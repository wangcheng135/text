序号：48397
名称：神效补天丹
组成：制耆5两，巴戟4两半，枸杞4两，熟地4两，杜仲3两，白术3两，白芍3两，人参3两，故纸3两，菟丝饼3两，块苓2两半，远志2两半，边桂2两，枣仁2两，萸肉2两，龙骨2两，当归2两，柏仁1两半，五味1两半，附子1两半，复盆子1两半，鹿胶3钱，黑驴肾1具，砂仁2两。
出处：《全国中药成药处方集》（吉林、哈尔滨方）。
主治：气虚血亏，百病蜂起，瘦弱难支，纳入便溏，气息微弱，动则作喘，腰酸痠软，健忘怔忡，自汗眩晕，寐而不实；并治肾虚阳痿，肾虚滑精，阳痿不举，举而不坚，见色自泄，精汁清冷，缺乏子嗣。
加减：None
功效：补气养血，添精壮阳。
用法用量：每服2钱，早、晚空腹各服1次，白水或淡盐汤送下。
制备方法：先将驴肾用滑石烫焦，再合诸药一处碾细，炼蜜为小丸，如梧桐子大，包于纸袋内严封，贮于瓷罐内。
临床应用：None
各家论述：None
用药禁忌：忌食生冷，相火盛者勿服。
附注：None
