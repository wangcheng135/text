序号：9493
名称：水银葶苈丸
组成：水银3两（3日3夜煮），葶苈子1升，椒目1升，衣鱼20枚，水萍1两，瓜蒂1两，滑石1两，芒消3两。
出处：方出《千金》卷二十一，名见《普济方》卷一九一。
主治：水肿，诸体肉肥厚，按之不陷，甚者臂粗，着衣袖不受，及十种大水。
加减：None
功效：利小便。
用法用量：初服1丸，次服2丸，次服3丸，次服4丸，次服5丸，次服6丸，至7日，还从1丸起，次服2丸，如是每至6丸，还从1丸起。始服药，当咽喉上有历子肿起，颊车肿满，齿龈皆肿，唾碎血出，勿怪，不经3-5日即消，所苦皆愈，亦止服药；若下多，停药以止利，药至五下止；病未愈，更服，病愈止。
制备方法：上药捣葶苈令细，下水银更捣，令不见水银止，别捣椒目令细，捣瓜蒂、水萍下筛，合和余药，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
