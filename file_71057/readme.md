序号：71073
名称：葱白补骨脂汤
组成：杜仲、远志、当归、川芎、陈皮、甘草、瞿麦、补骨脂、香附、牛膝、葱白、车前子。
出处：《陈素庵妇科补解》卷五。
主治：产前内积冷气，产时尿胞运动，产后腹胀如鼓，小便不通，闷乱欲死。
加减：None
功效：温下焦，行水。
用法用量：None
制备方法：None
临床应用：None
各家论述：冷气积于膀胱，产后血虚受冷，凝滞不行，以致腹胀闷绝，理或有之，热结寒亦结，《经》所谓脏寒生满病也。方以川芎、当归辛温以补血，远志、骨脂辛热以补暖下焦，陈皮、香附行气，车前、甘草、瞿麦行水，葱白开窍，杜仲、牛膝引药下行，其性最速。方中皆辛温通利之药，庶复冷气除，小便得温而自通矣。
用药禁忌：None
附注：None
