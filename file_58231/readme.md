序号：58246
名称：盐煎散
组成：良姜（炒）12两，苍术（去皮）12两，缩砂（去皮）5两，茴香（炒）5两，肉桂（去粗皮，不见火）2两，丁皮2两，橘红10两，甘草（炒）6两，青皮（去白）4两，山药半斤。
出处：《局方》卷三（续添诸局经验秘方）。
主治：男子、妇人一切冷气，攻冲胸胁，及前后心连背膂疼痛，转项拘急；或脾胃虚冷，不思饮食，时发呕吐，霍乱转筋，脐腹冷疼，泄泻不止；及膀胱成阵刺痛，小肠气吊，内外肾疼。
加减：None
功效：None
用法用量：每服2钱，水1盏半；入盐1字，煎至8分，空心食前服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
