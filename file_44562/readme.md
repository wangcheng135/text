序号：44577
名称：威灵仙丸
组成：威灵仙（去苗土）4两，牛膝（去苗，锉）1斤，天麻（锉）半斤，巴戟天（去心）4两（以上4味用好酒2斗浸两宿，焙），肉苁蓉2斤（洗，切，以前浸药酒银石器慢火熬成膏），何首乌1斤（米泔浸软，切片，于黑豆中蒸烂为度，焙干），石斛（去根）4两，海桐皮（锉）半斤。
出处：《圣济总录》卷五十二。
主治：肾脏风毒流注，腰脚疼痛。
加减：None
功效：久服辟风邪，调营卫，顺三焦，乌髭鬓，平补壮气活血，驻颜轻身健骨。
用法用量：每服20-30丸，空腹温酒送下，不嚼，1日3次。性温无毒。
制备方法：上药除苁蓉外，焙干为末，用苁蓉膏为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：神仙八味丸（原书卷一八五）。
