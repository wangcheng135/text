序号：2703
名称：天门冬煎
组成：天门冬2两半（去心焙），紫菀1两（去苗土），桔梗1两（去芦头），贝母1两半（煨微黄），赤茯苓3两，桑根白皮1两（锉），木通1两（锉，以上都捣罗为末），生地黄汁5合，生麦门冬汁3合，生姜汁1合，藕汁2合，酥2合，白蜜3合。
出处：《圣惠》卷四十六。
主治：咳嗽，痰唾稠粘，上气促急，心胸烦满，不能饮食。
加减：None
功效：None
用法用量：上将诸药汁及酥蜜，纳于银锅中。入诸药末，都搅令匀，以慢火熬成膏。不计时候以粥饮调下1茶匙。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“天门冬膏”。
