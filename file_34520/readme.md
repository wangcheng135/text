序号：34530
名称：驱风丹
组成：川乌1两（生用，不去皮尖，大者破作4边，小者破两边，锉如骰子大，用麻油煎令黄色，勿焦，去油，焙），草乌4两（择鸡心者，生用，不去皮尖，每个破作二片，择白净者用，心黑者不用，锉如骰子大。2两用油制，如制川乌法；2两用盐1两，拌和，水微淹着，浸3昼夜，1日1度漉，转候日数足取出去水，再用干盐1两同炒，候干，去盐勿令焦；再用卤汁浸3昼夜，去卤汁，炒干用，无卤汁，盐代），地龙4两（置阴润处少时，用纱帕裹定，轻搥，筛去土，取肉用，不必更秤），鸡心槟榔1两（锉，焙），何首乌1两（锉，焙），干木瓜1两（锉，焙），赤小豆1两（择去绿豆、黄豆）。
出处：《朱氏集验方》卷一。
主治：脚气，远年近日，不问阴阳二证，脚足肿痛不可忍；或患历节白虎，及筋脉拘挛等症。
加减：None
功效：None
用法用量：每服30丸，空心麝香、冷盐酒送下，不用温酒。服讫不可便食热物。若服药后觉遍身麻木，不可因麻木弃而不服。恐是丸数过多，或食热物太早。
制备方法：上为细末，用黄子醋一碗，好头面半两，煮糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌香菜、茵陈、豆豉，一切动风毒物。
附注：None
