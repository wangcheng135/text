序号：40126
名称：苓甘五味姜辛汤
组成：茯苓4两，甘草3两，干姜3两，细辛3两，五味半升。
出处：《金匮》卷中。
主治：气逆上冲，服茯苓桂枝五味甘草汤后，冲气即低，而反更咳胸满者。肺经感寒，咳嗽不已。
加减：None
功效：None
用法用量：上五味，以水8升，煮取3升，去滓，温服半升，每日3次。
制备方法：None
临床应用：None
各家论述：《金匮要略心典》：眼前汤己，冲气即低，而反更咳胸满者，下焦冲逆之气即伏，而肺中伏匿之寒饮续出也。故去桂枝之辛而导气，加干姜、细辛之辛而入肺者，合茯苓、五味、甘草消饮驱寒，以泄满止咳也。
用药禁忌：None
附注：五味细辛汤（《鸡峰》卷十一）、苓甘味姜辛汤（《普济方》卷一四○）、桂枝五味甘草去桂加姜辛汤（《张氏医通》卷十三）。
