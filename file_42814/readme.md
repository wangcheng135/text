序号：42829
名称：保命丹
组成：滑石1两，缩砂1两，青黛1两，山栀子1两，白茯苓1两，草龙胆1两，寒水石1两，甘草1两，管仲1两，黄芩1两，干葛1两，大豆（以上生用）1两，益智半两，地黄（生干者）半两，大黄半两，山豆根半两，桔梗半两，紫河车半两，马勃半两，薄荷半两，黄药子半两，花粉半两，百药煎半两，兰根（上生用）半两。
出处：《鸡峰》卷二十七。
主治：一切诸毒，并瘴疫、久年暑积、咽喉不利之患。
加减：None
功效：化解诸毒。
用法用量：每服1丸，细嚼，新水送下。
制备方法：上药各拣择令洁净，为末，用生蜜为丸，如鸡头子大，青黛为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
