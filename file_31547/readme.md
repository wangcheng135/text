序号：31556
名称：沉香大腹皮散
组成：连皮大腹子3两，沉香（锉）1两，槟榔（锉）1两，桑白皮（锉，微炒）1两，乌药（锉）1两，荆芥穗1两，陈皮（洗，去瓤，焙干）1两，茴香（炒）1两，白茯苓（去皮）1两，木通（锉）1两，紫苏子（微炒）1两，紫苏叶1两，甘草（炒）1两，干木瓜2两半（去瓤），枳壳（麸炒，去瓤）1两半。
出处：《御药院方》卷三。
主治：湿气郁滞经络，脚气肿满，沉重疼痛，筋脉不利。
加减：None
功效：宣通经络，使上下无碍，血气和平，腿脚轻利。
用法用量：每服5钱，水1盏，加生姜5片，萝卜5大片，同煎至7分，去滓，食前温服，1日2次；10日之后，1日1次。病愈即止。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：如无萝卜，用萝卜干一分（微炒，捣碎）同煎代之；如觉大便干燥，即服加减神功丸。
