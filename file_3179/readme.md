序号：3179
名称：天雄丸
组成：天雄3分（炮裂，去皮脐），人参半两（去芦头），丹参半两，沙参半两（去芦头），白花蛇1两（酒浸，去皮骨，炙令微黄），羚羊角屑半两，芎半两，白僵蚕3分（微炒），独活半两，防风3分（去芦头），牛膝3分（去苗），萆薢半两（锉），麻黄3分（去根节），甘菊花半两，天麻1两，桂心3分，当归半两，枳壳半两（麸炒微黄，去瓤），干蝎半两（微炒），蝉壳半两（微炒），细辛半两，白蒺藜半两（微炒，去刺），仙灵脾3分，白附子3分（炮裂），蔓荆子半两，阿胶3分（捣碎，炒令黄燥），麝香3分（细研）。
出处：《圣惠》卷二十二。
主治：柔风。皮肤虚缓，四肢不收，或时顽痹，腰脚无力。
加减：None
功效：None
用法用量：每服20丸，食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
