序号：63357
名称：清宫丹
组成：权壳4钱，寸冬1两，黄芩5钱，法夏4钱，花粉4钱，柴胡4钱，生石膏1两，桔梗5钱，薄荷3钱，朱砂4钱，山栀4钱，郁金4钱，云苓4钱，胆草4钱，羌活3钱，独活3钱，白参4钱，甘草4钱，犀角5钱，雄黄4钱。
出处：《全国中药成药处方集》（抚顺方）。
主治：温热病发疹，感冒发热烦渴，头疼身痛，干呕烦躁，寒热往来；麻疹初期，发热畏寒，隐疹潜伏，应出不出，烦热咳嗽；大头瘟、丹毒、头面红肿，发热畏冷，心烦欲吐，便秘神昏。
加减：None
功效：清凉解热，透表化毒。
用法用量：大人每服1大丸，5-10岁每服1小丸，2岁以下小儿每服小丸3分之1。用桑叶、薄荷、菊花煎汤送下；白水、茶水亦可。
制备方法：上为细末，炼蜜为丸，每丸1钱4分重，蜡皮封。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。忌腥辣厚味。
附注：None
