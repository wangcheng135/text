序号：40073
名称：肾沥汤
组成：羊肾1具，磁石5两，玄参4两，茯苓4两，芍药4两，芎3两，桂心3两，当归3两，人参3两，防风3两，甘草3两，五味子3两，黄耆3两，地骨皮2升（切），生姜8两。
出处：《千金》卷八。
主治：肾寒虚为厉风所伤，语言謇吃不转，偏枯，胻脚偏跛蹇，缓弱不能动，口言音混浊，便利仰人，耳偏聋塞，腰背相引痛。虚劳损羸乏，咳逆短气，四肢烦疼，耳鸣面黧黯，骨间热，小便赤黄，心悸目眩。
加减：None
功效：None
用法用量：上(口父)咀。以水1斗5升，煮羊肾取7升，下诸药，取3升，去滓，分3服，可服3剂。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：原书卷十九有泽泻。
