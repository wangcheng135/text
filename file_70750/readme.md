序号：70766
名称：紫菀膏
组成：枇杷叶、木通、款冬花、紫菀、杏仁、桑白皮各等分，大黄减半。
出处：方出《本草衍义》卷十八，名见《得效》卷五。
主治：肺热久嗽，身如炙，肌瘦，将成肺痨。
加减：None
功效：泻肺中积热。
用法用量：食后、夜卧各含化1丸。
制备方法：各如常制，同为末，炼蜜为丸，如樱桃大。
临床应用：肺痨：有妇人患肺热久嗽，身如炙，肌瘦将成肺痨，以枇杷叶、木通、款冬花、紫菀、杏仁、桑白皮各等分，大黄减半，各如常制，治讫，同为末，炼蜜为丸，如樱桃大。食后、夜卧各含化一丸，未终剂而愈。
各家论述：None
用药禁忌：None
附注：枇杷叶丸（《普济方》卷一五九）、紫菀丸（《金匮翼》卷七）。
