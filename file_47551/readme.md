序号：47566
名称：神仙大验备急黑神丸
组成：雄黄（研）1两，硇砂（研）1两，丹砂（研）1两，硫黄（研）1两，水银1两，先以慢火于生铁铫内溶硫黄销，次倾入水银，急以火箸搅，焰起即离火以湿布搭灭，候冷刮取，与上3件同捣研；取1湖南烧药罐子，先用六一泥固济待干，入上末，实按令平，连盖子泥4缝，只留1寸缝不泥；合慢火匀养1复时，加火近罐子烧令通赤，缝中有烟焰出，急抽火令人按盖子急泥合缝周遍，用净筛土窨定药罐子，不得令露透出药气，上以大盆合之；次日取出药捣罗，用湿重帛包裹，以净湿土内窨盆合，出火毒3日3夜，逐日起盆微洒水，日满取药，再研如粉，入后药末。犀角（镑）1两，鹿茸（酥炙，去毛）1两，牛黄（研）半两，天竺黄（研）1两，升麻1两，天麻1两，干蝎（酒炒）1两，木香半两，阿胶（慢火炙燥）2两，天南星（牛胆煮1复时，晒干）1两（上除研药外，捣罗为末）。
出处：《圣济总录》卷五。
主治：中风、急风，牙关急，口噤不开。
加减：None
功效：None
用法用量：每服2-3丸，用豆淋酒、生姜汁去滓研药，斡开口灌下；如人行5里，更1服；连3服，汗出解。
制备方法：上为末，用青州大枣蒸熟，去皮核，研膏为丸，如梧桐子大（合此药宜3月3日、5月5日，或腊日）。
临床应用：None
各家论述：None
用药禁忌：宜食生姜酒、粟米粥。
附注：本方方名，《普济方》引作“备急黑神丸”。
