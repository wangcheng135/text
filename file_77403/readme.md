序号：77419
名称：跌打损伤膏
组成：当归5两，川芎2两，白及3两，赤芍3两，大黄4两，苏木3两，血藤3两，郁金2两，良姜3两，刘寄奴3两，五加皮3两，青木香3两，羌活3两，姜黄3两，桑寄生3两，三棱3两，莪术3两，黄柏3两，血余3两，土鳖1两，红花2两，桃仁2两，川牛膝3两，玄胡3两，秦艽3两，防风3两，白鲜皮3两，大葱白1斤。
出处：《全国中药成药处方集》（沙市方）。
主治：一切跌打损伤，破皮流血等症。
加减：None
功效：None
用法用量：用时将膏药在火上烘融，摊开贴患处。
制备方法：上药切片，香油18斤，将药入油内浸泡，春秋二季7天，夏季4天，冬季10天，熬至药枯，去滓滤净，再熬至滴水成珠；再入炒黄丹，每斤油春、夏、秋8两，冬季7两，候冷，又入自然铜（醋煅）3两，乳香、没药各4两，血竭2两，儿茶2两，以上5味共研细末；又入樟脑、冰片各1两，上药7味下入膏内搅匀。
临床应用：None
各家论述：None
用药禁忌：非因跌打损伤致病及有痈疽症者，忌贴此膏；孕妇忌用。
附注：None
