序号：36210
名称：参鹿补膏
组成：参鹿补膏清膏1kg，砂糖1.29kg。
出处：《上海市药品标准》。
主治：阳虚畏寒，精神疲乏，气血不足，腰膝痠软。
加减：None
功效：益气养血，补肾壮阳。
用法用量：常用量每次1羹匙，开水冲服，1日2次。
制备方法：先将砂糖加水加热烊化，滤过，然后与参鹿补膏清膏混合，浓缩至比重1.365（热测），即得。密闭保存。
临床应用：None
各家论述：None
用药禁忌：None
附注：参鹿补膏清膏处方：红参80g、锁阳200g、淫羊藿300g、续断200g、狗脊（制）300g、墨旱莲草400g、白术（麸炒）300g、玉竹（制）100g、鹿肉100g、仙鹤草400g、鸡血藤800g、女贞子（制）600g、党参200g、熟地黄400g。以上14味药，先将红参水煎二次，每次3～4小时；鹿肉水煎4小时左右；再将参渣、鹿肉渣与余药同煎二次，每次3～4小时。参汁、鹿肉汁、药汁分别滤过，澄清，混合后浓缩至比重1.21（热测），即得。
