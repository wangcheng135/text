序号：38417
名称：油蜡膏
组成：珍珠1钱，头生儿胞衣1具。
出处：《本草汇言》卷十九引瓦氏祖传方。
主治：一切诸毒疽疮，穿筋溃络，烂肌损骨，破关通节，脓血淋漓，溃久不收之证。
加减：None
功效：None
用法用量：遇时患，以猪蹄汤淋洗毒疮净，将蜡油药以软抿子脚挑取，轻轻敷上，再以铝粉油膏贴之。
制备方法：以豆腐裹煮真珠一时许，拌灯草同研极细末；头生儿胞衣1具，以银簪穿孔数十，清水涤洗恶血净，火烘干燥，不可焦，研极细末，如不细，再烘再研，务要细如飞面者佳，如内有筋皮坚韧，研不细者，去之；白蜡1两，猪脂油1两，火上共熔化，和入胞衣末并真珠末，调匀，瓷器收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
