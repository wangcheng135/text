序号：36717
名称：固本锁精丹
组成：黄耆2两半，人参2两半，枸杞子2两，锁阳2两，五味子2两，石莲肉2两半，山药2两，海蛤粉2两半，黄柏2两（酒拌，晒干，炒黑色）。
出处：《古今医鉴》卷八。
主治：元阳虚惫，精气不固，梦寐遗精，夜多盗汗，遗泄不禁。
加减：None
功效：大补元气，涩精固阳。
用法用量：每服50丸，加至60-70丸，空心温酒或淡盐汤送下。
制备方法：上为末，用白术6两，水5碗，煎至2碗，倒过术汁另放；再用水4碗，煎至2碗，去滓，与前2碗同煎，熬至1碗如膏，收和前药末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：固本锁精丸（《准绳·类方》卷六）。《准绳·类方》有山茱萸、知母。
