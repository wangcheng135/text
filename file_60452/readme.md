序号：60467
名称：通灵黄金膏
组成：木香1两，当归（洗1两，焙）1两，金毛狗脊（去毛）1两，防风（去芦头）1两，白及1两，白蔹1两，香白芷1两，白术1两，乳香（别研）1两，松脂（别研）1两，枫香（别研）1两，杏仁（去皮尖1两，别研）1两。
出处：《杨氏家藏方》卷十二。
主治：打扑伤损，驴伤马坠，痈疽，瘰疬，鬼箭，骨疽，漏疮，软疖，眉疽，发背，脑疽，脚膝生疮，远年恶疮，臁疮、缠喉风，五般痔，漏耳，鼻内生疮，牙疼，耳痛。
加减：None
功效：None
用法用量：每先用膏药半分，蛤粉为衣，温酒送下；次用药摩病处。如损折者，以竹夹挟直，用药摩之；患缠喉风服药不下者，先用药于喉外摩之，候喉宽，然后服之；牙疼、齿浮出血者，以药填齿缝，如有清水吐之；耳内停风气，疼痛作声，纸捻紝药在耳内。
制备方法：上件除乳香、枫香、松脂外，各焙干锉细，用清油3斤，炼熟放冷，浸药于银石器内，文武火养3日，常似鱼眼，勿令大沸，恐损药力，候香白芷黄为度；滤过，别入净锅内，入黄蜡8两，细罗黄丹2两，次入已研者枫香、乳香，用槐、柳枝子不住手搅，再上慢火熬少时，候凝即成。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
