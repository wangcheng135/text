序号：71306
名称：紫石英丸
组成：紫石英1两（细研，水飞过），牛膝1两（去苗），柏子仁半两，阿胶半两（捣研，炒令黄燥），附子3分（炮裂，去皮脐），防风半两（去芦头），细辛半两，黄耆半分（锉），芎3分，杜仲1两（去粗皮，炙令黄，锉），熟干地黄1两，羌活（锉）3分，萆薢3分（锉），丹参1两，木香半两，人参半两（去芦头），麦门冬1两半（去心，焙），续断3分，泽兰3分，禹余粮3分（烧，醋淬7遍，细研），当归3分（锉碎，微炒），白芍药半两，桂心半两，石斛1两（去根，锉），鹿角胶1两（炙黄燥），甘草半两（炙微赤，锉）。
出处：《圣惠》卷七十。
主治：妇人虚损，血海风冷气，腰脚骨节疼痛，吃食减少，心神虚烦，气血不调，体瘦无力。
加减：None
功效：None
用法用量：每服30丸，食前以暖酒送下。
制备方法：上为末，入研了药，都研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
