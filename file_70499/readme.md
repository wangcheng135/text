序号：70515
名称：紫金丹
组成：黑附子半两（炮，去皮脐），丁香半两，硇砂半两，缩砂半两（去皮），当归半两，天南星半两（炮），半夏半两，肉豆蔻5个，自然铜1两（火煅，于醋内淬7遍），木香半两，防葵半两，青葙子半两，朱砂半两，水银1分。
出处：《博济》卷一。
主治：肾脏风，上攻下疰，虚肿疼痛；及妇人血风，血气流注，筋骨疼痛，或发寒热，口苦舌干，四肢烦倦，血海久虚。
加减：None
功效：补暖丹田，大进饮食，兼化痰涎。
用法用量：每服10丸，薄荷茶送下，或薄荷酒送下，1日2次。
制备方法：先将水银、朱砂、硇砂3味同细研，余即一处为细末，和匀，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
