序号：42940
名称：保真丸
组成：补骨脂1两（酒炒，研细末），人参1两3钱，茯苓1两，土炒白术1两5钱，炙甘草3钱（上4味以河水6碗，煎浓汁，去滓，和骨脂晒干听用），杜仲1斤（用盐水炒断丝，为细末），川芎8钱，当归1两5钱，酒妙白芍1两，熟地2两（上4味以水8碗，煎浓汁3碗，去滓，拌杜仲晒干），玫瑰膏子1斤（捣烂如泥。若干花瓣只用半斤，磨末听用），连腻皮核桃肉1斤（盐水炒，捣如泥）。
出处：年氏《集验良方》卷二。
主治：气血两虚，五劳七伤，遗精白浊，脾胃虚弱，阳痿腰痛，眼花头眩，吐血骨蒸，翻胃久嗽，盗汗，月经不调。
加减：None
功效：None
用法用量：每服1两，清晨淡盐汤送下；如吐血、骨蒸，童便送下。
制备方法：上用炼蜜2斤为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：如长服，不用人参，以玉竹一两（蜜炒）、黄耆一两（蜜炒）代之。
