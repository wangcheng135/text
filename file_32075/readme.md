序号：32084
名称：万应丹
组成：建青黛（水飞去渣，晒干）5钱，鸡肫皮（洗，炙干）1钱，牛胆消3钱，山栀仁（拣净仁，炒黑，碾细末）3钱，黄连末3钱，生黄芩3钱，真熊胆1钱，人中白（取经霜雪多年者，火煅3次）5钱，大红绒灰1钱，西牛黄1钱，雄黄1钱，青梅干（煅，存性）5钱（临时加才妙），硼砂3钱，枯矾2钱，儿茶3钱，铜青2钱，珍珠1分。
出处：《玉钥》卷上。
主治：咽喉口舌肿闭，并穿腮腐臭延烂。
加减：None
功效：None
用法用量：每用少许吹患处，日夜徐徐吹之，流出痰涎渐愈，如有腐臭，急用蚌水洗净，或用猪牙草、扁柏子和捣，加水去滓洗净，再以前药去青梅干，加滴乳香（去油），2钱，吹用。
制备方法：上为极细末，和匀。加真麝香5分，大梅片7分，再研和匀，入瓷罐内，以乌金纸塞紧口。
临床应用：None
各家论述：None
用药禁忌：None
附注：制青梅法：大青梅一斤，去核，略捣碎，入白矾、食盐各五钱，拌和，再加蜒蝤不拘多少，层层间之。一日夜取梅晒干，收尽汁再晒干，煅灰存性，临用加入。制胆消法：冬月约入朴消在黑牛胆内，挂在风前一百二十日，去皮用消。
