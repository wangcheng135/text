序号：47998
名称：神异膏
组成：归尾5钱，川芎5钱，赤芍2钱，生地黄4钱，防风5钱，羌活5钱，白芷5钱，玄参5钱，黄耆5钱，官桂3钱，桃仁49个，杏仁49个，木鳖子14个，何首乌3钱，牛子5钱，川山甲4钱，蜂房3钱，蛇退2钱，大黄2钱，黄柏2钱，乱发（男者）1团如鸡子，大槐柳皮49节（每长1寸）。
出处：《寿世保元》卷九。
主治：痈疽发背，诸疮毒，不拘已成已溃未溃者，皆可用之。诸疮溃脓后，不长肌肉，不合口者。
加减：None
功效：None
用法用量：诸毒甚者，每日换2-3次，中毒换1次。其药力方能胜毒。
制备方法：上用芝麻油3斤4两，将药入锅内浸，春5、夏3、秋7、冬10日，以桑柴文武火煎油黑色，以穿山甲浮起黑为度，绢滤去滓，再熬油，滴水成珠，陆续下黄丹14两，柳条搅不住手，成膏，软硬得所，再下乳香、没药各3钱，血竭3钱，降真香末3钱，次冷定，下麝香末2钱，水浸2-3日，去火性摊用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
