序号：61739
名称：接气丹
组成：沉香1两，硫黄2两（如黑锡丹砂子结，放冷，研为细末），黑锡（去滓秤）2两，牛膝半两（酒浸），白术半两（焙），苁蓉（酒浸）半两，丁香3钱，川楝子1两（去核用肉），木香1两，茴香1两（炒），肉豆蔻1两（煨），破故纸1两（炒），桂心1两（去粗皮），附子1两（炮，去皮脐），葫芦巴1两（炒），阳起石（煅）1两。
出处：《局方》卷五（淳祐新添方）。
主治：真元虚惫，阴邪独盛，阳气暴绝，或大吐大泻，久痢虚脱。
加减：None
功效：None
用法用量：每服50丸，温酒、盐汤空心送下。
制备方法：上药并砂子4两为细末，用糯米粉酒煮糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
