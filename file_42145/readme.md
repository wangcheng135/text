序号：42160
名称：青金丹
组成：天门冬（去心）1两，麦门冬1两，天麻1分，全蝎（大者）5个，牙消2钱，天竹黄2分，硼砂1钱，雄黄1钱，白附子2钱，紫粉4钱，辰砂1钱，片脑子半钱，生麝香半钱，金箔10片，银箔10片，水粉1两。
出处：《普济方》卷一一九。
主治：大人、小儿谵语，舌生白苔，痰盛气壅，烦渴引饮；及时行热极生风，并时行热疫，发热如火，连日不愈者。
加减：None
功效：清心解热，阴潮。
用法用量：每丸用麦门冬、生地黄、灯心、薄荷煎汤磨化服。合时加甘草、人参尤妙。
制备方法：上另研脑、麝、辰砂、水粉、金银箔，同前药末入白面2两，水为丸，以靛花为衣，如箸头大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
