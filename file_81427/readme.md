序号：81443
名称：鳖甲煎丸
组成：鳖甲3两（水浸3日3夜，去裙，米醋蘸炙令脆，为末），桃仁100个（汤浸，去皮尖，炒黄，细研），硇砂（汤化，去石，煎成霜）3分，上药用酽醋4升，砂盆中慢火熬成膏，更入后药：厚朴（去皮，生姜汁炙）1两，陈皮（去白）1两，神曲（炒）1两，肉桂（去皮）1两，肉豆蔻4个，槟榔2两，柴胡（去苗）半两。
出处：《普济方》卷二四九引《卫生家宝》。
主治：小肠气发不可忍，并治淋。
加减：None
功效：None
用法用量：每服20-30丸，以细切生葱热酒送下。
制备方法：上为末，再温前膏为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻、湿面。
附注：None
