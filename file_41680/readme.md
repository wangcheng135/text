序号：41695
名称：金精老奴丸
组成：木香5钱，灯心2钱，大蜘蛛7个，荜澄茄1两，胡桃肉1两，车前子1两，马蔺花1两，萆薢1两，牡蛎1两，韭子1两，木通1两，全蝎5钱，山茱萸5钱，破故纸5钱，桑螵蛸5钱，龙骨5钱，母丁香2两，紫梢花2两，蛇床子2两，肉苁蓉2两，菟丝子2两，白茯苓2两，淫羊藿（羊脂拌炒）2两，大茴香2两，巴戟2两，远志2两，当归2两，沉香3两，干漆3两，熟地5两。
出处：《全国中药成药处方集》（沈阳方）。
主治：中年虚损，五脏衰弱，元气不足，腿酸腰痛，下部虚寒，小肠疝气。
加减：None
功效：健脑添精益髓，促进生殖机能。
用法用量：每服1丸，空心淡盐汤送下；冬月温黄酒送下。
制备方法：上药除熟地捣膏外，余则碾面，加炼蜜为丸，2钱重。
临床应用：None
各家论述：None
用药禁忌：戒房事。
附注：None
