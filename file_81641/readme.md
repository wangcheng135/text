序号：81657
名称：黎洞丸
组成：三七2两，生大黄2两，阿魏2两，孩儿茶2两，天竺黄2两，血竭2两，乳香2两，没药2两，雄黄1两，山羊血（无真者，以小子羊鲜心血代之）5钱，冰片2钱5分，麝香2钱5分，牛黄2钱5分（以上各研细末），藤黄（以秋荷叶露泡之，隔汤煮10余次，去浮沉，取中，将山羊血拌入，晒干）2两。
出处：《金鉴》卷七十五。
主治：金疮跌扑伤，发背痈疽，恶疮，瘰疬，刑伤，疯犬咬伤，蜂、蛇、蝎毒。
加减：None
功效：续筋接骨，疏风活络。
用法用量：每服1丸，黄酒化下，外敷亦用黄酒磨涂此药。如在夏天修和，取天落水拌之为丸。
制备方法：取秋露水化藤黄，拌药捣千余下，如干，加炼蜜少许为丸，重1钱，黄蜡封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：(山黎)峒丸（《外科全生集》卷四）、黎洞丹（《种福堂方》卷四）、(山黎)峒丹（《青囊秘传》）、(山黎)峒丸（《北京市中药成方选集》）。《外科全生集》无山羊血、麝香。
