序号：557
名称：五痫丸
组成：天南星1两（炮），乌蛇1两（酒浸1宿，去皮骨，焙干称），朱砂1分（别研），全蝎2钱（去毒），半夏2两（汤浸7次），雄黄1钱半（研），蜈蚣半条（去头足，炙），白僵蚕1两半（炒去丝嘴），白附子半两（炮），麝香2字（别研），白矾1两，皂角4两（捶碎，用水半升，挼汁去滓，与白矾1处熬干为度，研）。
出处：《杨氏家藏方》卷二。
主治：癫痫朝发，不问久新。
加减：None
功效：None
用法用量：每服30丸，食后温生姜汤送下。
制备方法：上为细末，生姜汁煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：《北京市中药成方选集》：孕妇忌服。
附注：神应丸(《医级》卷八)、医痫丸、医痫无双丸(《北京市中药成方选集》)。按：本方方名，《景岳全书》引作“五痫神应丸”。《北京市中药成方选集》本方用法：用冷开水泛为小丸，用朱砂二钱五分、滑石粉二两五钱为衣，闯亮。每服一钱，日服二次，温开水送下。
