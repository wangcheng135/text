序号：9949
名称：止嗽青果丸
组成：藏青果960两，杏仁（去皮，炒）560两，苏子（炒）1200两，麻黄320两，贝母480两，桑皮640两，银杏1600两，法半夏1920两，石膏（煅）160两，黄芩640两，苏叶480两，冬花480两，甘草208两。
出处：《北京市中药成方选集》。
主治：肺热风寒，咳嗽痰喘，干嗽气呛，劳伤作嗽，老人犯喘，春秋举发。
加减：None
功效：润肺止嗽，化痰定喘。
用法用量：None
制备方法：上为细末，过罗，每160两细粉，兑冰片9钱5分。上药混合均匀，炼蜜为丸，重1钱2分，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
