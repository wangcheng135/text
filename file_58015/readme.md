序号：58030
名称：益母胜金丹
组成：熟地4两，当归4两，白芍（酒炒）3两，川芎1两5钱，牛膝2两，白术4两，香附（酒、醋、姜汁、盐水各炒1次）4两，丹参4两，茺蔚子4两，益母草1斤（酒、水各半，熬膏）。
出处：《医学心悟》卷三
主治：女人经血不调，及室女经闭成损。
加减：经水后期而来，小腹冷痛为寒，加肉桂5钱；经水先期妄行，自觉血热，加丹皮2两，酒炒条芩5钱；凡遇经水作痛，乃血凝气滞，加玄胡索1两。
功效：None
用法用量：每早服3钱，开水送下，晚服2钱，用清酒送下。
制备方法：炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：益母胜金丸（《医钞类编》卷十六）。
