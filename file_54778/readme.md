序号：54793
名称：桂心丸
组成：桂心1两半，人参1两半，熟地5两，茯神1两半（去木），柏仁3两（炒），远志1两半，辰砂1两半。
出处：《医略六书》卷三十。
主治：产后惊悸，脉虚弦细者。
加减：None
功效：None
用法用量：每服3钱，酒煎，去滓温服。
制备方法：上为末，炼蜜为丸。
临床应用：None
各家论述：产后血气虚寒，心阳亏损，而心气不振，心神失养，故惊悸不安焉。桂心补火以振心阳，人参扶元以补心气，熟地补血滋心明，茯神渗湿安神志，远志通肾交心，柏仁养心宁神，朱砂镇坠心气以安神也。蜜丸酒下，使血气内充，则虚寒自散，而心阳振发，心气自雄，安有惊悸不已之患乎。
用药禁忌：None
附注：None
