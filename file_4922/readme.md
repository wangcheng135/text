序号：4922
名称：木香槟榔丸
组成：槟榔1两半，木香1两半，枳壳1两半，青皮1两半，陈皮1两半，白蔻1两（去壳，炒），沉香1两，苏梗3两。
出处：《医略六书》卷三十。
主治：产妇气闭，脉沉滞者。
加减：None
功效：None
用法用量：每服3钱，米饮送下。
制备方法：上为末，粥为丸。
临床应用：None
各家论述：产妇气闭。素多忧怒，气滞于中，而肠胃不能传送糟粕，故大便不通。槟榔导滞气以疏利三焦，木香醒脾胃以调和中气，白蔻宽胸快膈，枳壳泻滞通肠，青皮破气以平肝，陈皮利气以和胃，苏梗顺气宽胸，沉香顺气降逆也；粥丸米饮下，使肝胃调和，则脾能健运，而诸气皆顺。
用药禁忌：None
附注：None
