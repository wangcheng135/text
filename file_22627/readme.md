序号：22635
名称：吴茱萸丸
组成：木香2分，青皮2分，白僵蚕4分，姜黄4分，泽泻4分，柴胡4分，当归身6分，炙甘草6分，益智仁8分，人参8分，橘皮8分，升麻8分，黄耆8分，半夏1钱，草豆蔻仁1钱2分，吴茱萸1钱2分，麦糵面1钱5分。
出处：《兰室秘藏》卷中。
主治：胸膈不通，善嚏，鼻流清涕，寒甚出浊涕，嚏不止，比常人大恶风寒，小便数而欠，或上饮下便，色清而多，大便不调，夜寒无寐，甚则为痰咳，为呕为哕，为吐为唾白沫，以至口开目瞪，气不交通欲绝者。
加减：None
功效：大理脾胃，调中顺气，外助阳气，内消阴火。
用法用量：每服20-30丸，温水送下，细嚼亦得。
制备方法：上为细末，汤浸蒸饼为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：勿多饮汤，恐速下。
附注：None
