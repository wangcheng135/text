序号：64368
名称：猪骨煎
组成：豮猪脊骨1条（去尾5寸，锉细，用好法酒6升，青蒿1握，乌梅10个，柴胡1两去芦，秦艽1两去芦，慢火同熬，耗一半，去滓，入蜜半斤，再熬成膏子），白茯苓（去皮）1两，当归（去芦）1两，川芎1两，人参（去芦）1两，肉苁蓉（酒浸）1两，巴戟（去心，酒浸）1两，五味子1两，牛膝（去芦，酒浸）1两，茴香（微炒）1两，破故纸（炒）1两，鳖甲（去裙，醋炙黄）半两，沉香半两，鹿茸（酒浸，醋炙）2两，附子（炮，去皮脐）2两。
出处：《百一》卷四。
主治：虚劳发热，热从脊骨上起。
加减：None
功效：None
用法用量：每服30丸，渐加至50丸，米饮送下，不拘时候。
制备方法：上为细末，用前猪骨膏子为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：猪膏煎（《袖珍》卷四）。
