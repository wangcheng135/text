序号：4460
名称：木香丸
组成：木香3分，白术3分，人参3分（去芦头），赤茯苓3分，吴茱萸半两（汤浸7遍，焙干，微炒），干姜半两（炮裂，挫），桂心3分，陈橘皮1两（汤浸，去白瓤，焙），诃黎勒1两（煨，用皮），槟榔1两，神曲1两（炒微黄），大麦蘖1两（炒微黄），当归半两（锉，微炒），川大黄1两（锉碎，微炒），桔梗半两（去芦头）。
出处：《圣惠》卷四十八。
主治：积聚气，脾胃虚冷，宿食不消，心腹气滞，胀满疼痛。
加减：None
功效：None
用法用量：每服30丸，食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
