序号：43570
名称：前胡汤
组成：前胡（去芦头）3两，细辛（去苗叶）3两，芎3两，麻黄（去根节，先煎，掠去沫，焙）3两，杏仁（去皮尖双仁，炒黄色）3两，枳壳（去瓤，麸炒）3两，防风（去叉）3两，泽泻3两，芍药3两，茯神（去木）4两，白术2两，旋覆花2两，甘草（擘破，炙）2两半，干姜（炮裂）2两半，半夏3两（水煮三五十沸，薄切放干，入生姜汁拌，炒黄色）。
出处：《圣济总录》卷一五六。
主治：妊娠伤寒，头痛恶寒，浑身壮热。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，入葱白1寸，同煎至6分，去滓，稍热服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
