序号：78158
名称：聚仙丸
组成：沙苑蒺藜1斤（先去刺，为末，取净末4两，余滓用水泡3-5日，取他汁浆，入锅内熬膏听用），莲蕊须4两（黄色者），芡实4两，枸杞2两，菟丝子饼2两，山萸肉（新者）4两，覆盆子（去蒂，酒拌蒸）2两，川续断（酒泡1宿，焙干）2两，金樱子（去外刺内瓤）3两，真龙骨（五色者，火煅，童便浸7次）5钱。
出处：《良朋汇集》卷五。
主治：遗精。
加减：求种子者，龙骨倍量，加金樱子熬2两。
功效：None
用法用量：每服3钱，盐汤、黄酒任下；求速效者，日进2服。
制备方法：上为细末，合一处，同前蒺藜膏，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
