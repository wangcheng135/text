序号：63746
名称：清热胜湿汤
组成：苍术5分（米泔制），黄柏5分（盐水炒），羌活5分，白芍5分（酒炒），陈皮5分（去白），牛膝5分（去芦，酒洗），木瓜5分，杜仲5分（姜汁炒），威灵仙5分，泽泻5分，甘草3分。
出处：《寿世保元》卷五。
主治：腰胯湿热作痛者。
加减：痛甚者，加乳香、没药为末各5分；水湿停下，入黑丑、槟榔各5分；血痛，加归尾、桃仁（去皮尖）各1钱，红花（酒洗）5分；冷风作痛，加熟附子1钱，虎胫骨（末）5分，减黄柏、泽泻各3分；倦怠，脚如沙坠，加苍术、防己、薏苡仁、白术各5分；游走而痛，加紫荆皮；湿热，加炒栀子；气不顺，加乌药；酸软，加牛膝、当归、地黄；肾虚，加破故纸（炒）5分。
功效：None
用法用量：加生姜3片，水煎服。
制备方法：上锉。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
