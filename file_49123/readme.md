序号：49138
名称：胜金丸
组成：干漆（炒烟尽）1两，乌头（炮裂，去皮脐）1两，狼毒1两，京三棱（锉碎，炒黄）1两，紫葳（即凌霄花）半两，没药半两，麒麟竭半两，槟榔半两（8味同为末），硇砂（别研如粉）1分，巴豆2-7枚（去皮心膜，研出油），大黄（单捣为末）1两，芫花（单捣为末）半两。
出处：《圣济总录》卷一五三。
主治：妇人血积，心腹(疒丂)痛气刺，少腹攻筑，或经候不行，肢节烦疼，痰逆癖块。
加减：None
功效：None
用法用量：每服7-10丸，当归酒调下。
制备方法：上12味，先以醋2升熬大黄、芫花2味末，次下巴豆、硇砂同熬令调；其余8味同为末，用熬者膏和剂；如干硬，更少入炼熟蜜同和剂令得所，为丸如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
