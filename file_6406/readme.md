序号：6406
名称：加味苍术膏
组成：苍术10斤（捣如泥，入大锅内，用水2桶，以文武火煮至10余碗，取出绢滤，入瓷罐内），人参4两，生地4两，熟地4两，黄柏4两，远志4两，杜仲4两，川芎4两，胡桃肉4两，川椒4两，故纸4两，当归4两，姜汁4两，青盐2两，朱砂1两，旱莲草汁2碗，白蜜2斤。
出处：《医学入门》卷七。
主治：男子精冷绝阳，妇人胞冷不孕。
加减：None
功效：通达诸身关节，流注遍体毛窍，养精养气养神，久服精满气盈，暖丹田，减相火，发白转黑，齿落更生。
用法用量：每空心酒、汤任下。
制备方法：上为末，共入膏内封固，大锅水煮，官香2炷为度，取出埋土中7日。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
