序号：8135
名称：加减定命丹
组成：蟾酥（酒浸1宿）1字，牛黄（别研）1字，朱砂（别研）1字，甘草（炙黄）1字，胡黄连1字，麝香（别研）1字，使君子肉1字，犀角屑1字，当归（洗焙）1字，天麻1字，细松烟墨（烧灰烟尽，地上出火毒）1字，羌活（去芦头）1字，全蝎2枚（去毒，微炒），棘刚子5枚（去壳取虫，微炒），半夏（汤浸洗7遍）1钱，天南星（牛胆制者）1钱，附子（炮，去皮脐）1钱，虎骨（蘸酒醋炙）1钱，乌蛇（酒浸1宿，取肉炙干）1钱，干姜（炮）1钱，丁香1钱，沉香1钱，肉桂（去粗皮）1钱，人参（去芦头）1钱，白茯苓（去皮）1钱，肉豆蔻（面裹煨熟）1钱，白术1钱。
出处：《杨氏家藏方》卷十七。
主治：小儿慢惊，瘈疭，目睛斜视，身体强硬，昏塞如醉；及治胎风成痫，发歇不定，荏苒经时。
加减：None
功效：None
用法用量：每服10丸，荆芥汤送下，不拘时候。
制备方法：上为细末，煮粟米粥为丸，如黍米大，青黛为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
