序号：36975
名称：夜合枝酝酒
组成：夜合枝5两，桑枝5两，槐枝5两，柏枝5两，石榴枝（生用，锉）5两，羌活（去芦头，别捣）2两，黑豆（用紧小者，生用）5升，糯米5升，细曲7斤半，防风（去叉，别捣）5两。
出处：《圣济总录》卷八。
主治：中风，手足不随，挛缩屈伸艰难。
加减：None
功效：None
用法用量：取清酒3合至5合，时饮之，常令有酒气，无令过醉，恐至吐，即悖乱正气。
制备方法：上10味，先以水5斗，将五枝同煎，取2斗5升，去滓，浸米、黑豆两宿，蒸熟入曲，与防风、羌活2味拌和造酒，依常法酝封21日，压去糟滓。
临床应用：None
各家论述：《慈禧光绪医方选议》：《本草图经》载：“合欢，夜合也”，夜合枝即合欢树皮。其叶似皂角，极细繁密，叶则夜合故名。《本草衍义补遗》称：“合欢，补阴有捷功，长肌肉，续筋骨”，但其治打扑伤损的功用，常为一般人忽略。《子母秘录》治搕损疼痛，取夜合花末，酒调服二钱匕。《圣惠方》治腰脚疼痛久不瘥，有夜合花丸，皆取其长肉生肌续筋接骨之功。本方取其树枝作酒，功能活络通经，故可治中风挛缩之症。
用药禁忌：None
附注：夜合枝酒（《本草纲目》卷三十五）、夜合酝酒（《金匮翼》卷一）。方中槐枝，《普济方》引作桂技。
