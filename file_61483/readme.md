序号：61499
名称：宿鸠丸
组成：宿鸠1只（去毛、羽、嘴、足、肠、胃，炙黄），羊肝1具（清油炼定血，去筋膜，劈作片子，焙），蔓荆子半斤（淘净，生绢袋盛，饭上炊3遍，焙），蜀椒（去目及闭口者，炒出汗）1两，楮实1两，仙灵脾1两，木贼1两，羌活（去芦头）1两，蝉壳（去土）1两，甘菊花（去萼）2两，荆芥穗2两，苍术（米泔浸，去皮）2两，蒺藜子（炒去角）2两。
出处：《圣济总录》卷一○九。
主治：肝肾气虚，眼生翳晕及见黑花。
加减：None
功效：None
用法用量：每服30丸，温酒或盐汤送下，不拘时候。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
