序号：28011
名称：补煮散
组成：黄耆6两，人参6两，独活6两，芎6两，防风6两，当归6两，桂心6两，萆薢6两，防己6两，茯苓8两，白术8两，丹参8两，附子（生用）4两，甘草（炙）4两，杏仁（去皮尖）、生地黄、生姜、磁石20分（碎如小豆）。
出处：《外台》卷十八引苏恭方。
主治：脚气。
加减：呕逆者，加半夏1两。
功效：None
用法用量：每服别以生姜2两，生地黄1两，杏仁14枚，碎，以水2升，煮取7合，布绞去滓，日晚或夜中服。
制备方法：上切，分为30服。
临床应用：None
各家论述：None
用药禁忌：忌猪肉、冷水、海藻、菘菜、生葱、桃、李、雀肉、醋物、羊肉、芜荑及饧。三日以后，并无禁忌。
附注：None
