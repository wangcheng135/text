序号：49968
名称：茯苓散
组成：白茯苓（去黑皮）1两，木香1两，杜仲（切，炒）1两，菖蒲1两，熟干地黄（焙）1两，柏子仁（研）1两，秦艽（去苗土）1两，菟丝子（酒浸，别捣，焙干）1两，青橘皮（汤浸，去白，焙）1两，诃黎勒皮（炮）1两，赤石脂1两，当归（切，焙）1两，五加皮（锉）1两，牛角（角思）（烧灰）1两，乌贼鱼骨（去甲）1两，艾叶灰（烧存性）1两。
出处：《圣济总录》卷一五一。
主治：妇人血海不调，因虚冷成积，月水不绝，及赤白带下，面色萎黄。腰脚沉重，胎气多损。
加减：None
功效：None
用法用量：每服2钱匕，糯米饮调下，温酒亦得，空心、食前服，日3次。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
