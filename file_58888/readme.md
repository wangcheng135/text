序号：58903
名称：秦王续命大八风散
组成：秦艽2两，防风2两，附子2两（炮，去皮），菖蒲2两，茯苓2两，牛膝2两，桔梗2两，细辛1两，乌头2两（炮，去皮），薯蓣1两，芎藭1两，远志2两半（去心），天雄1两（炮），石龙芮1两，蜀椒1两（去目及闭口者，汗），石斛2两，白芷1两，龙胆1两，白术1两，山茱萸1两，桂心1两，菊花1两，女萎1两，厚朴1两（炙），巴戟天1两，萆薢1两，牡荆子1两（无，用柏子仁），干漆1两（熬），肉苁蓉1两，五味子1两半，芍药1两，黄芩1两，白矾1两（烧汁尽），续断1两，白蔹1两，黄耆1两半。
出处：《千金翼》卷十六。
主治：诸风，五缓六急，或浮肿嘘唏，微痹，风虚不足，并风消胀满。
加减：心气不足，短气，加人参、甘草各1两；腹痛，加杜仲、羊肾各2两
功效：调和五脏，便利六腑。
用法用量：每服方寸匕，温清酒调下，日3次，不知，稍增之，可至2-3匕，以知为度。若苦心闷者，饮少冷水。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：宜断房室百日；禁生鱼、猪肉、菘菜；热人禁用。
附注：None
