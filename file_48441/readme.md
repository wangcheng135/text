序号：48456
名称：神效乾丹
组成：天雄3钱（去皮尖），雄精3钱，鸦片3钱，蜡酥3钱，母丁香（大者）4粒，人参3钱，樟脑（瓦上升净霜）3钱，乳香5分，没药（去油）5分，倭硫黄3钱。
出处：《纲目拾遗》卷二引《演揲儿集》。
主治：None
加减：None
功效：坚阳益肾种子，强筋力，和血脉。
用法用量：候干研擦。
制备方法：上为细末，用绢罗裹外。麝香2钱，研极细，另包。将白及（不拘多少，以敷用为度）放碗内，用滚水泡开，将白及装入绢袋内，拧汁去滓。再用苏合油3钱同白及汁和药调匀，将麝香末洒上，做成锭，放瓷盒内阴干，或将口封固，略晒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
