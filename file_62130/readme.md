序号：62146
名称：救苦汤
组成：桔梗1分，连翘1分，红花1分，细辛1分，当归身5分（夏月减半），炙甘草5分，苍术7分，草龙胆7分，羌活1钱，升麻1钱，柴胡1钱，防风1钱，藁本1钱，黄连1钱，生地黄1钱5分，黄柏1钱5分，黄芩1钱5分，知母1钱5分，川芎3钱。
出处：《兰室秘藏》卷上。
主治：眼暴发赤肿，睑高苦疼不任者。
加减：若苦痛，则多用苦寒者兼治本经之药，再行加减。若睛昏，加知母、黄柏1倍。
功效：None
用法用量：上（口父）咀。每服1两，水2盏，煎至1盏，去滓，食后温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：还阴救苦汤（《病机启微》卷下）。
