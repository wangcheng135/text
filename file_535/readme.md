序号：535
名称：五粒回春丹
组成：橘红3两5钱，胆南星3两5钱，防风3两5钱，竹叶3两5钱，茯苓2两，储蚕（炒）2两，甘草2两，金银花3两5钱，桑叶3两5钱，连翘3两5钱，麻黄2两5钱，薄荷2两5钱，蝉蜕2两5钱，山川柳1两5钱，赤芍2两5钱，川贝母2两5钱，杏仁（去皮，炒）1两5钱，羌活3两5钱，牛蒡子（炒）2两5钱（上共研为细粉，过罗），牛黄4钱，冰片4钱，麝香4钱，犀角1两，羚羊1两，珍珠（豆浆制）4钱，琥珀4钱。
出处：《北京市中药成方选集》。
主治：小儿毒热过盛，瘾疹不出，身热咳嗽，烦燥口渴。
加减：None
功效：清热解毒，透表豁痰。
用法用量：每5粒干重约2分。蜡皮封固。每服5粒，1日2次，用鲜芦根煎水送下；温开水亦可。3岁以下酌减。
制备方法：上为细末，过罗，混合均匀，用糯米6两熬水，泛为小丸，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：避风寒。
附注：None
