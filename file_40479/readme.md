序号：40489
名称：虎骨药酒
组成：虎骨（煎汁）8两，薏苡仁8两，当归5钱，五加皮5钱，青皮5钱，川芎5钱，白芍5钱，红花5钱，草乌（炙）5钱，红糖96两，萆薢8两，牛膝8两，羌活5钱，苍术5钱，独活5钱，白芷5钱，紫草5钱，橘皮8两，补骨脂（盐水炒）5钱，红曲20两，淫羊蕾（炙）8两，熟地8两，乌药5钱，防风5钱，丹皮5钱，木瓜5钱，枸杞子5钱，川乌（炙）5钱，萎蕤8两，蜂蜜160两，乳香面2两，官桂面5钱，檀香面5钱，砂仁面5钱，鹿茸面5钱，续断5钱，没药面2两，木香面5钱，紫豆蔻面5钱，蕲蛇5钱，麝香（研）2分，油松节4两，公丁香面5钱，佛手面5钱，人参面5钱，生杜仲5钱。
出处：《北京市中药成方选集》。
主治：风寒湿痹，筋骨疼痛，四肢麻木，腰膝乏力。
加减：None
功效：祛风活血，壮筋骨，强膝力。
用法用量：有酒量者，每服5钱，每日2次，温热后服；无酒量者酌减。
制备方法：上药装入罐内，加烧酒1760两，煮两沸，入缸封固，俟药滓沉淀后装瓶。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
