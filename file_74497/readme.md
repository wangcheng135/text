序号：74513
名称：摄风酒
组成：寻风藤1两，五加皮1两半，虎胫骨5钱，乌药5钱，石南叶5钱，苍术5钱，三角尖1两（石上生者佳），骨碎补7分半，青木香2钱半，威灵仙2钱半，川续断2钱半，当归2钱半，乳香2钱半，川羌活2钱半，北细辛2钱半，青藤根1两，川牛膝4钱，防风5钱，南木香2钱半，石薜荔1两（石上生者佳），苏木5钱，甘草节5钱，生姜1两半，大川乌1只（分作4份，只生用1份）。
出处：《得效》卷十三。
主治：白虎历节风，及诸般风湿，流注四肢，大风鹤膝，一切风疾，四肢拘挛，不能坐立，凡是骨节去处，皆尽浮肿，夜痛号哭，诸药不效。
加减：None
功效：None
用法用量：每日随意温服药酒，重者服至两料愈，不拘时候。如夏月恐停久作酸，只用半料，用小坛酒依上煮服。
制备方法：上锉，用无灰酒1坛，将药盛于布袋内放酒坛中，油纸封缚，仍以锅盛水，将坛于锅内用慢火自辰时煮至午时，连坛取出。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中苏木用量原缺，据《普济方》补。
