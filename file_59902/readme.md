序号：59917
名称：调经散
组成：吴茱萸1两半（去目、闭口，沸汤洗通3次），半夏1两（汤泡7次），当归1两（去芦，酒洗），人参1两半，麦门冬1两半（去皮），白芍药（京南者）1两，川芎（色如腊者）1两，牡丹皮1两，厚朴（去皮，不见火）1两，阿胶（蚌粉炒如珠子）1两，甘草（炙）1两。
出处：《普济方》卷三三六引《便产须知》。
主治：月候不调，或在月前，或在月后，或多或少，或逾月不至，或一月两来，不孕者。
加减：None
功效：None
用法用量：上（口父）咀。每服3钱，水1盏半，生姜5片，煎至8分，去滓，食前稍热服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
