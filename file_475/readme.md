序号：475
名称：五积丸
组成：沉香半两，木香半两，当归（洗，焙）半两，附子（炮，去皮脐）半两，青橘皮（去白）半两，丁香1分，大黄半两（酒浸，湿纸裹炮），缩砂仁1两，半夏半两（汤洗7次后，以生姜制曲），陈橘皮（去白）半两，京三棱半两（炮），蓬莪术半两（炮），槟榔1分（锉），胆矾半两（别研），细松烟墨半两（烧留性）。
出处：《杨氏家藏方》卷五。
主治：五种膈气，中脘痞闷，噎塞不通，饮食减少：积聚癖块，心腹作痛，一切沉积。
加减：None
功效：None
用法用量：每服20丸，加至20丸，食后临睡用橘皮汤送下。
制备方法：上药除胆矾外，并为细末，用肥枣50枚(去皮核)，入米醋2升，煮枣令烂，次下胆矾末，煮少时，与前药同和为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：按：《普济方》有厚朴(姜汁炙)半两。
