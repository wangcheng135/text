序号：70935
名称：小柴胡去半夏加栝楼汤
组成：柴胡8两，黄芩3两，人参3两，大枣12个（擘），甘草3两，生姜3两，栝楼根4两。
出处：《外台》卷五引《伤寒论》。
主治：疟发渴者，及劳疟。
加减：None
功效：None
用法用量：以水1斗2升，煮取6升，去滓，更煎取3升，温服1升，1日3次。
制备方法：上切。
临床应用：劳疟：伍某某，女，40岁，患劳疟已半年。每日下午开始畏冷，旋即头痛发烧，汗出口渴，小便短赤，舌红苔薄，脉弦细数，每次服奎宁可止，但遇劳即发。此体质虚弱，正不胜邪，拟扶正祛邪。用柴胡去半夏加栝楼汤：党参15g，柴胡10g，黄芩10g，栝楼根12g，甘草5g，生姜3片，大枣3枚，加醋炒常山10g。服3剂疟止。
各家论述：《金匮玉函经二注》：《内经》谓：渴者，刺足少阳。此证胃土被木火之伤，则津液涸而燥渴，故因柴胡、黄芩治木火，人参、甘草补胃，栝楼生津益燥，姜、枣发越荣卫。若劳疟由木火盛，荣卫衰，津液竭者，亦治以此。
用药禁忌：忌海藻、菘菜。
附注：小柴胡去半夏加栝楼根汤（《千金》卷十）、柴胡栝楼根汤（《御药院方》卷二）按：本方方名，《金匮》附方引作“柴胡去半夏加栝楼根汤”。
