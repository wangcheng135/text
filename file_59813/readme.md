序号：59828
名称：大黄左经汤
组成：大黄（蒸）、细辛（去苗）、茯苓、防己、羌活、黄芩、前胡、枳壳（麸炒，去瓤）、厚朴（去皮，锉，姜制，炒）、甘草（炙）、杏仁（麸炒，去皮尖，别研）。
出处：《三因》卷三。
主治：风寒暑湿流注足阳明经，使腰脚痹痛，行步艰难，涎潮昏塞，大小便秘涩，腹痛呕吐，或复下利，恶闻食气，喘满肩息，或自汗谵妄。
加减：腹痛，加芍药；秘结，加阿胶；喘，加桑白皮、紫苏；小便秘，加泽泻；四肢疮痒浸淫，加升麻。所加并等分。
功效：None
用法用量：每服4大钱，水1盏半，加生姜3片，大枣1个，煎7分，去滓，空腹热服。
制备方法：上锉散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
