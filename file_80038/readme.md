序号：80054
名称：摩风黄耆膏
组成：黄耆半两，当归（洗，焙）半两，防风（去芦头）半两，檀香半两，栝楼（去皮）半两，香白芷半两，甘松（去土）1分，零陵香1分，川芎1分，甘草1分，生干地黄1分，木香1分，藁本1分，白蘝1分，杏仁49枚（去皮尖），赤芍药1钱，麻油1斤（如水清者）。
出处：《杨氏家藏方》卷二十。
主治：None
加减：None
功效：嫩容，祛风。
用法用量：作面脂用。
制备方法：上锉，焙干，入油内慢火熬一伏时，去诸药不用，再称熬的油，每1两入黄明蜡4钱，再于火上化开蜡，熬少时，用新绵子滤去滓，盛于瓷器内。
临床应用：None
各家论述：None
用药禁忌：熬药时不得用铜、铁器，须在银、石器内熬。
附注：None
