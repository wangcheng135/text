序号：34399
名称：万病无忧酒
组成：防风1两，白芷1两，五灵脂1两，川牛膝1两，台芎1两，荆芥穗1两，甘草1两，天台乌药1两，八角茴香1两，木瓜1两，地骨皮1两，乳香5钱，没药5钱，南木香5钱，羌活5钱，赤芍药2两，五加皮两半，故纸2两（炒），自然铜2两（制），威灵仙2两，当归2两，紫荆皮半两，杜仲（制）半两，黑小豆2两（炒，去皮）。
出处：《医统》卷十一。
主治：None
加减：None
功效：除百病、理风湿，乌须鬓，健腰膝，快脾胃，进饮食，治伤损，补虚怯，消积滞，滋血养气，明目清心，去风活血和气。
用法用量：后开坛取酒，温饮之，或晨昏、午后，随量常饮不绝。
制备方法：生绢袋盛之，以无灰酒1大坛，入药在内，密封坛口，春、秋浸5日，夏3日，冬10日。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
