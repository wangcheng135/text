序号：83259
名称：乌鸡丸
组成：海金沙4两，侧柏叶（盐水炒，焙干）4两，香附（炒）1两，厚朴（姜炒）3两，当归（酒洗）3两，白术（去芦）2两，川芎2两，白芍（酒炒）2两，熟地2两，羌活1两半，防风1两半，人参1两，砂仁1两，粉草3钱。
出处：《寿世保元》卷七。
主治：妇人血海虚冷，经水不调，或前或后，或多或少，或时小腹疼痛，或下白带如鱼脑髓，或似米泔，不分信期，每日淋沥不止，头晕眼花，目眩耳鸣，面色萎黄，四肢无力，五心烦热，胸膈闷，不思饮食，肌肤减削。
加减：None
功效：None
用法用量：None
制备方法：上锉；用白毛乌肉膳鸡1只，不问3-5年俱好，宰后去肠屎、毛，将药末装入鸡肚中，放铜锅内，好酒5壶，水2瓶，文武火煮至干，取鸡去骨，取肉切细，同药晒干，为末，用粳米粉、酒、水煮糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
