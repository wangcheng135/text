序号：7854
名称：加减麻黄汤
组成：麻黄3钱，桂枝2钱，苏叶3钱，黄芩3钱（酒洗），芥穗3钱，滑石4钱，豆豉4钱，木通3钱，甘草1钱。
出处：《医学探骊集》卷三。
主治：伤寒初病，腿膝无力，头痛，发热恶寒，脉象浮洪而紧，亦有不紧者。
加减：None
功效：None
用法用量：葱头1个为引，水煎服。
制备方法：None
临床应用：None
各家论述：以麻黄为发汗之君药；佐以苏叶、芥穗清扬之品，发头部之汗，以桂枝、葱、豉温散之品，发四肢之汗；酒芩能清血中之热；滑石能清六腑之热；木通清心火通小肠，能引热从小便出，较之大黄利大肠，最为柔和；甘草能和中气，则汗出而病愈矣。
用药禁忌：None
附注：None
