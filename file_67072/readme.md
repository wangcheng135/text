序号：67088
名称：麻黄汤
组成：麻黄2两，射干2两，人参2两，茯苓2两，防己2两，前胡2两，枳实2两，半夏1两，犀角1两，羚羊角1两，青木香1两，橘皮1两，杏仁1两，升麻1两，生姜5两，独活3两，吴茱萸1升。
出处：方出《千金》卷七引苏长史方，名见《普济方》卷二四四。
主治：肿已消，仍有脚气者。
加减：若热盛喘烦者，加石膏6两，生麦门冬1升，去吴茱萸；若心下坚，加鳖甲1两。
功效：None
用法用量：上（口父）咀。以水1斗1升，煮取4升，分5次服，每服相去20里久，中间进少粥，以助胃气。此汤两口服1剂，取病气退乃止。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
