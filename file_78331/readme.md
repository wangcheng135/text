序号：78347
名称：蜀漆汤
组成：白薇2两，蜀漆2两，知母2两，甘草（炙）2两，苦参2两，升麻2两，龙胆2两，常山4两，大黄（别渍，后下）4两，鳖甲（炙）3两，石膏（碎）3两，茯苓3两，黄芩3两，香豉2合（裹），独蒜7颗（切），淡竹叶（切）1升。
出处：《外台》卷五引《救急方》。
主治：疟瘴疠，经百日或一年以上，服诸药不能愈者。
加减：None
功效：None
用法用量：以水1斗渍之，并春酒2升，合煮取3升，去滓，分3服，未发前1服，欲至发时又1服，皆温之。当发日勿见人，在一静房卧，药滓置病儿头边，仍以药汁涂手面，过时任出。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：忌肥腻、腥臊、滑物、生冷、海藻、人苋、大酢、菘菜、生葱、生菜。
附注：None
