序号：4040
名称：木瓜丸
组成：狗脊（去毛）6两，大艾（去梗，糯米糊调成饼，焙干为末）4两，木瓜（去瓤）4两，天麻（去芦）2两，当归（酒浸制）2两，萆薢2两，苁蓉（去芦，酒浸）2两，牛膝（洗去土，酒浸1宿）2两。
出处：《局方》卷五(宝庆新增方)。
主治：肾经虚弱，腰膝沉重，腿脚肿痒，疰破生疮，脚心隐痛，筋脉拘挛，或腰膝缓弱，步履艰难，举动喘促，面色黧黑，大小便秘涩，饮食减少，无问新久，并宜服之。
加减：None
功效：None
用法用量：每服20丸，渐加至30丸，空心、食前温酒吞下，盐汤亦可。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
