序号：50083
名称：茯神汤
组成：茯神（去木）3两，防风（去叉）2两，牛膝（去苗）2两，枳壳（去瓤，麸炒）2两，防己（锉）2两，秦艽（去土）2两，玄参（坚者）2两，芍药2两，黄耆（细锉）2两，白鲜皮（锉）2两，泽泻2两，独活（去芦头）2两，桂（去粗皮）1两半，五味子半升，人参半两，薏苡仁（炒）半升，麦门冬（去心，焙）半两，羚羊角（镑屑）2两，石膏（碎）半斤，甘草（炙，锉）1两半，磁石12两（烈火烧赤，醋淬10遍，淘用，别捣碎）。
出处：《圣济总录》卷九。
主治：中风，手足一边不收，精神健忘。
加减：None
功效：None
用法用量：每用药1两，磁石末半两，别人杏仁7枚（去皮尖，碎），以水4盏，同煎至2盏，去滓分2服，微热服之，空心并午时各1服。每自中春宜服，至季夏即住。
制备方法：上药除磁石外，为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
