序号：66460
名称：小儿化积膏
组成：橘皮1两，厚朴1两，芒消1两，甘草1两，甘遂1两，大黄1两，川乌（生）1两，草乌（生）1两，苍术1两，大戟1两，巴豆1两，芫花1两，郁金1两，水红花子1两，商陆1两，附子1两，莱菔子1两，三棱1两，莪术1两，官桂1两，干蟾1两，山甲1两。
出处：《北京市中药成方选集》。
主治：小儿停食积水，腹痛胀满，积聚痞块，身体消瘦。
加减：None
功效：消积聚，化痞块。
用法用量：微火化开，贴肚脐上。
制备方法：上药酌予切碎，用香油80两炸枯，过滤去滓，炼至滴水成珠，加黄丹30两，搅匀成膏，取出放入冷水中，出火毒后，加热溶化，另以甘遂3钱，芫花3钱，阿魏3钱，甘草2钱，大戟3钱，为细末，过罗。每16两膏油兑入上细末搅匀摊贴，大张油重4钱，小张油重2钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
