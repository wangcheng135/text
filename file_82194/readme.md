序号：82210
名称：薏苡根汤
组成：薏苡根（锉）3两，枳壳根（锉）3两，蒴藋枝2两（锉），吴茱萸1两（汤洗过，绵裹）。
出处：《圣济总录》卷八十四。
主治：脚气初发。
加减：None
功效：None
用法用量：上药纳釜中，以水2斗，煎至1斗2升，入浆水2斗，盐3两，更煎5-7沸，去滓，用新净瓦瓮子1口，可容5斗者，以板子阔3寸横着于瓮底，将煎得汁乘热倾入瓮中，候冷暖得所，于密室中就瓮蘸脚，频频以汤从骭面淋之，瓮外以糠火微温，其瓮中汤只可离脚面3寸，更互换汤，以得汗甚为度；若汗不多，可尽日蘸了，用生姜、木瓜各1两半（锉碎），同研如膏，涂于脚心，以蜡纸裹缠缚定，厚衣被盖覆取汗，更吃后木香汤。
制备方法：None
临床应用：None
