序号：12470
名称：白薇丸
组成：白薇1两，柏子仁（研）1两，附子（炮裂，去皮脐）1两，鳖甲（醋炙，去裙襕）1两，当归（切，焙）1两，黄耆（锉）1两，人参3分，桂（去粗皮）3分，石斛（去根）3分，芍药（炒）3分，牡丹皮3分，羌活（去芦头）3分，熟干地黄（焙）1两1分，肉苁蓉（酒浸，切，焙）1两1分，甘草（炙，锉）半两，芎藭半两。
出处：《圣济总录》卷一六四。
主治：产后蓐劳，寒热时作，肢体羸弱，饮食无味。
加减：None
功效：None
用法用量：每服20丸，温米饮送下，不拘时候。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
