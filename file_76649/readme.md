序号：76665
名称：葈耳散
组成：葈耳草（5月5日午时干地刈取，洗，晒干）。
出处：《千金》卷八。
主治：诸风。胃胀满，心闷发热；及三虫，肠痔。
加减：None
功效：None
用法用量：每服1方寸匕，酒送下，1日3次。作散若吐逆，可蜜和为丸，每服10丸（准前计1方寸匕数）。风轻易治者，日再服。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：若身体有风处，皆作粟肌出，或如麻豆粒，此为风毒出也，可以铍针刺溃去之，皆黄汁出尽乃止。五月五日多取阴干之，着大瓮中，稍取用之。此草辟恶，若欲看病省疾者，便服之，令人无所畏，若时气不和，举家服之，一周年服之佳，七月七、九月九皆可采用。
