序号：72300
名称：滋肾种子丸
组成：大怀熟地黄8两（用蒸不用煨煮者，须捣烂，另入群药），大怀山药4两，车前子4两（酒洗，晒干），山萸净肉4两（烘干，不宜炒），沙苑蒺藜2两（先用水洗去浮者，酒洗晒干），枸杞子3两（烘干，不宜炒），牡丹皮3两（粉口者，酒洗晒干），9蒸何首乌4两（用大黑豆蒸，不用料豆蒸，久蒸干），白莲须1两5钱（红莲勿用），家芡实2两，人参1两5钱，怀牛膝1两（酒洗，晒干），川萆薢2两（白色者，酒洗晒干），辽五味子1两5钱（烘干），菟丝子3两（洗去土，酒煮熟，晒干），杜仲1两5钱（盐水拌炒，去丝），鱼膘2两（蛤粉炒）。
出处：《幼科直言》卷六。
主治：None
加减：None
功效：种子。
用法用量：每日空心白滚水吞3钱。服药后随宜进饮食，将药压入下部。服药须恒，勿令间断。
制备方法：上为细末，炼白蜜成丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
