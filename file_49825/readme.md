序号：49840
名称：大内塞排脓散
组成：山茱萸1分，五味子1分，茯苓1分，干姜1分，当归4分，石韦4分，芎4分，附子2分，苁蓉8分，巴戟天8分，远志8分，麦门冬8分，干地黄8分，桂心3分，芍药3分，地胆3分，菟丝子3分，石斛5分，人参5分，甘草5分。
出处：《千金》卷二十二。
主治：发背痈肿经年，瘥后复发。此因大风，或结气在内，经脉闭塞，至夏月以来出攻于背，久不治，积聚作脓血，为疮内漏。
加减：None
功效：None
用法用量：每服方寸匕，酒送下，日3夜1。稍加之。常服，终身不患痈疖。
制备方法：上药治下筛。
临床应用：None
各家论述：《千金方衍义》：痈肿久败不瘥，或瘥后复发，而成内漏，总由中气告匮，气血乖离，不能化腐生新所致。故于温补肾脏气血药中，专赖地胆之破血攻毒，石韦之疏泄旺气，然后温补诸药，得以建内塞之功，寓泻于补之妙用。
用药禁忌：None
附注：None
