序号：80160
名称：毓麟丸
组成：白棉花子仁24两（用秋石1两6钱，加水溶化，浸1日晒干，再用陈酒浸片刻，取出，入木甑内锅上蒸半日，取出晒干，再用此法蒸棉花子仁黑色为度），熟地12两，潼蒺藜6两，线鱼胶6两，川萆薢4两，麦冬4两，五味子2两4钱，杜仲2两4钱，补骨脂2两4钱，杞子8两，当归3两2钱，牛膝3两2钱，茯苓5两，楮实子3两，柏子霜3钱。
出处：《饲鹤亭集方》。
主治：None
加减：如男有遗精，女有白带，去牛膝，加覆盆子2两4钱。
功效：填补精髓，妙合阴阳，求嗣得孕，益寿延年。
用法用量：男妇均可服。
制备方法：上为细末，用羊肾4条，盐酒浸，打烂为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
