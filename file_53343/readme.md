序号：53358
名称：健脾资生丸
组成：潞党参3两，炒白扁豆1两5钱，豆蔻仁8钱，川黄连（姜汁炒）4钱，炒冬术3两，莲子肉2两，六神曲2两，白茯苓2两，广橘红2两，山楂肉（蒸）1两5钱，炙甘草1两5钱，芡实1两5钱，广藿香1两，炒麦芽2两，怀山药2两，春砂仁1两5钱，桔梗1两，炒薏仁米，1两5钱。
出处：《全国中药成药处方集》（杭州方）。
主治：胃脾虚弱，食不运化，胸腔饱满，面黄肌瘦，大便溏泄，以及妇人妊娠呕吐，小儿疳积，神疲便溏。
加减：None
功效：健脾开胃，消食止泻，调和脏腑，滋养营卫。
用法用量：每服2-3钱，米饮汤或开水送下；妇人淡姜汤送下。
制备方法：上为细末，炼蜜为丸，或水为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
