序号：1992
名称：升阳益胃汤
组成：柴胡5分，葛根1钱，石莲子8分，茯苓1钱，升麻3分，当归1钱5分，丹皮1钱5分，川芎8分，白芍1钱，生地1钱5分，秦艽1钱，麦冬1钱5分，生草3分。
出处：《陈素庵妇科补解》卷一。
主治：经水不通，属二阳之病。
加减：None
功效：清心火，养脾血。
用法用量：None
制备方法：None
临床应用：None
各家论述：是方以升、柴、秦、葛升其清阳之气；石莲、麦、茯引入心经；四物、丹皮培养阴血；秦、葛引入于胃；苓、草引火下行，以通心气；丹皮、生地祛血中伏火，则阳明燥金不受伤，而水谷之气自能生律液以和营卫，月事必自通矣。
用药禁忌：None
附注：None
