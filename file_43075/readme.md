序号：43090
名称：养气活血丸
组成：大艾叶（炒焦，取细末）5两，干姜（炮，取末）2两半（用上醋2升半，无灰好酒2升，生姜自然汁1升，将姜、艾末同调于银器内，慢火熬成膏），附子（炮，去皮脐）2两半，白芍药2两，白术2两，椒红2两，川芎2两，当归（洗，焙）2两，紫巴戟（去心，糯米炒）2两，人参（去芦头）2两，五味子2两。
出处：《杨氏家藏方》卷十六。
主治：产后诸虚不足，劳伤血气，真元内弱，四肢倦乏，肌肉消瘦。及脾元虚损，不入饮食，或吐利自汗，或寒热往来。
加减：None
功效：None
用法用量：每服50丸，食前温酒或米饮送下。
制备方法：后9味为细末，入前膏子，并熟炒白面2两半，同和为剂，入杵臼内捣千下，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：养气活血丹（《女科百问》卷上）。
