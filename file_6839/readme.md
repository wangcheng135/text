序号：6839
名称：加减槟榔汤
组成：槟榔1两，香附子（去毛）1两，陈皮（去白）1两，紫苏叶1两，木瓜（去瓤）1两，五加皮1两，甘草（炙）1两（一方无香附子）。
出处：《普济方》卷二四○引《医方集成》。
主治：一切脚气脚痛。
加减：足痛而热者，加地骨皮煎；妇人脚气，多由血虚，加当归半两；室女脚痛，多由血实，加赤芍药1两半；如或大便秘结，虚弱者，加枳实，壮盛者加大黄；如脚痛不已加转筋者，加吴茱萸；痰逆或呕者，加半夏；小便不通，加木通；脚痛大便不通者，用此汤送下青木香丸亦可。
功效：顺气防壅。
用法用量：上(口父)咀。每服4钱，水1盏半，加生姜5片，煎至8分，去滓温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
