序号：47529
名称：神乌丸
组成：川乌（炮，去皮脐，切片，炒令变色）1两，虎胫骨（酥炙）1两，海桐皮1两，川萆薢1两，川牛膝（去芦，酒浸）1两半，肉苁蓉（酒浸）1两半，金毛狗脊（燎去毛）半两。
出处：《济生》卷三。
主治：远年近日干湿脚气。
加减：None
功效：None
用法用量：每服70丸，空心、食前温酒送下。
制备方法：上为细末，用木瓜膏为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：造木瓜膏法：先用好艾叶以盐水洒湿，蒸炊久，再洒再蒸，凡三次；用宣木瓜一个（生），去皮瓤，切下盖子作一瓮子，填艾叶在内，将盖子合定，再蒸极软，去艾不用，只将木瓜研细为膏。
