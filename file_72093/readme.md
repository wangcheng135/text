序号：72109
名称：搜病青黛丸
组成：青黛1分，槟榔1枚，木香1分，麝香半分（细研），黄连1两（去须），巴豆半两，川大黄半两（锉碎，微炒），鳖甲半两（涂醋，炙令黄，去裙襕），肉豆蔻1枚（去壳）。
出处：《圣惠》卷八十六。
主治：小儿气疳，腹内有积恶滞结之物。
加减：None
功效：None
用法用量：1-2岁儿，每服1丸，空心粥饮送下；3-4岁儿，每服3-4丸，每隔3日1服。取下恶物为效。次宜服诃黎勒丸补之。
制备方法：上先将黄连、巴豆2味，以淡浆水3碗，煮令水尽，候干，取出巴豆，去皮心研如膏，纸裹压去油，黄连晒干，然后与诸药共捣罗为末，用猪胆汁和丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
