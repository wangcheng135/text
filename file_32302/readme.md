序号：32311
名称：沉香汤
组成：沉香1两，柴胡（去苗）2两，秦艽（去苗土）2两，肉豆蔻（去壳）2两，白芷2两，黄耆（锉）2两，鳖甲（去裙襕，醋炙）2两，桔梗（炒）2两，桂（去粗皮）2两，当归（切，洗，焙）1两，芎1两，蓬莪术（炮）1两，麦门冬（去心，焙）1两，槟榔（锉）1两，芍药1两，人参1两，白茯苓（去黑皮）1两，海桐皮（锉）1两，枳壳（去瓤，麸炒）1两，甘草（炙）1两，熟干地黄（焙）1两，酸枣仁1两，木香1两，荆芥穗3两。
出处：《圣济总录》卷一五一。
主治：室女月水不调，或多或少，或断或绝，不快不利，攻刺疼痛，四肢无力，不思饮食，多困黄瘦，胸膈痞满。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，加生姜3片、乌梅半枚，同煎至7分，去滓温服，每日3次。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
