序号：3532
名称：太素神丹
组成：牡蛎（雪白，左顾极大者）1斤，硫黄1两，腻粉半两。
出处：《魏氏家藏方》卷七引刘德容方。
主治：久患痼冷，脏腑虚滑，痢下脓血；妇人血海虚冷，赤白带下，经候不时，久无子息；男子下部积冷，腰膝无力，寒疝，膀胱一切冷病。
加减：None
功效：None
用法用量：每服30-50丸，温米饮送下，食前服。
制备方法：上药先用炭3斤，烧牡蛎令通红，放冷，碾成粉，分为两处，各半斤。用大坩锅子1个，盐泥固济，只留口，以牡蛎4两实在锅子底，次将硫黄、腻粉同碾细，用无底小竹筒置牡蛎之上，锅子中心四边再以牡蛎实之，却取竹筒，要得不近锅子四边也，然后再以4两余牡蛎，实捺硫黄之上，去锅子口留2-3寸，周匝用熟火3斤簇，待锅子中焰出，以匙抄余牡蛎掺之，焰出又掺，以焰绝为度。放冷取出，再碾如粉。然后取大新砖1片，凿成1池子，深约半砖以上，将未经余煅牡蛎平分一半，实铺在池子底，次将已煅过硫黄、壮蛎在上，更将余一半牡蛎覆之，实捺平后，用新白瓦一口盖定，以木炭一秤周匝烧之，候火尽为度。却取出，于土内埋半日，令出火毒，研细，滴水为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
