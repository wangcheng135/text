序号：81113
名称：麝香金蟾丸
组成：黄连、黄柏、青皮、陈皮、三棱、莪术、槟榔、鹤虱、芜荑、川楝子（煨）、苦楝根等分。
出处：《医方类聚》卷二五五引《经验良方》。
主治：小儿一切疳疾。伤食呕吐，腹胀脾虚，不思饮食，小儿虫，赤白痢。
加减：None
功效：None
用法用量：疳泻黄白，紫苏木瓜盐米饮送下；疳虫蚀腹痛，史君子汤送下；伤食呕吐不止，藿香汤送下；退疳热，麦门冬汤送下；腹胀脾虚，木香陈皮汤送下；不思饮食，枣汤送下；取小儿虫，清油汤送下；赤白痢，干姜甘草汤送下；赤痢，甘草汤送下；白痢，干姜汤送下；常服磨积，米饮汤送下。
制备方法：上为细末，雄猪胆汁浸蒸饼为丸，如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名麝香金蟾丸，但方中无麝香，疑脱。
