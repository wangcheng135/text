序号：32073
名称：万应丸
组成：五倍子1两（去内虫屑），胡黄连1两，青皮1两（去白），陈皮1两（去白），黄柏1两，神曲1两，麦芽1两（净洗，焙干），三棱1两（炮，锉），莪术1两（炮，锉），芜荑1两，槟榔1两，龙胆草1两，川楝子肉1两，使君子肉1两。
出处：《活幼心书》卷下。
主治：诸般疳证，胃口有热，饮食不进，头发作穗，面色萎黄。
加减：None
功效：None
用法用量：每服30丸至50丸，或70丸，温米清汤送下，不拘时候或空心。儿小者，丸作粟谷大，粒数、下法同前。
制备方法：上除槟榔（不过火）、麦芽2味外，余12味锉碎，炒令微焦色，候冷，同前麦芽、槟榔研为细末，水煮面糊为丸，如麻仁大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
