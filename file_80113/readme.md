序号：80129
名称：缩脾饮
组成：白扁豆（去皮，炒）2两，干葛（锉）2两，草果（煨，去皮）4两，乌梅肉（净）4两，缩砂仁4两，甘草（炙）4两。
出处：《局方》卷二（宝庆新增方）。
主治：霍乱之后，因服热药大多，致添烦渴者。伏暑热，烦渴躁闷，干湿霍乱。
加减：None
功效：解伏热，除烦渴，消暑毒，止吐利。
用法用量：上(口父)咀。每服4钱，水1大碗，煎8分，去滓，以水沉冷服，或欲热欲温，任意服，代熟水饮极妙。
制备方法：None
临床应用：None
各家论述：《杏苑》：大暑烦渴，吐泻霍乱，用砂仁、扁豆、草果消暑气，止吐泻霍乱；乌梅收肺热；佐葛根生津液。
用药禁忌：None
附注：《得效》有生姜，无甘草。
