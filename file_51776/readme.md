序号：51791
名称：钟乳白泽丸
组成：白檀香（取末）1两，滴乳香（别研）1两，阳起石（煅令通红，研）1两半，附子（炮，去皮脐）1两半，钟乳粉2两，麝香（别研）1钱。
出处：《局方》卷五（续添诸局经验秘方）。
主治：一切虚劳之疾。丈夫诸虚损，五劳七伤，真气不足，元脏不固，神志俱耗，筋力顿衰，头目眩晕，耳内虚鸣，心腹急痛，气逆呕吐，痰嗽喘促，胸膈胀闷，脾泄下痢，遗精便浊，厥冷自汗，脉微欲绝；妇人血海虚冷，崩漏不止，赤白带下，经候不调，脐腹时痛，面无颜色，饮食不进。
加减：None
功效：久服补益精血，助阳消阴，安心神，定魂魄，延年增寿。
用法用量：每服1丸，以水1盏，煎至7分盏，空心热服。如急病，不拘时候。
制备方法：上和匀，滴水搜成剂，分作60丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
