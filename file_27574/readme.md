序号：27582
名称：补经固真汤
组成：白葵花（去萼，研烂）4分，甘草（炙）1钱，郁李仁（去皮尖，研泥）1钱，柴胡1钱，干姜（细末）2钱，人参2钱，生黄芩（细研）1钱，陈皮（留皮）5分。
出处：《兰室秘藏》卷中。
主治：妇人白带常漏，下流不止，诸药不效，心包尺脉微，其病在带脉。
加减：None
功效：None
用法用量：上件除黄芩外，以水3大盏，煎至1盏7分，再入黄芩，同煎至1盏，去滓，空心热服，少时以早饭压之。
制备方法：None
临床应用：None
各家论述：1.《济阴纲目》汪琪笺释：此方以用参、姜益阳，李仁、葵花润操；柴胡升清气于下，陈皮和胃气于中，生芩后煎，妙其清凉不滞也。_x000D_
2.《医略六书》：热伤元气，不能输化湿热而下注阴中，故为白带浸淫不已焉。人参扶元气之虚，黄芩泻湿热之溢，干姜温气化以从治，葵花泄湿热以清经，橘皮利气和中，李仁润燥解郁，柴胡升清以上行，炙草缓中州以益胃。水煎温服，使胃气调和，则元气来复而湿热顿化，经脉清和，何白带淫溢不已哉。
用药禁忌：None
附注：补阳固真汤（《医学六要·治法汇》卷七）、补真固经汤（《济阴纲目》卷三）、补真利经汤（《医略六书》卷二十六）。
