序号：80069
名称：豨莶丸
组成：豨莶。
出处：《政类本草》卷十一引《成讷方》。
主治：中风。
加减：None
功效：None
用法用量：每服20-30丸，空心温酒或米饮送下。服至二千丸，所患忽加，不得忧虑，是药攻之力；服至四千丸，必得复故；五千丸，当复丁壮。
制备方法：9蒸9晒，捣末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：火轮丹（《普济方》卷二二○引《鲍氏方》）、九蒸单豨莶丸（《医学入门》卷七）。江陵府节度使进豨莶丸方：臣有弟诉，年三十一，中风床枕五年，百医不差。有道人钟针者，因睹此患，曰：可饵豨莶丸，必愈。其药多生沃壤，高三尺许，节叶相对，其叶当夏五月以来收，每去地五寸剪刈，以温水洗泥上，摘其叶及枝头，凡九蒸九晒，不必大燥，但取蒸为度，仍熬，捣为末，为丸服。臣依法修合，与诉服，果如其言。钟针又言，此药与本草所述功效相异，盖出处盛在江东，彼土人呼猪为豨，呼臭为莶气，缘此药如猪莶气，故以为名。但经蒸晒，莶气自泯。每当服后，须吃饭3-5匙压之。五月五日采者佳。奉宣付医院详录。
