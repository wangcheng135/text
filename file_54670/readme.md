序号：54685
名称：栝楼汤
组成：栝楼根2钱，白甘逐1钱。
出处：《小儿药证直诀》卷下。
主治：小儿高热，惊风抽搐，癖结胀满。小儿慢惊。癖结胀满。小儿斑疹作搐。
加减：None
功效：None
用法用量：每服1字，煎麝香、薄荷汤调下，不拘时候。
制备方法：上用慢火炒焦黄色，研匀。
临床应用：None
各家论述：《小儿药证直诀类证释义》：白甘遂即蚤休，苦寒降泄，清热解痉，主治惊痫，摇头弄舌，胎风，手足抽搐等证，专用此一味，以治胎风，可见白甘遂是一味主痉的专药，钱氏用此加栝楼根治慢惊，是佐以润肺滑痰，解渴生津，使润而能收，猛而能缓，从二药性味分析，本方适用于小儿高热、惊风抽搐，但方前明言治慢惊，是谓治标之意。
用药禁忌：None
附注：栝楼散（《奇效良方》卷六十五）。
