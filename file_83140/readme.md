序号：83158
名称：乌龙丸
组成：乌蛇1条（全者，去首尾，约2寸作截子，轻捶动，浸酒1宿，去皮骨，焙干），荆芥穗4两，薄荷叶4两，威灵仙（只削取根上细茸，温水急洗过，令净，不得见火，只晒干）4两。
出处：《普济方》卷九十七引《经验方》。
主治：手足不遂，痰涎语涩。
加减：None
功效：None
用法用量：每服15丸至20丸，不拘时候，薄荷汤下，1日3服。服35日后，觉手足疼痛，乃是气通血行，不逾月自然平复，后每日只可两服1服。
制备方法：上为末，别用不蛀大皂角1斤，削去黑皮捶动，以长流水1斗浸1宿，洗净，揉取汁，生绢滤过，银石器中慢火上熬取5分，更入白沙蜜4两，再熬作稠膏，与药末拌和捣，可丸即丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
