序号：40526
名称：虎骨散
组成：虎骨（酥炙令黄）1两，乌梅肉1两，赤茯苓（去黑皮，锉）1两，肉苁蓉（酒浸，切，焙）1两，人参1两，鳖甲（去裙襕，醋浸，炙）1两，甘草（炙）1两，芍药1两，当归（切，焙）3分，前胡（去芦头）3分，白术1两，豉（慢火微炒）1两，紫菀（去土）1两，黄耆（锉，焙）1两，常山（炒）1分，枳壳（去瓤，麸炒）1分，犀角（镑）1分，知母（切，焙）1分，升麻3分，柴胡（去苗）3分，桔梗（炒）3分，天灵盖（酥炙）3分，桂（去粗皮）3分，木香3分，桃仁（汤浸，去皮尖双仁，炒）49枚。
出处：《圣济总录》卷十。
主治：白虎风，走注疼痛，发歇不定，积年不效者。
加减：None
功效：None
用法用量：每服2钱匕，温酒调下，空心、临卧各1服。若为丸，即炼蜜为丸，如梧桐子大。每服20-30丸，温酒送下。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
