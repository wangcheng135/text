序号：3161
名称：天麻煎
组成：天麻1（2）两（别捣，罗为末），附子（炮裂，去皮脐）1两，桂心1两，防风（去芦头）1两，白附子（炮裂）1两，独活1两，牛膝（去苗）1两，石斛（去根）1两，鹿角胶（捣碎，炒令黄燥）1两，补骨脂（微炒）1两，萆薢（锉）1两，当归1两，芎1两，山茱萸1两，白蒺藜（微炒，去刺）1两，海桐皮（锉）1两，仙灵脾1两，巴戟1两，沉香1两，木香1两，麝香1分（细研）。
出处：《圣惠》卷二十五。
主治：一切风。
加减：None
功效：暖脏腑，除风冷。
用法用量：每服1茶匙，食前以温酒调下。
制备方法：上为末。以无灰酒5升，入白蜜5合，同于银锅中煎令减半，先下天麻末，煎良久，次下诸药末，以柳木篦搅令稀稠得所，于瓷器中盛。
临床应用：None
各家论述：None
用药禁忌：忌生冷、鸡、猪、毒滑物。
附注：None
