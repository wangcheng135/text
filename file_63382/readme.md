序号：63398
名称：清神养荣汤
组成：川芎1钱5分，当归1钱5分，白芍药1钱，熟地黄1钱，人参1钱，茯神1钱，橘红5分，柴胡5分，羌活5分，甘草（炙）5分，香附子5分。
出处：《杏苑》卷八。
主治：神思昏愦，每日上午不得清爽，怕见明处，恶闻人语，至午后方可，常常腹疼，头亦昏重，睡卧惊惕，但加之劳动，或值月经来时，其症尤剧，此为不得遂意之所致也，主乎血虚。
加减：如心下不宽，加缩砂仁7枚，白豆蔻3分；如有痰，加南星、半夏（俱姜制）各5分；如气升，加沉香磨浓汁加入。
功效：None
用法用量：上(口父)咀。水煎熟，食前温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
