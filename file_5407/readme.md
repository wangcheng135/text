序号：5407
名称：二十五味流气饮
组成：荆芥穗1钱，川连3钱，归尾5分，黄柏5分，苦参2钱，连翘5分，地肤子2钱，防风1钱，独活1钱，羌活1钱，银花2钱，生地3钱，地骨皮2钱，鲜皮2钱，角刺1钱，丹皮2钱，滑石1钱，细辛1钱，寒水石2钱，牛子2钱，枳壳2钱，黄芩1钱5分，土茯苓1斤，川牛膝1钱5分，甘草节1钱5分。
出处：《人己良方》。
主治：鱼口便毒，疳疔顽癣。
加减：None
功效：None
用法用量：先将土茯苓切碎，用水1大锅，先煎，后入各药再煎至2大碗，分2次服之。须要连服，不可间服。

临床应用：None
