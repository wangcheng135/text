序号：5406
名称：二十五味药
组成：香白芷（醋炒）1两，紫金皮（醋炒）1两，刘寄奴1两，川当归（盐炒）1两，赤芍药（米泔浸）1两，黑牵牛1两，川牛膝（茶水浸）1两，生地黄（盐水浸，炒）1两，川芎1两，乳香1两，没药1两，破故纸（醋炒）1两，木通1两，自然铜1两，草乌（醋炒）1两，木香1两，川乌（火煨）1两，藿香1两，骨碎补1两，木贼1两，官桂1两，羌活1两，独活1两，熟地黄（盐水炒）半两，杜牛膝（茶水炒）半两。
出处：《奇效良方》卷五十六。
主治：跌扑损伤，骨碎骨折，筋断刺痛，不问轻重。
加减：骨不碎，金刀伤挫臼者，去自然铜，骨碎骨折，临好时用之，如早服致成他疾；孕妇，去草乌、川乌。
功效：None
用法用量：跌扑损伤，金刀箭镞，不问轻重，每服1丸，温酒磨化服，或细嚼，酒送下；刀伤全断，内损重者，以薄荷汤或木瓜汤、姜汤、灯心汤皆可送服。病在上食后服，病在下食前服，在中者不拘时服。
制备方法：上为末，炼蜜为丸，如弹子大，用黄丹为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：二十五味治损丸（《医统》卷七十九）。
