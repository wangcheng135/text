序号：49149
名称：胜金丹
组成：牡丹皮1两，川藁本1两，人参1两，川当归1两，白茯苓1两，赤石脂（别研）1两，香白芷1两，官桂1两，白薇1两，京藭1两，玄胡索1两，白芍药1两，白术（米泔浸1宿）1两，甘草（炙）半两，沉香（不见火）半两，没药（别研）半两。
出处：《百一》卷十八。
主治：妇人月水湛浊不通，久无嗣息，血癖气痛，四肢浮肿，呕逆心疼，虚烦劳闷，面色萎黄，崩漏带下，寒热蒸劳，头疼齿痛，血下无度，淋沥诸疾；产后胎结疼痛，伤寒烦渴，泻痢血晕，血劳筋挛，痰盛头疼，败血上冲，血刺泄泻，咳嗽喘急，咯血，血块起伏，气痞气膈，血作腰痛，小便不禁，子死腹中，失盖汗不出，血风，脚手痹顽，积年血风，半身不遂，凡产后诸疾并皆治之。
加减：None
功效：安胎催生。
用法用量：每服1粒，空心温酒送下。妊娠临月服5-6粒即易产；久无子息服20粒，当月有子。
制备方法：上件药材皆用温水洗净，捣罗为末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
