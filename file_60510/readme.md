序号：60525
名称：通明利气汤
组成：苍术（盐水炒）1钱，白术（瓦焙）1钱，香附（童便炒）1钱，生地黄（姜汁炒）1钱，槟榔1钱，抚芎8分，陈皮（盐水浸炒）1钱，贝母3钱，黄连（酒浸，猪胆汁炒）1钱，黄芩（同上制）1钱，黄柏（酒炒）1钱，栀子仁（炒）1钱，玄参（酒洗）1钱，木香5分，甘草（炙）5分。
出处：《回春》卷五。
主治：虚火上升，痰气郁于耳中，或闭或鸣；或痰火炽盛，忧郁病满，咽喉不利，烦躁不宁。
加减：None
功效：None
用法用量：加生姜，水煎，入竹沥同服。
制备方法：上锉作2剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方改为丸剂，名“通明利气丸”（见《便览》卷一）。《便览》有菖蒲（去毛）二钱，无竹沥。
