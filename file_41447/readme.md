序号：41462
名称：金莲种子仙方
组成：熟地黄（酒洗）3两，当归（酒洗）3两，白芍药（酒炒黄）3两，益母草3两，川芎（酒洗）3两，苍术（米泔水浸1宿）3两，蛇床子（酒洗炒）2两，条芩（酒洗）2两，覆盆子（炒）2两，玄胡索（微炒）2两，陈皮（水洗，去白）2两，丹参（水洗）2两，砂仁（去壳）1两5钱，山茱萸（酒浸，去核）5两，香附（四制）5两。
出处：《济阴纲目》卷六。
主治：血虚不孕者。
加减：如月信先期而至者，加黄芩、地骨皮、黄连各1两半，清水送下；如月信后期而至者，加黄耆1两、人参、白术各1两半，温酒或淡盐汤送下；如白带者，加苍术、白术、升麻、白芷各1两半，淡姜汤送下。
功效：种子。
用法用量：每服40-50丸，渐加至80-90丸，空心清米饮送下。
制备方法：上为极细末，用白毛乌骨雄鸡1只，预先喂养1月，勿令与雌鸡同处，临时将鸡缉死，不出血，干去毛，剖开去肠内污垢物并膆内宿食，肫内黄皮用酒洗净，一应时件仍入鸡肚内，不令见火，置缸内，入酒2斤，封固，重汤煮烂取出，割下净肉捣如泥，仍将鸡骨用酥油和原汁或酒炙炼为末，入前药末内拌匀，再用醋煮米糊，同鸡肉木臼内捣极细为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：金莲种子仙丹（《女科指掌》卷二）。本方加茴香，去熟地，名“梦熊丸”。
