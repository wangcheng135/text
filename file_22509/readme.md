序号：22517
名称：均气丸
组成：木香2两，胡椒2两，干姜（炮）2两，乌头（炮裂，去皮脐）2两，茴香子（炒）2两，荜澄茄2两，青橘皮（汤浸，去白，焙）2两，陈橘皮（汤浸，去白，焙）2两，蓬莪术（煨，锉）2两，桂（去粗皮）2两，牵牛半斤（拣净，炒，捣罗取末4两，余者不用）。
出处：《圣济总录》卷六十七。
主治：一切虚冷气，腹胁胀满，胸膈滞闷，呕吐酸水，不思饮食，脏腑滑泄，脐腹疼痛。
加减：None
功效：None
用法用量：每服15-20丸，炒生姜盐汤送下，不拘时候。
制备方法：上为末，生姜自然汁煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
