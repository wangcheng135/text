序号：26671
名称：延寿丸
组成：鹿茸3两（切片，燎毛，酥炙），沉香2两，苁蓉2两（酒浸），菟丝子2两（酒浸，蒸），杜仲2两（炒），当归2两（酒浸，焙），胡芦巴2两（炒），补骨脂2两（炒），枳实2两（去瓤），石莲2两（猪肉煮），续断2两（炒），枸杞子2两，五味子2两，川巴戟（去心）2两，胡桃仁14个。
出处：《袖珍方》卷二。
主治：元阳虚惫，诸虚不足，行履无力，肢体痠痛。
加减：None
功效：None
用法用量：每服50-100丸，空心盐酒送下。
制备方法：上为末，黄犬肉或羊肉2斤（切），用无灰酒并葱、椒、青盐、桂花、茴香同煮烂，入石臼杵成膏子，却入山药糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
