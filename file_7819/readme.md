序号：7819
名称：四黄涤痰丸
组成：川大黄4两（用竹沥1两，姜汁1钱，朴消3钱拌蒸3次），蒌仁4两（去油），蛤壳4两（煅，飞），橘红（炒）4两，茯苓3两，陈胆星3两（姜制，蒸），茅术（炒）3两，天麻2两（煨），浮石2两（煅，飞），蓬术2两（酒炒），白芥子2两（炒），薄荷叶1两6钱，石菖蒲1两，沉香1两，青黛（飞）1两，制半夏（竹沥、姜汁炒）6钱，川黄连（姜汁炒）5钱，天竹黄3钱，白蔻仁3钱，冰片1钱。
出处：方出《鸡鸣录》，名见《重订通俗伤寒论》。
主治：胸膈迷闷，气逆咳呛，及哮喘中痰。
加减：None
功效：化痰。
用法用量：每服1-2钱，开水送下。
制备方法：上为细末，以竹沥9分，姜汁1分为丸，如绿豆大，再用煨石膏5钱，牛黄2钱，辰砂1钱三味研细为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
