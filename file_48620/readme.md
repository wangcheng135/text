序号：48635
名称：神验摩风毒膏
组成：牛膝（去苗）1两，赤芍药1两，当归1两，白术1两，白芷1两，川椒（去目）1两，厚朴（去粗皮）1两，雷丸1两，半夏1两，桔梗（去芦头）1两，细辛1两，吴茱萸1两，附子（生，去皮脐）1两，木香1两，大腹皮1两，槟榔1两，酥2两，野驼脂5两，野猪脂5两。
出处：《圣惠》卷二十五。
主治：风毒积年，气脉不宣通，四肢挛急，肌肉顽痹，腹中百病。
加减：None
功效：None
用法用量：每服如枣大，于患处摩之，仍须避风；若腹痛，即取药如弹子大，空心酒化服。
制备方法：上锉细，以酒浸1宿，先煎猪脂，然后入诸药，从平旦至日入，以慢火煎之，其膏即成；以绵滤去滓，却入铛中，然后下酥及驼脂，待稍冷，收于瓷器中。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
