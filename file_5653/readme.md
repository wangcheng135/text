序号：5653
名称：加减白通汤
组成：附子1两（去皮脐），干姜1两（炮），官桂5钱，白术5钱，草豆蔻（煨）5钱，甘草5钱，人参5钱，半夏（炮）5钱。
出处：《医学启源》卷中。
主治：形寒饮冷，大便自利，完谷不化，脐腹冷痛，足胫寒逆。
加减：None
功效：None
用法用量：上(口父)咀。每服1两，水2盏半，加生姜5片、葱白5茎，煎至1盏2分，去滓，空心服。
制备方法：None
临床应用：None
各家论述：以附子大辛热，助阳退阴，温经散寒，故以为君；干姜、官桂辛甘大热，亦除寒湿，白术、半夏苦辛温胃燥脾湿，故为臣；草豆蔻、炙甘草、人参甘辛大温，温中益气；生姜辛大温，能除湿之邪；葱白辛温，以通上焦阳气，故以为佐。
用药禁忌：None
附注：None
