序号：57490
名称：润体丸
组成：当归3两，人参1两，黄耆1两，黄芩1两，桔梗1两，薄荷1两，石膏1两，苍耳子1两，玄参1两，僵蚕1两，蝉蜕1两，全蝎1两，黄连1两，乌蛇1条。
出处：《秘传大麻疯方》。
主治：鸡皮疯。因血燥气虚入风，致形如鸡皮，粗糙不润，以手磨则粗刺。
加减：None
功效：None
用法用量：先服驱风养血汤，又用防风、荆芥、归尾、苦参、黄柏烧汤洗浴后，用擦药方擦之，再服润体丸，每服100丸，空心好酒送下。
制备方法：上为末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：擦药方：水银、硫黄、雄黄各五钱、砒一钱、藜芦、黄连、黄柏各五钱，大枫子一两，共为末，后用香油调擦。
