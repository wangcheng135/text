序号：20468
名称：至圣来复丹
组成：灵脂2两，青皮2两，硫黄2两，消石（于瓷器内，文武火消，令匀，勿令太过，研细，慢火炒黄色），陈皮（不去白）2两，太阴玄精石1两。
出处：《幼幼新书》卷九引《养生必用》。
主治：阴阳不调，冷热相制，荣卫差错，心肾不升降，水火不交养，凡丈夫，女人、老寿、稚婴危急证候，胃气尚在者。如邪热炎上烦躁；冷气攻注疼痛；膈痞寒热不可忍；肾邪攻胁注痛，不可转动；诸霍乱吐泻，水谷汤药不住；大段吐逆，手足逆冷，脚转筋；着热烦躁，昏塞旋倒，不省人事；泻痢不问赤白冷热；非时吐逆气痞，食饮不下；小儿因惊成痫，发渴多日，变成虚风，作慢惊者。
加减：None
功效：补损扶虚，救阴助阳，常服和阴阳，益精神，散腰肾阴湿，止腹肋冷疼。
用法用量：小儿如麻子，看大小加减服之。每服20粒，病甚者30粒，轻者15粒；童稚10粒，婴儿3-5粒，新生1-2粒，化破，早晨粥饮送下。
制备方法：上为末，水煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：来复丹、正一丹（《同方》卷五吴直阁增诸家名方引铁甕城八角杜先生方）、养正丹、黑锡丹（《医宗必读》卷六）、二和丹（《杂病源流犀烛》卷四）、来复丸（《饲鹤亭集方》）。《局方》（吴直阁增诸家名方）：上用五灵脂、二橘皮为细末，次入玄精石末及前二气末拌匀，以好滴醋打糊为丸，如碗豆大。每服三十粒，空心粥饮吞下。甚者50粒，小儿3-5粒，新生婴儿1粒。小儿慢惊风或吐利不止变成虚风搐搦者，非风也，胃气欲绝故也。用5粒研碎，米饮送下。老人伏暑迷闷，紫苏汤送下。妇人产后血逆上抡闷绝，并恶露不止，及赤白带下，并用醋汤送下。应诸疾不辨阴阳证者，并宜服之。
