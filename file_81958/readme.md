序号：81974
名称：镇心丹
组成：苁蓉1两（焙干），牛膝1两（细锉，酒浸，焙），兔丝子1两（酒浸，煮研），五味子半两（拣），人参2两（去芦头），山药2两，鹿角霜2两，远志2两（去心），龙齿1两（飞），黄耆半两（蜜炙），茯苓2两（白者），石菖蒲半两，茯神2两（同茯苓一处用柏叶裹定蒸9次）。
出处：《卫生家宝》引俞山人方（见《普济方》卷十六）。
主治：忧愁思虑过伤，心气不足，恍惚惊悸，骨热诸劳，失精乱梦，飞尸鬼疰，肌瘦色黄，食衰倦怠，心腑不利，以致大恐所伤，及吐血便血，种种心疾。
加减：None
功效：常服安神，去百邪，调顺荣卫，补养真气。
用法用量：每服30丸，空心盐米饮或酒盐汤吞下，渐加至40丸；用辰砂为衣，食后人参汤下，闭目良久。
制备方法：上为细末，炼蜜为丸，如梧恫子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
