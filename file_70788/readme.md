序号：70804
名称：紫金膏
组成：蛇含石2两（用炭一秤，蛇含石用新铁铫盛，却入炭火中煅通红，用米醋2升，倾蛇含石入醋内），针砂5两（用水淘净，焙干，用米醋2升，同余粮于砂锅内煮干为度，却再入铁铫内，于炭煅通红，取出令冷，吹去灰，细研，全无砂声），禹余粮3两，赤石脂1两（细研，旋入），木香1两（切作片子，怀干），肉豆蔻1两（湿纸裹，炮香熟），川白姜1两（炮），川芎1两，白茯苓1两，羌活1两（去芦头），当归1两（去芦头，切片子，酒浸1宿），杜茴香1两（炒香），白蒺藜1两，川附子1只（重7钱者，炮，去皮尖），三棱1两（炮），青皮1两（去瓤），陈皮1两（去瓤），牛膝1两（切，酒浸1宿），官桂1两（怀干），蓬术1两（炮）。
出处：《传信适用方》卷三引赵师沔方。
主治：十种水气。
加减：None
功效：None
用法用量：每服30-50丸，空心、食前温酒送下。如不能饮酒，白汤亦得。
制备方法：上焙干为末，用无盐蒸饼汤泡为糊，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌盐三月，房事一年。患人若甲错、眼黑、脐凸、肚皮光，有青筋起，并不可下药。
附注：本方方名，据剂型，当作“紫金丸”。
