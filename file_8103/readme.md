序号：8103
名称：加减托里汤
组成：黄耆、人参、甘草、当归、白芍、白芷、山药、陈皮、木香、肉桂、糯米。
出处：《慈幼新书》卷五。
主治：痘疮实热之症，服寒凉而冰伏，呕吐泻利，或皮薄浆清，或塌陷无神，血色不活，躁痒烦渴，唇舌淡白洁清，大便不固者。
加减：泄泻，去当归，加升麻、诃子、肉蔻、炮姜、丁香；腹痛，加煨干姜、神曲；咳嗽，或烦渴，加麦冬、五味子；水泡作痒，加防风，倍白芷；抓烂，以松花粉或乔麦粉掩之；寒颤咬牙，四肢逆冷，唇舌淡白，加附子。
功效：None
用法用量：临服加人乳半杯，酒酿数匙，同饮。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
