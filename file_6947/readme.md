序号：6947
名称：四物加地榆汤
组成：生地炭5钱，小川芎1钱，白芍1钱半（醋炒），当归身3钱（醋炒），地榆炭3钱。
出处：《医略六书》卷三十。
主治：产后血崩，脉涩数者。
加减：脾虚，加焦术；气虚，加人参。
功效：None
用法用量：水煎，去滓温服。
制备方法：None
临床应用：None
各家论述：产后血室空虚，冲任失守，不能操蓄泄之权，而经血妄行，故血崩不止焉。生地凉血止血，地榆涩血定崩，小川芎行血海以升阳，白芍药敛阴血以固下，当归身养血以引血归经也。水煎温服，使经血内充，则冲任融和，而蓄泄有权，血无妄行之患，安有血崩之不止乎！脾虚加白术，气虚加人参，是因病制方，无不各有绳墨。
用药禁忌：None
附注：None
