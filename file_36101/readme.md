序号：36111
名称：参香散
组成：人参1两，黄耆1两，白茯苓1两，白术1两，山药1两，莲肉（去心）1两，缩砂仁半两，乌药半两，橘红半两，干姜（炮）半两，甘草（炙）3分，南木香1分，丁香1分，檀香1分，沉香2钱（一方有炮熟附子半两）。
出处：《三因》卷十三。
主治：心气不宁，诸虚百损，肢体沉重，情思不乐，夜多异梦，盗汗失精，恐怖烦悸，喜怒无时，口干咽燥，渴欲饮水，饮食减少，肌肉瘦瘁，渐成劳瘵。
加减：None
功效：常服补精血，调心气，进饮食，安神守中。
用法用量：每服4钱，水1大盏，加生姜3片，大枣1个，煎7分，去滓，食前服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
