序号：12810
名称：石斛丸
组成：石斛（去根），黄耆（锉）1两，鹿茸（去毛，酒浸1宿，酥炙）1两，地骨皮1两，附子（炮裂，去皮脐）1两，菟丝子（酒浸，别捣）1两，面三刀分，山茱萸1两1分，远志（去心）3分，熟干地黄（焙）3分，菖蒲（米泔浸1宿，锉，焙）3分，防风（去叉）3分，桂（去粗皮）半两，玄参1两。
出处：《圣济总录》卷一一四。
主治：肾虚耳内作声，或如蝉噪，或如风水声，诊其左手尺脉微而细，右手关脉洪而大。
加减：妊娠人去桂、附，加蜀椒3分，丹参半两。
功效：None
用法用量：每服30丸，空心温酒送下。以愈为度。
制备方法：上药将12味捣罗为末，入菟丝子末再罗，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
