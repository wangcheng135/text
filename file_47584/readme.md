序号：47599
名称：神仙风药酒
组成：秦归身1两，大川芎1两，片白术（去芦）5钱，白茯苓（去皮）5钱，大川乌（炮）5钱，软防风5钱，荆芥穗5钱，羊角天麻5钱，全蝎（炒）2钱，香白芷5钱，北细辛5钱，何首乌5钱，新草乌5钱，威灵仙5钱，金叉石斛5钱，川牛膝（去芦）5钱，川独活5钱，羌活5钱，麻黄节3钱，石楠藤5钱，薏苡仁1两，川干姜5钱，赤桂5钱，尖槟榔5钱，宣木瓜5钱，真石乳5钱，明没药2钱，川续断5钱，白苍术（米泔炒）1两，嫩黄耆5钱，两头尖5钱，南木香2钱，汉防己5钱，桑寄生5钱，赤茯神5钱，骨碎补5钱，甘草节3钱，虎骨胫（煅，乳浸）5钱。
出处：《寿世保元》卷五。
主治：虚寒之鹤膝，不能动履，肿痛难当，及风湿麻木。
加减：None
功效：None
用法用量：早晚随量饮酒。
制备方法：上合一处，用生头酒5斤，文武火熬熟，去火毒。
临床应用：None
各家论述：None
