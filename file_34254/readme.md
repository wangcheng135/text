名称：皂刺丸
组成：皂刺1两，桑寄生7钱，何首乌7钱，石楠藤7钱，白蒺藜7钱，五加皮7钱，地骨皮7钱，白鲜皮7钱，草乌5钱，枸杞5钱，牛蒡子5钱，归尾5钱，五灵脂5钱，蔓荆子5钱，胡麻子5钱，防风5钱，苦参5钱，虎胫骨5钱，地龙5钱，京墨5钱，木鳖5钱，天花粉5钱，白胶香3钱，乳香3钱，没药3钱。
出处：《医学入门》卷八。
主治：远年杨梅、痈、癣、顽疮，筋骨疼痛。
加减：痛甚，加麝香1字。
功效：None
用法用量：每服50丸，硬饭汤送下，每日2次。服2月断根。
制备方法：上为末，面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌狗肉、鱼腥、房事。
附注：None
