序号：37236
名称：定喘神奇丹
组成：人参2两，牛膝5钱，麦冬2两，北五味2钱，熟地2两，山茱萸4钱。
出处：《辨证录》卷四。
主治：痰气上冲于咽喉，气塞肺管，作喘而不能取息，其息不粗，而无抬肩之状，属气虚而非气盛者。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：此方人参宜多用，不用至二两则不能下达于气海关元，以生气于无何有之乡。非用牛膝不能下行，且牛膝能平胃肾之虚火，还能在补其下元之气也。麦冬益肺金，非多用则自顾不暇，何能生汪洋之水，以救燎原之炎耶！人喘则气散，非五味子何以能收敛乎。用熟地以益肾中之水也，肾水大足，自不去泄肺金之气，然非多加则阴不能骤生，而火不可以遽制。又益之以山茱萸，以赞襄熟地之不逮，自能水火既济，而气易还元也。
用药禁忌：None
