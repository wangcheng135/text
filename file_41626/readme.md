序号：41641
名称：金简煎
组成：乳香4两，牛乳1斗（银石器中盛，并乳香于重汤内煮，旋旋浇牛乳汁，以乳香化为度），赤石脂半斤（细研，水飞过，于纸箱内沙上渗干，入夹绢袋，盛于饭上炊9次，每次出焙干，又炊以饭熟为度），丹砂4两（研，水飞过，纸箱内盛，泣干），甘草半斤（为末），蜜（银器盛，重汤内熬，以绢袋滤过）4斤。
出处：《圣济总录》卷一九九。
主治：None
加减：None
功效：辟谷固气，却灭三尸，消荡九虫，安和五脏，镇守三田，轻身强志，拘魂制魄，延永胎根。
用法用量：每服抄1大匙许，仍须先均作120服，须先服药床；如要服药床，即忌盐醋；如要治中风瘫痪，半身不遂，口眼斜，语涩神昏者，分作140服，亦须先服药床。
制备方法：上6味，将蜜分作9分，每1分拌乳香等，入新竹筒内盛，大竹可3-4枚盛了；次用新桑白皮扎塞定筒口，令密，以黑豆、大麦水拌，铺在大甑底，上坐竹筒，次又以豆麦盖覆过，竹筒再盖覆如法，慢火炊1日，3-4次添水，炊至夜，取出筒，倾药出，以瓷器盛了，入焙笼内火焙1夜，或未干又焙，令极干；再取蜜1分拌药令匀，又以新桑白皮塞口了，再入水洒拌豆麦令匀，又依前再炊1日，如前用热水洒3-4次，炊1日至夜，依前取焙之；如此九炊九焙，放干成煎。
