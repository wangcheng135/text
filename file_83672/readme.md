序号：83690
名称：乌梅丸
组成：乌梅肉1两（微炒），鳖甲1两（涂醋，炙令黄，去裙襕），虎头骨1两（涂酥，炙令黄），天灵盖1两（涂酥，炙令黄），肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），恒山1两（锉），知母1两，川升麻1两，甘草半两（炙微赤，锉），柴胡1两（去苗），蜀漆1两，豉1合（炒干），桃仁49枚（汤浸，去皮尖双仁，麸炒微黄），枳壳1两（麸炒微黄，去瓤），猪苓半两（去黑皮），黄连1两（去须），犀角屑1分，地骨皮1两，木香1两，槟榔1两，栀子仁1两，川大黄1两（锉碎，微炒），麝香半两（细研）。
出处：《圣惠》卷五十二。
主治：劳疟，连年不愈。
加减：None
功效：None
用法用量：每服30丸，空心煎桑根白皮汤送下，晚食前再服。
制备方法：上为末，入麝香研匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
