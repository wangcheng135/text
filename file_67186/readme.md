序号：67202
名称：麻黄定喘汤
组成：麻黄（去节）8分，杏仁14粒（泡，去皮尖，研），厚朴（姜制）8分，款冬花（去梗）1钱，桑皮（蜜炙）1钱，苏子（微炒，研）1钱，甘草（生，炙）4分，黄芩1钱2分，半夏（姜制）1钱2分。
出处：《张氏医通》卷十三。
主治：寒包热邪，哮喘痰嗽，遇冷即发，脉浮紧数者。
加减：None
功效：None
用法用量：水煎去滓，以生银杏7枚捣烂入药，绞去滓，乘热服之。去枕仰卧，暖覆取微汗。
制备方法：None
临床应用：None
各家论述：《医略六书》：寒邪外束，热壅于内，升降失其常度，故胸膈逆满，喘急不已焉。麻黄开发肺邪，黄芩清肃肺热，苏子散痰解郁，杏仁降气疏痰，厚朴宽中散逆满，半夏醒燥温痰，桑白皮泻温热清肺，款冬花润肺燥散结，生甘草以和中缓急。水煎绞银杏汁服，俾痰热内消，寒邪外解，而经府清和，逆满无不退，喘急无不除矣。此疏邪降气之剂，为寒滞郁热喘满之专方。
用药禁忌：None
附注：None
