名称：前麓开翳散
组成：白蒺藜（炒，捣去刺）1两，苍术（洗，童便换浸2宿，晒）1两，蝉壳（洗，晒）半两，蛇蜕（去头尾及脊上一线皮，不堪用，用皂角水洗，新瓦焙）半两，菜花蛇皮半两，好川芎半两，杏仁（水浸，去皮）半两，防风半两，羌活半两，白芷半两，华阴细辛4钱，独活4钱，白附子（生）3钱，明烂石膏3钱，荆芥穗3钱，真蚌粉3钱。
出处：《直指》卷二十。
主治：眼生翳障。
加减：None

用法用量：每服1钱，沸汤点，茶清调，以舌浸于药中良久，毒涎自出；又别换药，食后临卧服。
制备方法：上为细末。
