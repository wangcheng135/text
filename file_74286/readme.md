序号：74302
名称：催生顺气饮
组成：当归、川芎、肉桂、木香、乌药、广皮、枳壳、冬葵子、红花、车前子、生芝麻。
出处：《陈素庵妇科补解》卷四。
主治：妇人临产，忽然胎反上冲心者，由气不顺，或胞浆先干，子道干涩，儿难转身，是以上逼也。
加减：None
功效：顺气催生。
用法用量：None
制备方法：None
临床应用：None
各家论述：妇人气血虚者，坐草努力太过，忽然胎反上逼，必有昏晕，不省人事之变。盖胎上冲，必有热血随之，惟顺气和血则胎自下。是方芎、归、红花行血和血，香、桂、乌、广、枳壳顺气行气，冬葵、车前、芝麻滑胎束胎。气顺、血和、胎滑三法具而催生之诀尽矣。
用药禁忌：None
附注：None
