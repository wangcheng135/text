序号：31031
名称：时令救急丹
组成：藿香叶8两，香薷4两，公丁香1两，沉香1两，白芷1两，茅茨菇1两，檀香2两，广木香1两5钱，木瓜3两，生神曲6两，厚朴（姜制）2两，茯苓（去皮）4两，红大戟（醋制）1两，千金子霜1两（上为细末兑入：冰片3钱，麝香1钱3分，牛黄3分，薄荷冰1钱5分，明雄黄面5钱）。
出处：《全国中药成药处方集》（天津方）
主治：时疫传染，中暑头昏，四肢厥冷，身烧头痛，肚胀腹痛，上吐下泻，干呕恶心。
加减：None
功效：清暑除疫，利湿行水，避秽排浊。
用法用量：每斤丸药用滑石粉2两4钱，朱砂面6钱研匀，用桃胶2钱化水上衣。每次服1钱，小儿酌减，白开水送下。
制备方法：上研细和匀，凉开水泛小丸。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
