序号：38075
名称：明眼金波膏
组成：宣连4两，蕤仁半两，杏仁49粒，金州柏3两（连、柏2味捶碎，以水2碗浸1宿，于器内熬取半碗，滤过）。
出处：《博济》卷三。
主治：睑眦赤烂，迎风泪出，或痒或痛。
加减：None
功效：去瘀肉，洗翳膜。
用法用量：点之如常法。
制备方法：上先将杏仁去尖双仁，以蕤仁于口内退皮，同杏仁研如粉，入前药汁内同熬，及1大盏，更滤过，入好蜜及药9分，入麝香1钱、白矾1字（飞过）、硇砂1字、空青3钱（如无，只以生青代之，略捶碎）、龙脑2钱（以绢袋子盛），在药内又熬及一半许，于冷水内滴，直候药在水上不散即成，用小瓶子1个，封闭令密，于饭甑上蒸三度，逐度于井内拔过，及冷为妙，瓷银器内收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：金波膏（《圣济总录》卷一○五）。
