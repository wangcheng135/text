序号：51763
名称：钟乳丸
组成：炼成钟乳3两，吴茱萸半两（汤浸7遍，焙干微炒），石斛1两（去根，锉），菟丝子3两（酒浸1宿，焙干，别捣为末），雄蚕蛾50枚（微炒），肉苁蓉3两（酒浸1宿，刮去皱皮，炙干）。
出处：《圣惠》卷三十八。
主治：丈夫衰老，阳气虚乏，手足常冷，心中少气，髓血虚耗，腰疼脚痹，体烦口干，不能饮食。
加减：None
功效：安五脏，补肠胃，息万病，下气消食，长肌肤，和中焦。
用法用量：每服30丸，空心及晚食前以温酒送下。服讫行数百步，更饮温酒3-5合。饮讫复行百余步以展药势，及吃干饭豆酱1日。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：不可闻见尸臭等气；勿食粗臭陈恶之物。
附注：None
