序号：83931
名称：乌蝎丸
组成：乳香5钱，没药（另研）5钱，地龙（去土）5钱，全蝎（去足翅）5钱，草乌5钱，乌药（炒）1两，麝香1两，蜈蚣1条（去足，炒），川乌2只（生用，去皮尖）。
出处：《得效》卷十三。
主治：手足拳挛，痛不可忍者。
加减：None
功效：None
用法用量：每服7丸、10丸、15丸，用麝香少许，空心好小酒送下。服至7日略利，至半月或满身发风丹，经月方没，多服其病全安。后常用生川乌、没药浸酒，1日2次，通气驱风汤加乳香、没药、生川乌、麝香少许，酒调服；不饮，木瓜汤送下。
制备方法：上为末，面糊为丸，如梧桐子大。
临床应用：痹证：《良方》云，吉安城内李秀才妻，年六十五岁，于辛亥年忽患手足不能屈伸，人以为历节风，壬子年二月，得永新州龙起宗出此方，合一料，服至七日，略溏利；至半月，风丹先从臀片上起，至遍身，七日方没，又服药至四分之一，其病全安，手足可行，至今无恙。
各家论述：None
用药禁忌：忌热食一时。
附注：None
