序号：69982
名称：滋荣活络汤
组成：川芎1钱半，当归2钱，熟地2钱，人参2钱，黄耆1钱，茯神1钱，天麻1钱，炙草4分，陈皮4分，荆芥穗4分，防风4分，羌活4分，黄连8分（姜汁炒）。
出处：《傅青主女科·产后编》卷上。
主治：产后血少，口噤，项强，筋搐，类中风症。
加减：有痰，加竹沥、姜汁、半夏；渴，加麦冬、葛根；有食，加山楂、砂仁以消肉食，神曲、麦芽以消饭食；大便闭，加肉苁蓉1钱半；汗多，加麻黄根1钱；惊悸，加枣仁1钱。
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：《胎产指南》有麦冬、草果，无甘草。《产宝》本方用法：用水二盏，煎至七分，稍热服。
