序号：51270
名称：药枕
组成：蔓荆子8分，甘菊花8分，通草8分，防风8分，羚羊角（屑）8分，犀角8分，石菖蒲8分，细辛6钱，白芷6钱，芎藭6钱，藁本6钱，白术4分，黑豆5合（拣择净）。
出处：《医部全录》卷三三一引《保生要录》。
主治：头风目眩。
加减：None
功效：None
用法用量：上为细末，相拌令匀，以生绢囊盛之；欲其气全，次用碧罗袋盛之如枕样，纳药直令紧实，置于盒子中，其盒形亦如此，纳药囊，令出盒子唇1寸半，晚来欲枕时，揭去盒盖，不枕即盖之，使药气不散。枕之日久，渐低，更入药以实之，或添黑豆，令如初，3-5月后，药气散则换之。

临床应用：None
各家论述：None
用药禁忌：None
