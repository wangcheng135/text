序号：44146
名称：复原固本丸
组成：北枸杞1两，红枣皮1两半，莲花须1两，淮牛膝1两，川续断（上皆酒蒸）1两，拣归身1两半，牡丹皮（2味酒洗）2两，川杜仲1两，光泽泻1两半，川萆薢（土盐水洗）2两，北五味（蜜蒸）1两，白云苓（乳蒸）1两，天门冬（去心）1两半，淮山药（微炒）1两，芡实米（微炒）1两，熟地黄1两，川黄柏1两，败龟版（酒炙）10两，真虎骨（酒炙10次）10两。
出处：《疯门全书》。
主治：年四十以上，六脉微细，气血衰败。
加减：None
功效：生气血，扶元神，健筋骨，活经络，润颜色。
用法用量：每早空心服4钱。
制备方法：将虎骨、龟板2味打碎，用佳酒24碗，熬至3-4碗，去滓取汁。预将余药为末，入汁内，打糊为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
