序号：47699
名称：卫生防疫宝丹
组成：粉甘草10两（细末），细辛1两半（细末），香白芷1两（细末），薄荷冰4钱（细末），冰片2钱（细末），朱砂3两（细末）。
出处：《衷中参西》上册。
主治：霍乱吐泻转筋，下痢腹疼，及一切痧症。头疼，牙疼，心下、胁下及周身关节经络作疼。气郁、痰郁、食郁、呃逆、呕哕。防一切疫疠传染。
加减：None
功效：醒脑养神。
用法用量：若治霍乱证，宜服80丸，开水送下；余证宜服40-50丸。服后均宜温覆取微汗。若平素含化以防疫疠，自1丸至4-5丸皆可。
制备方法：先将前5味和匀，用水为丸，如梧桐子大，晾干（不宜日晒），再用朱砂为衣，勿令余剩，装以布袋，杂以琉珠，来往撞荡，务令光滑坚实，如此日久，可不走气味。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
