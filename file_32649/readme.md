序号：32658
名称：羌活散
组成：羌活1两，天麻1两，麻黄（去根节）1两，香附子1两，人参（去芦头）1两，胡麻子半两，细辛半两，藿香半两，牛膝半两，犀角屑半两，芎半两，桂心半两，当归半两，天雄（炮裂，去皮脐）半两，蝉壳半两，白附子（炮裂）半两，地龙（微炒）半两，乌蛇肉（酒浸，炙微黄）半两，晚蚕蛾1分（微炒），干蝎1分（微炒），麝香1分（细研）。
出处：《圣惠》卷二十五。
主治：一切风。
加减：None
功效：None
用法用量：每服1钱，以温酒调下；薄荷汤调服亦得。
制备方法：上为细散，研入麝香令匀。
临床应用：None
各家论述：None
用药禁忌：忌猪、鸡、毒鱼等。
附注：None
