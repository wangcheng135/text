序号：58876
名称：秘真丹
组成：菟丝子（酒浸，炒）1两，韭子（炒）1两，柏子仁1两，龙骨（煅）5钱，牡蛎（煅，醋淬）5钱，山茱萸（去核取肉）5钱，赤石脂（煅）5钱，补骨脂1两（炒），远志（去心）7钱5分，巴戟天（去心）7钱5分，覆盆子7钱5分，枸杞子7钱5分，黄柏（盐酒炒黑色）7钱5分，山药7钱5分，芡实（去壳）1两，杜仲（姜汁炒丝断）1两，金樱子（半青黄者，去刺核取肉，焙干）2两，干姜（炒黑色）1两，鹿角胶1两5钱（炒成珠）。
出处：《医学正传》卷六。
主治：好色肾虚，遗精梦泄，白淫白浊。
加减：None
功效：None
用法用量：每服100丸，空心姜盐汤送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：秘真丸《不居集》上集卷十九）。本方方名，《医统》引作“经验秘真丹”。
