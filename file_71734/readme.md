序号：71750
名称：滋阴汤
组成：熟地2钱，山药1钱5分，麦冬（去心，微炒）8分，当归（酒洗，去尾）1钱3分，白芍（酒炒）1钱，甘草（炙）6分，阿胶（蛤粉炒）1钱，茯苓1钱，杜仲（淡盐水炒）1钱，丹参1钱3分。
出处：《会约》卷九。
主治：肝肾虚弱，不时失血，背痛咽干，咳嗽便短，倦怠遗精。
加减：咽干而五心热者，加元参1钱2分；骨蒸多汗者，加地骨皮1钱3分；血热妄动者，加生地1钱5分，青蒿1钱；阴虚不宁者，加女贞子1钱5分；咳嗽有痰者，加款冬花1钱，川贝母（微炒，研末）1钱；血来盛者，童便1杯，藕节汁或丝，茅根汁合服。服之而顺，可以多服，但中时必须以温脾汤以佐之。
功效：None
用法用量：水煎，早、晚服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
