序号：48560
名称：神效麝香丸
组成：麝香半两（细研），朱砂1两（细研，水飞过），牛黄半两（细研），天麻1两，羌活1两，芎1两，独活半两，防风1两（去芦头），干蝎1两（微炒），白僵蚕1两（微炒），甘菊花1两，天南星3分（炮裂），败龟2分（涂酥，炙微黄），白花蛇3分（酒浸，去皮骨，炙微黄），桂心3分，附子3分（炮裂，去皮脐），木香3分，蔓荆子1两，人参3分（去芦头），地龙1两（微炒），海桐皮3分（锉），干姜半两（炮裂，锉），当归3分（锉，微炒），虎胫骨3分（涂酥，炙令微黄）。
出处：《圣惠》卷二十五。
主治：一切风，及肢节走痛不可忍者。
加减：None
功效：None
用法用量：每服1丸，以暖酒或薄荷姜汤研下。
制备方法：上为末，炼蜜为丸，如小弹子大。
临床应用：None
各家论述：None
用药禁忌：忌生冷、毒滑物、鸡、猪肉。
附注：None
