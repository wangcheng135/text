序号：47887
名称：神功散
组成：川芎6两，当归6两，升麻6两，甘草6两。
出处：《古今医鉴》卷十四引何知府方。
主治：小儿痘疹。
加减：None
功效：None
用法用量：每服以朱砂末1钱，米末1钱，炼蜜2匙，好酒2匙，白沸汤1小钟，共一处调匀，用茶匙喂尽取效。痘疹初起，服之可令不出；若见标者，服之毒气即散；陷者，服之即起。
制备方法：上为细末，取东流水煎3次，每次用水3碗，文武火煎至1碗半，滤下又煎2次，共收药水4碗半听用；又用好朱砂4两，以绢袋悬入瓷罐，加前药水封固，水煮尽为度；取出焙干为末，以纸罗过所用；再将糯米2-3合以纸包紧，外用黄泥固济，入火炼红冷定，打碎，取米黄色者研末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
