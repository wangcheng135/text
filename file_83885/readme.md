序号：83903
名称：乌犀散
组成：乌犀角屑1两，赤箭3分，附子3分（炮裂，去皮脐），羌活3分，防风3分（去芦头），芎3分，桂心3分，羚羊角屑3分，独活3分，牛膝3分（去苗），五加皮3分，黄耆半两（锉），赤茯苓半两，麻黄半两（去根节），赤芍药半两，细辛半两，当归3分（锉，微炒），枳壳半两（麸炒微黄，去瓤），生干地黄1两，道人头1两，甘草1分（炙微赤，锉），酸枣仁3分（微炒）。
出处：《圣惠》卷六十九。
主治：妇人中风，筋脉挛急，四肢疼痛，不能行立，神思昏闷，言语謇涩。
加减：None
功效：None
用法用量：每服4钱，水半中盏，酒半中盏，加生姜半分，薄荷3-7叶，煎至6分，去滓温服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《医方类聚》引作“乌犀角散”。
