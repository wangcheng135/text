序号：6833
名称：四二五合方
组成：当归3钱，白芍3钱，川芎1钱，熟地4钱，覆盆子3钱，菟丝子3钱，五味子3钱，车前子3钱，牛膝4钱，枸杞子5钱，仙茅3钱，仙灵脾4钱。
出处：《刘奉五妇科经验》。
主治：血虚肾亏所引起的经闭，或席汉氏综合症。
加减：None
功效：养血益阴，补肾生精。
用法用量：None
制备方法：None
临床应用：None
各家论述：本方用五子衍宗丸补肾气，其中菟丝子苦平补肾，益精髓；覆盆子甘酸辛温，固肾涩精；枸杞子甘酸化阴，能补肾阴；五味子五味俱备，入五脏大补五脏之气，因其入肾，故补肾之力更强；车前子性寒有下降利窍之功，且能泻肾浊补肾阴而生津液。配合仙茅、仙灵脾以补肾壮阳。五子与二仙合用的目的就是既补肾阳又补肾阴。补肾阳能鼓动肾气，补肾阴能增加精液。肾气充实，肾精丰满，则可使毛发生长，阴道分泌物增多，性欲增加，月经复来。另外，与四物汤合方以加强养血益阴之效，再加牛膝能补肾通经。本方的功用不在于通而在于补。肾气充，肾精足，经水有源，则月经自复。
