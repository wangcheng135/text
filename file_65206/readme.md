序号：65222
名称：银液汤
组成：山泽银1斤（水煮取液），龙齿2两，生地黄（切，焙）半两，防已（锉）2两，羚羊角（镑）2两，远志（去心）2两，人参（去芦头）1两半，独活（去芦头）1两半，甘草（炙，锉）1两半，桂（去粗皮）1两半，细辛（去苗叶）1两，白茯苓（去黑皮）2两半，杏仁（汤浸，去皮尖双仁，炒）10枚。
出处：《圣济总录》卷十四。
主治：小儿风惊，恐怖不安，或发痫吐沫。
加减：热多，去桂，加钩藤2两。
功效：None
用法用量：每服5钱匕，以银液2盏，煮取8分，去滓，入竹沥少许，搅匀，温服，空心、午时、夜卧各1次。
制备方法：上除银液外，余为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
