序号：38380
名称：河车补阴丸
组成：紫河车1具（用热米泔水洗净，然后用麝香汤洗，用针挑去筋内红血水，漂数次者佳），川黄柏2两（盐酒拌，晒干，炒褐色），知母（盐酒炒）1两，人参，龟版1两（酥炙，去裙），熟地黄4两（酒浸），枸杞子2两，牡丹皮1两，茯苓1两，泽泻5钱，五味子5钱，青盐5钱。
出处：《医统》卷四十六。
主治：酒色过度，血气俱虚，肾脏羸惫，虚火上炎，咯血，咳痰多嗽，盗汗，劳热，渐成骨蒸。
加减：None
功效：None
用法用量：每服50-60丸，空心白汤送下。
制备方法：上为细末，山药糊丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
