序号：49206
名称：胜金散
组成：老生姜10两（切作如钱薄，摊箕内，令水缩；次用砒霜1钱重，细研，拌姜钱令匀，留少许砒放熔银甘砂锅子内，砒拌生姜于上，按令紧，用新瓦1片盖锅口，烈炭火煅令烟尽，倾于碗内，盏子合定存性，要成黑炭，不可白却），荜拨2钱，全蝎2钱（去肚内肠屎并毒），华阴细辛2钱（味辛如椒者），胡椒2钱。
出处：《传信适用方》卷二。
主治：肾热上蒸，牙龈浮肿，牙齿疼痛，或肾元虚冷，牙齿摇动，或赤肿痛，风牙虫牙。
加减：None
功效：None
用法用量：每用手指点少许如黑豆大，揩牙龈上。闭口，候有涎沫即吐去，立效。
制备方法：上4味焙干，姜炭亦为细末，都拌匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
