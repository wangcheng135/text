名称：神效药茶
组成：建曲4两，楂炭4两，滑石4两，麦芽4两，杏仁2两半，陈皮2两半，葛根2两，青皮2两，乌药2两，薄荷2两，防风2两，香薷2两，藿香2两，淡芩2两，木通2两，苏梗2两，羌活2两，白扁豆2两，泽泻2两，苍术2两，白芷2两，苍耳2两，大黄2两，钩藤2两，半夏（制）2两，枳壳2两，蔓荆子2两，槟榔2两，荆芥2两，甘草2两，独活2两，川芎1两半，升麻1两半，柴胡1两半，木瓜1两半，大青叶1两半，草果仁1两，麻黄（去节）1两，细辛1两，厚朴1两，苏木1两，鲜姜6两。
出处：《卫生鸿宝》卷一。
主治：外感风寒暑湿，身热头痛，时疫疟痢，霍乱吐泻，伤食饱胀，心口急痛。
加减：None
功效：None
用法用量：每服3钱，阴阳水煎服。小儿及虚弱减半。
制备方法：上药炭火煎浓，用雨前茶叶15斤，将药汁拌收，烈日中晒5-6日，干透，贮瓷瓶内。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
