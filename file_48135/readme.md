序号：48150
名称：神应酒
组成：茵芋（炙）2两，附子2两，天雄（并生用，去皮脐）2两，丹参2两，蜀椒（去目并闭口，炒出汗）2两，踯躅花2两，甘草（炙）2两，石菖蒲2两，桂（去粗皮）2两，干姜（生用）2两，乌头（生用，去皮脐）2两，独活（去芦头）2两，地骨皮2两，秦艽（去苗土）2两，防风（去叉）2两，芎2两，人参2两，当归2两，白芷2两，藁本（去苗土）2两，生干地黄2两，白鲜皮2两，栾荆（炙）2两。
出处：《圣济总录》卷十八。
主治：大风疾，及诸风疾。
加减：None
功效：None
用法用量：每日空腹饮半盏，渐渐加饮。每饮了以饭2-3匙压之。每饮1盏，即添1盏浸药，药味尽即止。
制备方法：上锉细，以无灰酒2斗，密器中浸，经7日成。
临床应用：None
各家论述：None
用药禁忌：忌热肉面、鸡、鱼、牛肉、油腻、果子、陈臭豉汁等物。
附注：None
