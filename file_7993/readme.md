序号：7993
名称：加减香棱丸
组成：木香2钱，丁香2钱，三棱2钱，枳壳2钱，青皮2钱，川楝肉2钱，茴香1钱，台乌3钱，香附3钱，莪术3钱。
出处：《中医妇科治疗学》。
主治：肝肾气郁，少腹两侧疼痛，拒按，有块不坚，推之可移，胸胁胀痛，痞满不思食，有时少腹中部亦痛，但不拒按，月经后期，舌淡苔白，脉弦滑。
加减：月经后期量少，加当归、川芎各2钱；少腹两侧痛甚，按之有块，去丁香，加荔枝核、橘核各2钱；包块疼痛拒按，去丁香、木香、茴香、川楝，加桃仁、丹皮各2钱，姜黄3钱，乳香、没药、檀香各2钱；腰痠腹痛，加杜仲5钱，续断3钱。
功效：理气行滞，和血散瘕。
用法用量：水煎，空腹时温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，据剂型，当作加减香棱汤。
