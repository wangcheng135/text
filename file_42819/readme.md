序号：42834
名称：保命龙虎丸
组成：白胶香（别研）、虎骨（酥涂、炙黄）、黑牵牛、乳香（别研）、地龙（去土炒）、古老钱（火烧、醋淬7遍）、木鳖子（去壳、别研）、当归（洗、焙）、川乌头（炮，去皮脐尖）、没药（别研）、附子（炮、去皮脐）、草乌头（锉、盐炒令黄、去盐不用）、牛膝（酒浸1宿、焙干）、肉苁蓉（酒浸1宿，焙干）、巴戟（去心），自然铜（火煅、醋淬7遍）各等分。
出处：《杨氏家藏方》卷一。
主治：瘫缓，走注疼痛，风劳气冷，腰膝软弱，四肢軃曳；及从高坠下损折，口眼歪斜，久卧床枕，起止不得。
加减：None
功效：None
用法用量：每服30丸，空心温酒或盐汤送下；妇人醋汤送下。
制备方法：上为细末，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
