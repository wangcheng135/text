序号：71410
名称：雄黄丸
组成：雄黄2两，磁石2两，朱砂2两，硫黄2两，牛黄1两（细研），麝香半两（细研）。
出处：《圣惠》卷九十八。
主治：None
加减：None
功效：补益精血，延年驻颜，益颜色，壮志气；久服可以无疾，身轻骨健，耳目聪明。
用法用量：欲修合时，须5月5日，收采青艾嫩者，纳1担以来择取，用水净洗，木臼中烂捣，于净布中绞取汁，可5升以来，先泥一炉，致银锅，以慢火煎令成膏，斟酌稀稠得所，即先下磁石，搅令匀；次下朱砂，又搅令匀；次下雄黄，又搅令匀；良久，去锅下火，即下硫黄，又搅令匀；次下牛黄，又搅令匀；次麝香，须细意熟搅，要药味匀。候可丸即丸，如绿豆大。每服5丸，空心温酒送下。
制备方法：上药前4味各于乳钵内细研，水飞过，于净瓷器中贮之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
