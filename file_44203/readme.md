序号：44218
名称：三痹汤
组成：人参1钱，黄耆（酒炒）1钱，白术1钱，当归1钱，川芎1钱，白芍1钱，茯苓1钱，甘草（炙）5分，桂心5分，防己5分，防风5分，乌头（炮）5分，细辛3分，生姜3片，红枣2个。
出处：《张氏医通》卷十四。
主治：治风寒湿气合病，气血凝滞，手足拘挛，治腰痛兼湿，痛必酸而麻木者。
加减：None
功效：None
用法用量：水煎，不拘时候热服。
制备方法：None
临床应用：None
各家论述：此方合保元、四君、内补建中、防己黄耆、防己茯苓汤、《千金》防己汤等方，但加防风以搜气分之风，川芎以搜血分之风，细辛以搜骨髓之风。于原方中削去生地、牛膝、杜仲、续断、秦艽、独活，增入防己、白术、乌头以祛除风湿，则参附、耆附、术附、桂附、真武等法俱在其中。彼用附子之雄以播真阳，此藉乌头之烈以祛痹着。盖杂合之气，须杂合之方，方为合剂。第恐地黄、牛膝辈阴柔之药难振迅扫之威，是不得不稍为裁酌。
用药禁忌：None
附注：本方方名。《观聚方要补》引作“改定三痹汤”。
