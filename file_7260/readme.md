序号：7260
名称：四妙丸
组成：肉豆蔻1两（用盐酒浸，破故纸同炒干燥，不用故纸），山药1两（酒浸，北五味子同炒干燥，不用五味子），厚朴2两（去粗皮，青盐1两同炒，青盐不见烟为度，不用盐），大半夏1两（每个切作2块，木猪苓亦作片，水浸，炒燥，不用猪苓）。
出处：《瑞竹堂方》卷二。
主治：脾胃虚弱，脾土不能化，痰成窠斗，停于胸臆，饮食既少复迟。
加减：None
功效：实脾土，下痰顺气。
用法用量：每服50-70丸，空心盐酒或米饮或盐汤送下。
制备方法：上为细末，酒糊为丸，如梧桐子大，辰砂1分，沉香1分，作二次上为衣，阴干。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“四炒丸”。
