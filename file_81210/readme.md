序号：81226
名称：麝香平气丸
组成：麝香（别研）半两，木香半两，沉香半两，丁香半两，肉豆蔻（去壳）半两，丹砂（别研）半两，槟榔（焙锉）1两，桂（去粗皮）1两，厚朴（去粗皮，涂姜汁炙）1两，乳香（生姜汁内煮软，候冷，别研如膏）1两，半夏（汤洗7遍，切，焙干，捣为末，姜汁和作饼子，焙干，别捣为末）1两。
出处：《圣济总录》卷六十三。
主治：心胸痞闷，痰逆恶心，吞酸噫食，腹胁疼痛。肢体倦怠。
加减：None
功效：和益脾胃，思进饮食，辟除邪气。
用法用量：每服15-20丸，食后温米饮送下。
制备方法：上药除别捣研外，共为末；次入丹砂、麝香、再研匀；将乳香，半夏末，入生姜汁，煮作薄糊，和前药硬软得所，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
