序号：22839
名称：西岳真人灵飞散
组成：云母粉1斤，茯苓8两，钟乳粉7两，柏子仁7两，人参7两，续断7两，桂心7两，菊花15两，干地黄12两。
出处：《千金》卷二十七。
主治：None
加减：None
功效：延年强身，悦颜固齿。
用法用量：先食饮服方寸匕，每日1次。
制备方法：上为末，生天门冬19斤取汁搜药，纳铜器中蒸1石2斗黍米下，米熟晒干为末。
临床应用：None
各家论述：《千金方衍义》：云母为辟除三虫、伏尸，荡练脏腑积阴之的药。更添《本经》云母性升，主中风寒热如在车船上，除邪气，安五脏，益精明目；钟乳补真阳，安五脏，通百节，利九窍；人参补五脏，安魂魄，除邪气，明目开心；益智久服轻身延年；茯苓守五脏正气，久服安魂养神，不饥延年；桂心利关节，益气补中，久服通神，轻身不老；柏子仁除风湿，安五脏，久服令人润泽美色，耳目聪明，不饥不老，轻身延年；得菊花补水制火，益金平木，久服利血气，耐寒延年；续断续筋骨，补不足，久服益气力；生地黄治伤中，逐血痹，填骨髓，长肌肉，久服轻身益气，延年不饥，且与茯苓、生地黄_x000D_
等皆能伏云母之性，使之驯良无悍，如无生者，不妨约取干者去心，杵烂熬膏和药，总在米下同蒸，生熟无异。服后白发尽落，故齿皆去，次以蜜丸服之而得更生，阴阳相济，非可言语形容。
用药禁忌：忌食胡蒜、羊血。
附注：灵飞散（《千金翼》卷十三）。《千金翼》有白术四两。
