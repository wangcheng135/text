序号：38621
名称：法炼灵乌散
组成：乌鸦1只（腊月者良，留毛去肠肚），朱砂半两，铁粉半两，蛇黄（烧红，醋淬3次）半两，黑铅半两（熔成汁，入水银半两在内，候化，急倾出，待冷用），黄丹2钱半，上除乌鸦外，并研令细，入在乌鸦腹内，用线缝合，入瓦罐内，以盐泥固济，晒干，用炭火3斤，煅烟出为度，次入后药：天南星（生姜汁浸3宿，焙干）1两，防风（去芦头）1两，羌活（去芦头）1两，川芎1两，荆芥穗半两，全蝎（去毒，微炒）半两，白僵蚕（炒去丝嘴）半两。
出处：《杨氏家藏方》卷十七。
主治：小儿胎风，诸痫，目睛斜视，涎潮壅噎，吐咽不下，口睛牵引，身体强直。
加减：None
功效：None
用法用量：每服半钱，麝香汤调下，不拘时候。
制备方法：上为细末，与前药同研匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
