序号：32763
名称：陈橘皮煎丸
组成：陈橘皮（汤浸，去白，焙）15两（别捣罗为末），巴戟天（去心）3两，石斛（去根）3两，牛膝（酒浸，切，焙）3两，肉苁蓉（酒浸，切，焙）3两，鹿茸（去毛，酒炙）3两，菟丝子（酒浸3日，别捣，焙）3两，杜仲（去粗皮，炙，锉）3两，阳起石（酒浸，研如粉）3两，厚朴（去粗皮，生姜汁炙）3两，附子（炮裂，去皮脐）3两，吴茱萸（汤洗，焙干，炒）3两，当归（切，焙）3两，干姜（炮）3两，京三棱（煨，锉）3两，萆薢3两，甘草（炙，锉）1两。
出处：《圣济总录》卷七十二。
主治：久积冷气，攻心腹疼痛，痰癖呕逆，腹胀不思饮食，肌肤瘦瘁，腰膝倦痛，下痢泄泻，疟疾肠风，并妇人血海久冷无子。
加减：None
功效：None
用法用量：每服20-30丸，空心温酒送下，盐汤亦得。
制备方法：上为末。先以好酒五碗，于银石器内煎橘皮末，令如饧，入诸药搅匀，稍干，更入酒少许为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
