序号：26662
名称：延寿丹
组成：白术1两（土炒），青皮1两，生地1两，厚朴1两（姜汁炒），杜仲1两（姜汁炒），故纸1两（微炒），广皮1两（去净白），川椒1两，青盐1两，黑豆2升，巴戟肉1两（去心），白茯苓1两，小茴香1两，肉苁蓉（竹刀剖净鳞，黄酒洗，晒干）1两。
出处：《验方新编》卷十一。
主治：痔漏疮毒，赤白带下。
加减：None
功效：添精补髓，健脾养胃，乌须延寿，轻身健体，返老还童，中阳复兴，少阳复起，调妇人经水；暖下安胎。
用法用量：每早空心服3钱，开水送下，不可间断。
制备方法：制好，人铜锅，或砂锅亦可，用水20小碗，桑柴文武火，煎至10小碗，将水盛出；复煎药滓，用水10小碗，煎至5小碗，去滓不用，惟用2次药水15碗，将黑豆放锅内，用火缓缓煎至水干，盛起候冷，入瓷罐装贮。
临床应用：None
各家论述：None
用药禁忌：妇人受胎之后，不可再服，恐受双胎。忌食牛、马肉。
附注：None
