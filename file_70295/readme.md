序号：70311
名称：煮酒药方
组成：当归5钱，人参5钱，茯苓5钱，草乌5钱，乌药5钱，杏仁5钱，川乌（去皮尖）5钱，缩砂5钱，何首乌5钱，五加皮5钱，枸杞子5钱，川椒5钱，肉苁蓉5钱，木香5钱，牛膝1两，枳壳1两，干姜（酥油炙）1两，香附子1两，白芷1两，厚朴1两，陈皮1两，麻黄1两，白术1两，川芎1两，独活1两，羌活1两，半夏1两，官桂1两，芍药1两，生地黄1两，天门冬1两，麦门冬1两，防风1两，五味子1两，小茴香1两，细辛1两，苍术1两，破故纸1两，甘草1两，沉香5钱，胡桃肉8两，酥油8两，小红枣8两，北蜜8两。
出处：《摄生众妙方》卷三。
主治：男妇久患诸风寒湿，左瘫右痪，一切风气，无问老少。
加减：None
功效：None
用法用量：空心服3盏，每日3次。渣晒干碾末，用本酒打糊为丸，如梧桐子大。每服30丸，用酒送下。
制备方法：上以生绢袋盛，用无灰好酒1大坛，浸药3日，放锅内煮3个时辰，取出，埋在土内3日出火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
