序号：81620
名称：麝香鹿茸丸
组成：鹿茸（火燎去毛，酒浸，炙）70两，熟干地黄（净洗，酒浸，蒸，焙）10斤，附子（炮，去皮脐）140个，牛膝（去苗、酒浸1宿，焙）1斤4两，杜仲（去粗皮，炒去丝）3斤半，五味子2斤，山药4斤，肉苁蓉（酒浸1宿）3斤。
出处：《局方》卷五。
主治：治下焦伤竭，脐腹绞痛，两胁胀满，饮食减少，肢节烦痛，手足麻痹，腰腿沉重，行步艰难，目视茫茫，夜梦鬼交，遗泄失精，神情不爽，阳事不举，小便滑数，气虚肠鸣，大便自利，虚烦盗汗，津液内燥。
加减：None
功效：益真气，补虚惫。
用法用量：每服20丸，食前用温酒送下；盐汤亦得。
制备方法：上为末，炼蜜为丸，加梧桐子大，每1斤丸子，用麝香末1钱为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
