序号：47773
名称：神仙保真丸
组成：川楝子（去核，炒）1两，蓬莪术（煨，切）半两，肉豆蔻（去核，面裹煨香）、木香半两，槟榔3枚，当归7钱半（焙），神曲半两（炒香），茴香（拣选一半炒香，一半生用）半两，附子1两（炮裂，去皮脐，令取细末），硇砂（光明不夹石者，用无灰酒1大建盏，将硇砂入酒内，于银石器内慢火熬化开，滤去滓，放冷，将附子细末再打糊）3分。
出处：《杨氏家藏方》卷十。
主治：真元不足，脏气虚弱，触冒寒气，闭塞下元，心腹绞痛，自汗厥逆，及奔豚发痛，上下有声，腹急胀。
加减：None
功效：None
用法用量：每服50丸，空心、食前细嚼炒桃仁7枚，同药用温酒1盏送下，盐汤亦可；如大痛时，并进2服。
制备方法：上除附子、硇砂外为细末，将熬下附子、硇砂糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
