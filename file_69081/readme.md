序号：69097
名称：黄连膏
组成：净黄连半斤，苦参4两，秦皮2两，杏仁49粒（冬月制，取雪水4碗，煎2碗，放净瓷器内；又以水煎，取1碗，放前汁内；又以水1碗，煎取半碗，用净汁，与前汁和一处，取净铜铫子入汁在内，慢火熬，以桑条不住手搅，勿令沉底，勿动灰尘入汁中，务宜仔细。待熬至1碗，再入马牙消半两，同煎至半碗，取起，以纸盖定），制过炉甘石末2两，硼砂（末）半两，乳香1钱，没药（末）1钱，胆矾（末）3钱，海螵蛸（末）2钱。
出处：《育婴秘诀》卷四。
主治：风热眼疾。
加减：None
功效：None
用法用量：以乳汁磨，点眼。
制备方法：和匀，入膏中取起，摊冷待干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
