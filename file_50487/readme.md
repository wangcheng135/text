序号：50502
名称：茵陈散
组成：栀子仁1两，石膏3两，川大黄1两（锉碎，微炒），栝楼（干者）1枚，甘草1两（炙微赤，锉），木通1两（锉），茵陈1两。
出处：方出《圣惠》卷五十五，名见《得效》卷三。
主治：酒食过度，风湿所搏，热气郁蒸而成黄疸，身目爪甲牙齿及小便皆黄，或身体多赤多青，心下热闷，小便闭涩。黄疸，身目皆黄，皮肤如曲尘色。黄疸，食已即饥，身体面目，爪甲牙齿，及小便悉黄寒热，或身体多赤、多青，皆由酒食过度，为风湿所搏，热气郁蒸而成。
加减：None
功效：None
用法用量：每服5钱，以水1大盏，加葱白7寸，煎至5分，去滓温服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：石膏茵陈散（《东医宝鉴·杂病篇》卷六）。
