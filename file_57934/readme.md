序号：57949
名称：益气内消散
组成：当归8钱，川芎8钱，白芍（酒炒）8钱，白术（去芦）8钱，青皮（去瓤）8钱，陈皮8钱，半夏（姜炒）8钱，桔梗8钱，羌活8钱，白芷8钱，独活8钱，厚朴（姜汁炒）8钱，防风1两，黄芩1两，乌药1两，香附1两，槟榔1两，苏叶1两半，沉香2钱，木香5钱，人参5钱，粉草5钱。
出处：《回春》卷八。
主治：瘰疬并诸瘤结核。
加减：None
功效：None
用法用量：水煎，温服。服10余剂即消；若再服，照分量制酒糊为丸，如梧桐子大。每服70丸，酒送下。
制备方法：上锉。
临床应用：None
各家论述：None
用药禁忌：None
附注：抑气内消散（《寿世保元》卷九）。
