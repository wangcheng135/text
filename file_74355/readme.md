序号：74371
名称：塌肿神应丸
组成：三棱1两，莪术1两，青皮1两，陈皮1两，干漆（烧）1两，芫花7钱半，大戟3钱，硇砂1钱，巴豆霜1钱。
出处：《玉机微义》卷二十引《经验方》。
主治：诸般虫毒，肚胀如鼓，脾癖癥瘕气块，饮积，气积，水积，血积，上喘气急，咳嗽倚息不得睡，服药不效者。
加减：None
功效：导气散郁。
用法用量：每服40-50丸，生姜汤送下，五更服。随用椒目6两，萝卜子半斤，炒香熟，用手帕盛，于患处熨，如冷再炒，熨至再三，大小便行后才住熨，以温稀粥补之。
制备方法：上前药和一处，用好米醋1大碗，慢火上煮，醋干为度，取出晒干，碾为末，醋和作丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
