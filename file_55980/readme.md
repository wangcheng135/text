序号：55995
名称：桑根煎丸
组成：桑根白皮（锉）2两，麻子仁（淘净，研）5合，清酒5盏（煮前2味药至3盏，绞汁去滓），枣30枚（大者，取肉），饴糖2两，阿胶（炙令燥）1两，蜜5大合（取枣糖胶，同煎1大盏，如膏），干姜（炮裂）半两，厚朴（去粗皮，生姜汁炙7遍，锉碎）半两，蜀椒（去目并合口者，炒出汗）1分，桂（去粗皮）1分，甘草（炙令赤）1分，黄柏（锉碎）1分，生干地黄（焙）1分，玄参1分，五味子1分，芍药半两。
出处：《圣济总录》卷一五八。
主治：妊娠堕胎后，血出不止，胸胁心腹满痛，时复寒热甚者。
加减：None
功效：None
用法用量：每服1丸，食前温酒化服。
制备方法：上药前7味共煎成稀膏，后10味为末，与前膏同和为剂涂酥为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
