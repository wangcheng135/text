序号：38644
名称：泻心导赤饮
组成：山栀子、黄芩、麦门冬、滑石、人参、犀角、知母、茯神、黄连（姜汁炒）、甘草。
出处：《古今医鉴》卷三。
主治：越经证。伤寒心下不疼，腹中不满，大便如常，身无寒热，渐变神昏不语，或睡中独语，目赤唇焦，将水与之则咽，不与则不思，形如醉人。
加减：None
功效：None
用法用量：加生姜1片，大枣2枚，灯心20茎煎。临服入生地黄汁3匙。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：泻心导赤汤（《寿世保元》卷二）。《寿世保元》本方用黄连八分，黄芩一钱五分，甘草八分，犀角五分，麦门冬三钱，滑石三钱，山栀三钱，茯神三钱，知母二钱，人参二钱。
