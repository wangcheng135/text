序号：40059
名称：肾沥汤
组成：远志（去心）1两，人参1两，五味子1两，石斛（去根）1两，泽泻1两，当归（切，焙）1两，桂（去粗皮）1两，甘草（炙，锉）1两，白茯苓（去黑皮）1两，桑上寄生1两，麦门冬（去心，焙）3两，熟干地黄（焙）1两半，栝楼根1两半，地骨皮1两半。
出处：《圣济总录》卷九十二。
主治：虚劳不足，小便利数，呼吸短气，烦渴引饮，膀胱满急。
加减：None
功效：None
用法用量：每服3钱匕，水1盏半，入羊肾1分（细切），先煎1-2沸，次入生姜半分（拍碎），大枣2枚（劈破），煎至7分，去滓，空腹温服，食后再服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
