序号：36546
名称：三台丸
组成：大黄1-2两（1方2两，椎碎、熬令变色），葶苈1升（熬令变色），附子1两（煨令坼），杏仁1升（熬令变色），消石1升，柴胡（1方前胡）2两，半夏1两（洗），厚朴1两，茯苓半两，细辛1两。
出处：《医心方》卷十引《范汪方》。
主治：五脏寒热积聚，胪胀腹大空鸣而噫，食不生肌肤，剧者咳逆。
加减：None
功效：None
用法用量：从5丸起，不知稍增。取大便调利为度。
制备方法：上为末，以蜜捣三万杵为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：三台丸取大黄附子汤，专破肾肝脾三经之结；兼大陷胸丸，但以消石易芒消，以泄肺胃逆气之满；更加前胡、厚朴、茯苓、半夏导气涤痰；方中仅有细辛一味达肝，于此可以默悟坚症积聚诸方，不独为肝脏一门设也。
用药禁忌：None
附注：方中厚朴、茯苓、细辛用量原缺，据《千金》补。
