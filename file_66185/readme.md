序号：66201
名称：野葛膏
组成：野葛2两（锉），蛇衔2两，犀角屑1两，川乌头1两（去皮脐），桔梗2两（去芦），茵芋2两，防风2两（去芦头），川椒2两（去目），干姜2两，巴豆30枚（去壳），川升麻1两，细辛2两，当归2两，附子2两（去皮脐），羌活2两，川大黄2两，雄黄2两（研如粉）。
出处：《圣惠》卷三。
主治：肝脏风毒，流注脚膝，筋脉挛急，疼痛。
加减：None
功效：None
用法用量：旋取摩病处，令极热，密室避风，日3次。
制备方法：上锉细，以酒5升，渍药一宿，以不中水猪膏5斤，以前药同纳于铛中，炭火上煎之，令药色变黄，又勿令焦黑，膏成，绞去滓，下雄黄，候冷，入瓷器中盛之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
