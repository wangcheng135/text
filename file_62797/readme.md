序号：62813
名称：清火豁痰丸
组成：大黄（酒蒸）3两，礞石（煅）5钱，沉香2钱，黄芩（酒炒）2两，黄连（酒炒）2两，栀子（炒）2两，连翘1两，天南星（制）2两，半夏（制）2两，白术（炒）2两，枳实（炒）2两，贝母（去心）1两5钱，天花粉1两，陈皮1两，白茯苓1两，神曲（炒）1两，青黛5钱，玄明粉7钱，甘草5钱，白芥子（炒）2两。
出处：《古今医鉴》卷四。
主治：上焦郁火，痰涎壅盛，胸膈不利，咽喉噎塞，吐不出，咽不下，如鲠状。
加减：None
功效：None
用法用量：每服40丸，生姜汤送下。
制备方法：上为末，生姜汁、竹沥为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
