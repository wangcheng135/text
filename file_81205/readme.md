序号：81221
名称：麝香双丸
组成：麝香2两，牛黄2两，黄连2两，桂心1两，雄黄1两，乌贼鱼骨（炙）1两，丹砂1两，附子1两，巴豆60粒（去皮炒），特生礜石（烧半日）1两，蜈蚣1个（净）。
出处：《幼幼新书》卷十一引《婴孺方》。
主治：客忤中恶发痫，乳哺不消，中风反折踧，口吐舌，并疰忤面青，目下垂，腹满，丁奚羸瘦，胫交，三岁不行。
加减：None
功效：None
用法用量：1月儿服米许，三百日服2麻子许，量加，日夜4-5服，汗出及痊为限。
制备方法：别研巴豆如脂，同蜜和末三千枚，合收勿泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：麝香丸（《圣惠》卷八十二）、麝香紫霜丸（《圣济总录》卷一七五）。方中桂心、雄黄、乌贼鱼骨、丹砂、附子用量原脱。《圣惠》本方麝香、牛黄、特生礜石、附子、雄黄，丹砂、桂心、乌贼鱼骨用各半两，黄连一两，巴豆三十枚，蜈蚣一枚。
