序号：64766
名称：硇砂膏
组成：血余（盐水洗）4两，山栀子8两，穿山甲（炙）6两，棉子油10斤，东丹（炒）100两，槐枝6两，杏枝6两，桑枝6两，柳枝6两，沉香2两，方儿茶2两，血竭3两，琥珀1两，象皮（微炒）1两，冰片5钱，麝香5钱，硇砂4两。
出处：《全国中药成药处方集》（上海方）。
主治：痈疽，瘰疬、乳疖。
加减：None
功效：None
用法用量：贴患处。
制备方法：先将血余、山栀子、穿山甲、槐枝、杏枝、桑枝、柳枝浸入棉子油内一夜，随后文火熬至药枯，去滓滤清，加入东丹，再熬至滴水成珠收膏，摊时再加沉香、方儿茶、血竭、琥珀、象皮、冰片、麝香、硇砂药粉和匀，摊布上，大号每张用药肉4钱，中号每张用药肉2钱半，小号每张用药肉1钱半。
临床应用：None
各家论述：None
用药禁忌：疔疮不可贴，不可入口。
附注：None
