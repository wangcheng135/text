序号：6178
名称：加味枳术丸
组成：归尾3两，白术1两半（炒），枳实1两半（炒），蒲黄3两，灵脂3两，没药3两，肉桂1两半（去皮），泽泻1两半，陈皮1两半。
出处：《医略六书》卷三十。
主治：产后血瘀腹胀，脉涩滞者。
加减：None
功效：None
用法用量：每服3钱，米饮送下。
制备方法：上为末，粥为丸。
临床应用：None
各家论述：产后脾气有伤，不能输化于中，而瘀血滞于肝脾，遂成腹胀焉，白术健脾气以运化，枳实破滞气以宽中，归尾破血活血，蒲黄破瘀散血，五灵脂破瘀降浊阴，明没药散瘀行血滞，紫肉桂温经通闭，建泽泻泻湿利水，陈皮利气以和中也。粥丸饮下，使脾气健运，则瘀血自化，而经府清和，安有腹胀之患乎？
用药禁忌：None
附注：None
