序号：30524
名称：还睛丹
组成：苁蓉（酒浸1伏时，切，焙干）2两，威灵仙（拣去土）2两，青葙子（拣净去土）2两，巴戟（去心）2两，蝉壳（去土）2两，甘菊花（拣净）2两，密蒙花2两，旋复花2两，防风（去芦头并叉）2两，枸杞子2两，天麻（酒浸1宿，焙干）2两，地骨皮2两，蛇蜕皮1两半（酒浸1宿，炒黄），香白芷1两半，桑花1两，麻子（水淘去浮者，炒香）1两。
出处：《御药院方》卷十。
主治：None
加减：None
功效：None
用法用量：每服50丸，空心、食前温酒、白汤、粥饮任下。
制备方法：上为细末，炼蜜为丸，如豌豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：《普济方》引本方有营实一两。
