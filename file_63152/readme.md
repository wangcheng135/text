序号：63168
名称：大温惊丸
组成：人参5钱，茯苓5钱，白术5钱，辰砂5钱，麦门冬5钱，木香5钱，代赭石5钱，甘草1两，酸枣仁1两，僵蚕钱半，桔梗尾钱半，全蝎5个，金箔6片，银箔6片。
出处：《医学入门》卷八。
主治：急惊潮热，慢惊夜啼，搐搦，心热烦躁。
加减：急惊潮热，加薄荷、竹茹；慢惊，加冬瓜仁，夜啼，加灶心土；搐搦，加防风；伤风，加荆芥；痘疹，加蝉退；常服用金银_x000D_
花、薄荷，俱煎汤下；治心热烦躁夜啼，常用安神定志去惊药；如惊风已退，神志未定者，加琥珀、远志。
功效：None
用法用量：量小儿大小服之。
制备方法：上为末，炼蜜为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
