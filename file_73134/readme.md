序号：73150
名称：山栀子丸
组成：山栀子（去皮）半两，山茱萸半两，地榆（洗，锉）半两，桔梗（炒）半两，细辛（去苗叶，炒）半两，羌活（去芦头）半两，独活（去芦头）半两，麻黄（去节，煎，掠去沫，焙）半两，甘草（炙，锉）半两，鹿茸（酒浸，炙去毛）半两，虎骨（涂酥，炙令黄色）半两，紫菀（去苗土）半两，白芷（微炒）半两，藁本（去苗土）半两，红蓝花（微炒）半两，防风（去叉）半两，乌蛇（酒浸，去皮骨，炙）半两，桂（去粗皮）半两，胡椒1分，干姜（炮）1分。
出处：《圣济总录》卷六。
主治：卒中风。
加减：None
功效：None
用法用量：不论破伤风、急风、慢风、摊缓风、洗头沐浴中风，口眼斜，及妊娠产后风，并得服之。所患深重日久、以生姜、薄荷、荆芥酒内调下散半钱，送下3丸至5丸；甚者豆淋薄荷酒调散半钱至2钱匕，送下7丸至10丸，日3夜1。
制备方法：上为末，一半为散，余炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
