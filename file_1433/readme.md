序号：1433
名称：分水丹
组成：广藿香2两，泽泻2两4钱，白扁豆1两6钱，川厚朴3两，半夏3两6钱，香薷草2两，砂仁2两4钱，木瓜2两4钱，紫苏2两4钱，白术2两，苍术3两，猪苓2两4钱，西滑石2两4钱，肉桂2两4钱，陈皮2两4钱，白茯苓2两4钱，石斛2两4钱，甘草3两，白芷2两4钱，桔梗1两6钱。
出处：《全国中药成药处方集》(禹县方)。
主治：暑天受寒，水泻不止，肚腹作疼，上吐下泻。
加减：None
功效：None
用法用量：每服1包，1日2次，10岁者服半包，姜汤送下；白开水亦可。
制备方法：上为细末，每包8分。
临床应用：None
各家论述：None
用药禁忌：孕妇忌用。
附注：None
