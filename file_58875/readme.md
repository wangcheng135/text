序号：58890
名称：秘授蜡矾丸
组成：黄蜡、白矾各胡桃大1块（研），银朱1钱，蛇蜕1条（阴阳瓦焙，研）。
出处：《疡医大全》卷九。
主治：None
加减：None
功效：定痛，厚膜，生肌化脓，解毒去秽。
用法用量：如遇患者，先令洗浴，饮热酒数杯，初服21丸，尽量饮醉，被盖取汗，初起即消；已成疼痛不可忍者，服之可止；一半已溃，服之必出；稠厚黄脓，看人虚实与服，头1日服21丸，第2日只服19丸，逐日递减2丸，服至1丸为止。
制备方法：先将蜡熔化，入蜂蜜少许，再下生矾、蛇蜕、银朱，研末搅匀，将铜勺放滚水内，急手丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
