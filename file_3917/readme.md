序号：3917
名称：无比丸
组成：紫河车1具（初生者佳，或无病壮年妇人者亦可。1说男病用女，女病用男者，若不可得，亦不必拘束。米醋浸1宿，焙干用），草龙胆2钱，甘草（炙）2钱，鳖甲（酥炙）5钱，桔梗2钱5分，胡黄连2钱5分，大黄（酒拌湿，蒸）2钱5分，苦参2钱5分，黄柏2钱5分，知母（去毛）2钱5分，秋石（另研，不必用煎炼者，但尿桶上凝结多年者亦可，长流水洗净用）2钱5分，贝母2钱5分（去心），犀角屑1钱5分，蓬莪术1钱5分，消石1钱5分，败鼓皮心（米醋炙黄）2钱5分，辰砂1两（另研）。
出处：《医学正传》卷三。
主治：劳极。
加减：None
功效：None
用法用量：每服20丸，加至30丸，温酒送下，胁热食前，膈热食后服。
制备方法：上为细末，炼蜜为丸，如梧桐子大，辰砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：紫河车丸、调鼎方。
