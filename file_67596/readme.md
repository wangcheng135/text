序号：67612
名称：小五石泽兰丸
组成：钟乳1两半，紫石英1两半，矾石1两半，白石英42铢，赤石脂42铢，当归42铢，甘草42铢，石膏2两，阳起石2两，干姜2两，泽兰2两6铢，苁蓉2两半，龙骨2两半，桂心2两半，白术30铢，芍药30铢，厚朴30铢，人参30铢，蜀椒30铢，山茱萸30铢，柏子仁1两，藁本1两，芜荑18铢。
出处：《千金》卷四。
主治：妇人劳冷虚损，饮食减少，面无光色，腹中冷痛，经候不调，吸吸少气无力。
加减：None
功效：补益温中。
用法用量：每服20丸，加至30丸，酒送下，1日3次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：本方主治全无风证，故去大五石泽兰丸中乌头、细辛、防风、芎、芷等药，但加阳起、矾石、石脂等味，专行固脱扶阳，故所加之味尤为必需。
用药禁忌：None
附注：None
