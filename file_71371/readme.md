名称：雄黄丸
组成：雄黄（研）半两，丹砂（研）半两，麝香（研）半两，木香3分，龟甲（醋炙）3分，鳖甲（去裙襕，醋炙）3分，虎头骨（酥炙）3分，羚羊角（镑屑）3分，犀角（镑屑）3分，白薇3分，玄参3分，当归（切焙）3分，知母（焙）3分，防风（去叉）3分，麻黄（去根节）3分，龙胆（去苗）3分，牡蛎（烧赤）3分，猪苓（去黑皮）3分，柴胡（去苗）3分，茯神（去木）3分，升麻3分，槟榔（锉）3分，地骨皮3分，赤芍药3分，栀子仁3分，黄连（去须）3分，乌梅肉（炒）3分，阿魏3分，桃仁（汤浸，去皮尖双仁，炒研）3分，安息香（研）3分，萎蕤3分，龙齿3分。
出处：《圣济总录》卷三十五。
主治：一切远年劳疟，结成癥瘕者。
加减：None
功效：None
用法用量：每服20丸，空腹白粥饮送下，未发前服。
