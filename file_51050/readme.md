序号：51065
名称：草果饮子
组成：半夏（泡）2钱，赤茯苓2钱，甘草（炙）2钱，草果（炮，去皮）2钱，川芎2钱，陈皮2钱，白芷2钱，青皮（去白）1钱，良姜1钱，紫苏1钱，干葛4钱。
出处：《妇人良方》卷二十一引《经效》。
主治：妇人产后疟疾，寒热相半，或多热者;感冒后四逆，手足不遂，牙关紧急；霍乱四逆，手足搐搦，欲成风者。
加减：None
功效：None
用法用量：上（口父）咀。每服3钱，水1大盏，加生姜3片、大枣2个，同煎至7分，去滓，当发日侵早连进3服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《校注妇人良方》引作“草果饮”。方中干葛，《医学纲目》作干姜。
