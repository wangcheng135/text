序号：72355
名称：紫元丹
组成：当归1两，独活1两，红花1两，羌活1两，秦艽1两，穿山甲（焙）1两，川断1两，僵蚕（生）1两，牛膝1两，延胡索1两，川郁金1两，香附1两，苍术1两，杜仲1两，川乌（姜汁制）1两，草乌（姜汁制）1两，麻黄（去根节，炒）1两，制乳香1两，制没药1两，全蝎1两，骨碎补4两（去毛，炒），蜈蚣10条（炙），蟾酥5钱（酒化拌药，共为细末），番木鳖1斤半（麻黄、绿豆煎水浸透，去皮心，入麻油内煎老黄色取起，拌土炒筛，去油，另为末）。
出处：《外科证治全书》卷五。
主治：一切阴疽，阴发背，失荣、乳癌、恶核、石疽、贴骨、流注、龟背、痰核等症。初起皮色无异，或微痛，或不痛坚硬漫肿。
加减：None
功效：None
用法用量：每服8分，身弱者5-6分，临卧热陈酒送下。出汗避风。如冒风发麻，姜汤、热酒可解。每间1-2日再服。
制备方法：上将制过木鳖末同前药末各半对和，水为丸。
临床应用：None
各家论述：None
用药禁忌：凡红肿痈毒及孕妇忌此。
附注：None
