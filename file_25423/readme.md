序号：25431
名称：八仙糕
组成：白术8两，白茯苓8两，莲肉8两，芡实（饭上蒸熟，晒干，临合微炒）8两，山药8两,陈皮3两，甘草3两，腊月炒米3斗。
出处：《仙拈集》卷三引《传家宝》。
主治：饮食不消，大便泻痢。
加减：None
功效：补脾胃，肥壮身体。
用法用量：食时旋入糖，用滚水冲服。
制备方法：每年腊月极冻之日，炒糯米，用大簸放天井中间，铺开冷透，同药共磨筛细，收瓷罐内。
临床应用：None
各家论述：None
用药禁忌：None
附注：腊月磨此糕，虽收三五年不坏不至，愈陈久功效愈大；若不在腊月炒磨，多生蛀虫网丝，难以久收，坛放高燥处，勿近地气潮湿，勿用盛酒盐者；白糖须食时旋入冲调，若同药拌入，糕俱潮坏。
