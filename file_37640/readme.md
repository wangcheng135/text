序号：37650
名称：三皮散
组成：海桐皮（去皮）1两，五加皮（自采者）1两，桑白皮1两，川独活1两，川牛膝1两，制杜仲1两，黑附子（炮，去皮）斟酌用多少，薏苡仁（生用）2两，生干地黄升半。
出处：《朱氏集验方》卷一。
主治：风毒湿气，流注脚膝，四肢挛拳，筋脉不伸，脚足疼痛，步履艰难。
加减：None
功效：None
用法用量：空心温服1盏，1日3-5次；如服药酒时，却饮它酒，常令酒气相接，勿令大醉，重者不过2剂。服此药后旬日，两腿必发大疹，渐渐破，出紫黑血，候干，赎磨风膏贴疮上，自然好也。
制备方法：上为粗末，用绵裹作1包，以无灰好酒1斗许，春浸7日，秋、冬14日，夏15日，滤起药。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
