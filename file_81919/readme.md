名称：镇心丸
组成：远志（去心）1两1分，铁精1两半，杏仁（去皮尖双仁，炒）1两半，芎1两半，麦门冬（去心，焙）1两半，牡蛎1两半，龙齿（研）2两，白茯苓（去黑皮）2两，防风（去叉）1两1分，当归（切，焙）1两1分，人参1两1分，鬼臼1两1分，白术1两1分，生干地黄1两1分，丹参1两1分，桔梗（去芦头，炒）1两1分，甘草（炙）1两1分，紫菀（去土）1两，卷柏（去土）1两，山茱萸1两，桂（去粗皮）1两，干姜（炮）1两，防己1两，白蔹1两，羚羊角（镑）1两，牛黄（别研）半两，麝香（别研）3分，银箔400片（别研），虎睛1对（酒浸，炙令黄，别研）。
