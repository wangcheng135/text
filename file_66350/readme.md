序号：66366
名称：鹿血丸
组成：桑上寄生2两，川续断（锉，酒润）1两，鹿茸（去毛，酒蒸）1两，麋茸（去毛，酒蒸）1两，鹿角（镑）1两，麋角（镑）1两，附子（炮，去皮）1两，川乌（炮，去皮）1两，钟乳粉1两，阳起石（煅）1两，川巴戟（捶，去心）1两，沉香（不见火）1两，川牛膝（去芦，酒浸）1两，川萆薢1两，菟丝子（淘，酒蒸，擂，焙）2两，五味子2两，宣木瓜2枚（去皮瓤，蒸烂），椒红（去目及闭口者，微炒出汗，取红）半两。
出处：《医方类聚》卷一五○引《续济生》。
主治：诸虚百损，精血俱耗，血少不能养筋，精虚不能实骨，筋骨痿弱，面色黧黑，耳鸣气短，目视昏花，腰脊疼痛．足膝痠弱，步履艰难，小便白浊，或小便频数，及妇人虚弱。
加减：None
功效：None
用法用量：每服100丸，空心食前用盐汤、盐酒任下，妇人用淡醋汤送下。
制备方法：上为细末，刺鹿血，乘热搜和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
