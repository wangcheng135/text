序号：41811
名称：降心丹
组成：熟干地黄（净洗，酒浸，蒸，焙干）3两，天门冬（去心）3两，麦门冬（去心）3两，茯苓（去皮）2两，人参2两，远志（甘草煮，去芦骨）2两，茯神2两，山药2两，肉桂（去粗皮，不见火）半两，朱砂（研飞）半两，当归（去芦，洗，焙）3两。
出处：《局方》卷五（吴直阁增诸家名方）。
主治：心肾不足，体热盗汗，健忘遗精，及服热药过多，上盛下虚，气血不降，小便赤白，稠浊不清。
加减：None
功效：常服镇益心神，补虚养血，益丹田，秘精气。
用法用量：每服30丸，煎人参汤吞下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
