序号：28775
名称：十四味建中汤
组成：当归（去芦，酒浸，焙干）、白芍药（锉）、白术（锉，洗）、甘草（炙）、人参（去芦）、麦门冬（去心）、川芎（洗净）、肉桂（去粗皮）、附子（炮、去皮脐）、肉苁蓉（酒浸1宿）、半夏（汤洗7次）、黄耆（炙）、茯苓（去皮）、熟地黄（洗去土，酒蒸1宿，焙干）各等分。
出处：《局方》卷五（宝庆新增方）。
主治：气血不足，脾肾久虚，虚损羸瘦，面白脱色，短气嗜卧，手足多冷，夜卧汗多，梦寐惊悸，大便频数，小便滑利；肾虚腰痛，不能转侧。荣卫不足，脏腑俱伤，积劳虚损，形体羸瘠，短气嗜卧，寒热头痛，咳嗽喘促，吐呕痰沫，手足多冷，面白脱色，小腹拘急，百节尽疼，夜卧汗多，梦寐惊悸，小便滑利，大便频数，失血虚极，心松面黑。肾虚腰痛，转侧不能，嗜卧疲弱。阴证发斑，寒甚脉微。伤寒中气不足，脉息虚大，一切虚斑。
加减：None
功效：None
用法用量：每服3钱，水1盏半，加生姜3片，枣子1个，煎至1盏，去滓，食前温服。
制备方法：上为粗散。
临床应用：None
各家论述：《医方集解》：此足三阴、阳明气血药也。黄耆益卫壮气，补中首药；四君补阳，所以补气；四物补阴，所以养血，阴阳调和，则血气各安其位矣。半夏和胃健脾，麦冬清心润肺，苁蓉补命门相火之不足，桂、附引失守之火而归元，于十全大补之中而有加味，要以强中而戢外也。
用药禁忌：None
附注：大建中汤（《证治要诀类方》卷一）。
