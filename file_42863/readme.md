序号：42878
名称：保坤丸
组成：当归4两，白芍3两，白术（炒）2两，茯苓4两，橘皮2两，党参（去芦）2两，丹皮2两，川芎2两，肉桂（去粗皮）2两，玄胡（炙）2两，香附（炙）4两，黄耆1两，熟地2两，藁本5钱，白芷5钱，木香1两，砂仁1两，甘草1两，艾炭1两，知母1两，黄柏2两。
出处：《北京市中药成方选集》。
主治：妇女血寒白带，月经不调，经期腹痛，气郁心跳。
加减：None
功效：调经养血，舒郁化滞。
用法用量：每日早、晚各服1付，温开水送下。
制备方法：上为细末，用冷开水泛为小丸，以滑石10两、朱砂5钱为衣，闯亮，每粒重5厘，每付40粒，每袋装4付。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
