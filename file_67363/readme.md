序号：67379
名称：黄土汤
组成：净黄土2两，广藿香2钱，生木香8分，宣木瓜2钱，陈橘皮1钱，紫厚朴8分，白扁豆3钱，活水芦根2两。
出处：《医略十三篇》卷十一。
主治：霍乱吐泻及转筋霍乱。
加减：夏月，加香薷1钱；三秋，加蓼花根1两；虚，加冬白术1钱半（土炒）；实，加鸡心槟榔1钱；寒，加理中丸5钱；热，加四苓散5钱；干霍乱，本方2剂加炒盐1两，童便1小碗多服，以手指按舌根探吐，得吐即泻，吐泻后去炒盐、童便，照常煎服。
功效：None
用法用量：长流水煎。
制备方法：None
临床应用：None
各家论述：用黄土为主，加藿香、木香之芳香以解秽浊，木瓜和胃舒筋以杜转筋，陈皮调畅气机，厚朴、扁豆消暑去湿，芦根致胃清和。犹是地浆之意，而胜于墙阴之不洁远矣。
用药禁忌：忌稠粘粥食。
附注：None
