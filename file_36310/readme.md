序号：36320
名称：和中启关散
组成：麦冬5钱，人参5分，甘草5分，柏子仁3钱，滑石（敲碎）1钱，黄连1钱，白芍5钱，桂枝3分，天花粉1钱5分。
出处：《辨证录》卷五。
主治：五志厥阳之火太盛，不能营于阴，遏抑于心胞之内，心液外亡，自焚于中，以致关格，吐逆不得饮食，又不得大小便，头上有汗者。
加减：None
功效：调其营卫。
用法用量：水煎服。1剂而上吐止，再剂而下团通矣。
制备方法：None
临床应用：None
各家论述：此方解散中焦之火，更能舒肝以平木。木气既平，而火热自灭。内中最妙者，用黄连与桂枝也，一安心以交于肾，一和肾而交于心，心肾两交，则营卫阴阳之气，无不各相和好，阴阳既和，而上下二焦安能坚闭乎！此和解之善于开关也。
用药禁忌：None
附注：None
