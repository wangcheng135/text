序号：46645
名称：点眼金丝膏
组成：硇砂（研）1钱，晋矾（研）1钱，青盐（研）1钱，乳香（好者，细研）2钱，片脑（研）2钱，当归（锉，净洗）半两，黄丹（研）半两，黄连1两。
出处：《奇效良方》卷五十七。
主治：男子、妇人目疾远年近日，翳膜遮睛，攀睛胬肉，拳毛倒睫，黑花烂弦，迎风羞明冷泪，及赤眼肿痛。
加减：None
功效：None
用法用量：每用1粟米大点眼。
制备方法：上用好蜜4两，除片脑外和七味，纳入青筀竹筒内，油单纸裹筒口5-7重，紧系定，入汤瓶中，文武火煮一周时，取出劈破，新绵滤去药滓，方下片脑和匀，瓷瓶收贮，再用油单纸5-7重封系瓶口，埋露地内去火毒，候半月取出。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
