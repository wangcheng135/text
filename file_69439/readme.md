序号：69455
名称：猬皮散
组成：猬皮1具，蜂房1具，地榆5分，附子5分，桂心5分，当归5分，续断5分，干姜4分，蜀椒4分，藁本4分，厚朴6分。
出处：《千金》卷二十二。
主治：痈疽脓血内漏，诸漏坏败，男发背、女乳房及五痔。
加减：加斑蝥7枚益良。
功效：None
用法用量：每服方寸匕，空腹以酒送下，1日3次。取愈。
制备方法：上药治下筛。
临床应用：None
各家论述：《千金方衍义》：猬皮、蜂房逐垢之品，椒、姜、桂、附破结之属，当归调血，续断和伤，藁本祛邪，厚朴泄滞，地榆止痛。人但知地榆止下部血，不知《本经》原有乳产疮疡，七伤五漏，除恶肉，疗金疮之治，所以痈疽痔漏俱得用之。专佐猬皮、蜂房涤除瘀积也。
用药禁忌：None
附注：None
