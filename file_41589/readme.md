序号：41604
名称：金锁丹
组成：桑螵蛸（微炙黄色）半两，晚蚕蛾（雄者，微炒）半两，紫梢花半两，蛇床子（微炒）半两，远志（去心）半两，鹿茸（酥炙黄色）半两，川茴香（炒）半两，穿山甲5片（炙焦），海马2对（炙黄），续断3钱，石燕子1对（炭火煅赤，淬七返，研），麝香（研）1钱，南乳香（研）2钱半，木香2钱半，黑牵牛1两（微炒，取头末3钱）。
出处：《御药院方》卷六。
主治：None
加减：None
功效：延年益寿，至耄无痿。添精和气，驻颜壮骨，养神调气。
用法用量：每服50丸，空心及晚食前温酒送下。其功不可言也。如不及作丸，只作散服更妙。
制备方法：上为细末，用酒煮薄面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
