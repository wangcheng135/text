序号：57673
名称：大理气丸
组成：牛膝2两，甘草2两，人参2两，茯苓2两，远志2两，恒山2两，苦参2两，丹参2两，沙参2两，龙胆2两，芍药2两，牡蒙2两，半夏2两，杏仁2两，紫菀2两，龙骨2两，天雄2两，附子2两，葛根2两，橘皮2两，巴豆2两，狼牙2两，大黄3两，牡蛎3两，白术3两，白薇6分，玄参10分，？？芦1枚（大者），生姜屑5两。
出处：《千金》卷十二。
主治：万病，疝瘕癥结。
加减：None
功效：理气。
用法用量：每服7丸，空腹酒送下，1日3次。
制备方法：上29味，捣筛27味生药令熟，又捣巴豆、杏仁如膏，然后和使相得，加白蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
