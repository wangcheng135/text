序号：29635
名称：鸡肫丸
组成：三棱（炒）20两，莪术（炙）20两，茯苓20两，白术（炒）20两，神曲（炒）20两，麦芽（炒）20两，青皮（炒）20两，砂仁20两，橘皮20两，香附（炙）20两，莱菔子（炒）20两，枳壳（炒）20两，厚朴（炙）20两，干蟾（烧）40两，山楂40两，鸡内金（炒）810两，全蝎10两，木香5两。
出处：《北京市中药成方选集》。
主治：小儿脾胃不和，饮食难消，积滞痞块，腹痛胀满。
加减：None
功效：理脾化滞，消积止痛。
用法用量：每服1钱，温开水送下，1日2次，3岁以下小儿酌情递减。
制备方法：上为细末，过罗。用冷开水泛为小丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
