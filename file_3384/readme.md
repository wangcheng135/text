序号：3384
名称：太一银朱丹
组成：黑铅3两（炼10遍，与水银结砂子，分为小块，同甘草水煮半日，候冷，取出研，去草不用），水银（结砂子）3两，铁粉3两，甘草（同铅煮）10两，天南星（炮，为末）3分，朱砂（飞研）半两，腻粉（研）1两。
出处：《局方》卷十。
主治：小儿惊风发热，涎盛发痫，手足搐搦，目睛上视；及风壅痰实，心胸满闷，呕吐痰涎，大便秘涩。
加减：None
功效：None
用法用量：1岁儿服1丸，乳食后用薄荷、蜜汤送下。微利为度，未利再服。
制备方法：上为末，以面糊为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：太乙银朱丹(《普济方》卷三七三引《幼幼新书》)。
