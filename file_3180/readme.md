序号：3180
名称：天雄丸
组成：天雄1两（炮裂，去皮脐），石斛3分（去根，锉），五味子3分，巴戟1两，白茯苓3分，熟干地黄1两，远志3分（去心），人参半两（去芦头），补骨脂3分（微炒），蛇床子1两，泽泻3分，薯蓣3分，石南3分，萆薢3分（锉），附子3分（炮裂，去皮脐），沉香3分，石龙芮3分，桂心3分，棘刺3分，黄耆3分（锉），白龙骨1两，菟丝子1两（酒浸3日，曝干，别杵为末），杜仲3分（去粗皮，炙微黄，锉），肉苁蓉3分（酒浸1宿，刮去皴皮，炙干）。
出处：《圣惠》卷七。
主治：肾气不足，体重无力，腰背强痛。脚膝痠疼，耳目不聪，忽忽喜忘，悲恐不乐，阳气虚弱，小便失精。
加减：None
功效：None
用法用量：每日30丸，空心及晚食前以温酒送下。
制备方法：上为散，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
