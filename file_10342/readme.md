序号：10342
名称：加味补中益气汤
组成：人参（随便），黄耆（蜜炒）2钱，白术钱半，当归钱半，炙草8分，陈皮8分，柴胡（酒炒）3分，升麻（蜜炒）3分，杜仲钱半，续断钱半，淮山药（炒）钱半，百合2钱，五味子15粒。
出处：《会约》卷十四。
主治：堕胎。
加减：如肺虚有热，或口渴溺赤，或咳嗽喉燥，加麦冬1-2钱；如脾寒泄泻，加炮干姜1钱；如气滞而胀，加腹皮8分，或加枳壳7分。
功效：None
用法用量：生姜、大枣为引。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：胎至七月，肺经养之，前此或堕，肺经受伤，后孕宜预于六月调补之，必须大剂，不可间断，保过七月，则无虞也。
