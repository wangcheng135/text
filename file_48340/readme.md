序号：48355
名称：神效左经丸
组成：苍术（米泔浸）4两，草乌（去皮）4两，葱白4两，干姜4两（上捣烂，装入瓶内，按实，密封瓶口，安于暖处3日，取出晒干，入后药），金毛狗脊2两，藁本2两，白芷2两，破故纸（酒浸，焙干）2两，抚芎2两，小茴香（炒）2两，穿山甲（炮）2两，牛膝（酒浸）2两，川乌（炮）1两，木瓜1两，白附子1两，虎胫骨（酥炙）1两，乳香（炙）1两，没药（炙）1两（另研）。
出处：《景岳全书》卷五十六。
主治：诸风寒湿痹，麻木不仁，肢体手足疼痛。
加减：此方加当归6两更佳。
功效：None
用法用量：每服30-40丸，空心酒送下。
制备方法：上为末，酒糊为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
