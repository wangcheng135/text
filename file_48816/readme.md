序号：48831
名称：禹余粮丸
组成：禹余粮1两（烧，醋淬7遍），龙骨1两，紫石英1两（细研，水飞过），人参半两（去芦头），桂心半两，川乌头（炮裂，去皮脐），泽泻1两，桑寄生1两，川椒1两（去目及闭口者，微炒去汗），石斛1两（去根，锉），当归1两（锉，微炒），杜仲1两（去皱皮，炙微黄，锉），肉苁蓉1两（酒浸1宿，微锉，去皱皮，炙干），远志半两（去心），五味子半两，牡蛎1两（烧为粉），甘草半两（炙微赤，锉）。
出处：《圣惠》卷七十三。
主治：妇人劳损，因成崩中，不可禁止，积日不断，故成漏下，致五脏空虚，肉色黄瘦。妇人病月经乍多乍少，或前或后，时发疼痛。
加减：None
功效：和其阴阳，调其气血，使不相乘，以平为福。滋补本气。
用法用量：每服2丸，晚食前以热酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：紫石英丸（《本事》卷十）。
