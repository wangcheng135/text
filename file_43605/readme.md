序号：43620
名称：前胡散
组成：柴胡（去苗）、前胡（去芦头）、胡黄连、乌梅肉各等分。
出处：《杨氏家藏方》卷十。
主治：童男、室女骨蒸潮热，及热在肌肉，吐血等疾。
加减：None
功效：None
用法用量：上（口父）咀。每服5钱，水酒、童便共1盏半，猪胆1枚取汁，猪脊髓1条，葱、薤白各3寸，同煎至8分，去滓，食前冷服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：柴胡梅连散（《玉机微义》卷九引《瑞竹堂方》）、柴前梅连散（《玉机微义》引《瑞竹堂方》，见《医方类聚》卷六十三）、八煎散（《医方类聚》卷二一五引《医林方》）、柴胡梅连汤（《傅青主女科·产后编》卷下）、清骨散（《胎产心法》卷下）。
