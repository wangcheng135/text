序号：40527
名称：虎骨散
组成：虎胫骨2两（涂酥，炙令黄），败龟2两（涂酥，炙令黄），麒麟竭3分，赤芍药3分，当归3分，没药3分，自然铜3分（细研），牛膝1两（去苗），天麻1两，槟榔1两，五加皮1两，羌活1两，防风3分（去芦头），白附子3分（炮裂），桂心3分，白芷3分，苍耳子3分（微炒），骨碎补3分。
出处：《圣惠》卷二十一。
主治：风毒邪气，乘虚攻注皮肤骨髓之间，血气相搏，往来交击，疼痛不定，少得睡卧，筋脉拘急，不能屈伸，及打坠腰痛。风毒走注，疼痛不定，少得睡卧。游走不定，昼静夜甚。
加减：None
功效：None
用法用量：每服1钱，以温酒调下。不拘时候。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：None
附注：乳香趁痛散（《局方》卷一吴直阁增诸家名方）、乳香定疼散（《奇效良方》卷二）。方中白附子，《奇效良方》作“附子。
