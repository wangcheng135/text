序号：71944
名称：滋阴荣血汤
组成：当归（酒洗，去芦）1钱5分，川芎（去梗）1钱，白芍药8分，甘草（炙）5分，熟地黄（酒洗）1钱，白术（去梗）1钱，广陈皮（洗净）1钱，白茯苓（去皮）8分，侧柏叶（去梗）1钱（春取东方，夏取南方，秋取西方，冬取北方），香附子（石臼内舂去毛）1钱5分，北五味（去梗）8分，知母（去毛，不见铁）8分，麦门冬（酒洗去心）10个，黄芩（条实者，去梗）8分。
出处：《摄生众妙方》卷四。
主治：血证。
加减：None
功效：None
用法用量：上(口父)咀，作1帖。用水2钟，加生姜3片，枣子1枚，煎至7分，去滓，食前服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌煎炒、鱼腥、胡椒、咸酸、油腻之物。
附注：None
