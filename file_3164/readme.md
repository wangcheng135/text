序号：3164
名称：天麻煎丸
组成：天麻半斤（净洗，焙干，捣末），牛膝（酒浸，切，焙）1斤（捣为末），杏仁（汤浸，去皮尖双仁，研细）4两，生地黄5斤（好者净洗，于木臼内杵，取汁3斤），胡芦巴3两，天雄（炮裂，去皮脐）1两，石斛（去根）1两，沉香（锉）1两，巴戟天（去心）1两，玳瑁（锉末）1两，桂（去粗皮）1两，白花蛇（酒浸，炙，去皮骨，焙）1两，槟榔（锉）半两，独活（去芦头）1两，芎1两，大腹1两（和皮锉），当归（切，焙）1两，木香1两，益智（去皮）3分，远志（去心）3分，干姜（炮裂）1两，酸枣仁（炒）1两。
出处：《圣济总录》卷一八七。
主治：肝肾久积风冷痰滞气，上攻眼目，肿涩疼痛，肌肉瞤动，心神多倦。
加减：None
功效：去痰涎，壮筋骨，补元气，益心利肺。
用法用量：每服25丸，加至30丸，酒送下，若常服空心1服，有疾早、晚各1服。
制备方法：上22味，前4味于银器内以水3斗煎至5-7升，以布绞取汁，却将滓于木臼内捣令极细，后以水3-5升浸取汁，同于银器内熬，更用无灰酒5升，安息香3两，慢火煎成膏。余药为末，入前膏内和为团，于木臼内杵1000-2000下，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
