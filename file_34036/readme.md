序号：34045
名称：附子丸
组成：附子（炮，去皮）2两，人参2两，芎半两，干姜2两半，矾石1两（炼），皂荚（炙，去皮子）5分，半夏（洗）5分，桂心5分，矾石5分（烧），吴茱萸3分，茯苓3分，黄芩3分，当归1两，细辛1两，蜀椒（汗，去目、闭口者）1两，芍药1两，麦门冬（去心）1两半，甘草（炙）1两半。
出处：《千金翼》卷十九。
主治：胸膈中寒温不和，心下宛宛痛，逆害饮食，气满嘘吸，干噫吞酸，胸背中冷，两胁急痛，腹中有冷水，抑抑作声，绕脐痛，头眩满闷，身体羸瘦。
加减：None
功效：None
用法用量：每服2丸，食前酒送下，日3次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
