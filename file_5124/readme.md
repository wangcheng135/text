序号：5124
名称：玉液金丹
组成：人参2两（老山者佳），归身1两2钱（酒炒），白术8钱4分（制），川芎2两4钱，茯苓6两4钱，阿胶2两6钱（酒化），甘草3两2钱，蕲艾6钱7分，生地1两2钱，黄耆1两2钱（蜜炙），白芍1两6钱（酒炒、），苁蓉1两2钱（漂淡），麦冬2两5钱（去心），香附2两6钱（四制），川贝2两2钱（去心），广皮1两6钱（盐水炒），川断6钱4分（酒炒），枳壳1两2钱，杜仲2两6钱（姜汁炒），楂肉8钱4分，血余8钱4分（煅净），厚朴1两5钱（姜汁制），山药4两3钱，苏叶2两5钱，建莲6两4钱（去心），羌活8钱4分，木香8钱5分，沉香1两6钱，砂仁2两9钱，西珀8钱4分，丹参4两2钱黄芩1两2钱，菟丝子3两2钱，益母草6两4钱，大腹皮8钱4分，潼蒺藜2两2钱。
出处：《良方集腋》卷下。
主治：月经不调；胎前、临产、产后以及室女停经不至、潮热等症。
加减：None
功效：None
用法用量：初孕疑似之间，腹胀呕吐，用蔻仁3分煎汤下、头晕，用防风8分，煎汤下；头眩，用炒金银花1钱5分，煎汤下；胎动不安，用艾绒5分，子芩1钱，煎汤下；子呛，用桑白皮5分，煎汤下；子烦，用淡竹叶7片，煎汤下；子悬，胎动不安，如物之悬于虚中，宕而难住，神昏身狂，用赤茯苓8分，葱白1个，煎汤下；子冒，危于子悬，血热心火太盛，胎气上冲于心，胞冒于心上，面红，牙关紧闭，气绝欲死，用麦冬1钱，羚羊角5分，煎汤下；子肿，用五加皮1钱，赤苓皮1钱，煎汤下；子淋，用车前子1钱，煎汤下；漏胎，用原生地2钱，煎汤下；尿血，用粳米煎汤下；小便不通，用冬葵子8分，煎汤下；潮热，用知母1钱5分，煎汤下；咳嗽，用杏仁1钱2分，桑白皮5分，煎汤下；感冒、疟疾，用苏梗4分，荆芥5分，煎汤下；跌扑损胎，用白术5分，当归1钱，煎汤下；半产，用益母草2钱，煎汤下；临产交骨不开，用龟板3钱，煎汤下；横逆难产，数日不下，及胎死腹中，用川芎1钱，当归2钱，煎汤下；胞衣不下，用牛膝2钱，檀香1钱，煎汤下；恶露不行，用五灵脂5分，桃仁5分，生蒲黄5分，煎汤下；产后喘，成藕汁半杯或姜汁3匙，当审症用之；虚脱，用人参5分，煎汤下；胎前产后痢，用米仁3钱，煎汤下；产后肿胀，用茯苓皮1钱5分，当归1钱，煎汤下；褥劳，用官燕3钱，煎汤下；倒经吐血，用藕汁下；崩漏，用淡白鲞3钱，煎汤下；经期或前或后不准，以致艰于受孕，每逢天癸时服3丸，即能调经受孕，开水送下；胎前产后患症不一，不及遍载，俱用开水送下。
制备方法：先选择药料，日中晒燥，各磨细末，照方称准，用炼蜜5斤，并酒化阿胶和匀，于石臼中杵六千硾为丸，每丸2钱，再晒极干，用朱砂为衣，白蜡为壳，藏贮燥处。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
