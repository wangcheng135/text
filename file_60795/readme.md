序号：60810
名称：通痹丸
组成：川乌头（生，去皮脐）3两，五灵脂（炒，别研）3两，川当归（去芦，酒浸，焙）3两，草乌头（生，去皮尖）5两，没药（别研）2两，木鳖子（取肉，炒）2两，地龙（洗，去土）2两，白胶香（银器内熔过）1两，全蝎（麸炒）半两，朱砂（别研）半两，乳香（别研）1两半，麝香（别研）2钱。
出处：《魏氏家藏方》卷一。
主治：中风，左瘫右痪，口眼斜，半身不遂，手足顽麻，语言謇涩及一切风疾。
加减：None
功效：None
用法用量：每服1丸，生姜汁磨化，温酒浸服。妇人血风，当归酒磨下；小儿惊风，每丸分作3服，金银薄荷汤磨下；大人急中仆倒，磨1丸灌之即苏。
制备方法：上为细末，滴水为丸，如鸡头子大，风干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
