序号：70153
名称：滋补肝肾汤
组成：北沙参15g，白芍15g，川断15g，菟丝子15g，女贞子15g，五味子12g，首乌12g，黄精12g，当归12g，生甘草9g。
出处：《古今名方》第四章引关幼波方。
主治：慢性迁延性肝炎，肝功能长期不正常，证属肝肾阴虚型者。症见腰腿痠软无力，劳累则肝区痛，睡眠多梦，精神疲倦，头晕目眩，有时盗汗，舌净无苔或舌质稍红，脉沉细弦。
加减：若腰痛甚，加狗脊、桑寄生各15g；盗汗多，加生牡蛎、生龙骨、浮小麦各15g，乌梅9g；失眠重，加远志、百合各12g；梦遗滑精，加芡实、补骨脂、诃子肉各12g；肝区痛重，加草河车9g，黄连4.5g。
功效：滋补肝肾。
