序号：490
名称：五积交加散
组成：羌活1钱，苍术（米泔浸）8分，防风（去芦）8分，枳壳（麸炒）8分，陈皮8分，柴胡8分，当归（酒洗）8分，川芎8分，独活8分，白芷8分，半夏（姜汁制）8分，麻黄8分，桔梗8分，白茯苓8分，厚朴（姜炒）8分，桂枝4分，甘草3分。
出处：《寿世保元》卷七。
主治：妇人34-35岁，因经水到时，当风坐卧，失于回避，腠理空虚，外邪乘入。遍身麻痹，不能转侧；肺经受风，咳嗽痰盛。
加减：再服去柴胡，加乌药、僵蚕各1钱。酒煎，热服。
功效：None
用法用量：上锉。加生姜、葱，水煎，热服。只可服1-2帖，勿多服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
