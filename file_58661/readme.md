序号：58676
名称：秘传神仙拨云散
组成：真珠1分，硼砂1分，石燕子（炒过）1分，冰片3厘，硇砂（升过）半分，牙消（炒过）半分，石蟹半分，牛黄半分。
出处：《直指附遗》卷二十。
主治：翳膜。
加减：None
功效：None
用法用量：点眼。再用后方末药点之。
制备方法：上为极细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：末药：乳香（制过）、没药（制）各五分，轻粉五厘，石燕子一个（醋浸九次），青盐（湿纸包，烧）一分，珍珠（湿纸包，烧过）、琥珀（放豆腐内煮五次用）、血竭各三分，雄黄、雄胆（用箬焙干）、朱砂各二分，枯矾、胆矾各五厘，海螵蛸三厘，白丁香三粒（水洗过），炉甘石（火煅，童便渍七次）一钱。上研极细末，加冰片少许，点之。
