序号：53639
名称：家韭子丸
组成：家韭子6两（炒），鹿茸4两（酥炙），苁蓉（酒浸）2两，牛膝（酒浸）2两，熟地黄2两，当归2两，巴戟（去心）1两半，菟丝子（酒浸）1两半，杜仲（去皮，锉制，炒断丝）1两，石斛（去苗）1两，桂心1两，干姜（炮）1两。
出处：《三因》卷十二。
主治：少长遗尿；男子虚剧，阳气衰败，小便白浊，夜梦泄精。
加减：None
功效：补养元气，进美饮食。
用法用量：每服50丸，加至100丸，空心、食前、盐汤温酒送下。小儿遗尿，别作一等小丸服。
制备方法：上为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《明医指掌》引作“韭子丸”。
