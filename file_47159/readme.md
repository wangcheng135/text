序号：47174
名称：珍珠散
组成：珍珠1分（以纸裹灯上烧过，吹去灰，细研），琥珀5厘，黄柏末1分半，血竭1分，孩儿茶1分，熊胆1分，轻粉1分，冰片1分，麝香5厘，乳香2分，没药2分（俱置箬上，于新瓦上炙去油），炉甘石2分半（制法：每石4两，用黄连、黄柏、黄芩各5钱，煎汤分作3半碗。将石火煅1炷香，浸在1碗内吃干，再煅1炷香，另浸1碗内吃干，又煅又浸，如此3次，取起细研）。
出处：《万氏家抄方》卷四。
主治：下疳疮。
加减：None
功效：None
用法用量：用苦参、防风、荆芥、甘草、黄柏、大黄各等分煎汤，洗净敷之；如肿，用桃叶、槐叶煎汤，入生矾5钱，冷洗；如再不消，用细茶、川椒、炒盐各1钱，煎汤冷洗自消。
制备方法：共研极细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
