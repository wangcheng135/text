序号：69936
名称：琥珀保生锭子
组成：琥珀（研极细）3两，肉桂2两，五灵脂（醋炒）3两，生蒲黄3两，丁香1两，延胡索4两，红花2两，香附（醋炒）4两，大黄（酒蒸5次，须黑色为度，再入饭甑上蒸3次）4两。
出处：《陈素庵妇科补解》卷五。
主治：产后风冷袭于胞门，恶露不下，上逆冲心，发晕，额出冷汗，口噤牙紧，甚至不测。
加减：None
功效：逐瘀血，生新血。
用法用量：血晕、虚，用佛手散，热，用荆芥一味散煎汤磨服。
制备方法：None
临床应用：None
各家论述：琥珀色赤，入手少阴、足厥阴血分，能消瘀血，破癥结，定魂魄；肉桂大热，行血消瘀；五灵脂甘温，散血通闭；蒲黄生用，通经消瘀；丁香辛温，开郁祛胀，治胃家呃逆；红花色赤辛温，少则补，多则行；香附辛温，行气开郁，理三焦诸气；大黄破瘀血，推积祛滞，有斩关夺纛之功，酒煮则性上行，凡败血之冲心、冲肺、冲胃及停蓄上中焦者。皆可散之。又以饭甑上蒸，则得谷气，一切猛厉之性皆缓而有功，不伤脾胃。是方以保生命名，诚夺命回生之剂也。而血虚极者，煎佛手散，调化服之。虚火引热血冲上而逆者，煎一味荆芥散，调化服之。
用药禁忌：None
附注：None
