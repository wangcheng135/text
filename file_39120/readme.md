序号：39130
名称：泽漆汤
组成：泽漆根10两，鲤鱼5斤（若无鲤鱼鲖鱼亦可用），赤小豆2升，生姜8两，茯苓3两，人参2两，麦门冬2两，甘草2两。
出处：《千金》卷二十一。
主治：水气，通身洪肿，四肢无力，或从消渴，或从黄疸支饮，内虚不足，营卫不通，气不消化，实皮肤中，喘息不安，腹中响响胀满，眼不得视。
加减：水甚不得卧，卧不得转侧，加泽漆1斤；渴，加栝楼根2两；咳嗽，加紫菀2两，细辛1两、款冬花1合、桂3两，增鱼汁2升。
功效：None
用法用量：上(口父)咀。以水1斗7升，先煮鱼及豆，减7升，去之，纳药煮取4升半，每服3合，1日3次；人弱服2合，再服气下喘止，可至4合。晬时小便利，肿气减，或小溏下。若小便大利，还从1合，始大利便止。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：泽漆根汤（《千金翼》卷十九）。
