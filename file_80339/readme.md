序号：80355
名称：癣药水
组成：白及5两，槟榔5两，紫荆皮5两，川椒5两，大风子5两，百部10两，斑蝥2两。
出处：《北京市中药成方选集》。
主治：一切癣症、干癣、湿癣、风癣、牛皮癣、松皮癣、脚气，皮肤病。
加减：None
功效：杀虫止痒。
用法用量：用棉花浸湿，擦患处、每日1-2次。
制备方法：上用白酒192两，先入白及、槟榔煎30分钟，将药锅取下；候温再下荆皮、川椒煎10分钟，取下药锅；候温再下大风子、百部，将坛封口，浸泡1月，药力尽出方有效。于用药前3天将坛起封，再下斑蝥浸泡3天，速将药水滤清装瓶，每瓶重1两。
临床应用：None
各家论述：None
用药禁忌：谨防入口。三岁以下禁用。
附注：None
