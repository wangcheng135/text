序号：23209
名称：人参散
组成：人参3两（去芦头），鳖甲3两（涂醋，炙令黄，去裙襕），羚羊角屑2两，赤茯苓2两，知母1两，柴胡3两（去苗），地骨皮2两，枳壳2两（麸炒微黄，去瓤），牛膝2两（去苗），赤芍药1两，生干地黄1两半，牡丹2两半，川大黄1两（锉碎，微炒），百部2两，贝母2两（煨令微黄），黄芩3分，栝楼根1两，当归3分，桃仁1两（汤浸，去皮尖双仁，麸炒微黄），草豆蔻1两（去皮），安息香半两，川朴消3分，甘草3分（炙微赤，锉），紫菀1两（洗，去苗土），麦门冬1两半（去心），天门冬1两半（去心，焙），天灵盖1两半（涂酥，炙令黄）。
出处：《圣惠》卷七十。
主治：妇人骨蒸劳，身体壮热，手臂疼痛，月水不通。日渐瘦瘁，两胁气刺，四肢羸弱，腹内块生，时有咳嗽，不欲饮食。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，大枣3枚，煎至6分，去滓，食前温服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
