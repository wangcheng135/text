序号：26691
名称：导滞汤
组成：香附（酒炒）2钱，玄胡（炒）1钱5分，归尾2-3钱，木香4分，泽泻1钱半，红花（炒黄）1钱，淮牛膝（酒炒）1钱5分，桃仁（去皮）1钱5分，苏木1-2钱。
出处：《会约》卷十四。
主治：妇人经期，血因气滞而停，瘀积作痛，拒按属实者。
加减：火盛内热，血燥切痛者，加炒栀子2钱；微热者，加白芍1钱半；瘀极而大便燥结者，加大黄2-3钱，不应，加芒消、蓬术；寒凝作痛，加肉桂1-2钱，或吴茱萸1钱。
功效：None
用法用量：水煎，加酒服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：以上诸证，必实见其有滞无虚，方可用之；若或兼虚，勿行g伐，以伤脾肾；血通瘀下，停药，勿得过服。
附注：None
