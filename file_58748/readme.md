序号：58763
名称：大黄丸
组成：大黄（破如米豆，熬令黑）1升，柴胡1升，朴消1升，芎5两，干姜1升，蜀椒2两，茯苓（如鸡子大）1枚。
出处：《千金》卷二。
主治：带下、百病、无子。
加减：None
功效：None
用法用量：先食服7丸，米饮送下，加至10丸，以知为度，5日微下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：此治妇人带下、百病、无子，故用大黄、朴消以散积血；即用干姜、蜀椒以温子脏；柴胡升发生气；芎窮理荣血；茯苓引领消、黄专行渗道，与后养胎令易产方蒸大黄丸用法相仿。
用药禁忌：None
附注：服十日下血，二十日下长虫及清黄汁，三十日病除，五十日肥白。
