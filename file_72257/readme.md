序号：72273
名称：紫菀丸
组成：紫菀5分，桑根白皮5分，贝母5分，半夏5分，五味子5分，射干5分，百部5分，款冬花4分，皂荚4分，干姜4分，橘皮4分，鬼督邮4分，细辛4分，杏仁8分，白石英8分，蜈蚣2枚。
出处：方出《千金》卷十八，名见《千金方衍义》卷十八。
主治：积年咳嗽，喉中呀声，一发不得卧。
加减：None
功效：None
用法用量：每服10丸，稍加至20丸，1日2次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：久嗽积年不愈，必有宿垢留伏于中，呀声一发则伏火内动，所以坐卧不安，故用皂荚、射干、督邮之属辟除恶毒；兼石英、姜、辛助力祛邪；紫菀、款冬等味助供佐使之用耳。
用药禁忌：None
附注：None
