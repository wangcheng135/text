序号：36705
名称：固本种子丸
组成：大怀熟地8两（酒煮，晒，杵膏），补骨脂8两（青盐2两化，酒炒，去衣），透明鱼胶8两（醋银牡蛎粉炒成珠），山萸肉4两，白茯苓3两（人乳拌，晒），枸杞4两，怀山药4两，辽五味3两，牛膝肉3两，杜仲3两（刮去皮，盐水炒），泽泻3两（去毛），菟丝子2两（酒煮），牡丹皮3两，厚肉桂1两（去皮，不见火）。
出处：《救产全书》。
主治：None
加减：如阳萎，加真川大附子1两（童便浸制）；如本虚，加头胎紫河车1具（河水洗，银针挑去血筋，酒蒸烂，捣膏）。
功效：种子。
用法用量：先服3钱，后渐加至5钱止，男用每早空心白滚汤服。
制备方法：上为细末，炼蜜为丸，如桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
