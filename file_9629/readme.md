序号：9629
名称：长春至宝丹
组成：鹿茄茸（炙）4两，蚕蛾（炒）4两，鹿角胶（牡蛎粉炒成珠）4两，巨胜子（炒）4两，人参4两，哺退鸡蛋7个（炙黄），枸杞子（酒蒸）4两，当归（酒洗）4两，肉苁蓉（酒洗）4两，楮实子（去毛）4两，杜仲（姜汁炒）4两，牛膝（酒洗）4两，金樱子（炒）4两，巴戟（酒浸）4两，锁阳（酥炙）4两，葱子4两，韭子（炒）4两，故纸（炒）4两，熟地8两，鸽子蛋5个（蒸熟入药），何首乌1斤（9次煎蒸，去筋）。
出处：年氏《集验良方》卷二。
主治：命门火衰，阳痿精冷。
加减：None
功效：健脾开胃，进食止泻，强筋壮骨，增精补髓，乌须黑发，明目聪耳，活血养筋，助阳种子。
用法用量：每服3钱。
制备方法：上为粗末，将鸽蛋捣烂，入药拌匀，晒干为末，蜜和后臼中杵千余下为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
