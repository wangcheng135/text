序号：35533
名称：使君子丸
组成：使君子（用水和生面，裹炮，以面熟为度，只取使君子用）1两，丁香1两，无食子1两，熊胆（研）1两，胡黄连1两，夜明沙（微炒）1两，青黛（研）1两，肉豆蔻仁1两，黄连（去须，微炒）1两，芦荟（研）1两，蟾头（干者）1枚（炙黄），蟾酥（1皂子大），麝香1钱（研），龙脑半钱（研）。
出处：《圣济总录》卷一七三。
主治：小儿五疳羸瘦，腹胀下利，不思饮食。
加减：None
功效：None
用法用量：每3岁服5-7丸，用温米饮送下，每日3次，不拘时候。
制备方法：上为细末，拌令匀，湿纸裹粟米烧饭和丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
