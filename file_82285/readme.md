序号：82301
名称：磨积锭
组成：白术（陈土炒）2两，陈皮2两，厚朴（姜炒）1两，槟榔1两，枳实（炒）1两，三棱1两半，莪术（2味醋炒）1两半，甘草1两，使君子（去核，净）1两7钱，半夏曲1两，山楂（去核）2两，神曲2两，阿魏（真者）1两，黑牵牛（头末）1两（半生半炒），巴豆霜3钱（另研），木香3钱，硇砂1钱（洗去砂土），苍术（麸炒）1两。
出处：《医便》卷四。
主治：小儿一切积滞。
加减：None
功效：None
用法用量：8岁以上每服1锭，7岁以下半锭，空心滚白汤磨下。微利1-2次不妨。
制备方法：上为末，神曲一半，麦芽面一半，打糊为块，捣千余下，即作锭子，每锭湿重2钱，阴干约1钱。
临床应用：None
各家论述：None
用药禁忌：无积不可服。
附注：None
