名称：独活散
组成：独活3分，汉防己1两，秦艽3分（去苗），黄耆半两（锉），赤芍药半两，人参半两（去芦头），茯神半两，白术半两，芎半两，远志半两（去心），石膏1两，川升麻半两，防风3分（去芦头），丹参半两，甘草半两（炙微赤，锉），天门冬半两（去心），薏苡仁半两，羚羊角屑3分，五加皮半两，生干地黄3分，麻黄1两（去根节），地骨皮半两。
出处：《圣惠》卷十九。
主治：风痱。心热烦闷，四肢不仁。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，煎至5分，去滓温服，不拘时候。
制备方法：上为散。
临床应用：None
