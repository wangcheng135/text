序号：83691
名称：乌梅丸
组成：乌梅肉1两（微炒），肉苁蓉1两（酒浸1宿，锉，去皱皮，炙干），恒山3分（锉），川升麻3分，人参3分（去芦头），桃仁1两（汤浸，去皮尖双仁，麸炒微黄），甘草3分（炙微赤，锉），知母3分，香豉1合（炒干），鳖甲1两（涂醋，炙令黄，去裙襕），麝香半两（研细），桂心3分。
出处：《圣惠》卷五十二。
主治：劳疟，及瘴鬼等疟。
加减：None
功效：None
用法用量：每未发前服30丸，空心以温酒送下，至发又服30丸。每日服之，以愈为度。
制备方法：上为末，入麝香研匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
