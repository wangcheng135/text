序号：74609
名称：新加木贼煎
组成：木贼草1钱半，淡香豉3钱，冬桑叶2钱，制香附2钱，鲜葱白3枚，焦山栀3钱，粉丹皮2钱，夏枯草3钱，清炙草5分，鲜荷梗5寸。
出处：《重订通俗伤寒论》。
主治：伤寒少阳证，热重寒轻。
加减：None
功效：和解偏重清泄。
用法用量：None
制备方法：None
临床应用：None
各家论述：何秀山：木贼草味淡性温，气清质轻，色青中空，节节灵通，与柴胡之轻清疏达，不甚相远，连节用之，本有截疟之功，故张景岳代柴胡以平寒热，俞氏加减其问，君以木贼，领葱、豉之辛通，从腠理而达皮毛，以轻解少阳之表寒；臣以焦桅，领桑、丹之清泄，从三焦而走胆络，以凉降少阳之里热；佐以制香附，疏通三焦之气机，夏枯草轻清胆府之相火；使以甘草和之，荷梗透之。合而为和解少阳，热重寒轻之良方。
用药禁忌：None
