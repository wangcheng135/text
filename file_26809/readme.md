序号：26817
名称：延龄广嗣酒
组成：头红花半斤（入袋候用），淫羊藿（去边茎，净洗）1斤（用羯羊油拌，入袋候用），羯羊油（（羊[品木]？？）而肥者，用腰眼油1斤，切碎火锅内熬化候冷，拌淫羊藿），厚杜仲2两（童便浸1日，用麸炒去丝），天冬（去心）1两（酒浸软，晒干），肉苁蓉1两（河水洗净，浸去鳞甲，晒干），人参1两，砂仁5钱（姜汁拌炒），破故纸1两（酒浸1宿，微火焙干），川牛膝（去芦）1两（酒浸，晒干），白豆蔻（去皮）5钱，真川附子1两（童便浸透，蜜水煮3炷香，晒干），真川椒（有小卵者真，去子，焙干）、甘枸杞子4两，甘草（去皮）5钱（蜜炙），地骨皮1两（蜜水浸1宿，晒干），生地2两（乳浸，焙干），熟地3两（9蒸9晒，焙干），当归2两（酒洗，晒干），白茯苓2两（牛乳浸透，晒干），甘菊花1两（童便浸，晒干），五加皮4两，白术4两（米泔水浸，土炒），苍术4两（米泔水浸，晒干），母丁香5钱（不见火），广木香5钱（不见火），沉香5钱（不见火），白芍1两（酒炒），麦冬（去心）1两（炒）。
出处：《同寿录》卷一。
主治：一切腰腿酸痛，半身不遂，肾精虚滑，小便急数，阳痿艰嗣，女人子宫寒冷，赤白带下，胎前产后诸疾。
加减：None
功效：补气血，壮筋骨，和脾胃，宽胸膈，进饮食，去痰涎，行滞气，消宿食，避寒邪风湿，壮阳种子，延年益寿。
用法用量：每日随量饮之。如做2酒，再用米2斗，面曲6斤，蒸法如前，下缸再火烧酒40斤，封3-7日榨出。如3次酒，只入烧酒40斤，不用米曲矣。头酒系上好者，二酒三酒，可串和匀，入瓶封固，日常慢慢饮之，亦妙。
制备方法：上药各为细末。入上好面曲内，拌匀，用元占米4斗，淘净，再浸一宿，如造酒法蒸透，取出候冷；用淘米第三次之极清米泔水20斤，入锅内，加葱白1斤，切寸许长，入浆内滚三沸，去葱白，只用净浆，候冷和人蒸熟之米饭内，然后拌上好细曲米4斤，粗曲米2斤，并药末一总和匀；将羊油所拌淫羊藿，同头红花二味，各入绢袋内，先置缸底，方将曲药拌匀米饭，拍实，上用干烧酒10斤盖之，春发三日、夏一日、秋二日、冬四日后，再加烧酒80斤，将缸口封固，过2-7日开看，木扒打转3-4百下；如喜用甜者，加红枣3斤，同糯米3斤，煮成粥倾入，又从底打起，2-3百下；再过2-7日，即成功矣。将酒榨清，入坛内封固，重汤煮三炷香，埋土内三日。
