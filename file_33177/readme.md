序号：33186
名称：万灵丹
组成：败龟壳7钱半（醋炙），五灵脂1两（微炒），虎胫骨1两（醋浸，火烧存性），自然铜1两（煅，醋淬），生地黄1两（酒浸，焙干），麻黄（去节）1两，川乌半两（生用），草乌（锉如豆，盐炒）半两（不用盐），乳香3钱半（别研），木香3钱半，干木瓜2两。
出处：《永乐大典》卷一三八八○引《风科集验方》。
主治：风寒湿三气合而成痹，腿脚沉重，手足麻木，久则偏枯，脚气不能行履，腰胯不能动移；痪风、湿痹、瘫中者。
加减：None
功效：None
用法用量：每丸分作2服，用生姜自然汁同温酒化开服，更饮好酒1-2盏，空心、临卧各1服。初服觉口唇吻微麻，勿怪。
制备方法：上为细末，炼蜜为丸，如小弹子大，每两作10丸。
临床应用：None
各家论述：None
用药禁忌：忌食油腻热物。
附注：灵龟丹（原书同卷引《极济方》）。
