序号：28801
名称：苏蜜煎
组成：苏小1升，蜜小1升（煎，去滓沫；如无，用甘葛煎），甘葛煎大3升，地黄煎大2合5勺，麦门冬煎大2合5勺，生姜大5升（舂绞，煎得大5合），大枣150个（取肉），练胡麻大3合（熬，舂），干薯蓣小3两（舂，筛），茯苓小2两（舂，筛）。
出处：《医心方》卷十三引《煎药方》。
主治：诸渴。
加减：None
功效：内补。
用法用量：每服2丸，日3次；或丸如鸡子大，每服1丸，日3次，俱以酒送下。
制备方法：先以苏入生姜煎，煎之令相得，次入蜜，次以甘葛煎和大枣、练胡麻绞去滓入，次入干薯蓣、茯苓，次入麦门冬煎、地黄煎，皆入诸药，微火上煎，不离手搅令和调，冷之，为丸如枣大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
