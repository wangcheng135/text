序号：32715
名称：补药酒
组成：鹿筋8两，蕲蛇2条，虎骨胶1两，鹿胶1两（炒珠），饭党3两，杞子5钱，乳香1钱，五加皮5钱（盐炒），防风3钱，当归2两，熟地4两，茯苓5钱，木瓜5钱（盐炒），松节4钱，没药1钱，木香2钱，白芷3钱，羌活3钱，天麻3钱，牛膝5钱（盐炒），杜仲5钱，白术6钱（土炒），香附3钱，川芎3钱，川乌3钱，炙草3钱，饭耆3两，红枣4两，核桃肉3两。
出处：《医方易简》卷十。
主治：跌打刀伤各症，身体虚弱及年老弱者。
加减：None
功效：None
用法用量：早、晚随量常饮，服完欲再饮者，即照前法泡制。
制备方法：以瓦钵盛盖，用米烧酒12两拌湿透一日，隔水蒸熟，待冷，装坛内，每料只用上好米酒10斤，泡至30天之久可饮，愈陈愈妙。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
