序号：3396
名称：太乙五行膏
组成：牛蹄甲5两，马蹄甲5两，驴蹄甲5两，猪蹄甲5两，羊蹄甲5两，连翘1钱5分，三棱1钱5分，莪术1钱5分，黑丑1钱5分，白丑1钱5分，木香1钱5分，胡连1钱5分，沙参1钱5分，地骨皮1钱5分，玄参1钱5分，柴胡1钱5分，白芥子1钱，天花粉1钱，山楂6分，麦芽6分，神曲6分。
出处：《经目屡验良方》。
主治：一切无名肿毒。
加减：None
功效：散肿拔毒生肌。
用法用量：摊贴患处。
制备方法：先将五蹄甲入麻油2斤4两，熬枯去滓，再入连翘等16味，熬焦，滤清，俟油熬至滴水成珠，再入陶丹1斤2两，水飞收膏。
临床应用：None
各家论述：None
用药禁忌：眉心、耳后忌贴。
附注：None
