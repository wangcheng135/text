序号：47869
名称：神功妙贴散
组成：大南星（圆白者）4钱，蓖麻子仁4钱，五倍子（淡红者）2钱，白芷（削片）2钱，姜黄2钱，半夏（生）2钱，贝母2钱，白及2钱，没药3钱，乳香3钱，花蕊石散2帖。
出处：《直指》卷二十二。
主治：痈疽。
加减：None
功效：收晕敛毒，使脓血化为水出。
用法用量：疮色黯晦者，先用姜汁从晕边抹收入里，留中间如钱大贴膏药；若疮开大，可用纱布摊药，将旧茶笼内白竹叶尾剪两片如疮势（久年篷仰上竹叶亦得），先贴药上，然后贴疮，竹叶出水，藉药以行之。凡敷药须是细末则不痛。
制备方法：上为细末，以井水入蜜调。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
