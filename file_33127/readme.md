序号：33136
名称：万应锭
组成：川黄连2两，胡黄连2两，明乳香2两，净没药2两，孩儿茶2两，生大黄2两，延胡索2两，麒麟竭1两，明天麻1两，真熊胆1两，陈京墨4两，自然铜5钱，梅花冰片2分，原麝香2分。
出处：《饲鹤亭集方》。
主治：痰火中风，半身不遂；疔毒归心，痔疮、漏疮，喉闭，乳蛾，牙疳；温疹，伤寒，中暑，痢疾，血热，霍乱，瘟毒，黄病，疟疾，牙痛；小儿痘疹、惊风；妇人月经不调；无名肿毒，臁疮，手疮。
加减：None
功效：None
用法用量：大人4-5分，小儿2-3分，俱用凉水送下；一切无名肿毒、臁疮、手疮，俱用醋磨，敷于患处。
制备方法：上为细末，用头胎男子乳化熊胆，杵和成锭，如鼠粪样，飞金千叶为衣，密储勿泄气，听用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
