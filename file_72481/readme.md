序号：72497
名称：犀角大丸
组成：马鸣退2两，人参（去头）1两，干姜（炮）1两，附子（炮，去皮脐）1两，川芎1两，藁本1两，白芜荑1两，柏子仁1两，白薇1两，白术1两，苍耳1两，白芷5分，当归1两，泽兰9分，桔梗3两，石膏2两，甘草1两，防风5两，芍药1两，川椒2两，食茱萸5分，厚朴（去皮，姜汁炙）5分，蝉蜕2两，生犀半两。
出处：《传家秘宝》。
主治：八风十二痹，寒气乳风，血瘀，子死腹中，兼胎不安及胞衣不下，腹中(疒丂)痛绕脐，呕逆气冲、心中烦满，及产后恶露不尽。中风伤寒汗不出，肠痛积聚、金疮，恶疮，阴中痛，月经往来不止，或前或后，绝产无子，泄痢赤白。
加减：如中风兼伤寒，汗不出者，加麻黄3分（去芦，杵为末）酒煎，送下1丸。
功效：None
用法用量：每服1丸，空心温酒化下。妊娠临月，日服1丸，产时不知痛。如汗出不止，只用酒下1丸便止。肠痛积聚，朝、暮进1丸。金疮败脓，恶疮生头不合，阴中痛，月经来往不止，多少前后不一，服3-5丸。绝产无子，朝暮服之。
制备方法：上为末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
