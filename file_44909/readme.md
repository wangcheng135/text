序号：44924
名称：既济固真丹
组成：北五味子1两，白茯苓1两，附子1两，沉香1两，龙骨1两，苁蓉（酒浸1宿，如无以鹿茸酥炙代之）1两，益智仁半两，柏子仁（去壳，炒）半两，补骨脂（炒）半两，酸枣仁（去壳，炒）半两，金铃子（去核，炒）半两，红椒（去目）半两，当归（酒浸）半两，川巴戟（去心）半两，菟丝子（酒浸，合研）1两半。
出处：《朱氏集验方》卷八。
主治：水火不济，精神恍惚，头目昏暗，阳道痿弱，阴湿多汗，遗沥失精，脾胃虚怯，心肾不宁。
加减：None
功效：壮阳固气，温脾益血。
用法用量：每服50-70丸，空心盐、酒任下。
制备方法：上为细末，酒糊为丸，如梧桐子大，以辰砂末3钱为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
