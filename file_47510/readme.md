序号：47525
名称：神丹
组成：天南星（光白大者）7个（剜作锅儿），硇砂1两（填入南星锅内），赤脚蜈蚣7个（去头足），全蝎7个（去毒，上2味焙干，为粗末，俱入南星锅内，装满，仍用剜下南星末醋调封南星口）。上同南星以纸数十重裹了，用酒、醋浸透；于泥地上掘1穴坑，方圆1尺，4周排定砖，用炭10斤烧坑至砖通红，扫净；以酒、醋各2升泼在坑内，却将南星包儿放在中间，更以砖盖定，次用炭团团护之；经宿，次日取出，细切，焙干，碾为细末，入后药：粉霜（另研）半两，辰砂半两（另研），麝香1钱（另研），凌霄花1两（研细末），针砂半两（米醋同炒，另研）。
出处：《奇效良方》卷六十四。
主治：诸痫攧倒，手足搐搦，或吐或不吐，喉间有痰，咽下方甦。
加减：None
功效：None
用法用量：3岁儿每服30丸，10岁儿每服100丸，以灯心枣汤送下，1日3次。
制备方法：将前后药末相和，再为极细末，分作4份，一份猪心血为丸、一份羊心血为丸、一份羊蹄草根汁为丸、一份煮枣肉为丸，如小绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
