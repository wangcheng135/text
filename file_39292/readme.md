序号：39302
名称：疠风丸
组成：老虾蟆1个，白砒3钱，铜绿2钱，人中白2钱。
出处：《仙拈集》卷四引《集验方》。
主治：大麻风。
加减：眉毛先落，肝经受病，加石决明；先损眼目，心经受病，加栀子；面起紫泡，肺经受病，加葶苈；遍身如癣，脾经受病，加白术；脚底先破穿，肾经受病，加破故纸。
功效：None
用法用量：每服5分，滚水或麻黄汤酒送下俱可。五死引药：脾死，麻木不仁，苍术汤送下；血死，手足生脓，生地汤送下；肉死，割切不痛，苦参汤送下；筋死，手足脱落，续断汤送下；肾死，脚下有孔，薏苡牛膝汤送下。
制备方法：上为末装蟆口内，线缝好，以泥包裹，火煅烟尽为度，取起去泥，约有5钱，配苦参、瓦松各1两，神曲糊丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
