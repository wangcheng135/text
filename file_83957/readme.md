序号：83975
名称：五子丸
组成：覆盆子2两，杜仲2两（去皮，姜制，炒去丝），菟丝子2两（淘净，酒浸，研成饼），巴戟2两（去心），枸杞子2两，远志2两（去心），五味子2两（去枝），茯神2两（去木），肉苁蓉2两（酒浸，去土），当归2两（酒浸，去芦），山茱萸2两（去核），牛膝2两（酒浸，去芦），干山药2两，萆薢2两，熟干地黄2两（洗），黄精2两，破故纸2两（炒），青盐2两（别研），柏子仁2两（别研），石菖蒲1两（去须）。
出处：《魏氏家藏方》卷六。
主治：None
加减：None
功效：固心肾，补元气。
用法用量：每服20-50丸，空心温酒、盐汤送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
