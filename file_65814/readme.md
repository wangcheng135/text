序号：65830
名称：菩提丸
组成：前胡、薄荷、苍术、厚朴、枳壳、香附、黄芩、砂仁、木香、槟榔、神曲、麦芽、山楂、陈皮、甘草、白芍、藿香、紫苏、羌活、半夏各等分。
出处：《惠直堂方》卷一。
主治：瘟疫时病，疟疾，暑症，伤风咳嗽，赤白痢，水泻，霍乱，心腹痛。
加减：None
功效：None
用法用量：每服1丸，小儿量减。瘟疫时病感寒，姜汤送下；疟疾，姜汤送下；暑症，香薷汤送下；伤风咳嗽，百部3钱煎汤入姜汁送下；赤白痢，车前子汤送下；水泻，姜茶汤送下；霍乱吐泻，胡椒49粒，绿豆49粒煎汤送下；心腹痛，姜汤送下。
制备方法：用薄荷煎汤，拌各药匀，晒干为末，蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
