序号：23294
名称：人参截疟饮
组成：人参8分，白术（去芦）8分，茯苓（去皮）8分，当归8分，青皮（去瓤，麸炒）8分，厚朴（姜汁炒）8分，柴胡8分，黄芩8分，知母（去毛）8分，桂枝3分，常山（酒浸）8分，草果（去壳）8分，鳖甲（醋炙）8分，乌梅1个，甘草3分。
出处：《回春》卷三。
主治：虚人疟疾，及一切疟疾。
加减：寒多，加官桂；热多，加柴胡；汗多，去苍术、藿香、川芎，加白术、黄耆；饱闷，加青皮、砂仁，去人参；渴，加麦门冬、知母，去半夏；泻，加炒白术、芍药，泻不止，加肉豆蔻，去厚朴、草果；呕哕，加白术、山药、炒砂仁、炒米，去草果、厚朴、苍术；痰多，加贝母、竹沥，去半夏、草果；内热盛，加炒黄芩，去半夏；长夏暑热盛，加香薷、扁豆，去半夏、藿香。
功效：截疟。
用法用量：加生姜1片，大枣2枚、桃脑7个，水煎，加酒少许尤妙，露1宿。临发日五更空心温服；滓待日午再煎服，糖拌乌梅下药。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：切忌鸡、鱼、豆腐、面食及房劳、怒气。
附注：None
