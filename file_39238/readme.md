序号：39248
名称：狗脊丸
组成：狗脊1两，萆薢（酒浸）1两，防风（去芦）1两，川乌头（炮，去皮脐）1两，牛膝（酒浸）1两，肉苁蓉（酒浸）1两，破故纸（炒）1两，巴载（酒浸，去心）1两，胡芦巴（炒）1两，甜瓜子（炒）1两，威灵仙1两，没药（另研）1两，自然铜4两（火煅，醋淬7遍，研极细），当归1两（去芦）。
出处：《奇效良方》卷一。
主治：肾脏风虚，毒气上攻下注，腿膝脚气肿痛。
加减：None
功效：None
用法用量：每服20丸，加至30丸，食前用温酒送下，1日2次。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
