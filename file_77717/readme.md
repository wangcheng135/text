序号：77733
名称：雄黄膏
组成：槟榔半两，雄黄（别研，如无，舶上硫黄代之）半两，轻粉（别入）半两，枯矾半两，黄蜡半两，蛇床子1两，黄柏1两，吴茱萸1两，苦参1两，黄连1两，五倍子6钱，海桐皮6钱，(艹闾)茹2两。
出处：《医方类聚》卷一六九引《居家必用》。
主治：顽恶疮疥癣，小儿奶癣，头疮，无时痛痒；大人脚气下疰。
加减：None
功效：None
用法用量：若治疥疮，加入舶上硫黄与雄黄同分两。
制备方法：上为细末，先将腊月猪肪脂1斤，入皂角5条，带须葱5茎，全蝎10个，巴豆30粒去壳，蓖麻仁40粒去壳，川椒3钱，同煎黑色，去滓，入前药末，再熬成膏子，方入轻粉，腊月内合者，瓷盒内收贮，可留10年余。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
