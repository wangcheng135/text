序号：39036
名称：泽兰丸
组成：泽兰1两半，防风（去叉）1两，附子（炮裂，去皮脐）1两，当归（切，焙）1两，白术1两，桂（去粗皮）1两，芎1两，柏子仁1两，熟干地黄（焙）1两，石斛（去根）1两，厚朴（去粗皮，生姜汁炙，锉）半两，甘草（炙，锉）半两，细辛（去苗叶）半两，人参3分，干姜（炮）3分，牛膝（酒浸，切，焙）3分，肉苁蓉（酒浸，切，焙）3分，白芷3分，黄耆（锉）3分，续断3分，桃仁（去皮尖双仁，炒）4两。
出处：《圣济总录》卷一六四。
主治：产后蓐劳。
加减：None
功效：补虚损，益气血。
用法用量：每服30丸，空心以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
