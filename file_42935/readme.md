序号：42950
名称：保真广嗣丸
组成：潞党参2两，车前子1两5钱，怀牛膝（酒浸）2两，天门冬2两，石菖蒲1两，炒远志1两，当归（酒洗）2两，五味子1两，山萸肉2两，怀山药2两，覆盆子1两5钱，杜仲（姜汁炒）2两，巴戟肉2两，赤石脂（另研）1两，地骨皮1两5钱，广木香2两，大生地2两，枸杞子2两，川椒（微炒）1两，泽泻1两5钱，菟丝子（酒炒）4两，淡苁蓉4两，大熟地2两，柏子仁2两，白茯苓2两。
出处：《全国中药成药处方集》（杭州方）。
主治：男子诸虚羸瘦，精神衰弱，腰膝痠痛，阳痿乏嗣；妇人下元虚冷，久不孕育。
加减：None
功效：补益气血，滋培肝肾。
用法用量：每服3-4钱，空腹淡盐汤送下；冬月温酒或开水送下。
制备方法：上为细末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
