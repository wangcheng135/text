序号：12481
名称：白薇丸
组成：白薇18铢，紫石英30铢，泽兰2两，太一余粮2两，当归1两，赤石脂1两，白芷1两半，芎藭1两，藁本20铢，石膏20铢，菴（艹闾）子20铢，卷柏20铢，蛇床子1两，桂心2两半，细辛3两，覆盆子2两半，桃仁2两半，干地黄18铢，干姜18铢，蜀椒18铢，车前子18铢，蒲黄2两半，人参1两半，白龙骨2两，远志2两，麦门冬2两，茯苓2两，橘皮半两。
出处：《千金》卷二。
主治：久无子，或断绪，上热下冷。
加减：None
功效：None
用法用量：每服15丸，酒送下，1日2次。渐增，以知为度，亦可至50丸。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：慎猪、鸡、生冷、酢、滑、鱼、蒜、驴、马、牛肉；觉有孕即停；三月正择食时，可食牛肝及心。
附注：None
