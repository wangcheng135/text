序号：44012
名称：受拜茴香丸
组成：舶上茴香（童子小便浸3日，1日换1次，3日毕，漉出，炒）2两，破故纸（如前法浸）2两，金铃子（炒）2两，杜仲（姜制，炒，去皮，断丝）2两，吴茱萸（汤泡7次，炒）2两，川椒（去目合口者，用酒煮，漉出，炒）2两，蛇床子（炒）2两，橘核（炒，拣紧小者，不要大，恐是柑核）2两，延胡索（炒）2两，南木香1两（不见火），川乌头3个（大者，炮，去皮脐），胡芦巴3两（炒）。
出处：《魏氏家藏方》卷二。
主治：一切疝气。
加减：None
功效：补肾，消疝，止痛。
用法用量：每服30-40丸，空心温酒送下；盐汤亦得。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
