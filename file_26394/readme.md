序号：26402
名称：异香鳖甲散
组成：鳖甲2两（醋炙黄），牛膝1两5钱（酒浸），熟地黄1两5钱，人参2两5钱，大黄3分（煨），黑附子（炮，去皮脐）1两，京三棱（炮）1两，白茯苓1两，羌活1两，枳壳（去瓤，麸炒）1两，肉桂（去皮）1两，厚朴（姜制）1两，五味子1两，木香（不见火）1两，当归1两，白术（炒）1两，白芍药1两，肉豆蔻1两。
出处：《普济方》卷二三二引《卫生家宝》。
主治：五脏虚劳气攻，四肢无力，手足酸痛，背脊拘急，日渐虚弱，心下气满，不思饮食。
加减：None
功效：None
用法用量：每服3钱，水1大盏，加大枣3个，生姜5片，同煎至7分，去滓温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
