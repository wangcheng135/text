序号：35447
名称：三才丸
组成：人参、天门冬（去心）、熟干地黄各等分。
出处：《儒门事亲》卷十五。
主治：阴虚咳嗽，痨瘵，肾经咳嗽，真阴涸竭，气血俱虚，精神不固，元阳失合，脾肺虚劳咳嗽，上下纯虚而不嗽者。
加减：None
功效：滋阴养血，生津润燥。
用法用量：含化服之。
制备方法：上为细末，炼蜜为丸，如樱桃大。
临床应用：None
各家论述：《医方集解》：天冬以补肺生水；人参以补脾益气；熟地以补肾滋阴。以药有天、地、人之名，而补亦在上、中、下之分，使天地位育，参赞居中，故曰三才也。
用药禁忌：None
附注：三才封髓丹、三才丹（《症因脉治》卷二）、大地人三才丸（《麻疹全书》）。本方改为膏剂，名“三才膏”（见《诚书》；改为汤剂，名“三才汤”（见《医方集解》）。
