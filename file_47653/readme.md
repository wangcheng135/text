序号：47668
名称：千捶膏
组成：松香（锅内溶化，倾入清水内片时，揉白取用）约1斤，蓖麻子6两（净），柏油2两，白蜡2两，大黄2两，银朱2两，左顾牡蛎2两（用粗草包好，入火内煨，存性）。
出处：《种福堂方》卷三。
主治：无名肿毒及己溃者。
加减：None
功效：None
用法用量：用麝香少许，初起者1张便效。若己溃者，用阿魏少许，即止痛，且易收口。
制备方法：捶膏之法：在光平青石下，先将松香1-2两，与蓖麻1-2两，铺于石上，用铁锤打碎，干则加蓖麻，湿则加松香，余药亦渐渐掺入，捶极细腻为度。遇无名肿毒，摊贴。
临床应用：None
各家论述：None
用药禁忌：忌见火，须隔汤炖软摊之。
附注：None
