序号：48138
名称：神应比天膏
组成：黄芩（枯者）5钱，黄耆5钱，青皮5钱，陈皮（去白）3钱，乌梅（去核）8个，诃子皮2两（火炮），木鳖子16枚（去壳），山楂子16个，桃仁24个，苏木5钱，麝香少许，三棱（火煨）3钱半，莪术（火煨）3钱半，槟榔3钱，白豆蔻3钱，黄柏3钱，牙皂3钱（去皮弦子），当归尾1两，没药3钱半，乳香2钱半，昆布5钱，巴豆霜5分，甘草2钱半，穿山甲16个（醋炙黄焦）。
出处：《济阳纲目》卷四十一。
主治：男子、妇人气聚，左右胁下及冲胸，优梁，或血块，或气结，酒色过度，有伤五脏致死，精神短少，肢体羸弱，并小儿大人，一切痞疾。
加减：None
功效：None
用法用量：量疾大小摊药贴敷，遇痒时用木梳往来搔之。
制备方法：上除麝香、没药、乳香、巴豆霜不入，将群药（不见铁器）为极细末，用清香油14两、黄蜡2两，熬至数沸，方将群药末下入砂锅内熬，滴水不散为度，方下麝香等4味，用瓷罐盛下。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
