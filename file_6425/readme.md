序号：6425
名称：加味滋阴大补丸
组成：枸杞子（去枝蒂，酒拌，蒸）4两，沙苑蒺藜（酒洗，蜜酒拌蒸）3两，当归身（酒洗）2两，人参（去芦）1两，黄耆（蜜炙）2两，山药（人乳拌晒3次）2两，山茱萸（水洗，去核，童便拌晒）2两，白茯苓（去皮，漂去筋膜，人乳拌晒3次）2两，牡丹皮（酒洗，去心）2两，怀生地黄（酒洗）2两，怀熟地黄（酒洗）2两，天门冬（水洗，去心）2两，麦门冬（水洗，去心）2两，黄柏（川秋石入酒炒褐色）1两5钱，知母（川秋石入酒炒褐色）1两5钱，龟版（酒洗，酥炙）2两，杜仲（去粗皮，姜汁炒断丝）2两，牛膝（去芦，酒洗，同黑豆蒸二时去豆）2两，补骨脂（酒浸，蒸）2两，鹿角胶4两，菟丝子（水淘去沙，酒浸蒸，捣成饼，焙干）2两，肉苁蓉（酒洗，酥炙）1两5钱，锁阳（酒浸，酥炙）1两2钱，虎胫骨（酒浸，酥炙）2两。
出处：《医便》卷五。
主治：None
加减：None
功效：养气血，滋肾水，固元阳，添精髓，壮腰膝，润肤体，育心神；久服驻颜延年。
用法用量：每服3钱，空腹用淡盐汤送下；温酒亦可。
制备方法：上药各为细末，先以鹿角胶用无灰好酒溶开，和炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
