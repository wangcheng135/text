序号：50924
名称：大圣夺命金丹
组成：大麻（炮）5钱，全蝎（去毒）5钱，防风（去芦）5钱，羌活5钱，天南星（大者）5钱，白附子（炮）5钱，茯神5钱，白僵蚕（炒）5钱，川芎5钱，远志肉5钱，桔梗（去芦）5钱，石菖蒲5钱，半夏（生）5钱，人参（去芦）5钱，白术5钱，白茯苓（去皮）5钱，乌蛇尾（酒浸，炙）5钱，酸枣仁（炒）5分，荆芥穗5分，北细辛5分，大川乌（炮焦），粉草3钱，大赤足蜈蚣1条（薄荷汁浸，焙），以上除全蝎制、蛇尾制、蜈蚣外制，余药制，同研细为末，入全蝎3味、沉香3钱，如法研细，方入下药：辰砂3钱，龙脑3钱，珍珠3钱，金箔30片，银箔40片，真琥珀3钱，麝香1钱，雄黄1钱。
出处：《普济方》卷三七三。
主治：婴孩急慢惊风，癫痫天钓，客忤物件，中恶；及初生脐风撮口，胎惊胎痫，牙关紧急，惊风痰热，搐搦掣颤，反引窜视，昏闷不醒。
加减：None
功效：None
用法用量：金钱薄荷汤研化服；惊风牙关不开，搐鼻不嚏，以上药搐鼻，开关。如欲存久，宜安暖处，常加晒焙，免失药味。
制备方法：上为细末，姜汁面糊为丸，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
