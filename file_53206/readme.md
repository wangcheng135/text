序号：53221
名称：倍力丸
组成：牛膝（去芦，酒浸）1两，羌活（生）1两，巴戟（去心）1两，官桂（去粗皮，怀干）1两，天麻（酒浸）1两，狗脊（生，去毛）1两，萆薢（生）1两，杜仲（去皮，姜汁制，炒去丝）1两，茴香（淘去沙，炒）1两，桐皮1两，附子（炮，去皮脐）1两，川乌头（炮，去皮脐）1两，青盐1两（别研），没药（制研）半两，木香（湿纸裹煨）半两，防风（去芦，洗）半两。
出处：《魏氏家藏方》卷八。
主治：腰痛。
加减：None
功效：None
用法用量：每服30丸，食前盐汤送下。
制备方法：上为细末，用黑牵牛4两，微炒取粉，和面同煮糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
