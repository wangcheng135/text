序号：54329
名称：柴胡达原饮
组成：柴胡1钱半，生枳壳1钱半，川朴1钱半，青皮1钱半，炙草7分，黄芩1钱半，苦桔梗1钱，草果6分，槟榔2钱，荷叶梗5寸。
出处：《重订通俗伤寒论》卷二。
主治：湿重于热，阻滞膜原。
加减：None
功效：和解三焦。
用法用量：None
制备方法：None
临床应用：None
各家论述：俞氏以柴、芩为君者，以柴胡疏达膜原之气机，黄芩泄膜原之郁火也；臣以枳、桔开上，朴、果疏中，青、槟达下，以开达三焦之气机，使膜原伏邪，从三焦而外达肌腠也；佐以荷梗透之；使以甘草和之。虽云达原，实为和解三焦之良方。
用药禁忌：若湿已开，热已透，相火炽盛，再投此剂，反助相火愈炽，适劫胆汁而烁肝阴，酿成火旺生风，痉厥兼臻之变矣。用此方者宜慎之。
