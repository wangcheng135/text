序号：3326
名称：天雄散
组成：天雄1两（炮裂，去皮脐），石龙芮3分，独活3分，防风3分（去芦头），麻黄1两（去根节），茯神3分，杜仲3分（去粗皮，炙微黄，锉），萆薢3分（锉），丹参3分，桂心1两，羌活3分，五味子3分，细辛3分，牛膝3分（去苗），当归3分（锉，微炒），人参3分（去芦头），枳壳半两（麸炒微黄，去瓤）。
出处：《圣惠》卷七。
主治：肾脏风邪所伤，语言謇急，腰脊不可转侧，腰膝缓弱疼痹，头旋耳鸣，身体沉重无力。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，煎至6分，去滓温服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：石龙芮汤（《圣济总录》卷十九）。
