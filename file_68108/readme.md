序号：68124
名称：黄耆芍药桂枝苦酒汤
组成：黄耆5两，芍药3两，桂枝3两。
出处：《金匮》卷中。
主治：黄汗，身体肿，发热，汗出而渴，状如风水，汗沾衣，色正黄如柏汁，脉自沉。伤寒脉沉，咽痛自汗。
加减：None
功效：None
用法用量：上3味，以苦酒1升，水7升，相和，煮取3升，温服1升。当心烦，服至6-7日乃解；苦心烦不止者，以苦酒阻故也。一方用美酒代苦酒。
制备方法：None
临床应用：黄汗：张某某，女，22岁。因家务劳作汗出，即用凉水浸毛巾擦洗身体，后发现上半身出汗，色黄，量多而粘，衣物均被黄染。自觉乏力，纳呆，微发热，有时干哕，月经正常，小便色略赤，大便色正常，巩膜皮肤无黄染，舌质正常苔薄白，脉略滑。辨证：黄汗。时值盛夏，暑热当令，劳则阳气张，遂汗出，复受水寒之气，致热伏于内，酿成外寒湿，内郁热之势，交相蒸郁，汗液排泄障碍，或发热汗出而色黄。治则：调和营卫，清泄郁热。方药：耆芍桂酒汤去苦酒加栀子、黄柏。药用：黄耆18g，白芍12g，桂枝9g，黄柏9g，水煎分2次服。服3剂黄汗已止。随访3年未再发现黄汗。
各家论述：1.《千金方衍义》：水湿从外渐渍于经，非桂之辛温无以驱之达表；既用桂、芍内和营血，即以黄耆外壮卫气以杜湿邪之复入；犹恐耆、芍固护不逮，而用苦酒收敛津液不使随药外泄。乃服药后每致心烦，乃苦酒阻绝阳气不能通达之故，须6-7日稍和，心下方得快，然非若水煎汤液之性味易过也。_x000D_
2.《金匮要略心典》：黄耆、桂枝、芍药，行阳益阴，得酒则气益和而行愈固，盖欲使营卫大行，而邪气毕达耳。云苦酒阻者，欲行而未得遂行，久积药力，乃自行耳，故曰服至6-7日而解。
用药禁忌：None
