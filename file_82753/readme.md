序号：82769
名称：中虚丹
组成：朱砂6钱（悬胎，酒煮1伏时，如酒干，旋添熟酒煮之，温水浴净），附子2枚（1两2钱净者，各切作4片，剜作盒子，分入煮了，辰砂在内，用线扎定。剜下附子末不用），獖猪心2个（各切开，去心中血，将朱砂盒子入在心内，合定，再用灯心铺遍，以麻皮横扎，甑蒸烂熟，去猪心不用），酸枣仁（去皮，炒）半两，滴乳香半两（并别研）。
出处：《魏氏家藏方》卷六。
主治：心血耗散，心志不宁。
加减：None
功效：None
用法用量：每服20丸至30丸，临卧煎人参汤送下。
制备方法：上将附子(去皮脐)为末，辰砂别研细，4味拌研令和，度药末多少，用干山药末打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
