序号：57928
名称：益元七宝丹
组成：何首乌1斤，赤白1斤（用米泔水浸1日，竹刀刮去皮，打块如棋子大），牛膝8两（用黑豆5升，木甑沙锅蒸3次，晒3次，为末，加盐1-2钱同浸），枸杞子8两（酒浸，洗净，晒干，为末），茯苓1斤，赤白1斤（赤者用牛乳浸、白用人乳浸1宿，晒干，研末），兔丝子8两（酒浸3日，晒干，为末），破故纸8两（炒干，为末），当归8两（酒浸1宿，晒干，为末）。
出处：《遵生八笺》卷十七。
主治：None
加减：None
功效：去五脏杂病，生津液，健体轻身，益精明目，乌发固齿，壮阳增力。
用法用量：每服1丸，早晨空心酒送下，午后姜汤送下，临卧盐汤送下。
制备方法：上药俱不犯铁器，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
