名称：消暑活产丹
组成：人参1两，当归2两，川芎1两，肉桂2钱，青蒿1钱。
出处：《石室秘录》卷六。
主治：产后忽感中暑，霍乱吐泻。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：产妇只补气血，气血既回，暑气自散，况方中又有祛寒解暑之味乎，所以奏功独神也。或疑感暑是热，胡为反用肉桂？不知气血大虚，遍身是寒，一感暑气便觉相拂，非有大热之气，深入腹中也，不过略成暑气，与本身之寒，两相攻击以致霍乱。今仍用肉桂以温其虚寒，以青蒿而解其微暑，用之于大剂补气血之中，是以驾御而不敢有变乱之形。此立方之妙，而建功之神也，又何必疑哉！
