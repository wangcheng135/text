序号：28042
名称：补脾人参散
组成：人参1两（去芦头），石斛2分（去根），黄耆3分（锉），桔梗3分（去芦头），白术3分，附子半两（炮裂，去皮脐），桂心半两（去皮），白茯苓半两，陈橘皮3分（汤浸，去白瓤，焙），丁香半两，草豆蔻半两（去皮）。
出处：《圣惠》卷五。
主治：脾虚身重如石，四肢不举，食少无力，腹胀肠鸣，神思昏闷。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，大枣3枚，煎至6分，去滓，不拘时候稍热服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻、湿面。
附注：补脾人参汤（《圣济总录》卷四十四）。本方方名，《普济方》引作“人参汤”。
