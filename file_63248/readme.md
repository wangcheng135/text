序号：63264
名称：清金汤
组成：粟壳（蜜炒）半两，甘草（炙）5钱，陈皮（去白）1两，茯苓（去皮）1两，杏仁（去皮尖，炒）1两，阿胶（炒）1两，五味子1两，桑白皮（炒）1两，薏苡仁1两，紫苏1两，贝母（去心）1两，半夏曲1两，百合1两，款冬花1两，人参5钱。
出处：《袖珍》卷一引《圣惠》。
主治：远年近日咳嗽，上气喘急，喉中诞声，胸满气逆，坐卧不安，饮食不下。
加减：None
功效：None
用法用量：上（口父）咀。每服8钱，以水1盏半，加生姜3片，大枣2枚，乌梅1个，煎至8分，去滓，食后温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：青金汤（《普济方》卷一六一）。
