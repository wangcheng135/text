序号：53265
名称：健步丸
组成：黄柏（盐炒）40g，知母（盐炒）20g，熟地黄20g，当归10g，白芍（酒炒）15g，牛膝35g，豹骨（制）10g，龟版（制）40g，陈皮（盐炒）7.5g，干姜5g，锁阳10g，羊肉320g。
出处：《中国药典》。
主治：肝肾不足，腰膝酸软，下肢痿弱，步履艰难。
加减：None
功效：补肝肾，强筋骨。
用法用量：口服，1次9g，日2次。
制备方法：将羊肉洗净，剔去筋膜、油，加黄酒40g煮烂，与黄柏等11味为细末，过筛，混匀，每100g粉末用糯米粉5～10g，调稀糊为丸，干燥，即得。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
