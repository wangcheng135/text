序号：9637
名称：牛膝丸
组成：牛膝2两（去苗），白芍药1两，远志1两（去心），黄耆1两（锉），肉苁蓉2两（酒浸一宿，刮去皱皮，炙干），杜仲2两（去粗皮，炙微黄，锉），续断1两，蛇床子1两，薯蓣1两，菟丝子2两（酒浸一宿，晒干，别捣为末），白茯苓1两，人参1两（去芦头），鹿茸2两（去毛，涂酥，炙微黄），巴戟1两，柏子仁1两，桂心1两，五味子1两，石斛2两（去根，锉）。
出处：《圣惠》卷二十六。
主治：五劳六极七伤，小便数，阳气弱，腰脊疼痛，上焦虚热，恒多健忘，不能久立。
加减：None
功效：补暖益精，明目驻颜，轻身强记。
用法用量：每服30丸，空腹及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
