序号：66326
名称：雀附丸
组成：雀儿40只（去毛嘴足肠胃，以酒5升煮令烂，去骨烂研，井酒都绞取汁），硇砂2两（细研，以汤化澄滤于银器中，煎成霜，将小大瓜1枚，去皮子细切，以酒1斤半煮令烂，同研用之），川椒红2两（微炒，捣罗为末），菟丝子3两（酒浸3日，晒干别捣为末，上药并入雀儿煎中相和，搅令匀，以慢火熬如膏，入后药），附子2两（炮裂，去皮脐），肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），天麻1两，鹿茸2两（去毛，涂酥炙令微黄），补骨脂1两（微炒），沉香1两，木香1两，茴香子1两，石斛1两（去根，锉）。
出处：《圣惠》卷九十八。
主治：None
加减：None
功效：补虚冷，暖下元，壮腰脚，祛风气，充肌肤，益颜色。
用法用量：每日空心以温酒送下30丸；盐汤送下亦得。
制备方法：上为末。以雀儿膏为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
