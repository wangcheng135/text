序号：25008
名称：如圣散
组成：鸡粪4两（用雌雄鸡2只，饿2日，次早用猎胰子切碎，拌糯米粉1-2合，徐徐喂之，6-7日接粪4两为度，晒干听用），雌黄6钱，雄黄6钱，明矾1两，皮消1两，胆矾5钱。
出处：《外科大成》卷二。
主治：诸痔。
加减：None
功效：None
用法用量：用唾津调敷痔上，良久去药，再上药，如此7次，看痔黑色，则不须上药，待7日，其痔自落。略用生肌散，2-3日收口。
制备方法：上为末，入倾银罐内，用瓦盖之，火煅青烟为度，取出加乳香、没药各3钱、冰片5分，共为末，瓷罐收封口。
临床应用：None
各家论述：None
用药禁忌：None
附注：如金散（《惠直堂方》卷三）。
