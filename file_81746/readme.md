序号：81762
名称：鹤寿丹
组成：天浆子7枚（内有物者，微炒），蝉蜕2-7枚，牛黄1钱（细研），青黛1钱（细研），地龙3条，（微炒），蟾酥1钱（研入），朱砂半两（细研，水飞过），防风半两（去芦头），蚕纸1张（烧灰），麝香1钱（细研），乌蛇半两（酒浸，去皮骨，炙令黄）。
出处：《圣惠》卷八十五。
主治：小儿急惊风，口噤，手足抽掣，眼目直视，多吐涎沫，四肢壮热。
加减：None
功效：None
用法用量：每服3丸，以新汲水研下，不拘时候。
制备方法：上为末，炼蜜为丸，如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：天浆子丸（《普济方》卷三七）。《普济方》有半夏。
