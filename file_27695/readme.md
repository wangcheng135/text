序号：27703
名称：补阴丹
组成：朱砂（去石）半两，硇砂（去石）半两，延胡素半两，木香半两，半夏（汤浸7遍）半两，芫花（醋浸，炒黄色）半两，斑猫（去翅足，酒浸后炒令焦黑止）半两，川苦楝子（醋浸，炒黄）1两，荆三棱1两，海蛤1两，蓬莪术1两，大附子（炮，去皮脐）1两，舶上茴香1两，青皮1两，肉豆蔻3枚，槟榔3枚。
出处：《博济》卷二。
主治：小肠气，膀胱气刺疼痛；妇人产后恶物不尽，变作血瘕者。
加减：None
功效：大健脾元。
用法用量：每服5-7丸，女用醋汤，男用温酒或盐汤送下，空心、临卧各1服。
制备方法：上为细末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“补阴丸”。
