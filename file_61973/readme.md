序号：61989
名称：救汗回生汤
组成：人参3两，当归2两，柴胡2钱，白芍1两，陈皮5分，甘草1钱，麦冬5钱。
出处：《辨证录》卷一。
主治：冬月伤寒5-6日，吐泻后又加发汗，气喘不得卧，发厥者。
加减：2剂后减去柴胡，将此方减去10分之6，渐渐调理。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：此救坏病之一法也。人见人参之多用，未必不惊用药之太峻，殊不知阳巳尽亡，非多用人参，何以回阳于无何有之乡？尚恐人参回阳而不能回阴，故又佐以当归之多，助人参以奏功。至于白芍、麦冬之多用，又虑参、归过于勇猛，使之调和于肺、肝之中，使二经不相战克，而阳回于阴之中，阴摄于阳之内，听柴胡之解纷，实有水乳之合也，何必以多用参、归为虑哉。
用药禁忌：None
附注：None
