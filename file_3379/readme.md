序号：3379
名称：太一神明陷冰丸
组成：雄黄2两，丹砂2两，礜石2两，大黄2两，当归3，巴豆1分，芫青5枚，桂心2两，真珠1两半，附子1两半，蜈蚣1枚，乌头8枚，犀角1两，鬼臼1两，射罔1两，黎芦2两，麝香1两，牛黄1两，人参1两，杏仁30枚，蜥蜴1枚，斑蝥7枚，樗鸡7枚，地胆7枚。
出处：《千金》卷十七。
主治：心下支满，寒热鬼疰，长病欬逆唾噫，客忤中恶，胸中结气，咽中闭塞，绕脐绞痛，按之挑手，心中愠愠如有虫状，毒注相染灭门。
加减：None
功效：破积聚，辟邪气。
用法用量：先食饮服2丸，每日2次。不知稍加之。以药2丸安门户上，令众邪不近。伤寒服之无不即愈。若至病家及视病人，夜行独宿，服2丸，众邪不敢近。
制备方法：上为末，炼蜜为丸，如小豆大。
临床应用：None
各家论述：《千金方衍义》：陷冰为名，毒邪得药，如冰泮溪也。方中二十四味，除人参、丹砂、犀角、珍珠，余皆大毒大劣、上吐下泄之味，允为辟除邪罔之用也。用雄、丹、乌、附恢复真阳；鬼臼、射罔开辟阴邪；犀角、珍珠分解毒蕴；藜芦、矾石、大黄、巴豆上下消泄，激发蜥蜴、斑蝥荡瘀之威；人参、当归气血兼该，鼓舞乌、附复阳之力。方名“陷冰”，《千金》取譬“叔孙陷围，公徒释甲，执冰而踞”之意。或云“陷”“凝”古字通用，阳为阴陷，于义亦近。
用药禁忌：None
附注：本方方名，《医方类聚》引作“太乙神明陷冰丸”。
