序号：31104
名称：拒胜汤
组成：赤芍药1钱，半夏（炮7次）1钱，泽兰叶1钱，人参（去芦）1钱，陈皮（去白）1钱，甘草（炙）1分。
出处：《医学正传》卷七引《局方》。
主治：产后败血入于脾胃，而脾不能运化，胃不能纳谷，以致腹胀满闷，呕吐不定。产后胁痛，呕涎，脉弦软涩滞者。
加减：None
功效：None
用法用量：加生姜3片，水1盏半，煎至1盏，温服。
制备方法：上细切，作一服。
临床应用：None
各家论述：《医略六书》：产后气亏血滞，痰饮不化而阻塞于胸臆间，有碍少阳清净之化，故胁痛呕涎不止焉。人参扶元以通血脉，赤芍破血以行滞血，半夏化痰燥湿，泽兰利血通经，陈皮和中利气，甘草和胃缓中，生姜豁痰涎以除呕也；水煎，温服，务使气行血化，则痰无不消，而脾胃清和，何胁痛呕涎之不已哉。
用药禁忌：None
附注：抵圣汤（《产育保庆集》），巨胜汤（《卫生家宝产科备要》卷四）、抵圣散（《张氏医通》卷十五）。
