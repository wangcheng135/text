序号：36274
名称：周氏回生丹
组成：五倍子2两，沉香3钱，千金子霜1两，六神曲（炒）5两，檀香3钱，公丁香3钱，山慈菇1两5钱，木香3钱，甘草5钱，红芽大戟（醋炙）1两5钱。
出处：《北京市中药成方选集》。
主治：中暑受寒，饮食不节，霍乱吐泻，腹中绞痛。
加减：None
功效：祛暑散寒，健胃止痛。
用法用量：每服10粒（约重5分），生姜汤或温开水送下。
制备方法：上共研为细粉，过罗，每12两7钱细粉兑入麝香3钱，朱砂6钱，雄黄3钱，冰片3分，为细末和匀，用冷开水泛为小丸（每两200粒），朱砂为衣，每包10粒（约5分），蜡纸袋装。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：时疫止泻丸（《赵炳南临床经验集》）。
