序号：38971
名称：泻热汤
组成：前胡3两，茯苓3两，龙胆3两，细辛3两，芒消3两，杏仁4两，玄参2两，大青2两，苦竹叶（切）1升。
出处：《千金》卷十五。
主治：脾实热，舌本强直，或梦歌乐而体重不能行。
加减：None
功效：None
用法用量：上(口父)咀。以水9升，煮取3升，食后分3服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：泻热汤专用龙胆、大青、前胡、杏仁以清肝热内动之风；茯苓、细辛、玄参、苦竹以祛沃土外蒸之湿；或梦歌乐，可卜胃气之充；舌强体重，洵是脾家之实。实热交并，得不以芒消荡热存阴为务耶？
用药禁忌：None
附注：方中前胡，《医心方》作“柴胡”。
