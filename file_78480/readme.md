序号：78496
名称：摩腰丸
组成：腻粉1分，麝香1分（细研），朱砂1分（细研），硫黄1两（细研），白矾灰1两，母丁香1两，干姜1两，木香1两，附子1两，吴茱萸1两（汤浸7遍，焙干，微炒），陈橘皮1两（汤浸，去白瓤，焙），雀粪1两（以绢袋子盛，于水中摆取白尽，取此水澄之，晒干），杏仁1两（去皮尖，研之，依前绢袋子盛水中摆取霜，晒干）。
出处：《圣惠》卷四十四。
主治：腰疼痛，俯仰不得。
加减：None
功效：None
用法用量：用时取生姜自然汁小半盏，于铫子中煎1-2沸，倾于盏内，浸药1丸，良久药破，以指研之令细，旋旋以指点摩腰上，候热彻，摩尽为度，便以绵裹肚系之。
制备方法：上为末，炼蜜为丸，如半枣大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
