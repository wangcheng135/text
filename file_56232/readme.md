序号：56247
名称：流气丸
组成：木香1两，川茴香（微炒）1两，菖蒲1两，青皮（去瓤）1两，蓬莪荗（炒，锉）1两，红橘皮（去瓤）1两，槟榔1两，萝卜子1两，补骨脂（微炒）1两，荜澄茄1两，缩砂仁1两，神曲（微炒）1两，麦糵（微炒）1两，枳壳（去瓤）1两，牵牛（微炒）1两半。
出处：《御药院方》卷八。
主治：五积六聚，癥瘕癖块留饮。皆系寒气客搏于肠胃之间，久而停留不去，变成诸疾。
加减：None
功效：消导滞气，通和阴阳，消旧饮。
用法用量：每服50丸，食后细嚼白豆蔻仁1枚，白汤送下。
制备方法：上为细末，面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：虽年高气弱皆可服。
