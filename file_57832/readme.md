序号：57847
名称：珠珀保婴丹
组成：东牛黄5分，天竺黄2钱5分，山参1钱，礞石2钱，东珍珠1钱，明雄黄1钱，油桂1钱，半夏2钱，台麝5分，犀牛角2钱，粉草2钱，南星1钱，梅片5分，川贝母2钱5分（去心），前胡2钱，川羌2钱5分，血琥珀3钱，僵蚕1钱，防风2钱，蜈蚣2条，辰砂3钱，全蝎4个，天麻2钱5分，纹军3钱。
出处：《全国中药成药处方集》（哈尔滨方）。
主治：小儿急惊风症，痰火内蕴，外感风邪，卒然暴惊，壮热抖战，角弓反张，痰壅气塞，四肢搐搦．口眼斜，面青口噤。
加减：None
功效：清火定风。
用法用量：周岁小儿每服1分，2-5岁每服2分，6-8岁每服3分，灯心、竹叶、薄荷、鲜姜为引。
制备方法：牛黄、珍珠、麝香、辰砂、琥珀均各另研，余药碾为细面，一处调匀。
临床应用：None
各家论述：None
用药禁忌：忌食辛辣、生冷、油腻等物，乳母亦忌。
附注：None
