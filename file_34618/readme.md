序号：34628
名称：驱蛔汤
组成：使君子6g（炒香），炒榧子9g，乌梅3g，鹤虱6g，胡黄连6g，槟榔9g，香附6g，厚朴6g，甘草3g。
出处：《临证医案医方》。
主治：肠蛔虫症。
加减：None
功效：驱虫，理气解痉止痛。
用法用量：以上为5岁儿童用量。水煎服。
制备方法：None
临床应用：None
各家论述：方中炒使君子驱虫消积；炒榧子驱虫缓泻，二药并用驱虫疗效好，副作用小；乌梅味酸，蛔得酸则安；胡黄连苦寒，蛔得苦即停；故用乌梅、胡黄连安蛔；鹤虱能驱多种肠道寄生虫；槟榔消积理气，有泻下作用，能驱除虫体；香附、厚朴理气宽肠；甘草调和诸药。上方配伍可起驱虫之功效。
用药禁忌：None
附注：None
