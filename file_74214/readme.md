序号：74230
名称：山楂内消丸
组成：山楂（炒）3两，麦芽（炒）3两，五灵脂（醋炙）3两，橘皮4两，香附（醋炙）4两，法半夏2两，青皮（炒）2两，厚朴（姜炙）2两，砂仁1两5钱，三棱（麸炒）1两，莪术（醋炙）1两，莱菔子（炒）2两。
出处：《天津市固有成方统一配本》。
主治：气血凝滞引起的倒饱吞酸，胸满气胀，癥瘕痞块疼痛，大便秘结。
加减：None
功效：开胃化滞，消食化痰。
用法用量：每服3钱，温开水送下，1日2次。
制备方法：上药除莱菔子外，将山楂等11味轧为细粉，再将莱菔子轧碎，陆续掺入细粉轧细，和匀过80～100目细罗。取上药粉用冷开水泛为小丸，晒干或低温干燥，每两约500丸。纸袋装，入盒密封，置室内阴凉干燥处。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
