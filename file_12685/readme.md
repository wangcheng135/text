序号：12691
名称：白雪丸
组成：硝石1两，硫黄1两，白矾1两。
出处：《圣惠》卷四十。
主治：头偏痛。
加减：None
功效：None
用法用量：每服5丸，于食后以冷水送下。
制备方法：上以固济了瓶子一所，先下消石，次下硫黄，后下白矾，其瓶盖上留一小窍子，先掘一地坑，内更着水瓶子一所，坐令安稳，便将药瓶坐于瓶上，用泥密固之，以慢火逼之，候窍中相次烟出尽，即却泥之后，用大火熬令通赤，候冷，开瓶取出药，于纸上摊，用甘草水洒地令湿，以盆合之，2日出火毒了，细研，用绿豆粉，以水调和，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《医方类聚》引作“白雪丹”。
