序号：34604
名称：鸡苏丸
组成：远志（去心，炙）320两，五爪橘红160两，紫菀160两，知母160两，白芍160两，甘草160两，麦冬160两，冬花160两，五味子（炙）160两，葶苈子160两，沙参160两，兜铃160两，天冬160两，瓜蒌仁（炒）160两，桑皮160两，百合160两，法半夏160两，黄芩160两，桔梗160两，前胡160两，橘皮160两，苏叶320两，苏子320两，麻黄640两，生石膏320两，杏仁（去皮，炒）640两。
出处：《北京市中药成方选集》。
主治：肺经湿热，咳嗽痰盛，经久不止，喘息气促，睡卧不宁。
加减：None
功效：润肺止嗽，化痰定喘。
用法用量：每服2钱，温开水送下，1日2次。
制备方法：上为细末，过罗，用生姜160两，红枣160两煎，水泛为小丸，每16两用滑石、细粉4两为衣，以外上墨汁衣闯亮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
