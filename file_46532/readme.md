序号：46547
名称：千金止带丸
组成：香附（炙）160两，椿根皮（麸炒）160两，红鸡冠花160两，补骨脂（盐水炒）40两，木香40两，白芍40两，杜仲（炒）40两，白术（炒）40两，砂仁40两，续断40两，青黛（上衣用）40两，玄胡索（醋炒）40两，小茴香（盐水炒）40两，牡蛎（煅）40两，人参（去芦）20两，川芎80两，当归80两。
出处：《北京市中药成方选集》。
主治：妇女带下，腹痛腰痠，四肢倦怠，精神不振。
加减：None
功效：补虚止带。
用法用量：每服2钱，温开水送下，1日2次。
制备方法：上为细末，过罗，用冷开水泛为小丸，用方内青黛40两，外加滑石170两为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
