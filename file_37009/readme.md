序号：37019
名称：奇效神应肥儿丸
组成：人参3钱（如力微者，用1两，或以乳汁蒸北直参1两代之），大川黄连（酒炒）2钱，臭芜荑（炒，擦去皮，取净者）5钱，君子肉（面粉包煨，去壳，并去内黑皮）45粒，夜明砂（水淘去土砂，取净砂，醋炒干）1两，漂白术（陈壁土炒）5钱（不宜白术者，以意苡仁炒代之），吴神曲（炒）5钱，生麦芽（炒）5钱，天浆子（即五谷虫肉，汤浸，洗净，炒）1两，淮山药（炒）3钱，小青皮（醋炒）3钱，胡黄连（酒炒）3钱，山楂肉（去子，汤洗）3钱，干蛤模（火炙，醋淬，极焦）3只，花槟榔3钱。
出处：《麻科活人》卷四。
主治：肌瘦面黄，或面青面白，泄泻少食，肚腹胀大，青筋满腹，或伤饮食，常有吐泻，尿如米泔，及一切疳证。
加减：None
功效：None
用法用量：每服2-3钱，空心以米饮调下。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，据剂型当作“奇效神应肥儿散”。
