序号：62674
名称：清心丸
组成：牛黄3钱，脑子2钱，雄黄2钱，麝香1钱，川芎4两，茯苓4两，柴胡4两，桔梗4两，蒲黄2两，芍药2两，犀角（屑）半两，白术半两，黄芩半两，黑豆（炒）半两，阿胶（炒）半两，麦门冬（去心）半两，杏仁（去皮尖）半两，人参3钱，神曲（炒）3钱，肉桂（去皮）1分，羚羊角1两，干姜（炮）1分，山药1两3分，甘草1两半，金箔300片。
出处：《普济方》卷三七八。
主治：小儿惊痫潮搐，精神昏慢，痰涎流溢，并虚实烦躁，头疼恶心，风眩不语，呕吐倦怠。
加减：None
功效：None
用法用量：每服1丸，竹叶汤化下。
制备方法：上为末，炼蜜为丸，如鸡头子大，金箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
