序号：83045
名称：乌头丸
组成：川乌头1两（炮裂，去皮脐），天麻3分，干姜3分（炮裂，锉），乳香3分（细研），天竹黄3分（细研），防风3分（去芦头），蝎尾3分（微炒），麻黄1两（去根节），白鲜皮3分，地龙3分（微晒干），独活3分，海桐皮3分（锉），自然铜1两（作1块者，大火中煅令赤，投酽醋中，如此2-7遍，细研）。
出处：《圣惠》卷二十。
主治：卒中风，四肢麻痹，缓弱不能行。
加减：None
功效：None
用法用量：每服30丸，温酒送下，不拘时候。
制备方法：上为末，入研了药，都研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
