序号：64784
名称：第一药神仙换骨汤
组成：人参1两（去芦），槐角子1两（净），黄耆1两（蜜炙），蔓荆1两（好者），天仙子1两（去土，别研），瓜蒌根1两（白实），随风子2两（净，生），白蒺藜1两（拣净），犀角（取末）半两，苦参1两半，肉天麻1两，何首乌1两半，白附子1两（好者），防风7钱半（去芦），血竭半两（好者），枸杞子1两（拣红活者），莲心3钱（要好青者），白蜡半两（好者），沙参1两（实者），轻粉2两，蝎梢半两（全者亦妙），白花蛇（酒醋炙；如无，乌蛇代）、颧嘴川乌1只（不蛀者）。
出处：《普济方》卷一○九。
主治：大风。
加减：None
功效：None
用法用量：每服3钱，江茶调下，1日3-5次，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：服药此件不应，宜次第依方服之。
