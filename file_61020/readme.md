序号：61035
名称：铁刷汤
组成：良姜（油炒）6两，茴香（炒）2两，甘草（炙）8两半，苍术（米泔浸1宿）8两。
出处：《局方》卷三（宝庆新增方）。
主治：男子脾积心气痛，妇人血气刺痛，中酒恶心，一切疟痢气疾，肠风下血脏毒，滑肠泄泻，四方之人不伏水土；小儿脏寒脱肛，冒暑伏热，四时非节疫疠，痧瘴。黄疸，面目遍身如金色。
加减：None
功效：None
用法用量：每服2钱，加生姜3片，盐1捻，水1盏，煎至7分，温服，或热酒调下亦得；如脾寒，用酒1盏煎，临发时连进3服；四方之人不伏水土，小儿脏寒脱肛。并加生姜3片，大枣1枚，水煎服；冒暑伏热，擦生姜冷水调下；行路早起，大枣1枚（去核），包药少许，同生姜3片嚼下。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：铁刷散（《景岳全书》卷五十八）。
