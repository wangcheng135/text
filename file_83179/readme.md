序号：83197
名称：乌龙膏
组成：公牛角炭1斤，血余炭1斤，青麻炭1两，煅龙骨2斤，黑铅粉10斤，陈粉子3斤，陈醋适量。
出处：《外伤科学》。
主治：骨折损伤。
加减：None
功效：接骨，活血，止痛。
用法用量：用时，将膏摊于布上，厚约0.3厘米，敷贴患处，隔日换药1次；如无严重肿胀者，每周换药1次亦可。
制备方法：将公牛角锉成细条，瓦器内封闭，用火焙焦成褐黄色炭状；血余除去污垢，瓦器内封闭，火焙焦成黑色有光泽的炭块；青麻瓦器内封闭，火焙焦干后，启器盖，用火引之，急闭盖，再待1小时左右即成；将上药研为细末，同黑铅粉、陈粉子和匀，再把陈醋(无陈醋用食醋代之，但必须由2斤浓缩至1斤方可使用)放在瓷器内(搪瓷亦可)，用火煮沸，将药粉向醋内撒布，边放边搅至稠糊状，即可停止放药粉，再煎熬约半小时即成膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：黑铅粉又名石墨，翻砂厂内常用此品。陈粉子即玉米膏粱之类，用水泡霉，磨成粉，亦可用麦麸代替。
