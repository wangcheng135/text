序号：29172
名称：阿魏万灵膏
组成：真芝麻油2斤4两（浸后药，春5、夏3、秋7、冬10日），当归1两，川芎1两，防风1两，白芷1两，肉桂1两，木鳖仁49粒，蓖麻仁25粒，巴豆仁49粒，穿山甲7大片，槐枝30寸，柳枝30寸。
出处：《活人方》卷四。
主治：顽痰积蔽，风湿客邪，停滞于皮肤、分肉、经络遂道之间，或积累成块，或阻塞为痛，药石难攻，宜乎外治者用之。
加减：None
功效：None
用法用量：凡用，看患之大小，摊厚青布上，先以水、姜擦过方贴，贴后以热手或热盐包熨之。
制备方法：上药油浸足日期，用文武火煎至药焦为度。以3-5重丝绵滤滓务净，将滓另于银标内烧出药油，共得净油约二斤；再以飞丹（炒黑，净）□斤，乘油出火，渐渐调入令匀；冷后再入真阿魏一两，用葱汁顿化，搅入令匀；再加滴乳香一两，没药一两，血竭一两，肉桂五钱，附子五钱，麝香一钱，研极细末，渐渐搅入令匀，收贮于有盖之厚瓷罐内封固，取用后，仍须盖紧。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
