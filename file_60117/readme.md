序号：60132
名称：透体异香丸
组成：沉香1两，木香1两，丁香1两，藿香1两，没药1两，陵零香1两，甘松1两，缩砂1两，丁皮1两，官桂1两，白芷1两，细茶1两，香附1两，儿茶1两，白寇1两，槟榔1两，人参1两，乳香5钱，檀香5钱，三奈5钱，细辛5钱，益智5钱，当归5钱，川芎5钱，乌药5钱，麝香2钱，朝脑2钱，薄荷1两。
出处：《鲁府禁方》卷四。
主治：五膈、五噎痞塞，诸虚百损，五劳七伤，体气，口气。
加减：None
功效：壮阳滋肾，补益丹田。
用法用量：每服1丸，清晨噙化，用黄酒送下。
制备方法：先将大粉草半片挫片，水煮汁，去滓，将汁熬成膏，将前药为末，炼蜜共膏，为丸，如芡实大。
临床应用：None
各家论述：None
用药禁忌：忌生冷，毒物。
附注：透体气口丸（《寿世保元》卷五）。
