序号：60769
名称：通真延龄丹
组成：五味子3斤，山茱萸2斤，菟丝子2斤，砂仁1斤，车前子1斤，巴戟天1斤，甘菊花2斤，枸杞子3斤，生地黄3斤，熟地黄3斤，狗肾4斤，怀山药2斤，天门冬1斤，麦门冬3斤，柏子仁2斤，鹿角霜2斤，鹿角胶4斤，人参2斤，黄柏1斤半，杜仲1斤半，肉苁蓉3斤，覆盆子1斤，没食子1斤，紫河车10具，何首乌4斤，牛膝3斤，补骨脂1斤，胡桃肉2斤，鹿茸1斤，沙苑蒺藜4斤（2斤炒磨，2斤磨粉打糊）。
出处：《广笔记》卷二。
主治：阳痿无火。
加减：None
功效：None
用法用量：每服5钱，空心、饥时各1服，龙眼汤吞下。
制备方法：上为末，同柏子仁、胡桃肉泥、蒺藜糊、酒化鹿角胶炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：有火者不可服。
附注：通真延龄种子丹、腽肭脐丸（《医学正印》卷上）。方中狗肾，《医学正印》用海狗内、外肾各一副。如无即本地黑狗或黄狗内、外肾各一副酥制。其服法用龙眼汤、淡盐、寒天好酒任下。
