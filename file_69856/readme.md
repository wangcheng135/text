序号：69872
名称：小活络丹加味方
组成：川乌5钱（炮，去脐皮），草乌5钱（去皮），胆星4钱，地龙5钱（洗净，焙干），乳香4钱（去油），没药4钱，天麻4钱，豨莶草6钱，于术8钱（生），当归1两，白芍6钱（炒），桑寄生8钱，抚芎4钱，生地1两，橘红5钱。
出处：《慈禧光绪医方选议》。
主治：风湿诸痹，肩背腰膝筋脉骨节疼痛，偏正头痛，或口眼歪斜，半身不遂，行步艰难，筋脉拘挛，肌肉顽麻沉重酸木，或皮肤作痒。无病之人，中年以后常服，亦可预防风症。
加减：None
功效：清心明目，宽胸畅膈，宣通脉络，滋益营卫，理脾除湿。
用法用量：每服1丸，老酒送下。
制备方法：上为极细末，老酒、白蜜各半为丸，重1钱5分，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
