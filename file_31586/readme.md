序号：31595
名称：杜蘅丸
组成：杜蘅（三月三日采根，洗，晒干，锉）1两，半夏（汤洗20遍，碎用，生姜炒）1两，白薇1两，桔梗（锉，炒）1两，附子（炮裂，去皮脐）1两，牛膝（切，酒浸经宿）1两，石菖蒲半两，蜀椒（去合口并目，炒出汗）半两，细辛（去苗叶）半两，厚朴（去粗皮，生姜汁炙）半两，沙参（去芦头）半两，防风（去叉）半两，干姜（炮，锉）半两，桂（去粗皮）半两。
出处：《圣济总录》卷一五三。
主治：妇人久无子，断绪。
加减：None
功效：None
用法用量：每服20丸，渐加30丸，早晨、日午温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
