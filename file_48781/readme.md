序号：48796
名称：大力丸
组成：蒺藜（酒洗，炒，去刺）1两5钱，白茯苓1两5钱，白芍1两5钱，苁蓉（酒洗）1两5钱，杜仲（酥油炒）1两5钱，菟丝子（酒煮）1两5钱，续断1两5钱，当归1两5钱，覆盆子1两5钱，威灵仙1两5钱，破故纸1两5钱，薏苡仁1两5钱，牛膝（酒洗）1两，无名异1两，自然铜（醋煅7次）1两，乳香5钱，没药5钱，朱砂（飞过）5钱，血竭5钱，青盐5钱，天雄2两（童便浸5日），象鳖10个（去头足翅，如无，用土鳖），跳百丈10个（去足），龙骨2两（酥油炙）。
出处：《良朋汇集》卷五。
主治：None
加减：None
功效：增力。
用法用量：每服1丸，早、晚盐汤或黄酒送下。少时，用力行功，散于四肢。
制备方法：上为细末，炼蜜为丸，每丸2钱半重。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
