序号：70324
名称：煮散
组成：茯神6两，防风4两，牛膝4两，枳实（炙）4两，防己4两，秦艽4两，玄参4两，芍药4两，黄耆4两，白鲜皮4两，泽泻4两，独活4两，桂心3两，五味子1升（碎），人参4两，薏苡仁1升（碎），麦门冬1两（去心），羚羊角2枚（屑），石膏1斤（碎，绵裹），甘草3两（炙），磁石24两（绵裹）。
出处：元侍郎希声引《张文仲方》（见《外台》卷十四）。
主治：诸风。
加减：None
功效：None
用法用量：每日1贴，加杏仁14枚（去皮尖双仁，碎），以水3升，煮取1升，去滓，空腹顿服。每春中夏初时服。
制备方法：上切，如麻豆大，分作24贴。
临床应用：None
各家论述：None
用药禁忌：禁生冷，忌醋、生葱、海藻、生菜。
附注：None
