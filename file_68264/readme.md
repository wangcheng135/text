序号：68280
名称：黄耆散
组成：黄耆1两（锉），续断3分，当归3分，熟干地黄1两，白术3分，五味子3分，石斛1两（去根，锉），桂心1两，白芍药1两，诃黎勒皮1两，人参3分（去芦头），木香半两，白茯苓1两，附子1两（炮裂，去皮脐），甘草半两（炙微赤，锉），麦门冬1两半（去心，焙），牛膝1两（去苗），陈橘皮3分（汤浸，去白瓤，焙）。
出处：《圣惠》卷三十。
主治：虚劳。少气，面色萎黄，四肢羸瘦，腹胁妨闷，吃食减少。日渐虚困。
加减：None
功效：None
用法用量：每服4钱，以水1中盏；加生姜半分，大枣3枚，煎至6分，去滓，食前温服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“黄耆汤”。
