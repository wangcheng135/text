序号：50640
名称：荆防败毒散
组成：柴胡4分，甘草4分，人参4分，桔梗4分，川芎4分，茯苓4分，枳壳4分，前胡4分，羌活4分，独活4分，荆芥穗4分，防风4分。
出处：《医学正传》卷八。
主治：痈疽疮疡初起，发热，脉浮数，及水肿邪在表者;伤寒温春发斑重者;痘疹，及时气风毒邪热;风水、皮水，凡在表宜从汗解者;肠风下血清鲜者;脑疽、甘疽、赤白游风、疥疮初起有表证，虚者。
加减：None
功效：发散痘疹,疏解寒热。
用法用量：用水1盏，煎至7分，温服；或加薄荷5叶。
制备方法：上细切，作1服。
临床应用：耳目赤肿：一人耳面赤肿作痛，咽干发热，脉浮数。先以荆防败毒散2剂，势退大半，又以葛根牛蒡子汤4剂而痊。
各家论述：None
用药禁忌：None
附注：消风败毒散（《医学六要·治法汇》卷五）。《医学六要·治法汇》有生姜三片。
