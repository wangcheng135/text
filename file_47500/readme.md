序号：47515
名称：祛瘴辟瘟丹
组成：厚朴1钱，苍术1钱，羌活1钱，防风1钱，陈皮1钱，枳实1钱，香附1钱，牛蒡子1钱，槟榔8分，白芷8分，藿香5分，川芎5分，细辛4分，甘草3分。
出处：《痧书》卷下。
主治：时疫痧瘴。
加减：无汗，加苏叶、薄荷；口渴，加花粉、葛根；身重汗出，加防己、石膏；温疟，加柴胡、半夏；遍身疙瘩肿痛，加兰叶、大黄、僵蚕；肌肉发红黑紫斑，加元参、大青、连翘；大便秘结，加大黄；先中热又中暑，加白虎、香薷；头疼，加川芎；风温身体灼热，加芩、连、栀子；咳嗽涕唾，头目昏眩，加荆芥、金沸草。
功效：None
用法用量：加姜、葱，水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
