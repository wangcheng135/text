序号：26502
名称：八味厚朴丸
组成：厚朴（去粗皮，生姜汁炙令紫）2两，陈橘皮（汤浸去白，焙）1两，诃黎勒（取肉，生用）1两，桂（去粗皮，取肉）1两，附子（炮裂，去皮脐）1两，干姜（炮）1两，白茯苓（去黑皮）1两，甘草（炙）1两。
出处：《圣济总录》卷四十五。
主治：脾胃虚，冷气上攻胸膈，三焦不调，不思饮食，及饮食不消，肌瘦少力。
加减：None
功效：开胃进食。
用法用量：每服30丸，空心、晚食前温酒或盐汤送下，嚼破。如大段膈气，进食不得，即留一半散，每服2钱匕，以酒2合，水2合，同煎至2合，和滓下丸药30粒。
制备方法：上为细末，拌令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：厚朴丸（原书卷四十七）、厚朴煎（《鸡峰》卷十三）。本方以醋糊为丸，名“烧胃丸”_x000D_
（见《普济方》）。
