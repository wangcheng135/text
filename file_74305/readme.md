序号：74321
名称：塞耳杏仁膏
组成：杏仁（去皮尖双仁，别研）1两，蓖麻子（去皮，别研）1两，食盐（别研）2分，乳香（别研）1分，巴豆（去皮心，别研）1分，附子（炮裂，去皮脐）1分，桃仁（去皮尖双仁，别研）1两，磁石（火烧醋淬1-7遍）1两，木通（锉）半两，蜡2两，菖蒲1两。
出处：《圣济总录》卷一一四。
主治：风聋久不愈者。
加减：None
功效：None
用法用量：绵裹塞耳中，1日4-5次。
制备方法：上除别研外，捣罗为末，后入别研者相和，捣丸，捻如枣核大。
临床应用：None
各家论述：None
用药禁忌：None
附注：蓖麻丸（《普济方》卷五十三）。本方方名，《普济方》引作“杏仁膏”。
