序号：61198
名称：铅丹散
组成：铅丹2分，胡粉2分，栝楼根10分，甘草10分，泽泻5分，石膏5分，赤石脂5分，白石脂5分。
出处：《千金》卷二十一注文引《古今录验》。
主治：消渴，小便数，兼消中。心神烦闷，头痛。
加减：None
功效：None
用法用量：每服方寸匕，水送下，日3次；壮人1匕半。渴甚者，夜2服，腹痛者减之。丸服亦佳，每服10丸。服此药了，经2-3日，宜烂煮羊肝空腹服之；或作羹亦得，宜汤淡食之。侯小便得咸，更即直服苁蓉丸兼煮散将息。
制备方法：上药治下筛。
临床应用：None
各家论述：《千金方衍义》：栝楼治肺胃燥渴；甘草解毒安中；石膏清胃止渴，兼解石药之悍；铅丹、胡粉、赤白石脂镇摄痰涎，分走血气。石膏解石药毒，甘草解草药毒，栝楼专主消渴，泽泻兼治水逆，总解热而化下毒。
用药禁忌：不宜酒下，用麦汁下之。
附注：胡粉散（《儒门事亲》卷十三）。
