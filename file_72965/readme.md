序号：72981
名称：渴龙奔江丹
组成：白矾1两半，火消1两3钱5分，黑矾1两，黑铅2钱半，水银（铅制）5钱，青盐5钱，明雄1钱5分，硼砂1钱5分，白砒1钱5分。
出处：《吴氏医方类编》卷四。
主治：一切恶疮疔毒。
加减：None
功效：None
用法用量：点疮口上。
制备方法：各为细末，用甘子土作罐，如元宝罐样，先以文火，次下白矾，再次下青盐，次下火消，硼砂，黑矾，以物搅之，俟结于罐底，先以大接白罐盛水令满，埋与地平，口内坐大白碗1个，将药覆碗内，靠罐边以毛头纸拈筋护住，炭火3分，碱土7分，盐水和泥填满碗，用瓦围好罐，沿上排炭火6斤，发火烧之，以炭尽为度，俟冷取起，将白碗底霜用鸡翎扫下，研细，江米糊和成条，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
