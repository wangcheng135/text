序号：37070
名称：定心汤
组成：人参（去芦）3两，白茯苓（去皮）3两，茯神（去木）3两，黄耆（蜜炙）3两，白术2两，赤石脂（研）2两，川芎2两，厚朴（姜汁制）2两，紫菀茸2两，防风2两，麦门冬（去心）1两半，官桂（去皮）1两，甘草（炙）1两。
出处：《简易方》引《叶氏录验方》（见《医方类聚》卷一五○）。
主治：心气不足，营血衰少，多畏不乐，精神昏昧，魂魄飞扬，心神离散，梦中失精，白浊。
加减：None
功效：None
用法用量：每服3钱，水1盏半，加赤小豆70粒，煎7分，去滓，食后通口服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
