序号：18627
名称：玉粉丹
组成：伏火硫黄2两，白龙骨（细末）2钱，钟乳粉2钱，附子（炮，去皮脐，取末）2钱，白石脂（细研）2钱。
出处：《杨氏家藏方》卷七。
主治：肠胃虚寒，下利清谷，或便纯白，肠滑不禁，少气羸困，全不思食。
加减：None
功效：None
用法用量：每服50丸，空心、食前米饮送下，次以温粥压之。
制备方法：上为细末，汤浸蒸饼为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：伏火硫黄法：先用硫黄五两，水飞去砂石，研为末，用瓷盒子盛，以水和赤石脂封口，盐泥固济，晒干；地内先埋一小罐子，盛水令满，置盒子在上，用泥固济慢火养七日七夜，候日足加顶火一煅，候冷取，研为细末。
