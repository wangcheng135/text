序号：75268
名称：蒲灰散
组成：蒲灰7分，滑石3分。
出处：《金匮》卷中。
主治：小便不利；厥而皮水者。
加减：None
功效：None
用法用量：每服方寸匕，饮调下，每日3次。
制备方法：上为散。
临床应用：None
各家论述：1.《金匮玉函经二注》：膀胱血病涩滞，致气不化而小便不利也。蒲灰、滑石者，本草谓其利小便，消瘀血。蒲灰治瘀血为君，滑石利窍为佐，皮水，用蒲黄消经络之滞，利小便为君；滑石开窍通水，通以佐之，小便利则水下行，逆气降。_x000D_
2.《金匮要略心典》：蒲，香蒲也，能去湿热，利小便，合滑石为清利小便之正法也。
用药禁忌：None
附注：本方方名，《张氏医通》引作“蒲黄散”。
