序号：68317
名称：黄耆散
组成：黄耆1两（锉），白术半两，续断半两，人参半两（去芦头），熟干地黄1两，茯神半两，附子3分（炮裂，去皮脐），当归半两（锉，微炒），肉桂3分（去皱皮），五味子半两，白芍药半两，赤石脂半两，陈橘皮半两（汤浸，去白瓤，焙），麦门冬1两（去心，焙），甘草1分（炙微赤，锉），干姜半两（炮裂，锉）。
出处：《圣惠》卷八十一。
主治：产后体虚乏力。四肢羸瘦，不思饮食。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，大枣3枚，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
