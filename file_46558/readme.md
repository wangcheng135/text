序号：46573
名称：千金定吼丸
组成：南星4两，半夏4两（用生姜、牙皂各3两，浸星、半1宿，切片，再加白矾2两入汤内，同星、半煮至汤干，去姜、皂不用，用南星、半夏），锦纹大黄（酒拌，9蒸9晒）1两，白附子1两，贝母1两（杵碎），枳实1两（麸炒），黄连1两（姜炒），连翘1两（去心），天麻1两，僵蚕（炒），瓜蒌1两，桔梗1两，沉香5钱，青礞石（用消煅如金色者）5钱，黄芩（酒炒）1两。
出处：《寿世保元》卷三。
主治：素患哮吼之疾，发则喘急，痰嗽上垂，不时举发。
加减：None
功效：None
用法用量：每服1丸，临卧口含化下；或丸如黍米大，姜汤送下亦可。
制备方法：上为细末，竹沥、姜汁和为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：慎劳役，戒厚味，节欲。
附注：None
