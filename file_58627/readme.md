名称：秘传阿魏万灵膏
组成：防风2两，荆芥2两，白芷2两，当归2两，黄连2两，黄柏2两，连翘2两，蛇蜕2两，蜂房2两，白蔹2两，苍耳草2两，接骨草2两，羌活2两，山栀2两，大风子2两，金银花2两，甘草2两，细辛2两，紫河车2两，何首乌2两，黑丑2两，桔梗2两，牡丹皮2两，车前子2两，苦参2两，白及2两，蓖麻子2两，大黄2两，川山甲酒10片，江子肉8钱，望见消2钱，木鳖子40个，虾蟆1两，柴胡1两，全蝎1两，半夏1两，升麻1两，南星1两，玄参1两，天花粉1两，川乌1两，牛膝1两，黄耆1两，两头尖1两，独活1两，斑蝥1两，地榆1两，五灵脂1两，槐角1两，苍术1两，藁本1两，赤茯苓1两，桃仁1两，三棱1两，莪术1两，小茴香1两，青木香1两，嫩松节1两，威灵仙1两半，天麻1两半，藕节1两半，薄荷1两半，贝母1两半，丹参1两半，生地黄1两半，乌药1两半，血余3钱（后入），八角风4两，叶下红4两，槐枝6两，柳枝大两，黄丹8两（水飞过，炒紫色）。
出处：《松崖医径》卷下。
主治：发背，瘰疬，疔肿，一切恶疮，瘫痪，痛风，脚气。
加减：None
功效：None
用法用量：每用油纸摊贴，留顶以出其毒。
制备方法：上细切，用水8碗，浸1日，煎稍干，下真麻油16斤，同煎至川山甲等药如炭黑，滤去滓；入血余煎无形影，滴水中不散，再入黄丹，徐徐顺搅，煎至滴水成珠，再入后项药：娱蚣2条，乌蛇肉4两，川乌、草乌、附子、白附子各1两，五加皮，紫荆皮各2两。上为细末，入膏药内，频频顺搅匀，退火入后项药：沉香、雄黄各1两，南木香、血竭、轻粉、赤石脂、龙骨各2两，乳香、没药各4两，麝香5钱阿魏1两（用水另溶化，再入膏药内）。上为细末，入膏药内，顺搅匀，出火毒。瓷器收贮。
