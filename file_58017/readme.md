序号：58032
名称：益母草子散
组成：益母草子1两，桂心半两，当归3分（锉，微炒），赤芍药半两，熟干地黄半两，大麦糵半两（微炒），鬼箭羽半两，红蓝花半两，川大黄3分（锉，微炒），赤鲤鱼皮灰半两，乱发灰半两，密陀僧半两（烧醋淬过），虻虫1两（去翅足，微炒），水蛭1两（炒令黄），麝香1分（研入）。
出处：《圣惠》卷七十九。
主治：产后恶血稽留，经久未消，致月水不通，面色萎黄，脐腹疼痛，肌瘦无力。
加减：None
功效：None
用法用量：每服2钱，食前以温酒调下。
制备方法：上为细散，以赤马尿半中盏，酒半中盏，拌和前药令匀，直候干，研入麝香。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
