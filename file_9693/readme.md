序号：9693
名称：牛膝丸
组成：牛膝1两（半）（去苗），柏子仁3分，桂心1两，白茯苓3分，白石英1两（细研，水飞过），黄蓍1两（锉），鹿茸1两（去毛，涂酥，炙令微黄），五味子3分，人参3分（去芦头），附子1两（炮裂，去皮脐），覆盆子1两，菟丝子1两（酒浸3日，曝干），山茱萸3分，芎藭3分，杜仲3分（去粗皮，炙令微黄，锉），熟干地黄3分，防风3分（去芦头），石斛1两（去根，锉），肉苁蓉1两（酒洗，去皱皮，微炙），磁石1两（烧，醋淬7遍，捣碎细研，水飞过），补骨脂1两（微炒）。
出处：《圣惠》卷七。
主治：肾脏虚损，骨痿无力，坐而难起，目视茫茫，短气不足，肌体羸瘦。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
