序号：33897
名称：没药除痛散
组成：蓬莪术（炮）1两，当归（焙）7钱半，玄胡索7钱半，五灵脂7钱半，肉桂（去粗皮）7钱半，良姜（炒）7钱半，蒲黄（炒）7钱半，甘草（炙）半两，没药半两。
出处：《女科百问》卷上。
主治：腹中坚痛，月经不调，脉紧涩滞者。
加减：None
功效：逐寒邪，调经。
用法用量：每服3钱，以温酒调下。
制备方法：上为细末。
临床应用：None
各家论述：《医略六书》：没药散瘀血以止痛，蓬术化瘀结以消坚，蒲黄破血瘀以通经，灵脂破瘀血以降浊，延胡索活血通经，炙甘草缓中除痛，肉桂温经暖血，良姜暖胃逐冷，当归养血脉以生新，而宿血自化也。为散以散之，温酒以行之，使瘀化寒消，则腹中坚痛自退；月经之至自无不调矣。
用药禁忌：None
附注：None
