序号：83803
名称：乌蛇丸
组成：独活（去芦头）1两，防风（去芦头）1两，吴白芷1两，人参（去芦头）1两，桂心1两，藁本（去土）1两，麻黄（去根节）1两，芍药1两，天麻（去芦头）1两，川乌头（去皮脐，捣碎，炒黄色）半两，藿香叶（去土）半两，乌蛇（酒浸，去皮骨）半两，全蝎（微炒去毒）半两，甘草（炙黄）半两，生犀（镑）半两，川芎7钱，羌活（去芦头）3钱，白僵蚕（微炒）3钱，远志（去心）3钱，牛黄（研）3钱，天南星（牛胆制，微炒）3钱，白附子4钱（炮裂），龙脑（研）2钱，朱砂（为衣）2钱。
出处：《御药院方》卷一。
主治：诸风疾，无问久新，半身不遂，手足麻木，精神不爽，咽嗌不利，及风虚头痛，目眩欲倒，心忪健忘，恍惚不宁，心气不得下通，脾气滞而不散，筋脉拘急，骨节烦疼。
加减：None
功效：None
用法用量：每服1丸，煎荆芥汤化下。或加至2丸，茶、酒化下亦得。
制备方法：上为细末，入研药令匀，炼蜜为丸，每1两作10丸，以朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：重身妇人不宜服之。
附注：None
