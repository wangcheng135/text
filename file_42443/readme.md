序号：42458
名称：青黛丸
组成：青黛1分（细研），牛黄1分（细研），朱砂半两（细研，水飞过），蜗牛1分（炒令黄），乌蛇1两（酒浸，去皮骨，炙令黄），干蝎2-7枚（微炒），胡黄连1分，白僵蚕1分（微炒），白附子1分（炮裂），麝香1钱（细研），狗胆2枚（取汁），蟾酥2片（如柳叶大，铁上焙焦）。
出处：《圣惠》卷八十五。
主治：小儿慢惊风，体热多涎，发歇搐搦。
加减：None
功效：None
用法用量：1-2岁儿，每服3丸，以薄荷汤送下，1日3次。3-4岁儿服5丸。
制备方法：上为末，入狗胆汁，与糯米饭为丸，如黄米粒大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
