序号：42944
名称：保真丸
组成：鹿角胶8两，鹿角霜（拌炒成珠）3两，白茯苓（人乳拌蒸）3两，山萸肉3两，北五味1两，熟地黄3两，川杜仲（盐水拌炒）3两，怀山药3两，益智仁1两，远志肉（甘草汤泡）1两，川楝子（酒煮取肉）1两，巴戟肉（酒炒）1两，沉香5钱（另为末），补骨脂1两，胡芦巴（2味同羊肾煮，晒干）。
出处：《女科指掌》卷二。
主治：None
加减：精薄者，加鳔胶6两。
功效：男子精亏不育。
用法用量：每服70丸，酒送下。
制备方法：上以肉苁蓉4两（酒洗去皮垢，切开心，有黄膜去之），酒煮烂捣，入药末，加炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中胡芦巴用量原缺。
