序号：76468
名称：干地黄丸
组成：生干地黄（微炒）1两1分，桃仁（汤浸，去皮尖双仁，麸炒黄）1两1分，芎1两，白芷1两，蒲黄1两，当归（微炙）3分，牛膝（酒浸，去苗，切，焙）3分，甘草（炙）3分，芍药3分，牡丹3分，干姜（炮裂）3分，人参3分，桂（去粗皮）3分，水蛭（以糯米少许同炒，米熟为度）30枚，虻虫（去翅足，微炒）30枚。
出处：《圣济总录》卷一五一。
主治：妇人月事欲下，腰腹刺痛，或多或少，或月内再来，或如清水，或似豉汁，心下坚满，沉重虚乏，日渐黄瘦。
加减：None
功效：None
用法用量：每服30丸，温酒送下；米饮亦得。
制备方法：上为末，炼蜜和丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
