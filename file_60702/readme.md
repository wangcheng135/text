序号：60717
名称：通络益气丹
组成：生山甲6两，豨签草1斤8两，木瓜1斤12两，苏地龙1斤2两，灵仙1斤8两，海风藤1斤12两，麻黄3两，橘红1斤2两，僵蚕1斤8两，薄荷1斤8两，生牡蛎3斤，生龙骨1斤8两，玄明粉1斤8两，黄柏12斤，川牛膝1斤2两，黑郁金12两，蒺藜1斤8两，生磁石1斤2两，生鳖甲4斤，夜交藤3斤，龙胆草1斤8两，生栀子1斤8两，车前子1斤1两，菟丝子4斤，蕲蛇肉4两5钱，紫贝齿1斤3两，乌药1斤3两，莲子心1斤1两，法半夏1斤8两，藿香1斤8两，砂仁1斤3两，生白芍1斤8两，黄豆卷12两，生赭石1斤3两，广寄生6斤，旋复花12两，全蝎12两，鸡血藤2斤4两，生石决明2斤，茯苓（去皮）2斤4两，青皮（醋炒）1斤2两（共为细粉，每细粉76斤5两5钱兑以下药物），琥珀面4两，朱砂面9两，瓜蒌3斤，胆星3斤，竹沥水1两5钱，生石决明6斤。
出处：《全国中药成药处方集》（天津方）。
主治：风湿内闭，痰涎壅盛，周身麻木，神志昏乱，肝热上冲，头晕头眩，身重脚轻。常服预防血压高。
加减：None
功效：除湿祛风，活络豁痰，镇静安神。
用法用量：每服4钱，白开水送下，日2次。
制备方法：将以上熬清膏和水打小丸。再用滑石4斤、生石膏8斤、知母4斤、花粉4斤、竺黄1斤8两、竹茹粉4斤研细为衣，装盒，每盒2两重。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
