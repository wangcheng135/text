序号：37708
名称：抵金膏
组成：花蕊石（火煅过，研如粉）2两，生硫黄（细研）2两，黄丹（细研）2两，牡蛎（火煅过，研如粉）2两，蚌粉（细研）2两，自然铜1两（火煅，醋淬，研细。上6味同研匀，用清油32两，同入银锅内，用炭火熬去油10两，次入后药），草乌4两（连皮尖，生用），骨碎补（去毛）2两，汉防己2两，龙骨2两，乌药2两，虎骨（如无，用败龟、五味各2两；上6味并为细末，入前油内熬成稠膏，次入后药）2两，乳香2两，没药2两，血竭2两，白胶香2两，安息香2两。
出处：《杨氏家藏方》卷十三。
主治：诸般痔漏，久不愈者。
加减：None
功效：None
用法用量：每服1小匙，温酒调下，不拘时候。
制备方法：后5味同为细末，入前膏内，急以杖子搅匀，少时取出，以瓷盒子盛，不得盖，收3日，候火力定。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
