序号：32952
名称：陈漆丸
组成：陈漆2升（以绵绞去滓），大黄6两（为末），薏苡仁5两（为末），无灰酒5升，蔓菁子3升（为末）。
出处：《圣济总录》卷九十三。
主治：传尸，飞尸，注气，癖块积气，上喘，水病，脚气，鬼注蛊毒，宿食不消，腹中如复杯，或九虫，妇人带下赤白，皮肤恶疮，腹大羸瘦，黄疸诸疾。
加减：None
功效：延年养性，黑须发。
用法用量：遇有患者，经宿勿食，明日清旦空心温酒送下10丸。年高或冷疾者，加至15丸，服之100日后，须发如漆色，有积年疮痕皆灭。初服4-7日内，泻出宿食或鱼粘脓血瘀恶物，勿疑。
制备方法：上先以清酒和蔓菁子末煎，不住手搅至半日许，滤去滓后，用银石器盛，重汤煮之，以竹篦子不住手搅一复时，后下陈漆、大黄、薏苡仁等末，更煮一复时，候药可丸，即丸如梧桐子大，置于不津器中，密封。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
