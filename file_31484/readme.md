序号：31493
名称：补肝细辛散
组成：细辛1分，桃仁3分（汤浸，去皮尖双仁，麸炒微黄），前胡3分（去芦头），当归3分（锉，微炒），附子3分（炮裂，去皮脐），陈橘皮3分（汤浸去白瓤，焙），人参3分（去芦头），柏子仁半分，芎3分，木香3分，白茯苓3分，吴茱萸半两（汤浸7遍，焙干微焙），桂心3分。
出处：《圣惠》卷三。
主治：肝脏虚寒，胸膈气滞，四肢厥逆，两胁疼痛。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，大枣3枚，同煎至6分，去滓温服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：细辛汤（《圣济总录》卷五十五）。
