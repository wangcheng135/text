序号：40004
名称：育婴金丹
组成：胆星1两，羌活1两，防风1两，麻黄1两，牙皂5钱，天麻1两5钱，薄荷2钱，全蝎18条，茯苓8钱，竺黄8钱，广皮8钱，礞石（制）1两，僵蚕30条，钩藤4两，琥珀3钱，冰片5钱，牛黄1钱，麝香5分。
出处：《全国中药成药处方集》（兰州方）。
主治：小儿伤风头痛，咳嗽气喘，咽喉肿痛，牙疼口干，鼻流清涕，或鼻塞不通，急惊风抽搐。
加减：None
功效：止咳化痰，镇惊安神，去风发汗。
用法用量：周岁以内服1/4；1-2岁服半丸；2岁以上服1丸，开水调服。
制备方法：上为细末，竹沥膏和蜜为丸，1钱重，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：忌刺激性食物。
附注：None
