序号：27529
名称：补水宁神汤
组成：熟地黄2钱，生地2钱，白芍药1钱半，当归1钱半，麦门冬（去心）1钱半，茯神1钱半，五味子30粒，甘草（用生）6分。
出处：《审视瑶函》卷五。
主治：神光自现症。
加减：None
功效：补肾水，宁心神。
用法用量：白水2钟，煎至8分，去滓，空心温服。
制备方法：上锉1剂。
临床应用：None
各家论述：肾水亏虚，真阴不足，故用熟地黄，乃天一生水之剂，大补真阴；生地黄有滋阴退热之效；麦门冬有清心降火之功；补血滋阴，须凭当归、白芍；神光荡漾，昼夜不宁，此神思间无形之火妄动故也，必用茯神与五味子，养精安神定志，能敛无精之气不走；细生甘草降神中之火，非此不能治。若然，则肾水上升，心火下降而神自宁，光亦可定矣。
用药禁忌：None
附注：None
