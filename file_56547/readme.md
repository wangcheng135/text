序号：56562
名称：大效使君槟榔丸
组成：肉豆蔻2个（炮），槟榔1个（生），宣黄连1钱，胡黄连1钱，陈皮1钱，青皮1钱，川楝子肉1钱（炒），芜荑1钱（炒，去皮），神曲1钱，麦芽1钱（并炒），木香1钱，夜明砂1钱（炒，去土），芦荟1钱，川芎1钱，麝1字。
出处：《活幼口议》卷十七。
主治：婴孩小儿食肉太早，伤及脾胃，水谷不分，积滞不化，疾作疳气等候。
加减：None
功效：None
用法用量：每服30-50丸，温饭饮送下。
制备方法：上为末，獖猪胆汁、薄荷为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方名大效使君槟榔丸，但方中无使君子，疑脱。《准绳·幼科》引作“使君槟榔丸”。
