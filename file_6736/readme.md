序号：6736
名称：加味调中健脾汤
组成：白术1钱，苍术1钱，厚朴1钱，陈皮1钱，茯苓1钱，半夏8分，枳实8分，人参7分，甘草5分。
出处：《济阳纲目》卷十二。
主治：None
加减：如头目眩痛，加川芎、白芷各1钱；如左胁气滞，加青皮、柴胡各1钱；如右胁痛，加枳壳1钱；如饮食肉物所伤，加神曲、麦芽、山楂各8分；如恶心呕哕，加藿香、砂仁各8分；如肺经有热，加黄芩7分；如小便涩，加栀子（炒）8分；如相火动，加黄柏、知母（炒）各8分；如大便涩，加黄芩、当归各1钱；如脾热，加芍药1钱。
功效：调养脾胃。
用法用量：加生姜3片，水煎，食远温服。
制备方法：上锉。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
