序号：33764
名称：没药丸
组成：没药1两，茺蔚子1两，干姜1两，苍术1两，川芎1两，熟干地黄1两，白芍药1两，当归1两，血竭半两。
出处：《产育宝庆》卷下。
主治：妇人月经不调，肌瘦发热，饮食减少。
加减：None
功效：None
用法用量：每服50-60丸，渐加至80-90丸，空心、食前以温酒或淡醋汤送下，日2次。
制备方法：除血竭、没药外，上（口父）咀，先炒茺蔚子焦黄色，次下干姜炒令黄，次下苍术微黄色，次下川芎等药，并令微黄，与血竭、没药等同为细末，醋煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中茺蔚子，《普济方》引作“香附子”，《御药院方》作“莨菪子”。
