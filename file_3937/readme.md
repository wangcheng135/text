序号：3937
名称：无价宝膏
组成：甘草1两，远志5钱，牛膝（去芦）5钱，肉苁蓉（去鳞）5钱，虎骨（酥炙）5钱，川续断（去芦）5钱，蛇床子（拣净）5钱，鹿茸（酥炙）5钱，天门冬（去心）5钱，生地黄5钱，熟地黄5钱，肉豆蔻（面煨）5钱，川楝子（炒黑色）5钱，麦门冬（去心）5钱，紫梢花5钱，木鳖子（去壳）5钱，杏仁（去皮尖）5钱，官桂（去皮）5钱，大附子（去皮）5钱，谷精草5钱，菟丝子5钱，金墨鴵鸰油5钱。
出处：《疡科选粹》卷八。
主治：高年阳痿。
加减：None
功效：补助真阳，返老还童。
用法用量：或縀皮摊贴，小腹用3个，5日1换，共9日。时常饮酒，引谷道肾经气通，再用钱大1个贴脐。
制备方法：上药用真香油1斤4两煎至黑色，去滓再煎至滴水成珠为度；下飞过黄丹8两，用柳条不住手频搅，不散为度；再下雄黄、龙骨、硫黄、赤石脂，再熬1次；又下乳香、没药、麝香、木香、阿芙蓉、海马2对、石燕子2对、沉香3钱、阳起石2钱，蟾酥2钱，丁香2钱，上为细末，方入膏内，搅匀出火，入瓷器盛之。
临床应用：None
各家论述：None
用药禁忌：None
附注：用法中雄黄、龙骨、硫黄、赤石脂、乳香、没药、麝香、木香、阿芙蓉等药用量原缺。
