序号：34092
名称：赤脚大仙种子丸
组成：全当归（酒洗）8两，肉苁蓉（酒洗、连蕊须）8两，绵杜仲8两，菟丝子（酒浸）8两，淫羊藿（酥炙）8两，潼蒺藜8两（盐水、童便、人乳分制），云茯苓（人乳蒸）8两，破故纸（盐水炒）8两，怀牛膝（盐水炒）8两，甘枸杞（青盐水炒）4两，猺桂心（不见火）2两，线鱼膘（牡蛎粉拌炒）2斤，大天雄（每重1两4-5钱者，面裹煨）2枚。
出处：《饲鹤亭集方》。
主治：None
加减：None
功效：补虚损，种子。
用法用量：每晨服100丸，淡盐汤送下，晚服百丸，陈酒送下，男妇不妨同服。附、桂2味，年逾五旬，方可用也。
制备方法：如法炮制，每药1斤，用炼蜜12两，开水4两为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
