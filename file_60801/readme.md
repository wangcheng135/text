序号：60816
名称：通瘀煎
组成：归尾3-5钱，山楂2钱，香附2钱，红花（新者，炒黄）2钱，乌药1-2钱，青皮1钱半，木香7分，泽泻1钱半。
出处：《景岳全书》卷五十一。
主治：妇人血滞血积，经脉不利，痛极拒按，及产后瘀血实痛，并男妇血逆、血厥等证。
加减：兼寒滞者，加肉桂1钱，或吴茱萸5分；血盛内热，血燥不行者，加炒栀子1-2钱；微热血虚者，加芍药2钱；血虚涩滞者，加牛膝；血瘀不行者，加桃仁30粒（去皮尖），或加苏木、玄胡索之类；瘀极而大便结燥者，加大黄1、2、3钱，或加芒消、蓬术亦可。
功效：None
用法用量：水2钟，煎取7分，加酒1-2小钟，食前服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
