序号：81871
名称：熟干地黄散
组成：熟干地黄1两，白芍药3分，柴胡1两，鳖甲2两（涂酥，炙令黄，去裙襕），当归3分（锉，炒微黄），苍术1两（锉，炒令黄），姜黄3分，琥珀3分（细研），羌活半两，芎3分，木香半两，厚朴3分（去粗皮，涂生姜汁，炙令香熟），桂心半两，陈橘皮3分（汤浸，去白瓤，焙），牛膝1两（去苗）。
出处：《圣惠》卷七十。
主治：妇人血风劳冷，气攻心腹疼痛，四肢不和，吃食减少，日渐羸瘦。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，煎至6分，去滓，稍热服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
