序号：65329
名称：大瓢李遗丸
组成：琥珀5钱，天麻1钱，真珠2钱，冰片2钱，朱砂2钱，胆星2钱，血竭2钱，僵蚕2钱，蝉蜕4钱，细辛4钱，川芎6钱，羌活6钱，防风6钱，远志5钱，茯神8钱，犀角3钱5分，菖蒲6钱5分，铁粉3钱6分，雄黄5钱6分，牛黄5钱6分，蛇含石（醋煅）4钱6分，白附子5钱2分，半夏4钱4分，麝香5钱5分，芦荟5钱5分，乌梢蛇5钱5分，牙皂1两，枫子霜8两，青礞石（煅）4两。
出处：《解围元薮》卷三。
主治：蛇皮、刺风、痒风、鸡爪、疙瘙、历节等风；并麻木冷痛，手足屈折痛痒，不知痿惫瘫痪，腐烂危笃等证。
加减：None
功效：None
用法用量：每服100丸，空心酒送下。如觉恍惚、困倦、麻木、萎厥者，以豨莶、苍耳、金银藤，炼膏服之。
制备方法：上为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
