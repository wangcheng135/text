序号：28469
名称：花苁蓉丸
组成：花苁蓉8分，泽泻4分，五味子4分，紫巴戟天4分（去心），地骨皮4分，磁石6分（研，水掏去赤汁，干之研入），人参6分，赤石脂6分（研入），韭子5分（熬），龙骨5分（研入），甘草5分（炙），牡丹皮5分，干地黄10分，禹余粮3分（研入），桑螵蛸30枚（炙），栝楼4分。
出处：《外台》卷十一引《古今录验》。
主治：消渴服铅丹丸得小便咸苦如常，后恐虚惫者。
加减：None
功效：None
用法用量：每服20丸，空腹以牛乳送下，日2次。
制备方法：上药治下筛，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌房劳、酢、海藻、菘莱、胡荽、芜荑。
附注：None
