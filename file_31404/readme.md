序号：31413
名称：沉香散
组成：沉香3分，枇杷叶3分（拭去毛，炙微黄），前胡1两（去芦头），半夏半两（汤洗7遍去滑），白术3分，诃黎勒皮1两，人参3分（去芦头），黄耆1两（锉），桂心半两，五味子半两，细辛半两，白茯苓1两，陈橘皮3分（汤浸，去白瓤，焙），甘草半两（炙微赤，锉）。
出处：《圣惠》卷三十。
主治：虚穷上气，脾胃气弱，胸膈多痰，食饮无味，神思昏闷，肢节烦疼，体虚乏力。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，大枣3枚，煎至6分，去滓，稍热服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
