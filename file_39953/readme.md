序号：39963
名称：肥肌方
组成：丁香半两，木香半两，肉豆蔻（去壳）半两（3味用面裹同入慢灰煨，令面熟为度取出，放冷），诃子（煨，去核）半两，使君子半两，芦荟（细研）1两。
出处：《普济方》卷三七九引《全婴方》。
主治：小儿羸瘦，脏腑怯弱，泄痢虚滑，乳食减少，引饮无度，心腹胀满。
加减：None
功效：None
用法用量：每服3-7丸，温米饮送下，乳食前服烂饭丸如黍米大，1岁20丸。一方黄羸指大溶入鸡子1个碎，和炒熟，令儿食尽，吞下上丸，并治休息疳泻如神，汤氏方一不用枣肉为丸，饭饮入脾也。
制备方法：上为细末。以枣肉为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
