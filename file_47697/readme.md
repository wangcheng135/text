序号：47712
名称：神仙玉粉丹
组成：精明舶上硫黄1斤（去砂石尽）。
出处：《幼幼新书》卷二十八引《张氏家传》。
主治：小儿、成人冷积暴泻。
加减：None
功效：驱除宿冷，补一切虚。
用法用量：每服10-15丸，空心米饮送下。
制备方法：上打碎，用豮猪肚7个，旋采桑根白皮3斤，寸锉；将猪肚1个净洗，只以疏黄实之，用麻线缝合，以水2斗先将桑根白皮1斤同煮一伏时，其余猪肚亦用慢火养之，不得令冷；候煮满一伏时，别以猪肚换之，又用白皮纳1斤同煮再一伏时；又换猪肚、桑白皮，过三伏时，不换白皮，只换猪肚，共煮七伏时，水耗以热汤添，不得用冷水，候满七伏时取出，用温水淘净，研至细，候烈日日中晒极热，再研，煮糯米粉糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
