序号：2508
名称：双丸子
组成：天麻（轻炙）、天南星（炮）、蚕蛾、生犀（末）、朱砂（另研）、羚羊角（末）、藿香叶、白檀香、蝎梢（须是锋全者）、乌蛇（酒浸，去皮骨，轻炙），零陵香1钱，天雄（尖）半两，麝香半两，牛黄1分，雄黄1钱，狐肝1具（水煮，薄切，焙干，另杵），乌鸦1只（去嘴爪肠肚，于瓦罐内烧为灰，另研，罗入诸药末内）。
出处：《博济》卷四。
主治：小儿瘫痪，一切风痰伤寒，小儿惊风等。
加减：None
功效：None
用法用量：每服2丸，大人白豆大，小儿绿豆大，薄荷汤送下，卒患并3服；瘫痪中风，用腻粉3大钱，水调，同药化下；小儿惊风，金银薄荷汤送下；妇人血风，并产前产后中风，手足弯曲，当归、红花酒送下；伤寒，每服3-5丸，豆淋酒送下。
制备方法：上药并拣择净，分两称足，依法修制，捣细，研令匀，炼蜜和，硬软得所，却于石上捶300下，用垍器盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
