序号：8548
名称：半夏汤
组成：秫米1升，治半夏5合。
出处：《灵枢》卷十。
主治：痰湿内阻，胃气不和之失眠，厥气客于五脏六腑，卫气不得入于阴，阴虚，目不瞑，;痰饮客于胆府，自汗不得眠;温病愈后，嗽稀痰而不咳，彻夜不寐。
加减：心脾亏虚加党参，心阴不足加麦芽，痰热扰心加黄连，胃不和加神曲。
功效：None
用法用量：以流水千里以外者8升，扬之万遍，取其清5升，煮之，炊以苇薪火，沸，置秫米1升，治半夏5合，徐炊令竭为1升半，去其滓，饮汁1小杯，日3次。稍益，以知为度。病新发者，覆杯则卧，汗出则已矣；久者，3饮而已矣。
制备方法：None
临床应用：不寐：笔者以半夏秫米汤加味治疗失眠收到满意效果。因药房不备秫米，遵吴鞠通意，用薏苡仁代之。方药组成：法半夏、苡仁各60g。加减：心脾亏虚加党参，心阴不足加麦芽，痰热扰心加黄连，胃不和加神曲。
各家论述：1.《温病条辨》：半夏逐痰饮和胃，秫米秉燥金之气而成，故能补阳明燥气之不及，而渗其饮，饮退则胃和，寐可立至。_x000D_
2.《古方选注》：今厥气客于脏腑，卫气独行于阳，阳跻气盛不得入于阴，阴虚目不瞑。用秫米汤者，以药石不能直入阳跻，故治胃以泄卫气也。半夏辛温，入胃经气分。秫，糯也，北地之膏梁茹粟也，甘酸入胃经血分。千里水扬之万遍，与甘澜水同义，取其轻扬，不助阴邪，炊以苇薪，武火也。火沸入药，仍徐炊令减，寓升降之法，升以半夏，从阳分通卫泄邪，降以秫米，入阴分通营补虚，阴阳通，卧立至，汗自出，故曰汗出则已矣。
用药禁忌：None
附注：本方方名，《景岳全书》引作“秫米半夏汤”、《兰台（车丸）范》引作“半夏秫米汤”。
