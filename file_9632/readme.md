序号：9632
名称：长春丹
组成：何首乌（用水浸，去粗皮，竹刀切片，赤白各2斤，黑豆拌蒸晒9次，为末，净）2斤，仙茅（竹刀刮去芦，用粳米泔浸去皮，黑豆拌蒸晒9次，净末）2斤，白茯苓（去皮为末，水飞，去筋，取沉底晒干，用粳米铺底，放上蒸3次，研净末）1斤，茅山苍术（米泔水浸，去粗皮，切片，老米拌蒸晒9次。1云：加桑椹汁1斤，拌苍术末尤妙，中年以后服极效）1斤，牛膝（去芦，酒浸1宿，同何首乌蒸3次，净末）1斤。
出处：《医便》卷一。
主治：None
加减：None
功效：补益肝肾，聪耳明目，却病延寿。
用法用量：每服100丸，空心滚白汤送下。
制备方法：上药各为末，和匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌牛肉、萝卜、葱、蒜。
附注：None
