序号：69756
名称：琥珀丸
组成：琥珀（别研）1两，木香1两，禹余粮（煅，醋淬）1两，白术1两，芍药1两，鳖甲（去裙襕，酒浸，炙令香）1两，桂（去粗皮）1两，附子（炮裂，去皮脐）1两，羌活（去芦头）1两，蓬莪术（炮，锉）1两，细辛（去苗叶）1两，牡丹（去心）1两，肉豆蔻（去壳）1两，人参1两，京三棱（炮，锉）1两，黄耆（锉）1两，当归（微焙）1两半，槟榔（锉）1两半，枳壳（去瓤，麸炒）1两半，柴胡（去苗）2两，芎2两，桃仁（汤浸，去皮尖双仁，炒黄色）2两，安息香半两（研）。
出处：《圣济总录》卷一五一。
主治：妇人虚冷，月水凝涩不利，腹内疼痛，四肢烦热，皮肤瘾疹，饮食减少。
加减：None
功效：None
用法用量：每服20丸，空心温酒送下。
制备方法：上为末，以生地黄自然汁1碗，与药末同拌，次用酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
