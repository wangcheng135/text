序号：78914
名称：酿猪肚
组成：饙猪肚1枚（净洗，去脂），杏仁1两（去皮尖，研），人参1两（去芦头），白茯苓1两，陈橘皮半两（汤浸，去白瓤，焙），干姜1分（炮裂），芜荑1分，汉椒1分（去目及闭口者，微炒去汗），莳萝1分，胡椒1分，黄牛酥1两，大枣21枚（去核，切），糯米5合（淘，看肚大小临时加减）。
出处：《圣惠》卷九十七。
主治：五劳七伤，羸瘦虚乏。
加减：None
功效：None
用法用量：每用药1两，入酥、枣、杏仁、米等分，相和令匀，入猪肚内，以麻线缝合，即于甑内蒸令熟，切作片。空心渐渐食之。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
