序号：53124
名称：大补枳术丸
组成：白术（去芦，炒）1两，陈皮（去白）1两，枳实（麸炒）1两，黄连（姜汁炒）5钱，黄芩（醋炒）5钱，黄柏（青盐水炒）1两，白茯苓（去皮）5钱，贝母（去心）8钱，神曲（炒）5钱，山楂（去核）5钱，麦芽（炒）5钱，加砂仁3钱，香附（醋炒）3钱。
出处：《寿世保元》卷二。
主治：人禀素弱，脾胃虚怯，上焦有火、有痰、有郁气、有食积，胸中不快，饮食少思。
加减：None
功效：None
用法用量：每服100丸，食后姜汤送下；有热，茶汤送下。
制备方法：上为细末，荷叶汤下粳术煮稀粥，同药捣和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
