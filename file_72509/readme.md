序号：72525
名称：温里退邪汤
组成：赤色鲜首乌8钱（打碎），当归5钱，醋炙鳖甲5钱，茯神5钱，半夏2钱（制），白芥子3钱，青皮1钱5分，柴胡5分，草蔻仁1钱（研）。
出处：《慈航集》卷下。
主治：足少阴肾经之疟，初病寒热俱盛，脊强腰痛，口渴，寒从下起，先脚冷，后腿冷至脐，脐冷至手而止，头以上不冷，呕吐恶心，状似寒证。
加减：如气虚体弱者，加人参1钱5分（或以上党参1两代之）；如寒多，加鹿角霜3钱；如热多，加青蒿3钱；如夜发者，加制附子1钱，炒升麻8分。
功效：None
用法用量：加煨姜3钱，大枣3个为引，用河井水煎药。加好酒1钟对服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
