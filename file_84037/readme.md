序号：84055
名称：五加皮酒
组成：五加皮1斤，枳刺2升（[口父]咀，炙），猪椒根皮8两，丹参8两，桂心3两，当归3两，甘草（炙）3两，天雄（炮）4两，秦椒（汗）4两，白鲜皮4两，通草4两，芎5两，干姜5两，薏苡仁半升，大麻仁3升（研为末）。
出处：《外台》卷十六引《删繁方》。
主治：筋痹，好悲思，颜色苍白，四肢嘘嗡，脚手拘挛，伸动缩急，腹中转痛。
加减：None
功效：None
用法用量：初服6-7合。稍稍加之，以知为度。
制备方法：上(口父)咀，以绢袋贮，酒4斗渍，春、夏4宿，秋、冬6-7宿。
临床应用：None
各家论述：None
用药禁忌：忌生葱、猪肉、冷水、海藻、菘菜。
附注：五加酒（《千金》卷十一）。
