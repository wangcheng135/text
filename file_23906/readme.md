序号：23914
名称：朱砂煎
组成：朱砂1两（细研），白蜜半斤，黄丹1两。
出处：《圣惠》卷三十二。
主治：胎赤，兼生翳膜疼痛等。
加减：None
功效：None
用法用量：以铜箸头粘药如绿豆大点之，每1复时只得1度。
制备方法：上药相和令匀，入有油瓷瓶内，用柳木楔子紧塞瓶口，又以生布一片，油单两重，密裹瓶口，勿令透气，便安瓶于大鼎内，座用一杖子横着鼎口，以绳子系瓶口悬之，用水常令至瓶项，以文火煮，如鼎内水耗，旋旋添汤，勿入冷水，从寅时煮至酉时，住火候冷，取出，以新绵滤过，用白龙脑1钱研入，以一新瓷瓶盛之，常令封闭，候三日外用之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
