序号：62856
名称：清地散花饮
组成：防风1钱，山楂1钱，当归1钱，地丁1钱，大力子8分，山慈姑8分，荆芥7分，前胡5分，青皮5分，蝉蜕3分，红花2分，茯神2分，水通草2分，生姜皮2分，大枣1枚。
出处：《慈幼新书》卷六。
主治：痘见标3日。
加减：夹疹，加蓬心9个、菊花5分、兰花根1钱；夹丹，加黄芩、生地、薄荷、山栀、小竹叶或黄紫草；腰腹痛，倍楂、青，加杜仲、胡桃肉；小腹痛，加厚朴、笋兜、青木香、延胡索；额上灰滞色，加菊花、梅花、兰花或用根；准头先标，加木香、笋兜，外以清肌汤裕之；色娇亮者，湿胜也，倍防风，少加大腹皮。
功效：清脚地，肃脏腑，理气活血，引毒达于四肢。
