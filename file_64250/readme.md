序号：64266
名称：大腹汤
组成：大腹皮（切）1两，槟榔（锉）1两，木通（锉）1两，防己1两，青橘皮（汤浸，去白，焙）1两，紫苏茎叶1两，桑根白皮（锉）1两，甘草（炙，锉）1两，枳壳（去瓤，麸炒）1两，草豆蔻（去皮）半两，丁香皮（锉）半两，大黄（锉，炒）半两，木香1分。
出处：《圣济总录》卷六十二。
主治：诸膈气，冷热不调，喜怒无度，胸中咽塞，不思饮食；或忧思过甚，不足之气蕴积心臆，日渐消瘦。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，加生姜2片，大枣1枚（擘），同煎7分，去滓温服，日3夜1。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：大腹皮汤（《普济方》卷二○四）。
