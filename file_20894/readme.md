序号：20902
名称：人参散
组成：人参1两（去芦头），柴胡1两半（去苗），白茯苓1两，厚朴2两（去粗皮，涂生姜汁，炙令香熟），白术1两，桔梗1两（去芦头），陈橘皮2两（汤浸，去白瓤，焙），五味子1两，黄耆1两（锉），当归1两（锉，微炒），槟榔1两，甘草半两（炙微赤，锉），桂心3分，半夏1两（汤洗7遍，去滑）。
出处：《圣惠》卷五。
主治：脾胃气久弱，肌体羸瘦；或加劳气，大肠不调，有时痰逆，不思饮食，四肢少力。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，大枣3枚，煎至6分，去滓，不拘时候温服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
