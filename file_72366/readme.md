序号：72382
名称：紫双丸
组成：巴豆18铢，麦门冬10铢，甘草5铢，甘遂2铢，朱砂2铢，蜡10铢，蕤核仁18铢，牡蛎8铢。
出处：《千金》卷五。
主治：小儿身热头痛，食饮不消，腹中胀满，或小腹绞痛，大小便不利，或重下数起。
加减：None
功效：None
用法用量：半岁儿服如荏子1双，1-2岁儿服如半麻子1双，3-4岁者服如麻子2丸，5-6岁者服如大麻子2丸，7-8岁儿服如小豆2丸，9-10岁，微大于小豆2丸，常以鸡鸣时服，至日出时不下者，热粥饮数合即下，丸皆双出也。下甚者，饮以冷粥即止。
制备方法：上以汤熟洗巴豆，研，新布绞去油，别捣甘草、甘遂、牡蛎、麦门冬，下筛讫。研蕤核仁令极熟，乃纳散更捣二千杵，药燥不能相丸，更入少蜜足之。
临床应用：None
各家论述：《千金方衍义》：小儿无异疾，惟饮食过度，故宜巴豆之辛散，兼甘遂之苦寒，以荡涤癖积；蕤仁除心腹邪热结气，麦冬治肠中伤饱，胃络脉绝，羸瘦短气，牡蛎散内结积热，蜂蜡清胃，甘草和中，丹砂安神，不使巴豆、甘遂侵犯正气也。
用药禁忌：None
附注：双丸（《活人书》卷二十）、双圣丸（《医学纲目》卷三十八）。
