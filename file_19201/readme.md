序号：19208
名称：甘露汤
组成：干饧糟6分（头酿者），生姜4分（和皮）。
出处：《普济方》卷三十六引《澹寮方》。
主治：翻胃呕吐不止，饮食减少。
加减：None
功效：快利胸膈，调养脾胃，快进饮食。
用法用量：每服入盐少许，沸汤调下，不拘时候。
制备方法：上拌和，研烂作饼子，或焙或晒干，每10两用甘草2两炙，同研为末。
临床应用：翻胃：常传一富人病翻胃，往京口甘露寺，设水陆，泊舟岸下，梦一僧持汤一杯与之。饮罢犹记其香昧，便觉胸隔少快。早入寺，知客供汤，乃是梦中所饮者，胸膈尤快。遂求其方，修制数10服后，疾遂愈。
各家论述：None
用药禁忌：None
附注：观音应梦散（原书同卷）、甘露饮（《寿亲养老》卷二）。
