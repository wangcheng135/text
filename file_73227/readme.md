序号：73243
名称：斑龙二至百补丸
组成：鹿角50两（新取连脑骨者佳，锯作2寸长段，长流水洗，米泔浸1宿，刷洗净，吹晒干，同后药和入瓷坛煮胶），黄精8两，枸杞子4两（甘州者），怀熟地黄4两，菟丝子4两（热水淘净），金樱子4两（去毛子净），天门冬（去心）2两，麦门冬（去心）2两，川牛膝2两（酒洗），龙眼肉1两，楮实子2两（热水洗）以上10味同角和匀，入净好金华坛内，层层放实，用新汲淡水注坛中平肩，以密棱布4层封口，以新砖压之，置大锅中井字架上，以木甑盖好，重汤煮3日夜，毋得间断火候。旁用小锅烧滚水，不时添注坛内并锅内，勿使干涸。日足，取起，滤去滓，将汁用罗底绢绞出，入净砂锅内，文火熬成膏，约1斤半。再炼蜜2斤（滴水成珠）掺入，调和后项药，杵烂为丸。鹿角霜10两，人参5两，黄耆（蜜炒）4两，鸡头粉4两，白茯苓（去皮）4两，怀山药4两（炒），山茱萸肉（连核者1斤，盐水洗过，取肉）4两，怀生地黄4两（酒洗，掐断，绢包，饭上蒸过），知母4两（盐水炒），五味子1两（去梗），夏月加川黄柏4两（炒褐色）。
出处：《医统》卷四十八。
主治：老年精血亏损，元阳虚惫，腰膝痠软，畏寒足冷，夜溺频多。肾虚腰痛，阳痿梦泄，精神衰弱，元气亏虚。
加减：None
功效：固本保元，生精养血；培复天真，大补虚损。益五内而除骨蒸，壮元阳而多子嗣。充血脉，强健筋骸；美颜色，增延龄算。聪明耳目，玄润髭须。
用法用量：每服80丸，空心时淡盐汤送下。随用煮熟莲子肉或晒干枣数枚以压之，俾纳丹田也。
制备方法：上为细末，用前膏和匀为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：斑龙二至丸（《中药成方配本》）、斑龙百补丸（《全国中药成药处方集》青岛方）。
