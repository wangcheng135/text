序号：35925
名称：参苏温肺饮
组成：人参1钱半，苏叶1钱半，肉桂1钱半（去皮），茯苓1钱半，白术1钱半（炒），五味8分，半夏1钱半（制），陈皮1钱半，甘草5分。
出处：《医略六书》卷二十二。
主治：肺虚寒滞喘促，脉紧细。
加减：None
功效：补肺散寒。
用法用量：水煎，去滓温服。
制备方法：None
临床应用：None
各家论述：寒束肺虚，阻遏肺气，失其升降之权，故呼吸不利，喘促不宁焉。人参扶元气补肺；苏叶疏肺络散邪；肉桂温营暖血，专祛寒邪遏伏；半夏燥湿化痰，兼理脾胃真元；白术培肺母以生金；五味收耗散以保肺；茯苓渗湿，理痰生之源；陈皮利气，杜生痰之因；更以甘草缓中和药也。
用药禁忌：None
附注：None
