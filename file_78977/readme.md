序号：78993
名称：增损理中丸
组成：人参2两，白术2两，甘草2两（炙），干姜6分（炮），枳实4枚，茯苓2两。
出处：《外台》卷三引《崔氏方》。
主治：时行4-5日，大下后或不下，皆心中结满，两胁痞塞，胸中气急，厥逆欲绝，心胸高起，手不得近。
加减：渴者，当加栝楼根2两，不渴除之；下者，当加牡蛎（熬）2两，不下勿用。
功效：None
用法用量：每服1丸，熟水送下，不歇复服。
制备方法：上为末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：忌海藻、菘菜、酢物、桃李、雀肉。
附注：枳实理中丸（《活人书》卷十八）、加减理中丸（《圣济总录》卷二十五）。本方改作汤剂，名“枳实理中汤”。（见《易简方》）
