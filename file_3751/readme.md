序号：3751
名称：开结枳实丸
组成：枳实（麸炒）半两，白术半两，半夏（汤洗）半两，天南星（炮）半两，白矾（枯）半两，苦葶苈（隔纸炒）半两，大黄半两，木香2钱，黑牵牛（头末）2两，大皂角（去皮子，酥炙）1两，青皮（去白）半两，或加旋复花1两。
出处：《御药院方》卷五。
主治：中痞痰逆，恶心呕哕，膈实，酒酲不解，宿物停积，两胁膨闷，咽嗌不利，上气咳嗽。黄疸。
加减：None
功效：宣导凝滞，消化痰饮，升降滞气，通行三焦，滋荣心肺，灌溉肾肝，补助脾元，养胃，转行百脉，去风结恶气，流畅大小肠。常服消食快气，下痰利膈。
用法用量：如单腹胀，上喘涎多，四肢肿满，每服30-40丸，食后生姜汤送下，以微利为度；妇人干血气，膈实肿满，或产后有伤，面目浮肿，小便不利，生姜、葱白汤送下；酒疸病，温酒送下。
制备方法：上为末，入牵牛头末令匀，生姜汁煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：木香利膈九(《医统》卷二十九引《医林》)、开结枳术丸(《医学入门》卷七)。
