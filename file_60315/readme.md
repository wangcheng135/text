序号：60330
名称：通水至奇汤
组成：人参3钱，莲子3钱，白果20个，茯苓3钱，甘草1钱，车前子3钱，肉桂3分，王不留行3钱。
出处：《石室秘录》卷三。
主治：膀胱气化不行，小便不通。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：此方之奇妙，全在用人参，其次则用肉桂3分，盖膀胱必得气化而始出。气化者何？心包络之气也，膀胱必得心包络之气下行，而水路能出。尤妙用白果20个，人多不识此意：白果通任、督之脉，又走膀胱，引参、桂之气，直奔于膀胱之中。而车前子、王不留行尽是通泄之物，各随之趋出于阴气之口也。
用药禁忌：None
