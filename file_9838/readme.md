序号：9838
名称：二贤散
组成：橘皮（去白取红）1斤，甘草4两，盐4两。
出处：《百一》卷五，名见《医学纲目》卷二十五。
主治：痰饮内停，食后胸满，翻胃，噎膈，肝气痛。膈中痰饮。肝气痛，常服除根。脾家冷积，每食已辄胸满不下，百药不效者，兼治一切痰气。翻胃及痛。
加减：None
功效：消积块，进饮食。清肺，消痰，定嗽，解酒毒。
用法用量：点服。
制备方法：水5碗，慢火煮，焙干，捣为细末。
临床应用：食后胸满外舅莫强中服之，腹痛，利下物数块如铁弹子，臭不可闻，旧苦食后胸满之疾豁然顿愈。
各家论述：《医方集解》：此足太阴、阳明药也。陈皮燥湿而利气，湿去则痰涸，气顺则痰行。食盐润下而软坚，润下则痰降，软坚则痰消。痰在膈中，故用甘草引入胃，甘草经蜜炙，能健脾调胃，脾胃健则痰自行矣。
用药禁忌：虚弱人慎用。
附注：二贤汤（《普济方》卷一六五）、三圣散（《医灯续焰》卷十二）、涤痰散（《回春》卷二）。本方改为丸剂，名“润下丸”（见《医方集解》）。
