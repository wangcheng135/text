序号：48134
名称：神应丹
组成：麻黄10斤（洗净，去土，捣烂，用河水4担浸1宿，砂锅熬至1担，去滓），贝母（炒，去心）2两，桑白皮（去土）2两，紫苏子2两，款冬花（去枝梗）2两，桔梗2两，知母2两，栝楼大者1个，皂角2挺（去皮弦子）。以上8味捣烂，入前麻黄汁内，熬至1半，去滓，澄取清汁，再熬成稠膏。白茯苓2两（去皮），紫菀（洗去土）1两（细末），天麻1两（细末），人参（去芦头）1两（为细末），阿胶（杵碎，炒为末）1两（入药拌匀）。
出处：《御药院方》卷五。
主治：诸远年喘咳。
加减：None
功效：None
用法用量：每服1丸，温齑汁化开，临卧时饮之，便去枕仰卧，不许久坐，如不得睡，乃药之效也。
制备方法：上以药末搜和前膏，杵熟为丸，如小弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
