序号：28753
名称：十子奇方
组成：凤仙花子3两（井水浸1宿，新瓦焙干），金樱子（竹刀切开，去毛子，水淘净，春碎熬膏）3两，五味子3两（酒浸，蒸，晒干），石莲子（研碎，用茯苓、麦冬各1两，煎汁拌蒸，晒干，净）3两，菟丝子3两（酒浸3宿，煮1昼夜，吐丝为度），女贞子3两（酒浸，9蒸九9晒），枸杞子4两（一半乳拌蒸，一半酒浸微炒），小茴香1两（微炒为末，白菊花2两，煎汁拌，晒干），桑子4两（极黑肥大者取汁，以瓷盆盛之，每日晒，成膏），大附子1个（重1两，蜜煮一日，换水煮半日，人参2两煎汁拌附子，晒干，附子须切片）。
出处：《惠直堂方》卷一。
主治：男子9种不育，四般精泄。
加减：None
功效：健脾壮筋，清痰理气。
用法用量：每空心服1钱5分，临卧服2钱。
制备方法：金樱子、菟丝、桑堪3味为膏，入诸药末，用淮山药4两，煮糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
