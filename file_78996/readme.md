序号：79012
名称：噙化上清丸
组成：五倍子（打碎，去内末净）1斤，水白酒曲2两。
出处：《鲁府禁方》卷四。
主治：None
加减：None
功效：香口生津，止痰清热，宁嗽、清头目。
用法用量：任意噙化。加片脑尤妙。如无梨汁，用柿霜白汤和之亦可。
制备方法：上为细末，合一处令匀，将细茶煎卤，冷和为糊，如烙饼样放瓷盆内，上用瓷拌盖严，放木桶内，上下周围俱铺穰草，口间上用草拍盖住，次日验看发动作热，用棍动仍旧盖住，看盖上有水擦净，如此1日2次，看搅擦水，至14日尝之，其味凉甜为止，后加薄荷3两，白硼砂2两，砂仁（焙），甘松（焙），玄明粉各5钱（为末），与前药一处，用梨汁熬膏，捣和为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
