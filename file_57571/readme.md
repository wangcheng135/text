序号：57586
名称：润涸汤
组成：熟地2两，白术1两，巴戟天1两。
出处：《辨证录》卷六。
主治：阴已痿弱，肾水燥，见色不举，若勉强入房，耗竭其精，则大小便牵痛，数至圊而不得便，愈便则愈痛，愈痛则愈便。
加减：None
功效：大补肾水，兼补肾火。
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：此方用熟地以滋肾中之真阴，巴戟天以补肾中之真阳，虽补阳而仍是补阴之剂，则阳生而阴长，不至有强阳之害，二者补肾内之水火，而不为之通达于其间，则肾气未必遽入于大小之肠也。加入白术以利其腰脐之气，则前后二阴，无不通达，何至有干燥之苦，数圊而不得便哉。
用药禁忌：None
附注：None
