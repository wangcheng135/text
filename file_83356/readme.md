序号：83374
名称：乌鸡饮
组成：雌乌鸡1只（去毛羽爪肚），鳖甲1两（涂醋，炙令黄，去裙襕），桃仁1两（汤浸，去皮尖双仁，麸炒微黄），川大黄3分（锉碎，醋拌，炒干），吴茱萸1分（汤浸7遍，焙干，微炒），桂心1两，鬼箭1两，牛膝1两（去苗），当归1两（锉，微炒），菴（艹闾）子1两，甘草（微炙）半两，芒消半两。
出处：《圣济总录》卷一六一。
主治：产后余血不尽，结聚成块，坚硬疼痛，腹肋胀满。
加减：None
功效：None
用法用量：上药除鸡外，为粗末和匀，以水4升，将鸡全煮，取汁，以瓷器澄令清。每服2钱匕，鸡清汁1盏，煎至7分，去滓，不拘时温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
