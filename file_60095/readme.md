序号：60110
名称：透风气散
组成：细辛1分，藿香1分，干蝎1分，羌活1分，白花蛇（酒浸1昼夜，非时用酒慢火上炙黄熟，去骨不用，只用肉）1分，独活1分，附子（炮，去皮）1分，天麻1分，牛膝1分，海桐皮1分，官桂（去皮）1分，豆蔻1分，半夏4铢，麝香1铢，麻黄（去节）半两，僵蚕3铢（去丝）。
出处：《普济方》卷一○三引《护命》。
主治：寒风所中，闷倒不知人事；中风时，涎少或无涎。
加减：None
功效：None
用法用量：每服4钱，空心浓煎姜汤调下，盖衣被，汗出为妙；若无汗，只闻得病处如虫行亦佳；若常服，每服1钱，薄姜汤调下，候风气已退，觉上焦有热，即宜吃调顺正气方。
制备方法：上为细末，入麝香同研令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：寒风所中，闷倒不知人事，先吃分涎散取涎了，却非时进此方。中风时涎少或无涎，只进此方亦得。
