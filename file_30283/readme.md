序号：30291
名称：阿魏膏
组成：羌活5钱，独活5钱，玄参5钱，官桂5钱，赤芍药5钱，川山甲5钱，生地黄5钱，两头尖5钱，大黄5钱，白芷5钱，天麻5钱，槐枝2钱，柳枝2钱，桃枝2钱，红花4钱，木鳖子20枚（去壳），乱发如鸡子大1块。
出处：《内科摘要》卷下。
主治：一切痞块。
加减：None
功效：None
用法用量：摊贴患处，内服丸药。黄丹须用真正者效。凡贴膏药，先用朴消随患处铺半指厚，以纸盖，用热熨斗熨，良久，如消耗，再加，熨之2时许，方贴膏药；若是肝积，加芦荟末同熨。更服胡黄连丸。
制备方法：上用香油二斤四两，煎黑去滓，入发煎，发化仍去滓，徐下黄丹煎，软硬得中，入芒消、阿魏、苏合油、乳香、没药各五钱，麝香三钱，调匀即成膏矣。
临床应用：None
各家论述：None
用药禁忌：None
附注：阿魏五香膏（《医级》卷九）。
