序号：79500
名称：碧雪
组成：川升麻2两，黄芩5两，钩藤5两，犀角屑5两，大青5两，青黛2两，虎睛1对，天竹黄半两，麝香1分，龙脑1分，川朴消1斤，竹沥3合。
出处：《圣惠》卷九十五。
主治：心热惊狂，诸痫热病。
加减：None
功效：None
用法用量：每服2钱，以金银汤调下，食后并夜临卧时服。
制备方法：虎睛、天竹黄、麝香，龙脑、青黛别研细入，余药并细锉，用水1斗，煎至3升，滤去滓，澄清，下朴消，微火更煎，以柳木篦搅，勿住手，候消散，下竹沥并研了药，更搅令匀，候稍凝，即于新瓦盆中盛，经宿即凝，捣罗为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：碧雪散（《普济方》卷三七八）。
