序号：39828
名称：三建登仙酒
组成：牛膝（去芦、酒浸）半两，当归（去芦，酒浸）半两，天麻（生）半两，杜仲（去皮,锉，姜制，炒去丝）半两，独活半两，薏米（略炒）半两，防风（去芦）半两，人参（去芦）半两，白术（麸炒）半两，川椒（去目、合口者，炒出汗）半两，木香（不见火）半两，肉苁蓉半两，熟干地黄半两，萆薢半两，羌活半两，附子（生，去皮脐）半两，肉桂（去粗皮，不见火）半两，川乌头（生，去皮脐）2钱半，白茯苓（去皮）2钱半，干木瓜2钱半，茴香（淘去沙，炒）2钱半，破故纸（炒）2钱半。
出处：《魏氏家藏方》卷八。
主治：中风瘫痪，及脚膝软弱，不能行步，顽麻疼痹；及老人、虚人、产妇一切脚气。
加减：None
功效：None
用法用量：上(口父)咀，以好酒5升（每升3盏）浸，春5日，夏3日，秋、冬7日，每取酒1盏，汤烫令热，空心、日午、临卧服，使酒气熏熏相续，如能饮两盏亦不妨。不能饮者可作5-6日饮尽，则病自除。如未效，更1料，其病可除矣。
