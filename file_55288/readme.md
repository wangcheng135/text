序号：55303
名称：大金箔丸
组成：金箔100片，银箔100片，辰砂1两，牛黄1钱，生犀1钱，丁香1钱，沉香1钱，真珠1钱，木香1钱，脑麝1钱，琥珀1钱，硼砂1钱，乌蛇肉（酒炙）1钱，天麻（酒炙）1钱，雄黄1钱，蝎梢1钱，白僵蚕1钱，附子（炮）1钱，天南星（炮）1钱，防风1钱，白附子1钱，甘草（炙）1钱，香墨半两。
出处：《幼幼新书》卷十一引《灵苑方》。
主治：一切风及痫。大人、小儿癫痫，无时发动，口吐涎沫，项背强直，神志昏愦。
加减：None
功效：解心胸壅热，消痰坠涎。
用法用量：每服大人5丸，薄荷酒送下；小儿3丸，薄荷汤送下。
制备方法：金银箔同水银3分研如泥；余药为细末，再同研，炼蜜为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
