序号：82888
名称：丹砂丸
组成：丹砂（研）1分，牛黄（研）1分，阿魏（研）1分，龙脑（研）1分，水银1分，天麻1分，防风（去叉）1分，芎1分，细辛（去苗叶）1分，羌活（去芦头）1分，麝香（研）2分，乌鸡血1两，乌蛇（酒浸，去皮骨，炙）2两。
出处：《圣济总录》卷六。
主治：中急风，昏乱不识人。
加减：None
功效：None
用法用量：每服10丸，热酒送下。汗出即愈。
制备方法：上药各为末，先以酒2升，煎蛇末令浓，和前药，又以盐、米醋、白矾末合研匀，候色光白，入铫内煮令热，下丹砂同煎，结成砂子，研和前药为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
