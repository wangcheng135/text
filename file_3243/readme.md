序号：3243
名称：九分散
组成：马钱子4两（去皮毛），麻黄4两（去节），乳香4两（去油），没药4两（去油）。
出处：《急救应验良方》（善化堂本）。
主治：跌打损伤，坠车落马，伤筋动骨，青肿疼痛。
加减：None
功效：活血止痛。
用法用量：遇有受伤人，即与准9分服下，以无灰老酒调。药力甚大，服时万不可过9分。外伤处破者，干上；若未破，只见青肿，用烧酒调涂。服药后，如觉胸中发闹，周身发麻，此是药力行动，勿恐。若受伤甚重，服后不见动静，过1个半时辰，再用无灰老酒调服九分。如此敷服，无论何样重伤，皆能起死回生，真破伤第1方也。
制备方法：上各为细末，再合研极细，收瓷瓶内，勿令泄气。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：本方方名，原书啸园本作“九厘散”，每服不过九厘。
