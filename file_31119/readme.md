序号：31128
名称：攻坚败毒膏
组成：当归1两，熟地1两，生地1两，白芍药1两，赤芍药1两，南星1两，半夏1两，三棱1两，蓬术1两，木鳖1两，两头尖1两，川山甲1两，巴豆仁1两，肉桂1两，五灵脂1两，桃仁1两，续断1两，玄参1两，玄胡索1两，萆麻子仁1两，白芷1两，羌活1两，独活1两，大黄1两，红花1两，川乌1两，草乌1两，苏木1两，川芎1两，防风1两，杏仁1两。
出处：《景岳全书》卷六十四。
主治：痞块、疮毒、痔漏。
加减：None
功效：None
用法用量：用麻油40两，浸诸药3日，桑柴火煎成丹，收后下细药：乳香（制）、没药（制）各1两，真阿魏1两半，麝香3钱。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：上方于细药中加芦荟、木香各一两蟾酥三钱，即名“消痞大成膏”。
