序号：57863
名称：珠黄保生丹
组成：僵蚕（炒）4两，防风4两，花粉4两，钩藤4两，天麻4两，竺黄4两，玄参（去芦）4两，黄芩4两，栀子（炒）4两，桔梗4两，胆星1两5钱，生石膏1两5钱，羌活1两5钱，薄荷1两5钱，牙皂1两，全蝎5钱，甘草4两（前诸药共为细末，每51两5钱细末兑），朱砂1两，冰片1两，羚羊粉1两，犀角粉5钱，牛黄5钱，珍珠（豆腐炙）3钱，麝香3钱（共研细末）。
出处：《北京市中药成方选集》。
主治：小儿内热火盛，头痛身烧，咳嗽痰喘，惊风抽搐。
加减：None
功效：清热退烧，祛风化痰。
用法用量：每服1丸，温开水送下，1日2次，周岁小儿酌减。
制备方法：上药混合均匀，炼蜜为丸，重5分，金衣36片，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《全国中药成药处方集》（北京方）引作“珠黄宝生丹”。
