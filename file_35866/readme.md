名称：参术遇仙丹
组成：人参1两，白术1两，川芎2两，皂角刺2两，藁本2两，蝉壳2两，天麻2两，羌活3两，独活3两，细辛3两，紫参3两，丹参3两，沙参3两，知母3两，玄参4两，当归4两，荆芥穗4两，红花4两，苍术4两，川山甲1两，僵蚕1两，蜈蚣1两，漏芦1两，萆薢1两，石斛1两，秦艽1两，乳香7钱5分，没药7钱5分，血竭7钱5分，麝香5分，木香1两5钱，地龙8钱，苦参皮8两。

主治：三十六种大风，诸恶危症。
加减：None
功效：None
用法用量：每服50丸，温酒送下。

各家论述：None
用药禁忌：忌牛、羊、猪肉、野味、鸡、鹅、烧酒、房事。避风为上。
