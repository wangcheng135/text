序号：69730
名称：琥珀丸
组成：琥珀5钱，乳香5钱，木香5钱，南星5钱，川乌5钱，当归5钱，沉香5钱，丁香5钱，檀香5钱，全蝎5钱，僵蚕5钱，天麻5钱，赤石脂5钱，延胡索5钱，五灵脂5钱，麝香2钱，辰砂2钱。
出处：《郑氏家传女科万金方》卷四。
主治：产后气虚恶食，胸闷腹胀，脾胃不和，寒热，夜睡多惊，昏眩泄泻；及新产血晕，不省人事。
加减：None
功效：None
用法用量：每服作4-5次，姜汤送下。新产血晕，不省人事，先用韭菜1握（切碎），以有嘴瓷瓶盛之，将米醋煮数沸沃之，以瓶口封没，将小嘴向产母鼻孔，令醋气透入即醒，急与琥珀丸即愈。
制备方法：上为末，同糯米糊为丸，如龙眼肉大，辰砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
