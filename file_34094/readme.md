名称：赤箭丸
组成：赤箭半两，羌活半两（去芦头），细辛（去苗叶）半两，桂（去粗皮）半两，当归（锉，炒）半两，甘菊花半两，防风（去叉）半两，天雄（炮裂，去皮脐）半两，麻黄（去根节）半两，蔓荆实半两，白术半两，杏仁（汤浸，去皮尖双仁，炒，研）半两，萆薢（锉）半两，茯神（去木）半两，山茱萸半两，羚羊角（镑）半两，芎半两，犀角（镑）半两，五加皮（锉）半两，五味子半两，阿胶（炙令燥）半两，人参半两，枫香脂（研）半两，天南星（炮）半两，白附子（炮）半两，龙脑（研）1钱，麝香（研）1钱，牛黄（研）1钱。

主治：肺感外邪，皮肤（疒帬）痹，项强背痛，四肢缓弱，冒昧昏塞，心胸短气。

制备方法：上23味为极细末，与研者5味拌匀，炼蜜为丸，如梧桐子大。
