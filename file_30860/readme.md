序号：30869
名称：苏桂姜辛汤
组成：苏叶3钱，桂枝3钱，甘草2钱，半夏3钱（炮），芍药3钱，细辛1钱，干姜2钱，五味子1钱。
出处：《四圣悬枢》卷二。
主治：太阳经病阳虚之人，水气停瘀，或原无积水而渴燥，饮冷蓄而不消，水气阻格肺胃，上逆则眩晕而呕咳，肝脾下陷则淋湿而泄利，外寒未解而里水又动，久而火败土崩则入三阴之脏。
加减：若下利，加赤石脂1钱；若渴者，去半夏，加瓜蒌根3钱；若小便不利，加茯苓3钱；若喘者，加杏仁3钱；若噫者，加附子3钱。
功效：外发表邪，内驱寒水。
用法用量：流水煎大半杯，热服。覆衣取汗。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
