序号：42070
名称：三黄宝蜡丸
组成：藤黄（以秋荷叶露泡之，隔汤煮10余次，去浮、沉，取中，将山羊血拌入，晒干）4两，天竹黄（无真者，九转南星代之）3两，红芽大戟3两，刘寄奴3两，血竭3两，孩儿茶3两，雄黄3两，朴消1两，当归尾1两5钱，铅粉3钱，水银3钱，乳香3钱，麝香3钱，琥珀2钱。
出处：《金鉴》卷七十五。
主治：金疮初起，伤破出血；一切恶疮。
加减：None
功效：None
用法用量：病重者每丸1钱，病轻者每丸5分，热黄酒调服。倘受伤至重，连服数次，服药后饮酒出汗，更妙。治一切恶疮，以香油化开敷之。
制备方法：上药各为极细末，称准和一处，将水银同铅粉放在铁锅内，火上热研成末，入前药内，共研匀；用炼净黄蜡24两，放瓷器内，坐滚水中化开，将药入内搅匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
