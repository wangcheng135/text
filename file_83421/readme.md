序号：83439
名称：乌金散
组成：血余半两，鲤鱼皮1两（2味各用1藏瓶去底入药，盛讫，却用瓦子盖，用好纸筋盐泥固济，用木炭火5斤烧通赤，取藏瓶放冷，打开出药），没药半两，红花1分（生用），伏龙肝1分（灶下取烧赤者土是），凌霄花半两（色鲜者，焙），好香墨半两（生用），干柏木1分（细研入，香好者可使），当归半两（去梢土，微炒令香用之）。
出处：《洪氏集验方》卷五。
主治：母热疾后胎死在腹，难产，生衣不下，产后血晕，起坐不得，如见异花，口干心闷，乍寒乍热，四肢浮肿，言语颠狂，见神见鬼，不语，泻痢，腰膝疼痛，小肠尿血涩痛，崩中，呕逆不安，咳嗽涕唾，喉中如猫作声，面色遍身黑，赤点子生，眼涩腰瘇，身发筋急等一切产后诸疾。
加减：None
功效：None
用法用量：以酒1盏，煎取8分，调药2钱，空心频服之。用无灰酒大妙。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
