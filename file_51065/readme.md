序号：51080
名称：草钟乳丸
组成：钟乳2两（别研令细），菟丝子1两（酒浸1宿，别捣），石斛1两，吴茱萸半两。
出处：《千金翼》卷二十二引曹公方。
主治：五劳七伤损肺，气急，丈夫衰老阳气绝，手足冷，心中少气，髓虚腰疼，脚痹身烦，口干不能食;脚气久虚，脉来沉细。
加减：多房者，加雄蛾30枚；若失精者，加苁蓉3两。
功效：安五脏，补肠胃，下气消食，长肌和中,潜补。
用法用量：空腹服7丸，1日2次。服之讫，行数百步，温酒3合饮之。复行200-300步，口胸内热，热如定，即食干饭豆浆。过1日，食如常，须暖将息。服过半剂觉有效，即相续服3剂。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：不得闻见尸秽等气，亦不用食粗臭陈恶食；初服7日，不可为房事，过7日后任性，然亦不宜伤多。
附注：曹公草钟乳丸（《外台》卷三十七）。曹公钟乳丸（《圣济总录》卷八十一）。
