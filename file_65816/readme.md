序号：65832
名称：菩提救苦丸
组成：紫苏叶2钱，天花粉1钱，玄参1钱，赤芍2钱，香附米2钱，川芎2钱，川朴2钱，生地2钱，防风2钱，羌活2钱，陈皮2钱，甘草2钱，粉甘葛2钱，黄芩2钱，苍术2钱，白芷2钱，细辛1钱，真蟾蜍1钱，麝香8分。
出处：《应验简便良方》卷下。
主治：春夏感冒风寒时症，以及瘟疫暑湿，头痛口渴，身热目胀，筋骨疼痛，恶心怯寒，脉象洪数。
加减：None
功效：None
用法用量：每服大人1丸，小儿半丸。内伤饮食，外感风寒，俱用神曲汤送下；余皆生姜汤或开水调服并渣一同眼下，惟暑勿用姜汤调服。
制备方法：上为极细末，用绢筛，用青荷叶并便同煎，为丸，每丸约重2钱5分。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
