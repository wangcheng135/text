序号：66422
名称：寸金丹
组成：神曲（炒）9两，苍术（炒）1两，白芷1两，甘草1两，川芎1两，茯苓1两，防风1两，草果仁1两，前胡1两，橘皮1两，砂仁2两，羌活1两，法半夏1两，藿香1两，苏叶1两，木香1两，厚朴（制）1两，薄荷1两，香附（醋炒）1两，乌药1两，白豆蔻1两，枳壳（炒）1两。
出处：《北京市中药成方选集》。
主治：中寒、中暑、感冒发烧，呕吐泻泄，胸满腹痛。
加减：None
功效：散寒解表，祛暑止呕。
用法用量：每服1至2丸，温开水送下。
制备方法：上为细末，过罗，每34两细末，兑朱砂1两，混合均匀，炼蜜为大丸，重2钱5分。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
