序号：67191
名称：麻黄连轺赤小豆汤
组成：麻黄1两，连轺1两，赤小豆半两。
出处：《此事难知》。
主治：身热不去，瘀热在里发黄，小便微利。
加减：None
功效：None
用法用量：每服1两，以水3盏，煎至1盏，去滓温服。
制备方法：上锉如麻豆大。
临床应用：荨麻疹：陆某，男，27岁。患荨麻疹状若地图形，全身瘙痒甚剧，时愈时作，缠绵6年。近年来复发次数增多，影响工作及睡眠，身感微恶寒，脉细数，苔薄白，体温37℃，其它无特殊症状。处方：麻黄连轺赤小豆汤加僵蚕。服药1剂后，症状大减，服2剂而荨麻疹消失。为巩固疗效，原方继服2剂，至今未再复发。
各家论述：None
用药禁忌：None
附注：None
