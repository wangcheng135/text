序号：59692
名称：调气汤
组成：代赭石18g（布包），旋复花6g（布包），牛膝9g，丹参15g，瓜蒌15g，郁金9g，白芍9g，柴胡6g，陈皮9g，枳壳9g，苏梗6g，桔梗6g，木香6g。
出处：《临证医案医方》。
主治：食管狭窄，食管憩室。吞咽梗阻不利，胸膈痞满，有时疼痛或逆气，舌苔白腻，脉弦。
加减：None
功效：理气、活血、解郁。
用法用量：None
制备方法：None
临床应用：None
各家论述：方中代赭石镇逆甲肝，有扩张食管的作用，为治疗食管狭窄之主药；牛膝降逆；丹参活血；瓜蒌、郁金宽胸解郁；白芍、柴胡舒肝；陈皮、枳壳、苏梗、桔梗、木香理气。
用药禁忌：None
附注：None
