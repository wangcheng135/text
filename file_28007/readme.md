序号：28015
名称：补虚款冬花汤
组成：款冬花3分，人参半两，升麻半两，桔梗3分（炒），杏仁（汤浸去皮尖双仁，炒）1两，白茯苓（去黑皮）3分，甘草（炙，锉）半分，干姜（炮）1分，柴胡（去苗）1两半，天门冬（去心，焙）半两，鳖甲（去裙襕，醋炙）1两，黄耆（细锉）半两，桑根白皮（锉，炒）3分，肉苁蓉（酒浸，去皴皮，炙）1两。
出处：《圣济总录》卷八十六。
主治：肺劳痰嗽，日渐羸劣。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，煎至8分，去滓，食后温服，1日2次。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：补肺款冬花汤（《普济方》卷二十七）。
