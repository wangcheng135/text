序号：25229
名称：妇宝丹
组成：艾附暖宫丸加阿胶。
出处：《医方集解》。
主治：虚寒，经水不调，带下淋浊，腰痠腿痛，四肢倦怠，崩中漏血，气促头眩，手足冰冷，气血两亏。
加减：None
功效：调经养血。
用法用量：None
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：妇科妇宝丹（《全国中药成药处方集》沈阳方）。《医林纂要》本方用当归四两（酒洗）、生地黄三两（酒润）、白芍药二两（炒）、川芎二两、艾叶二两、香附二两（童便、盐水、酒、醋各浸三日）、阿胶二两。《全国中药成药处方集》（沈阳方）本方用法：上为细末，阿胶化烊，炼蜜为丸，二钱重。每服一丸，白开水送下。
