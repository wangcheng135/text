序号：3528
名称：太岳活血丹
组成：乱发2斤（皂角水净洗，晒干，用清麻油2斤，入砂锅内炒，频以手拈看，脆乱如糊苔即止，不可令炒过），栗楔（谓栗3颗共1球，其中有扁薄者是，去壳，薄切，日干）1斤，皂角刺（烧通红，米醋内淬，焙）1斤，大黑豆（以湿布揩去尘垢，退黑皮，焙干）1斤，花桑枝（如臂大者，炭火烧，烟尽，米醋淬，取出，焙）1斤，蓖麻仁（别研，涂墨）3两，乳香（好者，细研，入米醋1碗，熬令熟香）4两，细墨半斤（一半用蓖麻仁3两，乳钵烂研，涂墨上，涂尽，用薄纸裹，以黄泥固济，日干，以火50斤锻令通红，放地上。盆盖，出火气，两饭久。一半用硇砂2两醋化，涂墨上，灸干），硇砂（光净者，醋化，涂墨上）2两。
出处：《局方》卷八。
主治：男子妇人外伤内损，狗咬虫伤，驴扑马坠，手足伤折，一切疼痛，腹中瘀血刺胁筑心，及左瘫右痪，走注疼痛，痈肿痔漏，妇人冷气入腹，血脉不通，产后败血灌注四肢，吹奶肿痛，血气撮痛。
加减：None
功效：None
用法用量：痛甚者每服1丸，轻可者服半丸，用无灰酒1盏，乳香1豆大，先磨香尽，次磨药尽，煎3-5沸，临卧温服，以痛处就床卧。如欲出汗，以衣被益覆，仍用药涂损处。妇人诸疾，更用当归末1钱，依法煎服。
制备方法：上药为末，入乳香膏内为丸，如弹子大。如乳香膏少，更入醋煮面糊。
临床应用：None
各家论述：None
用药禁忌：忌一切动风物，有孕者莫服。
附注：None
