序号：24706
名称：如妙辰砂化痰丸
组成：胆星1两，天麻1两，全蝎7钱（用苏叶包，煨），僵蚕7钱（用苏叶包，煨），白附子7钱（用苏叶包，煨），滑石5钱，蝉蜕5钱，人参5钱，川乌5钱，玄明粉5钱，防风4钱，天竺黄4钱，石菖蒲4钱，桔梗3钱，青黛3钱，枯矾3钱，礞石3钱，雄黄2钱，硼砂2钱，朱砂2钱，巴霜2钱，轻粉2钱，冰片3钱，麝香3钱，水银3钱，硫黄8钱（熔化，入水银同研），石菖薄1两，薄荷1两，荆芥1两。
出处：《幼科指掌》卷四。
主治：小儿风热，身热恶风鼻塞，身重咳嗽有痰，面赤唇干，微汗尿赤者。
加减：None
功效：None
用法用量：每服1丸，淡姜汤化下。
制备方法：上煎膏为丸，如芡实大，以青黛、朱砂、雄黄、金箔、银箔作五色衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
