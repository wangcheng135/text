序号：27675
名称：八珍益母丸
组成：当归（酒洗）、川芎（微炒）、白芍药（炒）、怀熟地、人参、白术（土炒）、白茯苓、炙甘草、香附（分4份，盐、醋、酒、童便各制听用）、阿胶（切碎，蛤粉炒珠）、益母膏。
出处：《摄生秘剖》卷三。
主治：胎前、产后诸虚百损，月事不调，子宫虚寒，久不受孕。
加减：None
功效：行气，养血，调经，种子。
用法用量：每服3钱，空心白滚汤送下。
制备方法：上药分两随证加减，各制为末，入益母膏，加炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：纯用四物则独阴不长，纯用四君子则孤阳不生，二方合用则气血有调和之益，而阴阳无偏胜之虞矣。香附行气生血，解郁散结；阿胶调经理血，治带止崩；益母膏者活血行气，有补阴之功。凡胎前、产后有所恃者，气血也，胎前无带，产后无虚，是其行中有补矣。命名益母者，所以利有子也。
用药禁忌：None
附注：None
