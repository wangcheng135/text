序号：4368
名称：九转长生神鼎玉液膏
组成：白术2斤（秋冬采，去粗皮），赤术（即苍术）16两（秋冬采，去粗皮）。
出处：《遵生八笺》卷十三。
主治：None
加减：更有加法，名曰九转。二转加人参3两，煎浓12次，熬膏，入前膏内，名曰长生神芝膏；三转加黄精1斤，煎汁熬膏，入前膏内，名曰三台益算膏；四转加茯苓、远志（去心）各8两，熬膏，加入前膏，名曰四仙求志膏；五转加当归8两，酒洗熬膏，和前熬膏内，名曰五老朝元膏；六转加鹿茸，麋茸各3两，研为末，熬膏，和前膏内，名曰六龙御天膏；七转加琥珀（红色如血者佳。饭上蒸1炊，为细末）1两，和前膏内，名曰七元归真膏；八转加酸枣仁（去核）8两，熬膏，和前膏内，名曰八神卫护膏；九转加柏子仁（净仁，研如泥）4两，入前膏内，名曰九龙扶寿膏。
功效：轻身延年，悦泽颜色。
用法用量：二药用木石臼捣碎，入缸中，用千里水浸1日夜，山泉亦好；次入砂锅煎汁1次，收起再煎1次，绢滤滓汁，去滓，将汁用桑柴火缓缓炼之，熬成膏，磁罐盛贮，封好入土，埋1-2日出火气。3钱1次，白汤调下，或含化。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌食桃、李、雀、蛤、海味等。
附注：None
