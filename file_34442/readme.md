序号：34452
名称：驱风散
组成：防风4两（去芦），白砂蜜半斤，朱砂1两（水飞，研令极细），薄荷4两（苗儿紫心者），蜗牛7个（瓦上炒去壳，细研，形如蜒蚰皆负壳者），皂角10条（不蛀者，去边寸，锉，先用水浸3日，去原浸水，却以1碗，挼取浓汁，去滓，入银器内熬去5-6分），天麻4两。
出处：《普济方》卷一○○引《卫生家宝》。
主治：暗风痫疾，涎潮不省人事，手足搐搦；又小儿惊风等疾。
加减：None
功效：None
用法用量：每服30-50丸，腊茶清送下，日3次。服1料，永除根。
制备方法：上为末，入朱砂、蜗牛，先以皂角膏子和匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌猪、鸡、鱼、面、动风物。
附注：None
