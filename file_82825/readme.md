序号：82841
名称：丹参膏
组成：丹参4两，蒴藋根4两，秦艽3两，羌活2两，蜀椒（汗，去目闭口者）2两，牛膝2两，乌头（去皮）2两，连翘2两，白术2两，踯躅1两，菊花1两，莽草1两。
出处：《千金翼》卷十六。
主治：伤寒时行，贼风恶气在外，肢节痛挛，不得屈伸，项颈咽喉痹塞噤闭，入腹则心急腹胀，胸中呕逆，缓风不遂，湿痹不仁，偏枯拘屈，口面喎斜，耳聋齿痛，风颈肿痹，脑中风痛，石痈，结核瘰疬，坚肿末溃，赤白瘾疹，诸肿无头作痈疽者，风结核在耳后，风水游肿疼痛翕翕。
加减：None
功效：None
用法用量：凡风冷者用酒服；热毒单服；齿痛绵沾嚼之；在腹内服之；在外摩之；缓风不遂，湿痹不仁，偏枯拘屈，口面喎斜，耳聋齿痛，风颈肿痹，脑中风痛，石痈，结核瘰疬，坚肿末溃，敷之取消；及赤白瘾疹，诸肿无头作痈疽者，摩之令消；风结核在耳后，风水游肿，疼痛翕翁，针之黄汁出，时行温气，服之如枣大1枚。小儿以意减之。
制备方法：上切，以苦酒5升，麻油7升合煎，苦酒尽，去滓，用猪脂煎成膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：丹参煎（《圣济总录》卷三十二）。
