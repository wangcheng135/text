序号：35410
名称：万病解毒丸
组成：射干2两，文蛤（即五倍子）2两，杏仁2两，石膏2两，续随子（去壳，去油）2两，蚤休（即金线重楼）2两，土朱2两，大戟2两，山豆根2两，山慈菇2两，白药子2两，大黄（酒蒸）2两，麝香2钱，青黛1两，威灵仙1两，白芷1两，黄连5钱，风化消5钱。
出处：《医学正传》卷六引《局方》。
主治：蛊毒，鼠莽、河豚鱼毒，菌毒，疫死牛马肉毒，竹木刺毒；喉痹、骨鲠，痈疽发背，疔肿疮疡，毒蛇犬咬，蜈蚣蜂蝎虿毒，刀斧、汤火伤；妇人鬼胎恶气，积块虫积，心胸痞满，肚腹膨胀。
加减：None
功效：None
用法用量：此药解一切毒，蛊毒，及鼠莽、河豚鱼毒，菌毒，疫死牛马肉毒，喉痹、骨鲤竹木刺毒，并用急流水磨下；痈疽发背，疔肿疮疡，毒蛇犬咬，蜈蚣蜂蝎虿毒，刀斧、汤火伤，并用井花水磨下，并涂伤处；妇人鬼胎恶气，积块虫积，心胸痞满，肚腹膨胀，并用好酒磨下。
制备方法：上为末，糯米糊为丸，如弹子大，青黛、滑石细研为衣，阴干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
