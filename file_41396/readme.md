序号：41411
名称：金疮铁扇散
组成：象皮5钱（切薄片，用小锅焙黄色以干为度，勿令焦），龙骨5钱（用上白者，生研），老材香1两（山陕等省无漆，民间棺殓俱用松香、黄蜡涂于棺内，数10年后，有迁葬者，棺朽另易新棺，其朽棺内之香蜡即谓之老材香。东南各省无老材香，即以数百年陈石灰1两代之，其效与老材香同），寸柏香1两（即松香中的黑包者），松香10两（与寸柏香一同熔化，搅匀，倾入冷水，取出凉干），飞矾1两（将白矾入锅熬透便是）。
出处：《经验广集》卷四。
主治：刀石破伤。
加减：None
功效：None
用法用量：遇有刀石破伤者，用药敷伤口，以扇向伤口处扇之。如伤口处发肿，煎黄连水用翎毛蘸涂之即消。
制备方法：上为极细末，贮瓷罐中。
临床应用：None
各家论述：None
用药禁忌：忌卧热处。
附注：None
