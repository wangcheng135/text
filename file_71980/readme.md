序号：71996
名称：滋阴降火汤
组成：百部3钱，生地1钱半，熟地1钱半，天冬1钱半，麦冬1钱半，知母1钱半，贝母1钱半，白术（炒）1钱半，白芍（酒炒）1钱半，白茯苓1钱半，黄耆（蜜炒）1钱半，地骨皮1钱半。
出处：年氏《集验良方》卷三。
主治：阴虚火动，发热咳嗽，吐痰喘急，盗汗口干。
加减：骨蒸夜热，加鳖甲3钱；痰中带血，加真阿胶3钱，倍加熟地黄；盗汗不止，加炒枣仁2钱，倍加黄耆；咽喉痒或痛，加桔梗、桑白皮，倍加贝母；咳嗽痰多喘急，加人参、沙参各2钱；遗精，加山药、芡实各5钱，牛膝2钱；小便淋闭，加车前子、萆薢各2钱；大便不实，加炒山药、扁豆各5钱。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
