序号：83354
名称：七水凌
组成：朴消5斤，芒消3斤（如雪者佳），滑石1斤半，玉泉石1斤，石膏1斤，卤碱5斤（如凌者），凝水石1斤（如雪者），冻凌水5升，霜水1升，雪水1升，露水5升半，寒泉水5升，雨水1升，东流水5升半。
出处：《千金翼》卷十八。
主治：大热及金石发动，金石凌不制者。
加减：None
功效：None
用法用量：前7味，各别捣粗筛；后7味，澄令清，铜器中纳上件七味散，极微火煎取7升，1宿澄清，纳瓷坩中净处贮之，以重帛系口120日，皆如冻凌，状成如白石英，有八棱，成就或大如著，有长1尺者，名曰七水凌。有人服金石发热者，以井花水和5分匕服之，1服极热即定；伤寒发热服1刀圭；小儿发热与麻子许。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：不可多用；服药得热退之后七日，乃慎酒、肉、五辛等物。
附注：None
