序号：25729
名称：朱砂散
组成：朱砂1两（细研，水飞过），紫檀1两（锉），漏芦1两，雄黄1两（细研），雌黄半两（细研），白蔹1两，石胆半两（细研），紫石英1两（细研，水飞过），牛黄3分（细研），阿魏半两（面裹煨，令面熟为度），犀角屑1两，龙骨1两，石膏1两（细研，水飞过），消石1两半。
出处：《圣惠》卷二十四。
主治：大风。
加减：None
功效：None
用法用量：每服1钱，食前以温酒调下。如饥，即宜吃小豆饭及枣汤，食勿令过饱，饱即其虫难出。服药后似淋，即以盆子盛著，当有虫出，或当日出，或2-3日出者，或至7日虫出，见虫即且服药，候虫尽便愈。
制备方法：上为细散，入石药末，更同研令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
