序号：66458
名称：小儿化毒散
组成：牛黄8g，珍珠16g，雄黄40g，大黄80g，黄连40g，甘草30g，天花粉80g，川贝母40g，赤芍80g，乳香（制）40g，没药（制）40g，冰片10g。
出处：《中国药典》（一部）。
主治：小儿疹后余毒未尽，烦躁，口渴，口疮，便秘，疖肿溃烂。
加减：None
功效：清热解毒，活血消肿。
用法用量：口服，1次0．6g，1日1至2次，3岁以内小儿酌减；外用敷患处。
制备方法：上药除牛黄、冰片外，雄黄、珍珠分别水飞或粉碎成极细末；其余乳香等8味粉碎成细末；将牛黄、冰片研细，与上述粉末配研过筛，混匀即得。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
