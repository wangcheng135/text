序号：47441
名称：祛毒汤
组成：土茯苓2斤，防风1两，荆芥1两，五加皮1两，白鲜皮1两，木瓜1两，威灵仙1两，白芍药1两，当归（酒洗）1两，川芎1两，生地黄（酒洗）1两，牛膝1两，白茯苓1两，杜仲（炒）1两，地骨皮1两，白芷1两，青藤1两，槐花1两，黄连1两。
出处：《疡科选粹》卷六。
主治：远年杨梅风瘘，筋骨疼痛。
加减：None
功效：None
用法用量：上作10帖。每帖水、酒各半，煎1钟，又煎滓，搅匀，作2次服。药滓晒干，3帖共煎汤，候温洗浴。但初起服5帖，疮势反盛，乃毒气攻外。轻者10余帖愈，重者亦不过20贴，定见奇效。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌房事、生冷，煎煿、鸡、鹅、羊，并猪头、猪蹄、鱼、虾一应动气之物。
附注：None
