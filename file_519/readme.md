序号：519
名称：五淋白浊丸
组成：公英6钱7分，地丁6钱7分，瞿麦6钱7分，萹蓄6钱7分，木通6钱7分，泽泻6钱7分，金砂6钱7分，灯心6钱7分，竹叶6钱7分，甘草6钱7分，猪苓6钱7分，土苓6钱7分，萝茶1两3钱4分，滑石1两3钱分，赤苓1两3钱4分，赤芍3钱分，蝉退3钱4分，车前1两，凤眼草1两，石韦1两，通草1两，山栀2钱，贡桂2钱。
出处：《全国中药成药处方集》(吉林方)。
主治：五淋白浊，女子赤白带下，横痃，下疳，膀胱发热，梦遗滑精，便溺不清，尿管混血，花柳诸症。
加减：None
功效：搜毒，止淋，消浊，利下，祛炎，镇痛。
用法用量：每服2钱，白水送下，1日2次，早、晚用之。
制备方法：上为细末，水泛为小丸，滑石为衣。
临床应用：None
各家论述：None
用药禁忌：服后忌饮茶水，孕妇忌用。
附注：None
