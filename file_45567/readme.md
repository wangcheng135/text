名称：柏子仁丸
组成：柏子仁（炒）1两，泽兰叶1两，甘草（炙，锉）1两，当归（切，焙）1两，芎1两，白术半两，白芷半两，桂（去粗皮）半两，细辛（去苗叶）半两，防风（去叉）1两半，人参1两半，牛膝（去苗，酒浸，切，焙）1两半，麦门冬（去心，焙）1两半，生干地黄（焙）1两，石斛（去根）1两，厚朴（去粗皮，生姜汁炙，锉）半两，藁本（去苗土）半两，芜荑半两，附子（炮裂，去皮脐）1两，干姜（炮）1两。

制备方法：上为末，炼蜜为丸，如梧桐子大。
