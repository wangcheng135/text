序号：57650
名称：大铁弹丸
组成：自然铜（烧红，醋淬7次）1两半，虎胫骨（酒浸，炙黄）1两，当归（酒浸，焙）1两，白附子（炮）1两，川乌（炮，去皮脐）1两，五灵脂（炒）1两，麻黄（去节）1两，没药半两，乳香半两，全蝎（焙）半两，安息香半两，白芷半两，直僵蚕（炒去丝）半两，乌蛇肉（酒浸，焙干）3分，木鳖21个（去壳，炒熟），朱砂1分，麝香1分。
出处：《直指》卷四。
主治：中风瘫缓，口眼喎斜，筋骨挛疼，肢体麻木。
加减：None
功效：None
用法用量：每服1丸，温酒磨下。
制备方法：上为末，以酒煮安息香，入飞白面为糊，丸如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
