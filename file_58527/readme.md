序号：58542
名称：秘传二仙糕
组成：人参半斤，山药半斤，白茯苓半斤，芡实仁半斤，莲肉（去皮心）半斤，糯米1升半，粳米3升半，蜜半斤，白糖10斤。
出处：《扶寿精方》。
主治：内伤脾胃虚弱，饮食不进者。痈疽脾胃虚弱，精神短少，饮食无味，食不作肌，及平常无病，久病但脾虚食少，呕泄者。
加减：None
功效：固齿黑发，壮阴阳，益肾水，养脾胃。轻身耐老，壮助元阳。
用法用量：每早1大匙，白汤调下。百日内见效。
制备方法：上为细末，合匀，将蜜糖溶化，和末掺挼得宜，小木笼炊蒸之，上以米1撮成饭，则药成矣，取起尽作棋子块，慢火上烘干作点心，或为末，贮瓷器。
临床应用：None
各家论述：None
用药禁忌：None
附注：秘传三仙糕（《东医宝鉴·杂病篇》卷四引《集略》）、八仙糕（《外科正宗》卷一）。
