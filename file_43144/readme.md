序号：43159
名称：三棱丸
组成：京三棱（煨，锉）5两，鸡爪三棱（煨，锉）5两，陈橘皮（汤浸去白，焙）5两，青橘皮（汤浸去白，焙）5两，巴豆（去皮心膜，出油）5两，石三棱（煨，锉）5两，槟榔10枚（半生用半炮，锉），肉豆蔻10枚（去壳，醋浸2宿），丁香1两，益智（去皮）1两，木香2两，硇砂1两半（研，飞过）。
出处：《圣济总录》卷七十二。
主治：多年积气癥癖。
加减：None
功效：None
用法用量：每服3丸，如当心气块，茱萸汤送下；左右气块，木香汤送下；本脏气块，茴香子汤送下。
制备方法：上为末，醋煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
