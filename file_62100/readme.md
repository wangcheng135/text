序号：62116
名称：救命丹
组成：川山甲3大片（用蛤粉炒熟，不用粉），甘草节2钱，乳香1钱，天花粉2钱，赤芍3钱，皂角刺5分（去针），贝母2钱，没药5分，当归1两，陈皮1钱，金银花1两，防风7分，白芷1钱，白矾1钱，生地3钱。
出处：《洞天奥旨》卷十四。
主治：痈疽。
加减：痈疽发背在头及脑后背脊，加羌活1钱，角刺倍之；在胸胁少阳经部位者，加柴胡1钱，瓜萎仁2钱；在腹脐太阴者，加陈皮5分，赤芍3钱，白芷1钱；生在手臂膊，加桂枝3分；生在腿膝，加牛膝2钱，防已5分，黄柏1钱，归尾3钱；如肿硬，加连翘2钱，木鳖仁5分；倘是疔疮，方中加紫河车3钱，苍耳子2钱。如人虚弱，不溃不起，加人参3钱，甘草1钱；如人壮实，加大黄2钱，麻黄1钱（连根节用）。
功效：None
用法用量：上用酒、水各数碗，煎8分，疮在上，食后服；疮在下，食前服。能饮酒者，再多饮数杯。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌酸醋、铁器。服毕宜侧卧，少暖有汗。
附注：仙传救命丹（《集验良方三百种》）。
