序号：67416
名称：黄公集圣丸
组成：芦荟2钱，五灵脂2钱，夜明砂2钱，砂仁2钱，橘红2钱，木香2钱，莪术2钱（煨），使君子肉2钱，川连3钱，川芎3钱，干蟾（炙）3钱，当归1钱半，青皮1钱半。
出处：《赤水玄珠》卷二十六。
主治：一切疳症。
加减：因于虚者，加人参2钱，白术3钱，去莪术，青皮；用于热者，加龙胆草3钱，去砂仁、莪术；因于吐泻下痢者，加白术2钱，肉果、煨诃肉各1钱5分，去青皮、莪术；因于积痛者，加煨3棱、川楝子肉、小茴香各1钱，去当归、川芎；因于疟者，加鳖甲（醋炙）3钱；因于虫者，加白芜荑1钱5分，川楝子肉2钱，去当归、川芎；因于渴者，加人参、白术各2钱，去莪术、砂仁。
功效：None
用法用量：看大小服，米饮送下。
制备方法：上为细末，用雄猪胆汁2个，面糊为丸。
临床应用：None
各家论述：此方不热不寒，补不致滞，消不致耗，至稳至妥。
用药禁忌：None
附注：None
