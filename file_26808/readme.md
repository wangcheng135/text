序号：26816
名称：延龄丹
组成：乌龙（即黑犬骨也，自脑骨至尾1条，全用好醋浸1宿，煮醋干，再用酥炙，为末，听用），鹿茸（酥炙）8钱，巴戟（酒浸）1两，沉香1两，石莲子（去壳心）1两，远志肉（炒）5钱，大茴香5钱，石燕子（雌雄各3对，烧红投姜汁内7次），故纸（炒）5钱（以上为末听用），何首乌（黑豆蒸9次）4两，熟地（酒洗）1两，床子（炒）2两，芡实肉2两，归身（酒洗）1两，川芎1两，白芍（酒炒）2两，生地1两（酒洗），天冬1两，麦冬1两，马蔺花1两，冬青子1两，楮实子（酒洗）1两，母丁香20个，枸杞子4两，金樱子1斤（去瓤核）。
出处：《年氏集验良方》卷二。
主治：阳痿。
加减：None
功效：None
用法用量：每服3钱。
制备方法：上药除药末外，共水1斗，煎至1升，去滓，取起晾冷，和入药内；又用黄雀49个，好酒煮烂，捣匀，同药末、乌龙骨为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
