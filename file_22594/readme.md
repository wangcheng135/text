序号：22602
名称：启脾散
组成：潞党参（元米炒黄，去米）3两，制冬术3两，建莲肉3两，楂炭2两，五谷虫炭2两，陈皮1两，砂仁1两。
出处：《成方便读》卷四。
主治：小儿因病致虚，食少形羸，将成疳积；或禀赋素亏，脾胃薄弱，最易生病者。
加减：None
功效：None
用法用量：每服2钱，开水下。
制备方法：上为末。
临床应用：None
各家论述：凡小儿之离母胎也，皆谓之后天，莫不藉谷食以为长养，因先天之禀赋不足者，难于接补；后天之气血亏弱者，易于滋培故古人每有借后天以济先天之法，不特脾胃为后天之源，抑且土为万物之母，土旺则四脏皆旺，正自充而病自除耳。方中党参、莲肉、冬术大补脾元；陈皮、砂仁助其健运；而以楂炭、谷虫消磨不尽之滞；兼广补药之功。
用药禁忌：None
附注：None
