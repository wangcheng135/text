序号：65697
名称：菖蒲导痰汤
组成：半夏9g，茯苓12g，橘红9g，甘草6g，菖蒲12g，南星6g，枳实6g。
出处：《中医内科临床治疗学》。
主治：痰气生厥，忽然眩仆，喉有痰声，或呕吐涎沫，平素可见痰多、胸闷、乏力等，脉多沉滑。
加减：None
功效：豁痰开窍。
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：菖蒲导痰汤，即二陈汤去乌梅，加菖蒲、南星、枳实而来。二陈汤燥湿化痰，理气和中；南星燥湿，祛风止痉，配半夏、陈皮可豁痰顺气；枳实行气化痰，散结消痞；菖蒲芳香化湿，开窍宁神，合之则本方有豁痰理气开窍的功效，用于发作将息甚为适宜。
用药禁忌：None
附注：先以通关散搐鼻开窍，继以菖蒲导痰汤治疗。
