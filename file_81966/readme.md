序号：81982
名称：镇心丸
组成：银屑1分半（研），牛黄9铢，丹砂（研）5分，甘草（炙）5分，麦门冬（去心）5分，远志（去心）5分，防葵4分，人参4分，防风4分，细辛4分，茯神4分，椒（汗）4分，附子（炮）4分，紫石英（研）4分，桂心6分，干姜6分，菖蒲3分，紫菀3分。
出处：《外台》卷十五引《深师方》。
主治：老小心气不足，虚弱，时苦小语，劳则剧；及风邪百病。
加减：None
功效：None
用法用量：每服3丸，食前服，日3次，不知，稍稍增之。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌海藻、菘菜、生菜、生葱、猪肉、生血、酢物、饧等。
附注：牛黄丸（《圣济总录》卷五）。
