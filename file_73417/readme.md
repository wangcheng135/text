序号：73433
名称：椒红丸
组成：川椒（微炒去汗，取红）5两，磁石3两（烧，醋淬7遍，捣碎细研，水飞过），白蒺藜1两（微炒，去刺），巴戟2两，附子3两（炮裂，去皮脐），硫黄2两（微炒，细研），厚朴3两（去粗皮，涂生姜汗，炙令香熟），茴香子2两（微炒），盐花2两。
出处：《圣惠》卷四十四。
主治：风湿积冷腰痛，行立无力，小便滑数。
加减：None
功效：None
用法用量：每服30丸，空心以温酒送下，晚食前再服。
制备方法：上为末。以羊肾3对，去尽筋膜，细研，用好酒2升相和，于银锅内熬成膏，和前药末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
