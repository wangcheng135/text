序号：544
名称：五黄丹
组成：生大黄2两，人中黄5钱，明雄黄5钱，广姜黄3钱，牛黄1钱，朱砂5钱，冰片5分，蝉退壳5钱，僵蚕1两5钱。
出处：《羊毛温证论》。
主治：一切温毒。
加减：None
功效：None
用法用量：治头面肿大，菊花1钱，薄荷8分，水煎去滓，和丹1粒，连服数次，以消为度：治羊毛温证，石膏1两，水煎去滓，化元明粉1钱，和丹服；治斑疹痧痘、火毒、赤游丹肿等证，石膏1两，犀角(镑屑)1钱，水煎去滓，和丹服；治温疟，寒少热多，青蒿2钱、石膏5钱，水煎去滓，和丹服；治红白毒痢腹痛坠胀，当归2钱、黄芩1钱，水煎去滓，和丹服；治伏热吐血，秋石5分，开水化和丹服；治伏邪胸闷头痛，薄荷1钱，川芎5分，水煎去滓，和丹服；治湿毒，瘴疬、蛊毒、脓疮、疥癣、痈肿疔疡，金银花1钱、甘草1钱，水煎去滓，和丹服；治小儿急惊阻厥，发热神昏，胸闷气喘，痫风抽搐，薄荷1钱，钩藤3钱，水煎去滓，和丹服。
制备方法：上为细末，用黄蜜、陈酒为丸，重2钱1粒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
