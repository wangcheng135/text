序号：26366
名称：当归木香汤
组成：青皮、五加皮、海桐皮、桑白皮、陈皮、地骨皮、丁香皮、牡丹皮、棕榈皮（上药全烧为灰末）10大钱，当归1两，木香半两，红芍药半两。
出处：《宣明论》卷九。
主治：妇人血气虚劳，令人头目昏眩，谵语声沉重，舌根强硬，言语謇涩，口苦不思饮食，白日间睡，夜发虚汗，神思恍惚，梦寐狂言，面色萎黄，频发喘咳，遍身疼痛，骨节气走注，四肢沉重，背胛拘急，发寒热，五心烦躁，唇干多渴，胸膈不利，咽喉噎塞，（兀王？？）羸瘦弱。
加减：如妇人血脏、脐下冷痛似刀搅，遍身肿满，室女经脉不通，用斑螫半两，大黄1两（炒，锉），2味为末，用黄狗胆汁，以温酒调下1钱；如脐下痛止，心间痰未止，不服二味。
功效：None
用法用量：每服1钱，水1盏，入小油2点．钱1文，同煎至7分，温服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
