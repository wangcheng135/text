序号：77716
名称：雄黄膏
组成：雄黄2两（细研），黄耆2分，漏芦3分，络石3分，续断3分，营实3分，紫葛半两，白蔹半两，桑寄生半两，商陆半两，连翘半两，汉防己半两，赤芍药3两，败酱半两，川升麻半两，莽草半两，当归1两，苦参1两，木通1两，紫苑1两（去土），芫花1两，藜芦1两（去芦头），白及1两，(艹闾)茹1两，黄丹15两，蜡4两，清油3斤。
出处：《圣惠》卷六十三。
主治：一切痈疽、发背、脑痈诸毒疮，及乳痈疼痛。
加减：None
功效：None
用法用量：用故帛上摊贴，逐日换药。以愈为度。
制备方法：上锉碎，以酒2升，拌1宿，先取油安铛内，以慢火煎令熟，即下药，煎白蔹赤黑色，滤去药，下蜡候溶，以绵滤过，拭铛，却安油入铛内，下黄丹，于慢火上以柳篦不住手搅，候变色黑，搅滴于水内为珠子，膏成也。去火，入雄黄末，调令匀，倾于瓷器中盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
