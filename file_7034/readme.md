序号：7034
名称：四物补经汤
组成：香附6分，当归6分，白芍（酒炒）6分，熟地黄5分，川芎5分，黄耆4分（蜜炙），白茯苓4分（去皮），白术4分（去芦），黄芩4分，玄胡索4分，陈皮4分，砂仁3分，小茴3分（酒炒），人参3分，阿胶（炒）3分，沉香（另研）3分，吴茱萸3分，粉草3分。
出处：《寿世保元》卷七。
主治：妇人25-26岁，血海虚冷，经水不调，或时小腹疼痛，或下白带如鱼脑髓，或似米泔，不分信期，每日淋漓不止，面色痿黄，四肢无力，头昏眼花目眩。
加减：None
功效：None
用法用量：加生姜3片，水煎，空心热服。
制备方法：上锉。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
