序号：74610
名称：新加玉女煎
组成：生石膏6钱（研），紫石英4钱（研），淮牛膝1钱半，大熟地6钱（切丝），灵磁石4钱（研），东白薇4钱，石决明5钱（杵），原麦冬3钱（朱染），知母2钱（秋石1分化水炒），青盐陈皮1钱。
出处：《重订通俗伤寒论》。
主治：肝挟胆火，化风上翔，冲气上而冲心，心中痛热，甚则为气咳，为呃逆，为晕厥，名冲咳、冲呃、冲厥。
加减：None
功效：清肝镇冲，育阴潜阳。
用法用量：先用熟地丝泡取清汤，先煎3石，百余沸，代水煎药。
制备方法：None
临床应用：None
各家论述：何秀山：本方以三石、白薇镇逆纳冲为君；臣以牛膝、决明，降逆气而潜肝阳，麦冬、熟地养胃液以滋肾阴；佐以秋石水炒知母咸苦达下；使以青盐陈皮辛润疏中。此为清肝镇冲，育阴潜阳之良方。
用药禁忌：None
附注：None
