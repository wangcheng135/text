序号：38602
名称：法制清金丹
组成：广陈皮（拣红者，净米泔水洗，略去白，锉大片晒干1斤，先用枳壳4两，去瓤净，用水6碗，浸1宿，煎浓汁2碗，拌橘皮，浸透1夜，次日蒸透晒干；2次，用甘草3两，去皮照前煎汤，浸蒸晒干；3次，用款冬花，去芦梗净，4两，用水照前煎浸蒸晒；4次，用桔梗，去芦净，4两，用水照上浸1夜，浓汁煎2碗，去滓，加白硼砂4钱，玄明粉4钱，青盐4钱，入汁化开，照前拌酒，浸1夜，蒸透晒干；5次，用竹沥浸拌，照前蒸晒；6次，用梨汁浸拌，照前蒸晒；7次，用姜汁、萝卜汁浸拌，照前蒸晒），沉香3钱，檀香3钱，山楂米1两，百药煎1钱，细茶1两，乌梅肉1两，白硼砂5钱，五味5钱，人参1两，天花粉1两，薄荷叶1两，半夏1两（姜汁炒）。
出处：《遵生八笺》卷十八。
主治：痰火咳嗽。
加减：None
功效：生津止渴，消食顺气，调中降火，清气化痰，止嗽。
用法用量：临卧，或有痰火涎嗽时含咽。
制备方法：上为细末，加白糖霜10两，炼熟蜜10两，和匀，入臼捣千杵，印成饼。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
