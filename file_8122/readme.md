序号：8122
名称：加减泻白散
组成：桑白皮3钱，桔梗2钱，地骨皮1钱半，甘草（炙）1钱半，知母7分，麦门冬5分，黄芩5分，五味子20个。
出处：《卫生宝鉴》卷十一。
主治：因膏粱而饮，劳心过度，肺气有伤，以致气出腥臭，唾涕稠粘，口舌干燥者。
加减：None
功效：None
用法用量：上(口父)咀。作1服，水2盏，煎至1盏，去滓，食后温服，日2次。
制备方法：None
临床应用：None
各家论述：因洪饮大热之气所伤，滋溢心火，刑于肺金，故以桑白皮、地骨皮苦微寒降肺中伏火而补气，用以为君；黄芩、知母苦寒，治气息腥臭，清利肺气，用以为臣；肺欲收，急食酸以收之，五味子之酸温以收肺气，麦门冬甘苦寒，治涕唾稠粘，口舌干燥，用以为佐；桔梗体轻辛温，治痰逆、利咽膈，为使也。
用药禁忌：忌酒、面，辛热之物。
附注：泻白散（《赤水玄珠》卷三）、加减泻白汤（《济阳纲目》卷一○五）。
