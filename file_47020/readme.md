序号：47035
名称：独活散
组成：独活（去芦头）3分，雄黄（研）半两，青黛（研）1分，麻黄（去节，先煎，掠去沫，焙干）半两，水银1分（与丹砂、雄黄点醋同研，以水银星子尽为度），阿魏（生，研）1分，雷丸（炮）半两，丹砂（研）半两，滑石（捣，研）半两，石胆（研）1分，牛黄（研）1分，紫石英（研）1分，斑蝥40枚（与糯米同炒，去翅足，米不用），芫青40枚（与糯米同炒熟，走翅足，米不用）。
出处：《圣济总录》卷十八。
主治：大风癞。
加减：None
功效：出五虫。
用法用量：每服1钱匕，空心温酒调下，晚更服。当有虫从小便中出，或当日，或3-4日至10日方出。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：不得饱食。
附注：None
