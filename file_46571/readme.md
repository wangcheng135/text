序号：46586
名称：千金封脐膏
组成：锁阳1两，川花椒1两，川附子1两，吴茱萸1两，韭菜子1两，紫梢花1两，白芷1两，生地1两，当归1两，熟地1两，天冬1两，麦冬1两，鹿茸（去毛）1两，杜仲1两5钱，海蛆5钱。
出处：《北京市中药成方选集》。
主治：诸虚不足，阳萎腰痛，遗精盗汗，虚寒腹痛。
加减：None
功效：补肾散寒止痛。
用法用量：微火烤化，贴脐下。
制备方法：上药酌予碎断，用香油128两炸枯，去滓过滤，炼至滴水成珠，入黄丹48两成膏，取出浸入水中，去火毒后加热溶化。每料兑入肉桂3钱，肉果5钱，没药3钱，乳香3钱，母丁香2钱5分，牡蛎（煅）1两，麝香1钱5分，沉香2钱5分，上药搅匀摊贴，每张油重：大者7钱，小者3钱5分，用红布光。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
