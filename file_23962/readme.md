序号：23970
名称：安魂琥珀丹
组成：天麻1两，川芎1两，防风1两，细辛1两，白芷1两，羌活1两，川乌（炮，去皮脐）1两，荆芥穗1两，僵蚕1两，薄荷叶3两，全蝎半两，粉甘草半两，藿香半两，朱砂（细研，水飞）半两，麝香1钱，珍珠1钱，琥珀1钱。
出处：《丹溪心法附余》卷一。
主治：中风，左瘫右痪，口眼斜，心神不宁；蛇伤，狗咬，破伤风，牙关紧急；小儿痘疹。
加减：None
功效：安魂定魄，疏风顺气。
用法用量：每服1丸，空心茶清或酒送下。若蛇伤，狗咬，破伤风，牙关紧急，先用1丸擦牙，后用茶清调下1丸；如小儿初觉出痘疹，即用茶清调1丸与服。
制备方法：上为细末，炼蜜为丸，如弹子大，金箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
