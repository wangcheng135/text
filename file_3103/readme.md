序号：3103
名称：天麻饼
组成：川乌（生，去皮尖）1两，天麻1两，防风1两，干姜1两，羌活1两，甘草（生）1两，半夏（洗）半两，天南星（生）半两。
出处：《普济方》卷四十六。
主治：头风，伤风鼻塞，一切头疼，偏正头风；并治中脘痰饮停积，及脾胃饮食所伤。
加减：None
功效：温中快膈。
用法用量：每服3饼，荆芥或葱茶细嚼送下。一方有川芎，无羌活，用生姜汁和捏成饼子，阴干1-2日，晒干服，更增饼数随意。头疼甚者，更于芎辛汤方中求之。若初成伤寒，因汗而解，尚余头痛者，浓煎葱白、生姜汤送下。
制备方法：上用汤浸炊饼，和捏成丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
