序号：37353
名称：宝灵丹
组成：朱砂1两（研，水飞），雄黄1两（研，水飞），黄丹1分（研），苦药子1分，续随子1分（去皮毛），山豆根1两（生），蜈蚣2条（1条微炙，1条生用），斑蝥1分（去头翅足，半生半炒），巴豆2分（去皮膜，出油尽），麝香2分（研）。
出处：《卫生总微》卷十五。
主治：小儿中蛊毒。
加减：None
功效：None
用法用量：若未能语儿，中毒即休；如能语儿，但将药1粒入门，即患儿先闻其香是验。以腊茶清送下1丸，不得嚼破。须臾患儿必言心头作响声，毒物便下。或从口鼻出，或自大便下。嫩即是血，老即加诸虫状，其形不一，药随物下，凝血中裹之。口噤者拗灌之。
制备方法：上为末，拌研匀。于端午日、重九日合之最佳，事急者不拘，用糯米粥为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
