序号：22273
名称：阳春酒
组成：人参（切片）5钱，白术5钱，熟地5钱，当归身（切片）3钱，天门冬3钱，枸杞3钱，柏子仁2钱5分，远志2钱5分。
出处：《外科正宗》卷二。
主治：脑疽、诸发已溃，流脓尽时，脾胃虚弱，肌肉生迟；或气血原不足，以致肉色淡白，不能生长收敛。
加减：None
功效：生长肌肉，强健脾胃，美悦颜色，滋润皮肤，去病延寿。
用法用量：每早、午、晚各饮1杯热服；如夏月天炎易坏，不堪久服，将药分作5份，每次用酒1斤随便浸服亦效；如酒将完，药尚有味，再添酒浸；饮之1次以后，药淡无味，不必再浸用之。
制备方法：上药用绢袋宽贮，以无灰好酒5斤瓷罐内浸至一伏时。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
