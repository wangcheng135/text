序号：4066
名称：木瓜丸
组成：木瓜5枚（将硇砂10两研细，汤浸绢滤澄清，银石器内煮成膏后，将木瓜削去皮切片，以硇砂霜拌匀，碗内蒸令熟，收藏旋用。每料用木瓜3两），雀40只（去头足肠胃，醋煮烂，砂盆研，布绞取肉。以硇砂2两、木瓜2两入干姜2两、椒红末2两，酒3升，慢火煎成膏），附子（炮裂，去皮脐）3两，菟丝子（酒浸3日，焙，捣末）3两，补骨脂（炒）1两，沉香（锉）1两，木香1两，天雄（焙裂，去皮脐）1两，石斛（去根）2两，肉苁蓉（酒浸去皴皮，切，焙）2两，天麻（酒炙）2两，蒺藜子（炒，去角）2两，羌活（去芦头）1两半，茴香子（炒）3分。
出处：《圣济总录》卷九十二。
主治：骨极，腰膝痛，风虚气衰，不能久立，脑髓痠痛。
加减：None
功效：补虚壮元。
用法用量：每服30丸，煨生姜盐汤送下。
制备方法：上药除膏外，捣罗为末，用前膏搜丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
