序号：28573
名称：苏葛饮
组成：苏木1钱半，紫葛1钱半，白芍1钱半（酒炒），当归3钱，蒲黄3钱，桂心1钱半，生地汁1合。
出处：《医略六书》卷三十。
主治：产后两胁痛胀，脉涩大浮者。
加减：None
功效：None
用法用量：水煎，去滓，生地汁冲服。
制备方法：None
临床应用：None
各家论述：产后素多郁怒，抑遏清阳，而肝血已虚，瘀血乘虚留结，放两胁疼痛，胀满不止焉。苏木疏肝胆以破血，紫葛行血分以升阳，当归养肝血之虚，蒲黄破瘀血之滞，桂心通闭平肝，白芍敛阴和血，生地汁壮水以滋荣肝木也。水煎冲服，务使水润木荣，则瘀血自化而新血完复，清阳无不升，何患其胁胀疼痛之不去耶。
用药禁忌：None
附注：None
