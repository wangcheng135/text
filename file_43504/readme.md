序号：43519
名称：前胡吐热汤
组成：前胡3两，白术3两，赤茯苓3两，枳实（炙）3两，细辛3两，旋覆花3两，龙胆3两，杏仁（去皮尖双仁）3两，常山3两，松萝3两，竹叶（切）1升。
出处：《外台》卷十六引《删繁方》。
主治：脾劳热，有白虫长1寸，在脾为病，令人好呕，胸中塞，呕而不出。胃咳，呕而不出。
加减：若腹中热满，加芒消、山栀子仁、黄芩各3两，苦参2两，加水2升。
功效：None
用法用量：以水1斗，煮取3升，去滓，分为3服。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：忌酢物、桃、李、雀肉、生葱、生菜。
附注：前胡汤（原书卷二十六）、前胡散（《圣惠》卷五十七）。方中枳实，《圣惠》作“枳壳”。
