序号：22196
名称：一块气丸
组成：官桂半两，沉香4两，玄胡索半两，江子1两（去壳、油），蓬术半两（火炮），锡灰4两，京三棱1两（炮，去皮），香附子1两（醋浸），姜黄半两，南木香4两，黑牵牛（头末）半两，砂仁半两，大麦芽4两（江子炒），使君子半两（去皮），枳实半两，陈皮1两（去白），槟榔半两，枳壳半两，青皮1两半，大黄半两（醋炙），雷丸半两，萝卜子1两（江子炒），白豆蔻半两，唐球1两半，川乌2钱半（火炒），芫花1两（酒浸，炒），丁香半两，皂角斤半（去皮，醋浸），胡椒1两。
出处：《普济方》卷一八二。
主治：一切气。
加减：None
功效：None
用法用量：每服5-7丸，各随其汤送下。妇人一切血气，当归酒送下；血崩，燕子泥汤送下；小儿脱肛，艾汤送下；小儿奶痞，橘皮汤送下；小儿惊风，一岁1丸，薄荷汤送下；白痢，姜汤送下；小儿脾积，使君子、猪胆、芦荟汤送下；赤痢，甘草汤送下；一切吐逆，生姜汤送下；心膈膨胀，新水送下；下元冷，好酒送下；风热闭塞，大小便不通，井花水煎豆粉汤送下；妇人经脉不通，红花，当归酒送下；赤白带下，蔓荆子汤送下；血红，当归酒送下；产前产后，吴茱萸1两，重酒1升，煎至2沸送下；血块、气血等，生姜、橘皮入醋少许煎下；常服者，淡姜汤送下；少女经脉不通，红花、当归酒送下；男子小肠气，茴香汤送下；咳嗽，乌梅汤送下；腰痛，牵牛汤送下；伤寒，葱白汤送下。
制备方法：上为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌一切热物，孕妇不可服。
附注：用法用量中“血红”二字，《准绳·类方》作“血昏”。
