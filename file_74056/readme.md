序号：74072
名称：填睛丸
组成：石决明1枚（净洗，别捣），白阳起石（饭上蒸五度，研）2两，磁石（饭上蒸五度，研）2两，陈橘皮（汤浸，去白，焙）2两，栀子花2两，肉苁蓉（去皴皮，切，焙）2两，黑石（饭上蒸五度，研）2两，人参2两，生姜（切，焙）2两，厚朴（去粗皮，生姜汁炙，锉）2两，苦参2两，白芷2两，黄芩（去黑心）2两，甘草（炙，锉）2两，白茯苓（去黑皮）2两，桂（去粗皮）2两，防风（去叉）2两，杏仁（去皮尖双仁，炒，研）2两，升麻8两，生干地黄（焙）8两，龙脑（研）1分，黄连（去须）4两，麦门冬（去心，焙）4两，槐子（炒）4两，黄柏（去粗皮）4两，车前子4两，乳香（研）4两，蕤仁3两，青葙子3两，乌贼鱼骨（去甲并咸味）1两。
出处：《圣济总录》卷一一二。
主治：青盲及内外有胬肉复睛，或吃石药热发，障，或因幼小泪出，或因久视伤明，或因热病愈后，两目俱灰，或因打损，即有胬肉复睛，或吃石药热发，两目作疮，或伤烟火，两目眇视，或两目畏日，远视不辨青赤，或两眦烂疮。
加减：None
功效：None
用法用量：每服6丸，空心米饮送下，服讫即食，食后更服10丸，渐加至20丸，食后即加，食前不加，食后仍以牛乳煎汤下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：二年勿食五辛、热面、陈物，一年勿食羊头肝肚、驴马免肉、毒鱼。
附注：None
