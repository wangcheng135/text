序号：41796
名称：钓藤饮子
组成：钓藤钩子3分，白僵蚕（去丝嘴，炒）1分，川芎1分，蝉蜕（去头翅足，炙）1分，蛇蜕（炙）1分，甘草（炙）1分，蜣螂（去头翅足，炙）3枚。
出处：《永乐大典》卷九八○引《养生必用》。
主治：小儿阴痫，多睡，手足冷，时瘈疭，目直视，乳食不进。
加减：None
功效：None
用法用量：每服2钱，水1盏，加生姜5片，煎至7分，去滓，量儿大小，分作3-4次灌下。若是阴痫，加附子（炮，去皮脐）指面大1块，依前法水煎，温服，日3夜2。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：钩藤饮（《旅舍》）、钩藤饮子（《卫生总微》卷五）、蝉壳饮（《普济方》卷三七一引《卫生家宝》）。
