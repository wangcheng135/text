序号：361
名称：五香连翘汤
组成：连翘3两，蜀升麻2两，熏陆香2两，淡竹沥1升，麝香1分（研），青木香2两，丁香1两，独活2两，寄生3两，射干2两，甘草2两，沉香1两，大黄4两（水1升别渍），朴消2两（熬干别纳）。
出处：《外台》卷二十四引《崔氏方》。
主治：恶疮、热毒肿，恐恶毒气入腹。
加减：None
功效：取利以泄毒气。
用法用量：以水1斗，煮取2升半、绞去滓，然后纳大黄、朴消、竹沥，更煮1-2沸，去滓，纳麝香，分3次温服。服毕相去如人行10里久，得利1-2行为度。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：慎鸡、猪、鱼、蒜、生冷、酢滑、油腻、面食、小豆、五辛、葵菜等。
附注：None
