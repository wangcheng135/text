序号：34011
名称：皂荚并目方
组成：皂荚10梃（不蛀者，捶碎，用水3升浸1宿，挼令浓，滤去滓，以慢火熬成膏），天门冬1两半（去心，焙），枳壳1两（麸炒微黄，去瓤），乌蛇3两（酒浸，去皮骨，炙令微黄），白蒺藜1两（微炒，去瓤），防风1两（去芦头），杏仁1两（汤浸，去皮尖双仁，麸炒微黄），川大黄1两（锉碎，微炒），苦参1两（锉），川升麻1两。
出处：《圣惠》卷五十三。
主治：消渴利后，热毒未解，心神烦热，皮肤瘙痒成疮。
加减：None
功效：None
用法用量：每服30丸，食后以温浆水送下。
制备方法：上为末，入皂荚膏，捣和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“皂荚煎丸”。
