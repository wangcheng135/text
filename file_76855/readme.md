序号：76871
名称：集仙丹
组成：大红三仙丹1两，明净腰黄2两，生漂牡蛎粉1两，飞净石膏4两，广丹1两，飞滑石3两。
出处：《疡科纲要》卷下。
主治：疮疡脓毒未消，恶腐不脱者。
加减：None
功效：提脓拔毒。
用法用量：掺入疮口，不妨略多。
制备方法：上各为细末，和匀听用。
临床应用：None
各家论述：三仙丹提毒化腐，性颇和平，不独脓毒未清恶腐不脱者赖以化毒去腐，即至脓水净尽，新肌盎然，亦可少少用之，即以生肌收口。但金石之性，藉炉火升炼而成，功最捷而吸力亦富，全在研之极细，掺之极匀。若扑药太重，即能作痛，恒有病家知是神丹，索药自掺，往往不知分量，用之大多，反而贻害，又不容靳而不予，致贻吝惜之讥。乃为汇集中正和中之品，俾三仙并行不悖，既有提脓拔毒之效，复无多用增痛之虞，是无法之法，命名集仙，以志其实。
用药禁忌：None
附注：None
