序号：81319
名称：鳖甲丸
组成：鳖甲（去裙襕，醋炙）3分，柴胡（去苗）2两，肉苁蓉（酒浸，切，焙）2两，羌活（去芦头）2两，知母（焙）1两，虎骨（醋炙）1两，常山1两，牛膝（切，酒浸，焙）1两，芍药1两，秦艽（去苗土）1两，附子（炮裂，去皮脐）1两，豉（炒）1两，黄连（去须）1两，乌梅肉（焙）1两半，青蒿1两半，白术1两半，桃仁（去皮尖双仁，炒黄）3两。
出处：《圣济总录》卷八十八。
主治：虚劳。寒热困劣，浑身疼痛无力。
加减：None
功效：None
用法用量：每服20丸，空心、临卧温米饮送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
