序号：30100
名称：赤芍药丸
组成：赤芍药1两半，大黄1两半（锉，炒），吴茱萸1两半（汤洗，焙，炒），干姜1两半（炮），厚朴1两半（去粗皮，生姜汁炙，锉），细辛1两半（去苗叶），牡丹（去心）1两半，芎2两，当归（炒，切）2两，桃仁（汤浸，去皮尖双仁，炒）2两半，附子（炮裂，去皮脐）1两。
出处：《圣济总录》卷一五一。
主治：妇人月水不调，或多或少，脐下块结，痛如锥刺，不治即成劳疾。
加减：None
功效：None
用法用量：每服20丸，渐加至30丸，久冷劳可服至40丸，空心酒送下。觉暖即减丸数。
制备方法：上为末，炼蜜和涂酥为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
