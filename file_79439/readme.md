序号：79455
名称：摩顶立成膏
组成：青莲花2朵，青黛（研）4两，龙脑（研）1两半，石膏（研）1两，麝香（研）1两半，芒消（研）2两，消石（研）2两，凝水石（研）1两，朴消（研）1两，桑寄生5两，莲子草3两，白杨木皮（锉）2两（一方去青莲花，用盐绿）。
出处：《圣济总录》卷十五。
主治：头风肿痒，脑热生疮，目暗赤痛。
加减：None
功效：散热毒。
用法用量：每用2钱，以生麻油研熟，下清水五匙更研，候如粥，用少许于前顶连囟百会两鬓处，涂摩数百遍。
制备方法：以水3升，先煎桑寄生、莲子草、白杨皮，滤去滓，再煎，下朴消、凝水石、消石，减火微煎，候凝取出，晒干细研，入诸药再研令匀，以密器盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
