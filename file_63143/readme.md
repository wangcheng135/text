序号：63159
名称：大温中丸
组成：香附（醋浸）1斤（春、夏1宿，秋、冬2宿），甘草2两，针砂（炒红，醋淬7次）1两，厚朴（姜制）5两，陈皮5两，山楂5两，苍木5两（泔浸），白术2两，茯苓2两，青皮6两，芍药5两，黄连5两，三棱5两，蓬术5两，苦参5两。
出处：《医学六要，治法汇》卷六引丹溪方。
主治：黄病久者。
加减：None
功效：None
用法用量：苍黑筋骨露，气实者，米饮送下50-60丸；肥白气虚者，白术汤送下30-40丸。服7日后，便觉手掌心热凉、口唇内有红晕，半月愈。
制备方法：上为细末，醋为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌一切油腻、生冷、肉、面、鸡、鹅、羊、鸭、糍粽难化之物。
附注：None
