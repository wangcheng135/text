序号：37670
名称：三军上马无敌丸
组成：杜仲（去粗皮）1两（细锉，酒浸，炒去丝），萆薢1两（细锉，酒浸，焙），川牛膝（去芦）1两（酒浸，焙），防风（去芦）1两，木爪1两（淡者好），金毛狗脊（去毛）1两（细锉，微炒），虎胫骨1两（文武火，用酥炙香），苍术1两（切片，酒浸，培干），天麻1两，桃仁1两半，乳香7钱（另研入），没药7钱（另研入）。
出处：《医方类聚》卷二十三引《经验秘方》。
主治：手足因感寒湿，在于行步，经络或冷或痛，骨节酸疼；左瘫右痪，不问久近。
加减：None
功效：None
用法用量：每服50丸，加至80丸，随病上下，食前食后温酒送下。
制备方法：上为细末，搅匀，将原浸药酒，煮面糊为丸，如梧桐子大，晒干。
临床应用：None
各家论述：None
用药禁忌：忌生冷、海味。
附注：None
