序号：2764
名称：天仙藤散
组成：天仙藤（洗，略炒）、香附子（炒）、陈皮、甘草、乌药（不须要天台者，但得软白、香而辣者良）各等分。
出处：《妇人良方》卷十五引陈景初方。
主治：子气。妊娠自3月成胎之后，两足自脚面渐肿腿膝以来，行步艰辛，以至喘闷，饮食不美，似水气状，至于脚趾间有黄水出者。
加减：None
功效：None
用法用量：每服3钱，水1大盏，加生姜3片，木瓜3片，紫苏3叶，同煎至7分，放温澄清，空心、食前服，每日3次。小便利，气脉通，体轻，肿渐消，更不须多服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：香附散(原书同卷)、天仙藤饮(《灵验良方汇编》卷三。)本方方名，《仙拈集》卷三引作“水肿饮”。
