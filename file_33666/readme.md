序号：33675
名称：牡丹大黄汤
组成：大黄4两，朴消4两，牡丹皮3两，桃仁1升，人参2两，阳起石2两，茯苓2两，甘草2两，水蛭2两，虻虫2两。
出处：方出《千金》卷四，名见《千金翼》卷八。
主治：妇人月经不调，或月头，或月后，或如豆汁，腰痛如折，两脚疼，胞中风冷。
加减：None
功效：None
用法用量：上（口父）咀。以水9升，煮取3升，去滓，内朴消令烊尽，分3服。相去如一饭倾。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：此方专取抵当下血为主，加朴消佐大黄，则有推陈致新之功；人参得阳起，有扶阳破明之勣；阳起得牡丹，则破血之力倍胜；茯苓、甘草则又人参之助；牡丹则桃仁之助也。
用药禁忌：None
附注：None
