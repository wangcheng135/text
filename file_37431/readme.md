序号：37441
名称：建中汤
组成：人参2两，甘草2两，桂心2两，茯苓2两，当归2两，黄耆3两，龙骨3两，麦门冬3两，大枣30个，芍药4两，附子1两，生地黄1斤，生姜6两，厚朴1两，饴糖8两。
出处：《千金》卷十九。
主治：虚损少气，腹胀内急，拘引小腹至冷，不得屈伸，不能饮食，寒热头痛，手足逆冷，大小便难，或复下痢口干，梦中泄精，或时吐逆恍惚，面色枯瘁，又复微肿，百节疼痠。
加减：咳者，加生姜1倍。
功效：None
用法用量：上(口父)咀。以水1斗2升，煮取4升，去滓，纳饴糖，每服8合，日三夜一。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：用黄耆建中参入当归、参、附峻补元阳，兼培荣气，加门冬、地黄以滋津液，厚朴、茯苓以泄滞气，龙骨以收耗散之精。
用药禁忌：None
附注：本方方名，《普济方》引作“附子汤”。
