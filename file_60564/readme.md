序号：60579
名称：通经益母丸
组成：益母草8两（用上截），香附米3两（七制），桃仁3两（去皮尖双仁，晒干，麸炒），红花3两（酒炒），当归4两（酒洗），白芍4两（酒炒），白术4两（土炒），白茯苓4两（乳拌，蒸透），粉甘草3两（蜜水拌炒），陈皮3两，丹皮3两（去骨），丹参3两（酒洗）。
出处：《履霜集》卷二。
主治：积块经闭者。
加减：None
功效：养血破积。
用法用量：病轻者，日用1丸，研末热黄酒送下，或蜜汤送下；有痰者，姜汤送下；病甚者，朝、夕各1丸，以愈为度。或丸如绿豆大，每服3钱。
制备方法：上为末，炼蜜为丸，每丸重3钱，晒干收用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
