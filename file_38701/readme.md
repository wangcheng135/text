序号：38711
名称：三阴煎
组成：当归2-3钱，熟地3-5钱，炙甘草1钱，芍药（酒炒）2钱，枣仁2钱，人参随宜。
出处：《景岳全书》卷五十一。
主治：肝脾虚损，精血不足，及营虚失血。凡中风，血不养筋，及疟疾汗多邪散而寒热犹不能止者；产后阴虚发热，怔忡恍惚。
加减：如呕恶者，加生姜3-5片；汗多烦躁者，加五味子14粒；汗多气虚者，加黄耆1-2钱；小腹隐痛，加枸杞2-3钱；如有胀闷，加陈皮1钱；如腰膝筋骨无力，加杜仲、牛膝。
功效：None
用法用量：水2钟，煎7分，食远服。
制备方法：None
临床应用：失眠：赵某，女，24岁。患失眠证月余，服多种中西药罔效。诊见：每晚不易入睡，入睡即多作恶梦而醒，醒后再难入眠，神疲乏力，心悸纳少，两目干涩，食后腹胀，时有便溏，舌质淡，苔薄白，脉细数。证属肝脾阴虚型失眠。治宜补脾、养肝、安神。方用三阴煎加黄芪、远志，水煎服，每午、晚各服一次。继服11剂疾愈。
各家论述：None
用药禁忌：None
附注：None
