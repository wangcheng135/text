序号：58589
名称：秘传加减平胃散
组成：苍术（米泔水浸，去皮）、白术、白茯苓、甘草（炙）、陈皮（去白）、砂仁、猪苓、泽泻。
出处：《松崖医径》卷下。
主治：脾胃虚弱，饮食所伤，及风寒暑湿之气所袭而致泄泻。
加减：泻如清水，脉来无力，加炮干姜、肉桂，甚不止，加制附子；泻如痢，黄赤稠粘，或乍泻口渴，脉来无力，加黄连、黄芩、炒干姜少许；泻而腹痛，右关脉来有力，加草果、枳实、山楂；泻，腹痛或呕吐，加木香磨姜汁服；泻，小便短赤，脉沉，加滑石、灯心；泻，腹如雷鸣，加煨生姜5大片；久不止，脉来无力，加人参、黄耆，甚不止者，加升麻、炒白术、苍术。
功效：None
用法用量：用水2盏，加生姜、大枣、灯心，煎至8分，去滓，食远服。
制备方法：上细切。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
