序号：28743
名称：十力大补丹
组成：防风1斤（醋煮，晒干为末），黑豆1斤（炒燥，入酒内浸1宿，取出晒干如前炒，又入酒浸3次，晒干为末），川乌半斤（不去皮），苍术1斤（米泔水浸1宿，洗、刮去皮，净晒干为末。以黑豆、防风、苍术末3味将纸裹在一处），草乌1斤1钱（不去皮，锉碎，用姜汁浸软，捣为泥），生姜1斤（为泥），葱白1斤。
出处：《普济方》卷一一五。
主治：诸般风疾，口眼斜，手脚肿痛，浑身拘急瘫软，半身不遂，头风脚气，下元虚冷无力。
加减：None
功效：None
用法用量：每服1丸，临卧睡时咬破，用温酒吞下，姜汤亦可。后加人参4两，当归4两，青黛4两。
制备方法：上用川乌、草乌、葱、姜捣碎，用皮纸裹，露1宿后，入新瓦罐内盛，紧裹器口，春5日，夏3日，秋7日，冬10日，如是日毕取出，入前3味末和匀，用糯米糊为丸，如团鱼子大许。
临床应用：None
各家论述：None
用药禁忌：服药后，一切热物汤酒不可食，此药麻人，大忌房事。
附注：None
