序号：13253
名称：人参丸
组成：人参半两（去芦头），白术1两，桂心1两，枳壳1两（麸炒微黄，去瓤），旋覆花（生干）1两，半夏1两（汤洗7遍去滑），厚朴1两（去粗皮，涂生姜汁，炙令香熟），赤茯苓1两，前胡1两（去芦头），木香半两，陈橘皮1两（汤浸，去白瓤，焙），川大黄1两半（锉碎，微炒），槟榔1两。
出处：《圣惠》卷四十三。
主治：心痛，痰饮多唾，心腹胀满，不能下食。
加减：None
功效：None
用法用量：每服20丸，不拘时候，以生姜、橘皮汤送下。
制备方法：上为末，炼蜜和捣200-300杵为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
