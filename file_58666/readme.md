序号：58681
名称：秘传神效红丸子
组成：巴豆49粒（去壳心膜及油，存性，炒热，于银石器内，慢火熬令黄色，乳细，以纸包压去尽油，细研，冬月少去油），杏仁15粒（须是真者，去皮尖，蒸，纸裹，去油，和巴豆同再研），木香3钱半（慢晒，勿见火），沉香1钱半，滴乳香3钱（为细末），槐花1钱半（为细末）。
出处：《医方类聚》卷一四一引《经验秘方》。
主治：肠胃虚弱，内受风寒，或饮食生冷，伤于脾胃，呕吐泄泻，脐腹疼痛，胁肋胀满，肠内虚鸣，积痢赤白，并酒积痢。
加减：None
功效：None
用法用量：每服10丸，加至15丸，白汤、姜汤任下；米饮亦可：如常饮食停滞膨胀，热汤下；温酒亦可。
制备方法：上相和，于乳钵内乳令匀细；用真黄蜡1两1钱，于银石器内，慢火化开，将好绵滤净，再将蜡溶开，却投前药于内，将竹篦不住手搅令匀；次下真杭州黄丹5钱，再搅，令极匀为度，倾出，碗内自凝，捏成块，如硬，入蜜少许，如不红，加黄丹；如用旋丸，如绿豆大，用油纸密包。
临床应用：None
各家论述：None
用药禁忌：小儿泄泻不可服。
附注：None
