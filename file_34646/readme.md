序号：34656
名称：鸡肉煎丸
组成：宣连2两，银柴胡1两，芜荑半两，鹤虱（川）半两，秦艽（净）1两，知母1两，使君子1两，子芩1两。
出处：《幼幼新书》卷二十六引《家宝》。
主治：小儿疳劳，骨蒸潮热，盗汗瘦弱，腹急面黄，食不生肌肉。
加减：None
功效：None
用法用量：每服10-20丸，空心麦门冬汤送下。15岁以上，温酒送下。
制备方法：上为末，黄雌鸡一只，重一斤，专以大麻子饲五日，开臀后去肠肚，洗，拭干入药，线缝。黑豆铺甑底，厚三寸，安鸡四旁及上，又以豆裹，日出时蒸，至晚取药。用鸡净肉和研，如干，入酒糊为丸，如麻子大，或如绿豆大。
临床应用：None
各家论述：None
用药禁忌：忌猪肉。
附注：蒸鸡丸（《玉机微义》卷五十）。
