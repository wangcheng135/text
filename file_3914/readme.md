序号：3914
名称：无比丸
组成：京三棱（煨，锉）1两，牵牛子1两，胆矾（研）1两，槟榔（锉）1两，芫花（醋浸，炒）1两，腻粉1分，续随子（去皮）半两，硇砂（研）半两，木香半两，铁粉（研）3分，大枣30枚（汤内略煮过，剥去皮核，取肉烂研）。
出处：《圣济总录》卷八十。
主治：水蛊，通身肿满。
加减：None
功效：None
用法用量：每服10丸，丈夫温酒送下；妇人醋汤送下。
制备方法：上药除胆矾、硇砂、枣肉外，同捣罗为末，用酽醋2大升，先下硇砂、胆矾、枣肉于银石器内，煎5、7沸，次下诸药末，1处搅匀，慢火熬，候可丸，即丸如豌豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
