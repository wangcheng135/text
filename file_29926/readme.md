序号：29934
名称：赤龙丸
组成：荆芥2两半，草乌2两半，破故纸（炒）2两半，羌活1两，白芷1两，乌豆1两，川牛膝1两，黑牵牛（炒）半两，茴香（炒）半两，紫金皮半两，川萆薢半两，川芎7钱半，木瓜3钱，独活1两半。
出处：《经验秘方》引江大丞方（见《医方类聚》卷二十三）。
主治：诸般风瘫，偏正头风，风眼牙痛，气滞腰痛，腿脚肿痛，风湿脚气，走注风，鸡爪风，皮肤燥痒，风疮下疰，破伤风，瘰疬，胎前产后诸风，痈疽肿痛，气血凝滞，肾经湿痒，筋骨拘挛，肠风痔漏，风痰喘嗽，口眼斜，诸痫，小儿惊风。
加减：None
功效：None
用法用量：每服20丸，诸般风瘫痪，黑豆汤送下；偏正头风，川芎茶送下；风眼牙痛，薄荷汤送下；气滞腰痛，茴香汤送下；腿脚肿痛，木瓜酒送下；风湿脚气，木瓜酒送下；走注风，防风酒送下；鸡爪风，麝香酒送下；皮肤燥痒，火麻子研酒送下；风疮下疰，赤豆酒送下；破伤风，乳香酒送下；瘰疬，荆芥茶送下；胎前、产后诸风，当归酒送下；痈疽肿痛，生姜酒送下；气血凝滞，生地黄研酒送下；肾经湿痒，飞过盐沸酒送下；筋骨拘挛，松节酒送下；肠风痔漏，槐花酒送下；风痰喘嗽，生姜半夏汤送下；口眼斜，木香、前胡酒送下；诸痫，朱砂、麝香酒送下；小儿惊风，朱砂、薄荷酒送下。
制备方法：上为细末，酒糊为丸，如梧桐子大，土朱为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
