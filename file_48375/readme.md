序号：48390
名称：神效吹喉散
组成：薄荷5钱，僵蚕5钱，青黛5钱，朴消5钱，白矾5钱，火消5钱，黄连5钱，硼砂5钱。
出处：《外科正宗》卷二。
主治：缠喉风闭塞，及乳娥喉痹，重舌木舌。
加减：None
功效：None
用法用量：吹患上。
制备方法：上为细末，腊月初一日取雄猪胆7-8个，倒出胆汁，用小半和上药拌匀，复灌胆壳，以线札头，胆外用青缸纸包裹，将地挖一孔，阔深1尺，上用竹竿悬空横吊，上用板铺以泥密盖，候至立春日取出，挂风处阴干，去胆皮青纸，瓷罐密收，每药1两加冰片3分，同研极细。
临床应用：None
各家论述：None
用药禁忌：None
附注：神效吹口药（《种福堂方》卷三）、吹喉散（《外科传薪集》）。
