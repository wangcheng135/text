名称：金不换膏
组成：川芎5钱，牛膝5钱，生草乌5钱，香附5钱，红花5钱，橘皮5钱，续断5钱，麻黄5钱，桑枝5钱，细辛5钱，防风5钱，羌活5钱，五加皮5钱，山药5钱，白芷5钱，清风藤5钱，远志5钱，桃仁5钱，白蔹5钱，何首乌（生）5钱，天麻5钱，熟地5钱，当归5钱，生杜仲5钱，桃枝5钱，威灵仙5钱，连翘5钱，穿山甲（生）5钱，乌药5钱，苍术5钱，赤芍5钱，独活5钱，槐枝5钱，荆芥穗5钱，蜈蚣5分，榆枝5钱，僵蚕5钱，苦参5钱，金银花5钱，柳枝5钱，大风子5钱，大黄5钱，生川乌5钱。
出处：《北京市中药成方选集》。
主治：手足麻木，腰腿疼痛，跌打损伤，闪腰岔气。
加减：None
功效：散风，活血、止痛。
用法用量：微火化开贴患处。
制备方法：上药酌予碎断，用香油240两炸枯，过滤去滓，炼至滴水成珠。入黄丹100两搅匀成膏，取出浸入水中去火毒后加热熔化。另兑血竭6钱，乳香6钱，没药6钱，樟脑6钱，轻粉6钱，5味共为细粉。每240两膏油兑以上细粉搅匀摊贴。大张油重6钱，中张油重4钱5分，布光。
