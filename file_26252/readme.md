序号：26260
名称：羊肝饼
组成：黑羯羊肝1具（去筋膜，切成方寸块，中间割开相连），白术1两，（小米泔浸1宿，切成咀，陈壁上炒黄色，为细末1两），左顾大牡蛎1个（重1斤者，炭火煅通红，候冷，为细末1两），真黄蜡1两（溶化开，入前药2味搅匀，乘热成饼，照肝块数目如肝块大，其饼重2钱，小者重1钱5分）。
出处：《医便》卷五。
主治：小儿惊积，左胁下有块；女人血痕，发热瘦弱。
加减：None
功效：None
用法用量：任小儿食之，1次2-3块。夏月将饼系于井中,勿令色变味臭，小儿不肯食也。重者不过1肝2肝，轻者数块则热止，7日后则积消腹软矣。
制备方法：上将蜡饼夹于肝内，用竹叶包裹，以线缚之，入新砂锅中，以水淹1寸，入粟米5-6合煮，以米熟为度，候冷，去竹叶。
临床应用：None
各家论述：None
用药禁忌：若积块在右，为食积，不宜吃此饼。
附注：None
