序号：56274
名称：流脑合剂
组成：生石膏60g，鲜生地60g，知母15g，连翘15g，大青叶30g，丹皮12g，黄连12g，黄芩12g，赤芍9g，淡竹叶9g，桔梗9g，甘草9g，水牛角120g（先煎，取汁200ml）。
出处：《古今名方》引江西中医学院方。
主治：流行性脑脊髓膜炎，证属气血两燔者，见有高热，头痛剧烈，呕恶肢痛，颈项强直，咽痛或红肿，皮肤出血点较明显，舌绛，脉数。
加减：None
功效：清热解毒，凉血救阴。
用法用量：先将石膏、大青叶煎汤代水，合牛角汁再煎诸药；先后煎2次，共煎药液200-400毫升，分3次服。1昼夜可连服2-4剂。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
