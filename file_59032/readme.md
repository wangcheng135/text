序号：59047
名称：秦艽散
组成：秦艽1两（去苗），赤箭1两，独活1两，桂心1两，五加皮1两，磁石3两（捣碎，水淘，去赤汁），甘菊花1两，汉防己1两，羚羊角屑1两，葛根1两（锉），赤芍药1两，麻黄2两（去根节），薏苡仁2两，防风1两（去芦头），芎藭1两，侧子1两（炮裂，去皮脐），杏仁2两（汤浸，去皮尖双仁，麸炒微黄），甘草1两（炙微赤，锉）。
出处：《圣惠》卷二十。
主治：瘫风，手足不遂，肌肉顽痹，筋脉拘急，心神不安，言语謇涩，胸膈痰涎不利。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，煎至6分，去滓温服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
