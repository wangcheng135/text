序号：70719
名称：紫雪玄霜丹
组成：飞金100页，石膏3斤，寒水石3斤，磁石3斤，滑石3斤，犀角屑5两，青木香5两，羚羊角屑5两，沉香5两，玄参3斤，升麻1斤，甘草8两，丁香1两。
出处：《全国中药成药处方集》（沈阳方）。
主治：疮毒内攻，温病热入脏腑，脚气，毒遍内外，口疮，吐血衄血，热淋发狂，小儿毒热惊痫等热性症。
加减：None
功效：清热解毒，镇惊。
用法用量：每服1-3丸，食后薄荷汤送下。
制备方法：以上清水煮，去滓，投木盆中，半日欲疑，冷2-3日，候凝成霜，紫色铅罐收贮，加麝香1两2钱5分，朱砂3两，水飞如法，调制为丸，重3分。
临床应用：None
各家论述：None
用药禁忌：忌食发物。
附注：None
