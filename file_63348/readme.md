序号：63364
名称：清带汤
组成：生山药1两，生龙骨6钱（捣细），生牡蛎6钱（捣细），海螵蛸4钱（去净甲，捣），茜草3钱。
出处：《衷中参西》。
主治：妇女赤白带下。
加减：单赤带，加白芍、苦参各2钱；单白带，加鹿角霜、白术各3钱。
功效：None
用法用量：None
制备方法：None
临床应用：1.白带：一妇人，年20余，患白带甚剧，医治年余不愈。后愚诊视，脉甚微弱。自言下焦凉甚，遂用此方，加干姜6钱，鹿角霜3钱，连服10剂全愈。_x000D_
2.赤白带：一媪年六旬，患赤白带下，而赤带多于白带，亦医治年余不愈。诊其脉甚洪滑，自言心热头昏，时觉眩晕，已半载未起床矣。遂用此方，加白芍6钱，数剂白带不见，而赤带如故，心热，头眩晕亦如故，又加苦参、龙胆草、白头翁各数钱。连服7-8剂，赤带亦愈，而诸疾亦遂全愈。
各家论述：此方用龙骨、牡蛎以固脱；用茜草、海螵蛸以化滞；更用生山药以滋真阴固元气。愚拟此方，则又别有会心也。尝考《神农本草经》龙骨善开癥瘕，牡蛎善消鼠瘘，是二药为收涩之品，而兼具开通之力也；乌鲗骨即海螵蛸，茹芦即茜草，是二药为开通之品，而实具收涩之力也。四药汇集成方，其能开通者，兼能收涩；能收涩者，兼能开通，相助为理，相得益彰。
