序号：62380
名称：旋覆花汤
组成：旋覆花1两，前胡（去芦头）半两，甘菊花（未开者）1两半，防风（去叉）1两，生干地黄（洗，切，焙）1两，羌活（去芦头）1两，杏仁（汤浸，去皮尖双仁，炒）1两，玄参半两，白僵蚕（炒）半两，黄芩（去黑心）半两，半夏（去末，姜汁作饼，晒干）半两，白术半两，藁本（去苗土）半两，甘草（炙，锉）半两，当归（切，倍）半两，人参半两，赤茯苓（去黑皮）半两。
出处：《圣济总录》卷十三。
主治：热毒风上攻，头旋倒仆．或吐不止，畏见日光，不喜喧处，不欲饮食．时时发动。
加减：None
功效：None
用法用量：每服5钱匕，以水1盏半，煎取1盏，去滓。食后良久服，1日2次。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
