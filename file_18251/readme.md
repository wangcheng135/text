序号：18258
名称：白术丸
组成：白术1两，黄耆1两（锉），牡蛎1两（烧为粉），人参1两（去芦头），赤茯苓1两，川乌头1两（炮裂，去皮脐），干姜半两（炮裂，锉），木香1两，当归1两（锉，微炒），赤芍药3分，桂心1两，甘草半两（炙微赤，锉），防葵半两，鳖甲1两（涂醋炙令黄，去裙襕），紫菀半两（去苗），槟榔1两，桔梗半两（去芦头），枳壳1两（麸炒微黄，去瓤）。
出处：《圣惠》卷四十八。
主治：积聚，宿食不消，腹胁下妨闷，四肢羸瘦，骨节痠疼，多有盗汗,血臌。
加减：None
功效：None
用法用量：每服30丸，于食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
