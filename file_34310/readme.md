序号：34319
名称：万灵膏
组成：清油8两，妇人油发1两，侧松枝1两，沥青1两，卷柏1两，太平白芷1两，当归1两，木鳖肉（切片）1两，柏枝香1两，白胶香1两，没药1两，乳香1两，黄丹1两，剪刀草末1两。
出处：《传信适用方》卷三。
主治：痈疽疮疖。
加减：None
功效：None
用法用量：先以葱盐汤洗疮，次用软帛火炙摊上，不得留孔，贴之。
制备方法：用净铛1只，下清油、妇人油发炭火熬令焦尽；次下侧松枝等，熬令紫色，用杨柳枝不住手搅，焦即锉头，以绵两重滤去滓；再熬油滚，依次下白胶香、没药、乳香、黄丹、剪刀草末，慢火熬成膏，滴水成珠子为度，就地用新汲井水沉去火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
