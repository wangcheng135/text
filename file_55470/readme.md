序号：55485
名称：丁香丸
组成：丁香2两，硇砂2两，木香2两，桂（去粗皮）2两，附子（炮裂，去皮脐）2两，干漆（炒烟出）2两，蓬莪术（煨，锉）2两，乳香（研）2两，青橘皮（汤浸，去白，焙）2两，京三棱（煨，锉）2两，墨大黄（锉，炒，捣末）2两，巴豆霜2两，芫花（醋炒焦）2两，猪牙皂角（去皮子，炙）2两，没药（研）2两，干姜（炮）2两。
出处：《圣济总录》卷四十四。
主治：脾胃虚寒，宿食不消。
加减：None
功效：None
用法用量：每服3丸至5丸，温酒或热水送下。得微利佳。
制备方法：上药除硇砂、大黄、巴豆霜、乳香、没药外，捣罗为末，以酽醋1升，化硇砂去滓，入银石器中，慢火煎之；次下巴豆霜搅匀，两食久；次下大黄末，熬成膏，拌诸药，更入醋煮面糊，和捣1000杵，为丸如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
