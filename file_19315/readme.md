序号：19322
名称：地黄丸
组成：玄参5分，熟地7分，当归7分，黄连1钱，大黄（煨熟）1钱，犀角1钱，木贼1钱，白蒺藜1钱，沙苑蒺藜1钱，蝉蜕1钱，谷精草1钱，羌活1钱，防风1钱，甘草1钱5分，生地1钱5分，木通1钱5分。
出处：《痘学真传》卷七引海藏方。
主治：痘后目疾生翳。
加减：None
功效：None
用法用量：口服。
制备方法：上为末，炼蜜为丸服。
临床应用：None
各家论述：二地、当归、沙苑以养血滋肾，二黄、犀角、玄参、木通以清火润下，木贼、蒺藜以消翳，蝉退、谷精以轻扬，羌活疏风，甘草和药，凡痘后目疾生翳，皆从血虚火炎，养血清火治其本也，消翳、轻扬、疏风治其标也。
用药禁忌：None
附注：None
