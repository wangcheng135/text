序号：3929
名称：无比散
组成：土瓜根8两，苦参粉3两，黄连5两（去毛），鹿茸3两（炙），栝楼3两，雄鸡肠3具，牡蛎5两（熬），白石脂3两（研），甘草3两（炙），黄耆3两，桑螵蛸3-7枚（炙），白龙骨5两（研），鸡肶胵30具，黄皮30具（熬）。
出处：方出《外台》卷十一引崔氏方，名见《普济方》卷一七七。
主治：消渴。
加减：None
功效：None
用法用量：每服6方寸匕，日2服，夜1服；用竹根10两，麦门冬4两(去心)、石膏4两，甘李根白皮3两，以水1斗2升，煮取3升五合，以下前散药。如难服，可取此药汁为丸，1服60丸，仍用此药汁下之。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：忌猪肉、海藻、菘菜。
附注：None
