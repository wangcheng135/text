序号：3903
名称：无上仙丹
组成：轻粉1两，水银3钱，朴消1两，樟脑5钱，石膏1两。
出处：《外科十三方考》。
主治：阴阳痈疽，久不收口，出脓水者。
加减：None
功效：None
用法用量：凡遇阴阳10恶大症久不愈者，将此丹少许撕于疮口，不日即可痊愈；若疮口久久不收者，加猫头骨灰少许于丹内和匀，撒于疮口。
制备方法：上为极细末备用，用时用井水调和，涂于碗内，复于阴处，候干透用竹片刮下，再研细备用。另用大碗1只，以皮纸糊于碗口上，将药粉摊撒纸上，再用艾绒盖于药上约半纸厚，四面点火，同时将艾燃着，药面见热即行降下，候冷吹去艾灰，将纸撕去，药在碗底，刮下收入瓷瓶，严密紧塞。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
