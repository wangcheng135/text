序号：61043
名称：铁柱杖丸
组成：草乌头1两（生，为细末）。
出处：《烟霞圣效方》引临清张先生方（见《医方类聚》卷二三八）。
主治：妇人产后血风，腰腿疼痛。疔疮发背，头风。
加减：None
功效：None
用法用量：每服7丸，食前以温酒送下；妇人产后，用温醋汤送下，每日2次。
制备方法：用葱白2根烧熟，去粗皮，将药末同溲得所，于臼内捣极烂为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：铁柱杖（《丹溪心法附余》卷十六）。《丹溪心法附余》本方用法：用葱白去须叶捣烂为丸，如豌豆大，以雄黄为衣。每服一丸，先将葱细嚼热酒送下，或有恶心吐三四口，用冷水一口止之，即卧以被厚盖，汗出为度。
