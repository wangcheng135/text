序号：44710
名称：封口药
组成：乳香1钱，没药1钱，儿茶1钱，当归1钱，杉皮灰1钱，麝香5厘，片脑1分，猪母聍叶1钱（如无此叶，葛叶、毛藤子叶亦可）。
出处：《准绳·疡医》卷六。
主治：刀斧伤，割喉断耳缺唇，伤破肚皮，跌破阴囊皮碎。
加减：None
功效：None
用法用量：如缺唇，先以小气针作三截针之，用绢线1条，两头搓猪毛，以唾蘸湿，抹封口药于线上，将药线三截穿定，却以麻药抹缺处，以剪刀口抹封口药，薄剪去些皮，以线即缝合，就以鸡子黄油搽患处，以金毛狗脊毛薄铺于上，却以封口药末搽于上，每日用药水轻洗去，搽油换药，每日1次，待8日剪去线，搽药。
制备方法：上各另研细末，和匀，入麝香，次入片脑，再和匀，瓷器收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
