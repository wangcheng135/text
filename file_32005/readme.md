序号：32014
名称：万生丸子
组成：沉香2分，藿香2分，丁香2分，青橘2分，牵牛子（炒）2分，白檀1分，海蛤1分，瞿麦1分，豆蔻1分，大戟3分（炙）。
出处：《医方类聚》卷二二七引《川玉集》。
主治：妇人妊娠、下死胎后，头面两眼并肿，脚胫细瘦，胸背疼闷，肚肿，脐下痛，旦晨惺惺，午后四肢无力，昏沉如醉，食饮不多，大便不通，小便赤涩，身体枯悴。
加减：None
功效：None
用法用量：每服5丸至7丸，粥饮送下。大肠取下。患未愈，隔日更再服之。依前下大小肠恶物后，服补虚散治之愈。
制备方法：上为末，每1两药末，用巴豆5个，于生铁铫子内麸炒令黄，研如泥，更入腻粉1钱相和，炼蜜为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：忌生冷、面、盐、毒物五十日。
附注：None
