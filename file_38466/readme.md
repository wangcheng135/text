序号：38476
名称：治毒紫霞丹
组成：西牛黄1钱2分，藤黄（制净）1两2钱，大黄1两2钱，参山漆1两2钱，天竺黄6钱，明腰黄6钱，粉儿茶6钱，梅冰片1钱2分，阿魏3钱，没药（去油）1两2钱，血竭6钱，麝香1钱2分，乳香（去油）1两2钱。
出处：《全国中药成药处方集》（杭州方）。
主治：痈疽发背，无名肿毒，横痃结核，或已成无肿，或漫肿不溃，以及跌打损伤。
加减：None
功效：化脓解毒，祛瘀生新。
用法用量：每服1丸，重则加倍，用温酒化服。外用浓茶汁抹敷患处。
制备方法：上各取净末，炼蜜为丸，每丸潮重5分，金箔为衣，蜡壳封固。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
