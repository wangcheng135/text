序号：76880
名称：集成定痫丸
组成：人参1两（切片，焙干），漂白术1两5钱（切片，土炒），白云苓（切片，姜汁拌蒸，晒干）1两，直广皮（酒炒）1两，法半夏1两，石菖蒲5钱（取9节者，切片），白当归1两（酒洗，晒，切），青化桂5钱（去木皮，津浔桂不用），杭白芍1两（酒炒），白蔻仁（酒炒）1两，漂苍术（用黑芝麻拌炒）1两，南木香5钱（忌火），真龙骨1两（火煅醋淬研末，水飞过，晒干，取5钱），赤金箔30张，镜面砂3钱（研末，水飞，晒干，研用）。
出处：《幼幼集成》卷二。
主治：小儿痫症，偶然有触则昏晕卒倒，良久苏者。
加减：None
功效：None
用法用量：每服1丸，姜汤化下，1日2次。痫症未久者服此，倘年深日久者，早服河车八味丸，午、晚服此。
制备方法：上为细末，炼蜜为丸，如龙眼核大，以朱砂为衣，贴以金箔，晒干，以瓷瓶收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
