序号：38103
名称：松子丸
组成：松子（去皮）1两，白茯苓1两，麦门冬（去心，焙）1两，柏子仁1两，薯蓣1两，枸杞子1两，五味子1两，菌桂1两，山茱萸1两，覆盆子1两，熟干地黄1两，泽泻1两，女贞实1两，石南1两，黄耆（锉）1两，远志（去心）1两，肉苁蓉2两（酒浸1宿，括去皱皮，炙令干），石斛1两半（去根，锉），杜仲1两半（去皱皮，炙令黄），甘草半两（炙微赤，锉）。
出处：《圣惠》卷二十七。
主治：虚劳百病，绝伤羸瘦。
加减：None
功效：久服强筋骨，长肌肉，悦泽颜色，延年不老。
用法用量：每服30丸，空心及晚食前温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
