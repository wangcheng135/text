序号：82249
名称：瘰疬妙方
组成：官粉1两半，乳香2钱，没药2钱半，孩儿茶2钱半，蛤粉5钱，龙骨2钱半，蜂房2个，密陀僧2钱半，血竭2钱，蓖麻子（去壳）120个。
出处：《古今医鉴》卷十五。
主治：瘰疬。
加减：None
功效：None
用法用量：贴患处，如神。先用养麦面作圈，围住疮上，用黄酒糟压干撒在疮上，用麝香入艾捶烂，铺糟上，火烧，艾过则再换，以疮内水干为度，后贴膏药。
制备方法：上为细末，用香油4两熬黑色，后将各药收在油内，熬数沸用瓦盆盛水，将药锅坐在上，出火毒，纸摊。
临床应用：None
各家论述：None
用药禁忌：忌食、鸡、鹅、羊肉、蛋、鲜鱼、辛辣、炙煿等物。
附注：None
