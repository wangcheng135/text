序号：58810
名称：秘传痔漏内消丸
组成：棕榈灰1两，发余2两，枯矾1两，刺猬皮1个（醋炙），牛角（角思）1个（烧存性），防风5钱，槐角5钱，黄连（炒）1两2钱，白术（炒）1两，黄芩（炒）5钱，地榆（炒黄）5钱，川山甲尾8钱（土炒成珠），苦参皮1两，石菖蒲1两，猪蹄壳20个（烧灰），雷丸1两，胡麻仁1两，漏芦1两，芜荑1两，青姜香5钱，象牙末5钱，乳没3钱。
出处：《医林绳墨大全》卷九。
主治：痔漏。
加减：None
功效：干脓止痛，退管生肌。
用法用量：每服1钱，每日2-3次。
制备方法：上为末，加麝5分，用黄蜡4两，加麻油1两，和药为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
