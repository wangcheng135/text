序号：8877
名称：水仙丹
组成：朱砂不拘多少（细研，水飞过，候干），木通（令为细末）1两，白及1两（锉，用麻油一小盏同入铫子内煎，令药焦黑色为度，去药，更煎油良久，以木箸点油向冷水中，成花子不散。是成。如未，更煎良久。倾入盏内收之）。
出处：《杨氏家藏方》卷九。
主治：水火不足，精神恍惚，怔忪健忘，遗精白浊，小便淋沥，消渴，吐血、衄血、溺血，及虚烦发热。
加减：None
功效：None
用法用量：每服2-7丸，空心新水送下。其浸药水1日1换。
制备方法：上药将煎来油和研细朱砂、木通末，看多少和如软面剂相似，用浓皂角水洗药剂数遍，令油尽，却以清水浸之。每日旋丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
