序号：74704
名称：辟秽驱毒饮
组成：西牛黄8分（研，冲），人中黄3钱，九节菖蒲5分，靛叶1钱半，忍冬蕊5钱（鲜者蒸露亦可），野郁金1钱。
出处：《集成良方三百种》卷中。
主治：鼠疫。秽毒内闭，毒气攻心，病者发头疼，四肢倦怠，骨节禁锢，或长红点，或发丹疹，或呕或泻，舌干喉痛，间有猝然神昏，痰涌窍闭。
加减：如见结核，或发斑，或生疔，加藏红花8分，单桃仁3钱，熊胆4分送服；大渴引饮，汗多，加犀角，金汁；神昏谵语，宜用至宝丹，或安宫牛黄丸，开水和服，先开内窍。
功效：芳香辟秽，解毒护心。
