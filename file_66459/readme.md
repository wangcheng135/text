序号：66475
名称：小儿百寿丹
组成：山楂5两，胆星2两5钱，滑石5两，竺黄2两5钱，苍术（炒）2两5钱，木香2两5钱，砂仁1两5钱，六神曲（炒）1两5钱，麦芽（炒）1两5钱，钩藤1两5钱，薄荷1两5钱，僵蚕（炒）1两5钱，茯苓1两，桔梗1两，甘草1两，橘皮2两5钱。
出处：《北京市中药成方选集》。
主治：停食发烧，消化不良，咳嗽痰盛，惊风内热，感冒风寒。
加减：None
功效：清热散风，化滞消食。
用法用量：每服1丸，温开水送下。
制备方法：上为细末，再兑朱砂1两，牛黄2钱，研细过罗，混合均匀，炼蜜为丸，每丸重8分，金衣36开，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
