序号：25994
名称：芎汤
组成：芎2两，干地黄2两，黄耆2两，芍药2两，吴茱萸2两，甘草2两，当归3两，干姜3两。
出处：《千金》卷四。
主治：带下、漏血不止；风虚冷热，劳损冲任，月水不调，崩中暴下，腰重里急，淋沥不断；及产后失血过多，虚羸腹痛；或妊娠胎动不安，下血连日，小便频数，肢体烦倦，头晕目暗，不欲饮食。
加减：若月经后因有赤白不止者，除地黄、吴茱萸，加杜仲、人参各2两。
功效：None
用法用量：以水1斗，煮取3升，分3服。
制备方法：None
临床应用：None
各家论述：1.《千金方衍义》：芎、归、地、芍虽专调血，不得甘、耆不能助卫和营。尤妙在姜、萸之辛温，使血无阻积，得以归经。而经后赤白不止，非但不可用茱萸之浊燥，即地黄之腻滞亦宜斟酌，故退二位而进杜、参以益气，精气充而血液固矣。_x000D_
2.《医略六书》：寒湿袭虚，经气不摄，故腹痛不止，崩且传漏焉。熟地补阴滋血，黄耆补气摄血，芎行血中之气以除腹痛，白芍敛失位之血以止崩漏，吴萸温中止痛，炮姜温中止血，当归行血归经，炙草暖中和胃。水煮微温服，使_x000D_
寒湿外散，则经气内充，而冲脉完固，何有_x000D_
腹痛不止，崩且传漏之患哉！
用药禁忌：None
附注：芎温中汤（《奇效良方》卷六十三）。本方方名，《普济方》引作“温中芎汤”。
