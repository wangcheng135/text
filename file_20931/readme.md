序号：20939
名称：人参散
组成：人参1两（去芦头），白茯苓1两，黄耆1两半（锉），熟干地黄1两半，肉苁蓉2两（酒浸1宿，刮去皱皮，炙干），五味子1两，附子1两（炮裂，去皮脐），陈橘皮3分（汤浸，去白瓤，焙），半夏3分（汤浸7遍去滑），柴胡1两（去苗），桂心1两，白术3分。
出处：《圣惠》卷十四。
主治：伤寒重病后，四肢沉困，肢体酸疼，翕翕少气；或两胁拘急，腰背强直，面少颜色，不能饮食，渐至虚羸。
加减：None
功效：None
用法用量：每服5钱，以水1中盏，加生姜半分，大枣5枚，煎至5分，去滓，不拘时候，稍热服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
