序号：59178
名称：胭脂膏
组成：胭脂（生用，为主）、珍珠（生用，研末）、豌豆（烧存性，为末）、头发（烧存性，研末，或炒发出油，取用之）。
出处：《医林纂要》卷九。
主治：贼痘，痘疔。凡报痘后将起胀时，诸痘未起，而有先起虚大如金黄者，名曰贼痘；有大而色黑者，名曰痘疔。
加减：None
功效：除血热壅结。
用法用量：以银簪刺破，口含清水，吸去秽血，用此膏填入疮内，则诸痘自皆红润；或用紫草油亦可。
制备方法：上为末，调入胭脂拌匀，候用。
临床应用：None
各家论述：胭脂以色，豌豆以形，血余以血活血，珍珠以阴和阳，要以除其血热之壅结者而已。去败群之羊，而群羊和矣。
用药禁忌：None
附注：None
