序号：81192
名称：麝香丸
组成：麝香（研）半两，蜥蜴（去头足，微炙）1两，鹳骨（微炒）3寸，羖羊鼻（炙令焦黄）2枚，干姜（炮裂）1两，鸡屎白（微炒）2两，巴豆（去皮心，麸炒出油尽）5枚，芫青（去翅足，微炒）20枚，藜芦（去芦头，微炙）1两，鬼臼（去毛，微炙）1两，丹砂（研如粉）1两，桂（去粗皮）1两。
出处：《圣济总录》卷一○○。
主治：遁尸。经年不愈，心腹刺痛，短气。
加减：None
功效：None
用法用量：每服2丸，空心、以米饮送下。每日2次，稍加至5丸，以吐利为度。
制备方法：上为末，炼蜜为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
