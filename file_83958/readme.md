序号：83976
名称：五子丸
组成：余甘子5两，覆盆子5两（酒浸，焙），菟丝子5两（去浮者，酒浸，蒸熟，焙），五味子5两（炒），车前子5两（酒浸，焙）。
出处：《普济方》卷二二三引《博济》。
主治：None
加减：None
功效：通流五脏，润泽血脉，返老成少，补助元阳，制金石药毒。
用法用量：每日30丸，空心温酒送下。如热，任意加减。
制备方法：上为末，取2-3月间枸杞嫩叶，捣研取汁2大升，和药末，令汁尽为度。又取杏仁1升(去皮尖)，与无灰酒同研，取汁5升，于银石器中煎令杏仁无苦味，然后下地黄汁半升，真酥5两，鹿角胶末5两，同于前汁中略煎过，次下五子末，以柳枝急搅之，慢火熬，可丸即并手丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：守仙五子丸（《遵生八笺》卷十八）。
