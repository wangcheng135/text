序号：47335
名称：祛风至宝丹
组成：防风1两半，石膏1两，川芎2两半，滑石3两，当归2两半，芍药1两半，甘草2两，大黄半两，白术1两3钱，连翘半两，荆芥穗5钱，薄荷叶半两，麻黄半两（去根不去节），山栀子6钱，黄芩1两，芒消半两，桔梗1两，熟地黄1两，天麻1两，人参1两，羌活1两，独活1两，黄连半两，黄柏半两，细辛半两，全蝎5钱。
出处：《杂类名方》。
主治：诸风热等证。
加减：None
功效：发表攻里。
用法用量：每服1丸，临卧细嚼，茶、酒任下。
制备方法：上为极细末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：祛风至宝膏（《法律》卷三）。《古今医鉴》有朱砂。
