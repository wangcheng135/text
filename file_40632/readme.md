序号：40642
名称：诜诜丸
组成：泽兰叶1两半，白术1两半，肉桂（去粗皮）半两，干姜（炮）半两，熟地黄（洗，焙）2两，当归（洗，焙）2两，川芎1两，石斛（酒浸，锉，炒）1两，白芍药1两，牡丹皮（去心）1两，延胡索1两。
出处：《局方》卷九（淳祐新添方）。
主治：妇人冲任虚寒，胎孕不成，或多损坠。
加减：None
功效：None
用法用量：每服50丸，空心温酒送下。
制备方法：上为细末，醋煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：《济阴纲目》汪琪笺：冲为血海，任主胞胎。此方以四物养荣，以白术、石斛养气，泽兰、丹皮、玄胡荡胞中之秽，干姜、肉桂暖子宫之寒。去旧生新，温中益胃，亦温和之正方也。
用药禁忌：None
附注：None
