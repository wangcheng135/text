序号：24120
名称：如意丹
组成：苍术（米泔浸1宿，晒）12两，厚朴（姜汁炒）12两，甘草（去皮）8两，木通（去皮）8两，莪术（醋炒）6两，陈皮12两，三棱（去毛）6两，枳壳（去瓤）10两，
出处：《惠直堂方》卷一。
主治：伤食气滞腹痛，隔食风寒，胸膈饱满，头痛发热，大便不实，或水泻，痢疾，嗳气吞酸噎满，痞积气块作痛等证。
加减：None
功效：None
用法用量：小儿1岁服1分，至15岁服2钱5分，16岁以上服3钱，不能服丸者，可研化服。肚腹痛，枳壳汤送下；食伤气滞腹痛，砂仁汤送下；隔食风寒，胸膈饱满，头痛发热，生姜葱头汤送下；心腹时常作痛，或大便不实，嗳气吞酸作胀，水泻及白痢，生姜汤送下；红痢，甘草汤送下；红白痢，甘草生姜汤送下；痞积气块作痛，生姜汤送下；停食，槟榔汤送下；逆气上升噎满，生姜汤送下；气塞痛，陈皮汤送下；其余诸病，俱滚汤送下。
制备方法：上为细末，将三年陈晚米1斗，巴豆490粒同炒至黄色，拣去巴豆，碾米为末，同前药水泛为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：孕妇忌用。
附注：None
