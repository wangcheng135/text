序号：35071
名称：龟鹿滋肾丸
组成：熟地8两，茯苓4两，阳春砂6钱，苁蓉（炙）2两，补骨脂（炙）1两，菟丝子1两，泽泻3两，当归5两，白术（炒）3两，远志肉（炙）1两，杞子4两，覆盆子3两，芡实（炒）4两，山药4两，莲子肉5两，丹皮3两，山萸肉（炙）3两，牛膝2两，杜仲炭4两，枣仁（炒）2两，人参（去芦）8两，鹿茸（去毛）3两，龟版胶3两，鹿角胶3两。
出处：《北京市中药成方选集》。
主治：肾气虚弱，阳痿精冷，夜寐多梦，遗精盗汗。
加减：None
功效：滋阴补肾，添精益髓。
用法用量：每服2钱，温开水送下，1日2次。
制备方法：上为细末，炼蜜为小丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
