序号：72685
名称：款气丸
组成：木香3分，陈橘皮（汤浸，去白，切，焙）3分，枳壳（去瓤，麸炒）3分，郁李仁（麸炒，去皮）3分，吴茱萸（汤洗7遍，炒）半两，桃仁（去皮尖双仁，麸炒）1两，京三棱（煨，锉）1两半，桂（去粗皮）半两，槟榔（锉）1两半，赤芍药1两，赤茯苓（去黑皮）1两，黑牵牛5两（拣净，捣罗为末）2两半。
出处：《圣济总录》卷六十七。
主治：气逆攻冲，肩膊拘急，或胁肋胀满，大便秘涩，手臂头面浮肿。
加减：None
功效：None
用法用量：每服20丸，空心温酒送下；生姜汤亦得。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
