序号：643
名称：五噎膈气丸
组成：半夏（汤浸7次，薄切，焙）2两，桔梗2两（炒），肉桂（不见火）1两半，枳壳（去瓤，麸炒）1两半。
出处：《本事》卷三。
主治：气、食、忧、劳、思虑，致成五噎膈气。
加减：None
功效：None
用法用量：每服30丸，食后、临卧姜汤送下。
制备方法：上为细末，姜汁糊丸，如梧桐子大。
临床应用：None
各家论述：《本事方释义》：半夏气味辛温，入足阳明；桔梗气味苦辛平，入手太阴，为诸药之舟楫，能引药达上；肉桂气味辛甘大热，入足厥阴；枳壳气味苦寒，入足太阴；姜汁丸、姜汤送，欲令药之入里也。此七情六欲之伤致成五噎膈气之疴，所用之药，乃苦辛以开其郁，使升降无阻，自然奏效矣。
用药禁忌：None
附注：None
