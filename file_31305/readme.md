序号：31314
名称：沉香交泰丸
组成：沉香3钱，白术3钱，陈皮（去白）3钱，枳实（麸炒，去瓤）2钱，吴茱萸（汤洗）2钱，白茯苓（去皮）2钱，泽泻2钱，当归（洗）2钱，木香2钱，青皮（去白）2钱，大黄（酒浸）1两，厚朴（姜制）半两。
出处：《医学发明》卷四。
主治：气机郁滞，食积内停，腹中胀满，膈下痞闷，大便燥结。
加减：None
功效：升清气，利浊气，化痞消食，开郁止痛。
用法用量：每服50-80丸，食前以温白汤送下。微利即止。
制备方法：上为细末，汤浸蒸饼为丸，如梧桐子大。
临床应用：None
各家论述：《杏苑》：用沉香升清气、降浊气；木香、厚朴、枳实、橘红、青皮、吴茱萸等散壅滞以疏胀满；白术健脾燥湿；用茯苓、泽泻分利水气；大黄以下湿热；当归理气血，使各归其所。
用药禁忌：孕妇忌服。
附注：None
