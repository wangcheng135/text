序号：53585
名称：夏季补肾肾沥汤
组成：附子1两（炮裂，去皮脐），桂心1两半，白茯苓1两，石南3分，山茱萸3分，石斛3分（去根，锉），人参1两（去竿头），杜仲3分（去粗皮，炙微黄，锉），当归1两（锉，微炒），五味子1两，熟干地黄2两，泽泻1两，肉苁蓉1两（酒浸1宿，刮去破皮，炙令干），磁石2两（捣碎，水淘去赤汁，以帛绢包之）。
出处：《圣惠》卷七。
主治：肾虚。
加减：None
功效：补肾。
用法用量：每服5钱，水1大盏，以羊肾1对，切去脂膜，入生姜半分、大枣3个，每与磁石包子同煎至5分，去滓，食前温服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
