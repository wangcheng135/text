序号：19343
名称：地黄丸
组成：熟干地黄（焙）1两，柏子仁（别研）1两，青橘皮（去白，炒）1两，诃黎勒皮（炮）1两，杜仲（去粗皮，锉，炒）1两，木香（炮）1两，白茯苓（去黑皮）1两，菖蒲1两，赤石脂1两，五加皮（锉）1两，菟丝子（酒浸，别捣）1两，秦艽（去苗士）1两，海浮石1两，艾叶（烧灰存性）1两，当归（切，炒）1两，牛角（角思）灰1两。
出处：《圣济总录》卷一五一。
主治：室女禀气怯弱，血海虚损，月水不断。
加减：None
功效：None
用法用量：每服20丸，米饮或温酒送下。
制备方法：上为末，醋煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
