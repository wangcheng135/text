序号：8049
名称：加减补中益气汤
组成：人参、黄耆、当归、生地、川芎、柴胡、陈皮、甘草、细辛、羌活、防风、白术。
出处：《玉案》卷二。
主治：劳力伤寒，头痛发热恶寒，但微渴自汗，身腿痠软无力，此内伤气血，外感风寒故也。
加减：如元气不足，加升麻少许；咳嗽，加杏仁；汗不止，去细辛，加芍药；胸中烦热，加山栀、竹茹；干呕，加姜汁炒半夏；胸中饱闷，去生地、甘草、黄耆、白术，加枳壳、桔梗；痰盛，去防风、细辛、加瓜蒌仁、贝母；腹痛，去耆、术，加芍药、干姜。
功效：None
用法用量：加生姜、大枣、葱，水煎，温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
