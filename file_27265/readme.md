序号：27273
名称：补肾肉苁蓉丸
组成：肉苁蓉2两（酒浸，去皴皮，微炒，炙），磁石2两（烧醋淬7遍，捣碎，细研水飞过），熟干地黄2两，山茱萸3分，桂心1两，附子1两（炮裂，去皮脐），薯蓣3分，牛膝1两（去苗），石南3分，白茯苓3分，泽泻3分，黄耆3分（锉），鹿茸2两（去毛，涂酥炙令微黄），五味子3分，石斛1两（去根，锉），覆盆子3分，远志3分（去心），补骨脂1两（微炒），萆薢3分，巴戟3分，杜仲1两（去粗皮，炙微黄，锉），菟丝子2两（酒浸3宿，晒干，别杵为末），白龙骨1两。
出处：《圣惠》卷七。
主治：肾脏久虚，面色萎黑，足冷耳鸣，四肢羸瘦，脚膝缓弱，小便滑数。
加减：None
功效：None
用法用量：每服30丸，空心温酒送下，晚食前再服。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
