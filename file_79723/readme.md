序号：79739
名称：漏芦汤
组成：漏芦1两，白蔹1两，黄芩（去黑心）1两，麻黄（去节）1两，枳实（麸炒，去瓤）1两，升麻1两，芍药1两，甘草（炙）1两，朴消1两，大黄2两。
出处：《外科精义》卷下。
主治：一切恶疮，毒肿丹瘤，瘰疬疔肿，鱼睛五发，凛疽。初觉1-2日，便如伤寒，头痛烦渴，拘急恶寒，肢体疼痛，四肢沉重，恍惚闷乱，坐卧不宁，皮肤状热，大便秘涩，小便赤黄。
加减：None
功效：None
用法用量：每服3钱，气实人5钱，水1盏半，文武火煎7沸，去滓，空心热服。
制备方法：上除消外，余(口父)咀，与消同和匀。
临床应用：None
各家论述：None
用药禁忌：好身莫服。
附注：None
