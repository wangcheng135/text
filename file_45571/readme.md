序号：45586
名称：柏子仁丸
组成：新柏子仁（研）2两，半夏曲2两，牡蛎（甘锅子内火煅，用醋淬7次，焙）1两，人参（去芦）1两，麻黄根（慢火炙，拭去汗）1两，吴白术1两，五味子（拣）1两，净麸半两（慢火炒）。
出处：《本事》卷六。
主治：阴虚盗汗。
加减：None
功效：戢阳气，止盗汗，进饮食，退经络热。
用法用量：每服30-50丸，空心米饮送下，1日2次。得效减1服，好愈即住。作散调亦可。
制备方法：上为末，枣肉为丸，如梧桐子大。
临床应用：None
各家论述：《医方集解》：心血虚则睡而汗出，柏子仁味甘辛平，养心宁神为君；牡蛎、麦麸咸凉，清燥收脱为臣；五味酸敛涩收，半夏和胃燥湿为佐；麻黄根专主肌表，引参术以固正气为使。
用药禁忌：None
附注：None
