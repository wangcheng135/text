序号：65361
名称：女经膏
组成：制鳖甲5两，白茯苓3两，益母草2两，大熟地4两，当归3两，炙甘草1两，地骨皮3两，淡黄芩3两，川芎1两5钱，南沙参3两，制香附3两，陈阿胶3两，炒白芍3两，丹参3两，雪梨清膏4两，青蒿3两，川断3两，白蜜4斤，焦白术3两，杜仲2两。
出处：《全国中药成药处方集》（南京方）。
主治：妇女阴虚有热，经期超前，经量或多或少，色紫，心烦，骨，蒸，口干，掌心灼热。
加减：None
功效：None
用法用量：每服3钱，1日2次，早、晚汗水和服。
制备方法：上药文火共熬浓汁去滓，滤清，用阿胶、梨清膏、白蜜收膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
