序号：78377
名称：蔷薇膏
组成：蔷薇根1斤，郁李根1斤，水杨皮1斤，牛蒡根（并细切）1斤，苍耳1升，露蜂房（碎劈）3枚，生地黄（切）1两，升麻1两，当归（洗切）1两，地骨皮半两，白芷半两，石胆（研）半两，熟铜粉（研）1分，麝香（研）1分。
出处：《圣济总录》卷一一七。
主治：风热上攻，口疮多年不愈。
加减：None
功效：None
用法用量：每含如弹丸大，吐津。
制备方法：上14味，先以前6味细切，水2斗，煎至5升，葛布绞去滓；次入地黄、升麻、当归、地骨皮、白芷再煎至2升，绵滤去滓；慢火又煎成膏，乘热下后三味研药，搅令匀，瓷器盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
