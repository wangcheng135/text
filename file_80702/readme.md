序号：80718
名称：麋角丸
组成：麋角霜1斤，白龙骨半斤（佳者，杵碎，以绢袋盛，蒸1日），天雄10两（长大者，酒浸一伏时，炮，未得出，再于酒内浸，再浸再炮，如此经7次，候放冷，取去皮脐），红椒半斤（去目并合口者，酒浸1宿，放微干，于新盆内以慢火炒去汗，杵筛取红），菟丝5两（淘去浮者，酒浸一伏时，蒸1次，捣烂细，焙干），牡蛎5两（水浸洗，火煅通赤，放冷，研），韭子2两，肉苁蓉2两，磁石1两，金钗2两，石斛2两，肉桂2两，巴戟2两，木贼2两，朱砂1两，泽泻2两，阳起石2两。
出处：《鸡峰》卷七。
主治：一切风气。
加减：None
功效：壮筋骨，实下元，秘精，安魂定魄，却老延年，补壮腰膝。
用法用量：每服30丸，酒或盐汤送下。
制备方法：上为细末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
