序号：45018
名称：枯药
组成：好白矾4两，生砒2钱半，朱砂1钱（生研，令10分细）。
出处：《魏氏家藏方》卷七引《李防御五痔方》。
主治：痔疮。
加减：None
功效：枯痔。
用法用量：先看痔头大小，将所煅白研末抄在手掌心上，入生朱末少许，2味作1处，以口津唾就掌心调令稀稠得所，用篦子涂在痔上，周遭令遍，1日上3次。须仔细看痔头颜色，只要色转焦黑，乃是取落之渐，至夜自有黄膏水流出，以布帛衬之，水尽多为妙，乃是恶毒之水，切勿它疑，至中夜更上药1遍，至来日依旧上药3次。纵有些小疼痛不妨。换药时用粗碗1只，盛新水或温汤在痔侧，以羊毛笔1管轻手刷洗痔上，去了旧药，却上新药，仍用护药，次用荆芥汤洗之。2-3日之后，黄膏水流出将尽，痔头焦黑为度，以篦子敲打痔头，见得渐渐坚硬黑色，却于枯药中增添生朱，减退白矾，自然药力慢缓。须用药2-3日以后方可增减，且以日子渐渐取之庶不惊人，全在用药人看痔头转色，增减厚薄敷药，方是活法。
制备方法：上为细末，将生砒安在建盏中，次用白矾末盖之，用火煅令烟绝，其砒尽随烟去，只是借砒气于白矾中，将枯矾取出，研为细末，须是研令十分细。
临床应用：None
各家论述：None
用药禁忌：None
附注：独圣散（《普济方》卷二九六）、枯痔散（《外科十三方考》）。此药只是借用砒气，又有朱砂在内少解砒毒，所以不甚疼痛者，盖非专用毒药也。
