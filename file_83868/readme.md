序号：83886
名称：乌犀丸
组成：乌犀角屑1两，赤箭1两，麻黄1两（去根节），天南星半两（炮裂），秦艽3分（去苗），汉防己半两，独活3分，羚羊角屑3分，防风1分（去芦头），白附子3分（炮裂），白僵蚕1分（微炒），芎3分，当归2分（锉，微炒），酸枣仁2两（微炒），桑寄生2分，龙脑1分（研入），阿胶1两（捣碎，炒令黄燥）。
出处：《圣惠》卷七十四。
主治：妊娠中风，口面僻，言语謇涩，身体强直，或时反张。
加减：None
功效：None
用法用量：每服20丸，以薄荷酒送下，不拘时候。
制备方法：上为末，入研了药令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
