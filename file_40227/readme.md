序号：40237
名称：苦参消石酒
组成：苦参、消石、好清酒。
出处：《千金翼》卷二十一。
主治：赤白癞风。
加减：None
功效：None
用法用量：渐渐服之。饮法：每服初7日，如半鸡子许，7日后可饮1升，任情饮之，空腹服之，1日3次。多则为善，患去则速，风动亦多，勿使醉吐，宁渐少饮，不用多饮，赤白二风，此药至日无有不愈，余非难治。
制备方法：先以清酒下消石，浸之14日或21日，然后与苦参同入酒瓮中，盛浸之7日。
临床应用：None
各家论述：None
用药禁忌：忌房室，大嗔怒，大热；禁粘食、五辛、生冷、大醋、酪、白酒、猪、鱼、鸡、犬、驴、马、牛、羊等肉，皆为大忌，自余不禁。
附注：None
