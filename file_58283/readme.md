序号：58298
名称：真料重明散
组成：炉甘石1两（黄连3钱，童便浸3日，滤出黄连，为末，汁碎石），井泉石4钱（山栀5钱，捶碎，浸水1盏，淬干；又用乳汁1蛤蜊许淬），石燕子2钱（用好醋2蛤壳许淬干），金星石2钱，银星石2钱。
出处：《医方类聚》卷六十七引《修月鲁般经后录》。
主治：None
加减：None
功效：明目。
用法用量：每用少许点眼中。
制备方法：上五石别制后，共为极细末。南明砂3钱，海螵蛸、白丁香（雄雀屎）、硇砂、青盐、枯矾、铜青、胆矾、焰消、片脑、乳香各1钱，芒消、半夏、麝香（少许），轻粉_x000D_
上药各为极细末，入前药研匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
