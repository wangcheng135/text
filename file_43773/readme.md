序号：43788
名称：厚朴丸
组成：厚朴4两（去浮皮，蘸生姜自然汁炙焦黄），苍术2两（米泔水浸洗，晒干，切片略炒），莪术2两（湿纸裹煨，去皮切片），青皮2两（洗，去瓤，晒干，锉细，略炒），陈皮2两（洗，去瓤，晒干，锉细，略炒），当归2两（去芦，净洗晒干），荆三棱2两（去芦，切片），白芷2两（切片）。上用老醋浸7日，春夏浸5日，取出焙干，研细末，入后药中：禹余粮（火煅醋淬凡10次，为细末，用水飞过，再入干锅内火煅通红，净称2两再研）、针砂2两（净洗，用醋浸1宿，煮干，研极细，净秤）。
出处：《朱氏集验方》卷四引鲁太丞方。
主治：None
加减：None
功效：宽中进食，推化积聚。
用法用量：每服70-80至100丸，食前陈米饮送下。
制备方法：醋煮面糊为丸，如梧桐子大，候干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
