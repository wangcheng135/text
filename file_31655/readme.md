序号：31664
名称：羌活胜湿汤
组成：羌活1钱，独活1钱，藁本5分，防风5分，甘草（炙）5分，川芎5分，蔓荆子3分。
出处：《内外伤辨》卷中。
主治：外伤于湿，郁于太阳，肩背痛，脊痛项强，或一身尽痛，或身重不能转侧，脉浮；邪在少阳、厥阴，卧而多惊。
加减：如经中有寒湿，身重腰沉沉然，加酒洗汉防己5分，轻者附子5分，重者川乌5分。
功效：None
用法用量：上(口父)咀，都作1服。水2盏，煎至1盏，空心食前去滓大温服。
制备方法：None
临床应用：None
各家论述：1.《医方考》：《经》曰：风胜湿。故用羌、防、藁、独、芎、蔓诸风药以治之，以风药而治湿，如卑湿之地，风行其上，不终日而湿去矣；又曰无窍不入，惟风为能。故凡关节之病，非风药不可。用甘草者，以风药悍燥，用以调之，此之谓有制之兵也。_x000D_
2.《医方集解》：此足太阳药也。《经》曰：风能胜湿。如物之湿，风吹则干。羌、独、防、藁、芎、蔓皆风药也，湿气在表，六者辛温升散，又皆解表之药，使湿从汗出，则诸邪散矣。藁本专治太阳寒湿；荆、防善散太阳风湿；二活祛风胜湿，兼通关节；川芎能升厥明清气，上治头痛；甘草助诸药辛甘发散为阳，气味甘平，发中有补也。
用药禁忌：None
附注：通气防风汤（《医学发明》卷五）、通气防风散（《普济方》卷九十七）、胜湿汤（《医级》卷七）。
