序号：3303
名称：天雄丸
组成：天雄2两（炮裂，去皮脐），石斛1两（去根，锉），补骨脂1两（微炒），天麻1两，麋角屑1两，泽泻1两，巴戟1两，五味子1两，柏子仁1两，沉香1两，肉苁蓉1两（酒浸1宿，刮去皱皮，炙，锉），鹿茸1两（去毛，涂酥，炙微黄），菟丝子1两（酒浸3日，曝干，别捣为末），山茱萸1两，续断1两，熟干地黄1两，杜仲1两（去粗皮，炙微黄，锉），防风1两（去芦头），腽肭脐1两（酒洗，炙令微黄），木香1两，龙骨1两。
出处：《圣惠》卷九十八。
主治：五劳七伤，元气衰惫，腰膝久冷，精气散失，小便稠浊。
加减：None
功效：补填骨髓，益壮血脉，驻精气，暖腰膝，润泽肌肉，补诸虚不足。
用法用量：每日30丸，空心以温酒送下；盐汤下亦得。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
