序号：33626
名称：运脾饮
组成：香附、半夏、苍术、厚朴、陈皮、甘草、茯苓、草豆蔻、山楂、泽泻、神曲。
出处：《陈素庵妇科补解》卷一。
主治：脾虚或外感风冷，内伤饮食而致脾气不实，经正行忽病泄泻者。
加减：风，加防风；寒，加羌活；伤食，加莱菔子（炒）。
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：脾主中州，主运化水谷。脾虚火衰，则失其健运之常，加以风寒外侵，饮食内伤，而泄泻之症作矣。虽属脾虚，初泻以消，健脾为先务，泻久以虚，补脾为上策，必兼外感、内伤，百无一失。是方香、夏、朴、陈、草蔻温中运脾，苍术力猛，祛风散寒，逐湿发汗，楂、曲消食宽中，苓、泻、甘草利水止泻。
用药禁忌：None
附注：None
