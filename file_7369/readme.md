序号：7369
名称：外痔神消散
组成：红栀子（捣碎）1两，黄柏1两，胡黄连1两。
出处：《疡医大全》卷二十三。
主治：外痔。
加减：None
功效：None
用法用量：每用，头1次用苦参1两，水3大碗煎成2碗，去滓；入制消末1两，煎2-3滚，倾盆内，先熏后洗，每日洗3次；第2日仍用洗剩药水，再加苦参1两，加水1大碗煎成，加制消末5钱，如上熏洗；第3日照第2日洗法；第4日药水俱不用，照第1日换新水煎苦参制消洗之。以3日又换1次，其痔自消，永不再发。
制备方法：水二宫碗，煎1碗，二煎、三煎俱煎1碗，共3碗，去滓；入皮消1斤燉化，撤去泥脚，取上清消汁，以文火慢炼成消。大田螺16个，每个入冰片1厘，待螺化成水，将水拌入消内，再加熊胆、儿茶各5钱，研细和入消内拌匀听用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
