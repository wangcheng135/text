序号：65908
名称：萆薢散
组成：萆薢1两（锉），茵芋半两，杜仲半两（去粗皮，炙微黄，锉），天雄3分（炮裂，去皮脐），石南半两，石龙芮半两，踯躅半两（微炒），独活2两，附子3分（炮裂，去皮脐），狗脊半两，当归半两（锉，微炒），麻黄3分（去根节），干蝎半两（微炒），桑螵蛸半两（微炒），菖蒲半两，赤箭2分，甘菊花3分，牛膝3分（去苗），木香3分，芎2分，麝香半两（细研）。
出处：《圣惠》卷七。
主治：肾脏中风，卧踞而腰痛，脚膝偏枯，皮肤顽痹，语言謇涩，两耳虚鸣，举体乏力，面无颜色，志意不乐，骨节酸疼。
加减：None
功效：None
用法用量：每服2钱，食前以温酒调下。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
