序号：23823
名称：安心煮散
组成：远志2两，白芍药2两，宿姜2两，茯苓42铢，知母42铢，紫菀42铢，赤石脂42铢，石膏42铢，麦门冬42铢，桂心30铢，麻黄30铢，黄芩30铢，萎蕤36铢，人参24铢，甘草10铢。
出处：《千金》卷十三。
主治：心热满，烦闷惊恐。
加减：None
功效：None
用法用量：先以水5升，淡竹叶1升，煮取3升，去滓，煮散1方寸匕，牢以绢裹煮，时动之，煎取8合为1服，日2次。
制备方法：上为粗散。
临床应用：None
各家论述：《千金方衍义》：此兼竹沥、茯神散二方之制，方中麻黄、远志、萎蕤、即茯苓散中升麻、桂心、麦冬之义，人参、甘草、黄芩，即竹沥汤中人参、白术、栀子之义，三方合，究其微，则滋中寓清，清中寓散，散中寓清之法。
用药禁忌：None
附注：本方方名，《普济方》卷十六引作“安心散”
