序号：81441
名称：鳖甲煮散
组成：鳖甲（去裙襕，醋炙）半两，羌活（去芦头）1分，柴胡（去苗）1分，黄耆（锉）半两，赤茯苓（去黑皮）1分，生干地黄（焙）1分，木香1分，牛膝（酒浸，切，焙）半两，天灵盖（酥炙）半两，续断半两，菟丝子（酒浸，别捣）半两，芎1两，藿香叶1两，陈橘皮（汤浸，去白，焙）1两，山茱萸1两，麝香（研）1分，独活（去芦头）1分。
出处：《普济方》卷二○○引《护命方》。
主治：劳疟。毛发枯焦，寒热不定，饮食减少，肌肉消瘦，面色青黑，两足无力，非时足冷，小便频数，大便不调，梦遗滑精。
加减：None
功效：None
用法用量：每服40丸，空心盐汤送下。
制备方法：上为散。每服1-2钱，水1盏，入盐少许，同煎7分，去滓，空心服。或炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：若大腑秘、骨蒸热勿服。
附注：None
