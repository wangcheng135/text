序号：48414
名称：神效松脂散
组成：黄香12两。
出处：《嵩崖尊生》卷十。
主治：一切疠风，并大麻风。
加减：None
功效：None
用法用量：用水10碗，入铜锅内煮1时辰，离火候温，即下手在水内，将香扭拔数次，冷定取出，再研再煮，如此9次，其香即酥，瓷器放贮，不可泄气。每服3钱，空心滚黄酒调下。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：服药后其毒从大小二便而出，预先令人在空野之地，挖一深坑，如病毒行，解在坑内，俟毒行尽，即用土盖之，恐毒染人。毒行之后，需用淡饭静心养之，不可用厚味动火发疾。并忌生冷腥荤诸般发物，大忌房事，如犯之则不可救。
附注：None
