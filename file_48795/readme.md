序号：48810
名称：神膏
组成：金银花8两，蒲公英8两，木连藤8两，真麻油8两，黄丹12两，乳香3钱，没药3钱，松香3两。
出处：《洞天奥旨》卷十五。
主治：发背，诸疮疡，不论阴阳痈毒，皆可贴之。
加减：阳疽，用冰片1钱，麝香2分，黄柏3钱，白芷3钱，五灵脂2钱，三七根5钱，洋参3钱，各为末，掺入膏药贴之；阴疽，用肉桂3钱，冰片3分，人参1钱，丹砂3钱，紫石英3钱，儿茶3钱，五灵脂2钱，各为末，掺入膏内贴之。
功效：None
用法用量：摊贴。
制备方法：上以麻油先煎金银花、蒲公英、木连藤至黑，滤去滓，入黄丹、乳香、没药、松香，煎成膏，去火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
