序号：77225
名称：酥雄丹
组成：上朱砂（研细，水飞净）1两2钱，真茅苍术（研净末）1两2钱，母丁香（研净末）1两2钱，明雄黄（研净末）1两2钱，真蟾酥1两2钱（净，以好酒化开，不住手搅粘）。
出处：《同寿录》卷末。
主治：一切痧胀恶症并暑气恶气，四肢痠胀，头晕眼花，心烦意乱；以及蝎螫、蜂叮发痒，疮疖。
加减：None
功效：None
用法用量：恶心腹痛及一切痧症，每用1丸噙于舌下，听其自化，微觉舌麻，不过1时即愈，至重者不过2丸，切勿多用；蝎蜂叮螫，发痒疮疖，每用1丸水浸化，敷患处。
制备方法：将各药入酥内擂匀，为丸如粟米大。
临床应用：None
各家论述：None
用药禁忌：忌生冷、辛辣、油腻。
附注：None
