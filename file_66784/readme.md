序号：66800
名称：鹿髓煎丸
组成：鹿髓5合，生天门冬汁3合（滤），生麦门冬汁3合（滤），清酒5合，牛髓5合（无牛髓，牛酥1升代），白蜜7合，枣膏5合，生地黄汁1升（上八味，先煎地黄、天门冬汁、清酒，5分减2分，次纳麦冬汁煎20沸，次纳酥、髓、白蜜、枣膏，煎如稠糖，倾出银石器中，复于重汤上煮，搅如稠膏，即入后药末），茯神（去木）1两，龙骨1两，人参1两，枳壳（去瓤，麸炒）3分，细辛（去苗叶）3分，防风（去叉）3分，白术3分，石斛（去根）3分，桂（去粗皮）3分，芎3分，黄耆（炙，锉）3分，五味子3分，甘草（炙，锉）1两半，陈橘皮（汤浸，去白，焙）半两，厚朴（去粗皮，生姜汁炙）半两，山芋半两，山茱萸（并子用）3分，柏子仁（炒）3分，枸杞子3分，远志（去心）半两，黄连（去须）半两，薏苡仁（炒）3分，槟榔（锉）3分。
出处：《圣济总录》卷十四。
主治：久积风热，发即惊悸，气满不安，四肢虚弱，不生肌肉。
加减：None
功效：None
用法用量：每服20丸，加至30丸，温酒送下，空心，日午、夜卧服。
制备方法：上三十一味，除八味为煎外，捣罗为末，入在煎中，和捣令匀为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
