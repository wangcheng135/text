序号：26749
名称：延龄聚宝丹
组成：何首乌1两（去皮），赤白1两，生地黄（肥嫩者）8两，熟地黄4两（鲜嫩者，俱忌铁），白茯苓4两（去皮），莲蕊4两，桑椹子4两（紫黑者），甘菊花4两（家园黄白2色），槐角子4两（11月11日采，炒黄），五加皮（真正者）4两，天门冬（去心）、麦门冬（去心），茅山苍术（去皮，泔浸1宿，忌铁）2两5钱，石菖蒲（1寸9节者）、苍耳子（炒，捣去刺）、黄精（鲜肥者）、肉苁蓉（酒洗，去甲心膜）、甘枸杞（去蒂，捣碎）、人参、白术（极白无油者）、当归（鲜嫩者）、天麻（如牛角尖者）、防风（去芦）、牛膝（酒洗）、杜仲（姜汁浸1宿，炒断丝）、粉甘草（去皮，炙）、沙苑白蒺藜（炒，舂去刺）。
出处：《扶寿精方》。
主治：None
加减：None
功效：畅快百骸，潜消百病，强身壮体，聪耳明目，固齿坚牙。
用法用量：每服50丸，无灰酒送下，每五更服3小杯药酒，仍卧片时；年久亦然；但觉腹空，并夜坐服3杯，最益。
制备方法：上锉，生绢袋盛，无灰醇酒9斗，瓷坛中春浸10日，夏浸7日，秋冬浸14日，取出药袋。控干，晒，碾为末，炼蜜为丸，如梧桐子大。
临床应用：养生：林以和，自39服本方至今64岁，宿病咸愈，身体强壮，须发不变，耳目聪明，齿牙坚固，精神胜常。
各家论述：None
用药禁忌：服酒后，切忌生冷葱蒜，韭白，菜菔，鱼，脱落尘事，诚心修服无间。
附注：保命丹（《扶寿精方》）、延龄聚宝酒（《遵生八笺》卷十七）。
