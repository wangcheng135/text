序号：29430
名称：连翘膏
组成：连翘1两半，陈油1斤半，猪脂10两，羊脂5两，黄耆1两半，黄丹14两，白芷1两半，白及1两半，白蔹1两半，乳香3分，松脂1两半，蜡2两，露蜂房1两半，乱发灰半两，青绢1尺2寸（烧灰），绯绢1尺2寸（烧灰），当归1两半，白芍药1两半，桂心1两半。
出处：《圣惠》卷六十三。
主治：一切痈疽发背，穿穴后。
加减：None
功效：排脓散毒止痛。
用法用量：用时旋于故帛上摊贴，日2换之。
制备方法：上件药，先将油及猪羊脂，以微火煎，侯脂消尽，锉碎黄耆、白芷、连翘、蜂房、白及、白蔹、当归、芍药、桂心九味，下入油内，以微火煎，候药黄黑色，次入松脂、蜡、乳香熔尽，即以绵绞去滓，再入铛内煎，即下黄丹，以柳木篦搅，勿令住手，候药变黑色，次下绯青绢灰，及头发灰，搅令匀，滴于冷处，凝硬得所，成膏，于瓷器内收。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
