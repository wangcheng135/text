序号：70119
名称：黑丸子
组成：山茵陈半两，蜀升麻半两，常山半两，芒消半分，麻黄（去节根）1两，官桂（去粗皮）1分，附子1个（烧黑留心）。
出处：《普济方》卷三九四引《灵苑方》。
主治：None
加减：None
功效：退热，定吐逆兼除食伤。
用法用量：每服5丸，吐不止，茅根竹叶汤送下；热攻泻血，蜜炒生姜汤送下；若吐血、眼眦出血者，生油、冷酒送下；伤寒手脚心冷，冷茶清送下；失音，竹沥酒送下。
制备方法：上为极细末，旋炒1大钱，入杏仁2粒（去皮尖，灯烧黑存性），巴豆○粒（压去油），寒食面糊为丸，如麻子大；大人丸如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
