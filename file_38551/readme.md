序号：38561
名称：法制化痰丸
组成：南星1两，半夏1两（用猪牙皂角5钱，白矾5钱，生姜5钱，水3碗，同浸1宿，秋冬2宿，煮熟，去皂角、生姜，只用星、半2味，晒干听用），瓜蒌仁（去壳）5钱，白术5钱，黄连（姜汁炒）5钱，香附（童便浸，炒）5钱，陈皮（去白）5钱，山楂5钱，萝卜子5钱，白茯苓（去皮）5钱，紫苏子5钱，片芩（酒炒）5钱，枳实（炒）5钱。
出处：《万氏家抄方》卷二。
主治：男妇虚火咳嗽，哮喘吐痰，胸膈饱胀，嗳气，一切痰症。
加减：None
功效：None
用法用量：每服60-70丸，淡姜汤或白滚汤送下，食远服。
制备方法：上为末，加竹沥1钟，姜汁1盏，入神曲末1两5钱为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
