序号：576
名称：五疳丸
组成：干蟾1只（置新瓦上，实以土，炼存性），连翘（微炒）2两，青皮2两，莪术2两，三棱2两，枳实（俱醋炒）2两，川黄连2两，胡黄连2两，芦荟2两，丹皮2两，白芜荑仁2两，桔梗2两，茯苓3钱，神曲（炒）3钱，泽泻（炒）3钱，白术（土炒）3钱，木香1钱5分。
出处：《慈幼新书》卷十。
主治：疳积。
加减：None
功效：None
用法用量：每服1钱2分，米饮送下。
制备方法：上为细末，用雄猪胆2枚，取汁和水泛为丸，如豌豆大。
临床应用：疳积：郑补翁：余幼女病疳，两目几废，偶得此方，服之神光复现。余友汪宗五子亦患此，肌肉瘦削，饮食大减，诸药无功，余以此授之，服一料而病去如扫。
各家论述：None
用药禁忌：None
附注：None
