序号：9582
名称：仙人粥
组成：何首乌1-2斤。
出处：《摄生众妙方》卷二。
主治：气血两虚，身体羸瘦，面色黄肿，手足疼痛，头昏耳鸣，头发早白，大便干燥。现用于老年性高血脂，血管硬化，大便干燥。
加减：None
功效：益肾抗老，养肝补血。
用法用量：每日5钱，用砂锅以白水滚烂，放白米3合，洗净，入内煮粥。每日空心服。
制备方法：以竹刀刮去皮，不可用铁器，切成细细如棊子面大。
临床应用：None
各家论述：None
用药禁忌：None
附注：首乌粥（《大众医学》）、何首乌粥（《长寿药粥谱》）。《长寿药粥谱》：何首乌粥：先用何首乌30～60g，入砂锅煎取浓汁，去滓，与粳米100g，大枣二三枚，冰糖适量，同煮为粥。
