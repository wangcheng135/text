序号：35225
名称：乳香法纸
组成：乳香（研细末）1两，甘草1两2钱，轻粉3钱。
出处：《外科正宗》卷四。
主治：臁疮作痛。
加减：None
功效：止痛生肌。
用法用量：临时随患大小剪纸多少，先用温汤洗疮，随将纸有药一面对疮贴之，绢扎，3日1换。如贴后内无水出，不必换贴自愈。
制备方法：以呈文油纸4张，每纸一张摊平，筛乳香末2钱5分，匀筛纸上，双折，捲一寸阔，将捲纸复作三折，两头以线扎之，用甘草1两2钱，水3碗，将捲过药纸浸入甘草汤内，上用重物压之，煮数滚，取起纸来，解去扎线，将纸摊开桌上，每张用轻粉3钱掺乳香上，用棕糊刷排剧相匀，提起药纸，带湿以无药一面对板壁上贴之，阴干收用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
