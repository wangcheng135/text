序号：65842
名称：菴（艹闾）子散
组成：菴（艹闾）子1两1分，瞿麦穗3分，槟榔（锉）3分，桂（去粗皮）3分，牡丹皮3分，芎3分，当归（切，焙）3分，甘草（炙，锉）3分，射干3分，木香3分，吴茱萸（汤洗，焙炒）3分，桃仁（汤浸，去皮尖双仁，炒黄）12枚，鳖甲（去裙襴，醋炙）1两，牛膝（酒浸，切，焙）1两，蒲黄1两，赤芍药1两，大黄（锉，炒）1两，熟干地黄（焙）1两，黄芩（去黑心）1两半，水蛭（微炒）1分。
出处：《圣济总录》卷一五一。
主治：妇人血气不调，经水不定，腹胁多胀，或5-6月一来，或2-3月一来，虽来色如煮小豆汁，其血复少者。
加减：None
功效：None
用法用量：每服1钱匕，稍加至2钱匕，温酒调下，1日2次。以效为度。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
