序号：593
名称：五福化毒丹
组成：桔梗（微炒）6两，玄参（洗，焙）6两，青黛（研）2两，牙消（枯）2两，人参（去芦）2两，茯苓（去皮）5两，甘草（炒）1两半，银箔8片（为衣），麝香（研）半钱，金箔8片（为衣）。
出处：《局方》卷十。
主治：小儿热毒内蕴，口舌生疮，常患疮疖。(匿虫)鼻疳疮，热疳肌肉黄瘦，雀日夜不见物。胎敛疮。小儿热极，皮肤火热，红晕成片，游走状如火丹。
加减：None
功效：清膈凉血。
用法用量：每1岁儿，1丸分4服，用薄荷水送下；及疮疹后余毒上攻口齿，涎血臭气，以生地黄自然汁化1丸，用鸡翎扫在口内；热疳肌肉黄瘦，雀目夜不见物，食后、临卧用陈粟米泔水化下。
制备方法：上为细末，入研药匀，炼蜜为丸，每两作12丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：青黛丸(《得效》卷十一)、化毒丸(《普济方》卷三六四)。
