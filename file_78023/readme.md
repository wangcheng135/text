序号：78039
名称：赛命丹
组成：蟾酥3钱，朱砂3钱，雄黄3钱，胆矾3钱，血竭3钱，乳香3钱，没药3钱，蜈蚣5分，麝香5分，细辛6钱，全蝎6钱，蝉退6钱，穿山甲6钱，僵蚕6钱，牙皂6钱，白矾（用信少许同枯，去信不用）5分，片脑5分。
出处：《医学入门》卷八。
主治：痈疽发背、疔疮乳痈，鱼口便毒，一切无名肿毒，及小儿脐风。
加减：None
功效：None
用法用量：每服3丸，用葱酒1小钟送下，被盖出汗，或不汗，再进1丸，服后吃白粥调理。
制备方法：上为末，端午日用酒糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：忌黄瓜、水前，一切动风之物。
附注：赛夺命丹（《简明医彀》卷八）。
