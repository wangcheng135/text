序号：3656
名称：廿四味流气饮
组成：半夏（汤洗7次）2两，陈皮（去白）2斤，厚朴（去粗皮，姜制，炒）1斤，青皮（去白）1斤，甘草（爁）1斤，香附（炒去毛）1斤，紫苏叶（去枝梗）1斤，人参4两，赤茯苓（去黑皮）4两，干木瓜4两，白术4两，白芷4两，麦门冬4两，草果仁6两，肉桂（去粗皮，不见火）6两，蓬莪荗（煨，切）6两，大腹皮6两，丁香皮6两，槟榔6两，木香（不见火）6两，木通（去节）8两，沉香6两，枳壳（去瓤，麸炒）4两，大黄（面裹，煨，去面，切）2两。
出处：《局方》卷三(新添诸局经验秘方)引《集验方》。
主治：诸气痞滞不通，胸膈膨胀，口苦咽干，呕吐少食，肩背腹胁走注刺痛；及喘急痰嗽，面目虚浮，四肢肿满，大便秘结，水道赤涩；忧思太过，怔松郁积，脚气风热，聚结肿痛，喘满胀急。
加减：None
功效：调顺荣卫，通流血脉，快利三焦，安和五脏。
用法用量：每服4钱，水1盏半，加生姜3片，大枣2枚，煎7分，去滓热服。如伤寒头痛，才觉得疾，入连根葱白3寸煎，升降阴阳，汗出立愈；脏腑自利，入粳米煎；妇人血气癥瘕，入艾、醋煎。不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《医方类聚》引作“二十四味流气饮”。《丹溪心法附余》引作“木香流气饮”。原书同卷木香流气饮，即本方去沉香、枳壳、大黄，加石菖蒲四两、藿香叶六两。
