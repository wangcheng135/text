序号：52432
名称：香甲散
组成：木香3分，鳖甲（去皮肉，醋炙令香）2两，牡丹皮1两，赤芍药1两，陈橘皮（去白）1两，官桂（去皮）1两，人参1两，茯苓1两，熟干地黄1两，秦艽1两，柴胡（去苗）1两，白术1两，当归（炒）1两，黑附子（炮制，去皮脐）1两，干姜3分（炮制），甘草半两（炙）。
出处：《博济》卷四。
主治：妇人血气虚劳，四肢少力，肌肉黄瘦，多困减食，遍体痠疼，真邪相击，心腹撮痛。
加减：如烦渴心躁，加乌梅1两（为末）。
功效：None
用法用量：每服2钱，以水1盏，加生姜3片，大枣2枚，同煎至7分，去滓，稍热服。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：香甲汤（〈圣济总录〉卷一五○）。
