序号：37728
名称：抽薪饮
组成：黄芩1-2钱，石斛1-2钱，木通1-2钱，栀子（炒）1-2钱，黄柏1-2钱，枳壳1钱半，泽泻1钱半，细甘草3分。
出处：《景岳全书》卷五十一。
主治：火热炽盛，瘟疫发狂，及孕妇外感发热。火炽盛而不宜补者。胃火发狂。
加减：如热在经络、肌肤者，加连翘、天花粉以解之；热在血分、大小肠者，加槐蕊、黄连以清之；热在阳明头面，或躁烦便实者，加生石膏以降之；热在下焦，小水痛涩者，加草龙胆、车前以利之；热在阴分，津液不足者，加门冬、生地、芍药之类以滋之；热在肠胃，实结者，加大黄、芒消以通之。
功效：None
用法用量：水1钟半，煎7分，食远温服。内热甚者，冷服更佳。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
