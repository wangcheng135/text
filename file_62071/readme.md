序号：62087
名称：大黄散
组成：羊胫炭（烧赤，酒淬10过）5两，大黄（小便浸7日，日1易，以湿纸裹煨，切，焙）3两半，巴豆肉（浆水煮黄色，焙）3两半，古铜钱（用半两烧赤，米醋淬，为粉，新水飞过，去粗取细者）2两。
出处：《苏沈良方》卷十引《灵苑方》。
主治：产后血晕及伤折内损，妇人血癥血瘕。
加减：None
功效：None
用法用量：上和研1日。每服半钱，当归1分，小便煎浓，稍温调下。产后血晕百疾，且当逐血者，至甚乃服；口噤者，挖开灌下，候识人，更1服。累经生产，有血积癥癖块，及败血风劳，寒热诸疾，当下如烂猪肝片，永无他疾。坠击内损，当归酒送下1字。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：托骨大黄散（《伤科汇纂》卷八）。
