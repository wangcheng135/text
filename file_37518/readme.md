序号：37528
名称：抱龙丸
组成：南星（腊月牛胆，制多次佳）4两，天竺黄（另研）1两，朱砂（水飞）5钱，雄黄（飞）5钱，甘草7钱，麝香5分（未出痘不必加），牛黄（陕西，千层）5分，琥珀5分。
出处：《简明医彀》卷六。
主治：小儿伤风、寒、暑发热，变蒸，惊风痰壅，喘急咳嗽；及大人风痰。
加减：None
功效：None
用法用量：每服1丸，临睡薄荷汤化下，病急多服。伤风感寒，姜、葱、紫苏汤下，取汗。大人每服3-5丸。如风痰，用竹沥入姜汁数匙化服，姜汤亦可。惊后调理，初生至长，微有不快，即宜服。
制备方法：上和匀，取腊雪水或早首汲井水，另煎甘草汁为丸，如芡实大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
