序号：33926
名称：灵飞散
组成：干蝎（全者，炒）1两，硇砂（去夹石者，生用）3分，巴戟天（去心）1两，阳起石（研细）3分，真珠（捣研）半两，木香半两，附子（炮裂，去皮脐）半两，芫花（醋炒）3两半，青橘皮（汤浸，去白，焙）半两，硫黄（研）半两，阿魏（研）1分，磁石（煅赤，淬7遍，研）2两。
出处：《圣济总录》卷八十二。
主治：脚气冲心闷极者。脚气发时，或肿或咳，不头痛，不嗽逆，心间妨闷，上气急促者尤宜服之。
加减：None
功效：None
用法用量：每服1钱匕，空心热酒调下，良久以饭压之，至午食前再服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
