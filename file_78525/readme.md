序号：78541
名称：獭肝丸
组成：獭肝（切碎，炙黄）1两半，胡黄连1两，鳖甲（去裙襕，醋炙）1两半，柴胡（去苗）1两半，犀角屑1两，知母（焙）1两，天门冬（去心，焙）1两，地骨皮1两半，升麻1两半，茯神（去木）1两，紫菀（去苗土）1两，百合1两，杏仁（汤浸，去皮尖双仁，炒令黄色，别研）1两，黄连（去须）1两半，前胡（去芦头）1两，贝母（去心，焙）1两，天灵盖（酥炙令黄）1两半，槟榔（锉）3两，麻仁（研）1两，甘草（炙）1两，生干地黄（焙）3两。
出处：《圣济总录》卷八十六。
主治：心劳热，胸膈聚痰，头目微痛，手足时烦，肌肤渐觉羸瘦。
加减：None
功效：None
用法用量：每服30丸，食后温水送下，夜卧再服。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
