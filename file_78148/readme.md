序号：78164
名称：膏药
组成：千里光自然汁10两，煮酒6两，肉当归5钱，大黄5钱，赤芍5钱，肉桂5钱，生地5钱，元参5钱，苦参5钱，踯躅花5钱。
出处：《外科图说》卷四。
主治：杨梅疮及癣疮、鹅掌风，结毒破烂。
加减：None
功效：拔毒吸脓，暖筋骨，长肌肉。
用法用量：摊之，用重汤顿软贴。
制备方法：文火煎浓，约存汁1碗许；用麻油24两，加头发3两，煎至滴水成珠，入前药汁，文火煎和，加研细铅粉，密陀僧各5两，缓缓搅转，候火候却好，滴水不老不嫩，离火，加入研细乳香、没药、黄占、白占各3钱，麝香3分，粉霜1钱，收罐坐水中，去火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
