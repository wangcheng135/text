序号：69543
名称：犀角散
组成：犀角屑1两，乌蛇（酒浸，去皮骨，炙）1两，细辛（去苗叶）1两，芎1两，独活（去芦头）1两，黄耆（锉）1两，蜀椒（去目并闭口者，炒出汗）1两，升麻1两，天麻（酒浸，焙）1两，羌活（去芦头）1两，苦参1两，龙骨（火烧）3分，酸枣仁（炒）3分，蔓荆实3分，枳壳（去瓤，麸炒）半两。
出处：《圣济总录》卷一六二。
主治：产后中风。角弓反张，筋急口噤。
加减：None
功效：None
用法用量：每服3钱匕，温酒调下，每日1次，或2-3服后，于温暖浴室内澡浴1次，令身内外和暖，浴后再服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：不可太汗出，慎风冷。
附注：None
