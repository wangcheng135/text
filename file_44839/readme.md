序号：44854
名称：扁鹊玉壶丸
组成：硫黄8两。
出处：《古方选注》卷中。
主治：阴寒恶疾，命门火衰，阳气暴绝，寒水膨胀。
加减：None
功效：补火扶阳。
用法用量：每服以硫3分为准，渐加至1钱，温开水送下。
制备方法：凡硫黄8两，配真麻油8两，将硫黄打碎，入冷油内炖炉上，炭火宜微勿烈，以桑条徐调，候硫溶尽即倾入大水缸内，急搀去上面油水，其色如金，取缸底净硫秤见若干两，仍配香麻油若干两，照前火候再溶、再倾，连前共三转；第四转用真棉花核油，配硫若干两，照前火候再溶，再倾入大水内，急搀去上面油水，其色如绛；第五转，用肥皂4两，水中同煮六时；第六转用皂荚4两，水中同煮六时，拔净制硫之油，搀去其水，其色如硫火之紫；第七转用炉中炭灰，淋碱水制六时；第八转用水豆腐制六时，拔净皂碱之性；第九转用田字草捣汁（田字草出水稻田中，其叶如田字，8-9月采），和水制六时；临用研如飞面，凡净硫1两，配炒糯米粉2两，或水法或湿捣为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
