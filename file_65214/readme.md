序号：65230
名称：银翘红酱解毒汤
组成：银花1两，连翘1两，红藤1两，败酱草1两，丹皮3钱，山栀4钱，赤芍4钱，桃仁4钱，苡仁4钱，延胡索3钱，炙乳香1钱半-3钱，炙没药1钱半-3钱，川楝子3钱。
出处：《妇产科学》。
主治：盆腔炎发热期。
加减：高热兼表症者，加荆芥、防风各1钱半-3钱，薄荷1钱；便溏热臭者，加葛根、黄芩各3钱，黄连1钱；便秘者，加大黄、元明粉（冲）各3钱；腹胀气滞者，加木香1钱，香附4钱；热毒甚者，加蒲公英、紫花地丁各1两；带多者，加黄柏3钱，椿根皮4钱；有血性分泌物者，加益母草5钱。
功效：清热解毒，活血化瘀。
用法用量：水煎服。每日2剂，每剂2汁，隔4-6小时服1次。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
