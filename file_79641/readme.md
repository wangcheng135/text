序号：79657
名称：熊油虎骨膏
组成：虎骨全副（捶碎），熊油10斤，当归8两，川芎8两，木瓜8两，牛膝8两，杜仲8两，天麻8两，南星8两，藁本梢8两，羌活4两，独活4两，防风4两，骨碎补4两，川断4两，胡芦巴4两，淫羊藿4两，草豆蔻4两，海风藤4两，钻地风4两，清风藤4两。
出处：《饲鹤亭集方》。
主治：None
加减：None
功效：强阳长力，壮骨填精，舒筋活络，胜湿祛风除痹。
用法用量：摊贴。
制备方法：用真麻油80斤，香油亦可，浸7日夜，如法熬膏。以炒黄丹20斤收膏，俟将凝定再下香料细药：肉桂、公丁香、乳香、没药、血竭、儿茶各8两，樟冰、原麝各2两，预研净末，徐徐搅匀，瓷坛密收。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
