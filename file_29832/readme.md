序号：29840
名称：十华散
组成：五加皮6两，陈皮（去白）6两，干姜（炮）6两，甘草6两，大川乌3两，附子（炮）6两，桔梗（炒）8两8钱，肉桂（去粗皮）8两8钱，绵黄耆（去芦，炒）8两8钱，苍术（去皮，炒）8两8钱，羌活8两8钱。
出处：《局方》卷二（绍兴续添方）。
主治：丈夫五劳七伤，浑身疼痛，四肢拘急，腰膝无力；脾元气虚，不思饮食，霍乱吐泻，四肢冷麻；二毒伤寒，脚气流注肿痛，行步不得及虚劳等患。
加减：None
功效：None
用法用量：每服2钱，水1盏，加生姜2片，大枣1个，煎6分，不拘时服；热盐酒调服亦得。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
