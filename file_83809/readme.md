序号：83827
名称：乌蛇散
组成：乌蛇（酒浸，去皮骨，炙）1两1分，防风（去叉）1两1分，青葙子1两1分，羌活（去芦头）1两，独活（去芦头）2两1分，麻黄（去根节，先煎，掠去沫，焙）2两1分，桔梗（炒）2两1分，秦艽（去土）2两1分，当归（切，焙）3分，细辛（去苗叶）3分，桂（去粗皮）3分，芎2两半，白芷（微炒）2两半，附子（炮裂，去皮脐）2两半，白芍药2两，蒺藜子（炒，去角）2两，人参2两，天麻2两。
出处：《圣济总录》卷十一。
主治：风不仁，皮肤麻，手足挛急。
加减：None
功效：None
用法用量：每服2钱匕，空心温酒调下，1日3次。微汗为度。
制备方法：上为极细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
