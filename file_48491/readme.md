序号：48506
名称：神效散
组成：苦参2两，川椒2两，苦葫芦2两，芫荽子2两，槐花2两，枳壳2两，荆芥2两，金银花2两，小茴香2两，白芷2两，连翘2两，独活2两，麻黄2两，牡蛎（煅）2两，威灵仙2两，椿树皮2两。
出处：《普济方》卷二九五。
主治：痔漏。多因嗜欲酒色过度，喜怒不常，致生痔漏，或如鼠乳连珠，或粪门肠头肿，流脓漏血，其痛如割，不可忍者。并治肠风下血。
加减：加老黄茄子2个，尤妙。
功效：None
用法用量：上（口父）咀。每用5钱，水6-7碗，葱白3茎，煎5-7沸，先以盆盛药水，上坐，先蒸后洗，却以乌龙膏贴之，临卧时再以药滓熬水如前洗之，如此3-5次，夜则以膏药贴之。常服葛花酒蒸香连丸。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
