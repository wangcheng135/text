序号：82371
名称：避瘟散
组成：檀香41两6钱，零陵香4两8钱，白芷11两2钱，香排草48两，姜黄4两8钱，玫瑰花11两2钱，甘松4两8钱，公丁香11两2钱，木香9两6钱。
出处：《北京市中药成方选集》。
主治：伤风头痛，鼻塞清涕、暑令受热，晕车晕船。
加减：None
功效：芳香辟秽，通窍止痛。
用法用量：每服2分，滚开水送下，或闻入鼻窍。
制备方法：以上9味，共重147两2钱，共研为细粉，过罗，每20两细粉兑麝香5分，甘油10两，冰片5两，朱砂粉24两，薄荷冰5两，共14味研细和匀收贮，勿令泄气，装盒，重2分8厘。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
