序号：24169
名称：安荣散
组成：麦门冬1钱（去心），通草1钱，滑石1钱，当归半两（去芦，酒浸），灯心半两，甘草（炙）半两，人参1钱，细辛（洗）1钱。
出处：《医方类聚》卷二二四引《济生》。
主治：妊娠子淋。
加减：None
功效：通利小便。
用法用量：每服3钱，煎麦门冬汤调服，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：《医方集解》：此手太阴足太阳少阴药也。陈来章曰：虚热宜补，故用人参、甘草之甘；淋闷宜通，故用木通、灯草之渗，滑石之滑；肺燥则天气不降，而麦冬能清之；肾燥则地气不升，而细辛能润之；血燥则沟渎不濡，而当归能滋之也。
用药禁忌：None
附注：安荣汤（《宋氏女科》）。本方改为丸剂，名“安荣丸”（见《中国医学大辞典》）。
