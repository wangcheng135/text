序号：70166
名称：滋阴清化丸
组成：怀庆生地黄2两（酒浸，竹刀切捣），天门冬2两（去皮心，晒），陈皮（去白，盐水拌，微炒）1两，天花粉1两，贝母1两，熟地黄（酒浸，竹刀切）1两，麦门冬（酒浸透，去心捣）1两，薏苡仁（绢包，同糯米于砂锅内蒸1炷香，去米不用，晒干）1两，白茯苓（去皮，得人乳浸透更妙）1两，干山药1两，甘枸杞1两，白芍药（酒炒）1两，川玄参1两，五味子5钱，生甘草5钱。
出处：《扶寿精方》。
主治：诸虚。阴虚火动而后嗽者。
加减：None
功效：滋化源，清痰火。
用法用量：每服1丸，空心、临卧不时津液噙化；沸汤调下亦可。
制备方法：上为细末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方去陈皮、天花粉、贝母，加黄柏、知母各一两，名“滋阴清化膏”（见《回春》卷四）。
