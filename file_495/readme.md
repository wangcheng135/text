序号：495
名称：五积饼
组成：三棱（醋炒）1钱，莪术（醋炒）1钱，青皮（去瓤）1钱，陈皮1钱，木香1钱，黄连（姜汁浸炒）1钱，川楝肉2钱，槟榔2钱，神曲（炒）3钱，麦芽（炒）3钱，砂仁3钱，使君子肉5钱，胡黄连5钱，白术（炒）6钱，龙胆草6分，山楂肉2两，干蟾蜍5只。
出处：《古今医鉴》卷十三。
主治：小儿疳积、食积、虫积、肉积、气积、冷积，腹胀大如鼓，青黄肌瘦，泄泻发热，不能服药者。
加减：None
功效：None
用法用量：每服3-5饼。服过半月大效。
制备方法：上为细末，用炒过白面5斤，黑糖2斤，并前药和匀，用印印作饼子，约重1钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
