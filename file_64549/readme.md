序号：64565
名称：理本汤
组成：人参1钱，白术5钱，麦冬3钱，山药5钱，芡实5钱，巴戟天3钱，肉桂1钱，桔梗5分，贝母5分，白芥子2钱，防已3分，茯苓3钱，豨莶草1钱。
出处：《辨证录》卷二。
主治：痹症。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：痹症：人有一身上下尽行作痛，有时而止，痰气不清，欲嗽不能，咽喉气闷，胸膈饱胀，二便艰涩，人以为肺气之不行也，谁知是风寒湿之犯于三焦乎。治三焦必宜治肾，肾气旺而下焦之气始通；更宜治肺，肺气肃而上焦之气始降；尤宜治脾胃，脾胃健而中焦之气始化。理肺、肾、脾胃之气，而益之散邪之药，则三焦得令，而风寒湿又难去也。方用理本汤，四剂而上中下之气乃通，一身之病尽解，再用四剂，诸症全愈。
