序号：38439
名称：治发背膏
组成：明乳香2钱半，没药2钱半，麒麟竭2钱半（同研细匀），木鳖子（去壳，细切）半两，当归（去芦头，细切）半两，杏仁半两（汤浸，去皮，研），妇人梳下油头发2两（皂角洗净），黄丹6两（水飞过）。
出处：《医方类聚》卷一七七引《经验良方》。
主治：发背。
加减：None
功效：None
用法用量：临用时，用黄柏煮过油单纸摊膏药，贴患处，1-2日1换。
制备方法：上药先用清油10两于砂铫内，将木鳖子、当归、杏仁、头发，慢火煎熬炒黑色，用绵帛滤去滓，复将油入铫内，却下黄丹于油内，便用桃、柳枝各10条，不住手搅，慢火熬，稍久将油滴在水面上，凝结成珠不散，用手指点起油珠成膏，即提起砂铫，不用火，却将乳香、没药、麒麟竭3味入铫内，不住手搅匀，别以瓦器张盛油膏，坐于新汲水内出火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
