序号：60912
名称：大黄汤
组成：大黄1两。
出处：《保命集》卷中。
主治：痢疾，脓血稠粘，里急后重，腹痛，一脉实。泄痢久不愈，脓血稠粘，里急后重，日夜无度，久不愈者。便血，身热，脉胜。痢初作及久不愈，或脉实。休息痢。湿热痢无表邪者。
加减：None
功效：None
用法用量：好酒2大盏，同浸半日许，再同煎至1盏半，去大黄不用，将酒分为2服，顿服之。痢止1服，如未止再服。以利为度。
制备方法：上细锉。
临床应用：None
各家论述：《玉机微义》：此乃阳明经荡涤邪热之药，用酒煎者，欲其上至顶巅，外彻皮毛也。
用药禁忌：None
附注：将军饮（《古今医鉴》卷五）、酒煎大黄汤（《症因脉治》卷四）。
