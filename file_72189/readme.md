序号：72205
名称：椒朴丸
组成：川椒3两（炒，去闭口），厚朴1两半（制），干姜1两半（炒），小茴3两（盐水炒），茯苓3两，益智3两（盐水炒）。
出处：《医略六书》卷二十五。
主治：伤冷腹痛，泄泻，脉紧者。
加减：None
功效：温中散冷。
用法用量：每服3钱，米饮送下。
制备方法：上为末，酒糊丸。
临床应用：None
各家论述：胃伤生冷，寒结于中，不能敷化精微四达，故泄泻腹痛不止焉。川椒补火温中以散冷，厚朴散满除湿以宽中，干姜暖胃止痛，茯苓渗湿止泻，小茴温经络化气，益智摄寒涎厚肠。酒丸以行药力，米饮以和胃气也。使生冷消化，则脾胃调和而敷化有权，腹痛泄泻有不止者乎？
用药禁忌：None
附注：None
