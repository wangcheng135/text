序号：24932
名称：如圣丸
组成：地胆32个（去头足），斑蝥50个（去头翅足），盐豉70个，轻粉半钱（为衣）。
出处：《普济方》卷二四七。
主治：男子五种疝气；妇人产前经血不来，赤白带下，经血不止；产后恶物不行，脐腹撮痛。
加减：None
功效：None
用法用量：每服50丸，先饮好酒3-5盏，后用温酒送下，如重车行5里，小腹微觉痛，取下胞中病如雀脑相似，是病根；妇人产后，脐腹痛，恶物末尽，服7丸，童子小便送下；男子妇人痃气并小腹脐痛，每服1丸，温酒送下，妇人用童子小便下。
制备方法：上药用枣肉为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：忌食杂鱼、湿面及鸡、猪、马、牛等肉。
附注：None
