序号：54768
名称：栝楼散
组成：栝楼1个（去皮，焙，研为末，急用则烂研，子多者有力），当归（净洗，去芦，焙，细锉）半两，甘草半两（细锉，生用），通明没药1分（别研），乳香1钱（别研）。
出处：《集验背疽方》。
主治：妇人乳疽，奶劳。吹乳、结核。
加减：None
功效：None
用法用量：上用无灰酒3升，同于银石器中慢火熬，取1升清汁，分3服，食后良久服。如有奶劳，便服此药，可绝病根。如毒已成，能化脓成黄水；毒未成，即于大小便中通泄。疾甚再合服，以退为妙。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：瓜蒌散（《丹溪心法附余》卷十六）。本方方名，《妇人良方》引作“神效瓜蒌散”。
