序号：48832
名称：禹余粮丸
组成：禹余粮1两（烧，醋淬7遍），白石脂1两，鳖甲1两（涂醋，炙微黄，去裙襕），当归1两（锉，微炒），狗脊1两（去毛），白芍药1分，白术1两，附子1两（炮裂，去皮脐），桑寄生1两，柏叶1两（微炒），干姜1两（炮裂，锉），厚朴1两（去粗皮，涂生姜汁，炙令香熟），吴茱萸半两（汤浸7遍，焙干，微炒）。
出处：《圣惠》卷七十三。
主治：妇人带下久虚，胞络伤败，月水不调，渐成崩漏，气血虚竭，面黄体瘦，脐腹里急，腰膝疼重，肢体烦痛，心忪头眩，手足寒热。
加减：None
功效：None
用法用量：每服30丸，食前以热酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：吴茱萸丸（《杨氏家藏方》卷十五）。
