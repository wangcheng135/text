序号：58831
名称：秘传槟榔丸
组成：槟榔1两，三棱（煨，去毛，醋炒）半两，莪术（醋炒）半两，青皮（去瓤，麦麸炒）半两，陈皮（去白）半两，雷丸（去壳）半两，干漆（炒无烟）半两，使君子肉半两，山楂肉半两，麦糵面（炒）半两，神曲（炒黄）半两，芜荑（水洗净）2钱5分，鹤虱（略炒）3钱，木香（不见火）3钱，胡黄连3钱，甘草（炙）3钱，良姜2钱（陈壁土炒），砂仁1钱。
出处：《松崖医径》卷下。
主治：小儿疳病，积气成块，腹大有虫。
加减：None
功效：None
用法用量：每服50丸，空心以淡姜汤送下。
制备方法：上为细末，醋糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
