序号：54611
名称：柴胡煎
组成：柴胡（去苗）3分，升麻3分，栀子仁3分，芍药3分，钓藤1分，凝水石（研）1两，黄芩（去黑心）1两，知母（切，焙）1两，生葛汁1合，甘草（炙）1分，蜜2合，淡竹叶（细锉）3握，杏仁（汤浸，去皮尖双仁，炒，别研）半两。
出处：《圣济总录》卷一七一。
主治：小儿频惊，壮热欲作痫。
加减：None
功效：None
用法用量：百日儿每服如绿豆大，1岁儿如杏仁大，温浆水化破服，每日3次。
制备方法：上10味为粗末。以水3升，入银石铫内，文武火煎至1升，绵滤去滓，再入锅内，下蜜并葛汁、杏仁等，煎如饧，以瓷器盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
