序号：74136
名称：愈风饮
组成：川芎1钱2分，当归1钱2分，生地黄8分（姜汁炒），熟地黄8分（姜汁炒），红花4分（酒炒），牛膝8分（酒炒），半夏1钱（姜制），甘草4分（炙），橘红8分（去白，盐水洗），羌活6分，防风6分，天麻1钱，南星（姜制）1钱，白术1钱5分，白茯苓1钱，薄桂枝6分（冬月7分），黄芩8分（酒炒），酸枣仁8分（炒），白芍药2钱（酒炒），黄柏3分（酒炒，夏月5分）。
出处：《医便》卷三。
主治：中风。半身不遂，手足欠利，语言费力，呵欠喷嚏，口眼斜宽弛，头目眩晕，痰火炽盛，筋骨时痛，头痛心悸。
加减：冬月，减黄芩3分，加炮川乌2分，桂亦减半；风病减川乌，桂；羌活风家要药，若冬有感冒加至1钱。
功效：活血消痰，疏风顺气，走肌表，利关节。
用法用量：上作1服。水2钟，煎1钟，临服入姜汁、淡竹沥各3茶匙，清晨温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：愈风汤（《采艾编翼》）。
