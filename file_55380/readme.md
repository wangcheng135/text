序号：55395
名称：大神丸
组成：熟川乌（黑豆水煮）5钱，制草乌（姜汁、甘草水煮）5钱，大风子（去油壳，净）5钱，白僵蚕3钱，北全蝎（姜汁炒）5钱，北蝉蜕5钱，川山甲（土炒成珠）5钱，明雄黄2钱，台乌药5钱，漂苍术（童便浸1宿）5钱，北防风4钱，荆芥穗4钱，苏薄荷4钱，绿升麻2钱半，全当归5钱，大川芎4钱，大秦艽（酒洗）5钱，炙甘草（去皮）5钱，羌活7钱，生地黄4钱半。
出处：《疯门全书》。
主治：麻疯。
加减：None
功效：None
用法用量：加减同小神丸法。每服100丸，茶送下。
制备方法：上以老米饭为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌铁。
附注：服小神丸后，尚未除根，服此以断根。倘服药欲呕，煮乌豆一勺食之。倘毒气再发，外复施针灸熏洗各法治之。若全愈，明年再服小神丸一料，连服二三年，病根永除。
