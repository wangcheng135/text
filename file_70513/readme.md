序号：70529
名称：紫金丹
组成：信砒1钱半（研，飞如粉），豆豉1两半（好者，水略润少时，以纸挹干，研成膏）。
出处：《本事》卷二。
主治：多年肺气喘急，(口勾)嗽，晨夕不得眠。
加减：None
功效：None
用法用量：每服15丸，临卧以腊茶清极冷送下。以知为度。
制备方法：上用膏子和砒同杵极匀，丸如麻子大。
临床应用：None
各家论述：《古方选注》：信，白砒，有大毒，须煅炼得法，庶不伤人。凡白砒一钱，用石膏一两同研匀，贮熔银罐中，勿盖，阳煅通赤，俟其烟尽，飞去砒毒，仅取石膏，得白砒之性，复借豆豉挫砒毒之余威，兼领石膏中砒性上升，迅扫肺经痼积之沉寒，不使少留，一如太空廓清，疾返冲和气象，其愈痼之功，莫有过于此者。又按《必效方》治痰积齁喘，遇阴气触发，用砒与白矾为丸，冷茶送下，高枕仰卧即愈。治法虽同，而砒与矾能烂人肉，不及《本事方》佳。
用药禁忌：None
附注：定喘紫金丹（《痘疹金镜录》卷一）。
