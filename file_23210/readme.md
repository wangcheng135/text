序号：23218
名称：人参散
组成：人参2两（去芦头），地骨皮2两，鳖甲3两（涂醋，炙令黄，去裙襕），羚羊角屑2两，赤茯苓2两，知母1两半，柴胡3两（去苗），枳壳2两（麸炒微黄，去瓤），牛膝2两（去苗），赤芍药1两，贝母3两（煨令微黄），黄芩3分，栝楼根1两，当归3分，桃仁1两（汤浸，去皮尖、双仁，麸炒微黄）。
出处：《普济方》卷三一九。
主治：妇人骨蒸劳，身体壮热，手臂疼痛，月水不通，日渐瘦瘁，两胁气刺，四肢羸弱，腹内生块，时有咳嗽，不欲饮食。
加减：None
功效：None
用法用量：每服4钱，以水1盏，加生姜半钱，大枣3枚，煎至6分，去滓，食前温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
