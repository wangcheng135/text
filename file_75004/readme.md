序号：75020
名称：鲊汤丸
组成：龙脑1字，麝香1字，青黛（炒，末）3钱半，白丁香（炒，末）3钱，水银2钱，轻粉2钱，天南星2钱，滑石（炒，末）2钱，巴豆36粒（浸，去皮，烂研，用纸裹，去油再研）。
出处：《幼幼新书》卷九引《惠眼观证》。
主治：急慢惊风，伤寒呕逆，壮热，大小便闭塞，腹胀虚膨，渴水，疳虫攒心，赤白滞痢，惊膈，霍乱吐泻，脾风等疾。
加减：None
功效：下涎。
用法用量：1岁每服15丸，2岁20丸，3-4岁下25丸，5-6岁下32丸，余更随大小、虚实加减。下疳虫攒心，用皂子21个（炮过）捶损煎汤送下；赤白泻痢，小鱼鲊煎汤送下；其余候，并以葱白煎汤送下，一更时吃，至天明通下青白粘涎。候众人食时，先以淡粥补之，次进匀气散。如患急惊，只以此药捶碎下，亦吐涎来。或慢惊至第2日、第3日，补实脾气，下此药压涎亦得，不拘时候。
制备方法：上药合和令匀，入巴豆霜内，一向研三四百下，又倾出研脑、麝香，方入前药都研，复倾出研饭少许。如硬，入水数滴令匀烂，方却用药为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：忌生硬食2日；涎末下，不得吃水。
附注：None
