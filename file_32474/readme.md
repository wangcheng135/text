序号：32483
名称：岐伯神圣散
组成：天雄1两，附子1两，茵芋1两，踯躅1两，细辛1两，乌头1两，石南1两，干姜1两，蜀椒2两，防风2两，菖蒲2两，白术3两，独活3两。
出处：《千金》卷二十三。
主治：痈疽，癞，疥，癣，风痿，骨肉疽败，百节痛，眉毛发落，身体淫淫跃跃痛痒，目痛眥烂，耳聋、齿龋，痔瘘等。
加减：None
功效：None
用法用量：每服方寸匕，日3次，酒调下。勿增之。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：忌猪肉、冷水、生菜、桃、李、雀肉、青鱼鲊、饴、羊肉等。如觉麻痹甚者，啜少许温甘豆汤止之。
附注：茵芋散（《圣济总录》卷十八）。本方方名，《外台》引作“歧伯神散”，有防葵、枳实。
