序号：36706
名称：固本种子丸
组成：巴戟天2两，远志肉2两，石斛4两，杜仲4两，川牛膝2两，五加皮3两，青盐2两，生地3两，当归2两，大茴香1两，山茱萸2两（去核），沙苑蒺藜4两，益智子2两，羚羊角4两，锁阳12两，鱼鳔8两，枸杞10两，人参10两，破故纸6两，覆盆子2两，巨胜子4两，桑螵蛸、阿胶1斤半，龟版胶1斤半，肉苁蓉半斤，何首乌3斤，白茯苓8两，鸽蛋200，淫羊藿、黑驴肾1具，黄狗肾10具（炙脆，为末），鳖头1个。
出处：年氏《集验良方》卷二。
主治：None
加减：None
功效：固本种子。
用法用量：每服3钱，黄酒送下。寅、午、戌3时服3次，或早、晚2次。
制备方法：炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：不宜多服。忌房事一月。忌诸样血，不可食。
附注：方中螵蛸、淫羊藿用量原缺。
