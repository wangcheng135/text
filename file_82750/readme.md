序号：82766
名称：中候黑丸
组成：桔梗4分，桂4分，巴豆8分（去心皮），杏仁5分（去皮），芫花12分。
出处：《肘后方》卷四。
主治：水肿，痰饮，心腹胀满，大小便秘涩。诸癖结痰饮，小儿欲发痫。澼饮停结，满闷目暗。水从头面至脚肿，头眩痛，身虚热，名曰元水，体肿，大小便涩。
加减：None
功效：None
用法用量：服1丸取剂，至2-3丸。儿生10日欲痫，皆与1-2丸如粟粒大。诸腹内不便，体中觉患便服。得1两行利，则好也。
制备方法：并熬令紫色，先捣3味药为末，又捣巴豆、杏仁如膏，合和为丸，如胡豆大。
临床应用：None
各家论述：《千金方衍义》：取杏仁熬黑，以涤胸中宿垢，与巴豆破积不殊；并取芫花利水，桂心散血，桔梗上通肺金，下走大肠。所以水肿先从头面至足，头眩身热，亦取用之。《本经》治腹满肠鸣幽幽，岂非下走大肠之一验乎！
用药禁忌：忌猪肉、芦笋、生葱等。
附注：中军候黑丸(《千金》卷十八)。本方方名，《普济方》引作“中候姜黑丸”。
