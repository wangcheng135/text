序号：65061
名称：羚羊角丸
组成：羚羊角屑1两半，防风1两半（去叉），麻黄1两半（去根节），人参1两半，柏子仁1两半，诃黎勒皮1两半，白槟榔2两（煨，锉），熟干地黄2两（焙），大麻仁（研）2两，羌活2两（去芦头），茯神2两（去木），桂2两（去粗皮），芎2两，枳壳2两（麸炒，去瓤），杏仁（去皮尖双仁，炒）1两。
出处：《圣济总录》卷五。
主治：脾脏中风，言语謇涩，神思昏沉，口干食少，肢体虚汗，大便秘塞。

功效：None
用法用量：每服30丸，空腹温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
