序号：22036
名称：人参散
组成：人参2两（去芦头），桂心2两，陈橘皮2两（汤浸，去白瓤，焙），厚朴2两（去粗皮，涂生姜汁，炙令香熟），干姜2两（炮裂，锉），赤茯苓2两，杏仁2两（汤浸，去皮尖双仁，麸炒微黄），白术2两，甘草2两（炙微赤，锉），麻黄4两（去根节）。
出处：《圣惠》卷九。
主治：伤寒头痛体疼，寒热不解。
加减：None
功效：解表，利四肢，和胃气，调理荣卫，去久滞风邪。
用法用量：每服4钱，以水1中盏，加生姜半分，大枣3枚，煎至6分，去滓，不拘时候热服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：十味人参散（《博济》卷二）、十味人参汤（《圣济总录》卷二十一）。
