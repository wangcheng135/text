序号：43240
名称：养血百补丸
组成：人参半两，牡丹半两，槟榔半两，吴茱萸半两，肉豆蔻半两，白芍药半两，泽泻半两，木香半两，远志半两，缩砂半两，枳壳半两，柴胡半两，麻黄半两，麝香半两，盐半两，乌梅2两，知母1两，升麻1两，甘草1两，鳖甲1两，苁蓉1两，白蔹1两，葳蕤1两，虎骨1两，桃仁1两，羌活1两，防风1两，茯苓1两，附子1两，青蒿1两，秦艽1两，厚朴1两，牛膝1两，半夏1两，桂1两。
出处：《鸡峰》卷七。
主治：真元衰弱，营卫虚微，久病羸瘦，咳嗽痰涎，唾如胶粘，或如红物，手足心热，虽思饮食而吃不多，睡眠不安，肩背拘急，百节烦疼，足胫痿弱，行步无力，腹中如空，气短喘促，精神烦扰，郁抑悲啼。
加减：None
功效：None
用法用量：每服30丸，加至50丸，空心温酒送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
