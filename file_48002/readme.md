序号：48017
名称：神曲丸
组成：神曲、大麦糵、生地黄、牛膝、桑耳1斤，白术8两，姜黄8两，当归14两，桃仁12两，杏仁12两，生姜1斤，橘皮8两。
出处：《鸡峰》卷十五。
主治：妇人腹内冷癖血块，虚胀，月经不调，瘦弱不能食，面无颜色，状如传尸病。
加减：None
功效：令病人能食及驻颜色。
用法用量：每服方寸匕，渐加至1匕半，酒饮下，每日2次。若不能散，为丸服，每服30丸。
制备方法：上切碎，于臼中以木杵之如泥，纳瓶中，以物盖之，封，勿令泄气，蒸干饭米中，饭熟出之，停屋下3日，开出晒干为末。
临床应用：None
各家论述：None
用药禁忌：初服10日内，忌生冷、难消之物，以助药势；过10日外，即百无所忌，任意恣口食之，唯忌桃、李。服丸时忌桃、李、雀肉、芜荑。
附注：方中神曲，大麦糵、生地黄、牛膝用量原缺。
