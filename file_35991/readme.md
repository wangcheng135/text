序号：36001
名称：参苓白术丸
组成：人参1两，白术2两（土炒），白茯苓1两，干山药（炒）1两，莲肉（去皮）2两，陈皮1两，半夏（制）1两，白扁豆（炒）1两，薏苡仁（炒）2两，桔梗2两，黄连（姜炒）1两，神曲（炒）1两，香附1两，砂仁5钱，甘草1两，当归1两，远志1两，石菖蒲5钱。
出处：《古今医鉴》卷四。
主治：病后元气虚弱。
加减：None
功效：进美饮食，壮健身体，充实四肢，清火化痰，解郁固本。
用法用量：每服100丸，空心白汤送下。
制备方法：上为末，生姜、大枣煎汤，打神曲湖为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌食生冷、油腻之物。
附注：None
