序号：79825
名称：干漆丸
组成：干漆1两（捣碎，炒令烟出），穿山甲1两（炙令微黄），槟榔3分，乳香半两，京三棱半两（微炮，锉），桂心3分，川乌头半两（炮裂，去皮脐），硇砂1两（不夹石者，细研），阿魏半两（面裹煨，面熟为度），朱砂3分（细研，水飞过），鳖甲1两（涂醋，炙令黄，去裙襕），木香半两，巴豆20枚（去皮心，研，纸裹压去油）。
出处：《圣惠》卷七十一。
主治：妇人积聚，及恶血不散，多攻心腹疼痛，面无颜色，四肢不和。
加减：None
功效：None
用法用量：每服5丸，不拘时候，以热生姜酒送下；当归酒下亦得。
制备方法：上为末，炼蜜为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
