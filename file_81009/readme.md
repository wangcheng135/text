序号：81025
名称：霹雳散
组成：桂枝6两，公丁香4两，草果2两，川椒5两（炒），小茴香4两（炒），薤白4两，良姜3两，吴茱萸4两，五灵脂2两，降香5两，乌药3两，干姜3两，石菖蒲2两，防己3两，槟榔2两，荜澄茄5两，附子3两，细辛2两，青木香4两，薏仁5两，雄黄5钱。
出处：《温病条辨》卷一。
主治：中燥吐泻腹痛，甚则四肢厥逆，转筋，腿痛，肢麻，起卧不安，烦躁不宁；再甚则六脉全无，阴毒发斑，疝瘕；一切凝寒固冷积聚。
加减：None
功效：None
用法用量：开水和服，大人每服3钱，病重者5钱；小人减半。再病甚重者，连服数次，以痛止厥回，或泻止筋不转为度。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：寒轻者，不可多服；寒重者，不可少服，以愈为度。非实在纯受湿、燥、寒三气阴邪者，不可服。
附注：None
