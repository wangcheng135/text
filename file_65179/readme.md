序号：65195
名称：萎蕤酒
组成：萎蕤3两，升麻3两，荠苨3两，人参3两，大黄（锉，炒）2两，黄芩（去黑心）4两，葛根（锉）4两，紫草（去芦头）4两，犀角（镑）4两，栀子仁2两，芒消2两，银屑2两半，猪脂（腊月者）3两，露蜂房5两，甘草（炙，锉）2两，大豆（浸1宿，晒干，炒，去皮）1合。
出处：《圣济总录》卷一八四。
主治：乳石发动，诸药不效。
加减：None
功效：None
用法用量：每服取酒1-2盏饮之。
制备方法：上药除猪脂、银屑外，为细末，以无灰酒2升，蜜封渍浸一宿，次将猪脂用好酒1升炼开，以银屑相和，研入前药酒内，更浸一宿。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“葳蕤酒”。
