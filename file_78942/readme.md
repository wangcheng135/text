序号：78958
名称：增损五积丸
组成：黄连肝积5钱，脾、肾积7钱，心、肺积1两半，厚朴肝、心、肺积5钱，脾、肾积8钱，川乌肝、肺积1钱，心、肾、脾积5分，干姜肝、心积5分，肺、脾、肾积1钱半，人参肝、心、脾、肺积2钱，肾积5分，茯苓1钱半，巴霜5分。
出处：《杂病源流犀烛》卷四。
主治：积块，不拘脐上下左右。
加减：肝积，加柴胡1两，川椒4钱，莪术3钱，皂角、昆布各2钱半；心积，加黄芩3钱，肉桂、茯神、丹参各1钱，菖蒲5分；肺积，加桔梗3钱，天冬、陈皮、青皮、白豆蔻各1钱，紫菀、川椒各1钱半；脾积，加吴萸、黄芩、砂仁各2钱，泽泻、茵陈各1钱，川椒5分；肾积，加元胡索3钱，苦楝肉、全蝎、附子、独活各1钱，泽泻、菖蒲各2钱，肉桂3分，广香5分。
功效：None
用法用量：初服每次2丸，渐加，以微溏为度。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
