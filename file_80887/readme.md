序号：80903
名称：门冬清肺饮
组成：紫菀茸1钱5分，黄耆1钱，白芍药1钱，甘草1钱，人参5分（去芦），麦门冬5分，当归身3分，五味子3个。
出处：《内外伤辨》卷中。
主治：脾胃虚弱，气促气弱，精神短少，衄血吐血。劳伤气虚，火旺咳嗽。
加减：None
功效：None
用法用量：分作2服。每服水2盏，煎至1盏，去滓，食后温服。
制备方法：None
临床应用：None
各家论述：1.《杏苑》：方中人参、黄耆补中益气为君；紫菀、麦冬、五味泄火清肺金为臣；白芍、归身救阴血为使。_x000D_
2.《张氏医通》：此生脉、保元合用，以滋金化源，其紫菀佐黄耆而兼调营卫，深得清肺之旨。其余芍药酸收，当归辛散，且走血而不走气，颇所非宜，不若竟用生脉、保元清肺最妥。先哲有保元、生脉合用，气力从足膝涌出，以黄耆实胃，五味敛津，皆下焦之专药耳。
用药禁忌：None
附注：麦门冬饮子（《卫生宝鉴》人卫本卷十）、门冬饮子（《卫生宝鉴》拔粹本）、麦冬清肺饮（《杏苑》卷三）。
