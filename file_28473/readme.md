序号：28481
名称：花鸠丸
组成：花鸠1只（去毛肠嘴足，炙熟），羊肝1具（切，炒），细辛（去苗叶）5两，防风（去叉）5两，桂（去粗皮）5两，黄连（去须）5两，牡蛎（熬）5两，甘菊花5两，蒺藜子（炒去角）5两，白茯苓（去黑心）4两，瞿麦穗4两，羌活（去芦头）3两，蔓荆子2升（蒸3次），蕤仁半斤，决明子2合。
出处：《圣济总录》卷一一二。
主治：内障青盲翳晕，及时暂昏暗，一切眼疾。
加减：None
功效：None
用法用量：每服20-30丸，空心、日午、临卧茶、酒任下。半月见效。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
