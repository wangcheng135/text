序号：67848
名称：黄耆丸
组成：黄耆1两，杜蒺藜1两（炒，去刺），川楝子1两，茴香1两（炒），川乌1两（炮，去皮脐），赤小豆1两，地龙1两（去土，炒），防风（去芦叉）1两，乌药2两。
出处：《局方》卷五。
主治：丈夫肾脏风虚，上攻头面虚浮，耳内蝉声，头目昏眩，项背拘急；下注腰脚，脚膝生疮，行步艰难，脚下隐疼，不能踏地，筋脉拘挛，不得屈伸，四肢少力，百节酸疼，腰腿冷痛，小便滑数；及瘫缓风痹，遍身顽麻；妇人血风，肢体痒痛，脚膝缓弱，起坐艰难。刺风，气血内虚，风寒蕴滞，寒热相搏，遍身如针刺；肾脏风攻注腰脚生疮，或虚肿热痛，行步不得。
加减：None
功效：None
用法用量：每服15丸，空心以温酒送下，盐汤亦得，妇人醋汤送下。
制备方法：上为细末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
