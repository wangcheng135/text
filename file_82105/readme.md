序号：82121
名称：蕤仁煎
组成：蕤仁（去皮尖，铺在银盂底）2两，黄连（去须，净洗，铺在蕤仁上）2两。
出处：《圣济总录》卷一一一。
主治：眼生翳障昏暗，目涩赤肿隐痛。
加减：None
功效：None
用法用量：先用白汤洗眼，灯心点药入眼内，日3次。勿令见风。
制备方法：上用古老钱49文，铺黄连上，以井花水2盏，浸不得过钱，用七年熟艾4两，紧打成一炷，在古老钱上烧，密盖盂口，不令出风，候烧过艾至，去灰并古老钱、黄连、蕤仁等，取下艾烟在水内，入白蜜2两，同煎1盏，去滓，次用麝香、龙脑各2分，蓬砂1字，细研和膏，再熬热密封，入井水内，浸7日，出火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
