序号：77236
名称：酥蜜膏酒
组成：酥1升，崖蜜1升，饴糖1升，生姜汁1升，生百部汁1升，大枣肉（研为脂）1升，杏仁（去皮尖）1升（研），柑皮5具（末）。
出处：《外台》卷十引《删繁方》。
主治：肺虚寒，厉风所伤，声音嘶塞，气息喘惫，咳唾。寒郁热邪，声音不出。
加减：None
功效：止气嗽，通声。
用法用量：以温酒1升送服方寸匕，细细咽之，日2夜1。
制备方法：上合和，微火煎，常搅，三上三下约一炊久，俟姜汁及百部汁各减半则停。
临床应用：None
各家论述：《千金方衍义》：肺窍为风寒所袭而致喘咳上气，语声嘶塞。故用姜汁、杏仁、柑皮、百部温散肺络之结胶、饴、枣肉、乳酥、崖蜜通行脾肺之津，津回燥润，津自复矣。
用药禁忌：None
附注：酥蜜膏（《张氏医通》卷十三）。方中酥至杏仁用量原缺，据《千金》补。
