序号：46479
名称：济坤丸
组成：紫丹参1两，益智仁1两，木通1两，当归3两，桔梗1两，生地2两，龙胆草2两，远志（炙）5钱，天冬2两，枣仁（炒）2两，麦冬3两，草豆蔻5钱，川楝子4钱，乌药8钱，茯苓2两，白芍2两，于术8钱，健神曲、阿胶（炒珠）5钱，丹皮1两，青木香8钱，藏红花2两，枳壳（炒）1两，橘皮8钱，熟地4两，香附（炙）4钱，稻芽（炒）1两，玄胡索（醋炙）3两，青皮（炒）1两5钱。
出处：《北京市中药成方选集》。
主治：妇女经期不准，胸膈不舒，食欲不振，心跳不眠。
加减：None
功效：健脾和胃，养血安神。
用法用量：每服1丸，1日2次，黄酒化服；或温开水送下。
制备方法：上为细粉，过罗，炼蜜为丸。重3钱，朱砂为衣，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
