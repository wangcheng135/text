序号：44996
名称：春雪膏
组成：好净蜜1斤，川黄连4两（于砂锅内加甜水3碗，慢火熬至1半，绢滤滓），猪胆3个（取汁），生姜4两（切细烂研，取汁），轻粉1钱，南硼砂1钱，梅花脑子半钱，麝香半钱，蕤仁10个（去皮净，钵研如泥用），斑蝥5个，芫青5个，红娘子5个（肥大者，去头翅足，焙，研极细）。
出处：《永乐大典》卷一一四一三引《野夫多效方》。
主治：眼内翳膜遮障，瘀肉攀睛涩痛。
加减：None
功效：None
用法用量：每用竹杖子或骨柱子点粟米大，夜卧用。
制备方法：前4味，慢火熬一顿饭久，看稀稠得所，取下火来，候冷，入轻粉、蕤仁等5味研，再入斑蝥、红娘子等3味，同研千百遍，瓷盒内收。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
