序号：80185
名称：槟榔煎丸
组成：槟榔3两（锉，捣为末，酒1升熬成膏），吴茱萸（为末，醋1升熬成膏）1两，京三棱（为末，醋半升熬成膏）1两，硫黄1两，巴豆1两（去皮，以绢袋子盛，用水5升与硫黄同煮及1升将硫黄与巴豆同研），木香1两，白豆蔻（去皮）1两，肉豆蔻（去壳）1两，桂（去粗皮）1两，陈橘皮（汤浸，去白，焙）1两，青橘皮（汤浸，去白，焙）1两，高良姜1两，荜拨1两，诃黎勒皮1两，白术1两，胡椒1分，当归（切，焙）半两，干漆（炒烟出）半两，草豆蔻（去皮）1两。
出处：《圣济总录》卷七十三。
主治：痃癖气及两胁积聚，并妇人血刺疼痛。
加减：None
功效：None
用法用量：每服3-5丸，生姜汤送下，食后服。
制备方法：上为末，与前三味膏同搜为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
