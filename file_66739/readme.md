序号：66755
名称：鹿茸肾气丸
组成：熟地5两，萸肉3两，鹿茸3两（锉），丹皮1两半，山药3两（炒），茯苓1两半（蒸），泽泻半两，菟丝3两（焙），龟版3两（盐水炙），巴戟3两（炒），石斛3两（焙）。
出处：《医略六书》卷二十一。
主治：肾虚不能纳气，眩晕脉虚者。
加减：None
功效：None
用法用量：每服3-5钱，淡盐汤送下。
制备方法：上为末，炼蜜为丸。
临床应用：None
各家论述：熟地补阴滋肾脏，萸肉秘气涩精海；鹿茸壮元阳以归肾，龟版壮肾水以滋阴；山药益脾阴，茯苓渗湿热，丹皮平相火，泽泻泻浊阴；菟丝补肾填精，巴戟补火温肾，石斛以退虚热也。丸以白蜜，下以盐汤，使肾水充足，则虚炎自退，而真气无不归原，何眩晕之有？此补肾纳气之剂，为肾虚眩晕之专方。
用药禁忌：None
附注：None
