序号：58825
名称：秘传愈疯丹
组成：防风（去芦）1两，连翘1两，麻黄（去节）1两，黄连（酒炒）1两，黄柏（酒炒）5钱，川芎1两，川归（酒洗）1两，赤芍药（酒浸）1两，薄荷叶1两，石膏1两，桔梗1两，何首乌1两，熟地黄（酒洗）1两，羌活1两，细辛（减半）1两，甘菊花1两，天麻1两，黄芩1两5钱，白术2钱5分，荆芥穗2钱5分，山栀仁7钱5分，滑石5两（另研），甘草（炙）2两，僵蚕（炒）5钱。
出处：《松崖医径》卷下。
主治：一切风疾，偏正头风，半身不遂；及诸恶疮毒，赤白痢疾，痛风。
加减：热甚，加大黄、朴消各1两。
功效：None
用法用量：每服1丸，细嚼，用茶清或酒送下。
制备方法：上为细末，炼蜜为丸，如弹子大，以朱砂、金箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：愈风丹（《丹溪心法·附余》卷一）。
