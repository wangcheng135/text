序号：68808
名称：黄连厚朴汤
组成：黄连1两1分（去须），厚朴（去粗皮，生姜汁炙）1两1分，桑耳1两半，茯神1两半（去木），天雄1两半（炮裂，去皮脐），射干1两半，黄耆（锉，炒）1两半，代赭1两（碎），枳壳1两（去瓤，麸炒），桔梗1两（锉，炒），地榆1两，当归（切，焙）1两，白术半两（锉，炒），桂半两（去粗皮），黄芩（去黑心）半两。
出处：《圣济总录》卷一五一。
主治：妇人经气不调。
加减：None
功效：None
用法用量：每服3钱匕，以水1盏，加生姜3片，煎取7分，去滓温服。
制备方法：上（口父）咀，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
