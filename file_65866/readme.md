序号：65882
名称：萆薢丸
组成：萆薢1两，酸枣仁1两（微炒），独活1两，附子1两（炮裂，去皮脐），芎3分，石斛1两（去根，锉），仙灵脾3分，丹参3分，牛膝3分（去苗），当归3分，防风3分（去芦头），桂心3分，狗脊3分，赤箭3分，虎胫骨1两（涂酥，炙令黄），干蝎半两（微炒），海桐皮3分，木香3分，槟榔1两，麝香1分（细研）。
出处：《圣惠》卷二十三。
主治：风毒所攻，四肢拘挛，骨节疼痛，脚膝无力。
加减：None
功效：None
用法用量：每服10丸，温酒送下，不拘时候。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
