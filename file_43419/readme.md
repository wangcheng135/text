序号：43434
名称：养真丸
组成：羊肚1枚（去膏膜），羊肾1对（去膜），白术2两（炒），神曲1两半（炒），丁香（不见火）7钱半，荜茇7钱半，沉香（不见火）1两，熟干地黄（洗）1两，大附子（炮，去皮脐）1两，干姜（炮，洗）1两，荜澄茄1两，白茯苓（去皮）1两，当归（去芦，酒浸）半两，厚朴（去粗皮，姜制，炙）半两，白豆蔻仁半两，人参（去芦）半两，半夏曲半两，钟乳粉半两，天门冬3两半（去心），益智1两。
出处：《魏氏家藏方》卷七。
主治：脏腑不固。
加减：None
功效：补诸虚弱。
用法用量：每服50-70丸，空心、食前米饮送下。
制备方法：上（口父）咀。用羊肾切细，入在羊肚子中，以线缝肚子口，于净甑蒸极熟为度，趁热于木臼中捣碎，晒干或焙干，再研为细末，用熟枣肉为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
