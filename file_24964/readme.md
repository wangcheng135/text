序号：24972
名称：如圣膏
组成：蔷薇根1两（锉），乳香1两（研），阿魏（研）1两，铅丹6两，柳枝3两（锉，长1寸），清油1斤。
出处：《圣济总录》卷一三○。
主治：一切恶疮疖毒；赤眼，头痛眼涩；驴伤马坠，妇人血气；癣疮。
加减：None
功效：None
用法用量：一切疮肿，并用故帛上涂贴之；如患赤眼，头痛眼涩，贴太阳两穴；驴伤马坠，妇人血气，并当归酒下3丸，如梧桐子大；癣疮先抓破，取膏涂贴，以愈为度。
制备方法：上药先熬油令沸，下柳枝、蔷薇根煎，候黄黑色，以绵滤过，下丹煎搅，候变黑色，次下乳香、阿魏，更搅令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
