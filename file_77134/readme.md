序号：77150
名称：黑虎膏
组成：当归1两，防风1两，大黄半两，赤芍药半两，黄芩半两，黄柏半两，生地黄半两，黄连半两，玄参半两，桔梗半两，官桂半两，白芷半两，木鳖子仁半两，杏仁半两，血竭半两，猪牙皂荚半两，没药半两，乳香半两（别研），香油2斤，黄丹1斤（别研）。
出处：《普济方》卷三一五。
主治：一切痈疖疽毒，发背，脑疽，肠痈，痔瘘，疔疮，乳痈，虎狼刀箭所伤，一应无名肿毒，及颠扑损伤，车马榼伤，杖伤，悬痈。
加减：None
功效：None
用法用量：如痔瘘，丸如枣核扑纤入；肠痈，丸如鸡实大，每服3丸，甘草汤送下。
制备方法：上(口父)咀，药入油浸3日，铫内同煎油药，候白芷焦色为度，每用槐、柳枝各数10条搅动其油，文武火熬，却用布帛滤去滓，再入铫下丹，并乳、没末，不住手搅，熬至紫色，及有青烟起，急去火，紧搅，滴水中成珠为度；看时候冷热，加减油并丹，临时通变，倾于净器盛之，于净室修合。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
