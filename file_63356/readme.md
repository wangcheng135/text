序号：63372
名称：清毒活血汤
组成：紫草茸6分，当归（俱汤洗）6分，前胡6分，牛蒡（炒）6分，木通6分，生地黄5分，生白芍（俱酒洗）5分，连翘5分，桔梗5分，黄连7分，黄芩（俱酒炒）7分，甘草4分，人参3分，生黄耆8分，山楂肉8分。
出处：《痘疹活幼至宝》卷终。
主治：痘不成脓，其色红紫干枯，或焦黑，毒炽血凝，又痘稠密红紫而陷顶者。痘不如期灌浆，板硬干黄或灰滞紫暗干枯，此毒火伤其气血而浆不行。
加减：烦渴者，去黄耆，加人参、酒炒花粉；形成壮实者，减去人参、黄耆。
功效：None
用法用量：加生姜1片，水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
