序号：53553
名称：凌霄散
组成：蝉壳7个，地龙（炒）7个，白僵蚕7个，全蝎7个，凌霄花半两。
出处：《洁古家珍》。
主治：疠风。
加减：None
功效：None
用法用量：每服2钱，酒调下，于浴室内常在汤中住1时许服药效。
制备方法：上为极细末。
临床应用：None
各家论述：《医方考》：疠风攻凿气血，木石不能获效者，非其类也，故用血气之属，能主风者以治之。蝉退主风热，地龙主风湿，僵蚕、全蝎主风毒，凌霄花主风坏之血，斯五物者，皆有微毒，用之以治疠风，所谓衰之以属也。然必坐于浴室汤中服药者，所以开泄腠理，使邪气有所出尔！
用药禁忌：None
附注：五九散（《儒门事亲》卷十五）、凌霄花散（《医学入门》卷八）。
