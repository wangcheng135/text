序号：41088
名称：金不换木香丸
组成：大戟5钱，芜花（炒）5钱，甘遂5钱，黑丑头末2钱，生大黄5钱，青皮5钱，陈皮5钱，南木香5钱，青木香5钱，胡椒1钱（病合倍用），川椒（去白）5钱，槟榔5钱，益智仁5钱，射干5钱，桑白皮5钱，苦葶苈5钱（炒），大腹皮5钱，泽泻5钱，木通（去皮）5钱，连翘5钱，砂仁5钱，巴豆（去壳，半生半熟）5钱。
出处：《寿世保元》卷三。
主治：蛊胀。
加减：None
功效：None
用法用量：每服50丸，壮盛人加70-80丸，第一消头面肿，五更初用葱白酒送下；第二消中膈胸腹肿，五更初用陈皮汤送下；第三消脐以下脚肿，五更初用桑白皮汤送下。
制备方法：上为末，醋煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌一切生冷毒物，油盐酱醋，鱼酢鹅鸭及房事等。
附注：None
