序号：24871
名称：如圣青金丸
组成：龙脑1钱，麝香1分，香墨1钱半，腻粉1钱，白面3钱，使君子2个（以白面裹，慢火煨令熟），金箔10片（如无，少用），银箔10片（如无，少用），青黛2钱。
出处：《博济》卷四。
主治：小儿体热，忽发吐逆，夜惊啼，荏苒不解，或秘或泄，变成慢惊，或为疳疾。
加减：None
功效：定搐搦，疗疳病，坠痰涎，镇心神。
用法用量：慢惊风，每服1丸，薄荷水化下。服讫，须臾便睡，睡立愈，后更服2-3服。如些须小惊者，及急惊，只服半丸以下；慢惊，取下清涎为效。
制备方法：上为细末，滴井水为丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：青金丸（《圣济总录》卷一七○）、青金丹、睡惊丸（《普济方》卷三七三引《全婴方》）。本方方名，《幼幼新书》引作“如圣青金丹”。《圣济总录》有芦荟，治慢惊，三岁以上服一丸，以下服半丸。《普济方》本方用法：若吐泻成惊者，先与神宝丹1-2服，次用此药，涎下后再与神宝丹；若只吐不泻，便服此药，涎下后再与神宝丹少许。
