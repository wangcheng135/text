序号：77096
名称：黑龙丹
组成：舶上硫黄1两（以柳木捶研2-3日），巴豆1两（和壳记个数）。
出处：《证类本草》卷四引《博济》。
主治：阴阳二毒伤寒。
加减：None
功效：None
用法用量：若是阴毒，用椒49粒，葱白2茎，水1盏，煎至6分，服1丸；阳毒，用豆豉49粒，葱白2茎，水1盏同煎，吞1丸，不得嚼破。
制备方法：用2升铛子一口，先安硫黄铺铛底，次安巴豆，又以硫黄盖之，醋半升以来浇之，盏子盖合令紧密，更以湿纸周回固济，缝勿令透气，缝纸干，更以醋纸湿之，文武火熬，常着人守之，候里面巴豆作声，数以半为度，急将铛子离火，便入臼中，急捣令细，再以米醋些子，并蒸饼些小，再捣令冷，可丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
