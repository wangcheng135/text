名称：黑末子
组成：雄鸡毛（烧）4两，桑炭4两，老松节（炒存性）4两，嫩松心4两，侧柏叶（醉煮）4两，当归2两，牛膝2两，何首乌2两，黑豆（制）2两，南星（制）2两，骨碎补2两，熟地黄2两，羌活2两，独活2两，赤芍药2两，川芎2两，白芷2两，细辛1两，肉桂1两，川乌（炮）1两，草乌（制）1两，木鳖子1两，南木香1两，五灵脂1两，降真香1两，乳香1两，没药1两，枫香1两，百草霜5钱。
