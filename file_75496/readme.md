序号：75512
名称：解毒汤
组成：连翘1钱，牛蒡1钱，枳壳6分，木通6分，防风5分，桔梗5分，紫草7分，川芎4分，升麻4分，虫退4分，黄芩4分，黄连4分，前胡1钱，麦冬8分，甘草3分。
出处：《种痘新书》卷四。
主治：痘疮外感风寒，毒气壅盛，憎寒壮热，咳嗽流涕，服加减升麻汤、扶元宣解汤后，依然大热熏蒸，眼红唇紫，舌有黄苔，口中气臭，狂言谵语，二便不通，恶风恶寒，嘎齿咬牙，腹中隐隐作痛者。
加减：None
功效：解毒。
用法用量：None
制备方法：None
临床应用：None
各家论述：翘、蒡、芩、连解毒除烦，枳、桔、防、木疏风开窍，芎、虫退达气上升。清毒之方，莫良于此。眼之觉烦闷少解则止，候痘出而中自安，而热亦渐退矣。盖热以发痘，痘未出，其热终不退也。
用药禁忌：None
附注：None
