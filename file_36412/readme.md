序号：36422
名称：和血通经汤
组成：当归5钱，京三棱（炮）5钱，广荗（炮）4钱，木香3钱，熟地黄3钱，肉桂3钱，红花2钱，贯众2钱，苏木2钱，血竭1钱（另研）。
出处：《卫生宝鉴》卷十八。
主治：妇人室女受寒，月事不来，恶血积结，坚硬如石，结为石瘕。寒侵子宫，瘀血积聚，小腹胀大，状如怀孕，经闭不通，时发胀痛，倦怠瘦弱。
加减：None
功效：和血化瘀。
用法用量：每服3钱，食前热酒1盏调下。
制备方法：上除血竭外，同为细末，和匀。
临床应用：None
各家论述：《医略六书》：寒气内凝，血积不散，女子不月，而成石瘕，故腹中疼痛不已焉。当归养血、和血脉，熟地补血、滋血室，苏木通经破瘀，血竭散瘀破血，贯众祛湿热之积，木香行结滞之气，红花活血散血，肉桂暖血温经，三棱消坚破积，广荗破血消癥。水、酒煎服，使寒凝解散，则坚积自消而经络清和，何血结石瘕之有哉？此破血消癥之剂，为寒凝石瘕之专方。
用药禁忌：忌生冷及当风大小便。
附注：本方方名，《增补内经拾遗》引作“和血通经散”、《中国医学大辞典》引作“和血归经汤”。《全国中药成药处方集》将本方改为丸剂，名“和血通经丸”。
