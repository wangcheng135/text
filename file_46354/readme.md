序号：46369
名称：活络丹
组成：萆薢4两，川乌5钱（去皮脐，切4块），金毛狗脊4两（切作片，去毛），苍术5钱（去皮，切作片，炒），破故纸（拣，炒）5钱，杜仲5钱（细切，姜汁浸，炒去丝），仙灵脾（切）5钱，吴茱萸（炒）5钱，续断5钱（切），小茴香（炒）1两，独活（切）1两，薏苡仁3两，猪牙皂角（去皮丝，切作1寸）2两。
出处：《普济方》卷九十三。
主治：男子、妇人瘫痪，筋挛骨疼，腰膝疼痛，口眼斜，言语謇涩，目晦耳聋，头风，心气痛。
加减：None
功效：None
用法用量：每服50-70丸，空心温酒或盐汤送下，与七乌丸相间服。
制备方法：上作一处，用好酒3升，于瓷瓶内浸1宿，次日以文武火煮至约酒汁1升，撩出焙干，为细末，用煮药酒拌面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：孕妇不可服。
附注：None
