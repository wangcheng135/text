序号：56052
名称：桑螵蛸汤
组成：桑螵蛸（炙）10枚，牡丹皮半两，白术（米泔浸1宿，锉，炒）1两，白茯苓（去黑皮）1两，当归（切，焙）1两，桂（去粗皮）1两，牡荆子（炒）1两，磁石（煅，醋淬7遍）1两，附子（炮裂，去皮脐）1两，菖蒲（米泔浸1宿，锉，焙）1两，熟干地黄（焙）1两，大黄（锉，炒）半两，细辛（去苗叶）半两，芎藭半两。
出处：《圣济总录》卷一一五。
主治：虚损耳聋。
加减：None
功效：None
用法用量：上（口父）咀。每服3钱匕，先以水3盏，煮猪肾1只，取汁1盏，去肾下药，煎至7分，去滓，食前温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
