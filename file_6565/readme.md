序号：6565
名称：二术丸
组成：白术8两（土炒），苍术4两（泔浸），生姜4两（切片），大枣100个（去皮核，同生姜屑煮）。
出处：《陈素庵妇科补解》卷一。
主治：妇人脾胃虚弱，始则行经血少而色淡，后且闭绝不通；并治男子、小儿脾虚洞泻。
加减：None
功效：None
用法用量：每日空心米饮下100丸。
制备方法：枣肉为丸。
临床应用：None
各家论述：苍、白术辛温，性雄壮，脾胃二经主药也。脾胃虚寒服之，则饮食倍进，肌肉渐充。然虚人服术恐中满，故又加姜、枣以辅之，大枣甘温，脾药也，用以为引；生姜辛温，气药也，用以为佐。不特治妇人，凡男子小儿脾虚洞泻者尤宜。
用药禁忌：None
附注：None
