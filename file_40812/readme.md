序号：40827
名称：软金丸
组成：大黄3两，牵牛3两，皂角3两，朴消半两。
出处：《宣明论》卷四。
主治：一切热疾。诸气愤郁，肠胃干涸，皮肤皴揭，胁痛，寒疟，喘咳，腹中鸣，注泄骛溏，胁助暴痛，不可反侧，嗌干面尘，肉脱色恶，及丈夫(疒颓)疝，妇人少腹痛，带下赤白，疮疡痤疖，喘咳潮热，大便涩燥，及马刀扶瘿之疮，肝木为病；老人久病，大便涩滞不通者。
加减：None
功效：None
用法用量：每服自10丸服至30丸，食后白汤送下。
制备方法：上为末，滴水为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：四生丸（原书目录卷四）、润肠丸（《儒门事亲》卷十二）。按：《儒门事亲》本方用各等分，为末，水丸，如梧桐子大，每服七八十丸，食后温水送下。
