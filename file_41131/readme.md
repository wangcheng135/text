序号：41146
名称：金牙散
组成：金牙（别研）1两，防风（去叉）2两，侧子（炮裂，去皮脐）2两，当归（切，焙）2两，石膏（别研细）2两，桂（去粗皮）2两，芎1两半，白术（微炒）3两，泽泻1两半，细辛（去苗叶，轻炒）1两半，黄芩（去黑心）1两半，赤茯苓（去黑皮）1两半，石南叶（洒酒炒）2两，人参2两。
出处：《圣济总录》卷二十。
主治：周痹，脚胫细瘦，痿弱不能行立。
加减：None
功效：None
用法用量：每服1钱半匕，渐加至2钱匕，空心温酒调下，日2，夜1。未觉，更增药至2钱半。
制备方法：上除金牙、石膏别研外，将12味为末，方入金牙、石膏末拌匀，重罗。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
