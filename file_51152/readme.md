序号：51167
名称：荜澄茄丸
组成：荜澄茄1两，白豆蔻（去皮）1两，肉豆蔻（去壳）1两，木香1两，草豆蔻（去皮，炒）1两，丁香1两，白术1两，缩砂仁1两，红豆蔻1两，桂（去粗皮）1两，益智（去皮）1两，诃黎勒（煨，去核）1两，人参1两，白茯苓（去黑皮）1两，附子（炮裂，去皮脐）1两，茴香子（舶上者，炒）1两，槟榔（锉）1两，胡椒1两，干姜（炮）1两，阿魏（面裹煨，去面）1两，青橘皮（汤浸，去白，焙）2两，陈橘皮（汤浸，去白，焙）2两，甘草（炙）4两。
出处：《圣济总录》卷五十五。
主治：胃心痛。腹胀满，口吐酸水，饮食无味，及一切气疾。
加减：None
功效：None
用法用量：每服1丸，细嚼，温酒或盐汤送下，妇人醋汤送下。
制备方法：上为末，炼蜜为丸，如樱桃大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
