名称：散阴膏
组成：老生姜1斤，葱白1斤，韭白1斤，蒜头1斤，白凤仙花（茎、子、叶、根全铢）1斤，桑枝1斤，槐枝1斤，柳枝1斤，桃枝8两，干柏叶4两，艾叶4两，上以小磨麻油5斤熬炒，黄丹收膏，入白芥子2两，川椒末2两，干姜1两，炮姜末1两，搅匀听用。生白附子4两，生附子3两，生川乌3两，生草乌3两，生大黄3两，苍术2两，甘遂2两，生南星2两，生半夏2两，川芎2两，当归2两，麻黄2两，天麻1两，荆穗1两，防风1两，紫苏1两，细辛1两，甘草1两，羌活1两，独活1两，蝎尾1两，陈皮1两，赤芍1两，白芷1两，紫荆皮1两，石菖蒲1两，灵仙1两，蓖麻仁1两，木鳖仁1两，乳香1两，没药1两，骨碎补1两，续断1两，官桂1两，广木香1两，延胡1两，灵脂1两，白及1两，白蔹1两，僵蚕1两，皂角1两，炮甲1两，桃仁1两，红花1两，自然铜（煅）1两，蛇蜕1两，露蜂房1两，干地龙1两，杜仲1两，香附1两，木瓜1两，五加皮1两，雄黄1两，明矾1两，苏木1两，远志1两，乌药1两，牛膝1两，蛇床子1两，秦艽1两，姜厚朴5钱，蚕砂1两6钱，发团2两4钱上以小磨麻油10斤熬炒，黄丹600两收膏，加提净松香8两，赤石脂（煅）2两，陀僧4两，陈石灰（炒）4两，鹿角胶（或用广胶代）4两，苏合油1两，樟脑1两，搅匀。
出处：《理瀹》。
主治：风寒湿痹，筋骨疼痛，及一切气血痰凝滞，阴酸漫肿疼痛，肾虚腰背痛，受寒腹痛，跌扑闪挫诸痛，疟疾，火衰泄泻，脾虚久痢，疝气，冻疮，阴疽，湿痰流注等。
加减：None
功效：None
用法用量：治三阴疟，膏上掺灵仙、肉桂、吴萸、丁香、白胡椒末，贴背石第三骨节下；火衰泄泻，掺肉寇仁末，贴命门穴；疝气，掺川楝、茴香末贴脐下；治风气痛，贴膏后，可用太乙针药末卷纸捻点火熏之，或桑枝扎把烧熏亦可，或炒蚕砂、或炒花椒熨之；治湿痰流注及阴疽等，贴膏后外用白及磨生姜汁围之则不走。
制备方法：将上两膏合并，摊贴。
临床应用：None
各家论述：None
用药禁忌：孕妇酌用。
附注：None
