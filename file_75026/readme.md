序号：75042
名称：慢白汤
组成：西党参9g，全当归9g，生白术9g，生黄耆15g，怀山药15g，云茯苓15g，熟枣仁15g，制首乌15g，银柴胡3g，炒白芍6g，大红枣6枚。
出处：方出《中医临证撮要》，名见《古今名方》。
主治：慢性白血病。头昏耳鸣，心悸气短，纳食不香，面色萎黄，浮肿，腰痠腰痛，疲乏无力，潮热，腹胀，大便时结时溏，舌苔淡薄，或薄白，脉象细濡涩，或沉微迟。
加减：骨蒸潮热，去生白术、当归、红枣，加生龟版、生鳖甲各24g，地骨皮、青蒿梗各9g；盗汗，去当归、银柴胡，加粉龙骨15g，牡蛎24g（先煎），麦冬12g，五味子3g。口干，去当归、白术、大枣，加北沙参9g，冬青子15g，粉丹皮6g；肝脾肿大，去生黄耆，加炙鳖甲24g（先煎），大丹参9g，粉丹皮6g颈部腋下瘰疬，去党参、黄耆、当归、白术、怀山药、红枣，加夏枯草12g，黑元参12g，川贝母9g，天花粉12g，生牡蛎24g，粉丹皮6g；鼻衄，去党参、黄耆、当归、白术、山药、银柴胡、大红枣，加焦山栀、肥知母各9g，侧柏叶、大生地、旱莲草、黑元参各12g，冬青子15g，京赤芍6g。
功效：补气血，益脾肾。
