序号：49130
名称：胜金丸
组成：槟榔4两，常山（酒浸，蒸，焙）1斤。
出处：《局方》卷八（宝庆新增方）。
主治：一切疟病。
加减：None
功效：截疟。
用法用量：每服30丸，于发前1日晚，临卧用冷酒吞下便睡，不得吃热物、茶汤之类；至四更尽，再用冷酒吞下15丸。一方用川常山16两为末，鸡卵15只取清为丸，服饵如前法。
制备方法：上为末，水面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌食一切热羹汤、粥食，午间可食温粥，至晚方可食热。忌一切生冷、鱼腥等物；忌油腻。
附注：胜金丹（《医学入门》卷七）。本方方名，《证治要诀及类方》引作“截疟丹”。《医学入门》本方用法：醋糊丸，绿豆大。每服三十丸，隔夜临卧冷酒送下，次早再进一服；血虚，当归煎汤送下；气虚，人参煎汤送下；痰多，贝母煎汤送下。
