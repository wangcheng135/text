序号：23258
名称：人参温脾散
组成：人参半两，白茯苓半两，白术（炮）半两，肉豆蔻（面裹，煨）半两，木香半两，黄耆（锉）半两，白附子半两，藿香半两，陈皮（汤浸，去白，焙干）半两，白僵蚕（直者佳）半两，防风半两，羌活半两，冬瓜子仁（微炒，取仁）半两，山药半两，芎半两，北南星半两（每个破作4分，同醋涂黄土煮，内1点黄豆大白为度，洗去泥用），甘草1钱（炙微赤）。
出处：《永乐大典》卷九八一引《方便集》。
主治：小儿慢脾风，脾虚胃冷，作吐泻。
加减：None
功效：温调脾胃。
用法用量：3岁儿每用1钱，水1盏，加生姜3片，淮枣1个，煎至5分，去滓温服，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
