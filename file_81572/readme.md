序号：81588
名称：镇心爽神汤
组成：石菖蒲（去毛）半两，甘草（炙黄）4钱，人参（去芦）3钱，赤茯苓（去皮）3钱，当归（酒浸）3钱，南星（炮）1分，橘皮（去白）2钱，干山药2钱，紫菀（去芦）2钱，半夏（汤洗7次）2钱，川芎（不见火）2钱，五味子（去梗）2钱，细辛（去苗）2钱，柏子仁（微炒）2钱，枸杞子2钱，酸枣仁（浸，去壳，炒）1钱半，通草1钱半，麦冬（去心）1钱半，覆盆子1钱半。
出处：《简易》引《叶氏方》（见《医方类聚》卷一五○）。
主治：心肾不交，上盛下虚，心神恍惚，多惊悸，小便频数，遗泄白浊。
加减：None
功效：镇心安神。
用法用量：每服3钱，水1大盏，加蜜1匙，煎取5分，去滓，入麝香少许，再煎1-2沸，放温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
