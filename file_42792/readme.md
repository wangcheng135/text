序号：42807
名称：保命丸
组成：苦参10斤，草胡麻5斤，当归5斤，防风5斤，芜荑5斤，白蒺藜5斤，大风肉2斤，薄荷叶2斤，土木鳖2斤，荆芥2斤，胡连12两，银柴胡12两。
出处：《疡医大全》卷二十八。
主治：大麻风。
加减：如身发多疮，或生湿毒，通身如蛇皮，或破流血水者，是脾经受毒，加白术10两；如手足破伤，麻木遍身，发紫块、白瘢癜，满身肉痛，此胃经受毒，加厚朴10两；如眉发脱落，遍身发疮癣，满头面上蝇虫痕，此肺家受毒，加黄芩10两；如面起紫泡，身如紫云朵，四肢先见者，乃肝属木而有此疮也，春2-3月、秋9-10月各发1次，此肝经受毒，加山栀、连翘各1斤；如脚底先穿，眼泪如珠流者，遍身骨节刺痛，又发血癣如桃花朵者，此心经受毒，加黄连、山栀各1斤；如遍身起紫块，手足拳挛，口眼歪斜，此肾经受毒，如破故纸10两；如五脏受病，则藏于内，不发于外，多生冷疾，手足如瘫痪软状者，加苍术、黄柏各8两，甘草4两；五脏受风者，加川续断、何首乌、威灵仙各半斤；遍身浮肿，加乳香（去油）、没药（去油）各4两；骨节疼痛，加虎骨3斤；如六腑受病，如紫云块，似核桃风，多由内热，两鼻出血，筋脉弛长，四肢无力，行步艰难，加威灵仙、川续断各8两，何首乌1斤。春则地气融和，万物发生，加连翘2斤；夏则火旺烦躁，加黄连2斤，薄荷1斤；秋则雾露乍寒乍热，加白术1斤，苍术2斤；冬则严寒地冻，加乌药2斤。
功效：None
用法用量：每日服4次，约2合，细茶送下。轻者不过7-8升，重者1斗5升，再重者2-3斗痊愈。
制备方法：上为末，以水为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
