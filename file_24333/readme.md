序号：24341
名称：八子丸
组成：青葙子1两，决明子（炒）1两，葶苈子（炒）1两，车前子1两，五味子1两，枸杞子1两，地肤子1两，茺蔚子1两，麦门冬（去心，焙）1两，生干地黄（洗，焙）1两，细辛（去苗叶）1两，肉桂（去粗皮）1两，赤茯苓（去黑皮）1两，泽泻1两，防风（去叉）1两，黄芩（去黑心）1两。
出处：《圣济总录》卷一一一。
主治：风毒热眼，翳膜侵遮，不计久新，及一切内外障眼，暴发赤痛，干涩昏花。
加减：None
功效：祛风热，补肝肾，疏气血。
用法用量：每服20丸至30丸，茶清送下，温米饮亦得，日3次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌食葱、蒜、辣物。
附注：None
