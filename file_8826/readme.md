序号：8826
名称：水金丹
组成：透明硫黄1斤，轻粉1两。
出处：《百一》卷一引钱观文方。
主治：男子妇人一切虚危痼冷，肠滑不禁，腹内缠疼，泻注不已，手足厥逆，饮食生冷吐泻不止；兼治妇人室女赤白带下，面黄萎瘦。
加减：None
功效：补暖丹田，壮元阳。
用法用量：每服7丸或10丸，空心人参汤送下。
制备方法：上先将硫黄研令极细，于1斤之内取研细硫黄1两，与轻粉1两合和同研一时辰许，别顿一处；先用真蚌粉10斤，于一片新瓦上，实填瓦口令平，次用银盂子1枚可盛硫黄末1斤以上者，顿瓦中心，四边用蚌粉紧拥作池子，极要实；然后轻手脱去盂子，将15两研细硫黄末用一大匙抄入池子内，次入合和轻粉硫黄末2两铺盖顶上，以匙捺，令小实；用熟火5斤，就瓦四边煅之，候硫黄成汁，透底造化，硫黄、轻粉二气融和，用细蚌粉一大盂猛罨药汁之上；其残火留经宿，直至寒炉；取之已成一片，刷去蚌粉尽，净研令极细，用面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
