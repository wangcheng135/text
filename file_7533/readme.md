序号：7533
名称：四神丹
组成：苦花子2两（又名毛连子，又名小叶金鸡舌，梗叶俱用），土木香2两（根名青木香，梗名天仙藤，花名马兜铃），仙人薯（用根，新鲜生者为妙，干者次之）2两，晚蚕砂1两。
出处：《准绳·疡医》卷二。
主治：因剥割瘴死牛、马、猪、羊、不避其气，以中其毒；或因食瘴死牛、马、猪、羊之肉者，或手足各处发疔毒，或起紫泡，或起堆核，初则创人，次渐肿大，疼痛不可忍，瞀闷发热，口渴心烦，四肢强痛，头目昏花，及一切瘴毒。
加减：None
功效：None
用法用量：上铡碎，擂，水和煮粽汁，冷服。次服劫瘴消毒散。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
