序号：32087
名称：万应丹
组成：香附4两（酒、醋、童便，乳汁4制，各拌浸1两），皂矾2两（隔纸炒），干漆4钱（炒净烟），陈皮5钱（盐水炒），山棱1两（醋制，炒），青皮5钱（炒）山栀5钱（炒），麦芽1两（炒），干姜5钱（炒），针砂3钱（炒），神曲1两（醋制，炒），枳壳1两（炒），枳实1两（炒），山楂2两（炒），莪术1两（醋制，炒），乌药2两（炒），大黄1两（酒制，炒），良姜5钱（炒），槟榔5钱（炒）胡连5钱（水炒），猪苓5钱（炒），泽泻5钱（炒），厚朴1两（去皮，姜汁制，炒），川楝子5钱（酒制，炒），使君子肉5钱（炒）。
出处：《良方合壁》卷上。
主治：黄疸，痞块；妇人痛经；小儿疳膨食积。
加减：None
功效：None
用法用量：男妇每服2钱，滚汤送下；小儿每服1钱，用黄糖调下。
制备方法：上药锅内炒黑，为极细末。
临床应用：None
