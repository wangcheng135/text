序号：55236
名称：桂蜜汤
组成：桂心2两，蜜1升，附子1两，干姜2两，甘草2两，当归2两，赤石脂10两。
出处：《千金》卷三。
主治：产后余寒下痢，便脓血赤白，日数十行，腹痛时时下血。
加减：None
功效：None
用法用量：上(口父)咀。以水6升，煮取3升，纳蜜煎1-2沸，分3服，日3次。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：夫蜜润肠，下痢所禁，反取用之，以产后泄脱无度，非四逆、桃花无以疗之，虽证属虚滑，而痢久津液逮匮，急需滋干导滞以协济之。考之本草，蜜性虽滑，熟则温中，姜虽辛热，炮则苦平，可知蜜有和解辛热之功，姜具散火安中之力，而桃花汤中粳米煎，胶蜡汤中已借用之。此方既有桂心、当归、甘草、熟蜜温中和血，可无籍于粳米也。
用药禁忌：None
附注：桂心汤（《千金翼》卷七）。本方方名，《外台》引作“桂枝汤”。
