序号：35201
名称：乳香应痛丸
组成：龙骨（酒浸1宿，焙干，研粉，水飞3度，晒干）4两半，蜈蚣6条（去尾针，以薄荷叶裹，煨熟），赤小豆（生用）6两，虎骨（酥炙焦）6两，白僵蚕（炒，去丝嘴）12两，草乌头（炮，去皮尖）12两，白胶香（拣净，炼过）3两，天麻（去芦，洗）3两，川牛膝（酒浸，去芦）3两，川当归（去芦，酒浸）3两，全蝎（去尾针，微炙）70个，乳香（研）6钱，木鳖仁72个（别研）。
出处：《局方》卷一（淳祐新添方）。
主治：一切风气，左瘫右痪，口眼斜，半身不遂，语言謇涩，精神恍惚，痰涎壅塞，筋脉拘挛；或遍身顽痹，走注疼痛，脚膝缓弱，行步艰难；又治打扑伤损，瘀血不散，痛不可忍；或行路劳伤，脚膝浮肿疼痛；或肾脏风毒上攻，面肿耳鸣，下注，脚漆沉重；及治偏正头痛，攻注眼目。
加减：None
功效：None
用法用量：每服5-7丸，冷酒送下，或冷茶清下亦得，不拘时候。
制备方法：上为细末，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌诸热物一时辰久。此药但临睡服尤妙，忌湿、面、炙煿、鲊脯、发热、动风等物。
附注：None
