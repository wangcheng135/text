序号：56402
名称：海桐皮散
组成：海桐皮（锉）1两，五加皮（去粗皮，锉）1两，萆薢（炒）1两，薏苡仁（炒）1两，虎骨（涂酥，炙，令黄）1两半，枳壳（麸炒，去瓤）1两半，赤芍药1两半，牛膝（去苗，酒浸，切，焙）1两半，恶实（炒）半两，防风（去叉）1两，续断1两，杜仲（去粗皮，锉，炒）1两，郁李仁（汤退去皮尖双仁，炒）1两，熟乾地黄（焙）1两。
出处：《圣济总录》卷五。
主治：肾中风。踞而腰痛，脚肿疼重，耳鸣面黑，志意不乐。
加减：None
功效：None
用法用量：每服2钱匕，温酒调下，渐加至3钱匕，空腹、食前各1次。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
