序号：47793
名称：神仙退云丸
组成：川芎1两半，当归1两半，犀角（酒洗）半两，枳实半两，川楝半两，蝉壳半两，甘菊半两，薄荷叶（不见火）半两，瓜蒌仁（生者）6钱，蛇蜕2钱，蜜蒙花2钱，荆芥穗2钱（此3味与甘草同焙干，去甘草不用），地骨皮（洗）1钱，白蒺藜（微炒，去刺）1钱，羌活1钱，生地（酒洗，焙干）1钱，川木贼1两半（去节，童便浸1宿，焙干）。
出处：《医学纲目》卷十三引李东垣方。
主治：一切翳晕，内外障，昏无晴者。阴虚有热，眼生翳膜。
加减：None
功效：None
用法用量：日进2-3丸，食后米泔汤调服；妇人，当归汤送下；有气者，木香汤送下。
制备方法：上为细末，炼蜜为丸，每1两分作10丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：视仙退翳丸（《脉因证治》卷下）、经验神仙退云丸（《直指附遗》卷二十）。
