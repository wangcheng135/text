序号：27267
名称：补肾肾沥汤
组成：磁石5两（烧醋淬7遍，捣碎，以帛包之），肉苁蓉1两（酒浸去皴皮，微炙），黄耆1两（锉），人参1两（去芦头），白茯苓1两，芎1两，肉桂1两（去破皮），菖蒲1两，当归1两（锉，微炒），熟干地黄1两，石斛1两（去根），覆盆子1两，干姜1两（炮裂，锉），附子1两（炮裂，去皮脐），五味子1两。
出处：《圣惠》卷七。
主治：肾脏久虚，体瘦骨疼，腰痛足冷，视听不利，食少无力。
加减：None
功效：None
用法用量：每服5钱，水1大盏，以羊肾1对切去脂膜，每与磁石包子同煎至5分，去滓，空心及晚食前温服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
