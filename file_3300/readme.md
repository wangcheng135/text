序号：3300
名称：九阳丹
组成：辰砂1两，雄黄1两，雌黄1两，阳起石1两，硫黄1两，石燕子1两，禹余粮1两，牡蛎1两，紫石英（并研）1两，钟乳粉1两，鹿茸1两，天雄半两，木香半两，舶上茴香半两，蛤蚧1对，桑螵蛸半两，麝香1分。
出处：《鸡峰》卷二十九。
主治：虚冷。
加减：None
功效：补益。
用法用量：每服1丸，空心温酒送下，盐汤亦得；虚惫者每服3丸，空心温酒或盐汤送下；气壮人用冷水送下。后以软羊肉压之，又吃酒亦佳。
制备方法：上将禹余粮末一半铺盒子底，次用辰砂，次用雄黄，次用硫黄，次用雌黄，又将禹余粮末一半在上；次用阳起石，上铺纸1张，用櫃头药：槐花、黄芩、马鞭草、草决明为末；次用赤石脂末固缝；又用纸、盐和泥固济，窨干，醋拌灰塚盒子，用纸火1斤烧，欲耗，再添熟炭火1斤，欲耗，再添炭火3斤，烧煅候冷，闻鸡子香取出，去火毒1宿，开，收药研1日、以细为妙；除前6味烧煅外，石燕子、紫石英、牡蛎研为细末，并钟乳粉共四味，与鹿茸等药不入火，后与烧者药同研匀，用醋煮半夏曲糊为丸，如皂角子大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
