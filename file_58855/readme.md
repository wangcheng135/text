序号：58870
名称：秘效丸
组成：橘红1两，青皮（醋炒）1两，砂仁（炒）1两，枳壳（麸炒）1两，桑白皮1两，草果仁（炒）1两，槟榔1两，芫花8钱，大腹皮8钱，茯苓皮8钱，厚朴8钱，丁皮（炒）8钱，肉桂（炒）8钱，南木香8钱，苍术（米泔浸，炒）8钱，益智仁8钱。
出处：《玉案》卷五。
主治：肚腹胀大，小便短涩，脐凸气喘，夜不得卧。
加减：None
功效：None
用法用量：体厚者5钱，体弱者3钱，第1服消上部，以葱汤送下；第2服消中部，以陈皮汤送下；第3服消下部，以桑皮汤送下，俱要五更时服，服后如未消，仍前汤引服之。重者6服收功，轻者不3服而愈。
制备方法：上为末，醋打面糊为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
