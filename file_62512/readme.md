序号：62528
名称：梦遗神应膏
组成：荔枝草（醋煅，末）1两，三角尖（醋煅，末）1两，益母草（醋煅，末）1两，清风藤（醋煅，末）1两，五味子（醋煅，末）1两，玄精石（醋煅，末）1两，粟壳1两，诃子肉1两，龙骨1两，牡蛎1两。
出处：《活人方》卷七。
主治：因劳烦过度，思虑无穷，谋为不遂，淫欲任意，致伤神动气，神气不守，则精无统摄，遂有淫梦自遗，白淫白浊，五淋滑脱；及妇人带下。
加减：None
功效：收敛固摄。
用法用量：外贴。内服益志固精丸。
制备方法：除玄精、龙骨、牡蛎外，先将7味用麻油2斤熬枯，漉去滓，再熬至滴水不散，方搅入炒黑铅粉12两，停火候冷，徐徐调入前3种末，摊粗皮上，用狗皮亦可。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
