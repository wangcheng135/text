序号：78455
名称：撞气阿魏丸
组成：茴香（炒）1两，青皮（去白）1两，甘草（炒）1两，蓬莪荗（炮）1两，川芎1两，陈皮（去白）1两，白芷半两，丁香皮（炮）1两，缩砂仁半两，肉桂（去皮）半两，生姜4两（切作片子，用盐半两淹1宿，炒黑色），胡椒2钱半，阿魏（醋浸1宿，以面同为糊）2钱半。
出处：《局方》卷三（绍兴续添方）。
主治：五种噎疾，九般心痛，痃癖气块，冷气攻刺，及脾胃停寒，胸满膨胀，腹痛肠鸣，呕吐酸水，丈夫小肠气，妇人血气，血刺等疾。
加减：None
功效：None
用法用量：丈夫气痛，炒姜盐汤送下1-2粒；妇人血气，醋汤送下；常服1粒，嚼烂，茶、酒任下。
制备方法：上为末，用阿魏糊和丸，如芡实大，每药丸1斤，用朱砂7钱为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
