序号：3372
名称：太一赤丸
组成：朱砂（上好者）半两，硼砂半两，硇砂半两，铅白霜半两，粉霜半两，舶上硫黄半两，干漆（细锉）半两，金箔10片，银箔10片，金牙石半两（别研），紫石英半两（别研），天麻半两，川羌活半两，独活半两，巴豆半两（去心，去油尽）。
出处：《传家秘宝》卷中。
主治：中风，积涎在膈下，四肢瘫痪，或不知人事。
加减：None
功效：None
用法用量：临卧时用糯米饮半盏，龙脑、腻粉、薄荷、自然汁同调下1丸。3日内取下风涎。
制备方法：上为细末，用黄蜡3两熔作汁，拌诸药匀，乘热为丸，如鸡头子大，用朱砂为衣，入瓷盆子内。
临床应用：None
各家论述：None
用药禁忌：None
附注：太乙赤丸（《普济方》卷八十七）。
