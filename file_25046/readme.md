序号：25054
名称：朱砂丸
组成：朱砂（细研，急水飞过，熟灰池渗干尤佳），白僵蚕（择去丝，取直者，洗过，焙干）半两，新罗白附子半两（以湿纸煨裹，候令纸干，取出油，切成片子，焙干），天南星（炮裂，去皮脐，切成片子）半两，麝香半钱（研入，和匀），干蝎1两（铫子内慢火炒，令极热，不可太过）。
出处：《幼幼新书》卷十引《张氏家传》。
主治：None
加减：None
功效：镇心，压惊，坠涎。
用法用量：每服10丸，用煎金银薄荷汤吞下，如遇惊取下后，且以此药服1-2服，无不效；或有虚汗，用麻黄根煎下。
制备方法：上药各为末，面糊为丸，如粟米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
