序号：26755
名称：延龄益寿丹
组成：何首乌4两（竹刀去皮，切片，用黑豆9蒸9晒，后用人乳拌1次），当归1两（酒洗净），知母2两（酒炒去毛），川芎1两，杜仲（去粗皮，姜汁炒去丝）2两，白茯苓1两（去皮），青盐1两，茯神1两（去皮心），远志1两（去芦心，甘草煎水浸半日），川椒1两（去目，出汗，留红皮，去白肉），牛膝（酒洗），朱砂1两（研碎，打零炒），蜜1斤（炼过镜光止），姜汁2两，黄柏2两（去皮，酒浸，日晒夜浸），破故纸（酒洗）、小茴香（去土，盐水洗，炒黄），天门冬（去心）1两，麦门冬（去心）1两，核桃肉4两（去油，炒黄），旱莲4两（水煎5滚），石菖蒲（盐洗将炒）1两，生地黄（酒洗）1两，熟地黄（酒洗）1两，石乳（去油）1两，川巴戟（酒洗净）1两，山精（用米泔水2-3碗，浸半日，竹刀刮去粗皮4两，烂者不要，要选好的切碎，捣烂放砂锅内，水3碗，煎至锅内汁干，取出，砂锅内浸要换米温水1-2次，然后酒煎成膏，同旱莲汁，姜汁拌诸药末）。
出处：《摄生众妙方》卷二引季全真方。
主治：None
加减：None
功效：存精固气，养神补身，阳消阴长，滋益肾水，补丹田，乌发润泽皮肤，助阳补孕。
用法用量：每服70丸，早晚盐汤任下；不饮酒，白汤送下。1月见效。
制备方法：炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
