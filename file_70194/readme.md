序号：70210
名称：犀角竹沥膏
组成：犀角12分（屑），升麻8分，蒴藋根4分，秦艽4分，独活4分，白及4分，菊花4分，白术4分，防己4分，白芷4分，当归4分，防风4分，芎4分，青木香4分，寒水石（碎）4分，苦参4分，漏芦根4分，蒺藜子2合，莽草2分，枳实2枚（四破），栀子仁7枚，竹沥3升，吴蓝1两。
出处：《外台》卷十五引《延年秘录》。
主治：风热发，即头顶脉掣动急强，及热毒疹痒。
加减：None
功效：None
用法用量：用摩风处，1日3次。
制备方法：上切，以竹沥渍1宿，明旦于炭火上和猪脂5升煎令九上九下，以候白芷色黄膏成，绞去滓，纳于不津器中。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
