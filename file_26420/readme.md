序号：26428
名称：当归丸
组成：当归9铢，吴茱萸（1作杏仁）半两，蜀椒半两，细辛18铢，干姜18铢，附子18铢，狼毒9铢，豉7合，巴豆10个。
出处：《千金》卷五。
主治：小儿胎寒（？？）啼，腹中痛，舌上黑，青涎下；亦治水癖。
加减：None
功效：None
用法用量：1月儿服如黍米1丸，日1夜2。不知稍加，以知为度。
制备方法：上九味，捣八种下筛，称药末令足，研巴豆如膏，稍稍纳末，捣令相得，蜜和，桑杯盛，蒸5升米饭下，出，捣一千杵。
临床应用：None
各家论述：None
用药禁忌：None
附注：黑丸（原书同卷）、蜀椒丸（《圣济总录》卷一七七）。《圣济总录》本方用法：一月儿服如黍米大一丸，早晨、晚后各一服；一岁儿两丸；三岁至五岁儿服五丸，并用米饮送下。
