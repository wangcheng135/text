序号：12015
名称：仙灵脾散
组成：仙灵脾1两，天雄1两（炮裂，去皮脐），天麻1两，独活3分，牛膝1两（去苗），芎藭3分，石斛1两（去根），肉桂1两半（去粗皮），茵芋3分，麻黄1两半（去根节），当归3分，侧子3分（炮裂，去皮脐），乌蛇肉1两（酥拌，炒令黄），虎胫骨1两（涂酥，炙令黄），桑螵蛸3分（微炒），丹参3分，五加皮3分，海桐皮3分，防风3分（去芦头），薏苡仁3分，干蝎3分（生用），牛黄1分（细研），麝香1分（细研）。
出处：《圣惠》卷二十三。
主治：中风，手足不遂，肌肉冷痹，骨节疼痛，缓弱不随。
加减：None
功效：None
用法用量：每服2钱，食前以温酒调下。
制备方法：上为细散，入研了药令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
