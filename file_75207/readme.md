序号：75223
名称：腽肭脐膏
组成：腽肭脐2两，当归2两，附子2两（去皮脐，生用），桂心3两，羌活1两，芎1两，麒麟血1两，乌蛇1两，乳香1两，木香1两，续断1两，生干地黄2两，白芷1两，穿山甲1两，猬皮1两，桃仁1两（汤浸，去皮尖），莨菪子2两，杏仁1两（汤浸，去皮），紫草1两，棘针1两，柳枝1两，槐枝1两，赤芍药1两，白蔹1两，防风1两，细辛1两，葱白14茎（连须），黄蜡1两，蜜陀僧1两，沥清香1两，驼脂2两，羊脂3两，猪脂20两，清麻油5两，黄丹3升（炒令紫色）。
出处：《圣惠》卷六十七。
主治：伤折。
加减：None
功效：接骨止痛。
用法用量：先似猪、羊、驼脂等于大锅内文火煎取清汁，去滓，后入麻油，煎令如鱼眼沸，次下棘针、柳枝、槐枝、葱白等4味。煎令黄焦，滤去滓，即下腽肭脐等药，以炭火养7日后，绵滤去滓，却入锅内，旋下黄丹，用柳枝子不住手搅，候转紫色，稀稠得所，即成膏，于瓷盒中盛。每用于纸上摊贴伤损处。
制备方法：上细锉。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
