序号：58523
名称：秘方换腿丸
组成：薏苡仁1两，石南叶1两，天南星（洗，姜制，炒）1两，川中膝《酒浸，焙》1两，肉桂（去粗皮）1两，当归（去芦）1两，天麻（去苗）1两，附子（炮，去皮脐）1两，羌活1两，防风（去叉）1两，石斛（去根）1两，萆薢（微炙）1两，黄耆（蜜炙）1两，续断1两，苍术（米泔浸）1两半，槟榔半两，干木瓜4两。
出处：《局方》卷一（续添诸局经验秘方）。
主治：肾经虚弱，下注腰膝，或当风取凉，冷气所乘，沉重少力，移步迟缓，筋脉挛痛，不能屈伸，脚心隐痛，有妨履地；干湿脚气，赤肿痛楚，发作无时，呻吟难忍，气满喘促，举动艰难，面色黧黑，传送秘涩。
加减：None
功效：舒筋轻足。
用法用量：每服30-50丸，空心以温酒或木瓜汤吞下，日2-3次。
制备方法：上为细末，面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：换腿丸（《普济方》卷二十九）、换脚丸（《古今医鉴》卷十）。
