序号：83254
名称：乌鸡丸
组成：乌鸡1只（不刀血，去毛，用醋5大碗煮熟，火煅存性，成灰为末），香附米10两（酒浸旬日，用醋煮，焙干），乌药2两，净艾2两（醋浸，炒白米饭少许，入杵臼内捣成饼，火上炙令干），当归3两（醋洗），川芎1两，白芍1两，熟地1两，小茴3两（醋炒），山药2两，牡蛎2两，破故纸（醋炒）5钱，良姜5钱，白姜1两半，丁香1两（不见火）。
出处：《古今医鉴》卷十一。
主治：下焦虚寒，赤白带下，脐腹冷痛。
加减：赤白带下不止，加龙骨1两、五倍子1两半。
功效：None
用法用量：每服50丸，空心醋汤送下。
制备方法：上为末，饭为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
