序号：74310
名称：催生神妙乳珠丹
组成：乳香（细研）。
出处：《妇人良方》卷十七。
主治：难产。胞浆先破，恶水来多，胎干不得卧，或胎横逆不顺，及胎死不下，或胎下胞衣未下。
加减：None
功效：催生。
用法用量：每服1粒，如催生，冷酒化下。良久未下，再服1粒；如大段难产时，以莲叶心蒂7个，水2盏，煎至1盏，放温化下1粒，良久未下，亦可再服；如胞浆先破，恶水来多，胎干不得卧时，须先与四物汤及通真丸补养其血气，次更浓煎葱汤，放冷如体，令坐婆洗产户，须是款曲洗，令气上下通畅，仍更用酥调滑石末涂产户里，次服此药；如胎死不下者，用黑豆3合，好醋半升，煮令豆烂，取汁1盏放温，化下药1粒，须臾便下矣，万一未下，亦可再服；若胎横逆不顺，即先服如神散，再服此药，复以此药催之。
制备方法：上以猪心血为丸，如梧桐子大，朱砂为衣，晒干。
临床应用：None
各家论述：None
用药禁忌：None
附注：神妙乳砂丹（《校注妇人良方》卷十七）。本方方名，《玉机微义》引作“催生乳香膏”、“乳朱丹”，《本草纲目》引作“开骨膏”。
