序号：83615
名称：乌骨鸡丸
组成：人参3两，生地5两，熟地5两，当归6两（酒洗），官桂3两，茯苓3两，黄耆6两，川芎3两，白术1两（麸炒），续断2两（酒洗），香附12两，芍药2两，石斛3两（酒浸），乌药2两（炒），杜仲2两（姜汁炒），地骨皮3两。
出处：《宋氏女科》。
主治：血海虚寒，乃无子嗣，数经堕胎，经水不时，暴下不止，月内丹行，或前或后，或崩中漏下，小便白浊并带，及腰胯疼痛。
加减：None
功效：常服除宿血，生新血，令人有孕，生子充实。
用法用量：每服80丸，用温酒送下；或米汤、或艾汤亦可。每药末1斤，用白面4两，打糊。
制备方法：上为末，用乌骨白鸡或黄鸡1只，男用雌，女用雄，将鸡笼住，用黄耆2两为末，加炒面1两和匀，水为丸，如豆大，喂鸡服尽，将鸡吊死，肚肠洗净，挦毛(扌遀)骨碎，入前药于鸡腹内，用酒酣5斤浸，火煮烂，取骨捣烂，为细末，将煮鸡、药汁和面糊，加酒酣打匀，同药末为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
