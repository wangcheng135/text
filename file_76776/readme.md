名称：黑龙丸
组成：羌活（去芦头）1两，独活（去芦头）1两，蔓荆子1两，薄荷叶（去土）1两，细松烟墨（烧红，醋淬）1两，川芎半两，白附子（炮）半两，甘草（炙）半两，山栀子仁半两，白僵蚕（炒去丝嘴）半两，香白芷半两，防风（去芦头）半两，荆芥穗半两，天南星（汤浸1宿，炮）半两，草乌头（炮，去皮尖）半两，川乌头（炮，去皮脐）半两。
出处：《杨氏家藏方》卷二。
主治：风毒上攻，头面多生赤(疒畾)。
加减：None
功效：None
用法用量：每服1丸，细嚼，食后茶、酒任下。
制备方法：上为细末，炼蜜为丸，每1两作10丸。
临床应用：None
各家论述：None
