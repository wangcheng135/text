序号：57800
名称：狸骨知母散
组成：狸骨、鲮鲤甲、知母、山龟壳、甘草、桂心、雄黄、干姜各等分。
出处：方出《千金》卷二十三，名见《普济方》卷二九三。
主治：鼠漏，始发于颈，无头尾，如鼷鼠，寒热脱肉。此得于鼠毒，其根在胃。
加减：None
功效：None
用法用量：每服方寸匕，以饮送下。日3次；仍以蜜和纳疮中；先灸作疮，后以药敷之，已作疮，不用灸。
制备方法：上药治下筛。
临床应用：None
各家论述：《千金方衍义》：狸专捕鼠，鼠漏用之每效，故用狸骨，佐以知母、龟壳、鲤甲、姜、桂、雄黄，都是肾家辟除邪毒之药，方下虽言其根在胃，恐是肾字之误。
用药禁忌：None
附注：狸骨散（《不居集》上集卷十八）。
