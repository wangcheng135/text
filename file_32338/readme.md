序号：32347
名称：妙应丸
组成：槟榔12两，黑牵牛3两，大黄1两，雷丸1两，锡灰1两，芜荑1两，木香1两，使君子1两。
出处：《医学入门》卷七。
主治：凡人面上白斑与唇红，能食心嘈，颜色不常，脸上生有蟹爪露者，便有虫也；山岚瘴气，传尸痨瘵，水肿疟痢，咳嗽黄疸，噎膈，肠风痔漏，一切风气食积疼痛，疮癞热痰痞块，赤眼口疮，女人经脉不调，血瘕血闭，赤白带下，小儿癫痫，一切疳积，虫积。
加减：如失声，加沉香、琥珀。
功效：追虫。
用法用量：每服4钱，五更葱汤或木香煎汤送下；取寸白虫，用东方上石榴根煎汤，面东服之；小儿服1钱或5分。天明取下病根，或虫，或如烂鱼肠，或如马尾、蛤蟆、小蛇，诸般怪物，或小便取下青、黄、红、白，或米泔等色。此丸4时可服。
制备方法：上为末，用葱白煎汤，露一宿，为丸如粟米大。
临床应用：None
各家论述：None
用药禁忌：孕妇禁用。忌生冷荤腥等物。
附注：其虫皆因饮食中所感而成。此药不比巴霜、甘遂、硇砂等剂，不动真气，有虫取虫，有积取积，有气取气，有块取块，一服见效。
