序号：67729
名称：黄芩牡丹汤
组成：黄芩2两，牡丹2两，桃仁2两，瞿麦2两，芎2两，芍药3两，枳实3两，射干3两，海藻3两，大黄3两，虻虫70枚，水蛭50枚，蛴螬10枚。
出处：《千金》卷四。
主治：妇人从小至大，月经未尝来，颜色萎黄，气力衰少，饮食无味。
加减：None
功效：None
用法用量：上（口父）咀。以水1斗，煮取3升，分3服。服2剂后，灸乳下1寸黑圆际各50壮。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：女子月经素未通，或郁热内戕，而至血结不行。故用黄芩、丹皮以化热，枳实、大黄以导滞，芎、芍、桃仁以和营，射干、瞿麦、海藻以降逆，虻、蛭，蛴螬以破血也。
用药禁忌：None
附注：黄芩散（《圣惠》卷七十二）。
