序号：71652
名称：寒通汤
组成：滑石1两，生杭芍1两，知母8钱，黄柏8钱。
出处：《衷中参西》上册。
主治：下焦蕴蓄实热，膀胱肿胀，溺管闭塞，小便滴沥不通。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：癃闭：一人，年六十余，溺血数日，小便忽然不通，两日之间滴沥全无。病人不能支持，自以手揉挤，流出血水少许，稍较轻松。揉挤数次，疼痛不堪揉挤。彷徨无措，求为诊治。其脉沉而有力，时当仲夏，身覆厚被，犹觉寒凉，知其实热郁于下焦，溺管因热而肿胀不通也。为拟此汤，一剂稍通，又加木通、海金沙各二钱，服两剂全愈。
