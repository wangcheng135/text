序号：39183
名称：炙肝散
组成：山芋1两，柴胡（去苗）1两，缩砂（去皮）1两，高良姜（炮）1两，陈橘皮（去白，焙）1两，桂（去粗皮）1两，白芷1两，木香1分，吴茱萸（汤洗，焙）半两，赤芍药（洗，焙）半两，厚朴（去粗皮，用生姜汁炙）半两，桔梗（锉，炒）半两，干姜（炮裂）半两，朴骨脂（炒）半两，青橘皮（去白，焙）半两，草豆蔻（去皮）半两。
出处：《圣济总录》卷八十七。
主治：冷劳，大便不禁，羸瘦困乏。
加减：None
功效：None
用法用量：每服用猪肝4两，薄批片子，掺药5钱匕，入葱、盐各少许，湿纸裹，慢火内煨令香熟，去纸，细嚼，米饮送下。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
