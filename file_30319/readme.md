序号：30327
名称：陀僧膏
组成：南陀僧（研末）20两，赤芍2两，全当归2两，乳香（去油，研）5钱，没药（去油，研）5钱，赤石脂（研）2两，苦参4两，百草霜（筛，研）2两，银黝1两，桐油2斤，香油1斤，血竭（研）5钱，孩儿茶（研）5钱，川大黄半斤。
出处：《医宗金鉴》卷六十二。
主治：鼠疮，溃破流脓。一切外科肿疡，已溃未溃，创破流血，疼痛异常。
加减：None
功效：拔脓生肌长肉，止痛散血消肿。
用法用量：贴患处。
制备方法：先将赤芍、当归、苦参、大黄入油内枯，熬至滴水不散，再下陀僧末，用槐、柳枝搅至滴水将欲成珠，将百草霜细细筛入搅匀，再将群药及银黝筛入，搅极匀，倾入水盆内，再收入瓷盆内，常以水渍之。
临床应用：None
各家论述：None
用药禁忌：不可入口。
附注：None
