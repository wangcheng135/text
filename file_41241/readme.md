序号：41256
名称：金沉膏
组成：云里砂、斑蝥20个，红娘子30个，川山甲5钱（锉），海金沙5钱，乳香3钱（研），没药5钱（研），血竭3钱，桑柴灰5斤，枣柴灰5斤（上二灰一处和）。
出处：《医方类聚》卷一九四引《经验秘方》。
主治：一切瘤子，瘰疬。
加减：None
功效：None
用法用量：若用时，将云里砂调成膏子，上药时离好皮子1韭叶，上药1昼夜，或2遍、3遍，直瘤子平了方住药。若瘰疬破不破，与瘤子一般上药，瘤子等或1月、20日、半月落下，即用青金散涂之。
制备方法：用滚水数碗，淋灰水12碗，入砂锅内熬百十余沸，先下斑蝥、红娘子，再滚百十余沸，下川山甲、海金沙，再滚百十余沸，熬至3碗，滤去滓，再入锅内熬，却下乳香、没药、血竭，再滚百十余沸，熬成膏子，用瓷器盛顿，牢封口。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中云里砂剂量原缺。
