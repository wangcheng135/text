序号：64521
名称：理中定风汤
组成：大熟地5钱，全当归2钱，山萸肉1钱，枸杞子2钱，炒白术3钱，炮姜炭1钱，潞党参2钱，炙甘草1钱，熟枣仁2钱（炒，研），上肉桂1钱，破故纸2钱，炙绵耆2钱。
出处：《喉科心法》卷下。
主治：小儿精神已亏，气血大坏，形状狼狈，瘦弱至极。
加减：如咳嗽不止，加缩壳1钱，金樱子1钱；大热不退，加白芍1钱；泄泻不止，加丁香6分，只服1剂，即去附子，只用丁香7粒，隔2-3日，只用附子2-3分。
功效：助气补血，却病回阳。
用法用量：加生姜3片，红枣3枚，胡桃2个（打碎）为引，仍用灶心上2两煮水煎药，取浓汁1杯，加附子5分煎水掺入，量儿大小分数次灌之。如法浓煎频频与服。
制备方法：None
临床应用：None

用药禁忌：None
