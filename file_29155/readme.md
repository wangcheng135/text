序号：29163
名称：阿魏丸
组成：阿魏1两半，当归1两半（切，醋炒），官桂半两，陈皮半两（去白，细切，醋炒），白及3分，吴白芷半两，蓬术1两，延胡索半两（锉碎，醋炒），木香3分，吴茱萸半两（醋炒），川芎半两（醋炒），附子半两（炮，去皮脐），干姜1两（炮），肉豆蔻3分（研细末），朱砂3分（研细末）。
出处：《博济》卷二。
主治：男妇一切气攻刺疼痛，呼吸不得，大肠滑泄。
加减：None
功效：None
用法用量：每服5丸，温酒送下；橘皮汤亦可；妇人，醋汤送下。
制备方法：上除阿魏、朱砂外，同为细末，以头醋半升，浸阿魏经宿，同生绢袋取汁，煮面糊为丸，如梧桐子大，以朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
