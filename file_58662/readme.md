序号：58677
名称：秘传神仙消痞丸
组成：斑蝥20个（去头足翼，用糯米半升同炒，候米焦黄色为度，去米不用），巴豆（去皮取霜）20粒。
出处：《局方》卷十（续添诸局经验秘方）。
主治：小儿寒温不调，乳哺失节，或啖生冷、果子、粘食等物，脾胃微弱，不能消化，五脏不利，三焦壅滞而致痞疾，结块腹内，坚硬如石，或发作寒热，有如疟证，不能饮食，渐致羸瘦。
加减：None
功效：None
用法用量：小儿3岁以前，每服3丸，五更初茶清下。
制备方法：先将斑蝥碾为细末，却入巴豆霜同研令匀，米糊为丸，如小绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“神仙消痞丸”。
