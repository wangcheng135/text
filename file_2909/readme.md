序号：2909
名称：天南星丸
组成：天南星（大者，逐日换水浸5日，慢火煮5-7沸，切作片子，曝干，麸炒令黄香）1两，乌蛇（酒浸，取肉炙干）1两，白僵蚕（直者，麸炒令黄）1两，天麻1两，干蝎（全者，擘破，炒黄色）3两，白附子（炮）3两，雄黄（研）3两，琥珀（杵，研）3两，麝香1分（研），牛黄（研）半两，龙脑（研）半两，丹砂1两半（研）。
出处：《圣济总录》卷十四。
主治：风惊。
加减：None
功效：镇心化涎。
用法用量：每服1丸，荆芥、人参汤化下。
制备方法：上药除研者外，捣罗为末，合研匀，干瓷器收，密封勿令透气，旋炼蜜为丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
