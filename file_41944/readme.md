序号：41959
名称：青龙丹
组成：草乌头4两（生，去皮尖），香附子（炒）2两，半夏（汤洗7次）1两，麻黄（去根节）1两，香白芷1两，白附子（生用）1两，血竭（别研）1两，天南星（生用）1两，川乌头（生，去皮脐尖）1两，赤芍药1两，薄荷叶（去土）1两，山药1两，螺青1两，白僵蚕（炒去丝嘴）半两，藿香叶（去土）半两，甘松（去土，焙）半两，零陵香半两，细松半两，烟墨（烧，醋淬）半两。
出处：《杨氏家藏方》卷一。
主治：男子、妇人左瘫右痪，手足拳缩，口面眼斜，遍身瘾疹，及伤风鼻塞脑痛，四肢顽麻，牙关紧急。
加减：None
功效：None
用法用量：每服1丸，生姜、薄荷同嚼，热酒送下，不拘时候。
制备方法：上为细末，醋煮面糊为丸，每1两作20丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
