序号：48658
名称：神授至圣保命丹
组成：全蝎14个（青色者），朱砂（水飞）2钱（好者），麝香半钱，防风（去芦并叉枝）1钱，金箔10片（研），天麻2钱，白僵蚕（去丝嘴，直者）1钱，白附子2钱（好者），天南星1钱，蝉壳（去土泥）2钱。
出处：《卫生总微》卷六。
主治：一切惊痫、风痓、中风，并胎惊内吊，腹肚坚硬，夜啼发热，急慢惊风，恶候困重，上视搐掣，角弓反张，倒仆不省，昏愦闷乱。
加减：None
功效：镇心神，退惊痫，安魂定魄，祛风逐邪，化涎消痰。
用法用量：每服初生儿半丸，周晬儿1丸，3-5岁有急候者2丸，5-7岁至10岁常服只1丸，乳汁或薄荷水化下。
制备方法：上为细末，粳米饭为丸，如樱桃大，以朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：至圣保命丹（《直指小儿》卷二）、保命丸（《婴童百问》卷三）、神效保命丸（《中国医学大辞典》）。
