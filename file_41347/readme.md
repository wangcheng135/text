序号：41362
名称：金沸草散
组成：前胡3两，荆芥4两，半夏1两（洗净，姜汁浸），赤芍药2两，细辛1两，甘草（炙）1两，旋覆花3两。
出处：《活人书》卷十七。
主治：伤寒中脘有痰，令人壮热，头痛，项筋紧急。
加减：None
功效：None
用法用量：每服2钱，水1盏，加生姜5片，大枣1枚，同煎至6分，去滓热服，未知再服。
制备方法：上为末。
临床应用：None
各家论述：《医方集解》：此手太阴药也。风热上壅，荆芥辛轻发汗而散风；痰涎内结，前胡，旋覆消痰而降气，半夏燥痰而散逆；甘草发散而和中；茯苓行水；细辛温经；盖痰必挟火而兼湿，故下气利湿而证自平。茯苓用赤者，入血分而泻丙丁也。
用药禁忌：None
附注：方中赤芍药，《医方集解》引作“赤茯苓”。
