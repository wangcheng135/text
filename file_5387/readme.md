序号：5387
名称：加减千金固胎饮
组成：人参1钱，生于术3钱，云苓2钱，炙甘草5分，酒炒黄芩1钱，枳壳1钱5分（炒），青皮1钱5分，柴胡5分。
出处：《慈航集》卷下。
主治：孕妇人虚，疟疾寒热不止。
加减：如恶心，加灶心土5钱、乌梅2个；如腹胀，加砂仁3钱，如热多，加青蒿3钱；如腰痛，加续断3钱；如痰多，加川贝母1钱5分、橘红8分；如大便结，加鲜首乌5钱。当归5钱；如作泻，加鹿角霜3钱；如腹痛，加煨广木香1钱5分；如泻热，加酒炒川连3分；如胎气下坠，加黄耆5钱（炙）、菟丝子2钱；如子气上逆，加砂仁3钱、葡萄干3钱。
功效：None
用法用量：用煨姜3钱，大枣3个，荷叶蒂3个为引，河井水煎。头煎在疟前服，二煎接服之。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：愈后饮食宜清淡，调理4-5日，庶疟不复。
附注：None
