序号：58049
名称：益母草煎丸
组成：益母草汁1升，青蒿汁1升，无灰酒1升，生姜汁3合，童便1升，蜜5合（以上同于银器中，慢火熬成膏），柴胡1两（去苗），人参3分（去芦头），麦门冬1两半（去心，焙），琥珀3分（细研），桃仁1两（汤浸，去皮尖双仁，麸炒微黄），地骨皮3分，白术3分，枳壳3分（麸炒微黄，去瓤），鳖甲1两（涂醋炙令黄，去裙襕），桔梗半分（去芦头），当归半分，赤芍药1两，生干地黄1两，鬼箭羽1两，麝香1分（细研）。
出处：《圣惠》卷七十。
主治：妇人热劳烦闷，四肢疼痛；经脉滞涩，腹胁妨闷，不欲饮食。
加减：None
功效：None
用法用量：每服30丸，食前以温水送下。
制备方法：上药前6味熬成膏；余药为末，用熬成膏和捣为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
