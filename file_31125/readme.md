序号：31134
名称：杏仁煎
组成：杏仁（汤浸，去皮尖双仁，炒黄）1两半，桑根白皮（锉，炒）1两半，贝母（去心）1两半，生姜汁1合半，地黄汁2合半，酥半两，大枣60个（去核），紫菀（去苗）3分，甘草（炙）1两，桔梗（炒）1两，五味子（炒）1两，赤茯苓（去黑皮）1两，地骨皮1两，人参3分。
出处：《圣济总录》卷一二四。
主治：肺胃壅滞，咽喉中如有物妨闷。
加减：None
功效：None
用法用量：食后含1匙头，细细咽津。
制备方法：上药先研杏仁，以水5升滤取汁，将草药细锉，同煎至2升，以绵滤去滓，续下酥及地黄汁，慢火煎成膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
