序号：44681
名称：宣胃散
组成：白芍药（炒）2钱，黄芩（炒）1钱半，黄连（炒）1钱半，木香1钱半，枳壳（炒）1钱半，槟榔1钱，甘草（炙）3分。
出处：《济阳纲目》卷二十二。
主治：痢疾。
加减：若腹痛，加当归1钱半，砂仁1钱，再加木香、芍药各5分；若后重，加滑石（炒）1钱半，再加枳壳、槟榔各5分，生芍药、条黄芩各5分；若痢已久而后重不去，此大肠坠下，去槟榔、枳壳，用条黄芩、升麻各1钱以升提之；若白痢，加白术、白茯苓、炒滑石、陈皮各1钱；稍久胃虚，减芩、连、芍药各7分，亦加上数味，惟去滑石、槟榔、枳壳，再加缩砂，炙干姜各5分；初欲下之，再加大黄5钱；兼食积，加山楂、栀子、枳实各1钱；若红痢，加当归、川芎、桃仁各1钱；稍久胃虚，减芩、连各5分，加当归、川芎、熟地、阿胶、朱砂、陈皮各1钱，白术1钱半；初欲下之，再加大黄5钱；若红白相杂，加芎、归、桃仁、陈皮、苍术各1钱半；若色赤黑相杂，此湿胜也，及小便赤涩，少加木通、泽泻、茯苓各1钱，山栀5分，以分利之；若呕吐食不下，加软石膏1钱半，陈皮1钱，山栀5分，入生姜汁缓呷之，以泻胃口之热；有气血虚而痢者，用四物汤加参、术、陈皮、黄芩、阿胶之类以补之；有寒痢者，加黄连、木香、酒炒芍药、当归、炒干姜、砂仁、厚朴、肉桂之类。
功效：None
用法用量：上锉。水煎服。
制备方法：None
临床应用：None

用药禁忌：None
