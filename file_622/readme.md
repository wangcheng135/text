序号：622
名称：五精煎丸
组成：白茯苓（去黑皮，别取末）4两，甘菊花（炊1复时，不住洒酒，晒干，别取末）4两，菖蒲（石上生者，酒浸3日，炊1日焙干，别取末）4两，桂（去皮，取心中好者，别取末）4两，天门冬（去心，焙）1斤，白术（切作片子，白者可用）1斤，人参1斤，牛膝1斤（捣碎，各以水并酒共1斗，浸药3日，绞取浓汁，滤去滓，于银器内慢火各熬成膏），生黄精5斤，生地黄5斤（2味各捣取汁，于银器内慢火熬成膏）。
出处：《圣济总录》卷一八七。
主治：上膈多热，下脏虚冷，皮肤不泽，气力乏少，大便秘涩，或时泄利，头旋痰滞，口干舌强。
加减：None
功效：益寿延年；
用法用量：每服30-40丸，食前后清酒或米饮送下。久服自觉神效。
制备方法：上10味，先将下6味逐味取汁，熬至半斤可住火，然后将膏6件共合成3斤，以前4味散药同和匀，晒干，再入膏和匀，再入臼中杵，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
