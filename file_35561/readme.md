序号：35571
名称：侧子酒
组成：侧子2两（炮），干姜2两，石斛8两，丹参3两，牛膝2两，甘草2两（炙），防风3两，干地黄4两，芎2两，当归3两，桂心3两，五味子3两，白术2两，秦艽3两，防己2两，椒2两（汗），独活3两，山茱萸4两，细辛2两，黄芩2两，茯苓4两，附子1两（炮）。
出处：《外台》卷十八引《崔氏方》。
主治：脚气虽愈，至冬季间常须服。
加减：None
功效：None
用法用量：以酒3斗5升浸，秋冬7日，春夏5日。每服4合，细细加之，以知为度，每日2次。
制备方法：上切，绢袋贮。
临床应用：None
各家论述：None
用药禁忌：得食羊、鹿、獐肉，鸡亦得食。忌海藻、菘菜、猪肉、冷水、桃、李、雀肉、生葱、生菜、芜荑、酢物。
附注：None
