序号：70212
名称：犀角汤
组成：犀角（镑）1两半，防风（去叉）1两半，生干地黄（焙）1两半，羌活（去芦头）1两半，菊花1两半，半夏（汤浸，去滑，姜汁制作饼，晒干）1两半，玄参1两半，黄芩（去黑心）1两半，白术1两半，甘草（炙，锉）1两半，旋覆花1两半，麦门冬（去心，焙）1两半，前胡（去芦头）1两半。
出处：《圣济总录》卷四十三。
主治：心脏实热，胸中满闷，嗔怒不常，或头旋运，或痛如破。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，加生姜1枣大（切），煎至8分，去滓，食后温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
