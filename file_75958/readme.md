序号：75974
名称：辟风锭子
组成：全蝎20个（生用），牛胆南星（腊月用肥泽无病犍牛胆1个，将南星入内，悬高处49日后取出，各晒干，收用）7钱，防风5钱，白附子5钱，干生姜3钱，川乌5钱，天麻5钱，川芎5钱，白芷5钱，人参5钱，牛黄3钱，辰砂1两，麝香2钱，片脑3钱，薄荷5钱，木香5钱，白术5钱，白僵蚕20个（生用），一方加天竺黄5钱。
出处：《摄生众妙方》卷十。
主治：小儿急慢惊风；大人诸风，破伤风。
加减：None
功效：None
用法用量：急惊风，手足搐搦，用金银磨汤化下；慢惊风，四肢不收，昏昏如眠，不省人事，淡姜汤化下，各量儿大小虚实，或半分，1分、2分、3分，斟酌与服；大人破伤诸风，温酒下。
制备方法：上为极细末，用麻黄1斤，甘草半斤，蜂蜜2两，煎作膏，令稀稠得宜，将前药末和匀为锭，金箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
