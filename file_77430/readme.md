序号：77446
名称：雄黄解毒丸
组成：雄黄2钱5分，玉金2钱5分，巴豆14粒（去油），枯凡2钱5分，皂角1钱，麝香8分。
出处：《治疹全书》卷下。
主治：小儿疹后服凉药太过，脾气不足，中气虚衰，变成慢惊，牙关紧急，痰涎壅盛，目直上视，手足搐溺，发无休止者；及小儿急惊风，大人中风，喉风等。
加减：None
功效：None
用法用量：每服1丸，以线穿好，续于箸上，令小儿仰卧，箸置于口外，将药入于喉中，听其自化，痰出立苏；如化3丸，痰不出者。艾炙颊车各3炷，即以好醋调胆矾末探之。
制备方法：共为细末，醋打飞罗面糊为丸，芡实大，以针穿孔，如念佛珠，阴干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
