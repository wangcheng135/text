序号：57838
名称：珠玉二宝粥
组成：生山药2两，生薏米2两，柿霜饼8钱。
出处：《衷中参西》上册。
主治：脾肺阴分亏损，饮食懒进，虚热劳嗽，并一切阴虚之证。
加减：None
功效：补肺健脾养胃。
用法用量：随意服之。
制备方法：上先将山药、薏米捣成粗渣，煮至烂熟，再将柿霜饼切碎，调入融化。
临床应用：劳嗽一少年因感冒懒于饮食，犹勤稼穑，枵腹力作，遂成劳嗽，过午发热，彻夜咳吐痰涎。医者因其年少，多用滋阴补肾之药，间有少加参、耆者，调治二月不效，饮食减少，痰涎转增，渐至不起，脉虚数兼有弦象，知其肺脾皆有伤损也，授以此方，俾一日两次服之，半月痊愈。
各家论述：山药、薏米皆清补脾肺之药，然单用山药，久则失于粘腻；单用薏米，久则失于淡渗；唯等分并用，乃可久服无弊。又用柿霜之凉可润肺，甘能归脾者，以为之佐使。病人服之不但(疒丂)病，并可充饥，不但充饥，更可适口，用之对证，病自渐愈。即不对证，亦无他患，诚为至稳善之方也。
用药禁忌：None
附注：None
