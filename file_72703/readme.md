序号：72719
名称：温中降气丸
组成：附子1两（生，去皮脐，锉如半枣大，用生姜自然汁半升，银石器内慢火煮姜汁尽为度，薄切，焙干），干生姜2两（连皮用），白术1两，人参（去芦头）1两，陈橘皮（去白）1两，神曲（炒黄）1两，半夏（汤洗7次）1两，白附子（炮）1两，当归（洗，焙）1两，天南星1两，高良姜（薄切，油炒）1两，丁香1两，木香1两，沉香1两，胡椒1两，肉桂（去粗皮）1两。
出处：《杨氏家藏方》卷六。
主治：中寒气痞，脾胃不和，饮食减退，脐腹虚疼；及中酒吐逆，胸膈不利。
加减：None
功效：None
用法用量：每服50丸，生姜汤送下，不拘时候。
制备方法：上为细末，用生姜自然汁煮曲糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
