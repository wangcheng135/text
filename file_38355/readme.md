序号：38365
名称：河车八味丸
组成：紫河车1具（头生男者，用白矾煎汤揉洗极净，用姜汁同酒煮烂），大地黄3两（姜汁、砂仁同酒煮烂），净枣皮1两（炒干），粉丹皮5钱（酒炒），宣泽泻5钱（盐水炒干），嫩鹿茸2两（切片，炒干），白云苓1两5钱（乳汁蒸，晒），怀山药5两5钱（酒炒），川熟附（切，焙干燥），青化桂7钱5分（去粗皮，研），北五味2两（去梗，炒干），大麦冬1两（去心，糯米拌炒）。
出处：《幼幼集成》卷二。
主治：小儿痫症，年深日远，肝肾已亏，脾肺不足，心血耗散，证候不时举发。
加减：None
功效：None
用法用量：每早1丸，用淡盐汤化服，以饮食压之；午及临卧，各用定痫丸一服。
制备方法：上为细末，炼蜜为丸，如龙眼核大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
