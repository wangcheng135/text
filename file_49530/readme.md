序号：49545
名称：茯苓人参散
组成：茯苓2斤（去黑皮，擘破如枣大，清水渍，经1日1夜再易水，出于日中，晒干，为末），人参7两（捣），甘草1两（炙，切），牛乳7升，白沙蜜1升5合。
出处：《外台》卷十引《救急方》。
主治：上气，胸胁满闷，霍乱，积痢。
加减：None
功效：益心力，除谬忘，能饮食，延年益寿。
用法用量：上药以水5升，纳甘草，煮取2升，除甘草，澄滤；纳茯苓，缓火煎，令汁欲尽；次纳白蜜、牛乳；次纳人参，缓火煎，令汁尽，仍搅药令调，勿许焦成，日中晒干，捣筛为散，以纸盛之。温乳及蜜汤和吃并得，亦不限多少。夏月水和当麨。
制备方法：None
临床应用：有人年40时，因患积痢，羸惫不能起止，形状如70老人，服此药2剂，平复如旧。
各家论述：None
用药禁忌：忌海藻、菘菜、大醋。
附注：None
