序号：80416
名称：藕珠丹
组成：辰砂10两（成块，有墙壁者），生藕汁3-4碗。
出处：《医方类聚》卷八十九引《吴氏集验方》。
主治：None
加减：None
功效：养心气，益神。
用法用量：辰砂1味，缝绢囊贮之，用生藕取自然汁3-4碗，入净瓶每服2丸，空心人参汤送下。
制备方法：入净瓶中，悬胎，炭火煮，候藕汁干为度（悬胎者，谓以竹棒1条，横瓶口，线系药囊，垂瓶中，不得着底），煮藕汁干，连袋焙干。用糯米5斗，以小蒸饭甑1个，先铺厚朴在甑中，安朱砂绢囊在厚朴上，又以厚朴周回裹定，却以米盖满，以盖(匚贛)之，桑火煮蒸1月，日夜火不绝，七日一换米汤，觉少便添熟汤，不得添冷水，1月满足，开袋，取辰砂焙干，入乳钵研细，糯米粽角为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
