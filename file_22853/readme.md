序号：22861
名称：西瓜霜
组成：西瓜1个。
出处：《疡医大全》卷十七。
主治：咽喉、口齿、双蛾喉痹，命在须臾；喉疼、火眼、火疮、肿毒、口烂、牙疼、外痔等一切热患。
加减：None
功效：止痛、防腐、消肿。
用法用量：每用少许吹之。
制备方法：用大黄泥钵1个，将西瓜1个照钵大小松松装入钵内，将瓜切盖，以皮消装满瓜内，仍以瓜盖盖，竹签扦定，再以一样大的黄泥钵一个合上，外用皮纸条和泥将缝封固，放阴处过数日，钵外即吐白霜，以鹅毛扫下收好，仍将钵存阴处，再吐再扫，以钵外无霜为度。收好。
临床应用：None
各家论述：None
用药禁忌：忌食辛辣食物；白喉忌用。
附注：咽喉独圣散（《疡科纲要》卷下）。
