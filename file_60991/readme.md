序号：61006
名称：酒癥丸
组成：巴豆160个（150个去皮膜，纸上炒去油，10个不去皮生用），半夏1钱半，粉霜1钱半，神曲半两，乳香1钱，面1斤，硇砂1钱半，轻粉1钱半。
出处：《鸡峰》卷九。
主治：酒积。
加减：None
功效：开胃取痰。
用法用量：每服1-2丸，食前开胃口，食后止痰涎嗽，生姜酒送下；消食，浓煎萝卜汤送下，中酒，嚼丁香，生姜酒送下3丸。
制备方法：上将麦糵末半两，用水半碗，熬麦糵末至8分一盏，去滓，再入黄连2-3钱熬成水黄，和硬软得所，少水只熬黄连，水添为丸，如小豆大，晒干，用陈粟米半升炒丸子如银褐色为度。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
