序号：63796
名称：清疳散
组成：枳壳2钱，青皮3钱，广皮3钱，楂炭3钱，槟榔1钱半，白术1钱半，半夏1钱半，白芍1钱，厚朴1钱，茯苓1钱，大黄1钱，朱砂1钱，象牙3分，牛黄1分，胆星8分，珍珠3分，麝香2分，冰片3分。
出处：《全国中药成药处方集》（沈阳方）。
主治：停食宿乳，体倦腹胀，午后发烧，面色萎黄，肌肉消瘦，目赤口臭，大便稠粘，尿色赤黄，积久变疳。
加减：None
功效：清肝解热，化积理脾。
用法用量：满1岁小儿每服1分，2岁至3岁者每服2分。
制备方法：先将枳壳等11味研为极细末，再将朱砂等7味研为极细面，兑匀。
临床应用：None
各家论述：None
用药禁忌：忌腥辣及硬性食物。
附注：None
