序号：63043
名称：清肺汤
组成：黄芩（去朽心）1钱半，桔梗（去芦）1钱，茯苓（去皮）1钱，陈皮（去白）1钱，贝母（去心）1钱，桑白皮1钱，当归7分，天门冬（去心）7分，山栀7分，杏仁（去皮尖）7分，麦门冬（去心）7分，五味子7粒，甘草3分。
出处：《回春》卷二。
主治：一切咳嗽，上焦痰盛。
加减：痰咯木出，加瓜萎、枳实、竹沥，去五味子；咳嗽喘急，加苏子、竹沥，去桔梗；痰火咳嗽，面赤身热，咯出红痰，加芍药、生地黄、紫花、阿胶、竹沥，去五味子、杏仁、贝母、桔梗；久嗽虚汗多者，加白术、芍药、生地黄，去桔梗、贝母、杏仁；久嗽喉痹，声不清者，加薄荷、生地黄、紫花、竹沥，去贝母、杏仁、五味；嗽而痰多者、加白术、金佛草，去桔梗、黄芩、杏仁；咳嗽身热，加柴胡；咳嗽，午后至晚发热者，加知母、黄柏、生地、芍药、竹沥，去黄芩、杏仁；咳嗽痰结，胁痛者，加白芥子、瓜蒌、枳实、砂仁、木香、小茴、竹沥、姜汁少许，去贝母、杏仁、山栀，亦加柴胡引经。
功效：None
用法用量：加生姜、大枣，水煎，食后服。
制备方法：上锉。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
