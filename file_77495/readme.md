序号：77511
名称：干地黄汤
组成：干地黄3两，芍药3两，当归3两，蒲黄3两，生姜5两，桂心6两，甘草1两，大枣20枚。
出处：《千金》卷三。
主治：妇人产后两胁满痛。
加减：None
功效：None
用法用量：上(口父)咀。以水1斗，煮取2升半，去滓分服，1日3次。
制备方法：None
临床应用：None
各家论述：1.《济阴纲目》汪淇笺释：世谓木得桂而枯，故用之以治胁痛，然不可不审寒热虚实也。此方以桂心为君，而以血药为辅，用之以温肝家血寒之痛则可，用之以治肝火作痛则不可，须详辨之。_x000D_
2.《千金方衍义》：产后虽两胁满痛而无寒热往来，知非少阳客邪，仍须内补建中，加地黄辅佐归、芍和营，加蒲黄辅佐桂心散血；以肝血不能循经输运而流入少阳部份，故专散血为主。
用药禁忌：None
附注：None
