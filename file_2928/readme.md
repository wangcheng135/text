序号：2928
名称：天南星散
组成：天南星1个（重9钱以上者。就地作小坎，深入8寸许，炭火烧通红，以真米醋半盏，洒入坎中，即纳南星于内，又以火炭密盖之，更用盆盖其上，1饭时取出，洗净，切，焙）。
出处：《医林纂要》卷九。
主治：惊痫。
加减：None
功效：祛风豁痰。
用法用量：或1钱，或5分，如风痫、肝痫，用生姜4片，紫苏5叶，同煎，加猪胆汁少许和服；如惊多及心痫，用琥珀(研)1钱，全蝎1钱，石菖蒲5分，同生姜汤调末服；如乳滞及脾痫，用巴豆霜少许，杏仁3粒，同和，以人参汤调末服；如痰涌肺痫，用皂角末少许，沙参汤调末服；如肾痫僵直，用乌蛇干肉1钱，附子制熟者1片，同炮姜汤调末服。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
