序号：54610
名称：柴胡煎
组成：柴胡（去苗）1两半，麻黄（去根节，汤煮，掠去沫）1两半，甘草（炙）1两半，木通（锉）1两半，紫菀（去苗土）1两半，五味子1两半，大青干1两半，百合1两半，款冬花3分，蓝叶3分，人参3分，赤茯苓（去黑皮）3分，大黄（锉，炒）半两，酥1斤，蜜（炼熟，去白沫）1升。
出处：《圣济总录》卷一八○。
主治：小儿脑热肺壅，鼻干无涕，喘息不得。
加减：None
功效：None
用法用量：5-6岁儿每服1钱匕，食后温熟水调下，日3次。
制备方法：除酥、蜜外，上锉细。水7升，慢火煎至3升，去滓入酥、蜜，同熬成煎，瓷器盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
