序号：73319
名称：普济消毒散
组成：黄连2两，黄芩（酒炒）2两，陈皮5钱，玄参5钱，生甘草5钱，川芎5钱，鼠粘子5钱，白姜蚕5钱，升麻5钱，柴胡5钱，葛根5钱，薄荷5钱，当归5钱，大黄5钱，连翘5钱，人参3钱，大兰根（如无，加靛花亦可）。
出处：《回春》卷二。
主治：大头瘟病。
加减：None
功效：None
用法用量：每服1丸，细嚼，白开水送下，发汗；如不及丸，用末药1钱2分，照前服；如未愈，再进一服，以汗为度，不可透风，若透风复肿，再服药，只是去皮一层方愈。
制备方法：上为细末，炼蜜为丸，每丸重2钱。
临床应用：None
各家论述：None
用药禁忌：忌酸、冷、羊、鸡、鱼之物并房事。
附注：方中大兰根用量原缺。
