序号：61053
名称：铁骨丹
组成：川乌头半两，草乌头半两（各炮，去皮脐尖），川芎半两，当归半两（酒浸，晒），辣桂半两，川续断半两（洗，晒），华阴细辛半两，补骨脂半两（炒），乌蛇（洗，酒浸，取肉，晒干）7钱半，直僵蚕半两（炒，去丝），木鳖子（去壳，炒熟）半两，天麻（酒浸，晒）半两，巴戟（酒浸，去心，晒）半两，防风半两，滴乳香半两，没药半两，麻黄（去节）半两，羌活半两，独活半两，坚白南星（炮熟）半两，白蒺藜（炒，捣去刺）半两，薏苡仁半两，苍术（炒）半两，萆薢（盐水煮干）1两，杜仲（去粗皮，锉，姜汁制，炒焦）1两，牛膝（酒浸，晒）1两，虎胫骨3两（洗，酒浸，炙焦），自然铜（烧红，醋淬7次）3两，白附子（炮）4钱，川五灵脂4钱，秦艽2钱半，全蝎（去毒，微炒）2钱半，麝香半钱。
出处：《直指》卷三。
主治：诸风瘫痪拳挛，半身不遂。
加减：None
功效：None
用法用量：每服1丸，温酒磨下，或宣木瓜煎汤送下、黑豆淋酒送下，不拘时候。仍间用高良姜（碎）1升煎汤围熏，汤浴最妙。未服药前，须以斑蝥21个（去翅足），用黑豆一建盏，慢火同炒焦，只用7个，并豆入全蝎14枚（微炒），五灵脂2钱半，为末，糕糊为丸，如麻子大，每服21丸，老酒送下，先去其风根。
制备方法：上为细末，乳香、没药、麝香别研，渐入拌和，以浸药酒调飞面煮糊为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中川乌头至补骨脂等八味用量原缺。
