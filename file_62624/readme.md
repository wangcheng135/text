序号：62640
名称：清中汤
组成：菖蒲（家生者，刮去皮须，切片，米汁浸3伏时，漉去苦水，称）8两，生姜5两（不去皮，片子），白术3两，甘草2两，白盐4两（炒，与菖蒲、生姜同淹1宿，焙干），麻黄（去根节）2两，细辛（去节叶）1两，五味子（微炒）2两，桂（去粗皮）1两，半夏（汤浸去滑，生姜汁制，焙干）2两（1方用干姜1两）。
出处：《普济方》卷一八三引《十便良方》。
主治：上气虚壅，精神倦怠，头目不爽。
加减：None
功效：None
用法用量：每服3钱，以水1盏半，加生姜1片，大枣3枚（去皮），煎至7分，去滓温服，每日3次。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
