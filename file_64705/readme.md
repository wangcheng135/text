序号：64721
名称：硇砂丸
组成：硇砂半两（水煎飞成霜），木香半两，桂半两（去粗皮），肉豆蔻半两（去壳，炮），茴香子半两（炒），附子半两（炮裂，去皮脐），青橘皮半两（去白，焙），陈橘皮（去白，焙）半两，山芋半斤，木瓜2枚。
出处：《普济方》卷二二二引《博济》。
主治：脾元虚弱，冷气攻上，饮食减少。
加减：None
功效：None
用法用量：每服20-30丸，盐汤或盐酒送下。
制备方法：上为末，先以木瓜切盖出瓤，如小瓮子样，存取原盖；逐个木瓜共入硇砂末半两，以原盖紧缚；用无灰酒3升，于银石器内，以慢火煮，候酒欲干，木瓜烂，取出入前八味药末相和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
