序号：68382
名称：黄耆鳖甲汤
组成：黄耆1两，鳖甲1两（去裙襴，醋炙），秦艽1两（去苗土），柴胡1两（去苗），当归1两（切，焙），知母（切，焙）1两，人参半两，芎半两，羌活（去芦头），赤茯苓半两（去黑皮），黄芩半两（去黑心），紫菀半两（去土），甘草半两（炙），芍药半两，桑根白皮半两，白鲜皮半两，款冬花半两，陈橘皮半两（汤浸，去白，焙），贝母半两（去心，炒），木香半两，桂半两（去粗皮），附子（炮裂，去皮脐）半两，丁香1分。
出处：《圣济总录》卷八十七。
主治：风劳。四肢倦怠，百节酸疼，饮食减少，行履不得，涕唾稠粘，多困少力，面色萎黄，小便赤涩。
加减：None
功效：None
用法用量：每服3钱匕，水1大盏，入乌梅1枚，生姜3片，大枣2枚（擘），同煎至7分，去滓，入麝香少许，稍热服。入酒一半同煎尤佳。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：鳖甲汤（《普济方》卷二二九）。
