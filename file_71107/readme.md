序号：71123
名称：照水丸
组成：龙脑1钱，滑石1钱，丹砂（通明者）1钱，乌贼鱼骨（去甲）1钱。
出处：《圣济总录》卷一一一。
主治：眼生翳障及一切目疾。
加减：None
功效：None
用法用量：先用黄蜡（皂子大）2-3块，纳新白瓷盏内，于慢火上熔，用纱帛子滤过，在净盏内再熔了，与前药末共同拌和，捏作饼子，如半破豌豆大，用薄绢或纱袋子盛了；以硇砂半两，放在净碗内，上交横安竹片，铺药饼，借硇砂气熏，用大棒1片，合碗口，勿令透气；掘一地坑，放药碗在坑内，用竹簟子1片盖了，然后以黄土盖之，7日取出，净瓷瓶中收药饼，硇砂不用。临卧将1饼扎在眼眦头，即睡至晓，用水1碗，向东觑水碗，其药自落在水中浮浴，却用绢帛子裹起，安洁净处，临卧依前再使，每饼可用半月，候药力慢时，方易1饼，如两目有疾，即用两饼。
制备方法：上各为细末，再同研匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
