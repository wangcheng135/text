序号：26590
名称：八宝眼药
组成：珠粉1钱，西牛黄3分，麝香3分，冰片1钱5分，珊瑚5分，玛瑙5分，熊胆6分，青鱼胆2只，制甘石5钱，海螵蛸7分，黄连2钱，荸荠粉2钱5分，蕤仁霜1钱。
出处：《中药成方配本》。
主治：暴发火眼，肿痛眵粘，障翳胬肉，羞明流泪。
加减：None
功效：清火止痛，消肿退翳。
用法用量：轻者每日点2至3次，重者点4至5次，点于大眼角。
制备方法：先将黄连煎汁去滓，化熊胆、青鱼胆，拌入制甘石内，晒干研末；再将珠粉、珊瑚、玛瑙各飞净末；后将其余西牛黄等6味，各取净末，与前药末一并和匀，共研至极细为度，约成粉1两2钱5分，分装120瓶，每瓶1分。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
