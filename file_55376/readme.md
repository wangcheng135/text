序号：55391
名称：大省风汤
组成：川芎1两，半夏1两，防风1两，甘草（炙）半两，全蝎（去毒）3个，附子（炮，去皮脐）半两，川乌（炮，去皮脐）半两，木香半两，南星半两。
出处：《得效》卷十三。
主治：诸虚风涎潮，痰厥神昏，头晕语涩，手足搐搦，半身不遂；及历节风痛，筋脉挛急。
加减：气虚，加沉香；气逆，加紫苏；胸膈不利，有痰，倍加半夏、人参；头晕头痛，加天麻半两，全蝎1个，煎熟入麝香；热风左瘫右痪，口眼斜，口噤不能言，手足顽麻，去附子、川乌。
功效：None
用法用量：每服4钱，水1盏半，加生姜10片，水煎，温服，不拘时候。
制备方法：上锉散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
