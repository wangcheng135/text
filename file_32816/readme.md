序号：32825
名称：花蕊石散
组成：乳香2钱，没药2钱，羌活2钱，紫苏2钱，细辛2钱，草乌2钱，蛇含石（便煅3次）2钱，厚朴2钱，白芷2钱，降香2钱，当归2钱，苏木2钱，檀香2钱，龙骨2钱，南星2钱，轻粉2钱，麝香3分，花蕊石（重便煅7次）5钱。
出处：《外科正宗》卷四。
主治：跌扑伤损及金疮、刀、箭、兵刃所伤，断筋损骨，疼痛不止，新肉不生者。金刃伤，肚破肠出者。
加减：None
功效：None
用法用量：葱汤洗净，用此掺之，软绵纸盖扎，1日1换。此药1时未备，可用多骨疽门生肌散代之暂用，亦可取效危急也。
制备方法：上为极细末，罐收听用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
