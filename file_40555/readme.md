序号：40565
名称：虎液
组成：龙砂（制，入阳城罐内封固，桑柴火煅红，1炷香毕，取出，冷成腻粉）2两，紫苏、薄荷、防风、荆芥、羌活、连翘、蕲艾。
出处：《眼科阐微》卷四。
主治：目赤肿痛疴痒；云膜胬肉，赤白翳障。
加减：None
功效：None
用法用量：饭后点大眼角，不可近黑珠，每日点10余次。
制备方法：上药后7味，用水5碗煎浓汁，滤去渣，再用水2碗，煎至1碗，去渣，共入一处，澄去沙泥，将甘石2两，共入铜锅内，入汁之时，以不见甘石为度，每次煅干，如此数次，汁尽为度；再以生姜汁煮三次，煅子，不必红，恐伤药性；入朱砂5分，听用。
临床应用：None
各家论述：None
用药禁忌：None
附注：甘石制法：甘石打碎如豆，入铜锅内，用白童便浸出二指，桑柴煮干，取起，再浸再煮，如此七次，尝苦咸味方止。如淡再煮。每次要焙干，取起听用。原书用本方，须兑五烹、龙砂、冰片合用。参阅“五烹”、“龙砂”条。
