序号：513
名称：五兽三匮丹
组成：鹿茸（酥炙）、麒麟竭（即草竭也）、虎胫骨（解片，酥炙）、牛膝（去芦，浸酒）、狗脊（亦草根也，燎去毛）各等分（上修事为细末，即五兽丹料），辰砂1两（细末），附子大者1只（生，去皮脐，旋中心空，入辰砂于内），宣木瓜1只（剜去心，仍薄薄去皮，入上附子于内，以旋附子末盖附子口，正坐于银暖罐中，重汤蒸10分烂，附子断白为度，即三匮丹也）。
出处：《医方类聚》卷九十八引《澹寮》。
主治：气血虚损，肝肾不足，两脚痿弱。
加减：None
功效：None
用法用量：木瓜酒和降气汤送下。
制备方法：上用三匮丹研膏，调和五兽末子，为丸如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
