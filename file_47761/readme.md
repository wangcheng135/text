序号：47776
名称：神仙养命驻颜草还丹
组成：川乌头1斤（净洗7遍，去皮脐），枸杞根1斤（冬月取，净洗，锉），何首乌1斤（净洗），威灵仙1斤（净洗，并控干），黑豆1斗。
出处：《医方类聚》卷二○四引《修真秘诀》。
主治：None
加减：None
功效：补精华，益骨髓，黑髭鬓，养神延寿，令邪气不侵凌，筋力不衰老、血脉通畅、神色不变。二十岁服之，一生容颜如故；中年服之，百年如故，五脏常安，四肢无疾，温黄寒热不侵。
用法用量：每服30丸，空腹、午、晚以温酒、盐汤任下。本方应于岁除日躬亲修合。
制备方法：取甑1只，先入黑豆之半铺底，次纳四药，而后将豆之另一半盖于药上；将药甑谨密盖覆，慢火蒸一伏时，取药细切，其豆不用，焙干为末，以无灰酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌犬肉，一生不可吃。
附注：None
