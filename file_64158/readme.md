序号：64174
名称：猪肾汤
组成：猪肾1具，白术4两，茯苓3两，桑寄生3两，干姜3两，干地黄3两，芎藭3两，麦门冬1升，附子（中者）1枚，大豆3合。
出处：《千金》卷二。
主治：妇女曾伤九月胎者。
加减：None
功效：预服养孕。
用法用量：上(口父)咀。以水1斗，煮肾令熟，去肾，纳诸药，煎取3升半，分4服，日3夜1，10日更1剂。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：孕至九月，可无伤损之虞矣。以前曾有所坠，必是肾气虚寒，不能司成实之令。故用猪肾引领术、附，大温肾气。然须十日更与1剂，不可多服，恐热遗子脏，反婴胎毒耳。
用药禁忌：忌猪肉、冷水、芜荑、桃、李、雀肉、酢物等。
附注：None
