序号：65870
名称：萆薢丸
组成：萆薢（锉）1两，石斛（去根，锉）1两，五加皮1两，防风（去芦头）1两，桂心1两，柏子仁1两，天雄（炮裂，去皮脐）1两，仙灵脾1两，酸枣仁（微炒）1两，山茱萸1两，钟乳粉1两，巴戟1两，菟丝子（酒浸3日，晒干，别捣为末）1两，鹿茸1两半（去毛，涂酥，炙微黄），牛膝1两半（去苗）。
出处：《圣惠》卷二十七。
主治：虚劳偏枯，手脚无力，肌肤消瘦，行立不得。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前温酒送下。
制备方法：上为末，研入钟乳粉令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
