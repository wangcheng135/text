序号：10382
名称：牛黄丸
组成：牛黄半两（研），蚱蝉3枚（微炒，去翅足），龙齿2两（细研），栀子仁2分，川升麻3分，犀角屑3分，胡黄连3分，钩藤3分，石膏2两（细研，水飞过），金箔90片（细研），银箔50片（细研），龙胆3分（去芦头），川大黄3分（锉碎，微炒），杏仁3分（汤浸，去皮尖双仁，麸炒微黄）。
出处：《圣惠》卷八十五。
主治：小儿热痫，发歇不定，眼目直视，身体壮热，吐沫，心神迷闷。
加减：None
功效：None
用法用量：每服5丸，以竹沥研服之。
制备方法：上为末，入研了药，同研令匀，炼蜜为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
