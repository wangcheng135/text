序号：67055
名称：麻黄止汗通肉解风痹汤
组成：麻黄（去节）3两，枳实（炙）3两，防风3两，白术3两，细辛3两，石膏8两（碎，绵裹），生姜4两，附子（炮）4两，甘草（炙）2两，桂心2两。
出处：《外台》卷十六引《删繁方》。
主治：肉极热，肌痹，淫淫如鼠走身上，津液脱，腠理开，汗大泄为脾风，肉色败，鼻见黄色。
加减：None
功效：None
用法用量：上以水9升，先煮麻黄，去沫，下诸药，煮取3升，分3次服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌猪肉、海藻、菘菜、生葱、生菜、桃、李、雀肉等。
附注：麻黄汤（《圣济总录》卷十九）、解风痹汤（《永乐大典》引《风科集验》）。
