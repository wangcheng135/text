序号：37922
名称：拨云锭子
组成：炉甘石（将炉甘石拣去隔石，选洁白者，尤以纸包石，用醋坛头上泥入桶内，以童便浸透炼熟，少少糊在纸外，又以火消末滚在泥球上，外再以厚泥包圆，大约炉甘石1斤可分作4-5个球；每1斤炉甘石，用火消1两研细，滚这4-5个球为度，球成晒干，如有缝以泥补之，另以砖砌1大炉，架火将球放炭火上炼1天，申刻火将完，可将球取起，翻转入炉，添火加炭，过夜不必守之，随炉中炭火化完为度；1炉可炼炉甘石2-3斤，每炉甘石4两，用川黄连5钱，龙胆草5钱，河水5碗浸1夜，煎数沸去滓，滤净清，将炉甘石煅红，倾入药汁内，取起又煅又淬，以汁尽为度，药水内落下炉甘石，再燉干药水，俱取入炉甘石内为妙，每料只用制炉石3钱，再加）、熊胆5分，冰片2分5厘，白硼砂3分，麝香5厘，朱砂（水飞）3分，活乌鸦翎2寸2分（煅）。
出处：《疡医大全》卷十一引刘长随方。
主治：一切眼疾。
加减：None
功效：开瞽复明。
用法用量：凡点眼时，以清水或人乳，或净唾润湿点眼，闭目少刻即效。临卧点之更妙。或用人乳化开，涂眼胞上下，揉入眼内，多涂过夜，即日见效。
制备方法：上为细末，用川芎、当归、赤芍药、生地、薄荷、防风、防己、川黄连、甘菊花、龙胆草各5钱，木贼草、黄芩、黄柏、羌活、大黄、白芷各2钱，河水6碗，浸1日夜，炭火熬出汁来，去滓，澄清沥净，再用文火熬成膏，和前药末和匀，搓成条子，重2分，用鹅毛管收藏，黄蜡塞口。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
