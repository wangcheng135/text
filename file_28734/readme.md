序号：28742
名称：十八味黄耆建中汤
组成：黄耆（蜜炙）2两，熟干地黄（洗）2两，肉桂（去粗皮，不见火）2两，甘草（炙）2两，人参（去芦）2两，当归（酒浸，去芦）2两，鳖甲（米醋炙）2两，白茯苓（去皮）2两，南木香（不见火）1两，地骨皮（去骨）1两，柴胡（去苗）1两，秦艽（洗净）1两，附子（炮，去皮脐）1两，五味子（酒洗）1两，川芎1两，阿胶（蚌粉炒）1两，半夏（汤泡7次）1两，白芍药4两。
出处：《魏氏家藏方》卷四。
主治：荣卫不调，五心烦热，状如劳疟，口苦舌干，不思饮食，一切虚损。
加减：None
功效：None
用法用量：上(口父)咀。每服2大钱，水1盏半，加生姜5片，大枣2个，同煎至7分，去滓，空心服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
