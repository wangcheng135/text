序号：583
名称：五痿汤
组成：人参1钱，白术1钱，茯苓1钱，甘草（炙）4分，当归1钱5分，苡仁3钱，麦冬2钱，黄柏（炒褐色）5分，知母5分。
出处：《医学心悟》卷三。
主治：五脏痿。
加减：心气热，加黄连3分，丹参、生地各1钱；肝气热，加黄芩、丹皮、牛膝各1钱；脾气热，加连翘1钱，生地1钱5分；肾气热，加生地、牛膝、石斛各1钱5分；肺气热，加天冬、百合各2钱；夹痰，加川贝、竹沥；湿痰，加半夏曲；瘀血，加桃仁、红花。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：《证因方论集要》：治痿之法不外补中祛湿，养阴清热而已。人参、白术、炙草以补中；当归、麦冬以养阴；茯苓、苡仁以祛湿；黄柏、知母以清热。
用药禁忌：None
附注：None
