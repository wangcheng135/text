序号：20924
名称：人参散
组成：人参半两（去芦头），半夏半两（汤洗7遍去滑），桂心半两，干姜半两（炮裂，锉），白术半两，草豆蔻1两（去皮），甘草半两（炙微赤，锉），陈橘皮1两（汤浸，去白瓤，焙），枇杷叶半两（拭去毛，炙微黄），荜茇半两，大腹皮1两（锉），丁香半两，诃黎勒皮1两，厚朴1两（去粗皮，涂生姜汁，炙令香熟）。
出处：《圣惠》卷五十。
主治：五噎，胃管气滞，心胸满闷，咽中噎塞，不能下食。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，煎至6分，去滓，不拘时候，稍热服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
