序号：38116
名称：松叶膏
组成：松叶（切）1升，天雄（去皮）4两，松脂4两，杏仁（去皮）4两，白芷4两，莽草1两，甘松香1两，零陵香1两，甘菊花1两，秦艽2两，独活2两，辛夷仁2两，香附子2两，藿香2两，乌头（去皮）3两，蜀椒（汗）3两，芎3两，沉香3两，青木香3两，牛膝3两，踯躅花1两半（并锉）。
出处：《外台》卷三十二引《延年秘录》。
主治：头风耳塞，头旋发落，白屑风痒。
加减：None
功效：None
用法用量：以涂发根，日3度摩之。
制备方法：上药以苦酒3升浸1宿，生麻油1斗，微火煎三上三下，苦酒乞尽，膏成，滤去滓，盛贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：松脂膏（原书同卷引《崔氏方》）。
