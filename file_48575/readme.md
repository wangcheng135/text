序号：48590
名称：神衰汤
组成：石决明12g，珍珠母12g，钩藤10g（后下），菊花10g，丹参10g，赤芍10g，夜交藤12g，合欢皮10g，淮小麦12g，炙甘草4.5g，鲜竹叶10g。
出处：《千家妙方》引黄文东方。
主治：神经衰弱，证履肝阴不足，肝阳上亢，心火偏旺，胃失和降者。
加减：None
功效：平肝潜阳，和胃安神。
用法用量：水煎服，每日1剂。
制备方法：None
临床应用：神经衰弱：某女，20岁，学生。由于学习紧张，以致严重失眠，每晚仅能入睡一小时许，食欲差，嗳气，舌质淡青，脉象弦细。脉症合参，乃肝阴不足，肝阳上亢，心火偏旺，胃失和降。治宜平肝潜阳，和胃安神。服神衰汤7剂，夜寐转佳，能入睡3－4小时，后以原方去钩藤，加入炒枣仁10g，再服14剂，睡眠增加，胃纳亦佳。改服补心丸，一月后追访，虽停药多日，但睡眠仍好。
各家论述：None
用药禁忌：None
附注：None
