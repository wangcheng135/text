序号：20262
名称：肉苁蓉散
组成：肉苁蓉1两（酒浸1宿，刮去皱皮，炙令干），熟干地黄1两，白茯苓3分，白芍药半两，桂心半两，牛膝3分（去苗），麦门冬1两（去心），白石英1两（细研），附子3分（炮裂，去皮脐），黄耆1两（锉），牡蛎1两（烧为粉），磁石1两（捣碎，水淘去赤汁），五味子3分，人参3分（去芦头），续断3分，萆薢半两（锉），地骨皮半两。
出处：《圣惠》卷五十三。
主治：大渴后，下元虚乏，日渐羸瘦，四肢无力，不思饮食。
加减：None
功效：None
用法用量：每服用獖猪肾1对，切去脂膜，先以水1大盏半，煎至1盏，去滓，入药5钱，加生姜1分，薤白3茎，煎至5分，去滓，食前温服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
