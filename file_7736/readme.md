序号：7736
名称：圣灵丸
组成：木猪苓（去乌皮）1两，京三棱（去皮）1两，青皮（去瓤）1两，白茯苓（去皮）1两，白术1两，麦糵1两，川白姜3钱，黑牵牛半两（炒黄色），泽泻1两。
出处：《医方类聚》卷一二九引《吴氏集验方》。
主治：水肿。
加减：None
功效：消水气。
用法用量：如早晨面上并眼胞上有气，巳时四肢皆肿，煎桑白皮汤，临卧送下15丸；如头面脚手不肿，只腹肿，当分二气证候，腹带黑色，名脾胃气蛊，若腹黄色，肚上青筋见，名蛊气，煎用木通汤，食后送下20丸。
制备方法：上为末，醋糊为丸，如梧桐子大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：忌盐、酱、面、糯米、虾蟹、无鳞鱼、花鸡、鸭、牛、羊、猪。
附注：None
