序号：84106
名称：五圣汤
组成：贯众半两，黄连半两（去须），甘草半两（微炙），吴茱萸半两（汤洗7次），白茯苓半两（去皮，以上5味并生用）。
出处：《杨氏家藏方》卷三。
主治：暑积年深，每遇夏月不进食饮，疲倦少力，见日色则头目昏痛，恶心多睡。暑病呕恶。
加减：None
功效：None
用法用量：上(口父)咀，平分2服。每服用水1碗半，煎至1碗，去滓放冷，候日午时，先取香熟甜瓜1枚，切去皮，作12片，先嚼瓜1片，呷药1-2呷送下了，再如前嚼瓜1片，呷药1-2呷，以药汁尽为度。不损脾胃，不动脏腑，须是大烦躁时服之。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
