序号：53631
名称：家传秘结祛痛散
组成：青皮（去白）2钱，灵脂（研飞，去沙土）2钱，川楝子肉2钱，穿山甲（土拌炒）2钱，良姜（香油炒）1钱5分，玄胡索1钱5分，没药1钱5分，沉香1钱，八角茴香2钱，槟榔1钱5分，木香1钱2分，砂仁少许。
出处：《保命歌括》卷三十。
主治：诸般心气疼痛，气滞不行，攻刺心腹，痛连胸胁，小肠吊疝，及妇人血气刺痛。
加减：None
功效：None
用法用量：每服1钱，加盐一星，用酒或滚水调下。
制备方法：上（口父）咀，为粗末，用木鳖（去壳）1钱2分切片，同药炒至香焦，去木鳖不用，研为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
