序号：48231
名称：神明膏
组成：栝楼1枚（去皮瓤，只取仁子），赤芍药1分，甘草（微炙）1分，黄耆1分，杏仁（汤浸，去皮尖）1分，香白芷1分，当归（洗，焙）1分，桃仁（汤浸，去皮尖）1分，人参（去芦头）1分，川芎1分，苍术（米泔浸1宿，焙）1分，桑白皮1分，沉香半两，零陵香半两，藿香叶（去土）半两。
出处：《杨氏家藏方》卷十二。
主治：痈疽，发背，一切疮肿，打扑伤损，汤火金疮，干湿癣，风痒顽麻。
加减：None
功效：None
用法用量：若内伤，用药1钱匕，酒化服；口疮，含化少许；恶疮多年不生肌者，先以葱汤洗净，用药敷之；鼻内有肉铃子者，以纸捻子蘸药点之，1月可取之；干湿癣、风痒顽麻，并以药摩之。
制备方法：上锉细，用清麻油15两，浸药49日，候日满先倾油入银锅中，慢火炼令香熟，放冷却入诸药，以文武火养1日，候药色半焦滤去滓，却用鹅梨3枚（取汁），黄蜡1两半，麝香1分，细研，并入药内重炼，候油不滚起，乃成膏也，用新绵滤过，待冷入研细生龙脑1分，搅匀，入新瓷器中盛之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
