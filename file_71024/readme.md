序号：71040
名称：稀痘保婴丹
组成：缠豆藤4两，真紫草茸4两（酒洗），荆芥穗2两，牛蒡子2两，升麻2两（盐水炒），甘草梢（生）2两，大辰砂3钱（麻黄、紫草、荔枝壳、升麻4味煮过，就将其水飞辰砂），防风2两，天竺黄（点少许于舌上，麻涩者真）1钱2分，牛黄（磨透者佳）1钱2分，蟾酥（自取赤睛者不用）1钱2分，赤小豆49粒（炒勿令焦），黑豆49粒（炒勿令焦），绿豆49粒（炒勿令焦）。
出处：《痘疹金镜录》卷下。
主治：痘未见点之先。
加减：None
功效：预服解毒。
用法用量：每服1丸，大人2丸，未出痘之先，浓煎甘草汤磨下；已发热之时，生姜汤磨下。微表之多者可少，已者可无，大有神效。
制备方法：上为细末，外又将紫草3两，入水3碗，熬膏半碗，入生沙糖半盏，将前末药用紫草膏为丸，如圆眼核大，将飞过辰砂为厚衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：保婴丹（《赤水玄珠》卷二十七）、秘传保婴丹（《准绳·幼科》卷四）。
