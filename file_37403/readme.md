序号：37413
名称：庚字化霉丸
组成：蝉蜕（炒去沙土）2钱，川山甲（炙）2钱，川贝母2钱，钟乳石（须择湖广产者，长大色白；用天葵、甘草水煮1日，研万遍。色杂性坚者不用）1钱，生牛乳1钱，郁金2钱，牛黄4分5厘，木香1钱5分，月月红1钱5分，乳香1钱5分，白鲜皮1钱5分，雄黄1钱5分，朱砂1钱7分。
出处：《疮疡经验全书》卷六。
主治：霉疮，肺经内外前后形症。
加减：None
功效：None
用法用量：每早空心服15丸，每晚空腹服10丸，人参汤送下；奇良汤亦可。
制备方法：上药各为末，用神曲末5钱打稠湖，入药捣匀为丸，如梧桐子大，另研朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
