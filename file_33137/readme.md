序号：33146
名称：万应膏
组成：嫩松香1斤（黄老者不用），蓖麻仁2两，巴豆仁2两，康青1两，杏仁（开水泡，去皮）1两，胆矾5钱。
出处：《经验奇方》卷上。
主治：跌打损伤，青肿疼痛；痛风，肝胃气痛，腰痛，胸腹膈气疼痛；痈疽初起。
加减：None
功效：None
用法用量：临用时须重汤炖烊，摊贴患处。
制备方法：上药先将松香用铁锅水煎3次，取起候干，隔1日，研细禾；再以蓖麻仁、巴豆仁各炒黑枯（若不黑枯，其性太烈不能用），康青、胆矾各研细末；先将杏仁落石臼，舂烂如泥；次下巴豆、蓖麻仁，一并舂烂；再下康青，胆矾，缓缓舂之；并将松香分数次下，随下随舂；再用麻油屡涂杵臼，勿致粘住，千捶成膏，储瓷器。
临床应用：None
各家论述：None
用药禁忌：皮破出血、痈疽已成作痛者忌之。
附注：None
