序号：83810
名称：乌蛇丸
组成：乌蛇尾（酒浸，去皮，焙干，和骨用）1两，天南星（炮）1两，天麻1两，羌活（去芦头）1两，独活（去芦头）1两，白附子（炮）1两，白僵蚕（炒）1两，乌头（炮裂，去皮脐）1两，乳香（研）1两，丹砂（研）半两，龙脑（研）1分，麝香（研）1分，牛黄（研）1分。
出处：《圣济总录》卷九。
主治：体有偏虚，风客半体，不自收持，痛麻痹痛。
加减：None
功效：None
用法用量：每服7-10丸，薄荷酒送下，1日3次。
制备方法：上13味，8味各捣罗为细末，5味各研，先将乌蛇、丹砂研末，与蜜同熬成膏，和诸药为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
