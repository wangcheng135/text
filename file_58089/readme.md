序号：58104
名称：益府秘传拨云龙光散
组成：蕤仁（5两，去粗壳取仁，用温水浸，去嫩皮膜尖心，用上好白竹纸包裹，捶去油以尽为度）5钱，牛黄2分5厘，白磁砂5分（即好白细瓷器4-5钱重，用头酸醋1碗，将瓷器以砂罐盛，放炭火内烧红，先投入醋内，以7次为度。又用童便1碗，烧红，投内以7次为度。又将醋、童便合1碗，又烧红投入以7次为度。先将瓷研炼，以水澄清，用中间阴干），好珍珠（8-9分，将雄鸡1只，以珠入鸡肚内过1宿，然后杀鸡取珠，用豆腐蒸过用）5分，硼砂2钱5分，琥珀5分，真熊胆（3分，以瓷瓦盛，放火上煨去水）2分5厘，硇砂（3-4分，将冷水1碗，以水煮干为度）1分，当门子1分，白丁香1分，海螵蛸（水煮过6-7次）2分，冰片1分，人龙（用男人孩子口内吐出食虫，即用银簪破开，河水洗刮令净，阴干）2分。
出处：《寿世保元》卷六。
主治：诸般翳障，攀睛胬肉，内障青盲。
加减：None
功效：None
用法用量：任意点眼。
制备方法：上药精制细研。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
