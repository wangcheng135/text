序号：69743
名称：琥珀丸
组成：熟干地黄（酒浸）1两半，白术（去芦）1两半（炒），续断（去芦，酒浸）2两，附子（炮，去皮脐）2两，蓬莪术（炒）2两，雄黑豆（炒熟，去壳）3两，刘寄奴（拣净，酒窨）3两，当归（去芦，酒浸）半两，白芍药半两，青橘皮（去瓤）半两，延胡索（蛤粉炒）半两，茴香（淘去沙，炒）半两，牡丹皮（炒）半两，乌药（炒）半两，蛇床子（炒）半两，陈橘皮（去白）半两，金钗石斛（去芦，酒浸1宿）半两，白芷（炒）半两。
出处：《魏氏家藏方》卷十。
主治：血海久冷，月经不调。
加减：None
功效：None
用法用量：每服30丸，空心、食前米饮送下，或温酒送下。久服自然有孕。
制备方法：上以米醋、无灰酒各3升，同煮干焙燥，入后药：防风（去芦）、琥珀（别研）、桔梗（炒）、蒲黄（隔纸炒）、官桂（去粗皮，不见火）各1两，共为细末，醋面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
