序号：45446
名称：久疟饮
组成：人参1钱，于白术2钱，云苓2钱，制半夏2钱，青皮8分，柴胡5分（炒），草蔻仁3钱（研），龙骨3钱，煨姜2钱，大枣3个。
出处：《慈航集》卷下。
主治：久疟不止，面色黄瘦，寒热不清。
加减：如腹胀气闷，加煨广木香8分；如作泻，加鹿角霜3钱；如夜发者，加炒升麻8分，夜明砂2钱；如口渴，加麦冬8分；如虚热，加青蒿3钱；如大便干结，加鲜首乌8钱，当归5钱；如阴虚咳嗽，加炙鳖甲5钱，川贝母1钱。
功效：None
用法用量：疟未来之前，早2时服，2煎接服之。1剂轻，2剂减去其半，3剂全止，4剂精神如常矣。
制备方法：河、井水煎，露1宿。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
