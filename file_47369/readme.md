序号：47384
名称：祛风清热散
组成：当归尾2钱，赤芍2钱，川芎1钱5分，生地黄3钱，黄连6分，黄芩2钱，栀子3钱，连翘3钱，薄荷8分，防风1钱5分，荆芥1钱，羌活2钱，桔梗8分，枳壳1钱，甘草8分，白芷梢1钱。
出处：《寿世保元》卷六。
主治：暴发眼肿如桃，并赤眼痛涩难开者。
加减：肿痛甚，加大黄、芒消；风热，加蔓荆子、牛蒡子；乌珠痛，加天麻、川乌（生用3片）；犯眼，加苍术、朱砂；眼生翳障，加白蒺藜；眼目被人打伤青肿，加大黄；如杖疮肿痛未破作憎寒壮热，打重血气攻心，加大黄、桃仁；如打扑伤损内重，瘀血不散，加桃仁、大黄。
功效：None
用法用量：加灯草7根，水煎，食后服。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
