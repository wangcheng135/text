序号：26760
名称：异类有情丸
组成：鹿角霜（以角之新者，寸截，囊置长流水中7日，瓦缶水煮，每角1斤，入黄蜡半斤，缶口用露酒1壶掩之，别沸流水旋添，勿令下竭，桑柴火足12时，其角软矣，竹刀切去黑皮，取白者，春细为霜）3两6钱，鹿茸（新如紫茄者，熏干，酒洗数过，酥油涂，炭火炙令透，为细末）2两4钱，龟版（8字文具者，醇酒浸7日，酥炙透黄）3两6钱，虎胫骨（新而真者，长流水浸7日，蜜酥和，炙透）2两4钱。
出处：《韩氏医通》卷下。
主治：丈夫中年觉衰。
加减：如厚味善饮之人，加猪胆汁1-2合。
功效：None
用法用量：每服50-70丸，空心盐汤送下。
制备方法：上为极细末，用水火炼白蜜，入豮猪脊骨髓9条，同舂剂为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
