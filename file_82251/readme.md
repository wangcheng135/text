序号：82267
名称：燕鼠膏
组成：全蝎（热水浸透，洗3次，晒干，净）2两，白芷1两，黄连1两，黄柏1两，黄芩1两，当归1两，山甲1两，生地5钱，赤芍5钱，官桂2两，海藻2两5钱（洗3次，晒干），番木鳖5钱（研碎）。
出处：《种福堂方》卷二。
主治：瘰疬痰核，痈疽发背肿毒。
加减：None
功效：None
用法用量：用时隔汤溶化摊贴，勿见火。
制备方法：用麻油1斤4两，浸药五日，熬焦黑色，去滓，将净油称准，每油2两，用飞净黄丹1两，收滴水不散，先入白占1钱5分，黄占3钱，即下黄丹，再下杭粉1两，用桑枝不住手搅成膏，候冷入水浸3-4日，再用文火熔化，再入没药3钱（去油），阿魏3钱，麝香1钱，血竭2钱，朝南燕窝泥5钱，雄黄1钱，朱砂1钱，两头尖7钱，白升丹4钱，上为极细末，入膏内搅极匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
