序号：58854
名称：秘制太和丸
组成：制香附4两，制苍术4两，广霍香4两，净防风4两，嫩前胡4两，紫苏叶4两，薄荷叶4两，川厚朴4两，草果仁4两，姜半夏4两，台乌药4两，广陈皮4两，焦麦芽4两，春砂壳4两，炒枳壳4两，焦山楂4两，白蔻米3两，广木香3两，茯苓3两，川芎3两，羌活3两，白芷3两，粉甘草3两。
出处：《萧山竹林寺妇科秘方考》。
主治：妇女信水不准，经行腹痛，腰痠带下，骨节疼痛，胸闷食少，停经化胀，脾虚泄泻，气血两亏，积年不孕。
加减：None
功效：健脾消积，化痰行气。
用法用量：每服1丸，温开水化服，每日2-3次。
制备方法：上为末，面糊为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
