序号：38490
名称：治消渴丸
组成：麦门冬（用上元柏桥鲜肥者）2大两，黄连（九节大者）1两。
出处：《普济方》卷一七七引《十便良方》。
主治：消渴。
加减：None
功效：None
用法用量：每服50丸，食后饮送下，1日2次。但服2日，其渴必定。若重者每服150丸，第2日120丸，第3日100丸，第4日80丸，第5日依次服之，至少可每日只服25丸。服讫觉虚，即取白羊头1枚，净去毛洗了，以水3大斗煮令烂，去头取汁，可1斗以来，细细服之，亦不用著盐，不过3剂平复。
制备方法：上为末，以肥苦瓜汁浸麦门冬经宿，然后去心，即于臼中捣烂，纳黄连末臼中，和捣为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
