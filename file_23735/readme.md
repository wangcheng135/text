序号：23743
名称：安坤赞育丸
组成：桑寄生8两，乳香8两，蕲艾8两，熟地8两，杜仲8两，制香附8两，山茱萸8两，鳖甲8两，没药8两，琥珀8两，白芍8两，乌药8两，当归8两，红花8两，龟版8两，泽泻8两，砂仁8两，柴胡8两，广陈皮8两，远志8两，酸枣仁8两，木香2两，川芎4两，沉香4两，青毛鹿茸4两。
出处：《全国中药成药处方集》（济南方）。
主治：妇女月经不凋，崩漏带下，腰酸腹痛，面色萎黄。
加减：None
功效：None
用法用量：每服1丸，白开水送下。
制备方法：上为细末，炼蜜为丸，重3钱，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：忌气恼及辛辣生冷等物。
附注：None
