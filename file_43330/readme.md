序号：43345
名称：养肺止嗽丹
组成：百合6两，生地3两，紫菀4两，川贝5两，白果4两，兜铃4两，天冬4两，五味子2两，枳壳2两，莱菔2两，木香3两，麻黄（炙）3两，白芍3两，杏仁4两，甘草4两，草蔻3两，苏子2两，蒌仁5两，玄参3两，陈皮3两，当归5两，京知母4两，熟地3两，阿胶珠8两，冬花3两，麦冬3两，橘红4两，乌药3两，桔梗2两，桑皮4两，槟榔3两，川朴4两，苏叶3两，粟壳10两，黄芩2两，茯苓3两。
出处：《全国中药成药处方集》（沈阳方）。
主治：新久咳嗽，痰中带血，痨伤咳嗽，骨蒸潮热，痰喘气短，身瘦食少，倦怠乏力，虚劳盗汗。
加减：None
功效：化痰止嗽，养肺补气。
用法用量：每服1丸，小儿半丸，白开水送下。
制备方法：上为细末，炼蜜为丸，2钱重。
临床应用：None
各家论述：None
用药禁忌：忌咸辣食物。
附注：None
