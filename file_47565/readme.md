序号：47580
名称：神仙六甲飞伏雄黄丹
组成：上等雄黄9两。
出处：《鸡峰》卷二十九。
主治：真阳不足，五脏气虚，肠风恶痢，百节之内大风积聚，妇人产后一切危证。
加减：None
功效：杀诸虫毒，去精邪，久服令人长寿。
用法用量：每服10丸，空心温酒送下。
制备方法：上为极细末，入于砂盒子内填实，上作一坑子，入生蜜4两，于蜜上放如莲子大六块防风，盖子合定，用盐泥7斤入纸同和熟软，固济盒子，阴干；于平地上作一地炉，深3尺2寸，阔3尺，内坐药盒子，再用细灰1斗盖之硬炭20斤，分4次烧之，须及一伏时火尽为度；候冷打开盒，其蜜或作数重，或重上须有雄黄，故曰飞伏也；其丹深紫为上，青亦妙，为极细末，其香如鸡黄（无雄黄之气也），酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
