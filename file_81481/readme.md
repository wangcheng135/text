序号：81497
名称：鳗鲡鱼煎丸
组成：大白鳗鲡鱼3条（用醋汤洗净，段截后以无灰酒于银锅内慢火煮熟，滤出取肉，细研，再入酒2升，慢火煎成煎），青蒿子1两，桔梗（锉，炒）1两，秦艽（去苗土）1两，柴胡（去苗）1两，知母（焙）1两，甘草（炙，锉）1两，鳖甲（九肋者，去裙襕，醋浸3日后，炙令黄熟）1两，人参1两，附子（炮裂，去皮脐）1两。
出处：《圣济总录》卷八十六。
主治：肺劳，咳嗽日久。
加减：None
功效：None
用法用量：每服20丸，空心、午后、临卧用炒栝楼根酒送下。
制备方法：除鳗鲡鱼为煎外，余为末，以鱼膏拌和匀，更捣百十杵，为丸如梧桐子大。或干，更入炼蜜少许。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
