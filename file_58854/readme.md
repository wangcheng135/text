名称：秘药方
组成：黄连1两，黄芩1两，黄柏1两，栀子1两，黄耆1两，薄荷1两，防风1两，荆芥1两，连翘1两，细辛1两，白芷1两，玄参1两，川芎1两，羌活1两，独活1两，山奈1两，槟椰1两，厚朴1两，苦参1两，甘草1两，木通1两，半夏1两，川乌1两，草乌1两，苍术1两，麻黄1两，赤芍1两，升麻1两，大黄1两，僵蚕1两，川牛膝1两，桔梗1两，射干1两，干葛1两，皂刺1两，车前1两，桑皮1两，五加皮1两，牛蒡子1两，麦冬1两，杏仁1两，地骨皮1两，山豆根1两，生地1两，归尾1两，花粉1两，生南星1两，银花参1两，三七1两，川槿皮1两，鲜车前草4两，骨牌草4两，金星草4两，五爪龙草4两，土牛膝草4两，紫背天葵草4两，地丁草4两。
出处：《青囊秘传》。
主治：咽喉诸症。
加减：None
功效：None
用法用量：每以铜吹筒取药少许，吹患上。预为修合，陈者愈佳。
制备方法：用新缸1只，清水浸之，日晒夜露49日，如遇风雨阴晦之日，用盖盖之，晒露须补足日期。取起滤去滓，铜锅煎之，槐柳枝搅之，煎稠如糊，再加下药：明雄黄5钱，青礞石（童便煅7次）、乳香（去油）、没药（炙）、熊胆（焙）、龙骨（煅）、玄明粉、血竭、石燕（醋煅7次）、海螵蛸（纸包，焙）、炉甘石（童便煅7次）、青黛各5分，枯矾、儿茶各1钱，轻粉、黄丹（水飞）各3分，月石7分，桑枝炭3钱。上为细末，入前膏内和匀，做成小饼，如指头大，晒露7日夜，放地上，以瓦盆盖之，1日翻1次，7日取起，置透风处阴干，收藏瓦罐内，3个月方可用之。用时为极细末，每饼2分，加后7味：冰片、珍珠、珊瑚（水飞）各4分，麝香2分，牛黄2分，轻粉1厘，月石2分，为细末，和匀，密收小瓶，封口勿令泄气。
