序号：76853
名称：骗马丹
组成：真川乌（炮）2两，川芎7钱，真苏木半两，地龙（去土）半两，白芷1两，草乌头（泡）1两，续断（浸酒）1两，牛膝（酒浸，去芦）1两，肉苁蓉（酒浸）1两，滴乳1两，明松香（研）1两，木鳖子（去壳，不去油）1两，虎胫骨（酒浸，炙）1两，骨碎补（酒浸）1两，自然铜（醋淬7次，水飞）1两，败龟版1两（煅红，好醋制净令黄色），全蝎3钱（去毒，炒）。
出处：《普济方》卷九十二引《瑞竹堂方》。
主治：中风，口眼斜，痰涎壅盛，语言謇涩，手足不仁，筋脉拘急，肢体不举；或寒湿相搏，肌肉顽麻，传入经络，筋骨疼痛，腰脚浮肿，难以屈伸；或寒湿脚气；或打扑损伤，筋骨体骱蹉跌，皮肤瘙痒，风毒疮疡。
加减：None
功效：None
用法用量：每服50丸，食后温酒送下。
制备方法：上为细末。用煮酒打陈米粉为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
