序号：75078
名称：鼠粘子酒
组成：鼠粘子1斗。
出处：《圣惠》卷九十五。
主治：一切风。
加减：None
功效：None
用法用量：每日平旦以物搅起令浊，即取温服1小盏，次1小盏服讫，封之。勿使气泄，良久方可饮食，晚间再服。病重者，服尽2硕即愈；若初觉即急服，不过1-2斗愈；亦疗历节风，痛贼风，风痹顽麻，重者不过5斗愈；腰脚疼痛，筋节急，重病后汗不留，四肢强直，服3斗愈；或因热食，体中如锥刺，口面戾，头旋心闷，呕吐，风在心脏，服3-4斗愈。
制备方法：以水淘去浮者，晒干，捣碎，于净砂盆内入无灰酒5升，研令极烂，即以绢罗滤取白汁，其滓再以酒5升研之，候滤白汁尽为度，续入酒2斗，相和令匀，纳不津器中密封，春、秋14日，夏7日，冬21日，日足则开。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
