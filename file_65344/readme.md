序号：65360
名称：女服紫石英丸
组成：紫石英1两（砂锅盛之，以石掩口，火煅红），禹余粮1两（火煅，醋淬），熟地黄1两（汤洗，清酒蒸，焙），辛夷仁1两，厚朴1两（去皮，不见火），卷柏1两（醋炙），石斛1两（去皮，锉，酒炒），川续断1两（酒浸），柏子仁1两（炒，别研），川乌1两（炮，去皮脐，锉），川芎1两，海螵蛸1两（醋炙），牡丹皮1两（去心），川当归1两（去芦，酒洗），粉草1两（炙），桑寄生半两，华阴细辛半两（去土叶，洗），山药半两，吴茱萸半两（炒），干姜半两（炮），人参7钱半，天门冬（去心）1两半，梓朴（去皮，姜汁涂炙）。
出处：《普济方》卷三三六。
主治：无子。
加减：None
功效：None
用法用量：每服70丸至100丸，空心米饮送下。以腹中热为度，尽剂当有娠。
制备方法：上为细末，醋为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：不禁房事，外出不可服。
附注：None
