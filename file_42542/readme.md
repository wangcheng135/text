序号：42557
名称：驻颜小还丹
组成：鹿角霜8两，龟版霜8两，虎胫骨（好酒炙）6两，天门冬（酒洗，去心）4两，熟地黄4两，人参（去芦）2两，松子仁2两，柏子仁2两，紫河车1具（焙干）。
出处：《赤水玄珠》卷十。
主治：诸虚百损。
加减：素有火者，加雄猪胆汁5枚，炼熟入之。
功效：久服返老还童。
用法用量：每服50-70丸，空心秋石汤送下。
制备方法：鹿角胶、龟版胶各4两，酒化开，同前药为丸，如梧桐子大。
临床应用：None
各家论述：此方用龟、鹿、虎者，以其多寿也，能壮人之筋骨；用天冬、地黄、人参等，法象三才，以补人之精髓；用紫河车以补人之元神；松、柏耐岁寒，皆足以养神气，非泛常草木可比，故有驻颜延寿之功。
用药禁忌：None
附注：None
