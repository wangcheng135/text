序号：36092
名称：参茸补血露
组成：当归5钱，川芎4钱，丹参1两，鹿茸2钱，枸杞3钱，五味子3钱，豆蔻3钱，焦术5钱，莲肉5钱，茯神4钱，远志5钱，节菖蒲5钱，甘草4钱，首乌4钱，生地5钱。
出处：《全国中药成药处方集》（沈阳方）。
主治：妇女气滞血亏，经闭经漏，赤白带下，腰腿痠痛，干血痨症。
加减：None
功效：补血益精。男服补精种子，女服调经受孕。
用法用量：温服，每次1杯，1日3次。
制备方法：上药以绢袋盛贮，用烧酒5斤，白糖5斤同置罐中，封口，放锅中滚水煮至3小时，止火待凉，置阴地3日出火毒，5日后即可去药用酒。
临床应用：None
各家论述：None
用药禁忌：虚而有热者忌服。
附注：None
