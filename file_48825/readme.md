序号：48840
名称：禹余粮丸
组成：蛇黄（大者）3两（以新铁铫盛入，炭火中烧蛇黄与铫子一般通赤，用钳取铫子出，便倾蛇黄入酽醋2升中，候冷，取出研极细则止，即含石），禹余粮3两，真针砂5两（先以水淘净，控干，更以铫子炒干，入禹余粮一处，用米醋2升，就铫内煮醋干为度，却用铫并药入炭火中，烧通赤，倾药净，砖地上候冷，研无声即止）_x000D_
以3物为主，其次量人虚实，入下项药：羌活半两，木香（煨）半两，茯苓半两，川芎半两，牛膝（酒浸）半两，白豆蔻（炮）半两，土茴香（炒）半两，蓬术（炮）半两，桂心半两，干姜（炮）半两，青皮（去瓤）半两，京三棱（炮）半两，白蒺藜半两，附子（炮）半两，当归（酒浸1宿）半两（虚人老人全用半两，实壮人减之）。
出处：《三因》卷十四。
主治：水肿胀满，小便不利，上气喘促，腹有积块。
加减：None
功效：逐阴固阳，扶危正命。暖水脏。
用法用量：食前温酒、白汤送下30-50丸，每日3服。兼以温和调补气血药助之。
制备方法：上为细末，拌极匀，以汤浸蒸饼，捩去水和药，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：切须忌盐；孕妇忌服。
附注：神仙万金丸（《百一》卷十二）、神授万金丹（《医方类聚》卷一二七引《澹寮》）、万金丹（《得效》卷九）、针砂丸、蛇含石丸（《兰台轨范》卷五）。
