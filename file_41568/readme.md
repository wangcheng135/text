序号：41583
名称：金粟丸
组成：胡黄连1分，犀角（镑）1分，丁香1分，木香1分，天竺黄1分，晚蚕蛾（微炒，为末）1分，牛黄（研）1分，丹砂（研）1分，雄黄（研）1分，龙脑（研）1钱，麝香（研）1钱，粉霜（研）1钱，蟾酥1钱。
出处：《圣济总录》卷一六九。
主治：小儿惊热涎盛，风虚吐逆，疳气黄瘦，吐泻后生风。
加减：None
功效：None
用法用量：每服1丸，温水化下。如惊风搐搦，先用1丸，温水化，灌在鼻内，随搐左右，良久以嚏为效，后用温水化下3-5丸；若吐逆不止，以倒流水化下2-3丸。
制备方法：上为末，再同研匀，用牛胆汁化蟾酥为丸，如黄米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
