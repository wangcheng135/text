序号：36
名称：五味子汤
组成：五味子2两，桔梗2两，紫菀2两，甘草2两，续服2两，地黄5两，桑根白皮5两，竹茹3两，赤小豆1升。
出处：《千金》卷十八。
主治：唾中有脓血，牵胸胁痛。咳嗽，皮肤干燥。
加减：None
功效：None
用法用量：上(口父)咀。以水9升，煮取2升7合，分3次服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：此火乘于肺，所以唾中有脓血，而胸胁牵痛乃血不荣筋之故，故用五味、紫菀滋培津血，桔梗、桑皮疏泄肺气，竹茹、小豆清膈上火，生地逐伤中血，川断通行经络，甘草平调中气。不独为火乘肺气之专药，并可为热伤肺痿之神丹。
用药禁忌：None
附注：None
