序号：3653
名称：巴鲫膏
组成：巴豆仁5钱，白及（切）5钱，番木鳖（切）5钱，川乌（切）5钱，草乌（切）5钱，商陆（切片）10两，漏芦2两，闹羊花2两，全归（切）2两，穿山甲（切）2两，元参（切）2两，虾蟆皮干（须新取收干）2两，蓖麻仁3两，白蔹（切）3两，川大黄（切）3两，雄鼠矢3两，苍耳子4两，黄牛蹄甲（敲研）1两，猪蹄甲（敲研）1两，乌羊角1对（敲研），鲫鱼2尾（重12两以上者）。
出处：《鸡鸣录·外科》。
主治：一切痈疽疔毒，未成即消，已成即溃。
加减：None
功效：None
用法用量：用纸摊贴。
制备方法：上药入大广锅内，真麻油3斤8两，浸3日，熬至各药焦黑，滤滓再熬沸，入飞净血丹24两，以槐、柳条不住手搅，熬至滴水成珠，息火待冷，再入上肉桂心5钱、乳香4钱、没药4钱、上芸香(各去油)4钱、上轻粉各4钱(此5味并研细徐徐掺入)，以铜箸搅匀，待凝冷，覆地上10余日，拔尽火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
