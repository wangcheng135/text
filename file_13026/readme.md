序号：13033
名称：夺命丹
组成：乳香（研）1分，琥珀（研）1分，天南星1分，防风1分，白僵蚕（洗，炒）1分，麝香肉（别研）1分，茯神1分，酸枣仁（去皮，炒）1两，远志（去心）1两，芸苔子（炒）半钱，蝉壳（洗净）4钱，全蝎（炒）半两，天麻（酒浸）8钱，白附子3钱，天浆子21个，蜈蚣2条（炙），木鳖子肉2钱（研）。
出处：《幼幼新书》卷九引易忠信方。
主治：小儿急慢惊痫及一切风证。
加减：如急惊盛，加龙脑少许同磨；如慢惊，即加附子少许同磨化下。
功效：None
用法用量：每服量大小加减1-2丸，金银薄荷汤磨化下。
制备方法：上为细末，水煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
