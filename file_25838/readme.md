序号：25846
名称：收黑虎汤
组成：玄参1斤，柴胡3钱，生甘草1两。
出处：《辨证录》卷十三。
主治：淫欲无度，又加之气恼忧郁，火乘其有隙之处，蕴藏结毒之无名肿毒，未破者或已破者。
加减：None
功效：补阴散郁。
用法用量：煎汤10碗为主，倘生于头面，加川芎2两，附子2钱，再煎汁，取3碗，分作3日服完。倘生于身中前后、左右，加当归2两，甘菊花1两，附子3分，亦如前煎服。倘生于手足四肢，加白术2两，附子5分，获苓1两，亦如前煎服。
制备方法：None
临床应用：None
各家论述：玄参最善退浮游之火，得甘草之助，能解其迅速之威，得柴胡之辅，能舒其抑郁之气，且又有各引经之味，引至结毒之处，大为祛除。妙在用至一斤，则力量更大。又妙是补中去散，则解阴毒而不伤阴气，所以奏功更神。
用药禁忌：None
附注：None
