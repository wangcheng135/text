序号：31472
名称：杜仲丸
组成：杜仲（去粗皮，锉，炒）3分，牛膝（去苗，酒浸，切，焙）1两，萆薢（微炒）1两半，酸枣仁（炒）1两，当归（切，焙）3分，防风（去叉）1两，丹参（微炙）3分，赤芍药3分，桂（去粗皮）半两，肉苁蓉（酒浸，切，焙）1两1分，石斛（去根，锉）3分，附子（炮裂，去皮脐）半两，郁李仁（汤浸去皮尖，炒）3分，槟榔（煨）1两。
出处：《圣济总录》卷五。
主治：肾中风，腰脚不随，骨节痠痛，筋脉拘急，行履艰难，两胁牵痛。
加减：None
功效：None
用法用量：每服30丸，空腹用温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
