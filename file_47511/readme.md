序号：47526
名称：神丹
组成：刺猬皮1个（铁器中炒，焦黑为度），皂角刺（烧存性）半两，硫黄（研）半两，猪牙皂角（去黑皮，蜜涂，炙）半两，白矾半两，枳壳（炒）半两，黄耆（蜜炙）半两，附子（去皮）半两，白鸡冠花子1两。
出处：《得效》卷七。
主治：肠风，痔漏。
加减：若服药觉热，加白鸡冠花子1两半或2两，更加3-5丸服之，脏腑自调。
功效：消诸痔。
用法用量：每服7-10丸，空心、食前以温酒送下；不饮酒者以米饮送下；久年痔漏，可服至30-40丸；外痔，用药10丸与朱砂细研，蜜调涂。
制备方法：上为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
