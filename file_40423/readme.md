序号：40433
名称：虎杖散
组成：杜牛膝5钱，丹皮3钱，归横须3钱，降香末3钱，琥珀（同研末）6分，两头尖3钱，桃仁泥3钱，麝香（同研冲）5厘。
出处：《吴鞠通医案》。
主治：由房事不遂而成小便淋浊，茎管痛不可忍。
加减：None
功效：None
用法用量：水煮3杯，分3次服。
制备方法：None
临床应用：淋痛：普，三十八岁，小便淋浊，茎管痛不可忍，自用五苓、八正、萆薢分清饮等淡渗，愈利愈痛，细询病情，由房事不遂而成。余曰：溺管与精管异途，此症当通精管为是。用虎杖散，现无虎杖，以杜牛膝代之。服药后，一贴而痛减，五帖而痛止，七帖浊净，后以补奇经而愈。
各家论述：None
用药禁忌：None
附注：None
