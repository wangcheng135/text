序号：73241
名称：斑龙二至丸
组成：鹿角（锯成段，长流水浸7日，入砂锅内，用桑柴火煮7日夜，取出，外去粗皮，内去血穰，研为细末）1斤，麋角（制法同前，净末）1斤，生地（酒浸1宿，晒干，为末）4两，黄柏（去皮，切粗片，酒炒老黄色，为末）半斤，熟地（酒浸1宿，晒干，为末）4两，天门冬（酒浸，去心，晒干）4两（为末），知母（去皮，盐、酒炒老黄色，为末）半斤，麦门冬（酒浸，去心，晒干）4两，当归（酒洗，晒干，为末）2两，白茯苓（去皮，为末，用水淘去筋膜）2两，何首乌（去皮，人乳拌匀，9蒸9晒，为末）2两（勿犯铁器）。
出处：《万氏家抄方》卷四。
主治：诸虚。精血欠充，高龄无子。
加减：None
功效：补养。补血，补阳。
用法用量：每服50丸，空心时黄酒送下，或盐汤亦可。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：高龄无子：少傅颖阳许相公，年五十八岁，如夫人年近三旬，从来十二年不孕。相公欲其有子，命宿诊视。六脉和缓，两尺大而有力（凡妇人两尺大而有力，皆有子）。告曰：此宜子之象也。尝诊相公，脉沉而缓，知精血欠充实耳。宜服大补精血药。市得麋鹿二角，煎胶，制斑龙二至丸一料。服未周年而孕，次年生公子。
各家论述：None
用药禁忌：None
附注：方中鹿角、麋角，《扶寿精方》作“鹿角霜”、“鹿角胶”。
