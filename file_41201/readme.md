序号：41216
名称：金刚活血酒
组成：金刚莲300g，甘草300g，当归250g，五加皮250g，川芎180g，紫草茸180g，三棱90g，莪术90g，苏木90g，枳壳90g，青皮90g，见血清120g，红毛藤根120g，桂枝150g，大血藤60g，白酒30公斤。
出处：《古今名方》引《张天乐十二秘方制药经验》。
主治：扭、挫伤和风湿痛患者。
加减：None
功效：通经活络，祛风止痛。
用法用量：成人每晚睡前10-20毫升，内服兼外搽。
制备方法：以上各药为饮片，均放入有盖大缸中，再入白酒20公斤，密封浸渍1周后，用纱布滤过，另放缸中密闭贮存。其残滓再用白酒10kg浸渍1周，如法用纱布滤过，将上两次浸渍溶液混合待用。此液为棕红色澄明液体，气微香，味甜可口。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
