序号：57633
名称：大通真丸
组成：蚕纸10张（烧灰），防风1两（去芦头），白芍药3分，桔梗1两（去芦头），石膏1两（细研，水飞过），白芷3分，当归1两（锉碎，微炒），干姜半两（炮裂，锉），附子1两（炮裂，去皮脐），芎半两，藁本半两，泽兰1两，白芜荑半两，川椒1两（去目及闭口者，微炒出汗），食茱萸3分，柏子仁1两（微炒），白薇半两，白术半两，苍术半两（锉碎，微炒），蝉壳半两（微炒），人参1两（去芦头），甘草半两（炙微赤，锉），厚朴3分（去粗皮，涂生姜汁，炙令香熟）。
出处：《圣惠》卷七十。
主治：妇人血风劳气，经络不调，腹内时痛，面色萎黄，四肢羸弱，心神昏闷，不欲饮食，及产后余疾。
加减：None
功效：None
用法用量：每服2丸，食前以温酒调下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
