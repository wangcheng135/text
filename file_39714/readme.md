序号：39724
名称：经济丹
组成：白茯苓1两，白茯神1两，白芍药1两，远志（去心）1两，乳香半两（别研），当归1两（酒浸），酸枣仁（去壳，炒），人参1两，没药1两（研），朱砂（别研）半两，石菖蒲1两（真者）。
出处：《女科百问》卷上。
主治：妇人血气不足，荣卫俱虚，心气不定，夜卧惊怖，梦寐不祥，心虚自汗，乏力倦怠，饮食减少，咳嗽痰实。
加减：None
功效：常服补心养血，安神定志，令人血壮气实。
用法用量：每服30丸，加至50丸，食前枣汤、参汤任下，饮酒亦可。
制备方法：上为末，炼蜜为丸，如梧桐子大，将朱砂为衣。只用枣仁丸亦得。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
