序号：68143
名称：黄耆饮
组成：黄耆4两，芍药4两，芎4两，甘草4两，生姜1斤。
出处：《普济方》卷二三一。
主治：虚劳崩中，吐血上气，短气欲绝，面黑如漆。
加减：酒客劳热，发痔下血，其谷道热，去生姜，用地黄代之。
功效：None
用法用量：上（口父）咀。以酒5升，浸1宿，明旦更以水5升，煮取4升，分4服，日3夜1。凡进2-3剂。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：凡夏月不得隔宿浸药；忌菘菜。
附注：黄耆汤（《医方类聚》卷八十六引《御医撮要》）。《医方类聚》引《御医撮要》本方用法：上为散。每服三钱，以水一盏，加生姜一块子（分拍碎），同煎至七分，去滓温服。
