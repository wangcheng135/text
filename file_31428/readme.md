序号：31437
名称：豆淋酒煎附子丸
组成：黑豆1升（炒令熟，入酒5升煎20-30沸，滤去滓，煎令稠），附（侧）子2分（炮裂，去皮脐），石南半两，牛膝半两（去苗），防风半两（去芦头），石斛半两（去根，锉），肉桂半两（去皱皮），萆薢3两（锉），麻黄半两（去根节），羌活半两，海桐皮半两（锉），赤茯苓半两，茵芋半两，独活半两，天麻半两，当归半两（锉，微炒），乌蛇1两（酒浸去皮骨，炙令微黄）。
出处：《圣惠》卷三。
主治：肝脏风毒，流注脚膝，筋脉疼痛，四肢缓弱无力。
加减：None
功效：None
用法用量：每服30丸，食前温酒送下。
制备方法：上为细散，以黑豆煎，和捣为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌猪肉、毒鱼等。
附注：石南丸（《圣济总录》卷四十二）。
