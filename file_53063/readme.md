序号：53078
名称：鬼哭饮子
组成：阿魏1分（使童便磨一处），东引桃枝（小者，捶碎）1大握，甘草（大者）3寸（捶碎），青蒿1大握（如用其子，只用1两半），槟榔1两（为末），葱白2寸（连根）。
出处：《博济》卷一。
主治：一切劳。
加减：None
功效：None
用法用量：上用童便2升，浸桃枝、甘草、青蒿、葱白1宿，来日五更初煎取6合，去滓，然后入阿魏更煎2沸，分为2服。每服临吃时入槟榔末半两同煎。如觉心头恶心，必吐，吐后更进第2服。如服前1服后心头安稳，即须进第2服，必然通转，当下，必见恶物。服时仍不得令人与患者面对，恐恶虫飞入人口鼻内。患者春吃2服，秋吃3服，每年5服，劳虫并尽，即去病根。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌猫、狗见。
附注：None
