序号：83614
名称：乌骨鸡丸
组成：人参（去芦）5钱，当归（酒洗）1两，熟地（姜汁浸，焙）1两，白芍（酒炒）1两，白茯苓（去皮）1两，香附（童便浸，炒）1两，川芎7钱，陈皮7钱，秦艽7钱，玄胡索7钱，贝母（去心）7钱，牡丹皮7钱，甘草5钱。
出处：《回春》卷六。
主治：妇人虚弱，咳嗽吐痰，或骨蒸劳热，或赤带下，或经水不调，形体瘦倦无力，或口干舌燥。
加减：None
功效：None
用法用量：每服50丸，空心清米汤送下。
制备方法：上俱锉成饮片听用。另用黄耆为末，拌饭喂乌鸡，喂至肌肥，眼生眵，缢死，燥去毛，破开取出肠胃，好酒洗净，入前药饮片在鸡肚内线缝住，用酒、醋等分，煮鸡烂如泥，捞起焙干或晒干，为细末，将鸡汁打面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
