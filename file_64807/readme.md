序号：64823
名称：续命汤
组成：竹沥1升2合，生地黄（汁）1升，龙齿4两，生姜4两，防风4两，麻黄4两，防己3两，附子3分，石膏7两，桂心2两。
出处：《千金》卷十四引徐嗣伯方。
主治：风眩。发则烦闷无知，口沫出，四体角弓，目反上，口噤不得言。
加减：有气，加附子1两，紫苏子5合，橘皮半两。
功效：None
用法用量：上(口父)咀。以水1斗，煮取3升，分3服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：此续命汤治风眩烦闷，但取麻黄、防风、桂心、附子、石膏、生姜六味开拓表里阴阳，调适经腑寒热，乃加竹沥以治经络四肢膜外之痰，地黄以治周身脏腑痹着之血，龙齿以治惊痫诸痉、癫疾狂走，防己以治中风挛急、风热诸癫。
用药禁忌：None
附注：None
