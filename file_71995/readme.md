序号：72011
名称：小菟丝子丸
组成：石莲肉2两，菟丝子（酒浸，研）5两，白茯苓（焙）1两，山药2两（内7钱半打糊）。
出处：《局方》卷五（吴直阁增诸家名方）。
主治：肾气虚损，五劳七伤，少腹拘急，四肢痠疼，面色黧黑，唇口干燥，目暗耳鸣，心忪气短，夜梦惊恐，精神困倦，喜怒无常，悲忧不乐，饮食无味，举动乏力，心腹胀满，脚膝痿缓，小便滑数，房室不举，股内湿痒，水道涩痛，小便出血，时有遗沥。色疸。赤白浊。
加减：None
功效：填骨髓，续绝伤，补五脏，去万病，明视听，益颜色，轻身延年，聪耳明目。
用法用量：每服50丸，空心温酒或盐汤送下；如脚无力，木瓜汤送下，晚食前再服。
制备方法：上为细末，用山药糊搜和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：菟丝子丸（《摄生众妙方》卷二）、菟丝丸（《郑氏家传女科万金方》卷四）。
