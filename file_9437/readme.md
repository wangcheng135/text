序号：9437
名称：水银膏
组成：黄柏1钱，黄连1钱，川大黄5分（上3味另研），雄黄3分，胆矾3分，青黛3分，儿茶铜青3分，轻粉4分，枯矾4分，珍珠1分半（生用，另研），大风子（去油，取霜，黑者勿用）5分，冰片1分半（另研），人言人壮者7厘，弱者5厘，中者6厘（上14味，为极细末，分作3份，约1钱8分），番打麻（另为末）疮重而人壮能食者每份用5分，人弱不能起者每份用3分，中者4分（以末入前药各份内研匀），水银人壮者每份用1两或8-9钱，中者五6钱，卧床不起而极弱者只可用3钱（不可再多）。
出处：《痘疹传心录》卷十五。
主治：杨梅风毒溃烂，危恶多年不愈者；或咽喉溃烂，或遍身牛皮疮癣及久烂臁疮。
加减：None
功效：None
用法用量：用此药擦手足四腕动脉处，每药1分，务分擦3日，每日早晚各擦1次，每次以600-700次为度，擦完用布包之，擦药时，凡周身略有破伤处，俱用无麝膏药贴之，膏药须厚摊，每2日1换，换时不可经风，常须避帐幔中，冬月须用厚被暖炕，他时亦须常暖，南方多用被褥盖垫，擦至7日，毒必从齿缝中发出，口臭吐涎，若口齿破烂出血，但用甘草、蜂房煎汤候冷漱解，不可咽下，轻者只以花椒汤漱之亦可，擦之必皮破，不可畏疼而少擦。
制备方法：上先将麻汞并前药各1分，俱入盏内，再入真麻油少许，用手指研开，务使汞药混为一家，渐次增油，久研以不见汞星为度，大约如稀糊可矣。
临床应用：None
各家论述：None
用药禁忌：忌盐十余日，多更好；并忌鱼腥、生冷、动气发风等物一个月，尤忌房事；外如牛肉、烧酒、团鱼之类须忌二三年；惟荞麦面、羊肉则终身忌之。
附注：秘传水银膏（《景岳全书》卷五十一）。
