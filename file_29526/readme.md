序号：29534
名称：陈曲丸
组成：陈曲（炒黄）8两，木香8两，厚朴（去粗皮，生姜汁炙）8两，甘草8两，槟榔8两，青橘皮（去白）8两，白术8两，枳壳（麸炒，去瓤）8两，京三棱（炮）8两，干姜（炮）12两，桂（去粗皮）12两。
出处：《圣济总录》卷五十四。
主治：中寒胃虚，饮食迟化，气不升降，呕逆恶心，留饮寒痰，癖结动气，胁下逆满，有时而痛，按之有形，或按有声，膈脘虚痞，食物多伤，噫气醋臭，心腹常疼，霍乱吐逆，烦闷不安。
加减：None
功效：None
用法用量：每服5-7丸，温米饮送下，不拘时候。
制备方法：上为末，水煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
