序号：25748
名称：托里散
组成：大黄3钱，牡蛎3钱，瓜萎根3钱，皂角针3钱，朴消3钱，连翘3钱，当归1两，金银花1两，赤芍2钱，黄芩2钱。
出处：《玉机微义》卷十五。
主治：一切恶疮发背，疔疽，便毒始发，脉洪弦实数，肿甚欲作脓者。
加减：None
功效：None
用法用量：每服半两，水、酒各半煎服。3服消尽。
制备方法：上为粗末。
临床应用：None
各家论述：《医方集解》：此足阳明、厥阴药也。金银花清热解毒，疮痈主药；当归、赤芍调营血；大黄、芒消荡胃热；黄芩清肺火；牡蛎软坚痰；连翘、花粉散结排脓；角刺锋锐，直达病所而溃散之也。
用药禁忌：None
附注：托里护心散（《明医指掌》卷八）。
