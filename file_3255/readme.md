序号：3255
名称：九仙山何处士黑神丸
组成：巴豆（新好者）1两（轻捶去皮，以长流水约两碗，浸1宿，然后更煮30-50沸后冷，粗去心膜，以布子拭去水，然后研如膏，用厚纸10数张裹，以重物压去油用），豆豉3两（须是新者好的，软者为妙。不得用盐煮干，与上巴豆膏同研细匀），京三棱半两（生用），大戟半两（生用，不去皮。其里面白如粉白者为妙），五灵脂1分（黑色者为上），杏仁半两（烧过后研，入药再研之）。
出处：《普济方》卷二五六引《博济》。
主治：瘟疫时气及中恶，心气闷绝，面青手冷，痢疾，疳疾，水肿胀满。
加减：None
功效：None
用法用量：上为极细末，入巴豆、豉膏研匀，后入杏仁，更研令极细，别研入飞罗面半匙，以井花水调如糊，渐次拌药搜和得所，入臼中捣2000-3000杵为丸，如绿豆大，晒干，收入瓷瓶内合顿，或微微火焙亦得，具汤使疗如下：瘟疫时气，阴阳二毒，伤寒及头疼壮热，每服10丸或15丸，用葱白连须一茎，好茶1盏，泼葱茶内盖定片时吞下，以衣被盖，或吐或泻，或只汗出便愈；如未吐未泻未出汗，更吃热茶1盏便愈，但避风2-3日将息；一切风，薄荷茶下5丸或7丸；伤寒腹满，姜汤送下7丸；一切气，橘皮、生姜汤送下5至10丸；肺气喘急，杏仁汤送下5丸；心痛，醋汤送下10丸；小便不通，葱汤送下5丸；血刺、血疰、血癥，煎当归酒送下5丸至7丸；呀呷、上气、杏仁汤送下5丸；淋疾，滑石汤送下7丸；赤眼，山栀汤送下5丸，食后服；中恶心气闷绝，面青手冷，桃仁汤送下7丸；水土痧气，虚胀满急，大小便赤涩，橘皮汤送下10丸，忌甘草；眼昏，葛粉汤送下5丸；疰刺气，桃仁汤送下7丸；水泻，新汲水送下3丸，忌热汤；赤痢，山栀子7粒、百草霜同煎送下7丸；赤白痢，山栀子、干姜汤送下7丸；疳疾蛔虫，粥饮送下5丸；食伤，茴香汤送下10丸；酒伤，嚼下10丸；奔豚气绞痛，茴香汤送下7丸；水气肿满，煎桑皮汤送下10丸；肾泻、送下5丸；气疾或上引攻心，七枣汤送下5丸至7丸；小儿五疳八痢，腹胀气恶，茴香汤送下5丸；四时宣转，以五更初温茶送下7丸，须臾热茶咽之，如转泻加多，以冷浆水服之即止。凡有诸般疾状，只用热茶酒任下5-7丸，无不瘥验。约人脏腑虚实，加减丸数。如修治巴豆子，先以黄连水净手。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：黑神丸（《活人书》卷十八）、九仙丹（《医方类聚》卷八十九引《王氏集验方》）。
