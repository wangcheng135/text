序号：80612
名称：燥湿化痰丸
组成：羌活5钱，独活5钱，防风5钱，防已5钱，川乌5钱，全蝎5钱，南星5钱，半夏5钱，僵蚕（炒）5钱，天麻5钱，陈皮（去白）5钱，猪牙皂角（瓦炒）5钱，当归（酒洗）5钱，生地黄5钱，木香5钱，沉香5钱，白术（炒）2两，苍术（米泔水浸）2两，杜仲（姜汁炒）2两，巴戟天（炒）1两，薏苡仁1两，牛膝（酒洗）1两，川芎1两，破故纸（炒）1两。
出处：《医统》卷十二。
主治：历节风。湿痰壅滞，昼夜疼痛无休者。
加减：None
功效：None
用法用量：每服100丸，空心、食前酒送下，1日1次，食干物压之。
制备方法：上为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：愈风燥湿化痰丸（《景岳全书》卷五十四）。
