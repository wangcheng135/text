序号：46355
名称：活命金丹
组成：川芎1两，甘草1两，板兰根1两，葛根1两，龙脑2钱，麝香2钱（研），牛黄（研）5分，生犀3钱，桂3钱，珠子粉半两，川大黄2两半，甜消1两，辰砂4钱（一半为衣），青黛3钱，薄荷5钱。
出处：《医学启源》卷中。
主治：风中脏，不语，半身不遂，肢节顽痹，痰涎上潮，咽嗌不利，饮食不下，牙关紧禁。及解一切药毒，发热腹胀，大小便不利，胸膈痞满，上实下虚，气闭面赤，汗后余热不退，劳病。
加减：None
功效：None
用法用量：如风毒，茶清送下；解毒药，新冷水化下；余热劳病及小儿惊热，薄荷汤化下。
制备方法：上为细末，炼蜜同水浸蒸饼糊为剂，每1两作10丸，别入朱砂为衣，就湿以真金箔40叶为衣，瓷器内收贮，多年不坏。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中川芎、葛根，《御药院方》作贯众、干姜。
