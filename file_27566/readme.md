序号：27574
名称：补泄防劳丸
组成：木香（不见火）12分，槟榔12分，大黄（生）12分，大麻仁12分，枳壳（麸炒，去瓤）1分，桂心（去粗皮，不见火）1分，萆薢1分，牛膝10分（酒浸），诃子（去核）4分，山茱萸（去核）4分，芎4分，独活4分，前胡4分，羚羊角屑4分，附子（炮去皮）。
出处：《魏氏家藏方》卷八。
主治：干湿脚气，浮肿，筋脉疼痛，行步不得，气冲心腹，不思饮食，元气弱不敢服泻药者。
加减：None
功效：开胸膈，调脾胃，除风毒。
用法用量：每服20丸，空心温酒送下，1日1次。微泄为度。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
