序号：24111
名称：如意丸
组成：威灵仙（去苗土）半两（为末，用好醋半盏浸1宿），附子（生，去皮脐）半两（为末，用好醋半盏浸1宿），硇砂（细研）1分，巴豆21粒（去皮心膜，出油。2味同研，用酒半升、醋半升同煎，与前2味同熬成膏），蓬莪术（煨，锉）半两，木香半两，青橘皮（汤浸，去白，炒）1两，大黄（锉，炒）3分，陈曲（炒）半两，丁香1分。
出处：《圣济总录》卷七十二。
主治：食后心膈妨闷。
加减：None
功效：消积化气，温胃思食。
用法用量：每服5-7丸，生姜汤送下。
制备方法：上药，将后六味为末，以前四味膏和，更别熬醋少许，研墨汁同为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
