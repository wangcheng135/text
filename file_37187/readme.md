序号：37197
名称：定崩四物汤
组成：生地5钱（炒松），白芷1钱半（炒黑），白芍1钱半（醋炒），川芎1钱，当归3钱（醋炒），蒲黄3钱（炒炭），阿胶3钱（血余炭炒），小蓟根3钱。
出处：《医略六书》卷三十。
主治：产后风湿袭于冲任，不能去宿生新，血崩如豆汁，腹胁阵痛，脉浮涩微数者。
加减：None
功效：None
用法用量：水煎，去滓温服。
制备方法：None
临床应用：None
各家论述：方中生地凉血止血，炒松能去阴中之湿；白芷祛风散湿，炒黑亦止崩漏之血；白芍敛阴血以固经；川芎入血海以升阳；当归引血归经；蒲黄散瘀止血；小蓟凉血散瘀；阿胶补阴益血，血余炭炒，以止血定崩也。水煎温服，俾风湿外解，则宿去新生而冲任完复，经色如常。
用药禁忌：None
附注：None
