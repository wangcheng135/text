序号：36287
名称：和中丸
组成：台白术2两，云茯苓2两，生甘草5钱，奎白芍2两，制首乌2两，银柴胡2两，知母2两，淮山药2两，地骨皮2两，使君子2两，生香附2两，木香1两5钱，川朴2两，陈广皮2两，油青皮2两，南楂肉2两，花粉2两，枳实2两，莱菔子2两，缩砂仁1两5钱，淡黄芩4两，柴胡4两5钱，木通2两，车前子2两，台乌药2两，泽泻2两。
出处：《青囊秘传》。
主治：None
加减：夏月，加香薷2两、六一散2两。
功效：和中理气，消痰祛湿。
用法用量：老年及小儿服1丸，多则2丸，饭后服。
制备方法：上药各炒，共为细末，用瓷瓶收贮，每于用时即将饴糖打糊为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
