序号：58629
名称：秘传杖疮膏
组成：香油4两（真者佳，将穿山甲、柏枝先入油中煎数沸，去二件滓，乘热将薄棉滤净油，复入锅中煎沸，以次下药。冬月用油5两），穿山甲1片，柏枝1根（以上二件取油煎汁，不用滓，取法见前），槐枝1茎（须另报开小条，不用大树上者，入药油用此频搅），府丹（即飞丹，净水飞去漂脚，取细末）1两（作2次入油），水花珠（净水飞去漂脚，晒干，取细末）2钱，血竭3钱，没药3钱，乳香3钱，孩儿茶3钱（捶碎和匀，共入铜锅，炭火上炒沸过，为细末），新珍珠1钱，新红象牙（各面包，烧存性，取细末，油旧者不用）1钱，面粉（炭火上烧黄）1钱，人指甲（炒黄）3分，三七（晒干，取细末）3分，石乳（铜锅内炒过，取细末）3分，黄连（细末）3分，黄芩（细末）3分，海螵蛸5分（细末），半夏（大者）10枚（为细末。以上俱用极细筛筛过，和匀分作5分，留起1分，看膏药老嫩加减，只用4分作4次下，下法如下），樟冰（细末）4钱，黄蜡2钱，冰片1分，麝香3分，阿魏（成块者）5分（以上四件待诸药俱下尽，临起锅时方下，搅极匀取出阿魏滓）。
出处：《准绳·疡医》卷六。
主治：打伤，金疮及无名肿毒、臁疮。
加减：None
功效：None
用法用量：贴时先将莱菔汁、桑叶煎汤，露中露过1宿，用以洗患处，方用贴之。既贴后每日洗1遍，不要换膏药，至2-3日后血散风去，方换收口黑膏药（即万应膏）。
制备方法：先将细末药分5分，其4分以次下锅如下，其1分留看药厚薄以为增减，如4分已下尽药尚薄，亦将此分渐下，如正好，留此1分，待点膏药时掺在患处尤妙。煎法用上好香油4两，入铜锅中炭火煎沸，沸时入柏枝1茎，穿山甲1片在内，煎数沸去二药滓，将薄棉纸乘热滤净油，揩净锅，复入油于锅中煎沸，下府丹5钱，用槐条急搅不住手，至成膏方止，候6-7煎后，用清水漱净口喷清水少许于锅中即取锅。一起锅时于前4分中细末入药将1分渐渐逐一挑下，急搅如前，此分药尽约匀和了，将槐条蘸药滴水且未要成珠，复置锅火上急搅，候沸起锅。二起锅复将前末药1分渐下锅中急搅如前，约匀和滴水要成珠，复置锅炭火上急搅，候沸起锅。三起锅渐下药搅如前，约匀和，将药滴水虽成珠尚要粘手，复置锅火上如前。四起锅渐下药如前急搅，约匀和将药滴成珠，珠要将至不粘手了，复置锅炭火上，候沸起锅。五起锅即下黄蜡2钱，府丹5钱急搅如前，将。药滴水成珠，要须不粘手，又不可太老了，如尚粘手，将前留下1分末药渐下以不粘手为度，如不粘手了，即下水花珠2钱，次下樟冰末4钱，急搅，方下麝香3分，阿魏5分，冰片1分，急搅不住手，量药已均和了，撩阿魏滓丢之，以药入瓷器内，浸冷水中片时，候凝，将药寻露天天阳净地，掘坎将瓷器倒覆于坎中，仍以土覆好，候7日后方起。藏法用油纸及箸包好瓶口以防泄气。摊膏药时用汤中煎过净油单纸摊上药，不用火烘，只用热汤入器中，将油纸放器上，以药放上摊开，又不用太厚，须于纸上照得见为妙，如以绢摊，用汤炖烊药摊上。
临床应用：None
各家论述：None
用药禁忌：跌伤及别样疮忌贴。
附注：None
