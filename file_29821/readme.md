序号：29829
名称：十全济阴丸
组成：当归身（酒洗）4两，熟地黄4两，香附子（童便煮）4两，干山药2两5钱，白术2两5钱，枸杞子2两，人参2两，蕲艾叶（去梗筋）2两（同香附用陈醋、老酒煮一时，捣烂，焙干），川芎1两5钱，白芍药1两5钱，牡丹皮1两5钱，紫石英（火煅淬）1两5钱，泽兰1两，紫河车一具（在净水内洗去秽血，用银针挑去紫筋）。
出处：《济阴纲目》卷六。
主治：月经不调，子宫寒冷不孕。
加减：None
功效：调经养血，顺气开菀。
用法用量：每服50丸，渐加至80-90丸，空心淡盐汤送下。随用早饭，使药下行。
制备方法：同河车入砂锅内，用陈老酒3碗，陈米醋1碗，清白童便1碗，米泔水数碗和匀，倾入锅内，浮于药寸许，如尚少，再加米泔，以锅盖盖密，勿令透气，桑柴火慢煮，以河车融化，汁干为度，同药俱取出，在石臼内捣极烂，捻作饼子，日晒夜露3昼夜，宜在月满之时，以受日精月华，仍焙干为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：此方以当归身养血和气为君，入手少阴经，以主心血也；入足太阴经，以脾裹血也；入足厥阴经，以肝藏血也，熟地黄补肾中元气，生心血，与芍药同用，又生肝血；川芎乃血中之气药，下行血海，通经导气为臣。人参通经活血，助熟地黄以补下元；白术利腰脐间血，与人参同用，补益脾气；香附疏气散郁，佐泽兰能生新血，而和平气体；牡丹皮养新血去坏血，固真气行结气；山药能强阴补虚；枸杞子补肾水，而止下血腰疼为佐；紫石英补心气，散心中结气，填补下焦；艾叶助香附和百脉，温子宫，兼行血药而平其寒；炙甘草通经脉血气而和诸药，且缓肝经之急为使。
用药禁忌：忌食生萝卜。
