序号：51079
名称：草神丹
组成：川附子（制）5两，吴茱萸（泡）2两，肉桂2两，琥珀5钱（用柏子煮过另研），辰砂5钱（另研），麝香2钱（另研）。
出处：《扁鹊心书·神方》。
主治：阴毒伤寒，阴疽痔漏，水肿臌胀，中风半身不遂，脾泄暴注久痢，黄黑疽，虚劳发热，咳嗽咯血，两胁连心痛，胸膈痞闷，胁中如流水声；童子骨蒸，小儿急慢惊风，痘疹变黑缩陷；气厥卒仆；双目内障；吞酸逆气，痞积血块，大小便不禁；奔豚疝气；附骨疽，两足少力，虚汗不止；男子遗精、梦泄，砂石淋，溺血；妇人血崩血淋；暑月伤食、腹痛，呕吐痰涎。
加减：None
功效：大补脾肾。
用法用量：每服50丸，米饮送下。小儿每服10丸。
制备方法：先将前3味为细末，后入琥珀、辰砂、麝香3味，共研极匀，蒸饼为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
