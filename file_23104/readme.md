序号：23112
名称：芎辛汤
组成：附子（生，去皮脐）、乌头（生，去皮尖）、天南星、干姜、甘草（炙）、川芎、细辛各等分。
出处：《三因》卷十五。
主治：伤风寒生冷及气虚、痰厥，头痛如破，兼眩晕欲倒，呕吐不定；一切头疼、痰厥、饮厥、肾厥、气厥等证，偏正头疼难忍者。
加减：None
功效：扶阳散寒。
用法用量：每服4大钱，水2盏，加生姜5片，茶芽少许，煎7分，去滓，食后服。
制备方法：上为散。
临床应用：None
各家论述：《医略六书》：真阳内虚，寒邪得以深入，元气不能外敷上奉，故厥逆而头痛眩晕焉。川乌、生附扶真阳以御寒，南星、细辛通关窍以豁痰，干姜温中散寒，生姜散寒快膈，川芎活经中之血，炙草缓中州之气，芽茶化热以清头目也。此扶阳散寒之剂，为阳虚寒厥痛晕之专方。
用药禁忌：发热者不宜服。
附注：大芎辛汤（《得效》卷十）。本方方名，《医学正传》引作“芎辛散”。
