名称：神仙万应膏
组成：穿山甲3钱，当归半两，大黄半两，黄芩3钱，黄皮3钱，玄参3钱，川芎半两，木鳖子3钱，知母3钱，贝母3钱，白薇3钱，白蔹3钱，羌活3钱，独活3钱，降真3钱，苏木3钱，柴胡3钱，芷梢3钱，赤芍药3钱，苦杖3钱，防风3钱，桔梗3钱，蜀葵花3钱，瓜蒌3钱，椿皮3钱，槐枝3钱，柳枝3钱，竹枝3钱，野紫苏叶3钱，天南星3钱，续断3钱，荆芥3钱，黄耆3钱，苦参3钱，草乌3钱，商陆3钱，甘草3钱，官桂半两，薄荷3钱，车前草3钱，桃仁3钱，杏仁3钱，槐花3钱，苍耳3钱，芒消3钱，地榆3钱，刘寄奴3钱，葛根3钱，通草3钱，泽兰3钱，桑桂3钱，黄栀子3钱。
出处：《普济方》卷三一四。
主治：一切疮肿伤折。
加减：None
功效：None
用法用量：用好香油2斤或3斤，除薄荷、车前草外，诸药俱入油中，煎至焦黄色，临熟下薄荷、车前草，熬1-2沸，滤去滓；以油入铫内煎，候油7分热下黄丹1斤，用槐枝条不住手搅成膏，于水内滴试，以不散为度；然后入乳香、没药各1两搅匀，倾入钵，候凝定，却入水中出火毒旬日。
制备方法：上锉。
