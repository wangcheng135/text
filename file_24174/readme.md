序号：24182
名称：安胎散
组成：编砂不以多少（熨斗内盛，慢火炒令热透，后去皮、取仁用）。
出处：《杨氏家藏方》卷十六。
主治：妊娠胎动下血，心腹疼痛，气逆呕吐及孕妇出痘动胎。
加减：None
功效：None
用法用量：每服2钱，热酒调下；不饮酒者，煎盐艾叶汤调下，食空服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：此药非八九个月内，不宜多用。
附注：安胎饮（《女科百问》卷下）、缩砂汤（《妇人良方》卷十二）、小安胎饮（《得效》卷十四）、独圣散（《景岳全书》卷六十一）、独圣安胎散（《种痘新书》卷十二）、独圣汤（《叶氏女科》卷二）。服此药后觉胎动处极热，即胎已安。
