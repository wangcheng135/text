序号：34769
名称：麦门冬汤
组成：麦门冬（去心）7分，桑白皮（蜜炒）7分，生地黄7分，紫菀茸5分，桔梗5分，淡竹叶5分，五味子3分，甘草3分，贝母6分，天门冬7分。
出处：《医学正传》卷二。
主治：诸病后，火热乘肺，咳唾有血，胸胁胀满，上气喘急，羸瘦，五心烦热，渴而烦闷。上焦热甚而声瘖者。
加减：None
功效：None
用法用量：加生姜3片，水1盏半，煎至1盏，温服。
制备方法：上细切，作1服。
临床应用：None
各家论述：《证因方论集要》：天冬、麦冬能清肺热，桑皮、紫菀能泻肺火，生地、贝母能润肺燥，五味能收肺气，淡竹叶功专清心，甘、桔除热利膈，火清而闭开矣。
用药禁忌：None
附注：方中贝母、天门冬用量原缺，据《景岳全书》补。
