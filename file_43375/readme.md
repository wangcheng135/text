序号：43390
名称：养胃进食丸
组成：人参（去芦头）1两，甘草（锉）1两，白术2两，白茯苓（去皮）2两，厚朴3两（去粗皮，生姜制，炒），陈皮（去白）1两半，神曲（炒）2两半，大麦糵（炒黄）1两半，苍术5两（去粗皮，泔浸）。
出处：《御药院方》卷三。
主治：脾胃虚弱，心腹胀满，面色萎黄，肌肉清瘦，怠惰嗜卧，全不思食。
加减：None
功效：滋养脾胃，进美饮食，消痰导气，去风寒暑湿冷邪气。
用法用量：每服30-50丸，食前用温生姜汤送下；或粥汤送下亦得。
制备方法：上为细末，水面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方改为汤剂，名“养胃进食汤”（见《类证治裁》卷六）。
