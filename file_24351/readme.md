序号：24359
名称：八风散
组成：藿香（去土）半斤，白芷1斤，前胡（去芦）1斤，黄耆（去芦）2斤，甘草（炙）2斤，人参（去芦）2斤，羌活（去芦）3斤，防风（去芦）3斤。
出处：《局方》卷一。
主治：风气上攻，头目昏眩，肢体拘急烦疼，或皮肤风疮痒痛，及治寒壅不调，鼻塞声重；疮疹既发，声音不出，形气俱病。
加减：None
功效：None
用法用量：每服2钱，水1中盏，入薄荷少许，同煎至7分，去滓，食后温服；或以腊茶清调1大钱亦得，小儿虚风，乳香、腊茶清调下半钱。
制备方法：上为细末。
临床应用：眩晕瘙痒：薛立斋治一人头目晕眩，皮肤瘙痒.搔破成疮，以八风散治之即愈。
各家论述：None
用药禁忌：None
附注：八风汤（《保婴撮要》卷二十）。
