序号：43996
名称：厚朴橘皮丸
组成：厚朴（去粗皮，用生姜制）3两，枳壳（麸炒，去瓤）1两2钱，干姜（火炮裂）1两2钱，良姜（锉）1两2钱，青皮（去白）8钱，陈皮（去白）8钱，五灵脂8钱，干蝎8钱，桂（去皮）8钱，肉豆蔻4个，草豆蔻4个，川乌头（炮裂，去皮脐）4个，大附子（炮，去皮脐）3个，槟榔3个，缩砂仁4钱，益智仁4钱，川椒（去目，出汗）4钱，胡椒4钱，丁香4钱，木香4钱，吴茱萸（汤洗7次，焙干，炒）4钱，木通（锉）3钱。
出处：《御药院方》卷四。
主治：伤冷溏下，腹满(月真)胀，其状如覆栲栳，喘鸣奔急，鼻张口（口去），厥气上下，不得分泄，又不任泻下药。
加减：None
功效：None
用法用量：每服30-40丸，食前煎生姜、橘皮汤送下。
制备方法：上为细末，面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
