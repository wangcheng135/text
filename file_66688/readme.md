序号：66704
名称：鹿茸丸
组成：鹿茸（去毛，酥炙）1两，石斛（去根，锉）1两，附子（炮裂，去皮脐）2两，熟干地黄（焙）2两，牡丹皮1两，泽泻1两，山芋1两，桂（去粗皮）1两，杜仲（去粗皮，炙）1两，萆薢（锉）1两，山茱萸1两，白茯苓（去黑皮）1两，五味子1两，肉苁蓉（酒浸1宿，切，焙）2两，补骨脂（炒）2两，远志（去心）1两，防风（去叉）1两，黄耆（锉）1两。
出处：《圣济总录》卷一八七。
主治：肾脏虚损，脚膝无力，腰脊拘急，口苦舌涩。
加减：None
功效：None
用法用量：每服20丸，空腹、晚食前用温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：石斛丸（《普济方》卷二二一）。
