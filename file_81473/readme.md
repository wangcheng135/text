序号：81489
名称：麒麟竭散
组成：麒麟竭（研）1两，赤茯苓（为末）1两，白茯苓（为末）1两，没药（研）1两，甘草节（捶碎）1两，老天萝子1个（连皮切碎，新者2个）。
出处：《普济方》卷二八四。
主治：痈疽毒疮。
加减：None
功效：None
用法用量：上用蜜1两，无灰酒2升，于银石器中煎天萝子、甘草节至3盏，分3服，调前药服。如曾服金石毒发，先吃猪羊血，方得服药；如不曾服金石药，只是气结，更不必用猪羊血，却以天萝子酒调，并进3服，其疮自穿，痛立止，未成脓者即自消散。已穿后，又进3服，候脓化，取下积热毒如鱼涎，并不动脏腑。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
