序号：52878
名称：香棱丸
组成：木香（不见火）半两，丁香半两，京三棱（锉细，酒浸1宿）、枳壳（去瓤，麸炒）、青皮（去白）、川楝子（锉，炒）、茴香（炒）、蓬术（锉细）1两（用去壳巴豆30粒同炒黄色，去巴豆不用）。
出处：《济生》卷四。
主治：五积，痰癖癥块，冷热积聚。
加减：None
功效：破痰癖，消癥块。
用法用量：每服20丸，炒生姜盐汤送下，温酒亦得，不拘时候。
制备方法：上为细末，醋糊为丸，如梧桐子大，以朱砂研极细为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：仙方香棱丸（《卫生宝鉴》卷十四）、香壳丸（《玉机微义》卷二十）、仙方香壳丸（《济阳纲目》卷四十一）。
