序号：78478
名称：摩障灵光膏
组成：黄连（锉如豆大，童便浸1宿，晒干，为末）1两，黄丹（水飞）3两，当归（酒洗）2钱，麝香5分，乳香5分，轻粉1钱，硇砂1钱，白丁香1钱，龙脑1钱，海螵蛸（俱另研细末）1钱，炉甘石（以黄连1两煎水，淬7次，研细）6两。
出处：《金鉴》卷七十八。
主治：混睛，目生翳障。
加减：None
功效：None
用法用量：点眼用。
制备方法：先用好白蜜10两，熬5-7沸，以净纸搭去蜡面，除黄丹外，下余药，用柳木搅匀，次下黄丹再搅，慢火徐徐搅至紫色，却将乳香、麝香、轻粉、硇砂和匀入上药内，以不粘手为度。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
