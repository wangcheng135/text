序号：31245
名称：沉香羌活汤
组成：沉香3钱，木香3钱，羌活3钱，白芍药3钱，槟榔3钱，甘草（炙）2钱，川芎2钱，枳壳2钱，青皮2钱，紫苏茎叶2钱，紫苏子2钱，木瓜2钱。
出处：《普济方》卷二四五。
主治：脚气，大小便秘，或入腹冲心，疼痛肿满。
加减：None
功效：降气疏肝。
用法用量：上（口父）咀。每服4钱，水1盏，加生姜3片，食前煎服。
制备方法：None
临床应用：None
各家论述：《医略六书》：羌活疏通经络以祛寒湿，能除百节之痛；紫苏疏理血气以散风寒，能除脚气之肿；白芍敛阴和肝；川芎活血调气；槟榔导逆气以下行；枳壳破滞气以降逆；香附调气解郁；苏子散郁除痰；木瓜舒筋脉和脾；甘草缓中气和药；生姜散风寒，豁痰涎也。更以药汁磨入沉香、木香，而独以沉香名方者，以木香能调气醒脾胃，而沉香顺气降逆，直入肾脏，较木香为得力耳。务使痰降气行，则风寒解散，而经络清和，脚气肿痛无不退，安有入腹冲心昏迷之危乎？
用药禁忌：None
附注：沉香羌活散（《奇效良方》卷三十九）、沉香导气散（《杏苑》卷七）、沉香导气汤（《张氏医通》卷十四）。
