序号：47567
名称：神仙小圣药
组成：红铅半盏（真女首经更佳，2-3次出者次之，其色红黄为上，纯红者为中，紫黑者不用），朱砂5钱（用辰州豆片者佳，有精神为最）。
出处：《回春》卷四。
主治：None
加减：None
功效：乌须发，延寿。
用法用量：用人乳送下。即归室中静养21日，然后方许出门动作。
制备方法：先将红铅拌入朱砂放瓷盆内，日晒月照49日毕，飞仙池文武火升3炷香，其药透蔑过一边，冷定开看，与金箔相似，用鸡翎扫下约1分8厘为上等；其次1分2厘，以乌金纸包，入小眼药罐内，以黄蜡封口，外以尿胞皮通身包裹，仍放大瓶内，以棉絮塞紧，仍用竹叶尿脬紧扎，用络以长绳引入井中去火毒，49日取出，用好乳香末半分研细末，以人乳2-3滴将圣药和匀作3丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：神仙小圣丹（《济阳纲目》卷六十四）。
