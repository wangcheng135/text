序号：37543
名称：抱胆丸
组成：水银2两，朱砂1两（细研），黑铅1两半，乳香1两（细研）。
出处：《百一》卷一。
主治：癫痫风狂，或因惊恐怖畏所致；及妇人产后血虚，惊气入心，并室女月脉通行，惊邪蕴结。
加减：None
功效：None
用法用量：每服1丸，空心用井花水送下。病者得睡，切莫惊动，觉来即安。再进1丸，可绝根本。
制备方法：将黑铅入铫子内，下水银结成砂子，次下朱砂、滴乳香2味，乘热用柳木槌研匀，丸如鸡头子大。
临床应用：None
各家论述：《济阴纲目》：人之神，以心为宅；水银之液，以朱砂为宅。朱之色，通乎心；水银之液，通乎神。神以妙应无方，银以园通不滞，二者皆万劫不衰，故以类从焉。然心火之下，阴精承之则安；水银之流，真铅制之则定。所以者，皆以真水制真火也。本方以乳香通心气，朱砂养心血，铅汞交心肾于倾刻，而引神归舍也，神归则睡安矣。
用药禁忌：None
附注：None
