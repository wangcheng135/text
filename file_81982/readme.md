序号：81998
名称：蕲蛇酒
组成：蕲蛇1条（酒洗，润透，去骨刺及近头3寸，只取肉）4两，羌活2两，归身2两，天麻2两，秦艽2两，五加皮2两，防风1两。
出处：《喻选古方试验》。
主治：中风伤酒，半身不遂，口目斜，肤皮（疒帬）痹，骨节疼痛；及年久疥癣，恶疮风癞。
加减：None
功效：None
用法用量：每饮1-2杯。仍以滓晒干研末，酒糊为丸，如梧桐子大，每服50丸，煮酒吞下。
制备方法：上以生绢袋盛之，入金华酒坛内，悬胎安置，入糯米生酒醅五壶，浸袋，箬衣密封，安坛于大锅中，水煮1日，取起，埋阴地7日取出。
临床应用：None
各家论述：None
用药禁忌：切忌见风、近色及鱼、羊、鹅、面发风之物。
附注：None
