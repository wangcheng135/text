序号：38437
名称：治风锭子
组成：防风5钱，荆芥5钱，麝香3分，川芎3钱，当归3钱，天麻3钱，乳香3钱，没药3钱，羌活3钱，黄芩3钱，枳壳3钱，麻黄1两，蝉退（去足）4钱，白附子3钱，川乌3钱，僵蚕3钱，天竺黄2钱，皂角2钱，自然铜（醋淬）5钱，金箔10片，人参1钱5分，茯神3钱。
出处：《活人心统》卷一。
主治：中风，左瘫右痪，痰涎壅盛，眼目不正，手足不遂，历节疼痛。
加减：None
功效：None
用法用量：每服1锭，痰涎壅盛，薄荷汤磨下；身热抽搐，姜汁同薄荷汤磨下；口眼喎斜，葱白汤磨下；冷痛、久痛，姜葱汤或酒磨下；口渴面红，身热恍惚，灯心汤磨下。
制备方法：上为末，用糯米粥捣胶和匀，作锭子，金箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
