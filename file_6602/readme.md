序号：6602
名称：加味黄芩汤
组成：黄芩4钱，厚朴3钱，吴茱萸4钱，毛苍术4钱，杭白芍3钱，升麻3钱，车前子4钱（炒），木通3钱，大枣6枚，甘草2钱。
出处：《医学探骊集》卷三。
主治：漏底伤寒。伤寒日久，不能忌口，饮冷食凉，触动脾胃，致令脾气虚衰，脾湿下陷，腹痛泄泻。亦治痢疾。
加减：赤痢，去车前子，加大黄4钱，服1剂，再加地榆炭3钱，粟壳4钱，服2剂；白痢，去大枣、车前子，加槟榔3钱，炮姜2钱，地榆炭3钱，粟壳4钱。
功效：None
用法用量：水煎，温服。
制备方法：None
临床应用：None
各家论述：此方用黄芩清热；白芍敛阴；厚朴温中；甘草、大枣和胃；苍术燥湿；吴萸止其腹痛；升麻提升清气；车前分其清浊；木通引热下行。
用药禁忌：None
附注：None
