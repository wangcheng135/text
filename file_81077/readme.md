序号：81093
名称：麝香饼子
组成：全蝎21个（每1个用薄荷2叶束定，先以生姜汁浸软麻黄，逐个以麻黄缠定，却蘸姜汁于慢火上炙，令麻黄黄色，又蘸姜汁炙，如此3遍，焙干），乳香（别研）3钱，朱砂（别研）3钱，雄黄（别研）3钱，白花蛇（酒浸，取肉，焙干）3钱，天南星（炮制，入生姜汁内浸1宿，切，焙干）3钱，乌蛇（酒浸取肉，焙干）3钱，白僵蚕（炒去丝嘴）3钱，附子1个（去皮脐尖，取末）3钱，麝香半钱（别研）。
出处：《杨氏家藏方》卷十七。
主治：小儿吐泻之后，内虚生风，已成慢惊，涎潮搐搦，头项反折，神情昏困。
加减：None
功效：None
用法用量：每服1饼子，煎荆芥汤化下，不拘时候。
制备方法：上为细末，次入研者药和匀，别用天南星末1两，调生姜汁煮糊，与前药为丸，如梧桐子大，捏作饼子，略见日色，阴干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
