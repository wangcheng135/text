序号：46504
名称：千金丸
组成：乌头、附子、川椒、桂心、吴茱萸、菖蒲、远志、人参、茯苓、苍术、枳壳、茴香、神曲、麦糵、胡椒、干姜、肉豆蔻、木香、缩砂、红豆、三棱、蓬术、陈皮、白豆蔻、丁香、藿香、甘草、桔梗、薏苡、山药、白扁豆、良姜、益智各等分。
出处：《普济方》卷二五六。
主治：中风虚寒，停留饮食，膈气噎气，五积六聚，反胃吐食，胸腹胀满刺痛，及心痛彻背，手足不收，言微气短；或结癥块在腹胁间，大如杯盘，时发痛，及吞酸吐苦，气逆喘咳，饮食不进，渐至浮肿，久病痢泄，霍乱吐下，劳伤虚损，羸瘦憔悴，痰多积水，及疝气痞气、奔豚等气。一切胃弱脏虚，腹内诸冷。
加减：None
功效：None
用法用量：每服1丸，汤饮嚼下；胸膈病，即食后服。
制备方法：上为末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
