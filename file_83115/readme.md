序号：83133
名称：乌头摩风膏
组成：川乌头（生，去皮脐）半两，防风（去芦头）半两，桂心半两，白芷半两，藁本半两，川椒（去目）半两，吴茱萸半两，白术半两，细辛半两，芎半两，白附子半两，藜芦半两，莽草半两，羌活半两，黄蜡5两，炼了猪脂1斤，生姜3两。
出处：《圣惠》卷二十五。
主治：风痛，及皮肤不仁，筋脉拘急。
加减：None
功效：None
用法用量：每有痛处，于火边熁手，乘热取膏摩之100-200遍。以手涩为度。
制备方法：上锉细，先以猪脂纳铛中煎之，以入诸药，煎令白芷色黄，候药味出尽，以新布绞去滓，更以绵滤过；拭锅令净，重入膏于锅中，慢火熬之；次下黄蜡令消，去火，待稍凝，收于瓷器中。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
