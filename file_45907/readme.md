名称：栀子香豉淡竹叶汤
组成：栀子（去壳取仁，用甘草水浸1宿取出，焙干，捣筛为末）13个，香豉小半合（用大黑豆3斗，6月内淘净沥干，蒸熟取出摊蓆上，待微温以青蒿覆之，每3日1看，待黄衣上遍即取，晒干，筛净，更用东流水拌润，干湿得所，以汁出指间为度，安瓮中，筑实，上以桑叶盖之，厚3-4寸，以泥密封，日中晒7日取出，摊晒1时许，又用秋秔拌入豆内，复安瓮中，晒7日取出，摊晒1时许，复安瓮中，晒7日，如此7遍取出，蒸之，摊令气歇，复收极净瓷瓮中，筑极实，密封瓮口，1月后即成矣），淡竹叶（粟米泔洗3遍，切碎）半斤，甘草（去头尾，蜜水润透，涂酒，炙黄色，锉碎）4钱，蜀漆（连根收采者佳，临用时去根，以甘草锉细，将东流水润透，拌入蜀漆内蒸之，勿使气漏矣，冷去甘草，取蜀漆，锉碎，又拌甘草水，干湿得所蒸之矣，冷晒干）1两，常山（临用去苗，同甘草末水润拌蒸矣，冷去甘草，取常山，锉碎，再用好酒拌润1宿取出，熬，捣）1两5钱，鳖甲（取9肋者，3两，洗去甲外黑皮，甲里皮肉置罐中，用酽醋煮干取出，炙燥，锉碎）1两3钱，石膏（研细，甘草水飞澄，晒，再研）2两，乌梅10枚（汤润，去核，入米中蒸烂，焙干）。
出处：《痎疟论疏》。
主治：心疟。
加减：None
功效：None
用法用量：上以水3升5合，煮取1升半，去滓，分3次温服，未发前令3服尽。
