序号：60303
名称：通气防风汤
组成：防风5分，羌活5分，陈皮5分，人参5分，甘草5分，藁本3分，青皮3分，白豆蔻2分，黄柏2分，升麻1钱，柴胡1钱，黄耆1钱。
出处：《内外伤辨》卷中。
主治：风热乘肺，肺气郁甚，肩背痛，汗出，小便数而少。
加减：None
功效：开泻肺经风热，解肌表和膀胱。
用法用量：上（口父）咀，都作1服，水2盏，煎至1盏，去滓，食后温服。
制备方法：None
临床应用：None
各家论述：《脾胃论注释》：用柴胡、升麻升阳气；黄耆、人参、甘草益肺气；防风泻肺经风热，用于升阳与益气之中，因为防风既是风药中润剂，又能助黄耆以益气；羌活、藁本散太阳的风寒，橘皮运脾气，青皮理胃气，蔻仁宣肺气，三使分途而出，少用黄柏引热下行，阳气舒伸，气通痛定溲便自调。
用药禁忌：如面白脱色，气短者，不可服；气盛者宜服。
附注：防风通气汤（《杏苑》卷三）。本方方名，《医学纲目》引作“防风汤”。
