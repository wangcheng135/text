序号：58432
名称：破结汤
组成：防风1钱，荆芥1钱，川芎1钱，当归（酒洗）1钱，连翘1钱，白鲜皮（炒）1钱，白牵牛（炒）1钱，牛膝7分，皂角刺1钱，生甘草5分，金银花1钱，细辛3钱，土茯苓4两。
出处：《医林纂要》卷十。
主治：杨霉结毒。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：防风、荆芥散经隧之毒；川芎、当归活血而行之；连翘散结热；白鲜皮泻毒，使出于小肠膀胱；白牵牛走气分，逐小肠膀胱之毒而出之，盖毒本自肾，故仍自肾腑而逐之；牛膝达周身之毒，使之下行；皂角刺达于毒所结之所，而破其结；生甘草，金银花解毒而以和缓；用细辛以尽拔肾部之积毒而散之行之；土茯苓解下部之毒为君，而臣以细辛以拔根本之毒。
用药禁忌：None
附注：None
