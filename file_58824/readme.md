序号：58839
名称：秘传膏药
组成：赤石脂2钱，舶上硫黄2钱，天门冬（去心）2钱，麦门冬（去心）2钱，熟地黄（酒浸）2钱，菟丝子（酒浸）2钱，木香（酥炙）2钱，肉苁蓉（酒浸）2钱，没药（另研）2钱，紫霄花2钱，杏仁（去皮尖，另研）2钱，鹿茸2钱，虎骨（生用）2钱，牛膝（酒浸）2钱，阳起石2钱，远志（去心）2钱，川续断2钱，蛇床子2钱，谷精草2钱，龙骨（煅）2钱，附子1个（炮，去皮脐），乳香5钱（另研），蟾酥（另研）1钱，麝香（另研）1钱，雄黄4钱（另研），生地黄2钱，沉香2钱，母丁香2钱，官桂3钱（另研），甘草3钱，松香（明净）3两（另研），木鳖子1两（去壳，碾）。
出处：《普济方》卷三一五。
主治：阳衰。
加减：None
功效：补益。
用法用量：用时以绯绢裁帛，逐渐厚摊如小碗大，贴脐上、腰上。
制备方法：上为末，除甘草、杏仁、木鳖子、官桂4味，用水6斤4两于砂锅内用桑柴火熬至1碗，去滓净，将松香末，小油2两，白及末1两，下砂锅内化开数沸，却下其余药末，以槐、柳、桑条不住手搅成膏药，稠粘为度。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
