序号：27599
名称：补阴丹
组成：磁石（紧者，烧赤，醋淬7次，水飞过，晒干称）3两，鹿茸3两（去毛，酥炙），生干地黄8两，石斛3两，泽泻3两，官桂1两半（去粗皮），杜仲2两（细切，炒去丝），山茱萸3两（生用）。
出处：《御药院方》卷六。
主治：发热怔忪，脚膝痹弱。
加减：None
功效：滋益肾水真阴，镇伏心火大热，坚强骨髓，补养精气，通调血脉，润泽肌肤，交泰心肾。
用法用量：每服50丸，空心、食前温酒送下，或盐汤送下亦得，日进1服。服2月觉功，100日见效。
制备方法：上为细末，入磁石末同研匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
