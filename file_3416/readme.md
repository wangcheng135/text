名称：太乙种子丸
组成：鱼鳔（炒成珠）4两，真桑螵蛸4两（炒黄），韭子（炒）2两，莲须2两，熟地2两（焙），杜仲2两（姜炒），牛膝2两（酒浸），枸杞子2两，沙蒺藜（炒）2两，人参2两，菟丝子2两（酒煮），天冬2两，龟板2两（炙），鹿茸2两（炙），破故纸2两（酒浸，炒），肉苁蓉2两（酒洗，去鳞甲），白茯苓2两，远志肉（去骨，甘草水泡）2两，当归2两（酒洗），青盐（泡）5钱。
