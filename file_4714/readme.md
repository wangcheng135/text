序号：4714
名称：木香硇砂煎丸
组成：木香1两，硇砂1两，官桂1两，附子1两（炮），干漆1两（去烟），猪牙皂角1两，细辛1两，乳香1两（研），京三棱1两（炮），广茂1两（炮），大黄1两（炒，令为末），没药1两（研），干姜1两（炮），青皮1两，巴豆霜半两。
出处：《卫生宝鉴》卷十四。
主治：妇人痃癖积聚，血块刺痛，脾胃虚寒，宿食不消，久不愈者。
加减：None
功效：None
用法用量：每服30-50丸，食后以温酒送下。
制备方法：上除研药外，同为末，以好醋1升，化开硇砂，去滓，纳银石器中，慢火熬，次下巴豆霜、大黄末，熬成膏，将前药末膏内和丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
