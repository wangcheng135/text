序号：55507
名称：桃仁芍药汤
组成：桃仁半升，芍药2两，芎藭2两，当归2两，干漆2两，桂心2两，甘草2两。
出处：《千金》卷三。
主治：产后腹痛。产后恶血未尽，攻心腹痛。
加减：None
功效：None
用法用量：上(口父)咀。以水8升，煮取3升，分3服。
制备方法：None
临床应用：None
各家论述：《医略六书》：产后冲任不调，不能操蓄泄之权，而瘀血内结，故腹痛环脐牵引不宁焉。桃仁破瘀血以开结，当归养冲任以荣经，赤芍破瘀泻血滞，桂心通闭温经脉，川芎行血中之气，漆灰化瘀结之血，甘草缓中州以和胃气也。水煎温服，使瘀化结开，则经脉清和，而腹痛无不止，安有牵引环脐之害乎。
用药禁忌：None
附注：桃仁散（《圣惠》卷八十一）。本方方名，《医略六书》引作“桃仁煎”。《圣惠》有干姜。
