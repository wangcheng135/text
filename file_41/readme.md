序号：41
名称：五味子汤
组成：五味子1两，人参1两，诃黎勒皮（炒）1两，白术1两，白茯苓（去黑皮）1两，桔梗（炒）1两，枳壳（麸炒，去瓤）1两，前胡（去苗土）1两，贝母（去心，炒）1两，陈橘皮（汤浸去白）1两，甘草（炙）1两，半夏（生姜汁和作曲，焙干）1两，麦门冬（去心，焙）半两，干姜（炮裂）半两，桂（去粗皮）半两。
出处：《圣济总录》卷一八七。
主治：痰盛，心经虚热，咽喉干煿，舌涩壅闷，口内生疮。
加减：None
功效：利胸膈，和脾肺气，止嗽思食。
用法用量：每服3钱匕，水1盏，加生姜1枣大(切)，煎至7分，去滓温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
