序号：3517
名称：太和丸
组成：白术（去油，土炒）4两，白茯苓（去皮）2两，怀山药2两，莲肉（去心皮）2两，当归身（酒炒）4两，白芍药（酒炒）2两，陈皮1两，川黄连（姜炒）1两，山楂（去子）1两，枳实（面炒）1两，半夏（汤泡，切片，姜炒）1两，神曲（炒）1两，香附（用童便炒）1两，木香5钱，龙眼肉1两，炙甘草5钱，人参5钱，白豆蔻（去壳）5钱，嫩黄耆（蜜水炒用）1两。_x000D_
。
出处：《寿世保元》卷四。
主治：None
加减：年幼、壮者，去参、耆。
功效：大补诸虚，专进饮食，清痰降火，解郁消滞，养气健脾，预防饮食失节损伤脾胃，劳役过度耗散元气，而成内伤诸病。
用法用量：每服100丸，食后、临卧米汤送下。
制备方法：上为细末，荷叶如掌大者煎汤，下陈仓米半钟煮稀粥和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
