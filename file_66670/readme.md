序号：66686
名称：鹿茸丸
组成：鹿茸（去毛，酥炙）1对，木香半两，葫芦巴1两，石斛（去根）半两，茴香子（炒）3分，巴戟天（去心）1两，附子（炮裂，去皮脐）半两，牛膝（酒浸，切，焙）1两，槟榔3分（锉），熟干地黄1两，破故纸1两（酒浸，炒），肉苁蓉1两（酒浸，焙），官桂半两，菟丝子1两（酒浸1日），蛇床子半两（酒浸），苦楝子1两，薯蓣半两，干姜半两。
出处：《圣济总录》卷一八七。
主治：劳伤。
加减：None
功效：补诸不足。
用法用量：每服50丸，空心温酒或盐汤送下。
制备方法：上为末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
