序号：37208
名称：定喘丸
组成：苏梗12两，白芥子（炒）12两，苏子（炒）12两，桑皮15两，苏叶12两，百合9两，杏仁（去皮，炒）18两，莱菔子（炒）2两，橘皮18两，天冬4两，川贝母15两，知母6两，法半夏6两，紫菀4两，麦冬8两，生地3两，冬花4两，白术（炒）10两，当归6两，首乌（炙）4两，阿胶（炒珠）32两，茯苓10两，黄耆16两。
出处：《北京市中药成方选集》。
主治：肺虚气亏，劳伤久嗽，喘咳痰盛，春秋举发。
加减：None
功效：理肺补气，止嗽定喘。
用法用量：每服2丸，温开水送下，1日2次。
制备方法：上为细末，每20两兑龙涎香3分，用梨膏为丸，重1钱，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
