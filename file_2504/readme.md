序号：2504
名称：午时茶
组成：杜藿香1斤半，紫苏1斤半，荆芥1斤半，青蒿1斤，前胡1斤，制半夏1斤，制川朴12两，广皮1斤，炒白术1斤，广木香1斤，枳壳1斤，青皮1斤，槟榔1斤，炒莱菔子1斤，焦山楂1斤，炒麦芽1斤，炒六曲1斤，豆蔻8两，西砂仁8两，红茶10斤，生姜5斤，面粉6斤半。
出处：《中药成方配本》。
主治：感冒食积，寒热吐泻。
加减：None
功效：解表疏中。
用法用量：每用1块至2块，绢包煎服。
制备方法：先将生姜刨丝打汁候用，上药除应炒者外，其余生晒，为粗末，将姜汁、面粉打浆和药为块，每块约干重5钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
