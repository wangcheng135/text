序号：70577
名称：紫芝丸
组成：紫芝1两半，山芋1分半，天雄（炮裂，去皮脐）1分半，柏子仁（炒香，别研）1分半，枳实（去瓤，麸炒黄）1分半，巴戟天（去心）1分半，白茯苓（去黑皮）1分半，人参3分，生干地黄（洗，焙）3分，麦门冬（去心，焙）3分，五味子（去茎叶，炒）3分，半夏（汤洗去滑，炒）3分，牡丹皮3分，附子（炮裂，去皮脐）3分，蓼实1分，远志（去心）1分，泽泻半两，瓜子仁（炒香）半两。
出处：《圣济总录》卷九十。
主治：虚劳短气，胸胁苦伤，唇口干燥，手足逆冷，或有烦躁，目视(目荒)(目荒)，腹内时痛。
加减：None
功效：安神保精。
用法用量：每服15丸，渐增至30丸，温酒送下，空心、日午、夜卧各1服。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
