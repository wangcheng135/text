序号：9330
名称：加参生化汤
组成：人参3钱（有倍加至5钱者），川芎2钱，当归5钱，炙草4分，桃仁10粒，炮姜4分。
出处：《傅青主女科》卷下。
主治：产后1-2日，血块痛虽未止，产妇气血虚脱，或晕或厥，或汗多，或形脱，口气渐凉，烦渴不止，或气喘急者。
加减：血块痛甚，加肉桂7分；渴，加麦冬1钱，5味10粒；汗多，加麻黄根1钱；如血块不痛，加炙黄耆1钱；伤饭食、面食，加炒神曲1钱，麦芽5分（炒）；伤肉食，加山楂5个，砂仁4钱（炒）。
功效：None
用法用量：加大枣，水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：产后发厥，块痛不止，不可加耆、术。
附注：None
