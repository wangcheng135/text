序号：53336
名称：健脾补肾汤
组成：党参15g，川续断15g，白术12g，茯苓12g，白芍12g，当归12g，五味子12g，菟丝子12g，川厚朴9g，香附9g。
出处：《古今名方》引关幼波方。
主治：慢性、迁延性肝炎，早期肝硬化，肝功能长期不正常，证属脾肾两虚者。
加减：腹泻，加苍术、芡实、诃子肉各9g；腰腿痛，足跟痛，加牛膝、生薏苡仁、仙灵脾各12g；夜尿多，加鹿角霜、女贞子各12g；腹胀甚，加冬瓜皮12g，木香4.5g；肝区隐痛，加桑寄生15g，木瓜12g。
功效：健脾补肾。
