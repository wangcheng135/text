序号：24846
名称：如圣青金丹
组成：青黛半两，瓜蒂1分，朱砂1分（水飞），轻粉1分，蝉壳（去土）3个，雄黄1分（水飞），芦荟1分，胡黄连1分，熊胆1分（化开），麝香少许。
出处：《卫生总微》卷六。
主治：小儿一切风痫搐搦，及寒热腹痛，诸虫五疳八利，肚大脚细，泄泻无时，天吊胎惊，暗风痫病，热疳口疮。
加减：None
功效：None
用法用量：诸般惊痫风疾，搐搦上视，温水化1丸，滴鼻中取嚏，更用薄荷水化下1丸；久患泄泻，腹胀肚大，脚细诸疳，米饮化下；疳虫蛔咬，苦楝子汤送下；疳蚀口疮鼻烂，乳汁研涂；疳眼雀目，白羊子肝1片，竹刀擘开，研药作末，掺在肝片内，麻缠，米泔煮熟，空心服。
制备方法：上为细末，用豬猪胆汁为丸，如梧桐子大，瓷盒贮之。
临床应用：None
各家论述：None
用药禁忌：乳母常忌鱼、大蒜、鸡、鸭、猪肉。
附注：None
