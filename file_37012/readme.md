序号：37022
名称：奇特六神丹
组成：辰砂1钱（研），蝎梢（去尖）1钱，白僵蚕（直者）1钱（浴净，姜汁浸，微炙），蜈蚣1条（大者，去头足，刮去腹中物，酒浸，炙香），真麝1字，甘草、半夏1钱（陈者，刮去脐，滚汤洗7次），人参（去芦，洗，切，焙）1钱，藿香（去尖）1钱。
出处：《永乐大典》卷九七八引《小儿保生要方》。
主治：小儿急慢惊风，涎潮气壅。
加减：None
功效：安神养魄，去风邪，定嗽喘，利膈正气。
用法用量：实及周岁儿，每服1丸，薄荷汤化下；或2-3周，未及周者，可以意加减。若疾势巳危者，不可守此例也。
制备方法：上为极细末，炼蜜为丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
