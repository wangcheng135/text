序号：60572
名称：通经逐瘀汤
组成：桃仁8钱（研），红花4钱，赤芍3钱，山甲4钱（炒），皂刺6钱，连翘3钱（去心），地龙3钱（去心），柴胡1钱，麝香3厘（绢包）。
出处：《医林改错》卷下。
主治：痘形攒簇，蒙头覆釜，周身细碎成片，或夹疹夹斑，浮衣水泡，其色或紫、或暗、或黑，其症或干呕、烦躁，昼夜不眠，逆形逆症，皆是瘀血凝滞于血管。
加减：大便干燥，加大黄2钱，便利去之；5、6日后，见清浆、白浆，将麝香去之，加黄耆5钱，将山甲、皂刺减半；至7、8日后，桃仁、红花亦减半，黄耆可用8钱。此方指4-5岁而言，若1-2岁，分两可减半；若7、8岁，分两可加1半。
功效：活血化瘀，解毒。
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：《医林改错评注》：方中用连翘、柴胡解毒，麝香、山甲、地龙、皂刺通络，赤芍、桃仁、红花逐瘀。
用药禁忌：None
附注：None
