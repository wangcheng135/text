序号：61428
名称：啄木散
组成：寒水石（火煨、别研）2两，铁粉1两（别研），附子1枚（重1两者，炮，去皮脐，取末），牛黄（别研）1分，麝香（别研）1分，龙脑（别研）1分，朱砂（别研）1分，啄木鸟1枚（腊月者，去嘴翅尾爪尖，用瓦罐子先铺荷叶、荆芥穗1寸厚，次入无灰酒1升，方下啄木鸟，更以荷叶、荆芥穗盖1寸厚，用纸封口，盐泥固济，炭火煅青烟出为度，候冷，只取啄木鸟，细研）。
出处：《杨氏家藏方》卷二。
主治：一切痫疾。
加减：None
功效：None
用法用量：每服1钱，用温酒1盏调下，便就枕睡少时。临发时服尤妙，不拘时候。
制备方法：上药同和匀，再研细。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
