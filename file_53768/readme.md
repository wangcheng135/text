序号：53783
名称：射干丸
组成：射干1两，半夏（汤洗10遍，炒干）1两1分，干姜（炮裂）1两，款冬花（去萼，焙干）1两，皂荚（去皮子，炙）1两，陈橘皮（汤浸，去白，焙）1两，百部（焙干）1两1分，五味子（拣净）1两1分，细辛（去苗叶）1两，贝母（去心，炒令微黄）1两，白茯苓（去黑皮）1两，郁李仁（汤退，去皮尖双仁，研如脂）1两。
出处：《圣济总录》卷六十五。
主治：久患呷嗽，喉中作声，发即偃卧不得。
加减：None
功效：None
用法用量：每服7丸，稍加至15丸，空腹饮下，日2次。
制备方法：前11味为末，与郁李仁同研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
