序号：35387
名称：京三棱丸
组成：京三棱3两（炮裂，锉），川大黄2两（锉碎，微炒），鳖甲2两（涂醋炙微黄，去裙襴），赤芍药1两，桂心1两，干姜1两（炮裂，锉），诃黎勒2两（煨，用皮），槟榔2两，川乌头1两（炮裂，去皮脐），吴茱萸1两（汤浸7遍，焙干，微炒），桃仁4两（汤浸，去皮尖双仁，麸炒微黄）。
出处：《圣惠》卷二十八。
主治：虚劳，积聚痞结，腹胁胀满。
加减：None
功效：None
用法用量：每服20丸，食前以温酒送下。渐加至30丸，下烂肉黑脓为度。
制备方法：上为末，熬醋如胶，和捣为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“三棱丸”。
