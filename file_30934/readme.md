序号：30943
名称：十珍丸
组成：猪背膂1条，羊背膂1条，鳖鱼1只，乌骨鸡1只（4味制净，去骨留肉，煮酒一大瓶，甏内煮熟，擂碎），大山药1条，莲肉半升，京枣100个，霜柿10个（上4味，用井水1大瓶，于沙糖甏内煮熟，擂细，却与前药一处，用慢火熬，却下），明胶4两，真黄蜡3两。
出处：《修月鲁般经》引《劳证十药神书》（见《医方类聚》卷一五○）。
主治：一切劳。大怯极虚甚惫，骨干津涸，血枯气竭，火乘金位者，服白凤膏愈后，即服此药。
加减：None
功效：此药服之半月，精神完复，气血津和。服之1月，饮食倍加，轻居轻快。服之两月，百病消除，诸气益补。服之3月，丹田常暖，水火既济。服之4月，老者反壮，行步如龙飞。服之5月，发白再黑，齿落更生。
用法用量：上2味，旋旋下，与前8味和作1处，擂成膏子，和平胃散末，四君子汤末，并知母1两，黄柏1两，共10两、搜和成剂。如十分硬了时，再入白蜜同熬，取起，放青石上，用水捶打如法，匀了，为丸如梧桐子大。每服100丸，枣汤吞下，不拘时候。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《十药神书》（陈修园注本）作“补髓丹”。
