序号：32074
名称：万应丸
组成：硇砂半两（水飞过，研），阿魏（醋研，去砂土）半两，大黄半两，吴茱萸（去枝梗）半两，青礞石（研细末，用焰消拌和，于银锅内煅，取净）半两，肉桂半两，木香半两，青皮（去瓤）半两，玄胡索半两，五灵脂（酒淘，去沙）半两，小茴香（炒）半两，川山甲（蛤粉炒）半两，乳香半两，没药半两，当归半两，石菖蒲半两，皂角（去皮弦子）半两，干漆（炒烟尽）半两，槟榔半两，陈皮（去白）半两，枳壳（去瓤，炒）半两，京三棱（煨）半两，丁香半两，莪术（煨）半两，良姜（炒）半两，甘遂半两，芫花（醋煮，焙）半两，大戟半两，雄黄半两，巴豆（去油膜）3钱。
出处：《医方类聚》卷八十九引《王氏集验方》。
主治：蛊气，血气，结块疼痛，癥瘕积聚，心气脾疼，食积，肉积、酒积，胃冷吐食，气膈噎塞不通，遍身水气浮肿，气急痰壅。妇人血气不行，腹肚疼痛，年深日久者。
加减：None
功效：破一切积，散一切气。
用法用量：每服30丸，空心生姜汤送下。利后以白粥补之。
制备方法：上为细末，醋煮面糊为丸，如梧桐子大。
临床应用：蛊胀：一妇人年四十余岁，经脉不行十三个月，腹肚蛊胀而疼，时肿时消，医以行经动胎之药服之，女口水浇石，脉息沉细而实。予曰：此非胎也，当作血气治之。予以此药，生姜汤下三十丸，大便如常，腹疼稍减。至次日五更初，再进五十丸，至天明，粪下异色，腥臭难闻，腹蛊稍消，旋以白粥补之。第三日早，又以五十丸进之，至天明，粪下如故，腹胀又减。如是者服药十日，其病全获安矣。
