序号：55918
名称：桑耳散
组成：桑耳1两（微炒），丹参1两，续断3分，芎藭3分，柏叶3分（炙微黄），熟艾3分（焙微黄），鹿茸1两（去毛，涂酥炙微黄），牡蛎1两（烧为粉），地榆3分（锉），阿胶1两（炙令黄燥），刺蓟根3分，龟甲1两（涂醋炙令黄），赤石脂1两，当归3分（锉，微炒），槲叶1两，熟干地黄1两，牛角腮1两（炙令微黄）。
出处：《圣惠》卷七十三。
主治：妇人赤白带下，无问远近。
加减：None
功效：None
用法用量：每服2钱，食前以温酒调下。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：None
附注：桑耳续断散（《圣济总录》卷一五二）。方中龟甲，《圣济总录》作“鳖甲”。
