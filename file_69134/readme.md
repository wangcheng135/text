序号：69150
名称：黄金汤
组成：黄土5钱，扁豆4钱（炒），谷芽2钱（炒），茯苓1钱，黑豆3钱，甘草8分，白芍1钱5分（炒），生姜3片，金银花3钱，五谷虫2钱（炒，研），扁豆花10枚。
出处：《杂症会心录》卷上。
主治：痢疾。
加减：体实受邪者，加黄连1味。
功效：解疫毒，救胃气。
用法用量：水2钟，煎8分，不拘时服。
制备方法：None
临床应用：None
各家论述：《证因方论集要》：黑豆、银花解毒；甘草、白芍理太阴腹痛，茯苓、扁豆醒脾开胃；谷芽消滞和中；扁豆花清暑；黄土治泄痢冷热赤白，腹内热毒绞痛；五谷虫止毒痢，且藉其秽以入大肠；生姜畅胃口而下食；是方寓平淡于神奇矣。
用药禁忌：None
附注：None
