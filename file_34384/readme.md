序号：34393
名称：万病丸
组成：硼砂半两，阿魏半两（醋研，去土），大黄半两，吴茱萸半两（去皮），青礞石半两（研细末，用缩砂拌和，锅内炒，取净），肉桂半两，木香半两，玄胡索半两，五灵脂半两（酒淘净），小茴香半两（炒），穿山甲半两（蛤粉炒），乳香半两，没药半两，当归半两，石菖蒲半两，皂角半两（去皮弦），干漆半两（炒烟尽），槟榔半两，陈皮半两（去白），枳壳半两，荆三棱半两（煨），丁香半两，莪术半两（煨），良姜半两（炒），甘遂半两，芫花半两（醋炒），大戟半两，雄黄半两，巴豆（去皮油膜）2钱半。
出处：《普济方》卷一六九引《医学切问》。
主治：气蛊、血气、结块、冷痛，癥瘕积聚，心气脾疼，食积、酒积、肉积，胃冷吐食，气膈噎塞不通，遍身浮肿，气急痰壅；妇人血气不行，肚腹疼痛，年深日久者。
加减：None
功效：破积散气。
用法用量：每服30丸，空心生姜汤送下。利行，白粥补之。
制备方法：上为细末，醋煮面糊为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
