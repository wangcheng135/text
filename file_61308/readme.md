序号：61324
名称：高良姜散
组成：高良姜1两，乌药1两，京三棱（并锉）1两，吴茱萸（水浸1宿，晒干）2两（与上3味相和炒，待茱萸焦即住火，净，拣去茱萸不用，将3味与后药同捣），丹参（锉，微炒）半两，沉香（锉）半两，莎草根（炒去毛）半两，当归（切，焙）半两，桂（去粗皮）半两，桃仁（汤浸去皮尖双仁，麸炒令黄，研）半两，槟榔（微煨）半两，麝香1分（别研入）。
出处：《圣济总录》卷五十五。
主治：厥心痛。面色青黑，眼目直视，心腹连季胁引痛满胀。
加减：None
功效：None
用法用量：每服3钱匕，煎茯苓汤调下；或炒桃仁酒调下亦得。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
