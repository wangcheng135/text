序号：83857
名称：乌蛇膏
组成：乌蛇1两（烧灰），马齿1两（烧灰），猬皮1两半（烧灰），乱发3分（烧灰），黄矾3分（细研），斑蝥3分（去翅足，糯米拌炒黄色），杏仁49枚（去皮，研如膏），麝香1分（细研），猪脂1升（腊月者），猪牙皂荚1分（炙，捣末），水银3分（入少胡粉，点水研令星尽）。
出处：《圣惠》卷六十。
主治：痔疾，年月深远，旁生孔窍，有头，脓血出，疮痒痛难忍。
加减：None
功效：None
用法用量：先煎猪脂候溶，滤去滓，入诸药煎20-30沸，欲成膏入麝香搅令匀，更煎2-3沸，入黄蜡3两，候冷，置于瓷盒内。每以少许，贴于痔上，1日2-3次。
制备方法：上为极细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
