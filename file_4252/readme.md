序号：4252
名称：木香丸
组成：人参1两，木香1两，茯苓（去皮，水煮如面者研）1两，没药1两，青橘皮1两，菴(艹闾)子1两，槟榔5个，白豆蔻仁5个，水银4两（水煮1伏时，同枣肉研星尽），当归8两，金牙石半两，麒麟竭半两，薄荷半两，荆芥穗半两，犬胆14个，不蚛皂角1挺。
出处：《鸡峰》卷十。
主治：血气及一切积聚败血为病。
加减：None
功效：None
用法用量：每服5-7丸，看病大小用之，汤使临时。
制备方法：上为细末，分一半，别入灯上燎者，巴豆21个，杏仁21个，同用面糊为丸，如梧桐子大；一半药末只炼蜜为剂，杵一千下，吃时旋丸小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：有巴豆、杏仁者名大药，治干血气积血气；无巴豆、杏仁者名小药，治产后及血气疰闷，不下食，血晕。
