名称：安全散
组成：人参（去芦）1钱（焙），白术（炒）1两，白附子（炮）1两，南星（姜汁1碗煮干切片，炒）1两，天麻（炮）1两，辰砂（别研）1两，当归（焙，去芦）1两，乳香（别研）1两，没药（别研）1两，吊藤（勾子者，焙）1两，白僵蚕（直者，炒去丝）1两，全蝎（去毒）1两，白茯苓（去皮，焙）1两，羌活（焙）1两，防风（去芦，焙）1两，川芎（焙）1两，甘草（炙）1两，麝香半钱。
