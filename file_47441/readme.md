序号：47456
名称：祛涎丸
组成：天南星4两，半夏9两半，白附子2两6钱，川乌头7钱半。上并生为细末，用生绢袋盛，以井花水揉洗澄滤，有滓更研，再入袋摆洗尽，瓷盆中日晒夜露，每至晓澄去宿水，别换井花水，搅匀晒，春5日，夏3日，秋7日，冬10日，去水晒干如玉片，方入后诸药：白花蛇（酒浸，去皮骨，焙干，称）1两，剑背乌梢蛇（酒浸1宿，去皮骨，焙干，称）1两，白僵蚕1两（炒去丝嘴），全蝎1两（去毒，微炒），川芎2两，天麻2两。
出处：《杨氏家藏方》卷八。
主治：风痰壅盛，头目昏痛，旋晕欲倒，呕哕恶心，恍惚健忘，神思昏愦，肢体烦疼，颈项拘急，头面肿痒，手足不举，或时麻痹。
加减：None
功效：None
用法用量：每服30丸，食后生姜、薄荷汤送下。
制备方法：上为细末，生姜自然汁煮糊为丸，如绿豆大，以飞研细朱砂1两，麝香末2钱为衣，风干，密器中盛之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
