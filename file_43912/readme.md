序号：43927
名称：厚朴散
组成：厚朴2两（去粗皮，涂生姜汁，炙令香熟），陈橘皮1两（汤浸，去白瓤，焙），草豆蔻1两（去皮），人参3分（去芦头），芎3分，白术3分，阿胶3分（捣碎，炒令黄燥），当归3分（锉，微炒），干姜半两（炮裂，锉），诃黎勒1两（煨，用皮），吴茱萸1分（汤浸7遍，晒干，微炒），甘草1分（炙微赤，锉）。
出处：《圣惠》卷七十五。
主治：妊娠冷气攻心腹痛，或不纳饮食。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加大枣3个，煎至6分，去滓，稍热服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
