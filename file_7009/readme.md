序号：7009
名称：四制鲮鲤丸
组成：归尾5钱，大黄2钱，荆芥2钱，桔梗2钱，乳香2钱（炙），没药（炙）2钱，黄芩3钱，连翘3钱，防风2钱半，羌活2钱半，全蝎1钱，蝉蜕20个（去头），僵蚕25个，牛皮胶1两（土炒），雄黄7分，金头蜈蚣4条（去头足，分作4样法制：1条用姜汁搽，焙干；1条用香油搽，焙干；1条用醋搽，焙干；1条用酥搽，炙），穿山甲4两（亦作4制：1两用红花5钱煎汤煮，焙干；1两用牙皂5钱煎汤煮，焙干；1两用紫草节5钱煎汤煮，焙干；1两用苏木5钱煎汤煮，焙干）。
出处：《种福堂方》卷三。
主治：一切无名肿毒，瘰疬。
加减：None
功效：None
用法用量：每服1丸，滚酒送下。
制备方法：上为细末，真米醋打糊为丸，每丸重1钱2分，朱砂1钱5分共为衣，瓷瓶收贮，瓶内放麝香5分以养之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
