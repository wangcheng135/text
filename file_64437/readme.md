序号：64453
名称：猫头丸
组成：猫头骨1个（酥炙），蝙蝠1个（以朱砂3钱填入腹内，瓦上炙焦），南星1两，白矾1两。
出处：《医学入门》卷八。
主治：瘰疬，马刀，不问远年近日，巳破未破。
加减：如风热实者，加防风、黄芩、山栀、蝉蜕、川芎、连翘、桔梗各5钱；虚者，加夏枯草2两；虚劳骨蒸，加玄参1两，胡黄连5钱；汗多，加牡蛎3钱；有咳，加麦门冬1两；血虚，加归、芍、生地；气虚，加参、术各1两；毒重，加雄黄；痛甚，加乳、没各2钱；坚硬，加海藻4钱；成漏，加穿山甲1两。
功效：None
用法用量：每服30丸，临卧米饮送下；便燥，用蜜为丸，空心及夜卧含化3丸。
制备方法：上为末，用黄蜡溶化为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
