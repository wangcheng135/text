序号：66858
名称：黄柏止泻汤
组成：黄柏3两，人参3两，地榆3两，阿胶3两，黄连5两，茯苓4两，榉皮4两，艾叶1升。
出处：《千金》卷二十。
主治：下焦虚冷，大小便洞泄不止。
加减：None
功效：None
用法用量：上（口父）咀，以水1斗，煮取3升，去滓，下胶消尽，分3服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：方中黄柏、榉皮皆苦寒清燥治热结肠胃下水断痢之剂，地榆亦苦涩微寒；专赖参、苓、阿胶平调气血以助诸药之力；惟艾性禀纯阳，能开发中外阴邪，必缘其人宿蕴湿热，暴感虚寒而洞泄不止，故以辛热之味行苦寒之性。湿热去而真阳复，三焦各司其职，可无下虚上盛之虞。否则，方下虚冷二字或有错误亦未可知。
用药禁忌：None
附注：None
