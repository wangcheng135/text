序号：3420
名称：太乙救苦万珍膏
组成：当归2钱，厚朴5钱，青皮2钱，丹皮3钱，白芍3钱，杏仁5钱，牛膝5钱，全虫3个，头翁2个，杜仲1钱5分，松节5钱，公英5钱，桔梗3钱，灵仙2钱，川连3钱，五灵脂2钱，海藻2钱，腹皮5钱，五味子3钱，榆白皮2钱，五加皮2钱，栀子5钱，槿皮2钱，桃仁3钱，甘草2钱，木通2钱，防风3钱，连翘5钱，蜈蚣1条，桑皮2钱，白芷2钱，木瓜3钱，枳壳3钱，桂心1钱，樟丹2两，香油5斤，山甲2钱，川芎3钱。
出处：《经验奇效良方》引凌文轩方。
主治：妇人久不生育；并治男女一切虚劳百损，腰膝疼痛，寒湿脚气，痰厥，刀伤热毒，远近头风，男子睾丸偏坠，手足冻伤，刀石磕碰，男子肾虚，小儿腹痛，手足麻木，噎膈反胃，胎前产后，胃口不开，牙齿疼痛，一切疔疮。
加减：None
功效：None
用法用量：每月贴1张，连贴3次。胃口不开，噎膈反胃；贴胃口；急慢惊风，贴命门穴；牙齿疼痛，贴面上；磕伤疼痛，贴患处；余皆贴肚脐。
制备方法：入油内浸5日，熬至滴水成珠，下丹。每贴重8钱。
临床应用：None
各家论述：None
用药禁忌：None
