序号：26281
名称：红灵丹
组成：明雄6钱，朱砂6钱，礞石6钱，火消6钱，月石6钱，麝香2分，洋片2分，佛金40张。
出处：《齐氏医案》卷六。
主治：感冒伤风，伤寒伤暑，痧胀，中恶中毒，心疼腹痛，哮喘痰嗽，牙痛，小儿急惊，五疳诸积，食伤饱胀，霍乱吐泻，时症瘟疫，痈疽疔毒疮疖，痰核痰疱，蜂螫虫咬，妇女月经不调。
加减：None
功效：None
用法用量：治感冒伤风，伤寒伤暑，用温茶送5丸；慢紧痧胀，稍冷茶下；中恶中毒，暴病五绝，将此丹水擦牙，下咽即活，重者3-5丸，勿过，过服冷水解；9种心疼、腹痛、哮喘、痰嗽，温茶送下；牙痛，碎1丸放痛处；小儿急惊，五疳诸积，食伤饱胀，霍乱吐泻，用3丸或2丸，放舌尖上，和津嚼之，见麻，冷水吞，寒症用温茶；时症瘟疫，沿门传染，用银簪点大眼角中，男左女右；治一切痈疽疔毒，阴阳疮疖，痰核痰疱，以及蜂螫虫咬，初起未陷，用葱头酒煎加蜜开擦，阳疮加猪胆汁擦，吞下3-5丸即消；妇女月经，或前或后，俱用黄酒送下5丸、7丸，取汗立效；佩之在身，不染瘟疫。
制备方法：各制合研极细末，瓷瓶收贮，勿令泄气，轻重量用；或烧酒、冷水为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：孕妇忌用。
附注：八宝红灵丹（《痧证汇要》卷一）、绛雪（《霍乱论》）、八宝红灵散（《慈禧光绪医方选议》）、红灵散（《中国药典》一部）。
