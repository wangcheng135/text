序号：80199
名称：漏芦汤
组成：漏芦（去芦头）1两，乌蛇（去皮骨，酒炙）1两，独活（去芦头）1两，黄耆（炙）1两，白蔹1两，白茯苓（去黑皮）1两，生姜（切，炒）1两，大黄（锉碎，醋炒）1两，升麻（生用）3分，麻黄（去根节，煎去沫，焙）3分，枳实（去瓤，麸炒）3分，芍药3分，防己3分，玄参3分，甘草（炙）3分，附子（炮裂，去皮脐）3分，栀子仁1两1分，石膏（碎）1两半。
出处：《圣济总录》卷十八。
主治：大风癞，身体成疮，眉鬓堕落，（疒咅）（疒畾）瘙痒，搔之黄水出者。
加减：None
功效：None
用法用量：每服5钱匕，以水1盏半，煎取1盏，去滓温服；要利，空心、临卧各1服。
制备方法：上锉如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
