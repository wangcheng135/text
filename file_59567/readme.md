序号：59582
名称：调中汤
组成：白术、干姜、白茯苓、甘草各等分。
出处：《阴证略例》。
主治：内伤寒。寒热间作，腕后有斑3-5点，鼻中微血出，两手脉沉涩，胸膈四肢，按之珠无大热。
加减：None
功效：None
用法用量：每服5钱，水1盏半煎。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：内伤寒，完颜小将军病寒热间作，腕后有斑3-5点，鼻中微血出，两手脉沉涩，胸膈四肢，按之珠无大热，此内伤寒也。问之，因暑卧殿角伤风，又渴饮冰酪水，此外感者轻，内伤者重，外从内病，俱为阴也，故先斑后衄，显内阴症，寒热间作，脾亦有之，非少阳之寒热也，与调中汤数服而愈。
用药禁忌：None
附注：None
