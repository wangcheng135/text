序号：26802
名称：延寿瓮头春
组成：天门冬（去心）1两，破故纸1两，肉苁蓉（麸炒）1两，粉草1两，牛膝（去芦）1两，杜仲（麸炒）1两，大附子（水煮）5钱，川椒（去目）1两（以上8味为末，入面内同和糜），淫羊藿1斤（米泔水浸），羯羊脂1斤（拌淫羊藿同炒黑色），当归4两，头红花1斤（捣烂，晒干），白芍1两，生地黄2两，苍术（米泔水浸，炒）4两，熟地黄2两，白茯苓4两，甘菊花1两，五加皮4两，地骨皮4两（以上12味，锉咀片，绢袋盛贮铺缸内），缩砂密5钱，白豆蔻5钱，木香5钱，丁香5钱（以上4味后用酒煮，为末用）。
出处：《寿世保元》卷四。
主治：None
加减：None
功效：通百窍，兴痿阳，延寿。
用法用量：每日量饮数杯。1-7日，百窍通畅，浑身壮热，丹田微痒，痿阳立兴。
制备方法：用糯米2斗，淘净浸1日夜；又淘1次，蒸作糜，取出候冷；用细面末4斤，同天门冬等8味调匀。却将淫羊藿等12味，贮于粗绢袋，置缸底，将前糜拍实于其上，然后投上品烧酒40斤，封固7日，榨出澄清，方入坛；加砂仁等4味固封，重汤煮3炷香，埋土中3日，能出火毒。
临床应用：None
各家论述：None
用药禁忌：切忌醉酒饱食行房。
附注：None
