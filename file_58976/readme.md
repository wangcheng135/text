序号：58991
名称：秦艽苍术汤
组成：秦艽（去苗）1钱，桃仁（汤浸，去皮，另研）1钱，皂角仁（烧存性，另研）1钱，苍术（制）7分，防风7分，黄柏（去皮，酒洗）5分，当归梢（酒洗）3分，泽泻3分，梭身槟榔1分（另研），大黄少许。
出处：《兰室秘藏》卷下。
主治：风热乘食饱不通，气逼大肠，致患痔漏，大便秘涩，必作大痛。
加减：如有白脓，加白葵花头5朵（去萼心），青皮半钱（不去白），入正药中同煎，木香3分（为细末）。
功效：None
用法用量：水3盏，煎至1盏2分，去滓，入槟榔等3味末，再上火煎至1盏，空心热服。待少时以美膳压之，不犯胃气也。
制备方法：上除槟榔、桃仁、皂角仁3味外，余药(口父)咀，如麻豆大，都作1服。
临床应用：None
各家论述：None
用药禁忌：服药日忌生冷硬物，及酒、湿面、大料物、干姜之类；忌桃、李、梅、杏果品、油、胡椒之类。
附注：None
