序号：32200
名称：扶桑至宝丹
组成：嫩桑叶（采数10斤，择家园中嫩而存树者，长流水洗，摘去蒂，晒干）、巨胜子。
出处：《寿世保元》卷四引胡僧方。
主治：眼目昏花，咳久不愈，肢麻便燥。
加减：None
功效：步健眼明，须白返黑。消痰生津，补髓添精。除风湿，润五脏。
用法用量：每服100丸，白开水送下，每日2次。3月之后，体生轸粟；此为药力所行，慎勿惊畏，旋则遍体光洁如凝脂然，服至半年之后，精力转生，诸病不作，久服不已，自登上寿。
制备方法：炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《医方集解》：此足少阴、手足阳明药也。桑乃箕星之精，其木利关节、养津液，其叶甘寒，入手足阳明，凉血燥湿而除风；巨胜甘平色黑，益肾补肝，润腑脏，填精髓。夫风湿去，则筋骨强；精髓充，则容颜泽，却病乌髭，不亦宜乎。
用药禁忌：None
附注：扶桑丸（《医方集解》）、桑麻丸《医级》卷八）。《医方集解》本方用嫩桑叶（晒干）一斤，巨胜子四两，白蜜一斤。
