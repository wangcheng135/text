序号：39133
名称：泽漆根汤
组成：生鲤鱼1头（重5斤，粗锉），麦门冬2两（去心），甘草2两（炙），人参2两，茯苓2两，泽漆根8两（生者）。
出处：《外台》卷二十引《深师方》。
主治：水在五脏，令人咳逆喘上气，腹大响响，两脚肿，目下有卧蚕，微渴，不得安卧，气奔短气，有顷乃复，小便难，少而数，肺病胸满引痛，水气迫肺，吸吸寒热。
加减：有寒，可纳干姜8两。
功效：利小便。
用法用量：以水1斗7升煮鱼，取1斗，去鱼以煮药，取4升，分服，1日3次，小便利为度，不利增服之；大便如利而小便未利者，增至4合。服1日气即下，得安卧。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
