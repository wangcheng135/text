序号：5426
名称：二丑夺命丹
组成：木通1两，香附（醋炒）1两，大黄1两，草果（炒）1两，芫花1两，槟榔1两，泽泻（去毛）1两，红芽大戟1两，小牙皂1两，甘遂1两，黑丑（炒）5钱，白丑（生用）5钱，雷丸3钱。
出处：《玉案》卷五。
主治：气蛊、血蛊，大小便不通，面足浮肿，肚大青筋，痰喘气急，饮食不进。
加减：None
功效：None
用法用量：每服2钱，白酒送下。泻3-4次。第2日服补脾丸药，第3日又服1钱5分。看行下何物，如血蛊血下，气蛊屁多，水蛊水多，食蛊粪多。服此药，如胀肿不消，以陈壁土煎水服之，即消散矣。
制备方法：上为末，以白酒浆同老米打糊为丸。
临床应用：None
各家论述：None
用药禁忌：忌盐、酱、发物、荤腥、房劳百日之外。
附注：None
