序号：26748
名称：延龄膏
组成：虾蟆、鹤骨、丁香、枣叶、鳗鲡、木香、猪牙皂角各等分，麝香（另研细）少许。
出处：《博济》卷一。
主治：一切劳。
加减：None
功效：None
用法用量：每日到辰、已间，用茅香熟水吞下100丸；更看病人强弱，渐渐服之。须得1度吃尽100丸后，以衣被盖之出汗，病甚者略露面，其虫逡巡后汗出，尸虫如麦麸大，余者皆微壮，出尽汗后，病人其身轻快，十去三四也。候汗干后，即一时换却原着衣服并卧物等。初服药，见当时随药便尽吐出，并不住，即难疗；或100丸存60-70丸，犹可医，别以新药补数，如药全住，疾无不退。病甚者，吃及2服至3服，即永除根本。治劳，煮散3-5服为妙。初出汗，频换手帕揩之，约两炊饭久，肠劳随泻下1度，别用盆1个盛之，其虫皆微壮，浮在上面动。服药后如不吐甚妙也。
制备方法：上为末，用羊肠（大者）盛之，缚定两头，于饭甑上蒸熟为度；取出候冷，以竹刀子割开，同肠研细，再入麝香，同研令匀为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：服药后忌一切动劳等物，一生永忌触犯药，苦参、人参、空青、麦冬、乌头切忌服之。
