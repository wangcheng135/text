序号：81865
名称：熟干地黄散
组成：熟干地黄1两，当归1两（锉，微炒），龙骨1两，人参1两（去芦头），甘草1两（炙微赤，锉），桔梗1两（去芦头），黄耆2两（锉），桂心1两，半夏3分（汤洗7遍去滑），茯神1两，远志半两（去心），枳壳1两（麸炒微黄，去瓤），白术半两。
出处：《圣惠》卷四。
主治：心气不足，恍恍惚惚，朝瘥暮甚，心中憧憧，胸满，不下食饮，阴阳气虚，脾胃不磨，不欲闻人声。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，枣3枚，白粳米50粒，煎至6分，去滓温服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：忌炙爆，热面。
附注：None
