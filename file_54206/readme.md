序号：54221
名称：大定心汤
组成：人参2两，茯苓2两，茯神2两，远志2两，龙骨2两，干姜2两，当归2两，甘草2两，白术2两，芍药2两，桂心2两，紫菀2两，防风2两，赤石脂2两，大枣20枚。
出处：《千金》卷十四。
主治：心气虚悸，恍惚多忘，或梦寤惊魇，志少不足。
加减：None
功效：None
用法用量：上(口父)咀。以水1斗2升，煮取2升半，分5服，日3夜2。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：本方即于小定心汤中加白术以理中气，辅桂心以和营血，更须龙骨、赤脂以镇心肝之怯，其余茯神，防风、当归、紫菀则又桂心、茯苓、芍药、大枣之佐也。
用药禁忌：None
附注：None
