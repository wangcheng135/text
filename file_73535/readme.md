序号：73551
名称：蛤蚧丸
组成：蛤蚧（酥炙）1对，胡黄连半两，知母（切，焙）半两，鳖甲（去裙襕，酥炙）半两，紫菀半两，桑根白皮（锉）半两，天门冬（去心，焙）半两，人参半两，黄耆（锉）半两，甘草（炙）半两，柴胡（去苗）半两，地骨皮半两，生干地黄（焙）半两，杏仁（汤浸，去皮尖双仁，炒）1分，细辛（去苗叶）1分。
出处：《圣济总录》卷八十七。
主治：热劳烦躁，面赤口干，骨节痠痛，夜多盗汗，咳嗽痰壅，力乏气促。
加减：None
功效：None
用法用量：每服20丸，食后卧时生姜汤送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
