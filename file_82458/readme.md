序号：82474
名称：蠲风饮子
组成：防风（去芦）1两，杜仲（去粗皮，姜汁炒）1两，羌活1两，白芷1两，川归（去芦头，酒浸洗）1两，川芎1两，生地黄（酒浸洗）1两，白芍药1两，川牛膝（去芦，酒洗）1两，秦艽（去芦）1两，何首乌1两，萆薢1两，苍术（米泔浸1-2宿）1两，白术1两，木通（去皮）1两，大枫子肉1两，威灵仙1两，血藤（即过山龙）1两，防己1两，丁公藤1两，荆芥穗5钱，海桐皮（去粗皮）5钱，五茄皮5钱，天南星（煨制）5钱，半夏（汤泡7次）5钱，橘红（去白）5钱，赤茯苓（去皮）5钱，桑寄生5钱，天麻5钱，僵蚕（炒）5钱，钩藤5钱，薄桂（去粗皮），草乌头（去皮尖）2钱半，甘草节2钱半，川乌（去皮脐，炮）2钱半，猪牙皂角2钱半，两头尖1两5钱，阴地蕨（一名地茶）1两5钱，大蓟1两5钱，小蓟1两5钱，理省藤1两5钱，桑络藤1两5钱，生姜1两（另研细）。
出处：《医学正传》卷一。
主治：中风瘫痪，口眼歪斜，及一切手足走注疼痛，肢节挛急，麻痹不仁等症。
加减：None
功效：None
用法用量：每日清晨、午前、午后、临卧各服1大白盏。
制备方法：上药切细。用无灰好酒2斗5升，以瓷罐1个盛酒浸药，以皮纸十数重包封罐口，冬半月，夏7日，秋、春10日。
临床应用：None
各家论述：None
用药禁忌：忌鸡、猪、鱼、羊、驴、马、飞禽、虾、蟹等肉味，及煎煿油腻、水果生冷、荞麦热面，一切动气发风之物。
附注：None
