序号：70161
名称：滋阴益肾丸
组成：熟地黄（酒浸，焙）6两，黄柏（酒浸，炒褐色）4两，菟丝子（酒蒸，焙）4两，牛膝（酒浸）3两，败龟版（酥炙黄）3两，虎骨（酥炙黄）3两，知母3两，白芍药3两，白术3两，山药3两，当归（酒浸）3两，枸杞子3两。
出处：《济阳纲目》卷六十四。
主治：虚损。
加减：如腰腿无力，加牛膝1两（酒浸），败龟版1两5钱（酥炙）。
功效：补元气，益肾水，降心火，生精补血，壮筋骨，悦颜色，益寿延年。
用法用量：每服70-80丸，空心淡盐汤送下。
制备方法：上为细末，地黄膏和炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
