序号：68713
名称：小沉香丸
组成：丁香1分，沉香1分，乳香1钱半，舶上茴香1两（炒），肉桂半两（去粗皮），槟榔2枚（冬春1枚），肉豆蔻5枚（夏加2枚），荜茇5钱，阿魏少许，巴豆15颗（去皮，不出油，别研）。
出处：《普济方》卷一八四。
主治：胸膈气不和，脏腑冷气，上攻迷闷。
加减：None
功效：消滞气，顺三焦，空胸膈，理脾元，兼化酒食毒。
用法用量：每服5丸，生姜汤送下。如胸膈气不和，脏腑冷气，上攻迷闷，加10丸，温酒送下。常服清茶送下，要微利，以意加服之。
制备方法：上为末，研入巴豆、阿魏令匀，煮白米饭为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
