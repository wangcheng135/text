序号：71282
名称：硫黄饼
组成：矾制硫黄1两。
出处：《医学入门》卷八。
主治：虫疮及冷疮，喜就火炙汤泡者。
加减：如退热，治干痒出血，须用黄芩、黄连、大黄，或松香，樟脑；退肿止痛，须用寒水石、白芷；止痒杀虫，用狗脊、蛇床子、枯矾；杀虫，用芜荑、水银、硫黄，甚者加藜芦、斑蝥；干脓，用无名异、松皮炭；头疮，加黄连、方解石；脚上疮，加黄柏；阴囊痒，加吴萸。
功效：杀虫止痒。
用法用量：临用先以柳、桃、桑、槐、楮五枝煎汤洗拭，然后用麻油调硫黄末搽之；如干疮，用猪油调搽。
制备方法：上为末，用水调成饼，贴瓷碗底，覆转，用蕲艾1两，川椒3钱，为末，火燃熏干硫黄。
临床应用：None
