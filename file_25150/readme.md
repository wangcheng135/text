序号：25158
名称：如意油
组成：防己6两，荆芥8两，香附4两，细辛10两，苦杏仁4两，沉香4两，藿香4两，桂枝10两，乳香6两，甘草8两，木香4两，丁香8两，大黄2两，血竭8两，冰片10两，棉油2斤半，樟脑油10斤，薄荷油6斤，檀香油1斤半，丁香油1斤。
出处：《全国中药成药处方集》（武汉方）。
主治：风湿头痛，关节痛，以及腹痛。
加减：None
功效：None
用法用量：外用搽涂患处，少量可以内服。
制备方法：先取棉油、樟脑油，合群药浸一星期后，再用蒸汽加温至七十度，经十八小时，冷后过滤去滓，再加薄荷油、檀香油、丁香油混合摇匀，成油17斤13两。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
