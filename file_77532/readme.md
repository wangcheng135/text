序号：77548
名称：干姜丸
组成：干姜（炮）1两半，吴茱萸（汤洗，焙，炒）1两半，附子（炮裂，去皮脐）1两半，黄芩（去黑心）1两，蜀椒（去目并合口，炒出汗）1两，熟干地黄（焙）1两，当归（切，焙）1两，大黄（锉，炒）1两，桂（去粗皮）1两，白术1两，赤芍药半两，人参半两，石韦（去毛）半两，桃仁（汤浸，去皮尖双仁，炒黄）35枚，薏苡仁2两。
出处：《圣济总录》卷一五一。
主治：妇人月水不调，绕脐痛，手足烦热，两脚痠疼。
加减：None
功效：None
用法用量：每服20丸，温酒送下，1日2次。未知稍加，以知为度。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
