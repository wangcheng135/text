序号：48071
名称：神妙五枝膏
组成：川乌6钱，草乌6钱，防风6钱，白芷6钱，当归6钱，熟地黄6钱，木鳖子（去壳）6钱，穿山甲6钱，大黄6钱，甘草6钱，槐1寸，桃1寸，柳1寸，椿1寸，楮（用枝）1寸，血余1握。
出处：《古今医鉴》卷十六。
主治：腰痛，咳嗽，痞块及诸般疮毒。
加减：None
功效：None
用法用量：腰痛贴痛处；咳嗽贴肺俞2穴；痞块贴块上；诸般疮毒随大小贴之。
制备方法：上锉，用香油1斤，入药用文武火煎至焦枯，滤去滓，将油再煎，随入黄丹（炒见火星为度）半斤，柳条搅不住手，滴水成珠为度，去火略待少时，入乳香1两、没药6钱、朱砂2钱、轻粉2钱，亦徐徐搅入内，倾碗中，坐水出火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
