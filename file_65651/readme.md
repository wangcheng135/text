序号：65667
名称：菊霜
组成：防风2钱，羌活2钱，石膏2钱，川芎2钱，川黄连2钱，荆芥2钱，元参2钱，甘草2钱，黄柏2钱，槐角2钱，连翘2钱，黄芩2钱，甘菊花2钱，薄荷2钱，白芷2钱。
出处：《玉钥续编》。
主治：风火牙痛。
加减：None
功效：None
用法用量：每用3-5厘擦风处，以涎出为度，擦过3次可保永不再发。
制备方法：上为细末，另将甘草5钱煎水，入药拌匀，质要干湿相得，放铜勺内，再用潮脑6钱，匀洒药上，净碗盖好，盐泥固封，微火升三炷香，切忌武火，恐其焦灼，升足取碗底白霜，瓷瓶收紧，勿使见风走气，升过药仍可拌甘草水，加潮脑，依法再升一次。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
