序号：450
名称：五疰汤
组成：大黄3两（别渍），甘草2两（炙），乌头10枚（炮，削皮），生姜半斤，桂心4两，芍药2两，当归2两，蜜1斤。
出处：《外台》卷十三引《小品方》。
主治：卒中贼风，遁尸鬼邪，心腹刺痛，大胀急。
加减：None
功效：None
用法用量：上切。以水9升，煮取3升，乌头别纳蜜中煎，令得1升，投着汤中；去滓，分服3合，如人行30里，又1服，1日3次。不知，可至4合。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：五疰汤中用乌头、姜、桂辛烈，祛辟阴邪，即兼归、芍护持营血，甘草、蜂蜜解毒安中。除邪养正之大略也。
用药禁忌：忌海藻、菘菜、猪肉、生葱。
附注：None
