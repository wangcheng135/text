序号：35844
名称：参术饮
组成：人参3钱，黄耆2钱，白术1钱，甘草（炙）5分，白豆蔻8分，陈皮7分，木香1钱，地榆1钱，升麻1钱，粟壳8分，当归（全用）1钱，砂仁7分，泽泻8分。
出处：《杏苑》卷四。
主治：中气下陷，脾胃虚败，久痢不止，滑脱不固，四肢倦怠，精神短少，饮食不消。
加减：None
功效：补中益气，温养脾胃。
用法用量：上(口父)咀。水煎，温服。
制备方法：None
临床应用：None
各家论述：用人参、白术、炙甘草等补中益气，砂仁、白蔻温脾和胃，陈皮、木香行滞气，地榆凉下焦之血，泽泻利小便以渗湿，升麻升提清气上腾，佐罂粟壳以固滑脱，当归分理气血，各归其所。
用药禁忌：None
附注：None
