序号：4357
名称：九宝散
组成：大腹并皮1两，肉桂1两，甘草（炙）1两，干紫苏1两，杏仁（去皮尖）1两，桑根白皮1两，麻黄（去根）3两，陈皮（炒）3两，干薄荷3两。
出处：《苏沈良方》卷五。
主治：感风伏热，肺气壅滞，咳嗽喘急，积年累发；或时下感冒，鼻塞流涕，一切咳嗽喘急；小儿因伤寒邪，不曾解利，致成远年嗽。
加减：None
功效：None
用法用量：每服10钱匕，用水1大盏，童便半盏，加乌梅2个，生姜5片，同煎至1中盏，滤去滓，食后、临卧服。
制备方法：上为粗末。
临床应用：喘证：两浙张大夫，病喘二十年，每至秋冬辄剧，不可坐卧，百方不愈，后得临平僧法本方服之遂愈。法本凡病喘三十年，服此药半年，乃绝根本，永不复发。凡服此药须久乃效。
各家论述：None
用药禁忌：虚劳自汗不可服。
附注：九宝汤（《易简方》）、九宝饮（《洪氏集验方》卷五）、苏沈九宝汤（《证治要诀类方》卷一）、苏陈九宝汤（《医统》卷四十四）。
