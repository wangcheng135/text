序号：74560
名称：新制蠲痹汤
组成：苍术1钱半，羌活1钱半，白术1钱半（炒），茯苓1钱半，官桂1钱半，泽泻1钱半，甘草5分。
出处：《医略六书》卷二十。
主治：寒湿袭虚，关节不利，故身体沉重痠疼，天阴转甚，脉弦缓者，谓之冷痹。
加减：None
功效：None
用法用量：水煎，去滓，冲竹沥、姜汁各数匙热服。
制备方法：None
临床应用：None
各家论述：苍术燥湿强脾，甘草缓中和胃；羌活散寒湿，关节可利；白术健脾土，营卫可行；茯苓渗湿清脾肺，泽泻泻湿利膀胱，务使外着之湿自无复留之地。更以官桂温营散寒，竹沥、姜汁行经活络也。俾经府交通，则三阳皆泰而寒湿并散，身重痠疼，无不自退，阴晦乃天道之常，岂必病人乎。此温经燥湿祛寒之剂，为身重痠疼冷痹之专方。
用药禁忌：None
附注：None
