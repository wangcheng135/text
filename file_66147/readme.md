序号：66163
名称：象皮膏
组成：真象皮2两（无真者，则以驴、马剔下之爪甲代之，可用4-5两），当归（全）2两，壮年人发（洗净垢）2两，大生地4两，玄武版4两，真麻油3斤。
出处：《疡科纲要》卷下。
主治：顽疮久不收口，脓水浸淫，浮皮湿痒，并不深腐之症；足胫湿臁久年不愈者。
加减：None
功效：None
用法用量：油纸摊贴，量疮口大小为度，外以布条轻轻缠之，2日1换。脓水少者，3-4日1换。此膏亦可摊于西法之脱脂棉纱上，较用油纸者易于收湿长肉。
制备方法：上先煎生地、龟版、象皮，后入血余、当归，熬枯去滓；入黄蜡、白占各6钱，川连汁煅制上炉甘石细末半斤，生石膏细末5两，文火上调匀，不煎沸，瓷器密收。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
