序号：47737
名称：神仙百解散
组成：白术2两，茯苓2两，藿香叶（去土）3两，橘皮（去瓤）3两，甘草（炙）3两，半夏（擘破，生姜汁制）3两，厚朴（姜汁蘸炙）3两。
出处：《普济方》卷一四七引《卫生家宝方》。
主治：四时伤寒，八般痞气，山岚瘴疟，浑身壮热憎寒，或中暑，或风疾灌注，曲挛手足，咽喉噎塞，十种膈气，不思饮食，冷物伤脾，脏腑不调，妇人产前产后及小儿一岁有病。
加减：None
功效：常服宽中进食。
用法用量：每服4钱，水1盏半，加生姜5片或7片，煎至7分，去滓热服，伤寒并吃2服。无论百病，初觉意思不快，便先进1-2服。
制备方法：上先将厚朴、半夏为粗末，用生姜4两烂研，同厚朴、半夏一处拌匀，于净器中淹1宿，次日焙干，却入前5味药，共为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
