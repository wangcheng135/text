序号：68697
名称：黄连炉甘石散
组成：炉甘石1斤，黄连4两，龙脑量入。
出处：《原机启微》卷下。
主治：眼眶破烂，畏日羞明。
加减：奇经客邪之病，量加朴消泡汤滴眼瘀肉黄赤脂上。
功效：None
用法用量：每用少许，井花水调如稠糊，临睡以箸头蘸敷破烂处；不破烂者，点眼内眦，锐眦尤佳。
制备方法：先以炉甘石置巨火中，煅通红为度；另以黄连用水1碗，瓷器盛贮，纳黄连于水内，却以通红炉甘石淬7次，就以所贮瓷器置日中晒干，然后用黄连研为细末；欲用时以1-2两再研极细，旋量入龙脑。
临床应用：None
各家论述：方以炉甘石收湿除烂为君，黄连苦寒为佐，龙脑去热毒为使。诸目病者俱可用，病宜者治病，不宜者无害也。
用药禁忌：不宜使入眼内。
附注：None
