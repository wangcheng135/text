序号：58092
名称：益寿比天膏
组成：附子（去皮脐）1两，牛膝（去芦）1两，虎胫骨（酥炙）1两，蛇床子1两，菟丝子1两，川续断1两，远志肉1两，肉苁蓉1两，天门冬（去心）1两，麦门冬（去心）1两，杏仁1两，生地1两，熟地1两，官桂1两，川楝子（去核）1两，山茱萸（去核）1两，巴戟（去心）1两，破故纸1两，杜仲（去皮）1两，木鳖子（去壳）1两，肉豆蔻1两，紫梢花1两，谷精草1两，川山甲1两，大麻子（去壳）1两，鹿茸1两，甘草2两（净末，看众药蕉枯方下），桑枝7寸，槐枝7寸，柳枝7寸。
出处：《回春》卷四。
主治：下元虚冷，五劳七伤，半身不遂，或下部虚冷，膀胱病症，脚膝痠麻，阳事不举。赤白带下，沙淋血崩，疮疖。
加减：None
功效：添精补髓，保固真情；善助元阳，滋润皮肤，壮筋骨，理腰膝，通二十四道血脉，坚固身体，返老还童。
用法用量：每1个贴60日方换。
制备方法：上锉细。用真香油1斤4两浸1昼夜，慢火熬至黑色；用飞过好黄丹8两，黄香4两入内，柳棍搅不住手；再下雄黄、倭硫、龙骨、赤石脂各2两，将铜匙挑药滴水成珠，不散为度；又下母丁香、沉香、木香、乳香、没药、阳起石、煅蟾酥、哑芙蓉各2钱，麝香1钱为末，共搅入内；又下黄蜡5钱。将膏贮瓷罐内，封口严密，入水中浸5日去火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
