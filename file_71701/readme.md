序号：71717
名称：琥珀定智丸
组成：南星半斤（先将地作坑，用炭10斤在坑内烧红，去灰净，用好酒10余斤倾在坑内，大瓦盆盖覆，周围以炭火拥定，勿令泄气，次日取出，为末），真琥珀1两（皂角水洗去油），大朱砂2两（公猪心割开，入内，用线缚住，悬胎煮酒2碗），干人乳（用姜汁制），好拣参（去芦）3两，白茯苓（去皮）3两，白茯神（去皮木）3两，石菖蒲2两（猪胆汁炒），远志（水泡过，去心）2两（猪胆煮过，晒干，再用姜汁制）。
出处：《古今医鉴》卷八引刘尚书方。
主治：惊战虚弱，气乏之病。
加减：None
功效：补心生血，定魄安魂，扶肝壮胆，管辖神魂。
用法用量：每服50-70丸，每夜卧时盐汤送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：琥珀定志丸（《回春》卷四）。晒干人乳法：用人乳数碗，入瓦盘内，莫搅动，四围晒干刮一处，干则再刮。乳干以姜汁拌晒用。方中干人乳用量原缺。
