序号：47330
名称：祛风导气化痰丸
组成：大川乌头半两重（炮，去皮脐用），乌迭泥3两，天南星3两，白附子3两（生用），天麻（酒浸，焙）1两，全蝎1两（不去毒，用薄荷叶炒），半夏7两。
出处：《普济方》卷九十四引《瑞竹堂方》。
主治：咳嗽气积，呕吐痰涎，头目昏晕，半身不遂，偏废，口眼斜，他药不疗者。
加减：None
功效：None
用法用量：每服50-70丸，食后临卧用姜汤送下，1日2次。药只阴干，不要晒干及焙。
制备方法：上各取净末，除南星、半夏生取末，以绢袋盛，安瓷器中，新水浸，日晒夜露3昼夜，每早换水，3日取出，晒干为细末，各依分两称，相和重罗，用苏合香油3两，如无苏合香丸，膏子亦可。如不敷，入糯米粉打薄糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
