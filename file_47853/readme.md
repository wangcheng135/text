序号：47868
名称：神功助化散
组成：地萹蓄5钱，瞿麦穗5分，大麦糵5钱，神曲2钱半，沉香1钱半，木香1钱半，甘草5分，大黄2两。
出处：《医学纲目》卷二十五引罗知悌方。
主治：腹中痞块，不拘气血食积所成。
加减：None
功效：None
用法用量：男以甘草、淡竹叶2味等分煎汤，妇人用红花、灯心、当归等分煎汤，黄昏时无灰酒同调服，且酒多于汤。大小便见恶物为度。
制备方法：上为细末，依分两和匀。
临床应用：None
各家论述：None
用药禁忌：忌油腻、动气之物，及房事一月。
附注：太无神功散（《医医偶录》卷二）。本方方名，《医灯续焰》引作“太无神功助化散”。改为丸剂，名“神功助化丸”（见《中国医学大辞典》）。
