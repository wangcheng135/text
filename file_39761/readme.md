序号：39771
名称：经验贴癖膏
组成：阿魏3钱，蜈蚣2条（炙），全蝎3钱（炙），硼砂3钱，血竭3钱，栀子2两（为末，5两亦得），大黄2两，芦荟3钱，雄黄2钱，胡黄连2钱，硇砂3钱（上药，俱要真正者，研细听用）。
出处：《便览》卷三。
主治：积聚癖块。
加减：None
功效：None
用法用量：每贴3钱或5钱，摊于生布上，外加布裹在病上，用一年老耐心人，昼夜常常用熨斗盛微火，慢慢熨之。每一贴贴一昼夜，待3-5日再一贴。待大便见脓血，见效。消后须服补药。曾经针灸过及病势大者，难治。
制备方法：用蜂蜜5钱，皮消2两，萝卜汁2两，黑狗脑子1个，滚发酒糟2两，葱白汁2两，鸡子清2个，各汁合前末药和成膏子。
临床应用：None
各家论述：None
用药禁忌：忌发物。
附注：None
