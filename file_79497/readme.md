序号：79513
名称：碧玉膏
组成：蓖麻仁（去皮尖，捣烂）49粒，杏仁（去皮，捣烂）49粒，铜绿2两7钱（用水1碗，将铜绿研细，投入水中，搅匀），片松香5斤（研细）。
出处：《疡医大全》卷七。
主治：痈疽发背，瘰疬马刀，乳痈乳岩，流火流注，肿块风毒，横痃痔漏，囊痈，冬瓜痈，贴骨疽，一切腰背臀腿毒疖，多骨疽，蟮拱头，脚隐漏蹄。
加减：None
功效：活血止痛，拔毒消肿，敛毒透脓，去腐生新。
用法用量：用时以热汤炖化，摊贴。
制备方法：用真麻油12两，入锅内熬滚，次下蓖麻、杏仁，熬至滴水成珠为度，夏布滤去滓，将油复入净锅内，用文武火熬滚，徐徐投下松香末，用桃槐枝不住手搅匀，倾入瓷盆内，候膏将凝，然后加水浸之，用手揉扯以去火毒，另用瓷罐或桐杓盛贮数月。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
