序号：21250
名称：至圣丸
组成：附子（炮裂，去皮脐）1两，牛膝（去苗，酒浸，焙干）1两，海桐皮（锉）1两，肉苁蓉（酒浸，切，焙）1两，防风（去叉）1两，萆薢1两，狗脊（黑者，去毛）1两，黄耆（锉）1两，蒺藜子（沙苑者，炒）1两，茴香子（舶上者，炒）1两，威灵仙1两，续断1两，木香1两，骨碎补4两，木鳖子（去壳）2两，乳香（研）半两，没药（研）半两。
出处：《圣济总录》卷十。
主治：风气身体疼痛，筋脉拘急，手足麻木，睡卧多涎，及丹田虚冷。
加减：None
功效：None
用法用量：每服20丸．空心嚼，木瓜酒送下。
制备方法：上药除研者外，为细末，再同和匀，以酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
