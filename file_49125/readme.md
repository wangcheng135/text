序号：49140
名称：胜金丸
组成：熟地5两，人参1两半，白术1两（炒），茯苓1两半，当归3两，白芍1两半（酒炒），川芎1两，桂心1两半，香附2两（酒炒），白薇1两（酒炒），延胡1两半（酒炒），炙草5钱。
出处：《医略六书》卷二十七。
主治：经迟不孕，脉虚者。
加减：None
功效：None
用法用量：每服3-5钱，温酒送下。
制备方法：上为末，炼蜜为丸。
临床应用：None
各家论述：熟地补血，以滋冲任；当归养血，以营经脉；川芎行血海；白芍敛阴血；人参扶元补气，兼通血脉；白术健脾统血，鼓运脾元；茯苓渗湿，以清血室；炙甘草缓中，以益胃气；延胡化血滞，力擅通经；香附解气郁，专主调经；桂心补火暖血；白薇降泄益阴。蜜以丸之，酒以行之，使血气内充，寒滞解散，而经气温暖，腹痛无不自退，岂有经迟不孕之患乎？
用药禁忌：None
附注：None
