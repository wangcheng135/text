序号：50582
名称：茸砂丹
组成：辰砂1两，草乌1两，瞿麦1两，黄药子1两。
出处：《秘传证治要诀·类方》卷四引魏氏方。
主治：肾厥头痛，因虚而致，服诸药不效，其痛愈甚,肾虚火炎头痛，必先眼黑头旋。
加减：None
功效：None
用法用量：瓷碗1只，以姜汁涂炙数次，入砂在内，上铺余药，复以盏盖了，掘1小坑，安碗在内，用熟炭火5斤，煅令火尽，吹去草乌药灰，取辰砂研末，或只独用辰砂末，每服1钱半，淡姜汤送下；或加鹿茸，爁去毛，切片，酒浸，为末，3两枣肉，丸如梧桐子大。每服30-40丸，人参汤送下。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：熟砂有毒，更宜斟酌。
附注：本方方名，《准绳·类方》引作“茸朱丹”。
