序号：48656
名称：神授东华益算膏
组成：五枝膏2两，净沥青1斤，净黄香半斤，乳香末1两，没药1两，轻粉2钱，黄蜡1两，血竭末2钱，麝香末1钱，安息香末1两，黄丹1两，瓜绿末2两（极细）。
出处：《(月翟)仙活人方》。
主治：一切无名恶疮，诸药不效者。
加减：None
功效：None
用法用量：用时以绯红绵帛可疮大小津唾摊贴，勿留口，不见火；如贴脑疽、发背溃烂之处，先用槐枝、葱白煎汤洗净患处，而后摊贴，3-5日1换。
制备方法：先熬五枝膏（以桃枝、柳枝、槐枝、榆枝、桑技、枸杞皮各5升，锉碎，用长流水一担，同熬至5分，去滓，加当归末4两，慢火熬成膏，滴水中不散为度），将川芎、白芷入油同煎（春夏用油4两，秋冬用油6两），油熟去药，下沥青、黄香、黄蜡溶开，次下五枝膏，用槐枝搅二百余次，下乳香、没药、血竭、轻粉、安息香、黄丹，再搅二百余次，下麝香、瓜绿，再搅三百遍，滴水盆内浮者为度，将药倾于水盆内，浮者似青荷叶为度，沉香色者再熬，拔扯二百余遍，掿成鸡子大块，水盆内浸1宿，捞出控干，用纸托盘内放之，冬温处、夏凉处。
临床应用：None
各家论述：None
用药禁忌：煎熬此药，勿犯铁器。
附注：None
