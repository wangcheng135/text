序号：47747
名称：神仙返魂丹
组成：丹砂3两，雄黄3两，雌黄3两。
出处：《圣济总录》卷二○○。
主治：None
加减：None
功效：起死扶衰，延龄却老。
用法用量：若有人猝死者，安在口中即活；常人服者，每月旦日服1丸，可终生无病。若中毒药，其毒自吐出。
制备方法：上先细研雄黄。雌黄2味，用竹筒盛，埋北方地中，3日取出，先倾出一半，当心置丹砂（亦研令细），却用倾出一半盖之，又用云母粉（用绢袋子揉洗出，晒干）塞筒子口，埋地内，只露筒口，用炭火于筒口畔围之，缓缓烧21日，养化成水为度，倾药水盛铜器内，垂井中5日，无井只用深盆中盛水3-4寸，更置大豆半斤在内，即安药器于水上，浸5日亦得，然后逐夜于露下露之，日出时收，候药水赤色，可丸即丸，如胡麻子大，密收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
