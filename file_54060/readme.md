序号：54075
名称：柴胡加大黄芒消桑螵蛸汤
组成：柴胡2两16钱，黄芩1两，人参1两，甘草（炙）1两，生姜1两（切），半夏1合（洗），大枣4枚（擘），芒消2两。
出处：《千金翼》卷九。
主治：伤寒不解，胸胁满而呕，日晡所发潮热而微利，服柴胡加芒消汤不解者。
加减：None
功效：None
用法用量：前7味，以水7升，下芒消3合，大黄4分，桑螵蛸5枚，煮取1升半，去滓，温服5合，微下即愈。
制备方法：None
临床应用：None
各家论述：《古方选注》：用柴胡汤，其邪必从少阳而来，热及于阳明者，加芒消；热实于阳明者，加大黄。其邪入阳明，而后可议下。然里虚之应下者，加芒消当佐人参以安中，若加大黄，当佐桑螵蛸固阴续绝以安下，此少阳而有阳明症者，下之之方也。
用药禁忌：None
附注：None
