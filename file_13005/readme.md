序号：13012
名称：石决明丸
组成：石决明1两（捣，细研，水飞过），黄连3分（去须），玄参3分，地骨皮3分，防风3分（去芦头），栀子仁3分，子芩3分，独活3分，茯神3分，甘菊花3分，车前子3分，青葙子3分，枳壳3分（麸炒微黄，去瓤），秦艽3分（去苗），五加皮3分，决明子3分（微炒），葳蕤3分，沙参3分（去芦头），蕤仁3分（汤浸，去赤皮），川大黄3分（锉碎，微炒），茺蔚子3分。
出处：《圣惠》卷三十二。
主治：肝脏热极，目赤涩痛，泪不止，风湿痒，心膈壅滞，头目常疼。
加减：None
功效：None
用法用量：每服20丸，以薄荷汤送下，不拘时候。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
