序号：42315
名称：青黄散
组成：青鱼胆1钱，黄瓜霜1钱，梅花、冰片1分。
出处：《梅氏验方新编》卷一。
主治：喉风，喉闭，一切喉疮、喉毒。
加减：None
功效：None
用法用量：吹时俟喉中流吐痰涎即愈。
制备方法：上为极细末。用瓷瓶收贮，勿令泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：制青鱼胆法：冬月取大青鱼胆，每个入糯米数粒在内，勿将胆中苦水倾出，挂在背阴处风干，听用。制黄瓜霜法：拣老黄瓜，用竹刀将瓜蒂切下，挖去瓜子、瓜瓤，用皮消贮满瓜内，仍将瓜蒂盖上栓好，挂在有风无日处，待霜结瓜外，扫下听用。倘瓜烂无霜，将消倒出，仍可换瓜重制，每制必须多备几条，恐防瓜烂。
