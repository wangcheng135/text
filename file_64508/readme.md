序号：64524
名称：理中茯苓汤
组成：白术3钱，干姜3钱，炙甘草3钱，人参3钱，茯苓3钱。
出处：《东垣试效方》卷四。
主治：中焦寒湿。
加减：None
功效：None
用法用量：每服2钱，水1盏半，煎至1盏，冰之令寒服之。
制备方法：上为细末。
临床应用：中焦寒湿：戊申春，一妇人六十岁。病振寒战慄，呵欠嚏喷，口亡津液，心下急痛而痞，身热近火，济下恶寒，浑身黄白睛黄，溺黄赤而黑频数，自病来身重如山便着床枕，其脉诊得左右关并尺命门中得弦而急极细，杂之以洪而极缓，左尺按之至骨，举指来实者，六脉按之俱空虚，先以轻剂去其中焦寒湿，兼退其洪大脉，理中汤加茯苓是也。
各家论述：冰之令寒服之，谓之热因寒，用其寒以对足太阳之假热也，以干姜之辛热以泻真寒也。故曰真对真．假对假。
用药禁忌：None
附注：理中加茯苓汤（《医学纲目》卷二十一）。
