序号：8501
名称：四虫丹
组成：芙蓉叶1斤，紫地丁1斤，千金子10两（去油壳），桑虫2两（炙干），活桑1两（晒干或炙干），姜汁半斤，蒜汁半斤，葱汁5两。
出处：《种福堂方》卷四。
主治：诸般疔疮发背，一应恶疮。
加减：None
功效：None
用法用量：临用时将井水化开，围患处，如火之热，其毒即时消退，可收下再治后人。如不煎膏，将前药晒干，洒烧酒，再晒再洒，酒尽为度，作末收藏。临用时筛细，以并水调围亦妙。
制备方法：上用阴阳水4斤，煎至半斤，去滓，再用红蛄3两，麝香3钱，雄黄1两（研），蜈蚣1两（研），烧酒3两，盛倾银罐内，将铁油盏盖定，炭火升过，候酒尽即起；再用烧酒1斤，并后五味入药内，熬成膏子，用瓷器收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
