序号：53915
名称：捉疳丸
组成：虾蟆1只，芦荟5钱，胡连5钱，香附5钱，地骨皮5钱，枳壳4钱，黄芩4钱，青皮4钱，胆草4钱，黄连3钱，陈皮3钱，人参3钱，芜荑3钱，柴胡3钱，砂仁3钱，广木香2钱，甘草2钱，神曲4钱，鸡心槟榔4钱。
出处：《幼科指掌》卷四。
主治：小儿恣食生冷油腻，损伤脾胃，致成脾疳，久则皮肤干燥，肢体消瘦，胸胁结块，面目浮肿，身热脚细，肚大硬胀，吐逆中满，水谷不消，泻下酸臭，尿如米泔，夜热日凉，困睡口渴。
加减：None
功效：None
用法用量：每服30-50丸，米饮送下。
制备方法：上为末，猪胆汁和蒸饼为丸，如黍米大，青黛为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
