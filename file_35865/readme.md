序号：35875
名称：参术散
组成：白术（用里白无油者去芦，刮去皮，炒）1两，人参5钱，白茯苓（去皮）5钱，砂仁5钱，炙甘草（去皮）5钱，薏苡仁（炒熟，拣净）5钱，家莲子（去心，炒）5钱，真神曲（炒）5钱，山楂肉5钱，肉豆蔻（面裹煨熟，去面切细，用火纸包，打去油净）4钱，诃子（煨，取肉去核）4钱，陈广皮（洗净，去筋膜，晒）4钱，南木香3钱。
出处：《活幼心法》卷八。
主治：小儿脾虚吐泻交作，其泻每日只溏粪1-2次，屡作不止。痘色淡白，脾胃虚寒，大便泄而青白滑利不止者。
加减：None
功效：None
用法用量：每用2钱，清米饮调，食前温服。儿有不肯服者，入稀粥内和服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
