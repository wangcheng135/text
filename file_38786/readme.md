序号：38796
名称：三和丸
组成：枳实（麸炒）2两，槟榔2两，半夏（汤洗）2两，木香1两半，青皮（去白）1两半，陈皮（去瓤）1两半，赤茯苓（去皮）1两半，丁香皮1两半，萝卜子（炒）1两半，白术1两半，京三棱4两，蓬莪术3两，白豆蔻仁1两，沉香1两，桂（去粗皮）1两，藿香1两，黑牵牛1斤（微炒，捣细，头末取半斤）。
出处：《御药院方》卷三。
主治：三焦不和，气不升降，心胸痞闷，胁肋疼痛，因伤冷物，传化失常。
加减：None
功效：None
用法用量：每服30丸至50丸，食后生姜汤送下。
制备方法：上为细末，酒、面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
