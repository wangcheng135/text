序号：27399
名称：补肾虚磁石丸
组成：磁石1两（烧令赤，以醋淬7遍，捣碎，水飞过），鹿茸1两半（去毛，涂酥炙微黄），人参1两（去芦头），黄耆1两（锉），白茯苓1两，远志3分（去心），附子3分（炮裂，去皮脐），牡蛎3分（烧为粉），牛膝1两（去苗），楮实子1两半（水淘去浮者，焙干），防风3分（去芦头），肉苁蓉3分（酒浸1宿，刮去皴皮，炙干），五味子半两，薯蓣3分，巴戟2分，石斛1两（去根，锉），桂心3分，熟干地黄2两。
出处：《圣惠》卷三十。
主治：虚劳肾脏乏弱，耳聋，或常闻钟磬风雨之声。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
