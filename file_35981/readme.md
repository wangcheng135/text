序号：35991
名称：参苓丸
组成：人参1两1分，赤茯苓（去黑皮）1两1分，细辛（去苗土）1两1分，枳壳（去瓤麸炒）1两1分，熟干地黄（焙）1两1分，当归（切，焙）1两1分，麦门冬（去心，焙）1两1分，附子（炮裂，去皮脐）1两1分，干姜（炮）1两1分，大黄（锉，炒）1两1分，厚朴（去粗皮，涂生姜汁炙）1两1分，桂（去粗皮）1两1分，甘草（炙，锉）1两1分，乌头（炮裂，去皮脐）1两，桔梗（炒）1两，紫菀（去苗土）1两，蜀椒（去目共闭口，炒出汗）1两。
出处：《圣济总录》卷七十二。
主治：积聚胀闷，减食黄瘦。
加减：None
功效：None
用法用量：每服7丸，空心温酒送下，1日3次。渐加至10丸，以知为度。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
