序号：39883
名称：三肾丸
组成：人参（去芦）16两，当归160两，鹿鞭子（代睾丸）5条，枸杞子80两，白术（炒）160两，川芎16两，驴鞭子（代睾丸）5条，杜仲炭160两，茯苓160两，生地160两，狗鞭（代睾丸）30条，续断160两，甘草16两，白芍80两，鹿茸（去毛）64两，川牛膝160两，葫芦巴（炒）160两，巴戟肉（炙）80两，补骨脂（炒）80两，锁阳160两，冬虫夏草48两，青盐64两，小茴香（盐炒）160两，菟丝子160两，韭菜子160两，肉苁蓉（炙）160两，五味子（炙）16两。
出处：《北京市中药成方选集》。
主治：肾水亏损，阳痿不举，命门火衰，精神疲倦。
加减：None
功效：滋阴益气，补肾壮阳。
用法用量：每服1丸，日服2次，温开水送下。
制备方法：以上除3鞭外，重2640两，用黄酒1600两下罐蒸（包括3鞭），蒸后晒干。共研为细末，过罗，炼蜜为丸，重3钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
