序号：81357
名称：鳖甲汤
组成：鳖甲（去裙襕，酥炙）1两，柴胡（去苗）1两，附子（炮裂，去皮脐）1两，白茯苓（去黑皮）1两，芍药1两，沉香半两，黄耆半两，桔梗半两，人参半两，芎半两，桂（去粗皮）半两，木香半两，黄芩（去黑心）半两，五味子半两，半夏（汤洗7遍，去滑，焙）半两，防风（去叉）半两，枳壳（去瓤，麸炒）半两，当归（切，焙）半两，麻黄（去根节，汤煮掠去沫，焙）半两，羌活（去芦头）半两，秦艽（去苗土）半两，槟榔1个，甘草（炙）1两半（锉），陈橘皮（汤浸，去白，焙）1分。
出处：《圣济总录》卷八十九。
主治：男子。妇人五劳七伤。四肢无力，手足疼痛，饮食无味。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，加生姜2片，大枣1个（劈破），同煎至7分，去滓温服。
制备方法：上(口父)咀，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
