序号：51806
名称：钟乳益黄丸
组成：定粉半两，黄丹半两，巴豆7粒（去壳，同定粉、虢丹3味研匀，水和作1块，阴干，置瓦上，熟炭火煅红，出火毒3宿，研细），钟乳粉（炼者）半两，丁香半两，石榴皮（炒焦）半两，益智子仁半两，人参（去芦头）2钱半，朱砂（别研）2钱半，木香7钱半，白豆蔻仁7钱，诃子（煨去核）7钱，黄连4钱，乌梅10枚（取肉，炒）。
出处：《杨氏家藏方》卷十八。
主治：小儿久利不止，及挟积作泻，疳气腹胀，全不思食。
加减：None
功效：None
用法用量：每服15丸，加至20丸，空心、乳食前煎木瓜、陈橘皮汤送下。
制备方法：上为细末，煮面糊为丸，如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：钟乳益黄丹（《普济方》卷三九六）。
