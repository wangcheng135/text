序号：72251
名称：犀角煎
组成：犀角（镑屑，捣为细末）2两，威灵仙10斤（紫色者），天麻（取细末）2两，附子（炮裂，去皮脐，取末）2两，龙脑（生者，研）半两。
出处：《圣济总录》卷九。
主治：中风。手足偏枯不随，或瘫或缓，脚气攻头面浮肿，口眼斜，语涩多涎，精神恍惚，大便风秘。
加减：None
功效：None
用法用量：如有患者，用薄荷汤化1钱匕服。
制备方法：上先将威灵仙用河水1石，煮至3斗，以绢滤去滓，只取清汁，更入醇酒1斗，同在银石器中，熬至1斗，更澄去细尘滓，次入天麻，附子末在药汁中，再以慢火煎成膏，放令温冷后，入犀角屑末、龙脑末一处同搅匀，用瓷盒子盛之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
