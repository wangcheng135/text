序号：53675
名称：家秘神验松葱膏
组成：明净松香1-2斤（放入大铜锅，清水煮烊，俟其融化，滚浮水面，用竹划缓缓闭去热水，速倾冷水盆中，少顷即趁热扯拔，如作米糖式，如太烧手，入冷水一冰即取出；冷定难以扯拔，则再入清水锅中煮烊，如前再拔，如是，少则5次，多则7次，拔多愈妙。用新鲜连须全葱3-4斤，洗净稍干，捣烂取汁，去滓不用，以葱汁将松香缓缓煮干，仍用冷水一倾，随意做成饼式，愈陈愈妙，称2两），明雄黄7钱，飞东丹5钱，炒黄柏2钱，洋青黛2钱（水飞），无名异2钱（炒，研极细），大梅片5分（另研），人中白3钱（煅），上官粉1钱5分（炒），净轻粉5分（炒），制铜绿5分，枯白矾1钱，孩儿茶2钱，真绿豆粉5钱（晒干，和匀）。
出处：《寿世新编》。
主治：小儿头面耳鼻一切干湿疳疮，及白秃疮。
加减：None
功效：None
用法用量：用时先将烂疮洗净，流水湿烂者，用绵蘸干即可扑之，2-3次即结痴而愈；如疮干用女人搽头陈香油调搽；若白秃疮，宜将头发剃去，洗净疮痂再搽。
制备方法：上选顶上料，共为极细末，瓷瓶内贮塞紧。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
