序号：46774
名称：牵正散
组成：白附子、白僵蚕、全蝎（去毒）各等分，并生用。
出处：《杨氏家藏方》卷一。
主治：中风，口眼斜，半身不遂。
加减：None
功效：None
用法用量：每服1钱，热酒调下，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：1.《医方考》：艽、防之属，可以驱外来之风，而内生之风，非其治也；星、夏之辈，足以治湿土之痰，而虚风之痰，非其治也。斯三物者，疗内生之风，治虚热之痰，得酒引之，能入经而正口眼。白附之辛，可使驱风；蚕、蝎之咸，可使软痰；辛中有热，可使从风；蚕、蝎有毒，可使破结。医之用药，有用其热以攻热，用其毒以攻毒者，《大易》所谓同气相求，《内经》所谓衰之以属也。_x000D_
2.《成方便读》：全蝎色青善走者，独入肝经，风气通于肝，为搜风之主药；白附之辛散，能治头面之风；僵蚕之清虚，能解络中之风；三者皆治风之专药，用酒调服，以行其经。
用药禁忌：None
附注：祛风散（《鲁府禁方》卷一）、三神散（《仙拈集》卷一）。本方改为丸剂，名“牵正丸”（见《慈禧光绪医方选议》）。
