序号：23832
名称：安中调气丸
组成：广皮2两，半夏（姜制）1两，白茯神1两，白术（土炒）2两，枳实（麸炒）1两，苏子（炒）6钱，川芎5钱，当归（酒洗）5钱，白芍药（盐酒洗，炒）8钱，木香1钱，甘草（炙）3钱，香附3两（长流水浸3日，洗净，炒黄色），神曲（炒）1两，黄连（姜汁浸，晒干，猪胆汁拌炒）1两，白豆蔻5钱，萝卜子（炒）5钱。
出处：《古今医鉴》卷五。
主治：一切翻胃痰膈之症。
加减：None
功效：None
用法用量：每服80丸，以白汤送下；清米汤亦可。不拘时候。
制备方法：上为细末，竹沥、姜汁、神曲打糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
