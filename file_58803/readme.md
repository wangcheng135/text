序号：58818
名称：秘传紫金丹
组成：乳香1钱，没药（2味灯心同炒，研，去油净）1钱，硼砂（纸裹，水打湿煨，净）1钱，天竺黄（揭起成片者真，令人多烧诸骨及蛤粉杂之）1钱，土鳖虫（即（庶虫）虫，无甲有鳞，八足圆如棋子，雄者刀切断，碗覆之，其虫能自接上，否则是雌，不用，生米酒栈中，烧酒浸炒）1钱，牛膝1钱半，无名异（即小黑石子，出桂林，去铜净，煅，酒淬3次）2钱，骨碎补（酒炒）2钱，自然铜（煅，醋淬7次）2钱，血竭（敲断有镜面，磨甲间透红者真）2钱。
出处：《卫生鸿宝》卷六。
主治：跌扑血胀，筋断骨折，昏迷晕倒。
加减：None
功效：None
用法用量：每服7厘，以酒送下。
制备方法：上为极细末，瓷瓶收贮（勿泄气）。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
