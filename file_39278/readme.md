序号：39288
名称：疟母丸
组成：青皮、桃仁、红花、神曲、麦芽、鳖甲（醋煮）、三棱、蓬术、海粉、香附（并用醋煮）。
出处：方出《丹溪心法》卷二，名见《医学纲目》卷六。
主治：疟母。
加减：None
功效：None
用法用量：每服50-70丸，白汤送下。
制备方法：上为末，为丸如梧桐子大。
临床应用：None
各家论述：《医略六书》：疟因痰食，久则气衰，不能统运营气，故癖结胁下为疟母焉。鳖甲滋阴散结，海粉泻热软坚，桃仁、红花破血活血，三棱、蓬术削癖消坚，青皮、香附理气破结，神曲、麦芽消积化滞；不用痰药者，食化气行，则津液流通，而痰无不化矣。丸以曲糊，下以参汤，总是鼓运诸药之力以奏续也，此化积消坚之剂，为疟久癖结之专方。
用药禁忌：None
附注：《医略六书》本方用法：人参汤送下三钱。
