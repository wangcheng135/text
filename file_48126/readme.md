序号：48141
名称：神应夺命丹
组成：辰砂（择墙壁镜面者，白纱囊盛之，用升麻、麻黄、紫草、连翘4味，同纱囊放沙罐内，入新汲水，以桑柴火煮1昼夜，取出辰砂，研细，仍将煮砂药汁重纸滤去滓，水飞取末，待干听用）2钱，麻黄（不去根节，酒蜜拌，炒焦黑色）8分，蝉蜕（水洗净，去足翅）3分，紫草（酒洗）5分，红花子5分，穿山甲（酒炙拌）5分，真蟾酥3分。
出处：《痘疹心法》卷二十二。
主治：痘疹。风邪倒陷，及痘毒入里。
加减：None
功效：None
用法用量：周岁者半丸，2岁者1丸，服至3丸，热酒化服，厚盖取汗，汗出痘随出也。
制备方法：上为细末，用醋酒拌丸，作10粒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
