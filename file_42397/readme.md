序号：42412
名称：青橘皮丸
组成：青橘皮2两（汤浸，去白瓤，焙），当归1两（锉，微炒），枳壳1两（麸炒微黄，去瓤），干漆1两（捣碎，炒令烟出），附子1两（炮裂，去皮脐），木香1两，白术1两，桃仁2两（汤浸，去皮尖双仁，麸炒微黄），桂心1两，川椒3分（去目及闭口者，微炒去汗），川大黄1两（锉碎，微炒），厚朴2两（去粗皮，涂生姜汁，炙令香熟）。
出处：《圣惠》卷四十八。
主治：积聚，心腹痛疼，全不欲食。
加减：None
功效：None
用法用量：每服30丸，食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
