序号：22690
名称：医痫无双丸
组成：南星1两，半夏1两（二味用白矾、皂角、生姜煎汤浸1日夜透，切片，随汤煮干，去研、皂、姜不用），川芎3钱，归身（酒洗）1两，软石膏1两，天麻7钱，僵蚕5分，生地黄（酒炒）1两，荆芥穗5钱，辰砂5钱，川独活5钱，乌犀角5钱，白茯苓（去皮）1两，（扌柬？？）参1两，远志（甘草水泡，去心）5钱，麦冬（去心）5钱，白术（去芦油）5钱，陈皮（去白）5钱，酸枣仁（炒）5钱，黄芩3钱，川黄连（去毛）5钱，白附子（煨）3钱，珍珠3钱，甘草3钱，金箔30片。
出处：《寿世保元》卷五。
主治：痫证。
加减：None
功效：祛风化痰，降火补益，养血理脾，宁心定志。
用法用量：每服50丸，空心白汤送下。轻者半料奏效，重者全料。
制备方法：上为细末，好酒打稀糊为丸，如梧桐子大，金箔为衣。
临床应用：一儿十五岁，御女后，复劳役，考试失意，患痫症三年，遇劳则发。用十全大补汤、加味归脾汤之类，更以紫河车生研如膏，入蒸糯米饭为丸，如梧桐子大，每服100丸，一日3-4服而愈。后患遗精，盗汗，发热，仍用本方及六味丸而愈。
各家论述：None
用药禁忌：None
附注：此方治痫，不拘老幼皆效。
