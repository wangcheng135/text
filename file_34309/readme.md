序号：34318
名称：万灵膏
组成：南青木香半两，连翘半两，木鳖仁半两，桃仁半两，蓖麻仁半两，巴豆仁半两，地黄半两（生用），白芷半两，防风半两，川芎半两，黄耆半两，羌活半两，当归半两，黄连半两，蓬术露半两，蜂房半两，槐枝半两，柳枝半两，桃枝半两（已上皆为(口父)咀），乳香半两（另研），没药半两（另研），轻粉半两，黄丹15两（水飞，煮去水，再换新水煮，如此3次，炒至紫黑色佳）。
出处：《医方类聚》卷一九一引《居家必用》。
主治：一切恶疮肿毒。
加减：None
功效：None
用法用量：None
制备方法：上药(口父)咀者入真香油30两浸，春、秋7日，夏5日，冬10日，慢火煎至巴豆、蓖麻仁抹开如黑泥，滤去滓，逐旋入丹，以槐、柳枝不住手搅，候丹尽，然后下乳香、没药，挑药滴入水中成珠为度，提起离火，不住手搅至微温，入轻粉搅匀至冷，以碗覆地上出火毒，收藏时，先用真蛤粉扑裹，却入净瓷器内收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
