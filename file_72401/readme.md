序号：72417
名称：棉柿丸
组成：当归身1两，怀生地1两（竹刀切片，烘脆），萸肉1两，真阿胶1两（将石膏2两研碎，和炒成珠，去石膏不用，候冷，研为细末），棉子仁1斤（燎去外面花衣，然后入锅内炒至逐粒暴开，并至焦黑存性），真柿霜（即柿饼上白霜也，不可经火，俟诸药研末后，方和入）。
出处：方出《种福堂方》卷二，名见《卫生鸿宝》卷二。
主治：肠风下血。
加减：None
功效：None
用法用量：每日空心服药末4钱，白滚汤一饭碗冲和，将箸调末，即半浮半沉，连汤饮下。若下血太甚，临晚再服3钱，俟粪色变黑，血渐止矣。
制备方法：上药逐味炒焦如墨色，又各要存性，共研为细末，和入柿霜拌匀。
临床应用：None
各家论述：None
用药禁忌：忌食胡椒、烧酒辛热之物。
附注：None
