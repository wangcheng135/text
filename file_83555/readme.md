序号：83573
名称：乌药散
组成：乌药6两半（去皮心，切片子），白芷6两，白术2两半，苍术3两（米泔浸1宿，切片子），甘草6两半（炒），青橘皮6两（去瓤）。
出处：《普济方》卷二五六引《卫生家宝》。
主治：伤脾伤暑，伤风伤气，伤冷吐泻恶心，寒热头痛，体重倦怠，不思饮食，荣卫不顺，肢节不和。
加减：None
功效：None
用法用量：诸证不论冷热百病，先进2-3服，大人小儿、孕妇、室女皆可服。每服6钱，加生姜2片，大枣1个，水8分盏，煎至5-6分，不计时候；沸汤、酒点服亦得。
制备方法：上药焙干，研为细末，炒，共30两重。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
