序号：23215
名称：人参散
组成：人参、茯苓、羌活、独活、桔梗、知母、麻黄（去根节）、枳壳（麸炒）、甘草（炙）、川芎、陈皮（去白）、白术、厚朴（姜制）、茱萸（水煮）、桂心（不见火）、前胡、苍术、甘草节各等分。
出处：《幼幼新书》卷十四引《朱氏家传》。
主治：伤寒，四时气疫，上焦虚热，心神恍惚，脾胃不和，饮食无味，口苦舌干，烦闷。
加减：None
功效：None
用法用量：每服2钱，水1盏，加生姜、大枣，煎至7分，如要出汗，加葱白、豆豉、生姜同煎；嗽，加杏仁、麻黄同煎；小儿入薄荷同煎；妇人只入生姜同煎；岚疟，柳、桃条2-7节同煎服。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中苍术，原作“削木”，据《普济方》改。
