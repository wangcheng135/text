序号：12806
名称：石斛丸
组成：石斛（去根）3分，牛膝（酒浸，切，焙）3分，续断3分，菟丝子（酒浸，别捣）1两，石龙芮（炒）1两，桂（去粗皮）1两，肉苁蓉（酒浸，切，焙）3分，鹿茸（去毛，酥炙）1两，杜仲（去粗皮，炙，锉）3分，白茯苓（去黑皮）3分，熟干地黄（切，焙）3分，附子（炮裂，去皮脐）1两，巴戟天（去心）半两，防风（去叉）3分，桑螵蛸（炙）半两，芎藭半两，山茱萸3分，覆盆子半两，补骨脂（微炒）3分，荜澄茄3分，五味子半两，泽泻1两，沉香3分，茴香子（微炒）3分，薏苡仁（炒）1两。
出处：《圣济总录》卷二十。
主治：肾虚骨痹，肌体羸瘦，腰脚痠疼，饮食无味，小便滑数。
加减：None
功效：None
用法用量：每服30丸，空心以温酒送下，1日2次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
