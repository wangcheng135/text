序号：37898
名称：拨云散
组成：川芎半两，楮实半两，龙胆草半两，羌活半两，薄荷半两，石决明半两，苍术半两，大黄半两，荆芥穗半两，甘草半两，木贼半两，密蒙花半两，连翘半两，川椒半两，草决明半两，桔梗半两，石膏半两，甘菊花半两，白芷半两，地骨皮半两，白蒺藜半两，槟榔半两，石燕1对重半两。
出处：《卫生宝鉴》卷十。
主治：眼因发湿热不退，而作翳膜遮睛，昏暗羞明，隐涩难开。
加减：None
功效：None
用法用量：每服3钱，食后茶清调下，1日3次。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：忌杂鱼、猪、马、荞面、辛热之物。
附注：《普济方》有地肤子、白术，无地骨皮。
