序号：32641
名称：赤石脂丸
组成：半夏1两6铢，赤石脂1两6铢，蜀椒1两，干姜1两，吴茱萸1两，当归1两，桂心1两，丹参1两，白蔹1两，防风1两，雚芦半两。
出处：方出《千金》卷四，名见《女科指掌》卷一。
主治：女入腹中十二疾：经水不时，经如清水，经水不通，不周时，生不乳，绝无子，阴阳减少，腹苦痛如刺，阴中冷，子门相引痛，经来冻如葵汁，腰急病。凡此十二病得之时，因与夫卧起，月经不去；或卧湿冷地，及以冷水浴，当时取快而后生百疾；或疮痍未愈，便合阴阳，及起早作劳，衣单席薄，寒从下入。
加减：None
功效：None
用法用量：每服10丸，空心酒送下，日3次。不知，稍加丸数，以知为度。
制备方法：上为末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
