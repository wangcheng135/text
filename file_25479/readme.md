序号：25487
名称：八味地黄丸
组成：熟地黄（酒蒸）8钱，山茱萸（酒浸去核，取净肉）8钱，丹皮2钱，泽泻2钱，白茯神（去皮木）4钱，山药（蒸）4钱，五味（去梗）5钱，麦冬（去心）5钱。
出处：《医部全录》卷三三一引《体仁汇编》。
主治：产后虚羸久咳，虚损劳热，咳嗽吐血，潮热盗汗。
加减：None
功效：滋补。
用法用量：每日70丸，空心白汤送下；冬天酒下亦宜。
制备方法：上为细末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None

味丸（《全国中药成药处方集》）。本方改为汤剂，名“冬味地黄汤”（见《胎产心法》），又名“麦味地黄汤”（见《金鉴》）。
