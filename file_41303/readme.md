序号：41318
名称：金花散
组成：半夏（汤浸7遍）半两，川乌头（炮，去皮脐）半两，郁金半两，胡椒半两，川楝（焙干，麸炒）半两，木香半两，马蔺花（酒浸，取肉）半两，当归半两（生），荆三棱半两，蓬莪荗（2味捶碎，用巴豆半两同炒褐色为度，去巴豆，只取2味药）半两，大腹子（用纸制）半两，白芜荑半两，白术（炒）半两，黄连（炒）半两。
出处：《传家秘宝》卷中。
主治：劳气腹胀，心脾泄泻，生疮面黄，肢瘦，腹内虚鸣，脐胁疝痛，不思饮食。
加减：None
功效：None
用法用量：每服3钱，用羯羊肝1具，去筋膜，批作片子，匀掺药末在内，更入盐3钱，干姜末2钱，大芜荑末2钱，葱白细切搅拌匀，和白面作肝角子，慢火烧令香熟，空心吃，温米饮下。如患五积气，五膈气，下胸膈，消化酒毒，并煎陈橘皮汤调下半钱。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：切忌冷水。
附注：None
