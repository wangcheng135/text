序号：77393
名称：越鞠丸
组成：苍术、香附、抚芎、神曲、栀子各等分。
出处：《丹溪心法》卷三。
主治：六郁。
加减：None
功效：解诸郁。
用法用量：None
制备方法：上为末，水泛为丸，如绿豆大。
临床应用：None
各家论述：1.《医方集解》：此手足太阴手少阳药也。吴鹤皋曰：越鞠者，发越鞠郁之谓也。香附开气郁；苍术燥湿郁；抚芎调血郁；栀子解火郁；神曲消食郁。陈来章曰：皆理气也，气畅则郁舒矣。_x000D_
2.《删补名医方论》：以气为本，若饮食不节，寒温不适，喜怒无常，忧思无度，使冲和之气升降失常，以致胃郁不思饮食，脾郁不消水谷，气郁胸腹胀满，血郁胸膈刺痛，湿郁痰饮，火郁为热，及呕吐、恶心，吞酸、吐酸、嘈杂、嗳气，百病丛生。故用香附以开气郁，苍术以除湿郁，抚芎以行血郁，山栀以清火郁，神曲以消食郁。五药相须，共收疏解五郁之效。
用药禁忌：None
附注：芎术丸（原书同卷）、越曲丸（《松崖医径》卷下）。
