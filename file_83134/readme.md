序号：83152
名称：乌龙丸
组成：乌龙骨1副（即黑犬骨也，白头至尾脊骨1条，洗净，用黄酒浸1宿，再用硼砂5钱研末，用奶酥油炸骨至黄色为度，秤骨24两足。如犬走去阳者不用。若1犬不足，用二犬骨，足分为佳），石莲子（去壳）1两，大茴香（酒炒）1两，远志肉（甘草水浸，去骨）1两，葫芦巴1两，石菖蒲1两，枣仁（去壳，炒黑）1两，巴戟（酒浸，去骨）1两，故纸（酒炒）1两，莲须1两，肥石斛2两，桑寄生（炒）2两，肉苁蓉（酒炒，去鳞甲）2两，海浮石（火煅，醋焠7次）2两，磁石（火煅，醋淬7次）2两，胡桃肉（去皮，炒黄）5两，楮实（去壳，炒）5两，朱砂（水飞）5两，鹿茸1对（酥），石燕子3对（去毛肠嘴爪，酒蒸熟，姜汁醋炒）。
出处：《集验良方》卷二。
主治：None
加减：None
功效：聪耳明目，乌须黑发，齿落更生，强阳生子，强壮阳事。
用法用量：每服2-3钱，空心以酒送下。
制备方法：上为细末，黄酒打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
