序号：67619
名称：小分气丸
组成：木香1两，槟榔半两（锉），陈橘皮半两（汤浸，去白，焙），楝实半两（锉，炒），干姜半两（炮），青橘皮半两（汤浸，去白，炒），蓬莪术（醋浸1宿，煨）1两，巴豆1分（去皮心膜，研出油），半夏1分（汤洗7遍，去滑，焙），大黄1分（煨，锉），雄黄（研）1两。
出处：《圣济总录》卷七十二。
主治：久积气块，宿食不消，胸膈痞闷，痰逆恶心，不思饮食，脐腹刺痛，醋心噎塞。
加减：None
功效：None
用法用量：每服5丸至7丸，食后、临卧温生姜汤送下。
制备方法：上为末，醋煮面糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
