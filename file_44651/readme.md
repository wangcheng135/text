序号：44666
名称：宣毒发表汤
组成：升麻、桔梗、甘草、焦栀、葛根、薄荷、前胡、牛蒡子、防风、苏叶、连翘、杏仁、银花各等分。
出处：《麻疹全书》卷四。
主治：天行时气，发热昏闷。预防麻疹。
加减：渴，加花粉；气逆，去升麻、桔梗，加淡竹；头痛，加蔓荆子；呕吐，去甘草，加陈皮；热甚，加黄芩；初起潮热，除升麻、桔梗、甘草，加紫苏叶、葱白；初潮咳嗽，除升麻、桔梗、甘草，加桑白皮；如潮热轻者，并除淡竹叶；初潮谵语，去升麻、桔梗、甘草，加桂、附、滑石、辰砂末；初潮泄泻，除薄荷、淡竹叶、升麻、桔梗、甘草，加赤茯苓、车前子；初潮大便出血，去淡竹叶、升麻、甘草、桔梗，加生地黄、丹皮，甚者加犀角；初潮小便赤者，除升麻、桔梗、甘草，加泽泻；初潮口鼻出血者，除升麻、桔梗、甘草，加炒栀仁、茅根、生地黄；初潮腹痛，除升麻、桔梗、甘草。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：气质虚弱不可与此汤。
附注：None
