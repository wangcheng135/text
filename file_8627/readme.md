序号：8627
名称：加味千金内托散
组成：黄耆（盐水炒）、人参、当归（酒洗）、川芎、白芍（酒炒）、白芷、防风、川朴（姜炒）、桔梗、官桂、瓜蒌仁（去壳）、金银花、甘草节。
出处：《寿世保元》卷九。
主治：气血凝滞，风毒壅结，致患痈疽疮疖，在5-6日间，已溃未溃而作痛者。
加减：痛甚，加乳香、没药，倍当归、芍药。
功效：发散外邪，流行气血，排脓止痛，生肌长肉。
用法用量：每服1两，水煎，入好酒半盏，去滓温服。日进2-3服之后，疮口有黑血出，及有汗出，此药之功也。不问证候猛恶，未成者自散，已成者即溃矣。
制备方法：上锉。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
