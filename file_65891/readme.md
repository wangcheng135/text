序号：65907
名称：萆薢散
组成：萆薢1两，人参1两（去芦头），细辛1两，牛膝1两（去苗），酸枣仁1两（微炒），附子1两（炮裂，去皮脐），羚羊角屑1两，独活1两，赤芍药1两，黄芩1两，茵芋1两，麻黄1两（去根节），葛根1两（锉），汉防己1两，桂心1两，赤茯苓1两，甘草1两（炙微赤，锉），芎1两。
出处：《圣惠》卷三。
主治：肝风。四肢拘挛急痛，不可转侧。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，大枣3个，煎至5分，去滓，入竹沥1合，更煎1-2沸，温服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：忌鸡、猪、鱼、蒜。
附注：None
