序号：31776
名称：良附丸
组成：高良姜（酒洗7次，焙研）、香附子（醋洗7次，焙研）。
出处：《良方集腋》卷上。
主治：肝郁气滞，胃寒脘痛，胸闷不舒，喜温喜按者。
加减：None
功效：温胃行气疏肝，祛寒止痛。
用法用量：用时以米饮汤加入生姜汁1匙，盐1撮为丸。服之立止。
制备方法：上二味，各焙、各研、各贮、否则无效。如病因寒而得者，用高良姜2钱，香附末1钱；如病因怒而得者，用高良姜1钱，香附末3钱；如病因寒怒兼有者，高良姜1钱5分，香附1钱5分。
临床应用：None
各家论述：《谦斋医学讲稿》：良姜长于温胃散寒，香附长于疏肝行气。
用药禁忌：None
附注：止痛良附丸（《饲鹤亭集方》）。
