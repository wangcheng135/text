序号：42072
名称：三黄宝蜡丸
组成：藤黄2两，天竹黄2两，大戟1两，归尾1两，牛黄1两，刘寄奴1两，麝香1两，琥珀1两，雄黄5钱，血竭5钱，儿茶5钱，乳香5钱，冰片5钱，水银5钱。
出处：《慈禧光绪医方选议》。
主治：诸疮恶毒，一切跌打损伤，闪腰岔气，伤力成痨及破伤风；或妇女经闭不通；或产妇胎衣不下；或半身不遂，不能动履。
加减：None
功效：破顽痰，保元气，解诸毒，活经络，接筋骨，消瘀血。
用法用量：外治内服匀可，外敷用清香油调化，鹅翎撢敷。
制备方法：上为细末，用净黄蜡12两为丸，每丸重1钱。
临床应用：None
各家论述：None
用药禁忌：外敷不可见火。服药初期，应忌食生冷、瓜果、烧酒及发物。
附注：None
