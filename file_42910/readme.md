序号：42925
名称：保胎牛鼻丸
组成：黄牛鼻1具，党参2两，蜜炙黄耆2两，白术1两，归身1两5钱，白芍1两5钱，熟地4两，阿胶1两，怀山药3两，川断3两，杜仲4两，黄芩7钱，炙甘草5钱，春砂仁7钱，卷心荷叶1两，蚕茧壳1两。
出处：《中药成方配本》（苏州方）。
主治：惯易流产。
加减：None
功效：补气血，安胎元。
用法用量：每次1钱5分至2钱，开水吞服，1日2次。
制备方法：先将黄牛鼻、荷叶、蚕茧壳等3味炙灰存性，为末候用；次将熟地捣烂，与诸药打和（阿胶除外），为细末，与前药末和匀；再将阿胶用开水烊化泛丸，如绿豆大，约成丸21两。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
