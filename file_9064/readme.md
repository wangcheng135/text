序号：9064
名称：风引独活汤
组成：独活4两，茯苓3两，甘草3两，升麻1两半，人参2两，桂心2两，防风2两，芍药2两，当归2两，黄耆2两，干姜2两，附子2两，大豆2升。
出处：《千金》卷七。
主治：风毒脚气。
加减：None
功效：None
用法用量：上(口父)咀。以水9升，清酒3升，合煮取3升半，相去如人行20里久更进服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：专取独活散膝胫风痹，黑大豆疗风毒脚气，仍合保元温助血气，兼取真武汤中茯苓，芍药以制独活、附子之燥烈，以除经脉牵引之掣痛，归、桂、升、防虽无麻、杏逐痹之力量，亦是驱风活血之巨擘。
用药禁忌：None
附注：None
