序号：7125
名称：四圣不老丹
组成：透明松脂1斤（放砂锅内入无灰酒，桑柴火煮数沸，竹"枝搅稠粘，住火，以瓷器盛水，倾入水内结块，复以酒蒸九遍、1日煮讫，次日亦如是，3日通计27遍，其脂莹然如玉，不苦不涩乃止。为细末，净用12两，凡煮不宜酒少，少则易干，煮之3分之2，即倾水内），白茯苓（去皮，净用）半斤，黄菊花（家园味甘者），柏子仁（纸包，捶去油）半斤。
出处：《扶寿精方》。
主治：None
加减：None
功效：强筋补益，振精力，健胃气，增饮食，补养延年。
用法用量：每服72丸，空心好酒送下。
制备方法：上为末。炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
