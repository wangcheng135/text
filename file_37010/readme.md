序号：37020
名称：奇效香薷丸
组成：香薷3两，茯苓1两5钱，广橘皮2两，白扁豆（略炒）2两，甘草（炙）5钱，藿香梗2两，厚朴（姜汁炒）1两5钱，宣木瓜2两，苍术（茅山者）2两，山楂肉（净炒）2两，真吴曲（煨）1两5钱，槟榔1两2钱，麦芽（炒）1两2钱，枳壳（炒）1两2钱，前胡1两，光泽泻（盐水炒）1两。
出处：《痢疟纂要》卷九。
主治：暑热及兼疫之痢。又治形寒饮冷，霍乱吐泻，呕哕恶心，吞吐酸水，腹痛膈胀，及小儿呕泄，痰食积热。
加减：None
功效：None
用法用量：临用研碎，滚白水调下。
制备方法：上为细末，炼蜜为丸，如龙眼大，晒干，入新瓦器收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
