序号：4910
名称：木香蛤蚧丸
组成：蛤蚧1对（尾全者，洗净，酥炙），生鳖甲7钱半（去裙，醋炙焦），白茯苓7钱半，川芎7钱半，当归7钱半，北五味子7钱半，牛膝7钱半，绵黄耆半两，柴胡半两，知母半两，贝母半两（去心），鸡心槟榔半两，明阿胶半两（炒酥），巴戟半两（酒浸，去心），桃仁半两（酒浸，去皮，焙），肉豆蔻3个（生），木香3钱，秦艽3钱（洗），羌活3钱，破故纸3钱（炒），生发3钱（纸燃火烧存性）。
出处：《直指》卷九。
主治：劳瘵久嗽声干，骨痿瘦瘁。
加减：None
功效：None
用法用量：每服1丸，煎枣汤空心嚼下。
制备方法：上为细末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
