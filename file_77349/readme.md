序号：77365
名称：遇仙丹
组成：白黑丑（头末）2两，槟榔1两，三棱5钱，蓬术5钱，牙皂3钱。
出处：《证治汇补》卷二。
主治：虫症，腹中坚实，腹痛肠鸣，脉沉紧有力者。
加减：None
功效：导饮杀虫。
用法用量：小儿1钱，大人3钱，空心糖汤送下。是日绝食，待虫下行，然后用薄粥汤。
制备方法：上为末，糖拌。
临床应用：None
各家论述：《医略六书》：白丑、黑丑导积逐饮，三棱、莪术破结攻坚，川椒温中杀虫，牙皂通窍杀虫，拌之以糖，诱虫争咂，蒸之以饼，药力得醇，下以糖汤，乃投虫所好，而易于蠢动下出也。此导饮杀虫之剂，为积饮生虫、坚实肠鸣之专方。
用药禁忌：None
附注：《医略六书》有川椒，无槟榔。
