序号：64707
名称：硇砂丸
组成：硇砂2两（细研），干蝎3分（微炒），阿魏半两（研入），桃仁半两（汤浸，去皮尖双仁，麸炒微黄），青橘皮半两（汤浸，去白瓤，微炒），木香半两，自然铜3分（细研），白附子半两（炮裂），茴香子3分，安息香半两，肉豆蔻3分（去壳），川乌头半两（炮裂，去皮脐），磁石3分（烧，醋淬7遍，捣碎细研，水飞过），附子半两（炮裂，去皮脐）。
出处：《圣惠》卷七。
主治：肾脏积冷，下焦久虚，邪冷气攻，心腹疼痛，汗出口干，阴缩声散，手足逆冷。
加减：None
功效：None
用法用量：每服10-15丸，以生姜酒送下不拘时候。
制备方法：上为细末，入研了药令匀，以醋煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
