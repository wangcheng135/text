序号：44237
名称：上上龙虎如意丹
组成：红硇砂3钱（拣高者），漂朱砂4钱，当门麝1钱，明雄精4钱，大梅片2钱，杜蟾酥5钱（晒、研），白降丹2钱（陈者佳），五倍子4钱，玄参3钱，乳香4钱（去油净），雌黄4钱，没药4钱（去油净），前胡3钱，胆矾3钱，轻粉5钱，寒水石3钱（漂净），明矾3钱，紫草5钱。
出处：《集验良方拔萃》卷一。
主治：痈疽、发背、对口、脑疽，无名肿毒，湿痰流注，附骨阴疽，一切疡科恶症。
加减：None
功效：拔毒除腐。
用法用量：外敷患处。
制备方法：上药各为细末，和匀，再研极细末，瓷瓶收贮，勿泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
