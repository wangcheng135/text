序号：70746
名称：紫琼膏
组成：忍冬藤（取鲜者，去泥垢）8两，土牛膝（取鲜者，去泥垢）10两，钗石斛5两，黄耆5两，熟地5两，赤首乌5两，白首乌5两，玉竹3两，麦冬3两，白术3两，党参3两，桔梗4两，当归1两5钱，白芍1两5钱，远志（去心）2两，取鲜梨100个（去蒂），浸水1宿，捣去渣，入前药熬膏至7分，下后药：参三七1两，真川贝1两，真紫金藤7钱，白螺壳5钱，乳香5钱，没药5钱，象牙屑5钱，琥珀4钱，龙骨3钱，人指甲3钱，珍珠2钱。
出处：《喉科紫珍集》卷上。
主治：风痹疳毒，以及悬疔诸症，破烂难于完口，久热已退，元气亏损。
加减：None
功效：None
用法用量：None
制备方法：上药各为极细末，无声为度，下入前膏，用槐枝搅，收成窖土地下，出火气，备用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
