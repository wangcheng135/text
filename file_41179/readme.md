序号：41194
名称：金玉散
组成：人参、白术、茯苓、干葛、紫苏、半夏、荆芥穗、桔梗、杏仁（去皮尖）、麻黄（去节）、防风、陈皮、甘草、桑白皮、枳壳、前胡各等分。
出处：《墨宝斋集验方》卷上。
主治：诸般咳嗽喘急。
加减：None
功效：None
用法用量：每服1-2钱，照后汤调服：心嗽面赤流汗，加干姜汤；肝嗽眼中泪出，加乌粟米汤；肺嗽上喘气壅，加桑白皮汤；脾嗽不思饮食，加生姜汤；胆嗽令人不卧，加姜汁调白汤；冷嗽，加葱白汤；暮嗽涕唾稠粘，加生姜汤；热嗽目胀闷，蜜水调下；气嗽胸腹胀满，加青皮汤；伤风流涕，加防风荆芥汤；痰结成块，加五味子汤；劳嗽四肢羸弱，加秦艽汤；血嗽连声不止，加当归汤；_x000D_
产后嗽背痛，加黄蜡泡汤。
制备方法：上为散。每料5钱，加生姜3片，乌梅1枚，好醋和匀浸1宿，挤去醋，文火炒干，以前醋拌炒，醋尽为度，待干，研极细末听用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
