序号：27436
名称：补血顺气药酒
组成：天门冬（去心）4两，麦门冬（去心）4两，怀生地黄半斤，怀熟地黄（肥大沉水，枯朽不用）半斤，人参（去芦）2两，白茯苓（去皮）2两，甘州枸杞子（去梗）2两，砂仁7钱，木香5钱，沉香3钱。
出处：《医便》卷一。
主治：痨疾。
加减：如热，去木香，减人参5钱；如下虚或寒，将韭子炒黄色，为细末，空心用酒3-5盏，每盏挑韭末1铜钱饮之；妇人下虚无子，久饮亦能生子，用核桃连皮过口。
功效：清肺滋肾，和五脏，通血脉，补虚损，乌须发，久服貌如童子。
用法用量：文武火煮半时，以酒黑色为度。
制备方法：上用瓦坛盛无灰好酒30斤，将药切片，以绢袋盛放坛内，浸三日。
临床应用：None
各家论述：None
用药禁忌：忌黄白萝卜、葱、蒜。否则令人须发易白。
附注：None
