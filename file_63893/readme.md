序号：63909
名称：清源止痢汤
组成：黄芩3钱，茯苓5钱，紫参3钱，诃黎勒3钱，甘草1钱，天花粉3钱，地榆3钱。
出处：《辨证录》卷七。
主治：人有受暑湿之毒，膀胱热结而气不化，水谷倾囊而出，一昼夜七八十行，脓血稠粘，大渴引水，百杯不止。
加减：None
功效：清肺经之热。
用法用量：水煎服。1剂减半，3剂痢止。
制备方法：None
临床应用：None
各家论述：此方清肺金化源之方也。用黄芩、地榆以凉肺，即所以凉大肠之热也。紫参疗肠胃之热，能消积聚，而通大小之便。诃黎勒能固肠脱，合而用之于茯苓，甘草诸药之内，则通中有寒，而寒中又有调和之妙，所以奏功特神也。
用药禁忌：None
附注：None
