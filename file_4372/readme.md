序号：4372
名称：九转灵砂丹
组成：破故纸5两（芝麻同炒香熟，去芝麻），杜仲3两（去皮，锉细，麦麸同炒去丝，麸不用），川当归2两（去芦皮及须用），川巴戟2两（汤浸，去心用肉），川萆薢2两（切作薄片，新瓦焙微黄），木香1两（形如枯骨者。薄切，醋浸一宿），苍术4两（酒浸2两，米泔浸2两，各一宿，去黑皮，切片，用黑叶葱头炒香黄），川茴香1两（盐炒微黄香，去盐），大胡桃30个（酒浸，去膜，切成薄片，铺于纸上一宿）。
出处：《医方类聚》卷一五三引《经验秘方》。
主治：腰脚疼痛，筋脉不舒，行步艰难。
加减：None
功效：补腰肾，祛风湿，壮脾肾。
用法用量：空心温酒送下：如不能饮，盐汤亦可；初服20丸，每日加5丸，至50丸；看药力未到，至七10丸。
制备方法：上先将前8味制度毕，称净分两定，为末罗过后，入胡桃同药再碾细，酒面糊丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌咸猪肥肉、湿面并房事7日。
附注：None
