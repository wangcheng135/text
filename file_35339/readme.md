序号：35349
名称：乳香膏
组成：油2斤，桑枝4两，槐枝4两（慢火煎令黄熟，去桑、槐技，后下诸药），蜥蜴3条，当归3分，芎3分，白芷3分，细辛（去苗叶）3分，乌蛇肉3分，郁金香半两，木香半两，沉香半两，桂（去粗皮）1两半，藁本（去苗土）1两（以上11味锉碎，入前油内煎令焦黄，漉出澄滤，取清油20两，入锅中徐徐火煎，次下后药末），铅丹8两，蜡6两，雄黄（别研）2两，乳香（研末）1两，没药（研末）1两，麒麟竭（研末）1两，麝香（研）1分，水银粉半两。
出处：《圣济总录》卷一三一。
主治：发背，痈疽，疮瘘。
加减：None
功效：None
用法用量：用时摊帛上贴，3日1换。次服连翘汤。
制备方法：上药次第煎熬，用文武火再煎前项膏，先下铅丹、蜡，不住手搅，次下诸药末，候成膏，入盒内盛，封闭，于井底出火毒7日。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
