序号：62621
名称：清上补下丸
组成：怀生地黄（砂锅内酒拌，蒸黑）4两，南枣（酒蒸，去核）2两，怀山药1两5钱，白茯苓（去皮）1两5钱，牡丹皮1两5钱，泽泻1两5钱，辽五味子1两5钱，天门冬（去心）1两5钱，枳实（麸炒）1两5钱，贝母1两5钱，麦门冬（去心）1两5钱，桔梗（去芦）1两5钱，黄连（姜炒）1两5钱，杏仁（去皮）1两5钱，半夏（姜汁炒）1两5钱，瓜蒌仁（去油）1两5钱，枯芩（酒炒）1两5钱，甘草5钱。
出处：《寿世保元》卷三。
主治：哮喘痼疾，逢寒即发，发则上气喘急咳嗽，痰涎上壅，年久不已。
加减：None
功效：None
用法用量：每服3钱，空心淡姜汤送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
