序号：73570
名称：福幼理中丸
组成：人参（去芦）2两，炒枣仁1两，枸杞子1两，熟地2两，干姜1两，白术（麸炒）1两，生黄耆1两，当归5钱，山萸肉（酒制）5钱，故纸（盐炒）1两，核桃仁5钱，生白芍1两，肉桂（去粗皮）5钱。
出处：《全国中药成药处方集》（天津方）。
主治：脾虚气弱，呕吐久泄，面色萎黄，身体消瘦，精神不振，食欲缺乏。
加减：None
功效：扶气健脾，温中祛寒。
用法用量：每次服1丸，周岁以内酌减，白开水化服。
制备方法：上为细粉，炼蜜为丸，1钱重。每斤丸药用朱砂面3钱上衣，蜡皮或蜡纸筒封固。
临床应用：None
各家论述：None
用药禁忌：忌食寒凉、生冷、油腻。积滞吐泻忌用。
附注：None
