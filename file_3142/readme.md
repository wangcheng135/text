序号：3142
名称：天麻散
组成：天麻1两，羌活1两，天南星1两（炮裂），桂心1两，乌蛇肉1两（酒拌，炒令黄），当归1两（锉，微炒），麻黄1两（去根节），防风1两（去芦头），牛膝1两（去苗），乌犀角屑1两，侧子1两（炮裂，去皮脐），柏子仁1两，白僵蚕1两（微炒），干蝎半两（微炒），朱砂1两（细研，水飞过），牛黄1分（细研），麝香1分（研入）。
出处：《圣惠》卷六十九。
主治：妇人中风，偏枯一边，手足不遂，皮肤（疒習）（疒習），不觉痛痒，言语謇涩，筋脉拘急。
加减：None
功效：None
用法用量：每服1钱，食前以豆淋酒调下。
制备方法：上为细散，入研了药令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
