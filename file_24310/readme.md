序号：24318
名称：人参鳖甲汤
组成：人参1两，鳖甲（去裙襕，醋浸炙）1两，附子（炮裂，去皮脐）1两，柴胡（去苗）1两，桃仁（去皮尖双仁，炒）半两，芍药半两，知母（焙）半两，桂（去粗皮）半两，乌梅（去核，炒）半两，陈橘皮（汤浸，去白，炒）半两，当归（切，焙）半两，秦艽（去苗土）半两，羌活（去芦头）半两，五味子半两。
出处：《圣济总录》卷三十一。
主治：伤寒后夹劳，寒热往来，进退不时，头痛体痛，口苦咽干，不思饮食。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，加生姜3片，煎至7分，去滓，早、晚食前温服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
