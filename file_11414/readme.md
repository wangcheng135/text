序号：11420
名称：牛黄豁痰丸
组成：胆星5钱，天竺黄5钱，熟大黄5钱，黄芩（枯）5钱，贝母3钱，黑丑（头末）3钱，玄明粉3钱，白附子3钱，天麻3钱，雄黄钱半，朱砂钱半，礞石（俱水飞）钱半，沉香1钱，牛黄5分，麝香3分，冰片2分。
出处：《简明医彀》卷四。
主治：中风颠狂，惊痫僵仆，不省人事。痰涎壅盛，牙关紧急，男妇老幼一切痰盛喘满。
加减：None
功效：None
用法用量：小儿每服1丸，用竹沥入生姜自然汁1匙调，或金银煎汤，或灯心姜汤化下。
制备方法：上药各为极细末，炼蜜为丸，如弹子大，金箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：危笃者多服，痰下即安，痰不下难治。
