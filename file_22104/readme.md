序号：22112
名称：一号扫风丸
组成：大风子3斤半，苡仁8两，荆芥8两，苦参4钱，白蒺藜4钱，小胡麻4钱，苍耳子4钱，防风4钱，白花蛇1两，苍术2两，白附子2两，桂枝2两，当归2两，秦艽2两，白芷2两，草乌2两，威灵仙2两，川芎2两，钩藤2两，木瓜2两，菟丝子2两，肉桂2两，天麻2两，川牛膝2两，何首乌2两，千年健2两，青礞石（制）2两，川乌2两，知母2两，栀子2两。
出处：《中医外科学讲义》。
主治：初期结核杆菌型麻风及晚期麻风。
加减：None
功效：祛风，利湿，杀虫。
用法用量：成人初用2钱，每日2次。3天后如无呕吐、恶心反应，可每次加5分，至第8天后，每日3次。最大剂量可增至每日1两8钱。
制备方法：上为细末，水泛为小丸，干燥后待用。
临床应用：None
各家论述：None
用药禁忌：忌吃鱼腥、葱、辣等。
附注：None
