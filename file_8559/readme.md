序号：8559
名称：半夏汤
组成：半夏1升，生姜1斤，桂心4两，甘草2两，厚朴2两，人参3两，橘皮3两，麦门冬3两。
出处：《千金》卷十七。
主治：肺劳虚寒，心腹冷，气逆游气，胸胁气满，从胸达背痛，忧气往来，呕逆，饮食即吐，虚乏不足。
加减：腹痛，加当归2两。
功效：None
用法用量：上(口父)咀。以水1斗，煮取4升，分4服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：劳乏而胸中阳气不布，浊阴上攻逆满，原非本虚之谓，故用参、桂、姜、半温中，麦冬、甘草滋肺，即兼厚朴、橘皮开泄滞气，胸中阳气得人参、姜、桂守护之力，则浊阴不复上矣。
用药禁忌：None
附注：人参厚朴汤（《三因》卷八）。
