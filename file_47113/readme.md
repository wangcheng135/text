序号：47128
名称：珍珠丸
组成：麝香、牛黄、犀（广）角、珍珠、藏红花、紫檀香、白檀香、海金砂、白云香、沉香、石膏、丁香、诃子、栀子、川楝子、方海、肉桂、草果仁、黑苣胜、白苣胜等29味。
出处：内蒙古自治区蒙药制药厂。
主治：白脉病，半身不遂，风湿，类风湿，肌筋萎缩，神经麻痹，肾损脉伤，瘟疫热病，瘰疬疮疡，久热不愈。
加减：None
功效：通络化痰，舒筋活血，除湿散风，清热解毒。
用法用量：每服2.5-3.5g，每日1-2次，温开水送服。
制备方法：上药除牛黄、犀（广）角、麝香、珍珠、藏红花5味各另研细粉外，其余诃子等24味分别挑选洁净，按处方要求炮制后，共研成细粉，与上述牛黄等5味细粉兑匀，加牛奶等辅料，凉开水泛丸，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：白脉病乃蒙医术语，白脉包括大脑，小脑，延脑，脊髓，以及各种神经。白脉病包括口眼歪斜，四肢麻木，肌肉萎缩，偏瘫，小儿麻痹，言语不清，神经衰弱，高血压等症。
