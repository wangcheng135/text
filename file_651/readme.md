序号：651
名称：五蹄膏
组成：川黄连1钱5分，黑丑1钱5分，白丑1钱5分，沙参1钱5分，元参1钱5分，柴胡1钱5分，连翘1钱5分，香附1钱5分，莪术1钱5分，三棱1钱5分，木香1钱5分，地骨皮1钱5分，神曲6分，山楂6分，麦芽6分，白芥子1钱，天花粉1钱，巴豆肉1两。
出处：《医方易简》卷十。
主治：外科疑难险证，一切无名肿毒，马、牛、驴、骡为鞍所破损。
加减：None
功效：肿毒未成即消，已成即拔毒收功。
用法用量：临用时，隔水灼化，摊贴患处，须将此滓研极细，过筛收存。治马、牛、驴、骡银鞍断梁破损，用花椒煎水洗破处，随将此末干敷填满，7日自愈。
制备方法：上药并生用，煎熬。将马、牛、羊、猪、驴5样蹄壳先洗净晒干，各称5两，香麻油2斤4两，先将各蹄壳入油熬烂，次入前药一同熬化，用桑木棍同搅，捞去滓，将油滤起，刷净锅，再用飞净黄丹1斤2两(炒干)，同此油一并于细火上熬至滴水成珠为度，取放水内浸7日。
临床应用：None
各家论述：None
用药禁忌：耳、眉心忌贴。
附注：None
