序号：33904
名称：苁蓉丸
组成：肉苁蓉（酒浸，切，焙）10两，牛膝（酒浸，切，焙）10两，天麻10两，何首乌（米泔浸1宿，竹刀刮去皮）10两，黄耆（锉）10两，木瓜（去皮，作片）10两（六味以好酒5升浸，候药泣酒干，取出与后三味同焙），狗脊（去毛，锉）2两，续断（锉）2两，萆薢（锉）2两。
出处：《圣济总录》卷八十一。
主治：风湿脚气，客搏筋脉，痹挛不仁。
加减：None
功效：None
用法用量：每服30丸，盐汤或酒任下，不拘时候。
制备方法：上为末，用木瓜3枚，去皮，剜作瓮子，入青盐1两在内，闭口，饭上蒸令烂熟，捣成膏，入上件药末，和为丸。如木瓜膏少，即入酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：木瓜丸（《传信适用方》卷上）、肉苁蓉丸（《金匮翼》卷六）。
