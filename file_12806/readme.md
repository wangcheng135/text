序号：12812
名称：石斛丸
组成：石斛（去根）1两，牛膝（酒浸1宿，焙干）1两，肉苁蓉（酒浸1宿，焙干）1两，杜仲（去粗皮）1两（姜汁浸，锉，炒），金毛狗脊7钱半（燎去毛），萆薢7钱半，木香1两，牡丹皮1两，人参（去芦头）7钱半，黄耆1两，山茱萸7钱半，防风（去芦头）半两，羌活（去芦头）7钱半，川芎半两，槟榔1两半，熟干地黄1两半。
出处：《杨氏家藏方》卷四。
主治：风湿客搏经络，筋骨冷疼，四肢重痛。
加减：None
功效：None
用法用量：每服50丸，空心、食前温酒或盐汤送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
