序号：472
名称：五疳散
组成：白术（蜜水炒）1两5钱，白茯苓7钱5分（碎，炒），使君子7钱5分（碎，炒），甘草1钱5分，山楂肉5钱，麦芽（炒）5钱，金樱子肉（炒）5钱，莲子心（隔纸炒）5钱，橘红5钱，麦冬（去心）2两，芡实（蒸）2钱5分，青皮2钱（麸炒）。
出处：《惠直堂方》卷四。
主治：小儿五疳。面黄肌瘦，烦渴吐泻，肚大青筋，手足如柴，精神疲倦。
加减：None
功效：小儿无疾预服，诸疾不生。元气虚弱者，服半月，自然身体轻健。
用法用量：每服1钱，清汤送下。
制备方法：上为细末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，据剂型，当作“五疳丸”。
