序号：83944
名称：乌癞白癞丸
组成：猬皮（炙）、魁蛤、蝮蛇头（炙），木虻4枚（去翅足，熬），虻虫（去翅足，熬）1枚，蛴螬1枚（并炙），陵鲤甲（去头足，炙），葛上亭长7枚（炙），斑蝥（去翅足，炙）7枚，蜈蚣（去头足，炙）3枚，附子3枚（炮，去皮），蜘蛛5枚（炙），水蛭1枚，雷丸30枚，巴豆15枚（去皮心，熬），水银（研）1两，大黄1两，真丹1两，桂心1两，射罔1两，黄连1分，石膏（研）2两，蜀椒（汗）3分，芒消1分（研），龙骨3分，甘遂（熬）1分，礜石（烧）1分，滑石1分。
出处：《外台》卷三十引《集验方》。
主治：乌癞恶风，初觉皮肤变异，或淫淫若痒如虫行，或眼前见物如垂丝，言语无定，心常惊恐，皮肉之中，或如桃李隐疹赤黑，手足顽痹，针刺不觉痛，脚下不得踏地；白癞语声嘶破，四肢顽痹，肢节火然，心中懊热，手足俱缓，背膂至急，肉如遭劈，身体手足隐疹起，往往正白在肉里，鼻有息肉，目生血珠挡瞳子，视无所见。
加减：None
功效：None
用法用量：每服2丸，1日3次。加之以知为度。
制备方法：上药治下筛，炼蜜为丸，如胡豆大。
临床应用：None
各家论述：None
用药禁忌：忌猪肉、冷水、生葱。
附注：None
