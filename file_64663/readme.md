序号：64679
名称：硇砂丸
组成：硇砂1两（细研），硫黄半两（细研），阿魏半两（面裹煨，令面熟为度），木香半两，附子半两（炮裂，去皮脐），巴戟半两，干姜半两（炮裂，锉），肉苁蓉半两（酒浸1宿，刮去皱皮，炙干），牛膝半两（去苗），桃仁半两（汤浸，去皮尖双仁，麸炒微黄），自然铜半两（细研），干蝎半两（微炒），萆薢半两（锉），石斛半两（去根，锉）。
出处：《圣惠》卷九十八。
主治：虚冷气，脐腹疼痛。
加减：None
功效：补暖下元，利腰脚，暖脏腑，益颜色。
用法用量：每服30丸，以温酒送下。
制备方法：上为末，入研了药令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
