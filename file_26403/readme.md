序号：26411
名称：当归丸
组成：苍术8两（米泔浸1宿，炒黄），陈橘皮（去白）6两，前胡4两，荆芥穗3两，高良姜3两（炒），蓬莪术3两（锉碎，醋炒），当归（洗，焙）3两，干熟地黄（洗，焙）3两，白芍药2两半，蒲黄2两（纸上炒），干姜2两（炮），甘草2两（炙），刘寄奴2两，泽兰叶2两，木香1两半。
出处：《杨氏家藏方》卷十五。
主治：妇人脾虚血弱，冲任不和，腹胁刺痛，月事不通，赤日带下，腰脚痠疼，四肢无力，上攻头目，致多昏晕，时发寒热，多困少食；产前伤冷，胎气不安；产后血虚，腹胁（？？）痛；及一切血气之疾。
加减：None
功效：None
用法用量：空心服50丸，温酒或盐汤送下。
制备方法：上为细末，醋煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
