序号：72549
名称：温气煮散
组成：木香半两，陈橘皮（汤浸，去白，焙）半两，当归（切，焙）半两，青橘皮（汤浸，去白，焙）半两，益智仁（去皮）半两，京三棱（炮，锉）半两，蓬莪术（炮）半两，茴香子（炒）1两，马蔺花（酒浸1宿，炒）1两，甘草（炙）1两，高良姜（炒）1分，沉香（锉）1分，丁香1分，肉豆蔻（去壳）1分，诃黎勒皮1分，槟榔3枚（炮，锉）。
出处：《圣济总录》卷四十四。
主治：脾虚，心腹刺痛，四肢乏力，不思饮食。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，入盐少许，同煎至6分，食前温服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：顺气汤（《普济方》卷三十五）。
