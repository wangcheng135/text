序号：34666
名称：鸡肉丸
组成：黄连（姜炒）1两，柴胡1两，鹤虱1两，秦艽1两，知母（酒炒）1两，黄芩（酒炒）1两，使君子（炒）1两，芦荟1两，芜荑5钱。
出处：《玉案》卷六。
主治：小儿疳痨壮热，形体羸瘦，眼闭不开，四肢渐小，肚腹渐大。
加减：None
功效：None
用法用量：用黄雌鸡1只，以大麻子饲之，七日缢死，去毛净，尾上开一孔，取肠净拭干，入前药末于内，缝密，以小甑先用黑豆铺底，安鸡上，又以黑豆盖之，厚3寸，早晨蒸至晚，俟冷去骨，捣烂为丸，如干加酒少许，如麻子大。每服10丸，5岁20丸，白滚汤送下。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
