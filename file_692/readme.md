序号：692
名称：元阳秋石丹
组成：小便。
出处：《得效》卷八。
主治：羸弱久嗽，针灸不效；头眩腹胀，喘满，积年肿满；年少色欲过度，未老眼昏膝疼，遗泄白浊，腰背时痛。
加减：None
功效：还元卫生。
用法用量：每服30丸，温酒送下，1日1次。阳炼法：小便不计多少。每小便1桶(约2桶为1担)，以清水挼好皂角浓汁去滓，入小便中1盏，用竹蓖急搅令百余遭乃止，澄清取浊脚1桶，以前法再搅，澄清取浊脚，凡10数担不过取得浓脚1-2斗，滤去滓，入净锅内熬干，刮下捣碎，再入锅以清汤煮令化，乃于筲箕内布筋纸两重淋过，取淋下清汁再入锅熬干，如此制作多遍，以熬干色白如霜雪即止；乃入固济沙盒内歇口火煅成汁，候呈莹白玉色即止；细研，入沙盒内固济，顶火四围养7昼夜，再研；每服2钱，空心温水下。或酒、枣肉为丸，如梧桐子大，每服30丸，空心日1服。
制备方法：阴炼法：小便3-5石，夏月虽腐败亦堪用，以新水1半相和，旋转搅百匝，放令澄清，辟去清者只留浊脚，再以新水同搅，又澄去清者，直候无臭气再研，以乳男子乳和如膏，烈日中晒干，如此9度，即丸如梧桐子大，晒干。
临床应用：None
各家论述：None
用药禁忌：None
附注：还元丹(原书同卷)、阴阳二炼丹(《本草纲目》卷五十二引《水云录》)。本方用法中之阴炼法、阳炼法，《东医宝鉴。杂病篇》引作“阴炼秋石丹”、“阳炼秋石丹”。
