序号：30922
名称：十宝散
组成：冰片1分2厘，麝香1分2厘，辰砂1钱2分，乳香（去油）1钱2分，子红花4钱，血竭1钱6分，雄黄4钱，儿茶2分4厘，归尾1两，没药1钱4分。
出处：《种福堂方》卷四。
主治：跌打损伤，金刃器械伤，骨折骨碎。
加减：None
功效：None
用法用量：治跌打损伤皮肉青肿未破者，用陈醋调敷患处，肿消即愈；刃伤并各器械伤皮破血出者，以药末掺上包裹，不可见风，血止即愈；内伤骨碎或骨已断折，先将骨节凑准，用陈醋调药末，厚敷患处，以纸裹，外加老棉紧包好，再用薄板片夹护，将绳慢慢捆紧不可移动，药性一到，骨自接矣；刃伤深重未致透膜者，先用桑皮线缝好，多掺药于上，以活鸡皮急急贴护；跌打昏迷不醒，急用少许，以陈醋冲服，自然醒转，以便调治。
制备方法：上为细末，贮瓷瓶，黄蜡封口，勿令走气。
临床应用：None
各家论述：None
用药禁忌：宜静养百日，忌犯房事。
附注：十宝丹（《药奁启秘》）。
