序号：32839
名称：羌活汤
组成：羌活（去芦头）3两，防风（去叉）3分，人参3两，白茯苓（去黑皮）4两，芎2两，远志（去心）2两半，薏苡仁（炒）3两，附子（炮裂，去脐皮）2两，麻黄（去节，先煎，掠去沫，焙干）2两，桂（去粗皮）2两，磁石（煅，醋淬）5两，秦艽（去苗土）2两，五加皮2两半，丹参2两，生干地黄（焙）半两，杏仁（汤退去皮尖双仁，炒）半两。
出处：《圣济总录》卷八。
主治：中风四肢拘挛筋急，或缓纵不随，骨肉疼痛，羸瘦眩闷，或腰背强直，或心忪虚悸，怵惕不安，服诸汤汗出后，又觉虚困，病仍未痊。
加减：有热，去桂，加葛根1两（锉）、白鲜皮1两（炙，锉）；四肢疼痛，痿弱挛急，加当归（切，焙）、细辛（去苗叶）各2两。
功效：None
用法用量：每服5钱匕，水2盏，加大枣2枚（擘破）、生姜半枣大（切），同煎至1盏，去滓温服，空心、晚食前各1服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
