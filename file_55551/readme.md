名称：桃仁散
组成：桃仁（汤浸，去皮尖双仁，炒）2两，刘寄奴（去根，锉碎），蓬莪术（炮，细锉）1两，当归（炙，锉）1两，茴香子（微炒）1两，乌药（锉）1两，陈橘皮（汤浸，去白，焙）1两，桂（去粗皮）1两，干姜（炮，锉）1两，木香1两，附子（炮裂，去皮脐）1两，芎藭1两，白术1两，桑黄（锉）1两，高良姜（锉）1两。
出处：《圣济总录》卷一五三。
主治：妇人因月水不通，血积不散，气攻疼痛，积聚成块。
加减：None
功效：None
用法用量：每服2钱匕，温酒或醋汤调下，空心、晚食前服。
制备方法：上为细末。
临床应用：None
各家论述：None
