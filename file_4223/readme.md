序号：4223
名称：木香丸
组成：木香（不见火）1两，破故纸（炒）1两，高良姜（炒）3分，缩砂仁3分，厚朴（去粗皮，姜制，炙）3分，赤石脂半两，陈皮（去白）半两，肉桂（去粗皮，不见火）半两，白术半两（炒），胡椒1分，吴茱萸（汤泡7次，炒）1分，槟榔1枚，肉豆蔻4两（面裹煨）。
出处：《魏氏家藏方》卷七。
主治：脏腑冷湿之气，留于脾经，注下不已，经年未效，米谷不化，饮食无味，肌肉瘦瘁，心多嗔恚。
加减：None
功效：None
用法用量：每服50丸至100丸，饮送下，不拘时候，1日3次。
制备方法：上为细末，用獖猪肝4两，陈米泔水煮，加盐1钱，葱白3茎，生姜20片，同煮肝熟，研成膏子，搜前药为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
