序号：77263
名称：雄仙丹
组成：雄雀1只（去皮毛，用黄泥固济令干，以文武火煅令香熟），白花蛇（酒浸，去皮骨，取肉）1两，乌蛇（酒浸，去皮骨，取肉）1两，肉桂（去粗皮）1两，川芎1两，当归半两，萆薢半两，藿香叶（去土）半两，天南星（炮）半两，蔓荆子半两，槟榔半两，菊花半两，牛膝（酒浸1宿）半两，犀角屑半两，白附子（炮）半两，全蝎（炒）半两，白僵蚕（炒，去丝嘴）半两，珍珠末半两，朱砂（别研）半两，龙脑（别研）半两，麝香（别研）半两，地龙（去土，炒）2钱半，防己2钱半，蝉蜕（去土）2钱半，天麻（去苗）2钱半，牛蒡子（炒）2钱半，人参（去芦头）2钱半，防风（去芦头）2钱半，藁本（去土）2钱半，独活（去芦头）2钱半，白茯苓（去皮）2钱半，羌活（去芦头）2钱半，麻黄（去根节）2钱半，干姜（炮）2钱半，踯躅花2钱半，香白芷2钱半。
出处：《杨氏家藏方》卷一。
主治：一切中风，左瘫右痪，半身不遂，口眼斜，卒暴中风，目瞪嚼舌，牙关紧急，不省人事，涎如锯声；及小儿一切惊搐。
加减：None
功效：None
用法用量：每服1丸，温酒化下；小儿1丸分4服，用金银薄荷汤化下，不拘时候。
制备方法：上为末，同研拌匀，炼蜜为丸，每1两作15丸，别研朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“神仙丹”。
