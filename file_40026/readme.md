序号：40036
名称：肺脓疡汤
组成：川贝母9g，桔梗9g，化橘红各9g，葶苈子6g，苇根30g，薏苡仁18g，金银花24g，连翘24g，旋覆花（布包）6g，代赭石（布包）12g，桃仁9g，杏仁9g，冬瓜子30g。
出处：《临证医案医方》。
主治：肺脓疡成痈期或溃脓期，咳吐腥臭脓痰或脓血，胸中烦闷而痛，脉滑数，舌苔黄腻。
加减：None
功效：清热解毒，降气化痰，排脓。
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：处方以葶苈子、旋覆花、代赭石、苇根等药组成，加入化痰排脓，清热解毒等药。方中清热解毒用金银花、连翘；降气化痰用旋覆花、代赭石、葶苈子；清肺化痰用川贝母、化橘红；排脓用桔梗、苇根、薏苡仁、冬瓜子；宣肺止咳用杏仁；活血祛瘀用桃仁。
用药禁忌：None
附注：None
