序号：4902
名称：木香煮散
组成：木香1两，白术1两，缩砂1两（去皮），益智1两（去皮），藿香1两（用叶），人参1两，丁香半两，青橘皮4两（汤浸，去白，焙），陈橘皮4两（汤浸，去白，焙），桔梗（炒）3两，桂2两（去粗皮），厚朴（去粗皮，生姜汁炙）2两，高良姜1分，甘草（炙，锉）1两半。
出处：《圣济总录》卷九十。
主治：虚劳冷气，心腹痞闷，肠鸣腹痛，饮食减少。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，加生姜3片，大枣2枚(擘破)，煎至7分，和滓稍热服，如不及煎，入盐少许，如汤点服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
