序号：48279
名称：神效（山黎）峒丸
组成：西牛黄2钱5分，梅冰片2钱5分，麝香2钱5分，真阿魏1两，明腰黄1两，大黄2钱，粉儿茶2钱，天竺黄2钱，参三七2钱，上血竭2钱，乳香（去油）2钱，没药（去油）2钱，山羊血5钱，藤黄（隔汤煮10余次，去红色浮沫，沉底杂物，用净者）2两。
出处：《全国中药成药处方集》（杭州方）。
主治：痈疽发背，恶疮瘰疬，跌仆损伤，瘀血内攻，昏晕不省，以及蛇蝎蜂毒，一切无名肿毒。
加减：None
功效：None
用法用量：每服1丸，重则2丸，温酒化服。盖被取汗。外症用浓茶抹敷患处。
制备方法：以上各取净粉，将藤黄烊化，加炼白蜜为丸，每重6分，蜡壳封固。
临床应用：None
各家论述：None
用药禁忌：忌一切生冷、发物。
附注：None
