序号：47546
名称：神水膏
组成：密佗僧半两（细研），栝楼根半分，淀花半分，丁香半分，附子半分（去皮脐），麝香半分（细研），莨菪子半合（水淘去浮者），皂荚1挺（去皮子），防风半分（去芦头），朱砂半分（细研），土花消1分，沙参半分（去芦头），人参半分（去芦头），芎半分，龙骨半分，槟榔半分，桂心半分，清麻油1斤，黄蜡2两。
出处：《圣惠》卷九十。
主治：小儿恶疮。
加减：None
功效：None
用法用量：用时涂于故帛上，外贴患处。3-5次即愈。
制备方法：上为末，先取油入铛中，而后下诸药末，以慢火煎2-3沸，后下黄蜡令消，次下麝香搅令匀，膏成，贮瓷盒中。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
