序号：63949
名称：清魂散
组成：泽兰叶1分，人参1分，荆芥穗1两，川芎半两。
出处：《产育宝庆集》。
主治：产后血气暴脱，未得安静，血随气上攻，迷乱心神，眼前生花，极甚者，令人闷绝，不知人事，口禁神昏气冷。产后恶露已尽，忽然昏晕不知人。
加减：None
功效：None
用法用量：每服1钱，温酒、热汤各半盏调匀，急灌之。
制备方法：上为末。
临床应用：None
各家论述：1.《医方集解》：此足厥阴药也。气血虚弱，故以芎窮、泽兰养其血，人参、甘草补其气；外感风邪，故以荆芥疏其风。风邪去，气血生，则神清矣。肝藏魂，故曰清魂。_x000D_
2.《成方便读》：荆芥芳香辛苦，独走肝经血分，搜散风邪，故以为君。病既因虚而来，故仍以人参甘草之补正。虽虚而得之产后，不免血气或有留滞，故以泽兰之祛瘀行水，川芎之活血理气。调以温酒者，助其解散之功耳。
用药禁忌：None
附注：清魄散（《三因》卷十七）、芎窮汤（《普济方》卷三四八引《仁存方》）。
