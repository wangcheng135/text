序号：66643
名称：鹿角膏
组成：鹿角1只（烧作炭，候冷，捣筛为末），麦饭石约半斤（净洗干，碎如棋子大，有作末者，去之，于净熨斗中熬令色赤，投于米醋中，良久滤出；又熬如此9遍讫，筛为末。麦饭石者，即磨刀石及硙石是），白敛1大两（捣罗为末）。
出处：《医方类聚》卷一七二引《千金月令》。
主治：发背。毒肿，痛不可忍。
加减：None
功效：None
用法用量：用故帛涂药贴疮上，1日1易，脓出为度，疮退，即膏敷之。
制备方法：上为细末，各取1大匙，以米酢5合，文武火煎之，酢少，又旋添，约煎五十沸已来，即止，令稀稠如糊，以新净瓷器盛之。
临床应用：None
各家论述：None
用药禁忌：None
附注：麦饭石膏（《圣惠》卷六十二）、三神膏（《圣济总录》卷一三一）、灵应膏（《外科精义》卷下）、麦饭石围散（《遵生八笺》卷十八）。本方改为散剂，名“鹿角散”（见《圣惠》）。
