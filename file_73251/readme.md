序号：73267
名称：普安正气丸
组成：藿香30两（微炒），紫苏20两（炒），枳实25两（麸炒），甘草5两（生研），淡豆豉30两（炒），当归20两（酒炒），制军15两（酒炒），陈皮20两（炒），半夏20两（姜汁炒），焦楂30两，炒麦芽30两，神曲20两（炒），槟榔20两（炒），草蔻仁30两（炒），牛蒡子15两（炒），桔梗15两（炒），葛根20两（炒），莱菔子30两（炒），黄芩8两（酒炒），山栀仁15两（炒黑），赤芍20两（炒），木通12两（炒），独活15两（酒炒），厚朴20两（姜汁炒）。
出处：《慈航集》卷上。
主治：夏秋感受寒暑，伤风头痛，恶寒发热，遍身骨节疼痛，霍乱吐泻，瘟疫疟痢，时毒斑疹及四时不正之气。
加减：None
功效：None
用法用量：照后汤送服。凡感冒伤风头痛，恶寒发热，遍身骨节疼痛，用煨姜3钱，葱头3个，煎汤化服1丸，取汁即愈；凡霍乱吐泻，用煨姜2钱，灶心土3钱，如火泻，加姜汁炒川连3分，煎汤服1丸；凡疟疾，用煨姜3钱，大枣3个，青蒿3钱，煎汤化1丸，于未来早一时服；凡红痢，用金银花3钱，川连3分（酒炒），煎汤化服1丸；凡白痢，用煨姜2钱，红砂糖3钱，煎汤化服1丸；不论红白，如腹痛，加广木香1钱5分，煎汤化服1丸；凡瘟疫，用赤饭豆5钱、青蒿3钱，煎汤化服1丸；凡斑疹，用葛根2钱，石膏5钱，炒升麻8分，生甘草5分，煎汤化服1丸；凡孕妇，用砂仁1钱5分，炒黄芩8分，煎汤化服1丸；凡产后，用当归8钱，益母草3钱，煎汤化服1丸。大人服1丸，小儿服半丸。
制备方法：上药如法炮制，各研细末，炼蜜为丸，每颗4钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
