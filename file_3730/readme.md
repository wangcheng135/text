序号：3730
名称：开郁和中汤
组成：人参（去芦）5分，白术（去梗，坚者）1钱，白茯苓（去皮）7分，甘草（炙）5分，香附子（童便浸，炒）8分，苍术（米泔浸，炒）7分，黄连（去须，炒）4分，川芎5分，陈皮（去白）7分，青皮（去瓤）3分，栀子仁（鲜红者，生姜汁炒）5分，柴胡（去苗）7分。
出处：《摄生众妙方》卷五。
主治：None
加减：气不和，少加木香3分；饮食不化，加枳实(炒)5分、山楂肉7分。
功效：开郁养胃进食，消积痞，和中，益元气。
用法用量：水1钟，加生姜3片，煎至8分，去滓，食远温服。
制备方法：上锉作一服。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
