名称：还睛丹
组成：羌活1两半，白芷1两半，干菜子1两半，细辛1两半，苍术1两半，川芎1两半，火麻子1两半，防风1两半，藁本1两半，当归1两半，栀子仁1两半，黄连1两半，桔梗1两半，甘草1两半，菊花1两半，薄荷1两半，连翘1两半，石膏1两半，密蒙花1两半，川椒1两半，枸杞1两半，天麻1两半，荆芥穗1两半，乌药1两半，木贼1两半，黄芩1两半。

主治：远年近日，久患双目不见光明，内外气障，拳毛倒睫，一切眼疾。
