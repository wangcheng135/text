序号：81617
名称：镇国将军丸
组成：锦纹大黄1斤（切薄片、分作4处，1分用川黄连1两，去毛、切片，水浸汁，拌大黄同炒干为度；1分用吴茱萸1两，去梗，用水泡成汁，拌大黄同炒干为度；1分用人乳汁浸拌大黄，炒干为度；1分用童便浸大黄，炒干为度）。
出处：《寿世保元》卷三。
主治：痢疾。
加减：None
功效：None
用法用量：每服半生半熟30丸，白痢，用吴茱萸煎汤送下；赤痢，用黄连煎汤送下；赤白痢，用吴茱萸黄连煎汤送下。
制备方法：上4分共合1处，为细末，酒打米糊为丸，如梧桐子大，将1半三蒸三晒，将1半晒干，各包听用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
