序号：12472
名称：白薇丸
组成：白薇1两，细辛1两，防风1两，人参1两，秦椒1两，白蔹（一云白芷）1两，桂心1两，牛膝1两，秦艽1两，芜荑1两，沙参1两，芍药1两，五味子1两，白僵蚕1两，牡丹1两，蛴螬1两，干漆20铢，柏子仁20铢，干姜20铢，卷柏20铢，附子20铢，芎藭20铢，紫石英1两半，桃仁1两半，钟乳2两，干地黄2两，白石英2两，鼠妇半两，水蛭15枚，虻虫15枚，吴茱萸18铢，麻布叩复头1尺（烧）。
出处：《千金》卷二。
主治：妇人子脏风虚积冷，经候不调，面无血色，肌肉消瘦，不能饮食；带下，久无子。
加减：None
功效：令妇人有子。
用法用量：每服15丸，酒送下，1日2次，稍加至30丸。当有所去，小觉有异即停服。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
