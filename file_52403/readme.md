序号：52418
名称：香术散
组成：广中莪术1两（煨），丁香半两，粉草1分。
出处：《妇人良方》卷十二。
主治：妊娠5个月以后，因喜怒忧虑过度，饮食失节，以致胸腹间气刺满痛，或肠鸣，呕逆减食。
加减：None
功效：None
用法用量：空心盐汤点服1大钱，觉胸中如物按下之状。
制备方法：上为细末。
临床应用：1.妊娠心腹痛：蔡元度宠人有子，夫人怒欲逐之，遂病。医官王师处此方，3服而愈，后用累验。_x000D_
2.妊娠恶阻：一妇患妊娠呕吐不止，粥药不下，众医袖手，吉安医官，用以此药，1服见效。后以治心脾疼痛呕逆之证，不问男女服之，良验。
各家论述：None
用药禁忌：None
附注：香莪散（《济阴纲目》卷八）。
