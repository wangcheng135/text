序号：11995
名称：仙灵脾煎
组成：仙灵脾1斤（细锉，以酒7升，煮至2升，滤去滓，入后药末），附子2两（炮裂，去皮脐），肉桂2两（去皱皮），赤芍药1两，牛膝（去苗）1两，鹿角胶（捣碎，炒令微燥）1两，萆薢（锉）1两，五加皮1两，酸枣仁1两，石斛（去根，锉）1两，当归1两，木香1两，乳香1两。
出处：《圣惠》卷二十五。
主治：一切风。
加减：None
功效：None
用法用量：每服1茶匙，食前以温酒调下。
制备方法：上为末，于银锅中入前煎在内，次入白蜜5合，下诸药末，用慢火熬，以柳木蓖不住手搅，令稀稠得所，收于瓷器中。
临床应用：None
各家论述：None
用药禁忌：忌猪、鸡。
附注：None
