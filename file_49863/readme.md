序号：49878
名称：大风水银膏
组成：牙消4两，绿矾（2味烧酒炒过）4两，白矾4两（同入罐内，泥封固，煅2炷香足，取出为极细末），水银2两，黑铅2两（入铁勺内化开，煅过，为细末），胆矾5钱（另研极细末），川椒（另研）2两，杏仁（去皮尖，另研）2两，蛇床子（另研）2两，半夏2两，硫黄（另研）2两，槟榔（以水浸1宿，细片，同水磨2-3次，连水晒干）2两，杜木鳖（去壳，细切薄片，另研）2两，雄黄（另研）2两，大风子（去壳，细切片）3两，油核桃3两（2味共研），桕油烛2斤。
出处：《疡科选粹》卷六。
主治：远年近日疮癞，脓窠风癣，及鹅掌风。
加减：None
功效：None
用法用量：用川椒、葱白、甘草煎汤，洗去腌脓，软绢拭干后用药。手内揉软，揩擦疮上，2-3次效。
制备方法：上各为极细末，和匀，桕油化开，为丸。
临床应用：None
各家论述：None
用药禁忌：忌食鸡、鹅、羊肉、猪首肠蹄、辛辣发毒之物。
附注：None
