序号：36981
名称：夜明八味汤
组成：熟地12g，丹皮9g，云苓12g，山药9g，泽泻6g，黄肉9g，肉桂3g，附子1.5g，夜明砂15g，苍术12g。
出处：《千家妙方》引齐强方。
主治：命门火衰，肾阳虚惫之视网膜色素变性。
加减：None
功效：温肾壮阳。
用法用量：水煎服。每日1剂。
制备方法：None
临床应用：视网膜色素变性：梁某某，女，24岁，汽车厂工人。近一年多来视力减退，经本厂医院治疗无效，于1973年7月16目介绍来诊求治。检查：视力右眼0.9，左眼0.7，外眼无异常。眼底：双眼视神经乳头色蜡黄，网膜有散在的褐色素斑点、斑块，视野缩小。全身情况尚好，脉弦细，舌质淡，苔薄。临床诊断为视网膜色素变性（双）。治疗用“夜明八味汤”加减，共投32剂，症状得除，眼底网膜较前改善。为巩固其效果，又投桂附地黄丸50丸，早、午、晚各服1丸。再次来诊检查，见眼底较前又有改善，追踪三年，已婚生育二胎，且未见复发。
