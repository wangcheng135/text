序号：37913
名称：拨云散
组成：生地、黄连、木通、荆芥穗、谷精草、甘草、赤芍、羚羊角，大黄1分至3分，木贼、甘菊、金银花、羌活、望月沙。
出处：《救偏琐言·备用良方》。
主治：痘后热毒在肝，两目通红，甚至起障生翳者。
加减：None
功效：None
用法用量：加灯心、白芙蓉叶煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：革三（《痧症全书》卷下）、五十一号临象方（《杂病源流犀烛》卷二十一）。本方方名，据剂型，当作“拨云汤”。《痧胀玉衡》本方用：生地一钱五分，川连三分，木通一钱，荆芥穗一钱，甘草四分，赤芍一钱，大黄一钱（酒炒黑，存性），羚羊角三分（磨汁），谷精草一钱五分，木贼八分（去节），甘菊六分（白者，去蒂用），银花一钱，羌活八分，望月沙三钱。
