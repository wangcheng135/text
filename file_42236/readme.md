序号：42251
名称：青娥丸
组成：补骨脂（即破故纸，四川合州者，洗净，酒浸少顷，纸炒香为度）4两，萆薢4两（真正者，切片分作4份用，1两盐水，1两童便，1两米泔水，1两无灰好酒各浸1宿，晒干），杜仲（姜汁炒去丝）4两，胡桃肉（汤泡，去皮）8两，黄柏（蜜炒）4两，知母（蜜炒）3两，牛膝（酒洗，去芦）4两。
出处：《摄生众妙方》卷二。
主治：肾虚腰膝疼痛无力，不孕，并耳聋，眩晕，足无力，耳鸣，头晕目眩。
加减：None
功效：补肾壮阳，强筋止痛，乌须，滋肾水，壮骨。
用法用量：每服50-80丸，空心盐汤或盐酒送下。以干物压之。
制备方法：上为细末，春夏用糊，秋冬用蜜，其糊用糯米1碗煮粥，将胡桃仁捣烂为膏，和匀，石臼捣为丸，如梧桐子大。
临床应用：不孕症《医学正印》：董廉宪五十无子，服此一年，连举二子。
各家论述：《医方考》：肾，坎象也，水火并焉。水衰，则阳光独治，而令肾热；火衰，则阴翳袭之，而令肾寒；水火俱衰，则土气乘之，而邪实于肾，均之令人腰痛也。是方也，破故纸、杜仲、胡桃味厚而温，黄柏、知母、牛膝味厚而寒，温者可使养阳，寒者可使养阴，均之味厚，则均之能走下部矣；若萆薢者，苦燥之品，足以利水土之邪而平其气也。曰青娥者，涵阳之坎也，假之以名方，明其全夫水火之真尔！
用药禁忌：禁房事。
