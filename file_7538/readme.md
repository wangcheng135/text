序号：7538
名称：四神丹
组成：雄黄5两，雌黄5两，硫黄5两，朱砂5两。
出处：《局方》卷五（吴直阁增诸家名方）。
主治：男子妇人真元虚损，精髓耗伤，形羸气乏，中满下虚，致水火不交，及阴阳失序，精神困倦，面色枯槁，亡血盗汗，遗沥失精，大便自利，小便滑数，梦寐惊恐，阳事不举，腰腿沉重，筋脉拘挛；及一切沉寒痼冷，痃癖疝瘕，脐腹绞痛，久泻久痢，伤寒阴证，脉候沉微，身凉自汗，四肢厥冷；妇人百病，胎脏久冷，绝孕无子，赤白带下，月候不调，服诸药久不瘥。
加减：None
功效：活血实髓，安魂定魄，悦泽颜色，轻身保寿。治百病，补五脏，远疫疠，却岚瘴，除尸疰蛊毒，辟鬼魅邪气。
用法用量：每服1粒，绝早空心新汲水吞下。
制备方法：上为细末，入瓷盒内，将马鞭草为末，盐泥固济，慢火四围烧煅，一日一夜取出，再研细末，以糯米粽研为糊，丸如豆大。
临床应用：None
各家论述：None
用药禁忌：妊妇不可服。忌羊血、葵菜。
附注：None
