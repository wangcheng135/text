序号：13262
名称：人参丸
组成：人参3分，麦门冬（去心，焙）3分，赤石脂3分，远志（去心）3分，续断3分，韭子（炒）1两，鹿茸（去毛，酥炙）3分，茯神（去木）1两，龙齿（研）1两，磁石（煅，醋淬）1两，肉苁蓉（酒浸，切，焙）1两，丹参半两，柏子仁（炒，别研）半两,熟干地黄（焙）1两半。
出处：《圣济总录》卷九十二。
主治：精极虚寒，少腹拘急，耳聋发落，行步不正，梦寐失精，惊悸遗沥，小便白浊，甚则劳弱咳嗽。
加减：None
功效：None
用法用量：每日服20丸，空腹温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：磁石千金方（《普济方》卷三十三）。
