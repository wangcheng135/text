序号：34307
名称：万灵膏
组成：香油4斤，白芷1两，赤芍1两，大黄1两，黄连1两，白芍1两，两头尖1两，草乌1两，玄参1两，川芎1两，生地1两，川椒1两，胎发1两（头生男者），穿山甲1两，熟地1两，杏仁1两，槐角1两，黄柏1两，归尾2两，木鳖子50个（去壳），黄香12两（化开，倾米泔水9次），蓖麻子120粒（去壳），巴豆120粒（去壳）。
出处：《医部全录》卷三九八引《医贯》。
主治：元气虚弱，女人赤白带下，子宫虚冷，血山崩。
加减：None
功效：None
用法用量：用时放滚水内，顿化摊开，贴丹田，熨120手。
制备方法：上各(口父)咀，入油内浸，春5日、夏3日、秋7日、冬10日，取倾锅内，熬枯黑色，滤去滓，将净油入锅，文武火熬，滴水成珠，方细退火；黄丹2斤，飞过焙干，徐徐下，以槐、柳、桃、杏、楮各2枝，不住手搅；再下黄香，去火少冷，又下阿魏1两，丁香1两，沉香1两，麝香2两、血竭3两，孩儿茶3两，乳香3两，没药3两，珍珠（制）5钱，琥珀3钱（各为极细末）入煎膏内搅匀，将好瓶贮之，放水内浸7日，出火气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
