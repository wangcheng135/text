名称：磁石丸
组成：磁石3两（醋淬，水飞），牛膝（酒浸）（制，净取）1两，巴戟（去心，酒浸）（制，净取）1两，肉桂（制，净取）1两，远志肉（制，净取）1两，干姜（炮）（制，净取）1两，附子（炮）（制，净取）1两，黄耆（蜜炙）（制，净取）1两，防风（制，净取）1两，覆盆子（炒）（制，净取）1两，柏子仁（炒，别研）（制，净取）1两，地骨皮（制，净取）1两，鹿茸（酒炙）2两，白茯苓2两，菟丝（酒蒸）2两，生干地黄2两，当归2两。
