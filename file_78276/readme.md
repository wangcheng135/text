序号：78292
名称：蜜煮朱砂丹
组成：辰砂10两（去夹石者），蜜10斤，人参1两，白茯苓1两，白术1两，附子1两，川椒1两，仙灵脾1两，龙胆草1两，白芍药1两，熟地黄1两，黄耆1两，肉苁蓉1两，远志1两，巴戟1两，破故纸1两，石斛1两，菟丝子1两，益智1两，五味子1两，柏子仁1两，黄菊花1两，覆盆子1两，麦门冬1两，枸杞子1两。
出处：《普济方》卷二六五《家藏经验方》。
主治：定心志，轻身体，明目。
加减：None
功效：None
用法用量：每服5-7丸，空心煎人参或枣汤送下。
制备方法：上锉细，各取一半于瓶内盛，以绢袋子盛好块砂10两，以麻线悬于瓶口竹片子上，注河水半瓶，重汤煮过三昼夜，取出候干，再入蜜10斤，并前件一半余药，倾于蜜内搅匀，复入砂子袋子内，依前悬挂，河水煮15昼夜，取出焙干，研过令细，以糯米粽子尖角为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
