序号：41774
名称：钓肠丹
组成：瓜蒌2枚（烧存性），猬皮2个（锉碎，罐内烧存性），鸡冠花（锉，微炒）5两，胡桃仁15个（不油者，入罐内烧存性），白矾（枯）2两，绿矾（枯）2两，白附子（生用）2两，天南星（生用）2两，枳壳（去瓤，麸炒）2两，附子（去皮脐，生用）2两，诃子（煨，去皮）2两，半夏（生）2两。
出处：《局方》卷八。
主治：新久诸痔，肛边肿痛，或生疮痒，时有脓血；以及肠风下血，脱肛。
加减：None
功效：None
用法用量：每服20丸，空心、临卧温酒送下。远年不愈者，服10日见效，久服永除根本；肠风等疾，1-2年内者，10服可愈。
制备方法：上为细末，以醋煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：钩肠丸（《疡科选粹》卷五）。《卫生宝鉴》有乱发，无瓜蒌。
