序号：52904
名称：香葛汤
组成：紫苏（去根）1两，白芍药1两，香附子（炒，去毛）1两，川升麻1两，白干葛1两，薄陈皮1两，白芷半两，大川芎半两，苍术（米泔浸，切，炒黄色）1两，大甘草半两。
出处：《得效》卷一。
主治：四时感冒不正之气，头痛身疼，项强寒热，呕恶痰嗽，腹痛泄泻，或风寒湿瘴。
加减：如发热无汗，遍体疼痛，加葱白2根，豆豉7粒，煎热服，得汗即解；呕，去苍术，加白术数片，藿香数叶；中脘胀满，大便秘，加枳实、槟榔各半钱；有痰，加半夏半钱；咳嗽，加五味子7粒；鼻塞，加桑白皮3寸；腹痛，加枳壳半片（去瓤，切）；泄泻，加木瓜3片；如伤寒不分表里，以此药导引经络，如热多，口渴心烦，脏腑坚，或加前胡；无汗，可加麻黄；汗太过，加麻黄根。
功效：None
用法用量：每服4大钱，水1盏半，加生姜3片，煎热服，不拘时候。
制备方法：上锉散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
