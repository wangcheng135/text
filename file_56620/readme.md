序号：56635
名称：消气散
组成：白术1两，薏仁1两，茯苓1两，人参1钱，甘草1分，枳壳5分，山药5钱，肉桂1分，车前子1钱，萝卜子1钱，神曲1钱。
出处：《石室秘录》卷六。
主治：气臌。乃气虚作肿，似水臌而非水臌也。其症一如水臌之状，但按之皮肉不如泥耳。必先从脚面肿起，后渐渐肿至上身，甚则头面皆肿者。
加减：None
功效：健脾行气利水。
用法用量：水煎服。日服1剂，初服觉有微碍，久则日觉有效，10剂便觉气渐舒，20剂而全消，30剂而全愈。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：必禁食盐3月，后可渐渐少用矣。即秋石亦不可用，必须3月后可用之。
附注：None
