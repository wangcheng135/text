名称：冶葛膏
组成：冶葛皮2两，黄连2两，细辛2两，杏仁2两，（？？）草2两，芍药2两，藜芦2两，附子2两，乱发2两，芦茹2两，芎2两，白芷2两，蛇床子2两，桂心2两，藁本2两，乌头2两，白术2两，吴茱萸2两，雌黄2两，矾石2两，天雄2两，当归2两，斑蝥1两，巴豆（去皮心）1两，蜀椒（去目汗闭口）1两，黄柏1两。
