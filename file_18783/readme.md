序号：18790
名称：人参禹余粮丸
组成：禹余粮2两，龙骨2两，人参2两，桂2两，紫石英2两，川乌头2两，桑寄生2两，杜仲2两，五味子2两，远志2两，泽泻2两，当归2两，石斛2两，苁蓉2两，干姜2两，川椒2两，牡蛎2两，甘草2两。
出处：《鸡峰》卷十五。
主治：冲任虚弱，荣卫不调，或阴乘阳，胞寒气冷，血不运行。经候乍多乍少，或前或后，脐腹时痛，面色不泽，久不治之，渐至虚损，令人断产，变生他病。
加减：None
功效：调阴阳，顺血气。
用法用量：空心，食前服20丸，米饮送下，1日3次。渐加至30丸。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
