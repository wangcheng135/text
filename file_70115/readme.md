序号：70131
名称：滋补丸
组成：生干地黄1两，熟干地黄（洗焙）1两，刘寄奴（去根）1两，泽兰叶1两，川芎1两，艾叶（醋炒）1两，当归（洗）1两，牡丹皮1两，五味子1两，紫巴戟（炒，去心）1两，白芍药1两，人参（去芦头）1两，赤芍药1两，白术1两，附子（炮，去皮脐）1两，香白芷1两，金钗石斛（去根，细锉，酒拌炒干）1两，五加皮1两，茴香（微炒）1两。
出处：《杨氏家藏方》卷十五。
主治：None
加减：None
功效：补诸虚不足，调养血气，悦泽颜色，充肌肤，益饮食。
用法用量：每服50丸，空心、临卧温酒或盐汤下。
制备方法：上为细末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
