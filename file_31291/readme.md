序号：31300
名称：沉香内补丸
组成：沉香5钱，广木香5钱，乳香、人参5钱，母丁香3钱，石燕1对（烧红，醋浸），海马1对（酥），鹿茸5钱（酥炙），仙灵脾5钱（酥炙），穿山甲5钱（灰炒），韭子5钱，八角茴香5钱，木通1两（炒），小茴香1两（炒黄），甘菊花5钱（盐炒），川楝子（酒浸1宿，去皮核）1两，蛇床子1两，白茯苓1两，大附子1个（炮，去皮），川椒1两（去目），枸杞1两，麝香少许，葫芦巴（入羊肠内，酒煮）1两，丁香5钱。
出处：《遵生八笺》卷十七。
主治：None
加减：None
功效：除百病，补诸虚，健脾胃，进饮食，添精补髓，延年益寿。
用法用量：每服30丸，空心以温酒送下，仍以干物压之。服之年余，身轻髓健。妇人服之尤妙。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌生冷、腐粉、鱼腥、诸血四十九日。
附注：方中乳香用量原缺。
