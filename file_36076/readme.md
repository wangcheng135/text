序号：36086
名称：参茸片
组成：熟地黄400g，当归320g，山药320g，茯苓200g，炒白术200g，牛膝200g，枸杞子200g，盐补骨脂200g，制远志120g，柏子仁霜120g，炒酸枣仁120g，人参120g，鹿茸120g，甘草120g，肉桂80g，陈皮80g，琥珀80g。
出处：《吉林省中成药暂行标准》。
主治：气血两亏，肾虚阳衰，腰痠腿痛，心悸多梦。
加减：None
功效：补气养血，壮阳益肾。
用法用量：每次5片，温开水送下。1日2-3次。
制备方法：将人参、鹿茸、琥珀、山药、当归、白术、陈皮、肉桂、茯苓共研细粉，与酸枣仁、柏子仁霜掺研，研细。将其余熟地黄等6味酌予碎断，煎煮3次，分次滤液，浓缩成膏。将药粉、浓缩膏混合均匀，干燥，粉碎，过100目筛，加适量的黄糊精，混合均匀压片。
临床应用：None
各家论述：None
用药禁忌：孕妇慎用。
附注：None
