序号：71218
名称：紫葛散
组成：紫葛2分（锉），鳖甲1两（涂醋，炙令黄，去裙襕），桂心半两，牛膝3分（去苗），京三棱3分（微煨，锉），桃仁半两（汤浸，去皮尖双仁，麸炒微黄），虻虫1分（微炒黄，去翅足），蒲黄半两，当归3分（锉，微炒），赤芍药3分，木香半两，牡丹3分，芎3分，川大黄1两（锉，微炒）。
出处：《圣惠》卷七十二。
主治：妇人月水不通，腹内有癥块，发来攻心腹，(疒丂)刺疼痛，吃食全少，四肢羸瘦。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，煎至5分，去滓，食前稍热服之。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
