序号：28647
名称：补天大造丸
组成：人参2两，黄耆（蜜炙）3两，白术（陈土蒸）3两，当归（酒蒸）1两5钱，枣仁（去壳，炒）1两5钱，远志（去心，甘草水泡，炒）1两5钱，白芍（酒炒）1两5钱，山药（乳蒸）1两5钱，茯苓（乳蒸）1两5钱，枸杞子（酒蒸）4两，大熟地（9蒸，晒）4两，河车1具（甘草水洗），鹿角1斤（熬膏），龟版8两（与鹿角同熬膏）。
出处：《医学心悟》卷三。
主治：None
加减：阴虚内热甚者，加丹皮2两；阳虚内寒者，加肉桂5钱。
功效：补五脏虚损。
用法用量：每服4钱，早晨开水送下。
制备方法：以龟、鹿胶和药，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
