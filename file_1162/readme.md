序号：1162
名称：内补丸
组成：熟干地黄（酒洒，9蒸9曝，焙）2两，当归（去芦，洗、切，焙干，微炒）1两。
出处：《本事》卷十。
主治：妊娠，冲任脉虚，胎气自痛。
加减：None
功效：补血安胎。
用法用量：每服30-40丸，温酒送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《本事方释义》：熟地黄气味甘苦微寒，入足少阴；当归气味辛甘微温，入手少阴、足厥阴。妇人怀妊，皆冲任脉用事，冲任脉虚，不能受胎，即使有娠，亦不能安固。故妇人补血安胎，在所必用。
用药禁忌：None
附注：内补当归丸(《女科百问》卷下)。本方改为汤剂，名“地黄当归汤”(见《保命集》)。
