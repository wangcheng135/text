序号：17685
名称：人参饮
组成：人参1两半，五味子1两半，熟干地黄（焙）1两半，赤芍药1两半，麦门冬（去心，焙）1两半，甘草（炙，锉）1两半，当归（切，焙）1两半，干姜（炮）1两，芎1两，黄芩（去黑心）1两，远志（去心）1两，白茯苓（去黑皮）1两，桂（去粗皮）1两。
出处：《圣济总录》卷五十三。
主治：骨极虚寒，面肿垢黑，腰脊痛，不能久立屈伸，梦寐惊悸，上气，小腹急痛，腰背四肢常冷，小便白浊。
加减：None
功效：None
用法用量：每服5钱匕，水1盏，加羊肾1只（去筋膜，切），同煎至8分，去滓，不拘时候温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
