序号：83379
名称：乌鸡煎
组成：鹿茸（酒炙）2两，肉苁蓉（酒浸1宿，切，焙干）2两，牛膝（酒浸1宿）1两，杜仲（去粗皮，生姜汁浸，炙）1两，山茱萸1两，川芎1两，覆盆子1两，肉桂（去粗皮）1两，续断（去芦头）2两，当归（洗，焙）2两，熟干地黄（洗，焙）2两，五味子2两，白芍药1两半，黄耆（蜜炙）1两半，五加皮1两半。
出处：《杨氏家藏方》卷十六。
主治：产后将理乖宜，劳伤气血，脏腑不和，肢体消瘦，久无子息，月水不调。
加减：None
功效：None
用法用量：每服30丸，空心、食前温酒或米饮送下。
制备方法：上为细末，用乌鸡肉1斤，酒煮烂研为丸，如梧桐子大；如硬，入少许酒糊和搜。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
