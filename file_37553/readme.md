序号：37563
名称：抵圣汤
组成：西赤芍1钱半（醋炒），江枳壳1钱半（炒），生人参1钱半，法半夏1钱半，泽兰叶3钱，新会皮1钱半，鲜生姜3片。
出处：《医略六书》卷三十。
主治：腹胀呕吐，脉弦细涩者。
加减：None
功效：None
用法用量：水煎，去滓温服。
制备方法：None
临床应用：None
各家论述：产后瘀血内滞，脾胃不化，致痰湿阻遏其间，而脾不能运，胃不能纳，故腹胀呕吐不止焉。赤芍破瘀泻滞血，泽兰利水通血脉，权壳泻滞气以宽胸，人参扶元气以助化源，半夏燥湿醒脾胃，陈皮利气和中州，鲜生姜温胃气以散痰湿也。水煎，温服，使瘀化滞行，则痰湿自消而脾胃调和，安有腹胀呕吐之患乎？
用药禁忌：None
附注：None
