序号：82669
名称：不二膏
组成：金石斛16两（去根，洗，切片），乳香4两8钱（去油），真川贝16两（去心，研），没药4两8钱（去油），明天麻6两8钱（洗，切片），甘草6两4钱（洗，切片），巴豆肉5两4钱（去油，研）。
出处：《经验秘方》卷下。
主治：痰症，疬串，乳疖，一切无名肿毒。
加减：None
功效：None
用法用量：倘疬串乳疗末溃者，少加樟脑于膏上，如已溃者不用。
制备方法：上用大麻油12斤，浸药数日，煎时下活大雄鲫鱼(不去鳞甲)两条，每条重1斤半左右，煎枯去滓存油，另用铅粉2斤(研，炒黄色)，筛下收膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
