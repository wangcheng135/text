序号：22503
名称：牡丹皮汤
组成：牡丹皮1两，桂（去粗皮）1两，芎1两，延胡索1两，白术1两，芍药1两，甘草（炙，锉）1两，京三棱（煨，锉）1两，羌活（去芦头）1两，当归（切，焙）1两，枳壳（去瓤，麸炒）1两，诃黎勒（炮，去核）1两，干姜（炮）半两，木香半两，陈橘皮（去白，焙）1两半，半夏（生姜汁制作饼，晒干）半两。
出处：《圣济总录》卷一五○。
主治：妇人血风虚劳，身体骨节疼痛，手足烦热，筋脉拘急，胸膈不利，大肠结燥，血积气痛，月水不调。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，生姜3片，大枣1个（擘破），煎至7分，去滓温服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
