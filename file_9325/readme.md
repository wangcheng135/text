序号：9325
名称：加味七子丸
组成：菟丝子（淘洗，酒蒸）1两3钱，川牛膝（去芦，酒蒸）1两3钱，麦门冬（去心，酒蒸）1两3钱，山茱萸（取肉）1两3钱，原蚕蛾1两3钱，五味子1两3钱，蛇床子（酒蒸）1两6钱，车前子（淘洗）1两7钱，大甘草（炙）1两，沙苑蒺藜子（马乳浸蒸）2两2钱，覆盆子2两2钱，补骨脂子2两2钱（淘洗，炒），肉苁蓉2两5钱（酒浸，去鳞膜）。
出处：《医学正印》。
主治：肾虚无子。
加减：None
功效：None
用法用量：每服30丸或40丸，淡盐汤送下，早、晚皆服。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
