序号：31548
名称：来苏膏
组成：皂角1斤（用好肥者，无虫蛀，去皮弦、切碎）。
出处：《医方类聚》卷一一九引《瑞竹堂方》。
主治：远年日近风痫心恙，风狂中风，涎沫潮闭，牙关不开，破伤风搐。
加减：None
功效：None
用法用量：如遇病人，取手掌大1片，用温浆水化在盏内，用竹筒儿盛药水，将病人扶坐定，头微抬起，将药吹入左右鼻孔内；良久扶起，涎出为验。欲要涎止，将温盐汤令病人服1-2口便止。
制备方法：上药用酸浆水一大碗浸，春、秋3-4日，冬7日，夏1-2日，揉取净浆水，浸透皂角，汁入银器或沙锅，以文武慢火熬，用新柳条、槐枝搅熬似膏药，取出，摊于夹纸上，阴干收顿。
临床应用：None
各家论述：None
用药禁忌：忌食鸡、鱼，生硬、湿面。
附注：None
