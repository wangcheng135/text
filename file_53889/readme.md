序号：53904
名称：挨积丸
组成：京三棱（炮）3两，丁香皮（不见火）3两，丁香（不见火）1两，青皮（去白）1两，干姜（炮）2钱半，巴豆（去皮膜油）2钱半。
出处：《局方》卷十（宝庆新增方）。
主治：小儿脾胃不和，宿滞不化，腹胀肠鸣，呕逆恶心，便利不调，乳食减少，或疳泻积泻，大便酸臭；丈夫、妇人胸膈不快，酒积食积，呕逆恶心，吐泻脾疼。
加减：None
功效：消积滞，进乳食，退黄长肌。
用法用量：每服50丸至60丸，2岁儿可服7至10丸，生姜汤吞下，熟水亦得，不拘时候。
制备方法：上为细末，入巴豆研匀，面醋糊为丸，如粟米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
