序号：79293
名称：臁疮膏
组成：净轻粉25g，铅丹25g，真铜绿15g，炙乳香15g，炙没药15g（以上共为细末），血余50g（净水洗清后晒干），蜂蜡50g，香油100g。
出处：《古今名方》引梁静山祖传方。
主治：臁疮日久不愈，甚至溃烂见骨，腥臭难闻，皮肉色黑者。
加减：None
功效：活血解毒，祛腐生肌。
用法用量：用艾叶煎水或温开水洗净患处，敷本药膏适量。敷药后休息，一般7天左右即可收效。
制备方法：取大勺1把，将香油100g倒入勺内，用炭火熔化，待开滚时，把血余缓缓倒入油中，并回旋搅拌，当血余炸至白丝状，油色变红时即捞除余滓，将药锅离大，趁热撤下药末搅拌之，随把已切成小块的蜂蜡边搅拌边入油内，待药油能滴水成珠，即可放置冷水中凝膏；若膏尚稀，可再加入少许蜂蜡。
临床应用：None
各家论述：None
用药禁忌：忌食辛辣刺激食品，忌房事2～3月。
附注：None
