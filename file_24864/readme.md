序号：24872
名称：如圣黄耆汤
组成：黄耆1两（锉），乌梅1两（去核），知母1两（焙），甘草1两（炙，锉），款冬花1两，秦艽1两（去苗土），贝母1两（去心，炒），半夏（汤洗7遍，焙）1两，糯米1分，桑根白皮（锉）1分，桃仁半两（去皮尖，麸炒），鳖甲（去裙襕，醋炙黄）半两，人参1两半，柴胡（去苗）2两。
出处：《圣济总录》卷九十。
主治：虚劳，心肺俱伤，咳唾脓血。
加减：None
功效：None
用法用量：每服3钱匕，水1盏半，加生姜3片，桃枝、柳枝、葱白、薤白各少许，同煎至6分，去滓温服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
