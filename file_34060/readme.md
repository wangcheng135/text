序号：34069
名称：芜荑煎丸
组成：芜荑仁2两（捣罗为末，酸米醋2升煎为膏），人参3分（去芦头），木香半两，陈橘皮1两（汤浸，去白瓤，焙），丁香半两，乳香半两（细研），肉苁蓉半两（去壳），附子3分（炮裂，去皮脐），缩砂3分（去皮），香附子3分，枳实3分（麸炒微黄），白术3分，厚朴3分（去粗皮，涂生姜汁炙令香熟），肉桂3分（去皱皮），荜茇3分，辛夷3分。
出处：《圣惠》卷二十六。
主治：脾劳，饮食不节，口苦舌涩，多吐清水，四肢黄瘦，虽食不成肌肤，大肠时时滑泄。
加减：None
功效：None
用法用量：每服20丸，渐加至30丸，空心及晚食前以粥饮送下。
制备方法：上为末，入芜荑煎和令匀，更入炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
