序号：35013
名称：麦煎散
组成：赤茯苓1两，当归1两，干漆（炒令烟尽）1两，鳖甲（醋炙）1两，常山1两，大黄（煨）1两，北柴胡1两，白术1两，生干地黄1两，石膏1两，甘草5钱。
出处：《普济方》卷三一九引《圣惠》。
主治：少男室女骨蒸，妇人血风攻注四肢，心胸烦壅。骨热。黄瘦口臭，肌热盗汗。
加减：有虚汗，加麻黄根1两。
功效：破血积痰。
用法用量：每服2钱，水1盏，加小麦50粒，煎至8分，食后、临卧时温服。
制备方法：上为细末。
临床应用：None
各家论述：1.《医方考》：血，阴也，阻而塞之，则积阴为疰，故令四肢攻注；曰风血攻注四肢者，风血内搏，四肢无力，而倦怠浮肿也。鳖甲、干漆，攻坚削积之品也，所以治精血之留结；柴胡、石膏，解肌清热之药也，所以去骨蒸之内热；思则火结于心包，故用常山以开其结；郁则气留于六府，故用大黄以推其陈；当归、生地，生新血也；白术、甘草，致新气也；赤茯苓所以导丙丁之邪；浮小麦所以止骨蒸之汗；而麻黄根之加，乃以其形中闭，为止汗之最。_x000D_
2.《法律》：此方治肝、肺、脾、胃火盛，灼干荣血，乃致口臭肌热可验。故用润血行瘀之法，以小麦煎之，引入胃中，盖胃之血干，热炽大肠必然枯燥，服此固可无疑，然更加人参助胃真气，庶可多服取效也。
用药禁忌：None
附注：麦煎汤（《鸡峰》卷十五）。
