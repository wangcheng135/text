序号：66405
名称：子油熏药
组成：大风子1两，地肤子1两，蓖麻子1两，蛇床子1两，祁艾1两，苏子5钱，苦杏仁5钱，银杏4钱，苦参子4钱。
出处：《赵炳南临床经验集》。
主治：牛皮癣（白疕）、鱼鳞癣（蛇皮症）、皮肤淀粉样变（松皮癣）。
加减：None
功效：软坚润肤，杀虫止痒。
用法用量：燃烟熏皮损处，每日1至2次，每次15～30分钟，温度以病人能耐受为宜。
制备方法：上为粗末，用较厚草纸卷药末成纸卷。
临床应用：None
各家论述：方中蓖麻子、苏子、银杏软坚润肤；蛇床子、地肤子润肤止痒；苦杏仁润肤软坚引药深入，渗透力强；苦参子润肤杀虫；祁艾润肤暖血；大风子杀虫止痒，解风毒而润肤。
用药禁忌：None
附注：本方药物经减压后干溜成焦油物质，用凡士林或祛湿药膏制成5～10％油膏，名“子油熏药油膏”。
