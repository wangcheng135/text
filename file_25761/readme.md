序号：25769
名称：扫风丸
组成：大风子1.75kg，苡仁240g，荆芥240g，苦参120g，白蒺藜120g，小胡麻120g，苍耳子120g，防风120g，白花蛇30g，苍术60g，白附子60g，桂枝60g，当归60g，秦艽60g，白芷60g，草乌60g，威灵仙60g，川芎60g，钩藤60g，木瓜60g，菟丝子60g，肉桂60g，天麻60g，川牛膝60g，何首乌60g，千年健60g，青礞石（制）60g，川乌60g，知母60g，栀子60g。
出处：《中医外伤科学》。
主治：初期轻型麻风。
加减：None
功效：祛风，利湿，杀虫。
用法用量：成人初用6g，日2次。3天后如无呕吐、恶心等反应，可每次加1.5g，至第8天后，日服3次。
制备方法：上为细末，水为小丸，干燥后待用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
