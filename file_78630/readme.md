序号：78646
名称：增减敛阳丹
组成：川楝子（去核皮，取净肉，微炒）1两半，沉香1两，木香（不见火）1两，茴香（去枝梗，净炒）1两，旧附子（炮裂，去皮脐）1两，钟乳粉2帖1两半，破故纸（去皮，炒）1两半，肉豆蔻（面裹煨令赤，去面）1两，厚朴（去粗皮，不见火）1两，葫芦巴（酒炒干，净）1两2分，嫩鹿茸（酒炙，不令伤火）2两，当归头（去芦，酒炙，净）1两半，灵砂（拣墙壁多者，研细）4两。
出处：《普济方》卷二二四引《经验良方》。
主治：诸虚不足，心肾不交。
加减：None
功效：安神益志，顺气调荣卫。
用法用量：每服50-60丸，空心盐汤或酒送下。
制备方法：上为细末，用无灰小酒，煮糯米糊为丸，如梧桐子大，晒干，不可焙。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
