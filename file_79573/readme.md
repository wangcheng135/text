序号：79589
名称：蝉花散
组成：蝉蜕（洗净去土）、谷精草（洗去土）、白蒺藜（炒）、菊花（去梗）、防风（不见火）、草决明（炒）、密蒙花（去枝）、羌活、黄芩、蔓荆子（去白皮）、山栀子（去皮）、甘草（炒）、川芎（不见火）、木贼草（净洗）、荆芥穗各等分。
出处：《局方》卷七（绍兴续添方）。
主治：肝经蕴热，风毒之气内搏，上攻眼目，翳膜遮睛，赤肿疼痛，昏暗，视物不明，眼涩难开，多生眵泪，内外障眼。
加减：None
功效：None
用法用量：每服2钱，食后、临卧用茶清调下；或用荆芩汤入茶少许调下亦得。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：蝉蜕散（《杏苑》卷六）。
