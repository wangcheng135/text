序号：64843
名称：续命散
组成：食茱萸2两，芍药1两1分，细辛1两1分（去苗叶），前胡（去芦头）1两1分，干姜2两半（炮），乌头（炮裂，去皮脐）2两半，紫菀1两1分（去苗叶），黄芩1两1分（去黑心），白术1两1分，白薇1两1分，芎1两1分，人参1两1分，生干地黄（焙）1两1分（一方用熟地黄），蜀椒2两半（去目及闭口，炒出汗），桂（去粗皮）2两半（一方无黄芩，有当归）。
出处：《普济方》卷二四八。
主治：寒疝积聚，邪气往来，厥逆抢心痛，羸瘦少气，胸胁满，不嗜食。
加减：None
功效：None
用法用量：食前米饮或温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌生冷油腻滑物。
附注：None
