序号：63176
名称：大紫苏饮
组成：大腹子3分，诃子（炮、去核）3分，桑白皮3分，厚朴（去粗皮，姜制，炒）3分，人参（去芦）3分，陈皮（去白）1两，紫苏叶1两，草果（炮）半两，五味子（去枝）半两，茯苓（白者，去皮）半两，甘草（炙）半两，桔梗（炒）半两。
出处：《魏氏家藏方》卷二。
主治：一切气。
加减：None
功效：通利胸膈。
用法用量：每服4钱，水1盏半，加生姜3片，盐1字，煎至8分，去滓温服，不拘时候。
制备方法：上为粗末。
临床应用：子悬《赤水玄珠·医案》：费少垣乃眷，妊己九月，痰多喘嗽，胎气上逆，眼撑不起，两太阳微疼。予曰：此子悬症兼痰火也。以大紫苏饮为主。才服一帖，逆即不逆，胸膈顿宽，唯咳嗽不止，与七制化痰丸而安。
各家论述：None
用药禁忌：None
附注：None
