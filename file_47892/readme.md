序号：47907
名称：神圣化脾丸
组成：京三棱（炮）5钱，广荗（炮）5钱，青皮（去白）5钱，陈皮（去白）5钱，沉香2钱，木香3钱，檀香2钱半，槟榔2钱半，脑子半钱，全蝎2钱。
出处：《普济方》卷三八○引《德胜堂方》。
主治：小儿脾疳。积聚成块，在皮膜外者，发热发渴，乳食不进，日渐羸瘦，服诸药不效者。
加减：None
功效：顺气进食。
用法用量：2-4岁者3丸；5-8岁者5丸；9-13岁者7丸，食前用白曳刺马溺半盏，水中浸后送下。服后食积推下便软，2服后渐消，3服去病7分，然后用药磨脾去积。
制备方法：上为细末，炼蜜为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
