序号：55701
名称：桃花丹
组成：羌活3两，当归3两，甘草3两，陈皮2两，黄柏2两，大黄2两，急性子2两，南星1两5钱，白芷1两5钱，赤芍1两5钱，马牙消1两，银朱1两，绿豆粉4两。
出处：《疡科纲要》卷下。
主治：疡疾红肿焮热，或尚未高肿色赤，乳痈疔毒，漫肿坚硬者。
加减：None
功效：None
用法用量：红肿焮热者，以忍冬藤杵自然汁调敷。大青叶、芙蓉叶、马蓝头、马齿苋等自然汁皆可用。时毒发颐，用防风3钱，薄荷叶2钱，煎汤调敷，或加薄荷油10滴许。小证红肿，用茶清调。小块初起，以药末3-4分，用太乙膏贴之。阳证初起，未红未热，以甘草煎汤乘热调敷。
制备方法：上各为细末。
临床应用：None
各家论述：是方清凉而不偏于阴寒，散肿软坚，疏泄郁热，以治阳发红肿焮热，或尚未高肿色赤，乳痈疔毒，漫肿坚硬者，无不应手捷效，其功实在金黄散之上。
用药禁忌：None
附注：None
