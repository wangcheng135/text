序号：71873
名称：散肿溃坚汤
组成：黄芩8钱（酒洗，炒一半，生用一半），草龙胆（酒洗，各炒4遍）5钱，瓜蒌根（锉碎，酒洗）5钱，黄柏（酒制）5钱，酒知母5钱，桔梗5钱，昆布5钱，柴胡4钱，炙甘草3钱，京三棱（酒洗）3钱，广荗（酒洗，炒）3钱，连翘3钱，葛根2钱，白芍药2钱，当归梢2钱，黄连2钱，升麻6分。
出处：《兰室秘藏》卷下。
主治：马刀疮，结硬如石，或在耳下至缺盆中，或肩上，或于胁下，皆手足少阳经中；及瘰疬遍于颏，或至颊车，坚而不溃，在足阳明经中所出，或二经疮已破流脓水。
加减：加海藻5钱（炒）亦妙。
功效：消坚散肿。
用法用量：上(口父)咀。每服6钱，水2盏零8分，先浸多半日，煎至1盏，去滓，食后热服。于卧处伸足在高处，头低垂，每含1口，作10次咽。服毕依常安卧，取药在膈上停蓄故也。另攒半料作细末，炼蜜为丸，如绿豆大，每服100余丸，用此药汤留1口送下。
制备方法：None
临床应用：None
各家论述：《医方集解》：此手足少阳足阳明药也。柴胡、连翘清热散结，升麻、葛根解毒升阳，花粉、桔梗清肺排脓，归尾、芍药润肝活血，甘草和中化毒，昆布散痰溃坚，三棱、莪术破血行气，黄芩、柏、连、龙胆、知母大泻三焦之火，而桔梗又能载诸药而上行也。
用药禁忌：None
附注：散毒溃坚散（《普济方》卷二九一）。《外科经验方》有黄耆，无广荗；《外科理例》有土瓜根，无瓜蒌根。本方改为丸剂，名“散肿溃坚丸”（见《外科发挥》）。
