序号：29294
名称：远志丸
组成：远志（去心，姜汁炒）2两，牡蛎（煅，取粉）2两，白茯苓（去皮）1两，人参1两，干姜（炮）1两，辰砂（别研）1两，肉苁蓉（净洗，切片，焙干）4两。
出处：《局方》卷五（续添诸局经验秘方）。
主治：丈夫、妇人心气不足，肾经虚损，思虑太过，精神恍惚，健忘多惊，睡卧不宁，气血耗败，遗沥泄精，小便白浊，虚汗盗汗，耳或聋鸣。
加减：None
功效：补益心肾，聪明耳目，定志安神，滋养气血。
用法用量：每服30丸，空心、食前煎灯心、盐汤送下；温酒亦可。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
