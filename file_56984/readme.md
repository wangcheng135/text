序号：56999
名称：消毒保婴丹
组成：缠豆藤（或黄豆、绿豆根上缠绕细红丝者，于8月福生生炁日采之，阴干听用）1两5钱，黑大豆30粒，赤小豆70粒，新升麻7钱半，山楂肉1两，荆芥（连穗）5钱，防风（去芦）5钱，生地黄（酒浸，焙）1两，川独活5钱，甘草（生）5钱，当归（酒洗）5钱，赤芍药7钱半，黄连（去枝梗）5钱，桔梗5钱，辰砂（另研，水飞）1两，牛蒡子（炒）1两，老丝瓜（隔年经霜者，取连藤蒂5寸，烧存性）2个。
出处：《痘疹心法》卷二十二。
主治：None
加减：None
功效：预防痘疹。
用法用量：每服1粒，浓煎甘草汤化下。
制备方法：上为细末，和匀，用净砂糖拌丸，如李核大。
临床应用：None
各家论述：None
用药禁忌：None
附注：神功消毒保婴丹（《摄生众妙方》卷十）、保婴丹（《古今医鉴》卷十四）、神效消毒保婴丹（《寿世保元》卷八）。《纲目拾遗》有连翘七钱五分。
