序号：47486
名称：祛湿健发汤
组成：炒白术15g，猪苓15g，萆薢15g，首乌藤15g，白鲜皮15g，车前子（包）9g，川芎9g，泽泻9g，桑椹9g，赤石脂12g，生地12g，熟地12g。
出处：《赵炳南临床经验集》。
主治：脂溢性脱发。
加减：None
功效：健脾祛湿，滋阴固肾，乌须健发。
用法用量：None
制备方法：None
临床应用：None
各家论述：方中炒白术、泽泻、猪苓、茯苓块、萆薢、车前子健脾祛湿，利水而不伤其阴；生熟二地、桑椹、首乌藤补肾养血，以助生发；川芎活血，且能引药上行；白鲜皮除湿散风止痒，以治其标；赤石脂能收敛，旨在减少油脂的分泌。
用药禁忌：避免用肥皂洗头。
附注：None
