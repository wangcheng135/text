序号：42965
名称：保真散
组成：黄芩3铢，沉香3铢，木香3铢，牡丹皮（去心）1分，前胡（去毛）1分，桔梗1分，柴胡（去毛）1分，贝母（去心）1分，天灵盖（酥炙黄黑色）1分，鳖甲（醋炙黄色）1分，麦门冬（去心）1分，杏仁（去尖双仁者）1分，茯苓1分，官桂1分，荆芥穗1分，麻黄4铢。
出处：《普济方》卷十九引《护命》。
主治：一切男子、女人肾病传心，心受劳气，五心烦躁，唇口干焦，精神不足，恍惚健忘，少喜多嗔，口无滋味，小便忽赤忽白，忽多忽少。
加减：None
功效：去火毒。
用法用量：每服1钱，水1盏，煎取8分，去滓，食后服。宜先吃去肾邪方，渐吃清心脏解邪气药。若病证已传在心，即宜服此方。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：须是大腑热，脉气数，有骨力，方可吃。
