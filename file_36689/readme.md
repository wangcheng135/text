序号：36699
名称：固本泻火汤
组成：生牡蛎5钱，生龙骨5钱，当归3钱，川芎2钱，白芍4钱，生地1两，生龟版4钱，金银花6钱，玄参3钱，知母3钱，寸冬3钱，黄连2钱，生甘草1钱，羚羊角5分。
出处：《眼科临症笔记》。
主治：萤星满目症（玻璃体出血）。两眼黑珠与平人无异，不疼不红，自视火光乱飞，视物昏花。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：眼底出血：左某某，女，干部。忽觉头晕、目昏，视有红花，确诊为眼底出血。六脉洪大。乃肝火旺盛，上冲于脑，波及于目而致。先服本方十余剂，血即止。改服平肝泻火汤，三月余视力上达1.2。
各家论述：None
用药禁忌：None
附注：None
