序号：62440
名称：梅花丸
组成：绿萼梅蕊3两，滑石7两，丹皮4两，制香附2两，甘松5钱，蓬莪术5钱，茯苓3钱5分，人参3钱，嫩黄耆3钱，砂仁3钱，益智3钱，远志肉2钱5分，山药1钱5分，木香1钱5分，桔梗1钱，甘草7分。
出处：《续名医类案》卷十八引沈月枝方。
主治：体虚，木土相乘，脘腹疼痛，呕吐泄泻。肝胃久痛。体虚多郁，血热气愆，木土相乘，呕泻腹痛，易感痧秽、霍乱者。体虚，肝木犯胃，腹胸痞，或上为呕恶，或下为泄泻。
加减：None
功效：久服可杜外患，兼除宿恙。消癥调经带，催生种子。
用法用量：每服1丸，开水调下。
制备方法：上为细末，炼白蜜12两为丸，如龙眼大，白蜡封固。
临床应用：None
各家论述：None
用药禁忌：孕妇慎用。
附注：None
