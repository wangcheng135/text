序号：67621
名称：小分清饮
组成：茯苓2-3钱，泽泻2-3钱，薏仁2钱，猪苓2-3钱，枳壳1钱，厚朴1钱。
出处：《景岳全书》卷五十一。
主治：小水不利，湿滞肿胀泄泻者；湿盛无寒而泻者；湿热下流，火伏阴中而遗精者；溺白证，饮食湿滞无内热者；湿热证热微者；淫浊初起，无火而但有窒塞者；气瘕，气结膀胱，小水不利者；小儿吐泻；痘疹，湿热下利，烦热大渴，小水热涩而腹痛者；小水不利，湿滞肿胀，不能受补者。
加减：如阴虚水不能达者，加生地2钱、牛膝2钱；如黄疸者，加茵陈2钱；如无内热而寒滞不行者，加肉桂1钱。
功效：消导，燥湿，利尿，理气。
用法用量：水1钟半，煎7-8分，食前服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
