序号：74199
名称：催生如意散
组成：官桂、甘草、川芎、白芷、陈皮、赤芍、当归、木香、厚朴、南星、车前、葵子、百草霜、枳壳、麻黄、益母草。
出处：《陈素庵妇科补解》卷三。
主治：妊娠身居富贵，口厌肥甘，忧乐不常，饮食不节，饱则即卧，贪闲久坐，血多饮溢，气壅痰生，致令胞胎肥厚，或偏，或侧，任其横仰，腹皮宽胀，行动艰难，临期难产，致有不测。
加减：None
功效：催生保产。
用法用量：None
制备方法：None
临床应用：None
各家论述：方中官桂行血；甘草补中瘦胎散中，用此为佐；川芎配归补血，但随众药则行；白芷芳香通窍；陈皮利膈，赤芍破气血瘀；当归同芎、桂补中有行；木香通利三焦；厚朴利气开结；南星痰盛可用；车前利水滑胎；葵子滑胎；百草霜辛温行血；枳壳瘦胎。又补按：此方芎、归、益母草、甘草以养胎；陈、芍、香、朴以顺气，使开结疏通；南星、白芷祛痰利窍；车前、冬葵以滑胎；枳壳宽肠；官桂、百草霜辛温行血；麻黄开腠泻肺，因宜酌用。
用药禁忌：九月以后，儿已转身，可服。劳苦之家，胎瘦形癯者勿服。
