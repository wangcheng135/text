序号：16238
名称：玉霜丸
组成：天雄10两（长大者，以酒浸7日了，掘1地坑，以半称炭火烧坑通赤，速去炭火令净，以醋2升泼于地坑内候干，乘热便投天雄在内，以盆合土拥之，经宿取出，去皮脐），磁石（醋淬7次，更多为妙）2两，朱砂（飞研）2两，泽泻（洗，酒浸1宿，炙）2两，牛膝（去苗，酒浸，焙干）2两，石斛（去根，炙）2两，苁蓉（去皮，酒浸1宿，炙干）2两，巴戟（穿心者）2两，茴香（炒）1两，肉桂（去粗皮）1两，家韭子（微炒）5两，菟丝子（酒浸1伏时，蒸过，晒干，杵，罗为末，去轻浮者）5两，牡蛎（火煅，捣为粉）3两，紫梢花（如无，以木贼代之）3两，鹿茸（用麻茸连顶骨者，先燎去毛令净，约3寸以来截断，酒浸1伏时，慢火炙令脆）半两，白龙骨1斤（粘舌者，细研如粉，以水飞过3度，日中晒干，用黑豆1斗，蒸1伏时，以夹绢袋盛，日晒干）。
出处：《局方》卷五。
主治：真气虚惫，下焦伤竭，脐腹弦急，腰脚软痛，精神困倦，面色枯槁，或亡血盗汗，遗沥失精，大便自利，小便滑数，肌肉消瘦，阳事不举。
加减：None
功效：续骨联筋，秘精坚髓，延年保命，却老还童，安魂定魄，换肌秘气，轻身壮阳，益寿住世；常服补真气，壮阳道。
用法用量：每服30丸，空心、晚食前温酒送下。
制备方法：上为末，炼酒、蜜各半和丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
