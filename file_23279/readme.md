序号：23287
名称：人参蛤蚧散
组成：蛤蚧1对（蜜炙），人参（去芦头）半两，百部半两，款冬花（去梗）半两，贝母（去心）半两，紫菀茸半两，阿胶（蛤粉炒）1分，柴胡（去苗）1分，肉桂（去粗皮）1分，黄耆（蜜炙）1分，甘草（炙）1分，鳖甲（醋炙）1分，杏仁（汤浸，去皮尖）1分，半夏（生姜汁制）1分。
出处：《杨氏家藏方》卷十。
主治：虚劳咳嗽咯血，潮热盗汗，不思饮食。
加减：None
功效：None
用法用量：每服3钱，水1盏半，加生姜3片，煎至1盏，温服，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：肉桂虽去风寒，有热人不宜服，则当改用细辛。
附注：本方方名，《普济方》引作“蛤蚧散”。
