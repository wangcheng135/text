序号：79883
名称：广荗溃坚汤
组成：广荗2分，红花2分，升麻2分，吴茱萸2分，生甘草3分，柴胡3分，泽泻3分，神曲3分，青皮3分，陈皮3分，厚朴（生用）5分，黄芩5分，黄连5分，益智仁5分，草豆蔻仁5分，当归梢5分，半夏7分。
出处：《兰室秘藏》卷上。
主治：中满腹胀，内有积聚，坚硬如石，其形如盘，令人不能坐卧，大小便涩滞，上喘气促，面色萎黄，通身虚肿。
加减：如渴，加葛根4分。
功效：None
用法用量：水2大盏，煎至1盏，食远稍热服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：忌酒、醋、湿面。
附注：服二服之后，中满减半，只有积不消，再服后药（原书同卷“半夏厚朴汤”）。
