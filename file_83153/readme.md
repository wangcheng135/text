序号：83171
名称：乌龙角
组成：白僵蚕6两（去丝嘴，炒），赤小豆6两，川牛膝6两（去芦），山桂（去皮）1斤，桔梗1斤，白及1斤，百草霜1斤，山枇杷（生锉，阴干）1斤，当归尾半斤，骨碎补（去毛，炒）半斤，北细辛（去苗）半斤，白芷10两，赤芍药10两，南星（煨）10两，何首乌10两，白蔹10两，知母3两，草乌3两（用姜汁煮）。
出处：《理伤续断方》。
主治：跌扑伤损，筋骨碎断。
加减：None
功效：None
用法用量：每用姜汁或冷水、茶水调，摊纸上，于痛肿处贴之，3日1洗换。贴骨碎须夹。
制备方法：上为末，如药润，亦可焙干碾之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
