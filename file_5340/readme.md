序号：5340
名称：加减乌金散
组成：柴胡8钱，桂枝两半，黄芩两半（酒炒），厚朴两半，白芍两半（酒炒），半夏两半（制），甘草5钱，生姜10片，葱白10枚。
出处：《医略六书》卷三十。
主治：产后寒伤腠理，热遏胸中，寒热往来，胸满呕恶，脉数弦滞者。
加减：None
功效：None
用法用量：每服3钱，水煎，去滓温服。
制备方法：上为散。
临床应用：None
各家论述：桂枝温经散寒，黄芩清里泻热，柴胡疏腠理之邪，厚朴散胸中之满，半夏醒脾燥湿，白芍敛阴和营，生姜温胃散寒邪，葱白解表通阳气，甘草以缓中和胃也。为散水煎，使寒邪外解则遏热自化，而胸中之阳气廓然，何胸满呕恶不退，往来寒热不定乎！
用药禁忌：None
附注：None
