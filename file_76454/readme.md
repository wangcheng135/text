序号：76470
名称：干地黄丸
组成：干地黄7分，蛇床子6分，远志10分，茯苓7分，苁蓉10分，五味子4分，麦门冬5分，杜仲10分，阿胶8分，桂心5分，天雄7分，枣肉8分，甘草10分。
出处：《千金》卷十九。
主治：五劳七伤六极，脏腑虚弱，食饮不下，颜色黛黯，八风所伤。
加减：None
功效：补虚益气，能食，资颜色，长元阳。
用法用量：每服20丸，酒送下，1日2次。加至30丸。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：地黄得天雄则滋而不壅，桂心得五味则辛而不散，阴阳兼济，寒热交通；麦冬、阿胶、茯苓助地黄之滋阴；苁蓉、远志、蛇床、杜仲助天雄之补火；枣肉、甘草通脾津而和寒热诸性也。
用药禁忌：None
附注：None
