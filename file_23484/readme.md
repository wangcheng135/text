序号：23492
名称：防风黄芩丸
组成：条芩（炒焦）、防风各等分。
出处：《校注妇人良方》卷十二。
主治：妇人肝经风热所致的吐血，衄血便血，尿血，血崩，漏胎。
加减：None
功效：None
用法用量：每服30-50丸，食远或食前米饮或温酒送下。
制备方法：上为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：《医略六书》：妊娠风热，干于血室，胎孕为之不安，致经血妄行，漏胎下血不止焉。子芩清热于里，防风流风于外，二味成方，尤以粥糊，下以米饮，使风热两除，则经脉清和而经血无不固，胎孕无不安，何漏胎之不愈哉？
用药禁忌：None
附注：防风子芩丸（《医略六书》卷二十八）、防风丸（《盘珠集》卷下）。
