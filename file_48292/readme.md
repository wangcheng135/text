序号：48307
名称：神效千捶膏
组成：松香1斤（熔化，滤净，下水缸中，多使人抽拔至白色莹亮，待干研末），木鳖子（去壳）半斤，杏仁（泡，去皮尖）半斤，蓖麻子（去壳）半斤，大枫子（去壳）半斤，铜绿（另研）2两，蛇床子2两，穿山甲（锉碎，炒）2两，樟脑（另研）2两，胆南星2两，白芷2两，面粉2两，半夏2两，川乌1两，甘草节1两，草乌1两，五倍子5钱，闹羊花5钱，红芽大戟5钱，金线重楼5钱，乳香5钱，没药（2味另研）5钱，孩儿茶5钱，血竭5钱，轻粉5钱，雄黄5钱，龙骨4钱，青竹蛇4钱，山慈菇4钱，甘遂4钱，白花蛇小半条，僵蚕1钱2分，麝香（另研）1钱2分，蜈蚣6条，癞虾蟆半个，全蝎2钱5分，阿魏2钱5分，莪术2钱5分，三棱2钱5分。
出处：《慈幼新书》卷十一。
主治：小儿一切无名肿毒；泄泻。
加减：None
功效：止痛内消。
用法用量：摊布贴之，不可见火。如日久放干，以温水烊开摊上，略烘柔软贴之。加麝香少许贴脐上，立止泄泻。
制备方法：葱汁、姜汁、柏油、桐油等分制净，将各药研细筛过，入油汁拌匀，放臼内杵三千下，令稠润成膏，如干渐加油汁。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
