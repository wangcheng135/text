序号：5455
名称：二气散
组成：大附子1只（生用，不去皮脐，切作三份，为片子令薄），大茴香6两（净洗过，滤干，分作3份，每份2两）。
出处：《朱氏集验方》卷八。
主治：肾经停冷，气滞力弱。
加减：None
功效：None
用法用量：第一度：用附子1份，茴香2两1份，同炒，令茴香黄色香熟，用瓦瓶盛贮，厚纸封盖，安在冷处1宿，次日除附子，以茴香为末，空心盐、酒调下。第二度：如第一度分两，炒法、存性皆如之。但次早只除附子1半，其1半附子同茴香为细末，如前法服之，第三度：凡分两、炒法、存性皆如前二度之法，但次早全用附子同茴香为细末，亦如前法服之。外余1分半附子，又可合别药用。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
