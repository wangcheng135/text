序号：49803
名称：大丸
组成：羌活半两，白术半两，陈皮3分，木通3分，黄耆3分，桑白皮3分，木香1分，黑牵牛10两（5两炒，5两生）。
出处：《三因》卷十四。
主治：通身肿满，及痰气食积，伤寒感风，脾气横泄。
加减：None
功效：None
用法用量：治风痰，散腹胁壅滞，清头目，浓煎生姜汤送下；取食伤，止赤白痢，煎枣汤送下；小便不利，灯心汤送下；伤寒，葱茶送下；如未快，用稀粥投之，用热茶汤亦可。须7日后方可服。已得泻，急欲止之，投冷白粥，即自止。
制备方法：上为末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：不得吃生冷、荤腥及滋味1日，只软饭淡粥可也。
附注：None
