序号：74198
名称：催生如意散
组成：当归（量人酌用）、川芎（量人酌用），冬葵子1钱，车前子1钱，肉桂（夏月不用，冬月8分，春秋6分），丁香5分，滑石2钱，红花1钱，乌药1钱，生芝麻2钱，黄杨头7个，枳壳1钱半（虚人减半）。
出处：《陈素庵妇科补解》卷四。
主治：妊妇本质强壮，奉养太过，起居安逸，绝无忧劳者。
加减：None
功效：行气滑胎催生。
用法用量：None
制备方法：None
临床应用：None
各家论述：是方用桂、丁辛热以行血破血；乌药，红花以行血顺气，通其壅滞；枳壳以宽肠；冬葵、滑石、芝麻以滑胎；车前利气，芎、归乃佛手散也，虚人难产则倍加大剂，以助行血气。
用药禁忌：九月以后，十月之内，可预服数剂。
附注：None
