序号：37280
名称：定痛和营汤
组成：当归3钱，赤芍3钱，川芎2钱，生地3钱，红花1钱，苏木2钱，琥珀1钱，血竭2钱，制乳香1钱，三七2钱，枳壳2钱，大黄2钱，甘草1钱，朱砂2钱，淮牛膝3钱，砂仁1钱半。
出处：《林如高正骨经验》。
主治：新伤气阻血滞。
加减：None
功效：和营定痛，活血祛瘀。
用法用量：春季，加泽泻2钱，续断3钱，甘草1钱，酒、水各半煎服；夏季，加麦冬3钱，天冬2钱，泽泻3钱，水煎服；秋季，加冰糖3钱，黄芩2钱，五味子3钱，酒、水各半煎服；冬季，加紫苏2钱，续断3钱，破故纸3钱，酒煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
