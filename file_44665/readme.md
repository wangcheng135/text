序号：44680
名称：宣毒膏
组成：豮猪（腊八日取尾后刺血1升，先用新泥盆盛）1两，好朱砂（细研，水飞）1两，拣乳香（细研）1两，甘草（末）半两，马牙消半两，脑子1分，麝香1分（细研）。
出处：《幼幼新书》卷十八引张涣方。
主治：毒盛甚，疮疹已出不快，倒黡。
加减：None
功效：None
用法用量：煎人参汤化下。若毒甚，疮毒倒黡服之者，疮疹红活再长，神妙。
制备方法：上件一处同猪血拌调细匀，用宽旧竹筒1个，底留一节，都入诸药在简内，用密纸数重，紧垂于大粪坑屋梁上，至清明日取出，晒干，更入脑、麝各1钱，研细匀，滴水为丸，如皂子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
