序号：71308
名称：紫石英丸
组成：紫石英3两，人参1两半，熟地黄5两，当归身3两，川芎1两，川乌头1两（炮），紫厚朴1两（制），桂心1两半，吴茱萸1两（醋炒），白干姜1两（炒）。
出处：《医略六书》卷二十七。
主治：血海久冷不孕，脉细涩者。
加减：None
功效：None
用法用量：每服2-3钱，温酒送下。
制备方法：上为末，炼蜜为丸。
临床应用：None
各家论述：血海久冷，冲任少薰育之权，故腹痛经迟，时发寒热而不能孕子焉。桂、附暖血海以逐冷，姜、朴温中气以散寒，当归养血脉，人参扶元气，秦艽活血通经，半夏化痰燥湿，白薇降泄以除寒湿热也。炼蜜以丸之，温酒以下之，俾血海温暖，则久冷顿消，而腹痛无不退，寒热无不除，何患天癸不调，不能孕子乎。
用药禁忌：None
附注：None
