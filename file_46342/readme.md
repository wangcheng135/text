序号：46357
名称：活命神丹
组成：真正当门子麝香1钱，月石（净末）3分，真正大泥冰片头、山豆根（净末）5分，真道地蟾蜍（不见火，晒，研，净末）1钱，老生姜（取汁澄粉）3分，新江子仁（去净油）1钱，大干地龙（去泥）2条。
出处：《喉科紫珍集》卷上。
主治：喉风喉痹，双单喉蛾。
加减：None
功效：None
用法用量：临时用小红枣1枚，去蒂去核，入药黄豆大。但取核，只开近蒂半截，免走药性。将枣开蒂孔一头，塞入鼻中，令病人闭口目，避风，少顷即能得涎嚏，或出脓，以银花、甘草汤漱之，喉中便觉通快。俟鼻内热时，即将药枣拿去。病甚者，再换药枣1枚，无不立效。凡左蛾塞左，右蛾塞右，双蛾，左右先后塞之。唯喉风喉痹，男左女右塞之。
制备方法：上药照方拣选道地，逐一研极细末，秤准，制，合匀，瓷瓶收贮，蜡口封固。
临床应用：None
各家论述：None
用药禁忌：虚人、孕妇及阴虚喉痛不可用。
附注：None
