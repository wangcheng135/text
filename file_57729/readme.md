序号：57744
名称：烧肝散
组成：黑附子1两，缩砂仁半两，川芎半两，青皮半两，陈皮半两，肉桂半两，益智半两，肉豆蔻半两，红豆半两，山茵陈半两，柴胡（去苗）半两，芍药半两，桔梗半两，白术半两，苍术（去皮，炒）半两，远志（去心）半两，干姜半两，白芷半两，良姜半两，细辛半两，蓬莪荗半两，芜荑半两，荜拨半两，大椒半两。
出处：《御药院方》卷六。
主治：五劳七伤，三十六风，二十四冷，脐腹寒痛，四肢少力困倦，黄疮久患。
加减：None
功效：None
用法用量：每服5钱，猪羊肝4两批开，葱白2枚细切，掺药重重尽，纸裹3-5重，文武火烧香熟。空心、食前吃，白面烧饼2个压，米饮送，次吃好酒2-3盏。频频吃此药多验。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
