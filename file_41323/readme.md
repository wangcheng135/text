序号：41338
名称：金枣儿
组成：苍术（米泔浸）1两，细辛（去叶）1两，白术1两，当归（酒洗）1两，天麻1两，草乌1两，川乌（炮去皮脐）1两3钱，防风（去芦）1两3钱，两头尖1两3钱，川芎1两3钱，香白芷8钱，没药5钱，乳香5钱，雄黄5钱，朱砂5钱，白花蛇（酒浸，去骨）5钱，穿山甲（酥炙）3钱，蝉蜕（洗）3钱，麝香2钱，金箔5贴。
出处：《济阳纲目》卷一。
主治：中风不语，左瘫右痪，口眼斜，不省人事；及破伤风，牙关紧急，角弓反张；或疯狗咬伤。
加减：None
功效：None
用法用量：每服1丸或半丸，温酒化服。
制备方法：上为细末，炼蜜为丸，如枣大，用金箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
