序号：37888
名称：拨云退翳丹
组成：水银1两2钱，青铅2两，硼砂1两6钱，火消1两2钱，明矾1两2钱，皂矾1两，防风2钱半，草决明2钱半，木贼2钱半，威灵仙2钱半，龙胆草2钱半，荆芥2钱半，归尾5钱。
出处：《银海指南》卷三。
主治：一切星障胬肉，顽翳老膜，诸般实症；并治瞽目。
加减：None
功效：None
用法用量：点眼。
制备方法：前6味中，先将水银熔化，入铅搅匀，倾出研细，然后每味各研和匀；次将后7味，用水5盏，煎至3盏，去滓，再煎至将干，下煎6味末结胎，盐泥固济，以3炷香为度，先文后武，取掇至地上出火气，于上刮取，其色要淡黄色为佳；每升药5分，配上好煅过炉甘石1钱，冰片、朱砂、雄黄、珍珠各2分，白丁香（飞过）、硇砂各1分，元明粉5厘，共研极细，瓷瓶收贮待用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
