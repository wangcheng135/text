序号：73453
名称：温中益气汤
组成：附子（炮，去皮脐）5钱，干姜（炮）5钱，草豆蔻3钱，甘草（炙）3钱，益智仁2钱，白芍药2钱，丁香2钱，藿香2钱，白术2钱，人参1钱半，陈皮1钱半，吴茱萸1钱半，当归1钱。
出处：《卫生宝鉴》卷十八。
主治：中气不足，四肢困倦，躁热恶寒，时作疼痛，不欲饮食，食则呕吐，气弱短促，怠惰嗜卧，医误作伤寒治之，发表攻里，中气愈损，爪甲微见青黑，足胫至腰冰冷，目上视而？？不转睛，咽噎不利，小腹冷气上冲心而痛，呕吐不止，气短欲绝。
加减：None
功效：None
用法用量：上(口父)咀。每服5钱，水2盏，煎至1盏，去滓，食前温服。病势大者服1两。
制备方法：None
临床应用：None
各家论述：《内经》曰：寒淫于内，治以辛热，佐以苦甘温。方中附子、干姜大辛热，助阳退阴，故以为君；丁香、藿香、草蔻，益智仁、吴茱萸辛热温中止呕为臣；人参、当归、白术、陈皮、白芍、炙甘草苦甘温，补中益气，调和血脉，协力为佐使也。
用药禁忌：None
附注：None
