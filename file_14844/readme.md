序号：14851
名称：夺命丹
组成：朱砂1钱半，胆矾1钱半，真血竭1钱，铜绿1钱，枯白矾2钱，雄黄3钱，蟾酥半钱，轻粉半钱。
出处：《经验良方》引曾守壹方（见《医方类聚》卷一七九）。
主治：疔疮毒气向里，邪气入内，淫闷不已，水食不下，兼治痈疽发背，恶疮。
加减：None
功效：None
用法用量：每服1丸，先用葱白3寸，令病人自嚼碎，吐出碎葱裹药在内，热酒送下，睡卧，如重车行五里之久，汗出或发热1阵即愈，或利为度，如病人不能嚼葱，擂碎葱白裹药，如前吞服，如不省人事，斡开口灌之。病在上，食后服；病在下，食前服。
制备方法：上为细末，面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：万灵夺命丹（《奇方类编》卷下）。
