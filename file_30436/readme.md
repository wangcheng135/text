序号：30444
名称：进退四物汤
组成：熟地5钱，当归3钱，酒炒白芍1钱5分，川芎1钱。
出处：《医方简义》卷五。
主治：None
加减：血热先期者，加丹参、丹皮、益母草各2钱；血寒后期者，加肉桂5分，牛膝、香附各2钱；先腹痛而经至，小腹(疒丂)痛者，加青皮（炒）1钱，泽兰、香附、延胡索各2钱，青木香4分，减去熟地1味，倍加川芎1钱；经净而腹痛者，加人参2钱，桂枝5分，香附、延胡各2钱，减熟地1半，加炒白芍1钱5分；带下，加鹿角霜2钱，煅龙骨2钱，左牡蛎4钱，仙居白术2钱，砂仁5分，去熟地、川芎2味；白淫多者，加茯苓、白术、东洋参各1钱5分，进白芍1半，退熟地1半；白淋多者，加琥珀1钱5分，乌贼骨1钱，滑石、淡竹叶各3钱；经水色黄而淡者，加肉桂5分，茺蔚子3钱，党参3钱，柴胡（醋炒）8分；心神摇漾，加茯神3钱，远志肉（炒）1钱，琥珀8分，灯草1扎；风虚眩晕者，加煨天麻1钱，姜半夏1钱，倍川芎1钱；如呕者，加姜3片，川连8分，姜半夏1钱5分，淡吴萸8分，减熟地1半；肢逆冷者，加桂枝7分，竹茹1丸半，姜3片。
功效：调经。
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
