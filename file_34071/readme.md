序号：34080
名称：赤膏
组成：桂心1两，大黄1两，白术1两，细辛1两，川芎1两，干姜2两，丹参5两，蜀椒1升，巴豆10枚，大附子2枚
出处：《千金》卷六。
主治：耳聋，齿痛。
加减：None
功效：None
用法用量：以苦酒2升，浸一宿，纳成煎猪肪3斤，火上煎，三上三下，药成，去滓，可服可摩。耳聋者，绵裹纳耳中；齿冷痛则着齿间；诸痛皆摩；若腹中有病，服如枣许大，酒调下。咽喉痛，取枣核大吞之。
制备方法：上（口父）咀。
临床应用：None
各家论述：《千金方衍义》：耳聋多缘痰湿闭遏，齿痛良由寒菀热邪，故用桂、附、椒、姜以破少阴之结，芎、细辛以散脑户之邪，白术、丹参逐湿和营，大黄、巴豆一开热结，一破寒结，苦酒、猪脂，与前陈醋、鸡膏同意，但鸡走肝而猪达肾，稍有不同，其润窍之用则一。可服可摩，可治久聋，可治齿痛，盖耳与齿总皆属肾也。
用药禁忌：None
附注：丹参膏（《圣济总录》卷一一四）。
