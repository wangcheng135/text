序号：39882
名称：三肾丸
组成：熟地6两，丹皮2两，广砂仁2两，锁阳2两，苁蓉2两，车前2两，茯苓2两，故纸2两，枸杞2两，川断2两，白术6两，附子5钱，川芎8钱，萸肉1两5钱，怀膝1两5钱，制草1两5钱，山药3两，杜仲2两，泽泻2两，当归2两，羊藿2两，丝瓜2两，白芍1两，肉桂5钱，广木香8钱，首乌2两，黑驴肾1具，黄狗肾1个。
出处：《全国中药成药处方集》（沈阳方）。
主治：肾脏衰弱，畏寒怕冷，过劳气喘，四肢疲乏，下焦虚寒，腰腿酸痛，生殖机能减退，&mdash;切肾病偏于寒凉者。
加减：None
功效：温暖肾脏，强精壮髓。
用法用量：每服1丸，淡盐汤送下。
制备方法：上为极细末，炼蜜为丸，重2钱。
临床应用：None
各家论述：None
用药禁忌：忌食生冷，禁房事。
附注：None
