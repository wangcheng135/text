序号：27354
名称：补益巴戟天丸
组成：巴戟天（去心，酒浸，焙）2两，肉苁蓉（去皴皮，酒浸，切，焙）2两，白龙骨2两，五味子2两，鹿茸（去毛，酥炙）2两，白茯苓（去黑皮）2两，天雄（炮裂，去皮脐）2两，续断2两，山芋2两，白石英2两，覆盆子3两，菟丝子（酒浸，别捣）3两，熟干地黄（焙）2两，蛇床子（炒，去皮）1两，远志（去心）1两半，干姜（炮裂）1两半。
出处：《圣济总录》卷二十。
主治：阳衰阴盛，痹气身寒。
加减：None
功效：None
用法用量：每服20丸，加至30丸，空心温酒送下，1日2次。
制备方法：上除菟丝子别捣外，同捣罗为末，入菟丝子拌匀再罗，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“巴戟天丸”。
