序号：42881
名称：保坤至圣丸
组成：香附子8两（童便、酒、醋、盐水各制1次），当归身2两（酒浸，晒干，醋拌炒），大熟地3两（酒洗蒸，醋炒），川芎1两5钱（醋炒），白芍1两5钱（酒拌炒），延胡索2两（醋炒），白茯苓2两（人乳拌蒸，晒），牡丹皮1两（酒洗，晒干，醋拌炒），白术2两（土拌，切片，麸炒），绵黄耆1两5钱（蜜水拌炙），粉甘草1两5钱（蜜水拌炙）。
出处：《集验良方》卷五。
主治：妇女经闭淋崩，产后诸虚百损，久无子嗣。
加减：None
功效：None
用法用量：每服50丸，空心淡盐汤或淡醋汤送下，1日2次。
制备方法：上为细末，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
