序号：26807
名称：延寿水仙丹
组成：人参半两，木通半两，白及半两。
出处：《普济方》卷十六。
主治：五脏六腑百病，心热多躁，心虚不足，夜卧不稳，口苦舌干，小便白浊，日夜无度，色如凝脂，其味或甜，肢体羸瘦，不生肌肉，有潮热，久坐生劳，倦怠乏力。
加减：None
功效：养精神魂魄，益气明目，通血脉，止烦满消渴。
用法用量：每服3丸，日正午时浸朱砂水送下。
制备方法：上用真麻油2两，入3味药在油内，用建盏盛之，火上煎，频将柳枝打匀，候3味药黑焦色，即去之，再煎油，滴水上如珠不散为度。用透明朱砂不拘多少，研细如粉，将炼油搜成2剂，用皂角水洗去油，入新水中浸之，逐日换水，就水旋丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
