序号：2155
名称：七宣丸
组成：大黄15两，枳实（炙）5两，青木香5两，柴胡5两，诃黎勒皮5两，桃仁6两（去皮尖，熬），甘草4两（炙）。
出处：《外台》卷三十一引《必效方》。
主治：风气结聚，宿食不消，兼沙石、皮毛在腹中；积年腰膝疼痛，寒冷如冰石；脚气冲心，愤闷将死，头旋暗倒，肩背重闷，心腹胀满，胃膈闭塞；风毒肿气连及头面，及大小便或利涩，脾胃气不理，不能饮食，夜卧脚转筋，脉掣痛，恍恍然眠寝不安。
加减：None
功效：None
用法用量：每服20丸，以酒送下，稍加至50丸，病在下，空腹服，病在上，食后服。以宣利为度。若是初生孩子，可与3丸、5丸，稍稍加之，取通利。
制备方法：上为末，蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：①《外台》引《必效方》：此药功效不可尽说，如前十数种病，则须服七宣丸。自外轻病，不妨与五补丸兼服，循环不辍。②按：本方为原书“五补七宣丸”之第二方。
