序号：33281
名称：一点消
组成：盐4两，矾4两，消4两，皂矾4两，汞4两，砒4钱。
出处：《灵药秘方》卷下。
主治：一切大小疮毒初起。
加减：None
功效：None
用法用量：治毒时，以米醋少许，灯草蘸点患处。毒小可点2-3点，起出1-2个白泡即消。毒大者，多点几点，或多点几次，亦无不消。如遇顽阴之毒，服奇命丹1眼，点之亦无不收功。
制备方法：上共研不见星，入罐，微火结胎，用木棍筑实，冷定覆于碗上。碗底放水盆1个，砖1块，放碗、罐于砖上，加水至碗底8分，另以大砖隔住，砌百眼炉，上火，下火看火到底即退火，冷定取药收固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
