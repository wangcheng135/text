序号：30299
名称：阿魏理中丸
组成：阿魏1分（用面二匙，醋和阿魏作饼子，炙令黄），京三棱（煨，切）1两半，蓬莪术（煨，切）1两半，青橘皮（去白）1两半，陈橘皮（去白）1两半，甘草（炙）1两半，干姜（炮）1两半，干木瓜1两半，肉桂（去粗皮）1两半，白术1两半。
出处：《杨氏家藏方》卷五。
主治：一切冷气攻刺疼痛，心腹胀满，胃冷吐逆，脐腹撮痛。
加减：None
功效：None
用法用量：每服1丸，食前细嚼，生姜、木瓜盐汤送下；如妇人血气攻刺，煎干姜、当归汤嚼下。
制备方法：上为细末，面糊为丸，每1两作15丸，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
