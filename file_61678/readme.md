序号：61694
名称：排石汤
组成：柴胡15g，黄芩15g，郁金15g，枳壳15g，姜黄15g，青皮15g，大黄15g（后下），白芍15g，山楂10g，川楝子12g，金钱草30g。
出处：《古今名方》引河南省人民医院方。
主治：肝胆管结石，总胆管结石，胆囊结石，胆道术后残余结石、胆道泥沙样结石等。
加减：腹痛重，加延胡索；呕吐，加竹茹、半夏；高烧感染，加金银花、蒲公英、连翘；湿热黄疸重，加茵陈、栀子、龙胆草。
功效：清热利湿，通淋排石。
用法用量：水煎服。或可配合“总攻”：上午8时口服排石汤，9时电针右侧日月、期门两穴2小时，10时服50％硫酸镁30-50毫升。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
