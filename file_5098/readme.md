序号：5098
名称：木通膏
组成：木通2两（锉），露蜂房2两，连翘2两（锉），黄芩2两（锉），商陆2两（锉），黄耆2两（锉），牛蒡根2两（锉），乳头香2两（细研），松脂2两，蜡1两，黄丹7两，羊肾脂3两，绯帛1尺（烧灰，细研），消石1两（细研），曲头棘针100枚。
出处：《圣惠》卷六十三。
主治：发背及诸痈疽疮。
加减：None
功效：None
用法用量：每用涂于帛上贴之。如肿未成脓，即肉消；已成脓，即日2贴之。
制备方法：上药以生麻油2斤于铛中，文火煎令香，下锉药，急火煎，候药色赤黑，下松脂、蜡消，以绵滤过滓，下黄丹及羊脂，搅勿住手，候色黑，时时点于铁上试看，凝如饧，去火，适火热下乳香、帛灰、消石等，搅匀，用不津器盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
