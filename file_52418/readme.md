序号：52433
名称：香甲散
组成：鳖甲3两（醋浸去裙，炙黄，又入醋蘸，炙7次），当归半两，木香半两，人参半两，羌活半两，川芎半两，沉香半两，肉豆蔻半两，酸枣仁半两，附子半两，槟榔半两，大腹子半两，北柴胡半两，厚朴1两，川牛膝1两，白茯苓1两，秦艽1两，桂心半两。
出处：《妇人良方》卷五。
主治：热病后虚劳，或四肢倦怠，脚手疼痛，饮食无味，肌肤黄瘦，或热疟盗汗，头晕虚烦。
加减：None
功效：久服驻颜。
用法用量：上（口父）咀。每服5钱，以水1盏，加生姜3片，乌梅1个，煎至8分，去滓，空心温服，日3次。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌生冷、面食、鸡肉、酢酱之类。
附注：None
