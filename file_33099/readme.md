名称：万应灵膏
组成：当归1两，赤芍1两，川军1两，白及1两，白蔹1两，羌活1两，乌药1两，木鳖子1两，苦参1两，连翘1两，皂角1两，生地1两，防风1两，甘草1两，山柰1两，五灵脂1两，半夏1两，槐枝1两，柳枝1两，桃枝1两，枣枝1两，桑枝1两。
出处：《青囊秘传》。
主治：筋骨疼痛，跌打损伤，一切泻痢，妇人赤白带下，肚痛。
加减：None
功效：None
用法用量：红布摊贴。
制备方法：麻油5斤，入药煎枯去渣，下净血余2两，烊化，再入炒过广丹2斤，熬成膏，入后细料药：细辛1两，附子1两，良姜1两，官桂1两，乳香1两，没药1两，丁香1两，甲片1两，洋樟1两，川草乌1两，阿魏1两；麝香1钱，研末，调入膏内。
临床应用：None
各家论述：None
