序号：34464
名称：驱疫丹
组成：生石膏5两，黄连3两，麻黄2钱，知母2两，白芷1两，生硼砂1两，槟榔1两，苏叶7钱，广木香5钱，母丁香1钱，檀香2钱，炒苍术4钱，菖蒲4钱，香薷1两，菊花1两，茅慈菇3钱，甘草5钱，黑郁金3钱，木瓜5钱，山柰2钱，藿香叶1两5钱，红大戟（醋制）5钱（上共为细末），血竭面5钱，雄黄面1钱，琥珀面1两，朱砂面6两，牛黄1钱，麝香1钱，苏合油5两，薄荷冰1两5钱，冰片1两5钱。
出处：《全国中药成药处方集》（天津方）。
主治：头昏呕吐，腹痛泄泻，手足厥冷，昏迷不省，中暑中寒，发烧发冷，四肢痠痛，湿郁闷胀。
加减：None
功效：避秽排浊，驱疫除瘟。
用法用量：每服2分5厘，白开水送下。重者加倍。
制备方法：先将苏合油、薄荷冰、冰片共研成水，再和以上细粉研匀。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
