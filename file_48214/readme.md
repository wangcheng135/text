序号：48229
名称：神明椒菊丸
组成：川椒1两，甘菊2两，生地黄（洗）1升。
出处：《普济方》卷八十三引《家藏方》。
主治：目睛失明，而睛不损者，或十分不见，以及冷泪及睑紧睑肿，肝虚肝热肾风攻注眼目及患昏花。
加减：None
功效：None
用法用量：每服30丸，以温水送下。半月取效，见2-3分，服药百日收全功。
制备方法：上生地黄控断水脉，入木臼内烂捣，或砂盆内烂研，以绢袋绞取自然汁，可得12两以上，去滓不用；将川椒、甘菊入地黄汁内，浸少时流出，俟水脉断，入慢火焙之，约8-9分干，再入地黄汁，再漉再焙，如此以汁尽为度，焙令透干，木臼内捣为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
