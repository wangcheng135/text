序号：44246
名称：上丹
组成：五味子半斤，百部（酒宿浸，焙）2两，玉女（即菟丝子，酒宿浸，焙）2两，苁蓉（酒宿浸）2两，思仙木（即杜仲，炒）2两，不凋草（即巴戟，去心）2两，细草（既远志，去心）2两，仙人杖（即枸杞子）2两，防风（无叉枝者）2两，白茯苓2两，思益（即蛇床子，炒）2两，柏子仁（另研）2两，干薯蓣2两。
出处：《元和纪用经》。
主治：男子绝阳，庶事不堪，女子绝阴，乃不能妊。腰膝重痛，筋骨衰败，面黑，心劳志昏，寤寐恍惚，烦愦多倦，余沥梦遗，膀胱邪气，五劳七伤，肌肉羸悴，上热下冷。
加减：夏，五味子加4两，通称12两；_x000D_
四季，苁蓉加6两，通称半斤，各18日四立之前也；秋，仙人杖加6两；冬，细草加6两；戊寅、戊申相火司天，中见火运，饭后兼饵养肺平热药。
功效：主养五脏，补不足，秘固真元，均调二气，和畅荣卫，保神守中；久服轻身耐劳，健力能食，明目，降心火，交肾水，益精气，开心臆，安魂魄，消饮食，养胃和中。
用法用量：每服20丸至30丸，食前温酒送下；不饮者，盐汤送下；春，干枣汤送下。
制备方法：上为末，蜜煎面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：不犯金、石、桂、附。
附注：本方为原书“耘苗丹”之第一方。
