序号：3835
名称：心丹
组成：朱砂五10两，罗参5两，远志（去心，甘草煮）5两，熟地黄（酒蒸，焙）5两，白术5两，石菖蒲5两，黄耆5两，当归（酒浸，焙）5两，麦门冬（去心）5两，茯苓5两，茯神5两，柏子仁5两，木鳖子（炒，去壳）5两，石莲肉5两，益智仁5两。
出处：《永类钤方》卷十三引《济生》。
主治：男子妇人心气不足，神志不宁，怔仲惊悸，一切心疾。
加减：None
功效：None
用法用量：每服20丸，食后参汤、枣汤、麦门冬汤任下。
制备方法：上先以人参等14味，各如法修制，锉碎拌匀，次将朱砂滚和，以夹生绢袋盛贮，线缚袋口，却用瓦锅1口，盛水7分，重安银罐1个于锅内，入白沙蜜20斤，将药袋悬之中心，不令着底，使蜜浸过药袋，以桑柴火烧令滚沸，勿使火歇，煮3日蜜焦黑，再换蜜煮，候7日足，住火取出，淘去众药，洗净朱砂，令干，入牛心内，仍用银罐子重汤内蒸，如汤干，复以热水从锅弦添下，候牛心蒸烂，取砂再换牛心，如前去蒸，凡7次，其砂已熟，即用沸水淘净，焙干，入乳钵玉杵研至十分细，米棕为丸，如豌豆大，阴干。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“法丹”。
