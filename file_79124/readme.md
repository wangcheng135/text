序号：79140
名称：磨风丸
组成：川当归1两，羌活1两，独活1两，川芎（小者）1两，明天麻1两，细辛1两，防风1两，荆芥1两，威灵仙1两，麻黄（去节）1两，何首乌1两，石京子1两，牛蒡子1两，虾（虫麻）叶1两，皱面草（即地松）1两，苍耳草1两。
出处：《医方类聚》卷二十四引《急救仙方》。
主治：疠风。
加减：None
功效：None
用法用量：每服30丸，食前食后皆可，日2次。
制备方法：晒干，为末，不可见火，好酒煮米糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：原书治病风，第一日服消风散，第二日服追风散，第三日服本方。同时用洗药：地骨皮、荆芥、苦参、细辛各三两。锉碎和匀，每次用二两，以河水煎，熏洗遍身，出血为效。如洗必用大浴桶或缸内，要汤宽，浸通身良久妙。
