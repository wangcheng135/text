序号：33119
名称：万应神曲
组成：荆芥8两，防风8两，桔梗8两，冬术8两，苍术8两，香薷8两，葛根8两，枳壳8两，苏叶8两，羌活8两，诃子8两，青蒿8两，川厚朴8两，淡黄芩8两，片黄柏8两，明天麻8两，生香附8两，粉甘草8两，蕲艾绒8两，绵茵陈8两，苍耳草8两，山楂肉8两，使君子8两，赤小豆8两，生谷芽8两，生麦芽8两，川连8两，川郁金8两，川桂枝8两，上沉香8两，广木香8两，阳砂仁8两，白芍药8两，广陈皮8两，槟榔片8两，石菖蒲8两，熟半夏8两，高良姜8两，天花粉8两，白扁豆8两，福泽泻8两，柴胡8两，薄荷叶8两，川木通8两，白杏仁8两，车前子8两。
出处：《续补集验良方》。
主治：四时不正之气，感冒发热，头眩咳嗽；及伤食腹痛，痞满气痛，呕吐泄泻，痢疾，饮食不进；不服水土，瘴气疟疾。
加减：外感发热头眩，咳嗽，疟疾，呕吐，加生姜；泄泻，加乌梅；痢疾，须加倍用，再同好陈茶心煎服。
功效：搜风解表，开胸快膈，调胃健脾，消积进食，和中解酒，止泻利水。
用法用量：大人每服3钱，水1汤盏，煎7分。小儿减半，或1钱，水1大茶钟，煎7分。每钱破作5-6块。
制备方法：上药46味，加酒药20丸，共研细末，用淮小麦1斗，煮熟捣烂，入诸药内拌匀，遏成曲，另用白干面2斤，将滚水泡成饼，遏存性，候过5-6日后，用热水将面饼化浆，共入石臼内捣烂，做成小块，重不过两，晒干。
临床应用：None
