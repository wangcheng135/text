序号：67108
名称：麻黄汤
组成：麻黄（去根节，先煎，掠去沫，焙干）1两半，独活（去芦头）1两，细辛（去苗叶）半两，黄芩（去黑心）半两。
出处：《圣济总录》卷八。
主治：中风四肢拘挛，百节疼痛，心烦，恶寒渐渐，不欲饮食。
加减：有热，加大黄2分（锉如麻豆，用醋炒令紫色）；腹满，加枳壳2分（去瓤，麸炒）；气逆，加人参2分；胁下悸满，加牡蛎灰2分；渴加栝楼根2分；素有寒，加附子1枚（炮裂，去皮脐）。
功效：None
用法用量：每服5钱匕，以水2盏，煎至1盏，去滓，空心温服，相去如人行5-7里再服。微汗即愈。病在四肢者，并为1次服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
