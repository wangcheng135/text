序号：2973
名称：天浆子散
组成：天浆子1分，蝎梢1分，犀角屑1分，丹砂（研）1分，雄黄（研）1分，附子（炮裂，去皮脐）1分，天南星（炮）1分，白附子1分，半夏（汤洗去滑，与生姜汁同捣，捏着饼子，晒干）1分，水银（黑铅结成沙子）1分，乳香（研）1分，白花蛇（酒浸，炙，用肉）1分，白僵蚕（炒）1分，腻粉1钱，牛黄（研）1钱，麝香1字，金箔3片，银箔3片。
出处：《圣济总录》卷一七二。
主治：小儿天钓，惊风。
加减：None
功效：None
用法用量：每服1字，薄荷汤调下，1日3次。
制备方法：上药除别研外，捣罗为散，入研药和匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
