序号：33053
名称：芦荟丸
组成：南木香2钱半，丁香2钱半，诃子（去核，取肉）半两，肉豆蔻半两，使君子肉4钱，芦荟4钱，枣肉1两（薄切，用屋瓦盛，慢火焙干）。
出处：《活幼心书》卷下。
主治：五瘠八痢，脏腑虚弱，身体瘦悴，头发焦疏，腹胀青筋，小便白浊，喝水无度，洞泄不时，谷食难化，遍身疮疥，神色干燥。
加减：None
功效：养胃壮气，止痢除虫，长肌健力。
用法用量：每服30-50丸，温米汤空心送下；儿小米汤化下。
制备方法：上除使君子肉薄切，于乳缽内，极细杵，仍将前南木香等4味，湿面裹，煨至香熟取出，地上候冷，去面锉，焙，同枣肉、芦荟为细末，再入乳缽，同使君子肉杵匀，炼蜜为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
