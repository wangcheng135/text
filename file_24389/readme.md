序号：24397
名称：八仙妙应丹
组成：雷丸1两，锡灰1两半，白芜荑1两，木香1两（不见火），锦纹大黄1两，槟榔12两（鸡心者），使君子1两（取净），黑丑头末3两（不见火）。
出处：《丹溪心法附余》卷十八。
主治：男子、妇人、小儿外感内伤，以致水谷停留肠胃，生虫成积，恶心呕吐，吞酸嘈杂，疟痢黄疸，水肿臌胀，膈噎翻胃；妇人癥瘕积聚，心腹疼痛；小儿疳癥，面黄肌瘦，肚大脚细，一切虫积。
加减：加以硇砂、甘遂尤妙。
功效：驱虫，化积，消气。
用法用量：每服4钱，病重年深体实者，加至5钱，葱白汤送下；或木香汤送下。十五岁以上者可服，三岁、七岁者，每1服作3服，早晨空心冷水送下，有虫即取下虫，有积即取下积，有气即消了气。
制备方法：上为细末，加葱白1斤，煮沸，露1宿，为丸，如粟米大。
临床应用：None
各家论述：None
用药禁忌：务在房内坐桶，不要见风、出外。如虚老之人用此推荡后，服四君子汤数帖尤好。
附注：None
