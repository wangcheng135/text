序号：3446
名称：太上五蛊丸
组成：雄黄4分，椒目4分，巴豆4分，莽草4分，芫花4分，真朱4分，鬼臼4分，矾石4分，藜芦4分，斑蝥30枚，蜈蚣2枚，獭肝1分，附子5分。
出处：《千金》卷二十四。
主治：百蛊，吐血伤中，心腹结气，坚塞咽喉，语声不出，短气欲死，饮食不下，吐逆上气，去来无常，状如鬼祟，身体浮肿，心闷烦疼，寒战，梦与鬼交，狐狸作魅，卒得心痛、胸胁刺痛，经年着床不起者。
加减：None
功效：None
用法用量：先食饮服1丸，余密封，勿泄药气。10丸为1剂，如不中病，后日增1丸，以下痢为度。下后7日将息，服1剂，30年百病尽除。
制备方法：上为末，炼蜜为丸，如小豆大。
临床应用：None
各家论述：《千金方衍义》：南粤之乡，多畜百虫，故治蛊之方，多用阳药。如太上五蛊丸，首取雄黄纯阳之精；佐以附子、椒目破除阴毒，以壮巴豆、莽草、鬼臼、芫花、藜芦、矾石破癥坚积、鬼痊蛊毒之咸；獭肝、蜈蚣、斑蝥，专杀尸疰蛊毒；真珠禀离方真火，力破五脏之百邪，为五蛊首列之神方也。
用药禁忌：忌五辛。
附注：五蛊丸(《圣济总录》卷八十引《膜外气方》)、雄黄丸(《普济方》卷二五三)。
