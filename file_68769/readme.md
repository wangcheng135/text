序号：68785
名称：小和中饮
组成：陈皮1钱5分，山楂2钱，茯苓1钱半，厚朴1钱5分，甘草5分，扁豆（炒）2钱。
出处：《景岳全书》卷五十一。
主治：胸膈胀满，或妇人胎气滞满；因食而成疟痞；病后浊气未净，或余火未清，胃口不开，饮食不进；呕逆；食饮寒凉，或误食性寒生冷等物，致伤胃气因而作呕，寒滞未散而兼胀兼痛；吐利因于过食，或瓜果生冷，以致食留不化，遂成痞膈霍乱；积聚，不堪攻击，只宜消导渐磨者；恶阻，饮食停滞作胀；小儿伤食呕吐，但有食滞而胃不寒者；小儿痞块，兼胃腔停积，食滞作胀；痘疹，饮食停滞，中满作痛者。
加减：如呕者，加半夏1-2钱；如胀满气不顺者，加砂仁7-8分；如火郁于上者，加焦栀子1-2钱；如妇人气逆血滞者，加紫苏梗、香附之属；如寒滞不行者，加干姜、肉桂之属。
功效：None
用法用量：水1钟半，加生姜3-5片，水煎服。
制备方法：None
临床应用：None
