序号：79791
名称：磁石丸
组成：磁石2两（烧令赤，以醋淬7次，捣碎，研，水飞过），鹿茸1两5钱（去毛，涂酥炙微黄），人参（去芦头）1两，黄耆（锉）1两，白茯苓1两，远志（去心）3分，附子（炮裂，去皮脐）3分，牡蛎3分（烧为粉），牛膝1两（去苗），防风3分（去芦头），楮实子1两5钱（水淘，去浮者，焙干），五味子5钱，薯蓣3分，巴戟3分，石斛1两（去根，锉），桂心2分，熟干地黄1两，肉苁蓉3分（酒浸1宿，刮去粗皮，炙干）。
出处：《普济方》卷二三四。
主治：虚劳。肾脏乏弱，耳聋或常闻钟磐风雨之声。
加减：None
功效：补肾虚。
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
