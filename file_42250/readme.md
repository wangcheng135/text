序号：42265
名称：青盐丸
组成：茴香3两（炒香），菟丝子4两，千山药2两，青盐1两。
出处：《本事》卷二。
主治：肾虚及足膝无力。肝肾虚损，腰膝无力，颤振軃曳。
加减：None
功效：壮力进食。
用法用量：每服30-50丸，盐酒、盐汤任下。
制备方法：上将菟丝子洗淘，无灰酒浸，日中煎7日，冬天近火煨之，晒干别末，将余药末和匀，酒糊为丸，如梧桐子大。
临床应用：足軃曳一妇人素患足軃曳，久服此药，履地如故。
各家论述：《本事方释义》：大茴香气味辛温，入肾；菟丝子气味甘平，入脾；干山药气味甘平，入脾；青盐气味咸、微寒，入足少阴。下焦肾虚致足膝行走无力，其始必因肾家气弱不能运水，故必补脾之药，佐以酒浸引药入肾，以驱湿邪而本脏自安也。
用药禁忌：None
附注：None
