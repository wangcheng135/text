序号：35131
名称：乳香丸
组成：乳香（细研）1两，乌头（炮裂，去皮脐）半两，羌活（去芦头）1两，雄黄（细研，水飞过）半两，白附子（炮）1两，羚羊角屑1两，附子（炮裂，去皮脐）半两，原蚕蛾（微炒）1两，防风（去叉）1两，乌蛇（酒浸去皮骨，炙）1两，白僵蚕（炒）1两，虎胫骨（酥炙）1两，腻粉1分，麝香（细研）1分，赤箭半两，牛膝（酒浸，切，焙）半两，没药（研）半两。
出处：《圣济总录》卷八。
主治：中风，四肢拘挛，筋骨疼痛。
加减：None
功效：None
用法用量：每服10丸，用豆淋热酒送下，如人行10里再服。
制备方法：上为末，再同和匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
