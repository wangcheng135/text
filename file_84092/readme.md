序号：84110
名称：五圣散
组成：沉香流气饮、人参败毒散、小续命汤除去附子、三和散、香苏散。
出处：《朱氏集验方》卷一。
主治：脚气。因坐立湿地，或因行脚热脱鞋及着湿鞋，风湿之气入侵，发作时行步艰难，痛不可忍。
加减：None
功效：None
用法用量：水1盏，加生姜3片，大枣1枚，木瓜3片，同煎至1盏，滤滓再煎2次，与头药合和，用瓦器盛贮，将纱布盖瓦器口，安顿屋上露1夜，次早取下暖热，调复元通气散半帖，空心服。
制备方法：上5件各1帖，加槟榔30个(搜面包，煨熟去面)，及川楝子10文，京南芍药20文，共三件并锉碎，同前五贴药共一处和匀，作5服。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
