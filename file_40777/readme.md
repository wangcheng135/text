序号：40791
名称：转呆丹
组成：人参1两，白芍3两，当归1两，半夏1两，柴胡8钱，生枣仁1两，附子1钱，菖蒲1两，神曲5钱，茯神1两，天花粉3钱，柏子仁5钱。
出处：《辨证录》卷四。
主治：呆病。终日闭户独居，口中喃喃，多不可解，将自己衣服用针线密缝，与之饮食，时用时不用，尝数日不食，而不呼饥，见炭最喜食之。
加减：None
功效：大补心肝气血，祛 痰开窍。
用法用量：水10碗，煎1碗，使强有力者，抱住其身，另用2人执拿其两手，以1人托住其下颔，1人将羊角去尖，插其口灌之。倘不肯服，不妨以杖击之，使动怒气，而后灌之。服后必然骂詈，少顷必倦而卧，听其自醒，切不可惊动，自醒则全愈，否则止可半愈也。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
