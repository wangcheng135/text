序号：59135
名称：索氏三和汤三倍加白术方
组成：白术3两，厚朴3两，陈皮3两，木通1两，槟榔2两，紫苏2两，甘草1两，海金沙1两，大腹皮1两，白茯苓1两，枳壳1两。
出处：《医学纲目》卷二十四。
主治：小腹胀。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：疟久腹胀一男子年四十余，患疟久而腹胀，脉不数而微弦，重取则来不滑利，轻重又皆无力，遂与本方入姜汁服之，数服而疟愈，小便利二三行，胀稍减，遂又小便短少，予作气血两虚，于前药内入人参、牛膝、当归身尾作大剂料，百服而愈。
各家论述：None
用药禁忌：None
附注：索氏三和汤（《济阳纲目》卷三十九）。①本方方名，《东医宝鉴·杂病篇》引作“三和汤”。②《济阳纲目》本方用法：上锉。每服八钱，加生姜三片，水煎服。
