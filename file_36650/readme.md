序号：36660
名称：固元丹
组成：菟丝子1分，益智仁（去皮）1分，牛膝1分，石斛1分，黄耆1分，干地黄1分，桑寄生1分，萆薢1分，茯苓1分，芎1分，五味子1分，山茱萸1分，羌活1分，木香1分，虎骨（酥炙黄黑色）1分，肉豆蔻1分，厚朴（去皮、姜制）3铢，青橘皮3铢，鳖甲（醋炙）半两，阿魏3铢，肉苁蓉1分。
出处：《普济方》卷二二八。
主治：五劳七伤，元气亏弱，血气虚损，一切虚冷，百般劳证。
加减：None
功效：None
用法用量：每服40丸，早晨盐汤送下。
制备方法：上为细末，以好酒少许化阿魏，与蜜合炼为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
