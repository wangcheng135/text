序号：15816
名称：玄参散
组成：玄参3分，川升麻3分，川大黄3分（锉碎，微炒），甘草半两（炙微赤，锉），犀角屑3分。
出处：《圣惠》卷三十六。
主治：心脾壅热，生木舌肿胀。
加减：None
功效：None
用法用量：每服5钱，以水1大盏，煎至5分，去滓温服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：《本事方释义》：玄参气味咸苦，入手足少阴；升麻气味辛温，入足阳明；大黄气味苦寒，入足阳明；犀角气味苦酸咸，微寒，入手足厥阴；甘草气味甘平，入足太阴，能缓诸药之性。因心脾气壅痹不宣，非下行不能杀其势，速下犹恐热不尽，故以甘平之品缓其下行之势，则壅热去而无不尽矣。
用药禁忌：None
附注：None
