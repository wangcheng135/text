序号：22376
名称：阴湿汤
组成：白术1两，白茯苓1两，苍术（米泔浸）1两，藿香叶（去土）1两，甘草（炙）1两，橘红1两，厚朴（制）1两，半夏1两，附子6钱（炮），生姜2两。
出处：《普济方》卷八十八引《余居士选奇方》。
主治：一切中风自汗，渐渐恶风，翕翕发热，呼吸少气，风湿风温，表实里虚，或表虚里实，腠理开疏，气道壅塞，虚汗盗汗，目黄身肿，小便不利，胸膈痞满，腰疼体痛，呕吐涎沫。
加减：None
功效：None
用法用量：每服3钱，水2盏，加生姜10片，煎至1盏，不拘时候。
制备方法：上以厚朴、半夏、生姜一处捣作饼子，焙干，同众药为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
