序号：36720
名称：固本膏
组成：羊腰子1对，附子1两2钱，海马3个，鹿角（镑）1两2钱，芙蓉叶2两，石脂1两，雄黄面1两，阳起石5钱，小茴香2两5钱，苁蓉2两5钱，官桂2两5钱，补骨脂2两5钱，大茴香2两5钱，生地2两5钱，熟地2两5钱，天麻2两5钱，紫梢花2两5钱，牛膝2两5钱，续断2两5钱，甘草2两5钱，蛇床子2两5钱，菟丝子2两5钱，冬虫草5钱，杜仲2两5钱。
出处：《北京市中药成方选集》。
主治：男子气虚，梦遗滑精，腰痠腿痛；妇女血寒，腹痛白带。
加减：None
功效：滋补散寒，固精止痛。
用法用量：外贴肾俞、肚脐。
制备方法：上药碎断，用香油240两炸枯，过滤去滓，炼至滴水成珠，入黄丹90两，搅匀成膏，取出入水中出火毒后，加热溶化，再兑鹿茸粉1两3钱，搅匀摊贴，每张油重5钱，布光。
临床应用：None
各家论述：None
用药禁忌：孕妇忌贴。
附注：None
