序号：73173
名称：山茱萸丸
组成：山茱萸1两，干姜（炮）1两，巴豆（去皮壳，炒，别研）1两。
出处：《圣济总录》卷一一四。
主治：久聋。
加减：None
功效：None
用法用量：绵裹塞耳中。食顷于，即易新药塞之。凡如此5日，当小愈；10日闻人声，愈即止。
制备方法：上3味，先捣前2味为末，入巴豆，同研令匀，绞葱汁和丸，如枣核大。
临床应用：None
各家论述：None
用药禁忌：常以发塞耳孔，避风。
附注：《圣惠》卷三十六治“耳久聋”方用“吴茱萸半两（生用），巴豆二枚（去皮心），干姜一分（炮裂），上件药，捣细罗为散，以葱涕和，绵裹枣核大，纳耳中食顷，干即去之，更和湿者纳之。如此五日，当觉病去，八九日便闻入语声。常以发塞耳，避风”。据此，本方中山茱萸，疑为“吴茱萸”之讹。
