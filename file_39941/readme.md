序号：39951
名称：肥儿糖浆
组成：山药11两1钱，芡实11两1钱，莲子11两1钱，党参11两1钱，苡米（炒）11两1钱，扁豆（炒）11两1钱，山楂11两1钱，白术（焦）5两5钱，麦芽（焦）8两4钱，茯苓8两4钱，白糖2672两，苯甲酸钠9两6钱，蛋白干1两3钱。
出处：《中药制剂手册》引武汉市健民制药厂。
主治：小儿由脾胃虚弱引起的食欲不振，消化不良，面黄肌瘦，精神困倦。
加减：None
功效：健脾胃，补虚弱。
用法用量：2-4岁每服10毫升（约1汤匙），日服3次，开水冲服。
制备方法：将山药至茯苓10味共轧成Ⅰ号粗末；取山药等粗末用煮提法提取二次，加热浓缩至约5000毫升，放冷；再加入一倍量95％乙醇，滤除沉淀，回收乙醇，浓缩至约3000毫升；另取白糖与适量清水置锅内加热溶化，并将蛋白干用水溶解，过滤，浓缩至约97000毫升，与前药液混合，灌瓶，每瓶100毫升。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
