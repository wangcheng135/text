序号：48520
名称：神效葶苈散
组成：甜葶苈3两（隔纸炒令紫色），牵牛子2两半（微炒），猪苓2两（去黑皮），泽泻2两，椒目1两半（微炒）。
出处：《圣惠》卷五十四。
主治：十水之病，百方不愈，面目四肢俱肿，气息喘急，寝卧不得，小便渐涩，肿胀气闷，水不入口，垂命欲死。
加减：None
功效：None
用法用量：取葱白3茎（切），以浆水1大盏，煎取半盏，入清酒半盏，搅令匀，稍热空腹，调下3钱，以人行5里已来，即煮浆水粥，切入葱白，煮令烂熟，更入清酒5合，搅匀，面向东，热吃令尽。至午后来，或小便下3-5升，或大便通利，气喘即定，肿减7分，隔日后再服，百日内切好将息。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：不得吃盐及诸面食。
附注：神助散（《局方》卷八）、葶苈散（《圣济总录》卷七十九）。
