序号：25568
名称：朱砂水银煎
组成：朱砂1分，水银1分，巴豆1分，粉霜1分，代赭5钱，蒌葱5钱。
出处：《鸡峰》卷十九。
主治：水气。
加减：None
功效：None
用法用量：患者安脐中，以软纸1张，折盖之后，用裹肚系定，令病人卧，只1两食饭间，其疾自小便出水约2-3斗，便住，取出药，恐过度。却用生姜汁调黄连末半两，作1枚安脐中，依前用纸盖裹肚系定。煎桑白皮汤煮大麦面馎饦吃之。愈后3日用药补之。
制备方法：上为细末，蒸饼为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：忌醃藏、生冷、盐、醋等物。
附注：补药：海蛤、破故纸、白甘遂、木香各等分。上为细末，每服半钱，空心米饮调下。
