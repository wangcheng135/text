名称：泽兰散
组成：泽兰叶（炙）1两，人参1两，蜀椒（去目并闭口，炒出汗）1两，厚朴（去粗皮，生姜汁炙）半两，桂（去粗皮）半两，细辛（去苗叶）半两，芜荑仁（微炒）半两，藁本（去苗土）半两，当归（切，焙）半两，干姜（炮）半两，代赭半两，山茱萸半两，防风（去叉）半两，柏子仁（炒）3分，芎3分，牡蛎粉3分，熟干地黄（焙）3分，甘草（炙，锉）3分，龙骨3分。
出处：《圣济总录》卷一五三。
主治：妇人血伤不止，兼带下不断，虚羸困倦。
加减：None
功效：补血益气。调中止血。
用法用量：每服2钱匕，温酒调下；米饮亦得。
制备方法：上为散。
临床应用：None
各家论述：None
