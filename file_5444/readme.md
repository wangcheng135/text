序号：5444
名称：二气丹
组成：大黄4两（别为末，醋1升，慢火熬成膏子），当归2两，白芍2两。
出处：《宣明论》卷十一。
主治：月水不调，断绝不产，面黄肌瘦，恒不思美食，经闭脉数涩，左右强弱不调者。
加减：如月水不通，加入干漆3钱（炒焦用），没药半两，硇砂3钱（研），官桂2钱，斑蝥3钱（去头足，炒热用，生用则吐泻）。
功效：None
用法用量：每服20丸，食前淡醋汤送下，每日3次。有燥热，以柴胡饮子相参服之。
制备方法：上为末，以膏子为丸，如梧桐子大。
临床应用：None
各家论述：《医略六书》：热瘀不清，经血暗耗，故经脉闭遏，月事不行焉。当归养既耗之血，白芍敛热伤之阴，大黄净汁，熬膏入药，丸服。醋以引之入肝，饮以漱之和胃，使热化血荣，则冲任蓄泄有权，何患经闭不通乎！
用药禁忌：None
附注：二气丸（《济阴纲目》卷二）。本方方名，原作“二气汤”，与剂型不符，据《医方类聚》改。
