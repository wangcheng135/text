序号：83787
名称：乌蛇丸
组成：乌蛇1两（酒浸，去皮骨，炙令微黄），天浆子20枚（去壳），天麻半两，天南星半两（炮裂），干蝎1分（微炒），白附子半两（炮裂），附子1两（炮裂，去皮脐），防风半两（去芦头），半夏半两（汤洗7遍，去滑。以上9味，都以酒浸7日后，取出焙干，捣罗为末），牛黄1分，龙脑1分，麝香1分，朱砂1分，雄黄1分（以上5味，同研如粉）。
出处：《圣惠》卷八十三。
主治：小儿中风痉，及天钓惊痫，一切诸风。
加减：None
功效：None
用法用量：每服3丸，用薄荷汤送下。
制备方法：上为末，用糯米饭为丸，如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：乌梢丹（《普济方》卷三六七）。
