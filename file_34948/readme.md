序号：34958
名称：麦冬汤
组成：青蒿1小握，葱白（1寸长）7根，蓝叶7片，苦楝根7寸。
出处：《慎斋遗书》卷七。
主治：心中烦热，惟欲露体，以衣被复之即闷，惊悸心怯，面无颜色，忘前失后，妇人患血风气者，多成此证。乃是心蒸之状。
加减：None
功效：None
用法用量：上药用童便1升半，煎取1半，去滓，加安息香1钱，苏合香1钱，阿胶1钱，朱砂5分，雄黄5分，雷丸5分，枯矾5分，硫黄5分，槟榔末1钱5分，麝香5分，五更初空心进1服，五更五点进1服。午时前后取出虫，净桶盛，急入油铫内煮，仍倾盖虫罐内，扎口埋之深山。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：本方名“麦冬汤”，但方中无麦冬，疑脱。
