序号：44814
名称：恒山丸
组成：恒山半两，野狸头骨1分，虎头骨1分，猢狲头骨1分，天灵盖1分，绿豆（末）3分，臭黄1分（细研），安息香2分，朱砂1分（细研），雌黄1分（细研），砒霜3分，乳香1分，阿魏1分，白芥子2分。
出处：《圣惠》卷五十二。
主治：寒疟，阳虚阴盛，内外俱寒，四肢颤掉。
加减：None
功效：None
用法用量：未发时，以绛囊盛，于中指上系1丸，男左女右。3日如不住，以熟水送服1丸。
制备方法：上药并生为末，用软饭和捣三二百杵，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：有娠妇人及小儿不得服。忌食热物。
附注：本方方名，《幼幼新书》引作“常山丸”。
