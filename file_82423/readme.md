序号：82439
名称：薯蓣丸
组成：干薯蓣2两，苁蓉4两，牛膝2两，菟丝子2两（酒渍），杜仲2两，赤石脂2两，泽泻2两，干地黄2两，山茱萸2两，茯苓2两，巴戟天2两（去心），五味子1两半，石膏2两（研），远志1两（去心），柏子仁1两，白马茎筋（干之）2两（炙）。
出处：《外台》卷十七引《古今录验》。
主治：丈夫五劳七伤，头痛目眩，手足逆冷，或烦热有时，或冷痹骨疼，腰髓不随，食虽多不生肌肉，或少食而胀满，体涩无光泽，阳气衰绝，阴气不行。
加减：None
功效：补十二经脉，起发阴阳，通内制外，安魂定魄，开三焦，破积聚，厚肠胃，消五脏邪气，除心内伏热，强筋练骨，轻身明目，除风去冷。
用法用量：每服20-30丸，空腹用酒送下，日2次。
制备方法：上药治下筛，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌大酢、芜荑、蒜，陈臭物。
附注：None
