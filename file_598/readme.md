序号：598
名称：五福延寿丹
组成：五味子2两，人参2两，远志2两，石菖蒲2两，山萸（去核）2两，大茴香2两，生地（姜酒炒）2两，熟地黄2两，杜仲2两，白茯苓2两，肉苁蓉（酒浸）4两，枸杞子3两，菟丝子3两，山药3两，牛膝（酒浸）7钱半，川椒（去目，炒）7钱半，缩砂1两5钱，黄柏（酒炒）8钱，知母（酒炒）1两，木瓜1两，覆盆子1两。
出处：《济阳纲目》卷六十四。
主治：一切元气虚弱，五劳七伤，身体羸瘦，膝痠疼。
加减：None
功效：消除百病，常服延年耐老。
用法用量：每服40-50丸，空心好酒或淡盐汤送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
