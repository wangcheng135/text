序号：55820
名称：桑白皮汁十味煎
组成：桑白（切）1升，地骨皮（切）3升（2味用水7升熟煎，取3升汁，去滓澄清），生地黄汁5升，生麦门冬汁2升，生姜汁1升，竹沥3升，生葛根汁3升，白蜜1升，牛酥3合，大枣膏1升。
出处：《外台》卷九引《许仁则方》
主治：咳嗽经久，将成肺痿，昼夜咳嗽不断，唾白如雪，细末粘稠，喘息气上，乍寒乍热，发作有时，唇口喉舌干焦，亦或唾血，渐觉瘦悴，小便赤，颜色青白，毛耸者。
加减：None
功效：None
用法用量：每取胡桃大，于临卧时含之，细细咽津，稍加至鸡子大。昼间服之亦得。
制备方法：上取生地黄汁以下，生葛根汁以上，于微火上和煎减半，内桑白皮等二物汁和煎之，3分减1，内酥蜜枣膏，搅之勿停手，得如稠饴状，置别器中收贮。
临床应用：None
各家论述：《医林纂要》：桑白皮甘酸微辛，补肺泻火，肃敛清气；麦冬、地骨皮补肺清金，下滋肾水；生地大滋肾水，以靖君相之火；葛根清提胃气，以解膻中之热；竹沥升散阴中之火，祛除经络之痰；白蜜润肺止咳，枣肉补土生金，牛酥滋阴润肺，养血止咳。方皆寒凉之味，非辛莫化，故以生姜行痰，并以调剂，亦反佐也。
用药禁忌：忌芜荑。
附注：本方方名，《医林纂要》引作“桑皮十味煎”。
