序号：83205
名称：乌龙德生膏
组成：黄耆1两，青木香1两，连翘1两，玄参1两，木鳖子（去油壳）1两，生地黄1两，桃仁（去皮尖）1两，防风1两，川芎1两，白芷1两，羌活1两，白及1两，白蔹1两，金银花1两，蓖麻子300枚（去壳），乱发1两（烧灰），桂花头5钱，五香连翘汤5钱，人参拔毒散5钱，复元通气散5钱，十奇内补排脓散5钱（1贴）。
出处：《普济方》卷三一三。
主治：一切恶疮肿毒，及小儿肿毒、脾癖坚硬。
加减：None
功效：None
用法用量：如小儿脾疳诸癖等证，量病坚硬大小，用纸或绯帛摊药贴之，候药力尽，自脱下再换；小儿疳泻痢证，贴肚皮上；咳嗽，贴脊梁中心，其病即愈。
制备方法：上将黄耆等17味、五香连翘汤4药(口父)咀，用小油3斤半，入铁锅内先浸5日，用慢火煎至药味黄黑为度，以槐1握，柳条1握，不住手搅，再用重绢滤去滓，秤净油3斤，先将黄丹1斤半炒黑色，下小油一处，于慢火同熬得所，滴入水中不散成珠；后下雄黄5钱，血竭5钱，乳香(另研)5钱，没药5钱，陀僧5钱，轻粉5钱，龙骨5钱，枫香5钱，麝香1钱(加苏合油半两火炒)，研末，下前膏内化开，搅千余遍和匀，又试水中得所，成膏药可摊为度。如坚，少加小油；如软，加些黄丹，须要搅匀成膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
