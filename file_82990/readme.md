序号：83006
名称：乌云煎
组成：香油2斤，柏子油2两（另贮），诃子半两，没食子6钱，百药煎3两，倍子（炒黑）1两，酸石榴皮半两，旱莲草半两，猪胆2枚（另贮），胆矾1钱。
出处：《医统》卷九十八。
主治：女人发黄不润。
加减：None
功效：None
用法用量：上为粗末，先将香油锅内熬数沸，然后将药末下入油内熬，少时倾出油入罐内，微冷、入柏子油搅匀渐冷，下后项细料药：甘松3钱，三奈3钱，白芷3钱，藁本3钱，零陵香3钱，藿香3钱为粗末，绢袋盛之，入油浸，每日搅3次。如此10日后，先1日以灰汤洗发净，次日搽油，不数日，其发黑绀润泽香美，永不惹尘，更不须再洗。

临床应用：None

用药禁忌：None
