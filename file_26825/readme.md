序号：26833
名称：延龄广嗣丸
组成：鹿角胶3两，巴戟肉2两，大熟地6两，海马1两，淡苁蓉2两，杜仲2两，潞党参4两，五味子1两，怀山药3两，白茯苓3两，大茴香1两，金樱子1两，胡芦巴4两，淫羊藿2两，贡沉香1两，枸杞子2两，蛇床子2两，白檀香1两，肉桂1两，菟丝子4两，川楝子2两，山萸肉3两，制附子1两，制乳香1两，怀牛膝（盐水炒）2两，补骨脂2两，制没药1两。
出处：《全国中药成药处方集》（杭州方）。
主治：男子下元虚损，阳痿精冷，久无子嗣，腰背酸痛，一切先天禀受不足，少年斫伤过度之症。
加减：None
功效：培元固本，补肾生精，健阳种子，延龄广嗣。
用法用量：每服3钱，淡盐汤送下。
制备方法：上为细末，将胶烊化，酌加炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
