序号：2477
名称：升麻膏
组成：川升麻1两，犀角屑1两半，玄参1两，杏仁1两（汤浸，去皮尖双仁），赤芍药1两，麻黄1两（去根节），栀子仁1两，甘草1两，川芒消1两，芎1两，蛇衔草1两，白蔹1两，黄芩1两，莽草1两，桑寄生1两，白芷1两，射干1两，蓝叶1两，地黄汁5合，猪脂4斤，醋1升。
出处：《圣惠》卷六十三。
主治：一切毒肿热疼。
加减：None
功效：None
用法用量：每日4-5次，涂摩肿处。
制备方法：上锉，以醋、地黄汁，渍药1宿，于铛内先消猪脂，入药，以慢火煎，候醋气歇，白芷黄赤色，膏成，绵滤去滓，盛瓷器中。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
