序号：47807
名称：神仙透空丸
组成：天麻1两（洗），香白芷1两，半夏1两（生用），天南星1两，干姜4钱（生用），川芎2两（洗），地龙半两（捶，洗，去土净），川乌1两（生用，去皮尖），草乌1两（生用，去皮尖），细辛半两，甘草2两（生用），白附子半两。
出处：《洪氏集验方》卷三。
主治：偏正头疼，恶心；产后体虚伤风，憎寒头痛，洗头沐浴伤风，壮热头痛，痰厥头痛，肾厥头痛，虚眩，项筋紧急等一切头痛。
加减：None
功效：None
用法用量：None
制备方法：上为细末，以药末1两入白面1两，用新汲水为丸，每1两分作10丸，于新板上排定，日内晒干，如合，须是伏中合。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
