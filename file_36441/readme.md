序号：36451
名称：和疟清痢饮
组成：紫苏1钱5分，当归8钱，藿香3钱，枳壳2钱（炒），槟榔1钱5分，青皮1钱5分，车前子3钱，炙甘草3分，广木香1钱5分，煨姜2钱。
出处：《慈航集》卷下。
主治：痢初起恶寒发烧，恶心呕吐，下痢腹痛，颇似疟疾而并非疟者。
加减：如恐寒热末全清，加炒柴胡5分，再服1剂，无有不清者；寒热俱清，去紫苏、藿香，加炒白芍8钱、炒菜菔子3钱，3-4服痢全止；如遍数多，里急后重，再加酒制大黄，虚者3钱，实者5钱；如红多，加酒炒川连7-8分。
功效：None
用法用量：此方1服。盖暖得汗，寒热恶心解，痢亦轻。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
