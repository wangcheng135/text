序号：38172
名称：松脂丸
组成：松脂3两（炼成者），松花3两，白茯苓1两，菖蒲1两，桂心1两，生干地黄2两，薯蓣1两，远志1两（去心），鹿角胶1两（捣碎，炒令黄燥），牛膝1两（去苗），甘草1两（炙微赤，锉），槟榔1两，肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），菟丝子1两（酒浸3日，晒干，别捣为末），鹿茸1两（去毛，涂酥，炙微黄）。
出处：《圣惠》卷九十八。
主治：风冷。
加减：None
功效：强筋骨，补五脏，除风湿，久服轻身耐老延年，益气，补诸不足。
用法用量：每日服30丸，空心以温酒送下。渐加至40丸。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
