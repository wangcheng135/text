序号：65951
名称：虚风丸
组成：天蓼木半两，吴白芷半两，白鲜皮半两，白茯苓（去黑皮）半两，川芎半两，独活（去芦头）半两，防风（去芦头）半两，天南星（酒浸，切作片子，酒煮）半两，天麻（酒煮）半两，乌蛇（酒浸，去皮骨）半两，全蝎（微炒）半两，人参（去芦头）半两，麻黄（去根节，炒）半两，甘草（锉炒）半两，白术半两，细辛（去苗叶土）半两，川乌头（炮裂，去皮脐）半两，白僵蚕（去丝微炒）半两，天雄（炮裂，去皮脐）3钱7分半，黑附子（炮裂，去皮脐）3钱7分半，马牙消（别研）钱半，雄黄（飞研）钱半，朱砂（飞研）钱半，龙脑半钱，麝香半钱。
出处：《御药院方》卷一。
主治：一切虚风，头痛眩运欲倒，呕吐痰涎，牙关紧急，手足无力，麻木不仁，不省人事。
加减：None
功效：None
用法用量：每服1丸，温酒化下，或荆芥汤亦得，食后、临卧服。
制备方法：上为细末，炼蜜为丸。每两作10丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
