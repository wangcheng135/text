序号：48534
名称：神效暧脐膏
组成：肉桂1两5钱（去皮），丹皮8钱，黄耆2两，党参2两，归身2两，生地2两，白芍1两，苁蓉1两，附子（炙）1两，木鳖子1两（去壳），荆芥5钱，防风5钱，麻黄5钱，桂枝5钱，柴胡5钱，前胡5钱，升麻5钱，葛根5钱，苏叶5钱，薄荷5钱，羌活5钱，独活5钱，白芷5钱，藁本5钱，川芎5钱，细辛5钱（一方有麝香5钱）。
出处：《慈禧光绪医方选议》。
主治：受寒受冷，腹痛腹胀，呕吐酸水；及久不孕育，腰骶疼痛。
加减：None
功效：镇疼止泻，祛风散寒，健肠胃，暖肚。
用法用量：None
制备方法：上以真麻油2斤，生姜4两、葱头4两（切碎），入油内慢火熬焦，去滓滤净汁，将油秤准，每油1斤，入飞净黄丹半斤，慢火熬至老嫩得所，以瓷器收盛，7日后方可用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
