序号：5546
名称：宁志膏
组成：琥珀1两（炼成收用），茯神3两，枣仁（炒）3两，丹皮3两，熟地5两，归身3两，川芎1两，白芍2两，半夏1两，麦冬1两，竹叶100片，丹参6两，郁金7钱。
出处：《陈素庵妇科补解》卷五。
主治：产后心血虚，败血、痰火、瘀血冲心，心神恍惚怖畏，乍见鬼神。
加减：None
功效：补心血，安心神，定心气，兼消瘀祛痰清火。
用法用量：加生姜3片，辰砂1钱、金饰2钱，煎汤化1盏服。
制备方法：None
临床应用：None
各家论述：四物加茯神、枣仁养血安神，佐以丹参生新去旧，丹皮泻火通经，半夏行痰，麦冬清心，竹叶降火，琥珀消瘀破结，郁金入心，专治败血攻心，癫狂错乱，而行以辛温之姜，引以重镇之辰砂、金饰。心神安，心血充，心气定，痰火。败血不攻而自退矣。
用药禁忌：None
附注：None
