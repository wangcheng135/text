序号：9646
名称：羚羊角丸
组成：羚羊角2两（镑），茯神2两（去木），木香2两，防风2两（去叉），升麻2两，赤茯苓3两（去黑皮），大麻仁（别捣如膏）3两，独活（去芦头）2两半，大腹7枚（连皮子锉），旋复花1两1分。
出处：《圣济总录》卷八十二。
主治：脚气。心腹妨闷，两肋虚胀，不思食，渐觉心满，气膈不通。
加减：腹内先冷，加附子（炮裂，去皮脐）2两；大肠秘涩，加大黄（锉，炒）2两半；兼气，加槟榔仁（锉）2两半；心胸间热，加麦门冬（去心）3两；筋急掣痛，加牛膝（切，焙）2两半；皮肤风痒，心下妨闷，加枳壳（去瓤，炒黄）1两半；脏气羸弱，加黄耆（锉，炒）3两。
功效：None
用法用量：每日空腹酒送下40丸，每日1服。
制备方法：上除大麻仁外，为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
