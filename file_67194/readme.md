序号：67210
名称：麻黄复煎散
组成：白术5分，人参5分，生地黄5分，柴胡5分，防风5分，羌活1钱，黄柏1钱，麻黄（去节，微捣，不令作末，水5大盏煎令沸，去沫，煎至2盏，入下项药再煎）2钱，黄耆2钱，甘草3钱，杏仁3个（去皮）。
出处：《兰室秘藏》卷中。
主治：风湿相博，下焦伏火而不得伸，阴室中汗出，懒语，四肢困倦无力，走注疼痛，躁热，一身尽痛。
加减：None
功效：发汗升阳。
用法用量：上（口父）咀，都作1服。入麻黄汤煎至1盏，临卧服之。勿令食饱，取渐次有汗则效。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：麻黄复煎汤（《准绳·伤寒》卷二）。
