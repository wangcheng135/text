序号：46587
名称：千金封脐膏
组成：天门冬2钱，生地黄2钱，熟地黄2钱，木鳖子2钱，大附子2钱，蛇床子2钱，麦门冬2钱，紫梢花2钱，杏仁2钱，远志2钱，牛膝2钱，肉苁蓉2钱，官桂2钱，肉豆蔻2钱，菟丝子2钱，虎骨2钱，鹿茸2钱。
出处：《寿世保元》卷四。
主治：男子下元虚冷，小肠疝气，痞疾，单腹胀满，并一切腰腿骨节疼痛，半身不遂。妇人子宫久冷，赤白带下，久不坐胎。
加减：None
功效：存精固漏，活血通脉，壮阳助气，返老还童。
用法用量：外贴，贴之60日，方无力，再换。
制备方法：上为末，入油1斤4两，文武火熬黑色，去滓，澄清，入黄丹半斤，水飞过松香4两熬，用槐柳条搅，滴水不散为度。再下硫黄3钱、雄黄3钱、朱砂3钱、赤石脂3钱、龙骨3钱，为末入内。除此不用见火，将药微冷定，再下腽肭脐1副、阿芙蓉3钱、蟾酥3钱，麝香1钱，不见火，阳起石3钱，沉木香3钱，俱不见火。上为细末，入内，待药冷，下黄蜡6钱，贮瓷器盛之，封口，放水中，浸3日，去火毒，取出摊煅子上，或红绢上亦可。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
