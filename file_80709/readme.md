序号：80725
名称：麋角鹿茸丸
组成：麋角饼子半斤，鹿角霜半斤，鹿茸2两，九节菖蒲2两，钟乳2两，覆盆子2两，石斛2两，蛇床子（酒煮，炒香）2两，当归（洗）2两，肉桂（去皮）2两，金铃子（去核，酒浸）2两，山药2两，泽泻2两，柏子仁（研细，另入）2两，续断2两，附子（炮，去皮，以地黄汁煮，焙干）2两，山茱萸（取皮）2两，萆薢（去须，蜜水涂，炙）2两，杜仲3两（麸炒丝断），天雄（去皮）1两，白茯苓1两，五味子（净洗）1两，人参（去芦）1两，槟榔1两，胡芦巴（酒浸，焙）1两，麝香（别研）1两，细辛1两，破故纸（酒浸，炒香）4两，远志（去心）4两，天门冬（去心）4两，牛膝（酒浸）4两，胡桃（去皮，研，另入）4两，巴戟（酒浸，炒）4两，苁蓉（洗，酒浸，焙干）4两，熟干地黄（净洗，焙）4两，茴香（炒）4两，菟丝子（酒浸，蒸3次，研）4两，防风1两。
出处：《百一》卷四。
主治：真阳不足，脾肾虚寒，下焦伤惫，脐腹疼痛，两胁胀满，手足麻痹，目视茫茫，遗泄失精，精神不爽，阳事虚弱，小便滑数，气虚肠鸣，大便自利，耳内常聋。
加减：None
功效：久服益脾元，壮肾气，助真阳，补虚损，散寒湿，养气滋血。
用法用量：每服30-40丸，渐加至50-60丸，空心温酒或盐汤吞下。
制备方法：上为末，用酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中麋角饼子，原作“鹿角饼子”，据《奇效良方》改。鹿茸原脱，据《奇效良方》补。
