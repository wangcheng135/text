序号：47582
名称：神仙化痞膏
组成：刘寄奴草4两，当归2两，川芎2两，白芷2两，黄柏2两，胡黄连2两，苏木2两，川乌2两，肉桂1两，丁香1两，巴豆肉1两，草乌1两，大黄3两，蜈蚣3两，穿山甲3两，白花蛇1条，桃枝30寸，柳枝30寸。
出处：年氏《集验良方》卷三。
主治：痞疾、积块。
加减：None
功效：None
用法用量：用桑皮油纸摊贴，以热手摩之。
制备方法：上用香油2斤浸5日，桑柴慢火熬黑色，去滓，放冷，滤清净，取1斤半，再入锅内熬至滴水成珠，下飞过黄丹3两，陀僧2两，仍慢火熬至沸止；再下黄蜡8两，熬至滴水成珠，方离火候微冷；再下去油乳香1两，去油没药1两，番硇砂1钱5分，麝香2钱，轻粉2钱，血竭5钱，阿魏5钱，陆续搅入膏内，以冷为度。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
