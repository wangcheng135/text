序号：59779
名称：调经补真汤
组成：独活2分，干姜（炮）2分，藁本2分，防风2分，苍术2分，麻黄（不去节）5分，炙甘草5分，人参（去芦）5分，当归身5分，白术5分，生黄芩5分，升麻5分，黄耆7分，良姜1钱，泽泻1钱，羌活1钱，柴胡4钱，杏仁2个，桂枝少许，白葵花7朵（去萼）。
出处：《兰室秘藏》卷中。
主治：冬后一月，白带再来，阴户中寒。
加减：None
功效：None
用法用量：先以水3大盏半，煎麻黄1味令沸，掠去沫，入余药同前至1盏零7分，再入生黄芩煎至1盏，空心服之。候1时许，可食早饭。
制备方法：上（口父）咀，除麻黄、黄芩各另外，都作1服。
临床应用：None
各家论述：None
用药禁忌：None
附注：调经固真汤（《东垣试效方》卷四）。
