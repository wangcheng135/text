序号：34313
名称：万灵膏
组成：鹳筋草1两，透骨草1两，紫丁香根1两，当归1两（酒洗），自然铜1两（醋淬7次），瓜儿1两，血竭1两，没药1两，川芎8钱，赤芍2两，半两钱（醋淬）1枚，红花1两，川牛膝5钱，五加皮5钱，石菖蒲5钱，茅山5钱，苍术5钱，木香3钱，秦艽3钱，蛇床子3钱，肉桂3钱，川附子3钱（制），半夏3钱（制），石斛3钱，萆薢3钱，鹿茸3钱，虎胫骨1对，麝香2钱。
出处：《金鉴》卷八十八。
主治：跌打损伤，兼去麻木风痰、寒湿疼痛。
加减：None
功效：消瘀散毒，舒筋活血，止痛接骨。
用法用量：None
制备方法：上除血竭、没药、麝香3味各研细末另包外，共23味，先将香油10斤微火煨浸3日，然后将群药入油内，熬黑为度，去滓，加黄丹5斤再熬，将至滴水成珠，离火；俟少时药温，将血竭、没药、麝香下入搅匀，取起出火气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
