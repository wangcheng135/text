序号：47234
名称：疯气膏药
组成：乱头发（洗净，晒干）半斤，地骨皮1两，寻风藤1两，当归1两，黄蜡2两，五加皮4两，防风1两，黄柏（去皮）4两，独活1两，木鳖子（去壳）1两，姜汁半斤，葱汁半斤。_x000D_
先将松香2斤下锅熬化，将棕2斤滤去滓，将清水浸之；然后将麻油2斤下锅熬，就下前粗药、葱姜汁入内，武火熬，各药将枯，再加端午日采百草入内共熬枯，将棕滤去药滓；再将锅抹净，复下药油入锅内，慢火熬，再下松香、黄蜡，滴水成珠，然后下百草霜半斤，细下入锅，收水气（将百草霜预先研极细，筛去粗者，然后下入药油内），候膏成住火，慢下细药。_x000D_
孩儿茶5钱，樟脑1钱，乳香（箬去油，碾）1两（研），没药（箬去油，研）1两，龙骨3钱，轻粉3钱，血竭1两，麝香5分。
出处：《墨宝斋集验方》卷上。
主治：疯气。
加减：None
功效：None
用法用量：听用。
制备方法：上为极细末，俟住火，将冷方渐下，不住手搅匀，将冷水浸之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
