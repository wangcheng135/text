序号：7213
名称：发背膏药
组成：滴乳香4两（箬包烧红，用砖压出油），净没药4两（照前式去油），鲜油血竭4两，白色儿茶4两，上好银朱4两，杭州淀粉4两，上好黄丹4两，上铜绿3钱。
出处：《串雅内编》卷二。
主治：发背。
加减：None
功效：止痛，化腐生新。
用法用量：临时照所患大小，用夹连泗油纸1块，以针多刺小孔，每张用药末5钱，用真麻油调摊纸上，再用油纸盖上，周围用线缝好，贴患处，用软绢扎紧。过3日将膏揭开，浓煎葱汤将患处洗净，软绢拭干，再将膏药翻过，照前法贴之。
制备方法：上药各另碾至无声为度，筛极细末，拌匀。
临床应用：None
各家论述：None
用药禁忌：此方破溃后用之最效，未溃未出大脓非所宜也。
附注：本方方名，《青囊秘传》引作“发背膏”。
