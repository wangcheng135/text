序号：59054
名称：秦桂丸
组成：肉桂1两（去皮），秦艽1两，附子1两（炮），当归2两，厚朴1两（制），人参1两，干姜1两（炒），白薇1两，半夏1两（制）。
出处：《医略六书》卷二十七。
主治：血海久冷不孕，脉细涩者。
加减：None
功效：None
用法用量：每服2-3钱，温酒送下。
制备方法：上为末，炼蜜为丸。
临床应用：None
各家论述：血海久冷，冲任少熏育之权，故腹痛经迟，时发寒热，而不能孕子焉。桂、附暖血海以逐冷，姜、朴温中气以散寒；当归养血脉，人参扶元气，秦艽活血通经，半夏化痰燥湿，白薇降泄以除寒湿热也。炼蜜以丸之，温酒以下之，俾血海温暖，则久冷顿消，而腹痛无不退，寒热无不除，何患天癸不调，不能孕子乎。
用药禁忌：None
附注：None
