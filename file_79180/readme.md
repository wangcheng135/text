序号：79196
名称：镇惊丸
组成：人参（去芦）3钱，粉草（半生半炙）5钱，茯神（去皮木根）5钱，僵蚕（去丝）5钱，枳壳（去瓤，麸炒）5钱，白附子2钱半，南星（锉碎，腊月黄牛胆汁酿经一夏）2钱半，白茯苓（去皮）2钱半，硼砂2钱半，牙消2钱半，朱砂（水飞）2钱半，全蝎（去毒）10尾，麝香1字。
出处：《活幼心书》卷下。
主治：急慢惊风，风痰上壅，手足抽掣，口眼斜，烦躁生嗔，精神昏闷。
加减：None
功效：常服宁心镇惊，疏风顺气。
用法用量：每服3-5丸或7丸，急惊用茶清磨下；慢惊用生姜、附子煎汤温服。
制备方法：上将牙消、硼砂、麝香、朱砂用乳钵乳细，余药为末，入乳钵内与前四味和匀，用糯米粉水煮糊为丸，如梧桐子大，以银朱为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
