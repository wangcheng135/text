序号：53322
名称：健脾导功丸
组成：人参（去芦）1两，于白术（东壁土炒）3两，白茯苓（饭上蒸）2两，粉甘草（蜜炙）5钱，广陈皮（饭上蒸）2两，制半夏（姜汁炒）2两，六神曲（炒）1两5钱，薏苡仁（炒）2两，陈枳壳（麸炒）1两5钱，泽泻（盐水炒）1两，五谷虫（新瓦培）2两，怀山药（炒黄）2两，谷芽（炒香）1两，菟丝饼（命火衰微始用），鸡肫皮（新瓦焙）2两。
出处：《疡医大全》卷二十六。
主治：妇女脾胃失调，饮食不进，面目痿黄，肌肤消瘦。
加减：None
功效：None
用法用量：每服2钱，每晚白汤送下。
制备方法：上为细末，水为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
