序号：51473
名称：轻脚丸
组成：宣州木瓜2枚（用竹刀切下盖子，剜去子，却填熟艾令满，以盖子盖好，用竹钉签定，入饭甑内蒸烂，然后以文武火焙干，入后药），肉苁蓉（酒浸1宿）2两，牛膝（酒浸）2两，萆薢2两，金毛狗脊（去毛净）1两，防风（去芦，去叉股者）1两，海桐皮（去粗皮）1两，天麻1两，全蝎（去毒）1两，青盐1两，川续断1两，甘草（炙）半两，乳香（别研）半两，没药（别研）半两。
出处：《魏氏家藏方》卷八。
主治：风寒湿留滞肝肾二经，下部虚大，发则攻筑，脐腹俱痛，自汗恶心，呕吐涎沫，肢体酸削，不进饮食。
加减：None
功效：常服壮脚膝，补虚损，下部之疾更不再发。
用法用量：每服30丸，渐加至40-50丸，早、晚食前温酒送下；盐汤亦得。
制备方法：上为细末，用川乌头5两，炮，去皮脐，为末，酒煮为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
