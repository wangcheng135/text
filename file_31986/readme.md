序号：31995
名称：坎离丸
组成：苍术8两（锉如豆大，泔浸3日，或焙或晒干，分作四处。一份用真乌豆1两，去皮脐，切作片子；又用川楝子净肉1两，同苍术炒焦黄色为度。一份用川椒去目1两，又用陈皮1两，破故纸1两，酒浸1宿，炒令干，次下苍术，川椒同炒黄。一份用茴香净1两，青盐1两半，食盐炒半两，先下苍术炒熟，次下茴香等同炒黄色。一份用醇酿酒、醋各1碗，浸苍术，令自干，炒燥，入后药），麦门冬（去心，焙干）3两，天门冬（去心，焙）3钱，茯神（去皮木，炒）3分，远志（去心，焙）2钱，沉香1两，鹿茸（燎去毛，酥炙）5钱，胡芦巴（酒浸，炒）5钱，川巴戟5钱（去心，酒浸，炒），当归半两（酒浸，焙），人参（去芦）半两，枸杞子半两，雀脑半两，川芎半两，陈皮（去瓤）半两。
出处：《瑞竹堂方》卷一。
主治：心、脾、肾三经不足。
加减：None
功效：None
用法用量：每服40-50丸，空心服。如补心，枣汤送下；补肾，温酒、盐汤送下。
制备方法：上为细末，好酒煮神曲2两，打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
