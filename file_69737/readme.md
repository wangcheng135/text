序号：69753
名称：琥珀牛黄丸
组成：琥珀1钱，猪牙皂1钱，木香1钱，人中白（煅）3钱，轻粉3钱，雄黄3钱，朱砂3钱，乳香（去油）3钱，没药（去油）3钱，白芷3钱，当归2钱，西牛黄3分，槐花（炒）1两，丁香（春、夏）1钱5分（秋、冬）3钱。
出处：《疡医大全》卷三十四。
主治：杨梅破烂，并一切疽痈久溃，脓水不干者。
加减：None
功效：None
用法用量：初服5丸，5日后服7丸，又5日后服9丸，又5日后仍服7丸，又5日后只服5丸，周而复始，俱用土茯苓、甘草煎汤送下。
制备方法：上为极细末，酒糊为丸，如萝卜子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
