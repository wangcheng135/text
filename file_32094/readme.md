序号：32103
名称：坎离丸
组成：酸枣仁（炒）1两，菟丝子（淘净，酒浸，研成饼）1两，柏子仁（炒，别研）1两，五味子（去枝）1两，薏苡仁（炒）1两，覆盆子1两，人参（去芦）1两，枸杞子1两，鹿茸（燎去毛，锉成片，酒浸，炙）1两，牛膝（去芦，酒浸）1两，肉苁蓉（酒浸）1两，当归（去芦，酒浸）1两，杜仲（姜制，炒去丝）1两，远志（去心）1两，地黄（洗）1两，茯神（去木）1两，沉香（不见火）半两，附子（炮，去皮脐）半两，龙骨（煅）半两，朱砂3钱（别研），麝香1钱（别研）。
出处：《魏氏家藏方》卷六。
主治：小便白浊，腰腿无力，心神不宁，下焦虚寒，阴冷遗沥。
加减：None
功效：平补五脏，升降心肾。
用法用量：每服50丸，空心温酒或人参汤送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
