序号：35528
名称：使君子丸
组成：使君子1分，丁香1分，没石子1分，熊胆（细研）1分，胡黄连1分，夜明沙（微炒）1分，青黛（细研）1分，黄连（微炒，去须）1分，肉豆蔻（去壳）1分，芦荟（细研）1分，龙脑1钱（细研），蟾头1枚（炙令焦黄），麝香1钱（细研）。
出处：《圣惠》卷八十六。
主治：小儿五疳，面色萎瘁，头热发干，胃气不和，心腹满闷，宿食不消，或时下痢，瘦弱无力。
加减：None
功效：None
用法用量：每服5丸，以粥饮或新汲水送下，每日服3次。3岁以上加丸服之。
制备方法：上为末，烧粟米饭为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
