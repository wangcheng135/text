序号：81401
名称：麒麟竭膏
组成：麒麟竭1两，桂心3分，木香半两，附子3分（生，去皮脐），槟榔半两，当归半两，白芷半两，川芎3分，诃梨勒皮半两，沉香半两，没药半两，白及半两，朱砂3分（细研），丁香半两，乳香半两，甘草半两（锉），麝香半两（细研），白檀香3分，甘松香1两，零陵香半两，槐枝1两，柏枝2两，垂柳枝2两，松脂3分，白蜡3分，黄丹15两，油2斤半。
出处：《圣惠》卷六十三。
主治：一切痈疽发背，日夜疼痛。恶疮毒肿，溃脓不止，肌肉不生。
加减：None
功效：解毒生肌。
用法用量：于蜡纸上摊贴，甚者每日早晚换之。
制备方法：先将油于铛中以炭火炼令香，细锉甘松香、零陵香、檀香、槐、柳枝等，入油内浸1宿，以文火煎，候三般枝黄黑色即去；却下松脂并蜡化了，以绵滤过；拭铛令净，却倾油入挡中，下黄丹，于火上煎，以柳木篦不住手搅，令沸转黑色；后将前十六味捣罗为末，微火上调入膏内，搅二三千遍，令匀，滴入水中作珠子即膏成，用瓷盒收。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
