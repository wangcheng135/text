序号：40332
名称：茄子根浸酒
组成：干茄子根2斤（未着霜者，细锉，饭上蒸一炊时），苍耳子1升（微炒，捣碎），牛膝1斤（去苗），鼠粘子1升（微炒，捣碎），大麻子1升（微炒，捣碎），牛蒡根1斤（切，酥炒黄），防风2两，萆薢2两，晚蚕砂半斤（微炒），枸杞子，败龟2两（涂酥，炙微黄，捣为末），虎胫骨2两（涂酥，炙微黄，捣为末），桔梗1两（去芦头），羌活2两，秦艽2两（去苗），附子2两（炮裂，去皮脐）。
出处：《圣惠》卷二十五。
主治：风证顽麻。
加减：None
功效：利关节，除疼痛，去挛缩，强腰膝。
用法用量：每日空心、日午、初夜量力温服1盏。候重者不过1剂愈。
制备方法：上锉细，以生绢袋盛，用好酒3斗浸，以瓷瓶盛，密封，经14日开之，开时不得面对瓶口。
临床应用：None
各家论述：None
用药禁忌：忌毒滑、鱼肉等。
附注：None
