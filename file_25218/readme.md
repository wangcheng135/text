序号：25226
名称：妇宝胶归丸
组成：生地8两，香附8两，芍药6两，山萸肉6两，丹皮4两，杜仲4两，续断4两，茯苓4两，白术4两，黄芩3两，椿皮3两，黑荆芥3两。
出处：《活人方》卷七。
主治：月事先期而至，红紫不一，甚则或崩或漏，淋漓不净，日久去血过多，气亦虚陷，非淋即带，腥秽绵绵，块结脐腹，痛连腰脊，胸膈痞闷，饮食日减，头目眩晕，肢体疲倦；多产成痨，或气虚半产，营卫虚极，形神羸弱，骨蒸烦热，四肢浮肿，昼则嗜卧，夜反无寐；先天不足，久不怀孕。
加减：None
功效：None
用法用量：早空心白滚汤吞服4-5钱，临睡服2-3钱。
制备方法：上药炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
