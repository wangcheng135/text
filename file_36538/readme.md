序号：36548
名称：三圣丸
组成：舶上硫黄1两，水银1两，硇砂（去砂石）1分。
出处：《博济》卷三。
主治：积年血气癥癖瘕聚，诸药疗理不瘥者。
加减：None
功效：None
用法用量：若因产后伤于饮食，结伏在腹胁，时发疼痛，不可忍者，当归浸酒1升，逐渐取酒少许，送下7丸至10丸；若取磨癖块，空心温酒送下3丸至5丸。所有药渣捩了，焙干为末，别入干地黄半两，真蒲黄1分，芫花（醋炒焦黄色）1分，同研为末，以前三圣丸所剩面糊为丸，如绿豆大。治妇人血脏冷气攻冲，心胸疼闷，及一切血块，温酒送下10丸。
制备方法：上3味，乳盆内滚研如粉，放生铁铫内，用文武火熬熔成汁，以铁火箸搅令匀，放冷，刀铲下，以纸裹，入地坑内埋1宿，取出再研令细，次以赤芍药1分，当归1分，荆三棱1分，蓬莪术1分，红花1分，生用并细锉，以法酒1升，煎及1半，漉出，于砂盆内研，生布捩汁，再熬放冷，入飞罗面为糊，丸如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
