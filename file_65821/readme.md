序号：65837
名称：菴（艹闾）子丸
组成：菴（艹闾）子1两，川乌头3分（炮裂，去皮脐），桂心3分，防葵半两，桃仁1两（汤浸，去皮尖双仁，麸炒微黄），吴茱萸半两（汤浸7遍，焙干，微炒），牛膝1两（去苗），当归1两（锉，微炒），生干地黄1两，芎1两，干姜半两（炮裂，锉），鳖甲1两（涂醋炙微黄，去裙襴），赤芍药半两，芫花3分（醋拌，炒令干），川大黄1两（锉碎，微炒）。
出处：《圣惠》卷七十九。
主治：产后余血不尽，腹内结成血瘕，月水不利，四肢羸瘦，不欲饮食。
加减：None
功效：None
用法用量：每服20丸。食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
