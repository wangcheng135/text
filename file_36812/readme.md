序号：36822
名称：固荣丹
组成：代赭石（火煅通红，醋淬21次，焙净，捣，罗过，再就乳钵内研极细）2两，五倍子（麸炒）1两，诃子（去核，锉，炒燥）1两，鹿角胶（锉，麸炒脆为度）1两，木贼（去节，锉，炒）1两，当归（锉，洗净，去芦）半两，川芎（用刷牙子洗，锉，炒）半两。
出处：《魏氏家藏方》卷七。
主治：痔大作，下血不止，或吐血，鼻衄，尿血，横流不摄。
加减：None
功效：None
用法用量：每服100丸，米汤送下，以止为度，未效再进。
制备方法：上为细末，入代赭石末和水煮，糯米粉厚糊为丸，如梧桐子大，阴干。
临床应用：None
各家论述：代赭石大治肠风下血痔疾，建脾缩小便，亦治鼻衄、吐血、尿血；诃子治肠风下血；五倍子治五痔下血不止；当归、川芎治肠风下血，血既多，以此滋血不致枯竭也；鹿角胶固血道、补血虚。或下血未已，急进此药，不辍则自然收敛，不渗失也。
用药禁忌：None
附注：None
