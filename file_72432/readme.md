序号：72448
名称：彭祖秘服接命丹
组成：何首乌10两，白茯神10两，赤茯苓10两，菟丝子10两，牛膝10两，破故纸10两，覆盆子10两，当归10两。
出处：年氏《集验良方》卷二。
主治：下元虚冷，五劳七伤，半身不遂，或下部虚冷，膀胱病症，脚膝酸麻，阳事不举，妇人赤白带下，血崩，沙淋，下生疮疖。
加减：None
功效：添精补髓保真，固精不泄，善助元阳，滋润皮肤，壮筋骨，理腰膝，令人行走康健，气力倍添，通二十四道血脉，固身体，返老还童。
用法用量：每服2钱，空心黄酒送下，早、午、晚进3服，服7日后每服3钱。
制备方法：上药不犯铁器，为细末，炼蜜调黄酒为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌食血；忌服芸苔、菜子油、萝卜。
附注：彭祖接命丹（《验方新编》卷十一）、接命丹（《中国医学大辞典》）。
