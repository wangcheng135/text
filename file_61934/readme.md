序号：61950
名称：救生丹
组成：鸡内金3-7枚（旋取，去却谷食，净洗，阴干，每夜露7宿），甜葶苈半两（洗，焙），黑牵牛子半两（用瓦上煿令下焦），砒信1分（别研细，每夜露7宿，至晚收于床下），半夏1分（洗净，焙，浸1宿，换水7遍，生用），黄丹半两（亦如砒信制）。
出处：《博济》卷二。
主治：远年日近肺气喘息，坐卧不能。
加减：None
功效：None
用法用量：食后、临卧温葱茶下7丸，甚者10丸。
制备方法：上为细末，煮青州枣（大者）12枚，去皮核，捣和为丸，如干，即人淡醋少许，丸如绿豆大，以朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：忌大冷、大热、毒食等。
附注：鸡膍胵丸（《圣济总录》卷四十八）。
