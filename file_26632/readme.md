序号：26640
名称：庆云散
组成：覆盆子1升，五味子1升，天雄1两，石斛3两，白术3两，桑寄生4两，天门冬9两，菟丝子1升，紫石英2两。
出处：《千金》卷二。
主治：丈夫阳气不足，不能施化，施化无成。
加减：素不耐冷者，去寄生，加细辛4两；阳气不少而无子者，去石斛，加槟榔15枚。
功效：None
用法用量：每服方寸匕，先食，酒调下，日3次。
制备方法：上药治下筛。
临床应用：None
各家论述：《千金方衍义》：庆云者，庆云龙之征兆。紫石英专温荣血，天雄峻暖精气，佐以覆盆、五味、菟丝温补下元，寄生主治腰痛，天冬能强肾气，石斛强阴益精，白术固津气而利腰脐间血；恐英、雄二味之性过烈，乃以天冬、石斛、寄生濡之，覆盆、五味、菟丝辅之，白术培土以发育万物；扶阳施化之功尽矣。若素不耐寒，则去寄生而加细辛，以鼓生阳之气；阳本不衰，当退石斛而进槟榔，以祛浊湿之垢。其法之可重端在乎此。
用药禁忌：None
附注：方中天门冬，《妇人良方》作“麦门冬”。
