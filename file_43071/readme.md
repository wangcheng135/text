序号：43086
名称：养气丹
组成：禹余粮石（火炼7次，醉淬7次，为末）半斤，紫石英（火煅1次）半斤，赤石脂（火煅1次）半斤，代赭石（火煅7次，醉淬7次，为未）1斤，磁石（火煅10次，醋淬10次）半斤，上五石各贮之，各为细末，又以水研之。挹其清者，置之纸上，纸用筲箕盛，欲使细末在纸上，而水滴在下，挹尽而上。既干，各用藏瓶盛贮，以盐水纸筋和泥固济，阴干；以好硬炭50斤，分，为5处，每1处用炭10斤，烧红作1炉子，煅此五药，以纸灰盖之；两日后，火尽灰冷，则再煅，如此3次，埋地坑内两日，出火毒，再研，入后药：附子（炮，去皮脐）2两，肉苁蓉（净洗，酒浸1宿，焙干）1两半，当归（酒浸1宿，焙干）1两，茴香（炒）1两，破故纸（酒炒香熟）1两，木香（不见火）1两，肉桂（去粗皮）1两，巴戟（盐汤浸，打，去心）1两，肉豆蔻（面裹；煨）1两，丁香1两，山药1两，鹿茸（酥炙）1两，白茯苓（去皮）1两，沉香1两，远志（去心）1两，已上各如法修制，同研为末，却入：乳香（别研）1两，五灵脂（去砂，别研）1两，没药（去砂石，研）1两，已上3味入众药同研，却入：朱砂（或煅或蒸）1两，阳起石（略煅，或只用酒煮）1两，钟乳粉1两，已上3味别研，临时入。
出处：《局方》卷五（宝庆新增方）。
主治：诸虚百损，脾元耗惫，真阳不固，三焦不和，上实下虚，中脘痰饮上攻，头目昏眩，八风五痹；或卒暴中风，痰潮上膈，言语謇涩，神昏气乱，状若瘫痪；及奔豚肾气，上冲胸腹连两胁，膨胀刺痛不可忍者；阴阳上下，气不升降，饮食不进，面无精光，肢体浮肿，五种水气，脚气上冲，腰背倦痛，夜梦鬼交，觉来盗汗，胃冷心痛，小便滑数，牵引小腹，足膝缓弱，步履艰难；妇人血海久冷，赤白带下，岁久无子，及阴毒伤寒，面青舌卷，阴缩难言，四肢厥冷，不省人事者；或触冒寒邪，霍乱吐泻，手足逆冷，六脉沉伏，唇口青黑，腹胁攻制；及男子阳事痿怯，脚膝痠疼，腹济虚鸣，大便自滑；兼疗膈胃烦壅，痰饮虚鸣，百药不愈者。
加减：肾虚，加热地黄3钱。
功效：助养真气，生阳逐阴，温平不僭，消磨冷滞，克化饮食，使五脏安宁，六腑调畅，百病不侵。固滑脱，镇虚逆，复阳助阴。
用法用量：每服5-10丸，空心用温酒吞下，或姜盐汤、或枣汤送下亦可；妇人用艾醋汤吞下。
制备方法：上为细末，用糯米粉煮糊为丸，每两作50丸，阴干，入布袋内，擦令光莹。
临床应用：None
