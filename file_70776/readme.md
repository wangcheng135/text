序号：70792
名称：紫金锭子
组成：鲜玄参（去芦）40斤，鲜地榆（去芦）40斤，鲜天麻12斤，鲜草乌（去芦）40斤，全蝎（洗，去土盐足）3斤，白面10斤，麻黄（去根）40斤，麝香（真正净肉）3两。
出处：《鲁府禁方》卷一。
主治：诸般风症，左瘫右痪，口眼斜，半身不遂，及破伤风。
加减：None
功效：None
用法用量：随症研服，轻者半锭，重者1锭，无灰黄酒研下。汗出即愈。
制备方法：每年五月初一日，采取鲜药，至初五日，先将玄参等4味，用水洗净，切片碾压，取自然汁，入瓷缸内，每日搅晒，至九月甲子庚申日，入全蝎同碾为细末，入麝香拌匀；将麻黄40斤，碾取细末一半，其滓入锅内熬水二三柱香，滤去滓，加入麻末及诸药末，白面打糊，搓成锭，用竹叶包裹。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
