序号：4433
名称：木香丸
组成：木香3分，鳖甲1两（涂醋，炙令黄，去裙襴），琥珀3分，柴胡1两（去苗），白术1两，干姜半两（炮裂，锉），陈橘皮1两（汤浸，去白瓤，焙），人参半两（去芦头），桂心半两，吴茱萸3分（汤浸7遍，焙干，微炒），厚朴1两（去粗皮，涂生姜汁，炙令香熟），当归3分（锉碎，微炒），赤芍药3分，京三棱3分（微煨，锉），延胡索3分，附子3分（炮裂，去皮脐），芎3分，壮丹3分，熟干地黄1两。
出处：《圣惠》卷七十。
主治：妇人冷劳气，经脉不调，脏腑气滞，四肢疼痛，饮食无味，渐加羸瘦。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
