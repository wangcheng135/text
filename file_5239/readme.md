序号：5239
名称：平安丸
组成：公丁香10两，橘皮10两，神曲（炒）10两，枳实（炒）10两，草果仁10两，木香10两，茯苓10两，山楂10两，延胡（炙）10两，砂仁10两，沉香10两，麦芽（炒）10两，肉果（煨）10两，白术（炒）10两，槟榔10两，青皮（炒）10两，香附（炙）10两，豆蔻仁10两，母丁香10两。
出处：《北京市中药成方选集》。
主治：气逆不舒，胸满腹胀，胃寒疼痛，嘈杂倒饱；肝气犯胃，肝胃不和，胃脘疼痛，吞酸。
加减：None
功效：顺气宽胸，暖胃降逆；疏肝理气，和胃止痛。
用法用量：每服2丸，温开水送下，日2次。
制备方法：上为细末，过罗，炼蜜为丸，重2钱，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
