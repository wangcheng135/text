序号：3080
名称：天麻汤
组成：天麻（酒炙）1两半，附子（炮裂，去皮脐）1两半，干蝎（去土，炒）1两，羌活（去芦头）1两，芎1两，白附子（炮）1两，牛膝（去苗，酒浸，切，焙）1两，麻黄（去根节）1两，白花蛇（酒浸，去皮骨，炙焦）1两，枸杞1两，白芷1两，人参1两，萆薢1两，海桐皮1两，防风（去叉）1两，桂（去粗皮）1两，酸枣仁（炒）1两，白蒺黎（炒）1两，当归（切，焙）1两，甘草（炙）1两，乳香（研）1两半。
出处：《圣济总录》卷四十二。
主治：肝脏风毒气注手臂、头项、肩髆、腰足，筋脉拳急，攻刺疼痛，或四肢虚肿、头目旋运，黑花昏暗，呕逆食减。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，加生姜3片，煎取8分，去滓温服。其煎药水，每用桃枝嫩者1两，柳枝嫩者1两，桑枝嫩者1两(净洗细锉)，甘菊叶半两，如无叶以花代，用水2升，煎取1升，去滓，若冬月，10日为1料；夏月，逐日修事服之。
制备方法：上药除研者外，锉如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：天麻煮散（《普济方》卷十五）。
