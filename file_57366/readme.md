序号：57381
名称：消翳复明膏
组成：黄丹1两，青盐2钱半，海螵蛸7分半，真珠7分半，熊胆2分，麝香2分，片脑5分（临时加入），诃子2个（去核，研末），槐枝49条，柳枝49条，冬蜜（熬1沸，去白沫，滤净）4两（上将蜜和黄丹炼至紫色，旋下余药，熬至滴水沉下成珠为度，除脑、麝成膏，后入），黄连2两半，防风5钱，当归5钱，龙胆草5钱，生地黄5钱，木贼2钱半，白菊花2钱半，蕤仁1钱，杏仁5分。
出处：《准绳·类方》卷七。
主治：眼目昏花，翳膜遮睛，内外障眼，一切眼疾。
加减：None
功效：None
用法用量：点眼，又以热汤泡化洗眼。
制备方法：上如前煎熬成膏，入蜜和匀，瓷碗盛，放汤瓶口上蒸炖成膏，滤净，入脑、麝和匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
