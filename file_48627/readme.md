序号：48642
名称：神捷膏
组成：香油半斤（先煎），黄蜡1两，松香5钱，上慢火熬至滴水成珠不散为度，取出候冷，加后药：乳香3钱，没药3钱，轻粉3钱，血竭3钱，孩儿茶3钱，枯矾3钱，龙（火煅）3钱，川椒4钱。
出处：《古今医鉴》卷十五引郑中山方。
主治：诸般顽疮，及内外臁疮，久年不愈者。
加减：None
功效：None
用法用量：若遇顽疮，先用花椒、细茶、艾叶浓煎水，频频温洗令净，却用油纸以封刺孔，比如疮口大，俱刺遍伤，药将孔面贴疮上，1日换3次，2日后1日换1次。每换药必须洗净方贴。
制备方法：上为细末，搅入煎膏内，瓷器收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
