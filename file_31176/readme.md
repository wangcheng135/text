序号：31185
名称：杏仁汤
组成：杏仁3钱，黄芩1钱5分，连翘1钱5分，滑石3钱，桑叶1钱5分，茯苓3钱，白蔻皮8分，梨皮2钱。
出处：《温病条辨》卷一。
主治：肺疟，舌白渴饮，咳嗽频仍，寒从背起，伏暑所致。
加减：None
功效：None
用法用量：水3杯，煮取2杯，1日服2次。
制备方法：None
临床应用：None
各家论述：《成方便读》：此为伏暑留于肺络而发也，故以一派轻宣肺气，清肃上焦之品，治之自愈。白蔻宣肺滞；杏仁降肺气，使肺金复其清肃之令；桑叶轻扬入络，散之于外；黄芩苦寒清金，降之于里；连翘散上焦之血凝气聚；梨皮利肺部之热蕴邪留；滑石、茯苓皆入肺引邪下导耳。
用药禁忌：None
附注：None
