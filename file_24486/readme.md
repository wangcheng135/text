名称：防风汤
组成：防风（去叉）1两半，当归（切，焙）1两半，泽泻1两半，威灵仙（去土）1两半，甘草（炙，锉）1两半，黄连（去须）1两半，虎杖1两半，石韦（去毛）2两，天门冬（去心，焙）2两，白石脂（研）2两，槐实（炒）2两，地榆2两，石膏（捶碎）3两，生地黄（切，焙）6两，大黄（锉，炒）1两，黄芩（去黑心）1两，犀角（镑）1两，消石（研）1两。
