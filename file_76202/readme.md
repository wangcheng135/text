序号：76218
名称：舒肝理气汤
组成：青橘叶9g，青皮9g，陈皮9g，枳壳9g，厚朴花6g，香附9g，苏梗6g，赤芍9g，白芍9g，柴胡6g，郁金9g，甘草3g。
出处：《临证医案医方》。
主治：慢性肝炎属气滞型。两胁窜痛，肝区脘腹胀满，舌苔白，脉弦。
加减：None
功效：舒肝，理气，止痛。
用法用量：None
制备方法：None
临床应用：None
各家论述：青橘叶、青皮走两胁，入肝胆，可舒肝理气；陈皮、枳壳、桔梗为一组常用理气药，善理中焦及两胁气滞；桔梗性升，枳壳性降，一升一降，气机通调，疼痛可止；芍药、柴胡柔肝舒肝，二药配合，一辛一酸，一舒一敛，一刚一柔，伍用甚妙，为治疗肝胆疾患的要药，临床可用于肝胆疾患的始终，柴胡又为肝胆经的引经药，可引诸药直达病所；郁金理血中之气，解郁止痛；芍药、甘草柔养缓急；厚朴花、香附、苏梗理气止痛。
用药禁忌：None
附注：None
