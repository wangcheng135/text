序号：9411
名称：牛黄丸
组成：牛黄半两，赤箭1两，独活1两，乌犀角屑1两，防风3分（去芦头），天南星1两（炮裂），牛膝3分（去苗），萆薢3分，茵芋3分，汉防己3分，麻黄1两半（去根节），仙灵脾1两，桂心3分，蝉壳半两，乌蛇肉1两（酒浸，炒令黄），川乌头半两（炮裂，去皮脐），天雄3分（炮裂，去皮脐），桑螵蛸半两（微炒），晚蚕蛾半两（微炒），干蝎半两（微炒），铅霜半两（研入），腻粉1分（研入），朱砂半两（细研），麝香1分（研入）。
出处：《圣惠》卷二十。
主治：瘫痪风。手足不遂，皮肤顽痹，口面斜，言语謇涩。
加减：None
功效：None
用法用量：每服10丸，以温酒送下，不拘时候。
制备方法：上为末，入研了药令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
