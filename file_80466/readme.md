序号：80482
名称：鳖甲丸
组成：鳖甲（去裙襕肉，用醋炙令黄）2两，附子（炮）2两，赤石脂2两，乌蛇（酒浸，去皮骨，炙黄）2两，黄耆1两，枳壳（去白，麸炒黄色）1两，当归1两，桂心1两，槐耳（微炙）1两，槐子（微炒）1两，猬皮（炙令焦黄）1两，槟榔1两，川大黄（锉，炒）1两，麝香半两（研入），皂荚10挺（不蛀者，汤浸，去皮，酥炙黄）。
出处：《医方类聚》卷一八三引《神巧万全方》。
主治：痔。肛边生鼠乳，及大肠疼痛，坐卧不得。
加减：None
功效：None
用法用量：每服30丸，空心食前温粥饮送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
