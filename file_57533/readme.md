序号：57548
名称：润肺止嗽丹
组成：天冬5钱，黄芩5钱，黄耆3钱，知母3钱，杏仁（去皮、炒）2钱，贝母3钱，青皮（炒）5钱，橘皮3钱，苏子（炒）3钱，生地3钱，桑皮5钱，淡竹叶3钱，紫菀5钱，天花粉5钱，冬花5钱，瓜蒌仁5钱，枣仁（炒）3钱，五味子（炙）5钱，桔梗2钱，前胡2钱，甘草（炙）2钱。
出处：《北京市中药成方选集》。
主治：肺气虚损，咳嗽喘急，痰涎壅盛，久嗽声哑，逢节必犯。
加减：None
功效：润肺定喘，止嗽化痰。
用法用量：每服2丸，日服2次，温开水送下。
制备方法：上共研为细粉，过罗，炼蜜为丸，重2钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
