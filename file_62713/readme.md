序号：62729
名称：清心莲子饮
组成：黄芩半两，麦门冬半两（去心），地骨皮半两，车前子半两，甘草（炙）半两，石莲肉7两半（去心），白茯苓7两半，黄耆7两半（蜜炙），人参7两半。
出处：《局方》卷五。
主治：心火偏旺，气阴两虚，湿热下注，遗精淋浊，血崩带下，遇劳则发；或肾阴不足，口舌干燥，烦躁发热。
加减：发热加柴胡、薄荷煎。
功效：清心养神，秘精补虚，滋润肠胃，调顺血气。益气阴，清心火，止淋浊。
用法用量：每服3钱，加麦门冬10粒，水1盏半，煎取8分，去滓，水中沉冷，空心，食前服。
制备方法：上锉散。
临床应用：None
各家论述：1.《局方》：药性温平，不冷不热。_x000D_
2.《回春》：此药温平，清火养神秘精。_x000D_
3.《医方集解》：此手足少明、足少阳太阴药也。参、耆、甘草，所以补阳虚而泻火，助气化而达州都，地骨退肝肾之虚热，柴胡散肝胆之火邪，黄芩、麦冬，清热于心肺上焦，使苓、车前，利湿于膀胱下部，中以石莲清心火而交心肾，则诸证悉退也。
用药禁忌：None
附注：本方方名，《医方集解》引作“莲子清心饮”。
