序号：65363
名称：女金丹
组成：玄胡索（醋炒）112两，白术（炒）112两，官桂112两，川芎112两，白芍112两，茯苓112两，没药（炙）112两，丹参112两，熟地112两，鹿角霜112两，吴茱萸（炙）112两，阿胶（炒珠）112两，藁本112两，白芷112两，甘草112两，赤石脂（煅）112两，白薇112两，橘皮224两，当归224两，香附（炙）336两，人参（去芦）32两，益母草320两，砂仁80两，党参（去芦）56两。
出处：《北京市中药成方选集》。
主治：子宫寒冷，经期不准，腹痛腰酸，四肢无力。
加减：None
功效：调经养血，温暖子宫。
用法用量：每服1丸，温开水或姜汤送下，1日2次。
制备方法：上为细末，过罗，炼蜜为丸，重3钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
