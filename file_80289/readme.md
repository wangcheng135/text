序号：80305
名称：覆盆子丸
组成：覆盆子1两，薯蓣1两，石斛（去根，锉）1两，熟干地黄1两，牛膝（去苗）1两，阳起石（酒煮半日，细研，水飞过）1两，桂心1两，巴戟1两，肉苁蓉（酒浸1宿，刮去皱皮，炙干）1两，菟丝子（酒浸3日，晒干，别捣为末）1两，蛇床子1两，山茱萸1两，枸杞子1两，五味子1两，人参（去芦头）1两，赤石脂1两，泽泻1两，鹿茸（去毛，涂酥，炙令微黄）1两，白茯苓1两，远志（去心）1两。
出处：《圣惠》卷九十八。
主治：五劳七伤。
加减：None
功效：强力益气，补虚损，壮腰脚，安五脏，驻颜色。
用法用量：每服20丸，空心温酒送下。渐加至30丸。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
