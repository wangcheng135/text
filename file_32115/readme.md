序号：32124
名称：坎离既济丸
组成：熟地黄（酒蒸，姜汁浸，焙）4两，生地黄（酒浸）2两，天门冬（去心）2两，麦门冬（去心）2两，山茱萸（酒蒸，去核）2两，山药2两，甘枸杞2两，肉苁蓉（酒洗，蒸）2两，黄柏（去皮，酒炒）2两，知母（酒炒）2两，当归（酒洗）2两，白芍药（酒炒）2两，白茯苓（去皮）1两半，牡丹皮1两半，泽泻1两，五味子1两，拣参1两，远志（甘草水泡，去心）1两。
出处：《鲁府禁方》卷二。
主治：虚劳。
加减：凡人年过40以后气血渐衰，可加斑龙胶4两，酒化开丸服。
功效：补心血，滋肾水。
用法用量：每服100丸，空心盐汤、黄酒任下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌铁器，忌三白。
附注：None
