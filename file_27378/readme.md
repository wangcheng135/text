序号：27386
名称：补肾熟干地黄散
组成：熟干地黄1两，五味子1两，桂心1两，当归1两（锉，微炒），白芍药1两，牛膝1两（去苗），杜仲1两（去粗皮，炙微黄，锉），石斛1两（去根，锉），人参1两（去芦头），附子1两（炮裂，去皮脐），白茯苓1两，荜澄茄3分，厚朴1两（去粗皮，涂生姜汁炙令香熟），白术1两，沉香1两。
出处：《圣惠》卷七。
主治：肾虚少气，腹胀腰疼，小腹急痛，手足逆冷，饮食减少，面色萎黑，百节痠疼，日渐无力。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，大枣3枚，煎至6分，去滓，不拘时候稍热服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
