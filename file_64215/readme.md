序号：64231
名称：大腹皮散
组成：大腹皮1两（锉），赤茯苓3分，桔梗3分（去芦头），牡丹3分，桃仁半两（汤浸，去皮尖双仁，麸炒微黄），槟榔1两，桑根白皮1两（锉），枳壳3分（麸炒微黄），鳖甲1两（涂醋炙令黄，去裙襕），郁李仁1两（汤浸，去皮，微炒），川大黄1两半（锉碎，微炒）。
出处：《圣惠》卷四十九。
主治：痃癖气，腹胁胀满，喘息促急，不思饮食。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中鳖甲、郁李仁、川大黄三味用量原缺，据《普济方》补。
