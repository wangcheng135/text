序号：60778
名称：通窍明目汤
组成：当归4钱，桃仁3钱（炒），红花2钱，丽参3钱，黄耆5钱，白术2钱（炒），菖蒲3钱，茯神3钱，川芎3钱，细辛2钱，甘草1钱。
出处：《眼科临症笔记》。
主治：暴盲症（视网膜中央静脉血栓）初期属虚脱证者。
加减：愈后头疼目胀，不时昏乱者，去细辛、牙皂，加楮实子5钱、茺蔚子4钱、川羌3钱、白芍4钱、生地8钱，以上加倍，共为丸，镜砂为衣。每服3钱，1日2次，白水送下。虽不能光明如初，亦可保守睛光不失。
功效：None
用法用量：水煎，并用牙皂5分，镜砂2分，赤金3张，共为细末，随药水冲服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
