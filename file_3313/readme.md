序号：3313
名称：天雄浸酒
组成：天雄（炮裂，去皮脐）3两，蜀椒（去目并闭口，炒汗出）1两半，乌头（炮裂，去皮脐）2两，茵芋（去粗茎）3两，干姜（炮）1两，附子（炮裂，去皮脐）2两，防风（去叉）1两半，羊踯躅（炒）1两半。
出处：《圣济总录》卷五。
主治：肾中风，筋急，两膝不得屈伸，手不为用，起居增剧，恶风寒，通身流肿生疮，风冷疾病，腰膝挛急缓纵。
加减：None
功效：None
用法用量：每服1合，渐加至1合半，空心、临卧各1服，酒尽将滓晒令干，为细散。每服半钱，渐加至1钱匕，服时候如前。
制备方法：上锉，如麻豆大，用生绢囊贮，以酒1斗5升浸之，春、夏5日，秋、冬7日开取。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
