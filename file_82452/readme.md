序号：82468
名称：鹳骨丸
组成：鹳骨3寸，雄黄4分，莽草4分，丹砂（一作丹参）4分，牡蛎4分（一作牡丹），藜芦2分，桂心2分，野葛2分，斑蝥14个，巴豆40个，蜈蚣1个，芫青14个。
出处：《千金》卷十七。
主治：遁尸，飞尸，积聚，胸痛连背，走无常处，或在藏，或肿在腹，或奄奄然而痛。
加减：None
功效：None
用法用量：每服2丸，每日3次。以知为度。
制备方法：上为末，蜜为丸，如小豆大。
临床应用：None
各家论述：《千金方衍义》：鹳善啖蛇，故其骨能疗尸疰，更以斑蝥、芫青、莽草、藜芦、蜈蚣、野葛等味汇入一方以毒攻毒。
用药禁忌：None
附注：方中鹳骨，《圣惠》作“鹤脑骨”，方名“鹤脑骨丸”。
