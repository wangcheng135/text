序号：76583
名称：蒺藜丸
组成：桔梗7两，蒺藜7两，木贼7两，羌活7两，蝉蜕7两，薄荷7两，防风7两，草决7两，覆盆子7两，当归7两，川芎7两，白芍7两，生地7两，白芷7两。
出处：《全国中药成药处方集》（吉林、哈尔滨方）。
主治：肝旺肾虚，目生障翳，视物昏花，迎风流泪，羞明畏光，雀目青盲；瘀火上灼，目赤焮肿，胬肉胀痛，热泪不止；白膜遮睛，血丝贯瞳，眼泡浮肿，瞳仁散大。
加减：None
功效：平肝明目，退翳清热。
用法用量：每服2钱，白水送下，1日2-3次。
制备方法：上为细面，用水泛为小丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌食辛辣；孕妇忌服。
附注：None
