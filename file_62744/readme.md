序号：62760
名称：清气化痰健脾丸
组成：白术（去黑心及梗，泔浸，炒）4两，枳实（去瓤，豉炒）2两，大半夏（姜片皂角水煮透）4两，南星（同上制）4两，白茯苓（去皮）4两，贝母（去心）1两，黄芩（炒）4两，黄连（姜汁浸炒）2两，栝楼仁（炒去油）4两，桔梗（去芦）3两，甘草（炙）2两，枯白矾2两，香附米（童便浸炒）2两，海石4两，紫苏子（炒）1两，杏仁（去皮尖双仁，炒）2两，神曲（炒）3两，麦芽（面炒），山楂肉2两。
出处：《便览》卷二。
主治：痰盛气滞，咳嗽喘满，脾胃虚弱少食，坐卧不宁。
加减：None
功效：None
用法用量：每空心临卧白汤、姜汤茶任下。
制备方法：上为末，用薄荷叶煎汁1碗，姜汁1碗，打神曲糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
