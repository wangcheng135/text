序号：58888
名称：秘授甘露饮
组成：童便。
出处：《玉钥》卷上。
主治：真阴亏竭，火炎灼肺，虚损失血，内热发为咽疮，喉癣。
加减：None
功效：降阴火。
用法用量：取出另倾茶碗内，与病者服下，每日早、晚共服2钟。取童便，须择无病无疮疖者5-6人，每早烹好松萝茶1大壶，令各童饮下，俟便出时，去头去尾不用，取中间者，以坛盛之。
制备方法：取童便半酒坛，要坛口大者，先用铁丝作四股络子，悬饭碗1个于坛内，约离童便3寸许，再用铅打成帽笠式，倒置坛口上，四围用盐泥封固，外加皮纸数层糊密，勿令泄气，再用砖搭成炉式，将坛放上，用桑柴文武火炼烧1炷香，去火候温，再将铅笠轻轻取起，勿令泥灰落下，则坛中所悬碗内自有清香童便露1碗。
临床应用：None
各家论述：None
用药禁忌：None
