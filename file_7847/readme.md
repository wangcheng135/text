序号：7847
名称：加减黄连解毒汤
组成：元参3钱，生地黄4钱，黄芩3钱，山栀子3钱，黄连2钱，煅石膏4钱，川大黄6钱，芒消2钱，木通3钱，甘草2钱。
出处：《医学探骊集》卷四。
主治：结热触动相火，客于咽嗌，咽喉肿痛，日复一日，水浆不能下咽，初起经刺少商出血及服药而肿仍不消、脉洪而数者。
加减：None
功效：None
用法用量：水煎，温服。
制备方法：None
临床应用：None
各家论述：此方用黄连、黄芩清其上焦之热，栀子、石羔清其中焦之热，木通清其下焦之热，元参、生地清热养阴，甘草和药调中，大黄、芒硝将其结热推（汤皿）而去，釜底抽薪。
用药禁忌：None
附注：原书治上症，并以锋针刺尺泽出血，泄其上焦之热。
