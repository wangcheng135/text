序号：54102
名称：柴胡石膏散
组成：赤芍药50两，柴胡（去苗）50两，前胡（去苗）50两，石膏（煅）50两，干葛50两，升麻25两，黄芩37两，桑白皮37两，荆芥穗（去土）37两。
出处：《局方》卷二。
主治：外感风热、少阳、阳明合病，壮热恶风，头痛身疼，鼻塞咽干，口渴心烦，咳嗽自汗，脉浮缓。时行瘟疫，心胸如满，寒热往来，痰实咳嗽，涕唾稠粘。伤风，及阳气郁遏，元气下陷。春夏感冒，声重声哑，甚至痰壅气喘，咳嗽咽干，自汗。
加减：None
功效：None
用法用量：每服2钱，水1盏，加生姜3片，豉10余粒，同煎7分，去滓，稍热服。小儿分作3服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：《医方集解》：柴胡平少阳之热，升、葛散阳明之邪，前胡消痰下气而解风寒，桑皮泻肺利湿而止痰嗽，荆芥疏风热而清头目，赤芍调营血而散肝邪，黄芩清火于上中二焦，石膏泻热于肺胃之部；加姜、豉者，取其辛散而升发也。
用药禁忌：None
附注：柴胡升麻汤（原书同卷宝庆新增方）、柴胡石膏汤（《奇效良方》卷十）。
