序号：53607
名称：家传大明膏
组成：大黄、苍术、柴胡、龙胆草、藁本、细辛、赤芍、菊花（倍）、红花、黄柏、黄芩、连翘、栀子、荆芥、防风、木贼、黄连、蒺藜、薄荷、羌活、独活、麻黄、川芎、白芷、天麻、蔓荆子、玄参、苦参、归尾、木通、生地黄、桑白皮、车前子、枳壳、皮消、甘草。
出处：《回春》卷五。
主治：翳膜攀睛，烂弦，赤障，胬肉，血灌瞳人，迎风流泪，怕日羞明，视物昏花，疼痛不止。
加减：如胬肉、云翳、昏矇、烂弦、风眼，入冰片少许。
功效：None
用法用量：点眼。
制备方法：上锉10大帖，用童便5碗煎熟，用炉甘石1斤，入炭火烧红，淬入药中10次，研烂去粗滓，将药入水铜盆内，重汤煮干成饼，晒干研千余下，每两入焰消8钱，黄丹5分，又研千余下，收入瓷罐内。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
