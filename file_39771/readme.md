名称：经验调经种子丸
组成：香附半斤（醋浸2两3日，酒浸2两3日，童便浸2两3日，盐水浸2两3日），当归（酒洗）2两，川芎2两，白芍药（酒炒）2两，麦门冬（去心）2两，川续断（酒洗）2两，条芩（酒炒）2两，牡丹皮2两，白茯苓2两，杜仲（盐水炒断丝）2两，白术（陈壁土炒）2两，牛膝（酒洗）2两，人参（去芦）2两，阿胶（蛤粉炒）2两，小茴香（炒）1两，艾叶（醋煮，捣烂作饼，新瓦烙干，研末）1两，怀熟地4两，黑豆（炒去壳）49粒。
出处：《医学正印》卷下。
