序号：47772
名称：神仙保命丹
组成：牛黄、冰片5钱，麝香、白芷、穿山甲、蛤粉1两（煅），乳香、胡椒2两，自然铜2两，大黄4两，没药2两，归尾2两，桃仁2两，苏木2两，五灵脂2两，红花2两，赤芍2两，木香2两，五加皮2两，血竭2两，青皮2两，无名异2两，甜瓜皮2两（炒），大戟、千金子（去油净）4两，地鳖虫1升（焙干），土豆根5钱，山茨菇1两，朱砂5钱。
出处：《伤科补要》卷四。
主治：跌打损伤，痈疽发背。
加减：None
功效：None
用法用量：每服1丸，重者2丸，老酒送下。
制备方法：上为末，炼蜜为丸，如弹子大，朱砂为衣，再用金箔为外衣，晒干，入瓷瓶内封固，不可泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中牛黄、麝香、白芷、穿山甲、乳香、大戟用量原缺。
