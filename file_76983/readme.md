序号：76999
名称：黑神丸
组成：木香1两，官桂2两，附子1两，当归2两，干姜2两，细墨1两，白术2两，荆三棱2两，陈橘皮4两（去白），芫花4两（以醋炒），巴豆2两（以好醋煮数沸，焙干，炒），槟榔3两，硇砂半两（入面煮糊），大黄半两（入面煮糊）。
出处：《普济方》卷一七三。
主治：积聚，酒食毒，冷气膨胀，五膈噎气，妇人血气，多年厌食。
加减：None
功效：消癥瘕，化酒食。
用法用量：如常服、化酒食，茶汤下3-5丸；心腹胀，橘皮汤下；癥块，生姜汤下；妇人血气，红花酒下；多年厌食，干柿裹10丸，生姜汤下，临卧服。
制备方法：上为末，用醋面糊为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
