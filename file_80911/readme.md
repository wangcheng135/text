序号：80927
名称：飞龙夺命丹
组成：犀黄2钱，辰砂（飞）2两，麻黄（去节）4钱，人中黄8钱，麝香3钱，腰黄1两，月石3钱，青黛（飞）5钱，珍珠3钱，蟾酥1钱5分，明矾5分，银消1钱5分，冰片4钱，牙皂3钱，灯草炭1两，真金箔300张。
出处：《青囊秘传》。
主治：痧胀腹痛，霍乱转筋，厥冷脉伏，神昏危急之症；及受温暑瘴疫、秽恶阴晦诸邪，而头晕痞胀，瞀乱昏狂；或卒倒舌强，遗溺不语，身热瘛疭，宛如中风；或时症逆传，神迷狂谵；小儿惊痫，角弓反张，牙关紧闭。
加减：None
功效：None
用法用量：取少许吹鼻取嚏；重者可用凉开水调服1分；小儿减半。
制备方法：上为极细末，和匀，装入瓷瓶中，固封无令泄气，每瓶1分。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
