序号：28736
名称：十八太保英雄丹
组成：老材香（即松香由黄转黑，代琥珀用）15g，象皮15g，血竭15g，胆星15g，藏红花15g，防风15g，白芷15g，马钱子15g，海螵蛸15g，升麻15g，乳香（去油）24g，没药（去油）30g，雄土元12g，生龙骨9g，当归9g，菖蒲9g，蟹骨9g，麝香9g。
出处：《古今名方》引少林寺秘方。
主治：跌打损伤，流血不止或未破而皮肤黑紫者，或被重物折压而致瘀肿疼痛难忍者。
加减：None
功效：祛瘀消肿，活血止痛
用法用量：血流不止者，将药面撒在伤处，立即止痛止血；未破溃者，用冷开水调和涂于患处。
制备方法：上为细末，研至无声为度，装入瓶内，勿使泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
