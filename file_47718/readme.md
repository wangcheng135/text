序号：47733
名称：神仙延寿酒
组成：生地黄2两，熟地黄2两，天门冬（去心）2两，麦门冬（去心）2两，当归2两，牛膝（去芦，酒洗）2两，杜仲（去皮，酒和姜汁炒）2两，小茴（盐酒炒）2两，巴戟（水泡，去心）2两，枸杞子2两，肉苁蓉2两，破故纸（炒）1两，木香5钱，砂仁1两，南芎2两，白芍（煨）2两，人参5钱，白术（去芦油）1两，白茯苓（去皮）2两，黄柏（酒炒）3两，知母（去毛，酒炒）2两，石菖蒲5钱，柏子仁5钱，远志（甘草水泡，去心）1两。
出处：《回春》卷四。
主治：虚人有热者。
加减：None
功效：和气血，养脏腑，调脾胃，解宿酲，强精神，悦颜色，助劳倦，补诸虚。久服百病消除。
用法用量：每随量饮之。
制备方法：上锉，用绢袋盛药入坛内，用酒60斤，煮3炷香为度，取出，埋土中3日夜，去火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
