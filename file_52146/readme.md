序号：52161
名称：面脂
组成：白术6分，茯苓6分，杜蘅6分，萎蕤5分，藁本5分，芎藭5分，土瓜根5分，栝楼5分，木兰皮4两，白僵蚕4两，蜀水花4两，辛夷仁4两，零陵香4两，藿香4两，菟丝子8分，栀子花3分，麝香（酒浸，绵裹）3分，鹰屎白3分，冬瓜仁5分，桃仁5合（并令碎），白蜡3两，羊脂（肾边者）1升，猪脂3升（水浸7日，日别易水），猪胰1具，白附子4分。
出处：《外台》卷三十二引《延年秘录》。
主治：None
加减：None
功效：泽面。
用法用量：酒2升，取猪胰、桃仁、冬瓜仁，绵裹纳酒中，挼令消，绞取汁，用渍药1宿，别煎猪脂令消，去滓，以鹅脂、羊脂、白蜡于铛中，用绵裹，纳铛微火煎，三上三下，药黄色，去滓，待澄候凝，纳鹰屎末，搅令匀。以涂面妙。
制备方法：上细切。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
