名称：大琥珀散
组成：琥珀（研）2两，干姜（炮）3两，石苇（去毛）3两，滑石（研）3两，牡丹皮3两，白茯苓（去黑皮）3两，芎3两，石斛（去根）3两，续断3两，当归（切，焙）3两，远志（去心）3两，人参3两，牛膝（去苗）3两，桂（去粗皮）2两半，肉苁蓉（酒浸，去皱皮，焙）4两，松脂（炼了者）4两，牡蒙4两，陈橘皮（汤浸，去白，焙）4两，荏子3升，松实（和皮用）3升，柏子仁3升，车前子1升，菟丝子（酒浸，别捣，焙）1升，菴（艹闾）子1升，枸杞子1升，胡麻子1升，芜菁子1升，麦门冬（去心，焙）1升，木通14两，蛇床子（炒）半升。
出处：《圣济总录》卷九十一。
主治：虚劳脱营，真气不足，形体毁沮，四肢沉重，咽干口燥，饮食无味，远视，惊悸不安，五脏虚损，病从内生。
加减：None
功效：轻身益气，消谷能食，耐寒暑，驻颜，润肌肤，力倍常人。
用法用量：每服3钱匕，以牛乳半盏，水1盏，同煎少时，和滓温热服。
制备方法：上为细散。
临床应用：None
