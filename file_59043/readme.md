序号：59058
名称：秦椒丸
组成：秦椒18铢，天雄18铢，玄参1两，人参1两，白蔹1两，鼠妇1两，白芷1两，黄耆1两，桔梗1两，露蜂房1两，白僵蚕1两，桃仁1两，蛴螬1两，白薇1两，细辛1两，芜荑1两，牡蒙20铢，沙参20铢，防风20铢，甘草20铢，牡丹皮20铢，牛膝20铢，卷柏20铢，五味子20铢，芍药20铢，桂心20铢，大黄20铢，石斛20铢，白术20铢，柏子仁1两半，茯苓1两半，当归1两半，干姜1两半，泽兰1两18铢，干地黄1两18铢，芎藭1两18铢，干漆2两，白石英2两，紫石英2两，附子2两，钟乳2两半，水蛭70枚，虻虫100枚，麻布叩复头7寸（烧）。
出处：《千金》卷二。
主治：妇人绝产，生来未产。
加减：None
功效：荡涤腑脏，使玉门受子精。
用法用量：每服10丸，稍加至20丸，酒送下，1日2次。若有所去如豆汁、鼻涕，此是病出，觉有异，即停。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：此即第一方白薇丸之立法，方中附子不逮，益以天雄、白术、虻虫、鼠妇；不逮，益以蜂房。以蜂房能治崩中漏下五色，又解钟乳、白术相反之毒，苏颂所谓下乳石毒也。
用药禁忌：None
附注：None
