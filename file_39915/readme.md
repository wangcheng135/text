序号：39925
名称：肥儿丸
组成：槟榔（用面剂裹，煨熟，去面，锉，焙）、陈皮（洗，去白）、青皮（洗，去白）、胡黄连、宣连（去须，锉碎，微炒）、白芜荑（炒，去扇）、使君子（煨，去皮）、肉豆蔻（如槟榔法煨）、人参（去芦）、夜明砂（微炒）、赤芍药、龙胆草（洗净，锉，炒）各等分。
出处：《传信适用方》卷四引荆南候医方。
主治：小儿五疳八痢，阴阳气不顺，虚痞腹胀，呕逆，腹痛泻痢。或小儿疳病累服药无效者。
加减：None
功效：消化乳癖积聚，肥肌，退面黄瘦，杀虫，安胃虫，进饮食。
用法用量：每服30-50丸，用紫苏、木瓜汤送下；泻痢，米饮送下，不拘时候，1日2-3次。
制备方法：上为末，薄面糊为丸，如萝卜子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：《普济方》有白芍药。
