序号：46582
名称：千金保胎丸
组成：当归（酒洗）2两，川芎1两，熟地（姜汁炒）4两，阿胶（蛤粉炒）2两，艾叶（醋制）1两，砂仁（炒）5钱，条芩（炒）2两，益母草2两，杜仲（去根皮，姜汁、酒炒）4两，白术（土炒）4两，陈皮1两，续断（酒洗）1两，香附米2两（醋、酒、盐水、童便各浸2日，炒）。
出处：《回春》卷六。
主治：女人受胎，气血不足，中冲脉有伤，经2月而胎堕者。
加减：None
功效：None
用法用量：每服100丸，空心米汤送下。
制备方法：上为细末。煮枣肉为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：节饮食，绝欲，戒怒。
附注：保胎丸（《仙拈集》卷三）。
