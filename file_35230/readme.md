序号：35240
名称：乳香宽筋丸
组成：乳香（别研）半两，没药（别研）半两，川乌（炮，去皮尖）半两，草乌（炮，去皮尖）半两，地龙（去土）半两，白牵牛（砂器内煮数佛）半两，何首乌1两（砂器内煮，去黑皮），白僵蚕1钱（炒去丝嘴），五灵脂（酒研，去石）7钱。
出处：《普济方》卷九十三引《卫生家宝》.
主治：左瘫右痪，口眼斜，走注风气，腰脚麻痹，一切风疾，赤眼白头风疮。
加减：None
功效：None
用法用量：每服15-20丸，食后茶、酒任下。如疮干研油调涂，疮湿燥掺，如打扑闪肭，皮骨损碎、研入没药、乳香各半钱调涂。
制备方法：上为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
