序号：84391
名称：七圣散
组成：白矾2钱，马牙消5钱，消石1两，铅丹3钱，硇砂1钱，蛇蜕半条，巴豆2枚（去壳）。
出处：《圣济总录》卷一二二。
主治：马喉痹，咽颊肿痛，吐气不快。
加减：None
功效：None
用法用量：上7味，先研白矾、牙消、硇砂3味，入罐子内，次入消石，次掺铅丹于上面，只用平瓦1小片盖，以慢火烧成汁，便用竹片子夹蛇蜕，搅5-7度，又入巴豆，更搅5-7度，取出候冷，研为散。如小可咽喉肿痛，咽津妨碍及口疮，只干掺1字；或大段喉痹及马喉痹，或腮颐生瘀肉，侵咽喉，即干掺半钱，安稳仰卧，其喉痈肿处自破。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
