序号：56540
名称：大香甲散
组成：沉香半两，鳖甲（汤浸，去裙襕，炙令黄香用）1两，柴胡（去芦）半两，人参半两，桔梗半两，茯苓（去皮）半两，川芎半两，藿香叶半两，羌活半两，木香半两，陈橘皮（去白）半两，牡丹皮半两，安息香半两，当归半两，厚朴半两（姜汁炙令香），京三棱半两（炮），官桂（去皮）半两，附子（炮，去皮脐）半两，牛膝（去苗）半两，桃仁（汤浸，去皮尖）半两，和皮大腹子1分。
出处：《博济》卷四。
主治：妇人血脏风虚冷气，肌肉黄瘦，饮食进退，经候不匀，心腹多胀，渐变如劳。
加减：None
功效：补血海，调气。
用法用量：上为末，分1半。每服2钱，水1盏，加生姜、乌梅各少许，同煎至8分，温服。余1半更入干漆1分，阿魏半两，赤芍药1分同为末，炼蜜为丸，如梧桐子大。每服20丸至30丸，空心煎乌梅、地黄汤送下。与散子相间服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：大香甲丸散（《妇人良方》卷五）。
