序号：41248
名称：金衣祛暑丸
组成：藿香1两，云苓1两5钱，丁香1钱5分，甘草7钱，木瓜2钱5分，苏叶2钱5分，薏仁5钱，朱砂5钱，香薷5钱，檀香5钱。
出处：《全国中药成药处方集》（西安方）。
主治：暑天贪凉，饮食冰冷，发热恶寒，头痛无汗，烦闷，精神困倦，急性肠胃炎，呕吐下利，腹痛。
加减：None
功效：解热发汗，健胃利尿，镇吐制泻。
用法用量：大人1次服1-2丸，1日2-3次，小儿减半，开水或姜汤送下。
制备方法：上为细末，炼蜜为丸，每丸重1钱，朱砂与金箔为衣。
临床应用：None
各家论述：None
用药禁忌：中暑，面赤心烦，卒然昏倒，心脏衰弱，大汗淋漓之病症最忌服用。
附注：None
