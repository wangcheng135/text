序号：70733
名称：紫葛丸
组成：紫葛3分，石膏3分，人参3分，丹参3分，细辛3分，紫参3分，苦参3分，玄参3分，齐盐3分，代赭3分，苁蓉3分，巴豆3分，乌头3分，干姜5分，桂心5分，独活5分。
出处：《千金》卷十二。
主治：诸热不调，腹中积聚，心腹满，心下坚，宿食痰饮，食吐逆；上气咳嗽，咽喉鸣，短气；黄疸，久疟，面肿，四肢烦重，身浮肿，坐起体重；热病湿(匿虫)下部痒，大肠出，热淋，关格不通，下利颜色不定；羸瘦无力，弱房少精，精冷；体疮痒，身体斑驳；从高坠下绝伤；坠胎后伤损血，皮肉焦烂，月水不定，或后或前，月水断，心下闷满，肩膊沉重；小儿百病，小儿癖气乳不消，小儿身常壮热，腹内有病。
加减：None
功效：None
用法用量：每服6丸，食前3丸，食后3丸。若觉体中大热，各减1丸服之。服药后10日，得利黄白汁大佳。妇人食前、食后只服2丸；2岁以下儿服米粒大。
制备方法：上为末，炼蜜为丸，如小豆大。
临床应用：None
各家论述：《千金方衍义》：紫葛丸治寒热不调而为方后诸病，故用巴豆、姜、桂、乌头以破寒积；紫葛、石膏、石盐、苦、紫、丹、元四参以散热结；独活、细辛以通经络；代赭治贼风、鬼疰、虫毒；苁蓉止茎中寒热痛，人参以助诸药之力；但紫葛药肆罕得，宜以紫菀代之，以本经专主胸中寒热结气也。
用药禁忌：忌五辛、猪、鸡、鱼、蒜。
附注：None
