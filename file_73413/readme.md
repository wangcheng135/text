序号：73429
名称：椒红丸
组成：椒红1两（微炒），沉香1两，附子1两（炮裂，去皮脐），蓬莪术1两，诃黎勒皮1两，当归1两（锉碎，微炒），高良姜半两（锉），肉豆蔻半两（去壳），丁香半两，白术1两，麝香1分（研入）。
出处：《圣惠》卷七十。
主治：妇人血风，气攻脾胃，脏腑虚冷，全不思食，脐腹多痛，体瘦无力。妇人血气不调，风冷邪气，乘虚客搏，胁肋时胀，面色萎黄，肌体羸瘦，怠惰嗜卧。
加减：None
功效：补虚损，暖下脏，逐痼冷，进饮食。
用法用量：每服20丸，食前以温酒送下。
制备方法：上为细末，以酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
