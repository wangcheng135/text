名称：獭肝金牙散
组成：獭肝1具（炙干），金牙（研）半两，丹砂（水飞，研）半两，狸骨（炙，捣研）半两，牛黄（研）半两，麝香（研）半两，龙胆半两，白薇半两，附子（炮裂，去皮脐）半两，当归（水洗，切，焙干）半两，蒲黄（炒）半两，鬼督邮半两，铜照鼻（烧，醋淬，碎）半两，苦参半两，龙骨（研）半两，沙参半两，蜈蚣（炙，去足）半两，蜀椒（去目及闭口者，炒出汗）半两，丹参半两，寒水石（烧，研）1两，禹余粮（烧红，醋淬）1两，消石（研）1两，玄参1两，牡丹皮1两，鬼臼1两，人参1两1分，鳖甲（去裙襕，醋炙）1两1分，干姜（炮裂）1两1分，菴（艹闾）子3合。
