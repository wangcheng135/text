序号：47591
名称：神仙太乙至宝万全膏
组成：当归半两，大黄半两，玄参半两，赤芍药半两，没药半两，肉桂半两，白芷半两，生干地黄半两，乳香半两。
出处：《医方类聚》卷一九四引《经验秘方》。
主治：八发痈疽，一切恶疮，不问远年近日，已未成脓，蛇虎蝎犬、汤火刀斧所伤，发背，久远瘰疬，诸痿疮，诸般疥疮，腰膝疼痛，妇人血气，赤白带下，吐血，风赤眼，咳嗽咽喉肿，一切风劳病。
加减：None
功效：None
用法用量：如摊时，用小器内分药，于文武火上化开摊之；如作丸，令如鸡头大，蛤粉为衣，煎汤使酒送下；蛇虎蝎犬、汤火刀斧所伤，并可内服外贴；发背，先以温水洗疮，拭干，用绵子摊膏药贴之，以温水下1丸；久远瘰疬，摊贴，温水下1丸；诸痿疮，盐汤洗贴，酒下1丸；打扑损伤，外贴，橘皮汤下1丸；腰膝疼痛，外贴，盐汤下1丸；妇人血气，木通甘草汤下1丸；赤白带下，酒下1丸；吐血，桑白皮汤下1丸；风赤眼，贴太阳穴，栀子汤下1丸；咳嗽咽喉肿，绵裹1粒，含化；一切风劳病，柴胡汤下1丸；一切疮疖并肿痛疮，及诸般疥疮，别炼入油少许，打膏令匀涂之；诸疾度其情而用之。
制备方法：上切如松子大，用香油1斤浸药，春5日、夏3日、秋7日、冬10日，然后以文武火于砂锅内熬白芷赤黄色为度，绢绵滤去滓，将油再熬得所，下黄丹半斤，以柳枝搅，至滴油在水中不散成珠，看硬软不粘手，即用瓷器盛之。
临床应用：None
各家论述：None
用药禁忌：None
