序号：2460
名称：升麻煎
组成：升麻4两，玄参4两，蔷薇根白皮4两，射干4两，大青3两，黄柏3两，蜜7合。
出处：《千金》卷六。
主治：膀胱热不已，口舌生疮，咽肿。
加减：None
功效：None
用法用量：上(口父)咀。以水7升，煮取1升5合，去滓，下蜜更煎两沸，细细含咽之。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：升麻性升，散风肿诸毒，疗喉痛口疮；玄参治肾虚真阴失守，膀胱之火僭逆，咽喉肿痛；射干疗喉痹，咽痛不得息，散结气；大青泻肝胆湿热，解毒杀虫；得薇根、相皮共襄厥功；蜂蜜解毒和中，滋润喉舌，留恋诸药性味，含之尤为得宜。
用药禁忌：None
附注：升麻饮(《圣济总录》卷五十三)。本方方名，《普济方》引作“升麻散”。
