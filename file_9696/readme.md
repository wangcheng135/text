序号：9696
名称：牛膝丸
组成：牛膝1两（去苗），川椒1两（去目及闭口者，微炒去汗），芎藭3分，附子1两（炮裂，去皮脐），木香半两，当归3分（锉碎，微炒），干姜3分（炮裂，锉），白术2分，熟干地黄1两，桂心1两，泽兰3分，蓬莪荗1两，肉豆蔻1两（去壳），硼砂1两半（研入），青橘皮3分（汤浸，去白瓤，焙）。
出处：《圣惠》卷七十。
主治：妇人久冷，血气凝滞，面色萎黄，四肢羸瘦，不思饮食，腹中多痛。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以暖酒送下。
制备方法：上为粗末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
