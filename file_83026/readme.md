序号：83042
名称：乌头丸
组成：乌头（炮，去皮）半两，巴豆（去心皮，熬）半两，人参1两，消石1两，大黄2两，戎盐1两半，苦参3分，黄芩3分，(庶虫)虫（熬）3分，半夏（洗）3分，桂心3分。
出处：《千金翼》卷五。
主治：心腹积聚，膈中气闷胀满，疝瘕，内伤瘀血，产乳众疾及诸不足。
加减：None
功效：None
用法用量：宿不食，酒服5丸。卧须臾，当下。黄者，心腹积也；青如粥汁者，膈上邪气也；下崩血如腐肉者，内伤也；赤如血者，乳余疾也；如蛊刺者，虫也。下已，必渴。渴，饮粥；饥，食苏糜。3日后当温食，食必肥浓。40日平复。
制备方法：上为末，纳蜜、青牛胆汁拌和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
