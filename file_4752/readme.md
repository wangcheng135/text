序号：4752
名称：木香散
组成：木香3分，诃黎勒1两（煨，用皮），前胡1两（去芦头），白术半两，丁香半两，人参半两（去芦头），厚朴1两（去粗皮，涂生姜汁，炙令香熟），陈橘皮1两（汤浸，去白瓤，焙），鳖甲1两（涂醋，炙令微黄，去裙襴），枳壳半两（麸炒微黄，去瓤），桂心半两，当归半两，槟榔半两，赤茯苓半两，甘草1分（炙微赤，锉）。
出处：《圣惠》卷二十八。
主治：气劳，心胸不利，腹中多气，少思饮食，四肢无力。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，大枣3枚，煎至6分，去滓，不拘时候稍热服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：忌苋菜。
附注：None
