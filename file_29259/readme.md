序号：29267
名称：远志煎
组成：远志（去心）半两，羚羊角屑半两，茯神半两，甘草（炙微赤，锉）半两，龙骨半两，杏仁（汤浸，去皮尖双仁，麸炒微黄）半两，紫菀（洗去苗土）半两，防风（去芦头）半两，龙胆1分（去芦头），蚱蝉1分（去翅足），百合1分，牛黄1分（细研），麝香1分（细研），川升麻3分，川大黄1两（锉，微炒），酥3两，蜜半斤。
出处：《圣惠》卷八十三。
主治：小儿身体壮热，惊悸，心神不宁。
加减：None
功效：安心神。
用法用量：用水3升，入银锅内，煎至半升，以新绵滤去滓，却入锅内，下牛黄、麝香、酥、蜜等，以柳篦不住手搅，慢火熬如稠饧方止，入瓷盒内盛。每服取两豆许大，用温水调下，日3-4次。
制备方法：上药先研牛黄、麝香为粉，除酥、蜜二味，粗捣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
