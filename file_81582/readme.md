序号：81598
名称：镇心当归汤
组成：当归（切，焙）2两，羚羊角（镑）2两，龙齿（碎）3两，茯神（去木）4两，人参1两，防风（去叉）2两，芎2两，杏仁（汤退去皮尖双仁，炒）2两，半夏（汤洗去滑7遍）4两，生姜（与半夏同捣，炒干）4两，桔梗（炒）2两，石膏（碎）3两，防己（锉）2两，桂（去粗皮）1两半。
出处：《圣济总录》卷十四。
主治：中风邪，虚悸恍惚，悲伤，或梦寐不安。
加减：None
功效：None
用法用量：每服10钱匕，以水3盏煎至2盏，去滓，入竹沥1合更煎2沸，分3服，每日空心、午时、夜卧各1服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
