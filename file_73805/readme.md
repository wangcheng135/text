序号：73821
名称：睡红散
组成：乌蛇（项下7寸，用酒浸1宿，去皮骨，炙黄色）1钱，青黛2钱，蝎稍10个（炒），牛黄半钱，硼砂半钱，脑子半钱，水银砂子半钱，珍珠末半钱，麝香1字，金箔10片，银箔10片，乌蛇尾（酒浸1宿，去皮骨，炙黄色）1钱，蛇黄（入火内烧令红，于米醋浸入，煅，如此3度）1钱，京墨（烧烟尽）1钱，天南星末（用生姜汁浸）1钱，半夏末（用生姜汁浸1宿）1钱。
出处：《幼幼新书》卷九引《家宝》。
主治：婴孩、小儿急慢惊风，手足搐弱，目瞪，口眼相引。
加减：None
功效：None
用法用量：每服婴孩半字，半岁1字，1-2岁半钱，2-3岁1钱，以意加减，金银薄荷汤调下。如一服搐定，即用调胃气观音散2-3服；如小儿再作气粗发搐，宜进鸡舌香散2-3服。
制备方法：上牛黄、麝香、硼砂、脑子、金银箔先研极匀，次入水银砂子再研，将余药捣罗为末，一处研匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
