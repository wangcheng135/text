序号：46876
名称：独活丸
组成：独活（去芦头）3分，防风（去叉）半两，白茯苓（去黑皮）半两，阿胶（炙令燥）3分，石膏（碎研）3分，玳瑁（镑）1两，人参1两半，甘草（炙，锉）半两，天南星（炮）1两，细辛（去苗叶）半两，丹砂（研）2两，白僵蚕（炒）半两，丁香1分，琥珀（捣，研）1两，牛黄（研）1分，麝香（研）1分，天麻半两，龙脑（研）半两。
出处：《圣济总录》卷十四。
主治：风惊邪及一切风，肢节不利，筋脉拘急，头目旋痛，恍惚心忪。
加减：None
功效：None
用法用量：每服1丸，甚者加1丸，细嚼，薄荷茶送下，不拘时候。
制备方法：上为末，再同研匀，别用安息香2两半（捣碎），以酒1升，研滤去滓，于银器内慢火熬成膏，和前药为丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
