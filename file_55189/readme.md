序号：55204
名称：桂星散
组成：辣桂4钱，川芎4钱，当归4钱，细辛4钱，净石4钱，菖蒲4钱，白蒺藜（炒，杵去刺）4钱，木通4钱，木香4钱，麻黄（去节）4钱，甘草4钱，大南星（煨裂）4钱，白芷梢4钱。
出处：《直指》卷二十一。
主治：耳聋、耳鸣、头痛。_x000D_
风虚耳聋。风入聋鸣，必有时头痛。寒虚耳聋。
加减：None
功效：None
用法用量：每服3钱，水1盏半，加葱白2片，紫苏5叶，生姜5片，水煎，食后服。晚少食，临卧加些全蝎服。
制备方法：上为碎。
临床应用：None
各家论述：None
用药禁忌：None
附注：桂香散（《医学入门》卷七）、桂辛散（《张氏医通》卷十五）、桂花散（《杂病源流犀烛》崇文书局刻本卷二十三）。
