序号：36093
名称：参茸补肾丸
组成：远志2两，核桃2两，萸肉2两，巴戟肉2两，杜仲3两，楮实子2两，川牛膝2两，山药3两，贡桂1两，茯苓3两，川附子2两，沉香1两，故纸2两，寸芸3两，母丁香2两，菟丝子3两，熟地6两，茴香1两，虎骨2两，茸片2两，鹿肾2两，淫羊蔻1两，柏仁霜2两，红参2两，枸杞子3两。
出处：《全国中药成药处方集》（抚顺方）。
主治：气血虚弱，瘦弱神倦，便溏久泻，气短无力，腰腿痠痛，怔忡健忘，失眠惊悸，肾虚阳痿，见色自泄，精汁清稀。
加减：None
功效：补气，养血，壮阳。
用法用量：每服2钱，食前白术送下。
制备方法：上为细末，蜜丸2钱重。
临床应用：None
各家论述：None
用药禁忌：忌食生冷；火盛者忌服。
附注：None
