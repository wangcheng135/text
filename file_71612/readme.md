序号：71628
名称：善应膏
组成：巴豆（去壳）5钱，僵蚕（去丝嘴）5钱，赤芍药5钱，白芷5钱，五倍子2钱，黄连1钱，乱发如鸡子大，桃枝7条，柳枝7条，萆麻子（去壳）30粒。
出处：《医方大成》卷八。
主治：一切疮疽，及伤折损痛。
加减：None
功效：None
用法用量：任意使用。
制备方法：用清油半斤，浸药3日，慢火煎熬，令乱发焦烂，出火候冷，用绢滤去滓，再澄，却入铫内上火再熬，次入飞过黄丹4两，以桃柳枝不住手搅青烟微出为度，要滴在水上不散方成膏，却出火，搅令温，再入乳香末5钱，没药末5钱，桂心末3钱，略上火再搅令匀，却以净瓷器收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
