序号：58819
名称：秘传遗粮汤
组成：山黄牛3两2钱，五加皮1钱5分，白鲜皮1钱5分，防风1钱5分，木瓜1钱5分，荆芥穗1钱5分，白芷1钱5分，当归（酒洗）1钱5分，白芍药（炒）1钱，生地黄（酒洗）1钱，地骨皮1钱，川牛膝1钱，黄连1钱，甘草1钱，槐花（炒）1钱，川芎1钱，威灵仙1钱，寻风藤1钱，白茯苓1钱，杜仲2钱（姜汁炒断丝），皂角子3粒（捣碎），白丑3粒（捣碎）。
出处：《松崖医径》卷下。
主治：杨梅疮，风漏筋骨疼痛。
加减：None
功效：None
用法用量：用水1钟半，酒半钟，煎至1钟，察病患上下饥饱服，滓再煎服，1日1剂。服至3-5剂，其疮愈肿，勿惧畏，以托出毒气故也。若轻者服10剂而愈，重者服20剂全愈。每剂服过，滓晒干煎水，常时浴洗患处。
制备方法：上切细，作1服。
临床应用：None
各家论述：None
用药禁忌：忌房事、生冷、茶清、煎炒、母鸡、鹅、羊、猪首蹄肉、鱼虾动风之物。
附注：None
