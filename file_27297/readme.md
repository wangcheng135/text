序号：27305
名称：补肺汤
组成：五味子3两，白石英（研，绵裹）3两，钟乳（研，绵裹）3两，桂心3两，橘皮3两，桑根白皮3两，粳米2合，茯苓2两，竹叶2两，款冬花2两，紫菀2两，大枣50枚，杏仁50枚，（去皮尖双仁），苏子1升，生姜5两，麦门冬4两（去心）。
出处：《外台》卷十引《集验方》。
主治：肺气不足，咳逆短气，寒从背起，口中如含霜雪，语无音声而渴，舌本干燥。
加减：None
功效：None
用法用量：以水1斗3升，先煮桑白皮、枣、粳米令熟，去滓，纳诸药，煮取4升，分3次服，日再夜1。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：忌大醋、生葱。
附注：None
