序号：80253
名称：礞石丸
组成：礞石（研）1两，干姜（炮）1两，蓬莪术（煨，锉）1两，猪牙皂荚（烧存性，为末）1两，芫花（醋炒焦）1两，桂（去粗皮）1两，大黄（蒸熟，锉，焙，捣碎末）1两，京三棱（炮，锉，捣末）1两，硇砂1两，木香1两，肉豆蔻（去壳）1两，青橘皮（汤浸，去白，焙）5钱，墨（烧过）5钱，白豆蔻（去皮）5钱，槟榔（锉）5钱，丁香1分，诃黎勒皮1分，胡椒1分，巴豆1两5钱（去皮心膜，研出油）。
出处：《普济方》卷二十三。
主治：脾胃虚寒，宿食不消，攻胁下痛。
加减：None
功效：磨气块，取虚积。
用法用量：每服5丸或7丸，生姜汤送下，量脏腑虚实加减；妇人心痛，醋汤送下。
制备方法：上除大黄，京三棱并研药外，为末，用醋3升飞硇砂，于银石器内慢火熬30沸，次入巴豆，又熬十数沸，又入京三棱末，又熬五七沸，次入白面2两，无灰酒半斤，又熬20沸，次入大黄末，又熬5-7沸，下诸药末，不住用柳枝搅成膏，硬软得所，候冷，入末臼捣一千杆，为丸三等，或如绿豆大，或如麻子大，或如黄米大。
临床应用：None
各家论述：None
用药禁忌：None
