序号：9488
名称：加味活命饮
组成：穿山甲（土炒）3钱，银花3钱，大黄3钱，归尾1钱半，陈皮1钱半，花粉1钱，赤芍1钱，生地1钱，薄荷1钱，防风1钱，白芷1钱，贝母1钱，甘草节1钱，乳香1钱，没药（净）5分，角刺5分，以上3味后下（1方无大黄、生地、薄荷）。
出处：《痧书》卷下。
主治：瘀后留滞热毒，发为痈肿、发背、疔疽。
加减：毒在背，加角刺1钱半；在腹，加白芷；在胸，加蒌仁2钱；在头面手足，加银花5钱。
功效：None
用法用量：加水入大瓦瓶封口煎，温服，侧睡。一方好酒煎。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌铁器，酸味、诸毒物。
附注：加味活命散（《杂病源流犀烛》卷二十一）。
