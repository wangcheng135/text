序号：74695
名称：雷丸丹
组成：雷丸（生）半两，鹤虱（生）半两，使君子（去壳，生）半两，胡黄连（微炒）半两，芦荟（研）半两，麝香半两（研入），蟾1枚（酒浸1宿，慢火炙熟，去皮足骨，焙），木香1分，肉豆蔻1分，芜荑1两（去皮，微炒，研入），朱砂2钱（研少许，为衣）。
出处：《幼幼新书》卷二十五引《万全方》。
主治：小儿一切疳，肚胀腹满，手脚枯细，眼目口鼻生疮，身体壮热，痢下泔淀，日渐羸瘦，面无光泽。
加减：None
功效：None
用法用量：每服5-7丸，空心、日午，临卧麦门冬熟水送下。
制备方法：上为末，用豮猪胆4个，将汁倾入瓷盏中，外以重汤煮过，和杵为丸，如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
