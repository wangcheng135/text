序号：6027
名称：加味青州白丸
组成：白附子2-1两，天南星2-1两，半夏2-1两，川姜2-1两，白僵蚕1-2两，天麻1-2两，干蝎1-2两，川乌头（去皮尖）半两。
出处：《普济方》卷九十四引《经验良方》。
主治：中风，半身不遂，口眼斜，痰涎闭塞，咳嗽咯血，胸膈满闷，小儿惊风，妇人血风。
加减：None
功效：常服去风痰，利壅膈、安神定志。
用法用量：每服30-50丸，生姜汤送下，不拘时候，如瘫风，温酒送下；小儿惊风，薄荷汤吞下5丸，一方以前8味1料为末，加真料苏合香丸2两，姜汁糊丸，如梧桐子大，薄荷汤送下；小儿偶患惊风，生姜、薄荷汤磨下3-5丸，灌之立定。
制备方法：上并生为细末，白面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
