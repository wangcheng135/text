序号：70439
名称：紫河车丸
组成：人参1两，天麻1两，炙草1两，犀角1两，远志（甘草汁浸）1两，滑石1两，白芍（炒）1两，茯神1两半，枣仁1两半，天竺黄5钱，朱砂（研）5钱，紫河车1具（烘，研），脐带（新瓦上炙焦，另研）3条。
出处：《医部全录》卷四三二引《幼幼近编》。
主治：小儿胎惊。
加减：急惊，去紫河车、脐带、人参，加白僵蚕（蜜炙）6钱，全蝎6钱，牛黄1钱2分，琥珀1钱，胆星8钱，麝香3分。
功效：None
用法用量：每丸重1钱2分，饥时、临卧以灯心、薄荷汤化下。
制备方法：上为细末，用钩藤汁4两和炼蜜半两为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
