序号：35117
名称：乳炼秋石奇方
组成：童便2桶，皂角12两，杏仁10两。
出处：《遵生八笺》卷十七。
主治：None
加减：None
功效：益寿延年，返元还本，发白变黑，百疾不生。
用法用量：每秋石1两，入柿霜3钱同和，每服1-2分起，至7-8分止，空心白滚汤调下。
制备方法：水9碗，煎皂角至3碗，倾入便内，用桃、柳枝搅打便水，淀清倾去浊脚；次将杏仁10两煎汁3碗，倒在便内，又如前搅打，去清留浊；又将猪脂油12两熬成汁，去滓，倾入便内搅，浮膜倾去，又淀清，将人乳汁用滚汤泡成块倾入便内，再搅如前，又淀1日，倾去清水下底浊粉浆水，用木勺乘起，倾桑皮纸上，先将毛灰一矼作一沉窝，将桑皮纸放灰上以渗便水，纸上干白腻粉，即成秋石矣。不可动摇，晒1-2日，瓷瓶收起。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
