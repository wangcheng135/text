序号：24309
名称：人参薯蓣丸
组成：生地黄1两，人参1两，防风1两，薯蓣1两，五味子1两，茯苓1两，麦门冬2两半，贝母半两，远志半两，熟地黄3分，百部3分，柏子仁3分，丹参3分，杜仲3分，茯神3分，黄耆3分。
出处：《鸡峰》卷七。
主治：虚劳，肾脏虚弱，客风流入四肢，四肢烦满沉重，腰背拘急，不能俯仰，体热身重，毒风上攻，心胸闷满，攻注颈项，志意不乐，肌肤消瘦，嗜卧无力，喜怒好忘，若服暖药，又加转甚者。
加减：None
功效：聪明耳目，保定骨髓，开心强记，去惊怖，除邪热。
用法用量：每服10丸，食后熟水送下。
制备方法：上为细末，炼蜜为丸，如樱桃大，或梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
