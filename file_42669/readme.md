序号：42684
名称：保幼化风丹
组成：南星1两，半夏1两，川乌1两，白附子1两（水洗净），郁金5钱。
出处：《古今医鉴》卷十三。
主治：小儿惊风，四证八候。
加减：None
功效：祛风痰，散惊热。
用法用量：每1两加入雄黄、朱砂、硼砂、焰消各1钱，片脑、麝香各少许，共为末，炼蜜为丸，如豌豆大。灯草、薄荷汤研化下。
制备方法：上为末，装入腊月黄牛胆内，阴干，百日取出，研为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：小儿有热，热盛生痰，痰盛生惊，惊盛发抽。又盛则牙关紧急，而八候生焉。搐、搦、掣、颤、反、引、窜、视是也。搐者，两手伸缩；搦者，十指开、合；掣者，势如相扑；颤，头偏不正；反者，仰向后；引者，臂若开弓；窜者，目直似怒；视者，晴露不活。是谓八候也。其四证者，即惊、风、痰、热是也。
