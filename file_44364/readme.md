序号：44379
名称：一粒金丹
组成：雄精乳香（去油）1钱，没药（去油），砂仁1钱，羌活1钱，半夏（姜汁炒）2钱，乌药2钱，巴豆（去心衣，炒黑）1两5钱，山豆根5钱，苍术（米泔浸）4钱，杏仁49粒（去皮尖），麝香3分。
出处：《惠直堂方》卷一。
主治：中风，痰厥，心气痛，痢疾，身热，肚胀，各种肿毒，蛇伤，疯犬伤，疟疾，小儿惊风，喉症，痄腮，胃气冲痛，疫疠温毒。
加减：None
功效：解毒，消肿，止痛。
用法用量：每服1丸，病在上部，研碎服；中风痰厥暴死，但心头有微热，用清汤送下；心气疼痛，艾醋汤送下；痢疾，甘草汤送下；气不顺，木香汤送下；身热，白汤送下；肚胀，香附汤送下；口眼歪斜，麻黄汤送下；诸般肿毒，老酒送下；蛇伤，雄黄汤送下；身肿，荆芥汤送下；疯犬咬，斑蝥7个（炒），防风汤送下；疟疾，井水送下；小儿惊风，薄荷汤送下；头痛，川芎汤送下；酒肉食积，盐汤送下；喉症，薄荷汤送下；痄腮红肿，赤芍汤送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大，辰砂为衣。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。服药期间忌猪、羊、鹅、牛、鸡、糟、面、生冷、油煎；服药后勿饮冷茶水。
附注：None
