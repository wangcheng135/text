序号：78489
名称：摩挲丸
组成：黑参（拣润者洗，焙干）8两，地榆（去苗）8两，川乌（炮，去皮脐）8两，木香8两，丁香8两，天台乌药4两，熏陆香（用滴乳香别研）4两，雄黄（研，飞）4两，乌犀（镑，别研细）4两，龙脑（别研）4两，辰砂（研，飞）4两，自然铜（烧赤，醋淬）4两，麝香（别研）4两，天麻（去苗）1斤，真珠末（细研）2两（缺，以龙齿代）。
出处：《局方》卷一。
主治：中风瘫缓，半身不遂，口眼斜，言语謇涩，精神昏塞，步履艰难，或肌肉偏枯，手足軃曳，或筋脉拘挛，不得屈伸，及气痹并诸风身体疼痛。
加减：None
功效：None
用法用量：每服1丸，温酒化下，不拘时候。病重者服一月全安，轻者半月愈，初患5-7服可安。
制备方法：上为末，研匀，炼蜜为丸，如楮实大。
临床应用：None
各家论述：None
用药禁忌：服讫，避风处衣被盖覆令汗出。
附注：None
