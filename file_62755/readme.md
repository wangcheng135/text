序号：62771
名称：清气饮
组成：杏霜2-3钱，桔梗1-2钱，蝉蜕（去头足）2-3钱，银花2-3钱，广藿香2-3钱，苏叶1钱或1钱5分，神曲2-3钱，谷芽3-4钱，广皮5-7分，半夏1钱，赤茯苓2-3钱。
出处：《辨疫琐言》。
主治：疫症初起2-3日内，发热恶寒，头疼身痛，胸满胁胀，头目蒙混，脉往来凝滞而有力者。
加减：疫症4-5日，郁深则热，如有烦渴面红等热象，去苏叶，易冬桑叶2-3钱，丹皮1钱或1钱5分；口燥渴，去广皮、半夏，加瓜萎根1-2钱，或芦根5-7钱；烦热口苦咽干，加黄芩1钱，或1钱5分；小便不利，加白通草4-5分，或飞滑石2-3钱；腹胀大便闭，喜冷恶热，加大黄3-5钱或7-8钱；如寸口脉微弱，为里阳不充，加玉竹5-7钱。
功效：轻清开肺舒气，芳香醒胃辟邪。
用法用量：上以水2小碗，煎1碗，温服。如未觉，更进1服。觉气通舒畅，是其验也。重者日3服。
制备方法：None
临床应用：None
各家论述：杏霜、桔梗，苦以开肺；蝉蜕轻清上升而从风化，上焦如雾，一经郁遏，则雾气弥漫，用蝉蜕者，取清风生、雾气潜消之义；银花、藿香、苏叶芳香辟秽，散胸中不正之气；谷芽乃稻浸害而成，神曲乃面蒸窨而成，凡蒸窨之物，能舒郁遏，同气相求也；广皮辛香通畅；半夏滑利通阴；赤苓利水。三焦通畅，何气不清，放曰清气饮。2小碗水煎1碗，略煎便成，取情芬未散耳。
用药禁忌：None
附注：None
