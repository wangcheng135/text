序号：22191
名称：一阴煎
组成：生地2钱，熟地3-5钱，芍药2钱，麦冬2钱，甘草1钱，牛膝1钱半，丹参2钱。
出处：《景岳全书》卷五十一。
主治：肾水真阴虚损，而脉证多阳，虚火发热，及阴虚动血；或疟疾伤寒，屡散之后，取汗既多，脉气虚弱而烦渴不止，潮热不退者。
加减：如火盛躁烦者，加真龟胶2-3钱化服；如气虚者，间用人参1-2钱；如心虚不眠多汗者，加枣仁、当归各1-2钱；如汗多烦躁者，加五味子10粒，或加山药、山茱萸；如见微火者，加女贞子1-2钱；如虚火上浮，或吐血，或衄血不止者，加泽泻1-2钱、茜根2钱，或加川续断1-2钱以涩之，亦妙。
功效：滋阴清热，润肺止咳，止血。
用法用量：水2钟，煎7分，食远温服。
制备方法：None
临床应用：None
各家论述：《中医内科临床治疗学》：二地、芍药、麦冬滋阴清热，合丹参以清血分之热，合牛膝引血下行，甘草调和诸药，故全方有滋阴清热，润肺止咳，兼有止血之功。
用药禁忌：None
附注：此治水亏火胜之剂，故曰“一阴”。
