序号：32838
名称：羌活汤
组成：羌活（去芦头）1两，麻黄（去根节，汤煮，掠去沫，焙）1两，防风（去叉）3分，木香半两，槟榔（锉）半两，附子（炮裂，去皮脐）半两，白术半两，乌头（炮裂，去皮脐）半两，草豆蔻（和皮）半两，陈橘皮（汤浸，去白，焙）半两，牛膝（酒浸1宿，切，焙）半两，当归（酒浸1宿，切，焙）半两，杏仁（去皮尖双仁，生用）半两，人参半两，白茯苓半两，甘草半两，芎半两，桂（去粗皮）各半两。
出处：《圣济总录》卷七。
主治：偏风瘫痪，脚气。
加减：大肠不通，加大黄末1钱，久不通者，加至3-5钱；小肠不通，心腹胀，加葶苈、滑石末各1钱，滑石汤调下；上膈壅滞，咳嗽气急，加半夏、升麻、天门冬、知母末各2钱。
功效：None
用法用量：每服1两，水3盏，加生姜7片，煎至1盏，去滓，取7分，温服。服讫衣覆取汗。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
