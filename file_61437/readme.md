序号：61453
名称：商陆散
组成：商陆1两，曾青1分（细研），黄芩1两，防风1两（去芦头），白矾1两（烧令汁尽），人参1两（去芦头），小蓟根1两，石胆1分（细研），甘草1两（炙微赤，锉），雌黄1两（细研），赤芍药1两，白芷1两，荏枝1两，知母1两，桔梗1两（去芦头），雄黄1两（细研），狸骨1两（炙令黄色），银星礜石1两（烧赤，醋淬7遍），地胆1分（去头足翅，糯米拌炒令米黄，去米），斑蝥10枚（去头足翅，糯米拌炒令米黄）。
出处：《圣惠》卷六十六。
主治：浮疽瘘。或生于颈，或发于腋，肿硬如指，久即穿溃，有脓。
加减：None
功效：None
用法用量：每日空心及临夜卧时以谈醋调下1字。30日知愈，70日平复，甚者100日，无复所苦。凡服药宁从少起，过度即令人淋沥，淋沥则减服之。
制备方法：上为细散，研了药更研令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
