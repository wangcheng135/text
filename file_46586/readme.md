序号：46601
名称：炮肾散
组成：巴戟（去心，麸炒）1分，甘遂（炒黄）1分，木香1分，苦葶苈（炒）1分，沉香（锉）1分，泽泻1分，腻粉1钱，槟榔（1枚生，1枚炮）半两，陈皮（去白）半两，芫花（醋拌炒）半两，麦糵半两。
出处：《奇效良方》卷四十。
主治：水气肿满。
加减：None
功效：None
用法用量：每服2钱，用猪腰子1枚，以竹刀割开，去筋膜，切作3片，掺药末在内，用湿纸裹，慢火煨令香熟；先煮葱白3茎令熟，细切，将葱白与粟米同煮粥1碗。先食粥一半，方食腰子，药后再食粥令尽，临卧时服。至五更大小便下赤黄恶物是效。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
