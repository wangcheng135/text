序号：41334
名称：金宝赴筵膏
组成：大黄3钱，黄耆3钱，地龙（去土）3钱，当归3钱，龙骨3钱，乳香3钱，没药3钱，粉霜3钱，硇砂3钱，川山甲3钱，轻粉3钱，脑子1钱，江子21粒（去皮壳），麝香少许。
出处：《普济方》卷二七五。
主治：诸恶毒疮，盘蛇疮，疔疮。
加减：None
功效：None
用法用量：每药1两，加石灰3两，黄丹半两，逐绞之，待煎滴水中直到底不散者方好。提取用饼封，要用者入麝香、脑子。
制备方法：上荞麦灰1斗煎，淋灰三复之，汁煎三分之一，下用雪里之雀粪5钱，重煎10来沸，提取放冷，澄清再熬；入大黄末煎3沸，次入朴消，花碱者各3两重。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
