序号：29719
名称：附子丸
组成：附子1两（炮裂，去皮脐），巴戟1两，天麻1两，牛膝1两（去苗），防风3分（去芦头），桂心3两，芎3分，独活3分，石斛1两（去根，锉），肉苁蓉1两（酒浸，去皱皮，微炙），补骨脂1两，干蝎3分（微炒），萆薢1两（锉），椒红1两（微炒去汗），仙灵脾1两，沉香1两，安息香1两，木香1两。
出处：《圣惠》卷二十三。
主治：脏腑虚，血气不足，受风冷之气，使人面青心闷，呕逆吐沫，四肢疼冷。
加减：None
功效：补虚损，暖脏腑，利腰脚。
用法用量：每服20丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：巴戟天丸（《圣济总录》卷十二）。
