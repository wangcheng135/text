序号：60164
名称：透骨丹
组成：地骨皮3两，甜瓜子（炒）3两，芸苔子（葱捣为饼）3两，乳香（另研）1两，没药（另研）1两，草乌头1两（锉，炒），苍术2两，牛膝（酒浸）2两，赤芍药2两，当归（去芦）2两，川乌头（炮，去皮脐）2两，自然铜（醋煅）2两，五灵脂2两。
出处：《准绳·类方》卷四引《集验方》。
主治：男妇一切走注疼痛不可忍。
加减：None
功效：None
用法用量：每服10丸，加至15丸，以温酒送下，不拘时候。
制备方法：上为细末，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：先用甜瓜子一两炒香，研烂，酒煎数沸，量虚实调黑牵牛末五钱服之，以利为度，然后服此方。
