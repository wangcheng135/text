序号：31261
名称：沉香降气汤
组成：香附子（炒，去毛）400两，沉香18两半，缩砂仁48两，甘草120两（爁）。
出处：《局方》卷三（绍兴续添方）。
主治：肝气郁结，脾失健运，胸脘痞闷，心腹胀满，恶心呕吐，食欲不振；妇女月经不调，少腹胀痛。
加减：None
功效：开胃消痰，散壅思食，通顺气血。
用法用量：每服1钱，加盐少许，凌旦雾露，空心沸汤点服。
制备方法：上为细末。
临床应用：None
各家论述：《医略六书》：气逆于中，肝气不降，此眩晕之发于气逆焉，郁怒人多此。沉香降气以疏逆，香附调气以解郁、砂仁理气醒脾胃，甘草缓中和脾胃也，为散沸汤下，使逆气降而肝气平，则脾胃调而运化如常，何气逆眩晕之不已哉。
用药禁忌：None
附注：沉香降气散（《准绳·类方》卷二引《说约》）。本方改为丸剂，名“沉香降气丸”（见《丸散膏丹集成》）。
