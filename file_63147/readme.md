序号：63163
名称：大温中饮
组成：熟地3-7钱，冬白术3-5钱，当归3-5钱（如泄泻不宜用，或以山药代之），人参2-5钱（甚者1两，或不用亦可），炙甘草1钱，柴胡2-4钱，麻黄1-3钱，肉桂1-2钱，干姜（炒熟）1-2钱（或煨生姜3-7片亦可）。
出处：《景岳全书》卷五十一。
主治：阳虚伤寒，及一切四时劳倦，寒疫阴暑之气，身虽炽热，时犹畏寒，即在夏月，亦欲衣被覆盖，或喜热汤，或兼呕恶泄泻，但六脉无力，肩背怯寒，邪气不能外达。小儿痘疹，气虚兼寒者。
加减：如气虚，加黄耆2-3钱；如寒甚阳虚者，加制附子1-2钱；头痛，加川芎或白芷、细辛；阳虚气陷，加升麻；如肚腹泄泻，宜少减柴胡，加防风、细辛亦可。
功效：None
用法用量：水2钟，煎7分、去浮沫，温服。或略盖取微汗。
制备方法：None
临床应用：None
各家论述：《景岳全书》：此元阳大虚，正不胜邪之候，若非峻补托散则寒邪日深，必致不救。温中自可散寒，即此方也。服后畏寒悉除，觉有燥热，乃阳回作汗佳兆，不可疑之畏之。此外，凡以素禀薄弱之辈或感阴邪时疫，发热困倦，虽未见如前阴证，而热邪未盛者，但于初感时，即速用此饮，连进二三服，无不随药随愈。
用药禁忌：此方宜与理阴煎、麻桂饮相参用；此方惟气血两虚而重感寒邪者宜之，非正伤寒治法。
附注：大温中汤（《医学从众录》卷八）。
