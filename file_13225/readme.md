序号：13232
名称：人参丸
组成：人参3分（去芦头），甘草3分（炙微赤，锉），赤茯苓3分，干姜3分（炮裂，锉），桂心3分，细辛3分，赤芍药3分，诃黎勒皮1两半，槟榔1两，陈橘皮1两（汤浸去白瓤，焙），厚朴2两（去粗皮，涂生姜汁，炙令香熟），草豆蔻1两（去皮）。
出处：《圣惠》卷五十。
主治：五膈气，心胸不利，痰饮留滞，宿食不消，或为霍乱，心痛醋心，心腹气满，积冷时多。
加减：None
功效：None
用法用量：每服20丸，不拘时候，以生姜、枣汤送下。如似有物在咽喉中，即取10丸并成1丸，含化咽津。
制备方法：上为末，炼蜜和捣200-300杵为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
