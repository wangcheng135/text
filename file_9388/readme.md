序号：9388
名称：水银丸
组成：水银半两（入黑铅半两结为砂子，细研），天南星1分（炮裂），铅霜1分（细研），朱砂1分（细研），雄黄1分（细研），天竹黄1分（细研），犀角屑1分，麝香半分（细研），牛黄半分（细研），龙脑半分（细研），马牙消1分（细研），金箔30（细研），白附子1分（炮裂），干蝎1分（微炒），腻粉半分。
出处：《圣惠》卷八十五。
主治：小儿慢惊风，面青口噤，吐涎，脚冷身热，频频搐搦。
加减：None
功效：None
用法用量：每服3丸，以薄荷汤化下，不拘时候。
制备方法：上为末，入诸药同研令匀，用雀儿饭甕内虫十枚炼蜜同研为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
