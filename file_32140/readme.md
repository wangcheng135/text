序号：32149
名称：抑肝化痰汤
组成：山栀10g，黄芩10g，龙胆草10g，菊花10g，防风10g，半夏6g，橘红6g，甘草6g，制川军10g，礞石30g，茯苓12g。
出处：《中医内科临床治疗学》引冷柏枝方。
主治：痰热肝火证，痰多而粘稠色黄，或结为块，胸膈痞闷，恶心时作，口干口苦，头晕目眩，胁痛目赤，烦躁易怒，少寐多梦，小便短赤，大便干燥，舌红，苔黄腻，脉弦滑有力，或弦数。
加减：None
功效：清热化痰，凉肝泻热。
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：山栀，黄芩清三焦郁火；龙胆草清肝胆实热；菊花、防风凉肝散风利头目，并能顺逆肝木上升之性；半夏、橘红燥湿化痰；制军、礞石开化顽痰而通肠利肺，使痰热从大便排出；茯苓、甘草健脾利湿。合之则肺、脾、肝三脏兼治。
用药禁忌：None
附注：None
