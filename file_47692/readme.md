序号：47707
名称：神仙巨胜子丸
组成：黄精2两，木通2两，当归2两，黄耆2两，莲子2两，广木香2两，枸杞子2两，肉苁蓉（酒浸）2两，熟地黄（酒浸）2两，何首乌2两，人参2两，破故纸（酒浸）2两，柏子仁2两，巴戟（酒浸，去皮）2两，山茱萸2两，巨胜子（煎，去皮，燥干）2两，干山药2两，菟丝子（酒浸）2两，杜仲（酒浸）2两，酸枣仁2两，五味子（酒浸）2两，天雄1对，石菖蒲（酒浸）1两，楮实子1两，甘菊花1两，牛膝（酒浸3日）1两，小茴香（炒）1两，川乌头（炮）1两，白茯苓1两，覆盆子1两，远志（去心，酒浸，焙）1两，天门冬（酒浸，去皮）1两。
出处：《普济方》卷二二四。
主治：耳聋眼暗，诸病。
加减：如无天雄，可以附子代之。
功效：除百病，补真气，乌发，驻颜，耐寒，种子，延年益寿。
用法用量：每服30丸，空心温酒送下，每日2次。服至1月，真气完成；至50日，头白再黑；百日，颜如童子。
制备方法：上为细末，春、夏炼蜜为丸，秋、冬枣肉为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
