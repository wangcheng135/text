序号：37367
名称：实气散
组成：白术2两半，当归（微炒）1两，厚朴（姜汁制）1两，白茯苓（去皮）1两，熟干地黄1两，黄耆（蜜炙）1两，川乌（炮，去皮脐）1两，桑白皮1两，续断（炒）1两，枳壳（炒）3分，香白芷（炒）3分，牡丹皮（炒）3分，茴香（炒）3分，威灵仙3分，白蒺藜（炒，去刺）3分，白芍药3分，川芎3分，五味子3分，山药3分，山茱萸3分，干姜（炮）3分，蓬术（炒）3分，甘草（炙）3分，五加子1两。
出处：《简易》引《叶氏录验方》（见《医方类聚》卷八十八）。
主治：胁肋膨胀，中满刺痛。
加减：None
功效：补五脏气虚。
用法用量：每服2钱，水1盏，加生姜3片，大枣1个，煎至7分；食前盐汤点亦可。
制备方法：上为细末，用桃仁3分（麸炒，去皮），研细和前药。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
