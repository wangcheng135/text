序号：8183
名称：加减补中益气汤
组成：人参1钱半，黄耆3钱（蜜炙），白术3钱（制），升麻5分，当归3钱，柴胡5分，白芍1钱半（炒），龙骨3钱（煅），牡蛎3钱（煅），熟地5钱。
出处：《医略六书》卷二十六。
主治：阴中挺出，脉软者。
加减：None
功效：None
用法用量：水煎，去滓温服。
制备方法：None
临床应用：None
各家论述：气血大虚，元气不能收摄于下，故阴中挺出。熟地滋阴补血，人参补气扶元，黄耆补中益气，白术培土益脾，当归养血脉以荣经，白芍敛肝阴以和血，升麻升阳明清气，柴胡升少阳清气，龙骨涩精秘气，牡蛎涩精固阴。水煎温服，使气阴内充，则清阳不复下陷，而阴挺自收也。
用药禁忌：None
附注：None
