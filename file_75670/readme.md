序号：75686
名称：葱豉桔梗汤
组成：鲜葱白3枚至5枚，苦桔梗1钱至1钱半，焦山栀2钱至3钱，淡豆豉3钱至5钱，苏薄荷1钱至1钱半，青连翘1钱半至2钱，生甘草6分至8分，鲜淡竹叶30片。
出处：《重订通俗伤寒论》卷二。
主治：风温、风热初起。
加减：咽阻喉痛，加紫金锭2粒（磨冲），大青叶3钱；胸痞，原方去甘草，加生枳壳2钱，白蔻末8分（冲）；发疹，加蝉衣12只，皂角刺5分，大力子3钱；咳甚痰多，加苦杏仁3钱，广橘红1钱半；鼻衄，加生侧柏叶4钱，鲜茅根50支（去衣）；热盛化火，加条芩2钱，绿豆2两煎药；火旺就燥，加生石膏8钱，知母4钱。
功效：辛凉发汗。
用法用量：None
制备方法：None
临床应用：None
各家论述：《肘后》葱豉汤，本为发汗之通剂，配合刘河间桔梗汤，君以荷、翘、桔、竹之辛凉，佐以栀、草之苦甘，合成轻扬清散之良方。
用药禁忌：None
附注：None
