序号：30487
名称：还睛丸
组成：拣人参1两半，天门冬（泡，去心）3两，麦门冬（泡，去心）3两，生地黄（酒洗）3两，熟地黄1两（酒蒸），当归（酒洗）1两，川芎7钱，白茯苓（去皮）1两，山药1两（蒸），菟丝子（酒饮烂捣饼，焙干）1两，甘枸杞1两半，肉苁蓉（酒浸）1两半，川牛膝（去芦）1两半，川杜仲（酒炒）1两半，石斛1两半，五味子7钱，川黄连7钱，川黄柏1两（酒炒），知母2两（酒炒），杏仁（泡，去皮）1两半，枳壳（面炒）1两，防风8钱（去芦），菊花（酒洗）1两，青葙子1两，草决明1两，白蒺藜（炒）1两，羚羊角1两（镑），乌犀角8钱，甘草7钱。（炙）
出处：《古今医鉴》卷九。
主治：远年近日一切目疾，内外翳障，攀睛弩肉，烂眩风眼，及老年虚弱，目昏多眵，迎风冷泪，视物昏花，久成内障。
加减：None
功效：降火升水，夜能读细字。养血安神，搜风明目。
用法用量：每服30-50丸，空心盐汤送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
