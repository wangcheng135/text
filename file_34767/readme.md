序号：34777
名称：麦门冬汤
组成：麦门冬（去心，焙）3两，茯神（去木）3两，防风（去叉）3两，地骨皮（去土）3两，人参2两，龙齿2两，远志（去心）2两，甘草（炙黄）2两，羚羊角（屑）2两，石膏2两，紫石英1两。
出处：《圣济总录》卷九十三。
主治：心中烦热，唯欲露体，复之即闷烦，惊悸心忪，面无颜色，忘前失后，妇人患血风气者，多成此疾，乃心蒸之状。
加减：None
功效：None
用法用量：每服3钱匕，以水1盏半，加大枣两枚，煎取半盏，去滓温服。服1剂，未全安再作之，以愈为度。
制备方法：上药各锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：曾经吐血者，服尤佳。若畏石药，不用紫石英亦佳。
