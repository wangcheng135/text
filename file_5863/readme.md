序号：5863
名称：加减仙茅丸
组成：仙茅2斤（米泔浸5日，去赤水，用铜刀子去皮，用铜刀锉碎，夏月只浸3日，阴干，不见日，干称1斤），苍术2斤（米泔浸5日，或2日亦得，去皮，焙干，秤1斤），白茯苓（去皮）8两，车前子12两，茴香（炒香）8两，枸杞子1斤，生干地黄（焙干）4两，熟地黄（焙干）4两，柏子仁（微炒黄，捣）8两。
出处：《御药院方》卷六。
主治：None
加减：None
功效：强筋骨，益精神，明目，黑鬚鬓。
用法用量：每服50-60丸，空心、食前温酒下，日2服，渐加至70-80丸。
制备方法：上为细末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
