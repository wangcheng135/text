序号：52425
名称：香甲丸
组成：柴胡3分，生干地黄3分，荆三棱3分，鳖甲（醋炙黄）1两，神曲（炒）1两，杏仁1两，熟干地黄1两，麦糵（炒）1两，牛膝半两，木香半两，姜黄半两，当归半两，白术1分，芎1分。
出处：《幼幼新书》卷二十一引《张氏家传》。
主治：男子、妇人、童男、室女气血虚疏，肌肤消瘦，百节痛，潮作温，五心烦热，四肢逆冷，不思饮食，中满气滞，妇人经血凝涩。
加减：None
功效：健脾胃，畅神气，充肌肤，泽颜色。
用法用量：每服10丸，空心清茶送下；米饮亦得。
制备方法：上为细末，白面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
