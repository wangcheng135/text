序号：30529
名称：还睛丸
组成：白术（生用）、菟丝子（酒浸，别研）、青葙子（去土）、防风（去芦）、甘草（炙）、羌活（去苗）、白蒺藜（炒，去尖）、密蒙花、木贼（去节）各等分。
出处：《局方》卷七（续添诸局经验秘方）。
主治：男子、女人风毒上攻，眼目赤肿，怕日羞明，多饶眵泪，隐涩难开，眶痒赤痛，睑眦红烂，瘀肉侵睛；或患暴赤眼，睛疼不可忍者；偏正头痛，一切头风，头目眩运。
加减：None
功效：None
用法用量：每服1丸，空心、食前细嚼，白汤送下，日3次。
制备方法：上为细末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：明目还睛丸（《全国中药成药处方集》杭州方）。
