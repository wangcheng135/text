序号：80157
名称：毓真膏
组成：当归5钱，远志5钱，人参5钱，白芷3钱，红花3钱，五味子3钱，附子3钱，肉桂5钱，苍术3钱，鹿茸1对，甘草3钱，黄耆5钱，白及3钱，紫梢花5钱。
出处：年氏《集验良方》卷二。
主治：一切腹痛，痞疾，梦遗，五淋，白浊，色欲过度，阳事不举；妇人经水不调，赤白带下。
加减：None
功效：固精保元，暖肾，补腰膝，去寒湿；久贴暖子宫，助生育，生阳气，暖命门，生精疏水，开脾胃，爽精神。
用法用量：如用时，将铜匙或磁杯盛滚水顿化开膏，方无火气，贴脐上。
制备方法：上用麻油2斤，春浸五日，夏三日，秋七日，冬十日，慢火熬黑色，滤去滓，入黄丹1斤，搅至滴水成珠，不粘手为度，随取起，热烟将尽即入麝香3钱，阳起石3钱，乳香3钱，丁香3钱，鸦片3钱，共为极细末，缓缓加上，不住手搅，收瓷器内盖好，掘地窖埋1月，取起作膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
