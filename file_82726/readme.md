序号：82742
名称：中和汤
组成：神曲1钱5分，生菔子1钱5分，淡黄芩（酒炒）1钱5分，姜半夏1钱5分，茯苓1钱5分，山楂1钱5分，茅山术1钱5分，川连（酒炒）1钱5分。
出处：《医方简义》卷三。
主治：痢下，不论红白，里急后重，不拘男女小人，皆宜服之。
加减：时痢身热者，加藿香、薄荷、桔梗各1钱，生姜3片；如白沫，则加广木香8分；如赤痢，加桃仁10粒；赤白相兼者，加桃仁10粒，红曲8分，木香5分；如醉饱受邪者，加葛根1钱，紫金锭2分冲入；如挟怒挟食者，加柴胡、厚朴各1钱。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
