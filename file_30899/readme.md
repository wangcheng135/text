序号：30908
名称：十宝丹
组成：梅矾（取青梅圆大而脆者，用刀切下圆盖，去核，将明矾末捺实在内，仍以盖覆上，竹钉签好，过1宿，用炭火煅之，去梅灰，只用其矾，白如腻粉，味极平酸）1两，薄荷（用青鱼胆汁收之，去筋梗）1两，僵蚕（洗净折断，无筋连者，去头足，瓦上炙脆）4钱，冰片（另研）4钱，孩儿茶2两，牛黄1钱，血竭3钱，珍珠3钱，琥珀3钱，生甘草5钱。
出处：《良方汇录》卷下。
主治：缠喉风，塞喉风一切急症及口喉诸病。
加减：如寻常口喉症，去牛黄、珍珠、琥珀。
功效：None
用法用量：None
制备方法：上为极细末，瓷瓶收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
