序号：24387
名称：八仙早朝糕
组成：白术（炒）4两，白茯苓（去皮）2两，陈皮（去白）2两，山药（姜汁炒）4两，莲肉（去皮心）4两，薏苡仁（炒）4两，芡实（去壳，净）4两，人参（去芦）2两，桔梗（炒干）1两。
出处：《医便》卷一。
主治：脾胃虚弱，膨闷泄泻，不思饮食。
加减：小儿，加山楂4两、麦芽面4两，去人参。
功效：补脾。
用法用量：饥时取3-5片食之，白汤漱口。
制备方法：上为末，白粳米5升半，糯米2升，共7升半，同粉，共药和匀，用蜜3斤（如无蜜，砂糖4斤代之）拌匀。如做糕法，入笼中，划片蒸熟，焙干，瓦罐封贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：《痘疹一贯》有砂仁，无桔梗。
