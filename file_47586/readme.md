序号：47601
名称：千金润下丸
组成：大麻仁（微炒）1两，郁李仁（泡，去皮）7钱，菟丝子（酒蒸）7钱，枳壳（麸炒）7钱，牛膝（酒浸）7钱，车前子7钱，山药7钱，肉苁蓉（酒洗）1两半，威参（蜜蒸）2两，大黄（酒蒸）2两，陈皮（去白）5钱，桃仁（去皮）5钱。
出处：《会约》卷八。
主治：大便燥结，胀闷之甚，而脉未至大虚者。
加减：None
功效：None
用法用量：每服40丸，白汤送下，早、晚各1次。若肠润而肛门紧，用猪胆汁，少加皂角末，和以导之。便下宜止，即服滋阴养胃之剂，以扶其本。
制备方法：先将大黄、威参杵成膏，后加药末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
