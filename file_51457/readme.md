序号：51472
名称：轻脚丸
组成：川乌头（炮，去皮脐）5两，木瓜2枚，牛膝（去芦头，酒浸1宿）7钱，巴戟7钱（紫色者，酒浸1宿），青盐1两，肉苁蓉（酒浸，细锉，焙干）1两，天麻（去苗）1两，海桐皮1两半，防风（去芦头）1两半，甘草半两（炙），金毛狗脊半两（去毛），萆薢2两。
出处：《杨氏家藏方》卷四。
主治：脚气，脚膝沉重，疼痛肿满。
加减：一方加羌活、乳香、没药各1两，兼治妇人产后脚软，行履不得，筋骨疼痛。
功效：None
用法用量：每服30-50丸，空心温酒送下。
制备方法：上先将木瓜用竹刀切下盖子，剜去子，却填熟艾末令满，将盖子盖定，用竹签签住，入饭甑内蒸烂，先将乌头末同木瓜研匀，次将诸药为细末，同和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
