序号：84444
名称：七制香附丸
组成：香附米14两（分作7分，酒、醋、盐、童便、小茴香2两，益智仁2两，莱菔子2两，凡浸，春、秋3日，夏1日，冬7日，同入砂锅内，用艾叶4两，无灰酒随煮随添，以黑色为度，制香附7两），归身4两（酒洗），熟地4两（姜汁焙），生地4两（姜汁焙），白芍4两（酒炒），抚芎3两，人参1两，白术（土炒）2两，白茯苓2两，枣仁2两（炒），炙甘草9钱，天冬2两9钱，益母草4两，条芩（酒炒）2两5钱，砂仁（炒）1两5钱，阿胶2两（炒），陈皮2两，山茱萸（酒蒸）2两，元胡索1两5钱（醋炒）。
出处：《奇方类编》卷下。
主治：妇人经脉不调。妇人郁怒伤肝，思虑伤脾，肢体困倦，面目枯黄，日晡潮热，夜静昼烦，胸膈(月真)胀，腰胁疼痛，饮食无味，神识不安，赤白带下，如是等情，渐致经水不调，或致半产漏下，久而不孕，亦有成劳；亦治山岚异气，老幼水土不服。
加减：None
功效：调经、保元理气，却病延年。
用法用量：每日空心服100丸。
制备方法：上为细末，用神曲四两，酒煮神曲糊为丸，如梧桐子大。
临床应用：None
各家论述：《卫生鸿宝》：香附为主，辛温能达各经，醋浸开气中之郁，消血中之滞；盐水浸，清坎中之阳；小茴香入水同炒干，以补腰滋肾；童便浸，滋离中之阳；益智仁入水同浸炒，培脾补肾强志；萝卜子入水同浸炒，化滞开胃；酒浸，通十二经络。当归、熟地、川芎、白芍、人参、白术、茯苓、炙甘草、枣仁、天冬为臣。益母草、山萸肉、陈皮为佐。加条芩清血热，平肝热，去膈热，解心热，泻肺热；砂仁保安胎产；炼蜜润肺滋阴。其性清上达下，导滞和中。早，白汤下，清上焦之营；晚，温酒送，养下焦之血；或用清米汤，则补肺健脾；或用桂圆汁，则养心和血。修合不易，气味和平，血病用之效，气病服之灵，不但无孕者能孕，即有孕者，可以却病延年也。
用药禁忌：None
