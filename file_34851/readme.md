序号：34861
名称：麦门冬散
组成：麦门冬（去心，焙）1两，防风（去叉）1两，玄参1两，地骨皮1两，远志（去心）1两，大黄（锉，炒）1两，车前子1两，茺蔚子1两，决明子（炒）1两，蔓荆实（去白皮）1两，细辛（去苗叶）1两，黄芩（去黑心）1两，黄连（去须）1两，犀角屑1两，甘草（炙）1两。
出处：《圣济总录》（人卫本）卷一○三。
主治：眼赤痛，生障翳，乍差乍发，多泪羞明，隐涩肿痒，心神烦躁。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，煎至7分，去滓，食后温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，原书（文瑞楼本）作“麦门冬汤”。
