序号：77380
名称：遇仙膏
组成：当归4两，白芷4两，两头尖4两，穿山甲25个，巴豆（研）120粒，蓖麻子120粒（研），土鳖21个（去壳），麻油1斤，黄丹10两（水飞，炒），乳香4钱，没药4钱，轻粉4钱，血竭4钱，麝香4钱。
出处：《疡科选粹》卷八。
主治：无名肿毒，痈疽，发背，痞块，疮疡，痢疾及二便秘结、咳喘。
加减：None
功效：None
用法用量：贴用火烘手，熨膏药上100余手，出汗妙。若痢疾及二便秘结，贴脐中；咳喘，贴肺俞穴。
制备方法：上两头尖等俱锉，入香油1斤内浸，春5、夏3、秋7、冬10日，入锅内熬白芷焦色，将锅取下温冷，用生绢滤去滓，再文武火熬，下黄丹，用桃、柳枝不住手搅，滴水不散，不老不嫩，入松香5两，搅匀，取下锅冷，乃下轻粉、麝香、血竭、乳香、没药搅匀用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
