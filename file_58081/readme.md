序号：58096
名称：益寿黑豆方
组成：旱莲草5钱，黑桑椹4钱，白何首乌3钱，故纸6钱，骨碎补6钱，金樱子4钱，何首乌3钱，杜仲4钱，明青盐2两，生地6钱，白茯苓6钱，柏子仁4钱，蛇床子3钱，肉苁蓉4钱，菟丝饼6钱，甘枸杞5钱，川续断4钱，川牛膝6钱，槐角子2两，远志肉6钱，川石斛4钱。
出处：《集验良方》卷二。
主治：None
加减：None
功效：补益。
用法用量：每服4-5钱，清晨空心用白滚水送下。
制备方法：腰子雄黑豆20碗，用水30碗，将前药煮至20碗，盛于净器，入黑豆在内浸过1宿，俟黑豆吃干药水，取起风干，用柳木甑子蒸透，即起风干，再将药渣用水20碗煎至15碗，取起药渣，仍入豆再浸1宿，候豆又吃干药水，取起再入甑蒸透，要不见火，不见日，风干，以新瓷罐收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
