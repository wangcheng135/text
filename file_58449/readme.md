序号：58464
名称：破壁擒王饮
组成：大黄4两，黄芩5钱，连翘4钱，防风3钱，黄连2两，莪术3钱，栀子5钱，荆芥3钱，黄柏5钱，槟榔3钱，薄荷2钱，桔梗1钱，前仁5钱，甘草1钱。
出处：《喉科种福》卷三。
主治：瘟疫黄喉。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：黄疫喉邑庠生某故弟，蔚南肄业师也。一日，弟过其馆，以喉初起微痛；令诊视，曰：得毋黄疫喉乎？曰：然。以破壁擒王饮与之，大黄、黄连各4两，3服不下，弟邀余同往视之，服大黄、黄连各6两，始下而愈。弟又治一老妇，年70，骨瘦如柴，染疫黄喉，亦投本方，大黄、黄连各重至6两始痊。
各家论述：破壁擒王一方，为专治黄疫喉之专剂而最峻者。以三黄攻毒；以荆、防、薄、翘解散其党；以莪术、槟榔作先锋，领猛将健卒直捣巢穴；以山栀泻三焦之热，桔梗载其上浮，车前引其下行，三路合兵，俾无隙漏；以甘草调和诸将，协力同心，以成大功。
用药禁忌：None
