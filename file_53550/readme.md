序号：53565
名称：哽生散
组成：丁香5钱，木香2钱5分，官桂心2分半，人参2分半，神曲2分半，川芎7钱5分，诃子7个（去核），巴豆14粒（水浸1宿如泥），草果仁2个，广术7分半。
出处：《普济方》卷三十六。
主治：反胃吐食。
加减：None
功效：None
用法用量：每服用1铜钱半子，令病人男左女右置药，米汤调成稀糊，以舌舐之，卧少时后，用细米粥汤压之，病人多年可服2服，病浅1服。
制备方法：上除巴豆外，并生用，为极细末，入巴豆泥和匀，瓷器内盛贮，密封之。
临床应用：None
各家论述：None
用药禁忌：忌生冷、硬物、油腻、湿面、盐醋等物，只服白粥百日。孕妇不可服。
附注：此药过腊月半辰时不见火方合药，于来年夏至后方可用。
