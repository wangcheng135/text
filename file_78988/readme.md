序号：79004
名称：增损黑锡丹
组成：黑锡丹头2两，川楝子半两，阳起石半两，木香半两，沉香半两，青皮（炒）半两，白豆蔻1两，茴香1两，官桂（去粗皮，不见火）1两，绵附（炮，去皮脐）1两，葫芦巴1两，破故纸（炒）1两，乌药（去木，锉）1分，磁石（火煅、醋淬7次，细研水飞）。
出处：《得效》卷三。
主治：下虚，阴阳不升降，上热下寒，头目眩晕，病至危笃，或服暖药上僭愈甚者。
加减：None
功效：None
用法用量：每服50丸，加至70丸，浓煎人参、茯苓、姜、枣汤空心吞下。
制备方法：上为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中磁石用量原缺。
