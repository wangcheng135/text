名称：万灵膏
组成：玄参1两，苦参1两，生地黄1两，黄连1两，黄芩1两，山栀1两，大黄1两，当归1两，川芎1两，白芷1两，赤芍1两，羌活1两，独活1两，防风1两，连翘1两，花粉1两，桔梗1两，五倍子1两，皂角1两，白及1两，白蔹1两，山慈姑1两，红大戟1两，官桂1两，蓖麻仁1两，木鳖仁1两，巴仁1两，山甲1两，杏仁1两，发团1两，槐枝8两，柳枝8两，桃枝8两，马齿苋8两（一方无百草霜，有两头尖5钱）。
