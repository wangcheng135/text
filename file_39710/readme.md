序号：39720
名称：经进骨碎补丸
组成：五灵脂（拣去砂石，别研极细）1两，木鳖子（去壳，锉碎，研细入众药用）1两，生姜（切作片子）1两，地龙（用麻包定石上捶去土）1两，黑牵牛（炒令微熟）1两，草乌（去皮尖，锉作块子，用盐炒令里边俱透）1两，白胶香（韭煮30-20沸，倾在冷水中，少时挖干，别研极细）1两，黑豆（炒令微熟）1两，威灵仙（用软脚，者1两，洗去土）1两，滴乳香（用糯米数粒，同乳香一处，研细入众药用）1两，天南星（慢灰火中炮去皮用）1两，麝香1钱（别研，临了将入药），防风（去苗并芦头）。
出处：《普济方》卷九十三引《卫生家宝》。
主治：左瘫右痪。
加减：None
功效：None
用法用量：初服5丸，加至7丸，食后薄荷茶酒送下。
制备方法：上为末，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
