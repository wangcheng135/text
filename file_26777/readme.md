序号：26785
名称：延龄种子仙方
组成：当归身（酒浸）4两，川牛膝（酒浸）4两，生地黄（酒浸）4两，熟地黄（酒浸）4两，片芩（酒浸）4两，麦门冬（去心，米泔水浸）4两，天门冬（去心，米泔水浸）4两，山茱萸4两，知母4两（盐酒各浸2两），黄柏（去皮）9两（蜜水、盐、酒各浸3两），辽五味2两，川芎2两，山药2两，龟版（酥炙）2两，白芍药（酒浸）2两，人参6钱。
出处：《济阳纲目》卷六十七。
主治：None
加减：None
功效：延龄种子。
用法用量：每服100丸，清晨盐汤，晚酒送下。
制备方法：上制如法，晒干，不犯铁器，为极细末，用白蜜3斤，不见火炼，将竹筒二节凿一窍孔，去瓤，入蜜在内，并入清水1小盏和匀，绵纸封固七层，竖立重汤锅内，柴火煮一昼夜，和药为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：男妇皆然，以服药之日为始，忌房事一个月，愈久愈妙。
附注：None
