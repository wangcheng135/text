序号：33535
名称：灵丹
组成：防风2钱5分，北细辛2钱5分，黄芩2钱5分，石膏2钱5分，元参2钱5分，羌活2钱5分，荆芥2钱5分，小生地2钱5分，连翘2钱5分，黄柏2钱5分，甘草2钱5分，白芷2钱5分，白菊花2钱5分，栀仁2钱5分，川芎2钱5分，百部2钱5分，薄荷2钱5分，真黄连3钱。
出处：《玉钥》卷上。
主治：一切牙痛。
加减：None
功效：None
用法用量：凡牙疼擦上，立止。
制备方法：上为粗末，置大铜锅内，外用甘草5钱，煎水一大碗，将药拌匀，再取潮脑3两，研碎，分作5-7次，洒药上，再以大碗盖住药，又用石膏和灰面、盐水调匀，密糊碗口，不可泄气，煮长香一柱时，方可起下，将上升在碗内的灵丹用竹刀刮下，仍将滓用甘草水拌匀，复洒潮脑于上，如此升取5-7次，候药性升尽为度，再以瓷瓶收固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
