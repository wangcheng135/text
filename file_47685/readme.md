序号：47700
名称：卫生宝
组成：天竺黄3钱半，钩藤3钱半，千金霜3钱半，琥珀1钱2分，麝香3分，僵蚕3钱半，重楼8钱4分，雄黄2钱半，牛黄8分，茅慈姑7钱1分，朱砂2钱半，文蛤1两1钱，江珠1钱半，大戟1两，红人参1钱半。
出处：《全国中药成药处方集》（抚顺方）。
主治：急痫高热，神昏痉搐项强，角弓反长，嗜眠昏睡；猩红热，斑疹伤寒，麻疹壮热烦渴，神昏谵语，狂躁干渴；急性胃肠炎，吐泻，并治疹后肠炎，毒泻毒痢：疔毒恶疮，毒火内攻，乍寒乍热，搅乱昏迷；咽喉肿痛，咽下困难。
加减：None
功效：消炎退热，镇静解毒。
用法用量：每服1丸，小儿服1小丸，白水送下。
制备方法：上为细末，炼蜜为丸，7分或3分重。上为细末，炼蜜为丸，7分或3分重。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服；四肢厥冷，体温低降之藿乱（虎列拉）、白喉禁用。
附注：本方改为散剂，名“卫生散”（见原书沈阳方）。
