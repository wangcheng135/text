序号：60354
名称：通玄二八丹
组成：黄连8两（去毛，雅州者），当归（酒浸）5钱，生地黄（酒浸）5钱，白芍药5钱，乌梅肉5钱。
出处：《扶寿精方》。
主治：积聚，泄痢肠滑，肠风下血。
加减：None
功效：能通能塞。
用法用量：每服70丸，治积聚，空心姜汤送服，泻1-2次即愈，用粥补；治泄痢，饭后茶汤送服，即止；若肠滑、肠风下血，可常服。
制备方法：上为细末，以雄猪肚1个，盐醋洗去秽气，煮将熟，取控干水，入药在内，置甑中，上下韭菜厚铺，自辰至酉，慢火蒸之，以银簪插试有黄色为度，乘热为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：通元二八丹（《济阳刚目》卷二十二）。
