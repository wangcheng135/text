序号：50008
名称：茯神丸
组成：茯神1两，鳖甲（醋炙黄色，去裙襕）1两，荗香1两，川大黄（锉碎，醋和炒）1两，川乌头（炮裂，去皮脐）1两，芫花（醋拌，炒令干）1两，京三棱（锉，拌醋微炒）1两，桂心1两，远志（去心）1两，干漆（杵碎，炒令烟出）1两，硇砂1两半（不夹石者，细研），蓬莪术3分。
出处：《医方类聚》卷&mdash;&mdash;一引《神巧万全方》。
主治：伏梁气结，固在心下，横大如臂，饮食渐少，肢体消瘦。
加减：None
功效：None
用法用量：每服7丸，渐加至10丸，空心酒送下。以取下积滞物为度，隔2日再服。
制备方法：上为细末，先以米醋3升，熬令稍调，入少面作糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
