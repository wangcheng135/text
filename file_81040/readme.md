序号：81056
名称：麝香膏
组成：麝香1两（细研），叶子雌半两（细研），龙脑半两（细研），麒麟竭2分（末），没药半两（末），槟榔半两（末），丁香半两（末），当归3分（末），木香半两（末），黄犬脂1两，朱砂3分（细研），白蜡3分，黄丹3两，油8两。
出处：《圣惠》卷六十三。
主治：一切痈疽发背，及风热毒结肿疼痛。
加减：None
功效：None
用法用量：用蜡纸上摊贴，每日2换，以愈为度。
制备方法：先将油于银锅中以慢火炼令香，下蜡，犬脂，去火，渐下黄丹，却用火煎，不住手以柳木篦搅，变色即去火，将前六味药末，并香药1处更研令匀，微火暖动，渐渐搅入令匀，膏成，以瓷盒盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
