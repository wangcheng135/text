序号：34302
名称：万灵筋骨膏
组成：大黄3钱5分，槟榔3钱5分五倍子3钱5分，香附3钱5分，山甲3钱5分，全蝎3钱5分，羌活3钱5分，防风3钱5分，杏仁3钱5分，芫花3钱5分细辛3钱5分，牵牛子3钱5分，土鳖虫3钱5分，厚朴3钱5分，甘遂3钱5分，木鳖子5钱，三棱5钱，莪术5钱，川乌（生）5钱，天麻子5钱，生地5钱，草乌（生）5钱，独活4钱，猪牙皂角4钱，黄柏4钱，肉桂4钱，大戟4钱，枳壳4钱，麻黄4钱，巴豆4钱，当归7钱5分，玄参（去芦）1钱，黄连2钱，柳枝8两，蕲蛇4两，蜈蚣2钱。
出处：《北京市中药成方选集》。
主治：受风受寒，腰腿疼痛，筋骨麻木，痞满腹胀。
加减：None
功效：散风活血，舒筋定痛。
用法用量：微火化开，贴患处。
制备方法：上药酌予碎断，用香油上160两炸枯，去滓过滤，炼至滴水成珠，入黄丹80两成膏，取出，放入冷水中去火毒后，再加热熔化摊贴，每张油重量1两1钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
