序号：44789
名称：急救丹
组成：苍术（炒）10两，橘皮8两，五加皮8两，厚朴（炙）8两，闹羊花8两，茯苓16两，槟榔10两，细辛4两，百草霜4两，猪牙皂12两，藿香12两，灯心草炭16两（共研为细粉，过罗），雄黄粉4两，朱砂粉4两，冰片4两，麝香4两，牛黄3两2钱。
出处：《北京市中药成方选集》。
主治：中寒中暑，感冒秽浊，昏迷气闭，四肢厥冷，呕吐恶心，腹痛作泄。
加减：None
功效：祛暑解毒，通关开窍。避瘴气，解浊秽。
用法用量：每服2分，温开水送下。外用闻入鼻内少许。
制备方法：上为细末，都拌匀，瓶装重2分。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
