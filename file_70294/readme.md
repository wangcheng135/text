序号：70310
名称：煮酒方
组成：当归3钱6分，川芎1钱8分，苍术6钱，白术9钱，白芍药1钱5分，枳壳9分，生地黄1钱2分，半夏1钱5分，甘草9分，乌药1钱5分，麻黄3钱，茯苓2钱半，防己1钱8分，陈皮1钱8分，木通1钱8分，薏苡仁1钱8分，五加皮1钱半，木香6分，桑寄生1钱2分，黄芩1钱8分，羌活1钱2分，独活1钱2分，牛膝1钱8分，木瓜1钱8分，海风藤3钱，苏叶2钱4分。
出处：《直指》卷四。
主治：肿风及白虎历节疼痛。
加减：None
功效：None
用法用量：None
制备方法：上为粗末，用好酒1坛，将药用绢袋盛，悬于坛口下角，用文武火煮1-2时辰，取出，放于湿泥地出火毒，住2-3日再服。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
