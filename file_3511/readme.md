序号：3511
名称：太极光
组成：孔雀尾4钱（甘草水洗，撮土搓之，复用水洗净晒干，为末纯用，尾端圆处更胜），乳香2钱，没药（各去油，净）2钱，蜈蚣2钱，全蝎2钱，磁石（火煅）2钱，麝香2钱，蝼蛄（晒干）2钱，雄黄（醋浸透，换白萝卜汁煮用）2钱，朱砂2钱，水银5钱，牙硝1两2钱5分，硫黄2两5钱。
出处：《经目屡验良方》。
主治：男女大小百病，恶疮肿毒，筋骨疼痛，左瘫右痪。
加减：None
功效：None
用法用量：每症各取药置患处，以火焠着灸之。灸时要避风，如遍身风气痛，则置药于各处骨节间温灸之，重症灸后须避风7日。
制备方法：上为末，用瓷碗1只，将药末每钱许，匙挑入碗内，以灯刀炒如米粒大小不等，勿令焦枯，收入瓷瓶听用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
