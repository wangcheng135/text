序号：28361
名称：芩连芍药汤
组成：黄蕉（炒）5分，黄连（炒）5分，白芍药（炒）2钱，枳壳（炒）5分，木香5分，槟榔1钱，甘草（炙）3分。
出处：方出《明医杂著》卷二，名见《医统》卷三十六。
主治：痢疾。
加减：腹痛，加当归1钱5分，砂仁各1钱，再加木香、芍药各5分；后重，加滑石（炒）5分，枳壳、槟榔、芍药、条芩各5分；白痢，加白术、白茯苓、滑石（炒）、陈皮各1钱，初欲下之，再加大黄5钱；红痢，加川芎、当归、桃仁各1钱5分，初欲下之，再加大黄5钱；红白相杂，加川芎、当归、桃仁各1钱5分以理血，滑石、苍术、陈皮各1钱5分以理气；食枳，加山楂、枳实以消导；白痢久，胃弱气虚，或下后未愈，去槟榔、枳壳，减芩、连、芍药各7分，加白术1钱5分，黄耆、陈皮、茯苓各1钱，缩砂、干姜（炙）各5分；红痢久，胃弱血虚，或下后未愈，减黄芩、黄连各5分，加当归、川芎、熟地、阿胶、木香、陈皮各1钱，白术1钱5分；赤黑相杂，此湿胜也，及小便赤涩短少，加木通、泽泻、茯苓各1钱，山栀仁（炒）5分，以分利之。血痢，加当归、川芎、生地黄、桃仁、槐花（炒）各1钱，久不愈，减芩、连各7分，去槟榔、枳壳，再加阿胶珠、侧柏叶、白术各1钱5分，干姜（炒黑）、陈皮各1钱；痢已久，而后重不去，此大肠坠下，去槟榔、枳壳，用条芩，加升麻1钱以升提之；呕吐食不得下，加软石膏1钱5分，陈皮1钱，山栀仁（炒）5分，生姜6分，缓呷之，以泻胃口之热；得痢而误服温热止涩之药，则虽稍久，亦宜用前法以下之，下后方调之。
功效：泻肠胃之湿热，开郁结之气，消化积滞，通因通用。
用法用量：加生姜，水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：《医统》本方用黄芩（炒）、黄连（炒）各一钱，白芍药一钱半，枳壳二钱，木香、槟榔各一钱，甘草五分；水1盏半，加灯心、大枣，煎八分，食前服。
