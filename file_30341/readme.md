序号：30349
名称：还元丹
组成：何首乌1斤（鲜者，只用竹刀刮去皮；干者，米泔水浸软刮皮。四制，忌铁，砂锅或瓦器盛酒，生脂麻蒸1次，晒干；羊肉1斤蒸1次，晒干；酒拌蒸1次，晒干；黑豆蒸1次，晒干。一方黑羊肉1斤、黑豆3合，量用水，上加竹炊笓置药，盖蒸熟透，晒干），生地黄3两，熟地黄3两（酒浸，焙干，各取末1两），天门冬4两，麦门冬4两（米泔水浸，去心，各取末1两），人参1两（取末5钱），白茯苓（去皮）3两（打成块，酒浸，晒干），地骨皮3两（童便浸，晒干，各取末1两，忌铁）。
出处：《扶寿精方》。
主治：诸虚。
加减：None
功效：千益百补。延年益寿。
用法用量：每服1-2匙，沸汤温漱，不拘时候。如首生乳难得，但凡人乳皆可。
制备方法：上取首生男孩乳汁6两，白蜜10两，炼同一器中，合前末为膏，瓷器贮，勿泄气。
临床应用：None
各家论述：None
用药禁忌：此方为阴虚血热者宜之，诸阳虚者不可用。
附注：不老丹（《医部全录》卷三三一引《体仁汇编》）、延年益寿不老丹（《摄生众妙方》卷二）、延年益嗣丹（《摄生众妙方》卷十一）、延寿不老丹（《医学入门》卷七）、延年益寿丹（《饲鹤亭集方》）。
