序号：78500
名称：獭肝丸
组成：獭肝1两（微炙），麝香1分（细研），犀角屑半两，鳖甲1两（涂酥，炙微黄，去裙襕），天灵盖3分（涂酥，炙令微黄），阿魏1两（麸裹煨，面熟为度），牛黄1分（细研），雄黄3分（细研），木香半两，龙胆半两（去芦头），胡黄连3分，知母3分，柴胡1两（去苗），地骨皮3分，赤芍药半分，麦门冬1两半（去心，焙），甘草半两（生用），白术半两，黄芩半两，赤茯苓2分，川升麻半两，朱砂3分（细研），槟榔3分。
出处：《圣惠》卷三十一。
主治：骨蒸劳热，体瘦烦疼。
加减：None
功效：None
用法用量：每服30丸，以清粥饮送下，不拘时候。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌苋菜。
附注：None
