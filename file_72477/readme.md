序号：72493
名称：犀角丸
组成：犀角屑半两，玄参3分，苦参3分，丹参半两，沙参半两（去芦头），甘菊花3分，旋覆花半两，车前子1两，槟榔3分，牵牛子1两半（微炒），杏仁1两（汤浸，去皮尖双仁，麸炒微黄），川大黄1两（锉碎，微炒），前胡3分（去芦头），黄柏根皮1两（微炒），知母半两，白鲜皮3分，槐子1两（微炒），赤芍药1两，芎3分。
出处：《圣惠》卷三十二。
主治：肝脏壅毒，眼目昏暗，热泪出不止。
加减：None
功效：None
用法用量：每服20丸，食后煎淡竹叶汤送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
