序号：23071
名称：芎犀丸
组成：石膏（细研）4两，生龙脑（别研）、朱砂（研，飞）4两（留1两为衣），生犀角1两，人参（去芦）2两，茯苓（去皮）2两，川芎4两，阿胶（碎，炒）1两半，细辛（去苗）2两，麦门冬（去心）3两，甘草（炙）2两，山栀子（去皮）1两。
出处：《得效》卷十。
主治：偏头疼，一边鼻塞不闻香臭，常流清涕，或作臭气一阵，加芎、蝎等遍服无效者。
加减：None
功效：None
用法用量：每服1-2丸，食后细嚼、茶、酒任下。服此不10数次，或作嚏，突出一铤稠脓，即愈。
制备方法：上除别研药后入，并为末，炼蜜为丸。
临床应用：None
各家论述：《金匮翼》：此方兼祛风清热之长，而得参、胶等安定气血，虽虚人亦可用之，安内攘外，并行不悖也。
用药禁忌：None
附注：本方改为散剂，名“芎犀散”（见《中国医学大辞典》）。
