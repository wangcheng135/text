序号：75901
名称：谢传万病无忧散
组成：草果1两2钱，黄连1两2钱，滑石1两2钱，泽泻1两2钱，枳壳8钱，木通8钱，厚朴8钱，陈皮8钱，赤茯苓8钱，车前子8钱，猪苓8钱，砂仁8钱，香薷2两，白扁豆2两，白术5钱6分，小茴香5钱6分，木香2钱半，甘草2钱半。
出处：《医学入门》卷七。
主治：夏月霍乱吐泻，烦渴尿赤，似疟非疟，似痢非痢，不服水土。
加减：None
功效：常服可防疟痢。
用法用量：每用2钱，滚水调服；素虚者，温酒或茶清调下，如不善服末者，煎三沸服，或摊冷服。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：忌米饮，孕妇忌服。
附注：本方方名，《东医宝鉴·杂病篇》引作“万病无忧散”。
