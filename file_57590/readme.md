序号：57605
名称：大疳丸
组成：白矾（枯）1两，绿矾1两，胆矾2钱，干虾蟆1枚（去肠肚，炙），莨菪子1分，葶苈（炒）1分，蜗牛4枚（以上7味入在瓶内，盐泥固济，候干，火烧通赤，取出研为细末），胡黄连1分，生蜗牛（研）3枚，麝香（研）1钱，丹砂（研）1钱，雄黄（研）1钱，牛黄（研）1钱，熊胆（研）2钱，诃黎勒皮2钱，细辛（去苗叶）2钱，青黛半钱。
出处：《圣济总录》卷一七三。
主治：小儿疳痢羸瘦。
加减：None
功效：None
用法用量：每服3丸至5丸，薄荷汤送下，不拘时候。
制备方法：上为末，和令匀，烧粟米饭为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
