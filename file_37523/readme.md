序号：37533
名称：抱龙丸
组成：琥珀5钱，辰砂3钱，雄黄7钱（香麻油煎12时，再用水萝卜汁煮），胆星2两1钱，僵蚕4钱（炒，去嘴足），全蝎3钱（研末，用石榴1枚剜空，以无灰酒调末填入盖定，坐文火上徐徐搅动成膏，取出冷用），牛黄1钱，麝香5分，天竺黄7钱，赤茯苓1两。
出处：《古方选注》卷下。
主治：肝惊魂升，搐搦不语。
加减：None
功效：熄风化痰，镇惊发音，保守肝魂。
用法用量：灯心、薄荷汤送下。
制备方法：上药各为末，蒸饼为丸，金箔为衣。
临床应用：None
各家论述：僵蚕、全蝎、薄荷辛以散肝风，天竺黄、胆星苦辛以泄风痰，辰砂、琥珀安五脏、镇魂魄，雄黄搜肝胆之伏风，麝香利骨髓之伏痰，金箔佐辰砂可镇胎惊，竺黄佐牛黄可发音声，赤苓、灯心可止夜啼。集肝经之药复方，初无深义。一方加人参二钱五分，培植正气以御肝风；草河车三钱五分（即蚤休草，切片，黑稆豆制），能伏牛黄、丹砂之毒，并可治惊祛风，二味却有妙义，当篡入方中。
用药禁忌：None
附注：按：本方加人参、草河车，名“人参抱龙丸”（见《饲鹤亭集方》。
