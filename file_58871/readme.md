出处：《直指附遗》卷二十二。
主治：一切肿毒及杨梅痈漏，恶疮，风气骨节疼痛，痞气积块，坐闪腰痛。
加减：None
功效：消肿毒，去腐生肌。
用法用量：油纸摊贴。
制备方法：上(口父)咀，用真香油10斤和药，浸7日，下锅熬，待药滓成炭、血余无形方可滤去药滓，再熬，滴水成珠，再将黄丹徐徐入内收为膏，再入后项药：乳香、血竭、阿魏、龙骨、胆矾、雄黄、轻粉、没药、孩儿茶各5钱，樟脑4钱，赤石脂7钱，沉香、木香各3钱，麝香1钱，冰片3分。上为极细末，入膏内搅匀，用瓷钵收贮，出火毒。
