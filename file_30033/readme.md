序号：30041
名称：赤金锭
组成：麻黄4钱5分，紫苏7钱5分，山茨菇2两5钱，五倍子2两5钱，香附子2两5钱，苍术1两5钱，半夏1两5钱，木香1两5钱，山豆根1两5钱，丹参6钱，鬼箭羽6钱，辰砂1两，千金子1两2钱，红芽大戟1两2钱，雌黄1两2钱，细辛1两2钱，川乌1两2钱，滑石1两2钱，麝香3钱。
出处：《仙拈集》卷四。
主治：天行时疫，瘟疫，伤寒，狂言乱语，霍乱，绞肠痧，腹痛，食物中毒，小儿急慢惊风，中风，中气，口眼斜，牙关紧急，筋脉拘急，妇人月水不调，腹中结块，男妇头晕，传尸劳瘵，自缢，落水，中恶，赤痢，白痢，汤火伤，蛇、蜈、蝎咬伤。
加减：None
功效：解百毒，治百病。
用法用量：天行时疫用绛囊盛之，悬之当胸，则不传染。瘟疫、伤寒、狂言乱语、霍乱、绞肠痧、腹痛、饮食中毒、小儿急慢惊风，俱薄荷汤磨服；中风、中气、口眼斜、牙关紧急、筋脉拘挛、妇人月水不调、腹中结块、男妇头晕，俱温酒磨服；传尸劳瘵、自缢、落水、中恶，俱冷水磨服；赤痢，凉水下；白痢，姜汤下；汤火伤，蛇、蜈、蝎咬伤，俱用酒磨服，水磨涂患处。
制备方法：依法炮制，净末称准，以糯米粉糊和之，石臼杵千下，用范子印成锭角，重1钱，作3次用之，阴干，万勿火烘。
临床应用：None
各家论述：None
用药禁忌：肿毒恶疮已溃者不宜服。
附注：None
