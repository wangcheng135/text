序号：42058
名称：三黄汤
组成：黄连钱半，黄芩钱半，黄柏钱半，山栀钱半（炒），玄参钱半，知母钱半，石膏5钱，甘草5分，淡豉钱半。
出处：《医略六书》卷十八。
主治：膏粱积热，三焦见诸火证，脉洪数者。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：膏粱积热，蕴蓄于中，不得舒泄，而弥漫三焦，故发火证。黄连清心、脾之火，黄芩清肺、肠之火，黄柏清肾、膀胱之火，栀子清三焦之火，玄参清浮游之火，知母清肠胃之火，石膏清胃泻热，甘草缓中和药，佐淡豉以发泄热邪也。使热化火降，则三焦肃清而经络宣通，安有火发之患哉！此清火疏热之剂，为三焦热盛之专方。
用药禁忌：None
附注：None
