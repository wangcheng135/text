序号：58573
名称：秘传牛黄清心丸
组成：天麻4两，防风2两（去芦），牛胆南星2两半，僵蚕（炒）2两半，全蝎2两半，白附子（生用）、干天罗（即丝瓜）5钱，川乌5钱，远志（去心）2两，川山甲（蛤粉炒）3两，蝉退2两（去土），蒿虫（不拘多少）、辰砂（天葵煮）1两，雄黄1两（2味另研），犀角（镑细）5钱，蜈蚣3钱，蟾酥5分（另研），沉香3钱，细辛5钱，龙齿5钱，琥珀2钱（另研），珍珠3钱（另研），天竺黄3钱，蛤蚧1对，金箔10帖，银箔10帖。
出处：《医便》卷四。
主治：小儿惊风，大人中风、中痰、中气，一切风痰。
加减：None
功效：None
用法用量：随症调引用。
制备方法：上药各制净，为末；外用荆芥1斤，麻黄1斤，木通1斤，皂角半斤，甘草4两，苍耳子4两，6味熬膏，入真酥合油，炼蜜为丸，如芡实大，金银箔为衣，蜡封。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
