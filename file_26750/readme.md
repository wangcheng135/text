序号：26758
名称：异方神仙济阴丹
组成：香附子1两，三棱1两，蓬术1两，陈皮1两，青皮1两，败姜1两（上6味，用黑豆半升、米醋5升同煮，豆烂取出焙干，留余醋打糊），官桂1两，当归1两，赤芍药1两，生地黄1两，熟地黄1两，泽泻1两，片姜黄1两，丹皮1两，干姜1两，川芎1两，刘寄奴1两，泽兰1两，人参1两，蒲黄（纸炒）1两，木香1两，白术1两，玄胡索1两（焙干）。
出处：《袖珍》卷四。
主治：妇人经事不行，及心脾疼，腰腹疼痛。
加减：None
功效：None
用法用量：每服50丸，空心、食前炒姜酒送下，艾醋汤亦得，经事不行，酒送下；心脾疼，姜汤送下，或酒送下亦妙。
制备方法：上为末，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：异方济阴丹（《丹溪心法附余》卷二十）。
