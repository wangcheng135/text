序号：22633
名称：吴茱萸丸
组成：茱萸1两（去枝梗，煮少时，浸半日，晒干），陈皮1两，苍术（米泔浸）1两，黄连2两（陈壁土炒，去土秤），黄芩1两（如上土炒）（或加桔梗1两、茯苓1两）。
出处：方出《丹溪心法》卷三，名见《杏苑》卷四。
主治：吞酸，因于湿热郁积于肝而出，伏于肺胃之间者。
加减：None
功效：疏郁滞，清湿热。
用法用量：每服20-30丸，食后服。
制备方法：上为末，神曲糊为丸，如绿豆大。
临床应用：None
各家论述：《杏苑》：用吴茱萸、橘红等诸辛以疏郁，苍术燥湿，苓、连等以清热。
用药禁忌：宜用粝食、蔬菜自养。
附注：按：《杏苑》本方用法：每服30-50丸，生姜汤送下，每日3次。
