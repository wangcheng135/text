序号：22550
名称：吴茱萸丸
组成：吴茱萸（汤洗，焙炒）1两，细辛（去苗叶）半两，芍药半两，柴胡（去苗）半两，旋复花半两，黄芩（去黑心）半两，紫菀（去苗土）半两，人参半两，白术半两，白茯苓（去黑皮）半两，干姜（炮）半两，桂（去粗皮）半两，附子（炮裂，去皮脐）半两，甘草（炙，锉）半两，半夏（汤洗7遍去滑）半两，当归（切，焙）半两。
出处：《圣济总录》卷九十四。
主治：寒疝心腹痛，或逆抢心，烦满不得卧，恶风惊惕不食，变发寒热。
加减：None
功效：None
用法用量：每服20-30丸，温酒送下，不拘时候。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
