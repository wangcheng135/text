名称：蠲毒换肌饮
组成：冷饭团（白色者，木槌打碎）4两（以长流水4大碗，入砂锅内煎至3碗，入后药），黄瓜蒌（连仁杆烂或细切）1个，黄耆（盐水炒）3钱，白芍药1钱半，当归1钱半，木瓜1钱，白芷1钱，风藤1钱，白鲜皮1钱，贝母1钱，天花粉1钱，穿山甲1钱，皂角刺1钱，甘草节1钱，汉防己7分，鳖虱、胡麻（炒研）2钱，金银花3钱，猪胰子（切碎）1两。
出处：《准绳，疡医》卷五。
主治：杨梅疮。
加减：None
功效：None
用法用量：上药再煎至1大碗，通口顿服；胃弱者分为2服，每日3次。
