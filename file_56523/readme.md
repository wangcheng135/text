序号：56538
名称：大顺散
组成：甘草（锉寸长）30斤，干姜4斤，杏仁（去皮尖，炒）4斤，肉桂（去粗皮，炙）4斤。
出处：《局方》卷二。
主治：冒暑伏热，引饮过多，脾胃受湿，水谷不分，清浊相干，阴阳气逆，霍乱呕吐；脏腑冷热不调，泄泻多渴，心腹烦闷，痢下赤白，腹痛后重；中阴暑，食少体倦，发热作渴，腹痛吐泻，脉沉微者。
加减：None
功效：温中散暑。
用法用量：每服2钱，水1中盏，煎至7分，去滓温服。如烦躁，井花水调下，不拘时候；以沸汤点服亦得。
制备方法：上先将甘草用白沙炒及8分黄熟，次入干姜同炒，令姜裂，再入杏仁又同炒，候杏仁不作声为度，用筛隔净，后入桂，一处捣罗为散。
临床应用：None
各家论述：1.《医方集解》：脾胃者，喜燥而恶湿，喜温而恶寒，干姜、肉桂散寒燥湿，杏仁、甘草利气调脾，皆辛甘发散之药，升伏阳于阴中，亦从治之法也。如伤暑无寒证者，不可执泥。_x000D_
2.《古方选注》：《局方》祖仲景大青龙汤，以肉桂易桂枝，而变为里法。病由暑湿伤脾也，故先将甘草，干姜同炒，辛甘化阳以快脾欲；再入杏仁同炒，利肺气以安吐逆；白沙，本草主治绞肠痧痛，用之拌炒，以燥脾湿；复以肉桂为散，俾芳香入阴，升发阳气以交中焦，去脾之湿，湿去而阳气得升，三焦之气皆顺，故曰大顺。
用药禁忌：None
附注：二宜汤（原书卷十）。①《景岳全书》：此方加附子，即名“附子大顺散”。②本方改为丸剂，名“杏仁丸”（见《普济方》）。
