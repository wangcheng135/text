名称：天雄散
组成：天雄（炮裂，去皮脐）1两，附子（炮裂，去皮脐）1两，五味子（炮）1两，白术1两，人参1两，白芷1两，细辛（去苗叶）1两，乌头（炮裂，去皮脐）3分，柴胡（去苗）3分，麦门冬（去心，焙）3分，干姜（炮）3分，麻黄（去根节）半两，山茱萸半两，蜀椒（去目并闭口，炒出汗）半两，桔梗（锉，炒）半两，当归（切，焙）1两半，防风（去叉）2两。
出处：《圣济总录》卷一六二。
主治：产后中风偏枯，手足不遂，痿弱无力。
加减：None
功效：None
用法用量：每服2钱匕，温酒调下，不拘时候。
制备方法：上为散。
