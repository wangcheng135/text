序号：64125
名称：猪肝丸
组成：猪肝1具（切，去脂膜，用醋5升煮令尽，取出研如膏），鳖甲1两半（涂醋炙令黄焦，去裙襴），厚朴2两（去粗皮.涂生姜汁炙令香熟），诃黎勒1两半（煨，用皮），陈橘皮1两（汤浸，去白瓤，焙），川椒3分（去目及闭口者，微炒去汗），柴胡1两（去苗），桂心3分，苍术1两，木香3分，桔梗3分（去芦头），乌梅肉3分（微炒），甘草半两（炙微赤，锉），紫菀1两（洗，去苗土），干姜3分（炮裂，锉），芜荑3分（微炒），当归3分。
出处：《圣惠》卷二十八。
主治：冷劳，肌体羸瘦，或时腹痛，食饮不消，日渐尪羸。
加减：None
功效：None
用法用量：每服30丸，食前粥饮送下。
制备方法：上为末，入猪肝膏内，和捣为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌苋菜。
附注：None
