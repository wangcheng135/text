序号：84199
名称：五伤接骨膏
组成：没药1两（好者，生用），乳香1分（好者，生用），川椒1两（拣择去子，生用），芍药1两（拣择，生用），川芎1两（好者，生用），川当归1两（拣择净，洗过，细切，炒令干），自然铜1两半（火烧令赤，候冷杵研令细，水飞过，纸上衬，于灰上吸干，取1两）。
出处：《博济》卷五。
主治：一切伤折，驴马伤堕，打扑闪挫，疼痛不可忍。
加减：None
功效：None
用法用量：每服1丸，用好酒1两同煎，煎散药丸为度，候通口呷讫，就痛处卧片时。只可1服止，大段疼痛者，2丸至3丸，永愈。
制备方法：上为末，入自然铜末拌和令匀，用黄蜡3两半，于铫子内熔为汁，次入药末，不住手搅令匀，为丸如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：五伤接骨丸《圣济总录》卷一四四)、没药丸(《得效》卷十八)。
