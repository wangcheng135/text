序号：76928
名称：鹅梨煎丸
组成：大鹅梨20枚（去皮核，用净布绞取汁），皂荚10挺（不蛀者，去黑皮，用浆水2升，揉取浓汁），生地黄半斤（净洗，研绞取汁），生薄荷半斤（细研取汁），蜜半斤（以上5味，同于银石器中慢火熬成膏，入诸药末），木香1两，人参1两，白茯苓（去黑皮）1两，白蒺藜（炒去角）1两，牛膝（酒浸1宿，切，焙干）1两，肉苁蓉（酒浸1宿，切，焙干）1两，羌活（去芦头）3分，防风（去叉）3分，白术3分，青橘皮（去白，焙）3分，桔梗（锉，炒）3分，山芋3分，半夏（汤洗7遍，焙干，炒过）1两，槟榔（煨，锉）2两，甘草（炙，锉）半两。
出处：《圣济总录》卷六十四。
主治：热痰。
加减：None
功效：凉心肺，利胸膈，解热毒，补元益气。
用法用量：每服15丸，加至20丸，食后荆芥汤送下，1日2次。
制备方法：上除5味为膏外，余为末，入膏拌和，杵令硬软得所，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
