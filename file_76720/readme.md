序号：76736
名称：槟榔汤
组成：槟榔1两，细辛（去苗叶）1两，半夏（陈者，汤洗7遍，焙干）5两，紫苏2两，甘草（炙，锉）2两，大黄（锉，炒）2两，陈橘皮（汤浸，去白，焙）2两，生姜（切，焙）3两，紫菀（去苗土）3两，柴胡（去苗）3两，附子1枚（炮裂，去皮脐），赤茯苓（去黑皮）4两。
出处：《圣济总录》卷七十一。
主治：积聚结实，腹满刺痛，泄利不止。
加减：若有癥瘕癖结，加鳖甲（去裙襕，醋炙）、防葵各2两；上气，加桑根白皮（锉）3两，枳壳（去瓤，麸炒），厚朴（去粗皮，生姜汁炙）各2两。
功效：None
用法用量：每服3钱匕，水1盏，煎至7分，去滓温服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
