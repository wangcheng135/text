序号：9700
名称：牛膝丸
组成：牛膝1两（去苗），赤芍药3分，甘草3分（炙微赤，锉），鬼箭羽3分，人参3分（去芦头），当归1两（锉，微炒），白术1两，牡丹2两，虎杖1两，桂心1两，乌梅肉半两（微炒），白薇半两，川大黄1两（锉碎，微炒），虻虫1分（去翅足，微炒令黄），水蛭1分（微炒令黄），蒲黄半两，熟干地黄1两。
出处：《圣惠》卷七十九。
主治：产后月水不调，小腹痞硬，乍寒乍热，食不生肌，心腹刺痛，口干多唾，手足沉重。
加减：None
功效：None
用法用量：每服20丸，食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
