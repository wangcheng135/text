序号：51196
名称：荜澄茄散
组成：荜澄茄半两，木香半两，白豆蔻半两（去皮），白术半两，槟榔半两，草豆蔻半两（去皮），诃黎勒皮半两，肉豆蔻半两（去壳），枳壳半两（麸炒微黄，去瓤），白茯苓半两，干姜半两（炮裂），桂心3分，丁香半两，陈橘皮半两（汤侵，去白瓤，炒），甘草半两（炙微赤，锉），厚朴1两（削去粗皮，涂生姜汁，炙令香熟）。
出处：《圣惠》卷五。
主治：脾胃虚冷，食即欲呕，心腹胀闷，水谷不消，四肢无力。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，入生姜半分，枣3枚，煎至6分，去滓，食前稍热服。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
