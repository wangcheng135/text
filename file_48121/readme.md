序号：48136
名称：神应乌玉丹
组成：棕榈2两，乱发2两，猪蹄甲14个（须后脚者），猬皮4两，牛角腮3两，苦楝树根2两半（洗净），槐角1两半，雷丸1两，芝麻1两（拣净），真麝香2钱，滴乳香半两。
出处：《中藏经·附录》。
主治：久新肠风、痔瘘，着床头痛不可忍，肠内有虫者。
加减：None
功效：None
用法用量：每服8粒，先细嚼胡桃1枚，空心、晚食前以温酒吞下，每日2次；如病甚，每日3次。切忌服别药。不过2-3日，永除根本。
制备方法：上除乳、麝二香别研细外，并细锉，入藏瓶或沙盒子，不固泥，用熟炭火煅；烟才尽，便去火，入二香同研匀，无灰醇酒打面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：乌玉丹（《百一》卷十四）、神应黑玉丹（《局方》卷六吴直阁增诸家名方）、马玉丸（《魏氏家藏方》卷七）、神应黑玉膏（《卫生宝鉴》卷十七）、黑玉丹（《万氏家抄方》卷三）。
