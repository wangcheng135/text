序号：40554
名称：虎骨散
组成：虎胫骨（酥炙）2两，天麻1两，木香1两，羌活1两，川芎1两，黄耆1两，蒺藜（炒，去刺）1两，青皮（去白，炒）1两，大腹皮1两，桂心1两，槟榔1两，沉香1两，白茯苓（去皮）1两，桃仁（去皮尖，炒）1两，山芋1两，葛根1两，海桐皮1两，五味子1两，败龟版（醋浸，炙）1两，白鲜皮1两，甘草（炙）半两，肉苁蓉（酒浸，焙）1两半，附子（炮裂，去皮脐）1两半。
出处：《奇效良方》卷二。
主治：风，腰脚疼痛，下注脚膝，行步不得，或肿痒，或在两膝肿疼痛，久疗不愈，渐致足胫细小少力。
加减：None
功效：None
用法用量：每服2钱匕，空心、临卧温酒或盐汤调下。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
