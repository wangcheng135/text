序号：6118
名称：加味泽泻汤
组成：泽泻1钱半，枳壳1钱半（炒），黑丑2钱，槟榔1钱半，赤芍1钱半，木通1钱半，赤苓3钱，猪苓1钱半，陈皮1钱半。
出处：《医略六书》卷二十四。
主治：脚气上攻，脉沉者。
加减：None
功效：None
用法用量：水煎，去滓温服。
制备方法：None
临床应用：None
各家论述：湿热内滞，气化不得流通，而脚气上攻，故两胫红肿疼痛不止焉，泽泻泻膀胱之湿，猪苓利三焦之湿，黑丑荡涤积水之湿，槟榔疏导结滞之气，赤苓渗脾胃之湿兼利营气，枳壳破肠胃之气能宽胸膈，木通降火利水，赤芍破血退肿，陈皮理气和中也。使湿热并化则水府肃清，而血气通利，两胫红肿无不退，脚气疼痛无不痊矣。此导气泻湿之剂，为脚气肿痛上攻之专方。
用药禁忌：None
