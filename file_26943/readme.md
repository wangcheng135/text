序号：26951
名称：当归散
组成：当归1两，香附子1两2钱，生干地黄1两，白芷1两（以上4味锉碎，不犯铁器，炒黑色存性为末），青盐半两（烧干，入诸药中1处和），皂角5斤（刮皮去子用），草乌头2两，生姜5斤（穿地坑烧红，埋在坑内5次，其生姜自干）。
出处：《鸡峰》卷二十一。
主治：None
加减：None
功效：乌髭鬓，牢牙齿。
用法用量：每日两次搽齿如常法，须发白者，1月变黑，遇旦望日用河水洗髭。
制备方法：上先将草乌头、生姜皆切作片子，及将皂角刮去黑皮，分作两片，内铺乌头、生姜了，用麻片缚夹定，炭火上用铲炙令通红，于地上用新瓦盆盖，如麸炭状，研细，与前药一处合和。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
