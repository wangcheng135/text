序号：42249
名称：青娥丸
组成：仙茅（酒浸，洗，勿犯铁器）4两，白茯神（去皮）4两，破故纸4两（酒浸1日），怀生地黄（酒浸，砂锅内蒸黑）4两，甘枸杞子4两，小茴香（盐酒炒）2两，黄精（酒蒸）4两，川杜仲（去皮，姜汁炒）4两，天门冬（去心）4两，菟丝子（酒炒，捣饼）4两，当归（酒洗）4两，肉苁蓉（酒洗，去鳞）3两，怀熟地黄（酒洗）4两，巴戟（盐水泡，去心）4两，青盐4两（水洗去泥，打碎，炒），牛膝（去芦，酒洗）4两，拣参2两，琐阳3两，鹿茸1两（炙），川萆薢4两（童便浸7日，流水洗净，去臭气），核桃肉（去壳，捣如泥）150个。
出处：《寿世保元》卷五引马伏所方。
主治：诸虚百病，素患腰痛者。
加减：None
功效：None
用法用量：每服100丸，空心好酒、盐汤任下。
制备方法：上为细末，好酒打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
