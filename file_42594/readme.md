序号：42609
名称：临汝药酒
组成：当归250g，高良姜250g，生草乌750g，丁香250g。
出处：《河南省药品标准》。
主治：风湿麻木，腰背冷痛，半身不遂，口眼歪斜，产后中风。
加减：None
功效：温中散寒，活血祛风。
用法用量：口服，每服1毫升，每日2次，早晚空腹服。
制备方法：取丁香制成粗粉，余药切片，混合装入袋内，加61℃白酒6000毫升，密闭，水浴加热，使内温达65～70℃，保持24小时，降至室温，过滤，压榨残渣，合并滤液与压榨液。另取红糖1000g，炒至棕色味苦，加入酒内搅匀，静置5～7天，纱布过滤，至澄清液灌装，灯检，包装即得。
临床应用：None
各家论述：None
用药禁忌：服后两小时内禁热饮食；高血压、心脏病、孕妇忌服。
附注：None
