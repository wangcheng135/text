序号：19482
名称：地黄丸
组成：地黄2斤，杏仁半斤（去皮尖），金钗4两，石斛4两，牛膝4两，防风4两，枳壳4两。
出处：《传家秘宝》卷中。
主治：肝虚血不足，肢节拘急，筋脉李痛及肾虚眼目昏暗。及肾脏虚惫，肝受虚热，及远年日近暴热赤眼，风毒气眼。兼治干湿脚气，消中消渴，及诸风气等由肾气虚败者。
加减：None
功效：补肾气。补肝益肾，驱风明目。
用法用量：每服1钱，空心用无灰豆淋酒送下，1日2次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌一切动风毒等物。
附注：金髓煎丸（《圣济总录》卷一○二）、明睛地黄丸（《局方》卷七续添诸局经验秘方）、明眼地黄丸（《得效》卷十六）、明目地黄丸（《原机启微·附录》）、经验地黄丸（《医方类聚》卷六十七引《经验秘方》）。
