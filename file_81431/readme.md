序号：81447
名称：鳗鱼丸
组成：干地黄4两（炒），女贞子1两5钱（蒸，晒），龟腹版2两（刮白，盐水炙），旱连草1两5钱（晒），天门冬1两（去心，焙），麦门冬1两5钱（焙，去心），左牡蛎2两（另研极细），川贝母1两5钱（去心），云茯神2两（切片，晒），淡贡菜2两（漂净，切片炒），人中白1两（漂淡，生用），鳗鱼粉8两。
出处：《良方集腋》卷上。
主治：三阴不足，阳明有余，阴虚火旺，吐血、咯血，传尸骨蒸。
加减：None
功效：杀蛲虫，疗劳瘵。
用法用量：每服4钱，开水送下。
制备方法：上为细末，用放胖旗参4两，川石斛3两，煎浓汁为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：制鳗鱼粉法：用鳗鱼三斤，切，勿经水。锅内入百部一斤，同水煎蒸，甑上用薄荷叶、牡丹皮二味铺满，将夏布一方盖在药面，然后将鳗鱼置在布上，蒸一炷香，起甑捣烂，入淮山药粉拌匀做饼，烘干，磨末听用，配药合丸。
