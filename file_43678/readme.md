序号：43693
名称：前胡膏
组成：前胡（去芦头）3两，白术（生用）3两，白芷（锉）3两（留1枚不锉，以验膏成），芎3两，椒（去目及闭口，生用）、吴茱萸（汤洗，焙干，炒）2两，附子（去脐皮，生用）5两，当归（细切）5两，细辛（去苗叶）3两，桂（去粗皮）3两。
出处：《圣济总录》卷九。
主治：肉苛。营虚卫实，肌肉不仁；疮癣疮痍；诸风？？麻疼痛；伤折及附堕损等。
加减：None
功效：None
用法用量：病在外，摩之；病在内，以热酒调化服樱桃大；若伤折及坠堕损，外摩内服。
制备方法：上锉，别以苦酒3升（拌匀，同窨1宿，以炼成猪膏5斤，入药微煎之，候白芷黄紫色，去滓膏成。
临床应用：None
各家论述：None
用药禁忌：None
附注：前胡散（《宣明论方》卷一）。
