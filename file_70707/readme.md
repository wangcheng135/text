序号：70723
名称：紫雪散
组成：升麻1两，寒水石1两，石膏1两，犀角1两，羚羊角1两，元参2两，沉香5钱，木香5钱，甘草8钱。
出处：《外科正宗》卷四。
主治：小儿赤游丹毒，甚者肚腹膨胀，气急不乳，伤寒热躁发狂，及外科一切蓄毒在内，烦躁口干，恍惚不宁。重腭，舌疔，及小儿赤游丹失治，毒气入里，腹胀坚硬，声音嘶哑，吮乳不下咽者。
加减：None
功效：None
用法用量：大人每用1钱，小儿2分，10岁者5分，徐徐咽之即效。病重者加1钱亦可，或用淡竹叶、灯心汤化服。
制备方法：水5碗，同药煎至5碗，滤清，再煎滚，投提净朴消3两6钱，微火慢煎，水气将尽欲凝结之时，倾入碗内，下朱砂、冰片各2钱，金箔100张，各预研细和匀，碗炖，水内候冷，凝成雪也。
临床应用：None
各家论述：None
用药禁忌：None
附注：紫雪丹（《囊秘喉书》卷下）。
