序号：22406
名称：阳起石丸
组成：远志（洗，取肉）半两，阳起石（煅）1两，沉香（不见火）1两，北五味1两，嫩鹿茸1两，酸枣仁（出皮）1两，桑螵蛸（微炒）1两，白龙骨1两，白茯苓1两，钟乳粉1两，天雄1两（姜汁制，去脐），菟丝子2两。
出处：《普济方》卷二二四引《孟氏诜诜方》。
主治：丈夫阴阳衰微，阳事不举，才交即泄，寒精自流，胸中短气，阴汗盗汗，冷痛或痒而生疮，出黄浓水。
加减：若强壮人服，觉火热，去天雄，加肉苁蓉。
功效：助阴壮阳。
用法用量：每服40-50丸，炒茴香、白茯苓煎汤吞下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
