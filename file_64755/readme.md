序号：64771
名称：移尸灭怪汤
组成：人参1两，山茱萸1两，当归3钱，乳香末1钱，虻虫14个，水蛭（火煅死）14条，二蚕砂末3钱。
出处：《辨证录》卷八。
主治：传尸痨。
加减：None
功效：None
用法用量：每日服100丸。
制备方法：上各为末，炼蜜为丸。
临床应用：None
各家论述：古人传祛逐痨虫之药，多至损伤脾胃，所以未能取效。今以人参以开胃，用山茱萸以滋肾，且山茱萸又是取虫之味，同虻虫、水蛭以虫攻虫，则易于取胜。尤恐有形之物，不能深入于尸虫之内，加当归以动之，乳香以开之，引其直入而杀之也。复虑虫蚀补剂以散药味，更加二蚕砂者，乃虫之粪也，虫遇虫之粪，则弃而不食，而人参、归、萸得行其功，力助诸药以奏效也。
用药禁忌：None
附注：None
