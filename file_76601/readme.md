序号：76617
名称：舒筋活络膏
组成：夏枯草3钱，鸡血藤膏5钱，金果榄3钱，冬虫夏草4钱，金银花6钱，连翘5钱，桑寄生6钱，老鹳草5钱，没药3钱，海风藤3钱，全当归4钱，生杭芍3钱，川芎2钱，细生地3钱，川羌活3钱，威灵仙3钱，独活3钱，宣木瓜3钱，广橘红3钱，川郁金3钱（研），半夏3钱，生甘草2钱，麝香面1钱（后入）。
出处：《慈禧光绪医方选议》。
主治：络阻筋伤。
加减：None
功效：None
用法用量：None
制备方法：用香油3斤，将药炸枯，滤去滓，入黄丹1斤收膏，老嫩合宜。
临床应用：None
各家论述：本方用四物养血，用风药舒肝，用滋阴药养肝，用藤药通络，更用麝香辛窜搜剔风邪，于是络阻筋伤之症可痊。
用药禁忌：None
附注：None
