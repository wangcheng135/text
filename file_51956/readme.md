序号：51971
名称：除湿丹
组成：槟榔2两，甘遂2两，威灵仙2两，赤芍药2两，泽泻2两，葶苈2两，乳香1两，没药1两（另研），黑牵牛半两，大戟2两（炒），陈皮4两（去白）。
出处：《宣明论》卷七。
主治：诸湿客搏，腰膝重痛，足胫浮肿，筋脉紧急，津液凝涩，便溺不利，赤瘾疹，疽痈发背，疥癣走注，脚气，疮疖。妇人腰胯疼痛、两脚麻木，恶寒喜暖者。闪脑膝踝足腕大痛及枚疮落马，坠堕打扑等。
加减：None
功效：None
用法用量：每服50丸至70-80丸，食前以温水送下。
制备方法：上为细末，面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：服药前后，忌酒1日，药后亦忌湿面。食温粥补暖；中病即止，虚弱者当慎。
附注：《普济方》引《经验良方》有青皮，无葶苈。
