序号：60030
名称：资生通脉汤
组成：白术3钱（炒），生淮山药1两，生鸡内金2钱（黄色），龙眼肉6钱，山萸肉4钱（去净核），枸杞果4钱，玄参3钱，生杭芍3钱，桃仁2钱，红花1钱半，甘草2钱。
出处：《衷中参西》上册。
主治：室女血枯经闭，饮食减少，灼热咳嗽。
加减：灼热不退，加生地黄6钱；咳嗽，加川贝母3钱、米壳2钱；泄泻，去玄参、加云苓2钱，或酌加白术用量；大便干燥，加当归、阿胶各数钱；小便不利，加车前子3钱、地肤子2钱；肝郁者，加生麦芽3钱、川芎、莪术各1钱；汗多者，加生龙骨、生牡蛎各6钱。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
