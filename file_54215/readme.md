序号：54230
名称：大岩蜜汤
组成：栀子15枚，甘草1两，干地黄1两，细辛1两，羊脂（青羊角亦得）1两，茯苓1两，吴茱萸1两，芎（《千金》用芍药）1两，干姜1两，当归1两，桂心1两。
出处：《千金》卷八（小字注）引《小品方》。
主治：贼风，腹中绞痛；并飞尸遁注，发作无时，发即抢心胀满，胁下如锥刀刺；并主少阴伤寒。
加减：若痛甚者，加羊脂3两，当归、芍药、人参各1两；心腹胀满坚急者，加大黄3两。
功效：None
用法用量：上(口父)咀。以水8升，煮取3升，去滓，纳脂令烊，温分3服，每服相去如人行10里顷。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
