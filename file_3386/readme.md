序号：3386
名称：太一散
组成：独活（去芦头）1两半，续断1两，杜仲（炒去丝）1两，肉桂（去皮）1两，牛膝（酒浸1宿）1两，黑附子（炮，去皮脐）1两，白茯苓（去黑皮）1两，人参（去芦头）1两，防风（去芦头）1两，白芍药1两，当归（去芦头）1两，川芎1两半，熟干地黄1两半，秦艽（去芦头土）1两半，甘草（锉，炒）1两半，细辛（去苗叶土头节）1两。
出处：《御药院方》卷一。
主治：阳明经虚，风邪客入，令人口眼喎斜，麻木不仁；及惊风痫窒，手足搐搦，不省人事。
加减：None
功效：None
用法用量：每服3钱，水1大盏，煎至7分，去滓温服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“太乙散”。
