序号：6629
名称：加减八味地黄汤
组成：怀熟地8钱至1两余，丹皮1钱，山茱肉2钱，茯苓2钱5分，山药2钱4分，泽泻（盐水炒）1钱，牛膝1钱，麦冬3钱，五味子6分，肉桂（盐煎，刮去粗皮）1钱。
出处：《冯氏锦囊·杂症》卷十二。
主治：凡咳嗽不止，痰唾槁枯，身热骨痛，头眩目胀，或时畏寒，六脉弦数，肌肉日瘦，夜不能寐，甚有两颊之间肿硬者。
加减：如尺脉无神者，加熟附子1钱。
功效：None
用法用量：水3大碗，煎1碗，食前温服，日1剂，不煎渣，服后随进饮食压之。数剂后，热退嗽减，六脉洪缓无力，身体倦怠，照前方冲参汤服。愈后，每早淡盐汤吞服八味丸4-5钱。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
