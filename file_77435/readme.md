序号：77451
名称：雄朱丹
组成：雄黄1分（细研），朱砂1两半（细研，水飞过），天麻1两，白花蛇2两（酒浸，去皮骨，炙微黄），乌蛇2两（酒浸，去皮骨，炙微黄），肉桂1两半（去粗皮），芎半两，蔓荆子半两，白僵蚕（微炒）半两，牛膝（去苗）半两，萆薢（锉）半两，羚羊角屑半两，白附子（炮裂）半两，槟榔半两，天南星（炮裂）半两，当归半两，藿香半两，甘菊花半两，地龙（微炒）1分，干姜（炮裂，锉）1分，羌活1分，独活1分，人参（去芦头）1分，麻黄（去根节）1分，天雄（炮裂，去皮脐）1分，踯躅（酒拌，炒干）1分，防风（去芦头）1分，汉防己1分，白芷1分，白茯苓1分，藁本1分，干蝎（微炒）1分，蝉壳1分，牛蒡子（微炒）1分，龙脑（细研）1分，麝香（细研）1分。
出处：《圣惠》卷二十五。
主治：一切风。
加减：None
功效：None
用法用量：每服1丸，以薄荷酒研下。
制备方法：上为末，入研了药，同研令匀，炼蜜为丸，如樱桃大，以朱砂末内滚过。
临床应用：None
各家论述：None
用药禁忌：忌生、冷、猪、鸡、毒鱼、粘滑、动风物。
附注：None
