序号：25345
名称：寻常煮散
组成：独活6两，汉防已6两，麻黄（去节）6两，茯苓6两，丹参6两，牛膝6两，磁石16两（碎），黄耆6两，防风6两，人参6两，犀角6两，升麻4两，青木香4两，桂心4两，石膏14两（碎），吴茱萸8两，生姜、半夏（洗）、槟榔（大者）、杏仁、大黄（切）。
出处：《外台》卷十九引《苏恭方》。
主治：诸脚气弱，未至大发。
加减：心下满，呕逆者，加半夏1两；大便涩者，去磁石，加大黄1两。
功效：None
用法用量：上21味，将16物捣切如豆大，分作30分，分和为1服服。以槟榔3枚、生姜1两，各合皮切，杏仁14枚（去皮尖），以水2升3合，煮取7合，去滓，日晚或夜中服之，每日1次；若气盛时，每日2次；可1服，或2日1服，或3日1服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌生葱、醋物、羊肉、饧。
附注：方中半夏至大黄用量原缺。
