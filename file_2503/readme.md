序号：2503
名称：午时茶
组成：甘草80两，薄荷40两，神曲（炒）160两，香薷200两，山楂160两，茯苓80两，法半夏80两，苏叶80两，川芎80两，厚朴（制）128两，柴胡80两，白芷160两，橘皮80两，防风80两，前胡80两，枳壳（炒）80两，大腹皮80两，砂仁80两，麦芽（炒）160两，苍术（炒）136两，羌活128两，麦冬80两，九菖蒲40两，桔梗80两，独活160两，扁豆160两，葛根160两，藿香160两，黄芩160两，香附（醋炒）80两，安化茶（清茶）1280两。
出处：《北京市中药成方选集》。
主治：感冒风寒，停食停水，咳嗽鼻塞，不思饮食；受暑中寒，腹痛作泄。
加减：None
功效：消食化滞，去暑。
用法用量：每服1块，煎水饮服。
制备方法：上为粗末，用鲜薄荷叶8两，生姜8两熬水，再将白面736两打稀面糊，拌合均匀，用模子印成小块，晒干，每块干重2钱，两块1包。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
