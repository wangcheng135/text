序号：72345
名称：紫正散合地黄散
组成：荆芥穗8分，北防风8分，北细辛4分（去茆），京赤芍8分，牡丹皮8分，紫荆皮2钱，小生地2钱，苏薄荷6分，牙桔梗8分，生甘草6分，净茜草1钱。
出处：《春脚集》卷一。
主治：咽喉诸症。
加减：孕妇，去丹皮，加四物汤；热盛者，加连翘、犀角；头痛闭塞，加开关散；烦渴，加银锁匙；潮热，加柴胡、黄芩；咳嗽，加麦冬、知母；大便秘结，小便赤涩，加木通；数日不大便，加元明粉；热壅肺闭，致气喘促，加麻黄5分（先滚去沫，再入药内同蒸）；痰稠，加贝母；阴虚，加四物汤。
功效：None
用法用量：每日1次，用开水泡药蒸服。
制备方法：引加红内消1钱（即茜草藤，5月5日采取，阴干），灯心20寸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
