序号：20927
名称：人参散
组成：人参1两（去芦头），附子1两（炮裂，去皮脐），神曲1两（微炒令黄），白术3分，麦糵1两（炒微黄），吴茱萸半两（汤洗7遍，焙干，微炒），厚朴1两半（去粗皮，涂生姜汁，炙令香熟），干姜半两（炮裂），陈橘皮1两半（汤浸，去白瓤，焙），甘草1分（炙微赤，锉），草豆蔻1两（去皮）。
出处：《圣惠》卷五。
主治：脾胃虚冷，食不消化，腹胁气胀，不思饮食，四肢少力。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加大枣3枚，煎至6分，去滓，不拘时候，稍热服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
