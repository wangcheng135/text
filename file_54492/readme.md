序号：54507
名称：柴胡散
组成：柴胡1两（去苗），白术3分，木香半两，紫菀2分（洗，去苗土），赤茯苓3分，赤芍药1两，鳖甲1两半（涂醋炙令黄，去裙襕），半夏3分（汤洗7遍去滑），杏仁3分（汤浸，去皮尖双仁，麸炒微黄），人参3分（去芦头），前胡3分（去芦头），诃黎勒1两半（煨，用皮），枳壳3分（麸炒微黄，去瓤），枇杷叶半两（拭去毛，炙微黄），甘草1分（炙微赤，锉）。
出处：《圣惠》卷二十六。
主治：肺劳，胸膈壅滞，痰嗽不止，四肢无力，少思饮食。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：忌饴糖、羊肉、苋菜。
附注：本方方名，《普济方》引作“柴胡汤”。
