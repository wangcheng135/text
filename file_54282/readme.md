序号：54297
名称：大金牙散
组成：金牙（烧）、雄黄、丹砂、龙胆、防风、玉支、大黄、曾青、茯苓、桂心、松脂、干姜、乌头（炮，去皮）、斑蝥（去翅足，熬）、亭长、细辛、消石、野葛、大戟、商陆、蛇蜕（熬）、芫青、鹳骨、芫花、附子（炮、去皮）、寒水石、人参、贯众、龙骨、蜀椒（汗、去目，闭口者）、露蜂房（熬）、巴豆（去皮心）、蜥蜴、蜈蚣（炙）、礜石（烧）、天雄狸骨（炙）、石胆、莽草各等分。
出处：《千金翼》卷二十。
主治：南方百毒，瘴气疫毒，脚弱肿痛，湿痹，风邪鬼疰。
加减：None
功效：None
用法用量：以绛囊佩带之，男左女右；每服1刀圭，未食以浆水或酒调下。以知为度。
制备方法：上为散。
临床应用：None
各家论述：.
用药禁忌：None
