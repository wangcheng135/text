序号：42508
名称：青黛散
组成：青黛2两，麝香1两，雄黄1两，朱砂1两，石盐1两，蚺蛇胆1两，盐绿1两，细辛1两，黄矾（烧汁尽）1两，薰陆香1两，黄连1两，黄柏1两，苦参1两，杏仁（去皮尖）1两，桂心1两，干姜1两，藜芦（烧灰）、莨菪子（炒）、附子（炮）半两。
出处：《幼幼新书》卷二十四引《婴孺》。
主治：小儿五种疳：内疳，眼涩，腹胀痢色无常定，或如泔淀，日渐羸瘦；头疳，鼻下赤烊，自揉其鼻，头上有疮，疮不着痂，渐流利达，耳生疮，有时目赤，头发渐稀，头皮光紧，渐渐羸瘦，头大项细；口疳，唇口被蚀，齿龈作五色，或尽峭黑，舌下白疮，上腭有孔子，口中见臭气，齿龈被蚀，口唇败烂；脊疳，疳蚀脊膂，十指皆痒，自咬甲，头发焦干，两膂虚空，脊梁如锯，右时腹胀，有时下痢；急疳，下部开张，痢下脓血，有时赤烂，痒不可忍，痢无其度，臭不可堪。并野鸡痔病，无名疮癣，及大人口中有疮，小儿白秃疮。
加减：None
功效：None
用法用量：量病传药，若疳在内，以井花水调下，3服止，且将息，不减，再服1杏仁许，3岁半钱；若口中有疮，用酥少许每夜安唇内，须臾，自至疮所；老鼻中疮及鼻赤烂，以酥和绿豆大2丸纳鼻中，日2-3度；若头上疮，以散敷之；下部外赤烂，以散敷之；若脊膂空虚，准前与服，仍以酥和散摩脊膂上；亦治野鸡痔病，绵裹纳之，外有头者，掐破，以散敷之；又治有疮无名，诸癣疥，用猪脂和涂之；大人口中有疮，绵裹含之；小儿白秃疮，以泔清洗去痂，拭干，先涂油，后敷散愈。
制备方法：上为散，垍盒收之。
临床应用：None
各家论述：None
用药禁忌：忌浆水、热面、猪、鱼、鸡、齏、蒜、滑腻、一切动风物。
附注：方中藜芦、莨菪子用量原缺。
