序号：74301
名称：催生安胎救命散
组成：乌药4两（别用醋炒黄色），前胡半两（拣净），菊花1两（去梗），蓬莪术2两（炮，乘热锉碎），当归半两（去芦须，洗，切，焙）。
出处：《卫生家宝产科备要》卷六。
主治：产难，死胎在腹，死血冲心，血山崩，产后一切血疾。
加减：None
功效：None
用法用量：如死胎在腹，每服3钱，用炒姜豆淋酒调下，连进3服，立下；死血冲心，每服2钱，用炒姜豆淋酒调下，入童子小便半盏；安胎，每服1钱，用热酒调下；血山崩，每服1钱，用热酒调下；寻常催生；每服3钱，用炒姜豆淋酒调下，只1-2服，立生。
制备方法：上锉，用好米醋炒干为度，同为末，用新瓷罐收，勿令失气味。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
