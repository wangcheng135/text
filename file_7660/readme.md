序号：7660
名称：二至丸
组成：鹿角（镑细，以真酥2两，无灰酒1升煮干，慢火炒令干）半斤，苍耳（酒浸1宿，炒干）半斤，麋角（镑细，以真酥2两、米醋1升煮干，慢火炒干）半斤，当归5两（细切，酒浸1宿，焙干），山药4两，白茯苓（去皮）4两，黄耆（蜜炙）4两，人参（去芦头）2两，沉香2两，沙苑疾藜（拣去土，净洗，焙干）2两，远志（去心）2两，肉苁蓉（酒浸1宿，切，焙干）2两，附子（炮，去皮脐）1两。
出处：《杨氏家藏方》卷九。
主治：None
加减：None
功效：补虚损，生精血，去风湿，明目聪耳，强健腰脚，和悦阴阳，既济水火，百疾不生。
用法用量：每服50至100丸，空心温酒，盐汤任下。
制备方法：上为细末，用酒3升，糯米3合煮烂，和捣为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
