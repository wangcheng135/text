序号：72014
名称：小营煎
组成：当归2钱，熟地2-3钱，芍药（酒炒）2钱，山药（炒）2钱，枸杞2钱，炙甘草1钱。
出处：《景岳全书》卷五十一。
主治：三阴亏弱，血虚经乱，无热无寒，经期腹痛，痛在经后者；妇人体本虚而血少，产后腹痛；产后阴虚发热，必素禀脾肾不足及产后气血俱虚，其证倏忽往来，时作时止，或昼或夜，进退不常，或精神困倦，怔忡恍惚，但察其外无表证，而脉见弦数，或浮弦豁大，或微细无力，其来也渐，非若他证之暴至者。血亏则涩而难产；胎衣不下。
加减：如营虚于上而为惊恐，怔忡不眠、多汗者，加枣仁2钱、茯神2钱；如营虚兼寒者，去芍药，加生姜；如气滞有痛者，加香附1-2钱，引而行之。
功效：专补真阴；培养气血；滑胎。临月服之易生。
用法用量：水2钟，煎7分，食远温服。
制备方法：None
临床应用：None

用药禁忌：None
