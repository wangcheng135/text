序号：68017
名称：黄耆汤
组成：黄耆（锉）1两半，附子（炮裂，去皮脐）1两，菖蒲（米泔浸1宿，切）1两，木通（锉）2两，磁石（火烧醋淬17遍）3两，五味子1两1分，防风（去叉）1两1分，玄参1两1分，人参1两1分，杜仲（去粗皮，锉，炒）1两1分，白茯苓（去黑皮）1两1分，熟干地黄（焙）1两1分。
出处：《圣济总录》卷一一四。
主治：风聋，飕飕如风雨钟磐声，或时出清水，或有脓汁；五聋鸣闹，不闻人声，出黄水；耳内生疮。
加减：None
功效：None
用法用量：每服3钱匕，以水1盏半，入生姜3片，大枣1枚（擘），同煎至7分，去滓，空心温服，日3次。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：菖蒲汤（原书卷一一五）。
