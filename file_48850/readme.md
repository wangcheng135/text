序号：48865
名称：秋石五精丸
组成：秋石1两（童男女洁净无体气者，沐裕更衣，各聚一室，用清洁饮食及盐汤与之，忌葱、蒜、韭、牛肉、不洁之物，椒、茶不用尤妙。聚便各盈缸，然后男女童便各另熬成秋石，各另盛瓦罐，盐泥固济，铁线固定，打一火炷香为度，连换铁线，打7火，然后男女秋石称匀，和成一处。如成块，研开用龙水将纸7层滤过，用锅仍熬秋石，其色雪白。用初生男胎之洁净妇人香浓乳汁和成，日晒夜露，但干即添乳汁，待日精月华取足49日后入药，用五精丸方见效），莲肉5钱，真川椒2分，小茴香5钱，白茯苓1两。
出处：《摄生众妙方》卷二。
主治：None
加减：None
功效：补养。
用法用量：上为丸，用初生男胎之妇人乳汁吞下。
