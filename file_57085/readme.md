序号：57100
名称：消息向导丸
组成：肉桂5钱，蛇床子5钱，川乌5钱，马蔺花5钱，良姜5钱，丁香2钱5分，韶脑2钱5分，木鳖子（去壳）2钱5分。
出处：《疡科选粹》卷八。
主治：腰胯疼痛，两足痠辛，下元不固，胞冷精寒，小便频数，遗精白浊，及妇人下寒，赤白带下，子宫冷痛，久不孕。
加减：None
功效：None
用法用量：每用1丸，以生姜汁化开，先将腰眼温水洗净后，将此药涂腰眼上，令人以手搓磨往来千遍，药尽方止，然后贴造化争雄膏。即用兜肚护住，初贴时忌7日，不得行房事，如入房，再用3钱贴脐上，又服中和丸1丸，然后行房，纵泄亦不多；如种子者，候女人经后1、3、5日将腰肾上膏药俱揭去，早上用车前子为末1钱，温汤调服，至晚交合，方得全泄成孕。
制备方法：上为极细末、炼蜜为丸，如弹子大，黄丹为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
