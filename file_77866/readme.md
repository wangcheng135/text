序号：77882
名称：薏苡仁丸
组成：薏苡仁1两，石斛（用细者，去根净，洗，细锉）3分，附子半两（炮，去皮脐），牛膝（酒浸，水洗，焙干）3分，生干地黄3分，柏子仁（研）半两，人参（去芦）半两，枳壳半两，细辛（去叶）半两，川芎（洗）半两，当归（洗，去芦，焙干）半两，甘草（炙）。
出处：《本事》卷七。
主治：悲哀烦恼伤肝气，至两胁骨疼痛，筋脉紧急，腰脚重滞，两股筋急，两胁牵痛，四肢不能举，渐渐脊膂挛急，手足枯悴。
加减：None
功效：None
用法用量：每服30-40丸，食前酒送下，日3次。丸子可食前，煮散食后，相兼服为佳。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
