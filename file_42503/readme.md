序号：42518
名称：青羹药酒
组成：苍术（米泔浸，炒）2两，乌药2两，牛膝（去芦）2两，杜仲（姜汁炒）2两，陈皮1两，厚朴（姜汁炒）1两，当归1两，枳壳（去瓤，麸炒）1两，独活1两，槟榔1两，木瓜1两，川芎1两，白芍1两，桔梗（去芦）1两，白芷1两，茯苓（去皮）1两，半夏（姜汁炒）1两，麻黄1两，肉桂1两，防己1两，甘草1两。
出处：《回春》卷五。
主治：男妇风湿相搏，腰膝疼痛，或因坐卧湿地，雨露所袭，遍身骨节疼痛，寒湿气。
加减：None
功效：None
用法用量：每服70-80丸，空心酒送下。
制备方法：上锉，以麻布袋盛之，用酒3斗，将药悬坛内，密封坛口，锅内煮一时久，然后则，过3日后，去药，随量饮之，滓晒干为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
