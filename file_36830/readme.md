序号：36840
名称：固真丹
组成：人参1两，干山药1两半（炒），当归（酒浸）1两，黄耆（炒）1两，黄柏（炒）1两，白术1两，杜仲（酒炙，炒断丝）1两，补骨脂（炒）1两，五味子半两（炒），泽泻半两，白茯苓1两，牡丹皮1两，山茱萸1两，熟地黄4两（汤头阑膏于石臼内，捣如泥）。
出处：《直指·附遗》卷九。
主治：诸虚百损，五劳七伤，水火不升，下元虚冷，脐腹疼痛。
加减：腰腿无力，加牛膝1两（酒炒）、败龟版1两半（酥炙）；夏天服，加桂半两；脉弱食少，再加附子半两（炮）。
功效：None
用法用量：每服80-90丸，空心淡盐汤送下。
制备方法：上为极细末，和地黄膏拌匀，却入炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：《济阳纲目》有羊肉，无当归。
