序号：71465
名称：紫脂膏
组成：好麻油4两，净花椒3钱，葱头7大个（连须7寸长）。
出处：《种福堂方》卷三。
主治：臁疮久不愈者。
加减：None
功效：None
用法用量：每膏贴5日1换。如痛者用甘草汤先洗，痒者花椒汤洗。若贴1膏即流尽黄水者，贴至5-6膏而愈；若贴至3膏方流尽黄水者，须贴至20膏而愈。凡初贴之膏出水者，膏中有毒气在内，揭下则无用，水尽后再贴之，膏须存之，以待后来将长肉结盖时用，此贴过旧膏贴之，以为收功最妙。
制备方法：3味同煎至葱焦脆，去滓，入白色松香5钱，黄占6钱，文火煎化，去上面浮出渣滓，煎至油面上有花纹，急离火，倾碗内，加入好银朱1钱，搅匀收之，待冷凝将碗合土地上3日，去火毒，摊夹纸膏贴之，纸只要一面刺孔。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
