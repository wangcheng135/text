序号：43300
名称：养阴驱邪汤
组成：全当归8钱（酒炒），川芎3钱，紫苏1钱5分，淡豆豉3钱，炮姜1钱5分，枳壳1钱5分，砂仁3钱（研），炙甘草3分。
出处：《慈航集》卷下。
主治：产后受瘟疫，头痛恶寒发烧。
加减：如恶心发呕，加藿香梗3钱；如作泻，加炒白芍5钱，车前子3钱；如热入血室，经到发昏，加益母草3钱，炒黑山楂3钱；如腹痛，加酒炒玄胡索2钱；如误药抽搐，反张发厥等证，加白僵蚕3钱，钩藤2钱，制南星2钱，橘红1钱5分；如大便结燥6-7日不通，发厥者，加制军5钱，玄明粉2钱。
功效：None
用法用量：上用酒半林对水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：1服盖暖，出汗自愈。
附注：None
