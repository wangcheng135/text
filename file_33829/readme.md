序号：33838
名称：苁蓉丸
组成：苁蓉（酒浸，焙干）2两，楮实子1两，枸杞子1两，地肤子1两，金毛狗脊（去毛）1两，五味子1两，覆盆子1两，菟丝子1两，干山药1两，补骨脂（微炒）1两，远志（去心）1两，石菖蒲1两，萆薢1两，杜仲（去粗皮，锉，炒）1两，熟干地黄1两，石斛（去根）1两，白茯苓（去皮）1两，牛膝（酒浸，焙）1两，泽泻1两，柏子仁（微炒，别研）1两，山茱萸（酒浸，取肉）1两。
出处：《御药院方》卷六。
主治：None
加减：None
功效：壮元气，养精神。
用法用量：每服60-70丸，食前温酒送下，日1-2次。
制备方法：上为细末，酒面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
