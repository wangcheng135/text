序号：40566
名称：虎液膏
组成：上好羊脑炉甘石8两（打如莲子大，1分重为则。用新铜罐盛入童便，浸49日，滤去宿童便，再入新童便煮1柱香久，咬咸酸味，不必再煮；又不可煮老。研为细末，用缸片1大块，将药放在上，用硬炭火煅，1柱香久，甘石渐渐转如松花色，谨慎取起）。
出处：《异授眼科》。
主治：内障迎风冷泪，怕日羞明，昏花；或胬肉板睛，赤白翳烂弦；及时行火眼，或年久云翳遮睛，不能行路，但见人影，如白衣人行，有血根板睛者。
加减：None
功效：None
用法用量：外用点眼。
制备方法：上药取一份，用姜汁煮三次，候干，细研筛过，另用磁罐盛之，不可出气。
临床应用：None
各家论述：None
用药禁忌：None
附注：原书用本方，治内障迎风冷泪，怕日羞明，昏花者，须兑羊脑玉、凤麟、冰片合用；治胬肉扳睛，赤白翳膜烂弦者，须兑羊脑玉、青龙、冰片合用；治时行火眼，须兑羊脑玉、朱砂合用；治年久云翳遮睛，有血根扳睛者，先用青龙、羊脑玉点眼，直点至翳开之后，再用本方兑羊脑玉、凤鳞、冰片、珍珠、琥珀合用。
