序号：9825
名称：二苓二术汤
组成：白术2钱，苍术1钱，白茯苓3钱，赤茯苓3钱，陈皮1钱，天仙藤2钱，通草1钱，草豆蔻1钱。
出处：《医方简义》卷二。
主治：湿证。
加减：如湿邪上受，而为外湿者，加羌活、独活、防己各1钱5分；如湿自下受，而为内湿者，加木瓜、淡附片各2钱；如湿伤腑阳，泄泻，小便短涩者，加淡干姜、川连各8分；欲呕，加姜半夏1钱5分；欲嗳，加厚朴1钱，代赭石1钱；挟食，加槟榔、枳实（炒）各2钱；腰重，加防己，生黄耆各2钱；湿注小肠，淋痛者，加琥珀8分，猪苓、滑石各3钱；肿而脉涩者，加姜3片，淡附片2钱，车前子（炒）3钱；痰多，加竹茹1丸大。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
