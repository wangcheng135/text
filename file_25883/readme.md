序号：25891
名称：收功万全汤
组成：黄耆（蜜水炒）2钱5分，人参1钱，白术（去芦，炒）1钱，白茯苓（去皮）1钱，当归身1钱5分，川芎7分，白芍（酒炒）7分，怀熟地黄1钱，官桂3分，白芷3分，陈皮5分，甘草3分，防风5分。
出处：《寿世保元》卷九。
主治：痛疽、发背、诸疮毒溃脓后，毒气已尽，气血虚弱，不长肌肉，不合口，脓清欲作余症。
加减：如渴，加麦门冬、五味子；烦躁，加生地黄，麦门冬；有痰，加姜制半夏；泄泻，加厚朴（姜炒）；小便不利，加泽泻；怔忡不寐，加远志、酸枣仁（炒）；胸膈不宽，加厚朴（姜炒）、楂肉。
功效：None
用法用量：加生姜1片，水煎，温服。
制备方法：上锉。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
