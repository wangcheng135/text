序号：83907
名称：乌犀膏
组成：威灵仙10斤（拣紫色条子用），真天麻（取细末）2两，生犀（取末）1两，黑附子（须是拣存生正者，去皮脐，取末）2两，龙脑（生者，另研）1两。
出处：《传家秘宝》卷中。
主治：中风手足不随，偏枯瘫痪，脚气攻注，头面浮肿，口眼斜，语涩痰涎，精神恍惚，大便风秘。
加减：None
功效：None
用法用量：每服1钱匕，用薄荷汤化下；烦躁者，薄荷自然汁化下。
制备方法：先将灵仙同河水1石煮至3斗，以生绢滤去滓，只取清汁，更入醇酒1斗，同以银石器内煮至1斗，更清澄去细尘滓，次入天麻、附子末在药汁中，再以文火熬成膏，放令温冷后，入生犀、龙脑一处同搅匀，用瓷盒子内盛之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
