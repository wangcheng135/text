序号：43478
名称：养脾肥儿丸
组成：人参、白术、甘草、陈皮、枳实、木香、茯苓、砂仁、山药、莲肉、麦芽、神曲、山楂、青皮。
出处：《幼科发挥》卷四。
主治：脾胃久虚。
加减：None
功效：None
用法用量：米饮送下。
制备方法：上为末，荷叶浸水煮粳米饭为丸，如麻子大。
临床应用：脾胃病：本县大尹朱云阁公子，常有脾胃病，向是韩医生调治。平时服养脾丸，伤食服保和丸，未有宁日。一日问余云；闻汝小儿甚精，小官人脾胃久虚，汝可治之？余曰：当攻补兼用，不可偏补偏攻。韩医云：密斋非所长也，如专补脾胃则饮食难化，如专消导则中气易耗。尹不听，曰：汝进一方来。乃进养脾肥儿丸，修合服之，大效，再无脾胃之病。尹犹相信，赐匾。
