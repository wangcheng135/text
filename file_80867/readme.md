序号：80883
名称：麝珀散
组成：独茎川当归1两（去芦头并梢，洗，以好酒浸1宿，次日漉出控干，切作薄片子，火焙令干），紫团人参（去芦头）8钱（切作片子，慢火焙干），琥珀4钱（别研极细末，候众药成末子，却入琥珀末，一处拌匀），柏子仁半两（拣去壳，如众药碾成末子，即将柏子仁一味入碾100-300百碾，却将众药末子又在碾内一处，又同碾100-200百碾，抄出，再用纱罗子隔筛，余者药碾又罗，只候张盖），芎1两（锉碎，焙干用），官桂（削去粗皮，锉碎，不得见火，只于日热干）3钱半（令碾作细末，却拌和入药），甘草1两（火上微炙透），熟地黄2两（须用酒漉匀，9蒸9次晒者），白术（干好者，锉碎，焙干）8钱，白茯苓1两（锉碎，微焙）。
出处：《普济方》卷三二八引《卫生家宝方》。
主治：妇人诸虚不足，产前产后等疾。
加减：None
功效：None
用法用量：每服3大钱，水1中盏半，煎至8分，去滓，乘热再入麝香少许，搅匀，急以盏子盖定药少时。空心食前通口服；如急要服，便服，抄2平钱，泡麝香汤调下。酒浸服亦得。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻、硬物。
附注：None
