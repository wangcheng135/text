序号：66525
名称：丁香丸
组成：丁香1分，木香1分，桂（去粗皮）1分，阿魏（面裹煨，去面，研）1分，麝香（研）1分，硫黄（研）1分，水银（二味于盏内结成砂子）1分，硇砂（研，飞过）1分，粉霜（研）1分，胡粉（研）1分，巴豆（去皮心膜，研，取霜）4钱。
出处：《圣济总录》卷七十二。
主治：久积食癖，心腹时发疼痛，胸膈不快，痰逆恶心，脏腑不调，不思饮食，或下利脓血，里急后重。
加减：None
功效：散恶气，逐滞结。
用法用量：每服3丸至5丸，临卧煎生姜、枣汤送下。更看虚实加减，取下积聚恶物为效。
制备方法：上药前3味为末，与后8味合研匀细，用安息香半两，酒化滤过，入蜜少许，重汤同熬，和剂为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
