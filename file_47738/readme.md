序号：47753
名称：神仙固真丹
组成：苍术1斤（切片，米泔水浸），川乌1两（炮，去皮尖，切片），青盐1两，川楝子（去核）、当归、枸杞子1两，茴香（炒）1两，破故纸（同术炒黄）1两，菟丝子（酒浸）1两，地黄1两（切细，焙干）。
出处：《普济方》卷二一九。
主治：男子元阳气虚，妇人七伤，日渐瘦弱，饮食无味，小肠膀胱清精寒湿，小便并多，妇人胎前产后诸般冷疾，赤白带下血崩，子宫久冷，面色痿黄，四肢倦怠。
加减：None
功效：None
用法用量：每服30丸，男子以酒送下，女子醋汤送下。
制备方法：上为末，同术1斤细末，酒和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中川楝子、当归用量原缺。
