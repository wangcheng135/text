序号：60332
名称：通仙五宝散
组成：钟乳粉3分，大丹砂2分，琥珀5厘，冰片5厘，珍珠2厘半。
出处：《回春》卷八引王范泉方。
主治：凡人病过杨梅、天泡、绵花等疮，致成一切难名状之疾；或杨梅疮烂见骨，经年不收口者；或筋骨疼痛，举发无时；或通身疙瘩不消；或手足皱破出血；或通身起皮发靥，好一层起一层；或赤癜、白癜，鹅掌风癣；或皮好骨烂，口臭难当，及年久臁疮不愈；一切顽疮恶毒。
加减：None
功效：None
用法用量：每服5厘，每1料分作12帖。每1日用土茯苓1斤，水煎作12碗，去滓，清晨用1碗入药1帖，搅匀温服。其茯苓汤须1日服尽，不可别用汤水并茶，日日如是。服尽1料，至12日即愈；或有不终剂而愈者；如病重，须再服1料，无不愈也。
制备方法：上为细末，另入飞白霜2分半（炒过）合作1服。
临床应用：None
各家论述：None
用药禁忌：忌食鸡、鹅、牛肉及房事。
附注：通圣五宝丹（《东医宝鉴·杂病篇》卷八）、通仙五宝汤（《寿世保元》卷九）。
