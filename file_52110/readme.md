序号：52125
名称：除痛丸
组成：木香半两，乳香（别研）半两，沉香半两，藿香叶（去土）半两，肉桂（去粗皮）半两，青橘皮（去白）半两，枳实（麸炒，去瓤）半两，吴茱萸（汤洗7次）半两，京三棱（煨香，切）半两，蓬莪荗（煨香，切）半两，黑牵牛4两（取1出细末1两半余不用），麝香1钱半（别研），陈橘皮（去白）半两（锉，用巴豆去壳2两，炒令紫色，去巴豆不用）。
出处：《杨氏家藏方》卷五。
主治：中焦积寒，心腹疼痛，呕哕清水，自汗短气。
加减：None
功效：None
用法用量：每服50丸，食后以温生姜汤送下。
制备方法：上为细末，入麝香乳香别研匀，水煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
