序号：58103
名称：益府秘传冲虚至宝丹
组成：紫草茸8两，当归4两，鲜笋1斤，红花4两，木通4两，麻黄4两，白芷4两，白及4两，牡丹皮4两，赤芍4两，怀生地4两，牛蒡子4两，甘草4两。
出处：《寿世保元》卷八。
主治：痘疮初起，气血两虚，倒塌陷黑不起，不分地界，或咳或泻。
加减：None
功效：None
用法用量：每服1丸，鲜鸡汤送下。立起分地而出。
制备方法：上（口父）咀，用水30碗，锅内煮去2分取起，再水10碗，煎至5碗，去滓，共前汁煎滴水成珠，加蜂蜜4两，再熬成珠为度，听用。另以梅花蕊1钱半，蟾酥3钱，紫河车1具（酒煮成膏听用），僵蚕（炒）1两，全蝎（酒洗）、穿山甲（炒）、川黄连（酒炒）、杏仁（去皮尖，另研）、黄芩、蜂房（炒）、连翘（炒）、地肤子（炒）、大胡麻各1两。为细末，煎膏1半为丸，如龙眼大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
