序号：62778
名称：清气散
组成：诃黎勒（炮，取肉用）1分，缩砂仁半两，白豆蔻仁（怀干）半两，白茯苓（去皮）半两，人参（去芦）半两，京三棱（洗，湿纸裹煨）半两，胡椒半两，良姜（炒）半两，檀香1两（不见火），丁香1两（不见火），木香1分（不见火），干姜（炮，洗）1两半，橘红1两半，甘草2两（炙），青皮（汤泡，去瓤）1分。
出处：《魏氏家藏方》卷二。
主治：脾胃虚弱，脏腑挟寒，中气不和，停痰积冷，腹内膨胀，清浊不分，肠鸣飱泄，手足厥冷，脐腹多疼，呕吐恶心，胸膈不快，多困少力，肢节怠堕。
加减：None
功效：和脾胃，快气利膈，化宿滞，消饮食，清神养气。
用法用量：每服2钱，入盐少许，煎大枣汤调下，或入盐沸汤点服亦得，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
