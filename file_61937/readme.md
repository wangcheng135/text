序号：61953
名称：救生丹
组成：腽肭脐1对（1两以上者，去膜，研），朱砂半两（研），附子4只（8钱重者，去皮脐，酒煮10沸，焙干，用黄土拌和，同蒸半时许），人参3两（去芦），白术3两（炒），远志3两（去心），当归3两（去芦，酒浸），天门冬（去心）3两，神曲5两（炒），鹿茸5两（爁去毛，酥炙），肉苁蓉（酒浸，去皮土）5两，沉香1两（不见火）。
出处：《魏氏家藏方》卷六。
主治：心肾不交，脾气虚寒，营卫不行，大肉俱陷，真气不守，津液枯少。
加减：None
功效：None
用法用量：每服30丸，加至50丸，空心、食前温酒送下。
制备方法：上为细末，用精羊肉2斤（细切，去皮膜秤，酒煮过，入砂盆内研烂），别用好酒5升，入腽肭脐、当归、肉苁蓉、天门冬末，同熬成膏，入余药末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
