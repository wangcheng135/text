序号：71028
名称：稀涎千缗汤
组成：半夏（大者）14枚，猪牙皂角1挺（炙），甘草1钱，白矾2钱。
出处：《古今名医方论》卷二。
主治：风痰不下，喉中声如牵锯，或中湿肿满。
加减：None
功效：None
用法用量：用生姜自然汁少许冲温水1盏，调末1钱灌之。得吐痰涎即醒。
制备方法：上为末。
临床应用：None
各家论述：柯琴曰：用生姜、半夏之辛以散之，甘草之甘以涌之，白矾之涩以敛之，牙皂之涌以开之，此斩关夺门之势，惟禀气素实而暂虚者可用。塞塞稍疏，续进他药，不可多用以伤元气。如平素虚弱者，又当攻补兼施，六君子汤加牙皂、白矾末以吐之，则庶几矣。若误作中风治之，去生便远。
用药禁忌：None
附注：稀涎汤（《时方歌括》卷上）。
