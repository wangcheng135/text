序号：23736
名称：利膈丸
组成：制野军4两，槟榔4两，莱菔4两，木香2两，茅术2两，陈皮2两，厚朴2两，果仁2两，枳壳2两，砂仁2两，山楂2两，神曲2两，麦芽2两，桔梗2两，青皮2两，藿香2两，甘草2两。
出处：《全国中药成药处方集》（吉林方）。
主治：肝瘀胃热，发为结胸，胸脘疼痛，咯气食少，肝气横逆，闭塞于胸，胸胁串疼，气促息急，湿蕴脾肺，痰如蛋清，漉漉有声，气喘咳嗽；食积结滞，中脘胀硬，大便闭塞，嘈杂吞酸。
加减：None
功效：开瘀顺气，消食化痰。
用法用量：每服1丸，日2次。
制备方法：上为细末，炼蜜为丸，每丸重2钱1分。
临床应用：None
各家论述：None
用药禁忌：忌食辛辣、油腻物。孕妇忌服。
附注：None
