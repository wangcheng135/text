序号：48469
名称：神效黄耆汤
组成：蔓荆子1钱，陈皮（去白）5钱，人参8钱，炙甘草1两，白芍药1两，黄耆2两。
出处：《兰室秘藏》卷上。
主治：浑身麻木不仁，或头面手足肘背或腿脚麻木不仁；及两目紧急缩小，羞明畏日，隐涩难开，或视物无力，晴痛昏花，手不得近，或目少精光，或目中热如火。
加减：如小便淋涩，加泽泻5分；如有大热证，加酒洗黄柏3分；如麻木不仁，虽有热不用黄柏，只加黄耆1两；如眼缩急，去芍药；如麻木甚者，加芍药1两。
功效：None
用法用量：上（口父）咀。每服5钱，水2盏，煎至1盏，去滓，临卧稍热服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：眼缩急者，忌酒、醋、面、大料物、葱、韭、蒜辛物。
附注：黄耆汤（《医学入门》卷八）。本方方名，《中国医学大辞典》引作“神功黄耆汤”。
