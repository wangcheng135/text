序号：48882
名称：种子丹
组成：何首乌8两，仙茅（川者）4两，牛膝（酒浸）1两，白茯苓（人乳砂锅煮、蒸）1两，赤茯苓（牛乳砂锅煮）1两，生地黄（酒浸，蒸，捣）1两，枸杞（甘州红细者）1两，人参（坚明）1两，当归身（酒浸）1两，杜仲（姜汁拌，盐酒炒去丝）1两，远志（甘草汁浸，剥肉）1两，柏子仁（去油）1两，山茱萸肉1两，菟丝子（煮饼）1两，破故纸（盐酒炒香）1两，大胡麻（酒蒸9次）1两，核桃仁1两，松子仁（俱另研）1两。
出处：《简明医彀》卷七。
主治：None
加减：None
功效：种子。
用法用量：每服100丸，空心酒送下；夏月盐汤送下。
制备方法：上除地黄、桃仁、松仁外，各为末，入地黄，重晒，磨，加二仁，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
