序号：76486
名称：干地黄丸
组成：熟干地黄（切，焙）1两半，大麻仁（炒，研）1两半，萆薢（炒）1两，五加皮（锉）1两，石斛（去根）1两，赤芍药1两，防风（去叉）1两，牛膝（酒浸，切，焙）3分，桂（去粗皮）3分，酸枣仁（炒）3分，羌活（去芦头）3分，木香3分，附子（炮裂，去皮脐）1枚，牡丹皮半两，槟榔2枚（锉）。
出处：《圣济总录》卷八。
主治：中风，腰脚不随，膝胫沉重，饮食减少，日渐无力。
加减：None
功效：None
用法用量：每服15丸至20丸，空腹温酒送下，1日2次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
