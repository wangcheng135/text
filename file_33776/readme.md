序号：33785
名称：沉香煎丸
组成：马蔺花1两（头醋2升熬干），芫花2两（头醋2升熬干），青橘1两（汤浸，去瓤，焙干秤），陈橘皮1两（汤浸，去瓤，焙干秤），蓬莪术3两（炮，纸裹，碎），干姜（炮）、吴茱萸1两（汤浸，去涎汁者，焙干，炒黄），川乌头1两（炮，去皮脐），巴豆20个（去皮、油，5个煎紫色）。
出处：《传家秘宝》。
主治：一切冷气，心胸痞滞，腹胁疼痛，伤冷心痛。
加减：None
功效：消化冷积。
用法用量：每服5-10丸，以温酒送下。
制备方法：上为末，用好沉香1两半为末，米醋2升，慢火熬成膏，加少许面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
