序号：56596
名称：大桂皮汤
组成：桂心6两，当归2两，细辛2两，黄芩2两，人参5两，厚朴（炙）3两，枳实（炙）3两，芍药3两，芎3两，黄耆4两，麦门冬（去心）1升，吴茱萸1升，半夏（洗）1升，蜜5合，附子1枚（炮，去皮），生姜2斤，五味子半斤，饴半斤，甘草6两（炙）。
出处：《千金翼》卷十九。
主治：气逆。叉胸，寒热往来，吸吸短气，恶闻人声，诸烦酸疼，咳逆不能饮食，饮食不生肌肉，溺黄里急绞痛，气上冲发咳，胃管有热，雷鸣相逐，寒冷厥逆，伤损五脏，语言难，喜直视，大便难。
加减：None
功效：None
用法用量：上(口父)咀。捣生姜取汁3升；以水2斗煮药，取6升，去滓，微火上煎，纳姜汁、蜜，饴，搅相得，煮取6升，每服1升，1日2次。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
