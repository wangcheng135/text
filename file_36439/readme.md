序号：36449
名称：和疟饮
组成：青皮1钱5分，柴胡1钱，制半夏2钱，草蔻仁1钱（研），厚朴1钱5分（姜汁炒），枳壳1钱5分（炒），槟榔1钱5分，甘草5分。
出处：《慈航集》卷下。
主治：疟，寒热分清，或一日一发，或间日一发者。
加减：如恶心呕吐，加藿香3钱；如热重，加炒黄芩1钱5分；如口渴，加知母2钱，川贝母1钱5分；如小便短赤，内热，加青蒿3钱；如腹痛作泻，加赤芍5钱，煨广木香1钱5分；如未病前食荤，加炒山楂3钱；如食鱼腥，加威灵仙1钱；如面食，加炒莱菔子3钱，一服即愈，如不愈，次日加黑豆49粒、炒川山甲1钱。
功效：None
用法用量：煨姜3钱、大枣3个为引，河井水煎，露1宿，疟前温服，二煎接服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：愈后饮食宜清淡，3-4日再食荤腥调理，庶无疟之反复。面食忌一月。
附注：None
