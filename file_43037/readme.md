序号：43052
名称：养心丹
组成：菖蒲1两，紫石英1两，茯神1两，苁蓉1两，远志1两，麦门冬1两，豆卷1两，柏子仁1两，当归1两，细辛1两，卷柏1两，干姜1两，人参1两，石膏1两，泽泻1两，薯蓣1两，秦艽1两，丹参1两，熟地黄1两，桔梗1两，白蔹1两，前胡1两，防风1两，白术1两，半夏1两，桂1两，牛黄1分，铁粉精1分，麝香1分，朱砂1分，金箔100片（1方有山药、甘草、芍药各1两）。
出处：《鸡峰》卷十一。
主治：None
加减：None
功效：补益心气，安神，去百邪，调顺营卫，补养肾气。
用法用量：每服30丸，人参汤送下。
制备方法：上为细末。枣肉为丸，如绿豆大。以牛黄等为衣。
临床应用：None
各家论述：None
用药禁忌：None
