序号：39076
名称：泽泻丸
组成：泽泻（锉）2两，白茯苓（去黑皮）2两，防风（去叉）2两，人参2两，紫石英（研）2两，秦艽（去土）2两，黄耆（锉）2两，白术2两，山芋2两，白蔹2两，麦门冬（去心，焙）2两，桂（去粗皮）1两，当归（切，焙）1两，远志（去心）1两，柏子仁（炒）1两，石膏（捣碎，研）1两，桔梗（去芦头，炒）1两，大豆黄（炒）1两，大黄（锉，醋炒）1两，蜀椒（去目并闭口者，炒出汗）3分，赤芍药（去土）3分，干姜（炮裂，切）3分，细辛（去苗叶）3分，甘草（炙令微紫，锉）2两。
出处：《圣济总录》卷十四。
主治：风惊恐，梦寐不安。
加减：None
功效：None
用法用量：每服20丸，空心、晚食前米饮送下。
制备方法：除紫石英、石膏2味别研外，余药为细末，入所研2味拌匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
