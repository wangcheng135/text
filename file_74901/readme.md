序号：74917
名称：槟榔丸
组成：黑牵牛4两（拣去杂物2两，炒令香起方得；2两生用，重洗，焙，杵为末；取3两用），槟榔半两，木香1分，陈橘皮1分（去白），干姜1分（炮）。
出处：《普济方》卷一○五引《博济》。
主治：None
加减：None
功效：通利三焦，疏逐风气，宽胸膈，化痰涎，散腹胁壅滞，清头目，化酒食毒。
用法用量：并夜卧，浓煎姜汤送下30丸。饮食伤，煎枣汤送下；痰涎壅喉痛头疼，食后生姜汤送下15丸；妇人产前后诸疾，煎生姜汤送下；止渴，枣汤送下。
制备方法：上为末，同煮粟米糊，细研，搜为丸，如梧桐子大，晒干，以瓷盒盛之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
