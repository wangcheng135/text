序号：3302
名称：天雄丸
组成：天雄1两（炮裂，去皮脐），蛇床子3分，细辛半两，川大黄（锉碎，微炒）半两，杜仲3分（去粗皮，炙微黄，锉），柏子仁3分，白茯苓3分，防风半两（去芦头），萆薢3分（锉），菖蒲3分，泽泻3分，栝楼3分，桂心3分，薯蓣3分，远志半两（去心），川椒半两（去目及闭口者，微炒去汗），牛膝3分（去苗），石韦半两（去毛），山茱萸3分，白术3分。
出处：《圣惠》卷三十。
主治：虚劳，阴萎湿痒，搔之汁出生疮，小便淋沥，或赤黄，茎中痛；甚者失精尿血，目视(目荒)(目荒)，得风泪出，脚弱不能久立。
加减：None
功效：None
用法用量：每服30丸，食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
