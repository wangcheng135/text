序号：32380
名称：妙应丸
组成：京三棱（炮，锉如豆）1两，青皮（去白，锉如豆）1两，石三棱（锉如豆）1两，鸡爪三棱（锉如豆）1两，厚朴（生姜制，锉如豆。以上五味同用好醋浸3日，取出焙干）1两，槟榔1两，肉豆蔻1两，白豆蔻1两，木香6钱，巴豆霜半两，硇砂1两（飞，别研），干漆6钱（炒出烟）。
出处：《御药院方》卷四。
主治：九种心痛，积年瘕聚，久癥癖块，或大或小，因伤寒疼痛，发无时，或心下坚结，上冲胸痞，或气攻两胁，呕逆苦水，或喉痹烦闷，吐出蛔虫。
加减：None
功效：None
用法用量：每服2-3丸，食后温醋汤送下。
制备方法：上药除巴豆霜、硇砂外，同为细末，后入硇砂、巴豆霜，同研极细，用原浸药醋打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
