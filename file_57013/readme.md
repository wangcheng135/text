序号：57028
名称：消毒散
组成：牛蒡子（爁）6两，荆芥穗1两，甘草（炙）2两。
出处：《局方》卷十。
主治：疮诊壮热出而不快；上膈壅热，咽喉肿痛．口舌生疮。小儿疮疹已出，末能匀透．及毒气壅遏，虽出不快，壮热狂躁，咽膈壅塞．睡卧不安，大便秘涩，胸膈不利。小儿急惊风毒，赤紫丹瘤，九道有血妄行及遍身疮疥。牙根臭烂。
加减：None
功效：None
用法用量：每服1钱，用水1盏，煎7分，去滓，食后温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：若大便利者，不宜服之。
附注：消毒饮（《易简》）、三味消毒散（《疮疡经验全书》卷八）、三味消毒饮（《麻科活人》卷二）、解毒三贤饮（《疡医大全》卷三十三）。本方方名，《本草纲目》引作“必胜散”。
