序号：842
名称：六味青散
组成：乌头15分，桔梗15分，白术15分，附子（炮）5分，防风、细辛。
出处：《外台》卷&mdash;引《范汪方》。
主治：伤寒，赤色恶寒者。
加减：None
功效：None
用法用量：每服1钱5匕，温酒下。不知稍增。服后食顷不汗出者，饮薄薄粥1升以发之，温覆汗出濈濈可也，勿令流漓，勿出手足也。汗微出勿粉。若汗大出不止，温粉粉之。不得汗者，当更服之。得汗而不解者，当服神丹丸。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：忌生菜、猪肉、桃、李、雀肉等。
附注：六物青散(《千金》卷九)。方中防风、细辛用量原缺。《千金》本方用：附子、白术各一两六铢，防风、细辛各一两十八铢，桔梗、乌头各三两十八铢。
