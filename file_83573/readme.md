序号：83591
名称：乌须羊肝丸
组成：黑羊肝1具（竹刀切片，瓷盆晒，涂之以羊胆，晒干又涂，至30-50枚，愈多尤妙。夏月用稀绢蒙之，免其蝇污，晒干听用），熟地黄6两（蒸），当归（酒浸）4两，川芎4两，白芍药（酒炒）4两，何首乌（如法制）4两，旱莲草（如法蒸）4两，覆盆子（炒）3两，山茱萸肉3两，白茯苓（人乳浸，日晒夜露，候干用）4两，生地黄（酒浸1宿，晒干）4两，血余（壮年人、童男女及自己发为佳，多至1-2斤，用皂角汤洗净，再用苡、椒汤洗，晒干，入砂罐，炭火烧成灰，放地上3日，出去火毒，取下净研）3两。
出处：《医统》卷六十六引《类要》。
主治：None
加减：None
功效：乌须，明目。
用法用量：空心服80-90丸，临睡服50丸，酒送下。
制备方法：上除血余不磨，以群药共为末，另用熟地12两，酒煮浓汁2碗，作糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
