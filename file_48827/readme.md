序号：48842
名称：禹余粮汤
组成：禹余粮（煅，淬7遍）1两，白僵蚕（去黑者，微炒）1两，乌贼鱼骨（去甲皮）1两，龙骨（碎，研）半两，桂（去粗皮）半两，灶下黄土（无灰者）半两，石韦（去毛）半两，干姜（炮）半两，滑石（研）半两，赤芍药半两，半夏（浆水浸1宿，生绢袋子揉洗去滑，切开，生姜汁炒黄）半两，代赭（丁头者，研）半两。
出处：《圣济总录》卷一五三。
主治：妇人胞胎寒冷，绝产无子。
加减：None
功效：None
用法用量：每服3钱匕，以水1盏，加生姜3片，煎至6分，去滓，食前温服，1日3次。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
