序号：27078
名称：当归荆芥散
组成：荆芥穗、川芎、人参（去芦头）、当归（洗，焙）、桔梗（去芦头）、附子（炮，去皮脐）、柴胡（去苗）、防风（去芦头）、丁香、白芍药、蒲黄（炒）、鳖甲（醋炙令黄）、香白芷、牛膝（酒浸1宿，焙干）、白薇、肉桂（去粗皮）、半夏（汤洗7遍）、羌活（去芦头）、杏仁（汤洗，去皮尖，麸炒）、木香、白茯苓（去皮）、续断、槟榔、没药（别研）、肉苁蓉、柏子仁、地骨皮各等分。
出处：《杨氏家藏方》卷十五。
主治：妇人血风攻注，四肢疼痛，饮食减少，胸满恶心，日渐羸瘦。及血海虚冷，经脉不调，夜梦多惊，瘕癖气块。
加减：None
功效：None
用法用量：每服3钱，水1盏半，加生姜5片，煎8分，温服，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
