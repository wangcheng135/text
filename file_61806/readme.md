序号：61822
名称：接骨草酒
组成：接骨草叶1斤。
出处：《中药制剂汇编》。
主治：骨折。
加减：None
功效：接骨续筋。
用法用量：先用手法复位，然后用接骨草酒湿敷于骨折部位皮肤，外用小夹板固定，必要时加牵引。每天将接骨草酒滴入夹板下之纱布，成人50毫升，儿童30毫升，每天1&mdash;2次。
制备方法：法一：将接骨草叶1斤，捣烂加少许酒精炒略带黄色，然后加水，交火熬6到8小时，搓挤出药汁过滤，配成45％酒精浓度的药酒500毫升便可应用。法二：取接骨草叶洗净切碎，加水过药面煎煮（第1次2小时，每2次1．5小时），合并滤液过滤，浓缩成适量，药液加95％乙醇，使含醇量为50-60度，药浓度为1：1或1：2，放置24小时过滤即得。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
