序号：73309
名称：普济正气丸
组成：冬白术20两（土炒），陈皮15两（炒），山楂30两（炒），茅苍术20两（芝麻炒），制半夏20两（姜汁炒），槟榔20两（炒），茯苓20两（炒），紫苏20两（微炒），独活20两（酒炒），官桂5两（生研），麦芽30两（炒），泽泻15两（盐水炒），广藿香30两（微炒），生甘草5两（生研），香附20两（盐水炒），枳壳20两（麸炒），神曲20两（炒焦），车前子20两（盐水炒），厚朴20两（姜汁炒），草蔻仁30两（炒）。
出处：《慈航集》卷上。
主治：夏秋感受寒暑，伤风头痛，恶寒发烧，遍身骨节疼痛，霍乱吐泻，瘟疫疟痢，时毒斑疹及四时不正之气。
加减：None
功效：None
用法用量：凡感冒伤风头痛，恶寒发热，遍身骨节疼痛，用煨姜2钱，葱头3个，煎汤化服1丸，盖暖出汗即愈，凡霍乱吐泻，用煨姜2钱，灶心土3钱，煎汤化服1丸；凡疟疾，用煨姜3钱、黑枣3个，煎汤化1丸，在未来之早一时服；凡红痢，用金银花3钱，川连3分，煎汤化服1丸；腹痛，加广木香1钱5分同煎服；凡白痢，用煨姜3钱，红砂糖3钱，煎汤化服1丸；凡瘟疫，用赤饭豆5钱、煨姜3钱，煎汤化服1丸；凡斑疹，用观音柳3钱，炒升麻8分，煎汤化服1丸；凡孕妇，用砂仁壳1钱5分，煎汤化服1丸；凡产后，用当归8钱（酒浸），川芎1钱5分，煎汤化服1丸。大人1丸，小儿半丸。
制备方法：以上药如法炮制各研净细末，炼蜜为丸，每颗4钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
