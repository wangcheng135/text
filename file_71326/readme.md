序号：71342
名称：筒骨煎
组成：地骨皮1两，粉草1两，北柴胡1两，前胡1两，乌药1两，麻黄（不去节）1两，干葛1两，青蒿1两，苦梗1两，知母1两，天仙藤1两，北黄芩1两，人参半两，生干地黄半两，秦艽半两，鳖甲半两，黄耆半两（一方加当归、白芍）。
出处：《妇人良方》卷五。
主治：诸虚劳疾，羸瘦乏力，腰背引痛，心烦喘嗽，唾脓呕血，顽涎壅盛，睡卧有妨，胸膈气促，夜多盗汗，发焦耳鸣，皮寒骨热，一切五劳七伤，骨蒸。
加减：None
功效：None
用法用量：上(口父)咀，每服3钱，水1盏，酒1分，猪筒骨1茎（炙焦，分为4服），桃、柳枝各7寸，杏仁5粒（去皮尖，捶碎），煎至7分，去滓温服。加乌梅半个尤妙。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
