序号：44656
名称：宣肺散结饮
组成：桑皮9g，桔梗6g，酒黄芩9g，赤芍9g，牡丹皮6g。
出处：《张皆春眼科证治》。
主治：金疳。白睛上出现一个或数个突起的灰白色小泡，周围绕以赤丝，往往反复发作。
加减：若小泡溃后不敛，此为正虚邪实，方中桑皮改炙桑皮，以免泻之过重，再加麦门冬9g以养阴润肺；若丝脉从两眦而来，可加生地9g，木通3g，以清心中之邪热；若赤脉从下方上冲而来，可加天花粉9g，茅根15g，以清胃火；若小泡位于风轮边缘，可加当归9g，补益肝血，以防肺邪侵入，或加青葙子3g，清肝中之邪热，以防二火相并，损伤风轮。
功效：清宣肺气，活瘀散结。
用法用量：None
制备方法：None
临床应用：金疳：苟某某，男，18岁。1972年3月10日就诊。右目涩痛流泪10余天，结眵羞明。检查：患眼大眦赤胀，赤脉从眦部射出，白睛内侧有一灰白色小泡，周围有赤丝缠绕，且兼溲赤，脉数，舌红。此为金疳。为心肺二经邪热上攻所致。治以宣肺散结饮加生地9g，木通6g，服药3剂而愈，未再复发。
各家论述：方中桔梗宣肺散结，桑皮降肺气，酒黄芩清肺热，这样一宣、一降、一清，肺中郁热均能祛除；更有赤芍、牡丹皮活血凉血，疏通络脉以退目中赤丝。
用药禁忌：None
附注：None
