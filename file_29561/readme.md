序号：29569
名称：陈氏咳喘膏
组成：川乌6钱，官桂8钱，当归6钱，白芷8钱，木鳖子8钱，白及6钱，连翘8钱，茯苓6钱，赤芍8钱，草乌6钱，白薇8钱，牙皂5钱，乌药6钱，桃枝5钱，桑枝5钱，柳枝5钱，槐枝5钱。
出处：《温氏经验良方》。
主治：多年咳嗽气喘。
加减：None
功效：None
用法用量：贴背脊骨，由上往下数第三骨节。入伏之日起，用膏药1帖，贴满3伏；再于冬至之日起，用膏药1帖，贴尽九九。去病除根。
制备方法：上同麻油三斤浸一夜，熬焦去滓，加飞黄丹一斤（如麦色），急以柳、桃棍二根，搅至滴水成珠，加没药、乳香末各四两收膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
