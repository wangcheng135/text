序号：74272
名称：川乌头丸
组成：川乌头半两（盐拌，炒令黄，去皮脐），白花蛇肉1两（酒浸，炙微黄），雄黄半两（细研），白僵蚕1两（微炒），天南星1两（微炒），麝香半两（细研），朱砂半两（细研），腻粉1分，天麻1两，当归1两，天雄1两（炮裂，去皮脐），干蝎1两（微炒），麻黄半两（去根节），蝉壳1分（微炒），独活1两，芎半两，地龙半两（微炒），乳香（半两）。
出处：《圣惠》卷二十三。
主治：历节风疼痛，发歇不止。
加减：None
功效：None
用法用量：每服15丸，以温酒送下。
制备方法：上为末，入研了药，更研令细，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
