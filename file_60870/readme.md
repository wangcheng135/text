序号：60885
名称：酒煎干蟾丸
组成：干蟾1枚（用无灰酒1升煎其酒半升以来却去蟾骨，煎令熟，以后于乳钵内并酒1时，研令如膏，次用后药），肉豆蔻2枚（去壳），槟榔1两，甘草1寸（炙微赤锉），乳香半两（研人），朱砂1两（研细），麻黄半两（去根节），腻粉1钱（研入），胡黄连半两，黄连半两（去须），丁香1分，芦荟1分（研入），麝香1钱（研细），牛黄1钱（研细）。
出处：《圣惠》卷八十六。
主治：小儿五疳，不生肌肉。
加减：None
功效：None
用法用量：每服5丸，以粥饮送下，日3-4次。
制备方法：上为末。都研令匀，入蟾膏内为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：干蟾丸（《普济方》卷三七九）。
