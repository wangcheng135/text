序号：22525
名称：围药
组成：五倍子（炒焦黑）5斤，陈小粉（炒黄黑色）5斤，龟板（烧灰）12两，白及12两，白蔹12两，朴消12两，榆树皮12两，大黄8两，白芷梢8两，南星8两，黄柏8两，半夏8两，黄连4两，牙皂4两，萆薢4两。
出处：《疡科选粹》卷八。
主治：一切痈疽、发背、便毒、横痃、吹乳及风湿疼痛，小儿热毒火丹，无名肿毒。
加减：None
功效：定痛散毒。初起围之即消；已成者围之即生头出脓。
用法用量：每用入白蜜、猪胆、醋3味和匀，围红肿处，中留一孔，绵纸盖之，如纸干，用醋以刷子刷上。
制备方法：上为末，陈米醋糊匀，入瓦瓮内，慢火熬成膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
