序号：25577
名称：朱砂丹
组成：朱砂3两，马牙消3两，消石2两。
出处：《圣惠》卷九十五。
主治：None
加减：None
功效：祛热毒风，镇心神，返老驻颜。
用法用量：每日早晨，含化1丸。如要多合，但依分两，酌度修炼为丸。妇人服之亦佳。
制备方法：上为末，入瓷瓶中，以重抄细纸三重，密固瓶口，重汤煮之，常如鱼眼沸，水耗，即以热水添之，不歇火，21日夜满，开瓶子，其消并在瓶四面，收之细研，任服；其朱砂即在中心，取出细研，以小瓷盒子中盛，固济，微火养一日，加火1斤，煅令通赤，放冷，开取细研，以枣肉和，每1两砂，可丸得360丸。
临床应用：None
各家论述：None
用药禁忌：忌羊血。
附注：None
