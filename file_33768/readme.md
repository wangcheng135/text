序号：33777
名称：没药丸
组成：芫花（去枝梗）2两（用好米醋3升煎至1升半，去滓不用，只将醋入石器内，入硇砂霜1两，巴豆肉7粒烂研，入醋内熬成膏，留丸药用），木香1两，没药（别研）1两，当归1两，桂心1两，荜茇1两，槟榔1分，肉豆蔻1枚（炮），斑蝥3枚（去头足翅，糯米炒令焦黄，去米研细），附子1两半（生用，去皮）。
出处：《妇人良方》卷七引《灵苑方》。
主治：五积气癖及惊忤，血积癥癖，血瘕，发歇攻刺疼痛，呕逆噎塞，心中迷闷，不醒人事，及血膨癥瘕胀满，经脉不行者。
加减：None
功效：None
用法用量：初服1丸，用醋炒萝卜子令焦黑，以酒浸，同煎1-2佛，放温吞下，渐加至5-7丸即止；如急卒血气攻心脾，以酒、醋共一银盏，煎沸吞下；妇人血瘕癥癖，结块攻心疼痛闷绝，久医不效者，加禹余粮1两火煅醋淬7次，研细，和药为丸，用苏木节2两细锉，酒3升煎至7合，去滓分为3服，吞药并进，3服当汗出，则瘕随大小肠逐下，其病立愈；人弱者每服只2丸。如是丈夫元脏小肠气、脾积气、癥块等疾，即入丹砂1两细研和停，每服3丸，以生姜盐汤吞下。
制备方法：除斑蝥、没药，余药为细末，与斑蝥、没药合研，入前膏子内为丸，如赤豆丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
