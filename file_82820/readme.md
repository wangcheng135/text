序号：82836
名称：丹参煮散
组成：丹参3两，芎2两，杜仲2两，续断2两，地骨皮2两，当归1两18铢，通草1两18铢，干地黄1两18铢，麦门冬1两18铢，升麻1两18铢，禹余粮1两18铢，麻黄1两18铢，牛膝2两6铢，生姜（切，炒取焦干）2两，牡蛎2两，甘草1两6铢，桂心1两6铢。
出处：《千金》卷十一。
主治：筋实极，则两脚下满而痛，不得远行，脚心如割，筋断折痛不可忍。
加减：None
功效：None
用法用量：以绢袋子盛散2方寸匕，以井花水2升，煮取1升，顿服，1日2次。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：忌海藻、菘菜、生葱、芜荑。
附注：丹参散（《普济方》卷十五）。
