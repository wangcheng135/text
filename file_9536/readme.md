序号：9536
名称：加味乌药顺气散
组成：白芷、桔梗、陈皮、天台乌药、枳壳、茴香、缩砂、天南星、川芎、当归、半夏、南木香、牛膝、木瓜、槟榔、香附子、甘草、萆薢各等分。
出处：《普济方》卷一一六。
主治：男子、妇人36种风，72般气，左瘫右痪，半身不遂，口眼歪斜，腰脚疼痛，及治妇人胎前产后血虚血晕，血气不调，四肢麻痹，忽然手脚不能动之瘫痪，一切血气风，又治男子寒疝，风湿脚气下痈等疾。
加减：None
功效：None
用法用量：每服2-3钱，水1大盏，加生姜3片，大枣2枚，煎至7分，去滓,看病上下服。如妇人患，用好当归服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
