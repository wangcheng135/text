序号：65825
名称：菩萨散
组成：菩萨石1两，金精石1两，银精石1两，太阴石1两，太阳石1两，雨余石1两，河洛石1两，矾矿石1两，云母石1两，炉甘石1两，井泉石1两，白滑石1两，紫英石1两，寒水石1两，阳起石1两，猪牙石1两，代赭石1两，碧霞石1两，鸟鱼骨1两，青盐1两，硇砂半两，蜜陀僧1两，铜青1两，黄丹4两，麝香、脑子1钱，轻粉1钱半，硼砂3钱，乳香2钱，雄胆1斤，白砂蜜2斤。
出处：《宣明论》卷十四。
主治：远年近日，一切眼疾。
加减：None
功效：None
用法用量：以井花水9大碗，熬就作4碗，占水内落下钱许，不大散可，如散者再熬，滤滓，过露旋点。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：菩萨膏（《普济方》卷八十五）。
