序号：20322
名称：肉苁蓉丸
组成：肉苁蓉（酒浸，切，焙）3两，山芋3两，熟干地黄（焙）3两，菟丝子（酒浸，别捣）1两半，五味子1两半，杜仲（去粗皮，炙，锉）1两半，泽泻1两半，覆盆子1两半，山茱萸1两半，远志（去心）1两半，续断1两半，桂（去粗皮）1两半，附子（炮裂，去皮脐）1两半，甘草（炙，锉）1两半，白茯苓（去黑皮）1两半，石斛（去根）1两半，鹿茸（去毛，酥炙）1两半，人参1两半，蛇床子1两半，巴戟天（去心）1两半。
出处：《圣济总录》卷四十三。
主治：瘈病，筋脉相引；及五劳七伤，小便数，腰疼，久立不得，坐即脚痹，腹肚不安。
加减：None
功效：None
用法用量：每服20丸，空心、日午、夜卧温酒送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
