序号：31682
名称：沉香丸
组成：沉香（锉）半两，木香半两，芎半两，白茯苓（去黑皮）半两，槟榔（锉）半两，楝实（炮）半两，白附子半两，人参半两，石斛（去根）半两，牛膝（酒浸，切，焙）半两，补骨脂（炒）半两，附子（炮裂，去皮脐）半两，茴香子（炒）半两，肉苁蓉（酒浸，切，焙）半两，泽泻（锉）半两，青橘皮（去白，焙）半两，白蒺藜（炒）半两，阿魏（醋化，去砂石，面和作饼，炙）半两，硇砂（醋飞）半两，桃仁（去皮尖双仁，炒，研）1两。
出处：《圣济总录》卷八十七。
主治：气劳肢体疼痛，心腹妨闷，减食无力，日渐羸瘦，怠惰呻吟。
加减：None
功效：None
用法用量：每服20丸，食前以温酒或盐汤送下。
制备方法：上为末，和匀，次用木瓜2枚（去皮核）蒸烂，研入众药末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
