序号：38969
名称：泻热方
组成：大黄4两，麻黄4两，黄芩4两，杏仁3两，赤茯苓3两，甘草3两，橘皮3两，芒消3两，泽泻3两。
出处：《千金》卷十五。
主治：有关上脉阴阳俱实者，足太阳与阳明经俱实也。病苦脾胀，腹坚抢胁下痛，胃气不转，大便难，时反泄利，腹中痛，上冲肺肝，动五脏，立喘鸣多惊，身热汗不出，喉痹精少。
加减：None
功效：None
用法用量：上(口父)咀。以水9升，煮取3升，绞去滓，纳大黄，煮两沸，去滓下芒消，分3服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：大黄、芒消泻里热；麻黄、杏仁泻表热；黄芩、甘草泻半表半里热；茯苓、泽泻泻不表不里热，使从水道渗泄；橘皮佐麻黄、杏仁引领里热从元府开发也。此方治热从内发，故可表里兼攻，若非伤寒之邪邪未传里，不可轻用里药之禁。
用药禁忌：None
附注：泻热汤（《圣济总录》卷六十）。
