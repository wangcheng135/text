序号：22261
名称：阳起石丸
组成：阳起石（飞过）1两，远志（去心）2两，山芋2两，巴戟天（去心）2两，附子（炮裂，去皮脐）2两，龙骨（研）1两，肉苁蓉（酒浸，切，焙）4两，蛇床子3两，牛膝（酒浸，切，焙）2两，杜仲（去粗皮，炙）2两，赤石脂2两，牡蛎（爆）2两，石斛（去根）2两半，黄耆（锉）2两半，续断2两半，五味子2两半，菟丝子（酒浸，别捣）2两半，地骨皮2两半，五加皮（锉）2两半，萆薢2两半，卷柏2两半。
出处：《圣济总录》卷八十六。
主治：肾劳虚损，腰脚酸疼，少腹急痛，小便滑数，面色黧黑。
加减：None
功效：None
用法用量：每服20丸，空心、食前以温酒送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
