序号：81494
名称：鳗鲡鱼丸
组成：鳗鲡鱼4两（炙令焦黄），野猪皮4两（炙令焦黄），瞿麦1两，蟾蜍1个（炙黄），斑蝥30个（去头足翅，糯米拌炒，令黄色），腊月猪脂5合（炼成者），巴豆15个（去皮心膜，纸裹压去油）。
出处：《圣惠》卷六十六。
主治：鼠瘘。
加减：None
功效：None
用法用量：每服2丸，空心以米粥饮送下。觉者当寒热，不觉者，来日平明更服3丸，稍稍增之。其虫当从小便中出，盛而视之，乃有百数个便愈也。
制备方法：除猪脂、巴豆外，上为末，用猪脂、巴豆和捣干杵为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：慎热饮食，当有烦闷；忌猪肉、芦笋。
附注：None
