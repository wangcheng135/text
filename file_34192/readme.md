序号：34201
名称：牡丹散
组成：牡丹皮1两，桂（去粗皮）半两，陈橘皮（汤浸去白，焙）3两，芎1两，延胡索半两，木香3分，白术3分，甘草3分（炙），芍药3分，京三棱半两（煨，锉），干姜（炮）半两，诃黎勒皮3分，半夏（汤洗去滑7遍，姜汁炒）半两，羌活（去芦头）1两，枳壳（去瓤麸炒）1两，当归（切，焙）1两半。
出处：《鸡峰》卷十五引《灵苑方》。
主治：妇人血风攻注，头目不利，不思饮食，手足烦热，肢节拘急疼痛，胸膈不利，大肠不调，阴阳相干，心下忪悸，或时旋运。
加减：None
功效：益血海，退血风，消寒痰、实脾胃，理血气。
用法用量：每服3钱匕，水1盏，生姜3片，煎至7分，去滓，食前温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：牡丹汤（《圣济总录》卷一五○）、大效牡丹皮散（《元戎》）、牡丹皮散（《普济方》卷二二九）、大效牡丹散（《准绳·女科》卷五）。
