序号：28939
名称：苏羌达表汤
组成：苏叶1钱半至3钱，防风1钱至1钱半，光杏仁2-3钱，羌活1钱至1钱半，白芷1钱至1钱半，广橘红8分至1钱（极重1钱半），鲜生姜8分至1钱，浙苓皮2-3钱。
出处：《重订通俗伤寒论》。
主治：伤寒挟湿。
加减：如风重于寒者，通称伤风，咳嗽痰多，去羌活、生姜，加仙半夏3钱，前胡2钱，苦桔梗1钱半。
功效：辛温发汗。
用法用量：None
制备方法：None
临床应用：None
各家论述：浙绍卑湿，凡伤寒恒多挟湿，故予于辛温中佐以淡渗者，防其停湿也；湖南高燥，凡伤寒最易化燥，仲景于辛温中佐以甘润者，防其化燥也，辛温发汗法虽同，而佐使之法则异。方以苏叶为君，专为辛散经络之风寒而设；臣以羌活辛散筋骨之风寒，防风、白芷辛散肌肉之风寒；佐以杏、橘轻苦微辛，引领筋骨肌肉之风寒，俾其从皮毛而出；使以姜、苓辛淡发散为阳，深恐其发汗不彻，停水为患也。立法周到，故列为发汗之首剂。
用药禁忌：None
