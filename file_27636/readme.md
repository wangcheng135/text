序号：27644
名称：八将驱邪散
组成：麝香3分，川山甲（炙）1两，蜈蚣（炙，去头足）3钱，土狗（炙）5钱，地龙（去上，炙）5钱，番木鳖（酥炙）5钱，金鼎砒5钱，雄黄5钱。
出处：《解围元薮》卷三。
主治：大风瘫烂败症。
加减：None
功效：None
用法用量：每服3厘，温酒送下，服7日，停7日。服退药1月，又服之。如人素弱，3服3日，就服退药，5日服补药，3日再服之。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：八将追魂丹，三厘散。退药：乳香、没药、血竭、朱砂、当归、元参、胡麻、桑寄生、牛黄、沉香各等分，为末，蜜丸服。补药：桑螵蛸、晚蚕蛾、银柴胡、仙灵脾、牛膝，防己、红花、破故纸，柏子仁、天冬，上为末，蜜丸，梧桐子大。每服五十丸，酒送下，日进二次。
