序号：42522
名称：青囊虎潜酒
组成：虎胫骨1对（羊酥炙），龟版（去墙，醋炙）3两，破故纸1两5钱，牛膝1两5钱，生地1两5钱，骨碎补1两5钱，枸杞子1两5钱，当归3两，川羌活1两，独活1两，川续断1两，桑寄生1两，海风藤1两，红花1两，白茯苓（人乳拌）1两，杜仲1两，川芎7钱，丹参7钱，滴乳香6钱，没药6钱，赤何首乌6钱，小茴香6钱，狗脊（去毛）6钱。
出处：《同寿录》卷一。
主治：风气，三十六种风症。
加减：如人弱者，加人参，不拘多少。
功效：舒筋壮步，止痛荣血。
用法用量：如热天先合半料，将完再合，其药滓第2次只放酒12斤，如前煮埋，饮完再用新药，另制亦可。
制备方法：上为粗片，入绢袋内，浸陈年好酒20斤，封固罐口，隔水煮3炷香，埋土内，二周时开饮随量。
临床应用：None
各家论述：None
用药禁忌：None
附注：虎潜酒（《喻选古方试验》卷三）。
