序号：66711
名称：鹿茸丸
组成：白马蹄5两，蒲黄4两，鹿茸4两，禹余粮4两，白马鬐毛4两，小蓟根4两，白芷4两，续断4两，人参3两，干地黄3两，柏子仁3两，乌贼骨3两，黄耆3两，茯苓3两，当归3两，艾叶2两，苁蓉2两，伏龙肝2两。
出处：方出《千金》卷四，名见《普济方》卷三三○。
主治：妇人血伤不止，兼赤白带下不绝，面黄体瘦，渐成劳疾。
加减：None
功效：None
用法用量：每服20丸，加至40丸，空心饮送服，1日2次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：此方温中散瘀，兼得其奥，允为调适久崩之合剂。
用药禁忌：None
附注：None
