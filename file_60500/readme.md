序号：60515
名称：通命丸
组成：茯苓6分，甘草6分（炙），杏仁6分（去皮尖，熬），牛膝7分，黄芩5分，阿胶3分（炙），防风4分，干天门冬6分（去心），芍药6分，大黄6分，当归6分，干姜6分，干地黄7分，人参6分，桂心3分，干漆4分（熬），紫菀5分，白术4分，苁蓉5分，吴茱萸3分，蜀椒3分（汗），石斛3分。
出处：《外台》卷十七引《古今录验》。
主治：虚劳百病，七伤六极，少气羸弱，不能饮食。
加减：None
功效：None
用法用量：每服7丸，食前服，日3次。不知渐增，以知为度。病剧者，夜更1服。
制备方法：上为末，以枣膏、蜜为丸。
临床应用：None
各家论述：None
用药禁忌：忌芜荑、鲤鱼、生葱、海藻、菘菜、桃、李、雀肉、酢等。
附注：None
