序号：51916
名称：除风解毒汤
组成：二花1两，公英8钱，生地1两，归尾4钱，赤芍3钱，防风3钱，石膏8钱，连翘4钱，牛蒡子3钱（炒），薄荷3钱，菊花4钱，黄芩3钱，甘草1钱。
出处：《眼科临症笔记》。
主治：风赤疮痍症（沙眼胞性湿疹）。初起赤疼，眵多流泪，隐涩羞明，睑肿而痒，重则眼睑内生票疮。
加减：None
功效：None
用法用量：水煎服。外涂三白散。
制备方法：None
临床应用：风赤疮痍症：董某某，男。外感风邪，忽觉头痛目赤，晨起眵多粘连，六脉皆数，关脉略带浮大，两目赤肿，热泪不止。此乃脾经风火上冲于脑。先刺合谷、上星、睛明、太阳，以泻其热；再用三白散涂之，内服除风解毒汤，4剂后，泪止肿消。
各家论述：None
用药禁忌：None
附注：None
