序号：48044
名称：神机万灵膏
组成：真麻油4斤，槐2枝，柳2枝，桃2枝，榴2枝，椿2枝，杏2枝，楮树枝2枝，两头尖2两，白芷2两，赤芍2两，大黄2两，川连2两，人参2两，穿山甲2两，白芍2两，草乌2两，苦参2两，川芎2两，当归2两，杏仁1两，生地1两，川椒1两，胎发1两，槐子1两，黄柏（去皮）1两，熟地1两，巴豆（去皮壳）120粒，木鳖子（去皮壳）50个，蓖麻子仁120个（去皮壳）。
出处：《同寿录》卷四。
主治：五劳七伤；肩背腰膝两足寒湿疼痛，脚气穿心疼痛；男子阳痿不起，阴痿瘦弱，遗精白浊，元气虚冷，女人子宫冷闭，赤白带下；赤白痢疾；痞块；左瘫右痪；偏正头风、头痛；舌胀；酒后呕吐，酒积，转食，暗风；风寒、风热咳嗽，痨病；胸膈不利，气喘不息；妇女月经不通；胎不安；伤寒，走黄结胸，兼赤白痢；打扑血凝；犬咬及蛇蝎伤；痈疽、发背、疔疮，一切无名肿毒初起；干湿疥癣，瘙痒，风疹；癞疮肿痛；疮疖；小儿癖疾等。
加减：None
功效：None
用法用量：用时取少许，隔汤炖化，量大小摊贴。五劳七伤，贴肺俞、肩井并三里、曲池穴，火烘双手熨百余下；肩背腰膝两足寒湿疼痛，脚气穿心疼痛，贴患处；男子阳痿不起，阴痿瘦弱，遗精白浊，元气虚冷，女人子宫冷闭，赤白带下，贴两阴交穴、关元穴；男女赤白痢疾，贴丹田穴，膏内加入捣细木鳖1个；男女痞决，先用面作圈围痞处，圈内入皮消1两，用重纸盖，上以熨斗盛火熨之，令纸热透进，然后去消并面圈，将膏贴患处，火烘双手熨百余下，令汗出，膏内加捣细木鳖1个；左瘫右痪，加捣细木鳖1个，贴丹田穴，仍服此药3丸，好酒送下；偏正头风，头痛，贴脐内；舌胀，贴心中肺俞并心坎下3寸；酒后呕吐，酒积、转食、暗风，贴肺俞兼心坎下2寸许；风寒、风热咳嗽、痨病，贴肺俞穴；胸膈不利，气喘不息，贴肺俞穴；妇女月经不通，贴陶康二穴骨上；胎不安，先将此膏贴脐内，再用一膏加入捣细木鳖1个，贴丹田穴；春三月患伤寒，或已过日期，用此膏贴脐上心坎下，如未过日期，用此膏2两半，贴脐中，手熨令汗出；夏三月伤寒，走黄结胸，用此膏2两，贴心坎下；秋三月伤寒，兼赤白痢，用此膏2两，贴脐中；冬三月伤寒，兼赤白痢，用此膏2两半，贴脐中；打扑血凝，贴疼处，手熨热即止；犬咬及蛇、蝎伤，贴伤处，不必熨；痈疽，发背、疔疮，一切无名肿毒，初起1-2日内，贴患处，手熨出汗即消，若4-5日肿硬有脓，亦以此贴之，易于出脓收口；干湿疥癣、瘙痒、风疹，贴脐中，手熨出汗即安；癞疮肿痛，膏内加捣细木鳖1个，贴脐中，手熨出汗即愈；凡疮疖，随大小贴之；小儿癖疾，以此贴患处，手熨觉腹热即止，或贴脐上亦可；此膏能治万病，皆对患处及脐中贴之，无不应验，贴后火烘双手熨百余下更妙。
制备方法：上锉，入油锅内浸，春5日，夏3日，秋7日，冬10日浸足，然后入铜锅熬煎，以药枯焦为度，起锅候冷，生绢滤去滓净，再将药油入锅熬煎，用槐柳枝不住手搅，加入黄丹（水飞净，火焙7次，燥者）2斤，于油内慢火熬，滴水成珠为度，再加入净明黄松香12两，研末搅匀，取起锅片时，减火性，乃下真阿魏1两（试法：将阿魏搽在铜器上，次日看铜色变白者真），沉香、丁香、麝香、广木香、血竭各1两，又乳香、没药各3两（出汗），共研极细末，入油内搅极匀，用凉水1大桶，将药投入水中，1日1换，浸7日夜，拔去火性，收入瓷瓶内。
