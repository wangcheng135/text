序号：84339
名称：五饮主方
组成：茯苓3钱，制半夏2钱，广橘红5钱，车前子1钱，嫩桑皮1钱，前胡5分，生姜2片。
出处：《活人方》卷四。
主治：五种水饮。
加减：溢饮，加白术1钱5分，更加枳实，去前胡；悬饮，加天麻2钱，佐荆芥、甘菊，去桑皮；在下部，加泽泻1钱，车前子5分，苡仁2钱，去前胡；支饮，加前胡1钱，柴胡1钱，白芥子1钱；伏饮者，加前胡1钱，羌活1钱；留饮，加白术2钱，苍术1钱；病久元气衰惫，形神虚萎，饮食不甘，脾胃泄泻，必须加人参2钱，白术3钱以培元气，兼于治饮；倘下元虚冷，则桂、附、干姜亦所不免。
功效：None
用法用量：水煎，午前、午后服。
制备方法：None
临床应用：None
各家论述：饮邪水类，茯苓浚水而能降气，故为君；半夏燥湿，利痰为臣；橘红佐桑皮调和脾肺之气，使有所统运；车前佐茯苓分利水道，所谓引而决之；前胡之辛，消痰下气，生姜温能治呕，辛能豁痰，此谓通治痰饮之方。
用药禁忌：None
附注：None
