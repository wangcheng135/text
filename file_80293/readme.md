序号：80309
名称：鹭鹚喀丸
组成：杏仁（去皮尖）2两，牛蒡子（炒）1两，生石膏2两，栀子（炒）2两，天花粉2两，紫苏（炒）2两，甘草4钱，瓜蒌皮2两，麻黄4钱，青黛1两，蛤壳2两，射干1两，白芥子（炒）4钱，细辛2钱（以上14味，计18两4钱，共研为细粉，过罗，每18两4钱细粉总之），龙延香5分，麝香2分。
出处：《北京市中药成方选集》。
主治：小儿咳嗽不已，连作数十声，甚则呛血，音哑，面浮肿。
加减：None
功效：宣解肺热，止嗽化痰。
用法用量：每服1丸，日服2次，温开水或梨汤送下。
制备方法：上药研细，混合均匀，炼蜜为丸，重4分，金衣36开，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：鹭鹚咳丸（《中药制剂手山》）。
