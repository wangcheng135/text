序号：32345
名称：妙应丸
组成：当归（去芦头）2两，延胡索（去土）1两，泽兰（取叶）1两半，白芍药（雪白者）1两，肉桂（去粗皮）1两（不见火），牡丹皮（去骨）1两，川芎1两，木香半两（不见火），石斛（去苗）1两，川姜1两（炮），熟干地黄（净洗，薄切，焙干）2两。
出处：《洪氏集验方》卷三引徐学谕方。
主治：妇人血经不调，月事湛浊，加之惊忧，遂发痎疟，汗出过多，损耗气血。
加减：None
功效：滋暖子宫，调血养气。
用法用量：每服20粒，空心及晚食前温酒或米饮送下。
制备方法：上药焙干为末，醋煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
