序号：74516
名称：摄生饮
组成：圆白南星（湿纸裹煨）1钱半，南木香1钱半，半夏（用白沸汤就铫蘸少顷）1钱半，辣细辛1钱，苍术（生）1钱，甘草（生）1钱，细节1钱，石菖蒲1钱。
出处：《直指》卷三。
主治：一切卒中。中风、中寒、中暑、中湿及痰厥、饮厥、气厥等。
加减：痰盛，加炙全蝎2枚。
功效：None
用法用量：每服以水盏半，加生姜七厚片，煎取其半，乘热调苏合香丸3丸。先以皂角肉为细末，管子揭些吹入鼻中，候喷嚏即灌药。牙噤者，用乌梅肉揉和南星、细辛末，以中指频擦自开。
制备方法：上锉，分2服。
临床应用：None
各家论述：None
用药禁忌：None
附注：《育婴秘诀》治大人、小儿卒中，以本方调惊气丸服。
