序号：4056
名称：木瓜丸
组成：木瓜7枚（大者，切头上1片为盖子，剜去瓤并皮子，入硫黄、青盐在内），硫黄（细研，水飞过）2两，青盐（细研）2两，木香1两，槟榔1两，肉豆蔻（去壳）1两，诃黎勒皮1两，桂心1两，白芍药半两，当归（锉，微炒）半两，胡椒半两，荜茇半两，草豆蔻（去皮）半两。
出处：《圣惠》卷九十八。
主治：脾胃积冷，腹胁疼痛，宿食不消，两脚转筋，时复泻痢。
加减：None
功效：None
用法用量：每服20丸，空心以温酒送下。
制备方法：上为末，入于木瓜中令尽，以盖子盖之，用竹签子劄定，以三五重纸裹木瓜，于饭甑内蒸令烂熟，研如膏，候可丸即丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
