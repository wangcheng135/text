序号：60463
名称：通灵丸
组成：荆三棱1两，酸石榴（大者）2枚，杏仁1两，苦葶苈1两，甘遂1两，大戟1两，大黄1两，巴豆1两，芫花1两，五灵脂1两，盐豉2两，乌梅2两。
出处：《博济》卷三。
主治：久患癖块，或因气不和，即发疼痛，胸多痞塞。
加减：None
功效：消化痰癥。
用法用量：每服3-5丸，姜汤送下；心胸痞闷疼，橘皮汤送下；吃酒食饱闷，生姜汤或茶汤、温水送下亦可。
制备方法：上锉，用水1斗2升，入药于锅内同熬，候水尽，就锅内炒令黄焦色止，取出，杵罗为末，更入木香、青橘末各1两，拌匀，醋煮面糊为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中荆三棱用量原缺。
