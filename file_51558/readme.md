序号：51573
名称：追命散
组成：半两钱40-50文（火煅通赤，淬酽醋中不计次数，于醋底淘取淬下碎铜末研之，粗碍乳锤者去之，别以水淘，澄取如粉者，纸上渗干）2钱1字，巴豆（去皮壳心膜）3钱半（用酸浆水1盏，煮至水欲尽，焙干，研如泥），大黄（绵纹紧实者）8钱（用小便浸7日，每日1换，日足，湿纸裹煨熟，薄切，焙干为末）3钱半，羊胫炭（即炭中圆细紧实如羊胫骨者，取3-4寸，却作十余段，别以着炭同烧通红，淬入醇酒中，如是7遍，烘干为末）半两。
出处：《卫生家宝》卷七。
主治：妇人血癥积聚疼痛，渐至经候隔绝消失、痨瘵；产妇危恶变证，胎死上冲，闷运欲绝，及产后血晕；打扑伤损，筋断骨折及破伤风。诸证人气已死，但心头尚暖者。
加减：None
功效：None
用法用量：每1服1字至半钱，浓煎当归酒和小便调下。伤折即全用酒，多饮不妨；若产妇血未定，及素饮酒人，即少借酒力行药，多以童便下之可也。
制备方法：上合和为散，于瓷罐子实筑，腊纸密封，收高处。
临床应用：None
各家论述：None
用药禁忌：None
