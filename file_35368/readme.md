序号：35378
名称：京三棱丸
组成：京三棱（煨，乘热捶碎，别捣为末）半两，木香半两，当归（切，焙）半两，桂（去粗皮）半两，肉苁蓉（酒浸，切，焙）半两，牛膝（去苗，酒浸，切，焙）半两，羌活（去芦头）半两，芎半两，赤芍药（锉）半两，防风（去叉）半两，枳壳（去瓤，麸炒）半两，白术半两，槟榔（生，锉）1两，大黄（锉，炒）1两，郁李仁（去皮双仁，别研如膏）1两。
出处：《圣济总录》卷九十七。
主治：痃癖，注气刺痛，大便秘涩。
加减：None
功效：None
用法用量：每服20丸，空腹米饮或温酒送下。以利为度。
制备方法：上除郁李仁外，捣罗为末，与郁李仁膏同研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
