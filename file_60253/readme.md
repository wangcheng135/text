序号：60268
名称：通化汤
组成：石菖蒲10g，连翘10g，竹茹10g，薄荷10g，橘红10g，黄连8g，桔梗8g，僵蚕10g，甘草10g，蜈蚣3条（研碎冲服），全蝎3g（研碎冲服）。
出处：《千家妙方》引高乐众方。
主治：脑外伤后综合征，辨证属肝郁风动，心火炽盛，风火痰扰者。
加减：None
功效：凉肝清心，祛风消痰。
用法用量：水煎服，每日1剂。
制备方法：None
临床应用：脑外伤后综合征《千家妙方》：刘某某，女，22岁，于1976年5月4日来诊。患者语謇，其父代述病情，称其女于春季大队建房当小工。坠砖击头而晕倒，经抢救回苏后即言语不清，头晕目眩，时如昏迷，四肢痿软，握物无力，行路蹒跚，三五步即摇摇欲跌，曾转治几个医院，皆诊为脑外伤后综合征，治疗四个月，未见好转。检查：体弱神疲，坐须人扶，面红目赤，问而不答，痴呆之象。脉细弦数，治用通化汤加川芎10g，水煎服。共服20剂，症除病愈。随访一年，疗效稳固。
