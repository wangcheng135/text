序号：49563
名称：茯苓丸
组成：茯苓8分，甘草7分（炙），杏仁50枚，人参7分，厚朴5分（炙），干姜7分，黄耆6分，桂心4分，当归8分，芎藭5分，干地黄8分。
出处：《外台》卷六引《删繁方》。
主治：下焦虚寒损，腹中瘀血，令人喜忘，不欲闻人声，胸中气塞而短气。
加减：None
功效：None
用法用量：初服20丸，加至30丸，清白饮送下，日2次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：下焦真阳亏损，则胸中大气不布，而致血涩不调，瘀滞腹内，故需辛温攻补兼施，方克有济。盖参、耆、甘草不得厚朴、杏仁之宣散则滞而不行；芎、归、地黄不得姜、桂之破结，则瘀而不化；茯苓一味，不独治畜血喜忘，并守五脏正气也。
用药禁忌：忌海藻、菘菜、生葱、酢物、芜荑。
附注：None
