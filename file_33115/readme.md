序号：33124
名称：万应清凉膏
组成：木鳖2两，蓖麻子2两，当归2两，生地2两，苦参2两，苍耳子2两，生大黄1两，黄芩1两，黄柏1两，赤芍1两，玄参1两，天花粉1两，桃仁1两，白芷1两，角刺1两，川山甲5钱，直僵蚕5钱，全蝎5钱，黄蜂房5钱，甘草8钱，槐枝2两，虾蟆14只。
出处：《外科方外奇方》卷二。
主治：外科一切大小疽毒。
加减：None
功效：提毒生肌长肉。
用法用量：摊贴。
制备方法：用麻油7斤，入前药浸，春5、夏3、秋7、冬10日，入锅熬至药枯，去渣滤净，复入锅内，武火熬至滴水成珠为度。称净油1斤，入炒黄铅粉8两研细，徐徐搅入，俟白烟起，倾井水内七日，出火气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
