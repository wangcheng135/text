序号：30471
名称：进食散
组成：青皮1分，陈皮（去瓤）1分，草豆蔻3个，甘草1分（炙），诃子（去核，煨）5个，高良姜（薄切，炒）1分，川乌头1个（炮，去皮脐），肉桂1分（去外皮）。
出处：《苏沈良方》卷四引李潜方。
主治：脾胃虚冷，不思食，及久病人脾虚，全不食者。
加减：None
功效：None
用法用量：每服1钱，水1中盏，加生姜2片，煎至7分，空腹时服。
制备方法：None
临床应用：脾虚不食：李潜在真州治贾使君女子，已五十余日，病脾多呕，都不进食，医绝无验。潜投此药一服，遂食蒸饼半枚，明日百味皆思。潜云：此药进食极神速。予疑此药太热，潜云不然，用之三十年，无不效者。
各家论述：None
用药禁忌：None
附注：理脾散（《普济方》卷二十三引《如宜方》）。方中草豆蔻，《局方》（淳祜新添方）作“草果肉”。
