序号：23805
名称：孙真人活命神丹
组成：麝香1钱，月石（净末）3分，冰片1钱，山豆根（净末）5分，蟾酥（不见火，晒干，净末）1钱，老生姜（取汁澄粉）3分，新江子仁（去净油）1钱，大干地龙（去泥）2条。
出处：《喉科紫珍集》卷上。
主治：喉风，喉痹，双单喉蛾。
加减：None
功效：None
用法用量：临时用小红枣1枚，去蒂去核，取核只开近蒂半截，免走药性，入药黄豆大，将枣开蒂孔一头，塞入鼻中，令病人闭口目。避风少顷，即能得涎嚏或出脓，以银花甘草汤漱之，喉中便觉通快，俟鼻内热时，即将药枣拿去。病甚者，再换药枣1枚。凡左蛾塞左，右蛾塞右，双蛾左右先后塞之，唯喉风喉痹，男左女右塞之。
制备方法：上为极细末。合匀，瓷瓶收贮，蜡口封固。
临床应用：None
各家论述：None
用药禁忌：阴虚喉痛、虚人、孕妇忌用。
附注：None
