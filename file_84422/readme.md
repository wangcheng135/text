序号：84440
名称：七制固脂丸
组成：固脂10斤（一制淘米水浸一夜，晒七日；二制用黄柏2斤，熬浓汁，泡一夜，晒七日；三制杜仲、四制生盐、五制鱼膘、六制核桃肉，俱照前；七制黑枣、糯米共煮粥，将固脂磨细末，和匀，捣融）。
出处：《验方新编》卷十一。
主治：命门火亏，下元虚损，耳聋眼花，腰痛腿软，肾冷精流，阳萎不举，小便过多，筋骨疼痛，肚腹畏寒，脾胃虚弱，饮食难消，夜多盗汗，精神疲倦，时爱躺卧。
加减：None
功效：补火壮阳，固精种子，保真元，壮筋骨，健脾胃，长精神，除疾病。
用法用量：每早空心服1钱，淡盐水送下。每服至一月后，加3分，加至2钱为止。若阴虚水亏者，早服此药，晚服六味地黄丸，半年后方效。
制备方法：为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌食羊血，油菜（又名芸苔）、菜油。忌铁器。
附注：None
