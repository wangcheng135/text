序号：77713
名称：雄黄膏
组成：雄黄3分（细研），当归3分，桂心3分，白芷半两，赤芍药半两，甘草3分，附子3分（生去皮脐），黄耆3分，枳壳3分，吴茱萸半两，白术半两，独活半两，槟榔3分，麝香半两（细研），乳香半两，突厥白3分，木鳖子半两（去壳），云母粉3分，松脂3分，白蜡2两，垂柳枝1两，槐枝1两，白檀香半两，零陵香半两，甘松香半两，黄丹10两，麻油。
出处：《圣惠》卷六十三。
主治：一切发背，乳痈恶疮，骨疽穿漏。
加减：None
功效：收毒止痛生肌。
用法用量：以蜡纸上摊贴，每日早晚换之。
制备方法：先将油于铛中，以炭火炼熟，下甘松、零陵、檀香、槐、柳枝等，以慢火煎令槐、柳黑色，即去之。细锉诸药，以酒半升，拌药1宿，后入油中煎，白芷色赤，以绵滤过，拭铛令净，都倾入铛内，下黄丹，于火上煎，变色黑，不住手搅三二十遍，有油泡子飞，即膏成，入雄黄、麝香搅令匀，安瓷盒内盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
