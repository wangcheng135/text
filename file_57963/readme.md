序号：57978
名称：益气保生丸
组成：真神曲（系6月6日造者良，研细，用黑羊肉2斤，煮烂取计，去滓，炒曲文武火焙干入药）1斤，当归（酒洗）4两，白芍（乳蒸）1两，川芎（童便、葱汁炒）3两，生地（酒炒）1两，白术（土炒）2两，茯苓（乳蒸）3两，陈皮（童便炒）2两，远志（猪心血炒）2两，车前子（盐水炒）2两，黄耆（蜜炙）3两，秦艽（蜜炙）3两，怀膝（酒炒）3两，杜仲（盐水炒）3两，粉葛（醋炒）1两，麻仁（乳蒸，童便炒）3两，黄芩（酒炒）3两，天冬（童便炒）2两，肉苁蓉（去甲，童便炒）3两，金箔（擂细和匀）300张。
出处：《点点经》卷一。
主治：伤酒。
加减：None
功效：调补。
用法用量：每服3钱，早晨开水送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
