序号：75110
名称：蒴藋蒸汤
组成：蒴藋根叶（切）3升，菖蒲叶（切）2升，桃叶皮枝（锉）3升，细糠1斗，秫米3升。
出处：《千金》卷十八。
主治：皮虚，主大肠病，寒气关格；皮肤一切劳冷。
加减：None
功效：None
用法用量：上以水1石5斗，煮取米熟为度，大盆器贮之，于盆上作小竹床子罩盆，人身坐床中，四面周围将蓆荐障风，身上以衣被盖覆。若气急时，开孔对中泄气，取通身接汗，可得两食久许，如此3日，蒸还，温药足汁用之，若盆里不过热，盆下安炭火。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：蒴藋治寒痹拘急，菖蒲通利九窍，桃叶辟邪散血，糠米蒸发肉腠，共襄作汗之功也。
用药禁忌：None
附注：None
