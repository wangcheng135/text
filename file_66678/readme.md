序号：66694
名称：鹿茸丸
组成：鹿茸（去毛，酥炙）3分，白薇（去苗）3分，覆盆子3分，细辛（去苗叶）3分，菴（艹闾）子3分，熟干地黄（焙）3分，山芋3分，蛇床子（炒）3分，白茯苓（去黑皮）3分，干姜（炮）半两，远志（去心）半两，当归（切，焙）半两，芎半两，桂（去粗皮）半两，续断半两，牡丹皮半两，人参半两，卷柏半两，龙骨半两，蒲黄半两。
出处：《圣济总录》卷一五三。
主治：妇人血伤带下，渐成劳疾。
加减：None
功效：None
用法用量：每服30丸，空腹温酒送下；米饮亦得。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
