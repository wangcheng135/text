序号：4147
名称：木防己加茯苓芒消汤
组成：木防己2两，桂枝2两，人参4两，芒消3合，茯苓4两。
出处：《金匮》卷中。
主治：膈间支饮，其人喘满，心下痞坚，面色黧黑，其脉沉紧，得之数10日，医吐下之不愈，用木防己汤后三日复发。心下痞坚而悸。
加减：None
功效：None
用法用量：以水6升，煮取2升，去滓，纳芒消，再微煎，分2次温服。微利则愈。
制备方法：None
临床应用：None
各家论述：《法律》：木防己味辛温，能散留饮结气，又主肺气喘满；石膏辛甘微寒，主心下逆气，清肺定喘；人参甘美，治喘消膈饮，补心肺不足；桂枝辛热，通血脉，开结气，宣导诸气。在气分，服之即愈。若饮在血分，深连下焦，必愈而复发，故去石膏气分之药，加芒消入阴分，开痞结，消血。石膏与茯苓，去心下坚，且伐肾邪也。
用药禁忌：None
附注：木防己汤去石膏加茯苓芒消汤(原书卷中)、防己加茯苓芒消汤(《医醇剩义》卷三)。
