序号：36637
名称：和解汤
组成：羌活1两，防风1两，川芎1两，人参1两，干葛半两，川升麻（轻者）半两，甘草（微炙）半两。
出处：《百一》卷十九。
主治：小儿四时感冒寒邪，壮热烦躁，鼻塞多涕，惊悸自汗，肢节疼痛，及麸疮、痘疮已发或未发者。痘出充满红活，而热不退者；寒战咬牙，属热者；不泻而渴，属实者。
加减：None
功效：解表和中。
用法用量：3岁儿每服1钱，水3分盏，加生姜半片，枣子少许，同煎至2分，去滓，不拘时候服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：和解散（《朱氏集验方》卷十一）、和劝汤（《疡医大全》卷三十三）《婴童百问》有芍药半两。
