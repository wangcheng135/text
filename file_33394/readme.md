序号：33403
名称：羌活膏
组成：羌活1分，独活（各去芦头）1分，人参（切）1分，白茯苓（去皮，切）1分，天麻（炙）1分，干蝎1分，青黛（研）1分，龙脑半钱，麝香半钱（研），水银1钱，硫黄1钱（结砂子）（1方有丁香，无龙脑、青黛）。
出处：《普济方》卷三九五。
主治：小儿吐利不止，烦渴闷乱，欲成脾风，手足微搐，但非时发热，不能辨认证候；小儿因惊发热，涎嗽，累经利动；或因伤乳食，吐泻后，气虚弱，精神昏倦，减乳食，手足厥冷，脉息微细，渐成慢惊。
加减：None
功效：None
用法用量：5-7岁每服3饼，2-3岁2饼，1岁半饼至1饼。如发热，煎荆芥汤下，或乳香汤下；手足厥冷，人参姜汤下。冬末春初最宜频服。
制备方法：上为末，炼蜜为丸，如皂子大；捻作饼子。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
