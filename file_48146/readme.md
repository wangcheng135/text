序号：48161
名称：神应散
组成：肥皂核（烧存性）5钱（另研，此味万不可少），荆芥穗1两，北防风1两，何首乌1两，天花粉1两，嫩苦参1两，白当归3钱，白鲜皮3钱，金银花5钱，薄荷叶5钱，白蒺藜3钱，净连翘3钱，粉甘草2钱。
出处：《外科十三方考》下编。
主治：杨毒疮。
加减：如兼有筋骨疼痛者，可酌加威灵仙、木瓜、苡仁、苍术等同为末。
功效：None
用法用量：每日用新鲜白土苓8两，雄猪肉1斤（精肉宜多），水数大碗，再入前药末5分，肥皂核末子1分，煮烂，滤去滓，其肉听食，其汤则代茶饮，不过10日，即可痊愈。如善肉食者，可作大量予之。善后用阴八味或二妙地黄汤收功，且保永无后患。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
