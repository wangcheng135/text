序号：65382
名称：子龙丸
组成：白蔻仁3两，川厚朴4两，制甘遂2两，红芽大戟2两，白芥子4两。
出处：《应验简便良方》卷下。
主治：颈项、胸胁、背、腰、筋骨牵引钩痛，流走不定，手足冷木，气脉不通，痰涎在胸膈上。喉中结气似若梅核，时有时无，冲喉闷绝，又遍身或起筋块如擂如栗，皮色不变，不疼不痛，但觉酸麻，或自溃串烂，流水如涎，经年不愈，有若管漏；又治瘰疬、鱼口、便毒、贴骨、一切阴疽。
加减：None
功效：None
用法用量：每服3分，淡姜汤送下。
制备方法：上药各为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：此乃治痰之本，痰之本水也，湿也，湿气与火则结为痰。大戟能泄脏腑水湿；甘遂能行经络水气，直达水气结聚之处以攻决；白芥子能散皮内膜外痰气，厚朴涤气温中，能去瘀生新；白蔻仁开胃健脾，温中顺气，惟善用者能获神效也。
用药禁忌：同日忌服甘草，因丸内有甘遂故也。
附注：None
