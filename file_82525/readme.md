序号：82541
名称：磨脾丸
组成：补骨脂6两（炒令黄焦），京三棱1片（大者，炮，细碾罗取粉4两用），荜澄茄6两，黑豆1斤（汤去皮，焙干，炒令黄熟，细碾取粉4两用），槟榔4两，木香4两。
出处：《圣济总录》卷四十四。
主治：三焦气不升降，脾脏衰弱，胃气虚满，不思饮食，旧谷未消，新谷又入，脾胃气弱，不能磨化，谷气减耗，肌肉瘦瘁，面目萎黄，寒湿结瘀，饮气下流，渍伤肝肾，足胫虚浮，怠情嗜卧，四肢不收。
加减：None
功效：剖判清浊，交通上下，使脾气和实而能磨化水谷。
用法用量：每服20丸，温米饮送下，不拘时候。
制备方法：上为末，水煮面糊为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
