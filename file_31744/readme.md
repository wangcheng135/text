序号：31753
名称：羌活汤
组成：羌活3两，桂心3两，芍药3两，葛根3两，麻黄3两，干地黄3两，甘草2两，生姜5两。
出处：《千金》卷八。
主治：中风身体疼痛，四肢缓弱不遂，及产后中风。
加减：None
功效：None
用法用量：上(口父)咀。以清酒3升，水5升，煮取3升，温服5合，每日3次。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：此葛根汤之变法，故下卷又名葛根汤。本太阳、阳明开肌表药方中，独去大枣，加羌活以治身体之痛，地黄以和经脉之血，即防风汤中知母之意，妙在地黄之滋降以制麻、葛之升发也。
用药禁忌：忌生冷、油腻、猪、鸡、鱼等。
附注：葛根汤（原书同卷）、羌活散（《圣惠》卷二十二）。
