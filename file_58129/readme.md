序号：58144
名称：益荣汤
组成：当归（去芦，酒浸）1两，黄耆（去芦）1两，小草1两，酸枣仁（炒，去壳）1两，柏子仁（炒）1两，麦门冬（去心）1两，茯神（去木）1两，白芍药1两，紫石英（细研）1两，木香（不见火）半两，人参半两，甘草（炙）半两。
出处：《医方类聚》卷一五八引《济生》。
主治：思虑过度，耗伤心血，心帝无辅，怔忡恍惚，善悲忧，少颜色，夜多不寐，小便或浊。
加减：None
功效：None
用法用量：上（口父）咀。每服4钱，水1盏半，加生姜5片，大枣1个，煎至7分，去滓温服，不拘时候。
