序号：83267
名称：丁香散
组成：丁香半两，半夏半两（汤洗七遍去滑），人参3分（去芦头），甘草1分（炙微赤，锉），柴胡3分（去苗），陈橘皮3分（汤浸，去白瓤，焙），干木爪1两，厚朴2两（去粗皮，涂生姜汁，炙令香熟），白豆蔻3分（去皮），诃黎勒1两（煨，用皮），附子1两（炮裂，去皮脐），高良姜3分（锉）。
出处：《圣惠》卷五。
主治：脾胃气虚，积有冷气，食不消化，面色萎黄，四肢无力，或时吐逆。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，大枣3个，煎至6分，去滓稍热服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：忌生冷，油腻，湿面，饴糖。
附注：None
