序号：76213
名称：舒肝和胃汤
组成：旋覆花3钱，生赭石3钱，藿香3钱，佩兰3钱，焦白术3钱，酒芩3钱，白芍4钱，当归3钱，草蔻2钱，香附3钱。
出处：《中医原著选读》引关幼波方。
主治：慢性、迁延性肝炎，转氨酶长期不降，证属肝胃不和型者。症见恶心欲吐，呃逆嗳气，食后胃脘胀闷，有时胃痛，两胁窜痛，厌油，大便时干时稀，脉弦滑，舌苔白或黄。
加减：若脘腹胀甚者，加沉香末5分（分次冲服）；两胁窜痛甚者，加木瓜4钱，延胡索3钱；大便于者，加瓜蒌4钱，焦四仙1两；纳少体虚，加党参5钱，云苓4钱。
功效：舒肝和胃，养血柔肝。
