序号：58079
名称：益阴固本丸
组成：熟地4两，山萸肉2两，丹皮2两，茯苓4两，白术2两（土炒），菟丝子2两，黄连5分，肉桂3分，芡实2两，金石斛5钱，牡蛎8钱（煅），莲须2两，杭芍5钱，淮山药4两（炒），麦冬8钱（去心）。
出处：《慈禧光绪医方选议》。
主治：时常滑精，心烦躁汗，夜寐不实，气短懒言，饮食减少。
加减：None
功效：滋补肾阴，收涩固精，交通心肾，兼顺中州。
用法用量：每服3钱，淡盐汤送下。
制备方法：上为细末，炼蜜为丸，如绿豆大。
临床应用：None
各家论述：本方亦宗六味地黄汤，去泽泻之通利，重在滋补肾阴，并仿金锁固精丸意小有加减，旨在收涩固精，合交泰丸以交通心肾，治其怔忡，另加健脾之品兼顾中州。
用药禁忌：None
附注：None
