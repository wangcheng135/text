序号：63167
名称：大温经汤
组成：人参5分，白术（土炒）5分，当归8分，白芍7分，川芎5分，熟地5分，砂仁（炒）4分，小茴4分，茯苓5分，甘草3分，香附8分（童便制），陈皮（炒）4分，沉香3分（另研），吴茱萸（炮）5分，玄胡索（炒）5分，鹿茸（酒炙）5分。
出处：《古今医鉴》卷十一。
主治：妇人气血虚弱，经水不调，或赤白带下，或如梅汁淋沥，或成片，有隔2-3个月，渐生潮热，饮食少进；四肢倦怠，日久生骨蒸，即成劳疾。血海虚寒，少腹冷痛。
加减：汗出不止，加黄耆、酸枣仁（炒）各4分：潮热。加柴胡，黄芩各5分；咳嗽，加杏仁、桔梗、五味子、半夏。
功效：None
用法用量：加生姜，水煎服。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：《寿世保元》有黄耆（蜜炒）五分、阿胶（炒）、肉桂各三分。
