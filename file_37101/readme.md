序号：37111
名称：定志丸
组成：菖蒲2分，远志（去心）2分，茯苓2分，人参3两。
出处：《古今录验》引陈明方（见《外台》卷十五）。
主治：心气不定，五脏不足，甚者忧愁悲伤不乐，忽忽喜忘，朝愈暮剧，或暮愈朝发，发则狂眩。能近视，不能远视。
加减：None
功效：益心强志，令人不忘。
用法用量：每服6-7丸，1日3次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《医方集解》：此手少阴药也。人参补心气，菖蒲开心窍，茯苓能交心气于肾，远志能通肾气手心。心属离火，火旺则光能及远也。
用药禁忌：忌酢物、羊肉、饧。
附注：开心丸（《医心方》卷二十六引《医门方》）、远志丸（《扁鹊心书·神方》）。
