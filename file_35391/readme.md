序号：35401
名称：万病无忧膏
组成：川乌6钱，草乌6钱，大黄6钱，当归8钱，赤芍8钱，白芷8钱，连翘8钱，白蔹8钱，白及8钱，乌药8钱，官桂8钱，木鳖子8钱，槐枝4钱，桃枝4钱，柳枝4钱，桑枝4钱，枣枝4钱，苦参5钱，皂角5钱（一方加苏合香2钱）。
出处：《回春》卷八。
主治：None
加减：None
功效：止痛箍脓，长肉生肌。
用法用量：治跌扑闪挫伤损，一切疼痛，心腹痛，俱贴患处；哮吼喘嗽，贴背心：泻痢，贴脐上；头痛、眼痛，贴太阳穴；无名肿毒、痈疽发背、疔疮疖毒、流注湿毒、臁疮，初觉痛痒，便贴患处即消。
制备方法：上锉，用真香油2斤浸药1宿，用火熬至药焦色，以生绢滤去滓不用，将油再熬一滚，入飞过黄丹12两炒过，陆续下，槐柳棍搅不住手，滴水成珠为度。离火，次入乳香、没药末各4钱，搅匀收贮，退火毒听用。
临床应用：None
各家论述：None
用药禁忌：None
附注：万应膏（《奇方类编》卷下）。
