序号：23887
名称：朱砂膏
组成：朱砂1两（别研细），金末1分（用箔子研），牛黄半两，麝香半两，生脑子半两，硼砂半两，生犀1两，玳瑁1两，真珠末1两（蚌末不可用），琥珀（别研）半两，羚羊角半两，苏合香（用油和药亦可）1分，铁液粉1分，安息香半两（酒蒸，去砂石，别研入药），人参1两，远志（去心）半两，茯苓半两，甘草1两（微炙）。
出处：《苏沈良方》卷五。
主治：虚损嗽血等疾。
加减：None
功效：镇心安神，解热。
用法用量：每用1皂子大，食后含化。
制备方法：上为细末，拌和炼蜜，破苏合油，剂诸药为小锭子，更以金箔裹，瓷器内密封。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
