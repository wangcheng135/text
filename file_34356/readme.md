序号：34365
名称：万金散
组成：石斛1两半，防风1两半，巴戟天1两半，天雄1两半（炮，去皮），干地黄1两半，石南1两半，远志1两半（去心），踯躅乌头1两半（炮去皮），干姜1两，桂心1两半，蜀椒半升（汗，去目、闭口者），瞿麦1两，茵陈1两，秦艽1两，茵芋1两，黄耆1两，蔷薇1两，独活1两，细辛1两，牛膝1两，柏子半两，泽泻半两，杜仲半两（炙），山茱萸3分，通草3分，甘草3分。
出处：《千金翼》卷十六。
主治：头痛眩乱耳聋，两目泪出，鼻不闻香臭，口烂恶疮，鼠漏瘰疬，喉咽生疮，烦热咳嗽，胸满脚肿，半身偏枯不遂，手足筋急缓不能屈伸，贼风猥退，飞尸蛊疰，江南_x000D_
恶气在人心下，或在膏育，游走四肢，针灸不及，积聚僻戾，五缓六急，湿痹，女人带下积聚，生产中风，男女五劳七伤。
加减：None
功效：None
用法用量：每用5分匕，冷酒调服，鸡未鸣时1次，白天3次。加至1匕。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
