序号：77138
名称：黑虎丹
组成：当门子1钱，大冰片1钱，公丁香（微焙，研）1钱，母丁香（微焙，研）1钱，全蝎（微焙，研）7个，蜈蚣（微焙，研）7条，僵蚕（微焙，研）7条，穿山甲7片，灵磁石（生研）1钱半，大蜘蛛（微焙，研）7个（各品生更妙）。
出处：《卫生鸿宝》卷二。
主治：眼目赤肿涩痛，内外翳障，眼丹，偷针眼，烂眩风眼，拳毛倒睫，眼皮外翻，眼珠突出，雀目，视物反常，视物倒植，瞳神反背，眼菌，眼疮，眼漏，通睛，疳眼。痈疽肿痛。
加减：如火证，加犀黄5分，珍珠5分。
功效：拔毒长肉。
用法用量：一切外证掺上膏药贴之，并掺眼目。
制备方法：上为极细末，和匀再研，瓷瓶收贮，勿令出气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
