序号：49083
名称：胎产金丹
组成：党参6两，炙黄耆5两，炒于术6两，茯苓6两，炙甘草4两，熟地20两，炒白芍4两，炒当归8两，炒川芎2两，炒川断3两，炒杜仲3两，炒淮山药2两，炒萸肉3两，盐水炒菟丝子3两，紫河车5两，蛤粉炒阿胶4两，杞子2两，炒丹皮2两，盐水炒黄柏2两，炒椿根皮3两，炙乌贼骨3两，沉香1两，肉桂1两，炮姜炭1两，炒荆芥2两5钱，炒艾叶3两，制没药2两，制香附5两，桑寄生4两，藁本2两，白薇3两，赤石脂10两，益母草5两。
出处：《中药成方配本》。
主治：妇女月经不调，赤白带下，胎前产后诸症。
加减：None
功效：补气养血，安胎保产。
用法用量：每用1丸，开水化服。
制备方法：先将熟地捣烂，与诸药打和，晒干为末，用白蜜72两炼熟，打和为丸，分做800丸，每丸约干重2钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
