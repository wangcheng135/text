序号：77631
名称：丁香胶艾汤
组成：熟地黄3分，白芍药3分，川芎4分，丁香4分，阿胶6分，当归1钱2分，生艾叶1钱。
出处：《兰室秘藏》卷中。
主治：崩漏不止。盖心气不足，劳役及饮食不节所得，经隔少时，其脉二尺俱弦紧洪，按之无力，其证自觉脐下如冰，求厚衣被以御其寒，白带白滑之物多，间有如屋漏水下，时有鲜血，右尺脉时微洪。
加减：None
功效：None
用法用量：川芎为细末，当归酒洗锉，熟地黄、丁香为细末，艾亦锉，都作1服。水5大盏，先煎5味作1盏零2分，去滓入胶，再上火煎至1大盏，带热空心服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
