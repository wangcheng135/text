序号：2798
名称：天灵盖散
组成：天灵盖如2指大（以檀香煎水洗过用，涂酥炙），槟榔5个（如鸡心者，生用），连珠甘遂半两（并为末），阿魏1分（别研），安息香2分（用铜刀子钵内研，和诸药），麝香3钱（即当门子，别研）。
出处：《普济方》卷二三七引《卫生家宝》。
主治：一切传尸。
加减：None
功效：杀一切劳虫。
用法用量：每服3钱，加薤白2-7茎，葱白2-7茎，青蒿1小握，桃枝7寸，柳枝7寸，桑枝7寸，酸石榴枝(并用东南条嫩)长7寸，乌梅5个，用童便4升，同入银器内，文武火熬至1升，滤去滓，分作3盏调下，五更初服。男子患女人煎，女人患男子煎。服药后，如觉有逆吐，即用白梅止之，天明觉脏腑鸣，须臾转下虫，及恶物黄水，异色粪秽。若1服未下，辰、巳间更进1服，各温吃。若泻不止，用龙骨、黄连、诃子，等分为末，熟水调下1-2钱立止；次以白粥补之，用温药调补。
制备方法：上6味，再研匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
