序号：42471
名称：青黛散
组成：青黛1两（细研），麝香半两（细研），雄黄半两（细研），朱砂半两（细研），蚺蛇胆半两，黄柏半两（涂蜜微炙，锉），苦参半两（锉），桂心半两，杏仁半两（汤浸，去皮尖双仁，麸炒微黄），干姜1分（炮裂，锉），白矾半两（烧令汁尽），细辛1分，黄连半两（微炒，去须），藜芦1分（去芦头），附子半两（炮裂，去皮脐），莨菪子半两（水淘去浮者，水煮令芽出，候干，炒令黄黑色）。
出处：《圣惠》卷九十三。
主治：小儿疳痢久不愈，日渐羸瘦。
加减：None
功效：None
用法用量：每服半钱，以井花水调下，1日3次；1岁儿服1字，3岁儿服半钱。若口有疮及鼻痒，酥和绿豆大，安鼻中；若头上疳疮，及下部有疮赤烂，并用散敷之。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
