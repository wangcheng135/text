序号：50469
名称：茵陈栀子汤
组成：茵陈1钱，茯苓5分，山栀3钱，苍术（炒）3钱，白术（炒）3钱，黄芩（生）6分，黄连2分，枳实（炒）2分，猪苓（去皮）2分，泽泻2分，陈皮2分，防己2分，青皮2分。
出处：《医学纲目》卷二十一。
主治：谷疸，趺阳脉紧，食谷即眩，风寒相搏，谷气不清，胃中苦浊，浊气不流，小便不通，身体尽黄。
加减：None
功效：None
用法用量：上（口父）咀，作1服。长流水煎，食前温服，2服可愈。
制备方法：None
临床应用：None
各家论述：山栀、茵陈能泻湿热而退黄，故以为君；枳实苦寒，泄心下痞满；肺主气，今热伤其气，故身体麻木，以黄芩苦寒泻火补气为臣；二术苦甘温，青皮苦辛温，能除胃中湿热，泄其壅滞，养其正气；防己苦寒，能去十二经滞湿；泽泻咸平，茯苓、猪苓甘平，导膀胱中湿热，利小便而去癃闭。
用药禁忌：None
附注：None
