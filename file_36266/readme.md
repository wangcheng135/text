序号：36276
名称：周先生枯痔法
组成：明矾，赤石脂5钱，辰砂（痛加）1钱，黄丹。
出处：《准绳·杂病》卷六。
主治：痔。
加减：None
功效：None
用法用量：先用郁金末护肛门，如无郁金，用姜黄末代之，调涂四周好肉，如不就，加绿豆粉打合，却将枯药敷上，如肛门疼急，浓煎甘草汤放温，拂四周肛门上，就与宽肠药：槐花、大黄、枳壳、木通、连翘、瞿麦、当归，半酒半水煎。枯药早晨上1次，日午1次，洗去旧药，申时又洗去又上1次。如要急安，至夜半子时又洗上1次。至次日且看痔头淡淡黑色，2-3日如乌梅，4-5日内用竹篦子轻轻敲打痔头，见如石坚，至7-8日便住，更不须上枯药。且待自然如萝卜根乃脱去也。洗用甘草、荆芥、槐花，洗去旧药，方上新药。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：枯痔散（《外科十三方考》）。方中明矾、黄丹用量原缺。
