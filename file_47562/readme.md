序号：47577
名称：神仙五子丸
组成：覆盆子1两，五味子1两，蛇床子（醋拌浸，炒干用）1两，菟丝子（酒浸3日，焙干用）1两，巴戟（去心）1两，白茯苓1两，续断1两，肉苁蓉1两，牛膝（2味酒浸2宿，焙干）1两，枸杞子1两，干山药1两，熟干地黄（细切，焙干用）1两，肉桂（去皮）1两，槟榔1两，黑附子（炮裂，去皮）1两，枳实1两（麸炒），干姜1两（炮）。
出处：《医方类聚》卷一五三引《烟霞圣效方》。
主治：None
加减：None
功效：生气力，坚精髓，温润肌肤，耐寒热，乌发，驻颜。
用法用量：每服20-30丸，空心温酒送下。服药10日气力生，半月阳气胜，20日精髓坚，1月气力俱壮，皮肤滑润，冬不冷，夏不热，白发再黑，壮若童颜，妇人服之，肌体温润。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
