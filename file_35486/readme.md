序号：35496
名称：三仙丸
组成：茴香3两（炒，香），川乌头1两（去皮尖，锉如骰子，用盐半两炒黄，去盐），苍术2两（米泔浸1宿，用竹刀刮去黑皮，切碎，用葱白1握，炒黄去葱）。
出处：《普济方》卷二二三引《杨氏家藏方》。
主治：耳聋并眼暗。肾经虚寒，元气损弱，神衰力怯。
加减：None
功效：补实下经，理脾健胃，顺气搜风，驻颜活血，增筋力、乌髭须。
用法用量：每服50丸，空心、食前盐汤送下，1日2次。
制备方法：上各为细末，酒和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌猪、羊血。
附注：长寿丸（原书同卷）、三仙丹（《局方》卷五、淳佑新添方）、三圣丸（《普济方》卷一二○）。
