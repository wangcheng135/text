序号：71311
名称：紫石酒
组成：紫石英1斤，钟乳4两，麻黄3两，茯苓3两，白术3两，防风4两，远志4两，桂心4两，甘草3两。
出处：《千金》卷十四。
主治：久风虚冷，心气不足，或时惊怖。
加减：None
功效：None
用法用量：每服4合，1日3次，亦可至醉，常令有酒气。
制备方法：上(口父)咀。以酒3斗渍，春3日。
临床应用：None
各家论述：《千金方衍义》：久风虚冷，袭于心包，虽用麻黄、防风不能使透膈膜，故用苓桂术甘汤和营健脾，逐湿祛痰，更取钟乳之反激助雄，紫石之搜逐阴邪，远志之通关利窍，渍之以酒，则麻黄、防风得以逞缓祛之力。
用药禁忌：None
附注：本方方名，《千金方衍义》引作“紫石英酒”。
