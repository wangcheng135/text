序号：76455
名称：已试鲤鱼汤
组成：杏仁3两（熬），贝母3两，桂心3两，橘皮2两，人参2两，甘草2两（炙），厚朴2两（炙），麻黄2两（去节），茯苓2两，胡麻2两，白前2两，鲤鱼5斤，生姜6两，半夏5两（洗）。
出处：《外台》卷十引《古今录验》。
主治：上气喘急，身浮肿。上气。上气喘急，胸中满闷，咽喉不利。咳嗽有水声，身浮肿。
加减：None
功效：None
用法用量：先以水2斗，煮鱼得1斗2升，去鱼纳药，煎取3升2合，分4服，日3夜1。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：忌海藻、菘菜、醋物、羊肉、饧、生葱等物。
附注：杏仁散（《普济方》卷一八三）。本方方名，《兰台规范》引作“鲤鱼汤”。
