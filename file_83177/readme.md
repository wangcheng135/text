序号：83195
名称：乌龙膏
组成：乌梅（去核）1两，柴胡1两，紫菀1两，生干地黄1两，木香1两，秦艽（实好者）3钱，贝母（面炒，去心）3钱，防风3钱，杏仁5两（面炒，为末），皂角60片（20片去黑皮，醋炙为末；20片烧灰存性；20片汤浸，去黑皮）。
出处：《永类钤方》卷六引《水丘先生紫庭治瘵秘方》。
主治：骨蒸劳瘵。
加减：None
功效：None
用法用量：每服20丸，空心麦门冬汤送下。甚者20日效。
制备方法：猪肉剁烂如泥，同皂角一处入水5升，细揉汁，加童便3升，无灰酒1升，并熬如膏，和前药末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
