序号：64607
名称：理脾养胃除湿膏
组成：党参2钱，于术2钱（炒），茯苓3钱，莲肉3钱，薏米3钱（炒），扁豆3钱（炒），藿梗1钱5分，神曲2钱（炒），麦芽3钱（炒），陈皮1钱5分，广砂1钱（研），甘草8分。
出处：《慈禧光绪医方选议》。
主治：脾胃虚弱，饮食不消。
加减：None
功效：None
用法用量：上药以水熬透，去滓，再熬浓汁，少加炼蜜成膏。每服2钱，白开水冲下。
制备方法：None
临床应用：None
各家论述：本方即参苓白术散化裁而来，去桔梗，加神曲、麦芽，功专理脾；易山药，加藿梗，是防滋腻。本方药性中和，无寒热偏胜之弊，于光绪帝脾胃虚弱，饮食不消病症至为合拍，故亦常服之。
用药禁忌：None
附注：None
