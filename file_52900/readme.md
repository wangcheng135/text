序号：52915
名称：香蒲丸
组成：松香20两（水澄化7次），草乌8两，川乌4两（上2味用水2桶，煎浓去滓），鲜菖蒲（煎浓去滓）2斤，防风1两，荆芥1两，羌活1两，甘草1两（水1桶煎浓汁，沉去滓泥脚）。
出处：《青囊秘传》。
主治：大麻风。
加减：None
功效：None
用法用量：初服1钱半，3日；次3日服2钱半，进2次；到7日，再服1钱5分，每日2次；到10日服2钱5分，每日2次，周而复始，空心酒送下。
制备方法：将二乌汁煎松香，干；次入防风等汁煮松香，干；又入菖蒲汁煮松香，干；又将醋1碗煮松香，干，熬之，水浸出火毒，取晒为末。病在上部醋为丸，病在下部面糊为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
