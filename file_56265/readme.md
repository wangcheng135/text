序号：56280
名称：浚川丸
组成：桑白皮5钱，大戟5钱，雄黄5钱，茯苓5钱，芫花5钱，甘遂5钱，商陆5钱，泽泻5钱，巴戟5钱，葶苈5钱。
出处：《医学入门》卷七。
主治：十种水气初起。
加减：从面肿起，根在肺，桑白皮加至1两；从四肢肿起，根在脾，大戟加至1两；从背肿起，根在胆，雄黄加至1两；从背肿起，根在皮肤，茯苓加至1两；从胁肿起，根在肝，芫花加至1两；从腰肿起，根在胃，甘遂加至1两；从腹肿起，根在肺，商陆加至1两；从阴肿起，根在肾，泽泻加至1两；从手肿起，根在腹，巴戟加至1两；从脚肿起，根在心，葶苈加至1两。
功效：None
用法用量：每服30丸，姜汤或木香汤送下。
制备方法：上为末，醋糊为丸。
临床应用：None
各家论述：None
用药禁忌：忌鱼、面、盐百日。
附注：十水丸（《医学入门》卷七）。
