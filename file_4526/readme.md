序号：4526
名称：木香当归散
组成：川芎、当归、人参、官桂、三棱（炮）、蓬莪术（炮）、青皮（炒）、神曲（炒）、厚朴、麦蘖、干姜、小茴香（炒）、木香、陈皮、甘草、枳壳（炒）各等分。
出处：方出《袖珍》卷二，名见《医方类聚》卷一○二。
主治：心腹疼痛，胸膈虚痞，肠鸣腹胀，口苦吐酸，身体倦怠，饮食减少，痰逆恶心，时发呕吐，大便不调，或即泄泻。
加减：大便闭倍枳壳，加槟榔1两(炒)，去干姜。
功效：和脾暖胃，补虚益气。
用法用量：每服8钱，水2盏，加生姜3片，枣子1枚，葱白3茎，煎至1盏，去滓，食前温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
