序号：49084
名称：胎产金丹
组成：当归1两，川芎1两，白芍1两，人参1两，赤石脂1两，白术1两，茯苓1两，桂心1两，藁本1两，白薇1两，白芷1两，丹皮1两，玄胡1两，没药1两，甘草1两。
出处：《仙拈集》卷三。
主治：妇女经闭成疾，麻木疼痛，头昏脚肿，血淋白带，不受孕，胎不安，难产，产后血崩、血晕，儿枕痛，胞衣不下，呕吐。
加减：None
功效：调经养血，助气安胎。
用法用量：经闭成疾，麻木疼痛，头昏脚肿，血淋白带，滚汤送下；不受孕，服至1月即受孕；胎不安者，俱用滚汤送下；受孕即服不辍，保全足月分娩无忧；临产，清米汤调服1丸，自然顺利，难产者倍用；产下，童便好酒调服1丸，自无崩晕之症；血崩，童便滚水送下；血晕，当归川芎煎汤送下；产后儿枕痛，山楂黑糖煎汤送下；胞衣不下，干姜煎汤送下；呕吐，淡姜汤送下。病轻者调服1丸；重者调服2-3丸。
制备方法：除石脂、没药另研外，其余皆以醇酒浸3-7日，烘干为末，称15两；外用香附米以水浸3日，略炒为末，称15两，和匀，重罗筛过，炼蜜为丸，如弹子大，瓷器收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：《全国中药成药处方集》有党参、乳香，无人参。
