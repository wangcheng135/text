序号：51394
名称：贴痞膏
组成：三棱3钱，陈皮3钱，地骨皮3钱，黄芩3钱，黄连3钱，五灵脂3钱，苦参3钱，玄参3钱，赤芍药3钱，两头尖3钱，草乌3钱，香附子3钱，当归3钱，香白芷3钱，大黄3钱，木鳖子16个（去皮），巴豆49个（去壳），乳香5钱，没药5钱，轻粉5钱，血竭5钱，阿魏5钱，麝香3钱，香油1斤4两，铅丹10两（去消，用水2碗，滚3-4次，去水，焙干）。
出处：《摄生众妙方》卷六。
主治：痞积。
加减：None
功效：None
用法用量：贴，每日换1次。如有痒，剥了用热鞋底烙下，再依法贴之，待药力尽自落，不要强去。
制备方法：先将香油入铜锅内，即将18味切碎粗药入油内，用桑柴慢火煎之，黑黄色为度，去粗滓，方入铅丹，用槐、柳条不住手搅千遍，将药滴入水内成珠，去火，才入6味细药，用绢摊。
临床应用：None
各家论述：None
用药禁忌：忌食生冷、油腻并一切发物。
附注：None
