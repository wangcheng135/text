序号：76436
名称：川楝子丸
组成：川楝子（净肉）1斤（分4处，4两用麸1合，斑蝥49个同炒至麸黄色，去麸、斑蝥不用；4两用麸1合，巴豆49粒，同炒至麸黄色，去麸、巴豆不用；4两用麸1合，巴戟1两，同炒至麸黄色，去麸、巴戟不用；4两用盐1两，茴香1两同炒黄色，去盐及茴香不用），木香1两，破故纸（炒香为度）1两。
出处：《百一》卷十五。
主治：疝气多年，肿痛缩小，一切下部之疾。
加减：None
功效：None
用法用量：每服50丸，空心、食前盐汤送下。甚者日进2-3服。
制备方法：上为东，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：四炒川楝丸（《医学入门》卷七）、四炒楝实丸（《回春》卷五）、四制川楝子丸（《景岳全书》卷六十）。
