序号：62144
名称：救苦回生丹
组成：乳香1两5钱，没药1两5钱，当归1两5钱，川芎1两5钱，五灵脂1两，檀香1两，松香1两，自然铜（醋煅）1两，威灵仙1两，虎骨（炙）5钱，地龙5钱，草乌5钱，天麻7钱，全蝎2钱，麝香3钱，荆芥1两2钱，白芷1两2钱，苦参1两2钱，番木鳖30个（炙），冰片3分，京墨1块，黑豆2合（炒），闹羊花5钱，僵蚕6钱。
出处：《解围元薮》卷三。
主治：历节、半肢、紫云、哑风、蛊风、干风，走注遍身，寒湿麻痹瘫痪，中风不语，口眼斜；妇人产后血晕，经闭，胎衣不下。
加减：None
功效：None
用法用量：每服1丸，薄荷酒磨下。如昏迷则病愈。若妇人血晕、经闭、胎衣不下，用炒焦黑豆，淋酒服之。
制备方法：上为末，糯米饭为丸，如龙眼大，朱砂为衣，金箔（飞）裹。
临床应用：None
各家论述：None
用药禁忌：None
附注：《疡医大全》有枫香、紫荆皮，无当归、檀香。
