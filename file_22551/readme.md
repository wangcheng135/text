序号：22559
名称：吴茱萸丸
组成：吴茱萸1分（汤浸，焙干炒），山茱萸、牛膝（去苗，酒浸，切，焙）、石斛（去根）半两，细辛（去苗）1分，芎1分，附子（炮裂，去皮脐）1分，菟丝子（酒浸，别捣）半两，白茯苓（去黑皮）1分，羌活（去芦头）1分，独活（去芦头）1分，木香1分，草薢半两。
出处：《圣济总录》卷五。
主治：肾中风，恶风多汗，面浮肿，腰脊痛，不能正立，面色枯黑。
加减：None
功效：None
用法用量：每服30丸，空心、日午、临卧盐汤或盐酒送下。
制备方法：上为末，醇酒半盏、炼蜜半盏为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：按：方中山茱萸、牛膝用量原缺。
