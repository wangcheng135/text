序号：59184
名称：胰腺清化汤
组成：柴胡15g，黄芩15g，白芍15g，厚朴10g，枳实10g，佩兰10g，金银花10g，大青叶30g，大黄10g（后下），芒消6g（冲服）。
出处：《急腹症方药新解》。
主治：急性胰腺炎及其并发症。
加减：高热，加生石膏30g，知母24g，连翘12g；腹痛重，加川楝子、玄胡各10g；黄疸，加茵陈30g；呕吐，加姜夏、竹茹各10g；胸闷，加全瓜蒌30g，陈皮10g；湿重，加藿香、茯苓各10g；腹中寒冷，加干姜6g；腹泻每日超过3-4次，去芒消；胆道蛔虫引起的胰腺炎，加苦楝皮根15g，使君子10g；麻痹性肠梗阻，加重大黄、芒消、枳实、厚朴用量；胰腺囊肿或假囊肿，加栀子、黄连、红藤、地丁和三棱、莪术、山甲、皂刺等。
功效：理气解郁，清热化湿。
用法用量：水煎服，每日1剂，早、晚分服；重症患者可每日服2剂，分4次服，每6小时1次。
制备方法：None
临床应用：None
各家论述：白芍、厚朴、枳实理气解郁；柴胡、黄芩、金银花、大青叶、佩兰清热化湿，大黄、芒消清热凉血，泻热导滞。
用药禁忌：None
附注：None
