序号：75471
名称：解郁和肝丸
组成：当归23两，栀子（炒）23两，黄芩5两，茯苓5两，橘皮5两，枳壳（炒）5两，厚朴（炙）5两，郁金5两，木香5两，香附（炙）17两，川芎9两2钱5分，白芍15两，神曲（炒）15两，山楂15两，砂仁3两2钱5分，青皮（炒）6两，甘草2两5钱，法半夏9两2钱5分，苍术（炒）9两2钱5分，柴胡3两2钱5分。
出处：《北京市中药成方选集》。
主治：肝郁不舒，气逆胸满，两胁膨胀，胃痛恶心。
加减：None
功效：舒肝开郁，顺气消胀。
用法用量：每服2钱，日服2次，温开水送下。
制备方法：上为细末，冷开水泛为小丸，每16两用滑石细粉3两5钱为衣闯亮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
