序号：22267
名称：阳起圣灵丹
组成：当归4钱，枳壳4钱，川芎4钱，虎骨（酥炙）2钱，牛膝2钱，木瓜2钱，生地2钱，桑寄生2钱，补骨脂2钱，天花粉2钱，乌药2钱，麻黄2钱，陈皮2钱，山药2钱，苍术2钱，自然铜2钱，赤芍3钱，僵蚕3钱，白芷3钱，桔梗3钱，黄芩3钱，红花3钱，黄耆3钱，甘草3钱，阳起石3钱，龙泉香3钱，防风1两，荆芥1两，连翘1两，风藤1两。
出处：《解围元薮》卷三。
主治：痛风不举伏床者。
加减：None
功效：None
用法用量：每服70-80丸，用葱酒送下，早晚服。
制备方法：上为末，用不见水鹅掌2只，酒煮焙干；又用狗蹄4只烧灰，用鹅血煮酒各半碗，不见水狗血1碗，加面少许为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
