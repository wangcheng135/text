序号：30051
名称：赤苍饮
组成：赤茯苓（去皮）1两半，苍术（去粗皮，米泔水浸1宿，滤干，锉片，炒微黄）1两半，枳壳（制）1两，藿香（和根）7钱半，半夏（汤煮透，锉，焙干）7钱半，净香附7钱半，紫苏叶（和梗）7钱半，厚朴（去粗皮，姜汁炙香熟）7钱半，陈皮（去白）7钱半，甘草（炙）1两2钱。
出处：《准绳·幼科》卷七。
主治：脾胃因虚受湿，面貌浮黄或遍身作肿，饮食减少，气不升降，小便赤色，肚膨胀，咳嗽有痰及肿。
加减：None
功效：None
用法用量：每服2钱，水1盏，加生姜2片，煎7分，不拘时候温服。
制备方法：上锉。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《中国医学大辞典》引作“赤苍散”。
