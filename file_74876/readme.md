序号：74892
名称：截疫保命丹
组成：大劈砂2两4钱（生研极细，水飞净，研至无声为度，用以为灰），腰雄黄2两4钱（生研极细，水飞净，研至无声为度），公丁香2两4钱（生研极细），广木香2两4钱（生晒，研极细），杜蟾酥2两4钱（好烧酒浸化，杵入），珍珠2钱（生研极细，研至无声为度），真西珀8钱（生研至极细无声为度），嫩薄荷1两（入怀中熩脆，研细），茅苍术2两4钱（去粗皮，生研极细），水飞滑石2两4钱（研至无声为度），锦纹大黄4两8钱（生晒，研极细），当门子麝香6钱（俟诸药末具齐修合时，研细和匀），云母石4钱（煅，研细），五灵脂6钱（研细，酒飞，去砂，晒干再研）。
出处：《良方合璧》卷上。
主治：腹痛吐泻，霍乱晕厥，里急后重。
加减：None
功效：御外邪，守内变，通正气，驱积秽，复绝脉。
用法用量：每服7丸，重者多至3服，此丹入口须含在舌心，令其自化，舌上发麻，然后咽下；洞泄无度者，藿香汤送下21丸。
制备方法：上药各为极细末，愈细愈佳，然后秤准分两，拌匀，以好烧酒浸蟾酥杵和为丸，如莱菔子大，水飞朱砂为衣，碗合箕播，摩荡令光坚，晒干，瓷瓶收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
