序号：61178
名称：铁魔丹
组成：大枫子1斤（用麻黄、闹杨花各4两，酒煮1昼夜），苦参皮（酒拌，九蒸晒）1斤，荆芥穗（净末）1斤，白蒺藜（微炒）1斤，狗虱胡麻（微炒，净末）1斤。
出处：《解围元薮》卷三。
主治：诸般风症。
加减：春，加柴胡、草麻、升麻、川芎、藁本各4两；夏，加桔梗、黄芩、半夏、银柴胡各4两；秋，加石膏、甘草、玄参、当归各4两；冬，加知母、生地、五加皮各4两；心经，加茯苓、朱砂、远志、山萸、蒲黄、当归各4两；肝经，加荆芥、白芷、风藤、羌活、白芍、甘草、地黄各4两；脾经，加荆芥、白术、陈皮、苍术、独活、乳香、没药、血竭各4两；肺经，加天麻、桔梗、半夏、贝母、柴胡、沙参、巴戟、胡麻各4两；肾经，加黄柏、知母、茯苓、当归、升麻、草乌、甘草各4两；胃经，加枳壳、藿香、苍术、半夏、厚朴、柴胡、益智、草果、白豆蔻各4两。
功效：None
用法用量：每服100丸，温酒送下，日3次。
制备方法：上为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
