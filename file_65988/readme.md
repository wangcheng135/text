序号：66004
名称：蛇头丸
组成：蛇含石10个（煅3度，醋淬，却用甘草汤煮，出酸气，研，飞，为细末），铁腻粉2两，五灵脂（酒浸，去砂）2两，神砂（研）2两，蝎梢2两，白附子（炮）2两，郁金（炮）2两，龙脑（别研）半两，麝香（研）1两，花蛇头10个（酒浸，去骨，用齿并肉）。
出处：《局方》卷十（续添诸局经验秘方）。
主治：小儿急慢惊风，手足抽掣，眼睛直视，角弓反张，证候危急者。
加减：None
功效：None
用法用量：每服1丸，薄荷自然汁磨，以井花水化开，量儿大小加减与服。
制备方法：上为细末，面糊为丸，如鸡头大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
