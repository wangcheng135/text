序号：53126
名称：大补益母丸
组成：益母草8两（用上截），香附2两（7制），嫩黄耆3两（蜜炒），人参2两（去芦），白术3两（土炒），白茯苓2两（蒸透），炙草2两，当归身3两（俱酒洗），白芍2两（酒炒），陈皮2两，熟地3两，砂仁2两（炒）。
出处：《履霜集》卷二。
主治：虚损而经候不调，或因虚损而经闭不行，或因虚损而吐衄崩带，或因虚损而小胎不稳，或因虚损而产后多疾。
加减：None
功效：调经安胎。
用法用量：为丸服。经不调，龙眼肉、炒枣仁、去心莲子煎汤送下；经闭，炒桃仁，炒红花煎汤送下；下血，生地、炒芩、丹皮煎汤送下；小胎不稳，炒芩、陈皮（去自）、苏梗煎汤送下，俱4、5分为率；产后恶露未净，腹中心硬疼，先用黄酒服救产丸，下净瘀血，继服此丸；若无恶露，多服此九，补虚为主；感寒，加生姜；发热，加童便。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
