序号：37264
名称：定痛生肌杖疮膏
组成：乳香（去油）5钱，儿茶5钱，象皮5钱（煅灰，为末），龙骨5钱（煅过，为末），没药5钱（去油），血竭5钱，冰片1钱，牡蛎壳1个（煅灰，研末）5钱。
出处：《医林绳墨大全》卷九。
主治：杖疮。
加减：None
功效：None
用法用量：如遇杖者，先将韭菜、葱头、猪肉3味煎汤净洗后膏涂患处，再将油纸贴上，加棉花裹好，再用布包，毋令出气，1日1夜换洗1次。如打见骨者，加细药掺上患处。
制备方法：上为细末，先用麻油12两，猪板油（净）4两，入砂锅内，下净头发2两，鸡子清5-6个同熬，候油滴水成珠，入白蜡2两，黄蜡2两烊尽，再入滚水泡过，飞净黄丹2两，铅粉2两，用槐条急搅成膏，取起离火，入前细药和匀。
临床应用：None
各家论述：None
用药禁忌：忌房事、诸发物。
附注：None
