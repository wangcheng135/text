序号：3497
名称：太阳汤
组成：白术3钱，干姜（炒）1-2钱，当归钱半，山药（炒）2钱，熟附子2-3钱（湿纸包，煨，热用），甘草（炙）1钱，白芍（煨）钱半，生姜1钱，红枣3枚。
出处：《会约》卷三。
主治：伤寒寒中三阴，战傈厥逆，呕吐昏迷，唇青囊缩者。
加减：如泄泻者，去当归；势危者，去熟附子，用生附子(湿纸包，煨热用)；如冬月寒伤太阳经，有表证者，加麻黄8分；头痛，加北细辛2-3分；如肉振汗多者，加制黄耆2-3钱；如泄泻者，加乌梅2个，去当归；如肝脉紧而郁滞者，加肉桂2钱；如小腹痛而喜按者，加吴茱萸7分(汤泡1次用)，又须兼用华佗救阳法外治，乃妙。
功效：None
用法用量：水煎服。假热，拒格不纳者，冰冷服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：华佗救阳法：治寒中三阴，阳脱无脉，昏倒强直等症。用葱白一二斤捣碎，炒热，绢绸包熨脐下；以二包更替熨之。脉渐出，手足温者生。
