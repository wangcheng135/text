序号：76538
名称：雷火针
组成：蕲艾1两，朱砂2钱，穿山甲（土炙）1钱，桃皮1钱，草乌1钱，川乌1钱，乳香1钱，雄黄1钱，没药1钱，硫黄1钱，麝香5分。
出处：《疡医大全》卷二十九。
主治：风寒湿气痰滞于经络血脉之中，闭塞不通而痛。
加减：None
功效：祛风散寒，通经活络。
用法用量：患处用青布或红布折二十四层，将针灯上点着吹息，隔布针之。患内暖极即止。不过三次即愈。
制备方法：上为细末，以蕲艾铺粗纸上，入药末3钱捲成筒，如爆竹式，将药筒入瓶内，以箬叶、油纸封口，埋屋下地中49日取出。
临床应用：None
各家论述：None
用药禁忌：None
附注：热熨针（《全国中药成药处方集》北京、南京方）。
