序号：4543
名称：木香汤
组成：木香1分，半夏（汤洗7遍，去滑）1分，人参1分，赤茯苓（去黑皮）1分，白术1分，干姜（炮）半两，甘草（炙，锉）半两，桂（去粗皮）半两，厚朴（去粗皮，涂姜汁，炙熟）半两，枳壳（去瓤，麸炒）半两，陈橘皮（汤浸，去白，焙）1两，草豆蔻（去皮）2个，槟榔1个（鸡心者，锉），诃黎勒5个（煨，去核）。
出处：《圣济总录》卷八十八。
主治：脾胃虚冷，痰饮不消，心腹时痛。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，加大枣两枚(擘破)，生姜1分(拍碎)，同煎取8分，去滓热服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
