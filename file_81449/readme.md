序号：81465
名称：鳖甲散
组成：鳖甲（炙）3分，干姜3分，芎1两，云母1两，代赭1两，乌贼鱼骨1分，龙骨1分，伏龙肝1分，白垩1分，猬皮（炙）1分，生鲤鱼头半两，桂心半两，白术半两，白僵蚕半分。
出处：《千金翼》卷八。
主治：妇人五崩，身体羸瘦，咳逆烦满，少气，心下痛，面上生疮，腰大痛不可俯仰，阴中肿，如有疮之状，毛中痒，时痛，与子脏相通，小便不利，常头眩，颈急痛，手足热，气逆冲急，烦不得卧，腹中急痛，食不下，吞酸噫苦，肠鸣，漏下赤白黄黑汁，大臭如胶污衣状，热即下赤，寒即下白，多饮即下黑，多食即下黄，多药即下青，喜怒心中常恐，一身不可动摇，大恶风寒。
加减：若头风小腹急，加芎、桂心各1两。
功效：None
用法用量：每服方寸匕，以淳酒纳少蜜送下，日3次，夜2次。久病者10日愈，新病者5日愈。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：忌生冷、猪、鸡、鱼、肉。
附注：None
