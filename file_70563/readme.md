名称：紫芝丸
组成：紫芝1分，胡黄连1分，当归1分，羚羊角1分，赤石脂1分，人参1分，马齿（炙）1分，川椒2分，杏仁2分，蚱蝉2分，乌蛇2分，雀瓮（并炒）2分，蜂房2分，丹参2分，干姜2分，芍药2分，龙骨2分，细辛2分，黄连2分，芎2分，丹砂（熬10上下）5分，牛黄5分，东门上鸡头（炙）1枚（一方有蜣螂3分，桂心3分）。
出处：《幼幼新书》卷三十九引《婴孺方》。
主治：小儿百病惊痫。
加减：None
功效：None
用法用量：1岁儿每服2丸，食前服，1日3次。不知，加。
制备方法：上为末，乳汁为丸，如豆大。
临床应用：None
各家论述：None
