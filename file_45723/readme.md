序号：45738
名称：柞叶汤
组成：柞木叶（干）4两，干荷叶4两，萱草根（干）2两，甘草节1两，地榆1两。
出处：《苏沈良方》卷九。
主治：发疽。
加减：None
功效：None
用法用量：每服半两，水2碗，煎去半，分2服，早、晚各服2服，滓并煎作1服。有脓血者自安，脓血在内者自大肠下，未成者自消。有疮者贴下药：通明牛皮胶1两（水半升，熬令化），黄丹1两，入胶中煮3-5沸。上放温冷，以鸡羽敷疮口，有疮即敛，未成疮者涂肿处即内消。
制备方法：上细锉。
临床应用：None
各家论述：《本事方释义》：柞木叶气味苦平，入足厥阴；干荷叶气味苦辛平，入足少阳、厥明；金樱根（萱草根）气味甘凉，入足厥阴，能利湿解毒；甘草节气味甘平，入足太阴，最能解毒，通行十二经络；地榆气味咸苦微寒，入手足阳明，此治痈疽发背之方也。大凡诸毒之发，皆由湿热壅痹，致气血凝滞而成，凉血之药，必兼分利湿热，则源头既清，病自消散矣。
用药禁忌：None
附注：柞木散（《本事》卷六）、柞木饮子（《外科精要》卷上）。
