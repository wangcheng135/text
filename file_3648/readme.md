序号：3648
名称：巴戟散
组成：巴戟1两，柏子仁1两，石龙芮1两，天麻1两，牛膝（去苗）1两，牡蛎（烧为粉）1两，菟丝子（酒浸1宿，焙干，别捣）1两，天雄（炮裂，去皮脐）1两，肉苁蓉（酒浸1宿，刮去皴皮，炙干）1两，萆薢（锉）3分，防风（去芦头）3分，当归3分，羌活3分，桑螵蛸（微炙）3分，肉桂2两（去皴皮）。
出处：《圣惠》卷二十七。
主治：风劳，气血不足，脏腑虚伤，肢节烦疼，腰膝无力，形体羸瘦，面色萎黄，小便数多，卧即盗汗。
加减：None
功效：None
用法用量：每服2钱，空心及晚食前以温酒调下。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
