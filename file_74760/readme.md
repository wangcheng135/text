序号：74776
名称：解毒消炎丸
组成：丁香450g，雄黄200g，蟾酥200g，朱砂150g。
出处：《山东省药品标准》。
主治：痈、疖、疔疮，毒虫咬伤，乳蛾，喉风，喉痛，一般咽喉红肿疼痛。
加减：None
功效：解毒，消肿，止痛。
用法用量：饭后服，1次4-6粒，1日3次；儿童1次2-3粒，婴儿1次1粒，外用适量，用冷开水或醋烊化后敷患处。
制备方法：上为细末，蟾酥粉碎过80目筛，加70％乙醇适量，待其胀透，加入朱砂、雄黄末搅拌均匀，45℃干燥后，粉碎成细粉，过筛，再加入丁香细粉混匀，过筛。用60～70％乙醇泛为细小水丸，45℃以下干燥，另取百草霜适量上衣，外包虫胶、玉米朊即得，每50粒重1g。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
