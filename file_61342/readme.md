序号：61358
名称：鸭通汤
组成：白鸭通5升（沸汤2斗半淋之，澄清，取2斗汁），麻黄8两，豉3升，冷石2两，甘草5两，石膏3两，栀子20枚。
出处：《千金》卷二十四。
主治：五石发动。
加减：None
功效：解散除热。
用法用量：上(口父)咀。以鸭通汁煮6升，去滓，纳豉3沸，分服5合。若觉身体冷，小便快阔，其间若热犹盛，小便赤促，服之不限5合。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：鸭通利水，取白以通金水二脏，为解石毒之专药；佐以冷石、石膏清解里热；麻黄开泄表气；栀子、豉、甘解毒除烦，兼通表里恶毒异气。
用药禁忌：宜小劳之，渐进食，不可令食少，但勿便多。
附注：None
