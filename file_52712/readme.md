序号：52727
名称：香砂养胃汤
组成：香附（炒）8分，砂仁8分，苍术（米泔制，炒）8分，厚朴（姜汁炒）8分，陈皮8分，人参5分，白术（去芦）1钱，茯苓（去皮）8分，木香5分，白豆蔻（去壳）7分，甘草（炙）2分。
出处：《回春》卷二。
主治：脾胃不和，胃寒不思饮食，口不知味，痞闷不舒。
加减：脾胃寒，加干姜、官桂；肉食不化，加山楂、草果；米粉、面食不化，加神曲、麦芽；生冷瓜果不化，加槟榔、干姜；胸腹饱闷，加枳壳、萝卜子、大腹皮；伤食胃口痛，加木香、枳实、益智；伤食泄泻，加干姜、乌梅、白术；伤食恶心呕吐，加藿香、丁香、半夏、乌梅、干姜。
功效：None
用法用量：加生姜、大枣，水煎服。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方改为丸剂，名“香砂养胃丸”（见《饲鹤亭集方》）。方中甘草用量原缺，据《寿世保元》补。
