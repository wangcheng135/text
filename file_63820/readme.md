序号：63836
名称：清营汤
组成：犀角3钱，生地5钱，元参3钱，竹叶心1钱，麦冬3钱，丹参2钱，黄连1钱5分，银花3钱，连翘2钱（连心用）。
出处：《温病条辨》卷一。
主治：暑温，邪入手厥阴，脉虚，夜寐不发，烦渴舌赤，时有谵语，目常开不闭，或喜闭不开及阳明温病，邪在血分，舌黄燥，肉色绛，不渴者。邪热初入营分，身热夜甚，口渴或不渴，时有谵语，心烦不眠，或斑疹隐隐，舌绛而干，脉象细数。
加减：None
功效：清营解毒，透热养阴。
用法用量：上以水8杯，煮取3杯，每日3服。
制备方法：None
临床应用：暑温：温邪入心包络，神昏痉厥，极重之症。连翘3钱，生石膏6钱，麦冬（连心）5钱，银花5钱，细生地5钱，知母2钱，丹皮3钱，生甘草1钱5分，竹叶2钱，今晚2帖，明早1帖，再服紫雪丹4钱。
各家论述：1.《成方便读》：方中犀角、黄连，皆入心而清火，犀角有轻灵之性，能解夫疫毒，黄连具苦降之质，可燥乎湿邪，二味为治温之正药；热犯心包，营阴受灼，故以生地、元参滋肾水，麦冬养肺金，而以丹参领之入心，皆得遂其增液救焚之助；连翘、银花、竹叶3味，皆能内彻于心，外通于表，辛凉轻解，自可神安热退，邪不自留耳。_x000D_
2.《方剂学》：犀角咸寒清解营分之热毒，为主药；热甚伤阴，故以玄参、生地、麦冬甘寒清热养阴，共为辅药；温邪初入营分，根据“入营犹可透热转气”的理论，佐以苦寒之黄连、竹叶心、连翘、银花清心解毒，并透热于外，使热邪转出气分而解，体现了本方气营两清之法。丹参清热凉血，并能活血散瘀，以防血与热结，亦为佐药。
