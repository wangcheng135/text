序号：43491
名称：削坚丸
组成：鳖甲（醋浸两宿，去裙襕，更蘸醋炙黄色，取末称）2两半，京三棱（锉如小枣大，好醋浸2宿，焙干，取末称）2两半，干漆（捣碎，炒令烟出，捣细，取末称）2两半，沉香半两，乳香2钱半（别研），槟榔半两，木香半两，干姜（炮）半两，没药（别研）半两，肉桂（去粗皮）半两，细松烟墨（烧去胶）半两，胡椒半两，萝卜子半两，干蝎（微炒，去毒）半两，硇砂（通明者，为末，重汤飞炼，别研）半两，粉霜2钱半（别研），轻粉2钱半。
出处：《杨氏家藏方》卷五。
主治：五积六聚，气结成块，食积癖瘕，心腹胀满，瘦瘁少食。
加减：None
功效：None
用法用量：每服20丸，淡醋煎生姜汤送下，日2次，夜1次。如未利渐加，微利即减。
制备方法：上为细末，拌匀，用好醋煮薄面糊为丸，如小绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
