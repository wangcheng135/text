序号：22735
名称：却老乌须健阳丹
组成：赤茯苓（牛乳拌）1斤，白茯苓（人乳拌，各浸1宿，晒干）1斤，白首乌（竹刀去皮，打碎）1斤，赤首乌（制同上）1斤，牛膝（同何首乌用黑豆5升砂锅内蒸3次）半斤，枸杞（酒浸洗，晒干）半斤，当归（酒浸1宿）半斤，茯神半斤，菟丝子半斤（酒浸3日，晒干），破故纸5两（炒黄）。
出处：《医部全录》卷三三一引《体仁汇编》。
主治：None
加减：None
功效：颐养补益。
用法用量：日进3丸：早1丸，空心酒送下；午后1丸，姜汤送下；临睡1丸，盐汤送下。初服3日，小便杂色，是去五脏杂病；14日唇红，口生津液，再不夜起；28日身体轻健，两乳红润；至1月后，鼻头辛酸，诸风百病皆出；49日，目视光明，两手火热，精通，白发反黑，齿落更生，阳事强健，丹田如火，行走如飞，气力倍加。
制备方法：上药各不犯铁器，为末，炼蜜为丸如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
