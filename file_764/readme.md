序号：764
名称：六合定中丸
组成：香薷4两，木瓜2两，茯苓2两，枳壳2两，紫苏4两，甘草5钱，厚朴2两，广木香1两，广藿香2两，阳春砂仁2两。
出处：《济急丹方》卷上。
主治：感冒风寒，四时痧症，受暑，痢疾，疟疾，伤食，山岚瘴气等。
加减：None
功效：解暑毒，祛风寒。
用法用量：每用1丸，小儿半丸，四时痧症、霍乱转筋，阴阳水(滚水、凉水各半)送下；感冒风寒，紫苏、葱头汤送下，或生姜汤送下，头痛发热，葱头汤送下，心腹饱胀，砂仁汤送下；疟疾，姜、枣汤送下；痢疾，红糖汤送下；伤食，炒萝卜子汤送下；受暑，凉藿香汤送下；山岚瘴气，槟榔汤送下。
制备方法：上药水泛为丸，每药末净重1钱3分为1丸，收贮瓷瓶。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
