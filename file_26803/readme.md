序号：26811
名称：延龄固本丸
组成：人参2钱，肉桂2钱，当归2钱，韭菜子（火酒煮）2钱，枸杞2钱，茯神2钱，山药2钱，菟丝饼2钱，山萸肉2钱，蛇床子2钱，牛膝2钱，熟地4两，何首乌（9蒸）4两，肉苁蓉（去甲，酒洗）4两，大附子1个（童便浸煮），鸽子蛋10个（去皮，炒成粉），黄狗肾（内外1具，酥炙），车前子1两，黑驴肾（连子1具，竹刀去筋，同肉苁蓉酒煮1日1夜），鹿茸1对（酥炙）。
出处：《良朋汇集》卷二引王世功集方。
主治：五劳七伤，诸虚百损，颜色衰朽，形体羸瘦，中年阳事不举，精神短少，未至五旬，须发先白，并左瘫右痪，步履艰辛，腰膝疼痛，下元虚冷等症。
加减：None
功效：None
用法用量：每服100丸，黄酒或滚水送下。
制备方法：上为细末，以驴肾、苁蓉捣膏为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
