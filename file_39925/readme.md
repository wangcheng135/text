序号：39935
名称：肥儿丸
组成：人参（去芦）3钱半，白术（去芦）3钱，白茯苓（去皮）3钱，黄连（姜汁炒）3钱半，胡黄连5钱，使君子（去壳）4钱半，神曲（炒）3钱半，麦芽（炒）3钱半，山楂肉3钱半，甘草（炙）3钱，芦荟2钱半（碗盛，泥封固，置土坑中，四面谷糠火煨透用之）。
出处：《古今医鉴》卷十三引刘尚书方。
主治：小儿脾虚虫积所致诸疳、癖疾，面黄体瘦，头大颈细，夜热冷汗，神倦嗜卧，吐泻纳呆，好食泥土，腹大硬痛，烦渴有癖块。饮食不进，四肢倦惰，各种疳证，以脾虚有虫积兼泄泻者为宜。吐乳，痴眠。心下痞硬，头大颈细，有时吐泻烦渴，大便腥粘。
加减：None
功效：消疳化积，磨癖清热，伐肝补脾，进食杀虫，养元气。润肌肤。
用法用量：每服20-30丸，米汤化下。或作小丸亦可。
制备方法：上为细末，黄米糊为饼。
临床应用：None
各家论述：None
用药禁忌：寒症忌用。
附注：参术肥儿丸（《幼科证治大全》引《济世全书》）。
