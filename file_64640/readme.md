序号：64656
名称：痔漏消管丸
组成：刺猬皮（烫）1两，石决明（煅）1两，僵蚕5钱，胡连5钱，木鳖子肉5钱，地龙肉8钱，连翘炭1两5钱，芥穗8钱，大黄1两2钱，当归尾1两，槐花1两5钱，乳香（炙）8钱，鹅管石（煅）1两，赤芍5钱，龟头2个，蜂房5钱，马钱子（炙）5钱，血竭5钱。
出处：《北京市中药成方选集》。
主治：痔疮漏疮，肛门红肿，痔漏生管，淋沥脓水。
加减：None
功效：化痔消管，解毒止痛。
用法用量：每服2钱，每日2次，温开水送下。
制备方法：上为细末，用冷开水泛为小丸，每16两用滑石粉4两为衣，闯亮，盒装重6钱。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
