序号：47671
名称：千捶膏
组成：长夏枯草1斤，北蓖麻肉240粒，甜杏仁240个，核桃肉16个（俱去粗细皮），轻粉2钱，铜青2钱，松香末1斤，黄占2两，乳香6钱，没药6钱，儿茶末6钱。
出处：《惠直堂方》卷四。
主治：诸毒诸疮，未溃及已溃者。
加减：None
功效：None
用法用量：重汤摊贴。
制备方法：将长夏枯草煎汁，入北蓖麻肉，甜杏仁、核桃肉，候熟，取起捣烂。又将轻粉、铜青二物，用袋盛，入汁煮熟，捞起为末。后入松香末，煮干，又入黄占搅匀。离火，入乳香，没药，儿茶末，并前煮熟药5味，共入臼捣千余下，收贮。
临床应用：None
各家论述：None
用药禁忌：臁疮、杨梅毒不贴。
附注：None
