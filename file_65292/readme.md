序号：65308
名称：大槟榔丸
组成：槟榔3两，桂心3两，附子2两（炮），栝楼3两，杏仁3两（熬），干姜2两，甘草（炙）2两，麻黄3两（去节），黄耆3两，茯苓3两，厚朴2两（炙），葶苈3两（熬），椒目3两，吴茱萸5合，白术3两，防己2两。
出处：《外台》卷二十引《范汪方》。
主治：老小水肿；虚肿；大病客肿作喘病。
加减：None
功效：None
用法用量：每服2丸，1日3次。不知，稍增至4丸；不知，又加2丸；不下，还服4丸。得小下为验。
制备方法：上药治下筛，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌海藻、菘菜、猪肉，冷水、生葱、桃李、雀肉、大醋。
附注：槟榔丸（《千金翼》卷十九）。
