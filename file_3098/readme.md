名称：天麻退翳散
组成：当归1两（好酒浸，焙干），熟地黄1两（酒浸，焙干），川芎1两5钱，赤芍药2两5钱（热水泡），白僵蚕1两（热水泡过，洗去丝，姜汁炒），蝉蜕50个（水泡洗，去头足），羌活1两，防风1两，荆芥1两，木贼（去根节）1两，石决明1两（烧过存性），白蒺藜1两5钱，白芷1两5钱，甘草7钱，麦门冬2两，黄芩尾1两，羊角天麻（炒存性）1两，厚枳壳（炒）1两，蔓荆子1两（打少碎），菊花1两，蜜蒙花7钱。
出处：《银海精微》卷下。
主治：垂帘翳障，昏暗不明。
加减：若眼红，加黄连。
功效：None
用法用量：每服加莲子3个，灯心7根，水1钟半，煎至8分，食后温服。
