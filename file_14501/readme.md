序号：14508
名称：先天大造丸
组成：紫河车1具（酒煮，捣膏），熟地黄4两（酒煮，捣膏），归身2两，茯苓2两，人参2两，枸杞2两，菟丝子2两，肉苁蓉（酒洗，捣膏）2两，黄精2两，白术2两，何首乌2两（去皮，用黑豆同煮，捣膏），川牛膝2两，骨碎补2两（去毛，微炒），仙茅（浸去赤汁，蒸熟，去皮，捣膏）2两，川巴戟（去骨）1两，破故纸（炒）1两，远志（去心，炒）1两，木香5钱，青盐5钱，丁香3钱，黑枣肉2两。
出处：《徐评外科正宗》卷七。
主治：风寒湿毒袭于经络，初起皮色不变，漫肿无头；或阴虚外寒侵入，初起筋骨疼痛，日久遂肿，溃后脓水清稀，久而不愈，渐成漏症；一切气血虚羸，劳伤内损，及男妇久无嗣息。
加减：None
功效：None
用法用量：每服70丸，空心温酒送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
