序号：19476
名称：地黄石斛丸
组成：生地黄5斤（研取汁，银石器中熬去半，入白蜜4两，慢火熬成膏），石斛（去根）1两，巴戟天（去心）1两，牛膝（去苗，切，酒浸，焙）1两，肉苁蓉（酒浸，去皴皮，切，焙）1两，桂（去粗皮）1两，补骨脂（炒）1两，鹿角胶（炒令燥）1两，菟丝子（酒浸，别捣末）1两，木香1两，附子（炮裂，去皮脐）1两，枸杞子（焙）1两，鹿茸（去毛，酥炙）1两。
出处：《圣济总录》卷一八五。
主治：None
加减：None
功效：补虚，益精髓。
用法用量：每服20-30丸，空心、临卧温酒或盐汤送下。
制备方法：上为末，入膏为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
