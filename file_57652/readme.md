序号：57667
名称：大断下丸
组成：附子（炮，去皮脐）1两，细辛（去土叶）7钱半，干姜（炮）1两半，高良姜1两半，肉豆蔻1两（面裹煨），诃子（煨，去核）1两，酸石榴皮（去瓤子）1两半（醋浸1宿，取出炙令焦黄），龙骨1两半（研），赤石脂1两半（研），牡蛎（火煅，称）1两（研），白矾（枯）1两。
出处：《杨氏家藏方》卷七。
主治：脏腑停寒，肠胃虚弱，腹痛泄泻，全不思食；下痢滑数，肌肉消瘦，饮食不入，脉细皮寒，气少不能言，有时发虚汗，及脏腑停寒，脐腹痛。
加减：None
功效：None
用法用量：每服30丸，食前温米饮送下。
制备方法：上为细末，醋煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
