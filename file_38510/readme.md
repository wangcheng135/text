序号：38520
名称：治痢安胎饮
组成：绵耆3-4钱（炙），生地炭3钱，归身炭2钱，茯苓3钱，泽泻2钱，升麻（炒）5分，银花3钱，条芩（酒炒）1钱5分，川连（酒炒）8分，广木香5分，范志曲2钱。
出处：《医方简义》卷五。
主治：妊娠下痢，腰痛气滞，里急后重，少腹疼痛。
加减：噤口者，水汤不能进而呕吐频频，加石莲子3钱，石菖蒲3钱，生姜3片，去生地炭1味；口渴，加青果1枚，乌梅1枚；赤痢，加地榆炭3钱；白痢，加白槿花1钱；腹痛甚，加川椒20粒，去升麻1味，更加白芍1钱；赤白兼者，加天仙藤2钱、驴胶1钱，去广木香。
功效：None
用法用量：加荷叶1角，水煎服。或外加扁豆叶20片以醒胃气。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
