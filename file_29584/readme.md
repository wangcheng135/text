序号：29592
名称：附姜白通汤
组成：附子（炮，去皮脐）5钱，干姜（炮）5钱，葱白5茎（取汁），猪胆（大者）半枚。
出处：《法律》卷二。
主治：暴卒中寒，厥逆呕吐，泻利色青色冷，肌肤凛栗无汗，盛阴没阳之证。
加减：None
功效：回阳散阴。
用法用量：用水2大盏，煎附、姜2味至1盏，入葱汁并猪胆汁，和匀温服。再用葱1大握，以带轻束，切去两头，留白2寸许，以一面熨热，安脐上，用熨斗盛炭火熨葱白上面，取其热气，从脐入腹，甚者连熨2-3饼。又甚者，再用艾炷灸关元、气海，各20-30壮。
制备方法：None
临床应用：None
各家论述：《重订通俗伤寒论》：以大剂附、姜回阳为君，臣以葱汁，得生阳之气独盛，以辛通脉道，反佐以一味胆汁者，恐阳药一饮即吐，格拒而不得入也。此为温热回阳，苦辛通格之良方。
用药禁忌：None
附注：姜附白通汤（《成方切用》卷六）。
