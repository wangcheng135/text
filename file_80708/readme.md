序号：80724
名称：麋角既济丸
组成：麋角1具（净水浸3日，刮去粗皮，镑为屑，盛在瓷瓶内，银瓶尤佳，以牛乳浸1日，乳耗更添，直候不耗，于角屑上乳深2指以来；用大麦只看瓶器大小，临时安顿甄内，约厚3寸，上置瓶，更用大麦周延填实，唯露瓶口，不住火蒸一伏时，如锅内水耗，旋添汤，直候角屑蒸得细腻如面相似，即住火，取出细研，别用下项药），龙骨2两，山药2两，人参（去芦头）2两，远志（去心）2两，山茱萸2两，石菖蒲2两，赤石脂2两，朱砂（别研）2两，五味子2两，全蝎（艾叶炒，去毒）2两，巴戟（去心）3两，附子（炮3两，去皮脐）3两，补骨脂（炒）3两，菟丝子（酒浸1宿，焙）3两，天雄（炮，去皮脐）3两，柏子仁（别研）4两，熟干地黄（洗，焙）4两，肉苁蓉（酒浸1宿，切，焙）4两。
出处：《杨氏家藏方》卷九。
主治：水火不济，精神恍惚，梦寐纷纭，阳道不兴，耳内虚鸣，小便白浊，遗沥失精。
加减：None
功效：常服能使火不上炎而神自清，水不下渗而精自固，壮阳固气，益血驻颜。
用法用量：每服100丸，空心温酒送下。
制备方法：上为细末，以麋角膏子和匀为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：麋茸角既济丸（《普济方》卷二一七）。
