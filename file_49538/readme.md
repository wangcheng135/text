序号：49553
名称：茯苓丸
组成：茯苓4分，白术4分，椒目4分，木防己5分，葶苈5分，泽泻5分，甘遂11分，赤小豆2分，前胡2分，芫花2分，桂心2分，芒消7分（别研）。
出处：《千金》卷二十一引《古今录验》。
主治：水肿。支饮上气，黄疸及脚气、消渴后成石水，腹胁坚胀，足胫浮肿，上气不得卧，口干，颈脉动，腹胀间冷，大小便不利。
加减：None
功效：None
用法用量：1日5丸，稍加，以知为度，蜜汤送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：丸中芫花、甘遂、葶苈、芒消、椒目、防己兼走二便；佐以茯苓、白术、桂心、泽泻、前胡、赤小豆利水下气之味，深得峻药缓攻之妙。
用药禁忌：None
附注：茯苓煎（《鸡峰》卷十九）。
