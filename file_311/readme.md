序号：311
名称：五香丸
组成：沉香（锉）1两，丁香1两，白檀香（锉）1两，茴香子（炒）1两，荜澄茄1两，青橘皮（去白，焙）1两，胡椒1两，缩砂（去皮）1两，赤茯苓（去黑皮）1两，白芷1两，牛膝（酒浸，切，焙）1两，甘草（炮）1两，木香1两半，麝香（研）3分，蓬莪术（炮，锉）半两，枳壳（去瓤，麸炒）半两，葛花1两半，肉豆蔻（去壳）5枚，槟榔（炮）3枚（锉），半夏3两（汤洗7遍，入生姜3两，同杵为末作饼，焙干），人参半两，桂（去粗皮）半两，荜拔半两，赤小豆花3两，葛根（炒）2两。
出处：《圣济总录》卷五十四。
主治：三焦虚胀，心腹满闷。
加减：None
功效：None
用法用量：每服1丸，细嚼，用淡生姜汤送下，不拘时候。或饮酒多气闷，即含化1丸。
制备方法：上为末，拌匀，炼蜜为丸，如樱桃大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
