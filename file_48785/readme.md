序号：48800
名称：大三五七散
组成：天雄3两，细辛3两，山茱萸5两，干姜5两，薯蓣7两，防风7两。
出处：《千金》卷十三。
主治：肝肾不足，风寒外袭，头痛眩晕，口眼喎斜，耳聋耳鸣，风寒湿痹。面骨痛，风眩痛。阳虚风寒入脑，头痛目眩，如在舟车之上，耳内蝉鸣，或如风雨之声应，风寒湿痹，脚气缓弱。产后风。
加减：None
功效：None
用法用量：每服5分匕，清酒送下，1日2次。不知稍加。
制备方法：上药治下筛。
临床应用：None
各家论述：《医方考》：大寒中于风府，令人头痛，项筋紧急者，此方主之。风府，脑后之穴，督脉之所主也。寒者，天地严凝之气，故令项筋紧急。干姜、附子，辛热之物也，可以散真寒；细辛、防风、气薄之品也，可使至高巅；山萸养督脉之阴，茯苓和督脉之阳。
用药禁忌：None
附注：天雄散（《圣惠》卷二十二）、三五七散（《圣济总录》卷五十一）。
