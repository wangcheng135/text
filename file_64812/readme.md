序号：64828
名称：续命汤
组成：麻黄6分（去节），大枣10个（擘），桂心3分，防风3分，细辛3分，芎3分，甘草3分，芍药3分，人参3分，秦艽3分，独活3分，黄芩3分，防己3分，附子（炮，去皮）3分，白术3分，生姜5分。
出处：《千金翼》卷十六。
主治：大风，风邪入心，心痛达背，背痛达心，前后痛去来上下或大腹胀满微痛，一寒一热，心中烦闷，进退无常，面或青或黄。皆是房内太过，虚损劳伤。交会后出汗，汗出未除，或因把扇，或出当风而成劳，五俞大伤，风因外入，下有水因变成邪。
加减：None
功效：None
用法用量：以水1斗3升，先煮麻黄1沸，去上沫，纳诸药，煮取5升，去滓，纳大枣，煎取3升，分为3服；老小久病，服5合取汗。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：忌生葱、海藻、菘菜、生菜、猪肉、冷水、桃、李、雀肉。
附注：None
