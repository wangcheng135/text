序号：58274
名称：真火汤
组成：白术5钱，巴戟天1两，附子1钱，防风1钱，牛膝3钱，石斛3钱，萆薢2钱，茯苓3钱。
出处：《辨证录》卷二。
主治：冷痹，脚膝疼痛，行步艰难，自按其皮肉直凉至骨。
加减：None
功效：None
用法用量：水煎服。连服4剂而皮肉温矣，又服4剂而骨髓热矣，再服4剂脚膝之痛去，更服4剂而步履无艰难之态矣。
制备方法：None
临床应用：None
各家论述：方中用巴戟天为君，补火仍是补水之药，而辅佐之味又彼此得宜，不用肉桂，当归之品，温其血分，实有意义。盖补气则生精最速，生精既速，则温髓亦速矣。若一入血分之药，则沾濡迟滞，欲速而不达矣。萆薢原忌防风，使之相畏而相使，更复相宜，所以同群而共济矣。
用药禁忌：None
附注：None
