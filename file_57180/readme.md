序号：57195
名称：消积化痞丹
组成：白术5两，茯苓3两，六曲2两，地栗粉8两，鳖甲1斤（醋炙），人参5钱，甘草1两，白芍3两，半夏1两，白芥子1两，萝卜子5钱，厚朴5钱，肉桂3钱，附子1钱。
出处：《石室秘录》卷二。
主治：痞块。
加减：None
功效：None
用法用量：每服5钱，临睡送下，以美物压之。
制备方法：上为末，炼蜜为丸。
临床应用：None
各家论述：此方有神功，妙在用鳖甲为主，则无坚不入；尤妙用地栗粉，佐鳖甲以攻邪，又不耗散真气。其余各品，俱是健脾理正之药，则健脾而物自化。尤妙用肉桂、附子，冲锋突围而进，则鳖甲大军相继而入，勇不可当，又是和平之师，敌虽强横，自不敢抵抗，望风披靡散走，又有诸军在后，斩杀无遗，剿抚并用，有不告捷者哉？
用药禁忌：None
附注：本方方名，《医门八法》引作“栗粉丸”。
