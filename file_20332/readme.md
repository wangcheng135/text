序号：20339
名称：肉苁蓉丸
组成：肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），熟干地黄1两，钟乳粉1两，天雄1两（炮裂，去皮脐），天门冬1两半（去心，焙），五味子1两，桂心1两，人参1两（去芦头），干姜1两半（炮裂，锉），白术1两，远志1两，杜仲1两（去粗皮，炙微赤，锉），巴戟1两，牛膝1两（去苗），山茱萸1两，覆盆子1两，甘草半两（炙微赤，锉），川椒1两（去目及闭口者，微炒去汗），菟丝子2两（酒浸3日，晒干，别捣为末）。
出处：《圣惠》卷九十八。
主治：下元久冷，水脏伤惫，风虚劳损，不思饮食。
加减：None
功效：驻颜，益髭发，补神益气。
用法用量：每服30丸，空心以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：苁蓉丸（《普济方》卷二二○引《圣济总录》）
