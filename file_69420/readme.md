序号：69436
名称：猢狲骨熁膏
组成：猢狲骨2两，穿山甲骨2两，狗食系骨2两，腽肭脐2两，虎胫骨2两，野狸骨1两，水獭骨2两，猫儿食系骨2两。
出处：《圣惠》卷六十七。
主治：一切伤折并磋跌，骨碎压肿，晓夜疼痛不可忍。
加减：None
功效：None
用法用量：每用时，先以醋煮黄米粥，看损折痛处大小，入药末半钱，调令匀，摊于油单子上，裹之，上面以绵裹，系缚。重者不过3度验。其伤折处骨，先依法度排正后，即封裹。如贴药时疼痛，先用温酒调药末半钱服之，药入口其痛处立定。
制备方法：上为粗末，以米醋拌，入瓶子，以泥密封头令干，以大火烧令稍热为度，候冷取出，捣罗为末，瓷器中密盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
