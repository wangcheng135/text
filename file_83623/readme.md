序号：83641
名称：乌贼鱼骨丸
组成：乌贼鱼骨（去甲，炙焦）1两半，鹿茸（酥炙，去毛）1两半，诃黎勒皮1两半，当归（切，焙）1两半，白芍药1两半，山茱萸1两半，黄耆（锉）1两半，酸枣仁（微炒）1两半，地榆1两半，芎1两半，覆盆子（去梗）1两半，玄参1两半，白茯苓（去黑皮）1两半，熟干地黄（焙）1两半，荜澄茄（微炒）1两。
出处：《圣济总录》卷一五一。
主治：妇人月水不断，脐腹冷痛，腰腿痠疼。
加减：None
功效：None
用法用量：每服30丸，空心、食前米饮送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：乌贼鱼丸（《圣济总录》卷一五三）。
