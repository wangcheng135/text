序号：71066
名称：葱白汤
组成：葱白（长3-4寸）14茎，半夏1升，生姜8两，甘草3两，当归3两，黄耆3两，麦门冬1升，阿胶4两，人参1两半，黄芩1两，旋覆花1合。
出处：《千金》卷二。
主治：妊娠七月，忽惊恐摇动，腹痛，卒有所下，手足厥冷，脉若伤寒，烦热腹满短气，常苦颈项及腰背强。
加减：汗不出者，加麻黄2两。
功效：None
用法用量：上(口父)咀。以水8升，煮减半，纳清酒3升及胶，煎取4升。每服1升，日3夜1。温卧，当汗出。一方以黄雌鸡1只，割咽取血，纳酒中煮鸡，取汁以煎药。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌羊肉饧、海藻、菘菜等。
附注：葱白雌鸡汤（《圣惠》卷七十六）。本方改为丸剂，名“葱白丸”（见《中国医学大辞典》）。
