序号：77953
名称：踯躅摩风膏
组成：踯躅花1两，羌活1两，防风（去芦头）1两，芎1两，杏仁（汤浸，去皮）1两，细辛1两，当归1两，白蔹半两，白及半两，白芷半两，丹参半两，苦参半两，玄参半两，桂心半两，附子（去皮脐）半两，川乌头（去皮脐）半两，皂荚（去黑皮）半两，汉椒（去目）半两，莽草半两，川大黄半两。
出处：《圣惠》卷二十五。
主治：风，肢节多痛，肌肉顽痹，或遍体疮癣，或隐疹风瘙。
加减：None
功效：None
用法用量：每取1弹子大，摩于痛上。
制备方法：上细锉，以米醋1升，拌令匀，湿经3宿后，慢火炒令干，用腊月猪脂2斤，以慢火同煎1日，候药味出尽，以新布绞去滓，更以绵滤过，再入锅中煎，以柳木篦不住手搅成膏，候凝，收于瓷合中。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
