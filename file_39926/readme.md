序号：39936
名称：肥儿丸
组成：人参（去芦）5钱，白术（坚白者，去芦）5钱，橘红（刮净）5钱，白茯苓（去皮）4钱，甘草（去皮，炙）2钱，青皮（四花者，去瓤）3钱，缩砂仁3钱5分，木香2钱5分，山药（刮净）5钱，莲肉（去皮，去心）5钱，使君子（去皮）3钱，山楂子（蒸，取肉）3钱，三奇神曲（炒）3钱。
出处：《广嗣纪要》卷十五。
主治：小儿脾胃素弱，食少而瘦；或素强健，偶因伤食成积而瘦；或因久病之后而瘦。
加减：None
功效：健脾胃，进饮食，消积滞，杀疳虫，补疳痨，长肌肉。
用法用量：每服25-35丸，至50丸，陈仓米炒熟煎汤送下，不拘时候。
制备方法：上为极细末，用生荷叶包粳米煮熟，去荷叶，将米杵烂，以净布包扭出，再煮成糊，为丸，如麻仁大。
临床应用：None
各家论述：None
用药禁忌：None
附注：保婴丸（《寿世保元》卷八）。
