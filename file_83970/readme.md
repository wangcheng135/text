序号：83988
名称：五子衍宗丸
组成：甘州枸杞子8两，菟丝子8两（酒蒸，捣饼），辽五味子2两（研碎），覆盆子4两（酒洗，去目），车前子2两（扬净）。
出处：《摄生众妙方》卷十一。
主治：肾虚腰痛，尿后余沥，遗精早泄，阳萎不育。
加减：None
功效：男服此药，填精补髓，疏利肾气，种子。
用法用量：每服空心90丸，上床时50丸，白沸汤或盐汤送下，冬月用温酒送下。修合日，春取丙丁己午，夏取戊己辰戍丑未，秋取壬癸亥子，冬取甲乙寅卯。
制备方法：上各药俱择道地精新者，焙、晒干，共为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《陕西中医》(1986；7：314)：本方皆为植物种仁，味厚质润，既能滋补阴血，又蕴含生生之气，性平偏温，擅于益气温阳。方中菟丝子温肾壮阳力强；枸杞填精补血见长；五味子五味皆备，而酸味最浓，补中寓涩，敛肺补肾；覆盆子甘酸微温，固精益肾；妙在车前一味，泻而通之，泻有形之邪浊，涩中兼通，补而不滞。
用药禁忌：None
附注：None
