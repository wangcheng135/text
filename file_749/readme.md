序号：749
名称：六龙固本丸
组成：怀山药4两，巴戟肉4两，山茱萸肉4两，川楝子肉2两，黄耆1两，补骨脂2两（青盐3钱煎汤，拌半日，搓去皮，黄柏5钱酒煎，拌骨脂，炒），小茴香1两（盐2钱煎汤，拌楝肉，同炒干），人参2两，莲肉2两，木瓜2两，当归身2两，生地黄2两，白芍1两，川芎1两。
出处：《寿世保元》卷七。
主治：妇人赤白带下，不孕，及小产、血崩、五劳七情等致虚者。
加减：None
功效：生血固真，补心益肾。
用法用量：每服100丸，空心淡盐汤送下。
制备方法：用水3碗，童便2钟，拌浸1日，烘，又浸又烘干，上为细末，用斑龙胶一料为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
