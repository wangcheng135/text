序号：29364
名称：阿胶丸
组成：阿胶2两（捣碎，炒令黄燥），肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），艾叶1两半（微炒），川椒1两（去目及闭口者，微炒去汗），白芍药1两，当归1两（锉，微炒），芎1两，延胡索1两，熟干地黄1两，桂心1两，川大黄1两（锉碎，微炒），牛膝1两（去苗），牡丹1两，附子1两（炮裂，去皮脐），黄耆1两（锉）。
出处：《圣惠》卷六十七。
主治：从高坠下，伤折腕损，内伤五脏，微者唾血，甚者吐血。
加减：None
功效：None
用法用量：每服30丸，豆淋酒送下，1日3-4次。
制备方法：上为末，先用酒1升，煎3-5沸，将一半药末入酒内，调入面糊，以慢火煎令稠，入余上药末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
