序号：35981
名称：参乳丸
组成：人参末、人乳粉各等分。
出处：《医方集解》。
主治：一切虚怯。
加减：None
功效：交补气血，滋燥降火。
用法用量：口服。
制备方法：炼蜜为丸。
临床应用：None
各家论述：1.《医方集解》：人参大补元气，人乳本血液化成，用之以交补气血，实平淡之神奇也。_x000D_
2.《医方论》：平补气血，一壮水之源，一益气之主。后人两仪膏药，从此化出。
用药禁忌：人乳能湿脾、滑肠、腻膈，久服亦有不相宜者。惟制为粉，则有益无损。须用一妇人之乳为佳，乳杂则其气杂：又须旋用，经久则油羶。
附注：炖乳取粉法，取无病年少妇人乳，用银瓢或锡瓢，倾乳少许，浮滚水上炖，再浮冷水上立干，刮取粉用，如摊粉皮法。
