序号：62398
名称：旋覆花饮
组成：旋覆花1两，赤茯苓（去黑皮）2两，吴茱萸（汤浸，焙炒）3分，前胡（去芦头）1两1分，木香1两半，郁李仁（去皮，炒）1分，半夏（汤洗7遍去滑，姜汁制）2两，大腹5枚（连皮子，锉）。
出处：《圣济总录》卷八十四。
主治：乍处江岭，未伏水土，食饮不宜，兼脾湿，脚气发动，时复心闷，面目脚膝浮肿，气乏，唇口青黑，胸膈烦热，见食吐呕，心腹妨痛，冷气结聚。
加减：或要疏利，别入槟榔末2钱匕服之。
功效：None
用法用量：每服5钱匕，水1盏半，加生姜1分（拍碎），煎取7分，去滓，早、晚食前服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
