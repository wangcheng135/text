序号：76533
名称：蒺藜子汤
组成：蒺藜子（炒，去角）2两，仙灵脾1两半，防风（去叉）1两半，芎1两半，萆薢1两半，白石脂1两半，枳壳（去瓤，麸炒）1两半，桂（去粗皮）半两，黄芩（去黑心）半两，白术1两，麻黄（去根节）1两，羌活（去芦头）1两，天雄（炮裂，去皮脐）1两，羚羊角屑1两，黄连（去须）1两，旋覆花（炒）3分。
出处：《圣济总录》卷十一。
主治：风疹发歇不愈，或赤或白，瘙痒至甚。
加减：None
功效：None
用法用量：每服5钱匕，用水1盏半，入生姜2片，乌梅肉半枣大，同煎至8分，去滓温服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
