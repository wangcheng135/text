序号：70419
名称：犀角丸
组成：犀角屑半两，羚羊角屑半两，牛黄1分（细研），麝香1分（细研），天麻1两，白附子1两（炮裂），蝉壳半两，牛膝半两（去苗），防风半两（去芦头），附子1两（炮裂，去皮脐），桂心半两，当归半两，芎半两，羌活半两，白僵蚕半两（微炒），五加皮半两，乌蛇肉1两（酒浸，炙微黄），薏苡仁半两，麻仁1两（去根节），朱砂1两（细研，水飞过），远志半两（去心），金箔50片（细研），银箔50片（细研），仙灵脾1两，道人头1两，地骨皮半两。
出处：《圣惠》卷二十二。
主治：风軃曳。手脚不能收摄，肌肤顽痹，筋脉不利，心神恍惚，骨节疼痛。
加减：None
功效：None
用法用量：每服20丸，渐至30丸，食前以温酒送下。
制备方法：上为末，入研了药令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌生冷羊血。
附注：None
