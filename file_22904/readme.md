序号：22912
名称：行气活血汤
组成：葛根9g，草河车9g，白芷9g，郁金9g，枳壳9g，生甘草9g，红花15g，泽兰15g，赤芍12g，白芍12g，五味子12g。
出处：《古今名方》引关幼波方。
主治：慢性肝炎，早期肝硬化，长期肝功能不正常，证属气滞血瘀型者。
加减：若气短心悸，加生黄耆15g；纳少腹胀，加党参15g，白术9g，焦山楂、麦芽、神曲各12g；有瘀斑，舌质绛者，加石见穿、蒲公英各15g；肝脾肿大不消，加王不留行、生牡蛎、地龙各12g；肝痛不止，加没药、五灵脂各9g。
功效：舒肝理气，活血化瘀。
