序号：46454
名称：济阴丹
组成：熟地5两，人参1两半，当归3两，川芎1两，肉桂1两半（去皮），干姜1两（炒），秦艽2两，木香1两，糯米1合（炒）。
出处：《医略六书》卷二十七。
主治：久不孕，脉软弦涩者。
加减：None
功效：None
用法用量：每服3钱，温酒送下。
制备方法：上为末，炼蜜为丸。
临床应用：None
各家论述：气血两亏，寒凝经脉，不能输泄而天癸来迟，久不孕育焉。熟地补血以滋血海，人参扶元以通血脉，当归养血荣经，川芎活血行气，肉桂温经暖血，干姜暖胃散寒，秦艽活血脉以通经，木香调胃气以化气，糯米实土膏以滋营经脉也。炼蜜丸之，温酒下之，使寒滞消散，则血气充盈而经脉调和，天癸无来迟之患，即久不孕者，尚可图效耳。
用药禁忌：None
附注：None
