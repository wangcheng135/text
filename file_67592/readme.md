序号：67608
名称：小乌沉汤
组成：乌药（去心）10两，甘草（炒）1两，香附子（沙盆内断去皮毛，焙干）20两。
出处：《局方》卷三（绍兴续添方）。
主治：气不调和所致的心腹刺痛、便血。心腹刺痛。气不调和，便血不止。气闪血瘀，心腹刺痛。
加减：None
功效：调中快气。
用法用量：每服1钱，入盐少许，或不着盐，沸汤点服，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：1.《张氏医通》：此乃《局方》乌沉汤之变法，中去参、姜，而易香附，其破气之力虽峻，而功力稍逊，故以小字加之，立方之意微矣。_x000D_
2.《成方便读》：此方全不用血药，但以香附、乌药辛苦而温，芳香而燥，专行气分之品，可宣可散；沉香之降气，上至天而下至泉，使表里上下无一毫留着，自然血自行而痛自消矣；用甘草者，痛则筋脉急而正气伤，以和中而缓急也。
用药禁忌：None
附注：乌附汤（《丹溪心法附余》卷十四）、抑气散（《张氏医通》卷十四）。按：本方名小乌沉汤，但方中无沉香，疑脱，《张氏医通》本方用法：三味水煎，即用药汁磨沉香五分，入盐一字，热服。
