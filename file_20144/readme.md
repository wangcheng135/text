序号：20151
名称：沉香羊肉丹
组成：羊肉1斤（去筋膜），葱白1握，陈皮1两，青盐5钱，破故纸（炒）、远志、生地黄、花椒5钱（去目合口者，用好酒煮糊，加葱白等再煮羊肉），牛膝1两，干地黄1两，木香1两，韭子1两，菖蒲1两，沉香1两，覆盆子1两，木瓜1两，北五味子1两，麝香1钱，胡桃肉2两，鹿茸（酥炙）4两，苁蓉1两，枸杞子1两，山药1两，茴香1两。
出处：《普济方》卷二一九引《兰室秘藏》。
主治：None
加减：None
功效：升降阴阳，调理三焦，通经络，生气血，壮元阳，补脏腑。
用法用量：每服30丸，空心以盐汤送下。
制备方法：上为细末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
