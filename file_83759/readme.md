序号：83777
名称：乌梅膏
组成：人参（去芦）半两，檀香（锉，晒）半两，薄荷叶半两，乌梅肉（薄切，用屋瓦慢火焙干）1两，干葛1两，缩砂仁1两，百药煎1两，粉草1两。
出处：《活幼心书》卷下。
主治：小儿诸渴不止，吐泻后口干无味，及病中昏闷，咽痛不利，心悸似惊。
加减：None
功效：None
用法用量：每服1丸或2丸，无时含咽。儿小者，麦门冬熟水化服。
制备方法：上除檀香不过火，乌梅肉另焙，余6味或晒或焙，仍同前2味研为细末，炼蜜为丸，如芡实大。
临床应用：None
各家论述：None
用药禁忌：None
附注：此药品味不寒不燥，用得其宜。暑月出路含化，则津液生而烦渴少，神效异常。又，本方方名，据剂型当作“乌梅丸”。
