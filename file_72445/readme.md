序号：72461
名称：痛宁酒
组成：当归2两，秦艽2两，川芎1两，白芍药1两，生地黄1两，苍术1两，羌活1两，黄芩1两，猪苓、泽泻、防风、茵陈（铃儿）、苦参、虎骨（打细，酒煮）。
出处：《简明医彀》卷三。
主治：风寒湿火相乘，遍身筋骨走痛，白虎历节诸风疼痛。
加减：上身痛，加灵仙、升麻；下身痛，加牛膝、黄柏；臂痛，加桂枝；冬加麻黄；痛甚，加乳香研末调服。
功效：None
用法用量：每服80丸，空心以酒送下。
制备方法：上药以绢袋盛内，用酒浸泡，封固，14日后取酒饮；渣晒干为末，米糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌发风、动火之物。
附注：方中猪苓以下诸药用量原缺。
