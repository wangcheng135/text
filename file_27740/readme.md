序号：27748
名称：诃子饮
组成：诃子（煨，去核）半两，青皮（去白）半两，麦门冬（水浥，去心）半两，槟榔4个，半夏3分（汤浸7次），甘草1分（炙）。
出处：《本事》卷三。
主治：咳嗽痰涎，致中膈不利，纳食减少。
加减：None
功效：利膈去涎，思食止嗽，健运中宫。
用法用量：每服4钱，水2盏，加生姜7片，同煎至7分，去滓温服，每日2-3次。
制备方法：上为粗末。
临床应用：None
各家论述：《本事方释义》：诃子气味温涩，入手阳明、足太阴；青皮气味苦辛温微酸，入足少阳、厥阴；麦冬气味甘寒微苦，入手太阴、少阴；槟榔气味辛温，入足阳明；甘草气味甘平，入脾。以辛温之药，健运中宫，气旺则肺金有所恃，孰谓肺病必用滋腻乎？
用药禁忌：None
附注：本方方名，《普济方》引作“诃子汤”。
