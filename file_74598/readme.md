序号：74614
名称：解酲止泻汤
组成：白术1两，山茱萸1两，茯苓1两，柞木5钱，黄连3-5分，白芍5钱，附子1分。
出处：《辨证录》卷七。
主治：酒湿伤脾肾，终年饮酒，不知禁忌，逞醉入房，过于泄精，久则脾气大伤，变成水泻，一感风寒，遂大泻不止，如溏如积。
加减：None
功效：大补脾肾，解其湿热之毒。
用法用量：水煎服。多服为佳。或10服之后，改为丸剂，朝夕服三月。
制备方法：None
临床应用：None
各家论述：此方脾肾双补之药也。用柞木、黄连以解其酒毒；用芩、术以消其水湿；用芍药以敛其耗脱之阴；用附子一分，引群药入肾，以扫荡其湿热，而非助其命门之虚阳也。
用药禁忌：None
附注：None
