序号：13213
名称：人参丸
组成：人参1两，白茯苓1两，川牛膝（去苗，酒浸）1两，地骨皮（真者）、川当归（去芦，酒浸1宿）、熟地黄各等分。
出处：方出《续本事》卷一，名见《普济方》卷二二六。
主治：丈夫、妇人房事不节，渐至虚损，行步如踏空，夜梦从高坠下，及梦大水、溺水、诸般水梦。
加减：None
功效：None
用法用量：每服30丸，空心用温酒或盐汤送下。常服只20丸，用3-5匙干饭压下。服3-5日后，每日饱饭后及临卧时，服局中鸡苏丸50粒，不嚼破，熟水吞下，次又服前药。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：青春丸（《普济方》卷二二六）。
