序号：83328
名称：七子丸
组成：五味子1两（净），菟丝子1两（先筛去灰，却用酒浸2-3日，蒸，擂细，焙干，研用之），韭子1两（炒），覆盆子1两（去蒂，酒洗），蛇床子半两，黑附子1两（炮，去皮脐），白茯苓半两（去皮），原蚕蛾1两（酒煮），肉苁蓉1两（酒焙干，先洗），鹿茸1两（酒炙，去皮毛），益智子1两（去皮），沉香半两（不见火），黄耆半两（蜜炙），远志半两（汤洗，去心），阳起石1两（煅，细研如粉），熟地黄1两（汤洗，酒拌蒸）。
出处：《普济方》卷三三六引《便产须知》。
主治：妇人闻凌霄花气不孕。
加减：弱甚者，加天雄半两（炮，去皮）；脚腰痠痛者，加杜仲1两（去皮，姜汁炒去丝），石斛1两（去根）。
功效：None
用法用量：每服60-70丸，空心、盐酒或盐汤吞下。只与男服。
制备方法：上为细末，酒煮糯米糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
