序号：69917
名称：琥珀还睛丸
组成：麦冬4两5钱，天冬4两5钱，党参（去芦）4两5钱，熟地4两5钱，知母4两5钱，黄柏4两5钱，川芎4两5钱，菊花4两5钱，山药4两5钱，枳壳（炒）4两5钱，苁蓉（炙）4两5钱，青葙子4两5钱，当归4两5钱，菟丝子4两5钱，茯苓4两5钱，枸杞子4两5钱，杜仲炭4两5钱，杏仁（去皮，炒）4两5钱，生地9两，沙苑子6两，羚羊1两5钱，黄连1两5钱，犀角9钱，石斛4两，炙甘草2两，琥珀3两。
出处：《北京市中药成方选集》。
主治：阴虚大盛，内外障翳，年老气虚，目昏多眵，瞳仁反背，视物昏花。
加减：None
功效：滋阴清热，明目退翳。
用法用量：每服3钱，温开水送下，日服2次。
制备方法：上方纳羚羊、犀角、琥珀另研兑入，余药共研为细末，过罗，炼蜜为小丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
