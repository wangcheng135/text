序号：2685
名称：天门冬酒
组成：天门冬5升（去心，捣绞取汁，慢火煎如稀饧），湿荆25束（每束3尺围，各长2尺5寸，当中心以火烧两头，盛取沥1升半，煎取7合），青竹30束（每束3尺围，各长2尺5寸，当中心以火烧两头，盛取沥3升，煎取1升半），生地黄（锉）5升（粗大者，洗净，控干，捣取汁3升，煎取1升半），生五加皮30斤（净洗，控干，锉，大釜内以水4石，煮至5斗，去滓，澄清取汁，以铜器盛入大釜内，重汤煎之至3斗5升，盛煎与前4味汁合匀），白糯米1石5斗，曲8斤（曝干，捣末）。
出处：《圣济总录》卷八十四。
主治：脚气疼痛。
加减：None
功效：None
用法用量：以上药汁，取6升浸曲末；5日曲中沸起第1酘，将米7斗净淘20遍，置净席上，以生布挪干，然后纳甑中蒸之为饙，即以前药汁纳净盆中，拌饭令匀，候冷热，如常醖酒法，入净瓮中密封。经3日后，入第2酘，更净淘米4斗，一如前法蒸炊后，放令冷热得所，投入前酒瓮中封头，又经3日，即入桂(去粗皮)6两、甘草(炙)6两，白芷6两，当归(切，焙)6两，芎6两，麻黄(去根节)6两，干姜(炮)1斤，五加皮(炙)1斤，附子(炮裂，去皮脐)5两，牛膝(去苗，锉)9两，白糯米4斗(净淘，控干)。上11味，除米外，并锉捣如米粒，入米拌匀，同纳甑中，久蒸极熟，倾出摊，候似人体冷暖。投入煎酒瓮中，3日后，可服少许尝之。若得中即密封瓮头，更候14日后，压取清酒，入不津器中盛之。每服4合，温饮之，1日2次，渐加至半盏，以知为度。饮此酒只可微温，不可过热。
