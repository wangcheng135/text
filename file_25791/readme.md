序号：25799
名称：当归膏
组成：当归1斤4两（酒洗），芍药8两（微炒），生地黄半斤（酒洗），薏仁1斤（糯米炒，去粉），茯苓6两，白术10两（泻者，黄土微炒），莲肉半斤（去心），山药8两（炒），陈皮4两，人参3两（脉微者，倍之），甘草1两（半炙半生），枸杞子4两。
出处：《医统》卷四十六。
主治：脾胃虚弱。
加减：内外俱热如蒸者，加青蒿汁1碗，银柴胡1两，胡黄连5钱；内热蒸者，加地骨皮4两，牡丹皮2两、知母1两；女人，加童便浸香附子1两，乌药2两，玄胡2两；男女胃脘痛者，加草豆蔻1两；寒，加肉桂；虚火阵阵作痛，加炒黑山栀仁半两；头昏目晕者，加天麻2两、钟乳粉1两；头虚痛者，加大川芎2两；咳嗽，加贝母3两，紫菀1两，五味子1两；肺热者，加麦门冬3两，天门冬1两，桔梗、百部各1两；足膝软弱或痠者，加牛膝4两，石斛2两；腰背痛者，加杜仲6两，橘核仁1两。
功效：养血和中。
用法用量：None
制备方法：上咀净称，用水20斤，文武火熬成膏，加熟蜜于内，冬用4两，春用5两，夏秋用6两，依法再熬。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
