序号：27409
名称：补益小泽兰丸
组成：泽兰2两，藁本1两，白术1两，白芍药1两，厚朴1两半（去粗皮，涂生姜汁炙令香熟），龙骨1两半，人参1两（去芦头），当归1两（锉碎，微炒），甘草1两（炙微赤，锉），阳起石2两（酒煮半日，细研，水飞过），赤石脂1两（细研），桂心1两半，紫石英1两（细碎，水飞过），钟乳粉1两半，川椒1两（去目及闭口者，微炒去汗），白石英1两（细研，水飞过），肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），白矾1两半（烧灰），干姜1两（炮裂，锉），石膏2两（细研，水飞过），山茱萸1两，芜荑3分，柏子仁1两，芎1两。
出处：《圣惠》卷七十。
主治：妇人劳冷虚损，饮食减少，面无光色，腹中时痛，女子月信不调，翕翕少气无力。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，入研了药令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
