序号：83565
名称：乌药顺气散
组成：乌药（去木）2两，麻黄（去根节）2两，橘皮2两，甘草（炙）1两，川芎1两，枳壳（麸炒，去瓤）1两，白僵蚕（去丝嘴，炒）1两，白芷1两，桔梗1两，白姜（炮）半两。
出处：《三因》卷二。
主治：风气不顺，手脚偏枯，流注经络，并湿毒进袭，腿膝挛痹，筋骨疼痛，头目旋晕；瘫痪，语言謇涩，筋脉拘挛；脚气，步履艰难，脚膝软弱；妇人血风，老人冷气，上攻胸臆，两胁刺痛，心腹膨胀，吐泻肠鸣。
加减：None
功效：疏风顺气。
用法用量：每服2钱匕，水1盏，加生姜3片，薄荷7叶，煎至7分，空心服。治气，去薄荷，用杏子2枚同煎。
制备方法：上为末。
临床应用：None
各家论述：《医方集解》：此手太阴、足厥阴药也。风盛则火炽，故有痰火冲逆而上，此里气逆也。然中风必由外感风寒而发，内虚而外邪乘之，此表气逆也。麻黄、桔梗，肺家之药，发汗而祛寒；川芎、白芷，头面之药，散风而活血；枳壳利气行痰；僵蚕消化散结；干姜温经通阳；甘草和中泻火；乌药能通行邪滞诸气。此乃先解表气而兼顺里气者，气顺则风散。风邪卒中，当先治标，若气虚病久者，非所宜也。
用药禁忌：孕妇不可服。
附注：乌药顺气汤（《嵩崖尊生》卷七）。
