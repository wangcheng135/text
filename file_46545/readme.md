序号：46560
名称：千金汤
组成：青蒿（酒浸1宿，焙）1两，柴胡（去苗）1两，秦艽（童便浸1宿，焙）1两，柳枝1两，桃枝1两，茯神（去木）1两，麻黄（去根节）1两，桂（去粗皮）1两，知母（切，焙）1两，白茯苓（去黑皮）1两，鳖甲（去裙襕，醋炙）1两，枳壳（去瓤，麸炒）1两，常山1两，天灵盖1枚（童便、酒共和浸，炙），槟榔（锉）半两。
出处：《普济方》卷二三八。
主治：尸疰，鬼疰，一切劳疾。
加减：None
功效：None
用法用量：每服5钱，水1盏半，童便半盏，加薤白5寸，豉5粒，同煎至1盏，去滓热服。10服后，十指甲中生毛为候。或青白色者不治。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
