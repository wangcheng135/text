序号：73952
名称：槐实丸
组成：槐实4两（干肥者，拣令净，水洗过，放干，慢火上麸炒令焦，微似黑色），皂荚6两（不蚛者，锉，长3-4寸，用长流水5升，黑豆1升同煮令豆香熟为度，去黑豆不用，取皂荚焙干，刮去黑皮涂酥，慢火炙令焦），木香半两，芎1两，枳壳（去瓤，麸炒）1两，菊花1两，牵牛子2两（慢火炒令微焦黑色，别捣罗，取末1两用），槟榔3分（鸡心者，锉）。
出处：《圣济总录》卷十二。
主治：风气头目昏眩。
加减：None
功效：化痰涎，利胸膈。
用法用量：每服20丸，食后、临卧荆芥汤送下。
制备方法：上为末，炼蜜成剂，再入臼内，捣令熟，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
