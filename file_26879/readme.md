序号：26887
名称：朴沉化郁丸
组成：广木香7钱，蔻仁7钱，砂仁7钱，青皮（醋炒）7钱，南柴胡7钱，沉香7钱，元胡（醋制）7钱，甘草7钱，公丁香7钱，檀香7钱，香附（醋制）3两，厚朴（姜制）1两5钱，枳壳（麸炒）1两，片姜黄3钱，广皮2两，莪术（醋制）5钱，良姜5钱，肉桂（去粗皮）3钱。
出处：《全国中药成药处方集》（天津方）。
主治：胸腹胀满，消化不良，呕吐恶心，停食停水，气滞闷郁，胃脘刺痛。
加减：None
功效：舒气化瘀，开胃消食。
用法用量：每服1丸，白开水送下。
制备方法：上为细末，炼蜜为丸，3钱重，蜡皮或蜡纸筒封固。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
