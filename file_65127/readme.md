序号：65143
名称：脱甲散
组成：柴胡3钱（去芦），川当归（净洗）2钱，龙胆草3钱（去芦），白茯苓2钱半，人参2钱，知母3钱，甘草（炙）4钱，川芎2钱，麻黄2钱（去根节）。
出处：《活幼口议》卷十六。
主治：婴孩小儿，伤寒体热，头目昏沉，不思饮食，夹惊夹食，寒热，大小便秘涩，或赤或白，烦躁作渴，冷汗妄流，夹积伤滞，膈满胀急，青黄体瘦，日夜大热；及疗伤风伤暑，惊痫客忤，筋骨、肾脏、疳气等热。
加减：None
功效：散热扶表救里。
用法用量：每服1大钱，水小小盏，入小葱白（连须）1寸同煎，至半温服，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
