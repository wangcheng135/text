序号：4401
名称：木香丸
组成：木香半两，人参半两（去芦头），赤茯苓半两，甘草半两（炙微赤、锉），汉椒半两（汤浸，去目及闭口者，微炒去汗），桂心1两，细辛半两，赤芍药半两，陈橘皮1两（汤浸，去白瓤，焙），川大黄1两（锉碎，微炒），干姜半两（炮裂，锉），附子半两（炮裂，去皮脐），郁李仁1两（汤浸，去皮，微炒），厚朴1两（去粗皮，涂生姜汁，炙令香熟），诃黎勒皮1两半。
出处：《圣惠》卷五十。
主治：膈气，饮食不下，呕逆不定，日渐羸瘦。
加减：None
功效：None
用法用量：每服20丸，以生姜汤送下，不拘时候。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
