序号：10511
名称：古庵心肾丸
组成：熟地3两，生苄（俱怀庆者，酒浸，竹刀切）3两，山药3两，茯神（去木）3两，山茱萸肉（酒浸，去核）1两，枸杞子（甘州者，酒洗）1两，龟版（去裙，醋炙）1两，牛膝（去芦）1两，鹿茸（火去毛，醋炙）1两，当归（去芦、酒洗）1两5钱，泽泻（去毛）1两5钱，黄柏（炒褐色）1两5钱，辰砂（为衣）1两，黄连（去毛，酒洗）1两，生甘草半两，牡丹皮（去心）1两。
出处：《丹溪心法附余》卷十九。
主治：肾水亏乏，心火上炎，发白无子及惊悸怔忡，遗精盗汗，目暗耳鸣，腰痛足痿。
加减：None
功效：补血生精，宁神降火。
用法用量：每服50丸，渐加至100丸。空心温酒或淡盐汤任下。
制备方法：上为细末，炼蜜为丸，如梧桐子大，辰砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中丹皮用量原缺，据《医学入门》补。
