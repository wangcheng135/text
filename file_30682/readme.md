序号：30690
名称：鸡骨丸
组成：雄鸡骨1具（炙令黄），赤茯苓半两，石膏半两（细研，水飞过），川大黄半两（锉碎，微炒），赤芍药半两，紫菀半两（洗去苗土），白矾半两（烧灰），陈橘皮半两（汤浸，去白瓤，焙），细辛半两（洗去苗土），附子半两（炮裂，去皮脐），黄芩3分，桂心3分，甜葶苈3分（隔纸炒令香）。
出处：《圣惠》卷八十八。
主治：小儿丁奚，骨中微热，腹内不调，食不为肌肤，或苦寒热，腹大。
加减：None
功效：None
用法用量：每服5丸，以粥饮送下，日3次。
制备方法：上为末，炼蜜为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：大鸡骨丸（《普济方》卷三九三）。
