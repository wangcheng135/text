序号：73153
名称：山栀子汤
组成：当归、芍药（赤者）、茯苓（赤者）、甘草、山栀子各等分。
出处：《鸡峰》卷十八。
主治：五淋及血淋。肾气不足，膀胱有热，水道不通，淋沥不宣，出少起多，脐腹急痛，蓄作有时，劳倦即发，或尿如豆汁，或如砂石，或冷淋如膏，或热淋便血。
加减：None
功效：None
用法用量：每服2钱，水1盏，煎至8分，温服。
制备方法：上为细末。
临床应用：None
各家论述：《医学实在易》：此方用栀、苓治心腹，以通上焦之气，而心火清；归、芍滋肝肾，以安下焦之气，而五脏阴复；甘草调中焦之气，而阴阳分清，则太阳之气化，而膀胱之府洁矣。
用药禁忌：None
附注：五淋散《局方》卷六（宝庆新增方）、五淋汤（《医学实在易》卷七）。
