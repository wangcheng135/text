序号：3107
名称：天麻浸酒
组成：天麻半两，骨碎补半两，松节（锉）1两，败龟（醋炙）1两，龙骨半两，虎骨（酒炙）半两，乌蛇（酒浸，去皮骨，炙）半两，白花蛇（酒浸，去皮骨，炙）半两，恶实根（切，焙）半两，附子（炮裂，去皮脐）1枚，羌活（去芦头）半两，独活（去芦头）半两，牛膝（酒浸，切，焙）半两，当归（切，焙）1两，芎1两，大麻仁2两，干熟地黄（焙）半两，茄子根（切，焙）2两，原蚕沙（炒）2两。
出处：《圣济总录》卷七。
主治：摊缓风，不计深浅，久在床枕。
加减：None
功效：None
用法用量：每服1盏，不拘时候温服。
制备方法：上（口父)咀，如麻豆大，用酒2斗浸，密封，春、夏3日，秋冬7日。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
