序号：23306
名称：芎丸
组成：芎1两，龙骨1两，白茯苓（去黑皮）1两，紫石英（捣研）1两，防风（去叉）1两，厚朴（去粗皮，生姜汁炙，锉）1两，细辛（去苗叶）1两，铁精（捣研）1两，甘草（炙，锉）1两1分，枳实（去瓤，麸炒）1两1分，丹参1两1分，桂（去粗皮）1两1分，蜀椒（去目并闭口，炒出汗）1两1分，人参1两1分，大黄（锉，炒）1两1分，干姜（炮）1两1分，附子（炮裂，去皮脐）1两1分，菖蒲（九节者，去须，米泔浸，切，焙）1两1分，白芥子（生，研用）3分，吴茱萸（汤浸，焙，炒）3分，禹余粮（煅，醋淬）1两3分，远志（去心）1两半。
出处：《圣济总录》卷五。
主治：心中风，惊恐愁忧，烦躁错乱，若风邪流入五脏，则往来烦闷悲啼，吸吸短气，发时恍惚喜卧，或心中涎涌，或怒起颠倒，手足厥冷，饮食呕逆。
加减：None
功效：None
用法用量：每服7丸，温熟水送下，食后、临卧各1次。
制备方法：上十九味为末，与别研三味和匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
