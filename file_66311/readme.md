序号：66327
名称：雀附丸
组成：雀儿30枚（取肉），附子4两（炮裂，去皮脐，捣罗为末），萆薢2两（锉），胡椒1两半，白芜荑1两半，干姜1两半（炮裂，锉），茴香子1两半，青橘皮1两（酒浸，去白瓤，焙），艾叶4两（捣罗为末，与附子、雀儿同于锅中，先铺艾叶末1重，次铺雀儿1重，次铺附子1重，以酽醋1斗，慢火熬成膏），川椒1两半（去目及闭口者，微炒去汗）。
出处：《圣惠》卷九十八。
主治：脾胃久积虚冷，心腹气痛，时自泄痢，水谷不消，少思饮食，颜色萎黄。
加减：None
功效：None
用法用量：每日空心以温酒送下30丸；盐汤送下亦得。
制备方法：上为末，以雀儿膏和，更捣300-500杵，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
