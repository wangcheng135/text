序号：72235
名称：紫金丸
组成：龙脑1钱，麝香1钱，滴乳香1钱，雄黄1钱，密陀僧1钱，砒霜半分，朱砂、阿魏1分，安息香1分。
出处：《博济》卷三。
主治：肠风泻血，痔漏。
加减：None
功效：None
用法用量：每服1丸，空心好温茶送下。心痛、水泻，生姜汤送下；食癥，面汤送下；气痛，温酒送下。
制备方法：上乳香、安息、阿魏以热水浸之通软，于大碗中，以乳石研令细，其有龙脑、麝香、朱砂、密陀僧、雄黄都研为末，其砒霜入绿豆粉2两同研令细，更用巴豆3粒，去皮心膜，以水1大碗浸之，一日六度换水，取出后用十重纸出油，亦研细，与药都一处，研匀，方入湿药中，全和，摊在碗中，俟通手，便丸之如豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方原名“紫金膏”，与剂型不符，据《圣济总录》改。方中朱砂用量原缺。
