序号：77384
名称：遇明丸
组成：皂角3斤（2斤烧成灰，装在新瓷罐内，用瓷碟盖口，不令出烟，不用碟子后，用纸2张，水湿过，盖罐口，纸干罐冷为度），何首乌（去粗皮）6两，牵牛头末3两（黑白各半），薄荷叶（去土）3两。
出处：《御药院方》卷十。
主治：风痰，头目昏眩，视物(目荒)(目荒)，目见黑花飞蝇。
加减：None
功效：清神水，行滞气，下流饮。
用法用量：每服20丸。食后煎生姜汤送下，渐加至30丸，1日1次。
制备方法：上为细末，后用皂角1斤，热水浸软，去皮弦子，用瓤，以酒2升，搓揉成浓汁，用新布滤去滓，入面1匙，同熬成膏子，入上4味为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
