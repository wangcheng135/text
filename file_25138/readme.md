序号：25146
名称：观音救苦丹
组成：茯苓2两，木瓜2两，麦冬2两，生地2两，当归2两，藿香1两，菖蒲1两，茅术1两，半夏1两，腹皮1两，白矾1两，陈皮1两，防风1两，黄芩1两，杏仁1两，山栀8钱，柴胡8钱，木通8钱，香附8钱，桔梗8钱，泽泻8钱，远志8钱，砂仁5钱，川芎5钱，油朴5钱，羌活5钱，独活5钱，青黛5钱，川连5钱，雄黄5钱，甘草5钱，黄柏5钱，神曲5钱，枳壳5钱，广木香5钱，青木香5钱，嫩苏梗5钱，广郁金5钱。
出处：《全国中药成药处方集》（吉林方）。
主治：四时感冒，伤食感冒，头痛体痛，发热畏冷，泛恶呕吐，泄泻。山岚瘴气，霉浊秽气，卒然中人，四肢清冷，呕吐搅闹，神昏吐沫；湿浊中阻，头痛身重，大便溏泄，不欲饮水。
加减：None
功效：解表和中，化湿解毒。
用法用量：每服1钱4分重丸1丸，或3分5厘重丸4丸，鲜姜汤为引。服后取汗。
制备方法：上为细末，炼蜜为丸，1钱4分重，或3分5厘重，均朱砂为衣，瓷罈存贮。
临床应用：None
各家论述：None
用药禁忌：忌食生冷。
附注：None
