序号：33009
名称：连翘丸
组成：连翘1两，桑白皮1两，白头翁1两，牡丹1两，防风1两，黄柏1两，桂心1两，香豉1两，独活1两，秦艽1两，海藻半两。
出处：《千金》卷五。
主治：小儿结风气肿。
加减：None
功效：None
用法用量：3岁儿饮服5丸，加至10丸；5岁以上者，以意加之。
制备方法：上为末，炼蜜为丸，如小豆大。
临床应用：None
各家论述：《千金方衍义》：方中防风、白头翁、香豉以散风毒；连翘、秦艽、独活、桑白皮、海藻以散气肿；然风药气药非得血药不能透达荣分，以散结核，又须黄柏、桂心寒热交攻；用牡丹者，专和黄柏、桂心之寒热也。
用药禁忌：None
附注：《圣惠》无桑白皮，有榆白皮。
