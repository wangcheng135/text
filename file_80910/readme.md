序号：80926
名称：飞龙夺命丹
组成：朱砂（飞）2两，明雄黄1两，灯心炭1两，人中白（漂、煅）8钱，明矾5钱，青黛（飞）5钱，梅冰4钱，麻黄（去节）4钱，真珠3钱，牙皂3钱，当门子3钱，硼砂3钱，西牛黄2钱，杜蟾酥1钱5分，火消1钱5分，飞真金300页。
出处：《霍乱论》卷下。
主治：痧胀绞痛，霍乱转筋，厥冷脉伏，神昏危急之证；及受温暑瘴疫、秽恶阴晦诸邪，而眩晕痞胀，瞀乱昏狂；或卒倒身强，遗溺不语，身热瘛疭，宛如中风；或时证逆传，神迷狂谵；小儿惊痫，角弓反张，牙关紧闭。
加减：None
功效：None
用法用量：以少许吹鼻取嚏；重者再用开水调服1分；小儿减半。
制备方法：上药各为细末，和匀，瓷瓶紧装。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
