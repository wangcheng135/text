名称：万应神曲膏
组成：前胡4两，大黄4两，苍术4两，莪术4两，防风4两，良姜4两，姜黄4两，山楂4两，柴胡4两，厚朴4两，紫苏4两，豆蔻4两，葛根4两，槟榔4两，薏米4两，黄芩4两，荆芥4两，麻黄4两，青皮4两，使君子4两，甘草4两，黄柏4两，百合4两，栀子4两，薄荷4两，羌活4两，陈皮4两，蒲黄4两，扁豆4两，杏仁4两，车前子4两，泽兰4两，独活4两，木香4两，益母草4两，麦芽4两，乌药4两，桔梗4两，腹皮4两，砂仁4两，诃子4两，猪苓4两，茯苓4两，三棱4两，芡实4两，草果4两，半夏4两，淮药4两，木通4两，藿香4两，枳实4两，建泽泻4两，香薷4两，菖蒲4两，黄连4两，木瓜4两，香附4两，枳壳4两，小豆4两，花椒4两。
出处：《验方新编》卷十一。
主治：四时不正之气，感冒发热，头眩咳嗽及伤食腹痛，痞满气痛，呕吐泄泻痢疾，饮食不进，不服水土。
加减：外感咳嗽，疟疾呕吐，俱加生姜同煎；泄泻，加真乌梅同煎；痢疾，大人用4-5钱，小儿用2-3钱，加陈茶叶同煎。
功效：搜风解表，开胸快膈，调胃健脾，消积进食，解酒，止泻利水。
用法用量：又用鲜青蒿4斤，凤尾草2斤，苍耳草3斤，大？？草3斤，小？？草3斤，以上5味浓汁同煎。又用小麦15斤（洗净，略蒸，晒干），曲粉6两。临时先将药与曲粉同拌，入药草水拌，揉做成块子，外用荷叶包好，以苎麻扎紧，上笼蒸1个时辰，取出凉3-4时，以冷为度，装入桶内，1层稻禾草，1层神曲，盖密，须12天取出，晒过月余，极干，然后刷去荷叶，再露7夜，晒7日，俟干透收藏听用，每月亦须晒数次，以免霉坏，大人每服3钱，小儿1钱（多则1钱半），水煎服。
制备方法：上为细末。
