序号：8068
名称：加减芩芍汤
组成：白芍3钱，黄芩2钱，黄连1钱5分，厚朴2钱，木香1钱（煨），广皮2钱。
出处：《温病条辨》卷二。
主治：滞下已成，腹胀痛者。
加减：肛坠者，加槟榔2钱；腹痛甚欲便，便后痛减，再痛再便者，白滞加附子1钱5分，酒炒大黄3钱；红滞加肉桂1钱5分，酒炒大黄3钱；红积加归尾1钱5分，红花1钱，桃仁2钱；舌浊脉实，有食积者，加查肉1钱5分，神曲2钱，枳壳1钱5分；湿重者，目黄、舌白、不渴，加茵陈3钱，白通草1钱，滑石1钱。
功效：None
用法用量：水8杯，煮取3杯，分3次温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌油腻生冷。
附注：None
