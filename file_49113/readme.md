序号：49128
名称：胜金丸
组成：泽兰4两，当归6分，芍药6分，芜荑6分，甘草6分，芎藭6分，干姜3分半，桂心3分半，石膏3分，桔梗3分，细辛3分，茱萸3分，柏子仁3分，防风3分，厚朴3分，乌头3分，白薇3分，枳壳3分，南椒3分，金钗石斛3分，石颔3分，蒲黄3分，茯苓3分，白术1分，白芷1分，人参1分，藁本1分，青木香1分。
出处：《周（廷页）传授济急方论》。
主治：产后血晕，血气及滞血不散，便成癥瘕兼泻，面色黄肿，呕逆恶心，头痛目眩，口吐清水，四肢萎弱，五脏虚怯，常日睡多，吃食减少，渐觉羸瘦，年久变为痨疾。
加减：None
功效：None
用法用量：每有所患，热酒研1丸。若死胎不下，胎衣在腹，以炒盐酒研服，未退再服。
制备方法：上为末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：忌腥腻、热面、豉汁、生葱、冷水、果子等。
附注：方中石颔，《普济方》作“石燕”。
