序号：82517
名称：磨翳水
组成：生炉甘石1两，硼砂8钱，胆矾2钱，薄荷叶3钱，蝉蜕3钱（带全足，去翅土）。
出处：《衷中参西》上册。
主治：目翳遮睛。
加减：若目翳甚厚，已成肉螺者，加真藏脑砂2分，另研调和药水中。
功效：None
用法用量：用时将瓶中水药调匀，点眼上，日5-6次。
制备方法：上药五味，将前三味药臼捣细，再将薄荷、蝉蜕煎水1大钟，用其水和所捣药末，入药钵内研至极细，将浮水者随水飞出，连水别贮一器，待片时，将浮头清水，仍入钵中，和所余药渣研细，仍随水飞出，如此不计次数，以飞净为度。若飞过者还不甚细，可再研再飞，以极细为度。制好连水贮瓶中，勿令透气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
