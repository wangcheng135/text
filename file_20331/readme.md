序号：20338
名称：肉苁蓉丸
组成：肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），菖蒲1两，磁石1两（烧令赤，醋淬7遍，捣碎研，水飞过），附子1两（炮裂，去皮脐），巴戟1两，菟丝子1两（酒浸3日，晒干，捣为末），鹿茸1两（去毛，涂酥，炙微黄），石斛1两（去根，锉），桂心1两半，桑螵蛸半两（微炒），杜仲1两（去皴皮，炙微黄，锉），牡蛎粉1两，补骨脂1两（微炒），熟干地黄1两半。
出处：《圣惠》卷三十六。
主治：劳聋，肾气虚损，腰脚无力，面黑体瘦，小便滑数。
加减：None
功效：None
用法用量：每服30丸，空心以温酒送下，晚食前再服。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
