序号：60669
名称：通神丸
组成：兔粪7粒，胎发（14天小儿胎发），人手指甲49片，人脚指甲49片，天灵盖1枚（圆者），穿山甲49片，硇砂半两（通明者），紫河车1具，上入瓦器内，盐泥固济，穿一地窍，方圆1尺，将药瓶置于地窍内，用槐木火固定，烧1日，取出细研，再入后药：雄黄（研）1两，鬼臼（炒，捣末）1两，鬼箭羽（炒，捣末）1两，麝香（研）1两，丹砂（研）1两。
出处：《普济方》卷二三七引《圣惠》。
主治：传尸劳，情思不悦，心多惊悸，盗汗羸瘦。
加减：None
功效：None
用法用量：每服10丸，渐加至20丸，早、晚食前米饮送下，日进2服。
制备方法：上将后5味末，与前烧药相合，研匀，炼蜜为丸，如梧桐子大，别用丹砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
