序号：41589
名称：金粟汤
组成：陈皮（去白）1两1分，车前子（炒）4两，干姜（炮）2两，甘草（炒）半斤，罂粟壳（去瓤蒂，蜜炒）半斤。
出处：《局方》卷六（宝庆新增方）。
主治：丈夫、妇人、室女、小儿一切下痢，无问新久，冷热不调，日夜无度，脐腹绞痛即痢，肢体困倦，小便闭涩，不思饮食，渐加羸瘦；伤生冷，脾胃怯弱，饮食不消，腹胀雷鸣，泄泻不止，连月不愈。
加减：None
功效：None
用法用量：每服2大钱，水1盏，加大枣1个，生姜2片，煎至7分，空心食前稍热服，或饭饮调下亦得。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻、鱼腥、鲊酱等。
附注：None
