序号：11697
名称：牛黄丸
组成：牛黄1分，龙脑1钱，麝香1钱，雄黄1分，丹砂半钱（五味同研），天麻半两，乌犀（镑）半两，干蝎（去土，炒）半两，羚羊角（镑）半两，羌活（去芦头）半两，独活（去芦头）半两，防风（去叉）半两，细辛（去苗叶）半两，白芷半两，赤茯苓半两，蝉蜕半两，麻黄（去根节）半两，牛膝（酒浸，切，焙）半两，芎藭半两，五加皮半两，白附子（炮）1分，乌蛇（酒浸，去皮骨，炙）1两。
出处：《圣济总录》卷七。
主治：一切风及瘫痪，言语謇涩。
加减：None
功效：搜风镇心。
用法用量：每服1丸，嚼破，温酒或荆芥茶送下，日3次。
制备方法：上为末，炼蜜为丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
