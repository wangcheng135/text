序号：26798
名称：延寿酒药仙方
组成：当归（去芦）5钱（净），人参（去芦）5钱（净），白茯苓（去皮）5钱（净），草乌（去皮）5钱（净），乌药5钱（净），杏仁（去皮尖）5钱（净），何首乌（去皮）5钱（净），川椒（去目）5钱（净），川乌（去皮尖）5钱（净），五加皮5钱（净），肉苁蓉（去皮尖）5钱（净），枸杞子5钱（净），砂仁5钱（净），木香3钱，牛膝3钱（去芦），枳壳3钱（去瓤），干姜3钱（火炮），虎骨3钱（酥炙黄色），川芎3钱，香附子3钱（炒去毛），香白芷3钱，厚朴3钱（姜汁浸），陈皮3钱（去白），白术3钱（炒），独活3钱，羌活3钱，麻黄3钱（去节），官桂3钱（去皮），白芍药3钱，半夏3钱（姜汁浸），生地3钱，熟地3钱，天麦门冬3钱（去心），五味子3钱，防风3钱，细辛3钱（拣净，酥酒洗，去芦），沉香3钱，苍术3钱（米泔浸，去皮），小茴香（盐炒黄）3钱，破故纸1两（酒浸，微炒），核桃仁1两（汤浸去皮），甘草（火炙，净）1两，红枣肉半斤，酥油半斤，白沙糖1斤。
出处：《遵生八笺》卷十七。
主治：男妇远年近日，诸虚百损，五劳七伤，左瘫右痪，偏正头风，口眼歪斜，半身不遂，语言謇涩，筋脉拘挛，手足麻木，浑身疥疮，肠风痔漏，紫白癜风，寒湿脚气，膀胱疝气，十膈五噎，身体羸瘦；腰腿疼痛，四肢无力，皮肤生疮，耳聋眼昏，下部虚冷，诸般淋沥；妇人经脉不调，脐腹疼痛，胁背臌胀，黄瘦面肌，口苦舌干，呕逆恶心，饮食无味，四肢倦怠，神鬼惊悸，夜多盗汗，时发潮热，月事或多或少，或前或后，心中闷塞不通，结成瘕块，时作刺痛，或子宫积冷，气毒虚败，赤白带下，渐成虚瘵。
加减：None
功效：补脾，养丹田，和气血，壮筋骨，益精髓，身体轻健，明眼目，安五脏，定魂魄，润肌肤，返老还童，延年益寿。
用法用量：每日用酒1小钟，病在上，食后服；病在下，空心服。饮酒毕后，将药滓晒干，研为细末。用好花烧酒打糊为丸，如梧桐子大。每服30丸，空心好酒送下。
制备方法：上药用细绢袋盛之，用烧酒一大坛，浸药三日，放在大锅内，用汤浸坛，煮两个时辰，取起掘一坑，埋三日，出火毒取出。
临床应用：None
各家论述：None
用药禁忌：None
