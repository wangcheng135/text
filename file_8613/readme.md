序号：8613
名称：玄精丹
组成：血余（自己发及父子一本者，及少壮男女发，拣去黄白色者，用灰汤洗2-3次，再以大皂角4两捶碎，煮水洗净，务期无油气为佳，将发扯碎晒干，每净发1斤用川椒4两，拣去梗核，于大锅内发1层，椒1层和匀，以中锅盖之，盐泥固济，勿令泄气，桑柴慢火煅3炷香，退火待冷取出，于无风处研为细末）4两，何首乌（用黑豆9蒸9晒，拣去豆，取净末）1斤，黑脂麻（9蒸9晒，取净末）8两，破故纸（炒，取净末）4两，生地黄（怀庆沉水者，酒浸杵膏）8两，熟地黄（酒浸杵膏）8两，桑椹（取净汁熬膏）4两，女贞实4两，旱莲草（取净汁熬膏）4两，胡桃肉（研膏）2两，胶枣肉（研膏）2两，槐角子（入牛胆内百日）4两。
出处：《准绳·类方》卷八。
主治：肝肾不足，精血枯涸，凡虚损在真阴水火者。
加减：None
功效：乌须发。
用法用量：每服60丸，空心用何首乌酿酒每温2-3杯送下，日3次。
制备方法：上以药末入诸膏和匀，加炼蜜1斤成剂，入石臼内春千余下，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：元精丹（《医部全录》郑一六四）。
