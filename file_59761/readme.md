序号：59776
名称：调经汤
组成：当归（洗焙）1两，半夏（汤洗7次）1两，甘草（炙）1两，麦门冬（去心）1两，五加皮1两，熟干地黄（洗焙）1两，川芎1两，吴茱萸（汤洗7次）1两，肉桂（去粗皮）1两，牡丹皮1两，赤芍药1两，乌药1两，人参（去芦头）1两，红花1两，没药半两（另研）。
出处：《杨氏家藏方》卷十五。
主治：冲任脉虚，风寒客搏，气结凝滞，每经候将行，脐腹先作撮痛，或小腹急胀，攻注腰脚疼重。
加减：None
功效：None
用法用量：上（口父）咀。每服5钱，水1盏半，加生姜5片，煎至1盏，去滓，食前温服，经欲行时，预前5日及经断后5日，并宜服之。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
