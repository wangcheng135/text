序号：62643
名称：清中汤
组成：黄连1钱半，青黛3钱，花粉3钱，池菊3钱（去蒂），会白1钱半，甘草1钱半，元参1钱半，薄荷1钱半（泡），钩藤5钱（迟入）。
出处：《医略六书》卷二十八。
主治：孕妇心膈热痛，脉数者。
加减：None
功效：None
用法用量：水煎，去渣温服。
制备方法：None
临床应用：None
各家论述：胎热气逆，肝火炽盛，故上侵心胞，而疼痛至急，谓之子痛。黄连清心包伏火，青黛清肝胆郁火，花粉清胃火以润燥，元参清肾火以益阴；池菊清郁热兼益金水之源，薄荷散郁火兼舒结伏之热，新会白和中启胃，生甘草泻火暖中，纯钩藤平上逆之邪以舒经脉也。水煎，温服，使热化气平，则心包宁静，而胎得所养，何心痛急暴之不瘳乎。
用药禁忌：None
附注：None
