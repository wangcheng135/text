序号：30559
名称：走马催生丹
组成：光明辰砂1两（研），麝香（研）1钱，桃嫩苗7茎，柳嫩苗7茎（如无取皮，须是向东方妙，取向东方者），雄蛇蜕1条（首尾全者，树上蜕者，是雄，不然只取中间5寸，烧烟欲尽，急速取出，用瓷器盆盖于地上，周围用湿土围之，良久取出，为末）。
出处：《普济方》卷三五六。
主治：难产危急，横生逆产，子死腹中，胎衣不下。
加减：None
功效：None
用法用量：难产者，每服1丸，淡醋汤送下。不久产下，便于儿手内取丸药，男左女右，手中把出。
制备方法：辰砂、麝香研极细，以桃柳嫩苗与上2味同研细，却入蛇蜕灰，又研匀，糯米饭为丸，如胡艽大，朱砂为衣，晒干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
