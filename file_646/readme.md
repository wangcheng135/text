序号：646
名称：五瘿丸
组成：菖蒲2两，海蛤1两，白蔹1两，续断1两，海藻1两，松萝1两，桂心1两，蜀椒1两，半夏1两，倒挂草1两，神曲3两，羊靥百枚。
出处：《千金》卷二十四。
主治：五瘿：石瘿、气瘿、劳瘿、土瘿、忧瘿。
加减：None
功效：None
用法用量：每日服3丸。
制备方法：上药治下筛，以羊、牛髓脂为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：方中菖蒲利窍，海蛤消坚，白蔹散肿，续断营筋，海藻软坚，松萝清风，桂心透经，蜀椒开痹，半夏涤垢，神曲消滞树中，倒挂草绝经络病根，羊靥通喉管结气。丸用羊髓，以滋肺、肾伏藏之风，五瘿之治备矣。
用药禁忌：忌羊肉、生葱。
附注：None
