序号：36265
名称：取积妙应丸
组成：槟榔1斤，大黄1斤，牵牛末半斤，杜仲半斤，芜荑仁半斤（另研），雷丸半斤，鹤虱4两，锡灰4两，阿魏2两（另酒化开揉）。
出处：《普济方》卷一六八。
主治：男子、女人、小儿诸般积滞气多，因茶、酒、生果、肉、面所伤，又为悲忧喜怒之气，郁结心怀，积成癖脾癥瘕，大如杯碗不消；诸种虫积；及治黄疸水蛊，遍身浮肿，翻胃吐食，九种心痛，一切风症；妇人血瘕淋滞，经脉不通，腹内如怀胎孕，及成鬼胎；小儿脾疳积滞。
加减：None
功效：None
用法用量：每服4钱，五更用葱白7根，熬汤送下，仍以枣儿3个，食之压药。次早天明，脏腑1行后，取下恶积之物。如有积者，尽去数行。如只见稀水无积物，即用温薄米粥补住。
制备方法：上为细末，用皂角（去皮弦子，净）一斤半，热水泡浸搓，橘水汁滤去滓，与药末、皂角水再入糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中杜仲，《奇效良方》作“贯众”。
