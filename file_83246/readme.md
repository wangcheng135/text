序号：83264
名称：乌鸡丸
组成：香附米1斤（酒、酯、童便、米泔浸，四制各4两），白茯苓（去皮）4两，当归2两，吴茱萸5钱（水浸，去闭口者），川芎1两，白芍1两，黄耆（蜜炙）5钱，黄柏1两，大附子1个（看虚实），怀生地黄（酒拌炒，蒸黑）4两，陈皮（去白）两半，山药1两，白术（去芦，陈土炒）1两，莲肉（去心皮）2两，酸枣仁1两，知母1两，小茴香2两，阿胶（蛤粉炒）5钱。
出处：《寿世保元》卷七。
主治：妇人无子。
加减：None
功效：None
用法用量：每服2钱，临经之日，每日3服。半月见效。
制备方法：用雄乌骨鸡1只吊死，去毛、屎，治净蒸熟，连骨扔烂，同前药为末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
