序号：31541
名称：沉香丸
组成：沉香半两，阿魏半两（以少面和溶，作饼子，炙令黄），木香1分，桃仁半两（汤浸，去皮尖双仁，麸炒微黄），槟榔半两，吴茱萸1分（汤浸7遍，焙干，微炒），茴香子半两，青橘皮1分（汤浸，去白瓤，焙），硼砂3两（不夹石者，细研，以汤1盏化，澄去滓取清，纳银器中煎成霜，研入），蛜螂1两（生用）。
出处：《圣惠》卷四十八。
主治：奔豚气，小腹积聚疼痛，或时上攻，心胸壅闷。
加减：None
功效：None
用法用量：每服20丸，食前以姜盐汤送下。
制备方法：上为细末，加硼砂令匀，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
