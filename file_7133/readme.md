序号：7133
名称：四圣丸
组成：紫河车胶12两，龟甲胶8两，糜角胶4两（上3味名三益膏），人参12两（为细末，人乳拌蒸，或晒、烘干，拌重至24两为度。若肺间有火，咳甚痰多者，不宜用参，可以茯苓代之，制亦如上法）。
出处：《何氏虚劳心传》。
主治：虚损。
加减：若己成虚劳内热骨蒸等症者，更宜参入壮水滋阴除蒸之品，如二地、二冬、青蒿、鳖甲、骨皮、女贞子之类。
功效：峻补精血。
用法用量：每服4-5钱，空心白汤送下。或竟三益膏中加茯苓，乳粉拌匀，麦冬汤化服亦可。
制备方法：浓麦冬汤入三益膏，隔汤煮烊，捣匀为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
