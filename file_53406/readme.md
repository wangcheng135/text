序号：53421
名称：凉血地黄汤
组成：犀角（乳汁磨，临服入药内；或挫末煎）4分，生地黄（酒洗）1钱，牡丹皮2钱，赤芍7分，黄连（酒炒）1钱，黄芩（酒炒）1钱，黄柏（酒炒）5分，知母1钱，玄参1钱，天门冬（去心）1钱，扁柏叶3钱，茅根2钱。
出处：《寿世保元》卷四。
主治：虚火妄动，血热妄行，吐血、衄血、溺血、便血。
加减：吐血成块，加大黄1钱，桃仁10个（去皮尖，研如泥）；衄血，加栀子、沙参、玄参；溺血，加木瓜、牛膝、条芩、荆穗、地榆，倍知、柏；便血，加黄连、槐花、地榆、荆穗、乌梅；善酒者，加葛根、天花粉。
功效：None
用法用量：上锉。水煎，入10汁饮同服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
