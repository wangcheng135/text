序号：34759
名称：麦门冬汤
组成：麦门冬（去心，焙）3分，赤茯苓（去黑皮）3分，生干地黄（焙）3分，石膏（碎）3分，升麻3分，人参3分，知母（焙）3分，芎3分，山栀子仁3分，小麦半升，黄耆（炙，锉）1两，甘草（炙）1两，枳实（麸炒）1两，芍药1两，黄芩（去黑心）1两半，前胡（去芦头）1两半。
出处：《圣济总录》卷一八三。
主治：乳石发动，痈疽发背热渴。
加减：None
功效：None
用法用量：每服5钱匕，水2盏，加生姜5片，大枣2枚（擘破），竹叶10片，同煎至1盏，去滓温服，不拘时候。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
