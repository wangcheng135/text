序号：22999
名称：达生散
组成：大苏梗1钱，当归（酒洗）1钱，白芍（酒洗）1钱，广皮1钱，川芎7分，炙甘草5分，大腹皮8分（黑豆汁洗净，晒干）（1方加车前子、黄芩）。
出处：《郑氏家传女科万金方》卷三。
主治：None
加减：虚者，加人参8分；小便闭结，头目眩晕，恶心呕吐，加黄连1钱（姜汁炒），山栀1钱（姜汁炒黑）；有痰，加天花粉8分；胸膈痞闷，加焦神曲1钱；8-9月后，可加白芷1钱半，川贝母1钱（去心），麸炒枳壳8分。
功效：安胎利产。
用法用量：水1钟，加生姜3片，煎至8分，食远服。妊妇每月服3帖，至5个月服5贴，6个月服6贴，逐月递加，至10月不必服。胎前平日，如未服过此药，临产服1贴。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
