序号：25228
名称：妇宝宁坤丸
组成：吉林人参2钱，大熟地5钱，制香附5钱，紫苏叶2钱5分，大生地5钱，驴皮胶2钱5分，全当归5钱，广橘红5钱，川牛膝2钱，于术5钱，沉香1钱，川芎5钱，台乌药5钱，西砂仁1钱5分，炒黄芩5钱，西琥珀2钱5分，白茯苓5钱，广木香2钱5分，炙甘草1钱5分，东白芍5钱，益母草3两。
出处：《全国中药成药处方集》（杭州方）。
主治：妇人气血两亏，月经不调，崩漏带下，诸虚百损，久不受孕，一切胎前产后诸病。
加减：None
功效：调经种子，养血安胎。
用法用量：每服1丸，开水化服。
制备方法：各取净粉，用柏子仁1两，煎汤去滓，和炼白蜜为丸，每重3钱，蜡壳封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
