序号：52417
名称：香术汤
组成：苍术（米泔浸1宿，切，焙）2两，陈橘皮（汤浸去白，焙）1两，防风1两，麻黄（去根节，汤煮，掠去沫，焙干）1两，当归（切，焙）1两，桂（去粗皮）1两，甘草（炙）1两，白茯苓（去黑皮）半两，吴茱萸（洗，焙干炒）半两，人参半两，厚朴（去粗皮，生姜汁炙）半两，羌活（去芦头）半两。
出处：《圣济总录》卷二十一。
主治：伤寒2-3日，头痛肌热，烦躁不解，四肢倦痛。
加减：None
功效：发汗。
用法用量：每服3钱匕，水1盏，加葱白1寸，生姜2片，同煎至6分，去滓温服，不拘时候。
制备方法：上除桂外锉碎，慢火炒黄，入桂，同为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
