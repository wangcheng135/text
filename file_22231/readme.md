序号：22239
名称：防病牛黄汤
组成：牛黄1分，芍药1分，杏仁1分（炒，去皮尖），蜣蜋1分，蜂房1分，黄芩1分，人参1分，葛根1分，甘草1分（炙），蚱蝉1分（炙），芒消1分，川芎1分，桂心1分，大黄3分，当归3分，石膏4分（碎）（一方有生姜3分，无芒消）。
出处：《普济方》卷三七八。
主治：少小心惊，又治痫发，众医不复治者。
加减：None
功效：None
用法用量：取豚五脏及卵，以水1斗，煮脏卵得3升，去滓澄清，纳诸药，煮取1升3合，去滓入芒消，烊尽，每服1合，日3夜1，临卧，末牛黄纳汤中。
制备方法：上细切。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
