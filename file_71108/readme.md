序号：71124
名称：照水丹
组成：蛇蜕皮1条（全者，净，烧灰，细研），水银1钱，黄丹1钱（2味同研砂子），乌鸡子1个（去膜，细研如粉），硇砂1钱（细研），乌贼鱼骨2钱（研如粉）。
出处：《永乐大典》一一四一三引《经验普济加减方》。
主治：远年日近，翳膜瘀肉，昏暗痒痛。
加减：None
功效：None
用法用量：每用2饼，临卧放贴于眼大眦，或眼翳膜上，明旦未洗面时，用水1大碗，取药放入水中照，其药自落于水中，却看下翳膜多少，未下，再贴必验。顽翳膜厚者，不过半月；小可者，3-5日效。
制备方法：取黄蜡半两熔汁入前药末，同熬成膏，以柳枝搅匀，为丸，如麻子大，或指甲上按作饼子。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
