序号：7862
名称：加减羚羊角散
组成：羚羊角、天麻、钩藤、龙胆草、桑寄生、川牛膝、鸡血藤、僵蚕、蜈蚣（焙研）、全蝎（焙研）。
出处：《杂病证治新义》。
主治：类中风，卒中回苏后，血压未降，口眼斜，舌瘖失语，半身不遂，脉象弦长有力者。
加减：None
功效：柔肝祛风，养血通络。
用法用量：水煎，后入羚羊水磨液、蜈蚣末、全蝎末调服。
制备方法：None
临床应用：None
各家论述：本方系《本事方》羚羊散及活络丹中悟出。以羚羊角、天麻、钩藤、龙胆草等柔肝泻火为主；僵蚕、蜈蚣、全蝎祛风为辅；桑寄生、鸡血藤、牛膝养血活血通络利痹为佐，共奏柔肝祛风养血通络之功。用于高血压脑出血卒中后遗之偏瘫、失语症有良效。
用药禁忌：None
附注：None
