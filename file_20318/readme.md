序号：20325
名称：肉苁蓉丸
组成：肉苁蓉2两（酒浸1宿，刮去皱皮，炙今干），附子1两（炮裂，去皮脐），巴戟1两，茴香子1两，石斛1两（去根，锉），补骨脂1两，桂心1两，川椒3分（去目及闭口者，微炒去汗），麋茸1两（去毛，涂酥负微黄），木香3分，牛膝1两（去苗），五味子1两，泽泻1两，槟榔1两，丁香3分，黄耆3分（锉），熟干地黄1两，人参3分（去芦头），诃黎勒皮3分，山茱萸3分，白术3分，干姜3分（炮裂，锉），朱砂1两（细研，水飞过），麝香半两（细研）。
出处：《圣惠》卷九十八。
主治：None
加减：None
功效：暖水胜，壮筋骨，益精气，利腰脚，聪耳明目，强志倍力，悦泽颜色，充益肌肤。
用法用量：每服30丸，空心以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
