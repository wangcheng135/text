序号：66452
名称：小七香丸
组成：甘松（炒）80两，益智仁（炒）60两，香附子（炒，去毛）120两，丁香皮120两，甘草（炒）120两，蓬莪术（煨，乘热碎）20两，缩砂仁20两。
出处：《局方》卷三（绍兴续添方）。
主治：中酒吐酒，呕逆咽酸，气膈食噎，饮食不下，冷涎翻胃，腹胀脾疼，远年茶酒食积，眼睑俱黄，赤白痢疾，脾毒泄泻。妇人脾血气，小儿疳气。郁怒忧思，或因闪挫扑，一切气滞腰痛。
加减：None
功效：温中快膈，化积和气。化积气，消宿食，止泻痢。
用法用量：每服20丸，温酒、姜汤、熟水任下：或气胀满，磨乌药水煎汤下；或酒食过度，头眩恶心，胸膈满闷，先嚼20丸，后吞20丸，生姜、紫苏汤送下。
制备方法：上为末，水浸蒸饼为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：七香丸（《百一》卷二引冯仲柔传徐家方）。
