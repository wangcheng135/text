序号：45473
名称：千里光明汤
组成：青木香5钱，黄连5钱，黄柏5钱，黄耆5钱，防风5钱，荆介5钱，苦参5钱，苍耳子5钱，蛇床子5钱，羌活5钱，升麻5钱，麻黄5钱，甘草5钱，鸡肠草（焙）、冬青叶（焙）。
出处：《寿世保元》卷九。
主治：杨梅疮。
加减：None
功效：None
用法用量：上作1剂，用布包，水煮。于无风处，服此煎药，即以此汤浴洗，凉了又加热，药汤煮热着实洗，微汗拭干，10日后不必频洗，其药滓并入，煎药滓再洗。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：①鸡肠草，一名“千里光明草”，又名“千里明”，俗名“藤枯买”。②方中鸡肠草、冬青叶用量原缺。
