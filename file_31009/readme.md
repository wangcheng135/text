序号：31018
名称：沉香荜澄茄散
组成：荜澄茄1两，沉香1两，葫芦巴（微炒）1两，破故纸（微炒）1两，官桂（去皮）1两，舶上茴香（微炒）1两，川苦楝子（炮，捶破，去核用肉）1两，木香1两，紫巴戟（穿心者）1两，黑附子（炮制，去皮脐）4两，桃仁（面炒，去皮尖）2两，川乌头半两（炮，去皮脐）。
出处：《博济》卷二。
主治：下经不足，内挟积冷，脐腹弦急，痛引腰背，面色萎黄，手足厥冷，胁肋虚满，精神困倦，脏腑自利，小便滑数及盲肠小肠一切气痛。
加减：None
功效：None
用法用量：每服2钱，水1大盏，加盐同煎至8分，温服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：荜澄茄散（《证治要决类方》卷三）。本方改为丸剂，名“沉香荜澄茄丸”，（见《御药院方》）。
