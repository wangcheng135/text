序号：28155
名称：苁蓉牛膝丸
组成：肉苁蓉（好酒焙干，净洗）2两，川牛膝（去苗，洗，焙干）2两，天麻（明白者，洗，焙干）2两，木瓜（干者，各锉碎，用酒3升，入瓶内密缠，春5日，取出急用沸汤滤过，焙，为末）2两，枸杞子（拣，净洗）2两，黄耆（洗，涂蜜炙）2两，真虎骨（酒浸，炙黄）2两，青盐（别研）2两。
出处：方出《传信适用方》卷二，名见《普济方》卷二四○。
主治：脚气。
加减：None
功效：None
用法用量：每服30-50丸，空心、食前温酒白汤送下。
制备方法：上药都拌匀，将前项浸药酒，入面糊为丸，如梧桐子大，焙干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
