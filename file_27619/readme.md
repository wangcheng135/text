序号：27627
名称：八物茜根汤
组成：茜根3两，升麻3两，犀角3两，桔梗1两，黄柏1两，黄芩1两，地榆4两，白蘘荷4两。
出处：方出《千金》卷二十四，名见《千金》卷十五。
主治：中蛊毒，下血状如鸡肝，腹中搅痛难忍者。
加减：None
功效：None
用法用量：上(口父)咀。以水9升，煮取2升半，分3服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：蛊毒下血如鸡肝，急需清热解毒，方中茜根，犀角专散毒血，黄柏、黄芩专散热结，升麻、桔梗升散于上，地榆、蘘荷解散于下。不特为蛊毒下血之专方，并可治热毒血痢之要药。
用药禁忌：None
附注：茜根散（《圣惠》卷五十六）、茜根饮（《圣济总录》卷七十七）、茜根汤_x000D_
（《圣济总录》卷一四七）。
