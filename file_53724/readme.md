序号：53739
名称：宽中散
组成：枳壳、桔梗、甘草、茯苓、半夏、芍药。
出处：《便览》卷二。
主治：脾胃气滞，膈塞腹满，胸胁不利，胃脘疼痛，吞酸嘈杂。
加减：膈塞腹满，加紫苏叶、青皮、大腹皮、厚朴、香附；气盛少食，加麦芽、砂仁、山楂；气结胸胁不利，或咳嗽，加炒瓜蒌仁、桑白皮；郁气胸膈痛，加香附、抚芎；冷气胃脘作痛，加青皮、陈皮、元胡、木香、草豆蔻；气郁胸中，心下满闷，加川连、神曲、贝母；气盛久郁，膈间上下游走，吞酸嘈杂刺心，加细辛、栀子、黄连、枳实，气病感寒作喘，加苏子、麻黄、杏仁、荆芥穗；病后气肿，加大腹皮、五加皮、苏子。
功效：None
用法用量：上药用水1钟半，加生姜3片，煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
