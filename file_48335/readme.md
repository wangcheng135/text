序号：48350
名称：神效风气膏药
组成：当归3两，川芎3两，芍药3两，防风3两，羌活2两，独活2两，红花2两，连翘2两，五灵脂1两5钱，川乌1两5钱，蝉蜕5钱，官桂4两，生地黄1两，熟地黄1两，乳香1两，没药1两，阿魏1两，荆芥穗2两。
出处：《摄生众妙方》卷三。
主治：诸风。
加减：None
功效：None
用法用量：None
制备方法：上为末，用麻油1斤文武火煎成膏，外用白瓷器收起；又用米醋、姜汁、葱汁各2大碗共一处，文武火煎成膏，又用瓷器盛起；用好松香1斤半，文武火煎化，方下醋、姜、葱汁，合成用槐棍打匀，又文武火煎一次，量加麻油药汁，后加乳香、没药、阿魏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
