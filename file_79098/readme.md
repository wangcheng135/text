序号：79114
名称：磨积三棱丸
组成：槟榔5钱，砂仁半两，阿魏（姜治）5钱，干漆（泥固，煅去烟）5钱，使君子1钱，芦荟2钱，神曲（炒）2钱，青皮（去白）2钱，木香1钱，陈皮（去白）2两，广术（炮）2两，大黄（酒浸2次）2两，麦芽（炒）2两，皂角（炙，去皮）3钱，官桂3钱，硇砂3钱，白牵牛（头末）4钱，黑牵牛（头末）4钱，京三棱（炮）、石三棱（炒，去脐）、鸡爪三棱（炒）、陈仓米半升（炒）。
出处：《医方类聚》卷一一一引《修月鲁般经》。
主治：一切沉积，酒食过伤，好食生破冷物停滞，胸膈痞满，积聚不散，远年近日，遂成酒积、食积、气积、血积，渐成结块，心腹胁肋膨胀刺痛，呕吐酸水，饮食无味减少，面黄肌瘦。又治小儿疳积腹痛，形体枯瘦。
加减：None
功效：顺气磨积，宽中消滞。
用法用量：每服大人40-50丸，小儿减丸数，食前淡姜汤送下。
制备方法：上为细末，糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
