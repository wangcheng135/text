序号：28107
名称：鸡膍胵汤
组成：鸡膍胵20具，鸡肠3具（洗），干地黄2两，当归2两，甘草2两，麻黄4两，厚朴3两，人参3两，生姜5两，大枣20枚。
出处：《千金》卷三。
主治：产后小便数。
加减：None
功效：None
用法用量：上(口父)咀，以水1斗，煮膍胵及鸡肠、大枣，取7升，去滓，纳诸药，煎取3升半，分3次服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：热在小肠，不能司膀胱气化而小便频数，小肠尽处傍通膀胱，膀胱渗洄，是必小肠有瑕，故取鸡之膍胵及肠专利小肠；厚朴、姜、枣专泄滞气；人参，甘草、地黄、当归兼滋气血；蒲黄匡佐膍胵，专清膀胱血热也。世本作麻黄，热既犯肠，断无复用麻黄之理。
用药禁忌：None
附注：方中麻黄，《千金方衍义》作“蒲黄”。
