序号：48372
名称：神效回生膏
组成：槐20条，柳20条，桃20条，榆20条，桑20条，枸杞（树条嫩者）20条（每条长2寸），上将6件树条，剥取嫩皮，用清油3升，文武火于大砂锅内煎，令嫩皮津液尽为度，将油滤过。白芷1两半，白及1两半，白蔹1两半，当归1两半，大黄1两半，黄柏1两半，赤芍药1两半，杏仁1两半，蓖麻子1两半，将上药锉碎，再下于前油内浸透，又用慢火煎炒，去药滓，再用油滤过。用黄丹12两，分作3次下，于油内熬令黑色，将筋觜蘸药油滴水内不散为度。血竭5钱，雄黄5钱，乳香5钱，没药5钱，轻粉3钱。
出处：《医方类聚》卷一七六引《瑞竹堂方》。
主治：痈疽疖毒，远近臁疮，打扑跌折伤损，暗毒发背，刀斧所伤，箭头在肉，蛇犬所伤，并皆治之。
加减：None
功效：None
用法用量：任意摊贴患处，2日外自觉病退。
制备方法：上为极细末，放油微温，下前药于油内，以瓦罐盛之，盖口，埋土内3日，去火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
