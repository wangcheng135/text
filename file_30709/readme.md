序号：30717
名称：鸡胵茅根汤
组成：生鸡内金5钱（去净瓦石糟粕，轧细），生于术（分量用时斟酌）、鲜茅根2两（锉细）。
出处：《衷中参西》上册。
主治：水臌气臌并病，兼单腹胀，及单水臌胀，单气臌胀。
加减：None
功效：None
用法用量：先将茅根煎汤数茶钟（不可过煎，1-2沸后慢火温至茅根沉水底，汤即成），先用1钟半，加生姜5片，煎鸡内金末，至半钟时，再添茅根汤1钟，7-8沸后，澄取清汤（不拘1钟或1钟多）服之。所余之滓，仍用茅根汤煎服。日进1剂，早、晚各服药1次。初服小便即多，数日后大便亦多。若至日下2-3次，宜减鸡内金1钱，加生于术1钱。又数日，胀见消，大便仍勤，可减鸡内金1钱，加于术1钱。又数日，胀消强半，大便仍勤，可再减鸡内金1钱，加于术1钱。如此精心随病机加减，仰其补破之力，适与病体相宜，自能全愈。若无鲜茅根，可用药房中干茅根1两代之。无鲜茅根即可不用生姜。所煎茅根汤，宜当日用尽，煎药后若有余剩，可当茶温饮之。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
