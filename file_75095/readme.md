序号：75111
名称：蒴藋膏
组成：蒴藋根2两，白蒺藜1两，附子1两（去皮脐），独活1两，犀角屑1两，蔷薇根2两，白芷1两，防风1两，苦参1两，川升麻1两，漏芦1两，汉防己1两，川椒1两，木香1两，蛇衔草1两，茺蔚1两，枳壳1两，莽草1两。
出处：《圣惠》卷二十四。
主治：风瘙瘾疹，皮肤中苦痒，搔之血出。
加减：None
功效：None
用法用量：每取涂摩所患处。累用即愈。
制备方法：上并生用，细锉，以头醋浸，令淹1宿，明旦用铜石银锅器中盛，于慢火上用腊月炼成猪脂2斤半与药同煎，令白芷赤色膏成，滤去滓，盛于瓷盒中。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
