序号：40037
名称：肺寒汤
组成：款冬花2两，紫菀（去土）2两，甘草（炙）2两，桂（去粗皮）2两，麻黄（去节）2两，干姜（炮）2两，五味子2两，杏仁（汤浸，去皮尖，炒）2两，半夏（汤煮软，焙干）2两，细辛（去苗叶）1两。
出处：《圣济总录》卷六十五。
主治：肺胃虚寒，咳嗽痰盛，呀呷有声，呕吐停饮，咽喉干痛，上气喘满，面目虚浮，自汗恶风，语声嘶破，背寒中冷，心下悸动，哕逆恶心，全不入食。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，生姜5片，大枣2枚（擘破），同煎至7分，去滓温服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
