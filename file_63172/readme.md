序号：63188
名称：大黑神丸
组成：曾青1两，硫黄1两，水银1两（与硫黄结为砂子），朱砂1两，雄黄1两，白石英1两，紫石英1两，白矾1两，黄丹1两半，金箔50片，银箔50片，消石1两，定粉1两，太阴玄精1两，上为细末，入于固济了瓶子中，其瓶子上开1小窍子，用出阴气。初以小火养2日，候阴气出尽，以盐泥塞窍，便以大火，渐令通赤，候冷，以湿沙土培瓶子1复时，出火毒毕，取出药，更研如粉。白龙脑1两，麝香1两，牛黄1两半，腻粉1两，天竺黄1两，真珠未半两，琥珀半两，以上都细研，与诸药临时合和。鹿胎1两半，乌蛇2两半（去皮骨），狸骨1两，虎胫骨1两半，甘草1两，败龟1两，以上并涂酥炙，捣罗为末。天南星1两（炮裂），白附子1两（炮裂），天麻1两，麻黄1两半（去根节），干蝎1两（微炒），蝉壳1两，桂心1两，木香1两，槟榔1两，独活1两，细辛1两，白术1两，附子1两（炮裂，去皮脐），白僵蚕1两（微炒），犀角屑1两半，羚羊角屑1两半。
出处：《圣惠》卷二十五。
主治：一切风。
加减：None
功效：None
用法用量：每服1丸，以豆淋酒研下。
制备方法：上为末，同研令匀，炼蜜为丸，如酸枣大。
临床应用：None
各家论述：None
用药禁忌：忌一切毒、滑、鱼、肉、动风物。
附注：None
