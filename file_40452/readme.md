序号：40462
名称：虎骨丸
组成：虎胫骨3两（涂酥，炙黄色），川芎1两半，熟干地黄（洗，焙）1两半，防风（去芦头）1两半，羌活（去芦头）1两半，当归（洗，焙）1两半，附子（炮，去皮脐）1两半，萆薢1两半，金毛狗脊（去毛）1两半，川乌头（炮，去皮脐尖）1两半，羚羊角屑1两半，白蒺藜（炒，去刺）1两半，天麻（去苗）1两半，白术1两半，杜仲（去粗皮，细锉，姜汁制，炒干）1两半，黄耆（蜜炙）1两半。
出处：《杨氏家藏方》卷一。
主治：瘫缓诸风，及风寒湿痹，脚膝缓弱，骨节疼痛。
加减：None
功效：None
用法用量：每服30丸，食前温酒送下。
制备方法：上细锉，如麻豆大，酒浸，春5、夏3、秋7、冬10日，取出焙干，为细末，以浸药酒，打面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
