序号：53385
名称：兼朴厚朴汤
组成：厚朴2两，芎藭2两，桂心2两，干地黄2两，芍药2两，当归2两，人参2两，黄耆3两，甘草3两，吴茱萸2升，半夏7两，生姜1斤。
出处：《千金》卷七。
主治：恶风毒气，脚弱无力，顽痹，四肢不仁，失音不能言，毒气冲心；并治清气咳嗽，逆气呕吐。
加减：None
功效：None
用法用量：上（口父）咀。以水2斗，煮猪蹄1具，取汁1斗2升，去上肥，内清酒3升，合煮取3升，分4服。每次1服，如人相去行20里，更进1服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：厚朴汤（《普济方》卷二四三）。原书治上症，经用麻黄汤、独活汤后，继用本方。
