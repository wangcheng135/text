序号：27338
名称：补益阿胶丸
组成：阿胶1两（捣碎，炒令熟燥），白芍药半两，干姜半两（炮裂，锉），卷柏半两，桂心半两，白龙骨1两，鹿茸1两（去毛，涂酥炙令黄），人参1两（去芦头），白茯苓1两，蒲黄半两，当归1两（锉碎，微炒），白术1两，厚朴1两（去粗皮，涂生姜汁炙令香熟），石斛1两（去根，锉），黄耆1两（锉），熟干地黄1两，艾叶3分（微炒），芎半两。
出处：《圣惠》卷七十。
主治：妇人风虚劳损，经血过多，脏腑虚乏，面色萎黄，四肢羸瘦，腹内时痛，不欲饮食。
加减：None
功效：None
用法用量：每服40丸，空心及晚食前以粥饮送下；温酒下亦得。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
