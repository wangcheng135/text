序号：9445
名称：风火两消汤
组成：白芍1两，炒栀子3钱，柴胡2钱，天花粉2钱，甘草1钱，车前子2钱，丹皮5钱。
出处：《辨证录》卷九。
主治：肝经内伤，感触风邪，木郁不泄，木乃生火，火郁不宣，火乃生风，动多气恼，大声骂詈，身热胸满，两胁作胀。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：此方治肝经之内火、内风也，然外来风火，未尝不可兼治，故两治之而奏功也。倘不用白芍为君，单用柴胡、栀子之类，虽风火亦能两平，肝中气血之虚未能骤补，风火散后，肝木仍燥，怒气终不能解，何如多加白芍，既能补肝，又能泻风火。
用药禁忌：None
附注：None
