序号：47597
名称：神仙水花丸
组成：消石1分，波斯青黛2分，青葙子2分，青木香2分，葵花2分，凌霄花2分，远志2分，柴胡2分，代赭2分，金牙石2分，元精2分，蜣螂2枚，槟榔1枚（生），橘皮（去瓤）2-7个，水蛭2-7个，虎睛1枚。
出处：《幼幼新书》卷二十五引《仙人水鉴》。
主治：孩子三岁内疳气，身如金色，瘦悴不下食，多不成肌肉，渐渐黑瘦，食入口即吐，时寒时热。
加减：None
功效：None
用法用量：上为细末，分为2份，1份炼蜜为丸，如麻子大，1岁以下，每服2-3丸，清水送下；1份为末，2岁以下，每服1字，米饮送下。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
