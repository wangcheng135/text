序号：7415
名称：圣验黑神丸
组成：防风（去芦，锉，焙）半两，川乌（炮）半两，好墨（锉，焙）半两，川附子（炮）半两，檀香半两，麦子（去心，焙）7钱，藿香叶（去土，焙）7钱，白茅7钱，何首乌（锉，用生姜汁同泡了甘草2钱浸2宿，焙，再浸，焙）7钱，干姜7钱，白僵蚕（油炒，去丝）7钱半，全蝎（姜汁浸1宿，焙）7钱半，天南星（炮裂熟，切片，以姜汁小半盏同泡了甘草3钱，锉，浸2宿，焙，再焙，姜汁尽为度）7钱半，白附子（炮10分裂熟，以姜汁同泡了甘草3钱，浸2宿，焙，再浸焙）7钱半，阿胶（水浸蒸，同糯米糊作丸）7钱半，细辛（去叶及土，锉、焙）8钱，白芷（锉、焙）8钱，香附子（炒，去毛，拍碎，以生姜汁同泡了甘草2钱，浸2宿，焙，余汁再浸）8钱，川芎（锉，焙）8钱，皂角灰（烧烟，煅），甘松（去土，锉，焙，川者尤妙），半夏（汤洗10次，炒沙令热，炮，切片，用生姜汁半盏泡了甘草3钱，浸2-3宿，焙，再浸再焙）半两，草豆蔻（拍碎去皮膜，生姜汁浸1-2日，焙，再浸焙）6钱，白术（锉，焙）1两2钱，缩砂仁（去膜皮，轻焙）1两，麝香（别细研，同药末滚）1钱，龙脑（同麝研，滚药末）1钱，白甘草（炮熟，锉，焙）1两7钱。
出处：《朱氏集验方》卷一。
主治：风疾。妇人血疼。
加减：None
功效：None
用法用量：若修锉生料，众药同片切，生姜半斤或7两，甘草3两，于瓷罐内，用水1斗5升上下，文武火煮水干为度，仍除出甘松、白茅、檀香、缩砂仁、藿香叶勿煮，止令项焙。捣为末，庶香气全，所有煮者众药亦令作一处，焙捣为细末就了，却令研脑、麝同香松、茅数件，大钵内滚2-3日，令药味和合，方和捣为丸。煮药下水之诀，以水高众药1寸余许，文武火煮。一切风疾，炒乌豆淬酒送下；卒中僵倒，或醉酒不醒，生姜汤研3-5粒灌下；小儿惊风天吊，五种痫疾，入京腻粉少许，煎金银、薄荷茶送下；沉寒痼冷，生姜汤送下；一切气疾，茴香汤送下；头风暗风，荆芥汤送下；咳嗽痰涎，半夏汤送下；肺气攻卫，杏仁汤送下；疟疾，乌梅汤送下；水泻，干姜汤送下；痢疾，甘草汤送下；脚疾湿气，木瓜汤送下；酒伤食积，胃气不和，陈皮姜汤送下；肠风，胡桃酒送下；伤寒次霎，葱豉汤送下；时行疫疾，山岚瘴气，不伏水土，藁本汤送下；中毒，甘草水研灌下；牙痛，夜卧含化；牙宣，填窍中；妇人血风、血气，当归酒送下；血脏虚冷，赤白带下，血淋血晕，烧棕榈灰研细调酒送下；腰背疼痛，浑身劳倦，温酒送下。常服，食后茶送下。
制备方法：上为细末，大钵中滚拌两日，极令匀，合糯米粉2两，添水煮糊，同阿胶膏和药为丸，如鸡头大，焙令十分干，瓷瓶收置暖处。
临床应用：None
各家论述：None
用药禁忌：孕妇休服。
附注：《普济方》有官桂、芍药。
