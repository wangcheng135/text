序号：80053
名称：摩风神验膏
组成：硫黄3两（细研），雄黄3两（细研），朱砂3两（细研），附子4两（生，去皮脐），天雄4两（生，去皮脐），人参3两（去芦头），当归3两，细辛3两，防风3两（去芦头），白芷2两，桂心3两，干姜3两，芎3两，川椒3两（去目及闭口者），独活3两，菖蒲3两，川大黄3两，藁本3两，白术3两，吴茱萸3两，松脂半斤（后入）。
出处：《圣惠》卷二十五。
主治：风，身体痛痹，头风目眩，伤风项强，耳鼻俱塞。
加减：None
功效：None
用法用量：病在内，即以酒服弹子大；病在外，即取弹子大，热炙手摩之。
制备方法：上锉细，以酒浸一复时，然后别取生地黄半斤捣绞取汁，同入猪脂中，以慢火煎之，以药味尽为度，以绵滤去滓，后入松脂、雄黄、硫黄、朱砂等，以柳枝不住手搅，至膏凝，收于瓷盒中。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
