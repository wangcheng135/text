序号：58689
名称：秘传药酒
组成：麻黄1两，白芷1两，桔梗1两，芍药1两，当归1两，川芎1两，肉桂1两，半夏1两，防己1两，甘草1两，陈皮2两，厚朴2两，枳壳2两，乌药2两，苍术1两半，槟榔1两半，川牛膝2两，川木瓜1两半，独活1两半，杜仲（制）2两。
出处：《医统》卷十一。
主治：风湿相搏，腰膝痛，或因坐卧湿地，雨露所袭，遍身骨节痛，风湿脚气。
加减：None
功效：None
用法用量：上为粗末，以绢袋盛之，用无灰酒3斗浸于坛内，密封坛口，锅内重汤煮1时，然后取出，过3日开，取酒饮之，量饮，日3次。滓晒干为末，酒糊为丸，如梧桐子大。每服70丸，空心以酒送下。
