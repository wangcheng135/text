序号：79454
名称：摩头散
组成：（艹闾）茹6分，半夏6分，蜀椒6分，乌头8分，莽草4分，桂心7分，附子1两，细辛1两。
出处：《千金》卷十三。
主治：头面风，头中50种病。风毒攻脑疼痛。
加减：None
功效：None
用法用量：摩头，记日数，3日头肤痛，4-5日后一着药如前，十日以酢浆洗头，复摩药即愈。若生息肉并咽喉中息肉，大如枣欲塞，以药摩之即愈。耳鼻齿有疾并用之良。
制备方法：上药治下筛，以大酢和。
临床应用：None
各家论述：《千金方衍义》：（艹闾）茹治大风恶毒，半夏逐湿开痰，蜀椒治寒热痹痛，乌头治寒湿痹，桂心利关节结气，莽草治头风痈肿，附子治寒湿拘挛，细辛治头痛脑痛，百节拘挛。八味辛散药中，得苦酒之酸收，使药内达以搜逐病根，何惮风毒不除，息肉不化耶。
用药禁忌：None
附注：摩顶散（《圣惠》卷四十）。
