序号：43987
名称：厚朴温中饮
组成：附子1钱半（炮），厚朴1钱半（制），白术1钱半（炒），泽泻1钱半，吴萸1钱（醋炒），木香1钱半，干姜1钱半（炒），青皮1钱半（炒），肉桂1钱半（去皮）。
出处：《医略六书》卷三十。
主治：产后腹寒胀，脉紧涩者。
加减：None
功效：None
用法用量：水煎，去滓温服。
制备方法：None
临床应用：None
各家论述：产后火虚，寒滞腹痛，而大便溏泄，小便涩少，遂成腹胀，谓之寒胀。附子补火壮阳以生脾土，肉桂补火散寒以暖血分，干姜温中逐冷，吴萸温中降逆，白术健脾气以运化，厚朴散滞气以宽中，青皮平肝破气，木香调气醒脾，泽泻泻湿以利小便也。水煎温服。使火暖土强，则寒滞自化，而小水自快，大便亦实，安有腹胀疼痛之患乎。
用药禁忌：None
