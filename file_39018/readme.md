序号：39028
名称：泽兰丸
组成：泽兰1两，赤石脂1两，牛膝（去苗，酒浸，切，焙）1两，人参1两，黄耆（锉）1两，熟干地黄（焙）1两半，白茯苓（去黑皮）3分，木香3分，萆薢3分，附子（炮裂，去皮脐）3分，续断3分，桂（去粗皮）3分，芎半两，白术半两，干姜（炮）半两，当归（锉，炒）半两，甘草（炙）半两。
出处：《圣济总录》卷一六四。
主治：产后虚羸，气血不调，四肢瘦弱，面色萎黄，饮食减少。
加减：None
功效：补益，壮气益血。
用法用量：每服30丸，空心、食前以温粥饮送下，1日3次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
