序号：33694
名称：灵宝救苦丹
组成：麝香5分，番蟾酥（酒化）5分，真狗宝5分，西牛黄5分，廉珍珠3分，梅冰片2分，明硼砂5分，草河车1钱，真熊胆3分，真山上百草霜1钱（如无，用陈京墨代），真血竭5分，去油乳香5分，去油没药5分，生大黄1钱（勿见火），明雄黄（制）5分，飞净朱砂5分，生玳瑁5分，真琥珀5分，乌沉香2分，青木香3分。
出处：《经验各种秘方辑要》。
主治：痈疽发背，疔疮、内痈，恶疠、蛊毒，烂喉丹瘀、霉疮结毒，一切无名肿毒，已溃未溃，毒邪毒火。
加减：None
功效：止痛护心，解毒化脓。
用法用量：每服1丸，重者2-3丸，陈绍酒送下，以醉取汗为度。并可外敷。
制备方法：上为极细末，用头生男孩人乳和化为丸，每丸1分，真金箔5分为衣。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
