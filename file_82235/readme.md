序号：82251
名称：瘰疬拔根方
组成：斑蝥7个（去头足并翅，同糯米炒用，米黄黑色为度，取蝥为末听用），鸡蛋1个（钻一孔，用银簪搅匀，入斑蝥末于内，再搅匀，另取鸡子壳补口，绵纸粘固，再用粗纸包七层，水湿透听用），土茯苓4两（择小而扁为佳，若长而大者名为奶肩，有毒，故不用），菜猪肉半斤（煎汤听用）。
出处：《外科大成》卷二。
主治：瘰疬。
加减：None
功效：None
用法用量：于五更时将药蛋用粗糠火煨熟，乘热食之，以前土茯苓肉汤送下。3日后，小腹胀痛，以益元散推之；如尚未利，再饮热水摧之，毒根从小便下，形如圆眼，如烂肉，其根下，其疬自消，甚验。或以木通、滑石、青黛之类导之，青黛善解斑蝥之毒。
