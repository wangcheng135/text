序号：50390
名称：茵芋漫酒
组成：茵芋1两半，细辛半两，天雄1两（炮裂，去皮脐），汉防己1两，川乌头1两半（炮裂，去皮脐），石斛1两（去根），踯躅1两（微炒），山茱萸3分，柏子仁3分，甘草（炙微赤）3分，木通3分，桂心3分，秦艽（去苗）3分，黄耆3分，干姜（炮裂）3分，熟干地黄3分，莽草（微炙）3分，附子（炮裂，去皮脐）3分，杜仲（去粗皮）3分，芎藭3分，王孙3分，泽泻3分，石南3分，防风（去芦头）3分，远志（去心）3分，牛膝（去苗）3分。
出处：《圣惠》卷二十五。
主治：八风十二痹，五缓六急，半身不遂，四肢偏枯，筋脉拘挛，肩髀疼痛，腰脊不能俯仰，胸胁膨胀，心烦，目眩耳聋，咽喉不利；或贼风所中，痛如锥刺，行人皮中，无有常处；或四肢肌体偏有冷痹，状如风吹。
加减：None
功效：None
用法用量：空心温服1小盏，晚食前再服。以愈为度。
制备方法：上锉细，用生绢袋盛，以酒3斗，浸10日。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻、动风物。
附注：None
