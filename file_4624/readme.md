序号：4624
名称：木香保命丹
组成：木香1两，白附子1两（生用），官桂1两，杜仲1两（去粗皮，炒去丝），厚朴1两（去皮，生姜汁炒干），藁本1两（去须土），独活1两，羌活1两（生用，去芦头），海桐皮1两（生），白芷1两，甘菊花1两（去土），牛膝1两（去苗，酒浸1日，焙干），白花蛇1两（酒浸3日，去皮骨，焙干称），全蝎1两（炒），威灵仙1两（水浸，去土），天麻1两（别捣，取末，去土），当归1两（去芦头，水浸，去土，干称），蔓荆子1两（生，去皮），虎骨1两（酒浸焦黄，去油，或酥炙，或用粗心），天南星1两（浆水煮5-7遍），大防风1两（去芦头，干称），山药1两（生用），甘草1两（酥炙微黄），赤箭1两（生用），麝香3钱（真者，别研），朱砂（上好者）1两半。
出处：《御药院方》卷一。
主治：男子、妇人体虚腠开中风，牙齿噤，口眼喎斜，手足偏枯，四肢拘挛，屈伸不得，麻痹不仁，惊痫，遍身瘙痒疼痛，头目昏暗，风入腹内，拘急切痛，体如虫行，心神恍惚，伤风瘴疫，偏正头疼，风病，诸般冷气；兼疗男子、妇人脾胃气虚，或伤冷物，心腹大痛，脏腑不调；妇人产前、产后中风病，壮热体重，头疼，旋运欲倒，气闭血涩，月事不行。或中酒、痰，昏倦力乏，饮食减少。
加减：None
功效：引血调养荣卫，升降阴阳，补益五脏。壮元阳，理筋骨腿膝；化风痰，快滞气，温脾胃，进饮食。
用法用量：每服1丸，细嚼酒下，不拘时候。如中风，加薄荷汤化下；如不能咽者，灌之；小儿急慢惊风，薄荷汤下1皂子大。
制备方法：上为细末，其药分作10份，将麝香1分拌匀，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
