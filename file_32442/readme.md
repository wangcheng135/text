序号：32451
名称：应痛膏
组成：当归半两，秦艽半两，何首乌半两，败龟半两，白蔹半两，白及半两，白术半两，白芷半两，杏仁（去皮尖）半两，木鳖子（去壳）半两，芎半两，延胡素半两，密佗僧（煅，研）半两（用麻油8两熬前药，令杏仁黄黑色为度，漉出药滓，入后药），乳香（研）1分，麒麟竭（研）1分，没药（研）1分，枫香脂（研）1分，铅丹3两。
出处：《圣济总录》卷一三○。
主治：痈疽发背，及诸种疮肿。
加减：发背，加附子末1分，同熬匀，用纸花子贴疮上。
功效：None
用法用量：None
制备方法：上先用油煎前十三味，去滓，入后五味再熬，用柳枝干搅匀，令黑色成膏青为度。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
