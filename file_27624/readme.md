序号：27632
名称：八金散
组成：金精石、银精石、阳起石、玄精石、磁石、石膏、滑石、禹余粮石各等分。
出处：《准绳·类方》卷五。
主治：癞病。
加减：None
功效：None
用法用量：令患人先洗疮，拭干，便用小油调稠硬作剂子，于有疮处擦上药。2次药了，用贯众汤嗽其口，不可咽下药汁，两手便洗净，不可近口、鼻、耳、目，第4日1伏时，依前上药。第7日不可更用，见效即止。
制备方法：上为末，入金、银钳锅子内盛之，用盐泥固济口，以文、武火煅炼红透，放冷，研如粉，入水银半两，轻粉1钱，研令不见星子，却入余药，再研匀。
临床应用：None
各家论述：None
用药禁忌：擦药之后，大忌饮水；宜禁身静坐至3日，口中涎出为度。
附注：None
