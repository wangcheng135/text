序号：45924
名称：栀子柏皮汤
组成：肥栀子15个（劈），甘草1两（炙），黄柏2两。
出处：《伤寒论》。
主治：伤寒，身黄发热者；衄血，或从口出，或从鼻出，暴出而色鲜，衄至1-2斗，闷绝者。
加减：None
功效：None
用法用量：上以水4升，煮取1升半，去滓。分2次温服。
制备方法：None
临床应用：None
各家论述：《温病条辨》：栀子清肌表，解五黄，又治内烦；黄柏泻膀胱，疗肌肤间热；甘草协利内外。三者其色皆黄，以黄退黄，同气相求也。
用药禁忌：None
附注：柏皮汤（《鸡峰》卷十）、柏皮散（《永乐大典》卷一○三三引《全婴方》）。《鸡峰》本方用法：为粗末。每服五钱，水二盏，煎至一盏，去滓温服。
