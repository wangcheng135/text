序号：25728
名称：朱砂散
组成：朱砂（拣去石，研）2钱，白附子2钱，附子1个（去皮脐），天南星（去脐）1钱，天麻1钱，干蝎（全者）1钱，半夏（汤洗7遍，去滑为度）1钱。
出处：《永乐大典》卷九七五引《灵苑方》。
主治：一切惊风，不问急慢，咬齿拗项，翻眼气粗，手足搐搦。
加减：None
功效：压惊，安魂定魄，镇心脏，退风热。
用法用量：2岁以下每服半字，4-5岁以下每服1字强，6-7岁半钱，13-15岁加至1钱。如是退一切惊热、风热、咳嗽，喉中壅隘，并宜食后用蜜和熟水调下；一切风不问急慢，咬齿拗项，翻眼气粗，手足搐搦，并用冷茶清调下。服良久，膈上有风涎则吐之，不尔汗出立愈；亦有寻时并定也。有猛发一上者，良久再进1服。疾定后更加牛黄少许，和前药用薄荷蜜水调下，日3次。若要为丸子，用薄荷、生姜汁、蜜、酒煮面糊为丸，如绿豆大，每服5丸，用薄荷汤送下。
制备方法：上各细锉，晒干，为极细散；次入朱砂末，再研合令匀，以瓷罐收之。
临床应用：None
各家论述：None
用药禁忌：忌动风毒物。
附注：丹砂散（《圣济总录》卷一七一）。
