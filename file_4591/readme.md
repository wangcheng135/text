序号：4591
名称：木香补肾丸
组成：怀庆生地4两（酒煮捣膏），菟丝子1两2钱，肉苁蓉1两2钱，黄精1两2钱，黑枣肉1两2钱，牛膝1两2钱，蛇床子1两2钱（微炒），茯苓1两2钱，远志1两2钱，当归身2两4钱，丁香3钱，大茴香6钱，木香6钱，枸杞子1两5钱，巴戟1两，杜仲1两，青盐5钱，人参5钱。
出处：《外科正宗》卷三。
主治：偏坠，一名木肾，不疼不痒，渐渐而大，最为顽疾，有妨行动，多致不便；诸疝，不常举发者；及精寒血冷，久无嗣息。
加减：None
功效：None
用法用量：每服60-70丸，空心温酒送下。偏坠者，灸后宜眼此，俱可内消。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：此药功效不独治疝，中年后服之益寿延年，黑发壮筋，填髓明目，聪耳补肾，助元阳，调饮食。其功不可尽述。妇人服之，颜如童女，肌肤莹洁如玉。
