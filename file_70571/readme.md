序号：70587
名称：紫沉消积丸
组成：沉香1两（为末），阿魏1分（研），巴豆霜4钱，硇砂1两（以上用酒蜜约度多少，一处熬成膏子，然后搜药），朱砂半两，丁香半两，干姜半两，硫黄1两，青皮1两，高良姜1两，槟榔1两，木香1两，人参1两，胡椒1两，官桂1两。
出处：《中藏经·附录》。
主治：脾积滞气，酒食所伤，饮食不化，恶心呕逆，胸膈不快，不思饮食，胸腹胀满，脐胁有块，心脾冷痛，口吐酸水，停饮冷痰，痃癖癥瘕，翻胃，黄瘦浮肿；脏腑不调，里急后重；及十膈气虚，妇人血气块硬。
加减：None
功效：None
用法用量：每服5-7丸，食后、临卧橘皮汤送下，常用1-2丸，更看虚实加减。
制备方法：上为末，将熬下膏子搜药匀和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：紫沉煎丸（《百一》卷八）。
