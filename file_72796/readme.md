序号：72812
名称：紫石英汤
组成：紫石英10两，白石英10两，白石脂30两，赤石脂30两，干姜30两。
出处：《千金翼》卷十二。
主治：心虚惊悸，寒热百病。
加减：None
功效：补虚，除痼冷，令人肥健。
用法用量：上(口父)咀，用二石英各取1两，石脂等3味各取3两，以水3升，合以微火煎，宿勿食，分为4次服，日3次，夜1次，服后午时乃食，日日依前称取昨日药，乃置新药中共煮，乃至药尽常然，水数一准新药尽讫，常添水，去滓，服之满40日止。服汤讫即行，勿住坐卧，须令药力遍身，百脉中行。若大冷者，春、秋各49日，服令疾退尽，极须澄清服之。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌酒、肉。
附注：None
