序号：62602
名称：混元胎丹
组成：首儿衣（二八月者不用，收时连带中元血。用长流水净洗，控干，入瓷瓶中，下无灰酒3大盏，脑子、麝香随力下之，多至1钱或半字，以纸封瓶口，下用文武火煅，候酒将尽取出，再入酒3盏，依前煅，却用竹篦或金银蓖不住搅，以搅烂糜似粥样，待冷，取入砂盆内，细研为粉，别入外料），人参2两，茯苓2两，乳香半两，朱砂半两（水飞过，晾干，取净），山药4两（上为细末，入所煮药内，如干，更用原煅酒，旋旋添）。
出处：《洪氏集验方》卷一。
主治：久无嗣息者。
加减：None
功效：补漏壮气，固神益髓，通神明，延寿命。
用法用量：每日空心50粒，加至100粒，用温酒、盐汤送下。良久，用甜淡饮食饱压，逐日荤味中减少5味。服至5-7日，微觉小腹连腰沉重，不须疑虑，乃是药力攻，元气相补助如此。
制备方法：上拌匀1处，搅搜为丸，如梧桐子大，慢火焙干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
