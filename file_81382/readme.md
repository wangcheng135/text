序号：81398
名称：麒麟竭膏
组成：白芷4两，白蔹4两，川芎4两，甘草4两，当归半两，丁香半两，木鳖子38个，没药1两半（另研），乳香1两半（另研），脑4两，干蟾半两，杏仁98个，鼠头（腊月者）两个，清麻油2斤，麒麟竭1两，真绯绢1尺（烧灰），黄丹10两，室女油头发1拳大团。
出处：《普济方》卷三一四。
主治：入发痈疽，一切恶疮软疖，无问年月深远，已成脓未成脓；汤火刀斧所伤。
加减：None
功效：None
用法用量：每用少许贴患处。
制备方法：上锉细，用好酒拌浸1宿，入铛内用油煎，候药深赤色，滤去滓；另入净铛，慢火煎，可少顷即入研者麒苓竭、乳香、黄丹、腊等，用柳枝子不住搅打，时时滴入水，试看软硬得所，即是成青；发背未脓者，半入银石器，慢火熬及半盏许，去滓，次下乳香（研碎），又熬之，候如1茶脚许，先将蜜熬去滓，放冷，却入前熬者膏子及众末，搅匀，再熬，候金漆状乃成，入不犯水磁器内收之。
临床应用：None
各家论述：None
用药禁忌：None
附注：麒麟膏（《奇效良方》卷五十四）。
