序号：51524
名称：追风散
组成：葶苈3分，胡麻（炒）2两，苦参2两，蒺藜（炒）2两，防风2两，花粉2两，全蝎2两（醋炙黄），僵蚕（炒）3两，蝉退3两，甘草（水洗）。
出处：《秘传大麻疯方》。
主治：大麻风浑身黑而不发出者。
加减：None
功效：None
用法用量：苦茶送1服。服3日后，唇口肿起，牙缝吐黑水2碗，必遍身疼痛；如痛、臭，用漱口散；服尽，必痢下五色便溺，乃脏腑根源毒气恶水皆出。
制备方法：上为末，加轻粉1两和匀，分作18次服。
临床应用：None
各家论述：None
用药禁忌：虚人忌用。
附注：痢下之物用瓦器盛之，埋于无人处。病人吃粥时，用漱口散药，吐尽毒物，方可吃粥。间人看望，须待漱口完后方可入所。吐毒物用盖桶盛之，埋入土内，须防毒气传染。方中甘草用量原缺。
