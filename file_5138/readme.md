序号：5138
名称：玉液至宝丹
组成：桔梗3分，前胡3分，山楂3分，木香3分，茯神3分，红花3分，青皮3分，通草3分，牛蒡子3分，糯米1撮，灯心7根。
出处：《痘疹仁端录》卷五。
主治：痘疮，自标痘至起胀6日内。
加减：None
功效：None
用法用量：水煎服。据症加减用之。
制备方法：None
临床应用：None
各家论述：《仁端录》：桔梗利咽喉，载气血上行，为舟揖之剂；前胡能托痘，除内外之痰热；山楂初时多用，最能生肌发痘；木香少用以提肾毒，2帖后不用；茯神安神养心血；红花少用以活血，多则逐血，4-5帖后不用；青皮能清脏腑，免后生痰，3帖后换陈皮，通草能通关节，3帖后换木通，小便清不必换。
用药禁忌：None
附注：方中通草，原书卷十四作“甘草”。
