序号：64736
名称：硇砂木香丸
组成：巴豆1两（去皮，以纸出油净为度，另研），硇砂半两（另研细后入巴豆，入诸药），附子1枚（炮去皮脐），官桂（去皮）、茱萸（炒）、舶上茴香、荆三棱（炒）、干姜（炮）、木香、丁香各等分。
出处：《博济》卷二。
主治：丈夫妇人一切冷气，攻刺疼痛，或成积聚，隐现不常，发则(疒丂)痛。
加减：None
功效：利气化痰。
用法用量：取食。利胸膈气，淡茶送下10丸；女人血气及诸般气，艾酒送下；丈夫脏腑气，葱酒送下；化疾，津液送下。
制备方法：同为末，用干柿1枚，洗过，蒸令软，和末为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
