序号：26026
名称：妙灵散
组成：斑蝥21个（去头翅足，同糯米炒黄，只用斑蝥末2字。），黑牵牛（头末）1两，荆芥穗1钱，僵蚕（炒去丝嘴）1钱，当归1钱半，木通1钱，滑石1钱。
出处：《医方类聚》卷一九○引《修月鲁般经》。
主治：瘰疬疮。
加减：None
功效：None
用法用量：作1服，五更初用无灰酒调下。必先于隔夜临睡服玉屑散1服，以米汤调下。次日五更，可服前药，天明，水道下新胡桃肉相似者是也。如不见，停3日，再依法服之。倘小便涩痛急，琥珀末以灯心汤调下，催之恶物，不下之，后可服活血丸调理。疮迹上贴云母膏。
制备方法：上为极细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
