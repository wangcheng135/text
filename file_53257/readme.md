序号：53272
名称：健步虎潜丸
组成：黄耆（盐水炒）、当归（酒洗）、枸杞子（酒洗）、龟版（酥炙）1两，知母（人乳汁、盐、酒炒）2两，牛膝（去芦，酒洗）2两，白术（去芦）2两，白芍（盐、酒炒）2两，生地黄2两，熟地黄2两，虎胫骨（酥炙）2两，杜仲（姜、酒炒）2两，人参（去芦）2两，破故纸（盐、酒炒）1两，麦门冬（水泡，去心）1两，白茯神（去皮木）1两，木瓜1两，石菖蒲（去毛）1两，酸枣仁1两，远志（甘草水泡，去心）1两，薏苡仁（炒）1两，羌活（酒洗）1两，独活（酒洗）1两，防风（酒洗）1两，黄柏（人乳汁、盐、酒炒）2两，五味子5钱，沉香5钱，大附子（童便浸透，面裹煨，去皮脐，切4片，又将童便浸，煮干）5钱。
出处：《回春》卷二。
主治：肝肾不足，中风瘫痪，舌强言謇；筋骨痿弱，腰腿痠痛，四肢无力。手足不能动，老年衰迈或壮年病后，筋骨无力。阴虚盗汗，遗精白浊，肾虚脚气。
加减：None
功效：祛风活血．壮阳益精。强筋壮骨，补肾填精，燥湿利下。
用法用量：每服100丸，温汤或酒送下。
制备方法：上为末，炼蜜和猪脊髓5条为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
