序号：24118
名称：安胎主膏
组成：党参2两，酒当归2两，熟地3两，酒条芩1两半，淮药1两半，白术1两半，酒川芎5钱，酒芍5钱，陈皮5钱，苏梗5钱，香附5钱，杜仲5钱，续断5钱，贝母5钱（一方加黄耆1两，生地1两）。
出处：《理瀹》。
主治：下血，于肿，子喘，子痫，肝脾血热．小便带血，胎动不安。
加减：下血者，加桑寄生、阿胶各5钱；子肿，加姜皮、茯苓皮、大腹皮、陈皮、栀子末调；子喘、加马兜铃、桔梗、贝母；子痛，加防风、独活、羚羊屑；止呕定痛，加砂仁少许；肝脾血热，小便带血，加紫胡、黑山栀；胎动不安，1月用乌雌鸡，十月用猪腰入药。
功效：安胎，止呕定痛。
用法用量：麻油熬，黄丹收，贴肾俞处。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
