序号：46551
名称：千金夺命鳖牙膏
组成：香白芷3分，红皮3分，白术3分，青皮3分，细辛3分，红芍药3分，白附子3分，败龟3分，通草3分，虎骨3分，骨碎补3分，苍术3分，海蛤3分，当归3分，蜜陀僧3分，干姜3分，鳖牙1钱半，没药1钱半，桂2钱，木鳖子5个，（月屈）（月奢）枝（（月葛）模枝也）2钱，补（月屈）（月奢）（根也）2钱，黑附子2个，血竭1钱半。
出处：《鸡峰》卷二十二。
主治：远年恶疮，疳疮，漏疮，无名恶毒疮。
加减：None
功效：None
用法用量：None
制备方法：上为细末，每料使药末半两，先用清油3两，铫子内煮沸，入黄丹1两，不住以柳枝搅，候黑色，取离火。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
