序号：77938
名称：醋石榴子方
组成：酸石榴7颗（去皮，置于盆内，炊饭甑上蒸烂，绞取汁），冬消梨20颗（去皮核，研绞取汁），羌活1两，犀角屑半两，防风1两（去芦头），干薄荷叶1两，茺蔚子半两，白附子半两（炮裂），苦参半两（锉），人参1两（去芦头），乌喙半两（炮裂，去皮脐）。
出处：《圣惠》卷二十四。
主治：紫癜风。
加减：None
功效：None
用法用量：每服20丸，以温酒调下，不拘时候。
制备方法：上药除汁外，为末，取前二味煎如膏，和丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：醋石榴丸（《圣济总录》卷十八文瑞楼本）、酸石榴丸（《圣济总录》卷十八人卫本）。
