序号：53079
名称：鬼哭饮子
组成：天灵盖（酥炙黄色）、鳖甲（醋炙黄色）、柴胡2钱半，木香1钱2分，桃仁22枚（另研），豉心（醋炙黄）、青蒿半握，阿魏1钱，贯众2钱半，安息香1钱，甘草1钱。
出处：《医统》卷七十八引《青囊》。
主治：None
加减：None
功效：取传尸虫。
用法用量：上（口父）咀，为粗末，先以童便2升隔夜浸露，至四更时煎至8分，去滓，分作3服。每服调后散子1帖，五更初温服，即稳卧片时，又进1服，至日出时觉腹中欲利；如未利，又进1服，已利勿服。服后利下恶物并虫，能传注后人，必置之烈火，或油煎之，以绝后患。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：散子方：槟榔二钱半，辰砂一钱二分，麝香一钱（另研），蜈蚣一条（姜汁浸），乌鸡屎（先将鸡五日前用火麻子喂之，五日后则用其粪）。为细末，作三帖，各入前煎汤内服。方中天灵盖、鳖甲用量原缺。
