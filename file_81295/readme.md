序号：81311
名称：鳖甲汤
组成：鳖甲（醋炙黄）1两，桑根白皮（锉）1两，甘草（炙，锉）1两，贝母（去心，炒）1两，麦门冬（去心，焙）1两，白术1两，人参1两，陈橘皮（去白，焙）1两，胡黄连半两，知母（焙）半两，麻黄（去根节）半两，黄芩（去黑心）半两，百部半两，紫菀（去苗土）半两，山栀子仁半两，栝楼根半两，柴胡（去苗）半两，款冬花半两。
出处：《圣济总录》卷九十。
主治：虚劳。烦躁，羸瘦发热，面红颊赤，咳吐脓血。
加减：None
功效：None
用法用量：每服3钱匕，水1盏半，煎至8分，加杏仁7个（去皮尖），再同煎至6分，去滓，临卧温服，如嗽止，不用杏仁，只用生姜少许同煎。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
