序号：32075
名称：万应丸
组成：大腹子2两（同槟榔制法），使君子20枚，贯众（去土）5钱，土朱5钱，雷丸（水浸，刮去黑皮）5钱（红者不用），木香3钱（另研），鸡心槟榔2两（用无灰酒浸半日，至心黑为度，以黄豆煎汤洗），自然铜1两（有金星小者佳，用醋煅7次，以酥为度），锦纹大黄5两5钱（湿纸裹，煨半生半熟），黑牵牛4两（炒半生半熟，取头末2两），滑石1两（白者佳）。
出处：《便览》卷三。
主治：虫积，面色萎黄，肌肤羸瘦，胸膈停痰，宿食不化，肚腹膨胀，虫咬心疼，吃泥炭米物。五脏中诸气血积聚，妇人癥瘕。
加减：None
功效：追虫取积。
用法用量：大人3钱，小人减半，莫食晚饭，四更时冷茶送下。行5-7次无妨。候虫积诸毒下尽，以温米粥补之。
制备方法：上为细末，皂角水煎成膏为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌腥冷面食。孕妇忌服。
附注：None
