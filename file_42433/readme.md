序号：42448
名称：青黛丸
组成：青黛（别研，留一半为衣）半两，使君子（肉）半两，槟榔（锉）半两，夜明砂半两，白芜荑（醋炒令紫）半两，蛇蜕1条（炙），肉豆蔻（面裹煨，去壳）1分，巴豆半分（用麸炒令紫，去壳，别研），麝香1钱（研），黄连（去须，炒）1分半，蛤蟆1枚（酒浸，炙令黄紫色，去骨）。
出处：《圣济总录》卷一七三。
主治：五疳。
加减：None
功效：杀虫，化食，长肌肉，退风热。
用法用量：3岁以下者，每服1丸，5岁2丸，10岁3丸，食后用熟水送下，1日3次。如风热大便涩，用葱汤送下5-7丸，以通利为度。
制备方法：上先研巴豆为细末，次用猪胆1枚取汁同研令匀，旋入其余药末为丸，如未就，更用少软饭再研得所，丸如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
