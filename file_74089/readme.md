序号：74105
名称：愈痛丸
组成：川萆薢1两，鳖甲1两，川当归（去芦，酒浸）1两，三棱（炮）1两，破故纸（炒）1两，神曲（炒）1两，蓬莪术（炮）1两，麦糵（炒）1两，熟干地黄（洗）1两，干漆（炒令烟尽）半两，延胡索（炒）半两，茴香（淘去沙，炒）半两，沉香（不见火）半两，肉桂（去粗皮，不见火）半两，没药半两（别研），麝香半钱（别研）。
出处：《魏氏家藏方》卷六。
主治：惊忧气滞，脾肾积寒，内夹冷气，久成痃癖癥瘕，透隐皮肤，或两胁牵痛不已，及小肠奔豚气痛。
加减：None
功效：None
用法用量：每服20-30丸，温酒或盐汤送下，不拘时候，1日2服。
制备方法：上为细末，醋煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
