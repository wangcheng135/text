序号：46299
名称：活血润燥丸
组成：当归梢1钱，防风3钱，大黄（湿纸裹煨）1两，羌活1两，皂角仁（烧存性，去皮）1两5钱，桃仁2两（研如泥），麻仁2两5钱（研如泥）。
出处：《兰室秘藏》卷下。
主治：大便风秘，血秘，常常燥结。
加减：None
功效：活血疏风，润燥通便。
用法用量：每服50丸，白汤送下3两，服后须以苏麻子粥，每日早晚食之。大便日久不能结燥也。
制备方法：上除麻仁、桃仁另研如泥外，为极细末，炼蜜为丸，如梧桐子大，以瓷器盛之，纸封无令见风。
临床应用：None
各家论述：《医方集解》：此手足阳阴药也。归尾、桃仁润燥活血，羌活搜风散邪，大黄破结通幽，麻仁滑肠利窍，血和风疏，肠胃得润则自然通利矣；皂角得湿则滑，湿滑则燥结自除。
用药禁忌：None
附注：活血润肠丸（《丹溪心法》卷二）。本方方名，《东医宝鉴·内景篇》引作“疏风润肠丸”。
