序号：79209
名称：增损泽兰丸
组成：泽兰（熬）7分，防风5分，干地黄5分，当归5分，细辛5分，桂心5分，茯苓5分，芍药5分，人参5分，甘草（炙）5分，藁本5分，乌头（炮）5分，麦门冬（去心）5分，石斛5分，紫菀5分，川芎5分，干姜4分，柏子仁4分，芜荑仁4分，厚朴4分，川椒（汗）4分，白术6分，黄耆6分，紫石英（研）8分，石膏（研）8分。
出处：《外台》卷三十四引《延年方》。
主治：产后风虚劳损，黄瘦。
加减：None
功效：None
用法用量：每服20-30丸，以酒送下，日2次。
制备方法：上为细末，炼蜜和丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌如常法。
附注：None
