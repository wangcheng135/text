序号：79327
名称：槟榔汤
组成：白槟榔4两（一半煨，一半生用），肉豆蔻（去壳）1两，木香1两，青橘皮（去白，焙）2两，厚朴（去粗皮，生姜汁炙透）2两，枳壳（去瓤，麸炒）2两，京三棱（煨，锉）2两，桂（去粗皮）2两，人参2两，白茯苓（去黑皮）2两，陈曲（炒）2两，麦糵（炒）2两，干姜（炮）2两，白术2两，诃黎勒（炮，去核）2两，甘草（炙，锉）2两。
出处：《圣济总录》卷五十四。
主治：三焦气满，虚胀及一切脏腑气疾。
加减：None
功效：None
用法用量：每服3钱匕，入生姜1块拍破，大枣2枚（去核），同煎至7分，去滓温服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
