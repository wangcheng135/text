序号：36791
名称：固齿延寿膏
组成：珍珠5钱（绢作小袋盛之，豆腐一方，中开1孔纳珠袋，仍以原腐掩孔，留袋上1线悬锅口上，勿令珠袋落底，恐伤珠原气，桑柴火煮1炷香为度），雄鼠骨（腊月内雄鼠1只，面作饼，将全鼠包完，外用盐泥复包，阴干，炭火烧红，冷定，破取骨）5钱，秋石2钱，龙骨（制法与鼠骨同）5钱，阳起石5钱，象牙5钱，鹿角霜5钱，广木香2钱半，沉香2钱，南川芎1钱，怀熟地黄1钱，白芍药1钱，当归1钱，乳香1钱，没药1钱，青盐1钱半，白芷5分，大小皂角5分，破故纸（炒香，忌铁）、细辛（去土，洗净，晒干）3分。
出处：《扶寿精方》。
主治：龈宣齿搞，黄黑腐败，风虫作痛，（思页）颊红肿。
加减：None
功效：坚固牙齿，驱逐垢腻，益肾生津，壮骨强髓，添精倍力。
用法用量：临卧贴在牙上1夜，次早取出，药条皆黑，齿牙渐贴渐固。
制备方法：上为极细末，白蜡5两，俱各作2分。用蜜煎罐1个，先将白蜡化开，次下1分药末，桑柴文火溶开蜡，将药搅匀，外用呈文纸二张，将前药1分散在纸上，用手擦磨药末，在纸上下周围，后将罐内药火化开，搅匀倾在纸上，周围俱用药汁走到，用刀作条。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中破故纸用量原缺。
