序号：40847
名称：软犀丸
组成：沉香1分，檀香1分，丁香半两，木香半两，肉豆蔻（面裹煨熟）半两，槟榔半两（6味同为细末），巴豆21粒（去壳），杏仁21个（去皮尖），白丁香1字，鹰粪白1字，百草霜1两。
出处：《杨氏家藏方》卷五。
主治：久虚沉积，心腹撮痛，肠滑下痢，脏腑不固，日渐羸弱。
加减：None
功效：None
用法用量：每服10-30丸，食后、临卧生姜汤送下。
制备方法：上药除沉香等6味外，将后巴豆等5味，用铁铫子内慢火炒至大烟出，无令太过不及，候得所，用1瓷盏密盖，放冷1宿，研细与前药末相和，再和研匀，用黄蜡1两半，麻油1分，同炼溶，和药成剂，为丸如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
