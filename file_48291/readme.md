序号：48306
名称：神效千捶膏
组成：土木鳖（去壳）5个，白嫩松香（拣净）4两，铜绿（研细）1钱，乳香2钱，没药2钱，蓖麻子（去壳）7钱，巴豆肉5粒，杏仁（去皮）1钱。
出处：《医宗金鉴》卷六十二。
主治：疮疡，疔毒初起，并治瘰疬，大人臁疮，小儿蟺拱头。红肿坚硬，瘰疬结核，臁疮溃烂，经年不愈。
加减：None
功效：活血消肿，化坚止痛。
用法用量：用时随疮大小，用手捻成薄片，贴疮上，用绢盖之。
制备方法：上合一处，石臼内捣三千余下，即成膏；取起，浸凉水中。
临床应用：None
各家论述：None
用药禁忌：None
附注：千捶膏（《药奁启秘》）。本方方名，《北京市中药成方选集》引作“瘰疬千捶膏”。
