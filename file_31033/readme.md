序号：31042
名称：李八伯杏金丹
组成：肥实杏仁5斗。
出处：《圣济总录》卷一九八。
主治：None
加减：None
功效：久服保气延年，变白，除万病。
用法用量：每服20丸，空心、日暮酒送下；或用津液送下。
制备方法：以布袋盛，用井花水同浸3日，次入甑中，以帛覆之，上布黄沙5寸，炊1日，去沙取出，又于粟中炊1日，又于小麦中炊1日，又于大麦中炊1日，压取油5升，澄清，用银瓶1只，打如水瓶样，入油在内，不得满，又以银元叶，可瓶口大小盖定，销银汁灌固口缝，入于大釜中，煮7复时，常拔动看油结，打开取药入器中，火消成汁，倾出放冷，其色如金，后入臼中，捣之为丸，如黄米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
