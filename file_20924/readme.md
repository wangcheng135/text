序号：20932
名称：人参散
组成：人参1两（去芦头），黄耆1两（锉），白术半两，当归半两（锉，微炒），白茯苓半两，木香半两，芎半两，草豆蔻1两（去皮），白芍药半两，诃黎勒皮2分，桂心半两，附子1两（炮裂，去皮脐），陈橘皮3分（汤浸，去白瓤，焙），甘草半两（炙微赤，锉），高良姜2分（锉），厚朴1两（去粗皮，涂生姜汁，炙令香熟）。
出处：《圣惠》卷八十一。
主治：产后虚羸，脾胃乏弱，四肢无力，全不思食，心腹气胀。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，大枣2枚，煎至6分，去滓，不拘时候温服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
