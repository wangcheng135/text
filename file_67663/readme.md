序号：67679
名称：小安肾丸
组成：香附子1斤，川乌1斤，川楝子1斤（以上用盐4两，水4升同煮，候干，锉，焙），熟干地黄8两，茴香12两，川椒（去目及闭口者，微炒出汗）4两。
出处：《局方》卷五（续添诸局经验秘方）。
主治：肾阳不足，夜多小便，腰膝沉重，肠鸣泄泻，耳鸣目暗，牙齿动摇出血，寒湿疝气。肾气虚乏，下元冷惫，夜多旋溺，肢体倦怠，渐觉羸瘦，腰膝沉重，嗜卧少力，精神昏馈，耳作蝉鸣，面无颜色，泄泻肠鸣，眼目昏暗，牙齿蛀痛。久泻；阳虚于下，虚火上浮，口不臭，牙不痛，但齿摇不坚，或微痛不甚而牙缝多出血者；痿证。小腹寒疝作痛。肾虚冷惫，阴火上升，喘嗽，齿疼，腰痛。女人胞门受寒，小腹疼痛。
加减：None
功效：补虚损，益下元。
用法用量：每服20丸至30丸，空心、卧时盐汤、盐酒任下。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：《杂病证治》：风寒袭入肾经，不能主骨而牙失所养，故牙齿疼痛、牙根摇动焉。熟地补阴坚齿牙，川乌入肾逐风寒，川椒温中逐冷，茴香温经散寒，香附调血中之气，楝子泻湿中之热以肃清经腑也，盐汤润下，酒服温行，使风寒外解，则肾气清融；而骨得所养，则齿无不坚，何疼痛之不痊哉。此补虚逐冷之剂，为风寒袭肾齿痛之专方。
用药禁忌：None
附注：None
