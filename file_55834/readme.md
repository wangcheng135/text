序号：55849
名称：桑白皮汤
组成：柴胡（去苗）1两半，桑根白皮1两半，天雄（炮裂，去皮脐）1两半，羌活（去芦头）1两半，枳壳（去瓤，麸炒）1两半，大腹（连皮锉）1两半，黄连（去须）1两，当归（切，焙）1两，麻黄（去根节）1两，桂（去粗皮）1两，甘草（炙，锉）1两，白梅（拍碎）4块，黄芩（去黑皮）半两，旋覆花（微炒）半两。
出处：《普济方》卷一六一。
主治：咳嗽，上气促急，心躁寒热，四肢烦疼，夜间甚者。
加减：None
功效：None
用法用量：每服5钱，水1盏半，入生姜3片，同煮8分，去滓，温服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
