序号：62450
名称：梅花饮子
组成：南硼砂半两，马牙消（另研）半两，芒消（另研）半两，甘草（炙）半两，人参1两，辰砂2钱半，片脑1字，麝香1字。
出处：《婴童百问》卷十。
主治：婴孩惊热、潮热、虚热、积热，五脏蕴热，上焦壅热，手足心热，喉中多痰，面色或红或白，变蒸(口兹)牙，鼻流清涕，气急，肝肺壅热，目赤咳嗽；或被人物所惊，夜啼睡卧不安，心中惊怖，情绪不快；或伤寒渐安，尚有余热未除。
加减：None
功效：常服镇心压惊，化痰退热安神，可除未来之患，通关。
用法用量：遇有此证，服1匙，麦门冬汤调下；气喘咳嗽，桑白皮汤调下；常服薄荷汤下。
制备方法：上为末，以瓷器收之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
