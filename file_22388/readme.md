序号：22396
名称：阴丹
组成：炉甘石1两，铜青1钱9分，硇砂6分2厘半，没药2分，青盐3分7厘半，乳香3分7厘半，熊胆1分2厘半，密陀僧2分半，以上8味，用黄连5钱、龙胆草2钱半，煎汁滤净，将前药和1处，入汁（石禹？？）细嫩，晒干，再（石禹？？）极细用之。白丁香1分7厘半，海螵蛸1分7厘半，白矾（生）1分7厘半，轻粉1分7厘半，硼砂2分半，雄黄1分2厘半，牙消1分2厘半，黄丹1分2厘半，血竭1分2厘半，朱砂1分2厘半，铅白霜7厘半，粉霜7厘半，鹰条7厘半，胆矾7厘半（一方有黄连6分2厘，胡连1分2厘半，脑1分2厘半，细辛1分2厘半，姜粉1分2厘半，草乌1分2厘半；一方有石蟹2分，贝齿2分，玄明粉2分，真珠2分，琥珀2分）。
出处：《准绳·类方》卷七。
主治：翳膜遮睛，血灌瞳仁，拳毛胬肉，烂弦风眼。
加减：None
功效：None
用法用量：如有翳膜，配合阳丹、一九、二八、三七、四六等丹点眼。
制备方法：上药各为细末，依方称合，和匀，硬令无声至千万余下，瓷器收贮听用。
临床应用：None
各家论述：None
用药禁忌：None
附注：黄连、胡连、脑荷、细辛、姜粉、草乌等六味并无去翳之功，不用更妙，恐有碍眼作痛害眼之祸也。石蟹、贝齿、玄明粉；真珠、琥珀等五味或多或少，皆可增入，以有磨翳消膜之功，不可或缺也。
