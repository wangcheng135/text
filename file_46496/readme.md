序号：46511
名称：千金丸
组成：雄黄（研）半两，鬼臼（去毛，炙）半两，徐长卿（炒）半两，礜石（煅）半两，雌黄（研）半两，干姜（炮）半两，蜀椒（去目及闭口，炒出汗）半两，地胆（去翅足，炒）8枚，野葛3分，斑蝥（去翅足，炒）10枚，射罔1分（一方加丹参半两，瓜蒂4枚）。
出处：《圣济总录》卷一○○。
主治：鬼注入腹，面目青黑，不知人，及心腹坚积结聚，胸胁逆满，呕吐，宿食不消。
加减：None
功效：None
用法用量：每服1丸，空心米饮送下，1日3次。不知，加丸数，以知为度。若百毒所螫，牛马踏伤，痈肿瘰疬，用1丸于掌中，津唾和涂痛处。岁旦以椒酒，长幼各服1丸，终岁无病。
制备方法：上为末，炼蜜为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
