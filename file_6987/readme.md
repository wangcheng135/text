序号：6987
名称：四灵丹
组成：硼砂3两，水银1两，朱砂1两，硫黄3两。
出处：《圣惠》卷九十五。
主治：筋有风，角弓风，肾脏风，热毒风，皮肤风，大风，感厥风。
加减：None
功效：None
用法用量：每日3丸，空心以温酒送下。10日后加至5丸。
制备方法：上件药，将硼砂、硫黄同研如面，于瓷盒中盛之，如法固济，令干，入灰炉中，其上灰厚3寸，以火3两，养7日，开取药，入水银、朱砂各1分同研，以水银星尽为度，依前入盒，养7日，如此4度，计28日开取，细研，以水飞过，入竹筒中，密封头，于饭上蒸两炊久，及热取出，于地上以纸衬，盆合1周时，出火毒了，用粟米饭为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：忌羊血。
附注：None
