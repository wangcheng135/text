序号：14145
名称：列节浸酒
组成：列节2两，防风1两（去芦头），茵芋1两，黄耆2两，羌活2两，桂心2两，海桐皮2两，虎胫骨2两（涂酥，炙微黄），牛膝2两（去苗），附子2两（炮裂，去皮脐），生干地黄1两，芎窮1两，当归1两，枸杞子1两，白芷1两，败龟1两（涂酥，炙微黄），黑豆半合（炒熟），五加皮1两，酸枣仁1两。
出处：《圣惠》卷二十五。
主治：风。
加减：None
功效：None
用法用量：每日空心、午时及夜卧热暖1小盏饮之，其酒旋旋更添。
制备方法：上锉细和匀，以生绢袋盛，用好酒3斗，密封瓶头，浸经7日后开取。
临床应用：None
各家论述：None
用药禁忌：忌生冷、猪、鸡、牛、马肉。
附注：None
