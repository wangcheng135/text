序号：57009
名称：消毒救苦散
组成：防风5分，羌活5分，麻黄根5分，升麻5分，生地黄5分，连翘5分，酒黄柏5分，当归身3分，黄连3分，川芎3分，藁本3分，柴胡3分，葛根3分，酒黄芩3分，生黄芩3分，苍术3分，细辛1分，生甘草1分，白术1分，陈皮1分，苏木1分，红花1分，吴茱萸半分。
出处：《兰室秘藏》卷下。
主治：小儿斑疹，未满21日而目疾作者，生翳羞明，眵泪俱多，红赤肿闭。
加减：None
功效：斑证悉具，消化便令不出，如已出稀者，再不生斑。
用法用量：每服5钱，水2盏，煎至1盏，去滓，稍热空心服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：消毒救苦汤（《东垣试效方》卷四）。消毒化斑汤（《原机启微》卷下）。
