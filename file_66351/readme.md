序号：66367
名称：鹿血丸
组成：黄柏（去皮，盐酒炒）2两，知母（去毛，酒炒）2两，山茱萸（去核）2两，枸杞子2两5钱，天门冬（去心）2两5钱，麦门冬（去心）2两7钱，熟地黄（酒洗）2两，生地黄（酒洗）2两5钱，人参2两，龟版（酥炙）3两，白茯苓（去皮）2两，川萆薢2两，山药2两5钱，五味子（去梗）1两3钱，当归身（酒洗）2两5钱，泽泻（去毛）1两2钱，牡丹皮1两，牛膝（去芦，酒洗）2两。
出处：《摄生众妙方》卷二。
主治：虚损。
加减：None
功效：None
用法用量：每服90丸，渐加至100-150丸，空心用滚水送下。
制备方法：上为细末，即杀鹿取血，加酒2-3盏，入药末内，和成丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
