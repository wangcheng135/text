序号：51508
名称：追风饼
组成：防风（去芦）1两，羌活1两，海桐皮1两，威灵仙1两，石膏（生用）1两，抚芎1两，细辛1两半，苍术1两半，草乌头（切，同苍术用盐1撮炒盐令黑）3两，天花粉3两，藁本半两，蔓荆子半两，萆薢半两，藿香叶（去土）半两，白芷半两，官桂3钱（去粗皮，不见火）。
出处：《魏氏家藏方》卷一。
主治：三十六种风，左瘫右痪，手足不随，面口斜，偏正头风。
加减：None
功效：None
用法用量：每服1饼，温酒或茶清嚼下；偏正头风，薄荷茶嚼下，半饥时服；头风食后服。
制备方法：上为细末，蒸饼糊和为饼子，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
