序号：50848
名称：草灵丹
组成：真川椒4两（去子，炒出汗），白茯苓1两（去皮，炒），川乌1两（去皮脐），茴香2两（盐炒），苍术4两（酒浸，焙干），甘草2两（粉者，去皮，炙），熟地3两（酒浸），山药3两。
出处：《遵生八笺》卷十七。
主治：老人阳气偏虚，便溺不禁，脾胃两亏，内挟寒湿。
加减：None
功效：延年益寿，添精补髓，乌须发，固齿牙，强筋骨，壮气血，聪耳明目，返老还童。
用法用量：每服30-50丸，空心温酒送下，以干物压之。服之1月，乃见其效。冬月服之，手足腮面如噀红，行步轻飞，70老人，健若少年。老年人服之10日，便不夜起。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌黑羊肉、鹁、鸽、桃李果子，恐减药力；服药者，不可赖此频行房事。
附注：None
