序号：78994
名称：增损续命长理石汤
组成：长理石（碎）半两，石膏（碎）半两，赤石脂（碎）半两，白石脂（碎）半两，滑石（碎）半两，桂（去粗皮）半两，大黄（锉，炒令香）半两，麻黄（去根、节，先煎，掠去沫，焙）半两，防风（去叉）半两，牡蛎（烧碎）半两，龙骨（碎）半两，栝楼根（锉）半两，甘草（炙）1两，寒水石1分。
出处：《圣济总录》卷一七四。
主治：小儿卒中风，状如欲绝。
加减：若有热者，加大黄；不汗者，加麻黄。
功效：None
用法用量：1-2岁儿每服1钱匕，水半盏，煎至3分，去滓温服，日3次。服后汗出，以粉敷之。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
