序号：70068
名称：滋阴百补丸
组成：熟地5两，杜仲3两，牛膝3两，枸杞子3两，当归2两5钱，茯苓2两5钱，山萸肉2两5钱，鹿角胶2两5钱，人参2两，黄耆2两，白术2两，白芍2两，肉苁蓉2两，龟板胶2两，锁阳1两5钱，知母1两5钱，黄柏1两5钱，肉桂1两。
出处：《活人书》卷三。
主治：脏腑不和，营卫不调，精神不足，气血不充，以致形衰色萎，骨软筋枯，腰膝痠痛，步力艰难，饮食减少，嗜卧懒言，皮寒内热，精寒阳痿。
加减：None
功效：None
用法用量：每服4-5钱，早晨空心白汤吞服。
制备方法：共研细末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
