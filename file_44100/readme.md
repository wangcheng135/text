序号：44115
名称：复目汤
组成：当归2钱，赤芍2钱，大熟地5钱（或用生地），黄芩1钱5分（酒炒），薄荷2钱，甘菊2钱，甘草5分，川芎1钱。
出处：《外科证治全书》卷一。
主治：目病。目痒肿痛，翳膜，目珠夜痛，目中赤脉，及肝虚泪多等。
加减：目痒，加蝉退、防风各1钱5分；目肿，加羌活、木通各1钱；目痛，加酒连5分，山栀仁1钱；目生翳膜，加木贼、刺蒺藜各2钱，柴胡1钱或8分；伤酒，加葛根2-3钱；目珠夜痛，加夏枯草、香附各2钱；目中赤脉，加密蒙花2钱；白眼上红不退，加桑白皮3钱；两瞳痛，去川芎，加泽泻1钱5分，盐炒黄柏1钱；肝虚泪多，加鲜首乌5-7钱或1两；红肿而不羞明者，加山萸肉、杜仲各2钱；肿痛、两眼如桃，合而为一、痛不可忍者，先用防风通圣散下之，然后服此方加连翘、蔓荆子各1钱5分。
功效：None
用法用量：上水煎，食后、卧时稍热服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
