序号：48886
名称：种子兜肚丸
组成：附子1个（重2两，切片，烧酒煎过，晒干），大茴（炒）1两，小茴（炒）1两，丁香1两，五味子1两，升麻4钱，木香4钱，甘草4钱，甘遂4钱，沉香1钱。
出处：《内外科百病验方大全》。
主治：赤白带下，腰腿痠痛，子宫寒；男子肚腹畏寒、遗精、白浊、偏坠、疝气，一切下部虚寒。
加减：None
功效：调经种子。
用法用量：上为末，用新蕲艾4两，搓融晒干，将前药放在艾中间，用线密缝兜肚，置丹田上，外用手帕包固，昼夜缚定，不可换动，1-2月后则去之。或加麝香2-3分更妙。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
