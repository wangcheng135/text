序号：31568
名称：来苏散
组成：黄耆3两（酒炒），阿胶3两（粉炒），白芍1两半（酒炒），神曲2两（炒炭），炙草1两，麦芽2两（炒炭），木香1两，糯米1合，生姜3片。
出处：《医略六书》卷二十九。
主治：临产胃虚挟滞，困乏，脉软涩者。
加减：None
功效：None
用法用量：每服4钱，水煎，去滓温服
制备方法：上为散。
临床应用：None
各家论述：临产坐草过劳，胃虚元气困乏，挟食滞而胸腹疼痛，无力送胎以分娩焉，黄耆补胃气之困乏，阿胶补阴血之虚衰，白芍敛阴和血脉，木香调气醒脾胃，神曲以消其食，麦芽以化其滞，糯米养脾益胃，炙草益胃缓中，稍佐生姜以温胃气也。为散水煎，使食滞消化，则胃气完复，而胸腹融和，安有疼痛之患？气力自然涌出，临产应无不顺之虞矣。
用药禁忌：None
附注：None
