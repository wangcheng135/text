序号：18681
名称：玉屑无忧散
组成：玄参（去芦）1两，荆芥穗1两，滑石（研）1两，黄连（去毛）1两，缩砂（去壳）1两，白茯苓（炒令黄）1两，贯众（去芦）1两，甘草（炙）1两，山豆根1两，寒水石（研、飞）2两，硼砂2钱。
出处：《局方》卷七。
主治：咽喉肿痛，舌颊生疮，风毒壅塞，热盛喉闭；或因误吞硬物，诸骨硬刺，涎满气急，或至闷乱，不省人事；小儿一切咽喉塞滞，口内疮；心腹胀满，脾积癥块；并蛇蝎诸虫咬，气入腹；但是心腹有疾，诸药不能治者；及湿痰风闭。
加减：None
功效：大解百药毒，偏润三焦，消五谷，除九虫，赶瘟疫。
用法用量：每服1钱，干掺舌上，后以新水咽下，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：此治实火，实痰之重剂，若虚火聚于咽喉，闭结不通者，万不可用。
附注：大圣夺命玉雪无忧散（《幼幼新书》卷三十四）、无忧散（《鸡峰》卷二十四）、大圣玉屑无忧散（《卫生总微》卷十七）、夺命无忧散（《普济方》卷六十引《如宜方》）。
