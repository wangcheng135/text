序号：74270
名称：川乌丸
组成：没药半两（为细末），川乌4两（去皮尖，切作片子，分两处，将一处用齑汁于砂石器内煮腐，取腐和后药，不用汁。如无川乌，草乌头亦得），麝香1钱（别研）。
出处：《普济方》卷一一六引《卫生家宝》。
主治：一切风疾，并脚气，腰痛，妇人赤白带下。
加减：None
功效：None
用法用量：每服1丸至2丸，如风疾，食后茶、酒任下；脚气，木瓜煎酒下；腰痛，核桃酒送下；头风，嚼生葱茶送下；妇人赤白带下，艾醋汤送下；软风，金银花煎酒送下；早晚进2服。
制备方法：上为末，杵烂川乌为丸，如鸡头子大，以朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
