序号：27902
名称：诃黎勒丸
组成：诃黎勒皮4两，五倍子4两，黄连（去须）4两，钟乳粉4两，白矾（熬令汁枯）4两，缩砂蜜（去皮）2两半，厚朴（去粗皮，生姜汁炙）2两半，当归（切，焙）2两半，酸石榴皮（锉，炒）2两半，胡椒2两半，樗根皮（锉）2两半，干姜（炮）1两半，干木瓜（锉）5两，乌梅肉（炒）5两，橡实（微炒）5两，肉豆蔻（煨，去壳）5枚，陈橘皮（汤浸，去白，焙）2两。
出处：《圣济总录》卷三十八。
主治：脾胃虚冷，呕逆霍乱，泻痢胀满。
加减：None
功效：None
用法用量：每服50丸，空心、食前以大枣汤送下，1日2次。
制备方法：上为末，醋并艾叶汁煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
