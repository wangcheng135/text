序号：41072
名称：金刀如圣散
组成：茅山苍术6两（米泔浸1日1夜），白芷2两，川乌4两（去皮脐，生用），防风4两（净，生用），细辛3两（去土，净），白术2两5钱，草乌4两，两头尖（无则以川芎）4两，雄黄5两（另研细末入药）。
出处：《伤寒标本》卷下。
主治：金疮、破伤风，蛇、蝎、疯狗咬伤，汤火烫伤，雷头风，干湿癣麻痹。
加减：None
功效：None
用法用量：一切金疮及多年恶疮，用自己小便洗过，贴药立效；破伤风紧急，用好酒调药半钱或1钱服之；蛇伤，入枯白矾少许调药敷之；蝎蜇伤，用吐服盖之汗也。如汗不出再服，或涎出亦验，伤处敷药也；疯狗咬伤处口噙水洗净，将药贴伤处；汤火烫伤者，以新汲水调药敷之；雷头风并干湿癣，麻痹，每服半钱，服之立效。
制备方法：上药俱各生用，晒干为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
