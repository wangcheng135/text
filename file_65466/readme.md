名称：羚羊角散
组成：羚羊角3分（镑），白鲜皮3分，黄耆3分（锉），白槟榔3分（煨，锉），山栀子仁3分，羌活1两（去芦头），甘草1两（炙，锉），恶实1两（炒），茯神1两（去木），桂1两（去粗皮），海桐皮1两（锉），附子1两（炮裂，去皮脐），郁李仁1两（炒，去皮），大黄1两（锉，醋炒），麻黄1两（去根节，汤煮，掠去沫，焙），酸枣仁1两（炒），独活1两（去芦头），芎1两，防风（去叉）1两，葛根3分（取粉），枳壳3分（麸炒，去瓤），地骨皮3分，车前子（炒）3分。
