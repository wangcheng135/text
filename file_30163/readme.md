序号：30171
名称：连翘败毒膏
组成：连翘16两，桔梗12两，甘草12两，木通12两，金银花16两，防风12两，玄参12两，白藓皮12两，黄芩12两，浙贝母12两，地丁12两，白芷12两，天花粉8两，赤芍12两，蝉蜕8两，大黄16两，蒲公英12两，栀子12两。
出处：《天津市固有成方统一配本》。
主治：诸疮初起，红肿疼痛，疮疖溃烂，灼热流脓，无名肿毒，丹毒疮疹，疥疮癣疮，痛痒不止。
加减：None
功效：None
用法用量：每服1两，日服2次，白开水送服。或制成水丸。名“连翘败毒丸”。每服3钱，日服2次，温开水送服。
制备方法：上药洗净切碎，加水浓煎成清膏，再加炼蜜（每清膏10两，加蜜20两）收膏。
临床应用：None
各家论述：None
用药禁忌：忌食腥荤及刺激性之物，孕妇慎用。
附注：None
