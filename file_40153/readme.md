序号：40163
名称：苦参十二味丸
组成：苦参6两，干姜6两，芎6两，玄参5两，丹参5两，人参5两，沙参5两，白术5两，地骨白皮4两，独活4两，薏苡仁2升，蜀升麻2升。
出处：《外台》卷十四引许仁则方。
主治：风热未退，频服汤饮，力不能及。
加减：若觉冷，即去玄参、沙参，加桂心4两，细辛3两；若觉热，别加10两主地黄；若觉有痛处，去沙参，加当归6两；若觉有气，去玄参，加橘皮4两；若大便涩，加大槟榔仁20枚。
功效：None
用法用量：初服15丸，稍加至30丸，薏苡仁饮送下，1日2次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌桃、李、生葱、生菜、芜荑。
附注：None
