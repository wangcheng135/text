序号：67567
名称：黄芩汤
组成：黄芩（去黑心）半两，柴胡（去苗）半两，地骨皮半两，人参半两，干漆（炒令烟出）半两，鳖甲（去裙襴，醋炙黄）半两，甘草（炙）半两，半夏（汤洗7遍，同生姜捣作饼子，晒干）半两，葛根（锉）半两，干青蒿半两，白茯苓（去黑皮）半两，麦门冬（去心，焙）1分。
出处：《圣济总录》卷八十七。
主治：热劳。心忪肌热，夜有盗汗，面黄肌瘦，饮食减少，骨节酸痛。
加减：None
功效：None
用法用量：每服5钱匕，先用水2盏，加小麦、乌梅、生姜各少许，煎5-7沸，去小麦等，入药末煎至1盏，去滓温服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
