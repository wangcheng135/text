序号：47770
名称：神仙青娥丸
组成：肉苁蓉（洗）2两，川牛膝（洗，去芦）2两，川萆薢2两，川椒（去目）1两，山茱萸（取净）1两，舶上茴香1两（用好酒浸，春夏3日，秋冬6日，滤出，焙干），川楝子（作4片，麸炒）3两，破故纸4两（麸炒），胡芦巴（麸炒）2两，白茯苓（去皮）2两，附子1只（7钱重者，炮，去皮脐）。
出处：《魏氏家藏方》卷入引胡应诚方。
主治：一切虚损，干湿脚气，妇人诸疾血气，一切小肠气，膀胱疝气。
加减：None
功效：延年不老，乌髭，治口齿，活血驻颜，大壮筋骨，补虚损。
用法用量：每服30-50丸，空心盐酒送下；干湿脚气，木瓜汤送下；妇人诸疾血气，煎艾醋汤送下。
制备方法：上为细末，用前浸药酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
