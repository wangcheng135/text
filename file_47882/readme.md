序号：47897
名称：神圣丸
组成：乌蛇头（醋浸1宿，炙黄）1两，蛇黄（火煅，醋淬3遍）1两，白僵蚕（炒，去丝嘴）1两，防风（去芦头）1两，天麻1两，五灵脂半两，代赭石半两（火煅，醋淬3遍），全蝎1钱（去毒，微炒），麝香1分（别研），朱砂1钱（别研）。
出处：《杨氏家藏方》卷十七。
主治：小儿急慢惊风，胸膈涎盛，口眼牵引，手足搐搦。
加减：None
功效：None
用法用量：1岁1丸，分3服，薄荷汤磨下；急惊，用磨刀水磨下；慢惊，煎荆芥汤磨下，不拘时候。
制备方法：上为细末，糯米煮糊为丸，每1两作40丸，金箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
