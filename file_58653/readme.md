序号：58668
名称：秘传枳术二陈汤
组成：白术（消洗，锉，土炒）1钱，黑枳实（麸炒）8分，陈皮（去白）8分，茯苓（去粗皮）1钱，香附子（童便浸，炒）1钱，半夏（汤泡7次）1钱，黄连（姜汁炒），槟榔（鸡心者）5分，白豆蔻5分，青皮（麸炒）3分，吴茱萸3分，生甘草3分。
出处：《直指附遗》卷七。
主治：痰气食膈，呕吐痰涎，翻胃嘈杂。
加减：气虚，加人参、黄耆；血虚，加当归、地黄；郁，加神曲、抚芎。
功效：None
用法用量：上(口父)咀。用水1钟，加生姜3片，大枣1枚，煎8分，食远服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：枳术二陈汤（《医统》卷二十七）。
