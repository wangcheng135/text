序号：3381
名称：太一追命丸
组成：蜈蚣1枚，丹砂1分，附子1分，矾石（1作礜石）1分，雄黄1分，藜芦1分，鬼臼1分，巴豆2分。
出处：《千金》卷二十四。
主治：百病，或中恶气，心腹胀满，不得喘息，心痛积聚，胪胀疝瘕，宿食不消，吐逆呕哕，寒热瘟病，蛊毒，妇人产后余疾。
加减：None
功效：None
用法用量：每服2丸，1日1次。伤寒1-2日，服1丸，当汗出，绵裹2丸，塞两耳中；下利，服1丸，1丸塞下部；蛊毒，服2丸；在外，膏和摩病上。在膈上，吐；膈下，利；有疮，1丸涂之，毒自出；产后余疾，服1丸；耳聋，绵裹塞耳。
制备方法：上为末，炼蜜为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：太乙追命丸、夺命丸(《普济方》卷二五一)。
