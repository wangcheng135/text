序号：70133
名称：滋补大力丸
组成：熟地4两，山药2两，茯苓2两，杞子2两，枣仁2两，萸肉2两，当归2两，冬术2两，杜仲2两，菟丝子2两，龟版2两，虎骨2两，白芍1两，苁蓉1两，补骨脂1两，覆盆子1两，自然铜（醋煅）1两，青盐3钱，乳香3钱，没药3钱，地龙5钱，地鳖虫20个。
出处：《饲鹤亭集方》。
主治：五脏虚衰，劳伤诸损。身体虚弱，腰膝痠软，跌伤血瘀，筋骨疼痛。
加减：None
功效：健脾开胃，强筋壮骨，填精髓，进饮食，肌肉充长，膂力过人。养血生精，强壮筋骨，活瘀止痛。
用法用量：None
制备方法：用大黄鳝1条煮熟，去骨，同熟地打烂，加蜜10两为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：《北京市中药成方选集》本方用法：将鳝鱼一条酒蒸去骨，和群药晒干，共研为细粉过罗，炼蜜为丸，重三钱。每服一丸，日服二次，温开水送下，黄酒亦可。
