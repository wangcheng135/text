序号：26566
名称：八宝回春汤
组成：附子（炮）1两，人参1两，麻黄（去节）1两，黄芩1两，防己1两，香附子（去毛）1两，杏仁（去皮）1两，川芎1两，当归1两，茯神1两半，陈皮1两，防风1两，白芍药5两，沉香半两，半夏1两半，川乌半两（炮），桂1两，白术2两，天台乌药半两，干姜1两，黄耆3两，甘草1两，熟地黄1两，生干地黄1两。
出处：《朱氏集验方》卷一。
主治：男子、妇人一切诸虚不足，风疾血气交攻，凝滞脉络，拘急挛拳，气不升降，瘫中疼痛，痰涎壅盛，脾胃不和，饮食不进。
加减：None
功效：去风，和气，活血。
用法用量：每服3钱，水1盏半，加生姜3片，大枣1枚，煎1盏，空心通口服。合2服，滓再煎。
制备方法：上为末。
临床应用：None
各家论述：方中二十四味药，祛风八味，和气八味，活血八味。故此方不专治风，而且和气；不专和气，而且又活血。血气和平，荣卫调顺，则其风证不攻而自去。
用药禁忌：None
附注：万宝回春汤（《医学入门》卷七）。大凡病风不可专服风药，攻之愈急则愈甚。但专服此药，轻者一月，重者二三月，自然而愈，且无再作。
