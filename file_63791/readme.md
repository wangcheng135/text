序号：63807
名称：清眩平肝汤
组成：当归3钱，川芎1钱半，白芍4钱，生地4钱，桑叶3钱，菊花3钱，黄芩3钱，女贞子3钱，旱莲草3钱，红花3钱，牛膝3钱。
出处：《刘奉五妇科经验》。
主治：妇女更年期综合征、经前期紧张症等，属于肝肾阴虚，肝阳亢盛，见有头晕、头痛（或血压升高），烦躁者。
加减：热重者，去当归、川芎，加马尾连3钱；肝阳亢盛者，加龙齿1两。
功效：滋肾养肝，清热平肝，活血调经。
用法用量：None
制备方法：None
临床应用：None
各家论述：方中当归、川芎、白芍、生地、红花、牛膝养血活血，引血下行以调经；女贞子、旱莲草滋补肝肾以培本；黄芩清肝热；桑叶、菊花清热平肝以治标。本方标本兼顾，使之补肾而不呆滞，清肝热而不伤正。在重用牛膝引血下行的同时，配合黄芩、桑叶、菊花清上引下，重点突出。
用药禁忌：None
附注：None
