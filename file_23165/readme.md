序号：23173
名称：芎散
组成：芎1两，防风3分（去芦头），独活3分，桂心3分，前胡3分（去芦头），甘菊花半两，附子3分（炮裂，去皮脐），麻黄1两（去根节），细辛半两，五味子3分，黄耆半两（锉），杏仁3分（汤浸，去皮尖双仁，麸炒微黄），人参3分（去芦头），茯神3分，山茱萸半两，甘草半两（炙微赤，锉）。
出处：《圣惠》卷六。
主治：肺脏中风，项强头旋，胸满短气，嘘吸颤掉，语声嘶塞，四肢缓弱。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，煎至6分，去滓稍热服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：忌生冷、毒滑、油腻。
附注：None
