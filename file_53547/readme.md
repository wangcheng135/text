序号：53562
名称：哮喘万灵膏
组成：川乌6钱，草乌6钱，连翅8钱，当归6钱，白芷8钱，木鳖8钱，白及6钱，官桂8钱，茯苓6钱，白蔹8钱，牙皂5钱，乌药6钱，桑枝5钱，枣枝5钱，桃枝5钱，柳枝5钱，槐枝5钱。
出处：《外科十三方考》。
主治：多年咳嗽气喘。
加减：None
功效：None
用法用量：用时以布或绫摊，贴肺俞穴（第3胸椎之下，旁开1寸5分），于初伏之日贴起，贴满3伏；再于冬至之日贴起，贴至九九。
制备方法：上药同麻油3斤先浸1宿，然后熬焦去滓，入飞黄丹1斤再熬至如漆色时，急以桃、柳棍2根搅至滴水成珠时，入乳香4钱，没药4钱，收膏备用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
