序号：59259
名称：脏连固本丸
组成：怀生地6两，干山药4两，茯苓3两（去皮），牡丹皮3两，泽泻2两，山茱萸4两（去核），黄连4两，黄柏3两，知母（去毛）2两，人参2两，当归2两，皂角2两，槐角3两，天花粉2两。
出处：《回春》卷四。
主治：膏粱富贵之人，饮食色欲，火酒犯房所致痔疾。
加减：None
功效：None
用法用量：每服100丸，空心以白汤送下。
制备方法：上为末，用豮猪大肠头1段去油，灌入药末，两头线扎住；用糯米1升煮饭，将半熟捞起入甑内，将药肠盘藏于饭之中如蒸饭之熟；待冷取出，去两头无药之肠，将药肠捣烂为丸，如硬，加饭为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：戒醇酒厚味。寡欲。
附注：None
