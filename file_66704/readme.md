序号：66720
名称：鹿茸丸
组成：附子1两（炮，去皮脐），鹿茸2两（去毛，涂酥炙微黄），苁蓉1两（酒浸1宿，去皮，炙干），巴戟1两（去心），防风3分（去芦），当归1两，羌活3分，桂心3分（去皮），萆薢3分（锉），酸枣仁3分（微炒），牛膝1两（去苗），木香3分，白蒺藜3分（去刺，微炒），石斛1两（去根，锉），补骨脂1两（微炒），白茯苓1两（去皮），桃仁1两（汤浸，去皮尖双仁，麸炒微黄）。
出处：《博济》卷一。
主治：虚劳伤惫，腰脚疼痛少力，精神不爽，饮食减退。
加减：None
功效：驻颜益气。
用法用量：每服20丸，空心及晚食前以温酒送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
