序号：4196
名称：木香丸
组成：木香（为末）半两，丁香（为末）半两，巴豆（去皮心膜，研出油）半两，硇砂（研）半两，大枣（去皮核）30枚，乌梅（去核，为末）30枚。
出处：《圣济总录》卷七十二。
主治：积聚凝滞，脏腑刺痛，饮食减少。
加减：None
功效：消食化气，利胸膈。
用法用量：每服2丸3丸，食后生姜汤送下；或随所伤物汤下。
制备方法：上6味，先将水拌白面，作一薄饼，以枣肉铺饼上，次以前4味药末和匀，复上铺枣肉，作馒头，裹就，用炭火四围炙烤，候面焦黑，约药透取出，地面上出火毒，候冷打破，去焦面不用，将药与乌梅末同捣，稀面糊丸如黄米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
