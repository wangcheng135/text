序号：27508
名称：补旧汤
组成：苦参皮1钱5分，牛蒡子1钱，人参1钱，首乌1钱，山栀1钱，僵蚕5分，白鲜皮5分，防风5分，连翘5分，天麻5分，蔓荆子5分，黄芩5分，全蝎4分，黄连4分，甘草4分，薄荷3分，羌活3分，独活3分，荆芥3分，干葛7分，黄柏7分，威灵仙8分，蒺藜8分。
出处：《解围元薮》卷三。
主治：壁泥风及血痹血风。
加减：怯弱者，加仙灵脾5分。3年身固绝根，再以每味加10倍为末，再加乳香、没药、血竭、沉香各1两，冰片、牛黄各1钱，麝香2钱，用米仁糊为丸，如梧桐子大，朱砂为衣。每服100丸，酒送下，临服时，加威灵仙末3分效速。
功效：None
用法用量：先用酒煎10帖服，再用水煎，须尽量饮酒。服之100帖，其眉须复生，肿块渐退，手足痿顿者有力。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：须戒色、省劳、避风、忌口，方能有功。
附注：None
