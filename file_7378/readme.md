序号：7378
名称：圣效散
组成：黄芩2两，北细辛2两，甘草2两，熟地黄2两，大黄2两，山栀子2两，赤芍药2两，当归尾（极细者）2两，牛蒡子2两，桑白皮2两（有翳膜加1两用），菊花5两（去梗）。
出处：《得效》卷十六。
主治：诸般风热，风痒热毒；大人小儿生翳、生膜、生血筋，属外障热眼者。
加减：None
功效：None
用法用量：每服4钱，水1盏半，煎8分，去滓，临卧温冷服；两滓再煎，早食、午食、晚食、临卧各1服。
制备方法：上锉散。
临床应用：None
各家论述：None
用药禁忌：内障是虚，不宜服。每日只白煮精猪肉咽饭，或山药、萝卜菜，更可吃枣子、柿子、银杏、土瓜、榧子、生葛。忌鸡、鱼、酒、面、糯米、咸、酸、热油诸般毒物。
附注：None
