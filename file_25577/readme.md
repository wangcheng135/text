序号：25585
名称：朱砂丸
组成：朱砂1两（细研，水飞过），龙脑1分（细研），牛黄1分（细研），龙齿1两（细研），天竺黄1两（细研），金箔100片（细研），银箔1百片（细研），虎晴3对（酒浸1宿，微炙），马牙消1两，麝香1分（细研），犀角屑1两，人参1两（去芦头），茯神1两，川升麻1两，羚羊角屑1两，天麻1两，麦门冬1两（去心，焙），独活1两，甘菊花1两，子芩1两，甘草半两（炙微赤，锉）。
出处：《圣惠》卷四。
主治：心脏风热，多惊恍惚，烦躁语涩。
加减：None
功效：None
用法用量：每服10丸，以荆芥汤送下。不拘时候。
制备方法：上为末，入研了药令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
