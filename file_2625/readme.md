序号：2625
名称：天一丸
组成：灯心10斤（以米粉浆染，晒干研末，入水澄之，浮者为灯心，取出，又晒干入药，用2两5钱。而沉者为米粉，不用），赤白茯苓（去皮，兼用茯神去木）5两，滑石（水飞过）5两，猪苓（去皮）2两，泽泻（去须）3两，人参1斤（去芦，切片，煎浓汤，去滓漉净。炼汤成膏，如糖饴）。
出处：《韩氏医通》卷下。
主治：小儿元气不足，心脾蕴热，心烦口渴，呕吐泄泻，小便不利，丹毒。孕妇难产不下者；痰胀，腹胀喘肿已减者；小儿脾胃不和。
加减：None
功效：清心利水，健脾止泄，通利水道，散火。
用法用量：每用1丸，任病换引。
制备方法：上药灯心等5味，各为细末，以人参膏和成丸，如龙眼大，朱砂为衣，贴金箔。
临床应用：None
各家论述：None
用药禁忌：None
附注：天乙丸(《医学入门》卷八)、天一水串(《串雅内编》卷三)。
