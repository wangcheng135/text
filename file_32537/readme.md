序号：32546
名称：羌活散
组成：羌活（去芦头）1两，白术1两，黄耆（锉）1两，青橘皮（汤浸，去白，炒）1两，桔梗（炒）1两，甘草（炙）1两，附子（炮裂，去皮脐）1两，五加皮（用茱萸半两，水1碗，煎水尽，焙干，去茱萸）1两，桂（去粗皮）半两，干姜（炮）半两。
出处：《圣济总录》卷三十一。
主治：伤寒后夹劳，肢体烦疼，早晚虚热，口苦嗌干，夜卧多汗，脚手麻痹，及风劳。
加减：None
功效：None
用法用量：每服2钱匕，温酒调下；用水1盏，加生姜3片，大枣1枚（擘），同煎至7分，温服亦得，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
