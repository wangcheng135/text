序号：16761
名称：平补镇心丹
组成：熟干地黄4两（一本7两），生干地黄4两（一本7两），干山药4两（一本7两），天门冬4两（一本7两），麦门冬（去心）4两（一本7两），柏子仁4两（一本7两），茯神4两（一本7两），辰砂（别研为衣）3两，苦梗（炒）3两，石菖蒲（节密者）16两，远志（去心，以甘草煮3-4沸）7两，当归（去芦）6两，龙骨1两。
出处：《局方》卷五（续添诸局经验秘方）。
主治：丈夫、妇人心气不足，志意不定，神情恍惚，夜多异梦，忪悸烦郁，及肾气伤败，血少气多，四肢倦怠，足胫痠痛，睡卧不稳，梦寐遗精，时有白浊，渐至羸瘦。
加减：None
功效：益精髓，养气血，明视听，悦色驻颜。
用法用量：每服30丸，空心饭饮吞下，温酒亦得，渐加至50丸。宜常服。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
