序号：81225
名称：麝香牛黄丸
组成：牛黄1钱，麝香1钱（研），蝎梢1钱（生），乳香1钱（研），没药1钱（研），蜈蚣1条（酥炙），花蛇肉（去骨）1钱（酒浸1宿），天南星2钱半（炮），草乌（炮去皮）2钱半，天麻2钱半（锉，研），防风（去苗并叉枝）2钱半，雄黄半钱（研、水飞），朱砂8钱（研）。
出处：《卫生总微》卷五。
主治：小儿急慢惊风，慢脾风。
加减：None
功效：None
用法用量：1-3岁儿，每服1丸，研细，煎金银薄荷汤调下；7-10岁，服2-3丸，煎人参汤调下；慢惊风，细研3丸灌下。不拘时候。
制备方法：上为细末，酒糊和剂，如皂子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
