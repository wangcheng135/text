序号：68718
名称：小灵丹
组成：代赭石4两，赤石脂4两，紫石英4两，禹余粮石4两。
出处：《杨氏家藏方》卷十四。
主治：真元虚损，精髓耗惫，本气不足，面黑耳焦，腰膝沉重，膀胱疝瘕，手足麻痹，筋骨拘挛，心腹疠痛，冷积泻利，肠风痔漏，八风五痹，头目昏眩，饮食不进，精神恍惚，疲倦多睡，渐成劳疾，妇人胎脏久冷，绝孕无子，赤白带下，月经不调，风冷血气。
加减：None
功效：助养真气，补暖丹田，活血驻颜，健骨轻身。
用法用量：每服1丸，空心温酒或新溪水送下。
制备方法：上药各用火煅赤，入米醋中淬，各7遍，同碾为细末，入1砂盒子内合了，外用盐泥固济，日中晒干，用炭20斤，顶火煅，以炭火尽力度，取出药盒，于润地上掘坑，埋一伏时取出，研3日令极细，次入乳香（别研）2两、没药（别研）2两、五灵脂（研细）2两。同前4味，一处研令极匀。水煮糯米饼子和得所，入铁臼中捣为丸，如鸡头子大，阴干。
临床应用：None
各家论述：None
用药禁忌：孕妇不可服。
附注：None
