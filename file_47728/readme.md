名称：神仙快活丸
组成：桂花2钱，木香1分，丁香1分，青皮1分，陈皮1分，官桂2分，荜澄茄1分，肉果1分，砂仁4钱，良姜5分，白果1钱，白芷2分，甘松4分，广三棱4分，檀香1分，沉香1分，茯苓6分，香附子5分，麝香1分，益智6分，大椒10个，红豆4个，藿香1两。
出处：《普济方》卷二十五。
主治：脾胃不和，气不升降，腹胀肠鸣，反胃吐食，呕吐酸水，不思饮食，心腹痞闷，水谷不消，渐成泄痢，酒食所伤，小儿奶癖。
