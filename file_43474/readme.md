序号：43489
名称：冠心通络丸
组成：丹参20g，旋覆花10g，杏仁10g，茯苓10g，茜草10g，干地龙10g，薤白10g，法半夏10g，山楂炭10g，五灵脂10g，生蒲黄15g，陈皮5g，建菖蒲5g，远志肉5g，琥珀末3g，甘草3g。
出处：《古今名方》引谭曰强经验方。
主治：冠状动脉硬化，心肌供血不足，胸闷气短，心悸心痛等。
加减：None
功效：活血通络，理气宽胸，宣痹止痛，定悸安神。
用法用量：每次10丸，每日3次。
制备方法：将丹参、蒲黄、菖蒲、远志、茯苓研细末，过筛，余药水煎2次，去滓过滤浓缩混合，制小丸或制片。
临床应用：None
各家论述：None
用药禁忌：孕妇忌用。
附注：None
