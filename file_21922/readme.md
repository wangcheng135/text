序号：21930
名称：何仙姑庆世丹
组成：枸杞子，菊花（去萼用）、远志（须用硬物捶破，去心）、车前子、巴戟、生地黄（用干者，去芦头）、覆盆子、白术、苁蓉（用有肉者，酒浸7日）、菖蒲（细小九节者）、牛膝（去芦头，酒浸7日）、地骨皮、菟丝子（酒浸7日，昼夜晒干，炒令黄色为度）、续断、细辛（去苗用）、何首乌（上各用本土所生者）各等分。
出处：《普济方》卷二二三引《卫生家宝》。
主治：一切危疾，及瘫痪痛楚，久在床褥，或五脏不安，四肢少力，口干气虚，神乱，骨节疼痛，毛发焦枯，或有恶疾，居体不安，行履艰难，饮食不进，或寝寐不安，或痛连筋骨。
加减：None
功效：还精定魂，安五脏，和六腑，添智慧，去邪，乌发黑髭，驻颜色，长肌肤，聪耳明目，强健四体，延年益智，服1-2年返老还童。
用法用量：每服20-30丸，空心、食前温酒下。
制备方法：上逐药择洗，为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
