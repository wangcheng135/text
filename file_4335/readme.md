序号：4335
名称：九味柴胡汤
组成：柴胡1钱，黄芩（炒）1钱，人参5分，山栀（炒）5分，半夏5分，龙胆草（炒焦）5分，当归5分，芍药5分，甘草5分。
出处：《校注妇人良方》卷二十四。
主治：肝经湿热下注，便毒肿痛，或小腹胁肋结核；肝胆经一切疮疡或风热结核瘰疬，阴痛，寒热，脉数洪涩。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：《医略六书》：湿热郁于厥阴营气有伤于肝胆、故寒热往来，阴部肿痛焉。龙胆草清热泻湿以清经脉；柴胡梢升阳散郁以清肝胆；人参扶元补气，兼通血脉：当归养血荣筋，善利经脉；黄芩清热燥湿：；半夏燥湿醒脾；山栀降湿火以屈曲下行；炙草缓中气以调和脾胃也，水煎温服，使湿热化而营气调，则寒热解而阴部之肿痛无不并除矣。
用药禁忌：None
附注：None
