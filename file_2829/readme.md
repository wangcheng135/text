序号：2829
名称：天竺饮子
组成：天竺黄5钱，川郁金（用皂角水煮，切作片，焙干）20两，甘草（炙）20两，大桅子仁（微炒）20两，连翘20两，雄黄（飞研）5两，瓜蒌根10斤。
出处：《局方》卷十。
主治：大人、小儿脏腑积热，烦躁多渴，舌颊生疮，咽喉肿痛，面热口干，目赤鼻衄，丹瘤结核，痈疮肿痛；又治伏暑燥热，疮疹余毒，及大便下血，小便赤涩。
加减：None
功效：None
用法用量：每服1大钱，小儿半钱，食后、临卧用新水调服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：天竺散(《永类钤方》卷十一)、天丝黄散(《得效》卷八)。方中天竺黄原脱，据《得效》补。
