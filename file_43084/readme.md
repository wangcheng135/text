序号：43099
名称：养正通幽汤
组成：川芎1钱或1钱5分或2钱，当归4钱或6钱，桃仁10粒（去皮尖），炙草5分，陈皮4分，麻仁1钱或1钱5分（炒），肉苁蓉1钱或2钱（酒洗，去泥甲）（1方无苁蓉）。
出处：《胎产心法》卷下。
主治：产后大便秘结。
加减：如有血块痛，加肉桂、炒延胡索各5分，不用苁蓉；如气虚多汗，加蜜炙黄耆1钱，人参1-2钱（1方再加麻黄根1钱）；燥渴，加去心麦冬1钱5分，人参1-2钱（1方再加五味子7粒）；腹满便实，加去心麦冬1钱，麸炒枳壳6分；汗出谵语便实，乃气血并竭神衰，心主失守，宜养气安神，加茯神、炒枣仁、制远志肉、炙黄耆、柏仁、苁蓉各1钱，人参、土炒白术各2钱。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
