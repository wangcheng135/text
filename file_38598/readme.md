序号：38608
名称：法制黑豆
组成：何首乌（炙）2两，远志（炙）2两，旱莲草2两，山萸肉（炙）2两，苣胜子2两，生地2两，黑芝麻2两，川芎2两，楮实子2两，茯苓2两，肉苁蓉（炙）2两，补骨脂（炒）2两，巴戟（炙）2两，菊花2两，节菖蒲2两，川花椒2两，覆盆子2两，菟丝子2两，蛇床子2两，女贞子（炙）2两，熟地3两，地骨皮3两，五味子（炙）3两，当归3两，枸杞子4两，桑椹4两，食盐16两，青盐8两。
出处：《北京市中药成方选集》。
主治：肾水不足，精神衰弱，身体羸瘦，梦遗滑精，腰腿痠痛。
加减：None
功效：补肾益精，强筋壮骨。
用法用量：每次用3-5钱，口内嚼咽。
制备方法：除食盐、青盐外，其余26味，水煎3次，过滤去滓，入雄黑豆800两，煮至八成熟，加入食盐、青盐同煎，熬至汤尽，取出晒干即得。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
