序号：73868
名称：硼砂煎丸
组成：硼砂3两（不夹石者，细研，以酒1升，醋1升，慢火熬令如膏），附子1两（炮裂，去皮脐），吴茱萸半两（汤浸7遍，焙干微炒），木香3分，桃仁1两（汤浸，去皮尖双仁，麸炒微黄，研入），防葵3分（锉碎，醋拌令黄），槟榔3分。
出处：《圣惠》卷四十八。
主治：奔豚气在小腹，积聚成块，发歇痛。
加减：None
功效：None
用法用量：每服15丸，食前以温酒送下。
制备方法：上为细末，入桃仁令匀，纳硼砂煎中，再入少蒸饼和溶为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“硇砂煎丸”。方中硼砂，《普济方》引作“硇砂”。
