名称：神效活络丹
组成：虎胫骨3钱，胆星8钱，防风6钱，半夏6钱，羌活6钱，川芎6钱，全蝎6钱，广红6钱，苍术6钱，川贝6钱，白附子6钱，独活6钱，桂枝6钱，当归6钱，乌药6钱，香附6钱，茯神6钱，石菖蒲6钱，麻黄2两4钱，牛黄1钱7分，沉香4钱6分，川附子3钱2分，钩藤1两，白芷1两，牛膝1两，天麻1钱6分，麝香1钱，冰片1钱2分，苏合油1两，僵蚕1两。
出处：《慈禧光绪医方选义》。
主治：风湿诸痹，肩臂腰膝筋骨疼痛，口眼歪斜，半身不遂，行步艰难，筋脉拘挛。
加减：None
功效：舒肝活血，除湿化痰。
用法用量：每服1丸，用温酒送下，白开水、茶清送亦可。
制备方法：上为末，炼蜜为丸，蜡皮封固，每丸重2钱。
临床应用：None
各家论述：None
用药禁忌：None
