序号：70994
名称：小续命丸
组成：汉防己1两，桂（去粗皮）1两，人参（去芦）1两，杏仁（去皮尖，炒赤）1两，白芍药1两，甘草1两，麻黄（去根节）1两，芎各1两（净），防风（去芦）1两半，附子（去皮脐）半两（人壮实不可用），何首乌4两，川牛膝1两，草乌头（去皮尖，切片，炒黄）半两，黄芩（去芦）。
出处：《医方类聚》卷二十三引《经验良方》。
主治：男子妇人久近风湿脚气。
加减：None
功效：None
用法用量：上为细末，先用大木瓜2个，好酒1瓶，小锅炭火慢煮10分烂，去皮瓤，于瓦盆内研烂，搜和前药，如硬燥，就用煮木瓜酒添，搜和为丸，如梧桐子大。每服30丸至50丸，空心温酒送下。
制备方法：None
临床应用：风湿脚气泰州苏教授患风湿穿心脚气，遍身浮肿，气喘；本州太守女患此疾，得一至人，授方服之愈，此方传与教授，服亦愈。
各家论述：None
用药禁忌：None
附注：按：方中黄芩用量原缺。
