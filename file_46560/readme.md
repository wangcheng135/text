序号：46575
名称：千金肥儿丸
组成：白术半斤，真茅山苍术半斤，陈皮1斤（不去白），甘草1斤（炙，为末用，留1半为衣），厚朴1斤（用干姜半斤，水拌令润透，同炒干，去姜不用），癞蛤蟆10只（蒸熟，焙干为末），禹余粮（煅）1斤（如无，以蛇含石代），川黄连1斤（用苦参4两，好烧酒1斤，2味拌合1时，焙干去参），神曲1斤（炒），牡蛎（煅7次，童便淬7次，净）1斤，青蒿1斤（童便制为末），山楂（去核）1斤，鳖甲（醋炙）1斤，胡黄连半斤，芦荟4两，夜明砂（淘净）4两，使君子（去壳，净肉）4两，鹤虱不拘多少。
出处：《医便》卷四。
主治：小儿疳证，肚大筋青，潮热咳嗽，胸前骨露。
加减：None
功效：调脾胃，养血气，消积杀虫，散疳热。
用法用量：每服：8岁以下50丸；9岁以上70丸，食前清米汤送下。
制备方法：上药各制净为末；外用小红枣5斤（去皮核），黄耆3斤，当归1斤，熬膏，入面1斤，打和作糊为丸，如绿豆大；以前甘草末半斤擂丸，小茴香末各4两为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
