序号：76822
名称：黑归脾丸
组成：熟地4两，人参2两，冬术2两，茯神2两，枣仁2两，远志2两，黄耆1两5钱，当归1两，木香5钱，炙草5钱，桂元1两，生姜1两，大枣50枚。
出处：《饲鹤亭集方》。
主治：心肾不交，劳伤过度，精血虚损，怔忡健忘，惊悸盗汗，发热体倦，食少不眠，肠红痔血，三阴亏损，疟疾不愈，及妇人带下。
加减：None
功效：补脾益肾，养心宁神。
用法用量：口服。
制备方法：炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：《中药成方配本》：将熟地、龙眼肉、枣子同坞烂，枣子坞烂后去皮核，与诸药打和晒干研末，用生姜煎汤泛丸，如绿豆大，约成丸十五两。每日二次，每次二钱，开水吞服。
