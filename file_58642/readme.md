序号：58657
名称：秘传疟丹
组成：甘草（生用）、信石（真者，生用）、少许寒水石（如琉璃亮者是）。
出处：《朱氏集验方》卷二。
主治：疟疾。
加减：None
功效：None
用法用量：每服大人5丸，小儿2丸，早晨用冷水吞下。如不丸者，用末予以冷水调，大人每服只用1字钱，小儿服半字钱，临发日服之。真实药料不过5-6服自然而愈，后服：草果、半夏（汤泡7次）、制厚朴、青皮、良姜（壁土炒）、甘草各等分，(口父)咀，每服3钱，水1盏乌梅煎，空心服。
制备方法：上为细末，面糊为丸，如绿豆大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：莫吃热食，只用温汤淘饭止饥，粥食一例放冷，恐热触犯药性。
附注：None
