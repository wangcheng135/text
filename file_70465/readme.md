序号：70481
名称：紫金皮丸
组成：紫金皮（醋炒）、刘寄奴、川当归（煨，盐水炒）、香白芷（醋炒，加减）、赤芍药、白芍药（米泔炒，加减）、黑牵牛、生地黄（盐煨，浸，炒）、川芎（米水浸）、川牛膝（茶水浸）、乳香（可加减）、没药（可加减）、破故纸（酒炒）、木通（去节）、木香（茶水炒）、自然铜（骨不碎折不用，临好时用）、藿香、木贼、官桂（可加减）、羌活、独活、半夏3钱（水炒，无痰不用），骨碎补1两，草乌（醋炒，孕妇不用）1两，川乌（火煨，孕妇则不用）1两。
出处：《普济方》卷三一○。
主治：打扑损伤。
加减：如骨折损，负痛不止，加乳香、没药、白芷、川芎各5钱，生姜酒送下。或不作丸，为末亦可。
功效：None
用法用量：或被攧扑损伤，金刃箭镞，不问轻重，每服1丸，如被刃伤筋断损内重者，以薄荷汤，或木瓜、姜、灯心诸汤下。
制备方法：自然铜、官桂、没药、乳香不炒，其余药或炒，或火焙，或日晒干，皆可，然后入不炒4味，同研为末，用蜜糊为丸，如弹子大，以黄丹为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
