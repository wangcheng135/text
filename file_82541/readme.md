序号：82557
名称：麝香散
组成：麝香（研）半钱，甘草如病人中指长（男左女右），东引桃枝1握，青蒿1握，东引柳枝1握，石榴枝1握，犀角（镑）半两，阿魏1两，柴胡（去苗）1两，葱白7茎，薤白7茎。
出处：《圣济总录》卷九十三。
主治：传尸骨蒸，冷热五劳。
加减：None
功效：None
用法用量：用童子小便2升半，浸1宿，别入槟榔末3钱，同煎至1升半，去滓，分温3服，男病女煎，女病男煎。初服讫，如人行5里，又进1服。恐恶心，可含白梅，病在上即吐，在下即泻，各出恶物如虫类及头发、马尾状，兼身上如蚁行。泻后以葱粥及软饭补之。仍服后方茯神汤，调和五脏。若远年病重，不过两剂。其吐下虫，腹红色者可治，黑色者或愈或否，白色者不可治也。
制备方法：上药除麝香外，同锉碎。
临床应用：None
各家论述：None
用药禁忌：宜避风一月；忌食油腻、湿面、咸酸并牛羊鸡猪犬肉、鱼腥。
附注：《医学正传》有天灵盖二钱半。
