序号：19997
名称：地黄煎丸
组成：生地黄2斤（以布袋取汁，留滓），生姜2斤（以布袋研取汁，留滓。以生姜汁炒地黄滓，以地黄汁炒生姜滓，至干），蒲黄4两，当归1两，延胡索（糯米内炒赤，用末）1两，南番琥珀3两。
出处：《鸡峰》卷十六。
主治：妇人胎前产后，眼见黑花，或即发狂，如见鬼状；胎衣不下，失音不语，心腹胀满，水谷不化，口干烦渴，寒热往来，口内生疮，咽中肿痛，心忪悸，夜不得睡；产后中风，角弓反张，面赤，牙关紧急，崩中_x000D_
下血，如豚肝状，身体烦躁，恍惚昏迷，四肢肿满；及胎不安，唇口指甲青，下血，脐_x000D_
腹满痛。
加减：None
功效：None
用法用量：每服1丸，食前当归汤送下，日2次。
制备方法：上为细末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：琥珀地黄丸（《妇人良方》卷十九）。
