序号：6216
名称：加味茯苓汤
组成：苍术2钱，白术3钱，茯苓1钱，甘草（炙）5分，猪苓8分，泽泻1钱，升麻5分，肉桂7分，柴胡6分，黄芩1钱，生草5分，川归1钱，白芍7分。
出处：《杏苑》卷四。
主治：水泄注下，日夜无度，小便短少，口渴咽干，腹中疼痛或变成赤白痢疾。
加减：None
功效：补中健脾，疏利湿热。
用法用量：上(口父)咀。水2钟，煎1钟，温服。
制备方法：None
临床应用：None
各家论述：此证中气不充，脾湿壅遏。法当补中健脾，疏壅湿热。故用苍术、白术、茯苓、甘草等补中健脾；猪苓、泽泻利小便，分消水湿；升麻升阳气上行，兼助柴胡、黄芩清热；生草泄火；下多则亡阴，故佐归、芍以助阴血。
用药禁忌：None
附注：None
