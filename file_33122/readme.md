序号：33131
名称：万应紫金膏
组成：沥青2斤半，威灵仙2两，蓖麻子100粒（去壳，研），木鳖子28个（去壳，研烂），乳香1两（笋箬炙，为末），没药1两（为末），黄蜡2两，生姜1斤（捣汁1碗），麻油（夏2两，春、秋3两，冬4两，先同灵仙熬，去渣，滴水不散为度）。
出处：《回春》卷八。
主治：跌扑伤损，手足肩背并寒湿脚气风毒，痛不可忍；泻痢，咳嗽、吐血，心疼。
加减：None
功效：None
用法用量：每用好厚绢纸摊贴，先将姜擦患处，后贴上，即用烘热鞋底熨之。泻痢、贴丹田；咳嗽、吐血，贴背心；心疼，贴心上。
制备方法：上将沥青研末，同2汁下锅熬化，看2汁尽时，却起火，桃、柳条不住手搅匀，却入前灵仙油同熬，再下木鳖子、蓖麻子捣匀入内搅，又下乳、没、黄蜡再搅，即成膏矣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
