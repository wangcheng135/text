序号：82699
名称：不换金卷云丹
组成：大石蟹1个（照后制法），黄连、黄柏、黄芩、大黄、细辛、桔梗、防风、荆芥、栀子、薄荷、羌活、乌药、陈皮、枳壳、前胡、当归、姜黄、木贼、菊花、干姜、桑白皮各等分。
出处：《医统》卷六十一。
主治：一切远年膜障。
加减：None
功效：None
用法用量：专治一切风热暴赤烂弦，迎风流泪，怕日羞明，或半年一发，或1年一发，歇作无时，悉以轻剂点之，不可轻用重药，病轻药重，反受其害，内服合病之剂为助。次轻号：丹头6分，冰片1分，麝香3厘，坯子1钱。上专治久患不愈，珠上必生薄翳，或有红筋赤脉，悉以此次轻药点之，日3-5次；若见退减，日点2次，愈则勿点。重号：丹头7分，冰片1分，麝香3厘，坯子1钱。上治眼患颇重，或翳障垂帘，或赤带痛涩，用此吹点，日3-4次，日渐好即止，吹药点数亦减，内服本经药为愈。至重号：丹头9分，冰片1分，麝香3厘，坯子1钱。上专治重眼厚膜遮睛，银疗白翳昏盲无见，方点此药，日点5次渐愈渐减。
制备方法：上将21味锉细，铜器用水3碗浸3日，用布滤去滓，却将石蟹微火煅令紫色，入药汁内，蘸冷取起，细研为末，就将药水淘飞浮清者，以净器盛浮水，安静室勿动，以物覆器上，毋使尘垢入内，俟其澄清，倾去药水，以蟹粉晒干取用，配后诸药：石蟹5钱，坯子5钱，熊胆2钱，胆矾2钱，硼砂2钱，朱砂1钱，银朱1钱，蕤仁（制）1钱，轻粉1钱，川椒5分，胡黄连5分，夜明砂5分，牛黄5分，珍珠5分，鹰条5分，血竭2分，巴豆霜2分，金墨2分。上各依制法合乳极细无声，瓷罐贮之听用，名曰丹头，随病轻重加减点眼。轻号：丹头5分，大冰片1分，麝香3厘，坯子1钱。上共研极细。
临床应用：None
各家论述：None
用药禁忌：None
附注：不换金拨云丹（《审视瑶函》卷六）。《审视瑶函》有柴胡。
