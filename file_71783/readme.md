序号：71799
名称：痧气丹
组成：天麻3两6钱，苍术（米泔水制）3两，麻黄3两6钱，大黄6两，甘草4两4钱，丁香6钱，麝香3钱，朱砂3两6钱，雄黄3两6钱，蟾酥（制）9钱。
出处：《中药制剂手册》引《济世养生集》。
主治：暑月贪凉饮冷，食物不慎引起腹痛，霍乱吐泻，牙关紧闭，四肢逆冷。
加减：None
功效：祛暑辟秽，开窍解毒。
用法用量：每服2-3分，温开水送服，1日1-2次；外用研粉，吹鼻取嚏。
制备方法：上先将雄黄、朱砂、麝香各研细粉，单包，蟾酥捣碎，用牛乳发透，晾干研为细粉；其余天麻等6味，共扎为细粉；另取糯米粉6两，以开水冲为稀糊，酌加冷开水调为稀液，与上药粉泛为小丸，取干丸，用雄黄1两6钱为衣。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服，非时令病不宜服。
附注：None
