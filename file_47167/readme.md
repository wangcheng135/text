序号：47182
名称：珍珠散
组成：珍珠1钱（人乳浸3日，夏天须每日换乳，研极细如飞面），血竭5分，儿茶5分，石膏1钱（煨），炉甘百1钱（以黄连5分煎汁煅淬，研极细，水飞净），赤石脂1钱（煅），陈年丝吐头5分（煅存性），冰片1分2厘。
出处：《徐评外科正宗》卷二。
主治：诸毒脓腐已尽，未收口者。
加减：None
功效：生肌长肉，平口收功。
用法用量：用时掺疮面。
制备方法：上各为细末，再称准，共研极匀，瓷瓶收贮，勿令泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：奇效八宝丹（《春脚集》卷四）、八宝丹、珍珠犀黄八宝丹（《梅氏验方新编》七集）、珍珠生肌散（《药奁启秘》）。方中珍珠，《春脚集》、《梅氏验方新编》均作“珠母”。
