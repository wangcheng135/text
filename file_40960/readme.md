序号：40975
名称：三消丸
组成：甘遂1钱，木香1钱，巴豆（去壳）1钱。
出处：《回春》卷三。
主治：肿胀。
加减：None
功效：None
用法用量：量人虚实用之，实者每服2分，虚者每服分半。先用五苓散加瞿麦、车前、木通、滑石煎服，后服此三消丸。消上用陈皮汤送下；消下用葱白汤送下。隔1日进1服，3服止。若动3-5次，以冷粥补之。消完后用白术3两，陈皮3两，甘草（炙）3两，厚朴（姜炒）2两，皂矾3两，用面炒尽烟，或用醋炒皂矾3-5次，同前药研为末，醋糊为丸，如梧桐子大。每服50丸，米汤送下，每日进3服，连服49日而安。
制备方法：上为末，寒粟米饭为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌恼怒，戒煎炒及无鳞鱼、诸般发物。
附注：None
