序号：28947
名称：苏合煎
组成：苏合香2两，麝香2两，白附子（炮）2两，女菀2两，蜀水花2两，青木香3两，鸡舌香1两，鸬鹚屎1两。
出处：《外台》卷三十二引《古今录验》。
主治：面皯黯。
加减：None
功效：None
用法用量：上先取糯米2升渐硬，炊1斗，生用1斗，合醇酢，用水1斛5斗，稍稍澄取汁合得1斛，煮并令沸，以绵裹诸药，纳着沸浆中煎得3升，药熟以澡豆洗皯处令燥，以药敷皯上，日2次。欲敷药，常以酢浆水洗面后涂药，徐药至3-4合，皯处当小急痛，好处微微剥去便白，以浆3洗，3敷玉屑膏讫，白粉粉之，若急痛勿怪，痒勿搔之，但以粉粉上面，按抑痒处满百日，可用脂胡粉取愈。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
