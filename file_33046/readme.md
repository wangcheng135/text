序号：33055
名称：芦荟肥儿丸
组成：五谷虫（炒）2两，芦荟（生）1两，胡黄连（炒）1两，川黄连（姜炒）1两，银柴胡（炒）1两2钱，扁豆（炒）2两，山药（炒）2两，南山楂2两半，虾蟆（煅）4个，肉豆蔻（煨）7钱，槟榔5钱，使君子（炒）2两半，神曲（炒）2两，麦芽（炒）1两6钱，鹤虱（炒）8钱，芜荑（炒）1两，朱砂（飞）2钱，麝香2钱。
出处：《金鉴》卷五十二。
主治：肝疳，面目爪甲皆青，眼失眵泪，隐涩难睁，摇头揉目，合面睡卧，耳疮流脓水，腹大青筋，身体羸瘦，燥湿烦急，粪青如苔。
加减：None
功效：None
用法用量：每服1钱，米饮送下。
制备方法：上为细末，醋糊为丸，如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
