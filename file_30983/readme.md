序号：30992
名称：十滴水
组成：扁豆3两，丁香半两，厚朴2两，香薷1两，花椒半两（开口者），云茯苓3两（连皮），藿梗3两，辣椒1两，猪苓1两，苍术3两，干姜1两，泽泻2两，白芷1两，陈皮2两，腹皮2两，干烧酒5斤。
出处：《集成良方三百种》。
主治：霍乱吐泻，受寒腹痛，恶心作呕，白痢水泻，一切痧症。
加减：None
功效：None
用法用量：每服1酒钟。如急用，将坛放入大锅内，加满水，煮沸1昼夜，凉透过滤即可用。每服10滴。病重酌加，温开水和服。
制备方法：上药连酒装入净瓷坛内，浸泡百日，打开过滤，去滓装小瓶，塞紧勿使泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
