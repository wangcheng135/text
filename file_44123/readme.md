序号：44138
名称：复明膏
组成：白丁香（腊月收者尤佳，水飞）8钱，拣黄连1两，防风（去芦，锉一指许）1两，新柳枝（方1寸者）3片。
出处：《儒门事亲》卷十五。
主治：外障。
加减：None
功效：None
用法用量：点眼如常。本方每药半合，用片脑1麦粒大，不用亦可。
制备方法：上用新水1升半，雪水更妙，春、秋两三时，冬月1宿，以银石器内熬至6分，滤去滓；另用蜜1斤，密陀僧（为极细末）3字，入蜜搅匀另熬，以无漆匙撩点，下蜜中急搅，候沸汤定，一人搅蜜，一人旋搅药汁，都下在内搅匀，再熬三两沸，色稍变，用新绵3两，重滤去滓，盛器内。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
