序号：73462
名称：棘针散
组成：棘针（倒勾多年者）32枚，大豆黄（生用）40枚，绯绢3条（每条阔1寸），乱发（如鸡子大）3团。
出处：《普济方》卷二七四引《肘后方》。
主治：疔肿。
加减：None
功效：内消。
用法用量：先研2团，令细，以温酒半钱调下。候觉疮四边软，即愈；过半日未效，更服1团，必愈。愈后无有触犯，如有即生3-5个赤黑脓窠；不触犯者，7-8日当愈，勿轻之。
制备方法：上分为三份，各以绯1片，裹棘针、豆子，用发1团缠裹绯帛，令周匝牢固，各于炭火烧令烟尽。
临床应用：None
各家论述：None
用药禁忌：None
附注：棘刺散（《圣惠》卷六十四）。方中大豆黄，《圣惠》作“生黑豆叶”。
