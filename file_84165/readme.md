序号：84183
名称：五石黄耆丸
组成：黄耆2两，紫石英2两（研），赤石脂2两，石硫黄2两（研），石斛2两，白石脂2两，白矾石2两（炼，研），桂心4两，乌头2两（炮，去皮），炼钟乳2两（研），芎2两，防风2两，茯苓3两，干姜4两，枣100枚，当归2两，细辛3两，人参2两，肉苁蓉2两，附子2两（炮），干地黄2两，芍药3两，甘草3两（炙），白术2两。
出处：《外台》卷十七引《古今录验》。
主治：五劳七伤，诸虚。
加减：None
功效：补益。
用法用量：每服10丸，空腹酒送下，1日3次。渐加之20丸。
制备方法：草、石各别为末，枣、蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌海藻、菘菜、猪肉、冷水、桃、李、雀肉、生葱、酢物、芜荑、生菜。
附注：None
