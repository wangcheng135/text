序号：73255
名称：斑龙宴
组成：鹿血。
出处：《韩氏医通》卷下。
主治：脱荣，心血少；痨瘵。
加减：None
功效：不拘男女老少，服之终身无疾而寿。
用法用量：此鹿不拘初生，但驯养牡鹿，或1-2只，每日用人参1两，煎汤饮之，渣和草料饲之。每用预夜减食，次晨空心以布缚鹿于床，首低尾昂，用三棱针刺眼大眦前毛孔（名天池穴），银管3寸许，插向鼻梁，吮其血，和以药酒（任意，或八珍散加沉香、木香煮者），尽量。月可1度，鹿无恙。若有屠刺鹿血，乘热和酒一醉亦妙。
制备方法：None
临床应用：None
各家论述：《增补内经拾遗方论》：白斑曰斑；鹿与游龙相戏，必生异角，故得称龙；宴，饮也，刺血饮之，故曰宴。斑龙者，鹿也，气禀纯阳，活血补血，非金石草木之比。
用药禁忌：None
附注：None
