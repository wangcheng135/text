序号：50362
名称：茴香橘核丸
组成：小茴香（炒）40两，香附（炙）40两，昆布40两，荔枝核80两，穿山甲（炒）20两，肉桂（去粗皮）16两，橘核（炒）40两，青皮（炒）40两，大茴香40两，补骨脂（炒）20两，木香20两，桃仁（去皮）16两，槟榔40两，玄胡索（炙）40两，川楝子80两，莪术（炙）20两，乳香（炙）20两。
出处：《北京市中药成方选集》。
主治：各种疝气，睾丸偏坠，坚硬肿痛。
加减：None
功效：散寒软坚，行气止痛。
用法用量：每服3钱，生姜淡盐汤或温开水送下，每日2次。
制备方法：上为细粉，用冷开水泛为小丸。
临床应用：None
各家论述：None
用药禁忌：忌食生冷。
附注：None
