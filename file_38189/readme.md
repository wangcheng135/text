序号：38199
名称：松焙饼子
组成：细墨半两（焙），芫花（醋浸，炒焦赤）半两，青礞石半两，大戟半两，干漆（炒）半两，五灵脂半两，荆三棱半两，蓬莪术半两，密陀僧半两，陈皮（去白）半两，牡蛎（烧）半两，巴豆1两（去皮，用湿纸三层裹烧），大干枣14个（去核，烧存性），白丁香1分，硇砂（研）1分，虻虫（去翅足）1分，斑蝥（去翅足）1分。
出处：《幼幼新书》卷二十九引《保生信效》。
主治：一切块癖积滞、气血瘕聚等10-20年者；大人、小儿久痢脓血、休息痢。
加减：None
功效：None
用法用量：每服1丸，记以所伤物煎汤，或面汤送下，须以齿啮咽之。其积渐渐移下，再服；再觉移下，更1丸，积自下。若寻常要宜转，只以面汤送下。
制备方法：上为细末，醋煮面糊为丸，如皂子大，捻作饼子。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
