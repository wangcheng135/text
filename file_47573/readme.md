序号：47588
名称：神仙化癖膏
组成：大黄2两，木鳖子21个，穿山甲10片，归尾5钱，白芷5钱，巴豆仁250个，栀子5钱，莪术3钱，蓖麻子仁120个，防风5钱，三棱3钱，官桂3钱，胎发1块（如无，少年亦可），槐枝20寸，柳枝20寸，上药入油，先炸至老黄色，去滓取净油24两，入飞过黄丹10两，熬至滴水成珠，下火待温，再入后细药末。全蝎10个（炙），蜈蚣2条，红娘子2钱，斑蝥2钱，片脑5分，硇砂3钱，阿魏5钱，硼砂3钱，血竭3钱，芦荟3钱，雄黄3钱，乳香5钱，没药5钱，蟾酥2钱，黄蜡3钱，松香5钱，麝香3钱，轻粉2钱，酥油1两。
出处：《便览》卷三。
主治：积聚。
加减：None
功效：None
用法用量：先熬皂角、皮消水洗搓病上良久，再用葱根搓搽良久，用绢帛摊贴患处。
制备方法：上用柳条一顺手搅匀，收瓷罐内。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
