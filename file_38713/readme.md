序号：38723
名称：三妙血
组成：鸡冠血（老白雄鸡更妙）、豕尾血、蚯蚓血各等分。
出处：《卫生鸿宝》卷四。
主治：痘五六朝，根赤转紫，而顶有孔，如针刺，如嵌顿，身热，苔黄，口渴，便秘。
加减：凡根紫甚至转黑，而顶下陷者，为毒陷，三妙合紫雪加金汁；如身热便秘，顶嵌根紫，或发水泡，而间有半浆者，将无浆之泡挑去，三妙入流气败毒之药。
功效：None
用法用量：酒冲服。一方用白酒酿。
制备方法：None
临床应用：None
各家论述：鸡冠居至高之分，取其阳气充足，天庭不起者，须此攻发，但系盛阳之品，故加豕、蚓等血为佐。豕，阴畜，尾又居至阴，凡血皆热，惟此清凉，尾善动，故尤活血。地龙善窜，活血通经，能引诸药直破恶毒所聚之处。
用药禁忌：None
附注：None
