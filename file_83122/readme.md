序号：83140
名称：乌术保真丸
组成：真茅山苍术5斤（糯米泔水浸1宿，竹刀刮去皮，晒干，为末），黑巨胜子4斤半（即芝麻，每次半斤，水磨，用麻布袋盛，去滓，去浮水，拌前苍术末，饭上蒸晒，如此9次），赤白何首乌5斤（竹刀刮去皮，晒干，为末），黑豆4升半（每次半升，熬豆烂，取汁，拌前首乌末蒸晒，如此9次）。
出处：《同寿录》卷一。
主治：None
加减：None
功效：保养真气。
用法用量：每早、晚服3钱，淡盐汤送下。其初服时，先服2钱，3-4日后，服2钱5分，再3-4日后加至3钱为止。凡有病痛，皆从小便，或冷气、或热气，直至1-2时，勿惊，自然止，病除。
制备方法：用白蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
