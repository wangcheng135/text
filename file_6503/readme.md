序号：6503
名称：二加龙骨汤
组成：龙骨2分，甘草（炙）2分，牡蛎3分（熬），白薇3分，附子3分（炮），芍药4分，大枣4枚（擘），生姜5分。
出处：《外台》卷十六引《小品方》。
主治：虚劳发热自汗，遗精梦交，吐血咳血；虚劳不足，男子失精，女子梦交，吐血，下利清谷，浮热汗出，夜不成寐；肾阳虚，肺阴虚，上热下寒之咳血。
加减：None
功效：清散上焦，温补下焦。
用法用量：以水4升，煮取1升半，分再服。
制备方法：None
临床应用：None
各家论述：1.《血证论》：此方用甘、枣，从中宫以运上下；姜、薇清散，使上焦之火不郁；附、芍、龙、牡温敛，使下焦之火归根。合观其方，以温为正治，以清为反佐，真寒假热，虚阳上浮，为对证。_x000D_
2.《江苏中医》（1986；11：22）：二加龙骨汤以白薇配附子，龙骨配牡蛎为两组主药，取附子温导浮阳，守而不走；白薇从阴中泄热，寒热互用，导火泄热，不治阴虚而阴自安。配以龙、牡镇潜摄纳，咸降益阴，合为用阳和阴之法。
用药禁忌：忌海藻、菘菜、生葱、猪肉、冷水。
附注：None
