序号：52093
名称：大补天丸
组成：黄柏（蜜炒褐色）3两，知母（乳汁炒）3两，龟板（酥炙）3两，怀熟地黄5两，牛膝（酒洗）两半，麦门冬（去心）两半，肉苁蓉（酒洗）两半，虎胫骨（酥炙）两半，山药（炒）两半，茯神两半，黄耆（蜜炙）两半，杜仲（制）2两，甘枸杞子2两，何首乌（制）2两，人参2两，当归（酒洗）1两，天门冬1两，五味子1两，怀生地黄（酒洗，用砂锅煮烂，捣）1两，白芍药（酒炒）2两（冬月只用1两），紫河车1具（取初胎者，米泔洗净，入小砂罐内，水1碗煮沸，候冷取起，放竹蓝中，四围纸糊密，烘干为末，入群药和匀）。
出处：《医统》卷四十八。
主治：男妇虚损劳伤，形体羸乏，腰背疼痛，遗精带浊。
加减：冬，加干姜半两（炒黑）。
功效：None
用法用量：每服80丸，空心淡盐汤送下，冬月酒送下。
制备方法：上为细末，炼蜜加猪脊髓3条为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
