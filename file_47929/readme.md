序号：47944
名称：神圣膏
组成：当归半两，杏仁49个，沥青1两，木鳖子5个（去壳），黄丹3两，乳香4钱（另研），麝香、鹰条、轻粉各不拘多少，桃枝（长3寸）49枝，柳枝（长3寸）49枝。
出处：《杂类名方》。
主治：瘰疬及一切恶疮。
加减：None
功效：None
用法用量：None
制备方法：用小油半斤，以缔裹当归、杏仁、木鳖子、桃柳枝，于沙石器内文武火熬，却用一枝粗槐稍缚短嫩枝搅之，药焦取出不用；乃离火下黄丹、沥青搅匀，再上火少时，以滴水不散为度，勿令伤火，软硬俱不中；后入乳香、麝香、轻粉、鹰条毕，倾于水盆内凝滞。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
