序号：55438
名称：丁字化毒丸
组成：牛黄4分，珍珠4分，蜈蚣（去头足，炙燥），4分，犀角1钱，生生乳1钱，牙皂1钱，月月红1钱7分，白鲜皮1钱7钱，朱砂1钱7分，雄黄1钱5分，乳香1钱5分，川山甲1钱5分，琥珀5分，贝母1钱，血竭1钱，郁金1钱，制大黄2钱。
出处：《疮疡经验全书》卷六。
主治：梅毒，毒结於心，小肠经者；毒注瞳仁，似乎内障，或见或不见；或毒聚舌本作肿；或十指惨痛无时；或疮生遍体，内有不易结痂，而腐烂不已者；或传他经，致生别病。
加减：None
功效：None
用法用量：每早空心服15丸，每晚空腹服10丸，人参汤送下；龙眼汤亦可，病去药减。如余毒未尽，药不可撤。
制备方法：上为末，用神曲末5钱，打稠糊入药，捣匀为丸，如梧桐子大，另研朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：百日内勿使大劳大怒，顺时调理。
附注：None
