序号：27340
名称：补益赤石脂丸
组成：赤石脂2两（细研），白薇3分，芎3分，琥珀1两，鹿茸1两（去毛，涂酥炙令黄），熟干地黄1两，人参半两（去芦头），五味子半两，藁本半两，桂心半两，甘草半两（炙微赤，锉），牡丹半两，牛膝3分（去苗），附子3分（炮裂，去皮脐），干姜半两（炮裂，锉），黄耆1两（锉），芜荑半两，丹参3分，白茯苓2分，肉苁蓉1两（酒洗去皱皮，炙干），细辛半两，当归半两（锉碎，微炒），羌活半两，杜仲1两（去粗皮，炙微黄，锉）。
出处：《圣惠》卷七十。
主治：妇人风虚，劳损羸劣，不能饮食，四肢疼痛，经络不调。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
