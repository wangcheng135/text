序号：27427
名称：补骨脂丸
组成：大草乌头4两（水浸1-2片，擦洗乌头皮尽，切片子，控干，每1斤乌头，用盐4两，淹，春、冬7日，夏、秋3日，候盐味入尽，晒干，慢火旋旋炒，直候色焦褐，取2-3片嚼，如不麻人方用），苍术4两（去皮，米泔浸1宿，连泔煮7-8沸，取出，以水洗去泔，切片子，焙燥）。二味用葱白大者10茎，湿纸裹，煨，切碎，烂研，拌上药，同淹1宿，焙干，草乌不须葱淹亦可。川楝子（炮，去核）2两，台椒（去目并合口者，蒸1宿）2两，补骨脂（炒）2两，巴戟（去心）2两，桃仁（汤浸，去皮尖，炒褐色，研如油，旋入）2两，舶上茴香（炒）2两，白茯苓（去皮）2两。
出处：《魏氏家藏方》卷六。
主治：脾湿流注肾经，渐成下部之疾。
加减：None
功效：None
用法用量：每服30丸，食前温酒、盐汤任下。
制备方法：上为细末，酒煮面湖为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
