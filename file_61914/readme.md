序号：61930
名称：救土通肠汤
组成：玄参2两，当归1两，生地1两，知母1钱，厚朴1钱，升麻5分，大麻子30粒。
出处：《辨证录》卷九。
主治：脾火作祟，大便闭结，口干唇裂，食不能消，腹痛难忍，按之益痛，小便短涩。
加减：2剂大便必通，减去大麻子、知母，再用4剂。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：此方玄参、生地补脾土之阴，又是泻命门、脾胃之火；当归取以润肠；知母、厚朴取其下行以解热；升麻提脾土之气，则阳升而阴自降；大麻子最润大肠而引火下行，不使阴气上升，正助升麻以提阳气。阳既升而阴又降，则津液无干涩之虞，何患大肠之不通哉。
用药禁忌：None
附注：None
