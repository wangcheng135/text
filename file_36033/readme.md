序号：36043
名称：参苓散
组成：人参2两，茯苓1两，白术20两（锉细），桔梗10两（去芦），橘红6两，香白芷3两，芍药1两，当归2两（去芦头），川芎1两半，半夏1两（汤泡7遍），厚朴2两（去皮，生姜汁制），官桂（去皮）春夏3两，秋冬4两，枳壳4两（麸炒令黄，同桂舂为末），干姜春夏1两半，秋冬2两，麻黄（去根节），春夏2两，秋冬3两，甘草3两。
出处：《传信适用方》卷一引何伯应方。
主治：伤寒，百节疼，一切虚劳气痰。
加减：None
功效：None
用法用量：每服1钱，水1盏，加生姜3片，葱2寸，同煎7分，通口服，不拘时候。
制备方法：上各净洗焙干，除芎、桂、枳壳别杵外，诸药同为粗末。分作6分，于大铁锅内以文武火炒令微赤黄为度，不可令焦，须搅匀取出，以净纸衬安板床上侯冷，勿令侵尘土，入前枳壳，官桂，川芎和匀，杵为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
