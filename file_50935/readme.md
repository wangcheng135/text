序号：50950
名称：大圣镇风金丹
组成：川乌头4两（去皮脐），全蝎1两（去毒，生用），晋矾2两（甘锅内枯，存性），附子4两（炮，去皮脐），白蒺藜2两（炮，去刺，另研），防风4两（去芦头），五灵脂2两（去石，别研），白附子4两（生用），白僵蚕2两（去丝嘴，微炒），朱砂半两（另研细），没药2两（去石，另研），麝香半两（另研细）。
出处：《普济方》卷九十二。
主治：卒患中风，左瘫右痪，口眼喎斜，涎潮搐搦，言语謇涩，偏正头风。
加减：None
功效：None
用法用量：每服1粒，食后、临睡生姜自然汁磨化，热酒调服，再饮少量热酒。即就暖处，覆以衣被，候汗出即愈。病小者，每粒分2服。
制备方法：上为细末，以熟汤放冷，磨京墨1两，成浓汁，搜和合匀，每温剂1两，可停分作4丸，只于风中干之，不可日晒，金箔为衣。
临床应用：None
各家论述：None
用药禁忌：忌发风物。孕妇不可服。
附注：None
