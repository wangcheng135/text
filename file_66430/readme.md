序号：66446
名称：小丁香丸
组成：丁香1分，沉香（锉）1分，乳香（研）1钱半，茴香子半两（炒），桂（去粗皮）半两，槟榔2枚（冬加2枚，锉），肉豆蔻2枚（夏加2枚，去壳），荜茇半两，阿魏（研）少许，巴豆15枚（去皮心，不出油，别研）。
出处：《圣济总录》卷五十四。
主治：三焦胀满，胸膈气不和，及元脏冷气，上攻迷闷。
加减：None
功效：消化滞气。
用法用量：每服5丸，生姜汤送下。如胸膈气不和，及元脏冷气，上攻迷闷，加至10丸，温酒送下；常服熟水亦得。要微动，以意加服之。
制备方法：上药除研者外，捣罗为末，次入乳香、巴豆、阿魏令匀，煮白米粥为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
