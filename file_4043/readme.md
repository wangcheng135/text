序号：4043
名称：木瓜丸
组成：木瓜（约及4两者）4枚（先炸熟，去皮核捣，与熟艾4两相和，蒸烂研匀，以备合后药），木香半两，白附子半两，羌活半两，没药半两，白术半两，舶上好茴香（炒）7钱半，川楝肉（炒）7钱半，白牵牛（炒熟）7钱半，威灵仙（折之内有白点者即不用，须子播净）1两。
出处：《脚气治法总要》卷下。
主治：脚气。
加减：None
功效：None
用法用量：每服20丸，食饮空时以木瓜汤或酒吞下。地道稍秘滞即空心服，有小壅亦可空心服。
制备方法：上为细末，先以前木瓜、艾和搜，俟少干，徐入少熟蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
