序号：51494
名称：追风丸
组成：川乌（炙）1两，没药（醋炙）1两，木瓜1两，当归2两，续断5钱，麻黄5钱，马钱子（烫去毛）6钱，草乌（炙）1两，生杜仲1两，南红花1两，千年健5钱，独活、蕲蛇（酒炙）5钱，乳香（醋炙）1两，菟丝子1两，虎骨（油炙）1两，追地风5钱，羌活5钱，甘草5钱。
出处：《北京市中药成方选集》。
主治：风寒湿痹，腰腿疼痛，手足麻木，半身不遂。
加减：None
功效：祛风散寒，活络止痛。
用法用量：每服1丸，温开水或温黄酒送下。
制备方法：上为细末，每15两6钱细末兑麝香5分，和匀，炼蜜为丸，每丸重2钱，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
