序号：81897
名称：熟艾方
组成：艾叶（别揉令熟）1两，砒霜（研）1两，水银（与雄黄同研）1两，腻粉（研）1两，硫黄（研）1两，丹砂（研）1两，阿魏（酒化，去沙石，面和作饼，炙）1两，附子（炮裂，去皮脐，捣）1两，雄黄（研细，入水银，点醋再研令星子尽，始入腻粉、砒霜、硫黄、丹砂同研）1两，麝香（研）半两，猪牙皂荚1斤（去皮子，别为末）。
出处：《圣济总录》卷十八。
主治：大风癞。
加减：None
功效：None
用法用量：None
制备方法：上除艾叶、皂荚别捣为末外，并合和细研，用纸4张，先布艾，次掺皂荚末，又次诸药末，卷刀切如饼餤，着碗内烧，以小木甑子笼碗，以衣衬四边，坐于上熏，以衣被遮壅定，不令透出气，热闷即俯伏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
