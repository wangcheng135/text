序号：34398
名称：万病无忧酒
组成：当归5钱，川芎5钱，白芷5钱，白芍1两，防风7钱5分，羌活1两，荆芥穗5钱，地骨皮5钱，牛膝5钱，杜仲_x000D_
（炒）1两5钱，木瓜5钱，大茴香5钱，破故纸1两，五加皮1两5钱，威灵仙1两，钩藤1两，石楠藤1两，乌药5钱，紫金皮1两5钱，自然铜（火煅），木香5钱，乳香5钱，没药5钱，甘草（炙）5钱，雄黑豆2两。
出处：《寿世保元》卷四。
主治：None
加减：None
功效：除百病，理风湿，乌髭须，清心明目；利腰肾，健腿膝，补精髓，疗跌扑损伤筋骨；和五脏，平六腑，快脾胃，进饮食，补虚怯，养气血；去风活血，养神理气。
用法用量：后取酒温饮之、或晨、午后、晚，随量饮之。其味佳。如饮一半，再加好酒浸饮妙。
制备方法：上药调匀，用缯布为囊盛之，无灰酒1大坛，入药在内，春、秋5日，夏3日，冬10日。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
