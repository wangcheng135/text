序号：48710
名称：卫经丹
组成：砂仁1两，丁香（不见火）1两，荜茇1两，厚朴（姜汁制1宿，炒）半两，白豆蔻仁半两，人参（去芦）半两，肉豆蔻（面裹煨）半两，神曲（炒）半两，半夏曲（炒）半两，附子（炮，去皮脐）半两，荜澄茄半两，陈皮（去瓤）半两，干姜（炮、洗）半两，白术（炒）半两，鹿茸1两（去毛，酥炙），麝香1钱（别研）。
出处：《魏氏家藏方》卷五。
主治：脾胃怯弱，久受虚寒，腰腹疼痛，泄泻无时，面无颜色，精神不爽，腰膝酸重，胸膈痞塞，呕吐恶心，痰唾稠粘。
加减：None
功效：大壮脾胃，美进饮食。
用法用量：每服50-60丸或70丸，米饮、盐汤任下，不拘时候。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
