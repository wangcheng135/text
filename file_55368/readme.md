序号：55383
名称：大活血丸
组成：青桑炭1斤，栗间1两6钱，骨碎补1两6钱，南星（制）1两6钱，白芍药1两6钱，牛膝1两6钱，川乌（炮）1两6钱，黑豆（酒煮）1两6钱，自然铜8钱，木鳖子8钱，细辛1两，降真香节3钱，枫香3钱，乳香6钱，没药6钱，血竭6钱。
出处：《准绳·疡医》卷六。
主治：打扑伤损，折骨碎筋，瘀血肿痛，瘫痪顽痹，四肢痠疼，一切痛风。
加减：None
功效：None
用法用量：每用1丸，用无灰酒磨化服。
制备方法：上为末，醋煮秫米粉糊，集众手搓为丸，缓则发裂，如弹子大，候干，用生漆为衣，久则不坏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
