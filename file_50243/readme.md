序号：50258
名称：茱萸鹿茸丸
组成：吴茱萸（汤洗3次）半两，附子（炮裂，去皮脐）半两，干姜半两，肉豆蔻（面裹煨）半两，白茯苓半两，黑龙骨（炭火3斤，烧通赤经宿，研细末，水飞）半两，杜仲（锉碎，酒浸，炒断丝）1两，五味子1两，苁蓉（酒浸1宿）1两，鹿茸（削去皮毛，劈开，涂酥炙）1两，赤石脂（粘舌者）1两，熟干地黄1两半。
出处：《百一》卷十八。
主治：本脏因虚生寒，月经行多，或来不及期，腹痛怯风，脏腑不和。
加减：None
功效：补气固血。
用法用量：每服50-70丸，空心、食前热米饮送下。1-2月血气已安，去龙骨加沉香半两，可以常服。
制备方法：上为细末，煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
