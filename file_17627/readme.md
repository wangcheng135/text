序号：17634
名称：人参补阳汤
组成：羌活6分，独活6分，白芍药3分，生地黄3分，泽泻3分，人参4分，白术4分，茯苓4分，黄耆4分，炙草4分，当归4分，柴胡5分，防风5分，熟地黄（酒洗，炒）4分。
出处：《原机启微》卷下。
主治：伤寒余邪不散，上走空窍，其病瘾涩赤胀，生翳羞明，头痛骨痛。
加减：None
功效：None
用法用量：上作1服。水2盏，煎至1盏，去滓热服。
制备方法：None
临床应用：None
各家论述：《原机启微》薛己按：上方分利阴阳升降上下之药也。羌活、独活为君者，导阳之升也；茯苓、泽泻为臣者，导阴之降也；人参、白术大补脾胃，内盛则邪自不容，黄耆、防风大实皮毛，外密则邪自不入，为之佐也；当归、熟地黄俱生血，谓目得血而能视，生地黄补肾水，谓神水属肾，白芍药理气，柴胡行经，甘草和百药，为之使也。
用药禁忌：None
附注：人参补胃汤（《审视瑶函》卷二）。
