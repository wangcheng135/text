序号：42837
名称：保命延寿丹
组成：硫黄2两，明雄黄2两，辰砂2两，赤石脂2两，紫石英2两，阳起石（火煅，醋淬3次）2两。
出处：《扁鹊心书·神方》。
主治：痈疽，虚劳，中风，水肿，臌胀，脾泄，久痢，久疟，尸厥，两胁连心痛，梦泄遗精，女人血崩白带，童子骨蒸劳热，一切虚羸，黄黑疸，急慢惊风。
加减：None
功效：None
用法用量：每服10粒，空心送下。童男女5粒，小儿2-3粒。
制备方法：上为细末，同入阳城罐，盖顶，铁丝扎定，盐泥封固，厚1寸，阴干，掘地作坑，下埋一半，上露一半，烈火煅1日夜，寒炉取出，为细末，醋为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
