序号：67611
名称：小乌犀丸
组成：乌犀1两，朱砂1两，天麻1两，羌活1两，芎1两，防风1两，干蝎1两，白僵蚕1两，甘菊花1两，地龙1两，蔓荆子1两，干姜半两（炮），牛黄半两（研入），麝香半两（研入），天南星3分（炮），败龟3分（酥涂，炙令黄），白花蛇3分（酒浸，去皮骨，炙黄），肉桂3分，附子3分（炮），木香3分，人参3分，海桐皮3分，虎胫骨3分（酥炙，令黄色），当归3分（微炒）。
出处：《医方类聚》卷二十引《神巧万全方》。
主治：一切风，走疰肢节，痛不可忍者。
加减：None
功效：None
用法用量：每服1丸，以暖酒或薄荷汤嚼下。
制备方法：上为末，炼蜜为丸，如小弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
