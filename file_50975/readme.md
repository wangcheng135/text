序号：50990
名称：大安汤
组成：麻黄（去根节，汤煮，掠去沫，焙）4两，防风（去叉）1两半，芎1两，羌活（去芦头）1两，桔梗（去芦头，锉，炒）1两，柴胡（去苗）1两，赤箭1两，白鲜皮1两，蔓荆实（去皮）1两，独活（去芦头）1两半，前胡（去芦头）1两半，甘草（炙，锉）半两，人参2两，松花2两，石膏（碎，研）3两。
出处：《圣济总录》卷十三。
主治：风邪伤人，寒热时作，头痛烦躁，周身疼痛，颈项拘急。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，加薄荷5叶，煎至8分，去滓温服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
