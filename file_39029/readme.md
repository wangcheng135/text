序号：39039
名称：泽兰丸
组成：泽兰2两，黄耆2两（锉），白术2两，柏子仁2两，赤石脂2两，白矾1两（烧令汁尽），桂心1两，木香1两，人参1两（去芦头），羌活3分，白茯苓1两，附子1两（炮裂，去皮脐），续断1两，芎1两，当归1两（锉，微炒），细辛1两，陈橘皮半两（汤浸，去白瓤，焙），龙骨1两，藁本半两，干姜半两（炮裂，锉），川椒1两（去目及闭口者，微炒去汗），厚朴2两（去粗皮，涂生姜汁，炙令香熟）。
出处：《圣惠》卷八十一。
主治：产后虚损，夹风劳气，吃食减少，面色萎黄，腹内冷疼，四肢乏力。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
