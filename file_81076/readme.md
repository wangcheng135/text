序号：81092
名称：麝香饼子
组成：川乌头（炮去皮脐）半两，天南星（炮）半两，干蝎梢半两，白花蛇（酒浸1宿，去皮骨，焙干）半两，干赤头蜈蚣2条（酒浸酥，炙黄）以上并捣，罗为细末，次用：朱砂（细研，水飞）1分，铁粉1分，乳香1分，牛黄（并细研）1分，好真麝香半两（另研）。
出处：《幼幼新书》卷九引张涣方。
主治：慢惊，因吐痢生风，及心肺中风。小儿吐泻之后，脾虚生风，目睛斜视，背脊强硬，手足瘈疭，及心肺中风，昏塞不省。
加减：None
功效：None
用法用量：每服1-2粒，煎人参薄荷汤化下。
制备方法：上1处研细，拌匀，酒煮白面糊，候冷，和为饼子，如芡实大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
