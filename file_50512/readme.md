序号：50527
名称：茵陈煎丸
组成：茵陈蒿3两（捣为末，用醋1升煎为煎），鳖甲（去裙襕，醋浸，炙令黄色）1两，京三棱（煨，杵碎）1两，干姜（炮裂）半两，附子（炮裂，去皮脐）半两，枳壳（去瓤，麸炒）半两，桂（去粗皮）半两，青橘皮（汤浸，去白，焙）半两，柴胡（去苗）1两，白术半两，厚朴（去粗皮，生姜汁涂炙）1两，吴茱萸（水浸1宿，焙干，炒）1两。
出处：《圣济总录》卷九十。
主治：虚劳，下焦虚冷，心腹痞满，吃食无味，舌涩口干，四肢少力。
加减：None
功效：健脾胃。
用法用量：每服30丸，空心生姜汤送下。如和不成，更入热蜜。
制备方法：上药除茵陈外，为末，用茵陈煎汤和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
