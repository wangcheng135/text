序号：63017
名称：清肺化热汤
组成：荆芥、玄参、桔梗、甘草、射干、连翘、犀角、生地、白芍、薄荷、大力子。
出处：《陈素庵妇科补解》卷三。
主治：妊娠喉痹喉风
加减：血虚，加阿胶、知母、川贝母；热甚伏火，大便闭，加大黄、元明粉、天花粉。
功效：清君相火，滋肺金而生肾水，利咽凉膈，清热解毒。
用法用量：None
制备方法：None
临床应用：None
各家论述：是方急治标，缓治本。标者，心肾二火，本者，胎也。荆、薄去头面咽嗌风热；花、射、大力清热解毒；甘、桔治咽喉，开郁利膈；犀、连、地、芍凉血滋阴。君相二火平则咽喉肃清，饮食可下，而水谷之精气能化血而养胎矣。
用药禁忌：None
附注：None
