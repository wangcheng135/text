序号：79722
名称：漏芦汤
组成：漏芦1钱，紫花地丁1钱，荆芥1钱，当归1钱，连翘1钱，薄荷1钱，白芷1钱，升麻1钱。
出处：《医林绳墨大全》卷九。
主治：肿毒。
加减：如治便毒，加猪苓、泽泻；如在上者，加川芎、桔梗；面上，加蔓荆子；下部，加牛膝、木瓜、薏苡仁；如红肿势凶，大便秘结，加大黄3钱，麻黄3钱，甘草4分；如大便不秘，减大黄1钱半，次日红肿尽退，只用神灯照之；若红肿未退，加大黄、麻黄各1钱半，甘草1钱，服至肿消，若红肿未尽，主药及加药各1钱5分，煎服。
功效：None
用法用量：水2钟，煎8分，热服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
