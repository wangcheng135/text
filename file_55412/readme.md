序号：55427
名称：乙字化毒丸
组成：牛黄5分，丁香5分，牙皂5分，琥珀（须择体坚燥者用之）1钱，郁金1钱，生生乳1钱，朱砂5钱5分，雄黄5钱5分，月月红5钱5分，白鲜皮5钱5分，乳香5钱5分，川山甲5钱5分，制大黄2钱，僵蚕4钱。
出处：《疮疡经验全书》卷六。
主治：梅疮毒结于肝胆二经，内作筋痛，攻走胁肋，上至于头，下至于足，转侧艰难，手不能举，足不能步，或颈项发块，或破烂上下，或传他经致生别病者。
加减：None
功效：None
用法用量：上为末，用神曲末5钱打稠糊入药捣匀为丸，如梧桐子大，另研朱砂为衣，每早空心服13丸，每晚空心服九丸，人参汤送下；炒米汤亦可。兼用煎剂调理，病去药减，如余邪未尽，药不可撤。
制备方法：制生生乳法：煅炼礜石3钱，云母石2钱5分，消石（即盆消）1两6钱，朱砂液（即朱水色粉红者为上）9钱6分，晋矾1两2钱，绿矾1两8钱，食盐半两，枯矾5钱6分，青盐3钱5分，上件共研不见星，入羊锡罐内，三方一顶火，俟药化，面上有霜头起离火，候冷用铁盏盖扎，盐泥固济，待罐口泥干，入八卦炉内，先用文火，候盏底热微，微擦水，加炭平口，用武火，三香足离火，先用甘草2钱，牙皂2钱，煎浓汁收盏底白丹砂，棉纸包裹，浸汁内片时取出，连纸埋土中3日夜，取来晒干，每两加冰7厘，麝7厘，辰砂9分，共研极细，外用乳香1钱2分，滚水燉化，和前末研匀为丸，每重1钱1分，外以黄蜡封固，即名生生乳。
临床应用：None
各家论述：None
用药禁忌：百日内勿使大劳大怒。顺时调理。
附注：余用生生乳配风药而治大麻风，配痨药而治传尸痨，配虫药而治诸虫疾，配膈药而治噎塞翻胃，配疮药而治顽毒顽癣，久漏骨痛，种种奇效，不独治广疮毒气之圣药也。大凡药性与禀性有异，人有杀药者，毒药服之竟不觉察，奏功亦缓。性有不杀药者，服之便觉时冒，奏效亦速，所以为医全在活泼。《经》曰：大积大聚，衰其大半而止，不必尽剂。须要体察病情，功效未全者，再宜进药，或间日再服，或停两三日再服，务宜消息增除，毋使过剂，以生药病。
