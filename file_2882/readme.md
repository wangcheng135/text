序号：2882
名称：天竺膏
组成：大风子4钱，蛇床子4钱，牛蒡子4钱，川羌活3钱5分，独活3钱5分，蓖麻子4钱，白练皮3钱5分，白及3钱5分，破故纸3钱5分，白芷3钱，蜂房1个，桑寄生3钱5分，防风3钱5分，南星3钱5分，陈皮3钱，土茯苓4钱，木鳖4钱，皂角刺3钱5分，白芍3钱5分，红花3钱5分，苍耳子4钱，川乌3钱5分，半夏3钱5分，归身4钱，归尾3钱，黄柏3钱，草乌5钱，甘草节3钱，穿山甲3钱5分，杜仲3钱5分，天花粉3钱，附子3钱，黄丹26两，姜汁2两，葱汁1两，头发2两（用鸡蛋清洗净），麻油5斤，桃枝1两，柳枝1两，槐枝1两，桑枝1两，枣枝1两。
出处：《集验良方拔萃·续补》。
主治：心痛，疟疾，五劳七伤，筋骨疼痛，腰膝软弱，左瘫右痪，手足麻木，筋脉拘挛，受寒泄泻，痰喘咳嗽，胃气疼痛，偏正头风，遗精赤白浊，赤白带下，月经不调，血疝崩漏，疝气，走气疼痛，寒热脚气，无名肿毒，诸般恶疮乳患，跌打损伤，积滞痞块等。
加减：None
功效：None
用法用量：一、治远年近日心痛，贴中脘穴；二、治大小疟疾，贴肺腧穴；三、治五劳七伤，遍身筋骨疼痛，腰膝软弱，贴两膏盲穴、两肾腧穴、两三里穴；四、治左瘫右痪，手足麻木，筋脉拘挛，贴两肩井穴、两曲池穴、两手腕穴、两膝眼穴、两三里穴；五、治腰痛，贴命门穴；六、治受寒泄泻，贴下脘穴；七、治痰喘气急咳嗽，贴华盖穴、肺腧穴、膻中穴；八、治胃气疼痛，贴上脘穴；九、治偏正头风，贴风门穴、两太阳穴；十、治男子遗精赤白浊，女人赤白带，月经不调，血癌崩漏，贴阴交穴、命门穴；十一、治小肠疝气，贴膀胱穴、丹田穴；十二、治走气疼痛，贴两章门穴；十三、治寒热脚气，贴三里穴、三阴交穴；十四、治一切无名肿毒，诸般恶疮乳患，跌打损伤，积滞痞块，劳伤内伤，闪挫等症，各贴所患之处。
制备方法：以上各药即入油内浸5日后，入锅煎，捞起诸药滤干，将药磨末，入油再煎，外加上肉桂3钱，麝香7分5厘，雄黄3钱，冰片3钱，苏合油2两，乳香3钱，白豆蔻3钱，木香3钱，没药3钱，丁香3钱，三蚕沙3钱，阿魏3钱5分。上药各为极细末，入油煎膏，滴水成珠便好。所有三蚕沙、阿魏(煎)、黄丹收用。
临床应用：None
各家论述：None
用药禁忌：凡大热火症以及孕妇忌贴此膏。
附注：方中白练皮，《膏药方集》作“白鲜皮”。
