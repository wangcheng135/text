序号：42382
名称：青蒿散
组成：天仙藤1两，鳖甲（醋炙）1两，香附子（炒，去毛）1两，桔梗（去芦头）1两，柴胡（去苗）1两，秦艽1两，青蒿1两，乌药半两，甘草（炙）1两半，川芎2钱半。
出处：《杨氏家藏方》卷十。
主治：虚劳骨蒸，咳嗽胸满，皮毛干枯，四肢懈惰，骨节疼痛，心中惊悸，咽燥唇焦，颊赤烦躁，涕唾腥臭，困倦少力，夜多盗汗，肌体潮热，饮食减少，日渐瘦弱。
加减：None
功效：None
用法用量：每服2钱，水1盏，加生姜3片，同煎至7分，温服，不拘时候。小儿骨蒸劳热，肌瘦减食者，每服1钱，水半盏，加小麦30粒，同煎至3分，温服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中柴胡，《普济方》卷二三六作“前胡”。
