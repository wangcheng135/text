序号：82778
名称：中满分消汤
组成：半夏1钱，厚朴5分，黄连5分，黄柏（上4味俱姜制）5分，川乌（炮）5分，干姜（炮）5分，吴茱萸（净，用开口者，炒）5分，草豆蔻（炒，研）5分，木香5分，人参5分，茯苓1钱5分，泽泻1钱5分，生姜5片。
出处：《张氏医通》卷十三。
主治：中满寒胀。
加减：身热、脉浮、喘满有表证，加麻黄5分；血虚至夜烦热，加归身、黄耆各5分；阳气下陷，便溺赤涩，加升麻、柴胡各3分；脾胃虚寒，饮食不磨，去黄柏，加益智仁、荜澄茄、青皮各2分。
功效：None
用法用量：水煎，稍热食前服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：大忌房劳、生冷、炙煿、酒面、糟醋、盐酱等物。
附注：None
