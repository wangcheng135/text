序号：48457
名称：神效剪红丸
组成：一上末：槟榔（生，研细，取净末）1斤（以2两为母，余14两上第1次，以1等罗筛过，取齐晒干），二上末：商陆（即樟柳根，白者可用，赤者杀人）4两，金毛狗脊4两，贯众4两（以上3味和1处，研极细末，上第2次，以2等罗筛过，取齐晒干）（一方不用贯众，则虫出来犹未死也），三上末：三棱（醋煮）8两，莪术（醋煮）8两，青木香4两，西木香4两，雷丸（醋煮）2两半，南木香2两（以上6味和1处，研极细末，上第3次，以3等罗筛过，取齐），四上末：大黄（铡碎，酒浸，晒干，研细，取净末）1斤（上第4次，以4等罗筛，取齐晒干），五上末：黑牵牛（半生半炒，研细，取头末）1斤（上第5次，以5等罗筛过，取齐晒干）（一方有枳壳1斤为母，有藿香4两，和入诸香）。
出处：《准绳·类方》卷八。
主治：一切虫积。凡因饮酒过度，食伤生冷，致使脾胃不和，心膈胀满，呕恶咽酸，常吐清水，面色萎黄，不进饮食，山岚瘴气，水肿、蛊胀，（鼻勾）齁咳嗽，痰涎壅滞，酒积、食积、气积、气块，反胃噎膈，呕逆恶心，肠风，痔漏，脏毒，酒痢，累蕴积热上攻，头目下生疮癣，妇人血气，寒热往来，肌体羸弱，月经不调，赤白带下，鬼气鬼胎，产后诸疾；小儿五疳，虫积，误吞铜铁，误食恶毒等物。
加减：None
功效：宣导四时蕴积。春宣积滞，不生疮毒；夏宣暑湿，不生热痢；秋宣痰饮，不生瘴疟；冬宣风寒，不生瘟疫。
用法用量：壮人每服5钱，弱人每服4钱，五更以茶清吞下，小儿减半。若病浅，即1服见效；若源深，更须再1服。
制备方法：上作五处，另研极细末，要作五次上末，却用茵陈半斤，大皂角1斤煎汁，滤净，法水为丸，如绿豆大，晒干后用丁香末1两，或加芦荟末1两亦妙，以前净汁煎一滚，洒入丸药，旋摇令光莹为度，再以阿胶2两（生），以前汁熬溶，洒入丸药，旋摇光莹，晒干。
临床应用：None
各家论述：None
用药禁忌：此药温和，不动元阳真气，亦无反恶。孕妇休服。
附注：药后用马桶盛粪于野地看之，庶见药功易辨，或虫、或积、或如烂鱼冻，或作五色等积。若一次未见虫积，更看第二三次下来，此即是病根。有积消积，有气消气，有虫取虫，有块消块。若病根去，其病自消。
