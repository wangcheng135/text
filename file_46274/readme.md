序号：46289
名称：活血养肝汤
组成：知母肉3钱，当归4钱，川芎2钱，玉竹3钱，胡黄连3钱，寸冬3钱，夏枯草3钱，菊花2钱，香附3钱，木贼2钱，石斛3钱，甘草1钱，田三七5分（另为末，冲服）。
出处：《眼科临症笔记》。
主治：花翳白陷症。两眼微红，风轮塌陷，白膜叠生，状如雪花，痠疼隐涩，热泪常流。
加减：None
功效：活血养肝。
用法用量：水煎服。
制备方法：None
临床应用：花翳白陷症：王廷宾之女，19岁。初患月经不调，行经腹疼，经后忽觉二目微疼，视物昏蒙，隔日头疼，热泪下流，白膜隐隐，遂往张先生处调治。每日挑拨点药，终未服药，月余头疼虽止，昏蒙加倍，饮食减少，辞医返里，就诊于余。视其目，睛光低落，白膜深沉，六脉虚脱，惟太阴为甚，此乃脾蕴湿热，肝血亏乏。余先略刺巨髎、上星，内服活血养肝汤，外点消炉散。月余白膜微退，自顾有余。又改服养荣平肝汤，再轮刺攒竹、鱼腰、临泣，半年始愈。
各家论述：None
用药禁忌：None
