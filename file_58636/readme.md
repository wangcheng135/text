序号：58651
名称：秘传明目补下丸
组成：人参3钱5分，川楝子（酒煮，去核）1两半，远志（去心）1两半，川巴戟（去心）1两，菟丝子（酒浸）1两，麦门冬1两，白术5钱，白茯苓（去皮）5钱，赤芍药（酒浸）5钱，青盐5钱，破故纸（炒）5钱，小茴香5钱，葫芦巴5钱，肉苁蓉（酒洗）5钱，黄耆5钱，甘草（炙）5钱，枸杞子5钱，砂仁（炒）5钱，黄柏（盐酒炒）5钱，山药（炒）5钱，知母（去毛皮，盐酒炒）5钱，熟地黄（酒洗，怀庆）5钱，五味子5钱，莲肉（去心）5钱，车前子2钱5分。
