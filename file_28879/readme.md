序号：28887
名称：苍耳酒
组成：苍耳子。
出处：《外台》卷十九引《苏（共水）方》。
主治：身诸风，骨髓中风若愈，或发疮，疮愈后，皮痛。
加减：若虚热羸瘦弱人，无问男女，加生地黄、牛膝根（锉）3升，丹参2升，天门冬各2升，松叶5升，枸杞根5升，杏仁1升（去皮尖），荆根若子2升，水3石，别煮牛膝、丹参、松叶等，取6斗，并苍耳汁总1石，渍5斗曲，用米2石5斗，分4度酘。杏仁末着第1酘饭中下；生地黄捣如泥，着第2酘饭中下；天门冬蒸熟，剥去皮，捣如泥，着第3酘饭中下；又大麻子1大斗，捣碎，着第4酘饭中下。头风者，得甘菊花1升，渍第5酘糟中下，搅之调酒熟，大小同服并得。如或少2-3两物亦得，不必俱备。将渍石斛等药酒弥佳。
功效：去皮节头足诸热风，补虚，延龄轻身，肤坚实光悦，腰脚甚便。
用法用量：每服5合，日2-3次。
制备方法：六月以后收取，晒干，至九月锉1大斛，水3斛，煮取4斗，渍2大斗曲三度，总以米1大斛，渍三日，如凡酿法将息酘之，酒熟。
临床应用：None
各家论述：None
用药禁忌：忌鲤鱼、芜荑。
附注：None
