序号：35677
名称：制绿豆
组成：大附子1个（去皮脐，切作两片用），绿豆2合半（水3碗半，入瓷器内煮，候干熟）。
出处：《朱氏集验方》卷四。
主治：十种水气，脾肾气浮肿。
加减：None
功效：None
用法用量：每1日临卧时吃豆，但依此资次。凡服4日，其水从小便下，肿自消退。如未退，再以前药服之。
制备方法：上取出，乘热空心只吃绿豆，其附子留住。次日将附子2片作4片，再用绿豆2合半，水3碗半，同煮干熟，乘热空心吃绿豆。第3日再别用附子1个，绿豆2合半，如前过度服之。又第4日亦如前第2日法度服之。
临床应用：None
各家论述：None
用药禁忌：忌生冷毒物、盐、酒60日。
附注：None
