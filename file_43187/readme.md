序号：43202
名称：养正膏
组成：鳖甲1两（醋炙），青蒿1握，淡豉30粒，葱白3茎，安息香1分（研），桃枝7茎，柳枝7茎，桑枝7茎，桃仁49个（去皮尖双仁），天灵盖用匕头大1片（酥炙）。
出处：《妇人良方》卷五。
主治：传尸。
加减：None
功效：出汗，取虫，辟邪。
用法用量：上药，隔夜以水1升浸至五更，煎至半升，再以童便半升同煎，取4合；又用槟榔1个（为末），麝香1钱匕，将所煎药去滓调下，至日高二丈时，放温顿服。以衣覆出汗，审看十指，汗出如藕丝，五色臭秽；汗出后，仍泻下虫状恶物尽。甚者，旬日再服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：养正丹（《普济方》卷二三七引《仁存方》）。
