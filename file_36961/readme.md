序号：36971
名称：夜光膏
组成：黄连（拣黄色肥者，去须，称4两，折碎，走水淘洗净，用槐柳枝拨出，控稍干，近腊月中气，用蜡水3升，浸于瓷罐内，封盖7日，放暖处，用雪水妙，冰水亦可），当归身半两（折碎，淘洗如上，与黄连同浸），诃子1对（即诃黎勒也，打破去核，将皮淘净，与黄连同浸），（豕宗）猪胰子2个（掐去脂膜，以秆草叶裹，清水内搓洗至净，细拣去草叶，用刀作切小片，腊水内濯去腥，槐柳枝漉出别放），甘石（取药末）4两（白色微黄者佳，置坩锅内，炭火围烧，上用砂锅片盖锅口上，一放熟火煽至锅红，用钳取出，倾于童子小便内，沸定为度，依前再烧蘸，通7遍，乳缽内研极细无声，少添水再研，再添水飞，不尽者再研，倒水再如此数次，去沉底滓，待细药沉底，将碗慢倾去水，药上铺净绢帛，上铺净纸2重，上摊桑柴灰渗水，纸并灰频换，至水干，日中晒，或暖炕上盖放至干为度，用净纸收裹，若得隆德宫所出者，名曰京石，为最佳），黄丹（细腻，掷于壁上不落者良，飞，干称）4两，铜绿（若用开元钱硇砂撞者，又名峒青，更妙；不须水飞，种法在后），雄黄2钱半（色如鸡冠，不夹石而不臭者良，研飞如甘石法），青盐半两（走水淘去泥，漉出吹去水，控干，研为极细），麝香1字（真者，手捻著指细腻；又云味苦者良），轻粉2钱半（与麝香同研），硇砂2钱半（先净，微青者良，瓷盏盛水，将硇砂放水中，汤瓶盛水置火上，瓶口上坐盏，待硇砂滚开，水向上有凝结者，用錍儿挑出，待干秤，不须研），马牙消2钱半（研细。以上飞研药味，除甘石、黄丹各另收裹，其余细药一同和匀），鹅梨10个（用水洗净，控干，竹瓜篱内捺碎，用净生绢纽），蜜1斤（陈者良）。
出处：《医方类聚》卷七十引《经验秘方》。
主治：青濛遮暗，内外障，不见分明。
加减：None
功效：None
用法用量：先洗眼用，无瘀肉翳晕，不用麝香、轻粉、雄黄、硇砂、马牙消亦可。每用大如鸡头子，净瓷盏内用温水二杓浸开，重温药盏，初头夹净绵少许蘸药，临卧热洗，眼涩为度；病大者，1日洗2-3次，10日1次换药，先用热水洗眼净，随后用药甚效。如干点，将药另作如鼠粪状，一头置于眼大角内及上下睑，觉行性，去药亦可。黄连等先滤下药滓，晒干收藏。若有眼新发者，用多半抄水2大盏，同煎数沸；若滓澄清者，重汤内温热洗眼浊，服之亦效。
制备方法：以上淘浸、飞研药味，并洗涤器皿，俱用腊水极妙，煎熬，并用度法；上项浸药至7日，洁净房舍内，用净砂锅，文武炭火，旋添碎炭，将先浸黄连等3味倾于锅内，每添炭时用板盖锅，以防灰尘，黄连等滚数沸，下猪胰，用滚猪胰化，用绵隔去滓，滤于原浸药罐内，将蜜下锅，用细槐柳嫩枝各1条，夹去须葱1枝，细线系定，搅蜜，滚数沸，蜜红色，以绵隔去滓，滤干净碗内，重热汤内浸碗，取不凝也；将先滚过药汁；再用绵滤于锅内，滚数沸，下梨汁；滚数沸，下蜜；滚数沸，下甘石，搅匀；下研细药味，搅匀；下黄丹。用小指大长尺许带皮槐柳枝各2条，下削作马蹄状，掐底及四边，始终不住，慢慢左手顺搅，熬至不粘手为度，杖上粘药用竹刀刮下锅内，乘热倾于瓷盖缽内，凉处收藏；锅内粘住药，火上坐锅，少添蜡水，用槐柳枝刷濯3-5次，锅净即止，药水瓷罐内收盖，天寒窝藏。
临床应用：None
各家论述：None
用药禁忌：忌湿面、杂肉、酒、蒜、姜、醋发病之物。
