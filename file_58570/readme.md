序号：58585
名称：秘传加味枳术丸
组成：白术3两，枳实（麦麸炒）1两，苍术（米泔水浸2宿）1两，猪苓（去皮）1两，麦糵面（炒）1两，神曲（炒）1两，半夏（汤泡透）1两，泽泻（去毛）7分，赤茯苓（去皮）7分，川芎7分，黄连（陈壁土炒，去土）7分，白螺狮壳（煅）7分，草豆蔻5钱，砂仁（炒）5钱，黄芩（陈壁土炒，去土）5钱，青皮（去白）5钱，萝菔子（炒）5钱，干姜5钱，生姜5钱，陈皮（去白）3钱，香附子（童便浸）3钱，瓜蒌仁3钱，厚朴（姜汁制，炒）3钱，槟榔3钱，木香2钱，甘草2钱。
出处：《松崖医径》卷下。
主治：清痰、食积、酒积、茶积、肉积在胃脘，当心而痛，及痞满恶心，嘈杂呕吐。
加减：久病挟虚，加人参、白扁豆、石莲肉各5钱；吞酸，加吴茱萸汤泡，寒月5钱，热月2钱5分；时常口吐清水，加炒滑石1两，牡蛎粉5钱。
功效：None
用法用量：每服70-100丸，清米饮汤送下。
制备方法：上为细末，用青荷叶泡汤浸晚粳米捣粉作糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：加味枳术丸（《医学正传》卷四）。
