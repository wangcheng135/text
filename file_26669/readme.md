序号：26677
名称：延年草
组成：青橘皮4两（浸洗，去瓤），甘草2两（为细末），盐2两半（炒）。
出处：《养老奉亲》。
主治：None
加减：None
功效：通滞气，进食顺气。
用法用量：先洗浸橘皮，去苦水，微焙，入甘草同焙干，后入盐，每早晨嚼3两叶子。
制备方法：None
临床应用：None
各家论述：《寿亲养老评释》：延年草以橘皮、甘草、食盐三味制成。青橘皮辛苦而温，功能理气健脾，燥湿化痰，开胃消食，善治食、气停滞胃脘引起的心腹气痛，胀满，食欲不振，呕吐泄泻，以及咳嗽痰多等症，以其理气消食而不伤正，老人服用堪称良药；甘草补脾胃，润心肺，清火解毒，调和诸药；食盐调和脏腑，消宿食，滋五味，长肉，通利大小便，主肠胃结热，喘逆，胸中满，令人壮健。三药相合，为顺气进食强壮之佳品，不仅宜于老人春时服用，在一年四季用之，对腹胀少食者，亦可收和胃却病延年之效。
用药禁忌：None
附注：延年散（《遵生八笺》卷三）。
