序号：70655
名称：紫苏饮
组成：紫苏、白芍、陈皮、川芎、当归、甘草、黄耆、大腹皮、白术、乌药、木香、香附、厚朴、黄芩、葱白、艾。
出处：《陈素庵妇科补解》卷三。
主治：子悬。妊娠胎上逼心，胀痛闷绝。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：妊娠胎气上冲，动而复安，此其常也。至于胎逼神明，甚或胀急痛闷则危甚矣。是方专为胀痛而胎不安，故用术、陈、乌、厚、附、腹、紫苏以消胀定痛，而芎、归、艾、耆、术则所以安胎而兼补气血也。甘草以和之，配白芍而止痛；葱白以开之，合紫苏而胀除，一剂之后，胀除痛定，仍用四物、四君、杜、断、益、芩为中正不易之法。
用药禁忌：None
