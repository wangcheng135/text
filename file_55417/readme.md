序号：55432
名称：丁公仙枕
组成：真川椒1两，桔梗1两，荆实子1两，柏子仁1两，姜黄1两，吴茱萸1两，白术1两，薄荷1两，肉桂1两，川芎1两，益智仁1两，枳实1两，全当归1两，川乌1两，千年健1两，五加皮1两，蒺藜1两，羌活1两，防风1两，辛夷1两，白芷1两，附子1两，白芍1两，藁本1两，苁蓉1两，北细辛1两，猪牙皂1两，芜荑1两，甘草1两，荆芥1两，菊花1两，杜仲1两，乌药1两，半夏1两。
出处：《验方新编》卷九。
主治：None
加减：None
功效：种子，消百病，长精神，延年益寿。
用法用量：务要顶好鲜明，咀片，研为细末，绢袋盛之，用槐木薄板做枕1个，高3寸3分，宽4寸5分，长1尺2寸，如天盖地，一面上钻孔128个，如梧桐子大，用上药装入枕中。药料3-5个月1换。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：丁公，康熙时人，年逾七十无嗣，遇异人授此方，不二年精力强壮，至八十一岁已生二十一子矣。如夫妇皆以此作枕，更见奇效。
