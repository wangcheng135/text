序号：53314
名称：健脾化痰丸
组成：半夏2两半，南星2两半（切作十字块，以长皂角肉1两半，明矾1两半，煮汁20碗，浸2味，经宿取出，次日文武火煮透，劈开南星、半夏内无白点为度，去皂角不用，将2味切作薄片，晒干，称足4两），瓜蒌仁（去壳，另研）1两，黄连（姜汁炒）1两，陈皮（去白）1两，白茯苓（去皮）1两，枳实（面炒）1两，山楂子肉（去子，蒸）1两，萝卜子1两（炒），生甘草7钱半（去皮），白术2两（炒），紫苏子7钱，香附子1两（童便浸1宿），黄芩1两（酒炒），干姜5钱（新瓦上焙黑）。
出处：《摄生众妙方》卷六。
主治：湿热气熏蒸而成郁结五色有形之痰。
加减：None
功效：None
用法用量：每服80丸，食后、临卧白汤送下。
制备方法：上为细末，姜汁煮薄糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
