序号：68072
名称：黄耆汤
组成：黄耆3两，芍药3两，桂心3两，麦门冬3两，五味子1两，甘草1两，当归1两，细辛1两，人参1两，大枣20枚，前胡6两，茯苓4两，生姜8两，半夏8两。
出处：《千金》卷十九。
主治：虚劳不足，四肢烦疼，不欲食，食即胀，汗出。
加减：None
功效：None
用法用量：上（口父）咀。以水1斗4升，煮取3升，每服8合，每日2次。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：小建中、黄耆建中、内补建中三方合用，表里兼赅；加细辛、前胡祛风下气；半夏利膈除痰；人参、茯苓、麦冬滋培津气；五味子以收津；以无胶饴，所以革去建中之名，而曰黄耆汤。
用药禁忌：None
附注：None
