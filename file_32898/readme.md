序号：32907
名称：羌活大丸
组成：防风（去芦头）3钱，天麻（酒浸1宿，焙干）3钱，全蝎（去毒，炒）3钱，人参（去芦头）3钱，羌活（去芦头）2钱半，白茯苓（去皮）2钱半，青黛（别研）2钱半，独活（去芦头）2钱，麝香（别研）2钱，川芎1钱半，脑子（别研）1钱，水银1钱（入硫黄1钱同研结砂子）。
出处：《杨氏家藏方》卷二。
主治：男子、妇人一切痫疾，涎潮搐搦；中风涎壅，语言謇涩，手足不遂；小儿急慢惊风。
加减：None
功效：None
用法用量：大人服2丸，小儿服1丸，1岁以下服半丸，食后、临卧并煎荆芥汤化下；病发时，不拘时候。
制备方法：上为细末，炼蜜为丸，每1两作20丸，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：羌活大风丸、羌活丸（《普济方》卷一○○）。
