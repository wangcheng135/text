序号：62544
名称：淋浇黄连汤
组成：黄连1两，地骨皮1两，羌活1两，防风1两（去芦头），木通1两，甘草1两，白芷1两，川大黄1两，狼牙1两，川升麻1两，莽草1两，藁本1两，黄耆1两（锉），赤芍药1两，细辛1两，桑根白皮1两，黄芩1两，白矾1两，葱白1两，麻黄1两。
出处：《圣惠》卷六十一。
主治：发背痈疽，穿穴时久，坏烂，恶气不可近，出骨露筋，余毒未解，攻刺疼痛不可忍。
加减：None
功效：None
用法用量：每贴用水3升，煎取2升，去滓，温暖淋洗疮上。后以热巾拭干，以生肌膏贴之。
制备方法：上锉细，分为7贴。
临床应用：None
各家论述：None
用药禁忌：None
附注：黄连汤（《普济方》卷二八九）。
