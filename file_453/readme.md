序号：453
名称：五疳丸
组成：青黛（细研）1分，雄黄（细研）1分，麝香（细研）1分，芦荟（细研）1分，熊胆（研入）1分，胡黄连1分，黄连（去须）1分，龙胆（去芦头）1分，苦楝根1分，白鳝鱼（炙令焦黄）1分，虾蟆灰1分，蜗牛（炒令微黄）1分，夜明砂（微炒）1分，蟾头1枚（炙令黄焦），五倍子1分，青橘皮（汤浸，去白瓤，焙）1分，天浆子（内有物者，微炒）1分。
出处：《圣惠》卷八十六。
主治：小儿五疳，乳食不养肌肤，心腹胀满，或时下痢。壮热昏沉，眼涩口干，爱吃生冷，毛发干竖，揉鼻多嚏，日渐羸瘦。
加减：None
功效：None
用法用量：每服3丸。以粥饮送下，1日3次。
制备方法：上为末，都研令匀，用粳米饭为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
