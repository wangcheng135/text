序号：76705
名称：葆真丸
组成：豨莶草1斤6两，白蒺藜1斤6两，天冬8两，熟地8两，人参8两，茯神4钱，枣仁4钱，枸杞4钱，牛膝4钱，杜仲4钱，续断4钱，五加皮4钱，山药4钱，山萸4钱，白术4钱，菟丝饼4钱，沉香4钱，朱砂4钱，南星4钱，沙苑子4钱，半夏4钱，鹿茸4钱，虎胆4钱，乳香2两，没药2两，黄芩2两，山楂2两，龙骨2两，地龙2两，土鳖2两，甜瓜子2两，骨碎补2两，肉桂2两，附子2两，炙甘草2两。
出处：《全国中药成药处方集》（沈阳方）。
主治：五劳七伤，左瘫右痪，腰酸腿痛，倦怠无力，少腹窜痛，男子遗精，肾虚头晕，心脏衰弱，失眠自汗。
加减：None
功效：滋补强壮，去风湿，壮筋骨。
用法用量：每服1丸，黄酒送下。
制备方法：上研极细末，炼蜜为丸，2钱重。
临床应用：None
各家论述：None
用药禁忌：忌食生冷辣物。孕妇忌服。
附注：None
