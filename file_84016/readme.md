序号：84034
名称：五加皮丸
组成：五加皮半两，芍药半两，当归半两，大腹子（连皮）半两，芎半两，牛膝半两，陈皮半两，石南叶半两，薏苡仁半两，赤小豆半两，麻黄（去节）半两，杏仁半两，木瓜1两，独活1两，杜仲（炒）1两，萆薢1两，牵牛头末2两。
出处：《御药院方》卷六。
主治：风寒湿气合而成痹，遍身疼痛，难以转侧，筋脉拘挛，不能屈伸，及头目眩晕，心腹胀闷，小便赤涩，大便秘滞。脚弱不能行走。
加减：None
功效：None
用法用量：每服30-40丸，木瓜汤送下，不拘时候。
制备方法：上为细末，酒浸蒸饼为丸，如豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
