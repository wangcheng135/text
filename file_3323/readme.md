序号：3323
名称：天雄散
组成：天雄1两（炮裂，去皮脐），麻黄半两（去根节），当归半两（锉，微炒），白术半两，半夏半两（汤洗7遍，去滑），肉桂1两（去粗皮），川椒1分（去目及闭口者，微炒去汗），干姜3分（炮裂，锉），厚朴1两（去粗皮，涂生姜汁，炙令香熟），陈橘皮3分（汤浸，去白瓤，焙）。
出处：《圣惠》卷十一。
主治：阴毒伤寒，身重背强，腹中(疒丂)痛，咽喉不利，毒气攻心；心下坚强，短气呕逆，唇青面黑，四肢厥冷，其脉沉细。
加减：None
功效：None
用法用量：每服3钱，以水1大盏，加生姜半分，大枣3个，煎至5分，去滓，不拘时候，稍热服。如人行10里未汗，再服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
