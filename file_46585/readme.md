序号：46600
名称：千金消癖丸
组成：芦荟1钱，阿魏1钱（另为糊），青黛1钱，木香1钱，厚朴1钱（姜炒），槟榔1钱，陈皮（去白瓤）1钱，麦芽（炒）4钱，使君子2钱（去壳），胡黄连2钱，山楂肉2钱，香附2钱（水浸），三棱2钱（醋炒），莪术2钱（煨，醋炒），水红花子4钱（微炒），神曲（炒）4钱，人参3钱（去芦），茯苓（去皮），白术（去芦）3钱，甘草（炙）1钱。
出处：《回春》卷七。
主治：小儿癖疾、积块。
加减：None
功效：None
用法用量：每服40-50丸，米饮、白汤送下。
制备方法：上为末，将阿魏1钱，白水和面打糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
