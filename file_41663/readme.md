序号：41678
名称：金箔散
组成：第一炉：密陀僧1两，黄丹1两，水银3铢，金箔2叶_x000D_
上药先以盐泥固济炉子，着金箔1叶子于炉下，细研上药令极细，着在炉中，以金箔1叶盖上，以大火一煅，通赤为度，钤出良久，放冷。煅用炭3斤以来。_x000D_
第二炉：代赭半两，白矾3铢，硇砂3铢，消石1两半（明净者），蛇蜕3铢（着在炉底）_x000D_
上件药依前法以大火一煅，通赤为度，钤出良久，放冷。
出处：《鸡峰》卷二十一。
主治：喉闭、缠喉风，诸梗物，及一切喉痛。
加减：None
功效：None
用法用量：以笔筒盛少许吹在咽喉处，时饷再吹，沥涎梗物自出。
制备方法：上两炉煅者药，于乳钵内滚研令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
