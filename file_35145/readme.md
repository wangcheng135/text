序号：35155
名称：乳香丸
组成：糯米（炒）2两，川乌头（炮，去皮尖）2两，五灵脂（去砂土）2两，乳香（研）1两，白芷（锉）1两，藿香叶（洗）1两，天南星（炮）1两，没药（研）1两，荆芥（去枝梗）1两，赤小豆（生）1两，骨碎补（去毛）1两，白附子（炮）1两，松脂（研）半两，香墨（煅）5两，草乌头（炮，去皮脐）5两。
出处：《局方》卷一（宝庆新增方）。
主治：一切风疾，左瘫右痪，口眼斜，半身不遂，语言謇涩，精神恍惚，痰涎壅塞，手足軃曳，筋脉拘挛，或遍身顽痹，走注疼痛，脚膝缓弱，行步艰辛；又治打扑损伤，瘀血不散，痛不可忍；或行路劳伤，脚膝浮肿疼痛；或肾脏风毒，上攻面肿耳鸣，下注脚膝沉重。
加减：None
功效：None
用法用量：每服10-15丸，冷酒吞下，茶清亦得，不拘时候。
制备方法：上为细末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌热物一时辰。
附注：None
