序号：8606
名称：玄霜
组成：金50两，寒水石6斤（研如粉），磁石3斤（碎），石膏5斤（碎），升麻1斤，玄参1斤，羚羊角8两，犀角4两，青木香4两，沉香5两，朴消末6升，芒消6升，麝香当门子1两（后入）。
出处：《千金翼》卷十八。
主治：诸热、风热、气热、瘴热、癃，恶疮毒，内入攻心热闷；服诸石药发动、天行时气、温疫，热入脏俯，变成黄疸；蛇螫虎啮、狐狼毒所咬，毒气入腹内攻，心热；小儿热病，毒风脚气，热闷赤热肿、身上热疮；卒热淋，大小便不通，原有患热者。
加减：None
功效：None
用法用量：此药无毒，心热须利病出，用水3-4合和1小两，搅令消服之，炊久当利，两行即愈；小儿热病服枣许大；毒风脚气、热闷赤热肿，身上热疮，水渍少许，绵贴取点上，即愈，频与2服。病膈上热，食后服；膈下热，空腹服之。卒热淋，大小便不通，服1两。
制备方法：金、寒水石、磁石、石膏4味以两斛水煮取6斗，澄清；升麻、玄参、羚羊角、犀角、青木香、沉香6味切，纳上汁中，煮取2斗，澄清；朴硝末、芒消、麝香3味、纳前汁中渍1宿，澄取清，铜器中微微火煎取1斗2升，以匙抄看，凝即成，下，经1宿当凝为雪，色黑耳。若犹湿者，安布上晒干，其下水更煎，水凝即可停之如初，闭密器贮之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
