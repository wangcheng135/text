序号：66227
名称：铜镜鼻汤
组成：铜镜鼻18铢（烧，末），大黄2两半，干地黄2两，芍药2两，芎2两，干漆2两，芒消2两，乱发如鸡子大（烧）、大枣30枚。
出处：《千金》卷三。
主治：产后余疾，恶露不除，积聚作病，血气结搏，心腹疼痛。
加减：None
功效：None
用法用量：上（口父）咀。以水7升，煮取2升2合，去滓，纳发灰、镜鼻末，分3服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：产后恶露不除，日久而成积聚，虽有大黄、芒消、干漆、地黄之属，不能消磨坚积，故取铜镜之鼻以磨砺之，深契《本经》“主治女子血闭癥瘕”之旨；芍药、芎、发灰、大枣，乃干漆之佐使耳。
用药禁忌：None
附注：None
