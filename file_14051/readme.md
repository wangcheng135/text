序号：14058
名称：全蝎观音散
组成：石莲肉（炒，去心）2两半，白扁豆（炒）2两半，人参2两半，神曲（炒）2两，全蝎1两，羌活1两，天麻（去苗）1两，防风（去苗）1两，木香（炮）1两，白芷1两，甘草（炙）1两，黄耆（捶扁，蜜刷炙）1两，茯苓（去皮）1两半。
出处：《局方》卷十（续添诸局经验秘方）。
主治：小儿外感风冷，内伤脾胃，呕逆吐泻，不进乳食，久则渐渐羸弱及小儿脾虚自汗，多出额上，沾粘人手。
加减：None
功效：温养脾胃，进美饮食。
用法用量：婴儿1字，2-3岁半钱，4-5岁1钱，用水1盏或半盏，枣子半个或1个，同煎至7分，去滓服，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
