序号：42668
名称：保元固本膏
组成：党参1两5钱，白术（炒）1两5钱，鹿角1两5钱，当归1两5钱，香附1两5钱，川芎1两，附子（炙）1两，独活1两，干姜1两，川椒1两，杜仲1两，鳖甲1两，荜茇1两，草果仁1两，白芍1两，生耆1两5钱。
出处：《慈禧光绪医方选议》。
主治：脾肾不足，肠胃功能失调。
加减：None
功效：脾肾双补，肾阴阳同治，兼顾先后天之本。
用法用量：None
制备方法：用麻油3斤，将药炸枯，去滓，再熬至滴水成珠，入飞净黄丹1斤2两，再入肉桂、沉香、丁香各3钱（共为极细末，候油冷加），搅匀成坨，重4-5两，候去火气，3日后方可摊贴。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
