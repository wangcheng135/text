序号：26761
名称：异圣散
组成：猪蹄甲不以多少（净洗，去带毛处，锉碎，于锅内炒令烟尽，取出于地上厚铺纸，薄摊其药，以盆盖之，候3日出火毒毕，为细末），金星石半斤，银星石半斤，青（？？）石半斤（3石同人1盒内，以马粪和泥固济，晒干，先用熟火渐渐烧之，次添炭火煅令通赤，取出，仍先令掘1地坑，深1尺，取药盒在地坑内，用土盖埋2日出火毒，为细末）。
出处：《卫生总微》卷六。
主治：小儿痫病日发数次，诸药不能愈。
加减：None
功效：None
用法用量：每服半钱，以浆水研100遍调下，空心食前服。
制备方法：上用猪蹄末2两，石末1两半，再研匀细，入生麻油半两再研极匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
