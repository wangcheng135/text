序号：66510
名称：丁香丸
组成：丁香1分，木香1分，舶上茴香（微炒）1分，乳香（别研取末）1分，沉香1分，青橘皮（汤浸，去瓤，焙干）1两，肉豆蔻2两，槟榔2两（捶碎，用黑牵牛3两，同醋浸软，却用慢火炒令牵牛子熟，只使槟榔），蓬莪术2两（用生姜4两，细切，以醋浸令术软，切作片子，用慢火炒软），茱萸3分（水淘7遍，净去浮者，都用醋浸1两，取出慢火炒干，只用1分），阿魏1钱（面裹，烧面熟用）。
出处：《传家秘宝》卷三。
主治：气滞，心腹胀满疼痛，痰逆，不思饮食。
加减：None
功效：None
用法用量：每服15丸至20丸，嚼烂，空腹时用温酒或生姜汤送下，每日2次。
制备方法：上件为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
