序号：44123
名称：复寿延春丹
组成：人参2两，白茯苓2两，远志（去心）1两，牛膝（酒浸）1两，杜仲（炒去丝）1两，黄耆1两，菖蒲1两，当归1两，苍术（泔浸，去皮）1两，甘菊花1两，地骨皮1两，天麻半两，半夏（制）半两，生地黄半两，熟地黄半两，益智半两，萆薢（酒浸）半两，羌活半两。
出处：《普济方》卷&mdash;一五。
主治：左瘫右痪，口眼斜，半身不遂。
加减：None
功效：补五脏，分阴阳，止咳嗽，助声音，驻颜色，黑髭发，明耳目，不忘记，添精髓，壮筋骨，美饮食，去一切诸风。
用法用量：每服30-50丸，空心以温酒送下。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
