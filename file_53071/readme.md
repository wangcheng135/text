序号：53086
名称：鬼箭丸
组成：鬼箭羽2分，芍药2分，白术3分（炒），当归（洗过，锉，焙）2分，桂心2分，甘草2分，牡丹皮3分，川大黄4分，干地黄3分，虻虫3分（炒），蒲黄1分半（炒），乌梅肉、人参3分。
出处：《传家秘宝》。
主治：妇人血脉不通，欲变成劳，寒热不调，不思饮食，肌肤消瘦，心腹刺痛，手足沉重。
加减：None
功效：None
用法用量：每服2钱，以水2盏，加生姜，煎至6分后，再入酒3分，重煎2-3沸，去生姜服之，空心服之后，更吃酒投之。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中乌梅肉用量原缺。本方方名，据剂型，当作“鬼箭汤”。
