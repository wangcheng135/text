序号：31725
名称：沉香丸
组成：沉香（锉）1两，附子（炮裂，去皮脐）1两，厚朴（去粗皮，生姜汁炙）1两，白术1两，芎1两，肉豆蔻（去壳）1两，茴香子（微炒令香）1两，胡椒1两，陈曲（炒）1两，桃仁（去双仁皮尖，炒）1两，楝实（取皮肉，炒）2两，阿魏半两，硇砂（无石者佳）半两（研）。
出处：《圣济总录》卷四十六。
主治：脾胃气虚弱，肌体羸瘦。
加减：None
功效：None
用法用量：每服20丸，早晨、晚间腹空时，以茶酒盐汤任嚼下。
制备方法：除硇砂、阿魏外，上为末，将硇砂、阿魏用好酒3升，银铜石锅内熬成膏，和上药为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
