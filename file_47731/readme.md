序号：47746
名称：神仙灵砂丹
组成：水银4两，硫黄1两半。
出处：《圣济总录》卷二○○。
主治：五脏百病，营卫不交养，阴阳不升降，上盛下虚，头旋气促，心腹冷痛，翻胃吐逆，霍乱转筋，脏腑滑泄，赤白下痢。痰涎壅盛，诸虚痼冷。
加减：None
功效：延年益寿，悦颜色，坚脏腑，壮腰脚，益血固精。益精养神，神气明目，安魂魄，通血脉，止烦满，杀邪魅，久服通神，轻身不老。
用法用量：每服1丸，空心井水送下。直到中脘，旋下丹田，当觉温暖。
制备方法：上先熔开硫黄即投水银，以铁匙炒作青砂子，称定4两，如重再炒，去尽黄乃已；方用煅药盒子1只，口差小者，入青砂在内，用新茶盏1只，底差大，平净而厚者，盛新汲水7分许，安盛砂盒上，以细罗赤石脂末水拌作泥，厚粘外缝令周密，盒下坐熟火猛炎得所，微扇熁之，盏中水耗旋添，令常有水，约半日许，令火自冷，取出盏底成灵砂一簇，打下称得多少，未尽者再用火依前熁之，砂成以绢袋盛，水煮3-5沸，或浸半日，滤干细研如粉，水煮半夏糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌羊血；忌猪、羊血，绿豆粉，冷滑之物。
附注：灵砂（《局方》卷五续添诸局经验秘方）、灵砂丹（《得效》卷四）、灵砂丸（《医统》卷十四）、灵妙丹（《医宗必读》卷九）。《局方》（续添诸局经验秘方》本方用法；糯米糊为丸，如麻子大。每服三丸，空心枣汤、米饮、井华水、人参汤任下。
