序号：32374
名称：妙应癣药酒
组成：土槿皮2两，白芨1两5钱，槟榔1两，白芷1两，斑蝥40枚，白信4分（研末），伏龙肝4两。
出处：《集验良方》卷一。
主治：风热湿邪侵袭皮肤，郁久风盛化虫之干癣，搔痒白屑，湿癣搔痒出粘汁，浸淫如虫行；风癣顽癣，搔则痹顽，不知痛痒；牛皮癣，状如牛领之皮厚且坚；松皮癣，状如苍松之皮，红白斑点相连，时时作痒；刀癣、轮廓全无，纵横不定者。
加减：None
功效：杀虫燥湿消毒。
用法用量：临用时，取1-2两另装小瓷瓶内，以笔扫涂患上，每日3次。如涂后肿痛起泡，系药力猛，多搽之故，不必疑惧，加新鲜香糟火酒少许和之则平矣。
制备方法：用高粱酒3斤，或顶香糟烧，并药入瓷瓶内封固，浸7日可用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
