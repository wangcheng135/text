序号：72748
名称：款肺汤
组成：知母（焙）1两，百合1两，百部1两，白前1两，芍药1两，黄耆（锉）1两，款冬花1两，马兜铃1两，贝母（去心）1两，五味子1两，前胡（去芦头）1两，枳实（麸炒）1两，甘草（炙）1两，葛根1两，防己1两，青橘皮（汤浸，去白，焙）1两，防葵1两，大黄（生，锉）1两，麻黄（去根节）1两，桃仁（去皮尖双仁，焙黄）1两，白术（锉，炒）1两，升麻1两，紫菀（去苗土）1两，大枣（去核，焙）1两。
出处：《圣济总录》卷四十九。
主治：肺脏壅热，咳嗽多痰，面赤口干，气急烦满，大肠不利。
加减：None
功效：None
用法用量：每服3钱匕，水1大盏，煎至7分，去滓温服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
