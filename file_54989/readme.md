序号：55004
名称：桂枝一白虎二汤
组成：桂枝（去皮）3钱5分，芍药（去黄赤皮1层，用蜜水拌蒸3次、晒3次，焙干，锉碎）6钱，生姜（切）7钱，大枣4枚（擘），秔米1合（淘洗），甘草（去头尾尖处各3寸，切作5寸长，好酒浸1宿，柳火上缓缓炙，表里皆燥为度，锉碎）7钱5分，石膏（取洁白如束针者，研极细，用甘草水飞3遍，澄清去水，晒干再研）2两5钱，知母（槐砧上锉细，干木臼杵捣数千下，勿犯铁器）7钱5分。
出处：《痎疟论疏》。
主治：阳明疟疾。
加减：None
功效：None
用法用量：以水3升5合，先煮秔米，减半升，去秔米，纳诸药，煮取1升2合，分2次温服，未发、临发各服1次。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
