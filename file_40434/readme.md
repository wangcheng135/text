序号：40444
名称：虎挣散
组成：马钱子1斤，穿山甲2两，川附子2两。
出处：《中医外科学讲义》。
主治：骨关节结核，已溃未溃。
加减：None
功效：宣通经络，调和营卫，健脾和胃，消肿止痛，化阴为阳。
用法用量：根据年龄病情和体质不同，给服不同分量。小儿：1-3岁，2-3厘；4-6岁，3-5厘；7-12岁，5厘-1分。成人1-2分。饭后1小时，黄酒送服。
制备方法：马钱子用清水浸15天，夏季每隔1日换水一次，冬天用温水浸之，换水一次，刮净皮毛，切1分厚细条，投香油锅中，煎至油沫净，再煎数滚，透心黄脆，再放入黄土内炒拌，至土粉有油气，入筛内筛去油土，再换土粉炒，如是三次，油尽，取出马钱子研细。穿山甲砂土炒松脆，研细。川附子用水浸3天，1日换水一次，晒干，再研细。以上3味药同研细末。
临床应用：None
各家论述：None
用药禁忌：虚弱者，用量酌减。
附注：None
