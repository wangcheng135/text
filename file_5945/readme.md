序号：5945
名称：加减归芍地黄汤
组成：生地5钱，萸肉3钱，山药3钱（炒），茯苓1钱半，当归3钱，丹皮1钱半，白芍1钱半（炒），麦冬3钱（去心），五味1钱半。
出处：《医略六书》卷二十六。
主治：交接出血，脉虚数者。
加减：None
功效：None
用法用量：水煎，去滓温服。
制备方法：None
临床应用：None
各家论述：阴虚阳浮，经气不固，而经血易动，故交接出血。生地滋阴壮水，萸肉秘气涩精，山药补脾阴，茯苓和脾气，当归养血以资血室，白芍敛阴以固冲任，丹皮平相火以凉血，麦冬润心肺以交肾，五味敛津液以固经脉也。水煎温服，使阴平阳秘，则经气完固，而经血无妄泄之患，何致交接出血而成阴蚀哉。
用药禁忌：None
附注：None
