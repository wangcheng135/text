序号：58597
名称：秘传去翳圣金膏
组成：炉甘石5两（童便煅浸30次，却研极细，用黄连1两，龙胆草1两，当归3钱煎水2碗，飞过讫，重汤蒸干，再研约100次，要如面极细，其石须拣色白面极细微者，黄色只好合粉），乌贼鱼骨半两（研细入煎），黄连5两（用水洗净，晒干，却将1两切碎，煎水4两，研为细末，重罗过，再研极细，用水飞过，却于砂铫内煮此药，最难细，冬月用雪水洗净，晒干再研方细），乳香3钱（要通明滴乳，用黄连水飞过），密陀僧半两（火燃醋淬，研细，水飞过），轻粉1钱（研细入药），没药3钱（用黄连等水飞过），南硼砂1钱（研细入煎），白丁香1钱（水飞过，重汤煮干，再研入药煎），黄丹1两（用铁铫火煅过，研细末，水飞过，重汤蒸干，再研多时，须极细，入煎），硇砂半钱（净水洗去泥，以水入铁铫煮干，如盐样白方好，再研细入药煎），龙胆草1两（截碎，水煎），蜜4两（用水1盏干铜铫内煎，用葱白2茎搅蜜，候煎了放于地上，用纸1片蘸取去面上蜡），当归半两（净，以一半焙干，研细末，再用些水研入药，一半煎水用），鹰条1钱（以水1碗飞过，用研丁香、没药用水淘飞过，合研人煎，须多淘，净称）上先将当归、黄连、龙胆草3味，截碎，用铫子煎2大碗水。用此水研乳香、没药飞过，再用此水飞过鹰条、白丁香，独将黄连4两，洗净令干，碾为细末重罗过，又碾飞过，或别作法研磨，但要极细，于砂铫内，和净蜜4两同煮。却旋入诸药，煎成膏，可丸即止，惟独后入下2味：麝香半钱（重研细，罗过），脑子半钱（重研细罗过，候前药成膏，却入此2味）。
出处：《普济方》卷七十八引《海上方》。
主治：眼内外障，远年眼疾。
加减：None
功效：None
用法用量：随病轻重，以为大小丸与之。每以净汤1大鸡子壳化开，日温洗5-7次；或如麦粒大，点眼尤妙。
