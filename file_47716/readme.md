序号：47731
名称：神仙延寿药酒丹
组成：人参（去芦）3两，白术（土炒）3两，甘草（炙）3两，白茯苓3两，当归2两5钱，川芎2两5钱，白芍药（炒）2两5钱，生地黄（姜汁炒）2两5钱，熟地黄2两5钱，枸杞子2两5钱，肉苁蓉（酒洗）2两5钱，何首乌（米泔浸）2两5钱，牛膝（去芦）2两5钱，天门冬（去心）2两5钱，麦门冬（去心）2两5钱，砂仁（炒）2两5钱，川椒（去梗目）1两，川乌（去皮脐）1两，草乌（圆者，泡）1两，乌药1两，五加皮2两，虎胫骨（酥炙）2两，枳壳（炒）2两，干姜（泡）2两，厚朴（姜汁炒）2两，陈皮（去白）2两，沉香2两，茴香（盐酒炒）2两，香附（童便浸，炒）2两，羌活2两，独活2两，防风（去芦）2两，白芷2两，麻黄（不去节）2两，细辛（酒洗）2两，半夏（制）2两，苍术（米泔浸，炒）2两，五味子2两，破故纸（炒）2两，桂2两，红枣（去核）8两，酥油8两，蜂蜜8两，胡桃肉（汤泡，去皮）。
出处：《古今医鉴》卷二。
主治：久近风邪，左瘫右痪，语言謇涩，手足拘挛，紫白癜风，风寒暑湿四气交攻，身体虚羸，腰疼膝痛，耳聋眼瞶，下部诸虚；女子经血不调，脐腹绞痛，胸膨胁胀，呕吐恶心，子宫虚冷，赤白带下。
加减：None
功效：追万病，补诸虚，和胃养丹田，益精壮筋骨，安和五脏，定魄宁魂，返老还童，延年益寿。
用法用量：每日饮1-2杯，随病之上下以定空心或食后。酒将饮尽，复以药滓晒干为末，酒为丸，如梧桐子大。每服30丸，空心酒送下。
制备方法：上锉，绢袋盛之，用烧酒1大坛，浸3昼夜，置锅中重汤煮三时许，取出，埋土内泄火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中胡桃肉用量原缺。
