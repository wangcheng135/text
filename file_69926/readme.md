序号：69942
名称：琥珀茯苓丸
组成：琥珀5钱，浙茯苓2两，猪苓5钱，木瓜5钱，牛膝梢3钱，粉丹皮3钱，泽泻8钱，制香附1两，车前5钱（炒），肉桂5钱，淡附片1两，蓬术4钱（炒），三棱3钱，山楂肉8钱，(庶虫)虫10个（炒），神曲5钱，羌活5钱，独活5钱，麦芽5钱（炒），广木香5钱。
出处：《医方简义》卷六。
主治：产后腹胀暴肿，及七癥八瘕。妇人产后子宫瘀血，腹胀疼痛，恶露不尽或寒闭尿症。
加减：None
功效：利水导滞，活血消瘀，温经调气。
用法用量：每服1丸，开水冲化，或以水1碗煎服亦可，用陈酒化服更妙，1日1次。
制备方法：上为末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
