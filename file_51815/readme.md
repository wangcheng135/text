序号：51830
名称：钩藤丸
组成：钩藤1两，铅丹（研）1两，茵芋叶1两，石膏（研）1两，杜蘅1两，防葵（炙）1两，秦艽（去土）1两，甘草（炙）1两，菖蒲（九节者，去节，焙）1两半，黄芩（去黑心）1两半，松萝（炒）半两，蜣螂（去翅足，炒）半两。
出处：《圣济总录》卷十五。
主治：小儿风痫，至大不除，发即百脉挛缩，行步不正，口面戾，言语无度。
加减：None
功效：None
用法用量：每于食后良久服，5岁以下5丸，10岁以下7-10丸，15岁以上及长年并15丸，金银汤送下。
制备方法：上将10味为末，与别研2味和匀，炼蜜为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
