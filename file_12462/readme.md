序号：12468
名称：白薇丸
组成：白薇5分，牡蒙5分，藁本5分，当归7分，干地黄7分，芎藭6分，人参6分，柏子仁6分，石斛6分，桂心6分，附子（炮）6分，五味子6分，防风6分，吴茱萸6分，甘草（炙）6分，牛膝6分，桑寄生6分，姜黄7分，禹余粮8分，秦椒2分（汗）。
出处：《外台》卷三十三引《广济方》。
主治：久无子,胞络伤损，宿受风寒，久无子息，或受胎不牢，多致损堕。
加减：None
功效：补调冲任，温暖子宫，祛下脏风冷，令人有子。
用法用量：每服20丸，空腹酒送下，加至30丸，日2次。
制备方法：上药治下筛，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌生葱、生菜、热面、荞麦、猪肉、葵菜、芜荑、菘菜、海藻、粘食、陈臭物；才觉妊娠即住服，已怀孕者尤不宜服之。
附注：None
