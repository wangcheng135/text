序号：66744
名称：鹿茸天麻丸
组成：鹿茸（去毛，酥炙）2两，天麻1两半，附子（炮裂，去皮肤）1两，巴戟天（去心）1两，菖蒲1两，石斛（去根，锉）1两半，干蝎（去土，炒）1两，萆薢（锉）1两，桂（去粗皮）1两，牛膝（酒浸，切，焙）1两，天雄（炮裂，去皮脐）1两，独活（去芦头）1两，丹参1两，当归（切，焙）1两，杜仲（去粗皮，炙，锉）1两，肉苁蓉（酒浸，切，焙）1两半，磁石（煅，醋淬，细研，水飞过）1两。
出处：《圣济总录》卷二十。
主治：肾脏气虚，骨痹缓弱，腰脊酸痛，脐腹虚冷，颜色不泽，志意昏愦。
加减：None
功效：None
用法用量：每服20丸，加至30丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
