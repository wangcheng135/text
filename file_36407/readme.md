序号：36417
名称：和血益气汤
组成：柴胡3分，炙甘草3分，生甘草3分，麻黄根3分，酒当归梢4分，酒知母5分，酒汉防己5分，羌活5分，石膏6分，酒生地黄7分，酒黄连8分，酒黄柏1钱，升麻1钱，杏仁6个，桃仁6个，红花少许。
出处：《兰室秘藏》卷上。
主治：口干舌燥，小便数，舌上赤脉。二阳之病久而生燥，传为风消息贲，病在里者。及中消有汗，血受火邪者。
加减：None
功效：生津液，除干燥，生肌肉。
用法用量：上(口父)咀，都作1服。水2大盏，煎至1盏，去滓温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌热湿面、酒、醋等物；忌房事。
附注：地黄饮子（《东垣试效方》卷三）。本方方名，《医学纲目》引作“和血养气汤”。
