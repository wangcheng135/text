序号：39737
名称：经效散
组成：人参1两半，黄耆3两（饴糖炙），厚朴6钱（炒灰），茯苓1两半，龙骨3两（煅），麦冬3两（去心，糯粉炒），生姜7片，大枣10枚。
出处：《医略六书》卷三十。
主治：产后气虚下陷，痢久发渴，挟滞气而小腹作痛，寒热不止。
加减：None
功效：None
用法用量：每服3钱，米饮煎，去滓温服。
制备方法：上为散。
临床应用：None
各家论述：方中人参扶元以补气之虚，黄耆补气以举气之陷，厚朴灰散滞气以止痛，煅龙骨涩虚脱以止痢，茯苓渗湿和脾气，麦冬生津、润燥渴，姜、枣调和营卫也。为散，米饮下，使元气内充，则阳不复陷，而滞气自化，津液上敷，何有痢久发渴，腹痛寒热之患哉？
用药禁忌：None
附注：None
