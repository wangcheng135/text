序号：80697
名称：鲮鲤甲汤
组成：鲮鲤甲10片，鳖甲（炙）1两，乌贼骨（去甲）1两，常山3两，附子1枚（炮）。
出处：《外台》卷五引《小品方》。
主治：山瘴疟。南方山岭溪源瘴气毒作，寒热发作无时，萎黄肿满，四肢痹弱。
加减：None
功效：None
用法用量：以酒3升，渍1夕，先疟发前稍稍服之。勿绝药味，兼以涂身体。断杂人，勿食饮，过时乃得通人，进饮食。
制备方法：上切。
临床应用：None
各家论述：《千金方衍义》：疟邪既犯厥阴，故取专走肝经血肉之味，兼附子通行经脉之力，鼓舞常山迅扫痰涎，一涌而涣散无余，瘴疟之金錍无愈于斯也。
用药禁忌：忌苋菜、生葱、生菜、猪肉。
附注：鲮鲤汤（《千金》卷十）、鲮鲤甲酒（《圣济总录》卷三十七）。
