序号：46963
名称：独活寄生汤
组成：熟地5钱，独活1钱半，当归3钱，官桂1钱半，白芍1钱半（酒炒），川芎1钱，茯苓3钱，杜仲3钱（酒炒），牛膝3钱（酒炒），桑寄生3钱（酒炒）。
出处：《医略六书》卷三十。
主治：产后脚气疼痛，脉虚涩弦浮者。
加减：None
功效：None
用法用量：水煎，去滓温服。
制备方法：None
临床应用：None
各家论述：产后血室空虚，邪气陷伏而下注于脚，故脚弱疼痛，谓之脚气。熟地补血以荣肝肾，独活疏邪以宣通经络，当归养血荣经，白芍敛阴和血，川芎行血中之气，官桂散营中之寒，茯苓渗湿和脾，杜仲强腰补肾，寄生祛痹强腰脚，牛膝下行壮筋骨。水煎温服，使血气内充，则筋脉滋荣而寒邪自散。
用药禁忌：None
附注：None
