序号：22556
名称：吴茱萸丸
组成：吴茱萸（瓦上炒）3分，胡椒半两，人参半两，当归（锉，焙）半两，甘草半两（一半生用，一半纸裹5-7重，醋浸令透，火内慢煨干，又浸，如此7遍），半夏1两（用生姜4两研汁，入沙罐子内，姜汁并水煮，候擘破看存2分白心，取半夏细研为膏），白矾（烧存性）半两。
出处：《圣济总录》卷四十七。
主治：年深膈气翻胃，饮食之物至晚皆吐出，悉皆生存不化，膈上常有痰涎，时时呕血，胸中多酸水，吐清水无时，夜吐辄至晓，日渐羸瘦，腹中痛楚，时复冷滑，或即闭结。
加减：None
功效：None
用法用量：每服7丸，桑、柳枝各21茎，银器内煎汤吞下，日3次。
制备方法：上为细末，以半夏膏和丸，如稍硬，添姜汁为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌诸毒物，惟可食油煎猪服脾软饭。
附注：茱萸矾石丸（《普济方》卷三十六引《卫生家宝》）。
