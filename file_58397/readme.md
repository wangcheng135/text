序号：58412
名称：破证夺命丹
组成：人参1两（去芦，薄切）。
出处：《百一》卷七。
主治：伤寒坏证，元气大亏，阳气暴脱，喘息脉微，吐血咯血等。伤寒阴阳二症不明，或投药错误致患人困重垂死，7-8日以后皆可服。气虚喘急。面色苍白，神情淡漠，肢冷汗出，脉息微弱；近代也用于大出血，创伤性休克，心力衰竭等重症的抢救。
加减：None
功效：益气固脱。
用法用量：水1大升，银石器内煎至1盏，以新水沉之，取冷1服而尽。汗不自它出，只在鼻梁尖上，涓涓如水，是其应也。
制备方法：None
临床应用：时疫坏证申某之子妇，产后病时疫已20余日，已成坏证，偶见闻，因劝其只服1味人参遂安。
各家论述：None
用药禁忌：None
附注：人参汤（《直指》卷二十六）、破证夺命散、独参汤、独柱汤（《内经拾遗》卷一）、夺命独参汤（《普济方》卷一三三引《德生堂方》）、破证夺命汤（《普济方》卷一三五）、坏证夺命散（《丹溪心法附余》卷一）、夺命散（《医统》卷七十六）。①本方方名，《本草纲目》引作“复脉汤”。②《内经拾遗》本方用法：水二钟，红枣十个，煎八分，食后温服。
