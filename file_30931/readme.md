序号：30940
名称：十将军丸
组成：三棱1两（去毛土，炮），莪术（生）1两，青皮（去白）1两，陈皮（去白）1两，草果（去壳）2两，川常山2两，砂仁1两，槟榔1两，乌梅1两，半夏（汤泡7次）1两。
出处：《丹溪心法附余》卷六。
主治：久疟不愈，腹痛，有疟母。
加减：None
功效：None
用法用量：将常山、草果2味锉，用好酒、醋各1碗，入瓦器内，先浸1宿，后入8味药，同浸至晚，用瓦铫内炭火煮干，取出晒，如无日色，用火焙干为末，半酒半醋打糊为丸。如梧桐子大。每服30-40丸，白汤吞下，日3次。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌生冷、鱼腥、咸、酸、油腻、面、诸死毒物。
附注：None
