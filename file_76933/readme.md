序号：76949
名称：黑神丸
组成：乌头1两，草乌（并炮，去皮）1两，芎1两，香白芷1两，白僵蚕1两，羌活1两，甘草1两，灵脂（净洗）1两（修事洗净，一处焙，研为末），好墨1寸（同药为末），麝香1字。
出处：《幼幼新书》卷九引《张氏家传》。
主治：头风，小儿惊风，伤寒，身上生疮，肠风痔疾，妇人血气、血风。
加减：None
功效：None
用法用量：药使如后：头风，茶汤嚼下1丸；伤寒，生姜、葱、茶嚼下1丸；身上生疮，蜜酒嚼下1丸；肠风痔疾，煎胡桃酒嚼下1丸；妇人血气、血风，当归汤嚼下1丸；小儿惊风，薄荷水磨下，每1丸为2服；头痛，菊花酒嚼下1丸；老人常服以好酒嚼下1丸。
制备方法：上为细末，用糯米2两研为末，煮糊为丸，阴干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
