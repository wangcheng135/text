序号：76804
名称：黑锡丹
组成：黑锡（4两，新铫中以火熔开，用香匙撇去锡滓，入硫黄半两同炒过，取出研为细末）2两，硫黄末2两（上2味加入铫中炼成汁，取下铫，候冷再见火，如此3次，候冷，研为细末），舶上茴香（略炒）1两，附子（炮，去皮脐）1两，木香（略炒）1两，川楝子肉（炒）1两，破故纸（炒）1两，肉豆蔻（面裹煨）1两。
出处：《杨氏家藏方》卷九。
主治：上实下虚，真元衰败，阳气耗损，阴气独盛，上喘气促，泄泻呕逆，自汗心忡，小便频数，一切虚寒。
加减：None
功效：None
用法用量：每服50丸，病势重加至100丸，空心、食前温米饮送下。服完以少干物压之。
制备方法：上为细末，却将黑锡末一处和匀，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：黑锡丸（《普济方》卷一二○）、乌金丸（《普济方》卷一二○）。
