名称：神异膏
组成：木香1两，川芎1两，牛膝1两，生地黄1两，细辛1两，白芷1两，秦艽1两，归尾1两，枳壳1两，独活1两，防风1两，大枫子1两，羌活1两，黄芩1两，南星1两，蓖麻子1两，半夏1两，苍术1两，贝母1两，赤芍1两，杏仁1两，白敛1两，茅根1两，两头尖1两，艾叶1两，连翘1两，甘草节1两，川乌1两，肉桂1两，良姜1两，续断1两，威灵仙1两，荆芥1两，藁本1两，丁香1两，金银花1两，丁皮1两，藿香1两，红花1两，青风藤1两，乌药1两，苏木1两，玄参1两，白鲜皮1两，僵蚕1两，草乌1两，桃仁1两，五加皮1两，山栀子1两，牙皂1两，苦参1两，穿山甲1两，五倍子1两，降真香1两，骨碎补1两，苍耳头1两，蝉退1两，蜂房1两，鳖甲1两，全蝎1两，麻黄1两，白及1两，大黄、蜈蚣21条，蛇退3条。
出处：《寿世保元》卷九。
主治：一切风寒湿气，手足拘挛，骨节酸疼，男子痞积、妇人血瘕，及腰胁诸般疼痛，结核瘰疬，顽癣顽疮，积年不愈，肿毒初发，杨梅肿块，腹痛泻痢，疟疾，咳嗽哮喘，受寒恶心，胸膈胀闷，面色微黄，心疼气痛，负重伤力，浑身痛，小肠气。
加减：None
功效：None
用法用量：肿毒初发，杨梅肿块未破者，俱贴患处；肚腹疼痛，泻痢、疟疾，俱贴脐上，利白而寒尤效；咳嗽哮喘，受寒恶心，胸膈胀闷，面色微黄，心疼气痛，俱败前心；负重伤力，浑身痛者，贴后心；腰眼痛、小肠气等症，贴脐下。
制备方法：上用桃、槐、榆、柳、楮、桑、楝七色树枝，各三七二十一，共俱切粗片，用真麻油17斤浸药，夏3宿、春5、秋7、冬10宿后，煎药枯油黑为度，用麻布滤去滓，贮瓷器内，另以松香不拘多少，先下净锅溶化后取起，每香2厅，用药油4两，搅匀，软硬得法，仍滤入水缸中，令人扯抽，色如黄金，即成膏矣。
