序号：3480
名称：太白丹
组成：天南星20两（炮），细辛（去土）2两，附子（炮，去皮脐）2两，芎2两半，天麻2两半，半夏15两（汤浸，洗去滑，切作片子，焙干），白附子5两（炮），蝎梢1两（炒），青皮（去白）3两，木香3两，寒水石（烧）10两（1半为衣），白僵蚕（炒去丝）3两。
出处：《御药院方》卷一。
主治：诸风头目旋晕，偏正头痛，肢体拘踡，痰盛气壅，鼻塞声重，咽膈不利。
加减：None
功效：清爽神志，解利四时邪气。
用法用量：每服30丸，生姜汤送下，不拘时候。
制备方法：上为细末，生姜汁面糊为丸，如梧桐子大，用寒水石为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
