序号：44892
名称：挺子膏
组成：附子1两（去皮脐，生用），赤芍药1两，当归1两，杏仁2两（汤浸，去皮尖双仁），黄连1两，赤柳皮4两，麒麟竭1两，没药1两，黄丹3两，清油2斤。
出处：《圣惠》卷六十三。
主治：一切痈疽恶毒疮痛。
加减：None
功效：None
用法用量：多年冷漏恶疮，先用甘草煎水洗，然后贴之。痈肿，每服1丸如弹子大，皂角酒调下。齿断痈肿贴之。
制备方法：上细锉，先将清油及诸药入于铛中，煎令焦黄色，待冷澄滤过，后下黄丹、麒麟竭、没药同煎，以柳木篦子不住手搅，候黑色，取少许滴水中成珠子即膏成，放冷，剂作挺子。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中没药用量原缺，据《医方类聚》补。
