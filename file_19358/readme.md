序号：19365
名称：地骨皮散
组成：地骨皮1两，柴胡1两（去苗），白茯苓半两，桑根白皮3分（锉），五加皮半两，人参半两（去芦头），黄耆3分（锉），甘草半两（炙微赤，锉），桂心半两，白芍药半两，前胡3分（去芦头），枳壳3分（麸炒微黄，去瓤）。
出处：《圣惠》卷七十。
主治：妇人血风，气体虚，发歇寒热。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：《济阴纲目》汪箕笺：黄耆、人参、桂心是补肺温气药，桑皮、枳壳、地骨皮是泻肺清热药，白芍、茯苓是降收之味，五加、前胡是行散之味，而乃并用之，重在气虚有火，当以分两中求之，疾苦中审之，则窍妙自得。
用药禁忌：None
附注：None
