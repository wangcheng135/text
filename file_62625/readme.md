序号：62641
名称：清中汤
组成：黄连1钱半，茅术1钱半（炒），黄柏1钱半，于术1钱半（炒），黄芩1钱半，泽泻1钱半，神曲3钱，木香1钱半，葛根1钱半。
出处：《医略六书》卷三十。
主治：膏粱积热痢，脉缓数者。
加减：None
功效：None
用法用量：水煎，去渣温服。
制备方法：None
临床应用：None
各家论述：产后素享膏梁，纵恣口腹，故热积肠胃，腹痛而下痢黄白焉。黄连清心脾之火，黄芩清肺肠之火，黄柏清肾膀之火，茅术燥肠胃之湿，泽泻通利膀胱，神曲消化积滞，于术壮胃健脾，木香醒脾开胃，葛根分解阳明之湿热也。水煎，温服，使积热消化，则脾气内强而胃气亦化，积滞无不一空，何腹痛不退，下痢不痊乎？
用药禁忌：None
附注：None
