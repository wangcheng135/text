序号：74569
名称：新加白虎汤
组成：苏薄荷5分（拌研），生石膏8钱，鲜荷叶1角（包），陈仓米3钱，白知母4钱，益元散3钱（包煎），鲜竹叶30片，嫩桑枝2尺（切寸）。
出处：《重订通俗伤寒论》。
主治：热汗烦渴，皮肤隐隐见疹，溺短赤热，甚则咳血昏狂。
加减：None
功效：清肝胃，辛凉心肺。
用法用量：先用活水芦笋2两，灯芯5分，同石膏半分，先煎代水。
制备方法：None
临床应用：None
各家论述：何秀山：本方以白虎汤法辛凉泄热，甘寒救液为君，外清肌腠，内清脏腑；臣以芦笋化燥金之气，透疹(疒咅)而外泄，益元通燥金之郁，利小便而下泄；佐以竹叶、桑枝通气泄热；使以荷叶、陈米清热和胃。妙在石膏配薄荷拌研，既有分解热郁之功，又无凉遏冰伏之弊，较长沙原方尤为灵活，此为辛凉甘寒，清解表里三焦之良方。
用药禁忌：None
附注：None
