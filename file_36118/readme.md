序号：36128
名称：参桂鹿茸丸
组成：茯苓80两，白芍80两，熟地80两，生地80两，鹿茸（去毛）80两，龟版（炙）40两，杜仲炭40两，秦艽40两，艾炭40两，山萸肉（炙）40两，泽泻40两，橘皮40两，续断40两，鳖甲（炙）40两，没药（炙）40两，枣仁（炒）40两，人参（去芦）40两，元胡（炙）30两，红花30两，石脂（煅）30两，红白鸡冠花60两，乳香（炙）30两，甘草20两，琥珀20两，阿胶（炒）120两，牛膝46两，黄芩50两，天冬54两，香附（炙）120两，川牛膝40两，藏红花20两（上药用黄酒1550两，入罐蒸3昼夜），当归80两，砂仁40两，肉桂（去粗皮）40两，白术（炒）60两，川芎60两，橘皮120两，沉香10两，木香10两。
出处：《北京市中药成方选集》。
主治：气虚血亏，身体衰弱，精神萎靡，腰膝痠软。
加减：None
功效：滋阴补肾，益气养荣。
用法用量：每服1丸，温开水送下，1日2次。
制备方法：上药中后8味，共串粗末，拌蒸下罐药料晒干，共研为细粉过罗，炼蜜为丸，重3钱。
临床应用：None
