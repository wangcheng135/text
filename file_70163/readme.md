序号：70179
名称：滋阴清热汤
组成：大生地6钱，熟地5钱，知母3钱，黄柏3钱，寸冬3钱，葛花4钱，地骨皮3钱，土茯苓3钱，菊花3钱，蔓荆子3钱，木贼3钱，甘草1钱，白蒺藜3钱（炒），车前子3钱（炒）。
出处：《眼科临症笔记》。
主治：聚散障症（角膜玻璃变性）。症见大小眺微带红色，惟黑珠上云翳层层，方圆无定，聚散不一，亦不疼痒，视物昏花。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：聚散障症：东明县萧某某，男。因素日嗜酒，肝火上攻于脑，头晕目赤，翳膜始生，经当地西医调治，半年稍安。年四十，嗜酒如故，肝火又起，翳膜往来不定，隐涩羞明。诊其脉，太阳虚数，厥阴浮数。是知膀胱之虚热，不能滋养肝经之木，邪火乘势上冲于脑，以致头疼目痠，点翳四起，视物昏蒙。初服活血解毒汤十剂，视力稍有加增，又改服滋阴清热汤年余，翳膜虽未退净，而视力无碍。
