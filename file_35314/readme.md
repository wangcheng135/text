序号：35324
名称：乳香煎丸
组成：乳香1两，没药1两，丹砂1两，木香1两，沉香1两，蓬莪荗（煨，锉）1两，枳壳（去瓤，麸炒）1两，槟榔（锉）1两，乌头（炮裂，去皮脐）1两，硇砂（别研，水飞）1两半，狼毒（锉，醋炒）1两半，干漆（炒烟出）1两半，阿魏1分（醋化，去砂石，面和作饼，炙），楝实（取肉，捣末）4两，芫花（醋炒焦）半两，青橘皮（汤浸，去白，焙）3分。
出处：《圣济总录》卷四十四。
主治：脾胃虚寒，心胸痞满，宿食不消。
加减：None
功效：None
用法用量：每服15丸，临卧温水送下。
制备方法：将乳香、没药、丹砂、硇砂别研，与楝实末同入石器内，用酽醋1升，慢火熬成膏，即以余药为末拌和令匀，为丸如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
