序号：40075
名称：肾沥汤
组成：磁石2两（捣碎，水淘去赤汁，以帛包之），肉苁蓉1两（酒浸，去皱皮，微炙），人参3分（去芦头），附子1两（炮裂，去皮脐），黄耆3分（锉），熟干地黄1两，桑螵蛸1两（微炒），桂心3分，石南3分，五味子3分，白龙骨3分，白茯苓3分。
出处：《圣惠》卷七。
主治：肾脏风虚耳鸣，四肢羸瘦，小便滑数，夜卧多寒，吃食减少。
加减：None
功效：None
用法用量：每服5钱，以水1大盏，用羊肾1对（切去脂膜），生姜半分，枣3枚，与磁石包子同煎至5分，去滓，空心及晚食前温服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
