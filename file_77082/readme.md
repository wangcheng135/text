序号：77098
名称：黑龙妙化膏
组成：川乌1两，草乌1两，当归1两，白芷1两，赤芍1两，生地1两，熟地1两，两头尖1两，官桂1两，三棱1两，莪术1两，穿山甲1两，木鳖子（去壳，净仁）1两，巴豆（去壳）100个，蓖麻仁100个上锉碎，用香油2斤，浸3日，文武火熬至焦黑，滤去滓，将油再熬至半炷香，下黄丹炒黑色1斤，研，同熬，以柳条搅不住手，滴水成珠，不散为度，取出入后药，乳香1两，没药1两，木香1两，麝香2钱，五灵脂1两。
出处：《古今医鉴》卷十三。
主治：癖块血积、气积、疳积、食积。
加减：None
功效：None
用法用量：量疾大小，用五倍子染过狗皮，摊贴半月，1易制药，2-3个月有效。
制备方法：上为细末，入内搅匀，瓷器盛之。
临床应用：None
各家论述：None
用药禁忌：忌食羊鱼等肉发物。
附注：None
