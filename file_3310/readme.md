序号：3310
名称：天雄沉香煎丸
组成：天雄4两（生用，锉碎），防风2两（生用），紧小黑豆2两（净拭，生用），汉椒4两，草乌头4两（生用），附子4两（生用），牛膝2两，沉香2两（生用），天麻2两（生用），丁香1两，木香1两，羌活1两，干姜1两，官桂3两（去皮），肉苁蓉3两（酒浸，去土，炙熟），紫巴戟2两（去心）。
出处：《博济》卷一。
主治：下元积冷伤惫，阳事不能，筋骨无力，或成下坠，及小肠气痛，并肾藏风毒攻注，脾胃不和，腰脚沉重。
加减：None
功效：明耳目，雄气海，驻颜色。
用法用量：每日20丸，空心温酒送下，加至30丸。
制备方法：上药先将前9味以无灰酒1斗于银锅内慢火煨，不得令大沸，酒尽为度，焙令干；再与后7味同杵为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
