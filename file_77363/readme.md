序号：77379
名称：遇仙补寿丹
组成：蝙蝠10个（捣烂，晒干），紫黑桑椹4升（取汁，滓晒干），杜仲6两，童子发6两，天门冬3两，黄精（蜜蒸晒9次）4两，何首乌4两，熟地4两，川椒4两，枸杞2两，当归2两（为末），旱莲草（为末）4两，秋石丹（为末）4两，玄胡索（为末）4两。
出处：《医学入门》卷七。
主治：None
加减：None
功效：补经络，起阴发阳，开三焦，闭精气，消五谷，益血脉，安五脏，除心热，和筋骨，去盗汗，驻颜乌发，轻身健体，夜视有光。
用法用量：每服不拘多少，随便饮下。
制备方法：用桑椹汁拌3味，晒蒸3次，酒煮3味，打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌萝卜。
附注：None
