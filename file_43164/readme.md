序号：43179
名称：三棱消积丸
组成：三棱3两（醋炒），蓬术3两（醋炒），于术3两（炒），枳实1两半（炒），香附3两（醋炒），延胡2两（酒炒），肉桂1两半（去皮），泽泻1两半，木香1两半。
出处：《医略六书》卷三十。
主治：产后五积，脉紧涩结滞者。
加减：None
功效：None
用法用量：每服3钱红花子煎汤送下。
制备方法：上为末，(庶虫)虫汁为丸，
临床应用：None
各家论述：产后夹气受寒，血气不化，而成五脏之积，或瘀痛不止，或腹胀不已。三棱破气中之血，蓬术破血中之气，二物俱消坚削积之剂；协之于术健脾运化以培本；枳实破滞消结以治标；香附调气解郁，治积之由来；延胡活血通经，治积之成就；木香调中气以解郁；泽泻泻浊阴以分化；肉桂暖血祛寒，以通闭结也。(庶虫)虫汁丸以活血，红花子汤以散其结，使寒散结消，则积块自化，脾气健运，安有胀满疼痛之患乎。
用药禁忌：None
附注：None
