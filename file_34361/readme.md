序号：34370
名称：万金散
组成：续断1两，杜仲（去粗皮，炙香，切）1两，防风1两，牛膝（酒浸，焙）1两，细辛（华阴者）1两，白茯苓1两，人参1两，桂（去粗皮，取有味处，不见火）1两，当归（切，焙）1两，甘草（炙）1两，川芎半两，独活半两，秦艽（去土）半两，熟干地黄半两。
出处：《医方类聚》卷二十引《叶氏录验方》。
主治：感中风邪，腰膝沉重，脚弱无力。
加减：None
功效：祛风补虚，顺荣卫，通血气。
用法用量：每服5钱匕，水2盏，煎至1盏；滤去滓，空心热服，不拘时候。
制备方法：上为粗末。
临床应用：手指无力：余尝左臂不随，后己痊愈，而手指不便，无力，试诸药不验，遂服此药，才半剂而愈。
各家论述：None
用药禁忌：None
附注：万金汤（《得效》卷十三）。
