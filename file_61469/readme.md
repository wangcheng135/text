序号：61485
名称：培荣滑胎散
组成：当归2两，川芎1两，熟地，白芍（酒炒）1两，丹参1两，肉桂1钱，生芝麻3钱（生捣），益母草2两，冬葵子（研）2钱，广皮1钱，香附（酒炒）1钱。
出处：《陈素庵妇科补解》卷四。
主治：沥浆生。胞破浆水先来，或1-2日，或2-3日，胎竟不下。
加减：None
功效：大补气血以助浆血。
用法用量：浓煎恣饮，再煎葱酒熏洗产户，令气通畅。
制备方法：None
临床应用：None
各家论述：此时孕妇且惊且惧，气结体疲，惟大补气血，以助其精神，逐瘀行血，和气滋营，产妇精神充足，仍可坐草。努力不能，血枯干闭。是方四物以大补阴血，丹参逐瘀生新，益母和营养血，冬葵、芝麻以滑胎，广皮、香附以破滞，肉桂辛热，使药性直入血分，引热下行，庶几有补于万一云尔。
用药禁忌：None
附注：None
