序号：82205
名称：薏苡附子败酱散
组成：薏苡仁10分，附子2分，败酱5分。
出处：《金匮》卷中。
主治：肠痈之为病，其身甲错，腹皮急，按之濡如肿状，腹无积聚，身无热，脉数，此为肠内有痈脓。
加减：None
功效：排脓消肿。
用法用量：每取方寸匕，以水2升，煎减半，顿服。小便当下。
制备方法：上为末。
临床应用：None
各家论述：1.《金匮玉函经二注》：血积于内，然后错甲于外，经所言也。肠痈何故亦然那？痈成于内，血泣而不流也。惟不流，气亦滞，遂使腹皮如肿，按之仍濡。虽其患在肠胃间，究非腹有积聚也。外无热而见数脉者，其为痈脓在里可知矣。然大肠与肺相表里，府病而或上移于脏，正可虞也。故以保肺而下走者，使不上乘。附子辛散以逐结，败酱苦寒以祛毒而排脓。务令脓化为水，仍从水道而出，将血病解而气亦开，抑何神乎。_x000D_
2.《金匮要略心典》：薏苡破毒肿，利肠胃为君；败酱一名苦菜，治暴热火疮，排脓破血为臣；附子则假其辛热以行郁滞之气尔。
用药禁忌：None
附注：附子汤（《圣济总录》卷一二九）、薏苡附子散（《准绳·疡医》卷二）、败酱散（《校注妇人良方》卷二十四）、薏苡败酱汤（《张氏医通》卷十四）。本方方名，《本草纲目》引作“薏苡仁附子败酱散”，《中国医学大辞典》引作“薏苡败酱散”。
