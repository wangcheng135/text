序号：39025
名称：泽兰丸
组成：泽兰叶1两半，芎1两半，牛膝（酒浸，切，焙）1两，防风（去叉）1两，禹余粮（煅，醋淬）1两，白茯苓（去黑皮）1两，附子（炮裂，去皮脐）1两，黄耆（锉）1两，芍药1两，当归（酒浸，切，焙）1两，柏子仁（研）半两，蜀椒（去目并合口，炒出汗）半两，桃仁（去皮尖双仁，炒，研）半两，桂（去粗皮）半两，木香半两，牡丹皮半两。
出处：《圣济总录》卷一五○。
主治：妇人风虚劳冷，四肢困倦，面色萎黄，经水不调，饮食减少。
加减：None
功效：补暖血脏。
用法用量：每服30丸，空心、日午、临卧以酒送下。
制备方法：上14味为末。与2味研者和匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
