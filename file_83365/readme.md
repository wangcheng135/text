序号：83383
名称：乌鸡煎丸
组成：乌雄鸡1只，人参（去芦）1两，白术1两，石床1两，牡丹皮1两，黄耆1两，乌药1两，草果半两，延胡索半两，熟干地黄（洗，焙）半两，木香半两，琥珀半两，肉豆蔻半两，陈皮2两，红花2两，川乌（炮）2两，海桐皮2两，白芍药2两，附子（炮，去皮脐）2两，肉桂（去粗皮）2两，蓬莪术2两，苍术（米泔浸，切，焙）1两半。
出处：《局方》卷九(续添诸局经验秘方)。
主治：妇人胎前产后诸般疾患。
加减：None
功效：None
用法用量：每服30丸，胎前产后伤寒，蜜糖酒送下；胎前气闷壮热，炒姜酒送下；赤白带下，生姜、地黄煮酒送下；产后败血攻心，童便炒姜酒送下；产后血块攻筑，心腹疼痛，元胡酒送下；胎前呕逆，姜汤送下；催生，炒蜀葵子酒送下；安胎，盐酒送下；室女经脉当通不通，四肢疼痛，煎红花酒送下；血气攻刺，心腹疼痛，煎当归酒送下；血晕，棕榈烧灰酒送下；血邪，研朱砂、麝香酒送下；血闷，煎乌梅汤研朱砂送下；子宫久冷，温酒或枣汤送下，空腹日1服；血风劳，人参酒送下；心腹(疒丂)痛，炒茴香盐酒送下；血散四肢，遍身虚浮黄肿，赤小豆酒送下；常服，温酒、醋汤任下，并空心食前服。
制备方法：上锉细，用乌鸡1只，汤挦去毛及肠肚，将上件药，安放鸡肚中，用新瓷罐、好酒1斗，同煮令干，去鸡骨，以油单盛，焙干为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：大乌鸡丸（《普济方》卷三二七）。
