序号：82257
名称：濒湖白花蛇酒
组成：白花蛇1条（取龙头虎口，黑质白花，尾有佛指甲，目光不陷者为真，以酒洗润透，去骨刺，取肉4两），真羌活2两，当归身2两，真天麻2两，真秦艽2两，五加皮2两，防风1两。
出处：《本草纲目》卷四十三。
主治：中风伤酒，半身不遂，口眼斜，肤肉(疒帬)痹，骨节疼痛；及年久疥癣，恶疮风癞。
加减：None
功效：None
用法用量：每饮1-2杯。仍以滓晒干，碾米酒糊丸，如梧桐子大。每服50丸，用煮酒送下。
制备方法：各锉匀，以生绢袋盛之，入金华酒坛内，悬起，安置入糯米生酒酪五壶侵袋，箬叶密封，安坛于大锅内，水煮1日取起，埋阴地七日取出。
临床应用：None
各家论述：None
用药禁忌：忌见风犯欲及鱼羊鹅面发风之物。
附注：None
