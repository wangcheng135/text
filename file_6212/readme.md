序号：6212
名称：加味茱萸内消丸
组成：吴茱萸（半酒半醋浸1宿，焙干）1两，舶上茴香（盐炒）1两，山茱萸（去核）1两，马兰花（醋洗，焙）1两，川栋子（取肉）1两，官桂1两，玄胡索（略炒）1两，黑牵牛（炒，取头末）1两，橘红1两，青皮1两，海藻（洗去咸盐）1两，桃仁（浸，去皮尖）半两，白蒺藜（炒，去粗）半两，木香半两。
出处：《保命歌括》卷十六。
主治：肾虚为邪所侵，留伏作痛，阴(疒颓)偏大，或生湿疮，出脓水。
加减：None
功效：None
用法用量：每服40丸，食前温酒、盐汤下。
制备方法：上为末，酒面稀糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
