序号：31246
名称：沉香牡丹丸
组成：沉香（锉）1两半，牡丹皮1两，赤芍药1两，当归（切，焙）1两，桂（去粗皮）1两，芎1两，黄耆（锉）1两，人参1两，白茯苓（去黑皮）1两，山芋1两，白芷1两，吴茱萸（汤浸，焙干，炒）1两，巴戟天（去心）1两，陈橘皮（汤浸，去白，焙）1两，木香1两，牛膝（去苗，酒浸，切，焙）1两，枳壳（去瓤，麸炒）1两，肉豆蔻（去壳）1两，厚朴（去粗皮，生姜汁炙）1两，干姜（炮）1两，白龙骨1两。
出处：《圣济总录》卷一五二。
主治：妇人血海久虚，经候不利，赤白带下，血气冲心，多发刺痛，四肢困烦。
加减：None
功效：None
用法用量：每服20丸，加至30丸，空心、日午、临卧温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中白术，《准绳·女科》作白芷。
