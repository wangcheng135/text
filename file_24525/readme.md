序号：24533
名称：老君益寿散
组成：天门冬5两（去心，焙），白术4两，防风1两（去芦头），干姜1两半（炮裂，锉），熟干地黄2两，细辛1分，桔梗1两（去芦头），天雄半两（炮裂，去皮脐），远志（去心）1两，肉苁蓉（酒浸，去皱皮）1两，泽泻1两，石斛（去根，锉）半两，桂心半两，柏实半两，云母粉半两，石韦（去毛）半两，杜仲（去粗皮，锉）半两，牛膝（去苗）半两，白茯苓半两，菖蒲半两，五味子半两，蛇床子半两，甘菊花半两，山茱萸半两，附子1两半（炮裂，去皮）。
出处：《圣惠》卷九十四。
主治：None
加减：None
功效：驻颜益寿。
用法用量：每服3钱，平旦酒下。冬月日3服，夏平旦1服，春秋平旦，日暮各1服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
