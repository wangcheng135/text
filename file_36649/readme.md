序号：36659
名称：固元丸
组成：八角茴香（炒）1两，复盆子（酒浸）1两，茴香（炒）1两，白茯苓（去皮）1两，川牛膝（酒浸）1两，磁石（火烧，酒淬7次）1两，龙齿（煅）1两，补骨脂1两（炒），天雄1两（用青盐1两研入酒1盏内，泡天雄，取酒尽为度），五味子（酒蒸，研）2两，菟丝子（净洗，酒蒸，研）2两，鹿茸（去毛）2两半，苁蓉（酒浸1宿）2两半，车前子半两（隔纸炒），麝香半钱（别研），钟乳粉1两半，川乌（炮）1两半，附子（炮，去皮脐）1两半，肉桂（去皮）1两半，巴戟（去心）1两半。
出处：《朱氏集验方》卷八。
主治：元气虚弱，肾水不能滋肝木，子母俱虚，而筋脉拘挛。
加减：None
功效：None
用法用量：每服50-100丸，食前温酒送下。
制备方法：上为细末，酒煮鹿角胶为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
