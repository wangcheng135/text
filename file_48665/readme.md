序号：48680
名称：神蛇酒
组成：白花蛇1条（去头尾），黑蛇1条（去头），僵蚕1两（炒），川乌1两，白芷1两，生熟地1两，玄参1两，白术1两，苦参5两，荆芥1两，防风1两，石菖蒲1两，细辛1两，天麻1两，浮萍3两，当归3两，秦艽3两，海桐皮3两，麻黄3两，狗脊3两，牛膝3两，葶苈3两，草乌3两，苍耳子3两，木瓜3两，灵仙3两，胡麻3两，白芍3两，人参3两，马鞭草3两，枳壳3两，肉桂3两，蛇床3两，枳实3两，皂角3两，白鲜皮3两，五味子3两，肉苁蓉3两，木鳖子3两，五加皮3两，土茯苓1两，青葙子1两，金银花1两，薄荷3两，全蝎1两，蜈蚣30条（去头），桑寄生2两，白茯苓2两，蝉退（去头足）2两，甘草2两，连翘3两，上为饮片，先用元米7升，酿成白酒浆，匀作4坛盛之，听用。又用水2斗，放入酒糟1升，另用大坛盛之，将前2蛇及药俱入坛封口，早煮至晚，取出待冷，开入4坛酒浆内。另将蛇取出，去皮骨，焙干，听用。又加下药：血竭1两，乳香1两，没药5钱，沉香1两，檀香1两，雄黄1两，辰砂1两，穿山甲1两，麝香2钱，牛黄2钱，阿魏2钱。
出处：《秘传大麻风方》。
主治：三十六种麻风，肌肤麻木，遍身瘙痒，癞疹瘾疹，面上游风如虫行，紫白癜风，眉落鼻崩，脚底穿烂，肉死痒痛，一切瘿瘤疬串，无名肿毒，梅花烂疮，并痛风。
加减：None
功效：None
用法用量：服1杯后泻6-7次，方可吃饭。已后将稀粥补之，早、晚各1次。
制备方法：与前二蛇共为细末，搅匀，4坛内封固，又煮1时。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
