序号：34336
名称：万金丸
组成：舶上硫黄1分，巴豆（去皮）半两（2味同以生绢袋子盛于浆水内，用文武火煮1伏时，放冷，另研极细），柴胡半两（去芦），附子1两（炮），干姜半两，陈橘皮（去白）1分，桔梗1分，青黛半两，当归1分。
出处：《博济》卷二。
主治：诸积，腹胁胀痛，冷气攻疰；妇人血气；小儿夜啼，胃冷痰涎等。
加减：None
功效：消化积滞，调三焦，宣利胸膈，理气止痛。
用法用量：每服2丸至3丸，温水送下。妇人血气，醋汤送下；小儿夜啼，常服1丸，温水送下；水泻，生、熟水送下；血淋，地榆汤送下；白痢，干姜汤送下；痰涎并多，生姜汤送下；一切气疾，煎生姜橘皮汤送下，不拘时候。
制备方法：上为细末，面糊为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
