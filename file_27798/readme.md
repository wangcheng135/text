序号：27806
名称：补真丸
组成：葫芦巴（炒）半两，附子（炮，去皮脐）半两，阳起石（煅）半两，川乌（炮，去皮）半两，菟丝子（淘净，酒蒸）半两，沉香（不见火，别研）半两，肉豆蔻（面裹，煨）半两，肉苁蓉（酒浸，焙）半两，五味子半两，鹿茸（去毛，酒蒸，焙）1两，川巴戟（去心）1两，钟乳粉1两。
出处：《济生》卷一。
主治：脾胃虚寒，饮食少思，大便不实，胸膈痞闷，吞酸嗳腐，食反不化。
加减：None
功效：None
用法用量：每服70丸，空心、食前用米饮、盐汤任下。
制备方法：上为细末，用羊腰子二对，治如食法，葱、椒、酒煮烂，入少酒杵和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
