序号：82425
名称：薯蓣散
组成：薯蓣2两，白茯苓2两，远志半两（去心），泽泻1两，黄耆2两（锉），人参1两（去芦头），龙骨1两半，白芍药1两，五味子1两，山茱萸1两，沉香1两，枳壳3分（麸炒微黄，去皮）。
出处：《圣惠》卷二十六。
主治：五劳六极七伤，脐下膨脝，两胁胀满，腰脊相引痛，鼻中干燥，目暗盲盲，愤愤不乐，胸中气逆，不下饮食，小便赤黄余沥，梦与鬼交失精，惊恐虚乏。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，大枣3枚，煎至6分，去滓，纳白砂糖如栗子大，更煎1-2沸，食前温服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
