序号：37666
名称：三龙眼膏
组成：朴消半两，草龙胆2分，白蒺藜1分，旋覆花1分半，仙灵脾2钱（锉）。以上药用黄土半斤，将5味药一处拌匀，于5更取井花水1碗，和黄土并药为泥，稀稠得所，安在垍碗内，以匙摊平，面铺白色开通钱5文，却以硇砂绿豆大20块子，便别于安排上，每文上安4块子，却以别碗合定，置净土上，用新黄土周回并上面培遍，仍透风气。得3-4昼夜开看，其硇砂钱上生半寸长如翡翠色，便取下，次垍器中收之。又用斑蝥（不去翅足），乳香1块（如枣大），秦皮3钱，胡黄连3钱，肥干枣3个，灯心1握（长7寸），古老钱7文（上7味，入无油石器中，取新井花水1大盏，煎至半盏，以棉滤入石器中；其滓再入水1盏，煎至半盏，滤去滓，合汁，以文武火熬成膏子，约得1匙头半。
出处：《鸡峰》卷二十一。
主治：翳膜。
加减：None
功效：None
用法用量：每日用银箸点1黄米大，次日依旧再点，翳膜自随泪下。点药亦不觉痛。如眼微昏，3-2日并愈。
制备方法：将已刮下空青和膏子，入新垍器中盛顿，以硼砂末、龙脑各少许同研匀，以银箸搅匀，用角合子收之，3-5年不坏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
