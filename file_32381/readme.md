序号：32390
名称：妙功十一丸
组成：丁香2钱半，木香2钱半，沉香2钱半，乳香2钱半，麝香2钱半，荆三棱（炮）2钱半，广荗（炮）2钱半，黑牵牛（微炒）2钱半，黄连2钱半，雷丸（炒）2钱半，鹤虱（炒）2钱半，胡黄连2钱半，黄芩2钱半，大黄（焙）2钱半，陈皮2钱半，青皮2钱半，雄黄2钱半，熊胆2钱半，炙甘草2钱半，赤小豆360粒（煮），白丁香（直尖者）360个，轻粉4钱，巴豆7个。
出处：《儒门事亲》卷十五。
主治：痫症。
加减：None
功效：None
用法用量：服时水浸1宿，化下1丸。大便出为验，不可再服。
制备方法：上为细末，赤小豆煮烂研泥，同荞麦面打糊，和作11丸，朱砂为衣，阴干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
