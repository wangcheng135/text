序号：24173
名称：安荣养胎汤
组成：人参（少者，以山药4钱炒黄代之）1钱半，白术1钱半，归身1钱半，熟地1钱半，杜仲（姜炒）1钱半，阿胶（蛤粉炒）1钱半，艾叶8分，条芩（酒炒）1钱，黄柏（炒）1钱，白芍1钱，炙草7分（或加三七根2钱）。
出处：《会约》卷十四。
主治：孕妇气血两虚，六脉微弱，不时下血者。
加减：如血热甚者，加生地、青藁、丹皮之属；如骨蒸多汗者，加地骨皮1钱半；如热甚而渴者，加石膏2、3钱；如五心热，加玄参1钱半；如下热便涩者，加栀子1钱。
功效：None
用法用量：生姜、大枣为引，水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
