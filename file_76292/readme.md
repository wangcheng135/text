序号：76308
名称：紫霞杯
组成：硫黄（袋盛，悬罐内，以紫背浮萍同水煮之数10沸，取出候干，研末）10两，珍珠1钱，琥珀1钱，乳香1钱，雄黄1钱，朱砂1钱，羊起石1钱，赤石脂1钱，片脑1钱，紫粉1钱，白芷1钱，甘松1钱，三奈1钱，木香1钱，血竭1钱，没药1钱，韶脑1钱，安息香1钱，麝香7分，金箔20片。
出处：《本草纲目》卷十一引《水云录》。
主治：妇人血海枯寒，赤白带下。
加减：None
功效：延年却老，脱胎换骨，大能清上实下，升降阴阳，通九窍，杀九虫，除梦泄，悦容颜，解头风，开胸隔，化痰涎，明耳目，润肌肤，添精髓，蠲疝坠。
用法用量：每旦盛酒饮2-3杯。
制备方法：上为末，入铜勺中慢火熔化，以好样酒杯1个，周围以粉纸包裹，中开一孔，倾硫入内，旋转令匀，投冷水中，取出。
临床应用：劳瘵：昔中书刘景辉因遭劳瘵，于太白山中遇一老仙，亲授是方，服之果愈。
各家论述：None
