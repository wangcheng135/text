序号：9923
名称：止痛雷火针
组成：蕲艾末1两，雄黄2钱，没药1钱，丁香1钱，白芷1钱，麝香3分，乳香1钱。
出处：《活人方》卷一。
主治：寒湿二气，有一流注于经络关节之间，便成痛痹。或着一处，或走不定，甚至气血虚寒，不能营运，加至风寒外袭，筋脉凝塞，不通而痛。或过食生冷、坚硬之物难消，胸腹胀满，窘迫而痛。或房劳亏损肾气，而寒邪侵于肾俞，督脉为痛。不分虚实，皆可通治。
加减：痞，加阿魏1钱。
功效：None
用法用量：用时灯上烧红，隔青布5-7层，于痛处针之。
制备方法：上为末，匀摊细草纸上，卷紧如筒，1钱粗细，外用棉纸封固，每料分作5条，晒燥收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
