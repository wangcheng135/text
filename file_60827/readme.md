名称：造化争雄膏
组成：炼松香（用小竹甑1个，用粗麻布1层，用明肥松香放其上，安水锅上蒸之，俟松香溶化，淋下清净者，初倾入冷水中，又以别水煮2-3滚，又倾入水中，如此数次后，复用酒如前煮之，俟其不苦不涩为度；2次炼，不用铁锅尤妙），飞黄丹（用好酒，入水中淘去底下砂石，取净，候干，炒之），真麻油3斤，粉甘草4两（先熬数沸，后下药），官桂（去粗皮），远志（油浸1宿，去心，焙干，为末）6钱，菟丝子（淘去沙，酒煮极烂，捣成饼，为末）6钱，川牛膝（去芦，酒浸1宿，晒干，为末）6钱，鹿茸（去毛，酥炙黄）6钱，虎骨（酥炙黄）6钱，蛇床子（拣净，酒浸1宿，焙干）6钱，锁阳（酥炙）6钱，厚朴（去皮）6钱，淮生地（酒浸1宿，焙干）6钱，淮熟地（酒浸1宿，焙干）6钱，玄参（去芦头）6钱，天门冬（去心）6钱，麦门冬（去心）6钱，防风（去芦）6钱，茅香（拣净）6钱，赤芍药（酒浸洗）6钱，白赤芍（酒浸洗）6钱，当归（酒洗）6钱，白芷6钱，北五味子6钱，谷精草6钱，杜仲（去皮，锉，盐酒炒去丝）6钱，荜茇6钱，南木香6钱，车前子6钱，紫梢花6钱，川续断6钱，良姜6钱，黄蜂、穿山甲（锉，以灶灰炒，为末）2钱，地龙（去土，炙）4钱，骨碎补2钱，蓖麻子、杏仁（去皮尖）4钱，大附子2个（重2两，面裹火煨，去皮脐），木鳖子（去壳）40个（研，纸裹压去油），肉苁蓉（红色者，酒浸，去甲，焙）7钱，桑嫩枝7寸，槐嫩枝7寸，桃嫩枝7寸，李嫩枝7寸（1方有红蜻蜒10只）。
出处：《疡科选粹》卷八。
主治：咳嗽吐痰，色欲过度，腰胯疼痛，两腿酸辛，行步艰难，下无不固，胞冷精寒，小便频数，遗精白浊，吐血鼻衄；妇人下寒，赤白带下，子宫冷痛，久不胎孕；恶毒痈疽顽疮，一切无名疔肿。
加减：None
功效：养精神，益气血，存真固精，龟健不困，肾海常盈，返老还童。
用法用量：如遇用时，入热水泡软，以手掌大红纻系1方，摊药在上，不用火烘。贴之。
制备方法：上药各依法制度完备，锉，入油内，用铜锅桑柴火慢煎候枯黑，取起，滤以生绢，去滓，锅亦拭净，其药油亦须滴水成珠为度，每药油1斤，用飞过黄丹8两，徐徐加入，慢火煎熬，用桑、槐、柳枝不住手搅，勿使沉底，候青烟起，膏已成，看老嫩得中住火，入炼过松香半斤，黄蜡6两，此亦以1斤油为率，搅匀放冷，膏凝结后，连锅覆泥土3日，取起，用别锅烧滚水，顿药锅在上，隔汤泡融，以桑、槐、柳枝不住手搅300-500遍，去火毒，入后药：麝香、蟾酥、霞片（疑鸦片）、阳起石（云头者）、白占各6钱，丁香、乳香、广木香、雄黄、龙骨、沉香、晚蚕蛾、倭硫黄、赤石脂、桑螵蛸、血竭、没药各4钱，黄耆（去皮头，蜜炙；为末）3钱。上件须选真正道地者，各制度过，为极细末，起手先熬药油，以上药渐投入药面中搅极匀和，即投膏入冷水中，捏成5钱1饼。
