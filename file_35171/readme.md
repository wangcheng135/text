序号：35181
名称：乳香乌龙丹
组成：骨碎补1两，葫芦巴（炒）1两，破故纸（炒）1两，川乌（炮，去皮脐）1两，苁蓉（酒浸）1两，牛膝（酒浸）1两，金毛狗脊1两，牡蛎（烧赤）1两，甜瓜子（炒）1两，黑牵牛1两，赤芍药1两，自然铜（烧赤，醋蘸7次）1两，川楝子1两，茴香（炒）1两，牡丹皮1两，五灵脂1两，白芍药1两，苍术2两（去皮，炒），官桂半两，细墨7钱，木鳖子仁7个，没药2钱半，乳香2钱。
出处：《医方类聚》卷一八八引《施圆端效方》。
主治：落马坠车，打扑闪肭，损伤筋骨，疼痛闷乱，及中风瘫痪，口眼斜，妇人血风，腰腹四肢走注疼痹。
加减：None
功效：活血止疼。
用法用量：每服7-10丸，温酒送下，1日3次。
制备方法：上为细末，研匀，醋面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
