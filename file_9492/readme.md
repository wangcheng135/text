序号：9492
名称：加味神功丸
组成：木香1两，人参2两（去芦头），大麻仁2两（别研如膏），枳实2两，桃仁2两，诃黎勒皮4两，大黄（锦文者）4两（面裹煨，去面）。
出处：《普济方》卷四十三引《余居士选奇方》。
主治：三焦气壅，心腹痞闷，六府风热，大便不通，腰腿疼痛，肩背沉重，头昏面热，口苦咽干，心胸烦躁，睡卧不安；及治脚气，并素有风人，大便秘结。
加减：None
功效：None
用法用量：每服20丸，食后、临卧以温水送下，或温米饮下皆可。如大便不通，可加丸数，以利为度。
制备方法：上为细末。入麻仁捣研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：神功丸（《原书卷三十七）。
