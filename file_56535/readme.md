序号：56550
名称：大效丸
组成：乌头1两，附子1两（2味去皮脐，为末，用醋1升，煎尽为度；又入好酒1升再煎，成膏后，入诸药），乌蛇（酒浸，去皮骨，炙）1分，细辛（去苗叶）1分，厚朴（去粗皮，姜汁炙）1分，人参1分，赤茯苓（去黑皮）1分，桂（去粗皮）1分，干蝎（炒）1分，木香1分，乳香（研）1分，草豆蔻（去皮）1分，硇砂（研）1分，胡桐泪1分，槟榔（锉）1分，腻粉1分，不灰木1分。
出处：《圣济总录》卷二十九。
主治：伤寒坏病。
加减：None
功效：None
用法用量：每服10丸，如阳盛，研牛黄、腻粉，水下；如阴盛发厥，心胸结痞，煎柳枝汤下。
制备方法：上将15味为末，入2味膏子内和捣为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
