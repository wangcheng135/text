序号：601
名称：五福灵丹
组成：朱砂（不夹石者，辰砂为上）1两，雄黄1两，雌黄（叶子者）1两，阳起石（钩牙白者）1两，硫黄1两（逐味研细）。
出处：《鸡峰》卷二十九。
主治：一切虚冷。
加减：None
功效：None
用法用量：每服1粒，空心温酒或水送下，以食物压之。
制备方法：上拌和一处，用沙盒子1个，内先铺草决明100粒、杏仁7个(嚼破)、海金砂2钱，后将五药末入在盒子内，上面更用草决明、杏仁、海金砂如前，又用紫石英小枣大放在诸药之上，盖盒子了，用赤石脂1两为极细末，醋调如膏，固口缝上，以重物压之1宿。候干，用蚯蚓粪、盐1两，好纸数张同作泥。固厚两指许，阴干，掘地坑子深5-6寸，阔尺余，内用新牛头砖，上放盒子于坑子中心，先将捍草一速逐旋放盒子上烧，草尽，去草灰一半，留一半拥盒子，然后用木炭5斤为祖火，候炭烧及2斤许，添生炭斤半，亦不要火大，亦不火慢，只以5斤火为则，候烧炭及一秤，更看火候加减。如火足便以新黄土罨1日，取出研为极细末，枣肉为丸，如梧桐子大，固济须是如法。
临床应用：None
各家论述：None
用药禁忌：None
