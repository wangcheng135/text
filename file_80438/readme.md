序号：80454
名称：藿香煮散
组成：藿香叶1两，木香1两，陈橘皮（汤浸去白，焙）1两，肉豆蔻（去壳）1两，诃黎勒皮1两，人参1两，白茯苓（去粗皮）1两，甘草（炙）1两，草豆蔻（去皮）1两，麦糵（炒）1两，陈曲（炒）1两，干姜（炮）半两，高良姜（锉，炒）半两，厚朴（去粗皮，生姜汁炙）1两半。
出处：《圣济总录》卷七十二。
主治：久积聚，宿滞不消，或翻胃吐逆，恶心干哕；及脾寒疾；水泻及肠风脏毒。
加减：None
功效：None
用法用量：每服2钱匕，水1盏，生姜1块拍破，同煎至7分，入盐1捻，热服。水泻及肠风脏毒，热陈米饮调下。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
