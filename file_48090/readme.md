序号：48105
名称：神应万验膏
组成：桃枝2两，柳枝2两，杏枝2两，桑枝2两，槐技（截作寸许长）2两。
出处：《疡医大全》卷七。
主治：一切无名肿毒，大疮恶疽，无论已破未破者。
加减：None
功效：None
用法用量：用时以井华凉水浸半日，捻成片，放布上，热汤熨化贴。
制备方法：用真麻油24两，小炭火熬滚，将枝次第入油熬枯成炭，滤去滓；再入人头发（男女各半，洗净油腻）1两5钱，入油炸化；再入穿山甲（剪碎）1两5钱，入油炸枯；再入象皮（剪碎）5钱，入油炸化；再入大栀子100个，逐个捻破，入油内离火浸1炷香，再用微火炖1炷香，再用大火炸成炭，取起冷定，用夏布滤去滓，再入净锅内，称准每油2两，人炒过黄丹1两，熬至滴水成珠不散，离火一刻，再入后药：真硇砂（透明白亮者）、血竭、儿茶各2钱，乳细，拌入膏内，坐冷水中，稍凉取起，用水湿手扯捻百下，使各药和匀，埋土内5日，去火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
