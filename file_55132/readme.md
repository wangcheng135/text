序号：55147
名称：桂枝麻黄汤
组成：桂枝1两，麻黄1两（去根节），赤芍药1两，杏仁1两（去皮尖，麸炒黄）。
出处：《医方类聚》卷五十四引《通真子伤寒括要》。
主治：阳明中风，头痛口苦，腹满微喘，发热恶寒，脉浮而紧，下之即小便难者；阳明病，5-6日至7-8日，如疟，热多寒少，1日再发，其脉微缓，为欲愈者；厥阴脉微而恶寒者，为阴阳俱虚，不可吐下者；厥阴发汗，面色赤，有热者，此欲解者。
加减：None
功效：None
用法用量：每服4钱，水1盏，加生姜5片，大枣2枚，煎至6分，去滓热服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
