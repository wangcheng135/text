序号：69641
名称：琥珀散
组成：琥珀1两，虎杖3分，牛膝1两（去苗），木香半两，鳖甲1两（涂醋，炙令微黄，去裙襕），赤芍药1两，柴胡1两（去苗），赤茯苓3分，桂心半两，桃仁3分（汤浸，去皮尖双仁，麸炒微黄），当归3分（锉，微炒），枳壳3分（麸炒微黄，去瓤）。
出处：《圣惠》卷七十九。
主治：产后月水不通，胁腹妨闷，四体烦疼，吃食减少，渐觉虚困。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，入生姜半分，煎至6分，去滓，每于食前温服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻、苋菜。
附注：《普济方》引本方有川大黄一两（锉碎，微炒）。
