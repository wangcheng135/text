组成：当归7分，石南6分，踯躅4两，秦艽4两，菊花4两，干姜4两，防风4两，雄黄4两，麝香4两，丹砂4两，斑蝥4两，蜀椒8分，鬼箭羽8分，连翘8分，石长生8分，知母8分，蜈蚣3枚，虻虫10枚，地胆10枚，附子4两，鬼臼11分，人参12分，石斛12分，天雄12分，王不留行12分，乌头12分，独活12分，防己12分，莽草12分，水蛭百枚。
