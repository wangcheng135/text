序号：26812
名称：延龄护宝丸
组成：禹余粮石2两（烧，醋淬7遍），龙骨1两，人参1两，桂1两，赤石脂1两，紫石英（研）1两，熟地黄1两，杜仲（去粗皮，锉碎）1两，桑寄生1两，续断1两，香白芷1两，芎1两，当归（锉，炒）1两，远志（去心）1两，金钗石斛（去根，锉，炒）1两，白茯苓（去皮）1两，阿胶（炒）1两，牡蛎1两，五味子1两，艾叶1两。
出处：《产乳备要》。
主治：妇人血脏虚损，经候过多，每行时暴下不可禁止，因成崩中，连日不断，致五脏空虚，失色黄瘦，崩竭暂止，日少复发，不耐动摇，小劳辄极剧。
加减：None
功效：None
用法用量：每服40-50丸，空心、食前温粥饮送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“延龄护宝丹”。
