序号：48744
名称：土瓜根散
组成：土瓜根3两，芍药3两，桂枝，(庶虫)虫3两。
出处：《金匮》卷下。
主治：带下，经水不利，少腹满痛，经1月再见者。
加减：None
功效：None
用法用量：酒服方寸匕，每日3次。
制备方法：上为散。
临床应用：None
各家论述：1.《金匮玉函经二注》：土瓜根者，能通月水，消瘀血，生津液，津生则化血也；芍药主邪气腹痛，除血痹，开阴寒；桂枝通血脉，引阳气；(庶虫)虫破血积，以消行之，非独血积冲任者有是证，肝藏血，主化生之气，与冲任同病，而脉循阴器，任、督脉亦结阴下，故皆用是汤治之。_x000D_
2.《张氏医通》：土瓜根，黄瓜根也，往往以栝楼根代用，考之《本经》，栝楼根性味虽同苦寒，而无散瘀血，通月闭之功，此治虽专，故以桂、(庶虫)弼之，芍药监之，与旋覆花汤之用新绛不殊。_x000D_
3.《金匮要略浅注》：土瓜，即王瓜也，主驱热行瘀；佐以(庶虫)虫之蠕动逐血，桂、芍之调和阴阳，为有制之师。
用药禁忌：None
附注：None
