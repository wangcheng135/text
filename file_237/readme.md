序号：237
名称：五虎神效膏
组成：蜈蚣6钱，生军6钱，川乌6钱，全蝎6钱，苦杏仁6钱，白芍5钱，羌活5钱，苏合香5钱，黄耆5钱，玄参5钱，甘草节5钱，皂角5钱，白芨8钱，赤芍8钱，连翘8钱，独活5钱，生地8钱，乌药8钱，白蔹8钱，乳香8钱，官桂8钱，当归8钱，木鳖子肉8钱，苦参8钱，炙没药8钱，蛇蜕3钱，血酥1两，蜂房（带子最好）4两，活大蟾2只（小者3只）。
出处：《丁甘仁家传珍方选》。
主治：一切无名肿毒及搭背、对口、大小痈疖；并治头风痛。
加减：None
功效：未成即消，已成即敛。
用法用量：外贴患处。
制备方法：外加桃树枝8钱，柳树枝8钱，槐树枝8钱，枣树枝8钱，桑树枝8钱，用真麻油11斤熬，去滓，红丹适量收膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中血酥，疑是“血竭”。
