序号：61483
名称：培坤丹
组成：炙黄耆3斤，白术3斤，炙草8两，广陈皮2斤，当归5斤，川芎1斤，杭芍1斤，砂仁9两，北沙参1斤，云茯苓2斤，枣仁2斤，寸冬2斤，杜仲2斤，核桃仁1斤4两，胡芦巴2斤8两，艾叶1斤，元肉2斤，山萸2斤，远志肉4两，熟地4斤，五味子8两，酥油4两。
出处：《全国中药成药处方集》（兰州方）。
主治：妇女贫血，消化不良，月经不调，赤白带下，小腹冷痛，精神不振，倦怠嗜卧，体温低降，不耐寒冷。
加减：None
功效：调经养血健胃。
用法用量：每服3钱。黄酒或白开水送下，1日2次。
制备方法：上药各为细末，以酥油熔拌微炒，炼蜜为小丸。
临床应用：None
各家论述：None
用药禁忌：抑郁气滞，内有蕴热者忌服。
附注：None
