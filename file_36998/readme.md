序号：37008
名称：奇应轻脚丸
组成：宣木瓜1个（用竹刀切开，顶作盖，剜去瓤，入熟艾实之，甑上蒸熟，薄切，焙干），肉苁蓉（酒浸1宿，去土）1两，防风（去叉头芦）1两，牛膝（酒浸1宿）1两，金毛狗脊（去毛）1两，川萆薢1两，青盐（别研）1两，海桐皮1两，川乌头4两（生用，去皮脐）。
出处：《魏氏家藏方》卷八。
主治：缓风湿痹，脚膝顽弱，腰腿疼痛，足胫肿满，或麻木不仁，或生疮不已。
加减：None
功效：None
用法用量：每服10-15丸，空心温酒或盐汤送下。
制备方法：上为细末，将乌头末酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
