序号：68776
名称：小麦面十四味煎
组成：小麦5升（以水硬搜之，别于水中揉挻，令面粉尽，面筋别成1块即止；以此面粉汁别器澄停，沥却清汁，即以稠粉盛于练袋子中漉，着令微燥），生葛根5挺（径3寸，长2尺，捶碎于水中揉挻，令葛根中粉汁尽，别器澄停，盛贮一如小麦面法），生栝楼5斤（捣如上法），胡麻3升（去皮，熬令熟，为散），堇竹根（切）1斤，生茅根（切）1斤，生芦根（切）1斤，乌梅50个（以上用水5斗，缓火煎取1升半，去滓澄取清），冬瓜汁2升，生麦门冬汁3升，生姜汁1升，牛乳1升，白蜜2升。
出处：《外台》卷二十七引《许仁则方》。
主治：消渴小便数。
加减：None
功效：None
用法用量：每夜含如此，初服1枣大，稍稍加至1匙，亦任性日日含之。欲作丸，饮服亦得。
制备方法：先取竹根等汁，和冬瓜以下汁，微火上煎减半，次纳牛乳、白蜜，又煎6-7沸，投小麦面粉、生葛粉、栝楼粉、胡麻散于诸汁中煎，和熟搅之，勿住手，候如稠糖即成，成讫，止火待冷，贮别器中。
