序号：69974
名称：煮肝丸
组成：雄猪肝1具（用米醋3升，煮醋尽为度），白矾（烧研）2两，柴胡（去苗）2两，厚朴（去粗皮涂生姜汁炙透）1两，干姜（炮裂）1两，黄连（去须）1两，陈橘皮（去白，焙）1两，桂（去粗皮）半两，附子（炮裂，去皮脐）半两。
出处：《圣济总录》卷八十七。
主治：冷劳。腹痛下痢，面色萎黄，四肢无力。
加减：None
功效：None
用法用量：每服7丸，空心温酒送下，晚食后再服，如不饮酒，生姜盐汤送下，重者不过3剂。
制备方法：上9味，捣罗8味为末。以醋煮猪肝极烂，入白面5匙相和，煎三、五沸，入诸药末一处，和捣为丸，如绿豆大，焙干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
