序号：38740
名称：三灵丹
组成：朱砂3两（细研如粉），雌黄1两半（细研如粉），硫黄半两（细研如粉）。
出处：《圣惠》卷二十三。
主治：中风偏枯不遂，口不收涎。
加减：None
功效：None
用法用量：每日3丸，空心以冷椒汤送下，渐加至五丸。服半月即愈。
制备方法：上药先将雌黄、硫黄于铛中消成汁，后下朱砂末，搅令匀，候冷，却下桑柴灰汁，煮3日3夜，旋旋添暖灰汁，候日足即住，刮入鼎子中，以文火逼干，出阴气，入盒子内固济，以20斤火煅，候火消至3-5斤，其药已在盒底作1片，候冷凿取，放瓷器中，入水煮1日，出火毒了，更研令细，入枣肉和研为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：忌羊血。
附注：None
