序号：47536
名称：神化丹
组成：黑丑（头末）3钱，母丁香3钱，槟榔3钱，何首乌3钱，荆芥3钱，荆三棱（醋炒）3钱，熟地3钱，蓬莪术（醋炒）3钱，巴豆3钱，五灵脂3钱，大黄3钱，白豆蔻（去壳）3钱，桂枝3钱，穿山甲3钱，当归3钱，赤芍药3钱，川乌3钱，小茴香3钱，草乌3钱，杏仁（炒）3钱，全蝎（去足）3钱，连翘3钱，麻黄3钱，甘草3钱，桔梗3钱，斑蝥3钱，雄黄3钱，朱砂3钱，乳香（去油）2钱，没药（去油）2钱，麝香5分，大蜈蚣1条。
出处：《疡医大全》卷七。
主治：痈疽疔毒，一切无名肿毒初起。
加减：None
功效：双解表里，疏通经络，以毒攻毒，削坚导滞。
用法用量：每服3分，以热酒吞下，尽醉为度。被盖出汗。
制备方法：上为细末，水泛为丸，如萝卜子大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：孕妇、体虚禁用。
附注：None
