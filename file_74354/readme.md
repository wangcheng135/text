序号：74370
名称：塌肿汤
组成：黄耆2两，白芍2两，川芎2两，当归2两，陈皮2两，甘草2两，麻黄（去节）2两，人参5钱，乳香（炙）5钱，没药（炙）5钱，罂粟壳（去顶蒂及筋，蜜炙）2两。
出处：《惠直堂方》卷三。
主治：一切恶疮、发背、痈疽、疔疮痛不可忍者；或疮毒入内，神思昏倦呕吐者；又治跌打损伤，筋骨疼痛，妇人产后肚痛，恶露不快，赤白带下。
加减：None
功效：None
用法用量：每服1两5钱或2两，水煎温服。凡疮科能专守此方，未有不获全功者。能使恶疮未成即消，已成即溃，不假砭蚀，恶毒自下。
制备方法：上锉为片。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
