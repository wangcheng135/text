序号：36825
名称：固根汤
组成：葳蕤1两，当归5钱，白芍5钱，熟地1两，麦冬5钱，甘菊3钱，菖蒲3分，柴胡5分。
出处：《辨证录》卷三。
主治：少年时丧元阳，又加时眼，不守色戒，伤损大眦，眦孔不闭，风寒透入其孔，内气即虚，外邪难杜，以致目痛后迎风流泪，至夜则目暗不明，一见灯光两目干涩。
加减：None
功效：None
用法用量：水煎服。连服4剂，即不畏风，再服4剂，见风不流泪矣，再服10剂痊愈。
制备方法：None
临床应用：None
各家论述：盖葳蕤最善止泪，加之当归、白芍以补肝，熟地以滋肾，益之麦冬以补心，佐之甘菊、菖蒲、柴胡以舒其风火，而引诸经之药以塞其泪窍。此固其根本而末症自愈也。
用药禁忌：None
附注：None
