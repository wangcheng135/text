序号：69698
名称：琥珀镇惊丸
组成：琥珀3钱，胆星5钱，竺黄5钱，茯神5钱，钩藤5钱，地龙2钱，犀角2钱，制军3钱，朱砂1钱，麝香1钱，冰片5分，雄黄3钱，苏珠2分，薄荷1钱。
出处：《全国中药成药处方集》（抚顺方）。
主治：小儿急痫；小儿目触异物、耳闻异声，发为惊悸抽搐；或热度太高，壮热神昏，牙关紧闭，二目天吊，角弓反张，痰鸣气喘，抽搐厥逆。
加减：None
功效：清凉，镇痉，化痰。
用法用量：小儿1岁以下每服半丸，2-4岁每服1丸，薄荷姜汤送下。
制备方法：上为细末，炼蜜为丸，重3分5厘，蜡皮封。
临床应用：None
各家论述：None
用药禁忌：大便泄泻，元气衰弱者禁用。
附注：None
