序号：46593
名称：千金种子丹
组成：沙苑蒺藜4两（净末如蚕种，同州者佳，再以重罗罗，2两极细末，2两粗末，用水1大碗，熬膏伺候），莲须（极细末）4两（金色者固精，红色者败精），山茱萸（极细末3两，须得1斤，用鲜红有肉者佳，去核取肉，为细末），覆盆子（南者佳，去核取极细末）2两，鸡头实500个（去壳，如大小不等，取极细末4两），龙骨5钱（5色者佳，火煅。煅法：以小砂锅，将龙骨入锅内，以火连砂锅煅红，去火毒方用）。
出处：《扶寿精方》。
主治：虚损梦遗，白浊。
加减：None
功效：延年益寿。
用法用量：每服30丸，空心盐汤送下。
制备方法：上用伏蜜1斤炼，以纸粘去浮沫数次，无沫，滴水中成珠者伺候，只用4两；将前6味，重罗过，先以蒺藜膏和作1块，再入炼蜜4两，入石臼内捣千余下方可，丸如黄豆样大。
临床应用：None
各家论述：None
用药禁忌：忌欲事20日。
附注：None
