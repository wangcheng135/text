序号：540
名称：五野丸
组成：牛黄（研）1分，麝香（研）1分，蜀椒（去目，汗）1分，雄黄（研）1分，大黄1分，当归1分，蜀乌头（炮）1分，蜀天雄（炮）1分，消石（熬，一方用芒消）1分，人参2分，桂心2分，朱砂2分，细辛2分，干姜2分，石蜥蜴1枚（炙），巴豆50枚（去心皮，熬），鬼臼2分。
出处：《外台》卷十三引《古今录验》。
主治：五连：尸疰、哭疰、冷疰、寒疰、热疰，身体寒热。短气、两胁下痛，引背腰脊，吸吸少气不能行，饮食少，面目痿黄，小便难，项强不得俯仰，腹坚癖，脐左右下雷鸣胀，手足烦疼，目不明，喜忘；久风湿痹，腰脊不随，喜梦寤。
加减：None
功效：None
用法用量：每服3丸，1日2次。不知稍增，以知为度。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌猪肉、芦笋、生葱、生菜、生血物。
附注：牛黄丸（《普济方》卷二三八）。
