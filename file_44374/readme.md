序号：44389
名称：一善膏
组成：木通2两，绵黄耆2两，羌活2两，川芎2两，生地黄2两，桃仁2两，白芷2两，连翘2两，玄参2两，防风2两，木鳖子仁2两，当归末2两，乳香（另研）2两，没药（另研）2两。
出处：《普济方》卷三一五。
主治：小儿脾证，大人一切风气，气积，食冷积，气块；痈疽，疖毒，疔肿，杖疮。
加减：None
功效：None
用法用量：凡贴之，数日不可揭去，速则作痛。
制备方法：上除乳、没、当归外，余并锉用。真麻油4斤半，炒黄丹24两，续挑入油内，以柳枝3-5条不住手搅之，丹不可老，火不可猛，直候丹变黑色，滴水不散为度；取出稍冷，却下乳、没、当归末，再搅匀，慢火养1时许，露地1宿，蛤粉养之，旋摊用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
