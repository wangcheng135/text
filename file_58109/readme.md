序号：58124
名称：益神散
组成：川楝子4两，炒麦芽4两，炒枳壳4两，使君子肉（醋制炒）4两，炒乌药4两，炒枳实4两，炒猪苓4两，炒山楂4两，炒川朴4两，炒泽泻4两，炒槟榔4两，大黄（酒制炒）1两2钱5分，莪术（醋制炒）1两2钱5分，三棱（醋制炒）1两2钱5分，胡黄连（炒）1两2钱5分，青皮（炒）1两2钱5分，青矾（隔纸炒）8两，六曲（醋制炒）8两，干漆（炒绝烟）7钱5分，苍术（醋制炒）7钱5分，四制香附12两，针砂5钱，陈皮1钱5分。
出处：《良方合壁》卷下。
主治：小儿肚大青筋，已成疳积，及妇人经水不调。
加减：None
功效：None
用法用量：每服1钱，清晨用黄沙糖拌和，开水调下。
制备方法：上为极细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
