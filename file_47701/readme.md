序号：47716
名称：神仙训老丸
组成：生干地黄5两，熟干地黄5两，川椒10两（不去核），牛膝5两（酒浸，为末），大黑豆1升（生用），干山药5两，赤何首乌10两，白何首乌10两，肉苁蓉5两，枸杞5两，藁本10两（洗）。
出处：《寿亲养老》卷四。
主治：None
加减：None
功效：补下元，光泽皮肤，常服延年益寿，气力倍常，齿落再生，发白再黑，颜貌如婴儿。
用法用量：空心温酒或盐汤送下。
制备方法：上将白何首乌为末，放水甑内，早晨蒸，日出晒，夜间露，如此九蒸、九晒、九露，数足焙焦为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌萝卜。
附注：神仙补老丸（《医学入门》卷七）。《医学入门》本方用法：每服五十丸。
