序号：68402
名称：黄连上清丸
组成：黄连8两，大黄256两，连翘64两，薄荷32两，防风32两，复花16两，黄芩64两，芥穗64两，栀子（炒）64两，桔梗64两，生石膏32两，黄柏32两，蔓荆子（炒）64两，白芷64两，甘草32两，川芎32两，菊花128两。
出处：《北京市中药成方选集》。
主治：头目眩晕，暴发火眼，牙齿疼痛，口舌生疮，二便秘结。
加减：None
功效：清热通便，消炎散风。
用法用量：每服水丸2钱或蜜丸2丸，每日2次，温开水送下。
制备方法：上为细粉，过罗，用冷开水泛小丸；或炼蜜为大丸，重2钱。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
