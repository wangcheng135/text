序号：58545
名称：秘传八味二花散
组成：桃花蕊5钱，梅花蕊（2味不拘多少，阴干）共1两6钱2分，穿山甲（取四足者，酒炒）1两，朱砂（水飞过）1两，紫河车（水洗，去红筋，焙干为末）1具，天灵盖1具（以皂角煎汤洗净，酥制为末），四方共1钱7分，鹿茸（去毛，酥制）3方共1两2钱，人参（官拣者）1两。
出处：《准绳·幼科》卷四。
主治：痘疹。
加减：None
功效：None
用法用量：痘疹末出之先，以朱砂1两为君，梅花2钱，桃花3钱共一处和匀，每服5分或3分或2分半，用雄鸡与酒2杯灌之与食，倒悬，刺血入林中，以热酒调前药同服；初发不起，以梅花1两为君，加桃花1钱，天灵盖5分；气血虚，灰白色，用紫河车1两为君，加天灵盖1钱，鹿茸1钱5分，梅花1钱，桃花1钱或8分；黑陷不起，以穿山甲1两为君，加桃花1钱，梅花8分，天灵盖7分，麝香5分；气血虚不能灌浆，以鹿茸1两为君，加紫河车2钱，桃花1钱，梅花8分，天灵盖7分，麝香5分，人参1钱；气血虚不能收靥，以鹿茸1两为君，天灵盖5分，桃花7分，梅花9分；落靥之后，瘢色白，气血虚，以人参1两为君，加紫河车2钱，鹿茸1钱，梅花1钱5分，以上咸照首条服法。
制备方法：上各为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
