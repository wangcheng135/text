序号：49081
名称：胎产金丹
组成：党参2两5钱，生地4两，香附4两，鳖甲4两，白术2两，白薇2两，当归2两，川芎2两，丹皮2两，黄芩2两，玄胡2两，蕲艾2两，青蒿2两，乳香2两，赤石脂2两，益母草2两，茯苓1两，五味1两，血琥珀1两，藁本1两，安桂1两5钱，白芍1两5钱，甘草1两5钱，沉香5钱。
出处：《饲鹤亭集方》。
主治：妇人胎前产后诸恙百病及子宫寒冷，艰于受孕，红白淋带疼痛，经停参前落后，行经腹痛，腰痠无力。
加减：None
功效：None
用法用量：None
制备方法：上为末，都拌匀，炼蜜为丸，每重2钱，辰砂为衣，蜡封口。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
