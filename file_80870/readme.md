序号：80886
名称：麝香丸
组成：麝香半两（细研），阿魏1分（面裹煨，以面熟为度），五灵脂3分，没药半两，蓬莪荗半两，芫花1两（醋拌炒令干），京三棱3分（微炮，锉），桂心半两，木香半两，当归半两（锉，微炒），桃仁3分（汤浸去皮尖、双仁，麸炒微黄），槟榔1两。
出处：《圣惠》卷七十一。
主治：妇人痃癖，冷气兼疰气，心腹痛不可忍。
加减：None
功效：None
用法用量：每服10丸，以醋汤送下，不拘时候。
制备方法：上为末，研入麝香令匀。用粳米软饭为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：大麝香丸（《普济方》卷三二五）、麝香丹（《医学入门》卷八）。
