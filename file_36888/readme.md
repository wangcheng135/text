序号：36898
名称：国老汤
组成：横纹大甘草1两。
出处：《古今医鉴》卷八。
主治：悬痈。脏毒。
加减：None
功效：None
用法用量：上药截长3寸许，取山涧东流水1碗（不用井水、河水），以甘草蘸水，文武火慢炙，须用3时久，水尽为度；劈视草中润透，再以无灰酒2碗，煎煮至1碗，温服，1日1次。半月消尽。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌煎炒、烟、酒、炙、煿、辛辣、发气等物。
附注：国老煎（《外科证治全书》卷三）。本方改为散剂，名“国老散”（见《医学心悟》）。用甘草七段，急流水一碗，浸之，炙干；又浸，又炙，以水尽为度，研细末。每服一钱，空心开水调下。
