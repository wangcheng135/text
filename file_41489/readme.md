序号：41504
名称：金旋散
组成：白附子（炮）、木香、肉豆蔻（去皮）、猪牙皂角（去皮，生）、桔梗、吴茱萸（麸炒）、肉桂（取心）、大黄（生）、川芎（净）、知母、白茯苓、当归、槟榔2个（一个生，一个熟）、巴豆（去皮，日日换汤，浸14日，又用麦麸水，煮1日，细研）、白芜荑（取仁）、芍药、白僵蚕2分（直者），黄连（取净）2两。
出处：《博济》卷四。
主治：卒中风，头旋，血淋，腰膝痛，肠风背阴，寸白虫，霍乱吐泻，肺气喘，小儿（鼻勾），小儿误吞钱及小儿天钓风。
加减：None
功效：None
用法用量：每用1字，汤使如后：卒中风，羊髓酒送下；头旋，菊花酒送下；血淋，大黄汤送下；腰膝痛，醋汤送下；吐血，竹茹汤送下；肠风，背阴繁柳草自然汁入热酒，又槲叶烧灰，调酒送下；寸白虫，先吃牛脯，后以芜荑汤送下；霍乱吐泻，新汲水送下；肺气喘，杏仁汤送下；小儿一切疾，米饮送下；小儿（鼻勾），蜜汤送下；小儿误吞钱，腻粉汤送下；小儿天钓风，以蝉壳烧灰入小便调下。
制备方法：上为细末。入巴豆子乳钵内同研令匀，然后入瓷器中密封，候至7日。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
