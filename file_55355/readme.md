序号：55370
名称：大养胃汤
组成：厚朴（去皮）2两，生姜2两，肥枣3两（锉，上3味同炒），白术1两，山药（炒）1两，人参1两，川芎1两，橘皮1两，当归1两，五味子1两，藿香1两，甘草（炙）1两，枇杷叶（刷毛，姜炙）1两，黄耆1两。
出处：《三因》卷十一。
主治：饮食伤脾，宿谷不化，朝食暮吐，暮食朝吐，上气复热，四肢冷痹，三焦不调；及胃虚寒气在上，忧气在下，二气并争，但出不入，呕不得食。
加减：None
功效：None
用法用量：每服4钱，水1盏半，加生姜3片，大枣1个，煎7分，去滓，空腹服；或为细末，米汤调下。
制备方法：上锉散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
