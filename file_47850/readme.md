序号：47865
名称：神功异宝丸
组成：附子1只（大者，生，去皮脐），赤小豆（生）1两，雄鸟豆（生）1两，川独活1两，青盐（别研）1两，川楝子（取肉，炒）1两，破故纸（酒浸1宿，炒）1两，川萆薢半两，黑牵牛（生用）半两，牛膝7分半（酒浸1宿），川乌头2钱半（生，去皮），草乌头2分半（生，去皮），地龙14条（如前者，去土），舶上茴香7钱半（去土，炒），蛜（虫祁）14条（头尾全者，去土，切下顶盖，去子并瓤，瓦上焙），蜈蚣（赤足者，葱袋叶裹，醋炙）2条，热艾2两，大木瓜2只。
出处：《魏氏家藏方》卷八。
主治：年深日近干湿诸般脚气。
加减：None
功效：None
用法用量：每服15丸，10日后加5丸，至30丸止，空心温酒或熟水送下。
制备方法：将青盐、熟艾拌入在木瓜内，用顶盖定蒸十数次，烂为度，研成膏，再入少许无灰酒煮面和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
