序号：48354
名称：神效四时加减养肺汤
组成：紫菀1两，五味子1两，干姜1两，款冬花1两，半夏1两，人参1两，糯米1两，杜仲1两，白术1两，桂1两，柴胡半两，茯苓半两，甘草半两，陈皮半两，丁香半两，细辛半两，射干半两，山药半两，独活半两，防风半两，钟乳半两。
出处：《鸡峰》卷十一。
主治：肺气不足，病苦气逆，胸腹满，咳逆上气抢喉，喉中闭塞，咳嗽短气，自惊，或笑，或歌，或怒无常，或干呕，言语过多，触风邪便发咳嗽，四时往来不愈。
加减：夏、秋，柴胡、独活、射干、细辛减半；久嗽虚寒人，加蜀椒1分。
功效：None
用法用量：每服2钱，水1大盏，加生姜3片，大枣1个，同煎至5分，去滓稍热服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
