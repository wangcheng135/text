序号：66498
名称：小三生丹
组成：朱砂（细研）2两，水银（细研）2两，硫黄（细研）2两，生铁15斤（磨洗后，以大火烧赤，投3斗浆水中淬10遍）。
出处：《圣惠》卷九十五。
主治：None
加减：None
功效：暖下元，益精气，黑髭鬓，驻颜色。
用法用量：上取平底铛1口，以前3味，用淬铁浆水煮之，21日，常令如鱼目沸，水耗，即暖浆水添之，日满，挑取少许于火上试之，如有鬼焰，又煎之，以无焰为度，泣干，却入瓶子中按实，以烧盐盖覆，如法固济，用火半斤，养7日满，以火5斤，煅令通赤，待冷，破瓶取之，投汤盆中，淘去盐味，澄取药，晒干，细研如粉，以甘草、余甘子水煮半日，出火毒，又研，以葛粉糊为丸，如麻子大。每服7丸，空心以温酒送下。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌羊血。
附注：None
