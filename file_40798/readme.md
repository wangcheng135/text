序号：40812
名称：软朱砂法
组成：颗块墙壁辰砂1两（研为粉）。
出处：《百一》卷一引赵从简方。
主治：废忘。男子元气虚损，酒色损惫，面色黄瘦，多睡少力，、精神恍惚，胸膈不利，多倦怠，夜多小便；兼治妇人血气血风等疾，及瘴疾。
加减：None
功效：补心气，轻健手足。壮筋骨，延年益寿。
用法用量：空心以新水送下7丸。若用1匙头许，以温酒化下亦得。
制备方法：上以好清麻油4两，白及2两，木通1两，于油内煎令焦黄，滤去滓，放令油如人体温；于瓷器内和辰砂末令如（饣兹）膏，以皂角浆水洗去油，却用新汲水洗去皂角浆，干瓷盒内以新水养之，每日早晨换水，旋丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：软朱砂延寿水仙丹（《普济方》卷二二三引《经效济世方》）。
