序号：3383
名称：太一流金大道散
组成：踯躅花1钱半，肉桂1钱半，干姜1钱半，附子（生）1钱半，细辛2钱，朱砂2钱，皂角（肉炒）2钱，藜芦2钱，牡丹皮4钱。
出处：《幼幼新书》卷十三引《灵苑》。
主治：大人小儿一切风，卒中或厥，不醒人事，搐搦戴目，伤寒头风等。
加减：None
功效：None
用法用量：气厥及厕上中恶，卒中或厥诸疾，不醒人事，手足搐搦，戴目闷绝，及元气、血气上冲，痛闷欲绝者，头风夹脑风，伤寒不解者，吹鼻内，大人1字，儿半字，嚏效。伤寒，以葱汁为丸，如绿豆大，服7丸，薄荷、姜、腊茶送下；伤寒久不解者，葱、酒下10丸。
制备方法：上药各为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
