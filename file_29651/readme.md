序号：29659
名称：附子丸
组成：附子1两（炮裂，去皮脐），肉苁蓉1两（酒浸1宿，锉去皱皮，炙干），巴戟1两，防风3分（去芦头），当归1两，羌活3分，桂心3分，萆薢3分（锉），酸枣仁3分（微炒），牛膝1两（去苗），木香3分，白蒺藜3分（微炒去刺），补骨脂1两（微炒），鹿茸2两（去毛，涂酥炙微黄），石斛1两（去根，锉），桃仁1两（汤浸，去皮尖双仁，麸炒微黄），白茯苓1两。
出处：《圣惠》卷三十。
主治：虚劳伤惫，腰脚疼痛。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
