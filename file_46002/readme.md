序号：46017
名称：洋参保肺丸
组成：米壳4两（折膏1两8分），杏仁（去皮，炒）2两，橘皮2两，枳壳（炒）2两，甘草2两，玄参（去芦）2两，川贝母2两，五味子（炙）1两，麻黄1两，生石膏1两，砂仁1两。
出处：《北京市中药成方选集》。
主治：劳伤久嗽，肺气虚弱，咳嗽痰喘，春秋必犯，睡卧不安。
加减：None
功效：滋阴补肺，止咳定喘。
用法用量：每服2丸，1日2次，温开水送下。
制备方法：上为细末，过罗；如用米壳膏，改用细粉16两，加米壳膏1两8分。每20两细粉兑洋参面1两5钱。上药混合均匀，炼蜜为丸，重2钱；如用米壳膏每丸改重1钱7分，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
