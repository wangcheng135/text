序号：32585
名称：羌活退翳丸
组成：黑附子（炮）1钱，寒水石1钱，酒防己2钱，知母（酒炒）3钱，牡丹皮3钱，羌活3钱，川芎3钱，酒黄柏5钱，生地黄（酒洗，炒）5钱，丹参5钱，茺蔚子5钱，酒当归身5钱，柴胡5钱，熟地黄8钱，芍药1两3钱。
出处：《兰室秘藏》卷上。
主治：内障，右眼小眦青白翳，大眦微显白翳，脑痛，瞳子散大，上热恶热，大便秘涩，小便如常，遇天气暄热，头痛睛胀。
加减：翳在大眦，加葛根、升麻；翳在小眦，加柴胡、羌活。
功效：None
用法用量：每服50-70丸，空心白汤下，宿食未消，待饥则服之。药后省语言，以食压之。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：滋阴地黄丸（《医学纲目》卷十三）、地黄丸（《普济方》卷七十九）、柴胡退翳丸（《银海精微》卷下）。
