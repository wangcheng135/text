序号：49859
名称：大车螯散
组成：车螯1个（黄泥固济，煅通红，去泥，置地上去火气），大戟（净洗）半两，芫花（醋炒）半两，漏芦半两，甘草（炙）半两，槟榔半两，甘菊（去梗叶）3分，大黄3分，腻粉1分。
出处：《卫济宝书》卷下。
主治：瘭、癌，疽，痼四发，初觉发作寒热，或不寒只热之状，疼痛肿赤，或在头背，_x000D_
或在肢节，或不见形状者。
加减：None
功效：None
用法用量：每服2钱，加车螯末2钱，腻粉1钱，拌和。于更初用瓜蒌酒下。如人行5里，即下恶毒脓血2-3行，不用止自住。凡后生只用1服，老少加减。
制备方法：上药各为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
