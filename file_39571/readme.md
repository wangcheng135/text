序号：39581
名称：细辛丸
组成：细辛2两，杏仁（去皮尖双仁，熬）2两，泽泻2两，干姜2两，白术2两，茯苓2两，桂心2两，甘草（炙）2两，附子（炮，去皮）5分，蜀椒（去目闭口者，汗）5分，大黄5分，木防己5分，芫花1两，甘遂1两。
出处：《千金翼》卷十五。
主治：冷热不调，痰结胸中强饮，百处不安。
加减：None
功效：消饮，去结澼，令胸膈无痰，无逆寒之患，又令人不眩满迷闷。散发五脏六腑三焦。久服强气。
用法用量：每服2丸，以酒送下，1日2次，不能者如大豆2丸，以知为度。饮家困于痰澼，服药患困者，参服此丸，暨相发助，又不令越逸。
制备方法：上为末，别治杏仁如脂，合捣百杵，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
