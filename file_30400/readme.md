序号：30408
名称：返魂散
组成：消石1两，牵牛1两，半夏3分，僵蚕（去头丝）1两，天南星1两。
出处：《普济方》卷六十三。
主治：喉咙肿痛，饮食不下，喘息不通，头项俱肿，命欲临死者。
加减：None
功效：None
用法用量：以旧笔管盛1钱，吹在痛处，停候片刻，吐出涎后，喘息已通，即便停药；若气未通，肿处未消未破，更进吹1钱半，待气通即住，若未通，加药吹之，以气通为度。若喉中喘息已通，更服下方。麻黄、荆芥穗、羌活、牡丹皮（去心）、射干、僵蚕（去丝）、连翘、消石各1分，大黄、牵牛各半两，半夏3铢，上为细末，用白面大小相滚，熟水为丸，如梧桐子大。每服50丸，食后煎葱汤送下；若作散，每服3钱，水1盏3分，煎1-2沸，食后合滓服；若服时吐逆，即浓煎去滓，服1-2盏，以通取转为度。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：量病势大小，使药轻吹，吹重则人闷绝。
附注：None
