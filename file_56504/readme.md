序号：56519
名称：大茵陈汤
组成：茵陈1两半，黄柏1两半，大黄3两，白术3两，黄芩1两，栝楼根1两，甘草1两，茯苓1两，前胡1两，枳实1两，栀子20枚。
出处：《千金》卷十。
主治：内实热盛发黄，黄如金色，脉浮大滑实紧数者。
加减：None
功效：None
用法用量：上(口父)咀。以水9升，煮取3升，分3服。得快下，消息3-4日，更治之。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：发黄本乎湿热，湿热本乎脾虚。此以枳、术、苓、甘加入茵陈蒿汤中，助脾逐湿；佐以前胡、栝楼下气通津，黄柏、黄芩燥湿清火，皆本经治诸热黄疸之专药。较茵陈蒿功力倍常，因以大字衔之。
用药禁忌：None
附注：None
