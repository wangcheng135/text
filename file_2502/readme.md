序号：2502
名称：午时茶
组成：陈茶叶8两，白芍1两，焦苍术2两，枳壳1两，焦麦芽2两，淡黄芩1两，焦山楂2两，防风1两，广藿香2两，苏叶1两，白扁豆2两，青蒿1两，飞滑石2两，陈皮1两，川厚朴1两，木瓜1两，砂仁1两，白芷1两，桔梗1两，茯苓1两，川羌活1两，甘草5钱，葛根1两，川黄连5钱，西香茹1两，薄荷5钱，西茵陈1两，大腹皮1两（煎汁）。
出处：《全国中药成药处方集》(南京方)。
主治：外感风寒暑湿，内伤饮食停滞，恶寒发热，头晕身倦，胸闷腹胀。
加减：None
功效：None
用法用量：每服1块或2块，开水泡或微煎服之。
制备方法：上药生晒，为粗末，大腹皮煎汁兑入所用稀粥内打糊，或兑入面糊内，和上药粗末，做成长方形块子，每块计重3钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
