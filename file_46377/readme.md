序号：46392
名称：活络通经丸
组成：川乌头2两（1两生，不去皮尖；1两炮，去皮尖），草乌2两（制如上法），木鳖子3两3分（别研），斑蝥100个（去头足并翅，醋煮香熟，焙干），乌蛇（酒浸，去皮骨，焙）1两，白花蛇（酒浸，去皮骨，焙）1两，好墨（火煅）1两，白胶香1两（别研），当归1两半，五灵脂3两3分。
出处：《三因》卷二。
主治：半身不遂，口眼斜，瘫痪诸风。
加减：None
功效：活络通经，宣导凝滞，壮筋骨，助血脉，起偏废。
用法用量：每服1丸，空心、食前温酒盐汤嚼下。
制备方法：上为末，将木鳖子末醋研为膏，和黑豆末1斤，好醋拌，1两作10丸，以墨为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
