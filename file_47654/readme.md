序号：47669
名称：千捶膏
组成：杏仁49粒，蓖麻仁49粒，琥珀（灯心同研）3分，冰片3分，珍珠（豆腐包煮）6分，麒麟竭6分，当门子6分，乳香（去油）6分，没药（去油）6分，铜绿6分，黄丹6分，龙骨6分，轻粉6分，水安息（龙眼肉大）3块，松香（入锅内，小火化开，用麻布滤去滓，冷定，用豆腐水煮数次，再用绿豆汤煮3次，又用葱、韭、生姜汁各1钟，煮干，研细末）8钱。
出处：《疡医大全》卷十八。
主治：瘰疬。
加减：若内觉有脓未熟，恐穿溃难于收功，可加木鳖子（去壳）7枚、黑驴蹄（研细）5分于膏内，即能隔皮取脓。
功效：None
用法用量：瘰疬初起，贴之自消；将溃贴之，毒从毛窍中出，不致穿溃。
制备方法：先将杏仁、蓖麻捣如泥，次将前药细末逐渐加入，捶千余下，用大红缎摊贴。
临床应用：None
各家论述：None
用药禁忌：忌见火。
附注：None
