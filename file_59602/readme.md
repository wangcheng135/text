序号：59617
名称：调中益胃汤
组成：人参2钱（如无人参，以明党参代之），苍术4钱（炒），熟地4钱，白芍3钱（酒炒），草果2钱，黄耆3钱（蜜炙），陈皮3钱，茯苓3钱，甘草1钱。
出处：《医学探骊集》卷三。
主治：中恶。凡人或入枯井，或入山洞，或入冷室，忽然晕倒，口目紧闭，不省人事。
加减：None
功效：None
用法用量：加生姜3片，水煎服。
制备方法：None
临床应用：None
各家论述：此方以人参为君，扶其元气；佐以黄耆、熟地，气血双补；苍术、草果能驱败气而扶正气；陈皮、茯苓调其中气；白芍敛阴，甘草和中。细审此方，似乎平庸，然与正气被伤之人服之，不寒不燥，最为稳妥。
用药禁忌：None
附注：None
