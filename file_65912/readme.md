序号：65928
名称：萎蕤丸
组成：萎蕤4两，车前子4两，熟干地黄（焙）4两，升麻1两半，黄芩（去黑心）1两半，秦艽（去苗土）1两半，枳壳（去瓤，麸炒）1两半，白茯苓（去黑皮）1两半，黄连（去须）1两半，独活（去芦头）1两半，地骨皮1两半，决明子（微炒）1两半，山栀子仁1两半，白槟榔（生，锉）1两半，赤芍药2两，芎2两，秦皮1两。
出处：《圣济总录》卷&mdash;一一。
主治：眼生翳膜，疼痛昏涩，视物不明。
加减：None
功效：None
用法用量：每服30丸，食后熟水送下，1日2次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
