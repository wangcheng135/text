序号：57639
名称：大造丸
组成：紫河车1具（米泔水洗净，新瓦上焙干。用须首生者佳。或云砂锅随水煮干，捣烂），败龟版（年久者，童便浸3日，酥炙黄）2两，黄柏（去粗皮，盐酒浸，炒褐色）1两5钱，杜仲（酥炙，去丝）1两5钱，牛膝（去苗，酒浸，晒干）1两2钱，怀生地黄2两5钱（肥大沉水者，纳入砂仁末6钱，白茯苓一块重2两，稀绢包，同入银罐内，好酒煮7次，去茯苓不用），天门冬（去心）1两2钱，麦门冬（去心）1两2钱，人参1两。
出处：《扶寿精方》。
主治：男子阳萎遗精，妇人带下，素无孕育；大病后久不能作声，足痿不任地者；虚损劳伤，咳嗽潮热。
加减：夏月，加五味子7钱；妇人，加当归2两，去龟版；男子遗精，妇人带下，并加牡蛎（煅粉）两半。
功效：None
用法用量：每服80-90丸，空心、临卧盐汤、沸汤、姜汤任下；寒月好酒下。
制备方法：上除地黄另用石木春1日，余共为末，和地黄膏，再加酒米糊为丸，如小豆大。
临床应用：1.阳萎、足痿：一人病弱，阳事大萎，服此二料，体貌顿异，连生四子。一人病痿，足不任地者半年，服此后能远行。_x000D_
2.补虚益寿：一妇年六十已衰惫，服此寿至九十，犹强健。
各家论述：《医方集解》：此手太阴、足少阴药也。河车本血气所生，大补气血为君；败龟版阴气最全，黄柏禀阴气最厚，滋阴补水为臣。杜仲润肾补腰，腰者肾之府；牛膝强筋壮骨，地黄养阴退热，制以茯苓、砂仁，入少阴而益肾精；二冬降火清金，合之人参、五味，能生脉而补肺气。大要以金水为生化之源，合补之以成大造之功也。
用药禁忌：None
附注：河车大造丸（《不居集》上集卷二）。
