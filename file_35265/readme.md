序号：35275
名称：乳香散
组成：乳香3两，天麻末3两，牛黄1两，麝香1两（细研），雄黄1两（细研），胡麻2斤（净水淘49遍，去浮者，取沉者用蒸，从卯时蒸至酉时止，用黑豆压之，次又用1重湿土盖之，恐釜内汤少，仍须时添热汤至酉时后取出，炒令香燥即住，候冷，捣罗为细散，与诸药末同拌令匀）。
出处：《圣惠》卷二十四。
主治：大风疾，肌肉欲坏，眼色变改，眉发落，语声散。
加减：None
功效：None
用法用量：每次2钱，空心时及晚食前用腊面条清调下。初服3日心逆，4日多睡，5日腰脚痛，可如醉人，是其候也，相次渐愈，只须长服此药，四癞必愈。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
