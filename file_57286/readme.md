序号：57301
名称：消痞狗次膏
组成：羌活4两，独活4两，防风4两，麻黄4两，威灵仙4两，桂枝4两，制川乌4两，制草乌4两，三棱4两，蓬莪术4两，当归4两，赤芍4两，乌药4两，秦艽4两，生香附4两，白芷4两，桃仁4两，草果2两，常山4两，大黄4两，槟榔2两，红花4两，干姜4两，制白附2两，五加皮4两，生地4两，僵蚕4两，全蝎4两（以上为甲组药），麝香1钱，阿魏2两，肉桂1两，白胡椒1两，公丁香5钱，广木香5钱，制乳香1两，制没药1两，山奈1两，苏合油1两（以上为乙组药）。
出处：《中药成方配本》（苏州）。
主治：癥瘕积聚。
加减：None
功效：软坚消痞。
用法用量：烘软贴患处。
制备方法：将甲组药用麻油50斤，浸1日，用文火煎至药枯，去滓滤清，再煎至滴水成珠，加入东丹（炒热）19斤，渐渐下入，搅匀为度，约成膏53斤。在摊膏时加入乙组药（研末）及苏合油（烊化）和匀，每张连皮，大者1两，小者6钱。
临床应用：None
各家论述：None
用药禁忌：孕妇慎用。
附注：None
