序号：76965
名称：黑神丹
组成：黑附子（炮裂，去皮脐）1两，天麻（去芦头）半两，天南星（炮裂）半两，桂（去粗皮）半两，半夏（浆水煮，焙干）半两，麻黄（去根节）半两，干姜（炮）半两，草乌头2两（炮裂，去皮脐），白附子（炒黄色）半两，麝香（去毛，细研）1两，天雄2两（慢火上炙热，好酒内蘸，如此7返，令折药力，更用童便内蘸7返，撅一坑子约深5寸，先用热火坑内炙干，去火，坑内洒酒约半升，天雄在内，瓷碗盖定，周围泥了，不教漏气，冷定取出用之）。
出处：《御药院方》卷十一。
主治：妇人产后大发热，消渴不止，烦躁不休，或汗病后胃脘硬、爱水者。
加减：None
功效：None
用法用量：发热、渴，用蜜水化服；欲出汗，热酒化服；汗病后胃脘硬，爱水，依前用蜜水化服1丸。
制备方法：上各修制，共为细末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
