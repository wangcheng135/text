序号：34856
名称：麦门冬散
组成：麦门冬1两（去心），石膏2两，知母1两，茅根1两（锉），葛根（锉）半两，玄参半两，川升麻半两，甘草（炙微赤，锉）半两，赤芍药半两，麻黄（去根节）半两，川大黄（锉碎，微炒）半两，子芩半两，人参（去芦头）半两，赤茯苓半两，远志（去心）半两。
出处：《圣惠》卷七十四。
主治：妊娠5月至10月足，患时气，5-6日不得汗，口干多吃冷水，热气上冲，喘急闷乱。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加淡竹茹1分，生姜半分，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
