序号：4193
名称：木香丸
组成：木香1两，肉豆蔻（去壳）1两，陈橘皮（汤浸去白，焙）1两，干姜（炮裂）1两，附子（炮裂，去皮脐）1两，郁李仁（去皮尖，炒，别研）1两，麦门冬（去心，焙）1两，熟艾（炒）2两，鳖甲（醋浸，炙，去裙襴）2两，陈曲（炒）2两，柴胡（去苗）2两，厚朴（去粗皮，涂生姜汁炙）3两，钟乳（炼成粉者）半两，桂（去粗皮）半两。
出处：《圣济总录》卷八十七。
主治：冷劳便利不调，腹胀呕逆，羸困少力。
加减：None
功效：None
用法用量：每服20丸，空心温酒送下；米饮下亦得。
制备方法：上药捣罗13味为末，入郁李仁相和研匀，用猪肝1具，去脂膜细切，以头醋3升，同熬令醋尽，烂研入药末，相和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
