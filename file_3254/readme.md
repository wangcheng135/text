序号：3254
名称：九仙丸
组成：黑驴肾（并肾子、腰子，全切片，以伏龙肝为末，铺锅底，将前物铺上，再用伏龙肝末盖之，慢火焙干，去伏龙肝），枸杞子2两，巴戟（去心）4两，核桃肉（去皮）4两，莲蕊2两，白芍（酒洗）2两，当归（酒炒）2两，破故纸（炒）2两，茯苓2两，胡芦巴（酒炒）2两，芡实2两，肉苁蓉（酒洗）2两，牡蛎（煅）2两，牛膝（酒蒸）2两，龙骨（煅，童便淬）2两，杜仲（盐水炒）2两，沙苑2两，蒺藜2两（炒），大茴1两。
出处：《惠直堂方》卷一。
主治：None
加减：None
功效：益肾固精，壮阳种子。
用法用量：每服2钱，清晨开水送下。如欲种子，可日3服，先忌房事3-7日效。此药须长服为妙。
制备方法：上为末，酒糊为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
