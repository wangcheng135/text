序号：71945
名称：滋阴降火清肺养脾丸
组成：当归（全用，酒浸，焙干）1两，川芎（酒浸）5钱，白芍（酒炒）1两，生地（酒洗，姜汁浸透，焙）1两，熟地1两（同上制），白术（去梗，泔浸，土炒）2两，南知母（去毛，酒炒）1两，黄柏2两（盐、酒焙），阿胶1两（蛤粉炒），麦冬（酒浸，去心，姜汁制透）1两，天冬（去心）1两（同上制），龟版2两（酒炙透），黄芩（酒炒）1两，陈皮（盐水浸）1两，大甘草（炙）5钱，瓜蒌仁（炒）1两，贝母（去心）1两，五味子5钱，黄连（姜汁浸炒），5钱，白茯苓（去皮）1两，紫苑（酒洗）1两，桔梗（去芦）1两，山栀（炒）1两。
出处：《便览》卷三。
主治：久痰喘嗽，胸膈痞闷，饮食少进，面黄肌瘦，或咯吐唾衄过血者，及已成未成劳疾，发热盗汗。
加减：None
功效：None
用法用量：每服100丸，空心、临卧用温水送下；有痰用姜汤送下。
制备方法：上为末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：久泄者不宜服。
附注：None
