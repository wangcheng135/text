序号：41147
名称：金牙散
组成：金牙（研）3分，曾青（研）3分，矾石（研，泥裹烧半日）3分，丹砂（研）3分，雄黄（研）3分，朴消（研）3分，寒水石（研）3分，代赭（研）3分，龙骨（研）3分，犀角屑3分，獭肝（炙）3分，鹳骨（炙）3分，狸骨（炙）3分，巴豆（去心皮，熬）3分，大黄3分，野葛皮（炙）3分，牛黄（别入）2分，麝香（别研）2分，升麻2分，桂心2分，附子（生用，去皮）2分，鬼臼2分，鬼督邮2分，黄环2分，鸢根（本草有鸢尾，此云鸢根即是用鸢尾之根也）2分，青木香2分，牡蛎（熬）2分，苏合香（别研入）2分，常山2分，茯苓2分，黄耆2分，知母2分，龙胆2分，露蜂房1分，玉支1分，菵草（1本作茵芋）1分，鬼箭羽1分，徐长卿1分，石长生1分，蜀漆1分，当归1分，桔梗1分，白薇1分，蜈蚣1枚（炙），蜥蜴1枚（炙），芫青（炙）39枚，地胆（炙）39枚，亭长（炙）39枚，椒49枚（汗）。
出处：《外台》卷十七引《苏恭方》。
主治：脚气毒遍内外，烦热，口中生疮，狂易叫走；及解诸石草热药毒发，邪热卒黄，瘴疫毒疠，卒死温疟，五尸五注，心腹诸疾，绞刺切痛，蛊毒鬼魅，野道热毒，小儿惊痫百病。
加减：None
功效：辟诸恶厉。
用法用量：以汤（疑为饧）如桃李许，和散3分匕，或如枣核服之；常患者，1日2次。平患取利吐者，服4分匕。或以绛袋裹方寸匕，3匕带之。或合药用腊月王相日
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：忌食生冷、芦笋、生葱菜、猪肉，冷水、醋物、陈臭、生血等物。
附注：None
