序号：29354
名称：连翘丸
组成：连翘（洗）240两，陈皮240两，青皮（洗）160两，蓬莪荗（炮）160两，肉桂（去粗皮，不见火）160两，好墨（煅）160两，槟榔80两，牵牛子（碾，取末）220两，三棱（炮）249两，肉豆蔻25两。
出处：《局方》卷三（绍兴续添方）。
主治：男子、妇人脾胃不和，气滞积聚，心腹胀满，干呕醋心，饮食不下，胸膈噎塞，胁肋疼痛，酒积面黄，四肢虚肿，行步不能；及久患赤白痢及大肠风秘，脾毒泻血。
加减：None
功效：None
用法用量：每服30丸，生姜汤送下；久患赤白痢及大肠风秘，脾毒泻血，黄连煎汤送下；妇人诸疾，姜醋汤送下，不拘时候。
制备方法：上为末，面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
