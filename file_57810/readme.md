序号：57825
名称：狼毒丸
组成：雄黄（生），狼毒，肉桂（去粗皮，不见火）4钱，大附子（炮，去皮脐）1两2钱，汉椒（去目，炒出汗）1两2钱，干漆（炒烟绝）1两2钱，甘遂（生用）1两2钱，当归半两（去芦），芫花（醋炒）1两，川大黄（生）1两半，槟榔（生用）1两半，大戟（生）1两，桃仁（连皮炒）1两，茱萸（生）1两，厚朴（去皮，姜制）1两，干姜（炮，洗）1两，枳壳（生，去瓤）1两，犀角（生用）1两，鳖甲（炙）1两4钱，银川柴胡（生用）1两4钱。
出处：《魏氏家藏方》卷九。
主治：腹胀水肿。
加减：None
功效：None
用法用量：每服10丸，以温汤送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中雄黄、狼毒用量原缺。
