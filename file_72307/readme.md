序号：72323
名称：滋肾丸
组成：当归1两，熟地1两，枸杞1两，白术1两，白芍1两，白茯1两，牛膝1两，胆草1两，覆盆子1两，肉苁蓉1两，川芎1两，玄参1两，菟丝子1两半（酒煮），苍术（米泔水浸）7钱，防己7钱，厚朴7钱，远志7钱，黄柏7钱，知母7钱，青葙子7钱，石决明7钱，蒺藜7钱，香附5钱，蒙花5钱，磁石（煅，醋淬）5钱，砂仁5钱，甘草4钱，人参3钱。
出处：《眼科全书》卷三。
主治：乌风内障。不痒不痛，其瞳仁不开大，渐渐昏沉，又无翳障，是由气涩使然。
加减：None
功效：None
用法用量：每服30-50丸，盐汤或酒下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
