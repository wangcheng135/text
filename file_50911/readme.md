序号：50926
名称：大圣泽兰散
组成：泽兰3两，生地5两，白芍两半（炒），当归3两，石膏3两，人参两半，甘草6钱，白薇两半，川芎8钱，柏子仁3两，茯苓两半，白术两半（炒）。
出处：《医略六书》卷三十。
主治：产后狂病，脉洪数软涩者。
加减：None
功效：None
用法用量：砂糖灰汤煎3钱，去滓温服。
制备方法：上为散。
临床应用：None
各家论述：产后气血大虚，阳明伏热而上扰心包，故神明失指，狂叛无知焉。今以四君补气，四物补血，石膏泻火清热，白薇抑阳扶阴，泽兰通利血脉，柏仁安定心神。为散，砂糖灰汤下，使血气安定，则伏热自解，而瘀去新生，心神得养，何有狂叛之患哉。
用药禁忌：None
附注：None
