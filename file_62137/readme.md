序号：62153
名称：救苦黄耆散
组成：黄耆1两5钱，甘草1两5钱，当归1两5钱，瓜萎根1两5钱，芍药1两5钱，悬蒌1对，熟地黄不拘多少，金银花2两，皂角棘针（为引）。
出处：《杂类名方》卷十。
主治：诸恶疮痈疖。
加减：None
功效：None
用法用量：疮在上，食后临卧服；在下，空心服之。
制备方法：上（口父）咀。每服5钱，无灰好酒1升，同引子装于瓷瓶内，将瓶用笋叶封，坐于锅内，上以大盆覆锅口，盆外用黄土封之，无令出气，煮之，外闻药香为度。取出瓶，澄定饮清，将药滓再添酒1升，依前煮服，若不饮酒者，以水煮服，若酒少者，酒、水各半煮服。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
