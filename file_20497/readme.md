序号：20504
名称：血余固本九阳丹
组成：血余（选黑者，不拘男女，用皂荚煎汤洗净，清水漂过，入口无油垢气为度，晒干，置大锅内，用红川椒去梗目，与发层铺上，用小锅盖定，盐泥秘塞上，锅底上用重石压之，先用武火煅炼1柱香，后用文火半柱香，以青烟去尽，无气息为度。冷定取出，研末，双绢筛过）1斤，何首乌（先用米泔水浸，竹刀刮去皮）赤者8两，白者8两，淮山药（共何首乌去皮，竹刀切成片，用黑豆2升，上下铺盖，蒸熟晒干）8两，赤茯苓（去皮，牛乳浸1日夜）8两，白茯苓（人乳浸1日夜）4两，破故纸（酒拌，沙锅炒以香为度）4两，菟丝子（人乳1碗，酒半碗，浸1夕，饭锅上隔布蒸熟，晒干，微炒，研为末）4两，枸杞子（去蒂梗，酒拌蒸熟）4两，生地黄（酒蒸）半斤，苍术（去皮，为未末）半斤，熟地黄（酒蒸）半斤，龟版（酥油炙）半斤，当归（去尾，酒浸）4两，牛膝（酒浸，黑豆蒸）4两。
出处：《广嗣纪要》卷四。
主治：None
加减：None
功效：调元固本，种子。
用法用量：每服50-60丸，药酒送下（药酒方：当归、生地黄、五加皮、川芎、芍药、枸杞子各2两，核桃肉1斤，砂仁5钱，黄柏1两，小红枣200个，用无灰白酒36斤，内分5斤，入药装坛内密封，隔汤煮之，冷定去滓，入前酒密封用）。
制备方法：上药各为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《景岳全书》引作“铁笛丸”。
