序号：75112
名称：蒴藋膏
组成：蒴藋根（切）1升，蒺藜子1升，附子3两，独活3两，犀角屑3两，蔷薇根3两，白芷3两，防风3两，苦参3两，白及3两，升麻3两，白敛3两，防己3两，川椒2两，莽草2两，青木香2两，蛇床子2两，蛇衔草2两，茺蔚子（切）1升，枳实5枚（炙），茵芋2两半（切）。
出处：《外台》卷十五引《延年秘录》。
主治：身痒，风瘙瘾疹。
加减：None
功效：None
用法用量：用摩风疹。
制备方法：上切，以苦酒渍，令淹匝1宿，明旦铜器中炭火上用猪膏5升煎，令三上三下，以候白芷色黄膏成，绞去滓，纳不津器中。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
