序号：32679
名称：诃黎勒丸
组成：诃黎勒皮1两，大腹皮1两（锉），半夏1两（汤洗7遍，去滑），桑根白皮1两（锉），前胡1两（去芦头），枳实1两（麸炒令微黄），汉防己半两，紫苑3分（洗去苗土），杏仁1两（汤浸，去皮尖双仁，麸炒微黄，别研如膏），甜葶苈1两（隔纸炒令紫色，别研如膏）。
出处：《圣惠》卷十一。
主治：伤寒气壅，心腹不利，上气咳嗽，腹胁妨闷。
加减：None
功效：None
用法用量：每服20丸，以生姜汤送下，不拘时候。
制备方法：上为末，加杏仁、葶苈更研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：大腹皮丸（《圣济总录》卷二十四）。
