序号：35106
名称：乳灵丸
组成：川乌（去皮脐，生用）半两，生干地黄（酒浸1宿）1两，草乌头（小者，去皮脐，炒，去盐）半两，木香（不见火）2钱半，五灵脂（拣去砂石，微炒）1两，麻黄（去根节，微炒去汗）1两，自然铜（火煅7次，米醋淬7次，另研）1两，虎胫骨1两（酥涂，火炙存性），干木瓜（酸者）2两，滴乳香（好者，和灯心研为末，去灯心）2钱半，败龟（米醋浸3日，炙黄色，再用醋淬）7钱半。
出处：《得效》卷十三。
主治：None
加减：None
功效：None
用法用量：每次以带皮生姜自然汁化开1丸，热酒服，更以酒半盏送下，病在下，空心食前服；病在上，食后停少时服，1日2次。口有少麻，勿疑。_x000D_
【主】痛风，数处不能举动，日夜呻吟。
制备方法：上研细如粉，炼蜜去上白沫，为丸如龙眼大，每1两重作12丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
