序号：72291
名称：滋肾固冲汤
组成：生地5钱，枸杞子3钱，山萸肉3钱，煅龙骨1两，煅牡蛎1两，龟版4钱，黄柏3钱，旱莲草4钱，侧柏叶1两，血余炭3钱，藕节炭4钱。
出处：《妇产科学》。
主治：功能性子宫出血，属肾阴不足者。症见出血量多，血色鲜红，两耳响鸣，舌红或光剥，脉象细数。
加减：如心火亢甚者，加黑山栀3钱，黄连5分；肝阳偏亢者，加丹皮3钱，贯众炭4钱。
功效：滋肾清热，养血止血。
用法用量：None
制备方法：None
临床应用：None
各家论述：方中生地、枸杞养血；山萸、龟版、黄柏滋肾清热；龙骨、牡蛎益肾固冲；旱莲、侧柏、血余炭、藕节炭止血。
用药禁忌：None
附注：None
