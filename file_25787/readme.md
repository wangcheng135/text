序号：25795
名称：戊己丸
组成：茴香3两（拣净），甘草1两（炙），胡椒5两（拣净），人参1两，白术2两，朱砂半两，白茯苓3两，香附子半两。
出处：《续本事》卷一。
主治：新婚男子女人，素禀虚寒滑泄。丈夫、妇人禀赋怯弱，饮食无味，气血衰败，肌肉不生，项背拘紧，腰脚无力，胸膈膨胀，多睡少寤，终日昏蒙，夜多异梦，及积年脾蛊时下，恶心噫酸吐水，小儿吐乳，大人翻胃。
加减：None
功效：护脾开胃，进饮食，长肌肉，生气血，化精益髓，全胃气，丹田不竭，肾经不虚。
用法用量：每服20丸，空心白汤送下，日2次。
制备方法：上为细末，生姜汁打面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
