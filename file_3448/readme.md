序号：3448
名称：太上灵应无比神异膏
组成：露蜂房1两（锉碎），蛇蜕皮半两（细剪），玄参半两（细锉），真绵黄耆3分（细锉），真杏仁1两（去皮尖，略捣破），黄丹5两（虢州者佳），童儿童女乱发（净洗，晒干）如鸡子大。
出处：《传信适用方》卷三。
主治：痈疽疮疖。
加减：None
功效：None
用法用量：摊用如常，日换2次，夜换1次，凡疮破有头，使用洗药洗讫，用此薄贴敷之。
制备方法：上药先用真麻油(不用菜子油)1斤入铛，便下发，慢火熬，候发消尽，次入杏仁，滤去滓；再入档，入余药熬，候焦黄，再滤去滓；次入黄丹，候变色，以新柳木篦子搅五千搅，滴入水中，看软硬得所，然后倾净瓷罐子，如尚未硬，更搅三百搅也。
临床应用：None
各家论述：None
用药禁忌：None
附注：神异膏(《外科精要》卷下)、太上无比灵应神异膏(《普济方》卷二八八)。
