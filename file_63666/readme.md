序号：63682
名称：清热平肝汤
组成：茵陈5钱，醋柴胡3钱，酒胆草3钱，小蓟5钱，赤芍4钱，丹皮4钱，石见穿5钱，白矾1钱，郁金3钱，泽兰1钱半。
出处：方出《中医原著选读》引关幼波方，名见《古今名方》。
主治：肝胆湿热型：多见于迁延性慢性肝炎、脂肪肝或合并胆道感染，谷丙转氨酶长期不降或胆固醇增高者。症见口干口苦，两胁持续作痛，小便黄赤，大便干燥，身体日见增胖，胃腹作胀，有时烦躁，脉弦滑，舌苔白腻或黄厚，舌质红。
加减：厌油者，加藿香3钱，佩兰3钱；胆固醇高者，加山楂5钱，草决明5钱；大便干者，加酒军3钱；小便黄者，加六一散（包）4钱。
功效：清热平肝，凉血解毒。
