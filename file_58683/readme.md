序号：58698
名称：秘传海上香连丸
组成：真川连（净）8两，木香2两（不见火），川当归1钱，白芍1钱（炒），川芎1钱5分，车前子1钱，生地黄2钱，厚朴2钱（炒），白茯苓1钱5分，木通8分，枳壳1钱（炒），陈皮2钱（炒），苍术1钱（炒），吴茱萸2钱（炒）。
出处：《医林绳墨大全》卷一。
主治：赤白痢疾，泄泻。
加减：None
功效：None
用法用量：每服30丸，白痢姜汤送下，红痢茶汤送下，泄泻百沸汤送下。
制备方法：上药除黄连、木香，用水7碗，煎至5碗，去滓，用汁煮黄连，以干为度，取起晒干，用木香为细末，用老米粉、好醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
