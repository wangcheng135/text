序号：22745
名称：时症丸
组成：茅苍术3钱，杏仁3两，公丁香6钱，九节菖蒲6钱，千金霜6钱，麝香3钱，梅片3钱，马牙消3钱，沉香3钱，蟾酥1两，郁金1两，山茨菇1两，胆星1两，天竺黄1两，五倍子1两，青礞石（醋煅7次）2两，硼砂2两，野大黄6两，明天麻3两4钱，明雄黄3两4钱，牛黄1钱5分，红芽大戟5钱。
出处：《全国中药成药处方集》（沈阳方）。
主治：山岚瘴气，中暑中恶，绞肠痧症，腹痛厥冷，恶心呕吐，吐泻不出，不省人事，口噤目瞪，及毒虫咬伤。
加减：None
功效：兴奋神经，清热解毒。
用法用量：每服7丸，不见效者可加1倍，用白开水送下。如有蚊虫咬伤，白水调敷患处。
制备方法：上除麝香、牛黄、梅片、雄黄另研外，余则共研细面，再用白薇、桑皮各1两5钱，明矾2两，煎浓汁和白酒为丸，如菜子大。
临床应用：None
各家论述：None
用药禁忌：忌食油腻生冷之物，孕妇忌服。
附注：None
