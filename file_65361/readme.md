序号：65377
名称：子午丸
组成：榧子（去壳）2两，莲肉（去心）1两，枸杞子1两，白龙骨1两，川巴戟（去心）1两，破故纸（炒）1两，真琥珀（另研）1两，苦楮实（去壳）1两，白矾（枯）1两，赤茯苓（去皮）1两，白茯苓（去皮）1两，莲花须（盐蒸）1两，芡实1两，白牡蛎（煅）1两，文蛤1两。
出处：《得效》卷七。
主治：心肾俱虚，梦寐惊悸，体常自汗，烦闷短气，悲忧不乐，消渴引饮，漩下赤白，停凝浊甚，四体无力，眼昏，形容瘦悴，耳鸣头晕，恶风怯冷。滑精。
加减：None
功效：None
用法用量：浓煎草解汤空心送下。
制备方法：上为末，酒蒸肉苁蓉1斤2两，烂研为丸，如梧桐子大，朱砂1两半重，细研为衣。
临床应用：None
各家论述：None
用药禁忌：忌劳力房事。
附注：None
