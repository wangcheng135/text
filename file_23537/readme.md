序号：23545
名称：防风汤
组成：防风3分，赤石脂3分，石膏3分，人参3分，生姜3分，白石脂3分，寒水石3分，龙骨3分，茯苓3分，桂心2分，紫石1分。
出处：《千金》卷十四引《徐嗣伯方》。
主治：风眩，服薯蓣汤后，四体尚不凉冷头目眩动者。
加减：None
功效：None
用法用量：以水8升，煮取3升，分3服。
制备方法：None
临床应用：None
各家论述：《千金方行义》：服薯蓣汤，四肢热尚未除，头目转觉眩晕，虚阳不能即敛放取风引汤中四石、龙骨、桂心，加参、苓以镇固之，生姜开发中气，防风开发表气，风引汤专取大黄引风热下行，此以人参助防风上泄，仍取江水煮取，乃薯蓣汤之成法，半夏汤之变法也。
用药禁忌：None
附注：此汤大都宜长将服，但药中小小消息之，随冷暖耳。
