序号：35743
名称：卷柏丸
组成：卷柏1两，钟乳粉1两，鹿角胶（捣碎，炒令黄燥）1两，紫石英（研细，水飞过）1两，阳起石（研细，水飞过）1两，桑螵蛸（微炒）1两，熟干地黄1两，禹余粮（烧醋淬7遍）1两，杜仲（去粗皮，炙微黄，锉）3分，芎3分，当归（锉，微炒）3分，桂心3分，桑寄生3分，牛膝（去苗）3分，五味子3分，蛇床仁3分，牡丹3分。
出处：《圣惠》卷七十七。
主治：妊娠数堕胎，因气血虚损，子脏风冷，致令胎不坚固，频有所伤者。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，都研令匀。炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
