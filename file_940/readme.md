序号：940
名称：六神散
组成：蝼蛄1枚（去肚肠），蜜蜂7枚（2味用泥半指厚裹遍，以熟炭1斤，湿煅候泥干通红，去火放冷，取出，于乳缽内细研，不罗），商陆根汁半盏，生姜汁3分，生蜜3分，绿豆末1钱匕。
出处：《圣济总录》卷十八。
主治：癞病。
加减：None
功效：None
用法用量：上药一处和匀，更用温酒调下，只作1服。约半日许发动，恶心涕唾汗出，仍吐出涎，渴时频与温米汤吃。若是吐多泻多，或更有血，勿疑，是病出也。如吐泻后腹痛，取下1块大小如鸡子，有诸般颜色者，此是病根也。如不思食，以甘草2寸，黄连2条，水1盏，煎至半盏，温吃其药。即不以老小强弱，都作1服。若是半日以来未发动，即更服樟柳根等汁投之，尚未觉，更与吃无妨。面上有如紫疼痛未消者，用干斑蝥未，以生油调成膏，敷患上约半日许，(疒咅)(疒畾)胀起，以软帛子揩拭去药，以棘针挑破近下、令水出自干，即不得剥其疮皮。斑蝥不得另入口眼。若面上有小尖(疒咅)(疒畾)子者，不用斑蝥，却以熟艾入胆矾少许，纸杵作钗股大小艾炷，灸之，每个灸1炷。如手脚上下有疮未愈，煎甘草热汤淋洗，更以白药子3两，甘草1分，荆芥10穗，一处捣罗为末，干贴。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：切忌风及房室事、动风发气、盐、醋料物等物。只得吃淡粥过100日。
附注：None
