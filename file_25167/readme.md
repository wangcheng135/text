序号：25175
名称：观音救急丹
组成：真朱砂6两，雄精6两，荜茇2钱，大梅片2钱5分，真佛金200张，当门子2钱5分，明矾1两，月石2两，牙消4两（后下）。
出处：《经验各种秘方辑要》。
主治：急痧，阴阳反错，寒热交争，四时不正之气，郁闷成瘀，绞肠腹痛，吐泻不止；小儿惊风闭急；伤暑泄泻。
加减：None
功效：None
用法用量：遇有急痧等症，先点两眼角，再取半分入脐内，膏药贴之；如遇重症，再将余丹放舌上，阴阳水送服，小儿减半。
制备方法：上为末，用瓷瓶每装1分，黄蜡封口。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：感应救急丹（《全国中药成药处方集》上海方）。《全国中药成药处方集》（上海方）无真佛金，诸药各取净粉，共研极细粉。每次服二分，温开水化服。
