序号：60053
名称：起枕汤
组成：当归3钱，赤芍1钱半，官桂1钱半，蒲黄1钱半，五灵脂3钱，白芷1钱半，丹皮1钱半，炙草1钱半。
出处：《医略六书》卷三十。
主治：产后冲任受风，波及阳明，而瘀血凝结，故恶露不下，脐腹作痛，脉浮滞涩者。
加减：None
功效：None
用法用量：水煎，去滓，入童便1杯，温服。
制备方法：None
临床应用：None
各家论述：当归养血活血脉，赤芍破血泻血滞，生蒲黄破瘀通经脉，五灵脂破瘀降浊阴，官桂温经通闭，白芷升阳散风，丹皮凉血散瘀，炙草缓中益胃也。水煎入童便温服，使风邪外解，则瘀结内消，而恶露无不下，何脐腹作痛之不止哉。
用药禁忌：None
附注：None
