序号：26779
名称：异功散
组成：人参（切去顶）、茯苓（去皮）、白术、陈皮（锉）、甘草各等分。
出处：《小儿药证直诀》卷下。
主治：脾虚气滞。饮食减少，胸脘痞闷，食入作胀，大便溏薄，神疲气短，身体羸瘦，或面部浮肿者。
加减：None
功效：益气补中，理气健脾。
用法用量：每服2钱，水1盏，加生姜5片，大枣两个，同煎至7分，食前温服，量多少与之。
制备方法：上为细末。
临床应用：1.咳嗽：一产妇咳而胸满不食，涕唾，面肿气逆，此病在胃而关于肺，用异功散而愈。_x000D_
2.喘：一小儿外感风邪，服表散之剂，汗出作喘，此邪气去而脾肺虚也。用异功散而汗喘止，再剂而乳食进。_x000D_
3.泄泻：一小儿患泻，乳食不化，手足指冷，服消乳丸，食乳即泻，余用五味异功散加木香，母子服之而愈。_x000D_
4.发热：一小儿发热，饮食少思，大便不实，常服芦荟等丸，视其鼻赤，此寒冷之剂复伤脾土而虚热也，用五味异功散，数剂而愈。
各家论述：《医略六书》：人参扶元气以补肺，白术燥湿气以健脾，茯苓渗湿清治节，橘红利气化痰涎，炙甘草以益胃气，姜汤煎服，使脾气鼓运，则痰涎自化而肺络清和。
用药禁忌：None
附注：五味异功散（《疠疡机要》卷下）。
