序号：83255
名称：乌鸡丸
组成：人参5钱，黄耆5钱，柴胡5钱，前胡5钱，黄连5钱，黄柏5钱，当归5钱，白茯苓5钱，熟地黄5钱，生地黄5钱，白芍药5钱，贝母5钱，五味子5钱，知母5钱，川芎5钱，白术5钱。
出处：《杏苑》卷五。
主治：虚劳客热，肌肉消瘦，四肢倦怠，五心烦热，咽干颊赤，心怯潮热，盗汗减食，咳嗽脓血。
加减：None
功效：None
用法用量：每服100丸，空心、食前米汤或沸汤送下。
制备方法：上(口父)咀；用乌骨雄鸡重2斤以上者1只，须新生肥壮者；去毛、血，洗净，入前药在肚，以线缝定，用好腊酒入锅中，放鸡在内，酒约过鸡背上1寸为度，肠脏放在鸡外，同煮极烂，拆开，同药晒干，研为细末，用原汁打蒸饼糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
