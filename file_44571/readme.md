序号：44586
名称：威灵仙汤
组成：威灵仙（酒浸，去芦头）3分，仙灵脾（锉碎，羊脂拌炒过）3分，防风（去叉）3分，人参3分，白茯苓（去黑皮）3分，羌活（去芦头）3分，独活（去芦头）3分，附子（炮裂，去皮脐）1两半，柴胡（去苗）1两半，秦艽（童便浸1宿，洗，焙）3分，槟榔3分，木香3分，鳖甲（童便浸1宿，炙黄，去裙襕）1两，黄耆1两，枳壳（去瓤，麸炒）3分，甘草（炙）3分，沉香3分，桂（去粗皮）3分，芎3分，苁蓉（酒浸，切，焙）1两，巴戟天（酒浸1宿，去心）1两，牛膝（酒浸1宿，切，焙）3分，半夏（生姜汁浸1宿，焙）3分，当归（酒浸1宿，瓦上炒）3分，乌药（生，米泔浸1宿，锉，焙）3分。
出处：《圣济总录》卷一八五。
主治：丈夫元脏风虚，气血劳伤，饮食减少，四肢疲乏，气劣心悸，上热下冷，口苦舌干。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，生姜5片，羊肾半枚，同煎减半，入酒半合，更煎2-3沸，去滓，适口服，空心、午前各1服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
