序号：23432
名称：杏胶饮
组成：黄明胶2两（锉，炙），马兜铃1两，甘草（炙）1两，半夏（姜汁浸3日）1两，杏仁（去皮尖）1两，人参半两。
出处：方出《续本事》卷五，名见《东医宝鉴·杂病篇》卷五。
主治：十六般哮嗽。
加减：None
功效：None
用法用量：每服1大钱，水1盏，随病有汤使，煎至7分，临睡、食后服。汤使于后。心嗽，面赤或汗流，加干葛同煎；肝嗽，眼中泪出，加乌梅1个，糯米3-4粒同煎；脾嗽，不思饮食，或1-2时恶心，加生姜3片同煎；胃嗽，吐逆酸水，加蚌粉同煎；胆嗽，令人临睡用药半钱，茶清调下；肺嗽，上喘气急，加桑白皮同煎；膈嗽，咳出痰如圆块，生姜自然汁调药咽下；劳嗽，入秦艽末同煎；冷嗽，天晓嗽甚，加葱白3寸同煎；血嗽，连顿不住，加当归末、枣子同煎；暴嗽，涕唾稠，加乌梅、生姜同煎；产嗽，背甲疼痛，加甘草3寸同煎；气嗽，肚痛胀满，加青皮（去白）同煎；热嗽夜甚，加蜜1匕，葱白同煎；哮嗽，声如拽锯，加半夏2个同煎；肾嗽，时复2-3声，加黄耆、白饴糖煎。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
