序号：74308
名称：催生柞木饮
组成：大柞木枝1大握（长1尺，洗净），大甘草5寸（并寸折）。
出处：《本草纲目》卷三十六引《产宝》。
主治：妇人难产。妇人横产倒生，死胎在腹，胀烂不出。
加减：None
功效：None
用法用量：以新汲水3升半，同入新沙瓶内，以纸三重紧封，文武火煎至1升半。待腰腹重痛，欲坐草时，温饮1小盏，便觉心下开豁；如渴，又饮1盏，至3-4盏，觉下重便生。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：切不可坐草太早，及坐婆乱为也。
附注：催生柞木叶饮（《圣济总录》卷一五九）、催生柞木饮子（《妇人良方》卷十七）、柞木饮（《回春》卷六）、催生神柞散（《寿世保元》卷七）、柞木饮子（《济阴纲目》卷十）、催生神柞饮（《大生要旨》卷三）。方中柞木枝，《圣济总录》用柞术叶一把（并细枝，锉）。
