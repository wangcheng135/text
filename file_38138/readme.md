序号：38148
名称：松实丸
组成：松实（去皮）、白茯芩（去黑皮）、麦门冬（去心，焙）、柏子仁（微炒，别研）、甘草（炙，锉）、山芋、枸杞子、肉苁蓉（酒浸，去皴皮，炙干）、五味子（去茎叶）、桂（去粗皮）、熟干地黄（焙）、陈橘皮（汤浸，去白，焙）、干姜（炮）、泽泻、远志（去心）、石斛（去根黑者）、女贞实、络石、杜仲（去粗皮，涂酥炙）各等分。
出处：《圣济总录》卷九十一。
主治：虚劳脱营，血气伤惫，羸瘦少气，畏恐多惊。
加减：None
功效：久服强筋骨、长肌肉，令人肥盛，光泽颜色，除解百病，安精神，少梦寐，强气血，倍力留年，益气长神。
用法用量：每服10丸，食前温酒送下，食后再服。不知，稍增之，可至20丸。
制备方法：上为细末，练蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
