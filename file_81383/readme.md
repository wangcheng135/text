序号：81399
名称：麒麟竭膏
组成：当归5钱，木鳖子5钱，知母5钱，五倍子5钱，细辛5钱，白芷5钱，槐枝14条，柳枝（长寸许者）14条，血竭3钱，乳香5钱，没药5钱，轻粉2钱，雄黄4钱，麝香2钱，松香10两（捣末），沥青2两（捣末）。
出处：《万氏家抄方》卷四。
主治：痈疽毒疮，疔肿结核，臁疮，臀股黄湿痒痛，打扑损伤，挫肭气闪，头疼，赤眼，暴伤风冷嗽，牙疼，面肿，风寒湿痒，臂腿痛。
加减：None
功效：None
用法用量：一切痈疽毒疮生者贴之即散，熟者即穿败生肌；一切疔肿结核，并贴患处；臁疮，先用齑汁、白矾入汤，用鹅翎洗净，以牛蒡子叶或金刚藤叶先贴疮半日，取净恶水，然后贴此膏；一切臀股黄湿痒痛等疮，并洗净捐干，贴患处；一切打扑损伤，挫肭气闪，俱贴患处；头疼，贴两太阳；赤眼，贴眼胞鱼尾际；暴伤风冷嗽，贴脊心；牙疼，刮药塞齿缝，面肿者再贴面上；一切风寒湿痒，臂腿痛，贴痛处。
制备方法：用麻油3两半同前粗药入锅，以文武火煎，用槐枝不住手搅，令焦色，滤去滓；再将油入锅，先下松香，沥青不住手搅，如沸溢取下火搅之，再上火熬，滴水成珠为度，下火；将血竭等六味细药下内，速搅令匀，倾于水盆中，半日后以手抟之，渐渐软和，反复揉扯如金丝状，再入水浸，如前揉扯，春、夏频换水，若急用亦浸1-2宿。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
