序号：41949
名称：青龙丸
组成：当归（去芦头，洗去尘土，微炙，切，焙干）4两，黄芩3两（生用），木贼（去节）3两，木鳖子3两，琥珀半两（研），麻黄（去节）1两，枸杞子2两，防风（去芦头）2两，荆芥穗1两半，甘草1两（生用，锉），橘皮（去瓤）1两半，乌鱼骨1两半，龙脑薄荷（阴干者，只用叶）2两。
出处：《普济方》卷七十五引《余居士选奇方》。
主治：风毒热气，上攻眼目，赤痛翳膜，冷热虚实，一切眼疾。
加减：None
功效：None
用法用量：每服1丸，食后、夜卧细嚼，腊茶送下，1日3次。
制备方法：上为末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：忌食鱼、面。
附注：None
