序号：68069
名称：黄耆汤
组成：黄耆（细锉）1两，柴胡（去苗）1两，鳖甲（去裙襴，醋炙）1两，肉豆蔻（炮，去壳）1两，白芷1两，秦艽（去苗土）1两，桂（去粗皮）1两，桔梗（炒）1两，麦门冬（去心，焙）半两，当归（切，焙）半两，白茯苓（去黑皮）半两，人参半两，枳壳（去瓤，麸炒）半两，甘草（炙，锉）半两，熟干地黄（焙）半两，海桐皮（锉）半两，芍药半两，木香半两，酸枣仁（炒）半两，沉香（锉）半两，荆芥穗半两，槟榔（锉）半两。
出处：《圣济总录》卷八十八。
主治：虚劳寒热，周身疼痛，咳嗽痰壅。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，加生姜3片，同煎至7分，去滓温服，空心、日午、近夜各1次。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
