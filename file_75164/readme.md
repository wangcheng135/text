序号：75180
名称：蒸脐方
组成：荞麦（以水和为一圈，径寸余，脐大者，经2寸）2钱，乳香2钱，没药2钱，虾鼠粪（即一头尖）2钱，青盐2钱，两头尖2钱，川续断2钱，麝香1分。
出处：《扶寿精方》。
主治：上部火或腹心宿疾，妇人月信不调，赤白带下，男子遗精白浊，或风热郁于腠理。
加减：None
功效：却疾延年。
用法用量：上各为末，入荞麦圈内，置脐上，上覆槐皮（去粗，半分厚），加豆大艾炷，灸至腹内微作声为度，不可令内痛，痛则反损真气，槐皮觉焦即更新者，每年中秋日蒸一次。若患风气有郁热在腠理者，加女子月信拌药则易汗，汗出而疾随愈。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
