序号：31588
名称：杞实粥
组成：芡实7钱（选净硬皮，滚水掏泡4-5次，又极滚水泡透听用），枸杞子3钱（选肥大赤色者，只用水掏1次，滚水泡透听用），粳米（晚熟者）大半茶钟（滚水淘洗4-5次听用）。
出处：《眼科秘诀》卷二。
主治：None
加减：None
功效：聪耳明目，延年益寿。
用法用量：上3味，今日如法制完，明日五更用砂锅1口，先将水烧滚，下芡实煮4-5沸；次下枸杞子煮3-4沸；又下大米，共煮至浓烂香甜。煮粥的水多加，勿添冷水。空腹食之，以养胃气。40日皮肤润泽，100日步履壮健，1年筋骨牢固。或为细末，滚水服亦可。
