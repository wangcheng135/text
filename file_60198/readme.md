序号：60213
名称：逐血补心汤
组成：红花2钱，赤芍药2钱，生地黄2钱，桔梗2钱，苏叶2钱，前胡2钱，茯苓2钱，防风2钱，牛胆南星2钱，黄连2钱，粉葛2钱，当归3钱，薄荷1钱5分，人参1钱5分，升麻1钱5分，半夏2钱5分，甘草1钱。
出处：《准绳·女科》卷五。
主治：产后失音不语。产后心肺二窍为血所侵，又感伤风。
加减：None
功效：None
用法用量：每服水1钟半，生姜3片，煎至7分，空心服，滓再煎服。
制备方法：上为散，分作2服。
临床应用：None
各家论述：《医略六书》：产后中风，痰热内滞，故心气闭塞，而心窍闭塞，令人不语焉。生地壮水以交心，黄连清热以燥湿，赤芍泻血滞，当归养心血，防风疏风邪之外袭，人参助心气之内虚，胆星豁痰以益肝胆，菖蒲开窍以通神明，生姜散痰涎，甘草和胃气也。水煎温服，使风邪外解，则痰热内消，而心窍自通，神机鼓舞，何不语之有哉。
用药禁忌：None
附注：None
