序号：11698
名称：牛黄丸
组成：牛黄3分（细研如粉），铁精3分（细研如粉），金箔50片，银箔50片（细研如粉），石膏3分，龙齿3分（细研如粉），地骨皮3分，茯神1两，川升麻3分，玄参3分，人参1两（去芦头），麦门冬1两（去心，焙），枳实半两（麸炒微赤），葳蕤3分，赤芍药3分，生干地黄3分，甘草半两（炙微赤，锉），黄芩3分，朱砂3分（细研如粉），虎睛1对（酒浸一宿，微炙）。
出处：《圣惠》卷四。
主治：心脏风邪，狂乱失志，不得安定。中风，心多恐怖。
加减：None
功效：None
用法用量：每服10丸，煎地骨皮汤送下，不拘时候。
制备方法：上为末，都研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
