序号：40994
名称：三清化毒黄蜡丸
组成：朱砂5两（用红亮者），当归1两，生地1两，人参5钱，犀角（镑）5钱，黄连4钱，白术8钱，牛蒡子7钱，连翘7钱，葛根4钱，升麻2钱，荆芥穗4钱，黄柏4钱，牛黄1钱5分，甘草4钱。
出处：《救产全书》。
主治：None
加减：None
功效：稀痘补气，和血解毒，益元清心。
用法用量：凡痘初出，即细研1丸，用薄荷汤调服，痘即减少轻快。
制备方法：上(口父)咀，将朱砂打作绿豆大块，用绢袋装缝，酒2碗，水10碗，同药15味入砂锅内，文武火煮之，以净剩1碗汁为度，滤清，拌朱砂晒干，以猪心血调糊为丸，干重5分，外以黄蜡护之。
临床应用：None
各家论述：None
用药禁忌：None
附注：化毒黄腊丸（《惠直堂方》卷四）。
