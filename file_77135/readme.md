序号：77151
名称：黑虎膏
组成：槐条70茎（每长7寸半），柳条70茎（每长7寸半），巴豆80枚（去皮），当归2钱，木鳖子仁5枚，白芷3钱，自然铜少许（为末），小油1斤1两，黄丹8两。
出处：《普济方》卷三一五。
主治：肠痈，乳痈，骨疽，眼目赤疼痛肿；妇人胎衣不下，瘀血冲心，月候不通。
加减：None
功效：None
用法用量：治肠痈、乳痈、骨疽者，每服15丸，如梧桐子大，甘草汤或漏芦汤送下，外贴患处；眼目赤疼痛肿者，以茶清或山栀子煎汤送下，仍贴两太阳穴；妇人胎衣不下，瘀血冲心，童便送下；月候不通，红花汤送下。
制备方法：先将小油锅内煎沸，下前药煎黄色，滤去滓，入丹熬成膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
