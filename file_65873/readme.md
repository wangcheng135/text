序号：65889
名称：萆薢分清散
组成：益智仁、川萆薢、石菖蒲、乌药各等分。
出处：《杨氏家藏方》卷九。
主治：真元不足，下焦虚寒，小便白浊，频数无度，漩面如油，光彩不定，漩脚澄下，漩如膏糊，或小便频数，虽不白浊。
加减：None
功效：None
用法用量：每服3钱，水1盏半，入盐1捻，同煎至7分，食前温服。
制备方法：上为细末。
临床应用：慢性前列腺炎：李某，男，32岁，结婚五年无子，会阴部胀痛，小便频数短赤，尿后泌出米汤样粘液，滴沥难尽，遗精、早泄、阳萎，经检查诊断为慢性前列腺炎，拟方：萆薢15g，茯苓12g，车前子12g，黄柏10g，山栀子10g，益智仁、乌药、芡实各12g，桃仁8g，当归尾10g，甘草梢5g，共服药30余剂，病告痊愈。次年喜得一子。
各家论述：None
用药禁忌：None
附注：分清散（《济生》卷四）、分清饮（《瑞竹堂方》卷一）、萆薢分清饮（《丹溪心法》卷三）、萆薢饮（《古今医鉴》卷八）、萆薢散（《寿世保元》卷五）。本方改为丸剂，名“萆薢分清丸”（见《北京市中药成方选集》）。
