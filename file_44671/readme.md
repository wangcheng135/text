序号：44686
名称：宣积丸
组成：京三棱（炮）5两，、，雷丸（炮）5两，黑牵牛（头末，生用）10两，大腹子（取末，生用）7两，莪术2两，广木香1两。
出处：《医方类聚》卷&mdash;一三引《经验秘方》。
主治：虫积。
加减：None
功效：None
用法用量：每服4钱，五更时用砂糖水，温冷不妨，天明去4-5行，看取出是何虫积，温白粥补之。
制备方法：上为细末，和匀。用好紫色皂角半斤，去皮弦，截碎，水1大碗半，浸1宿，冬月两宿，捞去皮滓，铜瓷器内熬数沸，白沫出为度。放冷，和匀，药须揉到，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：孕妇休服；忌生冷、腥硬物。
附注：None
