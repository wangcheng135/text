序号：4413
名称：木香丸
组成：鸡心2两，槟榔2两，陈橘皮（去白）2两，青木香5钱，人参5钱，厚朴5钱，官桂（去无味者）5钱，大附子5钱，羌活5钱，京三棱5钱，独活5钱，干姜（炮）5钱，甘草（炙）5钱，芎5钱，川大黄（切，微炒）5钱，芍药5钱，牵牛子1斤（淘去浮者，揩拭干，热捣取末4两，余滓不用），肉豆蔻6枚（去壳，止泻方用）。
出处：《苏沈良方》卷三。
主治：风劳冷气，脐下刺痛，口吐清水白沫，醋心，痃癖气块，心腹胀满；男子肾脏风毒，攻刺四体，阳毒脚气，目昏头痛，心间呕逆，两胁坚满不消；妇人血痢，下血刺痛，积年血块，胃口逆满，手足心烦热，不思饮食；小儿疳气，腹胀气喘；胸腹饱闷，泄泻不止；误食毒物，痈疽发背，山岚瘴疟，才觉头痛，背膊拘紧。阳毒伤寒，忽浑身壮热，四肢疼痛不可忍，口内狂言。
加减：None
功效：None
用法用量：心腹胀满，一切风劳冷气，脐下刺痛，口吐清水白沫，醋心，痃癖气块，男子肾脏风毒，攻刺四体，及阳毒脚气，目昏头痛，心间呕逆，及两胁坚满不消，卧时橘皮汤送下30丸，以利为度，此后每夜20丸；女人血痢，下血刺痛，积年血块，胃口逆，手足心烦热，不思饮食，姜汤送下30丸，取利，每夜更服20丸；小儿5岁以上，疳气腹胀气喘，空心温汤送下5-7丸，小者减丸数服；凡胸腹饱闷不消，脾泄不止，临卧温酒送下，取利。
制备方法：上为末，瓷器盛之，密封，临服用牵牛末2两，药末1两，同研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：木香槟榔丸(《圣济总录》卷三十七)。《幼幼新书》引《灵苑方》：阳毒伤寒，经三日，临卧温水下三十丸，未转加数。
