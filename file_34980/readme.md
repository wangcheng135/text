序号：34990
名称：麦汤散
组成：地骨皮（炒）半分，甘草（炙）半分，滑石半分，麻黄（去节）1分，人参1分，知母1分，羌活1分，大黄（湿纸裹，煨令熟）1分，甜葶苈（隔纸炒）1分。
出处：《幼幼新书》卷十五引《家宝》。
主治：小儿伤寒，咳嗽温壮，水痘。小儿水痘，遍身作痛，壮热烦躁，作渴饮冷，大便秘结，小便涩滞，喘嗽。
加减：None
功效：发表散邪，疏通内热。
用法用量：婴儿每服1字或半钱，3-5岁1钱，水半盏，加小麦7粒或14粒煎数沸服。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：参汤散（《小儿痘疹方论》）、麦煎散（《医方大成》卷十）、地骨皮散（《普济方》卷四○三）、参黄散（《外科启玄》卷十二）、麦汤饮（《痘疹仁端录》卷十一）。
