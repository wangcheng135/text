序号：37572
名称：抵圣散
组成：附子2枚重半两者（炮裂，去皮脐），槟榔2枚（一生一熟），大黄（锉）1分，肉豆蔻（去壳）1分，木香1分，当归（锉，焙）1分，吴茱萸（洗，焙）1分，黄连（去须）1分，芎1分，陈橘皮（汤浸去白，焙）1分，干姜（炮）1分，桂（去粗皮）1分，芜荑1分，猪牙皂荚（酥炙，去黑皮并子）1分，巴豆半两（去皮，以浆水煮20-30沸，麸炒黄，研出油），苍鼠（锉，焙干，研末）1两。
出处：《圣济总录》卷一四○。
主治：None
加减：None
功效：出箭头。
用法用量：每服1字匕，没药酒调下。只3服，箭头立出。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
