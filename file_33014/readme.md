序号：33023
名称：连茹绛复汤
组成：小川连4分（醋炒），真新绛1钱半，玫瑰瓣3朵（拌炒丝瓜络3钱），淡竹茹3钱，旋复花3钱（包煎），青葱管3寸，广郁金汁4匙（冲）。
出处：《重订通俗伤寒论》。
主治：肝病日久入络，血郁不舒，郁而化火，筋脉拘挛，胸胁串疼，脉弦而涩者。
加减：火盛痛甚者，加蜜炙延胡1钱半，醋炒川楝子1钱半，酸苦泄肝，以清火而止疼；瘀结痛剧者，加光桃仁20粒，杜红花8分，紫金片3分（开水烊冲）；肠燥便秘者。加元明粉3钱，净白蜜1两，煎汤代水，甘咸润燥以通便；血枯液结者，加鲜生地6钱，归身2钱，原麦冬3钱，南沙参3钱，甘润增液以滋血。
功效：清通肝络，行血止疼。
用法用量：None
制备方法：None
临床应用：None
各家论述：方中连、茹、绛、复清肝通络为君；臣以玫瓣拌炒瓜络，辛香酸泄以活络；佐以郁金活血疏郁；使以葱管宣气通络。
用药禁忌：None
附注：None
