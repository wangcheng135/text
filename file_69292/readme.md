序号：69308
名称：黄雪
组成：川朴消5斤，川大黄2两，黄芩3两，山栀2两，犀角屑1两，紫石英2两（细研），甘草3两（生用），竹茹3两，麝香半两（细研），朱砂1两（细研），羚羊角屑3两，郁金2两。
出处：《圣惠》卷九十五。
主治：风热，天行瘴毒。
加减：None
功效：压丹石，安心神，止狂热。
用法用量：每服1-2钱，以甘草汤调下。
制备方法：除朴消、紫石英、朱砂、麝香外，上锉细，以水1斗2升，煎至5升，去滓澄清，以文火更煎之，下朴消，以柳木搅，勿住手，候稍稠，即下紫石英、朱砂末搅令匀，候欲凝结，后下麝香末搅令匀，倾于新盆中，经宿取出，为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
