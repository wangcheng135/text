序号：78070
名称：裴公八毒膏
组成：蜀椒2两，当归2两，雄黄2两，丹砂2两，乌头1升，巴豆1升，薤白1斤，莽草4两。
出处：《千金》卷七。
主治：卒中风毒，腹中绞刺痛，飞尸入脏；及魇寐不寤，尸厥奄忽不知人；宿食不消；中蜈蚣、蛇、蜂等毒者；若岁中多温，欲省病及行雾露中者。
加减：None
功效：None
用法用量：温酒服如枣核大，得下止；若毒气甚，咽喉闭塞不能咽者，折齿纳葱叶口中，以膏灌葱叶中令下；病肿者，向火摩肿上；若岁中多温，欲省病及行雾露中，酒服之，纳鼻中亦得；诸蜈蚣、蛇、蜂等毒者，以膏置疮上，病在外，悉敷之摩之。
制备方法：上(口父)咀，苦酒3升渍1宿，用猪脂5斤，东向灶苇薪火煎之五上五下，候薤白黄色，绞去滓，研雄黄、丹砂如粉，纳之，搅至凝乃止，膏成盛不津器中。
临床应用：None
各家论述：《千金方衍义》：熬膏都用麻油，此则并用猪脂者，取异类有情，易通血气之病也。
用药禁忌：None
附注：八毒膏（《普济方》卷九十一）。
