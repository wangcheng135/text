序号：2790
名称：天灵盖丸
组成：天灵盖（如无，即髑髅骨有十字缝者，取中心1片，以茅香汤洗，以安息酒涂炙令黄色），紫河车1枚（头生儿男儿胞衣是也，于石上用新草鞋踏洗令净，控干，以麝香1两，渗拌令匀，入瓶以泥固济，如法火煅通赤留性，勿令煅过），石蜥蜴1条（和肠脏醋炙49遍令干），獭肝1具（薄切，瓦上焙干），赤足蜈蚣1条（酒浸，炙焦），黄鹰粪1钱，金线虾蟆（端午日收者，以醋炙令黄，只用头），蛤蚧1对（酥炙令黄，只取头尾用）。
出处：《圣济总录》卷九十三。
主治：传尸。
加减：None
功效：None
用法用量：每服40丸，温煎生姜汤送下。初更1服，二更初再1服。
制备方法：上为细末，以醋煮猪胆令热成膏，和搜为丸，如梧桐子大，即以槐花1钱，芫青1钱，研为末，熟绢筛过，为药衣，令青黄色。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
