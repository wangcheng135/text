序号：64684
名称：硇砂丸
组成：硇砂1两，肉豆蔻1两（去壳），木香1两，槟榔1两，雄（硫）黄1两（细研，水飞过），蛜（虫祁？？）1两（微炒），附子1两（炮裂，去皮脐），天麻1两，蓬莪术1两，青橘皮1两（汤浸，去白瓤，焙），茴香子1两半，桃仁半两（汤浸，去皮尖双仁，新炒微黄）。
出处：《圣惠》卷七。
主治：肾脏风冷气，腹胁疼痛，四肢无力。
加减：None
功效：None
用法用量：每服30丸食前用温酒送下。
制备方法：上为末用无灰酒3升，调药末，于银锅内，以慢火熬，看硬软得所，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：《普济方》有白附子、肉桂各一两，丁香、阿魏各半两。
