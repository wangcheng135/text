序号：34382
名称：万金膏
组成：黄连1分，黄柏1分，黄芩1分，白及1分，白蔹1分，龙骨1分，当归1分（洗，焙），厚朴1分（去粗皮），川芎1分，没药1分（研），槐枝1分，柳枝1分，猪牙皂角1分，鳖甲1分，苦参1分，香白芷1分，木鳖子仁1分，草乌头1分，乌贼鱼骨1分（同锉碎），乳香1钱（研），黄丹1两半，清麻油4两（冬月用半斤）。
出处：《杨氏家藏方》卷十二。
主治：痈疽发背，及诸般恶疮。从高坠堕，打扑伤损，脚膝生疮，远年臁疮，五般痔漏。
加减：None
功效：None
用法用量：如寻常膏药，摊在纸花上，看疮大小敷贴。
制备方法：上除黄丹外，银、石器中将诸药油内慢火煎得油色紫赤，滤去药不用，入一半黄丹在油内，不住手搅，煎微黑，更入一半黄丹，不住手搅，只是用慢火熬得紫黑色，滴在水上不散，捻不粘手，然后更入黄丹少许，再熬，如捻着硬时，却更入油少许，但不粘手即止。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
