序号：62663
名称：清化会厌退腐汤
组成：香银花5钱，连翘3钱，人中黄1钱5分，元参3钱，赤芍2钱，生地4钱，丹皮3钱，麦冬2钱，桃仁3钱，红花2钱，薄荷1钱，大贝母3钱，板蓝根3钱，芦根2两（去节）。
出处：《疫喉浅论》卷上。
主治：疫喉白腐，会厌腐溃。
加减：谵语神昏，加犀角：壮热烦渴，可与竹茹石膏汤相间服之；胸次饱满，加枳壳山楂、神曲、麦芽以消之；小便不通，加泽泻、车前子、灯心、莲子心以导之；大便秘结，加清宁丸、元明粉以行之，重则加大黄。
功效：清化会庆之腐。
用法用量：上以长流水煎，日2服，夜1服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
