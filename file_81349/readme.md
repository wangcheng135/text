序号：81365
名称：鳖甲汤
组成：鳖甲（去裙襕，醋炙）1两半，白前1两半，代赭（煅，醋淬）1两半，京三棱（炮，锉）1两半，附子（炮裂，去皮脐）1两半，延胡索1两半，大黄（锉，炒）1两，甘草（炙，锉）1两，木香1两，桂（去粗皮）1两，当归（切，焙）1两，桃仁（去皮尖双仁，炒）20个，熟干地黄（焙）3两，红蓝花3分，大腹皮（锉）2两半。
出处：《圣济总录》卷一五一。
主治：妇人经候不通，已经2-3月者。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，煎取8分，去滓温服，不拘时候。
制备方法：上(口父)咀，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
