序号：50577
名称：茸附养气丹
组成：代赭石2两，紫石英2两，禹余粮石2两，赤石脂2两（上4味，用醋淬煅7-8次，以酥为度，共碾为末，入瓦瓶内，盐泥固济，炭火煅养1日1夜后，移埋别地，出火毒3昼夜，入后药），天雄1个（炮，去皮脐），附子1个（炮，去皮脐），干姜（炮）1两，鹿茸（爁去血毛，盐酒炙）1两，当归（去芦，酒浸）1两，沉香1两，木香1两，丁香1两，肉桂（去粗皮）1两，阳起石1两，肉苁蓉1两，磁石1两，石燕1两，胡芦巴（炮炒）1两，破故纸（炒）1两，肉豆蔻（煨）1两，舶上茴香（炒）1两，胡椒1两，荜澄茄（去蒂）1两，牡蛎（煅）1两，乳香（别研）1两，安息香（别研）1两，血竭（别研）1两，朱砂（别研）1两，没药（别研）1两，钟乳粉1两。
出处：《普济方》卷二二六引《澹寮》。
主治：诸虚不足。
加减：None
功效：壮真气，敛元阳。
用法用量：每服5粒、7粒或9粒，早晨枣汤吞下，或参汤、盐汤送下。
制备方法：上共为细末，糯米糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
