序号：5994
名称：加减四物汤
组成：大熟地1两（九蒸），白芍3钱（酒炒），当归5钱（酒洗），川芎2钱（酒洗），白术5钱（土炒），黑芥穗3钱，山萸3钱（蒸），续断1钱，甘草1钱。
出处：《傅青主女科》卷上。
主治：妇女血虚，经水过多，行后复行，面色萎黄，身体倦怠，而困乏愈甚者。
加减：None
功效：补血归经。
用法用量：水煎服。4剂而血归经。10剂之后，加人参3钱，再服10剂，下月行经，适可而止矣。
制备方法：None
临床应用：None
各家论述：方中四物汤乃补血之神品，加白术、荆芥，补中有利；加山萸、续断，止中有行；加甘草以调和诸品，使之各得其宜。所以血足而归经，归经而血自静矣。
用药禁忌：None
附注：None
