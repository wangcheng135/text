序号：4911
名称：木香锉散
组成：木香1分，鳖甲（用7肋者，炙令黄色）1两，银州柴胡1两（去芦头），秦艽3分（去芦头），黄耆1两，知母3分，茯苓3分，人参1两，桔梗3两，白术1两，甘草1两（炙），防风3分，肉豆蔻1分，半夏半两（用生姜3两，取汁，煮令汁干），枳壳3分（汤浸，去瓤，炒），芍药3分。
出处：《传家秘宝》。
主治：虚劳气弱，寒热往来，不思饮食，口舌生疮，四肢劳倦，五心烦躁，肌肤不泽。
加减：None
功效：None
用法用量：每服半两，加生姜1分、大枣3个，水1升，煎7分。以生绢子滤，分2次温服，晨暮各服尽，再将两服渣依前煎作1服。
制备方法：上药并洗，锉细，如大豆大或豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
