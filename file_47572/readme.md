序号：47587
名称：神仙化癖膏
组成：真香油2斤4两，秦艽5钱，三棱5钱，黄丹1斤2两（水飞过，炒紫色），黄柏5钱，穿山甲14片，当归3钱，莪术5钱，全蝎14个，大黄3钱，蜈蚣5钱，木鳖子7个，将药入油内熬黄色为度，滤去滓，捣烂待用，油冷时下黄丹，用文武火熬，槐、柳条不住手搅，出黑烟气，熬至滴水成珠，手试软硬适度方可离火，次将后项细药并入捣烂粗滓内。真阿魏2两，乳香5钱，没药5钱，麝香1钱，皮消3钱（风化为末）。
出处：《寿世保元》卷八。
主治：癖积气块，身体发热，口内生疳。
加减：如有马刀疬子疮，加琥珀1两。
功效：None
用法用量：食用时坐水中熔开（不可火上化），用狗皮摊贴患处，每个重7钱。贴3日止热，7日觉腹微痛，10日大便下脓血为验。
制备方法：上调匀，以瓷器内盛之。
临床应用：None
各家论述：None
用药禁忌：忌生冷、腥荤、发物百日。
附注：清凉化痞膏（《何氏济生论》卷五）、化痞膏（《杂病源流犀烛》卷十四）。
