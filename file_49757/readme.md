序号：49772
名称：茯苓苏
组成：茯苓5斤（灰汁煮10遍，浆水煮10遍，清水煮10遍），松脂5斤（煮如茯苓法，每次煮40遍），生天门冬5斤（去心皮，晒干作末），牛酥3斤（炼30遍），白蜜3斤（煎令沫尽），蜡3斤（炼30遍）。
出处：《千金》卷二十七引彭祖方。
主治：None
加减：None
功效：养性。
用法用量：先1日不食，欲不食先须吃好美食，令极饱，然后绝食，即服2两，20日后服4两，又20日后8两；细丸之，以咽中下为度。第2度以4两为初，20日后8两，又20日2两。第3度服以8两为初，20日2两，又20日4两，合180日，药成自后服3丸将补，不服亦得，恒以酥蜜消息之，美酒服1升为佳。
制备方法：上6味，各捣筛，以铜器重汤上，先纳酥，次蜡，次蜜，消讫，纳药，急搅之勿住，务令大均，细瓷器中，蜜封之，勿泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
