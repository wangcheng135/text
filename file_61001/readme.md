序号：61016
名称：铁井阑
组成：大五倍子（去蛀屑，微炒成团，候冷研细）3两，杜蟾酥（干，研细）5钱，藤黄3两（先以好醋入铜杓上微火化烊，绢漉去滓听用），明矾1两（研），胆矾8钱（研），大黄2两，皂角2两，白及2两，山慈菇2两，天南星1两。
出处：《疡科纲要》引朱阆仙方。
主治：痈疽大毒漫肿无垠，根脚四散，其毒不聚；疮发于骨节转侧之间，酿脓化毒。
加减：None
功效：退肿移毒，收束疮根。
用法用量：临用时以醋磨浓，涂疮根四围，干则润之以醋，1日洗去，再涂；欲移毒使偏，则如上法涂其一偏，而涂药处自能退肿，其毒聚于未涂药之一偏，可保骨节不致损害。
制备方法：先以后五物用陈米好醋2大碗文火熬浓，绞去滓，乃和入醋煮之，藤黄同熬成膏，俟极浓，乃和入5倍、蟾酥、二矾细末调匀，离火，再入麝香细末3钱杵匀，制成锭子，阴干收藏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
