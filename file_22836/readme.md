序号：22844
名称：西州续命汤
组成：麻黄3两，生姜3两，当归2两，石膏2两，芎1两，桂心1两，甘草1两，黄芩1两，防风1两，芍药1两，杏仁40枚。
出处：《千金》卷十五。
主治：肉极。虚热，肌痹淫淫，如鼠走身上，津液开泄，或痹不仁，四肢急痛。
加减：None
功效：None
用法用量：以水9升，先煮麻黄，去沫，下诸药，煮取3升，去滓，分4服，日2次。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：续命为风痱身体不能自收、正虚风中之首药，乃《古今录验》方中除去人参，加入黄芩，谓之西州续命。更添小续命中芍药、防风二味，并以生姜易干姜，即小续命中除去附子、防已，专力开发风痹。以无脾虚喘之，故无取于人参；以无肾虚逆冷，故无取于附子；以无下体疼重，故无取于防己也。
用药禁忌：None
附注：本方方名，《普济方》引作“续命汤”。
