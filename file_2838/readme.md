序号：2838
名称：天竺黄丸
组成：天竺黄（研）1分，牛黄（研）1分，雄黄（研）1分，龙脑（研）1分，犀角（镑）1分，麝香（研）1分，水银1分，丹砂半两（研为末，1半纳铫子中，入前水银，更入1半丹砂，热熔搅匀，下火刮取），西甘石（研）半两，天麻半两，乌蛇（酒浸，去皮骨，炙）半两，干蝎（酒炒）半两，白僵蚕（炒）半两，蝉壳（微炙）半两，桑螵蛸（炙）半两，羚羊角（镑）半两，莎草根（炒去毛）半两，附子（炮裂，去皮脐）半两，白附子（炮）半两，羌活（去芦头）半两，独活（去芦头）半两，蔓荆实（去白皮）半两，麻黄（去根节，先煎，掠去沫，焙）半两，狐肝1具（炙干）。
出处：《圣济总录》卷六。
主治：中急风。
加减：None
功效：None
用法用量：每服2丸至3丸，以豆淋薄荷酒送下，加至5丸，后以热稀姜粥投之，日夜可3-4服，汗出多即减服数。
制备方法：上药除研药外，为细末，再入研药拌匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
