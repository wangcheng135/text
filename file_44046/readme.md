序号：44061
名称：复元通气散
组成：舶上茴香（炒）2两，穿山甲（锉，蛤粉炒，去粉）2两，南木香（不见火）1两半，延胡索（擦去皮）1两，白牵牛（炒，取末）1两，陈皮（去白）1两，甘草（炒）1两。
出处：《局方》卷八（续添诸局经验秘方）。
主治：疮疖痈疽，方作焮赤，初发疼痛，及脓已溃、未溃，小肠气、肾痈、便毒，腰痛气刺，腿膝生疮，及妇人吹奶。气不宣流或成疮疖，并闪挫腰胁，气滞疼痛。
加减：None
功效：None
用法用量：每服1大钱，热酒调服。病在上，食后服；病在下，食前服；不饮酒人，煎南木香汤调下。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：复元通圣散（《万氏家抄方》卷三）。本方方名，《准绳·类方》引作“复原通气散”。
