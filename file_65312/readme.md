序号：65328
名称：大瓢李末药
组成：当归5钱，防风5钱，川芎5钱，白芷5钱，细辛5钱，麻黄5钱，荆芥5钱，全蝎5钱，天麻5钱，藁本5钱，雄黄5钱，羌活5钱，甘草5钱，朱砂3钱，人参3钱，白花蛇3钱，大茴香3钱，两头尖3钱，香蛇7钱，川乌4两，草乌4两，苍术4两。
出处：《解围元薮》卷三。
主治：男妇疠风瘫痪，口眼歪斜，面如虫行，身痛如切，或皮肉淫痒难忍，久而手足反张者。
加减：None
功效：None
用法用量：每服5-6分，渐至1钱，临卧以无灰酒调服。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：忌一切热物。服此后用凤仙花梗煎汤洗浴，汗出为度。
附注：服药后若有汗麻木，身痒，乃药力至也。
