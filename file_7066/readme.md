序号：7066
名称：四物驱风汤
组成：当归1两，川芎5钱，白芍2钱，熟地1两，肉桂1钱，秦艽2钱，枳壳1钱半，红花1钱半，冬葵子1钱半，车前子1钱半，生芝麻3钱。
出处：《陈素庵妇科补解》卷四。
主治：孕妇平日不善调养，肝脾二经受伤，肝脾虚而风邪乘之，临产忽然口噤目翻者。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：口噤、目翻，虚风之症现矣。若胎不下，而反上逼则危在须臾。然过于补虚，又恐留而不产，反为害事。是方四物补血，秦艽祛风，肉桂、红花行血，冬葵、车前滑窍，枳壳宽肠。胎下则气不上壅，血不逆行，虚风之症，可以渐退矣。
用药禁忌：None
