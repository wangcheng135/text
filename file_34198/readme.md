序号：34207
名称：牡蛎术散
组成：防风10分，白术9分，牡蛎3分（熬）。
出处：方出《深师方》引赵子高方（见《外台》卷十五），名见《元和纪用经》。
主治：汗发过多，头眩，汗未止，筋惕肉者。
加减：恶风，倍防风；少气，倍术；汗出面肿，倍牡蛎。
功效：None
用法用量：每服方寸匕，增至2-3匕，日3次，酒调下。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：忌桃、李、雀肉、胡荽、大蒜、青鱼。
附注：防风白术散（《伤寒总病论》卷二），防风白术牡蛎汤（《活人书》卷十六）、牡蛎白术散（《圣济总录》卷十三）、防风白术牡蛎散（《校注妇人良方》卷三）、防术牡蛎汤（《医学入门》卷四）。《伤寒总病论》本方用法：每服二钱，每日2-3次，温米饮调下。
