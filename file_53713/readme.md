序号：53728
名称：宽中健脾丸
组成：白术6两，人参2两，黄耆2两，苍术2两，茯苓2两，五加皮2两，黄连（用茱萸水炒）2两半，白芍2两半，泽泻2两半，陈皮（用盐水炒）3两，半夏3两，香附3两，薏苡仁3两，山楂3两，草豆蔻1两半，苏子1两半，萝卜子1两半，沉香6钱，大瓜蒌2个（每个镂一孔，用川椒末3钱，多年粪碱末2钱，装入瓜蒌内，纸糊瓜口，盐泥固济，晒干，煅红为度，去泥与黑皮）。
出处：《医学入门》卷七。
主治：单腹胀，及脾虚肿满，膈中闭塞，胃口作痛。
加减：None
功效：None
用法用量：每服100丸，白汤送下。
制备方法：上药同为末，用荷叶、大腹皮煎汤煮黄米糊丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
