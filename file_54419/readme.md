序号：54434
名称：柴胡消痈汤
组成：柴胡10～20g，夏枯草10g，紫地丁10g，三颗针10g，黄芩5～10g，白芍5～10g，芒消5～10g，野菊花5g，郁金3～5g，大黄3～5g，法半夏3～5g，甘草3～5g。
出处：《古今名方》。
主治：胆道感染，痈疡型（脓毒型）。
加减：热重，重用柴胡、黄芩，选加紫地丁、野菊花、黄连；大热、大渴、脉洪大者，加石膏、知母、天花粉、鲜芦根；湿重，重用茵陈、郁金、金钱草、栀子、大黄；痛重，加玄胡、川楝子；呕吐，加半夏、竹茹、生姜；驱蛔，加苦楝根皮、槟榔、使君子；瘀血，加丹参、川芎、红花；气阴欲脱，加独参汤、生脉散。
功效：清热解毒，通里泻火。
