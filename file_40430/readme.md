序号：40440
名称：虎肚散
组成：蟾酥1两（同葱捣烂，白面包，火内煨热），厚朴15两（姜汁炒），红芽大戟2两5钱（紫色者佳），赤金2钱（放煎银罐内，用硫黄末将金花碎）。
出处：年氏《集验良方》卷四。
主治：噎食病。
加减：None
功效：None
用法用量：年少者，每日清晨用无灰热酒冲服3分，10日共服3钱；年大者，每日清晨用无灰热酒冲服5分，10日共服5钱，即愈。其饭食用京米煮饭，熟时每1碗饭，将柿饼切如米粒大，止用半碗拌在饭内，又复蒸烂食之，以大好为度。
制备方法：用虎肚1个，其肚内之物不可倒出，将各药共为粗末，入虎肚内，放在铁锅内用大火煅炼成灰，研成细末。
临床应用：None
各家论述：None
用药禁忌：忌汤水并气怒、劳碌、房事，如渴极时，汤水少用些。
附注：None
