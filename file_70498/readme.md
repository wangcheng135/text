序号：70514
名称：紫金丹
组成：铁渣1斤（淘净，控干），硇砂2两，硫黄半斤（水飞过），上3味先取铁滓、硫黄2味，于瓷器内用米醋1斗，慢火煎，候煎硫黄火上无焰即煿干，刮此2味，入瓷合内固济了，用大火煅3度毕，取盒内药，再用水飞，不用铁渣，控硫黄令干，入细瓷盒内，坐在平地，别用火一秤，煅至火尽，取出入硇砂2两，同研令细，再入盒内，用火1斤，就灰池中养3日，放冷，取2两，再入下项药：硇砂半两，木香（末）1钱，丁香（末）1钱，腻粉（研）1钱，肉豆蔻（去壳，末）1钱，丹砂（研）1钱，干漆（炒烟尽，研）1钱，胡椒（末）1钱，阿魏（用醋化面和作饼子，烧热为末）1钱，砒霜（末）1字半。
出处：《圣济总录》卷七十二。
主治：男子、妇人久积、气块、癥癖，两胁下积冷，胸腹气刺痛。
加减：若取积滞癥癖，及酒食积急气、冷气、一切滞气等疾，加巴豆仁（研）1两，六戟末半两，芫花末半两，草乌头末1分，五灵脂末1两，腻粉2钱，硇砂（研）半两。上以米酽醋2升铫子内熬成膏，便与前药末搜为剂，丸如梧桐子大。量虚实加减服，逐1丸加，用醋汤或茶清送下。
功效：None
用法用量：每服5丸，渐加至20丸。
制备方法：上为细末，再同硫黄研匀，用酒醋各半煎五灵脂、薄面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
