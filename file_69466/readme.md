序号：69482
名称：犀羚三汁饮
组成：犀角尖1钱，带心连翘2钱，东白薇3钱，皂角刺3分，羚角片1钱半，广郁金3钱（杵），天竺黄3钱（老式），粉丹皮1钱半，淡竹沥2瓢，鲜石菖蒲汁2匙，生藕汁2瓢。
出处：《重订通俗伤寒论》。
主治：邪陷包络，挟痰瘀互结清窍，症必痉厥并发，终日昏睡不醒，或错语呻吟，或独语如见鬼，目白多现红丝，舌虽纯红，兼罩粘涎。
加减：None
功效：开窍透络，豁痰通瘀。
用法用量：先用犀、羚二角，鲜茅根50支（去衣），灯心5分，活水芦笋1两，煎汤代水，三汁和匀同冲。临服调入至宝丹4丸和匀化下。
制备方法：None
临床应用：None
各家论述：以犀、羚凉血熄风，至宝芳香开窍为君；臣以带心连翘宣包络之气郁，郁、丹通包络之血郁，白薇专治血厥，竺黄善开痰厥；尤必佐角刺，三汁轻宣辛窜，直达病所，以消痰瘀；使以芦笋、茅根、灯心，轻清透络。庶几痰活瘀散，而包络复其横通四布之常矣，此为开窍透络豁痰通瘀之良方。
用药禁忌：None
附注：None
