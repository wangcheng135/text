序号：27284
名称：补肾方
组成：磁石2两（研，绵裹），生姜2两，防风2两，桂心2两，甘草1两（炙），五味子2两，附子1两（炮），玄参2两，牡丹皮3两，大豆24枚。
出处：《外台》卷十七引《深师方》。
主治：肾气不足，心中悒悒而乱，目视茫茫，心悬少气，阳气不足，耳聋，目前如星火；一身悉痒，骨中痛，少腹拘急，乏气咽干，唾如胶，颜色黑。
加减：None
功效：消疽痔。
用法用量：以水1斗2升，先于银器中扬300遍，煮药，取6升，去滓，更煎取2升入合，分3次服。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：忌海藻、菘菜、猪肉、冷水、生葱、胡荽等。
附注：磁石散（《普济方》卷二十九）。
