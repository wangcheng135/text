序号：38089
名称：服天门冬方
组成：天门冬（晒干）。
出处：《千金》卷二十七。
主治：虚劳绝伤，年老衰损，羸瘦，偏枯不遂，风湿不仁，冷痹，心腹积聚，恶疮痈疽肿癞疾，重者周身脓坏，鼻柱败烂，阴痿耳聋，目暗。
加减：None
功效：补中益气，愈百病，白发变黑，齿落复生，延年益命。
用法用量：每服方寸匕，食后服，1日3次，可至10服。小儿服尤良。与松脂若蜜丸服之益善。或捣取汁，微火煎取5斗，下白蜜1斗，胡麻炒末2升，合煎，搅之勿息，可丸即上火，下大豆黄末和为饼，径3寸，厚半寸，每服1枚，1日3次。或釀酒服。
制备方法：上药捣下筛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
