序号：77031
名称：催生丹
组成：麝香（别研）1字，乳香（别研极细）1分，母丁香（取末）1钱，兔脑髓（腊月者，去皮、膜，研）。
出处：《局方》卷九。
主治：产妇生理不顺，产育艰难，或横或逆。
加减：None
功效：None
用法用量：每服1丸，温水送下。
制备方法：上拌匀，以免脑和丸，如鸡头瓤大，阴干，用油纸密封贴。
临床应用：None
各家论述：None
用药禁忌：None
附注：神效催生丹（《卫生家宝产科备要》卷六）、顺生丹（《校注妇人良方》卷十七）、兔脑丸（《医学六要》卷七）、催生丸（《良朋汇集》卷四）、兔脑催生丹（《女科指掌》卷四）、催生兔脑丸（《灵验良方汇编》卷上）、手握丹（《胎产心法》卷四）、速产兔脑丸（《饲鹤亭集方》）。
