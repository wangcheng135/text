序号：27355
名称：补益天雄丸
组成：天雄（炮裂，去皮脐）1两，菟丝子（酒浸1宿，焙干别捣）1两，柏子仁1两，石斛（去根，锉）1两，巴戟1两，天门冬（去心，焙）1两，牛膝（去苗）1两，干漆（捣碎，炒令烟出）1两，肉苁蓉2两（酒浸1宿，利去皱皮，炙令干），熟干地黄2两，肉桂2两（去皱皮）。
出处：《圣惠》卷二十七。
主治：风劳气，身体疼痹，手足无力，气血虚损，颜色萎黄，精神昏沉，饮食无味。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。渐加至40丸。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
