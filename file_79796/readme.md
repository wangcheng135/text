序号：79812
名称：干漆丸
组成：干漆半两（捣碎，炒令烟出），熟干地黄1两，山茱萸半两，五味子半两，牛膝1两（去苗），白术半两，续断半两，蛇床子半两，甘草半两（炙微赤，锉），桂心半两，肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），石斛1两，（去根，锉），菟丝子1两（酒浸1宿，晒干，别捣罗为末），巴戟半两，酸枣仁半两（微炒），柏子仁半两，薏苡仁半两，鹿茸1两（去毛，涂酥炙微黄）。
出处：《圣惠》卷二十六。
主治：五劳六极七伤，虚羸不足。
加减：None
功效：令人肥健，益气力。
用法用量：每服30丸，空腹及食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
