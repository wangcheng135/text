序号：61086
名称：铁粉丸
组成：铁粉2分（细研），蛇蜕皮半两（烧灰），鬼督邮3分，龙齿半两，寒水石2两，败天公1两（烧灰），防风1两（去芦头），沙参半两（去芦头），羚羊角屑1两半，龙胆2两（去芦头），乌犀角屑2两，蚱蝉1两（微炙），地骨皮2两，商陆1两，牛黄1分（细研），石膏2两（细研，水飞过），黄连半两（去须）。
出处：《圣惠》卷六十九。
主治：妇人风邪癫狂，每发狂乱妄语，倒错不识人。
加减：None
功效：None
用法用量：每服20丸，煎地骨皮汤送下，不拘时候。
制备方法：上为末，入研了药，同研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
