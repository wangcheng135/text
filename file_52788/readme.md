序号：52803
名称：香盐散
组成：香附子1斤（连皮，生姜半斤取汁，浸1宿，煮干），孛姑缨1斤（少者半斤，根花叶皆用），食盐8两，破故纸6两，象兜5两，大黄5两，青盐4两，地黄4两。
出处：《医方类聚》卷八十三引《经验秘方》。
主治：None
加减：None
功效：乌发，嫩容，牢牙，补肾。
用法用量：每早、晚蘸药刷牙，将口内咸唾津吐手心，润搽面皮并髭髯。每月初六、十六、二十六日，摘去白者。常用此药，白者自黑，又时用酥油、胡桃油、乌臼油搽润发髯，用之日久，自知功效。
制备方法：上为末。先将瓶1个，用纸筋、盐、韭和泥固济，窨干盛药，以小瓦片盖瓶口，泥封之。若泥缝有裂处，以泥补合，候干，择成定日，就地做大深土盆1个，径阔2尺，先于内放炭火，厚5寸，上坐药瓶，周围及顶上亦用炭火，煅炼约多半日，存性，火冷，埋土内去火毒，为细末，瓷瓶收之。
临床应用：None
