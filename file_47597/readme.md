序号：47612
名称：千金散
组成：千金子（取白仁，去油）约1两，枳实（炒）5钱，青皮（炒）5钱，陈皮5钱，香附5钱，山楂肉5钱，木香5钱，砂仁5钱，云术（土炒）5钱，沉香3钱。
出处：《惠直堂方》卷二。
主治：一切臌胀。
加减：None
功效：None
用法用量：五更尽，用淡姜汤调下。天明利3-4次，不甚泻，每日1服，连服7日为止。如人虚，2日1服，病浅者3-5服能愈。愈后除千金子外，9味末，以陈米糊为丸，每服1钱，空肚清汤送下。
制备方法：9味为末，称5分，加千金子霜8分，入生蜜调丸。
临床应用：None
各家论述：None
用药禁忌：忌生冷、牛、羊、猪、鹅、油腻、煎炒、糟、面、盐、醋等物两个月，终身忌团鱼、河豚、骡马、母猪、牛肉、王瓜、南瓜、荞麦，犯之立复。
附注：None
