序号：83127
名称：乌头煎丸
组成：乌头1斤（大者，刮去黑皮，锉作小片子，用雪水3升浸1宿，入盐6两，同煮干，炒令黄），牛膝3两（酒浸1宿，切，焙），肉苁蓉（细切，用法酒1升浸1宿，焙令干）4两，巴戟天（去心）1两，大枣半斤（汤浸，去皮核，焙），桃仁（浸，去皮尖，麸炒熟）5两，陈橘皮（汤浸，去白，焙）半斤，蜀椒（去目并闭口，炒出汗）2两，厚朴（去粗皮，生姜汁炙）3两。
出处：《圣济总录》卷一八七。
主治：None
加减：None
功效：壮筋骨，明耳目，补虚爽神益气。
用法用量：每服30丸，空心盐汤送下。
制备方法：上为末，以酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
