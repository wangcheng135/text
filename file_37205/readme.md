序号：37215
名称：定喘止咳糖浆
组成：麻黄150g，苦杏仁450g，五味子112.5g，甘草150g，杏仁水20ml，蔗糖625g，香精适量、苯甲酸钠3g。
出处：《河南省药品标准》。
主治：哮喘性支气管炎，慢性支气管炎。
加减：None
功效：定喘止咳。
用法用量：口服，1次10-15毫升，1日3次，小儿酌减，用时摇匀。
制备方法：取五味子、甘草、苦杏仁，麻黄加水煎煮2次，煎液过滤，合并浓缩至约1升，静置36小时，取上清液，加蔗糖，苯甲酸钠，煮沸溶解并浓缩至热测比重1.22，趁势过滤，放冷，加杏仁水，香精及水至1升，搅拌均匀，即得。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
