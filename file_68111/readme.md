序号：68127
名称：黄耆防风饮子
组成：蔓荆子半钱，黄芩半钱，炙甘草1钱，黄耆1钱，防风1钱，葛根1钱半，细辛2分。
出处：《原机启微》卷下。
主治：眼棱紧急，以致倒睫卷毛，损睛生翳，及上下睑眦赤烂羞涩难开，眵泪稠粘。
加减：None
功效：None
用法用量：水2盏，煎至1盏，去滓，大热服。
制备方法：None
临床应用：None
各家论述：方以蔓荆子、细辛为君，除手太阳、手少阴之邪，肝为二经之母，子母平安，此实则泻其子也；以甘草、葛根为臣，治足太阴、足阳明之弱，肺为二经之子，母薄子单，此虚则补其母也；黄耆实皮毛，防风散滞气，用之以为佐；黄芩疗湿热，去目中赤肿，为之使也。
用药禁忌：None
附注：None
