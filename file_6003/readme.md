序号：6003
名称：加减平阳汤
组成：葛根3钱，云苓5钱，甜白术5钱，麦冬5钱，川贝母3钱（去心，研），生石膏3钱，柴胡1钱，枳壳1钱5分，橘红1钱5分。
出处：《慈航集》卷下。
主治：疟疾。初病令人先寒洒浙，洒浙寒甚，发热头痛鼻于，渴欲饮水，目眴眴不得眠，甚则烦躁谵语，畏火，厌听人声喧哗，寒久乃热，热退汗出，喜见日月光火气，心乃快然，善饥而能食。
加减：如体虚者，加人参1钱，贫人无力者，以上党参1两代之；如恶心，加广藿香3钱；如胃口饱闷作胀，加草蔻仁2钱，槟榔1钱5分；如寒多，加煨姜3钱，大枣3枚；如内热口渴，加知母2钱；如皮外热，加青蒿2钱；如作泻，加炒白芍5钱，车前子3钱。
