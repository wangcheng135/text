序号：35450
名称：三才大补膏
组成：生地黄1斤，熟地黄1斤，天门冬4两，麦门冬4两，人参4两，甘枸杞4两，牛膝4两，何首乌8两。
出处：《古今医鉴》卷七。
主治：None
加减：None
功效：补益。
用法用量：上(口父)咀，勿犯铁器，同入大砂锅内，用水20碗，煎至7碗，取汁别贮；药渣如前再煮9次，共得汁70碗，滤渣极净；别用中等砂锅，入汁7碗，慢火煎熬，耗汁1碗，方添1碗，63碗皆添尽，则汁已浓矣，盖只得汁6碗；却用山白蜜去蜡，可1斤半，同前药入砂锅内，重汤煮汁，滴水不散，则成膏矣。瓷罐盛之，埋上中7日，取出，如前再煮1昼夜，再埋1宿，乃分贮小罂内封固，自煎至煮，但用桑柴火，药本寻常，妙在火候。不拘时以醇酒调服，味美而功多。若惩忿窒欲之人，又深居简出，时服此膏，亦可以擅其天年矣。
