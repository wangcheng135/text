序号：9249
名称：牛黄丸
组成：牛黄（研）1分，丹砂（研）半两，雄黄（研）半两，马牙消（研）1两，麝香（研）1钱，龙脑（研）1钱，大黄（饭上炊3遍）1两，黄芩（去黑心）半两，山栀子仁半两，栝楼根（锉）半两，白药子半两，甘草（炙）半两，天竺黄（研）半两，郁金（皂荚水浸3宿，煮软，切作片，焙干）1两。
出处：《圣济总录》卷一六九。
主治：小儿挟热多惊，心神烦躁，赤眼口疮，遍身壮热，大小便多秘，或生疮癣，咳嗽多涎，睡卧惊叫，手足搐搦，急慢惊风，渴泻。
加减：None
功效：None
用法用量：每服旋丸1黑豆许，用金银薄荷煎汤化下。
制备方法：上为末，再同研匀，炼蜜和成挺子。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
