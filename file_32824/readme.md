序号：32833
名称：羌活汤
组成：羌活1钱半，苍术（米泔浸）1钱半，黄芩（酒炒）1钱半，当归1钱半，芍药（炒）1钱半，茯苓（去皮）1钱半，半夏（姜汁炒）1钱半，香附1钱半，木香（另研）7分，陈皮7分，甘草3分。
出处：《回春》卷五。
主治：痛风症，遍身骨节走注疼痛。
加减：风痛，加防风；湿痛，加苍术；热痰痛，倍酒芩、瓜蒌、枳实、竹沥；血虚痛，加生地黄；上痛，加白芷、威灵仙；下痛，加黄柏、牛膝；痛甚，加乳香；发热，加柴胡；小水短涩，加木通；手臂痛，加薄桂。
功效：活血疏风，消痰去湿。
用法用量：加生姜3片，水煎服。
制备方法：上锉一剂。
临床应用：None
各家论述：None
用药禁忌：不可食肉，戒油炒、热物、鱼、面。
附注：None
