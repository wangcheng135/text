出处：《经验广集》卷四。
主治：风寒湿气，腰膝痠疼，男子痞块，女人血瘕；肿毒恶疮，瘰疬，顽癣；肚腹疼痛，泻痢，疟疾，咳嗽喘急，胸膈胀闷，面色痿，胃脘疼痛；负重伤力，浑身拘痛；小肠疝气。
加减：None
功效：None
用法用量：风寒湿气，腰膝痠疼，男子痞块，女人血瘕，及肿毒恶疮，瘰疬，顽癣，俱贴患处；肚腹疼痛，泻痢疟疾，俱贴脐上，痢白而寒者尤效；咳嗽喘急，胸膈胀闷，男妇面色萎，并心胃脘疼痛，俱贴前心；负重伤力，浑身拘痛，贴后心腰眼；小肠疝气，贴脐下。
制备方法：上切为粗片，用真麻油12斤浸药，春5、夏3、秋7、冬10，煎药枯滤去滓，倾瓷器内，另以松香先下油锅熔化，后方加油药，大约油4两入松香2斤，试软硬得所，仍滤入水缸中，令人抽扯，色如黄金即成禽矣。一料计17斤，摊大膏药。
