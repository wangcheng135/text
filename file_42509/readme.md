序号：42524
名称：青麟丸
组成：大黄20斤，黄柏8两，黄芩8两，猪苓8两，赤苓8两，泽泻8两，木通8两，车前子8两，米仁8两，粉萆薢8两，生侧柏8两，玄参8两，广皮8两，薄荷8两，制香附8两。
出处：《中药成方配本》。
主治：湿毒疮疡，目赤口碎，头痛齿痛，二便不利。
加减：None
功效：清热利湿。
用法用量：每次1钱，开水送下，1日2次。
制备方法：先将14味药（大黄除外）煎汁去滓，将汁拌大黄蒸黑，晒干，将蒸余之汁和加黄酒5斤，再拌大黄蒸3小时，晒干；将锅内余汁拌大黄，再晒干，研末，每斤净粉用白蜜4两炼熟化水泛丸，如椒目大，约成99斤。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
