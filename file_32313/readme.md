序号：32322
名称：沉香汤
组成：沉香1两，肉豆蔻仁半两，桂（去粗皮）半两，木香半两，厚朴（去粗皮，生姜汁炙）半两，槟榔半两，青橘皮（汤浸，去白，焙）半两，诃黎勒皮半两，白术半两，当归（焙）半两，京三棱（醋浸一宿，煨，锉）半两，人参半两，枇杷叶（炙，刷去毛）半两，芎半两，干姜（炮）半两，蓬莪术（煨，锉）半两，黄耆半两，郁李仁（汤浸，去皮并双仁）半两，附子（炮裂，去皮脐）半两，白茯苓（去黑皮）半两，石斛（去根，酒浸，微炙）半两，前胡（去芦头）半两，枳壳（去瓤，麸炒）半两，甘草（炙）半两。
出处：《圣济总录》卷一八七。
主治：None
加减：None
功效：补虚治气，调顺三焦，安和脏腹，进饮食。
用法用量：每服5钱匕，水1盏半，加生姜3片，大枣3枚（擎破），煎至8分，去滓，稍热，食前服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
