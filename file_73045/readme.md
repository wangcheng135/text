序号：73061
名称：曾青丸
组成：曾青（蒸痞倍之）2分，干姜（寒痞倍之）2分，(庶虫)虫（血痞倍之）2分，紫石英2分，牡丹（虫痞倍之）2分，桂心2分，大黄5分，龙骨（蛇痞倍之）5分，蜀漆7分，龟甲（鱼痞倍之）3分，鳖甲（气痞倍之）3分，珍珠3分，蜚蠊（风痞倍之）3分，细辛6分，附子（炮）4分。
出处：《幼幼新书》卷二十二引《婴孺方》。
主治：小儿八痞。一、蒸痞，当心下坚痛，大如小杯；二、蛇痞，如板起于胁下，抢心；三、鱼痞，夹脐如手；四、寒痞，绕脐腹雷鸣；五、虫痞，当心如杯，不可动摇；六、气痞，心下如盘；七、血痞，生于寒热，腰背痛状如疟；八、风痞，脓出腹痛。
加减：None
功效：None
用法用量：5-6岁儿未食服4丸，小儿丸子大小量儿与之，日3服。服之当微烦，勿怪。
制备方法：上为末，蜜和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌猪、鱼、菜物。
附注：None
