序号：56555
名称：大效双丸
组成：甘遂半两（煨令微黄），巴豆霜半两，麦门冬半两（去心，焙），牡蛎1分（烧为粉），蕤仁1分（汤浸，去皮，研入），真珠末1分。
出处：《圣惠》卷八十八。
主治：小儿百病，身热头痛，饮食不消，心腹胀满；或心腹(疒丂)痛，大小便不利；或重下数起；或别无异疾，惟饮食过度，不知自止，哺乳失覆；或惊寒热；或蒸候，哺食减少，气息不快，夜啼不眠，是腹内不调。
加减：None
功效：None
用法用量：半岁儿以温水服1双，如荏子大；1岁服1双，如半麻子大；2岁、3岁服1双，如麻子大，常以鸡鸣时服。至日出当下；如不下者，以热粥饮投之，即下。
制备方法：上为末，入研了药，都研令匀，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
