序号：45232
名称：枳壳煮散
组成：枳壳（去瓤，麸炒黄）4两，细辛（去叶）4两，桔梗（炒）4两，防风（去钗股）4两，川芎4两，葛根1两半，甘草2两（炙）。
出处：《本事》卷七。
主治：悲哀烦恼伤肝气，致两胁骨疼痛，筋脉紧急，腰脚重滞，两股筋急，两胁牵痛，四肢不能举，渐至脊膂挛急。
加减：None
功效：None
用法用量：每服4钱，水1盏半，加生姜3片，煎至7分，滤去滓，空心，食前温服。
制备方法：上为粗末。
临床应用：None
各家论述：《本事方释义》：枳壳气味苦寒，入足太阴；细辛气味辛温，入足少阴；桔梗气味苦辛平，入手太阴；防风气味辛甘微温，入足太阳；川芎气味辛温，入足少阳、厥阴；葛根气味辛甘平，入足阳明；甘草气味甘平，入足太阴，能缓药之性。此由肝气不舒，故用苦降升散之药，再佐以甘缓，及姜之和卫，使升降不致偏胜也。
用药禁忌：None
附注：None
