序号：72003
名称：小续命汤
组成：麻黄（去根节）1两（气实者全用，气虚者一半，以威灵仙代一半），木香1两（不见火），缩砂仁1两，人参（去芦）1两，芎，甘草（炙）1两，杏仁（去皮尖，炒）1两，汉防己1两，桂心（去粗皮）1两，北防风1两半，附子（炮裂，去皮脐）半两，川乌（炮）3分，白芍药1两，黄芩7钱，独活1两。
出处：《普济方》卷九十六引庞安常方。
主治：半身不遂，口眼斜，手足战掉，语言謇涩，肢体麻痹，神思昏乱，头目眩重，痰涎壅盛，筋脉拘挛，屈伸转侧不便，涕唾不收。
加减：None
功效：None
用法用量：每服3钱半，加生姜5片，枣子1个，煎至7分，去滓，食前温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
