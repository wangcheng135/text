序号：26261
名称：羊肝退翳丸
组成：怀生地3两，熟地黄3两，白茯神（人乳拌，蒸晒）3两，怀山药（炒）3两，甘枸杞4两，夜明砂（淘净）4两，木贼草（蜜水拌炒）2两，蜜蒙花（蜜拌炒）2两，青葙子2两，草决明2两5钱（捶碎，用水浸拌炒），川黄连8钱（白酒浸1宿，微炒），黑羊肝1具（去外膜，蒸熟）。
出处：《疡医大全》卷十一。
主治：雀盲眼，一切昏花老眼。
加减：None
功效：None
用法用量：每服2-3钱，空心淡盐汤送下。
制备方法：上为粗末，同羊肝捣匀，再烘晒令干，再为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌萝卜、胡椒、鸡鸭蛋。
附注：None
