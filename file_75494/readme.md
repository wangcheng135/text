序号：75510
名称：解毒寿婴丹
组成：黄连3钱，连翘7钱，赤芍5钱，玄参3钱，生地3钱，甘草2钱，丝瓜近蒂3寸（霜降后取存性），先用前料为细末，次入后药：西牛黄1钱，生玳瑁1钱（如无以好琥珀代之），珍珠7分，乌犀（磨末）3钱，羚羊角（磨末）3钱，真麝香2分，冰片1分，好青黛1钱（研极细）。
出处：《痘疹仁端录》卷十三。
主治：解胎毒稀痘。
加减：None
功效：None
用法用量：量儿大小，用灯心汤磨服。
制备方法：令前后和匀，以甘草膏为丸，如芡实大，朱砂3钱为衣，再以金箔外裹，熔蜡为丸。
临床应用：小儿肥疮：一儿头面多生肥疮，身上生狗癣疥，服此药，大便解去垢秽，如涎涕者，诸症悉愈。
各家论述：None
用药禁忌：None
附注：None
