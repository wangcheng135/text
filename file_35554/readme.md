序号：35564
名称：侧子丸
组成：侧子（炮裂，去皮脐）1两3分，白芷（微炒）1两1分，附子（炮裂，去皮脐）1两1分，天麻（酒浸，切，焙）1两1分，龙骨1两1分，防风（去叉、制）1两1分，蔓荆实1两半，白蒺藜（炒）2两，白术（炒）1两，人参3分，山芋3分，生干地黄（焙）3分，当归（切，焙）3分。
出处：《圣济总录》卷十八。
主治：大风癞，眉须堕落，身上有疮，手足胀闷，（疒帬）痹挛缩，鼻梁未坏者。
加减：None
功效：None
用法用量：每服30丸，渐加至50丸，空心温酒送下，每日2次。唇舌微痹为效。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：慎避外风。
附注：None
