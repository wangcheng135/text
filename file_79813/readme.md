序号：79829
名称：干漆丸
组成：干漆1两（捣碎，炒令烟出），牡丹3分，赤芍药半两，琥珀1两，桃仁1两（汤浸，去皮尖双仁，麸炒微黄），牛膝1两（去苗），桂心3分，吴茱萸3分（汤浸7遍，炒），川大黄1两（锉，微炒），水蛭30枚（炒令黄），蝱虫30枚（去翅足，微炒），菴（艹闾）子1两，乱发灰1钱，(庶虫)虫35枚（微炒），大麻仁半两，鳖甲1两（涂醋，炙令黄，去裙襕），蛴螬13枚（微炒）。
出处：《圣惠》卷七十九。
主治：产后恶血不散，结成症块，经脉不利。
加减：None
功效：None
用法用量：每服20丸，空心以温酒送下。
制备方法：上为末，炼蜜和丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
