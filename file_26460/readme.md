序号：26468
名称：当归芍药和疟汤
组成：全当归1两（酒洗），白芍1两（酒炒），益母草3钱，炮差1钱5分，青皮1钱5分，柴胡8分（炒），草蔻仁1钱（研），炙甘草5分。
出处：《慈航集》卷下。
主治：产后疟疾，夹痰夹滞，寒热不止。
加减：如腹痛积瘀，恶露未清，加炒山楂3钱，酒炒元胡索3钱；如恶心，加灶心土5钱；如头痛，加川芎2钱；如大便结，加桃仁泥3钱，酒洗红花3钱；如作泻，加车前子3钱；如胸口饱胀，加炒枳壳1钱5分；加热盛，加青蒿3钱；如寒多，加入煨姜3钱、大枣3枚。
功效：补血和解。
用法用量：河、井水各半煎，露1宿，早服。3服全愈。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
