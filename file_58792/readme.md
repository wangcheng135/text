序号：58807
名称：秘传涤风散
组成：川乌4两，草乌（并火炮，去皮尖）4两，苍术（米泔浸）4两，人参2钱，白茯苓2钱，两头尖2钱，僵蚕7钱（用纸隔炒），甘草（炙）3两，白花蛇（酒浸3日，弃酒，火炙，去皮骨）1两，石斛（酒浸）1两，川芎5钱，白芷5钱，细辛5钱，当归（酒洗）5钱，防风5钱，麻黄5钱，荆芥5钱，全蝎（新瓦上焙干）5钱，何首乌（米泔水浸，忌铁），天麻5钱，藁本5钱。
出处：《松崖医径》卷下。
主治：痔漏及一切疯证。
加减：None
功效：None
用法用量：每服3分或5分，临卧酒调服下；若不用酒者，茶清调服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：忌多饮酒并一切热物。
附注：None
