序号：33180
名称：万灵丸
组成：黑牵牛1斤（取头末10两，生用），大腹子1斤（扁者，取7两末，生用，如尖者是槟榔），京三棱5两（炮），广木香5两（面裹煨干为度），雷丸5两（炮），蓬术（煨）2两。
出处：《医方类聚》卷一一三引《瑞竹堂方》。
主治：None
加减：None
功效：取虫宣积。
用法用量：每服4钱重，五更用沙糖水送下，温冷不妨。至天明去4-5利者，看取下是何虫积，以温白粥补之。
制备方法：上为细末，和匀，用好紫色皂角半斤，去皮弦，切碎，用水两大碗浸1宿，冬月两宿，捞去粗滓，铜、瓷器内熬数沸，白沫出为度，放冷，和药必须揉杵捣，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌生冷腥硬之物，孕妇勿服。
附注：万应丸（《准绳·类方》卷八）。
