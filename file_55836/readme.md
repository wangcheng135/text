序号：55851
名称：桑白皮汤
组成：桑根白皮（锉，炒）1两，青橘皮（去白，炒）半两，半夏（汤浸，洗去滑，姜汁制）半两，沉香1两，柴胡（去苗）1两，贝母（去心）1两，附子（炮裂，去皮脐）1两，干姜（炮）1两，白茯苓（去黑皮）1两，赤芍药1两，白芷1两，甘草（炙，锉）1两，白术1两，鳖甲（去裙襕，醋浸，炙）1两，细辛（去苗叶）1两，麻黄（去节）1两，大黄（煨）半两，木通半两，乌梅（炒，去核）半两，黄耆（锉，炒）半两，玄参半两，石斛（去根）半两，陈橘皮（去白，炒）半两，常山半两。
出处：《圣济总录》卷八十九。
主治：虚劳损伤，骨节痠痛，肌热咳嗽。
加减：None
功效：None
用法用量：每服3钱匕，水1盏半，同煎至1盏，去滓，温服，不拘时候。
制备方法：上（口父）咀，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
