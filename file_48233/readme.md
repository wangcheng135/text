序号：48248
名称：神保丹
组成：川芎1两，天南星（汤泡7次）1两，甘松1两，香白芷1两，藿香叶（洗去土）1两，香附子（去毛，炒）1两，牛膝（去芦，酒浸）1两，桔梗（炒）1两，防风（去芦）1两，茴香（淘去沙，炒）1两，羌活1两，藁本1两，麻黄（去节）1两，当归（去芦，酒浸）1两，草乌头1两半（并生，去皮尖），大川乌1两半（并生，去皮尖），甘草4两（生），荆芥穗2两，石膏半斤（生）。
出处：《魏氏家藏方》卷一。
主治：左瘫右痪，或一手顽痹，一足不仁，或半身不遂，口喎，喉肿，及脚上生疮，耳内虚鸣，眼肿。
加减：None
功效：None
用法用量：每丸分作4服，头风，薄荷酒嚼下；喉闭，生姜、薄荷酒送下；妇人血风久瘫，豆淋酒（用黑豆炒熟，以酒投之，去豆，只用酒）送下；伤骨，乳香酒送下；脚上生疮，木瓜酒送下；耳内虚鸣，煨猪肾酒送下；眼肿，菊花酒化下。小儿每丸分作8服。瘫风，先服顺气散，后服此药，瘫风3-5日者，服之甚效尤捷。
制备方法：上为细末，糯米糊为丸，每以3钱半重作1丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
