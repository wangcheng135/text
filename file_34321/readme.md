序号：34330
名称：万金丸
组成：木贼1两，何首乌1两，荆芥穗1两，防风（去芦头）1两，鸡冠花（焙）1两，芷壳（去瓤，麸炒）1两，五倍子1两，黄耆（锉，焙）1两，槐花（炒）1两，槐角1两（同为细末），猪蹄甲（用猪蹄向后小爪，不着地甲）1两，皂角子1两，皂角刺1两，麝香皮子1两，榼藤子1两，百药煎1两，刺猬皮1两（入瓷罐内盐泥固济了，候干，用炭火煅至青烟出，取出与前药末同研细，次入麝香2钱）。
出处：《杨氏家藏方》卷十三。
主治：一切痔漏，久不愈者。
加减：None
功效：None
用法用量：每服50丸，食前米饮送下，病久者当常服，即去根本。
制备方法：上药用酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
