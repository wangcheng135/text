序号：42489
名称：青黛散
组成：青黛（细研）1两，雄黄（细研）1两，朱砂（细研，水飞过）1两，石盐（细研）1两，白矾（烧令汁尽）1两，薰陆香（研入）1两，麝香（细研）半分，蚺蛇胆（研入）半分，细辛半分，黄连（去须）半分，青矾（烧令通赤）半分，黄矾（烧令通赤）半分，盐绿半分，黄柏（锉）半分，苦参（锉）半分，桂心半分，杏仁（汤浸，去皮尖双仁，麸炒微黄）半分，干姜（炮裂，锉）半分，藜芦（去芦头）半分，附子（炮裂，去皮脐）1分，莨菪子（水淘去浮者，水浸令芽出，焙干，炒令黑黄色）1分，熊胆（研入）1分，石胆（细研）1分，蛤蟆1枚（涂酥炙微黄）。
出处：《圣惠》卷八十六。
主治：小儿一切疳。腹肚胀满，手脚枯细，眼目口鼻生疮，身体壮热，痢下泔淀，日渐羸瘦，面无光泽。
加减：None
功效：None
用法用量：如疳在内，3岁每服半钱，以井花水1合调下，1岁1字，3岁以上，临时加之；若口内疳疮，以蒜1片研和少许散，每夜涂之，须臾，自然流引涎出；若鼻内有疮，用蒜如皂荚子大研和少许散，纳于鼻中；若外有疳疮，以猪脂和散涂之。立愈。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：None
