序号：19951
名称：夺命金丹
组成：天南星（为末，用腊月黄牛胆汁拌和，入于胆内，挂当风处阴干，百日可用）1两，天竺黄1两，天麻1两，防风（去芦）1两，白附子（炮）1两，白僵蚕（炒，去丝嘴）1两，朱砂（另研）1两，蝉蜕（去土）半两，麝香2钱（另研），天浆子（炒）21个，赤足蜈蚣1条（脊上开路，入麝香1钱于内，却用纸裹，焙干用），牛黄半钱，蝎梢49枚（炒），干蟾1枚（炙黄，去足）。
出处：《简易方》引黄医传秘方（见《医方类聚》卷二五八）。
主治：小儿急慢惊风，无药可理者。
加减：None
功效：None
用法用量：每服先下白末子（即《局方》青州白丸子研为末）半钱，薄荷汤送下，后用生姜自然汁磨化下1粒，不拘时候。
制备方法：上为细末，用东流水煮白面糊为丸，如鸡头子大，金箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
