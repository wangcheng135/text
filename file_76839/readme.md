序号：76855
名称：鲁公酿酒
组成：干姜5分，踯躅5分，桂心5分，甘草5分，芎5分，续断5分，细辛5分，附子5分，秦艽5分，天雄5分，石膏5分，紫菀5分，葛根4两，石龙芮4两，石斛4两，通草4两，石南4两，柏子仁4两，防风4两，巴戟天4两，山茱萸4两，牛膝8两，天门冬8两，乌头20枚，蜀椒半升。
出处：《千金》卷八。
主治：风偏枯半死，行劳得风，若鬼所击，四肢不遂，不能行步，不自带衣，挛躄，五缓六急；妇人带下，产乳中风，五劳七伤。
加减：None
功效：None
用法用量：先食服半合，每日2次，待米极消尽，乃去滓晒干，为末服。
制备方法：上(口父)咀，以水5升渍3宿，法曲1斤合渍，秫米2斗，合酿3宿，去滓，炊糯米1斗，醖3宿药成。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
