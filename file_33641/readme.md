名称：灵感膏
组成：大黄1两，生地1两，防风7钱，三棱1两，羌活8钱，白芷8钱，花粉7钱，蜈蚣10条，桃仁7钱（研末），香附7钱，厚朴7钱，槟榔7钱，黄柏8钱，大戟8钱，蓖麻子2两（研），蛇蜕5钱，杏仁7钱（研），皂角8钱，巴豆8钱（研），肉桂8钱，麻黄8钱，细辛7钱，黄连5钱，甘遂2两，木鳖子1两（研），莪术1两，川乌1两，枳实8钱，独活7钱，穿山甲7钱，全蝎7钱，当归1两5钱，草乌1两，元参7钱，五倍子7钱。
