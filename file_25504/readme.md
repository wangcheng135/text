序号：25512
名称：竹沥汤
组成：麻黄1两半，防风1两半，芎1两，防已1两，附子1两，人参1两，芍药1两，黄芩1两，甘草1两，桂心1两，生姜4两，石膏6两，杏仁40枚，竹沥1升，羚羊角2两，生葛汁5合。
出处：《千金》卷八。
主治：风痱。
加减：None
功效：None
用法用量：以水7升煮减半，加沥煮取2升5合，分3服取汗，间5日更服1剂，频与3剂。
制备方法：None
临床应用：None
各家论述：《医略六书》：人参补气以生阴血，附子扶阳以振生气，黄芩、石膏清热于内，防风、麻黄疏风于外，防已泻血分湿热，杏仁降胸中逆气，芎活血脉，芍药敛营阴，羚羊清肝火，甘草缓中州，桂心温血于营，姜汁行气于卫，竹沥、葛汁解燥热以滋液豁痰，俾痰化液充，则筋脉得养，何患瘫风之不愈哉。
用药禁忌：忌猪肉、冷水、海藻、菘菜、生葱。
附注：服讫，渐觉少损，仍进后方。本方方名，《张氏医通》引作“竹沥饮子”。
