序号：5120
名称：木鳖膏
组成：真麻油3斤，番木鳖147粒。
出处：《伤科汇纂》卷七引《顾氏家秘》。
主治：跌打损伤肿痛，一切疮疡、诸风。
加减：None
功效：None
用法用量：摊后加后掺头药。若伤损血积龌龊者，先贴无掺药膏1个，贴1-2时辰揭起，则伤口血迹被膏揭净，然后用掺药之膏贴之，倘无木鳖膏，即常膏药亦可用。
制备方法：上入锅内熬至番木鳖黑脆为度，熬时以柳枝频搅，将木鳖子捞起，再入铅粉炒黄色30两，徐徐投下，逐渐成膏。以缸盛井水，将膏倾入，置露处出火气1宿，捞起听用。
临床应用：None
各家论述：None
用药禁忌：None
附注：掺头药：山奈、北细辛、川乌、樟冰、肉桂、当门子、大茴、母丁香、乳香(去油)、没药(去油)、甘松、自然铜(煅)、半夏、大黄、荜茇、皂角、干姜、白芷、小茴香各五钱，阿魏三钱。上各忌见火，即自然铜亦须放倾银缸内燃，逐味另研，合匀密贮，勿泄香气，多少随用。如骨碎者，不可多用麝香，以其性热而散能耗髓也。
