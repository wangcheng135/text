序号：44092
名称：复方胎盘片
组成：胎盘粉1.6kg，麦芽100g，党参400g，橘皮100g，黄耆400g。
出处：《上海市药品标准》。
主治：神经衰弱，贫血，消化不良。体倦乏力，血虚眩晕，白细胞减少。
加减：None
功效：益精气，健脾胃。
用法用量：口服，每次4片，1日3次。
制备方法：将胎盘粉、橘皮、麦芽3味各为细粉，过100目筛，各取净粉，和匀。再将黄耆、党参2味水煎二次，每次4小时，药汁滤过，澄清，混合后浓缩成清膏。然后将白糊精120g、饴糖640g，与清膏混合成浆，加入上述混合细粉，制成颗粒，干燥，每100g干燥颗粒拌加饴糖5～8g，润滑剂1g，压制成片，片重0.24g，外包糖衣即得。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
