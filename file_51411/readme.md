序号：51426
名称：赵氏加味六子丸
组成：菟丝子（淘洗，酒蒸）1两3钱，川牛膝（去芦，酒蒸）1两3钱，麦门冬（去心，酒蒸）1两3钱，山茱萸（取肉）1两3钱，原蚕蛾1两3钱，五味子1两3钱，蛇床子（酒蒸）1两6钱，车前子（淘洗）1两7钱，大甘草（炙）1两，沙苑蒺藜（马乳浸，蒸）2两2钱，覆盆子2两2钱，破故纸2两3钱（淘洗，炒），肉苁蓉2两5钱（酒浸，去鳞）。
出处：《准绳·女科》卷四。
主治：None
加减：None
功效：种子。
用法用量：每服30丸或40丸，清盐汤送下，早、晚皆服。2-3月后必孕成矣。
制备方法：上药俱焙干，锉碎为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
