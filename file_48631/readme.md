序号：48646
名称：神授卫生汤
组成：羌活8分，防风6分，白芷6分，穿山甲（土炒，研）6分，沉香6分，红花6分，连翘6分，石决明（煅）6分，金银花1钱，皂角刺1钱，归尾1钱，甘草节1钱，花粉1钱，乳香5分，大黄（酒拌炒）2钱（脉虚便利者不用）。
出处：《外科正宗》卷一。
主治：痈疽发背，脑疽对口，丹瘤，瘰疬，恶毒疔疮，湿痰流注及一切疮症已成未成者。
加减：None
功效：宣热散风，行瘀活血，解毒消肿，疏通脏腑。
用法用量：水2碗，煎8分，病在上部，先服药，随后饮酒1杯；病在下部，先饮酒1杯，随后服药，以行药势。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：药性平和，功效甚速，诚外科首用方也。
