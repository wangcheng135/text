序号：41506
名称：金液丸
组成：京三棱（炮）1两，蓬莪术（炮）1两，白术1两，丁香皮（刮去粗皮）1两，牵牛子（麸炒）1两，青橘皮1两，陈橘皮（并汤浸，去白焙）1两，肉豆蔻（大者，去壳）1两，槟榔（炮）1两，干姜（炮）半两，丁香半两，硇砂（研）半两，巴豆半两（和皮秤，去皮，研如膏，纸压去油尽，以不污纸为度）。
出处：《圣济总录》卷七十一。
主治：脾积痞气，痰逆恶心，腹胁满闷，胸膈噎塞，不思饮食。
加减：None
功效：None
用法用量：每服5丸，食后米饮下。
制备方法：上为末，搅拌匀，用头醋煮稠面糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
