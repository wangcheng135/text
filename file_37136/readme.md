序号：37146
名称：定命保童丸
组成：白矾1两（烧令汁尽），干地龙（炙）1分，朱砂1分，麝香少许。
出处：《医方类聚》卷二五四引《保童秘要》。
主治：疳。
加减：None
功效：None
用法用量：疳蛔重者每服7丸，轻者5丸每日平明时先用□者1粒，于盏中以热水浸一炊久，用箸头研成□，将衽子点药于鼻中，候嚏喷三五遍，然后用□丸者，以熟水化破，空心服之。如头疳，即发坚□，皮肤干，鼻下赤，或□中生疮，此即每日先须□鼻，然后服药。如疳在腹内，泻痢无度，或作枣花食不消化，常无心绪，即初服药，日点鼻，后每日服药，不须点也，如常服，即每日1丸。
制备方法：上为细末，面糊为丸，如麻子大，小者如粟米。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
