序号：23413
名称：防风汤
组成：防风（去叉）1两，五加皮（锉）1两，附子（炮裂，去皮脐）1两，萆薢1两，薏苡仁1两，桂（去粗皮）1两，牛膝（酒浸，切、焙）1两半，独活（去芦头）1两半，赤茯苓（去黑皮）1两半，当归（切，焙）1两半，杜仲（去粗皮，炙）1两半，海桐皮1两半，木香半两，枳壳（去瓤，麸炒）半两，仙灵脾半两。
出处：《圣济总录》卷十一。
主治：风腲腿，肌肉虚满，四肢不收，骨节疼痛。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，加生姜1枣大（拍碎），同煎至1盏，去滓温服，不拘时候。
制备方法：如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
