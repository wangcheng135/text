序号：55662
名称：桃红散
组成：蟾酥少许，信石少许，蝉蜕3个（去足翘），蜈蚣头1个，斑蝥3个（去足翅），国丹5分，风化石灰1两（砂锅盛，瓦片盖，炭火煅2时久，取出）。
出处：《秘传外科方》。
主治：诸疔疮，虽凶证迭见，六脉俱绝，垂死者。
加减：None
功效：None
用法用量：指爪甲刮葱白内涎调药。先以禾叶针破疮口，令恶血出尽，别将蟾酥1粒如麦粒大，入疮内，却以葱白所调药敷疮口，莫敷在好肉上，用冷水浸湿纸20-30重，贴在药上，封固疮口。如清早封固，至晚觉疮口热，即去纸，水洗令净，用红玉散掺疮口。
制备方法：上为极细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
