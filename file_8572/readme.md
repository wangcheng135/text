序号：8572
名称：玄菟丹
组成：菟丝子（酒浸通软，乘湿研，焙干，别取末）10两，白茯苓3两，干莲肉3两，五味子（酒浸，别为末）7两。
出处：《三因》卷十。
主治：消渴、遗精、白浊,肾水枯竭，心火上炎，津液不生，消渴诸证;肾气虚损，目眩耳鸣，四肢倦怠，遗精尿血，心腹胀满，脚膝痠痿，股内湿痒，小便滑数，水道涩痛，时有遗沥等证;下焦虚而不能摄水，以致小便多而有降无升。
加减：None
功效：常服禁精，止白浊，延年。
用法用量：每服50丸，空心、食前米汤送下。
制备方法：上为末，别碾干山药末6两，将所浸酒余者，添酒煮糊，搜和得所，捣数千杵，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：须是戒酒，并火上炙馎之物。
附注：玄菟煎（《易简方》）、茯菟丹（《直指》卷十七）、茯菟丸（《丹溪心法》卷三）。
