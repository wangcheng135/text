序号：66666
名称：鹿茸丸
组成：洋参1两，鹿茸1两，熟地2两，大云1两5钱，当归2两，黄耆2两，枣仁8钱，淮药1两，于术3两，枸杞3两，巴戟2两，菟丝1两5钱，枣皮8钱，天雄2两，杜仲2两，茯苓1两，远志8钱，淮膝5钱，五味1两，菖蒲5钱，车前4钱，大枣1两，川姜6钱，泽泻4钱，朱砂2两，甘草1两。
出处：《全国中药成药处方集》（昆明方）。
主治：病后体虚，心脏衰弱，怔忡惊悸，遗精，妇人带下。
加减：None
功效：None
用法用量：每服1丸，幼童减半，早、晚用开水各服1次。
制备方法：上为末，炼蜜为丸，外装蜡壳封固。
临床应用：None
各家论述：None
用药禁忌：感冒及一切热症忌服；忌酸冷食物。
附注：None
