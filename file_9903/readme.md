序号：9903
名称：止血益母丸
组成：益母草（上截）8两，大蓟4两（阴干），香附3两（用童便制），丹参3两，条芩4两（去皮，酒炒），熟地黄8两（杵膏，忌铁），萸肉4两（去核），干山药4两（酒炒），白茯苓3两（去皮，蒸熟），丹皮3两（去骨，酒洗），泽泻3两（去净毛）。
出处：《履霜集》卷二。
主治：妇人失血，新起属实热者。
加减：None
功效：养血清火。
用法用量：病轻者，日用1丸，研末，或热黄酒送下，或蜜汤送下；有痰者，姜汤送下；病甚者，朝、夕各1碗，以愈为度；或丸如绿豆大，每服3钱亦可。
制备方法：炼蜜为丸，丸重3钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
