序号：47722
名称：神仙夺命丹
组成：南薄荷叶1两，天南星（汤泡透，切片，姜汁炒）5钱，姜蚕3钱，南羌活5钱，荆芥穗2钱，川椒（去目）1钱，辽细辛2钱，牙皂（刮去皮弦）8两，石脑油（真者）2两，硼砂1两。
出处：《鲁府禁方》卷一。
主治：中风，痰厥，气厥，牙关紧，不省人事。
加减：None
功效：None
用法用量：临用时剪方寸1块，以温浆水溶化盏内，用2苇筒吹入二鼻孔中，良久，吐痰涎即省，若吹之太重，或药水太热，致鼻出血勿得，即饮淡盐汤1-2口便止。
制备方法：上将前八药入瓷盆内，用好酸酱水4碗浸泡（春、秋5日，夏3日，冬7日），临熬时滤去滓，存净汁，入银锅或铜锅内，用桑柴火熬，以槐柳枝频搅；熬数10沸，方入石脑油、硼砂，再熬成膏，形如琥珀色，乘热摊于厚连四纸上，干收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
