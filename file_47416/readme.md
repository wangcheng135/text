序号：47431
名称：祛疟丹
组成：人参1两，槟榔1两，茯苓（去皮）半两，半夏（汤泡7次）半两，草果半两，常山3两（酒浸2宿），良姜7钱，川芎7钱，香白芷7钱。
出处：《医方类聚》卷一二三引《经验秘方》。
主治：烟瘴寒疟。
加减：None
功效：None
用法用量：每服50丸，温酒送下；如不饮酒者，皂角子2-7枚，捶碎熬汤，放令温送下。白日发，晚夕1服，半夜1服，平明1服；夜发，早1服，午时1服，晚夕1服，计150丸。
制备方法：上为细末，面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌荤腥5-7日，切忌鸡肉、湿面，一月方可食；服药之后，1日不许食辛辣之物、湿热、鱼肉。
附注：None
