名称：紫金锭
组成：羌活（去芦）5钱，白附子（炮）5钱，防风（去芦）5钱，天竺黄5钱，西牛黄7分，胆南星2两，大黄（煨）2两，枳实（麸炒）2两，黄连（姜汁炒）2两，僵蚕（炒去丝）2两，天麻（煨）2两，白术（土炒）6钱，青礞石（煅）6钱，雄黄2钱，川芎2钱，茯神（去木）1两，全蝎（去毒）1两半，冰片5分，麝香5分，辰砂2两（水飞）。
出处：《诚书》卷八。
主治：急慢惊风四证八候。

制备方法：上为末，甘草煎汁，打糊为锭，焙干，金箔为衣。
