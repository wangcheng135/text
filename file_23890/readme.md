序号：23898
名称：安神止痛汤
组成：琥珀6g，莲子24g，党参15g，白芍15g，生地9g，淮山15g，黄耆9g，酸枣仁9g，三七3g，醋元胡6g，制乳香4.5g，制没药4.5g，朱茯神9g，远志6g，甘草3g，双钩藤9g。
出处：《林如高骨伤验方歌诀方解》。
主治：重伤痛极，夜寐不安。
加减：None
功效：宁心安神，益气镇痛。
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：本方用琥珀、远志、酸枣仁、茯神安神宁心；双钩藤平肝止痉；“心藏神”，心主血脉，神赖气血濡养，党参、黄耆、淮山、生地、白芍、甘草补气补血；莲子养心补肾，益十二经脉血气；佐以乳香、没药、三七、玄胡活血化瘀，消肿止痛，遂成宁心安神、益气镇痛之功。
用药禁忌：None
附注：None
