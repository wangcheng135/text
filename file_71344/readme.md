序号：71360
名称：痢疾奇妙丹
组成：槟榔1两6钱，川黄连2两4钱，车前子2两，地榆1两6钱，白芍1两6钱，白茯苓1两6钱，全当归1两6钱，甘草8钱，黄柏2两4钱，莱菔子1两6钱，猪苓1两6钱，陈皮1两6钱，枳壳1两6钱，广木香4两，南山楂3两2钱，槐花1两6钱，吴茱萸2两，诃子2两，黄芩2两，川厚朴2两4钱，升麻1两6钱。
出处：《全国中药成药处方集》（禹县方）。
主治：红白痢疾。小腹疼痛，里急后重，下坠脱肛。
加减：None
功效：None
用法用量：每服1包（24粒），每日2次。满10岁者服半包。
制备方法：共研细面，水迭为丸，如豌豆大。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
