序号：58281
名称：真传腽肭脐丸
组成：前服：当归（酒洗）5钱，熟地（砂仁、沉香炒）5钱，生地（姜、酒炒）5钱，肉苁蓉（酒浸）5钱，栀子（炒）5钱，黄芩3钱，黄柏（盐、酒炒）2钱，木香1两3钱，沉香1两3钱，槟榔4钱，腽肭脐4钱，海牛1个，海马1对，蛤蚧1对（酥炙）后服：枸杞子5钱，五味子5钱，肉苁蓉（酒洗）5钱，当归（酒洗）5钱，生地黄（酒洗）5钱，熟地黄（酒蒸）5钱，牛膝（酒浸）1两，白茯苓1两，沉香1两，木香1两，天冬（去心）5钱，麦冬（去心）5钱，甘草3钱，枳壳5钱，海马1对（酥炙），海牛1个（酥炙），蛤蚧1对（酥炙），腽肭脐4两（酥炙）。
出处：《济阳纲目》卷六十四。
主治：None
加减：None
功效：补精益血，壮元阳，暖丹田，滋容颜，利腰膝。
用法用量：每服60-70丸，空心温酒或盐汤下。
制备方法：上2料，俱为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
