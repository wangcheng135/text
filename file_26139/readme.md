序号：26147
名称：朱砂膏
组成：朱砂（好者，别研）半两，硼砂（通飞者，研）半两，甘草（炙）1分，牙消1两半（煅过，少分生，别研），麝香（研）1字，龙脑（研）1字。
出处：《幼幼新书》卷八引《惠眼观证》。
主治：襁褓内牙儿等，因惊风后，余涎响。及初生下，患鹅口重舌腭，心热夜啼，发病搐搦，项背强直，痰涎壅并目带上翻，进退无时。
加减：None
功效：None
用法用量：每服皂子大，更加减吃。若更滚涎，用鸡子清化下；常服甘草汤。
制备方法：上先研朱砂400-500转，又别研硼砂同前数，入诸药再研，出，方研脑子，再入诸药末，滚合滴水，研成膏，摊一宿以油纸单内。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
