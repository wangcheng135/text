序号：920
名称：六神开瞽散
组成：五烹1两，入冰片7分，龙砂1两，入冰片1钱，虎液1两，入冰片1钱3分。
出处：《眼科阐微》卷四。
主治：目睛赤肿痛痒，云膜胬肉，赤白翳障。
加减：None
功效：None
用法用量：饭后，点大眼角，不可近黑珠，每日点10余次。
制备方法：凡目赤肿痛疴痒，俱系内癥、重者，虎液为主，用7分；龙砂为辅，用3分；五烹为佐，用2分。如内症轻者，五烹为主，用5分；虎液为辅，用3分；龙砂为佐，用2分。凡云膜胬肉、赤白翳障，俱系外障、重者，龙砂为主，用7分；虎液为辅，用3分；五烹为佐，用2分。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中五烹、龙砂、虎液，各详专条。
