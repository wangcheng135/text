序号：22263
名称：阳和膏
组成：官桂1两，甘松1两，山柰1两，丁香5钱，乳香5钱，没药5钱，上玉桂5钱，牛蒡子5钱。
出处：《经验各种秘方辑要》。
主治：痈疽、发背、流痰，一切无名肿毒，及风热肿胀。
加减：None
功效：None
用法用量：凡疮初起未成时贴之皆可消散，但须连四围根脚贴进，不可但贴头上，反致不效；即已溃之疽贴此膏亦可收束，不致蔓延；方内玉桂、官桂并用者，因玉桂价昂，故兼用之，非重味也。
制备方法：上药内乳香须熬烊去油，余皆晒干生研，各为极细末，用沪上姜衍泽堂太乙膏药肉烘烊，将药末拌匀摊膏，其摊膏纸用棉料油纸并白纸裱褙双层，大者用红布摊之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
