序号：57434
名称：涤饮丸
组成：黑丑3两，枳实1两半（炒），朴消3两，白矾3两，牙皂3两，枳壳1两半（炒）。
出处：《医略六书》卷十九。
主治：留饮。大腹胀满，便闭，脉沉紧数者。
加减：None
功效：逐饮涤结。
用法用量：每服2-3钱，白汤送下。以胀退、大腹和软为度。
制备方法：上为末，萝卜汁为丸。
临床应用：None
各家论述：饮留于中，气结不化，故大腹胀满，大便不通焉。黑丑逐饮，枳实破结，朴消荡热逐留饮，枳壳泻滞通便闭，白矾、牙皂消溶水湿以通窍也。萝卜汁为丸，白汤下，使饮化气行，则肠胃肃清，而大便无不通，胀满无不退矣。此逐饮涤结之剂，为留饮气结胀闭之专方。
用药禁忌：None
附注：None
