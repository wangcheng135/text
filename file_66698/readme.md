序号：66714
名称：鹿茸丸
组成：鹿茸2两（去毛，涂酥炙微黄），磁石2两（烧，醋淬7遍，细研，用水飞过），白茯苓1两，熟干地黄1两，肉苁蓉（酒浸1宿，刮去皴皮，炙干）1两，菟丝子（酒浸3日，晒干，别捣为末）1两，人参（去芦头）1两，附子（炮裂，去皮脐）1两，薯蓣1两，远志（去心）1两，桂心1两，牛膝（去苗）1两，杜仲（去粗皮，炙微黄，锉）1两，巴戟1两，续断1两，五味子1两，山茱萸1两，泽泻1两，补骨脂1两，蛇床子1两。
出处：《圣惠》卷九十八。
主治：None
加减：None
功效：暖脏腑，壮腰膝，补下元，养精气，美颜容，长肌肉，补诸虚损。
用法用量：每服30丸，空心以温酒送下。
制备方法：上为末，入磁石研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
