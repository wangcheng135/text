序号：25905
名称：扫雾丹
组成：丹头1钱（兔脑炉甘石1两，要上等雪白、轻者方佳，火精石1两，捶为细末，以水合之如泥，作1饼，如无火精石，用混元球煅之，或用煅无灵丹药汁淬之。将甘石打作小块，包入精石内，外再用黄泥包之，待干，入百眼炉内煅之，红透取出，待冷听用，去泥，取甘石入乳缽内，擂极细，用水飞过3-5次，有微粗再飞过，以不刺牙为妙，晒干为丹头），冰片（必上好4-6者）2分5厘，真麝香1分5厘，熊胆3分5厘，蕤仁3分（去油尽）。
出处：《眼科秘诀》卷一。
主治：一切翳膜障眼。
加减：None
功效：None
用法用量：日点3次，内服揭障丹，外敷冲翳散，再用本方点70-80次。极重者，半月有验矣。
制备方法：上擂万遍，以瓷罐贮之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
