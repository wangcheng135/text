序号：44806
名称：急痧至宝丹
组成：蟾酥3钱（活蛤蟆，取下晒干，临用切片，烧酒化开），西黄3分（研），茅术4钱（土炒焦），丁香2钱（研细），朱砂1钱5分（水飞净），木香2钱（研细），雄黄3钱（水飞净），沉香2钱（研细），麝香1钱（拣净）。
出处：《良方集腋》卷上。
主治：霍乱吐泻，腹痛昏瞶；及一切痧气、暑气、瘴气、途行触秽，中暑热，绞肠痧。
加减：None
功效：止痛。
用法用量：每服3丸，轻者1丸，重者5丸，泉水送下；或口内含化，津液咽下。
制备方法：上件药先期各为极细末，都拌匀，同蟾酥加糯米粽尖5个，捣千余下，为丸，如椒子大，晒干，盛于瓷盖碗内；再用朱砂1钱5分，烧酒调涂碗内，盖好，摇一二千下，则光亮，收贮瓷瓶内。
临床应用：None
各家论述：None
用药禁忌：孕妇少服。服药后停烟、茶、酒、饭两时。
附注：None
