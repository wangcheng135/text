序号：18722
名称：人参固本酒
组成：人参2两，天门冬（去心）4两，麦门冬（去心）4两，怀生地4两，怀熟地4两，枸杞子4两，虎胫骨2两（酥炙黄脆），龟版2两（酥炙黄脆），何首乌4两（大者佳，竹刀去皮，切片，米泔水浸1宿，用黑豆3升浸软，1层豆，1层药，密盖蒸熟，9蒸9晒），当归1两。
出处：《墨宝斋集验方》卷上。
主治：None
加减：None
功效：固精健骨，补精髓。
用法用量：用好酒30斤，盛入2罐内，将药分1半，用好绢小袋盛药，吊入罐内，将面封固，桑柴文武火煮2炷香为度，埋土内。一七后，空心任服。
