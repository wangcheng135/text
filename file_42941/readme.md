序号：42956
名称：保真汤
组成：川芎1钱，当归2钱，生地2钱，白芍2钱，麦冬1钱，天冬1钱5分，川贝5分，茯苓5分，桔梗8分，五味10粒，骨皮1钱，炙甘草4分。
出处：《胎产秘书》卷下。
主治：产后热蒸成痨症。此由嗜欲无节，起居不时，以致真阴耗竭，虚火上炎，或蒸而热，或往来寒热，似疟非疟，或咳血咯血，自汗盗汗，或心神恍惚，梦与鬼交，或经水闭塞，身渐羸瘦。
加减：虚，加人参1钱，黄耆6分；胃弱，加茯苓、山药各2钱，砂仁2-3粒。
功效：清热止咳，润肺泻火，滋补真阴，以复其元。
用法用量：上加大枣2枚，水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：患此者，必须寡欲内养，方能有效。
附注：None
