序号：26818
名称：延龄广嗣仙方
组成：怀生地（酒制）3两，何首乌（酒煮）3两，旱莲草3两，鹿衔草（真者绝少，用仙灵脾代之）3两，干山药（乳拌）1两，白茯苓（乳拌）1两，当归身（酒炒）1两，真青盐1两，石菖蒲5钱，菟丝子5钱，肉苁蓉（酒浸，去膜）5钱，补骨脂5钱，五加皮5钱，骨碎补5钱，淮牛膝5钱，白甘菊5钱，原杜仲（酒炒断丝）5钱，枸杞子5钱，蛇床子5钱，槐角子5钱，金樱子5钱，覆盆子5钱，川黄连5钱，建泽泻5钱。
出处：《纲目拾遗》卷八。
主治：肾虚目暗，上盛下虚者。
加减：None
功效：令人须发再黑，齿落更生，耳目聪明，手足便利，壮阳补肾，固本还元，多育子息，多增年寿。
用法用量：每服4钱，清晨滚汤送下。
制备方法：上除去青盐，锅内煎汁至半，沥滓；再将滓煎过半，沥清，冲和煎浓，人马料豆3升7合，女贞子1升7合，煮数10滚；将青盐研细，倾入同煎，以汁尽为度，取豆晒干，收贮瓷瓶。
临床应用：None
各家论述：None
用药禁忌：None
附注：延龄广嗣丸（《中国医学大辞典》）。上二十四味，俱合二十四气。
