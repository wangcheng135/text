序号：46446
名称：济阴丸
组成：香附米4两（1分醋浸，1分米泔浸，1分酒浸，1分童便浸，各浸3日，焙干为末），益母草2两（忌铁器），艾叶1两（醋煮），阿胶2两（蛤粉炒），熟地黄2两（酒洗过，姜汁炒），川芎1两，当归1两5钱（酒洗），白芍药1两3钱（盐酒炒），陈皮1两（去皮），白术1两5钱（土炒），半夏（汤泡，姜汁浸，香油炒）1两，白茯1两（去皮），甘草（炙）3钱，条芩1两（炒焦），丹皮1两（酒洗），吴萸5钱（汤泡），玄胡索4钱，小茴香（盐酒炒）5钱，没药5钱，续断1两（酒洗），麦冬1两（去心）。
出处：《古今医鉴》卷十一。
主治：None
加减：None
功效：顺气养血，调经脉，除白带，益子宫，育胎孕。
用法用量：每服100丸，空心米汤送下；温酒白水亦可。
制备方法：上为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：种子济阴丹（《回春》卷六）、种子济阴丸（《医学正印》卷下）。
