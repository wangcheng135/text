序号：52428
名称：香甲汤
组成：沉香（锉）半两，青木香半两，人参半两，白茯苓（去黑皮）半两，柴胡（去苗）半两，槟榔（锉）半两，桂（去粗皮）半两，黄耆（锉）半两，赤芍药半两，山芋半两，甘草（炙，锉）半两，干姜（炮）1分，熟干地黄（焙）1两，厚朴（去粗皮，生姜汁炙，锉）1两，白术1两，鳖甲（去裙襕，以童便浸，炙）1两。
出处：《圣济总录》卷八十七。
主治：气劳。不思饮食，身体疼痛，胸膈妨闷，四肢烦热。
加减：None
功效：None
用法用量：每服3钱匕，以水1盏，加生姜1枣大（拍碎），大枣2枚（去核），煎至7分，去滓，食后良久温服，每日3次。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：香甲煮散（原书卷八十九）。
