序号：70558
名称：紫苏子汤
组成：紫苏子1两，大腹皮半两，草果仁半两，半夏（汤泡7次）半两，厚朴（去皮，姜制，炒）半两，木香（不见火）半两，橘红半两，木通半两，白术半两，枳实（去瓤，麸炒）半两，人参半两，甘草（炙）半两。
出处：《医方类聚》卷一三○引《济生》。
主治：忧思过度，邪伤脾肺，心腹膨胀，喘促胸满，肠鸣气走，漉漉有声，大小便不利，脉虚紧而涩。
加减：None
功效：None
用法用量：上(口父)咀。每服4钱，水1盏半，加生姜5片，大枣2枚，煎至7分，去滓温服，不拘时候。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：紫苏饮子（《赤水玄珠》卷五）。
