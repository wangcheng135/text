序号：718
名称：六一顺气汤
组成：大黄、枳实、黄耆、厚朴、甘草、柴胡、芒消、芍药。
出处：《伤寒六书》卷三。
主治：伤寒表里有实热，大便秘结，口舌干燥，神昏谵语，胸腹硬满而痛；或阳明失下而呃逆；或下痢赤白，表里俱实者。
加减：结胸证，心下硬痛，手不可近，燥渴谵语，大便实者，去甘草，加甘遂、桔梗；凡伤寒过经，及老弱并血气两虚之人，或妇人产后有下证，或有下后不解，或有表证尚未除，而里证又急，不得不下者，去芒消。
功效：None
用法用量：上先将水2钟，滚3沸后入药，煎至8分，临服时入铁锈水3匙调服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
