序号：37651
名称：三石丸
组成：凝水石（火煅通赤，研为细末）3两半，石膏（煅令通赤，研为细末）1分，阳起石（煅令通赤，研为细末）1钱1字，天南星（炮裂，去皮，为末）半钱，天雄（炮裂，去皮脐，为末）半钱，草乌头（炮裂，去皮脐，为末）半钱，白附子（炮，为末）半钱，干蝎（去土，炒，为末）半钱，白僵蚕（炒，为末）半钱，蝉壳（去土，焙干，为末）半钱，龙脑（研），麝香（研）1钱，海蛤（烧通赤，细研）2两。
出处：《圣济总录》卷九。
主治：中风半身不随，病苦悲伤，恶闻人声，少气多汗，偏臂不举。
加减：None
功效：None
用法用量：每服1丸，细嚼，荆芥、薄荷汤送下：茶、酒亦得，不拘时候。
制备方法：上药再同研令细，入白面1钱半令匀，炼蜜为丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
