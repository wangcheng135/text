序号：73946
名称：溯源丹
组成：当归（酒洗）2两，熟地黄（酒蒸）2两，蕲艾（醋炒）2两，香附（醋浸，炒）3两，川芎（米泔制）1两2钱，人参1两2钱，白芍（酒炒）6钱，阿胶（蛤粉炒）6钱，白术（去芦）6钱，茅根6钱，椿根皮（酒炒）1两，黄柏（酒炒）1两，地榆7钱，白茯苓（去皮）8钱，白石脂7钱。
出处：《寿世保元》卷七。
主治：妇人赤白带下。上热下寒，口出恶气，或咽干，或牙痛，或耳鸣，或遍身流注疼痛，发热憎寒，或口吐酸水，或心腹气痛，或下五色腥臭。
加减：None
功效：None
用法用量：每服50-60丸，空心米汤送下。
制备方法：上为细末，米醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
