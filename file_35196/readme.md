序号：35206
名称：乳香没药丸
组成：抚芎180两，踯躅花（炒）72两，木鳖仁72两，白胶香（拣净）72两，藿香（拣，炒）72两，白僵蚕（洗，焙）72两，五灵脂（拣）72两，白芷（拣）72两，当归72两，地龙144两，何首乌244两，威灵仙（洗）222两，草乌头（炒）648两。
出处：《局方》卷一（绍兴续添方）。
主治：男子妇人一切风气，筋骨疼痛，手足麻痹，半身不遂，暗风头旋，偏正头风，小中急风，手足疼痛，牙关紧急，四肢软弱。肾脏风毒，上攻头面，下注腰脚生疮，遍体疼痠。
加减：None
功效：通经络，活血脉。
用法用量：每服5丸，不可多服，食后用薄荷茶送下；温酒亦得。
制备方法：上为末，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：有孕妇人不可服。
附注：通经活络丹（《普济方》卷一一五）。本方名乳香没药丸，但方中无乳香、没药，疑脱。
