序号：41445
名称：金珠化痰丸
组成：皂荚仁（炒）1两，天竺黄1两，白矾（光明者，放石铁器内熬汁尽，放冷，研）1两，铅白霜（细研）1两，半夏（汤洗7次，用生姜2两洗，刮去皮，同捣细作饼子，微炙黄色）4两，生白龙脑（细研）半两，辰砂（研，飞）2两，金箔（为衣）20片。
出处：《局方》卷四。
主治：头痛眩运，心忪恍惚，胸膈烦闷，涕唾稠粘，痰实咳嗽，咽喉不利。
加减：None
功效：清痰热，安神志，除头痛。
用法用量：每服10-15丸，生姜汤送下，食后、临卧服。
制备方法：上以半夏、皂荚子仁为末，与诸药同拌研匀，生姜汁煮曲糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
