序号：48099
名称：神妙膏
组成：乳香半两，没药半两，头发半两，大黄半两，肉桂半两，当归半两，玄参半两，续断半两，莪术半两，生地黄半两，赤芍药半两，白芷半两，射干半两，巴豆半两，明矾半两，黄芩半两，柳枝半两，香油1斤，黄丹8两，麝香1钱。
出处：《普济方》卷三一五。
主治：诸般疮疖痈疽，攧伤损及折伤。
加减：None
功效：None
用法用量：每用油纸安刀上，摊以药，量大小贴患处。治杖疮，宜中间贴，此膏药用大黄、黄柏皮、黄芩3味焙干为末，鸡子清调涂四边，用皮纸条封，1日1次，换膏药，第3日葱、椒、盐汤熏洗疮。内服乌药顺气散。
制备方法：上锉，如豆大，油浸1宿，煎柳枝搅令色黑，滤去滓，油再入铫，微冷下丹煎，不住手搅，以黑色、滴水中不散、不粘手为度，下乳、没、麝香搅匀，取出。
临床应用：None
各家论述：None
用药禁忌：忌醋、面、肉。
附注：None
