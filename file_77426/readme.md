序号：77442
名称：雄黄暖膏药
组成：黄丹48两，麻油5斤，猪脂2斤，松脂1斤，羊脂10两，蜡10两，野驼脂10两，当归2两，乌蛇2两，生干地黄2两，连翘花1两，续断2两，白芷1两，露蜂房1两，川乌头1两（去皮脐），细辛1两，棘针1两，芎1两，羌活1两，人粪1两（干者，烧灰），紫草1两，虎胫骨1两，鲮鲤甲1两，猬皮1两，莨菪子1两，吴茱萸1两，白蔹3分，紫葛3分，玄参3分，桑木耳3分，木通3分（锉），杏仁3分（汤浸去皮尖），青帛7尺，绯帛7尺（烧令烟尽），白术3分，葱和根3-7茎，槐树枝4两，杨柳枝4两，防风3分（去芦头），桑根白皮3分，赤芍药3分，香附子3分。
出处：《圣惠》卷六十七。
主治：一切伤折。
加减：None
功效：接骨止痛。
用法用量：有患者，于绢帛上微火摊，贴于折损处，1日1度换之。
制备方法：以上药，先将油猪脂、羊脂、野驼脂于锅内煎为油，入柳枝，槐枝、棘针、葱、紫草、露蜂房，先于脂油内，以慢火煎半日，滤去滓，其余诸药，细锉入于熟油内，慢火煎半日。次入松脂蜡，更煎半日，滤去滓，净拭锅内，细罗黄丹，炒令紫色，热下药汁中。以柳枝搅不令住。候色变紫成膏，住火，次入雄黄3两（细研），丁香3两，乳香4两，沉香3两，木香3两，桂心3两，麒麟血3两，附子3两（去皮脐）。以上捣罗为末，入膏中调令匀，用瓷盒中盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：雄黄暖膏（《普济方》卷三一五）。
