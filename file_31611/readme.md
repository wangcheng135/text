序号：31620
名称：束胎丸
组成：炒黄芩复1两，春、秋7钱半，冬半两，白术1两（不见火），茯苓7钱半（不见火），陈皮3两（忌火）。
出处：《丹溪心法》卷五。
主治：妊娠7-8个月，恐胎气展大难产。
加减：None
功效：妊娠八月束胎。扶助母气，紧束儿胎。
用法用量：口服。
制备方法：上为末，粥为丸。
临床应用：None
各家论述：《医方考》：凡患产难者，多由内热灼其胞液，以致临产之际，干涩而难；或脾气怯弱，不能运化精微，而令胞液不足，亦产难之道也。故用白术、茯苓益其脾土而培万物之母；用黄芩清其胎热，泻火而存胞液；用陈皮者，取其辛利，能流动中气，化其肥甘，使胎气不滞，儿身勿肥耳。此束胎之义也。
用药禁忌：None
附注：八月束胎丸（《女科指掌》卷四）。本方方名，《准绳·女科》引作“缩胎丸”。《玉机微义》本方用法：上为末，粥为丸，如梧桐子大。每服30-40丸，空心、白汤送下。
