序号：78514
名称：摩顶膏
组成：生油2升，黄牛酥3两，莲子草汁1升，淡竹叶1握，大青1两半，葳蕤1两半，曾青1两（细研），石长生1两半，吴蓝1两，槐子1两半，川朴消1两半，青盐2两，栀子仁1两半。
出处：《圣惠》卷三十二。
主治：肾虚眼暗，内障，花翳，赤眼，流泪，睛痛，脱发，偏正头风。
加减：None
功效：生发，退热毒风。
用法用量：每以铁匙取少许，夜间临卧时涂顶上，细细以匙摩，摩令消散入发孔中，顿觉清凉，轻者不过5-6次，重者用膏半剂即愈。摩膏之法：每隔三夜摩1次，并日恐药驱风毒太急，乍有触动。摩膏后2-3日便能生发，风毒自散也。合药取莲子草汁，须是8-9月采之，其汁方浓有力，余时不堪也。
制备方法：上锉细，绵裹于铛中，先下油、酥及莲子草汁，然后下诸药，以文火煎半日，即以武火煎之，候莲子草汁尽其膏即成，去滓，更细澄滤过，油瓷瓶盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：摩顶青莲膏（《圣济总录》卷一一一）。
