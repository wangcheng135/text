序号：60787
名称：通淋散
组成：瞿麦穗3两，赤茯苓1两半，条黄芩1两半，白芍药1两半，麦门冬3两（去心），生甘草1两半，桑白皮1两半，车前子3两，冬葵子3两。
出处：《医略六书》卷二十八。
主治：子淋，脉滑数者。
加减：None
功效：None
用法用量：每服3钱，水煎去滓，温服。
制备方法：上为散。
临床应用：None
各家论述：妊娠湿热，渍于胞门，脾气不得施化而溺窍闭塞，故小便涩痛，淋沥不已，谓之子淋，胎孕何以能安？瞿麦降心热，以通淋闭；条芩清肺热，以安胎元；白芍蔹阴护胎，最滋阴血；麦冬润肺清心，得振水源；赤苓利营渗水道；生草泻火，缓涩痛；桑白皮肃清肺金；冬葵子滑利溺窍；车前子清降以利小水也。为散水煎，使湿热并解，则气化有权，而胞门清肃，小便快利，何淋沥涩痛之不已者？胎孕无不安吴。
用药禁忌：None
附注：None
