序号：76289
名称：紫霜丸
组成：代赭石（火煅，米醋淬不拘遍数，以手拈得碎为度，研细，水飞）半两，牛黄2钱（研），朱砂1分（研），麝香1钱（研），鳖甲1分（醋炙），巴豆1分（去皮心膜，出油尽），枳壳（去瓤，麸炒）2钱，当归（去芦并土，炙）2钱，甘草（炙）3分，木香1分，生犀末2钱，大黄3分。
出处：《卫生总微》卷十二。
主治：小儿诸疳发热，肚大脚细，发穗面黄，宿滞不消，或作寒热，腹内疼痛，经年瘦弱，及中恶等病。
加减：None
功效：None
用法用量：每服3-5丸，米饮送下，不拘时候。
制备方法：上为细末，炼蜜为丸，如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
