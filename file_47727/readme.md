序号：47742
名称：神仙妙应丸
组成：槟榔2两，黑白牵牛2两，大黄1两，使君子半两，芜荑半两，雷丸半两，鹤虱半两，干漆5钱半（去烟）。
出处：《普济方》卷三八○。
主治：疳脾癥瘕，气积成块，或如小黄瓜，横担腹肚，胁左右青筋明现，腹鼓急，手足瘦小，形貌焦枯黄黑，发直口干，虫积。
加减：None
功效：None
用法用量：随儿大小，每服100-200丸，五更葱白熬汤送下，枣儿压之。次早大便见其得病根源之物。年10岁服2-3钱。
制备方法：上为细末，用皂角4两，去皮弦子，切碎，热水浸泡，搓揉，浓水滤过，和面为丸，如栗子大，或粟豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
