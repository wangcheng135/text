序号：43968
名称：厚朴散
组成：厚朴2两（去粗皮，涂生姜汁，炙令香熟），丁香半两，木香半两，附子1两（炮裂，去皮脐），白术1两，当归1两（锉，微炒），人参1两（去芦头），诃黎勒1两（煨，用皮），干姜3分（炮裂，锉），白茯苓1两，桂心1两，甘草半两（炙微赤，锉），陈橘皮1两（汤浸，去白瓤，焙）。
出处：《圣惠》卷五。
主治：脾气虚，腹胁胀满，吃食难消，面色萎黄，四肢少力。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，大枣3个，煎至6分，去滓，食前稍热服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻、湿面、黏滑。
附注：厚朴汤（《鸡峰》卷十二）。本方原名厚朴丸，与剂型不符，据《医方类聚》改。
