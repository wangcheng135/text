名称：清肝膏
组成：鳖甲1个（用小磨麻油3斤，浸熬听用），柴胡4两，黄连3两，龙胆草3两，元参2两，生地2两，川芎2两，当归2两，白芍2两，郁金2两，丹皮2两，地骨皮2两，羌活2两，防风2两，胆南星2两，薄荷1两，黄芩1两，麦冬1两，知母1两，贝母1两，黄柏1两，荆芥穗1两，天麻1两，秦艽1两，蒲黄1两，枳壳1两，连翘1两，半夏1两，花粉1两，黑山栀1两，香附1两，赤芍1两，前胡1两，橘红1两，青皮1两，瓜萎仁1两，桃仁1两，胡黄连1两，延胡1两，灵脂1两（炒），莪术1两（煨），三棱1两（煨），甘遂1两，大戟1两，红花1两，茜草1两（即五爪龙），牛膝1两，续断1两，车前子1两，木通1两，皂角1两，细辛1两，蓖麻仁1两，木鳖仁1两，大黄1两，芒消1两，羚羊角1两，犀角1两，山甲1两，全蝎1两，牡蛎1两，忍冬藤1两，甘草1两，石决明1两，吴萸5钱，官桂5钱，蝉蜕5钱，生姜2两，葱白2两，大蒜头2两，韭白4两，槐枝8两，柳枝8两，桑枝8两，冬青枝8两，枸杞根8两，凤仙4两（全株），益母草4两，白菊花4两，干桑叶4两，蓉叶4两，侧柏叶2两，菖蒲1两，木瓜1两，花椒5钱，白芥子5钱，乌梅5钱。
出处：《理瀹》。
主治：肝经血虚有怒火，或头晕头痛，眼花目赤，耳鸣耳聋，耳前后痛，面青口酸，寒热往来，多惊不睡，善怒，吐血，胸中痞塞，胁肋乳旁痛，大腹作痛，少腹作痛，阴肿阴疼，小儿发搐，肝疳。外症颈上生核。
加减：None
功效：None
用法用量：量部位大小摊贴。头眼病贴两太阳，耳病夹耳门贴。内症上贴胸口，并两胁、背心（肝俞）、脐上、脐下，余贴患处。加锭子，醋磨敷。
制备方法：以上两料共用油24斤分熬，丹收。再入煅青礞石4两、明雄黄、漂青黛各2两，芦荟、青木香各1两，牛胶4两（酒蒸化），俟丹收后，搅至温温，以一滴试之不爆，方取下。再搅千余遍，令匀，愈多愈妙，勿炒珠。
临床应用：None
