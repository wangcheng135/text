序号：11672
名称：牛黄丸
组成：牛黄1分（细研），川大黄3分（锉碎，微炒），雄黄1分（细研），附子1分（炮裂；去皮脐），珍珠1分（细研），甘草1分（炙微赤，锉），细辛1分，人参1分（去芦头），朱砂1分（细研），鬼臼1分（去须），莽草1分（微炙），川乌头1分（炮裂，去皮脐），麝香半两（细研），川椒半两（去目及闭口者，微炒去汗），紫菀半两（洗去苗土），巴豆20枚（去皮心，研，纸裹压去油），鬼箭羽半两，赤茯苓半两，桂心半两，干姜3分（炮裂，锉），地胆5枚（糯米拌，炒令黄色，去翅足），野葛1分，芫青7枚（糯米拌，炒令黄色，去翅足），蜥蜴1枚（微炙），樗鸡半两（微炒）。
出处：《圣惠》卷五十六。
主治：诸尸，及中恶疰忤不恻之病。
加减：None
功效：None
用法用量：每服3丸，以温酒送下，不拘时候。
制备方法：上为末，入研了药及巴豆，都研令匀，炼蜜为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
