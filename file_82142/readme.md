序号：82158
名称：蝎附散
组成：附子（炮，去皮脐）3钱，川乌（炮，去皮尖）3钱，麻黄（去节）3钱，僵蚕（炒）3钱，南星3钱，防风3钱，雄黄1钱半，朱砂1钱半，全蝎1钱半，白芷半两，藁本半两。
出处：《妇人良方》卷四。
主治：一切风邪头痛，夹脑风气，痰涎壅盛，呕逆恶心，口吐清水，暗风眩晕，眼见黑花，牙关紧急，口眼斜，面目瞤动，头项拘急，肩背引痛，耳痒目昏，四肢麻木；及洗头浴出暴感风邪，头目昏痛，两太阳穴痛，远年头风，乍愈乍发，服他药无效者。
加减：None
功效：None
用法用量：每服半钱，葱茶调下，食后服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：孕妇不可服。
附注：None
