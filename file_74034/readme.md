序号：74050
名称：暖宫丸
组成：沙参（净洗）半两，地榆半两，黄耆半两，桔梗半两，白薇半两，牛膝（酒浸1宿）半两，杜仲（去粗皮，姜汁炙）半两，厚朴（去粗皮，姜汁炒）半两，白芷半两，干姜（炮）1分，细辛（去苗）1分，蜀椒（去目及闭目，炒出汗）1分，附子（大者，炮，去皮脐）1个。
出处：《局方》卷九。
主治：冲任虚损，下焦久冷，脐腹(疒丂)痛，月事不调，或来多不断，或过期不至，或崩中漏血，赤白带下，或月内再行，淋沥不止，带下五色，经脉将至，腰腿沉重，痛连脐腹，小便白浊，面色萎黄，肢体倦怠，饮食不进，渐至羸弱；及治子宫久寒，不成胎孕。
加减：None
功效：None
用法用量：每服20-30丸，空心温酒或枣汤吞下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
