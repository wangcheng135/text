序号：42183
名称：青金定命丸
组成：胡黄连末1两，芦荟（研）3分，青黛（研）3分，白槟榔1枚（为末），肉豆蔻（去壳）1枚（为末），诃黎勒5枚（去核，为末），木香（为末）半两，麝香（研）半两，丹砂（研）半两，密陀僧（捣，研）半两，丁香（为末）半两，红雪（研）1分，鹤虱（为末）1分。
出处：《圣济总录》卷一七三。
主治：小儿宿有疳气，又因肠虚下痢，寒湿相乘，虫因虚动，侵食脏腑，或口齿生疮，或肛门伤烂，病名疳(匿虫)。
加减：None
功效：None
用法用量：每服5-7丸。奶疳，腊茶送下；气疳，丁香汤送下；脑疳，黄连汤送下；肺疳，橘皮汤送下；急疳，干笋汤送下；食疳，生姜汤送下；脾疳，大枣汤送下；肝疳，盐汤送下。
制备方法：上为末，用酒煎豮猪胆膏为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
