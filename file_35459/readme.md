序号：35469
名称：三仁五子丸
组成：菟丝子（酒浸1宿，别捣，焙干）1两，五味子1两，枸杞子1两，覆盆子1两，车前子1两，柏子仁1两，酸枣仁（炒）1两，薏苡仁（微炒）1两，沉香1两，鹿茸（醋涂，炙黄，锉）1两，肉苁蓉（酒浸1宿，切，焙）1两，巴戟（去心）1两，当归（洗，焙）1两，白茯苓（去皮）1两，乳香（别研）1两，熟干地黄（洗，焙）1两。
出处：《杨氏家藏方》卷九。
主治：血气耗虚，五脏不足，睡中惊悸，盗汗怔忪，梦遗失精，四肢倦懈，肌肤瘦弱，或发寒热，饮食减少。
加减：None
功效：常服养心益肝，生血补气，润泽肌肤，倍进饮食。
用法用量：每服50丸，空心温酒或盐汤送下。
制备方法：上为细末，次入研了药和匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
