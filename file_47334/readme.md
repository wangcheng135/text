序号：47349
名称：祛风保安丹
组成：乌蛇（酒浸，去皮骨取肉，焙干）半两，附子（炮，去皮脐）5钱，赤箭天麻（去苗）3钱半，朱砂（别研，为衣）3钱半，白附子（炮）3钱，防风（去芦头）3钱，没药（别研）3钱，白术3钱，细辛（去叶土）3钱，羌活（去芦头）3钱，独活（去芦头）3钱，黄耆（生用）3钱，白僵蚕（炒去丝嘴）3钱，藁本（去土）3钱，香白芷3钱，五灵脂（微炒，别入）3钱，赤芍药3钱，乌药3钱，川乌头（炮，去皮脐尖）3钱，当归（洗）3钱，木香2钱半，全蝎（去毒，微炒）2钱半，川芎2钱半，干姜2钱半，乳香（别研）2钱半，石莲肉（去心）2钱半，麝香（别研）1钱半。
出处：《杨氏家藏方》卷一。
主治：中风左瘫右痪，一切风气攻注，荣卫凝滞，筋骨疼痛，手足拘挛，口眼不正，肢体偏废。
加减：None
功效：None
用法用量：每服1丸，细嚼，茶、酒任下；金银薄荷汤或豆淋酒亦得。
制备方法：上为细末，炼蜜为丸，每1两作15丸，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
