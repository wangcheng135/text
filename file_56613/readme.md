序号：56628
名称：消化片
组成：枳实1kg，木香1kg，厚朴1kg，神曲1kg。
出处：《云南省农村中草药制剂规范》。
主治：积食气滞，胸腹胀痛。
加减：None
功效：健脾散积，理气止痛。
用法用量：None
制备方法：取枳实、木香、厚朴置简易挥发油&mdash;&mdash;浸膏连续提取器中进行煎煮，至挥发油收集器中挥发油量不再增加为止，收取油层滤取煎液。药渣再煎煮1小时，滤过，合并2次滤液，浓缩至稠膏状。加入约1/2神曲粉（过90目筛）混合，烘干，粉碎，过90目筛。与其余神曲粉充分混匀，用80％乙醇制粒，过14目筛，60℃烘干，取小量干燥颗粒研细，用此细粉吸附提取的挥发油，充分混匀，再将其余干燥颗粒接等量递加法混匀。加硬脂酸镁、滑石粉适量，混合压片。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
