名称：五福延寿丹
组成：五味子6两，肉苁蓉4两（酒浸，焙），牛膝3两（酒浸），菟丝子（酒浸，炒）2两，杜仲（姜炒断丝）3两，天冬（去心）2两，广木香1两，巴戟（去心）2两，山药2两，鹿茸（酥油炙透）1两，车前子（炒）2两，菖蒲（焙）1两，泽泻（去毛）1两，生地1两（酒洗），熟地1两（酒制），人参（去芦）1两，乳香1两（另研），没药5钱（另研），枸杞子1两，大茴（炒）2两，覆盆子1两，赤石脂（煅）1两，地骨皮2两，杏仁（去皮尖）1两，山茱萸（去核）2两，柏子仁1两，川椒（去目，合口炒）7钱，川楝肉（炒）1两，远志（去心）1两，龙骨（煅）5钱，白茯苓（去皮）1两，当归（酒洗）1两。_x000D_
。
出处：《便览》卷三。
主治：男子女人诸虚百损，五劳七伤，未及半百而须发早白，行路艰难，形容羸瘦，眼目昏花，远年近日咳嗽，吐痰见血，夜梦遗精，并妇人久不生育。

用法用量：每服30丸，空心盐汤或盐酒送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
