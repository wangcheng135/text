序号：78904
名称：酸浆膏
组成：酸浆草根（生者）1握（细锉，以洗净乱发缠裹成一团。酸浆草成小窠子，结实红色，似栀子，中心有子如樱桃。又名苦蚺也），皂荚2挺（不蛀者。捶，锉），附子（去皮脐，生，为末）半两，白矾（研）1钱，麝香一皂子大（研，留在乳钵内）。
出处：《圣济总录》卷一七二。
主治：小儿牙疳出血，牙龈臭烂；风牙、走马疳、蛀牙等。
加减：None
功效：None
用法用量：患者先用盐汤漱剔牙缝，令净，然后以指蘸药膏揩之；如龈烂，以帛子摊药贴。
制备方法：先用米醋1碗，入酸浆草根及皂荚2味，慢火煎至半碗，去滓，入附子、白矾末，更熬成膏，取出，候冷，刮入麝香乳钵内，研匀，以瓷盒收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
