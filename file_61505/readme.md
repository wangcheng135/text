序号：61521
名称：密补固真丹
组成：天南星半两，半夏1两（制），神曲1两，麦芽1两，茴香1两（炒），荆三棱（炮）1两，白附子1两，干生姜1两，川乌头（生）1两，巴豆7个，牵牛3两，代赭石2两，官粉1分。
出处：《宣明论》卷七。
主治：脾肾真元虚损，泄泻痢疾，痰嗽，哕痞，水谷酸臭，饮食无味，脐腹冷痛，肢体麻痹，下虚痰厥，上实壅滞，肾虚耳鸣，脾虚困惫，耳焦齿槁，面黧身悴，唇黄口燥，发堕爪退，风虚偏枯，中满膈气。
加减：除泄泻外，并加大黄1两。
功效：补养脾胃，宣通气血。
用法用量：每服10丸，加至50丸，温水送下。
制备方法：上为末，水为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
