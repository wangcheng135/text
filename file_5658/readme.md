序号：5658
名称：生乳灵
组成：穿山甲30kg，沙参10kg，天花粉50kg，丝瓜络50kg，白马悬蹄60kg，鹿角10kg。
出处：《天津市中成药规范》。
主治：妇人气血不足，经络不通，奶汁稀薄及奶汁灰黄。
加减：None
功效：通经活络，下乳。
用法用量：每次服40毫升，每日服3次，温热后服用。
制备方法：天花粉用25％醇按浸渍法提取浸液，穿山甲、沙参、丝瓜络、白马悬蹄、鹿角按水煮法取其煮液，将煮液合并过滤，减压浓缩至适量，兑入单糖浆1200升，苯甲酸钠0.5％，待冷后加入天花粉浸液，搅拌均匀，比重1.2～1.24，25℃静置7天后即可灌封。每瓶120毫升。
临床应用：None
各家论述：None
用药禁忌：None
附注：生乳糖浆（《中药制剂手册》）。
