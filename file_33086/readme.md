序号：33095
名称：芜荑丸
组成：黄连（去须，微炒）半两，黄柏（去粗皮）半两，甘草（微炒）半两，青橘皮（去白）半两，龙胆草（去芦头）半两，干蟾1个（酥炙），胡黄连1分，白芜荑仁（炒）1分，使君子14个（炮，去壳），青黛1钱，麝香半钱（别研）。
出处：《杨氏家藏方》卷十八。
主治：小儿五疳骨热，面黄肌瘦，饮食虽多，不长肌肤，牙齿宣露，或有盗汗，疳疮湿痒，小便白浊。
加减：None
功效：None
用法用量：每服20丸，温米饮送下，不拘时候。
制备方法：上为细末，研匀，用獖猪胆汁和得所，分药入猪胆内，各令七分满，以线系定，于银、石器中用浆水煮5-7沸，当风挂1宿后剥去猪胆不用，只取药，再和令匀为丸，如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
