序号：73606
名称：蓬莱雪
组成：黄芩（生，研细末）3钱，黄连（生，研细末）3钱，栀子（炒黑，研细末）3钱，雄黄3钱，硼砂3钱，牛胆消3钱，鸡内金1钱，人中白1钱，枯矾1钱，制青梅（煅存性）5分，青黛5分（上为细末，入后7味），牛黄3分，麝香3分，铜青5片，熊胆5片，珍珠5片，冰片5片，儿茶8分。
出处：《喉科紫珍集》卷上。
主治：咽喉七十二症。
加减：None
功效：None
用法用量：每遇喉症，以少许吹于患处，1日夜10余次。徐徐流出痰涎，渐愈。如有腐烂臭秽，用猪牙皂、扁柏叶同捣，和水去滓，灌漱令净。
制备方法：上共研匀，收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：制青梅法：大青梅一斤（去核），入白矾、食盐各五钱，拌匀，再以蜒蚰虫不拘多少，层层间之，一日夜取梅，晒干，以汁尽为度，煅存性用。
