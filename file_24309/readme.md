序号：24317
名称：人参鳖甲丸
组成：杏仁（汤浸，去皮尖，炒）1两，人参1两，当归（洗，焙）1两，赤芍药1两，甘草（炙）1两，柴胡（去苗）1两，桔梗（去芦）1两，地骨皮1分，宣黄连（去须）1分，胡黄连1分，肉桂（去粗皮）半两，木香半两，麝香（别研）半分，鳖甲1枚（重2两者，醋炙黄色为度）。
出处：《局方》卷九（淳祐新添方）。
主治：妇人一切虚损，肌肉瘦悴，盗汗心忪，咳嗽上气，经脉不调，或作寒热，不思饮食。
加减：None
功效：None
用法用量：每服50丸，温酒送下，不拘时候。
制备方法：上为细末，用青蒿1斤，研烂，绞取汁，童便5升，酒5升，同熬至2升以来；次入真酥3两，白沙蜜3两，再熬成膏，候冷，方下众药末，搜和令匀为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
