序号：5101
名称：木通膏
组成：春用：木通3两，白芷2两，细辛2两（去苗叶），摩勒香2两，夏用：木通3两，续断3两，白芷2两半，黄耆2两，芍药2两，秋用：木通3两，熏陆香3两，黄耆2两，白芷1两半，冬用：木通3两，黄耆3两，木香3两，当归（锉，焙）2两，芎1两半，摩勒香3两，熏陆香1两。
出处：《圣济总录》卷一三○。
主治：诸疮肿，四时可用。
加减：None
功效：未成脓即消，已成脓即破。
用法用量：欲用药先嚼少盐擦疮上，次贴之。
制备方法：上各依四时，并(口父)咀，以清麻油2升煎，候白正黄色，即去滓，绞取油放冷，下黄蜡五两，候消尽，更入铅丹10两，先下6-7两，看硬软得所即止，亦不须入尽10两，以急火煎，用柳篦搅，勿令住手，点物上，其色如漆即膏成。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
