序号：51538
名称：追风膏
组成：牛膝2两，桃仁2两，麻黄2两，当归2两，生草乌2两，良姜2两，独活2两，肉桂2两，赤芍2两，海风藤2两，红花2两，威灵仙2两，大戟2两，天麻2两，羌活2两，生山甲2两，细辛2两，乌药2两，蛇退5钱，苏木1两，蜈蚣5钱，生地1两，熟地1两，生川乌5钱，川断1两，白芷2两，五加皮5钱。
出处：《北京市中药成方选集》。
主治：筋骨疼痛，四肢麻木，行步艰难，腰膝无力。
加减：None
功效：舒筋活血，追风散寒。
用法用量：微火化开，贴患处。
制备方法：上切，用香油192两，文武火炸枯，去滓过滤，炼至滴水成珠，再入黄丹72两成膏，取出放入冷水中出火毒后，加热溶化，另兑细料丁香1钱，没药3钱，雄黄3钱，檀香3钱，血竭3钱，麝香1钱，乳香3钱，冰片1钱（研细和匀）。每大张重1两，小张5钱。
临床应用：None
各家论述：None
用药禁忌：孕妇忌贴腰腹部。
附注：None
