序号：80524
名称：藿香左金汤
组成：杜藿香3钱，吴茱萸2分，小川连6分，新会皮2钱，姜半夏1钱半，炒枳壳1钱半，炒车前1钱半，赤苓3钱，细木通1钱，建泽泻2钱，猪苓1钱半，六一散4钱（包煎）。
出处：《湿温时疫治疗法》。
主治：热霍乱。上吐黄水，或呕酸水，暴注下迫，泻出稠粘，心烦口渴，胸闷腹痛，溺赤短热，脉多弦急，舌苔黄腻，或黄多白少。
加减：None
功效：None
用法用量：先用鲜括淡竹茹5钱，炒香鲜枇杷叶1两，并井、河水各1碗，煎至1碗，分2次服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：服后毋多饮茶，多饮茶则连药吐出，不得药力矣，切宜忍耐。
附注：None
