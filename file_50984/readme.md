序号：50999
名称：大安散
组成：鳖甲1枚重2两（好醋煮1时辰，令裙襕自脱而出，净洗，别以醋煮，为粗末，取半两净者），大草果5枚（去皮），半夏5枚，生姜5块（如大拇指大，连皮用），甘草5寸（炙），大枣5枚（去核）。
出处：《普济方》卷二○○。
主治：久疟不瘥，不拘寒热多少。
加减：None
功效：None
用法用量：上同捣烂，并作1服。以无字白纸数重裹之，水湿透，入灰火内煨令香熟取出。以水5盏，煎1盏，去滓；再以水2盏煎至1盏，连前者和匀，分2服，每服发日五更暖服1盏，至欲发时再暖服1盏。轻者当日即止。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
