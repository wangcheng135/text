序号：35340
名称：乳香膏
组成：乳香7钱，没药7钱，白芷5钱，当归5钱，羌活5钱，独活5钱，川牛膝5钱，川芎5钱，自然铜5钱，石膏5钱，刘寄奴5钱，黑牵牛5钱，黄柏皮5钱，破故纸5钱，白胶香5钱，生地黄5钱，熟地黄5钱，赤芍药5钱，白芍药5钱，黄丹5钱，紫金皮5钱，黄蜡1两。
出处：《得效》卷十八。
主治：老金疮、杖疮。
加减：None
功效：None
用法用量：贴敷疮口。久留可再用瓦下盛，须封裹。孔深者，拍成膏条，穿入孔中，不问浅深，放疮上作热，加轻粉、梅花脑子、朴消入膏内。外贴。
制备方法：上为末，用真清油4两煎沸，却入药同煎，留胶香、黄蜡、黄丹末入，用柳枝不住手搅，试将欲成膏，却入3味，更试成膏，生布滤净，用瓦器盛水，倾在水中，用篦摊开。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
