序号：4514
名称：木香如意丸
组成：木香3钱，丁皮3钱，丁香3钱（不见火），青木香1两，枳实、枳壳（麸炒），青皮1两（去瓤），陈皮（去白），大黄5钱，小姜黄3钱，黑牵牛4两（头末2两，炒），巴豆1两（去壳，不去油），小茵陈3两，大麦蘖6两（用巴豆肉炒麦芽，文武火候，豆焦赤为度，用纸包裹1夜，次早拣去巴豆不用，以麦蒙和前药1处为末），皂角5梃（去皮弦）。
出处：《普济方》卷一八二。
主治：胸膈不快。
加减：None
功效：None
用法用量：每服20丸，热水送下。
制备方法：以水2碗，煎至1碗，取汁打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
