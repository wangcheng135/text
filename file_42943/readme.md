序号：42958
名称：保真汤
组成：当归5分，生地黄5分，熟地黄5分，黄耆（蜜水炙）5分，人参5分，白术5分，甘草5分，白茯苓5分，天门冬（去心）1钱，麦门冬（去心）1钱，白芍药1钱，黄柏（盐水炒）1钱，知母1钱，五味子1钱，软柴胡1钱，地骨皮1钱，陈皮1钱，莲心5分。
出处：《准绳·类方》卷一。
主治：诸虚百损，五劳七伤，骨蒸潮热，咳嗽，诸汗、诸血。
加减：None
功效：None
用法用量：上以水2钟，加生姜3片，大枣1枚，煎8分，食远服。
制备方法：None
临床应用：None
各家论述：《法律》：按此方十八味，十全大补方中已用其九，独不用肉桂耳。然增益地黄，代川芎之上窜，尤为合宜。余用黄柏、知母、五味子滋益肾水，二冬、地骨皮清补其肺，柴胡入肝清热，陈皮助脾行滞，其意中实不欲大补也。
用药禁忌：None
附注：None
