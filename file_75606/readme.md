序号：75622
名称：解炎化酒汤
组成：人参1两，柞木枝2两，黄连3钱，茯苓5钱，菖蒲1钱，寒水石3钱。
出处：《辨证录》卷十。
主治：恣饮烧酒，力不能胜，一时醉倒。
加减：苟无人参，以黄耆2两代之。
功效：None
用法用量：水煎1碗，以井水探冷灌之。
制备方法：None
临床应用：None
各家论述：此方以柞木解其酒毒；黄连，寒水石解其火毒；菖蒲引入心中；用茯苓以分消其酒湿之气；然必用人参以固其气者，使气不随酒俱散，盖烧酒系气酒也，热极则气易散越，固其真气，而火可泻，毒可解也，倘只泻其火而解其毒，火泻毒解而气脱矣，气脱而身将何在哉？此人参之所以必用。
用药禁忌：None
附注：None
