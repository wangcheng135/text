序号：76642
名称：舒筋丸
组成：大天南星（炮裂）半两，杏仁（汤浸，去皮尖双仁，麸炒黄，别研）1两，山栀子（取仁，略炒）1两，川乌头（炮裂）半两，自然铜1两（大小烧酒内淬10数遍，研细，水飞过，淘，别研）。
出处：《普济方》卷九十七。
主治：中风，手足筋急，开展艰难；妇人血风，挛却手足。
加减：None
功效：None
用法用量：每服7-10丸，空心煎葱白酒送下，1日2次。3-5日后，其身上如虫行是效，未觉即加丸数。
制备方法：上为末，入研者再拌匀，更用胡桃肉2两，汤去黄皮，研烂拌匀，法酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
