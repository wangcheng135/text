序号：37877
名称：拨云丹
组成：川芎1两半，黄连1两半，当归1两半，白蒺藜（微炒）1两半，羌活7钱，川椒（去子）7钱，柏子7钱，荆芥穗1两，密蒙花1两，甘菊花1两，蝉壳1两，栝楼2钱，薄荷叶2两，蔓荆子2两（微炒），地骨皮（去土，焙干）、蛇蜕皮（甘草汤煮，焙干），木贼2两（去节，童便浸1宿，焙干）（一方无羌活、木贼）。
出处：《普济方》卷七十八。
主治：眼翳。
加减：None
功效：None
用法用量：细嚼，食后茶清送下。内障气者，木香汤送下；睛暗，当归汤送下，好酒亦可；妇人血晕，当归汤送下；小儿斑痘疮翳疔子，半夏煮谷精草汤送下。
制备方法：上为细末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
