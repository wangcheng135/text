序号：35569
名称：侧子酒
组成：侧子4两，牛膝4两，丹参4两，山茱萸4两，蒴藋根4两，杜仲4两，石斛4两，防风3两，干姜3两，蜀椒3两，细辛3两，独活3两，秦艽3两，桂心3两，芎3两，当归3两，白术3两，茵芋3两，五加皮5两，薏苡仁2升。
出处：《千金》卷七。
主治：风湿痹不仁，脚弱不能行。
加减：None
功效：None
用法用量：上(口父)咀，绢袋盛。清酒4斗，渍6宿。初取3合，稍加以知为度。患目昏头眩者弥精。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：侧子浸酒（《圣济总录》卷二十）、牛膝酒（《圣济总录》卷八十四）。方中丹参，《圣济总录》作“丹砂”。本方方名，《外台》引作“金牙侧子酒”。
