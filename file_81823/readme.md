序号：81839
名称：蕤仁丸
组成：蕤仁（去皮）2两，地肤子2两，石决明（净洗，别捣）2两，人参2两，细辛（去苗叶）2两，地骨皮（去土）2两，白茯苓（去黑皮）2两，白术2两，楮实3两，石胆（研如粉）半两，空青（别研如粉）1两半，防风（去叉）1两半，熟干地黄（焙）3分，鲤鱼胆5枚，青羊胆1枚。
出处：《圣济总录》卷一一二。
主治：眼见黑花飞蝇，涩痛昏暗，渐变青盲。
加减：None
功效：None
用法用量：每服20丸，食后、临卧米饮送下，日2次。
制备方法：上除胆及研药外，细锉，焙，为末，入研药拌匀，胆汁和，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
