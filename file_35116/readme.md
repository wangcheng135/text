序号：35126
名称：乳香丸
组成：乳香（研）1两，丁香1两，木香1两，附子（炮裂，去皮脐）半两，五灵脂（为末）半两，干姜（炮）半两，桂（去粗皮）3分，芫花（醋拌，炒焦黄）3分，青橘皮（汤浸，去白，炒）3分，猪牙皂荚（去皮，酥炙）1两，巴豆（去皮心膜，别研如膏，新瓦内摊，去油，取霜）1钱。
出处：《圣济总录》卷七十二。
主治：积聚气滞，胸膈满闷，心腹（疒丂）痛，不化饮食。
加减：None
功效：None
用法用量：每服2-3丸，温生姜汤送下，量虚实加减，食后临卧服。
制备方法：上药除乳香、五灵脂末、巴豆霜外，共为末，入上3味搅匀，煮陈曲糊丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
