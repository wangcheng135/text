序号：61427
名称：啄木散
组成：腊月啄木鸟1个（用无灰酒3升，先以瓦罐子1个，底铺荆芥穗叶厚1寸，上顿啄木鸟；又用荆芥稳叶盖1寸厚，倾酒在内，仍用纸封合，盐泥固济，炭煅之，候酒干，青烟出为度，去火放冷，只取啄木鸟研为末，次入以下项药），石膏2两（煅，研），铁粉（用浆水半升煮尽，研细末用）1两，朱砂1分（研，水飞），附子1两（正生者，炮裂，去皮脐），麝香1分（研），脑子1钱（研）。
出处：《卫生总微》卷六。
主治：多年痫病。
加减：None
功效：None
用法用量：每服1钱，先令病人呷温水2-3口，以温酒1盏，先少许调药饮之，余酒送之，服毕便就枕睡少时。临发时1服。候1-2日更服1次，不过10服即愈。小儿可减服。
制备方法：上为细末。
临床应用：痫病：予有一亲姓王，患痫病十余年，发即涎潮，手足僵踡，颠仆不省，服此药，百服而愈。
各家论述：None
用药禁忌：None
附注：None
