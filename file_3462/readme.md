序号：3462
名称：太平丸
组成：陈皮3钱，厚朴3钱，木香3钱，乌药3钱，白芥子3钱，草豆蔻3钱，三棱3钱，蓬术（煨）3钱，干姜3钱，牙皂（炒断烟）3钱，泽泻3钱。
出处：《景岳全书》卷五十一。
主治：胸腹疼痛胀满，及食积、气积、血积、气疝、血疝、邪实秘滞，痛剧者。
加减：如欲其峻，须用巴豆2钱。
功效：None
用法用量：每用3分或5分，甚者1钱，随证用汤引送下。凡伤寒停滞，即以本物汤送下；妇人血气痛，红花汤或当归汤送下；气痛，陈皮汤送下；疝气，茴香汤送下；寒气，生姜汤送下；欲泻者，用热姜汤送下1钱，未利再服；利多不止，用冷水1-2口即止。
制备方法：上为细末。巴豆(用滚汤泡，去皮心膜)1钱，用水1碗，微火煮至半碗，将巴豆捞起，用乳钵研极细，仍将前汤搀入研匀，然后量药多寡，入蒸饼浸烂捣，丸前药如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
