序号：9111
名称：加味三仁葱豉汤
组成：鲜藿香2钱，杏仁2钱，苡仁4钱，白蔻1钱，厚朴2钱，法半夏2钱，白蒺藜3钱，菊花2钱，僵蚕2钱，豆豉3钱，葱白（后下）3寸，六一散（包煎）5钱，竹叶1钱半。
出处：方出《蒲辅周医疗经验》，名见《千家妙方》卷上。
主治：风暑湿合病（乙型脑炎）。
加减：None
功效：祛风利湿，调和三焦。
用法用量：None
制备方法：None
临床应用：乙型脑炎陈某某，女，4岁。1964年8月15日诊。发热8天，住院5天，诊为乙脑。头痛剧烈，烦躁，昏睡，汗出时体温即降，小便少，大便干，舌淡苔黄腻，脉浮滑数。由风暑湿合病，治宜祛风利湿，调和3焦。投以上方。8月17日复诊，周身有微汗，体温正常，头痛已除，大便尚干，前方去葱白，豆豉，加神曲1钱半、槟榔1钱半，调治而愈。
各家论述：None
用药禁忌：None
附注：None
