序号：59698
名称：调气饮
组成：当归1钱5分，远志肉1钱5分，川芎1钱，青皮1钱，乌药1钱，香附1钱5分，红花6分，大茴香8分，肉桂5分，延胡1钱，山楂2钱，艾叶（熟）1钱，砂仁、生姜、川断。
出处：《陈素庵妇科补解》卷一。
主治：妇女经欲来而腹痛者。
加减：寒者，倍肉桂；因怒者，加木香，柴胡；饮食停滞，加神曲、枳壳；血少气滞，加人参、白术、丹参；肥人多痰者，加半夏、茯苓；暑令去肉桂。
功效：行气和血。
用法用量：None
制备方法：None
临床应用：None
各家论述：妇人当经期欲来而腹先痛，是气滞而血亦随滞，故未来而腹先痛也。青皮乌药、香附之辛温以行气；红花、延胡、肉桂之辛温以行血；艾叶、茴香以暖命门，归、芎远志、川断以补血和血；山楂兼行气血之滞，腹痛自止。
用药禁忌：None
附注：方中砂仁、生姜、川断用量原缺。
