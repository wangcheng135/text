序号：70991
名称：小理中丸
组成：红豆1两，莪术1两（煨，乘热捣碎），缩砂仁1两，草豆蔻2两（煨），青皮2两（去白瓤），陈皮2两（去白），干姜2两（炮），京三棱2两（煨，乘热碎捣），肉桂2两（去粗皮），良姜3两，牵牛3两（炒香熟），阿魏3两（醋化，去沙石，研）。
出处：《局方》卷三（绍兴续添方）。
主治：三脘气弱，中焦积寒，脾虚不磨，饮食迟化，吃物频伤，胸膈满闷，胁肋疠刺，呕吐哕逆，噫醋恶心，腹胀肠鸣，心腹疼痛，噎塞膈气，翻胃吐食，饮食减少。
加减：None
功效：None
用法用量：每服30丸，生姜、橘皮汤送下，温汤亦得，不拘时候。
制备方法：上为末，水煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：此药无利性，不损气，脾胃偏虚寒者最宜服。
