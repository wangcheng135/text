序号：36244
名称：参燕百补戒烟丸
组成：参燕百补丸。
出处：《中国医学大辞典》。
主治：None
加减：None
功效：戒鸦片烟。
用法用量：如烟瘾1钱，服丸亦1钱，1日吸烟几次，服丸亦几次，瘾前服，每7日减去一成，逐次减除，以戒尽为度。再常服参燕百补丸以善其后。
制备方法：上为细末，称取若干，用清烟膏1成，枣糖汤内化开泛丸，如梧桐子大。
临床应用：None
各家论述：嗜烟之人，精神委顿，此气虚也；形瘦口燥，此血虚也。平日所以能振作者，推赖烟力耳，一经戒绝，则百病丛生矣。此丸益正气，滋阴血，气旺则精神振，血足则形体充，故戒时无委顿之苦，戒后则壮健逾恒。
用药禁忌：None
附注：None
