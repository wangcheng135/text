序号：5748
名称：汉防已膏
组成：汉防己1两，野葛1两半，犀角屑1两，莽草2两半，川乌头1两（去皮脐），吴茱萸1两，川椒1两（去目），丹参1两半，踯躅花2两，川升麻1两，干姜1两，附子1两（去皮脐），白芷2两，当归1两，桔梗1两，巴豆1两（去皮心），雄黄1两（细研），蛇衔1两，防风1两（去芦头），鳖甲1两。
出处：《圣惠》卷四十五。
主治：脚气风毒，筋脉拘急，肿满疼痛。
加减：None
功效：None
用法用量：每取摩所患处。
制备方法：上锉细，用绵裹，以醋2升，浸1宿，以猪脂3斤，慢火煎令药色黄，膏成，绞去滓，，盛瓷盒中。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“防己膏”。
