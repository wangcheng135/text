序号：27327
名称：补肺散
组成：桑白皮2两，熟地黄2两，人参（去芦）1两，紫苑1两，黄耆1两，五味子1两。
出处：《普济方》卷二三一引《本事》。
主治：劳嗽。
加减：None
功效：None
用法用量：每服3钱，加四君子汤、秦艽、黄蜡，加蜜少许，水煎，食后服。
制备方法：上为细末。
临床应用：None
各家论述：1.《医方考》：参、耆脾胃药也，肺虚而益脾胃，乃虚则补其母也；地黄滋肾药也，肺虚而益肾，恐其失养而盗气于母也；五味子酸收药也，咳多必失气，故用酸以收之；紫菀凉肺中之血；桑皮清肺中之气，所谓随其实而泻之。益其所利，去其所害，则肺受益，故曰补肺。_x000D_
2.《医方集解》：此手太阴、足少阴药也。肺虚而用参、耆者，脾为肺母，气为水母也。用熟地者，肾为肺子，子虚必盗母气以自养，故用肾药先滋其水，且熟地亦化痰之妙品也。咳则气伤，五味酸温，能敛肺气，咳由火盛，桑皮甘寒，能泻肺火，紫菀辛能润肺，温能补虚，合之而名曰补肺，盖金旺水生，咳嗽自止矣。
用药禁忌：忌房劳，一切生冷、鱼腥、成毒、醃藏等物。服药止可食淡煮猪蹄肉，仍须先煮熟肉去原汁，再以白汤熟煮。
附注：补肺汤（《妇人良方》卷五）。
