序号：4031
名称：木瓜万补丸
组成：人参1两半，白术1两，阳起石1两（火煅，细研），肉苁蓉2两（净洗，酒浸1宿，切，焙），肉桂1两（去皮），缩砂仁1两，赤石脂1两，肉豆蔻1两（面裹煨熟），当归1两半（切，焙），钟乳粉1两，草豆蔻1两半（去皮，炮），沉香1两，地榆半两，荜茇1两半（乳半盏，慢火煎干，焙），白姜1两（炮），茴香1两（炒），大麦蘖半两（炒），神曲半两（炒），丁香1两，厚朴1两（去皮，姜制），乳香1两半，白茯苓1两，罂粟壳（和米者）20枚（炙），大附子1个（7钱以上者，炮，去皮脐），嫩茄茸1两半（酥炙微黄）。
出处：《传信适用方》卷二。
主治：脾胃久虚，大肠积冷，下痢白脓，或肠滑不固。
加减：None
功效：None
用法用量：每服30丸，并频加至50-70丸，空心、食前米饮送下。
制备方法：上为末，入别研者匀，以木瓜去瓤蒸烂，同药末捣和得所为丸，如梧桐子大，晒干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
