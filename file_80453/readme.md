序号：80469
名称：鳖甲丸
组成：鳖甲1两半（涂醋，炙令黄，去裙襕），川大黄1两（锉碎，微炒），干漆半两（捣碎，炒令烟出），熟干地黄1两，赤芍药1两，芎3分，桂心半两，延胡索半两，牡丹半两，蛴螬14个（微炒），（庶虫）虫14个（去翅足，微炒），水蛭3分（炒令黄），当归3分（锉，微炒），干姜半两（炮裂，锉），虻虫14个（去翅足，微炒）。
出处：《圣惠》卷七十九。
主治：产后积聚。按之跃手，食饮不为肌肤，萎黄不耐劳苦，呕逆上气，月水闭塞。
加减：None
功效：None
用法用量：每服10丸，食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
