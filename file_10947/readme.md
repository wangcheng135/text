序号：10947
名称：二益丹
组成：草果2斤，砂仁2斤，紫蔻1斤，广木香2斤，丁香1斤，母丁香1斤，肉桂3斤，附片2斤，蛇床子2斤，炙草2斤，煅龙骨2斤，炒吴萸2斤，云苓皮2斤，北细辛2斤，花椒2斤，檀香2斤，枯矾2斤，当归6斤，白芷10斤，山奈2斤，海蛸2斤。
出处：《全国中药成药处方集》（兰州方）。
主治：经血不调，赤白带下，行经腹痛，心口痛疼。
加减：None
功效：调经，止带，暖宫。
用法用量：每丸1钱2分重。每付10丸，粘金1张，作丸时加酥油少许。每日服2次，每次1丸，早、晚用黄酒送下；开水亦可。
制备方法：上为细末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻等食物。
附注：None
