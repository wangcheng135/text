序号：54243
名称：大建中汤
组成：黄耆（去芦）1两，附子（炮，去皮脐）1两，鹿茸（酒蒸）1两，地骨皮（去木）1两，续断1两，石斛（去根）1两，人参1两，川芎1两，当归（去芦，酒浸）1两，白芍药1两，小草1两，甘草（炙）半两。
出处：《济生》卷一。
主治：诸虚不足，小腹急痛，胁肋胀，骨肉痠痛，短气喘咄，痰多咳嗽，潮热多汗，心下惊悸，腰背强痛，多卧少气。
加减：咳嗽者，加款冬花；咳血者，加阿胶：便精遗泄者，加龙骨；怔忡者，加茯神。
功效：None
用法用量：上(口父)咀。每服4钱，水1盏半，加生姜5片，煎至7分，去滓温服，不拘时候。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
