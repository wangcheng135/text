序号：2482
名称：升麻鳖甲汤
组成：升麻2两，当归1两，蜀椒（炒去汗）1两，甘草2两，鳖甲（手指大）1片（炙），雄黄半两（研）。
出处：《金匮》卷上。
主治：阴毒为病，面赤斑斑如锦纹，咽喉痛，吐脓血，烂喉痧。
加减：阴毒，面目青，身痛如被杖，去雄黄、蜀椒。
功效：None
用法用量：以水4升，煮取1升，顿服之。老小再服取汗。
制备方法：None
临床应用：None
各家论述：1.《古方选注》：升麻入阳明、太阴二经，升清逐秽，辟百邪，解百毒，统治温疠阴阳二病。但仅走二经气分，故必佐以当归通络中之血，甘草解络中之毒，微加鳖甲守护营神，俾椒、黄猛烈之品，攻毒透表，不乱其神明。阴毒去椒、黄者，太阴主内，不能透表，恐反助疠毒也。_x000D_
2.《证治宝鉴》：以升麻透疠毒，鳖甲泄热守神，当归和调营血，甘草泻火解毒。
用药禁忌：None
附注：阴毒升麻鳖甲汤(《元戎》)、防毒升麻汤(《准绳·幼科》卷六)。
