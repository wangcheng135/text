序号：39210
名称：狐肝丸
组成：狐肝1具（腊月者），老鸦1只（去嘴爪翅尾，与孤肝同于瓷瓶内烧令烟尽，候冷细研），天南星1两半（炮裂），天麻1两，白附子1两（炮裂），乌蛇肉2两（酒拌炒，令黄），干蝎1两（微炒），桑螵蛸4两（微炒），蝉壳1两（微炒），晚蚕蛾1两（微炒），白僵蚕1两（微炒），朱砂2两（细研），牛黄1两（细研），麝香1分（细研）。
出处：《圣惠》卷六十九。
主治：妇人中风卒倒，眼黑头疼，胸膈多痰，言语謇涩，心神恍惚，皮肤顽麻。
加减：None
功效：None
用法用量：每服7-10丸，以豆淋酒送下，不拘时候。
制备方法：上为末，入研了药令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
