序号：70528
名称：紫金丹
组成：好朱砂1斤（生），土盐6两，马牙消6两（2味用坩锅子烧令盐通红）。
出处：《鸡峰》卷二十九。
主治：一切虚损。
加减：None
功效：不老驻颜。
用法用量：每服1丸，空心冷酒送下。
制备方法：将后2味安向鼎，不拘大小，用水8分，后将朱砂用绢袋子六重悬煮，7日取出，淘去沙净，用垍盒先将盐铺盒子底4两，后入朱砂，更用盖头盐4两，后用牡蛎4两，亦盖头了，后用盐泥固济垍盒子，厚半寸，次用醋2升，拌细灰，更将醋灰盖泥者垍盒子，更用10斤炭火烧通赤，用茆灰盖，火烧一伏时后，火灭令冷时，便取朱砂用水淘净，却安向鼎中，用解盐6两捣细，上水底火，先将纸灰铺在炉里，后用炭5两，如鼎高，火临时加减，其水不得煎干，7日取出，又淘去盐净，焙干，入乳钵内研如粉，用枣汤煮糯米粽子为丸，临时看大小丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
