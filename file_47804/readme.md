序号：47819
名称：神仙接骨丹
组成：黄丹（飞过）4两，密陀僧4两，自然铜4两，辰砂1两，血竭1两，乳香1两，没药1两，黄蜡13两，白矾（飞过）13两，腊月猪脂13两。
出处：《朱氏集验方》卷十三。
主治：一切折伤，不问内外轻重，虽已无气，但心头尚暖者。
加减：None
功效：None
用法用量：每服1弹，作15丸，热葱酒吞下；如已绝不能吞，即以热葱酒磨灌，但下喉即自省人识痛，病人痛亦定，仍取10丸，入少油火炙软，摊于帛上，贴痛伤处；如伤损处大，用2丸以上。此药不沾粘肉，即以寻常胶粘膏药。
制备方法：上用新锅子先下脂溶，滤去筋膜，次下蜡，成油，去锅于冷处，再下密陀僧、黄丹，自然铜末，以慢火煎至滴水不散，便出锅于冷处；下诸药，用柳木篦子搅匀，入瓷器中，再不住手搅至凝为丸，如弹子大，候极冷收入通油新瓷盒内，永不败坏。
临床应用：None
各家论述：None
用药禁忌：None
附注：神仙接骨膏（《普济方》卷三一五）。方中辰砂、血竭、乳香用量原缺，据《普济方》补。
