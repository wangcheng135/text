序号：84236
名称：五邪汤
组成：禹余粮（研）1两，防风1两，桂心1两，芍药1两，远志（去心）1两，独活1两，甘草（炙）1两，人参1两，石膏（碎，绵裹）1两，牡蛎（熬）1两，秦艽1两，白术1两，防己1两，菖蒲1两，黄丹1两，茯神1两，蛇蜕皮（炙）1两。
出处：《外台》卷十五引《深师方》。
主治：邪气啼泣，或歌或哭。
加减：None
功效：None
用法用量：以水1升半，纳3方寸匕，煮2沸，去滓服之，1日4次。
制备方法：上为粗末。
临床应用：None
各家论述：《千金方衍义》：五邪者，肝风内发则为风邪，治以防风、独活、蛇蜕之属；二火交煽则为热邪，治以桂心、石膏、芍药之属；脾湿生风则为湿邪，治以防己、秦艽、白术之属；痰迷空窍则为惊邪，治以雄黄、牡蛎、禹余粮之属；惊啼歌哭则为虚邪，治以远志、菖蒲、茯神之属；凡此五邪，靡不由心神怯弱所致，甘草、人参非特安神，并佐诸药之力，病久正虚者，非此不能克应。
用药禁忌：忌生葱、海藻、菘菜、桃李、雀肉、饧、醋等。
附注：禹余粮饮(《圣济总录》卷十四)。《外台》引《古今录验》有雄黄，无黄丹。
