名称：推云散
组成：石蟹1钱（滚水泡，去夹石并砂土，用竹纸包，槌碎，再研至无声为度），黄连1钱（去须），白丁香1钱，宫粉3分，金锡3分（坚重有金星，研飞），青盐3分（锅内水化开，澄去砂土，火上煎干），赤石脂3分，铜绿（古钱上者刮下，研细，水飞，晒干）3分，硼砂4分，乳香4分（透明者，箬包4-5层，铜锅内清水煮数沸，换水煮去油），没药（制同上）4分，熊胆1分（去皮膜），当门子1分，枯矾1分，冰片1分。
出处：《疡医大全》卷十一。
主治：诸般外障，云翳蟹睛，血翳赤膜。
