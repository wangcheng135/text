序号：72277
名称：紫菀丸
组成：紫菀（去苗）2两，蛤蚧1枚（大者，皂荚水浸1宿，涂酥，炙），白茯苓（去黑皮）2两，杏仁（去皮尖双仁，蜜浸1宿，炒）2两，款冬花（用蕊）1两，防风（去叉）1两，麦门冬（去心，焙）1两，人参半两，甘草（炙，锉）1两，马兜铃（炒）1两，黄耆（细锉）半两，赤芍药半两，当归（锉，焙）半两，贝母（生姜汁浸1宿，焙）半两，白药子半两，半夏（生姜汁浸1宿，焙）半两（以上6味并为细末），枣4两（蒸熟，去皮核），大麻子半升（水浸，研烂，去滓取汁），栝楼30枚（大，肉烂研取），龙脑（研）半字（以上4味并研为膏）。
出处：《圣济总录》卷六十五。
主治：肺咳唾血。
加减：None
功效：None
用法用量：每服30丸，煎麦门冬熟水送下。
制备方法：上20味，以前药末入在后膏内，为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
