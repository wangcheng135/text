序号：55579
名称：桃仁散
组成：桃仁1两（汤浸，去皮尖双仁，麸炒微黄），茜根1两半，虻虫2-7枚（微炒，去翅足），水蛭2-7枚（炒令微黄），赤芍药1两，木通1两（锉），川芒消1两，川大黄1两半（锉碎，微炒）。
出处：《圣惠》卷七十二。
主治：妇人月水不通，年月深远，面上皯(黑黾)，黑如噀墨，每思咸酸之物，食之不已，意无足时，此由凝血在脏，热入血室，即歌詠言笑，悲泣不止。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，煎至6分，去滓，空腹温服，如人行10里再服。良久当利下黑血黄涎，亦如泔淀。如下不多，次日再服，使命令绝其根本。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：1月以上不得吃面并驴、马、猪、牛等肉。
附注：《医方类聚》引本方有琥珀一两。
