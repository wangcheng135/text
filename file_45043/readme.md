序号：45058
名称：枳术汤
组成：枳实7个，白术2两。
出处：《金匮》卷中。
主治：心下坚大如盘，边如旋盘，水饮所作。
加减：None
功效：None
用法用量：以水5升，煮取3升，分3次温服。腹中软即当散也。
制备方法：None
临床应用：None
各家论述：1.《金匮玉函经二注》：心下。胃土脘也，胃气弱，则所饮之水，入而不消，痞结而坚，必强其胃，乃可消痞。白术健脾强胃，枳实善消心下痞，逐停水，散滞血。_x000D_
2.《金鉴》：上脘结硬如盘，边旋如杯，谓时大时小，水气所作，非有形食滞也。用枳实以破结气，白术以除水湿，温服三服，则腹软结开而硬消矣。此方君枳实，是以泻为主也。然一缓一急，一补一泻，其用不同，只此多寡转换之间耳。
用药禁忌：忌桃、李、雀肉等物。
附注：枳实白术汤（《外台》卷八引《备急》）、枳实汤（《产育宝庆集》卷上）、白术汤（《准绳·女科》卷五）。
