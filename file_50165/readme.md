序号：50180
名称：茱萸丸
组成：吴茱萸（汤洗7遍，焙）1两半，硇砂半两（用醋浆水淹搅500-700度，用纸滤过，瓷器内侵火逼令干），木香1两，京三棱3两（炮熟，捣），青橘皮（汤浸，去白）1两半，干姜（炮裂）1两半，附子（炮裂，去皮脐）1两半，半夏（汤洗10遍，微炒）1两半，巴戟天（去心，酒浸1宿）1两，茴香子1两（微炒），硫黄1两（甘草水洒研7日）。
出处：《圣济总录》卷一八七。
主治：None
加减：None
功效：补虚，下气去痰。
用法用量：每服10丸，空心盐汤送下。
制备方法：上为末，先用硫黄末，以精羊肉15两细切，拌和令匀，淹1宿，便用好醋1升半，煎汁尽，醋刮在臼内，后将药末并硇砂同入皿内，杵1000下，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
