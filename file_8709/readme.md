序号：8709
名称：二灵丹
组成：何首乌（雌雄各半，采、刮、捣，俱不犯铁，用第一淘米泔浸一伏时，漉出。于银器内，先排枣一重，各擘开，上铺何首乌一重，又用枣一重，复再铺何首乌一重，令尽，次日取清河水入于药内，药上有水约五指，用慢火煮，候枣极烂并何首乌稍软，取出，不用枣，只拣何首乌入在清冷水中浸少时，用竹刀子刮去黑皮及两面浮沫令净，竹刀切作薄片子，慢火焙干，取净）1斤，牛膝（拣去芦头并细梢，只取中间粗者，折作半寸，用好酒浸2宿，取出焙干，净）半斤。
出处：《御药院方》卷六。
主治：None
加减：None
功效：补暖腑脏，祛逐风冷，利腰膝，强筋骨，黑髭发，驻容颜，性温无毒，久服轻身，延年不老。
用法用量：每日服60丸，空心温酒或米饮送下。服至半月，加至70-80丸；又服至1月，加至100丸。服之百日，前疾皆去。
制备方法：何首乌（雌雄各半，采、刮、捣，俱不犯铁，用第1淘米泔浸1伏时，漉出。于银器内，先排枣1重，各擘开，上铺何首乌1重，又用枣1重，复再铺何首乌1重，令尽，次日取清河水入于药内，药上有水约五指，用慢火煮，候枣极烂并何首乌稍软，取出，不用枣，只拣何首乌入在清冷水中浸少时，用竹刀子刮去黑皮及两面浮沫令净，竹刀切作薄片子，慢火焙干，取净）1斤，牛膝（拣去芦头并细梢，只取中间粗者，折作半寸，用好酒浸2宿，取出焙干，净）半斤，上药1处拌和，入石杵臼内捣罗为细末，炼蜜为丸，如梧桐子大。
