序号：66495
名称：小八风散
组成：天雄5分，当归5分，人参5分，附子4分，防风4分，天门冬4分，蜀椒4分，独活4分，乌头3分，秦艽3分，细辛3分，白术3分，干姜3分，麻黄2分，山茱萸2分，五味子2分，桔梗2分，白芷2分，柴胡2分，莽草2分。
出处：《千金》卷八。
主治：迷惑如醉，狂言妄语，惊悸恐怖，恍惚见鬼，喜怒悲忧，烦满颠倒，邑邑短气不得语，语则失忘，或心痛彻背，不嗜饮食，恶风不得去帷帐，时复疼热，恶闻人声，不知痛痒，身悉振摇，汗出猥退，头重浮肿，搔之不知痛，颈项强直，口面喎戾，四肢不随，不仁偏枯，挛掣不得屈伸。
加减：None
功效：None
用法用量：每服半方寸匕，渐至全匕，酒送下，1日3次。以身中觉如针刺者，则药行也。
制备方法：上药治下筛，合相得。
临床应用：None
各家论述：《千金方衍义》：详小八风散所治之证，皆病久本虚邪实，虽用麻黄附子细辛汤及理中、三建等方之制，专守温理正气邪自退之法，药味虽峻而无栾荆、茵芋、踯躅大毒之味，故以小八风散称之。
用药禁忌：None
附注：None
