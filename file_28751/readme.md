序号：28759
名称：十六味保元汤
组成：黄耆1钱，石斛7分，巴戟肉2钱，白茯苓1钱，升麻7分，圆眼肉3钱，贯仲（去根土）3钱，人参2钱，山药1钱，川独活1钱，当归身2钱，莲蕊1钱，黄柏（酒炒）8分，生甘草3分，杜仲（小茴、盐、醋汤浸，炒）1钱5分，骨碎补（先以稻草火上烙去毛，以粗布拭净）2钱。
出处：《寿世保元》卷上。
主治：带下。
加减：潮热，加柴胡8分，黄芩（酒炒）1钱；带甚者，月经必少，其有聚而反来，适来适断，而淋沥不净者，加荆芥1钱，黄连（酒炒）7分，地榆8分；若五心烦躁而口舌干者，加知母1钱，麦门冬1钱，地骨皮8分。
功效：生血固真，补心益肾。
用法用量：水煎，空心温服。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
