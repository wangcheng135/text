序号：51197
名称：荜澄茄散
组成：荜澄茄3分，白豆蔻3分（去皮），丁香3分，沉香半两，木香半两，高良姜半两（锉），桂心半两，白术1两，人参2分（去芦头），陈橘皮1两（汤浸，去白瓤，焙），干姜半两（炮裂，锉），半夏半两（汤洗7遍，去滑），厚朴3分（去粗皮，涂生姜汁，炙令香熟），诃黎勒3分（煨，用皮）。
出处：《圣惠》卷五。
主治：脾胃气弱虚，不思饮食，胸中气满，四肢不和，食即呕吐。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，入生姜半分，枣3枚，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
