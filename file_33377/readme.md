序号：33386
名称：羌活散郁汤
组成：防风、羌活、白芷、荆芥、桔梗、地骨皮、大腹皮、川芎、连翘、甘草、紫草、升麻、鼠粘子。
出处：《痘疹金镜录》卷四。
主治：痘疮实热壅盛，郁遏不得达表，气粗喘满，腹胀烦躁，狂言谵语，睡卧不宁，大小便秘，毛竖面浮，眼张若怒，并风寒外搏，出不快者。
加减：None
功效：None
用法用量：水1钟，加灯心14根。煎6分，温服。
制备方法：上为粗散。
临床应用：None
各家论述：《医林纂要》：羌活、防风、白芷宣达阳气，荆芥去血中风热，桔梗降逆气，大腹皮宽中气，前胡畅滞气；川芎达肝气，地骨皮滋肾水，连翘散心火，木通泄心火，牛蒡子散肺中结热，去皮肤风热，紫草茸活血散瘀，甘草缓肝和脾。
用药禁忌：None
附注：羌防散郁汤（《医林纂要》卷九）。《医林纂要》有木通、前胡，无升麻。
