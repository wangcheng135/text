序号：17698
名称：人参饮子
组成：麦门冬2分，人参（去芦）3分，当归身3分，黄耆1钱，白芍药1钱，甘草1钱，五味子5个。
出处：《兰室秘藏》卷中。
主治：脾胃虚弱，气促气弱，精神短少，衄血吐血。暑月衄血。
加减：None
功效：None
用法用量：用水2盏，煎至1盏，去滓，稍热服。
制备方法：上为粗末，都作1服。
临床应用：None
各家论述：《证因方论集要》：《内经》云：必先岁气，无伐天和。故时当暑月则肺金受克，令人乏气之时也，理宜清金益气。清金故用麦冬、五味；益气故用参、耆、甘草；白芍之酸所以收其阴；当归之辛所以养其血；此亦虚火可补之剂也。
用药禁忌：None
附注：人参饮（《嵩崖尊生》卷八）。
