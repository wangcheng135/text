序号：56469
名称：海藻玉壶汤
组成：海藻1钱，贝母1钱，陈皮1钱，昆布1钱，青皮1钱，川芎1钱，当归1钱，半夏1钱，连翘1钱，甘草节1钱，独活1钱，海带5分。
出处：《外科正宗》卷二。
主治：瘿瘤初起，或肿或硬，或赤不赤，但未破者。肝脾不调，气滞痰凝。石瘿，坚硬如石，推之不移，皮色不变。
加减：None
功效：化痰软坚，消散瘿瘤。
用法用量：上药用水2钟，煎至8分，量病上下食前后服之。
制备方法：None
临床应用：None
各家论述：《方剂学》：本病多成于气滞痰凝，由气及血，以致气血结聚而成。故用海藻、昆布、海带化痰软坚，为治瘿瘤主药；青皮、陈皮疏肝理气，当归、川芎、独活活血以通经脉，配合理气药可使气血和调，促进瘿病的消散。象贝、连翘散结消肿，甘草调和诸药，共以收化痰软坚，行气活血之功。
用药禁忌：凡服此药，先断厚味、大荤，次宜绝欲虚心。
附注：海藻消瘿汤（《嵩崖尊生》卷六）。
