序号：33869
名称：补虚生荣汤
组成：黄耆（蜜炒）3-5钱，当归（去尾）2-3钱，白术2钱，茯苓1钱，熟地3钱，益母草2钱，甘草（炙）2钱，干姜（炒黑透心）3-5分，白芍（煨，酒炒）1钱半，荆芥穗（炒黑）7分。
出处：《会约》卷十五。
主治：产后气血两虚，无神无力，不时昏迷。
加减：若不得力，黄耆可加至两余，再加附子7-8分，以助药力。如有血气痛，名儿枕血，加山楂2钱，1剂愈，即去之，余药不可增入。
功效：None
用法用量：水煎，少加酒、童便和服。可服数剂，气血易于复元，日后体旺而百病消除矣。人能有力，加参更妙。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
