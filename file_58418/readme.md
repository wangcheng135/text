序号：58433
名称：破结散
组成：海藻3分，龙胆3分，海蛤3分，通草3分，昆布3分，礜石（一作矾石）3分，松萝3分，麦曲4分，半夏2分。
出处：《千金》卷二十四，名见《三因》卷十五。
主治：石瘿、气瘿、劳瘿、土瘿、忧瘿等。
加减：None
功效：None
用法用量：每服方寸匕，酒送下，日3次。10日知，20日愈。
制备方法：上为细末。
临床应用：瘰疬《医学正传》：有人于项上生疬，大如茄子，潮热不食，形瘦日久，百方不效，后得此方，去松萝，加真桑寄生一倍服。3-5日后，其疮软而散，热退而愈。屡医数人皆效。
各家论述：None
用药禁忌：禁食鱼、猪肉、五辛、生菜、诸难消之物。
附注：破血散（《秘传外科方》）。《秘传外科方》有贝母（去心）三分。改为丸剂，名“海藻散坚丸”（见《医学入门》卷八），蜜为丸，如绿豆大。每服三十丸，临卧葱白煎汤送下，并含化咽之。
