序号：57670
名称：大清凉散
组成：白僵蚕（酒炒）3钱，蝉蜕（全）12个，全蝎（去毒）3个，当归2钱，生地（酒洗）2钱，金银花2钱，泽兰2钱，泽泻1钱，木通1钱，车前子（炒，研）1钱，黄连（姜汁炒）1钱，黄芩1钱，栀仁（炒黑）1钱，五味子1钱，麦冬（去心）1钱，龙胆草（酒炒）1钱，丹皮1钱，知母1钱，甘草（生）5分。
出处：《伤寒温疫条辨》卷四。
主治：温病表里三焦大热，胸满胁痛，耳聋目赤，口鼻出血，唇干舌燥，口苦自汗，咽喉肿痛，谵语狂乱者。
加减：None
功效：通泻三焦之热，清热利水。
用法用量：水煎，去滓，加蜂蜜3匙，冷米酒半小杯，童便半小杯，和匀冷服。
制备方法：None
临床应用：None
各家论述：《血证论》：诸药清热利水，使瘟毒伏热，从小便去；妙在三虫引药及酒达于外，使外邪俱豁然而解，是彻内彻外之方。
用药禁忌：None
附注：《血证论》有天门冬。
