序号：73747
名称：睡惊丹
组成：蛇黄（火煅红，米醋淬5遍，再将醋煮干为度）半斤，天南星（碾为粉，用薄荷汁为饼，炙熟）半斤，茯苓（去皮）半斤，铁粉（重罗）半斤，使君子仁半斤，脑子（别研）半两，麝香（别研）1两，银箔（研）100片，金箔（研）100片。
出处：《局方》卷十（绍兴续添方）。
主治：小儿惊邪，风热痰壅，咽膈不利，夜卧不安，睡中啼哭，惊风搐搦。因惊者泄泻，其色必青。
加减：None
功效：安神镇心，定惊控痰。
用法用量：每服1丸，5岁儿分2服，3岁以下儿分3、4服，薄荷汤磨下，更量岁数加减。
制备方法：上为末，糯米糊为丸，如皂荚子大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：睡惊丸（《卫生总微》卷六）。
