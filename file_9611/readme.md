序号：9611
名称：仙方隔纸膏
组成：黄连半两，何首乌半两（去皮），草乌半两（去皮），当归尾半两，白芷半两，川乌（去皮）2分半，黄丹（夏用）2两，乳香半两，没药半两，血竭半两。
出处：《秘传外科方》。
主治：发背，痈疽，外臁，下蛀，诸般恶毒疮疖。
加减：None
功效：None
用法用量：贴患处。
制备方法：前6味，总为(口父)咀，用清油5两，同药一处入于铫子内，以文武火熬，待药黑色，用布滤去滓，仍将药油入铫内，下黄丹，用桃柳枝1把，不住手搅之，又黑色，即将血竭、乳、没细末入内，搅匀略煎，滴在水中，成珠不散，却用瓦碗盛之，沉在冷水中，浸1昼夜出火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
