序号：56339
名称：海马保肾丸
组成：海马1对，砂仁2钱，远志肉（炙）2钱，杞子3钱，鹿茸（去毛）3钱，黄耆1两3钱，山药3钱，白术（炒）3钱，肉桂（去粗皮）2钱，锁阳3钱，茯苓6钱，蛤蚧（去头足）1对，苁蓉（炙）1两，人参（去芦）3钱，熟地6钱，杜仲炭3钱，狗脊（去毛）3钱，钟乳石（煅）2钱，阳起石（炙）1钱，巨胜子1钱，黄精（炙）1钱，龟版（炙）1钱，淫羊藿（炙）5分。
出处：《北京市中药成方选集》。
主治：肾气虚寒，精神衰弱，脑亏健忘，四肢无力。
加减：None
功效：滋阴益气，补骨助阳。
用法用量：每服2钱，日2次，温开水送下。
制备方法：上为细末，过罗，用冷开水泛为小丸，用牡蛎粉2钱为衣。
临床应用：None
各家论述：None
用药禁忌：忌色欲及刺激性食物。
附注：None
