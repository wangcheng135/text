序号：26701
名称：导源饮
组成：白茯苓皮2两，车前子1两，大腹皮（水洗）1两，通草1两，木通1两，薏苡仁（炒）1两，天仙藤1两半，桑皮1两半，郁李仁1两半，冬瓜子（炒）1两半，木香（纳怀内取燥）1两半，赤石脂8钱，甜葶苈8钱（用黑枣拌匀蒸用），大杏仁1两（去皮尖双仁，炒令微黑，另捣），当归1两半（去芦，酒洗）。
出处：《普济方》卷一九一。
主治：各种水气。
加减：None
功效：None
用法用量：每服30丸，渐加至50-60丸，食前温酒送下；如患人素不饮，淡盐汤送下。
制备方法：上为细末，用真藕粉打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
