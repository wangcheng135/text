序号：75181
名称：蒸脐补气散
组成：五灵脂1两，夜明砂1两，枯矾1两。
出处：《内外科百病验方大全》。
主治：气虚体倦，肚腹畏寒，下元虚冷。
加减：None
功效：None
用法用量：每逢春分、秋分、夏至、冬至先1日，避风，用温水将脐眼洗净，纳麝香5厘于脐内，将蒸面为圈（面圈深寸许，横径1寸6-7分），烘微温安脐上，药一包铺圈内，以蕲艾绒作团（每团重1分，或6-7厘）放药末上，用香火烧燃，若干岁即烧若干团，烧完用荞面作饼如圈大（如无荞麦，麦粉亦可）盖圈上，俟药冷，缓缓取下。久久行之，不可间断，受益无穷。
制备方法：上为细末，分为四包，存贮听用。
临床应用：None
各家论述：None
用药禁忌：忌茶7日。
附注：None
