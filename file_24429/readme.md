序号：24437
名称：芒消紫丸
组成：芒消4两，大黄4两，半夏2两，代赭1两，甘遂2两，巴豆200枚，杏仁120枚。
出处：《千金》卷五。
主治：小儿宿食、癖气、痰饮，往来寒热，不欲食，消瘦。
加减：None
功效：None
用法用量：百日儿服1丸，过百日至1岁服2丸，随儿大小，以意为度。当候儿大便，中药出为愈。若不出，更服如初。
制备方法：上为末，别捣巴豆、杏仁治如膏，旋纳药末，捣三千杵，令相和合，强者纳少蜜为丸，如胡豆大。
临床应用：None
各家论述：《千金方衍义》：牛黄丸为膏粱者设，芒消紫丸为藜藿者设。方中芒消以代真珠之涤热；大黄、甘遂以代牛黄之荡实；半夏以代附子之破结，在粗厉之子，原无藉于峻温也。
用药禁忌：None
附注：芒消丸（《普济方》卷三八六）。方中甘遂，《普济方》作甘草。
