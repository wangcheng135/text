序号：30435
名称：还魂汤
组成：麻黄3两，桂心2两，甘草1两，杏仁70粒。
出处：《千金》卷二十五。
主治：卒感忤，鬼击飞尸，诸奄忽气绝，无复觉，或已死咬口，口噤不开。
加减：None
功效：None
用法用量：上（口父）咀。以水8升，煮取3升，分3服。口噤不开，去齿下汤，汤入口不下者，分病人发左右捉踏肩引之药下，复增，取尽1升，须臾立苏。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：此即《伤寒论》“太阳例”中麻黄汤，以桂心易桂技入肝以招其魂；麻黄入肺以通其魄；杏仁入络以降其逆；甘草入腑以缓其暴，暴逆散而魂魄安矣。
用药禁忌：None
附注：还魂散（《圣惠》卷五十六）、追魂汤（《普济方》卷二三七引范氏方）。
