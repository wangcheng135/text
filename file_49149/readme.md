序号：49164
名称：胜金丹
组成：朱砂3两（研），雌黄1两5钱，硫黄5钱。
出处：《遵生八笺》卷十八。
主治：一切风疾，半身不遂，日不收敛，身转不得。
加减：None
功效：None
用法用量：每服3-5丸，空心冷椒汤送下。服5钱即愈。
制备方法：二黄研如泥，用桑柴灰淋汁于锅中，投二黄化熔，入朱末同熬化搅匀，再入灰汁，旋添旋煮，3日3夜；药在锅内有泣声，刮起取药，入铁鼎内，以文武火逼干阴气，方加鼎楔，盐泥固济，如法用炭火30斤，煅至火剩3-4斤即止，待冷开看，药成1片在底，凿取如白银，研如粉；用甘草、余甘子2味，同药砂锅内煮1日，出火毒，取起研，同米饭为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：忌羊血。
附注：None
