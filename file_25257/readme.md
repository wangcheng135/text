序号：25265
名称：如意丸
组成：生地黄2两，熟地黄2两，天门冬（去心）1两，麦门冬（去心）1两，川椒（去目，炒）1两，胡芦巴（酒炒）1两，补骨脂（炒）1两，苁蓉（酒浸）1两，杜仲（炒去丝）1两，白茯苓1两，小茴香（炒）1两，菟丝子（酒浸）1两，川楝肉1两，地龙（酒浸，去土）1两，石菖蒲1两，枸杞1两，远志（去心）1两，青盐半两（炒），山栀（去皮）2钱（炒），川山甲14片（炙），甘菊花3钱半。
出处：《丹溪心法》卷三。
主治：None
加减：None
功效：补损。
用法用量：每服70-80丸，白汤温酒任下。
制备方法：上为末，用晋枣（煮，去皮核）2两，核桃肉（煮，去皮）2两，各研如泥，余再炼蜜和丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
