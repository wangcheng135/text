序号：5665
名称：玉液汤
组成：牛膝1斤，天麻5两，秦艽1两，防风2两，枸杞2两，蚕沙2两（拣净），大栗子（炒熟，用肉）1斤，桔梗2两，当归1两8钱，苍术2斤（柑浸，去皮，蒸熟），地黄1两8钱。
出处：《医方类聚》卷二十三引《经验秘方》。
主治：诸般风疾，麻痹瘫痪，动止不得。
加减：None
功效：除风痰。
用法用量：每服酒1盏，日3次。虽患年深，亦不过2斗酒。
制备方法：上俱各洗净，细切，用无灰酒2斗，瓷缸内浸，用白纸7重，封7日，取用药时，人面不得于缸口上觑。
临床应用：汴梁州判王义甫患风年半，服此药酒，未及2斗，疾痊。
各家论述：None
用药禁忌：忌湿面，动风之物。
附注：None
