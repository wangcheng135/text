序号：42902
名称：保养元气膏
组成：麻油1斤4两（加甘草2两，先熬六七滚，然后下诸药），生地黄5钱，熟地黄（俱酒洗）5钱，麦门冬5钱，肉苁蓉（酒洗）5钱，远志肉5钱，蛇床子（酒浸）5钱，菟丝子（酒浸）5钱，牛膝（酒洗）5钱，鹿茸5钱，川续断5钱，虎骨5钱，紫梢花5钱，木鳖子5钱，谷精草5钱，大附子5钱，肉桂5钱。
出处：《景岳全书》卷六十四引邵真人方。
主治：腰膝疼痛，五劳七伤，诸虚百损，半身不遂，膀胱疝气，带浊淫淋，阴痿不举。
加减：None
功效：助元阳，补精髓，周血脉，镇玉地，养龟存精；妇人经净之时，去膏而泄，则可成孕。
用法用量：每用膏7-8钱，红绢摊贴脐上或腰眼间，每贴50-60日再换。
制备方法：上熬成，以煮过松香4两、飞丹半斤收之。次下龙骨、倭硫黄、赤石脂各2钱，又次下阳起石3钱、麝香5分、蟾酥、鸦片各1钱，又次下黄占5两，上煎成，入井中浸3-4日。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
