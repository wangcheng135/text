序号：69763
名称：琥珀丸
组成：琥珀1两（细研），当归1两（锉碎，微炒），芎半两，木香半两，桂心半两，羌活3分，槟榔3分，没药半两，牛膝1两（去苗），朱砂3分（细研，水飞过），延胡索3分，桃仁3分（汤浸，去皮尖双仁，麸炒微黄），熟干地黄半两，硼砂3分（不夹石者，细研），鳖甲1两（涂醋，炙令黄，去裙襕），姜黄半两，苏合香半两，柴胡1两（去苗），赤芍药半两，牡丹半两，川大黄1两（锉碎，微炒），麝香1分（细研）。
出处：《圣惠》卷七十。
主治：妇人血风劳气，四肢羸瘦，骨节痠痛，口干心烦，经脉不利，或时腹痛，干呕，不思饮食，日渐困乏。
加减：None
功效：None
用法用量：每服30丸，食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
