序号：48300
名称：神效丸
组成：芎2两，羌活（去芦头）2两，猪牙皂荚5挺（半生半炙），何首乌（炒，去黑皮）4两，杏仁（汤浸，去尖双仁和皮，研），50枚。
出处：《圣济总录》卷十八。
主治：癞疾。
加减：None
功效：None
用法用量：每服5丸，黑参汤下，1日3次；初服5丸，后逐日每服加5丸，加至50丸，却逐日每服减5丸，减至5丸止。初服药至第5-7日，头痛，牙缝中出涎，是效。
制备方法：上除杏仁外，为末，腻粉半两，与杏仁同研极细，用白羯羊肉4两，薄切作大片，先铺绵，次铺肉，将上件药裹了，用线紧系定，入银器内煮，不可犯铁器，约用水1大碗，于药四面各余二指许水，方可用碗合定，煮候泣尽水，去绵不用，和肉杵三五千下，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：服药后，只得吃淡粥，候服补药，以次吃淡饭。须忌房室半年。
附注：None
