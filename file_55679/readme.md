序号：55694
名称：桃花丸
组成：桃花3分，苏合香3分，安息香3分，木香3分，槟榔3分，川芒消3分，水蛭半两（炒令微黄），虻虫半两（炒令微黄，去翅足），鳖甲（涂醋，炙令黄，去裙襕）1两，麒麟竭1两，附子（炮裂，去皮脐）1两，柴胡（去苗）1两，卷柏1两，当归（锉，微炒）1两，辛夷1两，白芷1两，紫石英（细研，水飞过）1两，禹余粮（炒，醋拌7遍）1两，芎藭1两，牡丹1两，细辛1两，麦门冬（去心，焙）1两，羌活1两，桂心1两，肉豆蔻（去壳）1两。
出处：《圣惠》卷七十二。
主治：妇人月水不通，无子，由子宫风冷，积血滞于膀胱，故致腰胯疼痛，手脚心热，背膊妨闷，经络不调，腹内多气，四肢乏力，面无血色，及多(黑干)(黑曾)。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前煎茅香汤送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
