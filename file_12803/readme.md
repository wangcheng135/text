序号：12809
名称：石斛丸
组成：石斛2两（去根，锉），肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），菟丝子1两（酒浸3宿，晒干，别捣为末），牛膝1两（去根，锉），熟干地黄1两，杜仲1两（去粗皮，炙微赤，锉），泽泻1两，枸杞子1两，山茱萸1两，桂心1两，白茯苓1两，补骨脂1两（微炒），覆盆子1两，附子1两（炮裂，去皮脐），巴戟1两，桑螵蛸1两（微炒），钟乳粉1两，车前子1两，牡蛎粉1两，龙骨1两，阳起石1两（酒煮半日，细研，水飞过）。
出处：《圣惠》卷九十八。
主治：None
加减：None
功效：补虚损，壮腰膝，暖水脏，止小便滑数。久服好颜色，强志倍力，耐寒暑，填精髓，令人肥健。
用法用量：每日30丸，空心以温酒送下。
制备方法：上为末，入研了药令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
