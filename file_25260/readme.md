序号：25268
名称：如意丸
组成：天麻2两，钩藤2两，僵蚕2两，蝉衣1两，麻黄5钱，细辛3钱，桂枝1两，琥珀1两5钱，沙参1两，薄荷6钱，槟榔1两，胡黄连5钱，雄黄（飞）5钱，川贝母1两5钱，枳壳1两5钱，胆南星1两5钱，广木香1两，沉香1两，全蝎3钱，法夏1两5钱，杏仁1两5钱，白附子8钱，防风1两5钱，天竺黄1两5钱，使君子1两，蟾酥5钱，牵牛子1两，熟大黄2两，冰片1钱6分，西牛黄5钱，芥子5钱，广陈皮1两5钱，麝香1钱6分，云黄连5钱，朱砂（水飞）10两5钱。
出处：《全国中药成药处方集》（南昌方）。
主治：小儿夜啼，发热惊恐，痰喘气促，及急惊风，手足搐搦，角弓反张。
加减：None
功效：None
用法用量：4个月内小儿，每服2-5粒；5个月至1岁，每服5-7粒；1-2岁服10粒；3-5岁，每服15粒；5-10岁，每服20粒，根据病情轻重，每日服1次或3次，温开水化服。幼儿服此药时，宜先检视口腔，如牙龈上发现粟粒形水泡，即以软绸开水洗净，去其恶血，然后服此丸药。
制备方法：上为极细末，水泛为丸，如芥子大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：体弱儿童忌服。
附注：None
