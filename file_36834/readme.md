序号：36844
名称：固真丹
组成：沉香半两，丁香半两，木香半两，茴香（炒）半两，人参半两，当归（微炒）半两，滑石半两，乳香（另研）5钱，没药（另研）5钱，干胭脂（另研，一半为衣），琥珀（另研）5钱，川山甲（蛤粉炒）5钱，全蝎（微炒）5钱，代赭石（水飞）5钱，干莲心（微炒）2钱半，木通（头末）5钱，灯草3钱，桑螵蛸（炒，或炙酥）2钱半，麝香（另研）2钱半，血竭（另研）5钱（以上同川山甲捣），温肭脐1对（酒浸，炙酥），蛤蚧1对（去头足，炙酥），火杴草（酒蜜洒，9蒸9晒）5钱，晚蚕蛾5钱，蜻蜓5钱。
出处：《杂类名方》。
主治：水火不济，心有所感，白浊遗精，虚败不禁。
加减：None
功效：实骨髓，养精神，水保遐龄。
用法用量：每服2-3丸，空心温酒送下，服讫干物压之。
制备方法：上为细末，于辰火日合，醋浸蒸饼为丸，如樱桃大。
临床应用：None
各家论述：None
用药禁忌：忌猪、羊血，蒜、骑马。
附注：按：方中干胭脂用量原缺。《奇效良方有山药、破故纸、地龙、茯神，无代赭石、火杴草、晚蚕蛾、蜻蜓。
