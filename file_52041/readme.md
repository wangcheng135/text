序号：52056
名称：大沉香丸
组成：天台乌药2斤半，白芷2斤半，甘松（洗，晒）2斤半，甘草2斤半，姜黄（去皮）20两，檀香20两，干姜（炮）20两，肉桂（去粗皮）20两，白豆蔻（去皮）10两，沉香20两，香附子（去毛）5斤。
出处：《局方》卷三。
主治：一切冷气攻心腹刺痛，胸膈噎塞，呕吐痰水，噫气吞酸，口苦舌涩，不思饮食；膀胱、肾间冷气攻冲，腰背拘急，脐腹绞痛，手足逆冷，小便滑数；又治卒暴心痛，霍乱吐利，疝瘕气痛，妇人血气刺痛。
加减：None
功效：None
用法用量：每服1丸，嚼破，炒生姜盐汤下，元气发动，炒茴香热酒下，空心、食前服。
制备方法：上为末，炼蜜搜和，每1两作20丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
