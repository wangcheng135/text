序号：51767
名称：钟乳丸
组成：钟乳粉2两，薯蓣1两，续断1两，桂心1两，肉苁蓉1两（酒浸1宿，刮去皴皮，炙干），附子1两（炮裂，去皮脐），牛膝1两（去苗），萆薢1两（锉），槟榔1两半，石斛1两（去根，锉），覆盆子1两，五味子1两，菟丝子2两（酒浸3日，晒干，别捣为末），山茱萸1两，蛇床子1两，狗脊2两，杜仲2两（去粗皮，炙微黄，锉），巴戟1两，熟干地黄1两。
出处：《圣惠》卷四十四。
主治：肾脏虚惫，风冷所侵，致腰间久痛，行立不得。虚劳少气，羸瘦无力。
加减：None
功效：None
用法用量：每服30丸，空腹以温酒送下，晚食前再服。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：钟乳粉丸（《圣济总录》卷八十九）。
