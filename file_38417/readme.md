序号：38427
名称：治中结阳丸
组成：赤石脂5分，吴茱萸2分，干姜1分，附子1分，当归1分，厚朴1分，白术1分，木兰皮1分，白头翁1分，黄连1分，黄柏1分，石榴皮1分。
出处：《千金》卷十五。
主治：小儿冷滞，下赤白青色如鱼脑，脱肛出，积日腹痛，经时不断。
加减：None
功效：None
用法用量：3岁儿每服5丸，3岁以上每服10丸，10岁以上20丸。暴下者，服少许便愈，积下者尽1剂，更合之。
制备方法：上为末，炼蜜为丸，如大豆大。
临床应用：None
各家论述：《千金方衍义》：此主积冷滞下如鱼脑，用白头翁汤中三味加干姜，附子以破沍寒，乃热为寒导之奥旨；茱萸佐当归和血；石脂佐榴皮固脱；木兰皮佐厚朴温散滞气；白术佐干姜助脾之阳。
用药禁忌：None
附注：None
