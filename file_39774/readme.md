序号：39784
名称：经验黄鸡煎丸
组成：柴胡1两，知母1两，秦艽（洗净）1两，川楝肉（炒）1两，宣黄连1两，胡黄连半两，芦荟半两，鹤虱半两，芜荑半两，槟榔半两，丹参半两，川芎半两，神曲（炒）半两，麦糵（炒）半两，青皮半两，五灵脂半两，使君子肉1两半，水银粉1钱，麻子5两，黑豆5升。
出处：《婴童百问》卷八。
主治：小儿疳劳，骨蒸潮热，盗汗瘦弱，腹急面黄，食不生肌肉，日夜啼，多渴少食。
加减：None
功效：None
用法用量：2岁20丸，米汤送下，不拘时候。如15岁，温酒送下。
制备方法：上为末，黄雄鸡1口．约斤余，笼之，专以火麻子饲之，至5日后．去毛令净，于背上开孔，去肠肚净，拭干，令前药入鸡腹内，以线缝之，小甑先以黑面餔甑底厚3寸，安鸡在甑中，四旁将黑豆围裹，上以黑豆盖之，自日出蒸至晚，候温冷，取鸡净肉研和得所，如硬，入酒面糊，同药末为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：黄鸡煎丸（《医部全录》卷四四四）。
