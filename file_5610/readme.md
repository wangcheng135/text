序号：5610
名称：必胜丸
组成：鲫鱼1个（去肠肚并子，入雄黄1粒鸡子大、硇砂1钱，在腹内，仰安鱼于炭火上烧烟尽，取出以全蜈蚣1条、蓬术半两、栀子5个、皂角2挺并烧、蓖麻子5个去皮，灯上烧，更用黄明胶三文、皂角2挺去皮，酥炙）。
出处：《三因》卷十五。
主治：瘰疬，不以年深日近，及脑后两边有小结，连后数个，兼痨瘵腹内有块。
加减：None
功效：None
用法用量：别用皂角2挺，去皮，捶碎，以水3碗，揉汁去滓，煮精羊肉4两烂软，入轻粉5厘，男子乳汁半两，同研成膏，和药末，丸如绿豆大，朱砂为衣。每服10丸，侵晨温酒送下，1日1次。至晚，下肉疙瘩子。若项有5个，则以5服药取之，视其所生多少，以为服数，既可更进数服。如热毒疮疖未有头脑者，1服亦须消散。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
