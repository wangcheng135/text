序号：41947
名称：青龙丸
组成：马前子（即番木鳖，制法照小金丹式）4两，山甲片（炒黄色为度）1两2钱，白僵蚕（炒断丝，研末）1两2钱。
出处：《集验良方拔萃》卷一。
主治：一切疔疮肿毒，并跌仆闪肭，伤筋挛痛，贴骨痈疽；兼治男妇大小颈项瘰疬，及乳串结核、痰气凝滞、硬块成毒，小儿痘后发痈。
加减：None
功效：None
用法用量：每服5分，量人虚实酌减，老年觉气血衰者，此丸口服4分，妇人新产半月以内者只服4分，如过满月者服5分。男妇瘰疬痰毒，夏枯草汤送下，或酒亦可。小儿周岁以内者服9丸，周岁以外者服11丸，3岁者服15丸，4-5岁者服19丸，5-6岁者服21丸，8-9岁者服23丸，10岁以上者服3分，15岁以上者服4分，20岁者照大人服法。如小儿不能吞送，以开水或甜酒调化送下。临睡时按部位用引经药煎汤送下。盖暖睡，勿冒风。如若冒风，觉周身麻木抽掣，甚则发抖，不必惊慌，过片刻即安。毒初起者，1-2服即消散，已成脓者，服此自能出毒，不必咬头开刀，诚外科家第一妙方也。外加煎引：头面，羌活、川芎各5分；肩背，角刺尖5分；两臂，桂枝5分；胸腹，枳壳5分；两肋，柴胡5分；腰间，杜仲5分；两足膝，牛膝、木瓜各5分；咽颈，桔梗、甘草各5分；跌仆挛筋，当归、红花各5分酒煎。
制备方法：上为末，黄米饭为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
