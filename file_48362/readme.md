序号：48377
名称：神效托里散
组成：忍冬草（去梗）5两，黄耆（去芦）5两，当归1两2钱，甘草（炙）8两。
出处：《局方》卷八（宝庆新增方）。
主治：痈疽发背，肠痈，奶痈，无名肿毒，焮作疼痛，憎寒壮热，类若伤寒，不问老幼虚人，并皆治之。
加减：None
功效：None
用法用量：每服2钱，酒1盏半，煎至1盏，若病在上食后服，病在下食前服。少倾再进第2服，留滓外敷，未成脓者内消，已成脓者即溃。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：神效散（《朱氏集验方》卷十二）、托里散（《医学正传》卷六引《疮疡集》）、神功托里散（《外科发挥》卷二）、金银花散（《外科发挥》卷五）、四妙汤（《医宗说约》卷六）、四金刚（《串雅内编》卷二）。
