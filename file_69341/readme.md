序号：69357
名称：黄雌鸡汤
组成：小黄雌鸡1只（去头、足、翅、羽、肠胃，洗，切），当归半两（锉，微炒），白术半两，熟干地黄半两，桂心半两，黄耆半两（锉）。
出处：《圣惠》卷八十一。
主治：女子产后虚羸，腹痛。
加减：None
功效：None
用法用量：先以水7升，煮鸡至3升，每服4钱，以鸡汁1中盏，煎至6分，去滓温服，每日3次。
制备方法：上为散。
临床应用：None
各家论述：《医略六书》：熟地补阴滋血以资经脉，黄耆补气益卫以健中州，白术健脾生血，当归养血荣经，桂心暖营血以滋血海，雌鸡滋血室以荣冲任也。蜜丸酒下，使血气内充，则血室滋荣而腹痛无不退，羸瘦无不复元矣。
用药禁忌：None
附注：本方改为丸剂，名“黄雌鸡丸”（见《医略六书》）。
