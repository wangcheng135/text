序号：31275
名称：沉香和血丸
组成：当归（酒浸）1两，乌药（酒炒）1两，沉香（不见火）1两，玄胡索（炒）1两，白芷（酒炒）2钱，苍术（炒）2钱，枳实（炒）2钱，干姜（炮）2钱，小茴香（炒）2钱，川椒（炒，去目）2钱，乳香（研）2钱，没药（研）2钱，牡丹皮2钱，澄茄1钱，白芍药2两，艾叶4两（醋浸1宿，煮干为末，入前药）。
出处：《袖珍》卷四。
主治：虚羸，血气冲任脉不调，气不升降，饮食不消，聚为痰饮，头目昏眩，四肢倦怠，百节酸疼，子宫久冷。
加减：None
功效：None
用法用量：每服50丸，空心以醋汤送下；米饮亦可。
制备方法：上为末，好米醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
