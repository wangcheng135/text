序号：9610
名称：仙方解霉生肌定痛散
组成：黄连1两，黄柏4两，木贼1两，防风1两，苦参4两，羌活、独活。
出处：《秘传外科方》。
主治：痈疽，发背，乳痈，人面、外臁、金刀诸般恶疮疖肿毒。
加减：有虫，加轻粉、苦参、百药煎、雄黄；水不干，加螵蛸（去皮）、无名异（煅）、蓼叶（烧灰）。
功效：None
用法用量：敷中间。
制备方法：上(口父)咀，大瓦盆盛水，入前药煎汤，以炉甘石10斤，用炭火煅通红，钳出在药汤内，不同石片大小者，皆要以酥、内青色方好，如石不酥，再将前药滓煎汤，再以石淬酥方了。却将瓦盆盖在地上一昼夜，收火毒，将起候干，研为极细末，此石10斤，用石膏20斤，别研极细，伴匀，和后药：赤石脂（煅）、谷丹（炒，此2味同前打和）、南木香、血竭、降真节、乳香、没药、白芷、黄连、黄柏、白蔹各等分，龙骨（煅）、朱砂、何首乌，上各为细末。与前药拌和用之。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中羌活、独活用量原缺。
