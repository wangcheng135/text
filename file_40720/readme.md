序号：40733
名称：败蒲散
组成：败蒲1两半（烧灰），牡丹1两，当归1两（锉，微炒），芎1两，赤芍药1两，豉心1合，蒲黄半两，生干地黄1两，川朴消1两，陈橘皮半两（汤浸，去白瓤，焙），桃仁1两（汤浸，去皮尖双仁，麸炒微黄）。
出处：《圣惠》卷六十七。
主治：伤折内损，瘀血不散。妇人因血涩内瘀不散，疼痛者。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：败蒲汤（《圣济总录》卷一四四）。《圣济总录》本方用法：每服三钱匕，水一盏，煎至七分，去滓，入地黄汁一合，朴消一钱匕，温服，一日三次。
