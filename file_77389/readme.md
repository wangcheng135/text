序号：77405
名称：跌打丸
组成：制乳香6钱，土鳖虫4钱，防风4钱，当归尾4钱，丹皮4钱，木通4钱，汉三七8钱，甜瓜子4钱，骨碎补4钱，生白芍6钱，甘草6钱，枳实（麸炒）4钱，苏木6钱，桃仁（去皮）4钱，色姜黄3钱，制没药6钱，血竭6钱，刘寄奴4钱，续断4两，赤芍8钱，桔梗4钱，红花6钱，三棱（醋制）6钱，煅自然铜（醋制）4钱。
出处：《全国中药成药处方集》（天津方）。
主治：跌打损伤，闪腰岔气，伤筋动骨，铁木打伤，青紫红肿，疼痛不止。
加减：None
功效：散瘀活血，消肿止痛。
用法用量：每服1丸，黄酒送下，白开水亦可。
制备方法：上为细末，炼蜜为丸，每丸1钱重，蜡皮或蜡纸筒封固。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
