序号：76599
名称：舒筋保安散
组成：干木瓜1两，萆薢1两，五灵脂1两，牛膝（酒浸）1两，天麻1两，续断1两，白僵蚕（炒去丝）1两，松节1两，白芍药1两，乌药（去木）1两，威灵仙1两，黄耆1两，川当归1两，防风（去叉）1两，虎骨1两。
出处：《三因》卷一。
主治：左瘫右痪，筋脉拘挛踡，身体不遂，脚腿少力；干涩脚气，及湿滞经络，久不能立。
加减：None
功效：宣导诸气。
用法用量：每服2钱，用浸药酒半盏调下；吃酒尽，用米汤调下。
制备方法：用无灰酒1斗，浸上件药27日，紧封扎，日数足，取药焙干，捣为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：舒筋保肝散（《医学从众录》卷四）。
