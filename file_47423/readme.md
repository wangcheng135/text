序号：47438
名称：祛疠神效丸
组成：丢子肉15斤，防风2斤半（去芦），白蒺藜2斤半，荆芥2斤半，银柴胡6两，胡黄连6两，草胡麻2斤半，当归2斤半（酒浸），芜荑2斤半，木鳖子15两（去壳），薄荷1斤。
出处：《玉案》卷二。
主治：大麻风。
加减：若脾经受病，加白术5两；肺经受病，加黄芩5两；胃经受病，加厚朴5两；肝经受病，加连翘5两；心经受病，加山栀仁、胡黄连各8两；肾经受病，加破故纸5两；五脏受病，加苍术4两，甘草2两；六腑受病，加威灵仙4两，续断4两，何首乌8两；春则气暖融和，加连翘1斤；夏则火旺烦躁，加胡黄连8两，薄荷8两；秋则乍寒乍暖，多生雾露，加苍术8两，白术8两；冬则严寒冰冻，加乌药1斤；面生浮肿，加白芷5两，续断8两；遍身浮肿，加苍术8两；脚底肿，加牛膝8两；手踡挛，加威灵仙8两；骨节疼痛，加虎骨1斤。凡遇是病，依前法加减，其验如神。
功效：None
用法用量：每服5钱，1日3次。
制备方法：上为末，以酒为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
