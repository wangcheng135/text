序号：32849
名称：羌活石膏散
组成：羌活、石膏、黄芩、藁本、蜜蒙花、木贼、白芷、萝卜子、细辛、麻仁、川芎、苍术、菊花、荆芥、甘草各等分。
出处：《医学入门》卷七。
主治：远年近日内外翳障，风热昏暗，拳毛倒睫，一切眼疾，及头风。
加减：None
功效：None
用法用量：每服2钱，蜜汤调服，日3次；或加当归、枸杞、山栀、连翘、柴胡、薄荷、防风、天麻、桔梗各等分，为丸服。
制备方法：上为末。
临床应用：None
各家论述：羌活治热脑头风，石膏、黄芩洗心退热，藁本治偏头痛，蜜蒙花治羞明，木贼退翳障，白芷清头目，萝卜子、细辛起倒睫，麻仁起拳毛，川芎治头风，苍术开郁行气，菊花明目去风，荆芥治目中生疮，甘草和药。
用药禁忌：None
附注：None
