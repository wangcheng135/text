序号：38183
名称：松脂散
组成：松脂5两（桑柴灰汁1斗，煮至3升巳来，倾入冷水内放凝；别用浆水1斗，再煮至3升巳来，又倾入冷水内放凝，又用清水1斗，煮至5升巳来，又倾入冷水内放凝，取出，干即同入药用），虎胫骨3两（涂酥，炙令黄），天雄3两（炮裂，去皮脐），牛膝3两（去苗），酸枣仁2两（微炒），薏苡仁3两，羌活3两，白附子2两（炮裂），桂心1两，当归1两（锉，微炒），没药2两，麝香1分（细研）。
出处：《圣惠》卷二十三。
主治：历节风，筋骨肢节疼痛久不愈。
加减：None
功效：None
用法用量：每服2钱，食前以温酒调下。
制备方法：上为细散，研令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
