序号：60638
名称：通变大柴胡汤
组成：柴胡3钱，薄荷3钱，知母4钱，大黄4钱。
出处：《衷中参西》卷五。
主治：伤寒温病，表证未罢，大便已实者。
加减：若治伤寒，以防风易薄荷。
功效：None
用法用量：None
制备方法：None
临床应用：伤寒一人，年20余，伤寒6-7日，头疼恶寒，心中发热，咳吐粘涎，至暮尤寒热交作，兼眩晕，心中之热亦甚，其脉浮弦，重按有力，大便5日未行。投以此汤，加生石膏6钱、芒硝4钱，下大便2次，上半身微见汗，诸病皆见轻，惟心中犹觉发热，脉象不若从前之浮弦，而重按仍有力，拟投以白虎加人参汤，恐当下后，易作滑泻，遂以生山药代粳米，连服2剂而愈。
各家论述：方中用防风、薄荷以散表邪，所以防邪之内陷；用柴胡以升之，所以防邪之下陷也。
用药禁忌：None
附注：None
