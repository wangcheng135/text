序号：44760
名称：思仙丸
组成：思仙木2两，干蝎半两，五加皮1两半，防风1两半，萆薢1两半，天麻1两半，薏苡仁1两半，续断1两半，白术1两半，羌活1两半，牛膝1两半，生干地黄1两半。
出处：《鸡峰》卷四。
主治：肝肾风虚弱，腿膝痠疼，不可履地，风湿毒流注脚气，行动不得，小便余沥，里急后重。
加减：老人，加附子1两。
功效：久服补五脏内伤不足，调中益气，凉血，坚筋骨，轻身。
用法用量：每服40-50丸，空心、食前温酒、盐汤任下。如膏稠，少添酒煮面糊为丸，1日3次。
制备方法：上为细末，宣州木瓜（去瓤、皮、子）半斤，先蒸过，切作片子，以好酒2升化青盐3两，同盐酒煮烂为膏，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
