序号：28444
名称：芫根白皮丸
组成：芫花根白皮（锉碎，炒干）1两1分，半夏（汤洗5遍，炒干）1两1分，射干1两1分，百部1两1分，五味子（拣净）1两1分，干姜（炮裂）1两，紫菀（去苗土）1两，款冬花（去萼）1两，白茯苓（去黑皮）1两，皂荚（酥炙，去皮子）1两，细辛（去苗叶）1两，贝母（去心，微炒）1两。
出处：《圣济总录》卷六十五。
主治：久患呷嗽，喉中作声。
加减：None
功效：None
用法用量：每服3丸，空腹以粥饮送下，渐加至5丸，以知为度。如泻多，用防风甘草汤解之。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
