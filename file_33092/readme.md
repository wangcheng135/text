序号：33101
名称：万应至宝膏
组成：松香2斤半，沥青2斤半，麻油1斤（随时加减）。
出处：《医方类聚》卷一七七引《新效方》。
主治：痈疽，诸发肿毒疮疖，风寒湿气，麻木疼痛，打扑损伤，闪肭跌磕，坠高落马，筋疼骨痛，肌肤青肿。
加减：None
功效：定痛追脓，生肌长肉，收敛疮口。
用法用量：疮口未破者，剪去当中1窍贴之。
制备方法：上先以麻油煎熟，次下松香、沥青，以草柴慢火煎，柳条不住搅，候沫消，如油之清亮即止，试令软硬得所，布滤入水盆，去滓，众手引拔令色白，可浮水上为度，仍置水中浸顿，侍后药水同煎：天花粉2钱半、贝母2钱半、黄连2钱半、草乌2钱半、黄柏2钱半、大黄2钱半、赤敛2钱半、细辛2钱半、知母5钱、威灵仙5钱、半夏5钱、白及5钱、马蹄香5钱、杏仁5钱、马鞭草5钱，白芷2两、独活4两、桃嫩枝头1两、柳嫩枝头1两、槐嫩枝头1两，惟春月者用。东流水7-8碗浸之，冬6日，夏2日，滤去滓，留清汁，滓再煎，取清汁，共入松香、沥青、膏子肉，同煎至7-8分干，却下生姜汁、葱汁各半碗，米醋小半碗，巴豆49粒、蓖麻子49粒，研碎，同煎至10分干，方以下项21味药：木香3钱、沉香3钱、血竭3钱、当归3钱、牛膝3钱、犀角3钱、没药3钱、天麻3钱、虎骨（酥炙）3钱、降真3钱、马勃3钱、白虎蛇（或乌梢蛇，酒炙）3钱、南星5钱、生地黄5钱、官桂5钱、五灵脂（炒）5钱、乳香5钱、木鳖子5钱、自然铜（煅、醋淬）5钱、五加皮1两、蜂房7钱半，上研为细末，细细撒入前膏内，搅匀，停火，入麝香2钱半，再搅匀，瓷器收藏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
