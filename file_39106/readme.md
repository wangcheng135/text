序号：39116
名称：泽泻散
组成：泽泻10分，术10分，麋衔5分。
出处：方出《素问》卷十三，名见《圣济总录》卷十三。
主治：酒风，身热懈惰，汗出如浴，恶风少气。因醉中风，恶风多汗，少气，口干善渴，近衣则身热如火，临食则汗流如浴，骨节懈惰，不欲自劳，名曰漏风。
加减：None
功效：None
用法用量：每服3指撮，饭前服。
制备方法：合为散。
临床应用：None
各家论述：1.《素问》王冰注：术，味苦温平，主治大风，止汗；麋衔，味苦寒平，主治风湿筋痿；泽泻味甘寒平，主治风湿，益气。由此功用，方故先之。饭后药先，谓之后饭。_x000D_
2.《古方选注》：麋衔祛在表之风，泽泻渗在里之湿，白术助脾胃之气以却邪。
用药禁忌：None
附注：薇衔汤（《普济方》卷一一八引《指南方》）、麋衔汤（《三因》卷二）、泽术麋衔散（《张氏医通》卷十五）。
