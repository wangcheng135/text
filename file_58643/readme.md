序号：58658
名称：秘传肥儿丸
组成：白术1两5钱，莲肉1两5钱，山楂1两5钱，白芍5钱，神曲5钱，五谷虫5钱，芡实1两，茯苓1两，泽泻4钱，陈皮4钱，甘草2钱。
出处：《仙拈集》卷三。
主治：小儿肚大青筋，骨瘦毛焦，泻痢疳热。
加减：瘦成疳，加芦荟；腹中泄泻，加肉果；内热口干，大便燥结，加黄连；潮热，加银柴胡；有虫，加使君子肉3钱；肚腹膨胀，大便稀水，肠鸣作声，加槟榔、木香各1钱。
功效：None
用法用量：每服2钱，空心以米饮送下，少加白糖亦可；若是腹痛，不用蜜丸，可作散，瓷器收贮。
制备方法：上为末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：肥儿丸（《良方合璧》）。
