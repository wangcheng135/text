名称：皂荚刺散
组成：皂荚刺1两（炙微黄），乌喙1两（炮裂，去皮脐），茵芋3分，白花蛇2两（酒浸，去皮骨，炙微黄），秦艽3分（去苗），天麻3分，独活3分，白蒺藜3分（微炒，去刺），蛇床子1分，麻黄3分（去根节），莽草3分（微炒），槐子仁3分（微炒），景天花3分，踯躅花3分（酒拌，微炒），枫香3分，枳壳3分（麸炒微黄，去瓤），麝香1分（细研入）。
出处：《圣惠》卷六十九。
