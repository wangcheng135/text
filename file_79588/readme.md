序号：79604
名称：熊胆膏锭
组成：炉甘石6两，黄丹3两，黄连1两，当归2钱，朱砂2钱，硼砂2钱，白丁香1钱，海螵蛸1钱，白矾（生）1钱，轻粉1钱，乳香5分，没药5分，熊胆5分，麝香5分，片脑1钱（除脑、麝外，余各另制细末，称，合和匀，入黄连末、当归末，水调匀，绵绢滤净去滓，入末碾至于万余下，晒干，入麝香碾极嫩罗过，次入片脑碾匀，复罗，却入后膏成剂），黄连半斤，龙胆草2两，防风2两，当归2两，生地黄2两，诃子8枚（去核，研末），蕤仁2钱半，鹅梨4个（取汁），猪胰子2个（用稻草挪洗，去膏膜，干净无油为度，再用布包捣烂入药），冬蜜2两（另熬干酥为度）。
出处：《准绳·类方》卷七。
主治：风热上攻，眼目昏花，眵多(目蔑)泪，眊臊紧涩，痒极难忍，胬肉攀睛，沙涩难开，翳膜覆瞳，目眶岁久赤烂，俗呼为赤瞎；一切眼疾。
加减：None
功效：None
用法用量：每以少许，井水化开，鸭毛蘸点眼，又以热汤泡化洗眼。当以棱针刺目眶外，以泻湿热，如倒睫拳毛，乃内脸眼皮紧，当攀出内睑向外，以棱针刺出血，以泻伏火，使眼皮缓则毛立出，翳膜亦退。
制备方法：黄连以下9味，洗净锉碎，以井水浸于铜器内或瓷器内，春五、夏二、秋三、冬七日，滤去滓，以滓复煎3-4次，取尽药力，以熟密绢开绵纸在上滤过，澄清，去砂土，慢火煎熬，槐、桑、柳枝各49条，长1尺，搅不住手，互换搅尽枝条，待如饴糖相类，入蜜和匀，瓷碗盛，放汤瓶口上，蒸顿成膏，复滤净，滴入水中沉下成珠，可丸为度，待数日出火毒，再熔化，入末和匀，杵为丸锭，阴干，金银箔为衣。
临床应用：None
