序号：36488
名称：和胃膏
组成：人参（去芦）1两，藿香叶（去土）1两，水银1两，枇杷叶（先炙去毛，生姜汁涂，炙令香熟）1两，白茯苓1两，甘草（炙）半两，肉豆蔻（面裹煨熟）半两，硫黄（研细，入铁铫同水银1处拌匀，于火上炒，不住手研如泥，放冷）半两。
出处：《卫生总微》卷十。
主治：小儿哭啼，饮乳气逆噎塞，及胃虚气不升降，胸膈痞满，吐逆不时；反胃。
加减：None
功效：None
用法用量：每1岁儿服梧桐子许，生姜、枣汤化下，量大小加减。若治反胃，服1皂子许。
制备方法：上为末，次将硫黄、水银炒匀入之，再研匀细，炼蜜和膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
