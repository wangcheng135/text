序号：67417
名称：黄升丹
组成：水银1两，火消1两，明矾1两。
出处：《中药成方配本》。
主治：痈疽溃后脓不出者。
加减：None
功效：提脓拔毒。
用法用量：每次少许，掺患处，以膏药盖贴。
制备方法：将火消、明矾研细末，和水银倒在小铁锅内搅匀，将小铁锅炖在炭火上，用细铁筋搅之，徐徐熔化取下（这一操作叫做结胎子）；将大瓷碗合在小铁锅内，用桑皮纸摊成粗条，洒水打潮，塞紧碗边，再用无名异（砂子）盖在上面，只留碗底露外，放上一角通草，以小铜钱压住，将锅放在炭炉上，用适当均匀火力烧之，见碗底上之通草发黄为度，将锅取下，侯冷取去无名异，将大碗揭开，用小刀铲下黄升丹，装入瓶中，研细用之。
临床应用：None
各家论述：None
用药禁忌：不可入口。
附注：None
