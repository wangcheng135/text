序号：48584
名称：神秘浸酒方
组成：何首乌1两，石菖蒲1两，生地黄7钱，明天麻7钱，白附子5钱，白茯苓5钱，苍耳子1两（炒，研细），五灵脂5钱（炒），牛膝7钱，天南星7钱（姜汁炒），二蚕沙5钱（炒），当归7钱，苍术5钱（米泔水浸炒），半夏7钱（姜汁炒），红花5钱，光草乌末5钱，陈皮5钱（去白），防风5钱，汉防己5钱，芍药5钱，甘草3钱，黄柏5钱，木瓜7钱，川芎5钱，桑树上络藤1两。
出处：《遵生八笺》卷十八。
主治：左瘫右痪，半身不遂，口眼歪斜，一切诸风，疼痛不可忍者。
加减：None
功效：None
用法用量：上（口父）咀，以布袋盛，悬入坛内，无灰好酒1斗，瓶口封固，重汤水煮5炷香。不拘时候，饮醉为妙。
