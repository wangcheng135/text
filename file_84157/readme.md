序号：84175
名称：五石汤
组成：紫石英2两，钟乳2两，白石英2两，赤石脂2两，石膏2两，茯苓2两，白术2两，桂心2两，芎2两，甘草2两，薤白6两，人参3两，当归3两，生姜8两，大枣20枚。
出处：《千金》卷三。
主治：产后虚冷七伤，时寒热，体痛乏力，并治百病。
加减：若中风，加葛根、独活各2两；下痢，加龙骨1两。
功效：补肾。
用法用量：以水1斗2升，煮取3升6合，去滓，分6次服。
制备方法：上药五石并为末，余药各(口父)咀。
临床应用：None
各家论述：《千金方衍义》：产后虚冷，多系临蓐血气过伤，不但寒热体痛，必然崩脱不止，乃致清阳下溜，浊阴上逆，故用紫白石英、钟乳、石脂固脱，参、苓、术、草益气，芎、归、桂心调血，石膏、薤白除膈上浊邪，生姜、大枣和营卫寒热。
用药禁忌：None
附注：None
