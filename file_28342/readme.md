序号：28350
名称：芩术汤
组成：子芩3钱，白术（蜜炙）1钱5分，阿胶（炒珠）1钱。
出处：《叶氏女科》卷二。
主治：胎气上逼。
加减：风邪，加干姜、豆豉各1钱；寒，加葱白3茎；热，加天花粉1钱；寒热，加柴胡1钱；项强，加葱白3茎；温热腹痛，加白芍1钱；腹胀，加厚朴（姜制）1钱；下血，加熟艾、地榆各1钱；腰痛，加杜仲（盐水炒）；惊悸，加黄连1钱；烦渴，加麦冬（去心）1钱，乌梅1个；思虑太过，加茯神1钱；痰呕，加旋覆花、川贝母（去心）各1钱，或酌用半夏曲1钱；劳役，加黄耆1钱；气喘，去白术，加香附（制）1钱；便燥，加麻仁1钱；素惯难产，加枳壳（麸炒）、苏叶各1钱；素惯堕胎，加杜仲（盐水炒）1钱；若素血虚，加当归、川芎各2钱。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
