序号：60046
名称：起死轻骨膏
组成：麻黄（去根节）5斤（河水二石熬成膏），桑白皮（土下者佳）2两，川芎2两，白芷2两，苍术（去皮）2两，甘松（只用腿子）2两，苦参3两半。
出处：《幼幼新书》（古籍本）卷十三引《保生信效方》。
主治：中风瘫痪，四肢不随，风痹，及小儿惊风。
加减：None
功效：None
用法用量：每服1丸，温酒1盏，研化顿服之。临卧取汗，5-7日间再服，手足当轻快。小儿惊风量与之；卒中涎潮，分利涎后用之。
制备方法：上为细末，麻黄膏为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：起死神应丹（《儒门事亲》卷十五）。本方方名，原书（人卫本）作“起死轻骨丹”。
