序号：56024
名称：桑寄生散
组成：当归4分，芎藭3分，阿胶2分（炙，临时入），葱白14茎，豉8合，桑寄生4分。
出处：方出《经效产宝》卷上，名见《圣惠》卷七十五。
主治：妊娠胎动不安，烦闷。腹痛闷乱。
加减：None
功效：None
用法用量：上用水2升，煎取8合，下阿胶，分2次空腹服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：寄生葱豉汤（《医方类聚》卷二二四引《胎产救急方》）、当归散（《普济方》卷三四二）。《圣惠》本方用桑寄生一两，当归一两（锉，微炒），芎藭三分，阿胶二分（捣碎，炒令黄燥）。上为散，每服四钱，以水一中盏，入豉五十粒，葱白七寸，煎至六分，去滓，稍热服，不拘时候。
