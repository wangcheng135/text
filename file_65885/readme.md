序号：65901
名称：萆薢酒
组成：萆薢1两，防风1两（去叉），菟丝子1两，杜仲1两（去粗皮，锉，炒），黄耆1两，菊花1两，天雄1两（炮裂，去皮脐），石斛1两（去根），生干地黄1两（焙），地骨皮1两，续断1两，金牙1两（煅，醋淬），石南1两，肉苁蓉1两（酒浸，切，焙），蜀椒（去目及闭口者，炒出汗）1两。
出处：《圣济总录》卷十九。
主治：血痹及五脏六腑，皮肤骨髓，肌肉筋脉等疾，不问新久。
加减：None
功效：None
用法用量：每日任性服，候减1升，即旋添酒1升，药力薄即别制。年老者亦可服。
制备方法：上(口父)咀，如麻豆大。夹绢囊盛贮，以无灰酒5升浸14日。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
