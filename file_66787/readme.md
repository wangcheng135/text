序号：66803
名称：麻子仁丸
组成：麻子仁2升，芍药半斤，枳实半斤，（炙），大黄1斤（去皮），厚朴1尺（炙，去皮），杏仁1升（去皮尖，熬，别作脂）。
出处：《伤寒论》。
主治：胃强脾弱，津亏便秘。
加减：None
功效：破气消积，滋润大肠，健胃通便。
用法用量：饮服10丸，每日3次，渐加，以知为度。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：老年性精神病：岳&times;&times;，男，66岁，1974年10月25日诊治。久有心烦失眠之症，常见头晕目眩。近1年来，大便干结，小便频数，时见神志失常，骂詈不休。经某院诊为老年性更衣性精神病，予以清热泻火安神之剂，病情稍有好转，旋即如故。今且大便干结已结五日，口苦心烦，急躁易怒，胸胁痞闷，舌红少津，边有瘀斑，苔薄黄，脉弦细。此津液不足，大肠干燥，肝胆失于条达，肺炎宣降，瘀热上犯，上蒙清窍所致。治直泻火逐瘀，润燥滑肠。方用：大黄（后下）9g，杏仁、白芍、火麻仁、枳实、厚朴各15g，蜂蜜60g，冲服。服3剂，泻下坚硬黑晦如煤之便，烦躁减轻，神识清楚，继服2剂，又泻3次，诸症好转，用上方改汤为丸，调治而愈。
各家论述：1.《伤寒论讲义》：本方是小承气汤加麻仁、杏仁、芍药而组成。取麻仁润肠滋燥通利大便为主药，配以杏仁润肺沉降，使气下行，并具有润肠道、通大便的作用。芍药和营而缓急。大黄、枳、朴泄热去实，行气导滞。以蜜和丸，渐加，以知为度取其缓缓润下之义。_x000D_
2.《方剂学》：方中麻子仁润肠通便，为主药；辅以杏仁降气润肠，芍药养阴和里；佐以枳实破结，厚朴除满，大黄通下；使以蜂蜜润燥滑肠，合而为丸，具有润肠、通便、缓下之功。
用药禁忌：气虚年老者，体弱而大便溏泄者，及孕妇、产妇忌服。忌食辛辣、油腻等物。
附注：麻仁丸（《外台》卷十八）、脾约麻仁丸（《局方》卷六）、脾约丸（《直指》卷四）、麻仁脾约丸（《治痘全书》卷十四）、麻仁滋脾丸（《全国中药成药处方集》）。
