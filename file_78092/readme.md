序号：78108
名称：癫狂丹
组成：公丁香3钱，沉香3钱，雄黄3钱，青皮3钱，黄芩3钱，胡黄连3钱，乳香2钱5分，麝香2钱5分，熊胆2钱5分，檀香3两，轻粉4钱半，黄连1两，牵牛1两，三棱1两，甘草1两，莪术1两，陈皮1两，雷丸（甘草水浸1宿，酒拌蒸）1两，鹤虱1两，大黄1两半，赤小豆3两，巴豆（去油）7个。
出处：《全国中药成药处方集》（沈阳方）。
主治：癫狂不安，言语失次，悲哭无常，凶狂怒骂，自歌自舞，神经错乱。
加减：None
功效：镇静安神。
用法用量：每服10粒，白开水送下。
制备方法：上为极细末，荞面糊为小丸。
临床应用：None
各家论述：None
用药禁忌：剧药，孕妇忌服。
附注：None
