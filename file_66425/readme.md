序号：66441
名称：寸金塌气丸
组成：陈皮1两，香附子1两，大黄（煨）半两，木香半两，青礞石（煅）半两，斑蝥（用身）3两，白丁香（尾起者）半两，青娘子（用身）1钱，虻虫（用身）2钱，京墨（烧烟尽）3钱，麝香1钱，三棱半两，蓬术半两，巴豆（和壳捶碎，炒）1两，干漆（炒烟尽）半两，干姜1两，槟榔5枚，芫花（醋浸，焙干）3钱，水蛭（炒）。
出处：《普济方》卷一六九引《鲍氏方》。
主治：一切积聚，一切气癥，小肠风气，膀胱气，横梁气，走注气、心脾气，血气胁气，气块，癥瘕，蛊毒，宿食、积饮结胸中，伤寒，夹食伤寒，风漏气。
加减：气块，减水蛭。
功效：None
用法用量：每服1丸，醋汤嚼下。泻三行自止。
制备方法：上为末，醋糊为丸，如皂角子大。
临床应用：None
各家论述：None
用药禁忌：产孕不可服；脏虚人勿服；气虚人减量服。
附注：None
