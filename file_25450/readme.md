序号：25458
名称：八石散
组成：代赭3两，凝水石8两，甘草（炙，锉，别为末）8两，不灰木8两，金星石4两，银星石4两，云母4两，石膏4两，太阴玄精石4两，阳起石2两（别生研）。
出处：《圣济总录》卷二十三。
主治：伤寒阳盛烦躁，及夏月中热发躁。
加减：None
功效：None
用法用量：每服半钱至1钱匕，生姜、蜜水或新汲水调下。
制备方法：上药除阳起石、甘草外，余8味固济瓷罐中，歇口，约1秤，炭火煅赤，频将代赭出，醋中淬5度，去火，候冷取出，湿地上纸衬盆合上盖两宿，捣罗更细研，2-3日后，入阳起石、甘草末拌匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
