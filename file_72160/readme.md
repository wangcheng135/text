序号：72176
名称：犀角散
组成：犀角屑半两，天竺黄（细研）半两，人参（去芦头）半两，卷柏半两，赤箭半两，藁本半两，羌活半两，防风（去芦头）半两，芎半两，桂心半两，独活半两，五加皮半两，黄耆（锉）半两，甘菊花半两，麻黄（去根节）半两，赤芍药半两，细辛半两，当归（锉，微炒）半两，枳壳（麸炒微黄，去瓤）半两，天门冬（去心，焙）半两，苍耳子半两，甘草（炙微赤，锉）半两，牛黄半分（细研入），麝香半分（细研入）。
出处：《圣惠》卷六。
主治：肺脏中风。项强，皮毛焦瘁，口干心烦，头目不利，四肢无力疼痛。
加减：None
功效：None
用法用量：每服1钱，以荆芥、薄荷汤调下，不拘时候。
制备方法：上为细散，入研了药令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
