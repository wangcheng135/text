序号：38079
名称：明睛膏
组成：炉甘石1两（火煅，水飞，第1），熊胆半两（去皮，蜜浸1宿，与蜜同熬，去粗），好蜜半两（铫内溶，去粗，共熊胆浸1宿，炉甘石入白丸半钱，飞，同黄丹入，第2），黄连半两（为细末，当归同入，第2），当归半两（洗净，为末），青盐1钱（择净别研，与硇砂入，第3），硇砂1钱（去土，细研，同青盐入，第3），密陀僧1钱半（生研，滑石同入，第4），滑石半两（别研），朴消1钱（去土），牙消1钱（煅，研细，朴消入，第4），硼砂1钱半（别研，石决明同入，第4），石决明半两（火煅），乳香1钱（用蛤粉为衣，铫内慢火炒，候成块，出火性10分冷，细研，同胆矾入，第5），胆研1钱（生用），麝香半钱（药入，第6），片脑半钱，轻粉半钱（药完备方入）。
出处：《医方类聚》卷六十七引《修月鲁般经》。
主治：None
加减：None
功效：明目。
用法用量：点亦得，洗亦得。
制备方法：上熬成膏子。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
