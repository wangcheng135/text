序号：69161
名称：黄金散
组成：生姜1斤4两（薄切，洗，炒令水气尽，再入米醋2升熬干为度），当归1两，白芍药1两，熟地黄（洗）1两，桂心（去皮）1两，大黄（炮）1两。
出处：《普济方》卷三三八。
主治：妇人产前后诸疾。
加减：None
功效：None
用法用量：妇人产后败血攻心，晕闷欲死，微有气存，细茶、老姜炒令水气尽，用童子小便2盏，同煎100沸，倾去滤去姜滓，调药2大钱，热服立醒；或口噤，以物挑开灌药；如分娩后，产母虽无病，亦依前汤使调药服之，便吃白粥压下，即不生血晕之患；或胎不下者，照前服，即下；产后仍每日进药1-2服，服后，5-7日1服，使逐去恶血，久后百病不生；产前后血气及妇人寻常气疾痛刺不可忍者，并用无灰酒调服。
制备方法：上锉细，都炒干，同前炒姜为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
