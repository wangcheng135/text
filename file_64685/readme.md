序号：64701
名称：硇砂丸
组成：硇砂1分（别研），没药半两（别研），桂半两（去粗皮），当归半两（切焙），乌头半两（去皮脐），大黄（锉炒）半两，干漆1分（炒烟出），青橘皮1分（去白，焙），芫花1分（别捣末），巴豆1分（去皮心膜，出油尽），芎1分，京三棱1分（煨，锉），蓬莪术1分（煨，锉），鳖甲（去裙襴，醋炙）1分。
出处：《圣济总录》卷七十一。
主治：积聚不消，心腹胀满。
加减：None
功效：化气消积。
用法用量：每服3-5丸，茶、酒、生姜汤任下。
制备方法：上药十味为末，用酽醋半升，于铜石器内，下芫花、硇砂、巴豆三味，慢火熬，渐添醋1升，即入十味并没药末，同熬成膏，放冷，别入陈曲末1两半拌和，丸如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
