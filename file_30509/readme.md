序号：30517
名称：还睛丸
组成：菟丝子1两（酒浸3日，晒干，别捣为末），真珠3分（细研），远志半两（去心），防风半两（去芦头），蔓荆子半两，车前子半两，石斛1两（去根，锉），白茯苓1两，玄参半两，人参半两（去芦头），木香半两，决明子半两，地肤子半两，蕤仁半两（汤浸，去赤皮），芎半两，羌活半两，羚羊角屑半两，熟干地黄1两，枸杞子半两，牛膝1两（去苗），薯蓣半两，甘菊花半两，黄耆半两（锉），地骨皮半两，覆盆子3分，兔肝2两（炙微黄）。
出处：《圣惠》卷三十。
主治：虚劳目暗。
加减：None
功效：None
用法用量：每服20丸，食前以温酒送下；清粥饮送下亦得。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌热面、荤辛、生冷。
附注：None
