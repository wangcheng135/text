序号：40622
名称：虎潜丸
组成：黄柏半斤（酒炒），龟版4两（酒炙），知母2两（酒炒），熟地黄2两，陈皮2两，白芍2两，琐阳1两半，虎骨1两（炙），干姜半两（一方加金箔1片，一方用生地黄，一方无干姜）。
出处：《丹溪心法》卷三。
主治：肝肾不足，筋骨痿软；阴虚劳热。痿厥之重者。诸虚不足，腰腿痠痛，行步无力。肾虚精髓衰乏，骨萎足软，行步艰辛。阴虚劳证。肾虚多唾。肾阴不足，精血亏损，骨蒸劳热。
加减：None
功效：壮元阳，滋肾水，养气血。养阴潜阳，强筋壮骨。
用法用量：None
制备方法：上为末，酒糊为丸或粥为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：健步虎潜丸（《饲鹤亭集方》）。《东医宝鉴·杂病篇》有当归。《医学入门》本方用法：上为末，猪脊髓为丸，如梧桐子大。每服五六十丸，空心盐汤送下，干物压之。《杏苑》有“金箔为衣”。
