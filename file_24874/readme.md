序号：24882
名称：如圣胜金锭
组成：硫黄2两（细研），川芎2两，腊茶2两，薄荷2两（去枝梗），川乌2两（炮），消石2两（研），生地黄2两。
出处：《局方》卷七（宝庆新增方）。
主治：急喉闭，缠喉风，飞疡，单双乳蛾，结喉，重舌木舌，腮颔肿痛。不能吞水粥。及冒暑伏热，不省人事，砂淋、热淋．小便出血。
加减：None
功效：分阴阳，去风热，化血为涎，化涎为水。
用法用量：每服先用新汲水灌漱吐出，次嚼生薄荷5-7叶微烂，用药1锭同嚼极烂，以井水咽下，甚者连进3服即愈；重舌腮肿，先服1锭，次以1锭安患处，其病随药便消；治冒暑伏热，不省人事，用生薄荷水调研1锭，灌下即苏；如行路常含1锭，即无伏热之患；口舌生疮，不能合口及食热物，如上法服讫，用水灌漱，嚼薄荷10叶，如泥吐出，再水灌漱，嚼药1锭含口内聚涎裹之，觉涎满方吐出，如此服3锭，便能食酒醋；遇食咸、酸、酢脯、炙煿，喉中生泡，须掐破吐血，方与薄荷数叶，以1锭同嚼，井水吞下；砂淋、热淋．小便出血，同车前草7叶、生姜小块研烂，水调去滓，嚼药1锭，以水送下。此药常常随身备急。小儿只服半锭。
制备方法：上为细末，绞生葱自然汁搜和为锭。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《玉机微义》引作“如圣胜金锭子”。
