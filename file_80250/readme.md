序号：80266
名称：鳖甲丸
组成：鳖甲1个（可重4两，净洗，以醋和黄泥固济背上厚3分令干），京三棱3两，锉（炮，锉），川大黄3两（锉碎，微炒），枳壳3两（麸炒微黄，去瓤），木香2两半，桃仁3两（汤浸，去皮尖双仁，麸炒微黄，细研如膏）。
出处：《圣惠》卷四十八。
主治：肥气。体瘦无力，少思饮食。
加减：None
功效：None
用法用量：每服20丸，空心温酒送下，晚食前再服。
制备方法：上除鳖甲外为末，后泥一风炉子，上开口，可安得鳖甲，取前药末并桃仁膏，纳鳖甲中，用好米醋2升，时时旋取入鳖甲内，以慢火熬令稠，取出药，却将鳖甲净洗，去泥焙干，为末，与前药同和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
