序号：76598
名称：舒筋饮
组成：羌活钱半，当归3钱，乳香2钱，片姜黄钱半，海桐皮3钱（酒炒），米仁5钱（炒），甘草6分。
出处：《医略六书》卷三十。
主治：产后筋脉拘挛，脉浮弦涩者。
加减：None
功效：None
用法用量：水煎，去滓，入姜汁1匙，温服。
制备方法：None
临床应用：None
各家论述：产后血虚挟滞，风湿伤于经络，故筋脉失养，拘挛疼痛不休焉。羌活散太阳之邪，除百节之痛，当归养厥阴之血，能舒拘挛，滴乳香活血止痛，海桐皮除挛祛风，米仁渗湿热以舒筋，姜黄行手臂以除痹，炙草以缓中益胃。水煎，入姜汁，以温散行经，使痹痛分解，能血活经和，而拘挛无不舒，切痛无不解矣。
用药禁忌：None
附注：None
