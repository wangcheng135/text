序号：57149
名称：消疳饼
组成：蛆（夏月取赖蛤蟆百余尺，端午前后取的更佳，去头、足、肠、肚、皮、骨，另放一处。先将肉香油煎熟与儿吃。再将皮、骨、肠、肚以钵头盛，放烈日中，上用稀筛盖之，任苍蝇攻钻生蛆，食骨上肉尽，然后取蛆洗净炒干，用重纸包，灰火内煨焦存性，为末）1两，胡黄连2两，山楂肉（去子净）4两，真芦荟2两，砂仁2两，青皮（去白，麸炒）1两，芜荑1两，槟榔2两，蒿心末1两，西涯木香5钱。
出处：《医便》卷四。
主治：小儿诸疳积。
加减：None
功效：消疳磨积。
用法用量：每日空心食1个，米汤送下。
制备方法：上为末，除滓净1斤，外用陈麦面10斤，沙糖2斤，饧糖1斤，将药面糖和匀，造成饼子，1两重1个。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
