序号：40066
名称：肾沥汤
组成：桔梗8分，桑皮1钱半，甘草5分，条芩1钱半，赤苓1钱半，山栀1钱半，麦冬3钱（去心），紫菀2钱。
出处：《医略六书》卷二十八。
主治：孕妇转胞，脉沉数者。
加减：None
功效：None
用法用量：水煎，去滓温服。
制备方法：None
临床应用：None
各家论述：妊娠胎热熏肺，不能通调水道而胞系了戾，故小便不利，谓之转胞。桔梗开提肺气，桑皮清肃肺金，条芩清肺热以安胎，山栀清肺热以降火，麦冬清润肺气，紫菀温润肺金，赤苓利营以渗水道，生草泻火以正胞系也。水煎温服，使热化气行，则胞系自正而水道清长，胎孕无不自安，何转胞之足患哉！
用药禁忌：None
附注：None
