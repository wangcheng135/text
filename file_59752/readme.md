序号：59767
名称：调经并荣种子丸
组成：鱼鳔2两（蛤粉炒），牡丹皮1两半（酒洗晒干），白芍1两（酒炒），沙菀蒺藜2两（酒洗），续断1两（酒洗晒干），白茯苓1两半，大熟地黄4两（另捣），黄苓1两（酒洗），菟丝子1两半（酒煮），山萸肉2两，杜仲1两（盐水炒），川萆薢1两（白色者，酒洗），山药2两，当归1两（酒洗晒干），车前子2两（酒洗），阿胶1两5钱（蛤粉炒，真者），益母草1两（取嫩尖晒干，酒拌，饭上蒸过，再晒干），香附米1两（童便浸7日后洗净炒）。
出处：《幼科直言》卷六。
主治：None
加减：None
功效：滋水生精，调经种子。
用法用量：每早白滚水吞3钱，空心服后，随宜进饮食，将药压入下部。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
