序号：52271
名称：顺气枳壳丸
组成：枳壳（麸炒，去白）3两，益智仁2两，玄胡2两，雷丸2两，白豆蔻仁2两，木香2两，当归（去芦头，锉，炒）2两，白术2两，半夏（汤洗7次，切，焙干）2两，缩砂仁4两，青皮（用汤浸去白）1两，牵牛20两（微炒，取头末10两），京三棱4两（煨熟，锉碎），蓬莪术4两（煨熟，锉碎）。
出处：《御药院方》卷四。
主治：痃癖，形身瘦弱，及腿脚沉重，不任攻击者。
加减：None
功效：宣通一切凝滞，消化宿食，清利头目，消磨积蕴痃癖，久服令人肥壮美，轻身进饮食。
用法用量：每服30-40丸，诸饮皆可送下，不拘时候。如觉内伤，每服可用70-80至100丸，有益无损。男子、妇人、老幼皆可服之。服1月后觉身轻为验。
制备方法：上为细末，用生姜半斤自然汁，同水打面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：孕妇不可服。
附注：None
