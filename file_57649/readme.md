用法用量：空腹五更温1盏服之。以四肢头面习习为度，勿辄。加减非理，造次必大吐利。欲服散者，以绢筛下之，每服方寸匕。只1服，勿再也。水饮、浆酒皆得服之。丸服者，蜜和服，如梧桐子7丸。惟不得汤服也。须补者，药少服令内消，即是补也。凡服此药法，先多服，令人大吐下利3-5度后，乃少服，方可得益也。
制备方法：上药勿熬炼，振去尘土，即捣，粗筛下药3两，黍米3升，曲末2升，上酒1斗5升；净淘米，以水5升，煮米极熟，停如人肌，下曲末熟搦，次下散，搦如前，次下酒，搅之百遍，贮不津器中，以布片盖之1宿，旦以1净杖子搅30匝。
