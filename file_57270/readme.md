序号：57285
名称：消痞丸
组成：黄连1两，干葛1两，黄芩半两，大黄半两，黄柏半两，栀子半两，薄荷半两，藿香半两，厚朴半两，茴香（炒）半两，木香1分，辣桂1分，青黛1两（研），牵牛2两。
出处：《宣明论》卷四。
主治：积湿热毒，甚者身体面目黄，心胁腹满，呕吐不能饮食，痿弱难以运动，咽嗌不利，肢体焦尪，眩悸膈热，坐卧不宁，心火有余而妄行，上为咳血、衄血，下为大小便血，肠风痔漏，三焦壅滞，热中消渴，传化失常，小儿疳积热。
加减：本自利者，去大黄、牵牛。
功效：None
用法用量：每服10丸，新水送下；温水亦得。
制备方法：上为细末，滴水为丸，如小豆大。小儿丸如麻子大。
临床应用：None
各家论述：None
用药禁忌：忌发热诸物。
附注：None
