序号：83544
名称：乌荆丸
组成：川乌1两（炮，去皮），荆芥穗2两。
出处：《苏沈良方》卷二。
主治：病风挛抽，颐颔宽亸不收；肠风下血。诸风缓纵，手足不随，口眼斜，言语謇涩，眉目瞤动，头昏脑闷，筋脉拘挛，不得屈伸，遍身麻痹，百节疼痛，皮肤瘙痒，搔成疮疡。又治妇人血风，浑身痛痒，头疼眼晕；及肠风脏毒，下血不止。
加减：None
功效：None
用法用量：每服20丸，酒或熟水送下，有疾，食空时，1日3-4服；无疾，早晨1服。
制备方法：上以醋糊为丸，如梧桐子大。
临床应用：颐颔宽亸不收：少府郭监丞，少病风挛搐，颐颔宽亸不收，手承颔，然后能食，服此六七服即瘥，遂长服之，已五十余年。年七十余，强健，须发无白者。
各家论述：None
用药禁忌：None
附注：None
