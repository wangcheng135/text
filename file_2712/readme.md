序号：2712
名称：天门冬煎丸
组成：生天门冬（去心，锉）3升半（捣取汁令尽），枸杞根（净洗，控干，锉）3升半（以水2石5斗，煮取汁1斗3升，澄，去滓），酥3升（炼过），生地黄（净洗，细切）3斗半（捣汁令尽），鹿髓1升，牛髓3升，以上药6味，先将3味汁，以水1石，入大釜中，煮取5斗3升，却入锅中，重煎至1斗，收入铜器中，入后药：桂（去粗皮）2两，白术2两，萎蕤2两，菖蒲2两，远志（去心）2两，泽泻2两，山芋2两，人参2两，石斛（去根）2两，牛膝（酒浸，切，焙）2两，杜仲2两，细辛（去苗叶）2两，蔓荆实（去皮）2两，独活（去芦头）2两，枳壳（去瓤，麸炒）2两，芎2两，黄耆（锉）2两，肉苁蓉（酒浸，切，焙）2两，续断2两，狗脊（去毛）2两，萆薢2两，白芷2两，巴戟天（去心）2两，五加根皮（炙）2两，覆盆子2两，陈橘皮（去白，炒）2两，胡麻仁（炒）2两，大豆卷（生用，焙干）2两，白茯苓（去黑皮）2两，甘草（炙，锉）1两，石南叶1两，柏子仁（炒）1两，蜀椒（去目及闭口，炒出汗）1两，阿胶（炙令燥）10两，鹿角胶（炙令燥）5两，大枣100枚（煮取肉），薏苡仁（炒熟）1升。
出处：《圣济总录》卷八十四。
主治：脚气。
加减：若小便涩，去柏子仁，加秦艽2两半，地黄6两；若阴萎失精，去萎蕤，加五味子2两；若头风，去柏子仁，加菊花、防风各2两；若小便利，阴气弱，去细辛、防风，加山茱萸2两；若腹中冷，去防风，加干姜2两。
功效：None
用法用量：每服20丸，空心温酒送下，加至40丸。
制备方法：上43味，捣罗37味为末，纳前3味煎中，更入鹿髓、牛髓、酥等，重汤煮，搅匀可丸，即丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：女人如有旧患，热者宜服，冷者不宜。
附注：此药宜于腊月修合，经夏至七月下旬即服之。当于舍北入地深六尺，填沙，置药沙中，上加土覆之。
