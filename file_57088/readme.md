序号：57103
名称：消核丸
组成：橘红（盐水洗，略去白）1两，赤茯苓1两（去皮），生甘草节（去皮）4钱，半夏曲（姜汁拌，焙）7钱，片芩（酒拌，炒）8钱，僵蚕（水洗，炒黄）6钱，玄参（酒拌，焙）7钱，牡蛎（火煨，童便淬，另研）7钱，山栀仁（连壳，炒焦）8钱，天花粉7钱，瓜蒌仁7钱（另研），大黄（煨）1两，桔梗（去芦）7钱，连翘（去枝梗）1两。
出处：《回春》卷五。
主治：颈项、耳后结核，三五成簇，不红、不肿、不痛、不成脓者。
加减：None
功效：None
用法用量：每服80-90丸，白汤送下。
制备方法：上为末，汤泡蒸饼为丸，如绿豆大，晒干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
