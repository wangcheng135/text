序号：39266
名称：瓮头春酒
组成：头红花1斤，羊藿1斤（去毛边），白芍2两（酒炒），羯羊油1斤（炒羊藿极黑），杜仲1两（童便浸1宿，炒），苍术（炒）4两，天冬1两，肉苁蓉1两（去鳞甲），牛膝4两，五加皮4两，白茯苓4两，砂仁5钱（炒），故纸1两（炒），人参1两，大附子5钱（制），白蔻仁（炒）5钱，归身2两5钱，川椒5钱（焙去汗，去目），丁香5钱，木香5钱，沉香5钱，枸杞3两，白术（炒）4两，甘草5钱，地骨皮1两（蜜水炒），熟地3两，干菊1两，生地2两。
出处：《奇方类编》卷下。
主治：女子宫冷、白带。
加减：None
功效：壮阳种子，填精补髓。
用法用量：人年40以后用之。
制备方法：上为末，好糯米4斗，淘净，再浸1日夜，去浆澄清，如蒸酒法，糯米为糜，取出候冷，用原淘米浆20斤，入锅温之，加葱白1斤，滚数沸，去葱白候冷，和入糜内，然后拌上细曲末4斤，粗曲末2斤。又将前药和入糜内拌匀，又将羊藿、红花2味各入绢袋，先置瓮底，方将此糜入瓮，按置实，落上面，用火酒10斤盖了。春、秋3日，夏1日，冬5日。后又加火酒80斤，仍将瓮口封固，至14日开缸，木扒打过三四百下，再加圆眼肉2斤，红枣5升，又煮糯米饭3升，候冷，投入瓮内，又从瓮底打起二百下，再过14日，榨出清酒，入坛封口，煮3炷香，埋3日。秋冬不必煮。第二次又用糯米2斗煮饭，拌曲末2斤，火酒50斤，入在糟内封固。过5日打扒，又封。过5日打扒，再过5日上榨。
