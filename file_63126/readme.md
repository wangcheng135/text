序号：63142
名称：大戟散
组成：红芽大戟（炒。主脾）1两，甜葶苈（炒。主肺）1两，黑牵牛（半生半熟。主肾）1两，续随子（炒，去壳。主肝）1两，甘遂（炒。主心）1两。
出处：《普济方》卷一九二引《经验良方》。
主治：脾元虚惫，水气肿满。
加减：None
功效：None
用法用量：观其五脏病证，其病证之药加1两，共2两，并为末。煎灯心汤，五更初服1钱。用药多少，相老少虚实加减。至天明其水下3-5次，其肿即消，却用生葱、姜煮粥止之，隔日服平胃散补贴。如体虚不堪再下者，只用前药面糊为丸，谓之磨化丸，每服20丸，3日1服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
