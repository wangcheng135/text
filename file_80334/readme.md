序号：80350
名称：瀛州学士汤
组成：赤芍药、防风、川山甲、黑山栀、没药、乳香、川黄连、升麻、川贝母、苏薄荷、木通、白芷、皂角刺、甘草、天花粉、当归、川芎、陈皮各等分。
出处：《喉科紫珍集》卷下。
主治：喉痈喉蛾诸症，红肿不消，疼痛难忍；及梅乳诸核，死蛾，一切痄腮，颏痈疮毒，阴疮疳疮，未成脓者。
加减：诸疮痈肿、梅核、死蛾，初服必加大黄，如老人加3钱，壮盛者加4-5钱，空心服之，利5-6次，有痰则痰从下行，有热则清退，有毒即溃，任其自止；后剂去木通、山甲，加桔梗，粘子。
功效：None
用法用量：灯心、淡竹叶为引，水2钟，煎7分服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
