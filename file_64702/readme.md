序号：64718
名称：硇砂丸
组成：硇砂（研）2两，陈橘皮4两（汤浸，去白，焙），桂4两（去粗皮），干姜4两（炮），当归4两（切焙），厚朴4两（去粗皮，生姜汁炙），芎4两，胡椒4两，缩砂4两（去皮），甘草4两（炙，锉），附子（炮裂，去皮脐）4两，白术3两，五味子1两半，阿魏（研）半两，青盐（研）2两。
出处：《圣济总录》卷四十七。
主治：胃气虚冷，不思饮食。
加减：None
功效：None
用法用量：每服20丸，生姜盐汤送下，不拘时候。
制备方法：上为末，用银石锅，入好酒1升，白蜜10两，先下硇砂、阿魏、青盐三味，并好面1两，同煎煮稀稠成糊，入药末和成剂，为丸加梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
