序号：5323
名称：半边散
组成：芫花（醋浸，焙干）3钱，大戟3钱，甘遂3钱，大黄3钱，土狗7枚（5月内取会飞的）。
出处：《本草纲目》卷四十一引《普济方》。
主治：水病。
加减：None
功效：None
用法用量：每服2钱，入土狗末和匀，用淡竹叶、天门冬煎汤调，五更服，候左边退，至第4日服右边，如或未动，只以大黄3钱，煎至一半助之，如更不动，茶清助之。
制备方法：上先以葱捣烂为饼，摊新瓦上，却将土狗安葱上焙干，去翅足嘴，每个剪作2片，分左右成对记之，再焙干为末，欲退左边肿，即以左边7片为末，入前药调服；右边依前4味末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
