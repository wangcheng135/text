序号：9639
名称：牛膝丸
组成：牛膝1两（去苗），黄耆3分（锉），侧子1两（炮裂，去皮脐），羌活1两，人参1两（去芦头），白附子1两（炮裂，去皮脐），肉苁蓉1两（酒浸一宿，锉，去皱皮，炙），防风3分（去芦头），芎藭1两，桂心1两，巴戟1两，干蝎3（半）两（微炒），白茯苓1两，五加皮1两，甘菊花3分，天麻1两，补骨脂1两（微炒），熟干地黄1两，萆薢1两（锉），茵芋1两。
出处：《圣惠》卷三十。
主治：虚劳痿痹，四肢不举，头目昏重，不能饮食，身体乏力疼痛。
加减：None
功效：None
用法用量：每服30丸，食前以暖酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
