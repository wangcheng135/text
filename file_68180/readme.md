序号：68196
名称：黄耆酒
组成：黄耆3两，乌头3两，附子3两，生姜3两，秦艽3两，蜀椒3两，芎3两，独活3两，白术3两，牛膝3两，苁蓉3两，细辛3两，甘草3两，葛根1两半，当归1两半，菖蒲1两半，山茱萸2两，桂心2两，钟乳2两，柏子仁2两，天雄2两，石斛2两，防风2两，大黄1两，石南1两。
出处：《千金》卷七。
主治：风虚脚疼痿弱，气闷不自收摄。
加减：大虚，加苁蓉3两；下痢，加女萎3两；多忘，加菖蒲3两。
功效：None
用法用量：先食服1合，不知，可至5合，日3次。
制备方法：上（口父）咀，无所熬炼，清酒3斗渍之。
临床应用：None
各家论述：《千金方衍义》：黄耆专主风虚，而所见诸病靡不由风虚所致。故用乌、附，用姜、桂，用防、独，用葛根，用芎、归，用大黄，靡不由风虚变证。然酒醴之制，非专一人，虽五脏受邪，靡不由风虚内贼，所以汇次诸药渍之以酒，随宜而用，非比汤液之随证施治也。
用药禁忌：None
附注：None
