序号：52089
名称：大补元煎
组成：熟地5钱，党参3钱，山药2钱，杜仲2钱，枣仁2钱，枸杞2钱，萸肉1钱，炙草2钱，故纸2钱，白术3钱，肉桂2钱，附子1钱。
出处：《验方新编》卷十。
主治：痘疹误服凉药，呕吐泄泻，痘不起发，危在旦夕。
加减：倘2-3剂后泄泻不止，酌加附子，更加龙骨、粟壳各1钱；倘泄泻全止，减去附子，若附子大多则小便闭塞。痘后减去附子，只用肉桂数分，调理数剂，计日可复元。
功效：大补气血。
用法用量：加生姜3大片，好核桃仁3个打碎为引，速宜大剂连进，不可减去附子。与6味回阳饮相间服之，立见奇功。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
