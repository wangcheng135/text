序号：54274
名称：大枣散
组成：芫花（微炒）1分，甘遂（炙）半两，大戟（煨，去皮）1分。
出处：《圣济总录》卷八十。
主治：遍体浮肿，腹胀上气，不得卧，大小便涩。太阳中风，下利，呕逆，短气，不恶寒，热汗出，发作有时，头痛，心下痞硬，引胁下痛。兼及水肿，腹胁胀，酒食积，肠垢积滞，痃癖肾积，蓄热极痛，上气久不已。并风热燥甚，结于下焦，大小便不通。
加减：None
功效：None
用法用量：每服1钱匕，以大枣10枚，水1盏半，煮枣20沸，去枣调药，空心顿服。当利勿止。如此3服后，可服海蛤丸。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：愈后3年，不得食肉、入房。不尔，病必重发。
附注：None
