序号：59055
名称：秦绫锦家二十四味养脾丸
组成：丁香1钱半，沉香1钱半，木香1钱半，附子6钱半（炮，去皮脐，秤），陈皮（去白）半两，大腹皮半两，神曲（炒）半两，白术1两半，大麦芽（炒）1两半，肉桂（去皮，不见火）1两半，厚朴（去皮,姜制）3两，诃子（炮，去核）1两3钱，人参（去芦）4钱，茯苓4钱，缩砂仁8钱，荜澄茄2钱，白附子（炮）2钱，高良姜（油炒）2钱，红豆（去红皮）2钱，胡椒（炒）2钱，荜茇2钱，甘草（炙）2钱，川姜（炮）2钱，生姜14两（切作片，焙干）。
出处：《传信适用方》卷一引黄子耕方。
主治：风冷寒湿邪气，腹胀痞满刺痛，肠鸣泄泻，吐逆吞酸，羸弱困怠无力，不思饮食，一切脾胃之疾。
加减：None
功效：None
用法用量：食前细嚼，沸汤送下。
制备方法：上为细末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：秦家二十四味养胃丸（《医统》卷二十三）。
