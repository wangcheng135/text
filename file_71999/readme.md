序号：72015
名称：小铜镜鼻汤
组成：铜镜鼻10铢（烧末），大黄2两，甘草2两，黄芩2两，芒消2两，干地黄2两，桃仁50枚。
出处：《千金》卷三。
主治：产后余疾，恶露不除，积聚作病，血气结搏，心腹疼痛。亦治遁尸心腹痛，及三十六尸疾。
加减：None
功效：None
用法用量：以酒6升，煮取3升，去滓，纳镜鼻末3分服。
制备方法：None
临床应用：None
各家论述：《干金方衍义》：产后恶露不除，日久而成积聚，虽有大黄、芒硝、干漆、地黄之属，不能消磨坚积，故取铜镜之鼻以磨砺之，深契《本经》主治女子血闭癥瘕之旨。方中药味其势稍缓，或元气委顿，难胜干漆峻攻，故但用桃仁、黄芩、甘草，酒煮以行药力。方后言亦治遁尸心腹痛者，取铜镜鼻镇邪不能遁形之义。
用药禁忌：None
附注：None
