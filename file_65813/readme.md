序号：65829
名称：菩提万应丸
组成：陈皮1两4钱，厚朴（姜汁制）1两4钱，苍术1两4钱，制半夏1两4钱，制香附1两4钱，柴胡1两4钱，薄荷1两4钱，黄芩1两4钱，枳壳1两4钱，山楂2两，麦芽2两，神曲2两，砂仁2两，甘草5钱，藿香5钱。
出处：《医方易简》卷四。
主治：夏、秋一切时症，中暑，霍乱，疟，痢。
加减：None
功效：None
用法用量：随饮服之。一切感冒及瘟疫时症头痛骨痛，咳嗽痰喘，用生姜、葱白煎汤调下；红白痢，用车前子煎汤调下；泄泻，用姜茶汤调下；久泻不止，糯米饮调下；水泻，小便不通，口渴，淡竹叶、灯心煎汤调下；疟疾，用姜汤调下；久疟，必用人参汤调下；霍乱吐泻，用胡椒7粒、绿豆49粒煎汤调下；黄疸，用茵陈汤调下；心胃痛，用摈榔煎汤调下；其余山岚瘴气，水土不服并胸膈饱闷，宿食不消，一切杂症，或在路途无引，俱用清茶开水调下。轻者1服，重者2服。
制备方法：用干荷叶煎汤拌前药，晒干为末，炼蜜为丸，如弹子大，每丸重1钱。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
