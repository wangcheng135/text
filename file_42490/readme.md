序号：42505
名称：青黛散
组成：青黛2两（细研），雄黄（细研）1两，青矾1两，黄矾1两，白矾1两，莨菪子（炒令黑色）1两，附子（生，去皮脐）1两，苦参（锉）1两，甘草（锉）1两，细辛1两，藜芦（去芦头）1两，麝香1分（细研）。
出处：《圣惠》卷三十四。
主治：牙齿急疳，蚀颊骨，疼痛不可忍。下部有痔。
加减：None
功效：None
用法用量：每取1小豆许，点于齿疳上，有汁勿咽。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：青矾散（《圣济总录》卷一七二）。《普济方》本方用法：若痔病，绵裹纳下部中；若下部有虫，浸甘草、苦参各一两，煎汤，和散半钱，灌之便止。
