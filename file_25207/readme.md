序号：25215
名称：当归散
组成：当归（洗）、木香（煨）、赤茯苓、桂心、槟榔、赤芍药、牡丹皮、陈皮、木通、白术（各锉，焙干）各等分。
出处：《三因》卷十四。
主治：水肿，喘息奔急，皮肤溢满，足胫尤甚，两目下肿，腿股间冷，口苦舌干，心腹坚胀，不能正偃，偃则咳嗽，小便不通，梦中虚惊，不能安卧。
加减：脏寒，去槟榔；脐已凸，加大腹皮、木猪苓各1两。
功效：None
用法用量：脚膝头面肿，大小便不快，每服2钱，水1盏，紫苏2叶，淡木瓜1片如指大，同煎8分温服，日3次；如己愈，常服，早晚2次。觉气下，或小便快，是效。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：忌乌鸡肉，咸酸海味物。
附注：None
