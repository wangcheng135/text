序号：52964
名称：香橘饼
组成：橘红4两，青皮4两，厚朴（姜汁炒）4两，青木香4两，山楂肉4两，茯神（去皮木）4两，神曲（炒）4两，麦芽（炒）4两，白术（炒）4两，香附（炒）2两，三棱2两，砂仁（炒）2两，蓬术1两，广木香5钱，甘草2两（炙），人参2两（虚者用）。
出处：《万氏家抄方》卷五。
主治：小儿久泻痢致脾虚脱肛不收，冷热不调，赤白脓血痢疾，小腹疼痛，或禁口不食，日夜无度，里急后重，经久不愈，及疳积下痢，泄泻不止。
加减：None
功效：None
用法用量：空心米汤送下；白痢，生姜饭汤送下。
制备方法：上为末，炼蜜作饼。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
