序号：48171
名称：神应散
组成：牛头角腮1只（酌中者），猪牙皂角7锭，穿山甲49片（或园取，或四方取，或1字取之），猬皮1两，蛇退皮1条。
出处：《儒门事亲》卷十五。
主治：肠风痔漏。
加减：None
功效：None
用法用量：如服药日，先1日临卧细嚼胡桃仁半个如糊，用温醇糯酒1盏送下，不语便睡。至次日交五更服药，验病年月远近，每用3钱至5-7钱，用水半大碗，醇糯酒半大盏，相合热，和药服之，至辰时再服。又法，依前服药，不须用胡桃仁，久病不过7服。
制备方法：上捶碎，盛在小口瓷器内，盐泥固定，日中晒干，瓶口微露出烟，用文武火烧红赤烟微少，取出放冷为细末。
临床应用：None
各家论述：None
用药禁忌：忌油腻、鱼、鳖、鸡、兔、猪、犬等物。
附注：神验散（《普济方》卷二九七引《神效方》）。
