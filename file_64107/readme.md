序号：64123
名称：猪肝丸
组成：豮猪肝1具（去筋膜，切作柳叶片，以醋1升，煎醋令尽），大蒜（煮令熟，去壳，研）2两，乌梅肉（炒干）1两，桂（去粗皮）1两，厚朴（去粗皮，生姜汁炙令紫）2两，干姜（炮）1两，陈橘皮（汤浸，去白，焙）1两，诃黎勒（煨，去核）1两，黄连（去须，炒）2两，当归（切，焙）1两。
出处：《圣济总录》卷七十七。
主治：冷劳气痢久不愈。
加减：None
功效：None
用法用量：每服20丸，空心热面汤送下，日午再服。
制备方法：上10味，除猪肝、蒜外，捣罗为末，将猪肝与蒜细研如面糊，入药和匀，捣为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
