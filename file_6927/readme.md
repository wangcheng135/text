序号：6927
名称：四炒枳壳丸
组成：枳壳4两（米泔浸，去瓤，分4处炒之：1份苍术1两同煮干，炒黄色，去苍术；1份萝卜子1两水同煮干，炒黄，去萝卜子；1份小茴香1两水同煮干，炒黄色，去茴香；1份干漆1两水同煮干，炒黄色，去干漆），香附2两，槟榔1两，玄胡索1两（微炒），三棱2两（同莪术法制），莪术1两（棱、莪1味，用童便1钟浸1宿，次日用完巴豆仁去壳30粒，同水煮干，炒黄色，去豆不用）。
出处：《回春》卷三。
主治：气血凝滞，腹内鼓胀积聚。
加减：None
功效：宽中快膈快气，消导饮食。
用法用量：每服70丸，清米汤送下。
制备方法：上为细末，用苍术、茴香，萝卜子、干漆煮汁，好醋1碗，同面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
