序号：5083
名称：木通散
组成：木通1两（锉），木香3分，诃黎勒皮3分，香薷1两，枳壳半两（麸炒微黄，去瓤）槟榔半两，桑根白皮1两（锉），子芩3分，鸡苏茎叶1两。
出处：《圣惠》卷七十五。
主治：妊娠身体浮肿，心腹胀满，小便不通。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，煎至6分，去滓，每于食前温服。
制备方法：上为粗散。
临床应用：None
各家论述：《医方考》：妊娠气血朝胎，营卫之行涩，故令身体浮肿，四肢胀急，而小便不利也。是方也，紫苏流气于表，桑皮、枳壳、木通、木香、槟榔流气于里，香薷流气中之湿，条芩流气中之热，诃子流气中之液。服药之后，营卫流行，气血健运，则浮肿诸疾可得而皆愈矣。
用药禁忌：None
附注：木通汤（《圣济总录》卷一五七）。
