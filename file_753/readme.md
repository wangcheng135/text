序号：753
名称：六合回生丹
组成：真铅粉1两，轻粉2分5厘，银珠2分5厘，雄黄2分5厘，乳香（蒻上炙黄）2分5厘，没药（蒻上炙黄）2分5厘。
出处：《疮疡经验全书》卷二。
主治：溃心冷瘘，生于心窝，初起则心头如火热，其毒先内溃心包，方出皮肤，令人心神恍惚，盗汗多出，2日皆红，舌如鸡金背，里外俱热；发背、痈疽溃烂，对口疮，臁疮，下疳。
加减：None
功效：拔毒定痛。
用法用量：先用好苦茶洗净疮口，软绢拭干后，剖猪腰子片，用药1-2分掺腰子上，却敷患上，待腰子发热如蒸，良久取去；若疮口出脓，不可手挤，第2日依前法再敷之，第3日亦敷之。恶甚可敷7-9次，疮小只敷1次可愈。猪腰子不发热勿治矣。
制备方法：上药各择真正好者研为极细末。
临床应用：None
各家论述：None
用药禁忌：羊、鱼、鹅、鸡、犬、鸭及发毒菜物俱忌之。
附注：六合夺命散(《准绳·疡医》卷。《准绳·疡医》：若臁疮日久不愈，用黄蜡少加黄丹，化摊纸上，量疮大小，裁其蜡纸，炙热，掺药一二分，粘在蜡纸上面，贴疮，绵帛缚住，任疮出尽恶水即愈。若患下疳，用猪腰子切作宽片，掺药，缚裹疳上，或以尖刀穿开猪腰子，纳药于内，笼套其疳亦良。
