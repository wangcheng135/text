序号：22499
名称：牡丹汤
组成：牡丹皮1两，当归（切，焙）1两，黄芩（去黑心）1两，芎1两，甘草（炙）1两，芍药1两，细辛（去苗叶）1两，桂（去粗皮）1两，人参1两，生干地黄（炒）1两半，大黄（锉炒）2两，水蛭（去翅足，炒）25个，干姜（炮）半两，桃仁（汤浸，去皮尖双仁，麸炒）25个，虻虫（去翅足，炒）25个，黄雌鸡1只（去肠肚，以水8盏，煮取4盏，澄清）。
出处：《圣济总录》卷一五一。
主治：室女月水不通，或天癸过期索未通者。
加减：None
功效：None
用法用量：每服3钱匕，取鸡汁1盏，同煎至7分，去滓下消石末半钱匕，更煎1沸，食前温服，每日1次。
制备方法：上除鸡外，为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
