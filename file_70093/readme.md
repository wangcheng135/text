序号：70109
名称：滋阴百补丸
组成：枸杞（甘州）2两，杜仲（姜炒，断丝）2两，当归（酒洗）2两，南知母（去毛，酒炒）2两，生地（酒洗）2两，熟地（酒洗）2两，人参（去芦）2两，牛膝（酒洗，焙）2两，干山药1两，山茱萸（去核）1两，菟丝子（酒煮）1两，黄柏（酒炒）3两，琐阳（酥炙）1两5，麦冬（去心）1两，天冬（去心）1两。
出处：《便览》卷三。
主治：诸虚百损。
加减：如心神不宁，眩晕恶逆，加菖蒲、茯神、远志，莲肉，姜半夏各1两。
功效：扶阴助阳，健脾胃。
用法用量：每服50-60丸，盐汤或盐酒送下。
制备方法：上各制净，称足分量，为细末；外将好白术1斤，去梗，水洗2-3次，切成片，水7-8碗，熬至2碗，留汁，再将滓用水5碗，熬至2碗，去滓不用，将前后汁4碗，共熬至2碗半，如稀糊，和前药末，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
