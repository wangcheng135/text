序号：25373
名称：竹沥汤
组成：防风1两半，麻黄1两半，芍药1两半，防已1两，桂心1两，黄芩1两，白术1两，附子1两（1本作杏仁40枚），羚羊角、竹沥1升，甘草1两（1本，作葛根2两），人参1两，穹1两，独活1两，升麻1两，生姜2两，石膏2两。
出处：《千金》卷八。
主治：风痱。
加减：若有气者，加橘皮、牛膝、五加皮各1两。
功效：None
用法用量：以水8升，煮减半，加沥煮取2升半，分3服，相去如人行10里更服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌猪肉、冷水、海藻、菘菜、桃、李、生葱、雀肉等。
附注：方中用量不全，《外台》引《千金》用防风、麻黄、芍药各1两半，防已、桂心、黄芩各1两，附子3分，甘草、白术、人参、芎、独活各1两，竹沥1升、羚羊角2两，升麻1两，石膏2两，生姜2两。
