序号：65277
名称：羚羊角汤
组成：羚羊角（镑）1分，芍药1两，枳壳1两（去瓤，麸炒），生干地黄1两（焙），当归1两（切，焙），桂1两（去粗皮），麻黄1两（去根节，煎掠去沫，焙），黄耆1两（锉），五加皮1两（锉），牛膝1两（酒浸，切，焙），独活1两（去芦头），羌活1两（去芦头），附子1两（炮裂，去皮脐），防风（去叉）1两，酸枣仁半两，白僵蚕半两（炒），白附子半两。
出处：《圣济总录》卷一五○。
主治：妇人中风，筋脉挛急，肢体疼痛，行履艰难，神识昏冒，语言不利。
加减：None
功效：None
用法用量：每服3钱匕，以水1盏，加生姜3片，薄荷5叶，煎至7分，去滓温服。
制备方法：上锉，如麻豆。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
