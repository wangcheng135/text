序号：6857
名称：四物调经汤
组成：香附2钱5分，川芎2钱5分，当归2钱5分，白芍2钱5分，熟地2钱5分，柴胡2钱5分，陈皮2钱5分，三棱2钱5分，小茴香2钱5分，莪术2钱5分，白芷2钱5分，黄芩2钱5分，青皮2钱5分，砂仁2钱5分，肉桂2钱5分，甘草2钱5分。
出处：《女科万金方》。
主治：女子15-16岁时，误食生冷，经脉不通，日夜乍生寒热，手足麻痹，饮食少进，头痛，恶心呕吐，腹中忽结一块，冲痛。
加减：None
功效：None
用法用量：分为4帖。每帖加生姜3片，葱3根，红花3分，水2钟，煎1钟，空心热服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：四物调经散（《郑氏家传女科万金方》卷二）。
