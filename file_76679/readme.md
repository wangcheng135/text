名称：舒筋续骨定痛膏
组成：川芎3两，细辛3两，僵蚕3两，赤芍3两，远志3两，熟地3两，苦参3两，赤石脂3两，柳枝3两，甘草3两，连翘3两，地榆3两，山甲（生）3两，栀子3两，荆芥3两，防风3两，槐枝3两，乌药3两，木通3两，榆枝3两，桃枝3两，橘皮3两，天麻3两，川楝子3两，牛膝3两，续断3两，草乌（生）3两，大风子3两，首乌3两，当归3两，苦楝皮3两，大黄3两，杜仲3两，桑枝3两，羌活3两，麻黄3两，生地3两，苍耳子3两，白芷3两，枳壳3两，苍术3两，蒺藜3两，五加皮3两，藁本3两，川乌（生）3两，白蔹3两，灵仙3两，独活3两，青风藤3两，海风藤3两，地风3两，乌梢蛇3两，千年健3两，桑寄生3两，寻骨风3两，老鹳草3两，松节3两，透骨草3两，片姜黄3两，骨碎补3两，小茴香3两，木瓜3两，海桐皮3两，大茴香3两，蜈蚣6两2钱。
出处：《北京市中药成方选集》。
主治：腰腿疼痛，筋骨拘挛，四肢麻木，跌打损伤。
加减：None
功效：散风活血，舒筋止痛。
用法用量：微火化开，贴患处。
制备方法：上药酌予碎断，每48两用香油240两炸枯，去滓过滤，炼至滴水成珠，入黄丹100两搅匀成膏，取出浸入冷水中去火毒后加热溶化，另兑细料，每膏药油400两兑麝香1钱5分，苏合油4钱，樟脑4钱5分，乳香、没药、血竭、丁香各3两2钱5分。共研为细末，搅匀摊贴，每大张油重6钱，中张油重4钱，布光。
临床应用：None
各家论述：None
用药禁忌：孕妇忌贴。
