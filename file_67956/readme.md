序号：67972
名称：黄耆汤
组成：黄耆1两（细锉），麦门冬（去心，焙）1两，熟干地黄3分（焙），人参3分，甘草（炙，锉）3分，白茯苓半两（去黑皮），当归半两（锉，焙），芍药半两，芎半两，桂半两（去粗皮），远志（去心）半两（一方有五味子）。
出处：《普济方》卷二八九。
主治：发背已溃者。痈毒内虚，毒不起化，及溃后诸虚，不能收口。
加减：None
功效：去脓汁，理虚劳，内补。
用法用量：每服3钱，水1盏，加生姜半分（拍碎），大枣2枚（擘破），同煎至5分，去滓，空心温服，日晚再服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：神效黄耆汤（《疡科心得集》卷上）。
