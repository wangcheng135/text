序号：81217
名称：麝香乌龙丸
组成：天麻（去苗）1两，苍术（米泔水浸1宿）1两，白蒺藜（炒去刺）1两，地龙（去土，炒）1两，没药（别研）1两，木鳖子（去壳，麸炒黄色）1两，川芎1两，羌活（去芦头）1两，白僵蚕（炒去丝嘴）1两，五灵脂（炒）1两，防风（去芦头）1两，香白芷1两，乳香（别研）半两，川乌头（炮，去皮脐尖）半两，草乌头（炮，去皮尖）半两，白胶香（别研）半两，全蝎20枚（去毒炒），麝香1钱（别研），脑子1字（别研）。
出处：《杨氏家藏方》卷一。
主治：一切风气攻注，腰背拘急，皮肤瘙痒，遍身麻木，疼痛。或中风口眼歪斜，语涩涎潮，半身不遂，偏枯軃曳。
加减：None
功效：None
用法用量：食后每服15-20丸，茶酒任下。
制备方法：上为细末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
