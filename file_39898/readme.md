序号：39908
名称：肥儿丸
组成：白人参2两，白术1两，青皮6钱，云苓8钱，香附8钱，芦荟8钱，神曲8钱，胡连1两，川朴1两，薏米2两，陈皮6钱，苍术1两，黄连2钱，君子仁1两，当归2两，山楂8钱，内金1两，麦芽8钱，炙草5钱，谷芽1两。
出处：《全国中药成药处方集》（抚顺方）。
主治：胃肠不调，营养缺乏，腹胀青筋，面黄削瘦，嗜食无厌，口臭唇裂，肌瘦发烧，蛔虫盘裹，腹疼呕吐。
加减：None
功效：健胃整肠，驱蛔虫。
用法用量：5岁以上者，每服1丸；5岁以下者，服二分之一。白水送下。
制备方法：上为细末，炼蜜为丸，1钱重。
临床应用：None
各家论述：None
用药禁忌：忌生冷、硬物、鱼腥。
附注：None
