序号：24089
名称：当归导滞汤
组成：当归1两，白芍1两，莱菔子4钱，车前子（炒，研）1钱，枳壳（麸炒）1钱，槟榔1钱，甘草（炙）1钱。
出处：《寒温条辨》卷五。
主治：痢疾。
加减：红痢，加桃仁；热，加黄连2钱，黄芩2钱；日夜无度，或里急后重之甚者，再加大黄、木香；温病后痢疾，加白僵蚕、蝉蜕。
功效：None
用法用量：水煎，入蜜温服。
制备方法：None
临床应用：None
各家论述：此方之奇妙，全在当归、白芍。盖泄泻最忌当归之滑，而痢疾最喜其滑也；白芍味酸，入肝以和木，使木不侵脾土；枳壳、槟榔消逐湿热之邪；车前分利其水湿，而又不耗真阴之气；莱菔辛辣，除热去湿，又能上下通达，消食利气，使气行于血分之中，助归、芍以生新血而荡涤其瘀血也，加百草、蜂蜜以和中，则又无过烈之患，奏功之神奇，实有妙理耳。
用药禁忌：None
附注：None
