序号：32040
名称：万安丸
组成：干蝎（炒）2两，白花蛇4两（酒浸，取肉炙）4两，桃仁（去皮尖双仁，炒，研）4两，肉苁蓉1两半（酒浸，切），槟榔1两半（锉），木香1两半，当归1两半（切，焙），茴香子1两半（炒），羌活1两半（去芦头），芎1两半，天麻1两半，桂1两半（去粗皮），沉香1两半（锉），白附子1两半（炮），阿魏1两半（米醋研用），安息香（研）1两半。
出处：《圣济总录》卷一八六。
主治：虚损。
加减：None
功效：补益，调气，除风。
用法用量：每服温酒或茶嚼1丸。
制备方法：上为末，用蜜1斤，拌和为丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
