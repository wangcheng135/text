序号：40459
名称：虎骨丸
组成：虎骨7钱半，当归7钱半，海桐皮7钱半，参7钱半，木香7钱半，附子7钱半，白花蛇7钱半，桂心7钱半，败龟7钱半，天南星7钱半，干姜1两，没药1两，麝香1两，地龙1两，蔓荆子1两，菊花1两，僵蚕1两，干蝎1两，防风1两，芎1两，羌活2两，天麻1两。
出处：《鸡峰》卷四。
主治：风湿，筋脉缓纵不随，言语謇涩，及中风脚气。
加减：None
功效：去麻痹止风，通利血脉。
用法用量：每服30-50丸，食前温酒送下。
制备方法：上为细末，入麝香、没药研匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方原名虎骨煎，与剂型不符，据《普济方》引《治风验方》改。《普济方》引《治风验方》本方无干姜、地龙，疑脱。
