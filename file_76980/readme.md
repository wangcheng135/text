序号：76996
名称：黑星丹
组成：曾青半两，杨梅青半两，胡椒青半两，桃花石半两，紫石英半两，白石英半两，硫黄1两，雄黄1两，光明砂1两，黄丹1两，定粉1两，水银1两，生金屑1分，山泽银屑1分，珍珠末1分（上药捣罗入水银，研令细，纳瓶子中，密盖头，以盐泥固济，候极干，以慢火养7日后，用大火煅之，候冷取出，用甘草水泼地，以纸衬药，摊于地上，以盆合之，一复时后细研），龙脑1两，牛黄1两，铅霜1两，麝香1两，光明砂1分，雄黄1分，犀角屑1分。
出处：《圣惠》卷二十五。
主治：一切风。
加减：None
功效：None
用法用量：每服1丸，以热酒送下。
制备方法：上为细末，以面糊为丸，如皂荚子大。
临床应用：None
各家论述：None
用药禁忌：忌毒滑鱼肉动风物。
附注：方中生金屑、山泽银屑用量原缺，据《普济方》补。
