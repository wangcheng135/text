序号：55297
名称：桃仁汤
组成：荆芥半分，（庶虫）虫30枚，大黄3两，芎藭3两，蒲黄5两，当归2两，桂心2两，甘草2两，桃仁30枚。
出处：方出《千金》卷二十五，名见《张氏医通》卷十四。
主治：腹中瘀血，痛在腹中不出，满痛短气，大小便不通。
加减：None
功效：None
用法用量：上(口父)咀。以水1斗，煮取3升，分3服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：破瘀诸汤莫如桃仁承气、抵当、下瘀血三方，最为吃紧。此于桃核承气采用其四，抵当汤中采用其二，下瘀血汤全用其三。既有（庶虫）虫，可无藉于虻、蛭，既非坚积，可无取于芒消，加荆芥、芎藭、蒲黄、当归，缓散其血，以和上药之迅也。
用药禁忌：None
附注：None
