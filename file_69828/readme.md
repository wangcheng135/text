序号：69844
名称：小青龙加石膏汤
组成：麻黄3两，芍药3两，桂枝3两，细辛3两，甘草3两，干姜3两，五味子半升，半夏半升，石膏2两。
出处：《金匮》卷上。
主治：肺胀，咳而上气，烦躁而喘，脉浮者，心下有水。胁下痛引缺盆，其人常倚伏。
加减：None
功效：None
用法用量：水1斗，先煮麻黄，去上沫，纳诸药，煮取3升，强人服1升，羸者减之，1日3次。小儿服4合。
制备方法：None
临床应用：None
各家论述：1.《金鉴》引李彣：心下有水，麻黄、桂枝发汗以泄水于外，半夏、干姜，细辛温中以散水于内，芍药、五味子收逆气以平肝，甘草益脾土以制水，加石膏以去烦躁，兼能解肌出汗也。_x000D_
2.《金匮要略论注》：《伤寒论》中寒得风脉而烦躁者，主以青龙汤，故此亦主小青龙；然壅则气必热，故仍加石膏耳。_x000D_
3.《金匮要略心典》：此外邪内饮相搏之证而兼烦躁，则挟有热邪。麻、桂药中，必用石膏，如大青龙之例也。心下寒饮，则非温药不能开而去之，故不用越婢加半夏，而用小青龙加石膏，温寒并进，水热俱捐，于法尤为密矣。
用药禁忌：None
附注：None
