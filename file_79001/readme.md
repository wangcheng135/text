序号：79017
名称：僻巽锭子
组成：牛胆南星7钱，防风3钱，干姜3钱，白附子5钱，牛黄3分，川乌5钱，白芷5钱，薄荷5钱，木香5钱，白术5钱，白茯苓5钱，人参5钱，朱砂1钱，麝香5钱，白僵蚕20个（生用），片脑5分。
出处：《银海精微》卷上。
主治：痰盛，肝胆受风，瞳人开大，眼不收而展缩者；及小儿通睛，瞳人阔大。
加减：None
功效：None
用法用量：小儿急慢惊风，手足搐搦，金银箔磨汤化下1锭；大人破伤风，酒化下3-4锭子。
制备方法：上为细末，冬用蜜2斤，甘草半斤煎作膏，稀稠得宜，将次药末和作锭子，金箔为衣。约1钱1个，或7分1个，夏用麻黄1斤，甘草半斤，用水3-4碗砂锅内煎至1钟之时，入蜜1斤，缓缓熬炼，滴水内成珠，方将前药搜和为丸，即作锭子也。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
