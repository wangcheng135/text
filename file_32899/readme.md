序号：32908
名称：羌活丸
组成：羌活1两，茯神1两，五加皮1两，鹿茸1两半（去毛，涂酥炙微黄），防风3分（去芦头），牛膝1两半（去苗），人参1两（去芦头），远志3分（去苗），薯蓣1两，桂心1两，五味子3分，附子1两（炮裂，去皮脐），酸枣仁1两（微炒），枸杞子3分，山茱萸1两，黄耆1两（锉），熟干地黄1两，羚羊角屑1两。
出处：《圣惠》卷三十。
主治：虚劳痿痹，腰脚不遂，头昏目暗，心烦健忘，身体沉重。
加减：None
功效：None
用法用量：每服30丸，食前以暖酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
