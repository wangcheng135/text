序号：663
名称：井黄煎
组成：虾蟆1枚（去皮骨肠胃，炙焦，捣末，以无灰酒1盏、獖猪胆1枚取汁，熬成膏），诃黎勒皮（炮）1分，使君子（去壳）1分，胡黄连1分，蝉蜕1分，无食子1分，芦荟（研）1分，芜荑1分，熊胆1分，夜明砂1分，丹砂（研）1分，雄黄1分，木香1分，肉豆蔻（去壳）春、夏半分，秋、冬1分，牛黄（研）1钱，麝香（研）1钱。
出处：《普济方》卷三七九引《全婴方》。
主治：小儿诸疳诸痢，食伤气胀，头大体羸，头发作穗，壮热不食，多困，齿烂，鼻疮，丁奚潮热，腹急，骨蒸消瘦，发坚面黄。
加减：None
功效：None
用法用量：每服5-7丸，米饮送下；惊疳，金钱薄荷汤送下；疳疮腹胀，桃仁茴香汤送下；疳虫，东安石榴、苦楝根汤送下。
制备方法：上为末，猪胆膏为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：若挟热而痢者，不可服。
附注：None
