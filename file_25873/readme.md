序号：25881
名称：曲术香棱丸
组成：荆三棱4两，莪术4两，陈皮4两，香附子4两（作1处，用好醋2斤煮1沸时，去醋，焙干），白豆蔻4两，沉香4两，缩砂仁2两，荜澄茄2两，木香2两，川姜2两，乌药2两，枳壳（炒）2两，槟榔2两，干葛2两，半夏曲（炒）2两，神曲2两，干葛花2两。
出处：《普济方》卷一七六引《德生堂方》。
主治：脾积气滞，胸膈痞闷，肚腹胀满，痃痞气块，往来走注，刺痛大肠，酒食呕吐酸水，饮食不思，妇人血气癥瘕，小儿伤食。
加减：None
功效：None
用法用量：每服5-10丸，随意酒、生姜汤送下。
制备方法：上为末，熬萝卜水煮糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
