序号：57533
名称：润膈丸
组成：阿胶（炒燥）1两，熟干地黄（焙）1两，白茯苓（去黑皮）1两，山芋1两，五味子1两，麦门冬（去心，焙）半两，贝母（去心，炒）半两，百部半两，柏子仁（炒，别研）半两，丹参半两，茯神（去木）半两，人参1两，远志（去心）1两，防风（去叉）1两，杜仲（去粗皮，炙，锉）半两。
出处：《圣济总录》卷六十六。
主治：积年咳嗽上气，涎唾稠粘，五心烦躁，不思饮食，心肺留热。
加减：None
功效：None
用法用量：每服1丸，水1盏化破，煎至6分，时时温呷。
制备方法：上为细末，炼蜜和丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
