序号：83862
名称：乌麻酒
组成：乌麻5升。
出处：《千金》卷七。
主治：风虚气满，脚疼痹挛，弱不能行。
加减：None
功效：除风气。充悦强壮。
用法用量：随所能饮之，尽更作。
制备方法：微熬，捣碎，以酒1斗，渍1宿。
临床应用：None
各家论述：None
用药禁忌：None
附注：《圣惠》：乌麻子投水中，掠去浮者，取一斗，九蒸九晒，炒令香，以木杵臼捣细，用疏生绢袋盛之，令极宽转，即结袋头。又以一细绳子接系袋处，悬于瓮中。下无灰酒五斗，以新盆覆瓮，其盆底上钻一小窍，引出系袋绳头，又系于小横木子上，以泥固封，莫使泄气。每日六七度引挽其绳，令药汁入于酒中，满七日药成，乃开瓮，举袋沥汁令尽。冬温夏冷，随性饮之，不令至醉。
