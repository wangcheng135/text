序号：24402
名称：安息香丸
组成：安息香半两（研），肉苁蓉半两（酒浸，切，焙），白附子半两（炮），羌活半两（去芦头），当归3分（切，焙），茴香子3分（炒），木香3分，天麻3分，桂3分（去粗皮），沉香3分，槟榔1两（锉），干蝎（去土）1两，白花蛇（酒浸，去皮骨，炙）2两，芎3分（14味为末），桃仁（去皮尖并双仁，研如膏）3两，阿魏1分（白面裹，发火内炮令黄熟为度，去面，研），硇砂1分（研），硫黄（研）1分。
出处：《圣济总录》卷十二。
主治：风冷及虚风头昏，心胸疰闷，痰唾不下，饮食气胀，腰腹疼痛。
加减：None
功效：None
用法用量：每服1丸，空心、食前以温酒送下嚼服；以姜盐汤送下亦得。
制备方法：先将桃仁、阿魏、硇砂、硫黄，用好酒5升，于银石器内慢火熬成膏，和前药末十四味；如硬，入炼蜜少许，为丸，每1两分作15丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
