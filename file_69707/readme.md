序号：69723
名称：琼浆药酒
组成：人参（去芦）2两，鹿茸（去毛）1两，桂元肉1两，川附片2两，陈皮3两，狗脊（砂烫去毛）4两，枸杞子4两，补骨脂（盐水制）4两，黄精（酒炙）2两，金樱肉（去毛）4两，韭菜子4两，淫羊藿（羊油制）4两，冬虫草2两，怀牛膝（去头）4两，灵芝4两，当归2两，佛手2两，驴肾2两，麻雀头50个（约1两），红糖6斤，红曲8两，白蜜10斤。
出处：《北京市中成药规范》第二册。
主治：体质虚弱，肾衰寒盛，神情倦怠，腰痠腿软，四肢无力，阳痿不举，遗精早泄，妇女白带。
加减：None
功效：补气血，助肾阳。
用法用量：口服，每次3-5钱，每日2-3次。
制备方法：上置洁净容器内，装入回流罐，另取45度白酒100斤，分别放入白酒50斤、30斤、20斤，加入红曲8两兑色，每次均加热至酒沸半小时后，放去药液，将残渣压榨，榨出液与三次浸出液合并，置罐内，混匀，储存1个月，静止滤过，分装即得。本品为橘红色液体，气清香，味辛微苦。每瓶装药酒重10两，上下不超过1％。含乙醇量应为35～39％。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
