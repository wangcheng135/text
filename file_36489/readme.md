序号：36499
名称：和荣汤
组成：白术1钱半，川芎1钱半，南星1钱，半夏1钱，芍药1钱，茯苓1钱，天麻1钱，川归8分，生地黄8分，熟地黄8分，牛膝8分，酸枣仁8分，黄芩8分，橘红8分，羌活6分，防风6分，官桂6分，红花4分，甘草（炙）4分，黄柏3分。
出处：《法律》卷三。
主治：血虚血热，挟痰挟火，经络肌表之间，先已有其病根，后因感冒风寒，或过嗜陈酒膏粱而助痰火，或恼怒而逆肝气，遂成半身不遂，口眼斜，头目眩晕，痰火炽盛，筋骨时疼。
加减：None
功效：补血活血，健脾燥湿消痰，清热疏风，开经络，通腠理。
用法用量：水煎，入竹沥、姜汁，晨服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
