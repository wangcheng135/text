序号：66309
名称：银粉散
组成：水银1两2钱5分，轻粉1两2钱5分，黑铅5钱，寒水石3钱5分，硼砂1两，珍珠1钱，锡6钱，朱砂2钱，官粉1两，夹纸数张。
出处：《全国中药成药处方集》（哈尔滨方）。
主治：下疳破烂，阴茎腐烂，阴唇破溃，疼痛作痒，毒水黄稠，疮口内陷，脓水过多，凡一切花柳梅毒所致，皮破为患者。
加减：None
功效：消毒杀菌，生肌止痛，消炎防腐。
用法用量：用甘草煎汤淋洗患处，拭干，以此药撒之。
制备方法：先将黑铅5钱化开，投入水银2钱5分，研不见星为度。再取好锡6钱化开，投入朱砂2钱，搅炒枯，去朱砂留锡；再化开，入水银1两，合匀倾出；再将轻粉、寒水石、硼砂、珍珠各研细，合黑铅、官粉于一处研匀，将夹纸铺好，卷药成一条，悬于静处，下端点火燃之，至纸尽吹去纸灰，加入轻粉1两，研细即成。
临床应用：None
各家论述：None
用药禁忌：忌食发物、辛辣、酒类。
附注：None
