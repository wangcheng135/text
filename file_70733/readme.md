序号：70749
名称：紫菀散
组成：人参、紫菀、知母、贝母、桔梗、甘草、五味子、茯苓、阿胶。
出处：《卫生宝鉴》引海藏方。
主治：咳嗽，唾中有脓血；虚劳证，肺痿变痈。
加减：None
功效：润肺止嗽。
用法用量：加生姜，水煎服。
制备方法：上为粗末。
临床应用：None
各家论述：1.《医方集解》：此手太阴药也，劳而久嗽，肺虚可知，即有热证，皆虚火也。海藏以保肺为君，故用紫菀、阿胶二药润肺补虚，消痰止嗽；以清火为臣，故用知母、贝母二药辛寒润燥消痰；以参、苓为佐者，扶土所以生金；以甘、桔为使者，载药上行脾肺，桔梗载诸药上行而能清肺，甘草辅人参补脾；五味子滋肾家不足之水，收肺家耗散之金，久嗽者所必收也。_x000D_
2.《血证论》：取参、草、胶、菀以滋补肺阴；又用知母以清其火；五味以敛其气；桔梗、贝母、茯苓以利其痰。火、气、痰三者俱顺，则肺愈受其益，此较保和汤、救肺汤又在不清不浊之间，用方者随其择其。
用药禁忌：None
附注：紫菀汤（《医方集解》）、紫菀饮（《医碥》卷六）。《普济方》本方用量：各等分。《张氏医通》：有麦门冬，无知母。
