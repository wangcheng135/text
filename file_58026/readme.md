序号：58041
名称：益母草留颜方
组成：益母草（5月5日收取，暴令干，烧作灰，取草时勿令根上有土，有土即无效）。
出处：《外台》卷三十二引《近效方》。
主治：面上(黑干)黑色，及老人皮肤兼皱。
加减：None
功效：None
用法用量：旋旋取洗手面，令白如玉；女项颈上黑，但用此药揩洗，并如玉色。
制备方法：烧之时，预以水洒一所地，或泥一炉烧益母草，良久烬无，取斗罗筛此灰，干则以水熟搅和搜之，令极熟团之，如鸡子大作丸，晒极干，取黄土泥作小炉子，于地四边各开一小孔子，生刚炭上下俱着炭，中央着药丸，多火经一炊久，即微微着小烧之，勿令火气绝，绝即不好，经一复时药熟，切不得猛火。若药熔变为瓷巴黄，用之无验，火微即药白色细腻，一复时出之于白瓷器中，以玉槌研（如无玉槌，以鹿角槌亦可），绢筛，又研3日不绝，收取药，以干器中盛，深藏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
