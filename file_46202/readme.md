名称：洞天酥香膏
组成：酒当归5钱，酒熟地5钱，杜仲5钱，制苁蓉5钱，炙黄耆5钱，酒天冬5钱，麦冬5钱，五味子5钱，高丽参5钱，怀牛膝5钱，鹿茸5钱，甜杏仁5钱，蛇床子5钱，酒川断5钱，紫霄花5钱，盐菟丝5钱，虎胫骨5钱，谷精草5钱，制香附5钱，酒生地5钱，制远志5钱，制山甲5钱，木鳖子5钱，男子头发（洗净）5钱，大蛤蚧1对_x000D_
上药如法炮制，锉碎，用香麻油2斤4两，同药入铜锅内，桑枝火熬枯净去滓，再熬至滴水成珠，候热尽，入后香料：_x000D_
水飞松香4两，花龙骨3钱，母丁香3钱，当门子3钱，水飞黄丹1两，赤石脂3钱，制乳香3钱，制没药3钱，摇桂5钱，腰黄3钱，沉香3钱，倭硫黄3钱，大土膏3钱，木香3钱，蟾酥3钱，阳起石3钱。
出处：《千金珍秘方选》。
主治：五劳七伤，淋泻痞证，元虚气喘，瘫痪。
加减：None
功效：通十二经血脉，固本全形，返老还童。
用法用量：摊膏时，将铜器或瓷杯取滚水化糯米糊涂缎为妙。贴脐上或命门，药7-10天1换。
制备方法：上研极细末，用桑、槐、柳条不住手搅匀，盛瓷罐内，浸井水中，或天水缸中7日夜，出尽火气方可用，剪红煅摊膏，计重3钱，衰弱倍之。
