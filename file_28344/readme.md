序号：28352
名称：芩术安胎饮
组成：白术（米泔水浸1宿，去芦，切片，晒干，黄土炒香。如脾脉虚弱细软，缓大无力，外证饮食少进，恶心、呕吐、泄泻，用1钱5分或2钱，若气体强壮，或气郁壅滞，胸腹膨闷胀满作痛，或素有奔豚枳聚上攻者，忌用）、条芩（如脉洪盛有力，素多内热，用1钱5分或2钱，如气体虚寒，脾肺脉弱，呕哕泄泻者，忌用）、当归身（酒洗）1钱5分或2钱（如嗽，有痰喘呕哕泄泻者，忌用：如只有泄泻而无别证，以黄土炒用），带壳砂仁（微炒）5-7分（内热者，3-4分），生知母1钱（素多内热者，或用1钱5分或2钱；如气体虚寒，呕哕泄泻者，忌用），炙甘草3-4分。
出处：《胎产心法》卷上。
主治：胎动不安。
加减：如脉弱虚细，或缓大无力，饮食减少，口不知味，溏薄泄泻者，加人参2钱或1钱5分，炒白术1钱或1钱2分，或2钱，白茯苓1钱，广皮7-8分，炒条芩1钱，去知母；如血虚内热，肝肾脉洪数无力，腰疼，腿膝痠软无力者，加熟地3-5钱或7-8钱，生地2-3钱，酒洗芍药1钱或1钱5分，炒杜仲，酒洗当归1钱或1钱5分，炒续断1钱；如肝肾脉虚细濡弱，腰疼，腿膝麻木冷痛，加熟地3-5钱或7-8钱，川芎8分，制续断肉1钱，盐水炒杜仲1钱5分，酒洗归身1钱5分或2钱，去知母；如胸腹胀闷，加麸炒枳壳7分，制大腹皮8分，醋制香附米7分；如素多郁怒，加苏梗8分或1钱，醋制香附米8分或1钱，小柴胡7-8分，酒洗抚芎7分；如呕哕，加藿香8分或1钱，竹茹6-7分，制透半夏8分，陈皮8分，带壳砂仁4-5分，煨姜3片，去知母；胃寒呕哕，去条芩、知母、竹茹，加制去黄水吴茱萸3分；如虚烦，加去心麦冬1钱，竹茹7分；咳嗽，加去心麦冬1钱，蜜炙桑白皮8分或1钱，去皮尖杏仁8分，前胡1钱，麸炒枳壳8分；如小便淋沥不通，加车前子1钱，赤苓1钱，木通7分，甚者，加滑石1钱5分或2钱；如胎动下血，倍加生知母，纹银1小锭，忌铁器。
功效：安胎。
用法用量：水煎，食远服。
制备方法：None
临床应用：None
