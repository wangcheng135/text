序号：63172
名称：大犀角汤
组成：犀角2两，旋覆花2两，白术2两，桂心2两，防己2两，黄芩2两，香豉1升，橘皮2两，茯苓2两，前胡4两，桑白皮4两，紫苏茎叶1握，大枣10枚。
出处：《千金》卷七。
主治：脚气毒冲心变成水，身体遍肿，闷绝欲死者。脾肾经脚胫痛痹，小腹顽麻，上攻头面，遍身红肿，小便不利，上气喘满，闷绝欲死。
加减：None
功效：None
用法用量：上(口父)咀。以水9升，煮取2升7合，分3次服，每次间隔约人行10里久。取下气为度。若得气下，小便利，脚肿即消，能食。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：犀角汤（《圣济总录》卷八十三）。
