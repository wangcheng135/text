序号：263
名称：五神丸
组成：东方：巴豆（去油）5钱，麝香1分；南方：官桂5钱，朱砂1钱；西方：白矾5钱，白芷2钱；北方：青黛5钱，黑附子3分；中央：硫黄5钱，雄黄1钱。
出处：《摄生众妙方》卷四。
主治：疟疾。
加减：None
功效：None
用法用量：每服用1丸，绵裹，于未发日晚男左女右塞鼻孔中，立效。如用了的药还收藏，再有患者，用醋过，重又绵裹与患人。1丸可治8-9人
制备方法：上于5月5日修制，各另包，按方放，午时取五家棕尖为丸，如梧桐子大。
临床应用：None
各家论述：《医方考》：是方也，位按五方，药按五色，气按五气，味按五味，月按五月，日按五日，粽用五家，此医流而兼阴阳家之识也。故疟邪入于肝则青黛之凉可以清肝，麝香之燥可使直达；疟邪入于肺，则白芷之辛可以泻肺，矾石之腥可以清燥；疟邪入于心，则丹砂之重可以镇心，官桂之焦可以益火；疟邪干于肾。则黑豆甘咸可以益肾，巴豆之腐可以泻邪；疟邪干于脾，则硫黄之温可使健中，雄黄之悍可使辟秽。以疫气无形，由鼻而入，故亦就鼻而塞之，塞其一窍者露其一窍者，围师必缺之道也。
用药禁忌：忌生冷、鱼腥、鸡、羊发物；以上之药，忌火烘。
附注：《医方考》本方无黑附子，有黑豆三十粒。
