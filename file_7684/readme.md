序号：7684
名称：二妙地黄丸
组成：熟地黄8两（微火焙燥），山茱萸4两（去核，酒拌炒），牡丹皮4两（焙），白茯苓3两（焙），怀山药4两（炒黄），汉泽泻3两（淡盐水拌，晒干，炒），川黄柏7钱，熟附子5钱（2味盐、酒同浸1宿，各拣开，黄柏炒褐色，附子焙燥），茅山苍术2两（切大块，米泔水浸透，切片，黑芝麻拌，炒黄）。
出处：《冯氏锦囊》卷十一。
主治：湿热内郁而为便浊。
加减：如湿多热少，用附子7钱，黄柏5钱；如湿少热多，用附子5钱，黄柏7钱。
功效：None
用法用量：每早、晚食前白汤送服3钱。
制备方法：上为细末，用金石斛4两，煎浓汁，入白蜜20两，同炼为丸。
临床应用：None
各家论述：None
用药禁忌：忌食酒、面、鸡、鱼、湿热炙煿之物。
附注：None
