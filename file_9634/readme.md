序号：9634
名称：牛膝丸
组成：牛膝2两（去苗），虎胫骨1两（涂酥，炙微黄），羌活1两，海桐皮3分，当归3分（两）（锉，微炒），巴戟3分，芎藭3分，薏苡仁3分，防风3分（两）（去芦头），桂心3分，杜仲1两（去粗皮，微炙，锉），鹿茸1两（去毛，涂酥，炙微黄），石斛3分（去根，锉），附子1两（炮裂，去皮脐），熟干地黄1两，酸枣仁3分（微炒），肉苁蓉1两（酒浸一宿，刮去皱皮，炙干），仙灵脾3分，补骨脂3分（微炒），干蝎3分（微炒），天麻3分，木香3分，槟榔1两。
出处：《圣惠》卷七。
主治：肾脏风毒流注，腰脚筋骨疼痛，行立艰难。
加减：None
功效：None
用法用量：每服30丸，食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
