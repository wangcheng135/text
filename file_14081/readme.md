序号：14088
名称：全羊丸
组成：羊羔1只（养带胎母羊1只，每日常食黑豆8两，拌小茴、羊藿、菟丝喂养，足月产羊1对，捡1雄羔听用；养小羊足月，以好酒饮之满腹，取血听用；滚水退去毛，五脏洗净，将故纸、肉苁蓉、青盐、小茴各4两装入腔内，用线缝固，荷叶包裹，蒸3炷香，取出炙干听用），白茯苓8两（用羊乳浸蒸，晒干），枸杞子6两（小羊血拌蒸，焙干），牛膝4两（酒浸，蒸），大地黄8两（砂仁酒蒸5炷香），山萸（去核，酒浸）4两，沙蒺藜4两（酒浸），山药（炒）4两，归身4两（酒洗），杜仲4两（去粗皮，姜汁盐水炒去丝），巴戟4两（酒浸），虎胫骨1对（酥炙），故纸4两（羊乳汁蒸1次，黑芝麻油炒），小茴4两（小羊血拌炒），肉苁蓉4两（酒洗，去鳞甲），青盐4两（净小羊血拌炒）。
出处：年氏《集验良方》卷二。
主治：一切虚损劳疾。
加减：None
功效：壮筋骨，补气血。
用法用量：每服2-3钱，滚水送下。
制备方法：上为细末，入前全羊4味，再研极细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
