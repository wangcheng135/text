序号：78232
名称：磁石汤
组成：磁石5两，白术5两，牡蛎5两，甘草1两，生麦门冬6两，生地黄汁1升，芍药4两，葱白1升，大枣15枚。
出处：方出《千金》卷六，名见《普济方》卷五十五。
主治：肾热，背急挛痛，耳脓血出，或生肉塞之，不闻人声。
加减：None
功效：None
用法用量：上(口父)咀，以水9升，煮取3升，分3服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：肾开窍于耳，肾热则风生于下，风生则激于上，故欲治肾热，先须镇摄真精，所以首推磁石，《本经》专主耳聋；佐以牡蛎，兼地黄、门冬，则滋益精气；芍药收敛营血；白术、甘草杜风实脾；葱白散邪通阳；大枣助诸经，补身中不足也。
用药禁忌：None
附注：None
