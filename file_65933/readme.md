序号：65949
名称：虚风丸
组成：白附子（炮裂）1两，天南星（炮裂）1两，乌头（炮裂，去皮脐）1两，天雄（炮裂，去皮脐）1两，防风（去叉）1两，天麻1两，附子（炮裂，去皮脐）1两，芎1两，人参1两，白茯苓（去黑皮）1两，天蓼木1两，乌蛇肉（酒浸，炙）1两，白鲜皮1两，白芷1两，麻黄（去根节）1两，细辛（去苗土）1两，恶实（炒）1两，甘草（炙）1两，雄黄（别研细）1两，牛黄（别研细）1两，丹砂（别研细）1两，蝎梢1分（炒），龙脑1分（别研细），麝香半两（别研细）。
出处：《圣济总录》卷五。
主治：一切虚风等疾，心神迷闷，头目旋运，耳内虚鸣，唇面冷麻，口面斜，语言謇涩，舌本紧强，神志昏塞，涎液不收。
加减：None
功效：None
用法用量：每服1丸，荆芥汤嚼下，或化下亦得，不拘时候。
制备方法：上为末，炼蜜为丸，如樱桃大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
