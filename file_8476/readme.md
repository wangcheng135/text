序号：8476
名称：半夏干姜散
组成：半夏、干姜各等分。
出处：《金匮》卷中。
主治：干呕吐逆，吐涎沫。
加减：None
功效：None
用法用量：每服方寸匕，浆水1升半，煎取7合，顿服之。
制备方法：上为散。
临床应用：None
各家论述：1.《金匮玉函经二注》：赵以德：干呕吐涎沫者，由客邪逆放肺，肺主收引，津液不布，遂聚为涎沫也。用半夏、干姜之辛热，温中燥湿；浆水之寒，收而行之，以下其逆，则其病自愈矣。_x000D_
2.《金匮要略心典》：干呕吐逆，胃中气逆也；吐涎沫者，上焦有寒，其口多涎也。此是阳明寒气逆气不下而已。故以半夏止逆消涎；干姜温中和胃；浆水甘酸，调中引气止呕哕也。
用药禁忌：None
