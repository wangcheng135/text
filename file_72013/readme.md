序号：72029
名称：小黄耆酒
组成：黄耆3两，附子3两，蜀椒3两，防风3两，牛膝3两，细辛3两，桂心3两，独活3两，白术3两，芎3两，甘草3两，秦艽2两，乌头2两（《集验》用薯蓣3两），大黄2两，葛根2两，干姜2两，山茱萸2两，当归2两半。
出处：《千金》卷七。
主治：风虚痰癖，四肢偏枯，两脚弱，手不能上头，或小腹缩痛，胁下挛急，心下有伏水，胁下有积饮，夜喜梦，悲愁不乐，恍惚善忘。此由风虚，五脏受邪所致。或久坐腰痛，耳聋卒起，眼眩头重，或举体流肿疼痹，饮食恶冷，涩涩恶寒，胸中痰满，心下寒疝。及妇人产后余疾，风虚积冷不除者。大风虚冷，痰癖偏枯，脚肿满。
加减：大虚，加苁蓉2两；下痢，加女萎3两；多忘，加石斛2两、菖蒲2两、紫石2两；心下多水者，加茯2两、人参2两，薯蓣3两。
功效：耐寒冷，补虚。
用法用量：可先食服1合，不知，可至4-5合，1日3次。酒尽，可更以酒2斗重渍滓。服之不尔，可晒滓捣下，酒服方寸匕，不知，稍增之。此药攻痹甚佳，亦不令人吐闷，小热宜冷饮食也。
制备方法：少壮人无所熬炼，虚老人微熬之，以绢袋中盛，清酒2斗渍之，春、夏5日，秋、冬7日。
临床应用：None
各家论述：None
用药禁忌：None
附注：黄耆酒（《千金翼》卷十六）、黄耆汤（《普济方》卷一八六引《鲍氏方》）。
