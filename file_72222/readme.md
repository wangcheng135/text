序号：72238
名称：紫金丸
组成：乌头（生，去皮尖）4两，地龙（去土，炒）2两，木鳖子（去壳）2两，白胶香（研）1两，乳香（研）半两，没药（研）半两，丹砂（研）半两，五灵脂1两，附子（炮裂，去皮脐）1两，白花蛇（酒浸，去皮骨，炙）1两，天麻1两，当归（切，焙）1两，龙脑1钱，麝香（研）1钱。
出处：《圣济总录》卷六。
主治：急风，筋脉紧急，身背强直，面黑鼻干，口噤不语，甚者壮热汗出，直视唇青，涎盛咽塞。
加减：None
功效：None
用法用量：每服半丸，生姜自然汁和酒磨下。一切风证，或乳香、龙脑、麝香、薄荷、茶、酒临时作汤使。
制备方法：上为末，取东流水为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
