序号：52094
名称：大补心丹
组成：黄耆（蜜炙）1两，茯神1两，人参1两，酸枣仁（炒）1两，熟地黄1两，远志（去心，炒）半两，五味子半两，柏子仁半两（别研）。
出处：《三因》卷九。
主治：狂证，因忧愁思虑过多，而致神志不宁，魂魄失守。阳虚外泄则自汗，呕吐，泻利频数，诸阴不生，则语言重复，忪悸眩晕；兼治大病后虚烦不得眠，羸瘦困乏。
加减：None
功效：安心神，调血脉，镇惊补心。
用法用量：每服30丸，米汤、温酒任下；盗汗不止，麦麸汤送下；乱梦失精，人参、龙骨汤送下；卒暴心痛，乳香汤送下；肌热虚烦，麦门冬汤送下；吐血，人参卷柏汤送下；大便下血，当归、地榆汤送下；小便尿血，赤茯苓汤送下；中风不语，薄荷、牛黄汤送下；风痫涎潮，防风汤送下。
制备方法：上为末，炼蜜为丸，如梧桐子大，用辰砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：补心神效丸（《百一》卷一）、安神补气丸（《中药制剂手册》）。
