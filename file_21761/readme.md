序号：21769
名称：红蓝花散
组成：红蓝花1两，柴胡1两半（去苗），当归1两，生干地黄1两，赤芍药1两，鬼箭羽1两，虎杖1两，大腹皮1两（锉），麦门冬1两（去心），土瓜根1两，地骨皮1两，枳壳1两（麸炒微黄，去瓤），甘草半两（炙微赤，锉）。
出处：《圣惠》卷七十。
主治：妇人热劳，羸瘦，四肢少力，经脉不通。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，煎至6分，去滓，不拘时候温服。
制备方法：上为粗散。
临床应用：None
各家论述：《济阴纲目》：方中赤芍、虎杖、土瓜根皆破血通经；腹皮、枳壳宽胸理气；其他不过清心凉血解热而已。
用药禁忌：None
附注：None
