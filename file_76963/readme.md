序号：76979
名称：黑金散
组成：麻（米凡）2斤（细杵，不用罗），地黄3斤（拣择匀停好者，然后晒干），皂角6挺（不蛀、肥好者，每挺截作3段），青盐3两（杵碎，不用罗），东引桃枝15截（如大指大长1握），柳枝15截（如大指大长1握），桑枝15截（如大指大长1握），墙头豆（生者）、马齿1斤（及盛时采，采到即于净室阴干，拘数收用）。
出处：《博济》卷三。
主治：髭鬓斑白，兼治牙齿一切疾病。
加减：None
功效：乌髭鬓。
用法用量：每日不以旦暮或食后揩齿，表里精细耐烦揩之，悉令周回，即合口须臾。若不倦，久长用之，髭鬓自然黑润异常，牙齿坚白无脱落，口气香洁。
制备方法：用1斗3升瓦罐新好者，将前件药物逐味分作三处，逐一味一重重下于罐子内，药绝，用一新瓦片，可罐子口盖覆，仍于盖于中心开一窍于，出烟，罐子周围以好黄土纸筋泥固济，候干，于净房内用炭火四向慢慢烧之，须是用火得所，烟尽，拨却火，塞眼子，却黄土焙1-2宿，候冷，取出药，其色如鸦羽，于瓷器内研细，切勿犯铁；更入升麻、白芷各1两为细末，入于末内令匀，以瓷器内收，密封闭，逐渐取出一匙头许，亦以小瓷器内收。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
