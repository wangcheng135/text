序号：30988
名称：十黄散
组成：雄黄5分，人参5分，黄芩3分，大黄3分，桂心3分，黄耆3分，黄柏3分，细辛3分，黄连1分，黄昏1分，蒲黄1分，麻黄1分，黄环2分，泽泻2分，山茱萸2分（一方有生黄2分）。
出处：《千金》卷十四。
主治：脏腑气血少，自觉不安，忽忽喜悲，善恐怖。
加减：羸劣者，更加人参5分。
功效：None
用法用量：未食前温酒服方寸匕，每日3次。不知，加至2匕。
制备方法：上药治下筛。
临床应用：None
各家论述：《千金方衍义》：十黄散专主本虚惊恐及风水外激，故取雄黄以辟百邪，以治阴邪浊恶之患；人参、黄耆、桂心、山萸护持心肾血气；麻黄、细辛外发肌表之邪；黄芩、黄连、大黄、黄柏、蒲黄、泽泻内泄脏腑血闭。惟黄昏、黄环人所未详，黄昏一名合欢，《本经》安五脏，和心志；黄环本名狼跋，《本经》治鬼魅邪气寒热，但非常用之品，以故罕有识者。
用药禁忌：None
附注：None
