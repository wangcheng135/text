序号：70837
名称：紫金膏
组成：铜青（研）7钱半，硇砂7钱半，硼砂7钱半，羚羊角7钱半，雄黄7钱半，青盐7钱半，琥珀7钱半，明矾7钱半，当门子8钱半，片脑8钱半，胆矾8钱半，深中青8钱半，玄精石半两，黄连半两，乳香半两，水银半两，小丁香、炉甘石1片（火炮7次，醋淬7次），石燕子2两（火煅，醋淬7次），金星石1两7钱5分（炼通红），黄丹8两，银星石1两2钱5分（火炼），海螵蛸1两7钱5分（为末），轻粉1钱，沙糖3斤，水3升。
出处：《普济方》卷七十一。
主治：赤肿风烂眼目。
加减：None
功效：None
用法用量：每用鸡头大，沸汤化开，浸汤半盏，乘热温洗。
制备方法：上以水同蜜于锅内熬沸去沫，入黄丹，以柳木篦急手搅匀，约熬2-3沸，却下炉甘石，又复搅入乳香、硇砂，又下雄黄，次下丁香，再入轻粉，方下片脑，依次第下药，文武火各熬2-3沸，须不住手搅匀成膏，候粘手为度。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中小丁香用量原缺。
