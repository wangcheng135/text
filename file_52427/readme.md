序号：52442
名称：香艾丸
组成：净香附1斤，干艾叶4两。
出处：《活幼心书》卷下。
主治：None
加减：妇人血气素虚无生育者，加琥珀2两，同作丸服，粒数汤使皆依前法，或用大枣汤送下。
功效：小儿常服，惊积自除，色泽殊异，手足肥健，脾胃调和；兼理男子、妇人诸虚不足，生气血，暖中焦，固养精神，消进饮食；男子服之身体强壮，寒暑耐安；妇人投之百病不生，经脉通顺。
用法用量：每服30-50丸，或70丸，汤、酒、米饮随意送下，不拘时候。
制备方法：上瓦器盛之，用醇醋浸经7日，于净锅内用火煮令醋尽，就炒干为细末，仍用醋煮粳米粉为糊，入乳钵和匀，小儿丸如萝卜子大，大人丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
