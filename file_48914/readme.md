序号：48929
名称：结毒灵药
组成：水银1两，朱砂3钱，雄黄3钱，硫黄3钱。
出处：《外科正宗》卷三。
主治：杨梅结毒，腐烂作臭，或咽喉、唇鼻腐坏日甚者并效。
加减：None
功效：None
用法用量：治寻常腐烂之症，灵药5钱，轻粉5钱，和匀碾细，小罐盛贮，纱封罐口，临用甘草汤洗净患上，将罐倒悬，纱眼内筛药患上，用后单油膏药盖之，1日1换，自效。男子、妇人咽烂者，灵药1钱，加人中白2分研细吹之，日用3次，内服不二散，其疼即止，随可饮食。
制备方法：上为细末，入阳城罐内，泥固，铁盏梁兜固紧封口，点3炷香为度，用水擦盏内，火毕，次日取出盏底灵药，约有1两5-6钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
