序号：5522
名称：加减小柴胡汤
组成：人参（去芦），北柴胡（去芦）4两，黄芩（去心）1两半，甘草（炙）1两半，半夏（泡7次，切）1两1分。
出处：《医方类聚》卷五十六引《管见大全良方》。
主治：伤寒3-4日，传少阳胆经，胸胁痛，耳聋，口苦，舌干，往来寒热而呕者。
加减：咳嗽者，去人参、大枣、生姜，加北五味子2两，干姜半两；若胁下满痛者，去枣，每服加煅过牡蛎半钱；若腹痛者，去黄芩，加芍药1两半；妇人经脉适行，或适断，寒热如疟，昼日明了，暮则谵语，如见鬼状，此为热入血室，每服加生干地黄1两；若心下悸，小便不利者，去黄芩加赤茯苓2两；若不渴，外有微热者，去人参，加桂枝1两半；若胸中烦而不呕者，去半夏、人参，加栝楼实1枚，用4分之1；若渴者，去半夏，更加人参3分，栝楼根2两。
功效：None
用法用量：上(口父)咀。每服5钱匕，水1盏半，加生姜4片，大枣3枚，煮取8分，去滓温服，不拘时候。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
