序号：50300
名称：茴香子丸
组成：茴香子3两（微炒，为末，以米醋2升熬如饧），附子（炮裂，去皮脐）1两，青橘皮（汤浸，去白，焙）3分，木香3分，狼毒（炒）3分，当归（切，焙）3分，阿魏1两（研，以酒1升，煎取半），硇砂1两半（沸汤化，澄熬取霜，入阿魏煎中同熬如饧，入茴香煎，搅匀），自然铜（煅，醋淬，研）1两半。
出处：《圣济总录》卷七十一。
主治：肾脏久积气在膀胱，虚胀上攻，膨满（疒丂）痛。
加减：None
功效：None
用法用量：每服15-20丸，空心温酒送下。
制备方法：上药除煎研外，为末，同入煎内和捣为丸，如梧桐子大，如硬入炼蜜少许。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
