序号：46345
名称：活血膏
组成：轻粉4钱，红粉1钱，乳香面4钱，没药面4钱，儿茶面2钱，血竭面4钱，黄丹2两，蜂蜡2两，头发2钱，蛇蜕2钱，香油8两，麝香2分，冰片1钱。
出处：《北京市中药成方选集》。
主治：跌打损伤，痈疽疮疖，已破未破，红肿高大，日久溃烂，久不收口。
加减：None
功效：活血化瘀，消肿止痛。
用法用量：敷于患处。
制备方法：先将轻粉、红粉、乳香、没药、儿茶、血竭等6味共研为细粉过罗；麝香、冰片研为极细粉，和匀，将香油放入勺内，加热熬开，入蛇蜕，头发2味，炸枯去滓过滤后，徐徐撒入黄丹及蜂蜡，不停搅拌，候呈黑色，再兑入轻粉等6味细粉，搅拌均匀，离火片刻，另将麝香、冰片细粉兑入和匀，倾入冷水盆中去火毒，收成软膏，每盒重5钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
