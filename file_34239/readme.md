序号：34248
名称：疔疮立效膏
组成：松香（制）4两，黄蜡2两，没药（去油）6钱，乳香（去油）6钱，百草霜1两，铜绿1两，白蜡4钱，蟾酥（隔水炖研，和入）3两，麻油3两，麝香（研细，后入）3钱。
出处：《中国医学大辞典》。
主治：疔疮初起，顶如粟，四围肿硬，或麻痒疼痛。
加减：None
功效：None
用法用量：每用1丸，呵软捻扁贴之，外益膏药。痛即止，次日肿消而愈；已走黄者用之亦效。
制备方法：上为细末，用桑柴火先将麻油入锅煎滚；次下松香，候稍滚；三下白蜡，候滚；再下黄蜡，候滚；再下乳香，稍滚；下没药，滚；即下铜绿，再滚；将百草霜下于锅内，滚数次；再后搅下蟾酥、麝香，即息火，冷透搓成条子，为丸如桂圆核大，藏净瓷器内，勿令泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
