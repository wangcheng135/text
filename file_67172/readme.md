序号：67188
名称：麻黄豆蔻丸
组成：木香2分，青皮2分，红花2分，厚朴2分，苏木3分，荜澄茄4分，升麻5分，半夏（汤洗）5分，麦糱面5分，缩砂仁5分，黄耆5分，白术5分，陈皮（去白）5分，柴胡5分，炙甘草5分，吴茱萸5分，当归身5分，益智仁8分，神曲末（炒）2钱，麻黄（去节）3钱，草豆蔻仁5钱。
出处：《兰室秘藏》卷上。
主治：客寒犯胃，心胃大痛不可忍。
加减：None
功效：None
用法用量：每服50丸，白汤送下；或细嚼，汤下亦可。
制备方法：上为细末，汤浸蒸饼为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：麻黄草豆蔻丸（《准绳·类方》卷四）。
