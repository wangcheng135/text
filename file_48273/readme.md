序号：48288
名称：神效万灵膏
组成：当归6钱，川芎6钱，赤芍6钱，生地黄6钱，熟地黄6钱，防风6钱，羌活6钱，独活6钱，连翘6钱，山栀6钱，黄连6钱，大黄6钱，玄参6钱，苦参6钱，白芷6钱，两头尖6钱，皂角6钱，桔梗6钱，白及6钱，白蔹6钱，红牙大戟6钱，五倍子6钱，山茨菇6钱，天花粉6钱，官桂6钱，蓖麻子60个，木鳖子40个，杏仁40个，巴豆肉40个，穿山甲10片。
出处：《寿世保元》卷九。
主治：诸疮肿毒。
加减：None
功效：None
用法用量：随意摊贴。倘膏嫩，加杭粉，不拘多少，不粘手为度。
制备方法：上为散，用真麻油2斤4两，发余4两，入药浸，春、秋3日，夏2日，冬5日，油药放铁锅内，文武火熬，用槐柳枝长寸许，各30根，同熬焦色，用麻布滤去滓，再放油锅内熬，滴水成珠不散，倾出瓶内，秤准油2斤，下山东黄丹1斤，松香2两，姜汁煮过黄蜡2两、桐油3两，熬至不老不嫩，冷了下乳香、没药、血竭、孩儿茶、阿魏、百草霜各3钱，麝香5分或1钱，轻粉3钱，马齿苋膏3钱，俱为细末，药油将好投下，早了恐泄药气，再熬，不粘手为度；将膏药埋土内3-4日，出火毒，瓷瓶内收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
