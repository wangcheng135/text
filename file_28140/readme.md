序号：28148
名称：苁蓉羊肉粥
组成：肉苁蓉4两（水煮令烂，薄切，细研），精羊肉（分为4度）。
出处：方出《证类本草》卷七引《药性论》，名见《药粥疗法》。
主治：精败面黑，劳伤。肾阳虚衰所致的阳萎、遗精、早泄，女子不孕，腰膝冷痛，小便频数，夜间多尿、遗尿，以及平素体质羸弱，劳倦内伤，恶寒怕冷，四肢欠温，脾胃虚寒，脘腹隐痛，老人阳虚便秘。
加减：None
功效：补肾助阳，健脾养胃，润肠通便。
用法用量：上药以米煮粥，入五味。空心服之。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：苁蓉羊肉粥属温热性药粥方，适用于冬季服食，以5-7天为一疗程，夏季不宜服食。凡大便溏薄，性机能亢进的人，也不宜选用。
