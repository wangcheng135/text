序号：79456
名称：摩顶明目膏
组成：生麻油2升，真酥5两，车前叶半两，淡竹叶（洗，锉）半两，吴蓝1分，大青1分，黄连（去须）1分，山栀子仁1分，黄芩（去黑心）1分，甘草（炙）1分，麦门冬（去心）1分，槐白皮1分，柳白皮1分，马齿苋实（研）1分，生犀角（镑）1分，马牙消（别研）1分，朴消（别研）1分，盐花（研）半两。
出处：《圣济总录》卷一○五。
主治：风热冲目，赤脉胬肉，昏障痛涩。
加减：None
功效：祛目中热毒。
用法用量：每日饭后及卧时用药滴头顶心，以生铁熨斗摩顶一二千下。
制备方法：上药除消、盐、油、酥外，锉细，绵裹入通油瓷瓶中，绵罩口，重汤煮三复时，挼去滓，更新绵滤过，置生铁器中。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
