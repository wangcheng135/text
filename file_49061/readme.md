序号：49076
名称：胎元饮
组成：人参随宜，当归2钱，杜仲2钱，芍药2钱，熟地2-3钱，白术1钱半，炙甘草1钱，陈皮7分（无滞者不必用）。
出处：《景岳全书》卷五十一。
主治：气血两虚而胎不安者，六脉微弱，神昏气倦，一切不足之证。
加减：如下元不固而多遗浊者，加山药、补骨脂、五味之类；如气分虚甚者，倍白术，加黄耆，但耆、术气浮，能滞胃口，倘胸膈有饱闷不快者，须慎用之；如虚而兼寒多呕者，加炮姜7-8分或1-2钱；如虚而兼热者，加黄芩1钱5分，或加生地2钱，去杜仲；如阴虚小腹作痛，加枸杞2钱；如多怒气逆者，加香附无妨，或砂仁亦妙；如有所触而动血者，加川续断、阿胶各1-2钱；如呕吐不止，加半夏1-2钱、生姜3-5片。
功效：None
用法用量：水2钟，煎7分，食远服。或间日，或2-3日，常服1-2剂。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
