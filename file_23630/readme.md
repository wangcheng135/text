序号：23638
名称：戒烟膏
组成：明党参4钱，云茯苓4钱，黄耆（炙）4钱，潞党参4钱，玉竹（炙）4钱，炮姜炭4钱，罂粟壳4钱，杜仲（炒）4钱，橘红4钱，枸杞4钱，旋覆花（绢包）2钱4分，甘草（炙）2钱4分，法半夏2钱4分，益智仁2钱4分，枣仁2钱。
出处：《中国医学大辞典》引林文忠公方。
主治：None
加减：肚腹下坠者，加沉香2钱。
功效：None
用法用量：烟瘾1钱，膏亦1钱，每日吸烟几次，服膏亦几次，瘾前服，服7日，减去1成，逐次减去，以减尽为度。体丰阳虚者，服此方极佳。
制备方法：加红枣四钱，赤砂糖2两，清水煎取浓汁，去渣收成膏，称见若干，加清烟膏一成，搅入和匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方改为丸剂，名“戒烟丸”（见原书）。
