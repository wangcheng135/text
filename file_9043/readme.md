序号：9043
名称：司马灵通膏
组成：当归（洗）半两，赤芍药半两，地骨半两，木香3钱，白芷3钱，蜂房3钱，佗僧（研）3钱，木鳖子14个（掟），杏仁19个，黄丹2两，巴豆14个，血竭2钱，轻粉1钱半，乳香1钱半，没药1钱半，血余（鸡子大）1块，沥青半斤，白胶香半斤，香油半斤。
出处：《袖珍》卷三。
主治：一切疮疡并痔漏。
加减：None
功效：None
用法用量：取东南槐柳枝条，如筷子长，各49根，同当归、木香等10味，同香油在沙锅内熬黑色，去滓；次入沥青，胶香熔尽，去滓，上火再熬，入佗僧、黄丹，用柳条枝不住手搅，滴水中不散为度。稍冷，入乳香、没药、血竭、轻粉搅匀，瓷器内盛，湿土埋，出火毒。冬雪埋，腊月熬尤妙。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
