序号：69280
名称：黄香膏
组成：松香、白水煮透，取出放冷水内搓洗数10下，再煮再洗，如此9次，倒地得冷取起。每1两加：轻粉3钱，银朱1钱，白蜜少许。
出处：《验方新编》卷八。
主治：赚疮；及一切痈毒大疮，日久不愈。
加减：None
功效：None
用法用量：上药炼老成珠，加菜油少许，炖热搅匀，看疮之大小作饼。置疮上，将绸条扎住。1周时取下，用滚水搓洗极净，翻转再贴，周时取下，再洗再贴。只要1个药饼直贴到好，不须另换。待疮好，将此药饼洗净收好，如遇此疮，再与别人贴，仍前1周时1洗1贴。此饼若医过3人之后，贴上即好，若医过10人，贴上更能速愈，奇绝妙绝。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
