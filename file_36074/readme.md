序号：36084
名称：参茸卫生丸
组成：人参（去芦）80两，鹿茸（去毛）80两，巴戟天（炙）80两，党参（去芦）80两，山药80两，桑寄生80两，白芍80两，莲子肉80两，锁阳80两，苍术（炒）32两，乳香（炙）32两，黑附子32两，川牛膝112两，熟地160两，酸枣仁（炒）160两，甘草160两，香附（炙）160两，杜仲（炒）160两，何首乌（炙）48两，麦冬48两，牡蛎（煅）48两，枸杞子48两，龙骨（煅）48两，肉桂（去粗皮）48两，远志（炙）40两，复盆子64两，补骨脂（盐水炒）64两，茯苓240两，于术240两，没药（炙）16两，桂圆肉320两，琥珀96两，黄耆96两，砂仁152两，山茱萸128两，当归128两，红枣（去核）176两，肉苁蓉160两，续断48两，沉香48两，橘皮320两，生地32两，木香80两，白术（炒）160两。
出处：《北京市中药成方选集》。
主治：身体衰弱，精神不足，梦遗滑精，腰膝痠软，食欲不振。
加减：None
功效：滋补肝肾，健脾益胃。
用法用量：每服1丸，温开水送下，1日2次。
制备方法：续断、沉香、橘皮、生地、木香、白术6味研为粗末餔槽，余者下罐，加黄酒4000两，蒸3日夜，与餔槽之群药末拌匀晒干，共研为细粉过罗。每16两细粉，兑朱砂3钱6分，炼蜜为丸，重3钱，蜡皮封固。
