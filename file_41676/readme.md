序号：41691
名称：金箔镇心丸
组成：紫河车（用黑豆煮软，切作片，焙干）25两，山药150两，牙消（枯）15两，甘草（爁）50两，人参（去芦）50两，茯苓（去皮）50两，朱砂（研，飞）100两，龙脑（研）10两，麝香（研）5两，金箔1200箔（为衣）。
出处：《局方》卷十。
主治：小儿风壅痰热，心神不宁，惊悸烦渴，唇焦颊赤，夜卧不安，谵语狂妄。
加减：None
功效：安镇心神，散败邪热，凉咽膈，止惊啼。
用法用量：每服1丸，薄荷汤化下，含化亦得，食后临卧服。
制备方法：上为细末，炼蜜为丸，每1两半作50丸，以金箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
