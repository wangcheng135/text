序号：57739
名称：烧肝散
组成：茵陈半两，犀角半两，石斛半两，柴胡（去苗）半两，白术半两，芍药半两，干姜1两，防风1两，紫参1两，白芜荑1两，桔梗1两，人参1两，胡椒1两，吴茱萸1两，官桂1两。
出处：《博济》卷一。
主治：三十六种风，二十四般冷，五劳七伤。一切痢疾，脾胃久虚，不思饮食，四肢无力，起止甚难，小便赤涩，累年口疮，久医不愈。
加减：None
功效：None
用法用量：每日空心以生姜汤调下。大段冷劳，不过3服见效。
制备方法：上为末。以羊肝1具（如无，即豮猪肝代之），分作3份，净洗去血脉脂膜，细切，用末5钱，葱白1茎细切，相和，以湿纸3-5重裹之，后掘地坑，内以火烧令香熟。
临床应用：泄痢《苏沈良方》：庐州刁参军，病泄痢日久，黑瘦如墨，万法不愈，服此1-2服，下墨汁遂安。
各家论述：None
用药禁忌：None
附注：None
