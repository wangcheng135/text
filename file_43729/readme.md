序号：43744
名称：南星膏
组成：牛胆南星（腊月以南星为末，填入黄牛胆中，风处阴干，百日取用，宜亲手修制者佳）5钱（炒用），人参3钱，白术3钱，山药（炒）3钱，白茯苓（去皮）3钱，白茯神（去心）3钱，羌活3钱，甘草（炙）3钱，白僵蚕3钱，全蝎（去毒，以薄荷汁浸，炙）3钱，辰砂2钱（水飞另研），麝香1分。
出处：《松崖医径》卷下。
主治：小儿精神不定，恍惚不宁，恐畏多哭。
加减：None
功效：祛风退热，消痰镇心，除百病。
用法用量：每服1丸，食后用薄荷汤调化服下。
制备方法：上各为细末，一处和匀，炼蜜丸，如芡实大，金箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
