序号：59855
名称：大黄汤
组成：大黄3两，栀子50个，升麻2两，黄芩3两，芒消1两。
出处：《鬼遗》卷三。
主治：痈疽，发背，时毒焮肿赤痛，烦渴便秘，脉实数；年四十已，还强壮，常大患热痈无定处，大小便不通
加减：None
功效：None
用法用量：以水5升，煮取2升4合，去滓，下消搅调，分温3服。快利为度。
制备方法：上切。
临床应用：时毒：一男子冬月耳面赤肿，发热口干，脉洪实而便秘，此三阳蕴热症也。必舍时从症治之，以五利大黄汤一剂便行二次，赤肿稍退，内热稍疏。又以升麻解毒汤二服，肿消而病愈。此为用寒远寒之意也。
各家论述：《千金方衍义》：升麻升举清阳于上，消、黄荡涤热毒于下，黄芩、栀子兼清表热，从渗道而泄也。
用药禁忌：None
附注：五利汤（《千金》卷二十二）、五利大黄汤（《圣济总录》卷一三○）。
