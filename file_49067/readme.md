序号：49082
名称：胎产金丹
组成：当归（酒洗）2两，丹皮（水洗，晒干，勿见火）2两，蕲艾（醋煮）2两，延胡索（酒拌，炒干）2两，川芎2两，益母草（取上半截，童便浸，晒干）2两，青蒿（人多内热者更宜，不用亦可）2两，白薇（洗净，人乳拌）2两，人参2两，赤石脂（火煅，水飞亦可）2两，白茯苓2两，川藁本（洗净）2两，白术（土炒）2两，生地（酒洗，煮不犯铁器）4两，鳖甲（醋炙）4两，香附4两（醋、酒、盐、童便各浸1两），桂心1两2钱，没药（去油）1两2钱，粉草（酒炒）1两2钱，北五味1两（去梗，焙），沉香6钱。
出处：《胎产心法》卷中。
主治：临经腹痛，腰痠带多，面黄肢倦，子宫虚寒，难于受孕。脾胃虚弱，胎前漏血，腰腿痠痛，四肢浮肿，气血双亏，作冷作烧，不思饮食，自汗盗汗，骨蒸潮热。肚腹疼痛。
加减：None
功效：培补气血，安胎调经，益精种子。
用法用量：妇人临产，米汤化服1丸，助精神气力，分娩顺利；产下，童便好酒服1丸，神清体健，再无崩晕之患；产后，每日服1丸，服过5日，气血完固，自无他病；行经后，川芎当归汤服1丸，服之3日，必然有孕；苦于小产者，胎动欲产，白滚汤服1丸，睡半日，其胎自安，每月常服2-3丸，保全足月分娩无忧；产后血崩，童便好酒服1丸，即止；产后血晕者，当归川芎汤服1丸，即醒；产后惊风，防风汤服1丸，即解；儿枕痛者，山楂黑沙糖汤服1丸，即止；胞衣不下，干姜炒黑煎汤服1丸，即下；产后虚怯者，川芎当归汤每日服1丸，10丸痊愈；凡产后诸证，俱加好酒童便服。
制备方法：上为细末；再用新鲜头次男胎紫河车一具，长流水浸半日，洗净；黑铅打成大铅罐一个，将河车放在铅罐内，再将黄柏4两放在河车下，加白酒酿2斤，清水2碗，灌满铅罐，仍以铅化封口；再以铁锅盛水，将铅罐悬在锅内，煮2日夜为度，取出捣烂，和入药内，拌匀，晒干，再研为末，炼蜜为丸，如弹子大，每丸重3钱5分，水飞朱砂为衣，再以黄蜡为皮，如蜡丸式收贮。
临床应用：None
各家论述：None
用药禁忌：忌食生冷；忌生气。
