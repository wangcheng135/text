序号：4128
名称：木瓜煎丸
组成：木瓜（去子，蒸令熟）2枚，黑豆（拣细紧者）1升，陈皮（酒浸，去白）4两，吴茱萸4两，附子（炮裂，去皮脐）4两，桂（去粗皮）1两，芎1两，羌活（去芦头）1两，肉豆蔻（去壳）1两，槟榔（锉）1两。
出处：《普济方》卷二二○引《博济》。
主治：脏腑虚弱，眼目昏暗，腰膝无力，或脑冷痠痛，饮食减少。
加减：None
功效：补暖。
用法用量：每服15丸至20丸，空心、夜卧盐汤送下。候见效即减5丸。
制备方法：上药先煮黑豆令熟，浥干，却入陈皮、木瓜、吴茱萸3味拌匀，用好酒1升，入豆内煎泣干漉出，令烂研似面糊，然后将附子以下6味，捣罗为末，入在前膏内搅和令匀，再杵得所，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
