序号：35512
名称：京三棱煎丸
组成：京三棱（煨，锉）1两，蓬莪术（煨，锉）1两，芫花（醋炒焦）1两，半夏（汤洗7遍，焙）1两，青橘皮（去白，焙）1两，硇砂（去石，研）半两，附子（炮裂，去皮脐）半两，桂（去粗皮）半两，延胡索（醋炒）半两，大戟（腻粉调，酒炙）半两，干漆（炒烟出）半两，猪牙皂荚（去皮子，炙）半两，五灵脂（醋炒）半两。
出处：《圣济总录》卷七十一。
主治：五积六聚，血瘕气块，聚散不定，及一切气疾。
加减：None
功效：None
用法用量：每服5丸，食后生姜汤送下。
制备方法：上为末，用好醋3升，入药二停，熬成膏，再入一停，为丸如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
