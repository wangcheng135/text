序号：34344
名称：万金丹
组成：藿香叶1两，踯躅花1两，天南星（炮裂）1两，甘松（去土）1两，麻黄（不去节）1两，吴白芷1两，甘草（炙）1两，蔓荆子（去白皮）1两，川乌3两，何首乌3两，草乌头7个（生用），五灵脂（酒浸1宿，去沙石）1两，白胶香（细研）1两，麝香2钱，没药3钱（研细），蛤粉（为衣）1两。
出处：《普济方》卷一一六。
主治：一切风，左瘫右痪，筋骨拘急，遍身生疮，及风毒攻疰，妇人血风、暗风、夹脑风，偏正头痛。
加减：None
功效：None
用法用量：每服1丸，茶、酒任下；若牙关紧急，酒磨灌下，不拘时候。
制备方法：上药除蛤粉为衣外，15味并捣罗为末，糯米1盏，好酒1升，煮粥为丸，如鸡头子大，窨干。
临床应用：None
各家论述：None
用药禁忌：忌热物少时。
附注：None
