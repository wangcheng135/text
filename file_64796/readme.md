序号：64812
名称：续阴救绝汤
组成：人参2两，白术3两，附子1钱，巴戟天1两。
出处：《辨证录》卷八。
主治：房事大泄，精尽阳脱。
加减：None
功效：补阳。
用法用量：水煎服。1剂血止，2剂阴生，连服4剂，可以不死。
制备方法：None
临床应用：None
各家论述：此方补阳气之圣药也。用人参回绝续于无何有之乡，用白术以通利其腰脐之气，用附子以追其散失之元阳，用巴戟天补其心肾之阳，纯是补阳之药，则阳回而阴亦回也。倘不用人参，只用附、术、巴戟亦可夺命于须臾，然无参为君主之味，则附子之热无以驾驭，恐有阳旺阴消之弊。倘能以补之药济其后，亦不至有偏胜耳。
用药禁忌：None
附注：None
