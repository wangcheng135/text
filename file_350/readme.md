序号：350
名称：五香连翘汤
组成：木香3分（不见火），沉香3分（不见火），连翘（全者，去蒂）3分，射干3分，升麻3分，黄耆3分（拣无叉附者，生用），木通3分（去节），甘草半两（生用），丁香半两（拣去枝杖，不见火），乳香半两（别碾），大黄（微炒，锉）半两，麝（真者，别碾）1钱半，桑寄生3分（难得真者，缺之亦可），独活3分。
出处：《集验背疽方》。
主治：痈疽，一切积热恶核、瘰疬、痈疽、恶疮、发脑、发背。
加减：若无真桑寄生，则升麻分量当倍用。
功效：None
用法用量：每服3大钱，水1盏，煎至7分，去滓服。留滓2服，用水2盏再煎作1服。积四散滓，用水2盏，又再煎作1服，然后不用其滓。一方用银器煎药。如无银器入银1片同煎。
制备方法：上为粗末，和匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：李氏五香连翘散(《医方类聚》卷一七五引《澹寮》)。
