序号：22427
名称：阴阳兼培丸
组成：熟地8两，枣皮4两，淮药4两，茯苓4两，鹿角胶6两（蛤粉炒成珠，或酒蒸溶合，炼蜜为丸），附子3两，杜仲（淡盐水炒）3两，枸杞4两（酒蒸），淮牛膝3两（酒蒸），北五味1两半（微炒），当归3两（酒蒸），白芍2两（煨，酒炒），菟丝子（淘净泥沙，酒蒸，晒干）4两（或加肉桂3两）。
出处：《会约》卷十四。
主治：先天不足，精亏阳萎，后天不足，食少体倦，一切不足之证。
加减：阳萎，加补骨脂（盐炒）3两、巴戟3两、胡桃肉4两；下焦虚滑，去淮牛膝。
功效：None
用法用量：每服7-8钱，早用淡盐水送下。
制备方法：先将地黄、枣皮、枸杞、当归捣如膏，后入药末、鹿胶，量加炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
