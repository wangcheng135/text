序号：7045
名称：四物汤加桃仁红花竹沥姜汁方
组成：四物汤加桃仁、红花、竹沥、姜汁。
出处：方出《丹溪心法》卷一，名见《医方考》卷一。
主治：半身不遂在左属死血瘀血者。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：《医方考》：芎、归、芍、地，生血药也，新血生，则瘀血滑而易去；桃仁、红花，消瘀药也，瘀血去，则新血清而易生；然亦加夫竹沥、姜汁者，以痰之为物，靡所不之，盖无分于左右而为患也。
用药禁忌：None
附注：四物加桃仁红花竹沥姜汁方（《明医指掌》卷二）、四物加竹沥汤（《医林纂要》卷五）。《医林纂要》本方用当归（酒洗）四钱、川芎二钱、生地三钱、芍药二钱、桃仁七粒，红花一钱、竹沥半杯，姜汁三匙，水煎服。
