序号：43932
名称：厚朴散
组成：厚朴1两（去粗皮，涂生姜汁，炙令香熟），木香半两，当归3分（锉碎，微炒），熟干地黄1两，半夏半两（汤洗7遍去滑），人参3两（去芦头），白茯苓3分，白芍药半两，干姜半两（炮裂，锉），桂心半两，牛膝3分（去苗），陈橘皮3分（汤浸，去白瓤，焙），白术3分，附子3分（炮裂，去皮脐），甘草半两（炙微赤，锉）。
出处：《圣惠》卷七十。
主治：妇人冷劳气，面色萎黄，四肢羸瘦，多卧少起，不欲饮食，身体虚困。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分、大枣3个，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
