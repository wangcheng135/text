序号：70807
名称：紫朴开郁丸
组成：槟榔1两，蔻仁5钱，香附5钱，广木香2钱5分，草蔻仁5钱，琥珀5钱，苍术5钱，焦三仙1两5钱，枳壳2钱5分，广陈皮5钱，厚朴5钱，桔梗5钱，藿香5钱，青皮5钱，砂仁5钱，炙甘草5钱。
出处：《全国中药成药处方集》（沈阳方）。
主治：肝气横逆，胃不消化，心胃疼，肚子疼，打饱咯，吐酸水，膨闷胀饱，两胁攻痛，食前腹痛，食后胃痛，饮食不消，胃弱不食，胸腹串痛，噎膈反胃，气裹食积，胃连腹痛。
加减：None
功效：舒肝健胃，理气解郁，宽中散满，消食顺气，调经。
用法用量：每服1丸，白开水送下。
制备方法：上为极细末，炼蜜为丸，2钱重。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
