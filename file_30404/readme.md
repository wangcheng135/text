序号：30412
名称：辰砂散
组成：辰砂1两（须光明有墙壁者），酸枣仁（微炒）半两，乳香（光莹者）半两。
出处：《苏沈良方》卷二。
主治：风邪诸痛，狂言妄走，精神恍惚，思虑迷乱，乍歌乍哭，饮食失常，疾发仆地，吐沫戴目，魂魄不守，医禁无验。
加减：None
功效：化痰定惊。
用法用量：量所患人饮酒几何，先令恣饮沉醉，但勿令至吐，静室中服药讫，便安置床令睡，以前药都为1服，温酒1盏调之，顿服令尽；如素饮酒少人，但随量取醉；病浅入1-2日，深者3-5日，睡不觉，令家人潜伺之，觉即神魂定矣；慎不可惊触使觉，及他物惊动，一为惊寤，更不可治。
制备方法：None
临床应用：心病：上枢正肃吴公，少时病心，服一剂，三日方寤，遂愈。
各家论述：《医方集解》：此手少阴药也，辰砂镇心泻火，乳香入心散瘀血，枣仁补肝胆而宁心。
用药禁忌：None
附注：朱砂酸枣仁乳香散（《医方考》）、灵苑辰砂散（《准绳·类方》卷五）、灵苑丹（《类证治裁》卷四）。
