序号：43748
名称：南烛草煎丸
组成：南烛草5斤，酸石榴叶5斤，旱莲子苗5斤。
出处：《圣惠》卷四十一。
主治：血气虚惫，须发秃落不生，纵生，色黄不黑。
加减：None
功效：None
用法用量：每日空腹服30丸，温酒送下，晚食后再服。
制备方法：以上3味，于5月上旬收于瓷瓶子中，泥封令密，安日中，至6月中旬取出，皆如黑饧，研之，又以生地黄5斤，绞取汁，及白蜜5合，同煎上件3味成膏，次下后诸药末：地骨皮4两，熟干地黄4两，诃黎勒皮2两，秦椒2两（去目及闭口者，微炒去汗），白芷2两，旋覆花2两，桂心2两，杏仁3两（汤浸，去皮尖双仁，麸炒微黄）。上为末，入上件膏中，用缓火煎，可丸即丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌生葱、萝卜、大蒜等。
附注：None
