序号：56263
名称：流注汤
组成：土茯苓（刮去皮，打碎）4两，龙胆草2钱，贝母3钱，僵蚕3钱，银花3钱，槐花3钱，五倍子3钱，橘红1钱，防己1钱，防风1钱，木通1钱，甘遂7分，皂角子9个。
出处：《仙拈集》卷四引《顾体集》。
主治：湿痰流往，漫肿无头，皮色不变，久而不治，则发热作脓，未破者。
加减：虚弱，加石斛、苡仁；痰在头顶、胸，加夏枯草；在脊背，加羌活；胁肋，加柴胡；在肚腹，加赤芍、泽泻；在臂，加独活；在腿脚，加木瓜、牛膝；已破头者，只服4-5剂，随服十全大补汤加川贝母、石斛、乳香10余剂方能全愈。
功效：None
用法用量：水3碗，煎3大钟。每日早、中、晚各热服1钟，痰在上，食后服；在下，食前服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：方内有甘遂，忌甘草。
附注：None
