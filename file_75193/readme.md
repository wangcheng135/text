序号：75209
名称：腽肭补天丸
组成：腽肭脐1两半，人参1两半，白茯苓（姜汁煮）1两半，当归1两半，川芎1两半，枸杞1两半，小茴香1两半，白术2两半，粉草（蜜炙）1两，木香1两，茯神1两，白芍2两，黄耆2两，熟地黄2两，杜仲2两，牛膝2两，故纸2两，川楝2两，远志2两，胡桃肉3两，沉香5钱。
出处：《医学入门》卷七。
主治：男妇亡阳失阴，诸虚百损，阴痿遗精，健忘白带，子宫虚冷。
加减：男，加知、柏；女，加附子。
功效：None
用法用量：每服60丸，空心盐酒送下。
制备方法：上为末，用制腽肭酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
