序号：30296
名称：阿魏散
组成：野狐皮（焙，炙）7枚，豹鼻（焙）7枚，狐头骨1具（炙），雄黄1两，腽肭脐1两，鬼箭羽1两，露蜂房1两，白术1两，虎头骨（炙）1两，驴毛4分，驼毛4分，驹毛4分，牛毛4分（烧作灰，若骨蒸，气如泥，加死人胸骨1两炙），阿魏2两（炙）。
出处：《普济方》卷二五四。
主治：梦与鬼交通，及狐狸精魅等。
加减：None
功效：None
用法用量：每用1分，于座下熏患者。欲熏之时，盖覆衣被，勿令药气轻泄，别捣雄黄为末以藉药，烧药节度，一如熏香法。
制备方法：上为散，搅使调匀。先以水煮松脂，候烊接取以和散，和散之时，勿以手搅，将虎爪搅和为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：忌桃、李、雀肉等。
附注：None
