序号：69947
名称：滑石代赭汤
组成：百合7枚（擘），滑石3两（碎，绵裹），代赭石1枚（如弹丸大，碎，绵裹）。
出处：《金匮》卷上。
主治：百合病下之后者。
加减：None
功效：None
用法用量：先以水洗百合，渍1宿，当日沫出，去其水，更以泉水2升，煎取1升，去滓，别以泉水2升煎滑石、代赭，取1升，去滓，后合和，重煎取1升5合，分温服。
制备方法：None
临床应用：None
各家论述：1.《金匮玉函经二注》赵以德：百合安心定胆，益志五脏，为能补阴也；用滑石、代赭佐以救之，滑石开结利窍，代赭除脉中风痹瘀血。_x000D_
2.《金匮要略心典》：百合病不可下而下之，必伤其里。百合味甘平微苦，色白入肺，治邪气，补虚清热；复以滑石、代赭者，盖欲因下药之势，而抑之使下，导之使出、也在下者引而竭之之意也。_x000D_
3.《金匮要略释义》：以百合润肺而养阴，滑石清热而利小便，赭石重镇而降逆气。
用药禁忌：None
附注：百合滑石代赭汤（《千金》卷十）、百合代赭汤（《伤寒全生集》卷四）、百合滑赭汤（《医学入门》卷四）。
