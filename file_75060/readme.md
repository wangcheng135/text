序号：75076
名称：鼠粘子汤
组成：鼠粘子4两（炒香），甘草1两（炙），防风半两，荆芥穗2两。
出处：《活人书》卷二十一。
主治：小儿疹痘欲出，未能得透皮肤，热气攻咽喉，眼赤心烦者。大人、小儿内蕴邪热，咽膈不利，痰涎壅嗽，眼赤睑肿，腮项结核，痈肿毒聚，遍身风疹，瘅毒赤瘰，及疮疹已出未出，不能快透者。
加减：None
功效：利咽膈，化痰涎，止嗽。春、冬间常服免生疮疖。疏风散热。
用法用量：每服2钱，食后临卧沸汤点服，逐日3服。老幼皆宜服。
制备方法：上为末。
临床应用：None
各家论述：《医略六书》：方中防风散风邪以胜湿，荆芥走血分以疏风，甘草缓中泻火，大力子力能疏散风热，以发痧疹也。水煎温服，使风散湿除，则遏热亦外泄而痧疹无不透矣，此疏风散热之剂，为痧疹不出之专方。
用药禁忌：None
附注：消毒犀角饮（《局方》卷六吴直阁增诸家名方）、消毒犀角饮子（《御药院方》卷九）、消毒犀角散（《普济方》卷二七二）、鼠粘汤（《普济方》）卷四○三）、消毒饮子（《明医指掌》卷六）、消毒犀角汤（《玉案》卷三）、鼠粘子散（《医略六书》卷二十）。
