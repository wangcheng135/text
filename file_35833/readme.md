序号：35843
名称：参术饮
组成：四物汤加人参、白术、半夏（制）、陈皮、甘草。
出处：《丹溪心法》卷五。
主治：妊娠转胞。
加减：None
功效：调养荣卫，化痰理气，清升浊降。
用法用量：上(口父)咀，加生姜，煎，空心服。
制备方法：None
临床应用：转胞：一妇人怀胎，患转胞病，两脉似涩，重则弦，左稍和，此得之忧患，涩为血少气多，弦为饮。血少则胎弱，而不能自举，气多有饮，中焦不清而隘，则胎知所避而就下，故喜坠。以四物汤加参、术、半夏、陈皮、生甘草、生姜煎，空心饮，随以指探喉中出药汁，候少顷，气定又与一帖，次早亦然，至八帖安。
各家论述：None
用药禁忌：None
附注：参术散（《广嗣纪要》卷十三）。
