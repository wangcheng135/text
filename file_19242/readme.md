序号：19249
名称：生地黄汤
组成：生地2-3钱，干漆1钱（炒烟尽），生藕汁1小盅（如无以大蓟12钱代之），蓝叶钱半，大黄12钱（生、熟酌用），桃仁1钱（去皮，研），归尾2钱（酒洗），红花6分（酒洗）。
出处：《松峰说疫》卷二。
主治：蓄血。
加减：None
功效：None
用法用量：水与藕汁同煎服。
制备方法：None
临床应用：None
各家论述：抵当汤、丸，今总难用，以此代之，甚觉和平。原方水蛭、虻虫今改用归尾、红花。蓄血有上中下之殊，上焦胸中手不可近而痛者，犀角地黄汤；中脘手不可近，桃仁承气；脐下小腹手不可近，抵当嫌峻猛，此汤主之，或再加枳实、苏木，用者酌之。
用药禁忌：None
附注：None
