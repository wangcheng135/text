序号：81916
名称：熟干地黄散
组成：丹参（去芦头）半两，防风（去芦叉）半两，当归（去芦，微炒）半两，细辛（去苗）半两，藁本（去芦，洗）半两，芎半两，人参（去芦）1两，熟干地黄（酒洒蒸，焙）1两，白茯苓（去皮）1两，肉桂（去粗皮）1两，白术1两，续断3分，附子（炮，去皮脐）3分，黄耆（去芦）3分。
出处：《局方》卷九。
主治：妇人劳伤血气，脏腑虚损，风冷邪气乘虚客搏，肢体烦痛，头目昏重，心多惊悸，寒热盗汗，羸瘦少力，饮食不进。
加减：None
功效：None
用法用量：每服4钱，水1盏半，加生姜半分，大枣3个（擘破），煎至1盏，滤去滓，食前温服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
