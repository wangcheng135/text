序号：7451
名称：生肌膏
组成：防风1分（去芦头），白蔹1分，赤芍药1分，当归1分，芎藭1分，桑根白皮1分，杏仁1分（汤浸，去皮尖双仁），甘草1分，垂柳枝（锉）3合，乱发1两（洗令净），黄丹5两，木香1分，丁香1分，麻油1斤（清者）。
出处：《圣惠》卷六十八。
主治：灸疮久不愈，且疼痛。
加减：None
功效：None
用法用量：每用看灸疮大小，以纸上匀摊贴之，每日2-3度换，仍煎柳枝汤洗。勿令伤风。
制备方法：上药除香2味捣罗为末，余并细锉。以酒（油）浸1宿，慢火熬令柳枝色黄黑，绵滤去滓，澄清，拭铛令净，慢火熬药油，入黄丹，用柳木篦不住手搅，令丹色稍黑，取少许滴于水内，捻看得所，入香末，又搅令匀，倾于不津器中盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
