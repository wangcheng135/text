序号：48224
名称：神明补心丹
组成：远志（去心）8钱，紫石英（飞研）8钱，石菖蒲8钱，熟地黄半两，白茯苓（去皮）半两，麦门冬（去心）4钱，卷柏（去根土）4钱，人参（去芦头）4钱，丹参4钱，黄耆4钱，白术4钱，泽泻4钱，山茱萸4钱，防风4钱，秦艽4钱，桔梗4钱，柏子仁2钱半，川姜2钱半，干山药2钱，白蔹2钱，芍药2钱，石膏（飞研）2钱，铁粉（飞研）2钱，神曲（炒）2钱，当归2钱，半夏（生姜制）2钱，牡丹皮2钱，朱砂（研飞）4钱。
出处：《御药院方》卷六。
主治：心气不足，神志不定，恍惚多惊，虚烦少睡，心情沉默，恶闻人声，一切心虚之证。
加减：None
功效：None
用法用量：每服1丸，煎人参汤化下，温酒亦得，不拘时候。
制备方法：上为细末，入朱砂令匀，炼蜜和丸，每两作10丸，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
