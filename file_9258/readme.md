序号：9258
名称：牛黄丸
组成：牛黄（别研）半两，腻粉半两，天麻（酒炙）1两，桂（去粗皮）1两，白附子（炮）1两，干蝎（炒，去土）1两，天南星（炮）1两，石菖蒲1两，附子（炮裂，去皮脐）1两，羌活（去芦头）1两，麻黄（去根节，汤煮，掠去沫）1两，芎藭1两，干姜（炮）1两，当归（切，焙）1两，独活（去芦头）1两，防风（去叉）1两，麝香1分，乌蛇（酒浸，湿纸裹煨，取肉）3两。
出处：《圣济总录》卷十五。
主治：首风。头项急疼，脑重，四肢拘急，行步有妨。
加减：None
功效：None
用法用量：每服5丸，空心温酒送下。
制备方法：上18味，先将牛黄、麝香同研，后渐入诸药末，并腻粉一半，炼蜜为丸，如梧桐子大，于一半腻粉内滚为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
