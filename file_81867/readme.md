序号：81883
名称：熟干地黄丸
组成：熟干地黄1两，当归半两（锉碎，微炒），芎半两，鳖甲1两（涂醋炙令黄，去裙襕），人参3分（去芦头），白芍药3分，白术3分，桂心半两，五味子半两，黄耆3分（锉），牛膝3分（去苗），附子3分（炮裂，去皮脐），陈橘皮1两（汤浸，去白瓤，焙），白茯苓3分，甘草1分（炙微赤，锉）。
出处：《圣惠》卷七十。
主治：妇人冷劳虚损，肌体消瘦，颜色萎黄，四肢无力，月候不调，少思饮食。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：熟地黄丸（《普济方》卷三一九）。
