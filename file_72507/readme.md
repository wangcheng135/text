序号：72523
名称：温补鹿茸丸
组成：鹿茸（去毛，酥炙）4两，人参1两，天雄（炮裂，去皮脐）1两，五加皮（锉）1两，五味子1两，牛膝（酒浸，切，焙）1两，防风（去叉）1两，远志（去心）1两，石斛（去根）1两，山芋1两，狗脊（去毛）1两，肉苁蓉（去皱皮，酒浸，切、焙）3两，熟干地黄（焙）3两，白茯苓（去黑皮）1两1分，菟丝子（酒浸，别捣）1两1分，覆盆子2两，石龙芮2两，萆薢3分，石南3分，蛇床子（炒，去皮）3分，白术3分，巴戟天（去心，酒浸，焙）1两半，天门冬（去心，焙）1两半，杜仲（锉，炒）1两半，干姜（炮裂）3分，桂（去粗皮）3分，吴茱萸（炒）3分，附子（炮裂，去皮脐）3分，细辛（去苗叶）3分，蜀椒（去目及闭口者，炒出汗）3分。
出处：《圣济总录》卷二十。
主治：阳气虚，阴气盛，痹气内寒，如从水中出。
加减：None
功效：None
用法用量：每服20丸，加至30丸，空心，食前酒送下，1日3次。
制备方法：上除菟丝子外共为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
