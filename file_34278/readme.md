序号：34287
名称：牡蒙丸
组成：牡蒙18铢，厚朴18铢，消石18铢，前胡18铢，干姜18铢、（庶虫）虫18铢，牡丹18铢，蜀椒18铢，黄芩18铢，桔梗18铢，茯苓18铢，细辛18铢，葶苈18铢，人参18铢，芎18铢，吴茱萸18铢，桂心18铢，大黄2两半，附子1两6铢，当归半两。
出处：《千金》卷四。
主治：妇人产后十二癥病，带下无子，皆是冷风寒气；或产后未满百日，胞络恶血未尽，便利于悬圊上，及久坐，湿寒入胞里，结在小腹，牢痛为之积聚，小如鸡子，大者如拳，按之跳手隐隐然，或如虫啮，或如针刺，气时抢心，两胁支满，不能食，饮食不能消化，上下通流，或守胃管，痛连玉门背膊，呕逆短气汗出，少腹苦寒，胞中创，咳引阴痛，小便自出；子门不正，令人无子，腰胯疼痛，四肢沉重淫跃，一身尽肿，乍来乍去，大便不利，小便淋沥，或月经不通，或下如腐肉，青黄赤白黑等如豆汁，梦想不祥。
加减：None
功效：None
用法用量：每服3丸，日3次，空心酒送下。不知则加至5-6丸。下赤白青黄汤如鱼子者，病根出矣。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：方下主治最繁，总不出冷风寒气四字为致病之纲，所以首推牡蒙、前胡专祛胞门风气，余皆因病变证之治法，与当归丸二方参看，其义自明。
用药禁忌：None
附注：None
