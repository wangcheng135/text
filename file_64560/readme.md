序号：64576
名称：理劳神功散
组成：秦艽1钱，续断1钱，杜仲1钱，香附7分，当归8分，骨碎补1钱，陈皮7分，甘草3分，五加皮8分，金毛脊8分，柴胡8分，葛根8分。
出处：《不居集》上集卷十。
主治：伤筋动骨，劳苦太过，损气耗血，而邪有不能外出者。
加减：若发热，加柴胡7分，干葛8分；若咳嗽，加白前、桔梗6分；若久嗽，加紫菀、百部8分；若腰痛，加破故纸1钱；若骨蒸夜热，加地骨皮、青蒿、鳖甲8分；若胸满，加砂仁、木香6分。
功效：None
用法用量：加生姜、大枣。
制备方法：None
临床应用：None
各家论述：用力太过，则气血不和而营卫虚；劳伤筋骨，则正气不充而邪易入。秦艽、续断善理劳伤；柴胡、葛根托邪外出；当归、杜仲养血舒筋，而宣通脉络；陈皮、香附宣郁壅滞，而理气宽中；骨碎补、金毛脊、五加皮活血荣筋．大能坚肾；生姜、甘草、大枣调和营卫，且能逐邪。
用药禁忌：None
附注：None
