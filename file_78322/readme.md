序号：78338
名称：酸枣汤
组成：酸枣仁2升，甘草1两，知母2两，茯苓2两，芎2两。
出处：《金匮》卷上。
主治：虚劳，虚烦不得眠，盗汗。
加减：None
功效：None
用法用量：以水8升，煮酸枣仁，得6升，纳诸药，煮取3升，分温3服。
制备方法：None
临床应用：1.失眠：用酸枣仁汤提取物2.5g，一日三次，连续四周，对31例失眠症患者进行治疗。结果：对“入睡”、“熟睡感”二项指标的效果较显著，给药二周即有良好疗效。对于“睡中觉醒”、“醒后舒适感”、“白天精神”等也有明显改善。综合评定：茯中等度以上改善者8例，占25.8%，轻度以上改善者20例，占64.5%。对病情轻者疗效较好。_x000D_
2.卑惵症：以酸枣仁汤原方加大剂量，治愈卑惵症1例。患者表现为怠情沉闷，日处内室，自愧无能，遇人则不胜(赤及)颜，惭惧羞怯，或隐匿而避之，烦而少眠，易惊颤，爪甲枯白而凹，唇舌色淡，脉弦细。原方服30剂告愈。后依方制丸，巩固四个月。随访五年，再未复发。
各家论述：1.《法律》：虚劳虚烦，为心肾不交之病。肾水不上交心火，心火无制，故烦而不得眠。方用酸枣仁为君，而兼知母之滋肾为佐，茯苓、甘草调和其间，芎入血分，而解心火之躁烦也。_x000D_
2.《古今名医方论》罗东逸：枣仁酸平，应少阳木化，而治肝极者，宜收宜补，用枣仁至二升，以生心血，养肝血，所谓以酸收之，以酸补之是也。顾肝郁欲散，散以川芎之辛散，使辅枣仁通肝调营，所谓以辛补之。肝急欲缓，缓以甘草之甘缓，防川芎之疏肝泄气，所谓以土葆之。然终恐劳极，则火发于肾，上行至肺，则卫不合而仍不得眠，故以知母崇水，茯苓通阴，将水壮金清而魂自宁，斯神凝魂藏而魄且静矣。此治虚劳肝极之神方也。_x000D_
3.《金匮要略心典》：虚劳之人，肝气不荣，则魂不得藏；魂不藏故不得眠。酸枣仁补肝敛气，宜以为君；而魂既不归容，必有浊痰燥火乘间而袭其舍者，烦之所由作也。故以知母、甘草清热滋燥，茯苓、川芎行气除痰，皆所以求肝之治而宅其魂也。_x000D_
4.《古方选注》，虚烦、胃不和、胆液不足，三者之不寐，是皆虚阳溷扰中宫，心火炎而神不定也，故用补母泻子之法，以调平之。川芎补胆之用，甘草缓胆之体，补心之母气也；知母清胃热，茯苓泄胃阳，泻心之子气也；独用枣仁至二升者，取酸以入心，大遂其欲而收其缓，则神自凝而寐矣。_x000D_
5.《成方便读》：凡有夜卧魂梦不安之证，无不皆以治肝为主；欲藏其魂，则必先去其邪。方中以知母之清相火，茯苓之渗湿邪；川芎独入肝家，行气走血，流而不滞，带引知、茯，搜剔而无余；然后枣仁可敛其耗散之魂，甘草以缓其急悍之性也。虽曰虚劳，观其治法，较之一于呆补者不同也。
