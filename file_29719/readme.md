序号：29727
名称：阿魏丸
组成：三棱（炒）10两，莱菔子（炒）10两，莪术（炙）10两，枳壳（炒）10两，槟榔10两，厚朴（炙）10两，枳实（炒）10两，全蝎10两，青皮（炒）10两，干蟾（烧）20两，大黄10两，山楂20两，砂仁10两，鸡内金（炒）80两，橘皮10两，阿魏40两，香附（炙）10两，芜荑40两。
出处：《北京市中药成方选集》。
主治：小儿宿食停滞，积聚痞块，呕吐腹胀。
加减：None
功效：健胃和中，消积化痞。
用法用量：每服5分，温开水送下，1日2次。周岁内儿酌减。
制备方法：上为细末，过罗，用冷开水泛为小丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
