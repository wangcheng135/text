序号：78342
名称：酸枣仁煎
组成：酸枣仁5两（半生半炒），萆薢1两，赤箭1两，羌活1两，海桐皮（锉）1两，仙灵脾1两，白蒺藜1两，石斛（去根，锉）1两，牛膝（去苗）1两，巴戟1两，附子（去皮脐）1两，当归1两，桂心1两，丹参1两，防风（去芦头）1两，芎1两，踯躅1两，骨碎补1两，羚羊角屑1两，木香1两，杜仲（去皱皮，锉）1两，狗脊1两，威灵仙1两。
出处：《圣惠》卷二十五。
主治：一切风。
加减：None
功效：None
用法用量：每服1茶匙，食前以温酒调下。
制备方法：上为末，以好酒5升，白蜜5合，同于银锅内以慢火熬，后下诸药末，用柳木篦不住手搅，令稀稠得所，入麝香末1钱令匀，于瓷器中盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
