序号：47835
名称：神仙紫霞丹
组成：朱砂1两，天南星、半夏各等分（为末），取1小粉盒子，先将前末药1钱铺在下，再安朱砂放中，上又以末药1钱盖头顶，次用赤石脂（为末）水调糊缝口，外用铁线扎定，以六一盐泥固济，厚1寸许，6-7日后先以慢火养半日，后用火煅通红为度，候冷取出，研为末。次用：禹余粮4两，针头代赭石2两，紫石英1两（上用火煅通红，以好醋淬7遍），赤石腊3两（生用），硫黄（醋煮干）1两，附子2个（炮制，去皮脐），海螵蛸1两（去壳）。
出处：《普济方》卷二六五引《余居士选奇方》。
主治：妇人子宫久冷，血海虚惫。
加减：None
功效：久服壮元阳，益真气。
用法用量：每服2-5丸，空心煎浓枣汤送下。
制备方法：上为细末，再用乳钵研无声为度，次用新糯米粽烂研为丸，如黍米大，盛于新布袋中，将桂府滑石打作片子，同药打光，瓷盒盛之。
临床应用：None
各家论述：None
用药禁忌：忌猪、羊血。
附注：None
