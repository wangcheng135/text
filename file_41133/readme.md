序号：41148
名称：金牙散
组成：金牙（别研）、雄黄（研）、丹砂（研）、矾石（泥裹烧半日）、寒水石、芫青（熬）、巴豆（去心皮，熬）、朴消、桔梗、茯苓、人参、贯众、附子（炮）、蜀椒（去汗，目）、露蜂房（炙）、龙骨、干姜、牡桂、乌头（炮）、石膏（研）、莽草（炙）、苁蓉、大戟、芫花（熬）、防风、貍骨、商陆根、大黄、细辛、蛇蜕（炙）、玉支（1作玉泉）、贝母（1作牙子，即狼牙也）各等分。
出处：《外台》卷十三引《集验方》。
主治：邪魅，心腹刺痛。
加减：None
功效：None
用法用量：每服5分匕，酒调下，1日3次。
制备方法：上药治下筛。
临床应用：None
各家论述：None
用药禁忌：忌猪肉、冷水、生菜、生血肉、大醋、芦笋。
附注：None
