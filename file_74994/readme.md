序号：75010
名称：馏水石膏饮
组成：生石膏（轧细）2两，甘草3钱，麻黄2钱。
出处：《衷中参西》上册。
主治：胸中先有蕴热，又受外感，胸中烦闷异常，喘息迫促，其脉浮洪有力，按之未实，舌苔白而未黄者。
加减：None
功效：None
用法用量：上用蒸汽水（僻处若无汽水，可用甘澜水代之）煎2-3沸，取清汤1大碗，分6次温服。前3次，一点钟服1次，后3次，一点半钟服1次。病愈则停服，不必尽剂，下焦觉凉者，亦宜停服。
制备方法：None
临床应用：伤寒：奉天东站经理矿务钱慕韩，愚之同乡，其妇人于仲冬得伤寒，四五日间喘不能卧，胸中烦闷异常，频频呼唤，欲自开其胸。诊其脉浮洪而长，重按未实，舌苔白厚。知其证虽入阳明，而太阳犹未罢也。为拟此方，汤成后，俾徐徐分六次服之，因病在上焦，若顿服，恐药力下趋，则药过病所，而病转不愈也。服至三次，胸间微汗，病顿见愈，服至尽剂，病愈十之八、九。再诊其脉，关前犹似浮洪，喘息已平，而从前兼有咳嗽未愈。继用玄参一两、杏仁（去皮）二钱，蒌仁、牛蒡子各三钱，两剂全愈。
各家论述：此方取汽水轻浮之力，能引石膏上升，以解胸中之烦热；甘草甘缓之性，能逗留石膏不使下趋，以专其上行之力；少佐麻黄解散太阳之余邪，兼借以泻肺定喘，而胸中满闷可除。
用药禁忌：None
附注：None
