序号：48569
名称：神秘万灵丹
组成：何首乌（去皮，用黑豆九蒸九晒，忌铁器）5钱，川当归（酒浸）5钱，两头尖5钱，川乌（去尖，用火炮）2两，草乌（去尖，用火炮）2两，大茴香2两，川芎2两，人参（去芦）2两，防风（去芦尾）2两，白芷2两，荆芥穗2两，桔梗（米泔浸）2两，麻黄（水煮4沸，去节）2两，炙甘草2两，天麻2两，白术（米泔浸）5钱，木香（不见火）5钱，辽细辛5钱，血竭（另研）5钱，苍术半斤（米泔洗过，入酒浸1宿，晒干，为末）。
出处：《鲁府禁方》卷三。
主治：妇人一切胎前产后诸般病症，三十六种冷血风，八十二种风疝病，乳中风，淋血，胎孕不安，死胎不下，胎衣不下，产后腹内搅痛，脐下如刀刺者；胎前产后，赤白带下，呕逆填塞，心气烦满；经脉不通，或来频并，饮食无味，面赤唇焦，手足顽麻，遍身生黑点血癍者；及产后伤寒中风，体如板者。
加减：None
功效：None
用法用量：每服1丸，细嚼黄酒送下；产后伤寒中风，体如板者，用麻黄汤送下。
制备方法：上为细末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
