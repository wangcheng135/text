序号：51355
名称：衍庆丸
组成：当归（酒洗）4两，肉苁蓉（酒洗）4两，山药（乳拌蒸）4两，枸杞4两（酒蒸），鱼胶1斤（麸炒），核桃肉10两（去皮，捣烂），补骨脂1斤（米泔水加盐浸，春2、夏1、冬5日），菟丝子8两（酒漫1宿，煮吐丝），熟地4两（酒洗），吴茱萸3两（酒蒸，炒），覆盆子4两（酒浸），人参5钱（黄耆煎汤浸透，晒干）。
出处：《惠直堂方》卷一。
主治：身体衰弱，精气不足，肾虚寒冷，偏坠疝气。
加减：None
功效：补益身体，强精种子。
用法用量：空心淡盐汤送下，初服1钱，次服1钱5分，3服2钱，4服2钱5分，5服3钱。初服禁房事3-7日，便觉药力有效，至3月后，用酒送下3钱，即可得孕。
制备方法：上炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：忌食猪、鸡、鹅、鸭等血，并君达菜。
附注：方中鱼胶，《全国中药成药处方集》（沈阳方）作“阿胶”。
