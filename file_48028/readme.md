序号：48043
名称：神机万应秘传膏
组成：香白芷5钱，两头尖5钱，赤芍药5钱，白芍药5钱，生地黄5钱，熟地黄5钱，当归1两（1个），蓖麻子50粒，木鳖子50个，巴豆50个，乳香5钱，没药5钱，五灵脂5钱，阿魏5钱，穿山甲（大者）5个（炙黄，为末），黄丹1斤（飞过，炒至黑色），槐枝（用木许筋大）48根，柳枝（与槐同）48根，香油2斤（真者）。
出处：《摄生众妙方》卷八。
主治：一切疔疮，发背，无名肿毒，臁疮，对口，风气，癣疥。
加减：None
功效：None
用法用量：用时以绢绫摊之为上，纸次之，凡用贴疔疮，以火焙手熨300度，发背等疮200度，无名肿毒150度，臁疮、对口130度，风气170度，癣疥100度，余不拘。
制备方法：先将巴豆以上诸药切为细片，乳香以下诸药研为细末，将香油2斤放瓷罐中，入巴豆以上药浸之，春5日、夏3日、秋7日、冬10日，浸毕取出，入铜锅内，并入槐、柳枝，文武火熬至槐、柳枝黑色为度，用细绢滤去药滓，再入黄丹在油内同熬，外以槐枝1尺（比筋大者）频频搅之，看火色将好，油已成膏，滴水如钱，方入乳香以下诸药末，愈加频频搅良久，至药提起有细丝三五七根、尺长不断，然后盛入2-3小瓷罐内，放土地内以受五行之气月余方可用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
