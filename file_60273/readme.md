序号：60288
名称：通气丸
组成：附子（大者）1只（生，去皮脐，切薄片），大蒜头5枚（剥去皮苗，捶令碎），赤小豆5两（拣净。以上3味放于砂锅内，加水3升煮，渐添至5升，慢火煮干为度，只取附子焙干为末，余药不用），白花商陆根半两，南木香、沉香2钱（不见火），车前子。
出处：《魏氏家藏方》卷九。
主治：脾肾气虚，肾水流溢，四肢作肿，面目虚浮，腰脚肿胀，游走不定，小便赤涩，大便秘结，胀满气痞，脚膝无力，食少倦怠，渐成水肿。
加减：None
功效：None
用法用量：每服40-50丸，空心、食前以薏苡仁煎汤送下，1日2-3次，病重者1日5次。
制备方法：上同附子为细末，用薏苡仁末水煮作糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中南木香、车前子用量原缺。
