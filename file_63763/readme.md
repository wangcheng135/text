序号：63779
名称：清热解郁汤
组成：龙胆草6g，丹皮9g，生地9g，白茅根15g，赤芍9g，白芍9g，银柴胡9g，金银花9g，连翘9g，山栀9g，竹叶6g，枳壳6g，郁金9g。
出处：《临证医案医方》。
主治：慢性肝炎郁热型。肝区痛，有热感，五心烦热，舌尖红，脉弦微数。
加减：None
功效：清热凉血，舒肝解郁。
用法用量：None
制备方法：None
临床应用：None
各家论述：方中用生地、丹皮、茅根、赤芍凉血清热；山栀、竹叶引热下行；金银花、连翘清热，散热结；郁金、枳壳疏肝理气解郁，郁解则热除；银柴胡可清虚热，有调节少阳的作用，可使血分之热邪转到气分而解。诸药配伍，共奏清热凉血，疏肝解郁之功。
用药禁忌：None
附注：None
