序号：77923
名称：螽斯胜宝丸
组成：黄耆（蜜炙）2两，人参（去芦）2两，白术（去芦）2两，白茯苓（去皮）2两，当归（酒洗）2两，川芎2两，白芍（酒炒）2两，肉桂2两，大附子（面裹，火煨，去皮）2两，干姜（炒）2两，胡椒2两，小茴香（盐、酒炒）2两，破故纸（酒炒）2两，艾叶（醋炒）2两，乌药（炒）2两，吴茱萸3两（盐水炒），香附6两（醋炒），苍术4两（米泔浸，炒），甘草（炙）1两。
出处：《回春》卷六。
主治：妇人经水不调，脐腹冷痛，赤白带下，一切虚寒之疾，久无子嗣。
加减：None
功效：None
用法用量：每服50丸，空心好酒吞下。
制备方法：上锉作片，用白毛乌骨鸡1只，重1斤半或2斤者，吊死，水泡，去毛、肠屎并头、脚、翼尖不用；将鸡放砂锅里，将前药盖上，入好酒煮烂为度；取去骨，同药在锅焙干，为末，将煮鸡酒汁打稀米糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
