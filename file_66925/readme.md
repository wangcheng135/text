序号：66941
名称：温卫补血汤
组成：生地黄1分，白术1分，藿香1分，黄柏1分，牡丹皮2分，苍术2分，王瓜根2分，橘皮2分，吴茱萸2分，当归身2分半，柴胡3分，人参3分，熟甘草3分，地骨皮3分，升麻4分，生甘草5分，黄耆1钱2分，丁香1个，桃仁3个，葵花7朵。
出处：《兰室秘藏》卷中。
主治：耳鸣，鼻不闻香臭，口不知谷味，气不快，四肢困倦，行步欹侧，发脱落，食不下，膝冷，阴汗，带下，喉中介介，不得卧，口舌益干，太息，头不可以回顾，项筋紧，脊强痛，头旋眼黑，头痛欠嚏。
加减：None
功效：None
用法用量：上(口父)咀，作1服。用水2大盏，煎至1盏，去滓，食前热服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：温胃补血汤（《医学正传》卷七）。
