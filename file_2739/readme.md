序号：2739
名称：天王补心丹
组成：人参5钱，五味子5钱，当归（酒洗）5钱，天门冬（去心）5钱，麦门冬（去心）5钱，柏子仁5钱，酸枣仁（炒）5钱，玄参5钱，白茯神（去皮）5钱，丹参5钱，桔梗（去芦）5钱，远志（去心）5钱，黄连（去毛，酒炒）2两，生地黄（酒洗）4两，石菖蒲1两。
出处：《回春》卷四。
主治：健忘。内伤嗽血。
加减：None
功效：宁心安神，益血固精，壮力强志，令人不忘，除怔仲，定惊悸，清三焦，化痰涎，祛烦热，疗咽干，养育精神。
用法用量：每服30丸，临卧时灯心、竹叶煎汤送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
