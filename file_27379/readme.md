序号：27387
名称：补肾熟干地黄丸
组成：熟干地黄（切，焙）2两，肉苁蓉（酒浸，切，焙）2两，磁石（煅，醋淬）2两，山茱萸3分，桂（去粗皮）1两，附子（炮裂，去皮脐）1两，山芋3分，牛膝（酒浸，切，焙）1两，石南3分，白茯苓（去黑皮）3分，泽泻3分，黄耆（锉）3分，鹿茸（去毛，酥炙）2两，五味子3分，石斛（去根，锉）1两，覆盆子3分，远志（去心）3分，补骨脂（微炒）2两，萆薢（锉）3分，巴戟天（去心）3分，杜仲（去粗皮，炙，锉）1两，菟丝子2两（酒浸，别捣），白龙骨1两。
出处：《圣济总录》卷二十。
主治：肾虚骨痹，面色萎黑，足冷耳鸣，四肢羸瘦，脚膝缓弱，小便滑数。
加减：None
功效：None
用法用量：每服30丸，空心以温酒送下，日3次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
