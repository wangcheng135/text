序号：39717
名称：经进地仙丸
组成：川牛膝（酒浸1宿，切、焙）4两，肉苁容（酒浸1宿、切，焙）4两，川椒（去目）4两，附子（炮）4两，木鳖子（去壳）3两，地龙（去土）3两，覆盆子2两，白附子2两，菟丝子（酒浸，研）2两，赤小豆2两，天南星2两，防风（去芦）2两，骨碎补（去毛）2两，何首乌2两，萆薢2两，川羌活2两，金毛狗背（去毛）2两，乌药2两，绵黄耆1两，人参1两，川乌（炮）1两，白茯苓1两，白术1两，甘草1两。
出处：《易简》引陶隐居方（见《永乐大典》卷一一六二○）。
主治：丈夫妇人，五劳七伤，肾气衰败。精神耗散，行步艰辛，饮食无味，耳焦眼昏，皮肤枯燥。妇人脏冷无子，下部秽恶，肠风痔漏，吐血泻血，诸风诸气。肾脏虚惫，风湿流注，脚膝痠疼，行步无力。
加减：None
功效：None
用法用量：每服30-40丸，空心温酒送下。
制备方法：上为细末，酒煮面糊为丸，如梧桐子大。
临床应用：风气疾时有人母，幼年得风气疾，久治不愈五十余年。隐居处此方修合，日进二服。半年，母病顿愈，发白返黑，齿落再生，至八十岁，颜色如少年人，血气筋力倍壮，耳目聪明。其家老仆七十余岁，窃服此药，遇严御（纟希）葛，履霜雪无寒色，有别业去家七十里，每使老仆，往返不移时，又能负重，非昔时比，几成地仙。
各家论述：None
用药禁忌：None
