序号：61927
名称：救亡生阴汤
组成：人参2两，熟地4两，山茱萸2两，北五味5钱，茯神1两，白芍1两。
出处：《辨证录》卷六。
主治：中暑热症，发汗亡阴，大汗如雨，一出而不能止者。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：此方熟地、山萸、五味子均是填精补水之味，茯神安其心，白芍收其魂，人参回其阳，此人之所知也。阴巳外亡，非填其精髓，何以灌溉涸竭之阴；阳以外亡，非补其关元，何以招其散失之阳？山萸，五味补阴之中，仍是收敛之剂，阴得补而水生，则肾中有本；汗得补而液转，则心内无伤；又得茯神以安之，白芍以收之，则阳回阴返，自有神捷之机也。
用药禁忌：None
