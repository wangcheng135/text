序号：8179
名称：加减补中益气汤
组成：人参5钱，黄耆3钱（生用），柴胡1钱，甘草1分，当归3钱（酒洗），白术5钱（土炒），茯苓1两，升麻3分，陈皮3分。
出处：《傅青主女科》卷下。
主治：妊娠5月，脾肺气虚，肢体倦怠，饮食无味，先两足肿，渐至头面遍身俱肿。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：补中益气汤原为升提脾肺之气，血非气不生，补气即所以生血。今湿气乘脾肺之虚而相犯，未便大补其血，恐阴太盛而招阴也。只补气而助以利湿之品，则气升而水尤易散，血亦随之而生矣。重用茯苓一两为君，于补气之中，虽曰渗湿，而仍是健脾清肺之意。且凡利水之品，多是耗气之药，而茯苓与参术合用，实补多于利，所以重用之以分湿邪，即以补气血耳。
用药禁忌：None
