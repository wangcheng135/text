序号：2796
名称：天灵盖饮子
组成：天灵盖半两（涂酥，炙微黄，捣为末），鳖甲半两（涂醋，炙微黄，去裙襴，捣为末），桃仁10枚（汤浸，去皮尖双仁，麸炒微黄），柴胡半两（去苗），知母半两，青蒿半两，甘草1分（生用），豉心半合，葱白2茎（并须）。
出处：《圣惠》卷三十一。
主治：骨蒸劳，四肢疼痛，筋脉拘急，寒热进退，发作如疟，日渐萎黄，不能饮食。
加减：None
功效：None
用法用量：上锉细，拌令匀，都用童便3大盏从午时浸至来日五更，煎取1盏，去滓，食前分2次温服；服讫衣盖，卧至日出。良久审看手十指，节间有毛，如藕丝状，烧之极臭，毛色白者必愈；黑者难治。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌苋菜。
附注：None
