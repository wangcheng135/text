序号：54625
名称：柴胡鳖甲汤
组成：柴胡（去苗）1两，鳖甲（小便浸3日，逐日换小便，炙黄，去裙襕脊骨）1两半，秦艽（去苗土）1两，桔梗（炒）1两，人参1两，芎藭1两，当归（切，焙）1两，白茯苓（去黑皮）1两，桂（去粗皮）1两，槟榔（锉）1两，紫菀（去苗土）1两，桑根白皮（锉）1两，地骨皮1两，生干地黄（焙）1两，白术1两，知母（焙）1两，芍药1两，甘草（炙，锉）3分。
出处：《圣济总录》卷八十八。
主治：虚劳潮热，心神烦躁，咳嗽盗汗，肢节痠痛，夜卧不安。
加减：None
功效：None
用法用量：每服3钱匕，水、童便各半盏，同煎至7分，去滓，通口服，空心、日午、临卧各1次。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
