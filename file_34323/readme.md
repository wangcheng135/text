序号：34332
名称：万金丸
组成：天雄1两半，附子1两半，草乌头1两半（并炮，去皮脐尖），骨碎补3两（去毛），白胶香2两（研），川乌头1两2钱（炮，去皮脐尖），防风1两（去芦头），天南星1两（炮），锡蔺脂1两（甘锅盛，火煅7遍，醋淬7遍），萆薢1两，白附子1两（炮），自然铜1两（火煅7遍，醋淬7遍），胡芦巴1两，破故纸1两，白僵蚕1两（炒，去丝嘴），五灵脂1两（炒），赤小豆1两（生用），糯米1两（炒焦），乳香1两（别研），没药（别研）1两。
出处：《杨氏家藏方》卷一。
主治：男子妇人，左瘫右痪，口眼喎斜，打扑伤损，筋骨疼痛，语言謇涩，行步艰难；一切风疾，下注腰脚，鹤膝风等。
加减：None
功效：None
用法用量：每服20丸，如腰以上疼痛等病，食后细嚼，荆芥茶送下；如腰以下疼痛等病，食前嚼，胡桃酒送下。
制备方法：上为细末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
