序号：77526
名称：干地黄散
组成：生干地黄5斤，苁蓉8两，白术8两，巴戟天8两，麦门冬8两，茯苓8两，甘草8两，牛膝8两，五味子8两，杜仲8两，车前子5两，干姜5两。
出处：方出《千金》卷十九，名见《圣济总录》卷五十一。
主治：肾气虚寒，阴痿，腰脊痛，身重缓弱，言音混浊，阳气顿绝。
加减：None
功效：None
用法用量：每服方寸匕，食后酒送下，1日3次。
制备方法：上药治下筛。
临床应用：None
各家论述：《千金方衍义》：此以肾气不能流布中外而腰痛身重，阴痿阳绝，故用地黄、牛膝、五味滋阴，苁蓉、巴戟、杜仲助阳，干姜、白术、甘草固本，麦冬、茯苓、车前治标，滋培气化以资通调之力。
用药禁忌：None
附注：None
