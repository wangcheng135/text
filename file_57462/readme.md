序号：57477
名称：润字丸
组成：大黄3两，前胡1两半，枳实1两半（炒），杏仁2两，牙皂1两半，花粉3两，槟榔1两，楂肉3两（炒），橘红1两半，半夏1两半（制）。
出处：《医略六书》卷十九。
主治：实痞喘嗽，大便闭结，脉沉者。
加减：None
功效：疏痰通闭。
用法用量：每服2-3钱，空心白滚汤化下。
制备方法：上为末，水泛为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：痰实内壅，不得施化，而大便闭结，遏热刑金，故喘嗽不止焉。杏仁疏痰降气，牙皂搜风涤痰，橘红利气化痰，半夏燥湿化痰，楂肉消滞化积，前胡降气疏痰，槟榔破滞气以消积，枳实攻坚积以推陈，花粉清热邪壅结，大黄荡地道不通。泛丸汤下，俾痰消热降，则胸宇廓然，而肺金清肃，喘嗽自宁；津液施化，大便无闭结之患矣。此疏痰疾通闭之剂，为痰实顺嗽秘结之专方。
