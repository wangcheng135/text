序号：30447
名称：还睛紫金丹
组成：白沙蜜20两，甘石10两（烧7遍，碎，连水浸拌之），黄丹6两（水飞），拣连3两（小便浸，碎为末），南乳香3钱，当归3钱，乌鱼骨2钱，硇砂（小盏内放于瓶口上熏干）1钱，麝香1钱，白丁香（直者）5分，轻粉1字。
出处：《兰室秘藏》卷上。
主治：目眶岁久赤烂。眼生倒睫拳毛，两目紧盖，内伏火热而攻阴气。
加减：None
功效：None
用法用量：每用1丸，温水化开洗。
制备方法：上将白沙蜜于沙石器内，慢火去沫，下甘石，次下丹，以柳枝搅，次下余药，以粘手为度，为丸如鸡头大。
临床应用：None
各家论述：None
用药禁忌：None
附注：治目眶岁久赤烂，当以三棱针刺目眶外，以泻湿热；如眼生倒睫拳毛，法当去其热内火邪，眼皮缓则毛立出，翳膜亦退，用手法攀出，内睑向外，以针刺之出血。
