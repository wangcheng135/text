序号：72297
名称：滋肾复明汤
组成：熟地15g，枸杞子9g，桑椹子12g，菟丝子9g，女贞子9g，车前子9g，肉苁蓉9g，青盐少许。
出处：《张皆春眼科证治》。
主治：肾精亏虚，视物不见，眼内干涩，头晕耳鸣，腰痠遗精，脉细弱者。
加减：None
功效：滋补肾阴。
用法用量：None
制备方法：None
临床应用：None
各家论述：方中熟地、桑椹子、女贞子大补肾阴以填精；枸杞子、菟丝子、肉苁蓉阴阳双补以温肾，阴生阳长，神光自足；车前子利水道固肾窍，既防诸药腻膈伤胃，又不伤阴；青盐乌发明目，且能引诸药直达肾经。若见阴虚火旺者，可去菟丝子、枸杞子、肉苁蓉甘温之品，加知母、元参、牡丹皮育阴清热之剂。
用药禁忌：None
附注：None
