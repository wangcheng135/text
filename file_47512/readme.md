序号：47527
名称：神丹
组成：狼毒5钱，巴豆2两，蓖麻2两，芫花2两，蜜4两，辰砂不拘多少。
出处：《鸡峰》卷二十九。
主治：肌肉如柴瘦，皮如树皮，痰多不食者。
加减：None
功效：None
用法用量：每服量病浅深虚实，寻常以2-3丸，空心以米饮送下。
制备方法：先将好纸淹烂和泥，入盐2两，将成砂子，以纸裹，用线系定，以蜜先滚了，次滚狼毒末，次将蜜和麻、巴、芫三末固济砂子，入泥再固济，周围两指厚，日中晒干；以醋和灰2-3斗自炉底固济，次和泥毱包之，厚两指，次发火，用硬炭25斤（如无，用软炭30斤）；再用炭30斤依前法煅之，用好砂5两为一料，用前草药分两固济，候煅了，研极细，以半夏曲糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
