序号：6803
名称：加味益气养血救脱汤
组成：酸枣仁（炒）36g，制何首乌9g，玉竹9g，熟附子12g，生菟丝子24g，炙黄耆12g，炒白术15g，归身9g，丹参12g，柏子仁12g，砂仁9g，益智仁9g，覆盆子12g，鸡血藤9g，竹茹9g，红花6g。
出处：方出《刘惠民医案》，名见《千家妙方》卷上。
主治：营养不良性干瘦病，属脾气久亏，气血虚极，元阳欲脱者。
加减：None
功效：益气养血，温肾助归。
用法用量：水煎2次，得药液约250毫升。分2次温服。另用人参2g，琥珀0.9g，共为极细末，分2次冲服。服药3付，休药1天，有效再服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
