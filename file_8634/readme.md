序号：8634
名称：玄青丸
组成：黄连半两，黄柏半两，大黄半两，甘遂半两，芫花（醋拌炒）半两，大戟半两，牵牛（4两取末）2两（以上同细末），轻粉2钱，青黛1两。
出处：《宣明论》卷十。
主治：下痢势恶，频并窘痛，或久不能止，须可下之，以开除湿热痞闷积滞而使气液宣行者。积热，酒食积，黄瘦中满，水肿腹胀。小儿惊疳，积热乳癖诸证。
加减：None
功效：None
用法用量：初服10丸，每服加10丸，空腹、日午、临卧3服。以快利为度。后常服15、20丸，数日后得食。久病未痊除者，再加取利，利后却常服，以意消息，病去为度，后随证止之。小儿丸如黍米或麻子大。退惊疳，热积不下者，须常服10丸。
制备方法：上为末，水为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：唯泄泻者勿服。
附注：玄青丹（《赤水玄珠》卷五）。
