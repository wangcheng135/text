序号：6613
名称：加味麻风丸
组成：大胡麻1斤4两，小胡麻1斤4两，川牛膝4两，白蒺藜1斤4两，苦参1斤，防风8两，荆芥8两，当归6两，茅苍术6两，川断4两，苡仁4两，黄柏6两，浮萍20两，马齿苋24两。
出处：《青囊秘传》。
主治：疠风初起。
加减：None
功效：None
用法用量：每日早、午、晚3次，每服3钱或2钱。每丸1钱，照数加风子膏，春、秋用8厘，夏用5厘，冬用1分，拈圆，以茅尖茶叶1分煎汤过口。
制备方法：上为细末，水泛为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：制风子膏法：风子肉铜锅内炒至三分红色，七分黑色为好，太过无力，不及伤眼。炒后研成膏，如红沙糖一样，用铜勺盛，置火上熬四五滚，倒在纸上，放土面上，以物盖之待用。如上面霉，拭去后，仍可使用。
