序号：81932
名称：镇心丸
组成：紫石英2两（研），丹砂1两（研），雄黄（研）2两，白茯苓（去黑皮）2两，茯神（去木）2两，银屑2两，菖蒲2两，桔梗（去芦头，炒）2两，人参2两，干姜（炮）2两，远志（去心）2两，甘草（炙，锉）2两，防风（去芦头）1两，防己1两，当归（切，焙）1两，桂（去粗皮）1两，铁精1两，细辛（去苗叶）1两。
出处：《圣济总录》卷一○○。
主治：心气怯弱，常多魇梦，恍惚谬忘。
加减：None
功效：None
用法用量：每服10丸，食后熟水送下，日3次，稍增之。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
