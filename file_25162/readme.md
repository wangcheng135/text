序号：25170
名称：防风赤芍汤
组成：双防己2两，木防己3两，赤芍药1两半，秦艽肉2两，苡米仁5两，宣木瓜3两，川续断3两（酒炒），杜牛膝3两（酒炒）。
出处：《医略六书》卷三十。
主治：产后瘀血内滞，湿热不化，下注而致脚气，足胫红肿疼痛不止，脉数涩大者。
加减：None
功效：None
用法用量：每服3钱，水煎，去滓温服。
制备方法：上为散。
临床应用：None
各家论述：汉防己泻血分湿热，木防己散血分风邪，赤芍破瘀血以通经，米仁渗湿热以除痹，秦艽通肌活血，木瓜醒脾舒筋，续断续经脉，牛膝利血脉也。为散水煎，使瘀血消散，则湿热自化而经脉清和，何红肿疼痛之不退哉。
用药禁忌：None
附注：None
