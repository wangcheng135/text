名称：日精月华光明膏
组成：炉甘石8两，黄丹8两，绿豆粉（炒黑）4两，黄连1两，当归2钱，朱砂2钱，硼砂2钱，玄明粉2钱，决明粉2钱，轻粉1钱，白矾（生）1钱，白丁香1钱，海螵蛸1钱，自然铜1钱，硇砂1钱，熊胆2分半，乳香2分半，没药2分半，鹰条2分半，雄黄2分半，青盐2分半，胆矾2分半，铜青2分半，牙消2分半，山猪胆2分半，麝香5分，片脑1钱，樟脑半钱（又方有贝子煅1钱，贝齿1钱，石燕1钱，石蠏1钱，水晶1钱，真珠1钱，玛瑙1钱，琥珀1钱，珊瑚1钱，若加此9味，要去绿豆粉不用，有豆粉即半真半伪，上各制为细末，依方称合和匀，（石禹）至千万余下，熟绢罗过，入后膏子成剂），鸡柏根1斤，黄连半斤，龙胆草2两，黄柏2两，生地黄2两，苦参2两，大黄1两，黄芩1两，栀子1两，赤芍药1两，防风1两，菊花1两，玄参1两，当归1两，羌活5钱，木贼5钱，蒺藜5钱，连翘5钱，蔓荆子5钱，细辛5钱，川芎5钱，白芷5钱，夜明砂2钱半，蛇蜕2钱半，蝉蜕2钱半，冬蜜半斤（又方福建地有后11味草药在内，用之效速，他处无此草药，不用亦效：苦花子生采1握，地薄荷生采1握，地西瓜生采1握，田茶菊生采1握，七层楼生采1握，千里光生采1握，铁梗子生采1握，地园荽生采1握，地胡椒生采1握，蛇不见生采1握，水杨梅根皮生采1握，捣烂另煎取浓汁，入前药同煎成膏）。
出处：《准绳·类方》卷七。
主治：翳膜胬肉，诸般眼疾。
加减：None
功效：None
用法用量：每以少许，井水化开，鸭毛蘸点眼，又以热汤泡化洗眼。
制备方法：上锉，入井水于铜器内浸3宿，慢火煎熬浓汁，滤去滓，以滓再煎再滤，慢火煎熬，槐、柳、桑枝搅，熬如饴糖，入蜜和匀。更入羯羊胆2枚，雄猪胆2枚和匀，瓷碗顿放，汤瓶口上蒸成膏，复滤净，滴沉水中成珠，可丸为度。待数日出火毒，再溶化，入诸药末和匀，杵丸为锭，阴干，用金银箔为衣。
