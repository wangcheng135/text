序号：68491
名称：黄连丸
组成：黄连不限多少，生栝楼汁、生地黄汁、羊乳（无，即用牛乳及人乳亦得）。
出处：方出《千金》卷二十一，名见《千金》同卷注文引《张文仲方》。
主治：岭南山瘴气兼风热毒气入肾中，变成寒热脚弱，虚满而渴。脾胃有热，烦渴不止。
加减：None
功效：None
用法用量：每服30丸，渐加至40-50丸，以麦饮送下，每日3次。轻者3日愈，重者5日愈；若药苦难服，即煮麦饮汁送下亦得。
制备方法：取三般汁乳和黄连末任多少，众手捻为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌猪肉、芜荑。
附注：羊乳丸（《千金》卷二十一注文引《张文仲方》）、黄连羊乳丸（《圣济总录》卷五十九）。
