序号：33619
名称：返元丸
组成：川芎5钱，羌活5钱，独活5钱，细辛5钱，白芷5钱，当归5钱，黄耆5钱，牛膝5钱，蝉退5钱，狗脊5钱，首乌5钱，全蝎5钱，防风20两，苦参1斤，大枫子20两，牛黄2钱，血竭5两。
出处：《秘传大麻疯方》。
主治：紫云疯，形如紫云。
加减：如鼻塞，加皂角4两；骨节痛，加谷树皮4两；眉落，加皂针灰3钱；生姜汁擦；通身疮，加羌活，独活各5钱；眼目昏花发热，加姜黄1两，又用桃、槐、柳、椿、桑枝1把煎汤洗浴。
功效：None
用法用量：每服50丸，渐加至100丸为度，清茶送下。如未全愈，再服防风通圣散。
制备方法：上除血竭、牛黄、枫子三味另研，余药为末，入三味和匀，老米粉糊为丸。
临床应用：None
各家论述：None
用药禁忌：忌酒色劳碌，猪、羊、鲜鱼、油腻、生冷、动气、发风之物，只吃白鸭、鲞鱼，食淡更妙，重者半年，轻者三月。
附注：None
