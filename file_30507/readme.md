序号：30515
名称：还睛神明酒
组成：黄连5两，石决明3两，草决明3两，生姜3两，石膏3两，蕤仁3两，黄消石3两，山茱萸3两，当归3两，黄芩3两，沙参3两，车前子3两，淡竹叶3两，朴消3两，甘草3两，芍药3两，柏子仁3两，川乌头3两，泽泻3两，桂心3两，荠子3两，地肤子3两，桃仁（去皮尖双仁者）3两，防风3两，辛夷3两，人参3两，川芎3两，白芷3两，细辛3两，瞿麦3两，龙脑3钱，丁香半两，珠子（生）25颗，秦皮3两。
出处：《苏沈良方》卷七。
主治：目盲，瞳子俱损，翳如云，赤白肤肉如乳头。
加减：None
功效：None
用法用量：食后服半合。勿使醉吐，稍稍增之。100日后，目明如旧。
制备方法：上（口父）咀，绢囊盛，用好酒5斗，瓮中浸之，春、秋14日，夏7日，冬21日。
临床应用：失明：晋大夫于公失明经十二余年，不辨明夜，两目俱损，无瞳子，时年七十，服此酒一百日，万病除，两目明，见物益明。
各家论述：None
用药禁忌：忌热面、鮓、葵、秽臭五辛，鸡、鱼、猪、马、驴肉、生冷粘滑、入房、恚怒、大忧愁、大劳、大寒热悉慎之。
附注：None
