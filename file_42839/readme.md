序号：42854
名称：保和丸
组成：白术（泔浸，土炒）2两，苍术（泔浸，炒）2两，厚朴（姜汁制）2两，陈皮（去白）2两，甘草（炙）5钱，莪术（醋炒）1两，三棱（醋炒）2两，香附（炒）2两，砂仁（炒）5钱，益智（炒）6钱，萝卜子（炒）1两，山药8钱，人参（去芦）5钱，肉果（去油）40个，白豆蔻4钱，槟榔3个，木香5钱，神曲（炒）1两，麦芽（炒取粉）、山楂2两，茯苓（去皮）1两，使君子肉1两，干荸荠1两。
出处：《直指小儿附遗》卷四。
主治：小儿乳食所伤，吐泻积滞，肚腹疼痛。
加减：None
功效：None
用法用量：每服1丸，米饮化下；吐多，生姜汤化下。
制备方法：上为细末，炼蜜为丸，如龙眼大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中麦芽用量原缺。
