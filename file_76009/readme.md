序号：76025
名称：葶苈丸
组成：甜葶苈1两（长流水洗净，微火熬，捣为末），山杏仁半两（汤浸，去皮），盐花2钱。
出处：《圣惠》卷三十六。
主治：耳聋，耳中常有声哄哄声者。
加减：None
功效：None
用法用量：每次绵裹1丸，纳耳中，2日1换。初安药，2-3日耳痛，出恶脓水，四体不安，勿惧之。
制备方法：上为末，更入腊月猪脂1钱，和研如泥，硬软得所，丸如枣核大。
临床应用：None
各家论述：None
用药禁忌：一百日内，慎一切毒、鱼、肉、生冷、滑腻等。
附注：葶苈膏（《圣济总录》卷一一四）。《圣济总录》本方用法：上捣研极烂，入猪膏中，以银器盛，慢火煎成膏，倾入瓷盒中。以绵裹枣核大，塞耳中。
