序号：59319
名称：莨菪丸
组成：莨菪子1升，羖羊肺1具（青羊亦佳）。
出处：方出《千金》卷二十一，名见《千金翼》卷十九。
主治：水气肿，鼓胀，小便不利。
加减：None
功效：None
用法用量：每服4丸，食后一食久，以麦门冬饮服，日3次。以喉中干，口粘浪语为候，数日小便大利佳。
制备方法：上2味，先洗羊肺，汤微渫之，薄切，晒干作末，以三年大酢渍莨菪子一晬时出，熬令变色，熟捣如泥，和肺末，蜜合捣3000杵作丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：莨菪走而不守，故须醋制稽留其性，以去痰涎垢腻；用羚羊肺为引，以通气化；服用麦门冬饮，以通肺之津液也。
用药禁忌：None
附注：None
