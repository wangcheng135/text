序号：72467
名称：痞气丸
组成：厚朴（去皮）4钱半，黄连（去须）8钱，吴茱萸（洗）3钱，黄芩2钱，白茯苓（去皮，另为末）1钱，泽泻1钱（另为末），川乌头（炮，去皮）半钱，人参（去芦）1钱，茵陈（酒制，炒）1钱半，巴豆霜4分，干姜（炮）1钱半，白术2钱，缩砂仁（去皮）1钱半，桂（去皮）4分，川椒（炒）半钱。
出处：《东垣试效方》卷二。
主治：脾积。在胃脘覆大如盘，久久不愈，令人四肢不收，发黄疸，饮食不为肌肤。
加减：秋、冬，加厚朴5钱半，减黄连1钱，黄芩1钱；黄疸，积大不能退，加巴豆霜1分，附子（炮）1钱，砒少许。
功效：None
用法用量：初服2丸，每日加1丸，至便溏为度，食前淡甘草汤送下。
制备方法：上除巴豆霜、茯苓另研为末旋入外，同为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：积减大半勿服。
附注：None
