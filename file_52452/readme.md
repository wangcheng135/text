序号：52467
名称：香芎丸
组成：净香附（盐水炒）半两，川芎半两，赤茯苓（去皮）半两，海金砂1两，滑石1两，枳壳（水浸润，去壳，锉片，麦麸炒微黄）2钱半，泽泻（去粗皮）2钱半，石苇（去毛梗，取薄叶）2钱半，槟榔（不过火）2钱半。
出处：《活幼心书》卷下。
主治：小儿谱淋证。小肠受气，客于膀胱，销烁肾水，水道涩而不利，小便涩痛。
加减：None
功效：None
用法用量：每服33-55丸，或77丸，空心用麦门冬煮水送下。若小便涩痛，3-5点滴者，取长江顺流水，用火微温，入盐少许调匀，空心咽服。
制备方法：上为末，糯米粉煮为清糊，为丸如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
