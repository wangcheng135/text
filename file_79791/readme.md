序号：79807
名称：干漆丸
组成：干漆1两（捣碎，炒令烟出），牡丹1两，射干1两，黄芩1两，桃仁2两（汤浸，去皮尖双仁，麸炒微黄），桂心1两，吴茱萸1两（汤浸7遍，焙干，微炒），川大黄1两（锉，微炒），水蛭半两（炒微黄），柴胡1两（去苗），菴（艹闾）子1两，虻虫半两（炒微黄，去翅足），乱发灰半两，(庶虫)虫半两（微炒），蛴螬20枚（微炒），大麻仁1两（别研如膏），鳖甲2两（涂酥炙令黄，去裙襕）。
出处：《圣惠》卷七十二。
主治：妇人脏腑宿冷，恶血凝结，月水不通，致令无子。
加减：None
功效：None
用法用量：每服20丸，以浸药酒送下，1日2次。
制备方法：上为末，以酒煎干漆为膏，和捣为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
