序号：39075
名称：泽泻丸
组成：泽泻1两，附子1两（炮裂，去皮脐），桂心1两半，远志3分（去心），牛膝3分（去苗），人参3分（去芦头），白茯苓3分，甘草3分（炙微赤，锉），牡丹3分，防风3分（去芦头），鹿茸1两（去毛，涂酥，炙微黄），杜仲3分（去粗皮，炙微黄，锉），云母粉1两，石斛3分（去根，锉），薯蓣3分，山茱萸3分，肉苁蓉1两（酒浸1宿，刮去皱皮炙干），巴戟3分，五味子3分，熟干地黄1两。
出处：《圣惠》卷九十八。
主治：男子五劳七伤，四肢无力，腰膝冷痛，夜多小便，面色萎黄，不能饮食。
加减：None
功效：None
用法用量：每服30丸，渐加至40丸，空心以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
