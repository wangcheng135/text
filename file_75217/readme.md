序号：75233
名称：蒜红丸
组成：拣丁香1两，木香1两，沉香1两，槟榔1两，青皮（去白）1两，陈皮（去白）1两，缩砂仁1两，蓬莪荗（炮）1两，牵牛1两，草果子1两，肉豆蔻（面裹，煨）1钱，粉霜1钱，白茯苓（去黑皮）半两，人参半两，蒜200枚（一半生用，一半火煨熟）。
出处：《百一》卷二引华宫使方。
主治：脾积。腹胀如鼓，青筋浮起，坐卧不得者。
加减：None
功效：None
用法用量：每服5-15丸，食后淡盐汤送下。
制备方法：上为细末，以生、熟蒜研细，生绢扭取汁，旋用药末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌咸、酸、鱼、酢、茶、酱、淹藏鸡鸭、生冷、马牛杂肉之类，只吃淡白粥100日。
附注：None
