序号：42995
名称：保喉片
组成：连翘20两，甘草30两，麦门冬10两，党参20两，百部1钱，元参20两，乌梅20两，黄耆10两，云故纸20两，诃子肉10两，僵蚕10两，桔梗20两，天花粉10两，蟾酥2钱5分，冰片5钱，薄荷油1两，薄荷脑3钱，白糖190两。
出处：《中药制剂手册》。
主治：阴虚肺热，咽痛喉炎，声音嘶哑，咽干咳嗽。
加减：None
功效：生津止咳，润肺利咽，滋阴消炎。
用法用量：每服1-2片，含化，1日3-4次。
制备方法：取桔梗、天花粉共为细粉，取部分细粉与蟾酥共为细粉，再将其余细粉陆续配研。取连翘至僵蚕等11味，用煮法提取2次，浓缩稠膏约40两。取桔梗等细粉与连翘等浓缩膏搅匀，加入白糖，共为细粉，制粒压片。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
