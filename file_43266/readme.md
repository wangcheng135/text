序号：43281
名称：养血清热固胎饮
组成：归身8钱（酒炒），川芎2钱，枳壳1钱5分（麸炒），细黄芩1钱2分，紫苏1钱5分，淡豆豉3钱，砂仁2钱（研），炙甘草3分。
出处：《慈航集》卷下。
主治：妇人怀孕染瘟疫者。
加减：如恶心，加藿香梗3钱，灶心土3钱；如作泻，加炒白芍5钱，红枣5个；如腹痛，加煨广木香1钱；如腰痛，加制杜仲3钱，川续断5钱；如咳嗽，加川贝母8分；如内热，加青蒿3钱；如口渴，加花粉2钱；如咽喉痛，加桔梗2钱，牛蒡子2钱，换生甘草1钱；如心里热，加连翘1钱5分；如胎火旺，呕吐不止，加姜汁炒黄连3分，乌梅肉1钱5分；如大便结，加鲜首乌8钱；如子气上逆，加葡萄干5钱，纹银1两同煎服；如肝火化风，痰厥，加钩藤2钱，白僵蚕3钱，羚羊角1钱5分，磨郁金1钱和服。
功效：None
用法用量：煨姜2片为引，水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：盖暖取汗。
附注：None
