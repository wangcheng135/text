序号：41725
名称：金翼丸
组成：沉香半两，木香半两，青橘皮（汤浸去白，焙）半两，陈橘皮（汤浸去白，焙）半两，京三棱（煨，锉）半两，五灵脂半两，芫花（醋炒焦色）1分，干漆（炒烟出）1分，寒食面（炒）3分，墨（烧，研）1分。
出处：《圣济总录》卷七十二。
主治：积聚不消，心腹胀满，痞闷不快。
加减：None
功效：None
用法用量：每服3丸，食后、临卧温熟水送下；心胸痞闷不快，温甘草汤送下。
制备方法：上为末。每秤药末1两，用去皮心膜巴豆仁1钱，入净臼内捣令极细，方入药末，再捣匀，用硇砂末半钱，新汲水浸化，旋入臼内，再捣千百下，候硬软得所，为丸如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
