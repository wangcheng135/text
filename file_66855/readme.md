序号：66871
名称：黄柏汤
组成：黄柏2分，黄连2分，白头翁（一作白蔹）2分，升麻2分，当归2分，牡蛎2分，石榴皮2分，黄芩2分，寄生2分，甘草2分，犀角1分，艾叶1分。
出处：《千金》卷十五。
主治：小儿夏日伤暴寒，寒折大热，热入胃，下赤白滞如鱼脑，壮热头痛，身热手足烦。或以利药下之，便数去赤汁如烂肉，或下之不愈，后以涩药断之，下即不止，倍增壮热，或是温病热盛，复遇暴寒折之，热如腹中，下血如鱼脑。
加减：None
功效：None
用法用量：上（口父）咀。以水3升，煮取1升3合，100日儿至200日每服3合，200余日至期岁每服2合半。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：寒折热邪而痢下赤白，温下不瘳，热涩不止，惟有苦燥一法可以转危就安，乃于白头翁汤中用寄生代秦皮，兼取犀角、黄芩以散热，当归、艾叶以和血，牡蛎、榴皮以固脱，升麻、甘草引入脾家，以散伏匿之邪也。
用药禁忌：None
附注：None
