序号：30986
名称：十疰丸
组成：雄黄2两，巴豆2两，人参1两，甘草1两，细辛1两，桔梗1两，附子1两，皂荚1两，蜀椒1两，麦门冬1两。
出处：《千金》卷十七。
主治：10种疰。气疰、劳疰、鬼疰、冷疰、生人疰、死人疰、尸疰、食疰、水疰、土疰等。
加减：None
功效：None
用法用量：空腹服5丸，每日2次。稍加，以知为度。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：十疰丸取桔梗丸中4味，不用藜芦而用雄黄、细辛，蜀椒之辛热辟邪，且得人参、甘草、麦冬，虽云助长辛烈之性，究竟良药可保护津气，不似桔梗九之一派峻锐耳。
用药禁忌：None
附注：《千金翼》有藁本，无细辛。
