序号：73088
名称：犀角汤
组成：犀角屑2两，青木香2两，羚羊角屑2两，人参2两，竹茹2两，沉香2两，射干2两，麦门冬（去心）3两，茯苓3两，麝香2两，鸡舌香2两，石膏8两（碎，绵裹）。
出处：《外台》卷十八引《苏恭方》。
主治：脚气。风热轻，但毒气入胃，唯心闷烦，索水洒胸面，干呕，好叫唤，欲断绝者。
加减：若呕逆不下食，水浆即吐出者，加半夏4两（洗），生姜2两，橘皮1两，加水1升半，煮取2升3合服。
功效：None
用法用量：以水6升，煮取2升2合，分4次服，相去如人行6-7里，晚再服；如觉眼明心悟，若强人，作3服。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：忌五辛、羊肉、饧、醋物。
附注：None
