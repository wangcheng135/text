序号：71303
名称：紫石英丸
组成：紫石英2两（细研，水飞过），细辛1两，厚朴1两（去粗皮，涂生姜汁，炙令香熟），川椒1两（去目及闭口者，微炒出汗），桔梗1两（去芦头），鳖甲1两半（半生用），防风1两（去芦头），川大黄1两（锉碎，微炒），附子1两（炮裂，去皮脐），硫黄1两（细研），牡蒙3分，人参3分（去芦头），桑寄生3分，半夏半两（汤洗7遍去滑），白僵蚕半两（微炒），续断半两，紫菀半两（洗去苗土），杜衡半两，牛膝半两（去苗），白薇1两，当归1两（锉碎，微炒），桂心1两。
出处：《圣惠》卷七十。
主治：妇人由子脏久积风冷，阴阳不能施化而久无子。
加减：None
功效：None
用法用量：每服30丸，空心以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
