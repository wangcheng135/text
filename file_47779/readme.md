序号：47794
名称：神仙退云丸
组成：白蒺藜1两半（炒），川椒1两半（炒），川芎1两半，当归1两半，楮实半两，黄连半两，蝉壳半两，薄荷叶半两，瓜蒌根6钱，地骨皮1两，甘草花3钱，荆芥3钱，密蒙花3钱，蛇蜕3钱（以上4味用甘草先熬，水浸过，焙干），蔓荆子2两（炒），木贼2两（去节，用童子小便浸1宿，焙干）。
出处：《医方类聚》卷七十引《经验秘方》。
主治：眼睛青盲，气障眼，眼常昏，头风眼昏，小儿痘疮成翳。
加减：None
功效：None
用法用量：每服1丸，每日2次。眼睛青盲者，当归汤送下；气障眼，木香汤送下；眼常昏者，好酒送下；头风眼昏，茶汤送下；小儿痘疮成翳，谷精草汤送下。
制备方法：上为细末，炼蜜为丸，每两作10丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
