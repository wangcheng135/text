序号：46521
名称：千金不换挝痞膏
组成：血竭1钱半，乳香（另研）2钱，没药2钱（另研），阿魏2钱，大黄3钱，雄黄3钱，米壳3钱，巴豆（去油）3钱，人言3钱，川山甲3斤（炙），芥子5钱（另研），鸽粪（醋烹）5钱，皮消5钱，野葡萄（根皮，炒干）5钱，凤仙草5钱，蓖麻子5钱（炒黄）。
出处：《鲁府禁方》卷三。
主治：小儿癖疾，并男妇一切积块。
加减：None
功效：None
用法用量：贴患处，每贴加麝香5分，独蒜1头（捣10下），红绢1方，将药摊上，如干用醋润之，3炷香尽去药，3日1次。
制备方法：上为末，用小黄米做成粉子，炒糊4两研细，用陈醋和成膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
