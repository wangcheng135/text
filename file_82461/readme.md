序号：82477
名称：蠲饮枳实丸
组成：枳实（麸炒，去瓤）2两，半夏（汤洗，浸1宿，切，焙干）2两，陈橘皮（去白）2两，黑牵牛半斤（取头末3两，余不用）。
出处：《杨氏家藏方》卷八。
主治：痰饮，饮伤脾胃。
加减：None
功效：逐饮消痰，导滞清膈。
用法用量：每服50丸，生姜汤送下，不拘时候。
制备方法：上为细末。煮糊为丸，如梧桐子大。
临床应用：None
各家论述：《脾胃论注释》：方中枳实麸炒以减其攻破之性，治积滞内阻所致的痞满胀痛；半夏燥湿除痰，和胃降逆；陈皮芳香健胃，行气化痰，牵牛决三焦壅滞，其性滑利，有通便泻水的作用。为丸饭后服，以减轻药物的副作用。姜汤送服，既解半夏之毒，又制牵牛之寒。面粉和生姜均为保护胃气而设。
用药禁忌：None
附注：蠲饮枳术丸（《袖珍》卷二）。
