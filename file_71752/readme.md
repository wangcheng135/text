序号：71768
名称：紫菀杏仁煎
组成：紫菀（去苗土）1两半，杏仁半升（去皮尖双仁，别细研），生姜汁3合，地黄汁5合，酥2两，蜜1升，大枣肉半升，贝母（去心）3两，白茯苓（去黑皮）1两，五味子（炒）1两，人参1两，甘草（炙，锉）1两，桔梗（锉，炒）1两，地骨皮1两。
出处：《圣济总录》卷六十五。
主治：肺脏气积，喉中呷嗽不止，皆因肺脏虚损，致劳气相侵，或胃中冷膈上热者。
加减：None
功效：None
用法用量：上为末，调和诸自然汁，并酥、蜜、杏仁等，同于铜银器中以文武火煎，频搅令匀，煎百十沸，成煎后再于甑上煎3遍。每服1匙，食后服，头便仰卧少时，渐渐咽药，1日2次。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
