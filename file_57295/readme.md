序号：57310
名称：消痞膏
组成：朴消5钱，蒜头（杵）5钱，大黄3钱，急性子3钱，三棱4钱，莪术4钱，乳香（制）2钱，真阿魏2钱，没药（制）2钱。
出处：《不知医必要》卷二。
主治：积聚痞块。
加减：None
功效：None
用法用量：以白布或厚油纸摊贴，白天日1换。或见大便有脓血勿以为异，亦有不下脓血者。
制备方法：捋三棱、莪术、大黄、急性子研末。用脂麻油4两，煎蒜头、朴消及药末，煎好去蒜头，下黄丹2钱。候已成膏，以乳香、没药、阿魏3味研粉加入，再加麝香2分，搅匀，贮有盖瓦器内，俟3两日，火气去净。
临床应用：None
各家论述：None
用药禁忌：忌房事及一切生冷物。
附注：None
