序号：47730
名称：神仙延寿丹
组成：天门冬2两（去心），远志2两（去骨），山药2两（去苗），巴戟2两（去骨），赤石脂1两（炒），车前子1两（炒），石菖蒲1两（炒），柏子仁1两，泽泻1两，川椒1两（去目），熟地黄1两，生地黄1两，枸杞子1两，白茯苓1两，覆盆子1两，杜仲1两（炒去丝），菟丝子1两（酒炒），肉苁蓉4两（炒干），川当归1两，川牛膝1两（酒洗），地骨皮1两，五味子1两，山茱萸1两，人参1两。
出处：《摄生众妙方》卷二。
主治：None
加减：None
功效：养血黑须鬓，延年益寿。
用法用量：每服20-30丸，清晨温酒或盐汤送下。服至100日后颜色永无衰朽，发白返黑，虽是80老人，阴阳强健，目视10里，气力不衰，常行远地不乏。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
