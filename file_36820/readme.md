序号：36830
名称：固真丸
组成：川乌头（锉，盐炒黄色，去盐不用）2两，熟干地黄（洗，焙）2两，秦椒2两，肉桂（去粗皮）1两，茴香（酒浸，炒）1两，威灵仙（去土）1两，仙灵脾1两，山药1两，五味子（炒）1两，萆薢半两，附子（炮，去皮脐）半两，白茯苓（去皮）半两，当归（浸，焙）半两，牛膝（酒浸1宿）半两，石菖蒲半两。
出处：《杨氏家藏方》卷九。
主治：诸虚不足。
加减：None
功效：补益五脏，接助真阳，滋润肌肤，悦颜色。
用法用量：每服50丸，空心、食前温酒、盐汤任下。久服不生诸疾。
制备方法：上为细末，炼蜜为丸，杵千余下，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
