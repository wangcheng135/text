序号：39201
名称：狐仙封脏丸
组成：枸杞子（去蒂，酒拌蒸）4两，菟丝子4两，白茯苓（乳拌，蒸晒5次）4两，赤茯苓4两，大生地（竹刀切片）4两，大熟地4两，甘菊花4两，女贞子4两，何首乌（同女贞子蒸晒5次）4两，山萸肉4两，远志肉（甘草水浸2日）4两，当归身4两，人参4两，莲须4两，柏子仁4两，天门冬4两，龙眼肉4两，麦门冬（去心）4两，酸枣仁4两，北五味2两，川牛膝2两，粉丹皮2两，石菖蒲2两，泽泻2两。
出处：《疡医大全》卷二十三。
主治：痔。
加减：None
功效：延年益寿。
用法用量：每服2钱，白汤送下。
制备方法：炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：禁色欲。
附注：此方并不治漏，而治痔有神效之功。十日见效。服完一料，永不再发。
