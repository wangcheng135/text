序号：54295
名称：大金牙酒
组成：金牙1斤，侧子3两，附子3两，天雄3两，人参3两，苁蓉3两，茯苓3两，当归3两，防风3两，黄耆3两，薯蓣3两，细辛3两，桂心3两，萆薢3两，萎蕤3两，白芷3两，桔梗3两，黄芩3两，远志3两，牡荆子3两，芎3两，地骨皮3两，五加皮3两，杜仲3两，厚朴3两，枳实3两，白术3两，独活半斤，茵芋2两，石南2两，狗脊2两，牛膝3两，丹参3两，磁石10两，薏苡仁1升，麦门冬1升，生石斛8两，蒴藿4两，生地（切）2升。
出处：《千金》卷七。
主治：瘴疠毒气中人，风冷湿痹，口面戾，半身不遂，手足拘挛，历节肿痛，甚者小腹不仁，名曰脚气。凡是风虚，四体小觉有风痾者，皆须将服之。
加减：None
功效：None
用法用量：温服1合，日4-5次，夜1次。药力和善。
制备方法：上39味，石药细研，别绢袋盛，余药咬碎，以酒8斗同渍7日。
临床应用：None
各家论述：《千金方衍义》：金牙专辟瘴疠毒风、鬼疰、恶气，而汤液罕用，惟酒散中间有用之者。侧、附、天雄辟除阴毒，桂心、石南、萆薢，狗脊坚强筋骨，牡荆、茵芋、祛逐贼风，皆药中颖锐；磁石一味，《本经》治周痹风湿，肢节中痛、与金牙相为表里，一追风痹外散，一杜恶气内入也；人参、茯苓、黄耆、白术，以助侧、附、天雄之威；苁蓉、牛膝、杜仲、远志，以助桂、南、萆薢之势；细辛、防风、独活、白术，以助蒴、荆、茵芋之力；其余门冬、萎蕤、山药、薏苡，又为人参、黄耆之辅；丹参、川芎、当归、地黄，又为苁蓉、牛膝之辅；石斛、地骨、加皮，又为细辛、独活之辅；黄芩、桔梗开发毒风于外；枳实、厚朴疏通恶气于内。无坚不克，非独为脚气之金，而八风五痹，靡不疗之。
用药禁忌：忌猪肉、冷水、生葱、生菜、桃李、雀肉、芜荑等。
附注：金牙酒（《脚气治法总要》卷下）。
