序号：3537
名称：太傅白膏
组成：蜀椒1升，附子3两，升麻（切）1升，巴豆30铢，芎30铢，杏仁5合，狸骨1两半，细辛1两半，白芷半两，甘草2两，白术6两（一方用当归3两）。
出处：《千金》卷七。
主治：伤寒咽喉不利，头项强痛，腰脊两脚疼，有风痹湿肿难屈伸，不能行步，若风头眩，鼻塞，有附息肉生疮，身体隐疹风搔，鼠漏瘰疬，诸疽恶疮，马鞍牛领肿疮；及久寒结坚在心，腹痛胸痹，烦满不得眠，饮食咳逆上气，往来寒热；妇人产后余疾，耳目鼻口诸疾。
加减：None
功效：None
用法用量：上(口父)咀，苦酒淹渍1宿，以猪脂4斤，微火煎之。先削附子1枚，以绳系著膏中，候色黄膏成，去滓。伤寒心腹积聚，诸风肿疾，颈项腰脊强，偏枯不仁，皆摩之，每日1次；痈肿恶疮，鼠瘘瘰疬，炙手摩之；耳聋，取如大豆，灌之。目痛炙，缈缥白瞖如珠当瞳子，视无所见，取如穄米，敷白上，令其人自以手掩之，须臾即愈，便以水洗，视如平复，且勿当风，30日后乃可行；鼻中痛，取如大豆纳鼻中，并以摩之，龋齿痛，以绵裹如大豆，着痛齿上，咋之；中风，面目鼻口喎僻，以摩之；若晨夜行，辟霜雾，眉睫落，数数以铁浆洗，用膏摩之。
