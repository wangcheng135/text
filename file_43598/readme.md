序号：43613
名称：前胡散
组成：前胡半两（去芦头），半夏1分（汤洗7遍去滑），甘草半两（炙微赤，锉），桂心半两，人参半两（去芦头），赤茯苓半两，白芷1分，白术半两，干姜1分（炮裂，锉），当归半两（锉，微炒），葛根半两（锉），柴胡半两（去苗），陈橘皮半两（汤浸去白瓤，焙），木香半两，旋覆花半两。
出处：《圣惠》卷九。
主治：伤寒4日，虽经发汗后，心胸不利，头目多疼，胃气不和，少思粥食。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，入生姜半分，大枣3枚，煎至6分，去滓温服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
