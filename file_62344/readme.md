序号：62360
名称：旋覆花丸
组成：旋覆花5分，桂心5分，枳实5分，人参5分，干姜6分，芍药6分，白术6分，茯苓8分，狼毒乌头8分，礜石8分，细辛4分，大黄4分，黄芩4分，葶苈4分，厚朴4分，吴茱萸4分，芫花4分，橘皮4分，甘遂3分。
出处：《千金》卷十八。
主治：停痰澼饮，结在两胁，腹胀满，羸瘦不能食，食不消化，喜唾干呕，大小便或涩或利，腹中动摇作水声，腹内热，口干好饮水浆，卒起头眩欲倒，胁下痛。
加减：None
功效：None
用法用量：每服5丸，酒送下，1日2次。加之，以知为度。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：旋覆花开结下气，行水消痰，开发肺与大肠水气之味，为主；其间参、术、桂心、芫花、甘遂、葶苈、大黄等，与五饮丸相同者十余味，更加乌头、干姜、狼毒、礜石，其破积散癖之力，不减五饮丸中大戟、巴豆之烈也。
用药禁忌：None
附注：None
