序号：30045
名称：赤金丹
组成：苍术2两，雄黄1两，木香1两，炙草5钱，朱砂5钱，血竭5钱，乳香5钱，没药5钱，沉香5钱，麝香1钱，冰片1钱，大金箔30张（为衣）。
出处：《仙拈集》卷四。
主治：伤寒感冒，胸膈膨胀，乳蛾，肿毒，大小便不通，疟疾，泄泻，赤白痢，头痛，霍乱，小儿惊风，胃气痛，妇女经水不调。
加减：None
功效：None
用法用量：大人空心服5丸，小儿3丸。服后盖暖睡1时。伤寒感冒，葱白汤送下；胸膈膨胀，陈皮汤送下；乳蛾，井花水送下；肿毒，升麻大黄汤送下；小便不通，竹叶汤送下；大便不通，火麻仁、大黄汤送下；疟疾，杏仁汤送下；赤痢，甘草汤送下；白痢、泄泻，姜汤送下；赤白痢，乌梅汤送下；头痛，川芎汤送下；霍乱，藿香汤送下；惊风，薄荷汤送下；胃气痛，艾醋汤送下；经水不调，丹参汤送下；小儿不能服药，研碎抹乳上食少半丸。
制备方法：上为末，炼蜜为丸，如绿豆大，外用金箔为衣，阴干，瓷器收贮，置高燥处，恐致霉湿。
临床应用：None
各家论述：None
用药禁忌：忌生冷荤腥。
附注：None
