序号：45
名称：五味子汤
组成：五味子10株，甘草12株，当归12株，大黄6铢，芒消5株，麦门冬6株，黄芩6株，前胡6铢，石膏1两，黄连6铢。
出处：《千金》卷五。
主治：小儿伤寒，病久不除，愈后复剧，瘦瘠骨立。
加减：None
功效：None
用法用量：上(口父)咀。以水3升，煮取1升半，每服2合。得下便止。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：小儿伤寒，病久不除，或愈后复剧，此必邪从火化而陷伏不解，所以瘦瘠骨立，非急投三黄下夺无以泄之；三黄不逮，济以芒消、石膏；其前胡、麦冬、五味、甘草、当归以滋律气之燥。在婴儿固为合剂，以元神未动，不虑其引邪内贼也。
用药禁忌：None
附注：None
