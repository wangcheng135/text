序号：5921
名称：四海舒郁丸
组成：青木香5钱，陈皮3钱，海蛤粉3钱，海带2两，海藻2两，昆布2两，海螵蛸2两（俱用滚水泡去盐）。
出处：《疡医大全》卷十八。
主治：因七情抑郁不伸，肝脾气郁不舒致气颈，结喉之间，气结如胞，随喜怒消长，甚则饮食嗌碍。
加减：None
功效：None
用法用量：每服3钱，不拘酒、水，日3次；滓沉在碗底内者，敷气颈上。
制备方法：上为细末。
临床应用：气颈：曾治疗4例女性患者，肿大的甲状腺均渐变软、变小，其他伴随症状减轻或消失。作者认为：气颈多属气郁痰阻所致，治用四海舒郁丸理气化痰，软坚散结，一般在服药1-1.5个月后，颈间肿大之甲状腺都能变软、变小，连服2-3个月，可以消散。所有眼突、心悸、失眠、手颤等现象，亦能逐渐消失。
各家论述：None
用药禁忌：None
附注：愈后用黄药子四两，生酒三大壶，煮三炷香，窨一七去火毒，早、晚任饮数杯。
