序号：25208
名称：如意通圣散
组成：罂粟壳2钱5分，丁香2钱5分，麻黄2钱5分，赤芍2钱5分，防风2钱5分，荆芥2钱5分，当归2钱5分，川芎2钱5分，羌活2钱5分，独活2钱5分，白芷2钱5分，甘草2钱5分，黄芩2钱5分，威灵仙2钱5分，草乌2钱5分（炒黄色），桔梗2钱5分，葛根2钱5分。
出处：《解围元薮》卷三。
主治：白虎历节，痛风，寒湿，手足不能举，浑身走注，抽掣疼痛。
加减：None
功效：None
用法用量：水煎，去滓，乳香、没药末各3分，盖被取汗。趁热冲服。如病在肩背上，冲服白芷末2钱。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
