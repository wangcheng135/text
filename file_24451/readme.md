序号：24459
名称：托里败毒散
组成：绵黄耆（盐水炒）3钱（或5钱，或8钱，或1两），甘草节（水炙）2钱（可加至4-5钱），赤芍药2钱，金银花3钱，茜草（江西出，细如灯芯者佳）3钱，何首乌（鲜者）5钱，真白僵蚕（炙，研）6分，白及2钱5分，皂角刺1钱，贝母（去心）2钱，栝楼根3钱，穿山甲（土炒，研）1钱，鼠粘子（炒，研）1钱，蝉蜕（去翅爪）1钱。
出处：《广笔记》卷三。
主治：肿毒。
加减：阴症去后五味，加人参3钱，麦门冬5钱。
功效：None
用法用量：先用夏枯草5两，河水5大碗，煎3碗，入前药同煎至1碗，不拘时候服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
