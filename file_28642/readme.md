序号：28650
名称：补天丸
组成：紫河车（男用女胎，女用男胎，俱以初胎为佳，若不可得，即壮盛妇人者亦可）3两，黄柏（酒炒）3两，龟版（炙）3两，杜仲（酥炙）2两，牛膝（酒浸）2两，陈皮1两。
出处：《摄生众妙方》卷十一。
主治：虚劳。
加减：冬，加干姜5钱；夏，加五味子1两。
功效：None
用法用量：每服100丸，空心温酒或白沸汤送下。
制备方法：上为细末，以河车水洗净，布绞干，或用酒煨熟，入诸药末捣匀，焙燥，再为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：《医方集解》：黄柏、龟版滋肾之药，杜仲、牛膝腰膝之药，皆以补肾而强阴也。河车名曰混沌皮，用气血以补气血，借后天以济先天，故曰补天。加陈皮者，于补血之中而兼调其气也。冬月寒水用事，故加干姜以助阳；夏月火旺灼金，故加五味子以保肺。
用药禁忌：None
附注：补元丸（《中国医学大辞典》）。
