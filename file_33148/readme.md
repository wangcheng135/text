序号：33157
名称：万应膏
组成：透骨草2钱，附子2钱，桂心3钱，江子20个（去皮，勿研），山甲2钱（生），香油半斤，漳丹4两，（又）冰片5分，轻粉5分，麝香1分（细研，瓶盛备用）。
出处：《医学探骊集》卷五。
主治：腰痛，并连腿痛。
加减：None
功效：None
用法用量：将附子、透骨草、桂心为粗末，江子勿研，用布袋装1处；再将香油入铁勺内，将药袋入油内，生甲片亦入油内炸之，俟甲片浮起，将甲片并药袋捞出，入漳丹稍为熬之，以滴水成珠为妙，摊布上，再将冰片等3味药面，用1小捻撒在膏上，贴之极效。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
