序号：47185
名称：珍珠散
组成：珍珠1钱5分，玛瑙1钱5分，琥珀1钱5分，珊瑚1钱5分（以上4味俱用豆腐煮过再研），硼砂5分，熊胆5分（用笋壳盛，烘脆为末），龙脑4分，麝香2分5厘，瓜竭7分5厘，朱砂（细研水飞）7分5厘，黄连末（去须芦，细研）5分，明乳香（箬上炙干）5分，没药（箬上炙干）5分，芦甘石（按法炮制为主）1两5钱。
出处：《医学心悟》卷四。
主治：风火红眼，障翳胬肉，迎风流泪，老年昏花。
加减：None
功效：清火止痛，消肿退翳。
用法用量：听用。
制备方法：上各为细末，用上细粉罗筛过，再照分数称定，合为一处，研万匝，复以棉纸筛下，瓷罐收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：《中药成方配本》有石蟹，无血竭。
