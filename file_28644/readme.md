序号：28652
名称：补天丸
组成：紫河车（初胎者1具，米泔洗净，入砂锅内，用水1碗煮沸，候冷取起，放小竹篮中，用纸密糊烘干）、黄柏（蜜炒）、知母（乳炒）、龟版（酥炙）3两，熟地5两（煮），牛膝（酒洗）1两半，苁蓉（酒洗）1两半，麦冬1两半，山药1两半，虎胫骨（酥炙）1两半，茯神1两半，杜仲3两，首乌3两，人参3两，白芍3两，生地3两，天冬3两，当归3两，五味3两，枸杞2两。
出处：《医级》卷八。
主治：男妇虚损劳伤，形体羸乏，腰背疼痛，遗精带浊。
加减：冬，加干姜。
功效：None
用法用量：每服70-80丸，空心淡盐汤送下。
制备方法：上为末，猪脊髓三条，蒸熟，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
