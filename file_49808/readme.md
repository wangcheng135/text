名称：大乌犀散
组成：犀角（镑）1分，羚羊角（镑）1分，龙脑（研）1分，麝香（研）1分，雄黄（研）1分，熊胆（研）1分，牛黄（研）1分，乳香（研）1分，阿魏（研）1分，丹砂（研）1分，水银（与丹砂同研，慢火搅匀，结砂子，研）1分，干蝎（酒炒）1分，猪牙皂荚（酥炙，去皮、子）1分，乌头（炮裂，去皮脐）1分，附子（炮裂，去皮脐）1分，白附子（炮）1分，干姜（炮）1分，天麻1两，升麻1两，独活（去芦头）1两，狗脊（去土）1两，细辛（去苗叶）1两，秦艽（去苗土）1两，芎1两，杜仲（去粗皮，切，炒）1两，当归（切，焙）1两，厚朴（去粗皮，生姜汁炙）1两，黄耆（锉，炙）1两，藁本（去土）1两，莨菪子（淘去浮者，炒）1两，麻黄（去根节，先煎掠去沫，焙）1两，蜀椒（去目并闭口，炒出汗）1两，白鲜皮（锉）1两，防风（去叉）1两，天南星（炮）1两。
出处：《圣济总录》卷六。
主治：破伤中风。
加减：None
功效：None
用法用量：每服1钱半匕，加至2钱匕，豆淋酒调下，生姜酒投。甚者用米醋与水同煎服之。
制备方法：上35味，将26味为散，与9味研者拌匀，再罗1遍。
