序号：7733
名称：圣灵丹
组成：川乌（生，去皮脐，切片）半两，草乌（去皮尖，切片，盐炒香熟，去盐）半两，麻黄（去根节，微炒去汗）1两，生地黄（洗，去苗，锉碎，酒浸1宿，焙酥黄色）1两，自然铜（醋淬7次，研）1两，五灵脂（拣去砂石，微炒不得过），虎胫骨（醋浸，炙酥黄色），广木香2钱半，乳香（另研）钱半，没药（炙研）钱半，干酸木瓜（生者）8两，甜瓜子（炒黄色）1两，沉香5钱（镑），败龟版（卜卦者，醋炙黄酥）7钱半。
出处：《瑞竹堂方》卷二。
主治：男子妇人，风湿相搏，气痹传于手足，麻肿疼痛，久则偏枯，及脚气不能行履，风湿瘫痪。
加减：None
功效：None
用法用量：每服2丸，隔夜以生姜自然汁入磁盏同浸，至天明化开，空心温酒调服，再饮半盏热酒送下，日进2服。唇吻微麻无妨。
制备方法：上为细末，炼白沙蜜冷定，和药成剂，每一两分为一十二丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
