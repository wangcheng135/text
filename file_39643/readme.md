名称：细辛散
组成：细辛3分，肉桂半两（去芦头），人参3分（去芦头），麻黄3分（去根节），附子3分（炮裂，去皮脐），杏仁3分（汤浸，去皮尖双仁，麸炒微黄），紫菀半两（洗去苗土），赤茯苓3分，白术半两，干姜半两（炮裂，锉），桔梗半两（去芦头），前胡3分（去芦头），百合3分，厚朴2两（去粗，皮，涂生姜汁炙令香熟），甘草半两（炙微，赤，锉），赤芍药半两。
出处：《圣惠》卷十二。
主治：伤寒，咳嗽喘促，鼻塞。
加减：None

用法用量：每服4钱，以水1中盏，加生姜半分，大枣3枚，煎至6分，去滓温服，不拘时候。
制备方法：上为散。
