序号：25483
名称：八味回阳饮
组成：人参（无者，以蜜炒黄耆1两代之），附子2-3钱，干姜（炒）2-3钱，当归身3钱（如泄泻者，或血热动血者，去之），熟地数钱或1-2两，甘草（炙）1钱，白术3-4钱，黄耆（蜜炒）3钱。
出处：《会约》卷三。
主治：伤寒脉虚将绝，阴阳将脱。白喉，其痛甚，其无白色处，色紫红，脉沉紧者。
加减：如泄泻者，加乌梅2个；虚火上浮者，加茯苓2钱、麦冬1钱：如肝滞而胁胀痛者，加肉桂钱半。
功效：None
用法用量：水煎，温服。
制备方法：None
临床应用：None
各家论述：《喉科种福》：此为阴盛格阳于上之证，宜回阳饮，热药凉用。按其用姜、附、归、地也，回阳子肾以温中；其用参、耆、术、草也，暖气于肺以达外。服后如发战下利，则加倍再服，惟归、地不可再加，以归、地为阴药故也。
用药禁忌：None
附注：回阳饮（《喉科种福》卷五）。
