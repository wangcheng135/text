序号：56228
名称：梨浆饮
组成：青蒿（取花头，用童便浸1-2次，晒干为度）、柴胡（去芦）、人参、黄芩（去心）、前胡、秦艽（去土）、甘草（炙）。
出处：《活幼口议》卷十六。
主治：脾积寒热，其状如疟，或头痛呕逆，久则2-3岁不歇，左胁有块，小者如桃李，大者似杯碟。胃热口气，痰饮呕逆，不思饮食。
加减：None
功效：None
用法用量：上(口父)咀。每服1岁儿半钱，2岁1钱匕，水1小盏，入生藕、生梨、薄荷2叶，生地黄1寸，同煎至半，去滓，通口空心、食前服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：梨浆饮子（《袖珍》卷四）。方中青蒿，《医统》作“茵陈”。
