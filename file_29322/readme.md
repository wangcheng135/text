序号：29330
名称：远志汤
组成：远志3两，干姜3两，铁精3两，桂心3两，黄耆3两，紫石英3两，防风2两，当归2两，人参2两，茯苓2两，甘草2两，芎2两，茯神2两，羌活2两，麦门冬4两，半夏4两，五味子2合，大枣12枚。
出处：《千金》卷十四。
主治：心气虚，惊悸喜忘，不进食。
加减：None
功效：补心。
用法用量：上（口父）咀。以水1斗3升，煮取3升半，分5服，日3夜1。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：生脉、保元滋降心火，理中、四君固实脾土，心降则心肺安，土实则肾肝固，五脏相率受其益矣。更须用芎、归、桂心以和营血，远志、茯神以利关窍，羌、防以祛风湿，半夏以涤痰涎，紫石英以镇怯，大枣以和脾气，并和诸药性味也。
用药禁忌：None
附注：None
