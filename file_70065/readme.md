序号：70081
名称：滋阴补肾丸
组成：熟地黄（酒蒸）2两，生地黄（酒浸）2两，白术2两，人参1两，麦门冬（去心）1两，五味子1两，当归（酒浸）1两，白芍药（酒炒）1两，川芎1两，黄耆（盐水炙）1两，山药1两，蛤粉（另研极细）1两，茯神（去皮木）1两，砂仁1两，知母（炒）1两半，黄柏（炒）2两。
出处：《准绳·伤寒》卷七。
主治：病后阴虚，精血不足，四肢少力、心神不宁，夜梦遗精，或虚热盗汗，饮食进少，不为肌肉，身体羸弱，面色青黄而无血色。
加减：None
功效：滋肾水，制虚火。
用法用量：每服50丸，空心淡盐汤送下。
制备方法：上为细末，炼蜜和成于石臼内杵千余下，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
