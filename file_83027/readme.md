序号：83043
名称：乌头丸
组成：川乌头不计多少。
出处：《圣惠》卷九十八。
主治：风气。腰脚痠痛，脐腹虚冷。
加减：None
功效：祛风气，暖水脏。
用法用量：每服10丸，以温酒或盐汤送下。渐加至20丸为度。
制备方法：上净洗晒干。宽掘地作坑子一所，却用好土实填筑，于好土内取作坑子一所，刮削坑子口，周回如法。然后用炭火烧令通赤，即去火，用酽醋1碗，倾在坑内，候渗尽，便匀排乌头在内，使好土盖覆，厚1寸，筑却。然后坑口头以炭火密排烧之，斟酌欲熟，于坑口边取出1枚看之，以软熟为度。其药又不得令生，又不得太焦，切在用意。取出晒干，捣罗为末，以粳米饭为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
