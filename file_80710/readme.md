序号：80726
名称：麋角散
组成：麋角半斤（先用桑柴灰2斗煎汤淋，取汁3斗，次截麋角入灰汁中慢火煮，尽汁为度，候干，取4两用），芦荟半两，赤箭半两，蝎梢（酒炒）半两，麝香（研）半两，附子（炮裂，去皮脐）半两，干姜（炮）1分。
出处：《圣济总录》卷十八。
主治：大风恶疾，滑泄精气。
加减：None
功效：None
用法用量：每用5钱匕，入好腊茶末7钱匕和匀。凡患此疾，鼻梁未倒，语声未转，精气滑泄者，取药末1钱匕，用荆芥、薄荷汤如茶点热服；觉药力紧，每点入盐少许，要出汗即热服，厚衣覆出汗，慎外风。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
