序号：47980
名称：神龙抱珠灵膏
组成：乳香5钱（箬上烘去油），没药5钱（箬上烘去油，研细），大黄2两，肉桂2两（去皮，研极细末），土木鳖2两（去壳，净），真阿魏3钱（切薄片），白芷梢2两，血余2两，麻黄绒（用麻黄4-5两，入臼内捣成绒，筛去粗皮滓渣，再捣再筛，务令极细）2两，归身2两，羌活2两，元参2两（去芦），赤芍2两，生地3两，轻粉4两，黄丹40两（水飞过，盐矾炒干成紫色，研极细），桃枝7枝，柳枝7枝，槐枝7枝，桑枝7枝，枣枝7枝（每枝长1寸）。
出处：《膏药方集》引《外科枢要》。
主治：风寒暑湿，恶疮大毒，跌扑闪挫，五劳七伤，背心肩臂腰膝酸疼。
加减：None
功效：None
用法用量：临用摊贴。
制备方法：将前16味药中的10味粗药用麻油6斤半浸3日，先将血余熬溶后，再投各药熬至黄枯浮起，滤去药滓；次下黄丹熬至滴水成珠，将锅提起，停片刻，将细药（肉桂、乳香、没药、轻粉、阿魏）趁热放下，不住手搅匀，使黑如漆、明如镜，入水试不老不嫩为度；趁热投入大水缸内浸7日，捞起切块，油纸包裹，收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
