序号：47018
名称：独活散
组成：川芎半两，独活半两，羌活半两，防风半两，华细辛2钱，荆芥3钱，郓薄荷3钱，生地黄3钱。
出处：《御药院方》卷九。
主治：阳明经不利，邪毒攻注，牙齿龈肉虚浮宣露褪下，动摇发痛；及偏正头痛，渐渐攻注眼目，或发疼痛，视物不明。
加减：None
功效：清头目，发散风热。
用法用量：每服3钱，以水1盏，煎至8分，去滓，食后温服，1日3次。
制备方法：上为粗末。
临床应用：None
各家论述：《医略六书》：贼风伤经，袭入齿缝而伤风恶风，牙疼走注且吸风痛甚，可知病全在经而脏气未伤焉。独活去少阴伏风，羌活散太阳游风，荆芥疏风理血，防风散风行气，川芎活血行血中之气，生地滋阴壮肾中之水，细辛散浮热，薄荷清口齿。为散水煎，使风邪解散，则经气清和而齿无疼痛之患，何吸风更甚之有？此疏散调营之剂，为齿痛吸风疼甚之专方。
用药禁忌：None
附注：None
