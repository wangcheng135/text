序号：40863
名称：郁李仁丸
组成：郁李仁1两（汤浸，去皮，微炒），汉椒半两（去目及闭口者，微炒出汗），人参半两（去芦头），甘草1分（炙微赤，锉），桂心半两，干姜半两（炮裂，锉），细辛半两，赤芍药半两，陈橘皮1两（汤浸，去白瓤，焙），厚朴1两（去粗皮，涂生姜汁，炙令香熟），胡椒半两，附子半两（炮裂，去皮脐），川大黄2两（锉碎，微炒），木香1两，诃黎勒皮2两。
出处：《圣惠》卷五十。
主治：五膈气，心胸气壅，宿食不消，腹胃胀满，大便秘涩。
加减：None
功效：None
用法用量：每服30丸，以热酒送下，不拘时候。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
