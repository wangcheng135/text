序号：72521
名称：温补荣气汤
组成：当归3-5钱（若血虚有热者宜少用），熟地5-7钱，甘草（炙）1钱5分，白芍（酒炒）1钱5分，枸杞2钱，山药2钱，枣仁（炒）1钱。
出处：《会约》卷十四。
主治：心脾血虚，脉息细数，体亏气弱，心痛潮热。
加减：如心虚有火不可按者，加丹皮1钱半，或加生地1-2钱；如血寒凝滞，加肉桂1钱半；如气滞而胀者，加陈皮、香附之类；如气虚而不能生血者，加蜜炒黄耆1钱半，或加人参更妙；如血散而不收敛者，加五味2-3分，或用十全大补汤亦妙。
功效：None
用法用量：水煎，温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
