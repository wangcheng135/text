序号：56636
名称：消气散
组成：沉香半两，木香半两，人参半两，半夏（汤洗7次）半两，青皮（去白、炒）半两，桔梗（炒）半两，陈皮（去白，炒）1两，白茯苓（去皮）3两，草果仁（炒）3两，大腹皮（洗，焙）3两，紫苏（连梗）3两，木通3两。
出处：《简易方》引《叶氏录验方》（见《医方大成》卷三）。
主治：血气凝滞，心脾不和，腹急中满，四肢浮肿，饮食无味，小便不清。
加减：None
功效：None
用法用量：上（口父）咀。每服3钱，水1盏，加生姜4片，大枣1枚，煎，空心热服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《医方类聚》引作“消气汤”。
