序号：73352
名称：椒术养脾丸
组成：人参1两半，白术2两（炒），炙草5钱，苍术1两（炒），川椒2两（炒），木香1两，茯苓3两，砂仁1两（炒），干姜1两（炒）。
出处：《医略六书》卷十九。
主治：胃虚寒湿，腹痛泄泻，脉沉者。
加减：None
功效：益脾养胃，补火温中。
用法用量：每服3钱，米饮煎化，温下。
制备方法：上为末，米糊为丸。
临床应用：None
各家论述：脾虚胃弱，寒湿内滞而不能健运，故腹痛不止，泄泻不已焉。川椒补火温中，白术健脾燥湿，人参扶元补胃气，苍术燥湿强脾土，干姜暖胃祛寒，茯苓和脾渗湿，木香调气化，砂仁醒脾胃，炙草缓中益胃也。丸以米粥，下以米饮，总取益脾养胃之功，洵为补火温中之剂。
用药禁忌：None
附注：None
