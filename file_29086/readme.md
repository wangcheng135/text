序号：29094
名称：阿胶丸
组成：真阿胶4斤（火炙令热），蛇蜕皮1条（烧灰），熟艾半两（烧灰），败笔1管（用头烧灰），大麦花少许（炙干，如无此花，以麦糵上牙子代之亦可，为细末）。
出处：《博济》卷四。
主治：产前产后诸疾。
加减：None
功效：大安胎脏。
用法用量：妇人有身，10个月满足者，有诸般疾病，用井花水磨下1丸；产后有病，用通灵散1字，醋汤磨下1丸；如筑打着，及死胎在腹中，用醋汤送下1丸；产后咳噎，用千金散子1字，热水调下1丸；产前产后被血冲心，用黄散子半钱，醋汤调下1丸；产后遗沥不止，用烧盐半钱，无灰酒送下1丸；难产者，3-5日，服此立下，用通灵散子，醋汤送下1丸；如妇产时，衣先下，未见儿，足踏衣生，用通灵散子1字，调下1丸；如儿先下，衣未见，须臾，用醋汤送下1丸；如刺前后心，用通灵散子1字，调下1丸；如浆破后，经3-5日不生，用黄散子1钱，酒调下1丸；产前产后痢，醋汤送下1丸；血气，用艾枝煎汤送下1丸。
制备方法：上为细末，以软粳米饭为丸，如鸡豆大，如丸时粘手，以少许面为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：罂粟炒令黄，为末，是千金散，真阿胶炙令黄，为末，是黄散子；蛇蜕皮烧灰，为末，是通灵散子。
