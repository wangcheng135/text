序号：50508
名称：茵陈散
组成：茵陈蒿3分，犀角屑3分，石斛（去根）3分，紫参3分，人参3分，白术3分，柴胡（去苗）3分，桂（去粗皮）半两，芍药半两，防风（去叉）半两，吴茱萸（汤洗焙干，炒）1两，桔梗（炒）半两，白芜荑仁（炒）1分。
出处：《圣济总录》卷八十七。
主治：风劳瘦疾，七种冷气，六极，脾胃虚寒，不思饮食，疳痢休息或大小便涩；兼累年口疮，医治不愈者。
加减：None
功效：None
用法用量：白羊肝1具，细切，分3服，净去筋膜，每服入药末5钱匕，葱白5寸，细切，一处拌和，用湿纸裹，慢火煨熟，空心顿服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
