序号：43777
名称：厚朴丸
组成：厚朴3两（去粗皮，涂生姜汁，炙令香熟），神曲1两（微炒），当归1两，枳壳1两（麸炒微黄，去瓤），白矾1两半（烧灰），五味子1两，缩砂1两（去皮），黄连1两（去须，微炒），白龙骨1两，白石脂1两，干姜1两（炮裂，锉），诃黎勒2两（煨，用皮），白茯苓1两，人参1两（去芦头），附子1两（炮裂，去皮脐）。
出处：《圣惠》卷二十六。
主治：脾劳。心腹气冷痛，时时下痢。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以粥饮送下。
制备方法：上为末，以酒煮曲糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
