序号：70782
名称：紫金膏
组成：白芷6钱，闹洋花7钱，三柰7钱，大茴香7钱，青皮7钱，草乌7钱，川乌7钱，威灵仙7钱，甘松7钱，小茴香7钱，大黄7钱，独活7钱，干蟾1个，乱头发3两。
出处：《疡医大全》卷七。
主治：寒湿气，漏肩风，诸般疼痛。
加减：None
功效：None
用法用量：宜摊厚些贴。立刻止痛。
制备方法：麻油40两，同药入锅，熬至发化，滴水成珠，再下密陀僧（研细）11两，收成膏，再下松香、葱汁、姜汁、凤仙花汁，各煮1次，研细，5两，入膏化尽搅匀，倾在钵内，重汤炖化，再下：潮脑7钱，青黛、桂皮各6钱，丁香、雄黄各5钱，轻粉4钱，血竭、乳香（去油）、没药（去油）、儿茶各3钱3分，滑石3钱，龙骨2钱5分，麝香、冰片各5分，搅入和匀收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
