序号：77397
名称：越鞠保和丸
组成：苍术（米泔浸3宿，炒）1两，抚芎（酒洗）1两，神曲（炒）1两，香附（童便浸，炒）1两，栀子（炒）5钱，陈皮1两，半夏（炮）1两，白茯苓1两，连翘5钱，莱菔子（炒）5钱，枳实（麸炒）1两，白术3两，黄连（酒炒）1两，山楂（去核）2两，木香5钱，当归（酒洗）1两。
出处：《古今医鉴》卷四。
主治：忧思过度，损伤脾胃，郁结不舒，呃逆胸满。
加减：None
功效：扶脾开郁，行气消食，清热化痰。
用法用量：每服50丸，淡姜汤送下；或酒下亦可。
制备方法：上为末，姜汁泡蒸饼为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：越曲保和丸（《杂病源流犀烛》卷十八）。
