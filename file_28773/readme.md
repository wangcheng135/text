序号：28781
名称：十圣散
组成：人参5分，黄耆5分，白术5分，地黄5分，砂仁（炒）5分，炙甘草1钱，归身1钱，川芎1钱，白芍（炒）1钱，川续断8分。
出处：《广嗣纪要》卷八。
主治：胎动不安。因母疾病，气血衰少，不能护养其胎，以致不安者。
加减：腹痛下血者，加阿胶、艾叶。
功效：None
用法用量：水煎服。
制备方法：上锉。
临床应用：None
各家论述：《医略六书》：妊娠气血亏，不能滋荣胎息，故胎动不安。方中以人参扶元补气，黄耆补气固中，熟地补血以滋冲任，白术健脾以生血气，当归养血荣经脉，白芍敛阴安胎息，川芎调血海，续断续经筋脉，炙甘草缓中益胃，砂仁开胃醒脾，为散水煎，使脾胃调和，则气血内充而胎得所养，胎无不宁，何胎动之足虑哉。
用药禁忌：忌恼怒、生冷、一切辛热等物。
附注：None
