序号：70517
名称：紫金丹
组成：鹿脊1具（带骨肉全者，锉作小块），舶上硫黄半两，好硇砂2钱（以上各研细，入无灰酒3升，同浸鹿骨1日，取焙干，再浸再焙，酒尽为度，为末别用），破故纸3两，禹余粮4两（火煅醋淬3次），舶上茴香3两（炒用），川乌头3两（炮），巴戟1两（去心），石中黄1两（醋淬），太阴玄精石1两。
出处：《普济方》卷二二六引《卫生家宝方》。
主治：一切虚惫，腹中痞块，恶心泄泻不食。
加减：None
功效：None
用法用量：每服1丸，空心温酒送下。腊月合之，可久收。若无鹿脊骨，以麋脊骨代之亦得。
制备方法：上为细末，与前鹿脊骨末拌和极匀，酒糊为丸，如鸡头子大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
