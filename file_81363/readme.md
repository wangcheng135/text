序号：81379
名称：瀹经汤
组成：人参7分，白茯苓7分，熟地7分，小茴7分，白术（蜜炙）1钱，川芎1钱，甘草1钱3分，黄芩1钱3分，柴胡1钱3分，枳壳1钱3分（麸炒），当归1钱5分，白芍1钱5分，香附1钱5分，生姜3片，灯芯3寸（长7茎）。
出处：《叶氏女科》卷一。
主治：妇女17-18岁经脉不通、或阻隔半年百日，面色青黄、饮食不思，或作寒热，头痛眩晕，腹中结块，烦闷呕吐，或作膨胀。
加减：如肚痛，加干漆（炒令烟尽）、玄胡索各7分；呕吐恶心，加良姜、砂仁各7分；手足麻痹，加肉桂4分；咳嗽，加五味子、杏仁（去皮尖）、款冬花各7分。
功效：None
用法用量：水煎，空心热服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
