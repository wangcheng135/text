序号：32388
名称：坠翳丸
组成：人参1钱，川当归1钱，甘菊花1钱半，北细辛2钱，北五味2钱，川芎2钱，旋覆花2钱，黄连2钱半，荆芥3钱，甘草7钱半，黄芩3钱，防风3钱，羌活3钱，柴胡3钱半，知母4两，赤芍药半两，白蒺藜（炒）半两，木贼（去节）1两（上为散，晒干为末，不要火焙，然后入后药料），石膏3钱，真珠1钱，硼砂2钱半，琥珀2钱，生石决明2个（火炙），生麝香（硫黄气者不可用，大黄气者不妨），磁石1钱半（火烧，淡醋浸3次，烧了出火气用）。
出处：《普济方》卷八十引《海上方》。
主治：眼疾日深，及白瓷大者。
加减：None
功效：None
用法用量：临睡及食后细嚼3丸，白汤咽下；或用盐汤、枣汤、灯心汤3样皆可。
制备方法：上为末，前药一处，粳米糊为丸，如龙眼大，用朱砂3钱半，别研为衣，取日气阴干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
