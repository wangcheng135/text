序号：40419
名称：虎舌丸
组成：干虎舌半两（用石臼杵捣为末），生草乌头尖（末）3钱，磁石半两（性紧者，石臼杵捣为末），水银3钱（同磁石末1处研细，令水银星尽），硫黄（舶上者）1钱，消石2钱（同硫黄研），楮实（末）2钱，硇砂（透明者）2钱（研），丹砂1钱（研），金牙半两（石杵臼捣为末）。
出处：《圣济总录》卷一四○。
主治：金疮箭头不出。
加减：None
功效：None
用法用量：用两头尖角盒子盛贮。每出箭头1枚，用药1丸，于窍内将铜箸点入箭疮内，即以好酒少许，摩疮四畔。次将红散子摩疮，须臾觉疮极热而痒，其箭头当日内自出。次以生肌金华散掺疮内，以紫金膏封之。
制备方法：上为极细末，石脑油为丸，如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
