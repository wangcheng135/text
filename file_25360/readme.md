序号：25368
名称：导气除燥汤
组成：茯苓（去皮）2钱，滑石（炒黄）2钱，知母（细锉，酒洗）3钱，泽泻3钱，黄柏（去皮，酒洗）4钱。
出处：《兰室秘藏》卷下。
主治：血涩至气不通而致小便闭塞不通。
加减：None
功效：None
用法用量：每服5钱，水3盏，煎至1盏，去滓，稍热空心服；如急闭，不拘时候。
制备方法：None
临床应用：None
各家论述：《脾胃论注释》：方中黄柏、知母滋肾阴、润肾燥，清其源则流自洁；茯苓、泽泻甘淡，与咸寒为伍，有利尿降热的作用；滑石为石药中的润药，利六腑之涩结，下达膀胱，润滑肠道。本方并无调气药物，而冠以“导气”二字，在于黄柏、知母经过酒洗，酒性散发起先升后降的作用，滑石炒黄借火性以行润滑之药，故曰导气。
用药禁忌：None
附注：None
