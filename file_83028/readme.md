序号：83044
名称：乌头丸
组成：川乌头2两（以童子小便浸3宿，去皮脐，薄切，焙干，入诸药），白僵蚕半两（微炒），蛜（虫祁）半两（半生半熟，去足），白附子半两（炮裂），当归半两，天麻半两，萆薢半两，石斛半两（去根，锉），桂心半两，麻黄1两（去根节），天南星半两（炮裂），天雄半两（炮裂，去皮脐），白头翁1两，郁李仁1两（汤浸，去皮尖，微炒），羌活3分，白蒺藜1两（微炒，去刺），天蓼实1两，防风3分（去芦头），仙灵脾1两。
出处：《圣惠》卷二十。
主治：瘫痪风。荣卫气滞，经络涩壅，致手足不遂。
加减：None
功效：None
用法用量：每服20丸，食前以豆淋酒下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
