序号：44029
名称：咳血方
组成：青黛、瓜蒌仁、诃子、海粉、山栀。
出处：《丹溪心法》卷二。
主治：咳血。肺热咳嗽，痰中带血，咯痰不爽，心烦口渴，颊赤便秘，舌苔黄，脉弦数。
加减：咳甚者，加杏仁（去皮尖），后以八物汤加减调理。
功效：清热化痰，敛肺止咳。
用法用量：噙化。
制备方法：上为末，以炼蜜同姜汁为丸。
临床应用：None
各家论述：1.《医方考》：肺者，至清之脏，纤芥不容，有气有火则咳，有痰有血则嗽。青黛、山栀所以降火，瓜蒌、海粉所以行痰，诃子所以敛肺。然而无治血之药者，火去而血自止也。_x000D_
2.《医方集解》：此手太阴药也。肝者，将军之官。肝火上逆，能烁心肺，故咳嗽痰血也。青黛泻肝而理血，散五脏郁火；栀子凉心而清肺，使邪热下行，二者所以治火。栝楼润燥滑痰，为治嗽要药；海石软坚止嗽，清水之上源，二者降火而兼行痰。加诃子者，以能敛肺而定痰喘也。_x000D_
3.《医林纂要》：诃子肉苦酸涩，生用敛肺清金，降逆止咳；栀子苦酸，炒黑用，抑妄行之相火，决三焦之水道，敛肺宁心，降逆气，止妄血；海石咸涩，补心敛肺，清金降火，渗湿消痰；瓜蒌仁甘苦而能润，轻虚上浮，宁心润肺，泄逆清火，除痰去垢，开豁膻中之清气，亦治咳要药。青黛辛咸，此补肝而泻肺，然辛行肝气，使肝木自畅，则相火不至灼金；咸散肝血，则血各循经，而不至逆涌于上，且能解毒热。蜜亦润肺，能补清高之气。_x000D_
4.《医方论》：咳嗽痰血，固属君相之火犯肺。此方但清火而不治血，乃去所扰则自安之义。然业经失血，则肺已大伤，岂可置之不论不议。去诃子而加清养肺阴之药，始为得之。
用药禁忌：None
附注：肺血丸（《医林纂要》卷四）。《医学纲目》本方用量为各等分。
