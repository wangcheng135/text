序号：51177
名称：荜澄茄丸
组成：荜澄茄1两，藿香叶（去土）1两，人参（去芦头）1两，蓬莪荗（煨香，切）1两，甘草（炙）1两，丁香1两，茴香2两（微炒），木香1两半，肉豆蔻（面裹，煨熟）1分，麝香1钱（别研），安息香1两（酒煮，研开，滤去沙石）。
出处：《杨氏家藏方》卷六。
主治：脾虚胃弱，气滞不匀，心腹疼痛，宿冷不消，腹胁虚胀，不思饮食，面色痿黄，脏腑滑泄，气不升降。
加减：None
功效：None
用法用量：每服1丸，食前细嚼，橘皮汤或木香汤送下。
制备方法：上药除安息香外，并为细末，次入炼熟蜜半斤和丸，每1两作15丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
