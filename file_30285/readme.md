序号：30293
名称：阿魏雷丸散
组成：阿魏3分，紫雷丸3分，雄黄3分，紫石英3分，朱砂半两，滑石半两，石胆半两，丹砂半两，雚芦半两，白蔹半两，犀角半两，斑蝥（去足翅）40枚，芫青（去足翅）40枚，牛黄5分，紫铆1分。
出处：《千金翼》卷二十一引《耆婆方》。
主治：癞风。
加减：None
功效：None
用法用量：每服1钱匕，空腹以清酒2合和药饮尽。大饥即食小豆羹饮为良，莫多食，但食半腹许即止。若食多饱，则虫出即迟。日西南空腹更1服，多少如前。若觉小便似淋时，不问早晚，即更服药，多少亦如前，大饥即食。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
