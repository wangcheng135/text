序号：80679
名称：嚏惊丸
组成：牛黄3皂子大，芦荟3皂子大，熊胆3皂子大，生蟾酥10个，朱砂2皂子大，龙脑半皂子大，麝香半皂子大，雄黄5钱，全蝎半两（轻炒），白矾（枯过）1两，防风（焙）1两，荆芥穗1两。
出处：《幼幼新书》卷九引《张氏家传》。
主治：小儿急慢惊风。
加减：None
功效：None
用法用量：每服1丸，用倒流水化药，如小儿乎足牵搐，灌鼻内，良久打嚏即愈；如未定，再灌之；三次不嚏，恶候也，别用药治之。如疮疹倒靥，及疮平黑色斑出，急用鸡子壳盛酒半壳、生猪血半壳，合盛1壳，用药2-3丸化在内，火灰内暖热温，时时服之。重午日取酥合药。
制备方法：上除脑、麝外，1处为细末，然后别研脑、麝细，入前药内，再研，用蟾酥，少添数粒粳米饭和匀为丸，如芥子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
