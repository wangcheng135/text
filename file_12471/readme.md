序号：12477
名称：白薇丸
组成：白薇3分，柏子仁1两，牡丹3分，熟干地黄1两，芎藭半两，羌活半两，当归3分（锉，微炒），黄耆半分（锉），人参3分（去芦头），桂3分，附子3分（炮裂，去皮脐），石斛3分（去根，锉），白茯苓1两，白芍药半分，五味子半两，白术3分，甘草3分（炙微赤，锉），肉苁蓉3分（酒浸1宿，锉，去皱皮，炙干）。
出处：《圣惠》卷八十一。
主治：产后风虚劳损，寒热发歇，血脉虚竭，四肢羸弱，饮食无味。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
