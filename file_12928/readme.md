序号：12935
名称：石刻安肾丸
组成：苍术4两（1两用茴香1两炒，1两用青盐1两炒，1两用茱萸1两炒，1两用猪苓1两炒，各炒令黄色，取术用），川乌（炮，去皮脐）1两，附子（炮，去皮脐）1两，川楝子（酒浸，去该）1两，巴戟（去心，炒）1两，白术（炒）1两，陈皮（炒）1两，肉苁蓉（酒浸，炙）2两，破故纸（炒）2两，茯苓1两（炒），肉豆蔻（面裹煨）1两，木香（不见火）1两，当归（火焙干）1两，杜仲（炒去丝）2两，熟地黄（酒浸，蒸10次，火焙）1两，菟丝子（酒浸，炒）1两，茴香1两，黑牵牛（半生，半炒）1两，山药（炒）1两，晚蚕蛾（去头足翅，炒）1两，葫芦巴（酒浸，炒）1两，肉桂（不见火）1两，石斛（炒）1两，川牛膝（酒浸，炒）1两。
出处：《得效》卷八。
主治：真气虚惫，脚膝缓弱，目暗耳鸣，举动倦乏，夜梦遗精，小便频数，一切虚损。
加减：None
功效：久服壮元阳，益肾气，健筋骨，生血驻颜，扶老资寿。
用法用量：每服40丸，空心盐汤送下。
制备方法：上为末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
