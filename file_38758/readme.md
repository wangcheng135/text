序号：38768
名称：三豆蔻饮子
组成：肉豆蔻1两（锉），白豆蔻1两（锉），草豆蔻2两（锉），甘草1两半（锉），生姜7两。
出处：《魏氏家藏方》卷五。
主治：脾胃受冷过多，胸膈痞闷，气不舒畅，饮食之后，胸间噎塞，呼吸气短，全不思食，面无颜色，日渐气弱，遂成瘦怯者。
加减：None
功效：None
用法用量：每服1丸，旋打散，用水1大盏半，煎至1盏，入盐1捻，再煎1-2沸，约至8-9分，热服；并2服；滓再煎服，不拘时候。
制备方法：上先以生姜2两研烂，入前药拌和，盒一时许打开，再以生姜2两研烂如前，以前药拌和，盦1时，再打开，再以生姜3两研烂，入前药拌和，趁湿捻成丸，如鸡子大，焙干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
