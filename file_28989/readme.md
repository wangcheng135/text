序号：28997
名称：苏子降气汤
组成：紫苏子（炒）2钱半，半夏（汤泡）2钱半，前胡（去芦）1钱，甘草（炙）1钱，厚朴（去皮，姜制炒）1钱，陈皮（去白）1钱，川当归（去芦）1钱半，沉香7分。
出处：《准绳·类方》卷二。
主治：上盛下虚，气滞痰壅，咳嗽，喘急，头痛，胃脘痛。
加减：虚冷人加桂5分、黄耆1钱。
功效：None
用法用量：水2钟，加生姜3片，煎至1钟，不拘时候服。
制备方法：None
临床应用：None
各家论述：《血证论》：气即水也，水凝则为痰，水泛则为饮，痰饮留滞，则气阻而为喘咳。苏子、生姜、半夏、前胡、陈皮宣除痰饮，痰饮去而气自顺矣。然气以血为家，喘则流荡而忘返，故用当归以补血；喘则气急，故用甘草以缓其急；出气者肺也，纳气者肾也，故用沉香之纳气入肾，或肉桂之引火归元为引导。
用药禁忌：None
附注：None
