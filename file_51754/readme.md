序号：51769
名称：钟乳丸
组成：钟乳粉3两，泽兰2两，防风1两（去芦头），人参1两（去芦头），柏子仁2两（微炒），石膏1两半（研细，水飞过），芎藭1两，附子1两（炮裂，去皮脐），续断1两，白芷1两，牛膝1两（去苗），当归1两半（锉碎，微炒），木香1两，干姜1两（炮裂，锉），藁本1两，细辛1两，桂心1两，艾叶3分（微炒），麦门冬1两半（去心，焙），白芜荑1两，熟干地黄1两。
出处：《圣惠》卷七十。
主治：妇人风虚劳冷，羸瘦，四肢烦疼，脐下时痛，不能饮食，面目黄黑，忧恚不乐。
加减：None
功效：None
用法用量：每服30丸，食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
