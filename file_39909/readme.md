序号：39919
名称：肥儿丸
组成：白芜荑（去壳秤）、黄连（去须）、神曲、麦糵各等分。
出处：《幼幼新书》卷二十五引《朱氏家传》。
主治：小儿食积，脾热成疳，乳食不下，腹胀吐泻，腹大青筋，体瘦多涎，发稀白秃，龈烂目翳，口舌、周身生疮，小便澄白。以及惊风后失音。或治疳积，或治疳瘦。风后喑不能言。涎多，涎流不出口者，乃名脾热。五疳或白秃体瘦，腹胀成疳，或作泻不止，或食积脾疳，目生云翳，牙龈腐烂，发热瘦法。
加减：None
功效：None
用法用量：每服30粒，食前米饮送下。
制备方法：上为末，用豮猪胆煮糊为丸，如大麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：四味肥儿丸（《小儿痘疹方论》）、五疳芜荑丸（《医方类聚》卷二五五引《经验良方》）。本方用法：《直指小儿》：每服五丸，用陈皮、木香、使君子、炙甘草煎汤送下。《普济方》：木通汤送下。
