序号：77477
名称：雄黄散
组成：雄黄（研）半两，丹砂（研）1分，牛黄（研）1分，丁香1分，桂（去粗皮）半两，天麻3分，麝香（研）1两，天南星（炮裂）3分，龙脑1分（先研如粉，入麝香同研，次入前3味同研），半夏（为末，生姜汁作饼晒干）3分，麻黄（去节煎，掠去沫，焙）3分，白僵蚕（炒）3分，附子（炮裂，去皮脐）半两，干姜（炮）半两，大黄（锉，醋炒）半两。
出处：《圣济总录》卷十三。
主治：风气入中，蕴积生热，口干目黄，时发潮躁。
加减：None
功效：None
用法用量：每服半钱，加至1钱匕，空心、夜卧温酒调下。汗出即愈。
制备方法：上10味为散，与别研5味和匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
