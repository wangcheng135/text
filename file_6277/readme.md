序号：6277
名称：去腐白玉膏
组成：猪板油（腊月取，切碎，锅熬勿焦，去滓，凝后始白时用）4两，黄蜡（净者，化开，倾瓷盘内，露2-3夜自白）1两，白蜡（净者）1两，真铅粉2两，龙骨1钱，螵蛸1钱，象牙末1钱，凤凰退（焙黄）1钱，鸡内金（不见水，阴干，焙）1钱，乳香（去油）1钱，没药（去油）1钱，真轻粉1钱，蜗牛（焙）1钱，水银1钱，白粉霜2分，冰片2分。
出处：《医林绳墨大全》卷九。
主治：结毒，粉毒，疳蛀，烂臁，一切痈疽、顽疮。
加减：None
功效：None
用法用量：每用少许，摊于黑膏药中心。腐肉自化，条条片片，粘连而下。
制备方法：上药各为细末，另以樟脑研末，放铜盆内，以大碗盖定，湿绵纸封口，上用生面调糊，厚涂纸上，以微火离盆底拳高，少顷冷定，取升上碗者用4钱。先将猪油入铜锅内重汤煮化，投黄白蜡，次下铅粉，搅匀，离火方下龙骨等味，待将凝，下水银，急搅，缓则沉底，末下樟片，再搅极匀，瓷罐收固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
