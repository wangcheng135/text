序号：43079
名称：养心健脾丸
组成：党参3钱，于术3钱（土炒），茯神5钱，焦枣仁4钱，远志肉1钱，归身3钱（土炒），杭芍3钱（炒），炒杜仲4钱，陈皮2钱，薏米5钱（炒），广砂1钱5分（炒），沙苑蒺藜3钱，石莲肉4钱，谷芽3钱（炒），山药4钱（炒），炙甘草1钱5分。
出处：《慈禧光绪医方选议》。
主治：None
加减：None
功效：养心健脾。
用法用量：每服3钱，白开水送服。
制备方法：上为细末，枣汤、神曲糊为丸，如绿豆大，朱砂为衣。
临床应用：None
各家论述：本方从归脾汤和参苓白术散两方化裁而来，实具养心健脾之功效。归脾汤中去黄耆，以防补气升阳太过；去木香，以防芳香辛散耗阴；龙眼肉、当归之属似嫌滋腻，一并减去。用方大旨在于养心。参苓白术散功可和胃利湿，以沙苑蒺藜、白芍、杜仲易扁豆、桔梗，是为兼顾肝肾。立方稳健，寒热相宜。
用药禁忌：None
附注：None
