序号：52169
名称：面膏
组成：防风1两，藁本1两，辛夷1两，芍药1两，当归1两，白芷1两，牛膝1两，商陆1两，细辛1两，密佗僧1两，芎藭1两，独活1两，鸡舌香1两，零陵香1两，萎蕤1两，木兰皮1两，麝香1两，丁香1两，未穿真珠1两，蕤仁2两，杏仁2两（去皮尖），牛髓5升，油1升，腊月猪脂3升（炼），麞脑1具，鹿脑1具（若无，羊脑亦得）。
出处：《千金翼》卷五。
主治：面有(黑干)(黑曾)，及（疒咅）瘰并皮肤皴劈。
加减：None
功效：None
用法用量：即涂面上。甚妙。
制备方法：先以水浸脑髓使白，藿香以上（口父）咀如麦片，乃子脑髓脂油内煎之，三上三下，即以绵裹搦去滓，乃纳麝香及真珠末，研之干遍，凝。
临床应用：None
各家论述：None
用药禁忌：None
附注：据用法，组成中当有藿香。
