序号：43270
名称：养血调经膏
组成：当归2两，川芎1两6钱，白芍2两，益母草2钱，丹参1两6钱，泽兰2钱，茯苓5钱，木香1两6钱，牛膝6钱，杜仲8钱，柴胡8钱，鲜姜9两，续断8钱，香附1两6钱，陈皮2两，白术6钱，艾把2两5钱，腹皮2两。
出处：《北京市中药成方选集》。
主治：积气，积寒，神经衰弱，腰腿疼痛。
加减：None
功效：调经养血。
用法用量：贴时先将鲜姜煨熟，擦净肚脐部、腰部，各贴1张。
制备方法：上药酌予切碎，每锅用香油4斤6两，油热时先炸坚硬药品，后入叶草等药，将陈皮焖湿放在上边，分次撒于浮面，炸至焦枯，用细铁丝筛子过滤，去净滓，炼至滴水成珠为度；在火上下丹，每锅用樟丹1斤11两，搅匀，将锅端下用水喷，随喷随搅，烟出净即成。放入水中，消毒凝坨，加热溶化，兑入鹿茸粉3钱，人参尾5钱，搅匀，摊贴。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
