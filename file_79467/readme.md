序号：79483
名称：摩风膏
组成：白附子5钱，白芍药5钱，白茯苓5钱，零陵香5钱，白及5钱，白蔹5钱，白芷5钱，白檀5钱，藿香5钱，升麻5钱，细辛5钱，黄耆5钱，甘草5钱，杏仁（去皮尖）5钱，脑子1分，瓜蒌根1两，大瓜蒌2两（去皮），黄蜡6两，脂麻油1斤。
出处：《外科精义》卷下。
主治：头面五发疮肿，疥癣烫火破伤。
加减：None
功效：摩风止痛，灭瘢痕。
用法用量：旋用磨风涂之。
制备方法：上药锉，油内浸百日，于腊月慢木炭火上银石器内，煎至白芷微黄色，离火，入瓜蒌二味着内，煮百沸，重绵滤去滓，再慢火上炼油香，下削净蜡溶开为度，倾在瓷器内收贮，上掺脑子，密封。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
