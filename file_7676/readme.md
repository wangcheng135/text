序号：7676
名称：二妙丸
组成：黄柏末、苍术末各等分。
出处：《医学纲目》卷二十引朱震亨方。
主治：湿热下注之足膝肿痛，痿证，湿疮，湿疹，丹毒，白带，腰痛。下焦湿热肿痛，或流注游走，遍身疼痛。湿热腰痛。热痹，肌肉热极，唇口干燥，筋骨痛不可按，体上如鼠走状，属湿热伤气分者。脚气。热盛于下焦，而成痿证者。湿热下注，腿脚发沉作肿，及膝下生疮。湿热下注，足膝红肿热痛，下肢丹毒，白带，阴囊湿疹等。湿疮，臁疮等证，肌肤焮红，作痒出水，属于湿热内盛者。
加减：None
功效：清热燥湿。
用法用量：None
制备方法：炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《成方便读》：湿热之邪，虽盛于下，其始未尝不从脾胃而起，故治病者必求其本，清流者必洁其源。方中苍术辛苦而温，芳香而躁，直达中州，为燥湿强脾之主药；但病既传于下焦，又非治中可愈，故以黄柏苦寒下降之品，入肝肾直清下焦之湿热，标本并治；中下两宣。
用药禁忌：None
附注：阳明二妙丸、苍柏二妙丸（《症因脉治》卷三）。
