序号：32029
名称：万字丸
组成：白花蛇1条（去头皮脏骨），白蒺藜7两，白僵蚕1两，白附子1两，威灵仙1两，风子肉1两，土麻仁6两，川黄连（乳蒸）5钱。
出处：《疯门全书》。
主治：麻风。
加减：血热，加丹皮、栀仁；溃烂，加牛子、骨皮；拘挛拳曲，加天麻、钩藤；麻本甚，加蒺藜、土麻仁；面上红云红堆，加白附、僵蚕；面如油光，加白附、蝉蜕；鼻塞声散，加桑皮，黄芩；肉痹，加玄参；足麻木溃烂，加黄柏，银花、土茯苓；骨节疼痛，加独活；瘙痒，加丹皮、蝉蜕；癣多，加鲜皮、浮萍；红堆红圈，加山甲、皂刺；热甚，加川连；遍身麻木，非花蛇不能。
功效：None
用法用量：早、晚空心各服5钱。
制备方法：炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
