序号：2783
名称：天池膏
组成：天花粉半斤，黄连半斤，人参4两，知母（去壳）4两，白术（炒，去芦）4两，五味子3两，麦门冬6两（去心），藕汁2碗，怀生地黄汁2碗，人乳1碗，牛乳1碗，生姜汁2酒杯。
出处：《寿世保元》卷五。
主治：三消。
加减：None
功效：None
用法用量：每用2-3匙，安舌咽之，或用白汤送下。
制备方法：上先将天花粉7味切片，用米泔水16碗，入砂锅内浸半日，用桑柴火慢熬至5-6碗，滤清，又将渣捣烂，以水5碗，煎至2碗，同前汁又煎2-3碗，入生地等汁，慢熬如饧，加白蜜1斤，煎去沫，又熬如膏，乃收入瓷罐内，用水浸3日，去火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
