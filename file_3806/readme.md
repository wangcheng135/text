序号：3806
名称：引方六郁汤
组成：陈皮（去白）1钱，香附子2钱，半夏（泡，去皮）7分，山栀仁（炒）7分，赤茯苓7分，苍术5分，抚芎5分，砂仁（炒，研细）5分，甘草（炙）5分。
出处：《松崖医经》卷下。
主治：郁证。
加减：若气郁，倍香附、砂仁，加乌药、木香、槟榔、苏梗、干姜；若湿郁，倍苍术，加白术；若热郁，倍山栀，加黄连；若痰郁，加南星、枳壳、猪牙皂荚；若血郁，加桃仁、红花、牡丹皮；若食郁，加山楂、神曲、麦芽面。
功效：解诸郁。
用法用量：上切细，作1服。用水2盏，加生姜3片，煎至1盏，去滓温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
