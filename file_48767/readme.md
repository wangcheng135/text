序号：48782
名称：大七香丸
组成：香附子（炒）192两，麦糵（炒）100两，丁香皮330两，缩砂仁250两，藿香叶250两，甘松64两，乌药64两，肉桂（去粗皮）250两，甘草（炒）250两，陈皮（去白，洗）250两。
出处：《局方》卷三（绍兴续添方））。
主治：男子、妇人脾元气冷，胃气虚乏，不思饮食，心膈噎塞，渐成膈气，脾泄泻痢，气刺气注，中酒吐酒，冷痃翻胃，霍乱吐泻。脾胃不和，三焦痞滞，气不宣通，食饮迟化。
加减：None
功效：消谷进食。
用法用量：每服1粒，盐酒、盐汤嚼下；妇人脾血气，如经水不调，并用炒姜酒嚼下；醋汤亦得。
制备方法：上为末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：忌生冷、肥腻。
附注：None
