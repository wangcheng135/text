序号：9806
名称：二炁灵砂丹
组成：水银1斤，硫黄4两。
出处：《惠直堂方》卷一。
主治：诸虚痼冷，厥逆，及上盛下虚，痰涎壅盛者。
加减：None
功效：升降阴阳，和五脏，助元气，下逆气，扶危救急。
用法用量：每服3分或1分。常服，每用人参汤空心送下，或枣汤送下，疝气，木肾偏坠肿痛，茴香汤送下；白浊遗精，白茯苓汤送下；虚劳咳嗽，生姜、乌梅、苏梗汤送下，腰腹满痛，莪术汤送下；盗汗、溺多，煅牡蛎少许煎汤送下；疟疾不已，桃、柳枝汤送下；吐逆翻胃，丁香、藿香汤送下；中风痰厥，面青，木香汤送下；走注风痛，遍身作痛，葱白汤送下；脚膝痛，木瓜汤送下；气滞，生姜、陈皮汤送下；妇人血气作痛，延胡索、五灵脂、酒、醋各半送下；小儿慢惊沉困，胃虚呕吐，神脱，人参、丁香汤送下。
制备方法：入锅炒断星，入阳城罐升打5香，开出成束针纹赤色为佳，桑灰醋淋汁煮，制米糊丸，如芥子大。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。胃虚呕吐，霍乱，肺热生痰，心虚有火，神魂不宁者，俱不可用。
附注：None
