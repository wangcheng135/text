序号：47748
名称：神仙附益丹
组成：香附米1斤（童便浸透，取出，水洗净，露1宿，晒干，再浸，再露，再晒，如此2次，用好醋浸透过宿，晒干为末），益母草12两（东流水洗净，烘干为末）。
出处：《古今医鉴》卷十一引徐宪副方。
主治：妇人不孕，脉涩滞者。妇人百病。血虚不孕。
加减：None
功效：None
用法用量：每服50-70丸，空心、临卧淡醋汤送下。
制备方法：上用香附4两，北艾1两，煮汁3分，醋7分，将前2味和合为丸，如梧桐子大。
临床应用：None
各家论述：《医略六书》：血凝于络，气滞于经，故无癸不调，不能媾精而孕子焉。香附理血中之气，力能解郁调经；益母调冲任之血，性善生新去宿。艾汤以丸之，温酒以行之，使子宫温暖，则血活气行而经脉融和，天癸如度，岂有不孕之妇乎？
用药禁忌：None
附注：神仙附益丸丹（《济阴纲目》卷六）、附益类仙丹（《医略六书》卷二十七）。
