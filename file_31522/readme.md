序号：31531
名称：来苏散
组成：柴胡（去芦）2两，甘草（炙）2两，干姜2两，肉桂（去粗皮，不见火）1两，桔梗1两，防风1两，荆芥穗1两，五加皮1两，芍药半两，麻黄（去节）1两半，陈皮（去白）1两半，黄耆（蜜水浸一宿，炙）1分。
出处：《局方》卷二（续添诸局经验秘方）。
主治：四时瘟疫、伤寒，身体壮热，头痛憎寒，项脊拘急，浑身疼痛，烦渴闷乱，大小便涩，嗜卧少力，全不思饮食；及诸气疾，五劳七伤，山岚瘴疟，寒热往来。
加减：None
功效：和顺三焦，辟瘴气，进饮食。
用法用量：每服2钱，水1盏，加生姜3片，同煎8分，不拘时候热服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
