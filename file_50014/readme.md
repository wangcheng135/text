序号：50029
名称：茯神丸
组成：茯神（去木）5钱，白茯苓（去皮）5钱，姜黄5钱，白术（炒）5钱，黄耆5钱，羌活5钱，五加皮5钱，川乌（炮，去皮脐）5钱，薏苡仁5钱，木瓜（去皮瓤）5钱，半夏曲5钱，神曲（炒）5钱，天雄（炮，去皮）5钱，白芍药5钱，乳香（别研）5钱，没药（研）5钱，防风5钱，独活5钱，当归5钱，玄胡索5钱，桑寄生5钱，海桐皮（去粗皮）5钱，木香5钱，陈皮（去白）5钱，枳壳（炮，去瓤）5钱，南星（炮）5钱，地龙（捶，去土）5钱，荆芥穗5钱，麝香1钱。
出处：《得效》卷十三。
主治：四肢挛缩不伸。
加减：None
功效：None
用法用量：每服30丸，温酒吞下，不饮，木瓜汤送下。
制备方法：上用豮肝1具，洗净，煮存性，研烂，同酒面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：豮肝丸（《普济方》卷九十七）。本方方名，《普济方》引作：“豮猪肝丸。”
