序号：373
名称：五香流气饮
组成：金银花2两，姜蚕1两5钱，连翘1两5钱，羌活1两5钱，独活1两5钱，瓜蒌仁1两5钱，小茴1两5钱，藿香5钱，丁香1钱，木香1钱，沉香1钱，甘草1钱。_x000D_
。
出处：《外科大成》卷四。
主治：结核痰核及阴毒流毒，黄鳅痈。由肝、脾二经湿热凝结而成，生于小腿肚里侧，疼痛肿硬，长有数寸，形如泥鳅，其色微红。
加减：None
功效：None
用法用量：上分为10剂，水煎，随上下服。如为丸，绿豆大，雄黄5分为衣，滚水送下。
制备方法：None
临床应用：急性血栓性静脉炎：用本方治疗七例急性血栓性静脉炎，皆获显效。并发现本方具有迅速控制炎症，防止疾病茬再迁延之功。如一中年男性，因患大隐静脉曲张，血栓性静脉炎(急性期)，已服清热解毒、活血化瘀之品不见效，近四天病情加重，伴有身热纳差，红肿痛甚，由原来5公分发展到18公分。试投“五香”三剂后，全身症状改善，病灶停止发展，疼痛锐减。继进五剂，全身症状消失，局部红肿消退。又给五剂，复查惟见6公分之索状静脉，但无压痛，临床治愈，转用活血化瘀法治疗。
