序号：33125
名称：万应喉中散
组成：上犀黄1钱（透甲者真），滴乳石5钱（研净末），真珍珠1钱（大者无油为妙），劈辰砂1钱（漂净，末），灯草灰3钱（陈者更佳），儿茶5钱，大梅片1钱，香白芷2钱（生晒，研净末），片黄柏3钱（生晒，研净末），苏薄荷7钱（生晒，研净末），甘草3钱（生晒，研净末），青黛3钱（去石灰，净末），上血竭3钱。
出处：《集验良方》卷一。
主治：喉痹，缠喉风，双单乳蛾，喉痈，喉疮，阴虚咽痛。
加减：None
功效：None
用法用量：用时吹喉。
制备方法：上药各为细末，照药称准分两，和匀，再研极细无声，瓷瓶贮好，勿令泄气。
临床应用：None
各家论述：None
用药禁忌：戒口为要。
附注：None
