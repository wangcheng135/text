序号：46315
名称：活血散
组成：赤芍5钱，归尾5钱，红花5钱，紫草5钱，木香2钱，血竭1钱。
出处：《医学入门》卷八。
主治：痘色淡白，或热极血焦不红活。
加减：None
功效：None
用法用量：每服2钱。痘色淡白，酒调下；热极血焦不红活，紫草煎酒送下。
制备方法：上为末。
临床应用：None
各家论述：1.《痘学真传》：当归、赤芍、血竭、红花皆行血之品，紫草凉解郁毒，木香、川芎流走气血。_x000D_
2.《成方切用》：气贵利而不贵滞，血贵活而不贵凝。木香、川芎调其气滞，赤芍、归尾、紫草、红花、血竭理其血凝。
用药禁忌：None
附注：血竭活血散（《痘疹仁端录》卷十四）。《赤水玄珠》有川芎。
