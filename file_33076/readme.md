序号：33085
名称：芜荑酒
组成：生地、独活、丹参、白附、甘遂2两，赤脂2两半，干姜、芜荑、麦冬、芫花、苏子1两，柏子仁、苁蓉、茯神、金牙、薯蓣、白术、蔓荆子、杜仲、石楠、白芷、人参、乌头、山茱萸、狼毒、川椒、防风、细辛、牛膝、寒水石、麻黄、当归、柴胡、乌药、牡蛎、枸杞子、桔梗、狗脊、天雄、石斛、桂心。
出处：《疮疡经验全书》卷七。
主治：久患枯挛，三十年癞，着床，及诸恶风，眉毛脱落。
加减：None
功效：None
用法用量：上药以酒2斗浸，夏3日，春、秋6日，冬9日，合用。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：方中除甘遂、赤脂、苏子外，用量原缺。
