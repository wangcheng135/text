序号：58867
名称：秘金丹
组成：生地黄半斤（洗净，薄切，日晒干，入新砂盆内，慢火炒黄黑色），官桂半两（去皮），蒲黄3钱（以纸衬砂盆内，炒赤黄色），白芍药半两，川芎3钱（炒），鸡头粉半两，莲花蕊2钱（焙），白龙骨3钱，熟地黄1两，肉苁蓉3钱（酒浸1宿，焙干），北五味3钱，菟丝子3钱，远志3钱（去心），鹿茸半两（酥炙），川当归半两（去芦），木香3钱，丁香3钱，天雄1对（去皮，炮，切）。
出处：《医方类聚》卷二一○引《吴氏集验方》。
主治：带下。
加减：None
功效：暖子宫，滋气血。
用法用量：每服60丸，空心以酒醋汤送下。
制备方法：上为极细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
