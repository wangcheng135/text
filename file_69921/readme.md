序号：69937
名称：琥珀养心丹
组成：琥珀（另研）2钱，龙齿（煅，另研）1两，远志（黑豆、甘草同煮，去骨）5钱，石菖蒲5钱，茯神5钱，人参5钱，酸枣仁（炒）5钱，当归7钱，生地黄7钱，黄连3钱，柏子仁5钱，朱砂（另研）3钱，牛黄（另研）1钱。
出处：《准绳·类方》卷五。
主治：气血虚弱，心神失养，惊悸怔忡，失眠健忘，气短体倦，自汗口干，头昏心烦，面色少华。心血虚，或怔忡心跳。坐卧不安，寐则易觉，多魇。四肢倦怠，精神不爽，面黄肌瘦。
加减：None
功效：None
用法用量：每服50丸，灯心汤送下。
制备方法：上为细末，将牛黄、朱砂、琥珀、龙齿研极细，以猪心血为丸，如黍米大，金箔为衣。
临床应用：None
各家论述：《医略六书》：心虚热炽，心神失养，则心气不宁，故心跳不已，触事易惊焉。生地养心阴以制火，人参补心气以宁心，黄连清心火之妄动，龙齿定魂魄之飞扬，枣仁滋养心神，远志交通心肾，归身养血荣心，茯神安神定志，柏仁养心气，琥珀利心营，菖蒲开心气以通窍，牛黄凉心热以定惊，朱砂镇坠心气、安心神，更以猪心血引之入心，金箔制肝坠热，灯心泄热从小便去也。盖热从下泄，则心火自降而心气和平，安有心跳善惊之患乎？
用药禁忌：None
附注：None
