名称：栀子栝楼汤
组成：栀子5钱（去壳，取仁，用甘草水浸1宿取出，晒干，捣筛，为末），栝楼根（取大3围者，去皮，捣烂，以水澄粉晒干）7钱，香豉小半合（取如法修事者），淡竹叶（东流水洗，切）小半升，葛根（秋露润透，阴干，锉碎）1两1钱，猪苓（锉，去粗皮，切片，东流水浸1宿取出，同升麻、麻黄等分，水润拌蒸1炷香，勿令泄气，去升麻，麻黄，晒干）9钱，滑石1两（取洁白者，以竹刀刮净，研如粉，每两用牡丹皮2两同煮3柱香，去牡丹，以东流水淘过，晒干），牡丹皮（铜刀削去骨，锉碎，同桃仁等分，酒润，蒸晒3次，去桃仁）7钱，知母（槐砧上锉碎，干木臼中捣烂）5钱，生姜（切）7钱。
出处：《痎疟论疏》。
主治：瘅疟。
加减：None
功效：None
用法用量：上以水3升5合，煮取升半，去滓，分3次温服。
