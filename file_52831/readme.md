序号：52846
名称：香豉汤
组成：香豉1升，薤白1升，栀子4两，黄芩4两，地榆4两，黄连3两，黄柏3两，白术3两，茜根3两。
出处：《千金》卷二十。
主治：下焦热毒痢鱼脑，杂痢赤血，脐下少腹绞痛不可忍，欲痢不出。
加减：None
功效：None
用法用量：上（口父）咀。以水9升，煮取3升，分3次服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：毒痢势甚而饮食尚强，胃气未艾，故可用黄连解毒清燥之剂，其用香豉、薤白必缘邪热未除，后重不减，故专取2味为方中之首推，且以白术除热消食，茜根、地榆解散滞血，然惟毒势方张者为宜。
用药禁忌：若痢久元气下降而见虚滞不食者，禁用。
附注：None
