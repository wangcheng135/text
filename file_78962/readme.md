序号：78978
名称：增益归茸丸
组成：熟干地黄（酒浸，9蒸）4两，鹿茸（去毛，酥涂炙）4两，五味子4两，山药（酒浸）2两，山茱萸（去核）2两，大附子（炮，去皮脐）2两，川牛膝（酒浸1宿）2两，白茯苓1两半，牡丹皮（去骨）1两半，泽泻（酒浸1宿）1两半，大当归4两（去芦），黄连2两（去芦）。
出处：《得效》卷八。
主治：None
加减：None
功效：养肝肾，益心血、利足膝、实肌肤，悦颜色。
用法用量：每服50丸，空心温酒、盐汤送下。
制备方法：上为末，用真鹿角胶半斤，锉细入银石器中，酒湖为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：归茸丸（《寿世保元》卷四）。本方方名，《普济方》引作“补益归茸丸”。
