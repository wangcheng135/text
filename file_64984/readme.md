序号：65000
名称：绿消煎
组成：绿矾5两。
出处：《普济方》卷三○○引《圣惠》。
主治：甲疽。或因剔甲伤肌，或因甲长侵肉，遂成疮肿痛。后缘官靴研损，四边肿焮，黄水出，侵淫相染，五指俱烂，渐渐脚跌泡浆四边起，日夜倍增。
加减：None
功效：None
用法用量：先以盐汤洗疮，帛裹干，用此敷之。或有虫有黄水，当日洗，汁断疮干。若患痛急，即涂少酥令润，每1遍，盐汤洗濯。有脓处常洗使净，其痂干不须近。每洗、干敷药如初，胆急痛即涂酥，5日即觉上痂，渐剥起赤，依前洗敷药，10日即疮渐渐剥尽痂落。软处或更生白脓泡，即搽破敷药，自然总愈。
制备方法：上药形色似朴消而绿色，置于铁板上。聚灰封之囊袋，吹令火炽，其矾即沸流出，色赤如熔金汁者是真也。候沸定汁尽，去火候冷，取出研为细末，似黄色收之。
临床应用：张侍郎得此病，卧经六十日，困顿不可复言，京中医并经造问，皆随意处方，无效验，惟此法得效如神，故录之。
各家论述：None
用药禁忌：None
附注：绿矾散（《圣济总录》卷一二九）。
