名称：皂角苦参丸
组成：苦参1斤，荆芥12两，白芷6两，大风子肉6两，防风6两，大皂角5两，川芎5两，当归5两，何首乌（生）5两，大胡麻5两，枸杞子5两，牛蒡子（炒）5两，威灵仙5两，全蝎5两，白附子5两，蒺藜（炒，去刺）5两，独活5两，川牛膝5两，草乌（汤泡，去皮）3两，苍术（米泔水浸，炒）3两，连翘（去心）3两，天麻3两，蔓荆子3两，羌风3两，青风藤3两，甘草3两，杜仲（酥炙）3两，白花蛇（切片，酥油炙黄）2两，缩砂仁（炒）2两，人参1两。
