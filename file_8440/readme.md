序号：8440
名称：四君子汤
组成：人参1两，白术1两，茯苓1两，甘草1两。
出处：《鸡峰》卷十二。
主治：脾肺气虚，中土衰弱，食少便溏，体瘦神倦，或气短息微，皮聚毛落。
加减：None
功效：和胃进食。
用法用量：每服2钱，水1盏，加生姜3片，大枣1枚，同煎至6分，去滓温服，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：《成方便读》：人参大补肺脾元气，为君；白术补脾燥湿，为臣。以脾喜温燥，土旺可以生金，故肺脾两虚者，尤当以补脾为急，脾为后天之源，四脏皆赖其荫庇，不独肺也。而又佐以茯苓，渗肺脾之湿浊下行，然后参、术之功，益彰其效，此亦犹六味丸补泻兼行之意；然必施之以甘草，而能两协其平；引以姜、枣，大和营卫，各呈其妙，是以谓之君子也。
用药禁忌：None
附注：None
