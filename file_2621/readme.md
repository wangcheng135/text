序号：2621
名称：壬字化毒丸
组成：虎胫骨（酥炙）1钱6分，龟版（酥炙）1钱6分，川山甲（炙脆）1钱6分，朱砂1钱6分，月月红（即血余，用童子头发月剃者，煅）1钱5分，蝉蜕末2钱，没药1钱5分，乳香1钱5分，白鲜皮1钱5分，雄黄1钱5分，生生乳1钱，牛黄5分，土贝母2钱，沉香7分（取沉水、色黑、味甜香者），琥珀7分。
出处：《疮疡经验全书》卷十三。
主治：霉疮，见肾经内外前后形症者。
加减：None
功效：None
用法用量：每早空心服15丸，每晚空腹服10丸，人参汤送下；枸杞汤亦可。病去药减。如余邪未尽，药不可撤。
制备方法：上药各为末，用神曲末5钱打稠糊，入药捣匀，丸如梧桐子大，另研朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：切忌烦劳、恼怒、焦躁；茶、酒止可用十分之三。
附注：None
