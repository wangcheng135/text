序号：74123
名称：感冒清热冲剂
组成：荆芥穗200g，薄荷60g，防风100g，柴胡100g，紫苏叶60g，葛根100g，桔梗60g，苦杏仁80g，白芷60g，苦地丁200g，芦根160g。
出处：《中国药典》一部。
主治：风寒感冒，头痛发热，恶寒身痛，鼻流清涕，咳嗽咽干。
加减：None
功效：疏风散寒，解表清热。
用法用量：每次1g，开水冲服，1日2次。
制备方法：上取荆芥穗、薄荷、紫苏叶提取挥发油，蒸馏后的水溶液另器收集；药渣与其余8味加水煎煮2次，每次1．5小时，合并煎液，滤过，滤液与上次水溶液合并，浓缩至相对密度1．35～1．38（50～60℃）的清膏；取清膏1份，蔗糖3份，糊精1份及适量的乙醇制成颗粒，干燥，加入上述荆芥穗等挥发油混匀即得，每袋12g。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
