序号：37680
名称：三合济生丸
组成：川厚朴6两5钱，乌药2两，枳壳3两5钱，羌活4两，广藿香7两，木瓜1两3钱，紫豆蔻2两，茅术3两，半夏4两5钱，苏叶7两，香茹2两，草果2两，赤苓6两，香附3两，桔梗2两5钱，甘草3两，茯苓2两，川芎3两，白术1两5钱，檀香1两，陈皮6两5钱，防风3两，木香3两6钱，柴胡8钱，白芷5两，神曲5两，砂仁3两
出处：《伤科方书》。
主治：四时不正之气，头疼身热，腹痛胀闷，霍乱转筋，呕吐泄泻，四肢厥冷，绞肠痧气，伤寒，伤暑，伤食，疟，痢。
加减：吐泻转筋，用丸4服，加生姜、灶心土煎服。
功效：None
用法用量：每服1钱，重症加倍。舌苔白者，用藿香汤送下；黄者，用荷叶汤送下；寒重，用姜汤送下。
制备方法：上为细末，用薄荷、茶叶、大腹皮熬汁，米汤1碗为丸，朱砂为衣，每丸重7分，晒干收入小口瓷瓶不可泄气。
临床应用：None
各家论述：None
用药禁忌：忌食米粒。
附注：None
