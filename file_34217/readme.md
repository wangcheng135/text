序号：34226
名称：牡蛎丸
组成：牡蛎（煅研）1两，赤石脂（研）1两，栝楼根1两，肉苁蓉（酒浸1宿，切焙）1两，黄连（去须）3分，土瓜根（锉）3分，黄芩（去黑心）3分，知母（焙）3分，泽泻3分，天门冬（去心，焙）3分，鹿茸（去皮毛，酒浸，炙）3分，五味子3分，桑螵蛸（麸炒）3分，熟干地黄（焙）1两半。
出处：《圣济总录》卷五十九。
主治：消中，食已即饥，手足烦热，背膊疼闷，小便稠浊。
加减：None
功效：None
用法用量：每服30丸，煎陈粟米饮送下，日3夜1。
制备方法：上十四味，十二味为末，与别研二味和匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
