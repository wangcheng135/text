序号：46013
名称：洁白丸
组成：诃子（煨）370g，寒水石（平制）210g，翼首草85g，五灵脂膏178g，土木香26g，石榴子26g，木瓜26g，沉香19g，丁香20g，石灰华13g，红花6g，肉豆蔻13g，草豆蔻13g，草果仁13g。
出处：《中国药典》一部。
主治：胸腹胀满，胃脘疼痛，消化不良，呕吐泄泻，小便不利。
加减：None
功效：健脾和胃，止痛止吐，分清泌浊。
用法用量：每服1丸，1日2-3次，嚼碎，吞服。
制备方法：上14味，除五灵脂膏外，余粉碎成细粉，过筛，混匀，加五灵脂膏，炼蜜370g及适量水，泛丸，干燥，打光，即得，每丸重0.8g。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
