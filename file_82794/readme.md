序号：82810
名称：丹参汤
组成：丹参3分，木通（锉，炒）3分，当归（切，焙）3分，生干地黄（焙）3分，麦门冬（去心，焙）3分，禹余粮（烧令赤，醋淬7遍）3分，麻黄（去节煮，去沫，焙）3分，川芎1两，杜仲（去粗皮，涂酥炙）1两，续断（锉）1两，地骨皮1两，牛膝（酒浸，切，焙）1两，桂（去粗皮）半两，甘草（微炙，锉）半两，牡蛎（烧令通赤）1两1分。
出处：《圣济总录》卷九十二。
主治：筋实极，两脚下肿满而痛，不得远行，脚心如割筋断折，痛不可忍者。
加减：None
功效：None
用法用量：每服5钱匕，用水1盏半，加生姜半分(拍碎)，煎至1盏，去滓温服。空心、日晚各1次。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
