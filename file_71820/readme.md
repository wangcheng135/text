序号：71836
名称：散毒膏
组成：木鳖（去壳）20个，蓖麻（去壳）100粒，威灵仙2钱，当归2钱，川芎2钱，赤芍2钱，防风2钱，荆芥2钱，羌活2钱，独活2钱，生地黄2钱，白芷2钱，黄芩2钱，黄连2钱，黄柏2钱，姜黄2钱，蛇蜕1条，金银花、皂角刺、川山甲（切碎）。
出处：《简明医彀》卷八。
主治：一切痈疽发背，诸般肿毒热疖，蟮拱头毒，周身诸毒。
加减：None
功效：去腐消肿，抽脓拔毒。
用法用量：初起贴之即消，有脓者咬头出脓，脓尽亦能收口。
制备方法：上以麻油（冬7两、夏5两、秋6两）浸药1日，煎药焦，滤去滓，油入锅煎滚，下黄蜡2两，次入嫩松香2斤（老松添油），桃、柳枝搅化，滴水成珠，略不粘手，即取起扯黄，每斤用真铜青2两（米醋少许研化），真蟾酥2钱（酒少许研），2味拌扯入膏内。用随大小捏薄，捺布上贴；如小者，油纸摊。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中金银花、皂角刺、川山甲用量原缺。
