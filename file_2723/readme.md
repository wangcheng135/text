序号：2723
名称：天元接髓丹
组成：人参（清河者）2两，黄耆（绵白者）2两，橘红（连本蒂）半两，全蝉蜕半两，当归（去头尾）半两，怀生地（拣粗软者）半两，鹿茸（乳制）1两半，附子（连皮脐）半两，官桂（削外皮，煨时度候，只以米熟验之）半两。
出处：《准绳·幼科》卷五。
主治：男子17-18岁，或20-30岁，破阳亏元，患痘多密连布，欲其鼎峻充灌者。
加减：None
功效：None
用法用量：每钟加鲜人乳3杯，薄生姜2片，煮沸，温服。
制备方法：用澄清好酒浆10碗，放瓮中，慢火煨熟后，去滓出火。
临床应用：None
各家论述：None
用药禁忌：None
附注：《痘疹仁端录》以本方去鹿茸，名“天元接髓散”。
