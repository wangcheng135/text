序号：30443
名称：还魂丸
组成：巴豆（去心皮，熬）2两，甘草（炙）2两，朱砂2两，芍药2两，麦门冬2两（去心）。
出处：《外台》卷一引《古今录验》。
主治：伤寒4-5日，及数年诸癖，结坚心下，饮食不消，目眩，四肢疼，咽喉不利，壮热，脾胃逆满，肠鸣，两胁里急；飞尸鬼注邪气，或为惊恐伤瘦，背痛，手足不仁，口苦舌燥；天行发作有时，风温不能久住，吐恶水。
加减：None
功效：None
用法用量：每服2丸，葱、枣汤送下；小儿2岁以上，每服麻子大2丸，日2次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌海藻、菘菜、野猪肉、芦笋、生血物。
附注：None
