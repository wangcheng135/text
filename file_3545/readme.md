序号：3545
名称：少阳汤
组成：金银花2两，当归1两，川芎3钱，龙胆草3钱，夏枯草3钱（行肝胆经，除内热，散结气），栀子（炒）1钱（去三焦热），白芷1钱（阳明经脉亦与少阳经脉交络，且诸药性不上行，则用此使上行头面而去风热），薄菏1钱（行厥阴少阳，上行清头目风热）。
出处：《医林纂要》卷十。
主治：鬓疽，由胆及三焦之热毒上行者。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：方中金银花、当归、川芎、龙胆草、夏枯草行肝胆经，除内热，散结气；栀子去三焦热；白芷载药上行头面而去风热，薄荷行厥阴、少阳，上行清头目风热。
用药禁忌：None
附注：None
