序号：4227
名称：木香丸
组成：木香1两，犀角（镑屑）3分，生干地黄（焙）3分，萎蕤3分，杜仲（去粗皮，炙，锉）半两，沉香（锉）3分，白术3分，石膏（研碎）1两，当归（切，焙）1两，芎（锉）3分，知母（焙）3分，柴胡（去苗）1两，肉苁蓉（酒浸，去皱皮，焙）1两，槟榔（锉）半两，茴香子（炒）半两，人参（锉）半两，白茯苓（去黑皮）半两，附子（炮裂，去皮脐）1两。
出处：《圣济总录》卷二十九。
主治：伤寒后未平复合阴阳，毒气感动身体，热气冲胸，头重不能举，四肢拘急，小腹(疒丂)痛，或筋脉舒缓，气力疲乏，眠卧着床，不能摇动，甚者手足拳，即死。
加减：None
功效：None
用法用量：每服30丸，空心温酒送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
