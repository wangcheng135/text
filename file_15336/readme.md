名称：地骨皮丸
组成：地骨皮半两，白芷半两，升麻半两，防风（去叉）半两，赤芍药半两，柴胡（去苗）1两，生干地黄（焙）1两半，大黄（锉，炒）半两，黄芩（去黑心）半两，枳壳（去瓤，麸炒）半两，芎窮半两，知母（焙）半两，萎蕤半两，槟榔（锉）半两，细辛（去苗叶）半两，甘菊花半两，藁本（去苗土）半两，牵牛子（炒）半两，马牙消（研）半两，犀角屑半两，胡黄连1两，甘草（炙）1两。
出处：《圣济总录》卷一二○。
主治：风袭齿龈，肿痛有血。

制备方法：上为末，炼蜜为丸，如梧桐子大。
