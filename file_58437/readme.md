序号：58452
名称：破棺丹
组成：硫黄1两（无灰酒煮3日3夜，如耗，旋添暖酒，日足取出，研为末），丹砂1两（研匀细）。
出处：《中藏经》卷下。
主治：阴厥。面目俱青，心下硬，四肢冷，脉细欲绝者。
加减：None
功效：None
用法用量：有此病者，先于净室中，勿令人知，度病人长短，掘一地坑子，深1尺以来；用苜蓿火烧，令坑子极热，以醋5升沃，令气出，内铺衣被盖坑，以酒化下1丸，与病人服之。后令病人卧坑内，盖覆，少时汗出，即扶病者，令出无风处盖覆。令病人四肢温，心下软，即渐去衣被，令通风。然后看虚实调补。
制备方法：上以酒煮糊为丸，如鸡头大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
