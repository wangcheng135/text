序号：26639
名称：庆世丹
组成：何首乌（用赤白者）4两，生地黄2两，熟地黄2两，菊花（园中甘者）2两，车前子2两，地骨皮2两（去粗皮，用近骨者），茯神（白者）2两，远志2两（用柑子水煮，去心），石菖蒲2两（9节者，米泔水浸1宿），川牛膝2两（酒浸），肉苁蓉2两（酒洗，去鳞），山药2两，巴戟天2两（泡炒，去心），甘州枸杞子2两（酒浸）或加柏子仁2两、酸枣仁2两。
出处：《摄生众妙方》卷二。
主治：None
加减：None
功效：补养。
用法用量：每服100丸，清晨滚白汤送下，淡盐酒任下。
制备方法：上为末，春、夏用酒糊为丸，秋，冬炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
