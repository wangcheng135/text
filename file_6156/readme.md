序号：6156
名称：加味凉膈散
组成：白僵蚕（酒炒）3钱，蝉退（全）12枚，广姜黄7分，黄连2钱，黄芩2钱，栀子2钱，连翘（去心）3钱，薄荷3钱，大黄3钱，芒消3钱，甘草1钱，竹叶30片。
出处：《寒温条辨》卷四。
主治：大头、瓜瓤等温病危在旦夕。温证羊毛，火郁于上，壮热面赤，唇燥舌干，烦燥谵言，胸闷气滞，脉象数实。
加减：若欲下之，量加消、黄；胸中热，加麦冬；心下痞，加枳实；呕、渴，加石膏；小便赤数，加滑石；满，加枳实、厚朴。
功效：None
用法用量：水煎去滓，冲芒消，入蜜、酒冷服。
制备方法：None
临床应用：None
各家论述：连翘、荷、竹，味薄而升浮，泻火于上；芩、连、栀、姜，味苦而无气，泻火于中；大黄、芒消味厚而咸寒，泻火于下；僵蚕、蝉退以清化之品，除疵疾之气，以解温毒；用甘草者，取其性缓而和中也；加蜜、酒者，取其引上而导下也。
用药禁忌：None
附注：None
