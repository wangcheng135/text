序号：46214
名称：洗髓丹
组成：净轻粉2钱（炒至光色减去3分之2，研细。盖此药炒之则烈性少缓，若炒之过度，又恐无力，火候宜中，用其大片即净轻粉），净红粉1钱（研细，须多带紫黑片者用之，方有效验），露蜂房如拳大者1个（大者可用一半，小者可用2个。炮至半黑半黄色，研细，炮时须用物按之着锅），核桃10个（去皮，捣碎，炮至半黑半黄色，研细，纸包数层，压去其油，盖油多即不好为丸用）。
出处：《衷中参西》上册。
主治：杨梅疮毒蔓延周身，或上至顶，或下至足，或深入骨髓，无论陈、新、轻、剧，服之皆有奇效。
加减：None
功效：None
用法用量：服时，须清晨空心开水送下，至午后方可饮食。服后，口含柳棍，有痰涎即吐出，愈多吐愈好。睡时将柳棍横含，两端各系1绳，两绳之端结于脑后，防睡觉掉落，又须将柳棍勤换，即将药服完仍须如此，必待不吐痰涎时方可不含柳棍。其药日服1次，若恶心太甚者，可间日1服。制此药时，须自经手，将轻粉、红粉称极准，其称当以库称为定法，轻粉须称准后再炒。
制备方法：用熟枣为丸，如黄豆粒大，晒干，分3次服之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
