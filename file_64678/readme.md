序号：64694
名称：硇砂丸
组成：木香1两，沉香1两，巴豆肉（全者）1两，青皮2两（不去皮），铜青半两（研），硇砂1分（研）。
出处：《本事》卷三。
主治：膀胱疝气，外肾肿胀，痛不可忍。
加减：None
功效：None
用法用量：每服7-10丸，空心、食前盐汤吞下，1日2-3服。
制备方法：上二香，青皮三味细锉，同巴豆慢火炒令紫色为度，去巴豆为末，入青、砂二味研匀，蒸饼为丸，如梧桐子大。
临床应用：None
各家论述：《本事方释义》：木香气味辛温，入足太阴；沉香气味苦辛温，入足少阴；巴豆肉气味辛温，入足太阴、阳明；青皮气味苦辛酸，微温，入足厥阴；铜青气味酸平，入足少阳厥阴，能杀疳虫；硇砂气味咸若微温，入足太阳阳明厥阴。蒸饼和丸，盐汤送药，不欲药性之发于上也。
用药禁忌：None
附注：None
