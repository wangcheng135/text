序号：80057
名称：摩风膏
组成：防风（去叉）3分，羌活（去芦头）3分，芎3分，细辛（去苗叶）3分，蜀椒（去目并合口者，炒出汗）3分，当归（切，焙）3分，踯躅花3分，白蔹1分，白及1分，丹参1分，苦参1分，黑参1分，桂（去粗皮）1分，附子（去皮脐）1分，乌头（去皮脐）1分，皂荚（去黑皮）1分，莽草1分，杏仁（去皮尖并双仁）半两。
出处：《圣济总录》卷十八。
主治：风疾愈后，肌肉(疒帬)痹，遍体疮癣，或瘾疹瘙痒，及风湿着痹，一切风毒。
加减：None
功效：None
用法用量：每用少许，摩(疒帬)痹处。
制备方法：上锉细，如麻豆大。以米醋1升拌匀，浸3宿，熬干，用腊月猪脂2斤，再以文武火煎1日，以绵滤去滓，于瓷瓶内贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
