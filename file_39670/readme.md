序号：39680
名称：细辛散
组成：红椒（去目，炒）半两，鹤虱半两，牙皂半两，荜茇半两，缩砂（去壳）半两，荆芥（去梗）1两，细辛（去苗）1两，白芷2两，草乌2两。
出处：《局方》卷七（续添诸局经验秘方）。
主治：风蚛牙痛，或牙龈宣烂，牙齿动摇，腮颔浮肿。虫牙蛀，脉滑者。
加减：None
功效：疏邪杀虫。
用法用量：每用少许，于痛处擦之。有涎吐出，不得咽。少时，温水漱口，频频擦之。
制备方法：上为细末。
临床应用：None
各家论述：《医略六书》：牙蛀，本厥阴风化之气生虫蚀牙，而齿不坚牢，故齿骨损坏焉。荆芥疏血中之风，细辛散少阴之风，白芷散风燥湿，川椒温中杀虫，牙皂通窍杀虫，鹤虱祛湿杀虫，荜茇暖胃逐虫，砂仁理气开胃也。为散擦牙，使风虫消化，则齿日肃清，而齿骨坚固。
用药禁忌：None
附注：None
