序号：24629
名称：竹皮汤
组成：竹皮（一方用竹叶）2两，细辛2两，甘草1两，生姜1两，通草1两，人参1两，茯苓1两，麻黄1两，桂心1两，五味子1两。
出处：《千金》卷十六。
主治：噎，声不出。
加减：None
功效：None
用法用量：以水1斗，煮竹皮，减2升，去竹皮下药，煮取3升，分3次服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：噎本胃病，加以暴寒折之，所以声瘖不出，但欲彻外邪先固正气，麻黄、细辛专散暴邪，生姜、通草专通声气，人参、茯苓专培本元，竹皮清肺胃之虚火，桂心助参、苓之温补，五味收麻黄之开发，甘草和敛散之味也。
用药禁忌：None
附注：竹皮散（《奇效良方》卷十六）。
