序号：51373
名称：贴药乌龙角
组成：白僵蚕（炒）6两，赤小豆6两，川牛膝6两，山桂（去皮）1斤，桔梗1斤，百草霜1斤，白及（生，锉，阴干）1斤，枇杷叶1斤，骨碎补（去毛，炒）半斤，当归尾半斤，北细辛半斤，白芷10两，南星（煨）10两，赤芍药10两，何首乌10两，白蔹10两，知母3两，草乌（姜汁煮）3两。
出处：《医统》卷七十九。
主治：跌打损伤，筋骨寸断，差爻出臼。
加减：None
功效：None
用法用量：于患处贴之，2日1洗1换。若骨碎须用夹法。
制备方法：上为细末，每用以姜汁、冷水、冷茶调摊纸上。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中当归尾用量原缺。
