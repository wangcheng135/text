序号：57601
名称：大海藻汤
组成：海藻（洗去咸，焙）1两，芫花（炒焦）1两，猪苓（去黑皮）1两，连翘1两，泽漆（炒）1两，郁李仁（去皮尖双仁，研）1两，陈橘皮（汤浸，去白，焙）1两，桑根白皮（锉，炒）1两，白蒺藜（炒）1两，藁本（去苗土）半两，昆布（洗去咸，焙）半两，大戟（炒）半两，防己半两，葶苈（炒）半两，朴消半两，甘遂（炒）半两，杏仁（去皮尖双仁，炒）半两，槟榔（煨，锉）7枚。
出处：《圣济总录》卷七十九。
主治：十种水病。
加减：None
功效：None
用法用量：每服4钱匕，水1盏半，加生姜1枣大（拍碎），同煎至8分，去滓，空心温服，1日3次。以利为度。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
