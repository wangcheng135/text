序号：22113
名称：一号癣药水
组成：土槿皮10两，大风子肉10两，地肤子10两，蛇床子10两，硫黄5两，白鲜皮10两，枯矾2斤半，苦参10两，樟脑5两。
出处：《中医外科临床手册》。
主治：鹅掌风，脚湿气，圆癣等。
加减：None
功效：杀虫止痒。
用法用量：搽擦患处，每日3至4次。
制备方法：将土槿皮打成粗末，大风子肉捣碎，硫黄研细，枯矾打松，用50％酒精温浸。第1次加8升，浸2天后，倾取清液；第2次再加6升，再浸2天，倾取清液；第3次加6升，去滓取液。将3次浸出之药液混和。再以樟脑用50％酒精溶解后，加入药液中，俟药液澄清，倾取上层清液备用。
临床应用：None
各家论述：None
用药禁忌：有糜烂者禁用。
附注：None
