序号：35252
名称：乳香善应膏
组成：乳香5钱，没药5钱，血竭5钱，阿魏2钱，麝香1钱（另研），大黄1两，黄连1两，黄柏1两，防风1两，荆芥1两，芍药1两，白芷1两，玄参1两，当归1两，连翘1两，巴豆1两，苏木1两，大风子1两，木鳖子8个，川山甲8片，黄丹1斤（水飞过），槐嫩枝20寸，桃嫩枝20寸，柳嫩枝20寸，清油2斤。
出处：《丹溪心法附余》卷十六。
主治：痈疽发背，诸般恶疮，打扑伤损，筋骨疼痛。瘀血，年久烂疮，顽疮，小儿癖块。
加减：None
功效：None
用法用量：摊贴患处。
制备方法：上除乳香等5味另研为末，将其余药锉碎，入清油内煎令黑色，滤去滓，入黄丹不住手搅成膏，却入前5味药末，再搅令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：《便览》有官挂一两，无荆芥。
