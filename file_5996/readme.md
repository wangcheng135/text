序号：5996
名称：加减四物汤
组成：当归（切，焙）1两，川芎1两，熟干地黄1两，白芍药1两。
出处：《活人书》卷十九。
主治：妊妇产前腹痛，及月事或多或少，或前或后，胎气不安，产后血块不散，或亡血过多，或恶露不下。
加减：若妊妇下血，即入艾5-7叶，阿胶末1钱匕；同煎，服如前法，如因虚致热，热与血搏，口舌干渴，欲饮水，加栝楼1两，麦门冬3分；腹中刺痛，恶物不下，加当归、芍药各1分；血崩，加地黄、蒲黄各1两；因热生风，加川芎1分，柴胡半两；身热脉躁，头昏项强，加柴胡、黄芩各半两；秘涩，加大黄半两，桃仁1分（炒）；滑泻，加桂、附各1分；发寒热，加干姜、牡丹皮、芍药各1分；呕，加白术、人参各半两；腹胀，加厚朴、枳实各1分；虚烦不得眠，加竹叶、人参各1分；躁，大渴，加知母、石膏各半两；水停心下，微吐逆，加猪芩、茯苓、防己各1分；虚寒状类伤寒、加人参、柴胡、防风各3分。
功效：None
用法用量：每服4钱，水1盏半，煎至8分，取6分清汁，带热服，每日2-3次，以知为度；疾势甚大，散药不知，以4味各半两，细锉，以水4盏，煎至2盏半，去滓，分为4服，热吃，食前服，1日之中令尽，以和为度。平常产乳服至3腊止，如虚弱血脏不调，至1月止。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：增损四物汤（《朱氏集验方》卷十）。
