序号：41890
名称：青木香丸
组成：补骨脂（炒香）40两，荜澄茄40两，槟榔（酸粟米饭裹，湿纸包，火中煨令纸焦，去饭）40两，黑牵牛（240两，炒香，别捣末）120两，木香20两。
出处：《局方》卷三。
主治：胸腹痞满胀痛，呕逆不食，膀胱疝气，湿着腰痛。胸膈噎塞，腹胁胀痛，心下坚痞，肠中水声，呕哕痰逆，不思饮食。浴出身上未干，忽尔熟睡，致肾经肿痛，腰背挛曲。
加减：None
功效：宽中利膈，行滞气，消饮食。
用法用量：每服20丸，食后茶、汤、熟水任下。酒食后可每服5-7丸，小儿1岁服1丸。
制备方法：上为细末，入牵牛末令匀，渐入清水和令得所，丸如绿豆大。
临床应用：None
各家论述：None
用药禁忌：怀妊妇人不得服之。
附注：①方中黑牵牛，《医统》作黑豆。②本方《直指》治膀胱疝气用法：每服五十丸，以醇酒入葱白，煎五苓散送下。
