序号：53036
名称：骨填煎
组成：茯苓3两，菟丝子3两，山茱萸3两，当归3两，牛膝3两，附子3两，五味子3两，巴戟天3两，麦门冬3两，石膏3两，石韦4两，人参4两，桂心4两，苁蓉4两，大豆卷1升，天门冬5两。
出处：《千金》卷二十一。
主治：虚劳消渴。
加减：None
功效：None
用法用量：食如鸡子黄大，每日3次，亦可饮服之。
制备方法：上为末，次取生地黄、栝楼根各10斤，取汁，于微火上煎之，减半，便作数分，纳药，并下白蜜2斤，牛髓半斤，微火煎之。令如糜。
临床应用：None
各家论述：None
用药禁忌：忌酢物、鲤鱼、生葱、猪肉、冷水。
附注：本方方名，《外台》引作“填骨煎”，有远志，无苁蓉。
