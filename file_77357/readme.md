序号：77373
名称：遇仙方
组成：附子（炮，去皮脐）1两，川乌（炮，去皮脐）1两，当归（酒浸，焙）1两，川芎1两，羌活1两，肉苁蓉（酒浸，炮）1两，杜仲（去皮，炒去丝，姜汁制）1两，黄耆1两，白蒺藜（炒去刺）1两，白术1两，人参1两，川牛膝（酒浸，焙，去芦）1两，防风1两，天麻（去苗）1两，白茯苓1两，萆薢1两，狗脊（炒去毛）1两，续断1两，独活1两，肉桂（去粗皮）1两，赤芍1两，虎胫骨2两半（酥炙）。
出处：《医学纲目》卷十二。
主治：诸般痛风，手足艰难，筋骨疼痛，口眼斜，言语謇涩。
加减：None
功效：None
用法用量：每服50丸，空心用浸药酒1盏送下。
制备方法：上切细，以生绢袋盛之，用无灰酒浸，密封瓶口，春3日，夏2日，秋7日，冬10日，取出晒，焙干为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻、豆腐、面食、发风之物。
附注：None
