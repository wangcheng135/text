序号：80230
名称：鳖甲丸
组成：鳖甲（去裙襕，醋炙）1两半，防葵（锉）1两，人参1两，前胡（去芦头）1两，桔梗（炒）1两，枳壳（去瓤，麸炒）1两，当归（切，焙）1两，附子（炮裂，去皮脐）1两，干姜（炮）1两，白术1两，槟榔（锉）2两，大黄（锉，炒）2两，厚朴（去粗皮，生姜汁炙）3两，食茱萸3两，甘草（炙，锉）1两1分。
出处：《圣济总录》卷七十一。
主治：腹内积聚。心肋急满，时吐清水，不能食，时恶寒。
加减：None
功效：None
用法用量：每服20丸，渐加至30丸，温酒送下，早、晚各1次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
