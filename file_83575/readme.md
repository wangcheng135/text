序号：83593
名称：乌须固本丸
组成：何首乌半斤（米泔水浸2宿，竹刀刮去粗皮，切片，以黑豆5升，滚水泡1时，蒸熟去豆），黄精4两（用黑豆2升，同煮熟，去豆，忌铁器），生地黄2两（酒浸），熟地黄2两（酒浸），天门冬2两（去心），麦门冬2两（去心），赤茯苓2两，片术2两，人参2两，五加皮2两，巨胜子2两，松子仁2两，柏子仁2两，核桃仁2两，西枸杞2两。
出处：《摄生众妙方》卷二。
主治：None
加减：None
功效：乌须。
用法用量：每服70-80丸，加至100丸，空心温酒送下；盐汤亦可。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
