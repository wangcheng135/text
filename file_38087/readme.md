序号：38097
名称：服蛮煎
组成：生地2钱，麦门冬2钱，芍药2钱，石菖蒲2钱，石斛2钱，川丹皮（极香者）2钱，茯神2钱，陈皮1钱，木通1钱半，知母1钱半。
出处：《景岳全书》卷五十一。
主治：水不制火兼心肾微虚而狂。郁结不遂，疑虑惊恐，而致痴呆，言语颠倒，举动失常。
加减：如痰胜多郁者，加贝母2钱；痰盛兼火者，加胆星1钱5分；阳明火盛，内热狂叫者，加石膏2-3钱；便结胀满多热者，玄明粉2-3钱调服，或暂加大黄亦可；气虚神困者，加人参随宜。
功效：行滞气，开郁结，通神明，养正除邪。
用法用量：水1钟半，煎7分。食远服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
