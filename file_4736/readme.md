序号：4736
名称：木香散
组成：木香1两，桂心1两，人参1两（去芦头），细辛半两，诃黎勒皮半两，干姜半两（炮裂，锉），白术半两，甘草1两（炙微赤，锉），附子半两（炮裂，去皮脐），鳖甲1两半（涂醋，炙微黄，去裙襴），吴茱萸半两（汤浸7遍，焙干，微炒），青橘皮半两（汤浸，去白瓤，焙），京三棱3分，槟榔半两，赤茯苓3分，厚朴半两（去粗皮，涂生姜汁，炙令香熟），当归3分，茴香子半两。
出处：《圣惠》卷四十二。
主治：七气，心腹积聚，结块如杯，呕吐寒热，胸心中短气，不能下食。
加减：None
功效：None
用法用量：每服5钱，以水1中盏，加生姜半分，大枣3枚，煎至6分，去滓，每于食前稍热服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
