序号：37520
名称：抱龙丸
组成：天竺黄1两，胆南星5钱，人参2钱，辰砂3钱（水飞7次），雄黄3钱，珍珠3钱，琥珀3钱，檀香2钱，木香1钱，沉香1钱，麝香2钱5分，金箔20叶。
出处：《治痘全书》卷十三。
主治：小儿惊风潮搐，四时瘟疫，身热昏睡，痰涎壅盛，风热喘嗽，烦躁不宁，并痘疹欲出先惊搐，及蛊毒、中暑。
加减：None
功效：None
用法用量：每服1丸，婴儿半丸。
制备方法：上为末，甘草汁为丸，如大豆大。
临床应用：None
各家论述：《医方考》：明者可以安神，故用琥珀、珍珠；重者可以去怯，故用辰砂、金箔；气窜可以利窍，故用雄黄、沉、檀、木、麝；甘温可以固元，故用人参；辛燥可使开痰，故用南星；寒凉可使清热，故用竺黄。
用药禁忌：None
附注：None
