序号：35416
名称：万痊膏
组成：乳香4钱半，没药4钱半（另研），半夏半两，当归半两，续断半两，杏仁半两，桃仁半两，巴豆半两（和皮捶碎），木鳖子半两（去壳），芫花半两，大戟半两，川芎半两，熟地黄半两，芍药半两，苍术半两，防风半两，干姜半两（生用），桂半两，蛇床子半两，松枝2两，桃枝（新者）2两，乱发2块（如马打毬子大），澄清芝麻油10斤。
出处：《御药院方》卷十。
主治：一切疮疡，己溃未溃，脓水不绝，及灸疮久不愈。
加减：None
功效：敛疮生肌。
用法用量：每用粘在铁銲上，炭火炙消摊纸上，贴患处，1-2日1换。
制备方法：将前项药下在油内浸7日，慢火煎熬，令铁马杓搅至半夏黄黑色为度，用竹筛滤去滓，另研血竭3钱半，下在油内搅令匀，用新绵滤在盆器中，澄去滓油，揩锅并马杓至净，再用绵滤入锅内，入油每1斤用上好黄丹5两，若黄丹性紧者，只用四两半，准备冬春秋间使用，如夏月用者，使黄丹5两2-3钱，并看丹急慢调品，用慢柴火烧熬，不往手和，令候变黑色微溢住火，至沫下，依前用慢火熬，候黑烟出住火，如此2日后，用木炭火养，仍不得暂往手搅，直至通前4-5日以来，摊纸上不溢，硬软得所，盛在瓷器内，方欲凝时，用绢子裹水银搭在膏药面上，如用时揩去水银，如此不至膏药上面重干了，腊月内熬者佳。
临床应用：None
