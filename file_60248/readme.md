序号：60263
名称：遂心正气丸
组成：当归30两（酒炒），白芍30两（酒炒），柴胡15两（炒），薄荷8两（炒），紫苏30两（微炒），枳壳20两（麸炒），厚朴20两（姜汁炒），青皮20两（炒），草蔻仁30两（炒），制半夏20两（姜汁炒），山楂30两（炒焦），甘草5两（生研），藿香30两（炒），神曲20两（炒），麦芽30两（炒），槟榔20两（炒），山栀仁20两（姜汁炒），桔梗15两（炒），青蒿30两（炒），黄芩8两（酒炒），钩藤20两（炒），制军15两（酒炒），天竺黄8两（炙），香附30两（盐水炒）。
出处：《慈航集》卷上。
主治：夏秋感受寒暑伤风，头痛恶寒发烧，遍身骨节疼痛，霍乱吐泻，瘟疫疟痢，时毒斑疹，及四时不正之气。
加减：None
功效：None
用法用量：照后汤头，送服1丸。凡感冒伤风，头痛发烧，遍身骨节疼痛，用煨姜2钱煎汤化服1丸，出汗即愈；凡霍乱吐泻，用煨姜2钱，灶心土3钱，煎汤化服1丸；凡水泻，用川连3分（姜汁炒），煎汤化服1丸；凡疟疾，用煨姜3钱，黑枣3个，于疟未来早一时煎汤化服1丸；凡红痢，用金银花3钱，川连3分，煎汤化服1丸；凡白痢，用煨姜2钱，大枣3个，煎汤化服1丸；凡瘟疫，用赤饭豆5钱，薄荷3分，煎汤化服1丸；凡斑疹，用老鼠粪1钱，泡汤化服1丸；凡孕妇，用砂仁1钱5分，炒黄芩3分，煎汤化服1丸；凡产后，用当归8钱，益母草3钱，煎汤化服1丸。大人1丸，小儿半丸。
制备方法：上药如法炮制，各研净细末，炼蜜为丸，每颗重4钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
