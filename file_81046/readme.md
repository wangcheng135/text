序号：81062
名称：麝香膏
组成：天竺黄1分，白附子（微炮）1分，郁金1分，人参（去芦头）1分，真珠末1钱，犀角屑1钱，牛黄（别研）半分，朱砂（别研）半分，麝香（别研）1钱，龙脑（别研）1钱，金箔10片，银箔10片。
出处：《杨氏家藏方》卷十七。
主治：小儿壮热，涎盛，恍惚不安，夜卧狂语，咬牙弄舌、急惊潮发，目睛直视，牙关紧急，手足惊掣，项背强直，精神昏乱。
加减：None
功效：None
用法用量：每服1丸，食后、临卧煎薄荷汤化下。
制备方法：上为细末，次入朱砂、脑、麝香、金银箔同研令匀，炼蜜为丸，每1两作40丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：麝犀膏（《普济方》卷三七○）。
