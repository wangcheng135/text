序号：30212
名称：豆蒸丸
组成：原蚕蛾28枚（全者，炒），羊靥28枚（劈开，炙干），猪靥28枚（如前炙），安息香1两，犀角屑1两，没药（研）1两，人参1两，甘草（炙，锉）1两，何首乌1两，大黄（锉，炒）1两，远志（去心）1两，角蒿1两半，蒲公草1两半，苍耳1两半，地锦1两半，益母草1两半，土马騣1两半，杞子1两半（角蒿以下7味治净，用黑豆盖覆，甑中蒸令黑豆软熟，不用黑豆，将药晒干）。
出处：《圣济总录》卷十八。
主治：恶风。
加减：None
功效：None
用法用量：每服1丸，米饮嚼下，不拘时候。
制备方法：上为末，入鸭梨汁，并炼蜜同和为丸，如樱桃大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
