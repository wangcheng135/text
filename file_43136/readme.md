序号：43151
名称：三棱丸
组成：京三棱（炮，锉）半两，芍药半两，桔梗（炒）半两，干姜（炮）半两，槟榔（生，锉）半两，吴茱萸（汤浸，焙干，炒）半两，乌头（炮裂，去皮脐）半两，大黄（煨，锉）1两，诃黎勒（煨，去核）1两，鳖甲（去裙襕，醋炙）1两半，桃仁（汤浸，去皮尖双仁，麸炒，研）3分。
出处：《圣济总录》卷七十三。
主治：食气癖块，胸膈噎塞，冷气攻刺，吐酸水，不美饮食，腹胁胀痛，气不升降。
加减：None
功效：None
用法用量：每服15丸，炒橘皮、盐汤送下；如酒食所伤，胸膈不快，腹胀醋心，熟水送下。
制备方法：上为末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
