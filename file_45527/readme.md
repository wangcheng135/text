序号：45542
名称：枸杞酒
组成：米1石（黍糯并得酿酒米，用上好曲末1斗，加5升弥佳），枸杞30斤（去赤皮，半寸锉之，以水1石，浸之3日，煮取5斗汁），生地黄20斤（洗去土，细切，共米同炊之），秋麻子3斗（微熬细粉，蒸气出，以枸杞汤淋取计），豆豉2斗（以枸杞汤煮取汁）。
出处：《外台》卷十七。
主治：五内邪气，消渴，风湿，胸胁间气，头痛，五劳七伤，胃中宿食，鼻衄吐血，内湿风疰，恶血石淋，伤寒瘴气，烦躁满闷，虚劳喘息及脚气肿痹。
加减：None
功效：补中逐水，破积去瘀，逐热破血，利耳目，长发，坚筋骨，强阴，利大小肠，填骨髓，长肌肉，破除结气。
用法用量：初1度酿，用麻子2斗，多即恐令人头痛。日可饮3杯。
制备方法：上药地黄共米同蒸，余三物药汁，总合得5斗，分半渍米，（饣贲）半及曲和酿饭，如人肌温，总和一殴，盖瓮口，经14日，压取封泥，复经7日。
临床应用：None
各家论述：None
用药禁忌：慎芜荑、生冷、陈宿猪、犬、鸡、鱼、面、蒜、油腻、白酒、房室等。
附注：方中用法项用量原缺，据《圣惠》补。
