序号：30350
名称：还元丹
组成：紫河车1具（即产妇胞衣，用米泔洗涤污浊，以新瓦2片合定，上下用文武火炕干，为末听用。妇性嫌恶，勿与知之，密将药于别室中修合，与服亦不要言，致生疑惑），人参（上揀）1两半，黄耆（蜜炙）1两，当归（酒洗）1两，白术（土炒）8钱，芍药8钱，川芎8钱，熟地黄8钱，白茯苓8钱，牡丹皮8钱，肉桂炙5钱，甘草5钱。
出处：《医统》卷八十五。
主治：产后大虚，及一切虚劳。
加减：None
功效：补气补血，还元返本。
用法用量：每服60-70丸，酒或汤任下。未产之先备制各药，为未，候产后便合为妙。
制备方法：上将热地黄另捣外，余药为细末，和捣为丸，如梧桐子大；不成，加老米烂饭捣之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
