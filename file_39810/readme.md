序号：39820
名称：三奇散
组成：枳壳、黄耆、防风各等分。
出处：《普济方》卷二一三。
主治：痢后里急后重。
加减：None
功效：None
用法用量：上为末。每服2钱，蜜汤调下；米饮亦得。
制备方法：None
临床应用：肝郁胁痛：刘氏妇，青年寡居多郁，素有肝气不调之患，今秋将半，大便下坠，欲解不出，医用疏导之药，并进大黄丸，重闭愈增。两胁满痛，诊脉浮大而缓，饮食不进，四肢微热，小水甚利，月经不行，据此谛审，不得其法。细思独阴无阳之妇，值此天令下降之时，而患下坠之症，脉来浮大且缓，系中气久伤，继受风邪入脏无疑，两胁满痛，肝气郁而不舒，惟有升阳一着，四肢独热，亦风淫末疾之义，月经不行，乃风居血海之故，执此阳气下陷，用三奇散加升麻以提阳气，夏入当归，少佐桃仁以润阴血，果然应手而痊。
