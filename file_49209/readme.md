序号：49224
名称：胜骏丸
组成：附子1枚（炮，去皮脐），当归（酒浸1宿）2两，天麻（酒浸）2两，牛膝（酒浸）2两，酸枣仁（炒）2两，熟地黄（酒浸）2两，防风（去叉）2两，木瓜4两，乳香半两（别研），麝香1分（别研），全蝎（去毒）1两，木香1两，没药（别研）1两，羌活1两，甘草（炙）1两。
出处：《三因》卷三。
主治：元气不足，真气虚弱，及诸虚寒湿气进袭，手足拳挛，脚趾连脚面拘急，走注疼痛，筋脉不伸，行步不随，一切足弱鹤膝诸风。
加减：槟榔、萆薢、肉苁蓉（酒浸）、破故纸（炒）、巴戟（去木）各添1两，内熟地黄、当归各减1两尤妙。
功效：益真气，壮筋骨，黑髭须，滑皮肤。
用法用量：每服1丸，细嚼，临睡酒送下。地黄春、夏极多，遇冬或无地黄，只炼蜜为丸，如梧桐子大。每服50丸，盐汤温酒送下。
制备方法：上为细末，用生地黄3斤净洗，研烂如泥，入无灰酒4升，煮烂如膏，以前药匀和，杵令坚，每两作10丸。
临床应用：None
各家论述：如服此药五七日或半月，见效甚速，行步如飞，千里可至，乃名胜骏。
用药禁忌：None
附注：《外科正宗》有苍术。
