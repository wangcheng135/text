序号：9942
名称：止嗽太和丸
组成：胆星1两，茯苓（去皮）1两，厚朴（姜制）1两，前胡6钱，生桑皮6钱，生紫菀6钱，桔梗6钱，橘红6钱，制半夏6钱，黄芩6钱，薄荷叶4钱，川贝4钱，炒苏子4钱，麻黄4钱，生石膏4钱，香橼4钱，甘草4钱，熟军4钱，陈皮1两，杏仁（去皮，炒）4钱。
出处：《全国中药成药处方集》(天津方)。
主治：小儿感冒咳嗽，内热发烧，痰涎壅盛，喘嗽气促，呕吐恶心，烦躁不宁。
加减：None
功效：清热化痰止嗽。
用法用量：每服1丸，周岁以内酌减，白开水化服。
制备方法：上为细末，炼蜜为丸，1钱重，每斤丸药用朱砂面3钱上衣，蜡皮或蜡纸筒封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
