序号：11564
名称：牛膝汤
组成：牛膝（酒浸，切，焙）1两，柴胡（去苗）1两，荆芥穗1两，桔梗（炒）1两，陈橘皮（汤浸，去白，焙）1两，青橘皮（汤浸，去白，焙）1两，人参半两，肉苁蓉（酒浸，切，焙）半两，白茯苓（去黑皮）半两，秦艽（去苗土）半两，知母（焙）半两，乌梅15枚（去核），甘草（炙）3分，乌头（麸炒裂，安地上，用盏盖，出火毒，去皮脐尖）3枚，黄连（去须）3分，败龟（以醋1碗涂浸，炙用）1两。
出处：《圣济总录》卷九十。
主治：虚劳邪气攻击，心腹刺痛。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，加肥枣1枚(擘)，煎至6分，去滓温服。
制备方法：上(口父)咀，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
