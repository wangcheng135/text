序号：57378
名称：消癌片
组成：红升丹300g，琥珀300g，山药300g，白及300g，三七620g，牛黄180g，黄连150g，黄芩150g，黄柏150g，陈皮60g，贝母60g，郁金蕲蛇60g，犀角90g，桑椹90g，金银花90g，黄耆90g，甘草90g。
出处：《肿瘤的诊断与防治》。
主治：舌癌、鼻咽癌、脑癌、食道癌、胃癌、骨肉瘤、乳腺癌、宫颈癌等。
加减：如气虚，加用四君子汤；血虚，加用四物汤；气血俱虚者，2方合用。
功效：活血凉血，解毒消癌。
用法用量：每服1片，1日2-3次，饭后服。1个月为1疗程，4-6月为1治疗期，每疗程后停药1周左右。
制备方法：制成片剂，每片0.5g。
临床应用：None
各家论述：None
用药禁忌：服药期间，忌食蒜、葱、浓茶、鲤鱼等。
附注：None
