序号：73995
名称：煤红膏
组成：京红粉30g，利马锥4g（炼丹后下剩锅底的药粉），白蜡4g，凡士林加到100g，煤焦油若干。
出处：《中西医结合皮肤病学》。
主治：银屑病，湿疹，神经性皮炎。
加减：None
功效：None
用法用量：用时先洗浴，除去鳞屑，然后外搽煤红膏，扑以滑石粉，每1-2日换1次药，由低浓度煤红膏开始，根据病损消退情况，采用较高浓度煤红膏。
制备方法：先将前4味调制成京红粉膏，后以京红粉膏和煤焦油按不同比例调制1.10％（90g比10g）、2.15％（85g比15g）和3.20％（80g比20g）的煤红膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：上述煤红膏的汞含量较高，可有汞的中毒反应，所以近年来采用5％京红粉与10％煤焦油配成软膏治疗，也有较好的疗效。
