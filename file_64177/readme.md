序号：64193
名称：猪苓丸
组成：半夏1两（破如豆大），木猪苓4两。
出处：《本事》卷三。
主治：湿热内蕴，遗精滑浊，尿频涩痛。梦遗。年壮气盛，情欲动心，所愿不得，意淫于外，梦遗白浊。湿郁热滞精滑。小水频数。痰饮迷心。便浊涩痛。
加减：None
功效：开郁滞。
用法用量：每服30-40丸，空心温酒盐汤送下；常服．于申、未间．冷酒送下。
制备方法：先将一半猪苓炒半夏黄色，不令焦，地上出火毒半日，取半夏为末，糊为丸，如梧桐子大，候干，再用上猪苓末2两，炒微裂，同用不泄砂瓶养之。
临床应用：梦遗：一中年梦遗，医或与涩药反甚，连遗数夜。愚先与神芎丸大下之，却再制以猪苓丸服之，皆得安全。
各家论述：1.《本事》：半夏有利性，而猪苓导水，盖导肾气使通之意。_x000D_
2.《本事方释义》：木猪苓气味苦微寒，入足太阳；半夏气味辛温，入足阳明；送药以酒盐汤者，欲药性之下行也。
用药禁忌：None
附注：固真丹（《济生》卷四引袁氏方）、半夏丸（《丹溪心法》卷三）、半苓丸（《东医宝鉴·内景篇》卷一）。
