序号：64754
名称：硇砂煎
组成：硇砂2两（白色不夹石者，研），阿魏1分，京三棱1两，诃黎勒1两（炮），丁香半两，人参半两，附子半两（炮），青橘皮半两（去瓤），木香半两，舶上茴香半两，槟榔1两半，神曲1两半（别研为末）。
出处：《医方类聚》卷十引《神巧万全方》。
主治：脾脏虚冷，心腹有积滞气，发歇疼痛，心膈不利，两胁胀满，不能饮食。
加减：None
功效：None
用法用量：每服15-20丸，食前以生姜汤送下；温酒亦得。
制备方法：诃藜勒以下药为细末，以好酒1升，先煎硇砂，次入阿魏同煎5-7沸后，以帛滤过，再煎之，后下神曲末，搅令调，慢火熬成膏，拌和诸药末，入臼捣为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
