序号：53245
名称：倒流油乌髭三圣膏
组成：酸石榴（皮子皆用）、新胡桃（连青皮用）、新柿子（青者连蒂用）各等分。
出处：《御药院方》卷十。
主治：None
加减：None
功效：乌髭。
用法用量：如用时，先以温浆水洗净，以胆皮盛，临卧指蘸药捻之；如捻时自下捻之，其黑自下至髭根。
制备方法：用铁杵臼内捣烂如泥，用一小口新瓷罐子内，好黑锡3斤，拍作小钱大叶子，生牛乳同拌煎药令匀，罐口上用木柏子油绢密封，上用石灰磨捣泥之，日内晒令极干，于马粪内培1月，取出看药颜色，若深黑色即成；如未黑，再封于粪内培之。
临床应用：None
各家论述：None
用药禁忌：忌生葱、生萝卜、大蒜。
附注：三青膏（《瑞竹堂方》卷十）。
