序号：78706
名称：干姜黄连丸
组成：干姜1两，黄连1两，白术2两（炒），阿胶2两（粉炒），川芎1两，木香1两。
出处：《医略六书》卷二十八。
主治：孕妇赤白痢，脉紧数者。
加减：None
功效：None
用法用量：每服3钱，白痢，砂仁汤送下；赤痢，砂糖汤送下。
制备方法：上为末，乌梅肉3两，醋煮为丸。
临床应用：None
各家论述：妊娠冷热不调，肠胃敷化之权顿失，无以传送糟粕，故下痢赤白，胎孕不安焉，黄连清心脾之火以燥湿，干姜暖肠胃之冷以止痢，白术健脾生血，阿胶益血补阴，川芎行血海以升阳，木香调中气以醒脾胃也。为末，乌梅肉捣丸，白痢，熟砂仁汤下，赤痢，炒砂糖汤下，使血气各有所归，则肠胃之冷热无不化，而下痢之赤白无不除，何胎孕有不安者乎？
用药禁忌：None
附注：None
