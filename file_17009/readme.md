序号：17016
名称：巨胜子丸
组成：胡麻（拣净，不见水，酒淘净，黑豆上蒸，9蒸9晒）10两，生地（酒拌，9蒸9晒）10两，菟丝子（去沙，酒洗净，童便浸2日，捣为饼，晒干）10两，肉苁蓉（去中心膜，焙干）10两，枸杞子（人乳浸，蒸焙干）10两，破故纸5两（盐水拌炒），杜仲5两（去皮，童便浸，盐酒炒去丝），茯苓2两5钱（人乳拌蒸），当归2两5钱（酒洗，焙干），牛膝5两（酒洗，焙干），麦冬1两5钱（去心，焙干），五味子1两5钱（敲碎，蜜拌蒸，焙干），人参（中年）4两（少年）2两（童便浸，焙干），附子1个（重1两2钱者；用童便、甘草、防风煮透，用湿纸裹，慢火煨之），何首乌5两（竹刀刮去皮，米泔浸1宿，用黑豆上蒸，豆熟为度，去豆晒干，如此9次，晒干为末，少年不用）。
出处：《痘疹传心录》卷十八。
主治：补养气血，添精固髓，久服种子。
加减：None
功效：None
用法用量：每服3钱，空心白汤送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
