序号：48636
名称：神验摩风麝香膏
组成：麝香1两（细研），虎胫骨1两，细辛1两，防风1两（去芦头），独活1两，桂心1两，当归1两，芎1两，白芷1两，白僵蚕1两，生干地黄1两，白及1两，白术1两，川椒1两半（去目），附子1两（去皮脐，生用），旋覆花1两，赤芍药1两，连翘1两，甘菊花1两，木鳖子1两（去壳），天南星1两，栝楼根1两半，乌蛇1两半，牛膝1两（去苗），踯躅花1两，甘松香1两，石斛1两（去根），野驼脂10两，棘针2两，蜡5两，腊月猪脂2斤，醋3升，好酒2升。
出处：《圣惠》卷六十七。
主治：伤折，蹉跌筋骨，黯肿疼痛，及伤外风，风毒偏风，口面不正，但是伤风等。
加减：None
功效：软筋骨，润皮肉，止疼痛。
用法用量：用时先火上熁手心，点药摩痛处5-7度，并以温酒调药半匙服之。
制备方法：上净洗，晒干，锉细，入酒、醋中浸3宿，滤出阴干，却入腊月猪脂，驼脂内，以慢火煎，候白芷黄焦，则药成，以绵滤去滓，入麝香末调匀，以瓷盒盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
