序号：78440
名称：鳖甲丸
组成：鳖甲（去裙襕，醋炙）1两，桂（去粗皮）1两，京三棱（煨，锉）1两，牡丹皮1两，牛膝（去苗，酒浸，切，焙）1两，诃黎勒皮1两，琥珀1两，大黄（炮）1两，桃仁（去皮尖双仁，麸炒）1两，土瓜根1两，附子（炮裂，去皮脐）1两，赤茯苓（去黑皮）1两。
出处：《圣济总录》卷一五一。
主治：室女月经不调，或少不利，前后愆期，日月浸久，肌肉黄瘁，胁下积气结硬，时发刺痛，渐成劳状。
加减：None
功效：破血块、气块。
用法用量：每服20丸，煎桃仁汤送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：《普济方》有荆芥、白茯苓。
