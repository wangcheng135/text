序号：77003
名称：黑神丸
组成：木香半两，硇砂（研）半两，蓬莪术（煨，锉）半两，京三棱（煨，锉）半两，桂（去粗皮）1两，附子（炮裂，去皮脐）1两，干姜（炮）1两，干漆（捣碎，炒烟出）1两，大黄（煨，别为末）1两，青橘皮（汤浸，去白，焙）1两，墨（烧过）1两，巴豆（去皮心膜，细研出油）1两。
出处：《圣济总录》卷七十二。
主治：小肠疝气，妇人血气。
加减：None
功效：消积化气进食。
用法用量：每服3丸、5丸，茶、酒任下；如消食化气，生姜、橘皮汤下；小肠疝气，茴香酒下；妇人血气，当归酒下。
制备方法：上以好醋1大碗，先熬硇砂令沸，入巴豆又熬数沸，次又入大黄末熬成膏，余药并捣罗为末，以膏杵和为丸，如莱菔子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
