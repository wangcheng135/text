名称：煮酒方
组成：当归5钱，五加皮5钱，海风藤5钱，川芎2钱，牛膝2钱，黄芩2钱，木通2钱，薏仁2钱，木瓜2钱，白术1两，苍术1两，乌药1两，茯苓1两，甘草1两，寄生1两，白芍药3钱，生地3钱，陈皮3钱，半夏3钱，羌活3钱，独活3钱，麻黄3钱，金银花3钱，枳壳3钱，防风3钱，紫苏叶1钱，木香1钱。
出处：《医部全录》卷二二八。
主治：白虎历节疼痛。
加减：None
功效：None
用法用量：口服。
