序号：24810
名称：汝言化痰丸
组成：瓜萎、杏仁、海粉、桔梗、连翘、五倍子、香附、蛤粉、瓦楞子、风化消。
出处：《证治汇补》卷二。
主治：肺家老痰在于喉中，咯之不出，咽之不下。
加减：None
功效：泻热软坚。
用法用量：噙化；或作小丸，清茶送下。
制备方法：以姜汁少许，和竹沥捣入药，加蜜为丸。
临床应用：None
各家论述：《医略六书》：瓜萎泻热润燥以涤痰，桔梗清咽利膈以开结，瓦楞子消痰积、血积，风化硝化积热、结痰，五倍软坚豁痰，海粉泻热豁痰，连翘清热结，香附调血气，杏仁降气豁痰涎，蛤粉益阴利湿热，姜汁散痰，竹沥润液。和蜜捣丸，清茶化下，使湿化热降，则肺清润而老痰软，咯咽如常，安有咽喉窒塞之患？此泻热软坚之剂，为痰热固结之专方。
用药禁忌：None
附注：None
