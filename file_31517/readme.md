序号：31526
名称：补肝柏子仁丸
组成：柏子仁1两，黄耆1两（锉），白茯苓1两，楮实1两，覆盆子1两，五味子1两，附子1两（炮裂，去皮脐），石斛1两（去根），酸枣仁1两（微炒），鹿茸1两（去毛，涂酥炙令黄），桂心1两，白术1两，沉香1两，枳实1两（麸炒令黄），熟干地黄1两。
出处：《圣惠》卷三。
主治：肝虚寒，面色青黄，胸胁胀满，筋脉不利，背膊痠疼，羸瘦无力。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻。
附注：柏子仁丸（《圣济总录》卷四十二）。
