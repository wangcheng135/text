序号：58620
名称：秘传延寿丹
组成：何首乌（取赤白两种，黑豆汁浸1宿，竹刀刮皮，切薄片晒干，又用黑豆汁浸1宿，次早柳木甑桑柴火蒸3炷香，如是9次，晒干）共72两，菟丝子（先淘去浮空者，再用清水淘，挤去沙泥，5-6次，取沉者晒干，逐粒拣去杂子，取坚实腰样有丝者，用无灰酒浸7日方入甑，蒸7炷香，晒干，再另酒浸1宿，入甑蒸6炷香，晒干，如是9次，晒干磨细末）1斤，豨签草（5-6月采叶，长流水洗净，晒干，蜂蜜同无灰酒和匀拌潮1宿，次早蒸3炷香，如是9次，晒干为细末）1斤，桑叶（4月采取嫩叶，长流水洗净，晒干，照制豨签法九制，取细末）8两，女贞实（冬至日摘腰子样黑色者，剥去粗皮，酒浸1宿，蒸3灶香，晒干，为细末）8两，忍冬花（一名金银花，摘取阴干，照豨签草法九制，晒干，细末）4两，川杜仲（厚者是，去粗皮，青盐同姜汁拌潮，炒断丝）8两，雄牛膝（怀庆府产者佳，去根芦净，肉屈而不断，粗而肥大为雄，酒拌，晒干）8两，怀庆生地（取钉头鼠尾或原枝末，入水曲成大枝者有效，掐如米粒者，晒干，为细末）4两。
出处：《良方集腋》卷上。
主治：阴虚，脾虚，麻木，头晕，目昏，肥人痰湿多。
加减：阴虚，加熟地黄1斤；阳虚，加附子4两；脾虚，加人参、黄耆各4两，去地黄；下元虚，加虎骨1斤；麻木，加明天麻、当归各8两；头晕，加玄参、明天麻各8两；目昏，加黄甘菊、枸杞子各4两；肥人痰湿多，加半夏、陈皮各8两。
功效：乌须黑发，却病延年。
用法用量：None
制备方法：用四膏子（旱莲草熬膏1斤，金樱子熬膏1斤，黑芝麻熬膏2斤，桑椹子熬膏1斤）同前药末为丸，如膏不足，白蜂蜜增补，捣润方足。
临床应用：None
各家论述：None
用药禁忌：None
附注：延寿丹（《世补斋医书·文集》卷八）、首乌延寿丹（《中药成方配本》）。
