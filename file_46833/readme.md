序号：46848
名称：独角莲膏
组成：独角莲45g，皂角刺45g，白芷45g，防己45g，银花45g，连翘45g，生南星45g，刺猬皮45g，山甲片45g，当归45g，海桐皮45g，苏木45g，海带45g，大麻仁45g，豨莶草45g，干蟾3个，乳香35g，没药35g，血余45g。
出处：《朱仁康临床经验集》。
主治：痈肿，毛囊炎，瘢痕疙瘩，神经性皮炎。
加减：None
功效：提脓拔毒，消肿轻坚。
用法用量：用厚纸摊成大、中、小三号厚薄不同的膏药，用时烘烊，贴患处。
制备方法：用麻油6升，入大铁锅内，投入干蟾以上各药，熬枯去滓，再用强火熬至滴水成珠，离火，投入章丹（冬天约2.5kg，夏天约3kg），用铁棒急调，油渐变成黑色，最后将冷凝时，加入后药末，调和成膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
