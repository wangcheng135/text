序号：30470
名称：进食散
组成：半夏曲1两，肉豆蔻（面裹煨）1两，草果仁1两，高良姜（炒）1两，麦糵（炒）1两，附子（炮，去皮尖）1两，丁香1两，厚朴（去皮，姜炒）1两，陈皮（去白）1两，人参（去芦）半两，青皮（去白）半两，甘草（炙）半两。
出处：《医方大成》卷三引《济生》。
主治：脾胃虚寒，或为生冷所伤，或七情所扰，胸膈痞塞，不思饮食，痰逆恶心，大便溏泄。
加减：None
功效：None
用法用量：上（口父）咀。每服4钱，水1盏，加生姜5片，大枣1枚煎，不拘时候温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：进食汤（《普济方》卷二十五）。
