序号：39998
名称：育婴丸
组成：茯苓2两，法半夏2两，天麻2两，胆量1两2钱，僵蚕（炒）1两2钱，全蝎30个，白附子（炙）1两，钩藤1两2钱，薄荷1两2钱，防风1两2钱，羌活8钱，猪苓1两，厚朴（炙）1两，前胡8钱，白术（炒）1两，黄芩8钱，黄连8钱，藿香8钱，砂仁8钱，甘草2两。
出处：《北京市中药成方选集》。
主治：小儿急热惊风，四时感冒，咳嗽呕吐，痰喘气促，头痛身烧。
加减：None
功效：清热镇惊，熄风化痰。
用法用量：每服1丸，温开水送下，1日2次。
制备方法：上为细末，加麝香1钱、冰片1两、朱砂1两，混合均匀，炼蜜为丸，重5分，金衣三十六开，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
