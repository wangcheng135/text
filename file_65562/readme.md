序号：65578
名称：羚羊犀角散
组成：羚羊角屑1两，赤箭1两，酸枣仁1两，薏苡仁1两，白附子3分（炮裂），羌活半两，芎3分，犀角屑半两，当归3分（锉，微炒），白鲜皮半两，地骨皮半两，人参3分（去芦头），柏子仁3分，鹿角胶1两（捣碎，炒令黄燥），蔓荆子半两，牛黄2分（细研），麝香1分（细研）。
出处：《圣惠》卷六十九。
主治：妇人中风，身如角弓反张，筋脉拘急，言语謇涩，心神烦闷。
加减：None
功效：None
用法用量：每服1钱，以薄荷汤调下，不拘时候。
制备方法：上为细散，入研了药令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“羚羊角散”。
