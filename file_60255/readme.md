序号：60270
名称：通天再造散
组成：郁金半两（生），大黄1两（炮），白牵牛6钱（半生半炒），皂角刺1两（炮、经年黑大者）。
出处：《三因》卷十五。
主治：大风恶疾；疠疡下体患多。
加减：None
功效：None
用法用量：每服5钱，日末出面东以无灰酒调下，尽量为度。晚利黑头小虫，病稍轻者止利如鱼肠臭秽物，甚者不过2-3次。
制备方法：None
临床应用：疠疡：一男子冬间口苦耳鸣，阴囊湿痒，来春面发紫块，微肿麻木，至冬遍身色紫，不知痛痒，至春紫处俱大，至夏渐溃，又至春眉落指溃。此患在肝胆二经，今刺手指缝并臂腿腕出黑血，先与再造散2服，下毒秽，更以小柴胡合四物汤加白芷、防风、天麻、皂角刺，渐愈。又与换肌散，但遍体微赤，此血虚有火，因家贫未得调理，秋间发热，至春面仍发块，用前散并养血药，喜年少谨疾得愈。
各家论述：《医钞类编》：郁金散肝郁，下气破血，下虫毒；皂刺出风毒于荣血中；大黄利出瘀恶；牵牛利大小便，且苦寒皆能杀虫也。
用药禁忌：轻者忌毒半月，但食稠粥软饭，渐生眉毛，皮肤如常；甚者须将理，不可妄有劳动，并终身不得食牛、马、骡、驴等肉；忌房事。
附注：再造散（《丹溪心法》卷四）。
