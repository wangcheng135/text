序号：76296
名称：紫霞膏
组成：真小磨麻油1斤，象皮2两，当归2两，赤芍2两（象皮、当归、赤芍入油内，春、夏浸3日，秋、冬浸7日，将油熬至药枯，滤去滓，复入净锅内，熬至滴水成珠为度，务须勤看老嫩），松香（老嫩各半）100斤，葱100斤，生姜100斤（2味捣烂取汁，又将滓入水煮汁，去滓滤净，将汁入锅内，用蒸笼铺松毛于笼内，再将松香老嫩配搭，铺松毛上蒸化，松香汁滴在锅里葱姜汁内，捞起扯拔数百遍，放洁净地上数日听用。凡取用熬过松香1斤，加熬过药油4两，夏月只用3两5钱入锅内熬化，看老嫩火候得法，取起倾钵内，再入后药），乳香（去净）5钱，没药（去油净）5钱，血竭5钱，龙骨（煅）5钱。
出处：《疡医大全》卷七引张乘六方。
主治：痈疽发背，对口疔毒。
加减：None
功效：None
用法用量：任摊贴。
制备方法：上药各为极细末，入膏内，用槐柳条搅匀，再入漂朱、角朱（俱研至无声为度）各2两，又搅均匀，连钵头放在潮湿地上，顿多日出火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
