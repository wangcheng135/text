序号：32494
名称：寿星丸
组成：蛇含石1分，石燕（并火煅，酒淬3-5次）1钱，代赭石1钱，朱砂1钱，铁粉1钱，雄黄1钱，五灵脂、乳香、川乌（去皮，炮）、天浆子27个（炒），乌蛇肉（酒浸，炙，去骨）1钱，蛇皮（炙）、蛇头1个（酒浸，炙），僵蚕（皂角水浸1夕，焙，微炒）2钱，蝉蜕2钱，天麻2钱，蜂房（炒）2钱，蜈蚣（大赤足者）2钱，全蝎2钱（新薄荷自然汁浸1宿，焙，微炒），白附子1钱，南星（姜汁浸1夕，微炒，牛胆拌炒）1钱，羌活1钱，川芎1钱，麝香1钱，脑子半钱。
出处：《永乐大典》卷九七八引《全婴方》。
主治：小儿急慢惊风，荏苒经日，诸般痫病，累易医者无效，但是恶候，不问阴阳。
加减：如吐泻之后，加附子（炮去皮，随轻重入药）。
功效：None
用法用量：3岁每服1丸，薄荷入酒少许磨，或作散亦得。灌药1服，得睡即效。
制备方法：上为末，糊丸鸡头子大，金箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中僵蚕、蝉蜕、天麻、蜂房用量原缺。
