序号：40064
名称：肾沥汤
组成：鸡膍胵1两（微炒），远志1两（去心），人参1两（去芦头），黄耆1两（锉），桑螵蛸1两（微炒），泽泻1两，熟干地黄1两，桂心1两，当归1两，龙骨1两，甘草半两（炙微赤，锉），麦门冬2两（去心），五味子半两，磁石3两（捣碎，水淘去赤汁），白茯苓1两，芎2两，玄参半两。
出处：《圣惠》卷五十三。
主治：（疒肖）肾。肾气虚损，发渴，小便数，腰膝痛。
加减：None
功效：None
用法用量：每服用羊肾1对（切去脂膜），先以水1大盏，煮羊肾至1盏，去水上浮脂及肾；次入药5钱，生姜半分，煎至5分，去滓，空心温服，晚食前再服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方原名“肾沥丸”，与剂型不符，据《鸡峰》改。又本方方名，《普济方》引作“肾沥散”。
