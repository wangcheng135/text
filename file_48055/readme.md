序号：48070
名称：神妙乌髭方
组成：酸石榴1个。
出处：《普济方》卷四十九引《卫生家宝》。
主治：None
加减：None
功效：乌髭发。
用法用量：早、晚搽牙，须令擦得牙齿热，以使药力行。至10日永无白发，更无虫牙。若青春人用此药，一世更无白者，如摘白须后，以生姜汁擦，遂生黑者。
制备方法：于石榴上匀钉小铁钉49个，遇夜安顿在露地或屋上，至晓即收在屋下，勿令见日色，露三夜毕，取出去钉子，每一钉窍内，塞一丁香，每取出1个钉，便塞丁香，用纸裹石榴，以好米醋和黄泥，固济石榴，顿在风道略干，以炭火簇烧，令通红为度取出，移在一净地上，候冷去泥净，取烧过者石榴，碾为细末。
临床应用：None

用药禁忌：None
