序号：68250
名称：黄耆散
组成：人参3两（去芦），黄耆3两（洗，捶破，蜜水炙香），半夏3两（汤泡7遍，薄切旋入），白茯苓3两（去皮），当归3两（去芦，酒浸），麦芽3两（炒），白术（炒）3两，白芍药4两，甘草（炙）1两，肉桂1两（去粗皮，不见火），神曲（炒）1两。
出处：《魏氏家藏方》卷四。
主治：男子、妇人诸虚不足，病后羸乏，微发寒热，精竭力弱，血气劳伤，痰多呕逆，不思饮食，骨节酸痛，嗽喘气急，面色浮黄。
加减：None
功效：None
用法用量：上（口父）咀。每服3钱，加生姜5片，枣子3个，水1盏半，煎8分，去滓，食前温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
