序号：58558
名称：秘传乌鸡丸
组成：熟地3两，白术3两，川芎3两，白芍3两，厚朴3两，香附3两，甘草3两，人参2两，砂仁2两，海金砂4两。
出处：《郑氏家传女科万金方》卷二。
主治：妇人25-26岁，血海虚冷，气血俱虚，经脉不调，或时腹痛，或下白带如脑髓流，或如米泔，不分信期，每来淋沥不止，面色痿黄，四肢无力，头晕目眩眼花。
加减：None
功效：None
用法用量：每服80-90丸，每晨以米饮或酒送下。
制备方法：上药合一处和匀，用乌骨公鸡1只约2-3斤，不落水，去净毛，并去肚杂及头翅足，将前药取1分置于鸡腹内，及外余药一并置入铜罐内，用好酒5碗，水2钟，文武火煮至干，取出去骨，用净肉同药晒干，为末，酒糊粳米为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
