序号：75863
名称：葛根解托汤
组成：干葛8分，柴胡8分，前胡8分，防风6分，陈皮1钱，半夏1钱，泽泻1钱，生甘草3分，生姜、大枣。
出处：《不居集》（上集》卷十。
主治：正气内虚，客邪外逼，有似虚劳各症。
加减：如寒气胜者，加当归7分，肉桂5分；阴气不足者，加熟地1钱；若元气大虚，正不胜邪，兼用补托之法；如头痛者，加川芎、白芷各7分；气逆多嗽者，加杏仁1钱；痞满气滞者，加白芥子5-7分。
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：此症原非内虚，补之而邪益壅，托之而邪易解。盖解托之妙，妙用葛根，葛根味辛性凉，诸凉药皆滞，能遏表寒，惟葛根之凉，凉而能解；诸辛药皆燥，能发内热，惟葛根之辛，辛而能润；其用与柴胡互有短长，柴胡妙于升，能拔陷：前胡妙于降，能平气；干葛妙于横行，能托里；用二陈、姜、枣之辛甘温以和营卫。外有柴、前、防风以托出，内有泽泻以分消，解托之妙，尽于此矣。
用药禁忌：None
附注：方中生姜、大枣用量原缺。
