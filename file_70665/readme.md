序号：70681
名称：紫菀散
组成：紫菀半两（洗去苗土），人参3分（去芦头），半夏半两（汤洗7遍去滑），白茯苓1两，陈橘皮3分（汤浸，去白瓤，焙），麦门冬1两（去心，焙），当归半两（锉，微炒），黄耆1两（锉），白芍药半两，桂心半两，熟干地黄1两，甘草1分（炙微赤，锉），五味子3分，杏仁半两（汤浸，去皮尖双仁，麸炒微黄）。
出处：《圣惠》卷七十八。
主治：产后咳嗽，四肢无力，吃食减少。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，大枣3枚，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
