序号：12063
名称：王君河车方
组成：紫河车1具（首生并壮盛胞衣是也，挑血筋，洗数10遍，仍以酒洗，阴干，煮和各药），生地8两，牛膝4两，五味子3两，覆盆子4两，巴戟2两（欲多世事，加1两；女人不用），诃黎勒3两，鼓子花2两，苦耽2两，泽泻3两，甘菊花3两，菖蒲3两，干漆3两（炒黄），柏子仁3两，白茯苓3两，黄精2两，苁蓉2两（女人不用），石斛2两，远志2两，杏仁4两（炒黄，去皮尖），苣胜子4两，（一方有云母石3两）。
出处：《遵生八笺》卷三。
主治：None
加减：None
功效：驻颜，益寿。
用法用量：酒或盐汤送下。服3料，颜如处子。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
