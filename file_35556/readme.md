序号：35566
名称：侧子丸
组成：侧子1两半（炮裂，去皮脐），天麻1两，独活1两，石斛1两（去根，锉），杜仲1两（去粗皮，炙微黄，锉），鹿茸1两（去毛，涂酥，炙微黄），牛膝1两（去苗），附子1两（炮裂，去皮脐），当归1两，肉桂1两（去皱皮），威灵仙1两，五加皮1两，丹参1两，海桐皮1两（锉），安息香1两半，虎胫骨1两（涂酥，炙令黄）。
出处：《圣惠》卷二十一。
主治：风病，脚膝软弱，行立不得。
加减：None
功效：补益肾脏，强壮骨髓，利风毒，除缓弱。
用法用量：每服30丸，渐加至40丸，空心及晚食前以豆淋酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
