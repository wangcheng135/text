序号：58944
名称：秦艽白术丸
组成：秦艽（去芦）1两，桃仁（汤浸，去皮尖）1两，皂角仁（烧存性）1两，当归梢（酒浸）5钱，泽泻5钱，枳实（麸炒黄）5钱，白术5钱，地榆3钱。
出处：《兰室秘藏》卷下。
主治：痔疾并痔漏有脓血，大便燥硬而作疼痛不可忍。
加减：None
功效：None
用法用量：每服50-70丸，空心白汤送下，待少时以美膳压之。
制备方法：上为细末，和桃仁泥研匀，煎熟汤打面糊为丸，如鸡头子大，令药光滑，焙干。
临床应用：None
各家论述：①《兰室秘藏》：其疾甚者，当以苦寒泻火，以辛温和血润燥，疏风止痛，是其治也。以秦艽、当归梢和血润燥；以桃仁润血；以皂角仁除风燥；以地榆破血；以枳实之苦寒补肾，以下泄胃实；以泽泻之淡渗使气归于前阴，以补清燥受胃之湿邪也；白术之苦甘，以苦补燥气之不足，其甘味以泻火而益元气也。故曰甘寒泻火，乃借枳实之寒也。②《成方便读》：此方以秦艽入阳明，润燥宣风，能行能散；白术健脾胜湿，不特能利腰脐之血结，且可使离经之血仍统于脾，而协之以归尾、桃仁之破血润燥；枳实、皂角之导滞宽肠；泽泻以渗湿；地榆以凉瘀。丸以缓之，亦不欲其速化之意，立方之法，不愧为四大家之一耳。
用药禁忌：忌生冷、硬物、冷水、冷菜之类；并湿面、酒，及辣辛热、大料物之类。
附注：本方改为汤剂，名秦艽白术汤（见《外科经验方》）。
