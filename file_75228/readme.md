序号：75244
名称：腹痛广济散
组成：人参1两半，厚朴8钱（制），当归3两，陈皮1两半，茯苓1两半，炙甘草5钱，生姜7片，砂糖3两（炒灰）。
出处：《医略六书》卷三十。
主治：胁肋腹痛，脉软弦涩者。
加减：None
功效：None
用法用量：酒煎3钱，去滓温服。
制备方法：上为散。
临床应用：None
各家论述：产妇努力太过，中气大伤，不能统运营血，故胁肋胀满，腹痛不止。人参补中气之虚，厚朴散腹胁之满，当归养血以荣经，陈皮利气以和胃，茯苓渗湿气，炙草益胃气，煨生姜温胃散滞气，砂糖灰去宿生新血以行血滞也。为散酒煎，务使中气内充，则血不复滞，而经脉清和，何患胁肋疼胀不去乎。
用药禁忌：None
附注：None
