序号：63356
名称：清宫丹
组成：柴胡3钱，蒲黄5钱，枳壳3钱，生石膏3钱，寸冬5钱，黄芩3钱，郁金3钱，酒杭芍3钱，薄荷3钱，清夏3钱，桔梗3钱，云黄连2钱，山栀3钱，朱砂2钱，胆草3钱，川羌活3钱，独活3钱，犀角3钱，粉草2钱，胆星2钱。
出处：《全国中药成药处方集》（吉林方）。
主治：湿病，麻疹，大头瘟毒，瘟毒发斑，感冒。
加减：None
功效：退热解毒，透汗解表。
用法用量：大人每服1大丸，5-10岁小儿每服1小丸，2岁以上小儿每服2分之1小丸，周岁以内小儿每服3分之1小丸，用桑叶、菊花、薄荷、鲜姜煎汤为引。
制备方法：上将朱砂另研，余为细面，一处调匀，炼蜜为丸，大丸1钱4分重，小丸7分重，大赤金为衣，棉纸包裹，蜡皮封。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服；忌腥辣厚味。
附注：None
