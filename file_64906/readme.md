序号：64922
名称：续嗣壮元丹
组成：鹿茸（酥炙）1两，沉香1两，苁蓉（酒洗，去甲用）1两，天冬（去心）1两，麦冬（去心）1两，拣参1两，熟地（蒸）1两，巴戟（去心）1两，枸杞1两，茯苓1两，五味1两，当归（酒洗）1两，杜仲（酒洗）1两，牛膝（去芦，酒洗）1两，菟丝（酒洗令净，晒半干捣成饼后，晒干为末）1两，小茴（盐炒）1两，鳖甲（酥炙）1两，故纸（炒）1两，首乌（米泔浸）1两，石菖蒲（去毛）1两，山药4两，柏子仁4两，萸肉（酒蒸，去核）4两，朱砂5钱。
出处：《寿世保元》卷七。
主治：虚损，阳事不举，少壮纵情，痼冷，心肾不交，难成子嗣，遗精白浊，五劳七伤，一切虚损。
加减：None
功效：None
用法用量：空心、临卧以温盐汤送下。
制备方法：上为细末，酒打面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌烧酒、胡椒、干姜、煎炒之物。
附注：None
