序号：43374
名称：养胃汤
组成：香附7分，砂仁7分，木香7分，枳实（麸炒）7分，白术（去芦）1钱，茯苓（去皮）1钱，半夏（姜汁炒）1钱，陈皮1钱，白豆蔻（去壳）7分，藿香7分，厚朴（姜汁炒）7分，甘草（炙）2分。
出处：《回春》卷三。
主治：胸腹痞满。
加减：瘦人心下痞闷，加炒黄连，去半夏；血虚中满，加当归、白芍，去半夏；食积中满，加炒神曲、山楂、麦芽，去白术、半夏；肥人心下痞闷，加苍术；气虚中满，加人参，去半夏；痰膈中满，加瓜蒌仁、贝母、桔梗、竹沥、姜汁少许，去白术、半夏；脾泄中满，加炒苍术、炒白芍，去半夏；内伤元气而痞满者，宜大补气也。
功效：None
用法用量：加生姜3片，大枣1枚，水煎，食后服。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
