序号：27347
名称：补益白薇丸
组成：白薇1两，木香半两，当归半两（锉，微炒），桂心半两，泽兰半两，牛膝半两（去苗），熟干地黄1两，牡丹半两，人参半两（去芦头），芎半两，白术半两，枳壳半两（麸炒微黄，去瓤），白茯苓3分，细辛1两，赤石脂1两，龙骨1两，禹余粮1两（烧，醋淬7遍），附子3分（炮裂，去皮脐），黄耆1两（锉），续断半两，吴茱萸1分（汤浸7遍，焙干微炒），厚朴半两（去粗皮涂生姜汁炙令香熟）。
出处：《圣惠》卷八十一。
主治：产后风虚劳损，腹内冷气，脚膝无力，面色萎黄，饮食减少，日渐羸瘦。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：白薇丸（《普济方》卷三五○）。
