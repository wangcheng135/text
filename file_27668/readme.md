序号：27676
名称：八珍益母丸
组成：益母草4两（不见铁器，只用上半截带叶者），人参（去芦）1两，白术（土炒）1两，茯苓（去皮）1两，炙甘草（去皮）5钱，当归（酒洗）2两，川芎1两，白芍药（醋炒）1两，熟地黄（酒洗）2两。
出处：《医统》卷八十四。
主治：妇人气血两虚，脾胃并弱，饮食少思，四肢无力，月经违期，或先期而至，或腰疼腹胀缓而不至，或愆期不收，或断或续，或赤白带下，身作寒热，久不受孕。
加减：脾胃虚寒者，加砂仁1两（姜汁炒）；腹中胀闷者，加山楂1两（净肉，饭上蒸）多郁者，加香附子1两（童便制）。
功效：补气血，调月经。
用法用量：每次1丸，空心蜜汤送下。如不能嚼者，丸以细粒如小豆大，每服70-80丸。
制备方法：上为末，蜂蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中除益母草外，用量原缺，据《景岳全书》补。
