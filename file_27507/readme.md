序号：27515
名称：补母寿子方
组成：人参1钱（如弱人用2钱），当归2钱，白术2钱，川芎8分，怀生地（自蒸）2钱，条芩2钱，紫苏4分，陈皮4分，甘草4分。
出处：《胎产指南》卷一。
主治：屡产生子无气，或育而不寿，气血虚弱，孕成不安，或得孕数堕。
加减：虚弱人，陈皮去白，再加黄连5分，大枣3枚；脾胃弱，常泄泻，加莲子10枚，带壳砂仁3分，减地黄；多怒而泻，加木香2分；口常燥渴，加麦冬1钱；怔忡惊悸，加枣仁1钱，益智1钱，天员10个。
功效：益胎而分娩易，生子精神有寿。
用法用量：水煎服。每月服15贴，弱甚者，每日1帖。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
