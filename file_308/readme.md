序号：308
名称：五香丸
组成：牛黄（研）3分，犀角（屑）3分，升麻4分，沉香4分，熏陆香4分，当归4分，桂心4分，青木香4分，麝香（研）4分，雄黄（研如粉）4分，鬼箭羽4分，巴豆（去心皮，熬）4分，诃黎勒皮4分，朱砂（研）4分，槟榔仁4分，干姜4分，吴茱萸4分，甘草（炙）4分，豆蔻4分，桃仁（去尖皮，熬）5分，附子（炮）5分。
出处：《外台》卷三十一引《救急方》。
主治：诸毒疰气，心腹胀满，大小便不通，鬼疰，心痛不可忍。
加减：None
功效：None
用法用量：每服3丸至5丸，以暖水送下。如不利，更服，以利为度。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌海藻、菘菜、猪肉、冷水、生葱、芦笋、生血物等。
附注：None
