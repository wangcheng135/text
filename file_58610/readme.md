序号：58625
名称：秘传肉丹
组成：伏火朱砂4两（研如粉），代赭石4两（火煅，水飞过），鹿角胶7两（蛤粉炒），鹿角霜2两，鹿茸7两（酒浸，炙），白龙骨3两（生用，水飞），虎胫骨4两（酥炙），天雄2两（炮），桑螵蛸4两（炒），磁石4两（醋淬，水飞），牡蛎粉4两（韭菜叶和泥煅，水飞），附子1两（炮），钟乳粉2两，阳起石2两（煅，水飞），乳香1两，没药1两，沉香1两（1方用灵砂不使朱砂）。
出处：《朱氏集验方》卷八。
主治：None
加减：None
功效：养心肾，生精血，补虚损，壮筋骨，固下元。
用法用量：每服30丸，加至50丸，空心以盐汤送下。
制备方法：上以糯米粉用酒和成饼煮熟为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“朱赭妙丸”。
