序号：61819
名称：接骨神异膏
组成：姜4两，葱4两，韭4两，蒜4两，槿树皮4两，麻油2斤，猪油1斤，大黄2两，当归2两，桃仁2两，红花2两，川乌2两，草乌2两，羌活2两，独活2两，赤芍2两，苏木2两，骨碎补2两，五加皮2两，甘松2两，山奈2两（浸熬）。
出处：《理瀹》。
主治：骨碎。
加减：None
功效：None
用法用量：贴患处。
制备方法：上用油2斤，煎乱髮（勿洗）1斤，俟枯，合前药为1锅再熬，下丹1斤收，徐徐下松香末5斤收，再下土鳖虫（炒黑）4两，煅龙骨3两，血竭1两，醋煅自然铜、乳香、没药、炙虎骨、肉桂各2两，另研血竭末1两，搅匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
