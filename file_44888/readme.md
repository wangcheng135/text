序号：44903
名称：既济丸
组成：麦冬（去心）1两半（另捣膏），五味子5钱，菟丝子（淘去沙，酒煮1日，捣烂，捏作饼，晒干，研末）8钱，淮地黄（酒浸透，9蒸9晒，另捣膏）1两，远志7钱，山药（去皮，盐水炒）8钱，茯神（去木）1两，甘草（水煮，去心）7钱，枣仁。
出处：《采艾编翼》卷二。
主治：内伤。
加减：None
功效：None
用法用量：每早服1钱5分，白滚水送下，临卧再服1钱；服5-6日后，歇1-2日再服。
制备方法：上除麦冬、地黄、枣仁另捣膏外，其余药物共为细末，再入前膏捣匀，以荷叶蒸饭为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中枣仁用量原缺。
