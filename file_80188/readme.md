序号：80204
名称：漏芦汤
组成：漏芦6钱，连翘6钱，白蔹6钱，芒消6钱，甘草6钱，大黄1两，升麻9铢，枳实9铢，麻黄9铢，黄芩9铢。
出处：《千金》卷五。
主治：小儿热毒痈疽，赤白诸丹毒疮疖，眼赤痛，生翳障。
加减：None
功效：None
用法用量：上(口父)咀。以水1升半，煎取5合，儿生1-7日，取1合，分3服；8-15日，取1合半，分3服；16-20日，取2合，分3服；20-30日，取3合，分3服；30-40日，取5合，分3服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：漏芦连翘汤（《千金》卷十）、漏芦散（《圣惠》卷九十）、千金漏芦汤（《卫生总微》卷二十）、漏芦煮散（《普济方》卷二八五）。
