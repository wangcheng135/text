序号：62781
名称：清气散
组成：前胡（去苗，洗）、柴胡（去苗，洗）、川芎（洗）、枳壳（去瓤，锉，麸炒）、白术、青皮（去白）、羌活（去芦）、独活（黄色如鬼眼者，去芦，洗，焙，称）、甘草（炙）、茯苓（去皮）、人参（去芦）各等分。
出处：《本事》卷四。
主治：热气壅盛，痰涎，胸膈烦热。
加减：None
功效：调荣卫，顺三焦，消痰诞，退烦热。
用法用量：每服2钱，以水1盏，加荆芥1穗，煎7分服。
制备方法：上为末。
临床应用：None
各家论述：《本事方释义》：前胡气味苦辛微寒，入手足太阴阳明；柴胡气味辛甘平，入足少阳；川芎气味辛温，入肝胆；枳壳气味苦寒，入足太阴；白术气味甘温，入手足太阴；青皮气味辛酸微温，入肝胆；羌活气味苦辛甘平，入足太阳；独活气味苦辛甘平，入足少阴；甘草气味甘平，入足太阴；茯苓气味甘平淡渗，人足阳明；人参气味甘温，入脾胃；少佐以荆芥穗之辛温，盖即用古方败毒散增损者也。因荣卫不调，三焦不顺，风热变秘，痰涎上逆，故以补中之品扶持正气，以诸风药驱除外邪，则病退而元气不伤矣。
用药禁忌：None
附注：None
