序号：62461
名称：梅花普度丹
组成：藿香3两，黄芩3两，紫苏2两，香薷2两，细生地2两，荆芥穗2两，橘红（盐水炒）2两，制半夏2两，白术2两，泽泻2两，川连2两，川柏2两，牛蒡2两，黑豆皮2两，制香附1两5钱，青蒿1两5钱，防风1两5钱，川芎1两5钱，淡豆豉1两，黄菊1两，白蒺藜1两，六神曲1两，建神曲1两，白茯苓1两，赤苓1两，连翘1两，滑石1两，车前子1两，当归头1两，川贝1两，赤小豆1两，大麦芽5两，谷芽5两，煨木香5钱，砂仁5钱。
出处：《饲鹤亭集方》。
主治：暑痧疟痢，经络拘挛，头晕腹痛，手足厥冷，一切伤寒、伤风、痰痫诸症。
加减：None
功效：None
用法用量：每服1丸，随时用引，四月野蔷薇花2钱，梅花瓣3分（如用霜梅、乌梅，重者2个，轻者1个）煎汤送下；5月米仁1钱，梅花分两如前；6月鲜佩兰叶2钱，梅花如前；7月薄荷1钱，荷梗1钱，梅花如前；8月柴胡1钱，梅花如前；9月苏梗2钱，梅花如前；小儿照引加钩藤1钱，北地照引加大黄2钱，煎汤送下。
制备方法：上为末，另用梅花瓣5分（如无花时，用枝叶嫩头3个，无梅树处用霜梅、乌梅去核代之），桂枝5分，天泉水煎1碗，匀洒药末上，再用甘草8钱煎汤为丸，如弹子大，每丸重2钱，辰砂2两为衣。
临床应用：None
各家论述：None
用药禁忌：孕妇不忌。
附注：None
