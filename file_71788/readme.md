序号：71804
名称：痧药
组成：丁香21g，苍术110g，天麻126g，麻黄126g，大黄210g，甘草84g，冰片0.5g，麝香10.5g，蟾酥（制）63g，雄黄126g，朱砂126g。
出处：《中国药典》一部。
主治：夏季食凉饮冷，猝然闷乱烦躁，腹痛吐泻，牙关紧闭，四肢逆冷。
加减：None
功效：祛暑解毒，辟秽开窍。
用法用量：每次口服10-15粒，1日1次。小儿酌减。外用研细吹鼻取嚏。
制备方法：上除麝香、蟾酥，冰片外，雄黄，朱砂分别水飞，或粉碎成极细粉，其余丁香等6味粉碎成细粉，将麝香、蟾酥、冰片细粉与上述粉末（朱砂除外）配研，过筛，混匀，用水泛丸，阴于，朱砂粉包衣，打光。
临床应用：None
各家论述：None
用药禁忌：不宜过量多服；孕妇忌用。
附注：None
