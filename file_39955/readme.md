序号：39965
名称：肥皂丸
组成：白芷1两，白附子1两，白僵蚕1两，白及1两，猪牙皂角1两，白蒺藜1两，白蔹1两，草乌1两，山奈1两，甘松1两，白丁香1两，杏仁1两，豆粉1两，轻粉半两，密陀僧半两，孩儿茶3钱，肥皂（去裹外皮筋并子，只要净肉）1条盏，樟脑半两。
出处：《丹溪心法附余》卷二十四。
主治：None
加减：None
功效：去白斑、黑点、白癣、诸般疮痕，令人面色好。
用法用量：None
制备方法：上先将净肥皂肉捣烂，用鸡清和，晒去气息；将各药为末，同肥皂、鸡清和为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：玉容肥皂丸（《冯氏锦囊》卷十九）、玉容肥皂（《疡医大全》卷十二）。
