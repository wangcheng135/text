序号：58734
名称：大黄丸
组成：大黄1两（酒洗过，采下蒸熟，切片，晒干），川芎1两（锉），甘草1分（锉，炙），黑牵牛半两（半生熟炒）。
出处：《小儿药证直诀·附方》。
主治：风热里实，口中气热，大小便闭赤，饮水不止，有下证者。疮痂初起而能食，食而胀满，不大便而喘急，昏甚而谵语者。
加减：None
功效：None
用法用量：2岁每服10丸，温蜜水送下，乳后服。以溏利为度。末利，加丸数再服。
制备方法：上为细末，稀糊为丸，如麻子大。
临床应用：None
各家论述：《小儿药证直诀类证释义》：此方大黄、黑丑攻涤泻下，而以川芎升之，甘草缓之，相辅而行，使泻下而有所制。
用药禁忌：None
附注：None
