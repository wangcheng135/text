序号：9202
名称：加味四物汤
组成：当归3钱，鹿茸3钱，白芍3钱，香附3钱，川芎2钱5分，熟地2钱5分，黄耆2钱，白术2钱，茯苓2钱，黄芩2钱，陈皮（去白）2钱，砂仁2钱，人参2钱，阿胶2钱，小茴2钱，山萸2钱，沉香1钱，粉草1钱，延胡2钱。
出处：《女科旨要》卷一。
主治：妇人25-26，气血两虚，血海虚冷，经脉不调，或时腹下疼痛，或白带，或如鱼脑髓，或如米汁，信期不定，每日淋漓不止，面色青黄，四肢无力，头晕眼花。
加减：如咳嗽潮热，加5味子，杏仁各5分，竹沥少许。
功效：None
用法用量：分4帖。加生姜3片煎，空心服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
