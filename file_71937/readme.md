序号：71953
名称：滋阴养血汤
组成：生地9g，熟地9g，当归身9g，阿胶珠12g，白芍9g，首乌9g，枸杞子12g，炒枣仁12g，柏子仁9g，天冬9g，麦冬9g，鳖甲9g，龟版9g，甘草3g。
出处：《临证医案医方》。
主治：慢性肝炎（阴虚型）。头晕心悸，五心烦热，或易怒神疲，肝区隐痛，舌质淡，少苔，脉细弦或细数。
加减：None
功效：滋阴养血。
用法用量：None
制备方法：None
临床应用：None
各家论述：方中用龟版、鳖甲、生地、天冬、麦冬等滋阴；用熟地、当归身、阿胶珠、白芍、首乌、枸杞子等以养血；用柏子仁，炒枣仁以治心悸不安，甘草调和诸药。全方配伍，共起到滋阴养血的作用。
用药禁忌：None
附注：None
