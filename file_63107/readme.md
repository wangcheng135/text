序号：63123
名称：大戟丸
组成：大戟（去皮，锉，炒黄）半两，葫芦巴4两（炒），木香1两，附子（炮，去皮脐）1两，舶上茴香1两，诃子（煨，去核）1两，槟榔1两，川楝5两（后入），麝香半钱（别研）。
出处：《三因》卷十四。
主治：阴(疒颓)肿胀，或小肠气痛。
加减：None
功效：None
用法用量：每服5-7丸至10丸，空心温酒送下；姜汤亦得；潮发疼痛，炒姜热酒送下15丸。
制备方法：上为末，独留川楝，似好酒1-2升，葱白7枚，长3-4寸，煮川楝软，去皮核，取肉，和上药为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：麝香大戟丸（《局方》卷八续添诸局经验秘方）。
