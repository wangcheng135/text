序号：8898
名称：水杨膏
组成：水杨皮2两（锉），槐皮2两（锉），黄丹6两，麒麟竭1两（末），密陀憎1两半（细研），白松脂1两，蜡1两，白蔹1两（锉），降真香1两半，油2斤。
出处：《圣惠》卷六十三。
主治：一切痈疽发背。
加减：None
功效：生肌敛疮。
用法用量：用于软帛上摊贴，每日2度换之。
制备方法：先将油于铛内，微火煎水杨皮、槐皮，后下白蔹、麒麟竭、松脂、降真香，再煎；候水杨皮黄黑色，以绵滤去滓，再入铛内重煎，即入密陀憎，并黄丹、蜡等，用柳木蓖搅，勿令住手，候色变黑，旋滴于冷处，看硬软得所；膏成，盛于瓷器中。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
