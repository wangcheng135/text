序号：58817
名称：秘传紫府青津丸
组成：女贞实4两（用芩、连水浸1夜，次日蒸晒，如法3遍），白石膏4两（煅过，研细，用嫩桑叶4-5斤煎汁，取净汁1碗煮干，再用紫苏4两，荆芥1两，煎清汁，再待干听用），知母4两（净咀片，分4处：人乳、童便、青盐拌润过1宿，生用1分；俱微火炒），黄柏4两（净，照前四制如法），白芍药1两（用桑皮煎水煮干），贝母2两（姜矾水煮干），杏仁2两（去皮尖，青盐水煮干），天门冬2两（去心，切细，微火炒干），麦门冬2两（去心，微火焙干），人参1两（切大片，用好酒拌润1宿，取白酒、曲末炒热，下人参微炒干，去曲），茯神2两（去皮心，人乳拌润1夜，次日火倍干），黄耆1两（切片，蜜水拌润1宿，炒干），糖球肉5钱，当归1两（酒洗晒干，切片，酒拌润1宿，炒），陈皮1两（去白，炒），百合2两（姜汤泡过，焙干）
出处：《遵生八笺》卷十八。
主治：虚实痰火。
加减：None
功效：制伏相火，滋养真阴，津润肺腑，上降心火，下生肾水，清热化痰。
用法用量：每服3钱，早、晚白滚汤送下。
制备方法：上药各制精微分两，和一处，再焙大燥，为极细末，取梨汁半斤，炼蜜1斤，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
