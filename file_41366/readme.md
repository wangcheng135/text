序号：41381
名称：金虎丹
组成：牛黄（研）1两，丹砂（研）1两，粉霜（研）1两，腻粉（研）1两，雄黄（研）1两，龙脑（研）1两，铅白霜（研）1两，天竺黄3两，人参3两，麦门冬（去心）3两，甜消（研）2两，蓬砂（研）半两，白矾（研）1两半。
出处：《圣济总录》卷六。
主治：一切风涎，中风口噤不语，忽然仆地，涎潮壅塞。
加减：None
功效：None
用法用量：每服1丸，新汲水化下；伤寒伏热；在心，狂乱者，每服1丸，蜜水化下；小儿急慢惊风，口吐涎沫，1丸分作3服，薄荷新水化服。
制备方法：上为末，再同研匀，炼蜜为丸，如樱桃大。
临床应用：None
各家论述：None
用药禁忌：None
附注：金虎丸（《普济方》卷九十二）。
