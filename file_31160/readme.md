序号：31169
名称：杖丹膏药
组成：甘草1两，肉桂1两，蛇蜕1两，蝉蜕1两，露蜂房1两，连翘1两，白芷1两，白及1两，白蔹1两，白术1两，苍术1两，人参1两，玄参1两，苦参1两，芍药1两，南星1两，升麻1两，厚朴1两，栀子1两，百合1两，金银花1两，天花粉1两，川归1两，川芎1两，穿山甲（煨胖，另研）1两，羌活1两，独活1两，黄连1两，黄芩1两，黄柏1两，大黄1两，生地黄1两，红花1两，苏木1两，柴胡1两，鳖甲（酥炙，为末）1两，青木香1两，何首乌1两，防风1两，荆芥穗1两，藿香1两，云母石1两，花蕊石1两，乱发（壮年无病男子者）1块，干蟾（即风鸡）1只，凤凰胎（即壳中不转头鸡黄也，阴干用）1只，桃枝5茎，柳枝5茎，桑枝5茎。
出处：《医学正传》卷六引《外录验方》。
主治：杖疮及诸般痈疽疮疖，已溃未溃。
加减：None
功效：None
用法用量：临期看疮大小摊贴。受杖责后，如死血壅肿，宜先刺出恶血，然后以此膏贴之，3-4日平复。或早失调理成痈者，贴之即散。
制备方法：上各切细，用香油6斤，浸药3-5日，入锅内熬黑色，去滓，入黄丹3斤，别用槐柳枝不住手搅，膏成候温，入乳香、没药、龙骨、轻粉、血竭各1两，麝香2钱，搅匀，瓷器收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
