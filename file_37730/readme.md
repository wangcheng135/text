序号：37740
名称：拈痛散
组成：羌活（去芦头）1两，独活（去芦头）1两，细辛（去苗叶）1两，肉桂（去粗皮）1两，防风（去芦头并叉者）1两，白术1两，川乌（生，去皮脐）1两，良姜（锉）1两，麻黄（不去节）1两，天麻（去苗）1两，吴茱萸（生用）1两，葛根1两，乳香（另研）1两，小椒（去子及闭口者）1两，全蝎（生用）1两，当归（去苗）1两，川姜（生用）半两。
出处：《御药院方》卷八。
主治：肢节疼痛。
加减：None
功效：None
用法用量：熨烙痛处，不拘早晚，频用为效。如药冷，即再炒1次，用毕，其炒药不用。
制备方法：上为粗末，入乳香匀，每抄药10钱，痛甚者加至15钱，同细盐1升，一处炒令极热，用熟绢袋内贮药。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
