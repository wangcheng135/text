序号：35473
名称：三仁承气汤
组成：大麻仁3钱（炒香），松子仁3钱（研透），小枳实1钱半（炒香），大腹皮2钱，光杏仁3钱（勿研），生川军1钱（蜜炙），油木香5分，猪胰（略炒）1钱。
出处：《重订通俗伤寒论》。
主治：胃燥脾约，液枯便闭。
加减：None
功效：缓下脾脏结热。
用法用量：None
制备方法：None
临床应用：None
各家论述：脾与胃以膜相连，膜者，脂膜也；上济胃阴，下滋肠液，皆脾所司。若发汗利小便太过，则胆火炽盛，烁胃熏脾；胃中燥而烦实，实则大便难，其脾为约，约则脾之脂膜枯缩矣。故君以麻、杏、松仁等多脂而香之物，濡润脾约，以滋胃燥；然胃热不去，则胆火仍炽，又必臣从生军、枳实，去胃热以清胆火，所谓釜底抽薪是也；佐以油木香、大腹皮者，以脾气喜焦香，而油木香则滑利脂膜，脾络喜疏通，而大腹皮又能直达脾膜也；妙在使以猪胰，善去油腻而助消化，以洗涤肠中垢浊。此胃燥脾约，液枯便闭之良方。
