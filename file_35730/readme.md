序号：35740
名称：卷柏丸
组成：卷柏（去根）2两，当归（洗，焙）2两，熟干地黄（洗，焙）1两半，川芎1两半，柏子仁（微炒，别研）1两半，香白芷1两，肉苁蓉（酒浸1宿，焙干）1两，牡丹皮1两，川椒（去目及闭口者，微炒）3分，艾叶（炒）3钱。
出处：《杨氏家藏方》卷十五。
主治：冲任本虚，血海不足，不能流通经络，致月事不调，妇女带下。
加减：None
功效：常服调和经脉，补暖元脏，润泽肌肤，长发去(黑干)，除头风，令人有子。
用法用量：每服50丸，空心、食前温酒米饮任下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中香白芷，《普济方》作“香附子”。
