序号：30984
名称：十香膏
组成：沉香半两（锉），檀香半两（锉），丁香半两（末），郁金香半两（锉），甘松香半两（锉），麝香1分（细研），熏陆香半两（细研），白胶香半两（细研），龙齿半两（细研），黄丹6两，麻油1斤，苏合香半两（锉），木香半两（末）。
出处：《圣惠》卷六十三。
主治：风毒疮肿，痈疽，疔赘，瘤瘿。
加减：None
功效：None
用法用量：用软帛上摊贴，日3度换之。
制备方法：上药先取沉香、檀香、郁金香、甘松香等5味，于油中浸7日，都入铛内，以少炭火温养5日后，以武火煎20-30沸，滤出香，用绵滤过，净拭铛，油都入铛内，下黄丹，以柳木篦不住手于火上搅，候色黑，滴水中如珠子，软硬得所，去火，将煎丁香等6味，入膏中搅300-500遍，合成，盛瓷盒内。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
