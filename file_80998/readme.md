序号：81014
名称：麝香丸
组成：大活蟾1个（巴豆10粒，勿损，入蟾口，养罐中候自死，破取巴豆，洗灌蟾里外，去肠胃切数段、研巴豆如面，同水2升，蟾入银锅，文武火煎烂去骨，入无灰酒煎），夜明砂1两，宣连2两，大芜荑2两（上药同膏煎如杨，入后药），朱砂1分，鹤虱（纸上炒青）半两，芦荟半两，麝半两。
出处：《幼幼新书》卷二十六引刘氏方。
主治：小儿疳。腹胀，多渴，频泻腥白脓血，或痢，四肢疼痛，黄瘦，疳虫咬心，常吐清水，不食。
加减：None
功效：None
用法用量：1岁儿初服1丸，5-7渐加至2丸，百日至十岁皆宜。夏月疳，泻下恶物立止。
制备方法：上为末，入前膏为丸，如绿豆大，晒干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
