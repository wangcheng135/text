序号：6681
名称：加味槐角丸
组成：槐角2两，生地2两，当归身2两，黄耆2两，川芎半两，阿胶半两，黄连1两，条芩1两，枳壳1两，秦艽1两，防风1两，连翘1两，地榆1两，升麻1两，白芷半两。
出处：《丹溪心法附余》卷十一。
主治：痔漏，及肠风下血。
加减：None
功效：None
用法用量：每服50丸，渐加至70-80丸、100丸，空心温酒或米汤送下。
制备方法：上为末，炼蜜为丸，或酒糊为丸，如梧桐子大。
临床应用：None
各家论述：本方以槐角、生地生血凉血为君；当归、川芎、黄耆、阿胶补虚为臣；以诸药为佐使。盖黄连泻心火，条芩凉大肠，枳壳宽大肠，秦艽去大肠风，防风为血证上使，连翘为血证中使，地榆为血证下使，而连翘又能散经络中火邪，地榆又能凉血，升麻升散火邪，又与白芷引诸药入大肠经络，夫痔漏，经络之病也。
用药禁忌：None
附注：None
