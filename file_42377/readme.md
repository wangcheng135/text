序号：42392
名称：青蒿煎丸
组成：青蒿汁2升，薄荷汁2升，生地黄汁2升，童便5升，朱砂1两（细研，水飞过），麝香半两（细研），桃仁5两（汤浸，去皮尖双仁，研令细），柴胡3两（去苗，为末），鳖甲5两（涂醋，炙令黄，去裙襴，为末）。
出处：《圣惠》卷三十一。
主治：热劳烦心，口干，皮肤枯燥，渐渐羸瘦。急劳，心肺积热，鼻口焦干，食欲无味，神昏欲睡，心胸胀满，两目多涩，四肢无力，足胫痠疼，腰脚拘急。
加减：None
功效：None
用法用量：每服20丸，以麦门冬汤送下，不拘时候。
制备方法：上件药，取青蒿汁等及小便，相和一处，先煎令稠，然后下桃仁以下诸药，更熬令稀稠得所，候可丸，即丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：《圣济总录》有地骨皮。
