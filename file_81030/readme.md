序号：81046
名称：露蜂房膏
组成：露蜂房半两，蛇蜕皮半两，玄参半两，黄耆半两（锉），蛇床人1分，杏仁1两（汤浸去皮尖，双仁细研），乱发半两，黄丹5两，黄蜡1两。
出处：《圣惠》卷六十六。
主治：风瘘。热毒气毒，结为瘰疬。
加减：None
功效：消肿化脓。
用法用量：每取帛上涂贴，每日1换。以愈为度。
制备方法：上药除黄丹、蜡、杏仁、乳发外，为粗末，以绵裹，用油2-3度浸1宿。别用油半斤，纳杏仁及乱发煎令发消尽，后下诸药，同煎10数沸，绵滤，更下于铛中，然后下黄丹及蜡，又煎6-7沸，用柳篦子急搅令匀，滴于水中不散，成珠子，即倾于瓷器中盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：蜂房膏（《圣济总录》卷一二六）。
