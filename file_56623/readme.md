序号：56638
名称：消水肿归气饮子
组成：苏叶（连嫩枝）1两2钱，大腹皮3钱，川木通4钱，茯苓皮2钱，姜皮2钱，陈皮2钱，桑皮2钱，桔梗2钱，地骨皮2钱，五加皮2钱，茯苓2钱，麦门冬2钱，甘草2钱，五味子2钱，草果2钱。
出处：《普济方》卷一九二。
主治：面虚浮肿，四肢肿大，甚者入腹胁，致胸满气喘，微嗽，（冫食）泄，眠卧不能。
加减：None
功效：None
用法用量：每服7-8钱，用水2钟，加生姜3片，煎至8分，食前、临卧连滓服，日进4服。肿消药尽，6服全安。
制备方法：上为末，分为6服。
临床应用：None
各家论述：None
用药禁忌：宜只吃白粥10日、半月、20日。大忌生冷、油腻、酒醋、咸鸡、鸭子、面食、硬饭、蔬菜。
附注：None
