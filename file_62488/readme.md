序号：62504
名称：梅觉春丹
组成：沉香半两，木香半两，安息香半两，荜澄茄半两，丁香半两，牛膝1两（去须，酒浸），仙灵脾3钱，滴乳3钱（另研），肉豆蔻半两（面裹煨），赤石脂1两（水飞），巴戟1两（去皮心），川水通半两（吹透气者），通草3钱（肥大者），头红花1两，茯苓1两（雪白者，去皮），泽泻半两，吴茱萸3钱，山茱萸3钱，覆盆子3钱，朱砂半两（水飞），肉苁蓉2两（肥大者，酒浸1日，漉出控干），蛇床子半两（脊尖者，酒浸1伏时、绢帛扭干，微炒），川续断1两（折断，有藤花者），莲花蕊1两（新者），五味子半两（炒），川楝子2两（去皮取肉，微炒黄），川山甲1两（脊尖者，酒浸2宿，火内炮干），莲子心半两，白术半两，八角茴香1两（入盐少许，同炒香为度），鹿茸1两（酥煮）。
出处：《医方类聚》卷一五三引《经验秘方》。
主治：None
加减：None
功效：利气去痰，消饮食，壮筋骨，驻容颜，添精神，身轻健。
用法用量：每服50丸，空心温酒送下，干物压之。次日，比及服药，先煮萝白葱汤蒸洗之，至晚再取30丸。
制备方法：上除乳香、安息香、朱砂别研外，余者一处为细末，与前3味和匀，以无灰酒打糊为丸，如梧桐子大，以小瓷瓶收贮，勿令泄气，服之必用火日。
临床应用：None
各家论述：None
用药禁忌：忌猪、羊血。
附注：None
