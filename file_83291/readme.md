序号：83309
名称：丁香楝实丸
组成：当归（去芦，锉碎）1两，附子（炮制，去皮脐，锉）1两，川楝子（锉碎）1两，茴香（炒）1两（上四味锉碎，以好酒三升同煮，酒尽为度，焙干作细末，每称药末1两，再入下项药），丁香2钱，木香2钱，全蝎13个，玄胡1两（上四味同为细末，入前药末内拌和）。
出处：《医学发明》卷一。
主治：肾肝受病，男子七疝，痛不可忍；妇人瘕聚、带下。
加减：None
功效：None
用法用量：每服30丸至100丸，空心、食前温酒送下。
制备方法：酒糊为丸，如梧桐子大。
临床应用：None
各家论述：凡疝气带下者，皆属于风，全蝎治风之圣药；茴香、川楝子皆入小肠经；当归、玄胡和血止痛；疝气、带下，皆积寒邪入小肠之间，故以附子佐之，丁香、木香为其引导。
用药禁忌：None
附注：酒煮当归丸（《活法机要》）。《景岳全书》引本方有没药五分。
