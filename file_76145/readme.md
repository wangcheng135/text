序号：76161
名称：脾肾双补丸
组成：人参（去芦）1斤，莲肉（去心，每粒分作8小块，炒黄）1斤，菟丝子（如法另末）1斤半，五味子（蜜蒸，烘干）1斤半，山茱萸肉（拣鲜红肉厚者，去核烘干）1斤，真淮山药（炒黄）1斤，车前子（米泔淘净，炒）12两，肉豆蔻10两，橘红6两，砂仁6两（炒，最后入），巴戟天12两（甘草汁煮，去骨），补骨脂（圆而黑色者佳，盐水拌炒，研末）1斤。
出处：《广笔记》卷二。
主治：肾泄。
加减：如虚而有火者，火盛肺热者，去人参、肉豆蔻、巴戟天、补骨脂。
功效：None
用法用量：每服5钱，空心、饥时各1次。
制备方法：上为细末，炼蜜为丸，如绿豆大。
临床应用：None
各家论述：《饲鹤亭集方》：脾肾两亏，阴阳不固，以致虚寒飨泄，腹痛泻痢，食少神倦，或酒色过伤，脏真无火，此丸有健脾暖肾之功，故曰双补。
用药禁忌：忌羊肉、羊血。
附注：None
