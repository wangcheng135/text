序号：72087
名称：小嘉禾散
组成：木香3钱，丁香3钱，丁皮3钱，巴戟5钱（去心），紫苏叶5钱，白茯苓5钱，苍术5钱（浸，炒），肉豆蔻5钱（煨），附子5钱（炮），沉香3钱，苦梗5钱（去芦），粉草5钱，茴香5钱（炒），山药5钱，白豆蔻仁5钱，扁豆5钱（炒）。
出处：《得效》卷十五。
主治：荣卫不调，血气虚弱。面色萎黄，四肢无力，手足倦怠，盗汗并出，皮肉枯瘁，骨肉羸瘦，饮食不进，日渐卧床；妇人病后不能调理，变成崩漏。腰腿沉重，痛连脐腹，小便白浊。
加减：止泻，加黑豆（炒）；止痢，加粟壳（蜜炒）。
功效：None
用法用量：每服3钱，水1盏半，加生姜3片、大枣2个，水煎，温服。
制备方法：上锉散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
