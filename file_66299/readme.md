序号：66315
名称：随补羊肉羹
组成：羊肉不拘多少（或半斤或4两）。
出处：《会约》卷二。
主治：一切体弱神昏，不爱饮食，倦怠无力。
加减：None
功效：None
用法用量：照常加盐加酱烹调，如命门火衰，或脚膝冷，成身体冷，或腹冷腹痛，大便溏泄，不思饮食，或食不化，每两羊肉用熟附子1钱同煮，或食或楝去不食。若以附子研末同煮更妙.虚弱之症，须多用数会。如气虚者，四肢无力，神气短少，用蜜制黄耆煎水煮羊肉。如血虚者，唇白肤枯，或失血之后，或妇人生产之后，或月水之后，而色淡血少，用当归煎水煮羊肉。如脾土亏弱，不能多食，或泄泻，或瘦削，此等证候，小儿最多，大人亦不少，用淮山药炒黄，或稍加熟附子，共研细末敷羊肉服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
