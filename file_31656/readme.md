序号：31665
名称：羌活饮子
组成：羌活（去芦头）2钱半，独活（去芦头）2钱半，川芎2钱半，柴胡（去苗）2钱半，前胡（去芦头）2钱半，细辛（去叶土）2钱半，白蒺藜（炒，去刺）2钱半，麦门冬（去心）2钱半，山药2钱半，升麻2钱半，紫苏叶2钱半，黄耆（蜜炙）2钱半，乌梅（去核）半两，防风（去芦头）半两，枳壳（去瓤，麸炒）半两，蔓荆子半两，藁本（去土）半两，荆芥穗半两，甘草（炙）半两，桑白皮（炙）半两，干葛1两。
出处：《杨氏家藏方》卷二。
主治：风毒上攻，头面发热，颊赤唇焦，眼涩，鼻出热气，项背拘急。
加减：None
功效：None
用法用量：上(口父)咀。每服3钱，水1盏半，加生姜3片，薄荷5叶，煎至8分盏，食后去滓温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
