序号：35513
名称：京三棱煎丸
组成：京三棱8两（炮，锉），陈橘皮8两（汤浸去白瓤，焙），黑三棱4两（炮，锉），桃仁4两（汤浸，去皮尖双仁，研如膏），以上4味，除桃仁外，捣罗为末，用好酒5升，于锅中，以慢火煎，次下桃仁膏，熬，如稀饧，入后药末：槟榔2两，诃黎勒2两（煨，用皮），枳壳4两（麸炒微黄，去瓤），木香3两，硇砂1两（研入），鳖甲2两（涂醋炙微黄，去裙襴），硫黄2两（细研，水飞过），附子2两（炮裂，去皮脐），干姜3两（炮裂，锉）。
出处：《圣惠》卷二十八。
主治：虚劳癥瘕，心腹疼痛，胸膈不利。
加减：None
功效：None
用法用量：每服30丸，空腹及晚食前以生姜汤或温酒送下。
制备方法：上为末，入研了硇砂、硫黄等，重研令匀，入前药内为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌苋菜生冷。
附注：三棱煎丸（《鸡峰》卷九）。
