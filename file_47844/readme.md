序号：47859
名称：神功丸
组成：大黄3两，人参半两，诃子（泡，去核）2两，麻仁（另研）5两。
出处：《脚气治法总要》卷下。
主治：三焦气壅，心腹痞闷，六腑风热，大便不通，津液内枯，大肠干涩，里急后重，或下鲜血，痰唾稠粘，风气下流，腰疼脚重，脐下胀痛，溺赤如金。
加减：None
功效：疏解秘滞，调三焦，开胃口。
用法用量：每服30丸，温水送下，不拘时候。以通为度，未通更加丸数。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：麻仁丸（《圣济总录》卷一五七）、神效麻仁丸（《外科精要》卷中）、各神丸（《普济方》卷三十九）、神功麻仁丸（《普济方》卷二八三）。
