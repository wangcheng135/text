序号：47590
名称：神仙太一火煅丹
组成：圣知子（煎，掠取白花，焙干，研）2斤，绛矾（细研）5两，消石5两（细研，分作10份，每1度煅入1份），麻油100两（分作10份，每1度煅入1份）。
出处：《圣济总录》卷二○○。
主治：None
加减：None
功效：轻身延年，久服众疾皆愈，髭发如漆，筋骨轻健，耳目聪明。
用法用量：每服10丸，渐加至20丸，空心冷水送下。
制备方法：上先将矾盐入消一份拌和研匀，用麻油10两和了，入固济5升瓶内，将瓦盖瓶，以炭火10斤煅，焰绝炭尽为度；取出细研，更入消一份，油一份，准前用炭10斤煅尽，如此十度，计用炭100斤足；后取药捣罗为末，旋取少许，细研，汤浸蒸饼为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
