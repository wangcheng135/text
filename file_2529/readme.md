序号：2529
名称：双芝丸
组成：熟干地黄（后取末）、石斛（去根，酒炙）、五味子（焙）、黄耆（锉）、肉苁蓉（酒浸）、牛膝（酒浸）、杜仲（蜜水浸，炮）、菟丝子（酒浸3日，炒），麋鹿角霜半斤，沉香3钱，麝香2钱（研），人参1两，白茯苓（去皮）1两，覆盆子1两，干山药1两，木瓜1两，天麻（酒浸）1两，秦艽1两，薏苡仁2两（炒）。
出处：《宣明论》卷十二。
主治：诸虚。
加减：凡年50岁以上，加入黑附子(以青盐汤醮泡)，鹿角2大对(去顶3指)，硫黄半斤(浑用。以上用些油釜中以水同煮令微沸，勿大急甚，水耗只旋添温水，须用水以备添之，炼令角胶汁出尽，其角如霜，以手捻如腻粉，乃盛之取用，勿令秽污也)。
功效：补精气，填骨髓，壮筋骨，助五脏，调六腑，久服驻颜不老。
用法用量：每服20丸至30-40丸，温酒送下；盐汤、米饮亦可。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
