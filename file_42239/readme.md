序号：42254
名称：青娥丸
组成：胡桃（去皮、膜）20个，蒜（熬膏）4两，破故纸（酒浸，炒）8两，杜仲（去皮，姜汁浸，炒）16两。
出处：《局方》卷五（宝庆新增方）。
主治：肾虚为风寒温邪所伤，或坠堕伤损，气滞不散所引起的腰痛，头晕耳鸣，溺有余沥，妇女白带。
加减：None
功效：温肾暖腰，益精养血。壮筋骨，活血脉，乌髭须，益颜色，壮脚力。滋补下元，滋阴，齿落再生，返老还童。
用法用量：每服30丸，空心温酒送下，妇人淡醋汤送下。
制备方法：上为细末，蒜膏为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：青蛾丸（《普济方》卷一五四引《如宜方》、青娥不老丸（《明医指掌》卷九）、青娥不老丹（《嵩崖尊生》卷十三）。
