序号：51502
名称：追风应痛丸
组成：威灵仙4两，狗脊（去毛）4两，何首乌6两，川乌（炮，去皮脐）6两，乳香（研）1两，五灵脂（酒浸，淘去沙石）5两半。
出处：《局方》卷一（续添诸局经验秘方）。
主治：一切风疾，左瘫右痪，半身不遂，口眼斜，牙关紧急，语言蹇涩，筋脉挛急，百骨节痛，上攻下注，游走不定，腰痛沉重，耳鸣重听，脚膝缓弱，不得曲伸，步履艰难，遍身麻痹，皮肤顽厚；及妇人血风攻注，身体疼痛，面浮肌瘦，口苦舌干，头旋目眩，昏困多睡；或皮肤瘙痒，瘾疹生疮；暗风夹脑，偏正头疼。
加减：None
功效：轻身体，壮筋骨，通经活络，除湿祛风。
用法用量：每服15丸，加至20丸，食前麝香温酒吞下；只温酒亦得。
制备方法：上为末，酒糊为丸。
临床应用：None
各家论述：None
用药禁忌：孕妇不可服。
附注：None
