序号：40005
名称：育婴金丹
组成：葛根2两，黄芩4两，前胡4两，桔梗4两，防风2两，柴胡2两，木通4两，甘草4两，花粉4两，制半夏4两，枳壳（麸炒）2两，芥穗4两，人参（去芦）1两，神曲（麸炒）4两，苏叶2两。
出处：《全国中药成药处方集》（天津方）
主治：小儿感冒发烧，头疼头晕，痰喘气促，风火顿嗽，呕吐痰涎，口燥舌干，咽喉肿痛，小水赤黄，隐疹不透。
加减：None
功效：清肺，疏风，化痰。
用法用量：每服1丸，周岁以内酌减，白开水化服。
制备方法：上为极细末，每细粉2斤15两兑朱砂面1两、羚羊粉3钱，炼蜜为丸，1钱重，每斤丸药用朱砂面3钱上衣，蜡皮或蜡纸筒封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
