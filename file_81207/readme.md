序号：81223
名称：麝香天麻丸
组成：麝香（研）1钱半，天麻半两，天南星（炮）半两，白附子（炮）半两，羌活（去芦头）半两，赤茯苓（去黑皮）半两，干蝎（去土，炒）半两，丹砂（研）半两，防风（去叉）半两，桂（去粗皮）半两，蝉蜕（洗、炒）半两，乌蛇（酒浸，去皮骨、炙）2两，铅霜（研）1分。
出处：《圣济总录》卷十七。
主治：风头眩目黑，肩背拘急，恍惚忪悸，肢节疼痛。
加减：None
功效：None
用法用量：每服20丸，温酒送下，荆芥汤亦得，不拘时候。
制备方法：上药除研者外，捣罗为末。再同研匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
