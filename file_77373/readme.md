序号：77389
名称：雄朱丹
组成：雄黄半两，朱砂半两，麝半两，腻粉半两，白附子1两，半夏（汤洗7次）1两，天南星（炮）1两，川乌头1两，附子（各生去皮脐）1两，干蝎1两，羌活1两，天麻1两，川芎1两，肉桂（去粗皮）1两，白僵蚕1两，木香1两，白鲜皮1两，乌蛇1两，花蛇（酒制，炙）1两，巴豆（净肉薄荷汁煮50、70沸，去油）。
出处：《幼幼新书》卷十引《灵苑方》。
主治：小儿惊风，及冷气，宿食不消，大小便不通，急风，瘫痪，酒食伤，风气、血气，大肠秘涩，白痢、赤痢、赤白痢，风眼，头风，风疹，头痛伤寒，疳积。
加减：None
功效：None
用法用量：茶、酒下3丸，常服；冷气，姜汤送下；宿食不消，橘皮汤送下；大小便不通，甘草豆淋酒送下；急风，薄荷酒送下；瘫痪，豆淋酒送下；酒食伤，生姜汤送下；风气，茴香酒送下；血气，荆芥酒或醋汤送下；大肠秘涩，生姜汤送下；白痢，椒汤送下；风眼，淡竹叶汤送下，坐间便退；头风，槐枝汤送下；风疹，蜜酒送下；赤白痢，二宜汤送下；头痛伤寒，盐汤送下；赤痢，甘草汤送下；小儿疳，米饮送下；风眼，古井水煎淡竹叶汤送下：小儿惊风，薄荷乳汁送下。
制备方法：上为末，米粥为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
