序号：73870
名称：硼砂煎丸
组成：硼砂2两（不夹石者，细研，以酒醋各半升熬如膏），干漆1两（捣碎，炒令烟出），桂心1两，汉椒1两（去目及闭口者，微炒去汗），干姜半两（炮裂，锉），附子1两（炮裂，去皮脐），槟榔1两，川大黄2两（锉碎，微炒）。
出处：《圣惠》卷四十八。
主治：伏梁气，久积在心下，横大如臂，发歇疼痛，胸下拘急，腹胁满闷。
加减：None
功效：None
用法用量：每服15-20丸，空心温酒送下。
制备方法：上为细末，入硼砂膏中，更入蒸饼少许，和溶为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《医方类聚》引作“硇砂煎丸”。方中硼砂，《医方类聚》引作“硇砂”。
