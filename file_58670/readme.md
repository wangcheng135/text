序号：58685
名称：秘传茱萸内消丸
组成：吴茱萸（半酒半醋浸1宿，焙干）1两，山茱萸（蒸，去核）1两，马兰花（醋浸，焙）1两，川楝子（蒸，去皮核）1两，官桂1两，黑牵牛（炒，取末）1两，舶上茴香（用盐炒）1两，延胡索（略炒）1两，橘皮1两，青皮（去白）1两，海藻（浸，洗去咸，焙）1两，桃仁（浸，去皮，炒）半两，白蒺藜（炒，杵去刺）半两，木香半两。
出处：《直指》卷十八。
主治：肾虚为邪所袭，留伏作痛，阴(疒颓)偏大，或生疮出黄水。
加减：None
功效：None
用法用量：每服40丸，食前温酒、盐汤任下。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
