序号：8542
名称：归附地黄丸
组成：当归（酒洗）3两，川芎1两，白芍（酒炒）2两，熟地黄（酒蒸）1两，香附子（童便浸，炒）2两，陈皮1两半，黄柏（去皮，童便浸3日，晒干）1两半，知母（去毛）1两半（酒浸，晒干），五味子1两半，苍术（米泔浸，炒）2两，牡蛎（煅）5钱，椿根皮（酒炒）2两半（一方有白葵花；一方有山茱萸，酒蒸，去核，无五味）。
出处：《寿世保元》卷七。
主治：妇女赤白带下。
加减：人虚，加人参、白术。
功效：None
用法用量：每服50丸，空心淡盐汤下，后用于物压之。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌葱白、萝卜、胡椒、煎炒、发热之物。
附注：None
