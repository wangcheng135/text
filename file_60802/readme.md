序号：60817
名称：通瘀煎
组成：生蒲黄3钱，五灵脂3钱，川郁金1钱半，小枳实1钱半（炒），白术炭1钱半，建泽泻1钱半，西赤芍1钱半，桃仁泥3钱，明琥珀3钱。
出处：《医略六书》卷二十三。
主治：血瘀成臌，脉涩滞者。
加减：None
功效：None
用法用量：水煎，去滓，温服。
制备方法：None
临床应用：None
各家论述：血瘀不消，脾失健运之职，不能输化精微，故浊阴窒塞而胀满有加，是为血臌。蒲黄破瘀血，通经络；灵脂破瘀血，降浊阴；桃仁破瘀润燥；赤芍破瘀泻火；枳实消胀满；术炭健脾气；郁金调气开郁结；泽泻分清阳；琥珀散瘀血，以通渗道也，使瘀化气调，则冲脉清和，而肝脾气化，窒塞顿开，何腹脉之不退哉？
用药禁忌：None
附注：None
