序号：7454
名称：生肌膏
组成：麻油1斤（化胎发1团，熬滴水成珠），龙骨（煅）1钱（研细末），黄占1钱（研细末），熟猪油1钱（研细末），赤石脂1钱（研细末），乳香1钱（研细末），没药1钱（研细末），轻粉1钱（研细末），象皮（煅）1钱（研细末）。
出处：《串雅内编》卷二。
主治：疮疡疖肿，一切溃疡，腐肉不尽，新肉不生者。
加减：None
功效：生肌收口。
用法用量：摊贴，1日1换。仍以猪肉汤洗3-4次即平。
制备方法：入油内搅匀成膏。
临床应用：None
各家论述：《串雅内编选注》：本方用龙骨、赤石脂收敛渗湿；象皮生肌长肉；乳香、没药、黄蜡为膏，以助收敛生肌之力。
用药禁忌：None
附注：《串雅内编选注》：一般局部常规消毒，再上此药即可，不用猪肉汤洗亦可。
