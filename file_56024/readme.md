序号：56039
名称：桑椹河车丸
组成：河车2具（酒净，焙干），鹿茸1对（酥炙），黑驴肾（连腰子肾子；切片，酥炙）4具，黄狗肾（连腰子肾子，酒煮焙干）10具，熟地（9蒸晒）8两，枸杞（酒蒸）8两，生首乌8两，巴戟天（酒蒸）4两，破故纸（合桃拌炒）4两，山药（盐水炒）4两，萸肉4两，骨碎补（炒）4两，鱼鳔（蛤粉炒）4两，五味子4两，菟丝子（酒煮）4两，仙茅（米泔浸3次，去皮）4两，肉苁蓉（去鳞肠）4两，锁阳4两，茯苓4两，人参2两。
出处：《惠直堂方》卷一。
主治：None
加减：None
功效：补虚。
用法用量：每服5钱，空心清汤送下。
制备方法：上为末，桑椹熬膏，加炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
