序号：25005
名称：如圣散
组成：羌活、防风、川芎、白芷、柴胡、芍药、甘草、当归、乌药、半夏、黄芩。
出处：《赤水玄珠》卷十八。
主治：刚柔二痉，头摇口噤，身反张，手足挛搐，头面赤，项强急与瘛疭。
加减：有汗是柔痉，加白术、桂枝；无汗是刚痉，加麻黄、苍术。口噤咬牙者，如大便实者，入大黄利之。
功效：None
用法用量：水2钟，加生姜3片，煎之，临用入姜汁、竹沥温服。
制备方法：None
临床应用：None
各家论述：《医方集解》：此足太阳厥阴药也，羌防芎芷柴胡甘草，辛甘以发散风邪；用乌药者，治风须顺气也；用归芍者，治风先治血也；用半夏竹沥姜汁者，风必挟痰也；用黄芩者，风必生热也。柔痉加白术、桂枝，有汗欲其无汗；刚痉加苍术，麻黄，无汗欲其有汗；口齿属阳明，阳明实则口噤咬牙而便秘，故加大黄以泄胃热也。
用药禁忌：None
附注：如圣饮（《明医指掌》卷七）、如圣汤（《证治宝鉴》卷一）。按：《明医指掌》有人参。
