序号：48272
名称：神香苏合丸
组成：苏合香油2两，金银香1两，公丁香2两，广木香2两，贡沉香2两，生香附2两，犀角尖2两，飞朱砂2两，滴乳香（制）1两，生于术2两，梅冰片5钱，麝香5钱。
出处：《全国中药成药处方集》（杭州方）。
主治：小儿急惊，抽搐吐乳，目窜上视，牙关紧闭，痰涎潮壅，危急诸症。并治男妇中风痰厥，昏迷僵仆，寒证气闭，霍乱吐泻，时气瘴疟。
加减：None
功效：避秽开窍，祛痰行气。
用法用量：每服1丸，小儿酌减。中风昏迷，用薄荷汤化服；霍乱吐泻，用淡姜汤化服；惊风抽搐，用钩藤灯心汤化服，或用开水化服亦可。若瘟疫流行时，绢袋盛佩胸际，可避秽毒。
制备方法：上为细末，将金银香酒烊化成膏，和苏合香油，加炼白蜜打丸，每丸潮重5分，蜡壳封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
