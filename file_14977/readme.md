序号：14984
名称：艾附女珍丸
组成：香附5两（分4份：1童便，1米醋，1人乳，1盐酒浸），蕲艾（醋煮）2两，当归2两，川芎1两半，白芍1两半，熟地黄（酒蒸）1两半，黄芩1两半，阿胶（酒蒸）1两，臭椿根皮1两。
出处：《简明医彀》卷七。
主治：妇人气盛血衰，经期不准，或前或后，紫多淡少，赤白带下，崩漏淋沥，面黄肌瘦，四肢无力，倦怠嗜卧，精神短少，目暗耳鸣，头眩懒言，五心烦热，咽干口燥，夜寐不安者。
加减：None
功效：None
用法用量：每服100丸，空心米汤送下。
制备方法：上为末，捣地黄、阿胶和匀，加醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
