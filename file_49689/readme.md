序号：49704
名称：茯苓汤
组成：茯苓2两，防己1两半，橘皮1两，玄参1两，黄芩1两半，泽泻1两半，杏仁2两半（去尖皮），白术1两半，大豆1升半，郁李仁2两半，桑白皮2两半，泽漆叶（切）1升，猪苓1两半。
出处：《外台》卷七引《广济方》。
主治：鼓胀上下肿，心腹坚强，喘息气急，连阴肿，坐不得，仍下赤黑血汁，日夜不停者。
加减：咳者，加五味子2两。停2日服1剂。
功效：None
用法用量：以水1斗，先煮桑白皮、大豆、泽漆叶，取5升，去滓，澄去下淀，纳诸药，煎取2升，绞去滓，分3服。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：忌酢物、桃、李、雀肉、热面、蒜、炙肉、粘食、油腻。
附注：None
