序号：60419
名称：通关散
组成：旌德乌头4两（破皮，旌德有芦头肌白者），藁本半两，防风半两，当归半两，白芷半两，天南星半两，川芎半两，干姜半两，雄黄（细研）半两，桂半两（并生，勿近火）。
出处：《苏沈良方》卷二。
主治：诸中风伤寒。
加减：瘫痪，加牛黄、麝香。
功效：None
用法用量：每服1字或半钱，葱酒调下，亦可以薄荷酒调下。小儿药量减半。
制备方法：上为末。
临床应用：①中风偏瘫曾在江南，见市门有卧者，乃客贩因病偏风，医之，遂至病困，为邸家所委，时伯氏为邑，使人（臼艹）到令舍，调药（即通关散）饮之，又与10服。数日伯氏出，市有一人，扶倚床而呼曰：昔日卧者，今能扶榻而行矣，药尽，愿少继之。伯氏又与10服，服讫能起。②痉挛一吏病疮而挛，逾岁月卧矣。伯氏与散（即通关散）2钱匕，为8服。吏谬以为1服，服已，僵眩呕吐，几困将殆，数日疮挛悉除。
各家论述：None
用药禁忌：None
附注：None
