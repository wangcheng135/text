序号：3354
名称：天蓼木浸酒
组成：天蓼木10斤（细锉，以水1硕，煎至5斗，用此水造酒，须及5斗，熟后浸后药），石斛半斤（去根），防风半两（去芦头），地骨皮半斤，桑根白皮半斤，生地黄半两，远志（去心）半斤，牛膝（去苗）半斤，菟丝子半斤，槐子半斤，白蒺藜（微炒，去刺）半升，乌蛇1条（酒浸，炙令黄），乌鸡粪5合（炒黄）。
出处：《圣惠》卷二十三。
主治：中风，偏枯不遂，失音数年。
加减：None
功效：None
用法用量：上锉细，以生绢袋盛，入天蓼木酒中，密封闭，冬月21日，春夏14日。量性饮之，令常有酒容，如觉热，即减之。眼鼻及面口偏者，7日取正；手脚不遂者，半月内愈；失音，服之即语。

用药禁忌：None
