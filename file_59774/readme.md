序号：59789
名称：调经济阴丸
组成：生地5两，山药3两，茯苓3两，香附3两，当归3两，白芍3两，山萸肉2两，泽泻2两，杜仲2两，地骨皮2两，丹皮2两，青蒿1两3钱，蕲艾茸1两3钱，川芎1两3钱，知母1两3钱，黄柏1两3钱，牛膝1两3钱，鳖甲1两3钱。
出处：《活人方》卷二。
主治：阴虚内热，热郁于经脉之中，久而不清，遂成骨蒸劳热，火炎金燥，水涸精枯，先致咯血、吐血、咳嗽、音哑，渐及自汗盗汗，虚寒虚热，冲任不和，天癸闭绝。
加减：None
功效：开郁清热，滋阴济火。
用法用量：每服3钱，早、晚空心白汤下。
制备方法：上为细末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
