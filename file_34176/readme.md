序号：34185
名称：牡丹煎丸
组成：延胡索半两，缩砂仁半两，赤芍药1两，牡丹皮1两，山茱萸半两，干姜（炮）半两，龙骨（细研，水飞）2两，熟干地黄（酒浸）2两，槟榔2两，羌活2两，藁本（去土）1两，五味子1两，人参1两，白芷1两，当归（去芦，酒浸）1两，干山药1两，泽泻1两，续断（细者）1两，肉桂（去粗皮）1两，白茯苓1两，白术1两，附子（去皮脐）1两，木香1两，牛膝（去苗，酒浸1宿，焙）1两，萆薢（炮，为末，炒熟）1两，石斛（去根，酒浸）3两。
出处：《局方》卷九。
主治：妇人冲任本虚，少腹挟寒，或因产劳损，子脏风寒，搏于血气，结生瘕聚，块硬发歇，脐腹刺痛，胁肋紧张，腰膝疼重，拘挛肿满，背项强急，手足麻痹，或月水不调，或瘀滞涩闭，或崩漏带下，少腹冷疼，寒热盗汗，四肢痠痛，面色萎黄，多生（黑干）（黑曾），羸乏少力，心多惊悸，不欲饮食。
加减：None
功效：None
用法用量：每服20-30丸，空心、食前以温酒或醋汤送下，日2次。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：妊娠不宜服。
附注：None
