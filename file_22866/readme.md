序号：22874
名称：观音全蝎散
组成：黄耆1两，人参1分，木香1两，炙草1两，石莲肉1两（炒），扁豆1两（炒），白茯苓1两，白芷1两，全蝎1两，防风1两，羌活1两，天麻2两。
出处：《活幼口议》卷十四引东汉王氏方。
主治：小儿因吐而传为慢惊风者。
加减：None
功效：清神固气，补虚益脉，开胃止吐，生胃气，截风定痫。
用法用量：每服半钱或1钱，用大枣半个，水1小盏，煎至半盏，不拘时候服。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：观音散（《活幼口议》卷十四）、全蝎散（《永乐大典》卷九八一引《如宜方》）、全蝎观音散（《医统》卷八十九）。《医统》本方用法：上为末，每服二钱，用生姜、大枣汤调服或煎服。
