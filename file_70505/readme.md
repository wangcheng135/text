序号：70521
名称：紫金丹
组成：川乌头（炮）1两，草乌头（炮）1两，五灵脂半钱，木鳖子（去壳）半钱，骨碎补半钱，威灵仙半钱，金毛狗脊半钱，自然铜（醋淬7次）半钱，防风半钱，地龙（去土）半钱，乌药半钱，青皮（去白）半钱，陈皮（去白）半钱，茴香半钱，黑牵牛半钱，乳香2钱半，没药2钱半，红娘子2钱半，麝香2钱半，禹余粮石（醋炒）4两。
出处：《医学发明》卷三。
主治：打仆损伤，伤折疼痛不可忍。
加减：None
功效：定痛接骨。
用法用量：每服10-20丸，温酒送下，病在上食后，病在下食前。
制备方法：上为细末，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
