序号：59073
名称：积块丸
组成：京三棱1钱2分，莪术（各用醋煨）1钱2分，自然铜1钱2分，蛇含石（各烧红，醋淬7次）2钱，雄黄1钱2分，蜈蚣（全用，焙燥）1钱2分，辰砂8分，木香1钱半，铁华粉（用糯米、醋炒）1钱，芦荟4钱，天竺黄4钱，阿魏4钱，全蝎（洗，全用，焙干）4钱，沉香8分，冰片5分。
出处：《赤水玄珠》卷五。
主治：癥瘕积聚癖块，一应难消难化，腹中饱胀，或虫积疼痛。
加减：None
功效：None
用法用量：或服7-8分，重者1钱，五更以酒送下。块消即止，不必尽剂。
制备方法：上为极细末，用雄猪胆汁炼为丸（黑狗胆汁尤妙），如梧桐子大。
临床应用：臌胀《孙氏医案》：汪氏妇，年仅三八，经不行半载，腹大如斗，坚如石，时或作痛，里医尽技以治，月余弗瘳。予诊其脉，两关洪滑鼓指，按之不下，乃有余之候也。询其致病之源，因越食冷物，积而渐成臌胀。前任事者，误作气虚中满治之，胀而欲裂。乃用积块丸三下之，而胀消积去。后以丹溪保和丸调养一月而愈。
