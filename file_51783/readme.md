序号：51798
名称：钟乳补肺汤
组成：钟乳（碎如米粒）3两，桑白皮3两，麦门冬（去心）3两，白石英（碎如米粒）2两，人参（去芦）2两，五味子（拣）2两，款冬花（去梗）2两，肉桂（去粗皮）2两，紫菀（洗去土）2两。
出处：《局方》卷四。
主治：肺气不足，咳嗽上气，胸满上迫，喉咽闭塞，短气喘乏，连唾不已，寒从背起，口中如含霜雪，语无音声，甚者唾血腥臭，干呕心烦，耳闻风雨声，皮毛瘁，面色白。
加减：None
功效：None
用法用量：每服4钱，以水2盏，加生姜5片，大枣1枚（擘破），粳米30余粒，同煎至1盏，用绵滤去滓，食后温服。
制备方法：上除白石英、钟乳外，同为粗末，与白石英等同拌令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：补肺汤（《三因》卷八）。
