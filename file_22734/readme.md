序号：22742
名称：更生丸
组成：茯苓4分（若不消食，3分加1），菖蒲4分（若耳聋，3分加1），山茱萸4分（若身痒，3分加1），栝楼根4分（若热渴，3分加1），菟丝子4分（若痿泄，2分加1），牛膝4分（若机关不利，加1倍），赤石脂4分（若内伤，3分加1），干地黄7分（若烦热，3分加1），细辛4分（若目茫茫，3分加1），防风4分（若风邪，3分加1），薯蓣4分（若阴湿痒，3分加1），续断4分（若有痔，加1倍），蛇床子4分（若少气，3分加1），柏实4分（若少力，加1倍），巴戟天4分（若痿弱，3分加1），天雄4分（炮，若有风，3分加1），远志皮4分（惊恐不安，3分加1），石斛4分（若体疼，加1倍），杜仲4分（若绝阳腰痛，3分加1），苁蓉4分（若冷痿，加1倍）。
出处：《外台》卷十七引《素女经》。
主治：男子五劳七伤，阴衰消小，囊小生疮，腰背疼痛，不得俯仰，两膝膑冷，时时热痒，或时浮肿，难以行步，目风泪出，远视茫茫，咳逆上气，身体痿黄，绕脐弦急，痛及膀胱，小便尿血，茎痛损伤，时有遗沥，汗衣赤黄，或梦惊恐，口干舌强，渴欲饮水，得食不常，或气力不足，时时气逆，坐犯七忌，以成劳伤。
加减：None
功效：None
用法用量：每服3丸，食前，日3次。不知渐增，以知为度。亦可散服，以清粥饮服方寸匕，7日知，10日愈，30日余气平，长服老而更少。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：白茯苓丸（《圣惠》卷二十六）。
