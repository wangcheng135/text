序号：39181
名称：炙肝散
组成：石决明（洗）4两，谷精草（洗）4两，皂荚（炙，去皮子）1分，甘草（炙，锉）2两，木贼（锉）5两，黄芩（去黑心）5两，苍术（米泔浸7日，切，焙）半斤。
出处：《圣济总录》卷一一一。
主治：外障赤肉，翳膜遮障不明。
加减：None
功效：None
用法用量：每用獖猪肝1叶，去筋膜，切数缝，掺药末5钱，分于缝内，仍掺盐1钱合定，用旋斫湿柳枝3-4条搁起，慢火炙香熟，早晨空心冷吃尽，续吃冷饭1盏压之。仍于三里穴灸2-7壮，3日后有泪下为验，7日翳膜必退，每旦用新水漱口。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
