序号：79458
名称：摩顶油
组成：莲子草（5-6月收）3升，栀子叶3升，生麦门冬3升，生地黄3升，吴蓝（上5味并捣取汁）3升，连翘1两，秦艽（去苗土）1两，甘草（锉）1两，防风（去叉）1两，细辛（去苗叶）1两，地骨皮1两，大青1两，紫草茸1两，紫苏子叶1两。
出处：《圣济总录》卷十五。
主治：脑风头旋，恶心昏闷，发歇不定。
加减：None
功效：None
用法用量：每用时，量多少摩顶上。
制备方法：上药除汁外，为粗末，用绢袋盛，同五味药汁煎减半，去滓澄清后，入生麻油1升和匀，重煎又减半，收入瓶内14日，细研马牙消4两搅匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
