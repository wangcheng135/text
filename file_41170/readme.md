序号：41185
名称：金主杖鹿角霜丸
组成：鹿角霜12两，天门冬末（净）2两，麦门冬末（净）2两，生地末（净）2两，熟地末（净；俱如法制）2两，厚黄柏4两（先用酒浸1宿，焙干，又以盐水浸1宿，焙干，碾为细末，净）。
出处：《万氏家抄方》卷四。
主治：None
加减：冬，加干姜1钱5分（炒黑）。
功效：益力气，补精髓，强阳壮阴，固精补肾，滋血补脑，济阴阳，固元气，久服延年悦色，面如童少。
用法用量：每服50丸，空心温酒送下，或盐汤亦可。
制备方法：共前药末，搅匀，外用蜜12两炼，滴水成珠，入酒化鹿角霜4两，待温和前药搜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
