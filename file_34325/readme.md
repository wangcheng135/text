序号：34334
名称：万金丸
组成：安息香半两，乳香半两，丁香半两，木香半两，沉香半两（锉），无食子半两，肉豆蔻半两（去壳），当归半两（切，焙），麒麟竭半两，没药半两，密陀僧半两（煅），阿魏半两，巴豆3分（去皮心膜，醋煮），砒霜（入绿豆半两同研）3分。
出处：《圣济总录》卷一四三。
主治：肠风痔疾，远年不愈，疼痛不可忍者。
加减：None
功效：None
用法用量：每服1丸，空心冷茶咽下。
制备方法：上14味，先将乳香、安息香、阿魏、麒麟竭、没药等细锉，水少许浸1宿，来日细研如膏，余药捣罗为末，与膏同研为丸，如黄米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
