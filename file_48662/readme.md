序号：48677
名称：神符玉粉丹
组成：水银2两，黑铅1斤。
出处：《圣惠》卷九十五。
主治：None
加减：None
功效：久服延年驻颜。
用法用量：每日1丸，空心津送下。若养至1年，火力与小还丹同。
制备方法：于一新铁铛中，销铅成水，以白矾末2钱，入一小竹筒内，当铛中心下之，看沸定，即以小竹管，盛水银入铛中，送令到白矾上，以火养铅，常令成汗，候5日后，必有物出，向铅面上如金蚁子，即以物收之，候尽即止，秤知两数，研为末，入硫黄等分，结成砂子，更研如粉，入瓶密固济，候干，常以火4两，灰厚3寸，养至60日，沸开，依前取出，细研入瓶，更养60日，看紫色，即更固济，以火煅令通赤，待冷取出，以浓甘草汤沃之，候干，细研如粉，以饭和丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
