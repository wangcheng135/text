序号：47549
名称：神仙一井金丸
组成：牛膝3两（酒浸1宿，焙），肉苁蓉3两（酒浸1宿，切，焙），川椒（炒）2两，白附子（炮）2两，附子（炮，去皮脐）2两，乌药2两，何首乌（同黑豆半升煮，豆熟为度，去豆不用）2两，木鳖子（去壳）1两，萆薢（黑豆半升同煮，豆熟为度，去豆不用）1两，舶上茴香1两，防风（去芦头）1两，白蒺藜（炒，去刺）1两，覆盆子1两，绵黄耆（蜜炙）1两，赤小豆1两，骨碎补（去毛）1两，金毛狗脊（去毛）1两，全蝎（去毒，微炒）1两，五味子1两，青矾（火飞，枯尽）1两，地龙（去土，炒）1两，天南星（炮）1两，羌活（去芦头）1两。
出处：《杨氏家藏方》卷九。
主治：None
加减：None
功效：补益真元，大壮腰脚；久服髭鬓不白，牙齿牢壮，美进饮食，明目聪耳，行步轻快。
用法用量：每服50丸，煎五味子酒送下，空心、食前、日午各1服。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：乌发：余在淳安，主簿李渊云乃祖通判公，少服一井金丹，至老发不白。后在都城访杨五倅，问先和王晚年发不白所服何药？答曰：某未尝知。遂向老药童叩之，云：先和王常服一井金丹。后卢陵见前柳守赵鼎，六十余年，髭发皆不白，众以为润泽者，仆仔细观之，非是染者，渠云：自然如此。继过豫章，其人作酒官，托邱（亻辛）叔献询叩，渠云：大人平生，只服一井金丹，方知此药之妙。
各家论述：None
用药禁忌：None
