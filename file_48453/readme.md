序号：48468
名称：神效鹿胎丸
组成：萸肉5钱，萆薢5钱，熟地5钱，生地5钱，寸冬5钱，五味5钱，小茴5钱，故纸5钱，盆子5钱，鹿胶5钱，杜仲5钱，怀牛膝5钱，青盐5钱，柏仁5钱，归身5钱，巴戟5钱，远志5钱，锁阳5钱，苁蓉5钱，菟丝饼5钱，巨胜5钱，酒母5钱，酒柏5钱，川椒5钱，仙茅1两，枸杞1两，黄精1两，云苓1两，人参1两，山药1两，首乌2两，鹿胎1具。
出处：《全国中药成药处方集》（吉林、哈尔滨方）。
主治：肾虚阳痿，月经不调，子宫寒冷，虚劳。
加减：None
功效：补肾填精，调月经，温子宫，益气养血。
用法用量：早晚各服1丸，枣汤为引，或淡盐汤为引。
制备方法：将鹿胎洗净晒干，合诸药一处碾细，炼蜜为丸，重2钱1分，大赤金为衣，用棉纸包之，外用蜡皮封固，贮于玻璃瓶或瓷坛中。
临床应用：None
各家论述：None
用药禁忌：君相火盛，血热者忌服。
附注：None
