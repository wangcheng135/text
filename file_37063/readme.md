序号：37073
名称：定心神牛黄丸
组成：牛黄（别研）半两，珍珠末（别研）半两，琥珀（别捣罗）半两，铁粉（别研）半两，天竺黄（别研）半两，龙齿（别研）半两，金箔70片（与水银同研），银箔70片（与水银同研），水银（与金、银箔同研）半两，犀角（镑）半两，丹砂（与水银、金箔、银箔同研）半两，露蜂房（炙）1两，龙胆（去土）半两，升麻半两，防风（去叉）半两，黄芩（去黑心）半两，钩藤半两，知母（细切，焙）1两，天门冬（去心，焙）1两，白芍药1两，茯神（去木）1两，甘草（炙，锉）1两，菖蒲（九节者，米醋浸，刮去皮，切，酒炒）1两，麦门冬（去心，焙）1两，干蝎（酒炒）1两，麝香（别研）半两，人参半两。
出处：《圣济总录》卷十五。
主治：风邪变成癫痫，时时发动，不知人事。
加减：None
功效：None
用法用量：每服15-20丸，夜卧及食后煎新竹叶汤送下。
制备方法：上药除别研石药及麝香外，余药为末，与研药再罗令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“牛黄丸”。
