序号：8499
名称：四阴煎
组成：生地2-3钱，麦冬2钱，白芍药2钱，百合2钱，沙参2钱，生甘草1钱，茯苓1钱半。
出处：《景岳全书》卷五十一。
主治：阴虚劳损，相火炽盛，津枯烦渴，咳嗽，吐衄，多热。
加减：如夜热盗汗，加地骨皮1-2钱；如痰多气盛，加贝母2-3钱，阿胶1-2钱，天花粉亦可；如金水不能相滋而干燥喘嗽者，加熟地3-5钱；如多汗不眠，神魂不宁，加枣仁2钱；如多汗兼渴，加北五味14粒；如热甚者，加黄柏1-2钱（盐水炒用），或玄参亦可，但分上下用之；如血燥经迟，枯涩不至者，加牛膝2钱；如血热吐衄，加茜草根2钱；如多火便燥，或肺干咳咯者，加天门冬2钱，或加重便亦可；如火载血上者，去甘草，加炒栀子1-2钱。
功效：保肺清金。
用法用量：水2钟，煎7分，食远服。
制备方法：None
临床应用：None
各家论述：《成方便读》：生地滋肾水；参、麦养肺阴；白芍之色白微酸，能入肺而助其收敛；百合之甘寒且苦，能益金而兼可清神；茯苓以降其浊痰；甘草以散其虚热。名曰四阴者，取其地四生金也。
用药禁忌：None
附注：None
