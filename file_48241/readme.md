序号：48256
名称：神柞饮
组成：生柞树刺枝如小指大1握（水洗净，切碎），甘草5钱，新汲水1碗半。
出处：《胎产心法》卷中。
主治：横逆倒产，死胎在腹。
加减：None
功效：催生。
用法用量：用新瓦罐入水与药于内，以纸三层密封，文武火煎8分，温服。不煎滓。凡觉腹疼腰重欲坐草时，即将此药温服1盏，便觉心下开豁；如渴，又饮1盏，觉下重便产，更无横生倒逆之患。若遇横生倒逆，不过3服即正。子死腹中，不过3服即下。能保母子两全。
制备方法：None
临床应用：横产：一妇横产，儿手先出，致胛肿胀。欲截其手，不保其生，屡用催生药不效。以此药浓煎一碗与服，顷刻苏醒。再与一碗，困睡少时，忽云骨节都拆开了，扶起即血水涌下，拔出死胎，全不费力。
各家论述：None
用药禁忌：None
附注：None
