序号：84411
名称：七汁救命膏
组成：茅根汁1饭碗，竹沥汁1饭碗，萝卜汁1饭碗，韭菜汁1饭碗，生藕汁1饭碗，梨汁1饭碗，人乳1饭碗，童便（须清白者）1饭碗，无灰好陈酒1饭碗，姜汁1小杯（听用），川贝母6钱，白硼砂6钱，白茯苓6钱（为末），天冬4两，麦冬4两，地骨皮4两，泽泻1两，知母2两，黄柏2两。
出处：《同寿录》卷二。
主治：痨怯咳嗽。
加减：None
功效：None
用法用量：每用白滚汤调服2-3匙。
制备方法：以上粗药六味，用河水十大宫碗，煎至四碗，滤去滓，澄清，用重绢再沥过，方下前汁酒共十味，再熬至滴水不散为度，方下贝、硼、冬三末，收成膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
