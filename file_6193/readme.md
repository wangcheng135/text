序号：6193
名称：加味养荣丸
组成：当归（酒浸）2两，芍药（煨）1两5钱，熟地黄（酒浸）2两，白术2两，川芎1两5钱，茯苓1两，人参1两，甘草（炙）5钱，黄芩（炒）1两5钱，香附（炒）1两5钱，麦门冬（去心）1两，阿胶（炒）7钱，贝母1两，陈皮（去白）1两，黑豆（大者，炒，去皮）49粒。
出处：《摄生众妙方》卷十一。
主治：女人不孕。经脉参前，外潮内烦，咳嗽，饮食减少，头晕目眩，带下，血风血气，久无嗣息，一切痰火不受峻补，又治胎前胎动胎漏。
加减：None
功效：None
用法用量：每服70-80丸，食前空心盐汤、温酒任下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌食诸血。
附注：None
