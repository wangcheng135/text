序号：44117
名称：复老还童丸
组成：苁蓉（酒浸）1两，菟丝子（酒浸）1两，巴戟（酒浸，去心）1两，牛膝（去芦，酒浸）1两，山药1两，川楝子（盐炒）1两，蛇床子（炒）1两，茯神1两，八角茴香1两，黄芩1两，五味子1两，续断1两，人参1两，枳实1两，槟榔1两，干姜1两，丁香（母者佳）半两，乳香半两，木香半两，沉香半两，白檀半两。
出处：《普济方》卷二二一引《德生堂方》。
主治：None
加减：None
功效：补下元，乌髭须。
用法用量：每服30-50丸，空心温酒送下。
制备方法：上为细末，先入井花水1盏和药，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
