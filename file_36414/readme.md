序号：36424
名称：和血散痛汤
组成：羌活身1钱半，升麻1钱半，麻黄（去节）1钱半，桃仁10个，柴胡2钱，红花1分，归身1分，防风1钱，甘草（炙）2分，独活5分，猪苓5分，黄柏1钱，防己6分，知母（酒）1钱，黄连（酒）6分。
出处：《医学纲目》卷十二引李东垣方。
主治：两手十指，一指疼了一指疼，疼后又肿，骨头里痛，膝痛，左膝痛了右膝痛，发时多则5日，少则3日，昼轻夜重，痛时觉热，行则痛轻，肿却重。
加减：None
功效：和血止痛。
用法用量：上(口父)咀。分作4服，每服水1大盏，煎至一半，去滓，空心热服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：和血散瘀汤（《证治宝鉴》卷十二）；方中黄连用量原缺，据《赤水玄珠》补。
