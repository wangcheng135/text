序号：71781
名称：紫菀汤
组成：甘草1分，杏仁1分，紫菀1两，桑白皮1分，苦梗3分，天门冬1两。
出处：《妇人良方》卷十三。
主治：妊娠咳嗽不止，胎不安。
加减：None
功效：清润肺气。
用法用量：上(口父)咀。每服3钱，水1盏，竹茹1块，煎至7分，去滓，入蜜半匙，再煎2沸，温服。
制备方法：None
临床应用：None
各家论述：1.《医方集解》：此手太阴药也。子嗽由于火邪，当以清火润肺为务，桔梗、桑皮之凉以泻之，天冬、竹茹之寒以清之，紫菀、炙草之温，杏仁、白蜜之泽以润之也。_x000D_
2.《医林纂要》：肺气不足则生燥，胎热有余则烁金，故子嗽。肺燥润之，紫菀、天冬、杏仁、白蜜；肺热泄之，天冬、桑皮、桔梗、杏仁；炙草温之，竹茹散之，嗽可止矣。
用药禁忌：忌食鲤鱼。
附注：None
