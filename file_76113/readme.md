序号：76129
名称：蒋氏肥儿丸
组成：芸白术（土炒）4两，陈皮（炒）4两，香附（盐水炒）4两，厚朴（姜汁炒）4两，枳实（麸炒）4两，槟榔4两，神曲（炒）4两，麦芽（炒）4两，泽泻4两，菔子（炒）4两，山楂肉（炒）4两，白茯苓（蒸）4两，白芍药（炒）4两，木香（不见火）2两，甘草（炙）2两。
出处：《医宗说约》卷五。
主治：小儿瘦弱，食不生肌，肚腹疼痛，内热虫积。
加减：内热口渴，脉洪数，小便黄，加连翘、川黄连各3两；骨蒸热、背热、手心热、脚底热，口渴身瘦，溺赤，脉数细，加芦荟1两，胡连、银柴胡各3两，鳖甲（醋炙）4两；疟后寒热不止，胁下有块，加银柴胡、胡黄连、当归、青皮各3两，醋炙鳖甲4两，桂枝7钱；腹中膨胀，或有块痛，手不可按，加山棱、蓬术（醋炒）各3两，食蛆2两、干蟾2只；泄泻少食，或腹痛手按则减，面色(白光)白无神，六脉无力，倍白术、白芍，加人参、山药、米仁各4两；泻不止，加肉果（面裹煨）2钱；六脉迟，手足冷，口不渴，溺清白，肌肉日削，或悠悠腹痛无减增，或喜就暖处，或食生冷即泻，或脐下作痛，是属虚寒，加炮姜、肉桂各2两，倍白术、白芍；泄，加肉果（面煨）2两；虚甚，加人参、山药各2两；面上白斑，腹中作痛，口出清水，或吐虫，属虫积，加使君子肉、贯众各2两，雷丸、鹤虱各1两。
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《卫生鸿宝》引作“肥儿丸”。
