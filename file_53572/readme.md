序号：53587
名称：夏枯扶桑丸
组成：金银钝花2斤，百合1斤，真阿胶8两（炒），川贝母（去心）8两。
出处：《惠直堂方》卷一。
主治：一切疮疡，内伤阴虚，痨瘵咳嗽，痰喘血症，及目疾等。
加减：None
功效：None
用法用量：肺痿、肺痈，百合汤下；心颤，朱砂、麦冬汤下；久嗽，五味汤下；肠痈、乳痛，带壳瓜蒌仁汤下；肝中少血，烦躁不宁，白芍、地骨皮汤下；疬（疒串），雄黄冲开水下；疗疮，醋磨敷患处，仍用引经药服之；阴疮不可言者，痛痒难当，车前子、牡蛎粉煎汤下。
制备方法：上为末，用夏枯草、桑叶各20斤，熬汁煎膏为丸，重3钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
