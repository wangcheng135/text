序号：42752
名称：保安丹
组成：附子（炮）1两，当归1两，陈皮（去白）1两，干姜（炮）1两，蜀椒（去子）半两，厚朴（去皮，以姜汁炙令香熟）半两，吴茱萸半两，舶上硫黄1分（另研至细）。
出处：《博济》卷三。
主治：脾无虚滑及久患泻，服药未效，日夜不止，脐腹冷疼及一切气刺、气痛。
加减：None
功效：None
用法用量：每患一切气痛及宿酒食不消，炒生姜盐汤送下20丸；如患泻痢，米饮送下。
制备方法：上为细末，硫黄末和匀，以米醋和作剂，分为两团，别用白面半斤裹药令匀，如烧饼法，煨令面熟为度，杵烂为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：保安丸（《圣济总录》卷四十四）。
