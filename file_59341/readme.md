序号：59356
名称：莲子草膏
组成：莲子草汁2升，松叶4两，青桐白皮4两，枣根白皮3两，防风2两，芎藭2两，白芷2两，辛夷仁2两，藁本2两，沉香2两，秦艽2两，商陆根2两，犀角屑2两，青竹皮2两，细辛2两，杜若2两，蔓荆子2两，零陵香1两，甘松香1两，白术1两，天雄1两，柏白皮1两，枫香1两，生地黄汁5升，生麻油4升，猪鬃脂1升，马鬐膏1升，熊脂2升，蔓荆子油1升。
出处：《外台》卷三十二引《崔氏方》。
主治：头风、白屑。
加减：None
功效：长发令黑。
用法用量：欲涂头，先以好泔沐发后，以敷头发，摩至肌；又洗发，取枣根白皮锉1升，以水3升，煮取1升，去滓，以沐头发，涂膏。验。
制备方法：上细切，以莲子草汁并生地黄汁浸药再宿；如无莲子草汁，如地黄汁5小升浸药，于微火上纳油脂等和煎九上九下，以白芷色黄膏成，布绞去滓。
临床应用：None
各家论述：None
用药禁忌：None
附注：莲子膏（《普济方》卷四十八）。方中枣根白皮、青竹皮、蔓荆子，《圣惠》作桑根白皮、青竹茹、牡荆子。本方加升麻一两，名“旱莲膏”（见《圣济总录》）。
