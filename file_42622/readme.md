序号：42637
名称：保子七圣散
组成：赤白芍（各半）4分，当归4分，生大黄2分，甘草3分。
出处：《元和纪用经》。
主治：小儿胎寒腹痛，乳哺不时，温壮发热，吐利不常，诸经掣缩，二十五痫，肌肤喜疮，遇时而发作，口疮恶核，赤目黄瘦，大小变蒸。
加减：None
功效：None
用法用量：上以水3升煎，去滓服。月内儿服1杏核大，300日儿服1栗壳，1-2岁儿服两栗壳，3岁以上儿加之，每日3次。若发惊及温壮，外有触冒寒邪，以去节麻黄1分，水3升煮之，去沫滓，纳正药煎，如上服之；若惊风反折，戴眼掣缩，加细辛4分，纳1料正药，增水至4升，煮取1升5合服；若中风身体强直戴眼者，加独活2分，纳1料正药，加水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
