序号：38578
名称：法制何首乌丸
组成：何首乌（有赤白2种，各取停匀，用竹刀刮净，忌铁），牛膝（去芦，视何首乌折半），大黑豆1斗（水浸湿，用大柳木甑1个作平底篦，先铺黑豆一层，即铺何首乌一层，又放牛膝一层，又铺黑豆。如上安置放锅中，慢火蒸，以酒浇之；蒸至豆烂，取出晒干，去豆不用），赤白茯苓（先去皮，各研，取出筋膜，各用牛乳拌匀蒸过，取出晒干，秤何首乌4两，茯苓2两，牛膝2两）。
出处：《保命歌括》卷三十四。
主治：None
加减：None
功效：久服令人须发不白。
用法用量：每服50丸，空心食前酒送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌食猪羊血。
附注：None
