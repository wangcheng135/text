序号：38319
名称：枣肉平胃散
组成：陈橘皮（去白）3斤2两，厚朴（去粗皮，姜制，炒香）3斤2两，甘草（锉，炒）2斤，生姜2斤，红枣2斤，苍术（去粗皮，米泔浸2日，炒）5斤。
出处：《局方》卷三（新添诸局经验秘方）。
主治：脾胃不和，不思饮食，心腹胁肋胀满刺痛，口苦无味，胸满短气，呕哕恶心，噫气吞酸，面色萎黄，肌体瘦弱，怠惰嗜卧，体重节痛，常多自利，或发霍乱，及五噎八痞，膈气反胃。
加减：None
功效：常服调气暖胃，化宿食，消痰饮，辟风、寒、冷、湿四时非节之气。
用法用量：每服2钱，空心、食前盐汤点下。
制备方法：上锉碎，拌匀，以水浸过面上半寸许，煮令水干，取出焙燥，碾为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
