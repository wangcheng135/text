序号：43443
名称：养脏丹
组成：猪大脏半斤，大附子2只（炮，去皮脐），厚朴（姜制，炒为末）4两，硫黄2两（研细，同厚朴入猪大脏内，以麻绳系定两头，用水5升煮干，取出细研），龙骨（生用）1两，肉豆蔻（面裹煨）1两，木香（不见火）1两，白茯苓（去皮）1两，牡蛎（煅）1两，大诃子（炮，去核）1两，茴香（淘去沙，炒）1两，破故纸2两（炒）。
出处：《魏氏家藏方》卷七。
主治：脾元虚弱，久泻不止，肠胃不固，致成五泄。
加减：None
功效：None
用法用量：每服100丸，空心、食前盐汤或饭饮送下。
制备方法：上为细末。入麝香1钱，薄面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
