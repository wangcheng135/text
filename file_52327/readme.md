序号：52342
名称：顺流丹
组成：当归6两，川芎6两，升麻6两，甘草6两。
出处：《重庆堂随笔》卷上。
主治：痘证险逆。
加减：None
功效：None
用法用量：凡小儿1岁足者，用辰砂末、米末各1分，白蜜1茶匙，米汤半杯，醇酒半杯，其2味调匀，以茶匙徐徐喂服。未出痘者免出，已见点者必稀，陷没者即_x000D_
起。
制备方法：上为粗末，于腊月8日取东流清水7大碗，煎至3大碗，去滓收药，盛新砂锅内；再选明净完体辰砂4两，盛细绢袋内，以线扎口，悬系药汁中，约离锅底一指，以桑柴慢火煮至汁尽，取起研细末，瓷瓶收藏；另用好糯米半升，淘净控干水气；再以盐卤和净黄土，干湿得所，包米为围，放炭火内，煅令通红，速即取出，冷定劈开，拣米粒色黄尽性者，亦研细末，别盛瓷瓶收藏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
