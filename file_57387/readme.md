序号：57402
名称：涌泉汤
组成：漏芦1钱，瞿麦8分，茯苓8分，当归1钱，川芎1钱，三棱5分，生地8分，白芍（炒）6分，泽泻6分，香附6分，甘草4分。
出处：《胎产良方》。
主治：乳汁缺少。
加减：None
功效：None
用法用量：黄酒为引，水煎服。
制备方法：None
临床应用：乳汁还通赵某，女，27岁，已婚。1983年元月17日，生子4天无乳；两乳胀满作痛，情志郁闷不舒，有低热，善太息。证属精神郁闷，气滞不宣，脉络阻塞，乳胀而不得出。宜疏肝理气通乳。方用涌泉汤加味：漏芦12g，瞿麦9g，茯苓9g，川芎6g，三棱4．5g，生地9g，炒白芍6g，泽泻6g，香附6g，王不留行12g，青皮6g，山甲3g，蒲公英9g，两剂，每日1剂。服药后，乳即下，情志愉快，纳食增加。
各家论述：None
用药禁忌：None
附注：None
