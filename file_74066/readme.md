序号：74082
名称：暖肾散
组成：巴戟天（去心，麸炒黑）1分，甘遂（炒黄）1分，槟榔（1枚生，1枚炮）2枚，木香1分，苦葶苈（纸上炒）1分，大麦糵半两，芫花（醋浸，炒黄）半两，陈橘皮（去白炒）半两，腻粉1钱，沉香（锉）1分，泽泻1分。
出处：《圣济总录》卷七十九。
主治：水气肿满。
加减：None
功效：None
用法用量：每服2钱匕，用猪腰子1只，以竹刀子割开，去筋膜，作3片，掺药末在内，用湿纸裹，慢火煨令香熟，先煮葱白3茎，令熟细切，将葱与粟米同煮粥1碗，临卧先食粥一半，方食腰子，药后再食粥令尽，至五更大便并小便下赤黄恶物乃验。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
