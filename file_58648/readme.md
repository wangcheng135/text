序号：58663
名称：秘传降气汤
组成：桑白皮（炒）2两，骨碎补（去毛，炒）半两，草果仁（去皮，煨）半两，五加皮（酒浸半日，炒黄）半两，半夏（生，为末，生姜自然汁为饼，再碎炒）半两，桔梗半两，诃子（炮，去核）半两，甘草（炒）1两，枳壳（去瓤，麸炒）1两，陈皮（去白，炒黄）1两，柴胡（去芦）1两，地骨皮（炒黄），1两。
出处：《局方》卷三（宝庆新增方）。
主治：男子妇人上热下虚之疾，凡饮食过度，致伤脾胃，酒色无节，耗损肾元，水土交攻，阴阳关隔，遂使气不升降，上热则头目昏眩，痰实呕逆，胸膈不快，咽喉干燥，饮食无味；下弱则腰脚无力，大便秘涩，里急后重，脐腹冷痛；治以凉则脾气怯弱，肠鸣下利。
加减：痰嗽，加半夏曲；心肺虚，加人参、茯苓；上膈热，加北黄芩；下部大段虚，加少许炮附子，如使附子多，加生姜；妇人血虚，加当归。
功效：调顺荣卫，通利三焦，开膈化痰，和五脏。
用法用量：每服2钱，紫苏3叶，姜钱3片，水1盏，同煎至7分，食后通口服。后以所主药治之。
制备方法：上为粗末，和匀，再就蒸1伏时，晒干。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》卷一八一引作“降气汤”。
