序号：50572
名称：茸附丸
组成：黄狗脊1条（去两头，截作5-7段，带些肉用），硇砂1两（研，以浆水2升调匀，方下脊骨在汁中浸3日，炭火炙干，以汁尽令黄色，捣细后入诸药），肉桂（去皮）1两，附子（炮）1两，菟丝子（酒浸2日，蒸，熔干）1两，杜仲（姜制）1两，干姜（炮）1两，鹿角胶（炒）1两，肉苁蓉（酒浸，焙）1两，蛇床子（炒）半两，葫芦巴半两，阳起石（酒煮1日，研）半两，鹿茸（蜜炙）1两半，黄狗内外肾1付（酒煮，焙干）。
出处：《朱氏集验方》卷八。
主治：None
加减：None
功效：补下元伤惫，驻颜悦色，壮筋力，去百病。
用法用量：每服20丸，盐汤送下。
制备方法：上为细末，用枣肉5两，酥1两，相和为丸，如绿豆大，晒干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
