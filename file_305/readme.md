序号：305
名称：五食丸
组成：大戟（刮去皮）半两（生），甘遂半两（生），猪牙皂荚（生，去皮子）1两，胡椒1分，芫花半两（醋浸1宿，炒干），巴豆半两（去皮心膜，醋煮30沸。漉出，研）。
出处：《圣济总录》卷七十二。
主治：虚积、食气，蛊胀，水气，年深癥癖。
加减：None
功效：None
用法用量：每服5丸，用米、面、绿豆煎汤放温送下。量病人大小，加至7丸。
制备方法：上为末，合研匀，水煮面糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：忌油腻、粘滑物。妇人有胎者，不可服。
附注：神效五食汤丸(《卫生宝鉴》卷十四)。《卫生宝鉴》本方用法：每服五七丸，气实者十丸，夜卧，水一盏，用白米、白面、黑豆、生菜、猪肉各少许，煎至半盏，去滓，用汤温下。
