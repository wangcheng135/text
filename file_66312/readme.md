序号：66328
名称：雀瓮散
组成：棘刚子10枚（取虫，微炒），全蝎30枚（去毒，炒），蓖麻子20枚（去皮，研），石榴1枚（大者去却子，盛前3味在内，用黄泥裹作球，慢火炙干，烧赤，候闻药气透出，红熟，候冷，取出去泥，细研，次入后药），白僵蚕（炒去丝嘴）1分，天南星（炮）1分，半夏（汤洗7遍去滑），白附子（炮）1分，乳香1钱（别研）。
出处：《杨氏家藏方》卷十七。
主治：小儿慢惊潮搐，口眼相引，目睛上视，头项偃折，痰涎壅闭，神志昏愦。
加减：None
功效：None
用法用量：半岁儿每服1字，1岁儿取半钱，煎荆芥汤入酒2-3点同调下，不拘时候。
制备方法：后四味为细末，次入乳香并烧者药同研匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
