序号：56258
名称：流气饮子
组成：紫苏叶半两，青皮半两，当归（洗）半两，芍药半两，乌药半两，茯苓半两，桔梗半两，半夏（汤洗7遍，焙干为末，姜汁和，阴干）半两，黄耆半两，枳实（麸炒，去瓤）半两，防风半两，甘草（炙）3分，橘皮（洗）3分，木香1分，连皮大腹（锉，姜汁浸1宿，焙）1两，川芎3分。
出处：《全生指迷方》卷三。
主治：气晕。但晕而不眩，发则伏地昏昏，食顷乃苏，脉虚大而涩。由荣卫错乱，气血溷浊，阳气逆行，上下相隔：气复通则苏。妇人臂痛。
加减：None
功效：None
用法用量：每服5钱，水2盏，加生姜3片，大枣1个，同煎至1盏，去滓温服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
