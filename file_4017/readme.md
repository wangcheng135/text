序号：4017
名称：月蟾丸
组成：干蟾（大者，端午取）1枚，蛇蜕皮（大者）1条，谷精草2两（3味同入1瓶子内，以盐泥固济，烧灰），胡黄连1分，甜瓜蒂1分，丁香1分，熊胆（研）1分，芦荟（研）1分，天竺黄（研）1分，牛黄（研）1分，丹砂（研）1分，龙脑（研）1分，麝香（研）1分，雄黄（研）1分，青黛（研）半两。
出处：《圣济总录》卷一七三。
主治：小儿五疳，眼鼻多痒，寒热往来，腹脏不调，或泻脓血，肌体瘦弱，饮食不化，多困少力，眼涩饶睡；兼治惊风。
加减：None
功效：None
用法用量：每服3-5丸，米泔送下。后以桃、柳汤浴儿，着青衣盖。
制备方法：上为末，再研匀，用豶猪胆汁煮面糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
