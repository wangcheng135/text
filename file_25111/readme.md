序号：25119
名称：欢喜散
组成：防风2钱半（去芦并叉枝），人参2钱半（去芦），甘草2钱半（炙），天麻2钱半（去芦），前胡（去芦）2钱半，细辛1钱半（去苗），柴胡（去芦）1钱半，白茯苓2钱（去黑皮），桔梗（去芦）2钱，枳壳（去瓤，麸炒）2钱半，川芎3钱。
出处：《卫生总微》卷七。
主治：小儿伤风寒，发热头痛，无汗恶风；或温热鼻塞流清涕，泪出嚏喷。
加减：None
功效：None
用法用量：每3岁以上抄1钱，水6分，薄荷2叶，同煎2-3沸，通口服，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：清解散（《直指小儿》）、人参清解散（《奇效良方》卷六十四）。
