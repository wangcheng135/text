序号：32926
名称：羌活丸
组成：羌活（去芦头）2两，独活（去芦头）2两，青橘皮（去白，焙）2两，附子（炮裂，去皮脐）2两，虎骨（涂醋炙黄）3两，牛膝（酒浸，切，焙）3两，当归（去芦头，切，焙）2两，桂（去粗皮）1两，没药（研）1两，木香1分。
出处：《圣济总录》卷一八七。
主治：元脏诸虚，筋骨疼痛。
加减：None
功效：壮气去风，聪明耳目。
用法用量：每服20丸，空心温酒送下。
制备方法：上为末，入没药再拌研匀，用宣州木瓜2枚，切开头，剜去瓤、子，各满填熟细艾在内，以竹札原盖子盖了，入甑内烂蒸，取出入臼杵如膏，入诸药和，更杵为剂，如药末多，再入酒面糊少许为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
