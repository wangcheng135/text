序号：41421
名称：金科猪肚丸
组成：使君子1升，青皮（炒）、三棱（煨）、莪术（煨）、黄连、胡黄连、川楝子、芜荑（炒，研）、枳壳（炒）、黄梗皮、青木香、麦芽（炒）、槟榔（炒）、香附子、陈皮、杏仁（研）、茴香（炒）、吴茱萸（炒）、轻粉、巴豆（去心，去油）、神曲（炒）、龙胆草、石榴皮、诃子、肉豆蔻、南木香、芦荟、虾蟆（炙）、谷芽（炒）、青黛、白曲、干姜、玄胡索（炙）、朱砂、姜（炒）、郁金、皂角（烧）、山茱萸、没石子、良姜、干漆（炒令烟尽）、丁香各等分（一方加黄丹、鸡子黄、米粉）。
出处：《普济方》卷三八○引《傅氏活婴方》。
主治：一切疳积，面黄肌瘦，腹内痞癖气块，五疳多虫，骨蒸，疳寒热，瘦悴，面浮，无辜，丁奚。
加减：None
功效：None
用法用量：每服20-30丸，空心饭饮吞下。
制备方法：上为末。先用雄猪肚1个，以使君子肉1升，糯米2-3合，粳米2合，人猪肚内，蒸熟捣烂糊，再入猪胆汁3-4个研匀，却入众药，搜作饼子，臼中杵捣百遍，视色和匀为细丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
