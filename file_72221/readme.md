序号：72237
名称：紫金丸
组成：血竭2两，沉香1两，青皮2两，陈皮2两，枳壳（去瓤，麸炒）2两5钱，厚朴（姜炒）2两，百草霜1两，皂矾4两（用醋煮过），蓬术（醋炒）3两，香附（去毛）1两，针砂（醋炒）1两，干漆（炒过性）2两，槟榔2两，黄石榴矾2两（即金丝矾），秦艽1两，三棱（醋炒）3两，甘草5钱（一方加苍术1两，白术1两）。
出处：《寿世保元》卷三。
主治：酒疸、食疸，五积六聚，七癥八瘕，心腹疼痛，潮热。
加减：None
功效：None
用法用量：每服60丸，温酒送下；米饮亦可。
制备方法：上为细末，用大枣煮烂，去皮核，打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
