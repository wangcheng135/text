序号：40523
名称：虎骨散
组成：虎骨（酥炙黄）1两，败龟（酥炙黄）1两，何首乌（酒蘸去黑皮）半两，羌活（去芦头）半两，当归（细切，熔干）3分，芎3分，牛膝（去苗，酒浸，切焙）3分，秦艽（去苗土）3分，附子（炮裂，去皮脐）半两，威灵仙（洗焙）3分，原蚕沙（炒）3分，延胡索（与糯米同炒米赤为度）半两，皂荚（去黑皮并子炙黄）1两，槟榔（煨）3分，生干地黄（焙）1两。
出处：《圣济总录》卷二十。
主治：中诸风毒，冷痹偏枯不随，骨节疼痛，手足挛跎。
加减：None
功效：None
用法用量：每服3钱匕，温酒调下，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
