序号：47706
名称：神仙巨胜子丸
组成：生地黄4两，熟地黄4两，何首乌4两，巨胜子（九蒸九晒）2两，人参1两，肉苁蓉（酒浸）1两，牛膝（酒浸）1两，菟丝子（酒浸）1两，天门冬（去心，酒浸）1两，破故纸（酒浸，炒）1两，巴戟（去心，酒浸）1两，干山药1两，五味子1两，楮实（炙）1两，覆盆子（净）1两，鹿茸（嫩红色者，生用）1两，柏子仁（去壳，另研）1两，酸枣仁（去壳，另研）1两，白茯苓1两，西枸杞1两，核桃10枚（去壳取仁，另研烂后，入药内再研匀）。
出处：《普济方》卷二二三引《德生堂方》。
主治：None
加减：None
功效：滋血气，壮元阳，髭发反黑，安魂定魄，改易容颜，通神仙，延寿命，生骨髓，扶虚弱，展筋骨，润肌肤，补益丹田，接养真气，活血荣颜，百病永除，根本坚固，水火既济，常服身体轻健，气力倍加，行走如飞。
用法用量：每服50-70丸，空心以温酒或盐汤送下，服后干物压之，丸数任意加减。
制备方法：上为细末，用枣1斤，去皮核煮熟研烂，与药末和匀为丸，如梧桐子大。
临床应用：种子延年：昔有一老人，耳聋目昏，年至七十无子，服此药后，齿落更生，发白再黑，二妻生一十三子，寿至一百余岁。
各家论述：None
用药禁忌：None
附注：乌金丸（原书卷二二一）、神仙巨胜丸（原书卷二二一）。
