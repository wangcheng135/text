序号：21998
名称：利胆排石片
组成：金钱草250g，茵陈250g，黄芩75g，木香75g，郁金75g，大黄125g，槟榔125g，枳实（麸炒）50g，芒硝（精制）25g，厚朴（姜制）50g。
出处：《中国药典》。
主治：用于胆道结石，胆道感染，胆囊炎。
加减：None
功效：清热利湿，利胆排石。
用法用量：口服，排石：1次6-10片，1日2次；炎症：1次4-6片，1日2次。
制备方法：以上10味，木香、大黄芒硝粉碎成细粉；其余金钱草等7味经煎煮，提取，浓缩成稠膏状，加入上述细粉制粒，压制成1000片，包糖衣，即得。
临床应用：None
各家论述：None
用药禁忌：体弱，肝功能不良者慎用；孕妇禁用。
附注：None
