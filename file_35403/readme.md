序号：35413
名称：万病解毒丸
组成：文蛤（即五倍子）1两半，山慈姑（即金灯花根）1两（洗，焙），红芽大戟（洗，焙）7钱半，全蝎5枚，大山豆根半两，续随子（取仁去油，留性）半两，麝香1钱，朱砂2钱，雄黄2钱。
出处：《直指》卷二十二。
主治：痈疽发背，鱼脐毒疮，药毒，草毒，挑生毒，蛇兽毒，蛊毒，瘵虫，诸恶病。
加减：None
功效：解毒收疮。
用法用量：端午、七夕、重阳、腊日，净室修合。每服1丸，生姜、蜜水磨下，井水浸研敷患处。
制备方法：上药先以前5味入木臼，捣罗为细末，次研后4味，夹和糯米糊为丸，分作35丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：玉枢丹（《活人方》卷五）。
