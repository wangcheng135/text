序号：42228
名称：青城山葛真人秋乳丹
组成：秋石4两，钟乳粉2两（真者），云母粉2两，牡蛎（左顾者，用黄泥固一指厚，于文武火煨干后，以炭火煅通红，不以多少，去外黑者，用粉，研细）4两，寒水石半斤（生研为粉，纳信砒1两，以1沙盒纳，盛得恰好，上下以粉紧填信砒1两，外用纸筋、黄泥固1重约半指厚，阴干，以上10斤，火1煅，去尽砒烟，取出，放冷，用白矾3两，纳1两，飞过）。
出处：《普济方》卷一二○引《卫生家宝》。
主治：男子脾肾久弱，下部一切痼冷之疾，遗泄不禁，小便滑数，囊外湿痒；及脾元不固，饮食无味，久而脾泄变为寒热，似疟而多寒，滑泻白脓；及妇人子宫久冷，胎胞不固，赤白带下。
加减：None
功效：None
用法用量：候极干，空心服1丸，盐汤送下，妇人醋汤送下。一法用冷水滴为丸，只要研极细，阴干，复以坩锅盛，瓦片子盖头，大火一煅，功力尤大，半夏稀糊丸者，平稳，功力虽少，久服见效。
制备方法：上为极细末，用园正半夏10两为末，水熟煮成稀糊为丸，如梧桐子大。
临床应用：None
各家论述：None
