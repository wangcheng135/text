序号：32019
名称：万全饮
组成：阿魏1分（童便磨化），甘草（大者）2寸，东引桃枝（小者）1握，青蒿（1大握，用子皮）1两，槟榔1两（末），葱白2寸（并根）（一方有东向柳枝，无葱白）。
出处：《普济方》卷二三五引《圣惠》。
主治：一切劳疾，传尸劳瘵，不问远年近日。
加减：None
功效：None
用法用量：上用童便2升，浸桃枝、蒿、草、葱白4味1宿，来日五更初煎取6合，去滓，入阿魏更煎4沸，分为2服，每服临吃时入槟榔末半两同服。如觉恶心必吐，吐后更进第2服，服时不得与患人语，及不可与患人对面，恐恶虫飞入口鼻内。春2服，秋3服。1年5服，劳虫并尽。病若在上先吐，若在下则利，皆出黑虫如发，及如马尾。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
