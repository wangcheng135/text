序号：70174
名称：滋阴清肺肠
组成：生石膏8钱，知母4钱，玄参5钱，生地4钱，丹皮4钱，银花3钱，石决明6钱，连翘3钱，蒺藜2钱（炒），车前子3钱（外包），甘草1钱，羚羊角5分。
出处：《眼科临症笔记》。
主治：赤膜下垂症（沙眼性血管翳）。头疼，赤丝满目，热泪不止，从风轮上际生赤膜一片，下侵瞳神，视物昏涩酸疼。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：赤膜下垂症：鄄城县李某某，女，18岁。素体虚弱，因功课紧张，过用脑力，夜多失眠，常觉头疼，目痠，后觉目昏，痠疼流泪。诊其脉，寸脉浮数，尺脉沉细；视其目，气轮之上生赤膜一片，下侵风轮，痠疼流泪，怕日羞明，是知肾水不足，而肝火有余。先刺上星、合谷、太阳、攒竹等穴；内服滋阴清肺汤去羚羊角，加田七5分，胡黄连3钱。三剂后复诊，酸涩已止，赤膜稍退，再服滋阴清血汤，并点黄连膏即可。
各家论述：None
用药禁忌：None
