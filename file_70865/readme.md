序号：70881
名称：紫金膏
组成：炉甘石（好者，同火煅酥，研细无声，将黄连、当归身挑头童便浓煎汤滤净，飞，淘去沙石，焙干，粗者再研再淘；一法只用杨梨，亦名蓧采叶，浓煎汤滤净飞淘，焙干）1两，黄丹1两（水飞，细研），乳香1钱，硇砂1钱，雄黄1钱，没药1钱，白丁香（真者）1钱，当归1钱，轻粉1钱，麝香1钱（8味修制了，各为末，逐味用），脑子3钱，蜜4两。
出处：《普济方》卷八十。
主治：男子、妇人目疾，远年近日，翳膜遮障，攀睛胬肉，拳毛倒睫，黑花烂眩，羞明冷泪，及赤眼肿痛。
加减：None
功效：None
用法用量：每用鸡头大1块，沸汤化开，浸汤半盏以下，乘热食后熏洗之。
制备方法：大建盏内熬蜜沸，入黄丹，以柳木篦子急手搅匀，约熬2-3沸，却入炉甘石，复搅匀，下乳香，入硇砂，又下雄黄，次下白丁香，再入没药，又下当归，再入轻粉，方下脑子、麝香，依次第下药，用文武火各熬2-3沸，须急用篦子不住手搅匀，候熬成膏，不粘手为度。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
