序号：48712
名称：卫真汤
组成：人参1两半，当归1两（酒浸1宿）1两，青皮1两（去白），丁香1两，川牛膝2两（童便半盏、酒半盏，浸1宿），生地黄2两，白茯苓3两，木香3两，肉豆蔻3两，熟地黄3两（温水洗），山药3两，金钗石斛5两。
出处：《本事》卷二。
主治：丈夫、妇人元气衰惫，荣卫怯弱，真阳不固，三焦不和，上盛下虚，夜梦鬼交，觉来盗汗，面无精光，唇口舌燥，耳内蝉鸣，腰痛背倦，心气虚乏，精神不宁，惊悸健忘，饮食无味，日渐瘦悴，外肾湿痒，夜多小便，腰重冷痛，牵引小腹，足膝缓弱，行步艰难；妇人血海久冷，经候不调，或过期不至，或一月两来，赤白带下，漏分五色，子宫感寒、久不成孕。
加减：None
功效：补气固摄，实丹田，填五脏。
用法用量：每服3大钱，空心、食前酒调下，盐汤亦得；妇人诸疾，空心用童便同酒调下。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
