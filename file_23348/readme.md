序号：23356
名称：防风汤
组成：防风（去叉）2两，薏苡仁2两，麻黄（去根节，汤煮，掠去沫，焙干）4两，白术1两，芎1两，细辛（去苗叶）1两，羌活（去芦头）1两，茵芋（去粗茎）1两，牛膝（去苗，酒浸，切，焙）1两，狗脊（去毛）1两，萆薢1两，侧子（炮裂，去皮脐）1两，杏仁（去皮尖双仁，炒微黄）1两，赤箭1两，桂（去粗皮）1两。
出处：《圣济总录》卷二十。
主治：风寒湿痹，筋脉挛急，身体手足不随。
加减：None
功效：None
用法用量：每服4钱匕。水1盏，加生姜3片，同煎至7分，去滓温服，不拘时候。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
