序号：66505
名称：丁香丸
组成：丁香1分，木香1分，白豆蔻（去壳）1分，青橘皮（汤浸，去白，焙）1分，胡椒1分，荜茇1分，槟榔1分，麝香（别研）1分，乳香（别研）半两，巴豆半两（去皮，微炮，细研，纸裹，压油尽，研成霜）。
出处：《普济方》卷二十引《博济》。
主治：脾脏冷气，攻心腹疼痛，及妇人血飞。
加减：心痛，煎盐、醋汤下7丸；妇人血气，当归酒下5丸。
功效：None
用法用量：每服5丸，茶、酒任下。
制备方法：上将槟榔以上药先为末，次入余3味，和匀，用醋煮面糊为丸，如黍米大，再用朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中白豆蔻，《圣济总录》作“肉豆蔻”。
