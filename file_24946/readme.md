序号：24954
名称：如圣散
组成：苍术6两，川乌头（炮，去皮）4两，草乌头（炮，去皮）2两半，两头尖（炮，去皮）4两，白术1两半，川芎1两半，白芷1两半，防风2两半，细辛2两半，蝎梢5钱（微炒），雄黄5钱。
出处：《袖珍》卷一引《圣惠》。
主治：破伤风；一切金刃伤，及诸般伤损、中风搐倒，牙关紧闭。
加减：如损骨，加乳香5钱。
功效：止血定疼。
用法用量：每服1钱，酒调下，不拘时候。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：《医方类聚》引《袖珍》有天麻一两，无白术、蝎梢。《普济方》本方用法：金刃箭伤，先以药末一钱重，干贴伤口，后用温酒调一二钱服之。
