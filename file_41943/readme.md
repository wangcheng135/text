序号：41958
名称：青龙丹
组成：青黛1钱，硼砂1钱，白丁香2分，轻粉2分。
出处：《鸡峰》卷十九。
主治：水气，其势甚危者。
加减：None
功效：None
用法用量：上为末，滴水作块子，用蒸饼剂裹1重，桑柴灰内煨，候熟，再裹1重，又煨熟放冷，剥去面，用水煮面糊为丸，如梧桐子大。茵香汤送下。前件药，先勘7日内合吃的确丸数，依样大小服之。第1日3丸，2日4丸，3日5丸，4日7丸，5日6丸、9丸，每日3次。自第1日至7日，水未尽，中间不可断续也。第4-5日，其蓄水随小便出约1-2斗，候肚皮塌是效，如有余证即服向后准备药。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
