序号：42664
名称：保元汤
组成：人参1钱，黄耆3钱，甘草1钱，肉桂5分至7分。
出处：《博爱心鉴》卷上。
主治：元气虚弱，精神倦怠，肌肉柔慢，饮食少进，面青（白光）白，睡卧宁静，痘顶不起，浆不足，及有杂证。气血不足，婴儿怯弱，痘毒内陷，面色苍白，气陷久泻，肢体无力，肺脾虚弱，恶寒自汗。
加减：None
功效：补气温阳，滋养益气，扶弱补虚。
用法用量：None
制备方法：None
临床应用：None
各家论述：1.《博爱心鉴》：人参益内，甘草和中，实表宜用黄耆，助阳须凭官桂。前三味得三才之道体，后一味扶一命之巅危。_x000D_
2.《古今名医方论》引柯韵伯：参、耆非桂引道，不能独树其功；桂不得甘草和平气血，亦不能绪其条理。
用药禁忌：血热毒壅之火证禁用；禁忌生冷。
附注：参耆汤（《痘疹活幼至宝》卷终）、参耆饮（《简明医彀》卷六）、保元丹（《全国中药成药处方集》（沈阳方）。
