序号：80921
名称：飞龙夺命丹
组成：天南星1钱，半夏1钱，巴豆（去皮心）1钱，硇砂、信、黄丹、乳香、斑蝥16个（去翅、足），麝少许。
出处：《普济方》卷二七三。
主治：疮肿。
加减：None
功效：None
用法用量：每服3-5丸，或7丸，或9丸，随人虚实加减，以无灰酒送下。疮在上者，食后；在下者，食前服。如疮下者，加丸服。服药后觉疮痛皮肤红，是药之效也，如患者昏愦，牙关紧者，斡开口灌可治。服药后饮凉酒数口，药疾行也。
制备方法：上为细末，用5月5日蟾酥，和合为丸，如黄米粒大。
临床应用：None
各家论述：None
用药禁忌：药后忌热食、热汤、冷水1-2时辰，忌油腻、荤腥、湿面35日。
附注：方中硇砂、信、黄丹、乳香用量原缺。
