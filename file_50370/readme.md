序号：50385
名称：茵芋散
组成：茵芋半两，川乌头半两（炮裂，去皮脐），干姜半两（炮裂，锉），细辛半两，黄芩半两，桂心半两，天雄半两（炮裂，去皮脐），汉防己3分，秦艽1两（去苗），赤茯苓3分，防风1两（去芦头），当归3分（锉，炒），甘草3分（炙微赤，锉）。
出处：《圣惠》卷二十。
主治：贼风入腹，腹中拘急，烦乱恍惚，迷惑不知人事，口噤不开，手足缓弱，卧即惊恐，口干，身体沉重。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，煎至5分，去滓，入竹沥1合，更煎1-2沸，温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
