序号：64729
名称：硇砂丸
组成：硇砂1分（细研），干漆（炒烟出）1分，木香半两，丁香半两，蓬莪术半两（煨，锉），京三棱半两（煨，锉），青橘皮半两（汤浸，去白，焙），芫花（炒）半两，肉豆蔻5枚（大者，去壳），巴豆1分半（去皮心膜，压出油）。
出处：《圣济总录》卷七十二。
主治：食癥结块，疼痛发歇。
加减：None
功效：化气，消酒食。
用法用量：每服2-3丸。如要取转，生姜汤送下；妇人血气，醋汤送下；小儿疳气，甘草汤送下；男子膈气，龙脑汤送下；泻痢，干姜汤送下。
制备方法：将米醋1碗，浸蓬莪术、芫花、三棱一宿，后焙干，与五味同捣罗为末，入硇砂、巴豆拌匀，醋煮面糊为丸，如大麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
