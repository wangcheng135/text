序号：33352
名称：肠风黑散
组成：荆芥（烧）2两，枳壳（去瓤）3两（2两烧，1两炒用），乱发（烧）1两，槐花（烧）1两，槐角（烧）1两，甘草（炙）1两半，猬皮1两半。
出处：《局方》卷八（淳祐新添方）。
主治：荣卫气虚，风邪冷气进袭脏腑之内，或食生冷，或啖炙煿，或饮酒过度，积热肠间，致使肠胃虚弱，糟粕不聚，大便鲜血，脐腹疼痛，里急后重，或肛门脱出，或久患酒痢，大便频并。
加减：None
功效：None
用法用量：每服2钱，水1盏，煎至7分，空心温服；温酒调下亦得。
制备方法：上将各烧药同入瓷瓶内，黄泥固济，烧存三分性，出火气，同甘草、枳壳为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：用法中甘草，《奇效良方》作“木馒头”。
