序号：62442
名称：梅花丸
组成：腊月梅花不拘多少（阴干，另用），当归1钱5分，茯苓1钱，升麻5分，竹茹8分，甘草3分。
出处：《种福堂方》卷四。
主治：小儿痘疹。
加减：None
功效：None
用法用量：如男小儿病，用雄鸡1只，吊起左足良久，将竹枪入鸡喉内取血，调梅花末为丸，如绿豆大。每服2丸，滚水送下，即刻见功；如女小儿病，用老雌鸡吊右足，如前取血。制造晒干，以好瓷器收贮，不拘远年近日听用。此起死回生之药，小儿临危，任是毒甚，略有微气，用滚水送下，不拘时候，只不宜多服。
制备方法：用水1钟半，煎至8分，温热时将梅花拌浸1日，取出晒干，研为极细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
