序号：52073
名称：大芦荟丸
组成：芦荟（研）半两，木香半两，青橘皮半两，胡黄连半两，黄连半两，白芜荑（去扇称）半两，雷丸（破开，白者佳，赤者杀人，勿用）半两，鹤虱（微炒）半两，麝香2钱（另研）。
出处：《小儿药证直诀》卷下。
主治：小儿疳积、虫积。肚腹紧胀，心胸膨满，消瘦神困，肚胀青筋，肠鸣泻臭，食即呕哕。喜食酒肉，食不生肌，胸满胁胀，烦躁迷闷，眠不安席。肝脾疳积，食积发热，目生云翳；或疳热，颈项结核；或耳内生疮，肌体消瘦，发热作渴，饮食少思，肚腹膨胀；或牙龈蚀落，颊腮腐烂；阴囊、玉茎生疮；或胸胁小腹作痛。
加减：None
功效：治疳杀虫，和胃止泻。
用法用量：每服20丸，米饮送下，不拘时候。
制备方法：上为细末，粟米饭为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
