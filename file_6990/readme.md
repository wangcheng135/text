序号：6990
名称：四灵丸
组成：大蟾1枚（去却4足，劈开腹，去肠肚，入胡黄末1两在腹内，以线缝合，用湿纸3两重裹，以泥四面固济，令干，微火出阴气，便以炭火3斤烧令通赤即住，待冷去泥及纸灰，捣细罗为末），芦荟1分，麝香1分，熊胆1分。
出处：《圣惠》卷八十六。
主治：小儿五疳，头大项细，心腹胀满。皮肤干皴，毛发焦黄，鼻下赤烂，口舌生疮，泻利不止，日渐羸瘦。
加减：None
功效：None
用法用量：每服3丸，以粥饮或奶汁送下，日3次。3岁以上加丸服之。
制备方法：上为细末，以面糊为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：大蟾丸（《医部全录》卷四四五）。
