序号：32030
名称：万安丸
组成：冬白术20两（土炒），云苓20两（炒），泽泻15两（炒），广藿香30两（微炒），官桂8两（研），紫苏20两（炒），制半夏20两（姜汁炒），茅苍术8两（炒），草蔻仁20两（炒），神曲30两（炒），陈皮15两（炒），麦芽30两（炒），山楂30两（炒焦），枳壳20两（麦麸炒），厚朴20两（姜汁炒），羌活12两（炒），独活15两（酒炒），生甘草4两（炙），丹参30两（酒炒），苡仁米20两（炒）。
出处：《慈航集》卷上。
主治：夏秋感冒寒暑，伤风头痛，恶寒发烧，遍身骨节疼痛，霍乱吐泻，瘟疫疟痢，时毒斑疹，四时不正之气。
加减：None
功效：None
用法用量：照后汤头煎汤送服：感冒伤风，头痛，恶寒，发烧，遍身骨节疼痛，用煨姜2钱，葱头3个，煎汤化服1丸，取汗即愈；霍乱吐泻，用煨姜3钱，大枣3个，灶心土3钱，煎汤化服1丸；疟疾，用煨姜3钱，大枣3枚，煎汤化服1丸，于未来早一时服；红痢，用金银花3钱，炒黑荆芥2钱，煎汤化服1丸；如腹痛，加广木香1钱5分；白痢，用煨姜2钱，红砂糖5钱，煎汤化服1丸；斑疹，用生石膏5钱，炒升麻1钱，煎汤化服1丸；孕妇，用当归8钱，炮姜8分，煎汤化服1丸；产后用当归8钱，益母草3钱、炮姜1钱5分，煎汤化服1丸。大人1丸，小儿半丸。
制备方法：上药如法炮制，各研净细末，炼蜜为丸，每颗重4钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
