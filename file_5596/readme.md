序号：5596
名称：宁坤丸
组成：党参2钱，白术5钱，茯苓5钱，炙甘草1钱5分（姜汁炒），生地5钱，白芍5钱（姜汁炒），熟地5钱，炒当归5钱，炒川芎5钱，沉香5分，广木香2钱5分，制香附5钱，西砂仁1钱5分，乌药5钱，炒广皮5钱，川牛膝2钱，琥珀2钱5分，黄芩2钱，苏叶2钱5分，阿胶2钱5分，益母膏1两2钱。
出处：《中药成方配本》。
主治：血虚气滞，经闭经少。
加减：None
功效：和气血，调月经。
用法用量：每服1丸，开水化服。
制备方法：上药除阿胶、益母膏外，其余共为细末，阿胶、益母膏烊化，加白蜜四两炼熟，与诸药打和为丸，分做44粒，每粒约干重2钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
