序号：25184
名称：防风补煎
组成：防风3两，细辛3两，芎3两，白鲜皮3两，独活3两，甘草3两，橘皮2两，大枣3-7枚，甘竹叶（切）1斗，蜜5合。
出处：《千金》卷十一。
主治：肝虚寒，（目巟？？）（目巟？？），视物不明，谛视生花。
加减：None
功效：None
用法用量：以水1斗2升，先煮9味，取4升，去滓，下蜜，更煎2沸，分4服，日3夜1。若5-6月，以燥器贮，冷水藏之。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：防风、白鲜皮、甘竹叶上散头目诸风；细辛、独活、芎下通肾肝之结；甘草、橘皮、蜂蜜、大枣培土以御木邪之下陷也。
用药禁忌：None
附注：本方方名，《千金方衍义》引作“防风煎”。
