序号：53738
名称：宽中散
组成：宣姜2-3斤。
出处：《活人方》卷五。
主治：多忧多郁之人，中气虚寒之体，寒湿痰饮停滞三脘，自呕恶而成反胃，由噎塞而致关格，两关脉沉滑，或濡软无力者。
加减：None
功效：豁痰利气，温中散结。
用法用量：每服3钱，白汤调服。
制备方法：每块姜均切2片，粗线穿好，晒极干，浸于极陈无秽真金汁内7昼夜，取出，烈日晒露7昼夜，掛当风处，1浸1晒，各足49日，在地上筑一土堆，于中挖1大孔，放数斤炭火，入姜煨透，去净炭火，以砖盖闭，勿令透风，1周时开看，候姜成炭取出，星月下露7日，然后研为极细末，收贮瓷罐内，勿使透风经湿。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
