序号：61613
名称：常山饮
组成：常山1两，豉1合，葛根（锉）3分，柴胡（去苗）3分，升麻3分，牡蛎（熬）3分，大黄（锉，炒）3分，前胡（去头芦）3分，知母（焙）3分，山栀子仁3分，枳壳（去瓤，麸炒）3分，半夏（汤洗7遍去滑，炒）半两，麦门冬（去心，焙）半两，桃仁（去皮尖双仁，炒）半两，甘草（炙）半两，犀角（镑）半两。
出处：《圣济总录》卷三十四。
主治：疟疾。先寒后热，四肢黄瘦，头痛。
加减：None
功效：None
用法用量：每用5钱匕，水1盏半，入葱、薤白各3寸，切碎，煎取1盏，去滓，分2服，发前及发时热服之。
制备方法：上（口父）咀，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
