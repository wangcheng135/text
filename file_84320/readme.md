序号：84338
名称：五饮丸
组成：远志1两，苦参1两，黎芦1两，白术1两，甘遂1两，五味子1两，大黄1两，石膏1两，桔梗1两，半夏1两，紫菀1两，乌贼骨1两，前胡1两，芒消1两，芫花1两，当归1两，人参1两，栝楼根1两，大戟1两，贝毋1两，茯苓1两，芍药1两，黄芩1两，葶苈1两，桂1两，恒山1两半，薯预1两半，厚朴1两半，细辛1两半，附子1两半，巴豆30个，苁蓉1两，甘草3分。
出处：《鸡峰》卷十八。
主治：五饮留滞，停痰癖饮结在两胁，心腹胀满，羸瘦，不能饮食，食不消化，喜唾干呕，大小便或秘或利，腹中动摇作水声，腹内热，口干欲饮水浆，卒起头眩欲倒，胸胁下痛。
加减：None
功效：None
用法用量：每服1-2丸，临卧时以熟水送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
