序号：24256
名称：导赤丸
组成：赤芍药4两，茯苓（去皮）4两，滑石4两，生干地黄（焙）半斤，木通（去节）半斤，大黄（炒）15两，山栀子仁（炒）12两。
出处：《局方》卷六（续添诸局经验秘方）。
主治：心肾凝滞，膀胱有热，小便不通，风热相搏，淋沥不宣；或服补药过多，水道蹇涩，出少起数，脐腹急痛，攻注阴间；或心肺壅热，面赤心忪，口干烦渴；及痈肿发背，血脉瘀闭；内蕴风热，五般淋疾。
加减：None
功效：排脓，内消肿毒，疏导心经邪热。
用法用量：每服20-30丸，食后用温热水吞下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
