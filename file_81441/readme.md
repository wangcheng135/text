序号：81457
名称：鳖甲散
组成：鳖甲2两（醋炙），京三棱（炮）1两，茯苓1两，人参2两（去芦头），大黄3分（煨），黑附子（去皮脐）1两，枳壳（去白）1两，牛膝1两，半夏半两（去粗皮），羌活1两，槟榔1两，干地黄1两（米炒），厚朴（去皮，姜制）1两，五味子1两，木香1两，当归（炒）1两，白术1两，芍药1两，肉豆蔻1两，沉香1两。
出处：《传家秘宝》卷下。
主治：五脏虚劳气攻注，四肢无力，手足疼痛，日渐瘦弱，心下气满，不思饮食。
加减：None
功效：None
用法用量：每服3钱，水1盏，加大枣3个，生姜5片，同煎7分，去滓服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
