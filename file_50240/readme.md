序号：50255
名称：茱萸断下丸
组成：艾叶半两（炒），缩砂仁1分，附子（炮，去皮脐）1分，肉豆蔻1分，吴茱萸2两半（炒），赤石脂半两，川姜半两。
出处：《百一》卷六。
主治：脏寒腹痛，泄泻不止。
加减：None
功效：None
用法用量：每服50-70丸，食前米饮送下。
制备方法：上为细末，面糊为丸，如梧桐子大。
临床应用：1.久泻：赵从简通判，甲辰年丁母忧，食素之久，苦泻不止，日七八行，首尾几年，每服它药不过1-2日复作，得此方而愈，后数年间遇泻服之又效。_x000D_
2.久泻：新石街里正某妻，年30许，日溏泄六七行，既五阅月尚不愈。医与参苓白术散、附子理中汤、四逆汤等并不效。形肉羸脱，不能起床，既在死法中。友人橘尚贤劝余诊之，气口脉沉细稍有神，乃以茱萸断下丸料作汤，每服四神丸50粒，用之6-7日，泻利减其半，10余日而大便始见粪。调理50余日而平复。
