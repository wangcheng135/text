序号：61095
名称：铁粉散
组成：铁粉（研）4两，天麻1两，白僵蚕（直者，炒）1两，蝎梢（炒）1分，白附子（炮）半两，乌头（炮裂，去脐皮）3分，白花蛇（酒浸，去皮骨，炙）3分，桂（去粗皮）半两，麝香1分，龙脑1分，丹砂1两（3味同细研）。
出处：《圣济总录》卷五。
主治：中风涎潮搐溺，口眼斜，手足垂弹；破伤风、沐浴伤风、产后中风，及小儿惊风。
加减：None
功效：None
用法用量：每服1钱匕，薄荷汁和酒调下，腊茶清亦得；如病势危急，研龙脑、腻粉、薄荷水调服；小儿惊风，服半钱匕。
制备方法：上11味，以前8味为未，同后3味合研令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
