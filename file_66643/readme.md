序号：66659
名称：鹿柏固本丸
组成：鹿角霜半斤，（以角之新者，寸截，入布囊置流水中7日，瓦缶水煮。每角1斤入黄蜡半斤，去中用露酒1壶掩之别沸，流水旋添，勿令1竭，桑柴火足十二时，其角软矣，用竹刀切去黑皮，取白者，舂细为细霜），黄柏1斤（酒1制，乳3制，盐汤1制），天门冬4两（酒浸），麦门冬4两（酒浸），生地黄4两（酒洗浸），熟地黄4两（酒洗浸）。
出处：《丹溪心法附余》卷十九。
主治：虚损。
加减：None
功效：None
用法用量：每服50-60丸，温酒送下；不饮酒者，白汤送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：鹿角霜丸（《报生众妙方》卷二）。
