序号：76803
名称：黑锡丸
组成：硫黄2两（椎如皂荚子大，候铅成汁，入硫黄在内，勿令焰起，候硫黄化，倾出于9重纸，纳入一地坑，以碗盖火出），川楝子2两，黑铅（不夹锡者，先熔成汁）2两，阳起石（煅）半两，木香半两，沉香半两，青皮（炒）半两，肉豆蔻1两，茴香（炒）1两，官桂（去皮，不见火）1两，附子（炮，去皮脐）1两，葫芦巴（炒）1两，破故纸（炒）1两，乌药（去木，锉）1分。
出处：《三因》卷七。
主治：阴阳不升降，上热下冷，头目眩晕，病至危笃，或服暖药，僭上愈甚者。
加减：None
功效：None
用法用量：每服30-50至100丸，食前浓煎人参、茯苓、姜、枣汤送下。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
