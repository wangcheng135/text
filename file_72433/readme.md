序号：72449
名称：彭祖麋角丸
组成：麋角1对（炙黄），槟榔（上2味另捣，取净末）2两，通草2两，秦艽2两，人参2两，菟丝子（酒浸，别捣）2两，肉苁蓉（酒浸，去腐）2两，甘草2两（预散）。
出处：《张氏医通》卷十四。
主治：None
加减：None
功效：培理身心。
用法用量：每服30丸，空腹酒送下，日加1丸，至50丸为度，旦、暮各1服。服经1月，腹内诸疾自相驱逐，有微利勿怪。
制备方法：上以麋角、槟榔2味，共煎一食时倾，药似稠粥即止火，稍待热气歇，即投后六味散，搅令相得，仍待少时，渐稠粘堪作丸，如梧桐子大。
临床应用：None
各家论述：麋角走督脉而补阴中之阳；槟榔行腹内而破阴中之滞，兼通草、秦艽通血脉而运周身之气，菟丝、苁蓉填补肾脏，人参以助诸味之力也。
用药禁忌：百日内忌房室。
附注：None
