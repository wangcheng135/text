序号：39955
名称：肥气丸
组成：厚朴半两，黄连7钱，柴胡2两，椒（炒去汗）4钱，巴豆霜5分，川乌头（切，去皮）1钱2分，干姜（炮）半钱，皂角（去皮弦子，煨）1钱半，白茯苓（去皮）1钱半，广术（炮）2钱半，人参（去芦）2钱半，甘草（炙）3钱，昆布2钱半。
出处：《东垣试效方》卷二。
主治：肝之积，在左胁下，如覆杯，有头足，久不愈，令人发咳逆、痎疟，连岁不已。
加减：秋、冬，加厚朴半两（通前1两），减黄连1钱半；若治风痫，于1料中加人参、茯神、菖蒲各3钱，黄连只依春、夏用7钱，虽秋、冬不减，空心淡醋汤送下。
功效：None
用法用量：初服2丸，1日加1丸，2日加2丸，渐渐加至大便微溏，再从2丸加服。周而复始，积减大半勿服。
制备方法：上件除茯苓、皂角、巴豆霜外，为极细末；另研茯苓、皂角为细末，和匀；另研巴豆霜旋旋入末和匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
