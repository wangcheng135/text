序号：29671
名称：鸡鸣保肺丸
组成：知母1两，贝母1两，旋覆花1两，杏仁1两，阿胶1两，米壳1两，葶苈1两，款冬花1两，马兜铃5钱，甘草1两，清夏1两，麻黄3钱，五味子1两，广陈皮1两，桔梗1两，紫菀1两，天门冬1两，北沙参1两。
出处：《全国中药成药处方集》（沈阳方）。
主治：咳嗽痰喘，日轻夜重，感受风寒，饥饱劳役，冷热不匀，饮食过量，肺胃两伤，久咳声哑，夜间盗汗，食欲不振。
加减：None
功效：镇咳定喘，补肺化痰。
用法用量：每服1丸，开水送下，如恶寒时姜汤送下。
制备方法：上为细末，炼蜜为丸，每丸二钱重。
临床应用：None
各家论述：None
用药禁忌：忌生冷果瓜、烟、酒、肉、腥等。
附注：None
