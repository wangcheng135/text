序号：47564
名称：神仙三黄丸
组成：生地黄30斤（木臼捣取自然汁），生干地黄（焙）1斤，熟地黄（焙）1斤（为末），鹿角胶（炙燥，为末）4两，大麻仁（研）4两，干漆（捣末，点醋炒，烟尽为度）4两，甘草（炙，锉）半斤，杏仁（去皮尖双仁，研）半斤，蜜半斤。
出处：《圣济总录》卷一八五。
主治：None
加减：None
功效：平补换骨，延年驻颜。
用法用量：每服30丸，加至50丸，空心、食前面东温酒送下。
制备方法：上为末，先将无灰酒1斗、生地黄汁及蜜于银器内用慢火煎，以柳枝搅，将欲成膏，便入诸药同熬，候可丸即丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
