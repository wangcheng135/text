序号：33780
名称：沉香槟榔丸
组成：沉香半两，槟榔半两，檀香半两，南木香半两，丁皮半两，三棱（炮，锉）半两，莪术（炮，锉）半两，神曲（炒）半两，谷芽（洗，焙）半两，厚朴（去粗皮锉碎，每1斤用生姜1斤，薄片，切烂杵拌匀，酿1宿，慢火炒干用）半两，苍术（米泔水浸1宿，去粗皮，滤干，锉片，用火炒至微黄色）半两，使君子肉（锉，以屋瓦焙干）半两，青皮（去白）半两，陈皮（去白）半两，缩砂仁半两，益智仁半两，净香附半两，枳壳（水浸润，去壳，锉片，麦麸炒微黄）半两，良姜（锉，用东壁土炒）半两，粉草（炙）1两半。
出处：《活幼心书》卷下。
主治：面带痿黄，肌肤瘦弱，过食生果，停寒在里，乳癖腹胀作痛，及吐利疟肿，愈后诸疳虫积；痃痛。
加减：None
功效：和脾助胃，进食清神，宽胸快膈，顺气调中，悦颜色，壮筋骨。
用法用量：每服30-50丸，以温米清汤送下，不拘时候。儿小者不能吞咽，炼蜜为丸，如芡实大。每服1-2丸，以温汤化服。
制备方法：上除沉香、槟榔、檀香、木香、丁皮不过火，余十五味锉，焙，仍同沉香等为末，水煮面糊为丸，如麻仁大。
临床应用：None
