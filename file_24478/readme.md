序号：24486
名称：防风汤
组成：防风（去叉）1两，芎1两，麻黄（去根节）1两，独活（去芦头）3分，桂（去粗皮）3分，前胡（去芦头）3分，五味子3分，附子（炮裂，去皮脐）3分，杏仁（汤浸，去皮尖双仁，麸炒）3分，人参3分，茯神（去木）3分，细辛（去苗叶）半两，甘菊花半两，黄耆半两，山茱萸半两，甘草（炙，锉）半两。
出处：《圣济总录》卷十九。
主治：肺中风寒湿，项强头昏，胸满短气，嘘吸颤掉，言语声嘶，四肢缓弱，皮肤（疒帬？？）痹。
加减：None
功效：None
用法用量：每服4钱匕，水1盏半，加生姜5片，煎至8分，去滓稍热服，不拘时候。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
