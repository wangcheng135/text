序号：52965
名称：香橘饼
组成：南木香2钱半，陈橘皮（去白）2钱半，青皮（去白）2钱半，厚朴（去粗皮，锉碎，每1斤用生姜1斤薄片，切、烂待拌匀，酿1宿，慢火炒干用）7钱，缩砂仁5钱，神曲（湿纸裹炮）5钱，麦芽（净洗，焙）5钱，三棱（炮，锉）3钱。
出处：《活幼心书》卷下。
主治：婴孩过伤乳食，或吐或泻，及病后虚中感积成痢，气弱神昏，面黄目慢。
加减：None
功效：None
用法用量：每服1-3饼，用大枣汤化开，空心温服，米清汤亦可。
制备方法：上除木香不过火外，余7味锉、焙，仍同木香研为细末，炼蜜捻作饼子，如芡实大。
临床应用：None
各家论述：None
用药禁忌：None
附注：香橘丸（《慈幼新书》卷十）。
