序号：74572
名称：新定拯阴理劳汤
组成：牡丹皮1钱，当归身1钱（酒洗），麦门冬1钱（去心），甘草（炙）4分，苡仁3钱，白芍药7分（酒炒），北五味3分，人参6分，莲子3钱（不去皮），橘红1钱，生地黄2钱（忌铜铁器，酒、姜水炒透）。
出处：《医宗必读》卷六。
主治：阴虚火动，皮寒骨热，食少痰多，咳嗽短气，倦怠焦烦。
加减：肺脉重按有力者，去人参；有血，加阿胶、童便；热盛，加地骨皮；泄泻，减归、地，加山药、茯苓；倦甚，用人参3钱；咳者，痰燥也，加贝母、桑皮；嗽者，湿痰也，加半夏、茯苓；不寐，加枣仁；汗多亦用。
功效：None
用法用量：水2钟，加大枣1枚，煎1钟，分2次徐徐呷之。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：拯阴汤（《证治汇补》卷二）、救阴理劳汤（《冯氏锦囊·杂证》卷一）、拯阴理劳汤（《杂病源流犀烛》卷二）。
