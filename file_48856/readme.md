序号：48871
名称：秋莲丸
组成：莲肉1斤（先用酒浸1宿，待透入，于猪肚内线缝合，却用酒煮熟，取出晒干，猪肚不用），苍术（用茅山者）1斤（分作4份：一份用米泔水浸，小茴香、破故纸各1两同炒；一份用酒浸，川乌、川楝子肉各1两同炒；一份用醋浸，川椒1两同炒；一份用盐水浸，黑牵牛1两同炒，去牵牛），木香2两，五味子2两，枸杞子2两，熟地黄2两，牛膝2两，肉苁蓉2两，鹿茸2两。
出处：《奇效良方》卷二十一。
主治：诸虚百损。
加减：None
功效：壮筋骨，健脾胃，去痰涎，除风湿，添精气，和血驻颜，牢牙齿，乌髭发，理腰疼，疗疝气。
用法用量：每服50丸，空心温酒或盐汤送下。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
