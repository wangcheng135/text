序号：67346
名称：麻黄温痹汤
组成：麻黄10g，羌活10g，独活10g，制川乌10g，制草乌10g，八里麻1g，桂枝10g，黄耆20g，川牛膝12g，木瓜12g，威灵仙12g，鸡血藤10g，细辛3g，制附块10g，伸筋草10g，寻骨风10g，苍耳子10g，秦艽10g，桑寄生10g，炙甘草10g。
出处：《千家妙方》卷上引范国（木梁）方。
主治：风寒湿邪侵袭经络，留滞关节，关节肿大疼痛者。
加减：气血虚亏，可酌加党参、枸杞、当归、白芍以扶正气。
功效：祛风散寒，舒筋活络。
用法用量：水煎服，每日1剂。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
