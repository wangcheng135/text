序号：4011
名称：日精月华光明膏
组成：好琥珀1钱半（别研），马牙消（飞过）2钱半，铜绿1钱半，真胆矾1钱半（别研），硼砂1钱半（别研），没药4两（别研），乳香3钱（别研），青盐1钱半，朱砂1钱半，轻粉1钱（别研），麝香半钱（别研），片脑半钱（别研），防风1钱，天花粉半钱（前药各研，候后药成膏却下），黄连4钱（研），当归1两（研），诃子1对（去核，研），石决明2两（去瓤，细研），石膏1两半（碾，用腊八水或雪浸3日），大鹅梨20个（捶碎，用布扭去滓），猪胰2具（用草夹细，去筋膜），炉甘石4两（童便浸，淬烧5次），黄丹1两（用腊八雪浸3日）。
出处：《普济方》卷七十九。
主治：一切内障，翳膜遮睛，及攀睛胬肉，无问年久日深，或一目两目俱患，但能见人影者。
加减：None
功效：None
用法用量：以铜箸点眼。
制备方法：上先用黄连五味浸3日，却用大砂锅1口，纳药水，再添满7分，熬，重绵滤过。至4-5碗，却将大鹅梨、猪胰2味入内，又熬至3碗，再用滤过。再入锅，下炉甘石、黄丹，同熬至2碗，又滤过，却下马牙消等12味，不住手用槐柳条搅匀，候成膏，如前滤净，入瓶内，却入麝香、片脑等3味，十分搅匀。用油纸重封，无令水入，放冷水浸3日，然后旋取膏入瓶内。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
