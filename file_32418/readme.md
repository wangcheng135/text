序号：32427
名称：妙功丸
组成：丁香半两，木香半两，沉香半两，乳香（研）2钱半，麝香（另研）2钱半，熊胆2钱半，白丁香300粒，轻粉4钱半，雄黄（研）半两，青皮（去白）半两，黄芩半两，胡黄连半两，黄连1两，黑牵牛（炒）1两，荆三棱（煨）1两，甘草（炙）1两，蓬莪术1两，陈皮（去白）1两，雷丸1两，鹤虱1两，大黄1两半，赤小豆300粒，巴豆7粒（去皮心、膜、油）。
出处：《准绳·类方》卷五。
主治：诸痫。
加减：None
功效：None
用法用量：每服1丸，用温水浸1宿，去水，再用温水化开，空心服之。小儿加减服。10年病1服即愈，若未愈，3-5日再服，重者不过3服。
制备方法：上为细末，荞面一两半作糊和匀，每1两作10丸，朱砂1两水飞为衣，阴干。
临床应用：癫痫：有一人好酒，得痫病二十年，用药一服，取下虫一条，约长4-5寸，身有鳞，其病遂愈。
各家论述：None
用药禁忌：None
附注：None
