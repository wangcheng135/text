序号：57461
名称：涤痰汤
组成：南星（姜制）2钱半，半夏（汤洗7次）2钱半，枳实（麸炒）2钱，茯苓（去皮）2钱，橘红1钱半，石菖蒲1钱，人参1钱，竹茹7分，甘草半钱。
出处：《奇效良方》卷一。
主治：中风，痰迷心窍，舌强不能言。
加减：None
功效：豁痰清热，利气补虚。
用法用量：上作1服。水2钟，加生姜5片，煎至1钟，食后服。
制备方法：None
临床应用：None
各家论述：《医方集解》：此手太阴、足太阴药也。心脾不足，风邪乘之，而痰与火塞其经络，故舌本强而难语也。人参、茯苓、甘草补心益脾而泻火；陈皮、南星、半夏利气燥湿而祛痰；菖蒲开窍通心，枳实破痰利膈，竹茹清燥开郁，使痰消火降，则经通而舌柔矣。
用药禁忌：None
附注：涤痰散（《兰台轨范》卷二）。本方制成丸剂，名“涤痰丸”（见《丸散膏丹集成》）；其用量、用法为：为细末，用胆星烊化泛丸，如梧桐子大。每服二三钱，熟汤送下。
