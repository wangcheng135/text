序号：73901
名称：硼砂丸
组成：硼砂半两（细研），干姜半两（炮裂，锉），川乌头1两（炮裂，去皮脐），芫花半两（醋拌炒令干），皂荚半两（去黑皮，涂酥炙令焦，去子），京三棱半两（微煨，锉），五灵脂1两，巴豆半两（去皮，细研，纸裹压去油）。
出处：《圣惠》卷四十九。
主治：癖气，在两胁结聚不散。
加减：None
功效：None
用法用量：每服2丸，空心及临卧时以生姜橘皮汤送下。
制备方法：上为末，入硼砂、巴豆，同研令匀，用醋熬为膏，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“硇砂丸”，《鸡峰》作“大硇砂丸”。方中硼砂，《普济方》、《鸡峰》均作“硇砂”。
