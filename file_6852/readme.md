序号：6852
名称：打老儿丸
组成：石菖蒲（去须毛，嫩桑枝条拌蒸，晒干，不犯铁器）、干山药（蒸出晒干）、川牛膝（去头，用黄精自然汁浸，漉出，酒浸1宿，若无黄精，酒浸3日，漉出，细锉，焙干）、远志（去心，甘草汤浸1宿）、巴戟（去心，枸杞子汤浸1宿，漉出，酒浸1伏时，菊花同焙令黄，去菊花）、续断（去筋，酒浸1伏时，焙干）、五味子（蜜浸蒸，从已至申。又以浆水浸1宿，焙干）、楮实子（水浸3日，去浮者，晒干，酒浸1伏时，漉出蒸，从己至亥，焙干）、杜仲（去皮，酥蜜炒去丝）、山茱萸（取肉，暖火焙干）、茯神（去皮心，捣细，于水盆内搅，去浮者）、熟地（瓷锅柳木甑蒸之，摊令气歇，拌酒再蒸，晒干，勿犯铜铁器）、小茴香（酒浸1宿，炒）、肉苁蓉（酒浸1宿，刷去沙土浮甲，劈破中心，去白膜）、枸杞子各等分。
出处：《万氏家抄方》卷四。
主治：五劳七伤，体虚羸弱，尿频阳萎，遗精白浊，久无子嗣；脾肾虚寒，血气两亏，不思饮食，发热盗汗，遗精白浊，肌体瘦弱，牙齿浮痛；诸虚百损。
加减：None
功效：补精生血，益气力，健筋骨，久服却病延年。
用法用量：每服30丸，空心温酒送下；或白汤送下亦可。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：延寿丹（《医统》卷九十三）、还少丹（《一草亭》）。《一草亭》无五味子，用法以枣肉二百枚捣和，加炼蜜为丸。
