序号：76948
名称：黑神丸
组成：硫黄（研）1两，丹砂（研）1两，雄黄（研）1两，水银2两（上4味同结沙子），铅丹（研）2两，消石（研）2两，定粉（研）2两，乌蛇肉1两，藿香叶1两，干蝎（去土，炒）1两，白僵蚕（炒）1两，麻黄（去根节）1两，天麻1两，天南星（炮，去皮）1两，白附子（炮）1两，白芷1两，附子（炮裂，去皮脐）1两，麝香（研）1两。
出处：《圣济总录》卷六。
主治：破伤中风，身体强直，口噤不开；伤寒，一切风痰。
加减：None
功效：None
用法用量：每服1丸，伤寒，生姜、葱、酒磨下；中风，豆淋酒磨下；一切风痰，荆芥、薄荷汤磨下。
制备方法：上将前4味先结沙子，研末，次入铅丹、消石、定粉3味同炒黑色住火，次10味并用酒浸1宿，焙干，捣罗为末，与前药并后麝香合研令匀，用头醋煎膏为丸，如皂子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
