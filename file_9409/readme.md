序号：9409
名称：牛黄丸
组成：牛黄半两（以熟绢袋盛于黑豆1升中，炒豆熟为度，别研入），犀角屑半两，天麻1分，天竹黄半两（细研），白僵蚕半两（微炒），郁金1分，地龙半两（微炒），蜣蜋1分（去翅足，微炒），麝香1分（细研），朱砂1两（细研，水飞过），干蝎半两（微炒），天南星半两（炮裂），蚱蝉17枚（去翅足，微炒），白附子半两（炮裂），乌蛇肉2两（酒浸，炙微黄），乌鸦1枚（去翅足，泥裹烧为灰，用1两）。
出处：《圣惠》卷八十三。
主治：小儿中风，四肢不随，心神迷闷。
加减：None
功效：None
用法用量：每服5丸，以温酒送下。
制备方法：上为末，入研了药令匀，以糯米饭为丸，如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
