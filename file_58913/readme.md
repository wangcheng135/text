序号：58928
名称：秦皮散
组成：秦皮（去粗皮）2两，瞿麦穗1两半，升麻1两半，枳壳（去瓤，麸炒）1两半，黄连（去须）1两半，前胡（去芦头）1两半，栀子仁1两半，菥蓂子1两，车前子1两，大蓝实1两，防风（去叉）1两，决明子（炒）1两，苋实1两，铃羊角（镑）1两，黄柏（去粗皮，炙）1两。
出处：《圣济总录》卷一一二。
主治：眼昏晕，不以年月深浅，恐变为内障。
加减：None
功效：None
用法用量：每服20丸，食后米饮送下，临卧再服。加至30丸。
制备方法：上为散。炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，据剂型，当作“秦皮丸”。《普济方》本方用法：上为散，每服三钱，食后、临卧温浆水汤调下。
