序号：50381
名称：茵芋散
组成：茵芋（去茎）半两，杜仲（去粗皮，炙，锉）半两，石南半两，石龙芮半两，羊踯躅（微炒）半两，麝香（研）半两，狗脊（去毛）半两，当归（锉，炒）半两，干蝎（微炒）半两，桑螵蛸（微炒）半两，菖蒲半两，赤箭3分，独活（去芦头）3分，附子（炮裂，去皮脐）3分，天雄（炮裂，去皮脐）3分，甘菊花3分，牛膝（去苗，酒浸，切，焙）3分，木香3分，麻黄（去根节，煮，掠去沫，焙）3分，芎藭3分，萆薢（锉）1两。
出处：《圣济总录》卷十九。
主治：肾脏中风湿，腰痛，脚膝偏枯，皮肤（疒帬）痹，语声謇涩，两耳虚鸣，举体乏力，面无颜色，志意不乐，骨节痠疼。
加减：None
功效：None
用法用量：每服2钱匕，食前温酒调下，日2次。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
