序号：30952
名称：十珍散
组成：芫花（醋浸，焙）3钱，赤茯苓3钱，桑白皮（炒）3钱，泽泻3钱，葶苈（炒）3钱，黑牵牛（炒）3钱，川椒（并目）2钱，甘遂1钱，雄黄1钱，大戟1钱。
出处：《普济方》卷一九二。
主治：水气。
加减：先从足起，倍葶苈；先阴肿，倍泽泻；先眼肿，倍牵牛；先腹肿，倍川椒；先口肿，倍大戟；先手肿，倍桑皮；先胁肿，倍甘草；先头肿，倍芫花；先腰肿，倍茯苓；先肾肿，倍雄黄。如在先小便不通者，先服通小便方，小便通病不退，再用此方。
功效：None
用法用量：五更温酒下3钱匕。小便利则愈。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：须审度病体强弱，久近冷热，不可轻投。
附注：None
