序号：24026
名称：当归郁李仁汤
组成：郁李仁1钱，皂角仁1钱，枳实7分，秦艽5分，麻仁5分，当归梢5分，生地黄5分，苍术5分，大黄（煨）3分，泽泻3分。
出处：《兰室秘藏》卷下。
主治：痔漏大便硬，努出大肠头下血，苦痛不能忍，结热肠燥不便。
加减：None
功效：None
用法用量：除皂角仁别为末，水3盏，煎至1盏，去滓，入皂角仁末调，空心、食前服之。
制备方法：上锉，如麻豆大。
临床应用：痔漏：一男子，素不慎酒色，患痔焮肿，肛门坠痛，兼下血，大便干燥，脉洪大，按之则涩，以当归郁李仁汤加桃仁，四剂少愈；更以四物汤加红花、条芩、槐花，数剂而愈。
各家论述：None
用药禁忌：避风寒，忌房事、酒湿面、大辛热物。
附注：郁李仁汤（《医级》卷七）。
