序号：4072
名称：木瓜牛膝丸
组成：木瓜（大者）3-4个（切开盖，去瓤，先用糯米浆过，盐焙干，为末，却将盐末入瓜内令满，仍用盖针定，蒸3次，烂研作膏），川乌（大者，去皮尖，用无灰酒1升浸，薄切，酒煮干，研为膏）3两，牛膝1两（酒浸），萆薢1两，茴香1两（炒），羌活1两，青皮1两，青盐1两（别研入），狗脊1两（燎去毛），巴戟1两，海桐皮1两。
出处：《三因》卷三。
主治：寒湿脚气，冷湿下注，脚弱无力，或肿急疼痛，兼治妇人血风。
加减：None
功效：大固肾气，活血，壮筋络。
用法用量：每服50丸，食前盐汤或盐酒任下。
制备方法：上为末，入青盐和匀，将前2膏搜为丸，如硬，再入酒，杆数千下为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
