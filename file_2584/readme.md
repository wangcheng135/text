序号：2584
名称：双解饮子
组成：肉豆蔻2个，草豆蔻2个（1个用水和面裹煨，1个生用），厚朴2寸（一半用生姜汁浸、炙，一半生用），甘草（大者）2两（一半炙，一半生用），生姜2块如枣大（1块湿纸裹煨，1块生用）。
出处：《医学纲目》卷六引《局方》。
主治：疟疾及脾虚食少。
加减：None
功效：辟瘴气。
用法用量：上药各(口父)咀。每服分一半，用水1碗，煎至1大盏，去滓。空心温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：交加散（《鸡峰》卷十二）、草豆蔻散（《鸡峰》卷十四）、交解饮（《三因》卷六）、生熟饮子（《杨氏家藏方》卷三）、交解饮子（《御药院方》卷二）、交加双解饮子（《卫生宝鉴》卷十六）、交加饮子（《普济方》卷二○○）交加双解饮（《中国医学大辞典》）。
