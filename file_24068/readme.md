序号：24076
名称：安胎饮
组成：人参1钱（虚者倍用），当归2钱，熟地3钱，条芩1钱，川芎7分，白术2钱，陈皮4分，紫苏4分，甘草4分，元枣2枚。
出处：《胎产秘书》卷上。
主治：妊娠三月，胎动不安，盖因子宫久虚，气血两弱，不能摄元养胎致令不安欲堕。
加减：如虚肥人，陈皮去白，加川连5分；脾胃溏泻，加莲子10粒、砂仁5分，川连（炒）5分，去熟地、黄芩；怒而多泻，加木香3分；渴，加麦冬2钱；怔忡、惊悸，加枣仁2钱、益智仁1钱、龙眼肉10个。
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：《梅氏验方新编》本方用法：“水煎服”。
