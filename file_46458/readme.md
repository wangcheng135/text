序号：46473
名称：济阴保元汤
组成：滇珍参3钱，苡米仁4钱（拌水蒸透，咀片，再入姜，加米仁汁蒸，晒干），怀生地1两（砂仁、酒、姜3味拌蒸，九晒，收，再以瓦焙为炭），当归4钱，白芍3钱（酒炒），川芎2钱（去净油，米泔水浸洗，收干，再入酒浸），丹参4钱（酒洗透），茺蔚子4钱（酒蒸透），香附3钱（以姜、土、醋、盐、童便、甘草水、乳汁逐次制过），云白术5钱（陈土炒），女贞子3钱（以白芥、车前水浸，干用）。
出处：《纲目拾遗》卷三引《医铃》。
主治：妇人经血不调。
加减：如气血热，加丹皮、生地；气血寒，加肉桂数分；经闭，加牛膝。
功效：疏肝调经，济阴保元。
