序号：82438
名称：薯蓣丸
组成：薯蓣1两，熟干地黄1两，附子1两（炮裂，去皮脐），桂心1两，天门冬1两半（去心，焙），石斛1两（去根，锉），人参1两（去芦头），肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），远志半两（去心），鹿茸1两（去毛，涂酥，炙微黄），钟乳粉2两，白茯苓1分，菟丝子1两（酒浸3日，晒干，锉，捣），磁石1两（烧令赤，醋淬7遍，捣碎，细研，水飞过）。
出处：《圣惠》卷三十六。
主治：劳聋。脏腑久虚，肾气不足，肌体羸瘦，腰脚无力。
加减：None
功效：None
用法用量：每服30丸，空心以温酒送下，晚食前再服。
制备方法：上为末，入研了药令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中白茯苓用量原缺，据《普济方》补。
