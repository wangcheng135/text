序号：977
名称：内伤丸
组成：广七2两，桃仁2两，泽兰4两，大黄1两，明雄黄1两5钱。
出处：《中医正骨经验概述》。
主治：伤后吐血，咳血，三焦瘀血，及咳嗽、喷嚏、呼吸引起胸痛。
加减：None
功效：清热明心去内瘀。
用法用量：每服1丸，1日1至2次，饭后、睡前以白酒少许浸泡丸药，连酒服；或以童便1杯送服。不饮酒者，以白酒化开丸药后，除去余酒，用白开水送服。瘀血于下焦，服后则下出；瘀血于上焦，服后则上越。
制备方法：上各为极细末，拌匀(桃仁去油后入)，以适量蜂蜜，加入米汤内(糯米或普通米之粘稠煮液)搅合，倒入备好之药末内，调成软硬适度为丸。每丸重8分。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
