序号：40818
名称：软红丸
组成：朱砂（飞研）1分，龙脑（别研）1分，半夏（汤浸，煮洗7遍，焙）3钱，黄蜡3钱，粉霜2钱，水银1钱（入金箔3片，结沙子），牛黄半钱，腻粉半钱，蝎梢49个（微炒）。
出处：《幼幼新书》卷九引《刘氏家传》。
主治：小儿急慢惊风，惊痫涎潮，搐搦直视，牙关紧，项背强，喘咳多睡，发热不时。
加减：None
功效：None
用法用量：半岁儿可服2-3丸，荆芥、薄荷汤送下。
制备方法：上为极细末，先炼蜡去滓，入油3-5点，离火纳诸药，和搅令匀成剂。有病旋为丸，如黍粒大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《永乐大典》引作“红丸”。
