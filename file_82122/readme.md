序号：82138
名称：蝎梢半夏丸
组成：蝎梢（去刺，炒）半两，白僵蚕（生姜汁炒）半两，天南星（炮）半两，半夏（汤洗7次，用生姜汁制作曲）半两，明天麻（去芦头）半两，川独活（去芦头）半两，白花蛇（酒浸，取肉）半两，川芎半两，南青皮（去白）半两，紫苏叶半两，拣木香半两，防风（去芦头）半两。
出处：《御药院方》卷一。
主治：风壅痰实，咳嗽鼻塞，头目昏痛，手足麻木，颈项强直，筋脉不利。
加减：None
功效：祛风化痰，清爽头目。
用法用量：每服30-50丸，食后生姜汤送下。
制备方法：上用为细末，用生姜自然汁打糊为丸，如梧桐子大，以朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
