序号：82720
名称：中久丸
组成：麝香1分，乳香1钱（制），没药1钱（制），轻粉1钱，乌金石1钱，雄黄1钱，狗宝1钱，蟾酥2钱，粉霜3钱，黄蜡3钱，硇砂5钱，鲤鱼胆1个，狗胆1个，金头蜈蚣7条（全者，酥黄色），头胎男乳1两。
出处：《外科十三方考》。
主治：恶疮，身未烂者，及发背、脑疽、痈肿，遍身附骨肿痛。初发时大渴发热，四肢沉重，不论阴阳，俱可服之。
加减：None
功效：None
用法用量：每服1丸，重者3丸，用白丁香7粒(小儿减半)，研末，调冷开水送下，盖被出汗为度。如头上无疮肿者，1-2服即效。
制备方法：先将黄腊、乳汁2味熬成膏子，其余13味则共研细末，然后同黄蜡、乳汁膏调和为丸，如绿豆大(小儿服者如菜子大)。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
