名称：犀角散
组成：犀角屑1分，牛黄（别研）1分，麝香（别研）1分，羚羊角屑1分，丹砂（别研）半两，防风1分，天麻1分，独活（去芦头）1分，人参1分，茯神（去木）1分，沙参（去芦头）1分，天竺黄（别研）1分，升麻1分，龙齿1分，麦门冬（去心，焙）半两，白鲜皮1分，远志（去心）1分，龙脑（别研）半分，甘草（微炙）1分。
出处：《圣济总录》卷十九。
主治：心痹。精神恍惚，恐畏闷乱，不得睡卧，志气不定，言语错乱。
加减：None
功效：None
用法用量：每服3钱匕，煎麦门冬汤调下，不拘时候。
制备方法：上除别研者外，捣罗为散，同研药一处拌匀，再研细。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
