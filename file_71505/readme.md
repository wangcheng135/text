序号：71521
名称：循络丸
组成：五灵脂2两，防风（去芦）1两，川萆薢（炮，去皮脐）1两，狗脊（去毛）1两，大苍术（米泔浸）1两，虎骨（打碎，酒煮）1两，川乌（生用）1两，川乌1两（炮，去皮脐），没药1两（研，去砂），乳香半两（研，去砂）。
出处：《朱氏集验方》卷一。
主治：风湿流袭，足膝筋骨肉疼痛。
加减：None
功效：None
用法用量：每服30-50丸，食前温酒送下；不饮者，以木瓜汤下。此药可以常服，如痛大作，则以小续命汤加槟榔、羌活、青皮煎汤，痛止则勿服。
制备方法：上为细末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
