序号：44565
名称：威灵仙丸
组成：威灵仙（去苗土）5两，草乌头（炒，锉）7两，骨碎补（去毛）2两半，地龙（去土，炒）3两，天南星（炮）3两，自然铜（烧醋淬，研）1两半，苍耳4两，仙灵脾4两，侧子（炮裂，去皮脐）4两，防风（去叉）4两，羌活（去芦头）1两，蔓荆实（揉去皮）1两，泽泻1两，藁本（去苗土）1两，萆薢1两，独活（去芦头）1两。
出处：《圣济总录》卷七。
主治：摊缓风。脚膝无力，行履艰难，筋骨麻痹。
加减：None
功效：None
用法用量：每服5丸，渐加至15丸，空心温酒送下。觉唇口麻者，痹渐减也。
制备方法：上为末，合自然铜末研令匀，用好酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：才服后，便忌吃热物。
附注：None
