序号：13210
名称：人参丸
组成：人参3分，麦门冬（去心，焙）3分，半夏（汤浸10遍去滑）3分，黄耆（锉）3分，大黄（锉，炒）3分，苦参（锉）3分，矾石（烧汁尽，研）3分，甘草（锉，炙）3分，远志（去心）3分，黄芩（去黑心）3分，消石（研）半两，芎半两。
出处：《圣济总录》卷一七五。
主治：小儿伤食失衣，当风湿冷水浴，腹大丁奚，时下痢，寒热如疟，不欲饮食，虽食不充肌肉，又不能消化，羸瘦不耐。
加减：None
功效：None
用法用量：1-2岁儿每服3丸，米饮化下；4-5岁儿5丸，早、晚各1服。
制备方法：上为末，炼蜜为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
