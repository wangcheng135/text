序号：51166
名称：荜茇粥
组成：荜茇末2合，胡椒末1分，青梁米4合（淘）。
出处：《养老奉亲》卷一。
主治：老人冷气心痛，发动时遇冷气即痛;胃寒呕吐，食欲不振，脘腹疼痛，肠鸣泄泻。
加减：None
功效：温中、散寒、止痛。
用法用量：上以米煮作粥熟，下2味调之，空心食。常服尤效。
制备方法：None
临床应用：None
各家论述：《药粥疗法》：方中荜茇大辛大热而无毒，专入脾胃经，温胃散寒，下气止痛；胡椒入胃及大肠经，功同荜茇，二者一并煮粥，其温中散寒之力颇强，且与米配合，煮粥食用，还能温中补虚，健脾暖胃，同时也能使荜茇、胡椒的散寒作用缓缓发挥，以提高疗效。
用药禁忌：凡一切实热症及阴虚有火者忌用。
附注：None
