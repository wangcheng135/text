序号：57674
名称：大续命汤
组成：独活3两，麻黄3两，芎1两，防风1两，当归1两，葛根1两，生姜1两，桂心1两，茯苓1两，附子1两，细辛1两，甘草1两。
出处：《千金》卷八。
主治：大风经脏，奄忽不能言，四肢垂曳，皮肉痛痒不自知。
加减：若初得病，便自大汗者，减麻黄，不汗者，依方；上气者，加吴茱萸2两，厚朴1两；干呕者，倍加附子1两；啘者，加橘皮1两；若胸中吸吸少气者，加大枣12枚；心下惊悸者，加茯苓1两；若热者，可除生姜，加葛根。
功效：None
用法用量：上(口父)咀。以水1斗2升，煮取4升，分5服，老小半之。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
