序号：35321
名称：乳香黑虎丹
组成：草乌1斤，苍术1斤，生姜1斤，连须葱半斤（同均匀盫，春5，夏3，秋7，冬10日，每日拌1次，晒干），五灵脂5钱，乳香5钱，没药5钱，穿山甲2两，自然铜1两。
出处：《医学入门》卷七。
主治：男妇虚冷，血气衰败，筋骨寒冷，及外感风湿传于经络，手足麻木，腰腿疼痛，久侧偏枯瘫痪，口眼斜及诸中风不能行者。
加减：None
功效：None
用法用量：每服30丸，空心热酒送下，间日服尤妙；妇人血海虚冷，肚腹疼痛，临卧醋汤送下。只服30丸，不可过多，但觉麻木为效。
制备方法：上为末，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌生冷物，孕妇勿服。
附注：None
