序号：51754
名称：重型消黄汤
组成：茵陈90g，生石膏1两，炒知柏3钱，藿香3钱，佩兰3钱，鲜茅根1两，杏仁3钱，赤芍5钱，丹皮5钱，泽兰5钱，酒胆草5钱，六一散（包）3钱。
出处：《中医原著选读》引关幼波方。
主治：肝炎。
加减：高烧（39度以上），或兼神昏谵语者，加服安宫牛黄散4分，分2次冲服；或加服紫雪散1钱，分2次冲服；或加服局方至宝丹1丸，分2次服。便秘者，加酒军3钱，瓜蒌5钱。退黄时茵陈的用量要大，最大可用4两，并加活血药，以加速退黄的作用。
功效：清热利湿，活血解毒，芳香透表。
