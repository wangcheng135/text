序号：72652
名称：款冬花丸
组成：款冬花2两，石斛（去根）2两，紫菀（去苗土）2两，细辛（去苗叶）2两，防风（去叉）2两，芎2两，人参2两，当归（切，焙）2两，藁本（去苗土）2两，甘草（炙，锉）2两，蜀椒（去目并闭口，炒出汗）2两，白术（锉）2两，天雄（炮裂，去皮脐）2两，菖蒲（切）2两，麻黄（去根节，汤煮掠出沫）2两，半夏（汤洗7遍去滑，焙，生姜汁制）2两，桂（去粗皮）半两，独活（去芦头，锉）半两，芫花（醋浸，炒干）2两，钟乳粉（研）2两，桃仁（汤浸，去皮尖双仁，研）2两。
出处：《圣济总录》卷六十六。
主治：积年咳嗽，唾脓血，喘急不得卧。
加减：None
功效：None
用法用量：每服20丸，煎桑根白皮汤送下，日3夜1。
制备方法：上为末，和匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
