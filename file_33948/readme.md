序号：33957
名称：补肾丸
组成：麦门冬（去心，焙）、远志（去心）、干姜（炮）、防风（去叉）、乌喙（炮裂，去皮脐）、枸杞根、牛膝（去苗，酒浸，切，焙）、萎蕤、肉苁蓉（酒洗，切，焙）、棘刺、菟丝子（酒浸1宿，别捣）、桂（去粗皮）、厚朴（去粗皮，生姜汁炙）、防葵、石龙芮、萆薢、山芋各等分。
出处：《圣济总录》卷八十六。
主治：虚劳肾气不足，膝胫痛，阳气衰弱，小便数，囊冷湿，尿有余沥，精自出，阴痿不起，悲恚消渴。
加减：None
功效：None
用法用量：每服10丸，加至20丸，食前温酒送下，日3次。
制备方法：上为末，炼蜜和鸡子白为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
