序号：46424
名称：济世仙丹
组成：明雄5钱，火消4钱，白芷1钱，枯矾1两，牙皂5钱，菖蒲5钱，丁香3钱，荜茇3钱，北细辛4钱，苍术5钱，麝香3分，冰片7分，地胡椒3钱（即鹅不食草）。
出处：《集成良方三百种》卷上。
主治：痧症暴死，阴阳脱缩，阴寒证，胸腹积滞，羊毛寒疹，恶心翻胃，霍乱吐泻，气痛血痛，鼻塞头痛，中风中痰，风疳痢疾，山岚瘴气，天行瘟疫，牙关紧闭，人事不知，以及四时不正之气。
加减：None
功效：回阳救逆，起死回生。
用法用量：以此点眼角内；或用姜汤送下1-2分；或吹入鼻孔内；或以1-2分装入肚脐，外以膏药贴住。
制备方法：上为细末，过筛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
