序号：83001
名称：乌子肝丸
组成：黑羊肝1副（竹刀切片，摆瓷盆内，外用羊胆汁涂于肝上，晒干又涂，涂了又晒，约涂200-300胆汁为佳，即少亦须涂100个，晒极干），当归4两，生地10两（6两蒸熟，4两酒洗），白芍（酒炒）4两，川芎4两，何首乌4两（9蒸），盆子4两（炒），旱莲草4两（酒蒸），山萸4两（酒蒸），白茯苓4两，少壮血余灰4两。
出处：《奇力类编》卷上。
主治：None
加减：None
功效：乌须黑发，聪耳明目。
用法用量：每服100丸，早、晚白汤送下。
制备方法：上为末，再用大熟地12两，酒煮烂，捣入前药，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：诸药不可犯铁器。
附注：乌羊肝丸（《种福堂方》卷三）。
