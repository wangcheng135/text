序号：41590
名称：金粟黄芽丹
组成：丹母、大朱砂1斤，真金箔50片。
出处：《得效》卷八。
主治：诸虚百损，五劳七伤，八风五痹，沉寒痼冷，水肿蛊气，久痢久疟，一切男女老幼困笃之疾，百药不能疗者。
加减：None
功效：起死回生，脱胎换骨。
用法用量：依前汤引服。服时以平旦取1粒，在手内搓令暖，置口中。可加至10粒，或用枣肉丸，以井花水或人参枣汤送下。
制备方法：以大朱砂1斤抽汞，置丹母于鼎内，以汞安于丹母上，复以真金箔50片，于静室养火，朝东北，暮西南，1月后开鼎。如觉丹母伤火，以浓煎沉香水浴。一载后，丹成如谷芽样，或如花果样。用银剪刀剪下，入大萝卜内蒸一伏时，以萝卜深黑为度。取出用玉石杵研为细末，以木蜜为丸。丸如粟大。
临床应用：None
各家论述：None
用药禁忌：孕妇不可服。绝嗜欲，断五腥，不食鸡、鱼、大蒜、陈臭等物。
附注：原书注：此丹无方，惟有丹母，不知是何药修炼得成。
