序号：43755
名称：厚朴七物汤
组成：厚朴半斤，甘草3两，大黄3两，大枣10枚，枳实5枚，桂枝2两，生姜5两。
出处：《金匮》卷上。
主治：病腹满，发热10日，脉浮而数，饮食如故。腹满气胀。
加减：呕者，加半夏5合；下利，去大黄；寒多者，加生姜至半斤。
功效：None
用法用量：上药以水1斗，煮取4升。每服8合，温服，每日3次。
制备方法：None
临床应用：None
各家论述：1.《沈注金匮要略》：此有表证腹满也。发热十日之久，脉尚浮数，当责风邪在表。然风气内通于肝，肝盛乘胃，故表见发热，而内作腹满；风能消谷，即能食而为中风，所以饮食如故。用小承气荡涤肠胃之热，桂、甘、姜、枣调和营卫，而解在表之风耳。_x000D_
2.《张氏医通》：此本小承气合桂枝汤，中间裁去白芍之酸收，不致引邪入犯营血。虽同用桂枝、甘草，与桂枝汤泾渭攸分。其厚朴独倍他药，正以泄气之浊逆耳。
用药禁忌：忌海藻、菘菜、生葱、羊肉、饧。
附注：厚朴七味汤（《外台》卷七）、七物厚朴汤（《袖珍》卷三引《圣惠》）。
