序号：40331
名称：茄子根浸酒
组成：茄子根3斤（洗令净，晒干），苍耳子1升（微炒，捣碎），鼠粘子1升（微炒，捣碎），牛膝1斤（去苗），牛蒡根1斤，防风3两（去芦头），萆薢1两，桂心2两，羌活3两，秦艽2两（去苗），附子1两（炮裂，去皮脐），晚蚕砂半斤，败龟2两，大麻子1升，虎胫骨（涂酥，炙微黄），枸杞子1升半（半蒸，半微炒）。
出处：《圣惠》卷二十五。
主治：风毒攻注，腰脚骨髓疼痛，皮肤冷痹，筋脉拘挛，屈伸不得。
加减：None
功效：None
用法用量：每日空腹、午时、近夜各温饮1盏，常令醺醺为妙。
制备方法：上锉细，用生绢袋盛，以无灰酒5斗浸之，封闭勿令透气，经10日后开取，开时不得面向瓶口。
临床应用：None
各家论述：None
用药禁忌：忌毒滑、鱼肉、动风物。
附注：None
