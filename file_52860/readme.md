序号：52875
名称：香黄丸
组成：硫黄1两（研），硇砂1两（研，水煎成霜），生木瓜1枚（去皮，切取盖子，剜瓤尽，入硫黄、硇砂末在内，却盖，竹签定，蒸1复时，研膏），肉豆蔻（去壳，炮）3枚，槟榔3枚（生锉），当归（切，焙）半两，石斛（去根）半两，牛膝（酒浸，切，焙）半两，附子（炮裂，去皮脐）半两，巴戟天（去心）半两，肉苁蓉（酒浸，切，焙干）半两，茴香子（炒）半两，木香半两，沉香（锉）半两，白茯苓（去黑皮）1分，京三棱（炮，锉）1分，干姜（炮）1分，丁香1分，麝香1分，乳香（别研）1分，人参1分，桂（去粗皮）1分，荜澄茄1分，阿魏（醋化面调作饼子，炙干）1分。
出处：《圣济总录》卷一八六。
主治：痼冷。
加减：None
功效：补骨髓，益真气，润皮肤，悦颜色。
用法用量：每服20-30丸，空心、食前温酒送下。
制备方法：除木瓜膏外，为末，入木瓜膏为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
