序号：41967
名称：青龙妙应丸
组成：穿山甲15片（石灰炒），全蝎（去毒）3-7个，地龙（去土）1两，蜈蚣7条（生用），麝香1字（别研），草乌（生，去皮）1两，没药3钱（别研），乳香3钱（别研），松香半两，斑蝥7个（糯米炒，去头足），白僵蚕（姜汁炒）半两，五灵脂3钱（去砂石）。
出处：《医方类聚》卷二十一引《济生》。
主治：诸风挛急，遍体疼痛，游走无定，百药之所不效者。
加减：None
功效：None
用法用量：每服20丸，温酒送下，不拘时候。
制备方法：上为细末，酒糊为丸，如绿豆大，以青黛为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：妙应丸（《得效》卷十三）。
