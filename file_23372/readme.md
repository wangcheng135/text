序号：23380
名称：防风独活汤
组成：防风1两（去叉），独活1两（去芦头），秦艽1两（去苗土），黄耆1两，芍药1两，人参1两，茯神1两（去木），白术1两（锉，炒），芎1两，山茱萸1两，薯蓣1两，桂1两（去粗皮），天门冬1两（去心，焙），麦门冬（去心，焙）1两，厚朴4两（去粗皮，生姜汁炙），羚羊角4两（镑），升麻4两，甘草4两（炙），丹参4两，牛膝4两（去苗，酒浸，切，焙），五加皮4两，石斛4两（去根），地骨皮4两，远志（去心）4两，附子3两（炮裂，去皮脐），陈橘皮3两（汤浸，去白，焙），麻黄（去根节，先煎，掠去沫，焙）3两，甘菊花1升（半开者，微炒），薏苡仁1升，石膏6两（碎），熟干地黄（焙）6两。
出处：《圣济总录》卷五。
主治：中风初得，四肢不收，心神昏愦，眼不识人，不能言语，服防己竹沥汤后，风若未除者。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，入生姜半分（切），煎至8分，去滓，空心、日午、夜卧各温服。如觉心膈虚烦满闷，气喘面赤，即与荆沥汤相间服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
