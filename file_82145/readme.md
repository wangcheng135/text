序号：82161
名称：蝎附散
组成：天南星1枚（重2两者，捶碎），附子1枚（重7钱者，捶碎，上2味用生姜4两取汁，入好酒1盏，于银石器中同煮令汁尽，焙干为末），白附子7枚（炮裂），全蝎7枚（去毒，微炒），辰砂半两（别研），代赭2两（火煅，研细）。
出处：《杨氏家藏方》卷十七。
主治：小儿吐泻日久，或大病后生风，时发搐搦，目睛斜视，手足瘈疭，冒闷昏塞，身体强硬，角弓反张。
加减：None
功效：None
用法用量：每服1字，用酸浆水半盏，入麻油两滴，冬瓜子3粒，同煎3-5沸，放冷，取1茶脚许调服，候少时再服1服；如无酸浆水并冬瓜子，只用薄荷汤调亦得，不拘时候。
制备方法：上为细末，入脑子、麝香各1钱，研匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
