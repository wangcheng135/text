序号：57740
名称：烧肝散
组成：赤芍药半两，桔梗半两，红豆半两，白术半两，川椒（炒，去目）半两，胡椒半两，肉豆蔻半两，陈皮（去白）半两，缩砂仁1两，南附子（炮）1两，茴香（炒）1两，良姜（炒）1两，干姜（炮）1两，苍术（炒）1两，官桂1两，丁香2钱，干山药4两，木香2钱，吴茱萸3钱（炒）。
出处：《医方类聚》卷一四一引《经验秘方》。
主治：年深泄泻滑肠，一切痢。
加减：None
功效：None
用法用量：每服3钱，用羊肝或猪肝3两，薄批开，掺药在上，铺生姜、葱丝1重卷定，麻札纸数重，水湿，慢火内烧熟，细嚼，食前米饮汤下，日3次。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
