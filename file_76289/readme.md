序号：76305
名称：紫霞丹
组成：肉苁蓉（酒洗，去甲并内白膜，晒干）7钱，白茯苓（坚白无筋者，去皮）3钱，生地黄（酒浸，蒸，晒）3钱，鹿茸（慢火酥炙3次，另研）、雄雀脑7个，雌雄乌鸡肝2具（慢火瓦上焙），雄鸡肾2付（酒沃，慢火炙干，另研）。
出处：《扶寿精方》。
主治：None
加减：None
功效：固阳注颜，益精填髓，起痿延年。
用法用量：以鸡子12枚，每头开1小孔，去清黄净，盛丸在内，以纸壳封其孔，另将鸡子4枚同前12枚作1窝，与一伏鸡抱至4枚小鸡出为度，贮瓷器内，用麝少许，铺器内底，盖固封养7日方服。每服10丸，空心盐酒汤送下，干物压之，久久精自不泄。欲生子，以青黛、甘草、陈壁土调水饮之。
制备方法：上为细末，先将葱白10两，净苎麻叶包裹，外用绵纸三四层，水湿固之，火上煨熟，取起捣烂，合前药末杵千余下，为丸如梧桐子大，晒干。
临床应用：None
各家论述：None
用药禁忌：None
