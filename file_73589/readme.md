序号：73605
名称：蓬莱枣
组成：北红枣1斤（取肥大不破者，用千年叶、川椒煎汤，洗去枣上泥，阴干），五台草取自然汁10碗（一名猫耳眼，又名浓灌草），透骨草（即马鞭草）取汁1碗，左缠藤（即金银花）取汁1碗，夏枯草取汁1碗，透天龙（即茜草）取汁1碗，土风藤（即九龙草）取汁1碗，蒲公英取汁1碗，黄花根1两。
出处：《解围元薮》卷三。
主治：瘫痪、痿烂，臭恶困顿者。
加减：None
功效：None
用法用量：每服先吃1枚，3日外增3枚，5日外增5枚，常服之。如病人手指挛瘸屈倒5年者服2斤，10年者服4斤，二十年者不治。服时要在静处避风端坐养神。仍以扁柏、川椒汤洗手。其煮过枣汁，为疮疽围药，极好。或以金银花藤（蒸晒）为末，和为丸，外科服之甚妙，风科服之亦好。
制备方法：上各草于二月中旬收采，加白酒浆2碗，共入砂锅内文火慢煎汁尽，外用沉香、麝香末为衣更妙。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
