序号：59001
名称：秦艽酒
组成：秦艽1两，牛膝1两，芎藭1两，防风1两，桂1两，独活1两，茯苓1两，杜仲8两，丹参8两，侧子（炮裂，去皮脐）1两半，石斛（去梢，黑者）1两半，干姜（炮）1两半，麦门冬（去心）1两半，地骨皮1两半，五加皮5两，薏苡仁1两，大麻仁1合（炒）。
出处：《圣济总录》卷五十一。
主治：忧恚内伤，久坐湿地所致肾劳虚冷，干枯。胞痹，小便不利。
加减：None
功效：None
用法用量：以生绢袋盛，酒1斗浸，春秋7日，夏3日，冬10日成。每日空腹温服半盏，日2次。
制备方法：上锉细。
临床应用：None
各家论述：None
用药禁忌：None
附注：牛膝汤（《普济方》卷一八六）。
