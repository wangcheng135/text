序号：48609
名称：神验百子丸
组成：何首乌（赤白相半，先用米泔水浸2日夜，或竹刀、铜刀去粗皮，切成片，略晒爽，莫犯铁物，用黑芝麻拌匀蒸1次，晒干去芝麻，再用乌羊肉切成片，去筋膜油腻拌匀蒸1次，去羊肉晒干，再用极好无灰酒浸湿蒸1次，四用黑豆1层，首乌1层，又蒸1次，去豆晒干。蒸法俱用砂锅柳甑拌匀，蒸透熟为度，晒干，拣净，用石磨磨成细末，以1斤为祖）、熟地黄、天门冬（2味用生姜自然汁浸2日夜，晒干待用）、人参（去芦，净末）5钱，生地黄、麦门冬（2味用无灰酒浸1日，夜取出再用米泔水浸1日夜，取出晒略干），上4味，俱用石磨磨如泥浆，用杏仁（去皮尖）煎汤，化开前药，滤出滓再磨，磨尽如澄水粉样，待澄清，撇去清水，将药粉晒干细末各1两。_x000D_
白茯苓（去粗皮，为末，用童便泛去浮筋，用泥底者浸1夜，晒干末）4两，地骨皮（去骨，无灰酒浸1日，滤出，为细末）1两，牛膝（去芦，无灰酒浸1日夜，滤出，为细末）2两。
出处：《墨宝斋集验方》卷上。
主治：None
加减：None
功效：广嗣。
用法用量：每服50-60丸，用无灰好酒送下，晨昏各进1次。
制备方法：上为细末，用无疾好妇人养男孩的好乳汁6两，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌诸般血、豆腐、萝卜、大蒜、莲蓬、藕、败血之物。
附注：None
