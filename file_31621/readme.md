序号：31630
名称：羌活补髓丸
组成：羌活3两，芎3两，当归3两，桂心2两，人参4两，枣肉（研如脂）1升，羊髓1升，酥1升，牛髓2升，大麻仁2升（熬研如脂）。
出处：《千金》卷十二。
主治：髓虚脑痛不安，胆腑中寒。
加减：None
功效：None
用法用量：每服30丸，稍加至40丸，以酒送下，1日2次。
制备方法：上先捣五种干药为末，下枣膏、麻仁又捣，相濡为一家。下二髓并酥，纳铜钵中，重汤煎之，取好为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：补髓而用羌活之走督脉，桂心之温髓脏，芎之逐脑风，当归之润血脉，麻仁之滋津液，乳酥、牛羊髓之透骨髓，参、枣之温脾气，脾气温而髓腑完固，脑病自安然。
用药禁忌：None
附注：羌活丸（《普济方》卷三十三引《圣惠》）、补髓羌活丸（《圣济总录》卷五十三）。
