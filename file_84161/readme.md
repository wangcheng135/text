序号：84179
名称：五石护命散
组成：炼成钟乳1两，紫石英2两（细研，水飞过），白石英2两（细研，水飞过），硫黄1两，赤石脂2两，海蛤2两（细研），防风3分（去芦头），黄耆1两（锉），麦门冬2两（去心，焙），生干地黄1两，桂心3分，桔梗1两（去芦头），栝楼根1两，白术1两，干姜1两（炮裂，锉），细辛1两，人参1两（去芦头），附子3两（炮裂，去皮脐）。
出处：《圣惠》卷三十八。
主治：虚劳百病，羸瘦，咳逆短气，骨间有热，四肢烦痛，或腹鸣(疒丂)痛，大小便不利，尿多赤黄，头眩冒闷，恶寒风痹，食饮不消。
加减：None
功效：None
用法用量：每服2钱，空心及晚食前以温酒调下。服药后稍有力者，宜行百余步。所贵药势归下。
制备方法：上为细散，入研了药和令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
