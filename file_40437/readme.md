序号：40447
名称：虎胫骨丸
组成：虎胫骨5钱，熟地黄5钱，木瓜2钱，牛膝2钱，杜仲2钱，附子2钱。
出处：《霉疮证治》卷下。
主治：霉疮，毒结肌肉关节疼痛，或结毒偏枯。
加减：None
功效：None
用法用量：每服50丸，早旦、临卧空心温酒送下。
制备方法：上为细末，面糊为丸，如梧桐子大。
临床应用：阴疮横痃一奴年三十，尝患阴疮横痃，五宝、紫金、轻粉之类无所不用，肌肉枯瘦，两臂、两踝生瘿瘤大如拳，时常苦疼不能步履，两臂亦不能举，主人怜养十年于此。余施以家方虎胫骨丸，一月半两，踝瘿瘤溃烂如翻花，脓水淋沥，痛苦顿减，渐能步履，两臂痛亦随减矣。然溃烂臭秽不可近，教之浴于上野州草津温泉一月余，十年之病全愈。
各家论述：None
用药禁忌：None
