序号：59605
名称：调中健脾丸
组成：黄耆2两（蜜炙），人参2两，白术6两（土水拌炒），茯苓2两，陈皮3两（盐水制），紫苏子2两半（炒），萝卜子1两半（炒），山楂肉3两（炒），草豆蔻1两（酒炒），泽泻3两半，薏苡仁3两（炒），沉香6钱（另研），五加皮3两（炒），瓜蒌1两（用大瓜蒌2个，钻1孔，每个入川椒3钱，多年粪底1钱，敲米粒大，俱纳入瓜蒌内，外以绵纸糊完，再用绵筋、盐泥封固，炭火燃煅通红为度，取出择去泥，其黑皮一并入药）。
出处：《古今医鉴》卷六。
主治：单腹胀及脾虚肿满，膈中闭塞及胃口作痛。
加减：None
功效：None
用法用量：每服100丸，日进3次，白汤送下。
制备方法：上为细末，煎荷叶、大腹皮汤，打黄米糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：此药不伤脾气，大有补益。
