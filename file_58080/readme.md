序号：58095
名称：益寿固元膏
组成：熟地9两，杜仲3两，枣仁1两8钱，五味子3两，虎骨6两，远志1两8钱，吴萸3两，首乌3两，麦冬3两，茜草1两8钱，地骨皮3两，淫羊藿3两，艾叶2两4钱，黄耆3两，补骨脂3两，枸杞子3两，巴戟3两，附子3两6钱，肉苁蓉3两，当归9两，牛膝1两8钱，覆盆子3两，龟版6两，狗脊3两。
出处：《北京市中药成方选集》。
主治：男子气虚，梦遗滑精，偏坠疝气。妇女血寒腹痛，白带，腰腿疼痛。
加减：None
功效：补肾散寒，固精止痛。
用法用量：摊时每16两膏药加入细粉面2钱，每张油重5钱。微火化开，男子贴肾俞穴，女子贴脐部。
制备方法：上药酌予碎断，用香油400两炸枯，过滤去滓，炼至滴水成珠，入黄丹176两，搅匀成膏，取出放入冷水中去火毒后加热溶化。
临床应用：None
各家论述：None
用药禁忌：孕妇忌贴。
附注：摊时细料面：赤石脂二钱，硫黄一钱，狗肾二钱，乳香二钱，没药二钱，公丁香一钱，阳起石二钱，共为细粉。贴时加入细料：肉桂四两，冰片二钱，麝香一钱，丁香五钱，共为细粉。
