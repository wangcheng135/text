序号：43802
名称：厚朴丸
组成：厚朴（去粗皮，生姜汁炙令紫）1两，干姜（炮）1两，陈橘皮（汤浸，去白，焙）1两，诃黎勒（炮，去核）1两，白茯苓（去黑皮）1两，芜荑（微炒香）1两，阿胶（炙令燥）1两，熟艾（微炒，别捣）1两，胡粉（炒黄）1两，黄石脂（赤石脂亦可）1两，乌梅（去核，炒干）1两，当归（切，焙）1两，蜀椒（去目并闭口，炒出汗）1两。
出处：《圣济总录》卷七十七。
主治：积年冷痢，日三五行，胀闷肠鸣，食不消化，面黄渐瘦。
加减：None
功效：None
用法用量：每服50丸，空心用温浆水送下，日午再服。
制备方法：上除胶、艾2味外，为末，先以米醋1升半，于无风处煮艾水减约8分，绞去艾，次下阿胶，候消尽，乘热入药末和匀为丸，如梧桐子大，晒干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
