序号：42967
名称：保真膏
组成：麻油2斤半，甘草4两，天冬（去心）4钱，麦冬（去心）4钱，熟地（酒浸）4钱，生地（酒浸）4钱，官桂4钱，牛膝4钱，苁蓉（酒浸）4钱，鹿茸（酥炙）4钱，远志（甘草汤煮去骨，酒浸）4钱，川续断4钱，紫梢花4钱，木鳖子4钱，谷精草4钱，大附子4钱，白果4钱，杏仁4钱，蛇床子4钱，虎骨（酥炙，捶碎）4钱，菟丝子（水酒浸）4钱，松香4两，黄丹1斤（水飞过），硫黄4钱，雄黄4钱，龙骨4钱，赤石脂（细末）4钱，乳香4钱，没药4钱，丁香4钱，南木香（细末）4钱，当门麝香2钱，蟾酥2钱，真阳起石（细末）2钱，海狗肾1个，黄蜡6钱。
出处：《外科百效》卷一。
主治：腰膝疼痛，下元虚损，四肢麻木，半身不遂，五劳七伤，冷气攻刺。及去小肠膀胱气，二三十年劳证，远近风蛊，筋骨酸痛，阳事不举。
加减：None
功效：镇玉液，保精不泄，龟旺不死，通血脉，强身体，返老还童，须发复黑，固真精，善御器，虽数次不泄，滋皮肤。
用法用量：每用药3-4钱，摊绫绢上，贴腰眼二穴，1个膏药可贴得60日。
制备方法：用净锅一口，桑柴烧文武火，先下甘草与麻油，熬5-6沸；次下天门冬等19味粗末，熬至各味药枯焦黑，用细密绢巾滤去滓；次下松香、黄丹，以槐树条一把不住手搅药，滴水成珠不散为度；次下硫黄等细末，搅匀住火；然后下麝香、黄蜡搅匀。膏药好了，盛瓷器内，用油纸裹，浸井水中4-5日出火毒用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
