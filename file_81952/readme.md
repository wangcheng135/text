名称：镇心大牛黄丸
组成：牛黄半两（研如粉），真珠1两（研如粉），琥珀1两（研如粉），朱砂1两（研如粉），麝香1两（研如粉），天竺黄1两（研如粉），石膏1两（研如粉），龙齿1两（研如粉），雄黄1两（研如粉），马牙消1两（研如粉），铁粉2两（研如粉），天门冬（去心）半两，龙胆半两，防风半两，升麻半两，人参半两，黄芩半两，甘草（炙）半两，茯神（去皮木）半两，菖蒲半两，远志（去心）半两，露蜂房半两，秦艽半两，知母半两，犀角（末）半两，钩藤半两，川大黄1两，金箔50片，银箔50片，麦门冬半两（去心）。
出处：《普济方》卷十六引《博济》。
主治：心脏虚风，神情恍惚，往往惊悸，狂言妄语，或似癫痫。
加减：None
功效：None
用法用量：每服20丸，温水送下，日3次。
制备方法：上依法修制，麦门冬至大黄等味为细末，与上诸药同研匀细为度，炼蜜为丸，如梧桐子大。
