序号：75362
名称：川芎散
组成：川芎（洗）1两，柴胡（去苗，洗）1两，半夏曲半两，甘草（炙）半两，甘菊半两，细辛（去叶）半两，人参（去芦）半两，前胡（去苗，洗）半两，防风（去叉股）半两。
出处：《本事》卷四。
主治：风盛膈壅，鼻塞清涕，热气攻眼，下泪多眵，齿间紧急，作偏头疼。
加减：None
功效：None
用法用量：每服4钱，水1盏，加生姜4片，薄荷5叶，同煎至7分，去滓温服。
制备方法：上为粗末。
临床应用：None
各家论述：《本事方释义》：川芎气味辛温，入肝胆；柴胡气味甘平，入足少阳；半夏曲气味辛温，入足阳明；甘草气味甘平，入足太阴；甘菊花气味辛凉，入肝胆；细辛气味辛温，入足少阴；人参气味甘温，入足阳明；前胡气味苦辛微寒，入手足太阴、阳明，其功长于下气；防风气味辛温，入足太阳，再佐以生姜之辛温散，薄荷之辛凉而散，使上壅者得以宣降，则所患之症得甘平甘温之品护中，而辛温辛凉之药各得施其技焉。
用药禁忌：None
附注：川芎甘菊散（《杏苑》卷五）。
