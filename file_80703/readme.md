序号：80719
名称：麋角霜丸
组成：麋角1副（用水浸7日，刮去皴皮，镑为屑，盛在一银瓶内，以牛乳浸1日，如乳耗更添，直候不耗，于麋角屑上乳深2寸，用油单数重密封瓶口，别用大麦1斗，安在甑内，约厚3寸，上安瓶，更用大麦周围填实，露瓶口，不住火蒸一复时，如锅内水耗，即旋添热汤，须频取角屑，看烂如面相似，即住火取出，用细筛子漉去乳，焙干，每料用干角屑8两），附子（炮裂，去皮脐）3两，山芋3两。
出处：《圣济总录》卷一八五。
主治：None
加减：None
功效：补暖元脏，驻颜。
用法用量：每日服15-20丸，空心温盐酒送下。
制备方法：上为末，以枣肉为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
