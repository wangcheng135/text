序号：68025
名称：黄耆汤
组成：黄耆（锉）1两，白术（锉，炒）1两，牡蛎（熬为粉）1两，白茯苓（去黑皮）1两，防风（去叉）1两，生干地黄（焙）1两，麦门冬（去心，焙）1两。
出处：《圣济总录》卷一六四。
主治：产后荣卫虚损，汗出不止。产后阴虚，又遇邪，虚汗不止。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，煎至7分，去滓温服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：《济阴纲目》：黄耆得防风其功愈大，为易于固表也；牡蛎肾家药也，以肾液入心为汗，故止汗又宜固肾，其他可意解也。
用药禁忌：None
附注：《玉机微义》有甘草、大枣；《金鉴》有浮小麦、甘草；《女科指掌》有当归、甘草。
