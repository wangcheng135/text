序号：71498
名称：御米酒
组成：真薏苡米酒10斗（即100斤），霜茄根（切片）半斤，真桑寄生（锉）半斤，五加皮（切）半斤，苍耳子（炒去刺）半斤，甘州枸杞子（净）1斤，川牛膝（去芦）1斤。
出处：《医统》卷四十八。
主治：None
加减：None
功效：壮筋骨，畅经络，养精元，益神气。久服行步如飞，延年益嗣，有通仙之妙。
用法用量：早、晚随量饮，微醇为度，此药虽平易有奇效，不可以其简而忽之。
制备方法：上各制净称分两，每味分4剂，合为1剂，酒100斤，亦分4坛，各药煮3炷香，取放土地上退火性，二七可饮，久窨尤佳。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
