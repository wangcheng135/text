序号：58862
名称：秘制抱龙丸
组成：犀黄3钱，腰黄（飞）1两，僵蚕（炒）1两，蝉衣（去头足）1两，白附子（姜汁炮，炒）1两，天麻（粗纸湿包，煨）1两，防风1两，枳壳（面炒）1两，茯神1两，陈胆星1两，劈砂（研，飞）1两，全蝎（去头足，泡盐，晒）52只，薄荷（晒干）3钱，甘草5钱，当门子1钱。
出处：《华氏医方汇编》卷三。
主治：小儿惊风噤口，痰喘咳嗽。
加减：None
功效：None
用法用量：None
制备方法：上各为细末，以甘草、陈皮5钱煎汤化胆星为丸，如芡实大，先以雄黄为衣，晒干，再上朱砂为衣，香气不泄。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
