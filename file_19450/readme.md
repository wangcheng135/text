序号：19457
名称：地黄丸
组成：生地黄（净洗，细切）1斗（以好酒1斗浸之，经1宿即出。干即入酒中浸，以酒尽力度，候干），干漆2两（捣碎，炒令烟出），肉苁蓉2两（酒浸1宿，刮去皱皮，炙干），蛇床子2两，菟丝子2两（酒浸3日。晒干，别捣为末），桂心2两，远志3两（去心），人参3两（去芦头），牛膝2两（去苗），石斛1两（去根，锉），补骨脂2两（微炒）。
出处：《圣惠》卷九十八。
主治：虚损。
加减：None
功效：还精补髓，驻颜色，却老，安脏腑，暖下元，壮腰。
用法用量：每服30丸，空心及晚食前以盐汤送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
