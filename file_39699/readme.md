序号：39709
名称：绎皮丸
组成：当归4两（洗净，好酒浸1宿，漉出焙干，再浸，酒净为度），赤芍药2两，人参2两，白芍药2两，肉桂（去皮）2两，白术2两，益智仁2两，白薇2两，五灵脂2两，附子2两，陈皮2两，青皮2两，牡蛎（煅）4两，赤石脂4两，香附子（去尾尖）4两，玄胡索4两，牡丹皮（去木）4两，苍术4两，败姜4两，京三棱4两，蓬莪术4两，刘寄奴4两，艾叶4两，泽兰4两，生干地黄4两，熟干地黄4两，蒲黄3两（隔纸炒），雄黑豆10两（丸小者）。
出处：《朱氏集验方》卷十。
主治：妇人百病。
加减：None
功效：None
用法用量：每服40-50丸，空心黄酒或艾醋汤送下。
制备方法：上将苍术、败姜、陈皮、青皮、京三棱、莪术、刘寄奴、艾叶、泽兰、小雄豆等10味，用好醋煮药，候黑豆烂熟为度，醋少，又添煮为佳，焙干，和众药，为细末，以醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：有孕不可服。
附注：None
