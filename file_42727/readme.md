序号：42742
名称：保安丸
组成：附子半两（炮，去皮脐），白附子（炮）1两，天麻1两，全蝎（去毒，微炒）1两，蔓荆子1两，防风（去芦头）1两，羌活（去芦头）1两，川芎1两，肉桂（去粗皮）1两，白僵蚕（炒，去丝嘴）1两，当归（洗，焙）1两，麻黄（去根节）1分，乌蛇（酒浸，去骨取肉，焙干）半两，乳香1分（别研）。
出处：《杨氏家藏方》卷十七。
主治：小儿诸风惊痫，潮发搐搦，口眼牵引，项背强直，精神昏困，痰涎壅塞，嗳气喘急，目睛斜视，一切虚风。
加减：None
功效：None
用法用量：每服1丸，煎荆芥汤化下，不拘时候。
制备方法：上为细末，次入乳香，研匀，炼蜜为丸，每1两作40丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
