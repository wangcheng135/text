序号：26575
名称：八宝拨云散
组成：珍珠粉3分（水飞），玛瑙5分（水飞），珊瑚5分（水飞），琥珀5分（水飞），硇砂7分，煅熊胆5分，麝香3分，冰片5分，制甘石3钱（水飞），煅西月石1钱5分，飞朱砂5分，杜荠粉2钱（以鲜荸荠捣碎，滤取细汁，澄淀成粉晒干，用净粉2钱）。
出处：《全国中药成药处方集》（南京方）。
主治：目赤肿痛，胬翳攀睛。
加减：None
功效：None
用法用量：以点眼棒蘸凉开水点眼角。
制备方法：各取净粉，先分别乳细，再共合乳至极细无声为度，愈细愈佳。分装2分重1瓶，用玻璃瓶装，以白腊封口密藏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
