序号：57518
名称：润肠汤
组成：当归、熟地、生地、麻仁（去壳）、桃仁（去皮）、杏仁（去皮）、枳壳、厚朴（去粗皮）、黄芩、大黄各等分，甘草减半。
出处：《回春》卷四。
主治：大便闭结不通，为实热燥闭者。
加减：发热，加柴胡；腹痛，加木香；血虚枯燥，加当归、熟地、桃仁、红花；风燥闭，加郁李仁、皂角、羌活；气虚而闭，加人参、郁李仁；气实而闭，加槟榔、木香；痰火而闭，加瓜蒌、竹沥；因汗多，或小便去多，津液枯竭而闭，加人参、麦门冬；老人气血枯燥而闭，加人参、锁阳、麦门冬、郁李仁，倍加当归、熟地、土地，少用桃仁；产妇去血多，枯燥而闭，加人参、红花，倍加当归、熟地，去黄芩、桃仁。
功效：None
用法用量：水煎，空心热服。大便通即止药，不能多服，如修合润肠丸，将药加减各为末，炼蜜为丸，如梧桐子大。每服50丸，空心白汤吞下。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：切忌辛热之药。
附注：None
