序号：64791
名称：第二大竹沥汤
组成：竹沥1斗4升，独活2两，芍药2两，防风2两，茵芋2两，甘草2两，白术2两，葛根2两，细辛2两，黄芩2两，芎2两，桂心1两，防己1两，人参1两，石膏1两，麻黄1两，生姜3两，茯苓3两，乌头1枚。
出处：《千金》卷七。
主治：脚气。中风口噤不能言，四肢缓纵，偏痹挛急，风经五脏，恍惚恚怒无常，手足不随。
加减：None
功效：None
用法用量：上(口父)咀。以竹沥煮取4升。分6服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：其第二方乃以乌头代附子，生姜代干姜；盖乌头辛烈，祛风之力迅于附子，生姜性暴，开痰之功速于干姜，即二味之变通；又于《古今录验》续命方中采取人参、石膏、芎三味，以人参助麻黄、乌头，力开痹着；石膏佐黄芩、竹沥、涤除旺气；越婢全方，但少大枣一味；并采小续命中芍药佐芎入血搜风，复采《金匮》防己茯苓汤中茯苓，佐桂心逐湿安中土；《千金》防已汤全在其中，但不用苦酒煎服；更采仓公当归汤中独活，专祛下部风湿；且参茵芋一味，专通关节拘挛，性味虽劣，《外台》、《千金》恒用之，惜乎近世药肆罕得。
用药禁忌：None
附注：None
