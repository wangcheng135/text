序号：44041
名称：咽喉夺命丹
组成：珍珠2钱，金果榄2钱，真京墨（研）6分，川郁金2钱，犀角6分，飞辰砂2钱，赤金箔6分，煅中白5分，梅片5分，上血竭2钱，天竺黄2钱，上沉香2钱，苦甘草3钱，人指甲（炙脆）6分，川贝母3钱，真熊胆6分，上血珀6分，玳瑁（炙脆）7分，甜葶苈2钱，当门子3分。
出处：《千金珍秘方选》。
主治：咽喉险症。
加减：None
功效：None
用法用量：极重喉症含之立效。
制备方法：上为细末，用麻黄、钩藤、薄荷、陈皮各1两煎胶，用元米饮一勺，打和捣匀为丸，每丸重1钱，辰砂为衣，用蜡壳收置。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
