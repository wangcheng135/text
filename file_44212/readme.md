序号：44227
名称：三解散
组成：人参（去芦）2钱半，防风（去芦）2钱半，天麻2钱半，茯神（去皮、木根）2钱半，郁金（无，以山栀仁代）2钱半，白附子2钱半，大黄2钱半，赤芍药5钱，黄芩5钱，僵蚕5钱，全蝎15尾（去尖毒），枳壳2钱（水浸润，去瓤锉片，麸炒微黄），粉草6钱。
出处：《活幼心书》卷下。
主治：上焦蕴热，伤风，面红目赤，狂躁气急，渴水，惊啼烦闷，丹毒，口疮，痰嗽，搐掣；露丹，小儿生后，百日之内，半岁以上，忽两眼红晕微起，面青黯色，夜间烦哭，或脸如胭脂。初则满面状如水痘，脚微红而不壮，出没无定，次至颈项，赤如丹砂。
加减：None
功效：None
用法用量：每服半钱至1钱，用温薄荷汤或灯心汤调下，不拘时候。
制备方法：上药锉、焙，为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
