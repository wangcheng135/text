序号：83782
名称：乌蛇丸
组成：乌蛇（酒浸，去皮骨，炙）半两，乌头（炮裂，去皮脐）半两，五灵脂半两，羌活（去芦头）半两，天麻（酒浸，切，焙）半两，牛黄（研）1分，雄黄（研）1分，麝香（研）1分，干蝎（去土，酒炒）1分，天南星（炮裂，汤洗）1分，香墨（烧醋淬，研）3分，独活（去芦头）3分，皂荚（不蚛者，去皮子，酥炙）3分，白附子3枚（炮裂，汤洗），虎骨（酥炙）1两，黑豆21粒（先与乌头同捣烂，焙）。
出处：《圣济总录》卷六。
主治：中风，口面僻，言语不正。
加减：None
功效：None
用法用量：每服7丸，温酒送下；渐加至10丸，空心、午时、夜卧各1服。
制备方法：上16味，捣罗13味为末，入牛黄、麝香、雄黄末拌匀，重罗1遍，炼蜜为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
