序号：23148
名称：芎散
组成：芎3分，防风1两（去芦头），桂心半两，赤芍药半两，羌活3分，当归3分（锉，微炒），羚羊角屑3分，牛蒡子1两（微炒），酸枣仁3分（微炒）。
出处：《圣惠》卷七十八。
主治：产后中风，四肢筋脉挛急疼痛，背项强直。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：《医略六书》：产后风中于经，热遏于络，而筋脉失养，故项背强急，手足拘挛掣痛焉。西羌活散太阳之邪，羚羊角清厥阴之火；芎活血行气，当归养血荣筋；防风疏风于表，乳香活血止痛；牛蒡疏风散热，枣仁养心益营；桂心温营暖血以平肝木也。水煎温服，使风邪外解，则遏热自化，而经络清和，筋得所养，安有项背强急，拘挛掣痛之患哉。
用药禁忌：None
附注：《医略六书》有乳香。
