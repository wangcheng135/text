序号：69522
名称：犀角麻黄汤
组成：犀角屑2两，麻黄（去节）2两，甘草（炙）1两，茯苓2两，防己2两，黄芩1两，石膏3两，附子（炮）1两，白术1两，芎1两，防风1两，当归1两，生姜3分，细辛1两，桂心1两。
出处：《外台》卷十八引《崔氏方》。
主治：脚气冲心，烦闷。
加减：None
功效：None
用法用量：以水1斗，先煮麻黄，去沫，取汁8升，下诸药，煎取2升7合，分3服，相去10里久。服讫，覆取汗。待3-4日后，若其皮肤不仁，愈即停；不愈，宜更服之，不得过3剂，即愈。愈讫，脚中无力者，宜服独活犀角汤2-3剂，即愈。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
