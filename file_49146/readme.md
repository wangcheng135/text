序号：49161
名称：胜金丹
组成：香附12两，熟地4两，赤石脂1两，白术4两，赤芍1两半，琥珀5钱，白薇1两，甘草5钱，海沉1两，乳香1两，朱砂5钱，玄胡5钱，藁本2两，广边桂2两，云苓2两，白芍2两，当归1两半，川牛膝1两，没药1两，白石脂1两，红人参1两，远志1两半，川芎1两半，丹皮1两，白芷2两。
出处：《全国中药成药处方集》（抚顺方）。
主治：血崩漏血，赤白带下，月经不调，赶前差后，虚寒腹痛，久不孕育，颜面萎黄，腰膝疼痛。
加减：None
功效：补血调经。
用法用量：每服1丸，白水送下。
制备方法：上为细末炼蜜为丸，2钱重，蜡皮封。
临床应用：None
各家论述：None
用药禁忌：孕妇、于血痨及瘀血实证者均忌服之。
附注：None
