序号：73873
名称：硼砂煎丸
组成：硼砂2两（白色不夹石者，研），阿魏1分，神曲1两（为末），诃黎勒1两（用皮），丁香半两，荜菝半两，附子半两（炮裂，去皮脐），青橘皮半两（汤浸，去白瓤，焙），白芥子半两，茴香子半两，槟榔半两。
出处：《圣惠》卷五。
主治：脾脏虚冷，心腹有积滞气，发歇疼痛，心膈不利，两胁胀满，不能饮食。
加减：None
功效：None
用法用量：每服15-20丸，食前以生姜汤送下，或温酒送下。
制备方法：上诃黎勒以下诸药为末，以好酒1升，先煎硼砂，次入阿魏同煎5-7沸，后以绵滤过，再煎之，下神曲末，搅令匀，慢火熬成膏，拌和诸药末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，一作“硇砂煎丸”，方中硼砂，一作“硇砂”。
