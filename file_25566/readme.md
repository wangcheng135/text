序号：25574
名称：朱砂丹
组成：天门冬4两。
出处：《普济方》卷二三一引《十便良方》。
主治：劳嗽。
加减：None
功效：None
用法用量：每服3-5粒，以人参汤送下，不拘时候。
制备方法：上药碎锉，取一白垍小盒子，将天门冬入在一香炉内，三块两块烧之，上以物阁住盒子，令烟熏盒子内，十分光黑，仍以物遮烟，只就一处令出，令烟厚了，取砂2两，入在盒子内，上以川椒盖头令满，以盖子合定，蜜调茶土固封，土内仍着少盐，外以铁线系缠定，放地上，仍以一瓦子衬底上，以醋为灰家，下3斤火，候消得一半来，更下2斤，火冷取出，埋在湿地一宿，出火毒了，拣砂令净，研细，糯米糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
