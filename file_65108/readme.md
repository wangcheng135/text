序号：65124
名称：羚羊角汤
组成：羚羊角2分，防风12分，羌活8分，苦梗8分，败酱8分，桂心6分，柴胡6分，大黄（浸过）6分。
出处：《普济方》卷三五○。
主治：产后腹中坚硬，两胁膈胀，手足冷，心中热，欲饮水干呕，欲成关节劳、痉、中风之疾。
加减：None
功效：None
用法用量：上以水2升，煎取8分，空心2服，服毕即吐，良久更服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：羚羊角饮子（《准绳·女科》卷五）。《准绳·女科》本方用法：上(口父)咀。每服五钱，以水一大盏半，同煎至一盏，去滓温服，不拘时候。更服地黄酒：用地黄（切）一升，炒令黑，瓷瓶中下热酒三升，密封口，煮令减半，任意服之。
