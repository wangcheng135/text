序号：3602
名称：巴戟丸
组成：巴戟1两，硫黄（细研，水飞过）1两，桂心1两，补骨脂（微炒）1两，硇砂（细研）1两，附子（炮裂，去皮脐）1两，胡芦巴（微炒）1两，川椒红（微妙）1两，木香1两，肉苁蓉（酒浸1宿，刮去皱皮，炙干）1两，吴茱萸（汤浸7遍，焙干，微炒）1两。
出处：《圣惠》卷九十八。
主治：下元虚惫，脐腹疼痛，小便滑数，颜色萎黄，手足常冷，饮食无味，四肢少力。
加减：None
功效：None
用法用量：每日服30丸，空心温酒送下。
制备方法：上为末，入研了药令匀，以羊肾3对，切去筋膜，好酒3升，熬令稠烂，研和诸药末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
