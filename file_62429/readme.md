序号：62445
名称：梅花五气丹
组成：梅花片5分，当门麝5分，轻粉6分，辰砂6分，乳香1钱，没药1钱，瓜儿血竭1钱，明雄黄1钱，真酥散（预于端午前寻之，至午日，取酥2钱，用头男乳调膏）。
出处：《外科正宗》卷二。
主治：脑疽、发背、诸般疔肿，初起寒热交作，筋骨疼痛，有似伤风，恶心呕吐；但未成脓者。
加减：None
功效：None
用法用量：凡遇恶疮大毒，开器取出1枚，先用美馔食饱，次用无根水漱净口内，再含水一口，少顷待温，用葱白5寸同水嚼烂咽下，随将药丸安放舌下，睡于暖处，以被覆盖。药化苦水，徐徐咽之，疮势大者，2-3丸亦可；药尽其汗即到如淋，诸病若失。如冬月天寒难汗，噙后将葱白汤催之亦妙，凡治无有不效。如暗疔人所不知觉，及知觉而失治者，毒气入里，人便昏沉，一中便倒，不能依法服药，急用连须葱白7个，煎酒1杯，研药5丸灌下，药气到心，其功如汤泼雪，患者即便苏醒。
制备方法：上各为极细末，对准分数，于端午日辰时制度，候至午时，将上药9味和入蟾酥膏内，向日为丸，如芥子大，一时内晒干。用川椒27粒，灯心27段同药收于瓷罐内养之，以蜡封口，不泄药气为妙。
临床应用：None
各家论述：None
用药禁忌：None
附注：梅花五炁丹（《疡科捷径》卷上）。
