序号：26645
名称：延生护宝丹
组成：菟丝子（水淘净，酒浸软，取末）3两，肉苁蓉（酒浸，切倍）2两（2味浸药酒各多著，要熬子），家韭子4两（水淘净；用枣2两同煮，令枣熟，去枣，水淘净滓干，再用酒浸1宿，慢火炒干，称2两），蛇床子2两（水淘净，枣3两同煎，令枣熟，去枣焙干，称1两），晚蚕蛾（全者）2两（用酥少许，慢火微炒），木香半两，白龙骨1两（用茅香1两同煮1日，去茅香用帛裹，悬在井中浸1宿，取出），鹿茸，桑螵蛸（锉，炒香），莲实（去皮，炒熟）1两，干莲花蕊，胡芦巴（微炒）1两，丁香半两，南乳香（别研）半两，麝香（别研）2钱。
出处：《御药院方》卷六。
主治：None
加减：None
功效：补元气，壮筋骨，固精健阳，通和血脉，润肌泽肤，益寿延年。
用法用量：每服30丸，绝早日末出时，温酒入炒盐少许送下。静坐少时，想药至丹田，以意斟量渐加丸数。如阳道衰，精滑者，空心临卧各进1服。
制备方法：上药除乳香、麝香、菟丝子末外，同为细末；将前菟丝子末3两，用浸药酒2升，用文武火熬至一半，入荞麦面两匙（重1两），用酒调匀，下膏子内搅匀；次下乳香，麝香，不住手搅，轻沸熬如稠糊，放冷；此膏子都要用尽，恐硬，入酒少许，与前药末和为丸，如梧桐子大。
