序号：47225
名称：疮毒化毒散
组成：乳香3钱，没药3钱，赤芍3钱，花粉3钱，川军3钱，元连3钱，甘草3钱，绿豆面3钱，白芷3钱，贝母6钱，冰片5分，雄黄8分。
出处：《全国中药成药处方集》（沈阳方）。
主治：痈毒热疖，焮肿痛疼；小儿胎毒，头疮秃疮；斑疹余毒，流脓流水；项肿腮肿，溃破流脓；耳疮耳底流脓水；各种血毒流脓；风火毒，血风疮。
加减：None
功效：解毒活血，透达经络壅塞，退热消肿，宣通气血凝滞。
用法用量：周岁小儿每服1-5分，周岁以上者，每服5分至1钱，白开水送下。
制备方法：上为极细末，后兑冰片、雄黄，再共为细未调匀，贮瓷瓶内。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
