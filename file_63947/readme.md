序号：63963
名称：清膈丸
组成：人参1两，赤茯苓1两，木通1两，黄耆（蜜炙）1两，生干地黄1两，桑白皮（蜜炙）1两，青皮（去白）1两，防风（去芦）1两，甘草（炙）1两，枳壳（麸炒，去穰）1两，麦门冬（去心）半两。
出处：《简易方》引《叶氏录验方》（见《医方类聚》卷&mdash;一七）。
主治：肺气上壅，气促迫塞，面赤痰实，咽膈不利，头昏目眩，肩背拘急，及面生赤激瘙痒。
加减：None
功效：None
用法用量：每服1丸，以水7分盏，煎至6分，食后温服，每日3次。
制备方法：上为末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：一方只作散子，用蜜煎服。方中枳壳用量原缺。
