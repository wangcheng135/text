序号：66078
名称：蛇黄散
组成：蛇蜕皮（置净瓷器中，以烛焰爇之，火着去烛，匀烧令焦，取1两用之），臭黄1两，绿矾1分（烧熟）。
出处：方出《圣惠》卷六十五，名见《普济方》卷三○○。
主治：甲疽。
加减：None
功效：None
用法用量：上为细末，以铜盒子贮之，先以热小便2升，置于铜钞锣中，嚼20枚杏仁，吐于小便中，搅令相得，以疮脚浸之。候痒，即以铜篦子洗，拨出脓血，取烂帛裛之，候干，还以铜蓖子敷散令满，以故帛虚裹疮指，入大袜中。每日1洗，依前法用，每洗行药，软即拨去药，恐咬落疮筋。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：切慎房室及运动气力。忌面、蒜、酒一切物无妨。
附注：None
