序号：42779
名称：保寿丸
组成：麋茸2两（去毛，酒洗，炙微黄），钟乳粉2两，补骨脂1两（微炒），天雄2两（炮裂，去皮脐），硇砂2两（研细），腽肭脐1两（酒洗，炙），菟丝子2两（酒浸3宿，晒干，别杵为末），阳起石1两（酒煮半日，研细，水飞过），肉苁蓉1两（酒浸，去皴皮，炙干），青盐1两，巴戟1两，白马茎2两（涂酥，炙令微黄），雄雀儿30枚（去毛足肠肚，研如泥，以酒5升煎如膏），硫黄1两（研细，水飞过），桂心1两，雄鸡肝3具（切，焙干），黄戌茎并肾1对（切，焙干）。
出处：《圣惠》卷七。
主治：肾脏虚损，阳气全乏。
加减：None
功效：None
用法用量：每日30丸，空心及晚食前温酒送下。
制备方法：上为末，入雀肉膏为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
