序号：76633
名称：舒筋片
组成：麻黄160两，制马钱子160两（凡用越南、柬埔寨马钱子时，需按处方减少一倍投料），羌活12两，独活12两，千年见12两，钻地风12两，怀牛膝12两，乳香（炙）12两，没药（炙）12两，木瓜12两，杜仲（炒）12两，防风12两，桂枝12两，甘草12两。
出处：《天津市中成药规范》。
主治：风寒湿引起的四肢麻木，筋骨疼痛，行走不便。
加减：None
功效：祛风除湿，舒筋活血。
用法用量：每服2片，温开水送下，每日1次，不可过量。
制备方法：取千年见至甘草10味共为细末，制马钱子为细末，麻黄、羌活，独活为3号粗末，与马钱子细末和匀。取马钱子细末与麻黄等混合粗末，用6倍量70％乙醇按渗漉法提取，浓稠膏约90两。混合制粒压片，包橘红色糖衣（每100两用胭脂红1.25g，柠檬黄0.31g）打光。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
