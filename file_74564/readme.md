序号：74580
名称：新法半夏汤
组成：陈皮（去白）4两，炒神曲4两，炮干姜4两，丁皮7钱半，木香7钱半，白茯苓7钱半，甘草4钱半，草果（煨，去皮）、半夏曲2两3钱（炒）。
出处：《局方》卷三（宝庆新增方）。
主治：脾胃不和，中脘气滞，宿寒留饮，停积不消，心腹刺痛，胁肋膨胀，呕吐痰水；噫气吞酸，中酒吐酒，哕逆恶心，头痛烦渴，倦怠嗜卧，不思饮食。
加减：None
功效：温平破痰，开胃健脾，消酒进食。
用法用量：每服3钱点服，无时常服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：原书卷四（续添诸局经验秘方）“新法半夏汤”有桔梗、青皮、无神曲、木香、草果。主治相同。
