序号：42949
名称：保真丸
组成：牛膝10两（用黑豆3升铺锅内，9蒸9晒九露，黑豆一蒸一易，如数完，竹刀切片听用），生地黄12两（酒洗，用一半拌去皮砂仁5钱、白茯苓末5钱，蒸1昼夜，熟透捶用），补骨脂4两（用核桃肉2两，研碎拌匀，按实，瓷器内1日炒干用），何首乌20两（忌铁器，同牛膝蒸之），白茯苓6两（用牛乳2碗浸透晒干），白茯神6两（人乳2碗浸透晒干），当归6两（酒洗），天门冬6两（去心），菟丝子3两（酒浸，去壳），麦门冬6两（去心），柏子仁8两（汤泡7次，去油），枸杞子6两（去蒂），人参3两（去芦），山药4两，杜仲4两（炒去丝）。
出处：《墨宝斋集验方》。
主治：None
加减：阳气弱而精不固者，加山萸肉4两，锁阳4两，肉苁蓉4两；如健忘者，加九节菖蒲3两，远志3两；如思虑忧伤过度，损心太甚而不能寐者，加炒熟枣仁3两。
功效：补心神，固肾精，坚筋骨，润肌肤，泽容颜，乌须发，久服益寿延年。
用法用量：每服70丸。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
