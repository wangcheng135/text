序号：53953
名称：换骨丹
组成：桑白皮2两，川芎2两，吴白术2两，紫河车2两，威灵仙2两，蔓荆子2两，人参2两，防风2两，何首乌2两，地骨皮1两，五味子1两，木香1两，苦参1两，犀角半两，麝香半钱，龙脑半钱。
出处：《中藏经·附录》。
主治：一切卒中，手足顽麻，腰膝沉重，左瘫右痪，四时伤寒，妇人血滞，小儿惊搐。
加减：None
功效：None
用法用量：截四时伤寒，妇人血滞，产前产后，每服1丸，酒1盏，碎捶，浸至夜，温动化散，临睡和滓服；小儿惊搐，每服半丸，米饮化下。
制备方法：上为细末，用苍术半斤、槐角半斤，以地黄3斤去根、不去节，锉细，上用河水1斗8升（井水亦得），同熬至3-4升，去滓，留清者，再熬成膏，和前药，每两作8丸，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
