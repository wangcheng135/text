序号：31404
名称：沉香通气丸
组成：京三棱（慢火炮）1两半，丁香1两半，陈皮（去白）1两半，玄胡索1两，木香1两，木通（去皮），沉香1两，白术1两，槟榔半两，广术（慢火炮）2两，枳壳（麸炒，去瓤）2两，青皮（去瓤）1两半，茴香（盐炒）1两半，新罗参1两半，白茯苓（新者）1两半，白豆蔻仁3两。
出处：《医方类聚》卷一一三引《经验秘方》。
主治：积聚寒热，心腹闭满，胁肋刺痛，呕逆寒痰，气满不散，遍身骨节疼痛，寒热有时，荣卫不通。
加减：None
功效：顺气和血，消进饮食。
用法用量：每服30-50丸，以温水米饮汤送下，不拘时候。
制备方法：上为细末，姜汁打面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
