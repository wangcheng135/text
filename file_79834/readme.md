序号：79850
名称：干蝎丸
组成：干蝎（去土，微炒）1两，天麻1两，羌活（去芦头）1两，独活（去芦头）1两，附子（炮裂，去皮脐）1两，槟榔（锉）1两，沉香（锉）1两，木香1两，狼毒（切作块，先用醋煮3-5沸，焙干，锉）1两，牛膝（酒浸，切，焙）1两，白附子（微炒）1两，桂（去粗皮）1两，当归（切，焙）1两，枳壳（去瓤，麸炒）1两，巴戟天（去心）1两，防风（去叉）1两，昆布（洗去咸汁，微炒）1两，牵牛子（炒半熟）1两，人参1两，蒺藜子（炒去角）1两，高良姜1两，萝卜子（微炒）1两，肉豆蔻（去壳）1两，没药1两，白术1两，防己（锉，焙）1两，硇砂（醋化尽，去夹石，炼霜）1两，阿魏1分。
出处：《圣济总录》卷十。
主治：元脏风气攻注，腰脚疼痛，及一切风气。
加减：None
功效：None
用法用量：每日五更后，空心薄荷酒下20丸。向晓利一行即愈。
制备方法：上药先将没药、硇砂、阿魏3味，用好酒1升熬成膏。余药同为末，炼蜜同膏为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
