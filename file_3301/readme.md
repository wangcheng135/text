序号：3301
名称：天雄丸
组成：天雄1两（炮裂，去皮脐），桂心2两，羌活2两，当归3两（锉，微炒），五加皮2两，天麻2两，芎2两，酸枣仁1两（微炒），陈橘皮1两（汤浸，去白瓤，焙），续断1两，石斛1两（去根，锉），赤茯苓1两，鹿角胶1两（捣碎，炒令黄燥），薏苡仁1两，牛膝1两（去苗），木香1两，槟榔1两。
出处：《圣惠》卷二十六。
主治：筋虚极，身体拘急，胁下多痛，不可转动，善悲，色青，感于寒湿，筋不能动，十指皆痛。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以荆芥酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
