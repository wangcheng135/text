序号：56825
名称：消饮茯苓汤
组成：白茯苓（去黑皮）半斤，前胡（去苗）半斤，芎藭半斤，羌活（去芦头）半斤，桔梗（炒）半斤，人参半斤，独活（去芦头）半斤，甘草（炙）半斤，柴胡（去苗）半斤，陈橘皮（去白）4两。
出处：《圣济总录》卷二十一。
主治：伤寒表未解，气脉闭塞，津液不通，水饮气停在胸府，结而成痰，膈脘痞闷，倚息短气，体重多唾，头目旋运，嗜卧好眠，额角偏痛，腠理开疏，体常汗出。
加减：None
功效：None
用法用量：每服3钱，沸汤点服，不拘时候。或锉如麻豆，用前件药半两，水3盏，煎至1盏半，去滓，分2次温服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
