序号：31400
名称：沉香既济丸
组成：枳壳（去瓤，酒浸，麸炒）3两，川楝子（干用，青盐炒）3两，巴戟（去心，酒浸）3两，韭子（酒浸，炒焦）3两，八角茴香（就于青盐少许炒小茴香）3两，白茯苓3两，木香1两，沉香1两，麝香2钱，青盐1两，白马茎1条（微炒，晒干，切作片，另研为末；如无马茎，用黄狗茎13个，切，焙干；若有狐茎，止用9个，切，焙干，另研末）。
出处：《普济方》卷二一九引《德生堂方》。
主治：None
加减：None
功效：滋补下元，调顺诸气，壮健阳事，加进饮食。
用法用量：每服50-60丸，早晨空心以好酒送下，干物压之。
制备方法：上为细末，却将别研药末和匀，酒为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌食生姜、萝卜、豆粉、猪血。
附注：方中白马茎，《奇效良方》作“肉苁蓉”。
