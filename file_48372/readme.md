序号：48387
名称：神效观音救苦丹
组成：麝香1钱，朱砂2钱，硫黄3钱。
出处：《冯氏锦囊·杂症》卷十九。
主治：一切风寒湿气，流注作痛，手足踡挛，小儿偏搐，口眼斜，妇人心腹痞块攻疼，不问年深月久。小疖。
加减：None
功效：None
用法用量：用时将药置患处（重者用药米粒大，轻者用药牺粒大），以灯火点着，候至火灭，连灰罨于肉上，立见痊愈，只须1壮，不必复灸；若患处阔大，连排数壮，一起灸之（此药灸时不甚热痛，灸后并不溃脓）。
制备方法：上各为细末，先将硫黄化开，次入朱、麝同化，倾入瓷器内；候干再研末，隔火化开，候干，切作如粞、如米大，贮瓷瓶内，慎勿出气，珍藏听用。
临床应用：None
各家论述：None
用药禁忌：None
附注：观音救苦丹（原书目录）。本方方名，《卫生鸿宝》卷一引作“观音救苦针”，有蟾酥、冰片各一分。
