序号：61949
名称：救生丹
组成：龙脑1钱，朱砂1钱，雄黄1钱，牛黄1钱，芦荟1钱，胡黄连末1钱，麝香1钱，铅霜1钱，天竹黄1钱，曾青1钱，真珠末1钱，金箔50片，银箔50片，犀角屑1钱，干蝎末1钱，雀儿饭瓮3-7枚（内有物者）。
出处：《圣惠》卷八十五。
主治：小儿急惊风，四肢搐搦，多涎沫，身热如火，心神惊悸，发歇不定。
加减：None
功效：None
用法用量：每服1丸，以薄荷汤先研，滴在鼻内，男左女右，候嚏，即以薄荷酒服2丸，不拘时候。
制备方法：上为末，5月5日合和，用大活蟾10枚，子眉间各取酥少许，同研令匀，人饭和丸，如弹子大。着瓷碗内，用黄梢活蝎49枚，着碗内，令药弹丸触蝎毒，蜇入药，候毒尽，放蝎，然后重研药弹令匀，为丸如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
