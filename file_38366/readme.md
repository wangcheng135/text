序号：38376
名称：河车大造育麟丸
组成：熟地4两（9蒸9晒），紫河车1具（头胎肥大者，洗净蒸烂），山药（炒）1两5钱，白术（土炒）1两5钱，茯神1两，茯苓1两，人参1两，枣仁（去壳，炒，研）1两，麦冬（去心）1两，阿胶（蛤粉炒成珠）1两，续断肉（酒蒸）1两，杜仲（盐水炒断丝）1两，沙参1两，黄耆（蜜炙）1两，神曲（炒）1两，建莲肉（炒）1两，条芩（酒炒）1两，白芍药（酒炒）1两，丹皮1两，当归身1两5钱（酒洗），五味3钱（炒，研）。
出处：《胎产心法》卷上。
主治：妇人血气不足，苦于小产，或生而难育，或产下草迷而死。皆气血虚而胎不旺也。
加减：如血不热，去条芩、丹皮，加肉桂3-5钱亦可。
功效：None
用法用量：每服2-3钱，早、晚秋石或桂元汤任下。
制备方法：炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
