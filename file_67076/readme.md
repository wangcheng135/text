序号：67092
名称：麻黄汤
组成：麻黄1两，大枣20枚，茯苓3两，杏仁30枚，防风2两，白术2两，当归2两，升麻2两，芎2两，芍药2两，黄芩2两，桂心2两，麦门冬2两，甘草2两。
出处：《千金》卷七。
主治：恶风毒气冲心，脚弱无力，顽痹四肢不仁，失音不能言。
加减：None
功效：None
用法用量：上（口父）咀。以水9升，入清酒2升合煮，取2升半，分为4服，日3次，夜1次。覆令小汗，粉之，莫令见风。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：此方专祛上攻血脉之痹，故以芎、归、芍药、苓、防、大枣小续命汤中六味，参入麻黄汤中以开血脉之邪，兼升麻载诸药于上，苓、术渗湿著于下，麦门冬专为失音而设。
用药禁忌：忌海藻、菘菜、生葱、桃李、雀肉、酢物。
附注：None
