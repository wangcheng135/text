序号：27915
名称：诃黎勒散
组成：诃黎勒皮1两，赤茯苓3分，陈橘皮3分（汤浸，去白瓤，焙），枳实半两（麸炒微黄），桂心半两，白术3分，干姜1分（炮裂，锉），甘草1分（炙微赤，锉），人参3分（去芦头），木通半两（锉），厚朴3分（去粗皮，涂生姜汁，炙令香熟），半夏1分（汤浸7遍去滑）。
出处：《圣惠》卷四十七。
主治：上焦虚寒气滞，胸膈噎闷，饮食全少，或时痰逆。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，煎至5分，去滓，食后温服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“诃黎勒皮散”。
