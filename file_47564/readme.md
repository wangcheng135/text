序号：47579
名称：神仙六子丸
组成：菟丝子1两（细，酒浸1宿，焙），金铃子1两，枸杞子1两，覆盆子1两，五味子1两（焙），蛇床子1两（炒），何首乌1两（酒浸1宿，焙），地骨皮3两（酒浸1宿，焙），木瓜1两，舶上茴香2两（盐炒），熟地黄3两（焙），牛膝3两（酒浸1宿，焙）。
出处：《御药院方》卷六。
主治：男子气血衰败，未及年五十之上，髭鬓斑白，或年少人髭鬓苍黄。
加减：如要疾黑，前药内加人参、茯苓、石菖蒲各1两。药后百日内变黄白色如黑漆。
功效：常服养精髓，养气血，壮筋骨，补肾水，滑肌肤，驻容颜，黑髭鬓。
用法用量：每服50丸，空心食前温酒送下，日进1服。
制备方法：上为细末，用浸菟丝子酒澄清作面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌萝卜、生韭、薤、蒜菜。
附注：None
