序号：39226
名称：狗皮膏
组成：羌活2两，白芷2两，独活2两，良姜2两，川乌2两，草乌2两，麻黄2两，苍术2两。
出处：《全国中药成药处方集》（武汉方）。
主治：风寒或湿热所引起的一切肿痛麻痹。
加减：None
功效：None
用法用量：用时溶开贴患处，隔1-2日换1次。
制备方法：取上药，用麻油6斤，加鲜侧柏叶8斤、松毛尖8斤、生天雄1斤，同群药炸枯黑去滓，熬至滴水成珠；下黄丹32两（油、丹共重128两），搅匀，九折成膏115两2钱；用皮版摊，摊时加肉桂末16两，共成膏130两2钱。大张1两2钱，中张8钱，小张5钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
