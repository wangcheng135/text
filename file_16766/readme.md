序号：16773
名称：平补镇心丹
组成：白茯苓1两2钱半，茯神1两2钱半，麦门冬（去心）1两2钱半，五味子1两2钱半，车前子1两半，远志（制）1两半，天门冬（去心）1两半，山药（姜汁炒）1两半，熟地黄（酒浸）1两半，酸枣仁（炒）3钱，人参2两半，龙齿2两半，朱砂1两半（另研极细为衣）。
出处：《医统》卷四十八引《局方》。
主治：治心血不足，时或怔忡，夜多乱梦，如坠岸谷；心悬如大饥之状者。
加减：None
功效：常服安心神，益荣卫。
用法用量：每服80-90丸，早、晚米饮或温酒送下。
制备方法：炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：平补镇心丸（《风劳臌膈》）。
