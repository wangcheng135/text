序号：61482
名称：培坤丸
组成：炙黄耆3斤，白术3斤，炙草8两，广陈皮2斤，当归5斤，川芎1斤，杭芍1斤，拣砂仁9两，北沙参1斤，云茯苓2斤，枣仁2斤，寸冬2斤，杜仲（炒）2斤，核桃仁1斤4两，芦巴子2斤8两，醋炒艾叶1斤，元肉2斤，山药2斤，远志肉4两，熟地黄4斤，五味子8两，酥油4两。
出处：《全国中药成药处方集》（西安方）。
主治：妇人月经不调，赤白带下，子宫炎，身困肢懒，腹痛肢冷各症。
加减：None
功效：None
用法用量：每次3钱，黄酒或白开水送下。
制备方法：上药各为细末，以酥油溶拌微炒，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：中热肝郁者不宜服用。
附注：None
