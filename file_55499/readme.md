序号：55514
名称：桃仁承气汤
组成：桃仁半两，大黄（炒）2两，甘草2钱，肉桂1钱。
出处：《校注妇人良方》卷七。
主治：妇人瘀血，小腹急痛，大便不利，或谵语口干，漱水不咽，遍身黄色，小便自利；或血结胸中，手不敢近腹，寒热昏迷，其人如狂。
加减：None
功效：None
用法用量：姜水煎，发日5更服。
制备方法：None
临床应用：None
各家论述：《医略六书》：室女血瘀，冲任结滞小腹，而蓄泄不灵，故腹痛不止，经闭不通焉。桃仁生用破积血以开瘀结，大黄醋煮逐瘀血以通经脉，甘草和中缓胃，官桂通经活血也。水煎温服，使瘀血消化则冲任调和，而月事时下，何腹痛之有哉！
用药禁忌：None
附注：None
