序号：76337
名称：槐花酒
组成：槐花4-5两。
出处：《外科发挥》卷二。
主治：发背及一切疮毒，不同已成未成，但焮痛者；及湿热疮疥，肠风痔漏。
加减：None
功效：None
用法用量：上微炒黄，乘热入酒2钟，煎10余沸，去滓，热服。未成者2-3服，已成者1-2服。
制备方法：None
临床应用：1.肿毒：一人髀（骨甲）患毒痛甚，服消毒药不减。饮槐花酒一服，势随大退，再服托里消毒药而愈。_x000D_
2.发背：一人发背十余日，势危脉大，先饮槐花酒二服杀其势退，再服败毒散二剂，托里药数剂，渐溃，又用桑柴烧灸患处，每日灸良久，仍以膏药贴之，灸至数次。脓溃腐脱，以托里药白术、陈皮，月余而愈。_x000D_
3.肩疽：一人肩疽，脉数，用槐花酒一服，势顿退，更与金银花、黄耆、甘草十余服而平。
