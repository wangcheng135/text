序号：54219
名称：大垂云膏
组成：当归1两，附子（去皮脐，生用）1两，芎1两，防风1两，川升麻1两，槐子1两，细辛（去苗）1两，侧柏叶1两，桃仁（汤浸，去皮尖双仁）1分，杏仁（汤浸，去皮尖双仁）1分，甘草1分，桑根白皮1分，白及1分，黄耆1分，白僵蚕1分，垂柳1握（煎了不在吊），黄丹7两，雄黄半两，朱砂1分（细研），硫黄2分（细研），麝香1钱（细研），白芷1分，没药1分，麒麟竭1分（细研），龙脑1分（细研），黄蜡4两（细研），油1斤半。
出处：《圣惠》卷六十三。
主治：一切恶疮焮肿，发背，疽疮，风肿，肠痈，乳痈，瘰疬，疥癣，发鬓，牙痈，发脑，肾痈，马坠磕破骨损、及一切虫蛇毒物咬伤。
加减：None
功效：None
用法用量：发背疮，热酒调1钱服，外贴之。余症外贴，虎豹咬着，用甘草水洗后贴之。
制备方法：上药除研了药并丹外，细研，先熬油令沸，下锉药，煎候白芷黄赤色，以绵滤过，拭铛令净，再煎，下丹，以柳木篦搅，候变黑，即下蜡熔尽，滴于水中为珠子不散，即次下诸药末，搅令匀，以瓷盒盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
