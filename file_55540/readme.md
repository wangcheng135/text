序号：55555
名称：桃仁散
组成：桃仁半两（汤浸，去皮尖双仁，麸炒微黄），鳖甲2两（涂醋，炙令黄，去裙襕），琥珀1两（细研），肉桂1两（去粗皮），赤芍药3分，当归3分（锉碎，微妙），白术3分，木香半两，诃黎勒皮半两，干姜半两（炮裂，锉），人参半两（去芦头），延胡索3分，赤茯苓3分，陈橘皮1两（汤浸，去白瓤，焙），牛膝3分（去苗）。
出处：《圣惠》卷七十。
主治：妇人冷劳气滞，经脉不通，腹胁妨闷，四肢羸瘦，不思饮食。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，煎至6分，去滓，食前温服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
