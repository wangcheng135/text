序号：81067
名称：麝香膏
组成：麝香（研）1分，猪牙皂荚3挺（烧存性），白矾1两，绿矾1两半（与白矾同杵碎，入铫子内，烧令枯、研），腻粉（研）半两，水银半两，黄柏（去粗皮）1两，苦楝根白皮1两，密佗僧1两。
出处：《圣济总录》卷一七二。
主治：小儿齿损烂。及走马急疳，及牙齿诸疾。
加减：None
功效：None
用法用量：先净漱口，涂之。如久患者，取药半匙，并砒霜、粉霜末各少许拌匀使。有津吐之。
制备方法：上为细末，用无灰酒3升，熬成膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：九仙膏用法：九味捣研细，以好酒三升调药，用慢火熬成膏，瓷盒内盛，勿令泄气。小儿患口疳，即米泔化涂之，及米泔内服如录豆大三丸。如大人患口齿臭烂者揩之，亦用米泔内服5-7丸。牙疼，即先以米饮漱口，后以米泔化药如菜子大，点牙缝及蚛穴中。
