序号：66721
名称：鹿茸丸
组成：鹿茸2两（去毛，涂酥炙令微黄），莨菪子1两（水淘去浮者，水煮令芽出，候干，炒黄黑色），磁石1两（烧赤，醋淬10遍，细研，水飞过），附子1两（炮裂，去皮脐），天雄1两（炮裂，去皮脐），硫黄1两（细研，水飞过），蛇床仁1两，韭子1两（微炒），桂心1两，硇砂1两（细研），龙骨1两，熟干地黄1两。
出处：《圣惠》卷七。
主治：肾脏虚损，阳气萎弱，精泄不禁。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，用羊肾一对，去脂膜，研如泥，以酒2升，煎成膏，入诸药末，为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
