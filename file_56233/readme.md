序号：56248
名称：流气丸
组成：人参2钱，焦术2钱，茯苓2钱，炙甘草2钱，清夏2钱，广皮2钱，丁香2钱，沉香2钱，木香1钱，肉桂1钱，香附4钱，白芷2钱，紫苏2钱，草果仁2钱，青皮2钱，大黄2钱，枳壳3钱，厚朴3钱，槟榔2钱，莪术2钱，麦冬2钱，木瓜2钱，木通2钱，白蔻仁2钱。
出处：《全国中药成药处方集》（沈阳方）。
主治：胸膈闷满，饮食不下，气滞作痛，周身浮肿，气郁痰喘，小便不利，疮毒气肿。
加减：None
功效：散寒通滞，行气健胃。
用法用量：每服1丸，姜汤送下。
制备方法：上为极细末，炼蜜为丸，2钱重。
临床应用：None
各家论述：None
用药禁忌：忌生冷硬物。
附注：None
