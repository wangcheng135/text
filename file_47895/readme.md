序号：47910
名称：神圣北庭丸
组成：北庭（去砂石，研）1分，没药1分，木香1分，当归1分，芫花半两，莪术半两，巴豆（去皮膜心）40粒。
出处：《妇人良方》卷七引《灵苑方》。
主治：妇人积年血气，攻刺心腹疼痛不可忍者，及多方医疗未愈；或治男子血气；急喉闭者。
加减：None
功效：None
用法用量：每服5丸，临时加减丸数，用酒、醋各半盏煎数沸，通口服。不得嚼破，仍须吃尽酒、醋，立愈。或男子血气，亦依前方服食；如急喉闭者，男左女右，以1丸鼻中嗅之立愈。
制备方法：上先研北亭、没药、巴豆如粉，用好米醋3升同煮为稀膏，然后将余4味为细末，入于膏内搜合成块，用新瓦盒盛之，丸如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
