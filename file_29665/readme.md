序号：29673
名称：鸡鸣丸
组成：知母4两（炒），杏仁（去皮尖）2钱，桔梗（去芦）5钱，阿胶（麸炒）4钱，葶苈（火上焙）3钱，款冬花4钱，旋覆花1两，半夏（姜汁炒）3钱，甘草（炙）1两，陈皮（去白）1两，兜铃1两，五味子4钱，麻黄1两，人参5钱。
出处：《回春》卷二。
主治：风寒咳嗽，发热恶寒，头痛体痛，咳嗽痰清，以及肺经寒湿，痰饮喘嗽，交冬即发，喘咳难卧。
加减：None
功效：宣肺化痰定喘。
用法用量：每服1丸，五更，乌梅、生姜、大枣汤送下。
制备方法：上为细末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：忌食辛辣、腻物。
附注：鸡鸣定喘丸（《全国中药成药处方集》吉林方）。
