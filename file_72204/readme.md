序号：72220
名称：椒朴丸
组成：汉椒（去目）2两，厚朴（去粗皮，锉）2两，茴香2两，青盐（淘去沙土，取浮）2两。
出处：《苏沈良方》卷四。
主治：脾胃虚冷岁久，不思饮食，或发虚肿，或日渐羸瘦，四肢衰倦，吐利无节。
加减：None
功效：None
用法用量：每服30-40丸，空心米饮及盐汤送下，病深者，1日3次。
制备方法：上药以水2升，煮令干，焙燥，捣为末，面糊为丸，如梧桐子大。
临床应用：水肿予中表许君，病脾逾年，通身黄肿不能起，全不嗜食，其甥为本道转运使，日遣良医治之都不效，有傅主簿传此方，服十许日渐安，自尔常服，肌肤充硕，嗜饮美食，面色红润，年六十余，日行数十里，强力如少年。
各家论述：None
用药禁忌：None
附注：此方慎勿增他药，药之中病，自有奇功，否则却致不验。
