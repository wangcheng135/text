序号：81841
名称：熟地黄丸
组成：熟干地黄3两，牛膝（去苗）2两，远志（去心）2两，巴戟2两，石斛（去根，锉）2两，桂心2两，车前子2两，菟丝子（酒浸3日，晒干，别捣为末）2两，覆盆子2两，天门冬（去心，焙）2两，何首乌2两，白茯苓2两，黄耆（锉）2两，鹿茸（去毛，涂酥炙微黄）2两，附子（炮裂，去皮脐）2两，沉香2两。
出处：《圣惠》卷九十八。
主治：下元虚冷，腰脚无力。
加减：None
功效：益颜色，美髭鬓，补虚损。
用法用量：每日服50丸，空心以温酒送下。
制备方法：上为末。炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：熟干地黄丸（《普济方》卷二二一）。
