序号：60576
名称：通经妙灵丸
组成：黄连（酒炒）1两，苍术（米泔浸炒）2两，黄柏（盐酒炒）2两，肉桂（去皮）4两，南芎5分，当归（酒洗）1两，白芍（盐酒炒）1两3钱，汉防己（酒洗）3钱，白芷2钱半，桃仁（去皮尖）3钱，威灵仙1两（酒浸，蒸晒9次），羌活（酒洗）3钱，龙胆草（酒洗）1钱，红花（酒洗）5钱，防风（酒洗）5钱，龟版（酥炙）5钱，杜仲（姜汁炒）8钱。
出处：《古今医鉴》卷十。
主治：因酒色所伤，筋脉空虚，感受风寒湿热，损伤经络，致遍身走痛如刺，日轻夜重，左足痛尤甚。
加减：None
功效：疏筋活血行湿。
用法用量：每服100丸，空心陈酒送下，盐汤亦可。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
