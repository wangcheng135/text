序号：79607
名称：熏陆香丸
组成：乳香（别研）1分，沉香1分，木香1分，丁香2钱，肉豆蔻（面裹煨香）1分，人参（去芦头）1分，青橘皮（去白）3分，延胡索（炒香）2钱半，当归（洗，焙）2钱半，蓬莪荗（煨，切）1分，硇砂1钱（不夹石者，研细，用醋飞过），甘草（微炙）2钱，没药（别研）2钱，血竭（别研）2钱。
出处：《杨氏家藏方》卷十六。
主治：妇人血气凝涩，经候不行，有时作痛。
加减：None
功效：调经止痛，安和脏气。
用法用量：每服1丸，细嚼，炒生姜、盐汤送下，或用当归酒下，不拘时候。
制备方法：上为细末，炼蜜为丸，每1两作30丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
