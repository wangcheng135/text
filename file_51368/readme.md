序号：51383
名称：贴脐膏
组成：大川芎2钱，当归2钱，白芍2钱，地黄2钱，人参2钱，牡丹皮2钱，白术2钱，白苓2钱，黄耆2钱，厚桂2钱，泽泻2钱，大附子4钱，知母4钱，黄柏3钱，干姜4钱，北细辛4钱，胡芦巴4钱，白芷4钱，远志4钱，巴戟4钱，菟丝子4钱，蛇床子4钱，故纸4钱，苁蓉4钱，锁阳4钱，木鳖子4钱，蓖麻子4钱，龙骨4钱，石枣4钱，山药4钱，杏仁4钱。
出处：《膏药方集》引《外科活人定本》。
主治：男子遗精、白浊，女人赤白带、崩漏。
加减：None
功效：None
用法用量：摊用。
制备方法：水煎去滓，至大半干入油4两，桃、柳枝搅不住手，搅至水干，入密陀僧极细末1两半，成膏后入龙骨1钱5分，麝香1分，樟脑1钱5分。
临床应用：None
各家论述：None
