序号：60096
名称：逍遥散
组成：当归（酒洗）1钱5分，白芍（酒炒）1钱，柴胡1钱，黄芩1钱，川芎7分，熟地黄7分，半夏（姜炒）7分，人参5分，麦门冬（去心）5分，甘草4分。
出处：《寿世保元》卷七。
主治：室女17-18岁，脾胃受伤，气血俱弱，误食生冷，经脉不通，或百日或半年，颜色青黄，饮食少进，寒热往来，四肢困倦，头疼目弦，肚疼结块，五心烦热，呕吐膨胀。
加减：少睡，加酸枣仁（炒）以敛心血。
功效：和气血，扶脾胃。
用法用量：加生姜3片，水煎，热服。
制备方法：上锉散。
临床应用：None
各家论述：None
用药禁忌：None
附注：原书治上症，先用本方，次服加味八物汤，后服调经丸。
