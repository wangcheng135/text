序号：84173
名称：五石乌头丸
组成：钟乳（研，炼）2两半，紫石英（研，炼）2两半，白石英（研，炼）2两半，石硫黄（研）2两半，黄芩3分，白薇3分，白术3分，矾石2两（烧），干地黄7分，芍药1两，附子（炮，去皮）1两，乌头15枚（炮，去皮），吴茱萸2两半，蜀椒（去目、闭口者，汗）5分，人参5分，细辛5分，白石脂5分，赤石脂5分，山茱萸5分，天雄（炮，去皮）5分，芎5分，麦门冬（去心）5分，前胡5分，半夏（洗）5分，龙骨5分，桂心5分，远志15枚（去心），茯苓1两半，黄连1两半，当归1两半，紫菀1两半，禹余粮1两半，云母粉1两半，甘草（炙）1两半。
出处：《千金翼》卷二十二。
主治：男子五劳七伤，诸积冷，十二风痹，骨节沉重，四肢不举，食饮减少，羸瘦骨立，面目焦黑，时时或腹内雷鸣，膀胱常满，或下青黄，经时不止；妇人产后恶血不尽，腹内坚强，诸劳少气，百病间发，或时阴肿，或即脱肛及下出疼痛者。
加减：None
功效：None
用法用量：每服10丸，酒送下，1日3次。不知，可增至20丸，以心热为知力也。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
