序号：43214
名称：养血丸
组成：牡丹皮1两，白芍药1两，卷柏1两，当归（洗，焙）1两，石斛1两，白茯苓（去皮）1两，巴戟（去心）1两，熟地黄（洗，焙）1两，肉苁蓉（酒浸1宿，切，焙干）1两，杜仲（去粗皮，炙）1两，山药1两，柏子仁（别研）1两，白薇1两，枳壳（去瓤，麸炒黄色）1两，蒲黄（微炒）1两，肉桂（去粗皮）1两，京三棱（煨香，切）1两，蓬莪术（煨香，切）1两，枸杞子1两，覆盆子1两，附子（炮，去皮脐）半两。
出处：《杨氏家藏方》卷十五。
主治：月事阻滞，腹胁作痛；或结坚块，面黄发落，时发寒热，身体羸瘦。
加减：None
功效：补血海，疔虚弱，调经，快三焦，进饮食，久服令人肥盛有子。
用法用量：每服50丸，空心，食前以温酒或米饮送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
