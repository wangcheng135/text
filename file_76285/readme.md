序号：76301
名称：紫霞丹
组成：黑铅1两，雌黄3钱，雄黄3钱，硫黄5钱，白铅4两，阳城罐1个，盐泥固济，晒干，将黑白2铅铺底盖面，药放中间，以铁盏封固严密，铁线绊紧，架3钉上，外用八卦炉文武火5炷香，水升盏，研极细末。白茯苓末9钱5分，明没药（炙，去油，研末）9钱5分。
出处：年氏《集验良方》卷二。
主治：腹胁积聚，七癥八瘕，翻胃噎膈，攀睛胬肉，女人寒病带下。
加减：None
功效：None
用法用量：每服1钱，酒送下。每次用生药一半，如前封固，升炼9次，名九转仙丹。
制备方法：用头生男乳汁拌药，为丸如绿豆大。
临床应用：None
各家论述：None
用药禁忌：忌金石之药，不宜轻服。
附注：None
