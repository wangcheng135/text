序号：70613
名称：紫苏饮
组成：紫苏8分，枳壳6分，黄芩（炒）7分，柴胡6分，川芎8分，陈皮4分，茯苓5分，防风6分，当归6分，甘草4分，生姜3片。
出处：《妇科胎前产后良方注评》。
主治：胎前伤风而见恶寒发热头痛者。
加减：None
功效：发散风邪。
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：本条所用紫苏饮，系《局方》香苏散加味。方中香苏散为治四时温疫伤寒主方，柴胡、黄芩、防风疏风解表，当归、川芎、枳壳、香附、陈皮、生姜理气和中，养血安胎，茯苓、甘草健脾益气，合用之可使风寒得解而胎元自固，药虽平淡而方法合拍，可用于妊娠伤风而见恶寒发热头痛诸症者。
用药禁忌：None
附注：None
