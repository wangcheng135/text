序号：66470
名称：小儿四症丸
组成：广木香2钱，苏叶1两5钱，陈皮1两，厚朴1两（姜制），藿香1两，白术1两（麸炒），茯苓1两（去皮），炒麦芽1两，炒苍术1两，花粉7钱5分，泽泻7钱5分，山楂7钱5分，猪苓7钱5分，制半夏7钱5分，白芷5钱，桔梗5钱，滑石5钱，砂仁5钱，神曲（麸炒）7钱5分。
出处：《全国中药成药处方集》（天津方）。
主治：小儿消化不良，呕吐泻肚，小便不利，肚腹胀痛，中暑中寒，头痛身热，口渴舌干，烦躁不宁。
加减：None
功效：健胃消食，利尿止泻。
用法用量：每服1丸，周岁以内酌减，白开水化下。
制备方法：上为细末，每细末15两2钱，兑琥珀面5钱，和匀，炼蜜为丸，1钱重，每斤丸药用朱砂面3钱为衣，蜡皮或蜡纸筒封固。
临床应用：None
各家论述：None
用药禁忌：忌生冷油腻。
附注：None
