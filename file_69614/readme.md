序号：69630
名称：琥珀煎
组成：乳香末2钱，蕤仁（研）半两，滑石2两，铅丹2两，木鳖子（去壳）10枚，黄连末1两，秦皮1两，柳枝10枝，槐枝（并新青者）10枝（每枝长1寸半），白蜜4两，黄芩（去黑心）4两。
出处：《圣济总录》卷一○四。
主治：风毒冲目，肿赤痒痛。
加减：None
功效：None
用法用量：每用铜箸点，以目涩为度。熬、点不得犯铁器。
制备方法：上将槐、柳枝、秦皮、黄芩、滑石等为粗末，以水3碗，同煎至2碗，去滓，其余乳香、蕤仁、铅丹、木鳖子4味与蜜同熬，如琥珀色，却共前项药汁并黄连，同煎至1碗半，用熟绢滤去滓，入瓷器内密封，系垂在井底1夜，出火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
