序号：59655
名称：调元百朴膏
组成：当归身（酒洗）4两，怀生地黄2斤，怀熟地黄4两，甘枸杞子1斤，白芍（米粉炒）1斤，人参4两，辽五味子1两，麦门冬（去心）5两，地骨皮4两，白术（去芦）4两，白茯苓（去皮）12两，莲肉4两，怀山药5两，贝母（去心）3两，甘草3两，琥珀1钱3分，薏苡仁（用米粉炒）8两。
出处：《寿世保元》卷四。
主治：五劳七伤，诸虚劳极，元气虚损，脾胃亏弱。
加减：吐血，加牡丹皮2两；骨蒸，加青蒿汁、童便各2碗，同热服之。
功效：养血和中，宁嗽化痰，退热定喘，止泻除渴。
用法用量：每服3匙，白汤调下。
制备方法：上锉细末，和足水10斤，微火煎之，如干，再加水10斤，如此4次，滤去滓，取汁，文武火熬之，待减去3分，每斤加炼净熟蜜4两，春5两，夏6两，共熬成膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
