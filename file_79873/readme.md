序号：79889
名称：广嗣良方
组成：山茱萸（酒浸，去核）5两，香附子（去毛，4制）5两，川芎（酒洗）3两，熟地黄（酒洗，捣极烂）3两，白芍药（去皮，酒炒黄）4两，益母草3两，条芩（酒炒）2两，蛇床子（水洗净，微炒）2两，覆盆子（微炒）2两，玄胡索（微炒）2两，陈皮（水洗，去白）2两，苍术（米泔水浸1宿）3两，砂仁（去壳）1两5钱，丹参（水洗）2两，当归（酒洗，去芦，全用）3两，白丝毛乌骨雄鸡1只（预先喂养1月，不令与雌鸡同处，临合将线缉死，不出血，干去毛，剖开去肠内污物，并膆内宿食，肫内黄皮，用酒洗净，一应事件仍装入鸡肚内，不令见水，置土坛内，入酒2斤，封固，重汤煮烂取出，刮下净肉，捣如泥，仍将鸡骨酥油和原汁或酒炙酥为末，入药末拌匀）。
出处：《墨宝斋集验方》卷上。
主治：女子不孕。
加减：如月信先期而至者，加黄芩1两5钱，地骨皮1两5钱，黄连1两5钱，清米饮送下；如月信后期而至者，加黄耆1两，人参1两5钱，白术1两5钱，温酒淡盐汤任下，如下白带者，加苍术1两5钱，白术1两5钱，柴胡1两5钱，升麻1两5钱，白芷1两5钱，淡姜汤送下。
功效：None
用法用量：每服40-50丸，渐加至80-90丸，空心清米饮送下。
制备方法：上为极细末，同鸡肉地黄入醋煮米糊拌匀，木臼内捣极细，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
