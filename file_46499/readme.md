主治：眼生翳膜，血灌瞳神，迎风流泪，拳毛倒睫，赤烂风弦。
加减：眼害日久，有宿沙翳者，加螵蛸、珊瑚、曾青、珍珠，各研极细加入；病疮抱住黑睛者，加飞过灵砂少许，与白丁香研1处，用乌鸦翎搅匀；血灌瞳神，加官硼、曾青（即胆矾是也）、琥珀、朴消少许，研细入；束睛云翳者，加白翠、螵蛸、珊瑚、珍珠；有青红筋者，加轻粉，枯矾；内障气，加曾青、熊胆、珊瑚、琥珀、珍珠、神砂少许；胬肉攀睛者，加硇砂少许、鹰粪、人退；多年老眼云翳遮睛至厚者，全料点之；迎风冷泪，眼昏花者，用主方治之自愈，不必加别药，惟少加冰片；拳毛倒睫，加珍珠、冰片、琥珀；赤烂风弦者，加硼砂、珍珠，再用铜绿1两，用天茄汁，和艾熏透洗之妙，外用点药。
功效：None
用法用量：上药俱切为细片，用童便1桶将水澄，盛瓷盆中；入炉甘石3斤，浸之1日夜，澄清再浸，澄出；将炉甘石入混元球内煅红，入药水浸。如此10数次，冷定，取出炉甘石，入阳城罐内封固打火，每罐打3炷香升盏。轻清者，合后药可治瞎目；坠底者，可治火眼。诸药加减于后。如不入罐打火，将甘石研细，用水飞过，分清浊两用亦可。如制甘石10两，加琥珀5钱，珍珠8钱，俱各用混元球煅过，为极细，冰片3钱，官硼3两，铜器上飞过，海螵蛸6钱（生用），胆矾2两（用铜瓦片煅过），白翠2两（煅红入童便内，不拘遍数，以成腻粉为止），鹰粪3钱（用竹叶上焙过，研细），熊胆3钱（用缸瓦上煅过存性，为末），真正者人退1两（洗净，炒黄色存性，为细末），木贼1两（焙过，为细末），枯矾5钱，轻粉3钱，辰砂3钱，皮消3钱。此乃全料分两，亦当随其目疾而治之，无不取效矣。
