序号：19469
名称：地黄丸
组成：生干地黄2两，柴胡（去苗，净洗）半两，秦艽（净洗，去芦）半两，黄芩半两，赤芍药1两。
出处：《本事》卷十。
主治：尼师寡妇，独阴无阳，欲男子而不可得．以致恶风体倦，乍寒乍热，面赤心烦，或时自汗，厥阴脉弦长而上出鱼际。
加减：None
功效：None
用法用量：每服30丸，乌梅汤送下，日3次，不拘时候。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《本事方释义》：生干地黄气味甘苦微寒，入手足少阴、厥阴；柴胡气味辛甘平，入足少阳；秦艽气味苦平，入手足阳明，兼入肝胆；黄芩气味苦寒，入手太阴、少阳；赤芍药气味苦平，入足厥阴，能行血中之滞；乌梅汤送药，亦取其泄肝也。师尼寡妇，独阴无阳，情欲未遂，以致阴阳交争，乍寒乍热，将欲成劳者，非此不能治。
用药禁忌：None
附注：赤芍地黄丸（《医学入门》卷八）、抑阴丸（《古今医鉴》卷十一）、断欲丸（《寿世保元》卷七）。本方方名，《妇人良方》引作“生地黄丸”，《准绳·类方》引作“抑阴地黄丸”。
