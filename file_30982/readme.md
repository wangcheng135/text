序号：30991
名称：十鼓通证散
组成：大戟、甘遂、麻黄、乌梅、葫芦巴、葶苈、芫花、黑牵牛、细辛、汉防己、槟榔、海蛤、陈皮、桑皮。
出处：《摄生众妙方》卷六。
主治：十鼓证。气鼓、食鼓、热鼓、风鼓、劳鼓、湿鼓、虫鼓、血鼓、疳鼓，胸腹肿胀，并四肢肿者。
加减：None
功效：None
用法用量：每服1钱，或2-3钱，五更用生姜汤调服。
制备方法：上为细末。
临床应用：None
各家论述：方中大戟取膀胱水；甘遂取肝水；麻黄取肤水；乌梅取腹水；葫芦巴取胃水；葶苈取心水；芫花取遍身水；黑牵牛子取遍身水；细辛取气水；汉防己取胃水；槟榔取血水；海蛤取肺水；陈皮去白取牙水；桑皮取肠水。
用药禁忌：忌盐、醋、酱一百日。
附注：None
