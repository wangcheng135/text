序号：53844
名称：射干煎
组成：射干8两，大青3两，石膏10两（一作1升），赤蜜1升。
出处：《千金》卷十五。
主治：舌本强直，或梦歌乐而体重不能行。脾实，咽干口燥，舌本肿强，腹胁满胀，大便涩难。
加减：None
功效：None
用法用量：上（口父）咀。以水5升，煮取1升5合，去滓，下蜜，煎取2升，分3服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：射干苦寒有毒而能解毒，为喉痹咽痛专药，舌本强直，亦宜用之，以其能破宿血散结气也；大青解心下热毒，泻肝胆实火，正所以祛心胃之邪热；石膏治心下逆气，舌焦不能息，腹中坚痛，肢体沉重；赤蜜主心腹邪气，止痛解毒，且能安五脏，和百药。
用药禁忌：None
附注：本方方名，《外台》引作“射干汤”。
