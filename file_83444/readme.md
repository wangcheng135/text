序号：83462
名称：乌金散
组成：乌金子（即黑豆）5钱，紫葳（即凌霄花）5钱，大蓟根5钱，小蓟根5钱，当归5钱，肉桂（去皮）5钱，血余（无病者，烧存性）5钱，蒲黄5钱，木香5钱，青皮5钱，赤芍5钱，皂荚（不蛀者，烧存性）5钱，蚕蜕纸（烧存性）5钱，棕毛（煅）5钱，红花1两，川乌1枚（生用）5钱，辰砂少许，血竭少许。
出处：《胎产秘书》卷下。
主治：产后十八症：一、难产；二、胞衣不下；三、死胎不下；四、眼目昏花；五、口干心闷；六、寒热如疟；七、咳嗽寒热不定；八、败血如肝；九、败血入四肢浮肿；十、失音不语；十一、血邪癫狂言语；十二、心腹痛；十三、百节痠疼；十四、舌干津枯，鼻中出血，绕项生疮；十五、腰疼如角弓；十六、小便短缩；十七、喉苦蝉声；十八、胸膈气满，喘逆不食。
加减：None
功效：None
用法用量：每服1钱，姜汤或酒调下，甚至1日3剂。
制备方法：上除烧灰者另研外，共为细末，入烧灰药研匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
