序号：76532
名称：雷音丸
组成：巴豆2两（去仁不用，只用豆皮，每豆2两，可得皮3-4钱，微炒黄色，万不可用豆仁1粒），缩砂仁1两（炒），川大黄3钱（半生半炒），干姜3钱（炒黑），广木香3钱（炒黑），牙皂2个（去筋，炒），甘遂1钱5分（以甘草水浸3日，日换1次，看水无黑色为度，然后用面包，向火煨之，面俱黄色而止）。
出处：《回生集》卷上。
主治：水臌，酒积，食积。
加减：None
功效：None
用法用量：每服30-40丸，晨空心姜汤送下。每服可泄水二三次，日服日泻，日泻日消，大便渐实，小便渐长渐白，直服至水尽为度，但须量老少壮弱，泻之多寡，加减丸数，不可拘执。此药治病，多则一料，少则半料必愈。
制备方法：上为细末，绢罗过，醋打面糊为丸，如绿豆大，锅底烟煤研细为衣，晒干。
临床应用：None
各家论述：None
用药禁忌：忌盐酱一百多日。
附注：此药虽泄而不伤元气。
