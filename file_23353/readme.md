序号：23361
名称：防风羊角汤
组成：防风1两，赤芍半两（炒），桂心半两，羚羊角2钱，川芎2钱，羌活2钱，当归2钱，酸枣仁2钱（炒），牛蒡子（炒）2钱。
出处：《济阴纲目》卷十二。
主治：产后气血不足，血亏寒滞，热郁于经，中风挛痛，肢体挛痛，背项强直。
加减：None
功效：None
用法用量：每服4钱，水煎服。
制备方法：上锉。
临床应用：None
各家论述：《医略六书》：防风疏风邪之外客，羚羊清郁热之炽热，羌活除肢体之痛，牛蒡除项背之强，当归养血脉以荣经，川芎行血气以活络，赤芍泻血中之滞，桂心温血室之寒，枣仁养心润燥以资营血也。水煎温服，使滞化血荣，则热郁自泄而风寒解散，经络清和，肢体项背无不皆受其荫，安有挛急疼痛、牵强挺直之患乎？
用药禁忌：None
附注：防风羚羊角汤（《医略六书》卷三十）。
