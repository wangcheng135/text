序号：60047
名称：起死救儿丹
组成：人参3钱，玄参1两，金银花1两，白术2钱，当归3钱，麦冬3钱，甘草1钱，荆芥2钱，天花粉2钱，茯神3钱。
出处：《辨证录》卷三十四。
主治：小儿痘疹5-6日后，色变纯黑或炭灰之色，头顶陷下不起，饮食到口即吐，所谓坏症者。
加减：None
功效：None
用法用量：水煎服。1剂黑变为红，再剂而陷者起，干者润，饮食知味。
制备方法：None
临床应用：None
各家论述：此方之妙，全在金银花与玄参之多，即能解毒，复善散火，而又助之参、术、归、冬，则足以济二味之力，而益成其祛除之功，所以能变败而为胜，起死而变生也。万勿惊其药品之重与用参之多，而减其分量。盖药不重，火毒难消，参不多则阴阳难复矣。
用药禁忌：None
附注：None
