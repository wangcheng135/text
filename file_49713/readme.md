序号：49728
名称：茯苓汤
组成：半夏3两（汤洗10次），茯苓1两8钱，熟地黄1两8钱，橘皮1两2钱，细辛1两2钱，人参1两2钱，芍药1两2钱，川芎1两2钱，旋覆花1两2钱，桔梗1两2钱，甘草（炙）1两2钱。
出处：《三因》卷十一。
主治：忧怒兼并，气攻血溢，停留胃管，嗳闻血腥，呕吐食饮，及妊娠恶阻，中脘宿冷，冷血侵脾，恶闻食气。
加减：客热烦渴口疮者，去橘皮、细辛，加前胡、知母；肠冷下利者，去地黄，入桂心（炒）；胃中虚热，大便秘，小便涩，去地黄，加大黄1两8钱，黄芩6钱。
功效：None
用法用量：每服4大钱，水2盏，加生姜7片，煎7分，去滓空腹服。
制备方法：上锉散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
