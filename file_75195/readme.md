序号：75211
名称：腽肭脐丸
组成：腽肭脐1对，麋茸（去毛、酒浸1宿，炙）5两，鹿茸（去毛，酒浸1宿，炙）5两，苁蓉（酒浸1宿）5两，当归3两，茯神（去皮心）3两，朱砂（蜜煮）3两，牛膝（酒浸1宿）3两，五味子3两，巴戟（去心）3两，青盐（炒）1两，阳起石（酒煮1日）2两，沉香2两，附子3两（炮，去皮），菟丝子5两（净酒浸1宿）。
出处：《朱氏集验方》卷八。
主治：虚损。
加减：None
功效：补真助阳，益壮根本。
用法用量：每服70-100丸，食前盐酒汤送下。
制备方法：上为细末，用腽肭脐并鹿角胶为膏子，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
