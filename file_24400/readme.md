序号：24408
名称：安息香丸
组成：肉桂（去粗皮）2两半，诃子（炮，取皮）2两，阿魏（细研，白面少许搜和作饼子，炙令香熟）1分，茯苓1两半，当归1两半（汤洗，切片，焙干），干姜1两半（炮，去皮），肉豆蔻1两半（去壳），川芎1两半，丁香皮1两半，缩砂仁1两半，五味子1两半（微炒），巴戟1两半（去心，面炒），益智子1两半（去皮），白豆蔻（去皮）1两半，硇砂1两半（酒半盏化，去砂，入蜜），香附（去毛）1两半，茴香（微炒）1两半，胡椒1两，高良姜1两，木香1两，沉香1两，乳香（别研）1两，丁香1两。
出处：《局方》卷三。
主治：一切冷气，心腹疼痛，胸膈噎塞，胁肋膨胀，心下坚痞，腹中虚鸣，哕逆恶心，噫气吞酸，胃中冷逆，呕吐不止，宿饮不消，胸膈刺痛，时吐清水，不思饮食。
加减：None
功效：None
用法用量：每服1丸，细嚼，以温酒送下；浓煎生姜汤下亦得，食前服。
制备方法：除安息香、硇砂外，为细末，用蜜30两，入安息香、硇砂于蜜中炼熟，与上药为丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
