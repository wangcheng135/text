序号：13252
名称：人参丸
组成：人参2两，陈橘皮（汤浸，去白）1两1分，厚朴（去粗皮，生姜汁炙）2两，白术（锉）2两，干姜（炮裂）半两，甘草（炙，锉）1两，赤石脂1两，茯神（去木）1两，当归（切，焙）1两，薏苡仁1两，麦门冬（去心，焙）麦芽（炒）1两，紫苏子（炒）2合，细辛（去苗叶）3分，杏仁（去皮尖双仁，炒）3分。
出处：《圣济总录》卷八十八。
主治：虚劳，脾胃冷弱，饮食不消，气逆烦满，稍热即发虚烦。
加减：None
功效：None
用法用量：每服20丸，食前温酒送下，1日3次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
