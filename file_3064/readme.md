序号：3064
名称：天麻乌蛇丸
组成：天麻（酒浸1宿，切，焙）1两，乌蛇（酒浸1宿，去皮骨，炙）1两，天南星（浆水浸1宿，切，焙）2两，半夏（浆水煮过，切，焙）半两，藿香叶1两，乌头（去皮脐，生用）1两，白附子（生用）1两，腻粉（研）1两，仙灵脾（用叶）半两，雄黄（研）1分，铅白霜（研）1分，丁香1分，犀角（镑屑）半两，人参半两，麝香（研）1钱，龙脑（研）1钱，干蝎（全者，去土，炒）1分，丹砂（研）半两，槐胶1分，桑螵蛸20枚（炙），蛇黄（烧，醋淬7遍，研）1分。
出处：《圣济总录》卷十五。
主治：风痫心惊，身热瘈疭，摇头口噤，多吐涎沫，不自觉知。
加减：None
功效：None
用法用量：每服10丸，温酒送下，食后、临卧服。
制备方法：上药捣罗15味为细末，入研药6味拌匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
