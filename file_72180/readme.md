序号：72196
名称：椒艾丸
组成：蜀椒300粒，熟艾1升，干姜3两，赤石脂2两，乌梅100枚。
出处：《千金》卷十五。
主治：三十年下痢，所食之物皆不消化，或青或黄，四肢沉重，起即眩倒，骨肉消尽，两足逆冷，腹中热，苦筋转，起止须扶，阴冷无子。
加减：如不愈，加黄连1升。
功效：None
用法用量：每服10丸，日3服；不愈，加至20丸。
制备方法：上椒、姜、艾为末，梅著1斗米下蒸，令饭熟，去核，纳姜、椒末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：蜀椒、干姜温中，石脂、乌梅敛脱，熟艾恢复元阳，温暖子脏，故可治阴冷无子。服之若不愈，必有积热伏匿于中，则加黄连以分解之。
用药禁忌：None
附注：None
