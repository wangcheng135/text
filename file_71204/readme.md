序号：71220
名称：紫蔻丸
组成：红人参2两，陈皮5钱，云苓1两，丁香3钱，焦术1两5钱，紫蔻5钱，本色沉香5钱，焦楂3两，莱菔子1两，草蔻1两，广木香5两，香附2两，藿香5钱，榔片1两，广砂仁5钱，神曲1两，麦芽1两，枳壳1两，甘草5钱，官桂3钱，良姜5钱，青皮1两。
出处：《全国中药成药处方集》（抚顺方）。
主治：胃弱，食欲不振，食后闷满，嗳气不舒，颜面黄瘦，或暴饮暴食，呃逆脘闷，恶心呕吐，嘈杂吞酸，消化不良，气滞胃痛。
加减：None
功效：健胃助消化。
用法用量：每服1丸，每日早、晚用姜汤送下。
制备方法：上为细面，炼蜜为丸，2钱重，蜡皮封。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服；忌食生冷、硬物。
附注：None
