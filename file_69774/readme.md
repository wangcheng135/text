序号：69790
名称：琥珀调经丸
组成：香附子1斤（半斤童便浸，半斤好醋浸，各浸7日），好艾（择去枝梗，净者）4两（加入香附子内搅匀，再加好醋5碗，入砂锅内煮干为度，日中晒干，磨为细末），没药2两，当归2两（酒洗），川芎2两，熟地黄2两（酒蒸，另杵，入糊），生地黄2两（酒浸，另杵，入糊），芍药2两（煨），琥珀1两（另研）。
出处：《摄生众妙方》卷十一。
主治：妇人无子。
加减：None
功效：调经种子。
用法用量：每服100丸，空心艾醋汤送下。
制备方法：上为细末，用醋糊为丸，如梧酮子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中没药用量原缺，据《医学入门》补。
