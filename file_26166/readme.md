序号：26174
名称：朱砂膏
组成：腊月猪脂5两，黄蜡半斤（以上洗，煎），铅丹4两，自然铜4两（研），密陀僧4两（研），朱砂（研）。
出处：《普济方》卷三○九。
主治：攧扑刀伤。
加减：None
功效：接骨。
用法用量：凡伤碎骨者，大石压碎骨者，先用此药化开，然后夹定，用此药服之；须作小丸，如梧桐子大，每服10丸，葱酒送下；或伤损深者，捻成条子入孔中，浅者用油单纸贴之，甚者灯心草夹之；如药力散，再觉病痛，更1服痛即止；又痛甚者，贴之即止。
制备方法：上用新铛鼎先溶脂，次下蜡停冷，再下密陀僧、铅丹、朱砂、自然铜，缓火仍煎，入水中不散，取出鼎于冷处，用柳蓖搅匀，泻入瓷器内，不住手搅至凝，为丸如弹子大，且用笋皮之类衬之，极冷方收。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
