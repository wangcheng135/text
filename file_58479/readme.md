序号：58494
名称：秘方化滞丸
组成：南木香（坚实者，不见火）2钱半，丁香（去苞，不见火）2钱半，青皮（四花者，去瓤）2钱半，红橘皮（水温去白）2钱半，黄连（大者）2钱半，京三棱（慢火煨）4钱8分，莪术（慢火煨）4钱8分，夏曲（拣白净半夏为末，生姜自然汁和为饼，晒干）2钱5分（前8味晒干，和研为细末），巴豆（去壳，滚汤泡，逐一研开，去心膜，以瓦器盛，用好醋浸过1宿，慢火熬至醋干，称6钱，重为细末，将前药末和再研令匀，入后乌梅肉膏）4钱5分，乌梅（用肉厚者，打碎对核，细锉，火焙干，为细末，称5钱重，用米醋调略清，慢火熬成膏，和入前药）。
出处：《丹溪心法附余》卷三。
主治：停食饱闷，食泻，霍乱，呕吐，痢疾，气痛，小儿疳积。一切气滞积痛。
加减：None
功效：磨滞，理气，化积，通塞，调阴阳。
用法用量：每服5-7丸，人盛者10丸，五更空心用橘皮汤送下；不欲通泄，津液咽下；停食饱闷，枳壳汤送下；但有所积物，取本汁冷下；因食吐不止，津液咽下即止；食泄不休及霍乱、呕吐，俱用冷水送下；赤痢，冷甘草汤送下；白痢，冷干姜汤送下；心动，石菖蒲汤送下；赤白痢，冷甘草干姜汤送下；诸气痛，生姜橘皮汤送下；小肠气痛，茴香酒送下；妇人血气，当归汤送下；若欲宣积，滚姜汤送下；疳积常服，米饮送下，不拘时候。利多饮冷水一口补住，此药得热则行，得冷则止。小儿量岁数加减丸服。
制备方法：上药统和匀了，用白面8钱，重水调得所，慢火调糊为丸，如粟米大。
临床应用：None
各家论述：None
用药禁忌：孕妇勿服。
附注：化滞丸（《医统》卷二十三）。
