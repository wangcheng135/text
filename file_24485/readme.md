序号：24493
名称：防风汤
组成：防风（去叉）1两半，柴胡（去苗）1两半，白芷1两半，桔梗（炒）1两半，木通（锉）1两半，当归（切，焙）1两半，羌活（去芦头）1两，麻黄（去根节，煎，掠去沫，焙）1两，附子（炮裂，去皮脐）1两，甘草（炙，锉）1两。
出处：《圣济总录》卷一二九。
主治：风毒中人，留血脉不散，与荣卫相搏，结成风疽。身体烦热，昏冒肿痛。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，煎至8分，去滓温服，食后、临卧各1次。如欲出汗，空心并2服，后投以热姜稀粥，盖复取汗，慎外风。
制备方法：如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
