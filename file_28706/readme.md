序号：28714
名称：八痞丸
组成：桂心3分，曾青（无，代以空青）3分，牡丹3分，鳣头甲（头渍，炙令黄色）3分，干姜3分，蜀漆7分，细辛6分，龙胆5分，附子4分（炮）。
出处：《医心方》卷二十五引《产经》。
主治：小儿痞，面黄羸瘦，丁奚不欲饮食，食不生肌肤，心中嘈嘈烦闷，发时寒热，五脏胪胀，腹中绕脐痛，常苦下。
加减：None
功效：None
用法用量：每服2丸，1日3次。
制备方法：上药治下筛，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌猪肉、鱼、生菜等物。
附注：拓痞丸（《普济方》卷三九三）。方中鳣头甲，《幼幼新书》引《婴孺方》作“猬头”，并有“虻虫”。
