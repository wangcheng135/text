序号：35739
名称：卷柏丸
组成：黄耆（去芦，蜜水炙）1两半，熟地黄（洗）1两半，卷柏（醋炙）1两，赤石脂（煅，醋淬7次）1两，鹿茸（醋炙）1两，白石脂1两，芎1两，代赭石（煅，醋淬7次）1两，艾叶（醋炒）1两，桑寄生1两，鳖甲（醋炙）1两，当归（去芦，酒洗，微炒）1两，地榆1两，木香（不见火）半两，龙骨半两，干姜（炮）3分。
出处：《济生》卷六。
主治：妇人室女，腹脏冷热相攻，心腹绞痛，腰痛腿痛，赤白带下，面色痿黄，四肢羸乏。
加减：None
功效：None
用法用量：每服70丸，空心食前米饮汤送下。
制备方法：上为末，醋煮糯米糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
