序号：30906
名称：十宝丸
组成：破故纸（酒浸1宿，焙干）1两，附子（炮，去皮脐）1两，苍术（锉，泔浸1宿，焙干）1两，当归（去芦，焙）1两，石枣半两（去核），枸杞子（焙）半两，菟丝子（酒浸，焙干）半两，肉苁蓉（酒浸，焙干）半两，白茯苓（去皮）半两，地黄（去芦，拣肥壮者，酒浸，蒸，焙干，如此9次，透黑为度，焙干，妙处全在此1味）2两。
出处：《瑞竹堂方》卷一。
主治：None
加减：None
功效：补益肝、脾、肾三经。
用法用量：每服30-50丸，空心用温酒或盐汤送下，干物压之。
制备方法：上为细末，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
