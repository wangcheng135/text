序号：69744
名称：琥珀丸
组成：琥珀1两半，人参1两半，独活1两半，黄耆3两（酒炙），防风1两半，羚羊角1两半，赤芍1两半，川芎8钱，麦冬3两（去心），炙草1两半。
出处：《医略六书》卷三十。
主治：产后中风烦闷，脉浮软涩者。
加减：None
功效：None
用法用量：每服3钱，鲜姜汤煎，去滓温服。
制备方法：上为末，竹沥为丸。
临床应用：None
各家论述：产后血滞气亏，中风邪而遏热不解，故烦闷倦怠，四肢不能自收持焉。明琥珀安神散瘀，羚羊角清热熄风，人参扶元补气以御邪，黄耆益卫补中以托邪、独活开经气，防风泄风邪，赤芍泻血滞以调营，川芎活血气以调卫，麦冬清心润风燥，炙草缓中益胃气也。竹沥捣丸，姜汤煎服，使气行血活，则风热两解而经脉清和，胸中大气廓然，安有烦闷不痊，肢倦不瘳乎！
用药禁忌：None
附注：None
