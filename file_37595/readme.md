序号：37605
名称：三生丸
组成：半夏2两，南星1两，白附子1两。
出处：《本事》卷三。
主治：痰饮内伏，头晕目眩，呕吐酸水；中风痰多，口眼斜，半身不遂；痰厥头痛。中脘风涎痰饮，眩瞑，呕吐酸水，头疼恶心。痰厥头痛。中风昏迷，痰涎壅并，口眼斜，半身不遂，脉沉无热者。小儿暑风，暑毒入心，痰塞心孔，昏迷抽搦。
加减：None
功效：None
用法用量：每服10丸至20丸，生姜汤送下。
制备方法：上并生为末，滴水为丸，如梧桐子大，以生面滚衣，阴干。
临床应用：None
各家论述：《本事方释义》：半夏气味辛温入足阳阴，天南星气味苦辛温入手足太阴，白附子气味辛甘大温入足阳明。三味皆生用而以姜汤送者，以脘中之痰饮窃据为患，致瞑眩呕吐，头疼恶心，非峻利之药，不能扫除也。
用药禁忌：None
附注：None
