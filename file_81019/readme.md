序号：81035
名称：霹雳散
组成：熟附子、人参、甘草、白术、干姜、细茶1撮。
出处：《伤寒全生集》卷四。
主治：阴极发烦躁，阴极似阳，身热面赤，烦躁不能饮水，脉沉细或伏绝。
加减：烦躁欲坐卧泥水井中者，阴盛故也，加辰砂、远志、茯神；面赤者，下虚故也，加葱白9茎；身热者，里寒故也，加桂枝；泻不止，加炒白术、人参；呕吐不止，加姜汁、半夏、陈皮、丁香；腹痛，加砂仁、吴茱萸、木香，甚不止，加乳、没；虚寒气逆上，加沉香、苏子；有痰，加半夏、橘红；无脉，加猪胆汁1匙调服。
功效：None
用法用量：水煎，入蜜2匙，麝香少许调，顿冷服下。须臾汗出，得睡躁止乃愈。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：药后如不得睡无汗，复加烦躁不安，身热下利不止，脉不出者，死。
