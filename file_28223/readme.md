序号：28231
名称：苁蓉丸
组成：肉苁蓉（酒浸1宿，切作片子，焙干）2两，磁石3两（烧醋淬7遍，研），鹿茸（酥炙，去毛）1两，桂（去粗皮）1两，熟干地黄（焙）1两，巴戟天（去心）1两，附子（炮裂，去皮脐）半两，远志（去心）半两，地骨皮半两，黄耆（锉）1两，牛膝（酒浸1宿，切，焙）2两，五味子（炒）1两，白茯苓（去黑皮）1两，晚蚕蛾（炒）半两。
出处：《圣济总录》卷九十一。
主治：虚劳肾气不足，腹内拘急，目暗耳鸣，四肢困倦，行步乏力，脚如石沉，肌瘦羸弱，面色萎黄，脐下紧痛，心忪盗汗，小便滑数。
加减：None
功效：None
用法用量：每服20丸，空心食前温酒送下。
制备方法：上药除磁石外，为细末，同研匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
