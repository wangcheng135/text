序号：59059
名称：秦椒剪红丸
组成：秦椒、大黄、三棱、莪术、干漆、木香、槟榔、贯众、雄黄。
出处：《成方便读》卷四。
主治：虫膈、血膈。虫膈与血膈两端，皆饮食不下，皆有痛处，但虫则作止无常，攻动不一，或饥则盛，而稍得食则缓，血则口中自觉有血腥气，时作呃逆；或因大怒而血逆于上；或因受伤而血郁于中。
加减：None
功效：None
用法用量：五更时用鸡汤送下。
制备方法：上为末，神曲糊为丸。
临床应用：None
各家论述：然虫膈最易裹血，血膈每易生虫，故出一方而两治之。方中行气破血之药，居其大半，而以杀虫之药辅之，用神曲糊丸者，助脾而复其健运之职。送服用鸡汤者，以甘美之味而易入虫口也。
用药禁忌：None
附注：None
