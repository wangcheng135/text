序号：43866
名称：厚朴汤
组成：厚朴（去粗皮，生姜汁炙，锉）1两，白术1两，鳖甲（去裙襕，醋炙）1两，柴胡（去苗）1两，石斛（去根）1两，肉豆蔻（去壳）1两，地骨皮1两，犀角（镑）1两，白茯苓（去黑皮）1两，人参1两，甘草（炙，锉）1两，青木香半两。
出处：《圣济总录》卷八十七。
主治：气劳。心腹胀满，吃食不得，胸膈烦闷，面色萎黄，身体无力，不能行履。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，大枣2个（擘破），生姜1枣大（拍碎），煎至7分，去滓，食前温服，1日2次。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：厚朴散（《普济方》卷二二九）。
