序号：29802
名称：十全大补汤
组成：人参2钱，白术1钱5分，白茯苓3钱，当归2钱，川芎1钱5分，白芍2钱，熟地黄3钱，黄耆2钱，肉桂5分，麦门冬2钱，五味子3分，甘草（炙）8分。
出处：《寿世保元》卷四。
主治：元气素弱，或因起居失宜，或因用心太过，或因饮食劳倦，致遗精白浊，盗汗自汗，或内热晡热，潮热发热，或口干作渴。喉痛舌裂，或胸乳膨胀，或胁肋作痛，或头颈时痛，或眩晕眼花，或心神不宁，寤而不寐，或小便赤淋，茎中作痛，或便溺余沥，脐腹阴冷，或形容不充，肢体畏寒，或鼻气急促，或更有一切热症，皆是无根虚火。
加减：None
功效：None
用法用量：加生姜、枣子，水煎，温服。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
