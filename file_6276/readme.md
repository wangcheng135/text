序号：6276
名称：去腐灵药
组成：水银1两，火消2两，食盐3钱，枯矾3钱（3味炒燥），朱砂8钱，雄黄3钱，白砒3钱，硼砂3钱（一加硇砂3钱）。
出处：《外科大成》卷一。
主治：None
加减：发背未破，加花粉；已破、加乳香、没药；疔疮初起，加蟾酥；肿毒，加鹅管石，醋调敷；烂疮，加黑附子；囊痈烂，加贝母；瘰疬破，加发灰、皂角、白及，水调敷；疳疮，加滑石；鱼口，加皂角；结毒，加光粉、滑石；臁疮，加轻粉、黄丹；跌打，加文蛤、百草霜；乳蛾、走马疳、耳腮等，俱用茶调；蛇咬，加南星、川椒；虫咬，加雄黄。
功效：去腐。
用法用量：None
制备方法：上为末，入泥固罐内，盖盏封口，架三钉上，砌百眼炉，先底火2寸，点香1支，中火1枝，顶火1枝，随以水擦盏勿住，香毕去火，次日取升上者用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
