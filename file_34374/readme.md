序号：34383
名称：万金膏
组成：大甘草根节4两（锉，去皮），真麻油8两，黄丹4两（真好者）。
出处：《百一》卷二十。
主治：一切痈疖毒。
加减：None
功效：None
用法用量：每服50丸，甘草汤送下。
制备方法：上将甘草根节锉成寸段，捶破，内留1条长者搅药，用银、石器入油，煎甘草令焦黄，取出不用，入黄丹，以前所留长甘草1条，不住手搅，如黑色，点少许入水，试候成膏不散，用绵滤，入瓶，封令密，坎地2尺许埋药，20日取出，腊月合尤妙。敷贴如常法。发背，丸如梧桐子大。
临床应用：脑疽：沈仁父司理，年七八岁时，苦脑疽见骨，痛楚异常，沈德和尚书传此方，一夕敷之即减，不数日间，凡五换，遂痊愈。
各家论述：None
用药禁忌：None
附注：None
