序号：56644
名称：消风丸
组成：草乌头半斤（用油4两炒令黄色），麻黄（去根节，先煎，掠去沫，焙）半两，附子（炮裂，去皮脐）半两，白芷半两，防风（去叉）1两，白茯苓（去黑皮）1两，藿香叶1两，干姜（炮）1两，前胡（去芦头）1两，青橘皮（去白，炒子）1两，桂（去粗皮）1两，甘草（炙，锉）2两，龙脑（研）1分，麝香（研）1分，丁香1分，木香1分，白僵蚕（炒）1分，蝉蜕半两（炒），不蚛皂荚1两1分（去皮子，酥炙焦黄）。
出处：《圣济总录》卷六。
主治：风邪引颊，口僻，言语不正。
加减：None
功效：None
用法用量：每服1丸，温酒化下，不拘时候。
制备方法：上为细末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
