序号：3305
名称：天雄丸
组成：蛤蚧1对，朱砂2钱，沉香3钱，丁香3钱，阳起石3钱，钟乳粉半钱，木香2钱半，紫梢花半两，晚蚕蛾1两半，牡蛎粉2钱半，天雄1个，桂2钱半，石燕子1对（炭火烧，淬醋7次），鹿茸半两（酥炙），白术2钱半，苁蓉半两（酒浸3日，焙干），菟丝子3钱（酒浸，焙干），龙骨2钱半，海马1对，乳香3钱。
出处：《御药院方》卷六。
主治：真气不足，阳气衰惫，失精腰痛，脐腹痃急，及阳事不兴，男子本气脱者。
加减：None
功效：None
用法用量：每服1丸，空心细嚼，好酒煎木通，入麝香少许送下，不得过3服。
制备方法：上为细末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
