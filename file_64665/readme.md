序号：64681
名称：硇砂丸
组成：硇砂2两（细研），青盐2两（细研），生姜5斤（捣绞取汁），附子半斤（去皮脐，为末，将硇砂并青盐、附子，于姜汁内，用慢火煎令稠，取出焙干，捣罗为末，入后药），肉苁蓉2两（酒浸1宿，刮去皱皮，炙干），山茱萸2两，石斛2两（去根，锉），远志2两（去心），木香2两，巴戟2两，薯蓣2两。
出处：《圣惠》卷九十八。
主治：肾脏虚惫，腰间疼痛，小便滑数，冷气攻筑，虚损不足。
加减：None
功效：None
用法用量：每日空心以温酒送下20丸，渐加至30丸。
制备方法：上为末，与前药相和令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
