序号：35534
名称：使君子丸
组成：使君子（去皮）半两，夜明沙（炒）半两，白芜荑（炒）半两，胡黄连1两，麝香半钱（研），细辛（去苗叶）1分，芦荟（研）1分，雄黄（研）1分，蟾酥1分，槟榔（锉）1分，蝉蜕（去翅足，炒）1分，蜗牛10枚，干虾蟆2枚（用温水洗去肠肚恶物并骨爪后，慢火炙黄色为度，捣罗为末，用酒1升，同熬成膏）。
出处：《圣济总录》卷一七三。
主治：小儿疳气，身体壮热，毛发焦黄，目常有泪，满口生疮，脚手细弱，腹胁胀满，好吃泥土。
加减：None
功效：None
用法用量：每服5-7丸，米饮送下，1日3次，不拘时候。
制备方法：上除虾蟆外，为细末，入膏子和为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
