序号：84425
名称：七伤散
组成：茴香子（炒）1两，白术1两，人参1两，白茯苓（去黑皮）1两，陈橘皮（汤浸，去白）1两，芍药1两，桔梗（炒）1两，紫苑（去苗土）1两，白芷1两，苍术（去黑皮，米泔浸，切，焙）5两，柴胡（去苗）1两半，干姜（炮）2两。
出处：《圣济总录》卷八十六。
主治：脾劳腹胀，忧恚不乐，大便滑泄，不思饮食，肌肉羸瘦。
加减：None
功效：None
用法用量：每服3钱匕，用獖猪肾1对，去皮膜，批作片子，入盐1钱，与药拌匀，掺在猪肾内，湿纸裹，灰火内煨令香熟为度。细嚼，米饮下。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
