序号：43577
名称：前胡汤
组成：前胡（去芦头，锉）1两，半夏2两（以生姜自然汁1升半，浆水1升，同于银器内慢火煮，令水与姜汁尽，薄切，焙干），人参1两，木香1两（锉），厚朴（涂生姜汁炙令香熟，细锉）半两，枳壳（去瓤，麸炒）半两，旋覆花半两，陈橘皮（汤浸，去白，焙干）半两，桔梗（炒）半两，赤茯苓（去黑皮，锉）1两，白术1两，甘草3分（炙微令黄，锉）。
出处：《圣济总录》卷一五六。
主治：妊娠痰饮留滞，不思饮食。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，入生姜5片，同煎至7分，去滓温服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
