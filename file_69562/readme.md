序号：69578
名称：猴姜丸
组成：鲜猴姜数10斤（去毛洗净，亮去水气，捣烂揉汁听用），远志肉1斤2两（择肥大者，以甘草4两煎汤泡拌，晒干，加猴姜汁拌透，晒干，再拌再晒，如是数10次，候远志肉至2斤4两为度），鲜何首乌3斤（用竹刀切片，晒干，浓黑豆汁拌蒸，晒干，再拌再晒，直待首乌心内黑透为度），补骨脂1斤（以青盐1两，水拌透，炒干），石菖蒲1斤（蜜酒拌透，炒干），枸杞子1斤（蜜酒拌炒）。
出处：《惠直堂方》卷一。
主治：None
加减：None
功效：久服宁神喜睡，益记性，补下元。
用法用量：每服3钱，早、晚用盐汤送下。
制备方法：上为细末，用黑枣肉为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
