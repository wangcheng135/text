序号：78417
名称：蔷薇根皮散
组成：蔷薇根皮1两（锉），慎火草半两，白薇1分，桂心半两，败龟1两（涂酥，炙令黄），黄连1两（去须，微炒），干姜半两（炮裂，锉），细辛半两，熟干地黄1两，当归1两（锉，微炒），芎半两，石斛1两（去根，锉），白芍药半两，禹余粮2两（烧，醋淬7次），艾叶1两（微炒），牡蛎2两（烧为粉）。
出处：《圣惠》卷七十三。
主治：妇人崩中漏下赤白青黑，腐臭不可近，令人面黑，皮骨相连，月经失度，往来无常，小腹弦急，或时腹内(疒丂)痛，不欲饮食。
加减：None
功效：None
用法用量：每服2钱，食前温酒调下。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
