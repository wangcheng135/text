序号：62394
名称：旋覆花汤
组成：旋覆花1两，厚朴3两，白术3两，黄芩3两，茯苓3两，枳实3两，半夏2两，芍药2两，生姜2两。
出处：《千金》卷二。
主治：妊娠6-7月，胎不安。
加减：None
功效：None
用法用量：上(口父)咀。以水1斗煮取2升半，分5服。日3夜2，先食服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：此方专主妊娠气滞多痰，6-7月来胎息渐长，壅遏中气。故用旋覆开发痰气于上，枳、术健运脾气于中，苓、夏、姜、朴疏利滞气于下，黄芩、白芍专护胎气也。若妊娠体癯血热，中无痰湿阻碍胎气，即与此方无预也。
用药禁忌：忌羊肉、饧、醋、桃、李、雀肉等。
附注：None
