序号：58488
名称：秘方十补丸
组成：熟干地黄（净洗，酒浸，蒸过，焙干）4两，肉苁蓉（酒浸，焙干）2两，人参2两，绵黄耆（去芦，蜜使）2两，川芎2两，当归（去芦，酒浸，焙）2两，白芍药（浸）2两，白茯苓2两，白术（去芦，洗净，炒）2两，甘草（炙）半两，肉桂1两（去皮）。
出处：《普济方》卷三二二。
主治：妇人诸虚百损，荣卫不调，形体羸瘦，面黄背倦，口苦舌干，心忪多汗、血衰气盛，寒热往来，一切血崩带下，堕胎落孕。
加减：None
功效：None
用法用量：每服60-70丸，食前以米汤或温酒送下。
制备方法：上为细末，用好酒调山药未打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：十补丸（《济阴纲目》卷四）。
