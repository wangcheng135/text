序号：50021
名称：茯神丸
组成：茯神（去木）1两，五加皮（锉）1两，防风（去叉）1两，桂（去粗皮）1两，五味子1两，蛇床子（炒）1两，羌活（去芦头）1两半，鹿茸（去毛，酒炙）1两半，牛膝（酒浸，切，焙）1两半，菟丝子（酒浸，别捣）1两半，酸枣仁（炒）1两半，山茱萸1两半，巴戟天（去心）1两半，熟干地黄（切，焙）3两。
出处：《圣济总录》卷八。
主治：风，腰脚不随，行履不得，丈夫五劳七伤六极，诸风痹。
加减：None
功效：None
用法用量：每服20-30丸，温酒送下，早、晚食前各1次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
