序号：39741
名称：经验九藤酒
组成：青藤4两，钓钩藤4两，红藤（即理省藤）4两，丁公藤（又名风藤）4两，桑络藤4两，菟丝藤（即无根藤）4两，天仙藤（即青木香）4两，阴地蕨（名地茶，取根）4两，忍冬藤2两，五味子藤（俗名红内消）2两。
出处：《医学正传》卷四。
主治：远年痛风，及中风左瘫右痪，筋脉拘急，日夜作痛，叫呼不已。
加减：None
功效：None
用法用量：每服1盏，1日3次。病在上，食后及卧后服；病在下，空心食前服。
制备方法：上切细，以无灰老酒1大斗，用瓷罐1个盛酒，其药用真绵包裹，放酒中浸之，密封罐口，不可泄气，春、秋7日，冬10日，夏5日。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
