序号：53822
名称：射干散
组成：射干半两，桂（去粗皮）半两，牛膝（酒浸，切，焙）半两，牡丹皮半两，鳖甲（醋炙，去裙襕）半两，牵牛子（炒）半两，大黄（锉，炒）半两，荆芥穗1分，细辛（去苗叶）1分，半夏（为末，用生姜汁和成饼）1钱，狼毒1钱，芫花（与狼毒同用醋炒焦色）1钱。
出处：《圣济总录》卷四十一。
主治：肝脏积热，气滞血涩，或因食酸物过多，肝中血积不散，气血俱病，两胁下非时气动，每动左胁下有声，右胁相应，日渐胃脘结块，使人心腹满闷，上冲咽喉，头目不利，睡不安，如虫所啮，服泻肝汤数日后，病势不减者。
加减：None
功效：None
用法用量：每服1钱匕，食后煎葱汤调下。1-2服内如毒已下，即不用服。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
