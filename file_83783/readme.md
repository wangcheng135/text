序号：83801
名称：乌蛇丸
组成：乌蛇（酒浸，去皮骨，炙令黄）1两，玄参1两，沙参（去芦头）1两，人参（去芦头）1两，五加皮1两，防风（去芦头）1两，苦参（锉）1两，虎胫骨（涂醋，炙微黄）1两，天麻1两，败龟（涂酥，炙微黄）1两，羚羊角屑1两。
出处：《圣惠》卷二十四。
主治：大风，皮肤生疮肿瘙痒，肢节疼痛，心膈痰壅。
加减：None
功效：None
用法用量：每服30丸，食后以温水送下。
制备方法：上为末，用皂荚10挺，以水2升，揉取浓汁，去滓，煎似稀饧，和药末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中乌蛇、玄参、沙参、人参用量原缺，据《普济方》补。
