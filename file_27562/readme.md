序号：27570
名称：补经汤
组成：当归7分，鹿茸（酥炙）7分，香附（童便制）7分，白芍6分，川芎6分，熟地6分，黄耆（蜜炙）5分，白术（蜜炙）5分，白茯苓5分，黄芩（酒炒）5分，陈皮（去白）5分，砂仁5分，人参5分，阿胶（炒）5分，小茴5分，山茱萸5分，沉香2分，粉甘草2分，玄胡索5分。
出处：《叶氏女科》卷一。
主治：妇人二十五六岁，血海虚冷，经脉不调，腰腹疼痛，或下白带，或如鱼脑，或如米泔，信期不定，每月淋漓不止，面色青黄，四肢无力，头昏眼花。
加减：咳嗽潮热，加五味子、杏仁（去皮尖）各5分，竹沥少许。
功效：None
用法用量：加生姜3片，水煎，空心服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
