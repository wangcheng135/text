序号：441
名称：五柴胡饮
组成：柴胡1-3钱，当归2-3钱，熟地3-5钱，白术2-3钱，芍药钱半（炒用），炙甘草1钱，陈皮酌用或不用。
出处：《景岳全书》卷五十一。
主治：中气不足，外邪不散；伤寒，疟疾，痘疮。
加减：寒胜无火者，减芍药，加生姜3、5-7片，或炮干姜1-2钱，或再加桂枝1-2钱则更妙；脾滞者，减白术；气虚者，加人参随宜；腰痛者，加杜仲；头痛者，加川芎；劳倦伤脾、阳虚者，加升麻1钱。
功效：None
用法用量：水1钟半，煎7分，食远热服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：此与四柴胡饮相表里，但四柴胡饮止调气分，此则兼培血气以逐外邪，尤切于时用者。
