序号：64892
名称：续断汤
组成：续断2两，杜仲2两（去粗皮，锉，炒），桂2两（去粗皮），防风2两（去叉），牛膝2两（酒浸，切，焙），细辛2两（去苗叶），白茯苓2两（去黑皮），人参2两，当归2两（切，焙），白芍药2两，独活3两（去芦头），芎3两，秦艽3两（去苗土），生干地黄3两（焙），甘草（炙）3两。
出处：《圣济总录》卷十。
主治：肾气虚弱，卧冷湿地，风邪乘之，流入腰脚，冷痹疼痛。头眩瘈疭，搐掣欲作者，觉气上冲。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，煎至1盏，去滓温服。宜用蒴藋叶火燎厚安床上，及热卧上，冷即易之，冬月取根捣用。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
