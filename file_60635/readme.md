序号：60650
名称：通幽汤
组成：甘草（炙）3分，红花3分，生地5分，熟地5分，升麻1钱，桃仁泥1钱，归身1钱，麻仁3钱。
出处：《医学六要·治法汇》卷六。
主治：大便难，幽门不通，上冲，吸门不开，噎塞，不便，燥结不得下。痘症大便秘结。
加减：None
功效：以辛润之。
用法用量：临服加槟榔末半钱。
制备方法：None
临床应用：None
各家论述：《痘科类编释意》：此润燥下利之方。胃之下口曰幽门，服此通滞，故曰通幽。大肠得血则润，润则下行，亡血则燥，燥则秘结，故用熟地、当归以养血；初燥动血，久燥血瘀，故用桃仁、红花、麻仁与归尾并用，又能润燥而下行；血热，凉以生地；气热，凉以甘草；微入升麻，消风热又散抑郁。
用药禁忌：None
附注：通幽散（《痘科类编释意》卷三）。
