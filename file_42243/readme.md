序号：42258
名称：青桑枝饮
组成：青桑枝7茎并长4寸许，柳枝7茎并长4寸许，石榴皮7茎并长4寸许，桃枝7茎并长4寸许，梅枝7茎并长4寸许，鬼臼5钱，青蒿1小握，赤箭5钱（一方不用鬼臼、赤箭）。
出处：《得效》卷九。
主治：劳瘵。
加减：None
功效：取虫。
用法用量：用童便1升半，葱白7茎，去头叶，煎及一半去滓，别入安息香、阿魏各1分，再煎至1盏，去滓，调辰砂末半钱，槟榔末1分，麝香1字，分作2服调下，五更初1服，五更3点时1服。至巳时必取下虫，红者可救，青黑不治，见有所下，须进软粥饭，温暖将息。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：不可用性，及食生冷、毒物。
附注：青桑枝散（《奇效良方》卷二十二）。
