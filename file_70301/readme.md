序号：70317
名称：煮砂丹
组成：辰砂（有墙壁大块者）1两半，远志（汤泡，去心）1两半，山茱萸1两半，补骨脂（炒）1两半，石菖蒲（米泔浸1宿）1两半，石莲肉（去皮）1两半，白茯苓（去皮）1两半，柏子仁（别研）1两半，熟地黄（肥实者，水洗净，酒浸1宿，蒸5次，晒干，或焙干）1两半，穿心巴戟（去心）1两半，酸枣仁（汤煮1-2沸，去壳，炒紫色）1两半，北五味（去枝）1两半，人参（去芦）1两半，附子（炮，去皮脐）1两半，干山药3两，沉香1两（不见火）。
出处：《魏氏家藏方》卷六引王提点炳传方。
主治：None
加减：None
功效：专养心肾。
用法用量：每服50丸，食前枣汤下。
制备方法：别用獖猪心3个，灯心1两半，将朱砂用灯心裹，放猪心内，外以灯心缠之，麻线系定，于银石器内，水煮1日1夜，取出余物不用，将朱砂研极细，诸药为细末，别用法酒1升，熬沉香、山药末为膏，搜和诸药得所为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
