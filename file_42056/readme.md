序号：42071
名称：三黄宝蜡丸
组成：天竹黄3两，雄黄2两，刘寄奴3两，红芽大戟（去骨）3两，麒麟竭3两，归尾1两5钱，朱砂1两，儿茶1两，净乳香（去油）3钱，琥珀3钱，轻粉3钱，水银（同轻粉研不见星）3钱，麝香3钱（如无真天竹黄，以真胆星3两代之）。
出处：《金鉴》卷八十九。
主治：一切跌打损伤及破伤风，并伤力成痨；女人产后恶露不尽，致生怪证，瘀血奔心，痰迷心窍，危在旦夕。
加减：None
功效：None
用法用量：以上各称足分两，各为细末，再用好黄蜡24两，炼净，滚汤坐定，将药投入，不住手搅匀，取出装瓷罐内备用。重者1钱，轻者3分，用无灰酒送下，立刻全生；如被鸟枪伤，铅子在内，危在顷刻，服1钱，吃酒数杯，睡一时，汗出即愈；如外敷，将香油热化少许，鸡翎扫患处。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：服药后忌凉水、生冷、烧酒3日。如不忌此酒，则药无功。
附注：None
