序号：64605
名称：理脾和胃除湿膏
组成：党参1钱5分，生于术1钱5分，茯苓3钱，薏米3钱（生），莲肉3钱，炒谷芽2钱，陈皮1钱，香附1钱（炙），当归2钱（土炒），枸杞子2钱，白芍1钱5分（炒），生地2钱。
出处：《慈禧光绪医方选议》。
主治：None
加减：None
功效：理脾和胃。
用法用量：上药以水煎透，去滓，再熬浓汁少兑炼蜜为膏。每服2钱，白开水冲下
制备方法：None
临床应用：None
各家论述：本方虽重在理脾和胃，但寓八珍汤之意，惟因中州湿滞，故去甘草；因川芎辛温升散，光绪帝素体阴虚，故减去以防耗阴。并佐以薏米淡渗除湿之品，复加枸杞子滋补肝肾，亦属顾本之意。惟香附性虽和平，但苦燥亦能耗气，抑或因光绪帝精神不快，而以是药疏理肝气郁滞之故。综观方意,当为通补并行之方，功力和缓，宜于久服。
用药禁忌：None
附注：None
