序号：25142
名称：百灵膏
组成：生地黄5钱，熟地黄5钱，赤芍5钱，白芍5钱，白芷5钱，两头尖5钱，木鳖100粒，蓖麻100粒，巴豆50，穿山甲5片，真桐油1斤。
出处：《外科百效》卷一。
主治：恶疮及疖毒。
加减：None
功效：None
用法用量：贴恶疮及疖毒未破者用药引子，以五倍煅过为灰5分，笋箸灰4分，白丁香3分，饭为丸，如黄豆大，附于膏药中，1贴即破。
制备方法：将前药浸一宿，煎成炭浮起，用棕滤过，入炒过黄丹6两、水粉2两、百草霜2两，文武火熬成膏，滴水成珠，撚得硬时，便入血竭末5分，乳香、没药、五灵脂各2钱，搅匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
