名称：神仙训老丸
组成：何首乌（雌、雄）1斤，山茱萸4两，菟丝子4两，当归（酒洗）4两，白茯苓4两，地骨皮4两，甘州枸杞子（去核）4两，川芎4两，天门冬（去心）4两，麦门冬（去心）4两，淮生地黄4两，淮熟地黄4两，川牛膝（酒洗）4两，远志4两，甘菊花4两，山药4两，甘草（炙）4两，肉苁蓉（酒浸洗）4两，杜仲（酒炒去丝）4两，酸枣仁4两，补骨脂4两，生黑豆末4两，桑椹子4两。
