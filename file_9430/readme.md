序号：9430
名称：水蛭散
组成：水蛭80枚（炒令黄），虻虫80枚（去翅足，微炒），牛膝1两（去苗），牡丹半两，桃仁1分（汤浸，去皮尖双仁，麸炒微黄），桂心半两，菴（艹闾）子1两，当归1两（锉，微炒），鳖甲1两（涂酥，炙令黄。去裙襕），干漆1两（捣碎，炒令烟出），鬼箭羽3分，琥珀3分，吴茱萸半两（汤浸9遍，焙干，微炒），芫花半两（醋拌，炒令黄），麝香1分（研入）。
出处：《圣惠》卷七十九。
主治：产后恶血不尽，经脉日久不通，渐成癥块，脐腹胀硬，时时疼痛。
加减：None
功效：None
用法用量：每服1钱，食前以温酒下。
制备方法：上为细散，入研了药令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
