序号：56484
名称：海藻散
组成：海藻1两，大戟1两，锦纹大黄1两，续随子（去壳）1两（上锉碎，用好酒2盏，净碗内浸1宿，取出晒干后用），白牵牛（头末，生用）1两，桂府滑石半两，甘遂（麸炒黄）1两，肉豆蔻1个，青皮（去白）半两，陈皮（去白）半两。
出处：《卫生宝鉴》卷十四。
主治：男子、妇人遍身虚肿，喘，满闷不快。
加减：None
功效：None
用法用量：大人每服2钱，气实者3钱，平明冷茶清调下。至辰时取下水2-3行，肿减5-7分，隔2-3日平明又1服。小儿肿服1钱，5岁以下者半钱。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：妇人有孕不可服。忌食咸鱼、肉100日。
附注：None
