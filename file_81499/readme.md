序号：81515
名称：麝香散
组成：牛蒡子1两1分（微炒），犀角屑1两，细辛（去叶土）1两，黑参1两，麻黄（去根节）1两，白花蛇（醋浸1宿，炙去皮骨，取肉）1两，丹参1两，沙参1两，独活（去芦头）1两，人参（去芦头）1两，藁本（去土）1两，羌活（去芦头）1两，防风（去芦头）1两，甘菊花1两，全蝎（去毒1两，炒）1两，白僵蚕（炒，去丝嘴）1两，海桐皮3分，败龟（醋炙）3分，天竺黄半两，天麻（去苗）半两，赤箭牛膝（酒浸1宿，焙干）半两，白附子（炮）半两，石斛（去根）半两，阿胶（蛤粉炒成珠子）半两，虎骨（酥涂，炙黄）半两，槐实半两，半夏（汤洗去滑）半两，槟榔3个，酸枣仁（汤泡去皮）1分，肉桂（去粗皮）1分，朱砂（别研）1分，麝香（别研）1分，龙脑（别研）1分。
出处：《杨氏家藏方》卷一。
主治：一切风疾，筋脉缓纵，四肢軃曳，口眼斜，足膝不能步履，腰背不能屈伸，及头目旋晕，瘙痒顽麻。
加减：None
功效：None
用法用量：每服1钱，食前空腹用温酒调下。
制备方法：上为细末，次入研者药，和匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
