序号：5100
名称：木通膏
组成：木通1分，甘草1分，当归1分，白芷1分，防风1分（去芦头），细辛1分，栀子仁1分，黄连1分，垂柳枝（生，锉）3合，黄芩3分，黄丹6两，黄蜡2两，清麻油1斤。
出处：《圣惠》卷六十三。
主治：发背痈疽，热毒气结肿，疼痛坚硬。
加减：None
功效：None
用法用量：每使时，看肿结处大小，火畔熁，摊于故帛上贴，1日换2次。
制备方法：上锉细，于油内浸3宿，入净铛内，以慢火熬令柳枝黄黑色为度，绵滤去滓，澄清，却于铛内，慢火熬药油，相次下黄丹，用柳木篦不住手搅令匀，滴于水内，捻看硬软所得，入黄蜡，又搅令匀，倾于不津器内盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
