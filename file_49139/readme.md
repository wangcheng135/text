序号：49154
名称：胜金丹
组成：香附1斤（4制：童便、酒、盐、醋浸，春7、夏3、秋8、冬10日，炒），人参1两5钱，白薇4两（去芦），赤芍1两5钱，白芍1两5钱，当归1两5钱，白芷1两5钱，川芎1两5钱，熟地4两5钱，藁本3两，白茯苓2两5钱，丹皮2两5钱，牛膝2两5钱，杜仲2两5钱，甘草7钱5分。
出处：《惠直堂方》卷一。
主治：胎前产后，月经不调，淋浊，赤白带下，血枯不孕，小产难产，血晕血瘀，停胞死胎。
加减：None
功效：和经益精，补诸虚，种子。
用法用量：每服1丸，酒送下。如妇人行经腹痛，于经前5日服之，不过3日即愈；如素未受孕，服药数月即孕。
制备方法：上药俱用好酒浸，春5、夏3、秋7、冬10日，淘洗净，晒干为末；再用白石脂1两，赤石脂1两，醋浸3日，煅红、醋淬7次，烘干，研末，入前药末和匀；再用乳香、没药各1两，朱砂、琥珀各5钱，将4味用好酒研成膏，和前药炼蜜为丸，如弹子大，金箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
