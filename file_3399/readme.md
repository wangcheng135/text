序号：3399
名称：太乙灵丹
组成：丹参3两，赤小豆3两，鬼箭羽3两，红芽大戟2两，锦纹大黄2两，生香附1两，金银花1两，文蛤壳1两，滑石（飞）1两，法半夏1两5钱，桔梗1两5钱，雌黄1两5钱，山慈姑1两5钱，茅术1两5钱，紫苏叶1两5钱，新会皮1两5钱，广藿香1两5钱，千金霜7钱5分，明雄黄7钱5分，川乌（制）7钱5分，广木香7钱5分，山豆根7钱5分，生麻黄7钱5分，升麻7钱5分，朱砂（飞）5钱，北细辛6钱，麝香1钱5分。
出处：《中国医学大辞典·补遗》。
主治：一切瘟疫。
加减：None
功效：解毒气。
用法用量：伏疫时邪初起，寒热头痛，昏迷极闷者，薄荷汤送下；藿乱吐泻，藿香汤送下；绞肠痧，阴阳水送下；赤痢，山楂(炒焦)煎汤送下；白痢，淡姜汤送下；疟疾，向东南桃枝头3个，煎汤送下；偏正头风痛，温酒送下，并磨涂两太阳穴；无名肿毒初起，温酒送下，并涂患处；中风昏倒，口眼歪斜，二便闭者，石菖蒲汤送下；胸膈痞闷，心脾有病者，淡姜汤送下；风火牙痛，酒磨涂之；筋脉拘挛，骨节疼痛，陈酒送下；妇人腹中结块，经水过期，陈酒送下；痫证，桃枝7枚，柳枝7枚，煎汤送下；猪羊痈，石菖蒲汤送下；小儿百日胎毒，温汤送下；急慢惊风，钩藤汤送下；肝胀食积，山楂(炒焦)煎汤送下。此方于诸证初起时，轻者2-3服，重者3-4服。或用开水化服亦可。
制备方法：上为末，神曲糊为丸，每丸重2钱，辰砂为衣，晒干，瓷瓶密贮，忌火焙。
临床应用：None
各家论述：None
用药禁忌：孕妇及有血证者忌之。
附注：None
