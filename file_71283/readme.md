序号：71299
名称：紫石丸
组成：紫石英1两（细研，水飞过），肉苁蓉2两（酒浸1宿，去皮，炙令干），白石英1两（细研，水飞过），磁石2两（醋淬7遍，细研，水飞过），鹿茸（去毛，酥炙干）、菟丝子2两（浸3日，晒干为末），人参1两（去芦），黄耆1两，钟乳粉2两，熟地黄2两，巴戟1两半，白茯苓1两，补骨脂1两（炒），覆盆子1两，附子2两（炮，去皮脐），杜仲1两（去皮，炒），天门冬1两（去心），当归2两，五味子1两，石斛2两，桂心1两，柏子仁1两，蛇床子1两，棘刺1两，牛膝2两（去心），续断1两，腽肭脐1两（酒浸）。
出处：《普济方》卷三十二。
主治：肾脏虚损，羸瘦，饮食不进，肌肤骨痿无力，腰脚疼痛。
加减：None
功效：补益气力，令人健。
用法用量：每服30丸，渐加至50丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
