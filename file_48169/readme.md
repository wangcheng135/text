序号：48184
名称：神应膏
组成：栝楼1枚（去皮），零陵香1分，藿香1分，芍药1分，甘草1分，黄耆1分，杏仁（去皮）1分，白芷3分，龙脑1钱，麝香（并研）1钱，黄蜡1两半，清油6两。
出处：《圣济总录》卷一三○。
主治：一切疮肿，伤损，汤火烧。
加减：None
功效：除瘢痕，退(黑干) (黑曾)。
用法用量：摩风止痛痒，用薄绢摊贴之；口疮，含化1豆大；风毒气眼睑赤烂，疼痛不可忍者，用药涂之；口面风癣，以药涂擦，热彻为度；耳鼻中肉铃，用纸捻点1豆大，1月取下，并不疼痛。
制备方法：上除龙脑、麝香外，并锉细，于腊月用油浸7日，却出药，将油炼令香熟，放冷，秤6两，却再入诸药煎令黄，用夹绢袋滤去滓，再入锅内旋旋下蜡搅匀，滴水中成珠即止，去火候温，入龙脑、麝香打匀，倾出热瓷盒内，听用。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《医方类聚》引作“神愈膏”。
