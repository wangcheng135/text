序号：70
名称：五味子酒
组成：五味子1两，防风（去叉）1两，枸杞子（用根白皮，锉）1两，牛膝1两，牡丹（去心）1落，肉苁蓉（切，炒）1两，黄芩（去黑心）1两，白术1两，丹参（去土苗，微炙）1两，当归（切，焙）1两，五加皮（锉）1两，泽泻1两，甘草（炙，挫）1两，枳壳（麸炒，去瓤）1两，桂（去粗皮）1两，厚朴（去粗皮，涂姜汁炙5遍）1两，知母（焙）1两，细辛（去须叶，轻炒）1两，白芷（炒）1两。
出处：《圣济总录》卷九。
主治：荣虚卫实，肌肉不仁，病名肉苛。
加减：None
功效：None
用法用量：每服5合，空心、午时、夜卧温服。久服渐加至7合、1升，勿令醉吐。
制备方法：上锉，如麻豆大，生绢囊盛，用好酒2斗，密封浸，经7日。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
