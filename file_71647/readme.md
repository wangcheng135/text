序号：71663
名称：寒降汤
组成：生赭石（轧细）6钱，清半夏3钱，蒌仁（炒捣）4钱，生杭芍4钱，竹茹3钱，牛蒡子（炒捣）3钱，粉甘草1钱半。
出处：《衷中参西》上册。
主治：吐血、衄血，脉洪滑而长，或上入鱼际，此因热而胃气不降也。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：吐血：一童子，年十四，陡然吐血，一昼夜不止，势甚危急，其父通医学，自设有药房，亦束手无策。时愚应其邻家延请，甫至其村，急求为诊视。其脉洪长，右部尤重按有力，知其胃气因热不降，血随逆气上升也。为拟此汤，一剂而愈，又服一剂，脉亦和平。
