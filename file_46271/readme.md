序号：46286
名称：活血驱风散
组成：当归3钱，川芎3钱，白芷3钱，华阴细辛3钱，白蒺藜（炒去刺）3钱，桃仁（浸，去皮，焙）3钱，白芍药3钱，半夏（制）3钱，块润3钱，五灵脂3钱，甘草3钱，苍术（炒）、杜仲（姜汁炙，炒去丝）、辣桂、天麻、薏苡、橘红、槟榔、厚朴（制）、枳壳（制）。
出处：《直指》卷十九。
主治：肝肾风毒，肾囊湿痒，脚下疮癣。
加减：有热证，去乳香，加黑豆，煎服。
功效：None
用法用量：每服3钱，水1盏半，加生姜1钱5分，大枣2枚，煎7分，滤清，暖热，入乳香末少许，食前服。
制备方法：上锉细。
临床应用：None
各家论述：None
用药禁忌：None
附注：治血驱风散（《普济方》卷三○一）、活血祛风散（《杂病源流犀烛》卷二十八）。
