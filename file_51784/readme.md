序号：51799
名称：钟乳泽兰丸
组成：钟乳3两，泽兰3两6铢，防风42铢，人参1两半，柏子仁1两半，麦门冬1两半，干地黄1两半，石膏1两半，石斛1两半，芎藭30铢，甘草30铢，白芷30铢，牛膝30铢，山茱萸30铢，薯蓣30铢，当归30铢，藁本30铢，细辛1两，桂心1两，芜荑半两，艾叶18铢。
出处：《千金》卷四。
主治：妇人久虚羸瘦，四肢百体烦疼，脐下结冷，不能食，面目瘀黑，忧恚不乐。冲任虚损，月水不调，脐腹（疒丂）痛，腰腿沉重，四肢倦怠，百节痠痛，心忪恍惚，面少光泽，饮食无味；下脏风冷，带下三十六疾，崩中漏下五色，子宫久冷无子，及数堕胎，或因产劳损，冲任血气虚赢，肌瘦嗜卧。
加减：None
功效：补虚损，益血气；久服补暖元脏，润泽肌肤，长发去(黑干)，除头风，令人有子。
用法用量：每服20丸，加至40丸，酒送下，日2次：
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
