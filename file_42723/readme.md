序号：42738
名称：保合太和丸
组成：白术（去芦，炒）4两，当归（酒洗）4两，茯苓（去皮）2两，白芍（酒炒）2两，人参（去芦）1两，山药1两，陈皮（带白）1两，莲肉1两，半夏（姜制）1两，枳实（麸炒）1两，神曲（炒）1两，麦芽（炒）1两，山楂（去子）1两，香附（童便炒）1两，黄连（姜汁炒）1两，龙眼（取肉）1两，白蔻（去壳）3钱，甘草（炙）5钱。
出处：《鲁府禁方》卷二。
主治：None
加减：None
功效：培元气、脾胃之亏，壮气而增力，代劳任事，助困而不倦，当寒而耐饥。
用法用量：每服60-70丸，食后、临卧米汤送下。
制备方法：上为细末，荷叶煎汤，下大米煮稀粥为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
