序号：42977
名称：保婴百中膏
组成：沥青2斤半，威灵仙1两，蓖麻子120枚（去壳，研），黄蜡2两，乳香1两（另研），没药1两（另研），真麻油（夏）2两（春秋3两，冬4两），木鳖子（去壳）28斤（切碎，研）。
出处：《古今医鉴》。
主治：小儿疳癖泻痢，咳嗽，不肯服药；及治跌扑伤损，手足肩背，并寒湿脚气，疼痛不可忍者。
加减：None
功效：None
用法用量：如落马坠车，于破伤疼痛处，火上炙热，贴透骨肉为验。连换热水数次浴之，则热血聚处即消。小儿疳癖贴患处，泻痢贴肚上，咳嗽贴背心上。
制备方法：上先将沥青同威灵仙下锅熬化，以槐柳枝搅匀，须慢慢滴入水中，不粘手、拔如金丝状方可。如硬，再旋加油少许；如软，加沥青。试得如法，却下乳香、没药末，起锅在灰上，再用柳条搅数百次，又以粗布滤膏，在水盆内拔扯如金丝，频换水浸2日，却用小铫盛顿。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
