序号：47327
名称：祛风四物汤
组成：荆芥、防风、川芎、羌活、柴胡、白芷、甘草、蔓荆子、当归、白芍、天冬、甘菊、香附、黄耆、陈皮、苍耳子、黄连、茶叶。
出处：《陈素庵妇科补解》卷三。
主治：妊娠肝脏壅热，风充入脑，头旋目晕，忽然视物不明，腮颈颐项发肿结核。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：是方荆芥、防、羌、芷、芎、蔓、苍、柴以分各经之风热，而蔓、荆、苍耳上达巅顶以止头旋，荆芥、白芷散行两颊以清风肿核，归、芍、冬、草以清热养血，菊、连明目祛障，附、陈利膈顺气兼散肝郁，耆、防则引诸药通行十二经以疏风而固表，茶叶苦寒为使，通心降火头旋目眩自止，急则治标，缓则治本矣。
用药禁忌：病愈后一切炙煿、酒、面、辛热及毒物、鲜味、烦劳皆忌。如不守禁，两目必至失明。
