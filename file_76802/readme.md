序号：76818
名称：槟榔汤
组成：槟榔（生，锉）1两，大腹皮（锉）1两，白术1两，五味子（炒）1两，枳壳（去瓤，麸炒）1两，黄耆（锉）1两，桑根白皮1两，陈橘皮（汤浸，去白，焙）1两，防己1两，木通（锉）1两，厚朴（去粗皮，生姜汁炙）1两，桂（去粗皮）1两，木香半两，大黄（湿纸裹煨）半两，人参半两。
出处：《圣济总录》卷五十四。
主治：三焦积气，渐成水病，腹胀，四肢浮肿。
加减：None
功效：宽胸膈，利小肠。
用法用量：每服3钱匕，水1盏，加生姜3片，大枣2枚（劈破），同煎至7分，去滓温服，早晨临卧服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
