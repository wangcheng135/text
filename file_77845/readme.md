序号：77861
名称：罄脾丸
组成：陈皮4两（去瓤别为末），白面1两5钱，青盐4两，南木香1两（不见火），益智仁1两，青皮1两（去白，焙干），京三棱1两（炮），蓬莪术1两（炮），粉草1两，茴香1两（拣去枝梗）。
出处：《普济方》卷二十三引《卫生家宝》。
主治：脾气虚弱，四肢倦怠，面色萎黄，饮食减少。
加减：None
功效：常服补气益脾元，实脏腑，长肌肉，驻颜色，令百病不生。
用法用量：每服30-40丸，食前盐汤温暖酒、米饮任吞下。
制备方法：上件先将青盐细研，同陈橘皮末以水调作稀糊，慢火上煎搅数沸，入白面熬成膏，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
