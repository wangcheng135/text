序号：42970
名称：保婴丸
组成：麻黄16两，连翘4两，生石膏8两，滑石8两，羌活6两，天麻4两，栀子（炒）2两，细辛4两，防风4两，独活4两，甘草4两，全蝎2两，薄荷2两，白芍2两，大黄2两，荆芥2两，黄芩2两，川芎2两，桔梗2两，菊花2两，当归2两，白术（炒）2两，砂仁1两。
出处：《北京市中药成方选集》。
主治：小儿感冒风寒，头疼发热，鼻流清涕，身烧体倦。
加减：None
功效：清热解表，发散风寒。
用法用量：每服1丸，温开水送下，1日2次。
制备方法：上为细末，每16两细末兑百草霜2两，合匀，炼蜜为丸，每丸重1钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
