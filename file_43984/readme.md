序号：43999
名称：厚肠丸
组成：人参（去芦）2两，白术（炒）2两，厚朴（去粗皮，姜制，炒）2两，丁香（不见火）2两，荜茇2两，红豆2两，诃子肉（煨）2两，附子（炮，去皮脐）2两，肉豆蔻（面裹煨）2两，神曲（炒）2两，缩砂仁2两，麦糵（炒）2两，白豆蔻2两，良姜（炒）2两，槟榔4两，胡椒4两，荜澄茄4两，白芍药4两，陈皮（去白，洗）4两，甘草（炙）4两，干姜（炮）4两，肉桂5两（去粗皮，不见火），白茯苓（去皮）1两，当归（去芦）1两。
出处：《魏氏家藏方》卷七。
主治：肠胃虚寒，不能克消水谷，大腑飧泄。
加减：None
功效：None
用法用量：每服1丸或2丸，嚼细，白汤送下，不拘时候。
制备方法：上为细末，稀饧为丸，每2两作10丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
