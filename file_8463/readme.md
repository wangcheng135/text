名称：归来轩毡道人风药
组成：红曲半斤，苍术半斤，乌药半斤，天花粉半斤，何首乌半斤，白芷半斤，陈皮半斤，蝉壳半斤，川楝子半斤，补骨脂半斤，天南星半斤（炮），苏木半斤，藁本半斤，鹭鸶藤半斤，黑牵牛半斤，僵蚕半斤，五灵脂半斤，地龙半斤，宣木瓜半斤，香附子半斤，荆芥半斤，川芎半斤，当归半斤，细辛半斤，萆薢半斤，蚕砂半斤，赤芍药半斤，防风半斤，草乌22斤，川乌5斤，乳香4两。
出处：《袖珍》卷一。
主治：男子、妇人左瘫右痪，口眼斜，八种头风，五种腰疼，筋骨缩疼，半身不遂，手足麻木，血气燥痒，疮癣疥癞，眼流冷泪，一切风疾。
加减：None
功效：None
用法用量：每服2丸，早饭后、临卧各1服，嚼碎，茶送下；酒亦得。
制备方法：上为末，面糊为丸，如黄豆大。
临床应用：None
各家论述：None
用药禁忌：孕妇勿服。
附注：None
