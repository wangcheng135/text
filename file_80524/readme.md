序号：80540
名称：藿香散
组成：厚朴（去粗皮，姜汁炙）1两，甘草（炙）1两，半夏（切作4片，姜汁浸1宿，以粟炒黄）1两，藿香叶1两，陈皮（去白）半两。
出处：《局方》卷四。
主治：胸膈痞满，腹胁胀痛，短气噎闷，咳呕痰水，噫醋吞酸，哕逆恶心；及山岚瘴气。留饮宿食不消。
加减：None
功效：温脾胃，化痰饮，消宿冷，止呕吐。
用法用量：每服2钱，水1盏，加生姜3片，大枣1个，同煎7分，去滓热服，日2-3次，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：藿香汤（《圣济总录》卷六十四）。藿香正气散（《普济方》卷二○六引《指南方》）、藿脾饮（《证治要诀类方》卷二）、藿香脾饮（《准绳·类方》卷五）。
