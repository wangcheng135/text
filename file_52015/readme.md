序号：52030
名称：大红膏
组成：当归（锉）1两，赤芍药1两（锉），天台乌药1两（锉），小油半斤（以上3味浸油7日7夜），没药1两（研为细末），乳香2两（研为细末），琥珀1两（研为细末），沥青1斤，黄丹10两。
出处：《御药院方》卷十。
主治：从高下坠，落马伤损，瘀血结滞，筋脉挛急，肌肉肿硬，疼痛不可忍者。
加减：None
功效：None
用法用量：每用热铁篦子摊在厚软纸上，贴患处。
制备方法：上药先将沥青以银石器内慢火熬，铁篦子搅化开为度，觑时月看硬软，旋旋入浸药油，硬软停当：次入另研药3味搅匀，用绵滤在净水盆内，以手持拔，如锡白色；次入黄丹，再持拔令匀，盛在瓷盒内。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
