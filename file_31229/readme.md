序号：31238
名称：沉香附子汤
组成：沉香1块，附子1只（9钱，重者，炮，去皮脐，切片子）。
出处：《魏氏家藏方》卷六。
主治：肾阳不足，寒凝气滞，水湿停留，身面浮肿，胀满气喘，胸膈痞闷，小便不利。
加减：None
功效：None
用法用量：用水1盏，以沉香于砂盆内，旋以水少许，磨沉香300匝，以余水洗下，将附子分作3服，以沉香水煎，每加生姜5片，煎至7分，去滓，食前服，以吞既济丹尤佳。
制备方法：None
临床应用：None
各家论述：《瘴疟指南》：是方用附子，乃肾经本药，加以沉香，能引上焦阳气入肾，肾中有阳气则下元暖，根本固而邪风自息矣。
用药禁忌：None
附注：沉附汤（《朱氏集验方》卷四）、二味沉附汤（《景岳全书》卷五十八引《全集》）。
