序号：54136
名称：柴胡汤
组成：柴胡（去苗）3分，黄耆（锉）1两，厚朴（去粗皮，涂生姜汁炙）3分，半夏（汤洗去滑，焙干）3分，人参半两，白茯苓（去黑皮）半两，防风（去叉）半两，细辛（去苗叶）半两，当归（切，焙）2两，麦门冬（去心，焙）2两，陈橘皮（汤浸，去白，焙）半两，甘草（炙，焙）半两，杏仁（汤浸，去皮尖双仁，别研）半两，槟榔半两。
出处：《圣济总录》卷九十。
主治：虚劳羸瘦，心虚惊悸，气乏力劣。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，加生姜1分（切碎），煎至1盏，去滓，空腹顿服，夜卧再服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
