序号：52373
名称：首乌丸
组成：何首乌（制）360g，地黄20g，牛膝（酒制）40g，桑椹清膏70g，女贞子（酒制）40g，墨旱莲清膏48g，桑叶（制）40g，黑脂麻16g，菟丝子（酒蒸）80g，金樱子清膏70g，补骨脂（盐炒）40g，豨莶草（制）80g，金银花（制）20g。
出处：《中国药典》。
主治：肝肾两虚，头晕目花，耳鸣，腰酸肢麻，须发早白，高血脂症。
加减：None
功效：补肝肾，强筋骨，乌须发。
用法用量：口服1次6g，1日2次。
制备方法：何首乌等10味为细末，用金樱子等3味清膏，加炼蜜与适量之水为丸；稍干后，再用剩余之墨旱莲清膏加炼蜜10g包衣，打光，干燥即得。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
