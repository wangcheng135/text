序号：3265
名称：九仙饼
组成：人参2钱，南木香（不见火）2钱，南星（姜汁洗7次）2钱，甘草1钱，半夏（姜洗十次）5钱，枳壳（去瓤，面炒）10钱，白矾（明净者，火枯）10钱，豆豉（煅过）10钱，厚朴（姜汁浸，炒）15钱。
出处：《医方类聚》卷一○四引《急救仙方》。
主治：反胃噎食。
加减：None
功效：None
用法用量：每服1饼，用姜1大块，切作2片，夹饼子药，用纸裹浸湿，慢火熟煨，连姜及饼子上(口父)咀，以真料平胃散调汤吞下。
制备方法：上为细末，候夜间晴时露过，以人参、厚朴煎汤，糊作饼子，小平钱大，慢火焙干。
临床应用：None
各家论述：None
用药禁忌：令病者宽心开怀，服药调理。切忌诸般生冷。
附注：九仙夺命丹（《臞仙活人方》）。
