序号：40997
名称：三清快斑红蜡丸
组成：当归（红花汁浸，焙）2两，熟地（姜制）2两，生地（酒洗）2两，人参（去芦）7钱，白茯苓8钱，犀角（镑）3钱，川芎（酒洗）1两，白术1两，荆芥穗7钱，牛蒡子（炒）5钱，牛黄1钱5分，烧人粪5钱，人中黄3钱，甘草6钱，元参6钱。
出处：《救产全书》。
主治：痘不起，起而脓不全，全而不苍厚，烦满不宁，或遍身塌痒，间有干黑者。
加减：None
功效：益血，补气，化毒。
用法用量：3岁儿1丸，1岁儿半丸，薄荷、灯心汤调下，时常以胡荽酒些须与饮更妙。
制备方法：上为极细末，炼蜜为丸，如皂角子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：快斑红蜡丸（《惠直堂方》卷四）。
