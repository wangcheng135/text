序号：3378
名称：太一神明丸
组成：雄黄4两，真珠2两，丹砂2两，藜芦1两半，附子1两半（去皮，炮），斑蝥20枚（熬），杏仁80枚（去尖皮双仁，熬），地胆2-7枚，矾石1两（烧），赤足蜈蚣2枚（炙），巴豆70枚（去皮心），鬼臼3两，特生礜石5两（烧）。
出处：《千金翼》卷二十。
主治：腹中癥瘕积聚支满，寒热鬼疰，久病咳逆吐血，蛊注，胸中结气，咽中如有物，宿食久寒。
加减：None
功效：None
用法用量：纵不知病进退，绕脐相逐，上下不定，按之挑手，心中温温如有虫者，病走皮中，相次即取1丸摩病上，急挼手下皮青，不青当白黑，若有赤，病死皮中也；若为蜂蛇所中，中恶，服1丸，1丸著疮中，若不知，更加至3丸；卒得飞尸腹中切痛，服3丸，破1丸敷疮上即愈；夜梦寤惊恐，问病临丧，服1丸，渍1丸涂之，止恶邪气不敢近人；卒中鬼魅，狂言妄语，1丸涂其脉上，1丸涂人中，即愈；蛊毒病，一宿勿食，明旦服1丸，不知，增至2丸至3丸，以知为度；癥结，宿物勿食，服4丸，但欲癥消，每服1丸，1日3次，病下如鸡子白，或下蛇虫，下后以肥肉精作羹补之。
制备方法：上药治下筛，(口父)咀礜石令如麦大，桑白皮如钱大14枚，令于铁器中熬桑白皮焦黑止，捣二千杵，纳丹砂、雄黄诸药，合捣四千杵，白蜜和为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
