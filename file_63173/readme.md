序号：63189
名称：大黑神散
组成：生地黄（干秤）半两，熟地黄（干燥秤）1两（蒸20遍如黑角色，不可沾水），甘草（炙）1两，当归（酒浸半日，干燥秤）1两1分，肉桂（去皮，不见火）1两1分，干姜（炮）1两1分，白芍药1两，真蒲黄（纸衬，铫内漫火炒）1两，极小黑豆（炒去皮）1两半，附子（炮，6钱重，去皮脐用）2钱。
出处：《医方类聚》卷二三五引《简易方》。
主治：妇人产后众疾。
加减：月内不语，衣带断，胞衣不下，血晕口干，心闷，乍寒乍热，四肢虚肿，加独活末，酒调下：恶露不尽，血气刺痛，入炒三棱，加延胡索各半钱，酒调下；小便不通，或出血，加琥珀末半钱，木通汤下；大便秘结，加火麻仁末半钱，煎桔壳汤下；水泻，加干姜末，陈皮饮下；恶痢，浓煎罂粟汤下；中风，手足抽搐，加荆芥末，煎荆芥汤下；遍身疼痛，加黄耆末，酒下；血崩，浓煎艾汤下；咳嗽微汗，加人参、白术末，姜汤下；血渴，加蒲黄、干葛汤下：米饮亦可；呕逆恶心，煎人参、陈皮汤下；鼻衄，煎茅根汤下。
功效：None
用法用量：每服2钱，产后1旬内，并以童子小便温调下，胎毙腹中，温酒下。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
