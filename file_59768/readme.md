序号：59783
名称：调经育子方
组成：当归（酒洗）1钱，川芎7分，白芍（酒炒）1钱，熟地黄（姜汁炒）7分，陈皮8分，白术（去芦）1钱，香附（酒炒）1钱，砂仁2分，丹参5分，条芩（酒炒）1钱，甘草（炙）4分。
出处：《寿世保元》卷七。
主治：None
加减：月经先期者有热，加黄连（姜汁炒）7分，倍黄芩；后期者血虚，加黄耆（蜜炙）1钱，倍芎、归；腹痛有块，加玄胡索（炒）、牡丹皮各1钱；发热，加软柴胡、地骨皮；赤白带下，加柴胡、升麻（俱酒炒）各7分，半夏（姜汁炒）、白茯苓、苍术（米泔浸）、黄柏、知母（俱酒炒）、干姜（炮）；肥盛者，痰脂满子宫，加南星、半夏、苍术、茯苓；瘦怯者血少，不能摄精，倍芎、归；经血过多，加黑姜5分，荆芥穗（炒）8分，地榆9分；经闭不通，加桃仁、红花、苏木；气盛善恼，加乌药、香附、柴胡、陈皮。
功效：调经理脾，孕育子嗣。
用法用量：水煎，空心服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
