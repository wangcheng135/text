序号：84357
名称：五参散
组成：人参1两，沙参1两，丹参1两，玄参1两，苦参1两，木通（锉）1两，蔡黎子（炒，去角）1两，乌蛇（酒浸，去皮骨）1两，干蝎（去土）1两，天麻1两，赤箭1两，何首乌（去黑皮）1两，陈橘皮（汤浸去白，焙）1两，黄芩（去黑心）半两，原蚕蛾1分。
出处：《圣济总录》卷十八。
主治：恶风。
加减：None
功效：None
用法用量：每服3钱匕，热茶调下。服及半料，即减服2钱匕。初服3日后，先觉头项脊膂身上疼痛，乃药力与病相击也，当安卧两复时许即无事，如或身上有疮肿，服药1日后便觉渐减。
制备方法：上药并生用，捣罗为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
