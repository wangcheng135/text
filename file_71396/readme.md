序号：71412
名称：硫黄丸
组成：硫黄7分，干姜7分，吴茱萸7分，人参7分，当归7分，防风7分，礜石（泥裹，烧半日）8分，乌头8分（炮），桂心6分，天雄（炮）6分，甘草（炙）6分，蜀椒（汗）5分，皂荚（炙，去皮子）5分，枳实（炙）5分，细辛4分，甘菊花4分。
出处：《外台》卷十六引《删繁方》。
主治：肝劳虚寒，眩晕健忘，咳唾痰涎，忧恚内伤，面离色，目青盲。
加减：None
功效：None
用法用量：初服20丸，加至30丸，温清酒送服，1日2次。
制备方法：上为末，白蜜和丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌猪肉、冷水、生葱、生菜、海藻、菘菜。
附注：None
