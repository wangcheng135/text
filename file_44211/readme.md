序号：44226
名称：三解散
组成：白附子（炮）3钱，防风（去芦）3钱，黄芩3钱，桔梗（去芦）3钱，人参（去芦）3钱，全蝎5钱（薄荷汁炙），南星2钱（炮），北细辛3钱（去叶，土），厚朴5钱（姜制），缩砂仁5钱，赤芍药3钱，粉草2钱，郁金3钱（皂角水煮），曲饼5钱，半夏曲5钱（以半夏为末，姜汁作饼子晒干）。
出处：《普济方》卷三七三。
主治：夹惊夹食，伤寒，丹疮，赤眼，惊风，痰热等候。
加减：大便不通，加大黄、枳壳；急惊痰壅，加南星、半夏；惊搐不止，加金银箔、薄荷，鸡冠血下；里热壅实搐盛，加大黄、雄黄、朱砂，煎麦门冬汤，入竹沥下；惊风不语，石菖蒲汤下；食积，热酒饼汤调下；大热不退，用青龙汤、解肌汤同下，谓之和解；搐不止，加全蝎2个、不蛀皂角子7粒；惊风：紫苏、钩藤汤下。
功效：解惊风、积热、伤寒。
用法用量：金银箔、薄荷汤调下。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：药性稍凉，虚者莫服。
附注：None
