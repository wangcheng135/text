序号：80167
名称：毓麟膏
组成：人参1两，桑寄生1两，蚕沙1两5钱，生地1两，杜仲1两，续断1两，阿胶1两，地榆5钱，当归2两，熟地2两，砂仁1两。
出处：《惠直堂方》卷四。
主治：妇人久惯小产；肾虚腰痛，遗精白浊；女人淋带，血枯经闭；诸疮久烂。
加减：None
功效：能保胎10月无虞。
用法用量：摊贴，如惯于3月堕者，先1个月预贴腰眼，7日1换，保过3月之期，以后半月1换，至10月满而止，万无一失；遗精淋带经闭，贴肾俞穴、下丹田；其余俱贴患处。
制备方法：上药用麻油1斤半，按季浸，桑柴熬药枯去滓，下飞过红丹12两，黄占2两成膏，离火，下紫石英（火煅，醋淬）7钱，赤石脂（煅）7钱，龙骨（煅）3钱，为末，入膏内搅匀，收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
