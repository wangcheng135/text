序号：8093
名称：生料木香匀气散
组成：丁香1两，檀香1两，木香1两，甘草（爁）4两，缩砂（去壳）2两，白豆蔻仁1两，沉香1两，藿香（去土）4两。
出处：《得效》卷三。
主治：寒疝作痛，冷心痛，气痢腹痛，宿冷不消，气滞腹痛；气逆呕血、衄血；从高坠下，或打扑伤损，腰胁、心腹作痛。
加减：None
功效：和气。
用法用量：每服2钱，水1盏半，加生姜3片，紫苏叶5片，食盐少许，煎热服，不拘时候；或为末，炒茴香、盐酒调亦可。治气滞腹痛，用紫苏汤调下，未效，用乳香、没芍汤调服；治呕血，衄血，用侧柏叶、白茅花煎汤送下；治跌打损伤，加红曲末少许，童子小便同酒调，空心热服（如无红曲，红酒亦可）。
制备方法：上锉散。
临床应用：None
各家论述：None
用药禁忌：None
附注：木香匀气散（原书卷六）。
