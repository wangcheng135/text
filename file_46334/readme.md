序号：46349
名称：活命仙丹
组成：木鳖子3个（切片），斑蝥7个（陈土炒，去头足），米1撮（炒），大黄5钱，刘寄奴5钱，茯苓5钱，麝香1分。
出处：《辨证录》卷十三。
主治：癫狗咬伤。
加减：None
功效：None
用法用量：每服3钱，黄酒调下。
制备方法：上药各为细末，和匀。
临床应用：None
各家论述：是方用木鳖、斑蝥者，以狗最畏二物也；木鳖大凉，又能泻出热毒，得大黄以迅扫之，则热毒难留；刘寄奴善能逐血，尤走水窍，佐茯苓利水更速，引毒气从小便而出也；麝香虽亦走窍，然用之不过制斑蝥、木鳖，使之以毒攻毒耳。中有妙理，非漫然而用之也。
用药禁忌：服药切忌色欲，须二月不行房，并忌发物。
附注：None
