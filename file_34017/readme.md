序号：34026
名称：皂荚丸
组成：猪牙皂角7梃（3梃炮，2梃炙，2梃生，并去皮，都一处捶破，用温水1碗浸7昼夜，每日揉1遍，日满去滓绢滤，熬至半盏如糊，入药用），母丁香49个，龙脑（研）半钱，麝香（研）半钱，漏芦（去芦头）1分，红娘子（去头翅足）1分，苏枋木节（锉）1分，木通（锉）1分，滑石1分，粳米少许。
出处：《圣济总录》卷一二七。
主治：诸瘰疬。
加减：None
功效：None
用法用量：每服10丸，1日3-4次，空心、食前用丁香水送下。服时不得见日。
制备方法：上除皂荚外，捣研为末，都入皂荚汁中，更和寒食面少许为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：此药内消，不吐不利。
