序号：72913
名称：温胃饮
组成：人参1-3钱或1两，白术（炒）1-2钱或1两，扁豆（炒）2钱，陈皮1钱或不用，干姜（炒焦）1-3钱，炙甘草1钱，当归1-2钱（滑泄者勿用）。
出处：《景岳全书》卷五十一。
主治：中寒，呕吐吞酸，泄泻，不思饮食；及妇人脏寒呕吐，胎气不安。
加减：如下寒带浊者，加破故纸1钱；如气滞或兼胸腹痛者，加藿香、丁香、木香、白豆蔻、砂仁、白芥子之属；如兼外邪及肝肾之病者，加桂枝、肉桂，甚者加柴胡；如脾气陷而身热者，加升麻5-7分；如水泛为痰而胸腹痞满者，加茯苓1-2钱；如脾胃虚极，大呕大吐不能止者，倍用参术，仍加胡椒2-3分，煎熟，徐徐服之。
功效：None
用法用量：水2钟，煎7分，食远温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
