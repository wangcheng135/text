序号：22504
名称：牡丹皮汤
组成：牡丹皮2两，芍药（锉）2两，牛膝（酒浸，切，焙）2两，生干地黄（焙）2两，柴胡（去苗）2两，附子（炮袭，去皮脐）1两，当归（切，焙）1两，芎（锉）1两，细辛（去苗叶）1两，干姜（炮）1两，白芷1两，吴茱萸（汤洗，焙干炒）1两，人参1两，陈橘皮（去白，焙）1两，虎杖1两，延胡索1两，山茱萸1两。
出处：《圣济总录》（人卫本）卷一五○。
主治：妇人血风劳气，头目昏眩，胸背拘急，心烦体热，血脉不利，肌肉枯悴。
加减：None
功效：None
用法用量：每服5钱匕，水1盏，童便半盏，同煎至1盏，去滓温服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，原书（文瑞楼本）作“牡丹汤”。
