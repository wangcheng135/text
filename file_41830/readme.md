序号：41845
名称：降生散
组成：苍术（制）2钱，枳壳1钱，桔梗1钱，陈皮1钱，杨芍1钱，白芷1钱，川芎1钱，当归1钱，肉桂5分，干姜5分，厚朴5分，半夏5分，茯苓5分，木香5分，杏仁5分，麻黄5分，甘草5分。
出处：《古今医鉴》卷十二。
主治：临产生育艰难，痛阵尚疏，三两日不生，或产母气乏羡顿，产道干涩，致令难产。
加减：None
功效：None
用法用量：每服2钱，顺流水温暖送下。若觉热闷，蜜汤调；或锉散，姜枣顺流水煎服。冬月用之，甚为的当。隆暑之时，恐难轻服，但以五苓散，用葵子、灯心煎汤调下。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：才觉腹痛，但破水后，便可服此药，即生矣，如死胎亦下。未经破水，不宜服之。
附注：None
