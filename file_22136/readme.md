序号：22144
名称：一字散
组成：天南星（醋浸3日，焙干）1两，白附子（炮）1两，天麻1两，干蝎（全者，炒）1两，沉香（锉）半两，牛黄（研）半两，乳香（研）半两，麝香（研）半两，雄黄（研）半两。
出处：《圣济总录》卷五。
主治：脾风，多汗恶风，身体怠情，四肢不举，色黄面热，腹满短气。
加减：None
功效：None
用法用量：上药除4味研外，余药捣罗为细末，后入研药1处，研令极细。每服1字匕，温酒调下。如要丸时，用炼蜜为丸，如梧桐子大。急风，豆淋酒化3丸；一切风，头目昏暗，肢体疼痛，温酒化下1丸，小儿化半丸服。

临床应用：None
各家论述：None
用药禁忌：None
附注：None
