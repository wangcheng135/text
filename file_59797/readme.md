序号：59812
名称：大黄丸
组成：大黄（锉，炒）2两，干姜（炮）半两，人参2两，丹参（去芦头）1两，沙参1两，苦参1两，防风（去叉）1两，桂（去粗皮）半两，玄参1两半，(庶虫)虫（炙焦）8枚，附子（炮裂，去皮脐）半两，白术1两，赤茯苓（去黑皮）1两，葶苈（纸上炒）半两，杏仁（去皮尖双仁，麸炒，研）1分，蜀椒（去目并闭口者，炒出汗）1分，巴豆10枚（去皮膜，研出油尽）。
出处：《圣济总录》卷一七五。
主治：小儿哺露，胁下痞坚，腹满虚胀，手足烦热，往来无时。
加减：None
功效：None
用法用量：每服2丸至3丸，米饮送下。
制备方法：上17味，捣罗15味为末，与巴豆、杏仁同研，炼蜜为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
