序号：51974
名称：除湿去瘴汤
组成：南木1钱2分，黄柏（盐炒）1钱2分，威灵仙（酒洗）8分，山楂子1钱，当归尾（酒洗）8分，香白芷1钱，萆薢1钱，汉防己8分，木瓜1钱，川独活1钱，南星（火炮）8分，牛膝（去芦，酒洗）8分，红花5分，全蝎（炒，去刺）5分，厚朴（姜汁炒）7分，秦艽（去芦，酒洗，焙）7分。
出处：《摄生众妙方》卷四。
主治：十指顽木，屈伸直强，行步蹇难。
加减：None
功效：None
用法用量：水2钟，加生姜3片，大枣1枚，煎至1钟，空心热服；次早晨服滓。如效，后复发，将此方倍秤，为末，炼蜜为丸，如弹子大。每嚼1丸，空心酒送下；白汤亦可。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：如夏时，勿宜赤足。
附注：None
