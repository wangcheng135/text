序号：70826
名称：紫金锭
组成：炉甘石8两，黄丹8两，黄连（另研）1两，朱砂1两，当归5钱，硼砂5钱，海螵蛸1钱2分5厘，白丁香1钱2分5厘，生白矾1钱2分5厘，硇砂1钱2分5厘，轻粉1钱2分5厘，贝齿1钱2分5厘，珍珠1钱2分5厘，石蟹1钱2分5厘，熊胆1钱2分5厘，乳香1钱2分5厘，没药1钱2分5厘，麝香1钱2分5厘，冰片2钱（久留恐失气味，宜临用时加入）。
出处：《中国医学大辞典》。
主治：一切眼疾，诸般翳膜，血灌瞳仁，胬肉攀睛，拳毛倒睫，积年赤瞎，暴发赤肿，白睛肿胀，沙涩难开，眊矂紧涩，怕日羞明，眵多(目蔑)泪，烂弦风痒，视物昏花，迎风流泪，目中溜火。
加减：None
功效：None
用法用量：每用少许，新汲水浸化开，鹅毛蘸点眼大眦内；又可以热水泡化洗眼，冷则更暖之，每日洗5-7次，点10余次，甚效。
制备方法：除脑、麝外，余各为末，拌合和匀，入黄连水，碾至千万余下，晒干，次入麝香（研细，罗过），又次入片脑（研细，罗过），次用黄连1斤，当归、生地黄各4两，防风、黄柏、龙胆草各2两，蕤仁5钱，冬蜜8两（另熬，酥干为度），诃子8枚，鹅梨8枚（取汁），猪胰子4两（以稻草挪洗，去膏膜，洁净无油为度，再用布包，捣烂入药），各洗净，研为末，以水浸于铜器内，春5、夏3、秋4、冬7日，滤去滓，以滓复添水，熬3次，取尽药力，用密绢绵纸重滤过，澄去砂土，慢火煎熬，以槐柳枝各49条，互换搅拌，不可住手，搅尽枝条，至如饴糖，加蜜和匀，瓷器收盛，置汤瓶上，重汤蒸炖成膏，复滤净，至滴入水中，沉下如珠，可丸为度，待数日出火毒，再熔化，加入各末和匀，杵捣为丸锭，阴干，金银箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
