序号：43243
名称：养血参茸片
组成：太子参150g，当归90g，皂矾90g，桑寄生75g，紫河车75g，炒丹皮45g，炒栀子45g，甘草45g，鹿茸片3g（研细末），生地150g，天冬90g，菟丝子90g，麦冬45g。
出处：《古今名方》引《常见病的中医治疗研究》。
主治：阴阳气血俱虚。面色(白光)白，精神倦怠，腰痠腿软，头昏耳鸣，自汗盗汗，舌质淡，苔白，脉沉细无力。可用于再生障碍性贫血有上述证候者。
加减：None
功效：益气补血，滋阴补阳。
用法用量：每服4-8片，1日2次。
制备方法：上药水煎，浓缩成膏，加阿胶90g，制成片剂，每片0.5g。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
