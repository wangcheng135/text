序号：77103
名称：黑龙散
组成：穿山甲4两，虎头骨4两，洛粉3两（炒，带黄色为度），麝香1钱，龙脑1钱，丁香1两（新瓦焙，微见火气）。
出处：《博济》卷五。
主治：伤折手脚，骨髓冲出。
加减：None
功效：补碎续筋。
用法用量：用小黄米1大合，水2盏，煎成稠粥，续入药末1字、醋1小杓子，再煎，搅匀后，于损处先著手帕子1条，摊上面覆裹，勿令透气，再著纸裹3-5重，并油单都裹角，即1伏时1度换，依前法。如无黄米，即以糯米代之。
制备方法：上先用瓦罐子1个，投虎骨在罐子内，烧令烟起，次入穿山甲同烧，烟焰起为度，取出放冷，同前药杵罗令细。
临床应用：None
各家论述：None
用药禁忌：忌冷水、湿面，不得针灸。
附注：None
