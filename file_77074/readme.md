序号：77090
名称：黑豆浸酒
组成：黑豆5合（炒令熟），熟干地黄3两，杜仲2两（去粗皮，炙，微炒），枸杞子1两，羌活1两，牛膝3两（去苗），仙灵脾2两（去粗皮，炙微黄），当归1两，石斛2两（去根），侧子2两（炮裂，去皮脐），茵芋2两，白茯苓2两，防风3分（去芦头），川椒1两半（去目及闭口者，微炒去汗），桂心1两，芎3分，白术3分，五加皮1两，酸枣仁1两（微炒）。
出处：《圣惠》卷四十四。
主治：元气衰虚，风湿腰痛牵引，流入腿胫。
加减：None
功效：None
用法用量：每于食前暖1中盏服之。
制备方法：上细锉，用生绢袋盛，以酒2斗浸，密封，经10日后开。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
