序号：50376
名称：茵芋酒
组成：茵芋1两，乌头1两，石南1两，防风1两，蜀椒1两，女萎1两，附子1两，细辛1两，独活1两，卷柏1两，桂心1两，天雄1两，秦艽1两，防己1两，踯躅2两。
出处：《千金》卷七。
主治：大风头眩重，目瞀无所见，或仆地气绝，半日乃苏，口噤不开，半身偏死，拘急痹痛，不能动摇，历节肿痛，骨中痠疼，手不得上头，足不得屈伸，不能蹑履，行欲倾破，皮中动淫淫如有虫啄，疹痒，搔之生疮，甚者狂走。蛊风，皮肤尽痛。
加减：None
功效：None
用法用量：上（口父）咀。少壮人无所熬炼，虚老人薄熬之，清酒2斗渍之，冬7日，夏3日，春、秋5日，初服1合，不知，加至2合，宁从少起，日2次。以微痹为度。
制备方法：None
临床应用：None
