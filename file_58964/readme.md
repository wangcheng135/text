序号：58979
名称：秦艽汤
组成：秦艽3两（细切）。
出处：《外台》卷三十七引《古今录验》。
主治：服石药后饮酒，热盛充满经络，心腹少胀，欲心下(疒畐)(疒畐)不消，或时聚如坚，随复消者；或发黄，小便赤，心坚痛者。伤寒心神烦躁，口干烦渴。
加减：None
功效：None
用法用量：以牛乳1大升，煮取1小升。去滓，顿服之，得利即愈。若老弱可量气力进之。热气散后，黄色纵彻皮肤，是愈候，勿怪；热散后，慄慄寒颤，若困颤，黄复出外者，是谓余热欲散也，勿厚覆，但使饥肤中少寒颤即止。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：饮食宜清冷，不得浊热。
附注：秦艽牛乳二味汤（《外台》卷四引许仁则方）、秦艽煮散（《圣济总录》卷六十）、秦艽煎（《卫生总微》卷七）。
