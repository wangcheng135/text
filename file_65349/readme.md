序号：65365
名称：女金丹
组成：元胡（醋制）7两，生白芍7两，川芎7两，茯苓（去皮）7两，黄芩7两，陈皮14两，鹿角霜15两，白芷7两，党参（去芦）5两5钱，当归14两，白薇7两，丹皮7两，白术（麸炒）7两，制没药7两，肉桂（去粗皮）7两，熟地7两，生阿胶7两，藁本7两，甘草7两，砂仁5两，香附（醋制）15两，益母草1斤4两，煅赤石脂7两。
出处：《全国中药成药处方集》（天津方）。
主治：经血不调，赶前错后，腰腿痠痛，腹痛胀满。
加减：None
功效：调经养血，顺气活瘀。
用法用量：每服1丸，白开水送下。
制备方法：上为细末，炼蜜为丸，3钱重，蜡皮或蜡纸筒封固。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
