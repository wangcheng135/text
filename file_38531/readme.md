序号：38541
名称：泄泻丸
组成：白术（土炒）4两，橘红（留白）2两，白芍（煨）2两，白茯苓（去皮）2两，莲肉（去皮心）2两，芡实（取生肉）2两。
出处：《红炉点雪》卷二。
主治：火病便溏或泻、或下迫窘痛，或脾泄、肾泄者。
加减：腹痛后重，加木香2钱；久泻加肉蔻（面裹煨熟，去油）5钱，诃子（面裹煨，取肉）5钱；若每清晨溏泻1-2次，名肾泻，加破纸（炒）5钱；若食不消化，加麦芽（炒，取末）5钱。
功效：None
用法用量：每服100丸，或50-70丸，食远，或清晨，清米汤、滚白汤俱可下。
制备方法：上为细末。用淮山药6两，另末，以荷叶煮水糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
