序号：5937
名称：加减正元丹
组成：香附1斤（同艾2两，醋浸2宿，分作4分，1分用盐水炒，1分酥炙，1分童便浸炒，1分和乳瓦上炒），当归身（酒洗）5两，川芎2两，白芍药8两（酒浸，切片，半生半炒），生地6两（酒洗），阿胶4两（蛤粉炒成珠，无则鹿角胶代之），枳壳3两（江西者良，半生半炒），艾2两（用浸香附醋打糊饼，晒干），青蒿子3两，山茱萸肉3两，银柴胡1两，五味子3两，鳖甲（醋炙如法）4两。
出处：《广笔记》卷二。
主治：妇人月经不调，无子。
加减：如经后期，去青蒿子、银柴胡、鳖甲。
功效：None
用法用量：每服4钱，空腹淡醋汤吞下。
制备方法：上为末，米醋煮山药粉糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌白莱菔。如经调后，觉经不行，恐有妊娠，即勿服。
附注：None
