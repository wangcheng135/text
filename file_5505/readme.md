序号：5505
名称：加减小柴胡汤
组成：柴胡1钱，秦艽1钱，赤芍1钱，甘草5分，陈皮1钱5分，生姜1片，桑枝2钱。
出处：《医学心悟》卷三。
主治：疟疾。
加减：热多者，加黄芩1钱；寒多者，加黑姜5分；口渴甚者，加知母1钱，贝母1钱5分；呕恶，加半夏、茯苓各1钱，砂仁7分，生姜2片；汗少者，加荆芥1钱，川芎5分；汗多者，去秦艽，减柴胡1半，加人参1钱，白术1钱5分；饮食停滞，胸膈饱闷，加麦芽、神曲、山楂、厚朴各1钱；如欲止之，加白蔻仁8分，鳖甲（醋炙）2钱，更另用止疟丹1-2丸截之；若体虚气弱，加人参、黄耆、白术各2钱，当归、茯苓各1钱；久病成疟母，加白术1钱，木香、枳实各5分，鳖甲2钱。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
