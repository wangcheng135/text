序号：57116
名称：消热定惊煎
组成：柴胡7分，升麻7分，栀子仁7分，芍药7分，子芩8分，知母8分，寒水石12分，竹叶（切）1升，甘草2分（炙），杏仁6分（去皮）。
出处：《幼幼新书》卷十一引《婴孺方》。
主治：小儿惊热，欲发痫。
加减：None
功效：None
用法用量：上以水4升7合，煮取1升半，绞去滓，纳蜜、葛汁，文武火煎，搅勿住手，至1升2合。1月内及初满月儿，1合为3服，中间进乳；出1月，1服半合；50-60日儿，1服1合；100儿亦1合；出100日，服1合半；1-2岁2合；日2夜1，冬温服之。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
