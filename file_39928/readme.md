序号：39938
名称：肥儿丸
组成：三棱1两，莪术1两，青皮（俱醋炒）1两，焦神曲（炒）1两，川黄连1两，胡黄连1两，使君子（去壳，浸透，去皮）1两，芦荟5钱，坚槟榔5钱，香附子（炒）5钱，陈皮（去白）5钱，麦芽（炒）5钱，芜荑5钱，南木香3钱。
出处：《活幼心法》卷八。
主治：疳泻已久，脾胃极虚而不可单攻者。
加减：有癖块，加阿魏（酒浸，研化，和入），干漆（炒）各7钱。
功效：None
用法用量：2岁以下，每服3分；5岁以下，取5分；空心清米饮送下，临卧白滚水下。并用六神散与此方相间服之。
制备方法：上为细末。神曲、麦芽另研为细末打糊，和前药为丸，如粟米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：芦荟肥儿丸（《专治麻痧初编》卷三引《痘疹折衷》）。
