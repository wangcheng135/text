序号：32088
名称：万应丹
组成：乌药113两，防风113两，紫苏113两，半夏113两，川芎113两，厚朴113两，香附113两，白芷113两，甘草45两，枳壳45两，青皮40两，麦芽40两，白蔻75两，朱砂80两，草果37两5钱，神曲1048两。
出处：《同寿录》卷一。
主治：中风中寒，中气中暑，口眼歪邪，牙关紧闭，不省人事；红白痢疾，水泻疟疾，霍乱吐泻，腹痛转筋，山岚瘴气，吐酸，伤食生冷，胃口停痰，胸膈胀闷，不思饮食；远方不服水土，作泻，心痛，恶心；四时感冒伤寒，头疼发热，遍身疼痛，恶寒无汗，伤风咳嗽；妇人产后昏迷，恶露不尽；小儿急慢惊风。
加减：None
功效：None
用法用量：每服3钱，小儿每服1钱，淡姜汤送下。
制备方法：上各为末，姜汁水滴为丸。
临床应用：None
各家论述：None
用药禁忌：孕妇勿服。小儿未出天花勿服。
附注：None
