序号：43123
名称：三黄解毒汤
组成：黄柏、黄芩、黄连、山栀、大黄各等分。
出处：《广嗣纪要》卷十。
主治：妊娠伤寒5-6日后，表邪悉罢，并无头疼恶寒之症，只烦躁发热大渴，小便赤，大便秘，或利下赤水，六脉沉实。
加减：随五脏脉症加减。如烦满消渴，溲便难，尺寸脉沉弦有力，是肝经本脏受病，加当归1钱半，甘草5分，倍山栀；如腹胀满，_x000D_
谵妄，脉沉缓有力，是脾经本脏受病，加枳实（炒），厚朴（姜汁炒）各1钱半、倍大黄；如烦躁心痛，掌中热而哕，尺寸脉沉数有力，是心经本脏受病，加麦冬1钱，竹茹1团，倍黄连；如喘咳胸满，尺寸脉沉涩有力，是肺经本脏受病，加葶苈（炒）1钱，桔梗5分，倍黄芩；如下重，足胫寒而逆，尺寸脉沉而石，是肾经本脏受病，加干姜5分，熟地黄1钱半，倍黄柏。
功效：None
用法用量：水煎服。
