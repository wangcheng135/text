序号：42600
名称：鱼鳔种子丸
组成：上白鱼鳔1斤（牡蛎粉炒成珠，磨细），当归（酒洗，晒干）8两，淫羊藿（去枝梗荆刺，羊油酥炙炒）8两，白莲蕊（拣净，去灰）8两，肉苁蓉（酒洗、晒干）8两，川杜仲（去皮，青盐水炒断丝）8两，菟丝子（淘净灰土，用甜酒浸1宿，又用水煮，再合酒煮成饼，晒干）8两，沙苑蒺藜（碧绿，猪腰形者佳，去灰土，分4股，青盐、人乳、老酒、童便各拌2两，微炒）8两，云苓（去皮，切片、人乳拌蒸，晒干）4两，枸杞（红色肉厚者，拣净去带）4两，牛膝（肥长者佳，去芦、切片，酒洗、晒干）6两，破故纸（拣净，青盐水炒）6两，上肉桂2两（去粗皮、切片、不可见火），大附子2个（每个重1两4钱、去脐、切4块，以甘草水浸7日，每日1换、至期用面8两裹好，放炭火中煨熟，切片、焙干）。
出处：《中国医学大辞典》。
主治：身体虚弱，酒色过度、头眩、耳鸣、目花、腰膝酸疼，四肢无力、自汗盗汗、下元虚损，梦遗精滑、阳痿；或男子精寒，肾虚阳物不举，不能久坚，元阳衰败；或女子血寒气弱，子宫久冷，赤白崩带、经水不调、久不受孕。
加减：None
功效：却病保元。
用法用量：每早取100丸，盐汤送下，晚服100丸，陈黄酒或甜酒送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：方中桂、附二味，年老并虚弱者可用，壮实者少用。
附注：None
