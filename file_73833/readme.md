序号：73849
名称：痰火方
组成：广陈皮（去白）1两，好白术2两（陈壁土炒），黑枳实1两（麦麸炒），天花粉2两，陈枳壳1两（麦麸炒），前胡2两，山楂肉1两，生甘草4钱，大半夏2两（用姜汁泡3次，1次约用姜3两，捶碎，用水1碗熬滚，入半夏炮；如此者三。共要炮1日，取起，晒干用），大黄5两（用上好锦纹大黄1斤，将好水白酒5斤，入铁锅内煮，酒干为度，晒干，切片，再入锅内微火炒黑，细细夹碎，晒极干，同前药磨为末用）。
出处：《墨宝斋集验方》卷上。
主治：男妇老幼一切痰火。
加减：春月，加白芍药2两；夏月，加黄连2两（姜汁炒，秋冬不加）。
功效：清痰降火，止嗽定喘。少年服之无痨怯吐红之患，老年服之亦无中风痰厥之忧，解日用饮食煿炙五脏六腑之毒，兼消酒积，去皮里膜外湿痰。
用法用量：每服60-70丸，不拘清晨、晚间，用启滚汤送下。
制备方法：上药共为末，用老米作羹为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：痰火神丸（年氏《集验良方》卷四）。
