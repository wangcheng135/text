序号：69771
名称：琥珀丸
组成：琥珀（研）半两，辰砂（别研）半两，沉香半两，阿胶（碎，炒）半两，肉桂（去粗皮）半两，石斛（去根）半两，附子（炮，去皮脐）半两，五味子（拣净）半两，川芎半两，牛膝（去苗，酒浸1宿）3分，当归（去苗，炒）3分，肉苁蓉（切，酒浸1宿，焙）3分，人参3分，续断3分，没药（研）3分，熟干地黄1分，木香1分。
出处：《局方》卷九（续添诸局经验秘方）。
主治：妇人或老或少产前产后病，及疗三十六种血冷，七疝八瘕，心腹刺痛，卒中瘫痪，半身不遂，八风十二痹等，手足酸疼，乳中毒结瘀血，怀胎惊动伤犯不安，死胎不出，并衣不下。
加减：None
功效：生精血，去恶血。
用法用量：每服1丸，空心暖酒调下，午、晚食前再服。若人腹胁疼痛，绕脐如刀刺，及呕逆上气筑心，痰毒不思饮食，用姜汁少许和酒服；诸痢及赤白带，血冷崩中下血，漏胎下血，用生姜与艾，锉，炒令赤色，入酒同煎数沸，去滓调服；泄泻不止，陈米饮服；涩尿诸淋，煎通草、灯心汤服；血运不知人，煎当归酒调服；上热下冷，浓煎人参汤服；遍身虚肿水气，煎赤小豆汤服；产内二毒伤寒，及中风角弓反张，身如板硬，煎麻黄汤服，使被盖出汗；月经不通，或间杂五色，频并而下，断续不止，饮食无味，肌肤瘦劣，面赤唇焦，乍寒乍热，四肢烦疼，五心燥热，黑(黑干)，遍身血斑，赤肿走疰，及血风劳伤无力，用童便入姜汁少许调服，常服以小便为妙，若恐恶心，和以半酒；如怀胎人于难月1日1服，至产下不觉疼痛。或病人服至5服、10服，日倍饮食，是药功效矣。其功不能俱载，略述急用汤使于前。
制备方法：上为细末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
