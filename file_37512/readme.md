序号：37522
名称：抱龙丸
组成：南星（汤泡，去皮脐，锉片，微炒，为末，入黑黄牛胆中，悬风处阴干，春夏5钱，秋冬6钱）3钱，白茯苓（去皮）3钱，山药3钱，天竺黄1钱5分，雄黄（水飞，另研）1钱5分，琥珀（猪胆浸1宿，火焙，研）1钱5分，麝香1分，朱砂（水飞，另研，为衣）。
出处：《扶寿精方》。
主治：小儿惊悸，痰嗽，痘疹。
加减：None
功效：None
用法用量：葱头、薄荷汤送下。痰嗽，姜汤送下；痘疹见形，白汤送下；惊不安，灯心汤送下。
制备方法：上为细末，腊月取雪，新坛盛，埋土中，合药取1碗，甘草3钱，煎汁为丸；如无雪水，新汲水亦可，和匀如芡实大，阴干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
