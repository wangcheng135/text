序号：47574
名称：神仙中品黄龙丹
组成：赤石脂10两，黄牛乳汁3升，乳香（通明者）1斤，白砂蜜1斤，甘草3两（末），白粳米3斗5升（分作5次炊药，以熟为度）。
出处：《圣济总录》卷二○○。
主治：None
加减：None
功效：延年益寿，使四气调和，经络无滞，脏腑通快，精神清爽，除一切风劳气冷大病。
用法用量：初服时，选天德日欲明时，空心焚香，面东7拜，以好酒调下1匙许。
制备方法：上将赤石脂研末，以生绢夹袋子贮之，于甘水净盆内浸半日，以手在水中揉摆，候澄下，刮入纸箱中，控干，取5两精细者入银盒内盛之，第一次，于上7日淘米7升，如炊饭相似，安药在内炊之，饭熟为度，夜中去其盒盖，露之星辰下1宿；第二次，于月望日依前法炊之，亦去其盒盖，夜露月明中1宿；第三次，于24日早晨，依前法炊之，去其盒盖，于日中晒之，要取日月星辰之气足；第四次，先将牛乳3升入大盒内，于慢炭火上逼令如鱼眼沸，下乳香候化，次入前三次炊之赤石脂末，用柳木篦搅令匀，倾于乳钵内细研，复入盒中，依前用米7升，安盒在内炊之，米熟为度；第五次，以蜜2斤入盒内，依前慢火通之如鱼眼沸，即下前药，不住手用柳木篦搅匀，入甘草末3两，同熬令带湿便住；再用米7升，安药盒在内饮之，米熟为度，取盒于新水盆内浸半日，取出于净处顿。
