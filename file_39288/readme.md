序号：39298
名称：疟疾粉闻药
组成：苍术、白芷、川芎、桂枝。
出处：《中医杂志》（1959；4：27）。
主治：疟疾。
加减：None
功效：None
用法用量：临用时每种取出等份（共1g）用普通纱布（四层）包裹成长形（约2厘米）。在患者发疟前1、2、3或半小时不等，塞入任何1个鼻孔，令患者卧床休息，闻药时间不等（5小时或1天）。若闻药时症状仍发作者，勿将药取出，待症状发作后再取出，这样同样会发生疗效。
制备方法：上为细末，各用玻璃瓶密装（以免气味挥发）。
临床应用：疟疾：李端杰，男，34岁，广东人。1955年四月发过一次疟疾，经治疗停止症状，到同年10月下旬疟疾又复发。10月31日由畹町转入路西民族医院。查体发现，上腹部在右肋下缘处有压痛，脾部位疼痛，小便成酱油色，全身发黄，主要症状有呕吐发冷，发烧头痛等。入院后按黑尿热一般治疗后，但疟疾症状每日下午8时发作；每次发作症状特别严重，镜检原虫阴性。医院给他每日注射阿的平0.3g，连注射3天，疟疾症状依然按时发作，症状毫无减轻，后改用疾疟粉闻药，11月3日午后6时用闻药疟疾粉，到8时疟疾不发作，病人逐渐恢复健康，于11月12日出院。
