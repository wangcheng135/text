名称：神效活络丹
组成：乌蛇（去皮骨）半两（酒浸），麻黄2两（一半去节），木香2两，白芷2两（去土），细辛1两半（去土），全蝎（新者）1两，赤芍药1两，当归1两半（去芦），两头尖2两（刮去皮，一半小油浸，微炒），防风2两半（去芦），川芎2两，干葛1两半，没药1两（另研），血竭1两（另研），朱砂（水飞）1两（另研），乌犀末半两（镑），地龙半两（去土），甘草半两（炒，去土），丁香（净）1两，僵蚕1两（炒），乳香1两（另研），麝香1两（另研），片脑2钱半（另研），官桂2两，草豆蔻（净）2两，羌活（净）2两，天麻2两，香附子1两，玄参2两，人参1两，牛黄2钱半（另研），藿香2两（去土），威灵仙1两半（酒浸），何首乌2两（酒浸），沉香1两，天竺黄1两（另研），川附子1两（炮裂，去皮脐）。
出处：《普济方》卷一一四引《德生堂方》。
主治：风疾。凡男子妇人卒暴中风，不省人事，倒地不能起坐，或口眼斜，语言謇涩、身体麻痹，口噤失音不语，痰涎壅盛，筋脉拘挛，手足不能屈伸，骨节疼痛不能转侧，精神昏惯；或妇人产后暗风，洗头偏头风，旋晕欲倒，或破伤风，或内外一切伤寒。
加减：None
功效：None
用法用量：早晨空心、临卧各服1丸，细嚼烂，好酒送下。若治男子妇人卒暴中风，不省人事，倒地不能起坐，诸药不效，此药2丸，好酒1大盏，化开灌下；如口眼喎斜，语言謇涩，身体麻痹，口噤失音不语，痰涎壅盛，筋脉拘挛，手足不能屈伸，骨节疼痛不能转侧，精神昏债，临卧1丸，细嚼，或烂研，好酒咽下便睡，觉有汗受汗，无汗将病手背随即舒拳，至天明起，用人扶搀行动，至早饭后依前扶搀行动，至日西再服1丸，可痊；又治妇人产后暗风，洗头偏头风，旋晕欲倒，用药1丸，细嚼，或研开，好酒咽下即愈；破伤风，用2丸细嚼，热酒送下，半时不省，再服1丸，汗出为效；治内外一切伤寒，温酒嚼1丸亦可，以汗出为验；凡人年40以上者常服，间2-3服1丸，永无诸风之证。
制备方法：上为细末，炼蜜为丸，每药1两半重，作10丸，如弹子大，用金箔为衣。
