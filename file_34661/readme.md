序号：34671
名称：鸡艾汤
组成：艾叶2两，丹参2两，当归2两，麻黄2两，人参3两，阿胶3两，甘草1两，生姜6两，大枣12个。
出处：方出《千金》卷二引《徐之才逐月养胎方》，名见《圣惠》卷七十六。
主治：妊娠二月，始阴阳踞经，有寒多坏不成，有热即萎悴，中风寒有所动摇，心满脐下悬急，腰背强痛，卒有所下，乍寒乍热。
加减：None
功效：None
用法用量：上(口父)咀。用乌雌鸡1只，宿肥者，治如食法，割头取血，纳3升酒中相和，鸡以水1斗2升先煮，取汁，去鸡纳药煎，取3升，纳血、酒并胶煎，取3升，分3次温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
