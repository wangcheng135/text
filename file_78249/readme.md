序号：78265
名称：端效丸
组成：菟丝子（酒制）1两，枸杞子1两，破故纸（酒洗，微炒）1两，韭子（炒）1两，茴香（盐炒）1两，川山甲（炮）1两，金墨（烧烟尽）1两，远志（去心）1两，连花蕊1两，红花1两，莲肉（去皮心）1两，母丁香1两，芡实子1两，牛膝（酒洗）1两，木香1两，巴戟（酒洗，去心）5钱，益智仁5钱，川楝肉5钱，青盐5钱，沉香5钱。
出处：《医统》卷八十四。
主治：元气不足，肾虚阳脱，易萎易泄，尺脉俱微。
加减：None
功效：壮阳益气，补髓添精。
用法用量：每服50丸，空心酒送下；不饮酒者，盐汤送下，食干物压之。
制备方法：上为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
