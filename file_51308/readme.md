序号：51323
名称：虻虎汤
组成：西河柳1两许，玄参3钱，知母5钱，贝母3钱，麦门冬1两许，石膏1两半，竹叶70片。
出处：方出《广笔记》卷三，名见《麻科活人》卷二。
主治：痧疹因食肉饭而成极重者。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：痧疹：贺知忍少子病痧疹，家人不知，尚以肉饭与之。仲淳适至，惊曰：此痧症之极重者，何易视之？遂以本方2剂而痧尽现，遍体皆赤；连进4剂，薄暮矣。知忍曰：儿今无恙乎？仲淳曰：痧虽出尽，烦躁不止，尚不可保。再以石膏3两、知母1两、麦门冬3两，加黄芩、黄连、黄柏各5钱，西河柳1两，竹叶200片，浓煎饮之，烦躁遂定而愈。
