序号：78300
名称：蜀漆汤
组成：蜀漆1两，郁李仁（去皮，炒）1两，甘草（炙，锉）1两，当归（切，焙）1两，柴胡（去苗）1两，黄连（去须）1两，射干1两半，大腹1两半，桑根白皮1两半，芎1两半，牵牛子（炒）1两半，天雄（炮裂，去皮脐）1两半，陈橘皮（去白，焙）3分，桂（去粗皮）3分，苍术（去皮）3分，桃仁（去皮尖双仁）20枚。
出处：《圣济总录》卷六十六。
主治：三焦咳嗽，中满气逆，面目浮肿，咯唾痰饮。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，加生姜5片，煎至8分，去滓温服，不拘时候。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
