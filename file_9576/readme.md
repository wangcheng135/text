序号：9576
名称：仙方万金丸
组成：海浮石半两（用木柴炭火烧通红，却于好醋内蘸过，如此7烧7蘸），川乌头1两（于文武火内炮制），乳香1钱（细末），没药1钱（细末），巴豆49粒（去皮，炒黄色为末）。
出处：《医方类聚》卷一七九引《经验秘方》。
主治：疔黄，脑背疽等一应恶疮。
加减：None
功效：None
用法用量：若疮2-3日，每服10丸；若疮5-6日，每服13丸。如疮在上，食后；在下，食前。如觉呕吐勿虑，若吐出药时别服。小儿大小加减，病人如觉不动，再服3丸。每服药，先饮冷酒1盏，如泻不止，冷粥押之。
制备方法：上件海浮石与川乌于臼内捣为末，与诸药和匀，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：服药后，忌冷热物。
附注：神仙金丸（《普济方》）卷二七三引《经验良方》）。
