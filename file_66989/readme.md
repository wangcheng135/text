序号：67005
名称：麻油膏
组成：生乌麻油半鸡子许（着铜器内，以细砺石磨之，使浓不能流乃止），熟艾2升，杏仁1升，黄连1两，鸡粪1升，盐1合，乱发如半碗许大。
出处：《圣惠》卷三十二。
主治：20-30年风赤、胎赤眼。
加减：None
功效：None
用法用量：每夜临卧以铜箸点如黍米大，着目眦头，甚妙。
制备方法：穿地作一坑子，其形如瓶口，外小里大，先以火烧令干，于别处开一小风孔；以上药一重重布着坑中，状如灸炷，用火烧之；却将前所磨铜器盖坑口，候烟尽，即取铜器，刮取烟脂，为极细末，纳瓷合中盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：乌麻油膏（《圣济总录》卷一○二）。
