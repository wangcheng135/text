序号：307
名称：五香丸
组成：丁香3两，藿香3两，零陵香3两，青木香3两，甘松香3两，桂心1两，白芷1两，当归1两，香附子1两，槟榔1两，麝香1铢。
出处：《千金翼》卷五。
主治：一切肿，心痛，疔肿，口中、喉中、脚底、背甲下痈肿，痔漏。
加减：None
功效：下气散毒，令身香。
用法用量：含咽令津尽，日3夜1，1日1夜用12丸。当即觉香，5日身香，10日衣被香。五香汤法：取槟榔以前随多少皆等分，以水微微火上煮1炊久，大沸定，纳麝香末1铢，勿去滓，澄清，服1升。其汤不愈，作丸含之，数以汤洗之。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌食五辛。
附注：None
