序号：66807
名称：麻子汤
组成：秋麻子3升（净择，水渍1宿），防风2两，桂心2两，生姜2两，石膏（用绵裹）2两，橘皮2两，麻黄3两，竹叶1握，葱白1握，香豉1合。
出处：《千金》卷八。
主治：大风。
加减：患大风、贼风、刺风，加独活3两。
功效：None
用法用量：上（口父）咀。先以水2斗半煮麻子，令极熟，漉去滓，取9升别煮麻黄2沸，掠去沫，纳诸药汁中，煮取3升，去滓，空腹分3次服。服讫当微汗，汗出以粉涂身，极重者不过2-3剂，轻者1-2剂愈。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：麻黄、桂枝、防风、香豉、葱白、生姜透表之药，故取以治周身四肢挛急之风；麻仁、石膏、橘皮、竹叶清里之药，故取以治精神蒙昧；此风火扰乱神明，所以神识不清，殊非本虚之谓。
用药禁忌：忌葱菜。
附注：麻子仁汤（《圣济总录》卷八）。
