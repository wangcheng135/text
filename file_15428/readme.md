序号：15435
名称：人参汤
组成：人参2两，荆芥穗2两，柴胡（去苗）2两，白术2两，鳖甲（去裙襕，醋炙）2两，酸枣仁（微炒）2两，紫菀（去土）2两，黄耆（锉）2两，厚朴（去粗皮，生姜汁炙）2两，木香1两，桂（去粗皮）1两，白茯苓（去黑皮）1两，桔梗（炒）1两，五味子（炒）1两，陈橘皮（去白，焙）1两，枳壳（去瓤，麸炒）1两，细辛（去苗叶）1两，大腹皮1两，沉香（锉）半两。
出处：《圣济总录》卷一五○。
主治：妇人血风劳气，肌瘦寒热，咳嗽，盗汗，减食。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，加生姜3片，乌梅半枚，同煎至7分，去滓温服，1日3次。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
