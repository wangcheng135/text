序号：62702
名称：清心汤
组成：萍蓬根15钱，大黄8钱，当归6钱，川芎6钱，芍药6钱，干地黄6钱，黄芩6钱，黄连6钱，沉香6钱，人参4钱，槟榔子4钱，木香4钱，细辛4钱，桂枝4钱，丁子4钱，炙甘草4钱。
出处：《产科发蒙》卷三。
主治：妇人血晕，诸般杂病；胎前产后诸疾；及金疮，打扑损伤。
加减：None
功效：None
用法用量：每服3-4钱，以麻沸汤浸须臾，绞去滓服。再用滓以水1杯半，煮取1杯，温服。
制备方法：上药除丁子、沉香、肉桂、木香不炒外，余药合为1剂，用好酒1杯，将药润湿入锅内，炒至黄色为度，取起，摊地上候冷，入前4味。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
