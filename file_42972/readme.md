序号：42987
名称：保婴稀痘神验丹
组成：麝香5厘，朱砂1钱，大蓖麻子36粒（去壳取肉，拣肥白者）。
出处：《沈氏经验方》。
主治：None
加减：None
功效：预防痘疮。
用法用量：用手指蘸药搽小儿头顶心、前心后心、两手心、两脚心、两肘弯、两膝弯、两腋窝共13处。量药均搽如钱大，俱要搽到，勿使药有余剩。如小儿头发长者，将顶心头发剃去1块，务使药贴皮肤，其力方到。搽后听其自落，不可洗去。每药1料，只搽一儿，男女一样治法。搽1次出痘数粒；次年端阳午时再搽1次，止出1-2粒；又次年至端午再搽1次，其痘永不出矣。
制备方法：先将朱砂为末，次入麝香研匀，后将蓖麻子肉加入一处，共为细末，须要端阳午时，洁诚合制。
临床应用：None
各家论述：None
用药禁忌：None
附注：稀痘丹（《痘疹会通》卷四）。每年端午即搽一次，不可间断。如周岁小儿，再于七月七日、九月九日午时依前法搽之。
