序号：63986
名称：清膈煎
组成：陈皮1钱半，贝母2-3钱（微敲破），胆星1-2钱，海石2钱，白芥子5-7分，木通2钱。
出处：《景岳全书》卷五十一。
主治：痰因火动，气壅喘满，内热烦渴。
加减：如火盛，痰不降者，加童便1小钟。如渴甚者，加天花粉1钱；如热及下焦，小水不利者，加栀子1钱半；如热在上焦，头面红赤，烦渴喜冷者，加生石膏2-3钱；如痰火上壅，而小水不利者，加泽泻1-2钱；如痰火闭结，大便不通而兼胀满者，加大黄数钱，或朴消1-2钱。
功效：None
用法用量：水1钟半，煎7分，温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：清膈饮（《会约》卷十）。
