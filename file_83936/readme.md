序号：83954
名称：云母丸
组成：云母粉、石钟乳（炼）、白石英、肉苁蓉、石膏、天门冬（去心）、人参、续断、菖蒲、茵桂、泽泻、秦艽、紫芝、五加皮、鹿茸、地肤子、薯蓣、石斛、杜仲（炙）、桑上寄生、细辛、干地黄、荆花、柏叶、赤箭、酸枣仁、五味子、牛膝、菊花、远志（去心）、萆薢、茜根（洗去土，阴干）、巴戟天、赤石脂、地黄花、枸杞、桑螵蛸、菴（艹闾）子、茯苓、天雄（炮，去皮）、山茱萸、白术、菟丝子、松实、黄耆、麦门冬（去心）、柏子仁、荠子、冬瓜子、蛇床子、决明子、菥蓂子、车前子各等分。
出处：《千金翼》卷十二引华佗方。
主治：None
加减：None
功效：延年益寿，身体轻强，耳目聪明，流通荣卫，补养五脏，调和六腑，颜色充壮，不知衰老。
用法用量：先食服10丸，可至20丸，1日3次，久服。
制备方法：上药皆用真新好者，随人多少，捣为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
