序号：23259
名称：人参煮散
组成：人参4两，青皮（去白）12两，甘草（炙）10两，干姜（炮）6两，三棱（煨，捣碎）12两，芍药1斤，丁皮6两，茯苓（去皮）半斤，苍术（去皮）半斤。
出处：《局方》卷三（续添诸局经验秘方）。
主治：脾胃不和，中脘气滞，心腹胀痛，不思饮食，宿寒留饮，停积不消。或因饮冷过度，内伤脾气，呕吐痰逆，寒热往来，或时汗出。又治肠胃冷湿，泄泻注下，水谷不分，腹中雷鸣，胁肋虚满。并疗伤寒阴盛，四肢逆冷。
加减：None
功效：None
用法用量：每服2钱，水1盏，加生姜5片，大枣3个。同煎至7分，食前、空心温服。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《观聚方要补》卷三引作“人参煮煎”。
