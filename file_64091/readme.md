序号：64107
名称：猪肚黄连丸
组成：猪肚1具（以童便1斗煮令烂，切，研如泥），黄连（去须）1两1分，柴胡（去苗）1两半，白术1两，鳖甲（醋浸，炙令黄）1两半，紫菀（去苗土）1两半，杏仁（汤浸，去皮尖双仁，生研）1两半，桂（去粗皮）1两，陈橘皮（去白皮）1两，干姜（炮裂）3分，人参1两1分，芜荑仁（炒令香）3分。
出处：《圣济总录》卷八十九。
主治：劳瘦。
加减：None
功效：None
用法用量：每服30丸，空腹温酒送下。渐加至40丸。
制备方法：上除猪肚外，捣罗为末，入猪肚，更捣1000杵，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
