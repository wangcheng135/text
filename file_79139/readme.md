序号：79155
名称：蜡蜜丸
组成：盐豉80粒（炒香），巴豆14粒（去皮心膜，出油），大豆1鸡子大（炒），黄连3方寸，芫花1方寸，消石1方寸，白蜡1鸡子黄大。
出处：《幼幼新书》卷二十九引《婴孺方》。
主治：小儿诸注下及脓血，寒热不绝。
加减：None
功效：None
用法用量：四十日儿，服黍大1丸；一百日儿，2丸；二百日，麻子大2丸；1岁，胡豆大1丸；大人下病，如大豆3丸；每日1次。肠中病下，日中药力尽，至暮不止者，复服1丸；夜半病下，鸡鸣药力尽不止者，明早复服1丸。谓下赤白也，极者不过3服。
制备方法：上为末，研合，炼蜡丸之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
