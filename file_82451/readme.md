序号：82467
名称：鹳骨丸
组成：鹳骨3寸（炙），羚羊鼻2个（炙令焦），干姜1两，麝香2分（研），蜥蜴1个（炙），斑蝥14个（去翅足、熬），鸡屎白3两（熬），巴豆5个（去心皮，熬令黑），芫青20个（去翅足，熬），藜芦1两（去芦头，熬令黄）（一方无斑蝥、鸡屎白、巴豆、芫青、藜芦。）。
出处：方出《外台》卷十三引《广济方》，名见《圣惠》卷五十六。
主治：初得遁尸及五尸，经年不愈，心腹短气。
加减：None
功效：None
用法用量：每服3丸，空腹饮送下，每日2次，稍加至6-7丸。以知为度，至吐利。
制备方法：上为末，蜜和丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻、猪肉、蒜、粘食、陈臭、芦笋。
附注：None
