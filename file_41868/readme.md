序号：41883
名称：青云独步丹
组成：赤白何首乌共1斤（黑豆3升半，煮，拌，浸何首乌1昼夜，去汁后，将豆拌首乌木甑蒸，浸5次），当归身（酒洗）3两，赤茯苓半斤（用牛乳浸过，煮干），白茯苓半斤（用人乳浸过，煮干），补骨脂（盐、酒炒）4两，甘杞子（酒浸，焙）3两，菟丝子半斤（酒浸，蒸，捣饼，焙干），怀牛膝（甘草水泡）4两，怀生地黄（酒浸，入砂仁3钱，同蒸干，为末），真没药1两5钱（去砂）。
出处：《寿世保元》卷六。
主治：None
加减：None
功效：乌须黑发，延年益寿。
用法用量：每服30丸，空心酒送下；午间，生姜汤送下；临卧，盐汤送下。
制备方法：晒干，为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌铁器。
附注：None
