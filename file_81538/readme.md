序号：81554
名称：鲤鱼汤
组成：鲤鱼1头（重2斤），白术5两，生姜3两，芍药3两，当归3两，茯苓4两。
出处：《千金》卷二。
主治：妊娠腹大，胎间有水气。妊娠小便不利，身重恶寒，起则眩晕及水肿。
加减：None
功效：None
用法用量：上(口父)咀，以水1斗2升，先煮鱼熟，澄清，取8升，纳药煎，取3升，分5服。
制备方法：None
临床应用：None
各家论述：《医略六书》：妊娠肝脾两虚，不能输化，以制其湿，故遍身浮肿，小便涩少焉。鲤鱼下气利水，橘红化气利肺，当归养肝血以营经，白芍敛肝阴以安胎，白术健脾制湿，茯苓清肺和脾。煮鱼汁入药，务使肝脾气化则湿运，气调而小水自快，何患浮肿不退，胎孕不安乎。
用药禁忌：忌桃、李、雀肉、酢物。
附注：千金鲤鱼汤（《校注妇人良方》卷十五）。①本方方名，《外台》卷三十三引作“生鱼汤”。②《三因》有陈皮少许。
