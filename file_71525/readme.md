序号：71541
名称：搜风换骨夺命丹
组成：真五加皮（洗净）1斤8两，猪牙皂角（去壳）1斤8两，茅山苍术1斤（米泔水洗，晒干），麻黄（去根节）5斤，紫背浮萍（隔水晒干蒸过）2斤，上5味，用长流水1桶，砂锅内煎至一半，再入水煎，如此3-4次，细绢滤过，煮熬成膏入后药末：川乌（黑豆煮过）1两，土木鳖子（去油）1两，角沉香（黑色者）1两，广木香（不见火）1两，真血竭（嫩红者）1两，滴乳香（炒）1两，明天麻（大者）1两，南星（醋炙3次）1两，羌活（洗）1两5钱，荆芥穗1两3钱，防风（洗去土，去芦）1两3钱，锦纹大黄（大块，滋润者）1两3钱，白僵蚕（去丝嘴足）1两2钱，川甘松（去土）7钱，苍耳子（去土，炒）7钱，白芷梢（洗）7钱，真没药（另研）7钱，白蒺藜（去刺）7钱，姜黄7钱，川当归（去芦，酒洗）7钱，皂角刺（去尖，去木）7钱，真当门子麝香3钱，大风子（去壳油）5钱，牛蒡子5钱。
出处：《疡科选粹》卷六引《良方》。
主治：大麻癞风，紫云风，白癜风，鹅掌风；一切寒湿，手足拘挛，遍身骨节痠痛，麻木不仁，走气刺痛；破伤风；诸般疔肿、痈疽恶毒；杨霉疮，已结毒或末结毒；遍身疥癞风癣脓窠等疾。
加减：None
功效：None
用法用量：大病每用1丸，小疾半丸，放入小嘴酒瓶内，入无灰煮酒1大碗，油纸封固，隔汤煮2炷线香之久，取出热服，以出臭汗为度，元气厚者，隔日1服，虚弱者，3-4日1服。如大风等症，服至20丸病愈，此后又用20丸，入有嘴坛内，入无灰好酒30碗浸之，亦封固，照前汤煮，不可泄气，每饮3酒钟，拔去病根。临服时，先用防风、紫苏、凤仙根、槐枝、荆芥、蔷蔽根各两许，煎浓汤洗浴，然后服药。
制备方法：上各为极细末，入前膏子内，搜和为丸，如弹子大，每药4丸，用水飞朱砂为衣，次用金箔1片为衣，阴干，不见火，不见日。
临床应用：None
各家论述：None
用药禁忌：忌见风，戒食猪鱼羊肉、鸡鹅、麸面、油腻、炙煿、煎炒、辛辣发毒等物。
附注：None
