序号：26801
名称：延寿获嗣酒
组成：生地12两（酒浸1宿，切片，用益智仁2两同蒸1炷香，去益智仁），覆盆子（酒浸1宿，炒）4两，山药（炒）4两，芡实（炒）4两，茯神（去木）4两，柏子仁（去油）4两，沙苑（酒浸）4两，萸肉（酒浸）4两，肉苁蓉（去甲）4两，麦冬（去心）4两，牛膝4两，鹿茸1对（酥炙）。
出处：《惠直堂方》卷一引青诚霍氏家传方。
主治：素性弱，不耐风寒劳役，或思虑太过，致耗气血，或半身不遂，手足痿痹；或精元虚冷，久而不孕，及孕而多女，或频堕胎。
加减：有力者，加人参4两更妙。
功效：补真阴，添精益髓，乌须明目，聪耳延年。
用法用量：每晚男女各饮4-5杯，勿令醉。至百日后，健旺无比。
制备方法：上药用烧酒50斤，无灰酒20斤，白酒10斤，龙眼肉半斤，核桃肉半斤，同入缸内，重汤煮7柱香，埋土7日，取起勿令泄气。
临床应用：None
各家论述：None
用药禁忌：忌房事月余。
附注：None
