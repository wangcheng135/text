序号：29425
名称：连翘散坚汤
组成：柴胡1两2钱，草龙胆（酒洗4次）1两，土瓜根（酒制）1两，黄芩（酒炒2次）7钱，当归梢5钱，生黄芩5钱，广茂5钱，京三棱（同广茂酒炒）5钱，连翘5钱，芍药5钱，炙甘草3钱，黄连（酒炒2次）2钱，苍术2钱。
出处：《兰室秘藏》卷下。
主治：马刀。
加减：None
功效：None
用法用量：每服100余丸；一半咀。每服5钱，水2盏，先浸多半日，煎至1盏，去滓，临卧热服，去枕仰卧，每口作10次咽之，留一口送下丸药。服毕卧如常，更以龙泉散涂之。
制备方法：上另秤一半为细末，炼蜜为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：连翘溃坚汤（《玉机微义》卷十五）。《杏苑》无土瓜根，有天花粉。
