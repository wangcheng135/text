序号：59761
名称：调经乌鸡丸
组成：白毛乌骨未炖雄鸡1只（约重1斤，以糯米喂7日，勿令食虫蚁，以绳缢死，干挦其毛，去肚内杂脏不用，纳生地黄、熟地黄、天门冬、麦门冬各2两于鸡肚内，以好酒10碗，文火煮烂，取出肚内药，将鸡连骨用桑柴火焙干，仍以前煮过之生地等药酒，又浸又焙，至鸡骨肉枯为度，研极细末），人参5钱，肉苁蓉（酒洗净）1两，炒破故纸1两，砂仁1两，当归1两，白术1两，川芎1两，丹参1两，茯苓1两，甘草（炙）1两，杜仲（盐水炒）1两，香附米（醋制）4两。
出处：《竹林女科》卷一。
主治：月经愆期。由脾胃虚弱，冲任损伤，气血不足，致经来或前或后。
加减：None
功效：None
用法用量：每服50丸，空心米汤下。
制备方法：共为细末，入鸡骨肉末和匀，酒面糊为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
