序号：38222
名称：松葱膏
组成：明净松香1-2斤，明雄黄7钱，飞东丹5钱，炒黄柏2钱，洋青黛2钱（水飞），无名异（即漆匠所用之无名子灰）2钱（炒，研极细，筛之，粗者不用），大梅片5分（另研后和匀），人中白3钱（煅），上宫粉1钱5分（炒），净轻粉5分（炒制），制铜绿5分，枯白矾1钱，孩儿茶2钱，绿豆粉5钱（晒干，和匀）。
出处：《经验各种秘方辑要》引汪光焱方。
主治：小儿头面口鼻一切干湿疳疮，及白秃疮。
加减：None
功效：None
用法用量：用时先将烂疮洗净流水；湿烂者，将棉蘸药干扑之，不必再洗。2-3日即结痂收水而愈；干烂者，用女人搽头陈香油搽；若白秃疮，先将头发剃去，洗净疮痂，再搽；亦用香油调数次亦愈。
制备方法：先将松香放入大铜锅清水内煮炸，俟一齐融化，滚浮水面时，用竹签缓缓闭去热水，速倾冷水盆中，少倾即趁热扯拔，如作米糖式，倘嫌烧手，入冷水一冰即取出，不可太冷，恐凝定扯拔不动，须多手助力，各执一团扯拔，否则易冷；冷定后，复入清水内，如前再煮，再倾再扯，若是者少则五次，多则七次，如不厌烦，扯多愈妙，末用新鲜连须全葱白3-4个洗净，稍干水气，捣烂取自然汁，去滓不用，即以葱汁缓缓将松香煮干，仍用冷水一倾，随意做成饼式，愈陈愈妙，新者亦可用。每料另称2两，配下各药，余则存留可也；上为极细末，瓷瓶紧塞，勿令泄气。
临床应用：None
各家论述：None
用药禁忌：None
