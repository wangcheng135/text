序号：2633
名称：天下乌须第一方
组成：五倍子（不拘多少，捶碎，去灰，入砂锅内炒尽烟为度，以青布巾打湿扭干，包裹，脚踹成饼，为末听用）每用1钱半，乌黑霜（即炒黄好细面4两，当归尾1两为末，白及末1两，3味搅匀）每用1分半，红铜末（不拘多少，火内烧极红，投入水碗中，取出再烧，再投；取其水内自然之末，用水淘尽，将好醋煮数沸至干，随炒黑色听用）每用1分半，明矾末1分半，青盐1分2厘，没食子2厘半，诃子2厘半（2味俱用面包，入砂锅内，将柴炭同拌炒至焦干）。
出处：《古今医鉴》卷九。
主治：None
加减：None
功效：乌须发。
用法用量：上用细茶卤调如糊，瓷器内重汤煮。洗净搽上，干了洗去。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
