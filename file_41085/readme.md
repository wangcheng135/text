序号：41100
名称：金不换膏
组成：当归1两5钱，独活1两5钱，秦艽1两5钱，苍术1两5钱，白芷1两5钱，生杜仲1两5钱，羌活1两5钱，生川乌1两5钱，干姜1两5钱，良姜1两5钱，荆芥1两5钱，防风1两5钱，生草乌1两5钱，川芎1两5钱，玄参1两5钱，生地1两5钱，甘草7钱，麻黄6钱，生山甲7钱。
出处：《全国中药成药处方集》（天津方）。
主治：受风受寒，四肢麻木，腰腿酸痛，跌打损伤，伤筋伤骨，筋骨疼痛。
加减：None
功效：散风活血，强筋壮骨。
用法用量：每大张净油1两，小张净油5钱，贴患处。
制备方法：上用香油15斤，炸枯去滓滤净，炼至滴水成珠，再入章丹90两搅匀成膏。每膏药油15斤兑肉桂面1两5钱，麝香5分，乳香面1两，没药面2两，血竭面5钱，樟脑1两，海螵蛸面（去壳）5钱，煅龙骨面6钱搅匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
