序号：9012
名称：加味五宝丹
组成：真珠（豆腐煮）3钱，琥珀3钱，钟乳石（煅，为细末，人乳浸，饭上蒸过）4钱5分，辰砂（为末，飞）2钱5分，冰片1钱，牛黄2钱5分，山慈姑2钱5分，海参2钱5分（一方加麝香5分，旧琉璃烧存性，为末2钱）。
出处：《外科大成》卷四。
主治：杨梅风毒，筋骨疼痛，破脑崩鼻，蚀阳烂嗓，肺伤口臭；及癞癣，鹅掌风、身面出红黑白斑；并小儿遗毒。
加减：伤鼻，加辛夷3钱；咽喉腐烂，加升麻3钱，桔梗2钱；玉茎腐烂，加真僵蚕3钱，牙皂2钱；上部，加白芷1钱；胸腹，加白芍药1钱；下部，加牛膝1钱；四肢，加羌活、金银花、蒲公英各2钱。服前丹作呕者，生姜5钱取汁，煮鲫鱼食之，以助胃气；二便涩滞不通者，用百草霜2钱，井花水调服即解，如补鼻柱，长阳道，前古方5味，加制胎元1具，熏生蟹脚焙末7分，服如法。
功效：None
用法用量：每丹1两，配飞罗面5钱。如下部，易真绿豆粉5钱；如中部，则飞面、豆粉各用2钱半。每服3分，每日5服，土茯苓汤调下。每日用土茯苓1斤，随症加饮，煎汤5碗，每服1碗为率。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：制胎元法：干胎元大者一具。用粉甘草、人参各五钱，水三钟，煎钟半，涂抹胎元，炙汁尽为度。次用砂坛一个，入石膏末少半坛，入胎元于中，再入石膏，以满坛为度，封口，用金粟火煨一宿，取出胎元，如乌金纸色为佳，白色者不用。补鼻柱法：用油纸，以甘草水煮过，于好人鼻子上印一塑子，合患者鼻上，外以膏药盖之，俟长完，去纸，用皮消煎汤洗鼻三二次，则见风不痛。
