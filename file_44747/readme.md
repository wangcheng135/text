序号：44762
名称：思仙续断丸
组成：思仙木（即杜仲也。去皮，锉，炒令黑）5两，五加皮3两，防风（去叉股）3两，薏苡仁3两，羌活洗去土3两，川续断（洗，锉，焙干）3两，牛膝（洗，锉，焙，酒浸1宿，再焙）3两，萆薢4两，生干地黄5两。
出处：《本事》卷四。
主治：肝肾风虚气弱，脚膝不可践地，腰脊疼痛，风毒流注下经，行止艰难，小便余沥。
加减：None
功效：补五脏内伤，调中益精，凉血，坚强筋骨，益智轻身耐老。
用法用量：每服50丸，空心、食前温酒、盐汤任下。膏子少益以酒糊。
制备方法：上为细末，好酒3升，化青盐3两，用大木瓜半斤（去皮），以盐酒煮大木瓜成膏，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《医学纲目》引作“续断丸”。
