序号：3371
名称：太一护命丸
组成：阳起石1两，硫黄1两，附子（炮裂，去皮脐）1两，金星石半两，银星石半两，白石英半两，紫石英半两，礞石半两，滑石半两，消石半两，凝水石（烧赤）半两，磁石（火烧，醋淬）半两，太阴玄精石半两，甘锅子（累经销金银者）半两，不灰木半两，丹砂1分，巴豆49粒（去皮心，出油），杏仁49粒（汤浸，去皮尖双仁）。
出处：《圣济总录》卷三十三。
主治：伤寒后变成疟疾，或阴气结伏心下成块，或膨满呕逆喘促，面黑唇青，四肢逆冷，饮食不下，气结食毒。
加减：None
功效：None
用法用量：每服1丸，麸炭火内烧赤，取出，用2只盏子合在内，候冷细研，以生姜米饮送下。未知再服。
制备方法：上为末，再同研极细，用阿魏1分，与糯米粥同和匀为丸，如樱桃大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
