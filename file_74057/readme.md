序号：74073
名称：填睛育婴丸
组成：石决明1枚（洗刷），阳起石（饭上炊五度）2两，白芷2两，白茯苓（去黑皮）2两，桂（去粗皮）2两，防风（去叉）2两，杏仁（去皮尖双仁，炒）2两，陈橘皮（浸，去白，焙）2两，栀子花2两，肉苁蓉（酒浸，去皴皮，焙）2两，生姜（切，焙）2两，甘草（炙，锉）2两，厚朴（拌生姜，炒令烟尽）2两，磁石末（饭上炊五度）2两，人参2两，青葙子3两，蕤仁（水浸）3两，升麻（锉）8两，熟干地黄（焙）8两，龙脑1分，车前子4两，黄柏（去黑皮）4两，槐子4两，麦门冬（去心，焙）4两，黄连（去须）4两，乳香4两，乌贼鱼骨（去甲）1两，黄芩（去黑心）1两，苦参1两。
出处：《圣济总录》卷一○二。
主治：肝肾气虚，风毒上攻，两眼赤痒肿痛昏涩，迎风多泪，及有胬肉，或头风内外障，青盲，攀睛翳膜。
加减：None
功效：None
用法用量：每服6丸，空心白汤送下，食后更服10丸，渐加20丸。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
