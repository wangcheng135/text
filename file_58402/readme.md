序号：58417
名称：破经丸
组成：川大黄1两，硇砂（研）半两，川芎半两，红娘子49个，马鸣退灰3钱，当归6钱（切，焙）。
出处：《普济方》卷三一九引《仁存方》。
主治：妇人干血，气久滞，腿脐腹冷痛，寒热往来，血块，血刺，痃癖，癥瘕，四肢无力，饮食渐减，久变成虚劳痰嗽。
加减：None
功效：None
用法用量：每服10丸至15丸，食前煎红黄酒送下。三两时间觉脐腹下微痛，药行大小便取恶血浓枯为效；如未觉，加至20丸，3-5服效；要内消，只服3-5丸，每日常服，视患老少轻重，以意加减。
制备方法：上煎醋1小盏，入大黄末作膏，和后5味，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
