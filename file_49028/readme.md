序号：49043
名称：胆矾丸
组成：胆矾1钱（真者，为粗末），绿矾（真者）2两，大枣14个（去核），好醋1升（上药同煎，熬令枣烂和后药），使君子2两（去壳），枳实（去瓤，炒）3两，黄连1两，诃黎勒（去核，并为粗末）1两，巴豆27枚（去皮，破之。上5味同炒令黑，约3分干，入后药），夜明砂1两，虾蟆（烧灰存性）1两，苦楝根皮（末）半两。
出处：《小儿药证直诀》卷下。
主治：小儿疳病。
加减：None
功效：消癖进食，止泻和胃，追虫。
用法用量：每服20-30丸，米饮、温水送下，不拘时候。
制备方法：后三物再同炒，候干，同前四物杵罗为末；却同前膏和，入臼中，杵千下；如未成，更旋入熟枣肉，亦不可多，恐服之难化；太稠即入温水，可丸即丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
