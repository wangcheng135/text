序号：40577
名称：虎睛丸
组成：虎睛1具（酒浸1宿，炙），防风4分，秦艽4分，防葵4分，龙齿4分，黄芩4分，雄黄4分，防己4分，山茱萸4分，茯苓4分，铁精4分，鬼臼4分，人参4分，干地黄（一方云干姜）4分，大黄4分，银屑4分，牛黄4分，独活3两，远志3两，细辛3两，贯众3两，麝香3两，白蔹（1作白薇）3两，升麻3两，白鲜皮3两，茯神5两，石膏5两，天雄5两，鬼箭羽2分，露蜂房2分，寒水石6分，蛇蜕1尺。
出处：《千金》卷十四。
主治：风癫掣疭，口眼张大，口中出白沫，或作声，或死不知人。
加减：None
功效：None
用法用量：每服15丸，稍加至25丸，酒送下，每日2次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：虎睛、龙齿定魄安魂，牛黄、麝香涤痰利窍，蜂房、蛇蜕攻毒祛风，铁精、银屑辟邪镇惊，雄黄、石膏破恶散结，防葵、天雄出阴入阳，升麻、大黄升清除浊，人参、地黄导气和血，其余风药、毒药总皆匡佐之流耳。如无虎睛，珍珠代用可也。
用药禁忌：None
附注：按：《圣济总录》有芎、飞鸦头，无雄黄、茯神。
