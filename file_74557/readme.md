序号：74573
名称：新定胆制咽喉药
组成：真小川连1两，条子芩5钱，真川柏5钱，白僵蚕（炙燥）3钱，漂人中白2两，老月石1两，薄荷叶2钱。
出处：《疡科纲要》。
主治：风火喉证及口疳舌疮。
加减：None
功效：None
用法用量：红肿腐烂者皆效；若但红肿而未腐者，此药1两，可配枯矾2钱吹之。
制备方法：各为极细末，和匀，腊月收青鱼胆，带胆汁盛药末，线扎，挂当风处阴干，去胆皮，细研，每一胆，倾去胆汁一半，乃入药末，加指甲炭2钱，明腰黄5钱，西瓜霜1两，蜒蚰制青梅肉5钱，焙燥，研，每药末1两，加上梅片1钱，和匀密收。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
