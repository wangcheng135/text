序号：25181
名称：观音救苦甘露饮
组成：观音柳1枝（5钱），滑石3两，炒谷芽3两，焦神曲3两，苍术2两（泔水浸），云茯苓2两，柴胡1两5钱，川厚朴1两（姜汁炒），黄芩1两，枳壳1两，葛根1两，苏叶1两，姜半夏1两，陈皮1两（盐水炒），芍药1两，楂肉1两，乌药1两，香附5钱，木香5钱，甘草5钱，陈茶叶2斤（安化茶或六安茶之陈者佳）。
出处：《霍乱吐泻方论》。
主治：感冒时邪，瘟疫疟疾，伏暑停食，霍乱吐泻，头痛腹胀，口渴心烦，脾胃不调，吞酸嗳腐，一切不服水土。
加减：None
功效：避邪逐恶，祛风清热，疏滞和中。
用法用量：每次3钱，用阴阳水煎服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
