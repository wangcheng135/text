序号：81888
名称：熟干地黄汤
组成：熟干地黄（焙）1两，黄耆（锉）1两，人参1两，麻黄（去根节）1两，当归（切，焙）1两，芎1两，秦艽（去土苗）1两，鳖甲（去裙襕，醋炙）1两，延胡索3分，甘草（炙，锉）3分，赤芍药3分，桂（去粗皮）3分，前胡（去芦头）3分，地骨皮3分，柴胡（去苗）3分。
出处：《圣济总录》卷一五○。
主治：妇人血风虚劳，邪气相乘，肢节疼倦，口苦舌干，不思饮食，寒热头痛，虚汗不止。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，加生姜2片，大枣、乌梅各1枚，煎至6分，去滓温服，日3次。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
