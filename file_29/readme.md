序号：29
名称：五味子汤
组成：五味子半两，当归半两，麻黄6铢，干姜6铢，桂心6铢，人参6铢，紫菀6铢，甘草6铢，细辛3铢，款冬花3铢，大黄1两半（一方无款冬、大黄，有大枣3枚）。
出处：《千金》卷五。
主治：小儿风冷入肺，上气气逆，面青喘迫，咳嗽昼夜不息，食则吐不下。
加减：None
功效：None
用法用量：上(口父)咀。以水2升半，煮取9合，去滓。儿60日至100日，1服2合半；100日至200日，1服3合。其大黄别浸1宿下。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：小儿风冷入肺咳嗽，用麻、桂、干、辛、款冬、甘、菀当矣。以其有面青、喘迫、吐逆不下，知肺胃之气大虚，非藉人参不能安其胃气；非藉白术不能止其吐逆；非藉五味不能敛其喘迫；盖小儿面青逆冷，必非伏热假象，但其咳嗽昼夜不息，必有乳癖留滞于中，若系虚嗽，火动则剧，火静则止，定属虚中挟痰之象无疑，非藉大黄不能涤其乳癖；非藉当归不能和其血气。
用药禁忌：None
附注：按：本方方名，《外台》引作“五味汤”。
