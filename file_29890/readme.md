序号：29898
名称：十味香薷饮
组成：香薷叶1两，人参（去芦）半两，白术半两，陈皮（温汤浸少时，去皮）半两，白茯苓半两，黄耆（去芦）半两，厚朴（去粗皮，锉碎，生姜自然汁拌和，炒至黑色）半两，干木瓜半两，白扁豆（炒，去壳）半两，甘草（炙）半两。
出处：《百一》卷七。
主治：暑湿内伤，脾胃不和，食少腹胀；脾胃不和，乘冒暑气，心腹膨闷，饮食无味，呕吵恶心，五心潮热，力乏体倦；伏暑，身体倦怠，神昏头重，吐利。暑疟。
加减：None
功效：祛暑解表，益气和中。消暑，健脾进饮食。养阴避暑，调理阴阳。益气和中，安胎保孕。
用法用量：每服3钱，水1盏，加大枣1个，同煮至7分，去滓，不拘时服。
制备方法：上为粗末。
临床应用：None
各家论述：《医方考》：暑能伤气，故身体倦怠，神思昏沉；暑为阳邪，故并于上而头重：暑邪干胃，故既吐且利。火热横流，肺气受病，人参、黄耆，益肺气也；肺为子，脾为母，肺虚者宜补其母，白术、茯苓、扁豆、甘草，皆补母也；火为母，土为子，火实者宜泻其子，厚朴、陈皮，平其敦阜，即泻子也；香薷之香，散暑邪而破湿热；木瓜之酸，收阴气而消脾湿。脾气调则吐利自息，肺气复则倦怠自除。
用药禁忌：None
附注：十味香薷散（《杏苑》卷三）。
