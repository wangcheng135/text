序号：39504
名称：矾石酿酒
组成：矾石（烧）1两，石膏1两，代赭1两，恒山蜀椒（去目及闭口者，汗）1两，远志（去皮）1两，狼毒1两，半夏（洗）1两，芒消1两，礜石（炼）1两，玄参1两，麻黄（去节）1两，防风1两，桔梗1两，秦艽1两，石南1两，石韦（去毛）1两，黄连1两，莽草1两，干地黄1两，凝水石1两，菟丝子1两，甘草（炙）1两，白石英1两半，杏仁20枚（去皮尖，熬）。
出处：《千金翼》卷二十一。
主治：癞病。
加减：None
功效：服百日面白如桃李色，耳目聪明，邪气荡除，去魂还复。
用法用量：每服酒如1鸡子，酒势尽，复进之，1日2次，所治无有不愈，10日知，30日愈。
制备方法：上药治下筛，盛韦囊中，以时曲3斤，米3斗作酒，酒熟，合药封之，冬10日、春7日、夏3日、秋5日，出药去滓。
临床应用：None
各家论述：None
用药禁忌：服药当斋戒。
附注：None
