序号：25492
名称：八味汤
组成：吴茱萸（汤洗7次）2两，干姜（炮）2两，木香1两，橘红1两，肉桂（去粗皮）1两，丁香1两，人参（去芦头）1两，当归（洗，焙）1两。
出处：《杨氏家藏方》卷六。
主治：脾胃虚寒，气不升降，心腹刺痛，脏腑虚滑，内寒泄泻，恶食。
加减：None
功效：温中逐寒。
用法用量：每服5钱，水2盏，煎至1盏，去滓温服，不拘时候。
制备方法：上(口父)咀。
临床应用：None
各家论述：1.《杏苑》：用吴茱萸、干姜、肉桂、丁香等诸辛热以散寒，陈皮、木香行郁，人参补气，当归益血。_x000D_
2.《景岳全书》：此汤味太刚烈，当加炙甘草方妙。_x000D_
3.《医略六书》：人参扶元补气，肉桂暖血温营，丁香温中散滞气，炮姜逐冷暖中州，当归养血，木香调气，陈皮利气开胃，吴萸温肝降逆。
用药禁忌：None
附注：人参八味汤（《医略六书》卷十九）。
