序号：77744
名称：雄黄丹
组成：雄黄2两，磁石2两，朱砂2两，硫黄2两，牛黄1两（研细），麝香半两（细研）。
出处：《鸡峰》卷二十八。
主治：None
加减：None
功效：补益筋髓，延年，驻颜色，壮志气；久服可以无疾，身轻骨健，耳目聪明。
用法用量：每日空心以温酒下5丸。
制备方法：上药前4味各于乳钵内细研，水飞过，于净垍器中贮之，欲修合时，须5月5日收采青艾嫩者约一担，择取用水净洗，木臼中烂捣，干净布中绞取汁，可5升。先埿一炉，致银锅，以慢火煎令成膏，斟酌稀稠所得，即先下磁石搅令匀，次下朱砂又搅令匀，次下雄黄又搅令匀，良久去锅下火，即下硫黄又搅令匀，次下牛黄又搅令匀，次下麝香，须细意搅，要药末匀，候可丸，即为丸如绿豆大。
临床应用：None
各家论述：None
用药禁忌：忌羊血。
附注：None
