序号：80949
名称：飞黄散
组成：曾青1两，雌黄1两，白矾石1两，磁石1两，雄黄1两，丹砂1两。
出处：《外台》卷三十引《广济方》。
主治：诸恶疮肿。
加减：None
功效：None
用法用量：None
制备方法：上药各为细末，依四方色以药置色处，曾青东方，丹砂南方，白矾石西方，磁石北方，雄黄中央，瓦甕2枚，以黄泥下再三过，使厚5-6分，以雌黄屑著下，合筛诸药着上，后以半雌黄屑覆上，以泥密涂际，勿令气泄。土须厚，一宿如常点火，点火用2年陈芦作樵中调火，以新布沉水中，覆釜上，干复易，90沸止，若日暮，77沸亦足止；太熟1斛米饭顷发出药，恶肉青黑干，不复出汗，愈。以土釜2枚，如上法也。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
