序号：34325
名称：万和散
组成：茴香（炒）1两，萝卜子（生）1两，官桂（去粗皮）1两，蓬莪术（湿纸裹，煨）1两，香白芷1两半，陈皮1两1分（去瓤），麦糵1分，荆三棱3两半（用湿纸裹，炮），干姜3分（煨），甘草1两3分（炙），白术（米泔浸1宿）半两，桔梗半两，牵牛子（炒，过熟不妨）半两。
出处：《百一》卷四。
主治：男子妇人，一切气刺气闷，气胀食伤；及中毒积滞，两胁脐下四肢攻注；宿有气疾，心腹痞塞，呕吐，不思饮食；伤风烦闷，鼻出清水，夜多盗汗，渐成瘦弱；肠滑水泻不止。
加减：None
功效：None
用法用量：每服1钱，水8分盏，煎至6分，去滓，稍热服；或入大枣煎如汤，点服亦得；妇人血气，入当归少许；心痛，炒茴香酒调下；中毒，热酒调下；小儿久泻不止，及泻后伤动胃气，不思饮食，瘦瘁，并以药1钱，大枣半个，水5分，煎4分，热服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
