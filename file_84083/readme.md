序号：84101
名称：五圣丹
组成：雄黄半两，叶子雌黄半两，朴消半两，绿矾半两，明矾半两，白矾半两。
出处：《直指》卷二十三。
主治：痔。
加减：None
功效：None
用法用量：每抄1匙，煎甘草汤调敷，外用鸡羽扫药末盖之，日2次、夜1次，自然干硬而脱。逐日须用葱白煎汤，入朴消温洗，软帛拭干，然后敷药。
制备方法：上药各别磕碎，以银窝一入雄黄，二入雌黄，三入朴消，四入绿矾，五入白矾，圆瓦片盖定，炭火煅1日夜，看青烟出尽，有红烟上方好，候冷取出，纸衬顿地上，用盆盖，出火毒1日夜，入乳香末1钱，没药末1钱，同研极细。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
