序号：22250
名称：防葵丸
组成：防葵8分，当归8分，枳实8分（炙），厚朴8分（炙），楮实8分，人参8分，黄耆8分，茯神8分，白术8分，诃黎勒皮8分，郁李仁6分（去皮），柴胡6分，大麻仁6分，芍药6分，橘皮6分，防风6分，紫菀6分（洗去土），薏苡仁6分，鳖甲12分（炙），三棱根12分，桂心7分，仙鼠2枚（如无，以粪2合代），大附子2枚（炮），干姜（末）2分，甘草10分（炙），干地黄10分，大黄10分，五味子4分，槟榔4颗，牛膝2分。
出处：方出《外台》卷三十五引刘氏方，名见《普济方》卷三九一。
主治：小儿冷癖痃癖气，不下食，羸瘦，时时肋下痛。
加减：None
功效：None
用法用量：大小增减，以意量之，须饮服之，良。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
