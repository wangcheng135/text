序号：58548
名称：秘传十六味流气饮
组成：人参、当归、官桂5分，川芎1钱，防风1钱，白芷1钱，桔梗1钱，黄耆1钱，炙草1钱，厚朴1钱，木香1钱，白芍药1钱，大腹皮1钱，乌药1钱，枳壳1钱，苏叶1钱。
出处：《疮疡经验全书》卷四。
主治：痈疽。
加减：不退热，加茯苓、白术、地黄；不进饮食，加香附、砂仁；疼痛，加乳香、没药；水不干，加知母、贝母；疮不穿，加皂角刺；大便闭，加大黄、枳壳；咳嗽，加陈皮、枳壳、半夏、杏仁、生姜；小便闭，加麦门冬、车前子、木通、滑石、灯草；瘰疬，加羌活、夏枯草、连翘、青皮、柴胡、黄芩。
功效：发散风毒，调理气血，排脓止痛，长肉生肌。
用法用量：每服6钱，以酒调下，服药后饮酒以助药力。不饮酒者，木香汤代之，米饮亦可。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中人参、当归用量原缺。
