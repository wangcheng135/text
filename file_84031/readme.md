序号：84049
名称：五加皮药酒
组成：当归4两，青风藤4两，海风藤4两，川芎4两，灵仙4两，木瓜4两，白术（麸炒）6两，白芷6两，怀牛膝8两，红花8两，菊花8两，五加皮1斤，广皮1斤，党参（去芦）1斤8两，色姜黄1斤8两，独活2两，制川乌2两，制草乌2两，肉豆蔻（滑石煨）3两，檀香4两，蔻仁3两，公丁香2两，砂仁2两，广木香2两，肉桂（去粗皮）2两，玉竹4斤。
出处：《全国中药成药处方集》(天津方)。
主治：风湿痿痹，手足拘挛，四肢麻木，腰膝疼痛，肾囊湿冷。
加减：None
功效：舒筋活血，除湿散风。
用法用量：每次1两，1日3次，温服。
制备方法：先将白酒400斤放入缸内，用栀子3斤泡过，适合为度，去滓。以上各药用水熬汁，去滓滤净，炼成稀膏状，以稀膏化冰糖40斤兑入酒内。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
