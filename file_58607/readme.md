序号：58622
名称：秘传当归膏
组成：当归（酒洗）1斤6两，生地黄1斤（酒洗），熟地黄3两（酒洗），薏苡仁8两（米粉同炒），白芍药1斤（粉炒），白茯苓12两，白术1斤，莲子4两（去心），山药5两，人参4两（加洗用），甘草3两，枸杞子1斤4两（甘州者佳），贝母3两（去心），地骨皮4两，麦门冬5两（去心），天门冬2两（去心），五味子1两，琥珀1钱2分。
出处：《直指》卷九。
主治：五劳七伤，诸虚劳极，脾胃虚弱。
加减：吐血，加牡丹皮2两；骨蒸，加青蒿汁2碗，童便2碗；劳痰，加钟乳粉1两。
功效：养血和中，滋荣筋骨，养阴抑阳。
用法用量：None
制备方法：上各锉细，和足，以水10斤，微火煎之，再加水10斤，如此4次，滤其滓，取汁，文武火煎之，如法为度，每斤加炼熟净蜜4两，春5两，夏6两，共熬成膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：当归膏（《医学入门》卷七）。
