序号：22334
名称：防风雄黄丸
组成：赤芍药8两，防风4两（去芦头），香白芷4两，川乌头4两（炮，去皮脐尖），麻黄4两（去根节），白蒺藜（炒）4两，雄黄2两（水飞），白僵蚕2两（炒，去丝嘴），细辛2两（去叶土），天麻2两（去苗），川芎2两，甘草1两（炙），干姜1两（炮），藿香叶1两（去土），甘松（去土，焙）1两。
出处：《杨氏家藏方》卷一。
主治：左瘫右痪，手足麻痹，腰膝疼痛。或风气面浮，日苦舌干，头昏目运，并暗风、夹脑风、偏正头痛；兼治妇人血气、荣气虚，遍身疼痛，及洗头风，破伤风。
加减：None
功效：None
用法用量：每服1丸，细嚼，茶、酒任下，不拘时候。
制备方法：上为细末，炼蜜为丸，每1两作15丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
