序号：775
名称：六岁墨
组成：山慈菇1两，千金子1两，大戟1两，文蛤2两（去虫），麝香1分，川乌2两，草乌2两。
出处：《外科十三方考》。
主治：疔疮肿毒，口眼歪斜，牙关紧急；及山岚瘴气，死牛、死马、河脉中毒，砒毒，咽喉肿痛。
加减：None
功效：解毒止痛。
用法用量：每服1锭，病重者可连服2锭。通利之后，用温粥补之。凡疔疮肿毒、口眼歪斜、牙关紧急等症，俱用温酒磨服；其他一切疮毒，皆用醋磨搽。
制备方法：上为细末，以糯米煮糊捣匀，用模型铸为1钱重墨状条块，阴干备用。
临床应用：None
各家论述：None
用药禁忌：None
附注：原书按：太岁墨即太乙紫金锭之变方，以二乌、朱砂、雄黄，化和平为峻险，专作外用，不重内服，反不若紫金锭之安全稳妥。故在用本品处，皆代以紫金锭，其收效颇能如理想也。
