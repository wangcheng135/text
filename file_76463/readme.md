序号：76479
名称：干地黄丸
组成：干地黄6分，黄耆6分，防风4分，远志4分，茯神4分，栝楼4分，子芩4分，鹿茸（炙）3分，龙骨4分（五色者），人参5分，滑石12分，石苇（汤渍1宿，刮去皮）2分，当归2分，芍药3分，蒲黄3分，甘草（炙）3分，戎盐3分，车前子8分。
出处：《外台》卷二十七引《延年秘录》。
主治：心气虚热，小便赤色如浅红花汁。
加减：None
功效：补心神，益脾气，散客热。
用法用量：每服10丸，食后少时以粥清送下。日2-3服。稍加至15-20丸，以知为度。
制备方法：上为末，以蜜及枣膏各半相和，煎令消散，和药为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌忧愁在心，并勿食热食及冷水等。
附注：None
