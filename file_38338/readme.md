序号：38348
名称：枣膏丸
组成：葶苈（去芦，隔纸炒香）、陈橘皮（去白）、桔梗（炒）各等分。
出处：《本事》卷二。
主治：息贲，在右胁下，大如杯，令人洒淅寒热，喘嗽，发痈疽。
加减：None
功效：None
用法用量：每服5-7丸，米饮送下。
制备方法：先以后2味为末，入葶苈研匀，煮肥枣肉为丸，如梧桐子大。
临床应用：予常患停饮，久积肺经，食已必嚏，渐觉肺系急，服此良验。
各家论述：《本事方释义》：甜葶苈气味苦寒，入手太阴，性能行水下气；陈橘皮气味苦辛温，入手、足太阴，桔梗气味苦辛平，入手太阴，息贲令人酒淅寒热，喘逆而咳者，此肺家欲发痈疽之象，以泻肺之药，佐以枣之甘缓，不使药之下行他经，欲其专走肺也。
用药禁忌：None
附注：None
