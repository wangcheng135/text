序号：37537
名称：抱龙丸
组成：天南星（锉开，里白者，生为末，腊月内取黄牛胆汁和为剂，却入胆内阴干，再为末）半斤，天竺黄2两（别研），朱砂2钱（研，水飞），雄黄半两（研，水飞），麝香1钱（别研），牛黄1字（别研）。
出处：《斑疹备急》。
主治：一切风热，中暑惊悸，疮疹欲出，多睡，咳嗽涎盛，面赤，手足冷，发温壮，睡中惊，搐搦不宁，脉洪数，头痛，呕吐，小便赤黄。
加减：None
功效：None
用法用量：2岁儿每服1丸，竹叶或薄荷汤化下，不拘时候。
制备方法：上为极细末，甘草水为丸，如鸡头子大，窨干。
临床应用：None
各家论述：None
用药禁忌：None
附注：牛黄抱龙丸（《明医杂著》卷六）。《痘疹世医心法》有“金箔”。
