序号：29618
名称：附桂紫金膏
组成：防风2两，生杜仲2两，木瓜2两，白芷2两，生灵脂2两，独活2两，当归2两，川芎2两，羌活2两，生附子2两。
出处：《全国中药成药处方集》（天津方）。
主治：妇女经血不调，血海空虚，行经腹痛，经来黑紫，肚腹胀疼，以及体亏气弱，腰腿无力，周身痠疼。
加减：None
功效：温经散寒，补气养血。
用法用量：贴肚腹。
制备方法：上用香油15斤，炸枯去滓滤净，炼至滴水成珠，再入章丹90两搅匀成膏。每15斤膏药油兑乳香面、没药面、广木香面、肉桂面各2两，搅匀。每大张净油1两重；小张净油5钱重。
临床应用：None
各家论述：None
用药禁忌：孕妇勿贴。
附注：None
