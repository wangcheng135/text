序号：59042
名称：秦艽散
组成：秦艽1两（去苗），枳壳1两（麸炒微黄，去瓤），白术1两，丹参1两，羌活1两，人参1两（去芦头），柴胡1两（去苗），茯神1两，紫苏茎叶1两，薏苡仁1两半，桑根白皮1两（锉），防风1两（去芦头），石斛1两（去根，锉），大麻仁1两，甘草半两（炙微赤，锉）。
出处：《圣惠》卷四十五。
主治：脚气痹挛，不能行步，时发疼痛，烦躁恍惚。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，煎至6分，去滓温服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：秦艽汤（《圣济总录》卷八十一）。《圣济总录》有陈橘皮，无茯神。
