序号：60780
名称：通窍排脓汤
组成：细辛1.5g，薏苡仁9g，白芷3g，天花粉6g，黄耆9g，茯苓9g，甘草3g。
出处：《张皆春眼科证治》。
主治：大眦漏症溃后，肿核已消，肤色如常，按压眦部，或流脓液，或浊水外溢者。
加减：若脓水清稀，或浊泪外溢者，是气血两伤之候，应加当归12g以养血、加党参9g以补气。
功效：扶正祛邪，通窍排脓。
用法用量：None
制备方法：None
临床应用：None
各家论述：本方是清心排脓汤去生地、木通，加细辛、黄耆而成。因患眦漏已久，眦部已不肿不红，证情转虚，故去生地、木通清热之品。加黄耆补气健中，且助薏苡仁、白芷、天花粉托里排脓。加细辛者，是借其开散走窜之力，使泪窍畅通、排尽里蓄之脓。
用药禁忌：None
附注：None
