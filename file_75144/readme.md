序号：75160
名称：蒸鸡丸
组成：黄连1两，柴胡1两，芜荑半两，鹤虱半两，秦艽1两，知母1两，茯苓1两，使君子1两。
出处：《普济方》卷三七九引《金婴方》。
主治：小儿疳劳，骨蒸潮热，盗汗瘦弱，腹急面黄，饮食不生肌肉，日哭夜啼，多渴少餐。
加减：None
功效：None
用法用量：每服2岁12丸，米汤吞下，无时服；15岁儿以温酒送下。
制备方法：上为末，以黄雄鸡1只（重1斤者）笼之，专以大麻子饲之，至5日后，去毛令净，于臂后开孔，去肠肚净拭干，入前药于鸡腹内，以线缝之；取小甑，先以黑豆铺甑底，厚3寸，安鸡在甑中，四傍以黑豆围裹，上以黑豆盖好，自日出时蒸至晚，候温冷，取鸡净肉研和得所，如硬，入酒面糊同药末为丸，如赤豆大。
临床应用：None
各家论述：None
用药禁忌：忌食猪肉、黄雌鸡肉。
附注：《补要袖珍小儿》无茯苓，有丹参。
