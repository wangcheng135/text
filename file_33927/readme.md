序号：33936
名称：羌葛开膈散
组成：干葛3钱，羌活2钱，防己2钱，防风1钱，苍术1钱，小茴1钱，麻黄根1钱半，桂枝1钱半，川瓜1钱半，川芎1钱半，黄柏8分，姜炭8分，甘草2分。
出处：《点点经》卷四。
主治：身痛畏寒发抖，腰胀肢冷，肾子肿痛，两手酸麻如感冒之状。
加减：None
功效：None
用法用量：生姜为引，水煎服。取汗为度。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：如心气下陷，渗入肾水，膀胱火动，流注肾子，肿硬胀塞，肾囊紫红，若用敷法，必先用陈米泔水煎薄荷汤洗净肾囊，拭干污气，以开膈散末，用毡4-5层厚，照囊宽大长短，将烧酒喷润，烘大热，将末掺掩毡上，把肾囊周围包裹，只留茎物在外，如此扎紧，服前药取汗，敷药以对昼为度。
