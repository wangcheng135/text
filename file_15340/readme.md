序号：15347
名称：决明子丸
组成：决明子（炒）半两，细辛（去苗叶）半两，青葙子半两，蒺藜子半两（炒，去角），茺蔚子半两，芎藭半两，升麻半两，独活半两（去芦头），羚羊角半两（镑），防风（去叉）半两，菊花1两，黄连（去须）3两，玄参3两，枸杞子3两。
出处：《圣济总录》卷一○七。
主治：风热上冲眼目，或外受风邪，眼胞红肿，目昏赤痛，瞳仁昏暗，翳障遮睛。
加减：None
功效：补肾养肝，清头明目。
用法用量：每服20丸，以淡竹叶熟水送下。加至30丸。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌食辛辣、发物。
附注：决明丸（《医统》卷六十一）。
