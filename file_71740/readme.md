序号：71756
名称：犀角散
组成：犀角末1分，柴胡（去芦）1分，枳壳（面炒）1分，麦门冬（去心）1分，茯苓（去皮）1分，芍药1分，大黄1分，桑白皮1分，人参1分，黄耆1分，鳖甲1个（醋炙令黄）。
出处：《博济》卷四。
主治：小儿骨热，晚后多发热，面赤，五心烦闷，四肢无力，饮食减少，夜多盗汗，面色痿黄；大人盗汗。
加减：None
功效：None
用法用量：每服半钱，将桃仁9个（浆水煮），麦门冬19个（去心）与桃仁同研令细，入水1盏，与药同煎至6分，去滓温服，早、晚各1服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：犀角汤（《圣济总录》卷一七七）。
