序号：56662
名称：消风百解散
组成：荆芥4两，白芷4两，陈皮（洗，去白）4两，苍术4两，麻黄（去节）4两，甘草（炙）2两。
出处：《局方》卷二（宝庆新增方）。
主治：四时伤寒，身体烦疼，四肢倦怠，行步喘乏，及寒壅咳嗽，鼻塞声重，涕唾稠粘，痰涎壅盛，气急满闷者。感冒风邪，发热自汗。
加减：欲发散邪风，入连须葱白3寸同煎。
功效：None
用法用量：每服2大钱，用水1大盏，加生姜3片，乌梅1个，同煎7分，温服，不拘时候；或茶酒调下。
制备方法：上为细末。
临床应用：None
各家论述：《医方考》：伤风宜解肌，咳嗽宜利气，荆芥、白芷、麻黄，可以解肌；陈皮、苍术、甘草，可以利气。《经》曰：辛甘发散为阳。夫六物皆辛甘，则皆解散矣。
用药禁忌：None
附注：百解散（《保婴撮要》卷十）。本方方名，《北京市中药成方选集》引作“百解发汗散”。
