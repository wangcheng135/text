序号：83704
名称：乌梅丸
组成：乌梅肉1两（微炒），柴胡1两（去苗），知母3分，鳖甲1两（涂醋，炙令黄，去裙襕），桃仁3分（汤浸，去皮尖双仁，麸炒微黄），虎头骨3分（涂酥，炙令黄），人参半两（去芦头），恒山半两，秦艽3分（去苗），川升麻半两，白术半两，子芩半两，黄耆3分（锉），豉心1合，木香半两，甘草半两（生用），远志半两（去心），槟榔1两。
出处：《圣惠》卷三十一。
主治：骨蒸劳热，肢节疼痛，心膈壅闷，少思饮食。
加减：None
功效：None
用法用量：每服30丸，食前以清粥饮送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌苋菜。
附注：None
