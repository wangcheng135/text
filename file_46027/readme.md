序号：46042
名称：洗心散
组成：白术1两半，麻黄（和节）6两，当归（去苗洗）6两，荆芥穗6两，芍药6两，甘草（爁）6两，大黄（面裹，煨，去面，切，焙）6两。
出处：《局方》卷六。
主治：风壅壮热，头目昏痛，肩背拘急，肢节烦疼，热气上冲，口苦唇焦，咽喉肿痛，痰涎壅滞，涕唾稠粘，心神烦躁，眼涩睛疼；及寒壅不调，鼻塞身重，咽干多渴，五心烦热，小便赤涩，大便秘滞，并宜服之。
加减：None
功效：None
用法用量：每服2钱，水1盏，入生姜、薄荷各少许，同煎至7分，温服；如小儿麸豆疮疹欲发，先狂语多渴，及惊风积热，可服1钱，并临卧服；如大人五脏壅实，欲要溏转，加至4-5钱，乘热服之。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：七宝洗心散（《医方类聚》卷一二五引《简易》）。
