序号：76085
名称：蛜(虫祁)丸
组成：蛜（虫祁）3分（微炒），白附子3分（炮裂），防风3分（去芦头），天麻3分，天雄3分（炮裂，去皮脐），白花蛇1两（酒浸，去皮骨，炙微黄），黄耆3分（锉），萆薢3分，桂心1两，独活3分，丹参3分，当归3分（锉，微炒），安息香1两，海桐皮3分（锉），补骨脂3分（微炒），仙灵脾3分，牛膝1两（去苗），雄黄半两（细研，水飞过），麝香半两（细研），肉苁蓉3分（酒浸1宿，刮去皴皮，炙令干）。
出处：《圣惠》卷七。
主治：肾脏中风。脚膝麻痹无力，腰背强直疼痛，言语不利，面色萎黑，肌体羸瘦。
加减：None
功效：None
用法用量：每服30丸，以温酒送下，1日2次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
