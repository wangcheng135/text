序号：80897
名称：麝香丸
组成：麝香（细研）1分，朱砂（细研）1分，芦荟（细研）1分，雄黄（细研）1分，母丁香1分，鹤虱1分，白矾灰1分，密佗僧（细研）1分，没药1分，龙胆（去芦头）1分，地龙（微炒）1分，熊胆（细研）1分，肉豆蔻半两（去壳），黄连半两（去须），定粉半两（微炒），艾叶半两（炒令黄燥）（焦），蟾酥1钱。
出处：《圣惠》卷九十三。
主治：小儿疳痢不止，体瘦，食少腹痛，赢弱。
加减：None
功效：None
用法用量：每服3丸，以粥饮送下，每日3次。
制备方法：上为末，入研了药令匀，以面糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
