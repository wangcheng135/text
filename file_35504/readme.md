序号：35514
名称：京三棱煎丸
组成：京三棱3两（杵末，取2两），硇砂1两（飞过，同三棱用米醋3升煎成膏），当归（酒浸，切，焙）半两，大黄（锉，炒）半两，鳖甲（去裙襴，醋炙）半两，五灵脂（炒）半两，木香半两，没香（锉）半两，槟榔（锉）3分，桂（去粗皮）3分，干漆（炒令烟尽）3分，没药1分，马蔺花1分，蓬莪术（炮）1两。
出处：《圣济总录》卷四十七。
主治：胃热肠寒，食已复饥，小腹痛胀。
加减：None
功效：None
用法用量：每服7丸，空心、临卧温酒或盐汤送下。
制备方法：上为末，入三棱煎，搜和为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
