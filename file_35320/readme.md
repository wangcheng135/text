序号：35330
名称：乳香膏
组成：川乌（生，去皮脐，切）半两，乳香（研粗）半两，没药（研粗）半两，太平州白芷半两，赤芍药（切）半两，当归（洗，切）半两，绵黄耆（切）半两，白及（切）1分，桑白皮（切）半两，白蔹（切）1分，桂（切）1分，血竭（研）1分，防风（切）1分，巴豆21枚（去壳皮），连须葱7条，桃枝10条长4寸，柳枝10条长4寸（切）。
出处：《普济方》卷三一四。
主治：一切痈疽恶疮，结毒赤肿，疼痛呻吟不忍闻；或恶疮久而未愈者。
加减：None
功效：拔脓生肌止痛，未溃贴令内消，已溃贴收敛脓血。
用法用量：用如常法。
制备方法：上用麻油1斤，浸药3宿，慢火熬，直令白芷焦赤色为度，以绵滤过，入黄蜡2两，渐渐再熬片时，下黄丹3两，搅令匀，再熬候滴入水中成珠子为度，收入净瓷罐中密封。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
