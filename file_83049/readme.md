序号：83065
名称：乌头丸
组成：乌头1斤（劈破，以冷水浸2宿，洗净去脐，不去皮。用新黑豆1升，与乌头同煮，自早至夜，以豆烂为度。取乌头去皮，薄切，焙干后与诸药同杵为末，其豆亦同焙，别杵为末用），防风（去叉）2两，白附子（炮裂）2两，独活（去芦头）2两，海桐皮（锉）2两，羌活（去芦头）2两，桑螵蛸（炒）2两，乌药（锉）2两，芎2两，白头翁2两，当归（锉，焙）2两，狼毒2两，乌贼鱼骨（去甲）2两，牛膝（去苗，酒浸，切，焙）2两，天南星（炮裂）2两，威灵仙（去苗土）2两半，白僵蚕（炒）2两半，白花蛇（酒浸，取肉焙干）2两半，干蝎（炒）1两半。
出处：《圣济总录》卷八。
主治：风脚软，膝腕枢纽不用。
加减：None
功效：None
用法用量：每服20丸，空心温酒或盐汤送下。
制备方法：上为细末，用前黑豆末醋煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
