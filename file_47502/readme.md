序号：47517
名称：神人阿魏散
组成：阿魏3钱，青蒿1握（研），东北方桃枝1握（细切），甘草如病人中指许大（细锉）。
出处：《医学纲目》卷五引《济生》。
主治：骨蒸传尸等寒热羸劳，困倦喘嗽。
加减：None
功效：None
用法用量：上以童便2升许，隔夜浸药，明旦煎取1大升，分为3服，每次调入槟榔末3钱，空心温服，如人行10里，更进1服。服至1-2剂即吐出虫子，或泻出，更不须服余药；若未吐利，即当尽脓，病在上则吐，病在下则利，皆出虫如马尾、人发即愈。服药后觉五脏虚弱，魂魄不安，即以白茯苓汤补之。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：服药后忌油腻、湿面、生冷硬物。
附注：阿魏散（《永乐大典》卷八○二一引《普济经验加减方》）。
