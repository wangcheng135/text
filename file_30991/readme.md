序号：31000
名称：十膈气散
组成：人参（去芦头）1两，白茯苓（去粗皮）1两，官桂（去粗皮）1两，枳壳（麸炒，去瓤）1两，甘草（锉，炙）1两，神曲（炒令黄）1两，麦芽（炒黄）1两，诃黎勒皮（煨，去核）1两，吴白术1两，陈橘皮（去白）1两，干生姜（炮）1两，荆三棱（煨，锉）1两，蓬莪术（煨，锉）1两，厚朴（去粗皮，用生姜汁涂，炙）半两，槟榔（煨，锉）半两，木香半两。
出处：《御药院方》卷四。
主治：10般膈气。冷膈、风膈、气膈、痰膈、热膈、忧膈、悲膈、水膈、食膈、喜膈、上喘痰嗽，岁月渐深，心胸噎塞，渐致羸瘦。
加减：None
功效：None
用法用量：每服1钱，入盐1字，白汤点服亦得。如脾胃不和，腹胀，心胸满闷，用水1盏，加生姜7片，大枣2个，盐少许，同煎至8分，空心、食前和滓热服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：十膈散（《医统》卷二十七）。
