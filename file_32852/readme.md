序号：32861
名称：羌活汤
组成：羌活（去芦头）3分，蒺藜子（炒，去角）3分，沙参3分，丹参3分，麻黄（去根节）3分，白术3分，羚羊角（镑）3分，细辛（去苗叶）3分，萆薢3分，五加皮3分，五味子3分，生干地黄（焙）3分，赤茯苓（去黑皮）3分，杏仁（汤浸，去皮尖双仁，炒）3分，菖蒲（去毛）3分，枳壳（去瓤，麸炒）3分，郁李仁（汤浸，去皮尖，炒）3分，附子（炮裂，去皮脐）3分，桂（去粗皮）3分，木通半两，槟榔半两。
出处：《圣济总录》卷十九。
主治：皮痹。
加减：None
功效：None
用法用量：每服4钱匕，水1盏半，加生姜5片，煎至7分，不拘时候，去滓温服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
