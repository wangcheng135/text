序号：33135
名称：万应锭
组成：乳香1056两（炙），儿茶1056两，没药1056两（炙），香墨1056两，胡黄连1056两。
出处：《北京市中药成方选集》。
主治：中暑头昏，吐血衄血，无名肿毒，咽喉肿痛，及小儿热症。
加减：None
功效：清热祛暑，解毒止血。
用法用量：每服5分至1钱，每日2次，温开水送下；小儿每服5粒，三岁以下者，酌情递减。
制备方法：上为细末，过罗，每576两细粉兑麝香3两6钱，冰片3两6钱，研细，混合均匀，用牛胆汁360两为锭，每两约作200粒（干重），放在阴凉通风处晾干，每16两上金衣40张，阴干，蜡袋装1钱。
临床应用：None
各家论述：None
用药禁忌：忌食辛辣、油腻等物，孕妇忌服。
附注：None
