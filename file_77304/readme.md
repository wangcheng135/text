序号：77320
名称：锡蔺脂丸
组成：锡蔺脂半两，白僵蚕（炒）半两，芎半两，藿香叶半两，天南星（炮）半两，白芷半两，甘松（去土）半两，乳香（研）半两，枫香脂（研）半两，骨碎补（去皮毛）半两，乌头（新汲水浸1宿，去皮脐，切，焙干）1两，羌活（去芦头）1两，自然铜（煅，醋淬）1两，糯米（炒令黑色）2两。
出处：《圣济总录》卷十。
主治：风寒客搏，血气凝涩不通，历节疼痛，甚则短气汗出，肢节不得屈伸。
加减：None
功效：None
用法用量：每服5-7丸，细嚼，食后、临卧炒地黄酒送下。
制备方法：上为末，煮糯米粥为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
