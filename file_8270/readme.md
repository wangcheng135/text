序号：8270
名称：半夏茯苓汤
组成：半夏5两（洗），生姜5两，茯苓3两，旋覆花1两，陈橘皮2两，人参2两，桔梗2两，芍药2两，甘草（炙）2两，桂心1两。
出处：《外台》卷七引《小品方》。
主治：胸膈心腹中痰水冷气，心下汪洋，嘈烦，或水鸣多唾，口清水自出，胁肋急胀，痛不欲食，其脉喜沉弦细迟。
加减：欲得利者，加大黄；须微调者，用干地黄；病有先时喜水下者，加白术3两，除旋覆花；若大便不调，宜加大黄及干地黄，并用3两。
功效：None
用法用量：以水9升，煮取3升，分3服。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：忌羊肉、饧、酢物、生葱、猪肉、海藻、菘菜。
附注：None
