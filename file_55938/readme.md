序号：55953
名称：桑枝煎丸
组成：桑枝1斤，槐枝1斤，柳枝1斤（细锉），黑豆1升（淘洗过），天蓼木（研）半两（上药以水2斗，煎至5升，滤去滓，入酒1升，更熬令如稀饧），天麻2两，海桐皮1两，萆薢1两，芎藭1两，防风1两（去芦头），五加皮1两，酸枣仁1两（微炒），薏苡仁1两，桂心1两，生干地黄1两半。
出处：《圣惠》卷七十四。
主治：妊娠中风。手足缓弱，口眼斜，言语謇涩，肢节疼痛。
加减：None
功效：None
用法用量：每服20丸，食前以温酒送下。
制备方法：上为细末，入前煎中拌搜，入炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
