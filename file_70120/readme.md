序号：70136
名称：滋补汤
组成：生干地黄1两，熟干地黄（洗焙）1两，刘寄奴（去根）1两，泽兰叶1两，麻黄（不去节）1两，干葛1两，青蒿1两，苦梗1两，知母1两，天仙藤1两，地黄芩1两（炒），人参半两，秦艽半两，鳖甲半两，黄耆半两。
出处：《普济方》卷三二三引《杨氏家藏方》。
主治：None
加减：None
功效：补诸不足，调养血气，悦泽颜色，充肌肤，进饮食。
用法用量：上(口父)咀。每服3钱，水1盏，酒1分，猪筒骨一茎炙焦，分为4服，桃、柳枝各7寸，杏仁5粒（去皮尖，捶碎），煎至7分，去滓温服。加乌梅半个尤炒。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
