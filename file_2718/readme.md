序号：2718
名称：天中药茶
组成：细条1斤（武夷松罗更妙），石菖蒲1两，羌活1两，陈皮1两，川芎1两，北细辛1两，麦芽（炒）1两，吴茱萸1两，生甘草1两，干姜（不炒）1两，生枳实5钱，苏叶4两，上朱砂1两，明雄黄1两（另研极细）。
出处：《医方易简》卷四。
主治：男妇小儿感冒，一切时症。
加减：None
功效：None
用法用量：每服3钱，用沸汤泡少刻，候药味出，连服2-3次。有汗即止，无汗即出。蛇咬取汗亦妙。此药配合，宜5月5日午时。
制备方法：以上药茶和匀，晒干，日中喷上好火酒，待滋润拌之，又拌入朱砂、雄黄末，再喷火酒，再拌再喷，当天日中晒干，如无日色，即以火微微烘干，用瓷罐收贮，勿令泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
