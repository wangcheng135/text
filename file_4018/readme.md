序号：4018
名称：木一
组成：荆芥、连翘、防风、红花、青皮、桔梗、枳壳、山楂、卜子。
出处：方出《痧胀玉衡》卷中，名见《痧症全书》卷下。
主治：小儿痘前痧胀，心胸烦闷，痰证壅塞，甚至昏迷沉重，不省人事，脉滑疾。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：痘前痧胀：夏子亮幼子，五月发热，痰喘气急，四肢战动，两目无神，不省人事，口热如炉，面有隐隐红紫细点。延余看痘，阅其腿弯，有紫筋两条，必痘因痧胀而发。先用针刺出毒血，随用本方一剂，候稍冷饮之，其痘即发，至十二朝乃痊。
各家论述：None
用药禁忌：None
附注：五十七号兑象方(《杂病源流犀烛》卷二十一)。《痧症全书》有牛蒡子。
