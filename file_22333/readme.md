序号：22341
名称：两利汤
组成：白术5钱，茯苓5钱，薏仁1两，人参1钱，甘草5分，白芍1两，当归1钱，肉桂3分，防风5分，半夏1钱。
出处：《辨证录》卷二。
主治：卒中之后，手足流注疼痛，久之则麻痹不仁，难以屈伸。
加减：None
功效：None
用法用量：水煎服。连服4剂而疼痛止，再服10剂而麻痹愈，再服10剂而屈伸尽利矣。
制备方法：None
临床应用：None
各家论述：方中补多于攻，用防风以散风，而不用泽泻、猪苓以利水，盖因虚而成风湿，既祛其风，何可复泻其水。况方中白术、意仁未尝非利水之药也。于补水之中以行其利水之法，则水易流，而无阻滞之虞。水湿既去，而风难独留，故少用防风以表邪，而孤子之风邪，无水既于作浪，不必多用风药，而风无不除也。
用药禁忌：None
