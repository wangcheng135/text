序号：58842
名称：秘传赛宝丹
组成：黑丑10两（取头未4两，分2处），木香（末）半两，锡灰（醋炒，末）1两，槟榔（取净末）2两（分2处），雷丸（取净末）2两（分2处），陈皮（取未）半两，青皮（取末）半两，三棱（醋炒）1两，莪术（醋炒）1两。
出处：《直指附遗》卷五。
主治：追虫，取积。
加减：None
功效：None
用法用量：每服3钱，四更时候用冷茶吞下，复睡至天明，不可洗手、洗面、吃汤物，待取下或虫或积，恶毒滞气，并原药下尽，方可用冷水洗面。其药未下，宁耐片时，见其药下，再用药食补之。
制备方法：上为细末，再用使君子2两，鹤虱、皂角各1两，3味用水2碗，煎至1盏，用粟米1合，布包在药内煮，将熟起手，用黑丑末2两法起，次用槟榔末1两，再用雷丸末1两尽，再用木香、锡灰、三棱、莪术、陈皮、青皮等末尽后，再一用雷丸，二用槟榔，三用丑末盖在外，阴干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
