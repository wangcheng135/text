序号：79099
名称：镇惊造命丹
组成：蛇含石（微火煨熟，炭火煅红，醋淬7次，研细水飞，澄去水，晒干研细）4两，代赭石（如上煅研）5钱，辰砂（水飞）5钱，青礞石（煅金色，水飞，重研）5钱，南星（牛胆制）5钱，茯神5钱，僵蚕（洗，炒，末）3钱，蝉退（去土，末）3钱，白附子（末）3钱，使君子（末）3钱，天麻（末）3钱，牛黄（陕西）7分，麝香5分，冰片3分。
出处：《简明医彀》卷六。
主治：小儿胎惊，急慢惊风，癫痫不省人事，目直上视，惊风痰壅，睡中惊跳，夜卧不安，啼哭不止，客忤内钓，一切惊疾，奇形怪状，不能辨名者；及大人因惊忧劳损，卧不安寝，怔忡恍惚，恐怖癫狂；妇人产后不语，昏愦啼笑。
加减：None
功效：None
用法用量：大人每服2钱，小儿1钱婴儿3-5分，灯心、薄荷汤化服；金银煎汤尤好。
制备方法：上研匀，炼蜜和丸，金箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
