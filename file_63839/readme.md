序号：63855
名称：清暑和中散
组成：黄连（酒炒）1两，香薷（净穗）2两，厚朴1两，白扁豆（炒）4钱，猪苓1两5钱，泽泻1两5钱，白术7钱，赤茯苓（去皮）7钱，木通（去皮）1两，滑石1两5钱，枳壳（炒）1两，车前子（炒）1两，陈皮（去白）7钱，砂仁（炒）1两，木香3钱，草果仁1两5钱，甘草（炙）3钱，小茴香（炒）5钱。
出处：《古今医鉴》卷三引介石伯方。
主治：中暑诸证。
加减：None
功效：None
用法用量：每服1-2匙，随病用引。伏暑，冷水调下；腹痛，酒调下；呕吐泄泻，霍乱转筋，百沸汤调，热服出汗；呕吐甚而不止者，百沸汤和生姜汁调下；伤寒作疟者，葱白汤调下取汗。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
