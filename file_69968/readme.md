序号：69984
名称：滋荣益气汤
组成：川芎1钱，当归2钱，人参2钱，黄耆（生用）2钱，生地2钱，于术2钱，麦冬1钱，陈皮5分，升麻4分，防风3分，白芷4分，甘草（炙）4分，荆芥穗4分。
出处：《产宝》。
主治：产后半月崩来。
加减：汗多，加麻黄根5分，浮小麦1撮，大便不通，加肉苁蓉2钱；气不舒展，加木香1钱；有痰，加竹沥1匙，姜汁半匙；咳嗽，加苦杏仁2钱；惊悸，加炒酸枣仁1钱，柏子仁1钱；伤食，加神曲1钱，炒麦芽1钱；伤肉食，加炒山楂1钱，砂仁8分。
功效：None
用法用量：上药加黑枣1枚，用水1盏半，煎7分，稍热服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：本方去麦冬、防风，加黄连，名“滋荣益气止崩汤”（见《胎产心法》卷下）。
