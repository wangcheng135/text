序号：36336
名称：和中理脾丸
组成：香附（炙）32两，茯苓32两，苍术（炒）32两，厚朴（炙）32两，南山楂32两，神曲（炒）32两，麦芽（炒）32两，莱菔子（炒）32两，藿香32两，白豆蔻8两，白术（炒）48两，砂仁16两，橘皮64两，木香8两，甘草8两，法半夏16两，党参（去芦）16两，枳壳32两。
出处：《北京市中药成方选集》
主治：脾胃不和，饮食难消，倒饱嘈杂，呕吐恶心。
加减：None
功效：调理脾胃，益气和中。
用法用量：每服1丸，日服2次，温开水送下。
制备方法：上为细粉，过罗，炼蜜为丸，重3钱。
临床应用：None
各家论述：None
用药禁忌：忌食生冷及油面厚味。
附注：方中白豆蔻《全国中药成药处方集》（北京方）作“紫豆蔻”。
