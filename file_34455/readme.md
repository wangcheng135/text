名称：驱毒疏风方
组成：大羌活5分，搥鹿茸8分，川乌（湿纸包煨）3钱，草乌（黑豆煮）3钱，枫子肉（去油，净）2钱，漂苍术8分，大秦艽8分，白僵蚕（炒去丝）8分，北全蝎（洗去泥沙，姜汁炒）1钱，北蝉蜕（洗去泥）6分，荆芥穗5分，北防风6分，炒栀仁2钱，揀归身8分，大川芎8分，苏薄荷6分，条甘草3分。
出处：《疯门全书》。
主治：三十六种恶候。脸起红云，身有红块，四肢麻木。
加减：None
功效：None
用法用量：上以灯心为引，与凉肝八宝丹相间服之，每日各1次。覆滓时又送八宝丹100粒。
