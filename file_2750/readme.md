序号：2750
名称：天仙圣化丹
组成：川芎1两，防风1两，羌活1两，独活1两，胡麻子（微炒）1两2钱半，金毛狗脊（去毛）1两2钱半，苦参（去皮）1两2钱半，猪牙皂角（微炒，锉）1两2钱半，当归1两半，荆芥（陈者）半两，蝉蜕（去土）半两，全蝎（全者）半两，僵蚕（直者，炒）半两，何首乌（新者）半两，香白芷半两，苍耳草（蒸）半两。
出处：《普济方》卷一一五。
主治：一切风证。
加减：None
功效：None
用法用量：每服40丸，加至60丸。如病人面上浮肿，眉中痒不止；或是风气攀睛，手足拘挛，先服此药1料，茶清送下，1日4次，空心、食后、午后、临卧服之。或病人四肢麻木，手足刺痛，脚腿生疮，先服夺命丹1料，后服圣化丹药。10日后仍用白薄瓷碗打针：先于面上放血，次入膊上，后放腿脚上血；如遇天道睛明，5-6日间如此放血1次，量病轻重，不可放血太多。若妇人患病，放血多不妨。
制备方法：上为细末，用大风子2斤，去壳，烂杵如泥，与前药和匀，用陈米擂粉，打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
