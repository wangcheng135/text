序号：50845
名称：草灵丹
组成：生地黄32两（细切，用无灰酒1斗，夜浸昼晒，7日酒尽，焙干），鹿茸2两（酥炙黄，焙干，为末），肉苁蓉2两（酒浸7日，研为泥，焙干），牛膝1两（酒浸7日，焙干），桂心1两，蛇床子1两，菟丝子1两（酒浸7日，研为末，焙干），远志1两（去心），大枣100个（煮熟，去皮核，焙干）。
出处：《御药院方》卷六。
主治：None
加减：None
功效：补肾益真，滋荣养卫，填实骨髓，坚固牙齿，聪耳明目，延年不老，悦颜色，黑髭鬓。
用法用量：每服30丸，温酒送下。
制备方法：上为细末，炼蜜为丸，或酒、蜜、面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
