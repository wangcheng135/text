名称：消毒达原丸
组成：生军2斤，葛根2斤，滑石1斤，山楂1斤，神曲8两，木通8两，枳壳8两，黑栀8两，黑丑8两，黄芩8两，防风4两，荆芥4两，苏叶4两，龙胆草4两，苍术4两，羌活4两，草果4两，槟榔4两，厚朴4两，知母4两，车前子4两，甘草2两，生姜8钱。
出处：《卫生鸿宝》卷一。
主治：瘟疫头疼，筋骨痛，先寒后热，初起舌苔白色，3-4日黄色，5-6日黑色芒刺，大便不通，或水泻，小便黄赤短少，甚则谵语发狂，口鼻出血，拽衣摸床，呃逆不止。
加减：None
功效：None
用法用量：每服1-2丸，开水化下。
制备方法：上为末，糊为丸，每丸重2钱。
临床应用：None
各家论述：None
