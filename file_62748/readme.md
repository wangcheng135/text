序号：62764
名称：清气汤
组成：紫苏子1两，五味子1两，大腹子1两，枳壳1两，桑白皮（微妙）1两，菖蒲1两，地骨皮1两，白术1两，柴胡1两，秦艽1两，独活（干用）1两，干葛1两，甘草（炙）1两，地黄、泽兰、档子、防己、川乌、玄胡索各等分。
出处：《普济方》卷三二二。
主治：妇人血劳、产后藤劳，及羸瘦之人，阴衰阳盛，气弱而血热，则搏而不通，外蒸肌肉，内蒸骨髓，肌热骨瘦，劳时晕热，烦渴口干，颊赤头疼，饮食无味，心神惊悸，肢体酸疼，或时盗汗，或时咳嗽，或月水断绝，或经极少。
加减：None
功效：None
用法用量：每服2钱，空心酒调下。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
