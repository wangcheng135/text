序号：6232
名称：加味地黄汤
组成：熟地2钱（切片），山萸1钱，丹皮8分，川芎8分，山药1钱，泽泻8分，归身1钱，枸杞1钱，菟丝子1钱，菊花1钱，茯苓8分。
出处：《程松崖先生眼科》。
主治：肝肾亏虚，眼睛不红，不肿痛，眼胞不下坠，但视物不明，及病后眼睛看物不清楚，云翳退后不明，夜见灯有丝球者。
加减：None
功效：None
用法用量：水煎服。为丸亦可。若为丸，则用熟地8两，山药、山萸、归身、枸杞各4两，丹皮、云苓、泽泻、川芎各3两，菟丝子3两（酒蒸），菊花2两，共研细末，炼蜜为丸。空心每服4钱。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
