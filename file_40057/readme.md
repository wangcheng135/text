序号：40067
名称：肾沥汤
组成：磁石2两（捣碎，水淘去赤汁，以帛包之），巴戟1两，附子1两（炮裂，去皮脐），沉香半两，石斛半两（去根，锉），人参半两（去芦头），肉桂1两（去皱皮），白茯苓半两，牛膝3分（去苗），黄耆半两（锉），五味子半两，桑螵蛸半两（微炒），泽泻半两，防风半两（去芦头），熟干地黄1两，山茱萸3分。
出处：《圣惠》卷七。
主治：肾脏风虚，两耳常鸣。
加减：None
功效：None
用法用量：每服5钱，以水1大盏，用羊肾1对（切去脂膜），加生姜半分，与磁石包子同煎至5分，去滓，空心及晚食前温服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
