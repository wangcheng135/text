序号：47799
名称：神仙换骨丹
组成：槐角（炒黄熟）2两，桑白皮（去皮）2两，川芎2两，苍术（泔浸，去皮）2两，白芷2两，蔓荆子（去花）2两，人参2两，威灵仙2两，何首乌2两，防风2两，苦参1两，五味子1两，香附1两，麝半两（别研），麻黄10斤。
出处：《医学启源》卷中。
主治：气血凝滞，荣卫郁结，风热湿气相搏筋骨之间，内舍偏虚，发为不遂之病，气感八风，血凝五痹，筋挛骨痛，瘫痪偏枯，一切风证。
加减：None
功效：None
用法用量：每服1丸，以酒1盏浸至晚令溶化，临卧服。
制备方法：将麻黄去根、苗、节，用河水3石3斗3升，熬至6升，滤去麻黄，澄清，再熬至2升半，入其余药末，每1两3钱作10丸，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
