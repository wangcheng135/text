序号：32765
名称：陈橘皮煎丸
组成：陈橘皮1斤（汤浸，去白瓤，焙干，捣罗为末，用酒1斗入于银器中，以慢火成膏），附子2两（炮裂，去皮脐），萆薢3两（锉），京三棱3两（炮，锉），当归3两，桂心3两，干姜3两（炮裂，锉），桃仁3两（汤浸，去皮尖双仁，麸炒微黄）。
出处：《圣惠》卷二十八。
主治：虚劳癥瘕，食不消化，面色萎黄，四肢羸瘦，吃食全少，腹内常多冷气，大肠不调，腰胯疼痛。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以清粥饮送下。
制备方法：上为末，入前橘皮煎中，和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中萆薢，《普济方》作“黄柏”。
