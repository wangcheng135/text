序号：49399
名称：胡黄连丸
组成：胡黄连半两，青黛半两（细研），木香1分，蜗牛2-7个（炒令微黄），地龙半两（微炒），蟾酥1钱（研人），黄连半两（去须），槟榔1分，蜣螂5个（微炒，去翅足），朱砂1分（细研），麝香1分（细研），当归1分（微炒），犀角屑1分，干蝎1分（微炒），蛇蜕皮1分（烧为灰），芦荟1分（细研），独活1分，牛黄1分（细研），猪牙皂荚5挺（去皮，涂酥炙焦黄）。
出处：《圣惠》卷八十七。
主治：小儿脊疳，肌肤羸瘦，背脊骨高，身体寒热，面无颜色。
加减：None
功效：None
用法用量：每服5丸，以粥饮下，日3服。
制备方法：上为末，以猪胆汁和丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
