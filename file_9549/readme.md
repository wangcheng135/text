序号：9549
名称：水晶丹
组成：南星（锉作小块，汤煮少时）3钱，半夏（制）3钱，滑石4钱，轻粉50帖，净芜荑200片，巴豆50粒（去壳，全者，汤泡7次，又去心膜，作二边，水煮少时，晒干，碎切）。
出处：《活幼心书》卷下。
主治：惊积、食积、虫积，腹胀烦啼，恶心食减，面黄；及急惊后，风痰未尽，免生痴疾者。
加减：None
功效：None
用法用量：每服15-25丸，或35丸，糯米汤泡葱白，取汁小盏，五更初空心送下。过3-5行。进匀气散调补；下风痰，空心淡姜汤送下。
制备方法：上药前三味焙为末，拌和轻粉、芜荑、巴豆同碎切，在乳钵内细杵，入前药末，再杵匀；用粳大米饭包作棕子1大个，瓦瓶蒸水熟煮，候冷取出，在沙钵中烂杵，细布兜紧，捻出如稠糊，安在别器内，以前药末同杵细软丸，麻仁大。
临床应用：None
各家论述：None
用药禁忌：此药非常用之剂，有顽积、涎多、热极者乃可服。
附注：None
