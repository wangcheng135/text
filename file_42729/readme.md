序号：42744
名称：保安丸
组成：赤茯苓（去皮）3分，牡丹皮3分，白芍药3分，吴茱萸1分，沉香1分，人参（去芦）半两，当归（洗，切，焙）半两，桂（去皮）半两，牛膝（酒浸）半两，香白芷半两，木香半两，藁本（去芦）半两，麻黄（去根节）半两，川芎半两，附子（炮，去皮脐）半两，细辛（拣净）半两，兰香叶半两，甘草（锉，炙）半两，寒水石（烧）半两，防风（去芦）半两，桔梗（去芦）半两，蝉壳（去足翅）半两，马鸣退1两（炙），生干地黄1两。
出处：《产乳备要》。
主治：产前产后三十六种冷血气，半身不遂，手脚疼痛诸疾。
加减：None
功效：None
用法用量：每日空心用温酒化下1丸；或每1丸细嚼，空心温汤送下；产前伤寒中风，体如板，煎麻黄汤化下1丸。立效。
制备方法：上为细末，炼蜜为丸，如小弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：万灵丸（《普济方》卷三一六）。
