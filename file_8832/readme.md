序号：8832
名称：水府丹
组成：经煅花蕊石（研）1两半，硇砂（纸隔沸汤淋，熬取霜）半两，桂心（别为末）1两，木香1两，干姜1两，缩砂仁2两，红豆半两，斑蝥100个，腊月狗胆7枚，生地黄汁1升，童便1升，蚖青300个（斑蝥、蛇青2物并去头足翅，以糯米1升，同炒米黄，去米不用）。
出处：《妇人良方》卷七。
主治：妇人久虚积冷，经候不行，癥瘕癖块，腹中卒暴疼痛，羸瘠百病。
加减：None
功效：None
用法用量：每服1丸，嚼破，食前温酒送下；米饮亦可。
制备方法：上为末，同三汁熬为膏，和上末为丸，如鸡头子大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：孕妇莫服。
附注：None
