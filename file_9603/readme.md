序号：9603
名称：仙方羊肉丹
组成：肉苁蓉4两（酒洗），枸杞4两，山药4两，生干地黄1两，熟干地黄1两，远志1两，石菖蒲1两，破故纸1两（盐炒），干木瓜1两，牛膝1两（酒浸），木香、五味子、复盆子、沉香，茴香（盐炒）1两，韭子（同），龙骨5钱，鹿茸（酥炙）3两，麝香1钱。
出处：《医方类聚》卷一五三引《经验秘方》。
主治：None
加减：None
功效：升降阴阳，生气血，通经络，壮元阳，补脏腑，久服明目益肾，润颜色，阳事不倦，老来行步如飞，髭须不白。
用法用量：每服丸数不以多少，空心酒、汤任下。
制备方法：上以羊肾驱肉1斤，去筋膜，银器中用无灰酒入葱白1握，陈皮1两青盐半两，川椒（去目并合口者）半两，慢火煮肉极烂，去葱白、陈皮不用，研极烂，和前药为丸，以朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中木香、五味子、覆盆子、沉香用量原缺。
