序号：61646
名称：康复丸
组成：地黄200g，女贞子（酒蒸）60g，当归60g，太子参60g，续断60g，菟丝子（饼）60g，山药100g，五味子（醋蒸）60g，首乌藤100g，地骨皮100g，珍珠母100g，滑石粉适量。
出处：《山东省药品标准》。
主治：头昏，耳鸣，失眠，健忘，遗精盗汗，腰痠乏力。
加减：None
功效：滋肾，养血安神。
用法用量：每服30粒，1日3次。
制备方法：将山药、当归、太子参、五味子。珍珠母研细，过筛。余药置锅中煎2次，合并，滤过，取上清液，浓缩成稠膏，与药粉混匀，干燥，再研细，冷开水泛小丸，干燥，用滑石粉打光，每10粒重1g。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
