序号：51647
名称：退疔夺命丹
组成：防风8分，青皮7分，羌活1钱，独活1钱，黄连1钱，赤芍6分，细辛8分，僵蚕1钱，蝉退4分，泽兰叶5分，金银花7分，甘草节1钱，独脚莲7分，紫河车（1名金钱重楼）7分。
出处：《回春》卷八。
主治：疔疮。
加减：若有脓，加何首乌、白芷梢；在脚，加槟榔、木瓜；要通利，加青皮、木香、大黄、栀子、牵牛。
功效：None
用法用量：每服5钱，倍金银花1两、泽兰1两（少用叶）、生姜10片，同捣烂，好酒旋热泡之，去滓热服；不饮酒者，水煎亦可，然后用酒水各一半，煎生姜10片，热服出汗。病退减后，再加大黄5钱同煎，热服，以利2-3次，去除毒。
制备方法：上锉。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
