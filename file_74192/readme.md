序号：74208
名称：山药丸
组成：山药1两，菟丝子（洗净，酒浸1宿，研成饼）1两，附子（炮，去皮脐）1两，韭菜子（炒）1两，肉桂（去粗皮，不见火）1两，五味子（去枝）1两，牛膝（去芦）1两，白茯苓（去皮）1两，金钗石斛（酒浸）1两，肉苁蓉3两（酒浸，去皴皮），熟干地黄2两（洗），白龙骨1两半（煅，别研），山茱萸（去核）3分，牡丹皮3分，车前子3分。
出处：《魏氏家藏方》卷四。
主治：虚劳，肾脏衰弱，小便白浊，腿膝无力。
加减：None
功效：None
用法用量：每服30丸，食前温酒送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
