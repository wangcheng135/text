序号：48546
名称：神效膏
组成：好香油2两，花椒49粒，槐枝1寸长者49节，黄蜡1两，轻粉2分，枯矾1分。
出处：《医统》卷八十一。
主治：臁疮。
加减：None
功效：None
用法用量：贴时先将槐枝、葱、椒煎汤，洗疮令透，拭干，将膏纸贴上，外面再以油单纸掩护，以油帛缚定，1日取下，揭去近疮1张，复将煎汤洗净贴上，尽12张，无不愈者。
制备方法：先看疮大小，用绵纸裁成四方块12张，四角用小纸捻钉住听用，再以好香油入铜杓，文武火熬，下花椒煎黑取起，次下槐枝煎焦黑取起，次下黄蜡、轻粉、枯矾溶清，却下前纸浸油内令透，不可使焦，取起听用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
