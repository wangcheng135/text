序号：41481
名称：金铃子散
组成：金铃子1两，玄胡各1两。
出处：《袖珍》卷二引《圣惠》。
主治：热厥心痛；肝气郁热之胃脘，胸胁痛，疝气疼痛；妇女经行腹痛，其痛时发时止，口苦，舌红苔黄，脉弦数。或作或止，久不愈者。二维病。肝气郁滞。
加减：None
功效：行气疏肝，活血止痛。
用法用量：每服2-3钱，酒调下，温汤亦可。
制备方法：上为末。
临床应用：胃痛：用本方治愈胃痛15例。无论火郁，酒肉滞，肝阳犯胃，肝厥胃痛，胸痞脘痛，饥饱失时，阳微气阻等所致者，均用此方加味。一剂痛止，不出二剂痊愈。典型病例：覃某某，男，25岁，已婚。一九六三年秋间就诊，胃脘痛十余年，曾用中西药治疗，有时痛止，旋又复发。现已连痛三天，如刀刺，不想进食，时呕吐，症见神色颓丧，脉弦而涩，此因久病胃痛，胃络瘀滞所致。处方：金铃子、延胡、五灵脂、蒲黄、香附、半夏、陈皮。次日复诊，痛楚消失，精神安宁，追踪未见复发。
各家论述：1.《古方选注》：金铃子散，一泄气分之热，一行血分之滞。《雷公炮炙论》云：心痛欲死，速觅延胡。洁古复以金铃治热厥心痛。经言：诸痛皆属于心，而热厥属于肝逆，金铃子非但泄肝，功专导去小肠膀胱之热，引心包相火下行，延胡索和一身上下诸痛。时珍曰：用之中的，妙不可言。方虽小制，配合存神，却有应手取愈之功，勿以淡而忽之。_x000D_
2.《谦斋医学讲稿》：本方主治肝气肝火郁滞，胁痛，少腹胀痛。方仅两药，用量相等，而以金铃子为名，说明以疏肝气、泄肝火为主。金铃子只能走气分，并且偏于苦寒，配合延胡辛温活血，亦能行气止痛。_x000D_
3.《方剂学》：本方所治诸痛，乃由肝郁气滞，气郁化火所致。方中用金铃子疏肝气，泄肝火，为君药。玄胡行气活血，为臣使药。二药相配，气行血畅，疼痛自止，为气郁血滞而致诸痛的常用基本方剂。
用药禁忌：孕妇胃痛忌用，其他如胆结石及肝脉病，胃溃疡穿孔等均非本方适应症。
附注：金铃散（《杂病源流犀烛》卷十一）。
