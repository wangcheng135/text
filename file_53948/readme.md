序号：53963
名称：换腿丸
组成：石楠1两半，天南星（炮）1两半，石搬（酒浸）1两半，牛膝（酒浸）1两半，羌活1两半，薏苡仁（炒）1两半，防风（去叉芦）1两半，萆薢1两半，黄耆（蜜炙）1两半，天麻1两半，当归（酒浸）1两半，续断1两半，木瓜4两，槟榔2两半。
出处：《三因》卷三。
主治：足三阴经虚，为风寒暑湿进袭，挛痹缓弱，上攻胸胁肩背，下注脚膝疼痛，渐成风湿脚气，行步艰难，足心如火，上气喘急，全不进食。
加减：None
功效：None
用法用量：每服50丸，空心温酒、盐汤任下。
制备方法：上为末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
