序号：61011
名称：钻胃丸
组成：大黑附子1枚（去心，以刀子剜开成瓮子，入硇砂半钱、焰消半钱，以蒸饼剂生裹附子，慢火煨面焦熟，剥去面不用，只留附子用之），橘皮1两（须是好红色者，汤浸，去白瓤，每以一1片子裹巴豆1粒，同米半升入铫内，慢火炒令熟，巴豆不用，只留橘皮用之），半夏曲1两，青皮1两，丁香120粒，天南星1两，胡椒120粒，藿香1两，荜茇1两，蛮姜1两，柿蒂1两，白术1两。
出处：《鸡峰》卷二十。
主治：一切膈气，及一切气疾。
加减：None
功效：和暖脾胃，消进饮食。
用法用量：每服1丸，生姜1大块，纳药1丸在生姜内，湿纸裹煨熟，和生姜嚼吃，空心米饮送下。
制备方法：上为细末，枣肉为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
