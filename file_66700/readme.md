序号：66716
名称：鹿茸丸
组成：鹿茸1两（去毛，涂酥炙微黄），牛膝2两（去苗），巴戟1两，龙骨1两，补骨脂2两（微炒），附子2两（炮裂，去皮脐），干漆1两（捣碎，炒令烟出），熟干地黄2两，桂心1两，肉苁蓉2两（酒浸1宿，刮去皴皮，炙干），菟丝子2两（酒浸3日，晒干，别捣为末），阳起石2两（酒煮半日，细研，水飞过）。
出处：《圣惠》卷九十八。
主治：下元冷惫，风虚劳损。
加减：None
功效：补益脏腑，强壮腰脚。
用法用量：每服30丸，渐加至40丸，空心以温酒送下。
制备方法：上为末，入阳起石研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
