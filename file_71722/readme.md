序号：71738
名称：黑云膏
组成：当归2两，川乌2两，川椒2两，飞盐2两，赤芍5钱，白芷5钱，羌活5钱，木香5钱，僵蚕5钱，杏仁5钱，黑豆1升半，脂麻2升，蓖麻子100粒，苍耳子8两，白附子1两5钱。
出处：《解围元薮》卷四。
主治：大风大疠，紫黑肿块，疮癣恶形。
加减：None
功效：None
用法用量：None
制备方法：上各为末，和匀，以棉花油4两拌之，入瓷罐内筑实，以青槐枝数条插内通底，生布包瓶口，将地掘一穴，埋一阔口矮瓶在内，将药瓶倒入瓶内，盐泥封固，上面以棉花核2斗堆在瓶上，以桑柴200斤烧半日，去柴留炭火，煨至午间，则油滴在下瓶内，俟冷取出，加雄黄、乳香、没药、血竭各5钱，牛黄1钱，麝香2钱，共研末加入。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
