序号：60967
名称：大黄耆散
组成：黄耆（生，细锉）半两，款冬花（焙）半两，牛膝（去头，酒浸1宿，焙）半两，柴胡（去芦，洗）半两，秦艽（生）半两，青橘皮（去白，炒）半两，茴香（舶上者，炒）半两，木香（水调面裹，煨，忌伤火。以上拣净）半两，贝母（大者）7个（汤泡7次），杜仲（酒浸1宿，劈开，渗尽酒，炙色黄）1分，肉桂（去皮，不见火）1分，穿心巴戟（去心，生用）1分，甘草（炙黄）1分，萆薢1分，石斛1分，附子（大者）7钱（炮裂，去皮尖）。
出处：《普济方》卷二二三引《卫生家宝》。
主治：劳倦。
加减：None
功效：补虚，益颜色，填骨髓。
用法用量：每服2钱，水1盏，加生姜3片，大枣1枚，同煎至7分，倾向盏内，去滓，空心、食前温服，1日3次。
制备方法：上药焙，为末。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油面、炙煿、带壳物。
附注：None
