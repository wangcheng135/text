序号：77383
名称：遇仙膏
组成：炉甘石7两（用黄连锉碎，又用童便1大碗，浸黄连1宿，用砂锅1片，坐于火上，置甘石，上再用锅一片盖以煅红，时时蘸连水，水尽，取甘石细研如腻粉，称5两，黄连用3两），当归1两，乳香3钱，轻粉2钱，麝香半钱，黄丹（水飞，干）2钱，乌鱼骨（去皮）半两，硇砂4钱（温水半盏化开，澄去渣垢，用水煮干，刮药下，称2钱），白丁香粉1两。
出处：《医方类聚》卷七十引《经验秘方》。
主治：眼疾。
加减：None
功效：None
用法用量：每用皂角子大1块，水半盏，化开温洗。
制备方法：上为极细末，用好蜜2斤，于铜锅内熬数沸，去蜡，称20两，再下锅，速令人以竹片搅至赤色，下甘石，次下丹，余药相合，时时下药尽，成膏子，倾银器内。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
