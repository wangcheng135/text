序号：46467
名称：济阴百补丸
组成：当归（酒洗，晒干）6两，熟地（酒洗）1钱，香附子1斤（分4制：醋、酒、童便、盐水各浸3日，炒干），白芍药（酒炒）1钱，川芎1钱，益母草（5月5日采者佳，忌铁，净末）半斤，甘草（炙）1两，白茯苓（去皮）3两，玄胡索（炒）2两，人参（去芦）2两，木香（不见火）4两，白术（土炒）4两。
出处：《医便》卷四。
主治：女人劳伤，气血不足，阴阳不和，作寒作热，心腹疼痛，胎前产后，诸虚百损。
加减：None
功效：调脾胃，补虚损。
用法用量：每服60-70丸，渐加至80-90丸，空心米汤、酒任下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
