序号：28638
名称：补天育麟丹
组成：鹿茸1具，人参10两，山茱萸6两，熟地6两，肉苁蓉6两，巴戟天6两，炒白术8两，炙黄耆8两，淫羊藿8两，山药8两，芡实8两，当归4两，蛇床子4两，菟丝子4两，柏子仁3两，肉桂3两，麦冬5两，北五味2两，锁阳2两，人胞1个（火焙），海狗肾1根，蛤蚧2条，黄连1两，砂仁5钱。
出处：《辨证录》卷十。
主治：心肾两虚，早泄。
加减：None
功效：补心火。
用法用量：每服5钱，早、晚各1次，连服2个月。无海狗肾，可用大海马2个代之；不用蛇床子，可用附子7钱（甘草3钱，煮汤泡浸制）代之。
制备方法：上各为末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
