序号：2808
名称：天灵散
组成：天灵盖2指大，槟榔5个，麝香3钱，阿魏3钱，甘遂3钱，安息香3钱，朱砂1钱。
出处：《慎柔五书》卷四。
主治：None
加减：None
功效：驱虫。
用法用量：每服3钱，用薤白14茎，葱白14茎，青蒿两把，甘草5寸，桃枝5寸，柳枝5寸，桑白皮1片，石榴根皮1片，以童便4大碗，于瓷器内，文武火煎至1碗，去滓，分作3盏，调前药末，五更初服，男患女煎，女患男煎。服药后知觉欲吐，即用白梅含之。五更尽，须下虫及恶物黄水黑粪。如未下，良久又进1服，天明更进1服。如泻不止，用龙骨、黄连等分为末，白水调下，及白梅粥补之。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
