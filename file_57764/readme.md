序号：57779
名称：烫伤软膏
组成：白芷2两，忍冬藤2两，紫草2两，冰片1钱，麻油2斤，蜂蜡1两5钱。
出处：《浙江中草药制剂技术》。
主治：Ⅱ度烧伤。
加减：None
功效：解毒消炎。
用法用量：取纱布敷于烧伤面，外盖消毒纱布包扎，每3-4天换药1次。
制备方法：将麻油加热至140℃，加入白芷和忍冬藤，煎熬10分钟，再入紫草，继续加热维持140℃约20分钟（以白芷焦黄为度），加入蜂蜡搅溶，离火，立即用多层纱布滤去渣，待冷至40℃左右，加入研细冰片搅匀，乘热倾入盛于灭菌器中的适当大小的灭菌纱布中，使之浸透，备用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
