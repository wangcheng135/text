序号：9977
名称：止崩汤
组成：生地炭9g，熟地炭9g，阿胶珠12g，莲房炭15g，山萸炭9g，当归身9g，黑升麻3g，黑芥穗6g，仙鹤草12g，五味子3g，五倍子9g，茅根炭60g，鸡冠花炭15g。
出处：《临证医案医方》。
主治：崩漏(功能性子宫出血)。子宫骤然大量下血或淋漓不断，舌质淡，苔白，脉浮大无力，或沉细无力。
加减：None
功效：养血止血。
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：本方以养血止血为主。生地、熟地、阿胶珠、当归身养血；莲房、仙鹤草、五味子、五倍子、茅根、鸡冠花止血；山萸肉补肝肾，收涩止血；莲房炭为治疗子宫出血的专药，能走子宫，又为引经药；荆芥穗能入血分，炒炭用可止血，治崩漏下黑紫血块有效；升麻有升举阳气的作用，因崩漏为下部出血，用升麻可引血上行，与当归配伍能使血循经，恢复正常血液循环，则崩漏易止。
用药禁忌：None
附注：None
