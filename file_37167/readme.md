序号：37177
名称：定神琥珀丸
组成：琥珀（捣研）1两，珍珠（捣研末）1两，牛黄（研）1两，铁粉（研）1两，天竺黄（捣研）1两，龙齿（研）1两，腻粉（研）3分，犀角（镑）3分，甘草（炙，锉）3分，露蜂房（微炙）3分，龙胆3分，升麻3分，麦门冬（去心，焙）3分，丹砂（研）3分，防风（去叉）3分，黄芩（去黑心）3分，钩藤3分，人参3分，远志（去心）3分，知母（焙）3分，天门冬（去心，焙）3分，菖蒲（九节者，去须，米泔浸，切，焙）3分，白芍药3分，茯神（去木）3分，干蝎（酒炒）1两半，麝香（研）1分，金箔100片，银箔100片（与金箔、丹砂同研断星）。
出处：《圣济总录》卷五。
主治：心中风发动不知，渐成癫痫，惊悸恍惚。
加减：None
功效：None
用法用量：每服20丸，食后、临卧煎甜竹叶汤送下，日二夜一。
制备方法：上药除别研外，捣罗为末，入研者同研令断星，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
