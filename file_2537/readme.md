序号：2537
名称：双补内托散
组成：人参5分，黄耆1钱，熟地1钱，当归8分，柴胡8分，干葛8分，白术8分，秦艽7分，川芎6分，甘草3分，生姜、大枣。
出处：《不居集》上集卷十。
主治：阴阳两虚，不能托邪外出者。
加减：若寒盛阳虚者，加制附子7-8分；表邪盛者，加羌活、防风7-8分；头痛者，加蔓荆子8分；阳气虚陷者，加升麻3-5分。
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：宏格曰：阴阳两虚之人，气血亏衰，无力以拒邪也。故用人参、黄耆、白术以补其气，熟地、当归、川芎以补其血，柴胡、干葛、秦艽以托其外邪。如四君而不用茯苓者，恐其渗泄；如四物而不用芍药者，恐其酸寒；或加肉桂，有十全之功；佐以姜、枣，有通调营卫之美。虚人服之，邪可立散矣。
用药禁忌：None
附注：方中生姜、大枣用量原缺。
