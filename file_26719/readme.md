序号：26727
名称：导痰丸
组成：天南星半斤，大半夏（各不锉）半斤，白明矾半斤，生姜半斤（洗净，切作片），猪牙皂角4两（锉碎。上五味用瓦罐以水浸过为度，煮干令透，去白矾、生姜、皂角，止取南星、半夏锉焙，同后药同研为末），真紫苏子4两，萝卜子4两，麦糵4两（并炒），糖球子4两（去核，即山果子，又名猴楂）。
出处：《普济方》卷一六三引《经验良方》。
主治：痰饮气滞，胸膈不利，喘咳气促，胁肋满胀，咳嗽多痰，鼻塞稠涕，气不升降，胸膈痞结。
加减：None
功效：None
用法用量：每服50-60丸，淡姜汤送下，不拘时候。
制备方法：上为末，面糊为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
