序号：58259
名称：真人养脏汤
组成：丁香1两2钱半，木香1两2钱半，肉豆蔻（面裹煨，去面）1两2钱半，当归（洗去芦）1两2钱半，白茯苓（去黑皮）1两2钱半，罂粟壳（去顶蒂，炙）1两2钱半，人参（去芦）1两2钱半，楝草1两（炙），乌梅肉2钱半，酸石榴皮7钱半，陈皮（去白）7钱半，赤芍药7钱半，黄连（去须）7钱半，白芍药7钱半，厚朴（去粗皮，姜汁制，炒）7钱半，干姜（炮裂）7钱半，阿胶（蛤粉炒）7钱半，地榆7钱半，诃子（炮，去核）7钱半。
出处：《百一》卷六。
主治：一切痢疾。
加减：None
功效：None
用法用量：每服5钱，水1盏半，煎至8分，去滓，通口服，食前2服，滓再作1服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
