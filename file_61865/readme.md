序号：61881
名称：推云酒
组成：川乌3两（泡），苦参5钱，羌活5钱，防风5钱，胡麻5钱，甘菊5钱，荆芥5钱，风藤5钱，连翘5钱，粉草5钱，白芷5钱，黄连5钱，当归5钱，川芎5钱，黄芩5钱，芍药5钱，牛膝5钱，独活5钱，僵蚕5钱，蝉壳5钱，生地5钱，首乌5钱，威灵仙5钱，金银花5钱。
出处：《解围元薮》卷四。
主治：紫云疙瘩，挛困麻木，剜割不知者。
加减：None
功效：None
用法用量：每日3进，每进1钟。重者四坛痊愈，轻者1料，饮酒时以药汤频浴为妙。
制备方法：上药作2帖，用酒浆1坛，入药1帖，蜜封蒸之。
临床应用：None
各家论述：None
用药禁忌：忌猪、羊肉，房事、劳役；惟鳗鲡、乌鱼、白鸭啖之方效。
附注：浴药：菊花、干荷叶、藿香、白芷、甘松、麻黄、沙参各等分，为末，每水一桶，入药末三钱，加桃、柳枝各一把，煎四五沸，睡时于无风处热洗，久出。
