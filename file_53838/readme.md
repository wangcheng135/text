序号：53853
名称：射老丸
组成：蝉壳2两5钱，当归2两5钱，柴胡2两5钱，荆芥2两5钱，苦参3两，防风3两，全蝎4两，川芎1两5钱，独活1两6钱，羌活2两。
出处：《解围元薮》卷三。
主治：癞风。变形败体，一切恶症。
加减：None
功效：None
用法用量：每服80丸，白汤送下，日3次。3日后，其腿下黑紫块上污皮渐好；10日后，即服利药（江霜1钱，牙皂末3钱。饭为丸，如萝卜子大。每服2丸，白汤送下）1次，每月利3次。
制备方法：上晒，研为末，每药末1两，加大枫子肉1两6钱，为末，赤米糊为丸，如梧桐子大，西洋珠为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
