序号：61657
名称：惊气丸
组成：紫苏子（炒）1两，橘红半两，南木香半两，附子（生，去皮脐）半两，麻黄（去根节）半两，花蛇（酒浸，炙，去皮骨）半两，白僵蚕（微炒）半两，南星（洗，浸，薄切，姜汁1宿）半两，天麻（去苗）半两，朱砂（研）1分半（为衣），干蝎（去尾针，微炒）1分。
出处：《局方》卷一（吴直阁增诸家名方）。
主治：惊忧积气，心受风邪，发则牙关紧急，涎潮昏塞，醒则精神若痴。
加减：None
功效：None
用法用量：每服1粒，用金银薄荷汤化下，温酒亦得。
制备方法：上为末，入研脑、麝少许，同研极细，炼蜜为丸，如龙眼大。
临床应用：1.惊气：戊申年军中一人犯法，褫衣将受刃，得释，神失如痴，与一粒服讫而寐，及觉，疾已失。_x000D_
2.失心：江东提辖张载阳妻避寇，夹心数年，授此方，不终剂而愈。

用药禁忌：None
