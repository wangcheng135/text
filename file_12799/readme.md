序号：12805
名称：石斛丸
组成：石斛（去根）1两，肉苁蓉（酒浸，切，焙）1两，桂（去粗皮）1两，山茱萸1两，五味子1两，白茯苓（去黑皮）1两，山芋1两，泽泻1两，石龙芮1两，人参1两，木香1两，牛膝（酒浸，切，焙）1两，覆盆子（去梗）1两，柏子仁1两，菟丝子（酒浸，别捣）1两，熟干地黄（焙）1两，鹿茸（茄子者，去毛，酥炙）1两，附子（炮裂，去皮脐）1两，茴香子（舶上者，炒）1两，枳壳（去瓤，麸炒）1两，巴戟天（去心）1两，续断1两，木瓜1两，槟榔（锉），肉豆蔻仁半两，防风（去叉）半两，蒺藜子（炒去角）半两，蛇床子半两。
出处：《圣济总录》卷八十九。
主治：虚劳腰痛。
加减：None
功效：利腰膝。
用法用量：每服30丸，空心煎姜、枣汤或盐汤、温酒任下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
