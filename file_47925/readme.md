序号：47940
名称：神圣膏
组成：木香1两，雄黄1两（细研），桂心1两，赤芍药1两，当归1两，人参1两（去芦头），附子1两（生，去皮脐），丁香1两，白芷1两，黄耆1两，没药1两，芎1两，防风1两（去芦头），甘草1两，沉香1两，细辛1两，乳香1两，白檀香1两，甘松香1两，蜡2两，松脂1两，垂柳枝2两，柏枝3两，黄丹1斤，清麻油3斤。
出处：《圣惠》卷六十三。
主治：发背痈疽，疮肿结硬，痛不可忍。
加减：None
功效：None
用法用量：用时于帛上摊贴，每日早、晚换之。取愈为度。
制备方法：上锉细，先煎油沸，下甘松、檀香、柳、柏枝，以慢火煎半日，俟色赤黑，即滤去滓；下诸药，文火煎，候白芷色黑，滤出滓；下蜡、松脂令消，以绵滤过；净拭铛，却下药油，入黄丹，再着火煎，不住手搅，候变色黑，滴安水中如珠子即膏成，以瓷盒盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
