序号：84181
名称：五石肾气丸
组成：白石英10大分，紫石英10大分，钟乳10大分，赤石脂2两半，禹余粮2两半，薯蓣5分，远志（去心）5分，细辛5分，茯苓5分，菟丝子（酒浸1宿）5分，苁蓉5分，附子（炮，去皮）5分，干地黄5分，干姜5分，桂心5分，海蛤7分，白术7分，石斛1两半，五味子3分，山茱萸3分，人参3分，续断3分，杜仲（炙）3分，泽泻3分，蛇床子3分，桔梗3分，牛膝3分，天门冬（去心）3分，鹿茸（酒浸，炙）3分，当归3分，甘草半两（炙）。
出处：《千金翼》卷二十二。
主治：诸虚劳。
加减：None
功效：None
用法用量：每服5丸，1日2次，稍加至30丸，以酒送下佳。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
