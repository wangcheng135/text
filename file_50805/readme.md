序号：50820
名称：草乌散
组成：猪牙皂角5两，木鳖子5两，紫金皮5两，白芷5两，半夏5两，乌药5两，川芎5两，杜当归5两，川乌5两，舶上茴香1两，坐拿草（酒煎熟）1两，草乌1两，木香3钱。
出处：《得效》卷十八。
主治：损伤骨节不归窠者。
加减：伤重刺痛，手近木得者，更加坐拿、草乌各5钱，及曼陀罗花5钱入药。
功效：麻倒不识痛处。
用法用量：诸骨碎、骨折出臼者，每服2钱，好红酒调下，麻倒不识痛处，或用刀割开，或用剪去骨锋者，以手整顿骨节归原，端正，用夹夹定，然后医治；或箭镞入骨不出，亦可用此麻之，或用铁钳拽出，或用凿凿开取出。后用盐汤，或盐水与服立醒。
制备方法：上并无煅制，为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：麻药草乌散（《医统》卷七十九）。
