序号：33757
名称：没药丹
组成：没药1钱，当归、大黄1两，牵牛2两，轻粉1钱，官桂1分（上同研末），硇砂1钱（同研）。
出处：《宣明论》卷十一。
主治：产后恶血不下，月候不行，血刺腰腹急痛；或一切肠垢沉积，坚满痞痛，作发往来；或燥热顿渴，喘急闷乱，肢体痛倦。大小人心腹暴痛。
加减：None
功效：None
用法用量：每服5-10丸，温服下，以快利取积，病下为度；虽利后病末痊者，后再加取利；止心腹急痛，煎乳香汤送下，取大便利。。
制备方法：上为末，醋糊为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：孕妇自利，恶物过多，不宜服；燥热极甚，血液衰竭，不可强行，宜调气养血。
附注：None
