主治：痈疽发背，乳癖，便毒，闪腰挫气，跌打损伤，筋骨疼痛，手足顽麻，痞块疝气，杨梅，风毒，一切肿毒疮疖。

制备方法：麻油浸7日，熬黑枯色，去滓再熬，滴水成珠。每油2斤，入铅粉半斤，飞丹半斤，收成膏，入后细药：木香3钱，沉香3钱，檀香3钱，降香3钱，丁香3钱，藿香3钱，枫香3钱、麝香1钱、樟脑5分、乳香8钱，没药5钱，血竭5钱，雄黄5钱，为极细末，桑枝不住手搅匀，入水中，出火毒收用。
