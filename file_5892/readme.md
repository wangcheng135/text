序号：5892
名称：平肝顺气保中丸
组成：香附米3两（童便浸3日，炒），川芎2两，陈皮（去白）3两，白术4两（土炒），厚朴1两，枳实2两（炒），黄连（姜汁炒）2两，神曲（炒）2两，麦芽（炒）7钱，木香3钱，栀子（姜汁炒）1两，莱菔子（炒）1两，半夏（姜汁炒）1两半，白茯苓1两，砂仁（炒）4钱，干生姜1两，山楂（取肉）2两，青皮6钱（香油炒），甘草（炙）4钱。
出处：《古今医鉴》卷五。
主治：郁火伤脾，中气不运，胃中伏火，郁积生痰，致令呕吐，吞酸嘈杂，心腹胀闷。
加减：None
功效：常服顺气和中，健脾开胃，进美饮食，化痰消滞，清火抑肝。
用法用量：每服100丸，食后白滚汤送下，日2次。
制备方法：上为末，竹沥打神曲糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
