序号：1884
名称：化瘀通络洗剂
组成：骨碎补15g，苏木15g，桑寄生15g，伸筋草15g，威灵仙15g，桃仁9g，续断9g，当归尾9g，桑枝9g，川芎6g，红花6g。
出处：《林如高骨伤验方歌诀方解》。
主治：上肢骨折脱位后期，筋络挛缩痠痛者。
加减：None
功效：活血舒筋，化瘀通络。
用法用量：水煎，熏洗，每剂加黄酒60g，每日1剂，熏洗2次。
制备方法：None
临床应用：None
各家论述：本方用桃仁、红花、归尾、川芎、苏木活血化瘀；桑枝、威灵仙、伸筋草祛风除湿，舒筋通络；桑寄生、续断、骨碎补强筋壮骨；川芎、桑枝等药性上浮，主上升而向外，因而本剂适合于上肢熏洗。
用药禁忌：None
附注：None
