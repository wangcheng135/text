序号：79061
名称：噙化丸
组成：薄荷叶4两，桑白皮2两，天门冬（去心）2两，麦门冬（去心）2两，知母（去皮毛）2两，百部2两，贝母（去心）2两，柿霜2两，枇杷叶（去毛，蜜炙）1两半，诃子肉1两半，阿胶1两半，橘红1两半，紫菀1两半，款冬花1两半，栝楼仁（去油）1两，栝楼皮瓤1两，黄芩1两，杏仁（炒，去皮尖油，取净霜）1两，白茯苓1两，元明粉1两，铅白霜1两，桔梗1两，旋覆花7钱半，马兜铃7钱半，五味子7钱半，硼砂5分，冰片1钱（真者）。
出处：《准绳·女科》卷三。
主治：妇人咳嗽。
加减：None
功效：清肺化痰，止嗽定喘。
用法用量：噙化1丸。
制备方法：上为极细末，梨膏为丸，如无梨膏，则以白蜜、竹沥、梨汁熬至滴水不散为度，丸如龙眼大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
