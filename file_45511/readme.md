序号：45526
名称：枸杞羊肾粥
组成：枸杞叶1斤，羊肾1对（细切），葱白1茎，羊肉半斤（炒）。
出处：《饮膳正要》卷二。
主治：肾虚劳损，阳气衰败，腰脊疼痛，腿脚痿弱，头晕脑鸣，听力减退或耳聋，阳萎，尿频或遗尿。
加减：None
功效：益肾阴，补肾气，壮元阳。
用法用量：上4味拌匀，入五味，煮成汁，下米熬成粥，空腹食之。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：以冬季食用为好，对阳盛发热，或性功能亢进者，不可选用。
附注：《药粥疗法》本方用法：将新鲜羊肾剖开，洗净，去内膜，细切；再把羊肉洗净，切碎。用枸杞叶煎汁，去滓，同羊肾、羊肉、葱白、粳米一起煮粥，待粥成后，加入细盐少许，稍煮即可。
