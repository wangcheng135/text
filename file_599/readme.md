序号：599
名称：五福延龄丹
组成：沉香3钱，木香3钱，五味子2两（微炒），菟丝子3两（酒浸），苁蓉4两，天门冬2两，巴戟（去心）2两，杜仲3两（炒），山药2两，鹿茸（酥炙）1两，车前子（炒）1两，石菖蒲1两，泽泻1两，生地黄（洗，焙）1两，熟地黄（洗，焙）1两，枸杞1两，人参1两，山茱萸（去黑仁）1两，远志1两，赤石脂1两，白茯苓1两，覆盆子1两，杏仁（去皮，炒，另研）1两，柏子仁（微炒）1两，当归（酒浸，焙干）1两，牛膝（酒浸）1两，川楝子1两，川椒7钱半（去目）。
出处：《医方类聚》卷一五三引《经验秘方》。
主治：男女五劳七伤，颜枯骨疲，日渐羸弱，妇人久不成胎，男子未老阳事不举，精神怯弱，未及七旬，发鬓俱白，行步艰难，左瘫右痪。
加减：None
功效：延年益寿。
用法用量：每服30-50丸，空心温酒送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
