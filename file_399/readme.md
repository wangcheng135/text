序号：399
名称：五香鳖甲饮
组成：鳖甲（去裙襴，醋炙）2两，大黄（湿纸煨）3分，人参2两半，附子（炮裂，去皮脐）2两半，枳壳（汤浸去瓤，麸炒）2两半，牛膝（切，焙）2两半，桂（去粗皮，锉）半两，熟干地黄1两半（炒），厚朴（刮去粗皮，用生姜汁炙）1两，五味子（炒）1两，木香1两，丁香1两，当归（切、炒）1两，白术1两，芍药1两，白茯苓（去黑皮）1两，肉豆蔻（去皮）1两，沉香（锉）1两，京三棱（炮）1两，羌活（去芦头）1两，槟榔（煨）1两。
出处：《圣济总录》卷八十九。
主治：虚劳，身体气刺疼痛，日渐瘦弱，心下气满，不思饮食。
加减：None
功效：None
用法用量：每服3钱匕，以水1盏，加大枣1枚(擘)，生姜3片，同煎至7分，去滓温服，不拘时候。
制备方法：上(口父)咀，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：五香鳖甲散（《御药院方》卷六）。
