序号：3733
名称：开郁顺气丸
组成：柴胡2两，青皮1两5钱，榔片1两，香附1两，木香5钱，枳壳5钱，酒芍5钱，山栀5钱，黄芩5钱，姜夏5钱，川芎5钱，神曲5钱，紫补5钱，砂仁5钱，广皮5钱，苍术5钱，乌药5钱，茯苓5钱，盔沉5钱，当归5钱，甘草5钱，桔梗8钱，莱菔3钱。
出处：《全国中药成药处方集》(沈阳方)。
主治：胸膈胀满，两胁攻痛，饮食不消，胃脘胀痛，癥瘕痞块，肠痛肠肿，红白痢疾。
加减：None
功效：开郁养血，消食顺气，和胃健脾。
用法用量：每服1丸，早晚空心白开水送下。
制备方法：上为极细末，炼蜜为丸，2钱重。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
