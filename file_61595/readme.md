序号：61611
名称：常山饮
组成：知母2斤，川常山2斤，草果2斤，甘草（炙）2斤，良姜20两，乌梅（去仁）1斤。
出处：《局方》卷八（绍兴续添方）。
主治：疟疾。因外邪客于风府，生冷之物内伤脾胃，或先寒后热，或先热后寒，或寒热独作，或连日并发，或间日一发，寒则肢体颤掉，热则举身如烧，头痛恶心，烦渴引饮，气息喘急，口苦舌干，脊膂酸疼，肠鸣腹痛，诸药不治，渐成劳疟者。
加减：None
功效：None
用法用量：每服3钱，水1盏，生姜5片，枣子1枚，煎至7分，去滓温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：太医常山饮（《三因》卷六）、常山饮子（《卫生宝鉴》卷十六）。
