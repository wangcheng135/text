序号：23138
名称：芎补中汤
组成：干姜（炮）1两，阿胶（锉，蛤粉炒）1两，芎1两，五味子1两，黄耆（去芦，蜜水炙）1两，当归（去芦，酒浸）1两，白术1两，赤芍药1两，木香（不见火）半两，人参半两，杜仲（去皮，锉，炒）半两，甘草（炙）半两。
出处：《济生》引《校正时贤胎前十八论治》（见《医方类聚》卷二二四）。
主治：怀孕血气虚弱，不能卫养，以致数月而堕，名曰半产；劳役感寒，以致气虚下坠者。
加减：None
功效：产后用之，养新血，去瘀血，补虚扶危。
用法用量：每服4钱，水1盏半，煎至1盏，去滓，不拘时候通口服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：芎归补中汤（《回春》卷六）、芎归补血汤（《叶氏女科》）。
