序号：6194
名称：加味养荣丸
组成：当归身（酒洗，焙）3两，白茯神（人乳蒸）2两，肥玉竹（焙）2两，杭白芍（酒炒）2两，酸枣仁（炒熟）2两，丹参（酒炒）2两，远志肉（甘草汤焙）2两，柏子仁2两，川续断（酒炒）2两，橘红（饭上蒸）1两，杜仲（盐水炒）2两，秦艽（酒炒）1两5钱，女贞实（蜜拌蒸）2两，钩藤钩4两（同石斛蒸），郁金2两。
出处：《疡医大全》卷二十六。
主治：妇人脚气，心虚血少，肝脾不足，怔仲心悸。
加减：None
功效：调经养血。
用法用量：每早服3钱，白汤送下。
制备方法：上为细末，用金钗石斛1斤，同钩藤4两，熬膏，煮和丸，如豌豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
