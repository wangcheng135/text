序号：6047
名称：加味保和丸
组成：白术（去芦，炒）5两，枳实（麸炒）1两，陈皮（去白）3两，半夏（泡，姜炒）2两，白茯苓（去皮）3两，苍术（米泔浸，炒）1两，川厚朴（姜炒）2两，香附（酒炒）1两，神曲（炒）3两，连翘2两，黄连（酒炒）1两，黄芩（酒炒）1两，山楂肉3两，麦芽（炒）1两，萝卜子2两，木香5钱，三棱（醋炒）1两，莪术（醋炒）1两。
出处：《寿世保元》卷三。
主治：虚弱之人，腹内积聚癖块，胀满疼痛，面黄肌瘦，肚大青筋，不思饮食。
加减：None
功效：消痰利气，扶脾助胃，开胸快膈，消痞除胀，清热消食。
用法用量：每服50丸，加至70-80丸，食后白滚汤送下。
制备方法：上为细末，姜汁糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
