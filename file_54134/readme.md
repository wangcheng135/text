序号：54149
名称：柴胡汤
组成：柴胡（去苗）1两半，鳖甲（去裙襕，醋炙）半两，秦艽（去苗土）半两，知母（焙）半两，桂（去粗皮）半两，人参半两，白茯苓（去黑皮）半两，附子（炮裂，去皮脐）半两，黄耆半两，五味子半两，羌活（去芦头）半两，木香半两，沉香半两，枳壳（去瓤，炒）1分，枸杞子1分，槟榔（炮，锉）2枚。
出处：《圣济总录》卷九十。
主治：五劳七伤，四肢少力，肌瘦盗汗，遗精心忪，不思饮食，咳嗽唾脓血。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，煎至6分，去滓温服，不拘时候。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
