序号：31570
名称：沉香丸
组成：沉香1两，木香1两，槟榔1两，苦楝子1两，桂心1两，茴香子1两，当归1两（微炒），丁香2两，桃仁1两（汤浸，去皮尖双仁，麸炒微黄），肉豆蔻1两（去壳），干姜半两（炮裂，锉），吴茱萸半两（汤浸7遍，焙干，微炒），干蝎半两（微炒），阿魏1两（面裹煨，面熟为度），青橘皮半两（汤浸，去白瓤，焙），蓬莪术1两，硫黄1两半（细研，水飞过）。
出处：《圣惠》卷七。
主治：肾脏虚冷气攻，心神闷乱，四肢逆冷，腹胁胀满疼痛，喘促呕吐。
加减：None
功效：None
用法用量：每服30丸，以热酒送下，不拘时候。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
