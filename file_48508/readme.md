序号：48523
名称：神效黑神丸
组成：雄黄1两，硫黄1两，水银（与硫黄结为砂子，研）1两，朱砂1两，金星礜石1两，太阴玄精1两，白石英1两，紫石英1两，悉蔺脂1两，阳起石1两，曾青1两，定粉1两，黄丹1两，滑石1两，上为细末，入固济了瓶子内，盖上钻1窍子，盖口以六一泥固济令密，先以文火养1复时，令出阴气尽，以泥塞窍，后以大火烧令通赤，待冷，用湿土培瓶子1日，出火毒毕，取出细研，入后药：乌蛇肉2两（酒浸，炙微黄），天麻2两，天南星1两（炮裂），麻黄2两（去根节），犀角屑1两，桂心1两，白附子1两（炮裂），干蝎2两（微炒），牛黄1两（研细），麝香半两（细研），白龙脑半两（细研）。
出处：《圣惠》卷二十五。
主治：一切风。
加减：None
功效：None
用法用量：每服1丸，以豆淋酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌猪、鸡、毒滑、动风物。
附注：None
