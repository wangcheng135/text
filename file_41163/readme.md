序号：41178
名称：金丝膏
组成：当归尾3钱，川白芷3钱，杏仁（去皮尖）3钱，玄参3钱，猪牙皂角（去皮弦）3钱，草乌（生，锉用）3钱，连须叶葱（肥者）10根，滴青（明者）半斤，白胶香（明者）8两，乳香半两，没药（别研为末）半两，黄蜡（明者）1两，男子乱发（洗净，挼）如鸡子大。
出处：《得效》卷十九。
主治：伤筋动骨，损痛闪肭，风毒恶疮，风湿筋寒诸病。
加减：None
功效：None
用法用量：None
制备方法：上用清油半斤，将8味依法煎熬，滤却入胶香、滴青，搅匀，下黄蜡，又搅无烟，方下乳香、没药。
临床应用：None
各家论述：None
用药禁忌：None
附注：立应金丝膏（《医方类聚》卷一八七引《经验良方》。）
