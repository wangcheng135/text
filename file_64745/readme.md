序号：64761
名称：硇砂煎丸
组成：硇砂半两（水煎为霜用），附子（去皮脐，1半炮，1半生），沉香2两（锉），天雄2两（如附子修制），木香2两，巴戟天2两（去心），肉苁蓉2两（酒浸，切焙），牛膝2两（酒浸，切焙），茴香子2两（炒），桂2两（去粗皮），槟榔2两（锉），当归2两（切，焙），补骨脂2两（炒），干姜2两（1半生，1半炮），阿魏（用米醋1升化，以布滤过）2两，楝实（去核取肉，别杵）3两。
出处：《圣济总录》卷一八六。
主治：男子脾肾风劳。
加减：None
功效：补益元脏，和一切气。
用法用量：每服20丸，空心盐汤或温酒送下。妇人无孕可服，用当归酒下。合了用好瓷盒收藏，停久不妨。
制备方法：上除沉香、苦楝肉、硇砂、阿魏外，余药为细末。其沉香、苦楝二味同研令匀，用好酒6升，先下酒1升，入银石铫子内，便入硇砂水，以慢火熬；欲尽，又添酒1升再熬；欲干，次下阿魏水，又添酒1升，准前熬；相次又添入酒1升并楝子末同熬，已熬了4升酒，然后将前药末分作5分，内将二分入硇砂煎中慢火熬，徐徐添，尽酒2升，成膏为度，倾入净盆器内，将前三分药搜拌令匀，入臼杵一千下，丸如梧桐子大。
临床应用：None
