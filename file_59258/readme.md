序号：59273
名称：脑麝祛风丸
组成：白花蛇头1个（带项3寸，酒浸，炙），乌梢蛇尾2个（长7寸，酒浸，炙），川乌头尖3个（去黑皮），附子4个（去黑皮），天南星（炮）半两，半夏（姜制）半两，白附子半两，细辛（去叶）半两，防风（去芦）半两，天麻半两，全蝎（去毒，炒）半两，白僵蚕（去丝嘴，炒黄）半两，草乌头（炮）半两，脑1分，麝1分（研）。
出处：《普济方》卷九十三。
主治：左瘫右痪。
加减：None
功效：None
用法用量：每服50丸，煎小续命汤送下，不拘时候。
制备方法：上为细末，姜汁为丸，如梧桐子大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
