序号：27831
名称：补精养心丸
组成：牛膝（去苗，酒浸，焙）1两，干山药（焙）1两，山茱萸肉1两，白茯苓（去皮，水飞）1两，肉苁蓉（酒浸，去甲白膜）1两，远志（水浸，去心）1两，茴香（青盐水拌，炒）1两，杜仲（姜炒，去丝）1两，楮实子（去梗）1两，五味子1两，归身（酒浸）1两5钱，枸杞子1两5钱，熟地黄（酒浸，焙，姜汁浸）1两5钱，麦冬（去心，酒浸）1两5钱，人参（去芦，夏月减半）1两5钱，白术（土炒，去梗）1两5钱，虎骨（酥炙）1两5钱，黄柏（酒炒）1两。
出处：《便览》卷三。
主治：诸虚百损，精寒不固，腰眼疼痛或无力，及神思不定，恍惚不安，一切不足之症。
加减：None
功效：None
用法用量：每服100丸，空心淡盐汤送下；浸酒亦好。
制备方法：上各制净、称足，为极细末，炼蜜加酒一钟，姜汁一钟，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
