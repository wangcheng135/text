序号：43081
名称：养心益肾百补丹
组成：淮生地8两（酒蒸，另杵为膏），枸杞子4两（酒浸），山药4两，丹皮3两（去木），茯苓3两（乳蒸），山萸肉4两（酒蒸），柏子仁2两（炒），覆盆2两，泽泻2两（去毛），五味子2两，菟丝子3两（水洗净，酒蒸烂，研饼，焙干，又研细末）。
出处：《宋氏女科》。
主治：真精内乏，以致胃气怯弱，下焦虚惫，及梦泄自汗，头晕目黑，耳鸣，四肢无力者。
加减：None
功效：补益元气，培填虚损。
用法用量：每日空心服100丸，淡盐汤送下。
制备方法：上为末，用蜜8两，鹿角胶1两，先溶入蜜，浮小麦粉4两，芡实粉4两，水调，亦入胶蜜同炼熟，和诸药，杵千余下为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
