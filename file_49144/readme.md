序号：49159
名称：胜金丹
组成：人参1两半，白芍1两半，赤芍1两半，川芎1两半，丹皮1两半，肉桂2两半，茯苓2两半，牛膝2两半，当归4两，白薇4两，藁本3两（以上药合一处，酒浸1日，井水淘出，焙末），4制香附末1斤，熟地4两（打和一处），赤石脂2两，白石脂2两，乳香1两，没药1两，琥珀5钱，朱砂5钱。
出处：《妇科玉尺》卷三。
主治：虚劳妇人临产，子宫虚冷不育，积年手足麻痹，半身不遂，崩带、产后等疾。男子五劳七伤。
加减：None
功效：下死胎。
用法用量：酒送下。汗出愈。兼治子宫虚冷不育，服20丸即孕。
制备方法：上为末，炼蜜为丸，金箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
