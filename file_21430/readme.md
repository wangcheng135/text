序号：21438
名称：利生丸
组成：茅苍术2两，乌药（2味俱米泔浸1宿，晒干）2两，香附（1半童便浸，炒，1半米醋浸，炒）2两，藿香2两，纯苏叶2两，厚朴（姜汁炒）2两，陈皮2两，青皮（醋炒）2两，赤芍（酒炒）2两，砂仁（去壳）2两，小茴（微炒）2两，木香2两，草果（面裹，煨，去壳）2两，川芎（微炒）1两，归身（微炒）1两，黄芩（微炒）1两，枳壳（麸炒）1两，白茯苓1两，木通1两，鸡心1两，槟榔1两，粉甘草5钱。
出处：《惠直堂方》卷一。
主治：心腹胀痛，风痰喘嗽，膈食呕酸，赤白痢疾，疟疾，身面黄胖，湿痰流注，无名肿毒，疝气，妇人血崩，恶露不净，小儿痞积。
加减：None
功效：调气止痛，利湿祛痰。
用法用量：每服1丸，心痛，灯心2分，生姜1片，煎汤送下；肚痛，生姜1片捣碎，入炒盐3分，开水冲服；胸腹膨胀，生姜皮5分，大腹皮1钱，煎汤送下；疟疾发日，用桃脑7个、生姜1片，煎汤送下；风痰喘嗽，苏叶、薄荷汤送下；赤痢，白蜜2钱，米汤调下；白痢，红糖2钱、生姜汁1匙，同米汤调下；疝气，小茴川楝汤送下；隔食呕酸，小儿痞积，生姜汤送下；血崩，恶露不净，当归1钱，煎汤送下；身面黄胖，湿痰流注，无名肿毒，俱陈酒送下。
制备方法：上药日晒干为末，陈早米糊为丸，每重1钱5分，亦须晒干，每丸9分。
临床应用：None
各家论述：None
用药禁忌：上药不可烘，不可见火；忌生、冷、硬物。
附注：None
