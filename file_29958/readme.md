序号：29966
名称：赤金丸
组成：半两钱49枚（铁线穿，火煅通红取出，酽醋内淬过，煅，再淬5、7、10遍，候苏为末），硫黄1两（研细，与上药相间同入砂盒子内，以赤石脂和如泥，固济令干，复用火煅，候冷取出，细研入下项药），附子1两（炮，去皮脐），乳香1两半（别研），川乌头1两（炮，去皮脐尖），没药半两（别研），白胶香1两（别研），地龙（去土，炒）1两。
出处：《杨氏家藏方》卷一。
主治：卒暴中风，左瘫右痪，筋脉拘挛，不能行步。
加减：None
功效：None
用法用量：每服5-7丸，加至10丸，空心温酒送下。
制备方法：上为细末，醋煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
