序号：7015
名称：四制茱楝香壳丹
组成：吴茱萸1两（分作4份，1份装带尿猪尿胞，酒浸者；1份醋浸；1份童子小便浸；1份盐炒），泽泻4两，川楝子49个（巴豆20个炒黄，去巴豆），枳壳3两（去瓤，炒），木香3两，陈皮3两，青皮3两，茴香3两（盐炒），石菖蒲3两（炒），槟榔3两，三棱3两（煨），桃仁3两（去皮，炒），萆薢3两，蓬莪术3两（煨），官桂3两，白茯苓3两，乳香3两（别研），荔枝核49枚。
出处：《普济方》卷二四八引《德生堂方》。
主治：肾经虚寒，下元虚冷，小肠疝气，阳事衰败。
加减：None
功效：None
用法用量：每服50丸，空心盐汤或酒送下。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：四制茱萸香炼丹（《奇效良方》卷四十七）。
