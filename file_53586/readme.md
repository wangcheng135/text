序号：53601
名称：夏枯草膏
组成：京夏枯草1斤半，当归5钱，白芍（酒炒）5钱，黑参5钱，乌药5钱，浙贝母（去心）5钱，僵蚕（炒）5钱，昆布3钱，桔梗3钱，陈皮3钱，抚芎3钱，甘草3钱，香附（酒炒）1两，红花2钱。
出处：《金鉴》卷六十四。
主治：男妇小儿，忧思气郁，肝旺血燥，瘰疬坚硬。瘿瘤坚硬，结核肿痛，痈疖肿毒，目珠夜痛等症。
加减：None
功效：化硬消坚。
用法用量：每用1-2匙，滚水冲服；亦可用薄纸摊贴。
制备方法：上药共入砂锅内，水煎浓汤，布滤去滓，将汤复入砂锅内，慢火熬浓，加红蜜8两，再熬成膏，瓷罐收贮。
临床应用：None
各家论述：None
用药禁忌：戒气怒、鱼腥。
附注：None
