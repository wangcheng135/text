序号：53184
名称：大阿胶丸
组成：葶苈2两（炒），人参（去芦）1两，远志（去心）1两，防风1两，白茯苓（去皮）1两，防已1两，贝母（炒）1两，阿胶（炒）1两，五味子1两，熟地黄（洗）1两，杏仁（汤去皮尖）1两，山药1两，丹参半两，麦门冬半两，杜仲（去皮，锉，炒令黑）半两，柏子仁半两，甘草半两，百部半两。
出处：《简易方》引《必用方》（见《医方类聚》卷八十五）。
主治：肺有热，或因劳叫怒，肺胃致伤，嗽中有血。积年咳嗽上气，五心烦躁，不思饮食，心肺留热。
加减：None
功效：None
用法用量：每服1丸，水1盏，研化，煎6分，食后、临卧温服，每日2-3次。
制备方法：上为末，炼蜜为丸，如弹子大，瓷器收，勿泄气。
临床应用：呕血昔盛文肃太尉，因赴召甚急，后病呕血，医官独孤及为处此方，服之立效。
各家论述：None
用药禁忌：None
附注：润膈丸（《普济方》卷一六一）。
