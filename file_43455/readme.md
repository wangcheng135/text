序号：43470
名称：养脾丸
组成：人参1钱半，白术1钱半，当归1钱半，川芎1钱半，木香1钱，青皮1钱，黄连1钱，陈皮（炒）1钱，砂仁5分，山楂肉5分，神曲（炒）5分，麦芽（炒）5分。
出处：《痘疹心法》卷十二。
主治：痘疹伤食，但脾中满或痛，脾胃素弱者；惊泻，粪稠若胶，带青色。
加减：None
功效：None
用法用量：每服30-50丸，陈仓米饮汤送下。
制备方法：上为细末，水调神曲糊为丸，如麻子大。
临床应用：None
各家论述：《准绳·幼科》：参、术助气以补脾，芎、归活血以滋脾，曲、芽、青、楂消食以健脾，砂、陈、木香通气以和脾，黄连清火以厚脾。凡病后能食不节，面黄腹饱，或泄泻，或疳积，此方治之，统名养脾。
用药禁忌：None
附注：本方制成汤剂，名“养脾汤”（见《痘学真传》卷七）。
