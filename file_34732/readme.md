序号：34742
名称：麦门冬汤
组成：甘草（炙）2两，黄芩2两，大黄（别浸）2两，麦门冬（去心）2两，芒消2两，栀子30枚。
出处：方出《外台》卷三十八，名见《圣济总录》卷一○八。
主治：石发，腹胀头痛，眼眶疼，先有癖实不消，或饮消下食内热，或时时心急痛。诸石毒，眼睛疼，寒热时作。
加减：None
功效：None
用法用量：以水7升，煮取3升，分服之。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：None
附注：按：《圣济总录》本方用麦门冬（去心，焙）二两，甘草（炙，锉）、黄芩（去黑心）、大黄（锉，炒）、桅子仁各一两。上为粗末，每服五钱匕，水一盏半，煎至八分，去滓，下芒消一钱匕，食后、临卧温服。
