序号：58640
名称：秘传远志养心丹
组成：生地黄（酒洗）1两5钱，远志（甘草汤煮，去心）1两5钱，当归（酒洗）1两5钱，甘草（炙）1两5钱，柏子仁3两，酸枣仁3两，川芎1两，人参（去芦）1两，茯神（去木）7钱，半夏（姜汁泡）7次，南星（炮）5钱，朱砂（研末，为衣）5钱，麝香1钱，石菖蒲6钱，琥珀3钱，真金箔20片。
出处：《直指附遗》卷九。
主治：心虚手振。
加减：None
功效：None
用法用量：每服40-50丸，津唾咽下；有痰者，用姜汤送下。
制备方法：上为细末，汤浸蒸饼为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：经验远志养心丹（《保命歌括》卷十二）。
