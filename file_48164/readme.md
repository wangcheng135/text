序号：48179
名称：神应普济丹
组成：川大黄5两（1酒制，1姜制，1盐浸，1白矾浸，浸透，9蒸9晒），元参（净）3两（盐水浸透），紫苏3两（净末），葛根3两，柴胡3两，香薷3两，连翘2两5钱，羌活2两，白芷2两5钱，防风2两，荆芥2两，黄芩2两（生一半，酒炒一半），藿香2两，枳壳2两，天花粉2两，薄荷1两5钱，赤芍1两5钱，生草1两5钱（麸炒），威灵仙1两（酒炒），细辛6钱。
出处：《春脚集》卷三。
主治：时行瘟疫，斑点紫黑，舌唇紫黑；斑疹，咽喉肿，身热有斑点而发疹者；伤寒发热恶寒，头痛无汗；大头瘟；疟疾；痢疾、水泻腹痛者；孕妇身热发狂者。
加减：None
功效：None
用法用量：随症用引；时行瘟疫，斑点紫黑，舌唇紫黑，急用生大黄2-3钱、石膏1-2钱煎引；斑疹名红布者，多肿咽喉，此九死一生之症也，速用牛蒡子3钱、乌梅2钱、青黛3钱、桔梗3钱、甘草1钱，煎汤剂饮，再以此药为引，泡丸服之；头痛发热无汗，葱姜引；身热有癍点，而发疹者，升麻为引；时行瘟疫，大头瘟者，牛蒡子、青黛为引；疟疾，常山、草果为引；痢疾、水泻、腹痛，木通为引；孕妇身热发狂，麦冬、竹叶为引；伤寒发热恶寒者，葱、姜为引。
制备方法：上为细末，用嫩青蒿尖捣汁，和陈仓米糊为丸，重3钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
