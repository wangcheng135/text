序号：6361
名称：加味参附正气散
组成：人参2钱半，木香2钱半，白豆蔻2钱半，川芎半两，干姜半两，甘草半两，藿香半两，茯苓半两，黄耆半两，当归（去尾）半两，丁香半两，桂心半两，陈皮半两，白芷半两，缩砂仁半两，青皮（去白）半两，白术7钱，附子（炮）7钱，半夏曲7钱，加炮附子、枸杞子（去梗，炒，拣）、菊花蕊。
出处：《得效》卷十六。
主治：男子妇人目生内障，脑中有风，致视物昏暗，不生眵粪，瞳仁开阔，鼻流清涕。此因肾经虚损，元气虚惫使然。
加减：None
功效：None
用法用量：加生姜、红枣，水煎，食前空心服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：原书用本方治上证，与锦鸠、青盐，山药、八味等丸对和服，及四柱散，十全大补去熟地，加附子、枸杞子、菊花蕊、藿香正气散并用。
