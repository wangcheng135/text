序号：27379
名称：补益活络丹
组成：生黄耆5钱，生白芍3钱，桑枝5钱，党参（去芦）3钱，熟地3钱，丹皮3钱，茯苓（去皮）3钱，灵仙3钱，红花2钱，制首乌4钱，独活3钱，桃仁（去皮）1钱，当归3钱，木瓜3钱，赤芍2钱，川芎1钱5分，杜仲炭（盐炒）2钱，地龙3钱，防己3钱，香附（醋炒）2钱，甘草2钱。
出处：《全国中药成药处方集》（天津方）。
主治：跌打损伤，气血不足，筋骨痠疼，行动不利。
加减：None
功效：舒筋活络。
用法用量：每次服1丸，重症每次服2丸，白开水送下。
制备方法：上为细末，炼蜜为丸2钱重，蜡皮或醋纸筒封固。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
