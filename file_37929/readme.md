序号：37939
名称：斧磓丸
组成：没石子（白者）1个，肉豆蔻（去皮）1个，使君子14个（去皮），拣丁香半两，芦荟（细研）半两，木香（细锉）半两，硇砂（水飞）半两，荆三棱（微炒，锉碎）半两，胡黄连半两（以上9味同为末），白矾半两，绿矾半两，胆矾半两。
出处：《王氏手集》引韩道昌方（见《幼幼新书》卷二十四）。
主治：无辜疳。
加减：None
功效：杀疳，温脾胃，思食生肌。
用法用量：每服3-5丸，食前陈粳米饮送下。
制备方法：上3味矾用酽醋1升，去核熟枣肉1两，只得于石器内慢火熬成稀膏，后入9味药在内，熬稠取出，石上涂少熟油，槌千百下，为丸如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
