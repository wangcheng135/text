序号：32677
名称：羌活散
组成：川羌活1两，独活1两，前胡1两，柴胡（去芦，水洗）1两，川芎1两，桔梗1两，枳壳（汤浸，去瓤，细切，焙干，麸炒）1两，白茯苓1两，削术1两，防风1两，细辛（去苗叶）半两，官桂半两，甘草（炙）半两。
出处：《幼幼新书》卷十四引《王氏手集》。
主治：大人、小儿四时伤寒，热病时行，疫疠，山岚瘴疟，早晚中露雾，及暴中风寒。
加减：None
功效：None
用法用量：每服3钱，水满盏，加生姜3片，大枣2枚，煎8分，和滓热服，连3服，汗出便安；如路行不及，煎白汤点热酒调亦可。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
