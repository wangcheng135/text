序号：62367
名称：旋覆花丸
组成：旋覆花（微炒）1两，薏苡仁（炒）1两，升麻1两，赤茯苓（去黑皮）1两，地骨皮1两，白槟榔（煨，锉）5枚，前胡（去芦头，微炙）1两半，防风（去叉）1两半，芍药1两半，羌活（去芦头）1两半，麦门冬（去心，焙）1两半，大麻子仁（别研如膏）1两半，马牙消（别研）1两半，枳壳（去瓤，麸炒）3分，羚羊角（镑）3分，黑参3分，白蒺藜（炒去角）3分。
出处：《圣济总录》卷八十三。
主治：风毒脚气，壅热生痰，头项强痛。
加减：None
功效：None
用法用量：每服20丸，食后温浆水送下，日2夜1。
制备方法：上为末，入马牙消，大麻仁膏相和捣罗，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
