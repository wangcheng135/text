序号：44952
名称：星附定晕汤
组成：大天南星1两（劈作两片，一半炮，一半生），附子1只，天雄1只（并生作两，－半生，一半炮），川芎（半生，半炒）半两，橘红（半生，半炒）半两，川当归半两（去芦），丁香49粒（半生半微炒），半夏（团大者）30枚（一半姜汁同研为饼子，一半白矾汤煮片时），白附子14枚（半生，半泡），蝎稍20枚（半生，半炒），甘草10寸（半生，半炙）。
出处：《魏氏家藏方》卷上。
主治：风邪痰饮伏留于三阳之经，每遇将息失宜，或有感冒，饮食不调，则头晕呕吐，心胸痞闷，甚至旋晕呕逆，如屋旋倒者。
加减：None
功效：祛风化痰和胃，定晕止呕。
用法用量：上（口父）咀。每服3大钱，生姜10片，枣子1枚，水1盏半，煎至7分，去滓。不拘时候服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
