序号：83592
名称：乌须还少丹
组成：首生童子发4两（酒煮成膏），川椒4两半，胡椒5钱，阳起石2两，川乌1两半，何首乌1两半，草乌1两半，干漆1两半，辰砂1两半，针砂1两半（共为细末，与童子发膏拌匀，入阳城罐内封，桑柴火烧，以罐子红为度，埋在阴地之中，7日足取出听用），核桃仁（麸炒黄色）3两，柏子仁3两，生地黄（酒浸）3两，枸杞子3两，麝香3分（面包煨，甘草火煨，面熟为度）。
出处：《回春》卷五。
主治：None
加减：None
功效：服100日后须发如漆，面若童颜。
用法用量：每服1钱，好酒送下。100日以后，3日或7日服1次。
制备方法：上为细末，共前药合一处。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
