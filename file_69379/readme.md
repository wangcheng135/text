序号：69395
名称：犀角散
组成：犀角屑1分，朱砂（细研如粉）1分，防风（去芦头）1分，细辛1分，天竺黄（细研）1分，茯神1分，龙脑1分（细研），川大黄（锉碎，微炒）1两，羌活1两，麦门冬（去心，焙）1两，赤芍药1两，白僵蚕（微炒）1两，槟榔1两，羚羊角屑半两，甘草（炙微赤，锉）半两，栀子仁半两，子芩半两，麝香1分（细研）。
出处：《圣惠》卷四。
主治：心脏久积风热，脏腑壅滞，口干舌缩，神思不安。
加减：None
功效：None
用法用量：每服1钱，煎竹叶汤调下，不拘时候。
制备方法：上为细散，入研了药令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中犀角屑、朱砂、防风、细辛、天竺黄、茯神、川大黄、羌活、麦门冬、赤芍药、白僵蚕用量原缺，据《普济方》补。
