序号：32408
名称：妙应丸
组成：槟榔（锉）2枚，陈橘皮（汤浸去白，焙）半两，青橘皮（汤浸去白，焙）半两，木香1分，黄连（去须炒）1分，蓬莪术（煨，锉）1分，桂（去粗皮）1分。
出处：《圣济总录》卷一七六。
主治：小儿乳癖、积聚。按之苦痛，肌肤渐瘦，面色青黄；小儿阴阳气不顺，虚痞胀满，呕逆腹痛，成癥瘕痞结。
加减：None
功效：None
用法用量：每服2丸，食后生姜汤送下。
制备方法：上为末，每抄1钱匕，入巴豆1粒（去皮心膜，醋煮令黑色），并杏仁1粒（去皮尖），灯上烧作黑灰，同研令细，与药末再合研令匀，用白面糊为丸，如粟米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
