序号：79921
名称：磁石丸
组成：磁石3两（烧，醋淬7遍，细研，水飞过），雄黄2两（细研，水飞过），桂心1两，菟丝子2两（酒浸3日，晒干，别捣为末），雄雀粪1分，牛酥1分。
出处：《圣惠》卷九十八。
主治：None
加减：None
功效：暖腰肾，壮筋骨，明耳目，利脚膝；补暖强元气。
用法用量：每服3-5丸，空心以温酒送下。
制备方法：将磁石、雄黄二味，取鸡子2枚，打破小头，作孔，出白，调和二味令匀，却入于鸡子壳内，以数重纸糊定，后即与鸡同抱之。20日后取出，细研，并菟丝子、桂心二味，入牛酥等，以蜜为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
