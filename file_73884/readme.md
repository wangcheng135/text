序号：73900
名称：硼砂丸
组成：硼砂、细桑条子。
出处：《圣惠》卷四十九。
主治：痃癖冷癥块，及丈夫腰脚，妇人血气。
加减：None
功效：补暖水脏，暖子宫，杀三虫。
用法用量：每服5丸，空心以暖酒送下。
制备方法：上先取腊月细桑条子，不限多少，烧作灰，略以水淋却苦汁后，晒令灰干，收之；先研硼砂，以水化消，拌灰令干湿得所，每1两硼砂，管灰3两，取1个固济了瓷瓶子，底下先铺干灰半寸，次下硼砂灰填实，口头更着干灰覆盖，然后用文火烧，武火煅令通赤，候冷取出重研；于竹筲箕内铺纸三重，然后安灰，以水淋之，候药透过纸，待硼砂味断，即休淋水；别取小瓷钵子1-2个，盛药汁于热灰火内养之，常令鱼眼沸，直至汁尽，候干，别入固济了瓶子内，便以大火煅一食久，待冷取出细研，用粟米饭为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“硇砂丸”。方中硼砂，《普济方》引作“硇砂”。
