序号：42463
名称：青黛丸
组成：青黛（细研）1分，牛黄（细研）1分，麝香（细研）1分，芦荟（细研）1分，朱砂（细研）1分，雄黄（细研）1分，犀角屑1分，珍珠末1分，琥珀末1分，胡黄连1分，蟾酥1杏仁大（研入），夜明砂半分（微炒），瓜蒂半分，龙脑半分（细研），干蟾1枚（烧灰），蝉壳7枚（微炒），虎睛1对（酒浸1宿，微炙），母丁香10枚，蜣螂2枚（用大麦面作饼子裹，烧灰）。
出处：《圣惠》卷八十六。
主治：小儿惊疳，肌肤羸瘦，心神烦热，口鼻疳（匿虫）。
加减：None
功效：None
用法用量：每服3丸，用奶汁化破，1丸滴儿鼻中，2丸灌入口内。
制备方法：上为末，都研令匀，以猪胆汁为丸，如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
