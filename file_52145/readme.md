序号：52160
名称：面脂
组成：防风1两半（去芦头），萎蕤1两半，芎藭1两半，白芷1两半，藁本1两半，桃仁1两半（汤浸，去皮），白附子1两半，白茯苓2两，细辛半两，甘松香半两，零陵香半两，当归1两，栝楼瓤1两，川椒50枚（去目），鸬鹚粪3分（细研），冬瓜子仁3分，麝香1分（细研）。
出处：《圣惠》卷四十。
主治：None
加减：None
功效：令人面色润腻，鲜白如玉。
用法用量：任用涂面。
制备方法：上捣碎，以酒1斗，浸1宿，明日滤出，以薄绵裹之，用白鹅脂3升、羊脂2升，于银器中微火煎之令沸，看白附子色黄膏成，滤去滓，入麝香、鸬鹚粪等，搅令稠，以瓷器盛，用鹿角锤子研2日，唯多则光滑。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
