序号：58048
名称：益母草煎丸
组成：益母草2斤，青蒿2斤，桃枝1握（长1尺），柳枝1握（长1尺）（以上4味细锉，用童便1斗，于银铛中，煎至3升，绞去滓，煎成膏），柴胡2两（去心），朱砂1两（细研，水飞过），天灵盖1两（涂酥炙令微黄），鳖甲2两（涂醋炙令黄，去裙襕），木香1两，赤芍药2两，犀角屑2两，甘草1两（炙微赤，锉），麝香半两（细研），桃仁5两（汤浸，去皮尖双仁，生研如膏）。
出处：《圣惠》卷七十。
主治：妇人骨蒸劳瘦，月候不通，心神烦热，四肢疼痛，不能饮食。
加减：None
功效：None
用法用量：每服30丸，用乌梅、甘草煎汤送下，不拘时候。
制备方法：上为末，用益母草煎都和捣为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“益母草丸”。
