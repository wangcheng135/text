序号：49005
名称：胃关煎
组成：熟地3-5钱或1两，山药（炒）2钱，白扁豆（炒）2钱，炙甘草1-2钱，焦干姜1-3钱，吴茱萸（制）5-7分，白术（炒）1-3钱。
出处：《景岳全书》卷五十一。
主治：脾胃虚寒作泻，或甚至久泻，腹痛不止，冷痢。
加减：泻甚者，加肉豆蔻1-2钱（面炒），或用破故纸亦可；气虚势甚者，加人参，随宜用；阳虚下脱不固者，加制附子1-3钱；腹痛甚者，加木香7-8分，或加厚朴8分；滞痛不通者，加当归2-3钱；滑脱不禁者，加乌梅2个，或北五味子20粒；若肝邪侮脾者，加肉桂1-2钱。
功效：None
用法用量：上以水2钟，煎7分，食远温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
