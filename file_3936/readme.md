序号：3936
名称：无价宝
组成：川楝子2两，牛膝1两（酒浸），槟榔1两，菟丝子1两（另研，酒浸），蛇床子1两，干姜5钱，穿山甲一大片（酥炙），莲肉1两（不去心），乳香3钱（另研），沉香5钱（另研），白檀香5钱（另研），鹿茸1两（炙），巴戟1两，大茴香1两，仙灵脾3钱，破故纸5钱，凤眼草3钱，胡芦巴5钱，人参1两，泽泻1两，山药1两，五味子1两，熟地黄2两，麦门冬1两，肉苁蓉1两，茯苓1两，白芍药5钱。
出处：《伤寒标本》卷下。
主治：五劳七伤，四肢无力，脚腿沉困，骨节酸疼，面目无光，阳痿不起，下元虚冷，梦失精液。
加减：None
功效：None
用法用量：每服30丸，增至90丸，好酒送下，以干物压之。修合之日，加丁香1钱
制备方法：上药除乳香、沉香、白檀香、菟丝子4味另研为细末，其余23味各为细末，同前4味炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
