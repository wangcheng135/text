序号：72377
名称：疏经活血汤
组成：当归（酒洗）1钱2分，白芍（酒炒）1钱半，生地（酒洗）1钱，苍术（米泔浸）1钱，牛膝（去芦、酒洗）1钱，陈皮（去白）1钱，桃仁（去皮尖，炒）1钱，威灵仙（酒洗）1钱，川芎草6分，汉防己（酒洗）6分，羌活6分，防风（去芦）6分，白芷6分，龙胆草6分，茯苓（去皮）7分，甘草4分。
出处：《回春》卷五。
主治：遍身走痛如刺，左足痛尤甚。
加减：有痰，加南星、半夏各1钱；如上身及臂痛，加薄桂3分；如下身并足痛，加木瓜、木通、盐炒黄柏、薏苡仁各1钱；如气虚，加人参、白术、龟版各7分；如血虚，倍四物汤，以姜汁酒浸炒，用红花1钱。
功效：None
用法用量：加生姜3片，水煎，空心温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌生冷湿物。
附注：None
