序号：26030
名称：妙灵丹
组成：天竺黄7两，胆南星7两，生石膏7两，僵蚕（炒）7两，桔梗2两，连翘4两，金银花4两，薄荷2两，贝母2两，桑叶2两，黄芩2两，杏仁（去皮，炒）2两，生地4两，甘草1两，蝉蜕1两，钩藤1两。
出处：《北京市中药成方选集》。
主治：小儿发热，痰涎壅盛，惊悸不安，咳嗽气促。
加减：None
功效：清热镇惊，祛风化痰。
用法用量：每服1丸，每日2次，温开水送下。3岁以下小儿酌减。
制备方法：上为细末，每55两细粉兑：麝香6钱，冰片6钱，朱砂2两，研细，混合均匀，炼蜜为丸，重5分，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
