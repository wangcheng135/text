序号：47520
名称：神力丸
组成：牛膝（去芦头，酒浸1宿，焙干）2两，肉苁蓉（酒浸1宿，焙干）2两，何首乌2两，川椒（去黑子闭口者，炒）2两，木鳖子（去壳）1两，天南星（炮）1两，茴香（炒）1两，防风（去芦头）1两，萆薢1两，附子（炮，去皮脐）1两，地龙（去土，微炒）1两，羌活（去芦头）1两，乌药1两，白蒺藜（炒，去刺）1两，骨碎补（去毛）1两，金毛狗脊（去毛）1两，黄耆1两，赤小豆1两，覆盆子1两，白附子1两。
出处：《杨氏家藏方》卷四。
主治：风寒湿痹，客搏经络，四肢拘挛，及瘫缓軃曳，脚膝无力，筋骨疼痛。
加减：None
功效：None
用法用量：每服30-50丸，空心以温酒或盐汤送下。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
