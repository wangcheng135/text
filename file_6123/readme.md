序号：6123
名称：加味顺气散
组成：天台乌药5两，桔梗（去芦，炒）2两半，川白芷2两半，川芎2两半，甘草（炙）2两半，陈皮（去白）2两半，白术（无油者）2两半，麻黄（去根节）1两半，枳壳（去瓤，锉，麸炒令黄）1两半，天南星（炮制）1两，大腹皮（洗净）1两，川乌头（生，去皮脐）1两，干姜（炮）7钱半，人参（去芦）半两，木香（不见火）半两。
出处：《医方类聚》卷八十九引《经验秘方》。
主治：男子妇人，气脉不顺，虚风攻注，痰涎壅滞，头目昏眩，肌肤不遂，手足缓弱。心忡气短，行步不正。
加减：None
功效：None
用法用量：上(口父)咀，每服4钱匕,加生姜5片、大枣1个，水1盏半，同煎至8分，去滓，吞下白丸子。
制备方法：None
临床应用：None
