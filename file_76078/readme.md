序号：76094
名称：蛜(虫祁)丸
组成：蛜（虫祁）1两（炒，去足），虎胫骨3分（酒浸，炙黄），川乌头3分（炮裂，去皮脐），白蒺藜1两（微炒去刺），安息香3分，槟榔3分，芎3分，狗脊3分，赤茯苓3分，白花蛇2两（酒浸，炙令黄，去皮骨），肉桂3分（去皴皮），赤箭3分，枳实3分（麸炒微黄），防风3分（去芦头）。
出处：《圣惠》卷十九。
主治：风寒入于肌肉，气血不宣，肢体不仁，牵引腰背，风痹疼痛。
加减：None
功效：None
用法用量：每服10丸，以薄荷汤送下，不拘时候。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：《普济方》有麻黄。
