序号：78074
名称：鳖甲丸
组成：鳖甲（去裙襕，醋炙）1两半，杏仁（汤浸，去皮尖双仁，炒）1两半，苦葫芦（用瓤）1两半，天门冬（去心，焙）1两半，巴豆1分（去皮心膜，出油尽），猪牙皂荚（涂酥炙）1两，石菖蒲（微炒）1两，桂（去粗皮）1两，葶苈（隔纸炒）1两，甘遂（微煨）1两，苦参1两，大黄（锉碎，醋炒）1两，柴胡（去苗）1两，当归（切1两，焙）1两，羚羊角（磅）1两，龙骨（烧）3分。
出处：《圣济总录》卷一五三。
主治：妇人水分，肢体肿满不消，因经水不通者。
加减：None
功效：去水，调经血。
用法用量：每服10-15丸，食前温水送下，每日3次。如1-2服后小便利，即减丸数及间日服。
制备方法：上为末，炼蜜为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
