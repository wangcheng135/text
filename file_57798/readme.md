序号：57813
名称：狼毒丸
组成：狼毒2两半（醋浸，炙），肉桂2两（去皱皮），川鸡头半两（去皮脐，醋拌炒），京三棱1两（炮，锉），紫菀3分（洗，去苗土），附子1两（炮裂，去皮脐），川大黄2两半（锉碎，微炒），鳖甲2两（涂醋，炙微黄，去裙襕），甜葶苈3分（隔纸炒令紫色），槟榔2两，（鱼它）甲1两（炙），木香1两，桃仁2两（汤浸，去皮尖双仁，麸炒微黄），吴茱萸1两（汤浸7遍，焙干，微炒），皂荚3分（汤浸去皮，涂酥，炙黄焦，去子），芜花半两（醋拌，炒令干）。
出处：《圣惠》卷二十八。
主治：虚劳积聚，腹中坚硬，气胀喘急。
加减：None
功效：None
用法用量：每服10丸，空心以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌苋菜、湿面、生冷。
附注：None
