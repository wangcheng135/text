序号：19377
名称：地骨皮散
组成：地骨皮3分，百合3分，黄耆三分，赤茯苓3分，人参半两（去芦头），赤芍药3分，枳壳3分（麸炒微黄，去瓤），桑根白皮3分（锉），柴胡1两半（去苗），甘草半两（炙微赤，锉），麦门冬1两半（去心，焙），犀角屑3两，杏仁1两（汤浸，去皮尖双仁，麸炒微黄），鳖甲1两（涂醋，炙令黄，去裙襕），白前3分。
出处：《圣惠》卷三十一。
主治：虚劳骨蒸烦热，心神不宁，及小便赤涩，时有咳嗽，四肢羸弱疼痛。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：忌苋菜。
附注：None
