序号：47012
名称：独活散
组成：独活1两半（盐水炒），白芍1两半（炒），防风1两半（盐水炒），当归3两，远志1两半，生地5两，龙齿3两（煅），茯神2两（去木），人参1两5钱，炙草1两5钱。
出处：《医略六书》卷三十。
主治：产后风邪乘虚袭伤营阴，心神失养，心血心气俱馁，惊悸不安，脉浮虚微数。
加减：None
功效：None
用法用量：每服3钱，银器煎汤，去滓温服。
制备方法：上为散。
临床应用：None
各家论述：方中独活开经气，防风散风邪，生地壮水以滋心血，当归养血以营心经，白芍敛阴和血以宁心，远志通肾交心以保心，人参扶元补心气，龙齿定魄安神志，茯神定神志，炙草缓中州以益胃气。为散，银汁煎，使风邪外解则血气内充，而心神得养，心气自雄。
用药禁忌：None
附注：None
