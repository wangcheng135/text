序号：49742
名称：茯苓汤
组成：赤茯苓（去黑皮）2两，泽泻2两，麦门冬（去心，焙）2两，杜仲（去粗皮，炙）2两，桑白皮（锉）3两，桂（去粗皮）1两，磁石（捣如麻粒大，淘去赤水）4两。
出处：《圣济总录》卷五十九。
主治：三焦气不宣通，膈壅停水，不下至肾，肾消肌肉化为小便。
加减：None
功效：None
用法用量：每服6钱匕，水2盏，加大枣3个（擘破），薤白5茎（细切），煎至1盏，去滓，分2服，空腹温服，如人行10里，再服，至晚亦然。此药内消，不吐利，服1剂讫，津液未通，血脉未行，肌肤未润，更服1剂。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
