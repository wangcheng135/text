序号：65378
名称：子母两快汤
组成：熟地5钱，麦冬5钱，当归2钱，山茱萸3钱，茯苓2钱，芡实2钱，山药2钱，玄参5钱。
出处：《辨证录》卷一。
主治：冬月伤寒，身热5日，肾水干燥。不能润肝，人即发厥。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：此方纯用补肾之味，惟当归滋肝之血也。治肾而治肝在其中，何必再用白芍以平肝气那！且此症又不可用白芍也，以白芍虽平肝气，可以定热厥于须臾，然而白芍定厥，未免过于酸收。与补水之药，同用于无邪之日，易于生精；与补水之药，同用于有邪之顷，亦易于遏火。不若单用补肾之味，使水足以制火，而又无火留之害，为更胜也。故子母两快汤所以不用芍药而单用当归者，以当归之性动，不比芍药之酸收耳。且当归善助熟地、山萸以生水，生水以滋肝，即补肾以制肝也。
