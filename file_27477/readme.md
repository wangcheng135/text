序号：27485
名称：补气清痢汤
组成：人参1钱5分，炙黄耆3-5钱，甜白术3钱（土炒黄），云苓2钱，当归2钱（酒炒），白芍3钱（酒炒），车前子3钱，枳壳1钱5分，陈皮1钱5分，炙甘草5分。
出处：《慈航集》卷下。
主治：气虚下陷，久痢不愈，腹有微痛，脱肛下坠，虚寒怕冷，中气不接，形象欲脱。
加减：如痢中带白，加炮姜1钱5分；如恶心，加灶心土5钱；如痢不止，加铁莲子5钱，赤石脂2钱；如呃逆，换干姜1钱5分，柿蒂5个；贫人无力服人参，以上党参1两代之。
功效：None
用法用量：用煨广木香1钱2分，老姜2钱，红枣5个为引，水煎服。一服痢轻，再服更轻，三少服痊愈。此方愈后，多服更妙。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
