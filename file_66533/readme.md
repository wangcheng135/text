序号：66549
名称：丁香半夏丸
组成：肉豆蔻仁1分，木香1分，丁香1分，人参（去芦头）1分，陈皮（去白）1分，藿香叶半两，半夏（汤洗7次，姜汁炒黄色）3两。
出处：《局方》卷四。
主治：脾胃宿冷，胸膈停痰，呕吐恶心，吞酸噫醋，心腹痞满，胁肋刺痛，短气噎闷，不思饮食。脾胃久虚寒，痰壅滞，呕吐苦水，哕逆清涎，头痛目眩，咳嗽上喘，腹中水响。
加减：None
功效：None
用法用量：每服20丸，生姜汤送下，不拘时候。
制备方法：上为细末，以生姜汁煮面糊为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：半夏藿香丸（《鸡峰》卷十八）。本方方名，《景岳全书》：引作“半夏丁香丸”。
