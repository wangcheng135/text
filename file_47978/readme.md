序号：47993
名称：神异香连丸
组成：黄连1斤，赤茯苓1两，猪苓1两，白扁豆1两，泽泻1两，三棱1两，莪术1两，厚朴1两，枳壳1两，陈皮5钱，砂仁5钱，当归1两5钱，白芍7钱5分，川芎7钱5分，苍术1两2钱。
出处：《证治宝鉴》卷八。
主治：痢后遍身疼痛，手足拘挛，不能举动下床者。
加减：None
功效：None
用法用量：每服2钱5分，看虚实大小加减，红痢，槐角汤下，白痢，陈皮汤下。
制备方法：上除黄连外，用水煎浓汁，去滓再熬，入连浸，晒干为末听用。另用吴茱萸10两（水泡去毒），同连炒燥，如金色为度，去萸不用，再加木香5钱、肉果2两（面裹煨熟），共为末，黄米糊丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
