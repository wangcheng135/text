序号：40035
名称：肺脓疡合剜
组成：半枝莲15g，金银花15g，鱼腥草15-30g，虎杖12g，黄芩12g，桔梗12g。
出处：《古今名方》引金如寿方。
主治：急性肺脓疡（肺痈）。
加减：如高热不退，加生石膏30g、知母10g；痰中带血，加茅根30g、旱莲草15g；如果热退后，吐大量脓臭痰（排脓期）时，加桃仁9g、生薏苡仁18g，以祛瘀、化痰、排脓；如经X线检查，液面消失，脓腔全部显露时，加黄精、白及各15g，以助养阴补肺，促进脓腔闭合。
功效：清热解毒，化瘀排脓，清肺透热，清养肺阴。具有退热快、排脓多、空洞闭合迅速的效果。
