序号：5600
名称：必胜散
组成：芎、归、芍、生地、熟地、阿胶、前胡、甘草、天冬、麦冬、陈皮、黄耆、白术、茯苓、刺蓟、马勃、醉芩。
出处：《陈素庵妇科补解》卷三。
主治：妊娠吐血衄血者，皆由平日忧思惊恐伤于肝脾，结于经络，久则气逆以致经血妄行，口出曰吐，鼻出曰衄。心胸烦满，甚或喘急，胎气上逼则难治。
加减：None
功效：清热凉血，养血安胎。
用法用量：None
制备方法：None
临床应用：None
各家论述：古人云：胎前见血，十不活一，此甚言经血之不可伤也。夫血以养胎，胎藉血长，一有渗漏，胎元必伤，妄行过甚，孕妇有损，吐衄，从口鼻而出血，热极矣。清热凉血，胎或可安。芎、归、胶、芍、二冬、二地所以清血分之热，可养血固胎；醉芩、刺蓟、马勃专除血中之伏火；黄耆、术、芩、陈、甘补阳以生阴之道。微嫌川芎辛散上行，宜慎之。
用药禁忌：None
附注：None
