序号：22998
名称：达原解毒汤
组成：鲜生地15g，玄参12g，白芷12g，麦冬10g，浙贝母10g，金银花10g，牛蒡子10g，山豆根10g，花槟榔10g，射干6g，丹皮6g，厚朴6g，甘草6g，草果仁6g，土牛膝30g。
出处：《言庚孚医疗经验集》。
主治：急性喉炎，山岚瘴气，居伏膜原，蕴集肺胃，火动痰生，上蒸咽喉者。
加减：None
功效：疏风透达，清热解毒，滋阴凉血，豁痰开窍。
用法用量：水煎服，日1剂。
制备方法：None
临床应用：急性喉炎：吴某某，男，16岁，学生。咽壁红而肿胀，面青唇紫，呼吸急促，身热，脉弦数，舌红苔黄。诊断为急性喉炎（声音嘶哑）。治疗方法，外治刮瘀及药贴，内服“达原解毒汤”，一周后痊愈。
各家论述：None
用药禁忌：None
附注：None
