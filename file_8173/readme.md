序号：8173
名称：加减补阴益气汤
组成：生地5钱，人参1钱半，黄耆3钱（蜜炙），柴胡5分（盐醋炒黑），白芍1钱半（醋炒），升麻3分（盐醋炒），阿胶3钱（蒲黄灰炒），山药3钱（炒），血余3钱（炒灰），赤石脂3钱（醋炒）。
出处：《医略六书》卷二十六。
主治：交接出血，脉软数者。
加减：None
功效：None
用法用量：水煎，去滓温服。
制备方法：None
临床应用：None
各家论述：阴不藏精，虚阳不能固密，而经气漏泄，经血易动，故交接出血。生地滋阴壮水以资血室，人参补气扶元以固虚阳，山药补脾益阴，黄耆补中固气，阿胶补阴益血以填任脉，白芍敛血益阴以固冲脉。升麻升阳明清气，柴胡升少阳清气，血余灰去瘀生新，最能止血，赤石脂涩血镇怯善固经气。水煎温服，使气阴内充，则虚阳自敛，而经气完固，经血自无漏泄之虞。
用药禁忌：None
