序号：72422
名称：犀角丸
组成：犀角屑2分，麝香2分（碎），朱砂4分（光明者，研），桔梗2分，莽草2分（炙），鬼臼2分，附子2分（炮），桂心2分，贝齿5枚，甘草6分，芫花2分（熬），巴豆20枚（去心皮），赤足蜈蚣2枚（去足，炙）。
出处：《外台》卷七引《古今录验》。
主治：久心痛。腹痛积年，定不过一时间还发，发甚则数日不能食，又便出干血。
加减：None
功效：None
用法用量：每服1丸，日渐加至3丸，饮送下。以利为度。
制备方法：上为末，蜜和为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：犀、麝、雄黄等味辟除恶毒为主，明砂镇心安神，贝齿磨坚破血，桂、附鼓激诸药之性也。
用药禁忌：忌生葱、猪肉、野猪肉、芦笋、生血物。
附注：《千金》有甘遂，雄黄，无甘草。
