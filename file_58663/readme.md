序号：58678
名称：秘传神应眼药
组成：制炉甘石10两，黄连3两，黄柏3两，薄荷3两，甘草1两，朴消2两，上五味（口父）咀，用水数碗煎熟，滤去滓，用钵头盛之，将炉甘石研细，入银窝内，火煅深红，倾入药水内就研，飞过，净末听用。水银1两，黑铅1钱（上2味配作1块），白硼砂1钱，硇砂5分，食盐1两（用白水煮，炒干），枯白矾1两，皂矾2钱，火消（萝卜汁提过）7钱5分，上8味同1处，入阳城罐内，用盐泥固济干密，升打1炷官香，候冷取出听用。
出处：《直指附遗》卷二十。
主治：服目翳膜，昏朦。
加减：None
功效：None
用法用量：如点翳膜，加白硼砂5钱，硇1钱5分，火硝1钱5分，照前制过，名曰卷帘散。如点昏朦，加冰片，名日光明散。
制备方法：将前制炉甘石研细，再入5两在阳城罐底，次将前灵气7钱5分亦研细，上又安甘石5两，盖定，亦严封固，打1炷官香，冷定，出火毒，研极细，罗过。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
