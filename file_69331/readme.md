序号：69347
名称：黄蜡膏
组成：川连1钱，黄柏1钱，荆芥1钱，白芷1钱，紫草1钱，苦参1钱，郁金1钱，大枫子9个。
出处：《医部全录》卷三七四引叶心仰方。
主治：年久毒漏，杨梅痈疽，里外臁疮，身上烂者。
加减：None
功效：None
用法用量：None
制备方法：上作1贴，用麻油1盏，猪油1片如铜钱大，将药入油内同煎，煎至药滓黑色，用棕1片，滤去滓再煎，先放松香2钱熔化，后入黄蜡5钱，白蜡5钱，熔化，取起将碗盛著，再入后末药：轻粉1钱，枯矾5分，儿茶8分，自然铜（炼过，酢淬7次）6分，鳖壳（烧灰）2钱。五味共研末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
