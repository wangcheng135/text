序号：37366
名称：宝精丸
组成：白亮鱼胶8两（切作短块，用牡蛎8两炭火煅过，研末同炒。须炒得不可焦黑，黄色为度，去末用胶），熟地黄4两，山药3两，人参2两（虚甚加1两），沙苑蒺藜8两（酒洗，去衣，竹刀切开，去白膜），白茯苓4两（去皮，切片，入乳拌晒3次），牛膝3两（去芦，择粗壮者切碎，酒拌，微炒），甘州枸杞4两（去蒂与枯者，乳汁拌，晒干，如此者五次），鹿胶2两，菟丝子3两（水淘净，酒蒸熟，捣烂，晒干），山茱萸肉4两（酒拌，烘干），当归2两（去芦尾，取明亮者，酒洗，切片，晒干微炒）。
出处：《医学正印》卷上。
主治：None
加减：None
功效：种子，添精补髓，滋阴壮阳，健步明目益年。
用法用量：每服3钱，早晚淡盐汤送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
