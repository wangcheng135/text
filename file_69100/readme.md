序号：69116
名称：黄连磨积丸
组成：黄连1两（纳5钱吴茱萸同炒，5钱益智仁同炒，去2味不用，只用黄连），栀子仁5钱（炒，去杇），白芥子（醋浸炒）5钱，川芎1两，苍术1两（米泔浸7日），桃仁1两（去皮，存尖），青皮1两（去瓤），香附子1两（童便浸，炒），莪术1两（酒浸炒），山楂肉1两，莱菔子1两（炒，研），白术1两，三棱（用西安府者）1两5钱。
出处：《扶寿精方》。
主治：一切痰饮痰积，积聚拂郁，胁下闷倦，懒惰，饮食不消，或吐逆恶心，眩晕怔忡，时作时止。
加减：None
功效：破结磨坚，行气活血，消除积痞。
用法用量：每服50-70丸，以白汤送下。
制备方法：上为细末，汤浸蒸饼为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌热物，孕妇忌服。
附注：None
