序号：24350
名称：八风九州汤
组成：麻黄4两（去节），甘草（炙）3两，干姜3两，附子（炮）3两，防风3两，独活3两，石膏（绵裹）2两，茯苓2两，白术2两，芎2两，柴胡2两，当归2两，人参2两，杏仁40枚（去皮尖两仁），细辛2两。
出处：《外台》卷十四引《古今录验》。
主治：男子、妇人寒冷不自爱护，当风解衣，汗出卧冷湿地，半身不遂，手足苦冷，或不遂，或俯仰屈伸难，周身淫淫痹，四肢不收，状如风狂，饮食损少。
加减：None
功效：None
用法用量：上切。以水1斗，清酒5升，渍3夜。煮取4升，分为3服，1日令尽。若病人羸瘦者。用水煎服。药讫厚覆，当汗出微微，去上衣，汗解，以粉粉之。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌生菜、海藻、菘菜、酢、桃、李、猪肉、雀肉。
附注：None
