序号：51386
名称：贴敛药方
组成：麦饭石（粗磨石是也，曾作磨者尤佳。火煅7-8次，煅红入米醋中淬，煅至3-4次，其石定细研，用甘锅盛，煅通红淬醋中，煅过7-8次可用）、鹿角根（不用脑骨，不用角梢，只用角根3寸，火煅）、贝母（为末）各等分。
出处：《普济方》卷二九○。
主治：诸疮口不合。
加减：None
功效：合疮。
用法用量：先将旧净洁衣绢片洗净，候干，约疮大小，剪绢作1轮子，中留1小口，却用1小铫子热少米醋，约用多少，将前药投醋中，候冷，摊于绢轮子上，贴疮，1日1换。此方救人累效，不可疑药低贱而忽之。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
