名称：赵府神应比天膏
组成：当归2两，红花2两，生地2两，川芎2两，芍药2两，苏木2两，羌活1两，独活1两，蓬术（煨）1两，防风1两，荆芥1两，野菊花1两，骨碎补（去皮毛）1两，牙皂1两，苦参1两，牛膝1两，三棱（煨）1两，白蔹1两，山甲（炙）1两，续断1两，蝉蜕1两，全蝎（汤泡3次）1两，山豆根1两，地龙（去泥）1两，甘松1两，三奈1两，槐枝1两，柳枝1两，桃枝1两，榆枝1两，夏枯草1两，露蜂房1两，白果3个（去壳）,南星1两5钱,半夏1两5钱,男血余（皂角水洗）3两,胎发20丸，白花蛇1条（去头尾），桑白皮2两，连翘2两，金银花2两，川贝2两，山茨菇2两，木别仁2两，甘草2两，大黄2两，桃仁2两，杏仁2两，川连（去须）2两，首乌2两，五味2两，黄耆2两，合欢花2两，象皮2两，昆布（洗去盐味）2两，凤凰退2两，川附子1个，黄芩4两，射干（洗）4两，黄柏4两，乌药4两，玄参4两，五加皮4两，天麻4两，人参4两，大力子4两，肉桂4两，豨莶草4两（以上为粗药），雄黄2两，银朱6钱，朱砂2两，花蕊石2两（为粗末，用硫黄末2两搅匀，入阳城罐内封固，炼1日取出），石膏（煅）2两，赤石脂2两，自然铜2两（2味各入倾银罐内煅红，醋淬7次，埋土中1宿，去火气），云母石1两，乳香3两（同龙骨研），龙骨2两（照自然铜制），阿魏1两（同自然铜研），没药3两（炙，同赤石脂研），血竭2两5钱（同石膏研），儿茶2两（同云母石研），安息香5钱，珍珠5钱（同安息香研），丹珠1两（即人血，或用山羊血代），牛黄3两（同雄黄研），麝香4钱（同银朱6钱研），冰片2钱（同朱砂研），蝻蛇胆5钱（同雄黄研），沉香1两5钱，檀香1两5钱，丁香5钱，木香1两5钱，降香5钱（以上不用火），三七1两，苏合香2两5钱（以上为细末），黄蜡3两，白蜡3两，苏合油4两，淘鹅油4两。
出处：《惠直堂方》卷四。
主治：五劳七伤，遍身筋骨疼痛，腰脚软弱，腰痛，痰喘气息，咳嗽，小肠气，疝气，左瘫右痪，手足麻木、疟疾，男子遗精白浊，女人赤白带下，月经不调，血山崩漏，心气痛，偏正头痛，走气，寒湿脚气，无名肿毒，痈疽发背，对口及瘰疬臁疮，杨梅风毒，跌打损伤，指断臂折，痞块癥瘕。
加减：None
功效：接骨，化大毒。
用法用量：五劳七伤，遍身筋骨疼痛，腰脚软弱，贴两膏盲穴，两肾俞穴，两三里穴；腰痛，贴命门穴；痰喘气急，咳嗽，贴两肺俞穴，华盖穴，膻中穴；小肠气疝气，贴膀胱穴；左瘫右痪，手足麻木，贴两肩井穴，两曲池穴；疟疾，男贴左臂，女贴右臂即止；男子遗精白浊，女人赤白带下，月经不调，血山崩漏，贴阴交穴，关元穴；心气痛，贴中脘穴；偏正头痛，贴风门穴；走气，贴章门穴；寒湿脚气，贴两三里穴；一切无名肿毒，痈疽发背，对口及瘰疬臁疮，杨梅风毒，跌打损伤，指断臂折，痞块癥瘕，皆贴本病患处。
制备方法：真麻油15斤，将粗药浸，春5、夏3、秋7、冬10日，入锅，文武火煎枯，绢滤去滓，又煎油至滴水成珠，下淘鹅油、黄白蜡、苏合油，再下炒过黄丹7斤，柳枝搅匀，试其软硬得所，离火，下细药，冷定，沉水中3日，取起摊用。
临床应用：None
各家论述：None
用药禁忌：None
附注：比天膏（《膏药方集》）。
