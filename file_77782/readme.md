序号：77798
名称：雄朱蝎附散
组成：白芷1两，藁本1两，僵蚕（炒）1两，川乌（炮）1两，麻黄（去节）1两，南星（炮）3两，白附子（炮）半两，防风半两，雄黄2钱半，辰砂（并研）2钱半，蝎梢（炒）2钱半（一方有川芎、细辛、旋覆花、乳香、麝香）。
出处：《医方类聚》卷二十一引《管见大全良方》。
主治：一切风邪头疼，夹脑风气，痰涎壅盛，呕逆恶心，口吐清水，暗风旋晕，眼见黑花，牙关紧急，口眼斜，面目瞤动，头项拘急，肩背引疼，耳痒目昏，四肢麻木，及沐头浴出，暴感风邪，头目昏痛，两太阳穴疼，远年头风，经隔岁月，乍愈乍发，服诸药不效者。
加减：None
功效：None
用法用量：每服半钱，煎葱茶调下，食后服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：孕妇不可服。
附注：None
