序号：78403
名称：蜀椒汤
组成：蜀椒2合（汗），芍药3两，半夏（洗）2两，当归2两，桂心2两，人参2两，甘草（炙）2两，生姜汁5合，蜜1升，茯苓2两。
出处：《外台》卷三十四引《经心录》。
主治：产后心痛属大寒者。
加减：None
功效：None
用法用量：先以水9升煮椒令沸，下诸药，煮取2升半，去滓；下姜汁、蜜等更煎，取3升，1服5合，渐至6合，服尽为止。
制备方法：上切。
临床应用：None
各家论述：《千金方衍义》：产后心腹痛总宜温理血气，内补当归建中专和营解邪，产后恒宜服之。此以本气虚寒，故于方中加入蜀椒、人参、茯苓、半夏温散胃家痰气，生姜用汁取其迅速，蜜代饴、枣，不独安中，兼滋桂心、半夏之燥也。
用药禁忌：勿食冷物。
附注：None
