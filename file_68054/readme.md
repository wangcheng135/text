序号：68070
名称：黄耆汤
组成：黄耆半两，防风（去叉）半两，石斛（去根）半两，当归（焙）半两，白芷半两，藿香（择叶）半两，沉香半两，五味子半两，羌活（去芦头）半两，桂（去粗皮）半两，木香1钱，芎3钱，白蒺藜（炒，去角）3钱，桑寄生3钱，附子（炮裂，去皮脐）3钱，白术3钱。
出处：《普济方》卷十四引《护命方》。
主治：肝元虚冷，多困少力，口无滋味，耳鸣眼暗，面色青黄，精神不快。
加减：None
功效：None
用法用量：每服3钱，水1盏，枣1枚（擘破），煎1-2沸，去滓，空心、食前温服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
