序号：41984
名称：青龙膏
组成：上好羊脑炉甘石8两（打如莲子大，1分重为则。用新铜罐盛入童便，浸49日，滤去宿童便，更入新童便煮1柱香久，咬咸酸味，不必再煮，又不可煮老，研为细末，用缸片一大块，将药放在上，用硬炭火煅1柱香久，甘石渐渐转如松花色，细心谨慎取起，总称匀分作4份。取1份，用晚蚕沙3升，炒为灰，滚水淋灰汁大半钟，亦煮3次，候干）。
出处：《异授眼科》。
主治：胬肉攀眼，赤白翳膜烂眩；或年久云翳遮睛，不能行路，但见人影，如白衣人行，有血根攀睛者。
加减：None
功效：None
用法用量：点眼。
制备方法：上为细末，另用瓷罐收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：原书用本方治内障、胬肉攀睛、赤白翳膜烂眩者，须兑虎液、羊脑玉，冰片合用；治年久云翳遮睛，有血根攀睛者，先用本方兑羊脑玉点眼，直点至翳开之后，再用本方兑虎液，羊脑玉、凤麟、冰片、珍珠、琥珀合用。
