序号：33195
名称：万灵至宝仙酒
组成：淫羊藿（酒洗净，剪碎）10两，列当（如无，以肉苁蓉代之）4两，仙茅（糯米泔浸1宿，竹刀削去粗皮黑顶）4两，雄黄（研）2两，黄柏（去粗皮）2两，知母（去尾）2两，当归（酒洗，浸）8两。
出处：《医部全录》卷三三三引《身经通考》。
主治：肾阴阳亏虚，阳萎不举，妇人赤白带下，月水不调，肚冷脐痛，不孕。
加减：None
功效：生精益肾，助阳补阴。
用法用量：酒药同服，仍以干物压之。此酒用银壶或瓷壶重汤煮热服。酒后不可妄泻，待时而动，少则3月，多则半年，精泻胞成，屡试屡验。
制备方法：无灰酒15斤，装入瓶内封固，桑柴文武火悬煮3个时，埋地内3昼夜，去火毒取出，待7日将药捞出，晒干为细末，糯米粉打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌牛肉、铁器。宜阴脏人。
附注：None
