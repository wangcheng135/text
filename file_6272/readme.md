序号：6272
名称：去湿健脾汤
组成：党参1两，山药5钱，云苓5钱，薏米5钱，牛蒡子3钱（炒），连翘3钱，陈皮3钱，泽泻3钱，猪苓3钱，车前子3钱（外包），砂仁壳1钱，甘草1钱。
出处：《眼科临症笔记》。
主治：胞虚如球（非炎性眼睑水肿）。两眼珠微赤，稍痠不痒，无泪虚胀，不坚硬，皮不变色。
加减：None
功效：None
用法用量：水煎服；外以三物化坚汤罨之。
制备方法：None
临床应用：胞虚如球：昔濮县杨某某，女，37岁。时至中秋，忽觉四肢无力，眼胞沉涩，意为感冒所致，即服以姜葱发汗之物，至晨胞虚如球，不痒，稍觉微酸。六脉虚数，惟太阴为甚。此乃脾土虚热，清气下降，虚火上升。内服本方，外以三物化坚散罨之，2-3日即消。
各家论述：None
用药禁忌：None
附注：None
