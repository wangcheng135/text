序号：45575
名称：柏子仁丸
组成：柏子仁2两，白石英2两，钟乳2两，干姜2两，黄耆2两，泽兰9分（取叶，熬），藁本3分，芜荑3分，芎2两半，防风5分，蜀椒1两半（去目及闭口者，汗），人参1两，紫石英1两，石斛1两，赤石脂1两，干地黄1两，芍药1两，五味子1两，秦艽1两，肉苁蓉1两，厚朴（炙）1两，龙骨1两，防葵1两，细辛1两，独活1两，杜仲（炙）1两，白芷1两，茯苓1两，桔梗1两，白术1两，桂心1两，当归7分，甘草（炙）7分。
出处：《千金翼》卷七。
主治：妇人五劳七伤，羸弱瘦削，面无颜色，饮食减少，貌失光泽，及产后半身枯悴，伤坠断绝无子。
加减：None
功效：令人肥白。
用法用量：每服10丸，空肚温酒送下，不知，增至30丸，以知为度。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：禁食生鱼、肥猪肉、生冷。
附注：None
