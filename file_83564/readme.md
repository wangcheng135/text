序号：83582
名称：乌须大补丹
组成：何首乌1斤（铜刀切碎，黑豆3升，水泡，入甑内，与首乌层层铺盖，蒸1炷香尽，取出晒干，如此3次，听用），熟地2两，牛膝2两，故纸2两，当归2两，萆薢2两，苁蓉2两，琐阳1两，覆盆子1两，桑椹子1两，柏子仁1两，酸枣仁1两，没石子1两，川椒1两，小茴香1两，茯苓1两，巴戟5钱，百药煎5钱，槐角子5钱，青盐3钱，甘草3钱。
出处：《鲁府禁方》卷二。
主治：None
加减：None
功效：乌须发，壮阳事。
用法用量：上为末，石臼内不犯铁器，蜂蜜1碗，头生儿乳汁1碗，2味和匀，铜镟盛之，重汤煮3炷香，取出冷定，和药捣千下，不可间断1时。至20日，须发从根发黑。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌猪、羊、萝卜、豆腐。
附注：None
