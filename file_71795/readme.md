序号：71811
名称：登瀛散
组成：土茯苓2斤，防风1两半，荆芥1两半，五加皮1两半，白鲜皮1两半，木瓜1两半，当归（酒洗）1两，川芎1两，白芍1两，牛膝1两，生地黄（酒洗）1两，白茯苓（去皮）1两，杜仲（炒）1两，白芷1两，地骨皮1两，青藤1两，槐花1两，黄连1两，威灵仙1两半。
出处：《杏苑》卷八。
主治：远年梅疮，风漏，或筋骨疼痛，或近日初起。
加减：None
功效：None
用法用量：上(口父)咀，作10剂。每剂用水1钟半，酒1钟，煎至1钟，初一剂，加芒消、大黄各2钱，临熟下，离火，疮在上，食后服，在下，食前服。滓再煎服。余不用消黄，每日1帖。将逐日药滓晒干，积3-4次者，用锅煎熬，候温洗浴，初服5帖之内，疮势觉盛，乃毒气攻外，勿惧。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
