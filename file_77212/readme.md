序号：77228
名称：释麻丸
组成：白术2两（荷叶包蒸3次），广皮8钱，山楂5钱，川连5钱（吴茱萸拌抄，去萸），炙甘草8钱，黄芩（酒炒）7钱，木香1钱，枳实8钱（麸炒），苍术2两（茅山者佳，米泔浸晒3次，盐水拌匀），茯苓1两（人乳拌蒸），人参7钱，半夏8钱（白矾汤、姜同煮），当归1两（酒洗），白芍1两（酒拌炒），经霜桑叶4两（去筋）。
出处：《惠直堂方》卷一。
主治：指麻臂冷，忽觉一时昏瞆，及头晕眼花。
加减：None
功效：None
用法用量：空心服2钱，开水送下。
制备方法：上为细末，竹沥、姜汁、荷叶汁打神曲糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌猪首、鹅肉、发风动痰物。
附注：服此预防，一年后，可免中风之患。
