序号：56028
名称：桑菊感冒片
组成：桑叶800两，菊花320两，连翘480两，桔梗640两，杏仁640两，芦根640两，甘草256两，薄荷冰1两2钱8分。
出处：《中药制剂手册》。
主治：感冒风寒，发热头痛，咳嗽口渴，咽喉疼痛。
加减：None
功效：散温解表，清热止咳。
用法用量：每服4-8片，温开水送服，日2次。
制备方法：取桔梗512两，轧为细粉。取桔梗粗末与桑叶、菊花7倍量，用60％乙醇按澎漉法提取，浓缩稠膏约320两。将杏仁压榨去油，取杏仁饼约350两，用85％以上乙醇加热回流提取2次，浓缩膏约96两。取连翘、芦根用提取法提取2次，合并浓缩稠育约224两。粗提甘草酸，混合，制粒，压片。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
