序号：41515
名称：金液含化灵丹
组成：山泽银朱8两，朱砂1两5（金汁中浸5日了，逐块子用金箔裹两重）。
出处：《圣惠》卷九十五。
主治：None
加减：None
功效：去疾补益，延驻却老。
用法用量：每日空心含3丸，津液咽之。如要作油，每1两以桂心末1钱，大羊肾（月区）脂炼成者，如弹子大，入龙脑1钱，和研2日久，入银盒子中，埋于糠甑中，蒸三伏，当自化为油。每日含如豇豆大。
制备方法：上先铺银末1两于瓷盒子内，即排朱砂块子，勿令相着。上以银末盖之令匀，又市朱砂块子，又以银末盖之，候朱砂尽，即以盐花盖上，令满盒子口，实按如法，固济，入灰池中，盒子上灰厚4寸。常以2两火养7日7夜，勿令火猛，但令盒子热，可通人手为度。日满取出，重翻排过，一依前法重固济，以火4两养20日；后加火至2-3斤，烧可一炊，久放令极冷，取出细研，入龙脑半分，同研入粉，以糟汁和丸，如粟米大。
临床应用：None
各家论述：None
用药禁忌：忌羊血。
附注：None
