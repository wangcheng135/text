序号：31329
名称：沉香石斛丸
组成：沉香（锉）1两，石斛（去根）1两，人参1两，白茯苓（去黑皮）1两，菟丝子（酒浸1宿，别捣末）3分，山芋1两，麦门冬（去心，焙）1两，肉苁蓉（酒浸，切作片子，焙）半两，五味子3分，熟干地黄（焙）1两，百合3分，陈橘皮（汤浸，去白，焙）3分，枸杞子（焙）3分，黄耆（微炙，锉）半两，巴戟天（去心）半两，柏子仁（别研）3分，牛膝（酒浸，切，焙）1两。
出处：《圣济总录》卷五十四。
主治：三焦虚痞，心胸刺痛。
加减：None
功效：安和五脏，化痰利膈，止逆进食。
用法用量：每服15-20丸，空心、食前以温米饮送下；温酒送下亦得。
制备方法：上为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
