序号：78355
名称：蜀椒丸
组成：泽兰叶1两，芜荑仁（炒黄色）半两，石膏（火煅）半两，白芷半两，蜀椒（去目并合口者，炒出汗）半两，干姜（炮裂）半两，藁本（去苗）半两，厚朴（去粗皮，生姜汁炙）半两，人参半两，白术半两，细辛（去苗叶）半两，桂（去粗皮）半两，防风（去芦）半两，当归（切，焙）半两，芎半两，甘草（炙赤）半两，柏子仁（炒令黄）半两。
出处：《普济方》卷三五二。
主治：产后虚羸，月内不快，颜色萎黄，四肢无力。
加减：None
功效：调补气血。
用法用量：每服30丸，温酒或米饮送下，不拘时候。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
