序号：81108
名称：麝香荜澄茄丸
组成：麝香（细研）半两，硫黄（细研）3分，硇砂（不夹石者，细研）1分，石斛（去根）1两，荜澄茄1两，茴香子（炒）1两，补骨脂（炒）1两，木香1两，何首乌1两半，丁香1两，肉豆蔻（去壳）1两，桂（去粗皮）1两，当归（切焙）1两，吴茱萸（汤浸7遍，焙干，炒）1两，槟榔（锉）1两。
出处：《圣济总录》卷四十五。
主治：脾脏冷气，攻心腹撮痛，手足逆冷，霍乱呕吐，脏腑滑利，膈脘痞塞，不思饮食。
加减：None
功效：None
用法用量：每服20-30丸，空心、食前温酒送下；米饮亦得。
制备方法：上为末，入研药拌匀，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
