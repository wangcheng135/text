序号：50585
名称：茸珠丹
组成：鹿角胶（炒珠子）1两，鹿茸（去皮毛，切片，酥炙；无酥用浊酒炙）1两，鹿角霜1两，阳起石（煅，酒淬）1两，大附子（炮，去皮脐）8钱，当归（去芦尾）8钱，地黄（九蒸九焙）8钱，辰砂（别研）半钱，肉苁蓉1两，酸枣仁（去壳，捣膏）1两，柏子仁（去壳，同枣仁捣膏）1两，黄耆（蜜炙）1两。
出处：《袖珍方》卷二引《澹寮》。
主治：None
加减：None
功效：补精益血。
用法用量：每服50丸，空心温酒、盐汤任下。用干物压之为妙。
制备方法：上为末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：斑龙丸（原书同卷引澹寮）、茸珠丸（《普济方》卷二一九）。
