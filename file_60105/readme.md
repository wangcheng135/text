序号：60120
名称：透红丸
组成：缩砂仁100粒，杏仁100粒（去皮尖），巴豆50粒（去皮膜，取霜），坯子、胭脂1钱（别研），川芎1两（锉碎）。
出处：《杨氏家藏方》卷五。
主治：脾疼翻胃，膈气，水泻积痢；及妇人血气刺痛。
加减：None
功效：None
用法用量：大人每服2丸，小儿每服3丸；脾疼，石菖蒲汤送下；妇人血气刺痛，醋汤送下；翻胃膈气，丁香汤送下；水泻，倒流水送下；赤痢，甘草汤送下；白痢，干姜汤送下；赤白痢，甘草、干姜汤送下。空心、食前服用。
制备方法：上为细末，次入杏仁、巴豆、胭脂研匀，汤泡雪糕糊为丸，如梧桐子大；小儿为丸如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
