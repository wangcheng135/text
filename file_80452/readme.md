序号：80468
名称：鳖甲丸
组成：鳖甲（淡醋煮，去裙膜，洗净，酸醋炙黄）半两，猬皮（炙黄焦，锉）半两，穿山甲（锉，碎，蚌粉炒）半两，白矾（枯）半两，附子（炮，去皮脐）半两，猪牙皂角（炙焦存2分性）半两，麝香1分（研）。
出处：《本事》卷五。
主治：肠痔。
加减：None
功效：None
用法用量：每服20丸，食前米饮送下，每日3次。
制备方法：上为细末，蒸饼为丸，如梧桐子大。
临床应用：None
各家论述：《本事方释义》：鳖甲气味咸平，入足厥阴；猬皮气味甘平，入足厥阴、手阳明；穿山甲气味咸寒，入足厥阴、阳明；白矾气味凉涩，入手足太阴、阳明；附子气味辛咸大热，入手足少阴；猪牙皂角气味辛温开窍，入手足太阴、阳明；麝香气味辛温，入手足少阴。蒸饼糊丸引入下也，治肠风而有痔者。以咸辛平之药直入患处，以凉涩辛咸温之药佐之；又恐药性之不能即至病所，复使以辛香之品引入经络，焉有不中病者哉。
用药禁忌：None
附注：None
