序号：45398
名称：枸杞子丸
组成：枸杞子2两，牛膝1两（去苗），熟干地黄2两，漏芦3分，当归3分（锉，微炒），酸枣仁3分（微炒），人参1两（去芦头），防风3分（去芦头），羚羊角屑3分，桂心3分，白茯苓1两，黄耆1两（锉），羌活3分，麦门冬1两（锉，去心，焙），五加皮3分，白术3分，芎3分，甘草半两（炙微赤，锉）。
出处：《圣惠》卷八十一。
主治：产后风虚劳损，四肢疼痛，心神虚烦，不欲饮食。
加减：None
功效：None
用法用量：每服30丸，以温酒或荆芥汤送下，不拘时候。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
