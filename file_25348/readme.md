序号：25356
名称：竹沥汤
组成：竹沥5合，黄芩30铢，木防已6铢，羚羊角6铢，大黄2两，茵芋3铢，麻黄半两，白薇半两，桑寄生半两，萆薢半两，甘草半两，白术6铢（1方作白鲜）（1方无萆薢）。
出处：《千金》卷五。
主治：小儿头身患小疮，咳嗽，经治后变痫证，四肢缩动，背脊（身夭）（身兆），眼反。
加减：None
功效：None
用法用量：以水2升半，煮取药减半，加竹沥，煎取1升，分服2合，相去1食久，进1服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：咳嗽有六淫七情，经络脏腑，种种不侔，此专因胎毒发疮后变发痫。与竹沥治痰涤热，虽得小疏，然非吐下不能大朽其势，故用麻黄、白薇开发肺气于上，即用大黄、黄芩、竹沥疏利大肠于下，羚羊、防已、萆薢、寄生、茵芋专为热毒发痛而设，白术、甘草和中实脾，师旅之粮饷也。
用药禁忌：None
附注：小儿出胎200许日，头身患小小疮，治获小瘥，复发，五月中忽小小咳嗽，微温和治之，因变痫，一日20过发，四肢缩动，背脊（身夭？？）（身兆），眼反，须臾气绝，良久复苏，已与常治痫汤，得快吐下，经日不间，尔后单与竹沥汁，稍进，一日一夕中合进一升许，发时小疏，明日与此竹沥汤，得吐下，发便大折，其间犹稍稍与竹沥汁。
