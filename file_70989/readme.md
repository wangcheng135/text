序号：71005
名称：童叟卫生丹
组成：槟榔2两，郁金1两，三棱（炒）1两，莪术（炙）1两，于术1两，鲜姜1两，丁香1两5钱，鸡内金（炒）1两（共熬膏），莲子肉1两，朱砂（另研）2钱，茯苓1两，山药5钱，法半夏1两，使君子肉1两，冰片（另研兑）2钱，泻叶6两，柿霜3两。
出处：《北京市中药成方选集》。
主治：积滞不消，腹胀肚痛，面黄肌瘦，消化不良；及小儿积滞痞块，妇女癥瘕血块。
加减：None
功效：消食化滞，开胃健脾。
用法用量：每服2丸，温开水送下。小儿每服1丸。
制备方法：上除朱砂、冰片外，共研细末，过罗、兑入上膏内，冰片、朱砂细粉混匀，用膏合蜜为丸，重5分。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
