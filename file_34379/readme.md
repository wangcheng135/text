序号：34388
名称：万春膏
组成：桑枝4斤，槐枝4斤，柳枝4斤，麻油4斤，生大黄1两半，白芷8钱，当归8钱，红花8钱，防风8钱，羌活8钱，独活8钱，生香附8钱，南星8钱，木瓜8钱，佛手8钱，乳香8钱，没药8钱，沉香8钱，丁香8钱，木香8钱，白芥子2钱，肉桂5钱，麝1钱。
出处：《理瀹》。
主治：肝胃气，痞块，癥，鹤膝，疝气，脾虚泄泻，一切内癥疼痛，跌扑闪挫，风气。
加减：如火衰泄泻，加硫黄：痞，加米炒斑蝥（去头足）。
功效：None
用法用量：掺贴。
制备方法：将桑、槐、柳枝入麻油内熬，以铅粉收膏，桃枝搅。余药研末，和入膏内，忌火。
临床应用：None
各家论述：None
用药禁忌：None
附注：原书注云：原方有黄耆、川乌、牛膝、麻黄、茜草，无香附、木瓜、佛手，此从张刻本减，然原方力大。
