序号：77917
名称：豁脾煎
组成：苍术1钱2分，藿梗1钱，当归1钱，厚朴5分（炒），神曲2钱，楂肉2钱，广皮7分，茯苓1钱5分，赤芍1钱5分，炙草4分，广木香4分（磨汁，冲服）。
出处：《古方汇精》卷一。
主治：痢疾。
加减：如腹痛，脐下急痛，按之愈痛，此实热痢疾也，加生大黄1钱5分，可投2剂，取通利而愈；如痛不拒按，喜得热物，渴恶冷饮，此冷痢也，加附子4分（制熟用），滑石1钱5分，车前子8分，取和解而愈。
功效：None
用法用量：引加煨黑姜1片，粟壳4分。愈后服香砂六君子汤3-5剂，加慎调理。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
