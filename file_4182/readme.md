序号：4182
名称：木香三棱丸
组成：木香1两，丁香1两，砂仁1两，红豆1两，姜屑（炒）1两，甘松（水洗）1两，良姜1两，厚朴（姜制）1两，香附子（炒）1两，枳实（炒）1两，枳壳1两，萝卜子1两，荆三棱1两半，石三棱1两半，鸡爪三棱1两半，槟榔1两半，青皮（去瓤）1两半，陈皮（去白）1两半，莪术（醋炙）2两，神曲（炒）2两，麦芽（炒）2两，甘草（炒）2两，牵牛（炒）8两，苍术（泔浸一半，醋浸一半）8两，荜澄茄半两，白豆蔻半两，雷九半两，青木香半两，藕节半两。
出处：《普济方》卷一七一。
主治：胸膈痞闷，心腹胀满，胁肋疼痛，饮食迟化，四肢困倦，呕逆恶心，口苦无味，积滞冷物，不思饮食；癥瘕痃癖，坚硬气块；小儿伤食。
加减：None
功效：None
用法用量：每服40-50丸，食后温生姜汤送下。
制备方法：上为细末，滴水为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
