序号：82635
名称：澡豆
组成：白芷7两，芎5两，皂荚末4两，萎蕤5两，白术5两，蔓荆子2合，冬瓜仁5两，栀子仁3合，栝楼仁3合，荜豆3升，猪脑1合，桃仁1升（去皮），鹰屎3枚，商陆3两（细锉）。
出处：《外台》卷三十二引《崔氏方》。
主治：None
加减：None
功效：悦面，色如桃花，光润如玉，急面皮，去(黑干)(黑曾)粉刺。
用法用量：每洗面用浆水，以此丸当澡豆，用讫，敷面脂，如常粧餙，朝夕用之，亦不避风日。
制备方法：上为末，其冬瓜仁、桃仁、栀子仁、栝楼仁别捣如泥，其猪脑、鹰屎合捣令相得，然后下诸药，更捣令调，以冬瓜瓤汁和为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
