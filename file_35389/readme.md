序号：35399
名称：京三棱汤
组成：京三棱（煨，锉）1两半，大腹（连皮子，锉）1两半，延胡索1两半，天雄（炮裂，去皮脐）1两半，芎1两半，白术1两半，桃仁（汤浸，去皮尖双仁，炒）30枚，桂（去粗皮）1两，当归（切，焙）1两，消石1两，郁李仁（汤浸，去皮）1两1分。
出处：《圣济总录》卷七十二。
主治：腑脏不和，气血留滞，积聚胀满，心腹妨闷，食物减少，烦闷短气。
加减：None
功效：None
用法用量：每服4钱匕，水1盏半，加生姜2片，煎至7分，去滓温服。
制备方法：上(口父)咀，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“荆三棱煎汤”。
