序号：40513
名称：虎骨鹿茸丸
组成：虎胫骨1对（或14两，如无，以胶3两代之），鹿茸1对（羊酥蒸炙，如无，以胶4两代之），枸杞子1两，小茴（酒炒）3两，菟丝子3两，巴戟肉（酒炒）3钱，刺蒺藜（酒炒）2两，破故纸（盐水炒）1两5钱，肉桂1两5钱，陈皮1两，威灵仙1两5钱，防风1两，淫羊藿（羊油炙）3两，杜仲（姜汁炒）3两，全蝎梢（酒洗淡，炒）3钱，归身（酒炒）3两，川萆薢1两，龟甲（醋炙）2两。
出处：《胎产秘书》卷下。
主治：产后瘫痪。
加减：None
功效：None
用法用量：不拘大人、小儿，每服5钱，以绍酒浸红花、蕲艾少许送下。
制备方法：上为末，各胶熔化，将鹿筋（如无，牛筋可代）1斤炖烂化，捣如泥；再用米仁1斤，炒研末，打稠糊，和饴糖3斤溶化，与前各药和匀为丸。
临床应用：None
各家论述：None
用药禁忌：忌生冷。
附注：None
