序号：26993
名称：羊肉发药
组成：威灵仙1两，蝉衣3钱，川芎3钱，当归3钱，麻黄（春用7钱，夏用5钱，秋用9钱，冬用2两2钱）。
出处：《医宗说约》卷六。
主治：杨梅疮初起，皮肤燥痒，上身为多。
加减：None
功效：None
用法用量：先用羊肉1斤，水10碗，煎7碗，去羊肉，入前药于汁内，煎至3碗，去滓，1日内分3服，迨尽，即将羊肉吃完，水酒过口。用被盖取汗，或吐或泻，至3-4次方好，以粥补住。内毒自解，外毒自发。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌牛、狗、鸡、鹅、火酒、茶、醋等物，每早食仙枣三枚；如病人不能饮酒，或虚极者，以胰子汤。
附注：None
