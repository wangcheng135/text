序号：40611
名称：虎睛汤
组成：虎睛1具，茯苓3两，桂心3两，防风3两，独活1两，甘草1两，人参1两，天雄1两，露蜂房1具，鸱头1具，石长生10分，枫上寄生5分。
出处：《千金》卷十四。
主治：狂邪发无常，披头大唤，欲杀人，不避水火。风狂叫笑不时，喜怒无常，登高（俞）垣，言语不避人。
加减：None
功效：None
用法用量：上(口父)咀。以水1斗2升，煮取3升，分4服，日3夜1。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：虎睛、参、苓定魄安神，天雄、桂心、鸱头、蜂房、石长生辟除阴毒，余皆截风之味。
用药禁忌：None
附注：本方改为散剂，名“虎睛散”（见《圣惠》）。《圣济总录》有当归；鸱头，作“鸱头并肝”，用法云：每服时须去食稍远，恐药食相犯也。
