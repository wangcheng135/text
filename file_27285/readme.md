序号：27293
名称：补肺百花煎
组成：生地黄汁1升，生姜汁半升，黄牛乳1升半，藕汁1升，胡桃瓤10枚（研如糊），干柿5枚（锉细，研如糊），大枣21枚（煮去皮核，研如糊），清酒1升（以上数味1处入银锅中，煎候沸，方下后药），黄明胶（炙燥为末）半两，秦艽末半两，杏仁（汤浸，去皮尖双仁，炒，研如糊，入煎中）3两。
出处：《圣济总录》（人卫本）卷六十八。
主治：吐血不止，咳嗽。
加减：None
功效：None
用法用量：每服1匙头，糯米饮调下，酒下亦得，日3次。
制备方法：上十一味，相次下，煎、减一半，却入上色蜜4两，徐徐着火，养成煎后，入瓷盘中盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，原书文瑞楼本引作“百花煎”。
