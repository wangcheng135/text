序号：57039
名称：消毒散
组成：大黄1两，芙蓉叶（晒干为末）1两，麝香3分，冰片3分，五倍子1两，藤黄3钱，生矾3钱。
出处：《石室秘录》卷四。
主治：多骨疽，痈疽疖毒。
加减：None
功效：None
用法用量：米醋调成如厚糊一样，涂于多骨疽之左右四周。以药围其皮肉，中留一头如豆大，以醋用鹅翎不时扫之。
制备方法：上药各为末。
临床应用：None
各家论述：《串雅内编选注》：方中大黄、芙蓉叶清热解毒，麝香、冰片活血散瘀止痛，五倍子、藤黄、生矾有消肿止痛之功。凡痈疽疖毒初起，红肿痛热者，用醋调局部四周，有止痛消肿功效。
用药禁忌：此方只宜于痈疖等阳毒，但不可施之阴症，似于多骨疽不甚相宜。
附注：None
