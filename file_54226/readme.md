序号：54241
名称：大建中汤
组成：人参（去芦）2两，粉草（炙）2两，龙齿（研）3两，当归（酒洗，去芦）3两，酸枣仁（去皮）3两，黄耆（去芦）3两，白芍药4两，远志（去心）1两半，白茯苓（去皮）1两半，石莲肉（去心）1两半，泽泻1两半。
出处：《医方类聚》卷一五○引《管见大全良方》。
主治：思虑太过，心气耗弱，阳气流散，精神不收，阴无所归，小便或赤或白，甚则尿精滑数，夜梦鬼交，或睡而汗出，日渐瘦悴，或生虚热，六脉虚弱，或大而软，按之不应。
加减：None
功效：None
用法用量：每服3钱，水1盏，加生姜3片，大枣2枚，煎至7分，去滓，入饧少许，再煎溶，空心温服，1日2次。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
