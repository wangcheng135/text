序号：60158
名称：透骨丸
组成：附子1枚（重7钱者，面裹炮裂，去皮脐），麒麟竭30枚，白僵蚕（炮）30枚，地龙（白项者，炒）30枚，干蝎（全者）30枚（炒），草乌头（去皮）2两半，五灵脂1两，骨碎补（干者，去毛）1两，麻黄（去根节）1两（以上9味同为末），龙脑半钱，麝香半钱，丹砂1分，水银1分，消石1分，甜消1分，朴硝1分，马牙消1分，乳香1分，没药1分（以上10味同研细）。
出处：《圣济总录》卷八。
主治：风脚软，不能行步。
加减：None
功效：None
用法用量：上拌和令匀，又用飞罗面2两，无灰酒2两，煮糊为丸，每1丸重1钱3字，以布袋盛，挂当风处，年深不妨。若脚手麻软，每服半丸，滴酒研烂，用生姜汁1茶脚，麝香少许，好酒1盏相和温过，早、晚各1服。5日见效。若卒中暗风，取2丸研入生姜汁半盏，麝香半字，好酒4盏，分温4服，初服如一食久，下第2服；渐见手开，即下第3服，见通身汗出，即下第4服，若口开不能合， 不可治。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：方中麒麟竭用量原缺。
