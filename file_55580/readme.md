序号：55595
名称：桃仁煎
组成：桃仁1200枚。
出处：《千金》卷三。
主治：妇人产后百疾，诸气。痃癖，心腹疼痛，肌肤瘦弱，面无颜色。
加减：None
功效：补益悦泽。下三虫。
用法用量：纳汤中煮1伏时不停火，亦勿令火猛，使瓶口常出在汤上，无令没之，熟讫出，温酒服1合，1日2次，丈夫亦可服。
制备方法：捣令细熟，以上好酒1斗5升，研滤3-4遍，如作麦粥法，以极细为佳。纳长项瓷瓶中，密塞，以面封之。
临床应用：None
各家论述：《千金方衍义》：桃仁虽能逐瘀，然随乌药、莪术则专于破血；随芎、归、芍药则相胥和血。若单用1味，破之与和惟在多用少用之间。兼之以酒，为产后和血圣药。但所禀柔脆，坐草无伤，无所留滞者不在此例。
用药禁忌：None
附注：桃仁酒（《圣惠》卷九十五）。
