序号：40677
名称：败毒和中散
组成：连翘7分（去心，研碎），防风7分，荆芥7分，黄连1分（不用酒炒），牛蒡子7分（拣净，炒香，研碎），桔梗6分，陈枳壳5分（炒），前胡1钱，紫草茸5分（酒洗，研末），小川芎7分，升麻3分，木通7分，蝉蜕12只，如有闰月13只（洗净，晒，去头足），麦门冬1钱（去心），甘草2分（生，去皮）。
出处：《痘疹传心录》卷十九。
主治：发热1日，即见痘苗，极痘稠密，大热蒸蒸，烦躁昏沉，不省人事，唇焦口渴。
加减：若毒火内盛，大便结燥，2-3日不解，小便短涩又黄，加酒炒大黄8分；毒火盛，加酒炒黄芩2分。
功效：清热解毒疏利。
用法用量：加生姜1片，灯心50寸，水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
