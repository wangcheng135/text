序号：52266
名称：顺气和中汤
组成：半夏（制）6分，白茯苓7分，白术（土炒）8分，广皮（盐水浸，炒）1钱，枳实（麸炒）5分，甘草（炙）2分，香附（醋炒）1钱，山栀（姜汁炒黑）1钱，神曲（炒）6分，砂仁（炒）3分，黄连（姜汁浸，晒干，以猪胆汁拌炒）6分。
出处：《古今医鉴》卷五。
主治：呕吐翻胃，嘈杂吞酸。
加减：心胃痛，加姜汁3匙；气虚，加人参、黄耆各8分；血虚，加当归7分，川芎5分；恼怒或气不伸舒，加乌药5分，木香3分；胸膈饱闷，加萝卜子（炒）6分；心下嘈杂吞酸，加吴茱萸4分，倍黄连、白术；呕吐不止，加藿香7分；大便团结，加苏子、麻仁、桃仁、杏仁，俱研如泥，每服1钱，白蜜调下。
功效：None
用法用量：先以长流水入娇泥搅，澄清，取水1钟，加生姜3片，煎至7分，入竹沥、童便、姜汁，细细温服，不拘时候。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
