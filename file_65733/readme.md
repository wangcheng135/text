序号：65749
名称：菝葜八味汤
组成：菝葜3两，土瓜根3两，黄耆4两，地骨皮4两，五味子4两，人参3两，石膏8两（碎），牡蛎3两。
出处：《外台》卷二十七引《许仁则方》。
主治：消渴小便数。肺消饮少溲多；冷淋寒颤涩痛。
加减：None
功效：None
用法用量：以水1斗，煮取3升，去滓，分3次温服，每服如人行10里服1剂，服至5-6剂佳。隔5日服1剂，剂数满，宜合黄耆十四味丸服之。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：None
附注：菝（艹吉瓜）散（《圣惠》卷五十八）、黄耆汤（《圣济总录》卷四十八）、菝葜饮（《圣济总录》卷五十八）、菝葜汤（《普济方》卷一七八）。
