序号：75
名称：五味子散
组成：五味子3分，半夏半两（汤洗7遍，去滑，焙过），紫菀半两（洗去苗土），枇杷叶半两（拭去毛，炙微黄），前胡3分（去芦头），陈橘皮3分（汤浸，去白瓤，焙），桔梗半两（去芦头），杏仁半两（汤浸，去皮尖双仁，麸炒微黄），诃黎勒皮3分，赤茯苓3分，枳壳半两（麸炒微黄，去瓤），甘草半两（炙微赤，锉）。
出处：《圣惠》卷七十。
主治：妇人心胸痰壅，时有喘促，咳嗽，不欲饮食。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，煎至6分，去滓，不拘时候温服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
