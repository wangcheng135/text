序号：46594
名称：千金贴痞膏
组成：黄丹10两（水飞7次，炒紫色），阿魏3钱，乳香3钱，没药5钱，两头尖5钱，当归3钱，白芷5钱，川山甲10片，木鳖子10个，麝香1钱。
出处：《寿世保元》卷三引薛兵巡方。
主治：积聚。
加减：None
功效：None
用法用量：None
制备方法：上为细末，用香油2斤，槐2尺4寸、桃2尺4寸、柳2尺4寸、桑2尺4寸、榆2尺4寸，巴豆120个，去油壳，蓖麻子120个，去壳。先将铁锅盛油，炭火煎滚，入巴豆、蓖麻在内，熬焦，捞去滓，次下前药，用桃、柳等条，不住手搅匀，然后下丹，滴水成珠为度，瓷器收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
