序号：26874
名称：竹沥汤
组成：秦艽2两，甘草2两（炙），防风2两，当归2两，茵芋1两，乌头1两（炮），干姜1两，细辛1两，人参1两，黄芩1两，桂心1两，天雄1两（炮），木防已1两，茯苓1两，白术1两。
出处：《外台》卷十四引《深师方》。
主治：大虚挟风，及贼风入腹，腹中拘痛，烦乱恍惚，妄语迷惑不知人，口噤不开，手足缓纵，饮食不作肉，卧惊见屋中光，口干恶风，时时失精，梦寤沉重，及妇人产后余病，体虚受风，躁愦欲死。
加减：胸胁满，加前胡2两半，半夏2两（洗），术、附（炮）各1两；腹中病，加芍药2两，椒1两；汗烦，加知母1两；口干，加麦门冬1两（去心）；体痹，加麻黄2两（去节）。
功效：None
用法用量：以竹沥1斗半，煮取5升。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：忌海藻、菘菜、猪肉、冷水、生葱、生菜、桃、李、雀肉、酢物等。
附注：None
