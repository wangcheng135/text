序号：44366
名称：一粒金丹
组成：大川乌头（炮，去皮脐）2两，大黑附子（炮裂，去皮脐）2两，新罗附子（炮裂）2两，五灵脂1两，白僵蚕1两（去丝，炒），白疾藜1两（炒，去刺），没药半两（研），朱砂半两（研），白矾1两（枯，研），麝香半两（净肉，研），细香墨半两，金箔200片。
出处：《洪氏集验方》卷一引张真甫方。
主治：中风瘫痪，口眼歪斜，涎潮语涩，浑身疼痛，及一切风疾。癫痫倒卧，目瞑不开，涎盛作声，或角弓反张，目睛直视，口噤闷绝，牙关紧急；风搏于阳经，目眩头痛，耳作蝉鸣，皮肤瞤搐，频欠喜睡，项强拘急，不能回顾；及肾脏风虚，脚膝疼痛，步履艰难，偏风流注，屈伸不得，无问新久。
加减：None
功效：补益五脏，固密真元，通流关节，祛逐风邪，壮筋续骨。
用法用量：每服1丸，用生姜1两去皮，榨取自然汁，将药丸子于汁内磨化尽，用无灰热酒1大盏，同浸化，温服。更量性多少，吃温酒1-2升，投之以助药力，用衣被盖覆，汗出为效。痛势重者，日进2服。不拘时候。
制备方法：上药前六味同为末，后四味研匀，同合和令匀，用井花水1大盏，研墨尽为度，将汁搜和，杵臼内捣五百下，丸如弹子大，金箔为衣，窨干。
临床应用：None
各家论述：None
用药禁忌：忌发风物，孕妇不可服。
附注：大圣一粒金丹、保命丹（《百一》卷三）。
