序号：31388
名称：沉香猪肚丸
组成：沉香3两，丁香3两，木香3两，川椒（炒）3两，荜澄茄3两，陈皮3两，葫芦巴（炒）3两，破故纸（炒）3两，石茱萸3两，桂3两，巴戟（去心）3两，茴香（炒）3两，牛膝3两，肉苁蓉3两，附子（炮，去皮脐）3两，槟榔4两，肉豆蔻4两。
出处：《百一》卷四。
主治：男子、妇人久病气虚。
加减：None
功效：None
用法用量：每服50丸，以温酒送下；妇人以醋汤送下，与壮气丸相间服。
制备方法：上为细末，生猪肚一个去脂，先用生绢袋盛药末，令在猪肚内缝合，用酸浆水一桶于银石锅内煮令猪肚软，取出放冷，不用猪肚，将药焙干，酒面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
