序号：63510
名称：清荣养血丸
组成：当归身（酒浸，洗去土，晒干）1两，川芎（茶浸洗）7钱5分，白芍药（酒浸，微炒）1两，熟地黄（酒洗）1两，陈皮（去白用红）1两，白术（去梗）1两，生条黄芩2两，知母（去毛）1两，陈艾叶5钱，黄柏（生，炒）2两，泽兰叶1两，香附子（肥大沉实者）4两（分4份，醋、酒、童便、米泔水制俱妙）。
出处：《摄生众妙方》卷十。
主治：月水不调，紫黑成块，频并，不及期，烦热腰困，手足酸痿。
加减：None
功效：None
用法用量：每服50丸，空心米汤或淡醋汤下。
制备方法：上为末，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
