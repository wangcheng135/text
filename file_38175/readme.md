序号：38185
名称：松脂散
组成：松脂30斤。
出处：方出《千金》卷七，名见《外台》卷十九。
主治：一切风，及大风、脚弱风痹。
加减：None
功效：None
用法用量：每服方寸匕，1日2次，初和药以冷酒，药入腹后饮热酒。行药以知为度，如觉热即减；不减，令人大小便秘涩。若涩，宜食葱羹，仍自不通，宜服生地黄汁，令取泄痢。服松脂30日后，即觉有验，两脚如似水流下是效。如恐秘涩，和1斤松脂、茯苓与大枣，栗许大酥，即不涩。服经100日后，脚气当愈。
制备方法：以棕皮袋盛，系头，铛底布竹木，置袋于上，以石三五颗压之，下水于铛中令满，煮之，膏浮出得尽以后量，更20沸，接置于冷水中，易袋洗铛，更煮，如此9遍药成，为粗散。
临床应用：None
各家论述：None
用药禁忌：除忌大麻子以外，无所禁。
附注：None
