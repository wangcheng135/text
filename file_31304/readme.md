序号：31313
名称：沉香如意丸
组成：沉香3分，檀香3分，丁香3分，木香3分，全蝎3分，茴香3分，青盐3分，木通5钱，山药5钱，穿山甲（炙）5钱，韭子（酒浸）5钱，莲花蕊5钱，五味子5钱，白茯苓5钱，陈皮5钱，鹿茸（炙）5钱，山茱萸5钱，小茴香1两半，川楝子（去皮）1两半，葫芦巴1两半，破故纸（羊肠煮）1两半，巨胜子（炮）1两，菟丝子（酒浸）1两，肉苁蓉（酒浸）1两，知母1两，远志（酒浸）1两。
出处：《普济方》卷二二二。
主治：脐腹痛，小便滑，房室不举。
加减：None
功效：补虚壮阳，暖水脏，益精髓。
用法用量：每服20丸，空心以温酒送下，干物压之。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
