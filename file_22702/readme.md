序号：22710
名称：劫风酒
组成：防风2两，秦艽2两，川萆薢2两，当归2两，晚蚕沙2两，虎胫骨（酥炙）2两，人参1两，枸杞6两，豨莶草6两，苍耳子4两，茄根8两，松节1两，海风藤1两，桑寄生1两，鳖甲1两，川牛膝1两，苍术1两，陈皮1两，明天麻1两，羌活1两，杜仲（姜炒去丝）1两，生地1两，川芎1两，白芍1两，木瓜1两，五加皮1两，南星（姜制）1两，贝母1两，半夏（姜制）1两，薄桂1两，甘草节1两，木香1两。
出处：《万氏家抄方》卷四。
主治：诸风缓纵，手足挛急，行步艰难，一切风证。
加减：None
功效：None
用法用量：None
制备方法：上锉碎，绢袋装。用老酒一大坛约40斤，入药蒸1炷香，取起，过百日服。
临床应用：None
各家论述：None
