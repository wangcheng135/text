序号：80496
名称：鳖甲丸
组成：鳖甲1两半（涂醋，炙令黄，去裙襕），露蜂房1两（锉碎，微炒），牡丹3分，川椒3分（去目及闭口者，微炒去汗），川大黄1两（锉碎，微炒），牛膝3分（去苗），附子1两（炮裂，去皮脐），吴茱萸3分（汤浸7遍，焙干，微炒），干姜3分（微炒），虻虫1两（微炒），水蛭1两（微炒），皂荚半两（去皮子，涂酥，炙令黄），当归1两（锉，微炒），赤芍药1两，桂心1两，琥珀1两，防葵1两，蛴螬20个（微炒）。
出处：《圣惠》卷七十一。
主治：妇人腹中积聚。大如杯，上下周流，痛不可忍，食噫腥臭，四肢寒热，经水不通，恶血停滞，体瘦无力，面色萎黄。
加减：None
功效：None
用法用量：每服10丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
