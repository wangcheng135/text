序号：83331
名称：七乌丸
组成：草乌（去皮脐，生用，切作大块）2两，何首乌（作大片，忌铁器）2两，乌药（细切）2两，川乌2两，乌梅50个（捶碎），黑豆半斤（揀，洗净），猪牙皂角2两（去皮，切作半寸）。
出处：《普济方》卷一八五引《仁存方》。
主治：风湿寒痹，瘫疾，风湿腰痛，脚气。
加减：None
功效：None
用法用量：每服50-70丸，空心盐汤送下，病在上食后，病在下食前，酒吞；脚气，木瓜汤送下，小疮疡数服效，与活络丹相间服之。
制备方法：上用无灰酒2升，米醋2升，同上药1处，装于瓷瓶内，浸2宿，用文武火煮，约存1升药汁，取出焙干，为细末，仍用药汁打糊为丸，如梧桐子大；或药汁少，益以酒醋。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
