序号：27237
名称：补肾汤
组成：磁石（水飞，研，淘去赤汁）2两，附子（炮裂，去皮脐）2两，黄耆（锉）1两，五味子1两，当归（切，焙）1两，白茯苓（去黑皮）1两，石斛（去根）1两，芍药1两，人参1两，沉香1两，桂（去粗皮）1两半，陈橘皮（汤浸去白，焙）3分，枳壳（去瓤，麸炒）半两，蜀椒（去目并闭口，炒出汗）半两。
出处：《圣济总录》卷五十二。
主治：肾脏虚冷，气攻两胁下胀，小腹急痛，胸中短气。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，加生姜如枣大（拍碎），大枣3枚（擘破），煎至6分，去滓，食前温服。
制备方法：上(口父)咀，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
