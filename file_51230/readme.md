序号：51245
名称：荣芝丸
组成：鹿角霜5两，鹿茸（去毛，酥炙）3两，麝香（研）2两，沉香1两，白术1两，当归（去芦头）1两，熟干地黄1两，苁蓉（酒浸2宿）1两，牛膝（酒浸2宿）1两，菟丝子（酒浸，别研）1两，萆薢（蜜炒）1两，川芎1两，五味子1两。
出处：《御药院方》卷六。
主治：诸虚不足。
加减：None
功效：起阴发阳，安神定魄，补五脏，和六腑，活血脉，填骨髓，强骨生力，驻颜色，久服轻身，延年不老。
用法用量：每服30丸，空心粥水送下；或温酒盐汤亦得。渐加至50丸。
制备方法：上为细末，用面2两，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
