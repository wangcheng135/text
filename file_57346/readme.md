序号：57361
名称：消瘰丸
组成：玄参（蒸）4两，牡蛎（煅，醋研）4两，贝母（去心，蒸）4两。
出处：《医学心悟》卷四。
主治：瘰疬初起，痰核。
加减：None
功效：消瘰养阴，化痰软坚。
用法用量：每服3钱，开水送下，日2次。
制备方法：上为末，炼蜜为丸。
临床应用：None
各家论述：《中医方剂临床手册》：方用玄参滋阴降火，苦咸消瘰；贝母化痰消肿，解郁散结；牡蛎咸寒，育阴潜阳，软坚消瘰。合而用之，对瘰疬早期有消散之功；病久溃烂者，亦可应用。
用药禁忌：宜戒恼怒，断煎炒，及发气、闭气诸物，免致脓水淋漓，渐成虚损。
附注：消疬丸（《疡医大全》卷十八）。本方改为汤剂，名“消疬汤”（见《外科真诠》）。
