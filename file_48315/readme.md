序号：48330
名称：神效方
组成：水银、甘草、黄柏、黄连、松脂、腻粉、土蜂窠（著壁上者，南方多有之）。
出处：《百一》卷十六引赵百中方。
主治：一切恶疮，医所不识者；亦治疥疮。
加减：None
功效：None
用法用量：先以温水洗疮，帛拭干，涂之。一切无名疮，涂1次即愈；有黄水者，涂之随手即干；痒不堪忍者，涂之立止痛；甚者涂之立定。治疥，抓破敷药。
制备方法：上取水银放掌中，以唾擦为泥，入瓷器中，以生麻油和研，生绢滤如稀饧，和药末再研如稠饧。
临床应用：None
各家论述：None
用药禁忌：None
附注：松脂散（《得效》卷十九）、神效散（《仁术便览》卷四）。《仁术便览》本方用法：诸药同研，油调敷。
