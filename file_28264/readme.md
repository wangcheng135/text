序号：28272
名称：苍术膏
组成：苍术（新者，刮去皮，薄切）。
出处：《本草纲目》卷十二引《卫生杂兴》。
主治：None
加减：None
功效：除风湿，健脾胃，变白驻颜，补虚损。
用法用量：每服3匙，以温酒送下，清早、临卧各1服。
制备方法：米泔水浸二日，一日一换，取出，以井花水浸过二寸，春、秋五日，夏三日，冬七日，涌出，以生绢袋盛之，放在一半原水中，揉洗津液出，扭干；将滓又捣烂，袋盛于一半原水中，揉至汁尽为度；将汁入大砂锅中，慢火熬成膏。每1斤，入白蜜4两，熬二炷香。每膏1斤，入水澄白茯苓末半斤，搅匀瓶收。
临床应用：None
各家论述：None
用药禁忌：忌醋及酸物、桃、李、雀、蛤、菘菜、青鱼。
附注：None
