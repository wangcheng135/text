序号：70484
名称：紫金夺命膏
组成：川黄连2两，全蝎2两，穿山甲2两，黄芩2两，川黄柏2两，当归2两，白芷2两，赤芍药1两，番木鳖（切片）1两，生地黄1两，官桂4两，海藻4两。
出处：《中国医学大辞典》。
主治：恶疮，结毒，瘰疬，冷瘤，痞块，跌打骨断，久不收口。
加减：None
功效：None
用法用量：熬成不得加减药味分两。每用少许，摊贴患处。
制备方法：用水煎汁，去滓，用麻油22两，将药汁入内，熬尽水气，滴水成珠，方下血丹（炒过，飞）11两，搅匀成膏，再下黄蜡7钱，又下阿魏6钱（切片），掺膏药上，令其自化，候微冷，又下乳香、没药（均去油）、轻粉各6钱，麝香、血竭、朱砂、雄黄各2钱，雄鼠粪1两5钱，燕窝当底泥1两，俱研细末，入膏搅匀，收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
