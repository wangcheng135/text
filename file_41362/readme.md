序号：41377
名称：金英丸
组成：虢丹1两，麦门冬（去心）1两，牡蛎1两，知母1两，黄连2大两，干栝楼根2大两，苦参2大两，金100箔，银100箔，生栝楼根2大两（杵如泥，入药中）。
出处：方出《医方类聚》卷一二五引《神巧万全方》，名见《圣济总录》卷五十八。
主治：消渴不止。
加减：None
功效：None
用法用量：每服40丸，食后以饮下，1日2次，夜又进1服，当日渴止；10日已来，渐觉减，即每服35丸，1日2次；1月外，每服30丸，1日1次。夏月即用蜜为丸。服药之次，腹中忽冷痛，即取厚朴2小两（炙），橘皮3分（去白，焙），生姜2小两，以水2大升，煎取半升，去滓，分温2服，服讫良久，以饭压之，如腹中不痛，即不吃。
制备方法：上为末，用生栝楼根汁和为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
