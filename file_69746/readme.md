序号：69762
名称：琥珀丸
组成：当归2两，苏木2两，羌活2两，丹皮（盐水炒）1两，白术（炒）2两，赤芍1两，南星1两，陈皮1两，独活1两，川断1两，乳香（去油）1两，没药（去油）1两，川芎1两，黄芩1两，桂枝1两，青皮1两，白芍1两，木瓜2两，牛膝6两，苡仁6两，琥珀2钱，桑皮2钱，五加皮4两，甘草5钱，川柏3钱，黑豆2合，肉桂2钱，熟地2两。
出处：《接骨入骱》。
主治：跌打重伤。
加减：断骨，加自然铜、麻雀粪、密陀僧。
功效：None
用法用量：分作2次，空心陈酒送下。
制备方法：上为细末，用红糖油为丸，每丸重3钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
