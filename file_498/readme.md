序号：498
名称：五积散
组成：石榴皮2分，荆三棱2分，巴豆2分，五灵脂2分，甜葶苈2分，大戟2分，芫花2分，甘遂2分，杏仁2分，大黄2分，乌梅2分，盐豉2分。
出处：《普济方》卷一六八。
主治：五积六聚，并酒积蛊气所伤。
加减：None
功效：None
用法用量：每服1钱，空心温酒调下，病重2钱。如为丸，以醋打糊为丸，如绿豆大。每服15丸至30丸。伤酒，葛根汤送下；伤食，盐汤送下；气蛊，木香汤送下；水蛊，樟柳根汤送下。
制备方法：上药以头醋3碗，煮尽干，于砂锅内炒黄色，加陈皮、青皮、木香、瞿麦、豆蔻、砂仁6味在内，一处相和，研为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
