序号：51815
名称：钟乳散
组成：钟乳粉5分，附子5分（炮），白术14分，防风10分，牡蛎10分（熬），栝楼10分，干姜5分，桔梗5分，茯苓5分，细辛5分，桂心5分，人参5分。
出处：《外台》卷十七引《经心录》。
主治：阳虚气衰，畏寒气短，腹胀喘息，皮毛焦枯。伤损虚乏少气，虚劳百病。气极虚寒，阴畏阳气，昼差暮甚，气短息塞。气虚极，皮毛焦，津液不通，力乏，腹胀，甚则喘急，气短息塞。
加减：None
功效：令人丁壮能食，去风冷。
用法用量：每服方寸匕，渐加至2匕，酒送下，日2次。50以上可数服，得力乃止。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：忌食生菜、生葱、猪肉、冷水、桃、李、雀肉、大酢。
附注：None
