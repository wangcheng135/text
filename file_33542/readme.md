序号：33551
名称：灵宝化积膏
组成：巴豆仁100粒，蓖麻仁100粒，五灵脂4两，阿魏（醋煮化）1两，当归1两，白附子5钱，穿山甲5钱，乳香（炙，去油）5钱，没药（炙，去油）5钱，麝香3分，松香1斤半，芝麻油5两。
出处：《万氏家抄方》卷二。
主治：一切痞积。
加减：None
功效：None
用法用量：用狗皮摊贴患处，每日以热鞋底熨，令药气深入。
制备方法：除乳、没、麝、松、阿外，余药俱饮片，浸香油内三日，用砂锅煎药黑焦色，去滓再煎油，滴水成珠，每油4两入松香1斤，少煎一饭时，再入乳香、没药、麝香，取起入水中，抽洗金黄色，煎时以桃、柳枝不住手搅匀令枯。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
