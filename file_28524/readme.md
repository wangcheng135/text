序号：28532
名称：苏羌饮
组成：紫苏3钱，羌活2钱，防风1钱，陈皮1钱，淡豉2钱，葱白数段。
出处：《松峰说疫》卷二。
主治：四时寒疫，头痛身痛，身热脊强，恶寒拘急，无汗，或则往来寒热，气壅痰喘，咳嗽胸痛，鼻塞声重，涕唾稠粘，咽痛齿痛。
加减：如兼阳明症者，加白芷1钱；兼食积者，加炒麦芽、神曲各1钱；肉积者，加山楂1钱；风痰气壅，涕唾稠粘，加前胡1-2钱；咳嗽喘急，加杏仁1钱（泡，去皮尖，研）；心腹膨胀，加姜炒厚朴1钱；胸臆闷塞，加炒枳壳5-6分；呕逆恶心，酌加藿香、制半夏、生姜1钱；年高者、虚怯者，加人参1钱；阴虚血虚者，加熟地3钱，当归1钱；脾虚者、中气不足者，加参1钱、术1钱。
功效：None
用法用量：水煎服，不应再服。初觉，速服必愈，迟则生变。
制备方法：None
临床应用：None
各家论述：此足太阳药也。紫苏温中达表，解散风寒；羌活直入本经，治太阳诸症；淡豉解肌发汗，兼治疫瘴；防风能防御外风，随所引而至；陈皮利气而寒郁易解；姜可驱邪，葱能发汗，辅佐诸药，以成厥功。四时风寒，皆能治疗，甚毋以药味平浅而忽之，惟不治瘟疫。
用药禁忌：None
附注：本方组成，据原书方论当有生姜。
