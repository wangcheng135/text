序号：62583
名称：淡渗二苓汤
组成：泽泻1两，滑石2两，赤茯苓、白术、猪苓（1方有桂2钱半）。
出处：《医方类聚》卷五十六引《修月鲁般经》。
主治：中湿，病似伤寒，头汗自出，肢体疼重，难于转侧，小便不利者；欲作痫状，或表热里寒而自利，或误用巴豆热药下之而协热利不止，或表里俱热，自利或呕，或伤寒痞闷，脉尚浮而恶寒，表未解者；或一切留饮不散，水停心下，或太阳少阴俱病，或一切呕泻霍乱无问寒热，及小儿吐泻急慢惊风。
加减：嗽喘烦心不得眠卧者，加阿胶半两。
功效：None
用法用量：每服2钱，加生姜3片，大枣3枚，灯心5茎，水煎，热服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
