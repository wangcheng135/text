序号：65991
名称：蚺蛇胆丸
组成：蚺蛇胆1分（细研），丁香1分，黄连1分（去须），苦参3分（锉），青葙子1分，牛角屑1分，木香1分，朱砂1分（细研），雄黄1分（细研），青黛1分（细研），龙胆1分，麝香1分（细研），牛黄1分（细研），胭脂1分（细研），硫黄1分（细研），白矾灰1分，头发灰1分，铫绢灰1分，干虾蟆灰1分。
出处：《圣惠》卷八十七。
主治：小儿蛔疳，壮热，眼赤或涩，常多揉目，及发黄秃落，视物不明，手脚心热，时出蛔虫，下痢或青黄赤白无定，身体口鼻及下部生疮，虫蚀齿落，项边生无辜，肌体羸瘦，兄弟姊妹相传至死者。
加减：None
功效：None
用法用量：每服3丸，以粥饮送下；又以少许水化2丸，吹于鼻中；及有疮处，敷之。
制备方法：上为末，都研令匀，以炼蜜为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
