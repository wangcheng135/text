序号：33727
名称：灵砂宁神丸
组成：辰砂2两（不夹石，绢袋子盛，于银石器内悬，于器内用椒红3两，取井花水调椒红盛于器内，可7-8分，更用锅子坐，盛朱砂器在内，重汤煮令鱼眼沸。3昼夜为度，取出辰砂研，水飞），人参（去芦头）3两，白术3两，茯神（去木）3两，鹿茸（燎去毛，涂酥炙令黄）3两，黄耆（蜜炙）3两，石菖蒲2两。
出处：《杨氏家藏方》卷十。
主治：男子、妇人大病之后，伤损荣卫。或发汗、吐、下太过，或失血过多之后，精气亏损，不能复常，心神恍惚，不得睡眠，饮食全减，肌体瘦弱，怠堕倦乏，嗜卧无力，四肢痠痛。
加减：None
功效：补虚益气，滋养荣卫。
用法用量：每服20-30丸，空心、食前以温酒或米饮送下。
制备方法：上为细末，次入辰砂研匀，用枣肉和令匀熟，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：宁神丸（《普济方》卷二一八）。本方方名，《永类钤方》引作“灵砂宁志丸”。
