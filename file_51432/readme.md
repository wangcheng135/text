序号：51447
名称：轻骨汤
组成：知母（焙）1两，人参1两，天仙藤（洗，锉）1两，白术1两，秦艽（去土）1两，柴胡（去苗，洗，锉，焙）1两，鳖甲（去裙襕，醋炙）1两，黄耆（洗，打破，手劈如丝，以盐少许和水揉，猛火焙干）半两，常山半两，当归（切，炙）半两，前胡（去芦头）半两，芎藭半两，紫菀（洗，焙）半两，白茯苓（去黑皮）半两，甘草（生）半两。
出处：《圣济总录》卷八十九。
主治：虚劳身体倦怠，百节痠疼，羸瘦发热，神昏不爽。
加减：None
功效：None
用法用量：每服3钱匕，水1盏半，加乌梅半个，同煎至8分，去滓温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
