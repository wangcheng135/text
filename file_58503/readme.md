序号：58518
名称：秘方种子丹
组成：熟地2两4钱，淡苁蓉2两4钱，陈萸肉2两4钱，童木通2两4钱，飞龙骨2两，煅牡蛎2两，威灵仙2两，菟丝子2两，全当归2两，大茴香2两，巴戟肉2两，远志肉2两，荜澄茄2两，母丁香2两，干漆2两，车前子2两，茯苓1两4钱，广木香1两4钱，桑螵蛸1两4钱，蛇床子1两4钱，全蝎（去尾）5钱，灯心5钱，萆薢4钱，贡沉香3钱，马蔺花8分，蜘蛛14个。
出处：《饲鹤亭集方》。
主治：命门火弱，阳痿不兴，下元虚寒，精冷无子。
加减：None
功效：补气益血，添精壮阳。
用法用量：每服3-4钱，开水送下。
制备方法：上为末，炼蜜为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
