序号：47912
名称：神圣光明饼
组成：羚羊角（镑）5钱，白犀角（镑）5钱，密蒙花5钱，生地5钱，熟地5钱，独活5钱，藁本5钱，草决明5钱，栀子（炒）5钱，川芎5钱，细辛5钱，蔓荆子5钱，苍术5钱，木贼1两，甘草1两，白蒺藜1两，槐花1两，黄连1两，荆芥1两，青葙子1两，羌活1两，芒消1两，白附子（煨）1两，赤石脂1两，夜明砂（淘净末）1两，大麻子2两，大黄2两。
出处：《良朋汇集》卷三。
主治：诸般目疾疼痛，日久渐细，云膜遮睛，远不视物，并一切难治眼疾。
加减：None
功效：通大肠之火，祛除燥结。
用法用量：每服1丸，冷茶研化下。重者10丸即好。
制备方法：上为细末，炼蜜为丸，如弹子大，重2钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
