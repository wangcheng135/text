序号：73897
名称：硼砂丸
组成：硼砂半两（细研），青礞石1分（细研），穿山甲1分（炙令黄焦），磁石1分（烧，醋淬7遍，捣碎，研如粉），京三棱1分（微煨，锉），干漆1分（捣碎，炒令烟出），虻虫50枚（炒令微黄，去翅足），水蛭50枚（炒令微黄），巴豆15枚（去皮心，研，纸裹压去油），赤石脂1分（细研）。
出处：《圣惠》卷四十九。
主治：积年厌食癥，血气及癥块。
加减：None
功效：None
用法用量：每服3丸，小儿1丸，以烧蒸饼灰汤送下。一复时后，取下恶物。若是血气块，当归酒下，不过5服愈。
制备方法：上为末，入巴豆研令匀，用软饭为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《幼幼新书》引作“硇砂丸”。方中硼砂，《幼幼新书》引作“硇砂”。
