序号：4635
名称：木香枳壳汤
组成：木香1两，枳壳1两（去瓤，麸炒），黄连（去须）1两，麻黄1两半（去根节），贝母1两半（去心），百合1两半，紫菀1两半（去苗土），款冬花1两半（去梗），桑根白皮1两半，天雄1两半（炮裂，去皮脐），白石脂1两半，昆布（洗去咸，焙）1两半，黄芩（去黑心）半两，旋覆花（微炒）3分，杏仁（汤浸，去皮尖双仁，炒）10枚。
出处：《圣济总录》卷六十六。
主治：咳嗽气促。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，加生姜3片，同煎至8分，去滓温服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
