序号：83688
名称：乌梅丸
组成：乌梅肉1两（微炒），桂心1两，甘草1两（炙微赤，锉），虎头骨2两（涂酥，炙令黄），人参1两（去芦头），香豉1合（炒干），恒山2两（锉），鳖甲2两（涂醋，炙令黄，去裙襕），麝香1分（细研），附子半两（炮裂，去皮脐），桃仁半两（汤浸，去皮尖双仁，麸炒微黄），川升麻1两，肉苁蓉1两（酒浸1宿，刮去皱皮，炙令干）。
出处：《圣惠》卷五十二。
主治：肾疟，腰背痛，手足寒，食少无力。
加减：None
功效：None
用法用量：每服20丸，食前以粥饮送下。渐加至30丸。
制备方法：上为末，入麝香研匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
