序号：30847
名称：芦连消疳丸
组成：芦荟5钱，胡黄连5钱，宣黄连（酒炒）5钱，白术（米泔浸，焙）1两，白茯苓（去皮）1两，当归（全身，用酒洗）1两，白芍（酒炒）8钱，人参6钱，神曲（炒）6钱，使君子（去壳，晒干）7钱，山楂肉7钱，芜荑（炒）5钱，槟榔5钱，大甘草节（去粗皮）4两。
出处：《回春》卷七。
主治：小儿生疳，痞块发热，肚胀。
加减：None
功效：壮脾胃，消饮食，消肝火，磨积块。
用法用量：每服50-60丸，临晚米汤送下；或炼蜜为丸，如龙眼大。每晚噙化1丸，或米汤送下，或酒亦可。
制备方法：上为细末，汤泡蒸饼，打糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
