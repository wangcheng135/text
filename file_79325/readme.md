序号：79341
名称：薷苓清暑汤
组成：藿香、香薷、云苓、广皮、厚朴、麦冬、人参、白术、扁豆、泽泻、甘草、草蔻、竹茹、砂仁、生姜、乌梅。
出处：《陈素庵妇科补解》卷三。
主治：妊娠当盛夏时，身居闺阁，贪凉衣单，恣食生冷瓜果，体薄为阴寒所逼，暑邪袭之，忽然烦闷，身热多汗，或恶心呕吐。
加减：None
功效：温解安胎。
用法用量：None
制备方法：None
临床应用：None
各家论述：妊娠得此，先宜辛温之剂，散暑和中，使阴邪与暑邪两解。再加养血益气，利水解暑则胎自安。是方香薷辛温散暑；朴、陈、砂、藿、草蔻温中散寒；四君益元，暑伤气故也；扁、泽利水除湿，暑必兼湿故也；麦、茹清心，暑邪先入心，故令多汗，以安神清心为亟也；生姜、乌梅恐其呕逆。病退二三，再加归、芍、杜、续。
用药禁忌：None
附注：None
