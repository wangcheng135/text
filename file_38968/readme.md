序号：38978
名称：泻热黄连汤
组成：黄芩（酒制炒）1两，黄连（酒制炒）1两，草龙胆（酒制）1两，生地黄（酒制）1两，升麻5分，柴胡1两。
出处：《东垣试效方》卷五。
主治：眼暴发赤肿疼痛。内障，有眵泪眊臊。
加减：None
功效：泻热。
用法用量：上(口父)咀。每服2钱，将先煎药水纳入泻热黄连汤再煎，至1盏，去滓，于日午饭间热服之。
制备方法：None
临床应用：None
各家论述：《准绳·类方》：本方治主治客之剂也。治主者，升麻主脾胃，柴胡行肝经，为君；生地黄凉血，为臣，为阳明、太阳、厥阴多血故也。治客者，黄连、黄芩皆疗湿热，为佐；龙胆草专除眼中诸疾，为使；为诸湿热俱从外来为客也。
用药禁忌：午后服之，则阳逆不行；临睡休服，为反助阴也。
附注：黄连饮子（《景岳全书》卷六十）。方中生地黄，《准绳·幼科》作大黄。
