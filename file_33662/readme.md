序号：33671
名称：牡丹汤
组成：牡丹（去心）1两半，芎1两半，甘草（炙，锉）1两半，黄芩（去黑心）1两半，人参1两半，桂（去粗皮）1两半，干姜（炮制）1两半，吴茱萸（汤浸7次，焙干微炒）1两半，桃仁18个（汤浸去皮尖双仁，麸炒黄色），白茯苓（去黑皮）5分，当归（切，焙）5分，芍药5分。
出处：《普济方》卷三三四。
主治：妇人月水来而不利，攻脐腹痛不可忍。
加减：None
功效：None
用法用量：每服3钱，水1盏，生姜1分（拍破），同煎至6分去滓，下消石半钱，温服，如人行3-5里，再服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
