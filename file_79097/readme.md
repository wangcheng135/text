序号：79113
名称：磨积三棱丸
组成：木香半两，麦芽半两，京三棱（炮）半两，广术半两，枳壳（麸炒）半两，石三棱（去皮）半两，杏仁（麸炒）半两，干漆（炒烟尽）3钱，鸡爪三棱半两，葛根3钱，官桂2钱半，黑牵牛半两（半生半熟），丁香2钱，槟榔2钱，香附子2钱，青皮（去白）2钱，缩砂3钱，白牵牛半两（半生半熟），陈皮（去白）3钱。
出处：《卫生宝鉴》卷十四。
主治：远年近日诸般积聚，癖痃气块，或气积酒积诸般所伤，无问男子妇人老幼并宜服之。
加减：None
功效：常服进饮食。
用法用量：每服20丸，食后生姜汤送下，温水送下亦得，日2次。病大者40日消。
制备方法：上为末，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
