序号：28345
名称：芩连二陈汤
组成：青子芩2钱，仙半夏1钱半，淡竹茹2钱，赤苓3钱，川连8分，新会皮1钱半，小枳实1钱半，碧玉散3钱（包煎），生姜汁2滴，淡竹沥2瓢（和匀同冲）。
出处：《重订通俗伤寒论》。
主治：None
加减：None
功效：清肝和胃，蠲痰泄饮。
用法用量：None
制备方法：None
临床应用：None
各家论述：肝阳犯胃，症多火动痰升，或吐粘涎，或呕酸汁，或吐苦水，或饥不欲食，食则胃满不舒，甚则胀痛，或嘈杂心烦，故以芩、连、橘、半苦降辛通，调和肝胃为君；臣以竹茹、枳实通络降气；佐以赤芩、碧玉，使胃中之浊饮从小便而泄；使以姜、沥二汁，辛润涤痰，以复其条畅之性。此为清肝和胃，蠲痰泄饮之良方。
用药禁忌：None
附注：None
