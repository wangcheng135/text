序号：83387
名称：乌鸡酿酒
组成：乌鸡1只（其鸡先以笼养，用大麻子5升，与鸡吃，吃尽即便用鸡去毛及肠胃，净洗拭干，将其肉碎锉，毛即烧灰，亦同酿酒），羌活3两，桂心1两，牛膝4两（去苗），芎2两，附子3两（炮裂，去皮脐），防风3分（去芦头），萆薢3两，熟干地黄4两，乌蛇6两（酒浸，去皮骨，炙微黄），独活5两，石斛3两（去根），虎胫骨5两（涂酥，炙微黄），当归2两，海桐皮5两，丹参4两，白胶香1两，地骨皮1两，五加皮10两，百灵藤1两，松节10两。
出处：《圣惠》卷二十五。
主治：诸风。
加减：None
功效：None
用法用量：上锉细，以水2硕，煎取5斗，与曲末，如常法同酿酒。候熟，即日饮3-4盏。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌生冷、毒滑、动风物。
附注：None
