序号：28656
名称：补天丹
组成：杜仲2两，贡白术2两半，白芍2两，故纸2两，熟地2两，远志2两，当归1两5钱，枸杞1两5钱，核桃仁3两，牛膝2两，黄耆2两，海狗肾1具，川楝子2两，川芎1两5钱，人参1两5钱，沉香5钱，木香1两，小茴1两5钱，甘草1两，茯神1两。
出处：《全国中药成药处方集》（沈阳方）。
主治：肾虚阴痿，早泄遗精，腰腿痠痛，盗汗自汗，疝气腹疼，四肢厥冷，劳伤虚损，怔忡健忘，神经衰弱，形容焦悴，淋漓白浊，肾囊凉湿。
加减：None
功效：补肾固精，强心安神。
用法用量：每服1丸，盐汤送下。
制备方法：上为极细末，炼蜜为丸，2钱重。
临床应用：None
各家论述：None
用药禁忌：忌生冷。
附注：None
