序号：23857
名称：朱砂膏
组成：葱50-60斤（捣极烂，绞汁放锅内，投入嫩松香5斤，微火熬至葱汁滚，松香化，取下俟稍冷，即以手在汁中揉松香几百揉，然后再放火上再烊再揉，如此5-6次，揉至松香色白无油为度，配入后药），当门子5钱（即顶高麝香），樟脑12两，梅花冰片1两，蓖麻子1斤（去壳，研如泥，另贮），乳香3两5钱，没药3两5钱（俱用灯心草炒去油），朱砂6两（水飞）。
出处：《绛囊撮要》。
主治：一切无名肿毒，横痃，乳疖，恶疽疔毒。未成者即消，已成者即溃。
加减：None
功效：None
用法用量：None
制备方法：上除蓖麻子，余皆为极细末，将制好松香放于瓷钵内，隔水烊化，取出，即以前药末并蓖麻子泥一并投入，搅和摊贴；如干，可酌加蓖麻子油，以好摊为度；摊用柿漆单张桑皮纸，不可着火。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
