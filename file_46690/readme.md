序号：46705
名称：烂积丸
组成：二丑2两6钱7分，三棱2两6钱7分，莪术2两6钱7分，大黄2两6钱7分，槟榔2两6钱7分。
出处：《全国中药成药处方集》（吉林方）。
主治：五积六聚，胸中痞痛，胃脘腹胀，食噎气塞，咯气吞酸，胃积嘈杂，二便闭结，饮食不下。
加减：None
功效：消食化滞，攻泄宿积，利大便，开脾郁。
用法用量：每服2钱，每日2次，早、晚空心服之。7-12岁，每服1钱；4-6岁，每服5分；周岁-3岁者，每服3分，白开水送下。
制备方法：上为极细末，水为小丸，如梧桐子大，红曲为衣。
临床应用：None
各家论述：None
用药禁忌：虚寒之人及孕妇禁服；忌食辛辣生冷。
附注：None
