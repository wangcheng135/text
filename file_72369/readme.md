序号：72385
名称：疏凿饮子
组成：泽泻、商陆、赤小豆（炒）、羌活（去节）、大腹皮、椒目、木通、秦艽（去芦）、茯苓皮、槟榔各等分。
出处：《济生》卷五。
主治：水气，通身洪肿，喘呼气急，烦躁多渴，大小便不利，服热药不得者。
加减：None
功效：None
用法用量：上(口父)咀。每服4钱，水1盏半，加生姜5片，煎至7分，去滓温服，不拘时候。
制备方法：None
临床应用：None
各家论述：1.《医方集解》：此足太阳手足太阴药也。外而一身尽肿，内而口渴便秘是上下表里俱病也。羌活、秦艽解表疏风，使湿以风胜，邪由汗出，而升之于上；腹皮、苓皮、姜皮、辛散淡渗，所以行水于皮肤；商陆、槟榔，椒目、赤豆，去胀攻坚，所以行水于腹里；木通泻心肺之水，达于小肠，泽泻泻脾肾之水，通于膀胱。上下内外分清其势，亦犹神禹疏江凿河之意也。_x000D_
2.《医宗金鉴》：以商陆为君，专行诸水。佐羌活、秦艽、腹皮、苓皮、姜皮行在表之水，从皮肤而散，佐槟榔、赤豆、椒目、泽泻、木通，行在里之水，从二便而出，上下、内外，分消其势，亦犹神禹疏凿江河之意也。
用药禁忌：None
附注：疏凿散（《杏苑》卷六）。
