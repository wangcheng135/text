序号：7232
名称：四君子汤加竹沥姜汁方
组成：四君子汤加竹沥、姜汁。
出处：方出《丹溪心法》卷一，名见《医方考》卷一。
主治：半身不遂在右者，属痰有热并气虚。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：《医方考》：经曰：左右者，阴阳之道路也。故左属血，而右属气。气虚者，补之以甘，故用人参、白术、茯苓、甘草四件，称其为君子者，谓其甘平，有冲和之德，而无克伐之性也。其加竹沥，谓其行痰；其加姜汁，所以行竹沥之滞，而共成夫伐痰之功耳。
用药禁忌：None
附注：四君子加竹沥姜汁方（《明医指掌》卷二）。四君子加姜汁竹沥方（《成方切用》卷十二下）。
