序号：54238
名称：大建中汤
组成：黄耆3两，远志（去心）3两，当归3两，泽泻3两，芍药2两，人参2两，龙骨2两，甘草（炙）2两。
出处：《宣明论》卷一。
主治：房事过度，气血俱亏，精关不固，少腹急痛，尿频尿精，虚热，自汗或盗汗，形体羸瘦。蛊病，小腹急痛，便溺失精，溲而出白液。思虑太过，心气耗弱，阳气流散，精神不收，阴无所使，热自腹中，或从背膂，渐渐蒸热，日间小剧，至夜渐退，或寐而汗出，小便或赤或白或浊，甚则频数尿精，夜梦鬼交，日渐羸瘦。虚热盗汗，四肢倦怠，百节烦疼，口苦舌涩，心怔短气。
加减：None
功效：None
用法用量：每服3钱，水1盏，加生姜5片，煎至8分，去滓温服，不拘时候。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：大建中黄耆汤（《普济方》卷二一七引《究原方》）、黄耆建中汤（《普济方》卷二一八）。
