序号：57064
名称：消毒膏
组成：当归5钱，黄耆5钱，川芎5钱，杏仁5钱，白芷5钱，白蔹5钱，零陵香5钱，槐白皮5钱，柳枝（嫩者）5钱，木鳖子（用仁）5钱，甘松5钱（锉），乳香3钱，没药3钱，轻粉1钱，朱砂5分，朱红5分，麝香1分，黄丹（炒紫色）8两，黄蜡8两，芝麻油1斤。
出处：《外科精义》卷下。
主治：头面五发恶疮，及烧汤冻破溃烂。
加减：None
功效：止痛生肌，清血脉，消毒败肿，通气脉。
用法用量：None
制备方法：上将锉药油浸7日，木炭火上煎杏仁焦色，滤去渣，下黄蜡，候溶开出火，下丹，急搅百十转，下乳香、麝香、朱砂等6味，不住手搅至凝，瓷盒内收贮，白光绢上摊之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
