序号：4677
名称：木香流气饮
组成：川芎1钱，当归1钱，紫苏1钱，桔梗1钱，青皮1钱，陈皮1钱，乌药1钱，黄耆1钱，枳实1钱，茯苓1钱，防风1钱，半夏1钱，白芍1钱，甘草节5分，大腹皮5分，木香5分，槟榔5分，泽泻5分，枳壳5分，牛膝下部加1钱。
出处：《外科正宗》眷三。
主治：流注瘰疬，及郁结为肿，或血气凝滞，遍身走注作痛，或心胸痞闷，咽嗌不利，胁腹膨胀，呕吐不食，上气喘急，咳嗽痰盛，或四肢面目浮肿者。
加减：None
功效：None
用法用量：水3钟，加生姜3片，大枣1枚，煎8分，食远服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
