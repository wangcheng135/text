序号：45274
名称：枳实导滞汤
组成：小枳实2钱，生锦纹1钱半（酒洗），净楂肉3钱，尖槟榔1钱半，薄川朴1钱半，小川连6分，六和曲3钱，青连翘1钱半，老紫草3钱，细木通8分，生甘草5分。
出处：《重订通俗伤寒论》。
主治：温病热症，里有积滞。
加减：None
功效：下滞通便。
用法用量：None
制备方法：None
临床应用：None
各家论述：凡治温病热证，往往急于清火，而忽于里滞。不知胃主肌肉，胃不宣化，肌肉无自而松，即极力惊解，反成冰伏。此方用小承气合连、槟为君，苦降辛通，善导里滞；臣以楂、曲疏中，翘、紫宣上，木通导下；佐以甘草和药，开者开，降者降，不透发而自透发。每见大便下后，而疹癍齐发者以此，此为消积下滞，三焦并治之良方也。
用药禁忌：None
附注：None
