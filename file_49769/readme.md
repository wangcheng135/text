序号：49784
名称：茯苓饮
组成：茯苓3两，人参2两，白术3两，生姜4两，枳实2两（炙），橘皮1两半（切）。
出处：《外台》卷八引《延年秘录》。
主治：心胸中有停痰宿水，自吐水出后，心胸间虚气满，不能食。
加减：None
功效：消痰气，令能食。
用法用量：以水6升，煮取1升8合，去滓，分3次温服，如人行8-9里进之。
制备方法：上切。
临床应用：None
各家论述：《金鉴》：上、中二焦气弱，水饮入胃，脾不能输归于肺，肺木能通调水道，以致停积为痰，为宿水。吐之则下气因而上逆，虚与气结，满不能食，当补益中气，以人参、白术为君；茯苓逐宿水，枳实破诸气为巨；开脾胃，宣扬上焦，发散凝滞，则陈皮，生姜为使也。其积饮既去，而虚气塞满其中，不能进食，此证最多。
用药禁忌：忌酢物、桃、李、雀肉。
附注：外台茯苓饮（《金匮》卷中附方）、茯苓饮子（《鸡峰》卷十八）、茯苓汤（《校注妇人良方》卷六）。
