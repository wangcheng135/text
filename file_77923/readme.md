序号：77939
名称：凝真丹
组成：益智仁2两。
出处：《简易》引《诜诜书》（见《医方类聚》卷一四九）。
主治：治丈夫三丹不凝结，致真气不固，精清精滑，饮食不美，四肢怠情，昏困嗜卧。上丹不凝结，则常多感冒，鼻流清涕，头目昏疼。中丹不凝结，则发热自汗，心悸惊，恍惚健忘，不能饮食。下丹不凝结，则真气不固，梦遗白浊，胸中短气，面黄体虚，形瘦瘁，情思不乐，饮食减少，惊悸恍惚。
加减：None
功效：None
用法用量：每用少许搐鼻中。久用，清涕自止。治中丹不凝结，酸醋浸益智仁三宿，焙干，为细末，醋煮面糊为丸，如梧桐子大。每服30-50丸，盐汤送下。治下丹不凝结，以盐水浸益智仁三宿，焙干，为细末，盐煮面糊为丸，如梧桐子大。每服30-50丸，空心盐汤送下，不可用酒服，恐散真气。
制备方法：治上丹不凝结，用饼饽药，搜面裹煨，令面焦，去面，为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
