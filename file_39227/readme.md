序号：39237
名称：狗胆丸
组成：狗胆5枚（去汁），硇砂半两（胆汁浸21日），干漆半两（捣碎，炒令烟出），芫花半两（醋拌炒令干），牛李仁半两，延胡索半两，干姜1分（炮裂，锉），斑蝥1分（糯米拌炒令黄，去翅足），当归半两（锉，微炒），麒麟竭1分，砒霜1分，伏龙肝半两（细研），自然铜1两（锉，细研），虻虫半两（炒微黄，去翅足），水蛭半两（炒微黄）。
出处：《圣惠》卷七十二。
主治：妇人月水久不通，日渐羸瘦，变为血痨，及血气结聚疼痛。
加减：None
功效：None
用法用量：每服7丸，食前以温酒送下。
制备方法：上为末，用头醋1升，先入自然铜末，煎10沸以来，去却石脚，却入铫子内，入药末一半，以慢火煎如膏；后更入硇砂、狗胆及一半药末，和捣三二百杵，丸如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中硇砂，《普济方》引作“硼砂”，无牛李仁。
