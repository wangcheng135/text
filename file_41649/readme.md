序号：41664
名称：金箔丸
组成：金箔（研）100片，银箔（研）100片，犀角（细屑为末）1分，牛黄（研）1分，丁香1分，龙脑（研）1分，沉香1分，珍珠末1分，木香1分，麝香（研）1分，琥珀1分，硼砂（研）1分，乌蛇（酒浸，去皮骨，炙）1分，天麻（酒浸，切，焙）1分，雄黄（研）1分，蝎梢（炒）1分，白僵蚕（炒）1分，附子（炮裂，去皮脐）1分，天南星（炮）1分，防风（去叉）1分，白附子（炮）1分，甘草（炙）1分，丹砂（研）1两，墨（烧，研）半两。
出处：《圣济总录》卷六。
主治：风癔，奄忽不知人，喉中噫噫然有声，舌强不能言，身软自汗。
加减：None
功效：None
用法用量：每服大人5丸，薄荷酒下；小儿2丸，薄荷汤化下。
制备方法：上为细末，入研者药一处和匀。内将金银箔入水银3分，同研如泥，入诸药研，和匀，炼蜜为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
