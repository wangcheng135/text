序号：33725
名称：灵砂丹
组成：杏仁10个（去皮尖），南巴豆30个（去皮膜油），好白面1匙，黄丹3钱。
出处：《医方类聚》卷一一二引《经验秘方》。
主治：二十四般积证。
加减：None
功效：None
用法用量：每服3丸。比及取积，服药先服白粥3日，除粥外，都休吃他物。服此药人，不以大小生活并不得做，亦不得高声唱叫，也不得往来行走，只可睡坐。若妇人病患，服药须令男子将药丸递于患妇口中，用饮子药下。服药人并不大便。10年证候，10日取下来；5年证候，5日取下。
制备方法：上先以杏仁、巴豆杀研极细，入黄丹研如泥，方入面一匙，滴新水为丸，如黄米大。
临床应用：None
各家论述：None
用药禁忌：有孕妇人勿服。
附注：None
