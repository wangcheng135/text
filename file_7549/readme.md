序号：7549
名称：四神散
组成：当归1两，芍药1两，川芎1两，干姜半两（炮）。
出处：《苏沈良方》卷五引《灵苑方》。
主治：妇人血虚心腹痛，产后瘀血腹痛；腘血，血从委中出。
加减：None
功效：None
用法用量：每服2钱，暖酒调下。
制备方法：None
临床应用：None
各家论述：1.《胎产心法》：予谓白芍治血虚腹痛则可，若治瘀血作痛未必然也。今炒透而与炮姜合用，故血虚瘀痛兼治之。_x000D_
2.《医林纂要》：辛以补肝而主气分，盖当归补肝血之虚，川芎以行血中之气，芍药以敛津液而萃之，而干姜则亦能随三物以去血中所挟之寒耳。谓干姜能引血药入血分，则亦非也。此治血分之挟寒者，以挟寒故去生地而易用干姜，然虚寒亦可通治。四神之名，则未知所谓。
用药禁忌：None
附注：当归散（《医方类聚》卷二一八引《吴氏集验方》）、四神汤（《元戎》）、芎归散（《普济方》卷三十四）、当归汤（《普济方》卷三五一）、四顺散（《奇效良方》卷六十三）。
