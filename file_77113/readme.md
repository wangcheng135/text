序号：77129
名称：黑虎丸
组成：芫花（炒）半两，甘遂（炒）半两，乌头（炮裂，去皮脐）半两，大戟（炒，锉）半两，京三棱（煨，锉）半两，牵牛子（炒）半两，干姜（炮）半两，陈橘皮（去白，焙）半两，干漆2两（炒，烟出尽）。
出处：《圣济总录》卷七十二。
主治：诸积，宿食不消；伤寒，气痛，本脏气虚，疟疾；妇人血气，劳气；寸白虫。
加减：None
功效：None
用法用量：每服2丸，消食化气，温水送下；取积滞，米汤送下；温病、伤寒，姜醋汤送下；气痛，艾汤送下；本脏气虚，炒茴香子酒送下；疟疾，桃枝汤送下；妇人血气、劳气，醋汤送下；寸白虫，煎牛肉汤送下。
制备方法：上为末，以醋煮面糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
