序号：60641
名称：通变黑锡丹
组成：铅灰2两（用黑铅数斤熔化后，其面上必有浮灰，屡次熔化，屡次取之，研细），硫化铅1两（用黑铅4两，铁锅内熔化；再用硫黄细末4两，撒于铅上；硫黄皆着，急用铁铲拌炒，铅经硫黄烧炼，结成砂子，取出晾冷，碾轧成饼者去之，余者，再用乳钵研极细），麦曲1两半（炒熟）。
出处：《衷中参西》上册。
主治：痫风。
加减：None
功效：None
用法用量：每服5-6丸，多至10丸，用净芒消4-5分冲水送服。若服药后，大便不利者（铅灰、硫化铅皆能涩大便），芒消又宜多用。
制备方法：上用水为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
