序号：71419
名称：遇仙丹
组成：白牵牛（头末）4两（半炒半生），白槟榔1两，茵陈5钱，莪术（醋煮）5钱，三棱（醋煮）5钱，牙皂（炙，去皮）5钱。
出处：《摄生众妙方》卷一。
主治：邪热上攻，痰涎壅滞，翻胃吐食，十膈五噎，齁哈，酒积，虫疾，血积，气块，诸般痞疾，热疮肿疼，或大小便不利，妇人女子面色萎黄，鬼产，食吞铜铁银物等症。
加减：None
功效：涤饮攻积。
用法用量：五更时用冷茶送下3钱。天明可看去后之物，此药有积去积，有虫去虫。数服行后，随以温粥啖之。
制备方法：上为末，醋糊为丸，如绿豆大。
临床应用：None
各家论述：《医略六书》：白丑涤饮攻痰，槟榔破滞攻积，三棱破气中之血，蓬术破血中之气，茵陈祛湿热，牙皂搜痰涎，沉香降逆气以顺气也。醋丸茶下，使饮化气行，则血脉自活，而痰癖无不消，肢节肿痛无不瘳矣。此涤饮攻积之剂，为痰饮积结肿痛之专方。
用药禁忌：服后忌食他物。孕妇勿服。
附注：牛郎串（《串雅内编》卷三）。《张氏医通》有沉香五钱。《良朋汇集》有白术，无莪术。
