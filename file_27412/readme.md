序号：27420
名称：补骨脂丸
组成：补骨脂（炒）8两，松脂8两，山芋8两，白茯苓（去黑皮）8两，杏仁（汤浸，去皮尖双仁，炒）3升，胡桃肉1斤，枣1斤，鹿角胶（炙燥）10两，桂（去粗皮）1两，牛膝（酒浸，切，焙）1两，泽泻1两，菖蒲1两，薏苡仁1两，萆薢1两，槟榔（煨，锉）1两，独活（去芦头）1两，蒺藜子（炒，主角）1两，蛇床子1两，生地黄20斤（取汁）。
出处：《圣济总录》卷一八五。
主治：None
加减：None
功效：平补诸虚，益精壮阳。
用法用量：每服30丸，空腹温酒送下。
制备方法：上为末，煎地黄汁成煎，入药点蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
