序号：51071
名称：草果厚朴丸
组成：厚朴1两5钱（削去粗皮，洗，切，水煮数10沸，晒干，杵细，以姜等分研细，拌和罨2宿，焙干，入后药），陈皮1两（汤浸1日，干，不去白），干姜5钱（炮），草果子1两（纸裹水浸，炮令香熟，去皮），白术5钱（洗，锉，麸炒），诃黎勒1两（纸裹水湿，煨干取肉），桂半两（去粗皮，每1两取半两），缩砂仁1两（去壳，汤泡洗，再去膜）。
出处：《普济方》卷二十三引《卫生家宝》。
主治：脾胃虚弱，全不思饮食，腹痛滑泄，肠胃怯薄，关节不通。
加减：None
功效：去湿，厚肠胃，固元脏，大进饮食，充肌肤，去酒毒。
用法用量：每服50-60丸，空心、食前白汤送下。
制备方法：上为细末，水煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
