序号：60438
名称：通闭方
组成：大黄3两，槟榔1两半，赤苓2两，枳壳1两半（炒），诃子3两（炒），腹绒1两半。
出处：《医略六书》卷二十八。
主治：孕妇大便秘结，脉沉实大者。
加减：None
功效：None
用法用量：每服2-3钱，葱白煎汤，去滓，放温调下。
制备方法：上为散。
临床应用：None
各家论述：妊娠气滞于中，不能运化而大腹胀满，大便不通，是为气秘，胎孕因之不安。大黄通幽导滞；枳壳泻滞宽胀；赤苓利营渗水；腹绒除满泻气；槟榔下气，性如铁石；诃子涩肠，力能收肺。为散，葱白汤下，既不使病气滞而不通，亦不使元气虚而下脱，洵为导滞通幽、涩肠收肺、不致胎动之专方。《经》曰：有故无殒，亦无殒焉。
用药禁忌：None
附注：None
