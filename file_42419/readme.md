序号：42434
名称：青黛丸
组成：青黛1分，槟榔1枚，木香1分，麝香半分（细研），黄连1两（去须），肉豆蔻1枚（去壳），巴豆半两，大黄半两（锉，微炒），鳖半两（涂醋炙令黄，去裙襕）。
出处：《普济方》卷三八二。
主治：小儿气疳，腹内有积恶滞结之物。
加减：None
功效：None
用法用量：3岁每服2丸，空心以粥送下；3-4岁每服3丸至4丸，每隔3日1次。取下恶物为效。次宜服诃黎勒补之。
制备方法：上先取黄连、巴豆2味，以淡浆水3碗煮令水尽，候干取出巴豆，去皮心，研如膏，纸裹去油，其黄连晒干，然后与诸药都捣研为末，用猪胆汁为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
