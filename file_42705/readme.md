序号：42720
名称：保生锭
组成：牛黄3钱，天竺黄1两，辰砂1两，雄黄3钱，麝香5分，片脑5分，琥珀1两，珍珠5钱，赭石3钱（火煅7次），蛇含石3钱（火煅7次），金箔4帖，银箔4帖，天麻3钱，防风3钱，甘草3钱，茯神（去皮）3钱，人参3钱，僵蚕5钱，血竭5钱，远志3钱（去心），陈皮1两，牛胆南星1两。
出处：《摄生众妙方》卷十。
主治：小儿急慢惊风，痰涎壅盛，胎惊内钓，多啼，夜间恍惚不宁，久患癫痫，咳嗽发热，夏月中暑发搐。
加减：None
功效：镇惊安神宁心。
用法用量：用薄荷汤化下。
制备方法：上为细末，用粉米糊为锭，辰砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
