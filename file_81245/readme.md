序号：81261
名称：鳖甲黄连丸
组成：鳖甲（童便半盏，米醋半盏，慢火上蘸炙至尽色焦黄）半两，黄连（用巴豆7个，去皮膜，用水1盏同煮水尽，去巴豆不用，只使黄连）半两，白术半两，人参（去芦）半两，茯苓半两，甘草（炙）半两，川楝子肉半两，使君子仁半两，木香半两，草豆蔻（炮，去皮）半两，柴胡（去芦）半两，陈皮（去白）半两，草龙胆半两。
出处：《卫生总微》卷十二。
主治：小儿诸疳羸瘦，发热盗汗，寒热，肚大脚细，不肯进乳食，气粗促急，脾胃不调。
加减：None
功效：None
用法用量：每服10-20丸，米饮送下；如有潮热，体热不解，乌梅汤送下。
制备方法：上为细末，豮猪胆汁为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
