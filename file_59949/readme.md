序号：59964
名称：调荣汤
组成：白术、杜仲、牛膝、萆薢、独活、陈皮、肉桂、乌药、川断、当归、川芎、香附。
出处：《陈素庵妇科补解》卷五。
主治：产后两胯连臀俱酸痛者，由坐草久，劳伤筋脉，稳婆试水太早，或风冷乘于下焦，恶血停滞所致。
加减：None
功效：养血温经。
用法用量：None
制备方法：None
临床应用：None
各家论述：产后气血己亏，不能流通，加以风冷乘虚入宫，或恶血微有阻滞，则作酸痛。是方芎、归、川断补血，白术补气，能利腰脐间血，此四味为君；独活祛风，肉桂祛寒，乌药、香附、陈皮行气，萆薢治风湿，此六味为臣；杜仲、牛膝引药下行为佐。凡血气所不及至之处，诸药能引而至。
用药禁忌：None
附注：None
