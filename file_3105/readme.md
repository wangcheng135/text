序号：3105
名称：天麻饼子
组成：天麻5钱，草乌（汤泡，去皮）5钱，川芎5钱，细辛5钱，苍术5钱，甘草5钱，川乌（汤泡，去皮）5钱，薄荷5钱，甘松5钱，防风5钱，白芷5钱，白附子（去皮）5钱，雄黄3钱，全蝎3钱。
出处：《外科正宗》卷四。
主治：因风火湿痰上攻及杨梅疮毒所致头痛；兼治头目昏眩，项背拘急，肢体烦痛，肌肉蠕动，耳哨蝉鸣，鼻塞多嚏，皮肤顽麻，瘙痒瘾疹；又治妇人头风作痛，眉棱骨疼，牙齿肿痛，痰逆恶心。
加减：None
功效：None
用法用量：每服20-30饼，食后细嚼，葱头汤送下；属火热痰痛者，茶汤送下。甚者日进2服。
制备方法：上为细末，寒食面打糊捣稠，捻作饼子，如寒豆大。
临床应用：None
各家论述：None
用药禁忌：忌诸般发物。
附注：None
