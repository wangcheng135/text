序号：28717
名称：八解散
组成：人参1两，茯苓1两，甘草（炙）1两，陈皮（去白）1两，白术1两，藿香（去土）1两，厚朴（去粗皮，锉，生姜自然汁浸1宿，炒紫色）2两，半夏（汤洗7次）1两。
出处：《局方》卷二，（续添诸局经验秘方）。
主治：四时伤寒，头疼壮热，感风多汗，及疗劳伤过度，骨节痠疼，饮食无味，四肢疼倦，行步喘乏，面色痿黄，怠情少力，咳嗽寒热、羸弱自汗、胸膈不快，呕逆恶心。
加减：None
功效：None
用法用量：每服2钱，水1盏，生姜3片，枣子1枚，葱白3寸，同煎至7分，温服，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
