序号：39829
名称：三建膏
组成：天雄1枚，附子1枚，川乌1枚，桂心2两，观桂2两，桂枝2两，细辛2两，干姜2两，蜀椒2两。
出处：《张氏医通》卷十三。
主治：阴疽歹肉不化，腹痛泄泻，阳衰精冷，冷哮喘嗽，癥瘕冷积。
加减：None
功效：None
用法用量：麻油2斤浸，春5、夏3、秋7、冬10日，煎熬去滓，滤净再熬，徐下黄丹，不住手搅，滴水不散为度。阴疽，以葱汤洗净，摊成加银粉少许，贴患处；腹痛、少食、泄泻，摊成加丁香末少许，贴脐中及中脘；阳衰精冷，摊成加阿芙蓉少许，贴脐中及丹田；冷哮喘嗽，摊成加麝少许，贴肺俞及华盖、膻中；癥瘕冷积，摊成加麝香、阿魏少许贴患处。
制备方法：上切为片。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
