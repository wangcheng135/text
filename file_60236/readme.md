序号：60251
名称：逐瘀夺命丹
组成：益母草、白芷、泽兰、甘草、冬葵子、生地、丹皮、干姜、官桂、当归、附子、赤芍、南星、苏木、牛膝。
出处：《陈素庵妇科补解》卷五。
主治：产后儿已生，而胞衣不下，或儿生后，产母体疲，不能复用力，经停之间，外冷乘之，则血道阻涩或恶血流入胞中，衣为血所胀满，故胞衣不下也。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：①不孕症《陕西中医》（1986；1O：453）：张某，女，32岁。1983年4月30日诊。婚后7年未曾受孕，症见经期小腹冷痛，先后不定期，经量少色紫，夹有血块，面色不华，头痛唇干，心悸气短，腰痠肢冷，白带淋漓，舌黯苔白，脉细涩。辨证为瘀阻胞宫，寒凝脉络，气血两虚，津不上承。治宜活血化瘀，温宫通络，养血滋阴，益气调经。予本方加减化裁。处方：泽兰、益母草、当归、苏木、牛膝、赤白芍、冬葵子、生熟地、党参各10g，白芷、官桂、制附片、干姜、丹皮、甘草各6g，3剂。药后白带减少，经期小腹冷痛好转，诸恙亦有减轻，效不更方，守原方继进5剂。三诊诸症愈其大半，仍嘱原方连服7剂。四诊诸恙悉除，按原方略具增损还服7剂，以资巩固，后即怀孕得子。②闭经《陕西中医》（1986；10：453）：范某，女，18岁。1984年7月15日诊。于田间农活时，月经适来，突遇雨淋，回家后即感全身不适，继而畏寒发热，经治已愈。但月经未潮已半年。妇检未孕，诊为闭经。症见小腹冷痛，白带较多，形若蛋清，纳谷不香，有时恶心，头痛腰酸，舌暗红，苔薄白，脉沉涩。证属寒凝经脉，瘀滞胞宫。治应祛寒通经，逐瘀温宫。用本方化裁主治。处方：制附子、干姜、当归、泽兰、益母草、冬葵子、炒赤芍、苏木、川牛膝各10g，官桂、白芷、丹皮、制南星、甘草各6g，熟地20g，3剂。药后小腹冷痛轻，白带少，纳谷香，仍守原方连服5剂，闭经遂通。
各家论述：是方姜、桂、附子辛热以逐瘀；苏木、归尾以行血；白花、南星之燥，可以束胞中之水；冬葵、牛膝之滑利下行，可以使胞速下；益母、泽兰、芎、归、生地、丹皮、赤芍又能行血、养血、破血，胞中无瘀血填塞，则自下矣。
