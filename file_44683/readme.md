序号：44698
名称：宣解汤
组成：滑石1两，甘草2钱，连翘3钱，蝉蜕3钱（去足土），生杭芍4钱。
出处：《衷中参西》卷上。
主治：感冒久在太阳，致热蓄膀胱，小便赤涩；或因小便秘，而大便滑泻；兼治湿温初得，憎寒壮热，舌苔灰色滑腻者。
加减：若滑泻者，甘草须加倍。
功效：None
用法用量：None
制备方法：None
临床应用：风温：一叟，年六十五，得风温证。六七日间，周身悉肿，肾囊肿大似西瓜，屡次服药无效。旬日之外，求为诊视。脉洪滑微浮，心中热渴，小便涩热，痰涎上泛，微兼喘息，舌苔白厚。投以此汤，加生石膏一两，周身微汗，小便通利，肿消其半，犹觉热渴。遂将方中生石膏加倍，服后又得微汗，肿遂尽消，诸病皆愈。
