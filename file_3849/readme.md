序号：3849
名称：心咳汤
组成：北沙参3钱，石膏3钱（同薄荷头研），牛蒡子钱半，杏仁3钱（去皮尖），桔梗5分，甘草5分，麦冬3钱（去心），半夏1钱，茯神3钱，远志5分，小麦5钱。
出处：《退思集类方歌注》。
主治：心咳，咳则心痛，喉中介介如梗状，甚则咽肿喉痹。
加减：痰多，加川贝母；咽喉肿痛，去半夏；汗多，加五味子。
功效：None
用法用量：水3盏，先煮小麦减1盏，纳诸药，煎至1盏服。
制备方法：None
临床应用：None
各家论述：前人谓生脉散加茯神、远志，能治心咳。余因参入开泄肺经之药，重用小麦煎汤代水，治之乃验。盖小麦甘平，为心之谷，缓心宁气，大有殊功，即从厚朴麻黄汤意化出。
用药禁忌：None
附注：None
