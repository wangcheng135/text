序号：63768
名称：清热渗湿汤
组成：黄连1钱，茯苓1钱，泽泻1钱，黄柏（盐水炒）2钱，苍术1钱半，白术1钱半，甘草5分。
出处：《赤水玄珠》卷二。
主治：湿热证，面黄浮肿，肢节疼痛，疲困，烦渴，泄泻，溺赤。
加减：如单用渗湿，去连、柏，加陈皮、干姜。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：《医略六书》：方中黄连清心火，燥脾湿；黄柏清肾火，燥膀胱；苍术燥湿强脾；白术健脾燥湿；甘草缓中和胃；茯苓渗湿和脾；泽泻泻三焦湿热以通利膀胱也。使热降湿消，则津液四布，而口渴自止，溺亦清长，何患黄肿之不退哉，此消热渗湿之剂，为脾亏湿热之专方。
用药禁忌：None
附注：None
