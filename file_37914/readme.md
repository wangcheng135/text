序号：37924
名称：拨云膏
组成：黄丹4两（细研，水飞），炉甘石4两（用童便煅淬5-7次，研细，黄连水飞5-7次），青盐半钱，硇砂半钱，乳香半钱，雄黄半钱，川芎半钱，黄连半钱，枯白矾半钱，轻粉半钱，甘草半钱，密陀僧半钱，麝香半钱，龙脑半钱，当归半钱，白丁香半钱（研），朱砂3钱（研），没药3钱，海螵蛸（研，去甲）3钱。
出处：《瑞竹堂方》卷三。
主治：诸般眼疾，不问远年近日。
加减：None
功效：None
用法用量：热水泡开，点眼，不拘时候。
制备方法：上为细末，用白沙蜜15两慢火熬，初沸下黄丹，二沸下炉甘石，三沸下诸药末，不粘手为度，贮瓮盏内。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
