序号：65954
名称：虚劳神应膏
组成：枇杷叶50-60片（洗刷去毛），大梨3个（去皮心），白蜜半钟（先熬滴水成珠），大枣半斤，建莲肉4两（不去皮）。
出处：《卫生鸿宝》卷一。
主治：气血两虚，骨蒸劳热，身体羸瘦，四肢酸软，腰痛背痛，饮食不进，阴虚吐血，虚弱咳嗽。
加减：痰多，加川贝1两；吐血，加藕节捣汁同煮；不咳嗽者，去枇杷叶；大便溏泄者不用白蜜。
功效：益脏腑。
用法用量：随意温热吃。
制备方法：将枇杷叶砂锅内煎，取汁去叶，后入诸味拌，放锅内铺平，再将汁掩满，略高些，盖好，煮半炷香，翻转，又煮半炷香，枣煮好去皮，瓶贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
