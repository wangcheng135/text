序号：6255
名称：加味当归补血汤
组成：黄耆5钱，当归3钱，炙草钱半，防风钱半（盐水炒），羌活钱半（砂糖炒），竹沥1杯（冲），姜汁1杯（冲）。
出处：《医略六书》卷二十。
主治：产后去血过多，筋无血养，挛急发痉，脉浮软者。
加减：None
功效：None
用法用量：水煎去滓，冲2汁温服。
制备方法：None
临床应用：None
各家论述：产后血虚，不能荣养经络，而邪乘虚袭，故筋脉孪急而发痉焉，谓之虚痉。黄耆补虚生血，当归养血荣经，防风率领黄耆以益卫气，羌活统运炙甘草益胃气，以振运行之力，竹沥、姜汁活络行经，以除虚痉也。俾气能生血，则血液内充，而虚风自散，筋脉挛急自舒，何患虚痉之不退哉！此补气以统运营血之剂，为产后虚风发痉之专方。
用药禁忌：None
附注：None
