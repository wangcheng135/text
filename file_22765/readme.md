序号：22773
名称：沈氏血癥丸
组成：五灵脂5钱，大黄5钱，甘草梢5钱，桃仁泥5钱，生地7钱，牛膝4钱，官桂2钱，玄胡索6钱，归身6钱，三棱3钱，蓬术3钱，赤芍3钱，川芎3钱，琥珀1钱，乳香1钱，没药1钱。
出处：《杂病源流犀烛》卷十四。
主治：脏腑虚弱，寒热失节，或风冷内停，饮食不化，周身运行之血气适与相值，结而生块；或因跌仆，或因闪挫，气凝而血亦随结，经络壅瘀，血不散成块，而致血癥，心腹胁间苦痛，渐至羸瘦，妨于饮食。
加减：None
功效：None
用法用量：每服1钱，壮盛人1钱半。消过半即止，再随病体立方服药。
制备方法：酒糊为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：血癥丸（《类证治裁》卷三）。
