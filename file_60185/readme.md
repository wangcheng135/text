序号：60200
名称：逐气丸
组成：沉香2钱，破故纸（微炒）半两，槟榔半两，郁李仁25枚，黑牵牛4两（一半生，一半熟用），大皂角10挺（水浸，捶碎，滤去滓，慢火熬成膏，临膏将欲成，更下生蜜1匙，熬如稀饧相似，是膏也）。
出处：《御药院方》卷三。
主治：脾胃停饮，攻注腹胁，痞滞疼痛，或停痰饮，留溃胸膈，痞闷不快，或咳或喘，并水气流注，四肢浮肿，及大腹满。
加减：None
功效：None
用法用量：每服15丸至20丸，生姜汤下；食后如欲溏利，临时觑虚实加减服之；若治肿满及腹胀，用葱白汤送下。
制备方法：上将前5味为细末，用皂角膏和丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
