序号：57014
名称：消毒黄龙丸
组成：半夏半斤，茯苓2两，甘草（生）2两。
出处：《鸡峰》卷五。
主治：中暍烦躁，汗出身热，头疼，痰逆恶心，口燥多渴，胸膈不利，饮食减，昏困嗜卧；又暑毒热气内伏，久久不已，变成痎疟、黄疸，减食，日渐羸瘦。
加减：None
功效：None
用法用量：每服20丸，生姜汤送下；烦躁，新汲水送下，不拘时候。
制备方法：上为细末，生姜汁作糊为丸，如绿豆大。
临床应用：None
各家论述：《医方集解》：此方不治其暑，而治其湿。用半夏、茯苓行水之药，少佐甘草以和其中。半夏用醋煮者，醋能开胃散水，敛热解毒也。使暑气湿气俱从小便下降，则脾胃和而烦渴自止矣。
用药禁忌：None
附注：消暑丸（《局方》卷二绍兴续添方）、消毒丸（《三因》卷五）、黄龙丸（《永类钤方》卷二十一引《全婴方》）、消暑丹（《元戎》卷二）。
