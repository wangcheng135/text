序号：49168
名称：胜金元散
组成：白薇半两，人参3分，藁本3分，蒲黄3分，川乌头3分，丹参3分，吴茱萸2分，柏子仁2分，防风2分，厚朴2分，细辛2分，桂心1两1分，干姜1两1分，当归1两3分，芎藭1两3分，生干地黄8两，泽兰2两1分_x000D_
上除桂心外，同杵，以马尾罗子筛为粗末，重炒褐色勿焦，候冷，再杵为细末，入桂心末拌和匀，后分为两处；候合成，后药取一半，入在此药中；却将此药一半，入在后药中，丸子如后：_x000D_
延胡索3分，五味子3分，白芷3分，白术3分，石菖蒲3分，茯苓1两，桔梗1两，卷柏1两，川椒1两，黄耆1两，白芜荑1两3分，甘草1两3分，白芍药1两3分，石膏1两。
出处：《鸡峰》卷十五。
主治：风劳气冷，伤寒咳嗽呕逆，寒热不定，四肢遍身疮痒，血海不调，血脏虚惫，赤白带下，血运血崩，瘀血流入四肢，头痛恶心，血癥积滞，漏下，过期不产。丈夫肾脏虚风。
加减：None
功效：安胎，悦怿颜色。
用法用量：如有病证，每服用温酒调前散3钱，下此丸30丸；常服2钱，下此丸20丸。
制备方法：上药除石膏外同杵，以马尾罗子筛为粗末，重炒令褐色，候冷，依前再杵为细末，入石膏拌匀；亦分作两处，将一半换前药相和匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：妇人室女病至垂死，服之无不见效。若服丸子，不可无散子；服散子，不可无丸子。
