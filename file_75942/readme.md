序号：75958
名称：辟风汤
组成：独活（洗去土，焙干）1两，防风（去芦头）1两，吴白芷1两，桂藁本（去土）1两，麻黄（去节，微炒）1两，白芍药（去皮）1两，天麻1两，川乌头（炮裂，去皮，捶碎炒黄）半两，藿香叶（去土）半两，川芎7钱，羌活（去苗）3钱，甘草（锉，炒）半两，白花蛇（酒浸，去皮骨）半两，白僵蚕（炒黄）3钱，全蝎（去毒，炒黄色）半两，朱砂（为衣）2两，白附子（炮制，捣碎，炒微黄）4钱，天南星（牛胆酿，炒黄）4钱，远志（汤浸，去心，焙）3钱。
出处：《御药院方》卷一。
主治：诸风疾，无问新久者。半身不遂，口眼斜，语言謇涩，精神昏愦，痰涎并多，咽嗌不利；及风虚头痛目眩，旋运欲倒，或心忪健忘，恍惚不宁，手足麻痹，颤掉无力，筋脉拘急，骨节烦疼，行步艰难。
加减：None
功效：None
用法用量：每服1丸，细嚼或化服，用生姜汤送下，麝香汤亦得；如破伤风，豆淋酒送下；急风痫病，人参汤送下，不拘时候。
制备方法：上为细末，炼蜜为丸，每两作10丸，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
