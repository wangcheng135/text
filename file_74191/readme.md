序号：74207
名称：山药丸
组成：干山药1两半，牛膝（酒浸1宿，焙干）1两半，苁蓉（酒浸1宿，焙干）1两，石菖蒲1两，巴戟（去心）1两，楮实1两，山茱萸1两，五味子1两，远志（去白）1两，白茯苓（去皮）1两，杜仲（去皮，姜汁涂、炙）1两，枸杞子1两，茴香（盐炒）1两，熟地黄半两。
出处：《医方类聚》卷一五三引《经验秘方》。
主治：一切虚损，神志俱耗，筋力顿衰，腰脚沉重，身体倦怠，血气俱乏，小便浑浊。
加减：如身热，加山栀子1两；心气不宁，加麦门冬3两；精神短少，加五味子1两；阳弱，加续断1两。
功效：大补心肾脾胃。服后5日有力，10日精神爽健，半月气稍盛，20日明目，1月夜思饮食，冬月手常温；久服令人身轻体健，筋力壮盛，怡光难老；常服齿牢，永无瘴疟；妇人服之，滋养荣光。
用法用量：每服30丸，空心温酒或盐汤送下，1日3次。
制备方法：上为细末，蜜同枣肉为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
