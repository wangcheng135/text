序号：66459
名称：小儿化食丹
组成：神曲（焦）2两，山楂（焦）2两，麦芽（焦）2两，槟榔（焦）3两，莪术（醋制）1两，三棱（炒）1两，白丑（炒）2两，黑丑（焦）2两，大黄2两。
出处：《济南市中药成方选辑》。
主治：由伤食、伤乳引起的腹胀便秘，肚大青筋，或大便脓血。
加减：None
功效：消食，止痢。
用法用量：周岁每服1丸，周岁以上每服2丸，温开水送下，1日2次。
制备方法：上为细末，过80-100目细罗；取炼蜜（每药粉10两，约用炼蜜10两）与上药粉搅拌均匀，成滋润团块，分坨，搓条为丸，上药一料，约制5分重蜜丸608丸。
临床应用：None
各家论述：None
用药禁忌：忌食辛辣油腻之物。
附注：None
