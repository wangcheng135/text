序号：60642
名称：通宣丸
组成：巴豆15粒（去心膜）。
出处：《卫生家宝产科备要》卷七。
主治：产后崩中，状似鸡肝，寒热闭闷；或产后四肢浮肿及寒热；子烂腹中不下。
加减：None
功效：None
用法用量：每服5丸，熟水送下。
制备方法：上以生绢袋盛，于酽灰汁中煮10沸，取出用纸厚裹，压于重物下出油，研成霜，却取黑散（即原书琥珀黑神散）3钱匕，以无灰酒调成稀膏，入瓷器中，于铛内重汤煮令稠，入巴豆霜和合可丸，如绿豆大。
临床应用：死胎有村妇用毒药落胎不能下，子烂于腹中，脐穿时出小骨，病极，朱子东遂用此丸下枯骨一具而愈。
各家论述：None
用药禁忌：None
附注：原书治上症，先服琥珀黑神散，次服本方。
