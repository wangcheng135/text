序号：83184
名称：乌龙锭子
组成：大黄8两，五倍子4两，花粉4两，香附子4两，木鳖仁4两，蓉叶4两，萆麻仁4两，益母草4两，霜桑叶4两，苍耳草灰4两，皮消4两，雄黄4两，陈石灰4两，白及4两，苍术2两，黄柏2两，川乌2两，草乌2两，羌活2两，独活2两，生南星2两，生半夏2两，川芎2两，细辛2两，赤芍2两，白芷2两，甘遂2两，大戟2两，山茨菇2两。
出处：《扬州存济堂药局膏方》。
主治：肿毒，痰饮流注，跌打损伤。
加减：热，加猪胆汁；寒，加葱、姜汁。
功效：肿毒初起，敷之自散，已溃敷之不走，并拔脓收口。
用法用量：临用醋磨敷。
制备方法：共晒干，为末，用醋20斤，皂角净肉1斤，明矾4两，先熬去滓，下炒黑陈小粉8斤，再熬，俟于湿合用，倾在净桌上，即以前药末及榆面1斤，和入擦匀为锭。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
