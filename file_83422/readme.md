序号：83440
名称：乌金散
组成：槐白皮（锉）1两，猪牙皂荚1两，威灵仙（去土）1两，生干地黄1两，酸石榴皮（锉）1两，何首乌1两，青盐1两（以上7味锉细，泥固济，入罐子内，用瓦1片盖口，炭火10斤烧赤，放冷取出，研末），细辛（去苗叶）半两，升麻半两（并捣罗为细末），麝香1两（别研）。
出处：《圣济总录》(文瑞楼本)卷一二○。
主治：骨槽风，牙龈肿痒，风冷痛，齿宣有血。
加减：None
功效：牢牙、能髭鬃。
用法用量：每临卧用水调药半钱，涂在纸上，于牙龈上贴之，贴2-3次即愈。如早作齿药用尤妙。
制备方法：上为细末，相和令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
