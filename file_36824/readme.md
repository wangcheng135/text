序号：36834
名称：固真丹
组成：苍术（洗去泥土，干，米泔水浸，逐日易泔，春5日，夏3日，秋7日，冬10日，切作片子焙干）1斤（分作4份：4两用小茴香、青盐5分，同炒黄色为度；4两用川乌头切作片子，重5钱，川楝子1两，去皮并核，炒黄色为度；4两用酒半升，醋半斤，煮30来次；4两用川椒1两，去核，破故纸1两，同炒黄色为度）。
出处：《普济方》卷二二六引十便良方。
主治：诸虚不足，五劳七伤，元阳气虚，及小肠肾腧膀胱疝气，诸般淋疾，精滑精漏，小便白浊；及妇人赤白带下，漏下血崩，子宫久冷，血海虚冷，面色痿黄，产前后诸般冷病。
加减：None
功效：None
用法用量：每服20丸，空心盐汤或酒送下；妇人醋汤送下。此药性温无毒，小便频数为效。
制备方法：上为细末，用酒醋打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：神仙补益固真丸（原书同卷）、四炒丹（《医方类聚》卷八十八引《简易方》）、四妙丹、神仙固真丸（《普济方》卷三十三）、四炒固真丹（《医学入门》卷七）、固元丹（《本草纲目》卷十二）。
