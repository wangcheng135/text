序号：53911
名称：振颓汤
组成：生黄耆6钱，知母4钱，野台参3钱，于术3钱，当归3钱，生明乳香3钱，生明没药3钱，威灵仙1钱半，干姜2钱，牛膝4钱。
出处：《衷中参西》上册。
主治：痿废。
加减：热者，加生石膏数钱，或至两许；寒者，去知母，加乌附子数钱；筋骨受风者，加明天麻数钱；脉弦硬而大者，加龙骨、牡蛎各数钱，或更加山萸肉亦佳；骨痿废者，加鹿角胶、虎骨胶各2钱（另炖同服），然二胶伪者甚多，若恐其伪，可用续断、菟丝子各3钱代之；筋骨受风者，加明天麻；手足皆痿者，加桂枝尖2钱。
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：方中黄耆以补大气；白术以健脾胃；当归、乳香、没药以流通血脉；灵仙以祛风消痰；恐其性偏走泄，而以人参之气血兼补者佐之；干姜以开气血之痹；知母以解干姜、人参之热，则药性和平，可久服而无弊。其阳明有实热者，加石膏以清阳明之热，仿《金匮》风引汤之义也；营卫经络有凝寒者，加附子以解营卫经络之寒，仿《金匮》近效术附汤之义也；至其脉弦硬而大，乃内风煽动，其气不固之象，故加龙骨、牡蛎以息风敛真气；骨痿者加鹿胶、虎胶取其以骨补骨也；筋骨受风者，加明天麻取其能搜筋骨之风，又能补益筋骨也；若其痿专于腿，可但用牛膝以引之下行，若其人手足并痿者，又直加桂枝兼引之上行。
用药禁忌：None
