序号：34809
名称：麦门冬饮
组成：麦门冬25个，米25粒。
出处：《千金》卷二十一。
主治：肺热水肿，或疮疡误灸，火毒入里，浮肿而喘者。水气肿鼓胀，小便不利。肺热失其降下之令，不能通调水道，下输膀胱，渍于高源，淫于皮肤，肢体皆肿，少腹不急，初病便有喘满。痈疽阴疮，法当艾灸，或灸太过者，或阳疮不应灸而误灸者，以致火毒入里，令患者头项浮肿，神昏痰涌，吁吁作喘。
加减：None
功效：None
用法用量：以水1升，和煮米熟，去滓，食后送下丸药4丸，1日3次。以喉中干，口粘浪语为候。数日小便大利，佳。
制备方法：None
临床应用：None
各家论述：《医方考》：方中麦门冬清肺，以开其降下之源；粮米益脾，而培乎金之母气。此治病必求其本也。
用药禁忌：None
附注：麦门冬汤（《医方集解》）、麦冬粳米饮（《金鉴》卷六十一）、麦门冬粳米汤（《医林纂要》卷四）。用法中之丸药，用莨菪子一升，羖羊肺一具。先洗羊肺，汤微渫之，薄切，晒干作末，以三年大酢渍莨菪子一晬时，出，熬令变色，熟捣如泥，和肺末，炼蜜为丸，如梧桐子大。《医方考》、《医方集解》、《金鉴》等用本方治上证，均不用此丸。
