序号：75679
名称：葱豉汤
组成：葱白1虎口，豉1升。
出处：《肘后方》卷二。
主治：伤寒初起，头痛身热，脉浮大。利兼吐逆及呕者；妊娠热病；服乳石，固食仓米臭肉发动者；酒病。
加减：None
功效：None
用法用量：上以水3升，煮取1升，顿服取汗。不汗复更作，加葛根2两，升麻3两，水5升，煎取2升，分再服，必得汗；若不汗，更加麻黄2两，又用葱汤研米2合，水1升，煮之，少时下盐、豉，后纳葱白4物，令火煎取3升，分服取汗。
制备方法：None
临床应用：None
各家论述：《医方集解》：此足太阳药也。葱通阳而发汗，豉升散而发汗，邪初在表，宜先服此以解散之。
用药禁忌：None
附注：葱白豉汤（《活人书》卷十九）。
