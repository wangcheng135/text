序号：26789
名称：延胡索丸
组成：延胡索（米醋炒黄）3分，当归（切焙）半两，沉香（锉）半两，木香1两半，白术1两半，芎1两半，青橘皮（汤浸去白，焙）1两半，附子（炮裂，去皮脐）1两半，吴茱萸（汤洗，焙干，炒）1两半，桂（去粗皮）1两半，京三棱（湿纸裹煨，别捣为末）1两半，篷莪术（锉炒）1两。
出处：《圣济总录》卷一五三。
主治：妇人血脏久冷，血积气攻，心腹脐下疼痛，呕逆痰涎，不思饮食。
加减：None
功效：None
用法用量：每服20丸，煎生姜醋汤送下。日进3服，不拘时候。
制备方法：上为末，以酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
