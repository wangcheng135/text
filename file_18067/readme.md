序号：18074
名称：白术散
组成：白术1分，附子3分（炮裂，去皮脐），当归3分（锉，微炒），桂心半两，陈橘皮3分（汤浸，去白瓤，焙），人参3分（去芦头），木香半两，槟榔半两，干姜半两（炮裂，锉），赤芍药半两，芎藭3分，甘草1分（炙微赤，锉），吴茱萸1分（汤浸7遍，焙干，微炒），厚朴3分（去粗皮，涂生姜汁炙令香熟）。
出处：《圣惠》卷八十一。
主治：产后冷气攻心腹疼痛，四肢不和，少思饮食。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加大枣3个，煎至6分，去滓稍热服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
