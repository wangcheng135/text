序号：40057
名称：肾沥汤
组成：肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），汉椒半两（去目及闭口者，微炒去汗），五味子半两，附子1两（炮裂，去皮脐），干姜半两（炮裂，锉），人参3分（去芦头），黄耆3分（锉），泽泻3分，芎3分，牛膝3分（去苗），当归半两（锉，微炒），石斛3分（去根，锉），磁石2两（捣碎，水淘去赤汁，以帛包之），桂心半两。
出处：《圣惠》卷七。
主治：膀胱及肾脏虚冷，小便色白稠浊，日夜数无常，腰胁疼痛。
加减：None
功效：None
用法用量：每服5钱，以水1大盏，用羊肾1对（切去脂膜），大枣3枚，与磁石包子同煎至5分，去滓，食前温服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
