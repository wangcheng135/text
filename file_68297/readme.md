序号：68313
名称：黄耆散
组成：黄耆1两（锉），桂心半两，当归半两（锉，微炒），桑寄生半两，白茯苓半两，白芍药半两，人参半两（去芦头），牛膝3分（去苗），熟干地黄半两，麦门冬半两（去心，焙），鳖甲1两（涂醋，炙令黄，去裙襕），甘草半两（炙微赤，锉）。
出处：《圣惠》卷八十。
主治：产后蓐劳，或增寒壮热，四肢酸疼，头痛心烦。
加减：None
功效：None
用法用量：每服用豮猪肾1对，切去脂膜，先以水1大盏，入生姜半分，大枣3枚，煎至7分，去滓，下散5钱，更煎至4分．去滓，每日空心及晚食前温服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：黄耆煮散（《圣济总录》卷一六四）。本方方名，《普济方》引作“黄耆炙散”。
