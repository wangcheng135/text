序号：8637
名称：玄英丸
组成：雄黄1分（细研），牛黄1分（细研），龙脑半分（细研），白附子1两（生用），白花蛇1两（酒浸，去骨，炙微黄），天麻1两，白僵蚕1两（微炒），半夏半两（汤浸7遍去滑），天南星1两（生用），天雄1两（去皮脐，生用），麝香半分（细研），独活1两，干蝎1两（微炒），铅霜半两（细研），蝉壳1分，芎藭10两，腻粉1分，犀角屑1分，马牙消1两（烧令通赤，放冷出火毒），硫黄半两，水银半两（并硫黄结为砂子细研）。
出处：《圣惠》卷二十五。
主治：一切风。
加减：None
功效：None
用法用量：每服7丸，以薄荷酒研下，日3-4次。
制备方法：上为末，都研令匀，炼蜜为丸，如豇豆大。
临床应用：None
各家论述：None
用药禁忌：中风四肢缓弱，宜于淋浴后服，厚盖出汗，避风。忌猪鸡毒滑动风物。
附注：玄英丹（《普济方》卷一一四）。
