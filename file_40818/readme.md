序号：40833
名称：软金丹
组成：使君子（炒，为末）1分，香墨（烧研）1分，青黛（细研）1分，麝香（细研）1分，腻粉（研）1分，胡黄连（为末）1分，寒食面7钱半，天浆子7个（炒为末）。
出处：《局方》卷十。
主治：小儿惊疳积聚，壮热羸瘦，手足抽搐，目睛上视，项背强硬，牙关紧急，多睡昏愦，痰涎壅塞。小儿惊风壮热，多睡惊掣，精神昏愦，痰涎壅塞。小儿急惊，目睛直视，小儿急慢惊风。
加减：None
功效：治惊疳，下积聚。
用法用量：每服1丸，煎金银薄荷汤化下。5岁以上可服2丸，更量大小、虚实加减。不拘时候。
制备方法：上合研匀。以白面糊为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：软金丸（《圣济总录》卷一六九）。
