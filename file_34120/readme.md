序号：34129
名称：牡蛎汤
组成：牡蛎4两（熬），麻黄4两（去节），甘草3两（炙），蜀漆3两（若无用常山代之）。
出处：《外台》卷五引《伤寒论》。
主治：牝疟多寒者。
加减：None
功效：None
用法用量：上4味，以水先洗蜀漆3遍去腥，（口父）咀，以水8升，煮蜀漆及麻黄去沫，取6升，纳2味，更煎取2升，去滓，温服1升，即吐，勿更服。
制备方法：None
临床应用：None
各家论述：1.《千金方衍义》：此方中牡蛎即蜀漆散中龙骨之意，蜀漆得云母专升阳邪陷阴，故以纯阳之龙骨为佐；此方中麻黄即蜀漆散中云母之意，蜀漆得麻黄专开阴邪之固闭，故以纯阴之牡蛎为辅；甘草调和药性之阴阳也。_x000D_
2.《医门法律》：牡蛎汤一方，同治牡疟者，初感病时，风寒未清，传变为疟，结伏心下，故方中用麻黄以散风寒，并借之以通阳气耳。
用药禁忌：忌海藻、菘菜。
附注：None
