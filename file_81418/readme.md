序号：81434
名称：露蜂房丸
组成：露蜂房2钱（炒），蝉壳2钱（去土，炒），蛤蚧1只（重4钱，酥涂、炙干），丁香2钱，木香2钱，人参（去芦）2钱，地黄2钱，麻黄（去节根）2钱，马兜铃子2钱，五倍子（去虫）2钱，五味子（去枝梗）2钱半，贝母（去心，焙）2钱半，杏仁（童子小便浸2宿，去皮尖，炒）2钱半，半夏曲2钱半，款冬花（去枝梗）半两。
出处：《卫生总微》卷十四。
主治：肺胃虚寒，咳嗽喘满，呕逆不食。
加减：None
功效：None
用法用量：食后服20-30丸，生姜汤送下，每日3-4次。
制备方法：上为细末，炼蜜和丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
