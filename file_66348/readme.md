序号：66364
名称：鹿肉汤
组成：鹿肉3斤，芍药3两，半夏1升，干地黄2两，独活3两，生姜6两，桂心1两，芎1两，甘草1两，阿胶1两，人参4两，茯苓4两，秦艽3两，黄芩3两，黄耆3两。
出处：《千金》卷三。
主治：产后中风，风虚头痛，壮热，言语邪僻。
加减：None
功效：None
用法用量：上(口父)咀。以水2斗，煮肉得1斗2升，去肉纳药，煎取3升，去滓，纳胶令烊，分4服，日3夜1。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：产后为虚风所袭，非峻培气血，助其祛风之力，弗g有济。恐草木无情，不能速为取效，故取血肉之味，稍兼独活、秦艽鼓舞参、耆之性，不能助力祛邪，兼杜虚风复入，真补中寓泻之良法也。
用药禁忌：None
附注：None
