序号：80620
名称：燥脾丸
组成：羌活1两半，防风1两半，苍术1两半（制），半夏1两半（制），白芷1两半，陈皮1两半，柴胡8钱，升麻8钱，甘草5钱。
出处：《医略六书》卷二十一。
主治：风湿伤脾，两胞弦烂，脉浮缓者。
加减：None
功效：None
用法用量：每服3钱，米饮送下。
制备方法：上为末，滚水糊为丸。
临床应用：None
各家论述：风湿伤脾，清阳下陷，不能敷化精微，故湿渍两胞，眼弦湿烂。羌活散风胜湿，苍、半燥湿强脾，白芷散阳明之邪，陈皮利太阴之滞，柴胡升少阳清气，升麻升阳明清气，甘草缓中州以和胃气也。使脾胃调和，则清阳上举，而风湿两除，眼胞眩烂无不愈矣。此散风燥湿之剂，为风湿伤脾弦烂之专方。
用药禁忌：None
附注：None
