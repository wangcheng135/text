序号：26451
名称：当归汤
组成：当归（切，焙）2两，甘草（炙，锉）2两，芎2两，远志2两（去心），麦门冬（去心，焙）2两，芍药2两，赤茯苓（去黑皮）2两，生干地黄（焙）4两，黄芩（去黑心）1两，桂（去粗皮）1两，五味子3两。
出处：《圣济总录》卷一八四。
主治：乳石发后，虚热羸乏，胸膈痞滞，心腹胀满，或腰痛肾沥。
加减：None
功效：None
用法用量：每服5钱匕，先以水5盏，入白羊肾1个（去筋膜，切），煎至3盏，去肾下药，加生姜1枣大（拍碎），大枣3个（擘破），更煎至2盏，去滓，空心、食前分2次温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
