序号：75234
名称：腽肭脐丸
组成：腽肭脐1两（酒刷，炙微黄），鹿茸1两（去毛，涂酥，炙微黄），肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），菟丝子1两（酒浸3日，晒干，别捣为末），阳起石1两（酒煮半日，细研，水飞过），附子1两（炮裂，去皮脐），茴香子1两，桂心3分，山茱萸3分，独活3分，天麻3分，干蝎3分（微炒），人参3分（去芦头），石斛1两（去根，锉），芎1两，木香1两，补骨脂1两（微炒），白术3分，荜拨3分，熟干地黄1两，牛膝3分（去苗），远志1两（去心），铁粉1两（细研），槟榔1两，朱砂1两（细研，水飞过），麝香半两（细研）。
出处：《圣惠》卷九十八。
主治：肾脏风虚冷气。
加减：None
功效：温脾胃，思饮食，安心志，强气力，利腰脚，补虚损。
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，入诸研了药合匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
