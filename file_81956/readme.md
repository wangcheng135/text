序号：81972
名称：镇心丸
组成：紫石英3分，茯苓3分，菖蒲3分，苁蓉3分，远志3分，大黄3分，大豆卷3分，麦门冬3分，当归3分，细辛3分，卷柏3分，干姜3分，防风6分，人参6分，泽泻6分，秦艽6分，丹参6分，石膏3分，芍药3分，柏子仁3分，乌头7分，桂心7分，桔梗7分，甘草7分，薯蓣7分，白蔹2分，铁精2分，银屑2分，前胡2分，牛黄2分，白术8分，半夏8分，干地黄12分，（庶虫）虫12枚，大枣50枚。
出处：《千金》卷十四。
主治：虚损，梦寤惊悸，风邪鬼注、寒热往来，腹中积聚，忧恚结气；男子失精；妇人赤白注漏，或月水不利。
加减：None
功效：None
用法用量：每服5丸，加至20丸，日3次。
制备方法：上为末，蜜、枣和丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
