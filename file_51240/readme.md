序号：51255
名称：药艾条
组成：艾叶2.4kg，桂枝125g，高良姜125g，广藿香50g，降香175g，香附50g，白芷100g，陈皮50g，丹参50g，生川乌75g，雄黄12.5g。
出处：《中国药典》一部。
主治：风寒湿痹，肌肉酸麻，关节四肢疼痛，脘腹冷痛。
加减：None
功效：行气血，逐寒湿。
用法用量：直射灸法，1次适量，红晕为度，1日1-2次，或遵医嘱。
制备方法：上药，将艾叶碾至成艾绒，雄黄水飞，或粉碎成极细粉，其余9味粉碎成细粉，与雄黄粉末配研，过筛，混匀。先取艾绒20g，均匀平铺在一张长28厘米，宽15厘米白棉纸上，再均匀散布上述粉末88g，将棉纸两端折叠约6厘米，卷紧成条，粘合封闭，晒干即得。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
