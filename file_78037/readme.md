序号：78053
名称：赛金化毒散
组成：乳香1钱，没药1钱（出汗，研细，配众药），川贝母（去心，炒）1钱，雄黄1钱，黄连1钱，天花粉1钱（生用），大黄（半炒半生）2钱，甘草7分，赤芍2钱（炒），冰片1分半，牛黄2分，珠子4分（研至无声为度）。
出处：《痘疹金镜录》卷下。
主治：痘疮。小儿疹后，余毒未净，烦躁口渴，便秘，及小儿疮疖溃烂。
加减：None
功效：清热化毒。
用法用量：痘内有伏毒，啼号不已，经日不起，并发痈发疔者，用蜜汤调服此散。痘若抓伤或攒簇堆聚，或点见果恶干焦，紫黑板硬等象，以此散调入油胭脂内，用绵纸为膏药，贴之。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：化毒散（《痘疹金镜录》卷下）。
