序号：55711
名称：桃花行气丸
组成：桃花7钱，染胭脂5钱，白附子2钱，甘遂2钱，天南星2钱，大麦面10钱，威灵仙5钱，大戟3钱。
出处：《鸡峰》卷四。
主治：温湿风毒，客伏经络，气壅血涩，肢体闷痛，懊烦壮热，气奔逆冲，心腹痞满，腰脊重痛，脚膝难举，痛不能动，肩背拘强，眩冒神昏；或将温过度，客热内盛，随气升降，游注疼痛，及肿满水气。
加减：None
功效：疏风气。
用法用量：每服20丸，先用水1大盏煎桑白皮3寸，煎5-7沸，去皮不用，纳药再煮5-7沸，滤出并放温，将丸煮药汤下，食后临卧服。
制备方法：上为末，生用，滴水为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
