序号：45500
名称：千金丸
组成：甘草6铢，贝母6铢，秦椒6铢，干姜6铢，桂心6铢，黄芩6铢，石斛6铢，石膏6铢，粳米（1作糯米）6铢，大豆6铢，黄卷6铢，当归13铢，麻子3合（1方用蒲黄1两）。
出处：《千金》卷二。
主治：产难，胞衣不下。
加减：None
功效：养胎。
用法用量：每服1丸，用枣汤送下；1日3次；产难颠倒，胞不出，服1丸；伤毁不下，产余病汗不出，烦满不止，气逆满，以酒服1丸。
制备方法：上为末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：《千金方衍义》：此即前甘草散之变法，甘草散方中用桂、姜、芩、甘、麻、豆等味，为滑胎而设。此治产难胞衣不下，亦不出此，但加石斛、石膏、贝母、粳米，以清胃之上逆；秦椒以下恶气，与茱萸同为止逆下气之味。但彼用散，以利运动之机；此用丸，以祛毁伤之滞，取甘草散之变法，而为产难之变治也。
用药禁忌：None
附注：None
