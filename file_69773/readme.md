序号：69789
名称：琥珀珍珠丸
组成：巴豆7枚（取霜），附子尖14枚（半生，半炮），半夏10枚（半生，半炮），白花蛇头1枚（酒浸，焙干），白僵蚕14枚（一半生，一半用薄荷叶裹炙焦，去薄荷不用），白附子2枚（半生，半炮），全蝎14枚（一半生，一半用薄荷叶裹炙令黄，去薄荷），天南星3钱（半生，半炮），羌活（去芦头）1钱，白鲜皮1钱，琥珀（别研）1钱，天麻1钱，珍珠末（别研）1钱，朱砂（别研）1钱，龙齿（火煅）1钱，雄黄半钱（别研），麝香1字（别研）。
出处：《杨氏家藏方》卷十七。
主治：小儿急、慢惊风，涎留心经，上逆作吐，目睛直视，手足搐搦，及风痫瘈疭，涎潮昏塞，嚼舌摇头，作声狂叫。
加减：None
功效：None
用法用量：每服10丸，煎人参汤送下，不拘时候。
制备方法：上为细末，以猪心血同薄荷自然汁煮面糊为丸，如萝卜子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
