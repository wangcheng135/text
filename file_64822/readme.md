序号：64838
名称：续命汤
组成：人参2两，甘草（炙）2两，干姜2两，麻黄（去节）2两，独活2两，当归2两，芎2两，石膏（碎，绵裹）2两，附子1枚（炮），桂心3分，白术3分，细辛3分，防风5分，芍药2分，秦艽1两，杏仁40枚（去两仁尖皮），黄芩1两。
出处：《外台》卷十五引《深师方》。
主治：大风，风邪入心，或心痛彻背，背痛彻心，去来上下，惊恐，小腹胀满微痛，乍寒乍热，心中闷状如微温，进退无常，面赤或白或黄。
加减：None
功效：调和六腑，安五脏。
用法用量：上药，以水1斗，煮麻黄10余沸，纳诸药，煮取4升半，去滓，纳大枣10个，煎取3升，分5服，老小者5合，此以下以意消息。无芎，防己代之：无独活，天雄代之；无附子，乌头代之。汤成之后，服汤以椒10枚置汤中，温令暖服之。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌海藻、生葱、猪肉、桃、李、生菜、雀肉。
附注：None
