序号：70821
名称：紫金锭子
组成：炉甘石半斤，黄丹半斤，黄连（另研）1两，朱砂1两，当归半两，硼砂半两，海螵蛸1钱2分半，白丁香1钱2分半，白矾（生）1钱2分半，硇砂1钱2分半，轻粉1钱2分半，贝齿1钱2分半，珍珠1钱2分半，石蟹1钱2分半，熊胆1钱2分半，乳香1钱2分半，没药1钱2分半，麝香1钱2分半，片脑2钱（其片脑久留恐去气味，宜临用时加入；上除脑、麝外，余各别研制为末称，合和匀，入黄连水（石禹）至千万余下，晒干，次入麝香研细罗过，又次入片脑再研，复罗入后膏和作锭子，阴干），黄连1斤，当归4两，生地黄4两，防风2两，黄柏2两，龙胆草2两，蕤仁半两，诃子8枚，冬蜜8两（另熬，酥干为度），鹅梨8枚（取汁），猪胰子4枚（以稻草挪洗，去膏膜干净无油为度，再用布包捣烂入药）。
出处：《准绳·类方》卷七。
主治：一切眼疾，不分远年近日，诸般翳膜，血灌瞳仁，胬肉攀睛，拳毛倒睫，积年赤瞎，暴发赤肿，白睛肿胀，沙涩难开，眊矂紧涩，怕日羞明，眵多(目蔑)泪，烂眩风痒，视物昏花，迎烟泪出，目中溜火。
加减：None
功效：None
用法用量：每以少许新汲水浸化开，鸭毛蘸点眼大眦内，又可以热水泡化洗眼，药水冷又暖洗，日洗5-7次，日点10余次。
制备方法：上将黄连等8味洗净锉碎，以水浸于铜器内，春5、夏3、秋4、冬7日，滤去滓，以滓复添水熬3次，取尽药力，以蜜绢绵纸重滤过，澄去砂土，慢火煎熬；槐，柳枝各49条，互换一顺搅，不住手，搅尽枝条如饴糖相类，入蜜和匀，瓷器盛，放汤瓶口上重汤蒸顿成膏，复滤净，滴入水中沉下成珠，可丸为度，待数日出火毒，再熔化入末和匀，杵捣为丸锭，阴干，金银箔为衣。
临床应用：None
