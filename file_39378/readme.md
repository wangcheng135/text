序号：39388
名称：知母散
组成：知母3分，桔梗半两（去芦头），紫菀3分（洗去苗土），桑根白皮3分（锉），柴胡1两（去苗），人参半两（去芦头），赤芍药3分，半夏3分（汤洗7遍，去滑），秦艽1两（去苗），地骨皮1两，甘草3分（炙微赤，锉），生干地黄1两，天门冬1两半（去心，焙），赤茯苓1两，黄芩3分，鳖甲1两半（涂醋，炙令黄，去裙襕）。
出处：《圣惠》卷三十一。
主治：热劳。身体壮热，皮毛干枯，痰唾调粘，四肢疼痛，食少无力，渐加羸瘦。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：忌苋莱、醋物。
附注：None
