序号：73660
名称：缠金丹
组成：硇砂2钱半，乳香2钱半，杏仁（去皮尖）8钱半，巴豆（去皮心膜，出油）8钱半，黄蜡1两，朱砂1两，木鳖半两，白胶香1钱，黄丹2两半，砒霜（醋煮，煅）3钱半。
出处：《局方》卷六（吴直阁增诸家名方）。
主治：大人小儿一切泻痢，无问冷热赤白，连绵不愈，愈而复发，腹中疼痛者。
加减：None
功效：None
用法用量：每服1丸，小儿半丸，水泻，新汲水送下；赤痢，甘草汤送下；白痢，干姜汤送下；赤白痢，甘草干姜汤送下；并放冷，临卧服。
制备方法：上研为细末，熔蜡为丸，如麻子仁大。
临床应用：None
各家论述：None
用药禁忌：孕妇莫服。忌热物一二时辰。
附注：None
