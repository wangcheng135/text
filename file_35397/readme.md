序号：35407
名称：万病散
组成：黄耆半两，木通半两，桑白皮半两，陈皮半两，白术半两，木香半两，胡椒半两（作1服，别研置），牵牛子（微炒，勿过热，取头末）1两。
出处：《幼幼新书》卷三十九引《灵苑方》。
主治：诸风疮肿疥癣，脏腑积冷塑滞，结为风劳，膀胱宿冷，脏腑衰败，面黄，癥癖气块，疳蛔攻心痛，中寒脑痛，状如山岚时疫；或中风口、语謇，睡后涎出；久患腰膝疼痛；拜跪艰难，坐食不安；小儿疳痢脱肛，男女久泄气痢，状似休息；妇人久患血劳，痿黄无力。
加减：None
功效：消积快气，散饮逐湿。
用法用量：每服2钱，牵牛末2钱，天色晴明五更，姜汤小半盏调药顿服，更以汤送。平明宣2-3行，下多不妨，应脏腑百病悉出，转后吃1日白粥补。
制备方法：前7味，为细末。
临床应用：None
各家论述：None
用药禁忌：有孕妇人或遇阴晦时即不可服。
附注：无忧散（原书同卷）、万病无忧散（《御药院方》卷四）。
