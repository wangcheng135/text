序号：5253
名称：平肝开郁止血汤
组成：白芍1两（醋炒），白术1两（土炒），当归1两（酒洗），丹皮3钱，三七根3钱（研末），生地3钱（酒炒），甘草2钱，黑芥穗2钱，柴胡1钱。
出处：《傅青主女科》卷上。
主治：妇人怀抱甚郁，口干舌渴，呕吐吞酸，而血下崩者。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：方中妙在白芍之平肝，柴胡之开郁，白术利腰脐，则血无积住之虞；荆芥通经络，则血有归还之乐；丹皮又清骨髓之热，生地复清脏腑之炎，当归、三七于补血之中，以行止血之法。自然郁结散而血崩止矣。
用药禁忌：None
附注：平肝止血汤（《辨证录》卷十一）。
