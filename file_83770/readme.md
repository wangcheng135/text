序号：83788
名称：乌蛇丸
组成：乌蛇1两（用肉，酥炙黄），天麻1两，独活1两，附子（炮）1两，肉桂1两，人参3分，防风3分，细辛3分，当归（炒）3分，白术3分，羚羊角屑3分，薏苡仁3分，干蝎（炒）3分，牛膝3分，芎3分，茯苓3分，天南星3分，白僵蚕（炒）3分，麻黄1两半（去节），牛黄1分，龙脑1分，麝香1分（研入），朱砂半两（研入）。
出处：《医方类聚》卷二十引《神巧万全方》。
主治：脾脏中风，身体怠惰，四肢缓弱，恶风头痛，舌本强直，言语謇涩，皮肤顽痹。
加减：None
功效：None
用法用量：每服10丸，酒送下，加至15丸。
制备方法：上为末，入研了药和匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
