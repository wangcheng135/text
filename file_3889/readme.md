序号：3889
名称：文蛤散
组成：文蛤5两。
出处：《伤寒论》。
主治：伤寒病在阳，应以汗解之，反以冷水潠之，若灌之，其热被劫不得去，弥更益烦，肉上粟起，意欲饮水反不渴者；渴欲饮水不止者。
加减：None
功效：None
用法用量：每次1方寸匕，以沸汤5合和服。
制备方法：上为散。
临床应用：None
各家论述：《金鉴》：渴欲饮水，水入则吐，小便不利者，五苓散证也；渴欲饮水，水入则消，口干舌燥者，白虎人参汤证也。渴欲饮水而不吐水，非水邪盛也；不口干舌燥，非热邪盛也。惟引饮不止，故以文蛤一味，不寒不温，不清不利，专意于生津止渴也。或云：文蛤即今吴人所食花蛤，性寒味咸，利水胜热，然屡试而不效。尝考五倍子亦名文蛤，按法治之名百药煎，大能生津止渴，故尝用之，屡试屡验也。
用药禁忌：None
