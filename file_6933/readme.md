序号：6933
名称：四明丸
组成：芎藭1两，天麻1两（用水煮过，切，焙），半夏1两（水煮，洗去涎，切，焙），桑螵蛸1两（大者，锉，炒），旋覆花1两，羌活1两（去芦头），藁本1两（择粗者，洗，焙干），天南星1两（炮），青橘皮1两（汤浸，去白，焙），附子（炮裂，去皮脐）1两。
出处：《圣济总录》卷一○七。
主治：风毒气上，眼目昏暗，及偏正头疼，两目渐觉细小；及有夹脑风疼，目风眼寒。
加减：None
功效：None
用法用量：每服20-30丸，空心，临卧盐汤或米饮送下。
制备方法：上为末，用生牵牛3两，熟牵牛3两，杵取末2两，与前药末拌匀，生姜汁煮，面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
