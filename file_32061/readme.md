序号：32070
名称：万应丸
组成：没药半两（研），乳香半两（研），木香半两，白附子半两（炮），乌药半两，蒺藜子半两（炒，去角），乌头半两（炮裂，去皮脐）半两，硇砂（研）1分。
出处：《圣济总录》卷八十三。
主治：干湿脚气。
加减：None
功效：None
用法用量：空心冷酒送下10丸。如人行3-5里，吃荆芥茶半盏，每日1次。渐加至15丸止。以觉药力到脚膝下为效。
制备方法：上药各为细末。用木瓜2枚，去皮瓤，剜内空，留盖子，将硇砂入木瓜内，盖了，用竹签子签定，以湿纸3-5重裹，热灰煨令香熟，取出不令倾侧，别顿一处；次用好酒1升，银、石锅内煎，先下蒺藜、乳香末，熬去1-2分；次下诸药末同熬，少时取出，不得令干；却将煨了木瓜同捣三五百杵，可丸即丸，如豌豆大。
临床应用：None
各家论述：None
用药禁忌：None
