序号：449
名称：五疰丸
组成：丹砂（研）1两，雄黄（研）1两，附子（炮）1两，甘遂半两（熬），豉60粒（熬），巴豆60枚（去心皮，熬令变色）。
出处：《外台》卷十三引《华伦录佚》。
主治：中恶，五疰、五尸入腹。胸胁急痛，鬼击客忤，停尸垂死。
加减：None
功效：杀鬼解毒。破积去水。
用法用量：若有急疾，服胡豆2丸。不觉更益，以饮投之。若已口噤，将物强发开，若不可发，扣齿折以灌下药汤酒随进之，即效。
制备方法：上药治下筛，巴豆别研令如脂，乃更合捣取调。白蜜和之，藏以密器。
临床应用：None
各家论述：None
用药禁忌：忌生血物，猪肉、芦笋。
附注：雄黄丸(《圣惠》卷五十五)。《圣惠》：服后利不止，与酢饭一两匙止之。
