序号：3308
名称：天雄石斛丸
组成：天雄（炮裂，去皮脐）1两，石斛（去根）1两，肉苁蓉（酒浸，去皱皮，切，焙）1两，牛膝（酒浸，切，焙）1两，独活（去芦头）1两，巴戟天（去心）1两，桂（去粗皮）2两半，补骨脂（炒）2两半，生干地黄（焙）3两半。
出处：《圣济总录》卷一八五。
主治：None
加减：None
功效：壮元气，去风冷，益精髓，长肌肉。
用法用量：每服20丸，空心温酒或盐汤送下，如药稠，更入熟蜜。
制备方法：上为末；腽肭脐3两细锉，以酒2升，浸两宿，去筋膜，研取肉，布绞滤去滓，文武火煎膏，和前药末，捣为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
