序号：53347
名称：健脾养胃丸
组成：人参5钱，白术（土炒）1两，白茯苓1两，广陈皮1两，当归（酒洗）1两，白芍药（炒）1两，麦芽（炒）1两，木香5钱，半夏曲1两，山药2两，枳实5钱。
出处：《摄生秘剖》卷二。
主治：脾胃虚弱。
加减：None
功效：None
用法用量：每服3钱，食后白汤送下。
制备方法：上为末，陈米糊为丸，如椒目大。
临床应用：None
各家论述：人参、白术以益其气，当归、白芍以滋其血，木香、陈皮以利其滞，麦芽、枳实以消其积，半、苓、山药以燥其湿。湿者燥之，积者消之，滞者利之，血主濡之，气主呵之，则不刚不柔，无过不及之患。脾胃既得其养，又安有不强健者哉。
用药禁忌：None
附注：None
