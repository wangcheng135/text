序号：34711
名称：麦门冬汤
组成：麦冬（去心）、人参、白术、陈皮、川芎、半夏、当归、肉桂、乌梅、大附子、甘草、茯苓（去皮）。
出处：《瘴疟指南》卷下。
主治：哑瘴。神清目开，大小便如常，惟全不能出声，身热。
加减：None
功效：None
用法用量：上加生姜3片，水煎，温调黑神散服。
制备方法：None
临床应用：None
各家论述：哑瘴若神昏直视，不知人事，痰响者属痰；神昏不知人事，不痰响，能饮食，惟不能出声，此邪热涌沸其血，上塞心肺之窍，故不能言也。是方用六君子缓火邪以补脾救元气；门冬解心肺之热；乌梅生津，以收外泄阳气；归、芎行散上窍之血；血得热则行，故用桂、附之热以行之，且能引上焦之阳下入阴分；再调黑神散以驱逐其血，血散则心肺之窍开，而声音出矣。
用药禁忌：None
附注：None
