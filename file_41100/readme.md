序号：41115
名称：金凤丹
组成：五灵脂3两，天麻（酒浸1宿，焙）3两，乌头（炮裂，去皮脐）2两，枫香脂（研）2两，地龙（去土，炒）2两半，乳香1分，没药（研）半两，木鳖子（去壳）2两，海桐皮（锉）1两，黑豆（去皮）1合，草乌头（去尖）1两，干蝎（全者，去土）1两，狼毒（炮）1两，牛膝（酒浸，切，焙）1两，丹砂（研）半两，薄荷叶1两，附子（炮裂，去皮脐）1两，当归（切，焙）1两半，自然铜（煅，醋淬）1两，骨碎补（去皮）1两，虎骨（酥炙）1两，龙脑（研）1钱，麝香（研）1钱。
出处：《圣济总录》卷八。
主治：肝肾久虚，风邪攻注，腰脚不随，诸风疾。
加减：None
功效：None
用法用量：每服1丸，生姜、葱、酒下；走注风，乳香酒下；卒中风，薄荷酒下；卒中暗风，鸡冠血酒下；半身不随，煎松明酒下，不可食葱；妇人产后破血气，煎黑豆酒下，不可食葱；妇人产后虚肿及头面生疮，遍身痒痛，白芷酒下。
制备方法：上为末，以生姜、葱白汁为丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
