序号：17859
名称：龙骨汤
组成：龙骨1两半，桑根白皮1两半，赤石脂1两半，天雄（炮裂，去皮脐）1两半，厚朴（去粗皮，生姜汁炙）1两半，麻黄（去节根）1两半，白芷1两，黄连（去须）1两，地榆1两，桂（去粗皮）1两，当归（切，焙）1两，木香1两，白术1两，诃黎勒皮（煨）1两，黄芩（去黑心）半两，肉豆蔻（去壳）2枚。
出处：《圣济总录》（人卫本）卷七十七。
主治：新久泻痢。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，加生姜1枣大（切），煎至6分，去滓温服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名。原书（文瑞楼本）作“龙骨散”。
