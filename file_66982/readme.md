序号：66998
名称：麻豆煎
组成：大麻1石（末，入窖不郁（火邑）者），赤小豆1石（不得1粒杂）。
出处：《千金翼》卷十九。
主治：大腹水肿。
加减：None
功效：None
用法用量：欲服，取5升麻子，熬之令黄香，惟须缓火，勿令焦，为细末；以水5升，研取汁，令尽，净器密贮之。明旦欲服，今夜以小豆1升，净淘渍之，至晓干，漉去水，以新水煮，未及好熟，即漉出令干，纳麻子汁中煮，令大烂熟为佳，空腹恣意食之，每日3次。
制备方法：上取新精者，仍净拣择，以水掏，晒令干；蒸麻子使熟，晒令干，贮净器中。
临床应用：None
各家论述：None
用药禁忌：陈郁麻子，益增其病，慎勿用之。一切水肿，皆忌饱食，常须少饥。
附注：None
