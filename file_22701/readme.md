序号：22709
名称：劫风膏
组成：威灵仙（去芦）1两半（锉细，焙，研为末）。
出处：《准绳·幼科》卷一。
主治：小儿急慢惊搐，脐风撮口，牙关紧闭，痰涎壅盛，咽喉肿痛。
加减：None
功效：None
用法用量：先用盐梅肉擦牙根，次以此膏1丸或2丸，温白汤浓调，抹入左右牙关内即开，续进别药。熬时得瓦器为上，银器尤佳。及解风痰壅盛，淡姜汤调化，不拘时候少与含咽；咽喉肿痛，温茶清或薄荷汤调下。
制备方法：上用皂荚3两去皮弦，捶损，挪温水1碗绢滤过，慢火熬若稀糊，入醇醋半两，再熬3-5沸，去火候冷，用前药末停分乳钵内杵匀，为丸如芡实大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
