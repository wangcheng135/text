序号：33291
名称：一秤金
组成：半夏10斤，白矾5斤，生姜10斤，粉草10斤，金箔10张。
出处：《寿世保元》卷三。
主治：痰嗽，劳嗽。
加减：None
功效：None
用法用量：每病重者，用药2钱半，轻者2钱，金箔10张，和1大丸，与病人噙化。
制备方法：半夏用米泔水浸10日，换水3次，取出，切作两半，晒干；白矾用水1桶，入铁锅内化开，将半夏入矾水内，浸20日，取出，切作四瓣，晒干；生姜另研取汁，再入半夏，浸20日，取出晒干，为细末，听用；粉草去皮，为粗末，入锅内，添水煮数沸，取出，以布滤出渣，将净水仍入锅内，熬成膏子，和成剂。
临床应用：None
各家论述：None
用药禁忌：忌房事。
附注：None
