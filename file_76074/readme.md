序号：76090
名称：蛜(虫祁)丸
组成：蛜（虫祁）（微炒，去足）1两，天麻1两，附子（炮裂，去皮脐）1两，补骨脂1两，葫芦巴1两，牛膝（去苗）1两，石斛（去根，锉）1两，槟榔1两，巴戟1两，硫黄（细研，水飞过）1两，硇砂（细研）1两，阿魏（研入）1两，桃仁（汤浸，去皮尖双仁，别研如膏）1两。
出处：《圣惠》卷九十八。
主治：肾脏久积风冷，小腹气滞，腰膝痠疼，脐胁冷痛，饮食减少，四肢无力。
加减：None
功效：None
用法用量：每服20丸，以热生姜酒送下。
制备方法：上为末，入研了药令匀，用酒3升，入前药末中拌匀搅，以慢火熬如膏，和余上药末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：《普济方》有木香。
