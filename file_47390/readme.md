序号：47405
名称：祛风舒筋丸
组成：防风50g，桂枝50g，麻黄50g，威灵仙50g，川乌（制）50g，草乌（制）50g，苍术（炒）50g，茯苓50g，木瓜50g，秦艽50g，骨碎补（炒）50g，牛膝50g，甘草50g，海风藤50g，青风藤50g，穿山龙50g，老鹳草50g，茄根50g。
出处：《中国药典》。
主治：风寒湿痹，四肢麻木，腰腿疼痛。
加减：None
功效：祛风散寒，舒筋活络。
用法用量：口服，1次1丸，1日3次。
制备方法：上为细末，混匀。每100g粉末加炼蜜160～180g制成大蜜丸，每丸重7g。
临床应用：None
各家论述：None
用药禁忌：孕妇慎用。
附注：None
