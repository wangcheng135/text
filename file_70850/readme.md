序号：70866
名称：紫草丸
组成：紫草5钱，珍珠4分，朱砂5钱，牛黄2分，犀角3钱，羚羊3钱，青黛3钱，川贝3钱，羌活3钱，琥珀3钱，乳香3钱，没药3钱，玄参5钱，雄黄5钱，冰片2分（上为极细面，再加入下药），金银花2两，地丁2两，胡桃仁2两，菊花1两，生甘草1两。
出处：《全国中药成药处方集》（沈阳方）。
主治：痘疹将出末出，头痛呕吐，身热神昏，咳嗽喘促，食欲不振，疹毒内攻，昏睡似痉，小便赤涩。
加减：None
功效：解毒表疹，兴奋神经。
用法用量：每服1丸，白开水送下。
制备方法：用后5味熬成膏后，和前药面搅均一处，加蜜少许为丸，3分5厘重。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
