序号：8601
名称：半夏汤
组成：半夏5两，麦门冬5两，吴茱萸3两，当归3两，阿胶3两，干姜1两，大枣12个。
出处：《千金》卷二。
主治：妊娠9月，卒得下痢，腹满悬急，胎上冲心，腰背痛不可转侧，短气。
加减：None
功效：None
用法用量：上(口父)咀，以水9升，煮取3升，去滓，加白蜜8合，微火上温，分4服，痢即止。一方用乌雌鸡1只，煮汁以煎药。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：今以孕母卒得下痢，腹满悬急，故用半夏以辟肠垢；姜、萸以散腹满；归、胶以护荣血；冬、枣以行津液，此皆恒用之品，其理易明。独是白蜜奥旨崇古未讲，盖蜜能通肠，而利反用之，必四服痢止肠垢去而正气复，胎自安矣。
用药禁忌：None
附注：None
