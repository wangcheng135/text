序号：68868
名称：黄连清胃丸
组成：炒山栀2两，连翘2两，大黄2两，黄芩2两，生石膏2两，丹皮2两，薄荷2两，知母3两，朴消1两，荆芥1两，生地1两，黄连1两，升麻1两，防风1两，当归尾1两，甘草1两，白芷1两，赤芍1两，玄参1两，花粉1两。
出处：《全国中药成药处方集》（沈阳方）。
主治：口燥舌干，两腮焮肿，牙齿疼痛，齿根溃烂，口流热涎，烦渴饮冷，气息臭秽，头痛目赤，尿涩便结。
加减：None
功效：泻热解毒，清胃通便。
用法用量：每服2钱，以开水送下。
制备方法：上为极细末，朴消煎水泛为小丸。
临床应用：None
各家论述：None
用药禁忌：忌食五辛荤腥；孕妇忌服。
附注：None
