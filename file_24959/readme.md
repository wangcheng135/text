序号：24967
名称：如圣膏
组成：巴豆（取肉）27枚，密陀僧半两（别细研），天南星半两，附子1分，乳香（别研）1分，没药1分，木香1分，当归1分，防风1分，紫藤1分，白及1分，白蔹1分，香白芷1分，黄芩1分，黄耆1分，赤芍药1分，黄蜡1分（另入），盐花半钱（炒），头发1结（净洗，控干）。
出处：《普济方》卷三一四。
主治：不问年深日近，发背恶毒，痈疽漏疮，瘰疬。
加减：None
功效：None
用法用量：用清油1斤，熬令黄色，以绢袋滤去滓，煎令极沸。春、秋入黄丹7两，夏用8两，冬用6两，慢火煎熬成膏，于铁刀上试令软硬得所，摊得成靥为度。每用摊纸上，贴如常法。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
