序号：48185
名称：神应膏
组成：香油1两半，巴豆45粒（白净者），蓖麻45粒，黄蜡1两，沥青4两，乳香1两（为末），没药1两（为末），轻粉3钱，铜青6钱（为末，又名铜绿，滴香油1处研，用柳条调药）。
出处：《医方类聚》卷一九四引《经验秘方》。
主治：发背痈疽。
加减：None
功效：None
用法用量：如用时，冬月热水浸冷，令软捻开，如患处大小，于帛上贴；夏月略于口中，待软用。
制备方法：上用乳香、没药，先研极细末，方用油煎巴豆、蓖麻，候焦黄色，去豆、麻，油再入砂铫，溶蜡、沥青待化，下乳香、没药，用黄草布滤过，再入铫内，方入轻粉、铜青搅匀，倾入水中，捻成锭子。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
