序号：27746
名称：诃子散
组成：诃子1两（半生，半熟），木香半两，黄连3钱，甘草3钱。
出处：《保命集》卷中。
主治：久病滑泄不禁，气虚欲脱。
加减：如止之不已，宜归而送之也，诃子散加厚朴1两，竭其邪气也。
功效：None
用法用量：每服2钱，以白术芍药汤调下。
制备方法：上为细末。
临床应用：None
各家论述：1.《医方集解》：木香、黄连，香连丸也，行气清火，止痢厚肠；甘草、芍药，甘芍汤也，甘缓酸收，和中止痛；加诃子涩以收脱；加白术补以强脾；厚朴除湿散满，平胃调中，故更借以去余邪也。_x000D_
2.《医林纂要》：诃子酸苦涩，补敛肺气，止泻收脱，其用半生半煨者，生以上行肺，煨以下敛大肠；木香辛苦，能行下焦无形之气以达于上而调和气血，降上焦有形之物以行于下而决渎去秽；黄连苦以降火而能厚肠，用茱萸炒即左金丸，引肺气下行以止肝之过于疏泄，以黄连合之木香即香连丸，所以行大肠之郁滞而除其热；甘草以厚脾土而生肺金；用白术芍药汤调下，芍药以补敛肺金以敛大肠之气，白术以补土生金，补气而输之肺。
用药禁忌：外邪未已者，此方非所用。
