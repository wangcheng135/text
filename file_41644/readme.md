序号：41659
名称：金箔丸
组成：金箔49片，丹砂（研）1钱，水银沙子1钱，麝香（研）1钱，腻粉（研）1钱，牛黄（研）1分，青黛（研）1分，犀角末1分，白僵蚕（炒）1分，蝉蜕（去土）1分，麻黄（去根节）1分，白附子1分，干蝎（炒）1分，天麻（酒浸，炙）1分，天南星（炮，酒浸，焙）1分。
出处：《圣济总录》卷一六九。
主治：小儿惊风壮热，时气疮疹，摇头弄舌，咳吐目涩，多睡涎嗽；兼治寒邪发热，及下利脓血。
加减：None
功效：None
用法用量：每服以1丸分3服，煎人参、薄荷汤化下，不拘时候。
制备方法：上为末，再同研匀细，生蜜为丸，如梧桐子大，或如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
