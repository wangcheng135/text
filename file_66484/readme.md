序号：66500
名称：小三棱煎
组成：荆三棱4两，蓬莪术4两（洗净），芫花1两（去枝叶）。
出处：《博济》卷三。
主治：癥瘕积聚，痞气嗝噎，积滞水肿。
加减：None
功效：None
用法用量：每服3丸至5丸，用生姜、盐汤吞下；妇人醋汤送下。
制备方法：上药同入一瓷瓶内，用米醋5升浸满药，封却瓶口，以炭火煨，觉微干，即取出荆三棱、蓬莪术，便杵碎芫花，另以余醋炒微焦后，同2味猛焙干，捣罗为末，用米醋煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：小三棱煎丸（《圣济总录》卷七十二）、三棱煎（《全生指迷方》卷三）、三棱煎丸（《卫生总微》卷十三）、小三棱丸（《普济方》卷三二四引《医方集成》）。
