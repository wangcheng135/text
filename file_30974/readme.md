序号：30983
名称：十香膏
组成：沉香1钱，麝香1钱，木香5钱，丁香5钱，乳香5钱，甘松5钱，白芷5钱，安息香5钱，藿香5钱，零陵香5钱（同为细末），当归2两，川芎2两，黄耆2两，木通2两，芍药2两，细辛2两，升麻2两，白蔹2两，独活2两，川椒2两，藁本2两，菖蒲2两，厚朴2两，木鳖子2两，官桂2两，商陆根2两（锉碎），桃仁5钱，杏仁5钱，柏子仁5钱，松子仁5钱，槐枝2两，桑枝2两，柳枝2两，松枝2两（锉），没药2两，轻粉2两，雄黄2两，朱砂2两，云母石2两，生犀角2两，乱发灰2两，白矾灰2两（另研如粉），真酥猪脂2两，羊肾脂2两，黄丹1斤，清脂麻油3斤。
出处：《外科精义》卷下。
主治：五发，恶疮，结核，瘰疬，疳瘘，疽，痔。
加减：None
功效：None
用法用量：每用量疮口大小，绯帛上摊贴之，肠胃痈疽可作丸，梧桐子大。每服7丸，空心温酒送下。
制备方法：上先于木炭火炼油香熟，下16味锉碎药，并4枝、4仁，熬至紫黑色，出火，滤去滓，入脂酥，煎10余沸，再以新绵滤过，油橙清，拭铛令净，再入火上煎袖沸，下丹，用湿柳枝作箪子，不住搅，熬1日，滴在水中，成珠不散则成。离火，入10味药末，搅匀，再上火，入云母等粉8味，轻煎令沸，出火，不住搅1食时，于瓷盒内密封收。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
