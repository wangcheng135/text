序号：58661
名称：秘传金锁思仙丹
组成：莲花蕊10两（忌硫黄、蒜），石莲子10两（取净粉用），鸡头实10两（取其实并中，捣烂晒干，再捣筛，取净粉）。
出处：《直指附遗》卷十。
主治：男子嗜欲过多，精气不固。
加减：None
功效：None
用法用量：每服30丸，空心以盐汤送下。
制备方法：上以金樱子3斤，取霜后半黄者，木臼中转杵，却刺勿损，擘为2片，去水淘净，烂捣，入大锅，以水煎，不绝火，约水耗半取出，滤过重煎，如稀饧；市肆干者焙之，用水浸转，去子，煎令如法，入前药末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：金锁思仙丹（《医统》卷七十引万氏方）。
