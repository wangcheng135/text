序号：4100
名称：木瓜虎骨丸
组成：木瓜（宣州者）1枚（去皮瓤，焙），麒麟竭（研）1两，没药（研）1两，乳香半两（研。以上3味同研令匀，入在木瓜中，却以圆盖子盖定，用黑豆1斗，水淘过，安木瓜在内，都用豆盖，令蒸烂取出，沙盆内研成膏），虎胫骨1两（涂酒炙），木香1两，自然铜（醋淬7遍）1两，枫1两，香脂1两，败龟（醋炙，去裙襴）1两，骨碎补（去毛）1两，甜瓜子1两，桂（去粗皮）1两，当归（切，焙）1两，地龙（去土）2两，安息香1两（重汤内酒熬，去滓）。
出处：《圣济总录》卷八十一。
主治：风毒脚气，疼痛无力，筋脉拘急，行步艰难。风寒湿合而成痹，脚重不仁，疼痛少力，足下隐痛，不能踏地，腰膝筋挛，不能屈伸及项背拘急，手足无力，耳内蝉鸣，头眩目运，及诸证脚气，行步艰难。
加减：None
功效：None
用法用量：每服30丸，空心木瓜汤送下，1日2次。
制备方法：上药除前4味外，都捣罗为末，并安息香，同入木瓜膏内搜和，更入臼捣100-200下，如药稍干，入少好酒为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
