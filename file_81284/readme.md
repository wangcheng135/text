序号：81300
名称：鳖甲汤
组成：鳖甲（去裙襕，酒浸，炙）1两半，柴胡（去苗）1两，人参1两1分，白术1两1分，知母（焙）1两1分，半夏（汤洗去滑，炒）1两1分，桔梗（炒）1两1分，紫菀（去苗土）1两1分，桑根白皮（炙，锉）1两半，芎3分，芍药（炒）1两1分，当归（切，焙）1两半，秦艽（去苗土）1两半，地骨皮1两半，甘草（炙）3分，桂（去粗皮）3分，虎头骨（酒浸，炙）1两，生干地黄（焙）1两半，赤茯苓（去黑皮）1两半，槟榔（煨）1两半，附子（炮裂，去皮脐）1两半。
出处：《圣济总录》卷九十。
主治：五劳咳嗽，咯唾脓血，言语声嘎，日渐羸劣，不欲饮食。
加减：如有寒热，小便赤涩者，更加葱白1茎，豉40粒。
功效：None
用法用量：每服3钱匕，水1盏，加生姜1枣大（拍碎），煎至7分，去滓，空腹、食前温服，每日3次。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
