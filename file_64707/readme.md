序号：64723
名称：硇砂丸
组成：硇砂（飞）1两，硫黄1分（研），白矾（研）1分，上3味，同研匀，用莱菔1枚，重5两者，割开留原盖子，剜作坑子，填上件末在内，将原盖用竹签定，以面剂15两裹了，开地坑子1枚，方1尺，铺马粪厚3寸，安面球在上，更以马粪盖之，上发火烧，候面黑即止，去面，将莱菔药烂捣如膏，更入后药：肉豆蔻半两（去壳），葫芦巴半两，诃藜勒皮1两，附子1两（炮裂，去皮脐），补骨脂（炒）1两，茴香子1分（炒）。
出处：《圣济总录》卷七十八。
主治：冷热不和，下部痛，里急后重，虚滑或结涩，肠头脱出。
加减：None
功效：None
用法用量：每服15-20丸，空心食前温米饮送下。
制备方法：上为末，通前药膏同捣，滴少好酒捣令匀，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
