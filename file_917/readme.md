序号：917
名称：六神丹
组成：白僵蚕（直者）1钱（浴净，姜汁浸，微炒），辰砂1钱（研），蜈蚣（大者）1条（去头足，刮去腹中物，酒浸，炙香），蝎梢1钱（去尖），半夏（陈者）1钱（刮去脐，沸汤洗7次），真麝香1字，甘草1钱（炙匀），人参1钱（去芦，洗，切，焙），藿香叶1钱（去尘土）。
出处：《普济方》卷三七五引《卫生家宝》。
主治：小儿急慢惊风，涎潮气壅。
加减：None
功效：安神养魄，去风邪，定嗽喘，利膈。
用法用量：周岁儿每服1丸，薄荷汤化下；2-3周及1周者，以意加减。
制备方法：上为细末，炼蜜为丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
