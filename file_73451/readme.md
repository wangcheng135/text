序号：73467
名称：温中散滞汤
组成：陈皮1钱，半夏1钱半，甘草1钱，茯苓1钱，白芍1钱2分，厚朴（姜炒）1钱2分，紫苏1钱，木香5分，苍术1钱2分，砂仁（炒，研碎）6分。
出处：《会约》卷七。
主治：阴阳不和，吐泻腹痛。
加减：如冬天寒甚无汗，加麻黄7分；有汗，加桂枝8分；如腹中有热，或口渴拒按，大便赤热，加黄芩2钱，或加山栀（炒黑）1钱；如夏日伤暑，加扁豆2钱，香薷8分，木瓜，滑石各1钱；如泻甚，小便短赤者，加白术钱半，川萆薢3-4钱，泽泻、木通各1钱，木香（煨用）3分；如气滞之甚者，加白芥子、青皮、槟榔之类。
功效：None
用法用量：水煎，温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
