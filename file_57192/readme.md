序号：57207
名称：消积保中丸
组成：陈皮（去白）2两，青皮（清油炒）4钱，白茯苓（去皮）2两半，白术（土炒）3两，香附（醋炒）2两，半夏1两（泡7次，姜汁炒），木香3钱（不见火），槟榔7钱，莪术（醋浸，炒）8钱，三棱（醋浸，炒）8钱，莱菔子（微炒）1两，砂仁4钱，神曲（炒）1两，麦芽（炒）6钱，白芥子（炒）1两，黄连（姜汁炒）1两，真阿魏（醋浸）3钱，山栀仁（姜汁炒）1两，干漆（炒净烟）3钱。
出处：《回春》卷三。
主治：None
加减：None
功效：顺气化痞，理脾消滞，散痞结，除积块，进饮食，清郁热。
用法用量：每服80丸，食后白汤送下。
制备方法：上为细末，姜汁、酒打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：加人参五钱尤效。
