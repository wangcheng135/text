序号：64539
名称：理气丸
组成：蔻仁2两6钱7分，砂仁1两3钱4分，草果仁1两3钱4分，木香6钱7分，三棱6钱7分，槟榔1两，鸡内金1两3钱4分，盔沉1两3钱4分，甘草6钱7分，枳壳6钱7分，山楂6钱7分，姜夏6钱7分，白术6钱7分，乌药6钱7分，川军2两，二丑2两，神曲1两，公丁香1两，贡朴1两，贡桂1两，莪术6钱7分，青皮1两，陈皮1两。
出处：《全国中药成药处方集》（吉林方）。
主治：男女之气滞肝郁，小儿疳积。
加减：None
功效：舒肝理气，开郁导滞。
用法用量：每服20丸。
制备方法：上为细末，水泛为丸，如黄豆大，朱砂为衣。可用瓷坛贮存以免风干。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
