名称：轻骨丹
组成：独活（去土）1两，牛膝（酒浸）1两，菟丝子（酒浸）1两，苁蓉（酒浸）1两，萆薢（蜜炒）1两，金毛狗脊（去毛）1两，川心巴戟（盐炒）1两，骨碎补（去毛）1两，破故纸（炒）1两，胡芦巴（炒）1两，大附子（炮，去皮脐）1两，熟地黄1两，当归（去土）1两，天麻1两，防风（去芦头）1两，羌活（去土）1两，白芥子（炒）1两，川芎1两，五味子（炒）1两，川乌头（炮，去皮脐）1两，木香1两，木鳖子半两，甜瓜子半两（炒），地龙半两（去土），全蝎1两（炒），乳香半两（别研），没药半两（别研），续断1两。
出处：《御药院方》卷一。
主治：中风手足缓弱，肢节不伸，筋脉挛急，瘫痪偏风，半身不遂，口眼斜，语言謇涩，肌肉不生，一切诸风。
加减：None
功效：壮筋骨，补虚，驻颜色，强骨生力，益真气，除骨髓间风邪。
用法用量：每服30-40丸，食前温酒送下。
制备方法：上为细末，酒煮面糊为丸，如梧桐子大。
临床应用：None
