序号：70914
名称：小柴胡加地黄汤
组成：柴胡1两1分，人参半两，半夏半两（汤洗7次），黄芩半两，甘草半两，生干地黄半两。
出处：《普济方》卷三一八引《圣惠》。
主治：妇人室女伤寒发热，经水适来或适断，昼则明了，夜则谵语，如见鬼状。亦治产后恶露方来，忽尔断绝。
加减：None
功效：None
用法用量：每用5钱，水2盏，加生姜5片，大枣2个，同煎至8分，去滓温服。
制备方法：上为粗末。
临床应用：None
各家论述：《本事方释义》：柴胡气味辛甘平，入足少阳；人参气味甘温，入足阳明；半夏气味辛温，入足阳明；黄芩气味苦寒，入手太阴、少阳；甘草气味甘平，入足太阴，能缓诸药之性；生干地黄气味甘苦微寒，入手足少阴、厥阴；姜枣之辛甘，入荣卫。妇人病伤寒或发寒热，经水适来适断，昼则明了，夜则谵语，如见鬼状，谓之热人血室。外邪已入血分，更恐其深入至阴之处，故用小柴胡汤加生地，以泻其血分，则热缓而神安矣。
用药禁忌：None
附注：地黄汤（《女科百问》卷上）、小柴胡汤（《妇人良方》卷六）、人参汤（《普济方》卷三一八）、小柴胡加生地黄汤（《痘疹心法》卷二十三）。
