序号：6445
名称：加味葛根汤
组成：升麻、葛根、前胡、桔梗、山楂、青皮、木通、荆芥、抚芎、甘草、灯心。
出处：《医宗己任编》卷三。
主治：小儿发热，看有痘情，耳冷、尻冷、脚冷，眼如含水，懒于言笑。
加减：若热甚、气闷、谵语、腰无力，冬月加麻黄1钱；夏月加石膏1钱，甚者加至2钱；如冬月热甚，过4-5日，重症已见，而舌灰白色者，亦须以石膏合麻黄用之；夏月苏叶、薄荷俱可进退用之。
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：用此方服至见点。如服后而不见点，是毒盛也，其症必重。如见点如糠粞齐布，热甚口臭，此脾经痘也，死不治矣。
