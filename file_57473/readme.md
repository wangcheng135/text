序号：57488
名称：润体丸
组成：防风（去芦叉）1两半，白龙脑（别研）1两半，乳香（别研）1两半，羚羊角末（别研如粉）1两半，附子（炮，去皮脐）1两半，白僵蚕（微炒）1两半，槟榔1两半，肉豆蔻仁1两半，沉香1两半，蒺藜子（微炒）1两半，丁香1两半，蔓荆子（去白皮）1两半，牛黄（别研如粉）1两半，藿香叶1两半，麻黄（去根节）1两半，生犀角末（别研）1两半，雄黄（研飞）1两半，麝香（研如粉）1两半，木香1两半，辰砂（研飞）1两半，茯苓（去皮）1两半，白附子（炮）1两半，羌活（去芦）1两半，原蚕蛾（微炒）1两半，人参（去芦）1两半，肉桂（去粗皮）1两半，芎藭1两半，真珠末（别研如粉）3分，独活（去芦）3分，干蝎（微炒）2两，半夏（水煮30沸，薄切，焙干，入生姜汁炒）2两，川乌头（炮裂，去皮脐，捣碎炒黄）2两，白花蛇（酒浸，炙，去皮骨，取肉）3两，天麻（去苗）3两，琥珀（别研如粉）半两，腻粉（研）半两，白豆蔻仁半两，金箔60片（为衣）。
出处：《局方》卷一。
主治：诸风手足不遂，神志昏愦，语言謇涩，口眼僻，筋脉挛急，骨节烦疼，头目眩晕，恍惚不宁，健忘怔忪，痰涎壅滞；及皮肤顽厚，麻痹不仁。
加减：None
功效：None
用法用量：每服1丸，加至2丸，细嚼，温酒送下，荆芥茶下亦得。如破伤中风，脊强手搐，口噤发痫，即以热豆淋酒化破3丸，斡口开灌下，少时再服，汗出乃愈；若小儿惊风诸痫，每服半丸，薄荷汤化下，不拘时候。
制备方法：上为细末，入研药令匀，炼蜜为丸，如鸡头大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
