序号：41621
名称：金锁固精丸
组成：沙苑蒺藜（炒）2两，芡实（蒸）2两，莲须2两，龙骨（酥炙）1两，牡蛎（盐水煮1日1夜，煅粉）1两。
出处：《医方集解》。
主治：火炎上而水趋下，心肾不交之精滑不禁。真元亏损，梦遗滑精，盗汗虚烦，腰痛耳鸣，四肢无力。
加减：None
功效：补肾益精，固涩滑脱，交通心肾。
用法用量：盐汤送下。
制备方法：莲子粉糊为丸。
临床应用：重症肌无力：患者吴某，45岁，患重症肌无力，右眼上睑完全下垂，四肢无力，踡卧不起，咀嚼困难，呼吸喘息气短。诊前曾给新斯的明0.5毫升肌肉注射，上述肌无力症状在10分钟内消失，不久即如故。住院期间，曾用过补中益气汤、归脾汤、杞菊地黄丸等方药加减施治，西药除用新斯的明外，还用过氯化钾、维生素B1，维生素B2等药物，病情时好时坏，一直未能痊愈。根据中医辨证，患者有遗精、腰酸痛、腿冷、舌质红、少苔等肾阴虚表现，故改用金锁固精丸（成药）治疗。每次服12g，每日三次，淡盐水送下。二周后病情明显好转，共服金锁固精丸36瓶，病获痊愈，观察六年未见复发。作者认为，本方具有补肾固精之效，常用以治疗遗精病人。方中药物，沙苑蒺藜补肾益阴，芡实健脾利湿，龙、牡镇心安神，涩精秘气，莲子、莲须清心养胃，交通心肾。诸药合用，可治真气亏损，肾虚遗精，四肢无力诸症。据患者的证情表现，乃属肾阴虚，肾不纳气的表现，用本方是治本之法，当获捷效。
各家论述：1.《医方集解》：此足少阴药也。蒺藜补肾益精，莲子交通心肾，牡蛎清热补水，芡实固肾补脾，合之莲须、龙骨，皆涩精秘气之品，以止滑脱也。_x000D_
2.《成方便读》：夫遗精一证，不过分其有火无火，虚实两端而已。其有梦者，责相火之强，当清心肝之火，病自可已；无梦者，全属肾虚不固，又当专用补涩以固其脱。既属虚滑之证，则无火可清，无瘀可导，故以潼沙苑补摄肾精，益其不足。牡蛎固下潜阳，龙骨安魂平木，二味皆有涩可固脱之能；芡实益脾而止浊，莲肉入肾以交心，复用其须者，有赖其止涩之功，而为治虚滑遗精者设也。_x000D_
3.《方剂学》：方中沙苑蒺藜补肾涩精为君药；莲子、芡实助君药以补肾涩精，为臣药；君臣相配，以补不足为主；莲须、煅龙骨、牡蛎性涩收敛，专以涩精为用，共为佐使药。诸药合用，既可涩精液之外泄，又能补肾精之不足。但本方究以固涩为主，故遗精滑泄已止，便需用补肾之品，补虚固肾以治本。
用药禁忌：本方多为收敛之品，偏于固涩。如属心、肝火旺或下焦湿热所扰以致遗精者，禁用本方。
