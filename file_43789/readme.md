序号：43804
名称：厚朴丸
组成：厚朴（去粗皮，姜汁炙熟）1两半，陈橘皮（汤浸，去白，炒）1两半，白茯苓（去黑皮）1两半，人参1两半，干姜（炮）1两半，白术1两半，薏苡仁1两半，桂（去粗皮）1两1分，牛膝（酒浸，焙）1两1分，枳壳（去瓤，麸炒）3分，细辛（去苗叶）3分，食茱萸3分，大麦糵（炒）3分，石斛（去根）1两，甘草（炙）1两。
出处：《圣济总录》卷八十八。
主治：虚劳。脾胃气弱，呕吐，口干烦渴，不能饮食，四肢疼痛。
加减：None
功效：None
用法用量：每服20丸，加至30丸，1日2次，温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
