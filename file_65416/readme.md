序号：65432
名称：羚羊角散
组成：羚羊角屑1两，前胡1两（去芦头），桂心1两，芎1两，麻黄1两（去根节），秦艽1两（去苗），防风1两（去芦头），附子1两（炮裂，去皮脐），赤箭1两（微炒），天南星1两（炮裂），蝉壳半两，独活1两，茯神1两，槟榔1两，枳壳1两（麸炒微黄，去瓤），桑螵蛸半两（微炒），干蝎半两（微炒），牛黄1钱（研入），朱砂半两（研细），麝香1钱（研细），铅霜1分（研入）。
出处：《圣惠》卷十九。

制备方法：上为细末，入研了药，重研令匀。
临床应用：None
各家论述：None
