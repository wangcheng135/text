序号：30256
名称：豆蔻散
组成：肉豆蔻2个，自然铜2钱（烧酥），槟榔1个，板蓝根3分，蛀桑子2钱，甘草3钱，乌梅4个，川乌头1个（烧），金毛狗脊1个（去毛），红花2钱，红芍药3分，茅香3分，甘柿蒂14个，川楝子4个，香附子21个，苦丁香7个，官桂1分半（去皮），乳香2分。
出处：《普济方》卷三○六。
主治：驴涎马气，骆驼、骡子、兔子、狗子、蛇伤。
加减：None
功效：None
用法用量：每服2分，温酒1盏调下；如生气，食后服；热气，食前服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：如要验中毒，吃大麦面拌羮，吐逆者是，不吐非也。如吃大麦面羮1-2碗，令病人极吐，后下诸药大效。
