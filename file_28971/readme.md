序号：28979
名称：补骨脂丸
组成：补骨脂（炒香）1两，乳香（别研）1两，自然铜（烧红，酒淬7遍）1两，石亭脂（别研）1两，木鳖子（去壳，取肉，别研）1两，续断（酒浸，焙）1两，乌雄鸡1只（去皮毛肠肚肉皆不用，只取血并骨入药），乌龟1枚（刳去肠胃，如无，用龟甲代之）。
出处：《杨氏家藏方》卷十四。
主治：折扑伤损。
加减：None
功效：接骨止痛，生肌肉，续筋骨。
用法用量：每服50丸，加至100丸，温酒、盐汤任下，不拘时候。
制备方法：龟或龟甲和鸡血、鸡骨就砧上烂剁令极细，烈火焙干，为末，称与前药末等分，拌和令匀，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
