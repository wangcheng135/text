序号：7747
名称：圣饼子
组成：石决明（先捣碎，水飞细）1两，川椒（去子）1两，车前子1两，楮实子1两，羌活1两，牛蒡子（新瓦上炒）1两，青葙子1两，木通1两，苍术（米泔浸1宿）1两，木贼（去节）1两，独活1两，白蒺藜（去尖刺）1两，蛇退皮（洗，煿过令黄）1两，地肤子1两，太阴玄精石2两，滑石2两，寒水石2两，云母石2两，磁石2两（盐泡过挤干，以上五件入瓷瓶子内，用泥固济，入土坑子内，以慢火煅之令出火毒，以水飞令细，晒干），草决明2两，荆芥2两，甘草2两，甘菊花2两，旋覆花2两，蝉退1两（水洗净），密蒙花3两5钱。
出处：《普济方》卷七十八引《余居士选奇方》。
主治：内外障眼昏暗，久患风毒气眼。
加减：None
功效：None
用法用量：每服1饼子，腊茶嚼下，日3次，不拘时候。
制备方法：上为细末，入五石在内拌匀，炼蜜为丸，每1两药剂，分作十饼子。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
