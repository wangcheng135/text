序号：57388
名称：消癖丸
组成：木香3分，人参3分，白术3分，白茯苓3分，当归3分，枳壳（煨）3两，陈皮（去白）3两，三棱（炮）3两，莪术（炮）3两，桃仁（炒，去皮）3两，麦芽（生）3两，香附子（炒，去苗）3两，川楝肉3个，豆蔻（煨）3个，缩砂仁1两，腻粉（炒）1钱，全蝎3钱，代赭石3钱（火煅，醋淬），丁香2钱，白豆蔻肉1钱，阿胶3钱（炒），五灵脂1钱，半夏（生）3钱，南星（生）3钱，白姜（炮）3钱，巴豆3钱（去油），大曲饼5钱，厚朴3钱（姜制）。
出处：《普济方》卷三九一。
主治：小儿癖气。
加减：None
功效：None
用法用量：用甘遂、芫花各3钱（醋浸1宿，焙，为末）入前药内。又将硇砂3钱（水澄去泥土）瓦茶瓶上盏内熬成膏子，再入芫、遂、醋，同膏子煮面糊为丸，如粟米大，朱砂为衣。空心肉汁吞下。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：一切药内有甘草者不可服，忌3日。
附注：None
