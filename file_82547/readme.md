序号：82563
名称：囊虫丸
组成：茯苓5000g，水蛭875g，干漆875g，雷丸2500g，丹皮2500g，黄连1250g，大黄1250g，炒僵蚕（或僵蛹）3750g，生桃仁3750g，川乌300g，醋芫花300g，橘红1500g，五灵脂流浸膏6000g。
出处：《古今名方》引吉林省特产研究所制药厂方。
主治：囊虫病，脑囊虫及由脑囊虫引起的癫痫。
加减：None
功效：活血化瘀，软坚消囊，镇惊止痛，杀虫解毒。
用法用量：每服1丸，每日2-3次。
制备方法：制成蜜丸，每丸重5g。
临床应用：None
各家论述：None
用药禁忌：服药期间不要饮酒或吃刺激性食物。孕妇忌用。
附注：None
