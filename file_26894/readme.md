序号：26902
名称：朱犀散
组成：犀角半两（镑屑，研末），生麝香1分，大朱砂1分。
出处：《得效》卷十。
主治：中恶，中忤，鬼气，其证暮夜或登厕，或出效野，或游空冷屋室，或人所不至之地，忽然眼见鬼物，鼻口吸着恶气，蓦倒地，四肢厥冷，两手握拳，鼻口出清血，性命逡巡须臾不救。
加减：None
功效：None
用法用量：每服2钱，新汲井水调灌之。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：朱砂散（《疮疡经验全书》卷六）。此证与尸厥同，但腹不鸣，心胁俱暖。凡此切勿移动。即令亲眷多人围绕，打鼓烧火，或烧麝香、安息香、苏木、樟木之类，且候记醒，方可移归。
