序号：41609
名称：金锁比天膏
组成：紫花地丁1斤，刘寄奴（去泥根）1斤，野麻根1斤，苍耳草（连根叶子）1斤，豨签草1斤，山甲1具（或净甲1斤），蛤蚆皮100张（或干蟾100只更妙）。
出处：《惠直堂方》卷四。
主治：发背痈疽，无名肿毒，疔疮鼠串，马刀瘰疬，紫疥红丝，鸦焰漏睛等疮，两腿血风，内外臁疮，鱼口便毒，杨梅结核，金疮杖疮，蛇蝎虫咬，虎犬人伤，顽疮顽癣，久流脓血，万般烂疮，风寒痰湿，四肢疼痛，乳癖乳岩等。
加减：None
功效：None
用法用量：诸疮不论已破未破，并用葱椒汤洗净贴之；如初发势凶，将膏剪去中心留头出气，不必揭起。1膏可愈1毒。
制备方法：真麻油12斤，内将4斤先煎穿山甲枯焦，余药入8斤油内，加老酒、葱汁各2碗，文武火煎药枯，去滓，复煎至滴水成珠；每药油1斤，加飞丹8两，看嫩老得所，离火，不住手搅，下牙皂、五灵脂（去砂）、大黄各4两（皆为末）；待，温下白胶香（即芸香末）4两成膏，水浸3-4日用。
临床应用：None
各家论述：None
用药禁忌：摊膏时不可见火，须用重汤化开。
附注：None
