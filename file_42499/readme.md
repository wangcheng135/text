序号：42514
名称：青礞石丸
组成：南星2两（切作片，用白矾末5钱，水浸1-2日，晒干），半夏1两（汤泡，切作片，以皂角水浸1日，晒干），黄芩（姜汁炒）1两，茯苓1两，枳实（炒）1两，法制消（用莱菔水煮化，去莱菔绵滤，令结入腊月牛胆内风化）5钱（或只风化消亦可），礞石2两（捶碎，焰消2两，同入小砂罐内，瓦片盖之，铁线缚定，盐泥固济，晒干，火煅红，候冷取出）（一本礞石、南星各1两，无枳实）。
出处：《丹溪心法》卷二。
主治：None
加减：None
功效：清热化痰燥湿。
用法用量：每服30-50丸，白汤送下。一方加苍术半两，滑石1两，看病冷热虚实作汤使。
制备方法：上为末，神曲糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中枳实，《医统》作“枳壳”。
