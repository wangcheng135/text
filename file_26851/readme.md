序号：26859
名称：竹沥汤
组成：竹沥3合，荆沥3合，生姜汁半合，白蜜1合，葛根汁2合。
出处：方出《圣惠》卷三，名见《普济方》卷八十九。
主治：肝中风，心神烦热，言语謇涩，少得眠卧。
加减：None
功效：None
用法用量：每温服1合，宜频服。
制备方法：上药都拌匀。
临床应用：None
各家论述：《医略六书》：中风遇热直入厥阴，木火不得发舒，故烦热语涩，不能眠卧焉。弦为肝脉，数生火炎，可见木都火伏，乃不可以直折奏功，所以取用竹沥、姜汁行经豁痰，葛汁、白蜜疏风润燥，荆沥清气化痰也。俾痰化风消，则遏热顿降，而肝脏肃清，无不神宁语朗，何眠卧之有不安哉。
用药禁忌：None
附注：竹沥饮（《医略六书》卷十八）。
