序号：63873
名称：清湿化痰汤
组成：南星1钱（姜制），半夏1钱（姜制），陈皮1钱，茯苓1钱（去皮），苍术1钱（米泔浸），羌活1钱，片芩1钱（酒炒），白芷1钱，白芥子1钱，甘草3分，木香5分（另研）。
出处：《回春》卷五。
主治：湿痰流注经络；关节不利，而致周身四肢骨节走注疼痛，牵引胸背，亦作寒热，喘咳烦闷，或作肿块，痛难转例，或四肢麻痹不仁，或背心一点如冰冷，脉滑。
加减：骨体痛甚及有肿块作痛者，名曰痰块，加乳香、没药、海石、朴消；头项痛，加川芎、威灵仙；手臂痛，加薄桂，引南星等药至痛处；脚痛，加牛膝、黄柏、防己、龙胆草、木瓜。
功效：None
用法用量：加竹沥、生姜汁同服。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
