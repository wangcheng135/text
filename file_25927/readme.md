序号：25935
名称：护元丹
组成：防风2两，白鲜皮2两，麻黄2两，大枫肉（去油）2两，当归2两，牡丹皮2两，山药2两，菟丝子2两，牛膝2两，川续断2两，黄柏2两，黄耆2两，泽泻1两，白茯苓1两，黄芩1两，广桂枝1两，乳香1两，紫荆皮1两，没药1两，知母1两，白芷1两，荆芥1两。熟地4两，胡麻4合。
出处：《疡医大全》卷二十八。
主治：干疯，白癜风。
加减：None
功效：None
用法用量：先用愈风汤洗浴，切要避风。每服100丸，食后暖酒送下。即睡取汗。隔5日取汗1次，3次即愈。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
