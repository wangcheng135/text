序号：29297
名称：远志丸
组成：远志（去心）2两，山芋2两，柏子仁2两，巴戟天（去心）2两，续断2两，杜仲（去粗皮，炙，锉）2两，菟丝子（酒浸，焙干，别捣）2两半，荆实2两半，山茱萸2两半，五味子2两半，肉苁蓉（酒浸，切，焙）4两，牛膝（酒浸，切，焙）4两。
出处：《圣济总录》卷一八六。
主治：真元衰惫，耳焦面黑，精神不爽。
加减：若体涩，加柏子仁；精冷，加五味子；阳衰，加续断，各1倍。
功效：补血益气，强力益志。
用法用量：每服30丸，空心温酒送下。服之月余，气壮精倍。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
