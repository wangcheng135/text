序号：83660
名称：乌梅丸
组成：乌梅（割取皮）3两（火熬令干），黄连3两。
出处：方出《医心方》卷十一引《范汪方》，名见《圣惠》卷五十九。
主治：赤白滞下，昼夜数十行者。
加减：None
功效：None
用法用量：晨服10丸，不知稍增，可至20-30丸，昼夜可6-7服。若候不愈，可增服70-80丸。
制备方法：上药治下筛，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：黄连苦寒，本经主肠澼腹痛，专取苦燥以坚肠胃；配以乌梅益津开胃，不使木邪横干脾土。近世医师每谓初痢后重未除，不可便用酸收，而《千金》用此治暴痢，不致热毒上攻，全赖酸收之力，详此治例，又未可一概论也。
用药禁忌：服药期间欲食，勿与服药相近。
附注：黄连乌梅丸（《普济方》卷二一一）。
