序号：57024
名称：消毒散
组成：牛蒡子2两（炒，纸衬），甘草半两（锉，炒），荆芥穗1两，防风1两。
出处：《卫生总微》卷八引《千金》。
主治：外犯风寒，痘疹、麻疹出而不畅，咽喉肿痛。大便不通。疮发身痛。麻疹既出，1日而又没者，为风寒所冲，麻毒内攻。痘后将发痈毒。
加减：None
功效：None
用法用量：每服1大钱，水1盏，煎至5分，去滓温服。
制备方法：上为粗末。
临床应用：None
各家论述：《医方考》：牛蒡子疏喉中风壅之痰，荆齐穗清膈间风壅之热，生甘草缓喉中风壅之气：乃防风者，散诸风不去之邪也。
用药禁忌：下利者，不可与服。
附注：消毒饮（《保婴撮要》卷十七）、荆防消毒饮（《痘疹仁端录》卷四）。
