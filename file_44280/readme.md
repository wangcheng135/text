序号：44295
名称：下气海藻橘皮丸
组成：海藻3分，橘皮3分，杏仁2分，茯苓2分，人参1两，吴茱萸1两，白术1两，葶苈1两，桑根白皮2两，枣肉2两，昆布2两，芍药5分，桂心5分，白前3分，苏子5合。
出处：《千金》卷十七。
主治：风虚支满，膀胱虚冷，气上冲肺，息奔，令咽喉气闷往来。
加减：None
功效：None
用法用量：饮服10丸，每日2次。加至15丸，以利小便为度。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：海藻、昆布、葶苈皆破水涤痰伐肾之猛剂，佐以苏子、杏仁、橘皮、白前、桑皮泄肺诸品，不得不以参、术、萸、桂、苓、芍、枣肉以安中气之崩迫也。
用药禁忌：None
附注：None
