序号：62837
名称：清平丸
组成：槟榔1斤，川厚朴（姜汁炒）12两，广皮12两，藿香6两，制香附半斤，炒枳实半斤，酒白芍半斤，半夏曲12两，紫苏6两，草果仁半斤，制苍术12两，青皮半斤，柴胡半斤，炒黄芩半斤，莱菔子4两（炒），煨干葛6两，山楂肉半斤，甘草4两。
出处：《同生集》卷下。
主治：一切天行四时瘟疫，彼此传染，憎寒壮热，精神昏迷，身体倦怠，骨节疼痛，饮食不进，胸腹膨胀。及炎天受暑，痧症霍乱吐泻，春瘟夏疟秋痢，感冒风寒，山岚瘴气。噤口痢疾，饮水入口即吐者。
加减：None
功效：开通胃气。
用法用量：每服2丸，开水化服，重者4-5丸。如噤口痢疾，饮水入口即吐者，用1丸噙化，徐徐咽下。小儿减半。
制备方法：上加陈神曲3斤，武彝茶4两，共为细末，用生姜10斤，揭取自然汁，将红枣打泥泡浓汁，拌水泛为丸，每丸重1钱5分，晒干透，用大瓶盛贮，勿泄气。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
