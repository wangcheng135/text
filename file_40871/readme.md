序号：40886
名称：郁李仁散
组成：郁李仁1两（汤浸，去皮尖，微炒），川大黄1两（锉碎，微炒），柴胡3分（去苗），泽泻3分，赤芍药3分，猪苓3分（去黑皮），桔梗3分（去芦头），桑根白皮3分（锉），杏仁半两（汤浸，去皮尖双仁，麸炒微黄），赤茯苓半两，鳖甲1两（涂醋，炙令微黄，去裙襕），麻黄3分（去根节）。
出处：《圣惠》卷三十。
主治：虚劳通体洪满，腹坚胀喘急，不能饮食。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：郁李仁汤（《圣济总录》卷三十二）。
