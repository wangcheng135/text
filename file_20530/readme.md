序号：20537
名称：何首乌丸
组成：何首乌（雌雄各半，用第1淘米泔浸1伏时，次日流出，于银器内先排枣1重，各擘开，上铺何首乌1重，再上排枣1重，复再铺何首乌1重，令尽，次日入清河水于药上，有水约5指以来，用慢火煮，候枣极烂并何首乌稍软取出，不用枣，只拣何首乌入清水中浸少时，用竹刀刮去黑皮及两面浮沫令净，竹刀切作薄片子，慢火焙干，取净）1斤。
出处：《御药院方》卷六。
主治：None
加减：None
功效：补养五脏六腑，强筋壮骨，黑髭发，坚固牙齿，久服延年益寿，驻颜色。
用法用量：每服60丸，空心温酒或米饮送下，服至半月，加至70-80丸，又服1月，加至100丸，服之100日。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：刮、捣者俱不犯铁。
附注：二多丹（《普济方》卷二二一）。
