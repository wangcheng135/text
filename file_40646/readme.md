序号：40656
名称：败龟膏
组成：败龟1两，桂心半两，木香1分，木鳖子仁半两，防风3分（去芦头），白芷1分，当归1分，槐白皮1两，独活1分，川乌头1分（生，去皮脐），芎1分，藁本1分，黄丹1两，清油10两，松脂1两。
出处：《圣惠》卷六十三。
主治：一切风毒气流注骨节筋脉，结聚疼痛。
加减：None
功效：None
用法用量：每用时，看疼痛处大小，火畔煨，以纸上匀摊贴，日2-3度易之。
制备方法：上将败龟、木香、桂心为末，其余细锉，以油浸1宿，同煎令槐白皮黑色为度，绵滤去滓，澄清，都于铛内，以慢火熬，入黄丹，用柳木篦不住手搅，候黄丹色黑，滴入水内，看硬软得所，成膏，便入败龟等末，更搅令匀，倾于不津器内盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
