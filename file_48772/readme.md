序号：48787
名称：大人参半夏丸
组成：人参半两，茯苓（去皮）半两，天南星半两，薄荷叶半两，半夏1两，干姜1两，白矾（生）1两，寒水石1两，蛤粉1两，藿香叶1分。
出处：《宣明论》卷九。
主治：诸痰呕吐，痰逆、痰厥头痛，风气偏正头疼，风壅头目昏眩，耳鸣鼻塞，咽膈不利，心腹痞满，筋脉拘挛，肢体麻痹疼痛，中风偏枯，咳唾稠粘，肺痿，酒病。
加减：None
功效：化痰坠涎，止嗽定喘，宣通气血，调和脏腑，进饮食。
用法用量：每服20-30丸，生姜汤送下；食后温水下亦得。一法，加黄连半两，黄柏2两，水为丸，取效愈妙。
制备方法：上为末，面糊为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
