序号：38082
名称：易生汤
组成：苍术（净洗，去皮）12两，桔梗6两，枳壳（去瓤，麸炒）3两，麻黄（去根节）3两，附子（炮裂，去皮脐）3两，陈橘皮（汤浸，去白，焙）3两，芍药1两半，白芷1两半，芎1两半，当归（切，焙）1两半，甘草（锉）1两半，桂（去粗皮）1两半，半夏（汤洗7次）1两半，赤茯苓（去黑皮）1两半，厚朴（去粗皮，生姜汁炙）2两，干姜（炮）2两。
出处：《圣济总录》卷一五九。
主治：妇人血气不调，心腹（疒丂）痛，或月闭不通，每有妊娠，多难产。
加减：None
功效：None
用法用量：每服3钱匕，水1盏半，醋1合，入生姜3片，煎至1中盏，去滓，稍热服，不拘时候。
制备方法：上为粗末，除桂、枳壳2味外，其余慢火炒令色转，摊冷，次入桂、枳壳末令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
