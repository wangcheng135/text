序号：72151
名称：疏风散
组成：生地5两，防风1两半，连翘3两，白芷1两半，犀角1两，当归3两，川芎1两，桔梗1两，甘草1两。
出处：《医略六书》卷三十。
主治：产后乳汁不出，脉浮弦数者。
加减：None
功效：None
用法用量：薤白汤煎3钱，去滓，温服。
制备方法：上为散。
临床应用：None
各家论述：产后风伤营气遏热，而阳明之气不化，故乳房肿胀疼痛，乳汁不出焉。防风疏风于表，犀角清热于里，白芷开阳明之气，川芎行冲脉之血，生地滋阴凉血，连翘清热散结，桔梗开提气血，当归荣养经脉，甘草缓中泻火，以和胃气也。为散，薤白汤煎，使风热外解，则阳明之气无不化，而冲脉之血无不上荣，何乳房之疼肿不退，乳汁有不出者乎？
用药禁忌：None
附注：None
