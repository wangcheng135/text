序号：58421
名称：破故纸丸
组成：破故纸（炒）4两，葫芦巴（炒）4两，牛膝（酒浸1宿）4两，舶上茴香（酒浸1宿，炒）4两，肉苁蓉（酒浸1宿）4两，川续断（拣净，生）4两，杜仲（削去粗皮，生姜汁制1日1夜，炒丝断，黄）4两（1方无苁蓉）。
出处：《普济方》卷二四一引《海上方》。
主治：寒湿脚气，疼痛不止。
加减：None
功效：None
用法用量：每服50-70丸，食后温酒盐汤送下。
制备方法：上为细末，用艾4两（去枝梗秤），以大木瓜4个，切作合子，去尽瓤，以艾及药末实之，用麻线扎定，饭上蒸3次，烂研，和药为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
