序号：31991
名称：坎离丸
组成：全当归（用好酒浸洗2日，晒干，锉碎）4两，白芍（温水洗，锉碎，用好酒浸1日，晒干，炒赤）4两，川芎（大者，小者不用，清水洗净，锉碎）4两，厚黄柏（去皮）8两（2两酒浸，2两盐水浸，2两人乳浸，2两蜜浸，俱晒干，炒赤），知母（去毛，四制与黄柏同）8两，熟地黄8两（淮庆者佳，4两用砂仁，4两用白茯苓入绢袋，用好酒2壶煮干，去砂仁、茯苓二味，只用地黄）。
出处：《摄生众妙方》卷二。
主治：虚损。阴虚咳嗽。
加减：None
功效：生精益血，升水降火。
用法用量：每服80-90丸，空心盐汤送下；冬月温酒送下。
制备方法：上药和匀，平铺3-4分厚，夜露日晒三日三夜，以收天地之精，日月之华，为细末；用正冬蜜1斤8两，加水半碗，共炼至滴水成珠；再加水一碗，煎滚，和煎药为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：滋降降火丸（《便览》卷三）。
