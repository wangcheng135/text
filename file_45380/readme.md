序号：45395
名称：枸杞子丸
组成：枸杞子1两，苣胜子1两，菟丝子（酒浸软，别捣）1两，覆盆子1两，当归（洗，焙）1两，熟干地黄（洗，焙）1两，干山药1两，白茯苓（去皮）1两，白芍药1两，白术1两，白疾藜（炒去刺）1两，牛膝（酒浸1宿）1两，香白芷1两，延胡索1两，荜澄茄1两，破故纸2两（炒）。
出处：《杨氏家藏方》卷二十。
主治：None
加减：None
功效：滋补真元，通流血脉，润泽颜色，久服乌髭须，延年耐老。
用法用量：每服30-50丸，空心温酒或盐汤送下。
制备方法：上为细末，用无灰酒煮面糊为丸，如梧桐子大，候干，以苍耳叶罨1宿。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
