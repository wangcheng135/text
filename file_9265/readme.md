序号：9265
名称：牛黄八宝丹
组成：牛黄2分，珍珠4分，劈砂5钱（水飞），川黄连3钱（土炒），犀角3钱，羚羊角3钱，雄黄（透明者）5钱，青黛（水澄）3钱，川贝母（炒，净）3钱，冰片2分，琥珀2钱，羌活3钱（炒），玄参5钱（瓦上焙，或晒燥），乳香3钱，没药3钱（共出汗尽）。
出处：《救偏琐言·备用良方》。
主治：不拘已痘末痘，婴儿诸般恶疮恶毒。痧症发斑发狂，浑身赤紫，痧后恶疮毒疡。
加减：None
功效：None
用法用量：1岁左右者，日服1丸；3岁左右者，日服2丸，蜜调服。
制备方法：上为细末；外将拣净金银花2两、甘菊1两、甘草5钱、胡桃肉2两(击碎)，紫花地丁(连根带叶，理净勿杂青苔，锉断)2两，长流水5碗，砂锅内慢火煎至及半取汁，将渣绞干，以绵滤清，桑柴火熬膏，入炼熟老蜜盏许，再熬至粘筋，将前末为丸，每丸3分。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
