序号：64614
名称：理嗽汤
组成：霜桑叶1钱5分，百合3钱，桔梗1钱5分，前胡1钱5分，象贝母1钱，橘红8分，薄荷1钱5分，栀子（炒）3钱。
出处：《医方简义》卷四。
主治：咳嗽，不拘新久虚实。
加减：如身热气粗者，欲发风疹，加牛蒡子3钱（炒）；如久咳不已，则三焦受之，去薄荷，加麦冬、生地各3钱；如咳嗽伤络，痰中带血，加驴胶1钱，枣仁（炒）1钱，柏子仁1钱，淡黄芩1钱5分；如君火内炽，加麦冬3钱，川连8分，琥珀8分；如肝火刑金，加柴胡1钱，丹皮2钱；如土不生金，去桑叶、薄荷，加白术、茯苓各2钱；如肾咳者，加二地、二冬；大肠嗽者，每嗽必欲大便，加诃子（煨）1钱，炙粟壳1钱；小肠嗽者，每嗽必欲小溲，加东洋参1钱，麦冬3钱，五味子10粒；如膀胱胀痛而嗽者，气不化也，去象贝、薄荷，加桂枝2分，滑石3钱；如胆火上冲致嗽者，加夏枯草3钱；如咳而胃痛者，加左牡蛎4钱，川连7分。
功效：None
用法用量：加青果1枚，竹叶20片，水煎服。
制备方法：None
临床应用：None
