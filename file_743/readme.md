序号：743
名称：六生散
组成：生地黄根2斤，生姜1斤，生菖蒲根1斤，生枸杞根1斤，生乌头1斤，生章陆根1斤。
出处：《医心方》卷十三引《范汪方》。
主治：阴虚血瘀，风湿外侵，身体羸瘦，周身关节痠痛；五劳七伤，五缓六急；寒热，胀满大腹，中风垂曳；急风痹，身躯拘痛；腰膝痹。
加减：None
功效：消癥逐血，补诸不足，令人肥白。
用法用量：上6药合7斤，煮洗之，停令燥，粗切之。美酒2斗，都合渍3-4日，出晒之，暮则还着酒中，趣令汁尽，止，为末。每服半钱匕，酒下，1日3次。10日之后增至1钱。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌猪羊肉、冷水、芜荑、饧。
附注：菖蒲散（《圣济总录》卷二十）。
