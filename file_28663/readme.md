序号：28671
名称：补心宁志丸
组成：天竺黄（另研如面）5钱，沉香（另研如面）3钱，天门冬（去心，酒洗，蒸）2两，白芍药（酒炒）3两，白茯神（去心）4两，远志肉（甘草汁浸，蒸）2两，麦门冬（去心）2两，炙甘草6钱，旋覆花1两5钱，真苏子（研）1两，香附（醋浸，晒干，童便拌，瓦上炒）3两，半夏（姜汁拌，以明矾末少许同浸）2两，皂角荚（不蛀者，去黑皮，酥炒，去子取末）2两。
出处：《广笔记》。
主治：痫症。
加减：None
功效：None
用法用量：每服3钱，用竹沥点汤下。
制备方法：上为末和匀，怀山药粉糊为丸，如豌豆大，朱砂1两（研如法）为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
