序号：54789
名称：桂心丸
组成：桂（去粗皮）1两，诃黎勒（煨，去核）1两，厚朴（去粗皮，生姜汁炙）1两，上锉细，掘地作坑子，用炭火烧赤，吹去灰，以酽醋1茶盏，并药投于坑子中，急以瓷碗热盖，土培，匀令漏气，待冷即取出，焙干，入后药。干姜（炮）1两，草豆蔻（去皮）1两，陈橘皮（汤浸，去白，焙）1两，木香1两，阿魏（研）1分，槟榔（煨，锉）半两，郁李仁（汤浸，去皮尖，炒）半两，朴消2两。
出处：《圣济总录》卷五十七。
主治：胁助痛，腹冷，或食冷物不消，或块聚不转，心肋胀闷刺痛，霍乱。
加减：None
功效：None
用法用量：每服20丸，以温酒送下。初服急行200-300步，又服20丸，掣两脚各20-30下，贴膝便坐，气即散也。
制备方法：上为末，炼蜜为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
