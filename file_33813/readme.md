序号：33822
名称：沉香鳖甲散
组成：木香1两，沉香3分，鳖甲（九肋者1枚，净去裙襕，醋炙令黄香）1两半，常山1两，当归（去土并苗）1两，柴胡（去苗）1两，人参（去苗）1两，白茯苓（去黑皮）1两，官桂（去粗皮）1两，青橘（去瓤）1两，陈橘（去瓤）1两，生地黄1两，半夏1两（以汤洗7遍去滑止），槟榔3分，甘草3分（炙）。
出处：《博济》卷四。
主治：室女荣卫不调，经候凝滞，或时头目昏闷，上膈积涎，肢体不利，五心虚烦，饮食进退，多困少力。
加减：None
功效：None
用法用量：每服2钱，水1盏，加生姜3片，同煎至7分，去滓温服，空心、日午、临卧各1次。
制备方法：上各制好，焙干为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：沉香鳖甲汤（《普济方》卷三三二）。
