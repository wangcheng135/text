序号：73902
名称：硼砂丸
组成：硼砂1两（细研），鳖甲1两（涂醋，炙令黄，去裙襕），川大黄1两（锉碎，微炒），木香2分，肉桂2分（去皱皮），附子2分（炮裂，去皮脐），巴豆半两（去皮心，研，纸裹压去油），京三棱2两（微炒，锉），槟榔2分，干姜3分（炮裂，锉），皂荚5挺（不至者，捶碎，以醋浸两宿，挼绞取汁，熬成膏）。
出处：《圣惠》卷四十九。
主治：癥瘕，腹内疼痛。
加减：None
功效：None
用法用量：每服3丸，空心生姜汤送下。
制备方法：上为末，入研药令匀，以皂荚膏为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“硇砂丸”。方中硼砂，《普济方》引作“硇砂”。
