序号：58905
名称：秦皮汤
组成：秦皮2两，前胡2两，常山2两，黄芩2两，升麻2两，芍药2两，白薇2两，枳实2两（炙），大黄3两，甘草2两（炙）（1方加蕤仁1两，栀子仁半两）。
出处：《外台》卷二引《小品方》。
主治：毒病冲眼，忽生赤翳，或白，或肿肤起；或赤痛不得视光，痛入心肝；或眼外浮肿如吹汁出，生膜覆珠子。
加减：若盛热者，可加芒消2两。
功效：None
用法用量：上以水8升，煮取3升，分3服，相去二食顷，更服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌海藻、菘菜、生葱、生菜。
附注：秦皮大黄汤（《伤寒总病论》卷三）。方中前胡、白薇，《伤寒总病论》作“柴胡、白蔹”。
