序号：54347
名称：柴胡连翘汤
组成：中桂3分，当归梢1钱5分，鼠粘子2钱，炙甘草3钱，酒黄柏3钱，生地黄3钱，柴胡5钱，黄芩（炒）5钱，酒知母5钱，连翘5钱，瞿麦穗6钱。
出处：《兰室秘藏》卷下。
主治：男子、妇人马刀疮。男妇热毒、瘰疬，并气寒血滞经闭。
加减：None
功效：None
用法用量：每服5钱，水2大盏，煎至1盏，去滓，稍热，食后服之。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：《古方选注》：以柴胡散少阳之结气，连翘散外疡之血结气聚，瞿麦穗决上焦之壅肿，鼠粘子消上焦之热肿，生地、当归和手足少阳之血脉，黄芩、知柏解三焦之郁热，炙甘草调和寒热之剂。微加中桂者，马刀坚硬，用以消皮肤浮冻之气也。
用药禁忌：None
附注：None
