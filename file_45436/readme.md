序号：45451
名称：久道汤
组成：人参1钱，白术2钱，黄耆2钱，茯苓2钱，甘草5分，白芥子1钱，神曲5分，肉桂1分，麦冬2钱，北五味3分，苏子5分。
出处：《石室秘录》卷一。
主治：虚寒。
加减：心不宁，加生枣仁1钱；不寐，加熟枣仁1钱，远志1钱；饱闷，加白蔻1钱；口渴，加熟地3钱，当归2钱；梦遗，加芡实3钱，山药3钱；饮食不开，加麦芽1钱，山楂3-4粒；有痰，加半夏5分；咳嗽，加桔梗1钱；有浮游之火，加玄参2钱；头疼，加蔓荆子7分，或川芎1钱；有外感，加防风2钱；鼻塞，加苏叶1钱；目痛，加谷精草2钱；心微痛，加栀子5分；胁痛，加芍药1钱；腹痛，加肉桂3分。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
