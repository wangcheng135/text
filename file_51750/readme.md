序号：51765
名称：钟乳丸
组成：钟乳粉1两，金屑半两，银屑半两，真珠半两，珊瑚半两，水精半两（以上并研细，水飞过），琥珀半两（研细），密陀僧半两（研细），白檀香半两，千岁枣半两，乳香1两，零陵香1两，人参1两（去芦头），木香1分，诃黎勒皮1分，白附子1两，桃仁半两（汤浸去皮尖双仁，麸炒微黄），胡粉半两，黄鹰粪半两，丁香半两，光明砂半两（研细），牛黄半两（研细），辛夷1两，杏仁半两（汤浸去皮尖双仁，麸炒微黄）。
出处：《圣惠》卷四十。
主治：面上百病。
加减：None
功效：悦颜色，除(黑干) (黑曾)。
用法用量：每夜取1丸，以温水化破服。经1月，面如童子。
制备方法：上为末，入研药，以冻冷为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
