序号：4883
名称：木香散
组成：生姜1斤（细切，银石器内炒干，令黄），木香1两（炒），沉香1两（微炒），蓬莪术1两半（煨，捶碎），白术2两（炒），陈橘皮1两半（去瓤秤，炒），甘草2两，肉桂1两（不得近火），舶上茴香1两（炒）。
出处：《医方类聚》卷二○四引《修真秘诀》。
主治：妇人产后败血攻心，血气；小儿疳气腹痛，肚胀脚肿；室女经络不行；霍乱吐泻；老人元气发动。
加减：None
功效：None
用法用量：每服1钱，煨葱、酒及盐汤、饭饮、白汤调下并得。妇人产后败血攻心，炒生姜、小便调下；血气，橘皮汤下；小儿疳气腹痛，肚胀脚肿，饭饮调下少许；室女经络不行，炒姜并地黄酒下；霍乱吐泻，木瓜汤下；老人元气发动，煨猪肾酒下。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
