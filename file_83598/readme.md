序号：83616
名称：乌骨鸡丸
组成：乌骨白丝毛鸡1只（男雌女雄，制法同巽顺丸），北五味1两（碎），熟地黄4两（如血热加生地黄2两），上2味，入鸡腹内，用陈酒酒酿、童便于砂锅中煮，如巽顺丸。_x000D_
绵黄耆（去皮，蜜、酒拌炙）3两，于术（饭上蒸9次）3两，白茯苓（去皮）2两，当归身（酒洗）2两，白芍药（酒炒）2两，上5味，预为粗末，同鸡肉捣烂焙干，骨用酥炙，共为细末，入下项药：人参3两（虚甚加至6两），牡丹皮2两（酒洗净，勿炒），川芎1两（童便浸，切，晒），上3味，各为细末，和前药中。
出处：《张氏医通》卷十三引《制药秘旨》。
主治：妇人郁结不舒，蒸热咳嗽，月事不调，或久闭不行，或倒经血溢于上，或产后褥劳，或崩淋不止，及带下赤白、白淫；男子？？丧太早，劳嗽吐红，成虚损者。
加减：骨蒸寒热，加九肋鳖甲3两，银柴胡、地骨皮各1两5钱；经闭，加肉桂1两；崩漏下血，倍熟地，加真阿胶2两；倒经血溢，加麦门冬2两；郁结痞闷，加童便制香附2两，沉香半两；赤白带下，加真川萆薢2两，四制香附2两，蕲艾1两；白淫，倍用参、耆、苓、术。
功效：None
用法用量：侵晨人参汤或沸汤送下3钱，卧时醇酒送下2钱。大便实者，炼白蜜为丸亦可。
制备方法：另用干山药末6两打糊，将前药众手为丸，晒干勿令馊，瓷罐收贮。
临床应用：None
各家论述：None
用药禁忌：None
