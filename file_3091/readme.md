序号：3091
名称：天麻饮子
组成：天麻（酒浸）1两，防风（去芦）1两，当归（洗净，去芦，酒浸）1两，川芎1两，羌活1两，威灵仙（酒浸）1两，五加皮1两，白芍药（微炒）1两，肉桂（去粗皮，不见火）半两，木香（不见火）半两，酸枣仁（微炒，去皮）半两，犀角屑半两，海桐皮（酒浸）3分，人参（去芦）3分，白术（去苗，洗）3分，干葛3分，细辛（去苗，洗）3分，甘草（炙）3分。
出处：《普济方》卷九十七引《卫生家宝》。
主治：气血不足，正气与风邪相搏，浑身臂膊疼痛，潮热往来，倦怠，心间烦躁，恍惚不宁。
加减：None
功效：调血退热，解劳倦，进饮食，轻健四肢。
用法用量：每服3大钱，水1盏，加生姜2片，煎至7分，去滓热服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
