序号：21625
名称：羊骨煎丸
组成：羊脊骨1条（去肉，截成段，用硇砂2两，醋2升同煎，旋煎旋蘸，骨炙令焦黄，以醋尽为度，细锉焙干），沉香（锉）半两，木香半两，槟榔（锉）半两，桂（去粗皮）半两，人参半两，牛膝（酒浸，切焙）半两，白茯苓（去黑皮）半两，山芋半两，郁李仁（汤浸，去皮）半两，附子（炮裂，去皮脐）半两，白术半两，丁香半两，肉苁蓉（酒浸，去皱皮，切倍）半两，石斛半两，阿魏1分（醋化.入面和作饼，炙干）。
出处：《圣济总录》卷一八六。
主治：肾脏虚冷，不思饮食，倦怠。
加减：None
功效：None
用法用量：每服20-30丸，空心盐酒或盐汤送下。
制备方法：上为末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
