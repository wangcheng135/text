序号：8946
名称：加味小胃丹
组成：南星2两5钱，半夏2两5钱（用白矾、皂角刺及姜汁水煮15次），桃仁2两，杏仁（用白矾、皂角水泡）2两，红花2两，陈皮2两，枳实（用白矾水泡半月，炒）2两，白术2两，白芥子2两，苍术2两（用米泔，白矾、皂角水浸1宿，炒）。
出处：《疡科选粹》卷三。
主治：痰核生在下体，兼湿热者。
加减：None
功效：None
用法用量：上为末，姜汁、竹沥煮神曲为丸服，中风痰。痞积、眩晕、喉痹，淡姜汤送下；瘫痪不语，浓姜汤送下；惟痞块、头风、头痛宜临卧、食后服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：属火燥证者，断不可用。
附注：None
