序号：73899
名称：硼砂丸
组成：硼砂1两（细研），京三棱半两（煨，锉），芫花半两（醋拌，炒令黄），猪牙皂荚半两（去黑皮，涂酥，炙令黄），巴豆半两（去皮心，研，纸裹压去油），干漆半两（捣碎，炒令烟出），干姜半两（炮裂，锉），大戟半两（锉，炒令黄色），川乌头半两（炮裂，去皮脐）。
出处：《圣惠》卷四十九。
主治：食不消，结成癥癖，心腹胀痛。
加减：None
功效：None
用法用量：每服3丸，空心及临卧时以生姜橘皮汤送下。
制备方法：上为末，入研药令匀，于银锅子内，以头醋1升半，慢火熬，候可丸，即入油单内裹，旋丸如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“芫花丸”、“硇砂丸”。方中硼砂，《普济方》引作“硇砂”。
