序号：48987
名称：胃丹
组成：朱砂（大块不夹石老）50两，新罗人参4两，肉豆蔻（面裹煨）4两，缩砂仁4两，荜澄茄4两，白豆蔻仁4两，红豆4两，高良姜（锉，炒）4两，附子（炮，去皮脐）4两，白术4两，厚朴（姜汁炒）4两，丁香（不见火）4两，藿香叶4两，五味子4两，干姜（炮，去皮）4两，胡椒4两，益智仁4两，麦门冬（去心）4两，草果仁4两，橘红4两。
出处：《济生》卷一。
主治：真阳衰虚，心火怯弱，不养脾土，冲和失布，中州虚寒，饮食不进，胸膈痞塞，或不食而胀满，或已食而不消，痰逆恶心，翻胃吐食，脏气虚寒，米谷不化，心腹绞痛，泄利不止。一切脾胃诸疾。
加减：None
功效：None
用法用量：每服10粒，加至15粒，空心食前用人参汤送下，大枣汤亦得；如或呕吐，用淡生姜汤送下。
制备方法：上药各如法修制，锉如豆大，用白沙蜜5斤，将药一半同蜜拌匀，入铜锅内；以夹生绢袋盛贮朱砂，悬宕锅内，以桑柴火重汤煮4日4夜，换蜜5斤，又入前药一半，和匀，再煮3日3夜，取砂淘净焙干，入乳钵，用玉槌研细，米粽为丸，如绿豆大，阴干。
临床应用：None
各家论述：None
用药禁忌：忌食猪、羊血。
附注：None
