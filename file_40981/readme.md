序号：40996
名称：三清百解绿蜡丸
组成：归尾（红花汁浸）2两，生地（酒洗）1两2钱，白术（土炒）8钱，人参8钱，威灵仙（酒浸）5钱，牛蒡子（炒、研）8钱，犀角（镑）6钱，天花粉6钱，滑石（煅）6钱，槟榔（升麻汁浸）4钱，牡丹皮7钱，人中黄5钱，牛黄1钱，大黄（酒浸，蒸晒3次）1两2钱（祛毒复元全在此味）。
出处：《救产全书》。
主治：痘疮已靥，毒气内收、不能散降，伏于脏腑之间，乘虚窃发，变症百端。
加减：None
功效：None
用法用量：3岁儿1丸，1岁儿半丸。余剩者，仍以蜡封住，候后再用，愈久愈效。
制备方法：上为极细末，炼蜜为丸，如皂角子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：百解绿蜡丸（《惠直堂方》卷四）。
