序号：50979
名称：大决明散
组成：石决明1钱半（火煅如粉），荆芥穗8分，青葙子8分（酒炒，研），木贼8分，羌活8分，麦冬1钱半（去心），栀仁（炒）8分，白芍6分（酒炒），大黄3分（酒微炒）。
出处：《眼科秘诀》卷一。
主治：眼科七十二症。
加减：None
功效：None
用法用量：远重者，加雌雄石末1分，水煎8分，食后服。将诸药煎，冲倒在碗内，澄清去滓，外加雌雄石末搅匀服之。雄石体重沉碗底，将药吃完，以舌尖舐雄石末于口内，白水送下；点眼。头一七服7剂；二七不点，服7剂；三七点药，服7剂；症极重者服百剂；半重50-60剂，轻者30-40剂。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：①雌雄石即磁石吸针者，用醋煅七次，水飞过用之。②本方用法中所谓“点眼”，是指用原书中赛宝丹点眼中。
