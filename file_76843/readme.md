序号：76859
名称：雄麝汤
组成：雄黄（另研）1钱，朱砂（另研）1钱，真绿豆粉2钱，麝香（另研）1钱，乳香（另研）1钱，白芷2钱，茜草根2钱，紫花地丁草2钱，牡蛎1钱，僵蚕1钱，牛蒡子（炒）1钱，大黄1钱，金银花1钱，青木香1钱，栀子1钱，荆芥穗1钱，朴消1钱，甘草1钱，胡桃2个（去壳膜）。
出处：《医学正传》卷六。
主治：疔肿。
加减：欲利，倍加大黄、朴消2味，后下。
功效：解疔毒。
用法用量：上药白芷以后14味细切，用无灰酒1碗浸少时，擂细，又加水1碗，同煎至1碗，去滓及浊脚，入前雄黄等5味调匀，作1服，更审患处经络分野，依东垣引经泻火药加之尤妙。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
