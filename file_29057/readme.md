序号：29065
名称：阿胶黄连汤
组成：陈阿胶1钱半（烊冲），生白芍2钱，小川连6分（蜜炙），鲜生地6钱，青子芩1钱，鸡子黄1枚（先煎代水）。
出处：《重订通俗伤寒论》。
主治：None
加减：None
功效：滋阴清火。
用法用量：None
制备方法：None
临床应用：None
各家论述：手少阴心主血，中含热气，故《内经》云：少阴之上，热气治之。凡外邪挟火而动者，总属血热。其症心烦不寐，肌肤枯燥，神气衰弱，咽干尿短。故君以阿胶、生地，滋肾水而凉心血（阿胶必须真陈，庶不碍胃，生地用鲜，庶不碍阴）。但少阴只有热气，能温血而不致灼血，若挟肝胆之相火，激动心热，轻则咽干心烦，欲寐而不能寐，重则上攻咽喉而为咽痛，下奔小肠而使脓血，故臣以白芍配芩、连，酸苦泻肝以泻火，而心热乃平。白芍合生地，酸甘化阴以滋血，而心阴可复。妙在佐鸡子黄，色赤入心，正中有孔，能通心气以滋阴。此为润泽血枯、分解血热之良方。
