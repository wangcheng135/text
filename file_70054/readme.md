序号：70070
名称：滋阴百补药酒
组成：熟地3两，生地3两，制首乌3两，枸杞子3两，牛膝2两，沙苑蒺藜3两，鹿角胶3两，当归2两5钱，胡桃仁2两5钱，桂圆肉2两5钱，内苁蓉2两，白芍药2两，人参2两，白术2两，萎蕤2两，龟版胶2两，白菊花2两，五茄皮2两，黄耆1两5钱，琐阳1两5钱，牡丹皮1两5钱，杜仲1两5钱，地骨皮1两5钱，知母1两5钱，黄柏1两，肉桂1两。
出处：《活人方》卷六。
主治：None
加减：None
功效：大补气血，调和营卫，温经舒络，壮骨益髓。
用法用量：过黄梅开用，早、晚随量热饮。
制备方法：上锉碎，囊贮，以滚酒冲入大坛，泥固，外加厚纸蜜封，放窨地。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
