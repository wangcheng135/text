序号：78051
名称：酸石榴煎
组成：酸石榴7枚（并皮细切，研，后更入水1中盏再研，绞取汁，去滓），鹅梨10颗（捣，绞取汁），荆芥5两（细锉，入水1中盏，研，绞取汁），薄荷5两（细锉，入水1中盏，研，绞取汁），牛蒡根半斤（净洗，切，研，绞取汁），竹沥1中盏，生姜（地黄）汁1中盏，白蜜3两，以上诸药汁相和，于银石锅中慢火熬如饧，入后药末：赤箭2两，独活1两，羚羊角屑1两，防风1两（去芦头），桑寄生1两，阿胶1两（捣碎，炒令黄燥）。
出处：《圣惠》卷七十四。
主治：妊娠中风，口眼不正，言语謇涩，手足不遂。
加减：None
功效：None
用法用量：每服1大匙头，以温酒调下，不拘时候。
制备方法：后6味为细末，研令细，入前煎中搅令匀，瓷器中盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
