序号：83826
名称：乌蛇散
组成：乌蛇（酒浸，去皮骨，炙）3两，槟榔5枚，肉豆蔻5枚（去皮），桂（去粗皮）半两，人参半两，白茯苓（去黑皮）半两，当归（切，焙）半两，牛膝（酒浸，切，焙）半两，甘草（炙，锉）半两，麻黄（去根节）半两，白附子（炮）半两，天麻半两，芎半两，羌活（去芦头）半两，藁本（去苗土）半两，附子（炮裂，去皮脐）半两，细辛（去苗叶）1分，干蝎（去土，炒）1两，白芷半两，防风（去叉）半两，白鲜皮1分，木香半两，丹砂（研）1分，麝香（研）2钱。
出处：《圣济总录》卷十二。
主治：一切风气。
加减：None
功效：None
用法用量：每服1钱匕。治虚风气上攻，兼进饮食，葱白腊茶清调下；常服只用腊茶调下；大段风涎，手足不随，（疒帬）麻，用温薄荷酒调下。
制备方法：上药除研者外，捣罗为末，再同研令细匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
