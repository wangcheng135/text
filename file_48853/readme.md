序号：48868
名称：秋石还元丹
组成：男子小便10石（更多不妨）。
出处：《普济方》卷二二二引《十便良方》。
主治：诸般冷疾，及多年冷劳虚惫甚者。
加减：None
功效：能大补暖，悦色进食，益下元；久服去百痰，强骨髓，补精血，开心智。
用法用量：先搘大锅灶1副，于空屋内，锅上用深瓦甑接锅口令高，用纸筋杵石灰泥，涂甑缝并锅，勿令通气，候干下小便，只可于锅中及7分以来，灶下以稻火煮，令专人看之，若涌出，即添冷小便些少，勿令涌出；候干研细，入好合内，如法固济，入灰炉中煅之；旋取2-3两，再研如粉，煮枣瓤为丸，如绿豆大。每服5-7丸，渐至15丸，空心盐汤送下。其药末常近火收，或时复养火3-5日，功效大也。
