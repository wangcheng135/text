序号：2534
名称：双补丸
组成：鹿角霜3两，熟地黄（洗，再蒸）1两，沉香1两，菟丝子（酒浸，蒸，研，焙）1两，覆盆子（去枝蒂）1两，白茯苓（去皮）1两，人参（去芦）1两，宣木瓜1两，薏苡仁（炒）1两，黄耆（炙）1两，苁蓉（洗，酒浸）1两，五味子（去枝，炒）1两，石斛（去根，炒）1两，当归（去芦，油浸）1两，泽泻（切块再蒸）1两，麝香1钱（别研），朱砂半两（别研，为衣）。
出处：《简易方》引《究原方》(见《医方类聚》卷一五○)。
主治：一切虚损，五劳七伤，面色黧黑，唇口干燥，发渴，目暗耳鸣，心忪气短，食少神倦，夜梦惊恐，四肢痠疼，寒热盗汗，小腹拘急，小便滑数、妇人诸疾。
加减：None
功效：常服既济水火，益气安神。
用法用量：每服50-70丸，空心盐汤送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
