序号：41516
名称：金液汤
组成：软前胡1钱，白桔梗8分，直防风1钱，川独活3分，京芍药1钱，肥知母5分，荆芥穗5分，苏薄荷6分，蔓荆子7分（炒），北柴胡1钱（炒），片黄芩5分（炒）。
出处：《一草亭》。
主治：外障。
加减：如受风寒重者，初2剂加羌活5分，小川芎2分，白芷梢2分，后服仍去；如泪多者，加北细辛2分，家园菊5分；如肿胀者，加葶苈子3分；如痛甚者，加厚黄柏3分；如红甚者，加连翘3分，桑白皮4分，牡丹皮6分、红花3分；如翳膜者，加木贼4分，白蒺藜8分；如翳障胬肉者，加石决明1钱（煅）；如昏蒙者，加密蒙花8分，家白菊5分；如大眦红者，加栀仁7分（炒黑）；如小眦红者，加酸枣仁1钱（炒），远志肉（甘草煎水浸软，去骨，炒）1钱，麦冬1钱（去心），家白菊3分，生地黄1钱，当归尾3分，熟地黄1钱；如内热甚者，大便闭结，兼以体旺年少之人，加大黄1-2钱，通后除去。
功效：None
用法用量：上咀片，水煎，热服。如初发赤眼，服药6-7剂可愈，且无后患。如屡发者，风邪积热，入在经络、遇寒即发，服金液汤10余剂后，或作散，或作丸服，调理34日，外用玉华丹日点1次，即愈。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：如体虚者，须用加减地黄丸，空心服。饭后用金液散，此法最妙，如服金液汤，须饭后热服。每日只服1剂，不可骤进，恐伤胃气；服至42日，自愈。如外障等症，多是有余，不可妄投补剂，恐助邪，为害不浅。
附注：None
