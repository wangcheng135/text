序号：27538
名称：补肝散
组成：生地、熟地、当归、白芍药、石斛、丹皮、柴胡、甘草。
出处：《证治汇补》卷二。
主治：痨瘵。
加减：None
功效：养阴血，退虚热。
用法用量：None
制备方法：None
临床应用：None
各家论述：《医略六书》：方中生地滋阴壮水，专退疲劳之热；熟地补阴滋血，专资耗亡之阴；当归养血荣肝，白芍敛阴和肝，石斛益阴平热，丹皮凉血退蒸，生草泻虚火以缓中，柴胡引诸药以入肝也。益肝无补法，养阴滋血即所以补肝，水煎温服，使阴血内充，则肝得所养而疲劳自已。
用药禁忌：None
附注：《医略六书》本方用生地十两，柴胡五钱，熟地十两，归身六两，白芍（炒）四两，石斛三两，丹皮一两半，甘草一两。上为散，水煎，去滓温服。
