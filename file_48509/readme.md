序号：48524
名称：神效感应丸
组成：黑角沉1两，木香1两，檀香1两，全丁香1两，陈皮1两，青皮1两，黄连1两，砂仁1两，香附子1两，制半夏1两，三棱1两，莪术（并煨）1两（净，为末），肥乌梅肉100文重，巴豆300粒（肥白者，去衣膜心）。
出处：《永类钤方》卷十二引《浙方混元邓山房方》。
主治：宿食，气滞，酒后呕吐。
加减：None
功效：消宿食，除积滞。
用法用量：每服10丸，宿食，陈皮汤送下；气滞，茴香汤送下；酒后呕吐，淡姜汤送下。
制备方法：上用瓷器盛巴豆，上以乌梅肉盖之，以陈米醋浸，与乌梅肉平于甑上蒸极烂，以巴豆红色为度，却擂2味极烂，次用糯米粽和前药，捣千百杵，以黑色为度，众手丸如萝卜子大。
临床应用：None
各家论述：None
用药禁忌：常服不动脏腑。
附注：化铁丹《永类钤方》卷十二引《浙方混元邓山房方》）、邓山房感应丸（《玉机微义》卷二十）、邓山感应丸（《医统》卷三十三）。
