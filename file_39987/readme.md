序号：39997
名称：育真丹
组成：代赭石4两，左顾牡蛎（去两头，取中间者用）4两，紫石英4两，赤石脂4两，上药并为细末，米醋和成剂，匀分为6铤，入在甘锅子内烧通赤，半时辰取出，放冷，再捣为细末，次入：乳香2两（别研），茴香（微炒）2两，五灵脂（去砂石）2两，干姜2两（炮）。
出处：《杨氏家藏方》卷十五。
主治：妇人三十六疾，下脏久虚，沉寒痼冷，带下五色，变易不定，渐觉瘦弱。
加减：None
功效：None
用法用量：每服20丸，食前、空心煎茴香酒送下。
制备方法：上药乳香以下4味为细末，与前4味末和匀，醋煮糯米糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
