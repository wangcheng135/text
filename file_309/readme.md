序号：309
名称：五香丸
组成：沉水香1两，青木香1两，丁香1两，朱砂（别研）1两，麝香（别研）2两，犀角（锉，取屑）2两，熏陆香2两，栀子仁2两，连翘2两，石膏（别研）2两，芒消（熬）3两，蜀升麻3两，大青3两，干蓝3两，栝楼3两，干葛3两，茵陈3两，黄芩3两，肉桂3两，芎3两，茯苓3两，巴豆3两（去心皮，熬令变色，别研如脂），大黄2两。
出处：《外台》卷三十七。
主治：心腹臌胀冷泻，鬼气疰忤，时温热疰病，鬼疟病，疸黄垂欲死。
加减：None
功效：None
用法用量：有患时温热疰病，鬼疟病，心腹臌胀，疸黄垂欲死者，可服4-5丸，或至6-7丸。但取2-3行快利为度，利止即愈。
制备方法：上为末，炼蜜为丸，如梧桐子大，封以油蜡纸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
