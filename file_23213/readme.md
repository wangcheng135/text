序号：23221
名称：人参散
组成：人参、麝香、片脑各少许。
出处：《准绳·类方》卷三。
主治：噎膈胃反，关格不通。
加减：None
功效：None
用法用量：甘草汤调服。
制备方法：上为末。
临床应用：None
各家论述：《张氏医通》：此云歧子治噎膈胃反、关格不通九方之一。用独参汤峻补其胃，稍加脑、麝，以发越其气，得补中寓泻之至诀。乃肥盛气虚、痰窒中脘、及酒客湿热、郁痰固结之专剂。以中有脑、麝、善能开结利窍散郁也。
用药禁忌：None
附注：①《法律》：此方辄用脑、麝，耗散真气，才过胸中，大气、宗气、谷气交乱，生机索然尽矣，能愈病乎？②《法律》与《张氏医通》，均称此方出云歧子，考今《云歧子保命集》关格门无此方。
