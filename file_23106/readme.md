序号：23114
名称：芎辛散
组成：川芎1钱半，细辛1钱半，苍术1钱，甘草1钱，干姜1钱。
出处：《证治汇补》卷四。
主治：寒湿头痛，脉细者。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：《医略六书》：寒湿袭经，抑遏不散，故清阳之气不伸，不能分布，故头痛不止焉。川芎活血以上荣头角，细辛散寒以旁达肌表，苍术燥湿强脾兼主升阳，干姜温中散冷兼能补火，甘草援中以和诸药也。使寒湿解散，则清阳得伸而经气清和，头痛无不愈矣。此辛燥散寒之剂，为寒湿头痛之专方。
用药禁忌：None
附注：芎辛汤（《医略六书》卷二十一）。《医略六书》本方用法：水煎，去滓温服。
