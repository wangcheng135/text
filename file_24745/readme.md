序号：24753
名称：托里温中汤
组成：沉香1钱，丁香1钱，益智仁1钱，茴香1钱，陈皮1钱，木香1钱半，甘草（炙）2钱，姜活3钱，干姜（炮）3钱，黑附子（炮，去皮脐）4钱，生姜5片。
出处：《卫生宝鉴》卷十三。
主治：疮为寒变而内陷者。
加减：None
功效：None
用法用量：作1服。水3盏，煎至1盏，去滓温服。
制备方法：None
临床应用：None
各家论述：《内经》云：寒淫于内，治以辛热，佐以苦温。故以附子、干姜大辛热，温中外，发阳气，自里之表，故以为君；羌活味苦辛温，透关节；炙甘草甘温，补脾胃，行经络，通血脉；胃寒则呕吐，呃逆不下食，益智仁、丁香、沉香大辛热，以散寒为佐；疮气内攻，气聚而为满，木香、茴香、陈皮苦辛温，治痞散满为使也。
用药禁忌：忌一切冷物。
附注：托里温中散（《医林纂要》卷十）。
