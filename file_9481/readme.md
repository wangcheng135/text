序号：9481
名称：水银丸
组成：水银3分，硫黄半两（2味俱微炒之，细研），砒霜半两，芦荟半两（细研），朱砂半两（细研，用水飞过），蝉壳1分（微炒），天灵盖1分（涂酥，炙焦黄），鼓皮中蛀灰1分，白猪粪灰1分，蝉灰1分，蛤蚧1枚（涂酥，炙令微黄），乌驴蹄灰1分，雄黄1分（细研）。
出处：《普济方》卷三八三。
主治：小儿五疳，四肢黄瘦，腹胀气粗，发干作穗，眼鼻多痒，精神昏闷，不欲饮食。
加减：None
功效：None
用法用量：以麝香蜜水下。3天后便煎桃柳汤浴儿了，以青衣盖遍身，虫出，或泄恶气，并泻恶物，便是病源已出，小儿每3岁加1丸，宜粥饮送下，日3次。
制备方法：上为末，入研了药令匀，以苦参半斤锉碎，用水5升，浸一宿，煮至1升，去苦参，候熬成膏，入诸药为丸，如绿豆大，后、入去却汁獖猪胆内盛，悬于舍东阴七日，候干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
