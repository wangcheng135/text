序号：25992
名称：阳丹
组成：黄连1两，黄柏1两，大黄5钱，麻黄5钱，川芎5钱，白芷5钱，黄芩5钱，防风5钱，龙胆草5钱，细辛1钱半，千里光1钱半，脑荷1钱半，当归1钱半，连翘1钱半，羌活1钱半，荆芥1钱半，木贼1钱半，栀子3钱，白菊花3钱，生地黄3钱，赤芍药3钱，苦参3钱，苍术（一方有鸡柏树根不拘多少），上药以井水洗净，锉碎，以井水浸于铜器内，春3、夏2、秋4、冬5日，晒，常将手挪出药味，晒出药力，熟绢滤净，留清汁1碗以飞药，留浊汁3碗以淬药，却用熔铜锅子1个，装打碎甘石1斤在内，新瓦盖上，松炭固济，烧令透极红色钳出，少时淬入药汁内爆淬3次，就将留下清汁飞细，令千万余下，澄清去浊，晒干，再（？？）令无声为度，细绢重罗过，瓷器收贮听用，炉甘石1钱，麝香3厘，片脑1分。
出处：《准绳·类方》卷七。
主治：诸般外障，赤脉贯睛，怕日羞明，沙涩难开，胞弦赤烂，星翳覆瞳。
加减：None
功效：None
用法用量：点眼；如有翳膜，配合阴丹、一九、二八、三七、四六等丹。
制备方法：上为细末，次入片脑（石禹？？）嫩，熟绢罗过，磁器收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
