序号：58358
名称：真珠散
组成：真珠半两（细研），牛黄半两（细研），天竹黄3分（细研），黄芩1两，龙齿3分，朱砂半两（细研），防风半两（去芦头），人参3分（去芦头），茯神3分，麦门冬1两（去心，焙），远志半两（去心），白鲜皮半两，金箔50片（细研），银箔50片（细研），麝香1钱（细研），犀角屑半两，甘草3分（炙微赤，锉），胡黄连3分，铁粉3分，白附子3分（炮裂），甘菊花3分，羚羊角屑半两。
出处：《圣惠》卷二十。
主治：风狂乱语，心热狂走。
加减：None
功效：None
用法用量：每服1钱，以薄荷温水或梨汁调下，不拘时候。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：忌生血。
附注：本方方名，《普济方》引作“珍珠散”。
