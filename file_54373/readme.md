序号：54388
名称：柴胡知母汤
组成：柴胡1两，知母1两，桔梗（炒）1两，厚朴（去粗皮，生姜汁炙）1两，熟干地黄（焙）1两，白茯苓（去黑皮）1两，山芋1两，黄耆（锉）1两，紫菀（去苗土）1两，地骨皮1两，黄芩（去黑心）半两，甘草（炙，锉）3分，桂（去粗皮）3分，半夏（汤洗7遍，炒）3分。
出处：《圣济总录》卷三十一。
主治：伤寒后体虚成劳，遍身盗汗，四肢无力，口苦憎寒，又多咳嗽。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，加生姜1枣大（拍碎），大枣3枚（擘破），同煎至8分，去滓，空心温服，日2次。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
