序号：79076
名称：增减旋覆代赭汤
组成：旋覆花3钱（包煎），吴茱萸1分（拌炒），小川连6分，制香附2钱，代赭石3钱（拌），仙半夏1钱半，新会皮1钱半，沉香汁2匙（冲）。
出处：《重订通俗伤寒论》。
主治：肝气横逆，轻则嗳气胸痞，重则呃逆胃胀。
加减：呃逆甚者，加公丁香9支，柿蒂30个，辛通苦涩以止呃；痞胀甚者，加真川朴1钱半，槟榔汁两匙（冲），辛开重降以宽胀；食滞者，加莱菔子1钱半拌炒春砂仁8分，消食和气以导滞；便秘者，加苏子1钱半拌捣郁李仁4钱，辛滑流气以通便。
功效：清降肝逆。
用法用量：先用鲜刮淡竹茹4钱，鲜枇杷叶1两（去毛净，剪去大筋）煎汤代水。
制备方法：None
临床应用：None
各家论述：何秀山：方以旋、赭重降逆气为君；臣以莱、连、橘、半苦辛通降以清肝和胃；沉香，香附辛香流气以疏肝平逆；妙在佐以竹茹，使中结之肝气旁达；使以枇杷叶，令上逆之肝气清降。此为清肝降逆，佐金制木之良方。
用药禁忌：None
附注：None
