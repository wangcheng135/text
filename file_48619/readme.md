序号：48634
名称：神验噎膈方
组成：威灵仙2两（水浸1宿取出，捣汁），食盐1钱5分，狗宝末3分。
出处：《良方集腋》卷上。
主治：噎膈。
加减：None
功效：None
用法用量：上药共调和，炖温服。服之少顷，病者觉上焦胸膈之间气机旋扰作动，勿令呕，次日仍用威灵仙2两，浸之隔宿，如前捣汁，入食盐1钱5分，刮入狗宝末4分，调服之，觉动处略下；第3日仍如前法再用威灵仙2两，绞汁入食盐1钱5分，狗宝末5分，调服，少停，动更下，则大便下黑血痰涎。下之后正气虚耗。必须预备3剂，服之则气机通利，病即愈矣。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：愈后必得食淡一年，庶不再发，倘不能食淡，再发不治矣。
附注：None
