序号：56491
名称：浸虎骨酒
组成：防风2两（拣，去芦），秦艽2两，萆薢2两，羌活2两，虎骨2两（酥炙），川牛夕2两（去芦），败龟壳（捶碎）、晚蚕沙2两（炒黄色），枸杞子5两，苍耳4两，（炒黄色，捶碎），川乌2两（炮，去皮脐），松节2两，当归2两。
出处：《医方类聚》卷二十三引《经验秘方》。
主治：一切风疾。
加减：None
功效：None
用法用量：每日早、午、夜3时各取酒1盏服之，勿令药力断绝；候病痊时，将药晒干为末，酒糊为丸，如梧桐子大。每服30-50丸，空心用好酒送下，间服小续命汤佐之。
制备方法：上细锉碎，用无灰酒1斗，入药浸在坛内，牢固密封，俟14日，开坛取酒。
临床应用：风疾丞相史史深，节宣不谨，遂染风邪，手足拘挛，不堪跪拜，病越十年。饮此酒及1升，便手可梳头，2升手足屈伸有力，3升语言舒畅，行履如故，4升四肢通暖，5升百节俱遂，举步如飞。
