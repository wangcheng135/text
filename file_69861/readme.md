序号：69877
名称：小神丸
组成：白蒺藜（去刺）2两，北蝉蜕（去头足）3钱半，北全蝎（米汁洗，糯米水炒）2钱半（或姜汁炒），荆芥穗2钱半，北防风2钱半，大风子肉（壳不用，黑豆煮7次，去净油，否则伤目）2钱半，大羌活5钱，全当归（酒洗）3钱半，大川芎（酒洗）2钱半，土麻仁1两半，白苦参（酒洗）1钱半（无癣者，此味不用）。
出处：《疯门全书》。
主治：麻风。
加减：血热，加生地；胃热现面，加白芷、知母；鼻塞，加桑皮、黄芩；阴虚，加首乌；肝热，加丹皮；目昏，加蔓荆、菊花；痹多，加元参；面多红云、红堆，或油光，加白附、僵蚕；血枯，加丹参；面脚浮肿，加防己；拘挛，加钩藤，或茯神、白术。
功效：None
用法用量：每服4-5钱，空腹茶送下，早、晚各1次。间时方进饮食。服10日，即停3日，此3日服行药丸10粒，3日3次，后又服小神丸。
制备方法：上为末，老米饭捣烂和为丸。
临床应用：None
各家论述：None
用药禁忌：忌热汤、热茶。
附注：服完，未发外者即发，已发外者或更甚，盖毒既攻出，即遍身溃烂，亦无妨，但要麻木处渐轻渐狭，即渐浮渐阔，或别处再发一二处，亦是药力攻散之功，勿疑。
