序号：66952
名称：温中大黄汤
组成：干姜1分，桂心1分，厚朴1分，甘草1分，当归2分，人参2分，茯苓2分，白术2分，大黄6分，桔梗3分。
出处：《千金》卷十五。
主治：小儿暴冷水谷下，或乳冷下，青结不消；或冷实吐下，干呕烦闷，及冷滞赤白下者。
加减：None
功效：None
用法用量：上(口父)咀。以水2升半，煮取8合。凡儿30-60日，1服2合；70-100日，1服2合半；200日以来，1服3合。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：若已服诸利汤去实，胃中虚冷，下如水，干呕眼陷，烦扰不宜利者，可除大黄；若中乳，乳母洗浴水气未消，饮儿为霍乱者，但用大黄；小儿诸霍乱宜利者，使用大黄，不须利，宜温和者，则除之。
