序号：34306
名称：万灵膏
组成：羌活、防风、秦艽、苍术、独活、白芷、萆薢、官桂、大麻、川乌、草乌、干姜、当归、木瓜、川芎、牛膝、防己、豨莶、风藤、半夏、前胡、枸杞、南星、虎骨、白茄根、麻黄、苍耳子、高良姜、晚蚕沙、威灵仙、五加皮、玄胡索、川续断、红花、桃仁、苏木、芷壳、丹皮、骨碎补、乌药各等分，闹羊花、绵花子各倍用。
出处：《活人方》卷六。
主治：风寒湿痹，遍身经络骨节酸疼，跌打损伤，闷肭脞气，心胸腰背攻刺为痛。
加减：治癖，加入阿魏、雄黄。
功效：None
用法用量：凡用，先以生姜擦过贴，贴后以炒热艾或炒麸皮熨之。
制备方法：麻油熬，炒东丹收起，冷加细料香药：五灵脂、鸦片、血竭、木香、乳香、没药、冰片、麝香。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
