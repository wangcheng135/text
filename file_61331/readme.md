序号：61347
名称：高粘除秘授消散败毒万应灵膏
组成：当归2两，生地2两，白芷2两，银花2两，川乌2两，草乌2两，防风1两，荆芥1两，赤芍1两，羌活1两，独活1两，僵蚕1两，蝉蜕1两，刺蒺藜1两，灵仙1两，首乌1两，川牛膝1两，山甲1两，蛇蜕1两，鲜皮1两，甘草1两，黄柏1两，宫桂1两，乳香4钱，没药4钱，密陀僧（研，后入）8两，广丹（后入）1斤8两。
出处：《丁甘仁家传珍方选》。
主治：None
加减：None
功效：消散败毒。
用法用量：None
制备方法：上研细。用麻油6斤浸药，春5、夏3、秋7、冬10，日数足，入锅内慢火熬枯，去滓，净油入锅，熬至滴水成珠，下密陀僧末熬沸，离火置冷炉上片时，再投东丹，其丹不烘不炒，下为冷丹；或烘炒为熟丹。但下冷丹，极要仔细，热丹好收，此丹投入，不住手搅，候冷，收成膏时，再下乳香、没药搅匀，即成膏矣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
