序号：66899
名称：提毒异功散
组成：真血竭6分，斑蝥4钱，大梅片3分，制乳香6分（去净油），全蝎6分，麝香3分，制没药6分（去净油），元参6分。
出处：《白喉全生集》。
主治：白喉急症。
加减：None
功效：消肿止痛。
用法用量：用时将膏药摊开，放散于膏药中心，贴颈项，须对喉内肿处，喉左肿，贴左侧；右肿，贴右侧；左右肿，贴两侧。阅5-6时，揭去膏药，贴处必起水泡，用针刺破，揩净毒水。
制备方法：上除梅、麝外，先将斑蝥去头翅足，糯米拌炒，以米色微黄为度，炒后去米不用，后与上药共研细末，过绢筛，再合梅、麝乳细，瓷瓶收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
