序号：33228
名称：一味黄芩汤
组成：片芩1两。
出处：《本草纲目》卷十三引李杲方。
主治：骨蒸发热，肤如火燎，咳嗽吐痰，烦渴，脉浮洪。风劳肤如火燎，重按不热，日西更甚，喘嗽，洒淅寒热，目赤心烦。
加减：None
功效：None
用法用量：水2钟，煎1钟，顿服。
制备方法：None
临床应用：骨蒸发热：予年二十时，因感冒咳嗽既久，且犯戒，遂病骨蒸发热，肤如火燎，每日吐痰碗许，暑月烦渴，寝食几废，六脉微洪，遍服柴胡、麦门冬、荆沥诸药，月余益剧，皆以为必死矣。先君偶思李东垣治肺热如火燎，烦燥引饮而昼盛者，气分热也，宜一味黄芩汤，以泻肺经气分之火。遂按方用片芩一两，水二钟，煎一钟，顿服。次日身热尽退，而痰嗽皆愈。
各家论述：None
用药禁忌：None
