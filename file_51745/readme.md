序号：51760
名称：钟乳丸
组成：钟乳粉2两，熟干地黄1两半，人参1两（去芦头），薯蓣1两，肉苁蓉2两（酒浸1宿，刮去皱皮，炙干），牛膝1两半，黄耆1两，白茯苓1两，枸杞子1两，巴戟1两，杜仲1两半（去粗皮，炙微黄，锉），续断1两，天门冬1两半（去心，焙），石斛1两半（去根，锉），桂心1两，蛇床子1两，补骨脂1两（微妙），石龙芮1两，覆盆子1两，防风1两（去芦头），山茱萸1两，五味子1两，远志1两（去心），附子1两（炮裂，去皮脐），鹿茸2两（去毛，涂酥炙微黄），车前子1两。
出处：《圣惠》卷三十。
主治：虚劳乏弱，精少骨痿，腰膝无力，不能饮食，日渐羸困。
加减：None
功效：None
用法用量：每服30丸，食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
