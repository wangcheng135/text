出处：《外科理例·附方》。
主治：男妇卒暴中风，不省人事，斜口噤，失音，涎盛，拘挛，或产后暗风及破伤风，内外一切伤寒。
加减：None
功效：None
用法用量：每服1丸，细嚼，温酒茶清漱下，临卧、空心各1丸，随症上下，食前后服；头痛，茶送下；男妇卒暴中风，不省人事，喎斜口噤，失音，涎盛，拘挛，临睡烂研1丸，好酒化下便睡觉，有汗将病人手背随即舒拳，天明用人扶行，早饭、日西再服1丸，可痊；产后暗风及破伤风，内外一切伤寒，人年40以上，间2-3日服1丸，永无风疾。
制备方法：上为末，炼蜜为丸，如弹子大，每药1两半作10丸，金箔为衣。
