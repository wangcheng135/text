序号：67606
名称：小乌玉丹
组成：乳香半两（研），麝香2钱（研），芝麻1两（生用），苦楝根1两（生用），槐角子1两半（生用），雷丸1两（生，锉碎），乱油头发3两（剪碎，烧存性），黄牛角（角思）3两（生用），生猪前甲49个（篓内烧灰存性），南星1两（生用），穿山甲4两（盐固济，煅存性），白矾1两（飞过），半夏1两（生用），枳壳1两（烧）。
出处：《普济方》卷二九七。
主治：痔漏疮。
加减：None
功效：None
用法用量：每服20丸，空心用米饮送下，1日2次；如绿豆大亦得，可服30丸。
制备方法：上为细末，用好酒醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌鸡鹅肉，新姜、豆腐淹藏等物。
附注：None
