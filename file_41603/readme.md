序号：41618
名称：金锁补真丹
组成：川续断5分，川独活，谷精草5分，黄精草5分，莲花蕊1两（干用），鸡头粉1两（煮熟用），鹿角霜1两，金樱子5两（去皮尖）。
出处：《普济方》卷二一八引《德生堂方》。
主治：梦遗白浊。
加减：None
功效：升降阴阳，壮理元气，益气，补丹田，振奋精神，大能秘精。
用法用量：每服止1丸，空心温酒送下。服数日，自然益气补丹田，精神加倍。若欲药行，早晨另丸药50丸，如梧桐子大，温酒送下，应验。
制备方法：上为细末。次将金樱子捶碎。用水3升，煮至1升，去滓，银石器内用慢火熬至3合成膏，和匀，将药末为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
