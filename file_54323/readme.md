序号：54338
名称：柴胡抑肝汤
组成：柴胡2钱半，赤芍1钱半，牡丹皮1钱半，青皮2钱，连翘5分，生地5分，地骨皮1钱，香附1钱，苍术1钱，山栀1钱，川芎7分，甘草3分，神曲8分。
出处：《医学入门》卷八。
主治：寡居独阴，寒热类疟。
加减：None
功效：疏肝开郁，散结气结血，凉心启脾。
用法用量：水煎，空心临卧服。
制备方法：None
临床应用：None
各家论述：《医略六书》：生地黄滋阴凉血，地骨皮退热清肌，柴胡疏热解郁，青皮破气平肝，丹皮凉血以化瘀结，赤芍破血以通经隧，川芎行血中之气，山栀清屈曲之火，神曲消食滞，苍术燥脾湿，香附调气解郁，甘草泻火缓中也。
用药禁忌：None
附注：柴胡抑肝散（《古今医鉴》卷十一）、柴胡益肝散（《寿世保元》卷七）。
