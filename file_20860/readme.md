序号：20868
名称：至宝丸
组成：真金华香附1斤4两（择大而毛净者，童便浸3日，捣碎，晒干，为末，用马料黑豆数升，煮浓汁拌晒3次，复以人乳拌晒1次，磨为末，用12两），熟地黄3两，当归2两，杜仲（盐水炒）2两，川芎1两，人参1两，白茯苓1两，牡丹皮1两，延胡索1两，滴乳香1两，没药（2味瓦上焙去油，研）1两，赤石脂（细腻粘唇者，火煅醋淬）1两，白术1两5钱，白芍1两5钱，鹿角胶（牡蛎粉炒成珠，去粉）4两。
出处：《慈幼新书》卷首。
主治：胎前一切症。
加减：None
功效：调经止带，安胎补虚。
用法用量：每早空心服1丸，白汤送下。
制备方法：上药为末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
