序号：34326
名称：万宝丹
组成：水银1两，密陀僧1两，白矾1两，食盐（炒）1两，火消1两，明雄黄5钱，朱砂5钱，滁州青瓷器（打碎研细）2两。
出处：《灵药秘方》卷下。
主治：臌胀膈气。
加减：None
功效：None
用法用量：初服3分5厘，用雄鼠粪煎汤送下；以后只用竹青煎汤，微加姜汁服。
制备方法：先将水银、瓷末共研不见星，次下陀僧再研，再下矾、盐、消、雄、砂共研匀，入阳城罐内封口，升3炷香，取出灵药。二转加法，取前灵药，又加水银1两，研不见星，又下火消，盐、矾各1两，明雄5钱、朱砂5钱，研匀听用，再取出山铅4两，打薄剪碎，放阳城罐底上，再放药末在上，封固，打3炷香，取灵药配后药用。配药法，每前药1钱，用牛黄5分、狗宝5分，珍珠、琥珀、直僵蚕（糯米炒）、全蝎（酒洗，去头足，糯米炒）、沉香、川贝母、硼砂、朱砂、雄黄、元明粉、木香、川连、吴茱萸（煮）、川芎、白芥子、萝卜子，以上各1钱，巴豆仁（甘草水煮，去油）5分，麝香3分，牙皂8分（炒），金银箔各30张，五倍子1个，打1孔，入大黄末填满塞紧，入多年瓦便壶内封口，火煅候冷，取五倍子、大黄为末，与前诸药和匀，用小竹刮青煎汁，打糊为丸，萝卜子大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
