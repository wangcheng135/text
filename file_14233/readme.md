序号：14240
名称：伤心汤
组成：茯神3两，黄芩3两，远志3两，干地黄3两，甘草1两，阿胶1两，糖1两，半夏2两，附子2两，桂心2两，生姜2两，石膏4两，麦门冬4两，大枣30枚。
出处：《千金》卷十四。
主治：心气不足，腹背相引痛，不能俛仰。
加减：None
功效：None
用法用量：以水1斗，取煮3升，去滓，纳糖、阿胶更煎，取2升2合，分3服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：地黄、阿胶滋培阴血，桂心、附子、茯神、远志开通经窍，黄芩、石膏、麦门冬清理肌表旺气，半夏、姜、枣涤除隔上顽痰，甘草、饴糖和脾缓急，以和腹背相引之痛。
用药禁忌：None
附注：None
