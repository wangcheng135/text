序号：36288
名称：和中丸
组成：白术（陈土炒）4两，扁豆（炒）3两，茯苓1两5钱，枳实（面炒）2两，陈皮3两，神曲（炒黑）、麦芽（炒）、山楂（炒），香附（姜汁炒）2两，砂仁1两5钱，半夏（姜汁炒）1两，丹参（酒蒸）2两，五谷虫（酒拌，炒焦黄色）3两。
出处：《医学心悟》卷三。
主治：鼓胀。腹胀食积，疟后痰结，或血裹肝气，伏于胁下，时痛时止，而成痞积。
加减：若寒气盛，加干姜、吴萸、肉桂；若湿热盛，加黄连、连翘；若大便闭结，先用三黄枳术丸下之，随用本方渐磨之；若兼瘀血，加厚朴、赤芍；若脾气虚弱，用六君子汤吞服此丸，或以补中益气汤送下。
功效：消痞。
用法用量：每服2钱，上午、下午开水送下，每日2次。
制备方法：荷叶1枚，煎水迭为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
