序号：72298
名称：滋肾柔肝汤
组成：熟地20g，枣皮10g，山药20g，茯苓10g，丹皮10g，泽泻10g，杞子15g，菊花10g，当归10g，白芍60g，何首乌30g，甘草30g。
出处：《千家妙方》引白光中方。
主治：肝肾阴虚，目系失养而致之麻痹性斜视。
加减：None
功效：None
用法用量：水煎服，每日1剂。
制备方法：None
临床应用：贵某某，男，47岁。视物成双已四月余，曾赴成都某医院诊为“左眼麻痹性斜视”。查左眼正常，右眼球偏向内眦，转动不灵，自觉微干涩，全身尚好，舌尖红，少津，脉弦细微数。服上方二剂，眼球微能转动，但视远物仍成双，仍以上方加丹参30g，细辛1g，再服3剂，视物已不成双，只是原物宽大，纳食稍减，再以上方加陈皮、谷芽，服用四剂，眼球转动灵活，视物基本正常。后用杞菊地黄丸常服，以巩固疗效。
各家论述：None
用药禁忌：None
附注：None
