序号：31486
名称：补肝汤
组成：天门冬（去心，焙）、酸枣仁（微炒）、柴胡（去苗）、当归（切，焙）、羌活（去芦头）、防风（去叉）、桂（去粗皮）、细辛（去苗叶）、赤茯苓（去黑皮）、升麻、秦艽（去苗土）、黄耆（锉）、杜仲（去粗皮，炙，锉）、鳖甲（去裙襕，醋炙，锉）、鹿茸（去毛，酥炙）、牛膝（酒浸，切，焙）、天麻、黄明胶（炙燥）、山茱萸各等分。
出处：《圣济总录》卷八十六。
主治：肝劳。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，加生姜2片，大枣1枚（擘），煎至7分，去滓，食前温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
