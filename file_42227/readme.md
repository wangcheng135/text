序号：42242
名称：青冢丸
组成：广木香2两（青者佳），全蝎6个（焙），狐肾5个（去筋肉净，酥炙），犬肾10个（去筋肉净，酥炙），菟丝子6两（酒浸，取头末3两），小茴香3两（盐炒），白茯苓2两（白者佳），破故纸5两（酒浸，取头末2两），山茱萸1两（去心），川山甲1两2钱（酥炙），胡桃仁6两（汤洗，去黄皮，另研）。
出处：《普济方》卷二二二。
主治：男子下元虚冷，肾囊阴湿，行步艰辛，饮食不进。
加减：None
功效：添精髓，壮筋骨，美饮食；后服令人有子。
用法用量：每服3丸，空心温酒送下。
制备方法：上为细末，用炼过白沙蜜同胡桃膏子和前药，入石臼杵千余下，干物压之，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌饮酒及房室等事。
附注：None
