序号：66455
名称：小九转灵丹
组成：制灵砂1两，芦荟1两，制朱砂5钱，洛阳花5钱。
出处：《灵药秘方》卷下。
主治：遗精白浊，小儿急慢惊风，老人中风，伤寒有汗，结胸，阴症，痰嗽痰喘，脚气麻木不仁，诸般疼痛，黄疸，虫积，耳聋耳痛，口破痛烂，三焦烦热作渴，赤淋白带，诸般肿毒，癫症，痢疾，疟疾。
加减：None
功效：None
用法用量：每服3丸，男子遗精白浊，每清晨用灯心莲肉汤送下；小儿急惊，木香研细末，姜汁竹茹汤调匀化下，以痰降为度；小儿急慢惊，人参、白术、当归、陈胆星、半夏、竹沥、姜汁化下；老人中风，防风通圣散煎汤送下，如类中虚症，独参汤送下；结胸，大、小柴胡汤送下；伤寒有汗者，桂枝汤送下；阴症，附子、人参、肉桂、炮姜汤送下；痰嗽，半夏、茯苓汤送下；痰喘，当归、竹沥汤送下，虚喘加人参；脚气，防风、当归、木瓜、牛膝、羌活、秦艽汤送下；麻木不仁，黄耆、天麻汤送下；诸般疼痛，乳香、没药汤送下；黄疸，炒山栀、茵陈汤送下；诸虫积，桃仁、楝树根（朝南者）煎汤送下；耳病耳聋耳痛，黄柏、生地、石菖蒲汤送下；口破及痛烂等证，山豆根、黄芩、骨皮汤送下；三焦烦热作渴，人参、白术、麦冬、知母汤送下；赤淋白带，平陈汤送下；诸般肿毒，人参、麝香汤送下；癫症，蜈蚣、乳香、没药汤送下；中风不语，握拳咬牙，闭目不省人事者，人参5分、黄耆5发、白术5分、附子5分，川乌4分，甘草少许，竹沥、姜汁3匙，大枣2个煎汤灌之，俟苏醒后再用竹沥、姜汁汤送下；中风不醒，服前药后更进3丸，再用顺气散数剂，相其虚实调理。
制备方法：上为细末，炼蜜为丸，如绿豆大，金箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
