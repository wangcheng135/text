序号：41950
名称：青龙丸
组成：附子（炮裂，去皮脐）1两，芎1两，白术（米泔浸3日，每日换泔，取出焙干）1两，独活（去芦头）1两，蒲黄（用纸衬炒过）1两，藁本（去苗土）1分，麻黄（去根节，百沸汤中急煮过，焙）3分，丹砂（研）1分，牛黄（研）1分，龙脑（研）1分，麝香（研）半钱，熊胆1分（滴水1-2点，令化入药内）。
出处：《圣济总录》卷八。
主治：中风手足不随，涎涕胶粘。
加减：None
功效：None
用法用量：每服半丸或1丸，临卧以熟水化下。其患甚者，每夜研2丸。如小儿惊涎，每丸分作3服，用薄荷汤化下。
制备方法：上为细末，别用水银3两，以蒸枣50枚（去皮核）同研至水银星尽为度，次入诸药，炼蜜为丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
