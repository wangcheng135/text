序号：56561
名称：大效没药丸
组成：五灵脂2两，草乌头1两半，血余1两半（用纸筋泥固济1合子，令干，先入乌头，次灵脂，以血余盖头，泥固济，口绢牢，候干，以10斤炭火煅之，火3分去2，然后拨去火，候冷，取研为末），酸酱子25个，没药半两，当归半两，地龙（去土，研）半两。
出处：《普济方》卷三二三。
主治：妇人血风虚劳，身体壮热疼痛，肌肉黄瘦，饮食进退，多困少力，口苦舌干，百骨节痠痛，头目昏重，涕唾不利，心胸满塞，时发寒热。
加减：None
功效：None
用法用量：每服10丸，温酒送下，空心、日午、临卧各1服，不嚼破。多年患者，不过20服愈；近日患者，10服愈。
制备方法：上为细末，入前烧者药，纳在1处，罗3-5遍令匀，用水浸蒸饼和为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
