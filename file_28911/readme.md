序号：28919
名称：苍芎千里饮
组成：苍术（薄切，米泔水浸）、干木瓜、白芍药、川芎、人参、枳壳（去瓤）、白茯苓、甘草（炙）、大黄、陈皮（去白）、半夏（泡7次）、桔梗（炒）、前胡、干葛、紫苏、木香各等分。
出处：《朱氏集验方》卷一。
主治：久近脚气。
加减：None
功效：None
用法用量：上（口父）咀。每服5钱，水1碗半，加生姜7片，煎至1碗，去滓，食前温服。如服此药，先用大蒜3头，剥去皮，洗净擂细，捻作饼子，贴所患脚心，外用绢帛敷之，觉脚心微有蒜气，便温前药顿服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“苍芎千里散”。
