序号：79158
名称：蜥蜴丸
组成：蜥蜴2枚，蜈蚣2枚，地胆50枚，（庶虫）虫30枚，杏仁30枚，蜣螂14枚，虻虫30枚，朴消1两18铢，泽漆18铢，桃奴18铢，犀角18铢，鬼督邮18铢，桑赤鸡18铢，芍药1两半，虎骨1两半，甘草1两，巴豆1两18铢，款冬花18铢，甘遂1两6铢，干姜1两。
出处：《千金》卷十一。
主治：癥坚水肿，蜚尸，遁尸，百注，尸注，骨血相注，恶气鬼忤，蛊毒邪气往来，梦宿存亡，留饮结积，虎狼所啮，猘犬所咋，鸩毒入人五脏，妇人邪气鬼忤。
加减：None
功效：None
用法用量：先食饮服3丸，每日1次。不知，加之。不敢吐下者，1丸日1服。有人风冷注癖坚20年者，得愈。
制备方法：上为末，别治巴豆、杏仁如膏，纳药末研调，下蜜捣为丸如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中桃奴，《圣惠》作“桃仁”。
