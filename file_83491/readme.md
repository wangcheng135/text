序号：83509
名称：乌金煎
组成：黑豆2升，茄子根3斤（碎锉，以水3斗与豆同煮至烂熟，去滓更煎，取汁5升，入药末内），桂心1两，羌活1两，芎1两，酸枣仁1两（微炒），虎胫骨1两（涂酥，炙令黄），防风1两（去芦头），萆薢1两（锉），当归1两，仙灵脾1两，牛膝1两（去苗），附子2两（炮裂，去皮脐）。
出处：《圣惠》卷二十一。
主治：风脚膝软弱，骨节疼痛，肢节无力，行立艰难。
加减：None
功效：None
用法用量：每日早晨以暖酒调下1茶匙，日晚再服。
制备方法：上为末，入于前2味汁中，以慢火煎，柳木篦搅，勿令住手，可一炊时止，盛于瓷器中。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
