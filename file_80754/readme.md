序号：80770
名称：麋茸丸
组成：麋茸2两（去毛，涂酥，炙微黄），肉苁蓉2两（酒浸1宿，刮去皱皮，炙干），薯蓣1两，菟丝子2两（酒浸3日，晒干，别捣为末），石斛1两（去根，锉），桂心1两，熟干地黄1两，巴戟1两，牛膝1两（去苗），山茱萸1两，枸杞子1两，五味子1两，人参1两（去芦头），赤石脂1两，柏子仁1两，泽泻1两，白茯苓1两，远志1两（去心）。
出处：《圣惠》卷九十八。
主治：None
加减：None
功效：补虚劳，倍筋力，除脾胃冷气，充肌肤，益颜色，补暖。
用法用量：每日服40丸，空心以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
