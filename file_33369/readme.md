序号：33378
名称：羌活散
组成：羌活1两，桃仁1两（汤浸，去皮尖双仁，麸炒微黄），人参半两，木香3分，鳖甲1两（涂酥，炙令黄，去裙襕），白术3分，桂心半两，白茯苓3分，白芍药半两，当归半两（锉碎，微炒），附子3分（炮裂，去皮脐），牛膝1两（去苗），防风半两（去芦头），续断3分，芎3分，熟干地黄1两。
出处：《圣惠》卷七十。
主治：妇人风虚劳冷，四肢羸弱，不能饮食，面色萎黄，腹内时痛。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
