序号：56530
名称：大钩藤饮
组成：钩藤1两1分，黄芩（去黑心）1两1分，麻黄（去节）1两1分，当归（切）3分，龙齿（研）1两，石膏（碎）2两半，赤芍药（去黑皮）1两，桂（去粗皮）1两，龙胆（去土）1两，牛黄（研）1两，杏仁（去双仁皮尖，麸炒，研）半两，甘草（炙，锉）1分。
出处：《圣济总录》卷一七一。
主治：小儿发痫，壮热。
加减：None
功效：None
用法用量：上12味，10味为粗末。每服3钱匕，以水1盏，煎至6分，去滓；下牛黄、杏仁，加自蜜、竹沥各少许，炼如饧，汤调服，如人行5里再服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
