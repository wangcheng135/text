序号：74048
名称：暖宫丸
组成：生硫黄6两，禹余粮（醋淬，手捻为度）9两，赤石脂（火煅红）3两，附子（炮，去皮脐）3两，海螵蛸（去壳）3两。
出处：《局方》卷九。
主治：冲任虚损，下焦久冷，脐腹(疒丂)痛，月事不调，或来多不断，或过期不至，或崩中漏血，赤白带下，或月内再行，淋沥不止，带下五色，经脉将至，腰腿沉重，痛连脐腹，小便白浊，面色萎黄，肢体倦怠，饮食不进，渐至羸弱；及治子宫久寒，不成胎孕。
加减：None
功效：None
用法用量：每服15-20丸，空心、食前温酒或淡醋汤送下。
制备方法：上为细末，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
