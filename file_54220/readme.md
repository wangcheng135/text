序号：54235
名称：大建中汤
组成：甘草2两，人参3两，半夏1升，生姜1斤，蜀椒2合，饴糖8两。
出处：《千金》卷十九。
主治：虚劳寒澼，饮在胁下，决决有声，饮已如从一边下，有头并冲皮起，引两乳内痛，里急，善梦失精，气短，忽忽多忘。
加减：里急拘引，加芍药、桂心各3两；手足厥，腰背冷，加附子1枚；劳者，加黄耆1两。
功效：None
用法用量：上(口父)咀。以水1斗，煮取3升，去滓，纳糖消，服7合。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：此本《金匮》三物大建中汤，于中除去干姜之守中，易入生姜以散表，更加半夏以运痰，甘草缓急。药虽小变而大义不殊。
用药禁忌：None
附注：None
