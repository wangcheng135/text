序号：73052
名称：痛风药酒
组成：鳖甲（七轮者，醋炙）1两，甘菊花（净）1两，防风1两，杜仲1两，人参1两，甘杞子1两，粉丹皮1两，石菖蒲（去芦根）1两，秦艽1两，虎骨（酒炙）1两，川羌活1两，油松节1两，牛蒡子1两，白术5钱，黄耆5钱，远志（去心，甘草水煮）5钱，山萸肉5钱，桔梗5钱，苍耳子5钱，僵蚕5钱，白茯苓5钱，牛膝5钱，川萆薢5钱，石蟹5钱，天花粉5钱，熟地5钱，明天麻5钱，白芍5钱，白附子5钱，独活5钱，天南星（姜汁泡）5钱，荆芥5钱，广陈皮5钱，白芷5钱，茅苍术（米泔浸，炒）5钱，川芎5钱，当归5钱。
出处：《疡医大全》卷二十八。
主治：四肢不举。
加减：None
功效：None
用法用量：每空心饮1杯，1日3次；药滓焙干，研细，好酒和丸，如梧桐子大。每服70丸，空心好酒送下。
制备方法：上以无灰酒2斗浸之，春、夏二七，秋、冬三七，封固勿泄气。
临床应用：None
