序号：58402
名称：破血散疼汤
组成：羌活1钱，防风1钱，中桂1钱，苏木1钱5分，连翘2钱，当归梢2钱，柴胡2钱，水蛭3钱（炒去烟尽，别研），麝香少许（别研）。
出处：《兰室秘藏》卷中。
主治：乘马损伤，跌其脊骨，恶血流于胁下，其痛苦楚，不能转侧，妨于饮食。
加减：None
功效：None
用法用量：分2次服，每服酒2大盏，水1大盏，除水蛭、麝香另研如泥，煎余药作1大盏，去滓，上火令稍热，调2味，空心服之。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：破血散痛汤（《东垣试效方》卷六）、破血散（《玉机微义》卷四十三）。本方方名，《准绳·疡医》引作“破血消痛汤”。
