序号：52076
名称：大补二天膏
组成：熟地、丹皮、山茱萸、黄耆、白术、枣仁、云苓、泽泻、山药、远志肉、当归、白芍、茯神、龙眼肉。
出处：《陈素庵妇科补解》卷一。
主治：室女天癸已至，复止不来。
加减：None
功效：滋补阴血，补脾和胃。
用法用量：内服。作煎亦可。
制备方法：None
临床应用：None
各家论述：先后二天俱不足，是以任脉虽通，冲脉未盛，气血不能充满，当二七之期，天癸乍至而仍断也。是方用山茱萸、熟地、丹皮、山药、茯苓、泽泻以补肾水，即六味丸遗意也。用黄耆、当归、白芍、白术、茯神、远志、枣仁、龙眼肉以补脾土，即归脾汤治法也。先天不足则补肾，以益真阴；后天不足则补脾，以生阴血。气血充足，则月事自以时下矣。
用药禁忌：None
附注：None
