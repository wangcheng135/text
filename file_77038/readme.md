序号：77054
名称：黑神散
组成：雄黑豆（小者是，炒，去黑皮）2两，当归、芍药、甘草（炙）、干姜、蒲黄（用安石器内，炒赤色）、肉桂、熟地黄（温水洗）各等分。
出处：《经效产宝续编》。
主治：死胎不下，产后血运，恶露不尽，腹痛。
加减：若30岁以上生产少者，不用桂、姜，却以炒生姜、红花各2两。
功效：None
用法用量：每服2钱，空心温酒调下。
制备方法：焙干为末。
临床应用：None
各家论述：1.《医方考》：方中蒲黄能逐败血，熟地、芍药、当归能养新血，干姜、肉桂能引新血而逐败血，甘草、黑豆能调正气而逐败气。_x000D_
2.《医方集解》：此足太阴、厥阴药也。熟地、归、芍之润以濡血，蒲黄、黑豆之滑以行血，桂心、干姜之热以破血，用甘草者缓其正气，用童便者散其瘀逆，加酒者，引入血分以助药力也。
用药禁忌：None
附注：黑桂散（《圣济总录》卷一六○）、蒲黄黑神散（《卫生家宝产科备要》卷四）。
