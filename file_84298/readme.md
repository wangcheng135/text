序号：84316
名称：五补丸
组成：巴戟2两，牛膝2两，山药2两，白蒺藜2两，菟丝子（酒浸3日，洗，焙干）1两半，木香1两半，人参1两半，白茯苓1两半，黄耆1两半，川椒（微炒）1两半，苁蓉1两半，远志（去心）1两半，附子（炮，去皮脐）1两，桂（去皮）1两，舶上茴香1两，北五味子1两，山茱萸1两，破故纸（微炒）1两。
出处：《医方类聚》卷一五七引《吴氏集验方》。
主治：男子本脏虚惫，下元冷极，阳道衰弱，夜多小便，膀胱积滞，脐下疼痛，疝气攻注，梦中遗泄，精神恍惚，眼暗耳鸣，腰膝生痠，筋力困乏，齿牙浮动。妇人血海虚冷，面色无光，心腹冷疼，四肢瘦弱，产后诸疾。
加减：None
功效：None
用法用量：每服30丸，空心盐汤或温酒送下。
制备方法：上择净称，为末，以酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
