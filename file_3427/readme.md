序号：3427
名称：太乙紫金丹
组成：山慈菇2两，川文蛤2两，红芽大戟1两5钱，白檀香1两5钱，安息香1两5钱，苏合油1两5钱，千金霜1两，雄黄（飞净）5钱，琥珀5钱，冰片5钱，当门子5钱。
出处：《重庆堂随笔》卷上引薛生白。
主治：暑湿温疫之邪，弥漫熏蒸。神明昏乱，及霍乱吐泻，痧胀腹痛，水土不服，岚障中恶。
加减：None
功效：None
用法用量：每服1钱许，滚开水送下。
制备方法：上各为极细末，再合研匀，浓糯米饮为丸，如绿豆大，外以飞净辰砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方比苏合香丸而无热，较至宝丹而不凉，兼太乙丹之解毒，备二方之开闭，洵为济生之仙品。
