序号：79491
名称：獐骨丸
组成：獐骨4两（涂酥炙微黄），桑螵蛸2两（微炒），钟乳粉2两，石斛（去根，锉）1两，肉苁蓉（以酒浸1宿，刮去皱皮，炙干）1两，鹿茸（去毛，涂酥炙微黄）1两，菟丝子（酒浸3日，晒干，别捣为末）1两，龙骨1两，黄耆（锉）1两，五味子1两，牡蛎粉1两，巴戟1两，防风（去芦头）1两，诃黎勒皮1两，附子（炮裂，去皮脐）1两，桂心1两，羚羊角1两。
出处：《圣惠》卷九十八。
主治：男子水脏虚冷，诸有不足。
加减：None
功效：填精补髓。
用法用量：每服30丸，渐加至40丸，空心温酒送下。
制备方法：上药为末，入研了药令匀，炼蜜和捣为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
