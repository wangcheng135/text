序号：556
名称：五痨麝香散
组成：天灵盖2钱半，柴胡1两，犀角屑半两，甘草3寸（患人中指长，男左女右），东引桃枝1握，青蒿1握，东引柳枝1握，石榴皮1握，阿胶7寸，薤白7寸，葱白7寸，麝香2钱半。
出处：《准绳·类方》卷一。
主治：男子妇人传尸痨，骨蒸实热。
加减：None
功效：None
用法用量：上为末，用童便2升半，浸药1宿，明日早、晚煎至1升半，去滓，分为3服，加槟榔末3分，温服。初服约人行3-5里远，便再服1次。倘恶心，以白梅含止之。服3-5次，病止即泻出异物，若虫如头发马尾，身赤口黑，身上如蚁行不可名状，泻后葱粥饮补之，同时药煎补五脏茯神散。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌风1月。忌食油腻、湿面、咸味，并牛、猪、鸡、鸭、犬等物。
附注：None
