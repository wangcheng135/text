序号：6182
名称：加味春雪膏
组成：黄连4两（洗净，用童便2升，浸1宿，去渣用汁，淬芦甘石汁尽，留石为用），方芦甘石12两，好黄丹6两（水飞），乌鱼骨（烧存性）3钱，乳香3钱，当归3钱，白丁香半钱，麝香、轻粉各少许，硇砂1钱（研细，水调盏内，放汤瓶中，候干为度）。
出处：《卫生宝鉴》卷十。
主治：风热上攻眼目，昏暗痒痛，瘾涩难开，多泪疼痛，或生翳膜。
加减：None
功效：None
用法用量：每用1粒，新汲水少许化开，时时点之。
制备方法：上各为末，另裹起，用白砂蜜20两，炼去蜡，下芦甘石末，不住手搅，次下黄丹及诸药末，不住手搅，至紫色不粘手为度，搓作铤子。
临床应用：None
各家论述：None
用药禁忌：忌酒、湿面、猪肉、荞麦。
附注：①本方方名，《玉机微义》引作“春雪膏”，《张氏医通》引作“绛雪膏”。②方中硇砂，《张氏医通》作“硼砂”。
