序号：28439
名称：芫菁酒
组成：芫菁30个，巴豆（去皮心，熬）30个，斑蝥（去翅足，熬）30个，附子（去皮）1两，踯躅1两，细辛1两，乌头（去皮）1两，干姜1两，桂心1两，蜀椒（去目闭口者，汗）1两，天雄（去皮）1两，黄芩1两。
出处：《千金翼》卷十六。
主治：百病风邪狂走，少腹肿，癥瘕，霍乱中恶，飞尸遁注，暴癥伤寒，中风湿冷，头痛身重诸病，寒热风虚及头风。
加减：None
功效：None
用法用量：每服半合，日2次，因苦烦闷，饮1升水解之，以知为度。服酒当从少起，药发当吐清汁1-2升。
制备方法：上切，以酒1斗渍10日。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
