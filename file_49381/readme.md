序号：49396
名称：胡麻膏
组成：胡麻油1升，腊月猪脂1升，乌鸡脂1合，丁香1两半，甘松香1两半，零陵香3两，芎藭2两，竹叶2两，细辛2两，川椒2两（去目），苜宿香3两，白芷1两，泽兰1两，大麻仁1两，桑根白皮1两，辛荑1两，桑寄生1两，牡荆子1两，防风3两（去芦头），杏仁3两（汤浸，去皮尖双仁），莽草1两，柏叶3两。
出处：《圣惠》卷四十一。
主治：None
加减：None
功效：长发，令速生及黑润。
用法用量：净洗头，涂之，1日2次。30日发生。
制备方法：上细锉，米醋浸1宿，滤出，纳入油、猪脂、鸡脂中，以慢火煎，候白芷色焦黄，膏成，绵滤去滓，以瓷盒盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
