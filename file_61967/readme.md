序号：61983
名称：救产益母丸
组成：益母草8两，香附4两（盐、醋、酒、童便制），苍术4两（米泔浸，炒），泽兰叶4两，桃仁4两（去皮尖，麸炒，双仁勿用），延胡4两（酒炒），当归2两，川芎2两，牛膝2两（俱酒炒），炙草2两。
出处：《履霜集》卷二。
主治：产难，横生倒产，胎衣不下，并产后血晕眼花，言语错乱，心闷口干，寒热似疟，四肢浮肿，癫狂不语，泻痢腹疼，大小便闭结，下血如涌，胸膈气满，呕吐不安，咳嗽喉中似蝉声，面黄舌干，鼻中流血，遍身生黑点血斑。
加减：None
功效：None
用法用量：每服1丸，用黄酒送下。若子死腹中，产母腹冷胀疼，上则口角呕沫，下则小便流血，手足冰冷，指甲青黑，急以车前子1钱（去壳，炒香）研末，同丸热酒服之。
制备方法：上为末，大黄膏为丸，每丸晒干重2钱，收用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
