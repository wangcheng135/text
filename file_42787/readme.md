序号：42802
名称：保身丸
组成：党参3两，牡蛎2两，炙黄耆3两，巴戟天4两，当归3两，龙骨2两，甘草1两，杜仲2两，补骨脂2两，续断3两，菟丝子4两，川芎2两，益智仁2两，枸杞子4两，酸枣仁3两，淮牛膝2两，杭白芍2两，远志4两，白术3两，广陈皮1两，云茯苓3两。
出处：《全国中药成药处方集》（武汉方）。
主治：精神疲倦，腰痠肢软，虚烦盗汗，心悸不宁。
加减：None
功效：None
用法用量：每服2钱，白开水送下。
制备方法：小丸：取上药干燥，为细末，明净粉量加炼蜜50～52％，迭成小丸，每钱不得少于25粒。大丸：加炼蜜115～125％，和成大丸，每丸重2钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
