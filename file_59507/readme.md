序号：59522
名称：调卫汤
组成：黄耆3钱（蜜炙），炒白术1钱，人参1钱半，炒苍术1钱，桂枝5分，白芍（酒炒）1钱半，五味子1钱半，炙甘草5分。
出处：《医略六书》卷二十。
主治：卫虚多汗，脉浮软者。
加减：None
功效：None
用法用量：水煎，去滓，温服。
制备方法：None
临床应用：None
各家论述：卫虚气弱，不能卫外而腠理不密，故多汗不止，黄耆补卫气之虚，白术壮脾气之弱，人参扶元益气，苍术燥湿强脾，白芍敛阴，五味收津液，桂枝行阳气于卫，炙草益胃气于中。使气壮卫强，则腠理（纟致）密而汗可自止，安有不能卫外之患乎？此强中益卫之剂，为卫虚自汗之专方。
用药禁忌：None
附注：None
