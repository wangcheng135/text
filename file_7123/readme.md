序号：7123
名称：四圣丹
组成：珍珠3-5粒（犁尖铁器上煿微黄色，研），豌豆49粒（烧灰存性），头发（烧灰存性）。
出处：《丹溪心法附余》卷二十三。
主治：痘疮疔黑紫，头黑烂臭不可闻。
加减：None
功效：None
用法用量：将儿在漫燠处安存，忌风寒秽气，先用簪尖平拨开疔口，将药絍入疔内，即时变为红白色，余疮皆起。又尝见疔痘者，但挑破出其黑血即愈，或挑开用口咂去黑血，或用绵裹指甲掐其黑血展去亦可，盖自疔破而毒气得散也。
制备方法：上为细末，用搽面油胭脂调成膏子。
临床应用：None
各家论述：None
用药禁忌：None
附注：四圣散（《本草纲目》卷二十四）、四圣膏（《赤水玄珠》卷二十八）、四宝丹（《疡医大全》卷三十三）。
