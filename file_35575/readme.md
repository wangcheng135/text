序号：35585
名称：侧子散
组成：侧子1两（炮裂，去皮脐），桂心1两，汉防己1两，附子1两（炮裂，去皮脐），芎1两，人参1两（去芦头），麻黄1两（去根节），当归1两，赤芍药1两，秦艽3分（去苗），茯神2两，防风3分（去芦头），白术半两，细辛半两，甘菊花1两，甘草半两（炙微赤，锉）。
出处：《圣惠》卷六十九。
主治：妇人中风，偏枯一边，手足不遂，口面斜，精神不守，言语倒错。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，煎至7分，去滓，入竹沥半合，更煎1-2沸，不拘时候温服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
