序号：77305
名称：解肝煎
组成：陈皮1钱半，半夏1钱半，厚朴1钱半，茯苓1钱半，苏叶1钱，芍药1钱，砂仁7分。
出处：《景岳全书》卷五十一。
主治：肝郁气滞之胸胁胀满疼痛，泄泻，胎动不安。
加减：如胁肋胀痛，加白芥子1钱；如胸膈气滞，加枳壳、香附、藿香之属。
功效：None
用法用量：水1钟半，加生姜3-5片，煎服。
制备方法：None
临床应用：None
各家论述：《谦斋医学讲稿》：本方名为解肝，实际上除白芍养肝、苏叶兼能芳香舒气外，均属化湿行滞，调理脾胃之品，适应于土壅木郁的证候。因脾胃湿阻气滞，影响肝气条达，必须着重中焦治本，故方中不用柴胡疏肝而用苏叶，取其能舒肝郁，亦能和脾胃，脾胃健运则肝气自畅。故解肝的意义在于解肝之围，而不是直接治肝。
用药禁忌：None
附注：解恨煎（《笔花医镜》卷四）。《不知医必要》有藿香，无半夏。
