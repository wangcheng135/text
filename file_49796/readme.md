序号：49811
名称：大干枣三味丸
组成：大枣60枚（擘，去核），葶苈子1升（熬），杏仁1升（去皮尖、两仁者，熬）。
出处：《外台》卷九引《许仁则方》。
主治：饮气嗽，经久不已，渐成水病，大小便秘涩，头面身体浮肿。
加减：None
功效：None
用法用量：依前以桑白皮饮下之，初服7-8丸，日再服。稍稍加之，以大便通为度。病重者，时令鸭溏佳。亦有以前3味煮汤服之。
制备方法：上药合捣令如膏，可作丸。如硬燥不相著，细细下蜜作丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：枣杏丸（《朱氏集验方》卷五）。饮气嗽，已服细辛八味汤、葶苈子十五味丸，不觉可，见证如上者，服此方。
