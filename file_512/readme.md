序号：512
名称：五停五积丸
组成：甲先服方：使君子肉5枚，巴豆肉2枚半；乙接服方：老松香8斤，黄连2两，潮脑2两，朱砂（水飞）2两。
出处：《红蓼山馆经效方》。
主治：远年近日丹停(人面青黄，肚腹胀痛，小便不利短少)、水停(面黄浮肿胀痛)、酒停、食停(心胸胀或干呕)、气停(胸前不利)、妇人瘀血停(面黄青肿，月经不调)；妇女血疱、血块，气裹食积，干病潮热，或经水不通；及男妇血积、气积、酒积、食积、虫积膨胀。
加减：None
功效：None
用法用量：甲方：共捣成丸1枚，以此作为1剂，用红糖开水空腹送下。乙方：共为末，以开水调和成丸，如绿豆大。每服5粒，用红糖开水空腹送下。服此药后约1-2小时即泻下，或2-3次，或6-7次不等，随以酸筋草煎浓水1碗服之，以止其泻；1-2日后再以猪蹄1对，炖桐子根、通花根、臭草根、棬子根、打碗子根、百节藕、见肿消等草药，稍入盐服之。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌食糯米、菜油1月，并忌房事4月，病即除根。
附注：None
