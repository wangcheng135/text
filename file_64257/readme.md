序号：64273
名称：大腹汤
组成：大腹皮（锉）1两半，紫苏茎叶1两，干木瓜1两，桑根白皮（锉）1两，沉香（锉）半两，木香半两，茴香子根（切，焙）半两，羌活（去芦头）半两，木通（锉）半两，枳壳（麸炒，去瓤）半两，青橘皮（汤浸，去白，焙）半两，陈橘皮（汤浸，去白，焙）半两，槟榔（锉）半两，莱菔子（焙）半两。
出处：《圣济总录》卷八十三。
主治：风毒脚气上攻，头目昏眩时痛，脚膝痹弱，不能履地；或时发寒热，呕吐痰涎。
加减：None
功效：None
用法用量：每服2钱匕，水1盏，加葱白3寸（切），生姜3片，煎至6分，早、晚食后服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
