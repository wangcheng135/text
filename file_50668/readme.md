序号：50683
名称：荆芥四物汤
组成：生地5钱，荆芥1钱半（炒黑），白芍1钱半（炒），当归3钱（醋炒），条芩1钱半（炒黑），川芎1钱，香附1钱半（童便制），地榆2钱（炒黑）。
出处：《医略六书》卷二十六。
主治：风热伤于冲任经气，洋溢不能摄血，而血不归经，故经漏不止，脉浮数者。
加减：None
功效：None
用法用量：水煎，去滓温服。
制备方法：None
临床应用：None
各家论述：方中生地滋阴凉血，当归养血归经，川芎入血海以升阳，白芍敛营阴以止血，条芩清在里风热，荆芥散血中风邪，香附调气解郁，地榆凉血涩血。水煎温服，使风热外解，则经气清和而血无妄行之患，何经漏之不止哉。
用药禁忌：None
附注：None
