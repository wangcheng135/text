序号：71782
名称：散聚汤
组成：槟榔1钱（磨冲），半夏1钱半，桂心1钱，茯苓1钱半，橘红1钱半，当归2钱，杏仁3钱（去皮），甘草5分。
出处：《医略六书》卷二十三。
主治：瘕聚随气上下，脉弦。
加减：None
功效：None
用法用量：水煎，去滓温服。
制备方法：None
临床应用：None
各家论述：痰饮洋溢，随气升降，或上或下，而成瘕聚聂聂移动不止焉。半夏消痰涤饮，橘红利气除痰，茯苓渗水饮以和脾，炙草缓中气以益胃，桂心温经平肝气，槟榔逐饮破滞结，当归和血脉，杏仁降逆气也。水煎温服，使痰化饮消，则肝脾调和而癥瘕自散，何聂聂移动之有？此涤痰逐饮之剂，为瘕聚随气上下之专方。
用药禁忌：None
附注：None
