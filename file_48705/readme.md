序号：48720
名称：土七
组成：桃仁、赤芍、泽兰、玄胡索、红花、陈皮、乌药、独活。
出处：方出《痧胀玉衡》卷中，名见《痧症全书》卷下。
主治：内伤兼痧。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：内伤兼痧：曹洪宇子之外戚，争夺家产，讼公庭，有老妇造其家，互相争殴，发热沉重，咳嗽吐痰，胸中胀闷，诸亲戚唯恐毙于曹姓室中，邀余往视，余为诊之。知其内伤兼痧症也，刺痧筋20余针，付宝花散微温服之，胀闷稍松。又用上方治其内伤，服后下黑粪，瘀血俱消，诸症俱愈。
各家论述：None
用药禁忌：None
附注：四十七号讼象方（《杂病源流犀烛》卷二十一）。《痧症全书》有丹参。
