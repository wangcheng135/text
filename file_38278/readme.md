序号：38288
名称：林檎散
组成：麻黄（去节）1两半，肉桂（去粗皮）1两半，苍术（去皮）1两半，川大黄1两半，干葛1两半，石膏1两半，山栀子（去皮）1两半，木通1两，瞿麦1两，甘草（炙）1两，前胡1两，川芎1两，藿香（用叶）半两，川乌头（炮，去皮脐）半两。
出处：《局方》卷二。
主治：伤寒及时行疫疠，头痛项强，壮热恶寒，腰背四肢拘急烦疼，面赤咽干，呕逆烦渴。
加减：None
功效：None
用法用量：每服2钱，水1盏，入林檎糁十数片，新者亦得，煎至7分，去滓稍热服，不拘时候。相次再服，衣被盖覆，汗出为度。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：《鸡峰》有桔梗、白术，无苍术、前胡、川芎。
