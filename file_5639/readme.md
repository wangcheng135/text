序号：5639
名称：去三虫方
组成：生地黄汁3斗，清漆2升，真丹3两，瓜子末3升，大黄末3两。
出处：《千金》卷二十七。
主治：诸虫。
加减：None
功效：None
用法用量：先食服1丸，日3次。浊血下鼻中，30日诸虫皆下；50日百病愈，面色有光泽。
制备方法：东向灶苇火煎生地黄汁3沸，纳清漆。以荆匕搅之，日移1尺；纳真丹，复移1尺；纳瓜子末，复移1尺，纳大黄末，微火勿令焦，侯之可丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：首取漆之生者，搜逐脏腑之积荫，直捣三虫之窟穴，且与生地黄汁同煎，解其粘着之性，可无留积之虞；兼得铅丹、大黄、冬瓜子末，共襄厥功，一入虫口，则虫皆化为水，从魄门而下矣。详《本经》于漆条下原有去长虫之治。长虫者，九虫之长，是指三尸而言。以于者性缓，故专取生者之力锐；地黄亦须鲜者，方能逐血、能祛虫，若干地黄已经火焙，反资滞血，功用天渊；更取铅丹镇肝杀虫，除热下气；冬瓜子专涤肠垢，压丹石毒；大黄推陈致新，安和五脏，而祛之下夺。
用药禁忌：None
附注：三尸虫丸（《医方类聚》卷二○二引《得效》）。本方方名，《医方类聚》引作“去三虫丸”。
