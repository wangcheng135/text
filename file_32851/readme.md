序号：32860
名称：羌活汤
组成：羌活（去芦头）2两，桑根白皮2两，麻黄（去根节）2两，天雄（炮裂，去皮脐）2两，当归（切，焙）2两，桂（去粗皮）1两，旋覆花（微炒）1两，远志（去心）1两，大腹皮（锉）1两半，甘草（炙，锉）1两半，芎1两半，威灵仙（去苗土）1两半，枳壳（去瓤，麸炒）1两半，菖蒲1两半，杏仁（汤浸，去皮尖双仁，炒）21枚。
出处：《圣济总录》卷六。
主治：卒中风，闷乱，语言謇涩，牙关紧急。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，加生姜3片，煎至8分，去滓温服，不拘时候。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
