序号：55988
名称：桑根白皮散
组成：桑根白皮1两（锉），柴胡1两（去苗），大腹皮3分（锉），枳壳3分（麸炒微黄，去瓤），杏仁1两（汤浸，去皮尖双仁，麸炒微黄），赤芍药1两，赤茯苓1两，黄耆1两（锉），陈橘皮3分（汤浸，去白瓤，焙），麦门冬3分（去心），牛蒡子1两（微炒），甘草3分（炙微赤，锉）。
出处：《圣惠》卷四十六。
主治：咳嗽，面目浮肿，或四肢肿，气促不得眠卧。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，入生姜半分，煎至6分，去滓，温服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：桑根白皮汤（《圣济总录》卷六十六）。
