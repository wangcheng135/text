序号：40561
名称：虎骨膏
组成：生虎骨4两，石斛2两，赤芍1两5钱，白及1两，川芎1两，羌活1两5钱，桂枝2两，生杜仲1两5钱，生地4两，生川乌1两，白蔹1两，生山甲1两，独活1两5钱，麻黄1两，透骨草2两，当归4两，生草乌1两，红花1两，大黄1两，防风1两5钱，甘草1两。
出处：《全国中药成药处方集》（兰州方）。
主治：筋骨疼痛，四肢麻木，跌打损伤，闪腰岔气。
加减：None
功效：祛风散寒，舒筋活血，止痛。
用法用量：贴患处。
制备方法：以上药料用香油15斤炸枯，去滓滤净，炼至滴水成珠，再入章丹90两，搅匀成膏。每膏药油15斤兑：肉桂面1两5钱，乳香面1两，没药面1两，麝香5分，血竭面1两5钱，广木香面5钱，公丁香面5钱，搅匀。每大张净油1两，小张净油5钱。
临床应用：None
各家论述：None
用药禁忌：孕妇忌贴腹部。
附注：None
