序号：55562
名称：桃仁散
组成：桃仁1两（汤浸，去皮尖双仁，麸炒微黄），诃黎勒2两（煨，用皮），桂心1两，半夏1两（汤洗7遍去滑），五味子1两，槟榔1两，木香1两，赤芍药1两，鳖甲2两（涂醋，炙令黄，去裙襕），人参1两（去芦头），陈橘皮1两（汤浸，去白瓤，焙），白术1两，柴胡1两半（去苗），京三棱1两（煨，锉），郁李仁1两（汤浸，去皮尖双仁）。
出处：《圣惠》卷二十六。
主治：肺劳上气，胸满，痰唾不利，右胁有积聚，发歇寒热，肌体羸瘦，食少无力。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，煎至6分，去滓温服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
