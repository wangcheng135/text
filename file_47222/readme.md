序号：47237
名称：癸字化毒丸
组成：牛黄5分，鹿角屑3钱，沉香1钱，生生乳1钱，朱砂1钱半，雄黄1钱半，月月红1钱半，白鲜皮1钱半，乳香1钱半，穿山甲1钱半，神水1钱（用出山铅10斤打薄片20块，块上贴银箔，取尖底缸2只一样的，上缸开1孔，底中绳穿铅片悬上缸，下缸盛米醋、火酒各10隅，缸口架瓷盆1个，将缸合好，用面条封固，以文火下烧，俟酒醋干，取出盆中者是），人中白2钱5分（择乡间诚实人家，不生疮毒疾病者，取制入药有效），制何首乌3钱。
出处：《疮疡经验全书》卷六。
主治：毒结于膀胱并肾经，内作骨痛流注，上下抽掣，时痛发块，百会、委中、涌泉等穴，或阳物腐烂不已，或阴囊肿胀作溃，或生独脚杨霉疮，或传他经，致生别病。
加减：None
功效：None
用法用量：每早空心服15丸，晚空腹服9丸，人参汤送下；枸杞汤亦可。
制备方法：上为末，用神曲末5钱，打稠糊，入药捣均为丸，如梧桐子大，另研朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：病去药减，如余毒未尽，药不可撤，百日内勿使大劳大怒，顺时调理。
附注：None
