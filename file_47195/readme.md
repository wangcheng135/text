序号：47210
名称：疥疮膏
组成：麻黄1两，川椒1两，白芷1两，蛇床子1两，大风子3两，胡桃肉3两，斑蝥5钱，升华硫黄1两，轻粉3钱，信石2钱，煅枯矾1两，明雄1两，樟脑1两，黄蜡1两。
出处：《全国中药成药处方集》（武汉方）。
主治：疥疮。
加减：None
功效：None
用法用量：涂擦患处，视疮之大小酌用。
制备方法：先取麻黄、白芷、大风子、斑蝥、川椒、蛇床子、核桃仁等药用麻油1斤炸枯，去滓，加猪油1斤，黄蜡1两；再将轻粉、信石、枯矾、雄黄、硫黄、樟脑等细末兑入搅匀，成膏34两6钱，每盒重2钱。
临床应用：None
各家论述：None
用药禁忌：切勿入口，头面部及阴部禁用。
附注：None
