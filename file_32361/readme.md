序号：32370
名称：妙香丸
组成：辰砂2两（水飞过），巴豆150粒（肥好者，去心皮膜，不出油，研如面油），生龙脑1分，麝香1分，轻粉1分，大金箔35片，真牛黄半分，犀角1分。
出处：《传家秘宝》卷下。
主治：风热潮热，搐搦，伤寒时疾，阴阳气交结，伏毒气胃中，喘躁眼赤，潮发不定，一切惊热烦赤，睡卧不宁，泄泻积食。
加减：None
功效：None
用法用量：小儿每服1大丸，可分至10小丸，每服3-5丸，金银汤送下；或有惊风积滞，痰涎等，以生龙脑少许，轻粉1钱，研匀，用金银花汤送下5-7丸；如伤寒时疾，阴阳气交结，伏毒气胃中，喘躁眼赤，潮发不定，再经日数7、8日已下至半月日未安，医不能明其证候，脉息交乱者，可服1丸，如丸大难咽可分作3丸，用龙脑、腻粉、米饮调半盏送下。此1服，取转下一切恶毒涎，并药丸泻下，如要却收，水洗净，用朱砂末、龙脑、麝香内收之，可再与服。
制备方法：上药各为极细末，再一处同研令匀后，用上好黄蜡一两，溶化去脚，只取清者，放瓷器中和上件药，以竹筷子搅匀，再熔温化，令药匀，温软可丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
