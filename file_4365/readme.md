序号：4365
名称：九转大降丹
组成：白马（即火消）1两，黑铅2钱，青盐3钱，胆矾4钱，雄精5钱，明矾1两，大红硇7分，汞（即水银）1两，辰砂（即朱砂，如豆色紫佳）9分。
出处：《外科图说》卷一。
主治：痈疽、发背、疔毒、恶疮、蛇伤、犬咬。
加减：None
功效：None
用法用量：每用药厘许，以津唾调点毒上，以膏药盖之，毒气收聚顶上，自然结成黑肉1块，3-4日即脱落矣。
制备方法：先将上药9味，除水银外，都研细，放于阳城罐内，然后放下水银，上则放1小砂盆，缝处先用桑皮纸搓软，嵌于缝道内；又将光粉打潮湿，用以封口，极要留心，不可走气，外又将桑皮纸满糊。放于八卦炉中，然后举火，合五九四根半香为度，结胎时炷香3根6寸，合四九之数，用文火，谓之结胎。只须3小块，中匀炭火，不可旺也。炭火当设1总炉，以便换炭，四九之香数毕，炉中取出冷定，其胎自成。切忌开看，恐伤封口出气，即走炉也。后乃另取一小脚缸，注入水，只须齐小砂盆口下，再将预备铁丝做就的载火器套在阳城罐底，约露底1寸8分，然后取总火炉内旺炭，攒围罐底，随少随添，用扇搧拂；又炷9寸香，降乃毕矣。取出，择一净室中冷定，次日开看，砂盆内所有之霜，即大降也。另将瓷瓶收贮待用，愈陈愈妙。
