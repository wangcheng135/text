序号：56625
名称：消内灵砂丹
组成：黄蜡1两，巴豆14枚（去壳，作2片，入黄蜡内，熬巴豆黑为度）。
出处：《普济方》卷二○七。
主治：泻痢。
加减：None
功效：去积滞。
用法用量：小儿3岁，只服10丸，如黍米大，大人服10丸，如大麻子大，白痢，白姜汤送下；赤痢，甘草汤送下，空心服。如病人身体凉，脉微弦细，皆是好症，便用此药止。如病人身上有冷热，初不可便止，先服前药10丸，次服五苓散加车前同煎，3服小便清，退尽热，方可再用前药止住。
制备方法：上药去巴豆，将硇砂2钱，用水煮干，只用1钱重，仍入黄蜡，并再熬，就入朱砂末1-2钱，以黄蜡红为度，候冷刮下，如用时旋丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
