序号：55616
名称：桃奴丸
组成：桃果7枚（别为末），桃仁14枚（去皮尖，炒，别研），安息香1两（以无灰酒斟酌多少，研，飞去砂石，银器中入上2味，慢火熬成膏），生玳瑁（镑过，杵为细末）1两，琥珀3分（别研），雄黄（用桃叶煮，水研飞）3分，辰砂（研飞）半两，黑犀（石上以水磨，澄去水，取末）半两，脑1分，麝1分（别研）。
出处：《幼幼新书》卷十二引《养生必用》。
主治：小儿心气虚，有热，恍惚不常，言语错乱，尸疰客忤，魇梦不祥，惊痫。
加减：None
功效：None
用法用量：每服1丸，食后、临卧以人参汤送下。
制备方法：上为细末，以前膏为丸，如鸡头大，阴干，密器封，安静室。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
