序号：58875
名称：秘真丸
组成：龙骨（研）1两，诃梨勒（炮，取皮）5枚，缩砂仁（去皮）半两，丹砂（研）1两，（留1分为衣）。
出处：《圣济总录》卷九十二。
主治：心肾两虚，阳气衰微，精气不固，白淫不止，梦遗，小便不禁及有余沥，脉短涩。_x000D_
内虚里寒，冷气攻心，胁肋胀满，脐腹刺痛，呕逆泄泻，自汗时出，手足厥，久虚下冷，其气不足。精不禁，危急者。
加减：None
功效：温中助阳，固精止尿。
用法用量：每日空心热酒送下1丸，夜卧冷水送下3丸；或太秘欲通，用葱汤点茶服之。
制备方法：上为1末，煮糯米粥为丸，如绿豆大，以丹砂为衣。
临床应用：None
各家论述：《医略六书》：龙骨固涩肾气，收束浮弱之脉；灵砂镇坠心神，降抑虚浮之气；砂仁炒黑归肾，调其蓄泄之权；诃子炒黄收脱，兜其遗失之溺也。覆盆膏丸，人参汤下，使真气布护，则心肾相交，而脬气自固，安有遗尿之患乎。此镇坠固涩之剂，为心肾虚浮遗溺之专方。
用药禁忌：不可多服。
附注：秘精丸（《百一》卷十五）、秘元丹（《御药院方》卷六）、秘精丹（《普济方》卷二一七）、秘真丹（《证治汇补》卷八）。《医略六书》本方用法：覆盆膏为丸。每服三钱，参汤送下。
