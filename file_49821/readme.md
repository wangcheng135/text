序号：49836
名称：大五膈丸
组成：细辛3分，桂心3分，黄芩3分，食茱萸3分，厚朴（炙）3分，杏仁30枚（去尖），干姜3分，川椒（汗）3分，远志（去心）3分，小草2分，芍药2分，附子（炮）2分，当归2分，黄连2分。
出处：《外台》卷八引《古今录验》。
主治：膈中游气，上下无常处，脏有虚冷，气迫咽喉，胸满气逆，胁有邪气，食已气满，羸瘦著床骨立，往来寒热，腹中不调，或下痢呕逆咳嗽，骨肉销尽。
加减：None
功效：令人能食，长肌肉，强筋骨，利五脏，好颜色，补不足，益气力。
用法用量：每服2丸，1日3次。不知加之，以知为度。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌猪肉、冷水、生葱、菜等。
附注：None
