序号：46239
名称：活水无忧散
组成：益母草2两，急性子（即金凤子）4钱，当归4钱，陈枳壳1两，生地黄2钱，白芍药2钱，苏叶2钱，甘草8分，肉桂1钱，川芎1钱，陈艾1钱，生鲤鱼1个。
出处：《女科万金方》。
主治：产妇横产，胎死腹中。
加减：None
功效：None
用法用量：每服用水3碗，煎至2碗，临服之时加入好醋1匙，每1碗和调乌金丸1丸服。如其死胎不落，急取无根水煎药滓，连服2服，救其性命。
制备方法：上为散，分作2服。
临床应用：None
各家论述：None
用药禁忌：None
附注：活水推生无忧散（《郑氏家传女科万金方》卷二）、活命无忧散（《女科旨要》卷二）。《郑氏家传文科万金方》有秦艽、陈皮。
