序号：82971
名称：丹砂膏
组成：丹砂末2两，雄黄末2两，附子2两，天雄2两，干地黄2两，大黄2两，当归2两，秦胶2两，乌头4两，桂心4两，黄连4两，松脂4两，茵芋4两，蜀椒1斤（去目，汗），干姜2两，巴豆100枚（去皮心），蜈蚣4枚（去头足，赤者），石南草2两。
出处：《鬼遗》卷五。
主治：瘑疽、诸恶疮，经年不瘥。
加减：None
功效：None
用法用量：敷，疮有口，亦可兑，此脂多治合，即随多少，苦酒不必尽1斗，以意量用之。敷当火，须以意度之。
制备方法：上(口父)咀16味，以苦酒1斗，渍1夜，以猪脂6升，微火煎三上下，药色膏成，绞去滓，纳2石末，搅令调。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
