序号：635
名称：五膈散
组成：枳壳（去瓤，麸炒）1两，木香（不见火）1两，青皮（去白）1两，大腹子1两，白术1两，半夏曲（锉，炒）1两，丁香（不见火）1两，天南星（汤泡去皮）1两，干姜（炮）1两，麦芽（炒）1两，草果仁1两，甘草（炙）半两。
出处：《济生》卷二。
主治：五膈，胸膈痞闷，诸气结聚，胁肋胀满，痰逆恶心，不进饮食。
加减：None
功效：None
用法用量：每服2钱，水1中盏，加生姜5片，煎至6分，温服，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：五噎散(《医统》卷二十七)、五膈汤(《医学入门》卷七)。方中大腹子，《医学入门》作“大腹皮”；木香原脱，据《医方类聚》补。
