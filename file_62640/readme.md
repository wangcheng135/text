序号：62656
名称：清六丸
组成：六一散1料，红曲（炒）半两。
出处：《丹溪心法》卷二。
主治：泄泻，产后腹痛或自利、血痢。
加减：None
功效：去三焦湿热，补脾补血。
用法用量：每服50-70丸，白汤送下。治泄泻，与清化丸同用。
制备方法：上为末，饭为丸，如梧桐子大。
临床应用：None
各家论述：1.《医方考》：血痢者，此方主之。滑石能清六腑之热，甘草能调六腑之气，红曲能和六腑之血。_x000D_
2.《赤水玄珠》：此方专清六腑湿热，故名清六。内用红曲者，以其能消食积而化瘀血也。
用药禁忌：None
附注：本方方名，《医学纲目》引作“青六丸”。改为散剂，名“青六散”（见《金鉴》）。
