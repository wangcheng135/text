序号：7297
名称：四汁膏
组成：黄连末、天花粉末、人乳汁（又云牛乳）、藕汁、生地黄汁。
出处：方出《丹溪心法》卷三，名见《医统》卷五十二。
主治：三消。
加减：能食者，加软石膏、瓜蒌根。
功效：养肺降火生血。
用法用量：徐徐留舌上，以白汤少许送下。
制备方法：以后2味汁为膏，入前3味搜和，佐以姜汁和蜜为膏。
临床应用：None
各家论述：《医林纂要》：天花粉甘酸微苦，补肺敛气，泄逆宁心，此治上焦气分之热；黄连苦以泻火，此治中焦心脾血分之热；黄连亦入血分，且能厚肠胃；生地黄汁苦寒而浊，以治下焦肾命之火，清其本也；藕汁甘咸涩，敛阴散热，交心肾，济水火；牛乳甘咸，润肠胃，解热结，滋阴血，而引之上行；蜂蜜润燥去热，通利三焦，加姜汁为反佐，以行之。
用药禁忌：None
附注：乳汁膏（《杏苑》卷五）、藕汁膏（《东医宝鉴·杂病篇》卷六）、人乳膏（《明医指掌》卷七）、连花乳散（《医林纂要》卷四）。《医林纂要》本方用法：研黄连、天花粉为末，以二汁及乳调之，加姜汁、蜂蜜少许和服。
