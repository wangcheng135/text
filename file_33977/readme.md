序号：33986
名称：皂荚丸
组成：皂荚20梃。
出处：《圣惠》卷二十四。
主治：大风疾。
加减：None
功效：宣泻。
用法用量：每服20丸，空心以温酒送下。得利后，方可别服治大风丸散。
制备方法：上药以10梃去黑皮，涂酥炙令黄焦，去子，捣罗为末；10梃去皮子，捶碎，以水5升，煎碎皂荚至1升后，以生布裹，挼滤去滓，重煎成膏，和入皂角末，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：皂角丸（《直指》卷二十四）。《直指》本方用法：将二十条皂角先炙透，去皮弦核，多用酒，慢火煎稠粘，滤出清稠者，候冷，入雪糕，杵为丸，如梧桐子大。每服五十丸，不饥不饱时，用酒送下。
