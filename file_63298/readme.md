序号：63314
名称：清金解毒汤
组成：生明乳香3钱，生明没药3钱，粉甘草3钱，生黄耆3钱，玄参3钱，沙参3钱，牛蒡子3钱（炒捣），贝母3钱，知母3钱，三七2钱（捣细，药汁送服）。
出处：《衷中参西》上册。
主治：肺脏烂损，将成肺痈，或咳嗽吐脓，血，又兼治肺结核。
加减：将成肺痈者，去黄耆，加金银花3钱。
功效：None
用法用量：None
制备方法：None
临床应用：肺结核：一人，年48，咳吐痰涎甚腥臭，夜间出汗，日形羸弱，医者言不可治，求愚诊视。脉数至六至，按之无力，投以此汤，加生龙骨6钱，又将方中知母加倍，两剂汗止，又服10剂痊愈。

用药禁忌：None
