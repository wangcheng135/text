序号：3699
名称：开关散
组成：牙皂3钱半，细辛3钱半，明雄2钱半，法夏3钱，广木香3钱，陈皮2钱，藿香2钱，桔梗2钱，薄荷2钱，贯众2钱，白芷2钱，防风2钱，甘草2钱，枯矾5分。
出处：《验方新编》卷十一。
主治：朱砂症或感冒风寒及各种痧症，脉散牙紧发慌，手足麻木，闭目不语，喉肿心痛。
加减：None
功效：None
用法用量：朱砂症用此药3分，先吹入鼻孔内；再将药称足1钱，姜汤冲服；服后用红纸捻照心窝、背心2处，见有红点发现，用针挑破，内有红筋挑出，方保无事。
制备方法：上为细末，瓷瓶收贮，用蜡封口，不可泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
