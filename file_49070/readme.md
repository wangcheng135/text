序号：49085
名称：胎产金丹
组成：当归2两，茯苓2两，人参2两，白术2两，白薇2两，炒杜仲2两，蕲艾2两，藁本2两，赤石脂2两，川芎2两，川断3两，生地4两，阿胶4两，香附4两，条芩1两5钱，沉香6钱，甘草1两，五味子1两，炒杭芍2两，没药1两2钱，菟丝子4两。
出处：《全国中药成药处方集》（济南方）。
主治：妇女红白淋带，月经不调，腰腹作痛，习惯性流产，胎前产后诸症。
加减：None
功效：None
用法用量：每服1-2丸，白开水送下。
制备方法：鲜河车一具，用竹签挑去筋膜，洗净，用无灰酒煮烂，黄柏2两放在锅底，将前药共捣如泥，晒干，共为细粉，炼蜜为丸，重2钱，朱砂为衣，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
