序号：53341
名称：健脾饮
组成：厚朴（去粗皮，锉碎，每1斤用生姜1斤，切薄片，烂杵，拌匀，酿1宿，慢火炒干，再炒热，用酸醋淬透，仍以慢火炒干）1两，人参（去芦）1两，白茯苓（去皮）2钱半，肉豆蔻2钱半，半夏（制）2钱半，益智仁2钱半，净香附2钱半，良姜（锉片，东壁土炒）2钱，诃子肉2钱，甘草（炙）5钱。
出处：《活幼新书》卷下。
主治：小儿泻利，呕吐，及诸病后气血虚弱，有痰恶心，腹中微痛，饮食减，精神慢。
加减：None
功效：健脾养胃。
用法用量：上（口父）咀。每服2钱，水1盏，加生姜2片，大枣1枚，煎7分，空心温服，不拘时候。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
