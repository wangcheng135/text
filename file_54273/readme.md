序号：54288
名称：大败毒膏
组成：大黄10两，公英20两，橘皮8两，木鳖子2两，银花2两，黄柏10两，乳香（炙）2两，白芷6两，花粉6两，赤芍10两，当归2两，甘草2两，蛇蜕5钱，干蟾（烧）10个，蜈蚣20条，全蝎3钱。
出处：《北京市中药成方选集》。
主治：痈疽疮疡，坚硬不消，鱼口便毒，杨梅疥疮。皮肤结节性痒疹等。
加减：None
功效：消肿败毒止痛。
用法用量：上切，水煎3次，分次过滤去滓，滤液合并，用文火煎熬，浓缩至膏状，以不渗纸为度，兑芒消10两；每16两汁，兑炼蜜24两成膏；装瓶，每瓶重2两。每服5钱，1日2次，开水调服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：孕妇勿服。
附注：None
