序号：37509
名称：承泽丸
组成：梅核仁1升，辛夷1升，葛上亭长7枚，泽兰子5合，溲疏2两，藁本1两。
出处：《千金》卷二。
主治：妇人下焦三十六疾，不孕绝产。
加减：若腹中无坚癖积聚者，去亭长，加通草1两。
功效：None
用法用量：食前服如大豆2丸，1日3次，不知稍增。恶甘者，和药先以苦酒搜散，乃纳少蜜和为丸。
制备方法：上为末，炼蜜为丸。
临床应用：None
各家论述：《千金方衍义》：承泽丸专破子脏积血。子脏属冲脉，紧附厥阴而主风木。故取梅仁之酸平以泄厥阴风热，则亭长方得振破血之威；辛夷、藁本、溲（巟）三味，《本经》一治寒热风头脑痛，一主妇人阴中寒肿痛，一止遗溺利水道；更用泽兰子统理妇人三十六病，一举而内外风气悉除，胞户积血尽扫。
用药禁忌：None
附注：None
