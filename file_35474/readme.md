序号：35484
名称：三分散
组成：白术1两，茯苓1两，黄耆1两，川芎1两，芍药1两，熟地黄1两，当归1两，柴胡1两6钱，黄芩6钱，人参1两6钱，半夏6钱，甘草6钱。
出处：《保命集》卷下。
主治：产后日久虚劳，针灸、服药俱不效者；产后伤寒并痢。
加减：None
功效：None
用法用量：每服1两，水1盏，煎至半盏，温服清，每日1服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：三圣散（《丹溪心法附余》卷二十一引《济生》）、三合散（《医学纲目》卷三十五）。《奇效良方》有生姜三片，红枣一枚。本方方名，《东医宝鉴·杂病篇》引作“三_x000D_
合汤”。
