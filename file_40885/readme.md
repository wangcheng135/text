序号：40900
名称：郁金丸
组成：白芥子（炒）24两，枳壳（炒）24两，青皮（炒）24两，黄芩24两，白豆蔻24两，黄连24两，莱菔子（炒）24两，厚朴（炙）24两，片姜黄24两，槟榔24两，三棱（炒）24两，橘皮24两，当归24两，黄柏24两，木香24两，砂仁24两，郁李仁24两，郁金24两，熟大黄160两，生大黄160两，黑丑（炒）80两，牙皂40两，香附（炙）40两，灵脂（炒）40两，玄胡（炙）40两，沉香12两，莪术（炙）44两，桔梗44两。
出处：《北京市中药成方选集》。
主治：胸膈堵闷，胃口疼痛，积聚痞块，二便不通。
加减：None
功效：舒郁宽中，消食化滞。
用法用量：每服2钱，温开水送下，1日2次。
制备方法：上为细末，用冷开水为丸。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
