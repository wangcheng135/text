序号：63937
名称：清解汤
组成：黄芩1钱，生甘草4分，升麻3分，柴胡3分，紫草7分，川芎7分，麦门冬7分，荆芥7分，防风7分，黄柏7分，黄连7分，知母7分，牛蒡子6分（炒），蝉退6分，元参6分，山栀6分，桔梗6分。
出处：《医部全录》卷四九○引《沈氏心传》。
主治：痘疹。
加减：色赤稠密，不食，加地丁、金银花、地骨皮、灯心；渴，加葛根；小便白，去栀子、灯心、竹叶；痒，加防风、荆芥、羌活、蝉退、连翘；咳嗽咽痛，加山豆根，倍加麦门冬、牛蒡子；呕，加石膏；腹胀，加紫苏；泻，去麦冬、知母、紫草，切不可加诃子、豆蔻；出汗，去升麻、川芎、柴胡、荆芥、防风、连翘、羌活，切不可加黄耆。
功效：None
用法用量：上以水2钟，用竹叶数片，煎至1钟，陆续服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
