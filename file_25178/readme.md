序号：25186
名称：朱砂丸
组成：朱砂1分（细研），麝香半两（细研），砒霜1分（细研），恒山1分（锉），虎头骨1分（涂酥炙，令黄），猢狲头骨1分（炙黄），虎晴1对（酒漫，微炙），乳香1分，安息香1分，阿魏1分，巴豆3枚（去皮心及油），雄黄半两（细研）。
出处：《圣惠》卷五十二。
主治：寒疟，但寒不热，四肢鼓颤不止。
加减：None
功效：None
用法用量：用绯帛裹，男左女右，手把1丸；如痰疟，即以醋汤送下1丸。每把者药1丸可疗3人；若女人用者．即可疗2人。
制备方法：上为末，端五午修合，用醋煮面糊为丸，如豌豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
