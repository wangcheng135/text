序号：46861
名称：独参汤
组成：大人参2两（去芦）。
出处：《医方类聚》卷一五○引《劳证十药神书》。
主治：大汗大下之后，及吐血、血崩、血晕诸症。
加减：None
功效：劳证止血后，用此药补之。
用法用量：上（口父）咀。以水2盏，加大枣5枚，煎至1盏，细呷之。服后熟睡一觉，后服诸药除根。
制备方法：None
临床应用：None
各家论述：陈修园：失血之后，脏阴太虚，阴虚则不能维阳，阳亦随脱，故用人参二两，任专力大，可以顷刻奏功。但人参虽有补虚之功，而咳嗽者忌之。乘此大血甫止之际，咳嗽未作，急急饮之。若得熟睡一夜，则血从心脏而生。
用药禁忌：咳嗽去之。
附注：人参汤（《十药神书》周杨俊注）。
