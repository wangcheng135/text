序号：5397
名称：加减大补元煎
组成：党参3钱，口耆3钱（炙），当归身5钱（炒），熟地5钱，桂心1钱（冲），附片1钱（制），羌活2钱，山萸肉3钱（炒），乌梅肉3个，杜仲2钱（炒），枸杞2钱（炒）。
出处：《医门八法》卷三。
主治：虚寒腰痛，身痛。
加减：None
功效：None
用法用量：水煎服。或以煨姜5钱易桂、附亦可。
制备方法：None
临床应用：None
各家论述：治此证者，惟在补气补血暖寒而已。补气之品，莫良于参、耆；补血之品，莫良于归身、熟地。至于附片、桂心，能除沉寒痼冷，且能流通气血，可以为佐；羌活虽非虚证所宜，然能利周身百节之痛，可以为使；山萸肉、杜仲、枸杞，皆肾经药品，可为向导。此方从大补元煎加减，诚补虚暖寒之主剂也。
用药禁忌：None
附注：None
