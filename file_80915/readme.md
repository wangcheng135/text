序号：80931
名称：飞龙换骨还元水火仙丹
组成：制粉霜5钱，真阿魏5钱，槐花3钱，当归尾1钱，白芷1钱，小丁香1钱，乳香1钱（不去油），没药1钱（不去油），雄黄1钱，朱砂1钱，牛黄5分，冰片3分，制砒霜1分。
出处：《疡医大全》卷三十四。
主治：远年近来杨梅结毒，误服轻粉，筋骨疼痛，痈肿、下疳、阴蚀、湿痰流注，气瘰发背，喉舌溃烂，目鼻破损，遍身梅豆、梅癣，鹤膝、血疯、臁疮，鱼口便毒。
加减：杨梅结毒生于头顶，或烂见骨，至重者升粉霜只用3钱，升砒霜只用1分5厘，汤引内加川芎、藁本各5钱，煎汤送下。结毒生于眼上，升粉霜只用1钱5分，升砒霜只用1分，汤引内加川芎、白芷、藁本、僵蚕各2钱，煎汤送下。结毒生于鼻上，升粉霜只用2钱，升砒霜1分，汤引内加山栀、川芎各2钱，煎汤送下。结毒生于喉中，升粉霜只用2钱，升砒霜1分，汤引内加天花粉、石菖蒲、川芎各2钱，煎汤送下。结毒生于臂上，升粉霜只用2钱，升砒霜5厘，汤引内加川芎、柴胡各2钱，煎汤送下。结毒生于臂膊、上下两胁，升粉霜只用4钱，重者5钱，升砒霜只用2分，天花粉末5钱为丸，汤引内加杜仲、怀牛膝各2钱，煎汤送下。结毒生于阴囊上下、玉茎上者，升粉霜6钱，重者用8钱，升砒霜2分，牡蛎（煅红冷定，研极细末）8钱，人参5钱为丸，汤引内加黄耆、白术、天花粉、怀牛膝各2钱，煎汤送下；结毒生于腿上下，升粉霜5钱，升砒霜2分，天花粉3钱，防风5钱为丸，汤引内加木瓜、薏苡仁、怀牛膝各2钱，煎汤送下。结毒生于脚板底下，升粉霜6钱，升砒霜2分，天花粉、沉香各5钱为丸，汤引内加木瓜、怀牛膝各2钱，煎汤送下。结毒生于妇女阴户，肉上染风毒，年久不愈，溃烂见骨，或穿通肛门，肉烂极臭，手足不能伸缩，或生阴户上下两边者，升粉霜8钱，升砒霜6钱，天花粉3钱为丸，汤引内加怀牛膝2钱，煎汤送下，外用猪肝切长条蒸熟，乘温插入阴户内，冷则换1条，如此十余条。如阴户内生湿疮作疼不可忍者，另用轻粉、儿茶各3钱研细搽之，倘疮痒如虫1般，次早用枸杞根煎汤洗净，仍用猪肝插换，掺药数次即愈。结毒下疳，用升粉霜3钱，升砒霜1分，汤引内加怀牛膝，猪苓各2钱，煎汤送下；鱼口便毒，用升粉霜2钱，升砒霜1分，汤引内加怀牛膝、僵蚕各2钱，煎汤送下。瘰疬穿烂日久，用丹粉霜6钱，升砒霜2分。外用玳瑁（锉碎，绵帛包置妇女乳上1宿，次早取下，研为细末）5钱为丸，汤引内加夏枯草、海藻、滑石、天花粉、瓜蒌仁各2钱，煎汤送下。
功效：None
用法用量：每日初服4丸，10日后服5丸，20日后服至7丸，再不可多加，用土茯苓4两，猪牙皂角1条，河水2碗，井水2碗，煎成3碗，每早以1碗送丸药，余2碗留为午、晚当茶吃。但须看人虚实，虚者服此必发寒热头痛，喉疼口臭，不必惊恐，停止2-3日再服。
制备方法：上药俱用老米打糊为丸，如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：升粉霜法：水银一两五钱，火消一两三钱，白矾、皂矾、食盐各一两，硼砂三钱。共研匀入阳城罐内，盐泥封固，升打文火四炷香，武火三炷香，候冷开看，铁盏上药刮下听用。看人虚实，毒之轻重久近，至重者一料入粉霜七八钱，再不可多用。升白砒霜法：白砒霜四两，打碎如指头顶大块，听用，取绿豆一升，煮汁二碗，去绿豆不用，将绿豆汤入甘草一两，同煎至一碗，去甘草滓，将汁一碗浸砒霜一日，次早煎干为度。入阳城罐内，盐泥封固，打文火二炷香，武火一炷香，候冷取出，看铁盏内有时如石榴子，有时如灰样，此两色始堪入药，若非二色，不可用。
