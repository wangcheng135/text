序号：45569
名称：柏子仁丸
组成：柏子仁1两，熟干地黄1两半，防风3分（去芦头），黄耆3分（锉），人参3分（去芦头），麦门冬1两半（去心，焙），当归半两（锉，微炒），续断3分，羚羊角屑半两，白茯苓3分，泽兰1两，桂心半两，芎半两，白术半两，酸枣仁3分（微炒），紫石英1两（细研，水飞过），附子3分（炮裂，去皮脐），甘草1分（炙微赤，锉）。
出处：《圣惠》卷八十一。
主治：产后风虚劳损，四肢羸弱，心神虚烦，不能饮食，少得眠卧。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前温酒送下。
制备方法：上为末，入研了药令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
