序号：72188
名称：犀角散
组成：犀角屑1两，防风1两（去芦头），枳壳1两（麸炒微黄，去瓤），独活1两，桂心3分，秦艽1两半（去苗），当归1两半（锉，微炒），赤芍药1两，仙灵脾1两，葛根1两半（锉），人参1两（去芦头），赤茯苓1两半，牛膝1两半（去苗），熟干地黄2两，黑豆3合（淘净，炒熟）。
出处：《圣惠》卷二十三。
主治：中风。手脚不遂，关节疼痛，心胸躁热。
加减：若躁甚语涩者，每服加竹沥半合同服。
功效：None
用法用量：每服5钱，以水1大盏，加生姜半分，煎至5分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：犀角汤（《圣济总录》卷八）。
