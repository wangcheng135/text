名称：活络丹
组成：牛黄2钱5分，片脑1钱5分，麝香5钱，人参1两，犀角5钱，白花蛇2两，乌梢蛇2两，黑附子1两，乌药1两，白豆蔻1两，青皮1两，白茯苓1两，香附1两，当归1两5钱，骨碎补1两，麻黄2两，川芎2两，两头尖2两，白术1两，羌活2两，防风2两，全蝎2两，天麻2两，玄参2两，威灵仙1两半，白芷2两，草豆蔻2两，血竭7钱半，黄芩2两，黄连2两，地龙5钱，大黄2两，熟地黄2两，木香2两（陆的），沉香1两（陆的），丁香1两，乳香1两，没药1两，安息香1两，细辛1两半，干葛1两半，赤芍药1两，姜蚕1两，天竹黄1两，龟版1两，虎骨1两，藿香2两，甘草2两，朱砂1两，官桂2两，松香5钱，何首乌2两，金箔400张，酥油1两，黄蜡40斤，蜜糖11斤。
出处：《医便》卷五。
主治：风湿诸痹，肩背腰膝，筋骨疼痛，口眼斜，半身不遂，行步艰难，筋脉拘挛，一切风疾。
加减：None
功效：清心明目，宽膈活络，宣通气血。
用法用量：每服1丸，茶酒服之；病在上，食后服，病在下，食前服；以四物服之尤妙。年过40，当预服10数丸，至老不生风疾。
制备方法：上为细末，炼蜜为丸，如弹子大，金箔为衣。
