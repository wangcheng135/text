序号：62230
名称：救涸汤
组成：麦冬2两，熟地2两，地骨皮1两，丹皮1两，白芥子3钱。
出处：《辨证录》卷三。
主治：嗽血。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：麦冬与熟地同用，乃肺肾两治之法也；加入地骨皮、丹皮者，实有微义，盖嗽血必损其阴，阴虚则火旺。然此火旺者，仍是相火，而非阳火也，用地骨皮、丹皮以解骨髓中之内热，则肾中无煎熬之苦，自然不索于肺金，而肺中滋润，自然清肃之气下济于肾内，子母相安，则肾水渐濡，可以养肝木，可以制心火，外侮不侵，家庭乐豫，何至有损耗之失哉。至于白芥子，不过消膜膈之痰，无他深意。以阴虚咳嗽者，吐必有痰，故取其不耗真阴之气也。
用药禁忌：None
