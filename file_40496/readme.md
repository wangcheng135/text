序号：40506
名称：虎骨酒
组成：虎胫骨1两（酥炙），黄耆（锉）1两，桔梗（炒）1两，酸枣仁（炒）1两，茯神（去木）1两，羌活（去芦头）1两，石菖蒲（米泔浸1宿，切，焙）1两，远志（去心）1两，芎1两，牛膝（酒浸1宿，切，焙）1两，肉苁蓉（酒浸1宿，切，焙）1两，熟干地黄（焙）1两，附子（生，去皮脐，以新汲水浸半日，又破作2片，换水浸1日，焙干）1两，萆薢1两，石斛（去根）1两，防风（去叉）半两，羚羊角（镑）半两。
出处：《圣济总录》卷一八六。
主治：风痹，皮肤不仁或下注步履艰难。
加减：None
功效：温养肝肾，调顺气血，补虚排邪，理腰膝，久服无健忘，益心气，清头目，定神魂。
用法用量：每服温饮1盏，每日2次。如服尽，添酒5升浸之，又服尽，取滓焙干为末，每服1钱匕，酒调下；或以蜜为丸，如梧桐子大，每服30丸，空心温酒送下。
制备方法：上药锉细，以生绢袋盛，入醇酒1斗浸之，密封瓶口，春、夏3日，秋、冬7日。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
