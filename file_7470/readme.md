序号：7470
名称：生肌膏
组成：当归1两，黄耆1两，山慈菇1两，白芷1两，甘草1两，血余1两，天麻1两，独活1两，穿山甲1两，露蜂房1两，五倍子1两，天花粉1两，荆芥1两，金银花1两，白蔹1两，肉桂1两，牛蒡子1两，白芍1两。
出处：《同寿录》卷四。
主治：诸毒。
加减：None
功效：收功长肉。
用法用量：None
制备方法：净麻油3升，如法熬，去滓，入飞过黄丹1斤半收之，再入白占、黄占、血竭、铜绿各2两，待冷，再入轻粉、乳香、没药（各去油）、龙骨、象皮（炒）、樟脑、儿茶、赤石脂各1两、麝香5钱、冰片2钱（各制为末），和入搅匀，瓷瓶收贮待用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
