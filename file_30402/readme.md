序号：30410
名称：返魂神白散
组成：花蕊石5斤（捶碎、如皂子大），硫黄6两（捶碎，如皂子大）。
出处：《博济》卷五。
主治：打扑伤损及伤中；妇人产后血晕。
加减：None
功效：None
用法用量：刀伤损甚者，于伤处掺药，其血化为黄水，更掺药，其人便活，更不疼痛；如妇人产后血晕欲死者，若心头暖，即以童便调下1钱，取下恶血如猪肝片，终身不患血风等症；若膈上有血，化为黄水吐出，及随小便出便愈；若牛抵人，肠出未损者，急纳入，以桑白皮缝合，于缝上掺药立活，封裹不可有缝；如内痛血入脏腑，热煎童便并入酒少许，调服1钱。
制备方法：上用瓷盆子一个，先入蕊石一层，次入硫黄一层，重重铺尽，上用鸭舌草盖之，赤石脂和涂盒子口缝，又用盐泥固济，勿令有小缝纹，用新砖一块，上安盒子，用炭火2斤煅之，耗及三分，渐渐去之，取出盒子，地坑内埋一宿，细研为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：返魂散（《鸡峰》卷二十二）。方中硫黄用量原缺，据《鸡峰》补。
