序号：29236
名称：远志散
组成：远志半两（去心），菖蒲半两，铁精半两，桂心3分，黄耆1两（锉），防风3分（去芦头），当归3分（锉，微炒），人参半两（去芦头），甘草半两（炙微赤，锉），熟干地黄3分，芎半两，茯神3分，独活半两，紫石英1两（细研如粉），五味子半两，麦门冬3分（去心），半夏半两（汤洗7遍去滑）。
出处：《圣惠》卷四。
主治：心气虚，惊悸喜忘，不思饮食。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加入生姜半分，大枣3枚，煎至6分，去滓，食后温服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：《医方类聚》引《神巧万全方》有秦艽，无当归。
