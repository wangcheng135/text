序号：27222
名称：当归煎丸
组成：当归（切，焙）半两，附子（去皮脐，生用）半两，没药（研）1分，硇砂（研）1分，血竭1分，禹余粮（煅赤，醋淬7遍）半两，延胡索半两。以上捣研为末，用酒3升调匀，于石器内慢火熬成膏，和下药：柴胡（去苗）1两，鳖甲（去裙襕，醋炙黄）1两，人参1两，生干地黄（焙）1两，芍药1两，磁石（煅，醋淬7遍）1两，牡丹皮1两，木香1分，泽兰半两。
出处：《圣济总录》卷一五三。
主治：妇人血积血癖，脐腹（？？）痛，心膈满闷，四肢烦疼，口苦舌干，饮食减少，渐成劳瘦。
加减：None
功效：None
用法用量：每服20丸，空心、食前淡醋汤送下。
制备方法：上将后九味捣罗为细末，入前膏内为丸，如豌豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
