序号：918
名称：六神丹
组成：巴豆（去皮油膜）1两，蓬术1两，青皮1两，陈皮1两，白姜（炮）1两，黄连1两。
出处：《普济方》卷一六九。
主治：男子妇人，远年近日，饮酒过度，结成痞块，噫醋吞酸，肚腹膨胀疼痛，诸般积病。
加减：None
功效：None
用法用量：每服10-15丸，姜、盐汤送下；或米饮茶汤亦可；面色萎黄，已患伤寒不痊者，用药2钱，茶2钱，生姜3片，葱白1根，水1盏半，煎至1盏，吞下六神丹，用被盖出汗，宣3-5行；温粥补脾寒；妇人血气病，两胁痛，每服10丸，用红花酒送下；男子冷气或两肋刺痈，每服10丸，炒茴香酒送下；冷泻，每服10丸，酸醋汤送下；热泻，每服10丸，冷水送下；瘕聚积滞，随意斟酌，加减服之。
制备方法：上为细末，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
