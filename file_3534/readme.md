序号：3534
名称：太清四扇丹
组成：丹砂4两（碎如麻豆，上巳日采桃花20两，真蜜1斤，同入银器中，重汤煮至5月朔日，以新井水淘涤净，取砂研之），麒麟竭4两（选真者，5月2日入丹砂中研），没药4两（透明者，5月3日入丹砂研），麝香4两（全，以当门子5月4日入丹砂同研）。
出处：《圣济总录》卷二○○。
主治：一切恶气，痁疟；兼治妇人血气冲心疼痛。
加减：None
功效：补心导血，驻颜益气，调和百脉，安心辟邪，尤辟时疫。
用法用量：每以沉香温水2合，嚼下1丸或半丸。若疫岁，即全家斋洁，旦起面东，以井花水各人嚼下1丸，断不染疫。丈夫生姜酒下；妇人炒姜醋汤下。痁疟，腊茶下1丸或华丸，量疾大小服。
制备方法：上药先取上党人参1两，甘草1分，并细锉，5月1日日未出时，采露水1升。真留3两同浸之，至端午日，熬水至半，去滓。入麦面，作糊和药，稍硬剂之，为丸，如楝实大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
