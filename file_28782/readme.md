序号：28790
名称：十全丸
组成：麝香3钱，穿山甲（土炒脆）1两，广木香（生研）1两，血竭（另研）1两，雄黄（水飞）1两，山芝麻（酒炒）1两，番木鳖（黄土炒，焦黄为度，不可太枯，筛取净末）1两，自然铜（火煅醋淬9次，研细，水飞）1两，僵蚕（炒去丝，去头足）1两，川蜈蚣（去足尾，酒炙）21条（一方去木鳖子，加风茄花5钱，山芝麻亦用5钱）。
出处：《纲目拾遗》卷三引《绿竹堂方》。
主治：风痹，跌扑，痈疽初起。
加减：None
功效：None
用法用量：每用1丸，至重者再进1丸，用羌活、紫苏，酒煎化服。取汗避风，否则发战伤人。
制备方法：上为末，炼蜜为丸，如梧桐子大，以朱砂为衣，金箔裹之，蜡丸封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
