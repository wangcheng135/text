序号：35045
名称：龟肉丸
组成：川芎2两，当归2两，白芍2两（火煨），紫苏8钱（去梗），人参6钱，砂仁2两，花椒2两（取半为末），艾叶2两（醋炙，焙干），香附（酒便浸），白术2两（麸炒），陈皮2两（去白），熟地2两（酒浸，杵膏），干姜5钱（炒），五味3钱（去梗）。
出处：《宋氏女科秘书》。
主治：瘦怯妇人虚寒者。
加减：None
功效：None
用法用量：每服70-80丸，或米饧或温酒送下。
制备方法：用黄皮龟肉1斤8两，汤泡去皮，连爪，用好酣酒各1碗，漠烂如泥，焙干，和前药为末，将大枣煮烂，入生姜同煮，去皮核，去姜杵膏，和地黄膏为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
