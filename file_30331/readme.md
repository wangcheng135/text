序号：30339
名称：还少丹
组成：何首乌（黑豆蒸）半斤，牛膝6两，生地黄（酒蒸）6两，肉苁蓉（酒蒸）6两，黄柏（酒浸，炒褐色）3两5钱，补骨脂（酒浸，水蒸）3两5钱，车前子（微炒）3两5钱，柏子仁（微炒）3两5钱，干山药（微炒）3两5钱，秦当归2两5钱（酒洗），菟丝子（水淘，去砂，酒煮，捣成饼，晒干）2两，人参1两，五味子1两。
出处：《济阳纲目》卷六十四。
主治：虚损。
加减：None
功效：益精补髓，壮元阳，却病延年，发白返黑。
用法用量：每服60丸，空心盐汤、白汤、酒任下。
制备方法：上俱勿犯铁器，为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
