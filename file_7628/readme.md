序号：7628
名称：二白散
组成：山药、芡实各等分（约各4斤），万年青4大片。
出处：《辨证录》卷八。
主治：脾痨，胸前饱闷，食不消化，吐痰不已，时时溏泻，肚痛腹胀，空则雷鸣，唇口焦干，毛发干耸，面色黄黑，微微短气，怯难接续，便如黑汁，痰似绿脓。
加减：None
功效：None
用法用量：入白糖1斤，滚水调服。
制备方法：各炒，磨为细末。
临床应用：脾痨：一妇人得此症，脉又细数，余劝其单服二白散，遇饥即用，无论数次。头一日即服五大碗。约五月，每日如此，脾气渐服渐愈，竟得不死。问其前后所服几何，约百斤也。
各家论述：山药、芡实二味既能健脾，尤能补肾，脾肾兼治，所以奏功。况万年青杀虫于无形，入之于二味之中，虫亦不知其何以消灭于无迹也，此方不特单治脾痨，但不可责其近功耳。若加入人参二两以助胃气，则胃气更健，脾气尤易援耳。
用药禁忌：None
附注：None
