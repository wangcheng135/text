序号：39881
名称：三肾丸
组成：鹿肾1条，驴肾1条，狗肾1条，生黄耆2两，人参（去芦）2两，当归2两，熟地黄2两，龟版（醋制）2两，茯苓（去皮）2两，枸杞子2两，生于术1两，生阿胶1两，山茱萸（酒蒸）1两，制附子1两，淫羊藿（羊油炒）1两，蒺藜（盐炒）1两，补骨脂（盐炒）1两，菟丝子1两，鱼鳔（滑石烫）1两，杜仲炭（盐炒）1两，鹿茸（去毛）1两，肉桂（去粗皮）8钱。
出处：《全国中药成药处方集》（天津方）。
主治：腰肾不足，腰腿酸痛，肾囊湿冷，身体衰弱，倦怠少食。
加减：None
功效：滋补腰肾，强阴补阳。
用法用量：每次服1丸，白开水送下。
制备方法：先将方内3种肾洗净，用水煮烂，连原汤与群药共串一处晒干，共为细粉，炼蜜为丸，重2钱，蜡皮或蜡纸筒封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
