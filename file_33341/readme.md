序号：33350
名称：肝胃至宝丹
组成：莱菔子（炒）2两，香附（炙）1两，橘皮1两，焦三仙1两，三棱（炒）1两，厚朴（炙）1两，莪术（炙）1两，槟榔1两，枳壳（炒）8钱，白芍8钱，丹皮8钱，木香5钱，片黄姜5钱，旋覆花5钱，豆蔻仁5钱，川芎3钱，沉香3钱，丁香3钱，砂仁6钱，甘草2钱，赭石（煅）1两，青皮5钱，枳实6钱。
出处：《北京市中药成方选集》。
主治：胸胁痞闷，气滞不舒，呕逆胀满，嘈杂吞酸。
加减：None
功效：舒郁平肝，健胃化滞。
用法用量：每服2钱，以温开水送下，1日2次。
制备方法：上为细末，冷开水为小丸。每16两用朱砂3钱2分，滑石3两2钱为衣，闯亮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
