序号：79398
名称：蝉花丸
组成：蜜蒙花半两，威灵仙（去芦）半两，白蒺藜（炒，去尖）半两，木贼（去毛节）半两，草决明（炒）半两，菊花（去梗）半两，楮实子（炒）半两，石决明（火煅）半两，蝉退（麻油净洗）半两，青葙子（瓦上炒）半两，川芎半两，羌活（生）半两，旋覆花1两，甘草1两，荆芥穗1两。
出处：《普济方》卷七十八引《卫生家宝》。
主治：年深日近，诸般内外障眼，风毒气毒，翳膜赤脉，逆顺横关，胬肉攀睛，眩睑赤烂，多脓多泪，昏暗不明，视物茫茫。
加减：None
功效：None
用法用量：每服1丸，食后、临卧细嚼，用浓煎灯心、麦门冬汤送下，日3次。
制备方法：上为细末，炼蜜为丸，每两作15丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
