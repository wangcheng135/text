序号：46545
名称：千金化痰丸
组成：胆星4两，半夏（姜矾同煮半日）4两，陈皮（去白）2两，白茯苓（去皮）2两，枳实（去瓤，麸炒）1两，海石（火煅）1两，天花粉2两，片芩（酒炒）2两，黄柏（酒炒）1两，知母（酒炒）1两，当归（酒洗）1两，天麻（火煅）2两，防风（去芦）2两，白附子（煨）2两，白术（米泔浸，炒）2两，大黄（酒拌蒸9次）5两，甘草（生）3钱。
出处：《回春》卷二。
主治：痰饮。
加减：气虚，加人参8钱。
功效：健理脾胃，清火化痰，顽痰能软，结痰能开，疏风养血，清上焦之火，除胸膈之痰，清头目，止眩晕。
用法用量：每服60-70丸，茶送下。
制备方法：上为细末，神曲2两打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
