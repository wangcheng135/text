序号：33898
名称：没药降圣丹
组成：自然铜（火煅，醋淬12次，研末，水飞过，焙）1两，川乌头（生，去皮脐）1两，骨碎补（爁去毛）1两，白芍药1两，没药（别研）1两，乳香（别研）1两，当归（洗，焙）1两，生干地黄1两半，川芎1两半。
出处：《局方》卷八（绍兴续添方）。
主治：打扑闪肭，筋断骨折，挛急疼痛，不能屈伸，及荣卫虚弱，外受游风，风伤经络，筋骨缓纵，皮肉刺痛，肩背拘急，身体倦怠，四肢少力。
加减：None
功效：接续筋骨。
用法用量：每服1丸，捶碎，水、酒各半盏，入苏木少许，同煎至8分，去苏木，空心食前热服。
制备方法：上为细末，以生姜自然汁与蜜等分为丸，每1两作4丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
