序号：81002
名称：麝香丸
组成：阿魏半两（精明者，于砂石器中熬，入在羊肉内），猪牙皂角（炙去皮）3分，雄黄半两（细研），蓬莪术（生用）半两，柴胡（去苗）3分，槟榔（生用）1两半，芜荑仁（生用）1两，当归半两（洗，焙），麝香1分（别研），辰砂半两（别研）。
出处：《杨氏家藏方》卷十八。
主治：诸疳挟积，肌体发热，渐致羸瘦，虫作无时。
加减：None
功效：None
用法用量：每服30丸，温米饮送下，不拘时候。
制备方法：上为细末。用精羊肉去筋膜1斤，切细，以法酒煮如泥，取出细研，入阿魏并诸药同捣为丸，如萝卜子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
