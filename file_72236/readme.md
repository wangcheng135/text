序号：72252
名称：紫金丸
组成：羊脊骨（全）1条（以硇砂1分，酒2盏化开，浸骨一复时，取出，炙令焦黄，别为末），生地黄10斤（研绞取汁），杏仁5升（去皮尖双仁，炒），蜀椒（去目并合口者）半斤（炒出汗），附子（炮裂，去皮脐）半斤。
出处：《圣济总录》卷八十六。
主治：肺劳。胸满、气急、喘嗽，气不升降，饮食减少。
加减：None
功效：None
用法用量：每服10丸，空心温酒送下，服后良久，以饭压之，女子服亦得。
制备方法：上除地黄汁、脊骨末外，并为末，取地黄汁于银锅中用炭火以灰罨四面煎之，勿令火急，便入诸药末，以柳木篦搅三百下后方入脊骨末，又搅勿住手，但看稀稠，可丸即丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
