序号：37679
名称：三合保胎丸
组成：大怀地12两（用砂仁3两，老姜3两，将地黄入砂锅内，先以净水煮两昼夜，俟地黄将烂，始入好酒煮之，总以地黄糜烂为度，将酒煮干取起，拣去砂仁，姜滓不用，将地黄捣膏听用），大当归（去头尾，取身切片）12两（以好酒洗过，晒干听用），漂白术（取净干片）12两（以黄土研碎拌炒，极黄取起，筛去土。），实条芩（枯飘者不用，取小实者切片）6两（酒炒3次），棉杜仲（切片）12两（盐水拌炒，以丝断为度），川续断（切片）12两（酒炒）。
出处：《幼幼集成》卷一。
主治：素惯堕胎者。
加减：孕妇肥白气虚者，再加白术2两；黑瘦者，再加条芩1两，性躁者2两；至怯者，加人参。
功效：None
用法用量：每早盐汤送下3钱，晚临卧酒送下3钱，每日如此，不可间断，孕妇素怯者，须两料方可。自1月服起，过7个月方保无虞。
制备方法：上将后5味和为一处，火焙干燥，石磨磨细末，筛过，以前地黄膏和匀，少加炼蜜入石臼内，捣千余杵为丸，如绿豆大。
临床应用：None
各家论述：以古之内补丸、杜仲丸、白术散三方合凑，名三合保胎丸。以条芩清肝火而凉血，白术扶中气以健脾，当归养血宁心，熟地滋阴补肾，续断填损伤而坚胞系，杜仲益腰膝而暖子宫。药虽平易，功胜神丹，诚所谓针芥相投，捷如影响。
用药禁忌：None
附注：None
