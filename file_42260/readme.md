序号：42275
名称：青盐散
组成：青盐1斤，禹余粮2斤，白矾1斤。
出处：《圣惠》卷二十四。
主治：大风疾，眉鬓坠落，遍身结肿，皮肉顽痹。
加减：None
功效：None
用法用量：每服2钱，空心及晚食前荆芥茶调下。
制备方法：上为细散，更研令匀，入瓷罐子内盛，瓦子盖头，固济，初用炭火2斤烧，渐渐添火至一秤已来，从早晨烧至夜，常须添炭至一秤火尽为度，隔宿候冷，取之细研如粉，用夹熟帛包裹，取一生净土，水拌令浥浥，中心培药，2日出火毒，又别取胡麻子6斤（拣簸净洁，称之）九蒸、九晒毕，炒令香熟，捣罗为末，每3两胡麻末管1两烧者药末相和令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
