序号：4742
名称：木香散
组成：木香半两，草豆蔻半两（去皮），桂心半两，陈橘皮半两（汤浸，去白瓤，焙），神曲1分（微炒黄色），白术半两，荜茇半两，人参3分（去芦头），甘草半两（炙微赤，锉），柴胡半两（去苗），桃仁3分（汤浸，去皮尖双仁，麸炒微黄），厚朴3分（去粗皮，涂生姜汁，炙令香熟）。
出处：《圣惠》卷十一。
主治：伤寒食毒，脾胃虚乏，四肢少力，不思饮食，心腹气胀，或时下利，向晚憎寒。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，煎至6分，去滓，不拘时候稍热服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
