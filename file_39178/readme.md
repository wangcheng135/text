序号：39188
名称：炙肝散
组成：紫菀半两（洗去苗土），干姜半两（炮裂，锉），缩砂半两（去皮），芜荑半两，人参1分（去芦头），白茯苓1分，甘草1分（炙微赤，锉），当归1分，木香1分，陈橘皮1分（汤浸，去白瓤，焙），川椒1分（去目及闭口者，微炒去汗），厚朴半两（去粗皮，涂生姜汁炙令香熟），草豆蔻半两（去皮），桂心1分，胡椒1分，桔梗半两（去芦头），细辛半两，苍术半两，白术半两，附子半两（炮裂，去皮脐），芎1分。
出处：《圣惠》卷二十八。
主治：冷劳，羸瘦不能食，心腹多疼，四肢无力。
加减：None
功效：None
用法用量：每服用猪肝1具，去脂膜，薄切如角片，入散1两半，葱薤白1握细切，盐末相拌令有味，以竹箸子串，慢火炙令熟，空心食之，后饮暖酒1-2盏为妙。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
