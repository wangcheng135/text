序号：41741
名称：金蟾化管丸
组成：水银3钱，明雄黄1两（以2斤火酒，渐煮渐添，以酒尽为度）。
出处：《疡医大全》卷七。
主治：一切诸漏有管者。
加减：None
功效：化管。
用法用量：凡一切管，用1丸放膏药上，对管口，自入到底方回，嫩管自化，老管自退，7日见效；如未全退，再用1丸，无不除根。
制备方法：上乳细，用纸包好，取大蛤蟆1个，剖去肠，只留肝肺，将药包入于肚内，以线缝好，听用；再将银消、白矾各1两研匀，入阳城罐内，加水半茶钟，放火上熬令枯于罐底，取放地上，再纳蛤蟆于内，铁盏盖好，盐泥固济，升文火3炷香，中火1炷香，武火1炷香，冷定开着，盏上灵药刮下研细，用蟾酥（乳化）为丸，如芥子大，阴开。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
