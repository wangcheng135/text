序号：47790
名称：神仙轻脚丸
组成：远志（去心）1两，楮实子（炒）1两，巴戟（去心）1两，白茯苓（去皮）1两，五味子（去枝）1两，杜仲（锉，姜制，炒去丝）1两，舶上茴香（炒）1两，菟丝子（淘去泥，酒浸1宿，研成饼）1两，覆盆子（北地坚实者尤佳）1两，芡实（干者，去壳）1两，山茱萸（去核）1两，干山药1两半，肉苁蓉（酒洗，去皴皮，切片，再用酒浸1宿，焙）1两半，宣木瓜（去瓤，切片，焙）1两半，萆薢1两半，牛膝（酒浸，锉，再用酒浸1宿，焙干）2两。
出处：《魏氏家藏方》卷八。
主治：脚弱，不能久行久立。
加减：None
功效：壮筋骨，延年益气。
用法用量：每服30-50丸，空心温酒送下。
制备方法：上为细末，用北枣肥好不破者200个，蛇床子4两水煮令熟，去蛇床子，用枣子（去皮核取肉）搜药末杵丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
