序号：47785
名称：神仙济阴丹
组成：败姜1两，青皮1两，陈皮1两，三棱1两，蓬术1两，乌头1升（以上6味无以米醋2升煮，焙干为度），熟地黄半两，生地黄半两，赤芍药半两，当归半两，白芍药半两，刘寄奴半两，姜黄半两，肉桂半两，蒲黄半两。
出处：《普济方》卷三二八。
主治：妇人胎前产后，一切积气，血块，血癥，血瘕，血晕，血虚，血闷，血壅，血崩，血淋，血竭，赤白带下，月事不调，脐腹疼痛，腰膝沉重，干呕恶心，不思饮食，五心烦热，四肢倦怠，或寒热，呕吐酸水，头目晕眩，遍身隐痛，坐卧不安，经络凝滞，荣卫不调，气血虚弱，变成血劳热，面黄肌瘦，梦中惊悸，虚怯盗汗，或产后有失调理，至天阴雨下，浑身疼痛，或子宫久冷，或少腹痛，室女经脉不行，肠风食毒，下血不住。
加减：None
功效：常服升降阴阳，温暖血海。
用法用量：每服30丸，空心、食前姜酒送下。子宫久冷，或少腹痛，每服30-40丸，细嚼，炒姜酒送下；肠风食毒，下血不住，以槐花、刘寄奴各等分为细末，空心米汤送下。
制备方法：上为细末，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
