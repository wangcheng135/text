序号：58574
名称：秘传仙遗粮散
组成：仙遗粮（即土茯苓）2斤（忌铁），荆芥1两5钱，防风1两5钱，五加皮1两5钱，白鲜皮1两5钱，威灵仙1两5钱，木瓜1两5钱，生地（酒洗）1两，当归1两（酒洗），白芍1两（炒），白茯苓1两，川芎1两，牛膝1两（酒洗），杜仲（炒去丝）1两，白芷1两，地骨皮1两，青风藤1两，槐花1两，黄连1两。
出处：《医林绳墨大全》卷九。
主治：远年杨梅风漏，筋骨疼痛；轻粉毒。
加减：None
功效：补元气。
用法用量：上(口父)咀，作10帖。每1帖用水1钟，米白酒1钟，煎熟，疮在上，食后服；疮在下，食远服；滓再煎，每1日服1帖，煎2次合为一处，使浓淡得均，分作2次温服。其药滓逐日晒干，至3帖，共煎汤洗浴。初服5贴之内疮势觉甚，乃毒气攻外，勿俱，轻者至10日内可见效；重者虽服20-30帖无妨。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：切忌房事、茶、生冷、煎炒、鸡、鹅、羊、牛血，虾，猪蹄首各发物、动风之物。只用雄猪肉。
附注：None
