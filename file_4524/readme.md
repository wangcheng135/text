序号：4524
名称：木香导滞汤
组成：木香3钱，延胡索3钱，香附米3钱，枳实4钱，台乌药3钱，陈皮3钱，五灵脂3钱，郁金3钱，炮姜2钱，甘草2钱。
出处：《医学探骊集》卷五。
主治：贪食生冷，陈寒结气，滞塞脾胃，或受寒凉，或逢怒气，其胃脘处或寒块作痛，或寒条作痛，或胃腑作痛，轻者尚可饮食，重者呕吐酸水，饮食不下。
加减：初得此症，去陈皮、乌药，加大黄5钱，芒消2钱，1剂可永不发。若因气逆而得年久不愈者，将原方加韭菜根1把，鸡子2个，酒、水各半煎之，煎数滚，俟鸡子已熟，将鸡子取出入勺内，克破外皮，亦连皮入内煎之，俟药已煎好，将鸡子剥去皮，先食鸡子，然后服药。
功效：None
用法用量：水煎，温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
