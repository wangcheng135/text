序号：62168
名称：救苦膏
组成：大黄2两，花粉7钱，牙皂8钱，蓖麻子2两，全蝎7钱，枳壳8钱，生地黄1两，桃仁7钱，白芷8钱，草乌1两，五倍子7钱，莪术1两，羌活8钱，麻黄8钱，肉桂8钱，红大戟8钱，香附7钱，厚朴7钱，穿山甲7钱，蛇蜕5钱，当归1两5钱，甘遂2两，木鳖子2两，川乌1两，三棱1两，巴豆8钱，黄柏8钱，芫花7钱，杏仁7钱，防风7钱，独活7钱，槟榔7钱，细辛7钱，玄参7钱，黄连5钱，蜈蚣10条。
出处：《全国中药成药处方集》（沈阳方）。
主治：风寒湿痹，腰腿作痛，筋骨麻木，四肢不仁，半身不遂，口眼斜，症瘕积聚，肚腹疼痛；女子经血不调，赤白带下；膨闷胀饱，水臌痈疽，对口，无名肿毒。
加减：None
功效：解毒，散风，活血。
用法用量：贴患处。
制备方法：上用麻油50两，入群药浸数日，用慢火熬之，待滴水成珠后将药除去，兑入黄丹24两，密陀僧4两，成膏待用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
