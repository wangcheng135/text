序号：84224
名称：五色灵药
组成：食盐5钱，黑铅6钱，枯白矾2两，枯皂矾2两，水银2两，火消2两。
出处：《金鉴》卷六十二。
主治：痈疽诸疮已溃，余腐不尽，新肉不生者。
加减：None
功效：None
用法用量：None
制备方法：先将盐、铅熔化，入水银结成砂子，再入二矾、火消同炒干，研细，入铅、汞再研，以不见星为度，入罐内。泥固济，封口打3炷香，不可太过不及。1宿取出视之，其白如雪，约有2两，为火候得中之灵药。如要色紫者，加硫黄5钱；要黄色者，加明雄黄5钱；要色红者，用黑铅九钱，水银1两，枯白矾2两，火消3两，辰砂4钱，明雄黄3钱，升炼火候，俱如前法。凡升打灵药，消要炒燥，矾要煅枯。一方用烧酒煮干，炒燥方研入罐。一法凡打出灵药，倍加石膏和匀，复入新罐内，打1枝香，用之不痛。
