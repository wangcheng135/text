序号：17721
名称：白丸子
组成：附子（半炮半生，去皮脐）半两，白附子（半生半炮）半两，半夏（汤洗7遍，半生半偎）半两，天南星（热水洗，半生半泡）半两，天麻干蝎（生用）半两，白花蛇肉（酥拌，炒令黄）半两，甘菊花半两，羌活半两，防风（去芦头）半两，芎藭半两，桂心半两，白僵蚕（生用）半两，白鲜皮半两，木香半两，巴豆半两（去心，研，纸裹压去油，别研），朱砂1分（细研），雄黄1分（细研），麝香1分（细研）。
出处：《圣惠》卷二十五。
主治：一切风。
加减：None
功效：None
用法用量：每服3丸，以暖酒送下。
制备方法：上为末，入朱砂、雄黄、麝香等，研令匀，以糯米饭为丸，如梧桐子大，用腻粉滚过。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
