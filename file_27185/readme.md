序号：27193
名称：当归续断膏
组成：当归1两，续断1两，骨碎补1两，桂（去粗皮）1两，附子1两，泽兰1两，芍药1两，白及1两，牛膝1两，羌活1两，芎1两，木香1两，麒麟竭1两，生干地黄1两，白僵蚕1两，白附子1两，沉香半两，丁香半两，栝楼2枚（大者），乌蛇肉1两，白敛1两，白芷1两，玄参1两，杏仁3分，桃仁3分。
出处：《圣济总录》卷一四○。
主治：箭头入肉赤肿。
加减：None
功效：辟风敛疮。
用法用量：于疮口上涂摩之。
制备方法：上细锉，入麻油4斤，猪脂1斤半，驼脂3两，用文武火煎三日后，滤去诸药，入乳香3两，松脂6两，更煎一日，用生绢滤却粗滓。再用5斗大生铁锅，细罗铅丹3斤，炒令紫色，旋旋入前药油煎，以柳枝子搅令紫色，旋即退火。以药油滴少许水碗内，成珠子为度，以瓷石器密收。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“续断膏”。
