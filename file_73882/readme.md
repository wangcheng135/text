序号：73898
名称：硼砂丸
组成：硼砂半两，巴豆半两（去皮心，研，纸裹压去油），干姜半两（炮裂，锉），附子1两（炮裂，去皮脐），青橘皮1两（汤浸，去白瓤，焙），京三棱1两（微煨，锉），干漆1两（捣碎，炒令烟出），香墨半两（挺）。
出处：《圣惠》卷四十九。
主治：食癥及气块，攻刺心腹，疼痛不可忍。
加减：None
功效：None
用法用量：每服3丸，食前温酒送下。
制备方法：上各为细散，先取硼砂、巴豆细研，用头醋2碗煎为膏，然后入诸药末相和为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“硇砂丸”。方中硼砂，《普济方》引作“硇砂”。
