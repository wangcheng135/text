序号：34044
名称：皂荚煎丸
组成：皂荚2斤（不蛀、肥好者，用1斤生捣碎，以水1斗浸1宿，揉取汁），梨10个，生薄荷1斤，生荆芥1斤，上3味，入于皂荚水内，用揉洗令极烂，以生绢绞取汁煎。皂荚1斤（刮去黑皮，以酥3两薄涂，慢火炙令黄焦，酥尽为度），防风（去芦头）2两，威灵仙2两，独活2两，羌活2两，甘菊花2两。
出处：《圣惠》卷六。
主治：积年肺脏风毒，遍身生疮，大肠壅滞，心神烦躁。
加减：None
功效：None
用法用量：每服20丸，以温浆水送下，不拘时候。
制备方法：上为末，以一半入在煎药汁内，于银锅中慢火熬，看稀稠得所，入余一半药，同搜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：皂荚煎（《圣济总录》卷五十）。
