序号：24482
名称：安神镇惊丸
组成：天竺黄（另研）5钱，人参5钱，茯神5钱，南星（姜制）5钱，酸枣仁（炒）3钱，麦门冬3钱，当归（酒炒）3钱，生地黄（酒洗）3钱，赤芍药（炒）3钱，薄荷2钱，木通2钱，黄连（姜汁炒）2钱，山栀（炒）2钱，辰砂（另研）2钱，牛黄（另研）2钱，龙骨（煅）2钱，青黛1钱（另研）。
出处：《保婴撮要》卷三。
主治：小儿惊退后。
加减：None
功效：扶元安神，开膈祛痰，养气血。
用法用量：每服3-5丸，淡姜汤送下。
制备方法：上为末，炼蜜为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：百益镇惊丸（《全国中药成药处方集》（杭州方））。
