序号：12808
名称：石斛丸
组成：石斛（去根）半两，远志（去心）半两，槟榔（煨，锉）半两，牛膝（酒浸1宿，焙）半两，桑螵蛸（炙焦再炒）半两，桂（去粗皮）半两，干姜（炮）半两，五味子（炒）2两，覆盆子（微炒）2两，肉苁蓉（酒浸，去皴皮，切，焙）2两，巴戟天（去心，微炒）2两，枳壳（去瓤，麸炒）2两，柏子仁（研）2两，陈橘皮（汤浸，去白，焙）2两，鹿茸1对（去毛，酥炙），泽泻3分，白蒺藜（炒去角）3分，天雄（炮裂，去皮脐）2两，菟丝子（酒浸，捣烂，焙3日）2两。
出处：《圣济总录》卷一八五。
主治：None
加减：None
功效：平补诸虚不足。
用法用量：每服30丸，空心盐汤送下。
制备方法：上为细末，和匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
