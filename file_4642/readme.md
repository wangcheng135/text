序号：4642
名称：木香神效散
组成：南木香2钱半，青皮2钱半，陈皮2钱半，麦蘖（炒）2钱半，大枳壳（炒）2钱半，京三棱2钱半，蓬莪术2钱半，神曲（炒）2钱半，甘草（炙）2钱半，北白芍药2钱半，川白芷2钱半，肉桂（去皮）2钱半，元胡索2钱半，破故纸2钱半，荜澄茄1钱，丁香1钱。
出处：《朱氏集验方》卷四。
主治：远年近日一切脾病。
加减：None
功效：None
用法用量：上(口父)咀。每服3钱，水1盏半，加生姜3片，枣子1枚，煎至7分，临熟加盐1捻，再煎2沸，空心服。
制备方法：None
临床应用：脾病：乡里萧国贤云：自感脾病以来，遇食冷物或天气寒冷便发，发则胸间一点痛起，或引入背膂，痛不可忍，服之遂绝根源。屡用屡效，难以具述。
各家论述：None
用药禁忌：忌面食、豆腐、一切生冷。
附注：None
