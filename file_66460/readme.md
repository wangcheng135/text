序号：66476
名称：小儿至宝丸
组成：紫苏叶50g，广藿香50g，薄荷50g，羌活50g，陈皮50g，白附子（制）50g，胆南星50g，白芥子（炒）30g，川贝母50g，槟榔50g，山楂（炒）50g，茯苓200g，六神曲（炒）200g，麦芽（炒）50g，琥珀30g，冰片4g，天麻50g，钩藤50g，僵蚕（炒）50g，蝉蜕50g，全蝎50g，牛黄6g，雄黄50g，滑石50g，朱砂10g。
出处：《中国药典》（一部）。
主治：小儿感冒风寒，停食停乳，发热鼻塞，咳嗽痰多，呕吐泄泻，惊惕抽搐。
加减：None
功效：疏风镇惊，化痰导滞。
用法用量：口服，每次1丸，1日2至3次。
制备方法：以药除牛黄、冰片外，雄黄、朱砂分别水飞或粉碎成极细末；其余紫苏叶等21味粉碎成细末；将牛黄、冰片研细，与上述粉末配研，过筛混匀。每100g粉末加炼蜜110～140g制成大蜜丸，每丸重1．5g。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
