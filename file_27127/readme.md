序号：27135
名称：当归散
组成：当归（锉，微炒）2两，羌活2两，附子（炮裂，去皮脐）2两，防风（去芦头）2两，薏苡仁2两，麻黄（去根节）2两，茵芋1两，羚羊角屑1两，菖蒲1两，阿胶（捣碎，炒令黄燥）1两，干蝎（微炒）1两，木香1两，牛膝（去苗）1两，柏子仁1两，芎1两半，桂心1两半，麝香1分（细，研），乌蛇（酒浸，去皮骨，炙微黄）。
出处：《圣惠》卷七十八。
主治：产后中风，手脚顽痹，缓弱无力。
加减：None
功效：None
用法用量：每服2钱，以豆淋酒调下，不拘时候。
制备方法：上为细散，入麝香，相和令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
