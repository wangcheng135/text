序号：49855
名称：大牛角中人散
组成：牛角人1枚（烧），续断3两，干地黄3两，桑耳3两，白术3两，赤石脂3两，矾石3两，干姜3两，附子3两，龙骨3两，当归3两，人参1两，蒲黄2两，防风2两，禹余粮2两。
出处：《千金》卷四。
主治：妇人积冷崩中，去血不止，腰背痛，四肢沉重，虚极者。
加减：None
功效：None
用法用量：未食前服方寸匕，以温酒送下，每日3次。不知，稍加。
制备方法：上药治下筛。
临床应用：None
各家论述：《千金方衍义》：此即前小牛角散之变法。方中地黄乃阿胶之变；桑耳乃赤小豆之变；蒲黄乃乌贼骨之变；人参乃鹿茸之变；附子、白术专用温经；石脂、矾石专行固脱；防风一味，专祛子脏中风，且治腰背痛，四肢沉重，兼行地黄之滞。以其积冷虚极，非峻用温补兜涩，必难取效耳。
用药禁忌：None
附注：None
