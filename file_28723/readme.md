序号：28731
名称：十二味异功散
组成：木香3钱半，官桂2钱（去粗皮），当归3钱半，人参2钱半，茯苓1钱，陈皮2钱半，厚朴2钱半（姜制），白术2钱，半夏（姜制）1钱，丁香2钱半，肉豆蔻2钱半，附子（泡去皮）1钱半。
出处：《陈氏小儿痘疹方论》。
主治：痘出不光泽，不起胀，根窠不红，表虚痒塌。溃疡阴盛阳虚，发热作渴，手足并冷，脉虚无力，大便自利，至饮沸汤而不知其热者。
加减：None
功效：None
用法用量：每服3钱，水1大盏半，加生姜5片，肥枣3个，煎至6分，去滓，空心温服。3岁儿作3服，5岁儿作2服，一周两岁儿作3-5服。
制备方法：上为粗散。
临床应用：None
各家论述：1.《医方考》：中气有余，气血充满，则痘光泽起发，根窠红活，表无痒塌之患；中气不足，则表亦虚，而诸证作矣。是方也，人参、白术、茯苓、当归所以补胃；附子、肉桂、丁香、豆蔻，所以温胃；半夏、木香、陈皮、厚朴所以调胃，胃，阳明也。陈氏云：阳明主肌肉，胃气充足，则肌肉温暖，自然光泽起胀，而无痒塌之患，亦见道之论也。_x000D_
2.《古方选注》：异功散，治脾肾里虚之方也。证因内虚而变，故方中破滞之味轻，助阳之力大。人参。茯苓和胃，白术、广皮健脾，俾胃暖脾温而营卫无滞，自能升提痘毒，出于皮毛，附子理虚而收战栗，官桂鼓阳气以安塌痒，丁香安胃，木香理脾，半夏破滞化痰，厚朴温胃破滞，肉果温脾止泻，当归活血成浆。表虚里实，独用生姜，里虚表陷，姜、枣并用。
用药禁忌：None
