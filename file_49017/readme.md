序号：49032
名称：胃溃疡片
组成：海螵蛸280g，蒲公英200g，山药100g，白芨100g，姜半夏100g，洋金花10g。
出处：《山东省药品标准》。
主治：胃溃疡。
加减：None
功效：止痛，制酸，消炎。
用法用量：口服或嚼碎服。每次4-6片，每日3次。
制备方法：（l）取山药粉碎成细粉，过筛。（2）白芨、姜半夏、蒲公英煮提两次，每次2小时，将提取液澄清，滤过，蒸发至比重1.16（90℃测）。（3）将洋金花制粗粉，照渗漉法用50％乙醇作溶媒，浸渍24小时后，开始渗滤，蒸发至比重1．16（90℃测）。（4）取海螵蛸粉碎，去渣，过120目筛。（5）取（1）、（4）项细粉与（2）、（3）项稠膏照制颗粒法制粒，压片，即得。每片重约0.5g（相当原药材0．8g）。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
