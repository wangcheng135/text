序号：14571
名称：华佗救脱阳方
组成：葱白1握，附子1个（重1两，切8片），白术5钱，干姜5钱，木香2钱。
出处：《景岳全书》卷五十八。
主治：寒中三阳，口噤失音，四肢强直，挛急疼痛，似乎中风及厥逆，唇青囊缩，无脉或卒倒，尸厥脱阳等证。
加减：None
功效：None
用法用量：先用葱白1握微捣碎，炒热用布包，熨脐下，以二包更替熨之，甚者仍炙气海、关元20-30壮，脉渐出，手足渐温，乃可生也；次用附子1个（重1两），切8片，白术、干姜、木香，同用水2钟煎1钟。候冷灌服，须臾又进1服，或煎服回阳等汤。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
