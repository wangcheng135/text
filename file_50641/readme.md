序号：50656
名称：荆防散
组成：荆芥1两半，防风1两半，米仁5两（炒），通草4两，川芎8钱，茯苓3两，陈皮1两半，香附1两半（醋炒），紫苏1两半。
出处：《医略六书》卷三十。
主治：产后离褥太早，冒风致湿，经络不能流通，遍体浮肿，脉浮涩者。
加减：None
功效：None
用法用量：每服3钱，生姜皮汤送下。
制备方法：上为散。
临床应用：None
各家论述：荆芥理血疏风，防风疏风燥湿，米仁健脾气以渗周身之湿，紫苏理血气以泄遍体之风，川芎入血海行血滞；香附调气海行气滞；通草利湿通肺气；茯苓渗湿和脾气；陈皮利中气调胃气。为散姜皮汤下，使风邪和解，则经络清和，脾胃健运，湿无不化，何遍身浮肿有不退乎。
用药禁忌：None
附注：None
