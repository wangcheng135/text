序号：33113
名称：万应宝珍膏
组成：淡附子2两，鲜红花2两，血余2两，莪术2两，桂枝2两，羌活2两，独活2两，白僵蚕2两，秦艽2两，麻黄2两，当归2两，制川乌2两，防风2两威灵仙2两，制草乌2两，生大黄2两，赤芍2两，生山栀2两，桃仁2两，三棱2两，白芷2两，淡全虫2两，五加皮2两，良姜2两，生地4两，香附4两，乌药4两。
出处：《全国中药成药处方集》（杭州方）。
主治：跌打损伤，负重闪腰，筋骨疼痛，腿膝痠软，瘫痪风痹，手足麻木；以及痈疽发背，痰核流注，胸胃气痛，腹胀寒痛。
加减：None
功效：None
用法用量：临摊时每1斤药油再加香料2两，肉桂粉5钱，苏合香油5钱，同药油搅匀，摊红布上。用时烘热，按症各贴患处。
制备方法：用麻油15斤，入药熬枯去滓，熬至滴水成珠，加章丹60两收膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
