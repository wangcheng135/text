序号：50206
名称：茱萸内消丸
组成：吴茱萸（汤洗7次，焙）2两，陈皮（去白）2两，山药（焙）2两，川楝（蒸，去皮核）2两，山茱萸（去核）2两，青皮（去白）2两，茴香（炒）2两，马蔺花（醋炙）2两，肉桂（去粗皮，不见火）2两，木香（不见火）1两。
出处：《局方》卷五。
主治：肾与膀胱经虚，为邪气所搏，结成寒疝，伏留不去，脐腹（疒丂）刺，小肠气痛，奔豚痃癖，疼不可忍，阴核偏大，肤囊痈肿，结硬牵急，重大滋长，瘙痒疼痛，时出黄水，疮疡，腿沉重，足胫肿满，行步艰难。
加减：None
功效：补虚消疝，温养肾经。
用法用量：每服30-50丸，空心温酒、盐汤任下。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：《保命歌括》有破故纸，无山药。
