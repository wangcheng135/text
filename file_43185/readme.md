序号：43200
名称：三棱煎丸
组成：杏仁（汤浸，去皮尖，麸炒黄色）1两，硇砂（飞，研）1两，神曲（碎，炒）3两，麦糵（炒）3两，青皮（去白）2两，干漆（炒）2两，萝卜子（微炒）2两，三棱（生，细锉，捣罗为末）8两（以酒3升，石器内熬成膏）。
出处：《局方》卷三。
主治：食积痰饮，阻于中脘，气机不宣，脘痞腹胀，噫气不畅，呕吐痰涎，食欲不振，大便或溏或泻或秘；脾虚，为肉食所伤，停久不散，发为腹满膨痛；宿食积聚，翻吐酸秽；膨满，食积气块，伤食夹脐痛甚；癥瘕。
加减：None
功效：顺气宽中，消积滞，化痰饮，消胀软坚。
用法用量：每服15丸至20丸，食后温米饮送下。
制备方法：上为末，以三棱膏匀搜和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：大三棱煎丸（《古今医鉴》卷六）。《得效》：加阿魏五钱重，名阿魏丸，又名起祖三棱丸。《医方类聚》引《经验秘方》“三棱煎丸”有蓬莪术、木香。
