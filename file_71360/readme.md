序号：71376
名称：硫黄丸
组成：硫黄半两（细研），硼砂半两（不夹石者，细研），木香半两（为末），巴豆（去皮）49粒（取萝卜2枚，四破开，钻49窍，每窍纳巴豆1枚，依旧合之，藏在1尺深土坑中，49日后取出巴豆细研如膏，纸裹压去油后研入药中）。
出处：《圣惠》卷四十八。
主治：积聚气，多年不消，变成劳证，腹内结块疼痛，两胁胀满，常吐清水，食饮不下。
加减：None
功效：None
用法用量：每服5丸，空心以温酒送下，晚食前再服。以利为度。
制备方法：上取萝卜1枚，剜作坑子，纳入硫黄、硼砂，以萝卜盖头，用纸一重裹，以好黄泥固济，晒干，用大木火煅令通赤，候冷去泥取药与萝卜一同研细，入木香末及研了巴豆令匀，以醋煮面糊和丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
