序号：68084
名称：黄耆汤
组成：黄耆（薄切）1两，犀角（镑）1两，白茯苓（去黑皮）1两半，人参1两半，柴胡（去苗）1两，升麻1两，秦艽（去苗）1两，芎1两，木香1两，桑根白皮（锉）1两，枳壳（去瓤，麸炒）1两，防风（去叉）1两，芍药1两，黄芩（去黑心）1两，肉豆蔻（去壳，炒）1两，天麻1两，鳖甲（醋浸，炙，去裙襴）1两，地骨皮1两，甘草（炙，锉）1两，羌活（去芦头）3分，当归（切，焙）3分，青橘皮（去白，切，炒）3分，槟榔（锉）半两，桔梗（去芦头，炒）半两。
出处：《圣济总录》卷十三。
主治：风消，气血虚弱。
加减：None
功效：None
用法用量：每服3钱匕，以水1盏，煎至7分，去滓，空腹温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
