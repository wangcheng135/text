序号：46481
名称：济坤大造丸
组成：紫河车1具（制），人参1两（切片，焙干，研细和入），天冬（去心）1两，麦冬（去心）1两，当归1两，淮牛膝1两，山药1两，熟地4两，杜仲（姜汁同盐炒）5钱，黄柏5钱，五味子5钱。
出处：《产科心法》引丹溪方。
主治：气血本虚，不能摄元成孕；或濒堕胎，及生子不寿；成孕后虚热自汗，食少带多。
加减：如虚弱多汗，加黄耆2两（蜜炙），地骨皮、知母各1两；如脾胃虚，常大便溏泻，加白术2两，莲子2两（俱炒）；如少睡惊悸者，血少也，加炒枣仁、桂圆肉各2两。
功效：益气血，补子宫，种子。
用法用量：None
制备方法：上药各为末，同紫河车捣匀为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
