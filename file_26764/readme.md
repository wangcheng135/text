序号：26772
名称：延龄育子丸
组成：天门冬（去心）5两，麦门冬（去心）5两，怀生地黄5两，怀熟地黄（肥大沉水者）5两，人参（去芦）5两，甘州枸杞子（去梗），菟丝子（洗净，酒蒸捣饼，晒干）5两，川巴戟（去心）5两，川牛膝（去芦，酒洗）5两，白术（陈土炒）5两，白茯苓（去皮，牛乳浸，晒）5两，白茯神（去皮心，人乳浸，晒）5两，鹿角胶（真者）5两，鹿角霜5两，柏子仁（炒，去壳）5两，山药（姜汁炒）5两，山茱萸（去核）5两，肉苁蓉（去内心膜）5两，莲蕊（开者不用）5两，沙苑蒺藜（炒）5两，酸枣仁（炒）2两，北五味子（去梗）2两，石斛（去根）2两，远志（去芦，甘草灯心汤泡，去心）2两。
出处：《医便》卷一。
主治：少年斫丧，中年无子，妇人血虚，不能孕育。
加减：None
功效：None
用法用量：每服男人90丸，妇人80丸，空心滚白汤送下。
制备方法：上各为末，将鹿胶以酒化开，和炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《医方考》：男女媾精，乃能有孕。然精者，五脏之所生，而藏之肾者也。故欲藏精于肾者，必调五脏，五脏盛而精生矣。是方也，人参、五味，天麦门冬补肺药也；茯神、远志、柏仁、枣仁、生地补心药也；白术、茯苓、山药、石斛补脾胃也；熟地、枸杞、菟丝、巴戟、牛膝、茱萸、苁蓉、沙苑蒺藜补肝肾也；鹿角胶，血气之属，用之所以生精；角霜，莲须收涩之品，用之所以固脱。如是则五脏皆有养而精日生，乃能交媾而宜子矣。
用药禁忌：忌煎、炙、葱、蒜、萝卜。此方南人服效。
附注：延龄育子方（《医方考》卷六）。上药二十四味，合二十四气；一百单八两，合一年气候之成数，为生生不息之妙。
