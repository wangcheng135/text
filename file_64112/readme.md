序号：64128
名称：猪肝脯
组成：豮猪肝1具（割去苦胆，连血存之，不用水洗，竹刀割净白筋膜，切成柳叶薄片听用），南谷精草2两（以手断碎，不见铁，黄酒淘去泥土，又黄酒泡透听用），枸杞子7钱（黄酒泡透听用），甘菊花1两（去梗、蒂、叶、尘土净，黄酒泡透听用），玄参5钱（不见铁器，黄酒泡透听用），真秋石2钱（为细末，听用）。
出处：《眼科秘诀》卷二。
主治：翳障。
加减：None
功效：None
用法用量：每日吃5-7次，每次温热吃十数多片，不可太多。5、6、7旬以外，加人乳1碗，参汤1茶钟，酒2茶钟，当归汤2钟。11月、12月、正月、2月可用。
制备方法：上6味，除秋石外，将上五味合一处调匀，分作5份，肝亦分五份，用黑薄皮瓷罐子1个，底加一层药，药上排一层肝，肝上撒秋石，又药一层，肝一层，肝上仍加秋石末，如此3-4层，上用药盖肝，加酒1碗泡肝；用白净布1块水湿，布内夹纸5-6层，封固罐口，麻线扎住，入锅内重汤煮1日，锅内水耗，时时加热水，其水不直入罐内；候肝香气外闻，取开；当肝内无嫩血色，住火；俟火气尽，取出，即细嚼慢咽十数片；待冷，取出肝来，去药，将药收之，还煮1具肝，其肝瓷碗盛着。
