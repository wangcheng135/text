序号：31309
名称：沉香导气散
组成：沉香2钱半，槟榔2钱半，人参半两，诃子肉半两，大腹皮（锉，炒）半两，乌药（锉）1两，麦糵（炒）1两，白术1两，神曲（炒）1两，厚朴（姜制）1两，紫苏叶1两，香附（炒）1两半，姜黄4两，红皮4两，炙甘草4两，京三棱（炮）2两，广荗（炮）2两，益智仁2两。
出处：《医学发明》卷三。
主治：一切气不升降，胁肋刺痛，胸膈痞塞。
加减：None
功效：None
用法用量：每服2钱，食前沸汤点服。
制备方法：上为极细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：沉香升气散（《御药院方》卷四）、沉香升降散（《奇效良方》卷十五）。本方改为汤剂，名“沉香升气汤”（见《便览》）；《医学纲目》有红花。
