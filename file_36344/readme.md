序号：36354
名称：和太师牛黄丸
组成：石燕1两，蛇黄1两，磁石（上3味并火烧醋淬9遍，细研）1两，雄黄（研，飞）1两，辰砂（研，飞）1两，石绿（研，飞）1两，牛黄半两，粉霜（研）半两，轻粉（细研）半两，麝香（细研）半两，银箔（研）100片，金箔100片（为衣）。
出处：《局方》卷一。
主治：卒暴中风，眩运倒仆，精神昏塞，不省人事，牙关紧急，目睛直视，胸膈喉中痰涎壅塞，及诸痫潮发，手足瘈疭，口眼相引，项背强直。
加减：None
功效：豁痰镇痉，开窍清神。
用法用量：每服1丸，煎薄荷并酒磨下；老人可服半丸；小儿10岁以下，分为4服，蜜水磨下；4岁以下，分为5服；未满1岁，可分为7服。如牙关紧急，以物斡开灌之。
制备方法：上为细末，用酒煮面糊为丸，如鸡头大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
