序号：65356
名称：女贞膏
组成：黄连3两，黄芩3两，黄柏3两，黄耆3两，连翘3两，薄荷3两，山栀3两，山豆根3两，冬青叶1篮（清水洗净），菊花、千里光花、密蒙花。
出处：《遵生八笺》卷十八。
主治：远近烂眩，风翳障眼。
加减：None
功效：None
用法用量：每用银簪脚挑药些少，点眼两眦，1日3次。
制备方法：用长流水，煎浓汁，去滓再熬，下白蜜少许成膏。另用炉甘石（三黄煅过，为细末，以水飞5-7次，净末、）1两，大朱砂5厘、熊胆5厘、血竭5厘，乳香1分、没药1分，真珠1分、琥珀1分、牛黄1分、冰片1分、麝1分，硼砂3分，石斛1钱（蜜煅），胡黄连1钱5分，白丁香1分。共为细末，投入膏内搅匀，入罐，塞口。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
