序号：53623
名称：家传养肝丸
组成：羚羊角（镑，另研）5钱，生地黄（酒浸）1两，熟地黄（酒浸）1两，肉苁蓉（酒洗）1两，甘枸杞子1两，防风（去芦）1两，草决明（炒）1两，菊花1两，羌活1两，当归（酒洗）1两，沙苑蒺藜（炒）1两，楮实子（炒）5钱，羊子肝（小肝叶，煮，焙干，为末）。
出处：《寿世保元》卷六。
主治：肝肾不足，目失荣养，视力减弱昏花，二目艰涩，大眥赤色，迎风流泪，或翳膜不散。
加减：None
功效：补肝血，益肾气。
用法用量：每服50丸，加至70-100丸，早，盐汤下，午，茶下，临卧，酒下，不饮酒人当归汤送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
