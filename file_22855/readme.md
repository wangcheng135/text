序号：22863
名称：观音普济丹
组成：陈徽墨5钱（顶烟无麝者佳，先置烘箱烘软切开，再和后药研磨），百草霜5钱（微烘俟干透细罗），东天麻（透明者）4钱，广木香3钱（忌火）（上3味并忌泡水），飞面3钱（烘干罗净）。
出处：《卫生鸿宝》卷五引汪迈园方。
主治：难产，交骨不开，横生倒养，胎衣不下，子肠努出，胎死腹中；产后中风，血晕，血崩，鼻衄，瘀积，腹痛；妇女月经不调。
加减：None
功效：固气调血，催生。
用法用量：每服1丸，陈老酒送下。
制备方法：上药各为细末，罗去粗头，再入陈墨，细罗，取长流水为丸，每料分49粒，晒干瓷瓶收贮。
临床应用：None
各家论述：None
用药禁忌：忌烟、酒。
附注：None
