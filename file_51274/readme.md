序号：51289
名称：药酒
组成：淫羊藿1斤（去根梗，用叶，将麻布揩去背上毛，用酥油涂炙黄透，斤半取8两），当归8两（酒洗，去须头），仙茅4两（米泔水浸，去赤汁，用黑豆拌，蒸烂，去豆不用），鹿茸2两（酥炙3两5钱，取2两）。
出处：《墨宝斋集验方》卷上引茅鹿门方。
主治：气血不足。
加减：None
功效：固精壮阳，固筋健骨，补精髓，广嗣延年。
用法用量：每服20-30丸，清晨将原酒送下。半月以后见效。或加人参1两。
制备方法：上药各咀片，并入绢袋盛，悬坛内，用陈煮老酒50-60斤，打面封头并口，煮3炷香取起，埋地下7日夜，将药晒干为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌房事1个月。
附注：None
