序号：48282
名称：神效九分散
组成：马钱子4两（去毛皮），麻黄4两（去节），乳香4两（去油），没药4两（去油）。
出处：《春脚集》卷四。
主治：跌打损伤，无论青肿，错折破烂。
加减：None
功效：None
用法用量：遇有受伤人，即与准9分，以无灰老酒调服；外伤处破者，干上；若未破只见青肿，用烧酒调涂。服药后如觉胸中发闹，周身发麻，此是药力行动，勿恐；若受伤甚重，服药不见动静，过1个半时辰，再用无灰酒调服9分；再服后仍无动静，再过个半时辰，再服9分。如此敷服，无论何样重伤，皆能起死回生。
制备方法：上为极细末，收瓷瓶内，勿令泄气。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。此药药力甚大，服者万不可过九分。
附注：九分散（《理瀹》）。
