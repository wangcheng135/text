序号：40085
名称：肾炎汤Ⅰ号
组成：冬瓜子30g，冬瓜皮30g，车前子30g（布包），茯苓皮30g，赤小豆30g，薏苡仁30g，白茅根60g，泽泻12g，苏梗6g，桔梗6g，陈皮6g，旱莲草9g，蝉蜕6g。_x000D_
【功效】利尿消肿。
出处：《临证医案医方》。
主治：急性肾炎。周身浮肿，腰酸痛，小便量少色黄，舌苔薄白，脉浮数。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：冬瓜皮、冬瓜子、车前子、茯苓皮、赤小豆、薏苡仁淡渗利水；茅根、泽泻清热利尿，既消肿又消炎；陈皮、苏便、桔梗理气，加强利尿作用；蝉蜕散风热，善治面部浮肿（风水）；旱莲草凉血益肾。
用药禁忌：None
附注：None
