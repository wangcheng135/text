序号：36363
名称：和气地黄汤
组成：木香1钱，拣桂（去皮）1钱，茯苓（去皮）1钱，白芥子1钱（炒香），白术1分，干山药1分，川芎1分，当归1分（焙），桂花半钱，缩砂仁半钱，甘草半两（炙）。
出处：《宣明论》卷十二。
主治：沉积气结不散。
加减：None
功效：调养荣卫，补顺阴阳。
用法用量：每用蜜2斤，饧饴1斤，温好甜水5升，化匀开，炒前药，并杏仁20枚（去皮尖，洗净，炒香焦，捶碎），湿地黄根（切）长寸许，约取汁半盏，温服。常服以代汤、茶、酒、果。
制备方法：上为细末，入麝香少许（研匀），用数重油纸，或瓷器内密封起。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
