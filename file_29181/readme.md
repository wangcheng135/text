序号：29189
名称：阿胶散
组成：熟干地黄2两（洗，细切，酒浸，焙），艾叶（切，炒黄）1两，当归（洗，切，焙，去芦须）1两，甘草（炙，锉）1两，芍药（洗，锉）1两，阿胶（锉碎，以蚌粉拌和，铫内炒，泡起候冷，筛去蚌粉用）1两，芎（洗，锉）1两。
出处：《卫生家宝产科备要》卷三。
主治：妊娠不问月数深浅，因顿仆，胎动不安，腰腹痛，或有所下，或胎奔上刺心短气。
加减：若虚羸，加黄耆（蜜炙）1两；若胸中冷，逆气，加生姜5片，大枣5枚。
功效：安胎。
用法用量：每服4钱，水1盏半，煎至8分，去滓温服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
