序号：33289
名称：一柴胡饮
组成：柴胡2-3钱，黄芩1钱半，芍药2钱，生地1钱半，陈皮1钱半，甘草8分。
出处：《景岳全书》卷五十一。
主治：阴虚外感，内兼火邪，寒热往来，口中燥渴，妇人热入血室，及时感后阴虚而有潮热者。凡感四时不正之气，或为发热，或为寒热，或因劳因怒，或妇人热入血室，或产后、经后因冒风寒，以致寒热如疟等证，但外有邪而内兼火者。时感后阴虚未复，余邪潮热。四时不正之气，内外俱有火证而疟邪不散。肝燥胃渴。
加减：如内热甚者，加连翘1-2钱；如外邪甚者，加防风1钱；如邪结在胸而痞满者，去生地，加枳实1-2钱；如热在阳明而兼渴者，加天花粉或葛根1-2钱；热甚者，加知母、石膏。
功效：None
用法用量：水1钟半，煎至7-8分，温服。
制备方法：None
临床应用：None
各家论述：《退思集类方歌注》：此大柴胡变局也。去半夏、枳实、姜、枣，加陈皮、甘草调气，生地凉营分之热，如邪结胸而痞满者，仍宜去生地加枳实为妙。
用药禁忌：None
附注：柴胡饮（《会约》卷十）。
