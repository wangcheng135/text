序号：49114
名称：胜灵丹
组成：黄耆1钱，人参3分（如气短及不调而喘者加之），升麻1钱，真漏芦5分（勿误用白头翁），葛根5分，甘草5分，连翘1钱，当归身3分，牡丹皮3分，生地黄3分，熟地黄3分，白芍药3分（如夏月，倍之；如冬月寒证，勿用），肉桂3分（如阴证疡疮，少用；若为阴寒覆盖其疮，用此大辛热之剂去之；烦躁者勿用），柴胡（如疮不在少阳经勿用）、黍粘子3分（无肿不用），昆布3分（若疮坚硬甚者用之），广术3分，京三棱（炮）2分（此2味疮坚硬甚者用之；不硬者勿用），羌活1钱，独活1钱，防风1钱，麦芽1钱，益智仁2钱（如咳吐多者，或吐沫、吐食、胃中寒者加之），黄连（炒）3分，神曲（炒）2分（食不消化者用之），厚朴（姜制）1钱2分（如腹胀加之），黄柏（炒）3分（如有热或腿脚无力加之；若烦躁欲去上衣者，更宜加用）。
出处：《疡科选粹》卷四。
主治：马刀侠瘿从耳下或耳后，下颈至肩上，或入缺盆中属手中少阳经之分野者、其瘰疬在于颊下或至颊车属足阳明经之分野者。
加减：如气不顺，加橘红，甚者加木香少许，量病人虚实消息之；如只在阳明分者，去柴胡、黍粘子；如在少阳分，去独活、漏芦、升麻、葛根，加瞿麦3分；假令在上焦，加黄芩（半酒炒，半生用）；在中焦，加黄连（半酒半生）；在下焦，加酒制黄柏、知母、防己之类，选而用之；如大便不通而滋，其邪盛者，加酒制大黄以利之；如血燥而大便干燥者，加桃仁泥、大黄；如风结燥不行者，加麻仁、大黄以润之；如风涩而大便不行，加煨皂角仁、大黄、秦艽以利之；如气涩而大便不通者，加郁孪仁、大黄以除风燥；如阴寒证，为寒结秘而大便不通者，以《局方》半硫丸，或加炮附子、干姜煎成，候冰冷服之。
功效：None
用法用量：每服3钱，白汤送下。
制备方法：上为细末，汤浸蒸饼，捏作饼子，晒干，捣如米粒大。
临床应用：None
各家论述：黄耆护皮毛，实腠理，活血气，实表补元气，乃疮家之圣药也；人参补肺气；升麻、真漏芦、葛根三味俱足阳明经药；甘草能调中，和诸药，泻火益胃；连翘走十二经，凡疮中之药不可无者，能散血结与气聚，此疮家之神药也；牡丹皮去肠胃中留滞宿血；生地黄、熟地黄，诸经中和血、生血、凉血药也；白芍药其味酸，其气寒，能补中益气；肉桂大辛热，能散结聚；柴胡功同连翘；昆布味咸，咸能软坚也；麦芽能消食健胃；黄连治烦闷。
用药禁忌：None
附注：方中柴胡用量原缺。
