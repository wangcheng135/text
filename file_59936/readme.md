序号：59951
名称：调胃散
组成：藿香2两，甘草（炙）2两，陈皮（去白）2两，半夏曲（每1两用生姜3两半）2两，厚朴（每1两用姜1两拌制）2两。
出处：《御药院方》卷三。
主治：阴阳气不和，三焦痞隔，五劳七伤，山岚瘴气，八般疟疾，四时伤寒，头目肢节疼痛，心腹胀满，呕逆恶心，痰涎咳嗽，手足虚肿，五种隔气噎塞，寒热水泻诸痢，妇人产后蓐劳，脾胃不和，饮食减少。
加减：None
功效：None
用法用量：每服2分，水1盏，入生姜2片，同煎至7分，和滓温服，不拘时候。
制备方法：上同为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方为细末，生姜面糊和丸，如梧桐子大，名“调胃丸”。每服五十丸，食前生姜汤送下。
