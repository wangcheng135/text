序号：64877
名称：续断止血方
组成：续断1两，当归1两，桂心1两，干姜4两，干地黄4两，甘草2两，蒲黄1两，阿胶1两。
出处：《千金》卷二十。
主治：下焦虚寒损，或先便转后见血，此为远血，或便或不利，好因劳冷而发。
加减：None
功效：None
用法用量：上(口父)咀。以水9升，煮取3升半，去滓，下胶取烊，下蒲黄，分3服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：先便后血为远血，《金匮》主以黄土汤，专取术、附、灶土以破瘀结，胶、地、甘草以和营血，黄芩以化术、附之热。《千金》以血既下脱，不须复用破结之剂，乃以姜、桂代术、附，归、续代灶土，蒲黄代黄芩，虽用法稍平，而功用不殊。然须验其血色，晦淡则当用《金匮》法，鲜紫当用《千金》法，方为合辙。
用药禁忌：忌海菘菜、生葱、芜荑。
附注：止血汤（《圣济总录》卷四十）。本方方名，《外台》引作“续断止痢汤”，《普济方》引作“续断散”。
