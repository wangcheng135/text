序号：22740
名称：更生散
组成：钟乳2两半，白石英2两半，海蛤（各研）2两半，赤石脂2两半，防风2两半，栝楼2两半，干姜1两半，白术1两半，桔梗5分，细辛5分，人参3分，附子（炮，去皮）3分，桂心3分。
出处：《千金翼》卷二十二。
主治：男子、女人宿寒虚羸，胸胁逆满，手足烦热，四肢不仁，食饮损少，身体疾病，乍寒乍热，极者着床40-50年，服众药不愈。
加减：加硫黄，名“邵靳散”。
功效：None
用法用量：温酒和服1薄，须曳起行，随力所往，还欲坐卧，随意着衣乃卧，适体中所便，食时乃冷，不得热食，只得大冷。服药后20日复饮热食及房室，可渐随意，唯服药时不得耳。若头面中愦愦者，散发，风中梳百余遍，1日3饮5合酒讫，日下晡渴，便饮酒噉脯饭，常令体中曛曛有酒势；手足顿热，可冷水洗之。
制备方法：上药皆须新好州土者，捣筛为散，囊盛4两为8薄。
临床应用：None
各家论述：None
用药禁忌：忌食猪肉、羮臛、汤面，不得房室；忌猪肉、冷水，生菜、生葱、桃、李、雀肉。
附注：方中白石脂，《外台》作“白石英”。
