序号：3515
名称：太和丸
组成：人参（去芦）5钱，白术（去芦，土炒）4两，白茯苓（去皮）半两，陈皮1两，半夏（面炒）2两2钱，枳实（麸炒）1两，黄连（姜汁炒）1两，当归（酒洗）1两，山楂（蒸，去子）1两，木香5钱，白芍（酒炒）1两半，香附（童便炒）1两，神曲（炒）1两半，麦芽（炒）1两半，白豆蔻（去壳）1两3钱，龙眼肉1两3钱，大粉草（炙）7钱。
出处：《回春》卷二。
主治：元气、脾胃虚损，不思饮食，肌体羸瘦，四肢无力，面色萎黄。
加减：None
功效：补气生血，健脾养胃，开胸快膈，清郁化痰，消食顺气，平和调理。
用法用量：每服100丸，不拘时候，米汤送下。
制备方法：上为末，荷叶1个煎汤，打仓米糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
