序号：25858
名称：收膜汤
组成：生黄耆1两，人参5钱，白术5钱（土炒），白芍5钱（酒炒焦），当归3钱（酒洗），升麻1钱。
出处：《傅青主女科》卷下。
主治：妇人产前劳役过伤，又触动恼怒，以致肝不藏血，血亡过多，产后阴户中垂下一物，其形如帕，或有角，或二岐，往往出产门外者，至六七寸许，且有粘席干落一片，如手掌大者。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：或疑产后禁用白芍，恐伐生气之源，何以频用之而奏功也。是未读仲景之书者。嗟乎！白芍之在产后不可频用者，恐其收敛乎瘀也，而谓伐生气之源，则误矣。况病之在肝者，尤不可以不用，且用之于大补气血之中，在芍药亦忘其为酸收矣，又何能少有作祟者乎？矧下坠之物，正借酸收之力，助升麻以提升气血，所以奏功之捷也。大补其气与血，而少加升提之品，则肝气旺而易生，肝血旺而易养，肝得生养之力，而垂下之物自收。
用药禁忌：收垂下之物全赖白芍之功，不可用炭。
附注：收脂汤（《辨证录》卷十二）。
