序号：66063
名称：蛇衔生肉膏
组成：蛇衔6分，当归6分，干地黄3两，黄连1两，黄耆1两，黄芩1两，大黄1两，续断1两，蜀椒1两，芍药1两，白及1两，芎1两，莽草1两，白芷1两，附子1两，甘草1两，细辛1两，薤白1把。
出处：《千金》卷二十二。
主治：痈疽，金疮败坏。
加减：None
功效：None
用法用量：上（口父）咀，酢渍再宿，以腊月猪脂7升煎，三上三下，酢尽下之，去滓。敷之，日3夜1。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：蛇衔生肉膏中附子，即前方身啄之意；蛇街去风不逮，更加之以莽草、附子；散坚不逮，更加之以蜀椒；然恐遗热为患，故配之以芩、连、大黄。
用药禁忌：None
附注：None
