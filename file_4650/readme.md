序号：4650
名称：木香顺气丸
组成：陈皮96两，乌药96两，枳实（炒）96两，槟榔96两，厚朴（炙）96两，枳壳（炒）96两，香附（炙）96两，黄芩96两，青皮（醋炒）48两，黑丑（炒）48两，大黄48两，桔梗48两，木香48两，三棱（炒）24两，莪术（炙）24两，山楂96两，官桂12两，甘松12两，吴茱萸（炙）12两。
出处：《北京市中药成方选集》。
主治：气滞不舒，胸膈痞闷，腹胁胀满，大便不利。
加减：None
功效：舒气开郁，化滞通便。
用法用量：每服1钱至2钱，温开水送下。
制备方法：上为细末，过罗，用冷开水泛为小丸。
临床应用：None
各家论述：None
用药禁忌：年老气虚勿服；孕妇忌服。
附注：None
