序号：46193
名称：洗眼蚕茧
组成：黄连3钱，菊花3钱，金银花3钱，当归尾3钱，防风3钱，红花2钱，荆芥穗2钱，胆矾2钱，蕤仁2钱，蝉蜕2钱，蜀椒5分，冰片2分。
出处：《天津市固有成方统一配本》。
主治：暴发火眼，眼边赤烂，眼睑肿痛，迎风流血，羞明畏光，视物昏蒙，目眦涩痒。
加减：None
功效：散风清热，明目退翳。
用法用量：将药用新针刺数孔，以开水1杯浸药，乘热先熏，后用药棉蘸药水擦洗。洗眼器皿要保持清洁。
制备方法：冰片单包，将黄连等11味共轧为粗末，将冰片置乳钵内研细，再与黄连等粗末陆续配研和匀过罗。分装：先用白纸包成鸡心形，再用丝棉包严，用绳扎紧。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
