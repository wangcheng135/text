序号：73574
名称：福寿保生丸
组成：芦荟1两，木香1两，芜荑仁7两半，使君子7两半，砂仁7两半，胡黄连7两半，神曲7两半，广荗7两半，三棱（炒）7两半，陈皮7两半，干虾蟆7两半。
出处：《普济方》卷三八○引《德生堂方》。
主治：小儿一切脾疳癖积，饮食无时，诸物所伤脾胃，因食寒热，积作成气块，腹肚胀满，喜怒不常，坚硬为脾，身热虚汗，而且多睡，或黑或瘦，口鼻生疮。
加减：None
功效：None
用法用量：每服5-7丸，温米汤送下，量岁数加服，1日2次。8-9日后，大便黑为验。
制备方法：上为细末，水糊为丸，如粟米大。
临床应用：None
各家论述：None
用药禁忌：切忌荤、生冷、硬物。
附注：None
