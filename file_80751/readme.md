序号：80767
名称：麋茸丸
组成：麋茸1两（治如鹿茸，无麋茸以鹿茸代），舶上茴香半两（炒香），菟丝子（酒浸晒干，用纸条子同碾取末）1两。
出处：《本事》卷二。
主治：肾经虚，腰痛不能转侧。
加减：None
功效：None
用法用量：每服30-50丸，空心温酒、盐汤送下。
制备方法：上为末，以羊肾2对，法酒煮烂，去膜，研如泥，用上药为丸，如梧桐子大，阴干如肾膏，少入酒糊佐之。
临床应用：None
各家论述：《本事方释义》：麋乃泽兽也，气味甘温，入足少阴；菟丝子气味甘平，入脾肾；舶上茴香气味辛温，入肝肾；羊内肾气味辛甘温，入足少阴。此虽肾虚腰痛，必有水气阴湿之邪相感而起，故补肾药中必兼补脾之药。
用药禁忌：None
附注：None
