序号：77771
名称：雄朱丹
组成：朱砂2两，雄黄2两（上用沙盒子1个，先以牡丹皮2两，内外熏令黄色，入前药在内，用酽米醋和腊茶作饼子盖定，以赤石脂固盒子口缝，又用赤石脂泥裹盒子一重，再用黄泥纸筋又裹一重，约一指厚，先以草火烧令盒子极干，再用5斤火渐渐添至一秤，候火力渐消，取出，掘地坑1尺，埋1宿去火毒，取出细研续入后药），附子（炮裂，去皮脐，别为细末）1两，胡椒1两，官桂（去皮）1两，赤石脂1两，木香1两，沉香1两，荜茇1两，丁香1两，白术1两，乳香半两（与赤石脂同研细）。
出处：《百一》卷一引钱观文方。
主治：宿寒痼冷，饮食呕逆，经隔5-7年即疲瘠异形，变为劳瘵。
加减：None
功效：None
用法用量：每服10丸，空腹、食前以温酒或盐汤送下。
制备方法：上为细末，入前煅药同研匀，却以清酒2升3合，熬去2分，入附子末煮成糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
