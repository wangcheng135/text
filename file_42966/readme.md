序号：42981
名称：保婴育生丸
组成：银花5钱，连翘5钱，橘红5钱，贝母5钱，僵蚕（炒）3钱，竺黄3钱，天麻3钱，法半夏3钱，防风3钱，胆星3钱，钩藤3钱，焦三仙1两，黄芩4钱，甘草5钱，芥穗2钱，薄荷2钱，活蝎子12个，桑叶5钱（上除活蝎外，为末。），雄黄5钱，琥珀4钱，冰片2钱，麝香1钱。
出处：《北京市中药成方选集》。
主治：小儿外感风寒，停食发烧，咳嗽痰盛，急热惊风。
加减：None
功效：清热退烧，化痰消滞。
用法用量：每服1丸，温开水送下，1日2次，3岁以下者酌减。
制备方法：上和匀，炼蜜为丸，重5分，朱砂为衣，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
