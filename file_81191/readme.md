序号：81207
名称：麝香苏合丸
组成：苏合香油5钱（入安息香膏内），安息香1两（另为末，用无灰酒半升熬膏），丁香1两，青木香1两，白檀香1两，沉香1两，荜拨1两，香附米1两，诃子（煨取肉）1两，乌犀（镑屑）1两，朱砂（研，水飞）1两，薰陆香5钱，麝香7钱半。
出处：《准绳·幼科》卷九。
主治：一切邪神及胸膈噎塞，肠中虚鸣，宿食不消，传尸骨蒸，诸项劳瘵，卒暴心痛，鬼魅疟疾，霍乱吐泻、赤白下痢，小儿惊搐。
加减：None
功效：None
用法用量：小儿1丸，老人4丸，空心用沸汤送下，酒送下亦可。用蜡纸裹1丸，弹子大，用绊绢袋盛，常带之，一切邪神不敢近。
制备方法：上为细末，用安息香膏入炼蜜和剂为丸，如芡实大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
