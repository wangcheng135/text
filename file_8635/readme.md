序号：8635
名称：玄英散
组成：川朴消5斤（瓦瓶烧令通赤，细研如粉），淡竹沥1升。
出处：《圣惠》卷九十五。
主治：None
加减：None
功效：祛风热，利三焦，耐寒暑、驻容颜。
用法用量：每日食后，水调1钱服之。服至2-3斤，渐耐寒暑少汗；服至5-7斤，驻颜色，去百病，一生无汗，夏月可以衣裘，冒炎毒，履冰雪，无惧矣。
制备方法：上将竹沥拌消，令匀湿，用大竹筒1枚，先以牡蛎粉半斤，筑入筒中，次下消后，又以牡蛎粉半斤筑之，以蜡纸3重封之，勿令通气，安在甑中，四面以黑豆埋之，令没筒口，蒸1复时，待冷，去豆开筒，去牡蛎粉，取消细研如粉。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
