序号：48421
名称：神效活络丸
组成：薪蛇（酒炙）3两，甘草（炙）3两，大黄（酒制）3两，草豆蔻（炒）3两，黄芩3两，白芷3两，川芎（酒蒸）3两，藿香3两，白附子（制）3两，麻黄3两，香附（酒醋炒）3两，羌活3两，何首乌（酒蒸）3两，沉香3两，熟地黄3两，威灵仙2两4钱，天麻（姜制）3钱，当归2两4钱，僵蚕（姜水炒）3两，葛根2两4钱，青皮（醋炒）1两5钱，玄参1两5钱，白术（土炒）1两5钱，天竺黄1两5钱，虎骨（酒炙）1两5钱，茯苓1两5钱，赤芍（酒炒）1两5钱，细辛1两5钱，龟版（醋炙）1两5钱，骨碎补1两5钱，豆蔻（炒）1两5钱，木香3两，黄连（炒）9钱，全蝎7钱2分，地龙肉（炒）9钱，秦艽1两5钱，防风4两5钱，肉桂3两，桑寄生1两5钱，朱砂2两5钱，没药（醋炙）2两4钱，血竭1两2钱，乳香（醋炙）1两5钱，安息香1两5钱，松香7钱5分，冰片4钱2分，牛黄4钱2分，丁香1两5钱，犀角4分7厘5毫，麝香3分。
出处：《中药制剂手册》引广州陈李济制药厂方。
主治：风痰入络引起突然中风，痰厥，左瘫右痪，肢体麻木，以及痿软无力，痹症疼痛，手足拘挛。
加减：None
功效：舒筋活络，祛风豁痰。
用法用量：每丸重1钱2分，每服1丸，每日2-3次，温开水送服，噙化亦可。
制备方法：以上熟地黄、当归单放，朱砂至麝香11味单包，将朱砂研为极细粉，犀角锉研为细粉，麝香、牛黄、冰片先后研细过100～120目细罗，乳香、没药、血竭、松香、安息香分别轧为细粉，丁香轧细，熟地黄、当归用酒蒸透，干燥后与蕲蛇等37味，共轧为细粉，和匀过80～100目细罗，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
