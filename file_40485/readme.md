序号：40495
名称：虎骨胶丸
组成：虎骨2斤（锉碎，洗净，用嫩桑枝10两，金毛狗脊去毛10两，白菊花去蒂10两，秦艽2两，煎水熬虎骨成胶，收起如蜜样，如不足量，加蜜炼），大熟地4两，当归3两，牛膝2两，山药2两，茯苓2两，杜仲2两，枸杞2两，续断2两，桑寄生2两，熟附子7钱，厚肉桂（去皮，不见火）5钱，丹皮，泽泻8钱，人参2两（贫者以黄耆4两代之）。
出处：《济阳纲目》卷八十一。
主治：鹤膝风及瘫痪诸证。
加减：None
功效：None
用法用量：每早服3钱，开水送下。
制备方法：上为末，虎骨胶为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：虎骨膏丸（《医学心悟》卷三）。方中丹皮用量原缺。
