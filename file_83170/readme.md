序号：83188
名称：乌龙膏
组成：当归2两，白及2两，连翘2两，蝉蜕2两，大红扛2两，独活1两，羌活1两，川乌1两，草乌1两，细生地4两，血余4两，大黄4两，银花4两，番木鳖4两，麻黄1两5钱，泽兰1两5钱（上各药切片熬膏），全蝎2两，穿山甲2两，蛤蚆50只（活，放油内），瞎地鳖蛇两条（活，放油内），蜈蚣100条（大者，须活者）。
出处：《良方合璧》卷下。
主治：痈疽发背，对口搭手，一切无名肿毒。
加减：None
功效：去腐止痛，拔毒收敛。
用法用量：恶疮未成者、贴之即消；已成者，贴之即溃。
制备方法：上用麻油5斤，桐油8两，入锅内，并桃30段，柳30段，桑枝30段，每段长3寸许，生姜8两、葱8两，将枝煎枯取出，乃将瞎地鳖蛇放入锅内，急将锅盖揿住，蛇在油内跳跃不止，至不动时，又入活蛤蚆，然后，将山甲、全蝎、蜈蚣，并前药16味，熬至药俱枯黑，乃滤去渣，将锅拭净，再以密绢仍滤油入锅，用文武火熬至滴水成珠，将锅离火，再入上好洋丹3斤，以一手下丹，一手持硬木棍，不住手搅匀成膏，再入后药：乳香3两，没药3两，去油，麝香5钱，冰片5钱，4味另研，徐徐添入，搅匀成膏，收贮听用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
