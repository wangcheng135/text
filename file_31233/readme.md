序号：31242
名称：沉香阿魏丸
组成：沉香3分，木香3分，芎半两，当归（锉，焙）半两，蓬莪术（炮）半两，陈橘皮（去白，焙）半两，延胡索半两，槟榔（锉）半两，吴茱萸（醋浸1宿，炒）半两，益智仁半两，桂（去粗皮）半两，白术半两，附子（炮裂，去皮脐）半两，干姜（炮）半两，草豆蔻（去皮）半两，阿魏1两半（瓷器中醋浸1宿，生绢滤去砂，入面煮糊）。
出处：《圣济总录》卷四十五。
主治：脾胃冷热气不和，食毒，脾胃中宿积。
加减：None
功效：None
用法用量：每服5-7丸，以生姜、橘皮汤送下；血气，以温酒醋汤送下。
制备方法：上药捣罗十五味为细末，用阿魏糊为丸，如梧桐子大，丹砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
