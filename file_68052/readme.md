名称：黄耆汤
组成：黄耆3两，槐实1两半，小蓟1两半，桑耳1两半，干地黄（焙）1两半，当归（炙，锉）1两半，黄连（去须）1两半，白芷1两半，草豆蔻2枚（去皮），芎2两，赤石脂2两，天雄（炮裂，去皮脐）2两，龙骨2两，黄芩（去黑心）半两，红蓝花1两，诃黎勒皮1两，延胡索1两，厚朴（去粗皮，生姜汁炙）1两，桂（去粗皮）1两。
出处：《圣济总录》卷一四三。
主治：肠风下血。
加减：None
功效：None
用法用量：上（口父）咀。每服5钱匕，以水1盏半，加生姜1分（拍碎），煎取8分，去滓，食前温服，日2次。
