序号：73691
名称：槐子膏
组成：槐子3两，黄丹24两，头发2两，麻油2斤半，猪脂2斤，蜡5两，水杨白皮3两，桑根白皮1两，皂荚半两（去皮子），巴豆半两（去皮心），天雄1两（去皮脐），当归1两，槐白皮1两，雄黄半两（细研），麝香半两（细研）。
出处：《圣惠》卷六十七。
主治：伤折。
加减：None
功效：None
用法用量：逐日摊贴痛处。极效。
制备方法：上细锉。入脂油内，以慢火煎养1日，焦熟后，用绵滤去滓，于净锅中炒黄丹，令紫色，即下熟药汁，用柳枝搅，不令住手，候药成紫色，滴入水中成珠子，油力尽，即住火，入雄黄、麝香和匀，收于瓷盒中。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
