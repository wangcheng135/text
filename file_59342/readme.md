序号：59357
名称：莲子草膏
组成：莲子草汁3升，生巨胜油1升，生乳1升，甘草1大两（末）。
出处：《近效方》引婆罗门方（见《外台》卷三十一）。
主治：一切风，耳聋眼暗。
加减：None
功效：生发黑发，坚齿延年。
用法用量：每欲点，即仰卧垂头床下，一孔中各点如小豆，许久乃起，有唾唾却，勿咽之。起讫，即啜少热汤饮，点经一年，白发尽黑，秃处并出。
制备方法：上于锅中煎之，缓火熬令鱼眼沸，数搅之勿住手，看上沫尽，清澄滤，不津垍器中贮之；另用青莲蕊6分，龙脑花3分，郁金香2分，并为末，先煎诸药3分减1，次下汁及油等，膏成。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
