序号：3922
名称：无比神应膏
组成：白蔹1两，白及1两，木鳖子仁1两，香白芷1两，官桂1两，杏仁1两，当归1两，乳香1两，没药1两，桂花半两，苏合香1丸，黄丹2斤半，真香油5斤，槐条半斤，柳条半斤。
出处：《普济方》卷三一五。
主治：诸般恶毒疮肿、发背瘤疽、瘰疬、臁疮、脚气、打仆伤损、刀斧伤、汤浇火烧、马、犬、蛇、虫、蜈蚣、蜂、蝎咬伤；多年咳嗽、口内吐血；心疼腹痛、小肠疝气，赤白痢泄不止；牙疼，肉溃流脓，顽癣、腰痛、奶痈、痈痪、杖伤。
加减：None
功效：None
用法用量：贴之；多年咳嗽，口内吐血，贴背；心疼腹痛，小肠疝气，赤白痢泄不止，贴脐下；牙疼，贴腮上。
制备方法：上锉碎，除乳香、没药、黄丹、苏合香丸另研外，其余药于油内浸，春秋5日、夏3日、冬7日。过冬减黄丹3两，新铁锅内浸至日期用文武火熬，一顺搅，槐、柳条各黑色，尽去其滓，放温，入乳香、没药、苏合香丸，将药再熬，不住手搅，微滚2-3沸，放温，下黄丹毕，令文武火熬滚起，出火再滚，如此5-6次，不住手搅至数1000次，烟尽黑色为度，滴水不散方可，切不可过火。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
