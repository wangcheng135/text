序号：27200
名称：当归酒
组成：辣桂（去粗皮）半两，当归4钱，木香2钱，白芷2钱。
出处：《直指》卷二十二。
主治：痈疽阴证，头平向内，沉黯不疼，浑身患处不热。
加减：None
功效：None
用法用量：每服3钱半，醇酒1碗，慢火煎7分，加乳香末半钱，不饥饱温服，以排脓内补散、加味不换金正气散为佐，以熟餔、猪蹄、膂肉为养，荞麦面能起发，可煮食之。如更不发起，用《局方》姜附汤加当归、木香、炙甘草煎服。又更不发起，用穿山甲（头截片，蘸醋，炒焦）、生人牙（煅留性）各1分，为末，分作2服，用辣桂、当归、去节麻黄煎酒，食前调下。患处用生姜汁和面厚涂，或用川乌、硫黄、人牙（煅），并细末，酒调敷之。
制备方法：上锉细。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
