序号：63254
名称：清金宁肺丸
组成：法半夏128两，白及128两，蒌仁128两，麻黄128两，川贝母128两，芭蕉根128两，橘红80两，白芥子（炒）48两，曼陀罗叶48两，五味子（炙）80两，百部80两，苦桔梗80两，杏仁（去皮，炒）80两，甘草80两，冬花32两，桂枝32两，厚朴（炙）32两，地骨皮32两，陈皮32两，茯苓8两，沉香16两，贝母32两。
出处：《北京市中药成方选集》。
主治：肺气不清，咳嗽痰盛，喘促胸满，睡卧不宁。
加减：None
功效：清金宁嗽，化痰止喘。
用法用量：每袋装180粒。分4次服，每日2次，温开水送下。
制备方法：上为细末，过罗，用冷汗水泛为小丸，晒干。每16两用滑石细粉4两为衣，闯亮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
