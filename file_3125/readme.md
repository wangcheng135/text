序号：3125
名称：天麻散
组成：天麻半两，羌活（去芦头）半两，芎半两，防风（去叉）半两，蒺藜子（炒，去角）半两，桂（去粗皮）半两，当归（切，焙）半两，白附子（炮）半两，干蝎（全者，炒）半两，乌头（炮裂，去皮脐）半两，枳壳（去瓤，麸炒）半两，天南星（炮）半两，麻黄（去根节，煎，去沫，焙）半两，地骨皮半两。
出处：《圣济总录》卷一五○。
主治：妇人血风毒气，内外走注，身体皮肤骨节寒热疼痛，燥涩麻木。
加减：None
功效：None
用法用量：每服1钱匕，薄荷汤调下，温酒亦得，不拘时候。
制备方法：上为散，研入麝香半钱，和匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
