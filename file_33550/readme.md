序号：33559
名称：灵宝丸
组成：天麻（洗）1两，天南星1两，白附子1两，独活1两，白僵蚕1两，川乌头（炮）1两，羌活（洗）1两，干蝎（全者）1两，牛黄1分，龙脑1分（研细），麝香半两（研细，旋入众药）。
出处：《博济》卷四。
主治：小儿疥疮，及三十六种风疾。
加减：None
功效：None
用法用量：诸色风疾，每服5丸，薄荷、温酒送下；女人血风，更入少当归末，温酒送下；如瘫痪风，下床不得，每服30丸，先用白矾半两为末，葱10茎，煎汤温浴，后用薄荷汁、温酒送下；衣被盖出汗，别服补药；如是男子、妇人疥癣、瘰疬，并须依前法洗浴，服30丸，出汗，当日必愈；小儿疮疥亦须如常浴，每1岁1丸，并须出汗，并愈。
制备方法：上净洗，日内晒干，不用近水，杵为细末，炼蜜为丸，如豌豆大。
临床应用：None
各家论述：None
用药禁忌：忌食面、猪肉、鱼、毒等物。
附注：None
