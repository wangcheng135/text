序号：3723
名称：开郁汤
组成：香附（童便浸，炒）1钱半，贝母1钱半，苍术1钱，抚芎1钱，神曲（炒）1钱，山栀（炒）1钱，陈皮（去白）1钱，茯苓1钱，枳壳（去瓤，麸炒）1钱，苏梗1钱，甘草3分。
出处：《医便》卷三。
主治：恼怒思虑，气滞而郁。
加减：有痰，加半夏、南星各1钱；有热，加黄芩、黄连各8分，柴胡1钱；血郁，加桃仁、红花各8分；湿，加白术、羌活各1钱；气滞，加木香5分、槟榔8分；食积，加山楂、神曲各1钱、砂仁7分。
功效：None
用法用量：上加生姜1片，水2钟，煎1钟，食远服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：《赤水玄珠》有半夏，无苏梗、枳壳。
