序号：40479
名称：虎骨木瓜酒
组成：虎骨胶1两，木瓜4两，红花2两，官桂5钱，独活2两，当归4两，川芎1两，淮牛膝4两，川断1两，天麻1两，玉竹2斤，杞子1两，制乳香5钱，制没药5钱，桑枝4两，桑寄生1两，油松节2两，土烧酒50斤。
出处：《中药成方配本》。
主治：气血不和，风寒湿痹，关节痠痛，手足拘挛。
加减：None
功效：活血祛风。
用法用量：随量饮服，但每日至多不得超过4两。
制备方法：上切，用红曲8两，一并盛入夏布袋内，用五十度土烧酒浸10天，取出药滓，加以压榨，仍入原酒内，再浸10天，然后取出药滓，榨净去滓，将酒滤清；用黄酒5斤，微火炖暖，先将虎骨胶烊入，再将白砂糖100两，徐徐烊入滤清，和入前酒中，约成酒53斤，装瓶封口。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
