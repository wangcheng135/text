序号：42743
名称：保安丸
组成：巴豆（去皮心后，1两，研细，纸裹去油，入药内同研）半两，青橘（去白，切作片子，炒令转色）1两1分，黄连（去毛，锉，炒令紫色）1两，蓬莪术（锉，炒令黄色）1两，干姜（炮裂，切细，再炒少时）1两。
出处：《幼幼新书》卷二十九引《博济》。
主治：男子、女人一切酒食所伤。
加减：None
功效：取积滞，行冷气。
用法用量：常服白汤送下2丸，大人3丸；霍乱吐泻，用煨生姜汤送下5丸，小儿2丸；心气痛，醋汤送下3丸；白痢，干姜汤送下；赤痢，甘草汤送下5丸，小儿1-2丸。
制备方法：上4味同为细末，入前巴豆同研令匀，以米醋糊为丸，如麻子大，用朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
