序号：5646
名称：加减龙胆泻肝汤
组成：龙胆草（酒拌炒黄）1钱，泽泻1钱，车前子（炒）5分，木通5分，生地黄（酒拌）5分，当归尾（酒拌）5分，山栀（炒）5分，黄芩5分，甘草5分。
出处：《外科发挥》卷七。
主治：肝经湿热，阴部生疮，阴囊肿痛，小便赤涩，便毒悬痈，妇人阴挺。
加减：None
功效：None
用法用量：上作1剂。水2钟，煎8分，食前服。
制备方法：None
临床应用：None
各家论述：《济阴纲目》：泻肝而兼导赤，泻其子也；泻肝而用利水，肝主疏泄也，龙胆、山栀，假以降火；当归、生地，以滋肝阴；生甘草缓肝之急；炒黄芩助肝之气。
用药禁忌：None
附注：加味龙胆汤（《外科枢要》卷四）、龙胆泻肝汤（《校注妇人良方》卷二十四）。本方方名，《景岳全书》引作“加味龙胆泻肝汤”。
