序号：7746
名称：圣柳散
组成：苍术4两（米泔水浸1宿，焙），连翘1两（焙），甘草1两（炙），大黄半两（切），人参半两（切），赤茯苓半两（切），黄芩2钱半（煅，存性），桔梗半两，白术1分，枳壳1两（麸炒，去白），南木香1两（切），益智1两（去皮，切），白芷半两（焙），苦参1两（切）。
出处：《洪氏集验方》卷二。
主治：肿毒发背，一切痈疽，烦渴不已，疾势转增者。
加减：如渴时，加牙消半两，别研入。
功效：None
用法用量：每服2钱，水1盏，柳枝7寸，同煎至8分，去滓温服，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：忌鸡肉、湿面，疾愈半年方可食。
附注：None
