序号：65190
名称：萎蕤汤
组成：萎蕤2两，青木香2两，白薇（焙）2两，麻黄（去根节，煎去沫，焙）2两，独活（去芦头）2两，杏仁（汤浸，去皮尖双仁，炒）2两，芎2两，甘草（炙）3两，麝香（研）1分，石膏（研）3分。
出处：《圣济总录》卷十二。
主治：中风发热，头痛目眩，咽喉干，舌本强，胸背痛闷，心膈痞满，腰脊强急。
加减：若一寒一热，加朴消1分，烧令白，于湿地，纸衬出火毒，大黄1两，锉，醋炒，量人虚实用之，得下后，即减此二味。
功效：None
用法用量：每服5钱匕，水1盏半，煎至8分，去滓温服，空心、临卧并2服，取汗。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
