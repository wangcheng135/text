序号：13271
名称：人参丸
组成：人参（去芦头）3分，麦门冬（去心，焙）3分，半夏（汤洗7遍去滑）3分，黄耆（锉）3分，川大黄（锉碎，微炒）3分，白茯苓3分，柴胡（去苗）3分，黄芩3分，诃黎勒1两（煨，用皮），甘草半两（炙微赤，锉），鳖甲1两（涂醋炙令黄，去裙襕），芎半两。
出处：《圣惠》卷八十三。
主治：小儿哺露失衣，当风湿冷水浴，苦腹大时痢，或寒热如疟，不欲食，纵食不生肌肉，或不消化，四肢羸瘦。
加减：None
功效：None
用法用量：1-2岁儿每服3丸，以粥饮送下；4-5岁儿，服5丸，1日3次。
制备方法：上为末，炼蜜为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：人参黄耆丸（《圣济总录》卷一七五）。
