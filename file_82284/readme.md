序号：82300
名称：磨积搨气丸
组成：沉香半两，香附子半两，广术半两，半夏半两，麦芽半两，雷丸半两，川椒半两，京三棱半两，神曲半两，枳壳半两，木香3两，萝卜子4两（炒，另研），陈米1升（巴豆7钱半，去壳末一处炒，以米黄色黑色为度，去皮巴豆不用，只以陈米与前药一处用之）。
出处：《普济方》卷一八二引《德生堂方》。
主治：中焦气滞，胸膈痞闷，饮食迟化，呕恶气不升降，痃癖瘕癥，腹内膨胀疼痛；妇人血气之证。
加减：None
功效：None
用法用量：每服25丸，加至30丸，食前温白米汤送下。
制备方法：上末以萝卜5-7个熬水和药为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
