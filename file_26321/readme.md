序号：26329
名称：朴麝散
组成：朴消（研）5两，麝香（研）1分，甘草（炙）1分，淡竹叶1握，黄芩（去黑）半两，芦根（锉）1两，山栀子（去皮）1两。
出处：《圣济总录》卷一一九。
主治：重舌，及天行阴黄，丹石发动，一切热毒。
加减：None
功效：None
用法用量：每服半钱匕，冷水调服，绵裹咽津亦得。
制备方法：上七味，铁五味如麻豆大，入研者二味，以新汲水3升，将药入档中煎约半升，去滓澄清，取瓷瓶1个，倾药汁在内，以物盖定瓶口，用盐泥固济，慢火煅一复时，去火放冷却，将瓶安在水中，无令水至瓶口，浸经一宿，至明日打破瓶子，取药如金色，研为细散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
