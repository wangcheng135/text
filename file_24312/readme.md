序号：24320
名称：人参鳖甲散
组成：人参半两，桂心半两，当归半两，桑寄生半两，白茯苓半两，白芍药半两，桃仁半两，熟地黄半两，甘草半两，麦门冬半两，续断1分，牛膝3分，鳖甲（炙）1两，黄耆1两。
出处：《妇人良方》卷二十一引胡氏方。
主治：妇人产后未满百日，体中虚损，血气尚弱，失于将理，或劳动作伤，致成蓐劳。其状虚羸，乍起乍卧，饮食不消，时有咳嗽，头目昏痛，发歇无常，夜有盗汗，寒热如疟，背膊拘急，沉困在床。
加减：None
功效：None
用法用量：每服先以猪肾1对，去筋膜，以水2大盏，生姜半分，大枣3个，煎至1盏，去猪肾、姜、枣，然后下药末2钱、葱3寸、乌梅1个，荆芥5穗，煎至7分、去滓，空心、晚食前温服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
