序号：57680
名称：大续命散
组成：麻黄1两，乌头1两，防风1两，桂心1两，甘草1两，蜀椒1两，杏仁1两，石膏1两，人参1两，芍药1两，当归1两，竹茹1两，黄芩1两，茯苓1两，干姜1两。
出处：《千金》卷八。
主治：八风十二痹，偏枯不仁，手足拘急，疼痛不得伸屈，头眩不能自举，起止颠倒，或卧苦惊如堕状，盗汗，临事不起；妇人带下无子；风入五脏，甚者恐怖，夜多异梦，悲愁哭泣，忽忽欲走。
加减：None
功效：None
用法用量：每服方寸匕，酒送下，1日2次；稍加，以知为度。
制备方法：上药治下筛。
临床应用：None
各家论述：None
用药禁忌：None
附注：《千金翼》本方无(艹闾)茹，有芎。
