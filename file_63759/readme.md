序号：63775
名称：清热滋阴汤
组成：当归（酒洗）3分，川芎（酒洗）7分，生地（酒洗）2钱，黄柏（酒炒）3分，知母（酒炒）5分，陈皮（酒洗）3分，白术（炒）5分，麦门冬1钱5分，牡丹皮1钱，赤芍药7分，玄参1钱，山栀（炒黑）1钱半，甘草5分。
出处：《古今医鉴》卷七。
主治：吐血、衄血、便血、溺血。
加减：身热，加地骨皮1钱，柴胡5分，子芩1钱；吐、衄血，加炒干姜7分，柏叶、茜根、大小蓟各1钱；大便血，加炒槐花、地榆、百草霜各1钱；溺血，加炒黑山栀子、车前子、小蓟、黄连各8分。上四种血病俱用阿胶珠5分，姜汁、韭汁、童便同服。
功效：None
用法用量：水煎，温服。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
