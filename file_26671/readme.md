序号：26679
名称：延年护命丹
组成：没药2钱（另研），乳香2钱（另研），轻粉2钱，蓬莪术1两，京三棱（炮）1两，芫花1两半，鳖甲（醋蘸，炙黄色，去尖，捶碎）1两半，黑牵牛4两（取头末2两），陈皮半两（与芫花2味，好醋同浸1宿，漉去晒干，更焙），川大黄（一半生，一半醋浸1宿，软切作块子，先作大块，更作小块，切作片子，微晒干，更焙，勿令焦）。
出处：《医方类聚》卷一○二引《经验秘方》。
主治：男子妇人脾胃不和，饮食减少，心腹绞痛，反胃吐食，痰涎喘嗽，五般淋沥，伤寒结胸，大小便不通，泻血，肠风痔痿；或伤寒后热甚发黄，久患疟疾，滑泻痢米谷不消，酒疸，食劳黄，十种水气遍身黄肿，一切蛊毒，五脏积热，衄血不止；及痄腮喉闭口疮，遍身疥癣，九种心痛，三十六般积，24般气，诸药不效，不问年深日近；并妇人所患产后恶血冲心，令人欲死，口燥舌干，四肢困倦，血山崩漏不止，面色萎黄，赤白带下，血经瘀闭不通；并小儿36种惊风。
加减：None
功效：不损脏腑，通和百脉。
用法用量：细嚼，温水送下。临卧服毕，不用枕头，仰卧至一更后，任便睡卧，来日取下积块或片子，或虫或脓血为效；如病大者，3日后再服1丸；病小者，50日后再进1服；如遍身走注疼痛，用乳香、没药煎汤化下；鼻血不止，冷水化下；有虫者，麻子油化下；15岁以下，50以上，1丸分作2服；10岁以下，60以上，1丸分作3服；6岁以下，70以上，1丸分作4服；3岁以下，80以上，1丸分作5服。然临卧时更宜。
制备方法：后7味为细末，入前3味研匀，炼蜜和为块，入臼中杵三千下，每1两分作4丸。
临床应用：None
各家论述：None
用药禁忌：忌生冷硬物，油腻等。3日宜食白粥。
附注：None
