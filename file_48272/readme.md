序号：48287
名称：神效万应剪金丹
组成：老阳子（江子）35粒（不去皮油），老阴子（杏子，不去皮油）3钱，陈皮金（去白）3钱，青皮木3钱（去瓤），半夏（水烫7次）9粒，乌梅7个（全用），丹火2两（水飞7次，去粗）1两，黄蜡（生用2两，溶，水洗去粗）1两半，枳壳（罗去瓤）3钱，黄连（罗去须）3钱，乳香2钱，没药（炙）2钱，木香（蒸）2两，槟榔21个，粟米5钱。
出处：《袖珍》卷三。
主治：诸痢，霍乱吐泻，一切风疾，心疼，头身疼痛，咳嗽，五劳七伤，宿食不消，水泻，时气伤寒，落马折伤，小儿惊风，蛔虫寸白虫，脾胃寒痛，小儿惊风，二便不通，热病，一切疮痛。
加减：None
功效：None
用法用量：每服10丸，血痢，甘草汤送下；白痢，干姜汤送下；红白痢，草、姜汤送下；赤痢，椿根皮汤送下；噤口痢，莲肉、山药、防风、粟米汤送下；落马、折伤、血闷，酒送下；霍乱吐泻，干姜汤送下；水泻，五苓散送下；一切风疾，升麻汤送下；咳嗽，桔梗、杏仁汤送下；痢鱼脑脓汁，食脏汤加附子1片送下；寸白虫，槟榔汤送下；心疼，酒送下；头痛、腰痛、打伤、冷气冲心、下元虚，并用酒送下；时气，井水送下；大小便不通，木通茶汤送下；脐下疼，芥菜汤送下；五劳七伤，猪胆汤送下；一切疮痛，萝卜汤送下；气痛，宿食不消，生姜汤送下；产后痢，当归汤送下；小儿吊惊风，汉防己汤送下；血风劳，史君子汤送下；口吐清水，诃子汤送下；肠痛，葱白汤送下；蛔虫咬心，槟榔汤送下；阳毒伤寒，栀子、黄连汤送下；阴毒伤寒，附子、枣儿汤送下；浑身壮热，沙糖水送下；虚热，柴胡、竹茹汤送下；寒热，乌梅汤送下；上焦虚热，大黄汤送下；脾胃寒痛，热酒送下。
制备方法：上将黄蜡溶开，入众药和匀，杵千百下作一块，再分一半药末，以油单纸收，临用旋丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：万应剪金丹（《寿世保元》卷十）。
