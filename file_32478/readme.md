序号：32487
名称：寿脾煎
组成：白术2-3钱，当归2钱，山药2钱，炙甘草1钱，枣仁1钱半，远志（制）3-5分，干姜（炮）1-3钱，莲肉（去心，炒）20粒，人参随宜1-2钱，急者用1两。
出处：《景岳全书》卷五十一。
主治：脾虚不能摄血等证，凡忧思郁怒积劳，及误用攻伐等药犯损脾阴，以致中气亏陷，神魂不宁，大便脱血不止，或妇人无火崩淋。
加减：如血未止，加乌梅2个，凡畏酸者不可用，或加地榆1钱半亦可；滑脱不禁者，加醋炒文蛤1钱；下焦虚滑不禁，加鹿角霜2钱为末搅入药中服之；气虚甚者，加炙黄耆2-3钱；气陷而坠者，加炒升麻5-7分，或白芷亦可；兼溏泄者，加补骨脂1钱炒用；阳虚畏寒者，加制附子1-3钱；血去过多，阴虚气馁，心跳不宁者，加熟地7-8钱或1-2两。
功效：None
用法用量：水2钟煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：摄营煎（原书同卷）、寿脾汤（《会约》卷十一）、参姜寿脾煎（《顾氏医径》卷四）。
