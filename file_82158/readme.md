序号：82174
名称：薏苡仁酒
组成：薏米仁5两（炒），当归3两，防风1两半，白芍1两半（炒），防己1两半，海桐皮2两，五加皮2两，杜仲3两，牛膝1两半。
出处：《医略六书》卷二十四。
主治：脚气疼痛而肿，脉浮弱者。
加减：None
功效：None
用法用量：入红酒1瓮，蒸，窨，空心随量温服。
制备方法：None
临床应用：None
各家论述：米仁健脾气以泻湿，当归养血脉以荣经，防风散风胜湿，白芍敛阴和营，汉防己泻血分湿热，海桐皮疏气分湿热，牛膝壮筋骨，杜仲强腰脚，五加皮活络舒经以雄脚力。浸以红酒。助行药势，使风散湿消，则营血统运，而经气清和，何脚气痛肿之不痊哉。此养营分解之剂，为风湿脚气痛肿之专方。
用药禁忌：None
附注：None
