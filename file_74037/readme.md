名称：愈风化痰丸
组成：何首乌（酒浸）2两，僵蚕2两，光乌（酒浸，去皮）2两，全蝎2两，南星（姜制）2两，半夏（便煮）2两，桑寄生2两，荆芥2两，牙皂（瓦炒）2两，破故纸（炒）2两，陈皮（去白）2两，菟丝子（炒）2两，地黄（去土）8两，木香1两，沉香1两半，白芍药（酒炒）、川乌（酒浸，去皮）、天麻1两2钱，黄芩（酒炒）2两，防风（去芦）2两，汉防己2两，白术（炒）2两，当归（酒洗）2两，羌活1两5钱，独活1两5钱，杜仲（姜汁炒）4两，巴戟（去心）3两。
