序号：56459
名称：海藻丸
组成：海藻半两（洗，焙干），木香半两，槟榔半两，川椒（去目，炒）半两，甘遂半两，川乌半两（炒，小儿去之，老人用1两），白牵牛2两（取末半两），黑牵牛2两（取末半两），茴香（微炒）3钱，猪苓（去皮）3钱，泽泻3钱，吴茱萸4钱（椒炒）。
出处：《普济方》卷二四七。
主治：奔豚疝气，膀胱小肠气，卵顽肾木及小儿偏坠，一切肾气。
加减：None
功效：利大、小便。
用法用量：每服50-100丸，空心盐汤、酒任意下，黄涎下者验。15岁以下小儿，25-50丸。
制备方法：上为末，滴水为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌甘草。
附注：None
