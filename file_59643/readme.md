序号：59658
名称：调元肾气丸
组成：淮生地（酒煮捣膏）4两，山萸肉2两，山药2两，牡丹皮2两，白茯苓2两，人参1两，当归身1两，泽泻1两，麦门冬（捣膏）1两，龙骨1两，地骨皮1两，木香3钱，砂仁3钱，黄柏（盐水炒）5钱，知母（童便炒）5钱。
出处：《外科正宗》卷二。
主治：房欲劳伤，忧恐损肾，致肾气弱而骨失荣养，遂生骨瘤，其患坚硬如石，形色或紫或不紫，推之不移，坚贴于骨，形体日渐衰瘦，气血不荣，皮肤枯槁，甚者寒热交作，饮食无味，举动艰辛，脚膝无力者。
加减：None
功效：None
用法用量：每服80丸，空心温酒送下。
制备方法：上为末，鹿角胶4两，老酒化稠，加蜜4两，同煎滴水成珠，和药为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌白萝卜，火酒、房事。
附注：None
