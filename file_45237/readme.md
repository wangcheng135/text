序号：45252
名称：枳实丸
组成：枳实1两半，菊花1两，蛇床子1两，防风1两，白薇1两，浮萍1两，蒺藜子1两，天雄半两，麻黄半两，漏芦半两。
出处：《千金》卷五。
主治：小儿风瘙，痒痛如疥，搔之汁出，遍身(疒咅)(疒畾)，如麻豆粒，年年喜发，面目虚肥，手足干枯，毛发细黄，肌肤不光泽，鼻气不利。此则少时热盛极，身体当风，风热相搏所得者，不早治之，成大风疾。
加减：None
功效：None
用法用量：5岁儿饮用10丸，加至20丸，1日2次。5岁以上者，随意加之；儿大者，可为散服。
制备方法：上为末，炼蜜为丸，如大豆许。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
