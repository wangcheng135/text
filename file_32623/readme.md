序号：32632
名称：赤石脂丸
组成：川当归2两半（去芦，酒浸），赤石脂（一色不杂者）1两半，白茯苓（去皮）1两，熟干地黄1两（自蒸者。铺中者再蒸），鹿角胶1两（锉碎，炒成珠），吴茱萸（汤泡7次，炒）1两，宣州大木瓜1个（重半斤以上者，开一盖子，去瓤，用艾叶填满，将盖子盖定，用小竹针扎定，饭甑内蒸熟，取艾，同前药焙干；木瓜去皮，研成膏子）。
出处：《魏氏家藏方》卷六。
主治：男子、妇人大病之后，伤损营卫，或发汗吐泻太过，或失血过多，精气亏损，心神恍惚，不得眠睡，饮食全减，肌体瘦弱，怠情倦乏，嗜卧无力，四肢痠痛。
加减：None
功效：None
用法用量：每服50丸，空心米饮送下。
制备方法：上为细末，木瓜膏子为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
