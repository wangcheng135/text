序号：73310
名称：普济回春丹
组成：防风20两，连翘40两，豆根10两，薄荷10两，川芎10两，花粉20两，羌活20两，当归20两，赤小豆10两，大青叶40两，黄连20两，栀子（炒）40两，黄芩40两，升麻10两，荆芥40两，玄参（去芦）40两，牛蒡子（炒）20两，滑石40两，赤芍20两，雄黄20两，柴胡10两，甘草40两。
出处：《北京市中药成方选集》。
主治：小儿内热发烧，伤风头痛，乍寒乍热，隐疹不出。
加减：None
功效：清热退烧，解肌透表。
用法用量：每服1丸，日服2次，温开水送下，3岁以下小儿酌减。
制备方法：上为细末，炼蜜为丸，重1钱，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
