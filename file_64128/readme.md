序号：64144
名称：猪肝煎丸
组成：猪肝2具，猪肚（净洗，去脂膜）2具（上2味并切作片，于新砂盆内，以薄黄泥固济，外面泥如灶；初且用小便1斗巳来入肝肚，慢火煎，以柳木蓖搅，夜盖覆；旋入小便，更可2斗，次别入后药），桃仁（去皮尖双仁，炒）5两（研），阿魏2两（醋化，去沙石，以面裹，慢火煨，候面黄熟，去面，并桃仁同研入药），薄荷汁2升，青蒿头（研汁）2升，猪肥20枚（取汁。上5味并相次入前药煎，频搅；更入小便2斗，都可5斗已来，不住以慢火煎，渐渐入，不得令干，候煎如稀饧，以通油器盛贮，封盖）。鳖甲（去裙襕，醋炙）2两，京三棱（煨，锉）2两，槟榔（锉）1两，桂（去粗皮）1两，干漆（炒烟出）1两，厚朴（去粗皮，生姜汁炙）1两，附子（炮裂，去皮脐）1两，木香1两，蓬莪术（煨）1两，石斛（去根）1两，草豆蔻（去皮）1两，枳壳（去瓤，麸炒）1两，当归（切，焙）1两，白术（锉，炒）1两，牛膝（酒浸，切，焙）1两，桔梗（锉，炒）1两，紫菀（去苗土）1两，芎藭1两，芍药1两，诃黎勒皮1两，陈橘皮（去白，焙）1两，陈曲（炒）1两，地骨皮1两，肉豆蔻10枚（去壳），柴胡（去苗）3两。
出处：《圣济总录》卷八十八。
主治：五劳七伤，脏腑虚惫，四肢少力，骨节疼痛，胃气不调，日渐羸瘦，不思饮食。
加减：None
功效：None
用法用量：每服20丸，空心人参汤或温酒送下，加至30丸。
制备方法：上25味并为末，用前煎为丸，如梧桐子大。
临床应用：None
