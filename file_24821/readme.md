序号：24829
名称：安神清镇丹
组成：犀角2钱，朱砂1钱，干漆1钱，山甲片5分，轻粉5分，旱三七2钱，京牛黄5厘，青蒿1捻，甘遂2钱，大黄2钱，麝香3厘。
出处：《医学探骊集》卷五。
主治：痫病，忽然昏倒，目天吊，口嚼舌，手足不能举动者。
加减：None
功效：None
用法用量：匀2次服，每早服1次。
制备方法：上为细末，用猪脾1个，阴阳瓦焙焦成炭存性，细研，与药面合一处，稀糊为丸，如梧桐子大。
临床应用：None
各家论述：方用犀角、京黄、朱砂之凉，大清其热；用麝香、青蒿、山甲、轻粉，引药直达病所；干漆能化有形血块；三七能化血块为水；少佐大黄、甘遂，使其病随吐泻而出；将猪脾焙焦，为其能引药入积块更爽也。
用药禁忌：None
附注：None
