序号：39716
名称：经进龙虎金丹
组成：黑附子（去皮脐）1两，川乌头（去皮脐）1两，虎骨（醋炙）1两，古文钱（醋碎，研）1两，当归（生用）1两，白胶香（生，研）1两，地龙（去土）1两，木鳖子（火炮，去皮）1两，草乌头（盐炒令黄，发熬）1两，黑牵牛（生用）1两，苁蓉（酒浸）1两，牛膝（酒浸）1两，乳香半两（生，研），没药（生用）1两，巴戟（去心）1两，自然铜（醋淬）1两。
出处：《普济方》卷九十三引《卫生家宝》。
主治：一切瘫痪风疾。
加减：None
功效：None
用法用量：每服10丸，渐加至30丸，食前茶盐汤送下。
制备方法：上为细末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：黑附子退寒热，川乌头补肾经，虎骨治伤折，古文钱接骨，当归补骨活血，白胶香去麻痹，地龙能行血，木鳖子冲开滞气，黑牵牛除肾腧气，苁蓉补肾经，牛膝补劳，乳香治一切疼痛，没药化内损坏脓血，巴戟治八般腰疼，自然铜治伤筋动骨。
用药禁忌：None
附注：None
