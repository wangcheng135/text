序号：5593
名称：宁坤至宝丹
组成：益母草30两，香附（炙）5两，白芍5两，川芎5两，当归5两，橘红5两，熟地5两，生地5两，黄芩5两，乌药5两，茯苓5两，白术（炒）5两，阿胶（炒）2两5钱，木香2两5钱，苏叶2两5钱，砂仁2两5钱，甘草2两5钱，川牛膝2两（共研为细粉，过罗），每16两细粉加人参（去芦）3钱，沉香8钱，琥珀4钱。
出处：《北京市中药成方选集》。
主治：月经不调、，胸膈不舒，食欲不振，身体瘦弱，腰痛腿痠。
加减：None
功效：益气和荣，调经养血。
用法用量：每服1丸，温开水送下，日2次。
制备方法：上为细末，过罗，混合均匀，炼蜜为丸，重3钱，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
