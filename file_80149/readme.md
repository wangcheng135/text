序号：80165
名称：毓麟珠
组成：人参2两，白术（土炒）2两，茯苓2两，芍药（酒炒）2两，川芎1两，炙甘草1两，当归4两，熟地（蒸捣）4两，菟丝子（制）4两，杜仲（酒炒）2两，鹿角霜2两，川椒2两。
出处：《景岳全书》卷五十一。
主治：妇人气血俱虚，经脉不调，或断续，或带浊，或腹痛，或腰酸，或饮食不甘，瘦弱不孕。
加减：男子服，宜加枸杞、胡桃肉、鹿角胶、山药、山茱萸、巴戟肉各2两；妇人经迟腹痛，宜加酒炒破故纸、肉桂各1两，甚者再加吴茱萸5钱（汤泡1宿，炒用）；如带多腹痛，加破故纸1两，北五味5钱，或加龙骨1两（醋煅用）；如子宫寒甚，或泄或痛，加制附子、炮干姜随宜；如多郁怒气，有不顺而为胀为滞者，宜加酒炒香附2两，或甚者再加沉香5钱；如血热多火，经早内热者，加川续断、地骨皮各2两，或另以汤剂暂清其火，而后服此，或以汤引酌宜送下亦可。
功效：《北京市中药成方选集》：补气养血，调经种子。
用法用量：每服1-2丸，空心用酒或白汤嚼下，或为小丸吞服亦可。服1-2斤即可受胎。
制备方法：上为末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：毓麟丹（《医级》卷九）、毓麟丸（《北京市中药成方选集》）。
