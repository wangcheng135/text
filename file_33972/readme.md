序号：33981
名称：皂荚丸
组成：猪牙皂荚（去黑皮，酥炙，研）2钱，白马夜眼（炒令黑色，研）2钱，安息香（炒令黑色，研）2钱，斑蝥（以糯米炒令米黑色，去米并翅足不用，余即研之）2钱，蜈蚣1条（炙令黄色），蛇蜕1条（炒令黑焦，研），粉霜2钱（面2钱，水滴和为饼子，煨令黄色，研），雄黄（研）1钱，丹砂（研）1钱，硇砂（研）1钱，牛黄（研）1钱，犀角屑1钱，胡黄连1钱。
出处：《圣济总录》卷一○○。
主治：诸注。
加减：None
功效：None
用法用量：每服50丸，四更尽，以桃仁煎汤送下。
制备方法：上为末，令匀，以黄狗胆汁为丸，如梧桐子大，别以丹砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
