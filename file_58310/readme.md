序号：58325
名称：真珠丸
组成：真珠（研）1分，虎睛（左睛为上，酒浸，晒干，研）1只，露蜂房半两，麻黄（去根节）半两，钩藤半两，铁粉（研细）3分，防葵1两，大黄（锉，炒）3分，黄芩3分，龙齿（研）1分，银屑3分，栀子仁3分，独活（去芦头）半两，柴胡（去苗）3分，升麻3分，白鲜皮3分，雷丸1分，沙参半两，细辛（去苗叶）半两，蛇蜕（烧灰）1分，石膏（研）半两，牛黄（研）1分，蚱蝉（去翅足，熬）4枚。
出处：《圣济总录》卷一七一。
主治：小儿自1岁至大患癫痫，发动无时，口内沫出，小便不觉，呼唤不应。
加减：None
功效：None
用法用量：1-2岁儿每服5丸，研破，米饮送下。
制备方法：上为末，炼蜜为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
