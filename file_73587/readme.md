序号：73603
名称：蓬莱丸
组成：苍术8两（米泔浸透，陈壁土炒），半夏（姜汁制）4两，柴胡4两，黄芩4两，厚朴（姜汁炒）4两，广皮4两，枳实（炒）4两，羌活4两，苏叶4两，木通4两，山楂（炒）6两，莱菔子（炒）6两。
出处：《串雅内编》卷一。
主治：感冒、瘟疫时症。
加减：None
功效：None
用法用量：每服大人1丸，小儿、孕妇及吐血、虚损人半丸，头痛寒热，葱、姜汤送下；咳嗽痰喘，姜汁汤送下；中暑，香薷、扁豆汤送下；疟疾，姜汁冲服；红白痢，木香、槟榔汤送下；霍乱吐泻，藿香、砂仁汤送下；腹痛水泻，赤芍、车前子汤送下；饱闷，陈皮、木香汤送下；不服水土，广藿香汤送下；山岚瘴气、蛊毒虫积，槟榔汤送下；不识病原诸症，白滚水送下。
制备方法：上为末，鲜荷叶煎汤，和药晒干，加神曲6两，打糊为丸，重3钱，另用朱砂5钱，雄黄1两，研末为衣。
临床应用：None
各家论述：None
用药禁忌：服药后忌食生冷面食。
附注：None
