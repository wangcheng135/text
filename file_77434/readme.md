序号：77450
名称：雄黄散
组成：明矾4两半（研），朴硝半两（研），雄黄半两（研），鸭觜半两（研），胆矾半两（研），雌黄半两（研）。
出处：《医方类聚》卷一八四引《吴氏集验方》。
主治：一切内外痔，鸡冠翻花。
加减：None
功效：None
用法用量：每用甘草水调敷，日4-5次，夜2次。自然黑硬脱落，亦无疮口，肠头1-2日自收。如日敷2次，夜不敷，但脱迟。凡敷药先须用温甘草汤洗痔，帛子拭干敷药。初敷1-2日时觉硬痛，不妨，直待敷干落为度。
制备方法：上用雄黄、雌黄、朴消，顿沙锅内，却安胆矾在上，以白矾盖面令实，炭火煅令青烟尽，取出，纸衬于地上，出火毒，再研，别入好乳香、没药各少许。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
