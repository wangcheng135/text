序号：43203
名称：养生丹
组成：母猪大肠1尺（入朴消4两，两头扎住入瓦罐，水3碗煮将干，盐泥塞口，勿泄气，炭火煅存性），象牙末2两，刺猬皮2个（煅存性），麝香1钱，猪悬蹄24个（切片，土炒），川山甲24片（土炒），乳香3钱，没药3钱，雄黄3钱，地榆3钱，大黄5钱，青盐7钱，白芷1两，明矾5钱，小活龟3个（连肉入罐内，用泥封口，煅存性），蜂房1个（带子者，焙干为末），黄牛角（角思）1个（煅存性），朴消7钱，槐花5钱（炒），黄蜡1两，自然铜（煅，醋淬7次）5钱。
出处：《外科大成》卷二。
主治：痔漏。
加减：None
功效：内消痔漏。
用法用量：每服3钱，空心老酒送下，1日2次，服至半月出管。1月全愈，不用生肌。
制备方法：上为末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
