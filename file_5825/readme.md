序号：5825
名称：加减开邪散
组成：甜白术5钱（土炒），云苓5钱，制半夏3钱，青皮1钱5分，枳壳1钱（炒），柴胡1钱，山楂3钱（炒），草蔻仁1钱（研）。
出处：《慈航集》卷下。
主治：疟疾，足太阳膀胱之疟与手太阳小肠合证，初病令人腰痛头重，寒从背起，先寒后热，熇熇暍暍，热止汗出，遍身骨节痠痛，小便短赤。
加减：体虚脉弱者，加人参1钱，好肉桂心8分，另顿对服，贫人无力者，以上党参1两代之；恶心，加灶心土5钱；作泻，加炒白芍5钱；内热，小便短赤，加青蒿3钱；遍身骨节痠痛，加秦艽1钱5分；手足冷，青紫色，加桂枝2钱。
功效：None
用法用量：煨姜3钱，大枣3枚为引。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
