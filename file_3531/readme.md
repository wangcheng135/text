序号：3531
名称：太素丹
组成：炼成钟乳粉1两，真阳起石2钱（新瓦上用熟火煅过，通红为度，去火候冷，研极细）。
出处：《百一》卷一引周彦广方。
主治：停寒肺虚，痰实喘急，咳嗽经久，痰中有血；及气虚感冷，脏腑滑泄，脾胃羸弱，不进饮食。虚损痼冷，吐泻暴脱，伤寒阴证，手足厥冷。
加减：None
功效：益阳退阴。
用法用量：临和时入白石脂1钱，须大盘子不住手转，候8-9分坚硬，阴干，用新粗布以滑石末出光。每服2丸至3丸，空心人参汤或陈米饮送下。
制备方法：上为末，用糯米粽子尖拌和为丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：白丹(《普济方》卷二六五引《余居士选奇方》。
