序号：80715
名称：麋角丸
组成：麋角屑1斤（以酥拌炒微黄），熟干地黄3两，巴戟1两，黄耆（锉）1两，牛膝（去苗）1两，人参（去芦头）1两，独活1两，萆薢（锉）1两，白茯苓1两，桂心1两，肉苁蓉（酒浸1宿，去皱皮，炙）1两，附子（炮裂，去皮脐）1两，泽泻1两，续断1两，芎1两，槟榔1两，防风（去芦头）1两，当归（锉，微炒）1两，鹿角胶（捣碎，炒令黄燥）1两，白蒺藜（微炒，去刺）1两。
出处：《圣惠》卷九十八。
主治：五脏虚损，腰脚疼痛。
加减：None
功效：久服益肌肤，填骨髓，好颜色，耐寒暑，祛风破气，髭发润黑。
用法用量：每日服30丸，空心以温酒送下，加至40丸。
制备方法：上为末，以生地黄汁1大盏、酒1大盏相和，入酥半合，煎成膏，和诸药末，若硬，即炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
