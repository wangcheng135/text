序号：32830
名称：羌活汤
组成：羌活（去芦头）半两，桂（去粗皮）半两，防风（去叉）半两，天麻4两，甘菊花4两，旋覆花4两，白附子（炮）4两，栾荆（去叶，俗谓之顽荆）4两，天南星（水浸7日，切作片子，焙干）4两，乌头（盐水浸1日，切作片子，焙干，炒）4两，甘草（炙）4两，麻黄（去根节）4两，附子（炮裂，去皮脐）半斤，苍术（米泔浸1宿，切，炒）半斤，威灵仙6两，牵牛子（捣取粉）3两，陈橘皮（汤浸，去白，焙）半斤。
出处：《圣济总录》卷十。
主治：中风百节疼痛，头目昏眩；及伤寒头疼壮热，肢节疼痛，肩背拘急。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，加薄荷7叶，生姜3片，同煎至7分，去滓温服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
