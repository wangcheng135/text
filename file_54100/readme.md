序号：54115
名称：柴胡汤
组成：柴胡（去苗）1两半，五味子1两，桔梗（炒）1两，熟干地黄（焙）1两，白茯苓（去黑皮）1两，麦门冬（去心，焙）1两，紫菀（去苗）1两，人参1两，地骨皮1两，黄耆（锉）1两，白术1两，桂（去粗皮）1两，牡蛎（研粉）1两，半夏（去滑，汤洗7遍）3分，甘草（炙）3分。
出处：《圣济总录》卷三十七。
主治：寒热往来，夜卧盗汗，四肢无力，饮食口苦，上气咳嗽。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，加生姜半分（拍碎），大枣3枚（擘破），煎至6分，去滓温服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
