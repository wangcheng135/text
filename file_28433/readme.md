序号：28441
名称：芫菁丸
组成：芫菁20个（去头足翅，糯米拌，炒令黄），地胆10个（去头足翅，糯米拌，炒令黄色），斑蝥10个（去头足翅，糯米拌，炒令黄色），生犀角屑如枣大，黑豆黄50粒（炒热），牛黄半枣大（细研），蜈蚣1条（大者，去足，炙令焦黄）。
出处：《圣惠》卷六十六。
主治：狼瘘。
加减：None
功效：None
用法用量：初服药，隔夜少食，每服2丸，空心以温水送下。须臾可煮醋浆水薄粥，稍稍饮之。至日西甚虚闷，可煮蔓青菜羮食之。如壮人，隔日1服；人弱，隔2-3日1服。服药后，疮愈虫尽为度。若愈，仍将息1月。药欲尽，须预合，勿使断绝药气。
制备方法：上药前六味为末，入牛黄研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
