序号：27368
名称：补益紫石英丸
组成：紫石英1两（细研，水飞过），白石英1两（细研，水飞过），泽兰3分，木香半两，附子1两（炮裂，去皮脐），熟干地黄1两，芎3分，柏子仁3分，桂心3分，防风半两（去芦头），牛膝3分（去苗），续断3分，人参3分（去芦头），白茯苓3分，羌活半两，黄耆3分（锉），白术3分，当归3分（锉，微炒），甘草1分（炙微赤，锉），白薇3分，杜仲3分（去皴皮，炙微黄，锉），干姜半两（炮裂，锉），川椒半两（去目及闭口者，微炒去汗）。
出处：《圣惠》卷八十一。
主治：产后虚羸乏弱。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，入研了药令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
