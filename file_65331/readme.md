序号：65347
名称：大麝香丸
组成：麝香3分，牛黄2分，附子2分，鬼臼2分，真珠2分，莽草2分，犀角2分，矾石2分，细辛2分，桂心2分，獭肝2分，藜芦2分，蜈蚣1枚，蜥蜴1枚，丹砂2两，雄黄1两，巴豆50枚，杏仁50枚，地胆7枚，元青7枚，亭长7枚，斑蝥7枚，礜石8分。
出处：《千金》卷十二。
主治：鬼疰、飞尸诸病。
加减：None
功效：None
用法用量：每服1丸，1日2次。渐加至3丸。虫毒所螯，摩之。以知为度。若欲入毒疫疠乡，死伤病处，及恶鬼塚墓间，绛袋盛之，男左女右，肘后系之。又以少敷鼻下人中，及卧不魇。
制备方法：上为末，炼蜜为丸，如小豆大。
临床应用：None
各家论述：《千金方衍义》：玉壳丸中加入斑蝥、地胆、芫青、亭长、蜈蚣、蜥蜴、獭肝以攻毒邪，犀角、牛黄、麝香、真珠以和药毒，杏仁、细辛、莽草、桂心、矾石、鬼臼以佐玉壶丸中六味，并毒虫野兽之药，共襄厥功。
用药禁忌：None
附注：《千金翼》有大黄，无莽草。
