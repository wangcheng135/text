序号：3370
名称：太一护命丸
组成：甘菊花6两，麦门冬（去心，焙）6两，枸杞子（焙）6两，白术6两，人参6两，白茯苓（去黑皮）6两，远志（去心）6两，菖蒲（石上者）6两，桂（去粗皮）6两，熟干地黄1斤（焙）。
出处：《圣济总录》卷一八七。
主治：元脏虚损。
加减：None
功效：益精血，满骨髓，坚实脏腑，乌发，延年却病。
用法用量：每服20丸，空心、食前清酒送下。渐加至50丸，从50丸复渐减至20丸，终而复始。
制备方法：上为粗末。取春采生地黄50斤，绞取汁，同药末于银石器内，逐旋入地黄汁微炒，候入尽汁，焙干，再捣为细末，炼蜜为丸，更入酥少许为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：太乙护命丸（《普济方》卷二二一）。
