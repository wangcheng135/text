序号：62368
名称：旋覆花丸
组成：旋覆花2两，防风（去芦头）1两，吴白芷1两，甘菊花1两，天麻1两，天南星（炮）1两，白附子（炮）1两，半夏（汤洗）1两，陈皮（去白）1两，芎藭1两，蝎梢（去毒1两，炒）1两，僵蚕（炒去丝）1两，石膏（研）1两。
出处：《御药院方》卷一。
主治：诸风痰实，头目昏眩，旋运欲倒，呕哕恶心，恍惚不宁，神志昏愦，肢体倦怠，颈项强硬，手足麻痹；偏正头痛。身生小癞，大如酸枣，或如豆，色赤而内有脓血，此名疿痤。
加减：None
功效：除风化痰，清利头目。
用法用量：每服30-40丸，食后温生姜汤送下；或茶清亦得。
制备方法：上为细末，生姜汁煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
