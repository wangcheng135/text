序号：60025
名称：资生汤
组成：紫厚朴（姜汁炒）7分，蕲文（醋炒）7分，当归（酒洗）1钱5分，川芎1钱5分，川贝母（去心，另研）1钱，菟丝子1钱，川羌活5分，甘草5分，荆芥穗8分，生黄耆8分，枳壳（麸炒）6分，白芍（酒炒）1钱2分，姜3片。
出处：《竹林女科证治》卷二。
主治：向有难产，或惯滑胎，或偶动胎气。
加减：None
功效：None
用法用量：上用水2钟，煎8分，入川贝末和匀，空心温服。临月服3-5剂，永无难产之患。若7个月起服，7月服1剂，8月服2剂，9月服3剂，10月服3-5剂，临产再服1剂，甚效。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
