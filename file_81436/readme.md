序号：81452
名称：鳖肉煎丸
组成：生鳖肉半斤（治如食法），黄芩（去黑心）半两，柴胡（去苗）半两，蜣螂（去翅头足，炒）半两，鼠妇（炒，去足）1分，干姜（炮）1分，大黄（生锉）1分，海藻（洗去咸汁，焙）1分，葶苈子（纸上炒）1分，桂（去粗皮）1分，牡丹皮1分，厚朴（去粗皮，生姜汁炙，锉）1分，紫菀（切，焙）1分，瞿麦（去梗）1分，半夏（汤洗去滑，焙）1分，人参1分，大戟（锉）1分，（庶虫）虫（炒）1分，射干（炮）1分，阿胶（炙燥）1分，桃仁（汤浸，去皮尖双仁，别研）1分，石苇（去毛）1分，赤芍药1分，桑螵蛸（炒）1两。
出处：《圣济总录》卷三十五。
主治：疟久不愈，结为癥瘕，名曰疟母。
加减：None
功效：None
用法用量：每服20丸，未发前温酒送下。
制备方法：上为细末，取灶下灰3升，酒2升淋灰，取灰汁先煮鳖肉令烂，绞去鳖肉，后入诸药末，纳鳖肉汁，更煎成膏，可丸即止，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
