序号：43716
名称：南星丸
组成：蛇含石1分，石燕（并火煅，酒淬3-5次）1钱，代赭石1钱，朱砂1钱，铁粉1钱，雄黄1钱，五灵脂、乳香、川乌（去皮，炮）、天浆子27粒（炒），乌蛇肉（酒浸，炙，去骨）1钱，蛇皮（炙）、蛇头1个（酒浸，炙），僵蚕（皂角水浸一夕，微炒）、蝉蜕、天麻、蜂房（炒）、蜈蚣（大赤足者）2钱，全蝎2钱（新薄荷自然汁浸1宿，焙，微炒），白附子1钱，南星（姜汁浸一夕，牛胆拌炒）1钱，羌活1钱，川芎1钱，麝香1钱，脑子半钱。
出处：《普济方》卷三七五。
主治：小儿急慢惊风，经久诸般痫病，累医无效，但是恶候，不间阴阳。
加减：如吐泻后，加附子（炮，去皮，随轻重入药）。
功效：None
用法用量：3岁1丸，薄荷汤入酒少许，磨化服；或作散亦可。灌药1服，得睡即效。
制备方法：上为末，糊为丸，如鸡头子大，金箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
