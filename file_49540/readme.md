序号：49555
名称：茯苓丸
组成：白茯苓1两，牡荆子半两，天雄1两（炮裂，去皮脐），黄耆1两（锉），肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），薯蓣1两，巴戟1两，石长生3分，桂心1两，菟丝子1两（酒浸3日，晒干，别捣为末），杜仲1两（去粗皮，炙微黄，锉），牡蛎1两（烧为粉），山茱萸1两，熟干地黄1两，泽泻3分，石斛1两半（去根，锉），附子1两（炮裂，去皮脐），天门冬1两半（去心，焙），人参1两（去芦头），防风半两（去芦头），羌活3分，当归3分，甘草半两（炙微赤，锉）。
出处：《圣惠》卷三十。
主治：虚劳痿痹，手足厥冷，精气虚乏，骨节疼痛，头眩吐逆，腰脊强直。
加减：None
功效：丰盛体骨，光泽肌肤。
用法用量：每服30丸，食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
