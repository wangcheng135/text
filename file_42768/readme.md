序号：42783
名称：保寿散
组成：白茯苓1两，新罗参1两，川雄黄1两，牙消1两，甘草（炙）1两，片脑少许，麝香少许，牛黄少许（无亦得）。
出处：《婴童百问》卷十。
主治：婴孩惊热、潮热、风热、虚热，头额温壮，白日夜间多啼，伤湿鼻流清涕，喉咽时时有清涕，夜多睡卧不稳，或手足口舌生小热毒疮，或因吃着、喜怒、乳食，胸膈不快，时复吐呕乳，忽因人物所惊，日夜间手足心热，痰壅咳嗽，兼患天吊急惊风。
加减：None
功效：None
用法用量：1岁儿半钱，2-3岁儿1钱，并薄荷汤调下；金银薄荷汤尤好。
制备方法：上为细末，入锡合内收之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
