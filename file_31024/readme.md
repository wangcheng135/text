序号：31033
名称：李根皮散
组成：李根皮1升，通草1两，白蔹1两，桔梗1两，厚朴1两，黄芩1两，附子1两，甘草2两，当归2两，葛根3两，半夏5两，桂心4两，芍药4两，穹6两，栝楼根5两。
出处：《千金》卷二十二。
主治：痈疽发背，及小小瘰疬。
加减：None
功效：None
用法用量：每服方寸匕，酒下，日3次。疮大困者，夜再服之。
制备方法：上药治下筛。
临床应用：None
各家论述：《千金方衍义》：甘李根皮苦咸降逆；栝楼、葛根清胃解毒；通草、白蔹散结利窍；厚朴、半夏破气涤痰；桂心、附子化坚排脓；芎、归，芍药和营止痛；甘、桔、黄芩清热利气，疡溃本虚而脓未透者为宜。
用药禁忌：忌羊肉、饧、海藻、菘菜、猪肉、冷水、生葱。
附注：本方方名，《外台》引作“李根散”。
