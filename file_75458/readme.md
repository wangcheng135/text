序号：75474
名称：解郁调胃汤
组成：白术1钱，陈皮（盐水洗）1钱，白茯苓（去皮）1两，归尾（酒洗）1钱2分，赤芍（酒浸）8分，川芎6分，生地黄（酒洗，姜汁拌，晒干）8分，香附米8分，神曲（炒）7分，栀子仁（盐水炒）1钱2分，麦芽（炒）7分，桃仁（去皮）4两，生甘草4分。
出处：《回春》卷二。
主治：郁证。因怒、忧、思、虑、劳心而致胃脘血液耗损，痰火内郁，水浆易下而食物难消，若噎膈之症；或气分之火壅遏于中而时作刺痛者。
加减：若胸膈刺痛，加姜黄（酒炒）8分；若胸噎闷，加枳壳（麸炒）7分；胸内烦热，加黄连6分；大便不利，加酒蒸大黄2钱2分；有痰，加半夏（姜汁炒）8分，去地黄；饮食不美，去地黄，加白术5分；呕吐，加藿香1钱，去地黄、川芎、桃仁。
功效：None
用法用量：加生姜3片，水煎，热服。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
