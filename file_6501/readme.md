序号：6501
名称：二冬膏
组成：天门冬（去心）1斤，麦门冬（去心）1斤。
出处：《摄生秘剖》卷四。
主治：肺胃燥热，咳嗽痰少，痰中带血，咽痛音哑。虚损痰咳，烦渴热燥。咳逆上气，咽喉疼痛，燥渴音哑。燥咳痰少，痰中带血，鼻干咽痛。
加减：None
功效：清心润肺，降火消痰。清肺益肾，生津止渴。
用法用量：二冬入砂锅，水煎取汁，再将滓水煎，以无味为度，入蜜，熬成膏。空心白汤下2-3匙。
制备方法：None
临床应用：None
各家论述：1.《摄生秘剖》：是膏用天冬清金降火，益水之源，故能下通肾气以滋阴；更以麦冬气薄主升，味厚为阴，有清心润肺之功，堪与天冬相并而施膏泽，以濡其枯槁焉。_x000D_
2.《丸散膏丹集成》：二冬禀少阴水精之气。麦冬禀水精而上通于阳明，天冬禀水精而上通于太阳，夫冬主闭藏，门主开转，咸名门冬者，俱能开转闭藏而上达。合二冬制熬成膏，消痰润肺，生脉清心。久服则肾固气平，体健身轻，不老不饥，为益非浅。
用药禁忌：忌食辛辣之物；风寒咳嗽忌服；消化不良，便溏者不宜。
附注：天麦二冬膏（《全国中药成药处方集》西安方）。
