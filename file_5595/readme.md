序号：5595
名称：宁坤至宝丹
组成：嫩黄耆（蜜炙）3两，白术（陈壁土炒）2两，枣仁（炒香）2两，归身（酒炒）2两，香附（杵，米酒制）2两，川断（酒炒）2两，条芩（酒炒）2两，甘枸杞2两，血余（煅不见火）2两，阿胶（蛤粉炒）2两，杜仲（盐水炒）2两，茯苓（乳制）1两半，白芍（酒炒）1两半，丹参（酒炒）1两半，北五味（焙）6钱，甘草（蜜炙）1两，朱砂（飞为衣）1两，大生地（酒煨）4两。
出处：《卫生鸿宝》卷五。
主治：妇人经脉不调，带下，崩淋，虚劳，胎前产后百病。
加减：None
功效：None
用法用量：按症照引调服：凡久不坐孕，经脉不调，腹痛痠胀，或赤淋白带，腰痛胃痛，夜热心烦，食少，日服1丸，莲子汤送下；胎气失调，恶心呕吐，虚烦阻食，浮肿气急，腰腹痠痛，胎漏下血，或伤胎见红，每服1丸，莲子汤送下；甚者服数丸，人参汤送下；临产疼阵作时，服1丸，白汤送下，胎自顺下；如有横逆异产，每服数丸，汤和童便送下，保全母子；或难产者，冬葵子3钱，煎汤调下；产后下血过多，汤和童便送下；恶露不行，腹痛块瘀，山楂3钱，红花1钱，煎汤调下；或寒热往来，有外感者，荆芥穗1钱，煎汤调下；兼虚汗者，人参汤送下；虚烦狂躁，腹满气急，俱白汤送下；无论老少妇女，血崩尿血，或因血虚，周身筋骨疼痛者，白汤送下。
制备方法：上药各为细末，和匀，炼蜜为丸，每重3钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
