序号：66182
名称：野狐倒上树
组成：黑铅4两，汞2钱。
出处：《寿世保元》卷六。
主治：须发白。
加减：None
功效：None
用法用量：临用药时，将须发以温水洗净，就湿再用脂皮包手指拍药霜，粘在须发上，自然黑到根。
制备方法：先将铅化成汁．后入汞，凝成叶子剪成钱样，外用铁丝穿作2-3串，听用。再用大瓷罐1个，入无盐好醋3碗，将铅钱入罐内，悬于醋上，离醋2指，内泡卷柏2个，鸭嘴胆矾4钱，用瓷碟封口盖之，再用黄泥封固，夏月日中晒七日，冬月糠火煨七日，出，罐底摘一孔出醋不用，揭起碟来，扫下药霜，用脂皮包住收之。
临床应用：None
各家论述：None
用药禁忌：忌香肥皂水洗。
附注：None
