序号：47777
名称：神仙既济丹
组成：山药（酒蒸）3两，牛膝（酒洗）3两，杜仲（酥炙）2两，巴戟（汤泡）2两，五味子2两，白茯苓2两，枸杞（酒洗）2两，小茴（盐水炒）2两，苁蓉（酒洗）2两，山茱萸（酒蒸，去核，晒干）2两，石菖蒲（去毛）2两，远志（甘草水泡，去骨，晒干）2两，黄柏（酒炒）4两，知母（去毛，酒炒）2两，生地（酒蒸）2两，熟地（酒蒸）2两，麦冬（去心）2两，人参（去芦）2两，菟丝子（酒煮烂，捣成饼，晒干）2两，甘菊（酒洗）2两，山栀子（炒黑）2两，广橘红1两，天冬（汤泡）2两，当归（酒洗）2两，龙骨（火煅）2两。
出处：《古今医鉴》卷七。
主治：None
加减：None
功效：补诸虚百损，五劳七伤，滋肾水，降心火，补脾土，添精髓，益气和血，壮筋骨，润肌肤，聪耳明目，开心益智，强阴壮阳，延年益寿。久服坎离相济，阴阳协和。
用法用量：每服70-80丸，空心淡盐汤送下。
制备方法：上为末，炼蜜和枣肉为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
