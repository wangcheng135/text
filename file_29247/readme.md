序号：29255
名称：远志散
组成：远志（去心）2两，人参2两，赤小豆（炒熟）2两，附子（炮裂，去皮脐）2两，细辛（去苗叶）2两，桂（去粗皮）2两，干姜（炮）2两，防风（去叉）2两，龙齿（研）2两，熟干地黄（切，焙）2两，菖蒲（九节者，去须节，米泔浸，切，焙干）2两，黄耆（锉）4两，白茯苓（去黑皮）4两，白术4两。
出处：《圣济总录》卷十四。
主治：风惊恐，悲思恍惚，心常惕惕，梦寐不定。
加减：None
功效：None
用法用量：每服3钱匕，空心、晚食前以温酒调下。
制备方法：上药除别研一味外，余捣罗令细，即入研者拌匀，再罗。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
