序号：33394
名称：羌活散
组成：羌活（去芦头）1两，独活（去芦头）1两，前胡（去芦头，并锉）1两，人参（去芦头）1两，桔梗（去芦头）1两，芎1两，细辛（去苗）1两，防风（去芦头）1两，荆芥穗1两，甘菊花1两，土蒺藜1两，茯苓（去皮）1两，枳壳（麸炒，去瓤）1两，石膏（细捣研，水飞）1两，甘草（炙）1两。
出处：《圣济总录》卷一○三。
主治：风毒气攻注，头目昏眩，目碜涩疼痛，及皮肤瘙痒，瘾疹赤肿。
加减：None
功效：None
用法用量：每服2钱匕，不拘时候，茶、酒任调下。
制备方法：上十五味，除石膏外，同杵为散，再入石膏，和令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
