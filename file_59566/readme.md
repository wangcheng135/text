序号：59581
名称：调中汤
组成：木香1钱（不见火），防风（去芦）1分，黄耆（蜜炙）1分，炙甘草1分，人参（去芦）2两，白茯苓（去皮）2两，当归（去芦，酒浸）2两，熟干地黄2两（洗），罂粟壳（去顶蒂瓤）半两（剪碎，蜜拌炒令黄）。
出处：《魏氏家藏方》卷七。
主治：赤白痢。由肠胃虚弱，冷热之气，乘虚相博，血渗入肠，则为泻痢，重者血与脓相杂，状如浓涕，轻者浓血上赤脉，状如鱼脑，日夜不绝，脐腹(疒丂)痛，不思饮食。
加减：如血痢，加竹茹1块同煎。
功效：None
用法用量：每服3大钱，水1盏，加生姜3片，枣子1个，煎至7分，去滓，食前通口服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
