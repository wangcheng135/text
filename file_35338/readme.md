序号：35348
名称：乳香膏
组成：乳香2两，附子（生用）5两，乌头（生用）2两，木鳖子（去壳）2两，当归1两，秦艽1两，紫草（去苗）3两，苏枋木（锉）5两，头发灰1两，清油20两（各细锉，入油慢火熬，候诸药焦黑色大度，去滓，入后药），枫香脂5两，松脂2两（2味同研），黄蜡5两，铅丹2两，没药1两半（研）。
出处：《圣济总录》卷一三○
主治：痈疽发背，及一切疮肿。
加减：None
功效：None
用法用量：摊纸花子上，看疮大小用之。
制备方法：上药先煎前9昧，候色黑去滓，次下枫香等5味，依次等逐味下，用慢火熬，用柳木篦不住手搅，侯熬成膏，滴水中成珠为度。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
