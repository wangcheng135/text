序号：76540
名称：雷火针
组成：白芷1钱，独活1钱，川芎1钱，细辛1钱，牙皂1钱，川山甲（炮，倍用）1钱，丁香1钱，枳壳1钱，松香1钱，雄黄1钱，乳香1钱，没药1钱，杜仲1钱，桂枝1钱，硫黄2钱，麝香酌量，熟艾2-3两（一方有巴豆仁8分，斑蝥3钱，去头足翅）。
出处：《景岳全书》卷五十一。
主治：风寒湿毒之气留滞经络而为痛为肿，不能散者。
加减：None
功效：None
用法用量：用时向灯上点着，随用纸三五层，或布亦可，贴盖患处，乘热按于纸上。
制备方法：上为粗末，取艾铺底，掺药于上，用好皮纸卷筒，先须用线绊约两头，防其伸长，然后加纸再擀，务令极实，粗如鸡子尖样，用鸡子清尽刷外层，卷而裹之，阴干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
