序号：67630
名称：小牛角(角思)散
组成：牛角（角思）1枚（烧令赤），鹿茸2两，禹余粮2两，当归2两，干姜2两，续断2两，阿胶3两，乌贼骨1两，龙骨1两，赤小豆2升。
出处：《千金》卷四。
主治：妇人带下五贲，外实内虚。一曰热病下血，二曰寒热下血，三曰经脉未断，为房事则血漏，四曰经来举重，伤络脉下血，五曰产后脏开经利。
加减：None
功效：None
用法用量：每服方寸匕，空腹以酒送下，1日3次。
制备方法：上药治下筛。
临床应用：None
各家论述：《千金方衍义》：此方专主五贲下血。方用角（角思）以治带下血崩，鹿茸以治漏下恶血，一止一散，先为五贲之专药；禹余粮以治带下赤白，血闭癥瘕，能行能止，匡佐上二味之功益力；更以龙骨辅角（角思）、乌贼辅鹿茸，皆寓止散之机；阿胶专主内崩，干姜专温中气，小豆专清小肠，当归、续断专主冲带二脉之病，为崩带之紧关也。
用药禁忌：None
附注：None
