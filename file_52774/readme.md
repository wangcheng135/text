序号：52789
名称：香桂丸
组成：桂（去粗皮）半两，芎藭半两，肉豆蔻（去壳）半两，人参半两，赤茯苓（去黑皮）半两，附子（炮裂，去皮脐）半两，木香半两，白芷半两，当归（切，焙）半两，槟榔（锉）半两，黄耆（锉）半两，山芋半两，泽泻半两，京三棱（煨，锉）半两，枳壳（去瓤，麸炒）半两，干漆（炒烟出）半两，楮实（炒）半两，牛膝（去苗，酒浸，切，焙）半两，牡丹皮半两，陈橘皮（汤浸去白，炒）半两，独活（去芦头）半两，防风（去叉）3分，芍药3分，吴茱萸（汤浸，焙干，炒）3分。
出处：《圣济总录》卷一五○。
主治：妇人血风，荣卫气涩，经脉不调，皮肤不泽，肢体烦热，头目昏眩，骨节痠疼。
加减：None
功效：None
用法用量：每服20丸，空心、晚食前温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
