序号：43437
名称：养真丹
组成：补骨脂（炒）半两，益智仁半两，晚蚕蛾（微炒）半两，没药（研）半两，丁香半两，青盐（研）半两，川山甲（炙）半两，茴香3钱，白术3钱，乳香（研），南青皮3钱，沉香（锉）1两，香附子（炒）1两，姜黄1两，薯蓣1两，木香1两，甘草（炙）1两，巴戟（去心）1两，川楝子（去皮及子，麸炒黄色）1钱，牛膝（酒浸1宿）7钱，苁蓉（酒浸1宿）7钱，檀香7钱，苍术3两（酒浸2宿），蛤蚧1对，缩砂仁半两。
出处：《御药院方》卷六。
主治：阴衰消小，痿弱不举。
加减：None
功效：None
用法用量：每服40丸，空心及食前温酒送下，1日2次。
制备方法：上为细末，酒浸面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“养真丸”。
