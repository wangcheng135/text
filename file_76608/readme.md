序号：76624
名称：舒郁丹
组成：香附（炙）2两，厚朴（炙）1两，白芍5钱，枳壳（炒）1两，橘皮4两，川芎5钱，丹皮5钱，片姜黄1两，青皮（炒）1两，柴胡5钱，橘核（炒）5钱，山楂5钱，玄胡（炙）1两，槟榔5钱，六神曲（炒）5钱，川楝子5钱，大黄3钱，甘草2钱，麦芽（炒）5钱，当归2两，赭石（煅）1两，沉香3钱，砂仁5钱，豆蔻仁3钱，木香5钱，朱砂1钱。
出处：《北京市中药成方选集》。
主治：肝郁气滞，膨闷胀饱，两胁刺痛，呃逆恶心。
加减：None
功效：舒郁顺气，健胃化滞。
用法用量：每服2-3钱，温开水送下，1日2次。
制备方法：上为细末，水为小丸，每16两用滑石细粉4两为衣。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
