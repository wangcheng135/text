序号：47927
名称：神圣粉
组成：轻粉1分。
出处：《圣济总录》卷一一○。
主治：痘疮入眼睛，上作白翳，遮障不明，刺痛不可忍者。
加减：None
功效：None
用法用量：用时以指甲挑入不病眼边耳内，须侧卧，摇令药到耳底，以软纸塞耳。当时不痛，即开得眼。或未效，次日再如前法煅，取1豆大，挑入病眼边耳内，立效。
制备方法：先用少许炭灰渗地面如钟大，次烧小木炭10数片，簇定，吹令火盛，即倾粉其上，以新大瓷钟一只盖之，四下速拨灰塞缝，勿令烟透，更滴十数滴冷水于钟底，候冷取起钟；见粉烟飞粘钟内，旋以指点水刮洗下，用两重楮纸裹盛，绞出水银珠子，如赤豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
