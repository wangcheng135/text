名称：沉香快气丸
组成：京三棱（泡去皮）4钱，蓬术（煨）4钱，白茯苓4钱，青皮（去白）4钱，砂仁4钱，苍术（米泔水浸，炒）4钱，益智（去皮）4钱，白术4钱，神曲4钱，黑牵牛（头末）4钱，商陆（白的）4钱，大麦芽4钱，连翘4钱，藿香叶4钱，草果（去皮）4钱，丁香3钱，肉桂3钱，姜蚕3钱，沉香2钱，大腹皮2钱，雄附子5钱（看病冷热，热者不用）。
出处：《摄生众妙方》卷六。
主治：十种臌症。
加减：None
功效：None
用法用量：每服30-40丸。
制备方法：上为细末，面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌房事、辛辣、油腻、湿热之物，避暑湿，并忌盐酱油醋。
