序号：58555
名称：秘传大补元丸
组成：黄柏（蜜炒褐色）3两，知母（乳汁浸，炒）3两，龟版（酥炙）3两，淮熟地黄（酒洗）5两，牛膝（酒洗）1两半，麦门冬（去心）1两半，肉从蓉（酒洗）1两半，虎胫骨（好酒炙）1两半，淮山药1两半，茯神（去心）1两半，黄耆（蜜炙）1两半，杜仲（去粗皮，好酒炒断丝）2两，枸杞子（甘州者佳）2两，何首乌（篾刮去皮）2两，人参（去芦）2两，当归身（酒洗）1两，天门冬（去心）1两，五味子（去枝核）1两，淮生地黄（酒洗）1两，白芍药（酒炒）2两（冬月只用1两），紫河车1具（1名混沌皮，即今之胞衣，取初产者为佳。如无初产者，或壮盛妇人胎者亦可。取1具，用线吊于急流水中漂1昼夜，去其污浊血丝，取起，再用净米泔水1碗许，于小罐内微火煮1沸，取出勿令泄气，再用小篮1个，四周用纸密糊，将河车安于篮内，用慢火烘干，为末）。
出处：《直指附遗》卷九。
主治：男妇诸虚百损，五劳七伤，形体羸乏，腰背疼痛，遗精带浊。
加减：梦遗白浊，加牡蛎1两，白术，山茱萸各1两5钱，茯苓2两；冬加干姜5钱（炒黑色）。
功效：None
用法用量：每服80丸，空心以淡盐汤送下；寒月用温酒送下。
制备方法：上为极细末，入猪脊髓3条，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
