序号：18817
名称：瓜蒌煎
组成：瓜蒌2两，茯神1两，石斛1两，人参1两，肉灰蓉1两，甘草半两，黄连半两，当归半两，五味子半两，丹参半两，知母1两，胡麻1两，地骨皮2两，葳蕤2两，蜜5合，生地黄汁1升，牛髓1合，淡竹叶50片，生麦门冬汁5合，生姜汁1合。
出处：《鸡峰》卷七。
主治：虚劳燥渴，四体虚乏，羸瘦。
加减：None
功效：None
用法用量：每服不拘时候，以粥饮调下半匙。
制备方法：以水3升，煮地骨皮，葳蕤、胡麻、淡竹叶4味，去滓取汁1升，和地黄汁、麦门冬汁、牛髓、蜜、姜汁等，入前药未，搅令匀，又煎成膏，入于铜器中。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
