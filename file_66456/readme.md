序号：66472
名称：小儿回春丹
组成：橘红3两5钱，胆南星3两5钱，防风3两5钱，竹叶3两5钱，桑叶3两5钱，金银花3两5钱，连翘3两5钱，羌活3两5钱，茯苓2两，僵蚕2两，甘草2两，麻黄2两5钱，薄荷2两5钱，蝉蜕2两5钱，赤芍2两5钱，川贝2两5钱，牛蒡子2两5钱，三春柳1两5钱，杏仁1两5钱。
出处：《全国中药成药处方集》（北京方）。
主治：小儿热毒过盛，隐疹不出，发热咳嗽，烦躁口渴。
加减：None
功效：清热透表，化毒豁痰。
用法用量：每服5丸，鲜芦根煎水送下，温开水亦可，1日2次。小儿3岁以下者酌减。
制备方法：上为细末，兑入牛黄4钱、冰片4钱，麝香7钱2分，和匀，用糯米6两熬水泛小丸，朱砂为衣，每丸干重约2分，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：忌风寒，及一切荤食面食。
附注：None
