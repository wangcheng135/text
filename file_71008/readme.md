序号：71024
名称：葛根橘皮汤
组成：葛根2两，橘皮2两，杏仁2两（去尖皮），麻黄2两（去节），知母2两，黄芩2两，甘草2两（炙）。
出处：《外台》卷四引《小品方》。
主治：冬温未即病，至春被积寒所折，不得发，至夏得热，其春寒解，冬温毒始发出肌中，斑烂隐疹如锦纹，壮热而咳，心闷，呕但吐清汁。时气2-3日不解，头痛，壮热恶寒。
加减：None
功效：None
用法用量：以水7升，煮取3升，分3次温服。呕闷吐当先定，便且消息。
制备方法：上切。
临床应用：None
各家论述：《医方考》：冬月腠理闭密，故用麻黄以发表；肌属阳明，故用葛根以解肌；咳为肺气不利，故用橘皮、杏仁以利气；闷为心膈有热，故用黄芩、知母以清热；辛甘发散为阳，故佐以甘草，且调诸药而和中也。
用药禁忌：None
附注：葛根散（《圣惠》卷十五）、葛根汤（《圣济总录》卷二十二）、葛根陈皮汤（《永类钤方》卷八）。
