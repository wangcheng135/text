序号：52954
名称：香橘丹
组成：茯苓18两，苍术（炒）18两，白术（炒）18两，橘皮18两，香附（炙）18两，山药12两，法半夏12两，白扁豆12两，薏米（炒）12两，莲肉12两，枳实（炒）12两，厚朴（炙）12两，山楂12两，神曲（炒）12两，麦芽（炒）12两，砂仁6两，泽泻6两，甘草6两，木香3两。
出处：《北京市中药成方选集》。
主治：肠胃虚弱，消化不良，胃口不开，慢性肠胃炎；小儿脾胃衰弱，吐泻，久泻，久痢，大小便不分。
加减：None
功效：理脾止泄，健胃消食。
用法用量：每服1丸，温开水送下。1日2次。周岁以内小儿酌减。
制备方法：上为细末，炼蜜为丸，每丸重1钱。
临床应用：None
各家论述：None
用药禁忌：忌食生冷、油腻。便秘者勿服。
附注：小儿香橘丹（《中药制剂手册》）。
