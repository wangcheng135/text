序号：11469
名称：牛黄散
组成：牛黄3分（细研），龙脑半两（细研），天麻3分，桂心1两，人参半两（去芦头），芎藭半两，独活半两，乌蛇2两（酒浸，去皮骨，炙微黄），枳壳半两（麸炒微黄，去瓤），秦艽3分（去苗），防风3分（去芦头），蝎尾半两（微炒），天雄3分（炮裂，去皮脐），甘草半两（炙微赤，锉），金箔50片（细研），藁本3分，银箔50片（细研），当归3分（锉，微炒），天南星3分（炮裂），麝香半两（细研）。
出处：《圣惠》卷七十七。

制备方法：上为细散，都研令匀。
