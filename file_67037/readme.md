序号：67053
名称：麻黄引气汤
组成：麻黄5分，杏仁5分，生姜5分，半夏5分，石膏8两，紫苏4分，白前3分，细辛3分，桂心3分，竹叶（切）1升，橘皮2分。
出处：《千金》卷十七。
主治：肺劳实，气喘鼻张，面目苦肿。
加减：None
功效：None
用法用量：上（口父）咀。以水1升，煮取3升，去滓，分3次服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：劳役而邪并于肺，故用金匮厚朴麻黄汤中麻黄、石膏、细辛以泄肺满。泽漆汤中半夏、生姜、白前、桂心以涤痰垢，参入紫苏、橘皮、竹叶以助麻黄、半夏、石膏之力，引清气上升，浊气下降，喘息面肿随手可愈矣。
用药禁忌：None
附注：None
