序号：82522
名称：磨腰丹
组成：木香（不见火）1两，母丁香（不见火）1两，辰砂（别研）1两，附子（炮，去皮脐）1两，沉香（不见火）1两，藿香（去土）1两，干姜（泡，洗）1两，零陵香（不见火）1两，硫黄1两，陈橘皮（去白）1两，肉桂（去粗皮，不见火）1两，吴茱萸（汤泡10次，炒）1两，麝香（别研）1两，腻粉1两（别研）。
出处：《魏氏家藏方》卷八引史越王方。
主治：腰痛。
加减：None
功效：None
用法用量：每用时，先以生姜自然汁半盏于铫内，煎令沸，顷药在盏内，浸少时，以指头揩研令尽，放温于密室中，令人蘸药腰上摩之。用至10丸，骨健身轻，精华气至，骨体坚实。
制备方法：上为细末，炼蜜为丸，如鸡头大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
