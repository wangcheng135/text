序号：48430
名称：神效胜金丹
组成：香附16两，川芎1两半，丹皮2两半，当归1两半，玄胡1两半，牛膝2两半，远志1两半，熟地4两半，赤芍1两半，白术1两半，白薇4两，白芍1两半，炙草7钱半，白石脂1两，藁本3两，茯苓2两半，乳香1两，没药1两，赤石脂1两，白芷1两半，贡桂2两半，山参1两半，琥珀5钱，朱砂5钱，鹿茸2两。
出处：《全国中药成药处方集》（吉林、哈尔滨方）。
主治：气血虚脱，中气微弱，自汗形消，面色苍白，爪枯肤燥；经血暴崩或点滴不断，腰酸腿软，头晕气短；积湿浸带，带脉不宣，带下赤白，腰酸腿痛；子宫寒冷，血分虚弱，经血不调，久不受孕。
加减：None
功效：温补，收涩，益气，养血。
用法用量：每服1丸，白水调服。
制备方法：琥珀、朱砂均各另研，余药均一处研细，调匀，炼蜜为丸，大赤金为衣，每丸重2钱1分，除包装外，用瓷坛保贮。
临床应用：None
各家论述：None
用药禁忌：干血痨及瘀血实症均忌用。
附注：None
