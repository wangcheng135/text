序号：52086
名称：大补元汤
组成：人参2钱，淮山药（炒）2钱，黄耆（蜜炒）2钱，白术2钱，熟地2-3钱或多加，当归2-3钱，山茱萸1钱，枸杞2-3钱，甘草1-2钱，5味（蜜炒）7分，杜仲（姜炒）2钱，生姜8分，红枣3枚。
出处：《会约》卷十。
主治：气血虚甚，元气将脱，一时昏沉、掉摇等证。
加减：如元阳不足多寒者，加附子、肉桂、炮姜之类；如无力用人参者，再加黄耆3-4钱；如气滞，减黄耆2分，加去白陈皮1钱；如血滞，去山茱萸，加川芎1钱；如泄者，去当归，加补骨脂钱半、肉豆蔻1钱（面炒）；如腹痛喜按者，加吴茱萸，汤泡2次，用1钱。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
