序号：48820
名称：禹功散
组成：黑牵牛（头末）4两，茴香1两（炒），（或加木香1两）。
出处：《儒门事亲》卷十二。
主治：产后瘀血不行，脐腹疼痛；阳证水肿，寒湿水疝，属实证者。
加减：None
功效：泻水。
用法用量：以生姜自然汁调1-2钱，临卧服。
制备方法：上为细末。
临床应用：None
各家论述：1.《医方集解》：此足少阴、太阳药也。牵牛辛烈，能达右肾命门，走精隧，行水泄湿，兼通大肠风秘、气秘；茴香辛热温散，能暖丹田，祛小肠冷气，同入下焦以泄阴邪也。_x000D_
2.《古方选注》：禹功者，脾湿肿胀肉坚，攻之如神禹决水。牵牛苦热，入脾泻湿，欲其下走大肠，当从舶茴辛香引之，从戊入丙至壬，开通阳道，走泄湿邪，决之使下，一泻无余，而水土得平。
用药禁忌：此方峻猛，不可轻用。
附注：《得效》用生姜自然汁调药少许灌之鼻中。
