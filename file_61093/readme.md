序号：61108
名称：铁粉煎
组成：铁粉1两，牛黄1分（细研），菖蒲3分，酥3两，犀角屑1钱，人参（去芦头）1钱，茯神1钱，百合1钱，防风（去芦头）半两，川大黄（锉碎）1钱，青黛（细研）1钱，细辛1钱，远志（去心）1钱，芎藭1钱，麻黄（去根节）1钱，薯蓣1钱，甘草（炙微赤，锉）半两，蜜半斤。
出处：《圣惠》卷八十三。
主治：小儿心热，多惊悸，昼愈夜甚，象鬼神所著。
加减：None
功效：None
用法用量：每服2大豆许，以温水调下，每日3-4次。
制备方法：上为粗末，用水3升，入银锅中煎至半升，以新棉滤去滓，却入银锅内入研了药及酥、蜜，以慢火熬，不住手以柳篦搅如稠饧，收入瓷盒中。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
