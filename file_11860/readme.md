名称：风损膏药
组成：当归2两，川乌2两，草乌2两，枳壳2两，乌梢蛇2两，宣木瓜2两，薄荷2两，僵蚕2两，蝉蜕2两，白芷2两，木鳖子2两，海桐皮2两，何首乌3两，大川芎3两，汉防己3两，赤芍药3两，麻黄3两，防风3两，羌活3两，独活3两，苍术3两，五加皮4两。
出处：《穷乡便方》。
主治：风损骨节疼痛。
加减：None
功效：None
用法用量：用清油23斤，同前药煎枯，滤过去滓，先下陀僧10斤，后下松香11斤，内用姜1斤、葱1斤捣烂，共煮松香，用脚盆1双注水半盆，粗布滤净松香，入膏内。
