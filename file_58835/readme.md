序号：58850
名称：秘旨乌骨鸡丸
组成：丝毛乌骨鸡1只（男用雌，女用雄，溺倒，泡去毛，竹刀剖胁，出肫肝内金，去肠秽，仍入腹内），熟地4两，北五味（碎）1两（2味入鸡腹内，陈酒、童便各2碗，砂锅内水煮，旋添至磨烂汁尽），绵耆（去皮，蜜水拌，炙）3两，于术（饭上蒸9次）3两，白茯苓（去皮）2两，归身（酒洗）2两，白芍（酒炒）2两，5味为粗末，同鸡肉捣烂焙干，骨用酥炙，为粗末，入下项药：人参3两（无力者，党参代），川芎1两（童便浸，晒），丹参2两（酒浸，晒）3味研末入前药中。
出处：《卫生鸿宝》卷五。
主治：妇人郁结不舒，蒸热咳嗽，月事不调，或久闭，或倒经，产后蓐劳，及崩淋不止，赤白带下，白淫；男子斫丧太早，劳嗽吐血而致虚损。
加减：骨蒸寒热，加炙七肋鳖甲3两，银柴胡、地骨皮各1两半；经闭，加肉桂1两，崩漏下血，倍熟地，加阿胶2两；倒经血溢，加麦冬2两；郁结痞闷，加童便制香附末1两，沉香5钱；赤白带下，加萆薢、四制香附各2两，蕲艾1两；白淫，倍参、耆、苓、术；血热，加生地2两；虚甚，倍加人参。
功效：None
用法用量：清晨沸汤送下3钱，卧时醇酒送下2钱。
制备方法：用干山药末6两糊为丸，大便实者，蜜丸亦可，晒干瓶贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
