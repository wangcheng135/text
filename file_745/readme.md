序号：745
名称：六甲神丹
组成：雄黄9两（水磨为上，或研亦可。须选光莹者）。
出处：《鸡峰》卷二十八。
主治：妇人产前产后。
加减：None
功效：None
用法用量：早晨服1丸，温酒送下(另丸些小小者，备小儿服)。
制备方法：上药入在一砂盒子内，平实为度；入蜜6两，浸盖雄黄于其上，用防风6截子，长短满盒子，面用盖子合定。胶土6斤，青盐4两，水化和泥，入纸筋(看多少入)，候泥熟软，搓作索子，交加缠固，上用泥碗子裹合，阴7日，通风处用筛子盛，候10分干透；于地上作炉，深1尺5寸，阔1尺2寸，内坐丹盒子，上用细桑柴灰(如无，用松柴灰)1斗，盖之令平，上用硬炭20斤，作五遍烧，勘养一伏时(须渐渐添炭，使火气匀为佳，炭少更添不妨，不得太过)，取出破盒，如青色为上，紫色、红色为次，用油单裹悬于井中，离水1尺，悬一伏时，取出，研极细，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
