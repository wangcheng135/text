序号：68701
名称：小红丸
组成：骨碎补6两（姜制，焙取），土当归6两（焙取），川乌6两（煨），白杨皮6两（焙），肉桂4两（不见火），莪术2两（焙），丁香3两，干姜2两（焙），川芎3两，细辛4两（焙），附子3两半（煨，去皮），乳香3钱（别研，不焙），没药3钱（别研），芍药6两（焙）。
出处：《理伤续断方》。
主治：踒折伤损，皮破骨出，手足碎断，筋肉坏烂，疼痛至甚，昼夜叫呼，百治不止；手足久损，筋骨差爻，举动不得；损后伤风湿，肢节挛缩，遂成偏废；劳伤筋骨，肩背疼痛，四肢疲乏，动用无力。
加减：None
功效：壮筋骨，治经络，生气血。
用法用量：每服30丸，温酒送。下，或用生姜煎酒，或盐汤送下，不拘时候；敷用生姜自然汁煎酒或盐汤皆可，不拘时候。
制备方法：上补、药、归、杨4味用当土者，余8味研为细末，乳、没别制，和醋糊为丸，如绿豆大，信朱为衣。
临床应用：None
各家论述：None
用药禁忌：孕妇莫服。
附注：《医统》有续断一两。
