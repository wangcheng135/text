序号：24928
名称：如圣丸
组成：全蝎1两半（酒洗），连翘1两半，天麻1两半，防风1两半，荆芥1两，川芎1两，白芷1两，当归1两（酒洗），黄柏1两，羌活1两，桔梗1两，大黄1两（煨），滑石1两，石膏1两（煅），白术1两，麻黄1两，苦参1两，僵蚕1两（炒），蝉蜕1两，芍药1两，山栀1两，枳壳1两，细辛1两，皂角刺1两，大风子肉1两，独活5钱，人参5钱，郁金5钱，芒消5钱，黄连5钱。
出处：《外科理例·附方》。
主治：癞风。
加减：如疮破裂，只用大风子壳煎汤洗，春夏滑石、石膏依方用，秋、冬2味减半，过春分、秋分服防风通圣散1帖，空心服，利3-4次，以粥补之。
功效：None
用法用量：每服50-70丸，用六安条煎汤送下，每日3次，半年全愈。小便尿如靛水黑色，此病久深者，只用此药2料。
制备方法：上为细末，用红米糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：如眉毛须发脱落日渐生者，切不可食羊肉、鹅、鸡、猪头、蹄、鲤鱼、生冷，如肯食淡，百日全愈。
附注：如圣丹（《寿世保元》卷九）。
