序号：40426
名称：虎杖饮
组成：虎杖1两半，柴胡（去苗）1两半，五味子（炒）1两半，熟干地黄（焙）1两半，白茯苓（去黑皮）1两半，陈橘皮（去白皮）1两半，麦门冬（去心，焙）1两半，黄芩（去黑皮）1两半，甘草（炙，锉）1两半，人参1两，川大黄（锉碎，微炒）1两半，黄连（去须）1两半，当归1两半，赤茯苓1两半，黄耆（锉）3分，生干地黄3分，赤芍药3分，栀子仁半两。
出处：《普济方》卷二三○。
主治：男子妇人冷劳，身体羸瘦。
加减：None
功效：None
用法用量：每服20丸，以温水送下，不拘时候。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
