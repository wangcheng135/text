序号：37014
名称：奇效加味消积肥儿丸
组成：人参3钱，白术1两（蜜水拌抄），白茯苓8钱（蒸），橘红5钱，金樱子5钱（去毛，用肉，略炒），青皮（去瓤，麸拌炒）5钱，粉草1钱5分（蜜水炙），使君子7钱（炒），芡实5钱（蒸过），莲肉心5钱（隔纸炒干），门冬1两5钱（去心），五谷虫1两（水洗净）山楂肉5钱（蒸过），鸡肫皮10个（火焙，雄者佳），麦芽（炒黄色）5钱。
出处：《活幼心法》卷九。
主治：小儿疳积，肚大青筋，骨瘦毛焦，泻痢不止者。
加减：如身热咳嗽，加地骨皮、百部各5钱；肚腹胀大，便稀水，肠鸣作声，或虫出不和，加槟榔5分，木香1钱。
功效：None
用法用量：每日午间服1丸。或将前药为极细末，以炼蜜每次和2-3匙服亦可。
制备方法：炼蜜为丸，如弹子大，每个重一钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
