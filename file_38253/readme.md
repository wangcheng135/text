序号：38263
名称：枇杷叶散
组成：枇杷叶（去毛，炙）半两，陈皮（汤浸，去瓤，焙）半两，丁香半两，厚朴（去皮，涂姜汁炙）4两，白茅根1两，麦门冬（去心，焙）1两，干木瓜1两，甘草（炙）1两，香薷3分。
出处：《局方》卷二。
主治：冒暑伏热，引饮过多，脾胃伤冷，饮食不化，胸膈痞闷，呕哕恶心，头目昏眩，口干烦渴，肢体困倦，全不思食，或阴阳不和，致成霍乱，吐利转筋，烦躁引饮。暑毒攻心，呕吐鲜血。
加减：None
功效：None
用法用量：每服2钱，水1盏，加生姜2片，煎至7分，去滓温服；温水调下亦得。如烦躁，用新汲水调下，不拘时候。小儿3岁以下，可服半钱。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《兰台规范》引作“枇杷散”。
