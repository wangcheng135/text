序号：75520
名称：解毒玉壶丸
组成：白茯苓、贯众、硼砂（别研）、马屁勃2钱，紫河车（水煮）4钱，山豆根4钱，乌鱼骨4钱，金星凤尾草4钱，山药4钱，白术4钱，白僵蚕4钱，密陀僧4钱，大甘草1钱，寒水石3钱，坯子胭脂1钱，乳香4钱（别研），麝香半钱（别研），象牙末1钱。
出处：《普济方》卷六十一引《卫生宝鉴》。
主治：喉风喉闭，及误吞针钱竹刺物，及诸恶毒物。
加减：None
功效：None
用法用量：服时将药1丸，水1盏，浸1茶时，磨药数遍饮之，余药留再服，一药医数人之疾。
制备方法：上各炒焙为末，内乳香、麝香、胭脂、硼砂各研，入众药末中拌，用薄面糊为丸，如弹子大，再用煅过软石膏，研作末为衣，阴干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
