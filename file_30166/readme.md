序号：30174
名称：连翘败毒散
组成：柴胡、羌活、桔梗、金银花、连翘、防风、荆芥、薄荷叶、川芎、独活、前胡、白茯苓、甘草、枳壳。
出处：《古今医鉴》卷十五。
主治：痈疽，发有疔疮，乳痈，一切无名肿毒，初起憎寒壮热，甚者头痛拘急，状似伤寒；暑疡。
加减：如热甚并痛甚，加黄连、黄芩；大便不通，加大黄、芒消下之。
功效：None
用法用量：加生姜，水煎，如疮在上，食后服；在下，食前服。1日至4-5日者，2-3剂以解其毒，轻者则内自消散。若至6-7日不消，宜服真人活命饮，后服托里消毒散调理。
制备方法：上锉。
临床应用：None
各家论述：None
用药禁忌：None
附注：连翘散毒散（《杏苑》卷八）、败毒散（《杂病源流犀烛》卷十五）。
