序号：26832
名称：延胡索汤
组成：当归（去芦，酒浸，锉炒）半两，延胡索（炒去皮）半两，蒲黄（炒）半两，赤芍药半两，官桂（不见火）半两，片子姜黄（洗）3两，乳香3两，没药3两，木香（不见火）3两，甘草（炙）2钱半。
出处：《济生》卷六。
主治：妇人室女，七情伤感，遂使血与气并，心腹作痛，或连腰胁、或引背膂，上下攻刺，甚至搐搦，经候不调，一切血气疼痛。
加减：吐逆加半夏、橘红各半两。
功效：None
用法用量：每服4钱，水1盏半，加生姜7片，煎至7分，去滓，食前温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《得效》引作“玄胡索汤”；《东医宝鉴·外形篇》引作“玄胡索散”；方中片子姜黄，《妇科玉尺》作“姜汁炒黄连”。
