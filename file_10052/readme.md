序号：10052
名称：加味地黄丸
组成：地黄丸加五味子1两，肉桂1两。
出处：方出《万氏女科》卷三，名见《金鉴》卷六十二。
主治：盘肠产后下元虚者,痈疽已溃，虚火上炎，口干作渴者。
加减：None
功效：固下元。
用法用量：None
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：原书云，患盘肠产，欲免其苦者，应于此后无孕时多服地黄丸加五味子一两，肉桂一两。《金鉴》本方用熟地（酒蒸，捣膏）八两，山药（炒）四两，山萸肉（去核）五两，白茯苓四两，牡丹皮（酒洗）四两，泽泻（蒸）三两，肉桂六钱，五味子（炒）三两。为末，炼蜜为丸，如梧桐子大。每服三钱，空心盐汤送下。
