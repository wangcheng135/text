序号：37758
名称：拔生膏
组成：血竭1钱，蟾酥3钱，麝香5分，雄黄5钱，轻粉3钱，乳香2钱，没药2钱。
出处：《摄生众妙方》卷八。
主治：诸般恶疮，及瘰疬、鼠疮才起。
加减：None
功效：None
用法用量：凡诸疮点在当头，1日2次，次日又1次，疮头蚀破约5分，血水出为妙。恐日久药干，将前次留灰汤和用。
制备方法：上用荞麦稭灰或真炭灰1斗2升，淋灰8-9碗，用栗柴或桑柴文武火煎作3碗，取1碗收留，将2碗盛于好瓷器内候温，将前7昧药碾为极细末，入灰汤内用铁瓢或桑柳枝搅，又用好细石灰1升，入药灰汤搅匀，取出候冷过宿，盛入小白瓷罐内。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
