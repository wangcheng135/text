序号：48460
名称：神效救命丹
组成：朱砂1两，麝香半两，雄黄2钱半，黄丹2钱半，巴豆2分（去皮油），斑蝥2钱半（去头足，一半生一半炒用），蜈蚣2条（1生1熟上7味各细研），苦药子2钱半，山豆根1两，续随子2钱半（去皮，生用）。
出处：《鸡峰》卷二十七。
主治：诸蛇毒，蛊毒，一切药毒。
加减：None
功效：解中毒。
用法用量：每粒可救男5妇3。凡初中毒，便令人嚼生黑豆取验，急将药用真腊茶清送下1丸，须臾，患人自觉心头如断皮条之声，其毒从目口鼻出或大便下，中毒未久即血，久即成虫，更将药净洗收之再用；一切蛇、蝎、蜈蚣、马汗毒伤，以药点好醋磨汁，涂之立解。常于端午日修合，不及于9月9日亦得。
制备方法：上为细末，和匀，用糯米煮糊为丸，如10斤鱼眼大。
临床应用：None
各家论述：None
用药禁忌：大忌酒肉毒物1月。
附注：None
