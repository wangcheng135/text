序号：24231
名称：安胎饮
组成：绵耆3-4钱（炙），生地炭3钱，归身炭2钱，茯苓3钱，泽泻2钱，升麻（炒）5分，银花3钱，条岑（酒炒）1钱5分，川连（酒炒）8分，广木香5分，范制面2钱。
出处：《医方简义》卷五。
主治：妊妇患痢，名子痢。腰痛气滞，里急后重，少腹（？？）痛。
加减：如噤口，水汤不进而呕吐频频，加石莲子3钱，石菖蒲3分，生姜3片，去生地炭；口渴者，加青果1枚，乌梅1枚；如赤痢加地榆炭3钱；白痢，加白槿花1钱；如腹痛甚者，加川椒20粒，去升麻，更加白芍1钱；如赤白兼者，加天仙藤2钱、驴胶1钱，去广木香；或外加扁豆叶2片，以醒胃气。
功效：淡渗利湿，清热安胎。
用法用量：加荷叶1角，水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：口渴者，切忌生冷水果。
附注：None
