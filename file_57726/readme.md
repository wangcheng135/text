序号：57741
名称：烧肝散
组成：芍药1两，桔梗1两，缩砂仁1两，附子（炮，去皮）1两，茴香（炒）1两，干姜（炮）1两，苍术（炒）1两，良姜1两，桂1两，红豆半两，川椒（炒）半两，白术半两，肉豆蔻半两，橘皮半两，干山药4两。
出处：《医方类聚》卷一五三引《施圆端效方》。
主治：脾虚劳损，年深泄泻，久作滑肠，瘦困减食，饮食虽多，不生肌肉，口疮燥渴，男子诸虚百损，妇人血气劳伤，少食而胀满，体无光泽，真气衰乏，少力好睡。
加减：None
功效：None
用法用量：每用3钱，猪肝3两，薄批作3片，掺药上，铺生姜、葱丝1重，卷定，麻扎了，湿纸数十重裹，慢火烧熟，分3-4次细嚼，米饮下，每日1次。或细切肝，拌药末作馅，法面包作角子5-6个，烧熟，每日1次亦佳。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
