序号：77790
名称：雄鸡汤
组成：雄鸡1只（治如食法），甘草2两，人参2两，茯苓2两，阿胶2两，黄芩1两，白术1两，麦门冬5合，芍药4两，大枣12枚（擘），生姜1两（一方用当归2两，芎2两，不用黄芩、生姜）。
出处：《千金》卷二。
主治：妊娠三月，有寒，大便青，有热，小便难，不赤即黄，卒惊恐忧愁，嗔怒喜顿仆，动于经脉，腹满绕脐苦痛，或腰背痛，卒有所下。
加减：None
功效：None
用法用量：上(口父)咀。以水1斗5升，煮鸡减半，出鸡纳药，煮取半，纳清酒3升，并胶煎，取3升，分3服，1日尽之。当温卧。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌海藻、菘菜、酢物、桃、李、雀肉等。
附注：None
