序号：23001
名称：达生散
组成：大腹皮3钱，人参半钱，陈皮半钱，白术1钱，芍药1钱，紫苏（茎叶）半钱，甘草（炙）2钱，归身尾1钱。
出处：《丹溪心法》卷五。
主治：气血虚弱，胎产不顺。
加减：夏月，加黄芩（冬不必加）；春，加川芎；气虚，加参、术；气实，倍香附、陈皮；血虚，倍当归，加地黄；形实，倍紫苏；性急，加黄连；有热，加黄芩；湿痰，加滑石、半夏；食积，加山楂；食后易饥，倍黄杨树叶捎；有痰，加半夏；腹痛，加木香、桂。
功效：补益气血，顺气安胎，催产。
用法用量：上作1服。加青葱5叶，黄杨树叶梢7个，或加枳壳、砂仁以水煎，食后服。于妊娠8-9个月，服10数帖。
制备方法：None
临床应用：None
各家论述：1.《医方考》：《诗》云，诞弥厥月，先生如达。朱子曰：先生，首生也。达，小羊也。羊子易生而无留难，故昔医以此方名之。然产难之故，多是气血虚弱，营卫涩滞使然。是方也，人参、白术、甘草养其气；当归、芍药养其血；紫苏、腹皮、陈皮流其滞。气血不虚不滞，则其产也犹之达也。_x000D_
2.《医方集解》：此足太阴、厥阴药也。当归、芍药以益其血；人参、白术以益其气；腹皮、陈皮、紫苏、葱叶以疏其壅。气血不虚不滞，则临产自无留难之患矣。
用药禁忌：None
附注：束胎散（原书同卷）、束贻饮（《丹溪治法心要》卷七）、缩胎饮（《同寿录》卷三）、紫苏饮（《女科切要》卷五）。
