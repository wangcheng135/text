序号：26814
名称：延龄丹
组成：熟地黄半两，川芎半两，防风半两，槟榔半两，芜荑（炒）半两，蝉蜕（洗）半两，柏子仁（别研）半两，马牙消（烧）半两，人参半两，黄耆半两，白蔹半两，川椒半两，鲤鱼鳞（烧）1两，晚蚕砂（炒）1两，当归1两，木香1两，附子（炮，去皮脐）1两，石膏（煅）1两，泽兰1两，藁木1两半，厚朴（姜制，炒）1两半，甘草（炙）1两半，白姜（炮）1两半，红花（炒）1分，吴茱萸（洗）1分。
出处：《三因》卷十八。
主治：血瘕块痛，血劳血虚，血崩，血气痛，血风，血晕闷绝，胎死腹中，胞衣不下。
加减：None
功效：None
用法用量：每服1丸，血瘕块痛，绵灰酒送下；催生，温酒细嚼下；血劳血虚，桔梗酒送下；血崩，棕榈灰酒送下；血气痛，炒白姜酒送下；血风，荆芥酒送下；血晕闷绝，胎死腹中，胞衣不下，并用生地黄汁、童便、酒各1盏，煎2服调下；常服，醋汤温酒化下，并空心食前服。
制备方法：上为末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：妙应丹（《局方》卷九续添诸局经验秘方）。
