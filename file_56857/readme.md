序号：56872
名称：消肾脱钳丸
组成：泽泻6两，木猪苓（去黑皮）6两，白术（炒）6两，木通（去皮）6两，青皮（去瓤）6两，青木香6两，陈皮（去白）6两（为末），川楝子（炮，去核）12两，胡芦巴（炒）12两，茴香（淘去砂，炒）12两，破故纸（炒）12两（用斑蝥3两各炒香熟，并去斑螫不用），黑牵牛1斤半（半生用，半炒熟，为末1斤）。
出处：《魏氏家藏方》卷二。
主治：一切下部疾痛不可忍者。
加减：None
功效：None
用法用量：每服40或50丸，先用桃仁7粒，茴香1撮，葱头3茎同炒香熟，细嚼温盐酒送下。
制备方法：上为细末。酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
