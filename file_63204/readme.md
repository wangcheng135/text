序号：63220
名称：清热解毒消肿汤
组成：连翘5钱，公英5钱，金银花5钱，野菊花3钱，黄芩3钱，瓜蒌1两，生地5钱，甘草2钱。
出处：方出《赵炳南临床经验集》，名见《千家妙方》。
主治：鼻前庭疖肿。
加减：None
功效：清肺经热，解毒消肿。
用法用量：None
制备方法：None
临床应用：鼻前庭疖肿：关某某，男，34岁。患者于8天前右则鼻孔生疮，日渐增大，局部红肿，恶寒发热，恶心，大便秘结，口渴心烦，脉细数，舌质稍红，苔薄黄。体温38.7℃，局部脓头欲破溃。此系肺热不宣，火毒凝结。治以清肺经之热，解毒消肿。投以本方，外用化毒散软膏，3剂后红肿已消，身热已退，再进3剂治愈。
各家论述：None
用药禁忌：None
附注：None
