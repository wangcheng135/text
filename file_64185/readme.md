序号：64201
名称：大腹木香汤
组成：大腹（锉）1两，木香1两，前胡（去芦头）半两，肉豆蔻（去壳）半两，人参半两，白茯苓（去黑皮）半两，京三棱（炮，锉）1两半，干姜（炮）半两，青橘皮（汤浸，去白，焙）半两，诃黎勒（炮，去核）半两，陈曲（微炒）1两，桂（去粗皮）半两，大麦芽（微炒）半两，厚朴（去粗皮，生姜汁炙）1两，半夏（汤洗7遍，去滑，焙）半两。
出处：《圣济总录》卷四十六。
主治：脾气虚弱，心腹胀满，呕吐痰逆，胸膈不利，腹胁刺痛，不思饮食。
加减：None
功效：None
用法用量：每服3钱匕。水1盏，加生姜3片，大枣2枚（去核），同煎至7分，去滓，稍热食前服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
