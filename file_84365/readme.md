序号：84383
名称：七圣散
组成：黄雌鸡1只（料如食法，净去毛，勿令着水，于腹下开1小窍，去肠肚，令极净，却再入心、肝用），蜀椒（去目并合口者）1分，生地黄1升（洗，肥者），生姜（去皮）1两，黄耆（锉）1两，陈橘皮（汤浸，去白，焙）1两，人参1两。
出处：《圣济总录》卷九十三。
主治：骨蒸积癖，鬼气疰忤；男女虚损，手足烦疼，背膊酸重，至夜病甚，四肢清瘦，颜色萎黄，两膝疼冷，腹中雷鸣，时多泄利，饮食无味，行步不能；及五脏虚劳。
加减：None
功效：None
用法用量：每服1钱匕，米饮调下，日3次；其鸡劈碎掺少盐，令患人恣意食之，饱即止。良久厚衣被覆取汗。汗出多，即以牡蛎烧捣为粉敷之。
制备方法：上七味，除鸡外，各锉如麻豆大，和匀，入在鸡腹内，却缝合，以银石器盛，新布罩，坐于甑中蒸，甑一边用碗盛米并水半碗，同盖覆，勿令透气，候碗内米并鸡烂熟为度，取出药，别焙干，捣罗为散。
临床应用：None
各家论述：None
用药禁忌：勿冒风寒。
附注：None
