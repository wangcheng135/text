序号：17367
名称：玉杖丸
组成：大半夏14个（破作4片，修合时于屋下闲处地掘1小坑，深8寸，用火烧红。以薛荔包半夏，入坑内，沃以米醋1小碗，就用土盖之，周时取出半夏，用斑蝥21个，土狗49个，去翅足，炒半夏黄色，去斑蝥、土狗不用），石燕子2对（火煅醋淬，研细），阳起石1两半（青盐3钱，炒，去盐盐），灵砂1两半，龙骨1两半，诃子7个（煨，去核），砂仁25粒，麦门冬（去心）1两，原蚕蛾1两半（去翅足，炒，系未交公者），桑螵蛸1两半（斑蝥49个，炒，去斑蝥不用）。
出处：《医方类聚》卷一五三引《经验秘方》。
主治：肾虚生痰。
加减：None
功效：None
用法用量：每服50丸，空心温酒送下，干物压之。若要阳举，糯米粉为糊丸药。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
