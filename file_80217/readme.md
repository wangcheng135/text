序号：80233
名称：礞石丸
组成：硇砂1两（用米醋3升化开），巴豆霜2两半（2味同入醋煮两食久），青礞石半两（研），京三棱1两（醋浸1宿，煨，2味次之，煮半食久，入前醋中煮），白面2两（酒半升化，次入，煮半食久），大黄1两半（分三份，一生、一炒、一煨，次入，煮半食久），木香（以下并为细末）1两，槟榔1两，肉豆蔻1两，肉桂1两，猪牙皂角（去皮尖）1两，干姜（炮）1两，丁香1两，蓬莪术1两，青皮半两，芫花（醋浸1宿，炒，微令有烟）半两，好墨（烧，冷8分过）半两，白豆蔻半两，胡椒1分，粉霜1分（研）。
出处：《中藏经·附方》。
主治：脾积滞气，酒食所伤，饮食不化，恶心呕逆，胸膈不快，不思饮食，胸腹胀满，脐胁有块，心脾冷痛，口吐酸水，停饮冷痰，痃癖癥瘕，发痛无度，翻胃转食，面黄瘦乏，四肢头面浮肿，脏腑不调，里急后重，及十膈气，虚中有积，妇人血气块硬。
加减：None
功效：None
用法用量：每服3丸，酒、饮、姜汤杂下。
制备方法：上药次第煮了，次入木香等十四味，熬成膏为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
