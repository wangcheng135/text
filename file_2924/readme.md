序号：2924
名称：天南星丸
组成：天南星（腊月牛胆匮者）3分，白芷1两半，麻黄（去根节）1两，防风（去叉）1两半，羌活（去芦头）半两，独活（去芦头）半两，芎半两，天麻半两，白芍药半两，桔梗（锉，炒）半两，细辛（去苗叶）半两，白僵蚕（炒）半两，甘草（炙）1分半，干姜（炮）1分，龙脑（研）1钱，麝香（研）1分。
出处：《圣济总录》卷七。
主治：一切风，手足麻痹亸曳，或即肿痒疼痛。
加减：None
功效：None
用法用量：每服1丸，细嚼，以薄荷温酒送下，不拘时候；伤寒头目昏痛，肢节疼者，薄荷茶送下，并吃2-3服尤妙。
制备方法：上药除研外，为细末，和令匀，炼蜜为丸，如杏核大，丹砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
