序号：31161
名称：杜牛膝丸
组成：杜牛膝3两，甜肉桂1两半，瞿麦穗3两，当归尾3两，白通草1两半，冬葵子3两，飞滑石3两。
出处：《医略六书》卷三十。
主治：产后胞阻溺闭，脉沉者。
加减：None
功效：None
用法用量：每服3钱，童便煎，去滓温服。
制备方法：上为末，蜜为丸。
临床应用：None
各家论述：产后胞满气壅，阻塞膀胱窍道，故不能化气，而小便不通，胀急，危迫莫甚，牛膝去瘀血以利便，桂心暖瘀血以通闭，瞿麦穗降气通小便，当归尾破血下瘀结，通草利小便，滑石通利膀胱，冬葵子滑利窍道以下出胞衣也，蜜丸以润之，童便以降之，使气化溺通则胞门无阻而胞衣自出，何患胀急不退，危迫不安乎。
用药禁忌：None
附注：None
