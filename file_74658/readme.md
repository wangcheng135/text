序号：74674
名称：锦鸠丸
组成：草决明子3两，蕤仁（去皮）3两，羌活（去芦）3两，瞿麦3两，细辛（去苗）5两，牡蛎（洗，火煅取粉）5两，黄连（去须）5两，杜蒺藜（炒，去尖角）5两，防风（去芦）5两，肉桂（去粗皮）5两，甘菊花（净）5两，白茯苓（去皮）4两，斑鸠1只（去皮、毛、肠、嘴、爪，用文武火连骨炙干），羯羊肝1具（薄批，炙令焦），蔓荆子2升（淘洗，绢袋盛，饭甑蒸一伏时，晒干）。
出处：《局方》卷七。
主治：肝经不足，风邪内乘上攻，眼暗泪出，怕日羞明，隐涩痒痛，瞻视茫茫，多见黑花，或生翳膜。
加减：None
功效：None
用法用量：每服15-20丸，空心、日午、临卧以温水或温酒送下，1日3次。如久患内外障眼，服诸药无效者，渐加服50丸；暴赤眼疼痛，食后用荆芥汤送下20丸。
制备方法：上为末，炼蜜和丸，如梧桐子大。
临床应用：None
各家论述：《原机启微》：方以甘菊花、草决明主明目为君；以蕤仁、牡蛎、黄连、蒺藜除湿热为臣；以防风、羌活、细辛之升上，瞿麦、茯苓之分下为佐；以斑鸠补肾，羊肝补肝，肉桂导群药入热邪为使。
用药禁忌：None
附注：羊肝丸（《圣济总录》卷一一二）、神驻锦鸠丸（《原机启微》卷下）。
