序号：2242
名称：升麻汤
组成：升麻2分，当归2分，蜀椒（汗）1分，雄黄（研）1分，栀子1分，桂心1分，甘草2分（炙），鳖甲大如手1片（炙）。
出处：《外台》卷一引《古今录验》。
主治：伤寒1-2日便成阳毒，或服药吐下之后，变成阳毒，身重腰背痛，烦闷不安，狂言，或走，或见神鬼，或吐血下利，其脉浮大数，面赤斑斑如锦纹，喉咽痛唾脓血。
加减：阴毒，去雄黄。
功效：None
用法用量：以水5升，煮取2升半，分3次服。如人行5里久，再服。温覆手足，毒出则汗，汗出则解，不解重作服。亦取得吐佳。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：忌海藻、菘菜、生葱、苋菜。
附注：None
