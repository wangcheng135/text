序号：7417
名称：圣蚕丸
组成：白僵蚕（直者微炒）1两，天南星（浆水煮5-7沸）1两，防风（去芦）1两，半夏（先洗净，用浆水煮5-7沸）1两，白附子（浆水煮5-7沸）1两，人参1两，藿香半两（去尘土）。
出处：《永乐大典》卷九七五引《小儿保生要方》。
主治：小儿诸般生风，壮热，精神恍惚，痰涎壅塞，目睛上视，睡卧不安，头痛颊赤，多惊恐，肢体倦，不喜乳食。
加减：None
功效：None
用法用量：周岁儿每服5-7丸至10丸，用生姜。薄荷汤送下，不拘时候；2-3岁儿，可服15丸，余当以意加减。
制备方法：上为极细末，寒食面打薄糊为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
