序号：3198
名称：天雄丸
组成：天雄1两（炮裂，去皮脐），人参1两（去芦头），防风1两（去芦头），鹿茸1两（去毛，涂酥，炙微黄），远志1两（去心），牡蛎2两（烧为粉），薯蓣1两，泽泻1两，牛膝1两（去苗），黄芩1两（锉），五味子3分，山茱萸3分，肉苁蓉1两（酒浸1宿，锉，去皴皮，炙干），桃仁1两（汤浸，去皮尖双仁，麸炒微黄），熟干地黄1两。
出处：《圣惠》卷十四。
主治：伤寒夹劳，羸瘦，或时憎寒，卧即汗出，手足时颤，颊赤面黄。
加减：None
功效：None
用法用量：每服20丸，食前以姜、橘汤送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中黄芩，《普济方》引作“黄耆”。
