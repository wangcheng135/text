序号：77570
名称：干姜丸
组成：干姜（炮）1钱，巴豆（去心，炒黄，研）1钱，大黄（湿纸裹，甑上蒸）1钱，人参1钱（去芦）。
出处：《本事》卷四。
主治：忧愁中伤，食结积在肠胃，故发吐利。
加减：None
功效：None
用法用量：用温脾汤吞下1丸，米饮亦得。
制备方法：上除巴豆，余为末，同研，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《本事方释义》：干姜气味辛温，入手足大阴；巴豆气味辛温，入足太阴阳明；大黄气味苦寒，入足阳明；人参气味甘温，入足阳明。此即古方中之备急丸加参也。因忧愁中伤，食积久在肠胃，吐利频发，暑月更甚，以数年久不愈之证，欲攻病虑其体虚，欲补虚虑其留邪，故温下之药，佐以扶正，则两不相悖矣。
用药禁忌：None
附注：None
