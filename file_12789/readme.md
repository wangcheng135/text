序号：12795
名称：石斛丸
组成：石斛（去根，锉）1两，牛膝（去苗）1两，桂心1两，杜仲（去粗皮，炙微黄，锉）1两，续断1两，白茯苓1两，菟丝子（酒浸1宿，焙干，别捣）1两，枸杞子1两，五味子1两，山茱萸1两，黄耆（锉）1两，防风（去芦头）1两，肉苁蓉（酒浸1宿，刮去皴皮，炙干）1两，远志（去心）1两，人参（去芦头）1两，天门冬（去心，焙）1两，熟干地黄2两。
出处：《圣惠》卷二十七。
主治：风劳气，四肢羸弱，心神虚烦，食饮无味，肢节多疼，脚腰无力，夜多盗汗，小便赤黄。
加减：None
功效：None
用法用量：每服30丸，食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
