序号：3951
名称：无名异膏
组成：无名异（研）半两，没药（研）半两，麝香（研）半两，檀香（锉）半两，丹砂（研）半两，沉香（锉）半两，麒麟竭（研）半两，乳香（研）半两，突厥白（锉）半两，白蔹（锉）半两，白及（锉）半两，白芷（锉）半两，鸡舌香（研）半两，鸡骨香（研）半两，当归（切，焙）半两，芎（锉）半两，大黄（锉，炒）半两，牛膝（锉，酒浸，焙）半两，防风（去叉，锉）半两，槐枝（锉）半两，柳枝（锉）半两，桑枝（锉）半两，蜡4两，铅丹12两，青油2斤。
出处：《圣济总录》卷一八三。
主治：乳石、痈毒、发背。
加减：None
功效：None
用法用量：上药除油、蜡、丹及前8味研末外，并锉碎。先熬油令沸，下檀香等14味锉药，煎候白芷赤黑色，绞，去滓再煎，入蜡、铅丹，以柳蓖搅，候变黑色，滴于水中成珠子，软硬得所后，下无名异8味研末，搅令匀，以瓷合盛。用故帛涂贴疮上，每日换1次，以愈为度。
