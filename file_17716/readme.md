序号：17723
名称：白丸子
组成：安息香1两，胡桃仁（与安息香同研）1两，白胶香（研）1分，牛黄（研）1分，麝香1分，丹砂（研）1分，芎藭1分，当归（切，炒）半两，干蝎7枚（全者，酒炒），巴豆3粒（去皮心膜，研如膏，压去油）。
出处：《圣济总录》卷六。
主治：破伤风；兼治中风手脚挛缩及半身不随。
加减：None
功效：None
用法用量：每服1-2丸，温酒送下。量力饮之，但令醺醺，勿至大醉，兼理中风。手脚挛缩及半身不随，日2夜1，不可过多。
制备方法：上10味，将3味捣为细末，与别研7味和匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
