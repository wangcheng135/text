序号：20493
名称：何首乌丸
组成：何首乌5两（9蒸9晒，捣罗为末），干薄荷4两（捣罗为末），羊肉半斤（去脂膜），皂荚30梃（不蛀者。10梃去黑皮，涂酥炙令黄色，捣罗为末；10梃烧候火焰将尽，以碗合盖候冷，取出捣罗为末；10梃捶碎，用新汲水5升接取汁，生绢滤过）。
出处：《圣惠》卷六十六。
主治：风毒气滞，颈腋结成瘰疬，肿核不消。
加减：None
功效：None
用法用量：每服20丸，空心以温酒送下，薄荷汤下亦得。
制备方法：上先将皂荚水煮羊肉令烂，后取肉细研，入诸药末，和捣为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“祛风丸”，有玄参4两。
