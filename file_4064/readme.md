序号：4064
名称：木瓜丸
组成：熟干地黄（洗，焙）4两，陈皮（去白）4两，乌药4两，黑牵牛（炒）3两，石楠藤2两，杏仁（去皮尖）2两，当归2两，苁蓉（酒浸，焙）2两，干木瓜2两，续断2两，牛膝（酒浸）2两，赤芍药1两。
出处：《局方》卷一(续添诸局经验秘方)。
主治：肾经虚弱，下攻腰膝，沉重少力，腿部肿痒，疰破生疮，脚心隐痛，筋脉拘挛，或腰膝缓弱，步履艰难，举动喘促，面色黧黑，大小便秘涩，饮食减少，无问久新，并宜服之。
加减：None
功效：None
用法用量：每服30-50丸，空心木瓜汤吞下；温酒亦可。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：木瓜牛膝丸（《得效》卷九）。
