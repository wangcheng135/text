序号：30304
名称：阿魏良姜丸
组成：青皮3两，陈皮2两，良姜2两，红豆2两，桂（去粗皮）1两，缩砂（去皮）2两，蓬荗（炮）2两，草果子（去皮）2两，干姜（炮）、莱菔子2两（炒），木香2两，硇砂半钱，阿魏1分（并硇砂用醋化，去砂石，研）。
出处：《洪氏集验方》引上官驻泊方。
主治：三脘气弱，中焦积寒，脾不磨，饮食迟化，吃物频伤，胸膈满闷，胁肋刺痛，呕吐哕逆，噫醋恶心，腹胀肠鸣，心腹疼痛，噎塞膈气，翻胃吐食，饮食减少，及素来有沉寒积冷，腹中时作疼痛。
加减：若素来无沉寒积冷，去硇砂。
功效：久服大补益脾胃，空膈，令人能食，去寒湿，强中温暖。
用法用量：每服50粒，淡姜汤送下，不拘时候，1日2次。
制备方法：上为末和匀，面糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
