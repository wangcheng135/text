序号：48327
名称：神效手把丸
组成：猢狲头骨半两，虎头骨半两，猫儿头骨半两，砒霜1分（细研），恒山1两（锉），朱砂1分（细研），乳香3分（细研），麝香1分（细研），白芥子1分，蜈蚣1枚，阿魏1分。
出处：《圣惠》卷五十二。
主治：鬼疟。疟疾发作无时节，或一日三两度寒热，或两日一度发动，心神恍惚，喜怒无恒，寒则颤掉不休，热则燥渴不止。或愈而复作，或减而更增，经久不痊，连绵岁月者。
加减：None
功效：None
用法用量：以绯绢裹1丸，男左女右，臂上系之。发时解下，男左女右，以手把之，时时就鼻嗅之。4-5度效。
制备方法：上为末。炼蜜为丸，如皂荚子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
