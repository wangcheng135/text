序号：38480
名称：治疣汤
组成：熟地12g，杜仲6g，赤小豆9g，牛膝9g，丹皮9g，红花9g，白术9g，桃仁9g，赤芍9g，白芍12g，穿山甲3g，何首乌6g。
出处：《中医皮肤病学简编》。
主治：寻常疣，扁平疣，瘢痕疙瘩，瘙痒证，硬皮病，银屑病，症见皮肤干燥，刺瘊或扁平瘊，有厚鳞屑性皮疹，脉滑，舌质紫红，或有瘀斑。
加减：None
功效：养血化瘀，脱瘊止痒。
用法用量：水煎，分两次服，每次用烧酒30毫升作引子。
制备方法：None
临床应用：None
各家论述：熟地、白芍、何首乌养血润燥；丹皮、赤芍、桃仁、红花活血化瘀；牛膝引药下行，杜仲入肾；白术、赤小豆健脾利湿；穿山甲通络化瘀；白酒引药上行于一身之表。故该方可治一切皮肤肌表血燥血瘀之证，不限于治疣。
用药禁忌：None
附注：治瘊汤（《中西医结合皮肤病学》）。
