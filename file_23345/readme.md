序号：23353
名称：防风汤
组成：防风（去叉）半两，防葵（生用）半两，泽泻半两，白术半两，杏仁（汤浸，去皮尖双仁，炒）半两，黄耆半两，麻黄（去根节）半两，独活（去芦头）半两，赤茯苓（去黑皮）半两，大豆（炒熟）3合，赤小豆（炒熟）半升，桑根白皮（锉，炒）4两，陈橘皮（汤浸，去白，炒）3分，泽漆3分，麦门冬（去心，焙）3分，猪苓（去黑皮）3分，大戟（锉，炒）1两。
出处：《圣济总录》卷八十三。
主治：脚气肿满，变为风水。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，加生姜半分（拍破），煎至7分，去滓，空心、日午温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
