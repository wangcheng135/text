序号：48236
名称：神枕
组成：当归半两，芎半两，白芷半两，辛夷半两，杜仲（去粗皮）半两，藁本（去苗土）半两，肉苁蓉半两，柏实半两，薏苡仁半两，靡芜半两，秦椒（去目及合口者）半两，木兰皮半两，蜀椒（去目及合口者）半两，桂（去粗皮）半两，干姜半两，飞廉半两，防风（去叉）半两，款冬花半两，人参半两，桔梗半两，白薇半两，荆实半两，山蓟半两，白鲜皮半两，乌头（去皮）半两，附子（去皮尖）半两，藜芦（去芦头）半两，皂荚半两，莽草半两，半夏半两，矾石半两，细辛（去苗叶）半两。
出处：《圣济总录》卷一九八。
主治：None
加减：None
功效：令筋骨强壮，身面光泽，血气充实，百疾皆愈，齿发益壮，容色还童。
用法用量：每用冬至为首，枕及100日，筋骨强壮，身面光泽，即去一重纱；200日血气充实，百疾皆愈，又去一重；300日又去一重。3年后齿发益壮，容色还童。
制备方法：上（口父）咀，纳枕中（作枕应于5月5日、7月7日采山阳柏木，长1尺2寸、高4寸、广3寸5分、容1斗2升，选柏心赤者为盖，悉厚4分，钻黍粟大孔三行，每行40孔），后8味毒者安下，前24味香者安上，既满即用竹钉钉盖，四边悉用蜡封，惟上不用封，乃以绛纱三重裹之，其药一年一易。
