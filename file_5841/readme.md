序号：5841
名称：归掌地黄丸
组成：归掌2两（即归身），生地2两，熟地2两，天冬2两，麦冬2两，枸杞子2两，黑大豆2两（酒煨），何首乌2两，山药1两，茯神1两，黄耆1两（炙），白术1两，石决明5钱（童便浸，煅），草决明5钱，蜜蒙花5钱，谷精草5钱（只取花用），木贼5钱5钱（去节），甘菊（去蒂），丹皮5钱，川芎5钱。
出处：《卫生鸿宝》卷二。
主治：内障昏花，瞳神散大，或缩小不明；青盲黑暗，虚翳遮睛；及血少阴虚而微热者。
加减：热甚，加知母、黄柏各1两，肉桂2钱；虚寒，加菟丝子（酒浸，炒）、补骨脂各1两。
功效：None
用法用量：每服3钱，淡盐汤送下。
制备方法：上为细末，羊肝2个，不落水蒸熟，捣烂为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
