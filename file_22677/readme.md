序号：22685
名称：含化朱砂丹
组成：朱砂3两，马牙消3两，消石2两。
出处：《圣惠》卷九十五。
主治：None
加减：None
功效：祛热毒风，镇心辟惊，返老驻颜，充肌肤，延年益气。
用法用量：任服；其朱砂即在中心，取出细研，以小瓷盒子中盛，固济，微火养1日，加炭1斤，煅令通赤，放冷，开取细研，以枣肉为丸，每两砂得360丸。每日早晨含化1丸。
制备方法：上为末，入瓷瓶中，以重抄细纸3重，密固瓶口，重汤煮之，常如鱼眼沸，水耗，即以热水添之，不歇火，21日夜满，开瓶子，其消并在瓶四面，收之细研。
临床应用：None
各家论述：None
用药禁忌：忌羊血、咸水。
附注：含化丹砂方（原书卷九十八）。本方方名，《普济方》引作“含化丹砂丸”。
