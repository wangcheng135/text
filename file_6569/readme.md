序号：6569
名称：二术四神丸
组成：苍术（泔浸）1两，白术（土炒）2两，补骨脂（盐水拌炒）1两5钱，吴茱萸（汤泡去沫）5钱，厚朴（姜汁制）1两，广皮1两，神曲2两。
出处：《陈素庵妇科补解》卷五。
主治：产后血气虚损，饮食不能运化，或受风冷以致泄泻，甚则腹痛。
加减：None
功效：温肾健脾，兼消食。
用法用量：每服3钱，空心砂仁汤送下，1日2次。
制备方法：姜汁调前药，面糊为丸。
临床应用：None
各家论述：脾气虚弱，不能运化水谷，命门火衰，不能蒸腐水谷，加以风冷外袭，故泄泻也。补火以生土，扶土以制水，则泻自止。方中骨脂辛温补右尺；二术补脾，土恶湿而喜燥，故用术以祛湿止泄；茱萸温经散寒；陈、朴、神曲消食运气。
用药禁忌：None
附注：None
