序号：54805
名称：桂心丸
组成：桂心半两，没药半两，槟榔半两，干漆3分（捣碎，炒令黄燥烟出），当归半两（锉，微炒），赤芍药半两，川大黄1两（锉碎，微炒），桃仁1两（汤浸，去皮尖双仁，麸炒微黄），鳖甲1两（涂醋，炙令黄，去裙襕），延胡索1两，厚朴1两（去粗皮，涂生姜汁，炙令香熟），京三棱1两（微煨，锉），牡丹半两，青橘皮3分（汤浸，去白瓤，焙）。
出处：《圣惠》卷七十九。
主治：产后血气木散，积聚成块，上攻心腹，或时寒热，四肢羸瘦烦痛，不思饮食。
加减：None
功效：None
用法用量：每服30丸，以温酒送下，日3-4次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
