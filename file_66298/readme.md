序号：66314
名称：隆吉散
组成：硼砂12g，麝香3g，乳香6g，雄黄6g，熊胆6g，血竭6g，没药6g，儿茶6g，牛黄10g，山豆根10g，鸭嘴壳10g，山慈姑10g，冰片15g，黄柏12g（猪胆汁炒），花蜘蛛10个。
出处：《古今名方》引《言庚孚家传秘方》。
主治：白喉，咽喉肿痛。
加减：None
功效：凉血解毒，消肿止痛，利咽喉。
用法用量：愈陈愈好，百年不多，吹于咽喉患处。配合口服方，疗效更佳。
制备方法：将乳香、没药、儿茶去净油，黄柏、山慈姑去粗皮，鸭嘴壳、山豆根切片，皆用文火焙枯研末。用蜡线将花蜘蛛缠住，放铜瓢内，再以明矾60g研末，堆放蜘蛛上，用瓷碗盖住，置火上慢慢煅之，溶化成块稍枯，移至地上待冷取用。如无花蜘蛛，采用子壁钱亦可，以上各药，分别精制细末，再则称准每味分量，重新混合，加工研匀过筛，装瓶收贮密闭。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
