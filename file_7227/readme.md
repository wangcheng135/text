序号：7227
名称：四白头
组成：焰硝2两（研细），白矾3两（研细），石膏4两（研细），砒霜1两（研细）。
出处：《普济方》卷三十三引《仁存方》。
主治：虚寒，饮食作痰喘嗽；妇人白带，男子白浊遗精。
加减：None
功效：壮胃，清上实下。
用法用量：每服3粒，用冷水吞下，以干物压之。
制备方法：用一火鼎子，先以火炙，用生姜自然汁涂内外数遍，炙干，先下砒末半两，次以砒末半两，和焰硝末1两按实；又以焰硝末1两，和矾末按实；又以白矾末1两半，和石膏2两按实，却以石膏末2两紧按在上，用圆瓦片盖合口上，围簇炭5斤，发顶火煅烟尽为度，去火候冷，取药刮净，研如细粉，再加好白石脂（煅）1两，研细，和前药滴水为丸，如鸡头子大，候干；再入新锅内，用瓦盖走置砖上，簇炭1斤，一煅通红为度，用钤钤出，倾丹在厚瓷盆内，乘热搅动，候丹冷，出火毒。
临床应用：None
各家论述：None
用药禁忌：忌热物。
附注：None
