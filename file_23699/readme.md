序号：23707
名称：防风汤
组成：防风（去叉）1两，麻黄（去根节）3分，桂（去粗皮）3分，牛膝（去苗，酒浸，切，焙）1两，丹参半两，五加皮半两，杜仲（去粗皮，炙，锉）3分，芎3分，附子（炮裂，去皮脐）1两，细辛（去苗叶）半两，当归（切，焙）1两，芍药1两，羌活（去芦头）1两，续断1两。
出处：《圣济总录》卷三十三。
主治：伤寒后腰痛，或皮肉（疒帬？？）痹，腿膝疼痛，行履艰难，不可俯仰。
加减：None
功效：None
用法用量：每服5钱匕，用水1盏半，加生姜半分（拍碎），同煎至7分，去滓，食前温服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
