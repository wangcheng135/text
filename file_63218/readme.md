序号：63234
名称：清金一贯饮
组成：黄芩大3钱-5钱，黄芩中2钱-3钱，黄芩小8分-1钱半，牛蒡子大2-3钱，牛蒡子中1-2钱，牛蒡子小1钱，桔梗大2钱，桔梗中1钱，桔梗小7分，前胡大2-3钱，前胡中2钱，前胡小1钱，荆芥穗大1钱半，荆芥穗中1钱，荆芥穗小5分，木通大2钱，木通中1钱，木通小5分，青皮大2钱，青皮中1钱，青皮小7分，赤芍大2钱，赤芍中1钱，赤芍小5分，甘草大6分，甘草中4分，甘草小2分。
出处：《痘疹专门》卷下。
主治：麻疹。
加减：有郁症，即加大黄大5钱至1两，中3钱至6钱，小1钱至2钱；见点血热，即加生地大8钱至2两，中5钱至1两，小3钱至5钱；大热火盛，即加石膏大8钱至2两，中6钱至1两，小3钱至5钱；3日出齐，加元参大3钱，中2钱，小1钱；粗大，加大黄、蝉蜕5个至15个；稠密不匀，加大黄、山楂3钱至1两；浑红，加生地、丹皮3钱；艳红，加生地、大黄、丹皮、石膏、元参；干红，加大黄、紫草3钱，生地、桃仁、丹皮3钱；色紫，加大黄、紫草、生地、丹皮、石膏；色艳，加大黄、生地、石膏、紫草；色暗，加大黄、紫草、生地。桃仁；色白，加大黄、紫草、桃仁、归尾12钱；大泻，加石膏、大黄；舌刺，加石膏、大黄、生地、黄连3钱；大热，加石膏、生地、大黄；肢冷，加大黄、丹皮、石膏；谵语，加大黄、黄连、石膏、犀角3-5分；干呕，加大黄、黄连、甘草、滑石；哈舌，加大黄、黄连、石膏、生地；痰迷，加黄连、犀角、川贝母；发喘，加大黄、桑白钱许、石膏、枳实；喉痛，加山豆根、黄连、石膏；口秽，加石膏、大黄；咬牙，加生地、大黄、石膏；喉肿与头肿，加大黄、金银花2-3钱、石膏、山豆根；鼻衄，加大黄、山栀3钱、生地、犀角；小便短赤，加黄连、猪苓3钱、滑石；泄泻，加泽泻1-2线、条芩1-2钱；大肠逼迫，加大黄、槟榔钱许、滑石；大便不通，加大黄、枳实、滑石；腹痛，加大黄、赤芍、青皮；痢，加大黄、条芩、滑石。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
