序号：54144
名称：柴胡汤
组成：当归1钱2分，白芍1钱，柴胡1钱，黄芩1钱，熟地3分，甘草3分，半夏（制）7分，川芎7分，人参5分，麦冬（去心）5分，生姜3片。
出处：《叶氏女科》卷一。
主治：妇女17-18岁，因脾胃虚弱，气血不行，而致经脉不通，或阻隔半年百日，面色青黄，饮食不思，或作寒热，头痛眩晕，腹中结块，烦闷呕吐，或作膨胀。
加减：不睡，加枣仁（炒）；呕吐，加砂仁7分，白术6分（蜜炙），香附7分（制）；咳嗽，加杏仁6分（去皮尖），五味子5分，苏叶、桔梗各7分。
功效：None
用法用量：水煎，空心热服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
