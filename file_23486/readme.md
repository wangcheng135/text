序号：23494
名称：防风麻黄汤
组成：防风3两（去叉），麻黄3两（去根节，汤煮，掠去沫），独活3两（吉芦头），秦艽（去苗土）3两，当归2两（切，焙），防己2两，甘草2两（炙，锉），人参2两，黄芩2两（去黑心），升麻2两，芍药2两，远志1两（去心），石膏（碎）1两，麝香（研）半两，半夏（汤洗去滑，生姜汁制）2两。
出处：《圣济总录》卷八十一。
主治：脚气痹弱；或风毒脚气，攻身体发热，肢节不遂，精神不遂，语言謇涩。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，加生姜3片，煎至8分，去滓，连3服。衣覆取微汗。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：防风汤（原书卷八十三）。
