序号：62094
名称：大黄煎丸
组成：大黄9两（锦纹新实者，若微朽即不堪用，削去苍皮乃称）。
出处：《外台》卷十三引《崔氏方》。
主治：小儿无辜闪癖，或头干瘰疬、头发黄耸分去，或乍痢乍瘥。
加减：None
功效：下脓及宿结。
用法用量：儿年3岁，每服7丸，1日2次。当以下青赤脓为度；若不下脓，或下脓少者，稍稍加丸；下脓若多，丸又须减；病重者，或至7-8剂方尽根本；大人小儿不等，以意量之。
制备方法：上为散，以上好米醋3升和之，置铜碗内，于大铛中浮汤上，炭火煮之，火不用猛，又以竹木篦搅药，候堪丸，乃丸如梧桐子大，于小瓷器中密贮。
临床应用：None
各家论述：None
用药禁忌：禁牛、马、驴、鸡、猪、鱼、兔肉、生冷、粘滑、油腻、小豆、荞麦。乳母亦同此忌。
附注：大黄丸（《医方类聚》卷二五三引《神巧万全方》）。
