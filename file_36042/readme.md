序号：36052
名称：参苓琥珀汤
组成：人参5分，茯苓（去皮）4分，川楝子（去核，锉炒）1钱，琥珀3分，生甘草1钱，玄胡索7分，泽泻3分，柴胡3分，当归梢3分。
出处：《卫生宝鉴》卷十七。
主治：小便淋，茎中痛不可忍，相引胁下痛。
加减：None
功效：None
用法用量：上(口父)咀，都作1服。用长流水3盏，煎至1盏，去滓，空心、食前温服。
制备方法：None
临床应用：None
各家论述：《医方考》经曰：壮者气行则愈，怯者着而成病。是以房劳老弱之人，多有此疾。补可以去弱，故用人参、茯苓；滑可以去着，故用琥珀、归梢；泻可以会闭，故用泽泻、生甘草；用柴胡者，使之升其陷下之清阳；用玄胡、川楝者，使之平其敦阜之浊气；煎以长流水者，取其就下之意也。
用药禁忌：None
附注：参茯琥珀汤（《丹溪心法》卷三）。
