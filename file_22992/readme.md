序号：23000
名称：达营汤
组成：三棱60g，莪术60g，赤芍30g，生大黄30g。
出处：《中成药研究》（1978；3：30）。
主治：周期性精神病。
加减：None
功效：None
用法用量：上药水煎3次，压滓取汁，制成浓煎，每剂160ml，分2次服用，每日1剂。同服达营丸，直至下月发病日期后1星期，约服30～40剂，病情可控制，则停服汤剂，单服达营丸，每次15g，1日2次，连服3～4个月，疗效明显。
制备方法：None
临床应用：周期性精神病：44例周期性精神病（情感性精神病及精神分裂症）患者，经服用达营汤、丸，临床结果表明：有效40例，无效4例，治愈率为90.9%。随访中有8例复发，经首次治疗而缓解后停药1～3年间，再次用药症状缓解，长者达10年，未复发者32例。
各家论述：None
用药禁忌：孕妇及月经过多、易出血者忌用。
附注：本方改为丸剂，名“达营丸”（见原书）。
