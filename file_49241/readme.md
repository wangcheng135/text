序号：49256
名称：胡芦巴丸
组成：胡芦巴1两，茴香子1两，木香半两，桂心半两，当归半两（锉，微炒），附子1两（炮裂，去皮脐），阿魏半两（研入），硫黄1两（细研），青橘皮半两（汤浸，去白瓤，焙），沉香半两，白豆蔻半两（去壳），桃仁1两（汤浸，去皮尖双仁，别研如膏）。
出处：《圣惠》卷七。
主治：肾脏气虚，下焦积冷，气攻腹胁胀满，脐下疼痛，面色青黑，足胫多冷。
加减：None
功效：None
用法用量：每服20丸，以温酒送下，不拘时候。
制备方法：上为细末，入研了药令匀，好酒1升半，先熬桃仁膏令稠，拌和诸药末，捣200-300杵，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
