序号：31
名称：五味子汤
组成：五味子1两半，蒺藜子（炒，去角）1两半，麻黄（去根节，炒，掠去沫，焙）1两半，桑根白皮（锉）1两半，白石脂1两半，杏仁（去皮尖双仁，炒）1两半，百合1两半，贝母（煨，去心）1两，款冬花1两，枳壳（去瓤，麸炒）1两，紫菀（去苗土）1两，柴胡（去苗）1两，旋覆花半两，桂（去粗皮）半两。
出处：《圣济总录》卷六十五。
主治：咳嗽昼减夜加，不得眠睡，食即吐逆。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，加生姜5片，粳米50粒，煎取8分，去滓温服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
