序号：83609
名称：乌香正气散
组成：大香附子10两（去毛，刮净，熏醋过），好乌药（去心，炒黄）。
出处：《普济方》卷二五六。
主治：杂病。
加减：None
功效：None
用法用量：行当侵晨，冲冒风冷出入，盐点2钱匕，正气祛邪，辟鬼魅疫疠，祛风理气进食；兼治妊孕伤寒，葱白10茎，生姜2两，同煎1碗，作3服，调药热服出汗；治伤风冒冷，头眩项强，背皆痛，用热酒1盏，入苏叶调服；治产后败血攻心脾疼痛，煎童子小便调下；治妇人血海冷，面黄，发落稀少，米饮调下；妇人发落，血衰经脉不调，无颜色，醋汤调下；妇人血劳、血瘕、血癥，血气攻注疼痛，当归、乳香酒调下；妇人经脉过多，血崩不止，烧菴(艹闾)灰1盏，酒、醋同调下；妇人血气冲心，血气不通，血脉湛浊不匀，芫花酒调下；妇人难产，取酸草子吞3-7粒，以童便调药吞下，无草子，叶或根。男子妇人血风血热，遍身红痒，渐成癞疾，用荆芥酒调下；男子、小儿腹痛，脏毒泻血，用柏叶焙干，碾罗1钱末，同药3钱，米饮调下；男子、妇人痃风小腹急，男子小肠气，膀胱肾气，冷气攻冲，背膂绞疼痛，并炒盐、茴香、五灵脂，温酒调下；治蛊毒疰忤，鬼气神昏，用人参煎服；大人、小儿宿食不消，意气不顺，逆噎不通，一切气病，入生姜、大枣，同煎调下，或盐点服；治痈疖疥癞疮癣，荆芥茶或酒调下；治大人腹中有虫，小儿疳气诸虫，腹胀肚大，面黄发疏，服精肉瘦肚，并用槟榔磨汤调下，仍空心服；治大人、小儿冷热不解，泻痢交作，血气不和，用乌梅、干姜、甘草汤调下；治大人小儿积热不解，酸浆草研自然汁1合，井水同调下。
制备方法：上为细末。
