序号：53977
名称：晞露丸
组成：广荗1两（锉），京三棱1两（锉，并酒浸），干漆5钱（洗去腥，炒烟尽），川乌5钱，硇砂4钱，青皮3钱，雄黄（另研）3钱，茴香（盐炒）3钱，穿山甲（炮）3钱，轻粉1钱（另研），麝香半钱（另研），巴豆30个（去皮，切开）。
出处：《卫生宝鉴》卷十八。
主治：寒伤于内，气凝不流，结于肠外，久为癥瘕，时作疼痛，腰不得伸。肠覃，坚久作痛。
加减：None
功效：None
用法用量：每服20-30丸，空心、食前，姜汤送下；酒亦得。
制备方法：上除研药外，将巴豆炒三棱、广荗2味深黄色，去巴豆不用，共为末，入研药匀，生姜汁打面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
