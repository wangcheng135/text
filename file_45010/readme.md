序号：45025
名称：枯痔药
组成：白砒2钱（为末），白矾5钱（为末）。
出处：《外科百效》卷三。
主治：痔疮。
加减：None
功效：None
用法用量：None
制备方法：2味入铁杓内，先入白矾末1钱，又放砒1钱于正中，又放白矾1钱，砒1钱，白矾盖面，将慢火熬至烟净，出火毒，入后药：儿茶1钱，血蝎5分，蛇含石1钱（火煅，醋淬），芫花2分，朱砂5分，雄黄、硼砂各1钱，乳香、没药各5分（焙，去油），轻粉5分、炉甘石1钱（醋淬）。上为极细末，合五分之一，用液津调开，捏作薄饼子，阴干，听用。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《外科十三方考》引作“枯痔散”。
