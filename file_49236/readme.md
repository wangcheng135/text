序号：49251
名称：胡芦巴丸
组成：胡芦巴1两，补骨脂（炒）1两，肉苁蓉（酒浸，微炒）1两，巴戟天（去心）1两，附子（炮裂，去皮脐）1两，白豆蔻（去皮）1两，荜茇1两，茴香子（炒）1两，丁香1两，木香1两，硫黄（别研）1两，沉香（锉）1两，蓬莪术（煨）1两，桂（去粗皮）1两，当归（切，炒）1两，桃仁（去皮尖双仁，麸炒，别研）1两，阿魏（面和作饼子，炙黄）1两，肉豆蔻（去壳）6个，槟榔（锉）6个。
出处：《圣济总录》卷八十六。
主治：脾劳，大便不调，呕逆腹胀，羸瘦少力，饮食无味，面色萎黄。
加减：None
功效：None
用法用量：每服15丸，渐加至20丸，空心、食前盐汤送下；小肠气，炒生姜酒送下；妇人心痛，醋汤送下。
制备方法：上为末，用清米醋煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
