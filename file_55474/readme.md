序号：55489
名称：丁香丸
组成：丁香1两，木香1两，猪牙皂角（去皮，炙焦黑，为细末）1两，肉桂（去粗皮）1两，干姜（炮）1两，好墨（烧，醋淬）1两，青皮（去白）3两，附子（炮，去皮脐）2两，京三棱（炮，捣碎）2两，蓬莪术（炮，捣碎）2两，黑牵牛（炮）2两，川大黄（别为细末）2两，干漆（碎，炒令烟尽，为细末）2两，巴豆霜1钱半（先用醋煎硇砂令热，下巴豆霜，煎3两沸，下大黄末熬膏），硇砂别研2两。
出处：《局方》卷三。
主治：积滞不消，心腹坚胀，痰逆呕哕，噫酸吞酸，胁肋刺痛，胸膈痞闷；反胃恶心，食饮不下，气上冲胸，痞噎不通；食癥酒癖，血瘕气块，时发刺痛，全不思食。
加减：None
功效：消饮食，行滞气。
用法用量：每服1-2丸，茶、酒任下。如要化癥瘕癖块，用生姜汤送下7丸，并食后临卧服之。
制备方法：上以大黄、硇砂、巴豆膏为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
