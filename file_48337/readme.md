序号：48352
名称：神效仙方万亿丸
组成：朱砂5钱，巴豆（不去油）5钱。
出处：《古今医鉴》卷十六引张三峰方。
主治：小儿诸病。外感风寒发热，内伤生冷饮食，心痛，肠痛，霍乱吐泻，赤白痢，疟疾寒热，心膨气胀，伏暑伤热，诸虫作痛，大便闭结，小便不通，积聚发热，咳嗽喘急，小儿急慢惊风。
加减：None
功效：None
用法用量：每服3-5丸，外感风寒发热，姜、葱汤送下，出汗；内伤生冷饮食，茶清送下；心痛，艾醋汤送下；肠痛，淡姜汤送下；霍乱吐泻，姜汤送下；赤痢，茶清送下；白痢，淡姜汤送下；赤白痢疾，姜茶汤送下；疟疾寒热，姜汤送下；心膨气胀，姜汤送下；伏暑伤热，冷水送下；诸虫作痛，苦楝根汤送下；大便闭结，茶送下；小便不通，灯心汤送下；积聚发热，茶送下；咳嗽喘急，姜汤送下；小儿急慢惊风，薄荷汤送下。
制备方法：酒煎5钱寒食面，丸如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：神仙万亿丸（《古今医鉴》卷十六）、万亿丸（《回春》卷七）。
