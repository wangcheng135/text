序号：46744
名称：牵牛子丸
组成：牵牛子（生，捣罗为末，以生姜汁1升，慢火熬如饧）2两，硇砂（汤中慢火熬，取霜）1两，槟榔（煨，锉）3分，木香3分，桃仁（汤退，去皮尖双仁，炒黄，别研）1两半，附子（炮裂，去皮脐）3分，干姜（炮）3分，人参3分，丁香3分。
出处：《圣济总录》卷七十三。
主治：痃气，口吐酸水，醋心，常似有物在胸膈间。
加减：None
功效：None
用法用量：每服10丸，渐加至20丸，空心、食后煎生姜汤送下。以利下积滞物为度。
制备方法：上药除煎研外，为末，入硇砂、桃仁研令匀，同入牵牛子煎中和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
