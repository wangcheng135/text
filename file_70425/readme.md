序号：70441
名称：紫河车丹
组成：男子胞衣（取首生，以皂角水洗净，次放铜铫子内，以米醋揲洗控干，做一小篾笼子盛之，围以纸，密糊之，不令泄气，以烈火焙干）、人参1两5钱，白术1两，木香5钱，白茯苓5钱，茯神1两，川归1两，熟地黄1两，乳香4钱（另研），没药4钱（另研），朱砂2钱（另研），麝香2分。
出处：《医学正传》卷三引《青囊方》。
主治：飞尸鬼注，虚劳羸瘦，喘嗽痰气。
加减：None
功效：None
用法用量：每服50丸，煎人参汤送下，空腹服之，1日3次。
制备方法：上为细末，酒糊为丸，如梧桐子大，炼蜜为丸服亦可。
临床应用：None
各家论述：None
用药禁忌：None
附注：混元丹（《东医宝鉴·杂病篇》卷四）。
