序号：73283
名称：椒附丸
组成：蜀椒（去目及闭口者，炒出汗）1两，桔梗（锉，炒）半两，芍药半两，干姜（炮）半两，厚朴（去粗皮，生姜汁炙）半两，细辛（去苗叶）半两，附子（炮裂，去皮脐）半两，乌头（炮裂，去皮脐）1分。
出处：《圣济总录》卷九十四。
主治：七疝。或心腹厥逆，不得气息，痛达背膂；或心下坚痛，手不可近；或脐下坚痛，得寒冷食辄剧；或胁下坚痛大如手；或少腹胀满，引膀胱急痛；或女子月事不时。
加减：None
功效：None
用法用量：每服20丸，米饮或温酒送下，1日3次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
