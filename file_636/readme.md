序号：636
名称：五膈散
组成：人参1两，赤茯苓（去黑皮）1两，厚朴（去粗皮、姜汁炙）1两，枳壳（去瓤，麸炒）1两，桂（去粗皮）1两，甘草（炙，锉）1两，陈曲（炒）1两，诃梨勒皮1两，白术1两，陈橘皮（汤浸去白，焙）1两，干姜（炮）1两，京三棱（煨，锉）1两，槟榔（锉）1分，木香1分。
出处：《圣济总录》卷六十二。
主治：五膈气痞，心胸噎塞，渐至羸瘦。
加减：None
功效：None
用法用量：每服2钱匕，入盐点服。如脾气腹胀，心胸满闷，每服3钱，用水1盏，加生姜1块(切)，大枣2枚(擘破)，盐少许，同煎至8分，不拘时候，和滓热服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
