序号：51779
名称：钟乳丸
组成：炼成钟乳粉1两，续断1两，熟干地黄（焙）1两，石韦（去毛）1两，杜仲（去粗皮，锉，炒）3两3分，天雄（炮裂，去皮脐）半两，山茱萸1两，蛇床子1两，远志（去心）1两3分，肉苁蓉（酒浸，切，焙）1两3分，防风（去叉）1两3分，山芋1两3分，石斛（去根）1两3分，赤石脂1两3分，甘草（炙，锉）1两，牛膝（酒浸，切，焙）1两。
出处：《圣济总录》卷五十八。
主治：消渴后虚乏。
加减：None
功效：None
用法用量：每服30丸，温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中远志用量原缺，据《普济方》补。
