序号：31423
名称：沉香散
组成：沉香（锉）1分，乳香（炒软，候冷同杵）1分，安息香1分，丁香1分，没药（研）1分，青橘皮（去白，焙）1分，牛膝（去苗，酒浸，焙）半两，当归（切，焙）半两，威灵仙（去土）半两，羌活（去芦头）1两，莱菔子（微炒，别研如膏）1两。
出处：《圣济总录》卷八十一。
主治：脚气疼痛不仁，浸淫昏冒，乍发乍歇，食少无力，日渐羸瘦。
加减：None
功效：None
用法用量：每服1钱匕至1钱半，空心、午时、夜卧各以温酒调下；生姜汤调亦得。若要通气，即用大蒜1枚、生姜半两、绿豆2合，以水4升，一处煮豆烂，取汁1升，多用2合，调下1钱半匕，其绿豆亦须食尽。
制备方法：先将前十味为散，次入莱菔子膏拌，再罗令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
