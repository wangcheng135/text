序号：57260
名称：消暑十全散
组成：香薷2钱，扁豆（炒，捶）1钱，厚朴（姜制）1钱，陈皮1钱，甘草（炙）1钱，白术1钱，茯苓1钱，木瓜1钱，藿香1钱，苏叶1钱。
出处：《张氏医通》卷十三。
主治：伤暑，兼感风邪，发热头痛。
加减：None
功效：None
用法用量：水煎，热服，不拘时候。取微汗效。
制备方法：None
临床应用：None
各家论述：《医略六书》：暑伤心脾，风侵经络，故头痛恶心，发热口干焉。香薷散夏月之表，厚朴疏胸腹之里，扁豆助脾却暑，白术燥湿健脾，藿香快胃以祛暑，木瓜舒筋以和脾，陈皮理中气，茯苓清治节，苏叶散风邪，甘草和胃气也。此调中解表之剂，为风暑合邪之专方。
用药禁忌：None
附注：None
