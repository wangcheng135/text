序号：24460
名称：防风丸
组成：防风（去叉）1两，羌活（去芦头）1两，茯神（去木）1两，牛膝（酒浸，切，焙）1两，桂（去粗皮）1两，人参1两，枳壳（去瓤，麸炒）1两，五加皮（锉）1两，芍药1两，丹参1两，薏苡仁1两，玄参1两，麦门冬（去心，焙）1两，生干地黄（焙）1两，磁石（煅，醋淬）4两，槟榔（锉）2两，松子仁半两，大黄（锉，炒）半两，木香半两。
出处：《圣济总录》卷二十。
主治：热痹。
加减：None
功效：None
用法用量：每服30丸，加至40丸，空心、食前温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
