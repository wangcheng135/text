序号：26032
名称：妙灵丹
组成：麝香5钱，蟾酥5钱，雄黄5钱，母丁香5钱，朱砂5钱，真茅术1两（米泔浸透，剖去皮净，研末）。
出处：《古方汇精》卷一。
主治：各种急痧，胃疼，男妇阴症，伤寒时气，肚疼，喉痹，喉风，小儿急慢惊风。
加减：None
功效：None
用法用量：治各种急痧，用7丸，轻用5丸，生姜汤送下；治胃疼，用4-5丸；治男妇阴症，用21丸；治伤寒时气，用7丸；治肚疼，用7丸，以上俱生姜汤送下；喉痹，用5丸，未愈，再5丸；喉风，用5丸，未愈，再5丸，以上俱薄荷汤送下；小儿急慢惊风，1岁1丸，淡姜汤送下。
制备方法：上方宜于午月午日修制，各药取净细末，用真麦烧酒，将蟾酥泡透，搅粘，入群药和丸，如芥子大，阴干，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
