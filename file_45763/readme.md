序号：45778
名称：柳枝煎防风丸
组成：倒垂柳枝2斤，桑枝2斤，槐技2斤，天寥木枝2斤，仙灵脾叶2斤。
出处：《圣惠》卷二十四。
主治：大风疾，体生疮肿，瘙痒出脓，风毒极甚者。
加减：None
功效：None
用法用量：每服30丸，以温酒1中盏，入煎成药1茶匙调匀，食前送下。
制备方法：上锉，以水7斗，于大银锅中，煎取1斗，尽滤去滓；用晚蚕沙1升，炒令香，捣罗为末，入药汁中相和，再煎稀稠得所，取出，用瓮合盛，每用酒调下后丸药：防风1两（去芦头），羌活1两，五味子1两，人参1两（去芦头），五加皮1两，白蒺藜1两（微炒，去刺），赤茯苓1两，白鲜皮1两，甘菊花1两，松子1两（去壳），乌蛇3两（酒浸，去皮骨，炒令黄），露蜂房1两（微炙）。上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
