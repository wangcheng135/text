序号：7665
名称：二色丸
组成：天南星3钱，半夏3钱，甘遂3钱，大戟3钱，干姜2钱，胡椒2钱，桂2钱，荜茇2钱，代赭石1两，大黄（生用）3钱。
出处：《圣济总录》卷一二五。
主治：一切肿赤，皮肤毒气及瘤子；小肠气；妇人心气痛。
加减：None
功效：None
用法用量：此两等颜色（白者，或捏作饼子亦可），用治瘤子，每服1丸，嚼生姜酒送下，早、晚各1丸；如小肠气，3丸，切生姜3钱，炒焦，酒送下；如伤酒，2丸，飞白矾、生姜自然汁调下；妇人心气痛，醋汤送下2丸。
制备方法：上10味，取前4味，以浆水1升煮，水尽为度，晒干，余6味同捣为末。每用1钱，用巴豆3枚，烧得焰起，盏合，却候冷，与1钱药1处研，更用醋1盏，煎成膏，共药末同为丸，如绿豆大。分两处，1用丹砂为衣，1用腻粉为白衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中大黄用量原缺，据《普济方》补。
