序号：47191
名称：珍珠散
组成：青缸花5分（如无，用头刀靛花轻虚色翠者代之，终不及缸花为妙），珍珠1钱（不论大小，以新白为上，入豆腐内煮数滚，研至极细无声方用），真轻粉1两。
出处：《外科正宗》卷三。
主治：下疳皮损腐烂，痛极难忍及诸疮新肉已满，不能生皮；又汤泼火烧，皮损肉烂，疼痛不止者。
加减：None
功效：None
用法用量：凡下疳初起，皮损搽之即愈；腐烂疼痛者，甘草汤洗净，猪脊髓调搽；如诸疮不生皮者，用此干掺，即可生皮；又妇人阴蚀疮，或新嫁内伤痛甚者，亦可搽此极效，汤泼火烧痛甚者，用玉红膏调搽之。
制备方法：共研千转，细如飞面，方入罐收。
临床应用：None
各家论述：None
用药禁忌：None
附注：月白珍珠散（《外科大成》卷一）。
