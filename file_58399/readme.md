序号：58414
名称：破饮丸
组成：荜茇、丁香、胡椒、缩砂仁、乌梅肉、青皮、巴豆（去皮）、木香、蝎梢各等分。
出处：《三因》卷十三。
主治：五饮停蓄胸腹，结为癥癖，支满胸膈，傍攻两胁，抢心疼痛，饮食不下，反胃吐逆，九种心疼，积年宿食不消，久疟久痢，遁尸疰忤，癫痫厥晕，心气不足，忧愁思虑，妇人腹中诸病。
加减：None
功效：None
用法用量：每服5-7丸，临睡生姜汤送下；津液下尤佳。久服不伤脏气。
制备方法：将青皮、巴豆以浆水同浸1宿，次日滤出，同炒青皮焦，去巴豆，将所浸水淹乌梅肉，蒸1炊久，细研为膏，入药末和匀为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：破痰丸（《医统》卷四十三引《医林方》）。
