序号：66204
名称：野葛膏
组成：葛1两，黄连（去须）1两，细辛（去苗叶）1两，杏仁（去皮尖）1两，莽草1两，芍药1两，藜芦（去芦头）1两，附子（去皮脐）1两，乱发灰1两，（艹闾）茹1两，芎1两，白芷1两，桂（去粗皮）1两，藁本（去苗土）1两，乌头（去皮脐）1两，白术1两，吴茱萸（洗，焙干，炒）1两，雌黄（研）1两，矾石（研）1两，天雄（去皮脐）1两，当归1两，斑猫（去翅足）半两，巴豆（去皮）半两，蜀椒（去目及合口）半两，黄柏（去粗皮）半两，蛇床子半两，猪脂3斤半。
出处：《圣济总录》卷一三四。
主治：久瘑疮。
加减：None
功效：None
用法用量：取摩涂疮上，每日3-5度。
制备方法：上除雌黄、矾石、猪脂外，锉碎；先熬脂令沸，下诸药，煎候白蓝黄黑色流出，以绵布绞滤过，即下雌黄、矾石末，以柳蓖搅令匀，以瓷盒盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
