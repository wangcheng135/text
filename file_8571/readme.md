序号：8571
名称：玄菟固本丸
组成：生地黄（酒浸）4两，熟地黄（酒浸，蒸，俱不犯铁器）4两，天门冬（去心）4两，麦门冬（去心）4两，五味子（去皮）4两，茯神（去皮木）4两，干山药3两（白者，微炒），莲肉2两，人参（去芦）2两，枸杞子2两（甘州者佳），菟丝子1斤（酒煮数沸，捣烂，捻作饼子，晒干，净称8两）。
出处：《丹溪心法附余》卷十九。
主治：虚劳下元衰弱。
加减：None
功效：补肾,滋阴助阳。
用法用量：每服50丸，渐加至80-90丸，空心滚白汤送下，或淡盐汤、温酒送下亦可。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：固本丸（《竹林女科》卷四）。
