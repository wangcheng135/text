序号：81601
名称：镇心汤
组成：防风5分，当归5分，大黄5分，泽泻4分，白蔹4分（一方3两），菖蒲3分，人参3分，桔梗3分，白术10分，甘草10分，紫菀2分（一方3两），茯苓2分（一方3两），秦艽6分，桂心3分，远志3分，薯蓣3分，石膏3分，大豆卷4分，麦门冬5分（一方5两），粳米5合，大枣15枚，干姜2分，附子2两，茯神2两。
出处：《千金》卷十四。
主治：风虚劳冷，心气不足，喜忘恐怖，神志不定。
加减：None
功效：None
用法用量：上(口父)咀。以水1斗2升，先煮粳米令熟，去滓，纳药煮取4升，每服8合，日3夜1。

临床应用：None

用药禁忌：None
