名称：拔毒锭
组成：白及1两，白蔹1两，南星2两，牙皂1两5钱，花粉1两5钱，射干1两，白芷2两，全蝎3两，雄黄5钱，山甲2两5钱（炙），蟾酥1两，血竭2两，冰片5分，麝香3分，细辛1两，生军2两，木通1两，川连2两，山栀2两（炒），二宝花2两，防风1两，泽泻1两，草梢5分，白梅花3两，乳香2两，没药2两，江米4两（另研打糊）。
出处：《慈禧光绪医方选义》引《良方集成》。
主治：一切痈疽肿毒之火郁实证。
加减：None
功效：攻毒化脓止痛，清热解毒，提脓生肌。
用法用量：None
制备方法：上为细末，用木瓜酒粘为锭。
临床应用：None
各家论述：None
用药禁忌：若正气已虚，用之当慎。
