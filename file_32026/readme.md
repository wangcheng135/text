序号：32035
名称：万安丸
组成：肉苁蓉4两（酒浸），干薯蓣2两半，五味子2两半，杜仲3两（炒），牛膝2两（酒浸），菟丝子2两（酒浸），泽泻2两，白茯苓2两（酒浸），熟干地黄2两，当归2两，山茱萸2两（去核），巴戟3两（去心），赤茯苓（去皮）。
出处：《医方类聚》卷一五三引《瑞竹堂方》。
主治：下元极虚。
加减：None
功效：补下元，起阴阳，安魂定魄，和三焦，破积聚，消五谷，安脏腑，除心中伏热，强骨轻身，明目，去冷除风。
用法用量：每服50-70丸，空心温酒送下。
制备方法：上为细末，用苁蓉末半斤，酒熬膏和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌醋、陈腐、自死之物；忌猪羊肉、血7日。
附注：方中赤茯苓用量原缺。
