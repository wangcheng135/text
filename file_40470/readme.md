序号：40480
名称：虎骨木瓜酒
组成：虎骨（酥炙）1两，木瓜3两，川芎1两，川牛膝1两，当归1两，天麻1两，五加皮1两，红花1两，川续断1两，白茄根1两，玉竹2两，秦艽5钱，防风5钱，桑枝4两（一方有羌活、独活、千年健、山栀、陈皮；无天麻、防风、续断、白茄根）。
出处：《丸散膏丹集成》。
主治：风寒湿气流入经络，筋脉拘挛，骨节痠痛，四肢麻木，口眼歪斜，山岚瘴气，历节风痛。骨折伤筋后，筋络挛缩痠痛，痿软无力。
加减：None
功效：追风定痛，除湿驱寒，壮筋强骨，调和气血。
用法用量：任量饮之。
制备方法：上为细末，绢袋盛之，浸于高粱酒20斤内7日，滤清，加冰糖2斤。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
