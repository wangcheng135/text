序号：73504
名称：蛤蚧丸
组成：蛤蚧1枚（涂酥，炙微黄），人参半两（去芦头），白前1两半，杏仁1两（汤浸，去皮尖双仁，麸炒微黄），猪牙皂半两（去黑皮，涂酥，炙微焦，去子），汉防己1两半，紫菀1两（洗去苗土），甘草3分（炙微赤，锉），羚羊角屑3分，槟榔2两，贝母1两（煨，微黄），甜葶苈2两（隔纸炒令紫色），郁李仁2两（汤浸，去皮尖，微炒）。
出处：《圣惠》卷三十一。
主治：骨蒸劳，咳嗽，涎唾稠粘。
加减：None
功效：None
用法用量：每服20丸，以桃仁汤送下，不拘时候。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
