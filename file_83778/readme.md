序号：83796
名称：乌蛇丸
组成：乌蛇3两（酒浸，去皮骨，炙令微黄），犀角屑3分，连翘1两，玄参3分，昆布3分（洗去咸味），牛蒡子1两（微炒），川大黄1两（锉碎，微炒），黄耆1两（锉），漏芦1两，甘草3分（炙微赤，锉），枳壳3分（麸炒微黄，去瓤），大麻仁3分，郁李仁3分（汤浸，去皮，微炒），斑蝥1分（以糯米拌，炒米微黄，去翅足）。
出处：《圣惠》卷六十六。
主治：瘰疬成结核，肿痛不散，或破为脓水不绝。
加减：None
功效：None
用法用量：每服20丸，以清粥饮送下，不拘时候。
制备方法：上为末。炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
