序号：6136
名称：加味香苏散
组成：香附1两半，苏叶1两半，藿香3两，陈皮半两，甘草6钱，砂仁1两（炒）。
出处：《医略六书》卷二十八。
主治：孕妇感冒，吐泻，脉浮者。
加减：转筋，加木瓜；胎动，加白术；挟热，加黄连、白术；挟寒，加白术、炮姜。
功效：None
用法用量：每服3钱，水煎，去滓温服。
制备方法：上为散。
临床应用：None
各家论述：妊娠先伤于暑，复感于风，风暑合邪，肝胃受病，故吐泻不已，胎因不安焉。香附调气解郁，苏叶理血疏风，藿香快胃祛暑，陈皮利气和中，砂仁醒脾安胎气，甘草缓中和胃气也。为散水煎，使风暑并解，则肝胃调和而吐泻无不止，胎孕无不安矣。
用药禁忌：None
附注：None
