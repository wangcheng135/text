序号：15924
名称：生发膏
组成：丁香2升，甘松香2升，零陵香2升，吴藿香2升，细辛2升，蜀椒2两，白芷1两，泽兰1两，大麻子1两，桑白皮1两，桑寄生1两，牡荆子1两，苜蓿1两，辛夷仁1两，杏仁1两，芎藭1两，防风1两，莽草1两，胡麻油1升，竹叶半升，松叶半升，柏叶半升，腊猪膏1升，乌鸡肪1合，雁肪1合。
出处：《千金》卷十三。
主治：None
加减：None
功效：生发。
用法用量：涂头上，日2夜1。
制备方法：上(口父)咀，以醋渍1宿，纳油膏中微火煎三上三下，白芷色黄膏成，去滓。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》卷四十八引作“甘松膏”。
