序号：74422
名称：暖脐膏
组成：真麻油5斤，生天雄1斤，炮姜8两，广木香4两，香橼皮4两，小茴4两，黄丹30两，没药末2两，肉桂末12两。
出处：《全国中药成药处方集》（武汉方）。
主治：呕吐泄泻，脐腹疼痛。
加减：None
功效：None
用法用量：摊时重加肉桂末5两，母丁香1两，倭硫黄3两，生香附8两，麝香2钱，研粉。每张加药粉2厘，以红布为壳，每张重2钱。贴于腰脐上。
制备方法：取天雄、炮姜，广木香、香橼皮、小茴五味，加麻油5斤，浸7日，入油锅内，熬至药枯黑为度，滤净滓，再熬至滴水成珠；加炒黄丹30两，棍搅至烟尽微冷，再加没药、肉桂末12两，入膏内搅匀成膏，倾钵内收贮，浸冷水中3日，炖化去火毒，听用。
临床应用：None
各家论述：None
用药禁忌：孕妇忌用。
附注：None
