序号：59755
名称：调经丸
组成：当归（酒洗）2两，川芎1两，熟地黄（姜汁炒）1两，青皮（麸炒）1两，陈皮1两，枳壳（去瓤，炒）1两，白术（去芦）1两，厚朴（姜汁炒）1两，小茴香（炒）1两，艾叶（去筋）1两，三棱（煨醋炒）1两，莪术（煨醋炒）1两，砂仁1两，白芷1两，牛膝（去芦，酒洗）1两，玄胡1两，香附（醋炒）5两，粉草5钱，琥珀5钱（另研入）。
出处：《回春》卷六。
主治：经闭。
加减：肚痛，加苍术，去白术。
功效：None
用法用量：每服80-90丸，米汤送下；酒下亦可。
制备方法：上为末，醋打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：《寿世保元》有赤芍，无琥珀。
