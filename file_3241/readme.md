名称：九子还阳丹
组成：熟地16两，白芍6两，黄连4两，甘草8两，泽泻6两，杜仲炭8两，川贝4两，苁蓉（炙）6两，牡蛎（煅）8两，玉竹6两，砂仁4两，五味子（炙）4两，山萸肉（炙）8两，茯苓6两，黄耆（炙）6两，菟丝子6两，知母6两，檀香8两，远志（炙）6两，当归（酒炙）8两，人参（去芦）4两，枣仁（炒）6两，牛膝6两，丹皮6两，龙骨（煅）6两，丹参6两，芡实（炒）6两，枸杞子4两，乳香（炙）8两，山药8两，麦冬4两，木香1两，鳖甲（炙）6两，续断6两，肉桂（去粗皮）4两。
出处：《北京市中药成方选集》。
主治：身体衰弱，梦遗滑精，偏坠疝气，腰痠腿软。
加减：None
功效：补肾固精，散寒止痛。
用法用量：每服2丸，日服2次，温开水送下。
制备方法：上为细末，过罗，炼蜜为丸，重2钱2分，朱砂为衣。
临床应用：None
