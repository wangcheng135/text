序号：41860
名称：降痈活命饮
组成：大当归8钱，生黄耆5钱，金银花5钱，生甘草3钱。
出处：《经验奇方》卷上。
主治：一切有名肿毒，无论阴阳证。
加减：患在上部，加川芎2钱，在中部，加桔梗2钱，在下部，加牛膝2钱；如泄泻，加苍白术各2钱；呕吐恶心，加陈皮、半夏各1钱；不思食饮，加白术3钱、陈皮1钱；气虚，加党参5钱；阴疽肉白色淡，无论冬夏，加陈皮、麻黄各6分，瑶桂心、炮姜各1钱5分，切不可妄行加减。如排脓，加白芷2钱；欲破，加皂刺1钱5分，已破者不用；火气盛，加天花粉、黄芩各2钱；大便闭结，加熟大黄3钱，已通者不用。
功效：初起能益气和血，解毒托里；破后能排脓去腐，生长肌肉。
用法用量：上药用陈绍酒、清水各1碗，煎脓汁热服。服后宜暖睡出汗。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
