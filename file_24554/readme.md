序号：24562
名称：安胎丸
组成：生地黄4两（砂仁末1两拌酒蒸晒9次），当归身（酒炒）3两，白芍（酒炒）3两，白术3两（切片，饭上蒸晒5次，蜜炙），陈皮（去白）2两，条芩（酒炒）2两，川续断（盐水炒）2两，杜仲（盐水炒断丝）2两，麦冬（去心）2两。
出处：《叶氏女科》卷二。
主治：None
加减：脾虚泄泻，加淮山药、菟丝饼各3两；气虚，加人参2两；血虚，加阿胶（蛤粉炒珠）2两。
功效：和中保胎，养血调气，健脾进食。
用法用量：每早砂仁汤送下4钱。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：《大生要旨》：怀孕三月，恶心懒倦已退，脏燥已润，于四、五、六、七、八月逐月服此丸料。
