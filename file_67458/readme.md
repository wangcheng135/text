序号：67474
名称：黄龙丸
组成：红芍药半斤，川乌4两（去皮尖），防风4两，香白芷4两，天麻（去根节）2两，华阴细辛（去苗）2两，白僵蚕（炒，去丝嘴）2两，雄黄（别研）2两，川芎2两，白蒺藜（炒，去刺）、甘草1两，干姜（生用）1两，藿香叶1两，甘松（去土）1两。
出处：《普济方》卷九十三引《卫生家宝》。
主治：左瘫右痪，手足麻木，口眼斜，风痹疰腰脚疼痛；及妇人血风劳气，遍身疼痛，洗头伤风，头面乳肿，舌胀口干，头昏脑闷，多睡，暗风夹脑风，偏正头痛，破伤风。
加减：None
功效：None
用法用量：每服1丸，姜汁磨化，温酒调下。
制备方法：上为末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
