序号：34477
名称：驱邪散
组成：犀角1钱，羚羊角1钱，龙齿1钱，虎头骨（俱为末）1钱，木香1钱，沉香1钱，檀香1钱，降香（净）1钱，麝香2分，雄黄2钱，牛黄1分，朱砂2钱，羶羊肉2-3两，茯神5钱，枣仁5钱，远志5钱。
出处：《顾松园医镜》卷十三。
主治：卒中邪恶，头面青黑，口噤眼闭，昏不知人，手足厥冷，肌肤粟起，或错言妄语，或直视握拳，或遍身骨节疼痛非常。
加减：None
功效：None
用法用量：每服3钱，煎汤调下，1日2次。
制备方法：None
临床应用：None
各家论述：邪崇乘虚附人，与人之神魂相持，亦逼处不安，无隙可出，故用诸灵物（犀角、羚羊角、龙齿、虎头骨）之遗形，引以羊肉之羶，俾邪祟转附骨角，移从大便而出。邪气着人，则关闭窍塞，麝之辛香走窜，引芳香正气辟邪诸品（木香、沉香、檀香、降香），自内达外，则毫发骨节俱开，邪亦从此而出；雄黄、牛黄亦最辟邪之物；朱砂同茯神、枣仁、远志镇心神。
用药禁忌：羊肉，胃弱者不用。
附注：None
