序号：64644
名称：痔疮丸
组成：黄连1两，苦参1两，乳香（去油）1两，没药（去油）1两，雄黄1两，连翘2两，僵蚕2两，蝉蜕2两，防风2两，全蝎2两，槐角（入牛胆汁煮）2两，生地2两，牛膝2两，陈皮2两，穿山甲2两，当归2两，枳壳2两，地龙（去泥，晒干）2两，蜈蚣（焙，去头足）20条，象牙末5钱，人参2钱5分，蜂房1个（入玄明粉于眼内，草纸湿透包好，用微火煨之）。
出处：《疡医大全》卷二十三。
主治：痔疮。
加减：None
功效：None
用法用量：空心开水送下3钱。服过6-7日，再用荔枝草、马鞭草（阴干）各半斤，蒲公英2两，甘草3钱，入罐煎好，再入皂矾1两，略煎数沸，先熏后洗。
制备方法：上药各为末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：忌一切火酒、发物、房事。
附注：None
