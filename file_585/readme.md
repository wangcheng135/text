序号：585
名称：五福丹
组成：朱砂1两（研细），硫黄1两（研细），雄黄1两（研细），硼砂1两半，消石1两半（5味并用精者，拣去砂石，合研匀细）。
出处：《普济方》卷一四二引《保生回车论》。
主治：伤寒后发疟。
加减：None
功效：None
用法用量：每于发前1日临卧时，温粥饮吞下1粒，发作前1时，温粥饮吞下1粒，至日自愈。
制备方法：上用瓷盒子1个，里有油者，用乳香少许，香炉中烧烟起，即熏盒子内，入前药合定，油调蛤粉固口，再加纸筋盐泥固济一指厚，阴干。净室中砖上坐盒子，醋拌灰少许，拥抱盒子上，过口缝可一指厚，以3斤生炭，簇合周匝，次又下熟炭火1斤在盒上，渐令通灼，候火消至8分以上，即拨开残火，俟良久冷透，开盒取药，尽作汁有如镜面，捶碎，研极细，粳米饭和丸，如豌豆大及一等绿豆大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
