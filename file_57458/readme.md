序号：57473
名称：润生丸
组成：人参（不拘多少）4两，川乌4两，草乌4两，白芷4两，当归4两，槐角子4两，何首乌4两，枳壳4两，连翘4两，海风藤4两，乌药4两，杜仲4两，桔梗4两，石楠叶4两，肉桂4两，干姜1两，白僵蚕1两，甘草1两，赤芍药1两，升麻1两，虎骨1两，防己1两，白花蛇（酒浸）1两，乳香1两，没药1两，沉香1两。
出处：《疡医大全》卷二十八。
主治：大麻疯。
加减：None
功效：消风顺气。
用法用量：清晨送下1丸。
制备方法：上为细末。用麻黄20斤，去节，水煎1日，去滓，将汁熬成膏，入药末和为丸，每丸重5钱。
临床应用：None
各家论述：None
用药禁忌：不可见风。
附注：None
