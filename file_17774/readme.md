序号：17781
名称：生津饮
组成：人参1钱半，麦冬3钱（去心），知母1钱半（去毛），天冬3钱（去心），草梢1钱半，车前子3钱。
出处：《医略六书》卷二十八。
主治：孕妇淋沥涩痛，脉软数。
加减：None
功效：None
用法用量：水煎，去滓温服。
制备方法：None
临床应用：None
各家论述：妊娠气亏，湿热伤津液，而不能上敷下达，故口燥心烦，小便涩痛淋沥不已焉，人参扶元气以通津液，天冬润肺以资水源，麦冬生津润燥，知母清热除烦，草梢泻火以缓涩痛，车前子清肝以利小便也。水煎温服，使元气内充，则湿热自化，而津液四溢，无不上输心肺，而烦燥宁，下达膀胱肾而涩痛解，何小便淋沥之不痊者，胎孕无不自安矣。
用药禁忌：None
附注：None
