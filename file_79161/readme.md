序号：79177
名称：蜡脂
组成：白蜡10两（炼令白），桃花3分，菟丝子3分，白芷3分，木兰皮3分，细辛3分，辛荑仁3分，白茯苓3分，土瓜根3分，栝楼根3分，白附子3分，杜蘅3分，桃仁（去皮）3分，杏仁（去皮）3分，蔓青子油2升半，羊髓2合，牛髓2合，鹿髓脂2合。
出处：《外台》卷三十二引《崔氏方》。
主治：面病。
加减：None
功效：None
用法用量：任用，每以澡豆洗面后以涂之。
制备方法：上并细切，以苦酒渍1宿，用上件蜡油、髓脂等，煎如面脂法，其蔓青油、酒在前，煎令烟出后，始下蜡、髓讫，纳诸药，候白蓝色黄膏成。
临床应用：None
各家论述：None
用药禁忌：None
附注：蜡脂膏（《普济方》卷五十二）。
