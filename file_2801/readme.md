序号：2801
名称：天灵盖散
组成：天灵盖1两（涂酥，炙令黄），鳖甲1两（涂醋，炙令黄，去裙襴），柴胡1两半（去苗），诃黎勒1两半（煨，用皮），桂心1两半，栀子仁1两，人参1两（去芦头），赤茯苓1两半，贝母1两（煨令黄），桃仁1两（汤浸，去皮尖双仁，麸炒微黄），麦门冬2两半（去心），地骨皮1两，生干地黄2两，槟榔半两，黄耆1两（锉）。
出处：《圣惠》卷二十六。
主治：肺劳羸瘦，四肢无力，每至日晚，即烦热颊赤，痰嗽不利，骨节多疼，或乍寒乍热，饮食不成饥肤。
加减：None
功效：None
用法用量：每服5钱，以童便及水各1小盏，加葱白1茎，生姜半分，煎至1盏，去滓，食前温服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
