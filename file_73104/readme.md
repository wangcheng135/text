序号：73120
名称：山芋丸
组成：山芋1两半，车前子1两半，韭子（炒令熟）1两半，菟丝子（酒浸一宿，别捣、焙）1两半，附子（炮裂，去皮脐）1两半，白龙骨1两半，山茱萸1两半，五味子1两半，牡丹皮1两半，白茯苓（去黑皮）1两半，石斛（去根）1两半，牛膝（酒浸，切，焙）1两，桂（去粗皮）1两，熟干地黄（焙）5两，肉苁蓉（去皴皮，酒浸，切，焙）2两。
出处：《圣济总录》卷五十一。
主治：肾脏风冷气，胸中聚痰，夜梦泄精，腰膝无力，小便频数。
加减：None
功效：None
用法用量：每服40丸，空心暖酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
