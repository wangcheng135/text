序号：66985
名称：麻风丸
组成：鲜皂角刺2斤（好醋煮9日，晒干，取净末1斤，），番木鳖（羊油炙，得法如金色者，净末）8两，苦参（取净末）2两，紫背浮萍（晒干，取净末）1斤。
出处：《疡医大全》卷二十八。
主治：大麻风。
加减：None
功效：None
用法用量：空心服，每日3次。先用葱头、生姜各2斤捣烂，用麻油熬熟，将新鬆布包作6-7包。先将番木鳖刮去皮毛，为末，壮盛者服2分，老弱者服1分；随每人持药1包，在病人身上，周身擦过，用青布衫裤袜穿好，绵被盖紧，睡处不可透风，或生炭火，待大汗后渐渐揭下，乃起近前向火；然后服前药，自能痊可，如身若炭一般而作痒者，将石菖蒲打碎，煎汤，四周围住，不可漏风，连浴数次，黑皮脱去，平复如故。
制备方法：上再用苦参八两，好酒打糊为丸。
临床应用：None
各家论述：None
用药禁忌：若一见风，必成瘫痪，最宜谨防。
附注：None
