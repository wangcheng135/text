序号：44254
名称：上洞小丹
组成：辰砂1斤（用绵帛包裹），黄柏8两（锉），蜜8两。
出处：《杨氏家藏方》卷十四。
主治：诸虚百损，真元虚惫，形体羸瘦，脏腑虚滑，脐腹久冷；及妇人子宫宿寒，赤_x000D_
白带下，盗汗心忪，精神憔悴，服他药不能作效者。
加减：None
功效：None
用法用量：每服1丸，空心、食前米饮送下。
制备方法：上药一处用重汤同煮七伏时，取出辰砂，控干后，用羊蹄根、车前草2件等分，捣细，用砂盒子1枚，入2味药铺底，入辰砂在内，又以2味药盖头满筑，以盒盖子合定，铁线扎缚，赤石脂固缝，盐泥固济，令干，以炭2秤周围簇定，发顶火殿1伏时，火冷取出，去火毒，研令极细，用糯米粥和丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
