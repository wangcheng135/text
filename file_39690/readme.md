序号：39700
名称：细辛膏
组成：细辛（去苗叶）2两，乌喙2两，莽草2两，续断2两，石南2两，辛夷仁2两，皂荚2两，泽兰（去苗）2两，白芷2两，防风（去叉）2两，白术2两，松叶2两，竹叶2两，猪脂半斤，生麻油1斤。
出处：《圣济总录》卷一○一。
主治：头疮有虫，变成白秃。
加减：None
功效：None
用法用量：临卧时，先以热浆水洗头后用药涂匀。如痒，勿搔动。经宿即洗去，再涂。
制备方法：上除脂油外，锉细，以醋5升，入瓷瓶中，水浸1宿取出，用大铛先下脂油，微火煎一二沸，次下诸药，煎候白芷黄，即膏成，去滓，以瓷盒盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
