序号：78365
名称：蜀椒丸
组成：蜀椒（去目及闭口者，炒出汗）半两，人参3分，半夏（汤洗7遍，去滑）3分，菖蒲3分，柴胡（去苗）3分，桂（去粗皮）3分，桃仁（去皮尖双仁，麸炒微黄）3分，木香3分，吴茱萸（汤洗微炒）3分，干姜（炮裂，锉）3分，细辛（去粗叶）3分，桔梗（锉）3分，赤茯苓（去黑皮）3分，芎3分，大黄（锉，炒）2两。
出处：《圣济总录》卷六十七。
主治：诸气积聚坚牢，心腹胀痛。
加减：None
功效：None
用法用量：每服10丸，食前温酒送下。渐加至20丸。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
