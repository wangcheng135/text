序号：40322
名称：英明普救丸
组成：明雄黄5钱，郁金5钱，巴豆霜4钱，乳香1钱5分，没药1钱5分，陈皮1钱5分，木香1钱5分，皂角1钱5分，胆南星2钱，白豆蔻2钱，牛黄3分，麝香3分，琥珀3分。
出处：《全国中药成药处方集》（沈阳方）。
主治：胃脘疼痛，胀满结滞，食物不消，时行疫者。停食停水，积聚痞块，腹大青筋，面黄肌瘦，作冷作烧，内热痰盛。
加减：None
功效：化积开郁，解毒避疫。调胃通便，清热镇惊化痰。
用法用量：每服4小丸，小儿减半。
制备方法：上为细末，江米醋糊为丸，3厘重，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：普救丸（原书沈阳方）、英神普救丸（原书天津方）。
