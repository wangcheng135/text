序号：77399
名称：跌仆损伤急救散
组成：当归尾（酒洗）7钱，自然铜（醋煅7次）7钱，桃仁（去皮尖）7钱，红花7钱，陈麻皮3钱，地鳖虫（烧酒浸，焙）5钱，骨碎补（酒洗，蒸）2钱，大黄（酒洗）2钱，乳香（去油）5分，没药（去油）5分，胎儿骨5分，血竭5分，朱砂5分，雄黄5分，麝香5分。
出处：《疡医大全》卷三十六引周鹤仙方。
主治：跌死、打死尚有微气及骨折瘀血攻心。
加减：None
功效：None
用法用量：如遇跌死、打死，尚有微气者，用酒浆调2厘，入口即活；如骨折瘀血攻心，用药8厘，酒调灌之，其伤骨自上而愈。
制备方法：上为极细末，收贮勿泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
