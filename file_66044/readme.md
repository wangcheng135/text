名称：蛇酒
组成：百花蛇1条（去头尾），乌蛇1条（去头尾），当归3钱，槟榔3钱，灵仙3钱，菖蒲3钱，连翘3钱，薄荷3钱，海桐皮3钱，天麻3钱，风藤3钱，苍术3钱，杏仁3钱，蝉蜕3钱，麻黄3钱，红花3钱，陈皮3钱，麦冬3钱，枫子肉1斤（去油炒），胡麻5两，白芷5两，蒺藜5两，人参5两，辰砂5两，五味子5两，花粉5两，甘草5两，生地5两，熟地5两，木鳖5两，羌活5两，独活5两，防风5两，马鞭草1两，血竭1两，全蝎1两，乳香1两，雄黄1两，木香1两，茴香1两，沉香1两，大腹皮1两，天灵盖1两，黑枣1斤，五加皮8钱，虎骨1两，核桃4斤，首乌5两，白藓皮5钱，荆芥3两，草乌8两，甘菊3两，茯苓3两，蝉蜕5两，川乌5钱，没药5钱，阿魏5钱，蔓荆子、牛黄2钱，寸香5钱。
出处：《秘传大麻疯方》。
主治：漏蹄疯。脚底作痒，麻木肿起，底下开裂，脓水不干者，用神蛇酒治之，如不愈，服搜风散十帖，后服本方。
加减：None
功效：None
用法用量：每日空心、早、晚各服1杯。将药滓晒干，为末，入细药为丸。每服100丸，将前酒送下。
制备方法：上为片，另将粗药用细麻布袋盛之陈酒30斤，将袋入酒内，煮三炷香，隔宿取出药滓；药酒封固。
