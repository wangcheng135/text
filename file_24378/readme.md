序号：24386
名称：八仙玉液
组成：藕汁2杯，梨汁1杯，蔗浆1杯，芦根汁1杯，茅根（水煎取浓汁）1杯，人乳1杯，童便1杯，生鸡子白3枚。
出处：《顾松园医镜》卷十二。
主治：阴虚咳嗽，痰血。
加减：None
功效：None
用法用量：上将诸汁炖滚，与鸡子白和匀，频服之。尝用米仁1两，山药1两，莲肉1两，麦冬1两，白花百合2两、枇杷叶10片，煎浓汁1碗,冲入玉液，再加贝母末，真柿霜和匀，频饮之。
制备方法：None
临床应用：None
各家论述：藕汁性寒带涩，涤热止血；梨汁降火消痰，定喘止嗽；蔗浆消痰止咳：芦根汁清胃止呕；茅根凉金定喘；人乳补阴养血;童便引火下行。
用药禁忌：None
附注：None
