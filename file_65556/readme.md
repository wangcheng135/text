序号：65572
名称：羚羊清肺丸
组成：羚羊（另兑）1钱2分，浙贝8钱，花粉1两，银花1两，小生地1两，黄芩5钱，桔梗1两，玄参（去芦）1两，丹皮5钱，薄荷5钱，石斛2两，天冬5钱，陈皮6钱，大青叶5钱，板兰根5钱，杏仁（去皮，炒）5钱，桑皮5钱，前胡5钱，金果榄5钱，甘草3钱，熟军5钱，枇杷叶（去毛）1两，栀子（炒）1两，麦冬5钱。
出处：《北京市中药成方选集》。
主治：肺热咳嗽，咽喉肿痛，鼻衄咳血，舌干口燥。
加减：None
功效：清肺热，止咳嗽，利咽膈。
用法用量：每服2丸，温开水送下。
制备方法：上为细粉，炼蜜为丸，每丸重2钱，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
