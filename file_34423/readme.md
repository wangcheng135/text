序号：34433
名称：驱邪安神丸
组成：秦艽4两，桑寄生（不见火，捣末）4两，茯神4两，川续断（酒炒）4两，远志4两，海风藤4两，苍术（去粗皮，米泔浸）4两，制半夏4两，熟地（酒煮，晒干）8两，胆南星3两，防风3两，甘枸杞（酒薰）3两，杜仲（盐水炒）3两，川萆薢（酒炒，另捣）3两，牛膝（酒炒）3两，防己3两，肉桂、川乌（童便制）、草乌（童便制）、黄柏（酒炒）、甘草（炙）1两，何首乌重1斤1枚（黑豆煮，晒）。
出处：《疡医大全》卷二十八。
主治：癫疯，半肢、毛、历节疯。
加减：None
功效：None
用法用量：每服100丸，白汤送下，1日2次。
制备方法：上为细末，用鲜石菖蒲十斤去苗，捣汁拌前药末，不可太湿，晒干复磨为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
