序号：66325
名称：雀附丸
组成：雀儿30枚（去嘴脚毛羽胃肠胸骨，用好酒1升煮烂，熟研），补骨脂1两（微炒），木香1两，吴茱萸1两（汤浸7遍，焙干微炒），干姜1两（炮裂，锉），青橘皮1两（汤浸，去白瓤，焙），木瓜1两（捣罗为末），附子1两（炮裂，去皮脐），熟艾末2两（以米醋2升，煎如膏）。
出处：《圣惠》卷九十八。
主治：脏腑久积虚冷，腹胁多气，脾胃乏弱，少思饮食，羸瘦无力。
加减：None
功效：None
用法用量：每日空心及晚食前以温酒或盐汤送下30丸，渐加至40丸。
制备方法：上为末，以雀肉入于艾膏内，和药末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
