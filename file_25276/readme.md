序号：25284
名称：如童老奴丸
组成：稍花2两，灯心灰2两，蛇床子2两，车前2两，苁蓉2两，菟丝子2两，马蔺花2两，大茴2两，韭子2两，荜澄茄2两，故纸2两，川楝1两半，羊藿1两半，枸杞1两半，胡桃肉1两半，茯苓1两半，母丁香5钱，远志5钱，泽泻5钱，川附5钱，巴戟2两，蜘蛛7个（酒浸），柏仁1两半，桑蛸1两半，萸肉1两半。
出处：《全国中药成药处方集》（哈尔滨方）。
主治：精血衰弱，营养不足，神倦体软，气短心跳，不能支持，风寒湿痹，五劳七伤，阳事不举。
加减：None
功效：补血填精。
用法用量：每服2钱，淡盐汤送下，日服2-3次。
制备方法：上为细末，炼蜜为小丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
