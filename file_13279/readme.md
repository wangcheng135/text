序号：13286
名称：人参丹砂丸
组成：人参2两，丹砂（研）2两，紫石英（研）1分，白石英（研）1分，龙齿（研）1分，细辛（去皮叶）1分，赤箭1分，天门冬（去心，焙）1分，远志（去心）1分，生干地黄（焙）1分，菖蒲（九节者，米泔浸，切，焙）1分，龙脑（研）1分，白茯苓（去黑皮）3两，犀角（镑）半两，沙参半两，防风（去叉）半两，麝香（研）半分。
出处：《圣济总录》卷五。
主治：心中风，恍惚惊悸。
加减：None
功效：安神志，化痰涎。
用法用量：每服1丸，温酒嚼下，不拘时候。
制备方法：上为末，炼蜜为丸，如小鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
