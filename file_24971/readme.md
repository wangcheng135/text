序号：24979
名称：如圣煎丸
组成：麻黄（去根节，锉）4两，桑根白皮半斤（锉），桃心4两（锉），柳心4两（锉），黑豆2升，童子小便3升，河水15升（井水亦得，以上7味，同入银石器内熬3分至1分，去滓，别入后药），天南星（生用）2两，天雄（去皮脐，生用）2两，干蝎（去土，炒）2两半，天麻2两，天蓼花1两，麝香3分，腻粉1钱，乳香半斤，木鳖子（去壳）1两（以上9味，捣研为末，以生地黄汁入在前药水中，再用文火同熬成膏，以不津器密收），丹砂1两，云母末1两半，金箔10片，银箔10片。
出处：《圣济总录》卷七（人卫本）。
主治：瘫缓风疾。
加减：None
功效：None
用法用量：服时先取清酒3升，膏子1两，煅过丹砂（研）半两，同入银石器中，熬酒至1升，分作5服，先日午温服1服，日晚1服，更后1服。如人行20里后，取下恶物为候，随病人虚实加减用之，虚者少与，实者稍多。服药后，如汗出不止，煎麻黄根节汤令服，且于密室中将理20日则安。
制备方法：上药将后四味研如粉，用瓷盒盛之，用益母草固济，于地坑子内，用炭5斤，煅令通赤，用湿土盖，候冷取出，如粉红色为度。
临床应用：None
各家论述：None
用药禁忌：None
附注：金铁如圣煎丸（《普济方》卷九十三）。本方方名，原书文瑞楼本作“金映如圣煎丸”。据剂型当作“如圣煎”。
