序号：62777
名称：清气涤痰丸
组成：半夏曲（用齐半夏，选极大者，水浸2-3日，以透心去灰为度，用生姜自然汁1茶匙，同煅白矾4两煎化，将半夏为粗末，拌匀晒干，随症用）1斤，牛胆南星10两，橘红8两，楂肉8两，瓜蒌仁8两（去油），枳实8两，萝卜子8两（炒），茯苓8两，白术8两，黄连8两，香附6两（用青盐2两，水浸，炒），枯黄芩6两（微炒），甘草6两，真紫苏子6两，好沉香2两，白芥子3两。
出处：《遵生八笺》卷十八。
主治：None
加减：老痰，加天门冬肉4两，青礞石2两（消煅）。
功效：健脾胃，化痰涎，宽胸膈，进饮食。
用法用量：每服1钱5分，食远或临睡服。
制备方法：上为细末，竹沥为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
