序号：82055
名称：马蔺丸
组成：马蔺1两半（炒），肉桂3钱（去皮），桃仁1两半，海藻1两半，海带1两半，昆布1两半，厚朴6钱（制），枳实6钱，楝子1两半（炒），延胡1两半。
出处：《医略六书》卷二十四。
主治：男子七疝，妇人阴(疒颓)，脉弦涩滞者。
加减：None
功效：None
用法用量：每服3钱，淡盐汤送下。
制备方法：上为末，醋为丸。
临床应用：None
各家论述：气滞于中，湿热不化，伤厥阴之经，故男子内结七疝，妇人腹痛阴(疒颓)，且令不月焉。马蔺泻热散血；昆布泻热软坚；厚朴宽中散湿热；枳实破滞消结气；海藻化湿热；海带解湿热；桃仁破瘀润燥；延胡活血通经；川楝子泻湿热治疝；广橘核开结气除(疒颓)；肉桂温经，为寒因寒用之响导。醋丸淡盐汤下，使滞化气行，则湿热消散，而七疝无不愈，安有阴(疒颓)不月之患乎！此泻热软坚之剂，为阴(疒颓)七疝之专方。
用药禁忌：None
附注：方中无广橘核，而方论论及此药，疑脱。
