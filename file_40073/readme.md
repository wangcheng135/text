序号：40083
名称：肾附丸
组成：羊肾5对，附子5两（炮裂，去皮脐），钟乳粉2两，桂心2两，诃黎勒皮2两，赤箭2两，山茱萸2两，薯蓣2两，肉苁蓉2两（酒浸1宿，刮去皱皮，炙干），菟丝子2两（酒浸3日，晒干，别捣为末）。
出处：《圣惠》卷九十八。
主治：水脏风冷滞气。
加减：None
功效：耐寒暑，暖腰膝。
用法用量：羊肾去筋膜，批作片子，每1片上铺药末一层，如此层层相隔，以尽_x000D_
为度；然后用湿纸裹数重，更用盐泥重裹，入塘灰火中煨三炊久，候火气通透，亦不得令焦，出药看作紫黑色即住，如肾未熟，即重封更煨；候得所即捣3-5百杵，入少水浸蒸饼为丸，如梧桐子大。每服30丸，每日空腹及晚食前以温酒送下。3日后已觉水脏温暖。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
