序号：6572
名称：二术柴葛汤
组成：柴胡1钱半，白术1钱半，苍术（泔浸）1钱，干葛1钱2分，陈皮7分，甘草（炙）5分。
出处：方出《明医杂著》卷二，名见《医统》卷三十七。
主治：疟疾。
加减：若一日一发及午前发者，邪在阳分，加枯芩、茯苓、半夏各1钱；热甚头痛，再加川芎、软石膏各1钱；口干，加石膏、知母、麦门冬各1钱。若间日或三日发，或午后或夜发者，邪入阴分，加川芎、当归、芍药（酒炒）、熟地黄、知母（酒炒）各1钱，红花（酒洗）、黄柏（酒炒）、升麻各4分，提起阳分，方可截之。若间一日，连发二日，或日夜各发者，气血俱病，加人参、黄耆，白茯苓各1钱以补气，川芎、当归、芍药、熟地黄各1钱以补血。
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：二术柴胡汤（《医钞类编》卷八）。《医宗必读》本方用法：水钟半，生姜五片，煎八分服。
