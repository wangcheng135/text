序号：60674
名称：通神丹
组成：五灵脂2两（初1日用姜汁压至初5日，调井花水，去粗滓），干蝎1分（如僵蚕制度），大川乌2两（去皮，研为细末），草乌3分（如川乌制度），没药半两（用井花水磨膏），天南星1分（用姜汁压1宿，晒干，碾罗为末），白僵蚕1分（洗过，碾末），朱砂3分（飞过，半入药，半作衣），麝香半分（用法酒发1宿，碾细），木香半两（碾末），轻粉1分，滴乳香半两（用井水磨为膏）。
出处：《普济方》卷九十一引《卫生家宝》。
主治：卒中风，左瘫右痪，口眼斜，偏正头痛，夹脑风等；及小儿一切惊风。
加减：None
功效：None
用法用量：每服1丸，食后同生姜1片、薄荷1叶，用茶汤或酒嚼下。
制备方法：上药末于端午日或腊日合和匀，以腊水煮面糊为丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
