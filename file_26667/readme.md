序号：26675
名称：延年益嗣丹
组成：人参3两，天冬（酒浸，去心）3两，麦冬（酒浸，去心）3两，熟地黄2两（酒蒸，捣），生地黄2两，茯苓（酒浸，晒干）5两，地骨皮（酒浸）5两。
出处：《叶氏女科》卷四。
主治：男子阳极艰嗣，相火炽盛，灼伤真阴，以致阳极，阳则亢；或过于强固，强固则胜败不洽，是以无子。
加减：None
功效：None
用法用量：每服70-80丸，空心温酒下。
制备方法：上加何首乌半斤，米泔浸透，竹刀刮去皮，切片，置砂锅内，入黑羊肉1斤，黑豆3合，量著水，上用甑算，算上铺放何首乌，密盖，勿令泄气，蒸1-2时，以肉烂为度，取出晒干，为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
