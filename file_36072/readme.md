序号：36082
名称：参茸丸
组成：熟地1两，龟版8钱，山药8钱，归身8钱，益智5钱，茯神5钱，元肉5钱，茅术5钱，牛膝5钱，故纸5钱，枸杞5钱，辰砂5钱，远志3钱，焦栀3钱，草梢3钱，酒柏3钱，柏仁3钱，枣仁3钱，酒母3钱，山参3钱，鹿茸3钱，琥珀2钱，贡桂2钱，盐砂2钱。
出处：《全国中药成药处方集》（吉林方）。
主治：气血衰弱，体弱神倦，气短无力，腰膝痠痛，怔忡健忘，自汗晕眩，失眠惊悸，消化不良，溏泄清白，以及肾虚阳痿，遗精滑精。
加减：None
功效：补气养血，壮阳添精
用法用量：每服2钱，早、晚空腹服，白水送下。
制备方法：上为细末，炼蜜为小丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：阳事易举及火盛者忌服。忌食生冷。
附注：None
