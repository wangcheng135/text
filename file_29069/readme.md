序号：29077
名称：阿胶鸡子黄汤
组成：陈阿胶2钱（烊冲），生白芍3钱，石决明5钱（杵），双钩藤2钱，大生地4钱，清炙草6分，生牡蛎4钱（杵），络石藤3钱，茯神木4钱，鸡子黄2枚（先煎代水煎服）。
出处：《重订通俗伤寒论》。
主治：热邪久羁，灼烁阴血。筋脉拘急，手足瘛疭，类似风动，或头目眩晕，舌绛苔少，脉细数者。
加减：None
功效：滋阴养血，柔肝熄风。
用法用量：None
制备方法：None
临床应用：None
各家论述：本方以阿胶、鸡子黄为君，取其血肉有情，液多质重，以滋血液而熄肝风；臣以芍、草、茯神木，一则酸甘化阴以柔肝，一则以木制木而熄风；然心血虚者，肝阳必亢，故佐以决明、牡蛎，介类潜阳；筋挛者络亦不舒，故使以钩藤、络石，通络舒筋也，此为养血滋阴，柔肝熄风之良方。
用药禁忌：None
