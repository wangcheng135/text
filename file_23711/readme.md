序号：23719
名称：利膈化痰丸
组成：南星、蛤粉（研细）1两，半夏、瓜蒌仁、贝母（去心）、香附半两（童便浸）。
出处：《丹溪心法》卷二。
主治：一切痰涎壅塞，郁火热于胸膈之间，痰喘不利。
加减：None
功效：豁痰疏郁，泄火散热，降逆气，润肺止嗽。
用法用量：每服50丸，姜汤送下。
制备方法：上为末，用猪牙皂角14梃敲碎，水1碗半，煮杏仁（去皮尖）1两，煮水将干，去皂角，擂杏仁如泥，入前药搜和，再入姜汁泡，蒸饼为丸，如绿豆大，青黛为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中南星、半夏、瓜蒌仁、贝母用量原缺。《杏苑》本方用：南星2两、半夏1两5钱、贝母2两、哈粉1两、瓜蒌仁（另研）、香附各2两、牙皂、青黛各1两、杏仁（另研泥）1两5钱
