序号：24417
名称：导水茯苓汤
组成：泽泻3两，赤茯苓3两，白术3两，麦门冬（去心）3两，紫苏1两，木瓜1两，槟榔1两，陈皮7钱半，砂仁7钱半，木香7钱半，大腹皮7钱半。
出处：《普济方》卷一九一引《德生堂方》。
主治：水肿。
加减：None
功效：利小便。
用法用量：每服5钱，水2盏，加灯心25根，煎8分，去滓，空心服；服此药时，要如熬阿刺吉酒相似，水1斗，止取药1钱，服后小水行时，即渐添多，直至小便变清白色，方为痊愈。如病重者，前药可均作3大服，每服再加去心麦门冬2两，灯草1大把，均半两重，水1斗于砂锅内，下药5两，熬1大碗，再下小铫内煎至1大盏，五更空心服，滓再煎服，连进3服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：茯苓导水汤（《金鉴》卷五十四）。
