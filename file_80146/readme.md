序号：80162
名称：毓麟固本膏
组成：杜仲4两，熟地4两，附子4两，苁蓉4两，牛膝4两，故纸4两，续断4两，官桂4两，甘草4两，生地1两5钱，大茴香1两5钱，小茴香1两5钱，菟丝子1两5钱，蛇床子1两5钱，天麻子1两5钱，紫梢花1两5钱，鹿角1两5钱，羊腰1对，赤石脂1两，龙骨1两。
出处：《慈禧光绪医方选议》。
主治：None
加减：None
功效：补肾固精，温经散寒，种子。
用法用量：妇人贴脐上，男子贴左右肾俞穴各1张，丹田穴1张，用汗中缚住，勿令走动，半月1换。
制备方法：用香油8斤，熬枯去滓，用黄丹48两，再入雄黄、丁香、沉香、木香、乳香、没药各1两，麝香3分，阳起石5分搅匀成膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
