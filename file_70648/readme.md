序号：70664
名称：紫菀汤
组成：紫菀（去苗土）1两，贝母（去心）1两，黄耆（锉，炒）1两，柴胡（去苗）1两，人参1两，白茯苓（去黑皮）1两，麻黄（去根节）1两，杏仁（汤浸，去皮尖双仁者，炒，研）1两，款冬花1两，桂（去粗皮）1两，桔梗1两，陈橘皮（去白，炒）1两，当归（炙，锉）1两，大腹子（锉）1两，桑根白皮（锉，炒）1两，五味子（炒）1两，甘草（炙，锉）1两，生干地黄（焙）1两，半夏（洗去滑，焙）1两半。
出处：《圣济总录》卷八十六。
主治：肺脏虚劳，痰嗽不止，背膊并项筋痠疼，日渐羸瘦。
加减：None
功效：None
用法用量：每服5钱匕，以水1盏半，加生姜半分（切），大枣2枚（擘），同煎取7分，去滓温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
