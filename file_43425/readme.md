序号：43440
名称：养真汤
组成：苍术（净）4两（用姜4两制，微炒黄色，同炒），厚朴4两（去皮，同上制炒），人参1两，1两，白术1两，木香1两，藿香1两，茯苓（去皮）1两，青皮（去白）1两，陈皮1两，槟榔1两，缩砂1两，广术（煨）1两，泽泻1两，甘草（锉，炒）1两，干姜1两，小枣70个（焙干）。
出处：《医方类聚》卷一五三引《经验秘方》。
主治：脾胃不和，中脘气闷，心腹胀满，不思饮食，呕吐痰噫，逆气吞酸，面色痿黄，肌肉消疲，腹胁刺痛，便利不调，或泻痢不止，少力嗜卧，体重节重，五劳七伤，山岚瘴气，八般疟疾，及治咳嗽，脾胃虚弱，食饮不化，
加减：None
功效：None
用法用量：每服2钱，熬生姜、枣汤，空心调服。
制备方法：上为极细末。
临床应用：None
各家论述：None
用药禁忌：老幼气弱皆可服。
附注：None
