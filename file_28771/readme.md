序号：28779
名称：十圣天麻丸
组成：天麻（酒浸，焙干）3两，地榆（净洗，焙）3两，附子（以生姜半两，枣4枚，同煮1时辰，去皮脐，切碎，焙干，炒）3两，白附子（米泔浸，焙干）3两，丁香半两，木香半两，黄耆（细锉）3两，（以上7味，同捣罗为末），雄黄（研）半两，犀角（镑屑为细末）半两，珍珠（研）半两，牛黄（研）1两，麝香（研）1两。
出处：《圣济总录》卷九。
主治：中风，手足偏枯，跛蹇不随。
加减：None
功效：None
用法用量：每服1丸，空心、食前、临卧各1服，温酒嚼下，移时，更以热酒1盏投之。任加衣被盖覆。六十岁以下，两月平复；四十岁以下，一月平复；10日后或汗出，或如虫行，勿怪。如欲常服，每9分为4服，逐日1服。
制备方法：上12味，除7味捣罗外，5味别研，然后同拌和令匀，炼蜜为丸，如樱桃大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
