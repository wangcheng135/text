序号：370
名称：五香枳实汤
组成：青木香9铢，麝香6铢，鸡舌香半两，熏陆香半两，沉香半两，升麻1两，黄芩1两，白蔹1两，麻黄1两，防风半两，秦艽半两，枳实1两半，大黄1两18铢，漏芦半两。
出处：《千金》卷五。
主治：小儿着风热，(疒咅)(疒畾)坚如麻豆粒，疮痒搔之，皮剥汁出，或遍身头面年年常发者。
加减：None
功效：None
用法用量：上(口父)咀。以水5升，煮取1升8合，儿5-6岁者，1服4-5合；7-8岁者，1服6合；10岁至14-15者，加大黄半两，足水为1斗，煮取2升半，分3次服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
