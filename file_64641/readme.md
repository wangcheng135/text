序号：64657
名称：痔漏混元丹
组成：青礞石8两（黑色打碎有金星者佳），黑铅3两（同硫黄4两炒成青金色），陈明瓦3两2钱，明雄2两，水银2两，雌黄2两，白砒8钱，皂矾4两，牙消6两（同青礞石煅金色）。
出处：《疡医大全》卷二十三。
主治：痔漏。
加减：None
功效：None
用法用量：每空心酒下3-4丸，或5-6丸。服后倘喉痛破损，止药1-2日，待喉中不痛，照前又服，以愈为度。
制备方法：各研粗末，升量多少，好做混元球大小，先以灰如药数成球，庶不致药多球小也，球成将药装入混元球内，以泥盖盖之封固，入灰缸内，用金粟火煨一日至夜，又翻转四面俱煨到，取起去盖看时，如药茶褐色、黄色俱好；如白色、黑白色可再封口煨之，养火一日，取出配后药，每灵药1两，加风头贯众去心，净2两，以湿纸包，泥固济晒干，入炭煅红取出，闷息存性，研细末1两，黑枣肉1两2钱，同入石臼内杵为丸如黄豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
