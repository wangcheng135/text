序号：42121
名称：青龟丸
组成：乌龟1个，茯苓5两，薏仁6钱，羊蹄后爪4付，穿山甲5钱（俱用土炒），人参2两，青苔（干者）1两，黄耆1两，当归3两，瓦松2条（阴干，不可火焙），白芷1两，槐米1两。
出处：《辨证录》卷十三。
主治：痔漏。
加减：None
功效：祛湿败毒。
用法用量：每日服3钱，服至1月而漏疮干，服至2月漏疮满，服完全愈不再发。
制备方法：上药各为细末，将龟用石臼捣死，以药末拌之，饭锅内蒸熟，将龟肉用火焙干为末，同前药炼蜜为丸。
临床应用：None
各家论述：此方治漏，实有神效，非世上大概之方。况虽去湿而复不散气，虽败毒又不损血，补破于无形，填隙于有孔。
用药禁忌：服药时，务必独宿，戒酒色3月。倘服药时，不断酒色，不能奏功，不可不慎。
附注：None
