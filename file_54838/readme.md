序号：54853
名称：桂心酒
组成：桂心4两，牡丹4两，芍药4两，牛膝4两，干膝4两，土瓜根4两，牡蒙4两，吴茱萸1升，大黄3两，黄芩2两，干姜2两，虻虫200枚，（庶虫）虫70枚，蛴螬70枚，水蛭70枚，乱发灰1两，细辛1两，僵蚕50枚，大麻仁、灶突墨3升，干地黄6两，虎杖根5两，鳖甲5两，菴（艹闾）子2升。
出处：《千金》卷四。
主治：月经不通，结成癥瘕。
加减：None
功效：None
用法用量：上(口父)咀。以酒4斗，分两瓮浸之，7日并一瓮盛，搅令调，还分作两瓮。初服2合，每日2次，加至3-4合。
