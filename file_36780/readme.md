序号：36790
名称：固齿白玉膏
组成：龙骨1两，阳起石5钱（姜蚕49个，防风5钱，当归5钱，川芎5钱，牙皂5钱，青盐5钱，升麻5钱，白芷5钱，骨皮5钱，细辛3钱，藁本3钱，为粉末，长流水于砂锅内，桑柴火熬汁，去滓，再煎汁1碗，将龙骨、阳起石火煅通红淬药汁内7次，焙干为末），铅粉1两，珍珠3钱，象牙（末）5钱，麝香2钱。
出处：《外科大成》卷三。
主治：一切牙疼，及齿动摇而不坚固者。
加减：None
功效：None
用法用量：用时先漱口净，剪小条贴齿根上，闭口不语。
制备方法：上为末，和匀。黄蜡3两熔化，滤净再化，俟温方入前药和匀，乘热摊纸上，如冷，烧热熨斗，仰放，纸铺斗上摊之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
