序号：55280
名称：桃仁汤
组成：桃仁5两，吴茱萸2升，黄耆3两，当归3两，芍药3两，生姜8两，醍醐（百炼酥）8两，柴胡8两。
出处：《千金》卷三。
主治：产后往来寒热，恶露不尽。妊娠堕胎后，血不出腹痛。
加减：None
功效：None
用法用量：上(口父)咀。以酒1斗，水2升，合煮取3升，去滓，适寒温，先食服1升，日3次。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：此因恶露不通而见往来寒热，故用桃仁、芍药散血，即兼黄耆、当归补血。柴胡、吴茱萸、生姜专为寒热而设，吴茱萸散内痹之血，柴胡、生姜开外郁之气。用醍醐者，专通血结，产母痹约之故，若大便不固，殊非所宜，观后柴胡汤可知。
用药禁忌：None
附注：None
