序号：76015
名称：蛤粉膏
组成：蛤粉5钱，轻粉2钱半，青黛1钱半，川黄柏2钱半，石膏（煅）5钱。
出处：《中华皮肤科杂志》（1958；3：214）。
主治：酒渣鼻。
加减：None
功效：None
用法用量：临用时，先以温热水洗脸，将药膏加入适量冷水调稀，涂患处，每日早晚各1次。
制备方法：上为极细末，用芝麻油50或120毫升混合调匀，贮存备用。
临床应用：酒渣鼻：本方治疗酒渣鼻40例，初步疗效结果：治愈（皮疹消失，皮肤与正常皮色同）14例；显著进步（新疹出现减少，潮红减退）25例；恶化（潮红加剧，新疹增多，有时兼见渗出、疼痛肿胀）1例。治愈率35%，有效率87.5%。治愈之14例疗程为1-3个月，有效之25例尚在治疗中。
各家论述：None
用药禁忌：None
附注：None
