序号：76645
名称：舒筋丸
组成：苍术（削成块子）1两，川牛膝（去芦头，锉）1两，肉苁蓉（锉）1两，木瓜（去瓤）1两（并酒浸1宿，焙干），虎胫骨1两（酥炙），附子1两半（炮，去皮脐），乳香（研）半两，没药（研）半两，官桂（不见火）半两，血竭（研细）半两。
出处：《朱氏集验方》卷一。
主治：半身不遂，手足无力，皆由饮食起居失节，阳气不敛，风邪所侵。
加减：None
功效：生阳消阴，疏风去湿。
用法用量：每服50-70丸，空心温酒送下，1日2次。服之旬日，以手足轻，脚膝有力为验。
制备方法：上为细末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：可戒羊肉猪头动风气物。
附注：None
