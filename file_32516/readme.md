序号：32525
名称：应用膏
组成：当归8钱，连翘8钱，白及8钱，白蔹8钱，大黄8钱，山栀8钱，官桂2钱，苍术6钱，羌活6钱，天麻6钱，防风6钱，黄耆6钱，荆芥6钱，川甲6钱，甘草6钱，芫花6钱，方八1两，蓖麻子1两，小生地1两。
出处：《疡科心得集·家用膏丹丸散方》。
主治：疔、疽、流注、腿痈穿溃。
加减：None
功效：None
用法用量：摊用。
制备方法：用真麻油十斤，入药，文武火熏枯，滤去滓，再熬至滴水成珠；每斤净油，春、秋下淘净东丹5两，冬4两，夏6两，收成膏后，下乳香、没药末各一两搅匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：化脓生肌膏（《疡科心得集·方汇》卷下）。
