序号：33134
名称：万应愈风酒
组成：金毛狗脊（炙，去毛）1两，川牛膝1两，海风藤1两，广木香1两，川桂枝1两，左秦艽1两，大熟地1两，补骨脂1两，川杜仲1两，千年健1两，追地风1两，藏红花1两，枸杞子1两，肥玉竹1两，西羌活1两，独活1两，生川乌1两，官桂1两，黄耆1两，党参1两，肉桂1两，明天麻1两，黄皮1两，女贞子1两，淡附子1两，威灵仙4两，全当归4两，油松节4两，野桑枝（切）4两，红曲5钱，大枣1斤，赤沙糖1斤，白蜜糖8两，桂元肉2两，鹿角胶（炖）2两。
出处：《中国医学大辞典》引越人氏方。
主治：气血虚损，感受风湿，手足痠麻，腰膝百节疼痛，甚至半身不遂，口眼喎斜，及一切远近风证。
加减：None
功效：None
用法用量：轻者每服2-3斤即愈，重者不过6-7斤断根。
制备方法：装入夏布袋内，先用陈酒5斤，将药袋炖透，再合烧酒25斤，共装入坛内，加香味封固。待半月后取用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
