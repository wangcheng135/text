序号：75212
名称：腽肭脐丸
组成：腽肭脐（酒浸，微炙）1两，鹿茸（去毛，酥炙）1两，肉苁蓉（酒浸，切，焙）1两，牛膝（酒浸，切，焙）1两，人参1两，木香1两，独活（去芦头）1两，天麻1两，白术1两，防风（去叉）1两，巴戟天（去心）1两，麝香（研）1两，铁粉（研）1两，五味子1两，石斛（去根）3分，沉香（锉）3分，白茯苓（去黑皮）3分，远志（去心）3分，菖蒲（米疳浸，切，焙）3分，山芋3分，荜澄茄3分，丁香3分，肉豆蔻（去壳）3分，诃黎勒皮3分，槟榔（锉）1两半，熟干地黄（焙）1两半，萆薢1两半，松花1两半，丹砂（研）2两，赤石脂（研）2两。
出处：《圣济总录》卷一八六。
主治：元脏虚冷，腰膝无力疼痛。
加减：None
功效：滋润肌肤，悦泽颜色，进饮食。
用法用量：每服20-30丸，空心、食前温酒送下，粟米饮亦得。
制备方法：上为细末，同罗令匀，炼蜜和丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
