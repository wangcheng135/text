序号：55348
名称：大青散
组成：大青3分，犀角屑3分，玄参3分，黄芩3分，麦门冬3分（去心），川升麻3分，荠苨3分，甘草半两（炙微赤，锉），知母3分，石膏1两，枳壳1两（麸炒微黄，去瓤），川大黄1两（锉碎，微炒），红雪1两，吴蓝3分，葛根1两（锉）。
出处：《圣惠》卷四十五。
主治：服乳石，补养过度，饮酒、食肉热面大多，致脚气发盛攻心，烦热，躁渴，闷乱，神思恍惚。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，淡竹叶2-7片，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“大青龙散”。
