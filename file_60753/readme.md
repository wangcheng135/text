序号：60768
名称：通真丸
组成：当归（去尾）、苍术（切、炒）、肉桂、防风、川芎、人参、白芍药、白薇（去土）、熟地黄（酒炒）、牡丹皮、茴香、白术、白茯苓、桔梗、附子（炮）、泽兰叶各等分。
出处：《得效》卷十五。
主治：血崩，经脉不匀，赤白带下；血风瘾疹瘙痒；冷气块筑心腹，呕逆反胃；肠风泻血，赤白痢，月信不止；血风劳倦，头疼眼花；月信不行，室女红脉不通；产后诸风，中风不语迷闷；胎漏下血，气刺心腹。
加减：None
功效：None
用法用量：每服1丸，血崩，经脉不匀，赤白带下，炒当归酒送下；血风瘾疹瘙痒，薄荷蜜汤送下；冷气块筑心腹，呕逆反胃，炒盐汤送下；肠风泻血，赤白痢，月信不止，米饮送下；血风劳倦，青蒿酒送下；头疼眼花，荆芥茶送下；月信不行，室女红脉不通，产后诸风，中风不语迷闷，每服5丸，红花苏木汤送下；胎漏下血，气刺心腹胀满，炒姜酒送下。
制备方法：上为末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
