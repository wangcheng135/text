序号：34305
名称：万灵膏
组成：归尾5钱，红花5钱，大黄5钱，苏木（捶碎）5钱，桃仁5钱，杏仁5钱，三棱5钱，蓬术5钱，枳壳5钱，枳实5钱，苍术5钱，厚朴5钱，槟榔5钱，青皮5钱，白芥子5钱，香附5钱，青木香5钱，乌药5钱，水红花根5钱，野苧根3钱，生地3钱，川椒3钱，肉桂3钱，干漆3钱，皂角3钱，玄胡索3钱，白芷3钱，仙灵脾3钱，南星3钱，半夏3钱，防风3钱，荆芥3钱，羌活3钱，独活3钱，紫苏3钱，巴豆（去壳）3钱，麻黄3钱，秦艽3钱，木鳖子（去壳）3钱，大风子（去壳）3钱，赤芍3钱，海风藤3钱，防己3钱，川山甲3钱，蜂房3钱，白附子3钱，高良姜3钱，骨碎补3钱，川芎3钱，蜈蚣12条，蛇蜕2条，桑枝长3寸者30段，槐枝长3寸者30段，柳枝长3寸者30段，桃枝长3寸者30段。
出处：《万氏家抄方》卷二。
主治：痞积，并未溃肿毒，瘰疬痰核，跌打闪挫，及心腹疼痛、泻痢、风气、杖疮。
加减：None
功效：None
用法用量：如痞症，每膏药1斤，加阿魏5钱，酒化和入，用狗皮摊贴患处，常以热手摩之，令药气透。
制备方法：入麻油内，用铜锅煎药枯黑色，滤去滓，再煎，滴水成珠，取起；松香明净者不拘多少，先用水煮滤净，次用老酒煮入水中，抽扯数10次，每松香3斤，入葱汁1碗、姜汁1碗、蒜汁1碗、韭汁1碗、艾汁1碗，再熬汁干，又入水中抽扯数10次；然后每药油4两，入松香1斤，飞丹4两，熬成膏，取起，入后细药末：五灵脂、雄黄、木香各5钱，沉香3钱，乳香、没药各1两（焙去汗），黄蜡2两，樟脑2两。共为细末，配膏药1斤半，慢火熬，用槐枝不住手搅匀，再入水抽扯百余次用。
临床应用：None
各家论述：None
