序号：69317
名称：黄紫丹
组成：白术5钱，茯苓3钱，当归5钱，羌活1钱，紫苏1钱，甘草1钱，细辛5分，黄芩1钱，麦冬5钱，人参1钱，贝母1钱。
出处：《辨证录》卷五。
主治：内伤脾肾，而风乘虚以入肺，则经络之间不相流通，头痛发热，身疼腰重，骨节俱痠疼，恶风无汗。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：此方补多于散，何补之中又纯补脾而不补肾耶？人生后天以脾胃之气为主，脾健则胃气自开，胃开则肾水自润。况人参、白术原能入肾，而白术尤利腰脐，一身之气无不利矣。何况肺经为脾胃之子，母健而子亦健，力足以拒邪。又有紫苏、黄芩、羌活、贝母祛风、散火、消痰、泄水之药，足以供其战攻之具，自然汗出热解，而邪从外越也。
用药禁忌：None
