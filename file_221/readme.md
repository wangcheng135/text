序号：221
名称：五虎丹
组成：水银62g，白矾62g，青矾62g，牙消62g，食盐31g。
出处：《中医皮肤病学简编》。
主治：皮肤癌。痈疽疔疮，慢性瘘管，淋巴结核等需要腐蚀脱落者。
加减：None
功效：祛腐拔毒，生新。
用法用量：先将水银与矾磨研，以不见水银为度，再将余药加入共研细末。将上药末置入小铁锅内，盖大碗1只，用泥土密糊封闭，文火炼2-3小时，待冷却，轻轻除去泥土，将碗取出，碗底附着如霜之白色结晶，即为五虎丹。糊剂：五虎丹结晶体18g，熔炼0.5g，红娘0.5g，斑蝥0.5g，羊金花粉1g，用浆糊调成糊状，粘涂肿块上面，以普通膏药贴之；钉剂：药物份量同上，用米饭赋形。搓成两头尖的梭状条，每支长2～3cm，重0.65g，阴干，用时插入癌组织。肿块脱落坏死后，改用红升丹细粉末撒布，贴膏药至疮面愈合。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：不可口服。
