序号：22412
名称：阳液方
组成：黄米2升（醋煮粥），曲细末3升，糯米2升（醋煮粥）。
出处：《普济方》卷二二六。
主治：诸虚百损，气血劳伤，因病久深，变生膈气，胁肋刺痛，噎痞心胸，食结不消，哕逆呕水，翻胃吐食，大便硬秘，形体瘦枯，以致难救者。
加减：None
功效：颐生养气，实腹虚心。
用法用量：上量寒温，和器内，停候发过沉澄之时，又入饧稀6斤，等候去糟粕讫，自然上清下澄以成汤液，昼夜12时辰，停分3度，1度1服，俟药气下沉后，停待1时，温服汤液1盏液后，又停1时，更吃白饧数块，日夜长短品3服，饮液食饧均9度，饮食汤液造作，如前烹食药相应。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
