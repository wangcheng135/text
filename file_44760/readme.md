序号：44775
名称：急风散
组成：丹砂1两，草乌头3两（一半生用，一半以火烧存性，于米醋内淬令冷），麝香（研）1分，生乌豆（同草乌一处为末）1分。
出处：《局方》卷八（吴直阁增诸家名方）。
主治：年深日远，偏正头痛，肝脏久虚，血气衰弱，风毒之气上攻，头眩目晕，心忪烦热，百节痠痛，脑昏目痛，鼻塞身重，皮肤瘙痒，面上游风，状若虫行，及一切头风，妇人血风攻疰。
加减：None
功效：现箭头；止血定痛。
用法用量：破伤风，以酒1小盏调服半钱；如出箭头，先用酒1盏调服半钱，却以药贴箭疮上。
制备方法：上为细末，都拌匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中生乌豆，《普济方》引作“生乌头”，《杂病源流犀烛》作“生黑丑”。
