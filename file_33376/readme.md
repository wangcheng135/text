序号：33385
名称：羌活煮散
组成：羌活2两，荆芥2两（去梗），附子2两（去皮脐），秦艽2两（去芦），人参1两，麻黄2两（去节），茯苓1两（去皮），牛膝2两（酒浸1宿），白蒺藜2两（酒浸1宿，焙），沉香1两，牡丹皮1两，当归1两，半夏1两（姜汁浸1宿），汉防己1两（去皮），官桂1两（去皮），甘草1两，鹿茸（酥炙）、萆薢1两（姜汁浸）。
出处：《博济》卷一。
主治：风劳攻疰，四肢背胛酸疼，上焦虚热，心胸躁闷，面无颜色，四肢昏沉，多困少力，元脏虚惫，腰脚沉重，日渐羸瘦，冷气时攻，肠胁(疒丂)刺胀满，酒后痰唾稠多。
加减：None
功效：None
用法用量：每服2钱，水1盏，加大枣2枚，葱白1寸，煎至7分，温服，空心、午前、临卧各1服。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：忌动风物。
附注：羌活汤（《圣济总录》卷八十七）。
