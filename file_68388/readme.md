序号：68404
名称：黄连上清膏
组成：黄连2两7钱，黄芩11两，大黄11两，赤芍11两，生栀子3两4钱，川芎3两4钱，当归3两4钱，连翘3两4钱，菊花3两4钱，花粉3两4钱，甘草1两8钱，黄柏4两4钱，玄参（去芦）3两4钱，桔梗3两4钱，芥穗3两4钱，薄荷3两4钱，银花4两3钱，生石膏4两3钱。
出处：《北京市中药成方选集》。
主治：实火里热，头晕耳鸣，口舌生疮，牙龈肿烂，暴发火眼，大便秘结，小便赤黄。
加减：None
功效：清火散风，泻热消肿。
用法用量：每服3-5钱，温开水冲服。
制备方法：上药切碎，水煎3次，分次过滤，去滓，取滤液合并，用文火煎熬浓缩至膏状，以不渗纸为度，每两清膏兑炼炼蜜1两，装瓶重2两。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
