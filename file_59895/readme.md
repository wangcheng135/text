序号：59910
名称：调经散
组成：没药（另研）半钱，琥珀（另研）半钱，桂心半钱，赤芍药半钱，当归半钱，细辛半钱，麝香（另研）半钱。
出处：《产有宝庆集》卷上。
主治：产后四肢浮肿。因产后败血停积于五脏，循经流入于四肢，留淫日深，腐败如水，故令四肢面目浮肿。产后败血上干于心，心不受触，致心烦躁，卧起不安，如见鬼神，言语颠倒。
加减：None
功效：None
用法用量：每服半钱匕，生姜汁，温酒各少许调匀服。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：小调经散（《妇人良方》卷二十二）、小调经汤（《医统》卷八十五）。本方加炙甘草，名“调经汤”（见《妇科玉尺》），又名“调经饮”（见《医级》）。
