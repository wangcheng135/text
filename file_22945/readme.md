序号：22953
名称：防己丸
组成：防己半两，白前半两，五味子半两，紫菀（去苗上）半两，桑根白皮（锉）3分，马兜铃3分，麻黄（去根节）3分，桔梗（炒）3分，柴胡（去苗）3分，大腹皮（锉）3分，赤茯苓（去黑皮）1两，陈橘皮（汤浸去白，焙）1两，甘草（炙，锉）1分，杏仁50粒（汤浸，去皮尖双仁，炒）。
出处：《圣济总录》卷八十。
主治：水气肿满，肺气喘急，咳嗽胀闷，坐卧不得，喉中作声，心胸痞滞。
加减：None
功效：None
用法用量：每服15-20丸，温生姜汤送下，不拘时候。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
