序号：80868
名称：麝香丸
组成：麝香（细研）半两，牛黄（细研）半两，杏仁（汤浸去皮尖，双仁，研如膏）1两，丹砂（细研）1两，芍药1两，白茯苓（去黑心）1两，真珠（研如粉，水飞过）1分，甘遂1分，巴豆（去皮心，微炒，研如膏）3分，牡蛎（熬，别捣罗，研如粉）1分，虎睛2枚（微炙，研）。
出处：《圣济总录》卷一七一。
主治：小儿诸疾，一岁以上，三十六种无辜疳，湿闪癖，食痫，天行赤眼，急黄。
加减：None
功效：None
用法用量：候服取2丸，如麻子大，温水送下，随儿大小加减。
制备方法：除巴豆外，上药各为末，入巴豆，炼蜜和捣，入密器中贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
