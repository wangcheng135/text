序号：40510
名称：虎骨酒
组成：虎骨1升（炙焦碎如雀头），丹参8两，干地黄7两，地骨皮4两，干姜4两，芎4两，猪椒根5两，白术5两，五加皮5两，枳实5两。
出处：《千金》卷十一。
主治：肝虚寒劳损，口苦，关节骨疼痛，筋挛缩，烦闷。
加减：None
功效：None
用法用量：上(口父)咀，绢袋盛。以酒4斗，浸4日，初服6-7合，渐加至1升，1日2次。
制备方法：None
临床应用：None
各家论述：肝主筋，肾主骨，肝劳而至筋挛骨疼，风寒在下；口苦烦闷，风热在上也。虎骨、猪椒、芎内搜筋骨之风；干姜、白术、五加下追关节之痛；地黄、地骨上清烦扰之热；枳实、丹参中散瘀滞之血也。
用药禁忌：None
附注：None
