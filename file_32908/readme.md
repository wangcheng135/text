序号：32917
名称：羌活丸
组成：羌活（去芦头）半两，鹿茸（去毛，酒炙）半两，牛膝（酒浸，切，焙）半两，熟干地黄（切，焙）半两，菟丝子（酒浸，别捣）半两，酸枣仁（炒）半两，山茱萸半两，巴戟天（去心）半两，茯神（去水）半两，五加皮（锉）半两，防风（去叉）半两，桂（去粗皮）半两，五味子半两，蛇床子（炒）半两，生干地黄（焙）1两，黄芩（去黑心），白鲜皮1分，羚羊角（镑）1分。
出处：《圣济总录》卷八。
主治：风冷下注，腰脚不随，五劳七伤六极，并诸风痹。
加减：None
功效：None
用法用量：每服20-30丸，空心温酒送下，1日2次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
