序号：71292
名称：紫石英丸
组成：紫石英1两（细研，水飞过），肉苁蓉2两（酒浸1宿，锉，去皴皮，炙令干），白石英1两（细研，水飞过），磁石2两（烧，醋淬10遍，捣细研，水飞过），鹿茸1两（去毛，涂酥，炙微黄），菟丝子1两（酒浸3日，晒干，别捣为末），人参1两半（去芦头），黄耆2两（锉），钟乳粉2两，熟干地黄2两，巴戟1两半，白茯苓1两，补骨脂1两（微炒），覆盆子1两，附子1两（炮裂，去皮脐），当归1两（微炒），杜仲1两（去粗皮，炙令微黄，锉），天门冬1两（去心，焙），五味子1两，石斛2两（去根，锉），桂心1两，柏子仁1两，蛇床子1两，棘刺1两，牛膝2两（去苗），续断1两，腽肭脐1两（酒洗，微炙）。
出处：《圣惠》卷七。
主治：肾气虚损，食饮不为肌肤，骨痿无力，腰疼痛。
加减：None
功效：益气力，令人充健。
用法用量：每服30丸，空心及晚食前以温酒送下，渐加至50丸。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
