序号：73249
名称：斑龙丸
组成：鹿角胶（以酒浸胶数日，煮糊丸众药）10两，鹿角霜（碾为细末）10两，菟丝子（净洗，酒浸两宿，蒸，研）10两，柏子仁（净者，别研）10两，熟地黄（好者，酒浸2宿，蒸焙，余酒入在胶内）10两。
出处：《百一》卷四。
主治：虚劳，消渴。虚劳。真阴虚极及老人、虚人。
加减：None
功效：平补阴阳，填精益髓，强壮筋骨，安神定志，驻颜益寿。
用法用量：每服50-100丸止，早、晚空心盐汤或酒任下。逐日早晚服，久之大有功效。
制备方法：上先焙鹿角霜、菟丝子、地黄，干后研为细末，方入柏子仁在众药内研，却将鹿角胶酒约3-4升煮糊为丸，如梧桐子大。
临床应用：None
各家论述：1.《医方集解》：此手、足少阴药也。鹿角胶霜、菟丝、熟地皆肾经血分药也，大补精髓；柏子仁入心而养心气，又能入肾而润肾燥，使心肾相交。心志旺而神魂安，精髓充而筋骨壮，去病益寿，不亦宜乎？_x000D_
2.《医方论》：鹿角、菟丝阴中之阳也；地黄阴中之阴也，用以补肾，不偏不倚。
用药禁忌：None
附注：斑龙脑珠丹（《续易简方论后集》卷四）、仙传斑龙丸（《万氏家抄方》卷四）。
