序号：36964
名称：夜光育神丸
组成：熟地黄（洗，晒干，酒浸）、远志（净洗，就砧上捶碎，取皮去骨木）、牛膝（去芦）、菟丝子（净洗，晒干，以酒浸，别研如泥）、枳壳（净洗，去瓤，麸炒赤色）、地骨皮（须自取，净洗，净砧上捶打取皮）、当归（净洗，晒干，焙亦得）各等分。
出处：《寿亲养老》卷二。
主治：眼昏，健忘。
加减：None
功效：养神明，育精气，益智聪心，补血不壅燥，润颜色，远视移时，目不（目巟目巟），脏腑调适。久服目光炯然，神宇泰定，语音清澈，就灯永夜，眼力愈壮，并不昏涩，不睡达旦，亦不倦怠。服二三月后，愈觉神清眼明，志强力盛，步履轻快，体气舒畅。
用法用量：每服30丸，空心盐酒送下，加至40-50丸亦不妨。若不饮酒，盐汤亦得，但不如酒胜。常饵如饮食，日不可辍，惟在修合洗濯洁净，药材须件件正当，不宜草率。
制备方法：除地黄、菟丝子别器用酒浸，其余5味，同锉细，并入一缽内，或瓷瓮内，若每件10两，都用第一等无灰浓酒6升，同浸3宿，取出，文武火焙干，须试火令得所，不可太猛，恐伤药性，十分焙干，捣罗为末，以两手拌令十分匀，炼蜜为丸，如梧桐子大。炼蜜法，各五滚，夏六七滚，候冷，以纸贴惹去沫。丸后都入微火焙，少顷入瓮收。
