序号：24821
名称：妇康片
组成：益母草250g，熟地黄125g，当归100g，川芎75g，酒白芍75g，茯苓75g，炒白术75g，延胡索50g，蜜甘草50g，人参50g，阿胶25g。
出处：《吉林省中成药暂行标准》。
主治：气血两亏，体虚无力，月经不调，经行腹痛。
加减：None
功效：补气，养血，调经。
用法用量：口服，每次5片，每日2次。
制备方法：将当归、川芎、白术、阿胶共研细粉，过120目筛：将益母草、熟地黄、白芍、茯苓、甘草酌予碎断．煎煮三次，分次过滤，合并滤液，深缩成膏；将人参、延胡索分别以60％、70％乙醇按渗漉法提取，提取液浓缩成膏；将上述药粉、浓缩膏混合均匀．干燥，粉碎，过100目筛，加入适量的黄糊精，混合均匀，干燥，整粒，应出颗粒510g，公差率&plusmn;3％，加硬脂酸镁混合均匀，压片，每片重0．5g。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
