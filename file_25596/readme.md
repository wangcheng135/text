序号：25604
名称：朱氏神效吹药
组成：青皮、知母、白癣皮、陈皮、云茯苓、蒺藜、川槿皮、地骨皮、白术、甘菊、角刺、厚朴、麦门冬、麻黄、干葛、山奈各等分。上药共浸水内，春浸5日，夏浸3日，秋浸7日，冬浸10日，去滓煎干，后入以下诸药：儿茶1钱5分，血竭2钱5分，螵蛸5钱（煨），没药2钱（炙），硼砂1两，石蟹5分，石燕1钱6分，玄明粉3钱5分，金精石2钱5分（醋煅），银精石2钱5分（醋煅），铜青5钱，青礞石1钱5分，桑皮灰3钱，朱砂3钱，磁石1两（醋煅），龙骨3钱（煅），轻粉3钱，熊胆3钱（炙），明雄黄5钱（水飞），胆矾1两，飞丹5钱，制甘石6钱，乳香1钱5分（炙）。
出处：《喉科紫珍集·补遗》。
主治：咽喉七十二种，一切表里等证。
加减：None
功效：None
用法用量：None
制备方法：上为细末，入前膏内搅匀候干，贮瓷瓶中应用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
