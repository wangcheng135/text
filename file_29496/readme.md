序号：29504
名称：阿魏丸
组成：白术5两（用酥油炒3两，土炒2两），苍术3两（米泔水浸2日，去皮，再用芝麻2两，同浸，磨下，取粉晒干），半夏（姜制）1两，白茯苓（去皮）1两，陈皮1两，黄连（酒炒）2两，山楂（去核）1两，麦芽（炒）1两，枳实（面炒）2两，萝卜子（炒）2两，当归2两，红花1两，楮实子（炒）2两，牛黄1钱，水红花子（炒）3两，小桃红子（炒）3两，芦荟1两，阿魏1两，酥油2两，人中白（火煅）5钱，黄蜡3两（二味同化入药末内），桃仁（去皮）1两，海带2两，紫菜3两，干碱（炒）2两，三棱（煨）1两，莪术（煨）1两，胡黄连1两，沉香1两。
出处：《古今医鉴》卷十三引鲍思斋方。
主治：小儿癖疾。
加减：None
功效：None
用法用量：每服20-30丸，水红花子煎汤送下；白汤、黄酒亦可。
制备方法：炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
