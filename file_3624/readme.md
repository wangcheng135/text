序号：3624
名称：巴戟丸
组成：川楝子（去核）、胡芦巴（炒）、白姜（炮）、川椒（炒）、茴香、川牛膝（酒浸，焙）、破故纸（炒）、山药、木通、肉桂（去皮）、牡蛎（煅）、附子（炮，去皮脐）、赤石脂各等分。
出处：《朱氏集验方》卷八。
主治：诸虚不足，真阳衰惫。
加减：肾厥头痛，加川芎；肺虚咳唾，加五味子、款冬花；背膊劳倦，加沉香；脾胃不和，加荜澄茄；虚疟寒热，加蜀漆叶；心神不宁，加龙齿、酸枣仁(炒，去壳)。
功效：None
用法用量：每服50丸，空心盐汤、温酒任下。
制备方法：上为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方名巴戟丸，但方中无巴戟，疑脱。
