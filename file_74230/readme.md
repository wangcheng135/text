序号：74246
名称：山精寿子丸
组成：山药2两5钱（用心结实者，有蛀者勿用），黄精5两2钱（取真者，另杵膏待用。若9蒸9晒，干杵末用更好），黑枣7两5钱（择肥大者，去皮核及腐烂者，另杵膏待用），怀牛膝1两5钱（去芦净，酒拌蒸，或衬何首乌蒸，晒干用。或竟以牛膝易石斛亦可，然需加倍用。石斛生六安山中，形如蚱蜢髀，味甘体粘方真），大何首乌2两5钱（或3两亦可，用黑豆汤浸软，木棒打碎，置瓦器中，底注黑豆汤，务以豆汤拌湿，蒸1炷线香时，候冷取晒，俟水干，又拌蒸，如是9次，夏月1日3-4回蒸晒可也，晒极干，称准），川杜仲2两（炒，碾取净末，称准），川续断2两（酒润，剥净肉，锉，晒干），大熟地4两（煮熟者气味皆失，不堪用，必须9蒸9晒为妙），草覆盆子3两5钱（去蒂，以酒拌，焙干，研末用）沙苑蒺藜2两5钱（炒用），川巴戟天2两（酒浸，去骨，蒸熟，晒干用），肉苁蓉2两（酒洗，去泥甲，但不可过洗尽滑腻，恐伤去肉，隔纸烘干，再称准分两），远志2两（甘草汤浸，去骨，仍以甘草汤拌，炒干用，取净肉称准），菟丝子4两（择色黑而大者，去净，以布袋盛之，洗至水清，以瓦器蒸开肚皮，杵烂做饼，晒干称用），白茯苓2两（选洁白者，出六英山中或云南者佳，各处市买咀片多有连膜者，非为末水漂，其膜不能去，然过水力已减矣。或用云南整块茯苓，自去膜用，不令见水，盖不切为片，则膜易去），山萸肉2两（去核，取净肉称准，酒蒸，杵烂，晒干），辽五味子2两，甘州枸杞5两（去梗蒂净）。
出处：《胎产心法》卷上。
主治：真阳不足，壮年之男，种玉无成，幼岁之妇，从不受孕，或受胎而中怀堕落，或得正产而又生女非男，或生而不育，或育而夭，即苟延性命，难免多疾病者。
加减：脾虚易泄泻者，山药多用；阴虚之人，大熟地可用6两；阳萎者，草覆盆子多用；肝虚滑精，沙苑蒺藜多用；相火不足者，川巴戟天多用；滑精，经行多或淋沥不断者，山萸肉多用，肝气郁结者少用；肝气郁结，肺有热者，辽五味子少用。
功效：延寿。
用法用量：每服3-4钱，空心百沸淡盐汤送下，久服愈好。
制备方法：上药除精、枣2膏，余共为细末，徐徐上于精、枣膏内，杵和极匀，炼蜜为丸，如小豆大。
临床应用：None
各家论述：高益谦曰：补阳而专事参、附、耆、硫辈，骤补其火，不惟壮火食气，难免阳长阴消，阴不敌阳，而能寿能子又难。此方药性，不寒不热，类多平和，补阳不致阴消，久服长年无疾，效过多人，笔难罄书。
用药禁忌：如孕妇忌用牛膝，竟以石斛3两代之。
附注：此丸能延己寿，而生子又寿，无论有病者宜服，即无病服之犹妙。
