序号：52592
名称：香附丸
组成：香附（童便浸，炒）1斤，针砂（醋煅）1两，厚朴（姜汁炒）5两，甘草（炒）1两，陈皮（去白，炒）3两，白芍（炒）5两，制苍术5两，山楂肉（炒黑）5两，茯苓（乳蒸，晒，焙）3两，青皮（炒）6两，苦参（炒，春夏）2两（秋冬）1两，白术（土炒）3两。
出处：《杂病源流犀烛》卷十六引《杨氏黄病方》。
主治：脱力劳伤黄病，及一切黄胖病。
加减：None
功效：None
用法用量：每服1钱，米饮送下；弱者7-8分，白术汤送下。服过7日，手心即凉，内有红晕起，调理半月即愈矣。
制备方法：醋糊为丸。
临床应用：None
各家论述：None
用药禁忌：忌食一切生冷、油腻、发硬之物。
附注：None
