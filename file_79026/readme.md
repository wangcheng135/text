序号：79042
名称：僵蚕散
组成：白附子1两，僵蚕1两（炒），半夏1两（制），南星1两（制），天麻1两（煨），蝉衣1两。
出处：《医略六书》卷二十八。
主治：孕妇中风，痰涌口噤，脉滑者。
加减：None
功效：None
用法用量：每服5钱，水煎，去滓，加生姜汁1匙，温服。
制备方法：上为散。
临床应用：None
各家论述：白附子祛风开痹气，明天麻胜湿祛风邪；僵蚕疏风化痰，专行经络，蝉衣善脱衣肤，宣通元府；南星散风痰以快胸膈，半夏燥湿痰以醒脾胃也。为散水煎，加生姜汁以散豁痰涎，务使风邪外解，则痰涌自消，而经气清和，壅塞顿解，胸字无不廓然，何有口噤痰逆之患，胎孕无不自安矣。
用药禁忌：None
附注：None
