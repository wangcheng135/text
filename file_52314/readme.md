序号：52329
名称：顺经散
组成：韭子（汤浸，退取白仁，干称）1两，益智子（取仁，盐炒过）半两，琥珀半两（令研），石苇（去毛土）1钱，白茯苓3分，狗脊（爁去毛，净）半两，石燕子（火煅，醋炒，出火毒，令研极细）半两。
出处：《洪氏集验方》卷五。
主治：小儿10余岁，因惊之后，心气不行，小便淋沥，日及30余次，渐觉黄瘦。
加减：None
功效：None
用法用量：每服1钱，用韭菜白煎汤调下，空心、食前各1服，日午1服尤妙。
制备方法：上为末，和匀。
临床应用：惊悸：予表姪10余岁时，尝游慧山，归已昏暮，遇一巨人醉卧寺门，惊悸得疾。自是之后，1日便溺50-60度，医治数月不能效，即授此方，服药未几，日减1日，初则20-30度，最后10数度，凡服2料而愈。
各家论述：None
用药禁忌：None
附注：None
