序号：13370
名称：白花蛇丸
组成：白花蛇（酒浸，去皮骨，炙）半两，羌活（去芦头）半两，白附子（炮）1分，麻黄（去节，煎，掠去沫，焙）半两，桂（去粗皮）半两，芎藭半两，干蝎（去土，酒炒）1分，防己半两，附子（炮裂，去皮脐）半两，干姜（炮）1分，蜀椒（去目并闭口，炒出汗）1分，乌头（炮裂，去皮脐）1分。
出处：《圣济总录》卷五。
主治：肾中风，腰膝骨髓疼痛，转动不得，及一切风病。
加减：None
功效：None
用法用量：每服10-15丸，空心温酒送下。如要出汗，先浴后服药，热酒或葱酒送下。汗出避风。
制备方法：上为末。炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
