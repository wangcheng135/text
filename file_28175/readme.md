序号：28183
名称：苁蓉木瓜煎丸
组成：肉苁蓉1斤（以酒浸，净刮去皴皮及沙石尽，细切，焙干，捣罗为末，称），牛膝8两（去苗，酒浸，锉，焙干，捣罗为末，称），菟丝子（尝甜滑者，以水掏去浮者）4两半（酒浸3-5日，以软烂为度，沙盆内研如泥），木瓜4枚（如无花木瓜，只用小黄熟木瓜10枚，并削去皮子，别以酒煮烂为度，入沙盆研如泥，又用法酒1斗5升匀调，同煎四味，入银器内重汤慢火熬成膏，不住手搅，勿令焦，仍相度后药末多少，或硬，更入炼熟好蜜，亦须带软，搜所贵滋润易为丸也），附子6枚（炮裂，去皮脐，锉，以青盐末3两拌和匀，炒令黄色同用），麋角（锉末）5两（用酥拌和，炒黄色），椒红4两，肉豆蔻仁2两，补骨脂3两（拣净，称，炒香），楮实（红实熟成者）3两半（淘去浮者，焙干，称），巴戟天（去心，炒黄）3两，木香2两，鹿茸（去毛，酥炙3两），桂（去粗皮）3两，蛇床子（拣择净）2两，槟榔2两（锉），干姜（炮裂）3两。
出处：《圣济总录》卷一八七。
主治：肝肾气虚，耳目不聪明，及一切冷气，腹胁疼痛。
加减：None
功效：延年轻身，爽神益气，补壮筋骨。
用法用量：每服30丸，渐加至50丸，空心温酒送下，盐汤亦得，晚食前再服。
制备方法：上十三味为细末，将前膏和成剂，再入臼内捣2000-3000下，入真酥少许为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
