序号：42614
名称：举肛丸
组成：半夏5钱，天南星5钱，枯白矾5钱，枯红矾5两，鸡冠花（炒）5两，白附子5两，诃子肉（煅）1两，黑附子（生）1两，枳壳1两，猬皮2枚（炙），瓜蒌1枚（烧存性），胡桃仁15枚（烧存性）。
出处：《医方考》卷三。
主治：泄泻虚寒脱肛者。
加减：None
功效：None
用法用量：每服30丸，空心温酒下。
制备方法：上为末，醋糊为丸。
临床应用：None
各家论述：湿盛则濡泻，久泻则胃虚，胃虚则脏寒，脏寒则无阳以升举，故令肛肠脱而不止。燥能去湿，故用半夏、南星；枯能制湿，故用红、白枯矾；温能暖脏，故用黑、白附子。乃若鸡冠花、刺猬皮、枳壳所以驱风；而诃子、瓜蒌、胡桃仁之灰，取其涩以固脱也。
用药禁忌：None
附注：None
