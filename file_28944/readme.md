序号：28952
名称：苏合香丸
组成：苏合香（另研，白色者佳）2两，安息香（无灰酒熬，飞去砂土）2两，熏陆香（另研）1两，龙脑（另研）1两，丁香1两，麝香（别研，勿经火）1两，青木香、白术、沉香（另研极细）、香附（炒）、乌犀角（镑屑，另研极细）。
出处：《张氏医通》卷十三。
主治：传尸殗殜，心腹卒痛，僵仆不省，一切气闭属寒证。一切恶毒之气中人，关窍不通者。
加减：None
功效：通关辟邪解毒。
用法用量：每服1丸，临用剖开，井花水、生姜汤、温酒化下。
制备方法：上为末，逐一配匀，炼蜜为丸，分作50丸，另以朱砂1两水飞为衣，蜡护。
临床应用：None
各家论述：《古方选注》：苏合香能通十二经络、三百六十五窍，故君之以名其方，与安息香相须，能内通脏腑。龙脑辛散轻浮，走窜经络，与麝香相须，能内入骨髓。犀角入心，沉香入肾，木香入脾，香附入肝，熏陆香入肺，复以丁香入胃者，以胃亦为一脏也。用白术健脾者，欲令诸香留顿于脾，使脾转输于各脏也。诸脏皆用辛香阳药以通之，独心经用朱砂寒以通之者，以心为火脏，不受辛热散气之品，当反佐之，以治其寒阻关窍，乃寒因寒用也。
用药禁忌：None
附注：苏合丸（《伤科补要》卷三）。
