序号：62827
名称：清宁丸
组成：大黄20斤，牡丹皮8两，地骨皮8两，泽泻8两，薄荷8两，赤茯苓8两，川石斛8两，黄柏8两，侧柏8两，玄参8两，连翘8两，木通8两，当归8两，知母8两，车前子8两，猪苓8两，陈皮8两，生地8两，川萆薢8两，薏苡仁8两，青盐8两，韭菜8两，鲜藕1斤，甘蔗1斤。
出处：《全国中药成药处方集》（上海方）。
主治：目赤便秘，小溲热赤。
加减：None
功效：None
用法用量：每服1钱半至3钱，温开水送服。
制备方法：先将大黄用米泔水浸透，切开晒干，用鲜藕、甘蔗打汁拌蒸，再将余药煎汁拌蒸，至黑为度，然后晒干研细粉，用黄酒和水为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：禁忌刺激性食物。
附注：None
