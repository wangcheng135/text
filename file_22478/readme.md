序号：22486
名称：沃雪汤
组成：生山药1两半，牛蒡子4钱（炒，捣），柿霜饼6钱（冲服）。
出处：《衷中参西》上册。
主治：脾肺阴分亏损，饮食懒进，虚热劳嗽；及一切阴虚之证，兼肾不纳气作喘者。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：喘证：一人，年四十余，素有喘证，薄受外感即发。医者投以小青龙汤，一剂即愈，习以为常。一日喘证复发，连服小青龙汤三剂不愈，其脉五至余，右寸浮大，重按即无。知其前服小青龙汤即愈者，因其证原受外感；今服之而不愈者，因此次发喘原看无外感也。为拟此汤，服两剂全愈，又数服以善其后。
