序号：46468
名称：济阴至宝丹
组成：当归（酒洗）1钱，白芍（酒洗）8分，白茯苓（去皮）8分，白术（去芦）1钱，陈皮8分，知母8分（生用），贝母8分（去心），香附（便制）8分，柴胡（酒炒）3分，薄荷3分，地骨皮（去皮）8分，甘草3分，麦门冬（去心）8分。
出处：《古今医鉴》卷十一。
主治：妇人诸虚百损，五劳七伤，经脉不调，肢体羸瘦。
加减：None
功效：调经水，滋血脉，补虚劳，扶元气，健脾胃，养心肺，润咽喉，清头目，定心悸，安神魂，退潮热，除骨蒸，止喘嗽，化痰涎，收盗汗，止泄泻，开郁气，利胸膈，疗腹痛，解烦渴，散寒热，祛体疼。
用法用量：用煨生姜3片，水煎，温服。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
