序号：3289
名称：九龙定风针
组成：生草乌3钱，生川乌3钱，北细辛3钱，闹羊花3钱，漂苍术3钱5分，瑶桂心4钱，白芷2钱5分，艾绒5钱，麝香8分。
出处：《经验奇方》卷上。
主治：跌打损伤，或手足肩腰疯痛，年久不愈，酸痛隐在骨节筋间，非膏药煎剂之力所能到者。
加减：None
功效：None
用法用量：上药各为细末，和匀，再研极细。姜汁炼作条。如不好做，以白及研细末，滚姜汁搅作糊，捣药搓条，每条长寸许，如细笔杆粗，晒干，储锡瓶，封口，勿今泄气受蒸。临医时，先用竹筷头点痠痛之处，墨圈记之，令病人切勿稍动，以免墨圈移位，再以竹纸，春8、夏12、秋10，冬6层，纸放墨圈患上，将药条向灯火燃着，一头用力挂纸上。看准隔墨圈患处，1次呼吸之间火灭。纸面层见焦，余层如故。少顷针处起泡，用铁针挑破，火炮药搽之。无论患穴多少，凡竹筷头点之，觉甚痠痛者，均宜针之。1-2旬后，患泡两愈，永不复发。
制备方法：None
临床应用：None
