序号：51402
名称：贺兰先生解毒丸
组成：贯仲3两，茯苓3两，黄药子3两，篮根3两，干葛3两，地黄3两，雄大豆3两，甘草3两，滑石3两，缩砂仁3两，阴地厥3两，薄荷3两（好者用），土马鬃1两，绿豆粉1两，益智1两，寒水石1两，山豆根1两，紫河车1两，马譬孛1两，草龙胆1两，白僵蚕1两，百药煎1两，山栀子1两，大黄1两。
出处：《御药院方》卷七。
主治：药毒，酒毒，山岚瘴毒，果毒，肉毒，面食鱼菜痰，冬月丹毒，夏月暑毒，伤风后余热，小儿疮疹后毒，及喉闭之患，小儿急惊。
加减：None
功效：善解诸毒。
用法用量：每服1丸，细嚼，新水送下；小儿1丸分作4服，煎薄荷汤放冷磨下；小儿急惊，磨刀水下。此药长宜将带备急，若夏月频服，使诸疾不生。
制备方法：上为细末，蜜水浸蒸饼为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：保命丹、化毒丹。
