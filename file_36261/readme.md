序号：36271
名称：取癖丹
组成：定粉1两，舶硫黄1两，密陀僧（煅，醋淬7次）1两，木香半两，雷丸（不用红者）半两，黑牵牛（头未，半生半炒）半两，轻粉半钱，使君子3钱，大黄4两（醋煮黑，焙干）。
出处：《医方类聚》卷二五五引《新效方》。
主治：小儿疳癖，时发寒热，虚汗焦渴，面色黄瘦，肚大青筋，头面四肢浮肿，生疮口臭，牙疮鼻衄。
加减：None
功效：None
用法用量：1岁儿服1钱，3岁儿2钱，临卧米饮调下。或隔日汤泡炊饼为丸，如粟米大，米饮送下。天明取下恶物为验，以粥补之。如下恶物未尽，病未全除者，7日后依前再服，病重者不过3次服。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：忌荤腥1日，忌牛、马、驴、兔等肉并血100日。
附注：None
