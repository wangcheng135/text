序号：57300
名称：消痞狗皮膏
组成：三棱1两5钱，蓬荗1两5钱，米仁1两5钱，山栀1两5钱，秦艽1两5钱，黄连4钱，大黄9钱，当归9钱，甲片40片，全蝎40只，木鳖20个，巴豆10粒。
出处：《饲鹤亭集方》。
主治：一切气疾痞块，癥瘕血块，积聚腹胀疼痛。
加减：None
功效：None
用法用量：用时将膏在热茶壶上烘至暖烊，贴患处，以手心揉百转，无不效验。贴后能作寒热、肚痛下秽，其疾消，愈矣。
制备方法：上用麻油120斤，煎枯去滓后，下黄丹52两收膏，加入阿魏、阿胶、芦荟各1钱，麝香、乳香、没药各3钱，研末调和膏内。
临床应用：None
各家论述：None
用药禁忌：百日内禁忌酒色气恼，劳心劳力，诸般发物。
附注：None
