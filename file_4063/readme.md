序号：4063
名称：木瓜丸
组成：木瓜30个（大者，去皮瓤了，切，蒸烂为度，入盐花1斤，熟蜜1斤，更煎令稠，用和药末），沉香1两，阿魏3分，木香2两，肉豆蔻1两（去皮），红豆蔻1两，桂心2两，甘草1两（炙微赤，锉），缩砂2两（去皮），陈橘皮1两（汤浸，去白瓤，焙），胡椒1两，白术2两，芎2两，厚朴2两（去粗皮，涂生姜汁炙令香熟），附子2两（炮裂，去皮脐），神曲2两（微炒），桃仁3两（汤浸，去皮尖双仁，麸炒微黄），茴香子1两，藿香1两，荜茇1两，当归1两（锉，微炒），诃黎勒2两（煨，用皮），高良姜1两（锉），丁香1两，干姜2两（炮裂，锉），白豆蔻1两（去皮）。
出处：《圣惠》卷九十八。
主治：一切冷气，心腹胀痛，食不消化。霍乱。
加减：None
功效：None
用法用量：每服20丸，以生姜汤嚼下；温酒下亦得。
制备方法：上为末，以木瓜煎为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
