序号：81987
名称：镇心丸
组成：秦艽3分，柏实3分，当归3分，干漆（熬）3分，白蔹3分，杏仁（去皮尖双仁，熬）3分，芎3分，泽泻1两，干地黄6分，防风4分，人参4分，甘草1两（炙），白术2分，薯蓣2分，茯苓2分，干姜2分，麦门冬（去心）2两，前胡4分。
出处：《千金翼》卷十六。
主治：胃气厥实，风邪入脏，喜怒愁忧，心意不定，恍惚喜忘，夜不得寐，诸邪气病。
加减：None
功效：None
用法用量：每服10丸，食前以饮送服，日3次，不知，稍增之。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌海藻、菘菜、芜荑、桃李、雀肉、酢物等。
附注：None
