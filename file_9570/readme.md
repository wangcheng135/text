序号：9570
名称：仙方夺命丹
组成：白茯苓1两，密陀僧1两（炒黄色），紫河车1两，白僵蚕1两（直者），贯众1两（净），缩砂仁1两，甘草节1两，乌鱼骨（去皮）2钱半，麝香1字。
出处：《普济方》卷六十引《德生堂方》。
主治：缠喉风，木舌胀，双单乳蛾，喉闭，或误吞鸡鱼骨刺，竹木签刺，一切咽喉急证。
加减：None
功效：None
用法用量：每服1丸，用瓦器研碎，新水半盏，浸1茶时化开，用匙挑药，徐滴入喉中。勿急用，取药尽为度。
制备方法：上为细末，却入麝香研匀，同蒸饼，包白面蒸熟4两，和药，汲新井水为丸，如豌豆大，蛤粉为衣，放干10年不坏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
