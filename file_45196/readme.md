序号：45211
名称：枳壳散
组成：枳壳（去瓤，麸炒）3分，黄耆（炙，锉）3分，苍耳（炒）3分，白术（微炒）3分，蒴藋（锉）3分，白蒺藜（炒）3分，防风（去芦，生用）3分，升麻（生用）3分，栀子仁3分，漏芦（去芦头）半两，黄连（去须）半两，人参半两，天门冬（去心，焙）半两，防己半两，干姜（炮）半两，芎半两，丹参（去苗土，炙）半两，麻黄（去根节，煎，掠去沫，焙）半两，白芷（微炒）半两，甘草（炙，锉）半两，葛根（锉）半两，玄参（坚者）3两，乌蛇（酒浸，去皮骨，炙）1两。
出处：《普济方》卷一一一引《圣惠》。
主治：大风癞病，热毒风入五脏，眉发坠落，身体头面生疮。
加减：None
功效：None
用法用量：每服2钱匕，加至3钱匕，空心、夜卧时以温酒调下。
制备方法：上为散。
临床应用：None
