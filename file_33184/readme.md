序号：33193
名称：万灵汤
组成：前胡半斤（去芦头），柴胡半斤（去苗），秦艽半斤（去苗土），甘草（炙）半斤，茴香子1斤（炒），木香1斤，桂（去粗皮）1斤，槟榔10枚，肉豆蔻（去壳）半斤，芍药半斤，青橘皮（去白）半斤，芎半斤，甜葶苈半斤（微炒），桔梗4两。
出处：《圣济总录》卷二十二。
主治：虚寒痼冷，痰癖动气，心膈疼痛，噎闷呕逆，一切气疾。
加减：None
功效：和养三焦，调顺阴阳，升降痞滞，祛遣寒邪，温中散湿，暖胃和脾，滋助气血，思美饮食。
用法用量：每服3钱匕，水1盏，加大枣2枚（擘破），同煎至7分，去滓，食前温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
