序号：66684
名称：鹿茸丸
组成：禹余粮石（煅，米醋淬7次，别研）2两，熟干地黄（洗，炒）2两，当归（去芦，炒）2两，白艾叶（洗，醋浸炒）半两，卷柏叶（醋浸炒）半两，麒麟蝎（别研）半两，没药半两（别研），赤石脂（煅，别研）1两，附子（炮，去皮脐）1两，续断3两（酒煮）。
出处：《魏氏家藏方》卷十。
主治：经候过多，其色瘀黑，甚则崩下，吸吸少力，脐腹如冰，冷汗如雨，冲任虚损，风冷之气客于胞中，气不禁固。
加减：None
功效：None
用法用量：每服30-40丸，空心、食前用温酒或淡米醋汤送下。
制备方法：上为细末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方名“鹿茸丸”，但方中无鹿茸，疑脱。
