序号：74158
名称：催生膏
组成：大龟1个（要板黑者为佳，黄色者不佳，约2-3斤，愈大愈妙。用小磨麻油浸数日，熬枯去滓，再将油炼老，下炒黄丹收，加炒铅粉4两，搅匀）。
出处：《理瀹》。
主治：难产，数日不安，及交骨不开者。
加减：None
功效：安神息力以催生。如生产尚早，亦能安胎。
用法用量：每用1两，以姜葱汁、陈酒、醋调敷；胞干，用炼猪油调；夏天天热，用麻油、白蜜、鸡清调敷。如下死胎，加附子、官桂。睡醒自生。
制备方法：临用以黏3钱摊皮纸上，令产妇平身安睡，贴膏脐上，外加敷药：车前子2两，川芎、全当归各1两，冬葵子7钱，枳壳、白芷、半夏、白蔹各4钱，共研末，入榆面3两，益元散2两，和匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
