序号：51513
名称：追风逐湿膏
组成：豨签草3两，麻黄3两，川乌3两，草乌3两，风藤3两，半夏3两，南星3两，羌活3两，蓖麻子（打碎）3两，桂枝3两，独活2两，细辛2两，当归2两，白芷2两，苍术2两，大黄2两。
出处：《外科正宗》卷三。
主治：风寒暑湿相伤，以致骨节疼痛，筋挛不能步履，或麻木湿痹。
加减：None
功效：None
用法用量：渐用热汤顿化，绫缎摊贴。7日后，诸病可痊，百发百中。
制备方法：上药各切咀片，用葱汁、生姜汁各2碗拌药，先浸1宿，次日用香油半斤，同药入锅内，慢火煎至葱、姜汁将干不爆时，油方与药相煎，滓枯为度；细绢滤清，每油1斤，下飞过炒丹10两为准配用；再将前油入锅内煎滚，以油滴水成珠不散方下黄丹，徐徐搅入，其膏已成；再下松香净末1斤4两，再同熬化，取下锅来，以盆顿稳，再下乳香、木香、胡椒、轻粉各末2两，白芥子细末4两，渐入搅匀，倾入钵内盛贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
