序号：48384
名称：神效灰煎
组成：炭灰3升。
出处：《外台》卷二十九引《广济方》。
主治：疣赘，赤黑疵痣靥，秽疮疽，息肉结瘤。
加减：None
功效：None
用法用量：汤拌令湿彻，以热汤渍，令半日后，还以汤淋之（稍稍点汤，不得太速下，即灰汁不验），候汁下得2-3升。即纳一小铛中煎，令1-2沸，即别取1两石灰（风化者为佳，恐中湿者，须熬令极热）纳灰汁中和煎，以杖算搅之勿住手，候如煎饼面，少许细细取成膏，急泻著一瓷器中，搅令冷，不然，须臾干燥不堪用。常候此煎十分有一分堪久停。但有伤损，肉色须臾变赤黑色，痛如火烧状，若灸瘢发焮，经20余日，病自然脱落，无瘢痕。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：疮未愈间，忌小豆、姜，纵有瘢，亦不凸出。
附注：None
