序号：59740
名称：调肝散
组成：犀角屑1分，草龙胆半分，黄耆半两（炙锉），大黄1分（炒过），桑白皮1分（炙锉），钩藤钩子1分，麻黄1分（去根节），石膏（别研）半两，栝楼实半两（去瓤皮），甘草1分（炙）。
出处：《斑疹备急》。
主治：小儿疹痘疮及赤疮子，风胜气实，心肝血热，津液内燥，大便不通，毒气上盛，表热未散者。
加减：None
功效：败肝脏邪热，解散斑疹余毒，令疮疹不入眼目。
用法用量：每服2钱，水1盏，煎至5分，去滓温服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：《奇效良方》：方中栝楼、大黄治内燥；钩藤、龙胆治风血热；桑白皮、石膏治上焦热；麻黄、黄耆散肌热；犀角、甘草解毒热。
用药禁忌：None
附注：犀角汤（《奇效良方》卷六十五）。
