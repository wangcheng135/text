序号：60169
名称：透骨金丝万应膏
组成：牛膝1两（研），木鳖子1两（研），当归半两（锉），金刚骨9钱（切），自然铜半两，蓖麻子2两（切破），川乌头2枚（切破），紫花地丁半两（切），白龙骨半两。
出处：《普济方》卷三一三。
主治：疮肿。
加减：None
功效：None
用法用量：随疮大小，纸上摊贴。
制备方法：为细末，用小油1斤浸1宿，漫火煎，后入桃枝、槐枝同熬药焦，绵滤去滓，入黄丹6两，熬2沸，次入白胶半两，乳香1钱（研），白丁香1钱，雄黄1钱，没药1钱，密陀僧1钱，上为细末，同熬匀，再用绵滤过，看硬软取之收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
