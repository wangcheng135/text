序号：574
名称：五炁朝元紫霞丹
组成：南铅2两，北铅2两，雌黄2两，雄黄2两，倭硫黄5钱。
出处：《外科十三方考》引《红蓼山馆经效方》。
主治：一切诸风痰疾，反胃、哮嗽、齁喘老痰、瘘管，诸痿痹，虫积、阴毒，小儿急、慢惊风。
加减：冷齁，加桂心、附片、白蔻各1两；湿齁，加白茯苓、白术各1两；气齁，加广木香、沉香、家苏子各5钱；水齁，加芫花(姜汁和醋炒黑)3分，甜葶苈子1两，苡仁2两；痰齁，加法半夏、尖贝母各2两，橘红1两；食齁，加炒神曲、小枳实各5钱；火齁，加石膏、生桑皮、马兜铃各5钱；虫齁，加百部、榧子、槟榔各6钱；虚齁，加阿胶珠、北五味、沉香各3钱；盐齁，加苍术、猪苓、甘草各3钱。
功效：None
用法用量：大人每服3分，小儿半分。
制备方法：先将雌、雄、硫三味研细，再入南北二铅熬化，候冷，打成2盏，入前药在内，上覆1盏，入阳城罐内，石膏、盐泥封固，上仰一铁盏，入八方炉中，先文后武火升之，盏内添水勿令干，候线香五炷，其药即升于盏上，候冷，绢埋3日，取出研细，用大红枣蒸熟，去皮核，捣如泥，与药等分为丸，如粟米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
