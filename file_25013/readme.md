序号：25021
名称：如圣散
组成：狐心肝3副，乌鸦2只（去嘴爪翅，并腊月者，以上两味用大新藏瓶1枚，斡开口子纳之，瓦盖合留1眼子，外以纸筋泥固济，泥干以火烧令赤，候烟细无香气，去火盖定眼子，候冷取出，以绢帛裹地坑内碗，复出火毒1复时，取出研细），丹砂2钱（研），龙脑2钱（研），麝香（研）2钱，牛黄（研）半钱，犀角2字（镑），琥珀2字，真珠（麻子大者）5颗（小者10颗）（细研），金箔10片，银箔10片。
出处：《圣济总录》卷六。
主治：急风。
加减：None
功效：None
用法用量：每服1钱匕，用薄荷自然汁入酒少许调下，空心、日午、临卧各1服。
制备方法：上以丹砂以下诸药为极细末，同前药研匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
