序号：83859
名称：乌银丸
组成：巨胜子（炒）2两（别研），牛膝（去芦头）1两，覆盆子1两，荜澄茄（去枝）1两，肉桂（去粗皮）1两，白茯苓（去皮）1两，吴白芷1两，甘菊花（拣净）1两，远志（去心苗）1两，熟干地黄（焙）1两，旋覆花1两，旱莲草（去枝）1两。
出处：《御药院方》卷六。
主治：男子筋痿，少腹不利，小便频数，腰背疼闷，不能久立，久立则腿膝麻冷，难以屈伸，心意多忘，耳内蝉呜。
加减：None
功效：久服滋血气，壮元阳，髭发返黑，令人不老，添精补髓，益寿延年。
用法用量：每服50-60丸，食前温酒送下。服至7日，觉阳气坚壮是验。
制备方法：上除巨胜子外，同为细末，再与巨胜子拌匀，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
