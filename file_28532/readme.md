序号：28540
名称：苏荷生地汤
组成：紫苏1两，南薄荷1两，青蒿1两，生地8钱，条参8钱，连翘8钱，槐花7钱，柴胡6钱，川芎2钱，生黄耆5钱。
出处：《验方新编》卷十五。
主治：中蛇蛊、中疳蛊、中肿蛊、中癫蛊。
加减：大便结秘者，重加槐花、黄柏、黄芩、茨姑菜等味；小便赤者，加玄参、栀子、茯苓；肝火盛者，加生白芍；头痛，加白芷，蛇蛊，加白芷1两，三七2钱；带热咳嗽或咳血者，紫苏、薄荷减半，加百合1两、麦冬1两、生芍1两；或时而伤风带寒嗽者，去条参、连翘、槐花、青蒿、生地，加干姜、当归、半夏、陈皮、白芷；或人弱带痢者，去条参、连翘、槐花、青蒿、柴胡、生耆、生地，加百合、白芍、茯苓、麦冬、砂仁、白术、干姜。或兼有别病，则兼治之，而紫苏、薄荷则断不可去也。又口舌热烂，实火炎上，则去紫苏、薄荷，重加槐花、黄柏、黄芩、茯苓、白芍、玄参、泽泻、天冬、石膏等味。
功效：None
用法用量：酒、水煨服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：苏荷生地黄汤（《中国医这大辞典》）。
