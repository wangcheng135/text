序号：47828
名称：神仙眼药
组成：秦皮3钱（去粗皮，锉细），乳香1块（如枣大），胡黄连3钱，灯心1握（7寸长），枣子3个，斑蝥1个（去翅头足），古老钱7文（不锉）。
出处：《中藏经·附录》卷八。
主治：眼疾。
加减：None
功效：None
用法用量：入无釉器中（砂器尤佳），用井华水1大碗，熬去半碗，用绵绢挤过，再将滓以水半瓷碗，煎取1盏，入挤过汁同煎，汁入新碗中，熬似稠粥样，入小瓷盒中或角盒中盛，将空青并鹏砂1块（如2豆大，飞过，熬干。空青不熬）再研，入脑子（多不妨）、麝香少许，4味同入药膏内，搅匀。每点1粟米许在眼眦头，将手挪匀，仰面，候药微涩过，将沸盐汤用软帛片蘸洗。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
