序号：61583
名称：常山太守马灌酒
组成：天雄2两（生用），蜀椒1两，商陆根1两，乌头1枚（大者），桂心1两，白蔹1两，茵芋1两，干姜1两，附子5枚，踯躅1两。
出处：《千金》卷八。
主治：病在腰膝者。
加减：None
功效：除风气，通血脉，益精华，定六腑，明耳目，悦泽颜色，头白更黑，齿落更生。
用法用量：初服半合，稍加至2-3合；捣滓为散，酒服方寸匕，日3次，以知为度。夏日恐酒酸，以油单覆之，下井中近水，令不酸也。
制备方法：上（口父）咀，以绢袋盛，酒3斗渍，春、夏5日，秋、冬7日，去滓。
临床应用：None
各家论述：None
用药禁忌：None
附注：服药二十日力势倍，六十日志气充盈，八十日能夜书，百日致神明，房中强壮，如三十时，力能引弩，年八十人服之，亦当有子。
