序号：63193
名称：大慎火草散
组成：慎火草2两，白石脂2两，鳖甲（炙）2两，黄连2两，细辛2两，石斛2两，芎2两，干姜2两，芍药2两，当归2两，熟艾2两，牡蛎（熬）2两，禹余粮2两，桂心1两，蔷薇根皮4两，干地黄4两。
出处：《千金翼》卷八。
主治：妇人崩中漏下，赤白青黑，腐臭不可近，面黑无颜色，皮骨相连，月经失度，来无常，小腹弦急，或苦绞痛，上至于心，两胁肿胀，令人倚坐，气息乏少，食不生肌肤，腰背疼痛，痛连两脚，不能久立，但欲得卧。
加减：若寒多，加附子及椒（用椒当汗，去目及闭口者）；热多，加知母、黄芩，加石斛两倍；白多，加干姜、白石脂；赤多（1方云青黑），加桂心，代赭石各2两。
功效：None
用法用量：每服方寸匕，空腹酒调下，1日3次，稍增至2匕。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
