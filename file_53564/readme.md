序号：53579
名称：圆明膏
组成：柴胡5钱，麻黄（微捣）5钱（去节），当归身3钱，生地黄半两，黄连5钱，甘草2钱，诃子皮2钱（湿纸裹，煨）。
出处：《东垣试效方》卷五。
主治：劳心过度、饮食失节而致内障生翳，及瞳子散大。
加减：None
功效：None
用法用量：如常点之。
制备方法：先以水2碗，熬麻黄至1碗，掠去沫，外6味各(口父)咀，如豆大，筛去末，称毕，入在内同熬，滴入水中不散，入去沫，蜜少许再熬动。
临床应用：None
各家论述：①《医方集解》：此足少阳、厥阴药也。柴胡、麻黄发散表邪，当归、生地和肝养血，黄连清肝火，甘草和中州，瞳子散大，故加诃子以收之也。②《医林纂要》：障翳自内，火妄血枯，内枯而瞳散，外浊而翳生。故君生地而协以归、连、甘草，以治内也；翳成于外矣，散以柴胡、麻黄，自内而拔之于外；目阴欲尽，诃子皮以敛之。
用药禁忌：None
附注：None
