名称：小莽草散
组成：莽草2两（微炙），麻黄2两（去根节），天麻2两，萆薢1两（锉），防风1两（去芦头），芎1两，羌活1两，柏子仁1两，白术1两，细辛1两，松脂1两（炼过者），牛膝1两（去苗），山茱萸1两，泽泻1两，赤芍药1两，枳壳1两（麸炒微黄，去瓤），附子1两（炮裂，去皮脐），白附子1两（炮裂），天南星1两（炮裂），干蝎1两（微炒），乌蛇肉1两（酒浸，炙微黄），当归1两，石龙芮1两，犀角屑1两，杜仲1两（去皱皮，炙微黄），白僵蚕（微炒）3分，半夏半两（汤洗7遍去滑），铅霜3分（细研），牛黄半两（细研），麝香半两（细研）。
