序号：42972
名称：保婴五疳膏
组成：青皮（麸炒）2钱，橘红5钱，白术（去芦，蜜水炒）1两半，白茯苓7钱半，麦门冬（去心）1两，使君子肉（锉，炒）7钱5分，山楂肉5钱，麦芽（炒）5钱，金樱子5钱，金樱子肉（炒）5钱，芡实仁2钱半，莲心肉（隔纸炒）5钱，甘草2钱。
出处：《寿世保元》卷八。
主治：小儿五疳潮热，面黄肌瘦，烦渴，肚大青筋，手足如柴，精神困倦。
加减：身热咳嗽，加地骨皮、百部；肚腹饱胀，大便为稀水，腹鸣作声，或因虫出不知，加槟榔2钱，木香1钱；禀受气弱，加人参2钱半。
功效：无疾预服此药，则诸病不生。元气虚者，服半月身体健壮。
用法用量：每日中晌、晚间各服1-2茶匙，温水漱口。
制备方法：上为末，和匀，重7两，每次用药末1两，炼蜜4两，调和成膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
