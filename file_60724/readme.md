序号：60739
名称：通草汤
组成：通草7分，瞿麦1钱，柴胡1钱，天花粉1钱，桔梗2钱，木通5分，青皮5分，香白芷5分，赤芍5分，连翘5分，甘草5分。
出处：《古今医鉴》卷十二。
主治：乳汁不通属实证者。年少初产，乳虽胀，汁不流，有风热。产后血气盛实。
加减：None
功效：None
用法用量：水煎细饮，更摩乳房。
制备方法：上锉1剂。
临床应用：None
各家论述：《医略六书》：产后风热外遏、血气内壅而乳房肿胀、乳窍不通，故乳汁不出焉。瞿麦泻热以通气闭，赤芍破血以行血滞，连翘清利热结，通草通利阳明，花粉清热润燥，青皮破滞泻气，白芷解散阳明之经，桔梗开提气血之滞，柴胡疏乳房之腠理，甘草和厥阴之胃气也。为散水煎，使风热外解，则气行血活而经气亦清、乳窍通利，何乳房肿胀不退，乳汁乃有不出者乎？
用药禁忌：None
附注：通草散（《济阴纲目》卷十四）、立效散（《杂病源流犀烛》卷二十七）。
