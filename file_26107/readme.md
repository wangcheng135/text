序号：26115
名称：百花膏
组成：透骨草、忍冬藤、蒲公英、鹤虱草、九龙藤、野天麻、旱莲草、半枝莲、地杨梅、豨莶草、苍耳草、紫地丁、地锦草、旱辣蓼、大小青、薄荷叶、灵芝草、鱼腥草、见肿消、血见愁、淡竹叶、南天竹、枸杞、头橘树头、枳（木具）叶、五加叶、接骨木、石楠头、地蜈蚣、篇蓄草、马齿苋、野芥菜、蛇床叶、长青草、慎火草、太湖葱各等分。
出处：《解围元薮》卷四。
主治：疠风。
加减：None
功效：None
用法用量：每服1匙，酒下，日3次。
制备方法：捣汁，煎加蜜，炼成膏；再加沉香、檀香、冰片、麝香各等分为末入内，收贮于瓷瓶，勿泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
