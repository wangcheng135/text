序号：36577
名称：三圣散
组成：当归（洗，焙）、肉桂（去皮）、玄胡索（灰炒）各等分。
出处：《百一》卷三。
主治：中风瘫痪，腰痛，产后瘀血腹痛。中风手足拘挛，口眼斜，左瘫右痪，骨节痠疼，脚弱无力，行步不正。男妇腰痛。产后恶血凝滞，脐下作痛，或作寒热。闪肭血滞，腹中痛，产后服之更妙。
加减：None
功效：None
用法用量：每服2钱，温酒调下，空心，临卧日进3服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：舒筋散（原书同卷）、神应散（《普济方》卷一五四引《家藏经验方》）、如神汤（《妇人良方》卷四）、延胡散（《普济方》卷三五一）、延胡索散、《校注妇人良方》卷二十）、舒筋汤（《准绳·类方》卷四）、如神散（《治痘全书》卷十三）、舒筋三圣散（《张氏医通》卷十三）、元胡散（《仙拈集》卷二）。
