序号：43046
名称：养元辟谷丹
组成：黄犍牛肉15斤（去筋膜，切作棋子大片，用河水洗数遍，令血水净，再用河水浸1宿，次日再洗2-3遍，清水为度；用无灰好酒煮1夜，桑柴文武火，用砂罐煮，取出焙干，黄色者佳，黑焦不用。每牛肉末1斤，加入后药2斤），山药8两（用葱、盐炒山药至黄色，去葱、盐不用），莲肉8两（去心皮），白茯苓（去皮筋膜，为末，水飞过）8两，芡实肉（取粉）8两，白术8两（油者，不用炒），薏苡仁8两（炒），白扁豆8两（去壳，姜汁炒），人参（去芦）4两，小茴香（炒）4两，干姜（炒）2两，砂仁（炒）2两，青盐4两，川椒（去目，炒）2两，甘草（炙）4两，乌梅肉2两（熬浓汁半碗），粳米（洗净，炒黄）6斤。
出处：《古今医鉴》卷十六。
主治：None
加减：None
功效：安五脏，消百病，和脾胃，补虚损，固元气，实精髓，助脾健胃，瘦者令肥，老者健。
用法用量：每次2丸，不拘冷热茶汤嚼下，1日2-3次，永不饥。
制备方法：上为细末，与米粉、牛肉末和匀，用小红枣5斤，醇酒5斤，煮枣极烂，去皮核，捣膏，加炼蜜2斤半，共和为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
