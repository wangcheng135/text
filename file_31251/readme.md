序号：31260
名称：沉香降气散
组成：沉香1两，木香1两，丁香1两，藿香叶1两，人参（去芦头）1两，甘草（炮）1两，白术1两，白檀2两，肉豆蔻半两，缩砂仁半两，桂花半两，槟榔半两，陈橘皮（去白）半两，青皮（去白）半两，白豆蔻半两，白茯苓（去皮）半两，川姜（炮）2两，枳实（炒）2两。
出处：《御药院方》卷四。
主治：三焦痞滞，气不宣畅，心腹疼痛，呕吐痰沫，胁肋膨胀，噫气不通，哕逆醋臭，胃中虚冷，肠鸣绞痛，宿食不消除，反胃吐食不止，及五膈五噎，心胸满闷，全不思食。
加减：None
功效：None
用法用量：每服2钱，加盐少许，水1大盏，同煎至7分，和滓温服，每日3次，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《医方大成》引作“沉香降气汤”。
