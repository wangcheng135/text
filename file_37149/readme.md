序号：37159
名称：定岩散
组成：（豕叚）鼠粪（两头尖）3钱，土楝实3钱（经霜有核者佳，不用川楝），露蜂房3钱。
出处：《古方选注》卷下。
主治：乳癌溃烂经年，仅存内膜者。
加减：None
功效：止痛干脓，收敛合口。
用法用量：每服3钱，酒送下，间2日1服。
制备方法：上煅存性，各取净末3钱，和匀。
临床应用：None
各家论述：定，止也，溃岩服之，痛定而烂止也。（豕叚）鼠粪性主走阴。专入厥阴血分，通经下乳。楝实用土者，取其微苦力簿，走中焦乳间泄热，不似川楝力厚，直行下焦。露蜂房入阳明经，驱肝经风毒犯胃，有收敛之性，凡外疡之毒根在脏腑者，非此不愈。
用药禁忌：None
附注：托里散（《古方汇精》卷三）、定癌散（《医方歌括》）、乳岩散（《梅氏验方新编》卷二）。
