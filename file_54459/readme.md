序号：54474
名称：柴胡羚角汤
组成：鳖血柴胡2钱，归尾2钱，杜红花1钱，碧玉散3钱（包煎），羚角片3钱（先煎），桃仁9粒，小青皮1钱半，炒川甲1钱，吉林大参1钱，醋炒生锦纹3钱。
出处：《重订通俗伤寒论》卷二。
主治：妇人温病发热，经水适断，少阳内陷阳明厥阴，昼日明了，夜则谵语，甚至昏厥，舌干口臭，便闭尿短。
加减：None
功效：和解阴阳，大破血结。
用法用量：临服调入牛黄膏1钱。
制备方法：None
临床应用：None
各家论述：此方君以鳖血柴胡，入经达气，入络利血，提出少阳之陷邪；羚角解热清肝，起阴提神。臣以归尾、桃仁破其血结，青皮下其冲气。佐以川甲、碧玉散、炒生军直达瘀结之处，以攻其坚，引血室之结热，一从前阴而出，一从后阴而出。妙在人参大补元气，以协诸药而神其用。牛黄膏清醒神识，以专治谵语如狂。此为和解阴阳，大破血结，背城一战之要方。
用药禁忌：None
附注：None
