序号：72779
名称：紫石英丸
组成：紫石英（捶作小块，以葵菜叶煮半日，碾细，水飞）1两，鹿茸（切片，酒浸1宿，去毛，炙）1两，禹余粮（火煨，醋淬7遍，水飞）1两，当归（切，焙）1两，枳壳（去瓤，麸炒）1两，芎1两，侧柏（微炙）3分，艾（细锉，醋拌，炒黄）3分，阿胶（蛤粉炒黄，去粉）3分，赤芍药3分，桂（去粗皮）3分，白芷3分，乌贼鱼骨（去甲，微炙）半两，木香半两。
出处：《圣济总录》卷一五七。
主治：血气不足，子脏夹寒，妊娠数堕。
加减：None
功效：None
用法用量：每服30丸，空心、晚食前温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
