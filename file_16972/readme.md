序号：16979
名称：生血丹
组成：鹿角胶1两半，白茯苓（去皮）1两半，干山药1两半，柏子（别研）1两，牡丹皮1两，菟丝子（洗净，酒浸3日，研烂成饼）1两，枸杞子1两，五味子1两，人参（去芦）1两，牛膝（去芦，酒浸3日）1两，远志（去心）1两，当归（去芦，酒浸）1两1分，肉苁蓉（酒浸3日）1两1分，生干地黄（洗）4两，熟干地黄（洗）4两。
出处：《魏氏家藏方》卷二。
主治：血少气涩，肌肉不荣，脚膝无力，眼目多昏等疾。
加减：None
功效：None
用法用量：每服40-50丸，空心、食前温酒、盐汤任下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
