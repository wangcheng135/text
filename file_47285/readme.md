序号：47300
名称：砒霜丸
组成：砒霜半两，雄黄1两，丹砂（3味一处同研）1两，玳瑁（镑）1两，藜芦（炒）1两，椿叶（阴干者）1两，阿魏1两半（别研）。
出处：《圣济总录》卷三十七。
主治：山岚瘴疟，或入山早吸毒气，令人寒热如疟，终日不已，头痛痰逆，呕吐不下食，日渐羸瘦。
加减：None
功效：None
用法用量：每用3丸，男左女右，以绯绢袋盛，系于臂或衣带上。如疟不已，即用醋汤面东咽下3丸；如要吐，空心浆水下5丸。如吐甚，以冷水调绿豆末1-2钱匕，服即止。
制备方法：上为末，和匀，用安息香1两，酒浸，重汤煮为糊，为丸如赤小豆大。
临床应用：None
各家论述：None
用药禁忌：忌热物。
附注：None
