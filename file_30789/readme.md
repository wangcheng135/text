序号：30798
名称：灵光还睛膏
组成：川黄连4两（锉如大豆许，用童便浸1宿，滤去滓，晒干，为末），炉甘石6两（放在铁片上，炭火内烧红透，黄连汁淬之，烧淬7次，研为末），黄丹3两（研细，水飞净），当归2钱，乌鱼骨1钱，白丁香1钱，硇砂（另研）1钱，轻粉（研）1钱，麝香（另研）半钱，乳香（另研）半钱。
出处：《瑞竹堂方》卷三。
主治：一切眼疾。
加减：None
功效：None
用法用量：每用1丸，新吸水于小盏内化开，时时洗之。
制备方法：上为细末，用白沙蜜10两，银器内或沙锅内先熬5-7沸，以净纸搭去面上腊，取净，除黄丹外，下余药，用湿柳杆子搅匀，次下黄丹，再搅匀，于慢火内徐徐搅至紫色，不粘手为度，急丸如皂角子大，以纸裹之。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“灵光还睛丸”。
