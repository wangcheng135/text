序号：47806
名称：神仙紫金丸
组成：生地黄3两（大而圆者，以新铫子盛，用炭火1秤煅通赤，钳铫子出火，急倾入醋2升，候冷取，研如粉），针砂5两（真者，水淘净，控干，入生铁铫子同禹余粮3两1处，用酽醋2升煮，醋干，并铫子用炭火1秤煅赤，顿在净地上冷，研如粉），羌活半两，木香（炒）半两，白茯苓（去皮）半两，牛膝（去苗，酒浸1宿，焙干）半两，川芎半两，肉豆蔻（炮）半两，舶上茴香半两，蓬莪术（炮）半两，白术（炒）半两，桂（去皮）半两，干姜（炮）半两，青皮（去白，炒）半两，京三棱（炒）半两，陈皮（去白）半两，白蒺藜半两，附子（炮制）半两，当归（去芦须，浸，炒）半两。
出处：《产宝诸方》。
主治：十种水气，足膝浮肿，上气喘满，小便不利者。
加减：None
功效：逐阴固阳，扶危正命。
用法用量：每服30丸，空心、食前温酒或白汤送下。虚人壮人均用诸药，壮人减半，病自小便去后，每日1服，别以温和调气补脾胃血气药将理。
制备方法：上为细末，汤浸蒸饼为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌盐数月。
附注：None
