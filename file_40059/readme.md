序号：40069
名称：肾沥汤
组成：黄耆6两，芎6两，茯苓6两，五味子6两，防风6两，泽泻6两，独活6两，玄参6两，人参6两，牛膝6两，麦门冬（去心）8两，地骨皮8两，桂心，甘草3两（炙），丹参5两。
出处：《外台》卷三十一引《近效方》。
主治：None
加减：None
功效：除风下气，强腰脚，明耳目，除痰饮，理荣卫，永不染时气诸风疾。
用法用量：1剂分为24贴，每贴加生姜1两（切）、杏仁14枚（去尖，碎），以水3升，煮取1升，去滓澄清，取9合顿服，每日1贴，晚间以气下心胸空为妙。10服以后，身力不可当，常须护惜将养之，以饮食补之，每年春、夏、秋、冬各服1剂，胜服肾气丸20剂。
制备方法：上切，如大豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方名肾沥汤，但方中无肾，疑脱。
