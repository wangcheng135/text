序号：26841
名称：百合滑石散
组成：百合1两（炙），滑石3两。
出处：《金匮》卷上。
主治：百合病小便赤涩，脐下坚急。
加减：None
功效：None
用法用量：每服方寸匕，饮下，日3次。当微利者，止服，热自除。
制备方法：上为散。
临床应用：None
各家论述：1.《千金方衍义》：百合病若变发热，乃血脉郁而成热，佐滑石以通利之。_x000D_
2.《金匮方歌括》无犀按：百合病原无偏热之证，变发热者，内热充满，淫于肌肤，非如热之比。主以百合滑石散者，百合清金泻火，降逆气，从高源以导之；滑石退表里之热，利小便。二味合为散者，取散以散之之义，散调络脉于周身，引内外之热气，悉从小便出矣。
用药禁忌：None
附注：滑石散（《活人书》卷十八）。
