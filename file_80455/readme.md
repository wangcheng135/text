序号：80471
名称：鳖甲丸
组成：鳖甲1两半（涂醋，炙令黄），干姜1两半（炮裂，锉），赤石脂1两，丹参1两，代赭3分，甘草3分（炙微赤，锉），桂心1两，细辛1两，川椒1两（去目及闭口者，微炒去汗），附子1两（炮裂，去皮脐），鹿茸3分（去毛，涂酥、醋，炙令黄），当归1两（锉，微炒），禹余粮1两（烧令赤，醋淬7遍，细研），乌贼鱼骨3分，白僵蚕半两（微炒），牛膝1两（去苗），生干地黄1两。
出处：《圣惠》卷七十一。
主治：妇人虚冷，腹中积聚，月事往来，时苦腹满，绕脐下引腰背，手足烦，或冷或热，时腹心中闷，体瘦，不欲食。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《医方类聚》引作“龟甲丸”。方中鳖甲。《医方类聚》作“龟甲”。
