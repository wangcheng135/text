序号：9058
名称：叶氏痫症丸
组成：西牛黄1钱，琥珀3钱，羚羊角3钱，降香2钱，犀角尖3钱，麝香2分，朱砂4钱（水飞），天竺黄4钱，雄黄4钱（水飞），半夏曲4钱，僵蚕2钱（去头足，炒），冰片3分，沉香2钱，全蝎2钱（甘草水浸，去头足），羌活2钱，青礞石3钱（火煅），石菖蒲2钱，真金箔30张，白附子3钱，明天麻2钱（湿纸包，煨），胆星4钱。
出处：《痘学真传》卷七。
主治：痫症。
加减：None
功效：None
用法用量：每岁丸重2分，10岁以外者2钱。
制备方法：预选道地药品精制，为末，照方称准分量合和，再研，用雄猪心血1具，和甘草汁为大丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
