序号：51924
名称：除疟胜金丸
组成：酒炒透常山4两，草果2两，槟榔2两，制苍术2两。
出处：《重订通俗伤寒论》.
主治：痎疟因风寒而发，初起恶寒无汗，头身俱痛，继即往来寒热，发有定时，深者间日一发，极深者三日一发，发冷时形寒战慄，齿齘齘然有声，面头手足皆冷，甚则口唇指甲皆青，发冷过期，即发大热，皮肤壮热色赤，头甚痛，呼吸粗，渴欲饮冷，神倦嗜睡，或心烦懊憹，少则2-3时，多则4-5时，周身大汗，诸证若失，依此反复而作，累月经年，缠绵难愈，舌苔白得而腻，甚或灰腻满布，脉沉弦而迟，胁下无痞块而轻者。
加减：None
功效：温利积水，消化顽痰。
用法用量：每服20-30丸，清脾饮送下。
制备方法：上为细末，水法为小丸，外用半贝丸料为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
