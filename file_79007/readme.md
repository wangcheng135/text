序号：79023
名称：增损三黄石膏汤
组成：石膏8钱，白僵蚕（酒炒）3钱，蝉蜕10个，薄荷2钱，豆豉3钱，黄连2钱，黄柏（盐水微炒）2钱，黄芩2钱，栀子2钱，知母2钱。
出处：《寒温条辨》卷四。
主治：温病三焦大热，五心烦热，两目如火，鼻干面赤，苔黄唇焦，身如涂朱，烦渴引饮，神昏谵语。
加减：如腹胀痛，大便燥结，加大黄。
功效：清热解毒，生津止渴。
用法用量：水煎去滓，入米酒、蜜调，冷服。
制备方法：None
临床应用：None
各家论述：寒能制热，故用白虎汤；苦能下热，故用解毒汤。佐以荷、豉、蚕、蝉之辛散升浮者，以温病热毒至深，表里俱实，扬之则越，降之则郁，郁则邪火犹存，兼之以发扬，则炎炎之势皆烬也。此内外分消之法，犹兵之分击者矣。
用药禁忌：None
附注：None
