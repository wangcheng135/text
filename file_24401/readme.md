序号：24409
名称：安息香丸
组成：安息香3两（锉细，以无灰酒1升浸1宿，以瓷碗中盛，重汤煮成膏），沉香1两，肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），胡桃瓤3两（细研，入安息香膏内）鹿茸1两（去毛，涂酥，炙微黄），补骨脂2两（微炒），附子1两（炮裂，去皮脐），巴戟1两，丁香1两，桂心1两，牛膝1两（去苗），鸡舌香1两。
出处：《圣惠》卷九十八。
主治：肾脏虚冷，脐腹多疼，腰脚沉重，肌体羸瘦，颜色萎黄，食少无力。
加减：None
功效：None
用法用量：每日服30丸，空心以温酒送下。
制备方法：上为末，以安息香膏，更入少许炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
