序号：9485
名称：加味济坤大造丸
组成：紫河车1具（须壮妇人头产男胎连带者，洗净，用砂罐内隔竹片3或5根，剪蒲包1块，架住，放河车于蒲包上，下用白酒，不可令胞粘着，取酒气蒸极熟），人参1两5钱，当归2两，生地（酒洗蒸熟）2两，山药1两，天冬（去心）1两，牛膝（酒浸）1两，黄柏（炒）8钱，杜仲（姜汁、酒炒断丝）8钱，麦冬（去心）1两5钱，五味子5钱。
出处：《女科秘要》卷八。
主治：妇人气虚血弱，宫寒不孕。
加减：如虚弱多汗，潮热，加黄耆，地骨皮、知母各1两；脾胃虚弱久泻，加白术、莲肉各1两，血虚惊悸少睡，加枣仁（炒）1两，元眼肉2两。
功效：None
用法用量：空心每服60-70丸。
制备方法：上为末，捣河车于内，使极匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
