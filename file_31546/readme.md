序号：31555
名称：沉香丸
组成：沉香1钱，干姜（炮）1钱，羌活（去芦头）1钱，楝实1钱，木香1钱，甘草（炙，锉）1钱，肉豆蔻（去壳）1钱，诃黎勒皮1钱，延胡索1钱，肉苁蓉（酒浸，切，焙）1钱，芎1钱，当归（焙）1钱，蓬莪术（煨，锉）1钱，茴香子（炒）1钱，乌头（生，去皮脐）1钱，天麻1钱，人参1钱，丁香（大者）半两，白檀香（锉）半两，青橘皮（去白，焙）1两半，附子（炮裂，去皮脐）1两半，桂（去粗皮）1两半，巴戟天（去心）1两半，牛膝（酒浸，切，焙）1两半，蒺藜子（炒去角）2两，丹砂（研）1分。
出处：《圣济总录》卷六十七。
主治：一切冷气，两胁胀满，背膊拘急疼痛，饮食减少，噎塞不通，真气虚弱，精神昏暗，困倦少力。
加减：None
功效：None
用法用量：每服1丸，煨生姜、橘皮汤嚼下，温酒送下亦可，空心、食前各1次。
制备方法：上25味为末，加丹砂再同研匀，炼蜜为丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
