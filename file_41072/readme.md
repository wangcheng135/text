序号：41087
名称：金不换内消丸
组成：苍术半斤（制，去皮），枳壳1两半（麸炒），青皮（去白）1两，陈皮（去白）1两，三棱（醋煮）1两，蓬术（醋煮）1两，香附子（制）1两，大茴香（炒）1两，干漆（炒）1两，藿香（洗）1两，厚朴1两3钱，杏仁（去皮尖）1两3钱，砂仁1两3钱，猪牙皂角2两（去皮弦），黑牵牛2两，草果1两，百草霜1两（用乡村烧百草柴者佳）。
出处：《医统》卷三十三引《医林》。
主治：一切积聚，气蛊，胸膈膨胀，肚腹满闷，心紧束。
加减：None
功效：None
用法用量：每服50丸，临卧或日中，不拘时候，好酒、茶清、盐汤任下1-2服，便觉腹中宽快，并无泄泻之患。
制备方法：上为细末，面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：妊娠禁服。
附注：None
