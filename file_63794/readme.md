序号：63810
名称：清离滋坎汤
组成：怀生地黄1钱，熟地黄1钱，麦门冬（去心）1钱，当归（酒洗）1钱，白芍（酒炒）1钱，怀山药1钱，牡丹皮6分，炙甘草3分，天门冬（去心）1钱，白茯苓（去皮）1钱，山茱萸（酒蒸，去核）1钱，白术（去芦，土炒）1钱，泽泻（炒）5分，黄柏（蜜炒）5分，知母5分。
出处：《寿世保元》卷四。
主治：因房欲过度，而成阴虚火动劳瘵之症。发热咳嗽，吐痰喘急，盗汗，五心烦热，吐血衄血，咽喉声哑，夜梦泄精，耳鸣眼花，六脉沉数而涩；及阴虚火动之眩晕，赤白浊下。
加减：盗汗，加黄耆（蜜水炒）、酸枣仁（炒）；嗽甚，加五味子，痰盛，加贝母，瓜蒌仁；热盛，加骨皮；心下怔忡、恍惚不寐，加远志（去心），酸枣仁（炒）；遗精，加龙骨、牡蛎（煅）；胸中不爽，加陈皮；泄泻，加莲肉、陈皮，去知母、黄柏；吐血，衄血加犀角、玄参；气虚，加人参；阳虚，加熟附子；咽疮喉痛，加桔梗、玄参；眩晕，加川芎、天麻、山栀、竹沥少许；赤白浊下，加萆薢、牛膝、山栀、萹蓄、赤芍。
功效：None
用法用量：加生姜、大枣，水煎，温服。痰盛加竹沥，姜汁；热盛加童便，人乳同服。
制备方法：上锉。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
