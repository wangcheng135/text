序号：34320
名称：万灵膏
组成：木鳖（去壳）20个，蓖麻（去壳）100粒，威灵仙2钱，当归2钱，川芎2钱，赤芍2钱，防风2钱，荆芥2钱，羌活2钱，独活2钱，生地黄2钱，白芷2钱，黄芩2钱，黄连2钱，黄柏2钱，姜黄2钱，蛇退1条，麻油（冬）7两、（夏）5两、（秋）6两。
出处：《简明医彀》卷八。
主治：风寒湿气，跌扑损伤，风毒脚气，遍身疼痛，挫气闪肭，筋骨痠疼，及血风、脓窠、裙边、臁疮，手足一切诸疮毒。
加减：None
功效：None
用法用量：如风气闪挫，捣炒姜、葱擦患处，次贴膏药。泄泻、腹痛，贴脐上。
制备方法：浸药1日，煎药焦，滤去滓，油入锅煎滚，下黄蜡2两，次入嫩松香2斤（老松添油），桃、柳枝搅化，滴水成珠，不拈手；预备水半缸，稀麻布1幅，铺绵少许，二人扯定，将膏倾于布上，滤入水中，二人对扯，黄色入钵。陆续置小器中，微火燉，摊油纸或布。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
