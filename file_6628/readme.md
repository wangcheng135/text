序号：6628
名称：加减八味汤
组成：熟地5钱，萸肉3钱，附子1两半（炮），肉桂1钱半（去皮），山药3钱（炒），白芍1钱半（酒炒），五味1钱半，益智3钱（盐水炒），覆盆子3钱（炒）。
出处：《医略六书》卷二十六。
主治：肾虚遗溺，脉弱者。
加减：None
功效：None
用法用量：水煎，去滓温服。
制备方法：None
临床应用：None
各家论述：附子补火逐冷，肉桂暖血温经，熟地补阴滋肾，萸肉秘气涩精，山药补脾阴以固下，白芍敛阴血以固经，益智补火通心，兼摄涎水，五味补肺滋肾，收敛津液，覆盆子益肾膀以缩小便也。水煎温服，使火壮阳回，则关门扃固而蓄泄有权，岂有遗溺之患乎！
用药禁忌：None
附注：None
