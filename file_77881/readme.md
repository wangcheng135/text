序号：77897
名称：蟾酥丸
组成：蟾酥半两，麝香（研）半两，犀角（镑）半两，牛黄（研）半两，丹砂（研）半两，芦荟（研）半两，天竺黄半两，益智（去皮）10个，青黛（研）半两，干蜗牛5个（全者），白花蛇1寸（去皮骨，炙）。
出处：《圣济总录》卷一七二。
主治：小儿疳渴不止，及急慢惊风，胸膈有涎，天钓疳风。
加减：None
功效：利胸膈，坠涎，压心脏积热，顺气，进奶食。
用法用量：每服5-7丸，煎薄荷汤送下；惊风，用剪刀股研，薄荷汤送下；慢惊风，煎荆芥汤送下；疳气，麝香汤送下；惊风搐搦，目睛上视，煎金银酒化下。
制备方法：上为末，用豮猪胆汁为丸，如米粒大，丹砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
