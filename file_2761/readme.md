序号：2761
名称：天仙藤散
组成：天仙藤、香附、陈皮、甘草、乌药、茯苓、白术、大腹皮、人参、紫苏、当归、杏仁、厚朴、白芍。
出处：《陈素庵妇科补解》卷三。
主治：子气(亦名胎肿)。妊娠3月，两足自脚面渐肿至腿膝，行步艰辛，以致胸膈喘闷，饮食减少，似水气状，脚趾间有黄水出者。此系素有风气，或冲任二经有血风所致。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：是方四君补正气，归、芍养胎血；附、陈、乌、腹利气消胀；杏、朴快膈温中，天仙佐之；紫苏主治妇人血风之气。肿消之后，当用四物、四君、杜仲、远志、山药、木香之属，峻补气血。
用药禁忌：None
附注：None
