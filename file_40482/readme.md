序号：40492
名称：虎骨浸酒
组成：虎胫骨2两（涂酥，炙黄），熟干地黄2两，续断1两，赤箭1两，石斛1两（去根，锉），防风1两（去芦头），牛膝1两（去苗），丹参1两（去芦头），桂心1两，当归1两（锉，微炒），萆薢1两（锉），芎1两，酸枣仁1两（微炒），山茱萸1两，五味子1两，漏芦1两，五加皮1两，附子1两（炮裂，去皮脐），仙灵脾1两，骨碎补1两，荆芥半两，川椒半两（去目及闭口者，微炒去汗），海桐皮1两，肉苁蓉1两（酒浸1宿，刮去皱皮，炙令干），木香1两，乌蛇1条（重5两，用无灰酒浸3宿后取出，去头尾皮骨，炙微黄）。
出处：《圣惠》卷七。
主治：肾脏风毒流注，腰脚疼痛，行立无力。
加减：None
功效：None
用法用量：以无灰酒3斗浸7日。每于食前暖饮1小盏。
制备方法：上锉细，用生绢袋盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“虎骨酒”。
