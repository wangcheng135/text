序号：26682
名称：延年丸
组成：菟丝子（酒浸7日，炒黄）3两，枸杞子1两半（去梗），覆盆子1两半（去萼），车前子1两半（酒浸），巴戟天1两半（去心），远志1两半（去心），生干地黄1两半，细辛1两半（去苗叶），白术（炒），菖蒲1两半（锉），何首乌1两半（去黑皮），地骨皮1两半，牛膝1两半（酒浸1宿），续断1两半，菊花（去梗萼）1两半。
出处：《圣济总录》卷一八五。
主治：None
加减：None
功效：平补五脏，治百病；育神气，强力益志，美颜色，变髭发。
用法用量：每服30丸，空心常服；丈夫盐汤、妇人醋汤送下，午食前更1服。服至10服，其人病却有发时，是药动病本，功应也。
制备方法：上药除菊花外，以温水和酒少许洗过，焙干，杵罗为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：枸杞子丸（《圣济总录》卷一八六）。
