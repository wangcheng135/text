序号：14023
名称：全鹿丸
组成：当归身4两，知母（去尾净）4两，天门冬（去心、皮净）4两，怀熟地4两，人参（去芦）4两，白茯苓（去皮）4两，金樱子（去粗皮、刺净）4两，芡实肉6两，牛膝（去芦）6两，莲肉（去心）4两，山药4两，黄柏（去皮）4两，怀生地4两，白芍2两（炒），麦门冬（去心）4两，枸杞子（去蒂）8两，茯神（去皮心）4两，杜仲（酥炙，去丝）4两，白术（东壁土炒）4两，莲须4两，山茱萸（净肉）8两，女贞实8两，覆盆子4两，柏子仁6两，肉苁蓉4两（酒洗），黄耆（蜜炙净）4两，骨碎补4两，五味子2两（净），桑椹子4两（净），陈皮4两，菟丝子（酒煮捣烂，晒干，净）2两。
出处：《墨宝斋集验方》卷上。
主治：None
加减：None
功效：补肾种子。
用法用量：空心温酒、淡盐汤或白滚汤送下。
制备方法：用雄鹿1只，取精肉1-2斤，用血、髓、肾、肝、心、全角切作片，用无灰酒25斤煮熟，将前药再入酒肉内煮干为度，如末全熟，再加酒5斤，取出晒干，为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
