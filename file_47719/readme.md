序号：47734
名称：神仙延龄丹
组成：旱莲（取汁，晒干成膏子）半斤，破故纸（炒香，为末）1斤，五加皮（酒浸1昼夜，晒干）、赤茯苓（去皮，乳浸，牛乳可代）、生地黄2斤（酒浸1昼夜，取汁，晒成膏子），红枣（去皮，煮熟）、生姜2斤（取汁，晒干成膏子），杜仲（去皮，炙炒去丝，为末）半斤，核桃仁（去皮）半斤，川芎4两，枸杞（去蒂，酒浸）4两，没石子2两，蜂蜜（炼老熟）2两，细辛1两。
出处：《鲁府禁方》卷二。
主治：瘫痪，五劳七伤，颜色枯干，身体羸瘦，妇人久不成胎，男子精神减少，行步艰难，筋骨疼痛。
加减：None
功效：使衰返壮，折骨复坚，生瘢痕，耳聪目明，益寿延年。
用法用量：每服30-50丸，酒或盐汤送下。服20日后，退白生黑，久服延年。
制备方法：上除桃仁、红枣、蜜外，其余各为细末，将桃仁、红枣、蜜煮熟为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：按：方中五加皮、赤茯苓、红枣用量原缺。
