序号：80061
名称：摩风膏
组成：当归3两，白芷1两，附子3两（生，去皮脐），细辛2两，桂心1两，天雄3两（生，去皮脐），干姜2两，芎2两，川乌头2两（生，去皮脐），朱砂1两（细研），雄黄1两（细研），醋3升，松脂半斤，生地黄3两（捣，绞取汁），猪脂5斤（炼成者）。
出处：《圣惠》卷二十五。
主治：一切痛风。
加减：None
功效：None
用法用量：每取少许，摩于病处。如胁下聚如杯者，摩及涂之即愈；又面目黧黑消瘦，是心腹中冷，酒调服半匙，日3次。
制备方法：上锉细，以地黄汁及醋浸1宿，滤出，入猪脂中，慢火煎之，候白芷色黄即膏成，绵滤去滓，入丹砂、雄黄及松脂等，以柳木篦搅令匀，于瓷器中盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
