序号：83124
名称：乌头煎丸
组成：乌头半斤（不去皮尖，水浸3两宿），天麻1两，巴戟天（去心）半两，海桐皮（锉）1两，补骨脂（炒）半两，牛膝（去苗，酒浸，切，焙）1两半，肉豆蔻（去壳）半两，茴香子（炒）1两，萆薢半两，木香1两，石斛（去苗）半两，沉香（锉）1两，大枣15个（烂煮，去皮及核入膏中）。
出处：《圣济总录》卷一八六。
主治：None
加减：None
功效：补壮筋骨。
用法用量：每服30丸，空心盐汤送下。
制备方法：先用黑豆半升煮乌头，以豆熟为度，取出切作片子，焙干，用青盐2两炒令黄色，去盐，捣为细末；以醇酒2升，先煖酒令滚沸，次下乌头，熬成膏，然后以诸药焙干，捣罗为末，并枣膏入膏内，和纳得所为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
