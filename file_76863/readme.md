序号：76879
名称：集成沆瀣丹
组成：杭川芎（酒洗）9钱，锦庄黄（酒洗）9钱，实黄芩（酒炒）9钱，厚黄柏9钱（酒炒），黑牵牛（炒，取头末）6钱，薄荷叶4钱5分，粉滑石（水飞）6钱，尖槟榔7钱5分（童便洗，晒），陈枳壳4钱5分（麸炒），净连翘（除去心膈，取净）6钱，京赤芍（炒）6钱。
出处：《幼幼集成》卷二。
主治：小儿一切胎毒，胎热胎黄，面赤目闭，鹅口疮，重舌木舌，喉闭乳蛾，浑身壮热，小便黄赤，大便闭结，麻疹斑瘰，游风癣疥，流丹隐疹，痰食风热，痄腮面肿，十种火丹。
加减：None
功效：None
用法用量：月内之儿，每服1丸，稍大者2丸，俱用茶汤化服。但觉微有泄泻，则药力行，病即减矣；如不泄再服之，重病每日3服，以愈为度。此方断不峻厉，幸毋疑畏。
制备方法：依方炮制，和匀焙燥，研极细末，炼蜜为丸，如芡实大。
临床应用：None
各家论述：盖夫脏气流通者必不郁滞，或受毒于胎前，或感邪于涎后，遂尔中气抑郁，则见以前诸证。方内所用黄芩清上焦之热；黄柏清下焦之热；大黄清中焦之热，又藉其有推陈致新之功，活血除烦之力，能导三焦郁火从魄门而出。犹虑苦寒凝腻，复加槟榔、枳壳之辛散，为行气利痰之佐使。川芎、薄荷引头面风热从高而下趋；连翘解毒除烦；赤芍调营活血；牵牛利水，走气分而舒郁；滑石清润，抑阳火而扶阴，又能引邪热从小便而出。
用药禁忌：胎寒胎怯面青白者忌之，乳母切忌油腻。
附注：沆瀣丸（《麻疹全书》卷三）、沆瀣丹（《观聚方要补》卷十）。
