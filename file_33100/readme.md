序号：33109
名称：万应灵膏
组成：生地2两，川附2两，香附2两，乌药2两，五加皮1两，桂枝1两，当归1两，防风1两，羌活1两，独活1两，秦艽1两，天虫1两，全蝎1两，灵仙1两，川乌1两，草乌1两，白芷1两，良姜1两，大黄1两，麻黄1两，赤芍1两，莪术1两，三棱1两，桃仁1两，红花1两，六轴子1两，头发1两，麻油8斤。
出处：《饲鹤亭集方》。
主治：跌打损伤，闪腰挫气，筋骨疼痛，一切疟痞，内伤。
加减：None
功效：None
用法用量：照铜人图按穴贴之。
制备方法：浸数日，煎炼成膏，再加香料：官桂、丁香、木香、甘松、山柰、排草、辛夷、檀香、乳香、没药、白胡椒、苏合油，上药等分，官桂加重，胡椒减半，生晒为末，搅匀摊贴。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
