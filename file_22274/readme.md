序号：22282
名称：阳和至宝膏
组成：鲜紫苏8两，鲜牛蒡8两，鲜白凤仙4两，连根青葱8两，鲜萆薢8两，鲜薄荷8两，鲜苍耳草8两，以上7味，取叶、梗、根全草，用麻油10斤浸10日，煎枯去滓待冷，4天后再加青防风1两，荆芥1两，水红花子1两，木香1两，川附子1两，当归1两，天麻1两，穿山甲1两，陈皮1两，白芷1两，川芎1两，连翘1两，白芥子1两，官桂1两，乌药1两，草乌1两，僵蚕1两，天南星1两，桂枝1两，大黄1两，白蔹1两，赤芍1两，生半夏1两，青皮1两，蒲公英1两，青木香1两，熬枯去滓，熬至滴水成珠，入陶丹7两，文火收膏，渐温，入后药：制肉桂3两，炙乳香1两，没药1两，琥珀2两，芸香2两，丁香油4两，苏合油4两，当门子3两。
出处：《千金珍秘方选》。
主治：痰毒痰核，瘰疬乳疖，阴毒流注，以及外证之色不红者、皮肉所结之痰块。
加减：None
功效：None
用法用量：用时隔水炖化，摊用。修合时宜于夏末，膏必须熬老；如太老，加苏合油不拘多少。
制备方法：上为细末，入膏搅匀，瓷罐收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
