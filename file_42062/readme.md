序号：42077
名称：三黄败毒散
组成：防风2两，荆芥2两，连翘2两，白芷梢2两，黄芩2两，黄连2两，栀子2两，地骨皮2两，归尾2两，赤芍2两，川芎（上部疮多倍用）2两，五加皮2两，木瓜2两，苦参2两，黄柏2两，薏苡仁2两，僵蚕2两，蝉蜕2两，蒺藜2两，甘草2两，白鲜皮2两，皂角刺2两，木通（下部疮多倍用）2两，土茯苓（白者）3斤。
出处：《古今医鉴》卷十五。
主治：天泡、杨梅等疮。
加减：疮痛，加羌活、独活；体虚，去栀子，加入参、茯苓。
功效：None
用法用量：共锉作50剂。水煎滚，每日2服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌生盐、牛肉、烧酒。
附注：None
