序号：9667
名称：水煮青盐丸
组成：附子（生，去皮脐，锉，炒）1两，人参（去芦）1两，京三棱（炮）1两，肉桂（去粗皮，不见火）1两，木香（湿纸裹，煨）1两，鹿茸（爁，去毛，酒浸）1两，缩砂仁1两，蓬莪术（生）1两，益智仁1两，舶上茴香1两，阳起石（酒一盏，煮干，别研）1两，川椒2两（去目合口者，炒出汗），陈皮3两（去白），厚朴4两（去皮，姜汁浸一宿，炙干）。
出处：《魏氏家藏方》卷七。
主治：脾积泻，经年不效者。
加减：None
功效：None
用法用量：每服20丸，食前温酒盐汤送下。
制备方法：上锉，加青盐4两，水浸药平一指许，煮干为度，焙干，为细末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
