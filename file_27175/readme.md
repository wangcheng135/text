序号：27183
名称：当归散
组成：当归（去芦酒洗）2两，赤芍药2两，大黄（半生半炮）1两2钱，川芎半两，麻黄（制）半两，甘草（半生半炙）1两。
出处：《活幼心书》卷下。
主治：疳积曩泻，面赤萎黄，肚胀脚弱，头大项小，发稀直竖，肌肉削瘦，不思饮食，昼凉夜热，或腹内有疝瘕气块，泻则颜色不等，其臭异常，其泻有时，或一月半月一番，自泻自止；及小儿阴囊肿，阴茎全缩不见。
加减：None
功效：顺调气血，和解表里，爽利心腹，疏理百病。
用法用量：每服2钱，水1盏，加生姜2片，煎7分，温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：《幼科折衷》治疳积曩泻，先用本方加三棱、陈皮煎服；治阴囊肿，用本方加槟榔、苍术服。
