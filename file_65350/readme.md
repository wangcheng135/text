序号：65366
名称：女科乌鸡白凤丸
组成：白毛雄乌鸡1只（缢死，去肚杂物，用黄酒2斤煮烂，配入后药），党参4两，白芍（酒炒）2两，川断3两，桑螵蛸2两，炒于术2两，炙黄耆3两，广郁金2两，川藁本2两，茯苓（乳拌）3两，制香附4两，地骨皮（酒炒）2两，萸肉2两，炙甘草1两，杜仲3两，煅龙骨2两，丹皮2两，当归3两，丹参3两，煅牡蛎2两，延胡索1两5钱，川芎2两5钱，怀山药2两5钱，白薇2两，红花1两。
出处：《全国中药成药处方集》（杭州方）。
主治：妇人血虚阴亏，面黄肌瘦，神困体倦，虚劳成疾，月经不调，崩漏带下，骨蒸潮热，久不生育。
加减：None
功效：补益气血，调经种子。
用法用量：每服2至4钱，淡盐汤、米汤或开水送下。
制备方法：上为细末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
