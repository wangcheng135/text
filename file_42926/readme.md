序号：42941
名称：保真丸
组成：鹿胶8两（炒成珠），杜仲3两，山药3两（炒），白茯苓2两，五味子1两，菟丝饼1两5钱（酒煮），熟地2两，萸肉1两5钱，鹿茄茸1具（酥炙），川牛膝1两，益智仁1两，远志肉1两（甘草水泡），小茴1两（盐水炒），川楝子1两（去心核），巴戟肉1两（去油），故纸1两，胡芦巴1两，柏子仁1两，山甲片3钱，沉香3钱，人参2两。
出处：年氏《集验良方》卷二。
主治：虚损。
加减：None
功效：None
用法用量：每服3钱，温酒送下。
制备方法：上为细末，用肉苁蓉肉4两（洗净鳞甲白膜），好酒2两煮成膏，同炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
