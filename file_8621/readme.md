序号：8621
名称：比天保贞膏
组成：蛇床子10两，川楝子10两，熟地10两，生地10两，生杏仁10两，官桂10两，川断10两，川附片10两，牛膝10两，菟丝子10两，木鳖子10两，谷精草10两，紫梢花10两，天冬10两，麦冬10两，肉果10两，苁蓉10两，甘草64两，虎骨16两。
出处：《北京市中药成方选集》。
主治：男子气虚肾寒，阳事不兴，久无子嗣。妇女气虚血亏，行经腹痛，久不孕育。
加减：None
功效：滋阴补气，暖肾散寒。
用法用量：用时贴脐腹部或肾俞穴。
制备方法：上药酌予碎断，用香油640两炸枯，过滤去滓。加章丹336两，松香336两，鹿胶16两，熬成膏，再兑：麝香2两，冰片64两，硫黄面10两，赤石脂10两，龙骨面10两，阳起石面10两，蟾酥面10两，母丁香10两，乳香面10两，没药面10两，木香面1两，沉香面10两，雄黄面10两，摊成外用膏药，每张重5钱。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
