序号：76217
名称：舒肝调气丸
组成：陈皮1斤，玄胡（醋制）1斤，黑郁金1斤，菖蒲1斤，五灵脂（醋炒）1斤，枳实（麸炒）3斤，莪术（醋制）1斤，胆草2斤，丹皮1斤，郁李仁8两，厚朴花9两，炒黑丑8两，生白芍1斤，炒白丑8两，片姜黄12两，香附（醋制）3斤，厚朴（姜制）3斤，广木香1斤8两，沉香8两，蔻仁1斤8两，青皮（醋炒）2斤，炒莱菔子4两。
出处：《全国中药成药处方集》（天津方）。
主治：两胁胀满，胸中烦闷，呕吐恶心，气逆不顺，倒饱嘈杂，消化不良，大便燥结。
加减：None
功效：舒气开郁，健胃消食。
用法用量：每次服1袋，白开水送下。
制备方法：上为细末，水泛为小丸，每斤丸药用桃胶2钱化水，滑石3两为衣，2钱重装袋。
临床应用：None
各家论述：None
用药禁忌：孕妇及虚弱者忌服。
附注：None
