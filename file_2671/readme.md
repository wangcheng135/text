序号：2671
名称：天门冬大煎
组成：天门冬3斗半（切，捣，压取汁尽），生地黄3斗半（切，捣，压如门冬），枸杞根3斗（切，净洗，以水2石5斗，煮取1斗3升，澄清），獐骨1具（碎，以水1石，煮取5斗，澄清），酥3升（炼），白蜜3升（炼），上6味，并入大斗铜器中，微火先煎地黄、门冬汁，减半乃合煎，取大斗2斗，下后散药：茯苓2两，柏子仁2两，桂心2两，白术2两，萎蕤2两，菖蒲2两，远志2两，泽泻2两，薯蓣2两，人参2两，石斛2两，牛膝2两，杜仲2两，细辛2两，独活2两，枳实2两，芎2两，黄耆2两，苁蓉2两，续断2两，狗脊2两，萆薢2两，白芷2两，巴戟天2两，五加皮2两，覆盆子2两，橘皮2两，胡麻仁2两，大豆黄卷2两，茯神2两，石南2两，甘草6两，蜀椒1升，薏苡仁1升，阿胶10两，大枣100枚（煮作膏），鹿角胶5两，蔓荆子3两。
出处：《千金》卷十二。
主治：男子五劳七伤，八风十二痹，伤中六极，脚气。
加减：小便涩，去柏子仁，加秦艽2两，干地黄6两；阴萎失精，去萎蕤，加五味子2两；头风，去柏子仁，加菊花、防风各2两；小便利，阴气弱，去细辛、防风，加山茱萸2两；腹中冷，去防风，加干姜2两。
功效：None
用法用量：每服20丸，平旦空腹以酒送下，1日2次。加至50丸。有牛髓加3升，鹿髓加3升大佳。
制备方法：上药治下筛，纳煎中，煎取1斗；纳铜器重釜煎，令隐掌可丸，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：慎生冷、醋、滑、猪、鸡、鱼、蒜、油、面等。女人先患热者得服，患冷者勿服。
附注：天门冬丸（《鸡峰》卷四）。
