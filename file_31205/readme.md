序号：31214
名称：杏仁酝酒
组成：杏仁（汤浸，去皮尖双仁）3斗，糯米1石（簸去糠），麦曲20斤（焙令干，捣为末）。
出处：《圣济总录》卷十一。
主治：风腲腿，四肢不收，失音不语。
加减：None
功效：None
用法用量：每服5合，不拘时相续饮之。常令半醺，无至醉吐为妙。
制备方法：先取杏仁2斗捣，更入砂盆内烂研，渐入水8斗，旋研旋绞，取汁令尽去滓。煎取4斗尝之，若香滑则熟。倾入不津瓮中，如法盖覆，作3料酝酒。第一酝：取糯米6斗，炊作饭，用曲末12斤拌和；又取杏仁4升烂研，渐以水1斗6升，煎取8升，寒温适宜，投入前药瓮中酝之，令米糜溃。第二酝：取糯米2斗炊饭，用曲末4斤拌和；又取杏仁3升研烂，渐以水1斗1升，煎取6升，寒温适宜，投入前药瓮中。第三酝：用米、曲、杏仁水汁，一切依第二酝法。上三酝即毕，用蜡纸密封，莫令气泄，于静处安，候香熟。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
