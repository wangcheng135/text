序号：71372
名称：硫黄丸
组成：硫黄4两（酒煮令黑色，细研），雄雀儿50只（取肉，研），天雄4两（炮裂，去皮脐），阿魏2两（面裹煨令面熟为度），硼砂2两（细研），桂心2两，远志3两（去心），菟丝子2分半（酒浸3日，晒干，别捣为末），晚蚕沙2合半（醋浸1日，晒干）。
出处：《圣惠》卷九十八。
主治：None
加减：None
功效：暖下元，治风冷，益精髓，悦颜色，久服轻身倍力，耐寒暑，壮筋骨。
用法用量：每服20丸，空心盐汤或温酒送下。
制备方法：上为末，入研了药和匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：《普济方》有硇砂，无硼砂、天雄。
