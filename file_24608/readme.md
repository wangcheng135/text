序号：24616
名称：竹沥化痰丸
组成：海浮石（煅）16两，白术（炒）32两，橘皮96两，瓜蒌子32两，南星（炙）48两，茯苓32两，黄连16两，法半夏64两，百部草32两，南楂64两，香附（炙）64两，玄参（去芦）16两，熟军48两，莱菔子（炒）32两，枳实（炒）64两，苏子（炒）96两，黄芩96两，杏仁（去皮，炒）16两，贝母16两。
出处：《北京市中药成方选集》。
主治：痰火上攻，胸膈不畅，咳嗽喘满，咽喉堵塞。
加减：None
功效：清热化痰，快膈止嗽。
用法用量：每服2钱，温开水送下，日2次。
制备方法：上为细末，每16两加竹沥水1两，用冷汗水泛为小丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
