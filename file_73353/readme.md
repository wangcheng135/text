序号：73369
名称：滞下如金丸
组成：真川连（真姜汁浸，隔土如法炒9次）不拘斤两。
出处：《广笔记》卷一。
主治：痢疾腹痛，里急后重，便下赤白。
加减：如后重甚，加槟榔1钱5分，枳壳1钱，木香汁7匙，调入；口渴发热，调入滑石末3钱；小便赤涩，短少或不利，加滑石末3钱；赤多，加乌梅肉2钱，山楂肉3钱，红曲2钱；兼里急，用当归等加入前方；白多，加吴茱萸（汤泡1次）7分，酒炒黄芩1钱5分；恶心欲吐，即噤口痢，加人参、石莲肉、升麻（醋炒）各8分至1钱，白扁豆（炒）3钱，白芍药（酒炒）3钱；久痢不止，加肉豆蔻1钱，莲肉（去心，炒黄）3钱，砂仁（炒）1钱5分，人参3钱，白扁豆（炒，去壳）2钱，炙甘草1钱，橘红2钱，白芍药（酒炒）3钱，白茯苓2钱，共为细末，炼蜜为丸，如梧桐子大，每服3钱，米汤吞下；若积滞未尽，加滑石末3钱，每服4钱，白汤吞下；水泻无积滞者，用人参、橘红、炒砂仁煎汤吞服3钱。
功效：None
用法用量：每服4钱，如胃弱，以莲子40粒，橘红2钱，人参2钱，升麻（醋炒）7分，煎汤吞服；腹痛，以白芍药3钱，炙甘草1钱，黄柏1钱，升麻（醋炒）7分，煎汤吞服；里急甚，以白芍药3钱，炙甘草1钱，当归2钱，升麻（醋炒）7分，煎汤吞服。
制备方法：上为细末，姜汁和水跌丸，如梧桐子大，贮瓷器中封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
