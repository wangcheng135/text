序号：71118
名称：新制兰膏汤
组成：泽兰叶3钱，石膏5钱，蝉蜕壳12枚，白僵蚕2钱，桔梗2钱，甘草1钱，防风1钱，炒栀子1钱，薄荷叶1钱，黄芩1钱，新会橘红1钱，元明粉1钱，当归1钱，白芍药2钱，雄黄2分，黄蜜3钱。
出处：《羊毛温证论》。
主治：羊毛温邪，气血亏损，或产后半月内伏羊毛毒火，胸闷食少，寒热头身作痛，呕吐黄水，口苦粘腻，腹胀胁痛，遍身麻木，倦怠神昏，气阻发厥。并治余邪口淡作干，烦热不寐。
加减：如毒重深伏，加熟大黄2钱和服。
功效：None
用法用量：水煎去滓，下元明粉、雄黄，黄蜜和匀，温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
