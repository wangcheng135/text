序号：67950
名称：黄耆必安丹
组成：没药2两，全蝎2两（酒浸，焙干），羌活2两，虎骨2两（酥炙），独活2两，防风2两（去芦），川芎2两，当归2两，薏苡仁2两，半夏（姜制）2两，川乌头1两（炮，去皮尖），天麻1两，枳壳1两（去白，麸炒），前胡1两，陈皮红1两，细辛1两，朱砂（别研为衣）1两，白术半两，麝香1钱（别研），脑子半钱（别研）。
出处：《普济方》卷九十三引《卫生家宝》。
主治：血虚生风，左瘫右痪，口眼斜，语言謇涩诸疾。
加减：None
功效：None
用法用量：每服1丸，食后以酒嚼下，或荆芥汤送下。
制备方法：上为细末，糯米糊为丸，如龙眼大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
