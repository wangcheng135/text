序号：52806
名称：香积丸
组成：木香半两，枳壳（去瓤，麸炒）半两，羌活（去芦头）半两，独活（去芦头）半两，干姜（炮）半两，桂（去粗皮）半两，人参半两，陈橘皮（汤浸，去白，焙）半两，芎藭半两，甘草（炙，锉）半两，白术半两，附子（炮裂，去皮脐）半两，京三棱（煨，锉）半两，大黄（蒸过，切，焙）半两，肉豆蔻（去皮）1分，槟榔（锉）1两，牵牛子（净淘，拣，焙干）1斤（取粉半斤，别入用）。
出处：《圣济总录》卷十二。
主治：风气及心腹诸疾；妇人血风劳气，心腹胀痛；小儿疳痢、时疫、癥瘕。
加减：None
功效：除风气，利胸膈。
用法用量：每用时，旋称药末1两，牵牛子粉半两，和匀，炼蜜为丸，如梧桐子大。每服20-30丸；葱白、腊茶送下；生姜汤、温酒亦可。
制备方法：上除牵牛子外，为末，瓷合收，勿泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
