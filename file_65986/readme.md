序号：66002
名称：蛇头丸
组成：蜈蚣（姜汗炙干）20枚，花蛇头（酒浸1宿，焙干，碎）20枚，全蝎10两（净），天南星10个（姜汁煮1宿，焙），铅白霜（拣净）40两，铁粉30两，蛇黄石80两（醋煮7次，飞研），腻粉2两（研），脑子（细研）5两，珍珠（末，水飞）5两，麝香（研）3两，百草霜（研）3两，朱砂（研，飞）1两，血竭（细研）1两，芦荟（研）1两，白附子50两（炮裂），雄黄1两半（醋煮，水飞，焙干）。
出处：《幼幼新书》卷九引《聚宝方》。
主治：小儿急慢惊风，目睛上视，啮齿弄舌，面青口噤，背强啼叫，咽膈涎声，神昏不语，及内钓诸痫，腹内泄泻，夜卧时惊，潮热气喘。
加减：None
功效：None
用法用量：初生婴孩可服半丸，周岁已上可服1粒，不拘时候，并用薄荷汤化下。
制备方法：上为末。三家粽子为丸，如鸡头大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
