序号：63359
名称：清宫丹
组成：乌犀角5钱，元参4钱，杏仁3钱，寸冬4钱，粉葛根3钱，生石膏3钱，蒌仁4钱，桔梗4钱，生地4钱，枳壳3钱，芥穗3钱，牛蒡子3钱，木通3钱，连翘4钱，山栀4钱，明雄4钱，黄芩5钱，朱砂4钱，粉草4钱。
出处：《全国中药成药处方集》（沈阳方一）。
主治：流行感冒，头痛鼻塞，发热恶寒，遍身疼痛，烦渴干呕，咳嗽气促，神昏胸闷，大便秘闭，小便赤，两胁引痛。
加减：None
功效：解表散风，清热化毒。
用法用量：每服1丸，生姜汤送下，或用白开水亦可。服后宜令微汗。
制备方法：上为极细末，炼蜜为丸，每丸1钱4分重。
临床应用：None
各家论述：None
用药禁忌：忌食五辛发物及油腻之食物；孕妇忌服。
附注：None
