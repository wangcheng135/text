序号：79411
名称：蝉花无比散
组成：蛇蜕（微炙）1两，蝉蜕（去头足翅）2两，羌活3两，当归（洗，焙）3两，石决明（用盐同东流水煮一伏时漉出，捣研如粉）3两，川芎3两，防风（去叉枝）4两，茯苓（去皮）4两，甘草（炙）4两，芍药（赤者）13两，蒺藜（炒去刺）半斤，苍术（浸，去皮，炒）12两。
出处：《局方》卷七（续添诸局经验秘方）。
主治：远年近日一切风眼、气眼攻注，眼目昏暗，睑生风粟，或痛或痒，渐生翳膜，侵睛遮障，视物不明；及久患偏正头风，牵搐两眼，渐渐细小，连眶赤烂；及小儿疮疹入眼，白膜遮睛，赤涩隐痛。
加减：None
功效：常服祛风，退翳明目。
用法用量：每服3钱，食后米疳调下，茶清亦得。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：忌食发风毒等物。
附注：蝉蜕无比散（《异授眼科》）。
