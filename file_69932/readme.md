序号：69948
名称：滋燥丸
组成：熟地5两，枸杞3两，牛膝3两，茯苓2两5钱，当归2两5钱，黄耆2两，麦冬2两，白芍2两，人参1两5钱，知母1两5钱，黄柏1两5钱，牡丹皮1两5钱，五味子1两，黄连1两（茱萸制），甘草1两。
出处：《活人方》卷六。
主治：房劳内伤，肾水枯涸，肝木无所禀受，木燥火炎，本经无血可藏，精血既亏，则三焦之火乘虚攻刺于所经所络之地为痛，痛连腰肾、心胸，不能转侧，昼轻夜重，躁热憎寒，饮食减少，形容衰惫。
加减：None
功效：None
用法用量：每服4-5钱，早晨空心淡盐汤吞服。
制备方法：上为细末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
