序号：36200
名称：参耆鳖甲散
组成：人参（去芦）1两，黄耆（蜜炙）1两，鳖甲（去裙净，酒醋炙令黄）1两，白术（炒）1两，当归（酒浸，去芦）1两，白茯苓（去皮）1两，甘草（炙）1两，白芍药2两，附子（炮，去皮脐，锉，生姜自然汁浸1宿，蒸两次）半两，金钗石斛（酒浸，炒）半两，干姜（炮，洗）半两，肉桂（去粗皮，不见火）半两。
出处：《魏氏家藏方》卷四。
主治：劳倦。
加减：None
功效：补虚壮力，调荣卫，进饮食。
用法用量：每服2钱，水1大盏，加生姜3片，大枣子1个，乌梅1个，煎至7分，去滓。空心、食前服；温酒调下亦得。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
