序号：40253
名称：苦参散
组成：苦参（取头末）2两，猪肚1个。
出处：《儒门事亲》卷十五。
主治：疠风。
加减：None
功效：None
用法用量：每服30-50丸，温酒或茶清送下，不拘时候，1日3次；后用苦参、荆芥、麻黄煎汤冷洗。
制备方法：上以苦参末掺猪肚内，用线缝合，隔宿煮软，取出洗去原药，先不吃饭5顿，至第2日先饮新水1盏，后将猪肚食之，如吐了，再食之，食罢，待一二时，用肉汤调无忧散5-7钱，取出小虫一二万为效。后用皂角1斤（不蛀者，去皮弦及子，捶碎），水4碗，煮至1碗，用生绢滤去滓，再入苦参末搅，熟稀面糊膏子，取出放冷，后入：何首乌2两，防风1两半，芍药5钱，人参3钱，当归1两（焙），为细末，与皂角膏子为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
