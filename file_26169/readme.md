序号：26177
名称：朱砂膏
组成：附子1个（重5钱半，平正紧实者，生，去皮脐，取半两，不须用太大者），天南星1个（去皮脐，不得用小者），半夏2钱半（取中形圆正好者，汤洗7次，去滑，焙干，生用），天麻2钱半（明净好者，去两头，取中心，切），琥珀（研）2钱半，全蝎1分（生），赤足蜈蚣半两（去头尾足，炙，取2寸），白僵蚕半两，朱砂（光明者，研如粉）半两，麝香1分（净，研）。
出处：《幼幼新书》卷十引《吉氏家传》。
主治：一切惊风，诸痫，暗风，破伤，惊涎，心气不足，或久伏惊气，尸厥发即昏昧，涎壅；及因惊亡魂失魄，举动惊怕，梦魇，或歌哭不避亲疏，狂走不宁，中风缓弱顽痹，小儿慢惊风。
加减：None
功效：None
用法用量：每患用少许生姜自然汁化药1皂大，次用温酒调下；小儿生姜、薄荷汤化下豆大。
制备方法：上为细末，入研者朱砂、琥珀、麝香同研匀，炼蜜放冷为膏，密器收。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
