序号：76095
名称：蛜(虫祁)丸
组成：全蝎（微炒）1分，白僵蚕1分，雄黄（研）1分，白附子（炮）1分，天麻（锉碎）1分，朱砂（细研）1分，麝香（细研）1分，天南星（湿纸裹，炮）1分，半夏（汤浸5-7次，去黑脐，生姜3分，取汁煮令尽，焙干）1分，乌梢蛇（尾穿100足钱者佳，于项后粗处取）7寸（酒浸7日，去皮骨，慢火炙黄）。
出处：《幼幼新书》卷十二引《赵氏家传》。
主治：小儿因吐泻后虚风，眼涩多睡，潮搐惊痫；及丈夫、妇人一切虚风，头旋眼黑，恶心吐逆，筋脉紧缓，手足麻木，身体疼痛，精神不爽。
加减：None
功效：None
用法用量：3岁以下每服7丸，5岁以下10丸，5岁以上15丸，荆芥汤送下，不拘时候；大人丸如绿豆大，每服20丸，荆芥汤茶送下。如急用，即以20丸研碎，荆芥、生姜浓煎汤化下。
制备方法：上为细末，生姜汁煮糊为丸，如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
