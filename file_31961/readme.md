序号：31970
名称：应病散
组成：人参半两，白术（麦麸炒）半两，白茯苓（去皮）半两，白薇（去芦，用根）半两，白芷半两，京芍药半两，川芎半两，玄胡索（去皮）半两，桂（不见火）半两，大当归（酒浸，焙，去尾）半两，赤石脂（火煅红）半两，牡丹皮（去木骨）半两，藁本（去头）半两，甘草3钱，没药（不见火）2钱，沉香3钱（不见火）。
出处：《医方类聚》卷二一二引《经验良方》。
主治：妇人胎前产后百病，诸般崩漏，产后发热。
加减：骨蒸热，入童子小便1盏煎服，不能饮酒者，不用小便，炼蜜同生姜自然汁调服。
功效：None
用法用量：每服2钱，入炼熟蜜大半匙，热酒调服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
