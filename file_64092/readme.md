序号：64108
名称：猪肚黄连丸
组成：豮猪肚1具（以童便煮令熟，细切，焙干，捣为末），知母（锉，焙）2两，芜荑仁（炒）2两，紫菀（去苗土）1两，大黄（锉，微炒）1两，鳖甲（去裙襕，醋浸，炙）1两，槟榔（煨）1两，苍术（米泔浸，切，焙）1两，百部（锉，焙令干）1两，地骨皮1两，黄芩（去黑心）1两，桔梗（炒）1两，贝母（去心）1两，柴胡（去苗）1两，黄连（去须）1两，龙胆（去芦头）1两，人参1两，白茯苓（去黑皮）1两，黄耆（锉）1两。
出处：《圣济总录》卷九十三。
主治：骨蒸虚劳，传尸肺痿，咳嗽不止，困重羸瘦，壮热，或即憎寒，腹内冷胀，四肢烦疼，饮食无味，头疼口干，涕唾稠粘，渴不止。
加减：None
功效：None
用法用量：每服20丸，空腹温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
