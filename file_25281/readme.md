序号：25289
名称：百子建中丸
组成：香附1斤（分作4份：1份童便浸7日，1份酒浸7日，1份泔浸7日，1份盐水浸7日，各炒香），大艾叶4两（米泔浸7日，将米泔慢火煮半日，焙干为末），砂仁5钱，淮熟地（酒浸）3两，白芍药3两，玄胡索1两5钱，五味子5钱，杜仲（酒炒）1两，阿胶（炒）1两5钱，白术1两（麸炒）。
出处：《宋氏女科》。
主治：妇人久冷，赤白带下，肚腹疼痛，经水不调，四肢无力，久鲜子息。
加减：如妇人肥胖者，加陈皮、半夏各1两。
功效：温中暖脐，调经，开郁开胃。
用法用量：每服80丸，空心用淡酣汤送下。服至半月必有孕矣。
制备方法：上为末，用壬子日好米浴打粳米面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
