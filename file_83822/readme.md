序号：83840
名称：乌蛇散
组成：乌蛇肉1两（酒拌炒令黄），天南星1两（炮裂），天雄1两（炮裂，去皮脐），土蜂儿1两（微炒），干蝎半两（微炒），桑螵蛸半两（微妙），赤箭1两，麻黄1两（去根节），羚羊角屑半两，薏苡仁1两，酸枣仁3分，柏子仁半（3）分，芎1两，桂心半两，当归3分（锉，微炒），朱砂半两（细研，水飞过），麝香1分（细研）。
出处：《圣惠》卷六十九。
主治：妇人风痹，手足顽麻，筋脉抽搐，口眼不正，言语謇涩。
加减：None
功效：None
用法用量：每服1钱，食前以温酒调下。
制备方法：上为细散，入研了药令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
