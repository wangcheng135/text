序号：36827
名称：固真丸
组成：鹿角1对（用杀者，去顶骨，截作3寸长短，解作2半，秤见斤两，用河水浸7日，每日1换新水，浸候日数足，每角1斤，入好黄蜡1两，用瓷缸子纳，以河水用桑柴火煮3伏时，不得住火，如水少，渐渐添，浸着角后煮得角软，削去黑皮，只取霜，晒干，将煮角汁再以慢火熬成清胶）。
出处：《御药院方》卷六。
主治：肾经虚损，真元不足。
加减：None
功效：暖丹田，补真气，活血脉，健筋骨，添精固气，延年助阳。
用法用量：每服50丸，空心用温酒吞下，渐渐加至100丸亦得。若不吃酒，以清米饮吞下亦得。
制备方法：每用鹿角霜1斤，入上好者雪白茯苓5两（刮去黑皮）同捣罗为细末，将鹿角胶水搜和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：鹿角霜丸（《普济方》卷二二四）、固精丸（《万氏家妙方》卷四）。
