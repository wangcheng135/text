序号：51758
名称：钟乳丸
组成：钟乳粉3两，五味子3分，桂心3分，石菖蒲3分，鹿角胶1两（捣碎，炒令黄燥），白术3分，诃黎勒1两半（煨，用皮），木香3分，人参1两（去芦头），天门冬1两半（去心，焙），白茯苓1两，黄耆1两（锉），熟干地黄1两，川椒3分（去目及闭口者，微炒出汗）。
出处：《圣惠》卷二十六。
主治：气极。肺脏虚寒，腹胁胀满，呼吸短气，咳逆胸痛，四肢洒浙，皮毛干焦，肌体羸瘦，面无光泽。
加减：None
功效：None
用法用量：每服30丸，空心及食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌羊血、鲤鱼。
附注：None
