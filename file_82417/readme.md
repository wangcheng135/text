序号：82433
名称：薯蓣丸
组成：薯蓣1两，石斛2两（去根，锉），牛膝2两（去苗），鹿茸2两（去毛，涂酥，炙微黄），白茯苓2两，五味子2两，续断1两，巴戟2两，山茱萸2两，人参2两（去芦头），桂心2两，熟干地黄2两，泽泻2两，杜仲2两（去粗皮，炙微黄，锉），蛇床子1两，远志2两（去心），菟丝子1两（酒浸1宿，晒干，别捣为末），覆盆子1两，肉苁容2两（酒浸1宿，刮去皱皮，炙干）。
出处：《圣惠》卷二十六。
主治：男子五劳七伤，久虚损羸瘦，腰脚无力，颜色蒌瘁，下元衰惫，脾胃气寒，饮食无味，诸虚不足。
加减：None
功效：补脏腑，利腰脚，壮元气，充骨髓。
用法用量：每服30丸，空腹及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
