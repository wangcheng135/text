序号：33850
名称：苁蓉丸
组成：肉苁蓉（洗净，切，焙）1两，山芋1两，续断1两，人参1两，独活（去芦头）1两，牛膝（酒浸，切，焙）1两，山茱萸1两，陈曲（炒黄）1两，杜仲（去皴皮，涂酥炙）1两，巴戟天（去心）1两，菟丝子（酒浸1宿，烤干，别捣）1两，熟干地黄（焙）2两，桑寄生（炒）2两。
出处：《圣济总录》卷一○二。
主治：肝肾气虚，眼目昏暗，视物不明；眼青盲，并无赤痛，但不见物。
加减：None
功效：补肾。
用法用量：每服20丸，加至30丸，空心盐酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：补肾还睛丸（《圣济总录》卷一一二）。
