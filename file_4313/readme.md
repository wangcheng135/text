序号：4313
名称：九味仓廪汤
组成：潞党参1钱至钱半，羌活8分至1钱，薄荷1钱至钱半，茯苓2钱至3钱，防风1钱至钱半，前胡1钱至钱半，苦桔梗1钱至钱半，清炙草6分至8分，陈仓米3钱至4钱。
出处：《重订通俗伤寒论》。
主治：None
加减：None
功效：益气发汗。
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：此方妙在参、苓、仓米益气和胃，协济羌、防、薄、前、桔、甘，各走其经以散寒，又能鼓舞胃中津液上输于肺以化汗，正俞氏所谓“藉胃汁以汗之”也。凡气虚者，适感非时之寒邪，混厕经中，屡行疏表不应，邪伏幽隐不出，非藉参、苓、米辅佐之力，不能载之外泄也。
用药禁忌：None
附注：None
