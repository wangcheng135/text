序号：22734
名称：却老苁蓉丸
组成：肉苁蓉（酒浸，切，焙）2两，山芋1两1分，五味子（炒）1两1分，菟丝子（酒浸3日，焙干，别取末）1两，赤石脂（研）1两，白茯苓（去黑皮）1两，泽泻1两，熟干地黄（焙）1两，山茱萸（焙）1两，巴戟天（去心）1两，覆盆子（去梗）1两，石斛（去根）1两。
出处：《圣济总录》卷一八六。
主治：肾脏虚损，丹田风冷。
加减：None
功效：补真脏气，调顺阴阳，和胃气，进饮食。
用法用量：每服20-30丸，空心食前温酒送下；粟米饮亦得。
制备方法：上为细末，酒煮面糊，入蜜少许为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
