序号：48956
名称：绛雪散
组成：木通（锉）2两，桔梗（锉，炒）2两，槟榔2两，枳壳（去瓤，麸炒）1两半，犀角（镑）1两半，柴胡（去苗）2两，升麻2两，木香2两，赤茯苓（去黑皮）2两，桑根白皮（锉）4两，山栀子仁4两，桂（去粗皮）2两，人参2两，苏枋木5两，朴消（研）1斤，丹砂（研）1两，麝香（研）1分，诃黎勒（去核）5枚。
出处：《圣济总录》卷一二二。
主治：热结喉间，连颊肿不消，心膈烦满。
加减：None
功效：None
用法用量：每服1钱或2钱，食后临卧，以冷蜜汤调下。
制备方法：上除朴消、丹砂、麝香外，各锉细，以水2斗，于银器内慢火熬至7升，以生绢滤去滓，再煎至5升，下朴消，以柳木篦搅，勿住手，候稍凝，即去火，倾入盆中，将丹砂、麝香末拌令匀，瓷器盛之，勿令透气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
