序号：4005
名称：日用仙酥丹
组成：莲肉（去皮心）半斤，柏子仁（去壳）半斤，杏仁（去皮尖）6两（捣），胡桃仁（去皮）4两（捣），枣肉半斤（煮，去皮，捣），砂仁2两（碾末），酥油半斤，白蜜半斤。
出处：《扶寿精方》。
主治：None
加减：None
功效：补百损，除百病，返本还童。
用法用量：每服3匙，空心卧时温酒1-2杯送之。
制备方法：文火炼蜜，次入酥油搅匀，再数沸，方入莲、柏末，又数沸入桃、杏、枣膏，慢熬半炷香，量诸味皆热，入砂仁末搅匀，用瓷罐数个贮，置冷水中浸1日，出火气，油纸或脂膜封口。
临床应用：None
各家论述：None
用药禁忌：None
附注：日用仙酥汤（《济阳纲目》卷六十八）。
