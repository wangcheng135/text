序号：7753
名称：圣药丸
组成：川芎1两，当归1两，瓜蒌根6钱（生用），蝉壳5钱，川椒7钱（去子），蛇蜕皮3钱，甘草（浸洗）、蔓荆子2两，楮实子5钱，密蒙花1两，枸杞子1两，木贼2两（去节，用童便浸1宿），地骨皮1两，薄荷叶5钱，羌活1两，白蒺藜1两半（炒），干菊花1两，川黄连3两，荆芥穗1两。
出处：《医方类聚》卷七十引《经验秘方》。
主治：远年近日诸般内障，风暗气血，一切眼疾。
加减：None
功效：None
用法用量：每服1丸，细嚼，食后服之，日进2服。有翳者，清米泔水送下；睛暗者，当归汤送下；气昏瘴者，木香汤送下；如妇人血昏者，当归、薄荷汤送下。
制备方法：上为细末，炼蜜为丸，一两分作十丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中甘草用量原缺。
