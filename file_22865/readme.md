序号：22873
名称：西洋十宝散
组成：真血竭1钱6分，明雄黄4钱，上红花4钱，净儿茶2分4厘，辰州砂1钱2分，净乳香1钱2分，当归尾1两，净没药1钱4分，当门子3厘，梅片1分2厘。
出处：《外科全生集·新增马氏试验秘方》。
主治：跌打损伤，金疮。
加减：None
功效：None
用法用量：刃伤并各器械伤，皮破出血者，以末药掺上包固，不可见风；跌打损伤，皮肉青肿，未破者，用陈醋调敷患处；内伤骨碎，或骨已断折，先将骨节凑准，用陈醋调敷患处，以纸包裹，外加老棉絮包好，再用薄板夹护，将绳慢慢系紧，不可移动，药性到骨自接，须静百日，勿犯房事，犯则必成残疾；刀伤深重，未致透膜者，先用桑皮纸线缝好，多掺药末于上，以活鸡皮急急调护如前骨损养法即愈；跌打昏迷不醒，急用1钱，同陈酒冲服。
制备方法：上为细末，入乳钵研至无声，收入瓷瓶，黄蜡封口，勿令泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
