序号：83600
名称：乌须明目丸
组成：女贞实（酒拌，9蒸9晒，净末）1斤，甘菊花12两，何首乌（赤白各半，净）2斤（如法蒸晒），桑叶1斤，牛膝（酒蒸）1斤，怀生地（酒洗净）2斤（如法蒸晒），甘枸杞（去枯者）1斤，乳拌茯苓酥1斤，麦门冬（去心）1斤半，槐角子10两，苍术（蜜、酒浸，蒸，晒）12两，人参1斤（人乳拌，烘干），山茱萸肉（酒蒸）12两。
出处：《广笔记》卷二。
主治：须发早白，目视昏花者。
加减：None
功效：None
用法用量：每日服5钱，白酒送下，1日2次。
制备方法：乌饭子之嫩者取汁熬膏，每斤加蜜半斤为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌白莱菔、牛肉、牛乳、蒜、桃、李、雀、蛤。
附注：None
