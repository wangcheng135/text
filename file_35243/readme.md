序号：35253
名称：乳香搜风丸
组成：升麻4两，牛蒡子4两，胡麻4两，白鲜皮4两，连翘4两，豨莶草4两，苦参4两，桑寄生4两，当归4两，怀生地4两，秦艽4两，枸杞子4两，槟榔4两，何首乌4两，鳖甲1个（重1两佳），凌霄花1两，川芎1两，大风肉（去油，同川乌煮）1两，防己1两，虎胫骨（酥炙）2两，陈皮2两，牛膝2两，甘菊花2两，紫草2两，海风藤2两，木瓜2两，杜仲1两2钱，甘草1两5钱，白芷1两2钱，薏苡仁6两，芝麻2合，乳香3斤（用河水添换煮4、5次为度，又用川乌10两捣碎，煎汁1钵，入乳香煮至汁干为度，再用防风4两，石菖蒲4两，荆芥4两，苍术4两，煎汁1钵，入乳香煮至汁尽为度，又用石菖蒲4斤，捣汁1钵，入乳香煮干为末，每制乳香1斤，配药1斗）。
出处：《疡医大全》卷二十八。
主治：血痹（饣兹）（饣巴），痛疯半肢，蛓毛泥壁，水冷疯，漏蹄雁来鸡爪疯。
加减：None
功效：None
用法用量：初服1钱5分，3日后服2钱，又3日服2钱5分，再3日服3钱为止。服至20日，伸指酸麻，1月后容颜光洁，服至100日而愈。
制备方法：用鲜皂角刺3斤捣碎，水煎去滓熬成膏，入前药为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
