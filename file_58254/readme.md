序号：58269
名称：真方五色丸
组成：青丸子：青黛（别研）半两，天南星（生姜制）半两，巴豆霜半钱。红丸子：朱砂（水飞）半两，半夏（汤洗7次，生姜制）半两，巴豆霜半钱。白丸子：白附子（生）半两，寒水石（煅）半两，巴豆霜半钱。黑丸子：五灵脂（炒）半两，全蝎（炒）半两，巴豆霜半钱，黄丸子：大黄（煨）半两，郁金半两，巴豆霜半钱。
出处：《普济方》卷三九一引《保婴方》。
主治：小儿一切所伤，痰涎壅塞，胸膈不利，乳食不消，变生癖积，胁肋片硬，按之疼痛，及一切急慢惊风，发搐。
加减：None
功效：None
用法用量：每1岁服5丸，2-3岁儿10丸至15丸，乳汁送下；5-6岁如麻子大，每服15丸至20丸，温生姜汤送下；急惊风，煎金银薄荷汤送下；慢惊风，熬生姜全蝎汤送下，食后、临卧日进2服，不拘时候。
制备方法：上件前五色药，各另为细末，入巴豆霜半钱，同研令匀，水面糊为丸，如粟米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：真方五色丸子（《永类钤方》卷二十）、五色丸（《诚书》卷十）。
