序号：71300
名称：紫石门冬丸
组成：紫石英3两，钟乳石3两（鹅管通明者，2味各7日研之，得上浮即熟），天门冬3两，当归2两，芎2两，紫葳2两，卷柏2两，肉桂2两，干地黄2两，牡蒙2两，禹余粮（煅，醋淬）2两，石斛2两，辛夷2两，人参1两半，桑寄生1两半，续断1两半，细辛1两半，厚朴（姜制）1两半，干姜1两半，食茱萸1两半，艾叶1两半，白薇1两半，薯蓣1两半，乌贼骨1两半，甘草（炙）1两半，柏子仁1两。
出处：《准绳·女科》卷四。
主治：None
加减：None
功效：求子。
用法用量：每服10丸，3日渐增至30丸，酒送下，以腹中热为度，比来服者不至尽剂，即有娠。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：此方旧用乌头、牡丹、牛膝，据药证此三药俱堕胎，求子药中用之，盖胎未著之时，若服之已著，已著而未觉，服之未已，反为害也，今悉去之。增钟乳、艾叶、白薇，兹无疑矣。
用药禁忌：禁如药法。
附注：None
