序号：69704
名称：琥珀膏
组成：琥珀1分（细研），雄黄1分（细研），朱砂1分（细研），丁香1分，木香1分，当归1分，白蔹1分，芎1分，木鳖子1两（去壳），乱发1两（烧灰），生地黄2两（切），垂柳枝3合（锉），槐枝3合（锉），松脂1两，黄丹5两，清麻油15两。
出处：《圣惠》卷六十三。
主治：一切恶毒疮肿，坚硬疼痛。
加减：None
功效：None
用法用量：每用时，看疮肿大小，以故帛上涂贴，1日2度换之。
制备方法：上丁香、木香、当归、白蔹、芎为细散，以琥珀、雄黄、朱砂相和，细研，候膏成，乃下余药，并以油浸1宿，净铛内煎炼，以地黄色黑为度，绵滤去滓澄清，却于铛内慢火熬药油，相次入黄丹，以柳木篦不住手搅令色黑，取少许滴于水内，看硬软得所，入琥珀等搅令匀，倾于不津器内盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
