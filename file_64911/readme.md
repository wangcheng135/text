名称：维阳感召汤
组成：人参、天冬、麦冬、熟地、生地、茯神、犀角（镑）、羚羊角（镑）、琥珀（研）、龙齿（煅）、珍珠（研）、龟版（炙）、龙眼肉。
出处：《证因方论集要》卷一。
主治：阴不维阳，达旦不寐。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：《经》曰：阳不入于阴，则不能寐。人参、天冬、二地乃三才丹，以补手足太阴；麦冬、茯神入心，所谓热淫于内，以清胜之；犀角、羚羊，兽类之灵，凉心清肝；龙齿、龟版，介类之灵，镇心潜阳；琥珀，松脂入土而成实，珍珠，老蚌感月而结胎，故能安魂魄，定心神；龙眼肉，甘以悦脾。此方专用纯甘之味，复以物之灵，引人之灵，两相感召也。
用药禁忌：None
附注：None
