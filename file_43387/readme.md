序号：43402
名称：养荣丸
组成：白术（土炒）1两，黄耆1两，芍药1两，远志1两，当归（酒洗）1两，山药1两，熟地黄1两，陈皮8钱，人参1两，白茯苓2两，山茱萸半两，生地黄半两。
出处：《医统》卷四十八。
主治：脾肺气虚，荣血不足，惊悸健忘，寝汗发热，食多无味，身体疲瘦，色枯气短，毛发脱落，小便赤涩；亦治发汗过多，身振肢摇，筋惕肉瞤。
加减：咳嗽，加麦门冬、紫苑、贝母、款冬花各1两；热，加黄柏、知母各1两；遗精、带浊，加牡蛎1两，龙骨半两；吐血血腥，加牡丹皮、赤芍药各1两。
功效：None
用法用量：每服80丸，盐汤送下；冬月酒送下。
制备方法：上为细末，用鸭1只取血，入炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
