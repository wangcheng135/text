序号：48540
名称：神效膏
组成：木通（锉）1分，甘草（炙）1分，当归（炙，锉）1分，白芷1分，防风（去叉）1分，细辛（去苗叶）1分，栀子仁1分，黄连（去须）1分，黄芩（去黑心）1分，垂柳枝（锉）2两，铅丹6两，蜡半两，清油1斤。
出处：《圣济总录》卷一八三。
主治：痛疽发背，热毒气结，肿痛坚硬。
加减：None
功效：止疼痛。
用法用量：故帛上摊贴，1日2次，以愈为度。
制备方法：上除丹、蜡、油外，锉碎，先以油内浸药1宿，于火上煎，候白芷赤黑色，绞去滓再煎，即下丹、蜡，柳篦搅，候变黑色，滴水中成珠子，软硬得所，瓷盒盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
