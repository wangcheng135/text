序号：57268
名称：消暑清心饮
组成：香薷（去老梗）1两，泽泻（去粗皮）1两（炒熟，锉，去壳1斤，用生姜1斤切碎，煮烂拌匀，酿1宿，焙干），净黄连、羌活、猪苓（去皮）、厚朴5钱（去粗皮，锉碎，每1斤用生姜1斤切薄片，烂杵拌匀，酿1宿，慢火炒干），白术半两，干葛半两，赤茯苓（去皮）半两，升麻半两，川芎半两，甘草1两2钱。
出处：《活幼心书》卷下。
主治：小儿暑风。伏热中暑，烦躁作渴，神气不清，及有惊搐。
加减：None
功效：None
用法用量：上（口父）咀。每服2钱，水1盏，煎7分，不拘时候，带凉服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
