序号：82726
名称：中正丸
组成：紫河车1具，鹿角胶2两，龟胶3两，玄参3两，熟地8两，山茱萸4两，地骨皮5两，人参2两，白术5两，白芍5两，炒枣仁3两，枸杞子3两，麦冬3两，人乳2碗（浸熟地晒干），砂仁5钱。
出处：《石室秘录》卷二。
主治：劳瘵，虚劳，病入膏肓骨髓者。
加减：None
功效：None
用法用量：每日半夜白滚水送下5钱。此方不热不寒，可以长服。
制备方法：上药各为末，制以成丸。
临床应用：None
各家论述：None
用药禁忌：宜断绝色欲，口淡滋味，心戒贪嗔。
附注：晚服此丸方，宜朝服如下汤方：熟地九钱，山茱萸四钱，山药三钱，丹皮二钱，泽泻二钱，茯苓三钱，北五味一钱，麦冬二钱，芡实五钱。水煎服。服以上汤、丸，非数百剂不能奏效。
