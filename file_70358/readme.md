序号：70374
名称：犀角丸
组成：犀角屑半两，天雄半两（炮裂，去皮脐），鬼臼半两（去须），桂心半两，莽草半两（微炙），珍珠半两（细研），川大黄半两（锉碎，微炒），雄黄半两（细研），蜈蚣5节（微炙），贝齿5枚（烧赤），川乌头半两（炮裂，去皮脐），麝香1分（细研），巴豆15枚（去皮心，研，纸裹压去油），羚羊角屑半两。
出处：《圣惠》卷五十六。
主治：诸尸百病恶气，腹内疼痛，毒肿。
加减：None
功效：None
用法用量：每服1丸，空腹以粥饮送下，渐增至2丸。卒腹痛飞尸，服大豆大1丸；若恶气肿，以醋和涂之，以囊盛之，男左女右系之臂上，祛邪气极效。
制备方法：上为末，入研了药及巴豆，都研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
