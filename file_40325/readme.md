序号：40335
名称：茄和散
组成：枇杷叶（去毛，姜汁炙）1两，白茯苓（去皮）1两，砂仁（去皮）1两，薏苡仁（炒）1两，丁香1两，白豆蔻（去皮）1两，人参（去芦）1两，白术（炒）2两，桑白皮（炒）5钱，沉香5钱，五味子5钱，槟榔（炒）2钱半，青皮（去白）2钱半，谷芽（炒）2钱半，藿香2钱半，杜仲（去皮，姜、酒涂炙）2钱半，随风子2钱半，石斛（酒炒）2钱半，大腹子2钱半，陈皮（去白）2钱半，神曲（炒）2钱半，木香7分半，甘草（炙）1两5钱。
出处：《摄生众妙方》卷五。
主治：脾胃不和，胸膈痞闷，气逆生痰，不进饮食，五噎五膈。
加减：五噎，加干柿饼1枚；膈气吐逆，加韭白3寸，大枣5枚。
功效：None
用法用量：上(口父)咀。每服3钱，水1钟，加生姜3片，大枣1个，煎至7分，去滓温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
