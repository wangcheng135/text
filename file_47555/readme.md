序号：47570
名称：神仙不老丸
组成：人参（新罗者，须是团结、重实、滋润，去芦头，刷洗净，焙干，薄切，焙燥）2两，川牛膝（长3-4尺而滋润者，去苗，刷洗净，焙干，寸截，用酒浸1宿，焙燥）1两半，川巴戟（色黑紫，沉重大而穿心者佳，若色带黄而浮轻者非，洗刷净，焙干，细切，酒浸1宿，焙燥）1两，川当归（大茎其稍如马尾状，滋润辛温芳香者，去芦头，刷净洗，焙干，细切，用酒浸1宿，焙燥）2两，杜仲（截之多丝者，削去粗皮，只取其肉，如去肉桂之法，然后刷洗净，焙干，横理锉碎如豆，用麦面炒令丝断色黄，去面别磨）1两半，地黄（冬节前采，以水浸沉者为是，以其肥者捣取汁，浸令浃，蒸毕，焙干，色黑味甘为度，用时以生干、熟2种焙干，酒浸1宿，滤出，竹刀细切，焙干，各称1两，忌铁器）2两，菟丝子（小如芥子，极坚硬者佳，大而轻者非，用新布撮起，挪洗，焙干，以酒浸1宿，滤出，将温汤淋去酒，焙燥，别磨）2两，柏子仁（色红而滋润者，去壳取仁，细研，临时和入众药）1两，石菖蒲（紧细节密者，去毛，刷洗净，焙干，细切，焙燥）1两，枸杞子（色白而肥润，去蒂洗净，焙干，用酒浸1宿，焙干）1两，地骨皮（色黄，入手轻者佳，重者非，略去浮皮，净洗，焙干，薄切，焙干）1两。
出处：《寿亲养老》卷四。
主治：None
加减：None
功效：驻颜，乌髭发，大能温养荣卫，补益五脏，和调六腑，滋充百脉，润泽三焦，活血助气，添精实髓。
用法用量：每服70丸，空心、食前、临卧以盐酒、盐汤任下。
制备方法：上不可晒，只用慢火焙，若太燥则又失药气，只8分干，即于风前略吹，令冷热相激，便十分燥，取净称份量，磨如细散，火日炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌食葱白、薤白、芦菔、豆粉及藕、诸般血。
附注：神仙不老丹（《济阳纲目》卷六十四）。
