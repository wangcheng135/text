序号：67409
名称：黄丹膏
组成：黄丹（淘洗7次净）1两5钱，黄连5钱，川芎5钱，海螵蛸3钱，轻粉、朝脑、水龙骨。
出处：《疡科选粹》卷五。
主治：臁疮。
加减：None
功效：None
用法用量：第1日用2贴，第2、第3、第4、第5日用1贴，第6、7日以后，3日换1贴；尚存如钱大一处不能收口，以松香4两，葱头1把共捣烂，置于碗中，以滚白水冲下，良久去水，取药捻成饼，贴疮上；尚有针细1孔，流水不完，取蒜头、葱头内第2层白皮贴3-4日即好。
制备方法：上为极细末，以生桐油调为膏，夹纸做成，着肉面针刺数十孔。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中轻粉、朝脑、水龙骨用量原缺。
