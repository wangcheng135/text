主治：胃中血不足，燥火用事，或心烦口渴，或呕吐黄水，或噎食不下，或食下吐出，或消谷善饥，或大呕吐血，或大便难，或食亦及肺燥者、肾热者、挟肝火者。
加减：None
功效：None
用法用量：贴上、中、下三脘。
制备方法：共用油20斤，分熬丹收。再入生石膏80两、寒水石4两、青黛1两，牡蛎粉、元明粉各2两、牛胶4两，酒蒸化，俟丹收后，搅至温，以一滴试之，不爆方下，再搅千余遍，令匀，愈多愈妙。勿炒珠，炒珠无力，且不粘也。
