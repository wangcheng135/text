序号：53115
名称：大补阴膏
组成：茯神2两（去皮心），远志2两（去梗，炒干用），人参5钱（去芦），白术4两（切片，水洗，去油，晒干），茯苓2两（去皮），橘红1两5钱（去白），贝母1两5钱（姜汤煮过），甘草3钱（炙，去皮），紫菀1两（洗去土），阿胶1两（蛤粉炒成珠），五味子5钱，当归身3两（酒洗），生地黄1两5钱（酒洗），白芍药2两（炒），熟地黄1两5钱（酒洗，蒸9次，晒9次），天门冬1两5钱（去心），麦门冬1两5钱（去心），菟丝子2两（水洗去土，晒干），枸杞子3两（蒸，焙干），黄柏2两（去皮，盐水炒干），山茱萸2两（汤浸），知母1两（盐水炒干）。
出处：《遵生八笺》卷十七。
主治：None
加减：None
功效：安心神，健脾胃，滋肺金，补元气。
用法用量：每服3茶匙，空腹白滚汤调下。
制备方法：上切，用井花水24碗，入鲜姜4两2钱，胡桃肉、圆眼肉、枣肉、莲肉各24个，乌梅肉12个，春浸半月，夏不浸，秋浸1日，冬浸1日夜，于静室内，用炭火煎至5碗，去药滓，用好蜂蜜24两，煎1滚，用纸渗去面上沫，入前药同煎至滴水不散为度；用瓷罐盛，白纸封口，放水盘中，露罐口7日，去火毒，取出。
临床应用：None
各家论述：None
用药禁忌：忌食羊肉。
附注：None
