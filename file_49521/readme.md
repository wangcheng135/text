序号：49536
名称：茜根散
组成：茜根1两，黄芩1两，阿胶（蛤粉炒）1两，侧柏叶1两，生地黄1两，甘草（炙）半两。
出处：《医方类聚》卷八十五引《济生》。
主治：衄血不止，心神烦闷；吐血衄血，错经妄行，并妇人月信不止；阴虚衄血。
加减：None
功效：None
用法用量：上（口父）咀。每服4钱，水1盏半，加生姜3片，煎至8分，去滓温服，不拘时候。
制备方法：None
临床应用：None
各家论述：《证因方论集要》：肾阴虚则阳偏胜，故载血上行而致衄。是方也，阿胶能补虚，黄芩能养阴，甘草能缓急，茜根、侧柏、生地则皆去血中之热，能生阴于火亢之时者也。
用药禁忌：None
附注：茜根汤（《笔花医镜》卷二）。《玉案》本方用法：加童便半酒杯，温服。
