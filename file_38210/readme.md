序号：38220
名称：松黄颐寿丹
组成：松香1斤（嫩白莹净者，碾为末，筛过去滓，用新汲水10余碗，砂锅内桑柴火煮一炷香，不住手搅，冷定倾出苦水，仍换新水，更煮更搅，如此14-15次，直待水煮不苦为度，再用白酒4-5碗，亦煮一炷香，冷定，取出晒干，碾为细末），熟地黄半斤（淮庆肥大者，拣去不黄不用，浸，蒸烂，捣成膏），乌梅肉6两（安吉者佳，焙干，碾为末）。
出处：《遵生八笺》卷十七。
主治：None
加减：None
功效：益寿。
用法用量：每服30-50丸，食前茶汤、白酒任下。
制备方法：上为末，如干散难丸，加酒打面糊少许，和之易丸为度，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌豆腐，制药时不可犯铁器。
附注：None
