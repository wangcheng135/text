序号：39026
名称：泽兰丸
组成：泽兰叶（炒）1两1分，黄耆（细锉）1两半，藁本（去苗土）1两，当归（锉，焙）1两半，白芷1两，防风（去叉）1两半，芍药1两半，芎1两，桂（去粗皮）3分，柏子仁1两，细辛（去苗叶）半两，麦门冬（去心，焙）2两，熟干地黄（焙）1两1分，甘草（炙，锉）1两，五味子1两，石膏（研如面）1两3分。
出处：《圣济总录》卷一六四。
主治：产后诸疾，愈后虚羸无力。
加减：None
功效：补益。
用法用量：每服20丸，渐加至30丸，空腹以酒送下，1日2次。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
