序号：47768
名称：神仙金不换膏
组成：僵蚕7钱，青皮7钱，独活7钱，川附子7钱，防风7钱，生草乌7钱，何首乌7钱，白鲜皮7钱，秦艽皮7钱，穿山甲7钱，半夏7钱，生川乌7钱，青风藤7钱，马钱子7钱，桃仁7钱，生虎骨7钱，天麻7钱，归尾7钱，良姜7钱，威灵仙7钱，生姜2具，韭菜2具，生蒜2具，青葱2具。
出处：《全国中药成药处方集》。
主治：手足麻木，腰腿疼痛及跌打损伤。
加减：None
功效：散风，活血，止痛。
用法用量：临用化开，摊贴患处。
制备方法：用香油7斤半将上药炸枯去滓，熬之滴水成珠，加章丹3斤4两凉透，再将细药（乳香6钱、没药6钱、公丁香5钱、官粉1两3钱、冰片5分、木香8分，共研极细末）搅入膏内。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
