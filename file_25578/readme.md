序号：25586
名称：朱砂三棱丸
组成：石三棱（酒浸）半两，荆三棱（酒浸）半两，鸡爪三棱（酒浸）半两，蓬术（醋浸）半两，枳壳（去瓤，麸炒）半两，川楝子（去皮、子）半两，茴香（微炒）半两，青皮（去白）半两，当归（去芦）半两，槟榔7钱半，丁香2钱，木香3钱，巴豆（去壳、麸炒至深黄色为度）18粒（巴豆斟酌量用或只用巴豆炒三棱、枳壳、蓬术3味亦得，炒至黄色，去巴豆），肉豆蔻7钱半，朱砂5钱（水飞，1半入药，1半为衣）。
出处：《普济方》卷一七四。
主治：痃癖，气块留滞，一切内伤。
加减：None
功效：None
用法用量：每服15丸，生姜汤送下；温水亦得。
制备方法：上为细末。除朱砂外，生姜研汁，用面打糊为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
