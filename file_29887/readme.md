序号：29895
名称：十味养荣汤
组成：熟干地黄（酒浸）2两半，黄耆（蜜炙）2两半、五味子（去枝）1两，肉桂（去粗皮，不见尖）1两，牡丹皮（炒）1两，白芍药（炒）1两，白茯苓1两（去皮），当归（去芦，酒浸）1两半，川芎1两半，甘草7钱（炙），（一方加人参1两、地骨皮1两）。
出处：《魏氏家藏方》卷十。
主治：妇人劳疾，脏腑血气不足，冲任虚损，脐腹(疒丂)痛，寒热往来，心忪恍惚，忧虑不乐，面少光泽，月水不调，颜色多变，气道壅塞，体倦好睡。
加减：None
功效：None
用法用量：上(口父)咀。每服5钱，水1盏半，加生姜3片，枣子1枚，煎至7分，去滓，食前空心服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
