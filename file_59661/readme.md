序号：59676
名称：调化丸
组成：生黄耆8钱，白芍（炒）6钱，当归（酒洗）6钱，牛蒡（炒）7钱，连翘7钱（去心），黄芩（炒）8钱，川连8钱（酒炒），防风3钱，荆芥3钱，桔梗4钱，木通4钱，前胡6钱，紫草茸6钱，红花（酒洗）4钱，大生地（用新瓦烤酥，焙干即时为末）4钱，人中黄4钱，蝉退3钱，楂肉8钱，丹皮5钱。
出处：《种痘新书》卷三。
主治：痘出稠密红紫，潮热不退，烦闷，狂言，一切毒盛之症。
加减：弱甚者，加人参。
功效：清热解毒，扶元活血。
用法用量：None
制备方法：凡酒洗者，须先晒干，方可合诸药同焙，共合为末，米糊为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
