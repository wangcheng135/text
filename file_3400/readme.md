序号：3400
名称：太乙灵应膏
组成：玄参半两，生地黄半两，大黄2钱，黄柏3钱，黄芩2钱，槐花2钱，红花2钱，白芷半两，官桂半两，血竭半两，当归须1两，地榆2钱，川羌活4钱，乳香半两，没药3钱，赤芍药半两，杏仁21枚（去尖），防风3钱，猪牙皂3枚，白胶香4两，刘寄奴4两。
出处：《普济方》卷三一三。
主治：诸般恶疮、恶毒及杖疮。
加减：None
功效：None
用法用量：摊贴患处。并可内服1丸，每丸如樱桃大，蛤粉为衣。
制备方法：上锉散，以香油1斤浸，春5、夏3、秋7、冬10日。浸毕，就文武火熬，待药枯黑如炭，然后退火，令其自冷。将粗布一幅滤过，枯药不用。将药油再入锅内熬，先以黄丹10两，新瓦炒丹紫色为度，每熬油1沸，入丹2两，熬至5沸，入丹了毕，前后俱用柳枝条捶碎，去其末，搅之，去火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
