序号：71130
名称：煨肝茵陈散
组成：茵陈蒿1两，犀角（屑）1两，石斛（去根）1两，人参1两，芍药1两，桔梗（炒）1两，防风（去叉）1两，柴胡（去苗）1两，细辛（去苗叶）1两，白术1两，桂（去粗皮）1两，吴茱萸（汤洗，焙干，炒）1两，当归（切，焙）1两。
出处：《圣济总录》卷一五○。
主治：妇人血风劳，四肢疼痛，心腹胀满吐逆，面无颜色，经脉不调。
加减：None
功效：None
用法用量：每服5钱匕，用猪肝1具，切作5段，每服用1段，薄切作小片子，入药末拌令匀，以湿纸裹，慢火煨熟，取出细嚼，以米饮送下。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
