序号：48062
名称：神吸散
组成：鹅管石（火煅，好醋淬7次）1钱，余粮石（火煅，醋淬7次）1钱，粉草3分，枯白矾5分，石膏（煅）5分，款冬花5分。
出处：《寿世保元》卷三引颜心吾方。
主治：年久近日咳嗽，哮吼喘息。
加减：None
功效：None
用法用量：每服3分2厘，至夜食后静坐片时，将药放纸上，以竹筒5寸长，直插喉内，用力吸药，速亦不怕，吸药令尽为度，以细茶1口，漱而咽之。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：忌鸡、鱼、羊、鹅，一切动风发物，并生冷诸物，惟食白煮猪肉、鸡子，戒21日。宜用公猪肺一副，加肉半斤，栀子一岁一个，炒成炭，桑白皮不拘多少，用之同炒至熟烂，去药，至五更，病人不要开口言语，令人将汤肺喂之，病人嚼吃任用，余者过时再食。
附注：None
