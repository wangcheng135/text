序号：61237
名称：铅霜丸
组成：铅霜1两（研入），犀角屑半两，桑螵蛸半两（微炒），赤箭1两，白花蛇2两（酒浸，去皮骨，炙令微黄），白僵蚕1两（微炒），白附子半两（炮裂），干蝎半两（微炒），天南星半两（炮裂），附子1两（炮裂，去皮脐），半夏半两（汤浸7遍去滑），羌活2分，乌头半两（炮裂，去皮脐），羚羊角屑3分，防风半两（去芦头），麝香1分（研入）。
出处：《圣惠》卷二十二。
主治：急风，身强口噤，手足拘急。
加减：None
功效：None
用法用量：每服10丸，以温酒研下，不拘时候。
制备方法：上为末，入研了药令匀，水煮槐胶为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方原名铅霜散，与剂型不符，据《普济方》改。
