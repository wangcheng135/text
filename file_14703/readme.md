序号：14710
名称：回阳无价至宝丹
组成：川楝子（取肉）2两，乌药2两，川牛膝1两，熟地黄1两，蛇床子1两，茯神1两，穿山甲1两，肉苁蓉1两，巴戟1两，五味子1两，人参1两，泽泻1两，大茴香1两，槟榔1两，乳香3钱，沉香5钱，檀香5钱，凤眼草2钱，鹿茸1两，仙灵脾1两，甘草1两，破故纸1两，菟丝子1两，葫芦巴1两，莲心5钱。
出处：《遵生八笺》（（纟玄）雪居本）卷十七。
主治：五劳七伤，四肢无力，下元虚冷，夜梦遗精，阳痿。
加减：None
功效：None
用法用量：每服30丸，空心以好酒送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，原书巴蜀本作"壮阳无价至宝丹"。
