序号：10819
名称：加味生化汤
组成：川芎3钱，当归6钱，黑姜4分，桃仁10粒，炙草5分，荆芥4分（炒黑）。
出处：《傅青主女科·产后编》卷上。
主治：产后劳倦甚而晕，及血崩气脱而晕。
加减：如形色脱，或汗出而脱，皆急服1帖，即加人参3-4钱（1加肉桂4分），决不可疑参为补而缓服；痰火乘虚泛上而晕，加橘红4分；虚甚加人参2钱；肥人多痰，再加竹沥7分，姜汁少许；其血块痛甚，兼送益母丸，或鹿角灰、或玄胡散、或独胜散、上消血块方，服1服即效，不必易方，从权救急。
功效：None
用法用量：加大枣，水煎，速灌两服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
