序号：76850
名称：骗马丹
组成：骨碎补（去毛）1两，自然铜（醋煅）1两，虎胫骨（酒炙）1两，败龟（酒炙）1两，川乌（炮）4两，草乌（炒，去尖）4两，半两钱5文（醋煅），牛膝半两，地龙（去土）半两，五灵脂半两，当归半两，乳香半两，没药半两，川芎半两，萆薢（炙）半两，羌活半两，藁本半两，天麻半两，防风半两，威灵仙（去苗）半两。
出处：《朱氏集验方》卷一。
主治：风湿相搏，手足疼痛，及诸伤百损。
加减：None
功效：None
用法用量：每服30-50丸，温酒送下，病在上，食后服，病在下，食前服，遍身疼痛，不拘时服。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
