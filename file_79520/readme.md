序号：79536
名称：蜜膏
组成：松香（1斤4两，醋、葱汁煮过，为末，筛净）1斤，黄蜡1两，白蜡1两，轻粉1两，乳香5钱，没药5钱，樟脑5钱，象牙末（炒）5钱，竹蛀末5钱，龙骨（火煅）5钱，赤石脂（醋煅）5钱，面粉（炒）5钱，海螵蛸（去壳）5钱，人中白（煅）5钱，孩儿茶3钱，血竭6钱，白蜜1两，桐油13两。
出处：《疡科选粹》卷八。
主治：诸般疮肿恶毒，臁疮湿毒，瘰疬，杨梅结毒，下疳久不收敛者。
加减：None
功效：None
用法用量：None
制备方法：先将松香溶化，次下桐油，次下黄白二蜡，次下龙骨等味，次下轻粉、象牙末、乳没药、樟脑、白蜜。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
