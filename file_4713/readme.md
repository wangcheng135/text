序号：4713
名称：木香硇砂煎丸
组成：木香1分，大黄（炮）1分，荆三棱（生用）1分，巴豆（去皮膜，不出油用，细研）1分，官桂（去皮）1分，青皮（去白）1分，筒子漆（炒）1分，蓬术（炮）1分，附子（炮，去皮脐）1分，干姜（炮）1分，香墨1指节大（细研），硇砂半两（以好醋1盏浸1宿，去砂石）。
出处：《博济》卷三。
主治：癥瘕积聚，血结刺痛。
加减：None
功效：None
用法用量：每服5丸，伤冷食、冷酒、冷水，结聚腹内，气块疼痛，用干姜汤或橘皮汤送下；夹食伤寒，白汤送下亦可；粘食不消成气块，即用煮面汤送下；食牛、羊、鱼、鳖肉成气块不散，用所伤汁送下；宿酒不消，血气不调，当归酒送下；妊娠不服，要转，淡茶送下。加至7丸，小儿3丸，常服1-2丸。
制备方法：上将大黄末、荆三棱末、巴豆等3味，同于银石器内，以好醋1升，煎1-2沸；次入硇砂，同熬成膏；次入诸药末，和匀，再入臼杵，为丸如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：木香煎丸（《圣济总录》卷七十三）。
