序号：37266
名称：定痛如神汤
组成：栀子3钱，白芍5钱，茯苓1两，苍术3钱，大黄1钱，厚朴1钱，甘草1钱。
出处：《傅青主男科》。
主治：火结在大小肠，腹中痛不可忍，按之愈痛，口渴，饮以凉水，则痛少止，少顷依然大痛。若不急治，一时气绝。
加减：None
功效：舒肝经之气，利膀胱之水，泻水逐瘀。
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：《石室秘录》：此方妙在舒肝经之气，用白芍、甘草和其痛。尤妙多用茯苓为君，以利膀胱之水。更妙在栀子以泻郁热之气。又恐行之欠速，更佐之大黄走而不守，则泻火逐瘀，尤为至神也。
用药禁忌：None
附注：定痛至神汤（《石室秘录》卷六）。
