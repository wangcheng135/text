序号：2301
名称：升麻和气饮
组成：干姜半钱，熟枳壳半钱，干葛1两，熟苍术1两，桔梗1两，升麻1两，当归2钱，熟半夏2钱，茯苓2钱，白芷2钱，陈皮1两半，甘草1两半，芍药7钱半，大黄（蒸）半两。
出处：《局方》卷八(续添诸局经验秘方)。
主治：疮疥癞风，不时痛痒；及气滞痛经。疮疥发于四肢，臀髀痛痒不常，甚至憎寒发热，攻刺疼痛，浸淫浮肿；又癞风入脏，阴下湿痒，耳鸣，眼痛。风癣疮疥，热结大便不通，经来腹痛，不来腹亦痛，因气滞者。
加减：None
功效：None
用法用量：每服4大钱，水1盏半，加生姜3片，灯心15茎，煎至7分，去滓，食前服。
制备方法：上锉散。
临床应用：None
各家论述：None
用药禁忌：None
附注：和气饮(《证治要诀类方》卷二)。①本方方名，《丹溪心法》引作“升麻汤”。②《医方类聚》引《澹寮方》有官桂、厚朴、荆芥，无葛根。
