序号：41771
名称：钓虫黑白丸
组成：白丸子：磁石、云母石、蛇含石、甘草各等分（上为未，糯米糊为丸，如黄豆大。），黑丸子：针砂、青黛、枯矾、甘遂各等分（上为末，醋煮糯米糊为丸，如龙眼核大。）。
出处：《医学入门》卷七。
主治：虫病，肚腹作痛，以及肺窍失声者。
加减：None
功效：None
用法用量：先以灯心煎汤，送下白丸子1丸；后以粗线1条穿住黑丸子，仍以灯心煎汤送下，待病者作呕；若不呕，再喝乌梅水1口，又含冷水1口，方为病者捶打胸背，并略抽动其线，令病人吐去冷水，仍作呕声，如此3-4次，则黑白丸子挟病根瘀血俱吐泻，吐后须随各经病调治。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：须用极高山顶上泉水，或武当回龙水煮糊为丸，方不化。
附注：None
