序号：47708
名称：神仙巨胜子丸
组成：熟地黄4两，生地黄4两，何首乌4两，牛膝（酒浸3日）1两，官桂（研）1两，枸杞子1两，肉苁蓉（酒浸3日）1两，菟丝子（酒浸3日）1两，人参1两，天门冬（酒浸3日）1两，茯苓（去皮）1两，巨胜子（焙，去皮）1两，天雄（去皮脐）1两，覆盆子（炒）1两，山药1两，楮实1两，川续断1两，柏子仁1两，酸枣仁1两，破故纸（炒）1两，巴戟（去心）1两，五味子1两，广木香1两，韭子1两，鸡头实1两，莲蕊1两，莲肉1两。
出处：《普济方》卷二二二。
主治：气虚血亏，肾寒精冷，遗精白浊，腰腿无力。
加减：如无天雄，可以附子（去皮脐）代之；久服，去天雄、用鹿茸。
功效：安魂定魄，改易容颜，通神仙，延寿命，补髓驻精，益气，治虚弱，展筋骨，润肌肤，头白再黑，齿落更生，耳聋复聪，目视有光，心力无倦，行疾如飞，寒暑俱不能侵，能除诸病。
用法用量：每服20丸，渐加至30丸，空心以温酒或盐汤送下，每日2次。服1月元气充足，60日白发变黑，100日容颜改变，目明可黑处穿针，冬月单衣不寒。
制备方法：上为细末，加胡桃10个研细，春、夏炼蜜为丸，秋、冬枣肉为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：巨胜子丸（《北京市中药成方选集》）。《医统》有甘菊花八钱；《北京市中药成方选集》无天雄、胡桃，有香附、菊花。
