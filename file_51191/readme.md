序号：51206
名称：荠苨汤
组成：荠苨4两，茯苓1两，蔓菁子1升，芍药1两，人参1两，蓝子1两，黄芩1两，甘草（炙）1两。
出处：《千金翼》卷二十二引华佗方。
主治：石毒或年20年30年而发者，或慄慄如寒，或饮食，或不饮食。若服紫石英发毒者，热闷惽惽喜卧，起止无气力，或寒，皆腑气所主，藏气不和；矾石发热者，燥而战；石硫黄发热者，郁郁；如热极者，身并破裂。
加减：虚弱者，倍人参，减黄芩；若气上，倍茯苓，加荠苨1两。
功效：None
用法用量：上（口父）咀，以水1升，煮蔓菁子，取8升，去滓，纳诸药，煮取2升5合，分3服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《圣济总录》引作“芍药汤”。
