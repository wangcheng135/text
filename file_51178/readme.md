序号：51193
名称：荜澄茄散
组成：荜澄茄半两（炒），荆三棱3两（炮，趁热杵），陈皮2两（去白），香附子3两（炒），甘草2两（炮，）丁香2两，舶上茴香2两，厚朴1两（去皮，姜汁炙令黄色），蛮姜1两1分，官桂2两（去皮），桔梗1两，白盐12两（拣择净，炒，研细，后入），阿魏（皂子大1块，面裹煨，面黄为度，去面，另研细，旋入诸药）。
出处：《博济》卷二。
主治：脾元虚冷，气不和，心胸不快，肚腹膨胀，气刺气痛，不思饮食，酒色过度，面黄口淡。
加减：None
功效：None
用法用量：每服1钱，入生姜2片，水8分盏，煎5-7沸，热吃；如作汤，入生姜1钱点之。
制备方法：上同为末，细研和令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：荜澄茄煮散（《圣济总录》卷四十六）。
