序号：48502
名称：神效散
组成：南木香2钱半，青皮2钱半，陈皮2钱半，麦糵（炒）2钱半，大枳壳（炮）2钱半，京三棱2钱半，蓬莪术2钱半，神曲（炒）2钱半，甘草（炙）2钱半，北白芍药2钱半，川白芷2钱半，肉桂（去皮）2钱半，玄胡索2钱半，破故纸2钱半，荜澄茄1钱，丁香1钱。
出处：《得效》卷四。
主治：远年近日一切脾疼，遇食冷物，或天气寒，阴冷便作，胸间一点痛起，或引入背膂，痛不可忍。
加减：None
功效：None
用法用量：每服3钱，水1盏半，加生姜3片，枣子1个，煎至7分，空心服；临睡加盐1捻，再煎两沸服。
制备方法：上锉散。
临床应用：None
各家论述：None
用药禁忌：忌面食、豆腐、一切生冷。
附注：None
