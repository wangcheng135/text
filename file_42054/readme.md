序号：42069
名称：三黄补血汤
组成：牡丹皮1钱，黄耆1钱，升麻1钱，当归1钱5分，柴胡1钱5分，熟地黄2钱，川芎2钱，生地黄3钱，白芍药5钱。
出处：《兰室秘藏》卷上。
主治：吐血、衄血，六脉俱大，按之空虚，心动善惊，面赤，上热者。衄血不止，脉软数者。
加减：None
功效：None
用法用量：每服5钱，水2大盏，煎至1大盏，去滓，食前稍热服。
制备方法：上(口父)咀，如麻豆大。
临床应用：None
各家论述：1.《兰室秘藏》：此气盛多而亡血，以甘寒镇坠之剂，大泻其气，以坠其浮；以甘辛温微苦，峻补其血，_x000D_
2.《医方集解》：二地补血，丹皮凉血，黄耆补气，升、柴升阳。气旺则能生血，阳生则阴自长矣。_x000D_
3.《医略六书》：血气两虚，虚阳迫肺，不能摄血，而从鼻上溢，故衄血久不止焉。黄耆补肺气以摄血，白芍敛肺阴以止衄，生地滋阴凉血，熟地滋肾补阴，当归养血归经，丹皮凉血止血也。俾血气完复，则虚阳自敛，而肺气清宁焉，有衄久不止之患乎？
用药禁忌：None
附注：按：《医略六书》无升麻、柴胡。
