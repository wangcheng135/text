序号：57309
名称：消痞膏
组成：三棱1两，蓬术1两，穿山甲1两，木鳖仁1两，杏仁1两，水红花子1两，萝卜子1两，透骨草（晒干）1两，大黄1两，独头蒜4个。
出处：《景岳全书》卷六十四。
主治：癥积痞块，泻痢。
加减：None
功效：None
用法用量：用时以白布或坚白纸摊贴，8-9日1换。或见大便去脓血勿以为异，亦有不去脓血而自愈者。若治泻痢，可贴脐腹。凡贴癥积痞块，先用荞麦面和作1圈，围住患处4边，其块上放皮消2-3两，盖厚纸以熨斗熨，令热气内达，然后去消，用膏药贴之。
制备方法：上用香油1斤，入前药10味，煎油成，以飞丹收之，后下细药真阿魏、乳香、没药各1两，麝香3钱，搅匀待冷，倾水中浸数日，用瓷瓶收贮，勿使泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
