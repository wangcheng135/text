序号：42402
名称：青蛾丸
组成：破故纸6两（淘洗净者，焙干，隔纸炒香，为末），胡桃瓤4两（浸去膜，研如泥）。
出处：《普济方》卷二二○引《风科集验方》。
主治：夜自泄，腹冷洞泻，饮食少味，行步无力，肾虚腰疼。肝虚，腰腿重痛，风湿脚气。
加减：None
功效：温精髓，补劳伤，补五脏，去百病，益肌肤，壮筋骨，活血驻颜，黑髭乌发，秘精益阳，老者服之还童，少者服之行步如飞。
用法用量：每服30丸，渐加至50丸，空心、临卧温酒、盐汤任下。食猪、羊腰子以助药力，似觉水甜食美有效。或研如泥，和蜜瓷器内，以熟水或酒调服，便以饭压之为妙。
制备方法：炼蜜为丸，如梧桐子大。
临床应用：阳道痿绝《普济方》引《风科集验方》：予年过八十，出宦南海，忽忽不乐，况粤俗卑湿，寒燥不常，痛伤内外，阳道痿绝，钟乳、硫黄一二十方皆不效。有舟人李摩诃来，授予此方取之七日，强气壮，阳道微动，半月以来，意充力足，目明心悦，神效不可具述。
各家论述：None
用药禁忌：禁食美薹莱、羊血，恶甘草。
附注：青娥丸（《普济方》卷二十九引《如宜方》）。
