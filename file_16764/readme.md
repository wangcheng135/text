序号：16771
名称：平补镇心丹
组成：酸枣仁（去皮，隔纸炒）2钱半，车前子（去土，碾破）1两2钱半，白茯苓（去皮）1两2钱半，五味子（去枝、梗）1两2钱半，肉桂（去粗皮，不见火）1两2钱半，麦门冬（去心）1两2钱半，茯神（去皮）1两2钱半，天门冬（去心）1两半，龙齿1两半，熟地黄（洗，酒蒸）1两半，山药（姜汁制）1两半，人参（去芦）半两，朱砂（细研为衣）半两，远志（去心），甘草（炙）1两半。
出处：《局方》卷五（宝庆新增方）。
主治：丈夫、妇人心气不足，志意不定，神情恍惚，夜多异梦，忪悸烦郁；及肾气伤败，血少气多，四肢倦怠，足胫痠疼，睡卧不稳，梦寐遗精，时有白浊，渐至羸瘦。
加减：None
功效：常服益精髓，养气血，悦色驻颜。
用法用量：每服30丸，空心饭饮送下；温酒亦得，加至50丸。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：镇心丹（《证治要诀类方》卷四）。本方方名，《张氏医通》引作“平补正心丹”。
