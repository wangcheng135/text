序号：32018
名称：万全汤
组成：柴胡3分，白术3分，黄芩3分，神曲3分，白芍1钱，麦冬1钱，当归5分，茯苓2分，甘草2分，苏叶2分，山楂3个。
出处：《傅青主男女科》卷下。
主治：小儿不拘早晚发热。
加减：冬，加麻黄1分；夏，加石膏3分；春，加青蒿3分；秋，加桔梗3分；有食，加枳壳3分；有痰，加白芥子3分；吐，加白蔻仁1粒；泻，加猪苓1钱；惊风，加人参5分。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：《傅青主男科重编考释》：万全汤亦逍遥散之变方也。用逍遥散舒肝解郁，以清肝郁内热；苏叶、柴胡以解外风；麦冬、黄芩以清肺热；加山楂、神曲，调理脾胃而固胃气。用此方统治小儿早晚发热，实属万全也。傅氏治惊风用人参，人多不知其奥者，以只知人参大补元气，而不知人参尚有镇静、安神、止惊之功也。
用药禁忌：None
附注：None
