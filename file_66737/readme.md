序号：66753
名称：鹿茸补肝丸
组成：鹿茸（燎去毛，酒浸，微炙）1两，熟干地黄1两半（洗，焙），当归（洗，焙）1两，白术1两，黄耆（蜜炙）1两，人参（去芦头）1两，附子（炮，去皮脐）1两，柏子仁（炒）3分，石斛（去根）3分，枳壳（去瓤，麸炒）3分，白茯苓（去皮）10两，覆盆子10两，酸枣仁（炒）10两，沉香10两，肉桂（去粗皮）10两。
出处：《杨氏家藏方》卷十六。
主治：产后劳伤血气，肝经不足，头运忪悸，四肢懈倦，翕翕气短，目视茫茫，耳鸣听重。
加减：None
功效：补五脏，益肝血，驻颜色。
用法用量：每服50丸，空心、食前温酒或米饮送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
