序号：3450
名称：太上混元丹
组成：紫河车1具（用少妇首生男子者良，带子全。于东流水洗断血脉，入无灰酒3升，椒1大合，同入砂石瓷器内，慢火煮，候酒欲尽，去火候冷取出，不用椒。只将河车入大砂盆内，用大木槌研如泥，摊凉，袤过，入下药），沉香半两，人参1两，白茯苓2两，苁蓉1两，乳香半两，麝香少许。
出处：《传信适用方》卷二引韩伯成方。
主治：虚劳怯弱。
加减：None
功效：还本元，补益，轻身延年，补损扶虚。
用法用量：每服40-50丸，空心温酒送下。
制备方法：上为细末，河车搅匀，可丸即丸，如梧桐子大，真朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中麝香，《重订严氏济生方》作“安息香”。
