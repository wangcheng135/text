序号：22853
名称：西王母玉壶赤丸
组成：武都雄黄1两（赤如鸡冠），八角大附子1两（炮称），藜芦1两，上丹砂1两（不使有石者），白礜石1两（炼之1日1夜），巴豆1两（去皮、炙令紫色称之）（一方有真朱1两）。
出处：《医心方》卷十四引《深师方》。
主治：尸注，卒恶，水陆毒螫万病，积聚，心腹痛，中恶，痛疡，水肿胀满。男女与鬼交通，歌哭无常，或腹大绝经，状如妊娠；恶风逆气不得气息；忧恚气结在胸心，苦连噫及咳，胸中刺痛；澼饮，痰饮；风疝，寒疝，心疝，弦疝；腹中三虫；卒关格，不得大小便，欲死；卒霍乱，心腹痛，烦满吐下，手足逆冷；下痢重下；疟未发或巳发，寒热往来；伤寒敕涩，时气热病；淋沥瘦瘠，百节痠痛；头卒风肿；耳聋，脓血汁出及卒聋；风目赤或痒，视物漠漠泪出，烂（？？）；齿痛；妇人产后余疾，及月水不通，往来不时；妇人胸中苦滞气，气息不利，少腹坚急，绕脐绞痛；小儿百病，惊痫痞塞及有热；小儿大腹及中热恶毒，食物不化，结成积聚；小儿寒热，头痛身热及吐乳；小儿羸瘦，丁奚，不能食，食不化。
加减：None
功效：解毒。
用法用量：大人服之皆丸如小豆大，若本病将服者，禁食生鱼、生菜、猪肉；服以下病者，宿勿食，明旦服2丸，不知者，饮暖米饮以发之令下，下不止，饮冷水以止之；病在膈上吐，膈下者下，或但噫气而愈。或食肉不消，腹坚胀或痛，服1丸立愈；风疝。寒疝，心疝，弦疝，每诸疝发腹中急痛，服2丸；积寒热老痞，蛇痞，服2丸；腹胀不得食饮，服1丸：卒大苦寒热往来，服1丸；卒关格不得大小便，欲死，服2丸；瘕结，服1丸、日3次，取愈；若微者，射（？？）丸甚良；下利重下，服1丸便断；或夏天行下便断，卒上气，但出不入及逆气冲喉，暴积聚者，服2丸，日2次；疟未发服1丸，已发，服2丸便断：小儿百病痞寒中及有热，100日至半岁者，以如黍米大1丸着乳头与服之，1岁以上，服如麻子大1丸，日3次，皆以饮服；小儿大(月真？？)及中热恶毒，食物不化，结成坚积，皆服1丸，亦可以涂乳头使小儿乳之；伤寒勅色及时气病，以温酒服1丸，厚覆取汗，若不汗，复以酒服1丸，要取汗；欲行视病人服1丸，以1丸着头上，行无所畏；至死丧家，带1丸、辟百鬼；病苦淋露消瘦，百节痠疼，服1丸，日3次；妇人产生余疾，及月水不通，及来往不时，服2丸，日2次；卒霍乱心腹痛，烦满吐下，手足逆冷，服2丸；注病，百种病不可名，将服2丸，日2次；若腹中如有虫，欲钻胁出状，急痛，一止一作，此是风气，服2丸；若恶疮不可名，瘑疥疽，以膏若好苦酒和药，先用盐汤洗疮去痂，拭令燥，以药涂之即愈；恶风（？？）心、不得气息，服1丸即愈；耳出脓血汁，及卒聋，以赤楮皮裹2丸塞耳孔中即愈；痈肿痤疖瘰疬及欲作瘘，以苦酒和药涂之，齿痛，以绵裹小丸着齿孔中咋之；苦寒热往来，服2丸，若蛇蝮蜂蝎蛎所中及猘犬狂马所咋，以苦酒和涂疮中，并服2丸即愈；卒中恶欲死不知人，以酒若汤水和2丸，强开口灌喉中，捧坐令下；澼饮、留饮、痰饮，服1丸，以蜡和1丸如弹丸，着绛囊中以系臂，男左女右；中溪水毒，服2丸；巳有疮在身，以苦酒和3-4丸涂疮上；忧患之气结在胸中，苦连噫及咳，胸中刺痛，服如麻子大3丸，日3次；妇人胸中苦滞气，气息不利，小腹坚急，绕脐绞痛，浆服如麻子大1丸，稍增之如小豆大；心腹常苦切痛及中热，服1丸如麻子大，日3次，5日愈；男女邪气鬼交通，歌哭无常，或腹大经绝，状如妊身，皆服如胡豆大3丸，日3次，夜1次，又以苦酒和之如饴，旦以涂手间使，暮又以涂足三阴交及鼻孔，7日愈，又将服如麻子大1丸，日3次，30日止；腹中三虫，宿勿食，明平旦进牛羊肉，灸3膊，须臾便服如胡豆大3丸，日中当下虫，过日中不下、复服2丸，必有烂虫下；小儿寒热，头痛身热及吐（口见？？），服如麻子大1丸；小儿消瘦丁奚不能食，食不化，服2丸，日3次，又苦酒和如饴，涂儿腹良；风目赤或痒，视物漠漠，泪出烂眦，以蜜解如饴，涂注目眦头；卒风肿，以苦酒若膏和涂之，即愈；风头肿，以膏和涂之，以絮裹之；若为蛄毒所中，吐血，腹内如刺，服如麻子大1丸，稍益至胡豆大，亦以涂鼻孔中，以膏和，通涂腹背，亦烧之自熏；鼠瘘，以脂和涂疮，取駮舌狗子舐之即愈也。
制备方法：先治巴豆3千杵；次纳礜石，治3千杵；次纳藜芦，治3千杵；次纳雄黄，治3千杵；次纳附子、治3千杵；次纳白蜜，治3千杵；若不用丹砂而纳真朱2两，勿令泄气。
临床应用：None
各家论述：《千金方衍义》：辟除恶毒之药，非猛力峻攻，无以建克敌之功。方中雄黄治寒热死肌，杀精物恶鬼邪气，胜五兵；附子治风寒痿躄，破癥坚积聚；藜芦治蛊毒泄利，杀蛊毒，去死肌；丹砂治身体五脏百病，养精神，安魂魄，杀鬼精恶物，与礜石治寒热风痹、腹中坚癖邪气；巴豆破癥瘕积聚，坚积留饮，荡练五脏六腑，开通闭塞，除蛊毒鬼疰邪物，种种皆辟除邪毒峻药，无不本诸本经。治宿患痼疾，确有五兵荡练之绩。而方后有无丹砂，真朱代用之说，真来即矾红，取其涤除积垢，以安神识也。
用药禁忌：None
