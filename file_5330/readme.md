序号：5330
名称：半豆饮子
组成：半夏、白豆蔻、苍术、干姜、藿香、陈皮、归尾、川芎、人参、白术、甘草、猪苓、砂仁、莲子。
出处：《陈素庵妇科补解》卷五。
主治：产后霍乱，由脏腑虚损，触冒风冷，阴阳不和，饮食失调，或冷或热，致成上吐下泻，肚腹疼痛；或腹中一条梗起，上冲心胸甚绞而痛，昏闷，面黑，唇青，手足厥逆，自汗，与寻常霍乱无异，但属产后血虚。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：参、术、陈、甘、半，去茯苓加猪苓也；砂仁、莲子以止利；苍、藿、干、蔻、陈、夏、砂仁温中止吐；加芎、归以养血；不用地、芍者，以其酸寒也。
用药禁忌：None
附注：None
