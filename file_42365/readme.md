序号：42380
名称：青蒿散
组成：青蒿2两，龙胆3分半（去芦头），栀子仁3分，知母3分，黄连1两（去须），鳖甲2两（涂醋炙令黄，去裙襕），黄耆1两（锉），桑根白皮1两（锉），地骨皮半两，白术1两，甘草半两（炙微赤，锉），柴胡1两半（去苗）。
出处：《圣惠》卷七十。
主治：妇人骨蒸劳热，四肢烦疼，日渐羸瘦。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，煎至6分，去滓温服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：《济阴纲目》：栀、连、龙胆所以清水火之热，桑、地、知母所以泻肺金之热，柴胡、鳖甲、青蒿解骨蒸劳热，然肺泻则气伤于上，火去则土无以生，故用黄耆、白术、甘草者，所以发巨橘之粟，以保赤子于干戈之地也。
用药禁忌：None
附注：None
