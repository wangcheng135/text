序号：48589
名称：神翁地仙丹
组成：天仙子1两，川椒2两（去子并开口者），木鳖子4两，白胶香5两（煮过，别研），五灵脂3两（陈黑色好者，用好酒浸，投水淘去，绢滤过，晒干），黑牵牛6两，黑豆8两，草乌7两（小而坚实者洗净，用盐在油中并炒，令色焦黄，折裂候冷，以纸布之类揩令净），赤土（即土朱）9两。
出处：《医方大成》卷三。
主治：风痹脚气。
加减：None
功效：None
用法用量：每服30丸，茶清或酒送下，病甚者频进。
制备方法：上为末，同入白胶香、木鳖子末，用隔年好醋打面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：地仙丹（《普济方》卷二四一）。
