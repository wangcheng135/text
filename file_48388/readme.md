序号：48403
名称：神效鸡清丸
组成：木香2两，黄连2两半，肉豆蔻10枚（大者，生）。
出处：《普济方》卷二○七引《瑞竹堂方》。
主治：一切泻痢。
加减：None
功效：None
用法用量：每服50丸，空心米饮汤送下。
制备方法：上为细末，取鸡子清搜药作饼子，于慢火上炙令黄色变红，稍干，再为末，面糊为丸，如梧桐子大。
临床应用：None
各家论述：《活幼口议》：木香、黄连一阴一阳药，木香善导水利气脉，黄连厚肠胃，二味君臣相佐，阴阳相须，加之豆蔻温和脏腑，止泻痢功效弥良。凡儿患泻与痢，不问证轻重，并宜搀先与服，不问脏腑冷热，愈多愈效。然鸡清为物有毒，是以毒气引药致效，若去此一味，其功不作矣。
用药禁忌：None
附注：None
