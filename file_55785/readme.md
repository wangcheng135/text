序号：55800
名称：桑仁粥
组成：桑椹子20-30g（鲜者30-60g），糯米2两或加冰糖少许。
出处：《药粥疗法》引《粥谱》。
主治：肝肾血虚，头晕目眩，视力减退，耳鸣腰痠，须发早白，及肠燥便干。
加减：None
功效：补肝滋肾，养血明目。
用法用量：先将桑椹浸泡片刻，洗净后与米同入砂锅内煮粥，粥熟加冰糖稍煮即可。每日2次，空腹食之，5-7天为1疗程。
制备方法：None
临床应用：None
各家论述：桑椹味甘性寒无毒，入肝肾经，有滋补肝肾。养血明目作用。用于头晕目暗，失眠贫血，腰膝痠软，须发早白，及老人便秘。
用药禁忌：煮粥忌用铁锅，以砂锅为好；桑椹以紫者为佳，红者次之，青者不可用；平素大便稀溏或泄泻者不宜服。
附注：None
