序号：9961
名称：止嗽七汁膏
组成：枇杷叶（刷去毛，剪去边，抽去筋膜，蜜水炙约2斤，河水5碗、井水五碗，熬汁去渣）2碗，藕汁1碗，梨汁1碗，白果汁1碗，荸荠汁1碗，萝卜汁1碗，人乳1碗，秋露水1碗，童便1碗（一方有竹沥1碗）。
出处：《同寿录》卷二。
主治：男妇新久咳嗽，不分昼夜。
加减：None
功效：None
用法用量：每服4-5钱，早、午、晚空心白滚水调服。
制备方法：同入瓷罐内，熬至一半；加姜汁1小杯，真柿霜2两，又熬干碗许；加白糖2两，又熬干至碗许；加好雄猪板油(去皮，切小块)2两，又熬，不住手搅至滴水不散，下白蜜2两，收成膏，出火气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
