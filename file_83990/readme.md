序号：84008
名称：五分帛膏
组成：五分帛1寸，乱发2分（洗令净），黄芩1两，紫菀1两（洗去苗土），莨菪子1两，倒钩棘刺1两，乳香2两，石盐1两，黄蜡2两，麝香1两（细研），黄丹7两，胡粉1两，生麻油1升，松脂2两（与乳香着油同捣如膏），水银1两（并胡粉点少水研令星尽）。
出处：《圣惠》卷六十六。
主治：积年瘘疮，及一切恶疮救急者。
加减：None
功效：None
用法用量：每用摊子故帛上贴，日2-3换之。
制备方法：上药各捣研讫，先将酒入于铛中，炼烟少出，即下5分帛、乱发，用武火煎，发、帛消尽后下黄芩等4味；良久下松脂、乳香2味，又搅；良久下麝香、黄蜡，又熟搅；次下黄丹、胡粉、石盐、水银，又微用火急搅，勿住手，取1碗冷水点看，硬即收之。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“五方帛膏”。
