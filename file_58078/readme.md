序号：58093
名称：益寿比天膏
组成：牛膝2两5钱，生地2两5钱，杜仲2两5钱，木鳖子2两5钱，虎骨（生）2两5钱，巴戟2两5钱，续断2两5钱，苁蓉2两5钱，生山甲2两5钱，山萸肉2两5钱，远志2两5钱，熟地2两5钱，补骨脂2两5钱，肉果2两5钱，官桂2两5钱，菟丝子2两5钱，紫梢花2两5钱，蛇床子2两5钱，天麻子2两5钱，川楝子2两5钱，甘草5两，海胆1两2钱5分，桑枝7寸，槐枝7寸。
出处：《北京市中药成方选集》。
主治：气虚血亏，梦遗滑精，肾寒精冷，腰痠腹痛。
加减：None
功效：暖丹田，滋肾水，培元补气。
用法用量：贴脐部，肾俞。
制备方法：上药酌予碎断，用香油240两炸枯，去滓过滤，炼至滴水成珠，入黄丹100两搅匀成膏，取出放入冷水中去火毒后加热溶化，再兑入细料面3两，搅匀摊贴，每张油重5钱，布光。
临床应用：None
各家论述：None
用药禁忌：None
附注：益寿比天膏细料面：雄黄五钱，龙骨五钱，石脂五钱，母丁香一两，沉香一两，木香一两，乳香一两，没药一两，阳起石一两，芙蓉叶一两，鹿茸二两二钱，共研细粉过罗。
