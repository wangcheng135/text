序号：2991
名称：天通丸
组成：熟干地黄（焙）5钱，天门冬（去心，焙）1两5钱，白术（锉）1两5钱，干姜（炮）1两5钱，当归（切，焙）1两5钱，石斛（去根）1两5钱，甘草（炙，锉）1两5钱，肉苁蓉（酒浸，去粗皮，切，焙）1两5钱，芍药1两5钱，人参1两5钱，大黄（炙，锉，炒）1两5钱，紫菀（洗）1两5钱，白茯苓（去黑皮）3分，防风（去苗）3分，麻仁（生研）3分，白芷5钱，蜀椒（去目及合口，炒出汗）1两，杏仁3分（汤浸，去皮尖双仁，炒）。
出处：《普济方》卷二三四。
主治：心腹积聚，胁肋刺痛，肌体羸瘦，不欲饮食；及八风十二痹，气血不荣。
加减：None
功效：久服身体润泽。
用法用量：每服20丸，米饮送下，1日3次。
制备方法：上为末，炼蜜煮枣肉合为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
