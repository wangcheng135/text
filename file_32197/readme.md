序号：32206
名称：抑阳酒连散
组成：生地黄3分，独活3分，黄柏3分，防风3分，知母3分，蔓荆子4分，前胡4分，羌活4分，白芷4分，生草4分，黄芩（酒制）5分，寒水石5分，栀子5分，黄连（酒制）5分，防己3分。
出处：《原机启微》卷下。
主治：神水紧小，渐如菜子许，及神水外围相类虫蚀者，然皆能睹物不昏，微有羞涩之证。
加减：None
功效：抑阳缓阴。
用法用量：作1服。水2盏，煎至1盏，去滓，大热服。
制备方法：None
临床应用：None
各家论述：此方以生地黄补肾水真阴为君；独活，黄柏、知母俱益肾水为臣；蔓荆子、羌活、防风、白芷群队升阳之药为佐者，谓既抑之，令其分而更不相犯也；生甘草、黄茶、栀子、寒水石、防己、黄连不走之药为使者，推欲抑之，不欲祛除也；诸用酒制者，为引导也。
用药禁忌：None
附注：None
