序号：47334
名称：祛风至宝丹
组成：防风1两，薄荷1两，荆芥1两，羌活1两，独活1两，连翘1两，黄芩1两，黄柏1两，黄连1两，栀子1两，全蝎1两，天麻1两，细辛1两，枳实1两，桔梗1两，大黄1两，芒消1两，生地1两，石膏1两，甘草1两，盐梅50个（去核），干葛1两半，赤芍1两半，细茶1两半，麻黄3钱（临症详审或用或不用）。
出处：《回春》卷四。
主治：癫痫属风热者。
加减：血虚，加川芎、当归各1两；气虚，加参、术各1两。
功效：祛风清热。
用法用量：每服2丸，细嚼，临卧时茶、酒任下，不拘时候。
制备方法：上为末，炼蜜为丸，如弹子大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
