序号：53270
名称：健步虎潜丸
组成：人参（去芦）10两，生黄耆2斤8两，故纸（盐炒）2斤8两，枸杞根子5斤，生白芍2斤8两，龟版（醋制）2斤8两，怀牛膝5斤，熟地2斤8两，独活2斤8两，制附子10两，秦艽2斤8两，木瓜5斤，黄柏2斤8两，知母2斤8两，当归2斤8两，炒枣仁2斤，菖蒲2斤，制虎骨3斤12两，菟丝子2斤8两，茯苓（去皮）2斤8两，防风2斤8两，锁阳2斤8两，续断5斤，杜仲炭（盐炒）2斤8两，羌活2斤8两，远志肉（甘草水制）2斤8两。
出处：《全国中药成药处方集》（天津方）。
主治：四肢疼痛，筋骨痿软，腰痠腿疼，肾囊寒湿。
加减：None
功效：去风湿，散寒通络，强壮筋骨。
用法用量：每服1丸，白开水送下。
制备方法：上为细末，炼蜜为丸，3钱重，蜡皮或蜡纸筒封固。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
