序号：32099
名称：万应延寿丹
组成：麦门冬1两（去心），天门冬1两（去心），熟地黄1两（酒蒸），生地黄1两（酒洗），人参1两，干山药2两，牛膝3两（酒浸），巴戟2两（去心），泽泻1两，肉豆蔻4两（酒浸，洗），杜仲1两（姜汁炒去丝），枸杞子1两（酒浸，洗去须），赤石脂1两（煅），远志1两，白茯苓1两（去皮），覆盆子1两，地骨皮1两（酒洗，去土），五味子6两半，车前子1两，石菖蒲1两，柏子仁1两，川椒7钱，菟丝子3两（酒浸），山茱萸1两。
出处：《普济方》卷二二五。
主治：None
加减：None
功效：补诸虚不足。
用法用量：每服30丸，空心温酒送下，初服每日3次，2日至3日，每日只1次。渐加至50丸，老者加100丸。服49日见效，至百日百病散去，身体荣润。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：万应延龄丹（《奇效良方》卷二十一）。小便或落杂色恶物，是肾间病出；五日间气痞，是脏腑间寒热气出，至七日唇口生津液，粗觉腹痛勿怪；十日全体渐肥润；至二十日，鼻顶辛酸，消除腹中一切痛证；四十五日语言雄壮，胸膈微痛，或吐微血，去积滞思虑郁结；至百日百病皆散，身体强健，至诚修合服饵，其效不浅也。
