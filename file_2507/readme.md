序号：2507
名称：双丸
组成：上麝香2两，牛黄2两，黄连2两（宣州者），丹砂1两，特生礜石1两（烧），附子1两（炮，去皮），雄黄1两，巴豆60枚（去皮心，熬），桂心1两，乌贼鱼骨1两，赤头蜈蚣1两（熬）。
出处：《千金翼》卷十一。
主治：小儿新生客忤中恶，发痫发热，乳哺不消，中风反折，口吐舌，并注忤，面青目上插，腹满，癫痫羸瘦，疰及3岁不行。
加减：None
功效：None
用法用量：生10日、20日至1月，日服2丸，如黍米大；40日至100日服2丸，如麻子大；1岁以上，以意增加。有儿虽小而病重者，增大其丸，不必依此丸。小儿病客忤率多耐药，服药当汗出，若汗不出者，不愈也。1日1夜4-5服，以汗出为愈。凡候儿中人者，为人乳子未了而有子者，亦使儿客忤。口中衔血即月客也。若有此者，当寻服此药，即儿可痊也。口聚唾，腹起热者，当灸脐中，不过2-7壮，并勤服此药。若喜失子者，产讫儿堕落地声未绝，便即以手指刮舌上，当得所衔血如韭叶者，便以药2丸如粟米大服之，作7日乃止，无不痊也。若无赤头蜈蚣，赤足者亦得，3枚，皆断取前两节，其后分不可用也。
制备方法：上药各治下筛，别研巴豆如膏，乃纳诸药，炼蜜和捣三千杵，密塞之，勿泄气。
临床应用：None
各家论述：None
用药禁忌：None
