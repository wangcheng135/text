序号：59069
名称：积气丹
组成：槟榔2个，芫花1两，硇砂2钱，巴豆2钱半（生），青皮（去白）3钱，陈皮3钱，蓬莪术1两，鸡爪1两，黄连1两，京三棱1两，章柳根1两，牛膝1两，肉豆蔻3个，大戟半两，川大黄半两，甘遂半两，白牵牛半两，干姜半两，青礞石半两，干漆半两，木香2钱半，石菖蒲3钱。
出处：《宣明论》卷七。
主治：一切新久沉积气块，面黄黑瘦，诸气无力，癥瘕积聚，口吐酸水。
加减：None
功效：None
用法用量：每服1丸，临卧烧枣汤送下，每夜1丸或2丸。候肚内作声，病退为度。
制备方法：上为末，醋面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
