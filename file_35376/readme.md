序号：35386
名称：京三棱丸
组成：京三棱1两（炮裂），桂心1两，川大黄1两半（锉碎，微炒），槟榔1两半，吴茱萸半两（汤浸7遍，焙干，微炒），干漆1两（捣碎，炒令烟出），附子1两（炮裂去皮脐），木香1两，桃仁1两半（汤浸，去皮尖双仁，麸炒微黄），青橘皮1两（汤浸，去白瓤，焙），鳖甲1两半（涂醋炙令黄，去裙襴）。
出处：《圣惠》卷四十八。
主治：久积聚，气不消，心腹胀满，食少体瘦。
加减：None
功效：None
用法用量：每服20丸，食前以温酒送下。
制备方法：上为极细末，以醋煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“三棱丸”。
