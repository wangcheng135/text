序号：49163
名称：胜金丹
组成：大黄3两（用米醋浸2宿，以竹刀子细切，于甑上蒸9度，研为糊），地龙（去土，醋内炒过）半两，芫花1分（醋炒令黄色，于银器内炒，不得犯铁器），蓬莪术半两（炮），川芎半两，当归半两，蒲黄、延胡索半两（于银器内炒），杜牛膝半两，官桂（去皮）半两，赤芍药半两，干地黄半两（以醋微炒），刘寄奴1分（略炒）。
出处：《博济》卷四。
主治：妇人血海虚冷，脐腹冷疼，肌肉黄瘦，饮食进退，时多困倦，四肢发烦，产前产后诸疾。
加减：None
功效：None
用法用量：每日早晨、临卧用温酒化下1丸；如产后有疾，以炒生姜酒化下1丸；产后，只用温酒化下1丸。
制备方法：上为末，倾入大黄膏内，搜为丸，如鸡豆大。
临床应用：None
各家论述：None
用药禁忌：月数未多，莫服此药。
附注：None
