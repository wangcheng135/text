序号：54851
名称：桂心消积丸
组成：桂心1两半，当归3两，赤芍1两半（酒炒），桃仁3两，厚朴1两半（制），三棱1两半（醋炒），槟榔1两半，蓬术1两半（醋炒），大黄3两（醋煮），鳖甲3两（醋炙）。
出处：《医略六书》卷三十。
主治：产后积聚，脉数弦洪紧涩者。
加减：None
功效：None
用法用量：每服3钱，酒煎，去滓温服。
制备方法：上为末，炼蜜为丸。
临床应用：None
各家论述：产后血瘀气壅凝结，经久而遏热伤阴，遂成积聚，故腹大胀满疼痛不止焉。槟榔破气导滞以消其聚，当归养血活血以荣其经，桃仁破瘀清积，厚朴散聚宽胀，三棱破气中之血以消坚，蓬术破血中之气以消积，赤芍破血通经，鳖甲滋阴散结，大黄涤热通幽，醋煮引入血分，肉桂温经暖血，中心通闭力优。蜜丸酒煎，使瘀化气行，则积聚并散，而遏热顿清，营阴暗复，安有胀满疼痛之患乎？
用药禁忌：None
附注：None
