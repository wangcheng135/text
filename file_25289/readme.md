序号：25297
名称：灰煎
组成：石灰1斗5升，湿桑皮4斗，柞栎灰4斗。
出处：《外台》卷二十九引《深师方》。
主治：瘤赘，瘢痕，疵痣及痈疽，恶肉。
加减：None
功效：None
用法用量：上以沸汤令浥浥调湿，纳甑中蒸之，从平旦至日中，还取釜中沸汤7斗，合甑3淋之，澄清，纳铜器中，煎令至夜，斟量余5斗汁，微火徐徐煎，取1斗，洗乱发，干之，如鸡子大，纳药中即消尽，又取五色彩剪如韭叶大，量5寸，着药中亦消尽，又令不强，药成，以白罂子中贮之。外敷
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：其血瘤，瘤附左右胡脉，及上下悬壅，舌本诸险处，皆不可令消，消即血出不止，杀人。
