序号：32096
名称：万应回生膏
组成：生地5钱，独活2钱，玄参2钱，木鳖1钱5分，当归2钱5分，红花1钱5分，升麻2钱5分，韭子1钱5分，草乌5钱，檀香5钱4分，牛膝2钱半，虎骨5钱，荆芥1钱5分，苏木3钱，羌活5钱，麻黄2钱，青皮1钱半，川断2钱半，川乌5钱，杜仲1钱5分，熟地5钱，防风2钱半，赤芍1钱半，秦艽2钱5分，香附3钱，桂枝2钱半，松节2钱5分，乌药1钱5分，桃仁30粒，蒲黄5钱，补骨脂5钱，灵仙1钱5分，刘寄奴2钱5分，骨碎补5钱，玄胡3钱，灵脂3钱（打），丹皮2钱半，血余5钱。
出处：《伤科秘方》。
主治：跌打损伤，远年拳伤，筋骨疼痛，寒湿气，漏肩风，鹤膝风，各穴道伤。
加减：None
功效：None
用法用量：随时摊贴。
制备方法：用麻油煎好，滴水成珠为度，每斤油用炒过东丹4两，加入后9味药末：血竭5钱，元寸7分，肉桂5钱，丁香5钱，附子3钱，木香5钱，乳香1两，没药1两，苏合油，文火收好。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中苏合油用量原缺。
