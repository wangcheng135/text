序号：49155
名称：胜金丹
组成：白砒1钱，绿豆360粒（水浸去壳，同白砒研如泥，阴干），肥栀子40枚（去壳，晒干，勿见火，为末），雄黄1钱，雌黄（俱水飞）1钱，急性子（即白凤仙子，去皮，研）2钱。
出处：《张氏医通》卷十四。
主治：痴病。
加减：None
功效：涌吐兼利。
用法用量：每服7-8分，强人至1钱，临服入西牛黄5-7厘，冰片3-5厘，细细研匀，入糕饼内食之。一方加珍珠（腐内煮，研）、琥珀、狗宝各1钱，分作20服，临服亦如上方，入西牛黄5厘，冰片3厘，上好白面1两5钱，将面匀作2分，先将一半入白糖霜1钱，半拌药为馅，再入白糖1钱半，裹外作饼，熯熟与食，食后姜汤过口，少顷即上吐下泻而愈；不吐，以肥皂肉1钱，擂水灌吐，吐后锈钉磨水，频进6-7次，以镇其神。
制备方法：上为极细末，都拌匀，瓷罐收藏。
临床应用：None
各家论述：None
用药禁忌：失心风癫、悲愁不语、元气虚人禁用。大忌烧酒。
附注：本方方名，《丸散膏丹集成》引作“胜金散”。
