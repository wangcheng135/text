序号：47994
名称：神异透骨膏
组成：露蜂房（细剪至极碎）5钱，杏仁（去皮尖）5钱，清油10两，穿山甲4钱，当归1两，木鳖子8枚（去壳），白胶香4钱（明者），蛇蜕5钱（淡盐水洗浸），虢丹4两，连须葱10茎，乳香2钱，没药2钱，男子乱发（洗净）如鸡子大（用童男女者）。
出处：《普济方》卷三一三。
主治：远年近日，一切恶疮、毒注疮等。
加减：None
功效：消肿定痛生肌。
用法用量：None
制备方法：上用清油浸药1宿，慢火熬诸药黑色，生绢帛滤去滓，复将所滤油慢火上再熬，却将黄丹入油，长柳枝、槐枝不住手搅，候有微烟起，即提起药铫，滴水面上，凝结成珠，膏成矣，搅无烟，却入乳、没、丹搅匀，倾瓷器内，新汲水1日一换，将药器放水内3日，出火毒，此膏可加玄参半两，黄耆4钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
