序号：51084
名称：草寒食散
组成：钟乳（炼）1分，附子（炮，去皮）1分，栝楼根1分，茯苓1分，牡蛎1分（熬），桔梗1两，干姜1两，人参1两，防风1两，细辛5分，桂心5分，白术3两半。
出处：《千金翼》卷十五。
主治：心腹胁下支满，邪气冲上，又心胸喘悸不得息，腹中漉漉雷鸣，吞酸噫生食臭，食不消化，时泄时闭，心腹烦闷，不欲闻人声，好独卧，常欲得热，恍惚喜忘，心中怵惕如恐怖状，短气呕逆，腹中防响，五脏不调。
加减：有冷，加椒；有热，加黄芩。
功效：None
用法用量：每服2方寸匕，旦未食时，以淳美酒服，不耐者减之。建日服之，至破日止，周而复始。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
