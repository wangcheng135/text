序号：33162
名称：万应蝉花散
组成：蝉蜕（去土）半两，蛇蜕（炙）3钱，川芎1两，防风1两，羌活1两，炙甘草1两，当归1两，白茯苓1两，赤芍药3两，苍术4两，石决明（东流水煮1伏时，研极细）1两半。
出处：《原机启微》卷下。
主治：上焦有热邪，目久病，白睛微变青色，黑睛稍带白色，黑白之间赤环如带，谓之抱轮红，视物不明，昏_x000D_
如雾露，中睛白，高低不平，其色如死，甚不光泽，口干舌苦，眵多羞涩；亦治奇经客邪之病。大人小儿，远年近日，一切风眼气眼、攻注昏眼，睑生风粟，或痛或痒，渐生翳膜，或久患头风牵搐，两目渐渐细小，眼眶赤烂。
加减：None
功效：祛风退翳明目。
用法用量：每服2钱，食后临卧时浓米泔调下；热茶清亦得。
制备方法：上为细末。
临床应用：None
各家论述：上方制之复者也，奇之不去则偶之，是为重方也。今用蝉蜕又用蛇蜕者，取其重蜕之义以除翳，为君也；川芎、防风、羌活，皆能清利头目，为臣也；甘草、苍术通主脾胃，又脾胃多气多血，故用赤芍药补气，当归补血，为佐也；石决明镇坠肾水，益精还阴，白茯苓分阴阳上下，为使也。
用药禁忌：None
附注：None
