序号：318
名称：五香丸
组成：乳香2两（轻炒令熔，候冷，研细），木香2钱（生用不见火），沉香2钱（生用），枫香1分（水煮过，研细），麝香半钱（别研，同枫、乳后入），没药2钱（酒浸少时，研成膏子入糊），天麻2钱（酒浸讫，锉碎），白附子1两（酒浸少时，锉），干蝎1两（用火烧地令红，将醋泼地上，覆定1时去毒），川乌1两（酒浸1宿，炮裂，去皮尖。取半两），草乌半两（酒浸1宿，炮裂，去皮尖。取1分），白花蛇（去皮骨，取肉）1两（酒浸，炙令赤色），乌蛇1两（如前制），地龙1两（炒去土，取半两），大附子1分（炮制，去皮尖），蝉蜕2钱（用盐水浸少时，焙干），白蒺藜1两（去尖），麒麟竭2钱（别研），无名异2钱（同自然铜醋炒讫，研），自然铜2钱（醋炒，干研），朱砂2钱（别研，为衣）。
出处：《传信适用方》卷二。
主治：干湿脚气，风毒肿弱，遇发寒慄，不欲饮食，行立不得。
加减：None
功效：None
用法用量：每服15丸，或20丸，食前温酒或木瓜汤送下。痛甚者，三两服即止。
制备方法：上为末，酒糊为丸，如梧桐子大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
