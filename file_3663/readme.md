序号：3663
名称：开心薯蓣肾气丸
组成：肉苁蓉1两，山茱萸1两，干地黄6分，远志6分，蛇床子5分，五味子6分，防风6分，茯苓6分，牛膝6分，菟丝子6分，杜仲6分，薯蓣6分。
出处：《医心方》卷十三引《范汪方》。
主治：丈夫五劳七伤，髓极不耐寒，眠即肿胀，心满雷鸣；不欲饮食，虽食心下停淡不能消；春夏手足烦热，秋冬两脚凌冷；多忘。
加减：None
功效：健中补髓，填虚养志，开心安脏，止泪明目，宽胃养阴阳，除风去冷。
用法用量：每服10丸至20丸，日2夜1。若烦心，即停减之，只服10丸为度。服药5日，玉茎炽热；10夜，通体滑泽；15夜，颜色泽，手足热；20夜雄力欲盛；25夜，经脉充满；30夜，热气朗彻，面色如花，手纹如丝而心开，记事不忘，独寝不寒。年40以下1剂即足；50以上2剂即足。妇人断续者，服1剂，50得子。
制备方法：上药治下筛。炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌大辛、醋。
附注：None
