序号：31267
名称：沉香降气丸
组成：沉香（镑）半两，木香半两，荜澄茄半两，枳壳（去瓤）半两，缩砂仁半两，白豆蔻仁半两，青皮（去白）半两，陈皮（去白）半两，广术（炮）半两，枳实（麸炒）半两，黄连（去须）半两，半夏（生姜制）半两，萝卜子（另研）半两，白茯苓（去皮）1两，香附子2两（炒去皮毛），白术1两（煨），乌药1两半。
出处：《瑞竹堂方》卷一。
主治：胸膈痞满。
加减：None
功效：升降水火，调顺阴阳，和中益气，推陈致新，进美饮食。
用法用量：每服50-70丸，临卧煎橘皮汤送下，姜汤亦可，1日1次。
制备方法：上为细末，生姜自然汁浸蒸饼为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌生冷。
附注：None
