序号：47413
名称：祛老乌须健阳丹
组成：何首乌（赤白各1斤，米泔水浸，竹刀刮去皮，切碎），牛膝半斤（用何首乌、黑豆5升，砂锅内蒸3次），茯神半斤（乳拌），茯苓（赤1斤，牛乳浸1宿，白1斤，乳汁浸1宿），枸杞子半斤（酒浸，蒸，晒干），当归半斤（酒浸1宿，晒干），破故纸5两（炒黄），菟丝子半斤（酒浸，蒸，晒干）。
出处：《同寿录》卷一。
主治：None
加减：None
功效：健身体，去诸风，明眼目，乌须发，益气力。
用法用量：每服50丸，空心酒送下，午时姜汤送下，晚上盐汤送下，1日3次；或俱用酒下亦可。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
