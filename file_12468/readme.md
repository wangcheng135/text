序号：12474
名称：白薇丸
组成：白薇5分，细辛5分，厚朴（炙）5分，椒（汗）5分，桔梗5分，鳖甲（炙）5分，防风6分，大黄6分，附子（炮）6分，石硫黄6分（研），牡蒙2分，人参4分，桑上寄生4分，半夏（洗）2分，白僵蚕2分，续断2分，秦艽2分，紫菀2分，杜仲2分，牛膝2分，虻虫（去翅足，熬）2分，水蛭2分，紫石英（研）8分，朴消8分，桂心8分，钟乳8分，当归8分。
出处：《外台》卷三十三引《广济方》。
主治：妇人百病，断绝绪产。
加减：None
功效：None
用法用量：每服15丸，空腹温酒送下，1日2次。渐加至30丸。
制备方法：上药治下筛，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻、饧、生血物、人苋、生葱、生菜、猪肉、冷水、粘食、陈臭。
附注：None
