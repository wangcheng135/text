序号：41696
名称：金蝉丸
组成：白术（炒）16两，黄芩16两，山楂16两，莪术（炙）16两，陈皮16两，莱菔子（炒）16两，干蟾（烧）16两，银柴胡8两，使君子8两，麦芽8两，神曲（炒）8两，槟榔8两，茯苓8两，三棱（炒）8两，厚朴（炙）8两，鸡内金（炒）8两，青皮（炒）8两，甘草8两，黄连4两，胡连4两，芦荟1两，枳壳（炒）32两。
出处：《北京市中药成方选集》。
主治：小儿疳积痞块，食积腹胀，面黄肌瘦，口舌糜烂。
加减：None
功效：消疳化积，退烧杀虫。
用法用量：小儿每服1钱，1日2次，温开水送下，3岁以下小儿酌减。
制备方法：上为细末，过罗，用冷开水泛为小丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
