序号：48633
名称：神验膏
组成：糯米粉1合，飞罗面1合，马屁孛1碗（即灰菰），上3味拌和，于生布上筛，下用生姜自然汁半碗，调却入后药；牛皮胶2两半，酸枣醋1碗。
出处：《医方类聚》卷一八七引《经验良方》。
主治：打扑伤，损筋动骨，不问手足腰背。
加减：None
功效：None
用法用量：每遇磕损，以药膏摊于纸上，贴伤处，却用杉木片夹定，用厚衣服裹之，勿令冷，2-3日1换。若只伤筋骨，贴上如火，若筋损骨碎，贴上而痛如虫行为妙。初折伤者，只3次效。
制备方法：上先用醋熬牛皮胶化，便入前所调3味药，不住手搅成膏子，以瓷器盛之。
临床应用：None
各家论述：None
用药禁忌：None
附注：神效膏（《普济方》卷三一○）。
