序号：26784
名称：延寿散
组成：牡丹皮半两，附子半两（炮，去皮脐），柴胡半两（去苗），秦艽半两，鳖甲（酒浸，炙令赤色）半两。
出处：《博济》卷一。
主治：男子、女人、少童等传尸诸劳嗽，寒热百般变候，蒸汗劳虫。
加减：None
功效：None
用法用量：每服用猪石子1枚，葱白、椒末同一处，细锉如膏，入药末3钱匙，拌和，以童便1盏，水半盏，同煎令沸，温服。其患人贴体着皂纱衫一领，服后衣物盖覆，令恶汗出，粘腻尽出，乃脱皂衫，揩拭身干，避风将息。每七一度，服后汗轻即住。皂衣遂以皂角汤泡洗，安息香烘干。
制备方法：上为细末，入人中白半两同研，以瓷器盛之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
