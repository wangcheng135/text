序号：33379
名称：肝胃百合汤
组成：柴胡10g，黄芩10g，乌药10g，川楝子10g，郁金10g，百合30g，丹参15g。
出处：《夏度衡医案》。
主治：胃和十二指肠溃疡，胃脘疼痛，嗳气吞酸，心烦口苦，脘区压痛，大便色黑，舌质淡红，苔薄微黄，脉弦细。
加减：胃部灼热，加蒲公英15g；灼热喜按喜温饮，加高良姜3g；胸部痞满发胀，加九香虫3g；吐酸水，加生牡蛎30g，或瓦楞子30g；大便结，加火麻仁10g；大便色黑，加桃仁10g；脾胃虚弱，加明党参、黄耆各12g；十二指肠溃疡，加白芍12g，甘草10g。
功效：疏肝和胃，活血化瘀。
