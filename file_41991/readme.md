序号：42006
名称：三萸丸
组成：食茱萸1两，吴茱萸（汤浸，微炒）1两，桔梗（水浸1伏时漉出，慢火炒）1两，白蒺藜1两，青皮（去白）1两，山茱萸（去核取肉，微炒）1两，舶上茴香（淘去沙土，焙干）1两，五味子（净拣）1两半，海藻（洗，焙）1两半，大腹皮（酒洗过，晒干）1两半，川楝子（去核）1两半，玄胡索1两半。
出处：《普济方》卷三二六。
主治：因多服热药及煎煿，或犯非理房事兼意淫不遂，阴中生一物渐大，牵引腰腹胀痛，甚至不思饮食，名阴挺。
加减：下虚，加川芎、炮姜（去皮）、肉桂（去粗皮）各1两；腰腹痛甚，加桃仁（去皮尖，麸炒，别研）；肝郁，加青皮（去白）、枳实（去瓤）各1两，真南木香7钱半。
功效：None
用法用量：每服30-50丸，木通汤送下。
制备方法：上为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：三茱丸（《医统》卷八十三）。
