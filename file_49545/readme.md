序号：49560
名称：茯苓丸
组成：白茯苓（去黑皮）1两半，石斛1两半，人参1两，蒺藜子（炒去角）1两，羚羊角（镑屑）1两，枳壳（去瓤，麸炒）1两，五加皮1两，酸枣仁1两半，五味子1两半，菟丝子（酒浸1宿别捣，焙）3两，云母粉2两，黄耆（细锉）1两，防风（去叉）1两，细辛（去苗叶）1两，独活（去芦头）1两，杜仲（去粗皮，炙，锉）1两，鹿角胶（炙燥）2两，槟榔（煨，锉）2两，天雄（炮裂，去皮脐）1枚，续断（锉，焙）2两，泽泻1两半，当归（锉，焙）1两半，熟干地黄（焙）3两，甘草（炙）1两，肉苁蓉（酒浸，去破皮）1两半。
出处：《圣济总录》卷十八。
主治：诸癞脓溃，体虚气热，荣卫不清，风鼓于脉。

制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
