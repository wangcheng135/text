序号：38171
名称：松脂丸
组成：松脂20斤（以桑柴灰汁炼去苦汁，倾入水盆中凝取之），白茯苓（去黑皮）1斤，白术半斤，续断半斤，白蜜2两，牛酥2两，麦门冬（去心，焙）2两，生干地黄（焙）2两。
出处：《圣济总录》卷九十三。
主治：传尸骨蒸，积癖冷气，及腰脚衰弱，身体风痒，并诸疮癞疾、恶疮疥癣等。
加减：None
功效：None
用法用量：每服20丸，以温酒或米饮送下，渐加至50丸，1日2次。
制备方法：上8味，捣罗5味为细末；先以慢火炼蜜烊去沫，次下牛酥，次下松脂，候烊讫，即下药末，以竹篱搅勿住手，可丸即丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
