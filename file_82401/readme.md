序号：82417
名称：薯蓣鸡子黄粥
组成：生山药1斤（轧细过罗）。
出处：《衷中参西》上册。
主治：肠滑不固之久泄泻。
加减：None
功效：None
用法用量：每服用药7-8钱，或至1两，和凉水调入锅内，置炉上，不住以箸搅之，2-3沸即成粥服之。若小儿服，或少调以白糖亦可。
制备方法：None
临床应用：泄泻：一人年近五旬，泄泻半载不愈，羸弱已甚。遣人来询方，言屡次延医服药，皆分毫无效，授以薯蓣粥方。数日又来，言服之虽有效验，泻仍不止。遂俾用鸡子数枚煮熟，取其黄捏碎，调粥中服之，两次而愈。盖鸡子黄，有固涩大肠之功，且较鸡子白易消化也。以后此方用过数次，皆随手奏效。
各家论述：None
用药禁忌：None
