序号：44373
名称：一粒金丹
组成：麝香2钱5分，乳香6钱，没药7钱，当归7钱，地龙2两5钱，白檀香2两5钱，木鳖子5钱，草乌2两，五灵脂2两，京墨线胶（麸炒）2两5钱，紫萍2两5钱。
出处：《解围元薮》卷三。
主治：遍身亸曳，鱼鳞刺风，远年寒湿，手足痿痛，走注叫唤。
加减：None
功效：None
用法用量：每服1丸，酒送下，每日2次，至黑汁从足底出，乃为验。
制备方法：上为末，用去节麻黄2两煎汁，煮枣肉、米饭为丸，如龙眼大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：赤龙丸、一锭金（原书同卷）、一粒金丸（《青囊秘传》）。《疡医大全》有阿胶，无线胶、当归。方中白檀香，《青囊秘传》作“白胶香”。
