序号：34139
名称：牡蛎散
组成：黄耆（去苗土）1两，麻黄根（洗）1两，牡蛎（米泔浸，刷去土，火烧通赤）1两。
出处：《局方》卷八。
主治：虚劳不足，自汗盗汗，心悸遗精。
加减：None
功效：敛汗固表。
用法用量：每服3钱，水1盏半，小麦100粒，同煎至8分，去滓热服，1日2次，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：1.《医方集解》：此手太阴少阴药也。陈来章曰：汗为心之液，心有火则汗不止，牡蛎、浮小麦之咸凉，去烦热而止汗，阳为阴之卫，阳气虚则卫不固，黄耆、麻黄根之甘温，走肌表而固卫。_x000D_
2.《成方便读》：黄耆固卫益气，以麻黄根领之达表而止汗；牡蛎咸寒，潜其虚阳，敛其津液；麦为心谷，其麸则凉，用以入心，退其虚热耳。此治卫阳不固，心有虚热之自汗也。
用药禁忌：None
