序号：23888
名称：安神定志丸
组成：人参1两5钱，白茯苓（去皮）1两，白茯神（去心）1两，远志（去心）1两，白术（炒）1两，石菖蒲（去毛，忌铁）1两，酸枣仁（去壳，炒）1两，麦门冬（去心）1两，牛黄1钱（另研），辰砂2钱5分（水飞，另研，为衣）。
出处：《医便》卷一。
主治：咽干，惊悸，怔忡，健忘。
加减：None
功效：宁心保神，益血固精，壮力强志，清三焦，化痰涎，育养心神，大补元气。
用法用量：每服30丸，清米汤送下，日3次，不拘时候。
制备方法：上为末，龙眼肉4两熬膏，和炼蜜3-4两为丸，如梧桐子大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
