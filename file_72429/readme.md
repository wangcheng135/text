序号：72445
名称：彭祖丸
组成：柏子仁5合，石斛3两，天雄1两（炮），巴戟天3两（去心），续断3两，天门冬3两（去心），泽泻2两，菟丝子5两，人参2两，干地黄4两，薯蓣2两，远志2两（去心），蛇床子5合（取仁），钟乳3两（炼，研成粉），覆盆子5合，苁蓉6两，山茱萸2两，杜仲3两，菖蒲2两，五味子5两，桂心4两，茯苓2两。
出处：《外台》卷十七引《古今录验》。
主治：劳虚风冷百病。男女诸虚不足，老人精枯神耗。
加减：None
功效：延年益寿，通脏腑，安神魂，宁心意，固荣卫，开益智慧，令寒暑风湿气不能伤人。令目睛光明，冷泪不复出，筋力强健，悦泽肌肤。
用法用量：每服8丸，渐加至10丸，酒送下，勿令醉，1日2次。先服药，斋5日，不食脂、肉、菜、五辛。服20日断白沥，30日渐脱，60日眼瞳子白黑分明，不复泪出，溺血余沥断，80日白发变黑，腰背不复痛，行步脚轻，150日都愈，意气如年少时，诸病皆除，长服如神。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌鲤鱼、生葱、猪羊肉、冷水、酢物、芜荑、饧。
附注：小丹（《元和纪用经》）。
