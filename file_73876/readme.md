序号：73892
名称：硼砂丸
组成：硼砂1两（细研），硫黄1两（细研，水飞过），木香半两，槟榔1两，川大黄3两（锉碎，微炒），牵牛子3两（微炒），吴茱萸半两（汤浸7遍，焙干，微炒），京三棱1两（微煨，锉），当归1两（锉，微炒），肉桂1两（去皱皮），青橘皮1两（汤浸，去白瓤，焙），鳖甲1两（涂醋，炙令黄，去裙襕）。
出处：《圣惠》卷四十九。
主治：癥病不消，四肢羸困，不欲饮食，久不愈。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“硇砂丸”。方中硼砂，《普济方》引作“硇砂”。
