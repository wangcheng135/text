序号：36696
名称：固本还睛丸
组成：天门冬（去皮心，酒浸1宿，另杵如泥）3两，麦门冬（去心，焙干）3两，生地黄（酒浸，焙，勿犯铁）3两，熟地黄（酒洗净，用瓷器蒸，勿犯铁）3两，人参1两5钱，白茯苓1两5钱，干山药1两5钱，枸杞子1两5钱，川牛膝1两（酒洗），石斛1两（去芦，酒洗），草决明（微炒）1两，杏仁（去皮尖，另研）1两，甘菊花（用小金钱）1两，菟丝子（酒浸3宿，另研，焙干）1两，枳壳（麸炒黄色）1两，羚羊角1两（细锉，取净末8钱），乌犀角8钱（锉细，生用），五味子7钱（焙干），甘草7钱（炙），防风8钱（去芦），白蒺藜7钱（杵去刺），黄连7钱（去毛），川芎7钱，青葙子8钱（微炒）。
出处：《医学正传》卷五。
主治：远年一切目疾，内外翳膜遮睛，风弦烂眼，及老弱人目眵多糊，迎风冷泪，视物昏花等。
加减：None
功效：升水降火，平肝益肾，明目清心。
用法用量：每服50-70丸，盐汤送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：夜光丸（《济阳纲目》卷一○一）。本方方名，《景岳全书》引作“祖传固本还睛丸”。
