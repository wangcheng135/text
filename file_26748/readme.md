序号：26756
名称：延龄益寿丹
组成：茯神5钱，远志肉3钱，杭白芍4钱（炒），当归5钱，党参4钱（土炒焦），黄耆3钱（炙焦），野白术4钱（炒焦），茯苓5钱，橘皮4钱，香附4钱（炙），广木香3钱，广砂仁3钱，桂圆肉3钱，枣仁4钱（炒），石菖蒲3钱，甘草2钱（炙）。
出处：《慈禧光绪医方选议》。
主治：思虑太过，伤及心脾两脏，食少体倦，大便不调，健忘怔忡，惊悸少寐，脾虚不能统血，妇女月经不调与带下。
加减：None
功效：甘温养神，养心安神，畅利气机，延年益寿。
用法用量：每服2钱5分，白开水送下。
制备方法：上为极细末，炼蜜为丸，如绿豆大。朱砂为衣。
临床应用：None
各家论述：本方参、耆、术、草、茯苓、茯神甘温益脾；当归、远志、枣仁、桂圆濡润养心；木香之外又加石菖蒲，香附，更可畅利气机，是妇人长寿好方。
用药禁忌：None
附注：None
