序号：82377
名称：薤白膏
组成：薤白2握，白蔹1两，赤芍药1两，杏仁1两（汤浸，去皮尖双仁），续断1两，芎1两，白芷1两，郁金1两，生地黄1两（切），棘针1两，滑石3两，绯帛1尺（烧灰），青布1尺（烧灰），黄丹24两。
出处：《圣惠》卷六十七。
主治：搕打伤折，金疮。
加减：None
功效：生肌。
用法用量：贴痛上，日1换。
制备方法：上药除黄丹外，锉细，用麻油3升，先煎薤白、生地黄，后下诸药，以慢火煎半日，次下滑石、绊帛、青布灰等，再用慢火煎半日，以绵滤去滓，于净锅内炒黄丹，令紫色，旋下油内，以柳木枝不住手搅，成紫色，待油力尽，滴于水内成珠子，看不污人手即停火，入盒中收，用纸摊。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
