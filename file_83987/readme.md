序号：84005
名称：五仁橘皮汤
组成：甜杏仁3钱（研细），松子仁3钱，郁李净仁4钱（杵），原桃仁2钱（杵），柏子仁2钱（杵），广橘皮钱半（蜜炙）。
出处：《重订通俗伤寒论》。
主治：体虚便秘。
加减：若欲急下，加元明粉2钱，提净白蜜1两，煎汤代水可也；挟滞，加枳实导滞丸3钱；挟痰，加礞石滚痰丸3钱；挟饮，加控涎丹1钱；挟瘀，加抵当丸3钱；挟火，加当归龙荟丸3钱；挟虫，加椒梅丸钱半，或吞服，或包煎。
功效：润燥滑肠。
用法用量：None
制备方法：None
临床应用：None
各家论述：杏仁配橘皮，以通大肠气闭；桃仁合橘皮，以通小肠血秘；气血通润。肠自滑流，故以为君；郁李仁得橘皮，善解气与水互结，洗涤肠中之垢腻，以滑大便，故以为臣；佐以松、柏通幽，幽通则大便自通。
用药禁忌：None
附注：None
