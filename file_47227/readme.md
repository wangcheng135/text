序号：47242
名称：相传丸
组成：天门冬（去心，焙）3两，麦门冬（去心，焙）3两，贝母（去心，焙）3两，紫菀（去土）3两，百合3两，桔梗（炒）3两，人参3两，杏仁（汤浸，去皮尖双仁，炒）3两，生干地黄（焙）3两，桂（去粗皮）3两，半夏（汤煮软，切，焙干）3两，甘草（炙）3两，阿胶（炒至沸）3两，陈橘皮（汤浸，去白）3两。
出处：《圣济总录》卷六十六。
主治：肺寒外内合邪，停饮寒痰，咳嗽，语声不出，口中如含霜雪，咽喉妨闷，状若梅核，噎塞不通，膈气痞气；及咳嗽脓血。
加减：None
功效：None
用法用量：每服1丸，用生姜细嚼送下，嗽时服。
制备方法：上为末。煮糯米粉，并黄蜡1两成粥，更入蜜再熬匀，和前药为丸，如樱桃大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
