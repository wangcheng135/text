序号：51804
名称：钟乳健脾散
组成：成炼钟乳粉2两，人参（去芦）2两，肉豆蔻（面裹，煨）1两，诃子（煨，去核）1两，高良姜（炒）1两，厚朴（去粗皮，姜制，炒）1两，白茯苓（去皮）1两，甘草（炙）1两，陈皮（去白）1两，神曲（炒）1两，草果仁1两，麦糵（炒）1两，干姜1两半（炮，洗）。
出处：《魏氏家藏方》卷五。
主治：一切冷气，脾胃久虚，胸膈痞塞，中脘气滞，腹胀虚鸣，上气喘急，心腹绕痛，宿食不化，留饮停积，痰逆呕吐，嗳气不通，不进饮食，面黄肌瘦，四肢怠堕。膈气噎塞，霍乱吐泻。
加减：None
功效：None
用法用量：每服2钱。以水1盏，加生姜3片，大枣1枚，盐1捻，同煎至7分。通口服，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
