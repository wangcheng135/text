序号：60021
名称：资生丸
组成：白术3两（米泔水浸，用山黄土拌，九蒸晒，去土，切片，焙干），橘皮2两，山楂2两（蒸），神曲2两（炒），白茯苓1两5钱（人乳拌，饭上蒸，晒干），人参3两（人乳浸透，饭锅上蒸透），白豆蔻5钱（微炒），扁豆1两（炒），莲肉1两（去心，炒），山药1两半（炒），芡实1两半（炒），薏苡仁2两（炒）。
出处：《医学摘粹·杂证要法》卷二。
主治：胃有虚热，不能食，常觉饱闷，面黄赤，身常恶热，大便燥结。
加减：None
功效：None
用法用量：每服2钱，细嚼，淡盐汤送下。
制备方法：上为末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
