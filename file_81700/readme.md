序号：81716
名称：整骨丸
组成：白矾13两（飞过），黄蜡6两，黄丹4两，密陀僧（别研）4两，自然铜（别研为末）4两，乳香（别研）1两，朱砂（别研）1两，没药（别研）1两，猪脂12两（腊月者）。
出处：《杨氏家藏方》卷十四。
主治：从高坠下，筋断骨折，内外俱损，疼痛难任，不问轻重。
加减：None
功效：None
用法用量：每服20丸，热葱酒送下（葱白切如丝，于热酒中以盏合少时，勿出气）。如昏困不能咽药者，即用葱酒化开灌下。仍将2-3丸切开，于纸上以火炙之；如不化，入油少许，随所伤大小，摊成药靥，恐此药不粘，以云母膏涂靥周围贴之，良久麻木变成痒；若骨碎及蹉跌者，并皆平正，不成芦节，其伤破处便生肌肉；多年伤折，每遇阴晦发作疼痛者，亦可服，不拘时候。
制备方法：用银石器或新锅内，先下猪脂熬成汁，去筋膜滓；次下黄蜡候熔，退火，候稍冷，方下密陀僧、黄丹、自然铜末，添慢火熬，以柳木篦子搅，候滴水不散为度，抬下锅子于冷地上，入朱砂、乳香、没药、白矾、更以篦子搅匀为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
