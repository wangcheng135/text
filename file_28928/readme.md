序号：28936
名称：补骨脂丸
组成：补骨脂（微炒）1两，木香1两，附子（炮裂，去皮脐）1两，摈榔1两，肉豆蔻（去壳）1两，青橘皮（汤浸去白瓤，焙）1两，桂心（去皮）1两，牛膝（去苗）1两，干姜（炮裂，锉）1两，鹿茸（去毛，涂酥炙令微黄）1两，硫黄（细研，水飞过）1两，腽肭脐（酒刷，炙微黄）1两，肉苁蓉（酒浸1宿，刮去皱皮，炙干）1两，川椒（去目及闭口者，微炒去汗）1两。
出处：《圣惠》卷九十八。
主治：下无虚冷气。
加减：None
功效：温中强力，暖胃思食。
用法用量：每服30丸，空心温酒送下。渐加至40丸。
制备方法：上为末，入硫黄研令匀，用白羊肾5对，去筋膜，细剁，入前药末相和拌了，溲白面裹，煻火中烧令面熟为度，取出药，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
