序号：35202
名称：乳香应痛丸
组成：川乌（生用，不去皮尖，大者破4边，小者破2边，锉如骰子大，用麻油煎令黄色勿焦，去油，焙）1两，草乌（生用，不去皮尖，每个破作2片，择白净者用，心黑者不用，锉如骰子大，一半用油制，如制川乌法；一半用盐半两拌和，水微淹着浸3昼夜，1日1度漉，转候日数足取出去水，再用干盐半两同炒，候干，去盐勿令焦）1两，红花（锉，焙）1两，当归（酒洗，锉，焙）1两，天台乌药（锉，焙）1两，干葛（锉，焙）1两，白芍药（洗，锉，焙）1两，苏木（锉）1两，生干地黄（洗、锉）1两，骨碎补（燎去毛）1两，鸡骨降真香（锉）1两，金毛狗脊（燎去毛）1两，乳香（杵碎，用香炉盛火，竿箬叶1片虚架在香炉口上，约去火2寸许。次将乳香置箬叶上，候炙熔滚，用水润湿小竹篱子，炒拨，觉滚定取出候冷，研）1两，没药（杵碎，酒浸，烂研，候诸药末成，刮入同丸）1两，自然铜（入银窝内，炭盖口及4周；煅红，入好黄子醋内淬2-3次，可研为度）1两。
出处：《朱氏集验方》卷一。
主治：脚气风损，白虎历节。
加减：None
功效：None
用法用量：每服30丸，空心温盐酒送下。凡脚气发作，先服驱风丹，大痛稍愈，即间服此药；候痛痊愈，专服此药。
制备方法：上为细末。用好白酒浓脚1碗，好头面半两煮糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌香菜、茵陈、豆豉、一切动风毒物。
附注：None
