名称：祛风和脉调气利湿化痰膏
组成：羌活2两，独活2两，僵蚕3两，威灵仙1两5钱，川乌1两5钱，片姜黄1两5钱，橘络2两，鸡血藤3两（后入），秦艽1两5钱，桑寄生2两，归尾2两，穿山甲2两，红花2两，川续断2两，香附3两（生），没药1两5钱（后入），乳香1两5钱（后入），乌梢蛇1两5钱，防风2两，茅苍术2两，赤芍2两，台乌2两，青皮2两，半夏2两（炙），稀莶草2两，麝香5钱（后兑）。
出处：《慈禧光绪医方选议》。
