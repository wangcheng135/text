序号：3314
名称：天雄浸酒
组成：天雄3两（炮裂，去皮脐），川椒3两（去目），干姜2两（炮裂），茵芋2两，附子2两（炮裂，去皮脐），肉桂3两（去皱皮），牛膝3两（去苗），川乌头2两（炮裂，去皮脐），白蔹半两，踯躅花3两（微炒）。
出处：《圣惠》卷二十五。
主治：一切风证。
加减：None
功效：充壮血脉，益精气，明耳目，黑髯发，悦颜色，除久风湿痹，怯筋脉挛急，强腰膝，倍力气。
用法用量：每服1小盏，后渐渐增之；其药滓晒干为散。每服2钱，以酒调下，空心及晚食前服。
制备方法：上锉细，用生绢袋盛，以好酒2斗渍之，春、夏5日，秋、冬7日。
临床应用：None
各家论述：None
用药禁忌：忌生冷、猪、鸡肉、豆豉。
附注：夏日恐酒酸，以油单裹瓶，悬于井中，近水即不酸也。
