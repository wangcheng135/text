序号：70661
名称：紫菀汤
组成：紫菀（去土）1两，人参1两，陈橘皮（汤浸，去白，焙）1两，紫苏茎叶1两，诃黎勒（炮，去核）1两，枳壳（去瓤，麸炒）1两，细辛（去苗叶）1两，郁李仁（去皮尖，研如膏）1两，杏仁（汤浸，去皮尖双仁，炒，研如膏）1两，桂（去粗皮）1两，赤茯苓（去黑皮）1两，甘草（炙，锉）1两，当归（切，焙）1两，大黄（锉，炒）半两。
出处：《圣济总录》卷一六三。
主治：产后上气咳逆，烦闷。
加减：None
功效：None
用法用量：每服2钱匕，水1盏，煎至7分，去滓温服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
