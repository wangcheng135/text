序号：12792
名称：石斛丸
组成：石斛2两（去根，锉），蛇床子1两，牛膝1两（去苗），桂心1两，菟丝子2两（酒浸3日，晒干，别捣为末），肉苁蓉2两（酒浸1宿，刮去皱皮，炙令干），人参1两（去芦头），鹿茸1两（去毛，涂酥炙令微黄），熟干地黄2两，杜仲1两（去粗度，炙微黄，锉），木香1两，薯蓣1两，白茯苓2两，附子2两（炮裂，去皮脐），巴戟1两，防风1两（去芦头），钟乳粉2两，干漆1两（捣碎，炒令烟出），泽泻1两，山茱萸1两，覆盆子1两，补骨脂2两（微炒），五味子1两，石龙芮1两，槟榔1两。
出处：《圣惠》卷九十八。
主治：None
加减：None
功效：补虚损，利腰膝，暖水脏，祛风冷，强气力，悦颜色。
用法用量：每日30丸，空心以温酒送下。4时宜服。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
