序号：4291
名称：木香丸
组成：木香半两，京三棱1两（微煨，锉），槟榔1两，桂心半两，附子1两（炮裂，去皮脐），没药半两，阿魏半两（面裹煨，面熟为度），桃仁1两（汤浸，去皮尖双仁，麸炒微黄），鳖甲1两（涂醋，炙令黄，去裙襴），芎半两，虻虫1分（去翅足，微炒），水蛭1分（微炒令黄），当归半两（锉，微炒），牡丹半两，赤芍药半两，硇砂半两（细研），川大黄1两半（锉碎，微炒），干漆1两（捣碎，炒令烟出）。
出处：《圣惠》卷七十九。
主治：产后恶血不散。积聚成块，在脐腹下，坚硬疼痛。
加减：None
功效：None
用法用量：每服20丸，以温酒送下，1日3-4次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
