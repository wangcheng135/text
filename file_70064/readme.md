序号：70080
名称：滋阴百补丸
组成：香附子1斤（炒去毛，分四制：酒浸4两，盐水浸4两，醋浸4两，童便浸4两，俱炒，焙干），益母草半斤（端午日采去土），当归6两（酒洗），川芎4两，熟地黄4两（酒洗），芍药3两（炒），白术4两，人参2两，茯苓2两，玄胡索（炒）2两，甘草（炙）1两。
出处：《摄生众妙方》卷十。
主治：妇人劳伤，血气不足，阴阳不和，乍寒乍热，心腹疼痛，不思饮食，尪羸乏力。
加减：None
功效：理气补虚，调经种子。
用法用量：每服50-60丸，空心缩砂汤或醋汤或酒或滚水任下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
