序号：42526
名称：青麟丸
组成：绵纹大黄10斤（先以淘米泔浸半日，切片晒干，再入无灰酒浸3日，取出，晒大半干，第一次用侧柏叶垫甑底，将大黄铺上，蒸1炷香久，取起晒干，以后每次俱用侧柏叶垫底，起甑定气不用；第二次用绿豆热浓汁，将大黄拌透，蒸1炷香，取出晒干；第三次用大麦熬浓汁拌透，照前蒸晒；第四次用黑料豆熬浓汁拌透；第5次用槐条叶熬浓汁拌透；第六次用桑叶；第7次用桃叶；第八次用车前草；第九次用厚朴；第十次用陈皮；十一次用半夏；十二次用白术，十三次用香附；十四次用黄芩；以上俱如前煎汤浸透蒸晒，第十五次用无灰酒拌透，蒸3炷香，取出晒透）。
出处：《续名医类案》。
主治：一切热症。肝火便秘，小便不利，肠风便血等症。
加减：None
功效：利便利尿，消火解毒。
用法用量：每服2钱。
制备方法：上为极细末，每大黄1斤，入黄牛乳2两，藕汁2两，梨汁2两，童便2两，如无童便，以炼蜜2两代之，外加炼蜜6两，为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌食辛辣。
附注：秘制清宁丸（《全国中药成药处方集》吉林方）。
