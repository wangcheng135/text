序号：65336
名称：大藿香散
组成：藿香叶1两，木香1两，沉香（去白）1两，肉豆蔻（面裹煨）1两，诃子（煨，去核）1两，人参（去芦）1两，良姜（炒）1两，麦糵（炒，大麦炒）1两，神曲（炒）1两，白茯苓（炒）1两，甘草（炒）1两，青皮（去瓤，麸炒）1两，厚朴（姜汁制，炒）1两，缩砂仁1两，白干姜（炒）半两。
出处：《医方类聚》卷一○二引《经验秘方》。
主治：心气脾胃，变为万病。
加减：常服加紫苏。
功效：None
用法用量：每服2钱，加生姜3片，大枣1枚（擘开），水1盏，同煎7分，盏中先放盐1捻：将药倾在内，空心热服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
