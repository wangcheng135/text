序号：76283
名称：紫霞丹
组成：消石1两，水银1两，雄黄1两，朱砂1两，硫黄（与水银结为砂子）1两，金箔1两半。
出处：《圣惠》卷九十五。
主治：痔漏瘰疬，筋骨疼痛。
加减：None
功效：补暖脏腑，添益精髓，延年驻颜，祛风逐冷，益子宫。
用法用量：每服3丸，空心以温酒送下。
制备方法：上为末，取1瓷瓶子，盐泥固济，待于，入药于瓶子内，其瓶盖钻作一窍，如半钱孔大，盖瓶口讫，仍纳煻灰中煨之，不得使令火大，恐药飞走，专候窍中阴气尽，以盐泥闭塞其窍，以火半斤，养3日满，即用火1斤，烧17日，候冷取出，于土坑中出火毒，3日后细研，以枣肉为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：忌羊血。
附注：None
