序号：58828
名称：秘传解毒丸
组成：贯众4两，山豆根4两，黄药子4两，牙消4两，寒水石4两，龙胆草4两，干葛4两，雄大豆4两，百药煎4两，紫河车4两，甘草节4两，薄荷4两，栀子4两，大黄4两，豆粉4两。
出处：《普济方》卷三十九。
主治：上盛下虚，水火不能升降，大便秘涩，小便不通，赤眼口疮，便红泻血，吐血泄痢不止；及诸积气块，小儿脾病，妇人经脉不通，男子打扑伤损，一切诸毒疮痍，咽喉肿痛。
加减：加山慈菇2两大妙。
功效：None
用法用量：每服1丸，以温米饮送下，日3次，不拘时候。
制备方法：上为细末，炼蜜为丸，秤1两作10丸，银箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
