序号：67554
名称：黄芩汤
组成：黄芩1两，赤芍药1两，甘草半两（炙）。
出处：《医方类聚》卷五十三引《神巧万全方》。
主治：阳明病发热脉浮，口干鼻燥，衄血，及挟热下痢，寒热胁痛，疹疮不出。阳明病，口干但漱水不欲咽者，必衄也；阳明脉浮，发热，口鼻中燥，能食者，亦衄。引钱氏方，挟热下痢，头痛胸满，大渴；或寒热胁痛，脉洪大而实者。挟热作疹疮不出，烦躁不得眠。
加减：None
功效：None
用法用量：每服4钱，水1盏，煎至7分，去滓温服。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：黄芩芍药汤（《伤寒总病论》卷三）、芍药黄芩汤（《准绳·类方》卷六引东垣方）、黄芩甘草汤（《得效》卷十一）。
