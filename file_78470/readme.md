序号：78486
名称：麝香膏
组成：麝香半两（细研），零陵香1两，土瓜根1两，白蔹1两，防风1两（去芦头），沉香1两，栀子花1两，当归1两，藁本1两，木兰皮1两，白僵蚕3分，鸬鹚粪1两，桃仁2两（汤浸去皮），冬瓜仁1两，辛夷1两，白茯苓1两，白芷1两，商陆1两，丁香1两，牛脂半升，猪脂半升，鹅脂半升。
出处：《圣惠》卷四十。
主治：面上百疾。
加减：None
功效：None
用法用量：夜临卧，洗手面了，干拭涂之。
制备方法：上锉细，绵裹，用酒1斗，浸1宿，取脂髓等，用慢火于银锅中，与药同煎，候白芷焦黄为度，去滓，入麝香和匀，于瓷盒中盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
