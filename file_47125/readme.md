序号：47140
名称：珍珠丸
组成：鸡内金5钱，轻粉8分，巴豆霜8分，六神曲9两，枳实（炒）3钱，珍珠1钱，牛黄8分，黄连2钱，陈胆星3钱，天竺黄3钱，川贝母3钱，半夏（制）3钱，桔梗3钱，僵蚕3钱，全蝎3钱，雄黄3钱，玄参3钱，夏枯草5钱。
出处：《江苏省中药成药标准暂行规定汇编》第一册。
主治：由热结痰多引起的咳嗽喘急，腹部膨胀，疳积，虫积，及惊风抽搐。
加减：None
功效：化痰，消积，镇惊。
用法用量：1岁每服2粒，每增1岁增加1粒，10岁以上者10粒为度，日服2次，温开水送下。
制备方法：珍珠、牛黄、轻粉另研极细，余药共轧为细粉，混合再研，过罗，用冷开水泛为小丸，另取朱砂细粉1两6钱为衣。
临床应用：None
各家论述：None
用药禁忌：忌食生冷及不易消化之物。
附注：None
