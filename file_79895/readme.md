序号：79911
名称：磁石丸
组成：磁石1两半（烧，醋淬7遍，捣碎，研细，水飞过），朱砂半两（细研，水飞过），补骨脂半两（微炒），肉苁蓉3分（酒浸1宿，刮去皱皮，炙干），神曲3分（炒令微黄），远志半两（去心），木香半两，覆盆子半两，五味子半两，熟干地黄3分，巴戟半两，桂心半两，牛膝3分（去苗），石斛3分（去根，锉），薯蓣半两，甘草半两（炙微赤，锉），车前子半两。
出处：《圣惠》卷三十。
主治：虚劳目暗。
加减：None
功效：None
用法用量：每服30丸，食前煎黄耆汤送下。
制备方法：上为末，入研了药令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
