序号：42455
名称：青黛丸
组成：青黛（水飞）2钱，黄连（猪胆汁炒）2钱，石膏（火煅）2钱，连翘（去瓤）3钱，桔梗1钱半，升麻1钱半，黄芩（酒炒）2钱，薄荷2钱，防风2钱半，半夏（姜制）2钱，牛胆南星2钱，贝母2钱，枳实（麸炒）1钱半，莪术（醋炒）1钱半，木香2钱，槟榔2钱，香附（童便浸）3钱，山楂肉2钱，砂仁1钱半，人参（去芦）1钱半，白术（麸炒）3钱，茯苓（去皮）3钱，甘草（炙）1钱，紫苏2钱，麻黄2钱。
出处：《古今医鉴》卷十三引郭师傅方。
主治：小儿癖疾发热，上攻牙龈，腮颔肿痛生疮；诸热痰嗽，伤风身热，并痘疹出不快，身极热眼黄；大人伤酒、伤食、伤风、伤气、头痛。
加减：None
功效：None
用法用量：每服5-7分或1钱，身热，薄荷汤送下；咳嗽，五味子、桑白皮汤送下；头痛身热，川芎汤送下；痘疹，酒送下；伤风身热，麻黄、紫苏汤送下。又治大人伤酒、伤食、伤气、伤风、头痛，每服5丸，姜汤送下。
制备方法：上为细末，稀糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
