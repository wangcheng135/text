序号：45536
名称：枸杞浸酒
组成：枸杞子半升，晚蚕沙（炒）半升，恶实（炒）1升，苍耳子（炒）1升，防风（去叉）2升，大麻子（炒）2升，茄子根2斤（洗令净，细切，蒸1复时，须是9月9日采），牛膝（酒浸，细切）1斤，恶实根（切，炒）1斤，桔梗（锉，炒）2两，羌活（去芦头，锉）2两，秦艽（去苗土，焙）2两，石菖蒲（9节者，锉）2两。
出处：《圣济总录》卷八。
主治：中风，身如角弓反张，及妇人一切血风，上攻下注。
加减：None
功效：悦泽颜色，滋润皮肤，退风，益气强力。
用法用量：每服1盏，空心、食前、临卧温服。常令有酒容。
制备方法：上以夹绢袋盛，用好酒3斗浸，密封闭，勿令通气，7日方开，开时不得面对瓶口。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
