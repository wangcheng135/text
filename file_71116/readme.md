序号：71132
名称：煨肝散
组成：苍术3两，缩砂（去皮）1两，柴胡（去芦）1两，厚朴（姜汁炙，去皮）1两，桔梗1两，芜荑3分，桂心2分（去皮），陈皮（去白）半两，远志（去心）半两，北紫苑半两，胡椒1分。
出处：《博济》卷三。
主治：脾元虚冷，滑泄不止，口内生疮，腹中冷，不思饮食。
加减：None
功效：暖胃消食，止泻。
用法用量：每服用豮猪肝4两，切作3片，每片用末1大钱许，掺于肝上，入葱白、莳萝、盐等，令有滋味，一重重布了，麸片裹之，煻灰火内煨令通熟，面焦黄色即得，去面取肝，空心服之，其面可2分以来，或细切肝，以散拌和，如作角子，如常煿熟食之亦得。并以薄米饮下之。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：忌生冷、毒物等。
附注：None
