序号：77877
名称：螽斯丸
组成：附子（生，去皮脐）3钱，白茯苓（去黑皮）3钱，白薇3钱，半夏（汤洗7次）3钱，杜仲（去粗皮）3钱，桂心3钱，秦艽3钱，厚朴（去粗皮）3钱，防风2钱，干姜（生）2钱，牛膝2钱，沙参2钱，细辛（去苗）半两，人参4钱。
出处：《产乳备要》。
主治：妇人无子。
加减：None
功效：None
用法用量：每日服50丸，空心任下。如觉无益，稍加丸数为度；如服7日后，阴阳觉有娠，3日后不可更服。
制备方法：上为细末，炼蜜为丸，如小豆大。
临床应用：不孕：臣妻年二十七岁，无子，服此药有娠；又残药与前太子中舍字文妻李氏，年四十岁无子，服此药十三日有娠。
各家论述：《中国医学大辞典》：此方削去干姜，易入当归，以和阳药之性，不致阳无以化，且免经水紫黑、胎息不育之虞。其秦艽、朴、夏，专理痰积；沙参、膝、薇，专清浊带，使子宫温和、阳施阴化，孕自成矣。
用药禁忌：忌食牛、马肉。
附注：赐子丸（《产乳备要》）、秦桂丸（《三因》卷十七）、白薇丸（《产乳备要》注引《施圆端效方》）、暖宫丸（《一盘珠》卷六）。按：《医学入门》有香附，无附子。《东医宝鉴·杂病篇》有当归。
