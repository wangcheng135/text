序号：9615
名称：仙方点白还黑丹
组成：生地黄（取汁）1盂，桑椹子（取汁）1盂，旱莲草（取汁）1盂，共放铁锅内熬之，干碾为面，听用；三膏末1两，母丁香5钱，没石子5钱，真铅粉5钱（炒）。
出处：《墨宝斋集验方》卷上。
主治：None
加减：None
功效：乌须发。
用法用量：依后开拔白日期，用小镊子拔去白须，即以黑笔点记，然后用鲜姜汁调前药少许点孔中。6-7日后，再出黑，永不白。
制备方法：上为细末，以小瓷罐贮之，塞口，勿令泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：点白还黑丹（《医学启蒙》卷三）。拔白日期：正月甲子日，二月初八日，三月十三日，四月十六日，五月二十五日，六月十四日、十九日，七月二十八日，八月十九日，九月十六日，十月初十、十三日，十一月初十、十二日，十二月初七日。
