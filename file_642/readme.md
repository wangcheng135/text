序号：642
名称：五噎散
组成：人参2两，茯苓2两，厚朴（去粗皮，锉。姜汁制，炒）2两，枳壳（麸炒，去瓤）2两，桂心2两，甘草（炙）2两，诃子（炮，去核）2两，白术2两，橘皮2两，白姜（炮）2两，三棱（炮）2两，神曲（炒）2两，麦蘖（炒）2两，木香（炮）半两，槟榔半两，蓬术（炮）半两。
出处：《三因》卷八。
主治：五种噎，食饮不下，胸背痛，呕哕不彻，攻刺疼痛，泪与涎俱出。
加减：None
功效：None
用法用量：每服2钱，水1盏，加生姜3片、枣子1枚，煎7分，空心温服；盐汤点亦得。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：五噎汤（《医学入门》卷七）。
