序号：81854
名称：聪耳芦荟丸
组成：生熟川军5钱，芦荟5钱，青黛5钱，柴胡5钱，龙胆草1两，黄芩1两，山栀1两，当归1两，青皮1两，青木香2钱，杜胆星2钱，当门子5分。
出处：《重订通俗寒伤论》。
主治：黄耳伤寒。两耳红肿黄亮，扪之焮热而痛，两腮红肿痛甚，耳中亦红肿，筑筑然痛，时有黄涎流出，声如蝉噪，两目发黄，身热体痛，恶寒无汗，背脊拘挛串痛，强直难伸，不能转侧，溺短赤涩，脉右濡滞，左浮弦略紧，舌苔白腻带黄，边尖色红，病由风热挟湿温时毒引起的流行性中耳炎。
加减：None
功效：苦寒清利，解毒泻火。
用法用量：每服8分至1钱，以麻黄连翘赤小豆汤送下。外用清涤耳毒水灌耳。
制备方法：神曲糊丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
