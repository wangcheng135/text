序号：872
名称：六枝乌金膏
组成：桑枝1尺，槐枝1尺，榆枝1尺，柳枝1尺，桃枝1尺，枸杞枝1尺（粗如小指，俱1寸截，劈4破）。
出处：《圣济总录》卷一三二。
主治：打扑损伤，坠车落马，一切肿痛恶疮。
加减：None
功效：None
用法用量：涂疮上。
制备方法：用油4两炒令焦黑，滤去滓，加铅丹半两，蜡1两，复熬令黑色，倾在瓷盒内候冷，以新汲水浸出火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：六枝膏(《医方类聚》卷一八八引《烟霞圣效方》)。①《圣济总录》：先用熏法熏出虫，乃用此膏涂疮。熏药方：猫儿粪、猪粪、乱发、粳米糠各等分，都置在一地坑内，用火烧，上以方砖复之，其砖心钻一窍，令烟出，疮就烟熏之。有虫及恶物出尽，以温浆水净洗，然后涂六枝乌金膏，用帛覆之。②本方方名，《普济方》引作“乌金膏”。
