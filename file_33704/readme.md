序号：33713
名称：灵砂归命丹
组成：巴豆（去心膜皮，炒熟，研如面油）315粒，牛黄（研）3两，龙脑（研）3两，麝香（研）3两，腻粉（研）3两，辰砂（研飞）9两，金箔（研）90片。
出处：《局方》卷十。
主治：小儿蕴积邪热，潮热不除，颊赤口干，心膈烦躁，痰涎不利，睡卧不安，或发惊痫，涎潮搐搦。积滞不消，下利多日，腹中（疒丂）痛，烦渴呕秽，服药调和不能愈者。
加减：None
功效：None
用法用量：每服2丸，金银薄荷汤下。更量岁数加减。如惊痫搐搦，用龙脑、腻粉、蜜汤送下。服药先以冷水浸少时，服之见效尤速。
制备方法：上合研匀，炼黄蜡6两，入白沙蜜3分，同炼令匀，为丸如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
