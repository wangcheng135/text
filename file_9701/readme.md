序号：9701
名称：牛膝丸
组成：牛膝半两（去苗），柏子仁1两，白薇半两，杜仲3分（去粗皮，炙微黄，锉），牡蛎1两（烧为粉），干姜半两（炮裂，锉），细辛半两，防风半两（去芦头），川椒3分（去目及闭口者，微炒去汗），附子3分（炮裂，去皮脐），泽兰3分，桂心半两，紫菀半两（洗，去苗土），黄耆1两（锉），熟干地黄1两，当归半两（锉，微炒），五味子半两，萆薢半两（锉），紫石英1两（细研，水飞过），甘草半两（炙微赤，锉），白茯苓3分，厚朴3分（去粗皮，涂生姜汁，炙令香熟）。
出处：《圣惠》卷八十一。
主治：产后风虚劳损，腑脏乏弱，四肢羸瘦，不思饮食。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
