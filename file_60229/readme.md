序号：60244
名称：逐痰丸
组成：紫海蛤（如鸡子大者1斤，火煅红，淬入童便内，如此3次，为末，却用鲜瓜蒌拌粉，捣千余下乃匀，稀稠得宜，作饼子，将麻绳穿，悬当风处吹干，为末）4两，南星（牛胆制），黄连、半夏4两（用姜矾者，滴香油数点，煮令透，炒黄色），陈皮（去白）2两，青皮（炒）2两，大黄（酒拌，九蒸九晒）5两，青黛1两，木香5钱。
出处：《济阳纲目》卷七十四。
主治：上焦郁火，痰涎壅盛，胸膈不利，咽喉烦躁，噎塞如有所梗，吐不出，咽不下。
加减：None
功效：None
用法用量：每服30-40丸，姜汤送下。
制备方法：上为末，姜汁、竹沥为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中南星、黄连用量原缺。
