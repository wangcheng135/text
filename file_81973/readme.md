序号：81989
名称：镇心丸
组成：防风5分，人参5分，龙齿5分，芎1两，铁精1两，当归1两，干地黄5分，黄耆1两，麦门冬5分（去心），柏子仁1两，桂心1两，远志5分（去心），白鲜皮3分，白术5分，雄黄1两（研），菖蒲1两，茯苓1两，桔梗1两，干姜5分，光明砂1两（研），钟乳半两（研）。
出处：《千金翼》卷十二。
主治：损心不能言语，心下悬急苦痛，举动不安，数数口中腥，客热心中百病。
加减：None
功效：None
用法用量：先服玄参3两，干地黄3两，黄耆3两，地骨皮3两，苁蓉3两，丹参5两，牛膝3两，五味子3两，麦门冬3两（去心），杏仁2两（去皮尖），细辛3两，磁石5两，生姜3两（切），茯苓3两，橘皮2两，韭子半升，柴胡2两（去苗），(口父)咀，以水3斗，煮取3升，分3服；后3日乃服上丸5丸，渐加至15丸，稍加至30丸，1日2次，食后饮服。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：慎腥臭等，常宜小进食为佳，宜吃酥乳，倍日将息；药服讫仰卧少时，即左右换卧，及数转动，腰底安物，不得劳役身心；服丸后二日风动，药气冲头，两眼赤痛，久而不愈者，法取枣根（直入地二尺者）白皮一握，水一升，煮取半升，服之即愈。
附注：None
