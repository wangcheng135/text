序号：66616
名称：鹿角胶煎
组成：鹿角胶2斤（捣碎，作4分，于铛中熬令色黄），紫苏子2升（以酒1升，研滤取汁），生地黄1斤（取汁），生姜1斤（取汁），黄牛酥1升，白蜜3斤。
出处：《外台》卷三十一引《广济》。
主治：五劳七伤，四肢沉重，百事不任，怯怯无力，昏昏欲睡，身无润泽，腰痛顽痹，脚弱不便，不能久立，胸胁胀满，腹中雷鸣，春夏手足烦热，秋冬腰膝冷痛，心悸健忘，肾气不理，五脏风虚。
加减：None
功效：补五脏，益心力，实骨髓，生肌肉，理风补虚，耳聪目明。
用法用量：空腹以酒调2合服之，1日2次。
制备方法：上六味，先煎地黄汁、苏子汁、生姜汁等二十余沸，次下酥、蜜，又煎3-5沸，次以蜜并胶末下之，搅令相得，胶消尽，煎即成矣，以器盛之。
临床应用：None
各家论述：None
用药禁忌：忌羊血、芜荑。
附注：None
