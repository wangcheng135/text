序号：48367
名称：神效龙脑膏
组成：生龙脑1钱（研），腻粉1分，水银半两（用腊茶半钱，好酥1块如枣大，以水银1处揩磨调和，杀研之），天南星2钱（洗，去皮脐，湿纸裹，熟灰内煨，研），石脑油（冬用）1两（复用1分）。
出处：《博济》卷四。
主治：小儿惊风搐搦，痰塞在心，戴眼直视，或眼不开，口噤，四肢或冷或热，大便或秘或泄。
加减：None
功效：None
用法用量：1-4岁，每服1-2丸，煎乳香汤送下，不得化破，服后3-5顿食久，取下恶物痰涎即效。
制备方法：上为末，以石脑油和丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：龙脑膏（《普济方》卷三七四引《全婴方》）。
