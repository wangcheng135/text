序号：48266
名称：神禹疏凿丸
组成：吴茱萸6两（拣），茴香（拣）4两，苍术（锉）4两，陈橘皮（锉）4两，青盐4两，川椒（去子及闭目者）3两，厚朴（锉，先去皮）3两，干姜（锉）3两（上以水1盏，同于瓷器内煮，水尽为度，取出培干），附子（炮，去皮脐）6两，枳实（去瓤，麸炒焦黄）6两，木瓜4两（干者），川乌（炮）3两，赤茯苓3两，木香1两半，天雄（如无，以大附子代之）3两，椒目（炒）3两，木通（炮）3两，石硫黄（舶上者，去沙石，别研）3两，阳起石2两半（真者，酒煮半日，别研极细）。
出处：《普济方》卷二二六引《卫生家宝》。
主治：元气衰少，膀胱津液虽藏而不化，又遇上焦有寒，不能引导阴阳，开通闭塞，决凟之官，自失所司，水道无从而出，又与气相壅，停留膈间，久而为饮，渐渍脾土，脾即得湿，而化郁蒸淫于肢腹，上浮于肺，肾尤虚者，则独乘于下部，是以令人素盛而今衰瘦，膈间常有水声，胁肋胀闷，上气喘息，四肢沉重，遍身浮肿，或遍疰腰脚，小便减少，饮食不思。
加减：None
功效：壮元气，补脾肾，逐痰饮，除湿肿。
用法用量：每服50丸，温酒或米汤饮送下。
制备方法：上为细末，然后入阳起石、硫黄2味拌匀，生姜自然汁打面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
