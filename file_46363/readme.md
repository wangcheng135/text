序号：46378
名称：活络汤
组成：白术6钱（净者），杜仲6钱（去粗皮量，制，炒去丝了），牛膝半两（去根，酒浸，焙干），附子半两（炮，去皮脐了），甘草2钱半（炙），人参2钱半（洗，去芦），官桂2钱半（去粗皮了），川姜7钱半（炮），当归1两2钱半（洗去土，酒浸1宿，焙干了）。
出处：《普济方》卷二四一。
主治：寒湿脚气，筋骨手足一切疼痛。
加减：None
功效：None
用法用量：每服半两，水2盏，煎至8分，去滓，温热服。病在上者，食后服；病在下者，食前服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：活络丹（《赤水玄珠》卷十一）、活血丹（《观聚方要补》卷二）。
