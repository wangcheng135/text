序号：58989
名称：秦艽扶羸汤
组成：软柴胡2钱，人参（去芦头）1两半，鳖甲（米醋炙）1两半，秦艽1两半，地骨皮1两半，半夏（汤洗7遍）1钱，紫菀茸1两5分，甘草1两5分，当归（洗，焙）1两1分。
出处：《杨氏家藏方》卷十。
主治：肺痿。骨蒸劳嗽，或寒或热，声嗄羸瘦，自汗，四肢怠惰，不美饮食。
加减：None
功效：None
用法用量：上(口父)咀。每服5钱，水1盏，加生姜5片，乌梅、大枣各1枚，煎至8分，去滓，食后通口服。
制备方法：None
临床应用：None
各家论述：《医方集解》：此手太阴、足少阳药也。柴胡、秦艽散表邪兼清里热，柴胡解肌热，秦艽退骨蒸；鳖甲、地骨滋阴血而退骨蒸，地骨皮凉血，退有汗骨蒸；参、草补气；当归和血；紫菀理痰嗽，润肺除痰；半夏发音声。肺属金，声之所从出也，有物实之，则金不鸣，燥湿除痰，则金清而声自开矣。有声嘶而哑者，是肺已损也，难治。表里交治，气血兼调，为扶羸良剂。
用药禁忌：None
附注：None
