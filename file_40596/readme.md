序号：40606
名称：虎睛丸
组成：朱砂1分（别研），铅白霜1钱，白僵蚕末1钱，珍珠末1钱，轻粉1钱，牛黄1钱，犀角屑1钱，青黛1钱，乳香1钱，胡黄连1钱，白附子1钱，香墨（烧）1钱，脑半钱，麝半钱。
出处：《幼幼新书》卷九引《庄氏家传》。
主治：小儿急慢惊风，涎实壮热。
加减：None
功效：None
用法用量：每服1丸，若急惊，以薄荷汤蜜水化下；若慢惊，用乳香薄荷汤化下；若心神烦躁，膈实喘粗，用轻粉龙脑水化下；若痫，用薄荷自然汁、金银汤化下；若天钓惊，以荆芥薄荷煎汤化下；若常服，1丸分作4丸，薄荷汤化下。
制备方法：上为极细末，糯米饭为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方名虎睛丸，但方中无虎睛，疑脱。
