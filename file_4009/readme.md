序号：4009
名称：日精月华丹
组成：炉甘石4两（轻松不夹石，如羊脑者佳。用三黄汤煅淬五次，如粉净末，用1两3钱），黄丹（飞去土）9钱7分，川连1两（去毛，切，童便浸1宿，晒干，取头末3钱4分），归身（水洗，晒干）7分4厘，朱砂（飞）5分，月石5分，白丁香（壮直者为雄。水飞去砂）3分4厘，轻粉（真）3分4厘，海螵蛸（去皮，水泡去咸味，晒干，取净末）3分4厘，硇砂（重汤取碗沿浮白）3分4厘，熊胆1钱（箬炙，勿焦），乳香（炙）1分7厘，没药（炙）1分7厘，麝香1分7厘，片脑1分7厘，珍珠5分，琥珀5分。
出处：《惠直堂方》卷二。
主治：一切星障胬肉，瞳神昏花，拳毛倒生等症。
加减：None
功效：None
用法用量：井水少许化，加米饮，软鸭毛蘸点。
制备方法：上各(石禹)千万如尘，加蜜4两，滚数沸去沫，煎熟，绢滤净3两，入碗重汤文武火熬，柳条不住手搅，至紫色滴水如珠，撚丸不粘手，牵蜜有丝，是其候也。即离火渐入丹石搅匀为丸。如蜜老，不必晒，蜜嫩放箬上晒干。金箔为衣，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
