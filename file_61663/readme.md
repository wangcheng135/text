序号：61679
名称：排气饮
组成：陈皮1钱5分，木香7分或1钱，藿香1钱5分，香附2钱，枳亮1钱5分，泽泻2钱，乌药2钱，厚朴1钱。
出处：《景岳全书》卷五十一。
主治：气逆食滞胀痛。脐腹痛，痛时多在脐腹周围，喜手按或温掩，伴见肠鸣自利，饮食少味，消化迟钝，舌苔白腻。
加减：如食滞者，加山楂，麦芽各2钱；如寒滞者，加焦干姜、吴茱萸、肉桂之属；如气逆之甚者，加白芥子、沉香、青皮、槟榔之属；如呕而兼痛者，加半夏、丁香之属；如痛在小腹者，加小茴香；如兼疝者，加荔枝核（煨熟，捣碎）用2-3钱。
功效：None
用法用量：水1钟半，煎7分，热服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
