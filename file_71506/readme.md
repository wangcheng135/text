序号：71522
名称：循络丸
组成：没药（别研）2两，乳香（别研）2两，虎骨（酥炙焦）2两，败龟（酥炙）2两，当归（洗，焙）2两，五灵脂2两，白附子（炮）1两，天麻（去苗，酒浸，焙）1两，全蝎（去毒，炒）1两，天南星（炮）1两，附子（炮，去脐皮）1两，川乌头（炮，去皮脐尖）1两，杜仲（去粗皮，炒）1两，地龙（去土炒）1两，威灵仙（去苗）1两，牛膝（去苗，酒浸1宿）1两，续断1两，乌蛇（酒浸，去皮骨，取肉，焙）1两，肉苁蓉（酒浸，炙）1两，朱砂（别研）1两。
出处：《杨氏家藏方》卷一。
主治：风痹。气滞，血脉凝涩，筋脉拘挛，肢节腰膝强痛，行履艰难。
加减：None
功效：None
用法用量：每服30丸，食前温酒送下。
制备方法：上为细末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：循络丹（《永乐大典》卷一三八七九引《可用方》。
