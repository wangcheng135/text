序号：51026
名称：草豆蔻散
组成：草豆蔻2两（去皮），半夏半两（汤浸7遍去滑），肉桂3分（去皱皮），人参3分（去芦头），木香半两，前胡1两（去芦头），高良姜1两（锉），白茯苓1两，附子1两（炮裂，去皮脐），陈橘皮3分（汤浸，去白瓤，焙），厚朴2两（去粗皮，涂生姜汁，炙令香熟），白术1两，甘草1分（炙微赤，锉）。
出处：《圣惠》卷五。
主治：脾胃气虚弱，脏腑积冷，呕吐宿食，四肢少力，面无颜色。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分、大枣3个，煎至6分，去滓，稍热服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
