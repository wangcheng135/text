序号：74747
名称：解毒凉血汤
组成：犀角（镑）2-4分，生地炭5钱至1两，双花炭5钱至1两，莲子心3-5钱，白茅根5钱至1两，花粉5钱至1两，紫花地丁3-5钱，生栀仁2-4钱，蚤休5钱至1两，生甘草2钱，川黄连3钱，生石膏2-4两。
出处：《赵炳南临床经验集》
主治：外科感染性疾病，毒热入于营血，相当于败血症阶段。
加减：None
功效：清营，凉血，解毒。
用法用量：先用石膏煮水后，去滓，煮群药服。
制备方法：None
临床应用：None
各家论述：方中犀角清热凉血，解毒定惊，生地炭、双花炭能入血分清血分之毒热，又能养阴护心，两药同伍可有犀角之功能；地丁，蚤休清热解毒；花粉、白茅根、莲子心养阴凉血清心；栀子、黄连清三焦毒热而重点在于清心热；生甘草解毒调和诸药。
用药禁忌：None
附注：None
