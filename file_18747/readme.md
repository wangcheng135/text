序号：18754
名称：人参养肺丸
组成：黄耆（去芦，蜜涂炙）1两8钱，人参1两8钱，白茯苓（去皮）6两，瓜蒌根6两，杏仁（去皮尖，麸炒）2两4钱，皂角子（炒）300个，半夏（洗，为末，姜汁作曲）4两（炒）。
出处：《局方》卷四（宝庆新增方）。
主治：肺胃俱伤，气奔于上，客热熏肺，咳嗽气急，胸中烦悸，涕唾稠粘，或有鲜血，上气喘急，不得安卧，肢体倦痛，咽干口燥，饮食减少，渐至瘦弱喘乏；或坠堕恐惧，渡水跌仆；或因叫怒，醉饱房劳、致伤肺胃，吐血呕血。
加减：None
功效：None
用法用量：每服1丸，食后细嚼，用紫苏汤送下；如喘急，用桑白皮汤送下。
制备方法：上为细末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
