名称：万应膏
组成：附子2两，红花2两，血余2两，莪术2两，桂枝2两，羌活2两，独活2两，僵蚕2两，秦艽2两，麻黄2两，当归2两，川乌2两，防风2两，威灵仙2两，草乌2两，大黄2两，赤芍2两，山栀2两，桃仁2两，三棱2两，白芷2两，全虫2两，五加皮2两，良姜2两，生地4两，香附4两，乌药4两。
出处：《中医伤科学讲义》。
主治：跌打损伤，负重闪腰，筋骨疼痛，胸腹气痛，腹胀寒痛。

用法用量：摊贴。
制备方法：麻油15斤，加丹60两，收膏后，再加肉桂粉5钱，苏合油5钱，及香料药2两。
