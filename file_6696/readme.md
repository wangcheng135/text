序号：6696
名称：加味黑虎保安丹
组成：川芎1两，苍术1两，草乌（炮）1两，何首乌1两，白芷1两，荆芥1两，防风1两，麻黄1两，细辛1两，石斛1两，甘草1两，川乌5钱，全蝎5钱，樟脑1钱，两头尖1两，豨莶草1两，威灵仙1两，白花蛇1条。
出处：《扶寿精方》。
主治：治一切风气。
加减：None
功效：None
用法用量：病在上食后服，病在下食前服，随引嚼服。左瘫右痪，急闷或四肢顽麻，并热酒送下；初病劳瘵，吐血腥或小便不通，并好酒送下；眉须脱落，大麻风，口眼斜，心热风，腰足疼，偏正头痛，夹脑风，或腰痛耳聋，肾脏风，并茶清送下；狂言心邪，防风煎酒，入飞朱砂在内送下；紫自癜风，并鼻赤，肺脏风，并防风汤送下；耳作蝉鸣，骨气风，川乌酒送下；小儿吃泥土，或饮食无味，缠脏风，并皂角子汤送下；心气胀闷，噎食喷嚏，咳嗽，女人赤白带下，并生姜汤送下；手心退皮，天麻或益智仁汤送下；迎风冷泪，米汤送下；筋骨疼痛，乳香汤送下，膀胱肿痛，醋汤送下；指头破裂，烧梨汤送下；诸淋沥或发鬓退落，并炒盐汤送下。
制备方法：上为末，炼蜜和杵为丸，如樱桃大。
临床应用：None
各家论述：None
用药禁忌：忌过饮，及热物欲事，避风。
附注：None
