序号：58648
名称：秘传抱龙丸
组成：白附子2两（炮），胆星1两（姜；炒），羌活1两，僵蚕1两（炒，去嘴），前胡1两，橘红1两5钱，天竺黄2两，天麻1两（纸包，煨），青皮1两（醋炒），全蝎1两（米炒），黄芩（生）8钱，花粉2两，生黄连5钱，南薄荷1两，真琥珀3钱，盔沉香1钱，抱茯神1两，双钩藤1两。
出处：《寿世新编》。
主治：小儿惊风发搐，咳嗽痰喘，舌赤唇干，口渴便短，手足抽掣，痰涎塞盛，一切惊痫风热。
加减：None
功效：None
用法用量：每服1丸，重则2丸，开水化下。
制备方法：上为细末，炼蜜为丸，如芡实大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：面青，唇舌淡白，小便清利者不可服。
附注：None
