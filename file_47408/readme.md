序号：47423
名称：祛邪饮
组成：槟榔（煨）1两，大黄（锉，炒）1两，阿魏（研）1钱，丹砂（研）1分，麝香（研）1分，秦艽（去苗土）1分。
出处：《圣济总录》卷九十三。
主治：传尸劳，气满喘咳，大肠秘涩，羸瘦无力。
加减：None
功效：None
用法用量：每用东南桃柳梢各7枚，各长1握，细锉，青蒿1握，以童子小便1升半，隔夜浸至来日五更鸡鸣时，并药同煎10余沸，倾出，将细绢滤过，分为2服，空腹，相次服之。至明即转下恶物，似头发、马尾、鱼脑，转不止自住。
制备方法：上药除麝香丹砂外，细锉如麻豆大，相和令匀，分为5服。
临床应用：None
各家论述：None
用药禁忌：宜食软饭粥将息，7日后用劳药补之。
附注：None
