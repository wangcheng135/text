序号：82749
名称：中和种子丸
组成：菟丝子（拣净水淘，舂去粗皮，用无灰酒煮烂，以丝出为度，捣如泥，为薄饼，晒干，磨为末）4两，白茯苓3两，山茱萸（酒拌蒸，取净肉）4两，怀熟地（取大生地5两，酒洗净，用砂仁末3钱、好酒半斤，拌浸1宿，置瓷器，坐砂锅内，隔汤炖黑烂为度，另捣），怀山药3两，枸杞子（甘州者佳）4两，远志（甘草汤泡，捶去骨取肉，再用甘草场煮，晒干）2两，车前子（净，用泔浸蒸，晒干）2两，覆盆子（去蒂，酒蒸，晒干）4两，麦门冬3两（去心），五味子2两（辽东与北来者佳），鱼膘胶4两（用牡蛎粉炒成珠，去牡蛎），嫩鹿茸4两（酥油慢火炙透），当归身（酒洗，晒干）3两，柏子仁（去壳，取白净肉）3两（另捣），人参3两，川牛膝（盐酒炒）3两，沙苑蒺藜4两（微焙为末，入药；另取2两煮膏，同炼蜜为丸），川杜仲（盐酒炒）3两。
出处：《医学正印》卷上。
主治：元禀虚弱，或因色欲过度，以致气血两亏，心肾不交，百病内蚀，不能成育者。
加减：阴虚火盛，加盐、酒、蜜三制炒黄柏、知母各3两；虚寒无火甚，加童便制熟附子、肉桂各1两，或去附、佳，加肉苁蓉(去鳞膜)、巴戟肉、补骨脂(盐酒炒)各3两；肥人有痰，加广橘红3两，减熟地2两；瘦人上焦有火，加姜汁炒黄连2两；梦遗滑精，加蜜炙黄柏4两，砂仁末2两，酸枣仁(炒香)3两。
功效：None
用法用量：空心、淡盐汤送下3钱，临卧灯心汤送下2钱。
制备方法：上药除别捣者外，磨为极细末，隔汤炼真川蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
