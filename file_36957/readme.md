序号：36967
名称：夜光散
组成：宣黄连2两，诃子2两，当归1两，铜绿1钱，上（口父）咀，以河水3升，同浸两昼夜，于银石器熬取汁，约一大盏，纳8分来得所，看滓黑色为度，生绢纽取汁，再上文武火熬，槐柳条搅，滴水成珠为度，入后膏和剂：豭猪胰子2个（先去脂，以禾秆药梢裹，搅水内搓洗，令脂尽切，入黄连膏内，煮黑色，取出用之），芦甘石1两（童便1大碗，炭火烧红淬之，令小便尽，芦甘石粉白为度，研细末），黄丹4两（新汲水淘净，飞细，焙干），鹅梨10个（竹刀切去皮心，生布取汁用），青盐6钱（研细），蜜1斤（炼去蜡滓，一沸止）。
出处：《卫生宝鉴》卷十。
主治：赤眼翳膜昏花。
加减：None
功效：None
用法用量：每用铜筯蘸药，点入眼大眦内。
制备方法：上将梨汁，甘石膏子内熬5-7沸，入青盐，以杨柳枝搅至褐色，倾入瓷瓮，冷冰水浸，拔去火毒，腊月会为妙，正月、十一月次之，余月各不可合。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方制成丸剂，名“夜光丸”（见《医学纲目》卷十三）。
