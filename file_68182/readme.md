序号：68198
名称：黄耆酒
组成：黄耆1两（去芦），防风1两（去芦），官桂1两（不见火），石斛1两（去根），虎骨1两（酥炙），当归1两（去芦），白芍药1两，木香1两（不见火），云母粉1两，茵陈叶1两，仙灵脾1两，天麻1两，萆薢1两，甘草1两，川续断1两。
出处：《济生》卷三。
主治：风湿痹。身体顽麻，皮肤燥痒，筋脉挛急，言语謇涩，手足不遂，时觉不仁。
加减：None
功效：None
用法用量：每服1盏，温服之，不拘时候。常令酒气相续为佳。
制备方法：上锉，如麻豆大，以生绢袋盛，以好酒1斗浸之，春5日，夏3日，秋7日，冬10日。
临床应用：None
各家论述：None
用药禁忌：None
附注：《医方类聚》有白术。
