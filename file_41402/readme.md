序号：41417
名称：金砂丹
组成：白花蛇（酒浸，去皮骨取肉）1两，乌蛇（酒浸，去皮骨取肉）1两，蝎梢（炒）1两，白僵蚕（炒）1两，犀角屑1两，玳瑁屑1两，天麻1两，人参1两，白茯神1两，甘草（炙）1两，龙脑（研）1分，麝香（研）1分，朱砂（研）5两，牛黄（研）1分，雄黄（研）1分，珍珠（研）1分，天竺黄（研）1分，金箔230片，银箔230片，铁粉（研）1分。
出处：《御药院方》卷一。
主治：中风涎潮，失音不语，面赤，脉大数急，迷闷，口眼斜，俱系风热之疾，半身亸曳；或伤寒刚痉瘈疭，妇人产后角弓反张。
加减：None
功效：None
用法用量：每服1丸，人参汤化下；或竹叶汤、新汲水亦得。
制备方法：上为细末，真石脑油为丸，每两作10丸，瓷盒内收。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
