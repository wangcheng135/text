序号：38846
名称：泻肝汤
组成：荆芥穗半两，连翘半两，羌活（去芦头）半两，牡丹皮半两，黄芩（去黑心）半两，杏仁（去皮尖双仁，炒，研）半两，当归（切，焙）半两，芍药半两，栀子仁半两，鸡苏（去梗）半两，虎杖半两，麻黄（去根节）半两，大黄（锉，炒）半两。
出处：《圣济总录》卷四十一。
主治：肝脏积热，气昏血涩，或因食酸物过多，肝中血积不散，气血俱病，两胁下非时气动，每动左胁下有声，右胁相应，日渐胃脘结块，使人心腹满闷，上冲咽喉头目不利，睡卧不安，如虫所啮。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，生姜2片，煎至7分，去滓温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
