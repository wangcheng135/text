序号：81599
名称：镇心朱砂丸
组成：朱砂1两半（细研，水飞过），龙脑1分（细研），牛黄半两（细研），龙齿1两，天竹黄1两（细研），虎眼2对（酒浸1宿，微炙），蛇骨皮3分，紫石英1两（细研，水飞过），白僵蚕3分（微炒），马牙消1两（细研），金箔100片（细研），银箔100片（细研），赤筋1两，当归3分（锉，微炒），蔓荆子半两，麝香半两（细研），犀角屑1两，远志1两（去心），铅霜1两（细研），人参1两（去芦头），茯神1两半（去木），麦门冬1两半（去心，焙），独活1两，甘菊花1两，防风1两（去芦头），子芩1两，甘草半两（炙微赤，锉）。
出处：《圣惠》卷六十九。
主治：妇人血风，气壅多惊悸，头目旋痛，烦热恍惚。
加减：None
功效：None
用法用量：每服10丸，食后、临卧以荆芥薄荷汤加竹沥半合送服。
制备方法：上为末，入研了药，更研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
