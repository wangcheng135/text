序号：26289
名称：朴消煎
组成：朴消1斤，芒消8两，寒水石4两，石膏2两，金2两。
出处：《千金》卷十六。
主治：服石所致积热，困闷不巳。
加减：None
功效：清解悍毒，搜涤燥热。
用法用量：每服方寸匕，白蜜1合和冷水5合，搅和令消，顿服之，日3次。热定即止。
制备方法：上五味，先纳二消于8升汤中搅之令消，以纸密封一宿，澄取清，纳铜器中，别捣寒水石、石膏碎如豆粒，以绢袋盛之，纳汁中，以微火煎之，候其上有沫起，以箸投中，着箸如凌雪凝白，急下泻着盆中，待凝取出，烈日晒干。
临床应用：None
各家论述：1.《千金方衍义》：膏粱金石皆富贵之所，常嗜受如持虚，每致热发毒蕴，百治罔效，惟朴消煎可以荡涤脏腑，专取石药之寒以治石药之毒，如水沃火，顷刻冰消。贵高人加金以治五金之毒，皆同气相招之意。_x000D_
2.《医略六书》：石药气悍，积毒伤阴，故消瘅大渴，肌肉顿削焉。朴消、芒消荡涤积热之结滞；石膏、寒水石清解燥火以存阴。俾石药之悍毒顿消，则消瘅大渴自退，而肌体无不丰腴矣。此清解悍毒之剂，为搜涤燥热之专力。
用药禁忌：None
