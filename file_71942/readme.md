序号：71958
名称：滋阴种子丸
组成：知母2两（去毛皮为末，1两乳汁浸透，1两黄酒盐浸透，晒干），天门冬（去心）2两，麦门冬（去心）2两，黄柏2两（去粗皮为末，1两乳汁浸透，1两黄酒盐浸透，晒干，炒赤色），熟地黄（黄酒捣如泥，即和众药）2两，桑椹子2两，菟丝子（酒煮，晒干）2两，生地黄（黄酒洗过，与熟地黄总捣一处）2两，何首乌（黑白二色均用，同黑豆煮2次，去皮晒干）2两，干山药1两，牛膝（去芦）2两，黄精2两（对节生者真，酒蒸熟，与熟地捣一处），辽五味5钱，白茯苓（去皮，去红丝）1两，枸杞子1两5钱，柏子仁（水浸1日，连壳水磨成浆，绢袋滤汁去壳，又将水面浮油掠尽，去水，存结底，晒干）1两。
出处：《医学正印·男科》。
主治：男子精亏无子及阴虚有火者。
加减：None
功效：None
用法用量：上为细末，炼蜜为丸，如梧桐子大。每服70-80丸，早晨淡盐汤送下，服至百日。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
