序号：36098
名称：参茸养元膏
组成：天门冬3钱，紫霄花3钱，甘草3钱，川续断3钱，熟地黄3钱，牛膝3钱，菟丝子3钱，远志3钱，虎骨3钱，淡苁蓉3钱，杏仁3钱，番木鳖3钱，谷精草3钱，麦门冬3钱，蛇床子3钱，大附子3钱，生地黄3钱，官桂3钱。
出处：《中国医学大辞典》。
主治：男女忧思抑郁，劳倦色欲，诸虚百损，阳痿阴弱。
加减：None
功效：却病延年，助阳补髓，养气宁神，调营和卫，固本保元。
用法用量：每用1张，贴脐上或腰际，1月再换，常常贴之。
制备方法：上用花油2斤4两，熬枯去滓。次入人参、鹿茸、麝香，再后入母丁香、雌黄、雄黄、阳起石、没药、乳香、鸦片灰、木香、蟾酥、沉香、赤石脂、花龙骨各3钱、松香（制）4两、蛤蚧1对、黄丹8两。上为细末，收入成膏，每张摊3钱重。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
