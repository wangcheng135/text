序号：58297
名称：真料济阴丹
组成：净香附子8两（2两（口父）咀，好醋浸；2两无灰酒浸；2两盐汤浸；2两用2-3岁童便浸1日夕，长流水洗；如无童便，用姜汁，艾煎汁浸亦可，炒干），乌药半两，当归半两，泽兰半两，赤芍半两，百草霜半两，五灵脂半两，陈皮半两，熟苍术半两，川芎半两。
出处：《医方类聚》卷二一二引《仙传济阴方》。
主治：胎前产后百疾。
加减：无孕，加三棱、莪术、白芷各1两，桂半两，4味为末，醋糊为丸服；产后，加百草箱或添益母草，蜜浸炒干为末，入前药内用。
功效：None
用法用量：每服30丸，诸疾各汤使。
制备方法：上为末，醋煮面糊为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
