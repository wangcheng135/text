序号：41370
名称：金泥膏
组成：菖蒲（一寸九节者用）1两，远志（去心）1两，钩藤1两，人参（去芦头）半两，龙胆草半两，甘草（炙）半两（上为细末），水银1分，牛黄（别研）1钱，麝（研）1钱，金箔20片。
出处：《幼幼新书》卷十四引张涣方。
主治：伤寒邪热乘心，兼发惊病。
加减：None
功效：None
用法用量：每服1豆大，薄荷汤化下。
制备方法：将水银研如泥，与诸药一处拌匀，用蜜半斤，酥4两，用银锅或石锅中，先入水2升，除出金泥，酥，蜜外，先入诸药，慢火熬至1升，新绵滤去滓，方再下酥，蜜、金泥搅匀，用柳枝不住手搅，熬成膏，用瓷盒盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
