序号：5301
名称：平气丸
组成：巴豆（去壳）4两，黑牵牛4两，萝卜子4两，丁香皮1两，丁香1两，胡椒1两，肉桂（去粗皮）1两，五灵脂（炒）1两，青橘皮（去白）1两，桂花1两，陈橘皮（去白）1两，缩砂仁1两。
出处：《杨氏家藏方》卷六。
主治：胸膈噎闷，不思饮食，伤酒吐逆，气胀腹痛，小肠疝气。
加减：None
功效：None
用法用量：每服15丸，胸膈噎闷，不思饮食，煎葱白汤送下；中酒、吐酒，细嚼，煨生姜汤送下；气痛，煎石菖蒲汤迭下；气胀、面肿，煎大腹皮汤送下；疝气、小肠气，煎茴香汤送下；妇人血气，腹内刺痛，煎当归汤送下，不拘时候。
制备方法：上用陈粟米1升，炒巴豆黑色，去巴豆，将粟米与众药捣罗为末，醋煮面糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
