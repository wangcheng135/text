序号：48098
名称：神妙槐胶丸
组成：槐胶半斤（细捣，好酒5升浸1宿，后入锅内煎，滤汁1升），苏枋木4两（细锉，用长流水5升，煎至1升，滤过，与槐胶汁相和，次入后药），丹砂（研）1分，阿魏（研）1分，乳香（研）1分，没药（研）1分（以上6味，将后4味再研匀，入前2味药汁内，慢火再熬数沸，住火，次入后药），白花蛇（1全条，酒浸1宿，取肉炙）1两半，天麻1两，天南星（炮）1两，白附子（炮）1两，附子（炮裂，去皮脐）1两，甘草（炙）2两。
出处：《圣济总录》卷七。
主治：摊缓风。
加减：None
功效：None
用法用量：每服半丸，生姜酒嚼下，每日3-4次。
制备方法：上将后6味为末，与前药膏和捣五七百杵，丸如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：槐胶丸（《普济方》卷九十三）。
