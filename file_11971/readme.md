序号：11977
名称：仙灵脾丸
组成：仙灵脾3分，防风半两（去芦头），羌活3分，白附子3分（炮裂），天麻1两，天南星半两（炮裂），犀角屑3分，木香半两，槟榔半两，羚羊角屑3分，乳香3分（细研），虎胫骨3分（涂酥，炙令黄），桂心半两，附子3分（炮裂，去皮脐），当归3分（锉，微炒），牛膝3分（去苗），白僵蚕半两（微炒），鹿茸3分（涂酥，炙令黄，去毛），石斛3分（去很节），麝香1分（细研），海桐皮3分（锉），干蝎半两（微炒），乌蛇3两（酒浸，炙令黄，去皮骨）。
出处：《圣惠》卷十九。
主治：风湿痹，肢节疼痛，身体手足不遂。
加减：None
功效：None
用法用量：每服30丸，食前以温酒送下。
制备方法：上为末，入研了药令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
