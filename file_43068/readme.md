序号：43083
名称：养心滋肾丸
组成：人参1两，芡实（去壳）1两，酸枣仁（炒）2两，天冬（去心）2两，远志（甘草水泡，去心）1两，当归（酒洗）1两，莲蕊1两，柏子仁（去油，炒）1两，石菖蒲（去毛）6钱，熟地黄（酒蒸）2两，五味子1两，麦冬（去心）2两，知母（去毛，酒炒）2两，白芍（盐酒炒）1两5钱，白茯神（去皮木）1两，莲肉（去心皮）1两，牡蛎（火煅）1两，怀山药（炒）3两，生地黄（酒洗）2两，黄柏（去皮，盐水炒）2两。
出处：《寿世保元》卷五。
主治：遗精。
加减：None
功效：养元气，生心血，健脾胃，滋肾水，止盗汗，除遗精，降相火，壮元阳。
用法用量：每服70丸，空心盐汤送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
