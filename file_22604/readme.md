序号：22612
名称：吴仙丹
组成：白茯苓、吴茱萸（汤泡去沫）各等分。
出处：《百一》卷五引常子正方。
主治：痰饮上气，不思饮食，小便不利，头重昏眩，或头疼背寒，呕吐酸汁。
加减：None
功效：None
用法用量：每服30丸，熟水吞下，酒饮亦可，不拘时候。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：头痛：中丞苦痰饮，每啖冷食饱，或晴阴节变即发头疼背寒，呕吐酸汁，数日伏枕不食，而《千金》大五饮丸之类皆不效。宣和初，为顺昌司录，于太守蔡公安持达道席上得此方，服之遂不再作。每遇饮啖过多，腹满，服50-70丸，不2-3时便已旋，作茱萸气，酒饮随小水而去。前后痰药甚众，无及此者。
各家论述：None
用药禁忌：None
附注：茱苓丸（《得效》卷四）。
