序号：73254
名称：斑蝥丸
组成：斑蝥1分（以糯米拌炒，米黄为度，去头足翅），人参3分（去芦头），地胆1分（以糯米拌炒，米黄为度，去头足翅），当归3分，川升麻3分，麦门冬1两（去心，焙），白术3分，桂心3分，川大黄3分（锉碎，微炒），钟乳粉3分，甘草1分（炙微赤，锉），防风半两（去芦头），续断3分，麝香1分（细研），白矾1两（烧令汁尽）。
出处：《圣惠》卷六十六。
主治：转脉瘘。发于颈，肿痛，寒热，出脓水不止。
加减：None
功效：None
用法用量：每服10丸，以温酒送下，1日3次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
