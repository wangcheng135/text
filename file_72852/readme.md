序号：72868
名称：雄胆散
组成：黄芩3钱（生用），黄连3钱（生用），栀仁3钱（炒黑），制梅干5钱（煅存性），青黛5钱，雄黄1钱，硼砂3钱，鸡内金1钱（不见水），人中白5钱，枯矾1钱。
出处：《焦氏喉科枕秘》卷二。
主治：喉科七十二症。
加减：如有腐臭。急用蚌水漱净，或用猪皂、草乌、柏子和捣，加水去渣，灌净，吹之。
功效：None
用法用量：每用少许，吹入患上，1日夜吹十余次，徐徐咽下。流出痰涎渐愈。
制备方法：上药依法制度，为极细末，入冰片6分，麝香3分，再匀每药末5钱，加西黄2分，铜青5分，熊胆5分，珍珠5分，儿茶8分，共为极细末，以瓷瓶封收紧密。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
