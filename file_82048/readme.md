序号：82064
名称：马蔺花丸
组成：马蔺花（醋炒）1两，川楝实1两，橘核1两，海藻（洗净）1两，海带（洗净）1两，昆布（3味俱盐、酒洗，炒）1两，桃仁（去皮尖）1两，厚朴（姜制）5钱，木通5钱，枳实（麸炒黄色）5钱，玄胡索（杵碎，炒）5钱，肉桂（去粗皮）5钱，木香5钱，槟榔5钱。
出处：《医学正传》卷四。
主治：七疝(疒颓)气，及妇人阴(疒颓)坠下，小儿偏坠。
加减：脉沉细，手足逆冷者，加川乌头1个（5钱，炮）。
功效：None
用法用量：每服50-70丸，或酒，或姜盐汤送下。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：秘传马蔺花丸（《松崖医径》卷下）。
