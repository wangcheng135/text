序号：80920
名称：飞龙夺命丹
组成：蟾酥2钱（干者，老酒化），血竭1钱，乳香2钱，没药2钱，雄黄3钱，轻粉半钱，胆矾1钱，麝香半钱，铜绿2钱，寒水石1钱，朱砂2钱，海羊21个（即蜗牛，连壳用），天龙1条（即蜈蚣，酒浸，炙黄，去头足），脑子半钱（如无亦可）。
出处：《急救仙方》卷二。
主治：疔疮、发背、脑疽、乳痈、附骨疽、一切无头肿毒恶疮、狐臭。
加减：None
功效：消肿败毒。
用法用量：若丸不就，酒煮面糊为丸。每服只2丸，先用葱白3寸，令病人嚼烂，吐于手心，男左女右，将丸子裹在葱白内，用无灰热酒3-4盏送下。于避风处以衣被盖覆，约人行5-6里之久，再用热酒数杯，以助药力，发热大汗出为度。如病重汗不出，再服2丸，汗出即效。若初服2丸，但消3-5病，重者再进2丸。如疗疮走黄过心者，难治。汗出冷者亦死。如病人不能嚼葱，擂碎裹药，用酒送下。疮在上，食后服；在下，食前服。
制备方法：上为细末，将海羊研作泥，和前药为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：服药后忌冷水、黄瓜、茄子、油面、猪、羊、鱼肉。
附注：渊然真人夺命丹（《丹溪心法附余》卷十六引《仙传济阴方》）、再生丹（《增补内经拾遗》卷四）
