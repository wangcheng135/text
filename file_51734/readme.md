序号：51749
名称：重剂排石汤
组成：三棱15g，莪术15g，穿山甲9g，皂角刺9g，桃仁12g，赤芍15g，乳香9g，没药9g，牛膝15g，青皮12g，白芷15g，薏苡仁30g，枳壳15g，厚朴15g，金钱草30g，车前子15g。
出处：《急腹症方药新解》。
主治：尿路结石，石体较大，停留嵌顿时间较久，或结石部位有粘连。
加减：None
功效：破血破血，活血祛瘀。
用法用量：每日1剂，煎取150-200ml，每剂2-3煎，每日服2-3剂。
制备方法：None
临床应用：None
各家论述：方中三棱，莪术、桃仁、赤芍、乳香、没药活血祛瘀止痛；皂角刺消肿，下胞；薏苡仁利水排脓；白芷消肿止痛。
用药禁忌：None
附注：None
