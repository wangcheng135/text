序号：82876
名称：丹砂丸
组成：朱砂10两（作小块子者）春蜜3升，秋蜜3升。
出处：《圣惠》卷九十八。
主治：百病。
加减：None
功效：安五脏，坚筋骨，驻颜容，久服聪耳明目，却老延年，充益肌肤，能耐寒暑。
用法用量：每日空心置1丸于舌上，以自然津液咽之。
制备方法：用大竹1截，可3尺来，去却青皮一重，留底节，将砂入筒内，投蜜渍之，坐竹简安大鼎内，架定，用水煮竹筒，以炭火慢煮，日夜专看伺之，蜜耗，旋添蜜；自5月5日午时，日夜煮至7月7日住，取出，用暖水浴过，入一绛纱袋子，悬于一通油瓷瓶内，勿令着底及四边，以绳子系口，悬于一净井内，去水面5寸以来，不用汲着水，7日7夜满，取出，将砂于乳钵内研一千遍，建一高台，置乳钵于台上，朝太阳气，用纱笼罩却，免鸟雀粪，夜即朝太阴气，遇雨即收却，每日研一千遍，后即于台上置，至9月9日即止；用青州枣瓤为丸，如绿豆大，于瓷器中盛。
临床应用：None
各家论述：None
用药禁忌：忌羊血、咸水。
附注：None
