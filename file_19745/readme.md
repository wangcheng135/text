序号：19752
名称：壮元丹
组成：牛膝（酒浸3日）2两，苁蓉（酒浸1日）2两，熟地黄2两，川芎2两，覆盆子2两，石斛1两半（去根），菟丝子1两（酒浸3日），当归1两半，续断1两半，巴戟1两半，白茯苓1两半，山茱萸肉1两半，枸杞子1两半，肉桂1两半，五味子1两半，防风1两半，杜仲（炒）1两半。
出处：《普济方》卷二二四。
主治：肝肾虚，精血不足，眼昏黑花，迎风有泪，头晕耳鸣；或肾风下痉，腰脚沉重，筋骨痠疼，步履无力，阴汗盗汗，湿痒生疮。
加减：None
功效：耐寒暑，进饮食，黑髭发，润肌肤，壮筋骨。
用法用量：每服50丸，空心、食前以盐汤酒送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
