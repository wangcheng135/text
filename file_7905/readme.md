序号：7905
名称：加减通圣散
组成：防风5钱（去芦），连翘3钱（去蒂），川芎5钱，白芍药2钱，当归3钱（酒浸洗），薄荷2钱，荆芥穗5钱，麻黄3钱（去根节，汤泡），栀子3钱（去壳），桔梗5钱，枳壳（去瓤，面炒），石膏各5钱，甘草3钱，滑石3钱，黄芩3钱（去朽），柴胡5钱，黄连5钱，黄柏3钱，生地黄3钱5分（酒制），羌活5钱，熟地黄3钱5分（酒制），锦纹大黄6两，芒消1两，皂角刺1两（独生者，去尖）。
出处：《医学正传》卷六。
主治：疠风。
加减：None
功效：泻恶毒秽积。
用法用量：每服用水1碗半，煎至1碗，空心服，日进2服。5-6日后又进2服，待补养完，又行2次，然后服神仙紫花丸。
制备方法：上切细，分八服。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
