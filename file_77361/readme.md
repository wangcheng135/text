序号：77377
名称：遇仙如意丸
组成：白茯苓（去皮）1钱，陈皮（去白）1钱，青皮（去瓤）1钱，丁香2钱，木香2钱，人参2钱，白术（煨）3钱，白豆蔻仁3钱，缩砂仁3钱，官桂（去皮）3钱，京三棱（炮）3钱，石菖蒲（炒去毛）3钱，远志（去心）3钱，广荗（炮）3钱，干山药半两，甘草（去皮）少许，香附子3两，牵牛头末8两。
出处：《普济方》卷一一五引《瑞竹堂方》。
主治：治诸风疾病，及患恶疮；妇人月事不见，产后腹中恶物，气蛊，水蛊、气积，气胀，痃癖。
加减：若风疾，加地骨皮1两。
功效：None
用法用量：每服120丸，看老幼虚实加减丸数，临卧温水送下。气蛊水蛊，每服300丸，1服立消。此药微利3-5行，欲止脏腑，但吃凉水1口便住。利后服甘露散补之。
制备方法：上为细末，好醋为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：凡食不可太饱，可食粥5-7日。忌生冷、硬物、酒、肉、鱼、面。
附注：遇仙如意丹（《医统》卷三十）。
