序号：26722
名称：延寿内固丹
组成：辰砂3两半，黑附子1两（生），白术1两，没药半两，木香1两，胡芦巴1两半，硇砂半两，人参1两。
出处：《鸡峰》卷十五引马子得道人传《南岳魏夫人方》。
主治：男子脾肾气衰，有积及腰背冷，面黄，瘫痪中风；妇人虚冷带下。
加减：None
功效：返老还童。
用法用量：每服3丸或2丸，用盐汤温酒送下；若以酒化下尤佳。渐加至5丸止。
制备方法：上用冬月大萝卜作盒子1个，令厚一指以上，又深酌度盛药；或仓卒无大者，只用朱砂三分之一。硇砂、没药、朱砂细研，余药别捣细为末，同罗研匀，入萝卜盒内，先用赤石脂水调固盒子缝，外用六一泥6分，胶泥1分，纸筋固济，令厚一指，已上泥坐在砖上，日气中令5分干；用碳3斤煅，盒子上仍留一小窍子，以竹片子试劄，侯萝卜熟为度，侯盒窍子中气出及泥干萝卜熟，抽火，半炊饭许时，再添1-2斤火，专守火候，不得令萝卜焦，即恐药败，以泥稍黄熟是候；放令药气透，敲开泥，切开萝卜，取出，丹自软结而香，急丸如豌豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
