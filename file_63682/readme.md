序号：63698
名称：清热导滞汤
组成：黄连（酒炒）7分，槟榔7分，黄芩（酒炒）7分，生白芍7分，厚朴（姜汁炒）7分，枳壳（曲炒）7分，陈皮7分，青皮3分，甘草3分，连翘8分，牛蒡子（炒、研）8分，楂肉1钱，当归1钱，淡竹叶1钱，灯心50寸（引）。
出处：《麻科活人》卷三。
主治：麻已出透，身热未全退，毒气流注而成痢。
加减：原书治上证，去白芍、楂肉、甘草、厚朴，加生地黄、地骨皮、木通主之；红多者，加红花2分，地榆皮、桃仁（酒炒）各8分；闭涩甚者，与里急后重之极者，加酒炒大黄8分。
功效：None
用法用量：水煎，加犀角末3分，温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
