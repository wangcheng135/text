序号：28737
名称：十八味丁沉透膈汤
组成：白术2两，香附（炒）1两，人参1两，缩砂仁1两，丁香（炙）半两，麦芽半两，肉豆蔻（煨）半两，白豆蔻半两，木香半两，青皮半两，甘草（炙）1两半，半夏（汤泡7次）2钱半，藿香7钱半，厚朴（姜炒）7钱半，神曲（炒）2钱半，草果2钱半，沉香7钱半，陈皮7钱半（一本无丁香、白豆蔻，有白芷、槟榔各半两）。
出处：《局方》卷三（新添诸局经验秘方）。
主治：脾胃不和，中寒上气，胁肋胀满，心腹(疒丂)痛，痰逆恶心；或时呕吐，饮食减少，十膈五噎，痞塞不通，噫气吞酸，口苦失味。
加减：None
功效：None
用法用量：上(口父)咀。每4钱，水2大盏，加生姜3片，大枣1个，煎8分，去滓热服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：丁沉透膈汤（《丹溪心法》卷四）、十八味丁沉透膈散（《普济方》卷三十六）。
