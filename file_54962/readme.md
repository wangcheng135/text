序号：54977
名称：桂附汤
组成：黄柏（为引用）5分，知母5分，肉桂1钱，附子3钱。
出处：《兰室秘藏》卷中。
主治：白带腥臭，多悲不乐。阳气虚极，大寒之证，带久不止，下流白滑如涕。
加减：如少食多饱，有时似腹胀夯闷，加白芍药5分；如不思饮食，加五味子20个；如烦恼，面上如虫行，乃胃中元气极虚，加黄耆1钱5分，人参7分，炙甘草、升麻各5分。
功效：None
用法用量：上(口父)咀，都作1服。水2盏，煎至1盏，去滓，食远热服。
制备方法：None
临床应用：None
各家论述：《玉机微义》：此补阳气极虚，用黄柏等为引用，又升降阴阳药也。
用药禁忌：None
附注：方中黄柏，《卫生家宝》作“黄芩”。
