序号：4459
名称：木香丸
组成：木香1两，鳖甲（去裙襴，醋浸，炙）1两，羌活（去芦头）3分，柴胡（去苗）1两，半夏（汤浸7次，去滑）半两，桔梗（炒）3分，知母（焙）3分，陈橘皮（汤浸，去白，炒）3分，五味子半两，秦艽（去苗土）半两，天灵盖（酥炙）半两，桃仁（汤浸，去皮尖双仁，炒，研如膏）4两。
出处：《普济方》卷一四五。
主治：伤寒后夹劳，四肢无力，脾胃虚弱，饮食无味，向晚潮热，手足心烦，渐觉黄瘦。
加减：None
功效：None
用法用量：每服20丸，空心米饮送下，晚再服。
制备方法：上捣筛11味为末，与桃仁同研匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
