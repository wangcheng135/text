序号：42307
名称：青麻饮子
组成：青麻子枝（和叶锉）1斗（以中元节所有收用最佳，不然，秋间收者亦得，着土处头1尺弃之，旧方云，未沤麻亦同其效），小荷叶3片（亦是中元节收，晒干），当归半两，甘草（炮）3分，陈橘皮3分，生姜（拍破）3分。
出处：《医方类聚》卷二二八引《王岳产书》。
主治：诸患恶滞，胞衣不下。
加减：None
功效：下诸恶滞。
用法用量：每剂以好无灰酒1盏，小便1盏，煎取1盏。才产了得少时，便吃此饮子1服；若胞衣不下，服之立下；但分娩了无事，亦服此2服，保无运绝，诸患恶滞皆下也，未入所投月，宜预收贮药料。
制备方法：上锉，作饮子，分作4剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
