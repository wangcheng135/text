序号：53670
名称：家秘烂口神效散
组成：顶上人中白1两5钱（煅过），上孩儿茶4钱，洋青黛3钱（水飞），苏薄荷2钱（去梗），关黄柏1钱5分（炒），明雄黄1钱，大梅片2-3分（或5分），青果核3钱（炕，研极细末），制铜绿6分，枯白矾8分，鸡内金2钱（刷净秽，炕存性），白硼砂1钱5分
出处：《寿世新编》。
主治：口舌牙龈腐烂疼痛，走马牙疳，烂喉诸症。
加减：None
功效：None
用法用量：临用时，先用温水漱净口中涎秽，再蘸药少许搽烂处，含片刻，吐去毒涎，逾时又擦，数次即愈。
制备方法：共选道地药材，为极细末，瓷罐收贮，塞极紧。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
