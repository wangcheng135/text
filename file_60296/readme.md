序号：60311
名称：通气驱风汤
组成：天台乌药5两，桔梗（去芦）3两半，川白芷3两半，川芎3两半，甘草（炙）3两半，陈皮（去白）3两半，白术3两半，麻黄（去根节）1两半，枳壳（麸炒，去瓤）1两半，人参（去芦）半两。
出处：《得效》卷十三。
主治：男子妇人血气虚弱，虚风攻注，肌骨颤掉，肩背刺痛，手足拳挛，口角斜，半身不遂，头目旋晕，痰涎壅盛，语言謇涩，行步艰难，心忪气短；客风所凑，四肢拘急，鼻塞身重，头疼；脾胃不和，心腹刺痛，胸膈不快，少力多困，精神不爽，不思饮食，呕吐恶心，霍乱吐泻，胎前产后，气虚百病。
加减：None
功效：None
用法用量：每服3钱，紫苏、木瓜煎汤调下。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
