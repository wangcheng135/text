序号：73005
名称：犀角汤
组成：犀角（镑）半两，鳖甲（去裙襕，醋炒）半两，柴胡（去苗）半两，山芋半两，续断（锉）半两，熟干地黄（焙）半两，黄芩（去黑心）半两，紫菀（洗，焙）半两，甘草（炙，锉）半两，秦艽（去苗土）半两，防风（去叉）半两，厚朴（去粗皮，姜汁炙，锉）3分。
出处：《圣济总录》卷九十六。
主治：因有客热，积在脏腑，变为热劳。小便赤涩，四肢烦疼，心膈壅闷，面黄目赤，遍身壮热，骨节痠疼，饮食无味。
加减：None
功效：None
用法用量：每服4钱匕，水1盏半，加生姜3片，煎至8分，去滓，食前温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
