序号：79184
名称：增损禹余粮丸
组成：禹余粮2两，龙骨2两，人参2两，桂心2两，紫石英2两，乌头2两，寄生2两，杜仲2两，五味子2两，远志2两，泽泻3两，当归3两，石斛3两，苁蓉3两，干姜3两，蜀椒1两，牡蛎1两，甘草1两。
出处：《千金》卷四。
主治：女人劳损崩中，经来量多，不可禁止，积日不断，五脏空虚，失色黄瘦，崩竭暂止，少日复发，不耐动摇，小劳辄剧。
加减：None
功效：None
用法用量：每服10丸，加至20丸，空心以酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：发时服汤，未宜与此丸，若是疾久，可长与此方。
