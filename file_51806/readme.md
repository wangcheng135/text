序号：51821
名称：钟乳散
组成：成炼钟乳粉3两，上党人参3分，石斛3分，干姜3分。
出处：《千金》卷二十七。
主治：虚羸不足。六十以上人，瘦弱不能食者。
加减：None
功效：None
用法用量：平旦空腹温淳酒服1帖，日午后服1帖，黄昏后服1帖。3日后准此服之。凡服此药法，皆3日1剂。3日内只食1升半饭、1升肉，肉及饭惟烂，不得服葱、豉。3日服药既尽，3日内须作羹食补之，任意所便，仍不用葱豉及硬食也。3日服讫，还须准式服药如前，尽此1斤乳讫，其气力当自知耳，不能具述。一得此法，后服10-20斤，任意方便可知也。
制备方法：上为末，合和相得，均分作9帖。
临床应用：None
各家论述：None
用药禁忌：None
附注：问曰：何故三日少食，勿得饱也？答曰：三夜乳在腹中，熏补脏腑，若此饱食即推药出腹，所以不得饱食也。何故不得生食？由食生故即损伤药力，药力既损，脂肪亦伤，所以不得食生食也。何故不得食葱、豉？葱、豉杀药，故不得食也。本方方名，《外台》引作“炼钟乳散”。
