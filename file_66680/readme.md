序号：66696
名称：鹿茸丸
组成：鹿茸（酒浸，炙令黄）1两1分，附子（炮裂，去皮脐）1两1分，龙骨（碎，研）1两1分，黄耆（炙，锉）1两1分，桔梗（锉，炒）1两1分，生干地黄1两1分，牛膝（去苗，酒浸，焙）1两1分，芍药1两1分，人参1两1分，白茯苓（去黑皮）1两1分，枳壳（去瓤，麸炒）1两半，当归（切，焙）1两半，猬皮（炙焦）1两半，芎1两半，槐子（微炒）1两半，白矾（熬令汁尽）1两半，黄连（去须）1两半，桂（去粗皮）3分，蒲黄（炒）1两。
出处：《圣济总录》卷一四一。
主治：诸痔瘘。牡痔、牝痔、脉痔、肠痔、血痔等。
加减：None
功效：None
用法用量：每服30丸，空心煎柏叶汤送下，日晚再服。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
