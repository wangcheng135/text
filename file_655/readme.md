序号：655
名称：五霞至宝丹
组成：当归尾5两，玄胡（炙）5两，红曲2两，灵脂（炙）5两，槟榔5两，硇砂8钱，山楂5两，干漆（煅）5两，木香5两，赤芍5两，枳壳（炒）5两，吴茱萸（炙）5两，三棱（炒）5两，炮姜5两，川牛膝5两，莪术（炙）5两，黑丑（炒）2两，神曲（炒）5两，枳实（炒）7两，莱菔子（炒）10两，芒消10两，大黄200两。
出处：《北京市中药成方选集》。
主治：五积六聚，癥瘕血块，气血凝滞，胃痛腹胀。
加减：None
功效：破癥瘕，化积聚，舒肝导滞。
用法用量：每服10-20粒，1日2次，温开水送下。
制备方法：上为细末，过罗，芒消化水泛为小丸，如豌豆大，每16两用滑石3两5钱为衣闯亮。
临床应用：None
各家论述：None
用药禁忌：年老气虚勿服，孕妇忌服。
附注：None
