序号：5781
名称：加减地黄丸
组成：生地（干者）1斤，熟地（干者）1斤，石斛（去苗）4两，防风（去芦）4两，枳壳（炒）4两，牛膝（酒洗）4两，杏仁（泡，去皮尖，麸炒黄，入瓦器研去油）4两。
出处：《审视瑶函》卷六。
主治：男妇肝藏积热，肝虚目暗，膜入水轮，漏睛眵泪，眼见黑花，视物不明，混睛冷泪，翳膜遮障；及肾藏虚惫，肝受虚热；及远年近日，暴热赤眼，风毒气眼；兼治干湿脚气，消中消渴，及诸风气等疾。
加减：None
功效：补肝益肾，驱风明目。
用法用量：每服50丸，空心以豆淋酒送下；或饭饮及青盐汤亦可。
制备方法：上为细末，除杏霜另入，勿犯铁器，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌一切动风毒等物。
附注：None
