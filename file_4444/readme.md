序号：4444
名称：木香丸
组成：丁香2钱半，乳香（研）2钱半，木香2钱半，麝香（研）2钱半，安息香（研）2钱半，沉香（镑）2钱半，藿香2钱半，青橘皮（去瓤）1两，陈皮（去白）1两，槟榔（面裹，煨）1两，诃子皮1两，京三棱（略煨）1两，蓬莪术（煨）1两，肉豆蔻（面裹煨）1两，肉桂2两半（去皮），猪牙皂角（去皮弦）1两，巴豆7钱（去壳，不去油，别研入），细墨半两。
出处：《瑞竹堂方》卷一。
主治：酒食过伤，停饮。
加减：None
功效：消积、宽胸膈、快脾胃。
用法用量：每服5-7丸至10丸，食后姜汤送下。如欲推利，服15丸，利2-3行，勿多服。
制备方法：上用陈米4两，与皂角、墨、巴豆同炒令焦黄，用重纸裹，候冷，同前药碾为细末，白面糊为丸，如黄米壳大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“七香丸”。
