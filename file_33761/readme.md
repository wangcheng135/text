序号：33770
名称：没药丸
组成：当归（焙）1两，桂心半两，芍药半两，没药1分，桃仁（去皮尖，炒）1分，虻虫（去头足翅，炒）30枚，水蛭（炒）30枚。
出处：《全生指迷方》卷四。
主治：由冷热不调，或思虑动作，气乃壅遏，血蓄经络而恶血未尽，脐腹刺痛，或流注四肢，或注股内，痛如锥刺，或两股肿痛。
加减：None
功效：逐滞血。
用法用量：每服3丸，以醋汤送下。
制备方法：上为细末，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：《医略六书》：没药散瘀血，当归养新血，赤芍破血泻血滞，桃仁破瘀开血结，水蛭攻血之坚凝，虻虫攻血之疼胀。醋丸以搜之，酒煎以行之，使瘀结即化，则坚胀自消，而胞门清肃，恶露复行，何坚胀疼痛之不除哉。
用药禁忌：None
附注：桃桂当归丸（《东医宝鉴杂病篇》卷十）。
