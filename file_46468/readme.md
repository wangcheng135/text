序号：46483
名称：济泄丹
组成：木香1两，丁香1两，信（另研）1两，粉霜（另研）1钱，五灵脂1两半，肉豆蔻半两，诃子（去油，春4钱半，夏3钱半以上，秋3钱半以下，冬4钱半以上），硇砂（春3钱半，夏3钱，秋3钱，冬3钱半）。
出处：《医方类聚》卷一○二引《经验秘方》。
主治：脾虚积冷，胃脘停寒，食物多伤，不能克化，心下坚满，二胁胀痛，霍乱吐泻，中酒痰逆；小儿五疳八痢，乳食失节，蛔虫上攻，时发潮热，食癖，奶胎疟，食疟；妇人胎前产后血块。
加减：None
功效：None
用法用量：小儿1岁每服3丸，3岁每服5丸，大小加减服。大人每服1两末加江子末半钱，每服15-20丸，随汤物送下。
制备方法：上为末，好糯米粉煮饼为丸，生朱砂，麝香为衣；大人丸如芥菜子大，小儿丸如菜子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
