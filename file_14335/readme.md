名称：人参平肺散
组成：桑白皮（炒）7分，知母7分（炒），杏仁（去皮尖，炒）1钱，地骨皮1钱，紫苏1钱，橘红1钱，半夏（姜制）1钱，茯苓1钱，青皮1钱，人参1钱，五味子20粒（炒，杵），甘草（炙）5分。
出处：《痈疽验方》。
主治：心火克肺，痈疽喘急，恍惚嗜卧。
加减：None
功效：None
用法用量：水2钟，加生姜3片，煎8分，食远服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
