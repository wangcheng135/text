序号：65718
名称：菖蒲酝酒
组成：菖蒲（9节者，去须节，米泔浸，切）1斤，天门冬（去心）1斤，天雄（炮裂，去皮脐）3两，麻子仁1升（生用），茵芋（去粗茎）1两，干漆（炒烟出）3两，生干地黄（切，焙）3两，远志（去心）3两，露蜂房（微炒）1两，苦参1斤，黄耆（炙，锉）8两，独活（去芦头）5两，石斛（去根）5两，柏子仁（生用）2升，蛇蜕皮3尺（微炒），天蓼木（锉）2两。
出处：《圣济总录》卷十八。
主治：白驳，举体斑白，经年不愈。
加减：None
功效：None
用法用量：每温服4-5合，日2次夜1次。更煮菖蒲并药滓，取汤淋洗所患处。
制备方法：上为粗末，以水2石5斗煮菖蒲等，取汁1石，以酿1石2斗秫米，蒸酝如常法，用六月六日细曲于七月七日酿酒，酒成去糟，取清收干净器中，蜜覆。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
