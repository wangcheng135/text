序号：67356
名称：麻黄煎丸
组成：麻黄5斤（去根节），白花蛇肉1斤，乌蛇肉1斤，巴豆1两（去皮心，研如膏，与前3味同于釜内用水1石旋旋添水煮，水耗即添热汤，候两复时，热水及3-4斗已来，净滤去麻黄并蛇，将药水以生绢净滤过，再入锅内慢火渐熬，令稀稠得所，盛于净器中，则别入后药），硫黄1两（滴生甘草水研1复时），硇砂（浆水化去石，于铫子内熬令干）1两，干蝎1两（于瓷盒子内炒令褐色），桂心1两，附子1两（炮裂，去皮脐），防风1两（去芦头），天麻1两，沉香1两，羌活1两，天南星1两（炮裂），天雄1两（炮裂，去皮脐），羚羊角屑1两，槟榔1两，白僵蚕1两（微炒），当归1两（锉，微炒），牛黄半两（细研），犀角屑1两，白龙脑半两（细研），麝香半两（细研）。
出处：《圣惠》卷二十。
主治：瘫痪风，脚手肿满，骨节疼痛。
加减：None
功效：None
用法用量：每服1丸，以豆淋酒嚼下，不拘时候。
制备方法：上为散。入研了药，更研令匀，入麻黄煎内相和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：麻黄丸（《圣济总录》卷七）。
