序号：60220
名称：逐邪散
组成：皂子不计多少（以绢袋盛，入厕中浸3-7日，取出净洗，安地坑中，剜盖出毒1宿，焙干，捣为末），斑蝥50个（去头翅，麸慢火炒黄色，去却麸后捣为末）。
出处：《博济》卷五。
主治：瘰疬。由风邪毒气客于肌肉，随虚而停，结为疮，如李梅枣核大小，两三相连皮肉间，时发寒热，久则溃脓，连属而生，久不愈者。
加减：None
功效：None
用法用量：四更初服。至五更取下毒物，或从小便中下出，如有些小涩痛，不妨。
制备方法：上用皂子末2钱匕，斑蝥末1钱匕，水银粉2钱，生鸡卵1个，取白倾盏内，更入饭饮半盏，并药1处打匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
