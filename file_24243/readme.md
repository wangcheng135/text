序号：24251
名称：导赤片
组成：大黄270两，滑石粉72两，茯苓（去皮）72两，生地黄144两，栀子216两，木通72两。
出处：《中药制剂手册》。
主治：由于内热火盛引起的口舌生疮，咽喉肿痛，暴发火眼，两腮肿痛，大便不通，小便赤黄等症。
加减：None
功效：None
用法用量：每服4片，每日2次，以温开水送服。
制备方法：上取大黄为细末，与滑石粉混合；茯苓为粗末，用7倍量25％乙醇按渗漉法提取，并浓缩为稠液约80两；取生地黄、栀子，用煮取法提取2次，滤取药液；取木通加水14倍量煮沸3小时，滤取药液，与生地黄、栀子药液合并，浓缩为稠液约300两；另取淀粉100两、糊精50两，与大黄等细末混合，分次加入茯苓、生地黄等浓缩液，搅拌均匀，分成小块，干燥后为细末，依法压片，每片重0.3g，密封。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
