序号：64547
名称：理气汤
组成：人参、茯苓、白术、甘草、川芎、当归、白芍、枳实、枳壳、木香、乌药、香附、陈皮、砂仁、桔梗。
出处：《陈素庵妇科补解》卷三。
主治：妊娠胸痞，心下胀满，胃脘作痛，饮食不进。
加减：None
功效：温中和胃，除满去胀，养血安胎。
用法用量：None
制备方法：None
临床应用：None
各家论述：胸膈痞满，由妇人素有冷气，兼以客寒犯胃，宿冷与新邪相搏。受娠之后，经血闭而养胎，体质更弱，饮食失节，痰饮停积胸中，是以痞满而足痛也。治宜先祛寒邪，以辛温、辛散、辛凉之剂温中除满，再加甘温、苦温之药固本安胎，痞满自除。是方参、苓、术、草以益元气，芎、归、白芍以补阴血，木、砂、陈、附运气散寒，桔梗祛胸膈之满，乌药顺中下二焦之气，枳壳、枳实上下分消其滞。不独无形者立散，即有形者渐除。
用药禁忌：None
附注：None
