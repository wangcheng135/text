序号：67059
名称：麻黄加知母汤
组成：麻黄4钱，桂枝尖2钱，甘草1钱，杏仁2钱（去皮，炒），知母3钱。
出处：《衷中参西》上册。
主治：伤寒无汗。
加减：None
功效：None
用法用量：先煮麻黄5-6沸，去上沫，纳诸药，煮取1茶盅，温服。复被取微似汗，不须啜粥，余如桂枝法将息。
制备方法：None
临床应用：None
各家论述：方中用麻黄之性热中空者，直走太阳之经，外达皮毛，借汗解以祛外感之寒；桂枝之辛温微甘者，偕同甘草以温肌肉，实腠理，助麻黄托寒外出；杏仁之苦降者，入胸中以降逆定喘；原方止此四味，而愚为加知母者，诚以服此汤后，间有汗出不解者，非因汗出未透，实因余热未清也。佐以知母于发表之中，兼寓清热之意，自无汗后不解之虞。此乃屡经试验，而确知其然，非敢于经方轻为加减也。
用药禁忌：None
附注：None
