序号：70561
名称：紫苏子汤
组成：紫苏子1升，前胡1两，厚朴1两，甘草1两，当归1两，半夏1升，橘皮3两，大枣20枚，生姜1斤，桂心4两。
出处：《千金》卷七。
主治：寒痰上壅，咳嗽气喘，胸膈满闷。
加减：None
功效：常服清神顺气，和五脏，行滞气，进饮食，去湿气。降泄逆气。
用法用量：上(口父)咀，以水1斗3升，煮取2升半，分5次服，日3次，夜2次。
制备方法：None
临床应用：None
各家论述：1.《医方集解》：此手太阴药也。苏子、前胡、厚朴、橘红、半夏皆能降逆上之气，兼能除痰，气行则痰行也，数药也能发表，既以疏内壅，兼以散外寒也；当归润以和血，甘草甘以缓中；下虚上盛，故又用肉桂引火归元也。_x000D_
2.《千金方衍义》：脚气患者，浊气上攻，故以苏子。橘皮、前胡、厚朴辛温降气，半夏、生姜涤除痰湿，桂心、当归温散滞血，甘草、大枣调和中气，全以降泄逆气为主，故《局方》更名苏子降气汤。后世治虚阳上攻，痰涎壅盛，肺气喘满，服之气降即安。
用药禁忌：None
附注：苏子降气汤（《局方》卷三宝庆新增方）、降气汤（《普济方》卷一八三）、苏子降气饮（《杏苑》卷三）、紫苏汤（《景岳全书》卷五十四）。
