序号：84235
名称：五邪丸
组成：芎4分，龙角（无角用齿）4分，茯苓4分，紫石英（研）4分，防风4分，厚朴（炙）4分，铁精（研）4分，甘草（炙）4分，远志6分（去心），丹参5分，大黄5分，栀子仁5分，桂心5分，细辛5分，菖蒲5分，椒（汗，去目）5分，人参5分，干姜5分，附子（炮）5分，吴茱萸5分，芥子3分，禹余粮7分（研）。
出处：《外台》卷十五引《深师方》。
主治：邪气所中，涉于脏腑，心惊恐怖，梦寤愁忧，烦燥不乐，心神错乱，邪气经入五脏，往来烦闷，悲哀啼泣，常如苦怖，吸吸短气。当发之时，恍惚喜卧，心中踊踊，忽然欲怒，颠倒手足，冷清气乏，食即呕逆。
加减：None
功效：None
用法用量：末食，服20丸，夜服10丸，枣汤送下。不知，增之。
制备方法：上药治下筛，和以蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌海藻、菘菜、生葱、生菜、猪羊肉、饧等物。
附注：None
