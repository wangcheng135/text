序号：39031
名称：泽兰丸
组成：泽兰2两，赤芍药1两，当归1两（锉，微炒），白芷1两，蒲黄2两，芎1两，细辛1两，延胡索1两，牛膝1两（去苗），天雄1两（炮裂，去皮脐），桃仁1两（汤浸，去皮尖双仁，麸炒微黄），桂心1两，川大黄半两（锉碎，微炒），生干地黄1两，续断1两，皂荚1两（去皮，涂酥炙令焦黄；去子，别捣罗为末）。
出处：《圣惠》卷六十七。
主治：诸伤损，踠损蹉跌，筋骨疼痛。
加减：None
功效：散瘀血。
用法用量：每服30丸，以温酒送下，不拘时候。
制备方法：上为末，用酒、醋各1升，先将皂荚末煎成膏，入药末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
