序号：81491
名称：麒麟竭丸
组成：麒麟竭半两，没药半两，硇砂1两（入狗胆内浸7日），干漆1两（捣碎，炒令烟出），红蓝花1两，芫花1两（醋拌炒令干），延胡索3两，白附子半两，川乌头半两（炮裂，去皮脐），当归1两（锉，微炒），砒霜半两，伏龙肝1两，虻虫1两（微炒，去翅足），水蛭1两（微炒），巴豆1分（去皮心，研，纸裹压去油）。
出处：《圣惠》卷七十一。
主治：妇人经络痞涩，腹内有瘀血，疼痛不可忍。
加减：None
功效：None
用法用量：每服5丸，热酒送下，不拘时候。
制备方法：上为末。用生铁铫子，入头醋1升，先下硇砂搅匀，然后下药末一半，用慢火熬如膏，后下余药末和丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
