序号：75152
名称：蒿芩清胆汤
组成：青蒿脑1钱半至2钱，淡竹茹3钱，仙半夏1钱半，赤茯苓3钱，青子芩1钱半至3钱，生枳壳1钱半，陈广皮1钱半，碧玉散（包）3钱。
出处：《重订通俗伤寒论》。
主治：足少阳胆与手少阳三焦湿遏热郁，三焦气机不畅，胆中相火炽，致胸闷作呕，寒热如疟。
加减：None
功效：和解胆经。
用法用量：None
制备方法：None
临床应用：None
各家论述：1.《重订通俗伤寒论》：足少阳胆与手少阳三焦合为一经，其气化一寄于胆中以化水谷，一发于三焦以行腠理。若受湿遏热郁，则三焦之气机不畅，胆中之相火乃炽，故以蒿、芩、竹茹为君，以清泄胆火；胆火炽，必犯胃而液郁为痰，故臣以枳壳、二陈，和胃化痰；然必下焦之气机通畅，斯胆中之相火清和，故又佐以碧玉、引相火下泄；使以赤芩，俾湿热下出，均从膀胱而去。此为和解胆经之良方，凡胸痞作呕，寒热如疟者，投无不效。_x000D_
2.《中医大辞典·方剂分册》：方中青蒿、黄芩为君，清少阳胆热；配伍竹茹、陈皮、半夏、枳壳为臣，清胃降逆而化痰；合用赤茯苓、碧玉散为佐使者，既可导胆热下行，又能利湿和中调药。诸药合用，使少阳胆热可清，脾胃痰湿得化，则诸症自愈。
