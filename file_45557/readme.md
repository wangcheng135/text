序号：45572
名称：柏子仁丸
组成：柏子仁1两，泽兰半两，芎半两，桂心半两，黄耆半两（锉），禹余粮1两（烧，醋淬5次），人参半两（去芦头），熟干地黄1两，五味子半两，白术半两，木香半两，厚朴3分（去粗皮，涂生姜汁炙令香熟），当归3分（锉碎微炒），续断3分，白茯苓3分，紫石英1两（细研，水飞过），附子3分（炮裂，去皮脐），白薇3分，牛膝3分（去苗），干姜3分（炮裂，锉），干漆半两（捣碎，炒令烟出），防风半两（去芦头），牡丹半两，细辛半两，赤石脂1两。
出处：《圣惠》卷七十。
主治：妇人风虚劳冷，脾胃乏弱，四肢羸困，不欲饮食。
加减：None
功效：补虚助脾，思饮食，强气力。
用法用量：每服30丸，食前温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
