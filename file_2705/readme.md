序号：2705
名称：天门冬煎
组成：天门冬2两（去心，以水1升半，煮令极烂，候水尽细研），紫菀1两（洗，去苗土），桔梗1两（去芦头），贝母1两（煨令微黄），赤茯苓1两半，木通1两（锉），桑根白皮1两（锉）（以上同捣细，罗为散），生地黄汁4合，藕汁3合，生麦门冬汁3合，酥2合（两），白蜜3合。
出处：《圣惠》卷六。
主治：肺脏壅热，久嗽，涕唾稠粘，气促不能食。
加减：None
功效：None
用法用量：上药先下地黄、麦门冬、藕汁，煎6-7沸，次下前散搅令匀，即下酥，缓火煎如饧，收于盒中。每于食后及夜间含1茶匙，细细咽津。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
