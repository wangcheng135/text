序号：38464
名称：治疟安胎饮
组成：生地5钱，天冬3钱，南沙参3钱，知母（炒）1钱5分，条芩（炒）1钱5分，防风（炒）1钱5分，白术1钱5分，橘红8分，苏梗1钱。
出处：《医方简义》卷五。
主治：妊娠患疟。
加减：受风，加独活1钱；受寒，加牛蒡子（炒）2钱；受暑，加竹叶30片、生石膏2钱；受湿，加茯苓4钱、仙半夏1钱，以苍术易白术；欲呕，加川连（姜汁炒）8分；已成坏症而欲发斑疹，去白术，加牛蒡子2钱、薄荷1钱、马勃5分、活水芦根（即苇茎）1两。
功效：None
用法用量：加生姜1片，水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
