序号：3795
名称：开障去翳散
组成：黄连1钱，黄芩1钱，川军1钱，连翘1钱，小生地1钱，胆草1钱，菊花1钱，银花1钱，薄荷1钱，木贼1钱，川羌活1钱，蝉蜕1钱，赤芍1钱，防风1钱，荆芥1钱，甘草1钱，黄柏1钱。
出处：《春脚集》卷一。
主治：翳障。目珠云障，赤肿作痛，畏日羞明，迎风流泪，各种翳膜，视物不清，暴发火眼，沙眼。
加减：None
功效：磨云退翳。
用法用量：用凉水骨簪点少许。
制备方法：上药水煎浓汤，去净滓土，澄清，放碗中，拣上好羊脑炉甘石1两煅红，淬入药汤内，连煅、淬3次，即将甘石浸在药汤内，再将碗口用纸封好，勿令落尘。俟过数日后极干时，再加入：铜绿2分，胆矾2分，朱砂3分(水飞)，雄黄3分(水飞)，硼砂5分，冰片8分，麝香3分，共研极细如尘，收瓷瓶内，封固口。
临床应用：None
各家论述：None
用药禁忌：真有翳障，方可点之。
附注：None
