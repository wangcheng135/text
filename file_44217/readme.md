序号：44232
名称：三增茴香丸
组成：第一料：茴香（舶上者，用海盐半两同炒焦黄，和盐称）1两，川楝子（炮，去核）1两，沙参（洗，锉）1两，木香（洗）1两，第二料：加荜茇1两，槟榔半两，第三料：又加白茯苓4两（紧小实者，去黑皮），黑附子半两（炮，去皮脐秤。或加作1两）。
出处：《百一》卷十五。
主治：肾与膀胱俱虚，为邪气搏结，遂成寒疝，伏留不散，脐腹撮痛，阴核偏大，肤囊壅肿，重坠滋长，有妨行步，瘙痒不止，时行黄水，浸成疮疡；或长怪肉，屡治不痊，致令肾经闭结，阴阳不通，外肾肿胀，冷硬如石，渐渐丑大者。
加减：None
功效：温导阳气，渐退寒邪，补虚消疝，暖养肾经。
用法用量：第1料：为细末，以水煮米粉稠糊为丸，如梧桐子大。每服20丸，温酒或盐汤送下，空心食前，每日3次。小病此1料可安。才尽，便可服第2料。第2料：入前件药，共6味，重5两半，细末，依前法糊丸，汤使丸数服之。若病大未愈，便服第3料。第3料：通前件药共8味，重10两，并依前法糊丸，汤使丸数服之；加至30丸。新久大病，不过此3料可愈。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
