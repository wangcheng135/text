序号：29849
名称：十层膏
组成：黄芩2钱，黄柏2钱，白芷2钱，乳香（去油研）2钱，没药（去油，研）2钱，血竭（研）3钱，黄占1两，白占5钱，轻粉（研）1钱，血余2钱，象皮（炙，研）2钱，密陀僧（研）1两，珍珠（研）1钱。
出处：《疡科心得集·家用膏丹丸散方》。
主治：年久、新起臁疮。
加减：None
功效：生肌长肉。
用法用量：用麻油10两，先将柏、芩、芷3味入油煎枯，滤去滓。次下血余，煎枯，去血余；再下黄占，白占熔化，然后下乳、没、血竭，陀僧、轻粉、象皮、珍珠末，搅匀。将皮纸1张，分作6小张，以1张染膏提出，摊于台上，用手两面泥匀，再持1张，染膏如前法，摊在前1张上，共作10层。如遇臁疮，将此膏依疮大小剪下，扎于疮上，1日揭去1层，扎完疮愈。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：夹纸膏(《疡科心得集·补遗》)。方中珍珠，《青囊秘传》作“朱砂”。
