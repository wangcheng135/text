序号：70565
名称：紫苏子散
组成：紫苏子（微炒）半两，木香半两，诃黎勒皮半两，萝卜子（微炒）半两，杏仁（汤浸，去皮尖双仁，麸炒微黄）半两，人参（去芦头）半两，甘草1分（炙微赤，锉），青橘皮1分（汤浸，去白瓤，焙）。
出处：《圣惠》卷八十三。
主治：小儿咳逆上气，心胸壅闷，不欲乳食。咳逆上气，因乳哺无度，内挟风冷，伤于肺气，或呵气未定，与乳饮之，乳与气相逆，气不得下。
加减：None
功效：None
用法用量：每服1钱，以水1小盏，加生姜少许，煎至5分，去滓温服，不拘时候。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：None
附注：紫苏饮子散（《医方类聚》卷二四五引《经验良方》）、紫苏散（《普济方》卷三八七）、紫苏子汤（《医学入门》卷六）、紫苏子饮（《灵验良方汇编》卷一）。
