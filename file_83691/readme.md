序号：83709
名称：乌梅丹砂丸
组成：乌梅肉（炒）半两，丹砂（研）半两，鳖甲（去裙襕，醋浸，炙）半两，知母（焙）1分，常山（锉）1分，石膏（碎）1分，甘草（炙）1分，升麻1分，蜀漆叶1分，白薇1分，地骨皮1分，萎蕤1分，牛黄（研）半分，豉（炒）1合，麦门冬（去心，焙）半两，青蒿（干者）半两，麝香（研）半分。
出处：《圣济总录》卷三十五。
主治：劳疟，瘴疟，积年不愈。
加减：None
功效：None
用法用量：每服20丸，空心温酒送下、未发前再服。渐加至30丸。或吐出痰即安。
制备方法：上药除研者外，捣罗为末，研匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
