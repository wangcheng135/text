序号：83367
名称：乌鸡白凤丸
组成：人参（去芦）8斤，鹿角胶8斤，生白芍8斤，当归9斤，生牡蛎3斤，甘草2斤，生黄耆2斤，鳖甲（醋制）4斤，丹参8斤，香附（醋制）8斤，天冬4斤，桑螵蛸3斤，乌鸡32只（去净毛、肠子、爪尖，净重不得低于42斤）。
出处：《全国中药成药处方集》(天津方)。
主治：妇女血虚，月经不调，经期腹痛，白带淋漓，腰腿疼痛，肢体浮肿，产后身体衰弱，出虚汗发烧。
加减：None
功效：补气养血。
用法用量：每服1丸，白开水送下。
制备方法：上药用绍兴酒84斤装罐内(或不生锈的桶亦可)，将罐口封固，隔水蒸煮，至酒尽为度；再将以下鹿角霜3斤，熟地16斤，生地16斤，川芎4斤，银柴胡1斤10两，芡实(麸炒)4斤，生山药8斤，轧成粗末，再和所蒸的药料共和一起，搅匀晒干，共为细末，炼蜜为丸，3钱5分重，蜡皮或蜡纸筒封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
