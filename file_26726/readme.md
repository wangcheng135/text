序号：26734
名称：异方红丸子
组成：沉香1分，硇砂（别研）1分，使君子（去壳）1分，蓬莪术（炮，切）1分，京三棱（炮，醋浸过）1分，朱砂（别研）1分，木香1分，槟榔1枚（大者），肉豆蔻1枚（大者），母丁香5粒，巴豆20粒（肥好者，去皮心膜，不出油，研），黑牵牛1两（炒熟，取末半两入药，余者不用），荜澄茄1分。
出处：《杨氏家藏方》卷五。
主治：一切积聚，心腹疼痛；妇人血气攻注。
加减：None
功效：消酒食，破积气。
用法用量：每服3丸，食后茴香汤送下；欲微利，加至5-7丸。
制备方法：上为细末，面糊为丸，如绿豆大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
