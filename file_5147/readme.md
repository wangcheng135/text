名称：玉液金丹
组成：党参2两，川朴1两5钱，黄芩1两2钱，炙草1两2钱，琥珀8钱7分，粉丹皮4两5钱，沉香1两6钱，川贝2两6钱，川芎2两6钱，香附2两6钱，山药4两3钱，于术8钱4分，归身1两6钱，砂仁2两4钱，白芍1两6钱，广木香8钱4分，茯神6钱4分，川断6钱4分，橘红1两6钱，菟丝子3两2钱，杜仲2两6钱，枳壳1两2钱，益母草6两4钱，血竭8钱4分，山楂8两4钱，大腹皮8钱4分，云苓1两2钱，荷叶2两5钱，阿胶2两6钱，艾叶6两4钱，潼蒺藜2两2钱，麦冬2两5钱。
出处：《全国中药成药处方集》（兰州方）。
主治：经血不调，腰腿疼痛，气血双亏等症。
加减：None
功效：调经活血，益气养荣。
用法用量：每服1丸，日2次，白开水送下。
制备方法：上为细末，炼蜜为大丸，3钱重，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
