序号：35572
名称：侧子酒
组成：侧子4两（炮），生石斛8两（碎），磁石8两，独活3两，秦艽3两，甘草3两（炙），紫苏茎1握，前胡4两，防风3两，茯苓8两，黄芩3两，五味子4两，防己3两，桂心2两，丹参3两，蜀椒2两（出汗），山茱萸4两，芎2两，细辛2两，当归3两，白术4两，干姜3两，薏苡仁1升3合。
出处：《外台》卷十八引《崔氏方》。
主治：脚气不遂。
加减：None
功效：None
用法用量：每服4合，细细加至8-9合，温饮，每日2次。其中间觉热渴，得饮豉酒，豉仍须蒸晒之。
制备方法：上薄切，绢袋贮，以清酒4升，浸5日。
临床应用：None
各家论述：None
用药禁忌：忌海藻、菘菜、桃、李、雀肉、生葱、生菜及醋物等。
附注：None
