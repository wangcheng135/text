序号：23957
名称：当归活血丸
组成：当归3两，赤芍1两半（酒炒），桂心1两半，延胡1两半，秦艽1两半，丹皮1两半，乳香2两，牛膝1两半（酒炒）。
出处：《医略六书》卷三十。
主治：产后腰脚疼痛，脉弦涩滞者。
加减：None
功效：None
用法用量：为末，酒丸。白茄根3钱，煎汤送下。
制备方法：None
临床应用：None
各家论述：产后血瘀经络，挟湿热而流布下注，故腰脚红肿，疼痛不止焉。全当归养血以营运乎经脉；赤芍药破血以运行其血滞；桂心温经暖血，秦艽活血疏经；丹皮凉血散瘀，乳香活血散血；延胡索破血滞以通经，杜牛膝降瘀血以下行也。酒糊为丸，以行其经血，茄根汤下，以疏其经气。使瘀化血行，则经络通畅而湿热自化，红肿无不退，何腰脚疼痛之不痊乎？
用药禁忌：None
