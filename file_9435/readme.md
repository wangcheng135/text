序号：9435
名称：水银膏
组成：云母石、紫石英、鹅管石、花蕊石、玄精石、井泉石、石钟乳、寒水石、代赭石、滑石（或云金星石）、金牙石、银牙石各等分。
出处：《普济方》卷一○九。
主治：大风疾。
加减：None
功效：None
用法用量：去避风处搽之，日2次。7-8日药透，牙疼出臭涎1-2碗，出尽，用黄连、贯众各1两，煎汤漱之。如腹痛，用三黄丸下；若有麻痹不仁，用麻黄、木鳖子、防风、当归等分煎服，得汗即安。搽药后，有他病证变动者，随证用外方药治。
制备方法：上为细末，将轻粉、水银各7钱细研，入药同研，不见水银星后，用清油调。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
