序号：41369
名称：金泥煎
组成：金箔75片，水银1两半，远志1两（去心），菖蒲3分，钩藤3分，龙胆3分（去芦头），龙齿3分，人参3分（去芦头），赤茯苓3分，青黛1分，蚱蝉3枚（去翅足），麝香1分，虎睛1对（微炙），牛黄1分，甘草2分（炙微赤，锉），酥4两，蜜3斤。
出处：《圣惠》卷八十三。
主治：小儿心热，多惊悸。
加减：None
功效：None
用法用量：每服取2大豆许，以温水调服，1日3-4次。
制备方法：上件药，水银、金箔同研如泥，又别研麝香、虎睛、牛黄、青黛4味如粉，其余药捣筛为散，入银锅中，先以水2升，文火煎取半升，以新绵滤去滓，再入锅内，下酥、蜜及金泥，并研了药等，慢火煎，不住手以柳篦搅如稠饧，入瓷盒内盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
