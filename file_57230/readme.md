序号：57245
名称：消散片
组成：当归300g，麻黄（去根）100g，川芎100g，细辛100g，雄黄60g，羌活100g，防风100g，苍术800g，甘草100g，天南星（制）100g，蚯蚓（炙）300g，生川乌200g，荆芥油1g
出处：《上海市药品标准》。
主治：痈疽初起，发背流注，以及风湿疼痛。
加减：None
功效：温散寒凝，活血通络，解毒消肿。
用法用量：每服4片，日3次。
制备方法：上药先将当归、麻黄、川芎、细辛、雄黄、羌活并研细粉，过100目筛，和匀。再将剩余的6味水煎2次，每次2小时，药汁滤过，澄清，混和后浓缩成清膏。然后与上述混合粉760g，碳酸钙100g搅和，制成颗粒，干燥。干燥颗粒拌加荆芥油及润滑剂，压制2253片即得。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
