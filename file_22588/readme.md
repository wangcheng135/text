名称：吴府紫金丹
组成：朱砂10两（研细，使帛5寸，夹绢袋子盛之），附子1两半，甘草1两半，川乌头3两，苍术3两，草乌头3两，紫芫花3两，上为粗末，使生绢1尺5寸，缝作袋子3只，分上件3袋盛了，将上件草药3袋围朱砂袋子在内，使麻索子一处紧系袋口，于银锅内悬，胎水煮1日取出，解开，不用草药，将朱砂焙干，入乳钵内再研细，将大小水火鼎罐子1副，火前炕干，大罐子内使生姜自浆外涂数次，令干，将大片云母数片，先入在罐子底，次入银箔10片，金箔10片，在云母上，方入朱砂，又再入金箔10片、银箔10片盖砂，又入云母数片盖金银箔，次将小罐子，坐在大罐子口上，使醋拌赤石脂泥固济口缝，用铁线十字系定上下2个罐子作1处，使盐泥固济，约厚1分，阴干，以砖泥阔尺五一炉灶，候干，烧熟炭10斤，放在炉底，把药罐子虚悬坐于火上，其上下罐子内，常以汤瓶添水令满，罐子4面稍添火圈定煅之，每朱砂1两约干尽8两水为度，10两朱砂共使水5斤，水尽取出药炉放冷，将罐子敲开，朱砂目结成镜面锅子1团取出，其余砂石，并皆不用，将朱砂秤知钵两，用白夹绢袋子盛了，约每1两朱砂，使酽醋2升，用银锅悬朱砂袋子煮，候醋尽为度，将朱砂焙干，再依后段入盒子内煅。_x000D_
天南星4两，地骨皮4两，川椒4两，五倍子4两。
出处：《普济方》卷二六五。
主治：None
加减：None
功效：健神养气，除百病。
用法用量：每服3-5粒，空心浓煎枣汤送下。
制备方法：上为粗末，用蜜拌匀，以面径6寸，深6寸沙盒1个，先将蜜拌者草药铺1重在盒底，厚1寸半许，上铺橘叶1重，盖药令遍，却将朱砂劈作数块，滚蜜，使昆仑纸裹排在橘叶上，再用橘叶盖砂令遍，却将蜜拌者草药实满盒子，盖定，以醋拌赤石脂泥固济口缝，使铁线于十字紧系定，用盐泥纸，上用地黄龙粪固济盒子，约厚2分，阴干，放平地上，用醋炭5升作一（土冢？？），盖朱砂盒子，名曰老君（土冢？？），使阴阳炭30斤煅，约余剩烬2-3斤以来，去炭，用生土一担遍盖一宿，开（土冢？？）将盒子内草药不用，将朱砂吹净纸灰，入乳钵细研，米糊为丸，每朱砂1两，分作60-70丸。
