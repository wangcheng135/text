名称：阳起石丸
组成：阳起石1两（酒煮半日，细研，水飞过），乌犀角屑3分，防风3分（去芦头），羚羊角屑3分，石决明1两（捣，细研，水飞过），麦门冬1两（去心，焙），虎睛1对（酒浸1宿，微炙），真珠末半两，甘菊花半两，川升麻半两，空青（细研）半两，葳蕤半两，细辛半两，车前子半两，蔓荆子半两，人参（去芦头）半两，芎半两，赤芍药半两，青葙子半两，槐子（微炒）半两，蕤仁（汤浸，去赤皮）半两，黄芩半两，前胡（去芦头）半两，决明子半两，汉防已半两，黄连（去须）半两，茺蔚子半两，枳实（麸炒微黄）半两，川大黄（锉碎，微炒）半两，甘草（炙微赤，锉）半两。
出处：《圣惠》卷三十三。
主治：肝肾久虚，眼目昏暗，渐成内障。
加减：None
功效：None
用法用量：每服30丸，食后以麦门冬汤送下。
制备方法：上为末，入研了药令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
