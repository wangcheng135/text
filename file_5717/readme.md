序号：5717
名称：加减茱萸汤
组成：吴茱萸（汤洗7次，炒）1两半，桔梗半两，干姜（炮）半两，炙甘草半两，麦门冬（去心）半两，半夏（汤洗7次）半两，防风半两，细辛半两，当归（酒浸炒）半两，茯苓半两，牡丹皮半两，桂心半两。
出处：《三因》卷十七。
主治：妇人脏气本虚，宿挟风冷，胸膈满痛，腹胁绞刺，呕吐恶心，饮食减少，身面虚浮，恶寒战栗，或泄痢不止，少气羸困，及因生产，脏气暴虚，邪冷内胜，宿疾转甚等。
加减：None
功效：None
用法用量：每服4钱，水1盏半，煎7分，去滓，食前热服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：加减吴茱萸汤（《局方》卷九续添诸局经验秘方）。
