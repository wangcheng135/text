序号：30901
名称：十味淡斋方
组成：川贝母（去心，生研）1两，白芷（焙）1两，防风（焙）1两，海螵蛸（浸淡，漂净，去甲）1两，当归（炒）1两，川芎（炒）1两，金银花（晒）1两，花粉（晒）1两，半夏（姜汁制，炒）1两，南星（姜汁制，炒）1两5钱。
出处：《疡科心得集，方汇》卷下。
主治：下疳广疮，误服轻粉、升药致烂喉塌鼻，遍体节骱酸楚，或腐烂不堪，他药不效者。
加减：None
功效：None
用法用量：每服5钱，每日用鲜土茯苓1斤，不见铁器，于石臼内捣碎，放于瓦罐中，用河水12饭碗，煎6碗，去滓，下药末5钱，再煎至3碗；朝、午、晚各服1碗。服此63日收功。
制备方法：各药要囫囵，在瓦盆内炒，用木槌于石臼内打成末，筛净，称准分量，分作21服。
临床应用：None
各家论述：None
用药禁忌：服药忌一切盐味，煎药忌一切金、银、铜、铁、锡器。
附注：None
