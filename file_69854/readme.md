序号：69870
名称：小活血丹
组成：安息香2两，当归2两，延胡索2两，木香2两，桃仁2两，柏子仁2两，泽兰叶4两，牡丹皮4两，干姜4两，黄耆4两，桂心4两，艾叶4两，大附子2两，虎杖2两，山茱萸2两，吴茱萸2两，杜仲2两，肉苁蓉8两，厚朴8两。
出处：《鸡峰》卷十五。
主治：血脏虚冷，面黄肌瘦，胸膈痞闷，心腹撮痛，呕逆恶心，面生黑黯，鬓发脱落，头旋目黑，经候不匀，腰腿痠疼，胁肋胀痛，不欲饮食，手足烦热，肢节拘倦。一切血气虚衰。冲任不足，下焦大寒，脐腹疼痛，月事不匀，或来多不断，或过期不来，或崩中出血，或带下不止，一切血气虚寒。
加减：None
功效：None
用法用量：每服20丸，空心、食前温酒送下；淡醋汤亦可。
制备方法：上为细末，以前安息香杵碎，好酒同研，去滓，银器内慢火熬成膏，入白面少许同煮，作面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：活血丹（《卫生宝鉴》卷十八）。
