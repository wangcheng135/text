序号：67374
名称：肺痹汤
组成：人参3钱，茯苓3钱，白术5钱，白芍5钱，苏叶2钱，半夏1钱，陈皮1钱，枳壳3分，黄连3分，肉桂3分，神曲5分。
出处：《辨证录》卷二。
主治：气虚肺痹，咳嗽不宁，心膈窒塞，吐痰不己，上气满胀，不能下通。
加减：None
功效：None
用法用量：水煎服。连用2剂而咳嗽安；再用2剂而窒塞开矣；用10剂而诸症尽愈。
制备方法：None
临床应用：None
各家论述：或谓人参助气是矣，但多用恐助邪气，何以用之咸宜乎？不知肺气之虚以成痈，非肺气之实以成痹也。人参畏实不畏虚，况又有苏叶以治风；半夏以消湿；肉桂以祛寒，则邪何能作祟哉？而且白术、茯苓以健脾开胃；白芍以平肝；黄连、肉桂以交心肾。则肺气自宁，自然下降，正不必陈皮之助矣。
用药禁忌：None
附注：None
