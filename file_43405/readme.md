序号：43420
名称：养荣汤
组成：白芍药半两，川芎半两，熟干地黄（洗，焙）半两，当归（酒浸1宿，焙干）半两，青皮（去白）半两，姜黄半两，牡丹皮半两，五加皮半两，海桐皮半两，香白芷半两，牛膝（酒浸1宿，焙干）1分，延胡索1分，没药（别研）1分，五灵脂（去砂石）1分，肉桂（去粗皮）1分。
出处：《杨氏家藏方》卷十五。
主治：妇人血海虚弱，气不升降，心悸恍惚，时多惊悸，或发虚热，经候不调，可进饮食。
加减：None
功效：None
用法用量：上（口父）咀。每服5钱，水1盏半，生姜5片，乌梅1枚，煎至1盏，去滓温服，不拘时候。

临床应用：None

用药禁忌：None
