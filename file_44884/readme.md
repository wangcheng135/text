序号：44899
名称：既济丸
组成：鹿茸（爁去毛，酥炙）2两，沉香（不见火）2两，白术（炒）2两，五味子（去枝）2两，山药2两，补骨脂（炒）2两，远志2两，白茯苓（去皮）2两，续断2两，车前子（酒浸）2两，牛膝（酒浸，去苗）2两，覆盆子2两，舶上茴香（炒）2两，熟干地黄（洗）2两，白龙骨（黑豆蒸，去豆，火煅，水飞）3两，鹿角胶（蚌粉炒成珠）3两，巴戟（去心）3两，大附子（炮，去皮脐）4两，菟丝子（淘洗，酒浸，焙，再炒）4两，仙灵脾（去刺，酒浸，切，焙，微炒）4两，肉苁蓉（酒浸，去皮土）4两，杜仲（去皮，姜制，炒去丝）4两，莲子肉（炒）4两，桑螵蛸（酒浸，炙黄）4两，山茱萸（去核）4两，麝香半两（别研）。
出处：《魏氏家藏方》卷六。
主治：心肾气虚，客热上燥，神水下泄，阴阳不和，清浊相干，下元虚惫，腰脚疼重，心神不宁，水脏滑泄，饮食不进。
加减：None
功效：None
用法用量：每服50丸，食前温酒、盐汤送下。
制备方法：上为细末，糯米饭为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
