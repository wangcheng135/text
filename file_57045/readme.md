序号：57060
名称：消毒膏
组成：玄参2钱半，藁本1钱半，牛膝1钱半，续断1钱半，羌活2钱，葛根2钱半，柴胡（去苗）1钱，木鳖子（去皮）3钱，沉香3钱半，木香2钱半，当归（洗，焙）2钱半，升麻2钱半，赤芍药半钱，丹参1钱半，何首乌2钱，牡丹皮1钱半，芝麻2钱，槐白皮2钱半，甘草2钱半，白蔹2钱半，川芎2钱半，桃仁（汤浸，去皮尖）2钱半，杏仁（汤浸，去皮尖）2钱半，白附子2钱半，木通2钱半，赤茯苓2钱半，乱发（用水濯洗净，令干）2钱半，细辛1钱半，白芷3钱，防风半两，黄耆半两，苍术（去皮）1钱半，白及4钱，上好黄丹13两，腊日澄清芝麻油1斤4两。
出处：《御药院方》卷十。
主治：一切肿毒，结硬疼痛。
加减：None
功效：None
用法用量：每用时，用软白绢摊药匀，贴于患处。
制备方法：上为末，同乱发一处，用油浸7日7夜，于净石锅或银器中以慢火煎，候白芷焦黄色放温，以白棉滤去滓，于瓷罐子内密封3日3宿夜，候取出倾于锅内，慢火轻温，再滤去滓，倾在上好瓷碗中，用慢火再熬动，次下黄腊15两，用竹蓖子不住手搅令匀，次下黄丹，再搅令匀，以慢火再熬动，抬下搅令匀，续次再上火，3日方欲膏成，于瓷盒子内密封。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
