序号：1158
名称：内补丸
组成：萆薢4两，牛膝2两，五加皮2两，白术1两，川乌头（炮，去皮脐）1两，丹参1两，枳实（麸炒，去瓤）1两。
出处：《幼幼新书》卷一引《灵苑方》。
主治：受气虚弱，及五劳七伤，脏腑积冷，(疒去)癖癥块，虚胀；或经脉不调，疳冷，赤白带下，口苦舌干，面色萎黄，黑(黑干)，心烦惊悸，头目旋晕，不喜饮食，痰涕粘涎，手足百节热疼无力，肌肉消瘦，子息断绪，服一月当妊娠，百病皆愈。
加减：None
功效：None
用法用量：每服20丸，空心温酒送下，日、午、晚各1服。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
