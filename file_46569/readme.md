序号：46584
名称：千金保胎膏
组成：当归10两，白芍5两，生地8两，甘草3两，续断6两，黄耆5两，白术（炒）6两，苁蓉（炙）5两，木香1两，黄芩10两，益母草10两。
出处：《北京市中药成方选集》。
主治：妇人气虚血亏，胎元不固，屡经小产。
加减：None
功效：补气补血，保育胎元。
用法用量：贴脐上。
制备方法：上药酌予切碎，每锅用料子48两，香油240两入锅内，煎熬至枯黑，用铁纱罗过滤去滓，再熬炼至滴水成珠。兑入章丹100两，取出放入冷水中，出火毒后，加热溶化，再兑龙骨面3两，搅匀摊贴，每张油重5钱，布光。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
