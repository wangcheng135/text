序号：34655
名称：鸡肉膏
组成：大黄3两，黄芩3两，黄连3两，肉桂3两，土茯苓。
出处：《名家方选》。
主治：梅毒疳痼疾。
加减：None
功效：None
用法用量：以水3升，煮取1升5合，去滓，纳鸡肉1具，令煮和调而以如饴为度。服1-2日，乃用嗅药方。制鸡肉法：取鸡1只（去羽骨皮筋，薄锉之），渍清酒1宿，明日用之。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：①嗅药方：水银一钱，黑铅三分（纳土器上，武火沸时纳硫黄少许，则为黑霜），肉桂五分，甘草二分。上药和调，而更和陈艾六分，中令分捻而为七住，犹灯心状，纳盏浸香油灯，于方灯檠中嗅之。每日一炷，七日嗅尽之。嗅法：方灯檠四面上下以纸塞之，令气勿泄。一面穿小孔，令患人含水当鼻而嗅之。②方中土茯苓用量原缺。
