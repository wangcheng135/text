序号：40830
名称：软金丹
组成：牛黄（别研）1分，丹砂（研，水飞）1分，雄黄（研，水飞）1分，生犀（锉末，研细）1分，天麻1分，僵蚕（去丝嘴）1分，半夏（汤洗7次，为曲）1分，蝉壳（去土，汤洗净，焙）1分，南木香1分，使君子（取肉）1分，肉豆蔻（面裹煨）1分半，香墨1分半，天南星（炮裂）1钱，白附子（炮裂）1钱，腻粉1钱，水银（3钱半，用黑铅3钱半，结沙子用）半两，白花蛇1两，乌蛇（2蛇取项后粗处肉）1两（酒浸1宿，去皮骨，晒干，不见火），螺青1两2钱，麝香1钱（研），龙脑半钱（研），附子1枚（炮裂，去皮脐，虚者用之），蜈蚣（中者）2条（生用，须赤足佳），全蝎30个，天浆虫25个（去壳），大槟榔2个，丁香1钱半，蟾酥3皂子大，金箔15片，银箔15片。
出处：《卫生总微》卷五。
主治：小儿急慢惊风，三痫瘛疭，头项动摇，目睛上视，或牵斜偏搐，背脊强直，或反折如弓，口噤牙紧，或屈指如数，或温壮连绵，或服凉药过多，内生虚风，或因伤寒变搐发渴，或因吐利生风为痫，一切诸证，乳食不进，昏冒不省，但不喘急者；又治大人卒中风病，涎潮不省。
加减：None
功效：None
用法用量：每服1丸，金银薄荷汤化下。涎实者用水银丸化下，小儿量大小与，不拘时候。
制备方法：上为末，逐件旋入，研之细匀，用不蛀皂荚1铤（刮去皮弦），以好酒半斤，浸1宿，挼柔取汁，去滓，入石脑油3钱，银器内文武火煎十数沸放冷，别炼蜜少许投内，和诸药得所。大者丸如枣大，小者丸如皂子大，别以金银箔为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
