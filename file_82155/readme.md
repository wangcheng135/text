序号：82171
名称：薏苡仁酒
组成：薏苡仁2两，牛膝2两，海桐皮1两，五加皮1两，独活1两，防风1两，杜仲（姜汁炙）1两，白术半两，枳壳1两（炒），生干地黄2两半。
出处：《活人书》卷十八。
主治：脚痹。
加减：None
功效：None
用法用量：每日服1盏或半盏，空心温服，日3-4次，常令酒气醺醺不绝。久服常皮肤下如数百条虫行，即风湿气也。
制备方法：上为粗散，入生绢袋内，无灰酒5升浸，春，秋、冬14日，夏月盛热，分作数帖，遂帖浸酒。
临床应用：None
各家论述：None
用药禁忌：None
附注：海桐酒《医方类聚》卷九十八引《施圆端效方》。本方方名，《赤水玄珠》引作“苡仁酒”；《景岳全书》引作“薏仁酒”，有熟地，无枳壳、干地黄。
