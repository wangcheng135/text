序号：34707
名称：麦门冬汤
组成：麦冬3钱（去心），人参1钱半，生地5钱，阿胶3钱（糯粉炒），条芩1钱半（酒炒），白芍1钱半（酒炒），地骨皮3钱，甘草8分，大枣3枚。
出处：《医略六书》卷二十八。
主治：怀妊六月，脉大滑疾者。
加减：脾虚，加白术。
功效：None
用法用量：水1斗，煮药取3升，纳清酒1升，并胶烊尽，煎取1升，温服。
制备方法：None
临床应用：None
各家论述：阴阳凝结，胎渐长成，宜清热补阴以培养其母气，麦门冬清心肺以滋津液，人参扶元气以固胎元，生地滋阴壮水以资冲任，阿胶补阴益血以宁胎息，条芩清热安胎，白芍敛阴和血，地骨清肌退热，生草泻火缓中，大枣以益脾元也。肝虚亦用鸡汁煮药，并佐以清酒，而母气无伤，胎元无不日安日长矣。
用药禁忌：None
附注：None
