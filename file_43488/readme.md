序号：43503
名称：前胡丸
组成：前胡（去芦头）半两，赤茯苓（去黑皮）半两，芍药半两，枳壳（去瓢，麸炒）半两，黄芩（去黑心）半两，大黄1两（生），大麻仁1两半（细研）。
出处：《圣济总录》卷三十七。
主治：疟气急黄并痢。
加减：有痃癖者，加虻虫（去翅足，炒）20枚，黄耆（薄切）、天门冬（去心，焙）各1两，鳖甲（去裙襕，醋炙）、生干地黄（焙）各2两，赤茯苓（去黑皮）、人参各半两。
功效：None
用法用量：每服10丸，午食前以米饮及浆水送下，令溏利为度；不利，加至15丸。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
