序号：49229
名称：胜湿丸
组成：苍术（盐炒）1两，白芍药1两，滑石（炒）1两，椿根皮（炒）1两，干姜（煨）2钱，地榆5钱，枳壳（炒）3钱，甘草3钱。
出处：《济阴纲目》卷三。
主治：因湿热胜而下赤白带。
加减：None
功效：None
用法用量：空心米饮送下100丸。
制备方法：上为末，粥为丸，如梧桐子大。
临床应用：None
各家论述：1.《济阴纲目》汪淇笺释：此方加苍术以燥中宫，地榆以温下部，枳壳宽气于上，滑石利湿于下，干姜从而燥湿也。_x000D_
2.《医略六书》：湿热内滞，血气俱伤，故赤白带下，淫溢不止焉。苍术燥湿强脾，以清带脉；白芍敛阴和血，以安冲任；枳壳灰破滞化气；滑石末清热渗湿；地榆涩血。以止赤带；椿皮涩脱，以固带脉；炮姜温胃守中；甘草缓中和胃也。粥以丸之，饮以下之，使滞气调而湿热化，则清气和而血气各有所归，何赤白带之淫溢不止乎？
用药禁忌：None
附注：None
