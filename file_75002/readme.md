序号：75018
名称：鲊汤丸
组成：雷丸2钱，甘遂1钱，粉霜半钱（研），滑石2钱，白墡土1钱，轻粉半钱，芫花少许（用醋炒，或醋调面裹煨），青黛3钱，水银（同铅煅，入脑、麝少许）。
出处：《普济方》卷三九二。
主治：虚中坚积，外疾气蛊，腹大膨胀，遍身浮肿。
加减：None
功效：None
用法用量：未周者随月数与之，半岁不可喂；如惊或奶疳，金钱薄荷汤送下，或葱白汤送下；3岁20丸，疳瘦50丸；满身浮肿、毛发黄坚，用净洗鱼鲊煎汤送下；疳痨腹大汤使如前；一日止可两服，不可连进。5-6岁常服只15丸。自然疏泄利去滞积，如未甚退，加1-2服，积尽而泄自疏，但吃白粥补之，不可用甘草药调理，则成相反。
制备方法：上为末，烂饭捣成丸，如菜子大。
临床应用：None
各家论述：None
用药禁忌：忌甜物一月。
附注：None
