序号：46520
名称：千金不换内消丸
组成：苍术半斤（米浸，去皮），枳壳1两半（温水浸，麸炒），青皮（水浸，去瓤）1两，三棱（醋煮，去毛）1两，蓬莪（醋煮）1两，香附（炒，去毛）1两，大茴香（炒）1两，干漆（醋炒烟尽）1两，藿香（洗去土）1两，陈皮1两，厚朴（姜制）1两2钱，砂仁（炒，去皮）1两2钱，破故纸1两2钱，猪牙皂角（去皮弦）2两，黑牵牛2两，草果1两（去皮），百草霜1两。
出处：《济阳纲目》卷四十一。
主治：积聚，气蛊，胸膈膨胀，肚腹饱满，心肋紧束。小儿饮食停滞饱满。
加减：None
功效：健体扶阳。
用法用量：每服70丸，量人禀气、饮食厚薄加减，临卧好酒、或茶清、或盐汤、白汤任下；或不拘时候，照依前丸数汤引服之。暂得1-2时间，便食饭饮酒，自觉肚腹内宽快，不分多寡服。小儿用10数丸以上增添，咬碎，用茶清、米汤送下。
制备方法：上为细末，面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：孕妇不可服。
附注：None
