序号：54484
名称：柴胡散
组成：柴胡2两（去苗），秦艽1两（去苗），犀角屑3分，知母3分，桔梗3分（去芦头），人参3分（去芦头），杏仁2两（汤浸，去皮尖双仁，麸炒微黄），鳖甲1两半（涂醋炙令黄，去裙襕），葳蕤3分，生干地黄1两，甘草半两（炙微赤，锉），赤茯苓1两，桑根白皮3分（锉），栀子仁半两，紫菀1两（去苗土）。
出处：《圣惠》卷三十一。
主治：热劳体热，心烦不食，四肢无力。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：忌猪肉、菘菜、苋菜、醋物。
附注：None
