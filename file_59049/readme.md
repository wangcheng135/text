序号：59064
名称：积气丸
组成：桂（去粗皮）2两，附子（炮裂，去皮脐）半两，丹砂（研）4两，桃仁（汤浸，去皮尖双仁，研）1两半，巴豆（去皮心膜，压出油）100枚，京三棱（煨，锉）1两，干漆（炒烟出）1两，鳖甲（去裙襕，醋炙）1两，碗砂（研）2两，大黄（生用）1两，麝香（研）1两，木香1两。
出处：《圣济总录》卷七十二
主治：积聚，心腹胀满，宿食不消，(疒丂)刺疼痛，恶心呕吐，不思饮食。
加减：None
功效：None
用法用量：每服3-5丸，食后煎木香汤送下。
制备方法：上为末，先以好醋1升，熬成膏，和前药为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
