序号：47578
名称：神仙五子丸
组成：覆盆子1两，五味子1两，枸杞子1两，蛇床子1两，菟丝子（酒浸3日）1两，干山药1两，熟地黄1两，巴戟（去心）1两，白茯苓（去皮）1两，续断1两，苁蓉（酒浸2日）1两，牛膝（酒浸3日，焙）1两，肉桂1两，槟榔1两，附子（炮）1两，木香半两，沉香半两，乳香（另研）半两，没药（另研）半两，破故纸（炒）半两，木鳖子（去壳）半两，萆薢半两，茴香1两（盐炒，去盐），枳实2两。
出处：《医方类聚》卷一五三引《经验秘方》。
主治：男子失精，肌肉陷下，形色俱脱，骨蒸虚劳，诸风变易，脾胃久虚，全不思食，四肢怠情，夜梦泄精，阴囊肿痛，湿润瘙痒。
加减：None
功效：令白发返黑，活血驻颜。常服强阴气，补元肾，益子息。
用法用量：每服30丸，空心温酒送下。服至1月，气力俱壮，皮肤滑润，冬不至冷，夏不至热。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
