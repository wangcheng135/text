序号：9372
名称：牛黄丸
组成：牛黄半两（细研），龙齿2两，虎睛1对（酒浸一宿，微炙），安息香1两（入胡桃仁捣熟），朱砂2两（细研，水飞过），犀角屑1两，铁粉2两，人参1两（去芦头），独活1两，蜣蜋（去头足翅，微炒），麝香半两（细研），茯神1两，远志1两（去心），防风1两半（去芦头），甘草1两（炙微赤，锉）。
出处：《圣惠》卷二十二。
主治：风痫病。精神不全，常有痰毒在胸膈，呕吐不出，烦闷气壅。
加减：None
功效：None
用法用量：每服30丸，以荆芥汤送下，不拘时候。
制备方法：上为末，入研了药令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
