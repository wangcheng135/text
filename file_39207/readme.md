序号：39217
名称：狗头骨丸
组成：黄狗头骨1付（烧为灰，存性，不可白），紫石英2两，赤石脂2两，禹余粮2两，代赭石（各煅，醋淬）2两，香附子（炒焦）2两，当归2两，白薇2两，卷柏2两，牛膝2两，附子（炮）2两，覆盆子2两，熟艾（醋煮）2两，牡蛎（煅）2两，熟地黄2两，海螵蛸1钱，麝香1钱。
出处：《普济方》卷三三一引《仁存方》。
主治：冲任极虚，白浊，白沃，白带，脐腹疼痛，气体怯弱，饮食减少，久无子息。
加减：None
功效：None
用法用量：每服40丸，米饮送下，空心食前服。
制备方法：上为末，糯米糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
