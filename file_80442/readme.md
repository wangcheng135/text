序号：80458
名称：藿薷汤
组成：藿香正气散加香薷、扁豆、黄连。
出处：《准绳·类方》卷一。
主治：伤寒头疼，憎寒壮热，或感湿气霍乱吐泻。常服除山岚瘴气，伏暑吐泻，脚转筋。
加减：None
功效：None
用法用量：上(口父)咀。每服3钱，加生姜3片，大枣1个，水煎，热服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：《成方切用》：藿香正气散合三味香薷饮，名藿薷汤。藿香辛温，理气和中，辟恶止呕，兼治表里为君；苏、芷、桔梗，散寒利膈，佐之以发表邪；厚朴、大腹，行水消湿；橘皮、半夏，散逆除痰，佐之以疏里滞；苍术、甘草，益脾去湿，以转正气为臣、使也，正气通畅，则邪逆自除矣。
