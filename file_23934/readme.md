序号：23942
名称：安奠二天汤
组成：人参1两（去芦），熟地1两（9蒸），白术1两（土炒），山药5钱（炒），炙草1钱，山萸5钱（蒸，去核），杜仲3钱（炒黑），枸杞2钱，扁豆5钱（炒，去皮）。
出处：《傅青主女科》卷下。
主治：妊娠脾肾亏损，带脉无力，小腹作疼，胎动不安，如有下堕之状。
加减：None
功效：补脾肾，固胞胎。
用法用量：水煎服。
制备方法：None
临床应用：先兆流产：本方加减治疗先兆流产5O例，基本方：党参、白术各24g，熟地15g，山药20g，炒杜仲、续断、扁豆各10g，菟丝子15g，旱莲草30g，炙甘草6g。腹痛较甚者，加白芍30g；血热者，加黄芩10g；出血较多者，加地榆炭15-30g，阿胶10g（烊）；大便干结者，加熟大黄6g；纳差腹胀者，去熟地，加砂仁6g；恶心呕吐者，加法半夏10g，陈皮6g。少数病例适当配合西药治疗。结果：治愈45例，无效5例。
各家论述：None
用药禁忌：None
附注：None
