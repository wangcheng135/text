序号：26688
名称：导滞通经汤
组成：陈皮1两，桑白皮1两，白术1两，木香1两，茯苓（去皮）1两。
出处：《医学发明》卷六。
主治：脾湿有余，及气不宣通，面目手足浮肿。
加减：霖雨时加泽泻半两。
功效：None
用法用量：每服半两，水2盏，煎至1盏，食前去滓温服。
制备方法：None
临床应用：None
各家论述：《卫生宝鉴》：《内经》曰，湿淫所胜，平以苦热，以苦燥之，以淡泄之。陈皮苦温，理肺气，去气滞，故以为主；桑白皮甘寒，去肺中水气，水肿胪胀，利水道，故以为佐；木香苦辛温，除肺中滞气；白术苦甘温，能除湿和中，以苦燥之；以茯苓甘平，能止渴除湿，利小便，以淡泄之，故以为使也。
用药禁忌：None
附注：导气通经汤（《杏苑》卷六）。
