序号：4176
名称：木香万安丸
组成：木香1分，拣桂1分，甘遂1分，牵牛2两，大戟半两，大黄1两，红皮1两，槟榔1两，皂角2两（要得肥好者，洗净，水3盏，煮3-2沸，取出捣碎，揉取汁，再煮成稠膏，下蜜熬2沸，便取出），半夏1两，蜜1两。
出处：《宣明论》卷四。
主治：一切风热怫郁，气血壅滞，头目昏眩，鼻塞耳鸣，筋脉拘倦，肢体焦痿，咽嗌不利，胸膈痞塞，腹胁痛闷，肠胃燥涩，淋秘不通，腰脚重痛，疝瘕急结，痃癖坚积，肠滞胃满，久不了绝，走注疼痛，暗风痫病，湿病腹胀水肿。
加减：None
功效：None
用法用量：每服10丸至15丸，生姜汤送下；小儿丸如麻子大。水肿痫病诸积，快利为度。
制备方法：上膏为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
