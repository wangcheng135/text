序号：28172
名称：苋甲二仙种子膏
组成：活甲鱼1个重2斤4两准，好黄丹2斤，红苋莱2斤4两（连根带叶，晒干、切），真麻油5斤，新鲜桃条10寸（切碎），柳条10寸（切碎），桑条10寸（切碎），榆条10寸（切碎），槐条10寸（切碎）。
出处：《良方集腋》卷上。
主治：肾冷精寒，遗精白浊，一切下部虚损艰于得子以及妇女经水不调，赤白带下。
加减：None
功效：None
用法用量：用时摊布上，贴两腰左右穴并肚脐，贴至1月即可见效。100日即可种子。
制备方法：先将油入锅内，次入活甲鱼并苋菜、桃柳等条，用文武火将甲鱼等熬焦，去滓存油，再入黄丹，熬成膏，即倾入凉水内，浸三昼夜，再熔再倾，如此五次。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
