序号：66183
名称：野狐丸
组成：野狐鼻7枚，豹鼻7枚（炙），狸头骨1枚（炙），雄黄1两，阿魏2两，鬼箭羽1两，露蜂房1两（炙），白术1两，大虫头骨1两（炙），驴毛4分（烧作灰，若用骨加炒），马毛4分（烧作灰，若用骨加炒），狗毛4分（烧作灰，若用骨加炒），驼毛4分（烧作灰，若用骨加炒），牛毛4分（烧作灰，若用骨加炒），故人脑骨1两（炙）。
出处：《圣济总录》卷一○○。
主治：五注。与鬼神狐狸精魅鬼疰交通。
加减：None
功效：None
用法用量：用水磨下1丸，分一半，别捣雄黄末为衣，以床下火烧之，衣被复之，勿令泄药烟并气；未汗出，水磨2丸，分饮一半，须臾候等时；如汗不通，约行10里，再暖余1服，温吃。服后平身坐，少时平身卧，热烘衣被，和头通身盖，卧令厚暖。服药处须在暖房中，无令风入。如汗出，以灯烛照，手足指及节间，当有毫毛生出，如青黄白色，即1服见效；如赤色，须3次服；如毛黑色，必死之证。如汗出，以青绢拭遍身，日出时，于黑漆盆内洗，当有虫如麸片是验，大肠取下黑水紫黯恶黄白脓涎，如诸般虫等物是效，次即服和气药。
制备方法：上为末，搅和令调匀，又以水煮松脂候烊，接取以和之时，勿以手搅，将大虫爪和搅为丸。
