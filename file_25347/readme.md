序号：25355
名称：竹沥汤
组成：竹沥（汤成下）半两，甘草半两（炙，锉），秦艽半两（去苗土），细辛半两（去苗叶），黄芩半两（去黑心），白术半两（炒），桂半两（去粗皮），防已半两，干姜（炮）半两，麻黄（去根节，煎掠去沫，焙）3两，葛根1两，防风（去叉）1两，附子（炮裂，去皮脐）半枚，升麻3分。
出处：《圣济总录》卷八。
主治：风腰脚痹弱，或脚胫转筋，或皮肉胀起如肿，而按之不陷，心中烦懊，不欲饮食，或夙患风气者。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，煎至8分，去滓，入竹沥1合，更煎1沸，空心温服，日晚各1次。
制备方法：上药除竹沥外，锉如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
