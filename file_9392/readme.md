序号：9392
名称：水银丸
组成：水银1两（煮青州枣肉20颗，同研水银星尽），天南星半两（炮半，生使半），牛黄1分，白僵蚕半两（生用），干蝎1分（生用），麝香1分，白附子半两（生用），铅霜半两。
出处：《圣惠》卷八十五。
主治：小儿天钓，多惊搐搦，眼忽戴上，吐逆夜啼，遍身如火，面色青黄，不食乳哺，并无情绪。
加减：None
功效：None
用法用量：1-2岁儿，每服3丸，3-4岁儿，每服5丸，用薄荷汤送下，不拘时候。
制备方法：上药除水银膏，牛黄、麝香、铅霜三味研令如粉，余四味捣罗为末，都研令匀，用水银膏为丸，如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
