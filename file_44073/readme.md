序号：44088
名称：复方松塔丸
组成：松塔200两，生杏仁50两，黄豆40两，蒲公英100两，冰片2.5两。
出处：《北京市中成药规范》。
主治：老年慢性气管炎。
加减：None
功效：定喘止咳。
用法用量：每次2钱，温开水送服，1日2次。
制备方法：将药材加工洁净。取松塔40两，黄豆40两蒲公英100两，用水洗净，加入适量清水，加热煮提三次，时间分别为3小时、2小时、1小时。合并煮提液，过滤沉淀，减压浓缩至比重1.30～1.35，温度（50℃）的稠膏。取余松塔160两，粉碎为细粉，过一百孔罗，生杏仁50两，轧成泥状，串入药粉内，过60孔罗，兑入冰片2两5钱，混匀。取原粉及稠膏按比例泛为丸，低温干燥，每斤干丸药用滑石粉3两5钱，外用香墨3钱，熬水上衣闯亮。干燥后装入洁净瓶内，加盖。每百粒干重4钱，每瓶内装1两2钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
