序号：47254
名称：省风汤
组成：半夏（生用）半两，防风（去芦）半两，甘草半两（炙），全蝎（去毒）3个，白附子（生用）半两，川乌（生用）半两，木香半两，天南星（生用）半两。
出处：《医方类聚》卷二十一引《济生续方》。
主治：中风痰涎壅塞，口眼斜，半身不遂，不省人事。
加减：None
功效：None
用法用量：上（口父）咀。每服半两，水2盏，加生姜10片，煎至8分，去滓温服，不拘时候。
制备方法：None
临床应用：None
各家论述：《医方考》：风涌其痰，干于面部，则口眼喎僻；塞于胸中，则痰涎壅盛。是方也，防风、白附、全蝎、川乌，可以活经络之风痰而正口眼；南星、半夏、甘草、木香，可以疗脑次之风痰而开壅塞。方名曰省风者，省减其风之谓也。
用药禁忌：None
附注：醒风汤（《永类钤方》卷十一）。
