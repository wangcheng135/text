序号：24751
名称：托里温经汤
组成：人参（去芦）1钱，苍术1钱，白芍药1钱半，甘草（炙）1钱半，白芷2钱，当归身2钱，麻黄（去根节）2钱，防风（去芦）3钱，葛根3钱，新升麻4钱。
出处：《卫生宝鉴》卷十三。
主治：寒覆皮毛，郁遏经络，不得伸越，热伏荣中，聚而为赤肿，痛不可忍，恶寒发热，或相引肢体疼痛；痈疽脉浮紧，按之洪缓，牙关紧急，涕唾稠粘，饮食难下。
加减：None
功效：None
用法用量：每服1两重，水3盏，先煎麻黄令沸，去沫，再下余药同煎至1盏，去滓，大温服讫，卧于暖处，以绵衣覆之，得汗而散。
制备方法：None
临床应用：None
各家论述：麻黄苦温，发之者也，故以为君；防风辛温，散之者也，升麻苦辛，葛根甘平，解肌出汗，专治阳明经中之邪，故以为巨；香白芷，当归身辛温，以和血散滞，苍术苦甘温，体轻浮，力雄壮，能泄肤腠间湿热，人参、甘草甘温，白芍药酸微寒，调中益气，使托其里，故以为佐。
用药禁忌：None
附注：None
