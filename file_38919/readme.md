序号：38929
名称：泻肺散
组成：百部2两半，五味子2两半，茯苓1两，附子1两，苁蓉1两，当归1两，石斛1两，远志1两，续断1两，细辛7分，甘草7分，防风1两半，蜀椒1两半，紫菀1两半，桂心1两半，款冬花1两半，干姜1两半，桃仁60枚，杏仁30枚。
出处：《千金》卷十七。
主治：酒客劳倦，或出当风，喜怒气舍于肺，面目黄肿，起即头眩，咳逆上气，时忽忽欲绝，心下弦急，不能饮食，或吐脓血，胸痛引背，支满欲呕。
加减：None
功效：None
用法用量：每服方寸匕，以酒调服，1日3次，稍加至2匕。
制备方法：上药治下筛。
临床应用：None
各家论述：《千金方衍义》：泻肺而用紫菀、款冬、茯苓，专为酒客凑肺而设；防风、细辛、杏仁，专为风客于肺而设；五味、百部、石斛，专为木盛生风而设；当归、续断、桃仁，专为劳倦伤血，胸痛引背而设；甘草、干姜、川椒，专为饮食欲吐，面目黄肿而设；桂、附、远志、苁蓉，专为起即头眩，忽忽欲绝，上盛下虚而设。苟非固脾肾之根本，安能任肺气之疏泄欤。
用药禁忌：None
附注：None
