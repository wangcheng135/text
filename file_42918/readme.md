序号：42933
名称：保胎资生丸
组成：人参（人乳浸，饭上蒸，烘干）3两，白术3两，白茯苓（细末，水澄蒸，晒干，加人乳再蒸，晒干）1两半，广陈皮（去白，略蒸）2两，山楂肉（蒸）2两，甘草（去皮，蜜炙）5钱，怀山药（切片，炒）1两5钱，川黄连（如法炒7次）3钱，薏苡仁（炒3次）1两半，白扁豆（炒）1两半，白豆蔻仁（不可见火）3钱5分，藿香叶（不见火）5钱，莲肉（去心，炒）1两5钱，泽泻（切片，炒）3钱半，桔梗（米泔浸，去芦，蒸）5钱，芡实粉（炒黄）1两5钱，麦芽（炒，研磨取净面）1两。
出处：《广笔记》卷二。
主治：脾胃气虚，湿热蕴结，以及小儿疳积腹胀，面黄肌瘦，久泄久痢等一切脾胃不足之症。
加减：None
功效：妇人男子，调中养胃，饥能使饱，饱不使饥。调和脾胃，运化饮食，滋养荣卫，消除百病，可杜霍乱等患。
用法用量：用白汤或清米汤、橘皮汤、炒砂仁汤嚼化下。
制备方法：上为细末，炼蜜为丸，如弹子大，每丸重2钱。
临床应用：None
各家论述：1.《不居集》：此方以参、术、苓、草、莲、芡、山药、扁豆、苡仁之甘平，以补脾元；陈皮、曲、麦、豆蔻、藿、桔之辛香，以调胃气；其有湿热，以黄连清之燥之。既无参苓白术散之滞，又无香砂六君之燥，能补能运，臻于至和，名之资生，诚信不诬。_x000D_
2.《成方便读》：欲资生者，必先助其脾胃，故以四君子补益脾胃，合之山药、莲肉、扁豆、芡实之属以协助之。但脾者喜燥而恶湿，善运而不停，故以陈皮、白蔻香燥以舒之，苓、泽、苡米淡渗以利之，楂、曲、麦芽助其消导，藿香、厚朴借以温中，桔梗以引清气上行，黄连能使湿热下降。如是则脾复其常，可以资助生气矣。
用药禁忌：忌桃、李、雀、蛤、生冷。
附注：资生丸（原书同卷）、人参资生丸（《金鉴》卷四十）。
