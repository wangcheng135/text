序号：68119
名称：黄耆肉桂柴胡酒煎汤
组成：黄耆2钱，当归梢2钱，柴胡1钱5分，黍粘子1钱（炒），连翘1钱，肉桂1钱，升麻7分，炙甘草5分，黄柏5分。
出处：《兰室秘藏》卷下。
主治：附骨痈，坚硬漫肿，不辨肉色，行步作痛，按之大痛。
加减：None
功效：None
用法用量：上（口父）咀。好糯酒1大盏半，水1大盏半，同煎至1大盏，去滓，空心温服。少时便以早饭压之，不致大热上攻中上二焦也。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：内托黄耆酒煎汤（《东垣试效方》卷三）、内托酒煎汤（《医学入门》卷八）、黄耆柴胡汤（《杏苑》卷八）、内托酒煎散（《外科大成》卷二）。本方方名，《医学正传》引作“托里黄耆汤”；《疡科选粹》引作“柴胡黍粘子汤”。
