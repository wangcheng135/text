序号：47228
名称：疮药如神膏
组成：黄芩、黄柏、黄连、当归须、川芎、白芷、玄参、僵蚕、防风、斑蝥10枚，全蝎10枚，紫草、荆芥、蛇床子、花椒、槟榔、巴豆10粒。
出处：《普济方》卷二七九。
主治：疥疮。
加减：None
功效：None
用法用量：候冷，只将油敷疮。
制备方法：上（口父）咀，入香油半碗浸药，春、秋5日，夏3冬7日，熬黑色为度；再下乌桕油1小碗，半寸断柳槐条1握，用槐柳枝条搅令匀；滤净，再下硫黄、雄黄、黄丹、轻粉，随多少再下明矾，1钱为率；先用乳香研细下，急搅匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中黄芩、黄柏、黄连、当归须、川芎、白芷、玄参、僵蚕、防风、紫草、荆芥、蛇床子、花椒、槟榔用量原缺。
