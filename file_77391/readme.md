序号：77407
名称：跌打丸
组成：西大黄8两，青皮2两，刘寄奴4两，苏木2两，桃仁4两（去皮尖），炒枳实2两，五加皮4两，川芎2两，土鳖虫4两（酒浸），降香2两，山楂肉4两，京三棱2两，怀红花3两，凌霄花2两，西当归3两，赤芍2两，延胡索3两，威灵仙2两，川牛膝3两，花槟榔2两，牡丹皮3两，制乳香1两，制香附3两，制没药1两，蓬莪术2两，自然铜1两（煅），上血竭1两。
出处：《全国中药成药处方集》（南京方）。
主治：跌打损伤，及远年劳伤之筋骨疼痛。
加减：None
功效：舒筋活血，散瘀止痛。
用法用量：每服2-3钱，用开水或黄酒温下。
制备方法：上为细末，炼蜜为丸，每钱约做20丸。
临床应用：None
各家论述：None
用药禁忌：孕妇及小儿痘疹忌服。
附注：None
