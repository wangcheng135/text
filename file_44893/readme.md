序号：44908
名称：既济丹
组成：嫩鹿茸3两（酥炙），牛膝（酒浸1宿）1两半，肉苁蓉（酒浸1宿）1两半，熟干地黄（酒浸，蒸）1两半，当归（去芦，酒浸1宿）1两半，柏子仁（别研入）1两半，枸杞子（酒浸1宿）1两半，酸枣仁（微炒，别研）1两半，沉香（别研）1两半，山药（炒）1两半，远志（用甘草半两煮，去甘草不用）1两半，茯神1两半，附子2两半（炮，去皮脐）。
出处：《百一》卷四。
主治：None
加减：None
功效：升降水火，育神益血，久服延年，令人不老。
用法用量：每服50-60丸，空心、食前温酒、盐汤送下。
制备方法：上焙干，为细末，枣肉为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
