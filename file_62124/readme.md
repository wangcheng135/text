序号：62140
名称：救苦丹
组成：麻黄（去根节，洗净晒干）4两（研极细末，温水浸，用细布取汁，余滓再捣再浸再取汁，必要洁净细腻为上），甘草（炙，去皮净，研极细末）4两（凉水浸，照前取汁），赤芍1两（研极细末，温水浸，照前取汁），升麻（微炒）1两（研极细末，凉水浸取汁），朱砂1两5钱（研极细，水飞），雄黄1两5钱（研极细，水飞），当归身2两（研极细，温酒浸取汁），人参（去芦尽，研极细末）1两（温酒浸取汁），柴胡（去芦，研极细末）1两（温水浸取汁），细辛5钱（研极细末，温水浸取汁），枳实（去心，研极细末）5钱（温水浸取汁）。
出处：《万氏家抄方》卷一引通真子方。
主治：大人、小儿感冒伤寒。
加减：春、夏，加石膏5钱（水飞）；秋、冬，加桂皮5钱（研极细末，温水浸取汁）。
功效：None
用法用量：每服1丸，雄黄5分（研极细），新汲水半盏，调雄黄连药送下。用厚被盖暖处，香烧3寸，汗出即愈。
制备方法：上药各味浸汁与研细药共合一处，阴干再研，醋糊为丸，如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
