序号：60118
名称：透冰丹
组成：川大黄1两，益智子（去皮）1两，茯苓1两，茯神（去皮木）1两，蔓荆子（去花叶）1两，威灵仙（去土）1两，天麻1两，仙灵脾（去梗）1两，吴白芷1两，山栀子（七棱，小者为上，去皮）1两，麝香1分（另研），细墨1分，川乌头4两（生用，去皮脐）。
出处：《博济》卷四。
主治：一切风毒上攻，心胸不利，口舌干涩，风虚痰壅，不思饮食；及风毒下注，腰脚疼痛，脾虚体黄，肾败骨弱瘫痪等一切风疾；小儿惊风。
加减：None
功效：疏痰利膈。
用法用量：每服2丸，薄荷汁、温酒送下；如卒中，研4丸，用皂角、白矾温水送下；瘫痪风．每日服3-5丸；常服1丸，茶、酒任下；小儿惊风，入腻粉少许，薄荷汁化半丸灌之；瘰疬，葱茶清送下1丸。
制备方法：上并生为末，入麝香、墨拌匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌动风毒物。
附注：本方方名，《幼幼新书》引作“透水丹”。
