序号：49136
名称：胜金丸
组成：水银2两，硫黄1两（以上2味同结成砂子，细研），棕榈皮1两（烧灰），干漆1两（捣碎，炒令烟出），鲤鱼鳞1两（烧灰），自然铜1两（细研），狗胆1枚（干者），麒麟竭1两，当归1两（锉，微炒），延胡索半两，水蛭1分（炒令微黄），虻虫1分（微炒令黄，去翅足），乌蛇1两（酒浸，去皮骨，炙微黄），桂心半两，乱发1两（烧灰），没药半两。
出处：《圣惠》卷七十一。
主治：妇人久积瘀血在腹内，疼痛不可忍。
加减：None
功效：None
用法用量：每服10丸，以热酒调下，不拘时候。
制备方法：上为末，都拌匀，以酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
