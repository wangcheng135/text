序号：13990
名称：交泰丸
组成：干姜（炮制）3分，巴豆霜5分，人参（去芦）1钱，肉桂（去皮）1钱，柴胡（去苗）1钱5分，小椒（炒，去汗并闭目，去子）1钱5分，白术1钱5分，厚朴（去皮，锉，炒；秋冬加7钱）3钱，酒煮苦楝3钱，白茯苓3钱，砂仁3钱，川乌头（炮，去皮脐）4钱5分，知母4钱（1半炒，1半酒洗。此1味春夏所宜，秋冬去之），吴茱萸（汤洗7次）5钱，黄连（去须）6钱（秋冬减1钱半），皂角（水洗，煨，去皮弦）6钱，紫菀（去苗）6钱。
出处：《脾胃论》卷下。
主治：怠惰嗜卧，四肢不收，沉困懒倦。
加减：None
功效：升阳气，泻阴火，调营气，进饮食，助精神，宽腹中。
用法用量：每服10丸，温水送下。虚实加减。
制备方法：上除巴豆霜另入外，同为极细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
