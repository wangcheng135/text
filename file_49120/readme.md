序号：49135
名称：胜金丸
组成：芫花半两，大戟半两，甘遂半两，牵牛半两，荆三棱半两，干漆半两，青皮半两，陈皮（去白）半两，巴豆半两，硇砂1分，胆矾1分。
出处：《博济》卷二。
主治：一切气不和及女人血气肿；脾胃虚寒，宿食不消。
加减：None
功效：化积滞，消酒食。
用法用量：消酒食及常服，茶、酒任下3-5丸；妇人心气痛，醋汤下；水气及一切气肿并痢疾，生姜酒下；一切气不和，生姜汤下。
制备方法：上除硇砂、胆矾外，余并用生姜半斤捣碎，入好醋1升浸1宿，取出姜不用，控干，焙令干，为末；醋化硇砂并胆矾，煮面糊为丸，如豌豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：《圣济总录》有蓬莪术（煨，锉）。
