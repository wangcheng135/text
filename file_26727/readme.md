序号：26735
名称：延龄育子方
组成：腽脑脐（用桑白皮1两，楮实子1两，山楂1两，麦芽1两，神曲1两，补骨脂1两，黑芝麻1合，黑豆1合，以上8味用酒水各1半煎水；外用酒洗腽肭脐，入前酒水内，浸以软为度。后用竹刀切碎，去膜，用瓦1块，荷叶衬瓦上，上用瓦1块盖之，慢火烘干，碾细为末，听用），巨胜子5两（酒洗净，分为4份：芝麻、萝卜子、糯米、白芥子各炒1份），甘枸杞子（去梗蒂）4两，生地黄（肥大沉水者，酒洗净）5两，熟地黄（肥大沉水者，酒洗净）5两，麦门冬（去心）5两，白术5两（土炒1份，麸炒1份，神曲炒1份，枳壳炒1份），白茯苓（去皮心膜，乳浸，晒干）5两，菟丝子（酒洗净，浸1昼夜，蒸，捣饼，晒干）4两，人参（去芦）5两，柏子仁（炒，去壳）5两，山药（姜汁浸，炒干）4两，山茱萸（去核）5两，肉苁蓉（去甲膜，酒浸，晒干）5两，远志（去芦，甘草灯心汤泡，去核）2两，何首乌（黑豆汁蒸1份，盐水蒸1份，米泔水浸1份，醋浸1份）8两，鹿角霜5分，川巴戟（酒洗，去心）4两，石菖蒲（去芦，微炒）2两，当归（酒洗，去芦梢）2两，五味子（去梗）2两，川牛膝（去芦梢，酒洗，晒干）4两，沙苑蒺藜（炒）5两，川黄连（去须，吴茱萸汤浸1份，木香汤泡1份，姜汁泡1份，酒浸1份，晒干）3两，酸枣仁（去壳皮，炒）2两。
出处：《墨宝斋集验方》卷上。
主治：男子肾气虚弱，逢阴而痿，未媾先遗等症。
加减：None
功效：轻身延年，润养平和，延龄育子。
用法用量：每服90丸，滚白汤送下。
制备方法：上药各为末。春加姜汁、竹沥；夏加香薷、木瓜、薏仁；秋加姜、茶、茱萸、木香；冬加紫苏、薄荷、苍术、厚朴煎汁，用蜜炼为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：延龄育子丸、温脯脐真方（《何氏济生论》卷七）。
