序号：83809
名称：乌蛇丸
组成：乌蛇（酒浸，去皮骨，炙焦）2两，干蝎（炒，去土）2两，白附子（炮）2两，天麻2两，防风（去叉）2两，麻黄（去根节，先煎，掠去沫，焙）2两，五灵脂（炒）1两，白茯苓（去黑皮）1两，人参1两，槟榔（生用）1两，肉豆蔻（去皮）5枚，牛黄（别研）1分，白僵蚕（炒）1两半，阿胶（炙燥）1两半，天南星（炮）1两半，桂（去粗皮）1两半。
出处：《圣济总录》卷十一。
主治：风瘙隐疹，遍身肿起，或赤或白，痒痛难任。
加减：None
功效：None
用法用量：每服15丸，食前温酒送下，临卧再服。
制备方法：上16味，将15味捣罗为末，入牛黄研匀，炼蜜为丸，如赤小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
