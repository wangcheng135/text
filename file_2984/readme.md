序号：2984
名称：天真丹
组成：沉香1两，巴戟（酒浸，去心）1两，茴香（盐炒香，去盐用）1两，萆薢（酒浸，炒）1两，胡芦巴（炒香）1两，破故纸（炒香）1两，杜仲（炒去丝）1两，牵牛（盐炒香黑，去盐）1两，琥珀1两，肉桂半两。
出处：《医学发明》卷七。
主治：下焦阳虚，脐腹痼冷，腿肿如斗，囊肿如升，肌肉坚硬，按之不窅。
加减：None
功效：None
用法用量：每服50丸至70-80丸，空心温酒送下。
制备方法：上为细末，用原浸药酒打面糊为丸，如梧桐子大。
临床应用：None
各家论述：《古方选注》：是方用沉香入肾，消风水之肿毒，琥珀达命门，利水道，破坚瘀；巴戟疗脚气寒湿，胡芦巴搜下焦冷气潜伏；舶茴香辟膀胱冷气，除下焦气分之湿；补骨脂暖腰膝，逐囊湿；杜仲健腰膂，除阴下湿；肉桂除下焦沉寒痼冷；萆薢味苦，疗(疒帬)痹，去下焦风湿；牵牛子性大热，除气分之湿，三焦壅结，脚浮水肿。以上诸药，辛香者居多，其苦辛无香者，或藉酒浸，或令炒香，俾阳通湿去，其肿自消，肌肉自柔，于以迎阳下返，积气全形，命曰天真。形不坏也。
用药禁忌：None
附注：本方改为散剂，名“天真散”。(见《中国医学大辞典》)。
