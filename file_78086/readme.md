序号：78102
名称：蟾酥退毒丸
组成：制香附3两，西羌活3两，当归（全）3两，川断肉3两，生远志肉2两，明腰黄1两，白明矾1两，广地龙（去净泥垢，炒松弗焦）6钱，穿山甲片（炙透）5钱，藏红花5钱，上麒麟竭5钱，鸭嘴胆矾5钱，滴乳香8钱（去油净），净没药（去油净）8钱，真轻粉（净者）2钱，上西牛黄3钱，大梅花冰片3钱，当门麝香3钱。
出处：《疡科纲要》卷下。
主治：疡患初起，不论大小各症，阴发阳发。
加减：None
功效：宣通经络，行气活血，消散退肿，解毒定痛。
用法用量：小症每服分许；大症须服1钱至1钱5分。如初起痠痛坚肿，能饮酒者，用热黄酒吞丸。不能饮者，当归、木香煎汤送服。须囫囵吞，不可嚼碎。如肿痛已甚，势欲酿脓者，亦可服，但少减之；即脓成后，四围余块尚坚者，亦可服，以消尽坚肿为度。
制备方法：上各为细末，和匀，另用真杜蟾酥2两6钱，汾酒浸化，同杵为丸，如小绿豆大，辰砂为衣。
临床应用：None
各家论述：None
用药禁忌：头面疔毒忌之。
附注：None
