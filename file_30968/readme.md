序号：30977
名称：十香返魂丹
组成：公丁香2两，木香2两，乳香2两，藿香2两，苏合香2两，降香2两，海沉香2两，安息香1两，麝香1两，香附2两，诃子肉2两，僵蚕2两，天麻2两，郁金2两，蒌仁2两，礞石2两，甘草4两，建莲心2两，檀香2两，朱砂2两，琥珀2两，京朱黄1两，冰片5钱，大赤金300张。
出处：《春脚集》卷三。
主治：痰厥中风，口眼歪斜，牙关紧闭，昏厥欲死，或诸风狂乱。
加减：None
功效：芳香开窍，益智化痰。
用法用量：如见鬼神，自言自语，或登高者，姜汁送下：中暑卒晕死者，香薷煎汤送下；七情所伤而死者，灯心煎汤化下；夜寐怔忡，神魂游荡，重复又卧，醒后不知人事者，灯心、赤金煎汤送下；孕妇怀胎七、八、九月，突然死去，此为胎晕，人参、朱砂煎汤送下；孕妇胎动，莲子心煎汤送下；如醉，赤金、姜煎汤送下；小儿急慢惊风，天吊仰视，口吐涎沫，手足抽搐，薄荷、灯心煎汤送下；男女交合脱阳，脱阴欲死者，升麻煎汤送下。
制备方法：上为细末，甘草膏兑白蜜为丸，重1钱，金衣蜡皮封固。
临床应用：None
各家论述：《慈禧光绪医方选议》：此丹从《局方》苏合香丸加化痰祛风之品而来。取诸香辛窜，辟秽醒脑，以开窍闭；礞石、瓜蒌、郁金以化痰浊；僵蚕、天麻祛风；朱砂、血珀定神。诸药合用，窍道开，风痰化，凡卒厥昏死者，多可回甦。因方中有十味芳香药物，故名十香返魂。
用药禁忌：None
附注：十香返生丹（《北京市中药成方选集》）。《北京市中药成方选集》本方用法：每服一丸，日服二次，温开水送下。
