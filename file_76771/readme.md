名称：黑龙丸
组成：附子（炮裂，去皮脐）2两，乌头（炮裂，去皮脐）4两，乌蛇（酒浸1宿，去皮骨，炙）2两，干蝎（炒）1两半，苍术（锉碎，豉炒）2两，防风（去叉）2两，厚朴（去粗皮，生姜汁炙令香）2两，麻黄（去根节）3两，赤芍药（锉）1两，白芷（锉）2两，芎（锉）2两，陈橘皮（汤浸，去白，焙）2两，天南星（生用）半两，吴茱萸（净拣，用水淘7遍，微炒）1两，白术（锉碎）1两，自然铜1斤（杵碎，用生铁铫子内，以炭火一秤，渐以2-3斤，逼药铫子，令通赤，徐添火，可半日以来，其药有微焰起，闻腥气，又似硫黄香，药乃成，放冷取出；如药有五色者，甚妙，然后安向净黄湿土，上著纸，先衬药，用盆合之，令密不得通风，1宿出火毒，乳钵内细研，以水净淘黑汁，浓者收取，次更细淘，又收浓者，3-5次淘，澄定，去清水，用新瓦盆，内将纸封之，令泣干，如黑粉，称6-7两用）。
出处：《圣济总录》卷九。
主治：中风，由气血俱虚，腠理疏弱，风邪外中，真气失守，邪正相干，故半身不随，口眼斜，手足拘挛，或生軃曳，语言謇涩，心多悸惊。
加减：None
功效：None
用法用量：如中风瘫痪，半身不随，起止不能者，每日服1丸，空心临卧豆淋酒送下，60日内必愈；或患筋骨腰脚疼痛，走注不定，坐即刺腰，卧即刺背，行即入脚，服药20日定愈，亦豆淋酒送下，须臾以葱粥1盏投之，以衣被盖汗出，然后更服1丸，愈；如或患5-7日间未得汗，亦如前法，才入口汗立出，安；若男子元脏气痛，脐下撮痛不可忍者，以槟榔1枚，酒磨一半，以生姜汁少许，同煎5-7沸，研2丸服之，须臾以小麦麸醋拌炒，热熨脐下便止；或男子、妇人患破伤风、(疒帬)麻风、暗风、偏风，并豆淋酒送下1-2丸，立效；或患痃癖气，其病发时，牵痛甚者，用槟榔1枚，中分破，一半生、一半炙令黄色，都一处碾为末，用酒1盏，葱白1握同煎，葱熟倾盏内，候酒温，先呷1-2口，将葱白和药2丸烂嚼，以煎酒咽之，须臾分泄便止；些小风疾，口1服必愈。
制备方法：将自然铜粉，入诸末相和匀，捣罗为末，炼蜜为丸，如梧桐子大，于腊月内合甚妙。
临床应用：None
各家论述：None
用药禁忌：None
附注：资寿黑龙丸（《普济方》卷一一五引《简易》）。
