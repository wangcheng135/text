序号：69447
名称：猬皮丸
组成：猪左足悬蹄（烧灰存性）半两，猬皮1枚（烧灰存性），黄牛角（角思）（烧灰存性）半两，贯众半两，槐角子（炒）半两，雷丸半两，鸡冠花半两，槐花（炒）半两，油发灰半两，黄耆（去芦）半两，香白芷半两，当归（去芦，酒浸）半两，枳壳（去瓤生用）半两，玄参半两，黄连（去须）半两，防风（去芦）半两，鳖甲（醋炙）半两，麝香（别研）半钱。
出处：《济生》卷八。
主治：五种痔瘘。
加减：None
功效：None
用法用量：每服70丸，加至100丸，空心米饮送下。
制备方法：上为细末，米糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：年高虚弱，寒湿痔疾不宜服之。
