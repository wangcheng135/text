序号：57536
名称：润经汤
组成：当归1钱，白芍、川芎、香附（醋制）、熟地（焙）8分，阿胶（蛤粉炒珠）6分，黄芩6分，蒲黄（炒）6分，侧柏叶（盐水炒）6分，白术（蜜炙）6分，砂仁4分，炙甘草3分。
出处：《竹林女科》卷一。
主治：妇人27-28岁，身体困倦，饮食少进，经水时下，淋漓不止，或成片，或流赤白黄水，面色青黄目眩，眼花，四肢痠痛，将成崩漏。
加减：如咳嗽，加五味子、杏仁（去皮尖）各6分；气急，加半夏（制）、苏叶各4分；泄泻，加肉豆（煨）、粟壳各4分；肚痛，加枳壳（麸炒）、玄胡索、干膝（炒令烟尽）各8分。
功效：None
用法用量：加生姜3片，水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
