序号：2213
名称：升麻汤
组成：升麻3两，茯神（去木）2两，人参2两，防风（去叉）2两，犀角（镑）2两，羚羊角（镑）2两，羌活（去芦头）2两，桂（去粗皮）半两。
出处：《圣济总录》卷二十。
主治：热痹，风痹血脉，烦心悸眩，肌肉热极。
加减：None
功效：None
用法用量：每服4钱匕，水1盏半，加生姜1块(拍碎)，竹沥少许，同煎取1盏，去滓温服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：《医门法律》：方中以升麻为君，除阳明肌肉之热；然热甚必乱其神识，故以人参、茯神、犀角为臣而协理之；以官桂三分为反佐；以羌活为使。如秋月寒潭碧清可爱。鄙意羌、防使药更少减其半，非故饶舌，无非欲为引掖后来之助耳。
用药禁忌：None
附注：None
