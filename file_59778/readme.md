序号：59793
名称：调经种子神验秘方
组成：白归身1钱2分半（酒洗），白芍（酒炒）5分，川芎5分，熟地1钱3分（自制），四制香附1钱半，广皮8分，丹皮（酒炒）8分，白茯苓5钱（乳蒸），吴茱萸1钱（炒），延胡索8分。
出处：《郑氏家传女科万金方》卷二。
主治：不孕。
加减：若过期3-5日紫色者，乃血虚有外寒也，外加官桂1钱，炒干姜炭2钱，熟艾2钱；若先期3-5日紫色者，血虚有热也，外加黄芩3钱。
功效：None
用法用量：照方连进4剂，加生姜3片，水2钟，煎8分，清晨空心服；滓再煎，临卧服。俟经水来时服起，1日1服，药尽经止。如未成孕，俟后经来，如法再服4剂。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
