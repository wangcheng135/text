序号：52019
名称：大竹沥汤
组成：竹沥1斗4升，独活2两，芍药2两，防风2两，茵芋2两，甘草2两，白术2两，葛根2两，细辛2两，黄芩2两，芎2两，桂心1两，防己1两，人参1两，石膏1两，麻黄1两，生姜3两，茯苓3两，乌头1枚。
出处：《千金》卷七。
主治：卒中风。口噤不能言，四肢缓纵，偏痹挛急，风经五脏，恍惚恚怒无常，手足不随。脚气痹挛，风毒所致。口噤不能语，四肢顽痹缓弱，挛急疼痛。
加减：None
功效：None
用法用量：上(口父)咀。以竹沥煮取4升，分6服，先未汗者取汗。一状相当即服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
