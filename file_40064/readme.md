序号：40074
名称：肾沥汤
组成：羊肾1具（猪肾亦得），芍药3两，麦门冬（去心）3两，干地黄3两，当归3两，干姜4两，五味子2合，人参2两，茯苓2两，甘草（炙）2两，芎2两，远志（去心）2两，黄芩1两，桂心6两，大枣20枚（擘）。
出处：《外台》卷十六引《删繁方》。
主治：肾病，骨极虚寒，面肿垢黑，腰脊痛不能久立，屈伸不利，惊悸上气，少腹里急痛引腰，腰脊四肢常苦寒冷，大小便或白。
加减：遗尿，加桑螵蛸（炙）20枚。
功效：None
用法用量：以水1斗5升，煮肾取1斗，除肾纳药，煮取4升，去滓，分为4服，日3夜1。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：忌海藻、菘菜、生葱、酢物、芜荑。
附注：None
