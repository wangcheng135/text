序号：56544
名称：大健脾糕
组成：茯苓8两，白扁豆（炒去壳）8两，山药（炒）8两，莲子（去心）8两，芡实（去壳，净炒）8两，麦芽（炒）4两，砂仁4两，广皮2两，甘草（炙）4两，米5升（半糯米、半晚米，炒至老黄色，同上诸药磨为粉）。
出处：《灵验良方汇编》卷一。
主治：泄泻。
加减：None
功效：None
用法用量：饥时用白滚汤加白糖调服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：此方既大健脾，而又味皆可口，毫_x000D_
无药气。老人、小儿脾胃虚弱者，用此作点心，功效不可胜述。若欲常服省费，则茯苓、砂仁不用亦可。磨成粉后，须停三日，使火气尽出方可服。
