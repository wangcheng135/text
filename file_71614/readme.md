序号：71630
名称：善应膏
组成：黄丹2斤（细，上等者），没药（研）1两，白蔹（生）1两，官桂1两，乳香（研）1两，木鳖子（生）1两，白芨（生）1两，当归1两，白芷1两，杏仁（生）1两，柳枝1两（如箸条长）。
出处：《卫生宝鉴》卷十三。
主治：疮肿。五发，恶疮毒。
加减：None
功效：None
用法用量：用如常法。
制备方法：上除黄丹、乳香、没药外，余药用麻油5斤，浸1宿，于炭火上铁锅内，熬至变黑色，滤去药不用，将黄丹入油内上火，用柳条如小钱粗四指长，搅令微变褐色，出火，再用柳枝搅令出烟尽，入乳香、没药，再用柳条搅令匀，候冷倾入瓷盆内，切成块，油纸裹之。
临床应用：None
各家论述：None
用药禁忌：None
附注：神效善应膏（《医方类聚》卷一九四引《经验良方》）。
