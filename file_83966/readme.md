序号：83984
名称：五子内消丸
组成：香附子2两，橘核2两，汉防己2两，花椒子2两，玄胡索2两，山楂子2两，黄柏2两，山栀子2两，防风2两，川楝子（7制）2两，茅山苍术3两2钱，小茴香1两4钱，沉香5钱，人参1两，白茯苓3两。
出处：《医林绳墨大全》卷五。
主治：疝气。
加减：None
功效：None
用法用量：每服3钱，空心淡盐汤送下。外用茅术2两(炒)，艾1两，椒3钱，小茴香3钱煎汤。将衣盖熏，俟汤稍温，洗胞内，淋得毛孔冷水冷气尽化汗而出，不可间断。如只熏洗不服药，或服药不熏，皆难见效。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
