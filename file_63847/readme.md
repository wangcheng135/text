序号：63863
名称：清暑破疟饮
组成：广藿香3钱，紫苏1钱5分，青皮1钱5分，制半夏3钱，淡豆豉3钱，生甘草8分，槟榔1钱5分，枳壳1钱5分。
出处：《慈航集》卷下。
主治：疟疾。
加减：如无汗热重，此内伏热而外受寒暑，加香薷1钱；如有汗热重，加青蒿3钱；如口渴加葛根3钱，花粉2钱；如霍乱吐泻，加赤芍5钱，车前子3钱，白扁豆3钱；如周身酸痛，加秦艽1钱5分，独活1钱5分；如脉迟凉重，加煨老姜5钱，大枣3枚。一服头痛身痛发烧全退矣。大便结者，去紫苏。豆豉，加当归8钱，制军3钱，1服滞积全下，即愈。
功效：None
用法用量：爆老姜2钱为引，水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
