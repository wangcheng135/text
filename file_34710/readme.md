序号：34720
名称：麦门冬汤
组成：麦门冬（去心，焙）1两，桑根白皮（炙，锉）1两，生干地黄1两，半夏（汤洗7遍，焙干）3分，紫苑（去苗土）3分，桔梗（炒）3分，淡竹茹3分，麻黄（去根节）3分，五味子半两，甘草（炙）半两。
出处：《圣济总录》卷二十四。
主治：伤寒后伤肺。咳唾有血，胸胁胀满，上气羸瘦。诸病后火热乘肺，咳嗽有血，胸胁胀满，上气羸瘦，五心烦热，渴而烦闷。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，加生姜1分（拍碎），大枣3枚（劈破），同煎至7分，去滓，食后温服。
制备方法：上为粗末。
临床应用：None
各家论述：《医钞类编》：麦冬甘微苦寒，清心润肺，泻热除烦，火退金清，痰嗽自止；桑皮甘辛而寒，下气行水，泻肺中火邪，火退气宁，喘满自除；生地泻丙火，清燥金，血热妄行宜凉之；麻黄肺家专药，去荣中寒邪，风中风热；半夏行水润肾，亦能散血，火炎痰升，非此不除；紫苑专治血痰，为血劳圣药；桔梗开提气血，载药上浮，入肺泻药，痰塞喘促，宜辛苦开之，竹叶甘寒，能除上焦风邪，烦热咳逆喘促；五味敛肺，除热宁嗽定喘，火热咳嗽必用之药；甘草入凉剂则泻邪热，火热甚者以此缓之也。
用药禁忌：None
附注：千金麦门冬汤（《玉机微义》卷十）、麦冬汤（《嵩屋尊生》卷八。
