序号：12797
名称：石斛丸
组成：石斛1两（去根，锉），天门冬半两（去心，焙），五味子3分，巴戟半两，牛膝1两（去苗），肉苁蓉3分（酒浸1宿，刮去皴皮，炙干），干漆半两（捣碎，微炒），冤丝子1两（酒浸3宿，焙干，别捣为末），白术3分，远志半两（去心），白茯苓3分，熟干地黄3分，覆盆子半两，薯蓣半两，补骨脂1两（微炒），人参半两（去芦头），石龙芮3分，五加皮3分，萆薢3分（锉），狗脊半两，石南半两，杜仲2分（去粗皮，炙微黄，锉），天雄3分（炮裂，去皮脐），鹿茸1两（去毛，涂酥炙微黄）。
出处：《圣惠》卷七。
主治：肾脏虚损，头昏耳鸣，目暗茫茫，心中喜忘，恍惚不定，饮食无味，心恒不乐，多有恐思，时吐酸水，面无光泽，肌体虚羸，骨萎不能行立。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下，渐加至50丸。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
