序号：48516
名称：神效猬皮散
组成：猬皮1枚（炙黄），川椒3分（去目及闭口者，微炒去汗），附子3分（炮裂，去脐皮），当归3分（锉，微炒），露蜂房3分（微炙），地榆3分（锉），木通3分（锉），苦参1两（锉），斑蝥半两（以糯米拌炒，令米黄为度，去头足翅），鲮鲤甲4枚（炙令黄），桂心半两，细辛半两，樗鸡3分（炒黄），川大黄1两（锉碎，微炒），蜈蚣1枚（微炙），雄黄1两半（细研），蛇床子1两，蛇蜕皮半两（烧灰），蜥蜴1枚（炙黄），薏苡仁3分，(艹闾)茹1两，牡丹3分，龙胆3分（去芦头），鸡骨1两（炙黄），土瓜根3分，藿芦3分，白蒺藜3分（微炒，去刺）。
出处：《圣惠》卷六十六。
主治：狼瘘，出脓水不绝，寒热，肢节烦疼。
加减：None
功效：None
用法用量：每服1钱，食前以温酒调下，以愈为度。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
