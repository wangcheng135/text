序号：44083
名称：复方地锦片
组成：地锦草166两4钱，辣蓼166两4钱，车前草54两4钱。
出处：《中药制剂手册》引上海中药制药一厂方。
主治：细菌性痢疾，肠炎。
加减：None
功效：清热解毒，利水。
用法用量：每服3-6片，温开水送下，1日3次。
制备方法：取地锦草、辣蓼各33两，车前草11两，共为细粉。取地锦草等3味下余药料，用煮提法提取二次，合并浓缩为稠膏约77两，取地锦草等3味下余药料，用煮提法提取二次，合并浓缩为稠膏约77两，取地锦草等细粉，与上项稠膏搅软，制颗粒，压片，包淡红色糖衣（胭脂红、滑石粉各适量）。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
