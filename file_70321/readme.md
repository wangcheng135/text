序号：70337
名称：煮肝散
组成：白芍药1两，芎3分，桔梗3分（去芦头），陈橘皮1两半（汤浸，去白瓤，焙），厚朴3分（去粗皮，涂生姜汁，炙令香熟），桂心3分，干姜（炮裂，锉）半两，当归（锉碎，微炒）半两，柴胡（去芦头）半两，荆芥蔦（莳）萝半两，胡椒半两，芜荑半两，藁本半两，紫菀（去苗土）半两。
出处：《圣惠》卷七十。
主治：妇人冷劳。面色萎黄，不多思食，或时下痛，四肢少力，日渐羸瘦。
加减：None
功效：None
用法用量：每服半两，用獖猪肝1具，以盐醋葱白各少许相合，如寻常煮熟，空腹任意食之，后吃暖酒1盏。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
