序号：68067
名称：黄耆汤
组成：黄耆（锉）1两半，款冬花1两半，贝母（去心，焙）1两半，麻黄（去节）1两，柴胡（去苗）1两，甘草（炙，锉）1两，桂（去粗皮）1两，麦门冬（去心，焙）1两，人参1两，生干地黄（焙）1两，桑根白皮（锉）1两，紫菀（去苗土）1两，白茯苓（去黑皮）1两，杏仁（去皮尖双仁，炒）1两。
出处：《圣济总录》卷八十七。
主治：暴急劳疾，痰嗽喘满。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，加生姜7片，同煎至8分，去滓，食后温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中黄耆、款冬花用量原缺，据《普济方》补。
