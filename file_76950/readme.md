序号：76966
名称：黑神丹
组成：天麻（去芦）1两，蔓荆子1两，川芎1两，防风（去芦头）7钱，藁本（去土）7钱，白茯苓（去皮）7钱，细辛3钱半（去叶土），川鸟头2钱半（炮，去皮脐尖），牛膝（酒浸1宿）3钱，荆芥穗3钱，甘草（锉，炒）3钱，香白芷1两3分，赤芍药1两3分，白僵蚕（炒去丝嘴）4钱，细松烟墨（烧，醋淬）4钱。
出处：《杨氏家藏方》卷二。
主治：一切风气上攻，口眼不正，肌肉瞤动；偏正头疼，头风。
加减：None
功效：None
用法用量：每服1丸，细嚼，温酒送下；荆芥汤亦得，不拘时候。
制备方法：上为细末，炼蜜为丸，每1两作10丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
