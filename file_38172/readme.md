序号：38182
名称：松脂酝酒
组成：松脂2斗5升（大山川谷者，6月采），黍米2斗5升，细曲15斤半，糯米5斗，又一方加杏仁5升〔（去皮及双仁者，随料均分，汤退去皮，捣破，研如膏）〕。
出处：《圣济总录》卷十八。
主治：大风癞，皮肤瘙痒，搔之落如麸片，眉须堕落。
加减：None
功效：安腑脏，去胃中伏热，解咽干舌涩，除风痹虚羸；久服轻身、延年不老。
用法用量：以水1石煎，松脂浮上，掠取入冷水中，却又入汤，如此40-50次，每煮5次，即须换汤，晒干，捣研作粉，得1斗1升2合半。初酝法用水4斗，浸曲，曲发黍米1斗5升，以松脂粉拌饭，一如常酝法，相次成料，每曲随常酝法，入更炊1斗黍米，拌松粉下第1料，又相次更炊糯米3斗，入松粉和酘，又相次更炊糯米2斗，同松粉拌和匀，取其松脂粉，并须和饭用尽，每1斗米松脂粉1升5合相伴，入酘后，压去滓，取清酒。每服5合，细饮，日夜可服4-5次。渐渐加至1升，温任性饮之，常令醺醺，酒势相接。
