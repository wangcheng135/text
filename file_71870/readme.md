序号：71886
名称：散瘀活血汤
组成：当归3钱，泽兰1钱，桃仁7粒，红花1钱，川芎1钱，苏木1钱5分，丹皮1钱。
出处：《青囊全集》卷上。
主治：扑跌伤损。
加减：上至头顶后，加藁本；下至会阴班顶，加皂刺、甲珠、元寸；胁，加白芥；胸，加枳壳；手骨痛，加羌活、秦艽、桔梗；足，加川牛膝；遍身筋骨，加续断、桑寄生；喉，加甘草、桔梗；气，加香附、台乌；肩背，加皂刺；心，加菖蒲、良姜；头，加羌活；腰，加杜仲、故纸；肘，加桂枝；咳逆，加姜汁、苍术、厚朴、陈皮；左胁，加柴胡；老积，加三棱、莪术、甲珠；右胁，加片姜黄。
