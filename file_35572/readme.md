序号：35582
名称：侧子散
组成：侧子1两（炮裂，去皮脐），独活3分，桂心3分，汉防己3分，附子3分（炮裂，去皮脐），芎3分，人参3分（去芦头），麻黄3分（去根节），当归3分（锉，微炒），秦艽3分（去苗），茯神3分，防风3分（去芦头），白术3分，细辛3分，甘菊花3分，甘草半两（多微赤，锉），枳壳3分（麸炒微黄，去瓤）。
出处：《圣惠》卷二十一。
主治：偏风不遂，肢节烦疼，心胸满闷，缓纵不仁。中风，手足不随，言语謇涩。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，入生姜半分，煎至5分，去滓，入竹沥半合，搅令匀，更煎1-2沸，不拘时候温服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻、鸡、猪肉。
附注：《袖珍》引《圣济总灵》有白茯苓、赤芍药，无独活、枳壳。
