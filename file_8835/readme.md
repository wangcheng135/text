序号：8835
名称：水制黄香丸
组成：舶上硫黄（黑色者曰雌，黄色者曰雄，各半，打碎，滴花倾入酽醋内，取出里面之油，取净者）1斤（用竹筒1个削去青，入硫在内，以蜡封口，投入无水粪坑中，浸1年取起，放长流水中49日），明亮松香（溶化，加烧酒煮6、7沸，倾入冷水内，抽扯去内苦黄水，再煮再抽，一连7次，拔净，细白无脚方用）3两，茅术（米泔浸，刮去粗皮，用白净者）1斤，紫檀香4两，茅香（惧不见火）4两，白胶香4两，川乌（泡，去皮）4两，川芎4两，恶实（头末）3两，草乌（炮，去皮）3两，明天麻3两。
出处：《解围元薮》卷三。
主治：诸风危困。
加减：如皮肤发热，加番木鳖5钱，以麻油煮熟。1方加地龙2两，名“黄龙丸”。
功效：None
用法用量：每服50丸，滚汤送下。
制备方法：上为末，陈皮糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌猪肉、房事。
附注：None
