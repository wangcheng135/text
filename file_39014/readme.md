名称：泽兰丸
组成：泽兰叶3分，钟乳（别研）3分，细辛（去苗叶）3分，黄耆（锉）3分，紫石英（别研）3分，大黄（锉，炒）半两，远志（去心）半两，熟干地黄（焙）半两，白芷半两，苦参半两，柏子仁（微炒）半两，蜀椒（去目及闭口者，炒出汗）半两，白术半两，芎半两，附子（炮裂，去皮脐）半两，吴茱萸（汤洗焙干，炒）半两，麦糵（炒）半两，陈曲（炮）半两，前胡（去芦头）半两，大枣（去核，炒）半两，丹参3分，枳壳（去瓤，麸炒）3分，芍药3分，桔梗（炒）3分，秦艽（去苗土）3分，当归（切，焙）3分，沙参3分，桂（去粗皮）3分，厚朴（去粗皮，生姜汁炙，锉）3分，石斛（去根）3分，麦门冬（去心，焙）3分，人参半两。
出处：《圣济总录》卷一五一。
主治：妇人月水不利，累月不快，身体烦热，骨节沉重，日渐羸瘦。

功效：None
用法用量：每服20丸，渐加至30丸，空腹以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
