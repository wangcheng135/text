序号：61801
名称：接骨仙方
组成：千里马8双（烧灰存性，即穿破旧草鞋），沉香3两，木香3两，象皮2两（瓦上焙），琥珀2两（灯芯同研），冰片2两，骨碎补1两（去毛），血竭1两，虎胫骨（酥炙）1对，乳香（去油）1两5钱，没药（去油）1两5钱。
出处：《伤科汇纂》卷八。
主治：None
加减：None
功效：接骨。
用法用量：看伤之大小，用药钱许或8分，先用好米醋1茶杯，入铜杓内熬滚，入药再熬片时，调敷患上，不可太热，不可太凉，以绵纸裹好，新绵包紧，或8-10日1换。
制备方法：上各为细末，瓷瓶收贮。
临床应用：None
各家论述：None
用药禁忌：忌公鸡、鲤、鳝，不忌牛羊肉，戒房事百日。
附注：None
