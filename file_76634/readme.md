序号：76650
名称：葛花黄连丸
组成：黄连1斤（酒蒸，浸去酒，晒干），枳壳半斤（去瓤，炒），干葛4两，葛花4两（如无，以葛代之），槐花4两，木香3两。
出处：《普济方》卷二九七引《德生堂方》。
主治：因嗜欲恣情，酒色无厌，或食煎煿之肉，大肠受热毒之深，以致痔漏便血。或痔如鼠奶，连珠翻花鸡冠之状，其形不一，或在大肠头上，粪门左右，肿痛流脓出血，发不时，面色萎黄，饮食减少，或有粪后便血红黑，又有长丝缕之血，起卧艰难者。
加减：None
功效：None
用法用量：每服50-70丸，饭水送下，酒亦可，不拘时候，1日3次。
制备方法：上为末，留浸黄连酒作面糊丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
