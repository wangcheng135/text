序号：22397
名称：阳燧锭子
组成：蟾酥5分，朱砂5分，川乌5分，草乌5分，直僵蚕1条。
出处：《外科大成》卷一。
主治：痈疽发背，一切诸毒，瘰疬，便毒，蛇引疔，痞块，及风寒湿气疼痛。
加减：None
功效：None
用法用量：用时取甜瓜子大1块，要上尖下平，先用红枣肉擦灸处，粘药于上，用油灯草火点之，灸5壮或7壮、9壮毕，即饮米醋半酒钟，随用小膏药贴之，出黄水些须，其毒即消。如风气痛者，用箸子于骨缝箸中揿之酸痛处，于墨记灸之；如腿涌膝病，放鬼眼穴灸之；再肩担成疮，于肿处各灸1壮。
制备方法：上药各为末，和匀，用石硫黄1两5钱，置碗内微火炖化，入前蟾酥等末搅匀，离火，再入当门子麝香2分、冰片1分搅匀，即倾入湿瓷盘内，速盪转成片，俟冷，取收瓷罐内。
临床应用：None
各家论述：None
用药禁忌：None
附注：阳燧锭（《金鉴》卷六十一）。
