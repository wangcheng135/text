序号：72450
名称：彭真人还寿丹
组成：大辰砂（研细，水飞过）1两，补骨脂（酒浸炒）2两，核桃仁（去皮，炒）4两（捶去油），杜仲（姜酒炒）2两，牛膝（去芦，酒洗）1两，天门冬（去心）1两，麦门冬（去心）1两，生地黄（酒洗）2两，熟地黄2两，当归（酒洗）1两，白茯苓（去皮为末，水飞晒干，人乳浸再晒）、川芎1两，远志（甘草水泡，去心）1两，石菖蒲（去毛，盐水浸）、巴戟（酒浸去梗）1两，白茯神（去皮木，同煎，茯苓一样制）、青盐1面，黄柏（盐水炒）2两，小茴香（盐水炒）1两，知母（酒炒，去毛）2两，川椒4两（微炒去子，去白隔），乳香（箬炙）1两，拣参1两，黄精（米泔水煮1沸，拣去烂的，竹刀切片晒干，却用旱莲14两，生姜汁2两，各取自然汁，并酒3味，停兑熬膏，浸黄精半日，炒苍色）4两，何首乌（捶碎，煮于黑豆水上，9蒸9晒，再用人乳浸透晒干）4两（一方加山茱萸1两，枸杞子1两，菟丝子1两，山药1两，柏子仁1两）。
出处：《回春》卷五。
主治：None
加减：None
功效：补心生血，滋肾壮阳，黑须发，润肌肤，返老还童，延年益寿，种子。
用法用量：每服70丸，空心盐汤或酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
