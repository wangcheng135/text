序号：57109
名称：消核膏
组成：制甘遂2两，红芽大戟2两，白芥子8钱，麻黄4钱，生南星1两6钱，姜半夏1两6钱，僵蚕1两6钱，藤黄1两6钱，朴消1两6钱。
出处：《徐评外科正宗》卷五。
主治：瘰疬、乳核及各种结核。
加减：None
功效：None
用法用量：None
制备方法：用真麻油1斤，先投甘遂，南星、半夏，熬枯捞出；次下僵蚕；三下大戟、麻黄；四下白芥子；五下藤黄，逐次熬枯，先后捞出；六下朴消，熬至不爆。用绢将油滤净，再下锅熬滚，徐徐投入炒透东丹，随熬随搅。下丹之多少，以膏之老嫩为度。夏宜稍老，冬宜稍嫩。膏成，趁热倾入水盆中，扯拨数10次，以去火毒，即可摊贴，宜厚勿薄。
临床应用：None
各家论述：None
用药禁忌：已溃者不可贴。
附注：None
