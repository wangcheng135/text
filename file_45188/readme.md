序号：45203
名称：枳壳散
组成：枳壳（去瓤，锉，麸炒）半两，白术半两，香附子1两（麸炒，舂去皮），槟榔3钱。
出处：《本事》卷三引庞老方。
主治：心下蓄积，痞闷，或作痛，多噫败卵气。
加减：None
功效：None
用法用量：每服2钱，米饮调下，1日3服，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：《本事方释义》：枳壳气味苦寒，入足太阴；白术气味甘温，入手足太阴；香附子气味苦平，入足厥阴；槟榔气味辛温，入足太阴。此心下积聚痞闷，脘中不爽，多噫败卵气，胀疼者，皆由中气馁弱不振，以甘温守中，而用破气消积之药攻病，则正气不伤而宿病顿去矣。
用药禁忌：None
附注：None
