序号：30354
名称：还童丹
组成：熟地黄（酒拌蒸，临时杵成膏，忌铁）5两，牛膝（去芦，酒洗）4两，黄耆（破开，蜜水拦透炙）4两，五味子（去核）2两，覆盆子4两，地骨皮（去骨）4两，白茯苓（去皮）4两，白蒺藜（另杵，净，炒）4两，桃仁（去皮尖）4两，胡桃仁（温水浸，去皮）5两，菟丝子5两（先用水洗净，次用好酒拌，浸透半湿时件成饼，焙干，为末）。
出处：《摄生众妙方》卷二。
主治：肾水不足，髭须苍白，眼目昏花，腰腿疼痛。
加减：若50以前人服，可减胡桃仁2两，恐其太滑；大便燥，不必减也。
功效：固精壮阳。
用法用量：每服50-70丸，晨、晚好酒送下，或间用盐汤送下。
制备方法：上除胡桃仁、桃仁、熟地黄捣成膏，余药同为细末，和入前药，再入炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌葱、蒜、萝卜。
附注：None
