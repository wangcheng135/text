序号：46543
名称：千金化积散
组成：木香3钱，延胡索3钱，蜈蚣3条，旱三七1钱，炙山甲1钱，干漆2钱，芥穗炭3钱，全蝎5个，麝香1分。
出处：《医学探骊集》卷六。
主治：小儿痞疾，或久泻不愈，或饮食减少，正气日亏，四体羸瘦，肚大青筋，胁下结有病块，有生在胁下腹内者，有生在胁下肌肉中者。
加减：None
功效：None
用法用量：每早、晚各服半酒杯，米泔水调服。7-8岁者服2剂，5-6岁者服1剂半，3-4岁者服1剂，其痞可以全消。周岁上下者，不易服药，将原方研粗末，入砂锅内，先用1只鸡血阴干，铺在锅底，将药入内，再用火消1两、川白蜡1两，撒在药面上，用1黑碗醋洗，扣在药上，上用黄泥封好，下用麻秸火，微微炼之，俟药烟将尽，将碗起下，取下碗上之烟脂，入瓶内备用。每服1大耳勺，用水调服。亦可用乳汁调服，其痞可以尽消。
制备方法：上为极细末，入瓶盛之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
