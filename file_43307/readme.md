序号：43322
名称：养阴清燥汤
组成：大生地1钱，大麦冬1钱，川贝母8分，粉丹皮8分，玄参1钱，薄荷叶3分，生甘草5分。
出处：《玉钥续编》。
主治：肺肾阴虚，感燥而发，咽痛白腐缠喉，及口舌白疮，口糜唇疮。
加减：发热者，不必拘泥外感之有无，只照方投之而热自退；鼻塞，音微瘖，气急者，去薄荷，加玉竹2钱，北沙参2钱；舌苔黄色而唇燥者，加真钗斛1钱；肺热咳嗽，加干桑叶3片；大便闭结，3-4日未更衣者，加叭哒杏仁（去皮尖，研末）8分，黑芝麻3钱，或火麻仁2钱；时行燥疫，易于传染者，加陈人中黄3分；阴火盛而咽干不润者，加大熟地3钱，天门冬（去心）2钱，女贞子1钱；体质虚弱，两脉浮数无力，或潮热不退者，去生地，重用大熟地，而热自除；白腐已减，尚有些微，滞于咽间不得退净者，亦须重用大熟地至5-6钱，其白即除矣；喉白既已退净，可用炒白芍8分，甜百合2钱，以固肺气，淮山药亦可加入。
功效：None
用法用量：水1钟半，煎至5-6分，温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
