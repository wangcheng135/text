序号：30224
名称：豆附丸
组成：肉豆蔻（面裹煨）2两，附子（炮，去皮脐）2两，良姜（锉，炒）2两，诃子（面裹煨）2两，干姜（炮）2两，赤石脂（火煅）2两，阳起石（火煅）2两，龙骨（生用）2两，白矾（枯）2两，白茯苓（去皮）1两，桂心（不见火）1两，细辛（洗）1两。
出处：《济生》卷四。
主治：久虚下寒，泄泻不止，肠滑不禁，日夜无度，全不进食；一切虚寒泄泻困乏。
加减：None
功效：None
用法用量：每服70丸，空心、食前米饮送下。
制备方法：上为细末，酒煮面湖为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方白矾以上药物用量原缺“各”字，据《医方大成》补。
