序号：4266
名称：木香丸
组成：木香1两，茯神（去木）1两，人参1两，白术1两，枳壳（去瓤，麸炒）1两，当归（切，炒）1两，京三棱（煨，锉）1两，知母（焙）1两，鳖甲（醋炙，去裙襴）3分，大黄（锉，炒）1分，生姜（切，焙）1分，槟榔（煨，锉）1分，桂（去粗皮）1分。
出处：《圣济总录》卷一七六。
主治：小儿癖气，手脚心热，面色黄，不下食，日渐羸瘦，往往复卧，久不治变成恶病，入夏即泻痢。
加减：None
功效：None
用法用量：100-200日儿，每服1丸；2-3岁儿，2丸至3丸，并空腹、日晚米饮送下。微利下恶物为度。
制备方法：上为末，炼蜜为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
