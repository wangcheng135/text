序号：40517
名称：虎骨散
组成：虎胫骨1两（涂酥，炙令黄），败龟1两（涂酥，炙令黄），天麻半两，白附子半两（炮裂），乌蛇肉1两半（酒浸，去皮骨，炙令微黄），附子1两（炮裂，去皮脐），海桐皮3分，防风半两（去芦头），羌活半两，芎半两，桂心3分，骨碎补3分（去毛），干姜半两（炮裂，锉），牛膝1两（去苗），萆薢半两（锉），熟干地黄3分，当归3分（锉，微炒），麝香1两（研入）。
出处：《圣惠》卷二十一。
主治：风，腰脚冷痹疼痛，行履不得。
加减：None
功效：None
用法用量：每服2钱，食前以温酒调下。
制备方法：上为细散，研令匀。
临床应用：None
各家论述：None
用药禁忌：忌生冷、鸡、猪等肉。
附注：None
