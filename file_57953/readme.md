序号：57968
名称：益气快中丸
组成：沉香3两，木香3两，大黄1两，槟榔1两，厚朴1两，干姜1两，使君子1两，干漆1两，当归1两，麦芽1两，雷丸1两，小茴1两，大茴1两，茯苓1两，芫花1两，皂角1两，巴霜1两，丁香3钱，麝香3钱，黄连3钱，人参酌用。
出处：《仙拈集》卷四。
主治：胃脘疼痛，饱闷膨胀，膈噎，痞积癥瘕，惊风吐泻，诸气积聚。
加减：None
功效：None
用法用量：每服3分，空心白滚汤送下。病甚者，早、晚2服，弱者只宜服3-5丸，壮者10服为止，亦可间日1服，小儿每服3-5丸。
制备方法：上为末，陈老炒米磨面1斤，以滴醋、烧酒各1斤半，打糊为丸，如豌豆大，阴干，收瓷器，勿走药气。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
