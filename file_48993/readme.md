序号：49008
名称：胃灵冲剂
组成：甘草25两，海螵蛸25两，白芍21两，党参5两，白术13两，元胡（醋炙）13两。
出处：《中药制剂手册》引杭州第一中药厂方。
主治：胃炎及胃与十二指肠溃疡。
加减：None
功效：健胃，镇痛，消炎，止血。
用法用量：日服3次，开水冲化，摇匀服用。
制备方法：取甘草等6味，用煮提法提取2次。第一次加水10倍量，煮沸4小时；第二次加水8倍量，煮取3小时，滤取二次药液，沉淀过滤，浓缩成稠膏约30两。另取白糊精17两，与上项稠膏搅拌均匀，分成小块，晾干或低温干燥，轧为细粉，制粒。取上项细粉，喷洒适量冷开水搅拌成软材，过14～16目筛，制成颗粒，干燥后整粒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
