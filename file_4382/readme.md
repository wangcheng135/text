序号：4382
名称：九疸秦王散
组成：栀子仁、葛根、葶苈子、栝楼、秦椒（汗）、瓜蒂、石钟乳、凝水石、牡蛎、泽泻、白术各等分。
出处：《外台》卷四引《古今录验》。
主治：胃瘅、心瘅、肾瘅、脾瘅、肺瘅、舌瘅、肉瘅、髓瘅、肝瘅。
加减：None
功效：None
用法用量：上药随病所在加2分，捣合下筛。饮服5分匕，1日3次，稍加至方寸匕。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌桃、李、雀肉等。
附注：秦王散（原书同卷）、九疸秦椒散_x000D_
（《普济方》卷一九六）。《外台》：胃瘅，食多喜饮，栀子仁主之；心瘅，烦心，心中热，葛根主之；肾瘅，其人唇干，葶苈子主之；脾瘅，溺赤出少，心惕惕若恐，栝楼主之；肺瘅，饮少小便多，秦椒（汗）、瓜蒂主之（一云膏疸）；舌瘅，渴而数便，石钟乳主之；肉瘅，其人小便白，凝水石主之；髓_x000D_
瘅，目眶深，多嗜卧，牡蛎、泽泻主之；肝瘅，胃热饮多水激肝，白术主之。上十一味，随病所在加2分。
