序号：43854
名称：厚朴汤
组成：厚朴4两（炙），吴茱萸5合，人参3两，茯苓4两，桔梗3两，生姜8两，玄参3两，芎4两，白术4两，附子3两，橘皮3两（去赤脉）。
出处：《外台》卷六引《删繁方》。
主治：上焦闭塞，干呕，呕而不出，热少冷多，好吐白沫清涎，吞酸。
加减：None
功效：None
用法用量：以水9升，煮取3升，绞去滓，分3服。
制备方法：上切。
临床应用：None
各家论述：《千金方衍义》：干呕不出，热少冷多，肝气阻逆之验，虽用厚朴、橘皮、川芎、桔梗破滞之剂，不得生姜、茱萸不能开运闭塞，不得参、苓、白术不能运行药力，玄参一味专通膈上氤氲旺气。
用药禁忌：忌猪肉、桃、李、雀肉、大醋。
附注：None
