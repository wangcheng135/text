序号：40795
名称：转败汤
组成：人参2两，生黄耆1两，熟地2两，肉桂2钱，白术4两，当归1两，金银花4两，麦冬2两，山茱萸1两，远志3钱，北五味子1钱，茯苓3钱。
出处：《辨证录》卷十三。
主治：背痈溃烂，洞见肺腑，疮口黑陷，身不能卧，口渴思饮，属阴虚而不能变阳者。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：此方补其气血，而更补其肺肾之阴。盖阴生则阳长，阴阳生长，则有根易于接续，而后以金银花解其余毒，则毒散而血生，血生而肉长，肉长而皮合，必至之势也。倘日以解毒为事，绝不去补气血之阴阳，则阴毒不能变阳，有死而已，可胜悲悼哉！
用药禁忌：None
附注：None
