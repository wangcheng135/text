序号：66757
名称：鹿茸益精丸
组成：鹿茸（去皮，酥炙微黄）3分，桑螵蛸（瓦上焙）3分，肉苁蓉3分，当归3分，巴戟（去心）3分，菟丝子（酒浸软，研）3分，杜仲（截碎，姜汁淹，炒断丝）3分，川楝子（蒸，去皮取肉，焙）3分，益智仁3分，禹余粮（煅红，醋淬，以碎为度）3分，韭子（微炒）2分，故纸（炒）2分，山茱萸2分，赤石脂2分，龙骨（别研）2分，滴乳香1分。
出处：《直指》卷十。
主治：心虚肾冷，漏精白浊。
加减：None
功效：None
用法用量：每服70丸，食前白茯苓煎汤送下。
制备方法：上为细末，酒调糯米湖为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
