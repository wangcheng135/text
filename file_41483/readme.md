序号：41498
名称：金匮丸
组成：香附（童便制，酒、醋、盐水各分制）4两，没药6钱，枣皮（焙，去油）4两，当归（童便制）4两，茯苓4两，白术（米泔水浸）4两，白薇（洗，去芦）4两，阿胶（蛤粉炒）4两，白芍4两，生地（酒浸，洗，用益智仁2两，以酒同炒，去益智仁，净）8两，人参2两，川断（酒洗，五倍子炒，净）6两，黄芩（酒浸，洗，净）4两，砂仁（炒，去衣）2两。
出处：《女科秘要》卷四。
主治：浮肿症。因经闭，败血停积五脏，流入四肢，作浮肿者。
加减：None
功效：None
用法用量：每服50丸，空心白汤送下。
制备方法：上为细末，用山药12两为末，水打丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
