序号：61283
名称：顾步汤
组成：牛膝1两，金钗石斛1两，人参3钱，黄耆1两，当归1两，金银花3两。
出处：《辨证录》卷十三。
主治：脚疽。因气血大亏，不能遍行经络，火毒恶邪，固结干骨节之际，以致脚趾头忽先发痒，已而作痛，指甲现黑色，第2日脚指俱黑，第3日连足而俱黑，黑至脚上胫骨即死；及无名肿毒。
加减：None
功效：大补气血，泄毒。
用法用量：水煎服。1剂而黑色解，2剂而疼痛止，3剂全愈。若已溃烂，多服数剂，无不愈也。
制备方法：None
临床应用：None
各家论述：此方用金银花以解毒，非用牛膝、石斛不能直达于足趾，非用人参、归、耆亦不能流通气血而散毒也。故用此方治脚疽多效。
用药禁忌：None
附注：顾步保脱汤（〈中国医学大辞典〉）。
