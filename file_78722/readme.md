序号：78738
名称：干柿丸
组成：硇砂（研）1分，砒霜1分，粉霜1分，干漆（烧烟出）1分，鳖甲（去裙襕，醋炙）1分，黄连（去须）1分，旋覆花（炒）半两，荆三棱（炮）半两，杏仁（去皮尖双仁，麸炒）1两，干姜（炮）1两，皂荚4挺（不蛀者，去皮，酥炙），巴豆49粒（去皮心膜，出油）。
出处：《圣济总录》卷七十三。
主治：积聚，气块，癖瘕。
加减：None
功效：None
用法用量：每服3丸，用干柿裹药，临卧时烂嚼，温开水送下。
制备方法：上药各为细末，先将干漆、鳖甲、荆三棱3味药末，用粟米半盏，不淘洗，以酽醋5升，同熬成粥，后入众药拌和为丸，如豌豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
