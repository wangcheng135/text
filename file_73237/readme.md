序号：73253
名称：斑龙固本丹
组成：人参（去芦）2两，干山药2两，怀生地黄2两，熟地黄（酒蒸）2两，天门冬（去心）2两，菟丝子（酒煨，捣饼，焙干）4两，山茱萸（酒蒸，去核）2两，巴戟（酒浸，去心）2两，甘枸杞子2两，麦门冬（去心）2两，杜仲（姜炒）2两，五味子2两，肉苁蓉（酒浸）2两，牛膝（酒洗，去芦）2两，远志（甘草水泡，去心）1两，覆盆子2两5钱，泽泻1两，地骨皮1两5钱，老川椒1两，白茯苓（去皮）2两，石菖蒲2两，车前子1两5钱，大附子（面裹煨，去皮脐，切片，童便浸炒）1两，木香2两，虎胫骨（酥炙）2两，柏子仁2两。
出处：《寿世保元》卷二。
主治：诸虚百损，五劳七伤，形容羸瘦，颜色衰朽，中年阳事不举，精神短少，未至五旬，发须先白，并左瘫右痪，步履艰辛，脚膝酸软，小腹疝气；妇人下元虚冷，久无孕育。
加减：None
功效：大补虚寒。
用法用量：每服100丸，空心时温酒送下。服至半月，阳事雄壮；服至1月，颜如童子，目视10里，小便清滑；服至3月，白发至黑；久服神气不衰，身轻体健。
制备方法：上为细末，用好酒化五仁斑龙胶为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
