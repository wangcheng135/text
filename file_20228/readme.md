序号：20235
名称：芎归调血饮
组成：当归、川芎、白术（去芦）、白茯苓（去皮）、熟地黄、陈皮、乌药、香附（童便炒）、干姜（炒黑）、益母草、牡丹皮、甘草。
出处：《古今医鉴》卷十二。
主治：产后一切诸病，气血虚损，脾胃怯弱，或恶露不行，或去血过多，或饮食失节，或怒气相冲，以致发热恶寒，自汗口干，心烦喘急，心腹疼痛，胁助胀满，头晕眼花，耳鸣，口噤不语，昏愦等证；小产。
加减：如恶露不行，倍益母草、丹皮，加童便、黄酒同服；如去血过多，倍芎、归、干姜；如饮食停滞，胸膈饱闷，加枳实、厚朴、山楂、砂仁；如因气恼，倍香附、乌药；如口噤昏愦不语，加荆芥；如两胁痛，加青皮、肉桂；如小腹阵痛，加玄胡索、桃仁、红花、苏木，甚者加三棱、莪术；如有汗，加黄耆；如口干苦，加麦门冬。
功效：None
用法用量：加生姜1片，大枣1个，水煎，温服。凡产后，即用童便和热酒，随意饮之，百病不生。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：芎归补血汤（《回春》卷六）、芎归补血饮（《履霜集》卷二）。
