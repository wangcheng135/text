序号：7359
名称：圣僧散
组成：青盐3钱，铜青2钱，真胆矾3钱，黄丹3钱，南硼砂3钱，硇砂2钱半（以上六味纸衬略见火），麝香1钱，片脑，轻粉1钱，白矾，蕤仁半两（净去壳，研细7次去油），炉甘石2两（净，煅7次，每一次童便内淬，7次了，然后捻极细，井花水飞过，取净末，令干，不见火，方入众药）。
出处：《医方类聚》卷七十引《经验秘方》。
主治：眼目病。
加减：None
功效：None
用法用量：每用少许，于温汤内绵滤过，仰卧就，以绵沾药汁滴于眼中，勿令漏出药汁，然后起身。
制备方法：上十二味，于内坚硬者，各研细了，然后合和一处，研极细，用三重花纸罗过，收入砂合内，勿令透气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
