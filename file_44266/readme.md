序号：44281
名称：上清喉片
组成：儿茶720两，薄荷144两，硼砂72两，槟榔72两，甘草36两，乌梅36两，冰片36两，诃子或山豆根7两2钱。
出处：《中药制剂手册》。
主治：咽喉肿痛，声音嘶哑，口干唇燥。
加减：None
功效：清热，消炎，利咽喉。
用法用量：每服6片，温开水或清茶送下，1日3次。
制备方法：各药单放，先将冰片研为细末，再取乌梅、诃子，用少量水煎透，去核取肉连汁与薄荷拌匀同碾，晒干或低温干燥，再与儿茶、槟榔、甘草共轧为细末，另取淀粉200两，与上项细末和匀，喷洒适量乙醇，搅拌成软材，过16～18目筛网，制成颗粒，压片，每片重0．4克。
临床应用：None
各家论述：None
用药禁忌：忌辛辣刺激及生冷食物。
附注：None
