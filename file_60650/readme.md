序号：60665
名称：通神丸
组成：麻黄（去根，锉细，洒酒、醋炒黄色）1两，桑柴煤（2月2日未出时取东枝，火烧，醋淬）1两，乌菱角壳（11月取，火烧，醋淬）1两，当归（切片，洒酒、醋炒黄色）1两，晚蚕沙（5月5日收，洒酒、醋炒黄色）1两，没药（别研）1两。
出处：《杨氏家藏方》卷十四。
主治：一切颠扑，筋伤骨折，骡马踢踏，迷闷欲绝，及瘀血凝结，痛不可忍。
加减：None
功效：None
用法用量：每服1丸，以生姜、地黄自然汁各1合，酒半升，同煎5-7沸，磨药温服，不拘时候。
制备方法：上除没药外，俱为细末，先将米醋1大碗熬至半碗，入没药同熬成膏，次入余药捣匀，1两分作10丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
