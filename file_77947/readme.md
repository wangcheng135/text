序号：77963
名称：醉红散
组成：天南星1枚（大者，酒浸湿纸裹煨熟，焙干），蜈蚣1条（酒炙），白僵蚕（去丝嘴）7枚，全蝎（去毒）7枚，天仙子1字，朱砂（别研）1钱，紫菀1钱，杏仁1钱，龙骨1钱，防风1钱，龙胆草1钱，蝉衣1钱，百合1钱，牛黄（别研）1钱，白芷1钱，麝香1钱，升麻3钱，大黄（煨）4钱，酥1两，蜜3两。
出处：《小儿病源方论》卷三。
主治：小儿急慢惊风，潮搐涎盛，口眼偏斜，精神昏闷。
加减：None
功效：None
用法用量：周岁儿每服1鸡头实大，沸汤化下，1日2-3次。
制备方法：上药先将牛黄、麝香、朱砂各研为末，除酥、蜜外，余共为末，用水1升，入银锅内煎至3合，以新绵滤去滓，再入锅内，入牛黄、朱砂、麝香末及酥、蜜，以柳篦不住手搅，慢火熬如稠饧为止，入瓷合内盛，密盖收之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
