序号：47834
名称：神仙紫雪
组成：黄金100两，寒水石3斤，石膏3斤，犀角（屑）10两，羚羊角10两（屑），玄参1斤，沉香（镑）5两，木香5两，丁香5两，甘草8两，升麻6两（皆[口父]咀）。
出处：《斑疹备急》。
主治：大人小儿一切热毒，胃热发斑，痘疱麸疹，伤寒热入于胃发斑，小儿惊痫涎厥，走马急疳，热疳，疳黄，疳瘦，喉痹肿痛，及疮疹毒攻咽喉，水浆不下。脚气毒攻内外，烦热，狂易叫走。
加减：None
功效：消痘疱麸疹。解一切热毒。
用法用量：每服1钱，温水化下；小儿半钱1字；咽喉危急病，捻少许干咽之。
制备方法：上以水5斗，煮金至3斗，去金不用，入诸药再煎至1斗，滤去滓，投上好芒消2斤半，微火煎，以柳木篦搅勿停手，候欲凝，入盆中，更下研朱砂、真麝香各3两，急搅匀，候冷，贮于密器中，勿令见风。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
