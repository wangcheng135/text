序号：81977
名称：镇心丹
组成：远志（甘草水煮，去心）5两，熟地黄（酒洗，蒸，焙）5两，新罗人参5两，木鳖子（炒，去壳）5两，白术5两，麦门冬（去心）3两，当归（去芦，酒浸，焙）3两，石菖蒲3两，石莲肉（去心，炒）3两，黄耆（去芦）3两，茯神（去木）3两，柏子仁（拣净）3两，茯苓（去皮）3两，益智仁3两，朱砂50两。
出处：《丹溪心法附余》卷十九引《济生方》。
主治：男子妇人心气不足，神志不宁，一切心疾。
加减：None
功效：None
用法用量：将人参等14味各如法修制，锉碎拌匀，次将朱砂滚和，每服20丸，食后参汤。枣汤、麦门冬汤任下。
制备方法：以夹生绢袋盛贮，用麻线紧系袋口，用瓦锅1只，盛水7分，重安银罐1只于锅内，入白蜜10斤，将药袋悬之中心，不令着底，使蜜浸过药袋，以柔柴火烧令滚沸，勿使火歇，煮3日，蜜焦黑，候足7日侠火，取出淘出众药，洗净朱砂令干，入牛心内，入白蜜于重汤内，蒸如汤干，复以热水从锅弦添下，候牛心蒸烂，取朱砂再换牛心，如上法蒸，凡7次，其砂已熟，用汤水淘净，焙干，入乳钵，玉杵研至10分，米粽为丸，如豌豆大，阴干。
临床应用：None
