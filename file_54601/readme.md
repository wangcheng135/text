序号：54616
名称：柴胡解毒汤
组成：柴胡5-20g，黄芩5-10g，白芍5-10g，芒消5-10g，黄连3-5g，郁金3-5g，广木香3-5g，姜半夏3-5g，大黄3-5g，栀子3-5g，甘草3-5g，夏枯草10g，茵陈（后下）10g。
出处：《古今名方》。
主治：胆道感染湿热型或实火型。症见身热，口干，百苔黄腻，脉洪大。
加减：热重，重用柴胡、黄芩，选加紫地丁、野菊花、黄连；大热、大渴、脉洪大者，加石膏、知母、天花粉、鲜芦根；湿重，重用茵陈、郁金、金钱草、栀子、大黄；痛重，加玄胡、川楝子；呕吐，加半夏、竹茹、生姜；驱蛔，加苦楝根皮、槟榔、使君子；瘀血，加丹参、川芎、红花；气阴欲脱，加独参汤、生脉散。
功效：疏肝清热，通里攻下。
