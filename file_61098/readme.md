序号：61113
名称：铁弹丸
组成：乌头（炮裂，去皮脐）4两，五灵脂4两，附子（炮裂，去皮脐）4两，狼毒半两，防风（去叉）2两，地龙（炒）2两，桂（去粗皮）1两半，虎骨（烧存性）1两，自然铜（煅，醋淬7遍）3两，乳香（研）半两，没药（研）3两半，麝香（研）3钱，龙脑（研）2钱，发灰半两，当归（切，焙）1两，芎藭1两，干蝎（去土沙）半两，海桐皮（锉）2两，天麻1两，白花蛇（酒浸，去皮骨，炙）半两。
出处：《圣济总录》卷七。
主治：一切风。瘫缓卒中，(疒帬)麻；皮肤风毒，结聚肿痛；偏头痛，夹脑风；妇人血风攻注，四肢或痒或痛，心腹刺痛。
加减：None
功效：None
用法用量：每1丸分作4服，早、晚食后服；瘫缓风，生姜热酒化下；(疒帬)麻，薄荷酒送下；皮肤风毒，结聚肿痛，温酒送下；偏头痛，夹脑风，腊茶清送下；妇人血风攻注，四肢或痒或痛，心腹刺痛，当归酒送下。
制备方法：除研者外，上为末，生姜自然汁为丸，如樱桃大。
临床应用：None
各家论述：None
用药禁忌：None
附注：铁弹子丸（《普济方》卷一一四）。
