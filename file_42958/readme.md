序号：42973
名称：保婴出痘方
组成：生地2钱，当归2钱，赤芍2钱，金银花1钱，红花1钱，桃仁1钱，荆芥穗1钱，生甘草5分。
出处：《成方便读》卷四引《福幼编》。
主治：None
加减：None
功效：预防痘疮。
用法用量：上用水2茶杯，煎至1杯。再用小儿本人落下脐带2-3寸，炭火瓦上焙干，研末入药，1日内陆续与小儿服完，次日即出痘点，3日收功，不灌脓亦不结痂。须在小儿初生18日内服之有效。
制备方法：None
临床应用：None
各家论述：方中生地、归、芍益其虚，桃仁、红花行其滞，银花、甘草解其毒，而以荆芥穗从营透表，以达之于外。然无所引之物，则深藏之胎毒何由领出？故特用本人落下之脐带，同气相求，以为导引，且脐带为生命之蒂，元气所系，其阴阳未剖之前先具此物，灵通呼吸，混合太虚，较痘苗尤为功大耳。
用药禁忌：None
附注：保婴稀痘方（《医学集成》卷三）。
