序号：3923
名称：无比神验方
组成：羊粪2两半（半生半烧灰），瓦松2两半（烧灰，半曝干），铁粉2两，胡桃仁1斤，槐胶1两。
出处：《圣惠》卷四十一。
主治：None
加减：None
功效：令生发，兼黑光润泽。
用法用量：取此油调前羊粪等药，每日将头洗净后涂之，不久即生尺余。
制备方法：上药前3味为细散，其胡桃仁，槐胶2味，则捣为1团，填于小口瓶子中令实；又取槐子烂捣，作1片厚饼子，扎作孔子数个，盖瓶子口，更别取1瓶子，须盛得前药瓶口者，仰空瓶子向上相合，即以马粪火烧之1宿，候冷开之，其向下瓶子满中有清油。
临床应用：None
各家论述：None
用药禁忌：None
附注：槐桃膏（《圣济总录》卷一○一）。
