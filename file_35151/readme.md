序号：35161
名称：乳香丸
组成：乳香1两（细研），乌头半两（炮裂，去皮脐），雄黄半两（细研），白附子1两（炮裂），羚羊角屑1两，附子半两（炮裂，去皮脐），晚蚕蛾1两（微炒），羌活1两，防风1两（去芦头），白僵蚕1两（微炒），乌蛇3两（酒浸，去皮骨，炙令黄），麝香1分（细研），当归3分，腻粉1分。
出处：《圣惠》卷二十二。
主治：急风，四肢搐搦，筋骨疼痛。
加减：None
功效：None
用法用量：每服10丸，以热豆淋酒送下，不拘时候。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：原书卷二十三乳香丸无当归，治风，四肢拘挛，筋骨疼痛。
