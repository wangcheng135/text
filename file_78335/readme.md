序号：78351
名称：蜀金牙散
组成：金牙1分（研），蜈蚣（炙）1枚，蜥蜴（石上者，炙）1枚，附子（炮）1枚，人参4分，蜣螂7枚（炙），徐长卿14枚，芫青（炙）14枚，斑蝥（去翅足，熬）14枚，雄黄1分（研），桂心4分，鬼臼2分，野葛1分（炙），毒公3分，芎2分，石长生2分，椒（去汗目）2分，大黄2分，甘草（炙）2分，蛇蜕皮（炙）2分，露蜂房（炙）2分，曾青（无，蓝青代，别研）2分，真珠（别研）2分，丹砂2分，鬼督邮2分，乌头（炮）2分，狼毒2分，石膏5分（研），藘茹1分，芜荑2分，鬼箭2分，藜芦（炙）2分，鹳骨（炙）2分，雷丸2分，干漆（熬）2分，龟甲（炙）2分，狼牙4分，亭长7枚（炙），贝母2枚，凝水石5分，牛黄（别研）4分，胡燕屎4分，桔梗3分，铁精1分（研），消石2分（研）。
出处：《外台》卷十三引《深师方》。
主治：鬼疰风邪，鬼语尸疰，或在脊胁，流无常处，不喜见人，意志不定，面目脱色，目赤鼻张，唇焦，爪甲黄。
加减：None
功效：None
用法用量：每服1刀圭，食前酒调下，1日2次，不知稍增之。有虫随大小便出。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：忌猪肉、冷水、生葱、海藻、菘菜、生血、狸肉。
附注：金牙散（《千金翼》卷二十）。
