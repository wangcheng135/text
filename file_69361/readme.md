序号：69377
名称：犀角散
组成：犀角屑1两，羌活3分，羚羊角屑3分，人参半两（去芦头），甘菊花半两，独活半两，芎半两，白术半两，黄耆半两（锉），石膏1两，汉防己半两，防风半两（去芦头），黄芩半两，天麻半两，枳壳半两（麸炒微黄，去瓤），蔓荆子半两，当归半两（锉，微炒），甘草1分（炙微赤，锉）。
出处：《圣惠》卷三。
主治：肝中风。流注四肢，攻头面疼痛，心神烦热，言语謇涩，上焦风热，口眼斜，脚膝疼痛无力。
加减：None
功效：None
用法用量：每服3钱，以水1盏，加生姜半分，煎至6分，去滓温服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：《医方类聚》引《神巧万全方》有酸枣仁、白芷各半两，无汉防己、蔓荆子。
