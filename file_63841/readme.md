序号：63857
名称：清暑固阴汤
组成：白扁豆3钱（炒），茯苓2钱，滑石1钱半，香薷5分，川厚朴5分，葛根4分，木瓜1钱，姜炭2分。
出处：《温氏经验良方》。
主治：小儿伤暑初起，下泻上渴，身热，额与胸腹更甚，有时热退，旋又发热，心中烦躁，坐卧不安，小便短赤，粪色时黄时黑时红不定，试之烙手，口喜冷饮，关纹红，舌苔白。
加减：呕吐者，加藿香1钱，竹茹4分；不呕吐者，加甘草3分。
功效：None
用法用量：上略煎数开，冷服。热服反增腹泻。2岁小儿照本方服之；未及2岁者，2服分作3服；过2岁者，3服作2服，无论体之强弱，照此服；大人2服作1服。此方绝不寒中，故体弱者亦可。
制备方法：None
临床应用：None
各家论述：方中用扁豆者取其清暑和脾；茯苓渗湿补脾；木瓜利湿驱暑，又酸能泻肝以防侮脾；黑姜、厚朴温以养脾，苦以养心，且湿热内蕴，黑姜足为诸药响导，则香薷、葛根、滑石自无扦格不入之患；兼六一散能止泄泻而利小便，解肌热生津液。分之为清暑之方，合之为养脾养心之助。
用药禁忌：None
附注：None
