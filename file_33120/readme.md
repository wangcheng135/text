序号：33129
名称：万应琥珀膏
组成：全当归5钱，大黄4钱，上血竭4钱，川黄连4钱，小生地8钱，黄芩4钱，真琥珀4钱（另研极细末），生甘草4钱，大风子肉4钱，黄柏4钱，枯矾4钱。
出处：《急救经验良方》。
主治：一切癣、疥、结核，疮疖初起已破，刀伤出血及腐烂者。
加减：None
功效：生肌止痛。
用法用量：用时加黄色三仙丹少许调匀，随时擦之，看疮轻重加之。轻则以少为妙，重则稍多加，不可过多。金刃伤则不必加三仙丹。此膏愈陈愈妙。
制备方法：用好麻油1斤，将药10味熬枯，油色紫红，去滓，入琥珀末并黄蜡2两，候化开离火，用槐枝1根搅匀，冷定成膏，不可太老，埋入土内，退去火毒，一月取出。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
