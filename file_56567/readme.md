序号：56582
名称：大料神秘左经汤
组成：麻黄（去节）、干葛、细辛、厚朴（姜制，炒）、茯苓、防己、枳壳（麸炒，去瓤）、桂心、羌活、防风（去叉）、柴胡、黄芩、小草（即远志苗）、白姜（炮）、半夏（汤洗去滑）、甘草、麦门冬（去心）各等分。
出处：《三因》卷三。
主治：风寒暑湿流注足三阳经，手足拘挛疼痛，行步艰难，憎寒发热，自汗恶风，头眩腰重，关节掣痛；或卒中昏塞，大小便秘涩；或腹痛，呕吐下利，恶闻食臭，髀腿顽痹，缓纵不随，热闷惊悸，心烦气上，脐下冷痹，喘满肩息。
加减：自汗，加牡蛎、白术，去麻黄；肿满，加泽泻、木通；热甚无汗，减桂，加橘皮、前胡、升麻；腹痛吐利，去黄芩，加芍药、附子（炮）；大便秘，加大黄、竹沥；喘满，加杏仁、桑白皮、紫苏，所加并等分。
功效：下气，消痰，散风湿，退肿，进饮食，令人不虚。
用法用量：每服4大钱，水1盏半，加生姜3片，大枣1个，煎7分，去滓，空腹服。
制备方法：上锉散。
临床应用：None
各家论述：None
用药禁忌：None
附注：神秘左经汤（《得效》卷九）。
