序号：4065
名称：木瓜丸
组成：木瓜（切，曝干）2两，附子（炮裂，去皮脐）1两，磁石（燃，醋淬2-7遍，研）半两，甘遂（麸炒）半两，白牵牛（炒）半两，羌活（去芦头）半两，陈橘皮（汤浸去白，炒）半两，防己半两，巴戟天（去心）半两，茴香子（炒）半两，木香半两，地龙（炒）半两，干姜（炮）半两，干蝎（炒）半两，防风（去叉）半两。
出处：《圣济总录》卷五十二。
主治：肾脏风毒气攻注，四肢疼痛，及疮肿烦热。
加减：None
功效：None
用法用量：每服15丸，空心温酒送下。微利为效。
制备方法：上为末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
