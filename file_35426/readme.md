序号：35436
名称：三子养亲汤
组成：紫苏子、芥菜子、萝卜子。
出处：《杂病广要》引《皆效方》。
主治：高年咳嗽，气逆痰痞。
加减：若大便素实者，临服加熟蜜少许；若冬寒，加生姜3片。
功效：顺气降逆，化痰消食。
用法用量：上药各洗净，微炒，击碎。看何证多，则以所主者为君，余次之。每剂不过3钱，用生绢袋盛之，煮作汤饮，随甘旨，代茶水啜用，不宜煎熬太过。
制备方法：None
临床应用：None
各家论述：1.《韩氏医通》：紫苏子，主气喘咳嗽。白芥子主痰，萝卜子主食痞兼痰。_x000D_
2.《僵方考》：年高痰盛气实者，此方主之。痰不自动也，因气而动，故气上则痰上，气下则痰下，气行则痰行，气滞则痰滞。是方也，卜子能耗气，苏子能降气，芥子能行气。气耗则邪不实，气降则痰不逆，气利则膈自宽，奚痰患之有？此方为人子事亲者设也。虽然治痰先理气，此治标之论耳，终不若二陈有健脾去湿治本之妙也，但气实之证，则养亲汤亦径捷之方也。_x000D_
3.《医方集解》：此手足大阴药也。白芥子除痰，紫苏子行气，莱菔子消食。然皆行气豁痰之药，气行则火降，而痰消矣。_x000D_
4.《成方便读》：夫痰之生也，或因津液所化，或由水饮而成，然亦有因食而化者，皆由脾运失常，以致所食之物，不化精微而化为痰。然痰壅则气滞，气滞则伤肺，气失下行之令，于是为咳嗽，为喘逆等证矣。病因食积而起，故方中以莱菔子消食行痰；痰壅则气滞，以苏子降气行痰；气滞则膈塞，白芥子畅膈行痰。三者皆治痰之药，而又能于治痰之中各逞其长，食消气顺，喘咳日宁，而诸证自愈矣。
用药禁忌：None
附注：三子汤（《寿世保元》卷三）。
