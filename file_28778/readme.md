序号：28786
名称：十皮五子饮
组成：茯苓皮1钱5分，草果皮1钱5分，牡丹皮1钱5分，地骨皮1钱5分，五加皮1钱5分，大腹皮1钱5分，甘草皮1钱5分，菟丝子1钱5分，大腹子1钱5分，车前子1钱5分，生姜皮1钱5分，木通皮1钱5分，木瓜皮1钱5分，紫苏子1钱5分，葶苈子1钱5分。
出处：《冯氏锦囊》卷十四。
主治：一切鼓肿胀，并气虚中满，单腹胀。
加减：None
功效：None
用法用量：水煎服。如要断根者，将15味药等分为细末，用未下水之雄猪肝1个，先将温水煮1滚，取出，用竹尖钻孔数个，入药在内，蒸熟切片，捣蒜蘸食之。不过1-2个。永不发也。
