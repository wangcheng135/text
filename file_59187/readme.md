序号：59202
名称：胶艾芎归汤
组成：当归3钱，人参1钱半，艾叶1钱（醋炒），茯苓1钱半，阿胶3钱（糯米炒），川芎1钱，大枣3枚。
出处：《医略六书》卷二十八。
主治：孕妇小腹痛，脉弦细者。
加减：None
功效：None
用法用量：水煎，去滓。温服。
制备方法：None
临床应用：None
各家论述：妊娠劳伤冲任，阴血耗伤而经脉不足，络虚召寒，故小腹疼痛，胎因不安焉。人参扶元补气以充血脉，当归养血荣经以养经脉，阿胶补冲任之阻而络脉受荫，川芎行血中之气而经脉调和，茯苓清治节以通调，艾叶理血气以逐冷，大枣缓中益脾元。水煎温服，使血气内充，则经气布护而寒邪自散，何小腹疼痛之不已哉，胎孕有不安者乎？
用药禁忌：None
附注：None
