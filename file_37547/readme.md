序号：37557
名称：抵圣丸
组成：犀角末2钱，蝎稍21个，银末1两，朱砂1两，巴豆28枚（去皮膜），芫花2钱（同巴豆用好醋1盏，煮令醋尽，拣出巴豆，冷水浸洗，控干，再炒令干，捣末，取2分用）。
出处：《博济》卷四。
主治：下虚中积久，曾取转不得者。
加减：None
功效：None
用法用量：每服1岁以上、3岁以下2丸，小可只1丸，如小儿因惊积聚粘滑，毒物在于脾胃，累曾取下，变成虚积，枣汤送下；体热困闷，眼合不开，黄连、甘草、薄荷、桃仁汤化腻粉1字许送下；大人吃食吐逆，心腹胀满，夜有盗汗，日渐羸瘦，用姜、枣汤送下5-7丸，妇人血气，米醋汤送下5-7丸。更在临时约其虚实，加减用之。
制备方法：上为细末，再研如面，将巴豆另研如糊，和匀，以水煮面糊为丸，如小绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：紫金丹（《幼幼新书》卷二十二引《张氏家传》）。
