序号：78162
名称：膈气散
组成：肉豆蔻仁5两，木香5两，干姜5两，厚朴（去粗皮，生姜汁制，炒）5两，青皮（去白）5两，甘草（爁）5两，三棱（炮）10两，益智仁10两，莪荗（炮）10两，肉桂（去粗皮）10两，陈皮（去瓤）10两，槟榔10两，枳壳（去瓤，麸炒）10两。
出处：《局方》卷三。
主治：五种膈气。三焦痞塞，胸膈满闷，背膂引疼，心腹膨胀，胁肋刺痛，食饮不下，噎塞不通，呕吐痰逆，口苦吞酸，羸瘦少力，短气烦闷。
加减：None
功效：常服顺气宽中，消痃癖积聚，散惊忧恚气。
用法用量：每服2钱，水1盏，入生姜2片，大枣半个，同煎7分，和滓热服；如不及煎，入盐少许，沸汤点服亦得，不拘时候。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
