序号：52588
名称：香连膏
组成：白沙蜜5两（绢滤去滓），硇砂5钱（通明者，研），乳香（研）1钱，青盐1钱，铅丹1钱，黄连（去须）3两（为细末）_x000D_
以上6味除蜜外，并用新汲水3盏，于银石器内同煎至1盏，后入蜜，更用慢火熬成膏，不住手搅，候引之如丝线，以重绢绞去滓，入瓷瓶内盛贮。_x000D_
水银半钱，轻粉1钱，龙脑1钱，麝香（研）1钱。
出处：《普济方》卷七十八。
主治：眼生翳膜。
加减：None
功效：None
用法用量：除打损眼外，并可治。
制备方法：上除前膏外，将后4味一处细研匀，入药膏内，用油单封三五重系定。如春、夏、秋各时，即以麻绳子坠在井底，7日取出；若冬月合时，即于背阴处封闭14日出之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
