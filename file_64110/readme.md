序号：64126
名称：猪肝丸
组成：豮猪肝半具（去脂膜，以酒5升煮令烂，细切，后入药末），柴胡（去苗）1两，厚朴（去粗皮，生姜汁炙）1两，干姜（炮裂）1两，附子（炮裂，去皮脐）1两，缩砂（去皮）1两，白术1两，陈橘皮（汤浸，去白，炒）半两，当归（切，炒）半两，芍药半两，陈曲（炒）1分，肉豆寇（炮，去壳）1分，桂（去粗皮）1分，木香1分，黄（去须）1分。
出处：《圣济总录》卷八十八。
主治：虚劳，木思饮食，腹肚不调，目疮痰逆，及脏腑久冷。
加减：None
功效：None
用法用量：每服20丸，温酒送下，不拘时候。
制备方法：上15味，除猪肝外，为末，入木臼，将猪肝相和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
