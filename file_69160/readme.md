序号：69176
名称：黄金碧玉膏
组成：白占1两，黄占5钱，头发5钱，归身5钱。
出处：《发背对口治诀论》。
主治：发背、对口，腐肉已净，久不收口者。
加减：如痛，加乳香、没药各1钱5分，肉桂3钱（研），大附子3钱（研）；肉桂、附子2味阳毒不用，若阴毒久不收口，塌陷者加入如神。
功效：长肉生肌止痛。
用法用量：凡毒久不收不长肉以此膏敷之，外以好膏药盖之，或油纸亦可。至1昼夜以猪蹄汤洗去，3换3次而愈。
制备方法：上药用麻油6两，以头发先熬枯，去滓，再下归身熬枯去滓，后下黄白占，待化开再下乳香、没药2味，化开和匀成膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
