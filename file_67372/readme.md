序号：67388
名称：黄士汤
组成：伏龙肝（鸡子大）2枚，桂心1两，干姜1两，当归1两，芍药1两，白芷1两，甘草1两，阿胶1两，芎1两，细辛半两，生地黄2两，吴茱萸2升。
出处：《千金》卷十二。
主治：吐血，衄血。
加减：None
功效：None
用法用量：上(口父)咀。以酒7升，水3升，合煮取3升半，去滓，纳胶，煮取3升，分3服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》；《金匮》黄土汤治先便后血，《千金》取治内衄，于本方中除去附子、黄芩，参入姜、桂、萸、辛，佐伏龙肝以散结，芎、芍药佐胶、地以和营，以无附子之雄烈，且有地黄之滋血，故无藉于黄芩也。
用药禁忌：None
附注：干地黄汤（《普济方》卷一八八）。
