序号：53879
名称：息贲丸
组成：厚朴（姜制）8钱，黄连（去头，炒）1两3钱，干姜（炮）1钱半，桂（去皮）1钱，巴豆霜4分，白茯苓（去皮）1钱半（另末），川乌头（炮制，去皮）1钱，人参（去芦）2钱，川椒（炒，去汗）1钱半，桔梗1钱，紫菀（去苗）1钱半，白豆蔻1钱，陈皮1钱，青皮半钱，京三棱（炮）1钱，天门冬1钱。
出处：《东垣试效方》卷二。
主治：肺之积，右胁下覆大如杯，久不已，令人洒淅寒热，喘咳，发肺壅。
加减：None
功效：None
用法用量：初服2丸，1日加1丸，2日加2丸，渐加至大便溏，再从2丸加服，食远煎淡生姜汤送下，周而复始，积减大半止服。
制备方法：除茯苓、巴豆霜旋入外，上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
