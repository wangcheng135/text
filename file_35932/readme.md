序号：35942
名称：参附三白汤
组成：老东参1钱半，生白术1钱半，白茯苓1钱半，灼白芍1钱半，黑附块1钱，清炙草8分，生姜两片，大红枣2个。
出处：《重订通俗伤寒论》。
主治：阴证发斑。内伤脾阳，阳为阴逼者，斑点隐隐而稀，色多淡红，或夹淡灰，或夹(白光)白，多则六七点，少则三五点，形如蚊迹，只见于手足，或略见于腹部，似斑而实为细疹，四肢厥冷，神倦嗜卧，喜向里睡，神识似寐非寐，乍清乍昧，声低息短，少气懒言，大便多溏，溺色清白或淡黄，舌苔白而嫩滑，或胖嫩而黑润。
