序号：29269
名称：远志散
组成：远志1两（去心），白术1两，肉桂1两半（去皱皮），人参1两（去芦头），鳖甲1两半（涂酥炙令黄，去裙襕），天门冬1两（锉，去心，焙），杜仲1两（去粗皮，微炙令黄，锉），川椒1两（去目及闭口者，微炒去汗），牛膝1两（去苗），白茯苓1两，薯蓣1两，山茱萸1两，柏子仁1两，生干地黄1两，石斛1两（去根，锉），黄耆1两（锉），甘草半两（炙微赤，锉）。
出处：《圣惠》卷二十六。
主治：心虚，劳损羸瘦，四肢无力，心神昏闷。
加减：None
功效：除寒热，利腰脚，充肌肤，益气力。
用法用量：每服1钱，空心及晚食前以温酒调下。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：忌鲤鱼、苋菜。
附注：None
