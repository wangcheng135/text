序号：74078
名称：愈风丸
组成：白牵牛（半生半炒）1两，虎骨（炙，醋淬）1两，山栀子仁半两，甘菊花半两，当归（切，焙）半两，骨碎补半两，白芥子半两，草乌头3两（纳1两于童子小便内浸3日，余2两以桑柴烧为灰），地龙（去土）1分，硇砂（研）1分，牛膝（酒浸，切焙）1两，肉苁蓉（酒浸，切焙）1两，麻黄（去根节）1两，自然铜（煅，醋淬7遍）1两，萆薢1两，地骨皮1两，乌头（炮裂，去皮脐）半两，枫香脂（炒）半两，胡芦巴（炒）半两。
出处：《圣济总录》卷十八。
主治：大风疾。
加减：None
功效：None
用法用量：每服30丸，温酒送下，空心、日中、临卧各1服，加至50丸。第3日用淋渫药。
制备方法：上为末，醋煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
