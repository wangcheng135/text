序号：45272
名称：枳实导滞丸
组成：大黄1两，枳实（麸炒、去瓤）5钱，神曲（炒）5钱，茯苓（去皮）3钱，黄芩（去腐）3钱，黄连（拣净）3钱，白术3钱，泽泻2钱。
出处：《内外伤辨》卷下。
主治：伤湿热之物，不得施化，而作痞满，闷乱不安；脾胃湿热引起的胸满腹痛，消化不良，积滞泻泄，或下痢脓血，里急后重。
加减：None
功效：祛湿清热，消积导滞。
用法用量：每服50-70丸，食远温开水送下。
制备方法：上为细末，汤浸蒸饼为丸，如梧桐子大。
临床应用：None
各家论述：《医方集解》：此足太阴、阳明药也，饮食伤滞，作痛成积，非有以推荡之则不行，积滞不尽，病终不除。故以大黄、枳实攻而下之，而痛泻反止，经所谓“通因通用”也；伤由湿热，黄芩、黄连佐以清热，茯苓、泽泻佐以利湿；积由酒食，神曲化食解酒，温而消之；芩、连、大黄苦寒太过，恐伤胃气，故又以白术之甘温，补土而固中也。
用药禁忌：None
附注：枳术导滞丸（《脾胃论》）、导气枳实丸（《医学入门》卷八）。本方加木香、槟榔，名“木香导滞丸”（见《医学正传》卷二）。本方方名，《金匮翼》引作“导滞丸”。
