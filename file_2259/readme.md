序号：2259
名称：升麻汤
组成：川升麻半两，桔梗（炒）半两，薏苡仁半两，地榆半两，牡丹皮半两，芍药半两，子芩（刮去皮）半两，甘草3分（炙）。
出处：《本事》卷二。
主治：肺痈，吐脓血作臭气，胸乳间皆痛。
加减：None
功效：None
用法用量：每服1两，水1升半，煎至5合，去滓，日2-3服。
制备方法：上锉为粗末。
临床应用：None
各家论述：《本事方释义》：升麻气味苦辛微温，入足太阴、阳明之表药；桔梗气味苦辛平，入手太阴；薏苡仁气味甘微寒，入手足太阴、手少阴；地榆气味苦咸微寒，入手足阳明；子芩气味苦平，入手足少阳、阳明。此肺痈已成脓血，臭气上升，胸乳作痛，以表药提其清阳，以泄肺清热之药，泻其浊阴，戊己二味和中，清既得升，浊亦得降，焉不奏功耶?
用药禁忌：None
附注：牡丹散（《得效》卷十九）。
