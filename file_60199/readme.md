序号：60214
名称：逐血破瘀汤
组成：水蛭2至4钱，虻虫2至4钱，地龙3至5钱，（庶虫）虫2至4钱，黑丑3至5钱，路路通5钱至1两，透骨草3至5钱，水红花子3至5钱，盘龙参3至5钱，紫草3至5钱。
出处：《赵柄南临床经验集》。
主治：深部栓塞性静脉炎（血痹），腹腔瘀血（血瘕），腹腔肿物（癥瘕）。
加减：寒凉重者，加紫油肉桂1至2钱。
功效：活血破瘀，通经活络。
用法用量：None
制备方法：None
临床应用：None
各家论述：本方为活血汤方的重剂。方中水蛭、虻虫、地龙、（庶虫）虫破血逐瘀；紫草、水红花子软坚理气化痰；黑丑峻下，可以清除陈旧的瘀滞；路路通、透骨草活血通络化瘀；盘龙参益气滋阴而扶正。本方祛邪扶正兼顾，但以祛邪为主。
用药禁忌：None
附注：None
