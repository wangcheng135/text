序号：22876
名称：观音丸
组成：血竭2钱，熊胆（研）2钱，人参半两，蛇蜕（皂角水洗，新瓦倍）半两，木贼1两（去节，童便浸，晒），苍术1两（童便浸2宿，晒），威灵仙1两，鹰爪黄连1两（去须），地骨皮1两（洗，晒），蔓荆子1两，茺蔚子1两，车前子1两，川芎1两，当归1两，羌活1两，蝉蜕1两（洗，晒），石决明（煅，存半生）1两，蚕蜕纸20幅（炒焦）。
出处：《直指》卷二十。
主治：内外障失明，或欲结青光内障，或赤脉疼痛。
加减：None
功效：None
用法用量：每服70-80丸，温米泔送下或石菖蒲煎汤送下，食后常服。
制备方法：上为细末，用羯羊肝1具，去筋膜，慢火煮至半生熟，带血性和药同捣，以粟米粉用肝汁煮，糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
