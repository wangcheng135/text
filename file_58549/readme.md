序号：58564
名称：秘传反魂丹
组成：鹤虱3钱7分5厘，莪术3钱7分5厘，三棱3钱7分5厘，陈皮3钱7分5厘，黄连3钱7分5厘，大黄3钱7分5厘，胡黄连3钱7分5厘，雄黄3钱5分，枳壳3钱5分，青皮3钱5分，黄芩3钱5分，乳香1钱5厘，丁香1钱5厘，甘草减半，牵牛3钱7分，知母3钱7分，熊胆（渍水去腥气）3钱7分，麝香1钱2分5厘，白丁香（少炒赤小豆）150粒。
出处：《续名家方选》。
主治：一切癫痫，小儿内证诸疾，或疟疾久不愈而为疟母，及惊风癖疾，疳虫。
加减：None
功效：None
用法用量：凡癫痫新发者，服此丸30丸；或虽经年不愈者，1日1发、2日1发者，服之可至100日。与之后不出5日而1发者，数10日不再发者，愈之兆也。小儿5岁以上3丸，10岁以上7丸，15岁以上15丸。
制备方法：先17味为末，后纳麝香、熊胆和匀，荞麦面为丸，如梧桐子大，辰砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
