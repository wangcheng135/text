序号：67429
名称：黄犬肉丸
组成：磁石3两（煅，水飞），川乌2两（炮，去皮尖），附子2两（炮，去皮脐），桑寄生2两，鹿茸2两（燎去毛，酒蒸），麋茸2两（同上制），仙茅2两（酒浸），肉苁蓉2两（酒浸，切，焙），川巴戟2两（去心），葫芦巴（炒）2两，沉香1两（别研），阳起石1两（煅，研极细），龙骨1两（生用），虎胫骨1两（酥炙），覆盆子（酒浸）1两，青盐（别研）。
出处：《济生方》卷。
主治：真精衰惫，脐腹冷痛，小便频数，头晕耳鸣，足胫酸冷，步履无力，腰背拘痛，水谷不消，饮食无味，肌肉瘦悴，遗泄失精。
加减：None
功效：None
用法用量：每服70丸，空心盐酒、盐汤任下。
制备方法：上为细末，用犬肉2斤，以酒、葱、茴香煮烂，杵和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
