序号：35609
名称：侧矮丸
组成：侧耳根（干）15斤，女贞子15斤，矮茶风15斤，破故纸6.3斤，墨旱莲15斤，五味子5斤。
出处：《四川中草药通讯》（1972；1：44）。
主治：老年慢性支气管炎。
加减：None
功效：清热解毒，敛肺止咳。
用法用量：分2个阶段，先用侧矮合剂每日1剂，控制急性发作。急性发作控制后（脓痰消失、肺部罗音及哮鸣音消失，咳嗽基本控制），再用侧矮丸，每次1丸，每日2次，连服30天为1疗程。
制备方法：上将前5味药水煮，去滓，浓缩成浸膏，将五味子磨成细粉，加入浸膏，再加苯甲酸钠适量，混匀，制丸1000粒。
临床应用：None
各家论述：None
用药禁忌：None
附注：侧矮合剂：侧耳根、矮茶风、墨旱莲全草鲜品各一两或干品各5钱，水煎服。
