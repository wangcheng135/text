序号：44045
名称：咽喉噙化丹
组成：生地15g，熟地15g，白茯苓15g，天冬12g，麦冬12g，西瓜霜12g，硼砂12g，法夏12g，乌梅肉12g，人参6g，冰片6g，食盐6g，黄柏9g，知母9g，薄荷9g，诃子肉（煨）9g。
出处：《古今名方》引易玉泉家传方。
主治：虚火喉痹，虚火乳蛾。
加减：None
功效：滋阴润燥，降火利咽。
用法用量：用时含入口中，慢慢噙化。
制备方法：将以上植物药类先为细末，过细绢罗筛后，再下西瓜霜、硼砂、冰片、食盐，合研为极细末，将药末摊开，先喷食醋（约3份），再炼蜜（约7份）为丸，如龙眼大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
