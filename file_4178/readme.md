序号：4178
名称：木香三棱丸
组成：木香7钱，京三棱（炮，净，切）1两半，石三棱1两半（择净），肉豆蔻1两半（搓去尘），广术1两半（煨，净，切），鸡爪三棱1两半（择净），青皮（去瓤）1两半，槟榔（削净）4两，巴豆32个（去皮，全仁者，轻手麸炒黑焦，内外色同为度）。
出处：《医方类聚》卷一○九引《经验秘方》。
主治：肠胃积聚，上下不通。
加减：None
功效：利气，下痰，消食。
用法用量：每服20-50丸，生姜蘸盐少许，细嚼，熟汤送下。食前服，则消宿食，利气下痰；食后服，则消新食，利气下痰。
制备方法：上为细末，姜汁糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
