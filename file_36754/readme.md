序号：36764
名称：固肠散
组成：陈皮（炒）20两，木香1两（不见火），肉豆蔻（生用）2两，罂粟壳（去蒂盖，蜜炙）2两，干姜（炮）2两半，甘草（炙）2两半。
出处：《局方》卷六（吴直阁增诸家名方）。
主治：脾胃虚弱，内受寒气，泄泻注下，水谷不分；冷热不调，下痢脓血，赤少白多，或如鱼脑；肠滑而泻，遍数频并，心腹胀满而痛，食减少力。
加减：None
功效：None
用法用量：每服2钱，以酒1盏，入生姜2片、大枣1个，同煎至7分，温服，不拘时候；如不饮酒者，水煎亦得。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：忌酒、面、鱼腥等物。
附注：方中陈皮，《直指》作“陈米”。
