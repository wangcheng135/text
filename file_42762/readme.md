序号：42777
名称：保阴煎
组成：生地2钱，熟地2钱，芍药2钱，山药1钱半，川续断1钱半，黄芩1钱半，黄柏1钱半，生甘草1钱。
出处：《景岳全书》卷五十一。
主治：男妇带浊遗淋，色赤带血，脉滑多热，便血不止，及血崩血淋，或经期太早，凡一切阴虚内热动血等证。胎气热而不安，及产妇淋沥不止。
加减：如小水多热或兼怒火动血者，加焦栀子1-2钱；如夜热身热，加地骨皮1钱5分；如肺热多汗者，加麦冬、枣仁；如血热甚者，加黄连1钱5分；如血虚血滞，筋骨肿痛者，加当归2-3钱；如气滞而痛，去熟地，加陈皮、青皮、丹皮、香附之属；如血脱血滑及便血久不止者，加地榆1-2钱，或乌梅1-2个，或百药煎1-2钱，文蛤亦可；如少年或血气正盛者，不必用熟地、山药；如肢节筋骨疼痛或肿者，如秦艽、丹皮各1-2钱。
功效：None
用法用量：上以水2钟，煎7分，食远温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
