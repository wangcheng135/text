序号：35977
名称：参麦清补汤
组成：人参5分，麦门冬1钱（酒蒸，晒干），白花粉8分（酒蒸，晒干），前胡1钱，生黄耆3钱，牛蒡子8分（炒，研），炙甘草3分，生甘草2分，酒炒白芍5分，生白芍3分，当归身7分（酒洗），红花5分（酒洗），大川芎7分，桔梗7分，生地黄1钱（酒洗），山楂1钱（去核）。
出处：《痘疹传心录》卷十九。
主治：痘出稠密，毒火既盛，然气血虚弱，津液枯竭，不能制火，以致虚火炎蒸，或烦或渴，或咽喉痛，或鼻时出血，难任温补，不能成浆结痂者；又痘后因虚火口舌生疮者；又痘色虚陷灰白而音哑者。
加减：None
功效：None
用法用量：加生姜1片，龙眼肉3个，同煎，频频温服。
制备方法：None
临床应用：None
各家论述：None
