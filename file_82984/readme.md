序号：83000
名称：乌丸子
组成：赤小豆（炒）1两，白蔹1两，赤芍药1两，何首乌（醋煮）1两，细辛（去苗）1两，草乌（醋煮7次）1两，白及（煨）1两，山桂（去粗皮）1两，南星（面裹煨）1两，当归（酒浸1宿）1两，川芎1两，百草霜1两，骨碎补（去毛，炒）1两，天台乌药（乌豆酒煮后焙干）1两。
出处：《理伤续断方》。
主治：打扑伤损，骨碎筋断，瘀血不散；及一切风疾，筋痿力乏，左瘫右痪，手足缓弱，诸般风损；妇人血疾，产后败血不散，灌入四肢，面目浮肿。
加减：None
功效：None
用法用量：每服50丸，用煨葱酒或煨葱茶任下。
制备方法：上为细末，用煮豆酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：孕妇勿服。
附注：None
