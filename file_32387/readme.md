序号：32396
名称：坐导药
组成：皂荚1两，山茱萸1两，当归1两，细辛2两，五味子2两，干姜2两，大黄半两，矾石半两，戎盐半两，蜀椒半两（一本有葶苈半两，砒霜半两）。
出处：《千金》卷二。
主治：全不产，及断绪。
加减：None
功效：None
用法用量：以绢袋盛，大如指，长3寸，盛药令满，纳妇人阴中，坐卧任意，勿行走急，小便时去之，更安新者，1日1次。必下青黄冷汁，汁尽止，即可幸御，自有子也。若未见病出，亦可至10日安之。其药服朴消汤后即安之，经1日外，服紫石门冬丸。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：《千金翼》有葶苈、苦瓠，无山茱萸；《医学纲目》有吴茱萸、黄葵花，无大黄、山茱萸。
