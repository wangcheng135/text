序号：79881
名称：广胤丹
组成：黄耆（锉细）1两半，人参（上党者，去苗）1两，川续断（锉）1两，泽兰叶（去皮）1两，熟地黄（焙干）1两，牡丹皮（拣净）1两，延胡索1两，白芍药1两，川芎1两，白薇1两，嫩鹿茸（燎去毛，酥酒涂炙干，别杵）1两，白茯苓（去黑皮）1两，当归（去苗，洗净，切，炒干）1两，肉苁蓉（酒浸软，去皱皮，切，焙干）1两，防风（去苗及叉尾者）1两，藁本（去苗土）1两，华细辛（去苗叶土，吹搓，罗过）1两，陈皮（汤浸，去白，焙干）1两，蓬莪术1两，京三棱（2味各和白面裹，慢灰火中煨熟，去面，就热杵碎）1两，干姜（炮裂）1两，木香半两，肉桂（去粗皮）半两，山茱萸半两，甘草（锉，炒）2两，黑附子（炮裂，去皮脐）3钱，覆盆子（去萼枝，扶净）2两。
出处：《御药院方》卷十一。
主治：久无子息。
加减：None
功效：None
用法用量：每服1丸，空心、食前细嚼，温酒送下，1日3次。有孕住服。
制备方法：上为细末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
