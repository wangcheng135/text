序号：26646
名称：巩堤丸
组成：熟地2两，菟丝子（酒煮）2两，白术（炒）2两，北五味1两，益智仁（酒炒）1两，故纸（酒炒）1两，附子（制）1两，茯苓1两，家韭子（炒）1两。
出处：《景岳全书》卷五十一。
主治：膀胱不藏，水泉不止，命门火衰．小水不禁。
加减：如兼气虚必加入参1-2两更妙。
功效：None
用法用量：每服100余丸，空心滚汤或温酒送下。
制备方法：上为末，山药糊为丸，如梧桐子大。
临床应用：None
各家论述：《成方便读》：方中熟地、菟丝、骨脂、韭子，大补肾脏。然所以约束肾中之气者，又在于脾，故以白术、山药大补脾土；益智辛香温暖，独入脾家，且能于固摄之中，仍寓流动之意；附子助其火；茯苓去其邪水；而以五味子一味，固其关巩其隄也。
用药禁忌：None
附注：None
