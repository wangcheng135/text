序号：22856
名称：西川石刻安肾丸
组成：青盐4两，鹿茸（炙）1两，柏子仁（净）1两，石斛1两，附子1两，川乌（炮）1两，巴戟（去心）1两，肉桂1两，菟丝子1两，苁蓉1两，韭子1两，葫芦巴杜仲1两，破故纸（炒）1两，石枣1两，远志1两，赤石脂1两，茯苓1两，茯神1两，茴香（炒）1两，苍术4两，川楝子4两，川椒4两，山药4两。
出处：《玉机微义》卷十九引《局方》。
主治：真气虚惫，脚膝弱缓，夜梦遗精，小便滑数而清。
加减：None
功效：None
用法用量：每服70-80丸，空心盐汤送下。
制备方法：上为末，山药酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：西蜀石刻安肾丸（《医统》卷四十八）。
