序号：83789
名称：乌蛇丸
组成：乌蛇肉4两（酒浸，焙干），虎前胫（醋浸，净刮洗了，涂酥炙）2两，黄松节（酒浸，炙干）1两，天麻（酒浸1宿）1两，牛膝（洒浸1宿）1两，石斛（另取末）1两，萆薢1两，杜仲（锉，炒）1两，菟丝子（酒浸）1两，巴戟（去心）1两，独活1两，防风1两，桂（去皮）1两，肉苁蓉（酒浸）1两，金毛狗脊（去毛）1两，续断1两，荜澄茄（去蒂子）1两，当归1两，附子（炮，去皮脐）1两，木香半两，乳香（另研）半两。
出处：《脚气治法总要》卷下。
主治：风毒湿脚气攻注，脚膝疼痛；或即痒痹生疮，疮中黄水不止。
加减：None
功效：None
用法用量：每服30丸，空心温酒送下，1日2次。
制备方法：上为细末，研匀。用大木瓜去膈子了，蒸令烂，研如糊，以法酒化开，银石器中熬过，和剂前药，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
