名称：朱砂沉香丸
组成：丁香2钱半，木香2钱半，沉香2钱半，乳香2钱半，麝香2钱半，荆三棱2钱半，广荗2钱半，牵牛2钱半，黄连2钱半，雷丸2钱半，鹤虱2钱半，黄芩2钱半，大黄2钱半，陈皮2钱半，青皮2钱半，雄黄2钱半，甘草2钱半，熊胆2钱半，赤小豆300粒（60粒煮烂），白丁香360个（直立者好），轻粉4钱，巴豆7个（取霜）。
出处：《医方类聚》卷一六○引《医林方》。

用法用量：每服1丸，用新汲水浸1宿化开，食后服之。大便出。多年不过2丸。
