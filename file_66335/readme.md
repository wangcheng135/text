序号：66351
名称：雪煎
组成：麻黄10斤，杏仁1斗4升，大黄1斤13两（如金色者）。
出处：《千金》卷九。
主治：伤寒。
加减：None
功效：None
用法用量：上（口父）咀。以雪水5斛4斗渍麻黄于东向灶釜中3宿，纳大黄，搅令调，炊以桑薪，煮得2斛汁，去滓，复纳釜中，捣杏仁，纳汁中复炊之，可余6-7斗汁，绞去滓，置铜器中，又以雪水3斗合煎之，搅令调，得2斗4升，药成可丸，冷凝，丸如弹丸，密盛药，勿令泄气。有病者以3沸白汤5合，研1丸入汤中，适寒温服之，立汗出。若不愈者，复服1丸。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：方中大料麻黄、杏仁，虽有些微大黄，必从麻杏走表，然后缓通余热。
用药禁忌：None
附注：None
