序号：32072
名称：万应丸
组成：人参3分，白术4分半，茯苓6分半，甘草6分半，当归6分，川芎3分7厘，白芍9分，熟地3分6厘，半夏6分，柴胡6分，黄芩3分6厘，黄连3分7厘，地骨皮3分半，知母6分，桔梗7分，陈皮3分半，防风6分，薄荷6分，麻黄8分，枳壳5分，羌活9分，独活4分，藁本6分，石膏3分半，细辛9分，天麻6分，肉桂3分半，木香6分，僵蚕3分，全蝎10个（去头足），小茴6分，菟丝子4分，甘菊3分，杜仲6分，蔓荆子3分半，生地3分半，朱砂5分。
出处：《人己良方》。
主治：小儿1岁之中或月内诸症：脐风，天吊，惊痫，惊搐，外感风邪，头痛发热，痰涌咳嗽，疹子，麻子，痘疮，身热泄泻，痢疾，吐乳，呕逆。大人亦可用。
加减：None
功效：None
用法用量：百病俱用姜汤送下；咳嗽有痰，薄荷汤送下；寒，用姜汤送下；水泻，姜米汤送下；泻黄色，滚水送下；白漏腹痛，姜艾汤送下：麻疹后，作呕食及翻胃，淡姜汤送下。
制备方法：上为细末，炼蜜为丸，重1钱整。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
