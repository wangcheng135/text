序号：53640
名称：家韭子丸
组成：韭子3两（炒），鹿茸3两（酥炙），苁蓉3两（酒洗），熟地5两，当归2两，菟丝3两（饼），萸肉3两，巴戟3两（炒），杜仲3两（炒），肉桂1两半（去皮），干姜1两半（炒）。
出处：《医略六书》卷二十五。
主治：肾脏虚寒，遗溺，脉缓涩者。
加减：None
功效：温肾壮阳。
用法用量：每服3-5钱，淡盐汤送下。
制备方法：上为末，陈酒糊丸。
临床应用：None
各家论述：肾脏虚寒，真阳不秘，故闭藏失职，遗溺不止。韭子壮真阳以温肾，鹿茸补肾脏以壮阳，巴戟温肾脏以祛寒湿，苁蓉润肾脏以温精血，熟地填补真阴，肉桂温暖真阳，萸肉涩精秘气，当归养血营经，杜仲补肾脏以作强，予姜暖胃气以散寒冷，菟丝填补肾脏也。陈酒丸，盐汤下，俾肾脏充足，则真阳秘密，而寒邪自散，水府蓄泄有权，安有溲溺遗失之患乎！
用药禁忌：None
附注：None
