序号：57632
名称：大通青金丹
组成：曾青3分（螺髻者为上，研），金箔49片，丹砂（别研）1两，硫黄（研）1两，胡粉（研）1两，紫石英（研）1两，水银3分（与硫黄结砂子，研），铅霜（研）3分，铅丹（与曾青末、紫石英末、定粉、铅霜再同研）3分，雄黄（与沙子丹砂同研）3分。
出处：《圣济总录》卷五。
主治：中风。
加减：None
功效：None
用法用量：凡中风瘫痪风、手足挛急风、口面斜风、癫痫风、狂风邪风，并用温酒下5丸至7丸，空心、日午、夜卧各1服。
制备方法：上药取一湖南烧药瓶子并盖，用六一泥固济，火燲令干；先下金箔20片，次下沙子，同研4味末，实按平；又下金箔20片，次下曾青，同研5味末，又实按令平；次又下金箔9片，始用盖子，六一泥泥合盖子，留缝1寸不泥合；用火法并再捣罗、出火毒及再研法，并与上方大通水银丸同；用槐胶浓煮汤，去滓，停令温，入白面煮作稠糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
