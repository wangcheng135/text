序号：14764
名称：决明子丸
组成：决明子（微炒）1两，地肤子半两，麦门冬半两（去心，焙），玄参半两，车前子半两，赤茯苓（去黑皮）1两，远志1两（去心，焙），青葙子1两，茺蔚子1两，蔓荆实1两，地骨皮1两，柏子仁1两，山芋1两，人参1两，黄芩（去黑心）1两，防风1两（去叉），大黄（细锉，炒令香）1两，细辛（去苗叶）半两，甘草（炙、锉）1两，黄连（去须）1两。
出处：《圣济总录》卷一○五。
主治：风毒眼赤痛久患，及虚热眼暗。
加减：None
功效：None
用法用量：每服20丸，食后以米饮送下。加至30丸。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
