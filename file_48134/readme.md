序号：48149
名称：神应养真丹
组成：当归（酒浸）、天麻、川芎、羌活、白芍药、熟地黄各等分（1法有木瓜、熟阿胶等分，无羌活）。
出处：《三因》卷三。
主治：厥阴肝经脚气，为四气浸袭肝脏，左瘫右痪，涎潮，昏塞，半身不遂，手足顽麻，语言謇涩，头旋目眩，牙关紧急，气喘，自汗，心神恍惚，肢体缓弱，上攻头目，下注脚膝，荣气凝滞，遍身疼痛。兼治妇人产后中风，角弓反张；堕车落马，打扑伤损，瘀血在内。
加减：None
功效：None
用法用量：每服1丸，木瓜、菟丝子浸酒送下；脚痹，薏苡仁浸酒送下；中风，温酒米汤送下。
制备方法：上为末，炼蜜为丸，如鸡子黄大。
临床应用：None
各家论述：None
用药禁忌：None
附注：神应养真丸（《外科理例》）。
