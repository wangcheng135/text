序号：15361
名称：决明丸
组成：决明子1两（微炒），蕤仁1两（去皮，研），茯神1两（去木），桔梗1两（炒），麦门冬1两（去心，焙），黄连（去须）1两，青葙子3分，枳壳3分（去瓤，麸炒），防风3分（去叉），玄参3分，犀角3分（镑），槟榔3分（煨，锉），升麻3分，生干地黄3分（焙），龙胆3分，沙参3分，紫菀（去苗土）3分，百合半两。
出处：《圣济总录》（文瑞楼本）卷一○七。
主治：肝肺热，毒风目昏。
加减：None
功效：None
用法用量：每服30丸，食后以米饮送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，原书人卫本作"决明子丸"。
