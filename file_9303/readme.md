序号：9303
名称：牛乳方
组成：钟乳1斤（上者，细研之如粉），人参3两，甘草5两（炙），干地黄3两，黄耆3两，杜仲3两（炙），苁蓉6两，茯苓5两，麦门冬4两（去心），薯蓣6两，石斛2两。
出处：《千金翼》卷十二。
主治：None
加减：None
功效：补益。
用法用量：以水5升，先煮粟米7升为粥，纳散7两，搅令匀和，少冷，水牛渴，饮之令足；不足更饮水，日1。余时患渴，可饮清水。平旦，取牛乳服之，生熟任意。牛须3岁以上，7岁以下，纯黄色者为上，余色者为下。其乳常令犊子饮之，若犊子不饮者，其乳动气，不堪服也。其乳牛净洁养之，洗刷饮饲须如法，用心看之。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：慎蒜、猪、鱼、生冷，陈臭等物。
附注：None
