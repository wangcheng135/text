序号：66798
名称：鹿髓丸
组成：巴戟（去心）2两半，肉苁蓉（酒洗，去甲，酥炙）2两，葫芦巴（微炒）2两，破故纸（酒浸，炒）2两，川牛膝（酒洗，去芦）1两，白茯神（去木）1两，菟丝子（酒煮干）2两，甘枸杞（炒）2两，山萸（酒浸，去核）2两半，龙骨（火煅，童便、醋、盐淬九次，井水浸3日，晒干）1两，败龟版（去裙边，酥炙）1两，大附子（童便入盐共煮7次，去皮脐）1两或5钱。
出处：《济阳纲目》卷六十四。
主治：下元冷惫。
加减：None
功效：壮阳补肾。
用法用量：每服60-70丸，空心温酒、米汤。炒盐汤任下。
制备方法：上为细末，用鹿髓同炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
