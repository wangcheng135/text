序号：26422
名称：当归四逆汤
组成：附子1钱半（炮），官桂1钱半，白芍1钱半（酒炒），柴胡5分，当归3钱，吴茱1钱（醋炒），楝子2钱（酒炒），小茴3钱（醋炒），泽泻1钱半。
出处：《医略六书》卷二十四。
主治：阳虚寒疝，脉紧细者。
加减：None
功效：None
用法用量：水煎，去滓温服。
制备方法：None
临床应用：None
各家论述：阳虚于下，寒束于经，虚阳不能布敷而经气被遏，故虚疝时时发作焉。附子补火扶阳，官桂温经散寒，当归养血荣经，白芍敛阴柔筋，柴胡升清阳以除邪，吴茱降逆气以下达，泽泻泻浊阴清肾府，小茴温经气却疝疾，川楝子泻湿热以平虚疝也。
用药禁忌：None
附注：None
