序号：26553
名称：八宝丹
组成：九转胆星5钱，川羌活5钱，北全蝎5钱（酒洗，炒，去足），僵蚕5钱（姜汁炒），双钩藤（净末）8钱，雄黄2钱，天麻8钱（煨），白附子3钱（煨），怀山药5钱，白茯神5钱，远志肉5钱（甘草水洗），真麝香5分，石菖蒲2钱，大梅片5分，真琥珀3钱，珍珠1钱5分（豆腐内煮过，研末，水飞），真牛黄1钱，金箔5分，枳实3钱（炒）。
出处：《寿世新编》卷上。
主治：一切风痰发闭，惊风发搐，昏沉谵语，不知人事，危在顷刻。
加减：None
功效：None
用法用量：1-2岁者服3-5粒，3-4岁或5粒或7粒。
制备方法：上为末，用竹沥调和，少加姜汁为丸，如小痧药大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
