序号：69818
名称：小肾奇方
组成：沉香1钱，木香1钱，公丁香2钱5分，母丁香2钱5分，川楝子5钱（去皮，取肉），破故纸（炒香）5钱，龙骨3钱（煅），大茴香3钱，小茴香3钱，官桂3钱，葫芦巴2钱，陈皮3钱，白茯苓3钱，人参2钱，荔枝核3钱。
出处：《医林绳墨大全》卷五。
主治：大肾冷如冰，坚如石，大如斗。
加减：None
功效：None
用法用量：每服7丸或9丸，淡盐汤送下。
制备方法：共为细末，将黑铅3两熔化，投硫黄3两，俟化，入童便少许，如此9次，为末，前药用多少，入制硫黄多少，要相等，以醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
