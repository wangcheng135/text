序号：3614
名称：巴戟丸
组成：良姜6两，紫金藤16两，巴戟3两，青盐2两，肉桂（去粗皮）4两，吴茱萸4两。
出处：《局方》卷五(续添诸局经验秘方)。
主治：元气虚惫，面目黧黑，口干舌涩，梦想虚惊，眼中冷泪，耳作蝉鸣，腰胯沉重，百节痠疼，项筋紧急，背胛劳倦，阴汗盗汗，四肢无力，及妇人子宫久冷，月脉不调，或多或少，赤白带下。
加减：None
功效：补肾脏，暖丹田，兴阳道，减小便，填精益髓，驻颜润肌。
用法用量：每服20丸，日午、夜卧各1服，暖盐酒送下，盐汤亦得。
制备方法：上为末，酒糊为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《本草纲目》卷十八引作“紫金藤丸”。
