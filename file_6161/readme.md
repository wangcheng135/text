序号：6161
名称：加味健步虎潜丸
组成：龟胶（蛤粉炒成珠）2两，鹿角胶（蛤粉炒成珠）2两，虎胫骨（酥油炙）2两，何首乌（黑豆拌，蒸晒各9次）2两，川牛膝（酒洗晒干）2两，杜仲（姜汁炒断丝）2两，锁阳2两，当归（酒洗炒干）2两，威灵仙（酒洗）1两，黄柏（酒洗晒干，小盐少许酒炒）1两，人参（去芦）1两，羌活1两，干姜1两，白芍药（微炒）1两，云白术（土炒）1两，熟地黄3两，大川附子（童便、盐水各1碗，生姜2两，切片同煮1日，令极熟，水于再添，盐水煮毕取出，剥皮切薄片，又换净水，入川黄连5钱，甘草5钱，同煮长香3炷，取出晒干，如琥珀明亮色方用）1两5钱。
出处：《金鉴》卷八十九。
主治：跌打损伤，气血虚衰，腰胯膝腿疼痛，酸软无力，步履艰难。
加减：None
功效：舒筋止痛，活血补气，健旺精神。
用法用量：每服3钱，空心淡盐汤送下，冬日淡黄酒送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
