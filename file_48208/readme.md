序号：48223
名称：神明白膏
组成：吴茱萸1升，蜀椒1升，芎1升，白术1升，白芷1升，前胡1升，附子30枚，桂心2两，当归2两，细辛2两。
出处：《千金》卷七。
主治：中风恶气，头面诸病，青盲，风目，烂眦，管翳，耳聋，鼻塞，龋齿，齿根挺痛，及痈、痔、疮、癣、疥等。
加减：None
功效：清头风。
用法用量：病在腹内，温酒服如弹丸1枚，1日3次；目痛，取如黍米纳两眦中，以目向风，无风可以扇扇之；诸疮、痔、龋齿、耳鼻百病，皆以膏敷；病在皮肤，炙手摩病上，1日3次。
制备方法：上（口父）咀，醇苦酒于铜器中淹浸诸药1宿，以成煎猪膏10斤，炭火上煎3沸，三上三下，白芷色黄为候。
临床应用：None
各家论述：None
用药禁忌：None
附注：白膏（《普济方》卷三一五）。
