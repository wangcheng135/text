序号：78143
名称：膏药
组成：当归半两，官桂半两，川乌半两，香白芷半两，草乌半两，玄参半两，大黄半两，干地黄半两，赤芍药半两，桃枝21寸，柳枝21寸，红丹半斤，白胶香半斤，蜡2两半，密陀僧1两半。
出处：《医方类聚》卷一七七引《经验良方》。
主治：发背疮。
加减：None
功效：None
用法用量：None
制备方法：上用清油10两，先将当归、桃柳枝等10味入油内煎，以黑为度，去滓，滤净油，慢火再煎，徐徐却入红丹，用长柳枝频频搅匀，药将成，入白胶香，次入蜡，又次入密陀僧，将药滴入水中成珠为度，出药，坐水中1日夜，出火毒可用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
