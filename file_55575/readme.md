序号：55590
名称：桃仁煎
组成：桃仁1升，虻虫1升，朴消5两，大黄6两。
出处：《千金》卷四。
主治：带下，经闭不通。血瘕、血积，脉涩洪大。
加减：None
功效：None
用法用量：取1丸和鸡子黄投酒中，预1宿勿食服之，至晡时，下如大豆汁，或如鸡肝凝血、蛤蟆子，或如膏，此是病下也。
制备方法：上4味为末，别治桃仁，以醇苦酒4升纳铜铛中，炭火煎取2升，下大黄、桃仁、虻虫等，搅勿住手，当欲可丸，下朴消，更搅勿住手，良久出之，可丸乃止。
临床应用：None
各家论述：《医略六书》：妇人血瘀热结，渐成血积、血瘕，故经闭不行，脐腹闷痛不止焉。桃仁破瘀结以消癥积，大黄荡瘀热以化瘕聚，朴消软坚结，虻虫破积血也。醋煮以收之，酒下以行之，使热降瘀消，则冲任调和，而经闭无不通，血瘕无不化，安有脐腹闷痛之患哉！
用药禁忌：None
附注：桃仁煎丸、桃仁丸（《圣惠》卷七十二）、攻积桃仁煎（《医略六书》卷三十一）、桃黄煎（《顾氏医径》卷四）。
