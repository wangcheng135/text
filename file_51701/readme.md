序号：51716
名称：退翳丸
组成：当归32两，香附32两，防风32两，草决明32两，蝉蜕8两，玄参64两，青皮16两，连翘32两，菊花64两，蛇蜕8两，白芷32两，川芎20两，枳壳64两，密蒙花16两，柴胡32两，薄荷16两，黄芩32两，郁金16两，蒺藜64两（炒），木贼草64两，赤芍32两，橘皮32两，谷精草8两。
出处：《全国中药成药处方集》（北京方）。
主治：火眼外障，血翳贯睛，视物不清，羞明涩痛。
加减：None
功效：消障退翳，散风明目。
用法用量：每服2钱，温开水送下，1日2次。
制备方法：上为细末，水泛小丸，滑石为衣，闯亮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
