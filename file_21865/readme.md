序号：21873
名称：利气丸
组成：大黄（生用）6两，黑丑（头末）4两，香附米（炒）2两，木香2两，槟榔2两，枳壳（麸炒）2两，青皮（去瓤）2两，陈皮2两，莪术（煨）2两，黄连2两，黄柏3两（一方加黄芩1两，当归1两）。
出处：《古今医鉴》卷六。
主治：一切气滞，心腹胀闷疼痛，胁肋膨胀，呕吐酸水、痰涎不利，头目眩晕，并食积酒毒，及米谷不化，或下痢脓血，大小便结滞不快，气壅积热，口苦，烦燥，涕唾稠粘。
加减：None
功效：流湿润燥，推陈致新，滋阴抑阳，散郁破结，活血通径。
用法用量：每服50丸或100丸，临卧时淡姜汤送下。以大便通利为度。如不利，再加丸数。
制备方法：上为细末，水为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
