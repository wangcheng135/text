序号：83289
名称：丁香散
组成：丁香半两，人参3分（去芦头），槟榔半两，赤茯苓3分，草豆蔻半两（去皮），白术1两，大腹皮1两，前胡1两（去芦头），厚朴1两（去粗皮，涂生姜汁，炙令香熟），陈橘皮1两（汤浸，去白瓤，焙），诃黎勒皮1两，桂心3分，紫苏茎叶3分，半夏半两（汤洗七遍，去滑），甘草半两（炙微赤，锉）。
出处：《圣惠》卷十三。
主治：伤寒后，胃气不和，吃食痰逆，两胁妨闷，四肢少力。
加减：None
功效：None
用法用量：上为散。每服5钱，以水1大盏，加生姜半分，煎至5分，去滓稍热服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
