序号：28141
名称：苁蓉茸附丸
组成：鹿茸1两（先用草烧去毛，切作片子，用酥炙令香熟为度），苁蓉4两（酒浸1宿，切作片子，焙干）1两，菟丝子6两（酒浸2宿，炒令半干，捣作饼子，焙），牛膝2两（酒浸1宿，切，焙），熟干地黄2两（炒，焙），真乌药1两，川五味子1两，附子1两（炮，去皮脐），白术1两，天麻1两，补骨脂1两（炒），葫芦巴1两（炒），茴香1两（炒），干淡木瓜1两，沉香1分，木香1钱（面煨），丁香2钱（不见火）。
出处：《洪氏集验方》卷三引督府王翰林方。
主治：None
加减：None
功效：平补真元，益养脾肾，固精壮气，暖胃思食。
用法用量：每服30-50丸，空心、临卧以米饮、温酒、盐汤任下。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
