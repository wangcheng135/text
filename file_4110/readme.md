序号：4110
名称：木瓜酒
组成：干木瓜（宣州者）5两，五灵脂1两，川牛膝（去梗）1两，当归1两，白芍药1两，川萆薢1两，川天麻1两，天台乌药（连珠者）1两，防风（去芦）1两，黄耆（锦纹者）1两，威灵仙（去梗）1两，虎胫骨（酥炙）1两，川续断1两，乳香1两，没药1两，白僵蚕（去丝嘴）1两，松节（茯神中木）1两，川乌头（去皮脐，生用）1两。
出处：《医方类聚》卷九十八引《经验秘方》.
主治：干湿脚气，腿脚沉重，行步艰辛及筋脉拳挛，抽掣疼痛，不能伸舒，膝盖肿疼。
加减：None
功效：None
用法用量：每服50丸，食前温酒送下。
制备方法：以好酒2斗，用绢袋盛，浸于内14日。每服1盏，食前温服。晒干滓为末，面糊为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
