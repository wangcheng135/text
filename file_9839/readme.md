序号：9839
名称：二转杏林丹
组成：盐1两，矾1两，皂矾1两，消1两，汞1两。
出处：《灵药秘方》卷下。
主治：痔瘘，肿毒，杨梅，一切疮毒，小儿痞块。
加减：如搽一切疮毒，取药1两，入草乌末1钱，用猪胆汁或鹅胆汁调搽。
功效：None
用法用量：凡痔瘘服10丸，小儿痞块服3丸，肿毒杨梅服5丸，俱黄酒送下。
制备方法：上为末，入罐，升打3炷香，取药；加入明雄5钱，朱砂5钱，硫黄5钱，硼砂3钱，硇砂1钱，研匀入罐，再打5炷香，取药；配入乳香1钱，没药1钱，儿茶1钱，血竭1钱，麝香1分5厘，枣肉为丸，如绿豆大，收固用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
