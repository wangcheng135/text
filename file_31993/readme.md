序号：32002
名称：十膈散
组成：人参1两，茯苓1两，厚朴1两，黄橘皮1两，京三棱1两，枳实1两，神曲1两，甘草1两，白术1两，诃子1两，干姜1两，桂1两，槟榔1分，木香1分（一法添麦芽1两莪术1分，槟榔加1分，木香加1分）。
出处：《鸡峰》卷二十。
主治：冷、热、忧、悲、喜、怒、愁、恚、食、气疾10膈，并因忧惊冷热不调，又乖将摄，更加喜怒无则，贪嗜饮食，因而不化，滞积在胸中，上喘痰嗽，岁月渐深，胸膈噎塞，渐至疲羸。
加减：None
功效：None
用法用量：每服1钱，入盐点之。_x000D_
如脾虚腹胀，心胸满闷，以水1盏，加生姜3片，大枣2个，盐少许，煎至7分，和滓热服。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中枳实、黄橘皮，《卫生宝鉴》作“枳壳”，“青皮”。
