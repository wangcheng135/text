序号：25791
名称：当归摩膏
组成：当归（切，焙）1两半，细辛（去苗叶）1两半，桂（去粗皮）1两，生地黄1斤（切，研，绞取汁），天雄十枚（去皮脐，生用），白芷3分（留1块不锉，全用），芎半两，丹砂（研）1两，干姜（炮）3分，乌头（去皮脐，生用）1两3分，松脂4两，猪脂5斤（别炼，去滓）。
出处：《圣济总录》卷十九。
主治：诸风寒湿，骨肉痹痛。
加减：None
功效：None
用法用量：每用药用火炙，手摩病处千遍。
制备方法：先将八味锉如大豆粒，以地黄汁浸一宿，与猪脂、松脂同慢火煎，候至留者一块白芷黄色，以厚绵滤去滓，瓷盒盛，入丹砂末，不住搅，至凝即止。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
