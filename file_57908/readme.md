名称：益中丹
组成：鹿茸（蜜炙）2钱半，丁香2钱半，木香2钱半，茴香（炒）2钱半，山药2钱半，黄耆（蜜炙）2钱半，木通（油炒）2钱半，官桂（去皮）2钱半，干姜（炮）2钱半，青盐2钱半，石斛2钱半，天雄（炮）2钱半，附子（炮）2钱半，鹅管石（火煅，酒淬）2钱半，阳起石（火煅，酒淬）2钱半，肉豆蔻2钱半，川牛膝（酒浸，焙）2钱半，破故纸（炒）2钱半，葫芦巴（炒）2钱半，菟丝子（酒浸，焙）2钱半，金铃子（去核）2钱半，覆盆子2钱半，熟地黄（酒浸，焙）2钱半，荜澄茄2钱半，马兰花（醋炒）2钱半，肉苁蓉（酒浸，焙）2钱半，韭子（酒浸，焙）2钱半，沉香1钱1字，人参1钱1字，麝香3字。
出处：《朱氏集验方》卷八。
主治：下焦伤竭，脐腹绞痛，两胁胀满，饮食减少，肢节烦疼，手足麻痹，腰腿沉重，行步艰难，目视茫茫，夜梦鬼交，遗泄失精，神气不爽，阳事不举，小便滑数，气虚肠鸣，大便自利，虚烦盗汗，津液内躁。
加减：None
功效：益真气，补虚惫。
用法用量：每服30丸，加至50丸，空心温酒送下。
制备方法：上为细末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
