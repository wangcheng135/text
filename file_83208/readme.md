序号：83226
名称：乌豆仙
组成：青盐1两，生地1两，续断1两，破故纸（炒）1两，仙茅1两，枸杞子4两，何首乌（剖，去皮）4两，淫羊藿2两（去边刺，酥油炙），川椒1两，小茴香1两，牛膝3两，黑豆5升。
出处：《良朋汇集》卷三。
主治：None
加减：None
功效：乌须发。
用法用量：每空心服110粒，嚼烂，白滚水下。服2月须发尽黑，久服大有效验。
制备方法：上药用绢袋盛之，将黑豆铺上面，水10碗，酒10碗，炭火慢慢煮，将豆出搅数次，汤干为度，出锅，晒半干，再入锅，慢火炒干，收瓷器内，晴天时常晒，如夏月8日晒1次。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
