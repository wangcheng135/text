序号：66160
名称：象皮膏
组成：大黄2两，川芎1两，当归1两，生地1两，红花3钱，川连3钱，甘草5钱，荆芥3钱，肉桂3钱，白及3钱，白蔹2钱。
出处：《伤科补要》卷四。
主治：跌打断骨破伤。
加减：None
功效：None
用法用量：如老，加麻油；如嫩，加百草霜1两，搅匀。再入后药：地别虫1两，血竭5钱，象皮5钱，乳香（去油）5钱，没药（去油）5钱，龙骨3钱，海螵蛸3钱，珍珠2钱，人参2钱（共研细末）。入膏内，搅匀。
制备方法：用麻油1斤，先将药片入锅内煎，用柳枝搅匀，俟药滓枯色，以麻布绞去药滓，入黄白占各3两，白及、白蔹末同熬滚，至滴水不化，倾入净水缸内，将膏在水中捻长一块，分作五块，渐入大锅内溶化，膏入带水气，油花滚泛满锅，看药泛红黄色，渐渐化尽其膏，如镜面可以照人，将膏滴水试老嫩，贴手不粘为度。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
