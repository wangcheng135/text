序号：35565
名称：侧子丸
组成：侧子1两（炮裂，去皮脐），白附子半两（炮裂），天南星半两（炮裂），白僵蚕半两（微炒），汉防己半两，萆薢半两（锉），踯躅花半两（酒拌，炒干），牛膝半两（去苗），芎半两，乌蛇肉1两（酒浸，炙微黄），天麻1两，羚羊角屑半两，牛黄1分（细研），麝香1分（细研），硫黄半两（细研），干蝎半两（微炒），桂心半两。
出处：《圣惠》卷二十二。
主治：毒风亸曳，四肢不收，或挛急顽痹。
加减：None
功效：None
用法用量：每服70丸，食前以热酒送下。
制备方法：上为末，研入牛黄、麝香等，以水煮槐胶3两更入少熟蜜同和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻、毒滑、鱼肉。
附注：方中桂心用量原缺，据《普济方》补。
