序号：57292
名称：消痞五疳丸
组成：胡连5钱，木香3钱，麝香6分，芦荟3钱，赤茯苓4钱，蝉退8钱，使君肉5钱，泽泻4钱，蛤粉3钱，胆草3钱，粉甘草3钱，金灯15个，水獭肝1两，羊肝1两，防风3钱。
出处：《全国中药成药处方集》（沈阳方）。
主治：面黄肌瘦，懒食喜睡，腹胀泄泻，日晡潮热，目赤眵多，白翳遮睛，眼睑烂痒，口舌生疮，惊悸咬牙，龈肉肿痛，齿龈出血，喜食生冷，口唇蚀裂，咳嗽气喘，肚大青筋。
加减：None
功效：消痞除积。
用法用量：3岁以下每服3丸，3岁以上每服5丸。
制备方法：除麝香另研，獭肝、羊肝焙干外，余者共为细末，水泛小丸，3厘重。
临床应用：None
各家论述：None
用药禁忌：忌食生冷、硬物。
附注：None
