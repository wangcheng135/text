序号：4721
名称：木香散
组成：木香1两，肉豆蔻1两（去壳），人参1两（去芦头），附子2两（炮裂，去皮脐），当归2两（挫，微炒），干姜1两（炮裂，锉），甘草半两（炙微赤，锉），陈橘皮2两（汤浸，去白瓤，焙），苍术2两（锉，炒），吴茱萸半两（汤浸7遍，焙干，微炒），厚朴2两（去粗皮，涂生姜汁，炙令香熟）。
出处：《圣惠》卷五。
主治：脾脏虚冷，大肠泄痢，腹内疼痛，心腹(四肢)不和，少思饮食。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加大枣3枚，煎至6分，去滓，食前稍热服之。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：木香汤（《圣济总录》卷四十四）。
