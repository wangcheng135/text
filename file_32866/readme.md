序号：32875
名称：羌活汤
组成：羌活（去芦头）1两半，防风（去叉）1两半，黄耆（锉）1两半，五加皮（锉）1两半，牛膝（酒浸，切，焙）1两半，酸枣仁（炒）1合，丹参（锉）1两1分，桂（去粗皮）1两1分，芍药1两1分，麻黄（去根节，煎，掠去沫，焙）1两1分，槟榔4颗（锉），当归（切，焙）2两，玄参2两，木通（锉）2两。
出处：《圣济总录》卷十。
主治：下焦风虚，腰脚（？？）痹不仁，骨髓痠疼，不能久立，渐觉消瘦。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，煎取1盏，去滓，空腹食前温服。良久以热生姜稀米粥投，衣覆微出汗。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：慎外风。
附注：None
