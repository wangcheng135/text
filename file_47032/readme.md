序号：47047
名称：独活犀角汤
组成：独活3两，犀角2两（屑），石斛2两（先煮），丹参2两，侧子1两（炮），防风2两，防己2两，芎2两，生姜3两，当归2两，芍药3两，茯苓4两，桂心1两半，甘草2两（炙）。
出处：《外台》卷十八引《崔氏方》。
主治：脚气毒冲心，变成水，身体遍肿，闷绝欲死者。
加减：None
功效：None
用法用量：以水1斗，煮取2升6合，去滓，分3服，相去10里久再服。服讫任卧，不须取汗，凡服2-3剂，隔5日1服。初服此药，觉腹内气散，两脚有力，行动无妨，或可即停。可常服香豉酒，灸三里穴、绝骨各300壮。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：None
附注：独活汤（《圣济总录》卷八十二）。
