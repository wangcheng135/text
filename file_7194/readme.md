序号：7194
名称：生脉保元汤
组成：生脉散加黄耆、甘草。
出处：《中国医学大辞典·补遗》。
主治：None
加减：None
功效：夏月服之，令人气力充足。
用法用量：清水煎服。
制备方法：None
临床应用：产妇郁冒：以生脉散和保元汤加减（太子参15g，麦冬10g，五味子10g，黄耆15g，甘草10g，肉桂2g，炒白芍15g），治新产妇（包括人流或引产者）产后郁冒90例，症见阵热，热冒于上，头晕，自汗，盗汗，汗出冒解等。如下红逾半日不尽者，加茜根炭15g；失眠者，加柏、枣仁各10g；心惊，筋脉拘挛者，加煅龙骨15g；便闭者，加全瓜蒌15g；食差，苔白腻者，加炒白术10g；病程较长，贫血重者，加黄精15g，当归10g。疗效：痊愈62例，有效24例，无效4例，总有效率95.36%。
各家论述：None
用药禁忌：None
附注：None
