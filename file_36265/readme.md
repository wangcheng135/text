序号：36275
名称：周白水候散
组成：远志5分（去心），白术7分，桂心1两，人参3分，干姜1两，续断5分，杜仲5分（炙），椒半两（汗），天雄3分（炮），茯苓1两，蛇床仁3分，附子3分（炮去皮），防风5分，干地黄5分，石斛3分，肉苁蓉3分，栝楼根3分，牡蛎3分（熬），石韦3分（去毛），钟乳1两（炼），赤石脂1两，桔梗1两，细辛1两，牛膝3分。
出处：《千金翼》卷十二。
主治：心虚劳损。
加减：None
功效：令人身轻、目明。
用法用量：每服钱5匕，酒送下，服后饮酒1升，1日2次；不知更增1钱匕。服之30日身轻目明；80日百骨间寒热除；百日外无所苦，气力日益。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
