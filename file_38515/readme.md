序号：38525
名称：治痧救命丹
组成：马牙消1斤，雄黄2钱，黄丹4两，朱砂8分，皂矾2两。
出处：《千金珍秘方选》.
主治：七十二种痧症，腹痛身胀者。
加减：None
功效：None
用法用量：用之点眼，轻则点两大眼角，点上眼痛，则是痧症，其腹痛身胀自止而病愈矣。若点眼上不痛，乃非痧症，另法治之可也。重者须点数次，以眼不痛为度。
制备方法：上为细末，用井水和匀，入铜锅内，忌铁、锡器板，炭火煎熬，一定要桃、柳枝各3根，共扎一把子，以此炒药，炒至起金星为度。炒好，用新鲜竹简数个（干竹，重汁勿用），须两头有节者，将药趁热装入筒内，紧闭筒口，安放潮湿阴僻处石地之上，7日7夜，受过阴气，再行取出，用乳钵加工，研极细末，收贮瓷瓶内，勿令走风。
临床应用：None
各家论述：None
用药禁忌：切忌入口。
附注：None
