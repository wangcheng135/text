序号：4034
名称：木瓜丸
组成：木瓜（宣州者，去皮瓤，剜作瓮子）2枚，硇砂（以绢袋贮在木瓜内）半两，羊肾1对（研，以上用好酒4升置银器中，候硇砂尽，去袋子，熬成膏），雄雀1对（去皮毛嘴爪肠肚骨，用肉研），肉苁蓉（酒浸1宿，切，焙）2两，葫芦巴1两，附子（炮裂，去皮脐）1两，沉香（锉）1两，木香1两，茴香子（舶上者，炒）1两，楝实（锉，炒）1两，巴戟天（去心）1两，椒红1两，青橘皮（去白，焙）1两，槟榔（锉）1两，桂（去粗皮）1两。
出处：《圣济总录》卷一八六。
主治：肝肾虚，腰膝无力，四肢倦闷，腹胁冷痛。
加减：None
功效：壮筋力，和气血，补暖。
用法用量：每服30丸，空心盐酒送下。
制备方法：上药捣罗12味为末，以前四味膏为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
