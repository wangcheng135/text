序号：28757
名称：十六味冬青丸
组成：冬青叶150g，石榴25g，石膏75g，肉桂50g，豆蔻50g，木香50g，丁香50g，甘草50g，白葡萄干125g，沉香75g，拳参75g，荜茇50g，肉豆蔻50g，红花50g，广枣50g，方海50g。
出处：《中国药典》一部。
主治：胸满腹胀，头昏浮肿，寒嗽痰喘。
加减：None
功效：宽胸顺气，止咳定喘。
用法用量：每100g粉末加炼蜜110一130g，制成大蜜丸。口服，每次1丸，1日1-2次。
制备方法：以上16味，除白葡萄干外，其余冬青叶等15味为粗末；加白葡萄干粉碎，烘干，再粉碎成细末，过筛，混匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
