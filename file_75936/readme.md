序号：75952
名称：蛴螬丸
组成：蛴螬3分（微炒），生干地黄1两，牡丹3分，干漆半两（捣碎，炒令烟出），赤芍药3分，牛膝3分（去苗），土瓜根3分，桂心半两，桃仁3分（汤浸，去皮尖双仁，麸炒微黄），黄芩半两，琥珀半两，虻虫1分（炒微黄，去翅足），水蛭1分（炒微黄），甜葶苈3分（隔纸炒令紫色），赤茯苓1两，海藻3分（洗去咸味），桑根白皮3分（锉）。
出处：《圣惠》卷七十二。
主治：妇人月水久不通，或成肿满，气逆咳嗽，羸瘦食少。
加减：None
功效：None
用法用量：每服20丸，食前温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
