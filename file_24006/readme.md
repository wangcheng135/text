序号：24014
名称：当归养荣汤
组成：防风7分半，白芷7分半，白芍药1钱，熟地黄1钱，当归1钱，川芎1钱，羌活7分半。
出处：《原机启微》卷下。
主治：睛珠痛甚不可忍；又治红赤羞明，泪多眵少。
加减：None
功效：None
用法用量：上作1服。水2盏，煎至1盏，去滓。食后热服。
制备方法：None
临床应用：None
各家论述：以七情五贼劳役饥饱重伤脾胃，生意已不升发，又复血虚不能养睛，故睛痛甚不可忍。以防风升发生意，白芷解利，引入胃经为君；白芍药止痛益气，通血承接上下为臣；熟地黄补肾水真阴为佐；当归、川芎，行血补血，羌活除风引入少阴经为使。血为邪胜，睛珠痛者，及亡血过多之病，俱宜服也。
用药禁忌：None
附注：None
