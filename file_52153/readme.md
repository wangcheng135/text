序号：52168
名称：面膏
组成：杜蘅1两，牡蛎（熬，一云杜苦）1两，防风1两，藁本1两，细辛1两，白附子1两，白芷1两，当归1两，木兰皮1两，白术1两，独活1两，萎蕤1两，天雄1两，茯苓1两，玉屑1两，菟丝子3两，防己3两，商陆3两，栀子花3两，橘皮（一云橘仁）3两，白蔹3两，人参3两，甘松香2两，青木香2两，藿香2两，零陵香2两，丁香2两，麝香半两，白犬脂1升，白鹅脂（无鹅脂以羊髓代之）1升，牛髓1升，羊胰3具。
出处：《千金翼》卷五。
主治：None
加减：None
功效：泽面。
用法用量：每夜涂面，昼则洗却，更涂新者，10日以后色等桃花。
制备方法：以水浸膏髓等5日，日别再易水；又5日，日别一易水；又10日，2日一易水。凡20日止，以酒1升，挼羊胰令消尽去脉，乃细切香，于瓷器中浸之，密封1宿，晓以诸脂等合煎，三上三下，以酒、水气尽为候，即以绵布绞去滓，研之干遍，待凝乃止，使白如雪。
临床应用：None
各家论述：None
用药禁忌：None
