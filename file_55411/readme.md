序号：55426
名称：乙丑丸
组成：硇砂（细研，汤内飞过，去沙石，熬取霜）1两，乌头（生，去皮脐，为末）1两，沉香末半两，五灵脂末半两，干姜末半两，桂（去粗皮，为末）半两，胡椒末半两，巴豆（去皮心膜，研）半两，干漆末3分。
出处：《圣济总录》卷七十一。
主治：食积隐现时作，攻心胁(疒丂)刺痛。
加减：None
功效：None
用法用量：每服3丸，温木瓜酒送下；木瓜汤送下亦得，不拘时候。如大瘕癖积块及诸冷气(疒丂)刺疼痛，或泄痢脓血，食前服5丸至7丸。看虚实加减。
制备方法：上除巴豆外，同研匀，次入巴豆，再研极细，同熟枣肉和作1块，用湿纸裹3-5重，用纸筋黄土泥固济，约厚半指许，熁干，用熟炭火10斤，于乙丑日早渐进火烧，令香为度，以新盆器合，候冷取出，其焦纸灰不用，捣烂，看硬软，再入熟枣肉和捣千余杵得所，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
