序号：8357
名称：半夏散
组成：半夏1两（汤洗7遍去滑），白术1两，赤茯苓1两，人参3分（去芦头），甘草半两（炙微赤，锉），附子3分（炮裂，去皮脐），陈橘皮3分（汤浸，去白瓤，焙），桂心3分，木香3分，大腹皮1两（锉），诃黎勒1两半（煨，用皮），前胡3分（去芦头）。
出处：《圣惠》卷二十六。
主治：肉极，虚寒则胁下阴阴引背痛，不可以动，动则咳嗽胀满，留饮痰癖，大便不利，小腹切痛，膈上有寒。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，大枣3个，煎至6分，去滓，食前温服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：忌饴糖。
附注：None
