序号：43164
名称：三棱化瘿丸
组成：三棱6g，莪术6g，归尾9g，丹参12g，青皮6g，穿山甲6g（醋炙），生牡蛎12g，昆布9g，海浮石9g。
出处：《中医内科临床治疗学》。
主治：瘿气。颈部粗肿日久，赤络显露，按之硬痛，呼吸不畅，声音嘶哑，呛咳气急，或吞咽障碍，舌质暗红，脉象沉涩。
加减：None
功效：行气活血，软坚消瘿。
用法用量：早、晚各服1丸。
制备方法：水煎服；或加倍研细，枣肉为丸，每丸6克。
临床应用：None
各家论述：三棱、青皮疏肝行气；归尾、丹参、莪术、山甲珠活血祛瘀，消肿通络；昆布、海浮石、生牡蛎化痰软坚，散结消瘿；枣肉为丸，既能补脾益气，又能防止诸药攻伐之峻猛，损伤脾胃。组合成方，共具行气活血，散结消瘿之功效。
用药禁忌：虚弱患者，孕妇、哺乳妇均宜忌服。
附注：None
