序号：74223
名称：山萸肉粥
组成：山茱萸肉15-20g，粳米60g，白糖适量。
出处：《药粥疗法》引《粥谱》。
主治：肝肾不足，头晕目眩，耳鸣腰痠，遗精，遗尿，小便频数，虚汗不止，肾虚带下。
加减：肾虚病人，特别是老年肾虚患者，腰痠腿软，头晕耳鸣，配合枸杞子一同煮粥则更好；如出现遗精、遗尿、小便次数过多，_x000D_
可配合芡实煮粥尤妙；如虚汗不止，盗汗过久，则配合浮小麦煮粥。
功效：补益肝肾，涩精敛汗。
用法用量：先将山萸肉洗净，去核，与粳米同入砂锅煮粥，待粥将熟时，加入白糖稍煮即可。3～5天为1疗程，疾病完全治愈后，即可停服。或再间断食用1个时期，以巩固疗效。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：在发热期间，或小便淋涩的患者，均不可服食。
附注：None
