序号：59856
名称：大黄汤
组成：大黄2两，牡丹3两，芥子半升，消石3合，桃仁50枚（去皮，炒，切之）。
出处：《鬼遗》卷三。
主治：肠痈之为病，诊小腹肿痞坚，按之则痛，或在膀胱左右，其色或赤，或白色，坚大如掌，热，小便欲调，时色色汗出，时复恶寒。其脉迟坚，未成脓者；肠内生痈，腹痛，小便数似淋，身甲错，腹皮急，按之濡，如肿状，或连脐生疮。肠痈初起，小腹隐痛，小便淋涩，小腹坚硬如掌而热，按之则痛，肉色如故，或焮赤微肿。
加减：None
功效：None
用法用量：上(口父)咀。以水6升5合，煮取1升，分为两服，脓下；无者，下血大良。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：脉数脓成，不可服此方。
附注：None
