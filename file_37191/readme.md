序号：37201
名称：定惊至宝丹
组成：树上鸣蝉（中伏前1日采，取全翅足，用纸包扎，挂于有风之处，不可浸雨，待至次年5月5日午时修合）1个，朱砂5分，麝香5厘。
出处：《经验各种秘方辑要》。
主治：小儿惊风。
加减：None
功效：None
用法用量：用时以少许点舌尖上。1-2次即愈。
制备方法：上为末，用瓷瓶密封，勿令泄气。
临床应用：None
各家论述：蝉感风露之气以生，身轻音朗，得金气之发扬者，其蜕又象人身皮壳之肺经，故性能驱风热，定魂魄，义取金克木也；朱砂入心镇怯；麝香辟邪通络。点舌尖者，心之外候也。孩提纯阳多热，最易化风，风火相煽，惊从内生。此方定惊之至宝，保赤之灵丹，故赐其名曰定惊至宝丹。
用药禁忌：None
附注：None
