序号：37033
名称：奔豚丸
组成：厚朴（姜制）7钱，黄连（去须，炒）5钱，白茯苓（去皮，另末）2钱，川乌头（炮）半钱，泽泻2钱，苦楝（酒煮）3钱，玄胡1钱半，全蝎1钱，附子（去皮）1钱，巴豆霜4分，菖蒲2钱，独活1钱，丁香半钱，肉桂（去皮）2分。
出处：《东垣试效方》卷二。
主治：肾之积，发于小腹，上至心下，若豚状，或下或上无时，久不已，令人喘逆、骨痿、少气；及治男子内结七疝，女人瘕聚带下。
加减：秋、冬，加厚朴半两，通用1两2钱；如积势坚大，光服前药不减，于1料中加烧存性牡蛎3钱，(疒颓)疝、带下病勿加。
功效：None
用法用量：初服2丸，1日加1丸，2日加2丸，渐加至大便溏，再从2丸加服，食前谈盐汤送下。周而复始，病减大半勿服。
制备方法：上除巴豆霜、茯苓，另为末旋入外，为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
