序号：62601
名称：混元丹
组成：紫河车20两，竺黄10两，甘草100两，白梅花30两，滑石600两，丹皮200两，甘松40两，花粉100两，莪术（炙）30两，砂仁30两，益智仁60两，人参（去芦）10两，木香10两，黄耆10两，山药20两，香附（炙）100两，桔梗10两，茯苓50两，远志（炙，去心）25两。
出处：《北京市中药成方选集》。
主治：小儿先天不足，脾胃虚弱，慢惊抽搐、久泄不止。
加减：None
功效：理气健脾，利湿止泻。
用法用量：每服1丸，日3次，温开水送下。
制备方法：上为细粉，炼蜜为丸，每丸重1钱，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
