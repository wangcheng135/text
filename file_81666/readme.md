序号：81682
名称：凝水石酒
组成：凝水石1两，白石英1两，白石脂1两，代赭石1两，矾石1两，礜石1两，石膏1两，芒消1两，石南1两，石韦1两，天雄（炮裂，去皮脐）1两，附子（炮裂，去皮脐）1两，常山1两，续断1两，芫花1两，白术1两，防风（去叉）1两，黄芩（去黑心）1两，黄连（去须）1两，大黄（炒）1两，麻黄（去根节）1两，熟干地黄1两，山茱萸1两，杏仁（汤浸，去皮尖双仁）1两，玄参1两，（艹闾）茹1两，狼毒1两，半夏（汤洗7遍，焙）1两，藜芦1两，菖蒲1两，前胡（去芦头）1两，蜈蚣（炒）1两，甘草（炙）1两，龙胆1两，桔梗（锉，炒）1两，菟丝子（酒浸1宿，焙）1两，秦艽（去苗土）1两，芍药1两，紫菀（去苗）1两，白芷1两，远志（去心）1两，卷柏1两。
出处：《圣济总录》卷九。
主治：八风，十二痹，偏枯不随，宿食虚冷，五劳七伤。妇人产后余疾，月水不调。
加减：None
功效：None
用法用量：每服半鸡子壳，日3次。并晒囊中药滓。更捣筛，每服方寸匕，酒送下。以体暖为度。
制备方法：上药锉如麻豆，盛以绢袋，用水3斗，7月7日曲3斤，黍米3升作饭，依和酒法，以药袋著酿中，春、秋7日，冬10日，夏3日，酒成。
