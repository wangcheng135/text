序号：58866
名称：秘治跌打损伤膏
组成：全当归1两，川乌1两，白及1两，防风1两，木鳖子1两，生地黄1两，连翘1两，草乌1两，官桂1两，乌药1两，白芷1两，角刺1两，大黄1两，赤芍药1两，头发1两，白蔹1两，没药（去油）5钱，槐枝10寸，柳枝10寸，桑枝10寸，枣枝10寸，桃枝10寸。
出处：《伤科汇纂》卷八。
主治：跌打损伤；兼治疯痛闪挫，凝血闷气。
加减：None
功效：None
用法用量：临贴时加麝香半分，每张重5钱。病重者1张全愈，轻者1张可贴3-4人。
制备方法：上药入麻油2斤浸透，桑柴火熬枯去滓，复入净锅内，微火熬至滴水成珠，始下飞丹12两足，务使老嫩合宜，将凝，投去油乳香5钱化尽，倾入水盆内揉扯，以拔火毒，收用摊贴。
临床应用：None
各家论述：None
用药禁忌：此方不可加减，致取不效。
附注：None
