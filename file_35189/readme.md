序号：35199
名称：乳香应丸
组成：苍术半斤，川芎（炮）1两，地骨半两（去土），芍药半两，五灵脂2两（炒，研），自然铜（火煅）半两，葫芦巴半两，破故纸（炒）半两，甜瓜子（炒）半两，白姜半两，白附子半两，川椒（炒去汗）半两，茴香（炒）半两，两头尖半两，当归半两，牛膝半两（酒浸），乳香3钱，没药3钱，细辛3钱（去叶），血竭2钱半（研）。
出处：《普济方》卷一八五。
主治：手足麻痹筋痛。
加减：None
功效：补虚驻颜。
用法用量：每服5-7丸，空心温酒送下，干物压之，盐汤亦可。
制备方法：上为末，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
