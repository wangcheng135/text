序号：77276
名称：锉散
组成：天仙藤10斤，青蒿子（炒）10斤，桑白皮（炒）10斤，香附子（炒）10斤，荆芥穗10斤，前胡（生姜汁制，炒）10斤，柴胡10斤，桔梗10斤，麻黄（去根节）10斤，苍术（炒）10斤，干葛10斤，陈橘皮10斤，茴香（炒）2斤半，秦艽2斤半，川芎2斤半，白芍药2斤半，藁本2斤半，黄耆2斤半，半夏（为粗末，姜汁炙）2斤半，川羌活2斤半，甘草（炒）5斤，肉桂5斤，白芷（炒）5斤，厚朴（去粗皮，姜汁炒）5斤。
出处：《局方》卷二（绍兴续添方）。
主治：五劳七伤，感冷冒寒，气弱体虚，多倦少力。
加减：None
功效：壮筋骨，轻健肢体，进饮食。
用法用量：每服3大钱，水1盏半，加生姜、乌梅、大枣，煎至7分，去滓温服。并两滓作1服煎。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
