序号：67177
名称：麻黄防风汤
组成：麻黄（去根节，先煎，掠去沫，焙干）3分，防风（去叉）3分，芍药3分，防己半两，桂（去粗皮）半两，芎半两，黄芩（去黑心）半两，甘草（炙）半两，白术半两，人参半两，附子（炮裂，去皮脐）1两半，独活（去芦头）半两，竹沥（旋入）、升麻半两，石膏1两，羚羊角（镑屑）1两。
出处：《圣济总录》卷九。
主治：中风半身不遂。
加减：None
功效：None
用法用量：每用药15钱，以水4盏，加生姜1分（切），煎至2盏，去滓，入竹沥2合，更煎3沸，分3次温服。
制备方法：先将十五味（口父）咀，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
