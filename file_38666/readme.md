序号：38676
名称：泻火定痛汤
组成：栀子3钱，白芍5钱，甘草1钱，良姜3分，天花粉2钱，苍术1钱，杜仲1钱。
出处：《石室秘录》卷二。
主治：心中猝痛，手不可按者。
加减：None
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：此方妙在用栀子以清火，或疑心经之热，宜用黄连以凉之，何以不用？而反用栀子耶！盖心中火发，用黄连固宜。然黄连性燥，心火正在燥烈之时，以燥抽燥，正其所恶，不特不能去火，而转助其焰矣，不若栀子泻肝木之邪，母衰则子亦衰，不泻心火，正所以泻心火也。且栀子能泻六经之郁火，不专入肝经，亦能入心经，一味而两用之，此用药之妙，与白芍共用以泻肝，又加良姜数分，以引入于心中，复增天花粉以逐其火热之痰，则痰去火散而郁气益舒，此急治肝而正急其心也，又是急治之一法，余可推思。
用药禁忌：None
