序号：56398
名称：海桐皮散
组成：海桐皮（锉）1两，独活（去芦头）1两，五加皮（锉）1两，防风（去叉）1两，郁李仁（炒，去皮，别研如膏）1两，杜仲（去粗皮，炙，锉）1两半，枳壳（去瓤，麸炒）1两半，薏苡仁（炒）1两半，牛膝（酒浸，切，焙）1两半，虎胫骨（酥炙）1两半，恶实（炒）1两半，熟干地黄（焙干）1两半，朴消（别研）2两。
出处：《圣济总录》卷八十一。
主治：风毒流入脚膝，行履艰难，向夜筋脉痹挛疼痛。
加减：None
功效：None
用法用量：每服2钱匕，温酒调下，早、晚食后夜卧各1服。
制备方法：上药先将11味捣罗为散，次入郁李仁膏，并朴消同研匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
