序号：47363
名称：祛风追痰丸
组成：防风5钱，白附子5钱，枯矾5钱，天麻5钱，净全蝎2钱5分（炙，去毒），木香2钱5分，南星1两5钱（明矾水、皂角水各浸1半，经1宿），半夏3两（牙皂水、姜汁各浸1半，经1宿），猪牙皂（炒）5钱，僵蚕（净，炒，去丝）5钱。
出处：《幼科金针》卷上。
主治：小儿五痫，初起时未发声音。五痫者：马痫，声如马鸣，张口摇头；牛痫，目直视而腹胀；鸡痫，摇头反折，喜惊；羊痫，扬目吐舌；猪痫，喜吐沫。
加减：None
功效：None
用法用量：每服5丸，卧时薄荷汤化服。
制备方法：上为末，姜汁糊丸，如梧桐子大，辰砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
