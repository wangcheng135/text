序号：77879
名称：螽斯丸
组成：生地（酒洗）4两，熟地（酒蒸）4两，陈皮1两，白茯苓2两，川芎2两，赤芍2两，香附1斤（童便浸，春3、夏2、秋4、冬5日），当归（酒洗）4两，枳壳（麸炒）2两，黄芩（酒炒）2两，玄胡索（酒炒）2两，青皮2两，苏木1两，红花1两，五灵脂1两，干姜（炒）5钱，粉草2钱。
出处：《古今医鉴》卷十一。
主治：妇人赤白带下，经候不调，或前或后，或行时小腹作痛，腿膝麻痹，腰腹痛，子宫不能摄养。
加减：None
功效：None
用法用量：每服40-50丸，酒或白汤空心送下。
制备方法：上为末，用艾煎汤，入醋1盏，打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
