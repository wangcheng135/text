序号：79800
名称：熏黄散
组成：雄黄（研）1分，丹砂（研）1分，食盐（研）1分，青黛（研）1分，丁香1分，矾石（熬令汁枯，研）1分，铁衣1分，栀子仁1分，麝香（研）1分，莨菪子1钱，细辛（去苗叶）1钱，土瓜根1钱，干姜（炮）1钱，甜葶苈（纸上炒）1钱，菖蒲1钱，虾蟆（烧灰，研）1钱，蜀椒（去目及闭口，炒出汗）1钱，故靴底（烧灰，研）1钱，天灵盖（枯腐者）1分（炙）。
出处：《圣济总录》卷七十八。
主治：疳湿(匿虫)。下赤黑血，肛门虫蚀赤烂，日夜疼痛。
加减：None
功效：None
用法用量：每用1钱匕，以绵裹纳下部中，每日2次换药，有疮即敷其上。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
