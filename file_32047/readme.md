序号：32056
名称：万安膏
组成：川乌1两，草乌1两，归尾1两，蛤蟆1两，巴豆1两，白及1两，大黄1两，血余1两，连翘1两，蜂房1两，白蔹1两，川山甲1两，蒺藜1两，木鳖子1两，何首乌1两，槐枝1两，柳枝1两，桑枝1两，榆枝1两，楮枝1两，桃枝1两。
出处：《良朋汇集》卷三。
主治：跌打损伤，疮毒痞块，背寒肿痛。
加减：None
功效：None
用法用量：脂麻油7斤，将药入油泡，春5、夏3、秋7、冬10日；然后用火（火查）黑枯色，去渣尽，入飞过黄丹3斤，用槐条搅令烟净，滴水成珠，待温再入乳香1两，没药1两，血竭1两，麝香1钱，研末搅令匀，入水中退火毒。或绢或纸，任意摊贴。

临床应用：None
