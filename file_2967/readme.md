序号：2967
名称：天香膏
组成：白芷8钱，杏仁（去皮尖）8钱，麝香8钱，没药8钱，乳香（2味去油）8钱，白及8钱，官桂1两，白蔹1两2钱，归身1两2钱，苏木1两2钱，羌活1两2钱，陶丹20两（水飞，炒透），桃枝2两，柳枝2两，槐枝2两，桑枝2两（切），麻油3斤。
出处：《卫生鸿宝》卷二引《丛桂堂方》。
主治：瘰疬
加减：None
功效：None
用法用量：随疮大小摊用，摊时须热水坐化，不宜见火，无论已破未破，贴之即愈。
制备方法：上药除乳、没、麝、丹4种，余药入油浸，冬7夏3，春、秋5日，入锅，柴桑熬枯去渣，炼油成珠，再下黄丹，炒热，用柳条搅，武火熬，滴水成珠，渐下乳、没，又捞，将温，再下麝香成膏，贮用。熬时，极宜得法。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
