序号：30002
名称：豆蔻藿香汤
组成：藿香叶1分，桂花1分，甘松1分，陈皮（去白）5两，干姜（炮）5两，川芎2两，白芷2两，白术2两，益智1两，肉豆蔻1两，缩砂仁1两，人参1两，红豆1两半，茯苓（去皮）1两半，官桂1两半，五灵脂1两半，枇杷叶1两半，芍药1两半，苍术（净炒）半斤，甘草（炒）5两半，桔梗2两半，当归3两（焙），木香半两，厚朴（姜制）4两半。
出处：《医方类聚》卷八十九引《施圆端效方》。
主治：脾胃诸虚百损，气血劳伤，阳气久衰，下寒阴汗，中脘停痰，心腹痞闷，疼痛呕哕，减食困倦，泄泻肠滑，因病虚损，正气不复，妇人月信不匀，产后产前诸病，一切阴盛阳虚之证。
加减：None
功效：None
用法用量：每服2钱，浓煎生姜枣汤调下，食前，日进2服；或姜、枣同煎，和滓服亦妙。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
