序号：40560
名称：虎骨膏
组成：虎骨24两，川乌（生）1两，川芎1两，熟地1两，五加皮1两，桃枝1两，白术1两，续断1两，桑枝1两，槐枝1两，草乌（生）1两，白芷1两，天麻1两，何首乌（生）1两，生地1两，香附1两，青风藤1两，白蔹1两，独活1两，僵蚕1两，当归1两，细辛1两，牛膝1两，羌活1两，杜仲（生）1两，威灵仙1两，穿山甲（生）1两，苍术1两，榆枝1两，川楝子1两，柳枝1两，大风子1两，蜈蚣1钱。
出处：《北京市中药成方选集》。
主治：腰腿疼痛，筋脉拘挛，四肢麻木，行步艰难。
加减：None
功效：散风止痛，舒筋活络。
用法用量：微火化开，贴患处。
制备方法：上药酌予碎断，用香油240两，炸枯过滤去滓。炼至滴水成珠，入黄丹100两，搅匀成膏。取出浸于水中去火毒后加热溶化。另兑以下细粉：肉桂5钱，乳香5钱，公丁香5钱，血竭5钱，没药5钱，麝香3钱，搅匀摊贴。大张油重6钱，中张油重4钱5分，布光。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
