序号：5284
名称：半硫丸
组成：半夏（汤浸7次，焙干，为细末）、硫黄（明净好者，研令极细，用柳木槌子杀过）各等分。
出处：《局方》卷六。
主治：肾阳衰微，阴寒内结，命门火衰，阳气不运所致虚人、老人虚冷便秘或阳虚久泻；脾胃气弱，津液停积，湿久浊凝，痰浊咳嗽吐逆；或湿阻三焦，二便不通;心腹一切痃癖冷气，;痃癖冷气吐逆;小儿泄泻注下，或手足冷者，亦治咳嗽;湿凝气阻，三焦俱闭，二便不通。
加减：None
功效：温肾逐寒，通阳开秘，泄浊祛痰，止泻,润大肠;除积冷，暖元脏，温脾胃，进饮食。
用法用量：每服15-20丸，空心温酒或生姜汤送下；妇人醋汤送下。
制备方法：以生姜自然汁同煎，加干蒸饼末入臼内杵为丸，如梧桐子大。
临床应用：虚风便秘：吴，二气自虚，长夏大气发泄，肝风鸱张，见症类中，投剂以来诸恙皆减，所嫌旬日犹未更衣，仍是老人风秘。半硫丸1钱，开水送下，3服。
各家论述：1.《温病条辨》：湿阻无形之气，气既伤而且阻，非温补真阳不可，硫黄热而不燥，能疏利大肠，半夏能入阴。燥胜湿，辛下气，温开郁，三焦通而二便利矣。_x000D_
2.《成方便读》：此为命火衰微，胃浊不降而致，故以半夏和胃而通阴阳，硫黄益火消阴，润肠滑便，然后胃与大肠皆得复其常，所谓六腑皆以通为用也。
用药禁忌：None
附注：半桃丸（《三因》卷十二）、硫半丸（《良朋汇集》卷二）。
