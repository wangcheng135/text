序号：71662
名称：巽顺丸
组成：乌骨白丝毛鸡1只（男雌女雄，取嫩长者，溺倒，泡，去毛，竹刀剖胁，出肫肝，去秽，留内金，并去肠垢，仍入腹内），乌贼骨（童便浸，晒干为末，微炒黄，取净）4两，茹芦（去梢，酒洗，切片）1两，鲍鱼（切薄片）4两。
出处：《张氏医通》卷十三。
主治：妇人倒经，血溢于上，男子咳嗽吐血，左手关尺脉弦，背上畏寒，有瘀血者。
加减：None
功效：None
用法用量：每服50-70丸，空心百劳水送下。
制备方法：上3味入鸡腹内，用陈酒、童便各2碗，水数碗，砂锅中旋煮旋添，糜烂汁尽，捣烂熔干，骨用酥炙，共为细末，干山药末调糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：乌鸡丸（《类证治裁》卷八）。
