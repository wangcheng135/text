序号：71119
名称：新制阴阳攻积丸
组成：吴茱萸（泡）1两，干姜（炒）1两，官桂（去皮）1两，川乌（炮）1两，黄连（炒）8分，半夏（洗）8分，橘红8分，茯苓8分，槟榔8分，厚朴（炒）8分，枳实（炒）8分，菖蒲（忌铁）8分，玄胡索（炒）8分，人参（去芦）8分，沉香8分，琥珀（另研）8分，桔梗8分，巴霜（另研）5钱。
出处：《医宗必读》卷七。
主治：五积六聚，七癥八瘕，痃癖虫积，痰食，不问阴阳。
加减：None
功效：None
用法用量：每服8分，渐加1钱5分，生姜汤送下。
制备方法：上为细末，皂角6两，煎汁泛为丸，如绿豆大。
临床应用：None
各家论述：《医略六书》：虫、血、痰、食留滞，阴阳积结于中，皆能成积聚癥瘕痃癖，故胀满疼痛不已。吴萸温中散寒以开癥瘕，黄连清热燥湿以消痞结；厚朴散积聚之满，枳实消心下之痞；桔梗开提气血，槟榔化滞攻坚；干姜暖胃祛寒，官桂暖血散积；半夏燥湿痰，橘红利气痰；人参扶元气以助药力，菖蒲通窍门以开结气；琥珀散瘀血消积，沉香降逆气散聚；延胡活血滞，川乌透经络；茯苓渗湿和脾，巴霜攻坚荡实。皂角汁丸姜汤下，乃开其痰以散痃癖聚积也。使肠胃迅扫一空，则经络之积气自散，而胸中阳气敷布、阴霾顿灭，何患诸积不消，痛胀不去耶？此攻补热寒之剂，为夹攻阴阳诸积总方。
用药禁忌：None
附注：攻积丸（《杂病源流犀烛》卷十四）、阴阳攻积丸（《类证治裁》卷三）。《张氏医通》引本方无橘红、槟榔、厚朴、枳实、菖蒲、桔梗，名“乔氏阴阳攻积丸”。
