序号：3316
名称：天雄散
组成：天雄3分，防风3分，芎3分，人参3分，独活3分，桂心3分，葛根3分，白术6分，远志6分，薯蓣6分，茯神6分，山茱萸6分，莽草4分。
出处：《千金》卷十四引徐嗣伯方。
主治：头目眩晕，屋转旋倒，或三两日却明，发动无定，久成青盲。
加减：None
功效：None
用法用量：每服方寸匕，先食以菊花酒送下，1日2次。渐加至3匕，以知为度。
制备方法：上药治下筛。
临床应用：None
各家论述：《千金方衍义》：真元下虚，风毒上盛，而致头目眩晕，屋转旋倒。故用人参、茯神、薯蓣、白术、山萸、桂心、芎、远志填补脾肾，莽草、天雄、独活、防风、葛根专祛风毒也。
用药禁忌：None
附注：远志散(《圣惠》卷二十二)。菊花酒法：九月九日取邓州甘菊花晒干，作末，以米饙中蒸作酒。
