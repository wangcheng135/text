序号：81304
名称：鳖甲导经丸
组成：鳖甲、煎膏4两，当归3两，川芎1两，赤芍2两，白芍4两，生地4两，琥珀屑1两，熟地6两，麦冬3两，白术4两，茯神2两，枣仁3两，丹皮3两，阿胶2两，白薇1两，玉竹2两，红花1两5钱。
出处：《陈素庵妇科补解》卷一。
主治：室女年过二十，天癸闭而不通，心、肝、脾三经虚极，肌热，五心烦热，齿垢耳黑，目陷面黄，胸膈满，妨于食，或寒热往来，或梦与鬼交，或善怒，时时干咳。
加减：None
功效：清心和肝，补脾开胃。
用法用量：每服3钱，白汤入醋少许，空心送下。
制备方法：上为末，用龙眼肉并莲子（去心）煮烂，同上药末共捣如泥，合炼蜜为丸。
临床应用：None
各家论述：是方四物补血，合生、红、赤芍通经行血，胶、麦、玉、丹、阿胶、鳖甲滋阴退火，神、枣安心定神，琥珀则镇怯，白术补脾开胃。皆心肝脾三经要药，非干漆、大黄、水蛭、虻虫猛属伤营也。
用药禁忌：None
附注：None
