序号：3977
名称：无极丸
组成：锦纹大黄1斤。
出处：《本草纲目》卷十七引《医林集要》。
主治：妇人经血不通，赤白带下，崩漏不止，肠风下血，五淋，产后积血、癥瘕腹痛；男子五劳七伤；小儿骨蒸潮热。
加减：None
功效：None
用法用量：每服50丸，空心温酒送下。取下恶物为验、未下再服。
制备方法：上分作4分：一分用童便1碗，食盐2钱，浸1日，切晒；一分用醇酒1碗，浸1日，切晒，再以巴豆仁35粒同炒豆黄，去豆不用；一分用红花4两，泡水1碗，浸1日，切晒；一分用当归4两，入淡醋1碗，同浸1日，去归切晒；为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
