序号：81576
名称：镇癞宁心丸
组成：节菖蒲5钱，陈皮5钱，枣仁（炒）5钱，黄芩8钱，礞石（煅）4钱，乳香（炙）4钱，没药（炙）4钱，白术（炒）4钱，蒌仁（炒）4钱，生地4钱，白附子（炙）4钱，当归4钱，牙皂3钱，法半夏3钱，南星（炙）3钱，远志（炙）3钱，天麻3钱，僵蚕（炒）3钱，黄连3钱，白芍3钱，甘草3钱，茯苓4钱，人参（去芦）3钱。
出处：《北京市中药成方选集》。
主治：神经错乱，癫痫疯狂，痰迷心窍，烦躁不安。
加减：None
功效：镇惊，豁痰，安神。
用法用量：每服1-2丸，温开水送下，1日2次。
制备方法：共研细粉，过罗，加朱砂1钱，犀角1钱，沉香3钱，麝香5钱，研和令匀，炼蜜为丸，重5分，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：忌气恼忧思。
附注：None
