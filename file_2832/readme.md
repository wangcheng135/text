序号：2832
名称：天竺黄丸
组成：天竺黄（细研）1分，黄连（去须）1分，柴胡（去苗）1分，羚羊角屑1分，蔓荆子1分，犀角屑1分，防风（去芦头）1分，子芩1分，川升麻1分，麦门冬（去心，焙）1分，甘草（炙微赤，锉）1分，玄参1分，白蒺蒺（微炒，去刺）1分，朱砂（细研）1分，木香1分，龙脑（细研）1钱，麝香（细研）1钱，牛黄（细研）1钱。
出处：《圣惠》卷八十三。
主治：小儿惊悸壮热，黄瘦，不思乳食。
加减：None
功效：None
用法用量：每服5丸，以温水化下。
制备方法：上为末，与研了药，都研令匀，炼蜜为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
