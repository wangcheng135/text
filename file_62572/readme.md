制备方法：上药各为粗片，用真麻油12斤浸药在内，夏浸3日，冬浸半月，煎至黑枯色为度，麻布滤去滓，将油再秤，如12斤，加飞过黄丹5斤，如8斤加4斤，依数秤起，将油再下锅熬，黄丹徐徐投下，用槐柳棍不住手搅，火先文后武，熬至滴水成珠为度，春夏硬些，秋冬软些，外加乳香、没药、龙骨、轻粉各3两，研极细末，贮磁器内，临用时加入。
