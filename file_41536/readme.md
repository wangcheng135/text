序号：41551
名称：金雀万应膏
组成：金雀花根4两，生地2两，红花2两，丹皮2两5钱，银花2两，川乌2两，草乌2两，五加皮2两5钱，防风1两，荆芥1两，桃仁2两半，苏木2两，牛蒡2两（研），甘松1两，归尾1两，三棱1两，莪术1两，赤芍1两，续断1两，骨碎补1两，潮脑半斤，狗胆1个，自然铜1两（煅，醋淬7次，研），古钱6个（煅，醋淬7次，研），松香15两（研细，筛净，用姜汁，葱汁，白凤仙汁各1盏，研，炼）。
出处：《跌打损伤方》。
主治：跌打损伤。
加减：风湿诸般疼痛，宜加山甲，地龙各1两5钱，川椒2两。
功效：接骨。
用法用量：摊膏时再加入肉桂2分，麝香3厘，方有力量。
制备方法：以铜锅先入麻油5斤，次入群药熬膏枯去滓，入狗胆再熬，滴水成珠；入飞过黄丹半斤，次入松香，再入潮脑半斤，血竭1两，离火再入阿魏2钱，乳香、没药各2两，再入自然铜，古铜钱末，再入肉桂末1两，麝香2钱，再入淘鹅油，不住手搅匀，以烟尽为度，备用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
