序号：56128
名称：桔梗汤
组成：桔梗（炒）1钱，贝母（去心）1钱，当归（酒浸）1钱，瓜蒌仁1钱，枳壳（麸炒）1钱，薏苡仁1钱，桑白皮（炒）1钱，甘草节1钱，防己（去皮）1钱，黄耆（盐水拌炒）1钱5分，五味子（捣，炒）1钱5分，百合（蒸）1钱5分，葶苈（炒）5分，地骨皮5分，知母（炒）5分，杏仁5分。
出处：《外科枢要》卷四。
主治：肺痈咳嗽，胸膈两胁作痛，咽干口燥，烦闷作渴，时出臭浊。
加减：None
功效：None
用法用量：加生姜，水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：宁肺桔梗汤（《外科正宗》卷二）、十六味桔梗汤（《张氏医通》卷十六）。
