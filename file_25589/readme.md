序号：25597
名称：朱砂丸
组成：朱砂1分（细研），麝香1分（细研），恒山（锉）半两，川升麻半两，苦参半两（锉），猢狲脑盖半两（炙微黄），鳖甲半两（涂醋炙黄，去裙襕），乌梅肉半两（微炒），柴胡半两（去苗），虎头骨半两（涂酥炙黄），乌猫粪（烧灰），驴轴垢半两，木香半两，白术半两，川大黄半两（锉碎，微炒），地骨皮半两，巴豆3分（去皮心，熬令黄，别研如膏，纸裹，压去油）。
出处：《圣惠》卷五十二。
主治：疟，寒热发作不定，体瘦不能食。
加减：None
功效：None
用法用量：空心服4丸，以温水送下。渐吃冷茶半茶碗，良久，吐出痰涎，后更煎热茶下2丸，得吐利便效。
制备方法：上为末，入研了药令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
