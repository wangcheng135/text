序号：62712
名称：清心明目羊肝丸
组成：熟军32两，菊花32两，琥珀32两，生石决明32两，泽泻32两，白蒺藜（炒）32两，夜明砂32两，胆草32两，车前子（炒）32两，蝉退32两，芒消32两，川芎4两，桑叶4两，薄荷4两，防风4两，当归4两，黄芩4两，木贼4两，茯苓4两，蒙花4两，黄柏4两，知母4两，熟地4两，枸杞子4两，甘草4两，黄连10两，人参（去芦）10两，鲜羊肝320两（煮熟连汤制）。
出处：《北京市中药成方选集》。
主治：肝虚火盛，两目昏暗，羞明怕光，迎风流泪，夜盲内障。
加减：None
功效：清热散风，明目止痛。
用法用量：每服2钱，1日2次，温开水送下。
制备方法：上为粗末，将煮熟羊肝串入，晒干或烘干，为细粉，用芒消化水，泛为小丸，每16两用滑石细粉4两为衣，闯亮。
临床应用：None
各家论述：None
用药禁忌：忌服辛辣食物。
附注：None
