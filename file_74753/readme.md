序号：74769
名称：解毒承气汤
组成：银花3钱，生山栀3钱，小川连1钱，生川柏1钱，青连翘3钱，青子芩2钱，小枳实2钱，生锦纹3钱，西瓜消5分，金汁1两（冲），白头蚯蚓2支。
出处：《重订通俗伤寒论》。
主治：疫毒。
加减：如神昏不语，人如尸厥，加局方紫雪，消解毒火，以清神识。
功效：峻下三焦毒火。
用法用量：先用雪水6碗，煮生绿豆2两，滚取清汁，代水煎药。
制备方法：None
临床应用：None
各家论述：此方用银、翘、栀、芩轻清宣上以解疫毒；黄连合枳实，善疏中焦，苦泄解毒，即所谓疏而逐之也；黄柏、大黄、瓜消、金汁咸苦达下，速攻其毒，即所谓决而逐之也；雪水；绿豆清解火毒之良品。合而为泻火逐毒，三焦通治之良方。
用药禁忌：None
附注：None
