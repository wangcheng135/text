序号：4310
名称：九制硫黄丸
组成：硫黄。
出处：《内外科百病验方大全》。
主治：耳聋眼花，齿落发白，阳痿。
加减：None
功效：补先天本元，健脾胃，壮筋骨。
用法用量：每早用盐汤送服：1日用3分，第2日用4分，3日用6分，4日8分，5日1钱，每日递加，至2钱为度。如恐服久发毒，病愈则止，自无妨碍。
制备方法：用老白豆腐，每硫黄1斤，豆腐1斤（或黑豆煮亦可），将硫黄研末，用砂净锅，以竹篱夹锅底，篱上盖豆腐1层，铺硫黄1层，迭迭铺好，入水煮至豆腐黑黄为度，用清水漂净腐渣，再煮3次；2制用大生萝卜挖空，1硫2卜，将硫黄末入内盖紧，缚好，慢火煮至萝卜黄黑烂为度，清水漂净，复煮2次，或萝卜切片拌亦可；3制将紫背鲜浮萍洗净，1硫3萍，拌硫黄末，煮至硫、萍烂为度，但煮根须叶最多，清水漂净，或打烂取汁，拌煮亦可；4制用新绿豆拣淘洗净，1硫2豆，以取末拌，煮至豆烂为度，清水漂净；5制用石菖蒲或菖蒲，洗净，切小段，拌硫黄末，入水煮烂为度，取汁拌煮更妙；6制松柏叶各半，洗净，去枝用叶，剪碎拌硫黄，煮至叶烂为度，清水漂净：七制或藕或梨，或藕、梨各半，切片，同硫黄煮至藕、梨烂为度；8制肥壮猪大肠，洗净气味，将硫黄末研细漂净，装入大肠，两头扎紧，勿令走漏，煮至大肠熟烂为度，用清水漂过夜，澄出阴干；9制地黄2两，全归1两，天冬1两，麦冬1两，川芎5钱，陈皮5钱，枸杞5钱，杜仲5钱，茯苓5钱，炙草5钱，前胡5钱，防风5钱，泽泻5钱，蛇床子5钱，五加皮5钱，每硫黄1斤，用药1料，照硫黄递加，用清水煎浓，将硫末投入，煎至药汁干，起出阴干，用糯米煮粥，拌为丸，如绿豆大，阴干，用瓷瓶收贮。
临床应用：None
各家论述：None
用药禁忌：忌一切牲畜血及细辛。
附注：None
