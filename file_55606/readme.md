序号：55621
名称：桃奴散
组成：桃奴（12月收用）、玄胡索、豭鼠粪、香附子、官桂、五灵脂、砂仁、桃仁（去皮尖）各等分。
出处：《医学正传》卷三。
主治：妇人或室女月经不通，渐成胀满；及男子坠马，跌仆损伤，以致瘀血停积，成血蛊病。
加减：None
功效：None
用法用量：每服3钱，温酒调下。
制备方法：上为末。
临床应用：None
各家论述：《医略六书》：血瘀肝脾，不能鼓运气化，而成血臌，男女皆有之，惟经闭为女科所独焉。桃奴抑心气以生血，延胡化滞血以通经，桃仁破瘀血，灵脂降浊阴；鼠屎通幽降浊，香附解郁调经，肉桂温经以运乎经血，砂仁醒脾以鼓运乎经气也。为散以消之，酒煎以行之，使瘀结顿化，则经脉自通，而经闭无不行，胀满无不退矣。
用药禁忌：None
附注：本方原名“桃奴丸”，与剂型不符，据《医学入门》改。
