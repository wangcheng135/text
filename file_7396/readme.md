序号：7396
名称：圣散子
组成：御米壳5两（捣碎，醋炙黄色），甘草（炙黄）2两，赤石脂2两，乌鱼骨（去皮）2两，肉豆蔻（面包煨去面）2两，拣丁香2两，诃子皮2两，干姜（炮）2两。
出处：《普济方》卷二○九。
主治：男妇脾胃受湿，中脘停寒，吃物频伤，心胸满闷，胁肋膨胀，肠鸣虚痞，小腹坚痛，脐下强急；或大便不调，水谷迟化，里急后重，下痢脓血，或五色，或便如鱼脑，或如豆汁，或有鲜血，或如烂肉，日夜无度，久不愈，嗜卧怠惰，虚瘦，肢体沉重，寒热时作。
加减：None
功效：固养脾胃，温中止腹痛。
用法用量：每服先用水1盏，入乳香少许，煎7分，调药末2钱，食前和滓热服。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
