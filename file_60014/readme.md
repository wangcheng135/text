序号：60029
名称：资生健脾膏
组成：党参2两，于术1两5钱（炒），广砂仁1两（小粒，研），木香1两（研），茯苓2两（研），陈皮1两2钱，柏子仁1两5钱（炒），三仙4两（炒黄），山药1两，紫姜朴1两，小枳实1两2钱（炒，研）炙甘草5钱。
出处：《慈禧光绪医方选议》。
主治：脾胃病。
加减：None
功效：None
用法用量：每用4钱，白水冲服。
制备方法：上药以水熬透，滤去滓，再熬浓，加炼蜜为膏，瓷罐收盛。
临床应用：None
各家论述：本方为资生丸加减而得。方以参、术、苓、草、山药甘平补脾元，砂仁、陈皮、紫朴、三仙、枳实辛香调胃气，又以柏子仁润而通之，能补能运，为至和补养之良方。
用药禁忌：None
附注：None
