序号：4867
名称：木香散
组成：木香3分，人参半两（去芦头），赤茯苓3分，白术1两，陈橘皮1两（汤浸，去白瓤，焙），桂心半两，槟榔半两，草豆蔻半两（去皮），丁香1分，厚朴1两（去粗皮，涂生姜汁，炙令香熟），半夏半两（汤洗7遍，去滑），诃黎勒皮3分，甘草半两（麸炒微黄），附子半两（炮裂，去皮脐），枳实半两（麸炒微黄）。
出处：《圣惠》卷十三。
主治：伤寒后脾胃不和，腹胁气滞，痰逆，不纳饮食，四肢乏力。
加减：None
功效：None
用法用量：每服5钱，以水1中盏，加生姜半分，大枣2枚，煎至6分，去滓，不拘时候稍热服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
