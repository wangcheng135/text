名称：丹砂丸
组成：丹砂（研）1分，腻粉（研）1分，蛇蜕（炙）1分，兔头灰（研）1分，铜青（研）1分，硇砂（水少许浸，去石）1分，古字钱3文（烧赤投于硇砂水中，淬至水尽，捣），白矾（熬令汁枯，研）半两，龙骨（研）半两，老鸦灰（研）半两，盐花（研）半两，铅丹（研）半两，虎睛（炙，捣）1个，虎牙（炙，捣）1对，发灰（研）半分，金箔（研）5片，银箔（研）5片。
出处：《圣济总录》卷十五。
主治：五种风痫。
加减：None
功效：None
用法用量：每服2丸，发时及晚间温酒嚼下。
制备方法：上药，将4味为末，与别研13味和匀再罗，用猪血为丸，如樱桃大。
