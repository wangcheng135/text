序号：23339
名称：防风羌活汤
组成：防风6分，羌活6分，枳壳6分，桔梗6分，川芎6分，白芍（酒炒）1钱，甘草（炙）4分，白茯苓7分，陈皮5分，半夏5分（汤洗7次），白术5分，荆芥5分。
出处：《保命歌括》卷一。
主治：真中风初起，其邪在表，气虚血虚挟痰，无汗而拘急者。
加减：药后无汗者，加麻黄（去节）1钱，葱白3茎；头痛，加白芷、细辛各5分；血虚无汗，加生地黄，当归各5分；气虚有汗，加黄耆、人参各5分；口干有热，加柴胡、葛根、黄芩各5分；四肢恶寒，加桂枝1钱；风痰，加胆星1钱；胸中多痰、满闷，加竹沥、姜汁；搐掣，加白天麻、僵蚕（炒，去丝嘴）各8分。
功效：None
用法用量：加生姜，水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
