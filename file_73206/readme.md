序号：73222
名称：斑蝥酒
组成：斑蝥7个（去头翅足。若过1日，则加1个），番木鳖2个，糯米1合或1撮。
出处：《医林纂要》卷十。
主治：瘈犬伤甚重者，腹痛狂呼，见人欲啮，二便秘塞，时作犬声，亦切恶风，恶闻金鼓声，闻则腹中切痛，狂啮。
加减：None
功效：None
用法用量：上合炒至糯米透脆，去斑蝥、木鳖，细研糯米，酒调服；或合斑蝥、木鳖、糯米炒研，加辰砂1钱，酒送下；或木鳖3个（陈壁土炒），斑蝥7个（糯米炒），加大黄、茯苓、麝，研末，酒送下，俱可。受伤多日，斑蝥加至21个，又其头上必有红发3茎，急拔去之。毒自小便或大便而出，日浅只成血水，日深结血块，真如犬形，毒尽，腹痛止，二便通，更服黄连甘草汤以解余热。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：三月忌闻金鼓声及色欲、诸发毒物，终生忌羊、犬肉。
附注：None
