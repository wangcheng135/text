序号：4305
名称：九制牛膝葆真丸
组成：牛膝（去头尾）3斤（分作9分听制。1分用补骨脂1两，、巴戟肉1两，黄酒3斤，煎至斤半，将汁泡牛膝拌透晒干，俟汁尽为度，其2味渣不用；1分用川椒1两，狗脊1两，亦用酒煎，如上法；1分用肉苁蓉3两，洗净去甲，制如上法；1分用蛇床子、覆盆子各1两5钱，制如上法；1分用紫梢花、天门冬各4两，制如上法；1分用五加皮、菟丝子各1两，制如上法；1分用熟地、五味各1两，制如上法；1分用天雄2枚切片，用童便10碗，煎汁2碗，拌晒，制如上法；1分用小茴香1两，煎汁1碗，鹿茸血尖1两，研末，入小茴香汁，拌牛膝晒干）。
出处：《惠直堂方》卷一。
主治：None
加减：None
功效：补精神，强腰膝。
用法用量：清晨好酒送下3钱。少壮者只服6两，衰弱者8两，老迈者亦不过10两，诸症皆愈，精神强健。
制备方法：上为末，用核桃肉4两，去皮捣烂，加炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
