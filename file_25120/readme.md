序号：25128
名称：当归汤
组成：当归（切，炒）1两，红蓝花1两，延胡索1两，紫葳1两，琥珀半两（研），牡丹皮3两，姜黄3两，牡（？？）3两，鬼箭羽3两，麒麟竭1两，桃仁（去皮尖双仁，麸炒）1两，菴闾子1两，藕节1两（切，焙），没药（研）1两，桂（去粗皮）1两。
出处：《圣济总录》卷一五一。
主治：妇人经水不通，腰腹刺痛，拘倦少力，呕吐恶心，怠惰多睡，头旋眼涩，日渐羸瘦，饮食减少。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，酒半盏，同煎至7分，去滓，空心、食前温服。如无牡蒙亦得。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
