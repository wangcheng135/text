序号：3158
名称：天麻琥珀丸
组成：煨天麻2两，琥珀2两，乌药1两，茯神3两，肉桂5两，黄柏5钱，防己5钱，秦艽1两，煅牡蛎3两，豨莶草2两，钩藤1两，柴胡8钱，广郁金1两，怀牛膝2两。
出处：《医方简义》卷三。
主治：厥证，风动阳升，冲气上逆，足冷而厥。头眩头昏，肩背痠痛，四肢麻木，手足厥冷，心腹串痛，腰膝无力，逆气上冲，肌肉刺痛，中风中寒，中湿中气。
加减：None
功效：祛风降逆。
用法用量：每丸2钱，金箔为衣，白蜡封固。每服1丸，去蜡，水化服，加酒少许。
制备方法：上为细末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：忌食葱、蒜。
附注：本方加麝香一钱，蜡丸更妙。
