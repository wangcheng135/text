序号：78859
名称：鼻炎灵
组成：苍耳子（捣）60g，白芷60g，辛夷60g，冰片粉6g，薄荷霜5g，芝麻油500ml，液状石腊1000ml。
出处：《古今名方》。
主治：慢性鼻炎、萎缩性鼻炎、过敏性鼻炎、鼻息肉等。症见鼻粘膜充血，或干燥萎缩，鼻塞流涕，嗅觉失灵等。
加减：None
功效：疏风祛湿，芳香透窍，清热消肿，化瘀止痛，滋润粘膜，收缩息肉。
用法用量：用时仰头滴鼻，每次滴1～2滴，日滴1-2次。
制备方法：将前3味与芝麻油同放锅内，浸泡24小时，加热，待炸成黑黄色捞出，再下余3味药，搅匀，冷却后过滤，分装眼药水瓶内。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
