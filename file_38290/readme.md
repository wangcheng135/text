序号：38300
名称：枣仁汤
组成：枣核仁2合，人参2两，芍药1两，桂心1两，黄耆2两，甘草2两，茯苓2两，白龙骨2两，牡蛎2两，生姜2斤，半夏1升，泽泻1两。
出处：《千金》卷十九。
主治：虚劳，梦泄精，茎核微弱，血气枯竭，或醉饱伤于房室，惊惕松悸，小腹里急。便溺白浊，形体枯瘦，腰脚疼重。
加减：若不能食，小腹急，加桂心6两。
功效：None
用法用量：上(口父)咀。以水9升，煮取4升，每服7合，1日3次。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：肾司闭藏，肝司疏泄，今以过劳伤肝，不能司闭藏之令，故首取枣仁通肝气而愈惊，龙骨、牡蛎秘精气而固肾脱，参、耆、甘草保元气而补大虚，茯苓、泽泻利水道而除水气，生姜、半夏逐滞气而通清阳，桂心、芍药和荣气而除里急也。
用药禁忌：None
附注：人参汤（《圣济总录》卷九十一）。
