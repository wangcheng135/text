序号：19700
名称：壮脾丸
组成：丁香1两，附子（重6钱以上者，炮，去皮脐）1两，诃子肉1两，荜茇1两，白术1两，白茯苓（去皮）1两，肉豆蔻（面裹煨）1两，人参（去芦头）7钱，干姜（炮）7钱，荜澄茄7钱，乌药7钱，陈橘皮（去白，焙）7钱，沉香7钱，厚朴（去粗皮，细切，以生姜1两研烂，淹半日，炒干用）7钱，神曲（炒）7钱，熟艾6钱（研，糯米稀糊拌匀、炒干，乘热入碾末之），缩砂仁半两，甘草6钱（炙）。
出处：《杨氏家藏方》卷六。
主治：脾胃久弱，中焦停饮，腹内虚鸣，或多泄利，心腹胀满，饮食不入，精神怠惰，睡卧不安。
加减：None
功效：None
用法用量：每服50丸，空心、食前米饮吞下。
制备方法：上为细末，煮枣肉为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
