序号：46518
名称：千金不换刀圭散
组成：川乌2两，草乌（并用火炮，去皮尖）2两，苍术（米泔浸）2两，人参1钱半，白茯苓（去皮）1钱半，两头尖1钱，甘草（炙）1两半，僵蚕（隔纸炒）3钱半，白花蛇（酒浸3日，弃酒，火炙，去皮骨）5钱，石斛（酒洗）5钱，川芎2钱半，白芷2钱半，细辛2钱半，当归（酒洗）2钱半，防风（去芦）2钱半，麻黄2钱半，荆介2钱半，全蝎（瓦上焙干）2钱半，何首乌（米泔浸，忌铁器）2钱半，天麻2钱半，藁本2钱半。
出处：《鲁府禁方》卷一。
主治：男妇小儿，诸般风症，左瘫右痪，半身不遂，口眼歪斜，腰腿疼痛，手足顽麻，言语謇涩，行步艰难，遍身疮癣，上攻头目，耳内蝉鸣，痰涎不利，皮肤瘙痒，偏正头风，无问新旧；及破伤风，角弓反张，蛇犬咬伤，_x000D_
金刃所伤，出血不止；痔漏脓血，疼痛难禁。
加减：None
功效：None
用法用量：每服2分或5分，渐加至6-7分，临卧酒调下；不饮酒者，茶亦可。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：忌多饮酒并一切热物饮食，一时恐动药力。
附注：None
