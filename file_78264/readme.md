序号：78280
名称：蜜桃酥
组成：当归1两，川芎1两，白芍1两，生地黄1两，人参（去芦）3钱，白茯苓（去皮）3钱，白术（去芦）1两，陈皮1两，半夏（姜炒）1两，厚朴（姜炒）1两，苍术（用米泔浸2日）1两，香附1两，枳壳（去瓤）1两，乌药5钱，砂仁5钱，杏仁（去皮尖）5钱，木香5钱，沉香5钱，天门冬（去心）5钱，麦门冬（去心）5钱，五味子5钱，破故纸5钱，小茴5钱，牛膝（去芦）5钱，枸杞子5钱，川椒5钱，何首乌5钱，肉苁蓉5钱，川乌（炮，去皮尖）5钱，草乌（炮，去皮尖）5钱，细辛1两，白芷1两，麻黄1两，防风1两，羌活1两，独活1两，干姜1两，官桂1两，甘草1两，五加皮5钱，小红枣8两，北蜜8两，真酥油8两，胡桃肉8两（泡，去皮）。
出处：《回春》卷二。
主治：久患风寒湿痹，左瘫右痪。
加减：None
功效：None
用法用量：每服3盏，空心服，日3次。其药滓晒干为末，本酒打糊为丸，如梧桐子大。每服30丸，空心本酒送下。
制备方法：上锉。用生绢袋盛之，用好酒1大金华坛浸药3日，封固放锅内，悬胎煮3个时辰，取出埋土中3日，出火毒。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
