序号：64162
名称：猪肾子饮
组成：猪腰子1对（切作4片），当归1两，白芍药1两。
出处：《医学正传》卷七引《产宝》。
主治：妇女产后蓐劳，寒热如疟，咳嗽头疼，自汗体瘦，腹中(疒丂)痛。产后气血虚弱，饮食未复，虚乏倦怠，乍卧乍起，颜色憔淬，口干头昏，百节疼痛，时有盗汗，背膊烦闷。
加减：None
功效：None
用法用量：上以当归、芍药2味细切，用水3碗，煮至2碗，去滓；将腰子切碎如骰子状，入前药汁内，用晚粳米1合，香豉1两，葱白5-7根同煮糜烂，空腹食之，每日1服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：猪腰饮（《医统》卷八十五）、猪腰汤（《景岳全书》卷六十一）、猪肾汤（《叶氏女科》卷三）、猪腰粥（《卫生鸿宝》卷五）。
