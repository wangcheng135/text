序号：79002
名称：增损普济消毒饮
组成：元参3钱，黄连2钱，黄芩3钱，连翘（去心）2钱，栀子（酒炒）2钱，牛蒡子（炒，研）2钱，兰根（如无，以青黛代之）2钱，桔梗2钱，陈皮1钱，甘草（生）1钱，全蝉蜕12个，白僵蚕（酒炒）3钱，大黄（酒浸）3钱。
出处：《寒温条辨》卷四。
主治：大头瘟。初觉憎寒，壮热体重，次传头面肿甚，目不能开，上喘，咽喉不利，口燥舌干。
加减：None
功效：None
用法用量：水煎去滓，入蜜、酒、童便冷服。
制备方法：None
临床应用：None
各家论述：芩、连泻心肺之热为君；元参、陈皮、甘草泻火补气为臣；翘、栀、蒡、兰、蚕、蜕消肿解毒定喘为佐；大黄泻热斩关，推陈致新为使；桔梗为舟揖载药上浮，以开下行之路也。
用药禁忌：None
附注：None
