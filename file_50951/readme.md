序号：50966
名称：大生丸
组成：熟地黄（酒蒸）4两，当归身4两，续断（盐水炒）2两，阿胶（蒲黄末炒珠）2两，杜仲（盐水炒）2两，丹参（炒）2两，黄耆（蜜炙）1两5钱，白芍（酒炒）1两5钱，延胡索（炒）1两5钱，川芎1两5钱，广皮5钱，香附（四制者）1两。
出处：《竹林女科》卷四。
主治：经水不调，久不受孕。
加减：若先期色紫，改为煎剂，1两改作1钱，加黄芩8分，生姜3片，水煎，空心服，临卧再服；若后期色淡，加肉桂、熟艾、干姜各5分，生姜3片，水煎服；若经未至而腹痛，则用丹参1两为末，黄酒下2钱，俱以经尽为止。
功效：调经。
用法用量：每服3钱，空心白汤送下。行经时加2钱。
制备方法：上为末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
