序号：46325
名称：活血散
组成：乳香5钱，没药5钱，血竭（生）5钱，贝母3钱，羌活5钱，南木香2钱，厚朴3钱，川乌（制）1钱，草乌（制）1钱，白芷（生）8钱，麝香5分，紫荆皮（生）8钱，生香附5钱，炒小茴3钱，甲珠5钱，煅自然铜5钱，独活5钱，续断5钱，虎骨5钱，川芎5钱，木瓜5钱，上安桂（去皮）3钱，当归（酒洗）8钱。
出处：《中医正骨经验概述》。
主治：久伤不愈，经血不和，创伤出血，伤后肿胀，疼痛瘀血。
加减：None
功效：止血舒筋，活血散瘀，理气镇痛。
用法用量：开水冲调呈糊状，外敷，视伤状而酌量，一般扭挫伤用2-5钱。内服：药末5钱，配白酒1斤备用。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：上焦有热，出现口干舌燥者不宜用。
附注：None
