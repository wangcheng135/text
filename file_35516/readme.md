序号：35526
名称：使君子丸
组成：使君子（去壳）2两，南星（姜制）、槟榔1两。
出处：《医方集解》。
主治：虫积蛊胀腹痛，及食劳发黄，喜食茶米炭土等物。五疳，蛔虫，脾胃不和，心腹胀痛，食少体瘦。
加减：None
功效：杀虫。
用法用量：每晨砂糖水送下。
制备方法：上药合炒，如喜食生米，用麦芽1斤炒；喜食茶叶，用茶叶炒；喜食炭土，用炭土炒；取药为末，蜜为丸。
临床应用：None
各家论述：此手足阳明药也。使君子之甘，南星之毒，槟榔之苦，皆能杀虫。炒以诸物，因其所嗜。引以砂糖，诱之以甘也。
用药禁忌：忌食辛辣及不易消化食物。
附注：《中药成方配本》本方用法：共研细末，用白蜜三两，炼熟为丸，分作一百粒，每粒约干重五分，每日二次，每次三丸，糖汤送下，小儿减半，连服三天。
