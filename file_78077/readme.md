序号：78093
名称：蟾酥丸
组成：杜蟾酥（烧酒化开）3钱，明雄黄（水飞）3钱，丁香2钱，木香2钱，沉香2钱，茅山苍术（土炒焦）4钱，朱砂（飞）1钱5分，当门子1钱，西牛黄3分。
出处：《霍乱论》卷下。
主治：暑月食凉饮冷，食物不慎，兼吸秽恶，成痧胀腹痛，或霍乱吐泻。
加减：None
功效：None
用法用量：每服3粒（轻者1粒，重者5粒）泉水送下。
制备方法：上为极细末，择净室中研匀，同蟾酥，加糯米粽尖五个，捣千余下，丸如椒子大，晒干，盛于瓷碗内；再用朱砂1钱5分，烧酒调涂碗内，盖好，用力摇一二千下，则光亮矣，密收瓷瓶内。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
