序号：53274
名称：健步虎潜丸
组成：虎骨2两，黄耆（炙）2两，茯神2两，当归2两，木瓜2两，川羌活2两，独活2两，防风2两，石菖蒲2两，知母（炒）2两，薏苡仁（炒）2两，生地2两，熟地2两，白术（土炒）2两，枸杞2两，白芍（炒）2两，怀牛膝2两，盐黄柏2两，补骨脂（炒）2两，杜仲（炒黑）2两，麦冬2两，远志2两（炒），五味子半两，沉香半两，附子（制）半两，龟版1两半（炙），人参2两。
出处：《全国中药成药处方集》（济南方）。
主治：筋骨无力，行步艰难，下部虚损，腿痠腰软，四肢无力，阳事痿弱，阴囊湿汗。
加减：None
功效：None
用法用量：每服1丸，温开水送下。
制备方法：上为细末，炼蜜为丸，重3钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
