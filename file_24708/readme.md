序号：24716
名称：如神散
组成：香白芷5钱，川芎5钱，防风（去芦）5钱，细辛5钱，雄黄2钱半，草乌（去皮脐）4钱，两头尖（去皮脐）4钱，苍术（米泔浸1宿，去皮）1两。
出处：《袖珍》卷一。
主治：一切金刃所伤，血不止，牙关紧急。
加减：None
功效：None
用法用量：血不止，干贴；破伤风，牙关紧急，用热酒1盏，调服1钱，又贴破处；风牙虫疼，擦之流涎即愈；狗咬蛇伤，煎盐汤洗净干贴，用热酒调半钱服，如重车行5里，并服3服，如重，服1钱；小儿量力减服；多年不效恶疮，煎葱白、盐汤浆水口漱，洗去脓血1-2次，拭干贴；一切蜂蝎小虫所伤，唾调于痛处贴之；如妇人产后，败血上冲下注，热酒调服1钱。
制备方法：上为极细末。
临床应用：None
各家论述：None
用药禁忌：忌腥、荤、面、油腻等物。
附注：如圣散（《丹溪心法附余》卷十六）。
