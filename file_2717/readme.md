序号：2717
名称：天中茶
组成：厚朴5钱（姜汁炒），广陈皮3钱，山楂1两，羌活3钱，小青皮3钱，干葛3钱，防风3钱，乌药3钱，川乌3钱，枳壳3钱，白芷3钱，茱萸3钱，石菖蒲3钱，甘草3钱，广木香（勿见火，另研末）3钱，砂仁3钱（另研末），制香附5钱，广藿香5钱，茅术（米泔水浸、洗，切片）5钱，莪术5钱，槟榔5钱，茯苓5钱，麦芽1两，神曲1两，紫苏1两，木通8钱。
出处：《集验良方拔萃》卷二。
主治：一切感冒，伏暑停食，滞而不化，胸膈不宽，气逆呕痰，疟痢。
加减：None
功效：None
用法用量：每服3钱，水2碗，煎1碗，红痢加白蜜糖5钱，白痢加赤糖1两。
制备方法：上药除木香、砂仁另研入，其余俱要饮片制过，共合一处，磨如粗末，5月初4日夜，每料用白酒1斤，浸药于瓷缸内，端午日，用六安茶或红茶叶，每料2斤半，入药内拌匀，待至午时，每料加雄黄末3钱5分，同温烧酒8两，搅匀拌茶内，即于午时炒干，临上坛时，再将木香、砂仁末拌和，候凉透，再扎好坛口，勿令泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
