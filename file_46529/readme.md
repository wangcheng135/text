序号：46544
名称：千金化铁丸
组成：当归（酒炒）1两半，白芍（酒炒）1两半，川芎7钱，怀生地（酒洗）1两半，白术（去芦，炒）1两半，白茯苓（去皮）1两，陈皮（去白）1两，青皮7钱半，半夏（姜汁炒）1两，枳实（麸炒）7钱5分，木香（炒）7钱5分，香附（炒）1两，槟榔5钱，莱菔子（炒）5钱，三棱（炒）5钱，红花5钱，干漆（炒令烟尽）5钱，桃仁（去皮尖）5钱，莪术（醋炒）1两5钱，硇砂（为末，瓷器内煨过）5钱，琥珀5钱。
出处：《寿世保元》卷三。
主治：积聚。腹中有块，坚硬如石，有时作痛，肚腹膨闷，经水不调，或前或后，或多或少，或闭而不通，白带频下，夜间发热，脉急数。
加减：None
功效：None
用法用量：每服3钱，白汤送下，早、晚各进1服。
制备方法：上为细末，醋打面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
