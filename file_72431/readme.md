序号：72447
名称：彭祖固阳固蒂长生延寿丹
组成：麝香5钱，丁香3钱，青盐4钱，夜明砂5钱，乳香2钱，木香2钱，小茴4钱，没药5钱，虎骨5钱，蛇骨5钱，龙骨5钱，朱砂5钱，雄黄3钱，白附子5钱，人参7钱，附子7钱，胡椒7钱，五灵脂5钱，槐皮、艾叶。
出处：《医学入门》卷一。
主治：骨髓风寒暑湿，五劳七伤，及久嗽久喘，吐血寒劳，遗精白浊，阳事不举，下元极弱，精神失常，痰隔；妇人赤白带下，从无生育，子宫极冷。
加减：若妇人炙脐，去麝，加韶脑1钱。
功效：坚固元气，令百病不生，益气延年。
用法用量：另用白面作条，圈子脐上，将前药1料，分为3分，内取1分，先填麝香末5分入脐眼内，又将前药1分，入面圈内，按药令紧，中插数孔，外用槐皮1片盖于药上，艾火灸之。无时损易，壮其热气，或自上而下，自下而上，一身热透，患人必倦沉如醉，灸至50-60壮，遍身大汗，上至泥丸宫，下至涌泉穴，苟不汗出，则病未愈，再于3-5日后又灸，灸至汗出为度。
制备方法：上为末。
临床应用：None
各家论述：方中麝香引诸药入五脏六腑，周彻百节；丁香入肺补血，实脾胃；青盐入肾以实其子，使肺母无泄漏，如乳补下益其气脘；夜明砂透肺孔，补气不足，散内伤；小茴治湿沥之症，调达周流，升降其气，不致喘嗽；雄黄削除病根，扶弱助强；白附子循各经络有推前拽后之功；人参、附子、胡椒补元气，行血化痰为津液；五灵脂保肺气，削有余，补不足；槐皮能闭押诸气之性，使无走窜；艾叶取其火热，劫病去毒，起死回生。
用药禁忌：灸时慎风寒，戒油腻生冷，保养一月。
附注：方中槐皮、艾叶用量原缺。
