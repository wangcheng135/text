序号：31048
名称：时疫救急丹
组成：藿香叶8两，香薷4两，丁香1两，檀香2两，木香1两5钱，沉香1两，白芷1两，木瓜3两，红大戟1两（炙），茅茨菇1两，薄荷1两，茯苓4两，千金子霜1两，神曲（炒）6两，厚朴（炙）2两，甘草4两（上为细末，再兑：麝香3分，冰片3钱，雄黄5钱，薄荷冰1钱5分，牛黄3分）。
出处：《北京市中药成方选集》。
主治：中暑中寒，暑湿霍乱，头晕身烧，腹痛肠鸣，呕吐泄泻。
加减：None
功效：祛暑散寒，止痛止泄。
用法用量：每16两用朱砂5钱、滑石3两5钱为衣。每服5分至1钱，温开水送下。
制备方法：上共为细末，混合均匀，用冷开水泛为小丸。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
