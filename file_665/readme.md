序号：665
名称：仁香汤
组成：白蔻仁6分（分冲），杜藿香钱半，广木香6分，生香附钱半，春砂仁8分（同煎），白檀香5分，母丁香4分，广陈皮钱半，生甘草3分，淡竹茹3钱。
出处：《重订通俗伤寒论》。
主治：素有肝气，一受痧秽，即胸膈烦闷，络郁腹痛。
加减：None
功效：疏肝快脾，辟秽散痧。
用法用量：None
制备方法：None
临床应用：None
各家论述：凡素有肝气，一受痧秽，即胸膈烦闷，络郁腹痛，夏秋最多，吾绍通称痧气，故以二仁、五香为君，芒香辟秽，辛香流气；臣以广皮疏中，竹茹通络；使以些须生甘草，以缓和辛散之气，此为疏肝快脾，辟秽散痧之良方。
用药禁忌：勿过投，免致耗气劫液。
附注：None
