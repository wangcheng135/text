序号：76092
名称：蛜(虫祁)丸
组成：蛜（虫祁）3分，木香3分，当归3分（锉，微炒），附子3分（炮裂，去皮脐），肉桂半两（去皴皮），朱砂半两（细研），阳起石半两（酒煮半日，细研，水飞过），阿魏半两（面裹煨，面熟为度），硫黄1两半，水银1两，硇砂3分，自然铜半两（细研），槟榔半两，茴香子3分，干姜半两（炮裂，锉），磁石半两（烧醋淬7遍，捣碎细研，水飞过），桃仁半两（汤浸，去皮尖双仁，麸炒微黄）。
出处：《圣惠》卷七。
主治：肾脏积冷气攻心腹疼痛，四肢逆冷，面色青黄，或时呕吐，不思饮食。
加减：None
功效：None
用法用量：每服20丸，以热酒送下，不拘时候。
制备方法：上为未，将硫黄、水银同结为砂子，细研，入诸药末和匀，醋煮面糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
