序号：57742
名称：烧肝散
组成：山茵陈1两半，石斛（去根）1两半，当归（切，焙）1两半，木香1两，桂（去粗皮）1两，人参1两，紫菀（去苗土）1两，桔梗（炒）1两，赤芍药1两，干姜（炮裂）1两，防风（去叉）1两，白芜荑1两，犀角（镑）1两，吴茱萸（汤洗，焙干，炒）1两，白术1两1分。
出处：《圣济总录》卷八十七。
主治：冷劳，面色萎黄，泄痢。
加减：None
功效：补五脏，通气脉，和脾胃，止泄痢。
用法用量：每用猪肝1具细切，入药末15钱匕，葱白5茎（细切），入盐3钱匕，与肝拌和令匀，分作3服。每服用荷叶包，更以湿纸3-5重裹，慢火烧肝令熟，空心、食前吃，用米饮下。如患冷劳，面色萎黄，不过吃10服愈。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
