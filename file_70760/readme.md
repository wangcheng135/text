序号：70776
名称：紫菀散
组成：紫菀3分（去苗土），桑根白皮3分（锉），甘草3分（炙微赤，锉），栀子仁半两，赤茯苓3分，桔梗半两（去芦头），黄芩半两，乌梅肉3分（微炒），川大黄半两（锉碎，微炒），百合3分，柴胡1两（去苗），麦门冬3分（去心），鳖甲2两（涂醋，炙微黄，去裙襕），杏仁3分（汤浸，去皮尖双仁，麸炒微黄）。
出处：《圣惠》卷三十一。
主治：骨蒸肺痿咳嗽，胸膈痛，舌涩口干。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，豉50粒，桃柳枝各1握（长7寸），葱、薤白各7寸，同煎至6分，去滓温服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：忌苋菜。
附注：None
