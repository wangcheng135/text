序号：36909
名称：坤厚资生丸
组成：九制熟地4两，当归（酒蒸）4两，白芍（酒炒）3两，川芎（酒蒸）1两5钱，白术（陈土炒）4两，茺蔚子（酒蒸）4两，香附4两（醋、酒、生姜汁、盐水各炒1两），丹参3两（酒蒸）。
出处：《大生要旨》卷一。
主治：妇女经事不调，临期腹痛，不能受孕。
加减：月经先期而至，脉数有热，属血热，加生地、丹皮；后期而至，脉迟厥冷，属血寒，加肉桂；将行而腹痛，是气滞，加乌药、木香；食少气虚，面色(白光)白，四肢无力，是为气血两亏，减附子1半，加人参、黄耆、河车、茯神、枣仁、远志之类。
功效：None
用法用量：每早服4钱，开水送下。
制备方法：上为末，以益母草8两，酒、水各半，熬膏，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
