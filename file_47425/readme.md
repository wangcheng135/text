序号：47440
名称：祛毒牛黄丸
组成：牛黄（研）2钱半，人参1两，南琥珀半两，犀角屑（取极细末）半两，桔梗半两，生干地黄（沉水研）半两，雄黄（飞）2两，川升麻3钱，南玄参3钱，蛤粉（水飞）4两，南硼砂半两，朱砂（飞研）7钱，铅白霜1钱，脑子3钱，金箔（为衣）、寒水石（烧赤，去火毒）3两（研）。
出处：《御药院方》卷九。
主治：大人小儿咽喉肿痛，舌本强硬，或满口生疮，涎潮喘急，胸膈不利，饮食难进。
加减：None
功效：None
用法用量：每服1丸，浓煎薄荷汤温化下，或新汲水化服亦得，食后1日2-3次，更或含化咽津亦得。
制备方法：上为细末，炼蜜为丸，如小弹子大。金箔为衣，用瓷器内收。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
