序号：66745
名称：鹿茸世宝丸
组成：鹿茸（酥涂，炙）1两，附子（炮，去脐）1两，白术（炒）1两，阳起石（烧赤）1两，椒红（炒出汗）1两，成炼钟乳粉1两，苁蓉（酒浸，炙）1两，人参（去芦）1两，肉豆蔻（面裹煨）1两，川当归（炒）1两，牛膝（去芦，酒浸1宿）1两，白茯苓1两，沉香1两，巴戟（去心）1两。
出处：《洪氏集验方》卷三。
主治：诸虚不足，心脾气弱，腹胁胀急，肠鸣泄泻，腹疼，手足厥逆，顽痹，中满恶心，头疼怯寒，肢体酸痛，饮食少思，气短乏力，惊悸自汗。
加减：None
功效：None
用法用量：每服40粒，食前盐饭饮或盐汤送下，1日3服。
制备方法：上为细末，次入钟乳粉拌匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
