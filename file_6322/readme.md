序号：6322
名称：加味补益养血汤
组成：生黄耆15g，党参15g，山药31g，白术15g，茯苓12g，砂仁12g，远志12g，柏子仁15g，炒酸枣仁25g，狗脊（去毛）15g，枸杞子12g，菟丝子25g，当归15g，丹参18g。
出处：方出《刘惠民医案》，名见《千家妙方》卷上。
主治：白细胞减少症。
加减：None
功效：健脾益气，养血和血，补肾。
用法用量：水煎2次，混合，分2次温服。
制备方法：None
临床应用：白细胞减少症尹某，女，42岁。于1975年1月28日初诊。患者一年前感到头昏，疲惫，两腿沉重，乏力、腰痠，食欲不振，半年前发现白血球减少，为3000／mm3，最低时仅为1000／mm3，服用各种升白细胞药无效，伴肢体麻木，失眠多梦，面色暗黄乏泽，舌淡红，苔薄白，脉沉细无力，投以加味补益养血汤，服药10余剂后，症状逐渐减轻，白细胞已升至6500-7500／mm3。稳定在5000／mm3以上，恢复工作。
各家论述：None
用药禁忌：None
附注：None
