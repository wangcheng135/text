序号：10420
名称：加味二陈汤
组成：茯苓、半夏、陈皮、枳实、甘草、桔梗、杏仁、贝母、瓜蒌仁、黄连。
出处：《伤寒全生集》卷三。
主治：痰实结胸。喘咳，胸胁满痛，作寒热，脉洪滑，心烦口渴者。
加减：胸腹满，加砂仁，去甘草；痰渴，去半夏，加知母，天花粉；嗽，加5味；喘，加桑皮、苏子；胁满痛，加青皮、白芥子、木香；有热痰结，加柴、芩、竹沥、姜汁少许，去半夏；有寒痰结，加干姜、姜汁，去贝母、黄连；风痰结，加南星、竹沥、姜汁；火痰，加山栀、黄芩、竹沥、姜汁少许，去半夏。
功效：None
用法用量：加生姜，水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
