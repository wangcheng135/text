序号：76483
名称：干地黄丸
组成：熟干地黄3两半，白茯苓（去黑皮），肉苁蓉（酒浸，去皱皮，切，焙）1两，远志（去心）1两半，牛膝（酒浸，切，焙）1两半，山芋1两半，山茱萸1两半，蛇床子（微炒）1两半，续断1两半，黄耆（炙，锉）1两半，覆盆子（去萼）1两半，石斛（去根）1两半，巴戟天（去心）1两半，泽泻1两半，附子（炮裂，去皮脐）1两半，菟丝子（酒浸，别捣）1两1分，桂（去粗皮）1两1分，牡丹皮1两1分，杜仲（去皱皮，锉，炒）1两1分，人参1两1分，鹿茸（去毛，酥炙）1两1分。
出处：《圣济总录》卷五十二。
主治：肾脏虚损，腰重不举，阳气痿弱，肢体瘦瘁。
加减：None
功效：None
用法用量：每服30丸，空腹温酒送下，1日3次。加至40丸。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
