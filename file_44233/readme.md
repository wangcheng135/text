序号：44248
名称：上池饮
组成：人参（去芦）2钱，台白术（去芦，炒）1钱5分，白茯苓（去皮）5钱，当归（酒洗）1钱2分，川芎1钱2分，杭白芍（酒炒）1钱，怀生地黄（姜汁炒）1钱，熟地黄（姜汁炒）1钱，南星（姜汁炒）1钱，半夏（姜制）1钱，陈皮（盐水洗）8分，羌活6分，防风6分，天麻1钱（去油），牛膝（去芦，酒洗）8分，川红花（酒洗）4分，柳枝6分（寒月1分），黄芩（酒炒）8分，黄柏（酒炒）3分（夏月加1分），酸枣仁（炒）8分，乌药4分，甘草（炙）4分。
出处：《寿世保元》卷二。
主治：一切中风，左瘫右痪，半身不遂，口眼歪斜，语言謇涩，呵欠喷嚏，头目眩晕，筋骨时痛，头或痛；心中忪悴，痰火炽盛。
加减：言语謇涩，加石菖蒲。
功效：None
用法用量：水煎，入竹沥、姜汁，清旦时温服。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
