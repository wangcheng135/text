序号：12014
名称：仙灵脾丸
组成：仙灵脾1两，牛膝1两（去苗），芎藭1两，牛黄1分（细研），麻黄1两（去根节），乌蛇肉2两（酒浸，炙微黄），天麻1两，白附子1两（炮裂），天雄半两（炮裂，去皮脐），防风1两，独活1两，当归1两（锉，微炒），桂心1两，细辛半两，白僵蚕1两（微炒），莽草1两（微炒），朱砂半两（细研），麝香1分（细研）。
出处：《圣惠》卷二十二。
主治：风亸曳，手足麻痹，屈伸不得。
加减：None
功效：None
用法用量：每服10丸，食前以豆淋酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌毒滑、鱼肉、羊肉等。
附注：None
