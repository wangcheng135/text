序号：83408
名称：乌金丸
组成：香附（制）4两，川大黄4两，木香5钱，乳香（炙）5钱，没药5钱，官桂5钱，五灵脂1两，桃仁泥1两，玄胡索1两，天台乌药1两，蓬莪术1两，全当归3两，益母草2两，蚕茧2两。
出处：《中国医学大辞典》。
主治：妇人七情悒郁，气滞食减，口苦咽干，五心烦热，面黄肌瘦，胸胁刺痛，崩中带下；产后恶露上攻，败血不止。
加减：None
功效：散郁化瘀。
用法用量：每服3钱，熟汤或温酒、艾醋汤送下。
制备方法：上为细末。用黑豆(洗净)1升，煮汁去滓；红花2两，酒5碗，煎4-5沸，去滓用汁；苏木3两，水煎，去滓用汁，将3汁和蜜为丸。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
