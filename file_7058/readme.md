序号：7058
名称：四物绛覆汤
组成：细生地4钱（酒洗），生白芍钱半（酒炒），真新绛钱半，广橘络1钱，全当归2钱（酒洗），川芎5分（蜜炙），旋覆花3钱（包煎），青葱管3寸（切，冲）。
出处：《重订通俗伤寒论》。
主治：血虚脉络郁涩，血郁化火，致郁结伤中，脘胁串痛，甚则络松血溢，色多紫暗。
加减：痛甚者，加桃仁7粒，蜜炙延胡1钱半，活血止痛；挟火者，加川楝子1钱半，丹皮1钱半，苦辛泄热。
功效：滋阴濡络。
用法用量：None
制备方法：None
临床应用：None
各家论述：以生地、归、芍养血滋阴为君；臣以绛、覆、川芎辛润通络；佐以橘络舒络中之气；使以葱管通络中之瘀。此为轻清滋阴，辛润活络之良方。
用药禁忌：None
附注：None
