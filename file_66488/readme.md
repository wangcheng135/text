序号：66504
名称：丁香丸
组成：丁香半两，木香半两，五灵脂（去沙）半两，荆三棱（煨）半两，蓬莪术（炮）半两，茴香半两，干漆3钱（炒烟出），胡椒4钱，槟榔2枚，青橘皮（去白）1两，陈橘皮（去白）1两，巴豆（春夏40粒，秋冬100粒，去壳，将2橘皮同巴豆炒令巴豆黑色，不用巴豆）。
出处：《普济方》卷一六九引《博济》。
主治：积聚，停滞不消，胸膈痞满，心腹疼痛，呕逆。
加减：None
功效：磨积破块，消酒食毒。
用法用量：每服2丸至9丸，生姜汤送下；或温酒送下。
制备方法：上为细末，用硼砂2钱，酒浸去沙石，入醋1盏，面1两，煮糊为丸，如梧桐子大，朱砂、麝香为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：丁红丸（《杨氏家藏方》卷五）。
