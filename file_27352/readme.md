序号：27360
名称：补真丸
组成：厚朴（去粗皮）4两，苍术（净刮去黑皮）4两（二味用大枣1斤半、生姜2斤，细切，同入大锅，以浆水煮1日，耗更添之，慢火泣尽水脉，焙干用），陈橘皮（汤浸，去瓤）2两，鳖甲1两（小便、酒、醋各1升，同煮1日了，更将汁涂炙了，焙干），石斛（去根）2两，丁香2两，肉苁蓉（酒浸、切，焙）2两，木香2两，巴戟天（去心）2两，当归（切焙）2两，草豆蔻（去皮）2两，诃子皮2两，肉桂2两，五味子2两，槟榔（锉）2两，山茱萸2两，杜仲（去粗皮，炙，锉）2两，破故纸（炒）2两，人参1两，黄耆（锉）3两，附子（炮裂，去皮脐）1两，柴胡（去芦头）1两，茯苓1两，沉香（锉）1两。
出处：《传家秘宝》卷下。
主治：脾元脏虚冷，四肢无力，吃食不得，心腹满胀，或时下痢盗汗，冷劳玄癖。
加减：None
功效：None
用法用量：每服20丸，空心米饮送下；一半作散，米饮调下，或煮羊肝，每具用药10钱匕，盐花、白浆水煮熟，空心服之。
制备方法：上为细末，将一半用枣肉为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：《普济方》有吴茱萸五钱。
