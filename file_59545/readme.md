序号：59560
名称：调中汤
组成：大黄2两，葛根2两，黄芩2两，芍药2两，桔梗2两，茯苓2两，藁本2两，白术2两，甘草（炙）2两。
出处：《外台》卷一引《古今录验》。
主治：夏月及初秋，忽有暴寒，折于盛热，热结四肢，则壮热头痛；寒伤于胃，则下痢，或血或水，或赤带下，壮热且闷，脉微且数。
加减：None
功效：和胃气。
用法用量：以水9升，煮取3升，分3次服，服别相去二食久，勿以食隔。须取快下，壮热便歇，其下亦止。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：葛根、藁本、甘草解表药也；黄芩、芍药、甘草清热药也；大黄、黄芩、甘草攻里药也；茶、术、桔梗、甘草和中药也。为小儿寒郁热邪，腹痛下痢之的方，功用与人参败毒散相仿。
用药禁忌：忌海藻、菘菜，猪肉、酢物、桃李、雀肉等。
附注：None
