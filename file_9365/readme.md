序号：9365
名称：加味如圣散
组成：桔梗2钱，牛蒡子1钱5分，麦门冬1钱5分，甘草1钱，玄参1钱，荆芥1钱，防风7分，生犀角5分，黄芩5分。
出处：《医方考》卷六。
主治：痘症痰嗽风热，声哑喉痛,风痰热壅，烦渴不宁，痘色干红，在五六朝前不润泽起胀者。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：1.《医方考》：牛蒡子、麦门冬疗风痰而清肺热；荆芥、防风散风邪而升郁热；甘草、桔梗、黄芩利咽喉而清气热；犀角、玄参凉心膈而疗积热。热去则金清，金清则声哑愈矣。_x000D_
2.《痘学真传》：牛蒡、麦冬、黄芩清肺家痰热，荆芥、防风散肌表风邪，犀角、玄参凉心胸之郁热，甘草缓诸药寒凉之性，桔梗领诸药归于肺家，不使下行，独用2钱，然此味材堪舟楫，不能独建奇功，方中似宜减半。
用药禁忌：None
