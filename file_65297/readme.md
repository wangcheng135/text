序号：65313
名称：大镇心丸
组成：干地黄6分，牛黄5分（1方用牛膝），杏仁5分，蜀椒5分，泽泻2分，黄耆2分，茯苓2分，大豆卷2分，薯蓣2分，茯神2分，前胡2分，铁精2分，柏子仁2分，羌活8分，桂心8分，秦艽8分，芎8分，人参8分，麦门冬8分，远志8分，丹砂8分，阿胶8分，甘草8分，大黄8分，银屑8分，桑螵蛸12枚，大枣40枚，白蔹8分，当归8分，干姜8分，紫石英8分，防风8分。
出处：《千金》卷十四。
主治：男子妇人虚损，梦寤惊悸，或失精神；妇女赤白注漏，或月水不利，风邪鬼注。寒热往来，腹中积聚，忧恚结气诸病。
加减：None
功效：None
用法用量：每服7丸，酒送下，1日3次。加至20丸。
制备方法：上为末，炼蜜、枣肉为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
