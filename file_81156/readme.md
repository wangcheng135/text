序号：81172
名称：麝香丸
组成：麝香（研）1分，龙脑（研）1分，丹砂1两半（研），雄黄（研）1两，木香1两，赤箭1两，牛黄（研）半两，白花蛇肉（酒浸，炙）半两，乌蛇肉（酒浸半两，炙）半两，干蝎（炒半两，去土）半两，羚羊角（镑）半两，天南星（炮）2两，麻黄（去根节）2两，白附子（生）3分，天麻（酒浸，焙）3分，防风（去叉）3分，零陵香叶3分，藿香叶3分，天雄（炮裂，去皮脐）3分。
出处：《圣济总录》卷二十四。（人卫本）。
主治：伤寒头痛，目眩汗出。
加减：None
功效：None
用法用量：每服2丸，细嚼，温酒送下，不拘时候。
制备方法：上十四味为末，入麝香等五味研匀，炼蜜为丸，如小鸡头实大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，原书文瑞楼本作“麝香双丸”。
