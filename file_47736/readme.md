序号：47751
名称：神仙鸡鸣丸
组成：知母（去毛）1两，贝母（去心）1两，杏仁（去皮）1两，款冬（炒）1两，甜葶苈（隔纸炒）1两，甘草1两，法子1两，北味（炒）1两，广皮（去白）1两，桔梗（炒）1两，鸡苏（晒）1两，天冬（去心）1两，粟壳（炒）1两，旋覆花1两，沙参（炒）1两，东阿胶（面炒珠，无真正者，以黄明胶代之）1两。
出处：《齐氏医案》卷三。
主治：诸般咳嗽。
加减：None
功效：None
用法用量：每服1丸，用乌梅2枚，大枣3个，泡浓汤，细嚼送下，小儿1丸分4服。或以乌梅、枣肉、白蔻仁各1两，焙干，同前药磨末为丸，生姜汤送下。
制备方法：上为极细末，炼白蜜为丸，如弹子大。
临床应用：咳嗽：曾治淯水范三才患咳唾，痰血相兼，治愈已三载。一日忽感风寒，咳嗽，医家误用滋阴之药，酿成吐血不止，此新疾也，先宜发散，继以滋阴方为合法，今误早为滋阴，闭其肺窍，乃勉强以人参败毒散四剂，服之其咳愈剧，遂与鸡鸣丸，令每夜细嚼三五粒，日服补中汤，加麦、味，不数日而咳嗽如失，血亦不吐，遂服六味都气丸而康。
各家论述：None
用药禁忌：None
附注：None
