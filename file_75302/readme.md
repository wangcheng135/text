序号：75318
名称：川芎石膏汤
组成：川芎半两，赤芍药半两，当归半两，山栀半两，黄芩半两，大黄半两，菊花半两，荆芥半两，人参半两，白术半两，滑石4两，寒水石2两，甘草3两，桔梗2两，缩砂仁1分，石膏1两，防风1两，连翘1两，薄荷叶1两。
出处：《宣明论》卷三。
主治：风热上攻头面，目昏眩，痛闷，风痰喘嗽，鼻塞口疮，烦满淋闭，眼生翳膜，中风偏枯。目疾时发，壅生翳膜，烦躁多渴，疮癣皴揭。
加减：None
功效：清神利头，宣通气血。解中外诸邪，调理诸病劳复传染。
用法用量：每服2钱，水1盏，煎至6分，去滓，食后服；水调亦得。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：忌姜、醋、发热物。
附注：川芎石膏散（《审视瑶函》卷六）。
