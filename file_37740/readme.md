序号：37750
名称：拌肝散
组成：茵陈1两，犀角屑半两，石斛半两（去根，锉），白术3分，赤芍药半两，柴胡3分（去苗），缩砂半两（去皮），人参3分（去芦头），桔梗3分（去芦头），防风半两（去芦头），肉桂3分（去皱皮），白芜荑仁半两，肉豆蔻半两（去壳）。
出处：《圣惠》卷二十六。
主治：脾劳虚冷，大肠滑泄，不思饮食，口舌生疮，四肢无力，日渐羸弱。
加减：None
功效：None
用法用量：用猪肝1叶，净去筋膜，不洗，薄切作片子，葱白3茎细切，入散5钱，重重掺在肝上，用湿纸5-7重裹，以慢火煨令熟，空心食，食后吃暖酒半盏。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
