序号：35923
名称：参苏理肺糖浆
组成：浙贝母8公斤，桔梗4公斤，北沙参8公斤，桔红16公斤，厚朴（盐制）4公斤，前胡16公斤，黄芩8公斤，葛根8公斤，天花粉8公斤，生半夏16公斤，紫苏叶16公斤，杏仁8公斤。
出处：《天津市中成药规范》（附本）。
主治：四时感冒，风寒咳嗽，头痛无汗，鼻塞声重，畏风恶寒，鼻流清涕，四肢无力。
加减：None
功效：散风解热，止咳化痰。
用法用量：每服2毫升（1-2汤匙）1日2次，苏叶每公斤折合苏叶油1.6毫升。
制备方法：浙贝母、桔梗、北沙参、桔红、前胡、厚朴、黄芩、葛根、天花粉、按浸渍法提出浸液。生半夏用水泡透心，加入生姜同煎。紫苏叶提取挥发油，残余药渣略加浓缩。杏仁先加热榨油，然后水解蒸馏，得杏仁水24升，将第三次滓液及第二次回收液放在容器中，加入杏仁水，再加沙参等浸液，最后兑入苏叶油，再加入单糖浆搅拌均匀，静置7天即得。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
