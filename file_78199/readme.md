序号：78215
名称：磁石散
组成：磁石（煅，醋淬7遍）1两，熟干地黄（焙）1两，菖蒲（米泔浸1宿，锉，焙）1两，牡丹皮1两，白术1两，附子（炮裂，去皮脐）半两，人参半两，白茯苓（去黑皮）半两，芎半两，大黄（锉，炒）半两，牡荆子（微炒）半两，桂（去粗皮）半两，当归（切，焙）半两，桑螵蛸（切破，炙）半两，羊肾1对（薄切，去筋膜，炙干）。
出处：《圣济总录》卷一一四。
主治：肾气虚弱，气奔两耳作声，甚则成聋。
加减：None
功效：None
用法用量：每服1钱匕，温酒调下，日3次，加至2钱匕，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
