序号：78127
名称：糯米膏
组成：糯米（先洗7次，绢袋挂当风2-7日，炒）1斗，紫河车（去皮毛，生用根赤者，不用白者）5两，五倍子（瓦燥之）5两，白敛2两（真者，如白萝卜干，色白，味苦甘），黄柏皮（炒焦）5两，黄芩（炒干）5两，白及（生）2两，当归（酒浸、焙干）2两。
出处：《普济方》卷二八九。
主治：发背疽毒，一切恶证。
加减：None
功效：None
用法用量：上为末，酸醋调入瓶，时取敷患处。若已成欲破，加白丁香，为末，和上药点头上，却用不加之药敷四畔；内服用白术、黄耆3两（蜜炙）、木香、当归各1两为末，酒下2钱，病在四肢食后服；膈上眠时,膈下空心服之。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
