序号：75210
名称：腽肭脐丸
组成：腽肭脐1两（酒刷，炙微黄），附子3分（炮裂，去皮脐），石斛3分（去根，锉），鹿茸1两（去毛，涂酥，炙微黄），牛膝3分（去苗），肉豆蔻3分（去壳），山茱萸3分，桂心半两，人参半两（去芦头），白茯苓半两，沉香3分，蛇床子半两，覆盆子3分，黄耆半两（锉），熟干地黄1两，槟榔3分，木香3分，巴戟3分，泽泻半两，补骨脂3分（微炒），吴茱萸半两（汤浸7遍，焙干，微炒），肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），菟丝子1两（酒浸3日，晒干，别捣为末）。
出处：《圣惠》卷九十八。
主治：腑脏虚弱，肌体羸瘦，下元冷惫，腰膝疼痹，心腹胀满，脾气乏弱，不思饮食，面无颜色，虚损不足。
加减：None
功效：None
用法用量：每服30丸，空心以温酒送下，晚食前再服。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
