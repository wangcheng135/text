序号：41232
名称：金华散
组成：当归3两（醋炒），蒲黄1两半（炒黑），干姜6钱（炮黑），桂心6钱（炒黑），黄连6钱（炒黑），石膏6钱（煨过），灵仙6钱（炒黑）。
出处：《医略六书》卷二十六。
主治：暴崩腹痛，脉数涩者。
加减：None
功效：None
用法用量：每服3钱，米饮调下。
制备方法：上为散。
临床应用：None
各家论述：冷热不调，血瘀冲任，故腹痛心烦，暴崩不止焉。姜、桂温冲任之寒，膏、连平肝胃之热；当归醋炒，养血归经；蒲黄炒黑，破瘀止血；灵仙走经络以调经也。为散，以散其瘀；饮下，以和其胃，使胃气调和，则冷热无不调，而血无瘀逆之患，何烦痛不除，暴崩不止乎。
用药禁忌：None
附注：None
