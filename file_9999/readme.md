序号：9999
名称：止痛如神汤
组成：秦艽（去苗）1钱，桃仁（去尖皮，另研）1钱，皂角子（烧存性，研）1钱，苍术（泔浸，炒）7分，防风7分，黄柏（酒洗）5分，当归尾（酒洗）3分，泽泻3分，尖槟榔1分（另研），熟大黄1钱。
出处：《外科启玄》卷十二。
主治：痔疮。
加减：如肿有脓，加白葵花5朵(去蕊心)，青皮5分，木香3分；如大便秘甚，加大黄、麻仁、枳实；如肿甚，加黄柏、防己、泽泻、猪苓、条黄芩；如痛甚，加羌活、郁李仁；如痒甚，加黄耆、羌活、防风、甘草、麻黄、藁本；如血下多，加地榆、黄柏、槐花、荆芥穗、白芷；如小便涩数不通者，加赤茯苓、车前、灯心、萹蓄。
功效：None
用法用量：上(口父)咀，除三味另研外，余药用水2钟，煎至1钟2分，入此三味，再煎至8分。空心热服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌生冷、五辛、烧酒、肝肠、湿面等。
附注：None
