序号：47565
名称：神仙丸
组成：朱砂6钱（用5钱，以1钱为衣），人参半两，沉香半两，全蝎（微炒）半两，白僵蚕（微炒）半两，天麻（炙）半两，天南星1个（重3两者，炮），川芎1两，附子1个（重6钱者，炮），五灵脂1两（只用8钱，），乳香1钱半，蜈蚣2条（酒浸，和蛇头1处浸），白花蛇头1个（连皮骨，酒浸3-4宿），乌蛇头1个（连皮骨，酒浸3-4宿），花蛇7钱（项后由7寸以后和皮骨1-2两，取7钱净肉，连蛇头1处浸），牛黄1钱，麝香1钱，脑子1钱，没药1钱，血竭1钱，硇砂（细研）1钱，雄雀1个（去肠胃，纳硇砂，用盐泥固济，文武火煅）。
出处：《幼幼新书》卷九引《张氏家传》。
主治：小儿急慢惊风；兼治中风瘫痪。
加减：None
功效：None
用法用量：治大人中风瘫痪，每服半丸，早晨用酒磨下；小儿急慢惊风，每丸分为4服，以薄荷酒磨下。
制备方法：上为末，以绝好酒为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
