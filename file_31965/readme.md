序号：31974
名称：应效祛风丸
组成：木香1两，槟榔1两，川芎1两，陈皮（去白）1两，青皮（去白）1两，防风（去芦）1两，天麻（去芦）1两，半夏1两，姜屑1两，车前子1两，猪牙皂角（去皮弦）1两，大黄4两（老弱虚者可减1两），牵牛头末半斤（如减大黄1两，可减头末2两）。
出处：《医方类聚》卷一九七引《经验秘方》。
主治：男子、妇人卒中风疾。
加减：None
功效：解毒化痰，消酒进食，润滑肌肤，明目益力。
用法用量：每服30-50丸，临卧温茶酒或温水送下。初服2-3日，或小便转恶色，乃肾家之病也；至4-5日，是脱寒热气；至7-8日，唇红生津，50日后，自觉身轻，四体安宁，头风百病皆退，胸中忧虑、三焦积滞皆散，远行不困。
制备方法：上为细末，陈粟米饭为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
