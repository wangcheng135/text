序号：9614
名称：仙方脑麝丸
组成：黄药子3两，白药子3两，天花粉2两，川连1两（用心择过，研末，筛细，只用头末），广木香3钱，沉香2钱，麝香5分，片脑3分。
出处：《痧书》卷下。
主治：岚瘴，及赤白痢，一切火症；痧胀，面赤身热，喘急昏迷者。
加减：None
功效：解茶痰酒渴，除伏暑，退心热，止喉疼，开目雾。
用法用量：瘴气痰渴，老年痰火，临卧噙化3丸；暑路常噙1丸，止渴消暑；如感大热，用5-7丸，同好茶1撮，盐梅1个，擂碎，井花水调下；心热头疼目雾，噙化3-5丸；赤痢，用茅根汁擂7丸；白痢，用茶梅擂服。
制备方法：猪胆调蒸为丸，每丸1分。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
