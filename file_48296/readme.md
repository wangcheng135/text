序号：48311
名称：神效丹
组成：猪牙皂角（不蛀者）4两（粗锉，用羯羊肝1个，切作片子，用水3大碗，煮数10沸，去肝，用皂角），半夏（拣大而陈者）4两（用姜汁浸3宿，漉去晒干，每个切作4片，生朱末1两，同于铁铫内同炒，以半夏黄为度，拣出半夏，用朱末别研入药），天南星2两（生用），白矾（明者）2两（略枯存性），黑牵牛2两（炒紫色）。
出处：《普济方》卷三七六。
主治：小儿诸痫。
加减：None
功效：None
用法用量：每服30-40丸，生姜汤送下。服绝根。大人亦可用。
制备方法：上为细末，用生姜汁打糊为丸，如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
