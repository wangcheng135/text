序号：63186
名称：大黑虎丹
组成：蛤蚧3对（微炒），虾蟆1枚（涂酥炙令赤），丹砂5铢（细研），金箔5片（研），银箔5片（研），白鲜皮3钱，苦参3钱，蛇蜕皮1两（微炙），白狗粪1分（微炒），皮巾子6钱（3年者，炙令黄），金刚子3钱，乌驴蹄3钱（炙黄），硫黄1钱（细研），雄黄1分（细研），天灵盖1分（涂酥炙令微赤），麝香1两（细研），沉香1分，甲香3铢（微炙），乳香3钱，夜明砂6钱（雄者，两头尖），人中白1分。
出处：《圣惠》卷三十一。
主治：传尸复连，及一切劳证，不问冷热大小。
加减：None
功效：None
用法用量：每服药，先以新汲水1盏，入少许麝香、砂糖，以药10丸，浸于露下，来旦，先用煎茅香汤浴后，吃淡面1小盏子，便吃药。服了，以衣被厚盖卧，直候汗出。
制备方法：上为末，更都研令匀，用猪胆汁并软饭和捣为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
