序号：82846
名称：丹参膏
组成：丹参2两，蒴藋2两，秦艽1两，独活1两，乌头1两，白及1两，牛膝1两，菊花1两，防风1两，莽草叶半两，踯躅花半两，蜀椒半两。
出处：《肘后方》卷五。
主治：恶肉，恶核，瘰疬，风结，诸脉肿；疔肿、痈疽。
加减：None
功效：None
用法用量：上切，以苦酒2升，渍之1宿，猪膏4斤，俱煎之，令酒竭，勿过焦，去滓。以涂诸疾上，日5度，涂故布上贴之。此膏亦可服，得大行，即须少少服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：丹参膏虽云散血消肿，而实外敷毒风之峻药。
用药禁忌：None
附注：方中防风，《千金》作“防己”。《圣惠》有白术。
