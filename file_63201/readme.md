序号：63217
名称：清肺滋阴散
组成：川芎（酒洗）1钱，白芍（炒）1钱半，生地黄2钱，白术（炒）1钱，陈皮1钱，白茯苓8分，黄柏（蜜炒）1钱，知母1钱，贝母（去心）1钱，紫菀8分，五味子6分，款冬花8分，麦门冬1钱，地骨皮1钱，黄连（炒）5分，远志（甘草汤泡）8分，酸枣仁（炒）6分，甘草4分。
出处：《古今医鉴》卷七引杜次泉方。
主治：酒色太过，斫丧真阴，阴火上升，肺金受侮，以致唾痰稠浊，咳嗽咽疮。
加减：心下怔忡，夜卧不寐，加人参8分；心烦躁乱，加枳实6分，竹茹6分；如痰涎壅盛，加瓜蒌仁6分，天花粉1钱；如咽喉有疮，用通嗌散吹之。
功效：None
用法用量：加生姜1片，竹沥3匙，水煎服。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
