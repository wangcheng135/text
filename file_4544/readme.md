序号：4544
名称：木香汤
组成：木香半两，枸杞子半两，沉香半两，山芋半两，附子（炮裂，去皮脐）半两，天麻半两，半夏（汤洗7遍，焙）半两，秦艽（去苗土）半两，当归（切，焙）半两，鳖甲（去裙襴，醋炙）半两，黄耆半两，牛膝（酒浸，切，焙）半两，羌活（去芦头）1分，枳壳（去瓤，麸炒）1分，巴戟天（去心）1分，白茯苓（去黑皮）1分，肉豆蔻（去壳）4枚，柴胡（去苗）1两，人参1两，甘草（炙）1两。
出处：《圣济总录》卷八十七。
主治：气劳，身体羸瘦，四肢少力，面色萎黄，饮食减少，呕逆痰沫，咳嗽胸满。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，加生姜2片，葱白1寸，煎至7分，去滓温服，不拘时候。
制备方法：上（口父）咀，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
