序号：19317
名称：地黄丸
组成：熟地黄（洗）5钱，赤茯苓2钱，当归2钱，山茱萸（蒸，去核）2钱，川楝肉（焙）2钱，牡丹皮2钱，山药2钱，川芎2钱，使君子（煨）2钱。
出处：《普济方》卷三八一引《百一》。
主治：小儿肾疳。多由乳食不调，脏腑伏热所致。凡滋味入于脾而生虫，虫大则动，侵蚀脏腑，遂使小儿心闷。若上蚀腭龈，则口疮出血，齿色紫黑；下蚀肠胃，则下痢肛烂，湿痒生疮。或以走马命名，盖齿属肾，肾气虚才受邪热。疳气直奔上焦，初作口气，名曰臭口；次第齿黑，名曰崩砂；盛则龈烂，名曰溃槽；热血逆出，名曰宣露；甚者牙齿脱落，名曰腐根，其根即腐，齿不复生矣。外证脑热脱削，手足如冰，寒热时作，滑泄肚痛，口臭干渴，牙龈生疮，爪黑面黧，身疼疮疥。
加减：None
功效：None
用法用量：每服3丸，空心温汤送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：九味地黄丸（《明医指掌》卷十）。《婴童百问》有泽泻，无川芎。
