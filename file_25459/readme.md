序号：25467
名称：八制茯苓丸
组成：白茯苓2斤半（须皮光结实者，去皮，打碎如枣核大，分为8制），黄耆6两（切片，水6钟，煎3钟，煮茯苓1分，干为度），肉苁蓉4两（酒洗，去筋，水6钟，煎3钟，煮茯苓如前），人参6钱（水5钟，煎3钟，煮茯苓如前），甘枸杞6两（水8钟，煎3钟，煮茯苓如前），补骨脂5两（水8钟，煎3钟，煮茯苓如前），何首乌半斤（用黑豆1升，煎水3斤，浸首乌，春秋2日，夏1日，冬3日，将浸过首乌豆汁煮茯苓如前），秋石4两（水3钟化开，煮茯苓如前），人乳半斤（煮茯苓如前）。
出处：《广嗣纪要》卷四。
主治：一切虚损。
加减：None
功效：男子壮筋骨，生心血，乌须发；女子滋颜色，暖子宫，调经气。
用法用量：生子者，每日早晚1服，每服40丸，盐汤送下，乌须明目，用滚白汤送下。
制备方法：将制过茯苓放入石臼内捣为细末，用米筛筛过，上甑蒸熟，众手为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌烧酒、犬肉。
附注：None
