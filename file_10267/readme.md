序号：10267
名称：气宝丸
组成：茴香子（拣净，银石器内纸衬炒）2两，陈橘皮（汤浸，去白，焙）1两，槟榔（锉）1两，木香1分（4味同杵，罗为末），黑牵牛（拣净称）4两（用吴萸2两慢火同炒茱萸焦，只取牵牛子，1向杵取末2两）。
出处：《圣济总录》卷六十二。
主治：膈气呕逆，心胸痞满，食饮不下。一切滞气，腹中积聚，心胸痞满，腹闷喘急；及风邪久滞，痰饮咳嗽，酒食有伤，脾胃滞气，膀胱寒气，攻注体背，腰脊痛肿，不可俯仰。
加减：None
功效：None
用法用量：每服10丸至15丸，米饮或木香汤送下；有痰，即用槟榔末半钱，水半盏，煎数沸，放温送下；欲微疏利，加至30丸至40丸。看虚实，腹稍空服之。
制备方法：上药同拌匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
