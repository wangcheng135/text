序号：80065
名称：摩风膏
组成：黄耆（去粗皮）1两2钱，当归（去芦头）3钱，白芍药2钱半，茅香2钱半，甘草2钱半，防风2钱半，白芷3钱，杏仁（汤浸，去皮尖）3钱，桃仁（汤浸，去皮尖）3钱，藿香叶（去土）3钱，檀香3钱，川芎3钱，零陵香3钱，白附子2钱半，沉香2钱半，白及2钱半，白蔹2钱半，天麻2钱半，独活2钱半，木香2钱半，木通2钱，大瓜蒌瓤（锉）1个，龙脑（研）4钱，清油1斤2两，黄蜡（冬月用）9两半（夏月用）12两半。
出处：《御药院方》卷十。
主治：头面、唇、鼻诸疮，肌肉裂痛。
加减：None
功效：摩风止痒，消肿定痛。
用法用量：每用少许，摩擦患处。
制备方法：上锉，用清油浸7日，于净石器或瓷器或银器中，以慢火煎，候白芷微黄色，以白绵滤去滓，干净瓷器罐内密封澄1宿，再滤过，于上等瓷碗中慢火再轻温熬动，下黄蜡，和令匀，放温，次下研细龙脑末，于瓷盒内盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
