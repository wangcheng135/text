序号：75443
名称：解毒内消汤
组成：连翘1两，金银花1两，蒲公英1两，败酱草1两，冬瓜子1两，赤芍2钱，丹皮2钱，川军1钱，赤小豆3钱，甘草节2钱，土贝母3钱，犀黄丸3钱（分两次吞服）。
出处：《刘奉五妇科经验》。
主治：盆腔脓肿属于热毒壅聚者。
加减：None
功效：清热解毒，活血化瘀，消肿止痛。
用法用量：None
制备方法：None
临床应用：None
各家论述：方中重用连翘、银花、蒲公英、败酱草清热解毒消痈；丹皮、赤芍清热凉血活血；川军活血破瘀而又清热解毒，三者均能除败血生新血，消肿排脓；冬瓜子、赤小豆入血分，清热消肿排脓；甘草节、土贝母清热解毒消肿；另配犀黄丸以加强活血消肿清热止痛之效。
用药禁忌：None
附注：None
