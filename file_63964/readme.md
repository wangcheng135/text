序号：63980
名称：清膈散
组成：桑白皮5钱（炒，研用），紫苏2钱半，黄芩2钱半，当归2钱半，前胡2钱半，连翘2钱半，防风2钱半，桔梗2钱半，天门冬2钱半（去心），赤茯苓2钱半，甘草（炙）2钱半（1方有生干地黄）。
出处：《普济方》卷三八一。
主治：肺疳（即气疳）。由乳哺不调，壅热伤肺，风湿之气乘虚客于皮毛，入于血脉，鼻下两旁疮湿痒烂，是名(匿虫)。其疮汁所流，却又成疮，外证咳嗽喘逆，壮热恶寒，皮肤粟生，鼻干流涕，咽喉不利，颐烂唾红，气胀毛焦，泄痢频并，多啼，揉鼻咬甲，寒热。
加减：None
功效：None
用法用量：每服2钱，以水1盏煎，温服，不拘时候。次服化(匿虫)丸。
制备方法：上锉散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
