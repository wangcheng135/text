序号：23742
名称：安坤赞育丸
组成：益母草2斤（分作4分，用盐水、黄酒、姜汁、醋炙，干燥备用），香附4两（用盐水、黄酒、姜汁、醋分4分炙），黄芩4两，粉甘草4两，白芍（酒炒）4两，赤芍（酒炒）1两，川贝（去心）1两，血竭1两，丹参4两，陈皮1两，川芎1两，阿胶1两（蛤粉炒），南红花1两，生地2两，熟地3两，麦芽2两，当归（酒炒）1两，乳香（炒透）5钱，没药（炒透）5钱，砂仁2两，木香1两，云苓1两，丹皮1两，杜仲（炒炭）1两，蕲艾1两（炒炭），白术（土炒）1两，寸冬1两。
出处：《全国中药成药处方集》（济南方）。
主治：月经不调，经期腹痛，胸膈痞闷，赤白带下。
加减：None
功效：None
用法用量：早、晚各服2钱，腹痛下痢，呕吐，姜汤送服；产后诸症，黄酒送服；余皆白开水送下。
制备方法：上为极细末，水泛小丸，如绿豆大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：禁忌生冷食物。
附注：None
