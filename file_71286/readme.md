序号：71302
名称：紫石门冬丸
组成：远志（去心）2两，泽泻2两，肉苁蓉2两，桂心2两，紫石英3分，天门冬（去心）3分，五味子3两，禹余粮1两，蜀椒（汗）1两，乌头（炮）1两，卷柏1两，乌贼骨1两，寄生1两，石南1两，当归1两，杜仲2两，甘草（炙）2两，石斛2两，柏子仁2两，辛夷2两，人参2两，云母1两（烧）。
出处：《外台》卷三十三引《经心录》。
主治：风冷在子宫，有子常落；或始为妇，便患心痛，乃成心疾，月水未曾来者。
加减：None
功效：令人肥悦、有子。
用法用量：每服20丸，酒送下，1日3次，稍加至30、40丸。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌海藻、菘菜、猪肉、冷水、生葱、鲤鱼。
附注：紫石英天门冬丸（《千金》卷四）、紫石英丸（《圣惠》卷七十七）。方中紫石英，天门冬用量原缺，据《千金》补。
