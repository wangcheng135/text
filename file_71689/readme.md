序号：71705
名称：搜风镇心牛黄丸
组成：牛黄1分，龙脑1钱，麝香1钱，雄黄1分，丹砂5分（五味同研），天麻（去苗）半两，乌犀（镑）半两，干蝎（去土，炒）半两，羚羊角（镑）半两，羌活（去芦头）半两，独活（去芦头）半两，防风（去芦头）半两，细辛（去苗叶）半两，白芷半两，赤茯苓半两，蝉蜕半两，麻黄（去根节）半两，牛膝（酒浸，切，焙）半两，芎半两，五加皮半两，白附子（炮）1分，乌蛇（酒浸，去皮骨，炙）1两。
出处：《普济方》卷一一四。
主治：一切风，瘫痪，言语謇涩。
加减：None
功效：None
用法用量：上为细末，炼蜜为丸，如鸡头大。每服1丸，嚼破以温酒或荆芥茶送下，1日3次。
