序号：11899
名称：凤髓丹
组成：杜仲1两（生姜汁制），木瓜1两半，狗脊1两，川乌8两（炮，去皮），萆薢2两，牛膝1两，虎骨5钱（酥制），鹿茸5钱（酥制），麝香2钱，芎藭5钱，肉桂5钱（去皮），全蝎3钱，细辛7钱，乳香5钱，没药5钱，木鳖子3钱（去壳），海桐皮5钱，五加皮5钱，威灵仙7钱（去皮），薏苡仁5钱，自然铜7钱（醋淬7次），地龙2两半，木香5钱，骨碎补1两，朱砂5钱（为衣）。
出处：《普济方》卷一一四。
主治：肝肾二脏稍有不足，风寒之气侵之；每到中夜后腰背疼痛，转侧不得，不能行步；或左瘫右痪，肢体偏枯，起止不前，阳气衰绝，阴气大胜，久卧床枕，渐渐日深者。
加减：None
功效：None
用法用量：每服50丸，好酒温送下。
制备方法：上为细末，酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
