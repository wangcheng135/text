序号：53956
名称：换骨酒
组成：白茯苓3两，晚蚕沙（炒）3两，虎胫骨（酒浸，炙黄）半两，甘草1两，槟榔1两，郁李仁（汤浸，去皮）半两，附子（炮，去皮脐）半两，何首乌半两，防风半两，瓜蒌半两，牛蒡子根半两，牛膝半两，干菊花半两，杜仲（去皮丝）半两，黄耆半两，白附子1两，益智仁1两，石菖蒲半两，天麻1两，山茱萸1两，牡蛎半两，牡丹皮半两，枸杞子半两，蛇床子1两，肉苁蓉1两，羌活半两，鼠粘子半两，狗脊（去毛）1两，天雄（炮，去皮脐尖）半两，干姜（炮）半两，苍耳子（炒）半两，菟丝子1两，紫菀半两，白术、桔梗、白花蛇（酒浸，去皮骨，炙）半两。
出处：《普济方》卷九十六引《卫生家宝》。
主治：早年患偏风证，四肢不举，非人回转，不能自动，兼眼目疾。
加减：None
功效：益精补虚，活血驻颜，润皮肤，退一切风疾，乌髭发，轻身健骨，爽精神。
用法用量：每服1盏，温暖服，日进3服。久患者不过月，日近者只3-5服。其药滓阴干为末，炼蜜为丸，如梧桐子大。每服30-40丸，温酒送下。服药3日，便能手梳头，7日四肢渐舒，10日行步，半月觉身轻眼明。常服乌髭发，身轻骨健，爽精神，净房修养一月妙。
制备方法：上为粗末，用好无灰酒3-5斗，生绢袋盛药，在坛子内封闭，令蜜浸之，春、夏14日，秋、冬21日，开时人面不得向坛口上，取浸药酒。
临床应用：None
各家论述：None
用药禁忌：净房修养一月。
附注：本方改作丸剂，名“换骨丹”（原书同卷）。
