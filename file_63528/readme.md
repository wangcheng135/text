序号：63544
名称：清凉内消膏
组成：芝麻油2斤，大黄1两5钱，金银藤1两5钱，黄芩1两5钱，苦参1两5钱，荆芥1两5钱，玄参1两5钱，白芷1两5钱，僵蚕1两5钱，黄柏1两5钱，桃仁1两5钱，杏仁1两5钱，防风1两5钱，栀子1两5钱，羌活1两5钱，独活1两5钱，蜂房1两5钱，头发1两5钱，青藤1两5钱，连翘1两5钱，蛇蜕1两5钱，木鳖子1两5钱，川山甲1两5钱，芍药1两5钱，南星1两5钱，黄耆1两5钱，当归1两5钱，黄连1两5钱。
出处：《摄生众妙方》卷八。
主治：痈疽，发背，疔疮，大小疖。
加减：None
功效：None
用法用量：大小疖1-2日方起，用油纸量疮大小均贴之，不许揭动，待疮消散则除之。
制备方法：上将前药共锉碎，入油内以文武火熬之，待至白芷紫色住火，滤去渣，用黄丹3斤（先用水淘过炒热）并麝香1两（罗过），待煎药油熬滚渐渐加之，滴水中，看软硬适中不粘手为度。
临床应用：None
各家论述：None
用药禁忌：None
附注：若疮势已过四五日则难退。
