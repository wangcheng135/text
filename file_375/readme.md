序号：375
名称：五香麻黄汤
组成：麝香半两，熏陆香1两，鸡舌香1两，沉香2两，青木香2两，麻黄2两，防风2两，独活2两，秦艽2两，萎蕤2两，甘草2两，白薇2两，枳实2两。
出处：《千金》卷九。
主治：伤寒忽发肿，或著四肢，或在胸背，虚肿浮如吹状，亦着头面唇口颈项，剧者偏著脚胫，外如轴大，而不痛不赤；著四肢者，乃欲不遂。
加减：None
功效：None
用法用量：上(口父)咀。以水9升，煮取3升，分3次服。覆取汗后，外摩防己膏。
制备方法：None
临床应用：None
各家论述：《医方考》：上件诸肿，乃是余邪未去，营卫之行，不相顺接，逆于肉理，而为肿尔！是方也，用五香以开气窍；而麻黄、防风、独活、秦艽、萎蕤、白薇，皆辛散之，一以解其余邪，一以流其着气；甘草之补，所以致新；枳实之悍，所以推陈。
用药禁忌：None
附注：五香汤（《圣济总录》卷三十）。
