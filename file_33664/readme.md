序号：33673
名称：牡丹皮汤
组成：牡丹皮1两半，白芷1两半，桑耳1两半，诃黎勒皮（煨）1两半，代赭石（碎）1两半，龙骨（去土）1两半，当归（切，焙）1两半，黄连（去须）1两1分，黄耆（炙，锉）1两1分，地榆1两1分，鹿茸（去毛，酥炙）1两1分，苍术（米泔浸，切，焙）1两，附子（炮裂，去皮肤）1两，杏仁15个（去皮尖双仁，炒令黄），肉豆蔻（去皮）2个，黄芩（去黑心）半两。
出处：《圣济总录》卷一五一。
主治：妇人经水不调，腰背疼痛，食物不得。
加减：None
功效：None
用法用量：每服5钱匕，以水1盏半，加生姜5片，煎取8分，去滓温服。
制备方法：上（口父）咀，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
