序号：49727
名称：茯苓汤
组成：白茯苓（去黑皮）6两，麦门冬（去心，焙）6两，款冬花6两，独活（去芦头）6两，槟榔（锉）6两，桂（去粗皮）5两，防风（去叉）5两，防己5两，甘草（炙）4两，枳壳（去瓤，麸炒）4两，地骨皮（去土）10两。
出处：《圣济总录》卷九十三。
主治：心蒸。苦心惊悚粟，男子因读书损心气，伤思虑，过损心，吐血，心烦多忘，失精神，或身体痒瘙，风癣，或胸中气满。
加减：None
功效：None
用法用量：每服5钱匕，以水2盏，先煎山泽根，取水1盏半，入药并生姜半分（切），大枣3个（擘破），同煎取1盏，去滓温服，每早晨、日晚各1次。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
