序号：34392
名称：万病丸
组成：远志（去心）半两，泽泻半两，石斛（去根）半两，柏子仁（别研）半两，云母（水飞）半两，石韦（去毛）半两，杜仲（去粗皮，炙）半两，天雄（炮裂，去皮脐）半两，牛膝（去油，酒浸，切，焙）半两，白茯苓（去黑皮）半两，菖蒲半两，山芋半两，熟干地黄（焙）半两，肉苁蓉（酒浸，切，焙）半两，续断半两，干姜（炮）半两，甘菊花半两，桂（去粗皮）半两，五味子半两，蛇床子（炒）半两，山茱萸半两，桔梗（炒）1两，防风（去叉）1两，白术1两，附子4枚（炮裂，去皮脐），天门冬（去心，焙）1两半，细辛（去苗叶）3分。
出处：《圣济总录》卷一○○。
主治：五劳七伤，尸疰所侵，心腹痛，饮食不化，两胁鼓胀，皮肤挛缩等病。
加减：如久服，即减天雄、附子各1半。
功效：None
用法用量：每服20丸，空心温酒送下，春、秋日再服，夏、季日1服，冬季日3服。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
