序号：59753
名称：调经丸
组成：当归1两，陈皮1两，白芷1两，牛膝1两，三棱1两，蓬术（醋制）1两，玄胡索1两，白术1两5钱，川芎1两5钱，枳壳1两5钱，小茴香1两5钱，熟地1两5钱，香附2两，粉草5钱。
出处：《郑氏家传女科万金方》卷二。
主治：室女17-18岁，脾胃虚弱，误食生冷，经脉不通，或阻百日，或半年，颜色青黄，饮食少进，寒热往来，四肢困倦，头痛目弦，肚腹疼痛，五心烦热，呕吐膨胀。
加减：None
功效：补气血，扶脾胃，调经水。
用法用量：每服90丸，空心米饮或酒送下。先服逍遥散，次取加味八珍汤，再服此方。
制备方法：醋煮粳米为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
