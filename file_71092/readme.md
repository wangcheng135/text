序号：71108
名称：葱术散
组成：苍术（洗净）1斤，葱（连根须洗净）1斤，麻黄（不去节）4两，甘草2两（炙）。
出处：《传信适用方》卷一。
主治：时行瘟疫及寻常风吹雨洒，头目昏重，四体热倦，行步少力，骨节疼痛，不论阴阳二证，染患浅深。
加减：None
功效：None
用法用量：每服2钱，水1盏，生姜3片，枣1个，煎7分，去滓热服；如要出汗，连进2-3服，以衣盖之。常服解诸劳倦，行路缓急，最为先务。
制备方法：上将葱、术入臼中杵，令葱涎相入盆中，坚捺令热，冬间半月，春夏5-7日取出，锅中炒葱叶过焦，逐旋筛出，以术干为度，次入麻黄、甘草为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
