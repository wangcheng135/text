序号：66010
名称：蛇床子丸
组成：蛇床子（炒）1分，肉苁蓉（酒浸，去皱皮，切，焙）1分，细辛（去苗叶）1分，石韦（去毛）1分，山茱萸1分，矾石（煅，研）1分，防风（去叉）1分，远志（去心）1分，赤石脂1分，白茯苓（去黑皮）1分，泽泻1分，柏子仁（炒，别捣末）1分，菖蒲1分，栝楼根1分，天雄（炮裂，去皮脐）1分，牛膝（去苗，酒浸，锉，焙）1分，续断1分，山芋1分，杜仲（去粗皮，酥炙，细锉）1分。
出处：《圣济总录》卷九十二。
主治：五劳七伤，阴衰，小便余沥，阴中痛，精清，囊下湿，胸胁痛，两膝厥冷，不欲行，胃中热，远视泪出，口干肠鸣。
加减：None
功效：None
用法用量：每服30丸，空腹温酒送下，夜卧再服。
制备方法：上为细末。炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
