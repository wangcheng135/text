序号：57626
名称：大通丸
组成：金钗石斛1两，牛膝1两（酒浸），大附子2个1两（去皮脐），干姜3分（炒），豆蔻4个（去壳，面煨），槟榔4个，木香1分，菊花2两，硫黄1分，白花蛇（酒浸，去皮骨）2两，枸杞子2两（九蒸九晒，炒）。
出处：《普济方》卷三十二引《博济》。
主治：丈夫肾脏风，上攻下注，脚膝疼痛生疮；及小便膀胱宿冷，滞气攻刺腹胁；并妇人血风攻注，腰膝拘挛；风走注，身体疼痛，荣卫凝涩。
加减：None
功效：None
用法用量：每服空心温酒下；妇人，当归酒下。如作散，温酒调下。如吃了转觉脑骨内疼甚者，乃药效力；如疼发过后，只管吃即永愈。
制备方法：上为细末，以酒煮糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：《圣济总录》：每服二十丸，空心温酒送下；如作细散，每服钱半匕，温酒、粥饮任调下。
