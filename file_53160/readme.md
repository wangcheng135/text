序号：53175
名称：大连翘饮
组成：连翘4分，瞿麦4分，滑石4分，车前子4分，牛蒡子4分，赤芍4分，栀子4分，木通4分，当归4分，防风4分，柴胡1钱2分，黄芩1钱2分，荆芥1钱2分，蝉蜕5分，甘草1钱6分。
出处：《回春》卷七。
主治：小儿伤风感冒，发热痰壅，风热丹毒肿痛，颈项有核，腮赤痈疖，眼目赤肿，口舌生疮，咽喉疼痛，小便淋沥，胎毒，痘疹余毒，一切热毒；三焦积热，二便不利，重舌、木舌，咽痛疮疡。
加减：风痰热、变蒸，加麦冬；实热、丹热，加大黄；胎热、疮疹余毒，加薄荷叶；痈疖、热毒，加大黄、芒消。
功效：None
用法用量：加竹叶10个，灯心10茎，水煎，温服，不拘时候。
制备方法：上锉。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
