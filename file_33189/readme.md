序号：33198
名称：万灵神膏
组成：连翘1两，山栀1两，防风1两，羌活1两，独活1两，黄连1两，赤芍1两，生地1两，当归1两，川芎1两，大黄1两，玄参1两，苦参1两，白芷1两，五倍1两，桔梗1两，白及1两，白蔹1两，官桂1两，两头尖1两，山慈姑1两，花粉1两，蓖麻子1两，木鳖子1两，红芽大戟1两，杏仁40粒，巴豆40粒。
出处：《经验广集》卷四。
主治：咽喉喘嗽，负重伤力，男子遗精，女人白带，月经不调，膨胀，左瘫右痪，心胃肚腹疼痛，痞块，痈疽瘰疬，疥癣，一切肿毒，跌打损伤，四时伤寒，呕吐。
加减：None
功效：None
用法用量：咽喉喘嗽，或负重伤力，贴前胸，用火烤手摩百次；男子遗精，女人白带，月经不调，膨胀，俱贴气海穴；左瘫右痪及心胃肚腹疼痛，俱贴患处；男妇痞块，贴患处，再用面作圈放痞上，皮消1两，鸽粪5钱，共和匀捣烂入圈内，以熨斗频熨即愈；痈疽瘰疬初起，火烤手摩出汗即愈，其疥癣有脓血者不用摩，一切肿毒并跌打损伤，俱贴患处；四时伤寒贴背，火烤出汗为度；呕吐贴心口，或肺俞穴。
制备方法：麻油1斤半熬药枯黑，滤去滓，再入锅内熬，滴水成珠，倾出瓷器，称准熟油2斤，入松香2两、黄蜡2两、桐油2两，如嫩少加杭粉，熬至嫩硬得中，待温下后药：乳香、没药、儿茶、血竭、阿魏、麝香、轻粉，为细末，徐徐下油，以不粘手为度。土埋3日出火毒，瓷器收贮任用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
