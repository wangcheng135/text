序号：36332
名称：和中益气丸
组成：木香半两，丁香半两，肉豆蔻1两，茴香（微炒香）1两，京三棱（炮，锉）1两，桂（去粗皮）1两，白豆蔻仁1两，人参（去芦头）1两，缩砂仁2两，青皮（去白）4两，陈皮（去白）4两。
出处：《御药院方》卷三。
主治：脾胃不和，气不升降，呕吐减食，口苦吞酸，胸满短气，肢体怠堕，面色萎黄，中焦虚痞，不任攻击，脏气久寒，不受峻补；又疗心胸愊愊如满，五饮停滞不散，或大便不通。
加减：None
功效：和中益气，进美饮食。
用法用量：每服60-70丸，食后温生姜汤送下。
制备方法：上为细末，白面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
