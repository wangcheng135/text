序号：23800
名称：守中金丸
组成：干姜（炮）、甘草（爁）、苍术（米泔浸）、桔梗（去芦）各等分。
出处：《局方》卷三（吴直阁增诸家名方）。
主治：中焦不和，脾胃积冷，心下虚痞，腹中疼痛，或饮酒过多，胸胁逆满，噎塞不通，咳嗽无时，呕吐冷痰，饮食不下，噫醋吞酸，口苦失味，怠惰嗜卧，不思饮食；及伤寒时气，里寒外热，霍乱吐利，心腹绞痛，手足不和，身热不渴，及肠鸣自利，米谷不化；及脾胃留湿，体重节痛，面色萎黄，肌肉消瘦。
加减：None
功效：温脾暖胃，消痰逐饮，顺三焦，进美饮食，辟风寒湿冷。
用法用量：每服1丸，食前嚼服，以沸汤送下。
制备方法：上为细末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方改为汤剂，名“守中汤”（见《活幼心书》）。
