序号：63251
名称：清金止嗽西瓜膏
组成：大西瓜10个，杏仁（去皮）2两，苏叶2两，桔梗2两，麦冬2两，麻黄2两，米壳2两，桑皮2两，瓜蒌皮2两，茯苓2两，枳壳（炒）2两，生姜2两，杷叶1两，五味子（炙）1两，薄荷1两，清半夏1两，川贝母1两，冬花1两，甘草1两，化橘红1两，生地1两，萝卜8两。
出处：《北京市中药成方选集》。
主治：肺经停饮，外受风寒，咳嗽喘满，痰涎过盛，口干舌燥。
加减：None
功效：润肺止嗽，化痰定喘。
用法用量：每服5钱，1日2次，白开水冲服。
制备方法：上药酌予切碎，水煎3次，分次过滤，去滓，滤液合并，用文火煎熬，浓缩至膏状，每16两膏汁兑炼蜜32两，冰糖16两成膏，每瓶装重2两。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
