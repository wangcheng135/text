序号：6994
名称：四时加减柴胡饮子
组成：柴胡8分，白术8分，大腹槟榔4枚（并皮子用），陈皮5分，生姜5分，桔梗7分。
出处：《金匮》卷下。
主治：五脏虚热。
加减：如四体壅，添甘草少许，每贴分作3小贴，每小贴以水1升，煮取7合，温服；再合滓为1服，重煮，都成4服。
功效：None
用法用量：冬3月，加柴胡；春3月，加枳实，减白术；夏3月，加生姜3分，枳实5分，甘草3分；秋3月，加陈皮3分，各(口父)咀。分为3贴，1贴以水3升，煮取2升，分温3服，如人行4-5里进1服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：四时加减柴胡汤（《兰台轨范》卷三）。
