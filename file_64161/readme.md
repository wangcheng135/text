序号：64177
名称：猪肾荠苨汤
组成：猪肾1具，大豆1升，荠苨3两，石膏3两，人参2两，茯神（一作茯苓）2两，磁石（绵裹）2两，知母2两，葛根2两，黄芩2两，栝楼根2两，甘草2两。
出处：《千金》卷二十一。
主治：强中之病，茎长兴盛，不交精液自出。消渴之后，即作痈疽，皆由石热。中焦虚热注于下焦，烦渴引水，饮食倍常。
加减：None
功效：None
用法用量：上(口父)咀。以水1斗5升，先煮猪肾、大豆，取1斗，去滓下药，煮取3升，分3服，渴乃饮之。下焦热者，夜辄合1剂，病势渐歇即止。
制备方法：None
临床应用：None
各家论述：1.《法律》：此方用白虎等清凉之剂，加入猪肾、大豆、磁石，引诸清凉入肾，且急服之，凡热炽盛于上下三焦者，在所必用。_x000D_
2.《千金方衍义》：石药之悍，虽流布中外，其毒必伏匿少阴经中，所以借水兽之肾引领荠苨专主强中之味，与磁石、知母、黑大豆同入肾经，佐以石膏、黄芩、葛根、甘草、栝楼根，辅佐荠苨分解内外之毒，制剂虽专，不得人参阳药助胃以行其力，则毒匿幽深何由发越，孰谓人参壅补热邪而致扼腕耶。
用药禁忌：None
附注：石子荠苨汤（《三因》卷十）、二石荠苨汤（《医统》卷五十二）。本方方名，《证治要诀类方》引作“荠龙汤”。
