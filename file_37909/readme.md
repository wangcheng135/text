序号：37919
名称：拨云散
组成：羊脑炉甘石8两（拣没石性的，用砂茶吊1个，将炉甘石上火1煅，用水飞去细粉，粗滓不用，晒干听用），川黄连5钱，羌活5钱，连翘5钱，黄芩5钱（水3大碗，煎1碗，又水2碗，煎半碗，2次放1处；又将飞过炉甘石烧红，倒在童便内，如次3淬，第4次烧红方淬于煎药内，再勿见火；如有湿，待其自干听用），硼砂3两（生用），海螵蛸2两（煮去盐性），石决明（煅）1两，乳香（去油）5钱，没药（去油）5钱，瓜儿血竭5钱，熊胆3钱，麝香3分，冰片1钱。
出处：《良朋汇集》卷三。
主治：老年目昏，攀睛胬肉，拳毛倒睫，迎风流泪。
加减：None
功效：None
用法用量：点时用骨簪蘸凉水，点大眼角。
制备方法：上共乳无声方好，瓷罐内秘收。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
