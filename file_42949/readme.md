序号：42964
名称：保真种玉丸
组成：鹿茸（去毛）1两2钱，鹿肾2具，海马8具，虎骨（炙）1两，狗肾6具，熟地8钱，肉桂（去粗皮）1两，山药1两2钱，当归1两6钱，杜仲炭1两2钱，白术（炒）1两2钱，牛膝1两2钱，枸杞1两2钱，五味子（炙）1两2钱，茯苓1两2钱，党参（去芦）3两，补骨脂（炒）2两，菟丝子2两，核桃肉2两，小茴香（炒）2两，沙苑子2两，附子8两，苁蓉8两，巴戟肉（炙）8两，龙骨（煅）1两，母丁香3两，黄耆2两，山萸肉（炙）1两2钱，甘草（炙）1两。
出处：《北京市中药成方选集》。
主治：肾气亏虚，阳痿不兴，腰膝无力，久无子嗣。
加减：None
功效：滋补腰肾，添精益髓。
用法用量：每服1丸，温开水送下，1日2次。
制备方法：上为细末，炼蜜为丸，每丸重3钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
