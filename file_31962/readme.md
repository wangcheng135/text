序号：31971
名称：应病接骨膏
组成：龙骨（紧者）1两，虎胫骨1两，乌鱼骨1两，川山甲（炙）1两，地龙1两，黑狗脊1两，木鳖子（去皮）1两，牛膝1两，川乌头1两，泽乌头1两，草乌头1两，半夏1两，吴茱萸1两，山茱萸1两，石茱萸1两，细辛1两，芍药1两，肉苁蓉1两，川椒1两，乳香1两，没药1两，白芥子1两，防风1两，自然铜（制）1两，川楝子1两，猪牙皂角2两（别研）。
出处：《医方类聚》卷一八八引《烟霞圣效方》。
主治：一切损伤，肿痛伤折。
加减：None
功效：None
用法用量：量所折伤处骨节大小，用纸摊匀挟裹定，药力尽自离，然后更用药贴。30岁以上，不过2上，50岁以下，不过3上，定也。
制备方法：除乳香、没药外，余味为极细末，用好油二斤，于文武火上锅内一处同熬，槐枝不住手搅，烟起闻香才成，入黄丹半斤，并研细底乳香、没药细细熬，直至滴水不散成珠子，硬软得所成也。倾在水盆中去火毒，旋捞出，放盛在瓷器内收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
