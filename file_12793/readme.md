序号：12799
名称：石斛丸
组成：石斛1两（去根，锉），熟干地黄3分，麦门冬1两半（去心，焙），五味子半两，牛膝1两（去苗），泽泻半两，肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），防风半两（去芦头），芎藭3分，独活半两，秦艽2分（去苗），人参半两（去芦头），桂心3分，甘草半两（炙微赤，锉），细辛半两，附子1两（炮裂，去皮脐），黄耆半两（锉），石龙芮半两，白芍药半两，白茯苓3分。
出处：《圣惠》卷三十。
主治：虚劳痿痹，四肢挛急，肌体枯瘦。
加减：None
功效：None
用法用量：每服30丸，食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌生冷、猪、鸡、牛、马肉。
附注：None
