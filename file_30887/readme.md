序号：30896
名称：沉香桂附丸
组成：沉香1两，附子（炮，去皮脐）1两，干姜（炮）1两，良姜（锉，炒）1两，官桂（去皮）1两，茴香（炒）1两，川乌头（炮，去皮脐，锉作小块子如豆大，再炒令黄用）1两，吴茱萸（汤浸，洗去苦，炒）1两。
出处：《医学发明》卷六。
主治：中气虚弱，脾胃虚寒，饮食不美，气不调和，脏腑积冷，心腹疼痛，胁肋膨胀，腹中雷鸣，面色不泽，手足厥冷，便利无度；及下焦阳虚，七疝痛引小腹不可忍，腰屈不能伸，喜热熨稍缓。
加减：None
功效：退阴助阳，除脏腑冷气。
用法用量：每服50-70丸，空腹、食前以熟米饮送下，每日2次。
制备方法：上为细末，用好醋煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：《医略六书》：附子补火回阳以御邪，肉桂温经暖血以散邪，川乌逐在里之邪，泽泻泻逆上之邪，良姜暖胃散寒滞，吴茱平肝降逆气，沉香导厥气之上逆；小茴温气化以下达也。醋丸盐汤下，使真火内充，则厥气下潜而客邪解散，疝气自消，安有急痛欲死之患，手足逆冷之虑乎。
用药禁忌：忌生冷硬物。
附注：《医略六书》有泽泻，无干姜。
