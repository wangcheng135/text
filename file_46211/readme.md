序号：46226
名称：洞虚丹
组成：藁本1两5钱，天麻1两5钱，川芎1两5钱，细辛1两5钱，牛膝3两，羌活3两，大枫子4两，蝉壳2两，胡麻2两，防风2两，独活2两，僵蚕2两，荆芥2两，苏木2两，风藤2两，石膏2两，蒺藜2两，山栀2两，芍药2两，菖蒲2两，石蚕2两，黄芩2两，连翘2两，草乌2两，紫萍2两，升麻2两，红花2两，麻黄2两，白芷2两，石斛2两，当归2两，威灵仙2两。
出处：《解围元薮》卷三。
主治：恶风，麻木走注抽痛。
加减：None
功效：None
用法用量：每服100丸，用羊踯躅草根1斤4两，打碎，以酒20斤煮去滓，取1杯送下。
制备方法：上为末，面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌食盐物。
附注：None
