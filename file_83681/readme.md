序号：83699
名称：乌梅丸
组成：乌梅肉（微炒）1两半，鳖甲（酒浸，去裙襕，炙令黄色）1两，天灵盖（涂酥，炙令黄色）1两，虎头骨（酒浸，炙令黄色）1两，常山（细锉）1两，柴胡（去苗）1两，肉苁蓉（去皱皮，酒浸，炙干）1两，蜀漆叶3分，知母（焙干，锉）3分，甘草（炙）3分，升麻3分，大黄（锉）半两，豉（炒令黄）半两，桃仁29枚（汤浸，去皮尖双仁，生用）。
出处：《圣济总录》卷三十五。
主治：诸疟久不愈，发动无节，日渐虚困羸瘦。
加减：如愈后，觉冷损惫，即加生干地黄、防风、黄耆、桂各1两，香豉减半。日1服。
功效：None
用法用量：每服20丸，未发前空腹以枳壳、桑根白皮汤送下。渐加至30丸。
制备方法：上药除桃仁外，为细末，与桃仁相和，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
