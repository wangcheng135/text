名称：舒筋丸
组成：川牛膝1两，木香1两，补骨脂（盐炒）5钱，山甲（炙）5钱，杜仲（炒）5钱，熟地2两，白芷5钱，山茱萸（炙）1两，钻地风5钱，泽泻1两，藁本1两，马钱子（炙去毛）1两，续断5钱，蜈蚣1钱，全蝎5钱，厚朴（炙）5钱，钩藤5钱，红花5钱，当归5钱，黑豆8两，脂麻4两，桂枝5钱，羌活5钱，独活5钱，官桂5钱，秦艽5钱，狗脊（烫去毛）1两，白术（炒）1两，千年健1两，威灵仙1两，何首乌（炙）1两，川乌（炙）4两，草乌（炙）4两，土茯苓1两，肉桂（去粗皮）5钱。
