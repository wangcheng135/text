序号：3941
名称：无价散
组成：人粪（烧）、猫粪（烧）、猪粪（烧）、犬粪（烧）各等分少许。
出处：《御药院方》卷十一。
主治：斑疮发出不快，倒靥黑陷，一切恶疮。
加减：None
功效：None
用法用量：小儿每用1字，用蜜调服。
制备方法：将上4物于腊日早晨日未出盛贮于销1铤银锅子内，用木炭火大笼煅令烟尽白色为度。
临床应用：None
各家论述：None
用药禁忌：None
附注：化毒散(《普济方》卷四○三引《经验方》)、四味万两金丹(《疮疡经验全书》卷四)、万金散(《古今医鉴》卷十四)、四灵无价散(《治痘全书》卷十四)、万两黄金散、四圣散(《救偏琐言·备用良方》)。本方方名，《医学纲目》引作“四粪散”。本方入麝少许，名“健效化毒散”(见《普济方》卷四○三)。
