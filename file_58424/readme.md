序号：58439
名称：破积导饮丸
组成：槟榔5钱，陈皮（去白）5钱，广木香5钱，青皮（去白）5钱，枳壳（麸炒）5钱，枳实（麸炒）5钱，广茂（炮）5钱，半夏（泡7次）5钱，京三棱（泡）5钱，神曲（炒）5钱，麦糵（炒）5钱，干生姜5钱，茯苓（去皮）5钱，甘草（炙）5钱，泽泻5钱，牵牛（头末）2钱，巴豆3个（去心膜，取霜）。
出处：《卫生宝鉴》卷十四。
主治：积块坚硬，饮食不消，心下痞闷。
加减：None
功效：None
用法用量：每服30丸，食前温姜汤送下。
制备方法：上为末，入巴豆匀，生姜汁打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：破积导引九（《医统》卷三十三）。《理瀹》无茯苓、枳实，有黄连、车前子。
