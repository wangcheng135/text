序号：27376
名称：补益桑黄丸
组成：桑黄半斤，牛膝（酒浸，切，焙）1斤，桃仁（去皮尖双仁，炒研如膏）半斤，麦糵（炒）半斤，白术半斤，陈曲（炒）半斤，当归（切，焙）半斤，大黄（锉，炒）半斤，生地黄10斤（绞自然汁），生姜10斤（绞自然汁）。
出处：《圣济总录》卷七十一。
主治：积聚，女子诸疾。
加减：None
功效：暖血海。
用法用量：每服20丸，渐加至30丸，空腹温酒送下。
制备方法：上为末，与桃仁膏同入二汁内拌匀瓷器盛，甑内蒸一日，取出焙干，捣罗为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：桑黄丸（《普济方》卷三二八）。方中桃仁，《普济方》作“杏仁”。
