序号：3620
名称：巴戟丸
组成：巴戟1两，肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），石斛1两（去根，锉），鹿茸1两（去毛，涂酥，炙微黄），附子1两（炮裂，去皮脐），薯蓣3分，牛膝3分（去苗），桂心3分，山茱萸3分，泽泻3分，远志3分（去心），熟干地黄1两，菟丝子1两（酒浸3日，晒干，别捣为末），黄耆3分（锉），人参3分（去芦头），槟榔3分，木香3分，牡丹3分，仙灵脾3分，蛇床子3分，续断3分，枳壳3分（麸炒微黄，去瓤），白茯苓3分，覆盆子3分。
出处：《圣惠》卷九十八。
主治：丈夫下焦久积风冷，肾脏虚乏，腰膝痠痛，小便数；阳道衰，不能饮食，面无颜色，筋骨痿弱，起坐无力，膀胱虚冷，脐腹胀急。
加减：None
功效：久服驻颜色，养精志。
用法用量：每日服20丸，渐加至30丸，空心盐汤送下；温酒送下亦得。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
