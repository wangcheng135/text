序号：41640
名称：金煎药方
组成：升麻6分，连翘6分，桔梗5分，黄芩5分，生地7分，苏木5分，黄柏5分，黄耆3分，全蝎3分，人参3分，白豆蔻4分，甘草2分，地龙（去土，焙干）5分，桃仁3分，虻虫3分（去头翅），梧桐泪1分，川归4分，水蛭3分（炒烟尽），黄连3分，寸香少许。
出处：《秘传大麻风方》。
主治：麻风。
加减：None
功效：None
用法用量：上药除黄连、连翘、梧桐泪、豆蔻等外，先将寸香、水蛭、虻虫亦研为末，余药都作1服，水2杯，酒1钟，入连翘同煎去滓。入梧桐泪、白蔻、寸香再煎至7分，稍热服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌酒面生冷、咸味油腻。
附注：None
