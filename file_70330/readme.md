序号：70346
名称：煮砂方
组成：朱砂4两，黄连1两，鼠粘子1两，紫草5钱（先煮3味药2次，每次用水4碗，煎至2碗，去滓，用布裹砂穿吊锅上，离锅底一指，煮前药水，以干为度，取朱砂晒干），当归6两，川芎5两，升麻4两，甘草3两（如上煮2次，每次用水10碗，煎至5碗，仍如上煮砂，水干为度）。
出处：《痘学真传》卷七。
主治：痘症发热，或见红点者。
加减：None
功效：None
用法用量：每岁儿用朱砂、糯米各1分，入白蜜1匙，米汤小半瓶，每日调服1匙。
制备方法：用糯米1升，炒黄，贮瓷瓶封固。
临床应用：None
各家论述：朱砂得离火之色，禀坎水之性，能坠下，以解先天之毒；黄连、鼠粘子、紫草以助清火化毒之功；当归、川芎以养血分，升麻以升沉滞，甘草以和苦寒，可谓识痘毒之源，探稀痘之奥！
用药禁忌：None
附注：None
