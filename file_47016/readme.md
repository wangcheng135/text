名称：独活散
组成：独活2两，白芍药1两，白术1两，葛根1两（锉），白茯苓1两，防风1两（去芦头），茵芋1两，细辛1两，甘草1两（炙微赤，锉），汉防己1两，芎1两，酸枣仁1两，桂心1两，人参1两（去芦头），五加皮2两，麻黄2两（去根节），川乌头1两（炮裂，去皮脐）。
出处：《圣惠》卷二十二。
主治：卒中恶风，口噤不能言，四肢軃曳，缓弱疼痛，风经五藏，恍惚，恚怒无常。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，煎至5分，去滓，入竹沥1合，更煎1-2沸，稍热服，不拘时候。
制备方法：上为粗散。
临床应用：None
