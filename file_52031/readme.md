序号：52046
名称：大防风汤
组成：防风18铢，当归18铢，麻黄18铢，白术18铢，甘草18铢，黄芩30铢，茯苓1两，干地黄1两，附子1两，山茱萸1两。
出处：《千金》卷八。
主治：中风，发热无汗，肢节烦，腹急痛，大小便不利。
加减：大小便不利，纳大黄、人参各18铢，大枣30枚，生姜3两，煮取3升，分3服。《深师》加天门冬1两。
功效：None
用法用量：上(口父)咀。以水9升，煮取2升半，1服7合。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：中风外有六经形证，故用麻黄、防风；内有便溺阻隔，故用地黄、当归；肾主二便，大小便不利多属肾虚风燥，故用术、附为主，加茯苓、甘草，则真武汤中之二也；山茱萸，《本经》治心下邪气，温中逐寒湿痹，去三虫，佐地黄则有酸收肝肾虚风之功；黄芩，《本经》治诸热、黄疸，逐水，下血闭，佐麻黄则有解散肌表风热之用。
用药禁忌：None
附注：None
