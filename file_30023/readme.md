序号：30031
名称：赤金豆
组成：巴霜（去皮膜，略去油）1钱半，生附子（切，略炒燥）2钱，皂角（炒微焦）2钱，轻粉1钱，丁香3钱，木香3钱，天竺黄3钱，朱砂2钱（为衣）。
出处：《景岳全书》卷五十一。
主治：诸积不行，凡血凝气滞，疼痛肿胀，虫积，结聚癥坚。
加减：None
功效：None
用法用量：欲渐去者，每服5-7丸；欲骤行者，每服10-20丸，用滚水或煎药，或姜、醋、茶、蜜、茴香、使君煎汤为引送下；若利多不止，可饮冷水1-2口即止，盖此药得热则行，得冷则止也；如治气湿实滞膨胀，先用红枣煮熟取肉1钱许，随用7-8丸，甚者10-20丸，同枣肉研烂，以热烧酒加白糖少许送下；如治虫痛，亦用本肉加服，只用清汤送下。
制备方法：上为末，醋浸蒸饼为丸，如萝卜子大，朱砂为衣。
临床应用：None
各家论述：①《景岳全书》：此丸去病捷速，较之消、黄、棱、莪之类过伤脏器者，大为胜之。②《医略六书》：寒凝坚积，癥结于中，故痛急肿胀，厥逆不已。生附略炒，补火之功用稍峻；巴豆炼霜，荡涤之勇猛可除；丁香温中散滞；木香调气宽中；轻粉（？？）涎透经络；皂角豁痰通窍门；朱砂镇心养液，竺黄凉心宁神，二物并能保护心主，使悍烈之药勿上僭。盖火旺土温则寒邪外散，而坚积自消，癥结无不化，疼胀亦无不退矣。此扶阳涤结之剂，为癥坚疼急肿胀之专方。
用药禁忌：None
附注：None
