序号：34123
名称：赤箭丸
组成：赤箭2两，赤茯苓半两，芎半两，防风半两（去芦头），白附子半两（炮裂），桂心半两，羚羊角屑3分，白术3分，羌活半两，汉防己半两，附子半两（炮裂，去皮脐），当归半两，五加皮半两，牛膝半两（去苗），杜仲1两（去粗皮，炙微黄，锉），石斛半两（去根节），麻黄半两（去根节），海桐皮1两半，木香半两，枳壳半两（麸炒微黄，去瓤）。
出处：《圣惠》卷二十三。
主治：腲退风。脏腑虚弱，风湿所攻，致腰脚缓弱，肌肉虚满，肢节疼痛。
加减：None
功效：None
用法用量：每服20丸，食前以豆淋酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
