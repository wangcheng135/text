序号：67655
名称：小半夏汤
组成：半夏1升，生姜1斤，橘皮4两（一方用人参2两）。
出处：《千金》卷十八。
主治：痰饮胸满，呕逆恶心，头痛眩晕，喘逆咳嗽。心腹虚冷，游痰气上，胸胁满，不下食，呕逆，胸中冷者。呕逆恶心，头疼眩运，臂痛背寒，嘈烦多睡。痰壅涎嗽，久不已者。肺胃虚弱，好食酸冷，寒痰停积，呕逆恶心，涎唾稠粘，或积吐，粥药不下，手足逆冷，目眩身重；又治伤寒时气，欲吐不吐，昏愦闷乱；或饮酒过多，中寒停饮，喉中涎声，干哕不止。水饮喘逆而无火者。
加减：若心中急及心痛，纳桂心4两；若腹满痛，纳当归3两。
功效：养液润燥，解肌热，止咳嗽。温理中气。
用法用量：以水1斗，煮取3升，分3服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：羸弱及老人尤宜服之。
附注：橘皮半夏汤（《宣明论》卷九）。
