序号：4753
名称：木香散
组成：木香1两，人参1两（去芦头），白茯苓1两，当归1两（锉，微炒），白芍药半两，桂心半两，麦门冬1两（去心），远志1分（去心），五味子半两，京三棱半两（炮，锉），白术1两，诃黎勒半两（煨，用皮），厚朴1两（去粗皮，涂生姜汁，炙令香熟），陈橘皮1两（汤浸，去白瓤，焙）。
出处：《圣惠》卷五。
主治：脾胃气虚，不思饮食，精神恐悸，上气顿绝，身心昏昧，口干舌焦，四肢无力。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，大枣3枚，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻。
附注：None
