序号：27980
名称：连翘饮
组成：连翘1分，瞿麦1分，滑石1分，车前子1分，牛蒡子1分，赤芍1分，山栀仁半分，木通半分，蝉退半分，当归半分，防风半分，黄芩1分半，荆芥1分半，柴胡2分，甘草2分。
出处：《医学入门》卷八。
主治：小儿表里诸热；小儿痘疹14日时。
加减：如风热、痰热、变蒸热、肝热、大肠热、瘾疹热，加麦门冬；丹热、实热、血热、三焦热、小肠热，加大黄、灯心；麻痘热、温气热、已出未出症热，加紫苏、当归；余毒热、胎热、肺热、伤寒后、疮疹后余毒发热，加薄荷；项上生核作热、痈疖毒热，加大黄、朴消。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
