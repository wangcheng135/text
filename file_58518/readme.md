序号：58533
名称：秘方紫金膏
组成：黄连10两（净，捶碎），黄柏皮10两（去皮外，洗，锉），（上以腊水于净锅煎取1升，滤滓，再用水2斗，煎取1升，复以重绢绵滤），杏仁30粒（去皮尖，研极细，作膏），芜荑半两（去皮，研细）（上以前药水同于银石器熬成膏），黄丹5两（先用细绢罗了，再研极细），铜青（净）4分，轻粉4分。
出处：《普济方》卷八十六引《海上方》。
主治：一切眼疾。
加减：None
功效：None
用法用量：每用1丸，井水磨点。
制备方法：上同入前膏子内，不住手搅匀，再用艾叶20片重捣，烧烟熏药膏子了，然后为丸，如皂子大，阴干，密器收藏，用脑麝养，久留为妙。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
