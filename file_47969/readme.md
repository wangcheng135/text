序号：47984
名称：神传剪草膏
组成：剪草1斤（婺州者）。
出处：《本事》卷五。
主治：劳瘵吐血损肺，及血妄行。
加减：None
功效：None
用法用量：每服4匙，五更用匙抄药和粥服，良久用稀粟米饮压之，药冷服，粥饮亦不可太热，或吐或下皆不妨。如久病肺损咯血，只1服愈，寻常咳嗽血妄行，每服1匙可也。
制备方法：上洗净为末，入生蜜1斤，和为膏，以器盛之，不得犯铁，九蒸九晒，日一蒸晒。
临床应用：None
各家论述：《本事方释义》：剪草气味苦寒，入手太阴、手足厥阴，痨瘵而致久咳吐血不止，损伤及肺，血溢妄行，此方虽近似丹方，亦是培土生金之法。
用药禁忌：None
附注：仙传膏（《串雅内编》卷四）。本方方名，《医学纲目》引作“神传膏”。
