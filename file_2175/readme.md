序号：2175
名称：七珍丹
组成：胆星5钱，天竺黄5钱，淡全蝎1两，炒僵蚕1两，寒食曲1两，朱砂面5钱，净巴豆霜2钱，麝香1钱，明雄黄面5钱。
出处：《全国中药成药处方集》（天津方）。
主治：伤乳伤食，积聚痞块，消化不良，肚胀腹痛，咳嗽痰涎，惊风拘挛，大便不通。
加减：None
功效：清热，利便，化痰。祛风化痰，镇惊导滞。
用法用量：小儿3-4个月，每次服3粒；5-6个月服4-5粒；周岁服6-7粒，白开水化服。
制备方法：前5味共为细末，兑入后4味，研细和匀，凉开水泛小丸，如小米粒大，每斤丸药用朱砂面1两上衣。
临床应用：None
各家论述：None
用药禁忌：体弱者勿服。
附注：《中药制剂手册》：无寒食曲者，用馒头干代替。
