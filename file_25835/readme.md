序号：25843
名称：早夺汤
组成：人参1两，生黄耆1两，当归1两，远志3钱，生甘草3钱，金银花1两，大黄1两，石膏1两，柴胡2钱，白术1两，天花粉2钱。
出处：《辨证录》卷十三。
主治：杨梅疳疮初发，鱼口将生。
加减：病人阴虚阳燥，方中再加熟地数两．或玄参1两亦可，余品不可乱加。
功效：补中攻泄。
用法用量：水煎服。1剂而大泻恶物，臭秽不堪，再服2剂而臭物恶秽无留于肠胃矣，后可减去大黄、石膏，加土茯苓2两同煎药，再煎服4剂，则一身上下与头面之间，必有隐隐疮影视于皮肤之内，再服2剂，疮影亦渐消矣，再服2剂，永不发矣。
制备方法：None
临床应用：None
各家论述：大黄以泄毒，石膏以清毒，甘草、金银花以化毒，柴胡、天花粉以散毒，非多助之以大补气血之药，妙在用参、耆、归、术之类，自获全胜。
用药禁忌：None
附注：《洞天奥旨》有茯苓。
