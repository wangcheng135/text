序号：7466
名称：生肌散
组成：乳香5钱，没药5钱，血竭5钱，雄黄5钱，蒲黄5钱，梧子5钱，赤石脂5钱，白芷5钱，朴消5钱，寒水石5钱，陀僧5钱，龙骨5钱，轻粉5钱，花蕊石5钱，山甲5钱，螃蟹粉5钱，硼砂5钱，蟾酥5钱，朱砂3钱，乌药3钱。
出处：《跌损妙方》。
主治：跌打损伤，臁疮，疠症。
加减：若臁疮、疠症，再入麝香2-3分。
功效：None
用法用量：每膏1张各下数分，贴伤处。若臁疮、疠症，贴背心即安。
制备方法：共为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：《跌损妙方校释》：此散外治感染创口，有排脓解毒，生肌收口之效，适用于创伤感染创口中后期，以及外科溃疡创口，慢性骨髓炎等。
