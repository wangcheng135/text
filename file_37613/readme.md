序号：37623
名称：三白丹
组成：水银1两，白矾2两，焰消2两。
出处：《张氏医通》卷十五。
主治：梅疮结毒。
加减：None
功效：None
用法用量：每服3分，入飞面3钱，壮者分3服，中者分五服，赢者分7服。每日以上茯苓半斤，捶碎，用水7碗，煮至5碗，去滓，入前丹1服，再煎至3碗，1日服尽，明日如前法再服。2、3日后，喉腭肿痛，齿龈出水，7日毒尽自愈，肿甚者，用黄连1钱，犀角1钱，骨碎补1钱、黑豆1合，煎汤漱之。
制备方法：上3味，纳铁铫中，以厚瓷碗合定，盐泥固济，压定碗足，文火煅3炷香，升在碗内，取出放地1夕，以出火毒，瓷罐收贮，经年方可用之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
