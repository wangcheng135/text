序号：43389
名称：养胃沉香丸
组成：沉香半两，当归半两，广木香半两，白茯苓半两，陈皮（去白）半两，补骨脂半两，肉豆蔻（面裹煨）半两，白豆蔻半两，荜澄茄半两，青皮半两，丁香4钱，白术7钱，桂花3钱，大故脂1两（面炒，去瓤），人参3钱，萝卜3钱，藿香叶3钱。
出处：《医方类聚》卷一○二引《经验秘方》。
主治：胃气虚，脐腹冷痛，大便滑泄，咽膈塞闷，口苦无味，四肢无力，不思饮食，中满痰逆，噫气吞酸，困倦。
加减：None
功效：None
用法用量：每服30-40丸，米饮汤送下。
制备方法：上为细末，生姜自然汁、熟枣肉为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
