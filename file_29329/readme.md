名称：针砂丸
组成：针砂半斤（醋炒红），苍术4两（米泔浸），香附4两（童便浸），神曲2两（炒微黄），茵陈2两（姜汁炒），麦糵（麸炒）2两，芍药1两5钱，当归（酒洗，去头）1两5钱，生地黄1两5钱，川芎1两5钱，青皮（去瓤，炒）1两5钱，陈皮（去白）2两，莪术（醋煮）2两，三棱（醋煮）2两，栀子（去壳，炒）5钱，姜黄5钱，升麻5钱，干漆5钱（炒烟尽）。
出处：《医学正传》卷六引《集验方》。
