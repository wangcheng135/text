序号：32187
名称：扶羸黑白丹
组成：黑丹：麋茸（去床骨皮毛，酒浸一宿，酥炙令黄）、鹿茸（事治如麋茸之法）各等分，白丹：钟乳粉。
出处：《普济方》卷二六五引《家藏经验方》。
主治：年老气血虚耗，精血少不能荣卫经络，精神枯悴，行步战掉，筋脉缓纵，目视茫茫。
加减：None
功效：None
用法用量：None
制备方法：黑丹：共为细末，酒糊为丸，如梧桐子大。_x000D_
白丹：糯米糊为丸。_x000D_
上用此二丹，杂之而服。如觉血少，即多用黑丹。每服30丸，空心、食前温酒或米饮送下。
临床应用：None
各家论述：None
用药禁忌：None
附注：《寿亲养老》：如觉气不足，即多用白丹。
