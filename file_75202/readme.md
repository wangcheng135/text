序号：75218
名称：腽肭脐丸
组成：腽肭脐1对（慢火酒炙令熟），硇砂（研飞）2两，精羊肉（熟切碎烂，研）1斤，羊髓（取汁）1斤，沉香4两，神曲（炒）4两，以上6味，用无灰好酒1斗，同于银器内，慢火熬成膏，候冷入下项药：阳起石（用浆水煮1日，细研飞过，焙干用）2两，人参（去芦）2两，补骨脂（酒炒）2两，钟乳粉（炼成者）2两，巴戟（去心）2两，川芎2两，肉豆蔻（去壳）2两，紫苏子（炒）2两，枳壳（去瓤，麸炒）2两，木香2两，荜澄茄2两，葫芦巴（炒）2两，天麻（去苗）2两，青皮（去白）2两，丁香2两，茴香（舶上，炒）2两，肉桂（去粗皮）2两半，槟榔2两半，蒺藜子（炒）2两半，大腹子2两半，山药1两半，苁蓉（洗，切片，焙）4两，白豆蔻（去壳）1两，大附子（炮，去皮，脐，用青盐半斤，浆水1斗5升煮，候水尽，切，焙干）8两。
出处：《局方》卷五。
主治：五劳七伤，真气虚惫，脐腹冷痛，肢体酸疼，腰背拘急，脚膝缓弱，面色黎黑，肌肉消瘦，目暗耳鸣，口苦舌干，腹中虚鸣，胁下刺痛，饮食无味，心常惨戚，夜多异梦，昼少精神，小便滑数，时有余沥，房室不举，或梦交通，及一切风虚痼冷。
加减：None
功效：补虚壮气，暖背祛邪，益精髓，调脾胃，进饮食，悦颜色。
用法用量：每服20丸，空心温酒送下；盐汤亦得。
制备方法：上为末，入煎膏内搜成剂，于臼内捣千余杵，丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：大腽肭脐丸（《圣济总录》卷一八六）、赐方腽肭脐丸（《杨氏家藏方》卷九）。
