序号：60064
名称：起痿丹
组成：附子（炮、去皮脐）1两，枸杞子（拣去枝梗）1两，肉苁蓉（酒浸，焙干）1两，沉香（不见火）1两，官桂1两，朱砂（别研）1两，熟地黄（酒浸，蒸）1两，母丁香1两，木香（不见火）1两，阳起石（火煅）1两，天雄（炮，去皮脐，或鹿茸亦可）1两，硫黄1两，麝香（别研）1两，腻粉半两，白丁香（少许）。
出处：《医方大成》卷四。
主治：肾经虚败，遂成骨痿，腰脚难举，日加困乏。
加减：None
功效：None
用法用量：每用1丸，以生姜汁火上入药溶化，却用手点药于腰眼上，磨搽至药尽，用至20丸。若有他处瘫痪风疾，加皂角1片，去筋捶烂，姜汁浸1宿，瓦上焙干为末，入前药内，依法用。
制备方法：上为末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
