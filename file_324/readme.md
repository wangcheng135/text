序号：324
名称：五香丸
组成：丁香1分，木香1分，沉香1分，安息香1分，乳香1分，硇砂1分，丹砂（研）1分，肉豆蔻（去壳）1分，桂（去粗皮）1分，京三棱（煨，锉）1分，当归（切，炒）1分，陈橘皮（汤浸去白，焙）1分，槟榔（锉）1分，荜澄茄1分，附子（炮裂，去皮脐）半两，巴豆10粒（去皮心膜，出油）。
出处：《圣济总录》卷四十四。
主治：宿食不消，腹胀膨闷。
加减：None
功效：None
用法用量：每服5丸至7丸，熟水送下；生姜汤亦得。
制备方法：上药先将安息香、乳香、硇砂3味细研，用少洒浸良久，余11味，捣罗丹砂，巴豆，与前3味通研令匀，酒煮面糊为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
