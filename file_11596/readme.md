序号：11602
名称：牛膝丸
组成：苁蓉（酒浸）1两，牛膝（酒浸）1两，防风（去芦头）1两，萆薢（切碎，炒）1两，海桐皮（去粗皮）1两，自然铜（醋淬7次）1两，威灵仙（去土）1两，金毛狗脊（去毛）1两，川乌头1两（生，去皮脐），没药半两（另研），乳香半两（另研），地龙（去土）1两，骨碎补（去毛）4两，木鳖子（去壳，另研）4两。
出处：《御药院方》卷六。
主治：风气下注，脚膝无力，筋骨痠疼；或上攻，头面肿痛，遍身疼倦，壮热。
加减：None
功效：None
用法用量：每服50丸，食前温酒送下。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：接骨丹（《普济方》卷九十八）。
