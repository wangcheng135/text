序号：28987
名称：苏子煎
组成：苏子2升，生姜（汁）2升，白蜜2升，生地黄（汁）2升，杏仁2升。
出处：《外台》卷十引《深师方》。
主治：上气咳嗽。
加减：None
功效：None
用法用量：每服方寸匕，日3次，夜1次。
制备方法：上捣苏子，以地黄、姜汁浇之，绢绞取汁，更捣，以汁浇复绞，如此6-7过，令味尽，去滓，熬杏仁令黄黑，捣令如脂；又以向汁浇之，绢绞取汁，往来6-7过，令味尽，去滓，纳蜜和，置铜器中，于重汤中煎之，令如饴，煎成。
临床应用：None
各家论述：《千金方衍义》：咳嗽日久而客邪未除，蕴化为热，故用姜汁散邪，地黄滋热，且制入苏子顺气味中，更以杏仁熬黑，如大陷胸丸中法，专涤胸中宿垢，纳蜜熬煎，如饴，缓行搜逐而无骤伤中气之患。
用药禁忌：忌芜荑。
附注：None
