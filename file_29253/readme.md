序号：29261
名称：连翘丸
组成：连翘1两半，槟榔1两半，三棱1两半（醋炒），蓬术1两半，肉果1两半（面煨），牵牛1两半，肉桂1两半（去皮），青皮1两半（炒），陈皮1两半。
出处：《医略六书》卷三十。
主治：产后积坚聚结，阻碍肠胃，失其传化之职，下痢青黄，饮食不能遽下，脉紧弦涩者。
加减：None
功效：None
用法用量：每服3钱，米饮煎，去滓温服。
制备方法：上为末，以粥为丸。
临床应用：None
各家论述：方中连翘清热散结气，槟榔破滞降逆气；三棱破血中之气，蓬术破气中之血，二味俱消坚削积之品；牵牛导水下气，肉果固味涩汤，二味乃攻实治虚之品；青皮破滞气以平肝，陈皮利中气以和胃；肉桂温经暖血以化积滞也。粥丸米饮下，使积滞消化，则脾胃健运而饮食无艰下之患，何下痢青黄之不退哉。
用药禁忌：None
附注：None
