序号：41613
名称：金锁正阳丹
组成：砒1两（火煅），巴豆10两（去油），乌头1两（炮），木鳖6个，雄黄半两。
出处：《续本事》卷一。
主治：小肠气痛，滑肠脱肛，心气痛，气块，妇人红脉不行，及产后诸疾，眼多冷泪。
加减：None
功效：None
用法用量：每服1丸，常服空心盐汤送下；小肠气痛，炒茴香酒冷，下木通，煎汤送下；滑肠脱肛，干姜、艾同煎酒温下；心气痛，烧钱淬醋送下2丸；气块，嚼干柿子送下1丸；妇人红脉不行及产后渚疾，当归酒送下；眼多冷泪，盐椒汤送下。
制备方法：上为末，用黄蜡、沥青（好者）各1两半，黄丹1两，朱砂1两半，细研溶热，入前项药末，乘热为丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
