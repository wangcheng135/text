序号：59300
名称：荷叶散
组成：干荷叶2两，鬼箭羽1两，桃仁半两（汤浸，去皮尖双仁，麸炒微黄），蒲黄1两，刘寄奴1两。
出处：《圣惠》卷八十。
主治：产后恶露不下，腹中疼痛，心神烦闷。
加减：None
功效：None
用法用量：每服3钱，以童子小便1中盏、生姜半分、生地黄1分，拍碎，同煎至6分，去滓稍热服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：《医略六书》：产后瘀血凝结挟热，而心神烦闷，恶露不行，故胁腹阵痛不已。荷叶升阳散瘀以除胁痛，鬼羽破血辟邪以止腹疼；刘寄奴通经破血，生蒲黄破瘀通经，桃仁泥破血润燥以开瘀结也。为散，大黄以涤之，生姜以温之；童便以降之，使瘀热消化，则结闷自开，而恶露无不下，何胁腹烦痛之不止哉。
用药禁忌：None
附注：《医略六书》无生地黄，有大黄三分。
