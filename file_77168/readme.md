序号：77184
名称：锁阳固精丸
组成：鹿角霜20两，龙骨（煅）20两，牡蛎（煅）20两，芡实（炒）20两，韭菜子20两，锁阳20两，菟丝子20两，莲子肉20两，牛膝20两，补骨脂（盐水炒）25两，青盐25两，杜仲（炒）25两，苁蓉（炙）25两，大茴香25两，莲须25两，巴戟天（炙）30两，熟地56两，山药56两，山茱萸（炙）17两，茯苓11两，丹皮11两，泽泻11两，知母4两，黄柏4两。
出处：《北京市中药成方选集》。
主治：梦遗滑精，目眩耳鸣，腰膝痠痛，四肢无力。
加减：None
功效：温肾固精。
用法用量：每服1丸，淡盐汤或温开水送下，1日2次。
制备方法：将鹿角霜等22味为粗末，与熟地、山萸同串晒干，共为细末，过罗，炼蜜为丸，重3钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
