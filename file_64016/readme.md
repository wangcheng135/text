序号：64032
名称：渗湿汤
组成：苍术1两，白术1两，甘草（炙）1两，茯苓2两（去皮），干姜（爁）2两，橘红1分，丁香1分。
出处：《局方》卷二（吴直阁增诸家名方）。
主治：因坐卧湿处，或因雨露所袭，或因汗出衣衾冷湿，久久得之，寒湿所伤，身重腰冷，如坐水中，小便或涩或利，大便溏泄；或腰下重疼，两脚疼痛，腰膝或肿或不肿，小便利，反不渴者。
加减：None
功效：None
用法用量：上(口父)咀。每服4钱，以水1盏半，加生姜3片，大枣1枚，煎7分，食前温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：七味渗湿汤（《景岳全书》卷五十四）。《玉案》无干姜，有姜黄。
