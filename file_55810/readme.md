序号：55825
名称：桑白皮汤
组成：桑根白皮1两，苍术（去皮）1两，木通1两，桂（去粗皮）1两，当归（切，焙）1两，黄连（去须）1两，草豆蔻（去皮）3枚，天雄（炮裂，去皮脐）1两半，瞿麦穗1两半，大腹1两半，射干1两半，牵牛子（炒）1两半，桃仁（去皮尖双仁，炒）20枚，郁李仁（去皮，炒）3分，吴茱萸（炒）半两。
出处：《圣济总录》卷六十六。
主治：三焦咳嗽，面目虚浮，不得安卧，饮盛减食。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，入生姜7片，煮取8分，去滓，温服，不拘时候。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
