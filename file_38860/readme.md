序号：38870
名称：泻肝救肺汤
组成：柴胡6g，胆草3g，酒黄芩9g，夏枯草9g，知母9g，麦门冬9g，桔梗6g，牡丹皮6g，赤芍9g，玄参9g。
出处：《张皆春眼科证治》。
主治：白睛俱青。初起羞明流泪，目珠胀痛，痛连眼眶、头部，白睛略呈隆起，色淡紫而暗，继则症状消退，白睛呈青兰色。
加减：瞳神开大者，可加五味子3g，以收敛瞳神；瞳神缩小者，可加青葙子3g以散瞳。
功效：清肝泻火，养阴清肺。
用法用量：None
制备方法：None
临床应用：None
各家论述：方中柴胡、胆草、酒黄芩、夏枯草清泻肝胆郁热，夏枯草且有解毒散结之功；知母、麦门冬养阴清肺；桔梗宣肺散结，以除白睛郁结之邪；牡丹皮、赤芍药活血散瘀，且二味皆有清肝之力；玄参养阴滋肾，以防毒热内攻。
用药禁忌：None
附注：None
