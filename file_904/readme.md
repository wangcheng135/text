序号：904
名称：六神丸
组成：芦荟3钱，大槟榔半两，血竭（别研）半两，使君子（另捣极细）半两，肉豆蔻半两（面裹烧熟，去面不用），京三棱1两。
出处：《普济方》卷三九一引《保婴方》。
主治：小儿久患脾癖，面黄肌瘦，肚大腹胀；或有瘦虫，服诸药不效者。
加减：None
功效：None
用法用量：每服30-50丸，空心、食前温米饮汤送下，1日2次；或熬皂角子汤送下亦可。
制备方法：上除京三棱外，为细末，用乌鸡1个，去肚肠净，将京三棱装在肚内，缝合煮熟为度。不用鸡肉，只用京三棱切作片子，晒干，同前药一处为细末，枣肉为丸，如黄米大。
临床应用：None
各家论述：None
用药禁忌：忌一切肉、硬冷等物。
附注：None
