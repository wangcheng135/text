序号：56181
名称：桔梗散
组成：桔梗（锉，炒）半两，旋覆花半两，贝母（去心）半两，防风（去杈）半两，陈橘皮（汤浸，去白，炒）半两，麦门冬（去心，焙）半两，枳壳（去瓤，麸炒）半两，桑根白皮（锉）1分，人参1分，前胡（去芦头）1分，鳖甲（去裙襕，醋炙）1分，白茯苓（去黑皮）1分，蒺藜子（炒去角）1分，甘草（炙，锉）1分，黄耆（锉）1分，天门冬（去心，焙）1两半。
出处：《圣济总录》卷八十六。
主治：肺劳咳嗽，痰涎涕唾，上气喘急，时发寒热，疼痛；亦治肠风下血，诸气羸弱。
加减：None
功效：None
用法用量：每服3钱匕，沸汤点下，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
