序号：60963
名称：大黄耆汤
组成：黄耆3两，桂心3两，巴戟（去心）3两，石斛（酒浸）3两，泽泻3两，茯苓3两，干姜（炮）3两，防风2两，独活2两，人参2两，天雄（炮，去皮脐）1两，芍药1两，附子（炮，去皮脐）1两，半夏（汤浸7次）1两，细辛（去苗）1两，白术1两，黄芩1两，瓜蒌根1两。
出处：《三因》卷八。
主治：肉虚极，体重怠惰，四肢不欲举，关节疼痛，不嗜饮食，食则咳，咳则右胁下痛，阴引背及肩，不可动转。
加减：None
功效：None
用法用量：每服4钱，水2盏，加生姜7片，煎7分，去滓，食前服。
制备方法：上锉散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
