序号：6494
名称：木瓜丸
组成：侧子（炮，去皮）1两半，安息香1两半，牛膝1两半，肉桂（去皮）2两，虎胫骨（酥炙黄）2两，天麻1两，独活1两，白术1两，石斛（去根）1两，杜仲（去皮，炙）1两，鹿茸（去毛，酥炙黄）1两，陈橘皮（去白）1两，附子（炮，去皮）1两，当归1两，威灵仙1两，海桐皮1两，硇砂（研细，水飞过）1两。
出处：《医方类聚》卷九十五引《神巧万全方》。
主治：风脚膝软弱，行立不得。
加减：None
功效：补脾肾，壮骨髓，利风毒，除缓弱。
用法用量：每服30丸，渐加至40丸，空心、晚食前以豆淋酒送下。
制备方法：上为末，以木瓜去皮，用酒3升，并硇砂入银器中，熬成膏，和前药末，捣为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
