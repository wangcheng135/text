序号：6013
名称：加减四物汤
组成：香附（炒）1钱，当归（酒洗）8分，川芎8分，枳壳（去瓤，炒）8分，柴胡8分，白芍（酒炒）8分，黄芩6分，陈皮6分，三棱（醋炒）6分，莪术（醋炒）6分，熟地黄1钱，白芷5分，玄胡索5分，小茴（酒炒）5分，白术（去芦，炒）5分，青皮（去瓤）5分，砂仁5分，肉桂5分，甘草5分。
出处：《寿世保元》卷七。
主治：室女15-16岁，误食生冷，经脉不通，日夜寒热，手足麻痹，饮食少进，头疼恶心呕吐，腹中忽然结一块痛者。
加减：偏身痛，加羌活。
功效：None
用法用量：水煎，空心热服。
制备方法：上锉作1剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
