序号：43195
名称：三棱煎丸
组成：京三棱半两，蓬莪术（并炮）半两，芫花1分，鳖甲（去裙，米醋炙令焦）半两，淡豆豉2钱，巴豆21粒（去壳），川当归半两，杏仁（去皮尖）1分（炒令赤）。
出处：《活幼口议》卷十七。
主治：婴孩小儿食伤生冷、粘腻、热毒等物，脾胃积滞，久不克化，肚热脚冷，痞癖_x000D_
寒热；及疗癥瘕，中脘膨胀，上膈气壅，心腹不得宣通；诸积滞，食不化。
加减：None
功效：破气行血，和脾开胃。
用法用量：每服20丸，生姜汤送下。
制备方法：上前6味1处，以米醋1碗，煮令干，仍就炒起，更细截，焙为末；次入当归末，又入杏仁、巴豆、淡豆豉和匀，水煮面糊为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：化积丸、消痞丸。
