序号：24794
名称：妇科回生丹
组成：大黄1斤，红花3两，苏木3两，黑豆1斤，黄酒1斤，醋3斤（先将大黄轧成小碎块，红花、黑豆、苏木3味用清水熬汁，熬透去滓滤净，用汁煮大黄，待汁浸入，次将醋倒入，用微火徐徐煮之，须用铲不停地搅动，至稠膏形，再将黄酒倒入，微煮后起入盆内），当归2两，川芎2两，熟地2两，茯苓2两（去皮），炒苍术2两，香附2两（醋制），乌药2两，元胡2两（醋制），桃仁2两（去皮），炒蒲黄2两，川牛膝2两，生白芍5钱，广皮5钱，广木香5钱，三棱5钱（醋制），五灵脂5钱（醋炒），地榆炭5钱，羌活5钱，山萸肉（酒制）5钱，人参3钱（去芦），青皮（醋炒）3钱，白术3钱（麸炒），木瓜3钱，良姜4钱，制没药1钱，制乳香1钱，甘草5钱（以上轧成粉末，和煮制之大黄共和一起拌匀，晒干）。
出处：《全国中药成药处方集》（天津方）。
主治：经闭不通，肚腹疼痛，及产后恶露不净，腹胀头痛。
加减：None
功效：通经活血，化瘀止痛。
用法用量：每服1丸，白开水送下。
制备方法：上为细粉，炼蜜为丸，3钱5分重，蜡皮或蜡纸筒封固。
临床应用：None
各家论述：None
用药禁忌：孕妇及产后下血过多者忌服。
附注：None
