序号：53264
名称：健步丸
组成：石南叶1两，天南星（炮裂）1两，羌活（去芦头）1两，天麻（去苗）1两，薏苡仁1两，防风（去芦头）1两，续断1两，萆薢1两，黄耆（去芦头）1两，当归（去芦头，洗、焙）1两，石斛（去苗）2两，牛膝（切碎，酒浸1宿，焙干）2两，干木瓜4两，威灵仙1两，自然铜1两（烧红醋淬，碎研）。
出处：《杨氏家藏方》卷四。
主治：干湿脚气，腿膝麻痹、冷疼，足下隐痛，行步艰难，下注生疮。
加减：None
功效：None
用法用量：每服50丸，空心、食前温酒或木瓜汤送下。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
