序号：38966
名称：泻荣汤
组成：连翘6分，升麻6分，桔梗5分，生黄芩4分，生地黄4分，黄耆3分，苏木3分，黄连3分，地龙3分，全蝎3分，当归3分，白豆蔻2分，人参2分，甘草1分半，梧桐泪1分，麝香少许，桃仁3个，虻虫（去翅足，炒）3个，水蛭3个（炒尽烟尽）。
出处：《兰室秘藏》卷下。
主治：疠风满面连颈，极痒不任，眉毛脱落。
加减：None
功效：None
用法用量：水2盏，酒1盏，入连翘煎至1盏，去滓，再入白豆蔻2味并麝香等，再煎至7分，稍热，早饭后、午饭前服之。
制备方法：上锉，如麻豆大，除连翘、梧桐泪、白豆蔻另为细末，麝香、虻虫、水蛭3味同为细末，都作1服。
临床应用：None
各家论述：None
用药禁忌：忌酒、湿面、生冷、硬物。
附注：先砭其处，令恶气消尽，后服此药。
