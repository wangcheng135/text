序号：42001
名称：三脘散
组成：大腹皮（酒浸1遍，更以大豆汁洗3遍，焙干用）1两，白术3分，木香3分，甘草（微炙）3分，槟榔3分，陈橘皮（汤浸，去瓤）3分，川芎3分，独活1两，紫苏（并梗）1两，沉香1两，木瓜（干者）1两。
出处：《脚气治法总要》卷下。
主治：脚气，三焦气逆，大便秘滞，胸膈胀憋气，妇人中焦虚痞，两胁气痛，面目手足浮肿，大便秘涩。
加减：None
功效：None
用法用量：每服1分，水2盏，同煎至1盏，去滓温服。取大便通效。
制备方法：上为散。
临床应用：妇人胁痛《妇人良方》：一妇人，中焦虚痞，腹胁胀痛，大便秘结，六脉微弱，更数医服药无效，投此药不终剂而愈。
各家论述：None
用药禁忌：None
附注：大三脘散（《传家秘宝》卷三）、三脘汤（《普济方》卷三五四）。
