序号：3281
名称：九龙丹
组成：枸杞子、金樱子、山果子（又名山楂）、莲肉、佛座须（莲花心也）、熟地黄、芡实、白茯苓、川归各等分。
出处：《医学正传》卷六引丹溪方。
主治：肾水不足，精关不固，男子滑精，女子梦交。白淫。斫伤太过，败精失道，滑泄不禁。
加减：None
功效：None
用法用量：每服50丸，或酒或盐汤送下。如精滑便浊者，服2、3日，溺清如水，饮食倍常，行步轻健。妇人厌产者，2-3服便住孕，如仍欲产，服通利之药。
制备方法：None
临床应用：None
各家论述：《医方考》：精浊者，宜滋肾清心，健脾固脱。是方也，枸杞、熟地、当归，味厚者也，可以滋阴，滋阴则是以制阳光；金樱、莲须、芡实，味涩者也，可以固脱，固脱则无遗失；石莲肉苦寒，可以清心，心清则淫火不炽；白茯苓甘平，可以益土，益土则制肾邪；而山楂肉者，又所以消阴分之障碍也。
用药禁忌：None
附注：本方方名，《医统》引作“九龙丸。”方中山果子，《医统》作“山茱萸肉”。
