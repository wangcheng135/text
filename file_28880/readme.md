序号：28888
名称：苍耳酒
组成：苍耳（和茎叶花实取锉1石，八月收），牛膝根1升，松叶2斗，商陆根2升（白色者），鼠粘根1斗。
出处：《外台》卷三十一引《崔氏方》。
主治：大风恶疾，及一切诸风，乃至骨髓中毒风。
加减：None
功效：补养，益精神，令人充健。
用法用量：上锉讫量之，以水2石5斗，煮取6斗汁，如釜小可分煮之，即分3斗汁，将浸曲1斗2升，高量其曲，加于常法5分，为药力费曲故也。余3斗汁，留将拌（饣贲）料，糯米1石2斗，分作5酘，净淘干漉，以上并大斗。第1酘1日炊4斗米，取药汁9升拌酘熟，细切生地黄3斗，和米下之；第2酘3日炊3斗米，取药汁7升拌，酘熟，与杏仁1斗（去皮尖）碎和捣如泥下之；第3酘5日炊2斗米，取药汁6升半拌，酘熟，取木麻子1斗，捣碎和下之；第4酘7日炊2斗米，取药汁4升半拌，酘熟，取胡麻1斗，捣碎和下之；第5酘9日炊1斗米，取药汁3升拌，酘下之。上以前5酘法，须米消尽，即炊酘之，未必要须隔日，其酒如米少味薄，更炊1-2斗米下之，使味足，然后去糟取清，依常法饮半升，不能者，可量性多少，常使有酒气逼，夜饮最是所宜，此酒纵非风疾饮之。
