序号：42833
名称：保命丹锭子
组成：麝香（拣去毛皮，干研）3钱，辰砂（明透者，水飞净）3钱，冰片（梅花大块）1钱，珍珠（研细末）1钱，琥珀1钱，山豆根1两（熬汁另用），文蛤（洗净，煅）1两，山慈姑（洗去毛皮，净焙）2两，雄黄（鲜明大块，研净）3钱，千金子（白者，去油）1两，红毛大戟（浙江紫大戟为上，北方绵大戟不堪用。去芦根，洗净，焙干为末）1两5钱。
出处：《咽喉脉证通论》。
主治：咽喉、口齿新久肿痛。
加减：None
功效：解诸毒。
用法用量：病轻者1锭，重者连服2锭，磨服。
制备方法：上为末，以糯米粥和山豆根汁打糊为锭，每重1钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
