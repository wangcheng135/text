序号：6840
名称：加减槟榔汤
组成：槟榔1两，陈皮（去白）1两，紫苏叶1两，甘草（炙）半两。
出处：《济生》卷三。
主治：一切脚气。
加减：如脚痛不已者，加宣木瓜，五加皮煎；妇人脚痛，加当归煎；室女脚痛，多是肝血盈实，宜加赤芍药煎。师尼寡妇，亦宜服之。中满不食者，加枳实煎；痰逆成呕者，加半夏煎；小便不利，加木通煎；转筋者，加吴茱萸煎；脚痛大便不通者，用此汤下青木香丸；如更不通，加大黄煎；脚肿而痛者，加大腹皮、木瓜煎；足痛而热者，加地骨皮煎。
功效：None
用法用量：上(口父)咀。每服半两，水1盏半，加生姜5片，煎至8分，去滓温服，不拘时候。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
