序号：34801
名称：麦门冬饮
组成：麦门冬（去心，焙干）2两，白茯苓（去黑皮）3两，人参3两，龙齿3两，远志（去心，焙）3两，甘草（炙）1两，防风（去叉）3两，地骨皮（去土）3两，羚羊角（镑末）1两。
出处：《圣济总录》卷九十三。
主治：心蒸。心中烦躁，手足热疼，欲露其体，惊悸怵惕，咽干虚渴，面色萎黄，失前忘后，妇人血气衰弱，多传此疾。
加减：若大患烦热渴躁者，加淡竹沥1合煎服，若曾经吐血亦治。
功效：None
用法用量：每用5钱匕，以水2盏，加山泽银1分，大枣2枚（擘破），同煎取1盏，去滓，分2次温服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
