序号：40065
名称：肾沥汤
组成：人参1两（去芦头），石斛1两（去根，锉），麦门冬1两半（去心，焙），泽泻3分，桑寄生半两，远志半两（去心），甘草半两（炙微赤，锉），当归半两，熟干地黄1两半，栝楼1两，桂心3分，五味子3分，黄耆1两（锉），白龙骨1两，磁石2两（捣碎，水淘去赤汁），白茯苓1两，地骨皮1两。
出处：《圣惠》卷二十九。
主治：虚劳内不足，便数，四肢瘦，心神烦，不能食。
加减：None
功效：None
用法用量：每服用羊肾1对（切去脂膜），先以水1大盏半，煎肾至1盏，去肾，入药末半两，加生姜半分、大枣3枚，煎至7分，去滓，食前分为2服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
