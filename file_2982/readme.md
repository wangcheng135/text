序号：2982
名称：天真丸
组成：肉苁蓉2两（酒洗、去鳞甲及肉中白筋，净），甘枸杞5钱（酒蒸合研），独活2钱（酒蒸），沉香1钱5分（要将军帽油结者佳，忌火），芡实5钱（炒，研），巴戟5钱（去硬心，酒蒸），朱砂1钱5分（镜面者佳），母丁香1钱5分，菟丝饼5钱，阳起石2钱5分（煅红，盐水淬7次），锁阳3钱5分（酒蒸，焙，研，红者佳），知母7钱5分（去毛，忌铁，酒蒸，焙，研末合之），麝香1分（真当门子佳）。
出处：《良方集腋》卷上。
主治：虚劳。
加减：None
功效：补益，固精，益智。
用法用量：每晚好酒调服。
制备方法：如法制准，饴糖为丸。
临床应用：虚劳：壬年之春，与苏司马共任钦差，羡其不畏劳顿，气足神完。据云弱冠时多病，渐至饮食难受，气短形枯，诸药罔效，见者咸称不寿。承知友访知嵩山老人有天真丸，求药服两月后，身体渐康，饮食大进，复往虔求，始授此方，并以勿轻易予人，勿擅意增改，勿忽略间断为嘱。于是照方制服年余，竞称强健，自后娶室生子，获名出仕，二十余年以来，守而不失，亦从无染疾，极称应验。余即抄方遵服十余年，亦颇安顺。
各家论述：None
用药禁忌：None
附注：须查明六气，依天地阴阳五行配合君臣佐使，每气为君者之药，照分两加一倍。
