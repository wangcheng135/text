序号：28747
名称：十三味当归补虚汤
组成：当归（去芦，酒浸）1两，黄耆（捶破，蜜炙）1两，熟干地黄（洗净）半两，附子（炮，去皮脐）半两，白术（炒）半两，干姜（洗，炮）半两，白芍药半两，人参（去芦）半两，甘草（炙）半两，川芎半两，吴茱萸（去枝梗，汤泡7次，炒）1分，杜仲（去皮，锉，炒去丝）1分，良姜1分（炒）。
出处：《魏氏家藏方》卷十。
主治：妇人诸虚不足，心胸痞闷，四肢倦怠，头目昏眩，心间恍惚，饮食多伤，或时恶寒恶热，一切虚寒者。
加减：None
功效：None
用法用量：上(口父)咀。每服2钱，水1大盏，加生姜5片，枣子2个，煎至5分，去滓，空心服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
