序号：20857
名称：肉鹿茸丸
组成：鹿茸2两（先用草火燎去毛，锉，酒浸3宿，炙令黄色），菟丝子1两（净水淘，酒浸3宿，焙，别捣），茯苓1两（去粗皮），肉苁蓉1两（汤浸，刮去粗皮，酒浸3宿，作片，焙干），紫菀1两（去土，水淘洗，焙干），蛇床子（淘净，酒浸3宿，焙干）1两，黄耆1两（锉碎，慢火炙黄），桑螵蛸1两（锉开，慢火炙黄），阳起石1两（炒令赤色），沙菀蒺藜1两（净拣，微炒，别捣），大黑附子1两（拣，慢火炮，去皮脐），官桂（去粗皮）1两。
出处：《传家秘宝》卷中。
主治：五劳七伤。
加减：None
功效：补阴阳，益精神，止遗沥。
用法用量：每服30丸，空心酒送下。
制备方法：上药并须修制精细，为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
