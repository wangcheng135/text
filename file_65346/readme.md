序号：65362
名称：女金丸
组成：当归140g，白芍70g，川芎70g，熟地黄70g，党参55g，白术（炒）70g，茯苓70g，甘草70g，肉桂70g，益母草200g，牡丹皮70g，没药（制）70g，延胡索（醋制）70g，藁本70g，白芷70g，黄芩70g，白薇70g，香附（醋制）150g，砂仁50g，陈皮140g，赤石脂（煅）70g，鹿角霜150g，阿胶70g。
出处：《中国药典》一部。
主治：月经不调，痛经，小腹胀痛，腰腿酸痛。
加减：None
功效：调经养血，理气止痛。
用法用量：口服：每次1丸，1日2次。
制备方法：上为细末，过筛，混匀，每100g粉末加炼蜜120～150g制成大蜜丸，每丸重9g。
临床应用：None
各家论述：None
用药禁忌：孕妇慎用。
附注：None
