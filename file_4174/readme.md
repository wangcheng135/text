序号：4174
名称：木香人参汤
组成：木香半两，人参半两，黄耆（锉）半两，防风（去叉）半两，牛膝（酒浸，切，焙）半两，甘草（炙，锉）半两，当归（切，焙）半两，荆芥穗半两，巴戟天（去心）半两，半夏（用姜汁制）半两，鳖甲（去裙襴，醋炙）半两，桂（去粗皮）半两，白茯苓（去黑皮）半两，秦艽（去苗土）半两，柴胡（去苗）半两，附子（炮裂，去皮脐）3分，羌活（去芦头）3分，枳壳（去瓤，麸炒）3分，干蝎（全者，炒）3分，肉豆蔻（去壳）3分，熟干地黄（焙）3分。
出处：《圣济总录》卷五十二。
主治：肾脏风毒，攻四肢头面，腰膝生疮，口苦舌干，形容黑瘦，痰涎涕唾，不时心下满闷，肢节怠惰，状如劳疾。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，加葱白3寸(切)，生姜3片，同煎至7分，临熟入酒少许；不吃酒，即入童便少许，去滓，不拘时候温服。
制备方法：如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
