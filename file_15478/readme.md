序号：15485
名称：人参汤
组成：人参1两，槟榔（锉）1两，麦门冬（去心，焙）1两，大腹皮（锉）1两，牡丹皮1两，芍药1两，防己1两，芎1两，草豆蔻1两，白术1两，生干地黄（焙）1两，丁香皮1两，桔梗（炒）1两，枳壳（去瓤，麸炒）1两，茯神（去木）1两，当归（切，焙）1两，甘草（炙）1两，桂（去粗皮）半两，远志（去心）半两，大黄（锉，炒）半两。
出处：《普济方》卷三三三引《圣惠》。
主治：室女思虑太过，心气不足，气结不得宣利，月水不应时，或久不通，或血隔成痨，渐有寒热，肌肉消瘦，不思谷味。
加减：None
功效：通心气，行荣卫，滑经脉。
用法用量：每服3钱，水1盏，加生姜2片，大枣1枚（擘破），同煎至7分，去滓温服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：不可服破血有性之药。
附注：None
