序号：81653
名称：鹤顶丹
组成：明白矾1两，真银朱半两。
出处：《活幼心书》卷下。
主治：结胸，痰症发热，或咽喉如拽锯及痰厥。
加减：None
功效：None
用法用量：每服1丸，研细，茶清调匀温服，或入姜汁少许同炒下。听心上有隐隐微声，结者自散。不动脏腑，不伤真气，无问虚实证皆可投。
制备方法：上为末，用熨斗盛少炭火，坐小瓦盏在上，平抄矾、朱末1钱，入盏中熔化，急刮出就搓成丸。
临床应用：None
各家论述：白矾能化痰解毒，银朱是水银或硫黄炼成，专破积聚，故治结胸，胜陷胸、承气、泻心三药。
用药禁忌：None
附注：二仙丹（《外科传薪集》）。《医学入门》本方用法：上为末。每次一匕，入瓷器内熔化，乘热捻丸，如龙眼核大，薄荷煎汤化下。
