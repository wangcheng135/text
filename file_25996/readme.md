序号：26004
名称：羊肝散
组成：缩砂仁、白芍药、良姜（切，炒）、厚朴（姜制）、橘皮、胡椒半两，破故纸3钱，丁香3钱，白术3钱，木香3钱，吴茱萸（汤洗7次，焙）3钱，肉豆蔻3钱，官桂3钱。
出处：《施圆端效方》引《圣惠》（见《医方类聚》卷一五三）。
主治：诸虚百损，五劳七伤，年深泄痢，久作滑肠，心腹痞闷，块癥疼硬，下虚上热，口疮燥渴，一切阴盛阳虚之证。
加减：None
功效：None
用法用量：每日1剂，分3-5次细嚼，食前米饮送下。
制备方法：上为细末，羊肝3两，薄批开，掺药2钱半，铺葱丝一重，卷定麻扎，精火炙熟。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
