名称：经验肥儿丸
组成：神曲（微炒）5钱，麦芽（炒）5钱，山药5钱，山楂（煮软去核）5钱，水仙子5钱（微炒），陈皮（洗净）3钱，青皮（去瓤）3钱，枳壳（去瓤，麸，炒）3钱，前胡（去芦）3钱，苍术（米泔水夜浸日晒，7日为度）2钱，白术（蛤粉炒）2钱，半夏（姜汁夜浸日晒，7日为度）2钱，使君子（去壳，微炒）2钱，宣黄连（去芦）2钱，砂仁2钱，当归身2钱，人参1钱5分，胡黄连1钱，石莲肉1钱，粉甘草7分。
出处：《医便》卷五引陶国佐方。
主治：小儿疳积。
加减：None
功效：None
用法用量：每次2-3分，或4-5分，陈米汤送下；或稀粥调和米汤，俱可服下。
制备方法：上为细末，用晚米糊为丸，如小粟米大。
