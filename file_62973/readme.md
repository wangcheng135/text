序号：62989
名称：清明丹
组成：好猪肝1具（先用好醋浸半日，尽去瘀血皮膜，用竹刀切作片，研令极细），木贼草1两半（炒），川羌活1两半（鞭节者），苍术1两半（米泔水浸1宿，再炒），当归1两半（去芦，酒浸），远志1两半（去心），甘菊花1两半，荆芥穗1两，枸杞子1两（去蒂，酒浸），川乌头1两（炮，去皮脐），防风1两（去芦），车前子1两（炒），旋复花1两（去枝），地肤子1两（炒），白芷1两，黄芩1两，玄参1两，人参1两（去芦），海螵蛸1两（炙，去壳），雄小黑豆（炒，去壳）1两，白茯苓1两2钱（去皮），川芎2两，甘草（炙）2两，麦门冬2两半（去心），苦胡麻仁半两，蝉蜕半两（去土），草决明子半两（炒），荠莱子（春收者）半两。
出处：《魏氏家藏方》卷九。
主治：上热下冷翳障黑花，睹物茫茫。
加减：None
功效：明目，补肝清胆，还元固本。
用法用量：如肝少，入蜜。每服1丸，食后荆芥茶汤送下。
制备方法：上为细末，用前猪肝为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
