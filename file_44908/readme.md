序号：44923
名称：既济补真丹
组成：大附子2只（生，去皮脐，每只作4片），阳起石（酒煮3日，研如粉）1分，伏火灵砂1分（研细如粉），天雄1对（每只劈作4片，生，去皮，同附子入青盐半两，以水3升同煮，令水尽为度，切，焙干用），磁石（连吸5-7针者，火煅红，醋淬14次，研细如粉，水飞，去赤浊水）半两（别研），鹿茸（爁去毛，酥炙）1两，麋茸（爁去毛，酥炙）1两，舶上茴香（炒）1两，补骨脂（炒）1两，川当归（酒浸1宿，去芦）1两，牛膝（酒浸1宿，去芦）1两，钟乳粉半两，荜澄茄半两，夜明砂半两，肉豆蔻（面裹煨）半两，枸杞子半两，杜仲（去皮，盐炒）半两，丁香半两（不见火），菟丝子2两（淘净，酒浸3宿，焙干）（前药共为细末），腽肭脐（酒浸，研）半两，沉香（不见火）半两，神曲（炒）半两（并为细末），麝香半钱（别研），安息香1分（酒化，别研），羊髓2两（研烂），肉苁蓉1两（先去咸，研令极烂），羊石1对（去筋膜，研烂）。
出处：《魏氏家藏方》卷六。
主治：眼昏力弱，肤腠不密，脏腑不实，真阳虚惫，血弱气耗。
加减：None
功效：补诸虚不足，升肾水以制心火，降心火以暖肾水，交感阴阳，既济关元，生真精，和中焦，使上下升降，百骸安和，温暖脾胃，健壮脚膝，明目聪耳，添精补髓，益寿延年。
用法用量：每服100丸，空心食前，盐汤或温酒送下。
制备方法：将后8味用水2升，同于银石器内重汤熬，不住手搅成膏，和前药末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌食羊肉。
附注：None
