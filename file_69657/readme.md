序号：69673
名称：疏中丸
组成：制半夏2两，人参1两，白术5钱（生用），升麻1两，银州柴胡1两，猪苓1两，化州橘红5钱，泽泻1两。
出处：《伏阴论》卷上。
主治：伏阴病，心下痞塞，按之满闷者。
加减：None
功效：None
用法用量：每服3钱，生姜煎汤送下，日2次，夜1次，以愈为度。小便通利，其痞自消。
制备方法：晒干为末，米汤叠丸，勿令见火。
临床应用：None
各家论述：方以人参、白术益胃健脾，补中培土，使升、柴从九地之下升清于上，猪、泽从九天之上降浊于下，清升浊降，痞塞自开，而转痞为泰之功。在半夏、橘红之善开肺降逆耳，半夏散中有敛，力能敛清散浊，故数倍于橘红也。
用药禁忌：None
附注：None
