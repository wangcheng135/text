序号：44927
名称：既济解毒丹
组成：净黄连5分，黄柏（去粗皮）2钱半，黄芩（净者）2钱半，大黄2钱半，肉桂（去粗皮）2钱，枳壳（锉片，麸炒，清油润透1宿，焙干）2钱，白茯苓（去粗皮）2钱，甘草（生用）7钱。
出处：《活幼心书》卷下。
主治：小儿中热，睡中咬牙，梦语，惊悸不宁；或吐血、溺血，口渴引饮，手足动摇。
加减：None
功效：None
用法用量：每服30-50丸，用麦门冬熟水送下，不拘时候；吐血、溺血，栀子仁煎汤送下；儿小者，薄荷汤磨化投服。
制备方法：上除桂不过火，余药锉，焙，仍同桂共为末，滴水乳钵内熟杵为丸，如绿豆大，带润以水飞朱砂为衣，阴干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
