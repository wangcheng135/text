序号：32098
名称：万应如意痢疾丸
组成：紫苏4两，桔梗4两，前胡4两，陈皮4两，枳壳4两，神曲4两，羌活4两，山楂肉4两，防风3两，藿香3两，白芷3两，厚朴（姜汁炒）3两，川芎3两，薄荷3两，茅术3两，麦芽2两，萝卜子2两，当归（酒拌）2两，甘草2两，姜半夏2两，茯苓2两，广木香1两（另研），砂仁1两6钱（去衣，另研）。
出处：《续补集验良方》。
主治：伤风头痛，发热鼻塞，声重痰嗽，伤食停积，胸中膨胀，腹痛，胃气痛，时疫疟疾，水泻白痢。
加减：None
功效：None
用法用量：每服3钱，轻者2钱，淡姜汤送下；赤痢，白滚汤送下；霍乱吐泻，阴阳水送下。
制备方法：上药炒微脆，为细末，用姜汁糊为丸、如梧桐子大，晒于收贮。
临床应用：None
各家论述：None
用药禁忌：孕妇。痧胀忌服。
附注：None
