序号：24738
名称：安肺宁嗽丸
组成：嫩桑叶1两，儿茶1两，硼砂1两，苏子1两（炒，捣），粉甘草1两。
出处：《衷中参西》上册。
主治：肺郁痰火及肺虚热作嗽；兼治肺结核。
加减：None
功效：None
用法用量：早、晚各服1丸，开水送下。
制备方法：上为细末，炼蜜为丸，3钱重。
临床应用：None
各家论述：肺脏具辟之机，治肺之药，一过于散则有碍于（閤羽），过于敛则有碍于辟。桑得土之精气而生，故长于理肺家之病，以土生金之义也；至其叶凉而宣通，最解肺中风热，其能散可知；又善固气化，治崩带脱肛，其能敛可知，敛而且散之妙用，于肺脏（閤羽）辟之机尤投合也。蓬砂之性凉而滑，能通利肺窍；儿茶之性凉而涩，能安敛肺叶，二药并用，与肺之（閤羽）辟亦甚投合。又佐以苏子之降气定喘，甘草之益土生金，蜂蜜之润肺清燥，所以治嗽甚效也。蓬砂、儿茶，医者多认为疮家专药，不知其理痰宁嗽，皆为要品。且二药外用，能解毒化腐生肌，故内服亦治肺结核，或肺中损烂，亦甚有效验。
用药禁忌：None
