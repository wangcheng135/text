序号：27561
名称：补肺宁嗽汤
组成：人参1钱半，白术1钱半（炒），炙草6分，半夏1钱半（制），陈皮1钱半，葛根1钱半，茯苓1钱半，大枣3枚，生姜3片。
出处：《医略六书》卷十九。
主治：内伤邪陷，寒热咳嗽，脉弦浮者。
加减：None
功效：None
用法用量：水煎，去滓温服。
制备方法：None
临床应用：None
各家论述：内伤脾肺，外邪陷伏而卫气不振，痰涎壅盛，故寒热咳嗽不已焉。人参补肺扶元气，白术健脾燥湿气，炙甘草缓中气以益胃，半夏燥脾湿以化痰，陈皮利气和中，茯苓渗湿和脾，生姜散外邪，大枣益脾元，葛根升阳解肌，能使内伤陷伏之邪从外而出也。水煎温服，俾卫雄邪解，则寒热自退，而痰化气平，咳嗽无不宁矣，此调补升阳之剂，为邪陷寒热咳嗽之专方。
用药禁忌：None
附注：None
