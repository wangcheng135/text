序号：55596
名称：桃仁煎丸
组成：桃仁49枚（汤浸，去皮尖双仁，研如膏），生地黄汁1升，生牛膝汁1升，白蜜5两（以上4味，同于石锅中，慢火熬如稀饧），鳖甲1两半（涂醋炙令黄，去裙襕），京三棱1两（微煨，锉），当归1两（锉，微炒），延胡索1两，干漆1两（捣碎，炒令烟出），芫花半两（醋拌炒干），水蛭49枚（炒令黄），虻虫49枚（去翅足，微炒），槟榔1两，川大黄1两（锉碎，微炒），桂心2两，琥珀1两。
出处：《圣惠》卷七十九。
主治：产后恶血，结成癥块，羸瘦无力。
加减：None
功效：None
用法用量：每服20丸，食前以温酒送下。
制备方法：上为细末，入前煎中搜和，捣200-300杵为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
