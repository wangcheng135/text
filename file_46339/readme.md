序号：46354
名称：活命饮
组成：甘草书1钱，赤芍1钱，白芷1钱，天花粉1钱，贝母1钱，乳香1钱，防风7分，归尾1钱半，皂角刺1钱半，陈皮1钱半，金银花3钱，没药5分，大黄5钱，穿山甲3片。
出处：《医学入门》卷八。
主治：一切痈毒疮疡。
加减：如在背，皂刺为君；在腹，白芷为君；在四肢，金银花为君；在胸，加瓜蒌仁2钱；疔疮，加紫河车草根3钱；便调者，宜去大黄。
功效：排脓，止痛，消毒。
用法用量：用好酒瓦罐煎，密封罐口，勿令泄气，煎熟，随疮上下饮之。服后再饮酒2-3杯，侧卧而睡。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌酸物、铁器。已溃者忌服。
附注：None
