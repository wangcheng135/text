序号：77901
名称：蟾酥丸
组成：苍术3两6钱，生军6两，麻黄3两，天麻3两，沉香5钱，檀香1两，丁香6钱，广木香1两5钱，麝香3钱，雄黄3两，朱砂1两2钱，甘草2两4钱，蟾酥6钱。
出处：《饲鹤亭集方》。
主治：心腹暴痛，兼受四时不正之气，山岚瘴毒，癫狂迷乱，五痢八疳。
加减：None
功效：祛暑辟邪，利湿开窍。
用法用量：None
制备方法：上为末，将蟾酥酒化为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：蟾酥痧药丸（《全国中药成药处方集》南京方）。《全国中药成药处方集》（南京方）本方用法：用高粱酒泛为小丸，每分约二十粒，朱砂为衣，放于烈日下晒燥，趁热装蜡袋中，打光至亮为度。每服一分至二分，开水送下。
