序号：47664
名称：千捶膏
组成：蓖麻子（去壳，取净白肉）1斤，大天南星1两（研），乳香2两，没药2两（制好，去油，研细），急性子2两（研），银朱2两，血竭2两（研），上元寸3钱。
出处：《疡科纲要》。
主治：痈疡高肿，将欲成脓，又阳发初起，来势迅速，乳疡乳发，胸臂诸痈。
加减：None
功效：None
用法用量：痈疡内挟肝胆相火，不能用五瘟丹及温煦薄贴者，宜以此膏合清凉薄贴用之。未成可消；己成即提脓高肿，易于针溃，捷验异常。
制备方法：上先以蓖麻子石臼中捶极细，不见白星，如酱，乃入后7味细末，缓缓杵匀，瓷器密收听用。
临床应用：None
各家论述：None
用药禁忌：None
附注：此方以蓖麻为君，银朱、急性子为佐，消肿清解，捷于影响。阳发疡患初起，贴之消者八九。恒有一贴此膏，而肿块即退，移于膏药之旁者，以此知是膏并可作移毒用。古书称蓖麻能堕胎云云，亦以其流动，而过甚言之。然此膏恒贴孕妇痈疡，未有因此堕胎者，以此知古说之未可尽信也。
