序号：30239
名称：豆蔻香连丸
组成：黄连（炒）3分，肉豆蔻1分，南木香1分。
出处：《小儿药证直诀》卷下。
主治：泄泻，不拘寒热赤白，阴阳不调，腹痛，肠鸣切痛。
加减：None
功效：None
用法用量：每服10-30丸，食前米饮汤送下，日夜各4-5服。
制备方法：上为细末，粟米饭为丸，如米粒大。
临床应用：None
各家论述：《小儿药证直决类证释义》：此方用黄连苦降以清热，木香芳烈以行滞，肉豆蔻温涩以止泻。寒热并投，通涩兼施，故能统治一切泄利，尤适宜于里热气滞而兼久利滑脱之证。若湿热瘀积而见里急后重之滞下，应通而不应涩，此方肉果温涩，不宜早投。
用药禁忌：None
附注：豆蔻木香丸（《普济方》卷三九六）。
