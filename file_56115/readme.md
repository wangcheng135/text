序号：56130
名称：桔梗汤
组成：桔梗（炒，锉）半两，旋覆花半两，好贝母（去心）半两，防风（去杈）半两，陈橘皮（汤浸，去白，炒）半两，麦门冬（去心，焙）半两，枳壳（去瓤，麸炒）半两，桑根白皮（锉）1分，人参1分，前胡（去芦头）1分，鳖甲1分，白茯苓（去黑皮）1分，蒺藜子（锉，去角）1分，甘草（炙，锉）1分，黄耆（锉）1分，天门冬（去心，焙）1两半。
出处：《普济方》卷二十七。
主治：肺劳咳嗽，唾痰涎，上气喘急，时发寒热，疼痛；亦治肠风下血，诸气羸弱。
加减：None
功效：None
用法用量：每服3钱，沸汤调下，不拘时候。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
