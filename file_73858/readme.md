序号：73874
名称：硼砂煎丸
组成：硼砂1两（不夹石者，研），干漆末1两，京三棱末1两，巴豆1两（去皮心，研，纸裹压去油。以上4味，用头醋5升，于瓷器内，以慢火熬3日成膏，入后药末），川大黄1两（锉碎，微炒），附子1两（炮裂，去皮脐），青橘皮1两（汤浸，去白瓤，焙），香墨1两，当归1两（锉，微炒）。
出处：《圣惠》卷四十九。
主治：一切积滞气，胸膈不利，饮食难化，心腹结硬，欲成癥瘕，面色萎黄，脐腹多痛。
加减：None
功效：None
用法用量：每服3丸，以温酒送下。
制备方法：上为末，入前煎中相和为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“硇砂煎丸”。方中硼砂，《普济方》引作“硇砂”。
