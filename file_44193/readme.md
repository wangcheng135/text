序号：44208
名称：三痟丸
组成：好黄连（去须，细末）不计多少。
出处：《本事》卷六。
主治：消渴。
加减：None
功效：None
用法用量：每服30-40丸，以冬瓜汁煎大麦仁汤送下。寻常渴，只1服。
制备方法：锉冬瓜肉，研裂，自然汁和黄连末做成饼子，阴干；再为末，再用汁浸和，如是7次，即用冬瓜汁为丸，如梧桐子大。
临床应用：None
各家论述：《本事方释义》：川连气味苦寒，入手少阴；冬瓜气味微寒，入手太阳、手足阳明。此治三消之证致消渴不止者，皆由火气上炎，津液被劫，以苦寒、甘寒之味，制其上炎之火，而津液自振矣。
用药禁忌：None
附注：瓜莲丸（《直指》卷十七）、三消丸（《赤水玄珠》卷十一）。
