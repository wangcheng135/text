序号：42866
名称：保和汤
组成：知母（蒸）5分，贝母2钱，天冬（去心）3钱，麦冬（去心）1钱，苡仁5钱，北五味10粒，甘草8分，桔梗8分，马兜铃8分，百合8分，阿胶（蛤粉炒成珠）8分，薄荷2分。
出处：《医学心悟》卷三。
主治：肺痿久咳不已，时吐白沫如米粥者。肺津不足，痰凝火郁，肺痿咳嗽。
加减：虚者，加人参。
功效：清火降痰。
用法用量：水煎，加饴糖1匙，温服。
制备方法：None
临床应用：None
各家论述：《血证论》：肺经之津足，则痰火不生，而气冲和。若津不足，则痰凝火郁，痿咳交作，而气失其和矣。方用饴糖、甘草、阿胶，补胃以滋肺津；复加清火、祛痰、敛浮、解郁之品，凡以保护肺金，使不失其和而已。
用药禁忌：None
附注：方中天冬用量原缺，据《血证论》补。
