序号：56797
名称：消疔丸
组成：松香20两（用桑柴灰煎汁，澄清，入松香煮烂取出，纳冷水中，少时再纳灰水中煮，以色白如玉为度），百草（取烟煤用，如以别种柴烟煤不验）5两，铜绿（研细，过绢筛，再研，无声为度）5两，乳香3两，没药（2味去油）3两，白蜡2两（切为粗末），黄蜡10两（刮为粗片），麻油6两。
出处：《华氏医方汇编》卷二。
主治：疔疮。
加减：None
功效：None
用法用量：临用以1丸呵软，捻扁贴患处，倾刻止痛，次日肿消即愈。已走黄者贴之，亦霍然。
制备方法：择吉净室，先将麻油入锅煎滚，次下松香稍滚，三下白蜡稍滚，四下黄蜡，五下乳香，六下没药，七下铜绿，八下百草霜，滚过数次，于锅内冷透，搓成条子，为丸如桂圆核大，藏瓷器内。
临床应用：None
各家论述：None
用药禁忌：贴后忌荤腥、辛辣、沸汤、大热、生冷，发物、面食、豆腐、茄子、黄瓜、酒；忌水洗、恼怒、房事。
附注：神效消疔散、釜墨膏（《青囊秘传》）。
