序号：73805
名称：痰核瘰疬膏
组成：猫头骨牙爪1付（火煅存性），蜣螂虫（炙）5钱，磁石（醋煅）5钱，乳香1钱，没药1钱（去油），生明矾5钱（入雄猪脚爪壳内煅存性），海藻1两，大贝母1两，蓖麻子肉5钱。
出处：《种福堂方》卷二。
主治：瘰疬。
加减：None
功效：None
用法用量：临用摊贴。凡去滓后入细药时，仍用青州丹，少加松香、黄蜡，看老嫩得宜，方入猫头等末，始易成膏。如已穿破，再取客厕梁上尘加入。治未穿破者，贴之即消。
制备方法：用麻油4两，同上海、贝、麻3味，熬至滴水不散，滤去滓，入乳、没再熬，将稠离火，乘滚入猫头、蜣螂、磁石、飞矾搅匀，炖冷水中去火气，乘软取起打条。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
