序号：4049
名称：木瓜丸
组成：花木瓜（切下顶作盖，去瓤）1个，附子（炮，去皮脐，为细末）1只。
出处：《魏氏家藏方》卷八。
主治：一切脚气，腿膝疼痛。
加减：None
功效：None
用法用量：每服30-50丸，空心用温酒送下。
制备方法：上将附子末安在木瓜内，再以熟艾实之，将顶盖之，用竹签签定，复以麻线缚之，用米醋不拘多少，于瓷器内煮烂，石器中烂研成膏，却用2-3只碗，以匙摊于碗内，自看厚薄得所，连碗覆于焙笼上慢火焙，时时以手摸，如不沾手以匙抄转，依前摊开，勿令面上焦干，恐成块子。如此数次看干湿得所，方可为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
