序号：59775
名称：调经汤
组成：川芎7分，当归1钱，生地1钱，益母草1钱，白芍8分，香附8分，丹皮8分，茯苓8分，甘草3分，姜3片，枣1个。
出处：《女科秘要》卷四。
主治：因经闭，败血停积五脏，流入四肢而作浮肿者。
加减：如血热先期及血热过期，紫黑或块，加黄连7分（酒炒），血寒过期，加煨姜、肉桂各3分；临期正行作痛，加元胡、青皮各8分；临行继断不来，积块刺痛，加红花、苏木、桃仁各5分；经水过多，加黄芩1钱、蒲黄（炒）8分；经来饮食不思，加白术8分、陈皮、砂仁各5分；肥人多痰，赤白带下，加南星、苍术各8分；气虚血弱，四肢虚软，面无颜色，加人参、黄耆各5分。
功效：None
用法用量：空心温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
