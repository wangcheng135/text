序号：26561
名称：八宝玉枢丹
组成：山茨菰（俗名金灯笼。花似灯笼，色白，上有黑点，结子三棱；二月开花，三月结子，四月初苗枯即挖，迟则苗烂难寿。极类有毒老鸦蒜，但蒜无毛，茨菰有毛包裹结瓣。去皮洗极净，焙）2两，川文蛤（一名五倍子。打破，洗刮净，焙干）2两，红芽大戟（杭州紫大戟为上，江南土大戟次之。去芦，洗极净，焙干）1两5钱，千金子（一名续随子，去壳，拣色之白者，用纸包裹，更换研数十次，去尽油，以色白成霜，为末）2两，真麝香（拣净白毛皮壳，细研）2两，金箔10帖，牛黄3钱，珍珠3钱，琥珀3钱，朱砂3钱，雄黄3钱，乳香3钱，没药3钱。
出处：《摄生秘剖》卷三。
主治：None
加减：None
功效：解诸毒，疗诸疮，利关窍，治百病。
用法用量：每服1锭。病势重者连服，通利1-2行无妨，用温粥补住。内可以服，外可以敷。一切饮食药毒、蛊毒、瘴气、恶菌、河豚,吃死牛马驰羸等诸毒，并用凉水磨服；南方蛊毒、瘴疠伤人，才觉意思不快，即磨服1锭，或吐或痢随手便愈；诸蛊肿胀大，麦芽汤送下；痈疽、发背。对口疮、天蛇头、无名疔毒等诸恶疮，诸风瘾疹、赤肿未破时，及痔疮，并用无灰酒磨服，再用凉水调涂疮上，日夜各数次，觉痒立消。已溃出脓血者，亦减分数；阴阳二毒，伤寒心闷，狂言乱语，胸膈壅滞，邪毒未发，证宜下者，及瘟疫喉闭、缠喉风，凉水薄荷小叶磨服；传尸痨瘵，用檀香汤磨服；心气痛并诸气，用淡酒或淡姜汤磨服：久近疟疾，临发时东流水煎桃树枝汤磨服；赤白痢疾、泄泻，肚腹急痛，霍乱、绞肠痧等证，及诸痰症，并用薄荷汤磨服；男妇急诸癫邪，喝叫乱走，鬼交、鬼胎、鬼气，狂乱失心，羊儿猪癫等风，中风、中气，口眼歪斜，牙关紧急，语言蹇涩，筋脉挛缩，骨节风肿，手足腰腿周身疼痛，行步艰辛及诸痫症，并用暖酒磨服；自缢溺水已死，心头暖者，惊死或鬼迷，死未隔宿者，俱冷水磨灌下；年深日近头痛，或太阳痛者，用酒入薄荷叶研烂，敷纸贴太阳穴上；牙痛酒磨涂及含少许，良久吞下；小儿急慢惊风，五疳五痢，脾病黄肿，瘾疹疮瘤，牙关紧急，并用蜜水薄荷小叶磨下，及搽，量儿大小，1锭作2-3服；妇人女子经水不通，红花煎汤送下；打扑伤损，炒松节淡酒磨服；汤火伤，东流水磨涂；恶虫疯犬所伤，冷水磨涂，淡酒磨服。
制备方法：宜端午、七夕、重阳日将前药各为细末，搅和数百次，极匀，乃重罗1-2遍，方用糯米浓饮调和，于木臼内杵数千下，极光润为度，每锭1钱。
临床应用：None
各家论述：None
用药禁忌：孕妇及脾泄勿服。
