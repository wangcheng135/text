序号：41593
名称：金锁十益大安丸
组成：顶熟地1斤（人乳拌蒸9次），顶生地6两（酒洗），菟丝子5两（酒煮吐丝），黄耆（蜜炙）3两，麦冬（去心）3两，天冬（去心）3两，女贞子（人乳拌蒸9次）3两，白术（米泔浸，炒）3两，白芍（酒炒）3两，芡实3两，归身（酒洗）3两，杏仁3两（炒），杜仲（盐水炒断丝）3两，茯苓3两（乳拌蒸，晒），萸肉2两，山药（炒）4两，牡蛎4两（童便和，黄泥裹煨），鱼胶8两（蛤粉炒）（以上俱如法制过，照分量称准，为末听用），龙眼肉1斤，青核桃肉1斤（2味捣如泥），莲子8两（去心，焙研），真木枣肉5两（以上4味共捣匀）。
出处：《惠直堂方》卷一。
主治：None
加减：若加人参1两更妙。
功效：种子。
用法用量：早、晚各服5钱。男、女俱可用，至半料必孕。
制备方法：将药末和，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
