序号：58645
名称：秘传固本丸
组成：人参2两，生地2两，熟地2两，麦冬（去心）2两，菟丝子（酒制）2两，枸杞子2两，覆盆子2两，小茴（盐炒）2两，五味子2两，肉苁蓉2两，巴戟2两，山药2两，山茱萸（去核）2两，牛膝（酒制）2两，杜仲（姜炒丝尽）2两，当归（酒制）2两，茯苓（去皮）2两，川椒（去目合口，炒）2两，木通2两，黄耆（蜜炙）2两，木通1两，官桂5钱，黄柏4两（酒炒），知母（去毛，酒炒）4两，破故纸（炒）1两，虎胫骨（酥炙）1两。
出处：《便览》卷三。
主治：诸虚百损，腿膝无力。
加减：None
功效：生精血，补五脏，除百病，美容颜，平补气血，补下元诸虚。
用法用量：每服70-80丸，空心以盐汤或盐酒送下。
制备方法：上各制净，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
