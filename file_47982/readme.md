序号：47997
名称：神异痰火膏子
组成：生地黄4斤，熟地黄1斤，核桃肉1斤，红枣肉1斤，莲肉1斤，柿霜1斤，山茱萸（去核）1斤，甘枸杞4两，胡黄连4两，人参4两，知母4两，贝母4两，银柴胡4两，诃子肉4两，牡丹皮4两，地骨皮4两，山药4两，黄耆4两，黄芩4两，黄柏4两，陈皮4两，白沙参4两，杏仁（去皮尖）4两，桔梗4两，黄菊花4两，五味子4两，白芍4两，栀子4两，香附4两，松花4两，天门冬（去心）4两，麦门冬（去心）4两，厚朴（姜炒）4两，枳壳（去瓤）4两，当归4两，白术（去芦）4两，桑白皮4两，天花粉4两，瓜蒌仁4两，白茯苓4两，乳香4两，没药4两，玄胡索4两，玄明粉4两，鹿角胶4两，粟壳4两，柏子仁4两，梨汁5斤，藕汁2斤，五加皮6两。
出处：《鲁府禁方》卷一。
主治：痰火。
加减：None
功效：None
用法用量：每日早晨以滚水和食3钱，不拘食之前后。仍将诸药滓为末，炼蜜为丸，如梧桐子大。每服50丸，滚水送下，不拘时候。
制备方法：上用甜水1大锅，将生熟地黄煮熬稠浓，至10碗收起，又用水1大锅再煮熬，待稠浓至10余碗汁时再收起，将二黄用冷水磨细，绢袋滤滓；将上煎调药下锅，用水1大桶，煮1次，收水10碗，如此将药煮熬5次，取水50碗；将前二黄汁与诸药汁和匀，用细绢袋滤去滓，以净药水下铜锅，用文武火熬成膏子，下蜂蜜5斤熬一二沸，再下松花、玄明粉、白矾、乳香、柿霜、梨、藕，已成膏子熟美，用瓷罐盛之，勿令泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
