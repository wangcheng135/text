序号：37789
名称：拔疔膏
组成：野菊花1钱5分，山慈姑1钱5分，升麻（瓦炙）1钱5分，血竭1钱5分，天花粉1钱，七叶1枝花，紫花地丁3钱，木耳3钱，皂角刺（各瓦炙）3钱，朱砂（水飞，净）3钱，川贝母（去心）3钱，知母（瓦炙，或用黄酒煮透，焙干亦可）3钱，蟾酥3钱（酒化，不见火），生甘草5分，麝香5分，蓖麻子肉1两（去壳衣，捣烂用）。
出处：《经目屡验良方》。
主治：疔疮。
加减：None
功效：None
用法用量：用时先将银针刺破疔根，入此膏少许，掩以膏药1对。周时疔自拔出矣。
制备方法：上药除麝香、蟾酥、血竭、蓖麻子肉、朱砂、生甘草6味，余概用瓦炙存性，同前药为极细末，同蓖麻肉捣烂成膏。如干，加山东胭脂，如无，麻油亦可。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
