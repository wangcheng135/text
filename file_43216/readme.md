序号：43231
名称：养血当归膏
组成：当归4斤，川芎2两，党参4两，白芍4两，熟地4两，黄耆4两，茯苓4两，阿胶15斤，炙甘草2两。
出处：《全国中药成药处方集》（武汉方）。
主治：面色萎黄，肌肉消瘦，及妇女月经失常。
加减：None
功效：None
用法用量：每服1茶匙，1日3次。
制备方法：取当归粗粉4斤，加60％乙醇8斤，浸1星期，过滤，将乙醇收回；再将当归残滓与川芎、党参、白芍等药混合入锅，照量加水五倍，用常温（70℃）浸渍4小时，过滤，照例2次；最后滤液，加阿胶15斤，与当归液合并浓缩，再加蔗糖13斤，使成稠膏即得。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
