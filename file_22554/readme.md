序号：22562
名称：吴茱萸丸
组成：吴茱萸2分（汤浸7遍，焙干微炒），半夏3分（汤洗7遍去滑），细辛1两，紫菀1两（去苗土），甘草半两（炙微赤，锉），附子1两（炮裂，去皮脐），旋复花半两，前胡1两（去芦头），干姜3分（炮裂，锉），人参3分（去芦头），熟干地黄2两，白术1两，赤茯苓1两，当归3分（锉碎，微炒），赤芍药3分，桂心1两，诃黎勒1两半（用皮），木香1两。
出处：《圣惠》卷四十八。
主治：寒疝。
加减：None
功效：None
用法用量：每服20丸，以生姜汤送下，不拘时候。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
