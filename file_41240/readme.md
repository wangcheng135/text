序号：41255
名称：金汞灵丹
组成：金箔2钱半（以火煅过，用法酒淬50次为度，细剪如丝），水银半两，辰砂半两，好硫黄1两，自然铜4两（捣为末，用锅子1个盛之，瓦盖不封，于地炉内以炭一秤煅之，火尽放冷，取出研细，水飞候干，却同4味一处入乳缽内，研如面细，不见水银星子为度），生犀角半两，羚羊角3分（并镑），干蝎（炒）半两，白僵蚕（炒，去丝）半两，天南星（炮，去皮脐）半两，藿香叶半两，白花蛇3两（法酒浸软，去皮骨，培干秤），乌蛇3两（法酒浸软，去皮骨，焙干秤），官桂1两，白术3分，白芷3分，川芎3分，破故纸（炒）3分，葫芦巴（炒）3分，白附子（炒）3分，荜澄茄3分，羌活（去芦）3分，当归（炒，一云酒浸）3分，牛膝（酒浸1宿，焙干）3分，防风（去芦）3分，鹿茸（火燎去毛，切片涂酥炙）3分，附子（炮，去皮脐）1两1分，川乌（炮，去皮尖）1两1分，沉香半两，天麻1两半，木香3钱3分，安息香半两（别研）。
出处：《续易简》卷四。
主治：卒暴中风，奄忽不省，手足軃曳，口面斜，舌强痰盛，搐搦战掉，或角弓反张，目睛直视，口噤闭绝。
加减：None
功效：滋养五脏，补益真元，通流关节，祛逐风邪，强健筋骨，壮者不老。
用法用量：每服1丸，空心细嚼，温酒送下，1日3次。如中风数年不能步履，服至10丸复旧；新得中风，至3服可无事；常服半丸。
制备方法：上为末，却连前5味末拌和，入安息香膏搜和，再入臼中杵五百下，每1两作10丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
