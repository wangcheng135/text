序号：4907
名称：木香煮散
组成：木香1分，人参1分，白茯苓1分（去黑皮），白术1分，半夏1分（汤洗7遍，炒），厚朴1分（去粗皮，将厚朴入生姜1分同捣，炒干），干姜半两（炮），桂半两（去粗皮），枳实半两（去瓤，麸炒），甘草（炙，锉）半两，陈橘皮（汤浸，去白，焙）1两，槟榔（锉）1枚，草豆蔻（去皮）2枚，诃黎勒（煨，去核）5枚。
出处：《圣济总录》卷四十六。
主治：脾胃不和，不能饮食，心胸痞闷，口淡无味。
加减：None
功效：调顺中焦，兼解伤寒。
用法用量：每服3钱匕，水1盏，煎至7分，去滓热服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
