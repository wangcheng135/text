名称：犀角散
组成：犀角（镑）半两，羌活（去芦头）半两，桑螵蛸（炒）半两，白鲜皮半两，地骨皮半两，蔓荆实（去皮）半两，丹砂（研）半两，酸枣仁半两，乌头（炮裂，去皮脐）1两，白僵蚕（炒）1两，鹿角胶（炒令燥）1两，薏苡仁1两，白附子（炮）1两，当归（切，焙）1两，芎1两，人参1两，牛黄（研）1分，麝香（研）1钱。
