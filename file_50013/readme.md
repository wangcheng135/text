序号：50028
名称：茯神丸
组成：茯神（去木）1两，龙骨1两，龙齿1两，龙角（3味去土，1处研）1两，龙胆（去苗土）1两，铁精（捣研，入前3味同研）1两，蔓荆实（揉去白皮）1两，干姜（炮）3分，人参3分，远志（去心）3分，黄连（去须，炒）3分，大黄（锉，醋炒）1两半，芎藭半两，白芷半两，当归（切，焙）半两，黄芩（去黑心）半两，桂（去粗皮）半两。
出处：《圣济总录》卷十五。
主治：风癫瘛疭，神魂不定。
加减：None
功效：None
用法用量：每服15丸，渐加至20丸，空心、日午食前蜜汤送下。
制备方法：先将13味为末，入别研者4味，和令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
