序号：47524
名称：神子荆芥散
组成：黄芩3钱，栀子3钱，羌活3钱，细辛3钱，车前子3钱，升麻3钱，防风2钱半，荆芥2钱半，桑皮2钱半，知母2钱，连翘2钱，川芎1钱半，甘菊花2钱，旋覆花1钱，甘草半两，柴胡3钱半，赤芍药4钱。
出处：《普济方》卷八十五引《海上方》。
主治：诸色眼疾。
加减：眼血多，去羌活，加独活3钱、生地黄2钱半、当归（去须）2钱半；火烧痛者，乃热多，加大黄2钱（锉），一服后微利去热；眼青睛上生点者，加白蒺藜（炒香，用砖2片，磨去刺），木贼（去节）1两（去节）针刺痛者，乃是气眼，加枳壳2钱、紫苏子3钱半；眼角痛者，加白芷2钱、石膏末3钱，同前药煎服。
功效：None
用法用量：水煎服。
制备方法：上锉散，分为5股。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
