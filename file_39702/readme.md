序号：39712
名称：经进萃仙丸
组成：沙苑蒺藜8两（淘净，隔纸微焙，取细末4两入药，留粗末4两同金樱子熬膏），山茱萸（酒蒸，去核取净）4两，芡实4两（同枸杞捣），白莲蕊4两（酒洗，晒干，如无，莲须代之），枸杞子4两，菟丝子（酒浸，蒸烂，捣焙）2两，川续断（去芦，酒净）2两，覆盆子（去蒂，酒浸，9蒸9晒，取净）2两，金樱子（去净毛子）2两。
出处：《张氏医通》卷十四
主治：遗精。
加减：None
功效：None
用法用量：每服80丸，渐加至100丸，空腹淡盐汤送下。
制备方法：上为细末，以所留蒺藜粗末同金樱子熬膏，入前细末拌匀，再加炼白蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
