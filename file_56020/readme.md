序号：56035
名称：桑落酒
组成：糯米粉5升，白曲5斤，桑条末5斤（取桑枝法：于2-3月看朝东桑树发芽时，收取嫩枝吊檐下阴下，寸断，以川蜜拌匀蒸之，阴干后又拌又蒸，如此3次，晒极干听用）。
出处：年氏《集验良方》卷三。
主治：None
加减：None
功效：清心明目。
用法用量：将粉面以冷水和作饼子，用皮树叶包好，吊屋檐下，过7日取下，看内有菊花心听用。先用上熟糯米5斗，泪洗入甑内蒸熟，用桑条末5斤，前米曲2斤8两，共拌入饭内和匀，中按一窝，再以好酒洒上，不用火酒，过7日，即有酒浆潮来，再养7日榨出，再入黄蜡1斤，装坛内密封，重汤煮过3炷香，取起任饮。
