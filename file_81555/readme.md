序号：81571
名称：镇宫丸
组成：代赭石（火煅，醋淬7次）2两，紫石英（火煅，醋淬7次）2两，禹余粮（火煅，醋淬7次）2两，附子（醋炙）2两，阳起石（煅红，细研）1两，芎1两，鹿茸（燎去毛，醋蒸，焙）1两，茯神（去木）1两，阿胶（锉，蛤粉炒成珠子）1两，蒲黄（炒）1两，当归（去芦、酒浸）1两，血竭（别研）半两。
出处：《济生》卷十。
主治：妇人崩漏不止，或下五色，或赤白不走，或如豆汁，或状若豚肝，或下瘀血，脐腹胀痛，头晕眼花，久久不止，令人黄瘦，口干心烦不食。
加减：None
功效：None
用法用量：每服70丸，空心米饮调下。
制备方法：上为细末，用艾煎醋汁，打糯米和丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
