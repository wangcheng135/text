序号：66477
名称：小儿至宝丹
组成：厚朴（姜制）4钱，炒苍术3钱，制半夏4钱，桔梗3钱，黄连3钱，藿香叶3钱，山楂5钱，陈皮4钱，芥穗4钱，广木香2钱，砂仁4钱，炒麦芽3钱，花粉3钱，枳壳3钱（麸炒），甘草3钱，大黄3钱，木通3钱。
出处：《全国中药成药处方集》（天津方）。
主治：身体发热，腹痛便泄，呕吐胀满，赤白痢疾。
加减：None
功效：解热健胃，止呕止泻。
用法用量：1至2岁每次服1丸，周岁以内酌减，白开水化下。
制备方法：上为细末，兑朱砂面2两、冰片2钱。以上研细和匀，炼蜜为丸，5分重，蜡皮或蜡纸筒封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
