序号：30689
名称：鸡骨丸
组成：乌鸡骨1具（汤浸，炙令微黄），川大黄1两（锉碎，微炒），枳实半两（麸炒微黄），鳖甲1两（涂醋炙令黄，去裙襴），泽泻1两，柴胡1两（去苗），桔梗1两（去芦头），人参1两（去芦头），赤芍药1两，黄芩1两，防葵3分，（庶虫）虫5枚，（微炒令黄），杏仁3分（汤浸，去皮尖双仁，麸炒微黄）。
出处：《圣惠》卷八十八。
主治：小儿羸瘦，腹内有癖气，胁下坚满，时有腹痛，虽食不成肌肉。
加减：None
功效：None
用法用量：4-5岁儿，每服10丸，以粥饮送下，1日2次。
制备方法：上为末，炼蜜为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
