序号：3838
名称：心肾丸
组成：牛膝（去苗，酒浸）2两，熟地黄（洗，再蒸）2两，苁蓉（酒浸）2两，鹿茸（燎去毛，好酒涂炙）1两，附子（炮，去皮脐）1两，五味子（去枝）1两，人参（去芦）1两，远志（去苗，甘草水煮，捶去骨）1两，黄耆（蜜炙）1两，白茯神（去木）1两，山药（炒）1两，当归（去芦，酒浸）1两，龙骨（煅）1两，菟丝子（酒浸、蒸，碾成饼）3两。
出处：《医方大成》卷四引《究原方》。
主治：水火不既济，恍惚多忘，心忪盗汗，夜梦惊恐，目暗耳鸣，悲忧不乐，腰膝缓弱，四肢痠疼，小便数而赤浊，精滑梦遗。
加减：None
功效：养心神，补气血，生津液，进饮食，安神定志。
用法用量：每服50、70丸，空心，食前枣汤送下。常服。
制备方法：上为细末，用浸药酒，煮薄面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
