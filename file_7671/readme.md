序号：7671
名称：二防饮
组成：人参1钱，白术1钱，黄耆1钱，甘草（炙）5分，川归1钱，川芎1钱，芍药1钱，熟地黄1钱，防风7分，防己7分，羌活7分，牛膝7分，杜仲（姜汁拌炒）1钱，萆薢1钱，附子（童便浸3日，去皮脐）7分（冬月1钱）。
出处：《医学正传》卷三。
主治：鹤膝风，因痢后感冒寒湿，或涉水履霜，以致两足痛痹，如刀劙虎咬之状，膝膑肿大，不能行动。
加减：None
功效：补气补血，驱风散湿，行经络，散风寒，壮筋骨。
用法用量：加生姜3片，大枣2个，水2盏，煎至1盏，去滓，空心温服。
制备方法：上细切，作1服。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
