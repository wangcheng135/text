序号：63479
名称：清胃散
组成：丹皮1钱，青皮6分，甘草5分，石膏1钱，生地黄1钱，防风1钱，荆芥1钱。
出处：《嵩崖尊生》卷六。
主治：胃热牙痛面热。
加减：上四正牙痛，加黄连8分，麦冬1钱2分；下四正牙痛，加黄柏8分，知母1钱；左上板牙痛，加羌活1钱，胆草8分；左下板牙痛，加柴胡1钱，栀子1钱；右上板牙痛，加大黄1钱，枳壳1钱；右下板牙痛，加黄芩1钱、桔梗1钱；上两边牙痛，加川芎、白芷；下两边牙痛，加白芍、白术；头痛，加藁本；恶心，加厚朴；牙龈烂，用生姜、黄连捣烂贴上。
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
