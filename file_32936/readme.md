序号：32945
名称：陈猪油膏
组成：乳香2钱（去油），没药2钱（去油），儿茶1钱5分，血竭1钱5分，麝香少许。
出处：《集验良方》卷六。
主治：一切漏疮大毒。
加减：None
功效：None
用法用量：再加入上药，用槐枝或柳枝搅匀。春、夏黄蜡少减，秋、冬少加，必用铜锅熬，或砂锅亦可。其猪油即香脂油，必须隔年腊月收藏者方用，新猪油不可用。收猪油法：每年腊月八日，或腊月预买猪香脂油，不拘多少，收入净瓷罐内，将口封好，吊在背阴透风处，临时听用，以便年年添入。此油愈久愈佳。
制备方法：将陈猪油四两，入锅内化开，即下黄蜡一两，滚百滚，次下黄丹五钱，即起火，用绵纸滤去滓，盛入瓷器内。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
