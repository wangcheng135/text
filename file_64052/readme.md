序号：64068
名称：猪尾膏
组成：小猪儿尾尖血（刺血）1-2点。
出处：《活人书》卷二十一。
主治：疮子倒黡。锁唇痘，聚口唇内，肿裂，干黄板硬；蛇皮痘，出似蛇皮，隐隐簇簇漫无拘之，毒重者。
加减：None
功效：None
用法用量：上入生脑子少许同研，新水调服．
制备方法：None
临床应用：None
各家论述：《奇效良方》：古人用龙脑香凉心血，行荣卫；用猪尾血者，取其常动欲散外也；况有前狂躁证而未省者，以温酒化下，此意欲散而行荣卫故也，皆治热毒太盛者。后人见用猪尾膏，亦名龙脑膏得效，间有虚而陷伏者亦用之，误人甚多，深可怪也。又有当用一字，却用1钱，热少而不能当之，及以为害者亦多。如合用一字者，且与半字，如伤寒作渴甚，能饮1斗者且与5斗，正此之意。
用药禁忌：兼有他虚寒证见者，不可轻服。
附注：水调饮子（方出《阎氏小儿方论》，名见《医方类聚》卷二六四）。《奇效良方》本方用法；用紫草汤化下；烦躁狂闷未省者，以温酒浸服之。《金鉴》本方用法：锁唇痘，以泻黄散合猪尾膏；蛇皮痘，以必胜汤合猪尾膏。
