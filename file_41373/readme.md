序号：41388
名称：金城太守白薇丸
组成：白薇30铢，人参18铢，牡蛎18铢，牡蒙18铢，牛膝半两，细辛30铢，厚朴18铢，半夏18铢，沙参半两，干姜半两，白僵蚕10铢，秦艽半两，蜀椒1两半，当归18铢，附子1两半，防风1两半，紫菀18铢。
出处：《千金》卷二注文引《古今录验》。
主治：月水不利，闭塞绝产。
加减：None
功效：None
用法用量：食前服3丸。不知，稍增至4-5丸。此药不长将服，觉有娠则止。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：方中参、附、椒、姜以温血气；白薇、沙参以化辛热；辛、防、秦艽以祛血室之风；牛膝、当归以和冲脉之血；僵蚕以涤子户风痰；加杜衡者，师甄权之破留血也；牡蒙、紫菀者，法《本经》之下逆气及胸中寒热结气也。逆气下，结气散，而血行无滞；风气去，痰气除而子脏安和，故用半夏、厚朴、僵蚕专行清理风痰湿滞。搜剔脂腻，此方为最，所以服之匝月便能有子。
用药禁忌：忌饧、猪、羊肉。
附注：白薇人参丸（《圣济总录》卷一五三）。方中牡蛎，《千金》作“杜衡”；又，原书注：《崔氏》有桔梗、丹参各十八铢。
