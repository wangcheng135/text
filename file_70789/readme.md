序号：70805
名称：紫金膏
组成：明松香4两（夏用红者，冬用白者，秋冬红白各半，以火熬滚，入水内扯拔百十下，研末，若贴痘毒，松香用黄豆浸水入锅内煮化，待温照上扯拔，研细末），蓖麻仁2两（研细，放细筛罗底上，用穿山甲往来刮之，取罗下者用之，上面粗者去之），轻粉5钱，银朱2钱5分，铜绿2钱5分。
出处：《疡医大全》卷七。
主治：一切无名肿毒，恶疮；兼风湿流火，小儿痘毒。
加减：None
功效：None
用法用量：凡贴毒，将膏中剪1孔，露顶透气，能贴多年痘毒，若贴流火，竟贴顶上，不必剪孔。
制备方法：宜端午、七夕、重阳、天医、天德、月德日配合制毕，用猪油去衣膜，拌药放青石上，用铁槌捣数千下，盛瓷瓶内，用时摊油纸上贴。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
