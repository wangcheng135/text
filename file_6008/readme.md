序号：6008
名称：加减圣效散
组成：卜子（炒）8钱，砂仁（炒，研）8钱，槟榔8钱，陈皮8钱，延胡8钱，厚朴5钱，防风5钱，苍术5钱，藁本5钱，藿香叶5钱，柴胡5钱，独活5钱，石菖蒲5钱，泽泻5钱，枳壳5钱，细辛5钱，草豆蔻（去壳）10个。
出处：《痧书》卷下。
主治：伤寒时疫风湿，阴阳两感，表里未辨，或外热内寒，或外寒内热，肢节拘急，头项腰脊疼痛，发热恶寒，呕逆喘咳，鼻塞声重，及食饮生冷，伤在胃脘，胸膈饱满，肠胁胀痛，心下痞结，手足逆冷，肠鸣泄泻，水谷不消，小溲不利。
加减：None
功效：None
用法用量：每服5钱，水盏半，煎至1盏，去滓温服，不拘时候。取遍身微汗即愈。时气不和，空心饮之，可辟邪疫。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：加减圣功散（《痧症汇要》卷四）。
