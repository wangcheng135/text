序号：33991
名称：皂荚丸
组成：皂荚1两（去黑皮，涂酥，炙令黄，去子），紫菀3分（去苗土），款冬花半两，陈橘皮3分（汤浸，去白瓤，焙），细辛3分，桂心半两，麦门冬1两（去心，焙），紫苏子3分（微炒），杏仁1两（汤浸，去皮尖双仁，麸炒微黄，研如膏），干姜3分（炮裂，锉），当归3分（锉，微炒），甘草半两（炙微赤，锉），川椒1两（去目及闭口者，微炒去汗）。
出处：《圣惠》卷四十六。
主治：久咳嗽上气，心胸满闷，吃食减少。
加减：None
功效：None
用法用量：每服30丸，以姜、枣汤送下，不拘时候。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
