序号：38680
名称：泻白丸
组成：石膏（煨熟）1两，花粉1两，川贝母（去心）1两，陈香橼（去瓤）1两，胆南星1两，款冬花1两，薄荷叶1两，甘草7钱，细芽茶7钱（上9味共为极细末，听用），麻黄1两5钱，防风1两，桑皮（蜜炙炒）1两，杏仁（去皮尖，炒）1两，前胡1两，紫菀1两，苏子（炒，为末）1两，陈瓜蒌1大个，柿饼3两，山栀1两，葶苈子（炒）5钱。
出处：《医宗说约》卷五。
主治：风邪痰火咳嗽，嗽声不转者。
加减：None
功效：止嗽疏邪，消痰定喘，清热顺气。
用法用量：每服1丸，小儿灯心汤化下；大人噙化。
制备方法：上用水煎，去滓滤清，再入萝卜汁、水梨汁、餳糖各4两，姜汁5钱，煎成膏，滴水成珠，将前末药和匀为丸，每丸重1钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
