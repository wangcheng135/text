序号：78887
名称：磁石木香丸
组成：磁石3两（烧令通赤，以醋淬7遍，捣碎，研，水飞过），木香1两，附子3两（炮裂，去皮脐），干姜3两（用浆水1斗，盐花1合，与附子一处以慢火煮，水尽为度，切片，焙干），汉椒3两（醋浸1宿，取出，用炭火半称，先烧地令通赤，将椒薄摊于地上，以盆子盖却1宿，取出）。
出处：方出《圣惠》卷三十，名见《普济方》卷二二四。
主治：虚劳目暗，或见黑花。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以盐汤送下。
制备方法：上为细散，入磁石都研令匀，用羊肾2对，切去脂膜，入砂盆肉细研，同酒2升，同熬成膏，入药末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
