序号：42263
名称：青盐丸
组成：破故纸（炒）半两，茴香（淘去沙，炒）半两，石菖蒲半两，肉桂（去粗皮，不见火）半两，川椒（去目合口者，炒出汗）半两，牡蛎（煅）半两，木香（不见火）半两，陈皮（去瓤）半两，缩砂仁半两，当归（去芦，酒浸）半两，川楝子（去核，炒）半两，厚朴（去粗皮，姜制，炙）1两，鹿角霜1两，吴茱萸（炒）1两，桃仁（去皮尖，炒）4两，苍术（米泔浸1宿，炒）4两，草乌头2两（盐煮），青盐4两半（炒干）。
出处：《魏氏家藏方》卷五引王克明方。
主治：None
加减：None
功效：调理脾胃。
用法用量：每服60丸，食前盐汤送下；妇人醋汤送下。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
