序号：6975
名称：四制化痰丸
组成：半夏1斤（分作4份，勿切碎。1份用生姜1两，黄连各1两，水2碗同煮干，去姜、连；1份用知母1两，贝母各1两，水2碗同煮干，去2母；1份用人参1两，杏仁1两，去皮尖，水2碗同煮干，去参、杏；1份用桔梗1两，桑皮1两，水2碗同煮干，去桑、桔）。
出处：《万氏家抄方》卷二。
主治：男妇虚火咳嗽，哮喘吐痰，胸膈饱胀嗳气，一切痰症；肥人嗜酒好味，胃口生痰生火，脉数者。
加减：None
功效：None
用法用量：每服40-50丸，生姜汤送下。
制备方法：拣出半夏，切碎晒干，为末，水糊为丸。
临床应用：None
各家论述：《医略六书》：湿热生痰，滞于胃口，故痞满不消，恶心不止焉。半夏一味，功为燥湿化痰，煮以黄连，清火燥湿，生姜解郁散痰，知母清热润燥，贝母解热清痰，人参扶元益胃气，杏仁降气豁痰涎，桑皮肃金泻湿热，桔梗清咽利胸痞。使湿热消化，则脾胃清和，而痞满无不解，何恶心之不止哉！
用药禁忌：None
附注：None
