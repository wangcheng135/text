序号：64395
名称：猪悬蹄青龙五生膏
组成：猪后悬蹄3枚（炙黄），生梧桐白皮4两，生桑根白皮5分，龙胆5分，雄黄（研）5分，蛇蜕皮50（炙），生青竹皮6分，露蜂房（炙）3分，蜀椒3分（汗），猬皮（烧）4分，附子（炮）4分，生柏皮7分（炙），杏仁30枚（去皮尖）。
出处：《外台》卷十六引《删繁方》。
主治：肺虚劳损，致肠中生痔，名曰肠痔。肛门边有核痛，寒热得之，好挺出，良久乃缩而生疮。
加减：None
功效：None
用法用量：敷疮，井酒服如枣大。
制备方法：上药细切，绵裹，以苦酒2升半淹渍1宿，于火上炙燥捣筛，以猪膏3升和，微火上煎如薄糖。
临床应用：None
各家论述：None
用药禁忌：None
附注：青龙五生膏（《普济方》卷二十七）。
