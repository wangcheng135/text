序号：53847
名称：射干煎
组成：射干8两，紫菀半两，胶饴5两，细辛半两，干姜5两（末），生竹沥1升，芫花根半两，桑根白皮8两，款冬花8两，附子半两（炮），甘草半两（炙），白蜜1升半。
出处：《外台》卷十引《深师方》。
主治：咳嗽上气。
加减：None
功效：None
用法用量：上先切射干，合蜜、竹沥汁，煎5-6沸，绞去滓，（口父）咀诸药，以水1升4合，渍1宿，煎之，七上七下，去滓，乃合饴、姜末煎，令如餔。每服酸枣1丸许，日3夜1，不知，稍增之。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：用附子摄肾归源，细辛下通肾气，射干上散肺结，款冬花、紫菀滋燥润肺，姜汁开发桑皮、竹沥之性，流动胶饴、白蜜之滞也。
用药禁忌：忌海藻、菘菜、猪肉、冷水、生菜。
附注：None
