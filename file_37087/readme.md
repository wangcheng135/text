序号：37097
名称：定吐救生丹
组成：山大戟（浆水煮，切，焙干，为末）15两，乳香（别研）5两，丁香（为末）5两，粉霜（研）7两半，腻粉（研碎）7两半，龙脑（研）2两半，水银10两半，黄蜡10两半，黑铅（与水银同结砂子）10两半。
出处：《局方》卷十。
主治：小儿伏热生涎，心膈烦躁，壮热霍乱，乳食不下，呕哕恶心，或发吐逆。
加减：None
功效：除热化涎，下膈止吐逆。
用法用量：每1岁儿服1丸。如烦躁，研生脂麻、马齿水送下；如吐逆，煎丁香、马齿汤送下。食后临卧服之。
制备方法：上为末，每熔蜡1两，入炼蜜2钱半为丸，如黄米大。
临床应用：None
各家论述：None
用药禁忌：若胃虚伤冷，呕吐不止者，不可服。
附注：None
