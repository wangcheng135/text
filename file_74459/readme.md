序号：74475
名称：楞薤煅蒌丸
组成：瓦楞子（醋煅）、鸡内金1两，延胡5钱，没药5钱，香附5钱，桃仁3钱，蒌仁3钱，苏子3钱，白芥子3钱，萝卜子3钱，薤白3钱。
出处：《医级》卷九。
主治：癥瘕痰食，积滞留著，以致不时冲逆胸胁，攻注腹肋切痛。
加减：None
功效：None
用法用量：每服30丸，白汤送下。
制备方法：先用顶大瓜蒌1个，开一孔，去仁，将香附、桃仁、三子、薤白、蒌仁7味和匀，装入蒌壳内将孔盖好，麻扎纸糊，外和熟黄泥厚涂，火煅3炷香，候烟将尽，即取起置泥地候冷出火，然后打开泥取炭药，研极细，再将楞、肫、延、没4味研极匀，以荞麦面糊作丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
