序号：6196
名称：加味将军汤
组成：犀牛角2钱，羚羊角2钱，真锦纹川军4两。
出处：《医学探骊集》卷五。
主治：狂病或登高而歌，或弃衣而走，或妄见妄言，或打人骂人者。
加减：None
功效：None
用法用量：水煎，温服。早晨空心服药，俟其大泻后，至晚不可与食。其人不能饮烧酒者，用烧酒4两，香油2两，能饮2两者，用烧酒6两，香油3两，折杨枝百根，皆6寸许，将油、酒对一处，用杨枝搅之，每根搅50-60下，搅完将油、酒火上微温，令病人以羹匙饮之，饮尽为度。病者既1日不食，饮完此酒，必大醉思睡，任其睡去，不可惊动。饮此酒有呕者，有不呕者，其形不一，及其睡醒，再与粥或淡汤食之，其病若失。如觉有不爽之处，可取鸠尾、中脘针之，留5小时乃出针，针后服清镇丹1剂。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
