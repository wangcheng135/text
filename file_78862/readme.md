序号：78878
名称：酸枣仁浸酒
组成：酸枣仁3两（微炒），羌活1两，牛膝2两（去苗），山茱萸2两，桂心3两，仙灵脾2两，萆薢2两，芎2两，天麻2两，肉苁蓉2两（锉，去皱皮），虎胫骨3两（涂酥炙令黄），生干地黄3两，甘菊花1两半，天雄2两（炮裂，去皮脐），桑寄生1两半。
出处：《圣惠》卷二十五。
主治：风气。
加减：None
功效：暖脏腑，利脚膝，止疼痛。
用法用量：每服1盏，温饮，1日3-5度，常令醺醺。瓶中酒少，旋更添之，候药无味，即更修合，以愈为度。
制备方法：上细锉，以生绢袋盛，用好酒3斗，于净瓷瓶中浸，密封瓶口，5日后开取。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
