序号：4629
名称：木香枳壳丸
组成：木香1两，枳壳1两（炒，去瓤），槟榔1两，半夏1两（汤浸7次），青皮1两（去瓤），陈皮1两（去白），白茯苓（去皮）1两，白术（煨）1两半，京三棱3两3钱（煨），广荗（煨）3两3钱，黑牵牛（微炒，取末）3两，人参半两，神曲半两（微炒），大麦蘖半两（微炒），枳实（炒）半两，干姜（炒）7钱。
出处：《瑞竹堂方》卷一。
主治：中焦气涩，胸膈痞闷，饮食迟化，四肢困倦，呕吐恶心。
加减：None
功效：升降滞气，消化宿食，去痰，进饮食。
用法用量：每服50丸，食后姜汤送下。
制备方法：上为细末，水糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
