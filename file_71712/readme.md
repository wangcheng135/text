序号：71728
名称：琥珀抱龙丸
组成：甘草5钱，天竺黄1两2钱，防风1两2钱，天麻1两5钱，茯苓1两5钱，羌活1两5钱，川贝母1两5钱，白附子（炙）1两5钱，蝉退1两5钱，胆星1两5钱，桔梗1两5钱，全蝎9钱，僵蚕（炒）9钱，钩藤9钱，人参（去芦）9钱（以上共为细粉，过罗），牛黄5钱，珍珠（豆腐炙）5钱，琥珀1两，明雄黄6钱，朱砂6钱，麝香9钱。
出处：《北京市中药成方选集》。
主治：内热痰盛，咳嗽喘促，惊吓失魂，惊风抽搐。
加减：None
功效：清热化痰，镇惊安神。
用法用量：每服1丸，温开水送下，1日2次。
制备方法：上为细末，炼蜜为丸，重5分，金衣三十六开，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
