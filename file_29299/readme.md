名称：远志散
组成：远志（去心）半两，黄耆（炙，锉）半两，芍药1两，五味子1分，黄芩（去黑心）1分，赤茯苓（去黑皮）3分，牡荆实（轻炒）3分，秦艽（去苗土）1两，乌头（炮裂，去皮脐）半两，天雄（炮裂，去皮脐）半两，细辛（去苗叶，微炒）半两，山茱萸半两，菊花（未开者，炒）半两，防风（去叉）3分，狗脊（去毛）半两，桂（去粗皮）半两，芎半两，芜荑（微炒）半两，菖蒲（用米泔浸，去节皮，焙）3分，萎蕤（去土及须，焙）3分，白敛（生用）半两，山芋半两，附子（炮裂，去皮脐）半两，龙胆（去苗土）半两，厚朴（去粗皮，生姜汁炙，锉）半两，蜀椒（去目并闭口者，炒出汗）1分，巴戟天（去心）1分。
出处：《圣济总录》卷二十。
主治：周痹不仁。
加减：None
功效：None
