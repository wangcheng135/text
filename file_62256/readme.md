序号：62272
名称：敛阳丹
组成：灵砂（研末）、钟乳（研末）2两，金铃子（蒸，去核）1两，沉香（镑）1两，木香1两，附子（炮，去皮脐）、葫芦芭（酒浸，炒）1两，阳起石（煅成细粉，水飞）1两，破故纸（酒浸，炒）1两，茴香（炒）1两，肉蔻（面裹煨）1两，鹿茸（酒炙）1两，苁蓉（酒洗）1两，牛膝（酒浸）1两，巴戟（去心）1两，肉桂（去皮）半两。
出处：《医方大成》卷四引《澹寮》。
主治：老人气虚，面红自汗，阳气不敛者。
加减：None
功效：安神益志，顺气调荣。
用法用量：每服30丸，空心枣汤送下。
制备方法：上为末，和匀，酒煮糯米糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
