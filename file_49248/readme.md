序号：49263
名称：胡芦巴散
组成：胡芦巴2两，补骨脂（炒）2两，荜茇1两，荜澄茄1两，茴香子（炒）1两，木香1两，丁香1两，楝实1两，桂（去粗皮）1两，槟榔1两，巴戟天（去心）1两，京三棱（微锉）1两，青橘皮（汤浸，去白焙）1两，附子（炮裂，去皮脐）1两，枳壳（去瓤麸炒）1两。
出处：《圣济总录》卷一八七。
主治：肾脏虚惫，腰膝疼，小肠膀胱等气攻冲。
加减：None
功效：None
用法用量：每服15丸，空心盐汤送下。
制备方法：上为散。每服2钱匕，水、酒共1盏，同煎3-5沸，温服；如作丸，用酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
