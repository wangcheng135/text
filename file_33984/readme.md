序号：33993
名称：皂荚丸
组成：皂荚20挺（10梃去黑皮，涂酥，炙令焦黄，去子；10梃生捶烂，用好酒5升，按绞取汁，熬成膏），何首乌半斤，干薄荷半斤，蜗牛子4两（炒令微黄），硼砂1两（通白者），附子1两（炮裂，去皮肤），天麻1两，精羊肉4两（去脂膜，薄切，炙令干），天南星1两（炮），半夏1两（汤洗7遍，去滑）。
出处：《圣惠》卷六十六。
主治：风毒瘰疬，项腋下生如梅李枣核，肿痛。
加减：None
功效：内消。
用法用量：每服10丸，渐加至20丸，空心及晚食前浸牛膝酒送下。
制备方法：上为末，入皂荚膏，和捣为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
