序号：60051
名称：起废丹
组成：川乌头4两（炮，去脐皮尖），五灵脂（去砂石，炒）4两，附子（炮，去皮脐）2两，白花蛇（酒浸，去皮骨）2两，肉桂（去粗皮）2两，羌活（去芦头）2两，天南星（炮）2两，干姜（炮）2两，虎骨（酥炙）1两半，甘菊花1两半，零陵香1两半，金牙石（煅红、醋淬3次）1两半，藿香叶（去土）1两半，血竭（别研）1两，香白芷1两，川芎1两，麻黄（去根节）1两，甘草（炙）1两，狼毒（炮）1两，干蝎（去毒，炮）半两，皂角（炙，去皮子）半两，白姜蚕（炒，去丝咀）半两，朱砂（别研）半两，雄黄（别研）半两，细松烟墨（烧）半两，脑子1钱（别研），麝香1分（别研），生地黄4两，当归4两（同入少许砂盆内，研成膏子，又用无灰酒3升，煮膏子令半干）。
出处：《杨氏家藏方》卷一。
主治：一切中风瘫痪，口眼斜，语言謇涩，步履艰难，筋脉拳缩，骨节疼痛。
加减：None
功效：None
用法用量：每服1丸，用热豆淋酒（用黑豆炒焦，乘热以酒浸之，去豆取酒）送下，不拘时候。初得病之日，即服神柏散，次服本药，不致为废人。
制备方法：上为细末，入地黄当归膏子内，搜和，放木臼内捣千余杵，如膏子，和药，硬时即用浸花蛇酒打面糊，渐渐添入杵匀，每1两作10丸，阴干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
