序号：35761
名称：参五秦艽汤
组成：当归3钱，赤芍（酒炒）7分，苍术（童便浸）1钱，生地黄（酒浸）1钱，萆薢1钱，黑狗脊（去毛根）2钱，川芎7分，羌活1钱5分，秦艽（去芦）1钱5分，川独活1钱，五加皮2钱，黄连（姜汁炒）1钱，黄柏（酒炒）1钱，红花（酒洗）8分，黄芩（酒炒）1钱5分，黄耆（酒炒）2钱，人参2钱，牛膝（去芦，酒浸）1钱5分，杜仲（每1两用茴香1钱，盐1钱，水2钟拌炒）2钱，生甘草2分。
出处：《寿世保元》卷五。
主治：痛风。腰背手足肢节疼痛，因外淫侵入日久，及年近衰者，不善养生，肝血、肾水内损所致。
加减：如天将作雨，阴晦时日，而预先觉痛甚者，加防风、天麻、升麻；午后夜甚者，血弱阴虚，加升麻5分，牡丹皮1钱；早上午前甚者，气滞阳弱，加连翘、沉香、竹沥、乳汁；痛甚者，倍羌活、红花、酒炒黄芩，凉血则痛止。
功效：None
用法用量：桃枝7根，每长1寸半；灯心7根，水煎，临服入童便1盏，好酒1盏，空心温服。滓再煎服。
制备方法：上锉。
临床应用：None
各家论述：None
用药禁忌：忌食酒、面、鲤鱼、羊、鹅。
附注：None
