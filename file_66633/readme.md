序号：66649
名称：鹿角霜丸
组成：黄耆（蜜炙）2两，人参2两，白术2两，白茯苓2两，当归（酒洗）2两，川芎1两，肉桂1两，熟地黄2两，茴香（炒）1两，牛膝（去芦）1两半，木瓜1两半，白芍药（酒炒）2两，川乌1两半，羌活1两，独活1两，肉苁蓉（酒洗）1两半，槟榔1两，防风1两半，乌药（炒）1两半，破故纸（酒炒）2两，木香2钱，续断1两5钱，甘草5钱，苍术（米泔水浸）2两，附子1两（童便和煨），杜仲2两（姜汁炒去丝），虎胫骨（酥炙）1两5钱，鹿角霜1斤。
出处：《古今医鉴》卷二。
主治：诸风瘫痪，半身痿弱，或2-3年不能动履者。
加减：None
功效：None
用法用量：每服100丸，空心米汤送下。
制备方法：上为极细末，酒为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
