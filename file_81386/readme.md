序号：81402
名称：麒麟竭膏
组成：麒麟竭半两，雄黄半两（细研），密陀僧半两（细研），雌黄1分（细研），乱发半两，朱砂半两（细研），乳香1两（细研），黄耆1两，白芍药1两，牡丹1两，连翘1两，丁香1两，木香1两，桂心1两，当归1两，牛膝1两（去苗），细辛1两，白芷1两，松脂2两，腊3两，黄丹12两，麻黄2两，油2斤半。
出处：《圣惠》卷六十三。
主治：一切痈疽发背，恶疮毒肿溃后，日久脓水不住，肌肉不生，毒气未定。
加减：None
功效：收毒，止痛，暖肌。
用法用量：每日2次贴之。
制备方法：上药黄耆等十二味细锉，入油内浸1宿，后用文火煎诸药色黑，滤出；次下松脂、乳香、蜡消熔尽，以绵滤去滓；拭铛令净，却下药油，以慢火熬，相次入黄丹，不住手以柳木篦搅，候色变，滴于水碗内，捻看软硬得所，歇良久，入麒麟竭、雄黄、雌黄、密陀僧、朱砂等末，搅令匀，倾于瓷盒内，以纸上摊令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：麒麟膏（《医方类聚》卷一七八引《御医撮要》）。
