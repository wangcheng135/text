序号：30978
名称：十香定痛丸
组成：丁香2两，降香2两，乳香（炙）2两，没药（炙）2两，木香2两，母丁香2两，于术2两，良姜2两，白芍2两，延胡索（炙）2两，厚朴（炙）2两，片姜黄2两，灵脂（炙）2两，松罗茶2两，九菖蒲2两，槟榔2两，三棱（炒）2两，莪术（炙）2两，法半夏1两，茴香（炒）1两，檀香1两，红花1两，肉桂（去粗皮）1两，甘草1两，赤芍1两，生蒲黄1两，二丑（炒）4两，茯苓2两，山楂（炒）2两，砂仁2两，枳实（炒）2两，紫豆蔻5钱，香附（炙）4两。
出处：《北京市中药成方选集》。
主治：气郁结滞，胃脘疼痛，胸满腹胀。
加减：None
功效：舒郁散寒  和胃定痛
用法用量：每服1丸，温开水送下，1日2次。
制备方法：上为细末，过罗兑：伽楠香2两，苏合香2两，朱砂2两，安息香1两。研细混合均匀，炼蜜为丸，重2钱，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
