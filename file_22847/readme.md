序号：22855
名称：西川罗赤脚仙还少丹
组成：干山药1两半，牛膝（酒浸一宿，焙干）1两半，山茱萸1两，白茯苓（去皮）1两，五味子1两，肉苁蓉（酒浸一宿。焙干）1两，石菖蒲1两，巴戟（去心）1两，远志（去心）1两，杜仲（去粗皮，用生姜汁并酒合和，涂炙令熟）1两，楮实1两，舶上茴香1两，枸杞子半两，熟干地黄半两。
出处：《洪氏集验方》卷一引陈晦叔方。
主治：一切虚损，神志俱耗，精力不爽，腰脚沉重，肢体倦怠，气血羸乏，小便浑浊，子宫久冷；脾胃怯弱，心忪恍惚，精神昏愦，气血凝滞，饮食无味，肌瘦体倦，目暗耳聋。
加减：如身热，加山栀子1两；心气不宁，加麦门冬1两；少精神，加五味子1两；阳弱，加续断1两。
功效：大补心肾脾胃，常服令人身体轻健，筋骨壮盛，怡悦难老，齿牢，永无瘴疟；妇人服之，姿容光悦，去一切病。
用法用量：每服30丸，空心以温酒、盐汤送下，日3次。至5日觉有力，10日精神爽健，半月气力稍盛，20日目明，1月夜思饮食，冬月手足常暖，久服无毒。
制备方法：上为末，炼蜜入枣肉为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：还少丸（《杨氏家藏方》卷九）、还少丹（《卫生宝鉴》卷六）、滋阴大补丸（《医学正传》卷三）、真人还少丹（《中国医学大辞典》）。
