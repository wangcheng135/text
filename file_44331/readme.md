序号：44346
名称：一捻金散
组成：半夏2两，天南星（锉）2两，巴豆2两，皂角子6两，阿胶2两（锉），黄明胶3两（锉），杏仁6两，白矾1两半。
出处：《杨氏家藏方》卷八。
主治：虚损劳嗽，咯血吐血，心胸不利，上气喘急，寒热往来，盗汁羸瘦，肢节酸痛，肌肉枯槁，咳嗽不已，痰涎壅盛，夜卧不安；暗风痫病，倒仆不省人事，口吐涎沫。
加减：None
功效：截劳气，定喘满，化痰涎。
用法用量：每服半钱，生姜自然汁调成稠膏，临卧入齑汁半盏和服。
制备方法：上药都入藏瓶内，外留1眼子出烟，盐泥固济，候干；用炭半秤，煅令烟尽为度，却用泥塞合出烟眼子，放冷一宿，研为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
