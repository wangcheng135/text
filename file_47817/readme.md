序号：47832
名称：神仙紫花丸
组成：白花蛇（1具，出蕲黄州，黑质白纹，龙头虎口，背上24个方胜花，尾尖有1佛指甲，新鲜者佳，蛀腐者不堪用，去头尾各4-5寸，并1两为率，连皮骨用）1两5钱，何首乌4钱，荆芥穗4钱，威灵仙4钱，麻黄（连根节）2钱，胡麻子1钱，蛇床子2钱，上细切，同蛇用无灰酒1大碗，浸1宿，去蛇皮骨，通晒干，仍还原酒内，再浸再晒，酒尽为度，待晒极干，共为细末，另包。木香2钱5分，沉香2钱5分，人参1两，当归7钱5分，明天麻5钱，猪牙皂角5钱，麝香1钱5分，乳香1钱，没药1钱，明雄黄5分，辰砂5分（大块者佳），肉豆蔻1枚（煨），定风草（即天麻）2钱半，还瞳子（即草决明）1两，上麝至辰砂5味，各另研极细，不见火，其余草木味亦另研，细罗过，连前5味和匀另包。防风（去芦）5钱，羌活5钱，甘草5钱，细辛5钱，川芎5钱，独活5钱，苍术（米泔浸1宿）5钱，枇杷叶（去毛筋，焙干）5钱，白芍药5钱，白蒺藜5钱，金银花5钱，五加皮5钱，香白芷5钱，苦参5钱，胡麻子2钱5分，白附子（米泔浸，炒）2钱5分，麻黄2钱5分，川牛膝2钱5分，草乌头（米泔浸，炮）2钱5分，川乌（米泔浸，炮）2钱5分，石菖蒲2钱5分，上为细末，另包。
出处：《医学正传》卷六。
主治：疠风及诸般恶疮，风疮。麻风，麻木不仁，筋脉枯萎。
加减：鼻塞声重者，麝香倍用。
功效：None
用法用量：每服20丸，渐加至50-60丸，鸡鸣时、午时、临卧时各1服，茶清送下。轻者，1料可愈；重者，2-3料除根。
制备方法：用新鲜大枫子3斤，去油黄色者及壳，以瓷罐1个盛之，少入无灰酒，以皮纸竹箬重重包口，勿令泄气，顿滚汤中，勿令没罐口，外以物盖锅口，密封固，文武火蒸，候黑烂为度，杵无渣滓成油，分作3份；每一份入第二号药8钱重，第一号药6钱重，第三号药1两5钱重，和匀，加糯米饭捣极胶粘为丸，如梧桐子大，晒干，勿见火。
临床应用：None
各家论述：None
用药禁忌：忌房劳，咸酸，酒醋，糟腌，猪羊鸡马驴肉，鱼腥煎煿煿，水果，五辛，姜椒大料，辛辣热物，荞麦绿豆之类，其余肉味，病愈后一年可食，但猪羊鸡肉，终身用忌。
附注：《疡医大全》有赤芍、牛蒡子、石决明、无白芍药、川牛膝、草决明。
