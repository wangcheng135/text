序号：34349
名称：万金水澄膏
组成：乳香3钱，没药3钱（用灯心捻去油），广木香（不见火）3钱，轻粉1钱，雄黄4钱，辰砂4钱（以上俱另为末），白及8两，黄药子2两，乌骨鸡骨2钱6分（要白毛乌骨乌肉的1只，生刮去腿肉，用腿骨晒，称如前数，以火煅存性，另研和内，不许煮熟用之）。
出处：《疡科选粹》卷八。
主治：一切无名肿毒，发背，痈疽已溃、未溃。
加减：None
功效：None
用法用量：将底药敷红肿处，空1顶，再用绵纸贴于顶，纸穿1孔，以败毒气；候纸干，将前药水浸润纸外，勿令干，病轻每日2次，病重日夜3-4次。
制备方法：上为末，用冷井水半钟，以药少许调化，候药澄钟底，随以钟内余水滤去听用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
