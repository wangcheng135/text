序号：30518
名称：还睛补肝丸
组成：白芍（酒炒）2两，熟地2两，当归（酒洗）2两，天冬2两，五味子2两，炙甘草2两，白术2两，白茯苓2两，官桂2两，车前子（微炒）2两，白菊花2两，青葙子2两，玄参2两，川芎1两，羌活（去芦）1两，防风（去芦）1两，人参1两，骨皮1两，黄芩（酒炒）1两，柴胡1两，细辛1两，决明子1两，苦参1两，黄连（姜汁炒）5钱。
出处：《惠直堂方》卷二。
主治：眼目羞明多泪，翳膜侵珠，时歇时作，久病不愈。
加减：None
功效：None
用法用量：每服3钱，临睡白汤送下。久久服之，永不再发。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
