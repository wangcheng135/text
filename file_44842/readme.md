序号：44857
名称：扁鹊曾青丸
组成：曾青2分，寒水石3分，朴消2分，茯苓3分，大黄3分，附子3分（炮），巴豆2分。
出处：《古今录验》引殷仲堪方（见《外台》卷十二）。
主治：久寒积聚，留饮宿食。
加减：None
功效：None
用法用量：大人服大豆大2丸；小儿5岁以下如麻子大1丸；2-3岁儿如黍米大1丸，如服药以薄粉粥清下。当覆卧令汗出。吐下气发作服2丸，霍乱服3丸，泄痢不止服1丸可至2丸。
制备方法：上各异捣，下筛，巴豆、消相合，捣六千杵，次纳附子捣相得，次纳茯苓捣相得，次纳大黄捣相得，次纳曾青捣相得，次纳寒水石捣相得，次纳蜜和捣千杵。
临床应用：None
各家论述：None
用药禁忌：忌猪肉、冷水、芦笋、大酢。
附注：曾青丸（《外台》卷十二引《古今录验》）。
