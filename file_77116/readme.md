序号：77132
名称：黑虎丹
组成：苍术（米泔浸7日）1两，川乌（煨）1两，草乌（米泔浸，去皮，煨）1两，细辛1两，防风1两，荆芥1两，麻黄1两，两头尖1两，金钗石斛1两，天麻1两，川芎1两，白芷1两，甘草1两，何首乌1两，当归1两。
出处：《普济方》卷一一五。
主治：诸风。
加减：None
功效：None
用法用量：每服1丸，随引汤送下。治男子、妇人36种风疾，眉毛退落，冷茶送下；腰痛、耳聋，肾脏风，荆芥汤送下；遍身紫癜风，防风汤送下；饮食无味，脾胃风，皂角子汤送下；手心退皮，鼓掌风，天麻汤送下；迎风冷泪，肝脏风，米泔水送下；筋骨疼痛，气下注，乳香汤下；口眼斜，心热风，茶汤送下；指节破裂，断风，盐白汤送下；心腹胀闷，胸膈塞，气风，生姜汤送下；前后倒卧地，感厥风，生姜汤送下；发狂吐沫，急缓风，荆芥汤送下；五般淋沥，肾患风，盐白汤送下。
制备方法：上为细末，炼蜜为丸，如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：《便览》有乳香、朱砂。
