序号：6441
名称：加味椒红丸
组成：真川椒1斤（净，分作4处，浸1伏时，4两酒浸，4两醋浸，4两童子小便浸，4两米泔水浸），自然铜2两（用火煅，以童子小便浸7次，醋淬7次）。
出处：《普济方》卷二二八。
主治：男子五劳七伤，身体骨节酸疼；行履艰难，胸膈闷满，不思饮食。
加减：None
功效：轻身健体，顺气。
用法用量：每服35-40丸，温酒送下，日进3服。
制备方法：上将川椒1斤，置地上烧红，却将好醉洒于烧红地上了，地干，将花椒铺在地上，用瓦盆盖之，候地冷，取出花椒为末，却将自然铜、乳香、没药、血竭一处，用原浸花椒酒、醋、童子小便、米泔水打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
