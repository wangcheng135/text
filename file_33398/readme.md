序号：33407
名称：肝肾两舒汤
组成：熟地1两，玄参1两，茯苓3钱，白芍1两，柴胡1钱，当归5钱，甘草1钱，炒栀子1钱，丹皮3钱。
出处：《辨证录》卷六。
主治：肝火郁结不伸，闷烦躁急，吐痰黄块。
加减：None
功效：舒肝解火，补肾济水。
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：此方归、芍、柴、栀所以舒肝者，风以吹之也；熟地、玄、丹所以补肾者，雨以溉之也；茯苓、甘草又调和于二者之中，使风雨无太过不及之虞也，譬如夏令炎蒸，郁极而热，树木枯槁，忽得金风习习，大雨滂沱，则从前郁闷燔燥之气，尽快如扫，而枯槁者倏变为青葱，爽气迎人，岂犹有烦闷躁急等症哉。
用药禁忌：None
附注：None
