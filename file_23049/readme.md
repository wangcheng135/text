序号：23057
名称：芎附散
组成：小川芎、附子（炮，去皮脐）、黄耆（蜜炙）、白术、防风（去钗股）、当归（洗，去芦，薄切，焙干）、熟干地黄（酒洒，9蒸9晒，焙）、桂心（不见火）、柴胡（去苗，净洗）、甘草（炙）各等分。
出处：《本事》卷三。
主治：脾胃虚弱，卫气不温分肉，为风寒湿邪所着而致的五种痹证，腿并臂间发作不定。
加减：None
功效：常服不生壅热，兼消积冷。
用法用量：每服4钱，水1盏半，加生姜3片，大枣1个，同煎至7分，去滓，食前服，每日3次。
制备方法：上为粗末。
临床应用：None
各家论述：《本事方释义》：小川芎气味辛温，入肝胆；附子气味辛咸大热，入手足少阴；黄耆气味甘平，入手足太阴；白术气味甘温，入手足太阴；防风气味辛甘平，入足太阳；熟地黄气味甘寒微苦，入肾；当归气味辛甘温，入肝；桂心气味辛甘热，入肝；柴胡气味辛平，入足少阳；甘草气味甘平，能行十二经络，缓诸药之性。此卫气不温分肉间，为三气所乘，致邪留着不去，故发作无定。以补血养气之药护持正气，以风药及辛热之药搜逐留着之邪，则卫气坚固，无隙可乘，亦何从入哉！
用药禁忌：None
附注：芎附汤（《简明医彀》卷三）。
