序号：73306
名称：普济五蹄膏
组成：巴豆肉1两，胡黄连1钱5分，川黄连1钱5分，黑丑1钱5分，白丑1钱5分，沙参1钱5分，元参1钱5分，柴胡1钱5分，连翘1钱5分，香附1钱5分，三棱1钱5分，莪术1钱5分，木香1钱5分，地骨皮1钱5分，天花粉1钱，白芥子1钱，神曲6分，山楂6分，麦芽6分，麻油2斤4两。
出处：《理瀹》。
主治：外科疑难险症，一切无名肿毒，未成或已成者。
加减：None
功效：消肿拔毒。
用法用量：摊贴患处。
制备方法：先将猪、羊、牛、马、驴蹄各5两熬烂，再入前药一同熬枯去渣，下黄丹1斤3两收膏。
临床应用：None
各家论述：None
用药禁忌：耳后，眉心忌贴。
附注：None
