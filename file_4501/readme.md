序号：4501
名称：木香平气丸
组成：木香半两，沉香半两，丁香半两，肉豆蔻仁半两，丹砂（别研）半两，麝香（别研）半两，槟榔（湿面裹，慢火内煨熟，去面不用）1两，桂（去粗皮）1两，厚朴（去粗皮，生姜汁炙）1两，乳香（拣通明者，生姜汁内煮软，另研如膏）1两，半夏2两（汤浸7遍，切作片子，焙干，杵为末，以生姜汁和作饼子，焙干，再杵为细末）。
出处：《圣济总录》卷四十七。
主治：心胸有痰，噫醋吞酸。
加减：None
功效：None
用法用量：每服15丸至20丸，食后温米饮送下。
制备方法：上药除丹砂、乳香、半夏、麝香4味别研外，将木香等7味一处为细末；次入丹砂、麝香再拌匀研细后，将乳香、半夏末入生姜汁，煮作薄糊，拌和前药；如拌和未就，更以生姜汁煮薄糊，取和拌硬软得所为度，为丸如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
