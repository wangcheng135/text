序号：38600
名称：法制清气化痰丸
组成：半夏4两，南星（去皮尖）4两，白矾4两，皂角4两，干姜4两，陈皮2两，青皮2两，紫苏子2两，萝卜子（炒，研）2两，杏仁（去皮尖，炒，研）2两，葛根2两，神曲（炒）2两，麦糱（炒）2两，山楂子2两，香附子2两。
出处：《校注妇人良方》卷六。
主治：None
加减：None
功效：顺气快脾，化痰消食。
用法用量：每服50-70丸，临卧茶汤任下。
制备方法：上将白矾等3味，用水5碗，煎取水3碗，却入半夏2味，浸2日再煮，至半夏、南星无白点为度，晒干，与余药共研为末，蒸饼为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：脾胃虚弱者忌用。
附注：None
