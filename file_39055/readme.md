序号：39065
名称：泽兰散
组成：泽兰1两，当归3分（锉碎，微炒），延胡索3分，桂心3分，附子3分（炮裂，去皮脐），牛膝3分（去苗），赤芍药半两，干膝3分（捣碎，炒令烟出），续断半两，芎3分，柏子仁半两，牡丹半两，琥珀3分，没药3分，桃仁3分（汤浸，去皮尖双仁，麸炒微黄），木香3分，麝香1分（研入）。
出处：《圣惠》卷七十。
主治：妇人风虚劳冷，气攻心腹疼痛，肢节拘急，体瘦无力，经候不调，食饮减少。妇人寒湿，或服水银，以致子宫翻出肿湿。
加减：None
功效：None
用法用量：每服2钱，食前以酒调下。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：None
附注：泽兰叶散（《医学入门》卷七）。
