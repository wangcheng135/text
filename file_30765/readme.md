序号：30773
名称：苣根散
组成：莴苣根7个，韭根7个，蒜1瓣，黄丹1两，麝香1钱。
出处：《鸡峰》卷二十四。
主治：小儿一切疳疾。
加减：None
功效：None
用法用量：每有害疳孩儿，用线围脚，男左女右，自后跟至大（母）趾尖为定，却把线双缋定自第一，推至线尽处，用纸花摊药贴之，如药干旋入新蒜研药，用之敷定，更须调寒食面作纸花子覆定药，免走动也。闻儿口中作香，取皂帛包头上，有疳虫万千出，细如碎发，出见帛上，乃是验也。食生米者，则泻出虫儿也。
制备方法：上先将三根并蒜烂研，次入黄丹、麝香，再研极烂，垍合盛之，勿令透气，每于喘午日绝早合之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
