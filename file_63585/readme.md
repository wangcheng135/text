序号：63601
名称：清凉散
组成：山栀、连翘、黄芩、防风、枳亮、黄连、当归、生地、甘草各等分，桔梗减半，薄荷减半，白芷减半（或不用亦可）。
出处：《回春》卷五。
主治：一切实火咽喉肿痛。
加减：咽喉干燥，加人参、麦门冬，天花粉，去白芷茫；咽喉发热，加柴胡；咽喉肿痛，加牛蒡子、玄参，去白芷；痰火盛，加射干、瓜萎、竹沥，去白芷；咽喉生疮，加牛蒡子、玄参，去白芷；极热大便实，加大黄，去桔梗；虚火泛上，咽喉生疮，喉不清者，加黄柏、知母，去白芷。
功效：None
用法用量：如灯心1团，细茶1撮，水煎，磨山豆根调服。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
