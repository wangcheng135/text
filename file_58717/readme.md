序号：58732
名称：大黄丸
组成：川大黄1两（锉碎，微炒），川椒半两（去目及闭口者，微炒去汗），人参3分（去芦头），半夏3分（汤洗7遍，去滑），桔梗3分（去芦头），菖蒲3分，柴胡3分（去苗），赤茯苓3分，芎3分，桂心3分，桃仁3分（汤浸，去皮尖双仁，麸炒微黄），木香3分，吴茱萸3分（汤浸7遍，焙干，微炒），干姜3分（炮裂，锉），细辛3分。
出处：《圣惠》卷四十二。
主治：七气，积聚坚牢，心腹胀痛。
加减：None
功效：None
用法用量：每服10丸，食前以温酒送下。渐加至20丸。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
