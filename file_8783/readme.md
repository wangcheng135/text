序号：8783
名称：二参丸
组成：人参半两，桂（去粗皮）3分，牡蛎（煅，研成粉）3分，山芋3分，黄拍（去粗皮，蜜炙，锉）3分，细辛（去苗叶）3分，附子（炮裂，去皮脐）3分，苦参3分，麦门冬（去心，焙）1两，泽泻1两，干姜（炮）1分，生干地黄（焙）1分，菟丝子（酒浸一宿，别捣）半两。
出处：《圣济总录》卷九十二。
主治：虚劳，小便余沥，尿精。
加减：(疒帬)痹，加附子（炮裂，去皮脐）1分；妇人血伤，加干地黄（焙）半两，黄柏（去粗皮，蜜炙）1分。
功效：None
用法用量：每服30丸，空腹温酒送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
