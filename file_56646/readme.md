序号：56661
名称：消风百解散
组成：防风1两半，苍术1两（制），羌活1两半，甘草5钱，白芷1两，川芎1两，荆芥1两半，柴胡1两，蝉衣1两半，陈皮1两半。
出处：《医略六书》卷二十。
主治：风伤肺气，湿伏皮肤，遏郁不散，身热，白痧疹不出，脉弦者。
加减：None
功效：None
用法用量：每服3钱，水煎，去滓温服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：此方防风散肌表陷伏之风，羌活散皮肤百节之风，白芷散肌肉头面之风，俱能胜湿。而苍术又能燥湿强脾．盖脾属土，上强则自能制湿而生金，金生则肺病气自旺，而风邪不致陷伏。更以荆芥散血分之风，川芎行血中之气，陈皮利气和胃，甘草和胃缓中，蝉衣善透皮肤而发痧疹也，水煎温服，使中外调和，则风湿化而痧疹自透，以热无不退矣。柴胡一味，疏腠升阳，古今不同，恐引动虚阳，则气急鼻煽，势所必至矣。
