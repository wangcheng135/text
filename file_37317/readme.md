序号：37327
名称：定搐化风锭
组成：活蝎子40个，桔梗3钱，黄连3钱，蝉蜕5钱，甘草3钱，防风5钱，羌活5钱，大黄5钱，僵蚕5钱，法半夏5钱，麻黄5分。
出处：《北京市中药成方选集》。
主治：急热惊风，痰延壅盛，神志不宁，咳嗽发烧。
加减：None
功效：镇惊化痰。
用法用量：小儿周岁内每次服半丸，周岁-5岁每次服1丸，温开水送下。
制备方法：先将药料串碎，再将活蝎子用烧酒渍，放在碾上串碎。用药末将活蝎子搜净，取下晒干。再进行粉碎，研细粉过罗合匀。每细粉10两兑：朱砂粉5两，牛黄1钱5分，麝香1钱5分，冰片5钱。以上合匀研细，炼蜜为丸，重5分，满金衣九开，蜡皮封固，包棉纸。
临床应用：None
各家论述：None
用药禁忌：忌食油腻之物。
附注：None
