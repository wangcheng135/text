序号：48441
名称：神效鬼哭膏
组成：香油5斤，柳、槐、桑、杞枝4两半，苏木4钱，降真节4钱，甘草3钱，防风2钱，川乌2钱，草乌2钱，半夏2钱，黄柏1钱半，槐花2钱，红花4钱，厚朴2钱，黄连5钱，蓖麻3钱，江子2钱，牙草4钱，天花粉2钱，川楝2钱，当归须3钱，川椒2钱，南星4钱，五加皮2钱，杜当归4钱，穿山甲2钱，苍术2钱，白及2钱，木鳖子2钱，槟榔2钱，川芎2钱，贝母2钱，白芷2钱，妇人油头发。
出处：《普济方》卷三一三。
主治：杖疮不疼不发无痕，及痈疽，远年恶疮肿毒、风寒暑湿、疼不可忍者。
加减：None
功效：None
用法用量：None
制备方法：上将前药同五枝一处入油，熬至药成炭黑色，用铁笊篱捞去滓，离火候稍温，下黄丹35两，用槐条搅匀，再入火略滚1-2沸，药锅离火，再下乳香、没药、血竭末各1两，搅匀，用生麻布滤入别器内，将麝香1两研，轻粉7钱半在锅内和匀，候经1宿用。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中“妇人油头发”用量原缺。又用法中所言“五枝”，而组成中只有柳、槐、桑、杞四枝，疑脱。
