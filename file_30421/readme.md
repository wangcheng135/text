序号：30429
名称：辰砂既济丸
组成：人参、当归（酒洗）、黄耆（盐水洗，炒）、白山药、牡蛎（酒浸1宿，煅）、锁阳、甘枸杞（蜜拌）、熟地黄（酒洗）4两，知母（去毛，酒洗，略炒）2两，败龟版（酒浸一宿，酥炙）2两，牛膝（酒洗）1两半，破故纸1两2钱，黄柏（酒炒）6钱。
出处：《扶寿精方》。
主治：元阳虚惫，精气不固，夜梦遗精。
加减：None
功效：大补元气，涩精固阳。
用法用量：每服70丸，空心谈盐汤或酒送下，干物压之。
制备方法：上为末，用白术8两，水8碗，煎至一半，取滓再益水煎，漉净，合煎至二碗成膏为丸，如梧桐子大，辰砂研细为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中人参、当归、黄耆、白山药、牡蛎、锁阳、甘枸杞用量原缺。
