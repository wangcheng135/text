序号：48827
名称：禹余粮丸
组成：禹余粮2两半，乌贼骨2两半，吴茱萸2两半，桂心2两半，蜀椒2两半，当归1两6铢，白术1两6铢，细辛1两6铢，干地黄1两6铢，人参1两6铢，芍药1两6铢，芎1两6铢，前胡1两6铢，干姜3两，矾石6铢，白薇18铢，紫菀18铢，黄芩18铢，(庶虫)虫1两。
出处：《千金》卷四。
主治：妇人产后积冷坚癖。
加减：None
功效：None
用法用量：每服20丸，空心酒或饮送下，每日2次，不知，则加之。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：此与鳖甲丸第二方主治相类，彼用白芷，此用白薇；彼用僵蚕，此用前胡；彼用石脂，此用矾石；彼用丹参，此用紫菀；彼用白术，此用甘草；彼用鳖甲，此用(庶虫)虫；彼用鹿茸，此用人参，药虽变易，而功用仿佛。惟彼用附子以助鹿茸、姜、桂之雄，此用黄芩分椒、姜、细辛之悍，泾渭攸分，于此稍异。
用药禁忌：None
附注：None
