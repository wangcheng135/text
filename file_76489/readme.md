序号：76505
名称：葛根加半夏汤
组成：葛根4两，麻黄3两（去节），甘草2两（炙），芍药2两，桂枝2两（去皮），生姜2两（切），半夏半升（洗），大枣12枚（擘）。
出处：《伤寒论》。
主治：太阳与阳明合病，不下利，但呕者。
加减：None
功效：None
用法用量：上以水1斗，先煮葛根、麻黄减2升，去白沫，纳诸药，煮取3升，去滓，温服1升。覆取微似汗。
制备方法：None
临床应用：None
各家论述：1.《古方选注》：葛根汤，升剂也；半夏辛滑，芍药收阴，降药也；太阳、阳明两经皆病，升阖失机，故以升降法治之。麻、葛、姜、桂其性皆升，惟其升极即有降，理寓于其中。又有芍药、甘草奠安中焦，再加半夏以通阴阳，而气遂下，呕亦止，是先升后降之制也。_x000D_
2.《伤寒今释》：葛根汤虽能运输消化管中之水液，然水在胃而不下降者，因胃无吸收水分之能力，必加半夏以止呕降逆，使水液下达于肠，然后葛根汤能成其运输之功也。
用药禁忌：None
附注：葛根半夏汤（《伤寒图歌活人指掌》卷四）。
