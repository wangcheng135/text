序号：57648
名称：大造固真膏
组成：补骨脂6两（盐、酒浸1宿，炒香），胡桃仁（酒蒸，去皮，另研）3两，山药4两（炒黄），山茱萸（去核，酒蒸，焙）3两，菟丝子（酒洗，晒干，炒燥，另磨细末，不出气）4两，小茴香1两5钱（焙），肉苁蓉（酒洗，去鳞甲，焙）2两，巴戟天（酒洗，去心，焙）2两，鹿茸（去毛骨，酥炙）2两，五味子1两5钱（蜜酒拌蒸、晒干，焙），人参2两（锉片，隔纸焙），地黄12两（酒煮，去滓，熬膏），枸杞子6两（水煮，去滓，熬膏3两），干白术（米泔水浸1宿，锉片，晒干，人乳拌蒸，炒黄，水煮，去滓，熬成膏）3两，紫河车1具（酒洗，酒煨，去筋膜，熬成膏）。
出处：《冯氏锦囊·杂证》卷十四。
主治：阳萎。
加减：None
功效：填补精血，壮固元阳。
用法用量：每早、晚食前各服3钱，白汤、温酒任下。
制备方法：前药各制度，共为细末，用后4膏和剂，如干，加炼老蜜少许，杵千下为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：大造固真丹（《类证治裁》卷七）。
