序号：66447
名称：小丁香丸
组成：五灵脂12两，丁香3两，木香1两半，肉豆蔻（去壳）30个，巴豆（去皮膜，出油）210个。
出处：《局方》卷三。
主治：心腹胀闷，胁肋刺痛，胸膈痞满，噎塞不通。小儿宿食不消。
加减：None
功效：消积滞生冷、留饮宿食，止痰逆恶心、霍乱呕吐；常服顺脾胃，进饮食。止呕逆，利关膈，温脾胃，进乳食，定心腹痛。
用法用量：每服5丸至7丸，食后温生姜汤送下，橘皮汤亦得；如霍乱吐逆，煎桃叶汤放冷送下。小儿吐逆不定，3岁儿服3丸，5岁以下服4丸，用生姜、桃叶汤送下。
制备方法：上为细末，入巴豆令匀，面糊为丸，如黍米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：丁香丸（《普济方》卷三九四）。
