序号：25711
名称：芎汤
组成：芎半两，防风（去叉）半两，白术半两，白芷半两，牛膝（去苗）半两，狗脊（去毛）半两，萆薢（炒）半两，薏苡仁（炒）半两，杏仁（汤退去皮尖双仁，炒）1两，人参1两，葛根（锉）1两，羌活（去芦头）1两，麻黄（去根节，先煎，掠去沫，焙干用）2两，石膏（碎）1两半，桂（去粗皮）1两半。
出处：《圣济总录》卷九。
主治：中风，手足不随，身体疼痛，口面斜，一眼不合。
加减：None
功效：None
用法用量：每服12钱匕，以水3盏，煎取2盏，去滓，分3服，微热服之，日2服，夜1服。服药后宜依法次第灸诸穴，风池2穴，肩髃1穴，曲地1穴，支沟1穴，五枢1穴，阳陵泉1穴，巨虚、上下廉各1穴，灸9次即愈。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
