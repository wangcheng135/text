序号：58863
名称：秘制金毛狮子疮药
组成：黄芩2两，荆芥穗2两，黄连2两，五倍子2两，茜草2两，槐枝头2两，芒消2两，黄柏2两（上为粗末，再炒老黄色），土硫黄（醋煮）2两，枯矾2两（上为细末），巴豆肉3钱，蓖麻仁5钱，大风子（净肉）2两（上药捣如泥，同前药和研匀细），水银1两，黑铅1两（同研），雄黄5钱，潮脑5钱（共与前药研和一处）。
出处：《疡医大全》卷三十五引刘氏方。
主治：一切血风疥疮，手足诸疮。
加减：None
功效：None
用法用量：每用1钱，以前制疮药油调搓。凡脚上患血风疮多年不愈，先以药水洗净拭干，再以研细轻粉薄薄扫上，再搓疮药。凡搓疮药先1日用油调鹿角霜细末，厚敷疮上过夜，次日再搓疮药更妙。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：制疮药油法：雄猪油一斤，槟榔、大黄、黄柏、麻黄各一两，水三碗，同药入锅内熬至水干油出，滤去滓，冷定收贮，调一切疮癣擦药。
