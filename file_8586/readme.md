序号：8586
名称：归命丹
组成：蛇黄4两（紫色者佳，用火煅令通赤，取出以纸衬地上出火毒，1宿杵罗为末，更入乳钵研如面），朱砂半两，铁粉1两，獖猪粪2两（野放小硬干者，用饼子固济，烧烟尽为度，勿令白过，恐药少力，候冷，研令细），麝1钱（研）。
出处：《幼幼新书》卷十引《灵苑方》。
主治：感厥急风，心邪痫疾，小儿天钓、惊风及疳热。
加减：None
功效：None
用法用量：一切风，薄荷酒磨下1丸，小儿半丸；疳热，用冷水磨下1丸，分作4服；大人、小儿中风口噤，反张涎满者，灌下1丸，立醒；小儿被惊及发热，并以薄荷磨少许便安。端午及甲午日合；急用不拘。
制备方法：上药都入乳钵内同研极细，糯米粥为丸，如芡实大，挑漆盘于日内晒之。
临床应用：None
各家论述：None
用药禁忌：None
附注：神穴丹（原书同卷）、归命丸、神穴丸（《圣济总录》卷六）。
