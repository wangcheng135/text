序号：27369
名称：补益泽兰丸
组成：泽兰1两，防风1两（去芦头），芎1两，人参1两半（去芦头），肉苁蓉1两（酒洗，去皱皮，炙干），延胡索2两，细辛1两，柏子仁1两半，牛膝1两（去苗），麦门冬1两半（去心，焙），当归1两（锉，微炒），熟干地黄1两，芜荑1两，石膏1两（细研，水飞过），艾叶3分（微炒），薯蓣1两，山茱萸1两，桂心1两，石斛1两半（去根，锉），钟乳粉3两，藁本1两，五味子1两，甘草3分（炙微赤，锉）。
出处：《圣惠》卷七十。
主治：妇人久患羸瘦虚损，四肢百体烦疼，脐下结冷，不能饮食，面目黑，忧恚不乐。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
