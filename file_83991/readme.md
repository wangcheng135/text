序号：84009
名称：五斤丸
组成：大木瓜1斤（去皮瓤），牛膝1斤（去芦头，用无灰酒浸1宿，极干，切，焙），肉苁蓉1斤（酒浸1宿，切，焙），天麻1斤（透明者，切，焙），虎骨4两（涂酥，炙令黄），没药4两（别研），川乌头4两（炮，去皮脐），山药4两。
出处：《杨氏家藏方》卷四。
主治：精血不足，腰脚缓弱，行步艰难，腿膝无力，及寒湿脚气等疾。
加减：None
功效：常服活血驻颜，身轻体健。
用法用量：每服30丸，加至50丸，空心、食前温酒、盐汤任下。
制备方法：上将木瓜烂蒸，研作糊，和众药末，如和不就，更用原浸牛膝酒打面糊搜匀为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
