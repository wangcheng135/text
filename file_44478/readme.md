序号：44493
名称：姜粉散
组成：当归（洗，去芦须，切，焙）、官桂（去皮，不见火）、人参（去芦，切片）、茯苓（白者，去黑皮）、甘草（炙）、芍药（洗净）、知母（润者，切，炒）、川芎（洗）、大黄（略炒）、黄耆（蜜炙）、木香（不见火）、草豆蔻（去壳）、白术（锉，焙）、诃子（煨熟，去核）、高良姜（锉，炒）、青橘皮（去瓤）各等分，熟干地黄（洗，酒浸，焙）。
出处：《卫生家宝产科备要》卷五。
主治：None
加减：None
功效：荡尽儿枕，除百病源。
用法用量：才产后，用药末3钱，水1盏，加生姜3片，枣子1个（劈开）同煎至7分，乘热吃。如吃药之后，自然产母睡着，半日以来，睡觉再服，全除却腹痛。1日只可3服，至9服不可服，恐肚中冷。
制备方法：上药锉，焙干；次用附子1个结实半两者，炮裂，去皮脐，切；生姜1斤，研取自然汁，于碗器中停留一食久，倾去清汁，取下面粉脚，摊在箬叶中，入焙笼焙干，同众药捣罗为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
