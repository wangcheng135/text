序号：44141
名称：复肾散
组成：广狗肾2具，海马50g，鹿肾1对，土茯苓200g，淡菜100g，鹿角菜50g，鲍鱼50g，头发菜50g，砂仁50g，杜仲炭50g，杞果100g，冬虫草50g，酒生地50g。
出处：《千家妙方》引任继学方。
主治：肾功能不全（阴阳俱虚）。浮肿尿少，腹胀，恶心呕吐，纳呆，呼吸气短，不能平卧，口干不欲饮，腰痠膝冷，四肢欠温，颜面青白，口唇淡，舌体肥大，质嫩红，少苔，脉沉弦而数，证属阴阳两虚者。
加减：None
功效：补阴培阳。
用法用量：每服10g，以淡盐汤送下，1日3次。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
