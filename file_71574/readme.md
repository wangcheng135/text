序号：71590
名称：喉症汤药方
组成：大生地5钱，黑玄参4钱，白芍药（炒）2钱，象贝母2钱，粉丹皮2钱，破麦冬3钱，薄荷（后下）1钱2分，生甘草1钱，射干1钱，板蓝根2钱，粉前胡1钱，桔梗1钱，蝉衣5分。
出处：《吉人集验方》。
主治：喉症。
加减：咽喉肿胀，加煅石膏3钱；胸闷，加焦楂2钱，神曲3钱；口渴，加天冬3钱，马兜铃3钱；溺黄赤短，加小木通1钱，知母3钱，泽泻1钱5分，车前子3钱；身热舌黄，加金银花3钱，连翘1钱5分；便结，加大黄3钱，元明粉（冲）2钱；如唇焦舌黑，口出臭气，谵语神昏者，加犀角（磨，冲服）1钱，龙胆草2钱。如遇症重照各症见象加重可也。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
