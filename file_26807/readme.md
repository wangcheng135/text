序号：26815
名称：延龄丹
组成：牛膝（酒浸2宿）7钱半，苁蓉（酒浸2宿）7钱半，金铃子（去皮）7钱半，补骨脂7钱半（炒），川茴香（炒）7钱半，鹿茸（去毛）半两，益智仁半两，檀香半两，晚蚕蛾（炒）半两，没药（研）半两，丁香半两，青盐半两，川山甲半两（炙），沉香1两，香附子（炒）1两，姜黄1两，薯蓣1两，木香1两，巴戟（去心）1两，甘草（炙）1两，乳香（研）3两，白术3两，青皮3两，苍术3两。
出处：《御药院方》卷六。
主治：脾肾不足，真气伤惫，肢节困倦，举动乏力，怠惰嗜卧，面无润泽，不思饮食，气不宣畅，少腹里急，脐下（？？）痛，及奔豚、盲肠气攻充脐腹，发歇无时。
加减：None
功效：补五脏，养真阳，和血脉，壮筋骨。
用法用量：每服40丸，空心温酒送下；或煎茴香汤送下。
制备方法：上为细末。酒煮面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
