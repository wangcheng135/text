序号：61369
名称：乾坤一气膏
组成：当归1两，白附子1两，赤芍1两，白芍1两，白芷1两，生地1两，熟地1两，川山甲1两，木鳖肉1两，巴豆仁1两，蓖麻仁1两，三棱1两，蓬术1两，五灵脂1两，续断1两，肉桂1两，玄参1两，乳香1两2钱，没药1两2钱，麝香3钱，真阿魏2两（切薄片听用）。
出处：《外科正宗》卷四。
主治：痞疾，诸风瘫痪，湿痰流注，各种恶疮，百般怪症。男子夜梦遗精，妇人赤白带下。男女精寒血冷，久无嗣息者。
加减：None
功效：活血杀菌，驱风散寒，渗湿除痰，暖宫调经。
用法用量：痞病红缎摊贴，余病绫绢俱可摊之，有肿者对患贴之。男子遗精、妇人白带，俱贴丹田；诸风瘫痪，贴肾俞穴并效。
制备方法：上(口父)咀。用香油5斤，存下4味，余皆入油浸，春3、夏5、秋7、冬10，期毕，桑柴火熬至药枯，细绢滤清；每净油1斤，入飞丹12两，将油入锅内，下丹，槐枝搅搂，其膏候成，端下锅来，用木盆坐稳，渐下阿魏片，泛化巳尽，方下乳、没、麝香，再搅匀，乘热倾入瓷罐内，分3处盛之，临用汤中顿化。
临床应用：None
各家论述：None
用药禁忌：None
附注：一气膏〔《全国中药成药处方集》（吉林方）〕。
