序号：68202
名称：黄耆萆薢大黄汤
组成：黄耆5两，萆薢5两（即土茯苓），当归3两，芎3两，桂枝3两，防已3两，升麻3两，鲮甲2两，熟大黄1两，附子1两，甘草1两，营实2两（若无，以忍冬花代之）。
出处：《霉疮证治》卷下。
主治：霉疮。发头面手足腹背，似杨梅子紫赤色，有脓泡或无脓泡，或痒或痛，而其人实者；乃结毒坚硬如石，小便常涩痛，难消散，又难溃脓。
加减：None
功效：None
用法用量：加生姜，水煎服。
制备方法：None
临床应用：阴疮脓淋：一人年三30，患阴疮脓淋，杂治4年，今又两腿发横痃，且周身梅疮，筋骨疼痛，余诊之，脉症无虚候，乃制黄耆萆薢大黄汤与之，兼用加味化毒丹，两月诸症尽愈，后调理痊愈。
各家论述：None
用药禁忌：None
附注：None
