序号：82616
名称：橘连枳术丸
组成：白术3两（去梗），枳实1两（去瓤，麸炒），陈皮1两，黄连1两（酒浸，炒）。
出处：《丹溪心法附余》卷七。
主治：None
加减：None
功效：补脾和胃，泻火消痰。
用法用量：每服50丸，食后服。
制备方法：上为末，荷叶煮汤，打米糊为丸，如梧桐子大。
临床应用：None
各家论述：《冯氏锦囊》：易老枳术丸方用白术二两补脾，枳实一两消痞，取其补多消少，至东垣加橘皮一两以和胃，名橘皮枳术丸，则补消相半也。今更用白术三两，枳实一两，陈皮一两，黄连一两，名橘连枳术丸，仍补多消少，又兼清热也。丹溪云：心下痞，须用枳实炒黄连，是也。
用药禁忌：None
附注：None
