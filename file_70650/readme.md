序号：70666
名称：紫菀汤
组成：紫菀2两，五味子2两，生姜（合皮，切）2两，白石英（研，绵裹）2两，款冬花2两，桂心2两，人参2两，钟乳（研，绵裹）3两，麦门冬（去心）3两，桑根白皮3两，大枣20枚（擘），粳米1合。
出处：《外台》卷十引《广济方》。
主治：肺气不足，逆气胸满，上迫喉咽，闭塞短气，连唾相属，寒从背起，口如含霜雪，语无音声，剧者唾血腥臭，或歌或哭，干呕心烦，耳闻风雨声，皮毛悴，面白。
加减：None
功效：None
用法用量：上切。水1斗5升，先煮桑根白皮、粳米，取9升，去滓，纳诸药，煎取3升。去滓，分温3服，每服相去如人行7-8里久。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌生葱、热面、炙肉。
附注：None
