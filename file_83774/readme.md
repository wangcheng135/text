序号：83792
名称：乌蛇丸
组成：乌蛇（1条，酒浸，去皮骨头尾）3两，檀香（锉）3分，丁香3分，茜根（锉）3分，紫葛（锉）3分，防风（去叉）3分，苦参3分，独活（去芦头）3分，沙参（去芦头）3分，桅子仁3分，酸枣仁（生用）3分，槐子（炒）3分，白芷3分，附子（炮裂，去皮脐）3分，藁本（去苗土）3分，羚羊角（镑）3分，苍术（锉，醋炒）3分，犀角（镑）3分，羖羊角（镑）3分，防己3分，桃胶（生用）3分，萆薢（炒）3分，芜荑仁（炒）3分，栝楼根（锉）3分，秦艽（去苗土）3分，乌药（锉）3分，桑根白皮（锉）3两，木通（锉）1两半。
出处：《圣济总录》卷十八。
主治：大风癞疾。
加减：None
功效：None
用法用量：每服10-15丸，空心温酒送下，1日2次。不饮酒者，薄荷、槐白皮汤送下。若要速愈，宜用桑枝汤送下。
制备方法：上为末，炼蜜为丸，如赤小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
