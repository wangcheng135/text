序号：83441
名称：乌金散
组成：鲮鲤甲1两，犀角（镑）1两，黄明胶1两，赤鲤鱼皮1两，胎发1两半，独角仙1枚（去翅头足）。
出处：《圣济总录》卷六十八。
主治：吐血及一切血病，诸药不效者。
加减：None
功效：None
用法用量：每服1钱匕，旋入腻粉少许，吐血鼻衄不止，新汲水调下；产后血晕，昏迷闷乱，不知人，冷醋汤调下；血气，温酒调下；咯血及血积、脏毒下血、赤痢、血痢、蛊毒痢、肠风、五痔下血，并米饮调下，临卧空腹服；更相度虚实，可加至2钱匕。服药后，取下积聚物为效。
制备方法：上用瓦瓶1枚，底下开窍，纳药，以纸筋泥固济，晒干；用炭火5斤簇烧，候烟绝，拨去火，放冷取出，细研为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
