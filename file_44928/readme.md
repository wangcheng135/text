序号：44943
名称：星石降痰丸
组成：青礞石2两（焰消4两，将礞石捶碎，拌匀，装瓷瓶内，以瓦盖之，盐泥封固1寸厚，大火煅过，放地上1宿，取出研末），南星2两（掘地坑敲实，炭火烧红，将米醋2碗泼下，随下南星，以瓦盆盖之，四旁用泥封固，次日取出，为末），辰砂1两（水飞），沉香1两，滑石1两（甘草水煮过），猪牙皂角（去皮净）7钱，瓜蒌仁7钱，贝母7钱，黄芩5钱，枯矾5钱，荆芥穗5钱。
出处：《万氏家妙方》卷五。
主治：小儿惊风，痰壅喘急。
加减：None
功效：None
用法用量：白汤化下。
制备方法：上为末，生姜汁炼蜜为丸，如龙眼核大。
临床应用：None
各家论述：None
用药禁忌：None
