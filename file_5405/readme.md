序号：5405
名称：二十二味虎睛丸
组成：虎睛1只（生眼佳，晒干，酒浸令黄色），真珠3钱，蜂房3钱，麻黄2分（去节），钩藤3分，铁精4分，防葵4分，大黄4分，子芩4分，龙齿4分，银屑4分，栀子仁4分，羌活4分，柴胡3分，升麻3分，白鲜皮3分，雷丸（烧令赤）3分，人参3分，细辛1分半，蛇皮5寸（炙），石膏5分，蚱蝉4枚（去翅足，炙）。
出处：《颅囟经》卷上。
主治：孩子从1岁至大，癫发无时，口出白沫，小便淋沥不利。
加减：None
功效：None
用法用量：4-5岁5丸，大儿10丸，浓煎米饮送下，每日2次。
制备方法：上为末，炼蜜为丸，如赤豆大。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻。
附注：虎晴丸（《普济方》卷三七七）。
