序号：38165
名称：松烟饼子
组成：细墨5分（烧，研），陈皮5钱（去白），牵牛（别研，取头末）5钱，神曲（炒）5钱，三棱（火煨）5钱，密陀僧（研）5钱，五灵脂（研）5钱，硇砂（研）5钱，牡蛎（火煨，煅）5钱，麦糵（炒）5钱，大黄1两，北枣14个（烧存性），斑蝥1两（去翅足，糯米同炒），芫花（醋浸1宿，炒）1钱，干漆（炒去烟）1钱，白丁香（研）1钱，大戟（去芦）1钱，青礞石（研）1钱，蓬莪术（煨）1钱，巴豆1两（去皮，湿纸裹烧，黄色为度）。
出处：《瑞竹堂方》卷一。
主治：积气瘀血痞塞，大人、小儿久痢或休息痢，并男子、妇人年深不伏水土，及暑月变成恶痢，米汤不消，五痞块逆，隔胃吐食，心胸闷闭，酒疸食黄，劳嗽上喘，呕逆涎沫，心闭惊恐，口苦恶心，小便淋涩，大便不通，伤寒余毒，妇人胎前产后，败血结成积块，饮食平常，遍身疼痛，腰强腿硬，手足眩厥，九种心疼，十般积热，九般水气，霍乱吐泻，久病瘦弱。
加减：None
功效：消食快气。
用法用量：临用为粗末，记以所伤，煎汤送下，或面汤亦可，小儿3饼，大人看虚实禀气加4-5饼。其积块渐渐近下，再进1服，又觉近下。
制备方法：上为细末，水打面糊为丸，如皂角子大，捻为饼子。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
