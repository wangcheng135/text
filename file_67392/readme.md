序号：67408
名称：黄丹膏
组成：黄丹24两（微炒，细罗），麻油2斤半，猪脂8两（腊月者），松脂4两，紫花1两（去土），当归1两，防风1两（去芦头），黄芩1两，莨菪子2两，棘针49枚（头曲者），青绯帛各2尺（烧灰），人粪灰1两，青柏叶1两，蜥蝎7枚，乱发如鸡子大，蜡3两，葱（并根）20茎。
出处：《圣惠》卷六十三。
主治：痈疽发背，痈肿丹毒，一切疮疖。
加减：None
功效：内消止痛。
用法用量：摊在故帛上贴之。
制备方法：上锉，先下油脂于锅内煎令熔，次下药，以文火煎半日，次下松脂、蜡，候香熟，以绵滤去滓，都入药油于锅中，纳黄丹，不住手搅令匀，候色变紫色，收得油方尽，软硬得所，用瓷盒盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
