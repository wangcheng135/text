序号：31241
名称：沉香饮
组成：沉香1两，白蒺藜（炒去角）1两，补骨脂（炒令香）1两，巴戟天（去心）1两，酸枣仁（炒）1两，五味子（炒）1两，泽泻1两，磁石（煅，醋淬7度）1两，桂（去粗皮）1两，人参1两，陈橘皮（去白，焙）1两，枳壳（去瓤，麸炒）1两，牛膝（切，酒浸，焙）1两，芍药1两，石斛（去根）1两，鳖甲（醋炙，去裙襕）1两，槟榔3两，桑螵蛸3两，肉苁蓉（酒浸，切，焙）2两，当归（切，焙）2两，柴胡（去苗）2两，黄耆（锉，炒）2两，芎3两，附子（炮裂，去皮脐）1两半。
出处：《圣济总录》卷八十六。
主治：五劳七伤，肾气虚乏。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，加生姜5片，煎取8分，去滓，空心温服。
制备方法：上锉细。
临床应用：None
各家论述：None
用药禁忌：None
附注：沉香散（《普济方》卷三十）。
