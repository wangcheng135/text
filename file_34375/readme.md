序号：34384
名称：万金膏
组成：五倍子1钱，赤芍药2钱半，白芷2钱半，大黄2钱半，官桂2钱半，当归2钱半，玄参2钱半，干地黄2钱半（以上第1次煎），虢丹4两（第1次入），当归2钱半，羌活1钱，云母（别研）1钱，巴豆35粒（以上第2次煎），乳香1钱（别研），滑石1钱（别研），白胶香（别研）1钱，没药（别研）2钱半，虢丹4两（以上第2次入）。
出处：《百一》卷二十。
主治：诸般恶疮。
加减：None
功效：None
用法用量：None
制备方法：上锉细，用真清麻油1斤，先将半斤入铫煎沸，下五倍子、芍药、白芷、大黄、官桂、当归、玄参、地黄8件，以柳枝1条搅油，候药带焦色，用绵一绚滤去滓，再将药油入铫，候略沸，下虢丹4两，打转看候紫色，将油1-2点滴在水面上，成珠子，即便倾出，安稳处。再将清油半斤，如前煎沸，下当归、羌活、云母、巴豆及青杨柳枝皮指面大10-20片，同煎，候药带焦，用绵子滤去滓，再煎药油，下乳香、滑石、胶香、没药、虢丹同煎，候如前带紫色，滴入水中，才成珠子，却将前煎药油倾作1铫同煎，打匀为度。煎药时不要火太紧，恐煎过药味。
临床应用：漏疮：唐仲举云：渠令嗣颏颊间苦一漏疮年余，用此膏两枚而愈。后以治他人，亦多验。
