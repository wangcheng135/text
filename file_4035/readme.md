序号：4035
名称：木瓜丸
组成：木瓜（去皮瓤，切，焙）10枚，木香3两，人参1两半，肉豆蔻（去皮）半两，陈橘皮（汤浸，去白，焙）2两，胡椒3两，槟榔（挫）3两，吴茱萸（汤浸，焙炒）2两，草豆蔻（去皮）3两，厚朴（去粗皮，姜汁炙）2两，桂（去粗皮）1两，苍术（米泔浸1宿，刮去皮，焙）2两，缩砂（去皮）3两，高良姜3两，生姜1斤（捣绞取汁）。
出处：《圣济总录》卷四十。
主治：霍乱，心下气痞不通。
加减：None
功效：None
用法用量：每服10丸，熟水送下。
制备方法：上药除生姜汁外，为细末。取生姜汁拌匀，用瓷瓶盛于锅中，以重汤煮一复时，取出，更捣令匀，众手丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
