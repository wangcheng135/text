序号：48525
名称：神效楸叶煎
组成：楸叶15斤（秋分前后取）。
出处：《圣惠》卷六十六。
主治：瘰疬成瘘。
加减：None
功效：None
用法用量：凡患者，取麻油半合，蜡1分，酥1栗子大，同消如面脂，又取杏仁7粒，生捣如膏，米粉2钱，同入面脂中搅令匀，先涂疮上，经2日以来，净拭去，以篦子匀涂楸叶煎满于疮上，仍用软帛裹之，2日1度，拭去旧药，更上新药，不过5-6上。已作头者，便生肌平复；如未穴，即内消。
制备方法：上用水1石，于净釜中煎楸叶，取汁3斗；又重换锅，煎至1升，已成煎矣，盛于不津器中。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“楸叶煎”。
