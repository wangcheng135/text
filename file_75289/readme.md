序号：75305
名称：川朴消丸
组成：川朴消2两（细研），川芒消1两（细研），马牙消半两（细研），川乌头1两（生，去皮脐，捣罗为末），椒目1两（微炒，捣罗为末），甜葶苈1两（隔纸炒令紫色），莨菪子1两（水淘去浮者，水煮牙出，候干，炒令黄黑色），杏仁2两（汤浸，去皮尖双仁，麸炒微黄）。
出处：《圣惠》卷五十四引《神仙密藏经》。
主治：十种水气。
加减：None
功效：None
用法用量：每服20丸，空心以桑根白皮汤送下。
制备方法：上药将葶苈、莨菪、杏仁等相和，先捣一千杵，取大枣10个，煮取肉，与上药细研令匀，然后入炼蜜为丸，如梧桐子大。
临床应用：水病有人先患脚气十余年，发盛便成水病。四时之中，遍身肿满，腹硬如石，水饮难下，靡觉饥渴，但喘粗不得睡卧，头不着枕，二百余日，无问昼夜，即呷粥饮，常须倚物而坐，羸弱异常。因服此药，当日气散。十日后肚硬消尽，二十余日后气力如旧。
各家论述：None
用药禁忌：忌咸物。
附注：None
