序号：71194
名称：黑散
组成：麻黄半两，大黄6铢，杏仁半两。
出处：《千金》卷五。
主治：小儿变蒸，中挟时行温病；或非变蒸时而得时行者。小儿伤寒发热。
加减：None
功效：None
用法用量：1月儿服小豆大1枚，以乳汁和服，抱令得汗，汗出温粉粉之，勿使见风。百日儿服如枣核大。
制备方法：先捣麻黄、大黄为散，别研杏仁如脂，乃细细纳散，又捣令调和，纳密器中。
临床应用：None
各家论述：《千金方衍义》：于变蒸之中复挟时行邪气，非急为开提中外，何以保全万一？方中大黄荡涤内结，用麻黄开发表邪，杏仁疏利逆气。盖大黄原有安和五脏之功，麻黄兼有破除癥坚之力，杏仁交通中外，乃麻黄汤之变方，守真通圣、双解从此悟出。
用药禁忌：None
附注：黑膏子（《普济方》卷三六一）。
