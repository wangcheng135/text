序号：65699
名称：菖蒲汤
组成：菖蒲（米泔浸1宿，切）4两，木通（锉）3两，瞿麦2两（用穗），杏仁（去皮尖双仁，炒）3两，白术（锉碎，炒）3两，独活（去芦头）4两，山芋3两，甘草（炙，锉）2两，附子（炮裂，去皮脐）2两，桂（去粗皮）3两，茯神（去木）2两，人参3两，石膏2两，前胡（去芦头）3两，磁石（火烧醋淬7遍）2两。
出处：《圣济总录》卷一一四。
主治：风聋。
加减：None
功效：None
用法用量：每服3钱匕，以水1盏半，加竹叶7片，生姜1枣大（切），葱白1寸，同煎至7分，去滓温服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
