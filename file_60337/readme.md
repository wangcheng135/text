序号：60352
名称：通圣膏
组成：真麻油1斤，无蛀皂角1尺2寸（去核，捶碎），降真香2两（捶碎），巴豆40粒（去壳，劈开去心）。
出处：《普济方》卷三一四。
主治：痈疽漏疮，一切恶疮。
加减：None
功效：None
用法用量：摊纸上贴患处。
制备方法：上用柳枝1握，长4-5寸，以草系定，先煎油转黑色（火稍猛不妨），取下稍定，再下柳枝煎，少时除去，而后依次下降真香、皂角、巴豆，用火稍慢以防溢涌，又以尺许长柳2条，时时搅转，候巴豆紫黑色为度；以绵或绢绞去滓，入红色虢丹8两，分作2-3次下，不住手搅转，候黑光，滴少许入净冷水内，以不粘手为度，倾入石器中，乘热入少许乳香末尤妙。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
