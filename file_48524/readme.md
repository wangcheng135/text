序号：48539
名称：神效膏
组成：当归2两，白芷1两半，乳香3分（细研），松脂1两，芎1两，白蔹1两半，绯帛灰半两（细研），乱发灰半两（细研），甘草1两半，黄丹10两，木鳖子30枚（去壳），杏仁1两（汤浸，去皮尖双仁，炙），木香1两半，黄蜡2两，麻油2斤。
出处：《圣惠》卷六十三。
主治：一切痈疽发背，溃后肌肉不生。
加减：None
功效：排脓生肌。
用法用量：每用以故帛摊贴，每日换2次。
制备方法：上先取油安铛内，炼令香熟，将8味药细锉，下油中浸1宿，以文火煎白芷色赤黑，即滤出，次下松脂、蜡、乳香、绯帛、发灰等，更煎令消，以绵滤去滓，都入铛内，下黄丹，不住手搅，变黑光色，滴在水中为珠子，膏成，用瓷器盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
