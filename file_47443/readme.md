序号：47458
名称：祛疯酒
组成：大熟地2两，龙眼肉2两，全当归5钱，潞党参5钱，炙绵耆5钱，米仁5钱，茯神5钱，甘枸花5钱，炒白芍3钱，炒冬术3钱，千年健3钱，海风藤3钱，羌活3钱，独活3钱，虎胫骨3钱，钻地风3钱，五加皮3钱，杜仲3钱，忍冬藤3钱，川续断3钱，牛膝3钱，淡附片2钱，瑶桂心2钱，炙桂枝2钱，虎头蕉2钱，明天麻2钱，川芎2钱，炙甘草2钱，广木香1钱5分，红花1钱5分。
出处：《经验奇方》。
主治：一切疯痛，半身不遂。
加减：None
功效：None
用法用量：随量温饮，1日2次。
制备方法：上药用陈绍酒浸瓷瓶，瓷盘作盖，棉纸封口，重汤炖至点3炷香时为度。
临床应用：None
各家论述：None
用药禁忌：孕妇忌服。
附注：None
