序号：46969
名称：独活续断汤
组成：独活2两，续断2两，杜仲2两，桂心2两，防风2两，芎3两，牛膝2两，细辛2两，秦艽3两，茯苓3两，人参2两，当归2两，芍药2两（白者），干地黄3两，甘草3两（炙）。
出处：《外台》卷十七引《古今录验》。
主治：肾气虚弱，卧冷湿地，当风所得，腰痛，不时愈，久久流入脚膝，冷痹，痛弱重滞，或偏枯，腰脚疼挛，脚重急痛。
加减：None
功效：None
用法用量：以水1斗，煮取3升，分3服。温将息，勿取冷，宜用蒴藋叶火燎，厚安床上，及热卧上，冷即易之，冬日取根，捣用，事须熬之。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：忌芜荑、生葱、生菜、海藻、菘菜、酢物。
附注：None
