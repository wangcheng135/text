序号：50383
名称：茵芋散
组成：茵芋3分，桂心3分，汉防己3分，附子3分（炮裂，去皮脐），侧子3分（炮裂，去皮脐），芎藭3分，人参1分（去芦头），麻黄3分（去根节），当归3分（锉，微炒），赤芍药3分，秦艽1两（去苗），茯神3分，五加皮3分，防风半两（去芦头），白术半两，黄耆半两（锉），细辛半两，甘菊花半两，甘草半两（炙微赤，锉）。
出处：《圣惠》卷二十一。
主治：偏风，积年不愈，脚手枯细，口面斜，精神不守，言语倒错。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，加生姜半分，煎至5分，去滓，下竹沥1合，更煎1-2沸，温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻、毒鱼、滑物。
附注：None
