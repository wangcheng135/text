序号：33105
名称：万应灵丹
组成：川芎1两（瓦上焙脆），石菖蒲3钱（瓦上炒），白芷6钱（去梗，净），羌活8钱（晒），苏叶6钱（去梗，净），茅术1两（生切，晒脆），半夏3钱（生用，姜汁拌晒），薄荷8钱，大黄1两（生用），木香5钱（晒脆），川乌5钱（汤泡，去芦皮），草乌5钱（汤泡，去芦皮），独活4钱（晒脆），当归1两，葛根6钱，细辛3钱，胆星5钱（另研），甘草5钱（生用），牙皂3钱（生研），蟾酥5钱（另研），明矾5钱（另研），麝香1钱（另研）。
出处：《家用良方》。
主治：受暑感风，冒寒挟湿，气闭发痧，肚腹胀痛，呕吐泄泻，山岚瘴气，痰迷气逆，头风心痛，中邪中恶，厥气迷闷，羊癫诸风，及妇人产后惊风，小儿急漫惊风。
加减：None
功效：None
用法用量：视病之轻重，每服20-50丸，老幼减半，沸水待温送下。再研数丸，吸鼻取嚏。
制备方法：上药各为极细末，用鬼箭羽2两，煎浓汤滴为丸，如粟米大，飞雄黄为衣，晒干，瓷罐装存，勿令泄气。
临床应用：None
各家论述：None
用药禁忌：忌食鱼腥、生冷、面食及难消化之物；孕妇忌服。
附注：None
