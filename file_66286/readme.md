序号：66302
名称：银粉丸
组成：上好银2两（打作薄片，用猪脂2斤，煎令脂尽，又以好醋1升，亦煎令尽，细擘银片，以水银3两相和，火上熬令极热，即泻于水碗中，研令极细即止，纳入铁铛中，以瓷碗盖，以火逼飞，却水银令尽即出，捣罗为末，更细研入后药），人参1两（去芦头），茯神1两，石膏1两，虎睛1对（酒浸1宿，微炒），牛黄1分（细研），铅霜1两（细研）。
出处：《圣惠》卷二十二。
主治：风痫。失心狂乱，不识好恶。
加减：None
功效：None
用法用量：每服1丸，以金银汤研下，不拘时候。
制备方法：上为末，研入银粉令匀，以枣肉为丸，如皂荚子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
