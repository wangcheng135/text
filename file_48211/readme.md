序号：48226
名称：神明青膏
组成：蜀椒5合，皂荚3铢，黄芩3铢，石南3铢，黄连3铢，雄黄3铢，桂心3铢，藜芦3铢，白术7铢，芎7铢，大黄7铢，乌头5铢，莽草5铢，续断5铢，泽泻7铢，半夏12铢，当归12铢，干地黄11铢，萎蕤10铢，细辛10铢，附子2铢，桔梗2铢，干姜6铢，人参5铢，戎盐杏子大1枚。
出处：《千金》卷七。
主治：鼻中干。
加减：None
功效：None
用法用量：病在外，火炙摩之；在内，温酒服如枣核，1日3次，稍稍益，以知为度；鼻中干，灌之并摩服。
制备方法：上（口父）咀，以苦酒1斗渍之。羊髓1斤，为东南三隅灶，纳诸药，炊以苇新。作三聚新好土，药沸即下，置土聚上，三沸三下讫，药成，以新布绞去滓。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
