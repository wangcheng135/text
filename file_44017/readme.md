序号：44032
名称：咳特灵
组成：小叶榕。
出处：广州白云山制药总厂。
主治：慢性气管炎。
加减：None
功效：消炎镇咳，祛痰平喘。
用法用量：每服片剂3片，1日2次，或胶囊1粒，1日3次。
制备方法：上药加水煎煮二次，合并煮液浓缩加浸膏，在充分搅拌下加入乙醇，静置几小时，滤取上清液，浓缩成膏，干燥后加入扑尔敏，制片或装胶囊。
临床应用：以本方治疗慢性支气管炎291例，年龄40-86岁，病程2-40年，其中10年以上者90例（30.9%），10天为一个疗程，一般用1-3个疗程。结果：有效率（咳痰，喘症减少30%以上者）96.5%，显效（咳痰、喘症减少60%以上者）为83.85%。
各家论述：None
用药禁忌：None
附注：None
