序号：66317
名称：雀儿药粥
组成：雀儿10枚（剥去皮毛，剥碎），菟丝子1两（酒浸3日，晒干，别捣为末），覆子1合，五味子1两，枸杞子1两，粳米2合，酒2合。
出处：《圣惠》卷九十七。
主治：肾气不足所致的阳虚羸弱，阴痿（即性机能减退），早泄，遗精，腰膝酸痛或冷痛，头晕眼花，视物不清，耳鸣耳聋，小便淋沥不爽，遗尿多尿，妇女带下。
加减：None
功效：壮阳气，补精血，益肝肾，暖腰膝。
用法用量：将雀肉先以酒炒，入水3大盏，次入米煮粥，欲熟，下药末5钱，搅转，入五味调和令匀，更煮熟空心食之。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：发热病人和性机能亢进者忌服。
附注：None
