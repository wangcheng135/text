序号：2872
名称：天竺黄散
组成：天竺黄半两，郁金（无，以山栀仁代）半两，茯神（去皮）半两，甘草半两，硼砂2钱半，牙消2钱半，白芷2钱半，川芎2钱半，僵蚕（去丝）2钱半，枳壳（麸炒）2钱半，朱砂（水飞）2钱，麝香1字，蝉壳15个（洗，去泥土嘴足）。
出处：《活幼心书》卷下。
主治：上焦风热，口鼻生疮，两目赤肿，咽膈不利，痰涎壅滞，气不通畅，惊搐烦闷，神思昏迷。
加减：None
功效：None
用法用量：每服半钱或1钱，温薄荷汤或麦门冬汤调服，不拘时候。
制备方法：上除硼砂、牙消、朱砂、麝香4味乳缽细杵，余9味焙干，为末，同入乳钵内再杵匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
