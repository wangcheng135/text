名称：沉香续断丸
组成：沉香（锉）3分，续断3分，牛膝（炒）3分，石斛3分，茴香子（炒）3分，补骨脂（微炒）3分，荜澄茄3分，山茱萸3分，防风（去叉）3分，熟干地黄3分，白茯苓（去黑皮）3分，杜仲（去粗皮，炙）3分，肉苁蓉（酒浸，切，焙）3分，菟丝子（酒浸1宿，别捣）1两，桂（去粗皮）1两，鹿茸（去毛，酥炙）1两，附子（炮裂，去皮脐）1两，泽泻1两，石龙芮1两，巴戟天（去心）半两，桑螵蛸（炒）半两，芎半两，五味子半两，覆盆子半两，木香半两。

主治：骨髓伤败。

制备方法：上为末，酒糊为丸，如梧桐子大。
