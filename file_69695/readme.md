序号：69711
名称：琼液膏
组成：熊胆1钱，牛黄1钱，龙脑半钱（为末），蕤仁1钱（去皮），硼砂1钱（为末），黄连1两，蜂蜜2两。
出处：方出《臞仙活人心方》卷下，名见《医方类聚》卷七十。
主治：远年近日一切不疗眼疾。
加减：None
功效：None
用法用量：每用铜箸少许点于患目内，瞑目片时，候药性过，日点3次。
制备方法：上熊胆、牛黄、蕤仁、黄连4味，用长流水2大碗，于瓷器内熬至半碗，用重绵滤过，去滓，入蜜，再用文武火熬至紫色，蘸起牵丝为度，不可太过不及，取出，入硼砂、龙脑末和匀，瓷瓶内封固，入土埋7日，出火气。
临床应用：None
各家论述：None
用药禁忌：忌动风热物。
附注：None
