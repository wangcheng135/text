序号：40509
名称：虎骨酒
组成：虎骨1具（通炙，取黄焦汁尽）。
出处：《千金》卷十九。
主治：骨虚，酸疼不安，好倦，主膀胱寒。骨极者，腰脊酸削，齿痛，手足烦疼，不欲行动。
加减：None
功效：None
用法用量：碎之，如雀头大，酿米3石，曲4斗，水3石，如常酿酒法。酒熟封头50日开饮之。
制备方法：None
临床应用：None
各家论述：1.《千金方衍义》：虎骨追风定痛，强筋壮骨，酿酒以助其势而散膀胱之寒也。_x000D_
2.《医方考》：肾主骨，骨极者，骨内空虚之极也，故令腰脊酸削；齿者，骨之余，故齿亦痛；手足烦疼，不欲行动，皆骨内空虚之征也。以骨治骨，求其类也；以虎骨治骨，取其壮也；酿之以酒者，取酒性善渍，直彻于骨也。
用药禁忌：None
附注：None
