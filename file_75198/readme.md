序号：75214
名称：腽肭脐丸
组成：腽肭脐1对，鹿角胶2两，鹿角霜2两，麋茸2两，熟地黄1两，葫芦巴1两，菟丝子1两，巴戟（去心）1两，钟乳粉1两，乳香1两，没药1两，当归1两，牛膝1两，苁蓉1两，茴香1两，天雄1两，附子1两，沉香半两，朱砂2钱，麝香1钱。
出处：《朱氏集验方》卷八。
主治：虚损。
加减：None
功效：补真助阳，益壮根本。
用法用量：每服30-50丸，空心温酒送下。
制备方法：先将腽肭脐酒浸1宿，煮烂，杵成膏，将浸药酒化麋鹿胶同为膏子，次下乳香、麝香、没药、朱砂、乳粉研为末，将余药研为细末，再于干钵内同研千百下，别用羊白腰子3对，羊白脊髓5条，酒煮熟烂，研为膏，用腽肭脐，麋鹿角胶，搜拌药末成剂得所，若稍干打酒糊些小，同搜成剂，入臼杵三五千杵，丸如梧桐子大，窨1宿，慢火上焙干，用无油罐子盛，纸密封上。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
