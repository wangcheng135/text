序号：69668
名称：琥珀散
组成：琥珀半两（细研），硫黄半两（细研），硇砂1两，没药半两，麒麟竭半两，斑蝥1分（炒熟，去翅足），水蛭半两（炒令黄），桂心1两，干漆半两（捣碎，炒令烟出），海马子9枚，当归1两（锉，微炒），虻虫1分（去翅足，微炒），芫花1两（以醋拌过，炒令干），麟香1分（研入）。
出处：《圣惠》卷七十九。
主治：产后，脏腑夙有风冷，恶血下少，结积成血瘕，致月水不利。
加减：None
功效：None
用法用量：每服1钱，以酒半盏，童便半盏，桃仁7枚（去皮尖，研），同煎1-2沸，每日空心服。当下恶滞物，以愈为度。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
