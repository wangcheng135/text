序号：56737
名称：消石大丸
组成：消石12两（熬之令干），蜀椒1升2合（去目闭口；汗），水蛭100枚（熬），虻虫2两半（去翅足，熬），大黄1斤，茯苓6两，柴胡8两（去苗），芎藭5两，蛴螬30枚（熬）。
出处：《千金翼》卷十五。
主治：男子女人惊厥口干，心下坚，羸瘦不能食，喜卧，坠堕血瘀，久咳上气胸痛，足胫不仁而冷，少腹满而痛，身重目眩，百节疼痛，上虚下实。又主女人乳余疾带下，五脏散癖，伏热，大如碗，坚肿在心下，胸中津液内结，浮肿膝寒，蛊毒淫跃，若渴大虚。
加减：若女人月经闭，加桃仁30枚（去皮尖双仁，熬）。
功效：None
用法用量：每服5丸，空腹以饮送下，日3次，5日进10丸，此皆不下，自此以后任意加之，1日可数10丸。与羊臛自补，若利当盆下之，勿于圊。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：慎风冷。
附注：None
