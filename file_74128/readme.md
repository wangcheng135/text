序号：74144
名称：愈风散
组成：荆芥（略焙为末）。
出处：《妇人良方》卷十九引华佗方。
主治：产后中风，口噤，牙关紧急，手足瘈疭如角弓状，血晕，四肢强直，不省人事。
加减：None
功效：None
用法用量：每服3钱，豆淋酒调下，用童子小便亦可，口噤者灌，齿龈噤者吹鼻中皆效。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：如圣散（《证类本草》卷二十八引《经验方》）、青金散（《产宝诸方》）、荆芥散（《卫生家宝产科备要》卷六）、再生丹（《续医说》卷九引《曾公谈录》）、独行散（《本草纲目》卷十四）、华伦愈风散（《准绳·女科》卷五）、举轻古拜散（《明医指掌》卷九）。
