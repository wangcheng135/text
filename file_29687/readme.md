序号：29695
名称：阿魏丸
组成：阿魏（细研）1两，当归（切，炒）1两，芜荑仁（炒）1两，雌黄（研）半两，猪牙皂荚（去皮子，酥炙）半两，麝香（研）3分，蓬莪术（煨，锉）3分，柴胡（去苗）2两，白槟榔（锉）2两。
出处：《圣济总录》卷九十三。
主治：传尸骨蒸，女人血气，月候经年不通，痰嗽黄瘦，四肢羸弱，盗汗骨蒸，或时憎寒，饮食减少。
加减：None
功效：None
用法用量：每服50丸，五更温酒送下。仍饮令醉，以青绢被盖之，睡觉汗出通身，必有细虫在被间，日光内看，急须烧之，或泻下恶物并虫等是效。
制备方法：上为末，和匀再罗，用羊肉半斤，去皮烂煮，细切，研如膏，入诸药末，和捣三千杵，如硬，即添肉汁，为丸如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
