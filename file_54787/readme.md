序号：54802
名称：桂心丸
组成：桂心半两，白石英2两半，麦门冬2两半，枳实2两半，白鲜皮2两半，贝母2两半，茯神2两半，槟榔仁2两半，天门冬2两半，车前子1两，人参1两半，前胡1两半，橘皮1两半，白薇1两半，杏仁1两半，郁李仁3两，桃仁5分。
出处：方出《千金》卷十七，名见《普济方》卷一八三。
主治：积气热发，气上冲不得息，欲死不得卧。
加减：None
功效：None
用法用量：每服10丸，加至30丸，1日2次，以竹叶饮送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：本虚邪实，故用桂心、石英、人参、茯神、天麦二冬固本滋津；前、杏、橘、贝、白薇、白鲜皮疏肺泄气；槟榔、车前、郁李、桃仁分调血气而顺下之。更以竹叶饮子清润诸味，但加生姜之辛散而下前丸，上下开泄其气也。
用药禁忌：None
附注：None
