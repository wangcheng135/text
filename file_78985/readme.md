序号：79001
名称：增损普济消毒饮
组成：黄芩10g，黄连5g，元参10g，连翘12g，马勃5g，牛蒡子10g，薄荷5g，僵蚕10g，金银花15g，芦根30g，荆芥5g，板兰根15g，苦梗3g。
出处：《温病刍言》。
主治：流行性腮腺炎。头面部及咽喉肿痛，如无名肿毒，牙龈肿痛，急性扁桃体炎等。
加减：大便燥，加酒军。
功效：清热解毒，疏风消肿。
用法用量：None
制备方法：None
临床应用：None
各家论述：方中芩、连、银、翘、板蓝根、马勃清热解毒以消肿，且马勃清轻专上焦，故能治头面咽喉肿痛；牛蒡子、僵蚕、薄荷、荆芥疏风散邪，俾毒热由表而散；苦梗载药上行开泄上焦；元参治浮游之火以利咽喉。
用药禁忌：None
附注：None
