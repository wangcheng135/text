序号：54229
名称：大岩蜜丸
组成：防风、灵脂、延胡、生地、当归、甘草、赤芍、益母草、川芎、干姜、乌药、白芷、蒲黄、陈皮、细辛、吴茱萸。
出处：《陈素庵妇科补解》卷五。
主治：产后心痛，因虚寒血凝上冲于心之包络。
加减：None
功效：温经散寒。
用法用量：None
制备方法：None
临床应用：None
各家论述：产后虚寒血凝，故用四物加益母、蒲黄、甘草以养血，干姜、吴茱萸、乌药、陈皮以温中，防风、细辛、白芷以逐寒，温经散寒则血不滞；恐其凝而上冲也，加延胡、灵脂以祛瘀止痛。至于火因血虚而上升冲心包络致痛者，则非吴萸、干姜之辛热所能止者，即痛止而血必得热则行，血来太甚，产妇益弱矣。
用药禁忌：None
附注：None
