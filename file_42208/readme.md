序号：42223
名称：青金膏
组成：信砒1两，乳香1两，轻粉1两，粉霜1两，巴豆1两（同研），龙脑半字，麝香半字，青黛2钱（同研），黄蜡3钱。
出处：《宣明论》卷七引《信香十方》。
主治：周身中外阴阳不调，气血壅滞，变生百病，乃至虚羸困倦，酒食内伤，心腹满塞急痛，或酒积，食积，癥瘕积聚，痃癖坚积，中满膈气，食臭酸醋，呕吐翻胃；或膈瘅消中，善食而瘦，或消渴多饮而数小便；或肠风下血，痔瘘痒痛；或胃痈疢，或遍身痈疽恶疮，或疮毒已入于里，腹满呕吐，或成泻痢，或出恶疮息肉，或下痢腹痛；或一切风气，肢体疼痛；及中风偏枯，或痰逆生风痰涎嗽；兼产后腹疼及小儿疳疾，诸风潮搐。
加减：None
功效：行荣卫，调饮食。
用法用量：先服小丸1丸，净器盛水送下。病在上，食后服；病在下，食前服；病在中，不拘时候。
制备方法：上为细末，熔蜡入蜜半钱为丸，如绿豆至小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，据剂型，当作“青金丸”。
