序号：32992
名称：辰砂五苓散
组成：辰砂（研）12两，白术（去芦）12两，木猪苓（去黑皮）12两，泽泻（洗，锉）12两，赤茯苓（去皮）12两，肉桂（去粗皮）8两。
出处：《局方》卷二（宝庆新增方）。
主治：伤寒表里未解，头痛发热，心胸郁闷，唇口干焦，神思昏沉，狂言谵语，如见鬼神，及瘴疟烦闷未省者；中暑发渴，小便赤涩，五心烦热，焦躁多哭，咬牙上窜，欲为惊状。小儿邪热在心之夜啼证。
加减：None
功效：清导小便。
用法用量：每服2钱，沸汤点下，不拘时候。如中暑发渴，小便赤涩，用新汲水调下；小儿五心烦热，焦躁多哭，咬牙上窜，欲为惊状，每服半钱，温熟水调下。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：苓砂散（《卫生总微》卷七）。本方方名，《永类钤方》引作“朱砂五苓散”。
