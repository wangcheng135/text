序号：41177
名称：金丝膏
组成：白胶香7斤半，大黄4钱，黄芩4钱，黄丹5钱，全蝎3钱，半夏3钱，虎骨3钱，防风3钱，当归须6钱，独活5钱，穿山甲5钱，乳香2两（别末），川芎4钱，木香5钱，明矾5钱，白蔹3钱，姜黄3钱，木鳖3钱，龙骨3钱，白及5钱，血竭5钱（别末），没药2两（别末），白芷4钱，黄柏5钱，五灵脂5钱，芍药3钱，蓖麻子3钱，山栀3钱，黄连6钱，羌活5钱，香油2斤8两。
出处：《医方类聚》卷一九三引《修月鲁般经》。
主治：诸般寒湿疼痛，及上高上坠，挫其骨节，及挫气疼痛，跌扑损伤。
加减：None
功效：None
用法用量：贴患处，贴时须用热手熨之。
制备方法：先将诸品药放于瓷器内浸，春5，夏3，秋7，冬10日，然后用柴火煎，候诸药焦黑为度，然后将白胶香煎至溶成清汁，滤过，次入前药油，熬至三五沸，入乳香、没药、血竭等，再熬一二沸，滴膏药于水中，冬天用嫩些，夏天用老些，拔成金丝为度。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
