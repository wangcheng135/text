序号：83958
名称：云母水
组成：上白云母20斤。
出处：《千金》卷二十七。
主治：劳气风疹，腹中寒癖，龋齿。
加减：None
功效：长生延年。
用法用量：先取水1合，纳药1合，搅和尽服之，1日3次，水寒温尽自在。
制备方法：薄擘，以露水8斗作汤，分半洮洗云母，如此再过，又取2斗作汤，纳芒消10斤，以云母木器中渍之，20日出，绢袋盛，悬屋上，勿使见风日，令燥，以水渍，鹿皮为囊，揉挺之，从旦至中，乃以细绢下筛，滓复揉挺，令得好粉5斗，余者弃之，取粉1斗，纳崖蜜2斤，搅令如粥，纳生竹桶中薄削之，漆固口，埋北垣南岸下，入地6尺，覆土，春夏40日，秋冬30日出之，当如泽为成，若洞洞不消者，更埋30日出之。
临床应用：None
各家论述：None
用药禁忌：None
附注：服十日。小便当变黄，此先疗劳气风疹也，二十日腹中寒癖消，三十日龋齿除，更新生，四十日不畏风寒，五十日诸病皆愈，颜色日少，长生神仙。
