序号：76260
名称：紫蟾锭
组成：山慈菇2钱（去毛皮，焙），川文蛤2钱（去蛙，末，炒），千金霜1钱（去油净），红芽大戟1钱5分（去芦根，洗净，焙，惟须杭州紫大为佳，北方绵大戟性裂不堪用），原麝香1钱（拣净皮毛），明朱砂2钱（漂净），雄黄1钱（拣鲜红者），寒水石3钱（煅），铜绿1钱，胆矾1钱，明乳香1钱（去油净），没药1钱（去油净），蜈蚣2钱（去头足，炒），全蝎1钱（酒炒），川山甲1钱（炙），僵蚕1钱（洗去丝，炒），蟾酥2钱（酒化），血竭1钱，梅花冰片5分，枯矾1钱6分，藤黄4钱（酒化），轻粉5分，红砒3钱，皂角刺1钱（炒）。
出处：《徐评外科正宗·附录》。
主治：阴阳二毒。
加减：None
功效：None
用法用量：None
制备方法：上药各为末，每味若干称准，净末合和一处，再研极细，先用蜗牛21个，微捣去壳，再用蟾酥、藤黄和研稠粘，方入各药，共捣极匀，做成小锭，放石灰坛中收燥，另以瓷瓶装盛听用。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
