序号：79861
名称：干蟾丸
组成：虾蟆、食蛆（或虾蟆、食蛆俱烧存性），广陈皮1斤（炒），甘草（炙）4两，蓬术（炒）6两，厚朴（米泔浸，炒）8两，枳实（麸炒）8两，连翘6两，香附（米泔浸，炒）1斤，山楂6两，神曲（炒）6两，菔子（炒）8两，龙胆草6两，青皮子（炒）8两，川黄连（炒）8两，白术（土炒）8两，槟榔8两。
出处：《医宗说约》卷五。
主治：五疳五痢，泻蛔虫，脏腑虚弱，身体羸瘦，发竖焦黄，小便浊色，肚腹膨胀。
加减：虚者，加米仁、山药；虚甚，加人参；有虫，加川楝子、使君子肉、鹤虱。
功效：消积化食，健脾和胃，长肌肉，驱蛔虫。
用法用量：上为极细末，炼蜜为丸，如龙眼大，空心清米汤化下。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：方中虾蟆、食蛆用量原缺。
