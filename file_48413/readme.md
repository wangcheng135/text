序号：48428
名称：神效种子鱼肚丸
组成：鱼肚（蛤粉炒至无声，去蛤粉，加酥少许，炒过）1斤，菟丝子（酒水煮成饼，晒）2两，真北沙苑（隔纸炒）3两，真白莲须3两，归身（酒洗）3两。
出处：《惠直堂方》卷一。
主治：None
加减：None
功效：种子。
用法用量：每服3钱，清晨开水送下，男女皆服。20以上，服至4两受孕；30者，服至半斤；40者，服至1斤；50者，服至1料；60者，多服亦能生子。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌食鱼、羊、火酒。服药时以保精为上，二十外者，五日一御女；三十者，十日；四十者，一月；五十者，一季，愈久愈妙。
附注：鱼肚丸（《蕙怡堂方》卷一）。
