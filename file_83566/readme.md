序号：83584
名称：乌须方
组成：五倍子1两，硇砂春冬8分，秋夏3分，红铜末1钱，白矾1钱，没石子1钱。
出处：《鲁府禁方》卷二。
主治：None
加减：None
功效：染乌须发。
用法用量：先将须发用肥皂洗净，以布拭干；将药入于白茶盏内，又用浓茶、食盐些须，调前药放于锅内，煮3-4沸，看其不稠不稀，取起，趁热以眉掠挑药，染涂白处，以油纸包裹1-2小时，解去油纸，候干洗净，须发即黑。
制备方法：上药各为极细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：制五倍子法：用五倍子，不拘多少，捣碎，如黄豆大，用糠筛筛去细者，入无油净锅内，不住手炒，以黑色为度，不要黄色，不要焦枯了；用青布一方，水湿趁五倍热包裹在内，于地上板盖踏成饼，候冷取出，听用。制铜末法：用红铜末，将好醋铜末锅内炒干，如此七次，方好入醋，看末多寡酌量。
