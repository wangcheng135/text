序号：2698
名称：天门冬粥
组成：天门冬15-20g，粳米50-100g，冰糖少许。
出处：《药粥疗法》引《饮食辨录》。
主治：肾阴不足，阴虚内热，津少口干，及肺阴不足，肺虚有热，干咳少痰或无痰，痰中带血，以及午后低热，夜间盗汗的肺结核患者。
加减：None
功效：滋阴润肺，生津止咳。
用法用量：先用天门冬取浓汁，去滓，入粳米煮粥，沸后加入冰糖适量，再煮成粥。以3-5日为1疗程，间隔3日再服。
制备方法：None
临床应用：None
各家论述：《药粥疗法》：天门冬味甘微苦，无毒，肥厚多脂，入肺、肾经，所以能养肺肾之阴，善治肺肾虚热。在煮粥时，适当加些冰糖，不仅可纠正微苦之味，还能增强它的滋养作用。
用药禁忌：虚寒腹泻及外感风寒咳嗽者，均不宜用。
附注：None
