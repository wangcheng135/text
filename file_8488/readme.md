序号：8488
名称：半夏丸
组成：半夏1两6铢，赤石脂1两6铢，蜀椒1两，干姜1两，吴茱萸1两，当归1两，桂心1两，丹参1两，白蔹1两，防风1两，雚芦半两。
出处：《普济方》卷三二七。
主治：因与夫卧起，月经不去或卧湿冷地，及以冷水洗浴，或疮痍未愈，便合阴阳，及起早作劳，衣单蓆薄，寒从下起，至妇人怀中十二疾：经水不时、经来如清水、经水不通、不周时、生不乳、绝无子、阴阳减少、腹苦疼如刺、阴中寒、子门相引痛、经来冻如葵汁状、腰急痛。
加减：None
功效：None
用法用量：每服10丸，空心酒送下。不知，稍加，以知为度。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
