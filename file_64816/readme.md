序号：64832
名称：续命汤
组成：麻黄（去节）3两，茯神3两，生姜3两，附子（炮）两半，防己两半，甘草（炙）两半，芎3两，细辛3两，白鲜皮3两，杏仁（去皮尖双仁，碎）3两，人参3两，羌活3两，桂心3两。
出处：《外台》卷十四引《崔氏方》。
主治：卒中风欲死，身体缓急，口目不正，舌僵不能语，奄奄惚惚，神情闷乱。
加减：None
功效：None
用法用量：以水8升，煮取2升8合，去滓。分3服，服别相去8-9里许，覆取汗。可服3剂，间5日一进，慎如药法。若老弱虚羸，非间10日以上，不可频服。
制备方法：上切。
临床应用：None
各家论述：None
用药禁忌：忌猪肉、冷水、海藻、菘菜、生葱、生菜、大酢。
附注：None
