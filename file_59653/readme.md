序号：59668
名称：调元散
组成：干山药（去黑皮）5钱，人参（去芦）2钱半，白茯苓（去皮）2钱半，茯神（去皮木根）2钱半，白术2钱半，白芍药2钱半，熟干地黄（酒洗）2钱半，当归（酒洗）2钱半，黄耆（蜜水涂炙）2钱半，川芎3钱，甘草（炙）3钱，石菖蒲2钱。
出处：《活幼心书》卷下。
主治：禀受元气不足，颅囟开解，肌肉消瘦，腹大如肿，致语迟行迟，手足如痫，神色昏慢，齿生迟者。
加减：None
功效：None
用法用量：上（口父）咀。每服2钱，以水1盏，生姜2片，大枣1个，煎7分，温服，不拘时候。如婴孩幼嫩，与乳母同服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：调元地黄汤（《幼科金针》卷上）。
