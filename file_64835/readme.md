序号：64851
名称：续骨丹
组成：天麻半两（明净大者，酒浸1夕），白附子半两（炮），牛膝半两（洗，锉，焙，酒浸1宿，再焙），木鳖子（去壳，研）半两，乌头（炮，去皮脐）1分，川羌活（洗去土）半两，地龙（去土）1分，滴乳香2钱（乳钵坐水盆中研细），真没药（研）2钱，朱砂（水飞）1钱。
出处：《本事》卷四。
主治：两脚软弱，虚羸无力及小儿不能行。
加减：None
功效：None
用法用量：每服1丸，食前薄荷汤磨下。
制备方法：上以生南星末1两，无灰酒煮糊为丸，如鸡头子大，朱砂为衣。
临床应用：None
各家论述：《本事方释义》：天麻气味辛平，入足阳明、厥阴；白附子气味辛甘大温，入足阳明；牛膝气味酸咸平，入足厥阴；木鳖子气味甘温微苦，入足太阴；乌头气味苦辛大热，入足太阳少阴；羌活气味辛甘平，入足太阳；地龙气味咸寒，入足阳明、厥阴；乳香气味辛微温，入足少阴；没药气味苦平，入足阳明；朱砂气味苦温，入手少阴；南星，气味苦辛温，入手、足太阴。以酒丸薄荷汤送下，取其引药入络也。此治虚羸无力，两脚软弱及小儿不能行走者，皆用辛通热药，少佐以下行之品，不欲其停留在上，而任行于筋骨也。
用药禁忌：None
附注：None
