序号：80492
名称：鳖甲丸
组成：鳖甲（醋炙，去裙襕）2两，干姜（炮）1两半，大黄（锉，炒）1两半，硇砂（去砂石）1两半，附子（炮裂，去皮脐）1两，槟榔（锉）1两，桂（去粗皮）1两，干漆（炒出烟）1两，京三棱（煨）1两，木香1两，诃黎勒皮1两，水银（与诸药末同研）1两，墨（烧）半两。
出处：《圣济总录》卷七十三。
主治：痃癖气块。
加减：None
功效：None
用法用量：每服7丸，加至10丸，丈夫温酒送下，妇人醋汤送下，不嚼破，每日3次，不拘时候。取下血块，如鸡肝色是效。
制备方法：上为末，用曲末3两，浓醋2升，同煎成膏，和上药为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
