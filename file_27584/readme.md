序号：27592
名称：补阴汤
组成：当归1钱，白芍（酒炒）1钱，生地黄1钱，熟地黄1钱，陈皮1钱，茴香（盐、酒炒）1钱，故纸（酒炒）1钱，牛膝（去芦，酒洗）1钱，杜仲（去粗皮，酒炒）1钱，茯苓（去皮）1钱，人参5分，黄柏（去粗皮，酒炒）7分，知母（酒炒）7分，甘草（炙）3分。
出处：《回春》卷五。
主治：肾虚腰痛。
加减：痛甚大者，加乳香、砂仁、沉香，去芍药、生地、陈皮。
功效：None
用法用量：每服50丸，米汤送下；酒亦可。
制备方法：上锉1剂。加大枣2枚，水煎服，不拘时候。如常服合丸药，俱为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
