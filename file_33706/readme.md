序号：33715
名称：灵砂丹
组成：皂角（不蛀，肥实者，去皮弦子）2斤（用河水3升，、生姜自然汁半升，揉皂角，取浓汁，滤去滓，于银石器内慢火熬成膏），天南星（生用）4两，半夏（汤洗去滑）4两，白附子（生用）4两，白矾（枯）4两，猪牙皂角（肥实者，去皮弦，涂酥炙赤色）2两，朱砂1两半（研如粉）。
出处：《杨氏家藏方》卷八。
主治：痰饮头痛。
加减：如积年经久痼冷痰疾，加生附子4两，去皮脐用。
功效：None
用法用量：每服30丸，食后生姜汤送下。
制备方法：上为细末，入朱砂同研匀，将前皂角膏子搜和为丸，如梧桐子大，别用朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
