序号：6057
名称：加味保和丸
组成：白术1两5钱（土炒），神曲1两5钱，萝卜子1两5钱（炒），广皮1两5钱，连翘1两5钱，半夏1两5钱（炙），香附1两5钱（炙），茯苓1两5钱，黄芩1两5钱，黄连5钱，山楂1两（炒），厚朴1两（炙），枳实1两（炒），麦芽1两（炒）。
出处：《慈禧光绪医方选议》。
主治：食积、酒积。痰饮。除胸膈痞满，嗳气吞酸，腹痛便溏。
加减：None
功效：和血补血，消补兼施，消多于补。
用法用量：每服3钱，白开水送下。
制备方法：上为细末，水法为丸，如绿豆大。
临床应用：None
各家论述：本方为朱丹溪保和丸加枳实、香附、厚朴、黄芩而成。或谓保和丸原方麦芽伤肾，萝卜子伤肺胃之气，故主张以枳实、香附易之，不伤先后天之真气。
用药禁忌：None
附注：None
