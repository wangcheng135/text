序号：47534
名称：神化丹
组成：天麻（去苗）1两，附子（炮裂，去皮脐）1两，牛膝（酒浸1宿）1两，防风（去芦头）1两，地龙（去土，炒）1两，蝎梢（去毒，炒）1两，羌活（去芦头）1两，白僵蚕（炒，去丝嘴）1两，独活（去芦头）1两，肉桂（去粗皮）1两，当归（洗，焙）1两，干姜（炮）1两，虎胫骨（醋浸，炙黄）1两，半夏（生用）1两，败龟（醋浸，炙黄）1两，乌蛇肉（酒浸，去皮骨，取肉，焙干）1两，天南星（生用）1两，白附子（生用）1两，黑参1两。
出处：《普济方》卷一一六。
主治：一切风，及瘫痪偏枯，疠风，卒中喑哑，神昏志乱，手足不遂。
加减：None
功效：None
用法用量：每服1丸，温酒化下，不拘时候。
制备方法：上为细末，炼蜜为丸，每1两作15丸，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
