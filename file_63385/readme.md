序号：63401
名称：清神益气汤
组成：茯苓2分，升麻2分，泽泻3分，苍术3分，防风3分，生姜5分，青皮1分，橘皮2分，生甘草2分，白芍药2分，白术2分，人参5分，黄柏1分，麦门冬2分，五味子3分。
出处：《脾胃论》卷下。
主治：素有脾胃虚损病，目疾时作，身面目睛俱黄，小便或黄或白，大便不调，饮食减少，气短上气，怠惰嗜卧，四肢不收。
加减：None
功效：None
用法用量：以水2盏，煎至1盏，去滓，空心稍热服。
制备方法：上锉如麻豆大，都作1服。
临床应用：脾胃病：戊申6月初，枢判白文举年62，素有脾胃虚损病，目疾时作，身面目睛俱黄，小便或黄或白，大便不调，饮食减少，气短上气，怠情嗜卧，四肢不收；至6月中目疾复作，医以泻肝散下数行而前疾增剧。此当于脾胃肺之本脏，泻外经中之湿热，制清神益气汤主之而愈。
各家论述：茯苓、升麻、泽泻、苍术、防风，此药能走经，除湿热而不守，故不泻本脏。补肺与脾胃本气之虚弱，青皮、橘皮、生甘草、白芍药、白术、人参，此药皆能守本而不走经，不走经者不滋经络中邪，守者能补脏之元气。黄柏、麦门冬、人参、五味子，此药去时令浮热湿蒸。火炽之极，金伏之际，而寒水绝体，于此时也，故急救之以生脉散除其湿热，以恶其太甚。肺欲收，心苦缓，皆酸以收之；心火盛则甘以泻之，故人参之甘，佐以五味子之酸，孙思邈云：夏月常服五味子以补五脏气是也。麦门冬之微苦寒，能滋水之源千金之位，而清肃肺气；又能除火刑金之嗽，而敛其痰邪；复微加黄柏之苦寒，以为守位滋水之流，以镇坠其浮气，而除两足之痿弱也。
用药禁忌：None
附注：益气汤（《普济方》卷二十五）。
