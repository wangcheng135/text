序号：61931
名称：救中汤
组成：蜀椒（炒出汗）3钱，淡干姜4钱，厚朴3钱，槟榔2钱，广皮2钱。
出处：《湿病条辨》卷二。
主治：卒中寒湿，内挟秽浊伏阴与湿，眩冒欲绝，腹中绞痛，脉沉紧而迟，甚则伏，欲吐不得吐，欲利不得利，甚则转筋，四肢欲厥，俗名发痧，又名干霍乱。
加减：兼转筋者，加桂枝3钱，防已5钱，薏仁3钱；厥者，加附子2钱。
功效：急驱浊阴，救中焦之真阳。
用法用量：加水5杯，煮取2杯，分2次服。
制备方法：None
临床应用：None
各家论述：以大建中之蜀椒．急驱阴浊下行，干姜温中；去人参、胶、饴者，畏其满而守也；加厚朴以泻湿中浊气，槟榔以散结气，直达下焦，广皮通行十二经之气。
用药禁忌：None
附注：None
