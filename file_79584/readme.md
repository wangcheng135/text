序号：79600
名称：蝉花散
组成：蝉蜕（去土）半两，苍术2两半（米泔浸1宿，切，焙），荆芥穗1两半，甘草（炙）1两半，木贼（去节）1两半，密蒙花1两，甘菊花1两，旋覆花1两，黄连（去须）1两，石决明（火煅）1两，草决明1两，黄芩1两，谷精草1两，仙灵脾1两，青葙子1两，薄荷叶（去土）1两，羌活（去芦头）1两，川芎1两，防风（去芦头）1两，白蒺藜（炒去刺）1两，细辛（去叶土）半两，羯羊肝1具（切，焙干）。
出处：《杨氏家藏方》卷十一。
主治：肝经蕴积风毒，上攻眼目，肿痛昏暗，或生翳膜，视物不明。
加减：None
功效：None
用法用量：每服2钱，食后用川椒汤调下，茶清亦得。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
