序号：23260
名称：人参煮散
组成：人参（去芦头）4两，黄耆（蜜炙）3两，苍术（米泔浸1宿）3两，当归（洗，焙）2两，麻黄（去根节）2两，柴胡（去苗）2两，黄芩2两，川乌头（炮，去皮脐）1两半，羌活（去芦头）1两半，肉桂（去粗皮）1两半，高良姜1两，益智仁1两，干姜（炮）1两，陈橘皮（去白）1两，青橘皮（去白）1两，香白芷1两，枳壳（去瓤，麸炒）1两，白芍药1两，牛膝（酒浸1宿，焙干）1两，独活1两，青蒿子1两，鳖甲（醋炙黄）1两，川芎1两，荆芥穗1两，炙甘草1两。
出处：《杨氏家藏方》卷十五。
主治：妇人血风劳倦，骨蒸盗汗，心烦气劣，浑身疼痛。
加减：None
功效：None
用法用量：上(口父)咀。每服3钱，水1盏半，加生姜3片，乌梅半枚，同煎至8分，去滓温服，不拘时候。
