序号：3397
名称：太乙无名丸
组成：三棱2两，莪术2两，归尾2两，赤芍2两，元胡2两，槟榔2两，五灵脂2两，干漆2两，广木香2两，干姜2两，神曲4两，莱菔子（炒）4两，芒消4两，吴萸1两，黑丑（炒）半斤，山楂2两，牛膝2两，红曲半斤，枳实3两，枳壳（炒）3两，紫硇砂5钱，大黄5斤（酒洗），大腹皮3两（熬水）。
出处：《全国中药成药处方集》(济南方)。
主治：胃肠积聚，胁腹胀满，停食停水，大便不通。
加减：None
功效：None
用法用量：每服5分至1钱。小儿酌减。
制备方法：醋2斤，和水为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：忌辛辣、油腻等物。孕妇及体虚胃弱者忌服。
附注：None
