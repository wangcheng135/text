序号：7813
名称：圣化丹
组成：防风1两，荆芥1两，羌活1两，独活1两，胡麻（炒）2两半，金毛狗（去毛）2两半，苦参（去皮）2两半，牙皂（去皮核）2两半，当归2两半，蝉退1两1钱，姜蚕（炒）1两1钱，全蝎（去头足尾）1两1钱，白芷1两1钱，何首乌（去皮毛，炒）1两1钱，苍耳草（烧）5钱。
出处：《外科百效》卷四。
主治：疠疯症，眉毛脱落，鼻梁倒塌，遍身生癞者。
加减：None
功效：None
用法用量：每服40-50丸。如病人身上骨肿，眉上痒不止，或是风气拔睛，手足拘挛，先服此药1料，将青茶送下，日进4服。或妇人四肢麻痹，手若刺痛，腿膝生疮，先服夺命1料，后服此圣化丹。服药后10日，瓦锋面上放血，次于膊上放血，后于腿脚上放血，而遇天气清明，5-6日放1次。量病轻重，不可乱放血，若妇人多破血无妨。
制备方法：上为细末，大枫子二两去壳，捣烂如泥，同药和匀，将陈米饭打糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
