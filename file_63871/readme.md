序号：63887
名称：清脾汤
组成：青皮（去白）、厚朴（姜制，炒）、白术、草果仁、柴胡（去芦）、茯苓（去皮）、半夏（汤泡7次）、黄芩、甘草（炙）各等分。
出处：《济生》卷一。
主治：瘅疟脉来弦数，但热不寒，或热多寒少，膈满能食，口苦舌干，心烦渴水，小便黄赤，大便不利。妊娠疟疾。
加减：None
功效：None
用法用量：上（口父）咀，每服4钱，以水1盏半，加生姜5片，煎至7分，去滓温服，不拘时候。
制备方法：None
临床应用：None
各家论述：1.《医方考》：方曰清脾者，非清凉之谓，乃攻去其邪而脾部为之一清也。故青皮、厚朴清去脾部之痰，半夏、茯苓清去脾中之湿，柴胡、黄芩清去脾中之热，白术、甘草清去脾脏之虚，而草果仁又所以清膏梁之痰也。_x000D_
2.《医方集解》：脾虚恶寒，胃虚恶热。寒热间作，脾亦有之，不独少阳也。虽十二经皆能为疟，而脾胃受伤者实多。故仲景小柴胡汤人参、甘草、半夏、姜、枣，皆脾胃药，其治少阳，独柴胡一味而已。严氏宗之，故以小柴胡加减而立清脾饮，是明从脾胃论治矣。
用药禁忌：None
