序号：34183
名称：牡丹煎丸
组成：延胡索半两，砂仁半两，赤芍药1两，牡丹皮1两，山茱萸半两，干姜半两（炮）、龙骨（细研）3两，熟地黄（酒浸）3两，槟榔3两，羌活3两，五味子1两，人参1两，白芷1两，当归（酒浸）1两，干山药1两，肉桂（去皮）1两，白茯苓1两，白术1两，藁本1两，附子（炮去皮脐）1两，木香1两，牛膝（酒浸）1两，荜拨1两（水泡），石斛3两（酒浸）。
出处：《葆光道人眼科龙木集》。
主治：目病积年不愈，目数赤点者。
加减：None
功效：None
用法用量：每服20丸。空心温酒或醋汤送下，日3次。
制备方法：上为细末。炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：孕妇不可服。
附注：None
