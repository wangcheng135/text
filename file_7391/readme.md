序号：7391
名称：圣饼子
组成：丹砂、铁粉、牛黄、甜消、麝香、龙脑、蓬砂（7味并研），天麻、白芷、犀角（镑）、白僵蚕（炒）、芎藭、雌黄（别用水银、石脑油各1钱同研如泥），天雄1钱，乌头1钱，附子1钱，天南星1钱（四味同锉），狐肝1具（以甘草水洗3遍细切，与天雄、附子、乌头、天南星四味锉了拌匀，入罐子内黄泥固济，勿令透气，候干，以炭火5斤，烧存性，放冷取出细研）。
出处：《圣济总录》卷五。
主治：卒中风，涎潮昏塞，口眼斜，手足麻痹，言语謇涩；大治风痫。
加减：None
功效：None
用法用量：每服1饼，薄荷酒化下。小儿惊痫，1饼分作5服，薄荷汤化下。
制备方法：上为末，炼蜜为剂，分作六十饼。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
