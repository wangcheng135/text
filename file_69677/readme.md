序号：69693
名称：疏风芎归散
组成：当归3两，川芎1两半，人参1两半，紫苏1两半，葛根1两半（砂糖炒黑）。
出处：《医略六书》卷三十。
主治：产后伤风，脉浮涩者。
加减：None
功效：None
用法用量：每服3钱，加生姜2片，葱白3枚，水煎，去滓温服。
制备方法：上为散。
临床应用：None
各家论述：产后血气两虚，营卫不能布护，寒风得以伤之，故发热无汗而恶风寒焉。人参扶元气以内托；当归养营血以荣经；川芎行血气升生阳；紫苏理血气散风寒；葛根禀性轻扬，本解肌升阳药，自与产科无涉，砂糖炒黑，假之引入血分以解冲脉之邪也，为散，姜、葱汤煎，务使血气内充，则营卫布护而邪无容身之地，何发热无汗，恶风寒，为产后伤寒风之足患哉。
用药禁忌：None
附注：None
