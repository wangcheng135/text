序号：66420
名称：寸金丸
组成：雄黄半两，京三棱（炮、锉）半两，石三棱半两，鸡爪三棱半两，蓬莪术（炮）半两，桂（去粗皮）半两，木香半两，沉香（锉）半两，干漆（炒烟出）半两，半夏（汤洗7遍，焙）半两，丁香半两，肉豆蔻（去壳）半两，槟榔（锉）4枚，硇砂（研）1两，巴豆（去皮，出油尽，研）30枚，茴香子2两（炒），金铃子2两，大麦糵（炒）4两。
出处：《圣济总录》卷五十四。
主治：阴阳气不升降，心腹臌胀，胁肋刺痛，倦怠嗜卧，全不思食。
加减：None
功效：None
用法用量：每服先嚼枣1枚，下2丸干咽，不得嚼破，食后或临卧服。虚弱人有所伤，皆可服。
制备方法：上为末，同和匀，以糊饼剂作糊为丸，如梧桐子大，风干，用油炸令紫色为度，入瓷盒收贮，以研麝香1分熏之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
