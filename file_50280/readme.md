序号：50295
名称：茴香子丸
组成：茴香子（舶上者妙）2两，附子（炮裂，去皮脐）2两，巴戟天（去心）2两，蜀椒（去目及闭口，炒出汗）2两，牛膝（酒浸1宿，焙）2两，肉苁蓉（酒浸令软，细切，焙）2两，青盐（研）2两。
出处：《圣济总录》卷八十六。
主治：脾劳。肌肤瘦瘁，面色黄黑，四肢无力，脚膝疼痛，大便不调；或风虚上攻，头眩目暗，肢体沉重，昏愦嗜卧。
加减：None
功效：None
用法用量：每服20-30丸，空心盐汤或酒送下。
制备方法：上药除青盐外，为末，研拌令匀，烂煮羊肾或猪肾2-3对，去筋膜细切，烂研和药，入杵臼捣令匀熟为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
