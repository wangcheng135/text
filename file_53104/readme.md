序号：53119
名称：大补经汤
组成：当归（酒洗）6分，白芍6分，香附6分，川芎5分，熟地黄5分，白术（去芦）4分，白茯苓4分，黄耆4分，陈皮4分，玄胡索4分，人参3分，砂仁3分，阿胶（炒）3分，沉香（另研）3分，小茴（酒炒）3分，吴茱萸（炒）3分，肉桂3分，粉甘（炙）3分。
出处：《回春》卷六。
主治：妇人气血虚弱，血海寒冷，经水不调，或时心腹疼痛，或下白带如鱼脑髓，或似米泔色，错乱不分，信期每月淋沥不止，面色萎黄，四肢无力，头目眩晕，肌体羸瘦。
加减：None
功效：None
用法用量：加生姜、大枣，水煎服。
制备方法：上锉1剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
