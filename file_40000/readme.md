序号：40010
名称：育魂丹
组成：胆星6钱，半夏（制）6钱，茯神6钱，黄连（炒）6钱，远志（水泡）6钱，白术（炒）6钱，枣仁（炒）6钱，柏子仁（炒）6钱，干山药1两，竹茹5钱，白附（煨）5钱，天麻（酒洗）5钱，陈皮3钱，全蝎3钱2分，川芎5钱，犀角（镑）3钱5分，枳实（炒）1钱，辰砂2钱2分，牛黄2钱2分，羚羊角3钱5分，白矾（生用）3钱，麝香1钱，飞金24贴。
出处：《古今医鉴》卷七。
主治：一切惊痫、癫邪等证。怔忡惊怕恐惧之疾。
加减：None
功效：None
用法用量：空心淡姜汤送下，或用薄荷汤调下。
制备方法：上为细末，竹沥打甘草膏为丸，如芡实大。
临床应用：None
各家论述：None
用药禁忌：None
