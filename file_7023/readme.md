序号：7023
名称：四物补肝散
组成：熟地黄（焙干）2两，香附子8钱（酒制），川芎8钱，白芍8钱（酒洗，炒），当归身8钱（酒洗，炒），夏桔草8钱，甘草4分。
出处：《审视瑶函》。
主治：妇人产后，午后至夜昏花不明；雀目初起，头旋，常见五色不定，目中困倦，时暗时明。
加减：None
功效：None
用法用量：每服2-3钱，食后滚白汤送下。
制备方法：上为细末。
临床应用：None
各家论述：以熟地黄补血、当归养血为君；夏枯草入厥阴，补养血脉为臣；甘草益无气补脾胃，白芍补脾和血为佐；川芎助清阳之气上升，香附理气血，故为使耳。
用药禁忌：None
附注：四物补血汤（《眼科阐微》卷三）、四物补肝汤（《金鉴》卷七十八）。
