序号：63131
名称：大戟丸
组成：大戟1分（锉碎，微炒），甜葶苈1分（隔纸炒令紫色），芫花1分（醋拌，炒令干）甘遂1分（煨令微黄），泽漆1分，桑根白皮1分（锉），赤小豆1分（炒熟），巴豆1分（去皮心，研，纸裹压去油），泽泻1分。
出处：方出《圣惠》卷五十四，名见《普济方》卷一九一。
主治：十种水气、遍身肿满，喘急烦闷，心腹壅滞，大小便不利。
加减：None
功效：None
用法用量：每服3丸，空心粥饮送下。
制备方法：上为末，入巴豆研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌盐120日，缘盐能化水故也。外忌鱼酢、面食、一切毒物及生冷等物。
附注：None
