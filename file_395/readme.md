序号：395
名称：五香膏
组成：丁香1分（末），木香1分（末），白檀香1分（末），熏陆香1分（末），麝香1分（末），黄耆半两，白芷半两，细辛半两，防风半两（去芦头），芎半两，当归半两，甘草1两，桑根白皮1两，槐枝（锉）3合，乱发1两（烧灰），垂柳枝（锉）3合，黄丹10两，清麻油1斤4两。
出处：《圣惠》卷六十三。
主治：一切痈疽发背，及恶毒疮肿。
加减：None
功效：止疼痛，生肌。
用法用量：外贴，每日换2次。
制备方法：上药除五香末外，并锉细，安净铛内，以油浸1宿，以慢火煎令槐、柳枝色黄黑为度，以绵滤去滓，澄清，却于铛内慢火熬药油，相次入黄丹，用柳木篦不住手搅，候黄丹色黑，滴于水内，看硬软得所，入五香末搅令匀，倾于不律器内盛。每日用时，于火畔煨，以纸上涂贴。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
