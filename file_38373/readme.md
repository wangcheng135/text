序号：38383
名称：河车种玉丸
组成：紫河车1具（只要母气壮盛，厚大新鲜者，但去胞内瘀血，不必挑去鲜红血脉，以米泔水洗净，用布绞干，石臼内生杵如糊，用山药末4-5两收干，捻为薄饼8-9个，于砂锅内焙干，以香如肉脯为妙），大熟地（酒洗，烘干）8两，枸杞（烘干）5两，白茯苓（人乳拌，晒3次），归身（酒洗）4两，人参4两，菟丝（制）4两，阿胶（炒珠）4两，丹皮（酒洗）2两，白薇（酒洗）2两，沉香1两，桂心3两，山茱萸3两，香附米（用酒半碗，醋半碗，水半碗，浸3日，晒干略烘）3两，大川芎（酒浸，切片，晒干）2两。
出处：《景岳全书》卷六十一。
主治：None
加减：如带浊多者，加赤白石脂各2两，须以清米泔飞过用。
功效：令人孕育。
用法用量：每服百余丸，空心或酒、或白汤、盐汤任下。
制备方法：炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：服药后忌生萝卜、生藕、葱、蒜、绿豆粉之类。
附注：None
