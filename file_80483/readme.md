序号：80499
名称：鳖甲丸
组成：鳖甲1两（涂醋，炙令黄，去裙襕），紫菀1两（洗，去苗土），熟干地黄2两半，桂心3分，芎1两，羌活2分，防风1两（去芦头），牛膝1两（去苗），当归1两（锉），秦艽1两（去芦头），黄耆3分（锉），赤芍药3分，人参1两（去芦头），白术3分，桃仁1两（汤浸，去皮尖双仁，麸炒微黄），琥珀1两，鬼箭羽3分，虻虫3分（去翅足，微炒），水蛭2分（炒令黄），麝香1分（细研入）。
出处：《圣惠》卷七十。
主治：妇人血风劳气。四肢羸瘦疼痛，经络不利，饮食无味，渐加虚困。
加减：None
功效：None
用法用量：每服30丸，食前温酒送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
