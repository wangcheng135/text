序号：34396
名称：万病丸
组成：熟干地黄4两，当归4两。
出处：《产乳备要》。
主治：妇人久虚，血气衰少，怠惰嗜卧，饮食不进，肌体瘦悴，精神不足，妇人劳虚血弱，肌肉枯燥，手足多烦，肢节痠疼，鬓发脱落，面少_x000D_
颜色，腹中拘急，痛引腰背，去血过多，崩伤内竭，胸中短气，昼夜不得眠，情思不乐，怔忡多汗；诸虚不足，腹胁疼痛，翕翕发热，_x000D_
及妇人经病，月事不调。
加减：None
功效：None
用法用量：每服50丸，空心、食前温粥饮送下。
制备方法：上为细末，面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：滋阴养血丸（《鸡峰》卷十六）、滋阴万病丸（原书冀致君注引《十便良方》）、内补丸（原书冀致君注引《郑汝明产经》）。
