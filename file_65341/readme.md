序号：65357
名称：女宝丹
组成：当归6两（酒洗），生地6两（酒蒸），白芍3两（酒炒），川芎3两（酒洗），白术6两（漂净，土炒），条芩4两（酒炒），陈皮2两（炒），阿胶（酒浸，溶蜜内）3两，香附（童便、盐、酒、醋4制）6两，砂仁（炒）2两。
出处：《医宗说约》卷四。
主治：None
加减：月事后期来者，去条芩，加姜灰1两，蕲艾2两；肥者，加制半夏3两，白茯苓4两；有白带者，再加白薇4两；气虚甚者，加人参3两，茯苓4两，山药4两；如腰痛，加山药3两、杜仲3两。
功效：调经种子，安胎保孕。
用法用量：每服5钱，空心白汤送下，安胎用白蜜丸，不用益母膏。
制备方法：上为末，另将益母草2斤半煎膏，和炼蜜及阿胶为丸，如梧桐子大。
临床应用：1.种子：一妇年四十岁，有十余年不受胎矣，月事前后不准而又无子，脉来微细兼数，予制女宝丹，服至百日而孕，后产一男。_x000D_
2.安胎：一女子三十余岁，有孕至五月间必堕，已三四次矣，予亦以女宝丹加减付之，即产一子。
