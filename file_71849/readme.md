序号：71865
名称：散花去癫汤
组成：柴胡5钱，白芍7钱，当归5钱，炒栀子3钱，甘草1钱，茯神3钱，菖蒲1钱，麦冬5钱，元参3钱，白芥子3钱。
出处：《石室秘录》卷三。
主治：妇人花癫，忽然癫痫，见男子则抱住不放，此乃思慕男子不可得，忽然病如暴风疾雨，罔识羞耻，见男子则以为情人，此肝木枯槁，内火燔甚，脉弦出寸口。
加减：None
功效：平肝散郁祛邪。
用法用量：水煎服。如不肯服，用人灌之，彼必骂詈不休，久之人倦欲卧，卧后醒来，自家羞耻，紧闭房门者3日，少少与之饮食，自愈。1剂后不必更与之药也。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
