序号：36841
名称：固真丹
组成：木香2钱，母丁香2钱，沉香2钱，麝香半钱，头红花3钱，川楝子3钱，细墨2钱，韭子3钱（酒浸1宿），菟丝子3钱（酒浸1宿，擂细成饼，晒干，为末），牛膝3钱（酒浸1宿），破故纸3钱（同前制），巴戟3钱（同前），肉苁蓉3钱（同前），莲肉3钱，青盐3钱，莲心3钱，茴香3钱，川山甲3钱（酥炙黄），益智3钱，地龙（去土）3钱，川木通3钱，朱砂3钱，胡芦巴3钱。
出处：《普济方》卷二二六。
主治：诸虚。
加减：None
功效：None
用法用量：每服30丸，空心温酒送下，服讫干物压之。
制备方法：上青盐、莲心以墨别研，共为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
