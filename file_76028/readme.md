序号：76044
名称：葶苈丸
组成：葶苈（隔纸炒）2两，木香3分，陈橘皮（汤浸，去白，焙）3分，枳壳（去瓤，麸炒）3分，楮根白皮（炙，锉）3分，干姜（炮）3分，槟榔（煨，锉）3分，防己3分，马兜铃（去皮，微炒）3分，朴消（别研）3分，蓬莪术（煨，锉）3分，甘遂（微煨）1两。
出处：《圣济总录》卷一五三。
主治：妇人水分，浮肿不退，经脉不利。
加减：None
功效：None
用法用量：每服20丸，加至30丸，空心、日午、夜卧时用桑根白皮煎汤送下，取利为度。如水利即减丸数。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中楮根白皮，《普济方》作“桑根白皮”。
