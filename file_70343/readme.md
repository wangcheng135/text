序号：70359
名称：犀地清络饮
组成：犀角汁4匙（冲），粉丹皮2钱，青连翘2钱半（带心），淡竹沥2瓢（和匀），鲜生地8钱，生赤芍1钱半，原桃仁9粒（去皮），生姜汁2滴（同冲）。
出处：《重订通俗伤寒论》。
主治：热陷包络神昏。
加减：None
功效：轻清透络，通瘀泄热。
用法用量：用鲜茅根1两，灯心5分，煎汤代水，鲜石菖蒲汁2匙冲。
制备方法：None
临床应用：None
各家论述：热陷包络神昏，非痰迷心窍，即瘀塞心孔，必用轻清灵通之品，始能开窍而透络。故以千金犀角地黄汤，凉通络瘀为君；臣以带心连翘透包络以清心，桃仁行心经以活血；但络瘀者必有粘涎，故又佐姜、沥、菖蒲三汁，辛润以涤痰涎，而石菖蒲更有开心孔之功；妙在使茅根交春透发，善能凉血以清热；灯心质轻味淡，更能清心以降火。此为轻清透络，通瘀泄热之良方。
用药禁忌：None
附注：None
