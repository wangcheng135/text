序号：19712
名称：壮原汤
组成：人参2钱，白术2钱，茯苓1钱，破故纸1钱，桂心5分，大附子5分，干姜5分，砂仁5分，陈皮7分。
出处：《赤水玄珠》卷五。
主治：下焦虚寒，中满肿胀，小水不利，上气喘急，阴囊两腿皆肿，或面有浮气。
加减：有痰，加半夏1钱；喉中痰声，加桑白皮1钱，咳嗽亦加；脚跌面肿，加薏苡仁2钱；中气不转运，不知饿，加厚朴、木香；气郁不舒，加沉香、乌药，临服磨入；气虚甚者，人参加作5钱，大附子加作1钱半；汗多者，再加桂枝5分，白芍药（酒妙）过8分；若夏月喘乏无力，或汗多者，加麦门冬1钱、五味子11粒；夜梦不安者，加远志1钱；两胁气硬，加白芥子8分；若面浮肿，胁下气硬，加白芥子、紫苏子5分；若身重不能转动，加苍术1钱，泽泻7分；湿盛，加桑白皮、赤小豆。
功效：None
用法用量：水煎，食远服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：壮元汤（《风劳臌膈》）。
