序号：2986
名称：天真玉髓丸
组成：白蒺藜（炒去刺）4两，草胡麻（去土，微炒）4两，苦参（鲜明者）4两，荆芥4两，当归身（酒洗）4两，防风（去芦）4两，海风藤（香者为上，如马鞭根，切片，花纹如槟榔尤妙）2两，枳壳（去瓤净）2两，白术2两，木通2两，乳香（去油）1两，没药（去油）1两，牛膝1两，川桂枝1两，重全蝎7个，大风子5两（同天麻5钱煮，去白衣膜，石臼内捣碎和匀），虎骨（酥炙）2两。
出处：《疡医大全》卷二十八引《家秘》。
主治：大麻风、紫云风。
加减：有湿痰在胃则用白术，如胃不痛，无痰则不用；疼痛则用乳香、没药、虎骨，不痛则去之；病在下部则用牛膝，在上部则去之；手臂痛则用桂枝，不痛而鼻塞则不必用。
功效：None
用法用量：每早、午、晚各服3钱，白汤送下，用香椽片过口。如服此丸反觉饮食少进，身体倦怠疲困，则药力到矣，须耐心久服，可保全功。渐加至5钱、7钱更妙。
制备方法：上为细末，水法叠丸。
临床应用：None
各家论述：None
用药禁忌：忌食面酱、酱油、火酒、川椒、羊、鹅发物等味，惟乌鱼、芝麻相宜。
附注：None
