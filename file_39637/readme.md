序号：39647
名称：细辛散
组成：附子2分（炮，去皮），秦艽3分，人参3分，牡蛎3分（熬），蜀椒3分（汗，去目闭口者），干姜5分，桂心5分，茯苓1两，桔梗1两，防风1两半，白术1两，当归1两，独活1两，柴胡5分，黄芩3分，乌头半两（炮，去皮），甘草3分（炙），麻黄3分（去节），芎3分，石南半两，莽草半两，牛膝半两，天雄半两（炮，去皮），栝楼半两，杜仲半两（炙），细辛2分。
出处：《千金翼》卷二十。
主治：风入五脏闷绝，常自燥痛，或风注入身，冷注鬼注，飞尸恶气肿起，或左或右，或前或后，或内或外，针灸流移，无有常处，惊悸，腹胀气满，叉心头痛，或恍惚悲惧，不能饮食，或进或退，阴下湿痒，或大便有血，小便赤黄，房中劳极。
加减：None
功效：None
用法用量：旦以清酒服5分匕讫，如行十里势欲歇，更饮酒5合佳。
制备方法：上为散，仍别称之合和也。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
