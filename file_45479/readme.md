序号：45494
名称：千金丸
组成：野葛（炙）4寸，斑蝥10枚（去翅足，炒），雄黄（研）半两，雌黄（研）半两，干姜（炮）半两，鬼臼（去毛，微炒）半两，瓜蒂（炒）半两，丹砂（研）半两，礜石（煅）半两，沙参半两，莽草半两，椒目（炒）半两，地胆8枚（去翅足，炒）。
出处：《圣济总录》卷一○○。
主治：气注羸瘦，历年停滞，胸胁结痞，饮食或吐，宿食不下，中风鬼注。
加减：None
功效：None
用法用量：每服5丸，酒送下，1日2次；卒中恶，闷绝不知人，服2丸，老少1丸；牛马觝践，猪犬所啮，痈肿若虫毒所啮，用1丸于掌中，唾和涂疮口毒上，立愈；正月旦，以酒服，家中大小各1丸，1年中不病；若欲视病人，先服1丸佳。
制备方法：上为末，炼蜜为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：忌生血物。
附注：None
