序号：44795
名称：急救阴阳汤
组成：人参2两，黄耆3两，当归1两，熟地2两，甘草3钱，白术2两。
出处：《辨证录》卷一。
主治：冬月伤寒，阴阳两亡，大汗而热未解，腹又痛不可按。
加减：None
功效：None
用法用量：水煎服。1剂而腹痛顿止，身热亦解，汗亦尽止矣。
制备方法：None
临床应用：None
各家论述：此方用参、耆以补气，使阳回于阴之内；用当归、熟地以补血，使阴摄于阳之中；用白术、甘草和其肠胃，而通其腰脐，使阴阳两归于气海、关元，则亡者不亡，而绝者不绝世。倘认是阳症变阴，纯用温热之剂，加入肉桂、干姜、附子之类，虽亦能回阳于顷刻，然内无阴气，阳回而阴不能摄，亦旋得而旋失矣。
用药禁忌：None
附注：None
