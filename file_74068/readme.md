序号：74084
名称：暖肾疝气丸
组成：白术（炒）5两，茯苓4两，泽泻3两，橘核（炒）3两，肉桂（去粗皮）3两，木香2两，川楝子3两，牡蛎（煅）3两，硫黄（豆腐炙）4两，知母（炒）3两，川附子2两，干姜2两，黄柏（炒）3两，昆布2两，山甲（炙）2两，乳香（炙）2两，小茴香（炙）5两，丁香2两。
出处：《北京市中药成方选集》。
主治：小肠疝气，少腹抽痛，睾丸偏坠，坚硬肿大。
加减：None
功效：舒气止痛，暖肾散寒。
用法用量：每服2钱，日服2次，温开水送下。
制备方法：上为细末，用冷开水泛为小丸，每16两用滑石细粉4两为衣，闯亮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
