序号：72601
名称：温胃汤
组成：肉桂1钱（夏月减半），厚朴1钱，陈皮1钱，香附1钱，当归1钱半，川芎1钱，车前子1钱，枳壳1钱半，黑姜5分，桃仁1钱，半夏1钱，砂仁5分，生芝麻1钱，炒米50粒。
出处：《陈素庵妇科补解》卷四。
主治：孕妇临产，胃气虚寒，或冬月去衣太早，下体受寒，或胎上通心，气逆而忽然呕吐者。
加减：None
功效：和胃温中顺气。
用法用量：None
制备方法：None
临床应用：None
各家论述：临产呕吐，其故有三，胃气虚者，以温中和胃为主；外犯寒者，以散寒温胃为主；有寒邪而伤饮食者，以消食温经为主。临产之前呕吐频作，是胎上通心，以致气逆而吐，尤当顺气温胃。是方以朴、香、半、陈消食和中，温经顺气；桂、姜行血；枳壳宽肠；桃仁破滞；车前利窍；芎、归助血。气顺血行，胎易产，呕自止也。
用药禁忌：None
附注：None
