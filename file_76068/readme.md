序号：76084
名称：蛜(虫祁)丸
组成：蛜（虫祁）（头尾全者）1两，桃仁（生）1两，白附子1两，阿魏1两，桂心1两，白芷1两，安息香（用胡桃瓤研）1两，没药3分（上8味，用童便5升，无灰酒2升，银器内熬令厚），乳香3分，当归1两，北漏芦1两，牛膝1两，芍药1两，地骨皮（去土）1两，威灵仙1两，羌活1两。
出处：《苏沈良方》卷二。
主治：鹤膝风，及腰膝风缩。
加减：None
功效：None
用法用量：每服1丸，空心暖酒送下。
制备方法：上为丸，如弹子大。
临床应用：风疰：胡楚望博士病风疰，手足指节皆如桃李，痛不可忍，服之悉愈。
各家论述：None
用药禁忌：None
附注：干蝎丸（《圣济总录》卷八）。
