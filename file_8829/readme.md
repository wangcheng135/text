序号：8829
名称：水浸鳖甲汤
组成：鳖甲（九肋者，去裙襕，醋炙）半两，升麻半两，柴胡（去苗）半两，人参半两，白茯苓（去黑皮）半两，槟榔（锉）半两，肉豆蔻（去壳）半两，诃黎勒皮半两，犀角（镑）半两，青橘皮（汤浸，去白，焙）半两，陈橘皮（汤浸，去白，焙）半两，甘草（炙，锉）半两，缩砂仁半两，茴香子（炒）半两，陈曲（炒）半两。
出处：《圣济总录》卷八十七。
主治：气劳羸瘦，四肢疼痛，心腹妨闷，不思饮食。
加减：None
功效：None
用法用量：每服3钱匕，水1盏半，浸2日，煎至7分，去滓，空心细呷，以食压之。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
