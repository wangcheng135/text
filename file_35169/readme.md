序号：35179
名称：乳香中黄膏
组成：白芷3钱，当归3钱，赤芍药3钱，生地黄3钱，地龙3钱，川山甲3钱，血竭3钱，防风3钱，大黄3钱，玄参3钱，苦参3钱，藁本3钱，苍术3钱，草乌尖半两，牙皂10条，两头尖半两（无，草乌代），南星2钱，木鳖15枚，蓖麻子150枚（去壳），乳香1两，没药（各另研不入油浸）1两，蜂房2钱，生葱2把，桃枝半两，柳枝半两，槐枝半两。
出处：《普济方》卷三一三。
主治：疮疡。
加减：None
功效：None
用法用量：None
制备方法：上用桐油5两，浸三二时，锅内文武火慢慢熬蓖麻子焦，川山甲、葱白焦黄，用布滤去滓，药油另收，入松香在锅内熬开，药入在内同煎化，又用布滤去滓，药油与松香同熬，以桃、柳枝搅匀，滴水中不散为度，如硬入油少许，药嫩加松香，方入乳、没末搅匀，再以布滤入水盆中，候冷，用水抽洗黄色为度。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
