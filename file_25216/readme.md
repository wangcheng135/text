序号：25224
名称：妇科十味片
组成：香附250kg，川芎10kg，当归90kg，玄胡索20kg，生白术14.06kg，甘草6.5kg，红枣50kg，白芍7.5kg，赤芍7.5kg，熟地黄30kg。
出处：《北京市中成药规范》第二册。
主治：月经不调，经期腹痛。
加减：None
功效：补气，益血，调经。
用法用量：将生白术、白芍、赤芍、香附打碎，甘草切碎，熟地黄、口服，每次4片，每日3次。
制备方法：红枣破开；生白术、甘草、红枣、白芍、赤芍、熟地、香附20kg煮提3次，时间分别为3小时、2小时、1小时，合并药液，过滤沉淀，减压浓缩至比重1.38～1.40、温度（50℃）的稠膏；香附230kg、当归、川芎，玄胡索粉碎为细粉，过100目筛，混匀。每料用白沙糖20kg，加入适量清水制成糖浆，加入淀粉5kg冲浆或打浆，将碳酸钙粉32.5kg，加上述稠膏与糖浆，置搅拌机内搅拌，加入药粉，再加入淀粉糊，搅拌均匀以后过14目筛制粒，用60～70℃干燥，以12目筛整粒，加2％滑石粉，均匀压片，每片重0.3g。
临床应用：None
各家论述：None
用药禁忌：None
附注：妇科调经片（《湖南省中成药规范》）。
