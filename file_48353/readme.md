序号：48368
名称：神效产灵丹
组成：当归5钱，首乌5钱，两头尖5钱，白术5钱，广木香5钱，细辛5钱，血竭5钱，人参2钱，川乌2钱，草乌2钱，大茴2钱，川芎2钱，肉桂1两半，沉香6钱，琥珀1两，防风2两，白芷2两，芥穗2两，桔梗2两，麻黄2两，炙草2两，苍术半斤。
出处：《全国中药成药处方集》（西安方）。
主治：妇人产后，代谢机能衰弱，血循环发生障碍，恶露不下，或下之不尽，胸腹胀闷，两胁刺痛，头目眩晕；及产后虚怯，感冒风寒，恶寒发热，头身疼痛。
加减：None
功效：None
用法用量：每次1丸，陈黄酒送下。服后覆卧。
制备方法：上为细末，炼蜜为丸，2钱重，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：无瘀血而内燥热者不宜服。
附注：None
