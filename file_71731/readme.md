序号：71747
名称：滋阴壮阳丹
组成：熟地（用淮生地酒蒸9次，晒9次）4两，石菖蒲5钱，远志、甘草（水浸去心）1两，淮山药2两，五味子7钱，肉苁蓉（酒浸，洗去鳞甲白膜）2两，菟丝子（酒浸，炒）2两，牛膝（酒浸）1两，巴戟（去心，酒浸）1两5钱，续断（酒浸，洗）1两5钱，茯苓（去皮）1两5钱，益智仁（去皮）1两5钱，黄柏（盐酒炒）1两5钱，知母（酒炒）1两5钱，破故纸（盐酒炒）2两，枸杞子2两，山茱萸（净肉）2两，杜仲（去皮，盐酒炒断丝）2两，沙苑蒺藜（炒）2两，人参1两，虎胫骨（酥炙）1两。
出处：《医学正印》卷上。
主治：None
加减：None
功效：阴阳两补，种子。
用法用量：每服100丸，空心盐汤送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
