序号：3315
名称：天雄酒
组成：天雄2两（炮裂，去皮脐），杜仲1两（去粗皮，炙微黄），牛膝3分（去苗），仙灵脾3分，乌蛇3两（酒浸，去骨，炙微黄），石斛3分（去根），侧子3分（炮裂，去皮脐），防风3分（去芦头），桂心1两，芎3分，川椒3分（去目及闭口者，微炒去汗），白术3分，五加皮3分，酸枣仁1两（微炒）。
出处：《圣惠》卷四十四。
主治：腰痛，元气衰虚，风冷所侵，腰背拘急，俯仰不得。
加减：None
功效：None
用法用量：每于食前温1小盏服之。
制备方法：上锉细，以生绢袋盛，用酒2斗浸，密封，经7日后开。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
