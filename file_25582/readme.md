名称：朱珀镇惊散
组成：僵蚕3钱，天麻1钱，全蝎1钱，木通1钱，犀角1钱，薄荷1钱，大黄1钱，蝉蜕1钱，柴胡1钱，粉草1钱，枳实1钱，槟榔1钱，赤芍1钱，川连1钱，天竺黄1钱，菖蒲1钱，川贝1钱，羌活1钱，钩藤1钱，橘红1钱，前胡1钱，胆星1钱5分，生地1钱5分，灯心5分，梅片5分，辰砂5钱，麝香1分，牛黄1分，琥珀2钱。
