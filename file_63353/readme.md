序号：63369
名称：清毒拔翳汤
组成：酒炒黄连5分，酒炒当归5分，酒炒花粉5分，牛蒡5分，草决明5分，桔梗5分，甘草5分，白蒺藜（炒，研，去刺）5分，真甘菊4分，密蒙花4分，谷精草4分，川木贼4分，粉葛根4分，川芎3分，羌活3分，柴胡3分，防风3分，薄荷3分，生地3分，酒炒山栀3分。
出处：《痘疹活幼至宝》。
主治：痘毒留津于气血精华之分，落为眼患，赤肿而痛，不能开，或有翳膜遮蔽不能视者。
加减：大便秘者，加酒炒大黄，通即去之。
功效：None
用法用量：加生姜1片同煎，食后良久服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌用寒凉之药点眼。
附注：None
