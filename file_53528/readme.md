序号：53543
名称：凌花散
组成：当归（酒浸）、凌霄花、刘寄奴、红花（酒浸，候煎药1-2沸即入）、官桂（去皮）、牡丹皮（洗）、川白芷、赤芍药、延胡索各等分。
出处：《医方大成》卷九引《澹寮》。
主治：妇人月水不行，发热腹胀。妇人腹满，身体疼痛，瘦淬食少，发热自汗。
加减：None
功效：None
用法用量：上（口父）咀。每服4钱，水1盏，酒半盏，煎8分，再入红花煎，热服。
制备方法：None
临床应用：None
各家论述：《医略六书》：凌霄花破血降火；刘寄奴破血通经；当归养血，统营之运；赤芍破血，泻火之亢；延胡化血滞以通经脉，红花活血脉以浚血海；官桂温经通闭；丹皮凉血化血；白芷散阳明之邪以清冲任之脉也。为散以散之，温酒以行之，使瘀血顿化，则经气自调，而经血应时以下，何经闭发热之不瘳乎？
用药禁忌：None
附注：凌霄花散（《济阴纲目》卷二）。
