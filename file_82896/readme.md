序号：82912
名称：丹砂丸
组成：朱砂1两，天南星1两（生用），赤箭1两，附子1两（炮裂，去皮脐），防风1两（去芦头），牛膝1两（去苗），汉防己1两，白附子1两（生用），独活1两，白僵蚕1两（微炒），麻黄1两（去根节），芎1两，桂心1两，白花蛇肉1两（酥拌，炒令黄），蝉壳1两，川乌头1两（炮裂，去皮脐），羚羊角屑1两，干蝎1两（生用）桑螵蛸1两（微炒），乌犀角屑1两，雄黄半两（研入），麝香1两（研入），牛黄1分（研入），龙脑1分（研入）。
出处：《圣惠》卷二十三。
主治：中风，手足不遂，言语謇涩，缓纵不仁，肢节疼痛。
加减：None
功效：None
用法用量：每服10丸，空心及晚食前以温酒送下。
制备方法：上为末，入研了药，更研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
