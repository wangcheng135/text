序号：38921
名称：泻肺通窍汤
组成：苏子1钱，葶苈子1钱，莱菔子1钱，北芥子1钱，麻黄（捶，水泡过）1钱，杏仁7分，枳壳7分，黄芩7分，桑皮7分。
出处：《一盘珠》卷四。
主治：肺胀，两鼻煽动，汗出如流，胸高气喘，肺脉急数。
加减：大便不通，加大黄、槟榔。
功效：None
用法用量：None
制备方法：None
临床应用：一妇人年余六十，急患肺胀，两鼻如蝉翼，气喘，大汗如雨，诸医谓年老虚痰上涌，用姜桂六君子汤，汗更甚，请予诊之。诊得肺脉八九至，予曰：此肺胀也。若是虚汗，一二次必发寒厥，如此热汗如雨，淋淋不已，实为痰火内冲，肺窍闭塞。遂用泻肺通窍汤三剂，汗自止，吐痰数日而愈。后用八珍汤十剂而痊。又一小孩十二岁，患肺胀，热汗如雨，五六日不大便，气喘嗄声，面赤烦躁。余用泻肺通窍汤加大黄、黄连各一钱而愈。
各家论述：None
用药禁忌：None
