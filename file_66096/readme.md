序号：66112
名称：蛇蝎续命汤
组成：白花蛇（酒浸，去皮骨，焙）1两，全蝎（炒）1两，独活（去土）1两，天麻1两，附子（炒，去皮脐）1两，人参1两，防风1两，肉桂（去粗皮）1两，白术1两，藁本1两，白附子（炮）1两，赤箭1两，川芎1两，细辛（去叶）1两，白僵蚕（去土灰，炒）1两，甘草（炒）1两，半夏（汤浸糊，切）1两，白茯苓（去皮）1两，麻黄（去节，水煮三沸，去沫，细切）1两。
出处：《御药院方》卷一。
主治：卒中急风，牙关紧急，精神昏愦，口眼斜，不省人事，痰涎不利，喉中作声。
加减：None
功效：None
用法用量：每服5钱，加水1盏，入生姜5片，煎至7分，去滓，稍热服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
