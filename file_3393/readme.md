序号：3393
名称：太乙丹
组成：五灵脂半斤（去土石），木鳖子半斤（去壳，别研），草乌2两（去皮尖），僵蚕4两（炒，去丝嘴），钱子地龙半斤（布袋去土，不见火），川乌4两（去皮尖），防风4两（去芦），白胶香6两（别研），麝香半两（别研），脑子2钱（别研），好墨（烧用）4两，桑柴炭（好醋浸宿）4两。
出处：《普济方》卷九十三引《卫生家宝》。
主治：瘫痪诸风疾。
加减：None
功效：None
用法用量：每丸分3服，用生姜、薄荷研自然汁浸，温酒1盏嚼下。须腊日或端午日合。
制备方法：上为细末，酒糊为丸，如弹子大，阴干。
临床应用：None
各家论述：None
用药禁忌：忌发风物；孕妇不可服。
附注：None
