序号：51887
名称：闾茹膏
组成：闾茹2两，狼牙2两，青葙2两，地榆2两，藜芦2两，当归2两，羊蹄根2两，萹蓄2两，蛇床子6分，白蔹6分，漏芦2分，上为末，以苦酒渍1宿，明旦以煎成猪膏4升煎之三上三下，膏成，绞去滓，纳后药：雄黄2两，雌黄2两，硫黄2两，矾石2两，胡粉2两，松脂2两，水银2两。
出处：《千金》卷二十三。
主治：一切恶疮、疥、癣、疽、漏、瘑。
加减：None
功效：None
用法用量：煎膏法必微火，急则不中用。先研雄黄等令细，候膏小冷，即和搅敷之。
制备方法：上为末，看水银散尽，即倾前件膏中，搅匀，用瓷器贮之，勿令泄气。
临床应用：None
各家论述：None
用药禁忌：不可近目及阴。
附注：None
