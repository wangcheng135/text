序号：38576
名称：法制竹沥丸
组成：陈皮（去白）3两，白术（炒）3两，白茯苓3两，炙甘草3两，半夏曲3两，贝母3两，枳壳3两，神曲（炒）3两，桔梗3两，黄芩3两，玄明粉1两，香附子（制）1两。
出处：《医统》卷四十三引海上道人方。
主治：劳嗽。
加减：None
功效：清热降火，化痰利膈，止嗽，止呕吐，进食开胃。
用法用量：每服80丸，食后或卧时白汤送下。3日便见效验，久病者7日效，疲者1月痊愈。
制备方法：上为粗末，用大瓷盆以竹沥1碗，入姜汁，酒半盏和匀拌诸药，日中晒干，仍依法入竹沥、姜汁拌晒7次为度，磨罗为细末，酒水为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
