序号：47341
名称：祛风补益蛜(虫祁)丸
组成：蛜（虫祁）1两半（微炒），白附子1两（炮裂），沉香1两，肉桂1两（去皱皮），芎1两，槟榔1两，木香1两，天麻1两，石斛2两（去根，锉），牛膝2两（去苗），白蒺藜1两（微炒，去刺），附子1两（炮裂，去皮脐），巴戟3分，白僵蚕3分（微炒），羌活3分，肉苁蓉2两（酒浸1宿，刮去皱皮，炙令干），当归3分（锉，微炒），山茱萸3分。
出处：《圣惠》卷二十三。
主治：脏腑久虚，风冷所攻，四肢无力，背膊多疼，膀胱冷气流注，腰脚沉重。
加减：None
功效：None
用法用量：每服20丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
