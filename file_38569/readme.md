序号：38579
名称：法制杏子
组成：杏仁1斤（拣板杏真方用），苍术1斤，白沙蜜2斤，苍术1两（去粗皮，泔浸1宿，麸炒），半夏1两（姜汁浸1宿），木香1两，当归1两，人参1两。
出处：《瑞竹堂方》卷一。
主治：None
加减：若气不顺，加木香末1两；血不和，加当归末1两；若喘息，加人参末1两；若虚弱，加苍术末1两；痰盛，加半夏末1两；若妇人血气不和，加当归服。
功效：调补。
用法用量：每日空心细嚼30-50个，面汤送下。
制备方法：上将杏仁用热水洗净，顿于罐内，将生苍术1斤为粗末，水熬一二百沸，将汁浸杏仁，封其口，夏1日1夜，春秋3日3夜，冬5日5夜，去苍术及汁，锅内炒干，调入蜜半斤，再炒，少时取出，将木香、当归、人参、半夏、苍术俱为细末，拌于杏仁上，冷定，用瓷罐盛顿，用生蜜半斤浇上。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
