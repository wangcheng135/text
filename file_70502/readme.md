序号：70518
名称：紫金丹
组成：天麻1钱5分，胆星5钱，僵蚕1钱5分，丹皮1钱5分，竺黄2钱，白附2钱，全蝎3钱，胡连1钱，骨皮1钱5分，朱砂5分，牛黄4分，麝香2分。
出处：《全国中药成药处方集》（吉林方）。
主治：小儿急惊，壮热烦渴，痰壅气促，牙关紧闭，四肢搐搦；汗出当风，发为痉风，壮热无汗，角弓反张，身热足寒，颈项强急，面赤目赤，头摇直视，口噤神昏；并主风痰，咳嗽痰喘，痰鸣气促；感冒寒凉，发热畏冷，头痛体痛。
加减：None
功效：镇惊、疏风、涤痰、解表。
用法用量：每服1丸，病重2丸，鲜姜、薄荷、桑叶、菊花为引。
制备方法：上除牛黄、麝香、朱砂另研外，余均为细末，陆续调合一处，炼蜜为丸，2分1厘重，或3分5厘重，大赤金为衣。绵纸包裹，蜡皮封固。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻。
附注：None
