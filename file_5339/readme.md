序号：5339
名称：加减乌金散
组成：厚朴1钱5分，柴胡1钱5分，黄芩1钱5分，麻黄2钱，陈皮1钱5分，当归1钱5分，川芎1钱5分，桔梗1钱5分，茯苓1钱5分，桂枝1钱，苍术1钱，白芷1钱，枳壳1钱，羌活2钱，草果2钱，半夏2钱，甘草9分，白芍药1钱5分，熟地黄1钱5分。
出处：《准绳·女科》卷五。
主治：产后寒热似疟。三阴疟有锗杂之邪者。
加减：有汗，多当归、川芎、白芍药、熟地黄；有胀，多厚朴、陈皮；有热，多柴胡、黄芩；有寒，多苍术、草果桂枝；有痰，多半夏、桔梗、茯苓；有头痛，多川芎、白芷、羌活；有泻，去枳壳、甘草；有余血块在腹，作潮热疼痛，加三棱、莪术，多用延胡索、八角、茴香；遍身痛，加羌活、独活；寒热往来，加黄芩、柴胡。
功效：None
用法用量：每服用水1钟半，加生姜3片，葱3茎，煎至1钟，不拘时服。
制备方法：上锉为散，分作两服。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
