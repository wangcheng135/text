序号：67029
名称：麻桂饮
组成：官桂1-2钱，当归3-4钱，炙甘草1钱，陈皮（随宜用，或不用亦可），麻黄2-3钱。
出处：《景岳全书》卷五十一。
主治：伤寒、瘟疫、阴暑、疟疾，凡阴寒气胜而邪有不能散者。
加减：若阴气不足者，加熟地黄3-5钱；若三阴并病者，加柴胡2-3钱；若元气大虚，阴邪难解者，当以大温中饮更迭为用。
功效：None
用法用量：上以水1钟半，加生姜5-7片或10片，煎至8分，去浮沫，不拘时服。但取津津微汗透彻为度。四季皆可用之。
制备方法：None
临床应用：None
各家论述：此实麻黄桂枝二汤之变方，而其神效则大有超出二方者。
用药禁忌：None
附注：麻桂汤（《会约》卷十）。
