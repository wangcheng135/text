序号：2749
名称：天仙丹
组成：三仙大红升丹（须自炼者为佳）2两，天仙子6两（研极细），五虎拔毒丹1两，梅片3钱。
出处：《疡科纲要》卷下。
主治：疔毒及脑疽、背疽、腹皮大痈，溃后脓多，或腐肉不脱。
加减：None
功效：提脓拔毒，去恶腐。
用法用量：临用挹尽脓水，须以100倍石炭酸淋洗净，棉纸挹干，以此末子细细掺遍疮口，以膏盖之，1日2换，吸尽脓腐，不伤好肉，不觉痛苦，最为稳妥。
制备方法：上药各为极细末，和匀密贮。
临床应用：None
各家论述：广东药肆，有所谓天仙子者，其形小圆而扁，其色深黄，光泽滑润，一得水湿，则自有粘质，稠如胶浆，以治溃疡，吸取脓水，其力颇峻，寻常疮疖，嫌其吸力太富，反觉痛苦，惟疮脓多，及脑疽、背疮、腹皮痈等大证，腐化已巨，脓水甚多者，以此提脓吸毒，去腐极易，并不痛苦。考《本草纲目》有莨菪子，一名天仙子，而所载形色性情，实非此药，或粤省所独有，未入本草之物，颐用之有年，恃为利器，爰合以三仙丹数味，配为一种末子，专治大毒大腐。是新方之适宜于实用者，即以粤东之名是方，以旌其功，允足当佳名而无愧色。
用药禁忌：None
