序号：48837
名称：禹余粮丸
组成：桑寄生1两，柏叶（微炒）1两，当归（去芦，微炒）1两，厚朴（去粗皮，姜汁炒）1两，干姜1两，白术1两，鳖甲（醋浸，去裙，炒黄）1两，附子（炮，去皮脐）1两，禹余粮（烧，醋淬7次，细研）5钱，扁豆5钱（炒）。
出处：《普济方》卷三二二引《医学集成》。
主治：妇人带下久虚，胞络伤败，月水不调，渐成崩漏，气血虚弱，面黄肌瘦，脐腹里急，腰膝疼重，肢体烦痛，心忪头眩，手足寒热，不思饮食。
加减：止泻，加黑豆；止痢，加粟壳（蜜炒）。
功效：None
用法用量：每服3钱，以水1盏半，加生姜3片、红枣2枚煎，温服。
制备方法：上锉散。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，据剂型，当作“禹余粮散”。
