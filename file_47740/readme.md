序号：47755
名称：神仙备急丹
组成：沉香（略炒）半两，木香（湿纸裹，炮）半两，槟榔半两，白姜（泡，洗）半两，石菖蒲（酒浸1宿）半两，朱砂（别研）半两，牡蛎粉半两，桃仁（去皮尖，炒，别研）半两，磁石（火煅，酸醋淬7次，别研，水飞，令极细）半两，阿魏（酒化）半两，硫黄半两（别研），茴香（淘去沙，炒）7钱半，缩砂仁7钱半，红豆（炒）7钱半，禹余粮石（火煅，米醋淬7次，别研，水飞，令极细）7钱半，当归（去芦，炒）7钱半，神曲（炒）7钱半，附子（炮，去皮脐）7钱半。
出处：《魏氏家藏方》卷二引庆元府慈应大师方。
主治：脾肾气，时作雷鸣，腹胁胀满，不美饮食，胸膈隘滞，秘利不时，及暴下呕逆。
加减：None
功效：None
用法用量：每服30丸或40丸，空心煎姜汤送下。
制备方法：上为细末，研桃仁为膏，和入酒煮阿魏糊丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
