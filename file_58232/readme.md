序号：58247
名称：盐煎散
组成：槟榔1两半，厚朴1两半（制），草果1两（炒），良姜1两，澄茄1两，甘草5钱，青皮1两半，陈皮1两半，葛根1两半。
出处：《医略六书》卷十九。
主治：冷气攻冲，胸腹刺痛，脉沉者。
加减：None
功效：温中破滞。
用法用量：每服3钱，入盐少许，煎服。
制备方法：上为散。
临床应用：None
各家论述：《医略六书》：澄茄温中散冷，良姜暖胃驱寒，槟榔破滞气，厚朴除腹满，草果散寒消滞，甘草和胃缓中，青皮破气平肝，陈皮利气调胃，葛根升阳开胃，入盐润下以降逆也。俾逆气平而肝胃调，则清气升而浊气降，其胸腹冷气无不散，攻冲胸腹无不平，何刺痛之不瘳哉。此温中破滞之剂，为冷冲胸腹刺痛之专方。
用药禁忌：None
附注：None
