序号：4956
名称：木香鳖甲汤
组成：木香1分，鳖甲（9肋者，去裙襴，醋炙黄）1两，柴胡（去苗）1两，秦艽（去苗土）3分，黄耆1两，知母（焙）3分，白茯苓（去黑皮）3分，人参1两，桔梗（炒）3两，白术1两，甘草（炙）1两，防风（去叉）3分，肉豆蔻（去壳）1分，半夏半两（生姜3两取汁，煮令汁尽，焙），枳壳3分（去瓤，麸炒），芍药3分。
出处：《圣济总录》卷八十八。
主治：虚劳寒热往来，不思饮食，口舌生疮，四肢劳倦，五心烦躁，肌肤不泽。
加减：None
功效：None
用法用量：每服半两，加生姜1分(切碎)，大枣3枚，水3盏，煎至2盏，去滓分温2服，早晨、日晚各1次。
制备方法：上锉细，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
