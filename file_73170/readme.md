序号：73186
名称：山茱萸散
组成：山茱萸3分，五味子3分，白茯苓（去黑皮）3分，当归（切，焙）半两，附子（炮裂，去皮脐）半两，芎半两，芍药半两，石韦（去毛，炙）半两，桂（去粗皮）半两，人参半两，地脉草半两，石斛（去根，酒浸，焙）半两，菟丝子（酒浸，炙）半两，甘草（炙）半两，巴戟天（去心）1两，远志（去心）1两，麦门冬（去心，焙）1两，肉苁蓉（酒浸，去皴皮，炙）1两，生干地黄（炒）1两，干姜（炮）1分。
出处：《圣济总录》卷一二八。
主治：发背痈疽，经年不愈，热气结聚。
加减：None
功效：None
用法用量：每服2钱匕，温酒调下。加至3钱匕。醋浆水调下亦得。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
