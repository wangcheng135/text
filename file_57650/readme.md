序号：57665
名称：大救生丸
组成：牡蛎（生用）1两，苦参（生、为末）1两，栝楼（生捣）1两，知母（生、为末）1两，蜜陀僧（生用）1两，白蜡（熔）8分，水银（研）8分，黄丹半斤（研）（1方有天花粉半两；1方有灰坯，无密陀僧）。
出处：《普济方》卷一七六引《卫生家宝》。
主治：三消渴病，日夜饮水，百杯不歇；子母疮，或生于背，或生于鬓间；五漏疮。
加减：None
功效：None
用法用量：用猪肚1个贮药，以线缝合，用绳子系在新瓦砖上、不令走转；更用栝楼根半斤，细切，入在水中，1处和砖煮，早晨至午，取猪肚细切，与诸药末为丸，如梧桐子大，阴干。每服30丸，空心米饮送下，1日3次。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：忌酒、色、热面、咸物、豚、鱼、葱、蒜、炙煿等物。
附注：若饮酒则渴愈甚者，专心服饵之。数日后，见酒与水若仇，顿尔口中津润，小便缩减。五日后，小便赤色，是病毒归于下也。或令人吐，或腰背脚膝疼痛，或呕逆恶心，精神昏困，此乃药验，使病毒消散也。方中白蜡用量原缺。
