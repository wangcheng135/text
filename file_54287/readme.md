序号：54302
名称：柴胡汤
组成：柴胡（去苗）1两，桑根白皮1两，羌活（去芦头）1两，百合1两，当归（切，焙）1两，半石膏（碎），麻黄（去根节，先煎，掠去沫，焙）、天雄（炮裂，去皮脐）2两，黄连（去须）1两，贝母（煨，去心）1两，五味子1两，桂（去粗皮）1两，枳壳（去瓤，麸炒）1两，白石脂1两，款冬花1两，黄芩（去黑心）半两，杏仁10枚（去皮尖双仁，炒）。
出处：《圣济总录》卷二十四。
主治：伤寒后咳嗽经旬未愈。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，加生姜1枣大（切），煎至8分，去滓温服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
