序号：3938
名称：无价金丹
组成：白术（去芦，炒）3两，枳实（麸炒）1两，苍术（米泔浸，炒），猪苓1两，麦芽（炒）2两，神曲（炒）2两，半夏（汤泡）2两，泽泻7钱，赤茯苓（去皮）7钱，川芎7钱，黄连（陈土炒）7钱，白螺蛳（煅）7钱，砂仁5钱，草豆蔻5钱，黄芩（陈土炒）5钱，青皮（去瓤）5钱，莱菔子（炒）5钱，生姜5钱，陈皮（去净白）3钱，香附子（童便炒）3钱，瓜蒌仁3钱，槟榔3钱，川厚朴（去皮，姜炒）2钱，木香2钱，甘草2钱。
出处：《寿世保元》卷五。
主治：一切诸积在胃脘当心而痛，及痞满恶心，嘈杂呕吐，嗳气吞酸，脾疼，诸痛。
加减：吞酸，加吴茱萸（汤泡），寒月用5钱，热月用2钱半：久病挟虚，加人参、扁豆、石莲肉各5钱；时常口吐清水，加炒滑石1两，牡蛎煅5钱。
功效：清痰涎，消食积、酒积、肉积、茶积。
用法用量：每服70丸，多至100丸，米汤送下。
制备方法：上为细末，青荷叶泡汤浸晚粳米，研粉作糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
