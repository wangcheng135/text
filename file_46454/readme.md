序号：46469
名称：济阴寿子饮
组成：人参1钱，当归身（土炒）2钱，熟地（择顶大枝头无灰者，酒拌，9蒸9晒）2钱，白术（土炒）2钱，川芎8分，紫苏4分，陈皮4分，炙草4分。
出处：《胎产心法》卷上。
主治：None
加减：弱者，人参倍用之；虚肥人，陈皮去白，加蜜炙黄耆5分；泄泻，加莲子（去心）10枚，带皮砂仁3分，减地黄；多怒，加木香2分，磨冲药服；口躁，加去心麦冬1钱；怔忡，加炒枣仁1钱，益智仁1钱，龙眼肉10枚。
功效：大益胎元，济阴寿子，稀疮痘。
用法用量：加大枣1枚，水煎，食远温服，滓再煎服，不拘时候。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
