序号：57250
名称：消斑青黛饮
组成：柴胡、玄参、黄连、知母、石膏、青黛、生地黄、山栀、犀角、人参、甘草。
出处：《回春》卷二。
主治：热传里，里实表虚，血热不散，热气乘虚出于皮肤而为斑，轻如疹子，重则如锦纹，重甚则斑烂皮肤。
加减：None
功效：None
用法用量：加生姜1片，大枣1个，水煎，临服入醋1匙同服。
制备方法：上锉1剂。
临床应用：None
各家论述：《医方集解》：发斑虽出胃热，亦诸经之火有以助之。青黛、黄连以清肝火，栀子以清心肺之火，玄参、知母、生地以清肾火，犀角、石膏以清胃火。此皆大寒而能解郁热之毒者；引以柴胡，使达肌表，使以姜、枣，以和营卫，其用人参、甘草者，以和胃也。胃虚故热毒乘虚入里，而发于肌肉也。加苦酒者，其酸收之义乎！
用药禁忌：None
附注：None
