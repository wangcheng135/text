序号：57180
名称：消积丸
组成：砂仁半两，制青皮（锉）半两，净陈皮半两，莪术半两，三棱（并煨）半两，巴豆21个（去皮膜，不去油，同前5味炒令黄色，去巴豆不用）。
出处：《永类钤方》卷二十引《保婴方》。
主治：小儿乳食宿滞不化，胸膈不利，呕吐恶心，便利不调，大便酸臭，重伤乳食，或作诸痫。小儿一切所伤，面黄肌瘦，肚大腹胀，癖积块硬，头大脚细，虽进饮食，不生肌肉，腹内作痛，下痢腥臭。
加减：None
功效：消化癖积，美进乳食。
用法用量：每服20-30丸，温米汤送下，日2次。
制备方法：上为末，醋面糊为丸，如黄米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
