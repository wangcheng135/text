序号：67078
名称：麻黄汤
组成：麻黄（去根节，汤掠去沫，焙）2两，萆薢2两，附子（炮裂，去皮脐）2两，黄连（去须）1两，当归（切，焙）1两，桂（去粗皮）1两，枳壳（去瓤，麸炒）1两，甘草（炙，锉）1两，羚羊角（镑）1两，桑根白皮1两半，牡丹皮1两半，羌活（去芦头）1两半，芎1两半，旋覆花（炒）半两，杏仁（去皮尖双仁，炒）14枚。
出处：《圣济总录》卷六。
主治：中风，口眼邪。
加减：None
功效：None
用法用量：每服5钱匕，以水1盏半，加生姜半分（切），煎至8分，去滓温服。
制备方法：上锉，如麻豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
