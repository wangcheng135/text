序号：40077
名称：肾沥汤
组成：桑根白皮（锉）2两，黄耆（锉）3分，五味子（去梗）3分，肉苁蓉（酒浸，切，焙）3分，防风（去叉）3分，秦艽（去苗土）3分，泽泻3分，巴戟3分，天（去心）3分，桂（去粗皮）3分，山芋3分，丹参3分，茯神（去木）3分，牛膝（酒浸，切，焙）3分，石斛（去根）1两，磁石（煅，醋淬2-7遍）1两，杜仲（去粗皮，锉，炒）3分，人参3分。
出处：《圣济总录》卷八十五。
主治：五种腰痛，肾脏虚冷，脚弱不能行步。
加减：None
功效：None
用法用量：每服先用水2盏煮羊肾1只，至1盏去肾，入药末3钱匕，加生姜3片，煎至7分，去滓，空心、日午、夜卧温服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
