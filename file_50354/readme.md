序号：50369
名称：茵芋丸
组成：茵芋（去梗，锉）1分，朱砂（水飞）1分，薏苡仁1分，牵牛子1两半，郁李仁（去皮尖，微炒）半两。
出处：《本事》卷三。
主治：历节肿满疼痛。
加减：None
功效：None
用法用量：每服10-15丸，至20丸，五更初，温水送下。到晚未利，再1-2服，快利为度，白粥将息。
制备方法：上为细末，炼蜜为丸，如梧桐子大，轻粉滚为衣。
临床应用：None
各家论述：《本事方释义》：茵芋气味苦温，入手少阴；薏苡仁气味甘平淡渗，入手、足太阴；牵牛子气味苦辛微寒，入手、足阳明；郁李仁气味辛温而润，入手、足太阴、阳明；轻粉气味辛咸微寒，为衣者，取其能引药入下也。历节而致肿满，非下利不能杀其势，故渗湿行经，必使其下利也。
用药禁忌：None
附注：None
