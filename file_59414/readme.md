序号：59429
名称：莽草散
组成：莽草1两半（炙，去毛出汗），蝎梢1分，官桂（去皮）1分，当归1两，羌活半两，荆三棱1两，蓬莪术1两。
出处：《普济方》卷九十三引《博济》。
主治：瘫痪等风疾。
加减：None
功效：None
用法用量：如人卒中恶风，昏迷未省，风涎方盛，即时以药4钱，用2钱生末，先置盏中，以2钱末，用酒1盏半，煎至8分，倾在生药盏内，调匀带热服，但不得咽喉，扶起须臾，即吐出恶涎，立愈。如诸般风疾，并依此法。每日1服，永无中风及痰涎。如久患瘫痪，旦暮常服，经月余必被上有臭汗气，痕迹如人形状，其患永愈。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
