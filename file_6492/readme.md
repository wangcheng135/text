序号：6492
名称：木香散
组成：木香3分，破故纸3分，高良姜3分，砂仁3分，厚朴3分（姜汁炙），赤芍药半两，陈橘红半两，肉桂半两，白术半两，胡椒1分，吴茱萸1分（汤洗去黑水），肉豆蔻4枚，槟榔1个。
出处：《苏沈良方》卷四。
主治：脏腑冷极，及久冷伤惫，口疮下泄，谷米不化，饮食无味，肌肉瘦悴，心多嗔恚。妇人产后虚冷下泄，及一切水泄、冷痢。
加减：None
功效：逐下冷气。
用法用量：每服3钱，不经水猪肝4两许，去筋膜，批为薄片，重重掺药，置1鼎中，入浆水1碗，醋1茶脚许，盖覆，煮肝熟，加盐1钱，葱白3茎(细切)，生姜弹子许(捶碎)，同煮水欲尽，为1服，空心冷食之。初服微泻不妨，少时自止。经年冷利滑泻，只是1服。渴即饮粥汤送下。如不能食冷物，即添少浆水暖服。
制备方法：上为散。
临床应用：久泻：张某某，久泻，忽有人召食，以疾辞不往，主人曰：吾有良方，一服可瘥。煮药而召之，张至，先服药，便就席，熟醉而归，竟不复泻。
各家论述：None
用药禁忌：忌生冷、油腻物。
附注：按：本方改为丸剂，名“木香豆蔻丸”(见《普济方》卷二○七引《东坡家藏方》)。
