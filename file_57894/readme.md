序号：57909
名称：皱血丸
组成：菊花（去梗）3两，茴香3两，香附（炒，酒浸1宿，焙）3两，熟干地黄3两，当归3两，肉桂（去粗皮）3两，牛膝3两，延胡索（炒）3两，芍药3两，蒲黄3两，蓬术3两。
出处：《局方》卷九（续添诸局经验秘方）。
主治：妇人血海虚冷，气血不调，时发寒热，或下血过多，或久闭不通，崩中不止，带下赤白，癥瘕癖块，攻刺疼痛，小腹紧满，胁肋胀痛，腰重脚弱，面黄体虚，饮食减少，渐成劳状，及经脉不调，胎气多损，产前、产后一切病患。胞络伤损，宿瘀干血不散，受胎不牢，而致损堕。
加减：None
功效：暖子宫，种子。调补冲任，温暖血海，去风冷，益血。
用法用量：每服20丸，温酒或醋汤送下；血气攻刺，炒姜酒送下；癥块绞痛，当归酒送下。
制备方法：上为末，用乌豆1升醋煮，候干，焙为末，再入醋2碗，煮至1碗，留为糊，为丸。如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌鸭肉、羊血。
附注：None
