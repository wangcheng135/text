序号：42644
名称：保元丸
组成：鹿胶160两，五味子（炙）20两，补骨脂（炒）20两，山药40两，牛膝20两，芦巴（炒）20两，杜仲炭12两，益智仁20两，柏子仁20两，茯苓40两，全蝎3两，小茴香（炒）20两，淫羊藿（炙）20两，熟地40两，川楝子20两，沉香6两，菟丝子30两，巴戟（炙）20两，远志（炙）20两，山甲（炙）6两，山萸肉（炙）20两，苁蓉（炙）80两。
出处：《北京市中药成方选集》。
主治：气虚肾寒，腰膝无力，精神疲倦，小便频数。
加减：None
功效：滋阴补肾，益智宁神。
用法用量：每服1丸，温开水送下，1日2次。
制备方法：上为细末，炼蜜为丸，每丸重3钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
