序号：35852
名称：参术健脾丸
组成：苍术8两（2两盐水浸，2两米泔浸，2两醋浸，2两葱白炒），人参2两，白术（去芦）2两，白茯苓（去皮）2两，干山药（炒）2两，破故纸（酒炒）2两，枸杞子（去硬）2两，菟丝子（酒制，焙）2两，莲肉（去心）2两，川楝子（取肉）1两半，五味子1两半，川牛膝（去芦）1两半，川椒（去目，炒）5钱，小茴香（盐炒）5钱，陈皮5钱，木香（不见火）5钱，远志（甘草水泡，去心）5钱。
出处：《回春》卷三。
主治：脐腹冷痛，泄泻年久不止。
加减：None
功效：滋养元气，补理脾胃，益肾水，温下元，进饮食，调中下气，除寒湿，大补诸虚。
用法用量：每服80丸，空心盐汤送下，以干物压之。
制备方法：上为细末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
