序号：84172
名称：五石丸
组成：紫石英1两半（细研，水飞过），钟乳粉1两半，白石英1两半（细研，水飞过），赤石脂1两（细研），石膏1两（细研，水飞过），五味子1两，熟干地黄1两半，麦门冬1两半（去心，焙），黄耆1两（锉），白茯苓1两，白术1两，当归1两（锉，微炒），人参1两（去芦头），甘草半两（炙微赤，锉），桂心1两，芎1两。
出处：《圣惠》卷八十一。
主治：产后虚羸寒热，四肢瘦弱，不思饮食，心神虚烦，夜卧不安。
加减：None
功效：None
用法用量：每服30丸，以薤白汤送下，1日3次。
制备方法：上为末，入研了药，都研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
