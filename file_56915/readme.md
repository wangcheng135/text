序号：56930
名称：消胀丹
组成：白术3钱，茯苓1两，麦冬5钱，熟地5钱，山药1两，芡实5钱，苏子1钱。
出处：《辨证录》卷五。
主治：肺、脾、肾三经之虚，气喘作胀，腹肿，小便不利，大便亦溏，渐渐一身俱肿。
加减：None
功效：None
用法用量：水煎服。1剂而喘少定，2剂而胀渐消，10剂而小便利，20剂而一身之肿无不尽愈也。
制备方法：None
临床应用：None
各家论述：方中白术、茯苓以健其脾土，麦冬、苏子以益其肺金，熟地、山药、芡实以滋其肾水，自然脾气旺而不至健运之失职，肺气旺而不至治节之不行，肾气旺而不至关门之不开，水自从膀胱之府而尽出于小肠矣，安得而再胀哉！
用药禁忌：None
附注：None
