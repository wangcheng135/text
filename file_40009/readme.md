序号：40019
名称：肺伤汤
组成：五味子1两，紫菀1两，熟地黄2两，阿胶2两（炒），人参半两，杏仁1两（炒，去皮尖），黄耆1两（蜜炙），川当归2两，桑白皮1两（炒），甘草1两（炙），款冬花1两，桂1分，枳壳半两（去皮瓤，炒），干姜1两半（炮），青蒿半两（小便浸），黄芩半两。
出处：《普济方》卷一五七引《卫生家宝》。
主治：远近一切嗽疾，浑身劳倦，胁下疼，时作潮热，饮食减少。
加减：None
功效：None
用法用量：每服3钱，水1盏，入生姜3片，枣子、乌梅各1枚，煎至7分，去滓，空心温服，1日3次。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
