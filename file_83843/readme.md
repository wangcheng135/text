序号：83861
名称：乌麻地黄酒
组成：6月6日曲4升（净），王斯油麻6斗5升（出虢州，赤色者是。如无，别用巨胜替之，以脱去皮，晒干。脱乌麻法：以冷水浸经1宿出之，置筲箕中漉水令尽，春之即皮自脱去），生地黄4斗（冷熟汤洗，待水气尽便切之，更取生地黄1石，以水1石和煮，粗布绞去滓，即取汁6斗，又以蜡及麻子涂瓮内，蒸之令干，前3味总纳瓮中浸之），丹参2斤，生石斛2斤，牛膝2斤，杜仲2斤，萆薢2斤，生姜2斤，人参8两。
出处：《外台》卷二十一引《崔氏方》。
主治：风虚。
加减：None
功效：补不足，除百病。
用法用量：上切，以生绢袋盛，同纳前件熟地黄汁瓮中浸，封闭7日外，更取乌豆4大斗，摩使光净，分作4度，微熬令香，取无灰重醖酒2斗8升，3度淋豆，豆1熬3淋，淋讫并去豆，总计12度淋豆，取淋酒别器中盛，然后更泻曲汁等物及诸药，并出在大瓮中，以物闭头，及更将此酒先重蒸瓮看冷暖，还纳曲汁及药等安在瓮中，其日即用8斗精糯米，炊作饭，如常酿酒法，酘酒，即以淋豆酒投在瓮中，封闭经1-2宿，看米消尽，又炊4斗糯米饭酘之。此后更封闭经7日，其酒即熟。任性饮，多少量之，不拘时候。常微微觉身润，夜中稍加少许，或汗出佳。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌房室，避风，特禁毛桃、芥、生菜、热面并酢、蒜、牛肉、冷物。
