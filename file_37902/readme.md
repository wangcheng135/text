名称：拨云散
组成：当归尾3钱4分，防风3钱4分，胆草3钱4分，黄连3钱4分，连翘3钱4分，黄芩3钱4分，黄柏3钱4分，硼粉3钱4分，石决3钱4分，蒙花3钱4分，车前3钱4分，赤芍花粉3钱4分，谷精3钱4分，柴胡3钱4分，玄参3钱4分，川军3钱4分，菊花3钱4分，山栀3钱4分，木通3钱4分，蝉蜕3钱4分，荆芥3钱4分，木贼3钱4分，蒺藜3钱4分，生地3钱4分，羌活3钱4分，川芎3钱4分，甘草3钱4分，薄荷3钱4分，草决3钱4分，甘石3斤2两。
出处：《全国中药成药处方集》（吉林方）。
主治：云蒙翳睛，暴发火眼，烂眼边，瞳仁散大，迎风流泪。
加减：None
功效：蠲翳清蒙，收瞳明目，解痒止痛。
用法用量：用银簪蘸冷水点药，上于眼内或眼角。
制备方法：将前药熬水煅甘石，研面，用水飞过，再研极细面，按每1两兑梅片4分即成，装眼药瓶内严封。
临床应用：None
各家论述：None
用药禁忌：None
