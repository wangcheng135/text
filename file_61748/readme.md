序号：61764
名称：接骨丹
组成：自然铜2两（火烧，醋蘸7次），白芍药半两，血竭3钱半，甜瓜子半两（麦麸炒黄），木鳖子5个或3个（去皮油甚净），水蛭2钱或3钱（石灰炒），胡桃仁5钱（去皮油、净），当归5钱，半两钱3两个（烧红，醋蘸3-4次），血余灰3钱，没药半两，乳香半两。
出处：《医方类聚》卷一八七引《经验秘方》。
主治：损折。
加减：None
功效：None
用法用量：每服1铜钱，好酒调下，唯可食1服。如左边损折，用纸炒黑豆1颗，嚼试之，如能嚼破者，则髓不断，不能者，髓必断矣，然服之神效。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
