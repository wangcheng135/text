序号：34622
名称：驴尿一物饮
组成：驴尿。
出处：方出《外台》卷八引《救急方》，名见《医方考》卷三。
主治：胃反。
加减：None
功效：None
用法用量：每服2合。
制备方法：None
临床应用：翻胃：昔在幼年，经患此疾，每服食饼及羹粥等物，须臾吐出。正观中，许奉御兄弟及柴蒋等，时称名医，奉勅令疗，竭其术竟不能疗，渐至羸惫，死在旦夕。忽有一卫士云：服驴子小便极验，旦服二合，午食惟吐一半，晡时又服二合，人定时食粥吐即定。后奏知大内中，五六人患翻胃，同服，一时俱愈。
各家论述：《医方考》：火郁于中，治以辛香开胃之药。益滋其燥，非所宜也。驴尿辛膻，可使开郁，然为浊阴之所降，则可以济火矣。
用药禁忌：None
