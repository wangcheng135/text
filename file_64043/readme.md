序号：64059
名称：猪牙皂荚丸
组成：鳖甲1两（涂酥或醋炙令黄，去裙襕），猪牙皂荚半两（去黑皮，涂酥炙令焦黄，去子），桃仁半两（汤浸，去皮尖双仁，麸炒微黄），郁李仁半两（汤浸，去皮尖，微炒），天灵盖1两（涂酥炙令黄），甜葶苈1分（隔纸炒令黄或紫色），虎头骨半两（涂酥炙令黄），干青蒿半两。
出处：方出《圣惠》卷三十一，名见《普济方》卷二二九。
主治：热劳。或咳嗽气喘，两胁胀，不思饮食，大便秘涩，心脏燥热．恍惚不安。
加减：None
功效：None
用法用量：每服20丸，以麦门冬汤送下，不拘时候。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌苋菜。
附注：None
