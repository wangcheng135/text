序号：27714
名称：补肝丸
组成：杏仁1两，茺蔚子1两，青葙子1两，枸杞子1两，五味子1两，茯苓1两（去皮），干地黄3两，菟丝子2两，决明子1两，山芋1两半，车前子1两半，地骨皮（焙）1两半，柏子仁1两半，大黄1两半，细辛（去苗叶）1两半，甘草（炙，锉）1两半，人参1两半，黄芩（去黑心）1两半，黄连（去须）1两半，防风（去叉）1两半。
出处：《圣济总录》卷一一二。
主治：眼昏暗，将变成内障。
加减：None
功效：None
用法用量：每服20丸，加至30丸，食后米饮送下，临卧再服。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
