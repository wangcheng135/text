序号：25290
名称：百子附归丸
组成：真阿胶（蛤粉炒成珠）2两，蕲艾叶（去筋梗，醋煮干）2两，当归（肥大者，酒洗，去芦）2两，川芎（去芦）2两，怀庆熟地黄（去脑，取沉水者）2两，白芍药（肥长者）2两，香附（赤心者，去毛，杵成米，水、醋各淹1宿，晒，炼干）12两。
出处：《摄生众妙方》卷十一。
主治：阴阳不利，气血不足，不孕，胎前产后，月事参差，有余不足诸证。
加减：None
功效：调经养血，安胎顺气，种子。
用法用量：每服100丸，空心陈米醋点沸汤送下，日1次。
制备方法：上为极细末，用大陈石榴1枚，连皮捣碎，东流水3升熬去滓，打面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：滋血暖宫丸（《医统》卷八十四）、百子建中丸（《回春》卷六）、百子归附丸（《济阴纲目》卷一）。
