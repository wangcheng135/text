序号：60137
名称：透经丸
组成：天麻（酒浸1宿，焙干）半两，白附子（炮）半两，牛膝（酒浸1宿）半两，木鳖子（去壳，别研）半两，当归（酒浸1宿，焙干）半两，羌活（去芦头）半两，地龙（去土）1分（微炒），乳香2钱（别研），朱砂1钱（别研），没药2钱（别研）。
出处：《杨氏家藏方》卷十七。
主治：小儿筋脉拘挛，不得舒畅，手足软弱，虚羸无力，惊搐之后，偏废不举，及行步迟晚者。
加减：None
功效：透经络，活血脉。
用法用量：每服1丸，煎薄荷汤化开，入温酒少许，乳食空同调下。
制备方法：上为细末，次入木鳖子、乳香、朱砂、没药研匀，炼蜜为丸，每1两作40丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
