序号：43041
名称：养元汤
组成：生地5钱，归身2钱，白术2钱，条芩（炒）1钱，黄耆2至4、5钱（蜜炙）。
出处：《医方简义》卷五。
主治：None
加减：始受之胎，1月足厥阴养之，加桑寄生3钱；2月，加川贝母2钱，减黄耆1半，加条芩1倍炒；3月，加川连6分，减黄耆1半；4月，去黄耆，加阿胶（冲入），地骨皮3钱；5月，加茯苓、山药各3钱；1钱；6月，加川连5分，大枣3枚；7月，加知母（炒）1钱，天冬3钱，百合3钱，炙橘红8分；8月，去白术，加川连（酒炒）5分，党参3钱；九月，加党参3钱，杞子3钱，川芎8分，白芍（炒）1钱；10月，去白术，加川芎1钱，炒白芍1钱5分，党参3钱，枸杞子3钱，炙龟版3钱。
功效：养胎元。
用法用量：妊娠每月加减而服4剂，俟其气化有序而顺产。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
