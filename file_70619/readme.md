序号：70635
名称：紫苏木通汤
组成：紫苏茎（锉，焙）1两，木通（锉，焙）1两，桑根白皮（锉，焙）1两，青橘皮（汤浸，去白，焙）1两，荆芥穗（焙）半两，独活（去芦头，锉）半两，木瓜（去瓤，切，焙）半两，怀香子根（锉，炒）半两，羌活（去芦头，锉）半两，枳壳（去瓤，麸炒）2两，大腹（锉，并皮子用）20枚。
出处：《圣济总录》卷十二。
主治：体虚受风，因气鼓作，名为风气，冷则厥逆，热则烦惋。
加减：None
功效：None
用法用量：每服5钱匕，水2盏，加生姜1分，葱2寸，同煎至1盏，去滓温服。空心、日午、临卧各1次。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
