序号：28025
名称：补虚饮
组成：黄耆（锉，炒）2两，人参1两，茯神（去木）1两，麦门冬（去心，焙）1两，桂（去粗皮）1两，陈橘皮（去白，焙）1两，当归（炙，锉）1两，天门冬（去心，焙）1两，甘草（炙，锉）1两，熟干地黄（焙）1两，五味子（炒）1两。
出处：《圣济总录》卷八十六。
主治：肺脏因吐血后，四肢虚劣，气乏无力，手脚振掉，饮食不得。
加减：None
功效：None
用法用量：每剂以水3盏，加生姜半两（切），大枣7枚（擘），同煎取1盏，去滓，空心顿服。
制备方法：上为粗末，分作10剂。
临床应用：None
各家论述：None
用药禁忌：None
附注：补肺饮（《普济方》卷二十七）。
