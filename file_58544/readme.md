序号：58559
名称：秘传乌药顺气散
组成：乌药1两，川芎1两，熟地黄（酒洗）1两，防风1两，枳壳（去瓤，麦麸炒）1两，桔梗1两，白芷1两，僵蚕（汤洗净，姜汁炒）1两，羌活1两，当归（酒洗）1两，白芍药1两，木瓜1两，槟榔1两，南木香1两，秦艽1两，川独活5钱，甘草5钱。
出处：《松崖医径》卷下。
主治：痛风。
加减：None
功效：None
用法用量：取酒吞捉虎丹，随量饮之。如饮过一半，再添酒连绢袋煮熟饮之。
制备方法：上细切，以生绢袋盛药，同无灰好酒二十五斤，入不津坛内，春、冬浸一月，秋二十日，夏十日，紧封坛口，浸满日。
临床应用：None
各家论述：None
用药禁忌：忌食猪肉。妊娠妇不宜服捉虎丹。
附注：None
