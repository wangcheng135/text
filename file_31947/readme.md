序号：31956
名称：杏仁丸
组成：杏仁5升（肥好者，以童便，于瓦瓶中浸14日，和瓶于日中，每日换小便，日满以新汲水淘洗，去皮尖，便以微火焙干，别以小便1斗。于银锅内缓火煎，候杏仁随手破，即于久经用砂盆内，柳木捶研令如膏，更以细布捩过，入真酥1两，薄荷汁2大合，和令匀，即入后药），青蒿子2两，柴胡（去苗）1两半，鳖甲（涂醋炙令黄，去裙襕）1两半，乌梅肉（微炒）1两半，地骨皮1两半，赤茯苓1两半，知母1两，虎头骨（涂醋炙令赤）1两，生干地黄1两，肉苁蓉（酒浸1宿，锉去皱皮，炙令干）1两，人参（去芦头）1两，枳壳（麸炒微黄，去瓤）3分，当归3分，白术半两，木香半两，牡蛎半两，白槟榔2两，朱砂1分（细研），豉心1合。
出处：《圣惠》卷三十一。
主治：传尸。夜梦鬼交遗精，心腹冷癖，小腹与阴中相引痛，饮食不下，日渐为瘦。
加减：None
功效：None
用法用量：每日服15丸，渐加至30丸，空腹以温酒送下，服后觉似热即减，热定还添，以意斟酌；服经1月，诸候皆退，能食，夜卧安畅，面有色，即是药力已行，当劝服勿怠。
制备方法：上为末，以杏仁煎和，捣为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌人苋、冷水、白粥、生血、雀肉、桃李。
附注：None
