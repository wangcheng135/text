序号：5190
名称：加减六味地黄丸
组成：茯苓1两5钱，熟地4两，泽泻8钱，炙甘草5钱，枸杞1两5钱（盐水炒），萸肉1两5钱，青皮5钱（盐水炒），半夏8钱，粉丹皮8钱，煅龙骨1两，煅牡蛎1两，杜仲1两（炒黑），白芥子1两。
出处：《疬科全书》。
主治：寒痰凝结所致的阴火疠，颈际夹起，大如卵形，坚硬异常，或一边，或两边，或带小核数粒，体质羸弱或后天亏损者。
加减：若唇舌常白，面色萎黄，脉沉迟无力，须兼用附、桂。
功效：温补肝肾固脾。
用法用量：每服3钱，早、晚饭后淡盐汤送下。加减作汤剂亦可。
制备方法：上为细末，炼蜜为丸，如绿豆大，切勿火焙。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
