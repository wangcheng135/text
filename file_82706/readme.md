序号：82722
名称：中丹
组成：黄耆（白水者，半禀阴也；陇西者，半禀阳也）4两，白芍药4两，当归4两，黑附子（炮，去皮脐，大者佳）1两，黄芩1两（与黑附子同末，生姜汁和），蜀椒（去子）1两（出汗），白茯苓2两，人参2两，桂（去皮，辛者）2两。
出处：《元和纪用经》。
主治：百损久虚，体劣羸瘦不堪，荣卫不足，善惊昏愦，上焦客热，胃膈冷痰引饮，过食则心腹痞满，脾胃气衰不能消谷，血妄时崩，四肢沉困。
加减：None
功效：长肌肉，泽容色，实髓壮筋，不染邪疫，身轻目明耐老。
用法用量：每服20-30粒，酒送下，食前服。
制备方法：上为末，粟米粥为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方为原书“耘苗丹”之第二方。
