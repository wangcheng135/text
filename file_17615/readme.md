序号：17622
名称：人参羌活散
组成：柴胡（去苗）2两，独活（去芦）2两，羌活（去苗）2两，人参（去芦）1两，芎1两，枳壳（去瓤，麸炒）1两，茯苓（去皮）1两，甘草（炙）1两，桔梗半两，前胡半两，天麻（酒浸，炙）半两，地骨皮（去土）半两。
出处：《局方》卷十。
主治：小儿寒邪温病，时疫疮疹，头痛体痛，壮热多睡，及潮热烦渴，痰实咳嗽。初作急惊；小儿疹痘，因服热药，多发而不透，身体头面两目皆肿，连日风搐，奋身硬直。
加减：None
功效：散风邪，除风热。
用法用量：每服1钱，水7分盏，入薄荷少许，煎至5分，去滓温服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：羌活散（《幼幼新书》卷十九引《孔氏家传》）、惺惺散（《普济方》卷四○三）、人参羌活汤（《婴童百问》卷二）。
