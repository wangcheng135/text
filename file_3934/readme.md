序号：3934
名称：无比蔓荆子汤
组成：黄耆1钱，人参1钱，黄连7分，柴胡7分，蔓荆子5分，当归5分，葛根5分，防风5分，生草1钱，细辛叶3分。
出处：《原机启微》卷下。
主治：眼棱紧急，以致倒睫拳毛，损睛生留。及上下睑眦赤烂，羞涩难开，眵泪稠粘。
加减：None
功效：None
用法用量：作1服，水2盏，煎至1盏，去滓稍热服。
制备方法：None
临床应用：None
各家论述：肺气虚，黄耆、人参实之，为君；心受邪，黄连除之，肝受邪，柴胡除之，小肠受邪，蔓荆子除之，为臣；当归和血，葛根解除为佐；防风疗风散滞，生甘草大泻热火，细辛利九窍，用叶者，取其升上之意为使也。
用药禁忌：None
附注：None
