序号：32304
名称：沉香汤
组成：沉香（锉）半两，人参半两，木香（锉）3分，厚朴（去粗皮，擦生姜汁炙）3分，干蓝3分，升麻1两，玄参1两，知母（焙）1两，地榆（锉）1两，甘草（炙，锉）1两，钩藤皮1两1分，凝水石1两半（捣，研）。
出处：《圣济总录》卷一七一。
主治：小儿风痫，发作无度，颈核瘰疬，瘴毒乍寒乍热，欲成骨蒸，肠滑骨热，多变疳痢，皮肉干枯，不思乳食，身热生疮，喉闭肿痛。
加减：None
功效：None
用法用量：1-2岁儿以水半盏，药末半钱匕，煎至3分，去滓，不限早、晚，每日3次。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
