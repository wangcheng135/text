序号：23671
名称：护心散
组成：真绿豆粉1两，乳香半两。
出处：《本草纲目》卷二十四引《李嗣立外科方》。
主治：疽疾初期，及毒气冲心，呕逆者。
加减：None
功效：出毒气，预防毒气内攻。
用法用量：每服1钱，以生甘草浓煎汤调下，时时呷之。若毒气冲心，有呕逆之证，大宜服此。服至1两，则香彻疮孔中。凡有疽疾，1-3日之内，宜连进10余服，方免变证，使毒气外出。服之稍迟，毒气内攻，渐生呕吐，或鼻生疮菌，不食即危矣。4-5日后，亦宜间服之。
制备方法：加灯心同研和匀。
临床应用：None
各家论述：绿豆压热，下气消肿解毒；乳香消诸痈肿毒。服之一两，则香彻疮孔中。
用药禁忌：None
附注：None
