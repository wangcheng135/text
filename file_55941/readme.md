序号：55956
名称：桑枝酸枣仁煎
组成：酸枣仁3两（一半炒令香熟，一半生用），羚羊角屑1两，海桐皮2两（锉），羌活2两，仙灵脾1两，赤箭1两，萆薢1两（锉），杜仲1两（去粗皮，炙令微黄，锉），虎胫骨1两半（涂酥炙令黄），防风1两（去芦头），石斛1两半（去根，锉），牛膝1两（去苗），巴戟1两，附子1两（炮裂，去皮脐），木香1两，生干地黄1两，蜜4两，真酥1两，桑枝1握（长1尺，锉）。
出处：《圣惠》卷二十六。
主治：筋极。身体拘急，四肢疼痛，行立不得。
加减：None
功效：None
用法用量：每服1茶匙，食前以温酒调下。
制备方法：上除酥蜜桑枝外，共为散，用清酒7升，先煎桑枝，令色微黄，去桑枝后下药末，更煎10-20沸，次下酥蜜，煎成膏，看稀稠得所，以瓷合盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
