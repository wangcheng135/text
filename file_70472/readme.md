序号：70488
名称：紫金挺
组成：当归1两，续断1两，骨碎补1两，桂（去粗皮）1两，附子1两，泽兰1两，芍药1两，白及1两，牛膝1两，羌活1两，芎1两，木香1两，麒麟竭1两，生干地黄1两，白僵蚕1两，白附子1两，沉香半两，丁香半两，栝楼2枚（大者），乌蛇肉1两，白蔹1两，白芷1两，玄参1两（23味都一处捣筛），杏仁（去皮，细研）3分，桃仁（去皮，细研）3分。
出处：《圣济总录》卷一三○。
主治：疮肿疼痛。
加减：None
功效：辟风敛疮。
用法用量：每使时看疮大小用之。
制备方法：上药入麻油4斤，猪脂1斤半，野驼脂3两，用文武火锅内煎黑，去滓，再入乳香末3两、松脂6两，更煎烊后，滤去滓，细罗铅丹3斤，别炒令紫色，去火，滴水内成珠，即倾入瓷器内盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
