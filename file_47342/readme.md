序号：47357
名称：祛风牵正汤
组成：当归9g，川芎9g，赤芍15g，羌活9g，荆芥9g，白附子12g，全虫9g，僵蚕9g，白芷9g，薄荷9g，菊花9g，甘草6g。
出处：《常见病辨证治疗》。
主治：面神经麻痹发病初起。
加减：若发病日久，或气血较虚而身体弱者，加黄耆30g。
功效：祛风通络，益气活血。
用法用量：None
制备方法：None
临床应用：None
各家论述：方中当归、川芎、赤芍活血；白附子、荆芥，性味辛散，善治头面之风；薄荷、菊花轻而走上，疏风清热；全虫、僵蚕、羌活、白芷通经活络，祛风消肿；甘草益气清热，调和诸药，共奏祛风通络，益气活血之功效。
用药禁忌：None
附注：None
