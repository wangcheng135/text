序号：3995
名称：无敌大将军
组成：桑柴灰（将柴另烧，取其炭火，置1大缸内，待其自化成白灰，取1斗，绵纸衬入淘箩内，清滚水淋下汁，瓷缸盛贮，淋至汁味不苦涩咸则止，将汁入瓷碗中，重汤燖浓如稀糊为度），茄杆灰（淋制如前法）1斗，矿灰（即石灰，须柴烧者佳，淋汁如前法）1斗，3味熬调和匀，名三仙膏，亦可点痈疽之稍轻者，再和碱水熬育1两，加入后开细药，则成全方。每三仙膏5两，配入：蟾酥3钱5分（酒化令匀），梅花冰片2钱，真正牛黄1钱，珍珠2钱，（3味俱研如飞面），透明雄黄2钱，明矾3钱，朱砂1钱5分，白硼砂2钱（4味另研如飞面方妙），真麝香（须用当门子，即麝香最上乘者，碾匀）1钱，铜青1钱5分，硇砂2分5厘，火消3钱，轻粉2钱，乳香2钱（打碎，人乳浸烂，研匀），制没药1钱5分。
出处：《广笔记》卷三。
主治：痈疽、对口、疔疮、发背，一切无名恶肿毒。
加减：None
功效：None
用法用量：每遇毒，取少许涂其顶，干则以米醋和蜜少许润之，其毒黑血或毒水暴出，即时松解。或用荞麦面调。若系疔疮，加铁锈黄1分，研如面和入。多涂其顶，信宿其根烂出。内服紫金锭1锭，须内府者方效。若系痈疽等症，别服蜡矾丸及托里解毒之剂。
制备方法：上各为细末，和匀，再碾数千下，将前膏加入，搅得极匀，入瓷罐内，罐须小口者妙，以乌金纸塞口，封以好黄蜡，勿令一毫气走。
临床应用：None
各家论述：None
用药禁忌：忌着好肉上。
附注：无敌丹（《外科方外奇方》卷二）。
