序号：28646
名称：补天丹
组成：驴肾2两，制耆5两，柏仁1两半，杜仲3两，白术5两，川附子1两半，萸肉2两，五味子1两半，白参3两，白芍3两，云苓2两半，龙骨2两，故纸3两，菟丝子3两，杞子4两，砂仁6钱，巴戟4两半，熟地4两，当归3两，覆盆子1两半，鹿胶3两。
出处：《全国中药成药处方集》（抚顺方）。
主治：生殖器衰弱，肾虚滑精，阳痿不举，见色早泄，精液清冷，及气血衰弱，瘦弱难支，食少便溏，气息微弱，动则作喘，腰痠腿软，健忘怔忡，自汗晕眩，寐而不实。
加减：None
功效：添精壮阳，补气生血，强壮。
用法用量：每服2钱，早、晚食前各服1次，白水或淡盐汤送下。
制备方法：上为细末，炼蜜为丸，重2钱。
临床应用：None
各家论述：None
用药禁忌：火盛者勿服。
附注：None
