序号：51817
名称：钟乳散
组成：钟乳6两（无问粗细，以白净无赤黄黑者为上，铜铛中可盛3两斗，并取粟粗糠2合许，纳铛中煮5-6沸，乃纳乳煮水，欲减添之如故，一晬时出，以暖水净淘之，晒干，玉锤研，不作声止，重密绢水下，澄取之用），铁精1两，鹿角1两（白者），蛇床子3两，人参1两，磁石1两，桂心1两，僵蚕1两，白马茎（别研）1两，硫黄（别研）1两，石斛1两。
出处：《千金》卷十九。
主治：五劳七伤，虚羸无气力，伤极。
加减：None
功效：None
用法用量：每服30丸，酒送下，日2次。
制备方法：上为末，以枣膏为丸，如语桐子丸。
临床应用：None
各家论述：None
用药禁忌：忌房事及生冷、酢滑、鸡、猪、鱼、陈败之物。
附注：本方方名，据剂型，当作“钟乳丸”。
