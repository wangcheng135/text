序号：74735
名称：解毒通脉汤
组成：桃仁3钱，大黄2钱，水蛭2钱，虻虫2钱，银藤1两，生石膏8钱，丹皮2钱，连翘5钱，栀子3钱，黄芩3钱，延胡索2钱，赤芍2钱。
出处：《刘奉五妇科经验》。
主治：寒湿阻络，恶露不下，毒邪逆串经脉，气血壅滞，堵塞血脉，郁久化热所致的产后急性血栓性静脉炎。发热肢肿，疼痛难忍。
加减：None
功效：活血化瘀，清热解毒，通脉止痛。
用法用量：None
制备方法：None
临床应用：None
各家论述：方中水蛭、虻虫入血分，化瘀血，蚀死血；桃仁活血化瘀；大黄入血分，化瘀血，清解血分毒热；赤芍、丹皮清热凉血，活血破血；石膏、连翘、栀子、黄芩清热解毒而散结；金银藤清热解毒，通血脉活络；延胡索行气活血止痛。此方边清边通，使湿毒热邪得以清解，瘀血死血得以活散。
用药禁忌：None
附注：None
