序号：49433
名称：胡黄连丸
组成：胡黄连半两，人参（去芦头）半两，地龙（微炒）半两，代赭（细研）半两，赤石脂半两，蜗牛肉2-7个，大蜣螂5个（去翅足，微炒），猪牙皂荚2挺（去黑皮，涂酥炙焦黄，去子），青黛（研入）1分，木香1分，蟾酥（研入）1分，黄连（去须）1分，槟榔1分，朱砂（细研）1分，麝香（细研）1分，天麻1分，当归（锉，微炒）1分，犀角屑1分，干蝎（微炒）1分，蝉壳（微炒）1分，芦荟（细研）1分，羌活1分，使君子1分，白芜荑1分，驴胎耳（炙令焦黄）1分，牛黄（细研）1分，蛤蚧（头尾全者，涂酥炙微黄）1分。
出处：《圣惠》卷八十六。
主治：小儿风疳，剜鼻揉眼，不知痒处。
加减：None
功效：None
用法用量：每服3丸，空心以粥饮下。
制备方法：上为末，入研了药令匀，以豮猪胆汁和丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
