序号：3902
名称：方脉流气饮
组成：紫苏1钱，青皮（去白）1钱，当归（酒拌）1钱，芍药（炒）1钱，乌药1钱，茯苓1钱，桔梗（炒）1钱，半夏（姜制）1钱，川芎1钱，黄耆（炙）1钱，枳实（麸炒）1钱，防风1钱，陈皮（去白）1钱，甘草（炙）1钱，木香5分，大腹皮5分，槟榔5分，枳壳（麸炒）5分。
出处：《外科发挥》卷五。
主治：瘰疬，流注，及郁结聚结肿块，或走注疼痛，或心胸痞闷，咽塞不利，胁腹膨胀，呕吐不食，上气喘急，咳嗽痰盛，面目或四肢浮肿，大小便秘。
加减：None
功效：None
用法用量：水2钟，加生姜3片，大枣1个，煎8分，食远服。
制备方法：None
临床应用：1.瘰病：一男子因暴怒，项下肿痛结核，滞闷兼发热，用方脉流气饮二剂，胸膈利；以荆防败毒散，一剂而热退；后服它药而瘳。流注：一妇人暴怒，腰肿一块，胸膈不利，时或气走作痛，与方脉流气饮，数剂而止。更以小柴胡汤对四物，加香附、贝母，月余而愈。
