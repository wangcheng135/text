序号：58561
名称：秘传五疳散
组成：白术（蜜水炒）1两5钱，白茯苓（去皮）7钱5分，甘草1钱5分，麦门冬（去心）1两，使君子肉（切碎，略炒）7钱5分，山楂肉3钱，麦芽（炒）5钱，金樱子肉（略炒）5钱，芡实2钱5分，莲肉心（隔纸炒）5钱，青皮（去瓢，面炒）2钱，橘红5钱。
出处：《摄生众妙方》卷十。
主治：小儿五疳潮热，面黄肌瘦，烦渴吐泻，肚大青筋，手足如柴，精神悸倦。
加减：身热咳嗽，加地骨皮、百部各5分；肚腹饱胀，大便稀水，肠鸣作清或虫出不和，加槟榔2钱5分，木香1钱；禀受气弱，加人参2钱5分。
功效：None
用法用量：每日中晌、晚间各服1-2茶匙，温水漱口。途中无蜜，滚白水调服亦可。
制备方法：上为极细末，和匀，重7两，每次用药末1两，用炼蜜半斤或4两，调成膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
