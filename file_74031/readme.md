序号：74047
名称：暖宫丸
组成：香附6两，艾炭3两，当归3两，黄耆3两，吴萸3钱，白芍2两，川芎2两，川断1两半，熟地1两，贡桂5钱。
出处：《全国中药成药处方集》（哈尔滨方）。
主治：子宫寒冷，经血衍期，腹痛结块，腰腿疼痛，久不生育；肝郁气滞，气结胸脘，胸脘胀痛，纳少嗳气；积湿浸滞，带脉不宣，湿浊下注，带下白滑，腰痠腹痛，面苍体软；痛经气滞，白带。
加减：None
功效：养血散寒，理气化湿。
用法用量：每服1丸，经血寒者，红糖水为引，其它均白水送下，日服2、3次。
制备方法：上为细末，炼蜜为丸，每丸重2钱。
临床应用：None
各家论述：None
用药禁忌：忌食腥冷，孕妇勿服。
附注：None
