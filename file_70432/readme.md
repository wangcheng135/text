序号：70448
名称：紫朴汤
组成：紫朴、山楂、卜子、三棱、蓬术、枳实、连翘、青皮、陈皮、细辛各等分。
出处：《痧胀玉衡》卷下。
主治：痧有食气壅盛者。
加减：None
功效：宽中下气，消积化食。
用法用量：水煎，冷服。
制备方法：None
临床应用：None
各家论述：《慈禧光绪医方选议》：此方为治时令病方，宽中下气，消食化积。主药紫朴，即厚朴用紫苏叶、生姜煮汁浸泡炮炙而成，功能燥湿散满以运脾，行气导滞而除胀，温燥行气，芳香化湿。据现代药理研究，其煎剂对伤寒杆菌、霍乱弧菌、葡萄球菌、痢疾杆菌、绿脓杆菌均有抗菌作用，故可用于多种疾病。用上法炮制后，既可减轻厚朴刺激咽喉的副作用，又可加强其行气和胃的功效。
用药禁忌：None
附注：石八（《痧症全书》卷下）、十六号师象方（《杂病源流犀烛》卷二十一）。本方改为丸剂，名“紫朴丸”（见《慈禧光绪医方选议》）。
