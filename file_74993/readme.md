序号：75009
名称：颖曲氏回春膏
组成：当归3两，木鳖子3两，栀子2两5钱，牙皂2两5钱，白及2两5钱，川乌2两5钱，草乌2两5钱，乌药2两5钱，白蔹2两5钱，连翘2两5钱，苦参4两，槐枝4两，西大黄1两5钱，乳香（去油）1两5钱，没药（去油）1两5钱，血竭1两5钱，儿茶1两5钱，明雄1两5钱，樟脑1两5钱，麝香1钱，葱白1斤，生姜2斤，香油7斤，广丹3斤8两。
出处：《全国中药成药处方集》（禹县方）。
主治：疔毒恶疮，无名肿毒，跌打损伤，冻疮臁疮，牙疼痄腮，脚气狗咬，搭背对口，鼠疮瘰疬，附骨阴疽，手足麻木，胃口腹疼，鹤膝风症，气痞寒积，乳岩乳核，臂腰腿疼。
加减：None
功效：None
用法用量：按患处大小贴用，温开水暖软，夏天用水捏成薄片，摊在布上，贴到患处。
制备方法：以香油将药煎枯去滓，加丹熬成膏，下乳香、没药、血竭、儿茶、明雄、麝香、樟脑即成。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
