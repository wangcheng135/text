序号：25264
名称：如意丸
组成：枳壳（去瓤）2两，槟榔2两，橘红2两，半夏（汤炮7次）2两，蓬术2两，京三棱2两，干姜（炮）2两，黄连（去须）2两，巴豆3-7粒（连壳用）。
出处：《济生》卷四。
主治：中虚积冷，气弱有伤，不能传化，心中坚痞，两胁胀满，心腹疼痛，噫宿腐气；及霍乱吐泻，米谷不消；久痢赤白，脓血相杂，久病黄色羸瘦；及腹中一切食癥之疾。
加减：None
功效：None
用法用量：每服10丸，加至15丸，食后临卧清茶、姜汤任下。
制备方法：上除巴豆外，锉如豆大，用好醋合巴豆煮干，去巴豆，余药焙为细末，薄糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：孕妇不宜服。
附注：None
