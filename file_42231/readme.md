序号：42246
名称：青娥丸
组成：破故纸（隔纸炒香）4两，萆薢（真正者，4两分作4份，一份盐水，一份童便，一份米泔，一份酒，各浸1宿，晒干）2两，山茱萸（去核）2两，菟丝子（酒浸，蒸熟）1两，肉苁蓉（去角心，酒浸，炙）1两，牛膝（酒浸1宿）1两，枸杞子1两，骨碎补（蒸）1两，杜仲（酒浸，炒去丝）1两，五味子1两，虎胫骨（酥炙）1两，黄耆（蜜炙）1两，山药1两，陈皮（去白）1两，白茯苓1两，人参1两（黑瘦者，减半）。
出处：《济阳纲目》卷六十四。
主治：腰膝足痛。
加减：None
功效：滋肾水，壮元阳，益筋骨。
用法用量：每服80丸或100丸，盐汤、酒任下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
