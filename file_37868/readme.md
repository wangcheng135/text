序号：37878
名称：拨云丹
组成：珍珠子8钱（要未经穿眼用过者，敲碎，研细，水飞净），廉珠1钱（豆腐内熝1次，研极细无声为度），炉甘石1两（煅，水飞净），真西黄1钱（研极细，另入），片脑5分（研细，另入），辰砂3钱（研极细，水飞），麝香5分（研细，另入），硼砂1钱（研极细，另入），真熊胆1钱（另入），水晶3钱（敲碎，研极细，水飞），其血珀3钱（水飞），灵玛璃3钱（敲碎，研极细，水飞），玄明粉2钱（另入），生石蟹5钱（研极细，水飞）。
出处：《经验秘方类钞》卷上。
主治：眼中膜障，胬肉，翳星。
加减：None
功效：None
用法用量：用时将药末少许，以冷滚水调和点之，每日点2次。
制备方法：上药俱不可见火，凡研总以极细无声为度，其另入者，研细不必水飞，连前之水飞过者，各药一总倾入瓷钵内，共研和匀，再用极细绢筛筛过，盛瓷瓶内。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
