序号：57478
名称：润肌皮肤膏
组成：大风子仁16两，大麻子仁16两，潮脑16两，核桃仁16两，京红粉16两。
出处：《北京市中药成方选集》。
主治：酒渣鼻，蝴蝶脸，面部粉刺，白癜风，土癍，汗癍，脚气，骑马癣，风癣，钱癣。
加减：None
功效：消癍，祛湿，润皮肤。
用法用量：用细布包药，每瓶分3-4次，擦患处，每日2-3次，将药液擦入毛孔内，至皮肤觉痛时即药力发挥作用。擦药3-5日，如肉皮不疼，去布使药擦，如肉皮疼，即停止用药，过一星期再用。
制备方法：将大风子仁、大麻子仁、核桃仁研成细泥，红粉另研；用腊7两，香油13两5钱，松香1两。先用松香熬油，蜡溶化后，开起化净，下入大风子仁、潮脑等；其后京红粉冷后再下，混合搅匀，装瓶，重1两。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
