序号：43192
名称：三棱散
组成：人参（去芦）7钱半，三棱（炮，锉）1两半，净香附1两半，青皮（去白）半两，益智仁半两，陈皮（去白）半两，半夏（汤煮透，滤，仍锉，焙干）半两，枳壳（水浸润，去壳，锉片，麦麸炒微黄）半两，神曲（炒）半两，谷芽（洗，焙）半两，莪术（醋煮透，滤干，锉，焙）半两，大黄（半生半炮）半两，紫苏（去老梗）半两，甘草（半生半炙）1两2钱。
出处：《活幼心书》卷下。
主治：诸般停滞，疳积发热，泻痢酸馊，水谷不化。
加减：气虚劳，加白茯苓1两。
功效：和脾胃，进饮食，长肌肉，益神气。
用法用量：上(口父)咀。每服2钱，水1盏，加生姜2片，仓米100粒，煎7分，温服，不拘时候。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：方中谷芽，《奇效良方》作“麦芽”。
