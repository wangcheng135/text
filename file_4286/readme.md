序号：4286
名称：木香丸
组成：木香半两，茴香子（炒）半两，缩砂蜜（去皮）半两，青橘皮（汤浸，去白，焙）半两，蓬莪术（煨，捶碎）半两，红豆半两，高良姜半两，楝实（锉，炒）半两，丁香皮（锉）半两，陈橘皮（汤浸，去白，焙）半两，桔梗（炒）3两，芍药2两，益智（去皮）1两半，五味子1两，桂（去粗皮）1两，甘草（炙，锉）3分，胡椒半两。
出处：《圣济总录》卷四十六。
主治：脾胃气虚，不思饮食。
加减：None
功效：None
用法用量：每服15丸，加至30丸，生姜汤送下，如伤水，多嚼服亦得。
制备方法：上为末，用粟米饭为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
