序号：55558
名称：桃仁散
组成：桃仁3分（汤浸，去皮尖双仁，麸炒微黄），吴茱萸半两（汤浸7遍，焙干，微妙），木香半两，京三棱3分（炮，锉），芎藭半两，桂心半两，白术3分，青橘皮半两（汤浸，去白瓤，焙），柴胡1两（去苗），诃黎勒3分（煨，用皮），高良姜2分（锉），当归半两，槟榔半两，赤芍药半两，甘草半两（炙微赤，锉）。
出处：《圣惠》卷二十八。
主治：气劳羸瘦，膈胁痞坚，脐下冷疼，不欲饮食。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，大枣3枚，煎至6分，去滓稍热服，不拘时候。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
