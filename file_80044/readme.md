序号：80060
名称：摩风膏
组成：羌活半两，防风3分（去芦头），芎1两，踯躅花半两，甘菊花半两，附子1分（去皮脐），桂心3分，汉椒1两半（去目），川乌头1分（去皮脐），当归半两，皂荚1分（去皮子），鲮鲤甲3分，甘草1两，白及1分，栝楼根1分，紫葛1分，乌蛇半两，猬皮1分，莽草半两，细辛半两，杏仁1分（汤浸，去皮尖双仁），苦参1两，白蔹半两，蜡5两，露蜂房1分，猪脂3斤（切）。
出处：《圣惠》卷六十七。
主治：筋骨俱伤后，夹风疼痛。
加减：None
功效：None
用法用量：如有伤折筋骨者，取此膏摩之。
制备方法：上锉细，以米醋2升，拌匀，经2宿后，以火微微炒之令干，用猪脂和药，以慢火煎1日，以绵滤于瓷盒内盛，不令水污着。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
