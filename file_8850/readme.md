序号：8850
名称：水眼药
组成：羊脑炉甘石1两（童便浸，春5、夏3、秋7、冬10日，取出打碎，放新瓦上，火煅2次，漂净，焙干），山东黄丹1两（水飞过，焙干），辰朱砂4钱（研细，水飞），真麝香3分（研），真乳香4钱（熨，去油），真没药4钱（熨，去油），白硼砂2钱（研极细，水飞过），海螵蛸1两（去衣，研细，水浸，漂净，焙干），破大珍珠5分（入豆腐内煮过，研细，水飞）。
出处：《寿世新编》。
主治：诸般眼疾。
加减：None
功效：None
用法用量：点眼。
制备方法：上为极细末，研至无声为度，用白蜜8两，炼三次，绢滤三次，将药内调成膏子，用瓷罐盛，熔蜡封口，愈陈愈佳。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
