序号：3187
名称：天雄丸
组成：天雄1两（炮裂，去皮脐），独活3分，杜仲1两半（去皴皮，炙微黄，锉），附子1两（炮裂，去皮脐），牛膝1两半（去苗），干漆3分（捣碎，炒令烟出），桂心1两，没药3分，巴戟1分，鹿茸1两（去毛，涂酥，炙微黄），蝉壳1两（酒浸，晒干），虎胫骨3分（酒浸，炙微黄），萆薢1两（锉），乳香3分，蛜（虫祁）3分（微炒），天麻1两，白花蛇1两（酒浸，去皮骨，炙微黄），狗脊3分，川乌头3分（炮裂，去皮脐），当归3分（锉，微炒），芎3分，地龙1两（微炒），朱砂3分（细研，水飞过），败龟1两（涂醋，炙令黄），麝香半两（细研）。
出处：《圣惠》卷四十四。
主治：肾脏气衰虚腰痛，或当风湿冷所中，腿膝冷痹缓弱。
加减：None
功效：None
用法用量：每服30丸，食前以温酒送下。
制备方法：上为末，入研了药令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
