序号：32514
名称：应痛丸
组成：生苍术1斤，破故纸1斤（半炒半生），舶上茴香12两（炒），骨碎补1斤（去毛），穿山甲（去膜去皮，炒胀为度，柴灰亦可）、生草乌1斤（锉如麦大）。
出处：《医方大成》卷八引《经验方》。
主治：风邪入经，足三阴受湿，脚气挛急。寒湿风损，手脚疼痛。
加减：None
功效：None
用法用量：每服50丸，酒、汤任下。
制备方法：将草乌1斤，用生葱2斤，连皮生姜2斤，擂烂，将草乌一处淹2宿，焙干，连前药一处焙为未，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌热物。
附注：按：①本方方名，《普济方》引作“应病丸”。②《永类钤方》引本方，穿山甲用六两，桑灰炒胀为度，柴灰亦可。
