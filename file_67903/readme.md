序号：67919
名称：黄耆丸
组成：黄耆（锉）3两，肉苁蓉1两半（酒浸，去皱皮，切，焙），鹿茸（酥炙，去毛）1两半，菟丝子（酒浸1宿，别捣）3分，石斛1两，巴戟天（去心）1两，山芋1两半，远志3分（去心），柏子仁（炒，别研）3分，白茯苓（去黑皮）1两，泽泻3分，山茱萸1两，熟干地黄（洗去土，切，焙）6两，续断1两，桂（去粗皮）3分。
出处：《圣济总录》卷九十二。
主治：虚劳。肾气冷弱，小便余沥。
加减：None
功效：None
用法用量：每服30丸，空心煎枣汤送下，午食前再服。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
