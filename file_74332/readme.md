序号：74348
名称：填骨万金煎
组成：生地黄30斤（取汁），甘草1斤，阿胶1斤，肉苁蓉1斤，桑根白皮（切）8两，麦门冬2斤，干地黄2斤，石斛1斤5两，牛髓3斤，白蜜10斤，清酒4斗，麻子仁3升，大枣150枚，当归14两，干漆20两，蜀椒4两，桔梗5两，五味子5两，附子5两，干姜8两，茯苓8两，桂心8两，人参5两。
出处：《千金》卷十二。
主治：内劳少气，寒疝里急，腹中喘逆，腰脊痛。
加减：None
功效：None
用法用量：饮吞如弹丸1枚，1日3次。若夏月暑热，煮前转味，可以蜜、地黄汁和诸药成末为丸，如梧桐子大，每服15丸，不知，稍加至30丸。
制备方法：上药先以清酒2斗6升，纳桑根白皮，麻子仁、枣胶，为刻识之，又加酒1斗4升煮，取至刻，绞去滓，纳蜜、髓、地黄汁，汤上铜器煎，纳诸药末，半日许使可丸止，大瓮盛。
临床应用：None
各家论述：《千金方衍义》：此方专主填补骨髓，而于天门冬大煎方中采取温补药味，添入椒、附之辛烈，故可兼治寒疝里急等疾。
用药禁忌：None
附注：None
