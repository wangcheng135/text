序号：80463
名称：蟹爪饮
组成：蟹爪1升，甘草2尺，阿胶3两。
出处：方出《千金》卷二，名见《圣济总录》卷一五九。
主治：动胎及产难；子死腹中；并妊两儿，一死一生，令死者出，生者安。
加减：None
功效：None
用法用量：以东流水1斗，先煮二物得3升，去滓，纳胶冷烊，顿服之；不能，分再服，若人困，拗口纳药，药入即活。煎药作东向灶，用苇薪煮之。
制备方法：None
临床应用：None
各家论述：1.《千金方衍义》：此即前下血方中蟹爪汤于中除去桂心，加用甘草、阿胶也。夫桂心温散之品，子死腹中正宜加用，而反去之，必本妇去血过多，恐复走血，而不便用，非此证之不当用桂也。此用阿胶专补荣血，甘草专培胃气。_x000D_
2.《重庆堂医学随笔》：蟹爪尖专下死胎，汀草奠安中气，不使尸气上乘，阿胶滑利前阴。若双胎一死一生者，蟹爪又安生胎，阿胶专于育神，甘草培植生气，服之令死者出，生者安。
用药禁忌：None
附注：千金神造汤（《本草纲目》卷四十五）。
