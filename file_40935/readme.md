序号：40950
名称：三香神术丸
组成：香附子1两，苍术1两，厚朴1两，藿香1两，甘草1两，良姜1两，枳实1两，枳壳1两，青皮1两，陈皮1两，广术1两，三棱1两，槟榔1两，神曲1两，半夏曲1两，益智仁1两，瓜三棱1两，石三棱1两，曲糵1两，雷丸1两，干葛1两，黄连1两，木香1两。
出处：《普济方》卷二十四引《德生堂方》。
主治：中酒吐酒，呕逆吐酸，气壅食噎，饮食迟化，胸膈痞闷，腰胁刺痛，胃脘停痰，饮食无味，妇人血气，脐腹疼痛。
加减：None
功效：温中快膈，化积顺气。
用法用量：每服50丸，酒、水任意下。
制备方法：上为细末，萝卜熬水打曲糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
