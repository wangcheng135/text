序号：32176
名称：抓癖膏
组成：香油半斤，桐油半斤，生猪脑子半斤，男子血余（灰水洗净）不拘多少，桃仁4两，白蜡4钱（上俱下锅内，文武火熬的脑子尽，用布绢滤去滓，次下飞过黄丹14两，熬成膏，待温），胡黄连3钱，香白芷3钱，苏木3钱，红花3钱，三棱3钱，莪术3钱，当归尾5钱，硇砂5钱，麝香1钱半（各为细末，照分两重罗）。
出处：《回春》卷七引李沧溪方。
主治：小儿癖疾。
加减：None
功效：None
用法用量：如有积块，先用皮消煎水洗患处令净，次用生姜擦之，方用绢帛摊药贴上；贴后，用热鞋底炙热熨之50-70遍，觉内热方可。
制备方法：将药末入前膏内搅匀收贮，勿令泄气。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
