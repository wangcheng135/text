序号：6999
名称：四制黄柏丸
组成：黄柏（去粗皮，净）4斤（1斤以好酒浸，1斤米泔水浸，1斤蜜糖水浸）。
出处：《赤水玄珠》卷一。
主治：吐血，遗精。
加减：如相火周身疼痛，减黄柏2斤，加犀角1两为末，入前丸中。
功效：滋肾降火化痰。
用法用量：每服50-70丸，空心及临卧酒吞下，徐以干物压之。
制备方法：上俱用瓷器浸之，三味俱要没二指为度，冬月浸7日，夏浸3日，春秋5日，滤出晒干，仍存余汁待后用；再将黄柏1斤切作5寸长，用真酥油半斤，以瓷碗盛之，先将铜铫将水熬滚，再将酥油连碗入水溶化，将黄柏以微火炒热，用棕刷蘸酥徐徐刷上，且刷且炙，各使透彻，切忌焦黑，炙毕放于冷地上，以瓷碗复之2日，去火毒。并前共为细末，以前存原汁和为丸；如汁不敷，再加蜜酒兑匀和之为丸，如梧桐子大。
临床应用：None

用药禁忌：None
