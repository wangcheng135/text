序号：43897
名称：厚朴建中汤
组成：厚朴1斤，生姜1斤，大枣1斤，半夏（合杵，焙）4两，甘草4两，人参1两半，陈皮2两，良姜1两，草豆蔻仁1两，白术1两，神曲1两，藿香1两。
出处：《鸡峰》卷十二。
主治：脾胃虚弱，忽中湿冷，心腹暴痛，胁肋胀满，水谷化迟，肠鸣泻痢，后重里急，脐腹冷痛，胸满气逆，呕吐恶心，手足不和，体重节痛，哕噫吞酸，不思饮食，怠惰嗜卧，四肢少力。
加减：None
功效：调适阴阳，建中补气，辟风寒湿冷非节之气，山岚瘴疟等疾气。
用法用量：每服3钱，水1盏，煎至7分，去滓，食后服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
