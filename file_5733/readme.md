序号：5733
名称：四壁櫃朱砂
组成：针砂1斤，硫黄4两，朱砂3两，白矾7两，盐1两。
出处：《圣惠》卷九十五。
主治：None
加减：None
功效：除风冷，温暖骨髓，悦泽颜色，延年益寿。
用法用量：每服5粒，空腹以温水送下。
制备方法：上以浓醋1斗5升，煮针砂、硫黄二味令干，以火煅之，待鬼焰出尽后，放冷再研。别人硫黄2两，又用醋1斗5升更煮，候干，依前煅之，鬼焰尽即止，放冷，以水淘取紫汁，去其针砂，澄紫汁极清，去其清水尽，阴干，即入白矾、盐同研，纳瓷瓶中，四面下火煅之，候瓶内沸定即止，待冷出之，细研，以醋拌为枢，先用药一半，入铅桶中筑实，即以金箔两重、朱砂入櫃上，又以余櫃盖之，筑实，以4两火养21日，即换入铜桶中，密固济，用6两火养21日足，即用10斤火煅之，住火自消，寒炉出药，朱砂已伏，于湿地薄摊，盆合一复时，出火毒了，细研，以枣肉为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：忌羊血。
附注：None
