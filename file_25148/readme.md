序号：25156
名称：朱砂丸
组成：砒霜半两，朱砂1分，雄黄1分，阿魏1分，麝香3分（以上并细研），虎头骨1分（涂酥，炙黄），蛇骨1分（酒浸，炙微黄），恒山1分（锉），猢狲头骨1分（炙黄），天灵盖1枚（烧令白色，上用朱砂点作7星，取5月1日夜作）。
出处：方出《圣惠》卷五十二，名见《普济方》卷二○○。
主治：劳疟，发歇寒热，体瘦，四肢疼痛，或时须渴。
加减：None
功效：None
用法用量：患者以1丸，男左女右，把之愈，带之亦愈，以醋汤发前服1丸亦愈，插在耳内鼻里亦愈。
制备方法：上为末，用醋煮面糊为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：服药即忌食热物。
附注：None
