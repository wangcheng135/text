序号：59663
名称：调元益胃饮子
组成：川芎炭1钱，炒当归3钱，炮姜1钱，淡附子2钱，苍术炭8分，清炙耆3钱，姜半夏1钱5分，川连（吴萸8分拌炒）1钱，琥珀6分，泽兰1钱，茺蔚子（炒）3钱，车前（炒）1钱，广木香（煨）1钱。
出处：《医方简义》卷六。
主治：产后泄泻。产后脾胃虚弱，土不胜水，木旺侮土，停滞不运，更有血瘀不净，致患泄泻。
加减：脱肛者，加炙粟壳1钱；如挟食者，加神曲3钱；如有癥瘕者，加莪术（炒）1钱，山楂肉3钱，尖槟榔2钱；如瘀血未净者，加桃仁1钱，红花8分；如心悸者，加远志肉8分，枣仁1钱（炒）。
功效：None
用法用量：水煎，和陈酒半盏，徐徐服下，每日1剂，分作5-6次服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
