序号：55185
名称：桂苓白术散
组成：木香半两，桂枝半两，藿香半两，人参半两，茯苓（去皮）半两，甘草（炙）1两，白术1两，葛根1两，泽泻1两，寒水石1两，滑石2两，石膏1两。
出处：《医学启源》卷中。
主治：冒暑、饮食所伤转甚，湿热内甚，霍乱吐泻，转筋急痛，腹满痛闷；小儿吐泻、惊风。
加减：None
功效：None
用法用量：每服3钱，白汤调下，新水或生姜汤亦得。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：桂苓甘露散（《儒门事亲》卷十二）、桂苓甘露饮（《卫生宝鉴》卷十六）、桂枝白术汤（《嵩崖尊生》卷九）、桂苓白术汤（《风劳臌膈四大证治》）。方中滑石、石膏用量原缺，据《儒门事亲》补。
