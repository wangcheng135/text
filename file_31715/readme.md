序号：31724
名称：沉香丸
组成：雄黑豆1升（小圆者，淘），附子2两（炮裂，去皮脐），吴茱萸（汤浸，焙，炒）8两，青橘皮（汤浸，去白，焙）8两，生姜4两（切碎），上药用葛布作袋盛，纳大锅中，以水1斗煮令水尽，泣干，取袋中豆晒，焙干，捣罗为末，其余4味并不用；更入后药：沉香（锉）2两，肉苁蓉（去粗皮，切，焙）2两，白附子（炮）2两，巴戟天（去心）2两，牛膝（锉，酒浸，焙）1两，海桐皮（锉，炙）1两，独活（去芦头）1两，芎1两，泽泻（锉）1两，山芋1两，生干地黄（切，焙）1两，羌活（去芦头）1两。
出处：《圣济总录》卷八十一。
主治：风毒脚气，上冲脏腑，散入四肢，虚肿无力。
加减：None
功效：None
用法用量：每服20丸，空心以温酒送下，近晚再服。
制备方法：上除不用四味外，捣罗为末，拌令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
