序号：32835
名称：羌活汤
组成：羌活（去芦头）半两，独活（去芦头）半两，干姜（炮）半两，牛膝（酒浸，切，焙）半两，草豆蔻（去皮）半两，桂（去粗皮）半两，细辛（去苗叶）1分，藿香叶（去梗）1分，吴茱萸（汤洗，焙干，炒）半两，陈橘皮（汤浸，去白，焙）半两，干蝎（去土，炒）1分，半夏（汤洗，去滑，焙）1分，甘草（炙）4钱，芎1两，白术1两。
出处：《圣济总录》卷十二。
主治：寒风所中，面青，遍身骨节俱冷，两手拘急，筋脉牵抽，手足不仁，厥冷，得暖气则舒展。
加减：None
功效：None
用法用量：每服3钱匕，水1盏，煎至7分，去滓，稍热服，不拘时候。候身暖并筋脉舒展则止。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
