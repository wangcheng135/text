序号：67149
名称：麻黄汤
组成：麻黄4两，栝楼根4两，大黄4两，甘草1两。
出处：《千金》卷十。
主治：疟疾须发汗者。
加减：None
功效：None
用法用量：上（口父）咀。以水7升，煮取2升半，分为3服，未发前，食顷各服1次，临以服1次。服后皆厚覆取汗。
制备方法：None
临床应用：None
各家论述：《千金方衍义》，疟宜发汗，必壮热脉实，不得不用麻黄急开肌表以泄外淫之邪；更审便溺燥结，又不得不用大黄并疏里气以通内蕴之滞。麻黄力猛，甘草和之；大黄性暴，栝楼根濡之，方得兼济之炒。服后厚覆取汗，必非夏秋时疟治例，即当寒月，苟非北方禀赋之强亦难效用，用方者不可不审，反归咎于立方之过也。
用药禁忌：None
附注：None
