序号：24558
名称：羊肉汤
组成：肥羊肉3斤（去脂），当归1两，桂心2两，芍药4两，甘草2两，生姜4两，芎3两，干地黄5两。
出处：《千金》卷三。
主治：产后虚劳，摄养不善，内伤七情，阳陷阴逆，升降倒置，脏腑交病，表里均亏，乍起乍卧，饮食不化，时作嗽咳，目昏头痛，口渴盗汗，寒热往来，喘乏自汗，少气惊悸，腹中绞痛。。
加减：None
功效：None
用法用量：以水1斗半，先煮肉，取7升，去肉，纳余药，煮取3升，去滓，分3服，不愈重作。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：《金匮》当归生姜羊肉汤为产后腹中（？？）痛之圣药，即寒疝、小腹结痛亦得用之，况合四物以调其血，加桂以通其滞，用甘草者，以缓姜、桂之急也。缓急得宜，攻补兼济，非深入《金匮》之奥，何以及此。
用药禁忌：None
附注：《千金翼》有葱白一斤。
