序号：35074
名称：龟龄集
组成：黄毛鹿茸（去毛）2两，补骨脂（黄酒制）3钱，石燕（鲜姜炙）4钱，急性子（水煮）2钱5分，细辛（醋炙）1钱5分，生地8钱，杜仲炭2钱，青盐4钱，丁香（用生川椒2分炒，去川椒）2钱5分，蚕蛾（去足翅）2钱，蜻蜓（去足翅）4钱，熟地6钱，苁蓉（酒制）9钱，地骨皮（蜜炙）4钱，附子（炙）5钱，天冬（用黄酒1钱炙）3钱，山参（去芦）1两，甘草（炙）1钱，山甲（炒珠）8钱，枸杞子3钱（1钱蜜炙），淫羊藿（羊油制）2钱，锁阳3钱，牛膝（用黄酒3钱制）4钱，砂仁4钱，麻雀脑3钱，菟丝子（用黄酒2钱制）3钱，对海马（用苏合油3钱制）9钱，硫黄3分，镜面砂2钱5分。
出处：《北京市中药成方选集》。
主治：阳虚气弱，盗汗遗精，筋骨无力，行步艰难，头昏眼花，神经衰弱，妇女气虚血寒，赤白带下。
加减：None
功效：滋阴补肾，助阳添精。
用法用量：每服1钱，温开水送下。
制备方法：将麻雀脑、硫黄2味装入猪大肠内，用清水煮之，煮至麻雀脑和硫黄溶合一起时倒出，去猪大肠，晒干，再合以上药为粗末，装入银桶内蒸之。蒸至3尽夜，将药倒出，凉干装瓶，每瓶装1钱。
临床应用：None
各家论述：None
用药禁忌：忌生冷。
附注：None
