序号：75654
名称：解风散
组成：人参1两，川芎1两，独活1两，麻黄（去节，汤洗，焙）1两，甘草1两，细辛（去苗）半两。
出处：《宣明论》卷二。
主治：风成寒热，头目昏眩，肢体疼痛，手足麻痹，上膈壅滞。
加减：None
功效：解散虚风。
用法用量：每服3钱，水1盏半，加生姜5片，薄荷叶少许，同煎至8分，不拘时候。
制备方法：上为末。
临床应用：None
各家论述：1.《医门法律》：风入既久，胃气致虚，故以人参为君；臣以麻黄、川芎，佐以独活、细辛，使以甘草，而和其营卫，乃可收其外解之功也。_x000D_
2.《中国医学大辞典》：今人但知人参为补虚之药，不知人参有祛邪荡实之功。此证因虚风久袭，若独用麻黄，无人参助其胃气，必不能奏效也。
用药禁忌：None
附注：本方方名，《医学纲目》引作“解风汤”。
