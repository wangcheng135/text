序号：69289
名称：黄疸汤
组成：茵陈30g，山栀9g，金银花15g，连翘15g，败酱草15g，板蓝根15g，赤芍9g，白芍9g，柴胡6g，神曲15g，苏梗6g，桔梗6g，大豆黄卷15g。
出处：《临证医案医方》。
主治：急性黄疸型肝炎（阳黄）。巩膜黄染，周身皮肤发黄，小便黄赤，舌苔黄腻，脉弦数。
加减：None
功效：清热利湿退黄。
用法用量：None
制备方法：None
临床应用：None
各家论述：茵陈清热利湿退黄，大豆黄卷解湿热退黄，金银花、连翘、板兰根、败酱草、山栀清热解毒，苏梗、桔梗、柴胡舒肝理气，神曲发酵协助退黄，芍药柔肝养阴与柴胡配伍应用，为治疗肝胆疾患的要药。
用药禁忌：None
附注：None
