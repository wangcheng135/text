序号：57579
名称：润疮生肌膏
组成：槟榔1两，白芍药1两，丁香1两，细辛1两，黄连1两，川芎藭1两，杏仁1两（汤浸，皮尖双仁），桂心1两，天南星1两，牛膝1两（去苗），羌活1两，附子1两（生，去皮脐），藁木1两，防风1两（去芦头），木鳖子1两（去壳），当归1两，木香1两，白芷1两，乳香1两，白胶香1两，麝香半两（细研），蜡4两，羊脂1斤，野驼脂1斤，猪脂1斤。
出处：《圣惠》卷六十三。
主治：一切痈疽发背，肌肉不生，干急疼痛。
加减：None
功效：None
用法用量：看患处大小，涂贴于上，日2度换之。
制备方法：上药除脂、蜡、麝香外，都细锉，以米醋半升拌令匀，浸1宿，先取三般脂于铛内文火煎沸，即下诸药，煎半日，候白芷色赤漉出，下蜡令消，以绵滤过，瓷盒盛，调入麝香令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
