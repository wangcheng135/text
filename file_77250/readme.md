序号：77266
名称：雄朱丸
组成：雄黄（研）1钱，朱砂（研）1钱，龙脑（研）1钱，麝香（研）1钱，白僵蚕（去丝嘴，生）半两，白附子（生）半两，天南星（洗，生）半两，乌蛇（去皮骨，生）半两。
出处：《局方》卷一。
主治：中风涎潮，咽膈作声，目眩不开，口眼斜，手足不遂，及一切风疾。
加减：None
功效：None
用法用量：如中风涎潮，牙关不开，先用大蒜1瓣捣烂，涂在两牙关外腮上，次用豆淋酒化1丸，揩牙龈上即开，续用薄荷酒化下1-2丸；如丈夫风气，妇人血风，牙关紧急者，只用豆淋酒化药，揩牙龈上即开；如头风目眩，暗风眼黑欲倒者，急嚼1-2丸，薄荷酒送下。
制备方法：上除研外，余皆为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
