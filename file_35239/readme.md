序号：35249
名称：乳香黄耆散
组成：黄耆1两，当归（去芦）1两，川芎1两，陈皮1两，麻黄（去根节）1两，甘草1两，芍药1两，人参5钱，米壳2两（去根蒂，蜜炒），乳香5钱，没药5钱（另研）。
出处：《普济方》卷二八三。
主治：一切恶疮，痈疽，发背，疔疮，疼痛不可忍者；或疮气入腹，神昏不醒呕吐者；打扑伤损，筋骨疼痛；或妇人产后腹痛，恶物不下。
加减：None
功效：未成者速散，已成者溃败，脓不出则以刀砭，其恶肉自下。
用法用量：每服3钱，水1盏，煎至7分，去滓温服，如疮在上食后，如疮在下食前服。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
