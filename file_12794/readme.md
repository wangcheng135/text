序号：12800
名称：石斛丸
组成：石斛1两（去根，锉），天雄3分（炮裂，去皮脐），黄耆1两（锉），肉桂半两（去粗皮），鳖甲1两（涂醋炙令黄，去裙襴），当归3分，芎藭3分，白术3分，沉香3分，海桐皮3分（锉），牛膝1两半（去苗），杜仲1两半（去粗皮，微炙，锉）巴戟1两，五味子3分，干漆3分（捣碎，炒令烟出），枳壳3分（麸炒微黄，去瓤）。
出处：《圣惠》卷二十九。
主治：虚劳衰损，气血虚弱，风邪所乘，肢节不利，身体疼痛。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌苋菜。
附注：None
