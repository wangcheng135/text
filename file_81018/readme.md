序号：81034
名称：霹雳散
组成：附子（浓甘草汤煎去毒）3两，吴茱萸（泡去第1次汁盐水微炒）3两，丝瓜络（烧酒洗）5两，陈伏龙肝2两（烧酒1小杯收干），木瓜（络石屯7钱煎汁炒干）1两5钱，丁香（蒸晒）1两。
出处：《霍乱论》卷下。
主治：阳虚中寒、腹痛吐泻，转筋肢冷，汗淋，苔白，不渴，脉微欲绝者。
加减：None
功效：None
用法用量：外以醋半酒杯，盐1钱5分，藕肉1两5钱，煎滚，瓦上炙存性研，每服加3厘，每病止须用半服，参汤下。
制备方法：上为极细末，分作19服。
临床应用：None
各家论述：None
用药禁忌：确系寒证，此散固佳，若未辨阴阳，极宜审慎，勿轻试也。
附注：None
