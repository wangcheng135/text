序号：32653
名称：羌活散
组成：羌活1两，天麻半两，胡麻子半两，细辛半两，麻黄3分（去根节），藿香半两，附子1两（炮裂，去皮脐），牛膝半两（去苗），白鲜皮半两，芎半两，天南星半两，蝉壳半两，旋覆花1两，白附子半两（炮裂），地龙半两（微炒），乌蛇肉1两（酒浸，炙令黄），晚蚕蛾半两（微炒），干蝎1分（微炒），麝香1分（细研），甘草半两（炙微赤，锉）。
出处：《圣惠》卷二十。
主治：瘫痪风，手足垂軃，头痛目眩，涎唾不止。
加减：None
功效：None
用法用量：每服2钱，以薄荷温酒调下，不拘时候。
制备方法：上为细散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
