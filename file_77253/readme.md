序号：77269
名称：雄朱丸
组成：雄朱（研）半两，朱砂（研）半两，天麻半两，大天南星（炮裂，锉净）半两，白附子（炮裂）半两，白僵蚕（略炒）半两，狼毒（炮）半两，桂（去粗皮，取有味者，不见火）半两，槟榔半两，南木香半两，白花蛇2两，乌蛇（2味并用酒浸，去皮骨，取肉，焙干）2两，蝎梢1分（略炒），大川乌头（水浸1宿，去皮脐，切片阴干，只生用）1两半，麝香（研）1钱，龙脑（研）1钱。
出处：普济方卷九十六引《经验方》。
主治：中风，手足不遂，麻痹疼痛。
加减：None
功效：None
用法用量：每服15丸，薄荷温酒微嚼下，腹空时服。屡用此得效。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
