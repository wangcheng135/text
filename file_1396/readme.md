序号：1396
名称：分心气饮
组成：木通（去节）1两，赤芍药1两，赤茯苓1两，肉桂（去粗皮）1两，半夏（汤洗7次）1两，桑白皮（微炒）1两，大腹皮1两，陈皮（去瓤）1两，青皮（去白）1两，甘草（炙）1两，羌活1两，紫苏（去粗梗）4两。
出处：《局方》卷三(续添诸局经验秘方)。
主治：男子、妇人一切气不和，多因忧愁思虑，怒气伤神，或临食忧戚，或事不随意，使抑郁之气留滞不散，停于胸膈之间，不能流畅，致心胸痞闷，胁肋虚胀，噎塞不通，噫气吞酸，呕哕恶心，头目昏眩，四肢倦怠，面色萎黄，口苦舌干，饮食减少，日渐羸瘦，或大肠虚秘；或因病之后，胸膈虚痞，不思饮食。
加减：None
功效：消化滞气，升降阴阳，调顺三焦，和脾进食。
用法用量：每服3钱，水1盏，加生姜3片，大枣2个，灯心5茎，同煎至7分，去滓温服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
