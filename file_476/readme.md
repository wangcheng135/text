序号：476
名称：五积丸
组成：缩砂仁5钱，木香2钱，丁香2钱，肉豆蔻3个（面煨），大曲饼（生）3钱，三棱（煨）3钱，莪茂（煨）3钱，白茯苓（去皮）3钱，腻粉2钱（炒），人参（去芦头）3钱，白术3钱，代赭石（火烧醋淬）3钱，白姜（炮）2钱，麦芽3分（生），百草霜1钱（炒），巴豆3钱（去壳，纸捶去油）。
出处：《普济方》卷三九二。
主治：一切食积，乳积，积痢。
加减：None
功效：None
用法用量：空心白汤吞下，五更服。如取积未动，早晨再1服，乳饭放迟，温食，免药食相忤，吐逆恶心。如止痢，食白粥即止。
制备方法：上除巴豆、百草霜另研外，余味各制为细末，再入巴豆、百草霜拌匀，捣饭为丸。
临床应用：None
各家论述：此药内有温育补脾理气之剂，不损胃气。有积则利，积去则止，无积则不利。凡治积痢，先服之以去其积。
用药禁忌：None
附注：None
