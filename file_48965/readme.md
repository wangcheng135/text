序号：48980
名称：绝痧方
组成：甘草1两，明矾1两，食盐1两，川乌1钱，干姜3钱。
出处：《痧胀玉衡》卷下。
主治：数患痧症。
加减：None
功效：None
用法用量：每服5分，白汤温下。新犯痧者，1-2服即愈；久犯痧者，10服全愈，不复发矣。若人属虚寒者，必加倍多服，方能有效。
制备方法：上为细末，米饭捣为丸。
临床应用：None
各家论述：用甘草以助胃，用干姜、川乌以充胃，用明矾以解毒，用食盐以断痧，乌姜性热，恐人有宜有不宜，故每服止用五分为则，惟取其能绝痧根焉尔。
用药禁忌：必痧症已愈，然后可服，以绝其根，否则稍有痧气未除，断不可服，恐甘草作胀，热者助邪，反害之矣。
附注：木八（《痧症全书》卷下）、六十四号归妹方（《杂病源流犀烛》卷二十一）。
