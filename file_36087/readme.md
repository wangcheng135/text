序号：36097
名称：参茸养元膏
组成：甘草2两，牛膝1两，鹿茸8钱，生地8钱，熟地8钱，淡苁蓉8钱，菟丝子8钱，川附8钱，川断8钱，麦冬8钱，远志8钱，蛇床子8钱，虎骨8钱，精珠（即穿山甲）8钱，宵花8钱，方八（即番木鳖）2钱，木香2钱。
出处：《饲鹤亭集方》。
主治：五劳七伤，诸虚百损，腰膝疼痛，半身不遂，膀胱疝气，带浊淋沥，阴痿不起。
加减：None
功效：助元阳，补精髓，通血脉，镇玉池。养气保元，种子毓麟，待妇女经后去膏则_x000D_
可成孕。
用法用量：摊贴脐下或腰眼间，每贴月余再换。
制备方法：用麻油2斤煎之，再入安桂8钱，乳香4钱，赤石脂4钱，阳起石5钱，龙骨3钱，公丁香2钱，沉香2钱，鸦片2钱，倭硫4两，松香6两，黄蜡6两。上为末收膏。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
