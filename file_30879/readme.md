序号：30888
名称：走马散
组成：天麻半两，天南星（炮）半两，半夏（汤洗7遍，与生姜半两同捣，焙干）半两，白附子（炮）半两，附子（炮裂，去皮脐）半两，丹砂（研）1分，雄黄（研）1分，牛黄（研）1分，麝香（研）1分，犀角（镑）1分，腻粉（研）3分。
出处：《圣济总录》卷六。
主治：破伤中风，牙关紧急，口面斜，身体或硬或软，小儿惊风。
加减：None
功效：None
用法用量：入五味研者和匀。每服半钱匕，豆淋酒调下。汗出取愈，未汗再服1字，良久用热生姜稀饭投。要丸即用新炊饼为丸，如麻粒大。每服3-5丸，吐逆用生姜汤送下，出汗用生姜酒送下，热粥饮投之。
制备方法：上为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
