序号：23425
名称：防风浴汤
组成：荆芥穗4两，苦参3两，地骨皮3两，白牵牛3两，赤小豆3两，白蒺藜（炒，去刺）2两，防风（去芦头）2两，白僵蚕（炒，去丝嘴）2两，香白芷2两，蓖麻叶2两，杉木2两，蒴藋叶2两，木鳖子1两半（去壳，炒令焦黄色），当归（洗，焙）1两，独活（去芦头）1两，吴茱萸（汤洗7次）1两。
出处：《杨氏家藏方》卷四。
主治：风湿脚气，气血凝滞，皮肤粗涩，不自润泽。
加减：None
功效：None
用法用量：每用半两，水4碗，入连根葱白5茎，擘开，同煎5-7沸，倾出，入朴消末2钱，重搅匀，先熏，候通手淋渫。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
