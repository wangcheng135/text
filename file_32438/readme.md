序号：32447
名称：张走马玉霜丸
组成：大川乌（用蚌粉半斤，同炒，候裂，去蚌粉不用）8两，川楝子（麸炒）8两，破故纸（炒）4两，巴戟（去心）4两，茴香（焙）6两。
出处：《局方》卷五（吴直阁增诸家名方）。
主治：男子元阳虚损、五脏气衰，夜梦遗泄，小便白浊，脐下冷疼，阳事不兴，久无子息，渐致瘦弱，变成肾劳，眼昏耳鸣，腰膝痠疼，夜多盗汗。妇人宫脏冷，月水不调，赤白带漏，久无子息，面生（黑干？？）（？？黑曾），发退不生，肌肉干黄，容无光泽。
加减：None
功效：精元秘固，内施不泄，留浊去清，精神安健。
用法用量：每服30-50丸，空心、食前用酒或盐汤送下。
制备方法：上为细末，用酒打面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“玉霜丸”。
