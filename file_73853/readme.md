序号：73869
名称：硼砂煎丸
组成：硼砂2两（不夹石者，细研，以醋1升半，与芫花末同熬如膏），芫花1两（炒令黄，捣罗为末），川乌头半两（炮裂，去皮脐），川大黄1两（锉碎，微炒），鳖甲1两（涂醋，炙令黄，去裙襕），当归半两，木香半两，桂心半两，蓬莪术半两，京三棱半两（炮，锉），干漆3两（捣碎，炒令烟出），青橘皮3分（汤浸，去白瓤，焙）。
出处：《圣惠》卷四十八。
主治：积聚气，久不消散，腹胁胀痛，面无颜色，四肢不和。
加减：None
功效：None
用法用量：每服15丸，食前以温酒送下。
制备方法：上为细末，纳前煎中，更入少蒸饼为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《普济方》引作“硇砂煎丸”。方中硼砂，《普济方》引作“硇砂”。
