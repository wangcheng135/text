序号：84238
名称：五邪汤
组成：菖蒲1两，秦艽1两，桂心1两，当归1两，禹余粮1两，人参1两，附子（炮）1两，黄芩1两，甘草（炙）1两，远志（去心）1两，防风1两，龙骨2两，赤石脂2两，茯苓2两，芍药2两，芎2两，防己2两。
出处：《外台》卷十五引《深师方》。
主治：风邪恍惚，悲涕泣，狂走，如有神之状，身体强直或疼痛，口噤喉痹，水浆不通，面目变色，甚者不识人。
加减：None
功效：None
用法用量：上药治下筛，作粗散调和，取水2升(一方取东流水煮小沸)，纳散2两，煮取1升5合，未食服5合，日再夜1。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：忌羊肉、饧、海藻、菘菜、酢物。
附注：五邪菖蒲散(《圣惠》卷六十九)、五邪菖蒲汤(《圣济总录》卷十四)、菖蒲汤(《圣济总录》卷一二四)、菖蒲散(《普济方》卷三一七)。
