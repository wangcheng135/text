序号：48045
名称：神机丹
组成：黄耆（蜜炙）2两，白术（蜜炙）3两，茯苓1两，白扁豆（炒）1两，建莲肉（去心）1两，薏苡仁（炒）1两，山楂肉1两，炙甘草6钱，广陈皮6分，石菖蒲（九节，去皮，桑枝拌蒸）1两6钱。
出处：《叶氏女科》卷四。
主治：魃病。妇人先有小儿，未能行走，而母复有胎妊，使儿饮此乳，则作魃病，令儿黄瘦骨立，精神不爽，身体痿瘁。
加减：None
功效：None
用法用量：先用伏翼（即蝙蝠）烧灰，研细，米饮调下5分，每日4-5次，急令断乳。随用神机丹，月内小儿每服1分，逐月加1分，周岁儿每服1钱2分；2岁儿每服1钱5分；3岁儿每服2钱，5岁以后每服3钱，白汤调下。间服交泰丹。
制备方法：上为极细末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
