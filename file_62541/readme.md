序号：62557
名称：淋渫药丁香散
组成：紫稍花半两，丁香半两，肉桂半两，蛇床子半两，吴茱萸半两，细辛（去苗）半两，红豆1两，川芎1两，莨菪子1两，黑狗脊1两，藿香叶1两，甘松1两，山茱萸1两，蜀椒（微炒）1两，香附子3分，芫花3分，巴戟3分，木香3分，甜葶苈3分（炒），香白芷3分，槐子3分（炒），芸苔子3分，天雄（炮裂，不去皮）3分。
出处：《御药院方》卷八。
主治：肾气虚弱，阴囊多汗或冷，肿痛不消，或牵引少腹时发疼痛。
加减：None
功效：None
用法用量：每用酸浆水3升，药末1两，盐少许，煎5-7沸，渐渐乘热小浴下部，临卧用之。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
