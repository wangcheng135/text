序号：27299
名称：补肺汤
组成：罂粟壳2两（制），人参半两，粉草半两，陈皮1两，茯苓1两，杏仁（制）1两，白术1两，明阿胶（炒）1两，北五味子1两，桑白皮1两，薏苡仁1两，紫苏茎1两。
出处：《妇人良方》卷六。
主治：男子、妇人远年近日肺气咳嗽，上气喘急，喉中涎声，胸满气逆，坐卧不安，饮食不下，及肺感寒邪，咳嗽声重，语音不出，鼻塞头昏。
加减：None
功效：None
用法用量：每服3钱，水1盏半，加生姜3片，大枣2枚，乌梅半个，煎至1盏，临卧温服。
制备方法：上(口父)咀为末。
临床应用：None
各家论述：None
用药禁忌：None
附注：原书云：仆每用无效，遂加百合、贝母（去心）、半夏曲、款冬花各一两，服之良验。
