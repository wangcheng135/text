序号：66811
名称：麻子苏子粥
组成：紫苏子半合，大麻子半合。
出处：《本事》卷十。
主治：妇人产后郁冒多汗，大便秘，及老人、诸虚人风秘。
加减：None
功效：顺气，滑大便。
用法用量：用水再研取汁1盏，分2次煮粥啜之。
制备方法：上药净洗，为极细末。
临床应用：None
各家论述：《本事方释义》：紫苏子气味辛温，入手太阴、足厥明，能降逆下气。大麻子气味辛甘平而润，入手足阳明、足太阴，能润肠胃。
用药禁忌：None
附注：紫苏麻仁粥（《济生》）、苏麻粥（《寿亲养老》卷四）、苏子麻仁粥（《医统》卷六十九）、麻苏粥（《济阴纲目》卷十四）、麻仁苏子粥（《医方集解》。本方改为丸剂，名“苏麻丸”（见《中国医学大辞典》）。
