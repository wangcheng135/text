序号：3405
名称：太乙保生丹
组成：羚羊角1两（煎浓汤，拌诸药，晒研，如磨更妙），蝉衣3钱，牛蒡3钱，白附子（竹沥钱半、姜汁钱半，拌、晒，研）3钱，橘红3钱，僵蚕（漂淡）3钱，天麻（制）3钱，桔梗4钱，生南星（竹沥1钱、姜汁1钱，拌、晒、研）4钱，全蝎（漂洗）4钱，牙皂1钱，蟾酥（酒蒸化，合星、附拌晒，研）1钱，当门子1钱5分，蜣螂12只（去足翅，生合），车前5钱，天竺黄5钱，茯神5钱。
出处：《华氏医方汇编》卷三。
主治：小儿急惊，由外感风邪传之，化热化痰，动风动火所致者。
加减：None
功效：None
用法用量：每服1丸，重证2-3丸。
制备方法：上为细末，用粳米饭1钟，姜汁5钱，竹沥1两，化糊为丸，如梧桐子大，朱砂3钱为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
