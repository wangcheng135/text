序号：40425
名称：虎杖饮
组成：虎杖1两半，柴胡（去苗）1两半，五味子（炒）1两半，熟干地黄（焙）1两半，白茯苓（去黑皮）1两半，陈橘皮（去白，焙）1两半，麦门冬（去心，焙）1两半，黄芩（去黑心）1两半，甘草（炙，锉）1两半，人参1两，桂（去粗皮）2两，黄耆（锉）2两，芍药2两，当归（切，焙）2两。
出处：《圣济总录》卷八十七。
主治：男子、妇人冷劳。身体羸瘦，食不化，心腹痞满，呕吐吞酸，面色萎黄，甚则心腹常痛，大肠泄痢，手足逆冷，骨节痠痛。
加减：None
功效：None
用法用量：每服5钱匕，水1盏半，加生姜7片，大枣3枚（劈），同煎至8分，去滓温服，不拘时候。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
