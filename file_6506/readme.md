序号：6506
名称：二号扫风丸
组成：大风子（土炒，去毛）62g，何首乌31g，白蒺藜31g，菊花15g，橘红15g，防风15g，牛膝15g，全蝎15g，当归尾15g，黄连15g，车前子15g，金银花15g，马钱子4g（土炒，去毛，用香油拌，另研），黄芩6g，苦参7g，麝香6g，红白12g，胡麻12g，全蛇皮1条。
出处：《中医皮肤病学简编》。
主治：麻风。
加减：None
功效：None
用法用量：成人每次6g，1日2次，徐徐增量，每次增2g。1周后可改为1日3次，最大剂量，可增至每日25g，饭前开水泡陈茶送下。
制备方法：上为细末，用荞面4两和匀，煮成膏，放凉后，再将麝香放在半杯白烧酒内溶化，掺于前药内为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
