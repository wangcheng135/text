序号：30337
名称：还少丹
组成：莲花蕊3两，生地黄3两，熟地黄（怀庆者佳）3两，五加皮（海州者佳）3两，槐角子3两，没实子6个（三阴三阳，有孔阴，无孔阳）。
出处：《摄生众妙方》卷二。
主治：None
加减：None
功效：养血消痰，乌须黑发。
用法用量：每饭后取数饼噙化，其酒任意饮之，以醉为度，酒须连日饮尽，若久收恐味变也。酒药尽而须发黑矣，若欠黑，再照前制作2-3料可矣，多不过4料。若饼子难噙化，可作丸子，以酒咽之。
制备方法：上药木杵、石臼捣碎，将绢缝袋一个，长8寸宽6寸装药，用无灰好酒10斤，入不津瓷坛同浸，春、冬一月，夏10日，秋20日，满日取药晒干，仍用木杵、石臼捣为细末，炼蜜为饼，又以薄荷为末，一层饼放一层末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
