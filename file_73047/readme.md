序号：73063
名称：曾青汤
组成：曾青、甘草（炙）2分，当归3分，细辛3分，芍药3分，独活3分，大黄3分，麻黄（去节）3分。
出处：《普济方》卷三七六。
主治：少小二十五痫，日数百发者。
加减：若自汗出，去麻黄，加麻黄根1分；若腹中急痛，加当归、芍药各1分；若缩口聚唾吐乳者，加细辛1分；中风身强，戴眼反折者，加独活1分。总当视病增减药，药皆令精新。
功效：None
用法用量：上以水3升，煮取7合，一月儿服如杏核，二月儿服如二杏核大，以此为准。汤讫，当抱儿令汗出；若先下者，勿令汗出。或分五服，日三夜二，小有痫候，便可作，无病候亦可服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
