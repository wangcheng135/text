序号：34062
名称：芡实丸
组成：芡实（蒸，去壳）2两，莲花须2两，茯神（去木）1两，山茱萸（取肉）1两，龙骨1两，五味子1两，枸杞子1两，熟地黄（酒蒸，焙）1两，韭子（炒）1两，肉苁蓉（酒浸）1两，川牛膝（去芦，酒浸，焙）1两，紫石英（煅7次）1两。
出处：《济生》卷一。
主治：思虑伤心，疲劳伤肾，心肾不交，精元不固，面少颜色，惊悸健忘，梦寐不安，小便赤涩，遗精白浊，足胫痠疼，耳聋目昏，口干脚弱。
加减：None
功效：None
用法用量：每服70丸，空心盐酒、盐汤任下。
制备方法：上为细末，酒煮山药糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
