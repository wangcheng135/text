序号：28489
名称：芫花丸
组成：芫花（醋制干）1两，干漆（炒令烟尽）半两，狼牙根半两，桔梗（炒黄）半两，藜芦（炒）半两，槟榔半两，巴豆10个（炒微黑黄）。
出处：《本事》卷三。
主治：积聚停饮，痰水生虫，久则成反胃，及变为胃痈。
加减：None
功效：常服化痰，消坚，杀虫。
用法用量：每服2-3丸，加至5-7丸，食前姜汤送下。
制备方法：上为细末，醋糊为丸，如赤豆大。
临床应用：None
各家论述：《本事方释义》：芫花气味咸辛温，入手、足太阳，善能行水；干漆气味辛温，入足厥阴，降而行血；狼牙根气味苦辛寒，入足少阳、厥阴，善能杀虫；桔梗气味苦辛平，入手太阴，为诸药之舟楫；藜芦气味辛温，入手阳明，能行积滞；槟榔气味辛温，入足太阴、太阳，能下气消积；巴豆气味辛热，有毒，入手足阳明、足大阴，此积聚痰饮，久而不去，甚至生虫、反胃，胃变为痈，非有毒、行血下气、攻坚消积之药不能扫除沉痼也。
用药禁忌：禁酒即易治，不禁无益也。
附注：方中干漆，《准绳·类方》作牛膝。
