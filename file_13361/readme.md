序号：13368
名称：白花蛇丸
组成：白花蛇2两（酒浸，炙微黄，去皮骨），杏仁半两（汤浸，去皮尖双仁，麸炒微黄），黄耆1两（锉），葫荽子1两（微炒），蝟皮1两（炙黄焦），人参1两（去芦头），鲤鱼皮1两（烧灰），附子1两（炮裂，去皮脐），枳壳2两（麸炒微黄，去瓤），男儿发2两（烧灰），肉桂2两（去皱皮），当归1两（锉，微炒）皂荚树耳1两（微炒）。
出处：《圣惠》卷六十。
主治：肠风下血，日夜不绝，疼痛至甚。
加减：None
功效：None
用法用量：每服30丸，食前煎人参汤送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
