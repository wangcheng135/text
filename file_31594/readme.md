序号：31603
名称：杜仲酒
组成：杜仲（去粗皮，切，炒）1两，干姜（炮）1两，萆薢1两，羌活（去芦头）1两，天雄（炮裂，去皮脐）1两，蜀椒（去目及闭口者，炒出汗）1两，桂（去粗皮）1两，芎1两，防风（去叉）1两，秦艽（去苗土）1两，甘草（炙）1两，细辛（去苗叶）3分，五加皮3分，石斛（去根）3分，续断3分，地骨皮（洗）3分，桔梗1两半。
出处：《圣济总录》卷八十五。
主治：肾虚冷或感寒湿，腰脚冷痹，或为疼痛。
加减：None
功效：None
用法用量：上各锉细，用酒1斗，瓷瓶内浸，密封，以重汤煮2时辰，取出候冷开封。每次温服1盏，不拘时候，常令如醉。
