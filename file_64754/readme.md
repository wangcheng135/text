序号：64770
名称：硇魏丸
组成：硇砂（水净，去石，炒）3两，胡芦巴1两半，木香半两，沉香半两，陈皮4两，干姜4两，当归4两，厚朴4两，川芎4两，茴香4两，胡椒4两，砂仁4两，甘草4两，大附（炮）4两，白术、青盐、五味1两半，阿魏半两（醋化），好酒5升，好醋5升，好蜜10两，细面2斤，丁香。
出处：《普济方》卷一八二。
主治：脾元气弱，久积阴冷，心腹胁肋胀满刺痛，面色青黄，肌体瘦弱，怠惰嗜卧，食少多伤，噫气吞酸，哕逆恶心，腹中虚鸣，大便泻利，胸膈痞塞，饮食不下，呕噎霍乱，体冷转筋，五膈五噎，痃癖积聚，翻胃吐食，久病久痢。
加减：None
功效：None
用法用量：每服15-20丸，空心嚼破，姜酒汤送下。
制备方法：上为末，用银石锅，内入酒醋蜜，先下丁魏盐三味，并面同煎稠粘，便下药末半斤以来，更煎如稀糊，渐渐入药末，煎至得所，熄火取出，更入干药末，搜和成剂，捣杵为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌羊血豉汁。
附注：None
