序号：22973
名称：防己汤
组成：防己1两，大戟1两，木香1两，赤茯苓（去黑皮）1两，海蛤1两，犀角屑1两，胡椒1两，白术1两，葶苈1两，防风（去叉）1两，木通1两，桑根白皮1两，紫苏1两，陈橘皮（炙）1两，牵牛子1两，诃黎勒（去核）1两，郁李仁1两，白槟榔1两，大黄2两，麝香少许（汤成下，不用研）。
出处：《圣济总录》卷八十引《膜外气方》。
主治：膜外气水病，不限年月深浅，洪肿大端。亦治脚气，时时冲心。
加减：None
功效：None
用法用量：上20味并须新药，挫后称为2剂。以水3升宿浸，明日五更用铛文火煎，减去1升，绞取饮可3盏，平旦空腹服1盏，如人行5里更服1盏，又如人行5里更服1盏，至第2-3服，如药冷用重汤暖之，不可冷服。若久病腹中虚，服至第3盏，即微利2-3行。若腹中实者，至日午即转泻，宜用盆盛验之，必有恶浊黄水，或青黑恶物出3-5升，并气化为之。泻若不甚困，慎而止之，必自住，若觉力乏，即服浆水粥补之。后隔3-5日，更服1剂，还依此法服之，小儿及老人，随意加减。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：防己饮（《圣济总录》卷七十九）。同书防已饮中胡椒，作“胡黄连”。
