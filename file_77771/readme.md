序号：77787
名称：雄附汤
组成：天雄2只（炮，去皮脐），附子4只（8钱重者，炮，去皮脐），绵黄耆（蜜炙）2两，新罗参（去芦）2两，白术（炒）2两，白茯苓（去皮）2两，白芍药2两，肉豆蔻（面裹，煨）1两，木香（炮）1两，丁香（不见火）1两，川白姜4两（泡洗），甘草（炙）半两，沉香（不见火）半两，诃子（去核）半两。
出处：《魏氏家藏方》卷五。
主治：脏腑虚寒，泄泻。
加减：常服不须入乳香、鹿角霜；如脏腑坚固，不必用诃子亦得。
功效：健脾温中。
用法用量：上(口父)咀。每服3钱，水1盏半，入鹿角霜、乳香各少许，生姜5片，枣子2枚，同煎至1小盏，空心服之，并滓再煎。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
