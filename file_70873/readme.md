序号：70889
名称：紫草冬葵汤
组成：紫草茸1钱2分，山栀子1钱2分，黄芩1钱2分，秦艽1钱1分，苦参1钱1分，冬葵子1钱半，露蜂房1钱，白茯苓1钱，木通1钱，白芍药1钱，泽泻1钱，车前子1钱。
出处：《准绳·幼科》卷五。
主治：小便不通，毒气闭塞。
加减：如急数，茎中痛者，加甘草梢8分，苦楝子1钱；如痛甚欲死者，加川牛膝1钱3分；如有赤如血色者，加胡黄连1钱3分；如溺血者，加当归1钱，川芎1钱，龙骨（火煅）1钱，菟丝子1钱；红甚者，加生地黄；白溺者，加使君子各1钱3分，黄连1钱1分，韭子（研）1钱2分；浊甚者，加桑螵蛸1钱。
功效：None
用法用量：每服4钱，水煎，食远温服。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
