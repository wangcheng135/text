序号：41712
名称：金樱子粥
组成：金樱子10-15g，粳米（或糯米）1-2两。
出处：《药粥疗法》引《饮食辨录》。
主治：滑精遗精，遗尿，小便频数；脾虚久泻，妇女带下病，子宫脱垂等。
加减：None
功效：收涩、固精、止泻。
用法用量：先煎金樱子，取浓汁，去滓，用粳米或糯米煮粥。每天分2次温服，以2-3天为1疗程。
制备方法：None
临床应用：None
各家论述：金樱子味酸涩，性平无毒，入肾、膀胱、大肠经。《蜀本草》说能治脾泄，下痢，止小便利，涩精气。《滇南本草》：治日久下痢，血崩带下，涩精遗泄。中医认为，脾气虚则久泻不止，膀胱虚寒则小便不禁，肾气虚则精滑自遗，金樱子入三经而收敛虚脱之气，所以治疗上述病证有很好的效果。
用药禁忌：感冒期间以及发热的病人不宜食用。
附注：None
