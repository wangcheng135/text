序号：33004
名称：陈曲散
组成：陈曲（炒）3两，大麦糵（炒）3两，生地黄（切）9合，白术8两，牛膝（酒浸，切，焙）9合，桑耳（锉）9合，干姜（炮）8两，当归（锉）13两半，生姜（切）9合，桃仁（去双仁皮尖，炒）6合，杏仁（去皮尖双仁，炒）6合，陈橘皮（汤浸，去白）8两。
出处：《圣济总录》卷一五三。
主治：妇人腹内冷癖血块，虚胀，月经不调，瘦弱不能食，面无颜色，状如传尸。
加减：None
功效：None
用法用量：每服3钱匕，空腹温酒调下。炼蜜为丸服亦得。
制备方法：上捣如泥，纳瓶中，以物盖密封之，勿令泄气，于一石米上蒸之，饭熟出之，停屋下三日，开出晒干，再为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
