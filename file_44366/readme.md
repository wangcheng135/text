序号：44381
名称：一粒珠
组成：全穿山甲（一足用好醋制，一足用松萝茶制，一足用麻油制，一足用苏合油制，俱连一边身子，如鳞甲有不全处，须再取一具，视取原缺处者补全，同炙淡黄色为度，焦黑不可用）1具，犀牛黄3钱，真珠3钱，大劈砂4钱，明雄黄4钱，原麝香4钱，梅花冰片4钱。
出处：《绛囊撮要》。
主治：一切无名肿毒，对口搭手，痈疽发背,小儿惊风，闷痘初起,流注流痰，附骨阴疽，乳痈，乳癌。
加减：None
功效：消肿解毒。
用法用量：服时用人乳化开，真陈酒煮，冲服1丸，量佳不妨多饮，盖暖患处，重症倍服。小儿惊风，用陈胆星1分，钩藤3分，橘红3分，煎汤化服1丸；闷痘初起，用白芦根汤化服1丸。
制备方法：上为极细末，加入蟾酥1钱2分，人乳化，饭锅上蒸，再量入苏合油，打和为丸，每丸干重3分。
临床应用：None
各家论述：None
用药禁忌：怯弱、吐血、疔症、孕妇忌服。此丹药味贵重，峻利非凡，凡外科小症，幸勿轻用。
附注：一粒丹（《内外科百病验方大全》）、一粒珠丸（《中国医学大辞典》）。
