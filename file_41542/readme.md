序号：41557
名称：金黄散
组成：天花粉1两，黄柏5两，姜黄5钱，大黄5钱，白芷5钱，紫川朴2两，陈皮2两，甘草2两，苍术2两，天南星2两。
出处：《外科传薪集》。
主治：痈疽发背，诸般疔疮，跌仆，湿痰流注，大头时肿，漆疮火丹，风热天泡，肌肤赤肿，干湿脚气，妇女乳痈，小儿丹毒等。
加减：None
功效：None
用法用量：凡遇红肿，及夏月火令时，用茶汤同蜜水调敷；如微热欲作脓者，以葱汤同蜜水调敷；如漫肿无头，皮色不变，附骨痈疽、鹤膝等，俱以葱酒并调；如天泡、火赤游丹、黄水疮，俱以板兰根叶捣汁调和；烫伤，麻油调；其次诸引，又在临用之际，顺合天时调，窥病势也。
制备方法：上为末，以瓷器收贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
