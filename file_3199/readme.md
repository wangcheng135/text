序号：3199
名称：天雄丸
组成：天雄1两（炮裂，去皮脐），羚羊角屑半两，牛黄1分（细研），麝香1分（细研），天麻1两，桑螵蛸半两（微炒），蝉壳半两，牛膝半两（去苗），附子1两（炮裂，去皮脐），桂心半两，当归半两，芎半两，羌活半两，白僵蚕半两（微炒），五加皮半两，乌蛇肉2两（酒浸，炙微黄），薏苡仁半两，麻黄1两（去根节），防风半两（去芦头），干蝎半两（微炒），乳香1两，仙灵脾1两，道人头1两，朱砂半两。
出处：《圣惠》卷二十三。
主治：中风半身不遂，言语謇涩，肌肤顽痹，筋脉不利，骨节疼痛。
加减：None
功效：None
用法用量：每服20丸，以温酒送下，渐加至30丸，日3-4服。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中朱砂用量原缺，据《普济方》补。
