序号：31015
名称：羌活膏
组成：羌活（去芦头）半两，川芎半两，人参（去芦头）半两，赤茯苓（去皮）半两，白附子（炮）半两，天麻1两，白僵蚕（酒浸，炒黄）1分，干蝎（去毒，炒）1分，白花蛇（酒浸，取肉焙干）1分，川附子（炮，去皮脐）3钱，防风（去芦头，切，焙）3钱，麻黄（去节，秤）3钱，豆蔻肉2钱，鸡舌香（即母丁香）2钱，藿香叶2钱，木香2钱，轻粉1钱，珍珠1钱，麝香1钱，牛黄1钱，龙脑半字，雄黄1分，辰砂1分（上7味各别研入）。
出处：《小儿药证直诀》卷下。
主治：小儿脾胃虚，肝气热盛生风，或取转过，或吐泻后为慢惊，亦治伤寒。
加减：None
功效：None
用法用量：每服1-2丸，食前薄荷汤或麦冬汤温化下，不拘时候。
制备方法：上为细末，炼蜜和剂，旋丸如大豆大。
临床应用：None
各家论述：None
用药禁忌：实热、惊急勿服。
附注：羌活丸（《永乐大典》卷九八○引《大方》。
