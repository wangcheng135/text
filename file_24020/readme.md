序号：24028
名称：当归泽兰丸
组成：香附子（去衣，分作4处，童便4两、酒4两、醋4两，米泔4两各浸1宿）1斤，当归（去须，酒浸）2两，白芍药（炒）2两，熟地黄（酒制）2两，生地黄2两，泽兰叶1两5钱，艾叶1两5钱，白术1两5钱，黄芩1两，川芎2两。
出处：《摄生众妙方》卷十。
主治：妇人经脉不调，赤白带下，久无子者。
加减：None
功效：None
用法用量：每服60丸，空心白汤或酒送下。
制备方法：上为末，醋糊为丸，如赤豆大。
临床应用：None
各家论述：《医略六书》：血亏气滞，天癸愆期，而带脉不能收引，故赤白带下，经久不能生子焉。熟地补阴滋血，生地凉血滋阴，当归养血脉以荣经，白芍敛营阴以和血，川芎行血中之气，艾叶暖子宫之血，泽兰去宿生新，白术健脾燥湿，黄芩清肺气以肃生水之源，香附解郁结以调冲任之气。醋以丸之，汤以下之，使经脉有资，则血气调和，而天癸无不如度，带脉约束有权，何赤白带下之不除哉？自此带愈经调，天下应无不孕之妇矣。
用药禁忌：None
