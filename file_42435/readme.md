序号：42450
名称：青黛丸
组成：青黛（研）、天竺黄（研）、干蛤蟆1枚（黄泥裹，烧赤，去泥，研），干蜗牛壳（洗，炒，研）1分，黄连（去须）1分，地龙（炒）1分，人参1分，钩藤（炙）1分，龙胆1分，芦荟（研）半两，熊胆（研）半两，牛黄（研）1钱，麝香（研）1钱，雄黄（研）1钱，丹砂（研）1钱，夜明砂3分，胡黄连3分。
出处：《圣济总录》卷一七二。
主治：小儿天钓客忤，五疳八痢，十二惊痫。
加减：None
功效：None
用法用量：1岁每服1-2丸，粥饮送下。1岁以上，以意加减。
制备方法：上7味为末，与10味研者和匀，以烧饭为丸，如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
