序号：23406
名称：杏仁煎
组成：杏仁5两，五味子3合，甘草4两（炙），麻黄1斤（去节），款冬花3合，紫菀3两，干姜3两，桂心4两。
出处：《外台》卷十引《深师方》。
主治：咳上气，中寒冷，鼻中不利。
加减：None
功效：None
用法用量：食前服如半枣大，日3次。不知稍加之。
制备方法：上切，以水一斗，煮麻黄减2升，掠去沫，乃纳诸药，煮取4升，绞去滓，又纳胶饴半斤，白蜜1斤，合纳汁中，搅令相得，汤中煎如饴成。
临床应用：None
各家论述：《千金方衍义》：肺气通于鼻，肺为客邪所遏，以故鼻息不通，喘嗽痰清，非麻黄汤不能开发肺气，加干姜以温肾气，款冬、紫菀以温肺经，五味以收麻、杏之散耳。
用药禁忌：忌海藻、菘菜、生葱。
附注：杏仁煎丸（《圣济总录》卷一七六）。
