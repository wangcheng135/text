序号：25983
名称：百合地黄汤
组成：百合7枚（擘），生地黄汁1升。
出处：《金匮》卷上。
主治：百合病，不经吐、下、发汗，病形如初者。
加减：None
功效：None
用法用量：以水洗百合，渍1宿，当白沫出，去其水，更以泉水2升，煎取1升，去滓，纳地黄汁，煎取1升5合，分温再服。中病，勿更服。大便当如漆。
制备方法：None
临床应用：None
各家论述：1.《千金方衍义》：百合病若不经发汗、吐、下，而血热自汗，用百合为君，安心补神，能去中热，利大小便，导涤痰积；但佐生地黄汁以凉血，血凉则热毒解而蕴结自行，故大便当去恶沫也。_x000D_
2.《金匮要略心典》：百合色白入肺，而清气中之热，地黄色黑入肾，而除血中之热，气血即治，百脉俱清，虽有邪气，亦必自下；服后大便如漆，则热除之验也。
用药禁忌：None
附注：百合汤（《伤寒全生集》卷四）。本方方名，《外台》引作“百合生地黄汤”。
