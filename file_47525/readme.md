序号：47540
名称：神方夺命丹
组成：透明雄黄1两，肥巴豆120粒（不去油心），金鼎砒1两半，黄蜡4两（熔开）。
出处：《普济方》卷二七五。
主治：疔肿，痈疽，发背，诸恶疮，及食牛马肉发黄者。
加减：None
功效：None
用法用量：量老幼加减服之。每服50丸，多至200丸，临睡温熟水送下。不动，其丸经过脏腑，只下清黄黑水则病去；如药未下，再服则药病俱下矣。
制备方法：上药各为极细末，入蜡中搅匀，取出火，重汤泡匀为丸，如小麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：煅金鼎砒法：将透明砒四两，敲作米粒块，用黑铅一斤熔化，水中扑作珠子；先铺珠一层，次铺砒一层，层层相间，入角罐中，铅珠盖面，黄泥饼子又盖面上，其饼用箸杵十数窍，饼四缘略用泥固定，罐口须空一二寸；水鼎颠倒覆口，铁线扎定，须做把手提挈，略固口缝，安平地上，城砖围煅，下开四窍通风，一层熟火，一层生炭，层层相铺，平药处即止；又发火自上而下煅之通红，或提出或寒炉取出，其铅熔坠在下，其砒将在上，可得四两半。
