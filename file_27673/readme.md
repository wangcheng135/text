序号：27681
名称：八珍酒
组成：当归（全用，酒洗）3两，南芎1两，白芍（煨）2两，生地黄（酒洗）4两，人参（去芦）1两，白术（去芦，炒）3两，白茯苓（去皮）2两，粉草（炙）1两半，五加皮（酒洗，晒干）8两，小肥红枣（去核）4两，核桃肉4两。
出处：《回春》卷四。
主治：None
加减：None
功效：和气血，养脏腑，调脾胃，解宿醒，强精神，悦颜色，助劳倦，补诸虚，久服百病消除。
用法用量：每晨、午、夕温饮1-2小盏。
制备方法：上(口父)咀，共装入绢袋内，用好糯米酒40斤，煮2炷香，埋净土中5日夜，取出，过3-7日服。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
