序号：77391
名称：跌打丸
组成：香附（醋炙）14两，生蒲黄8两，白及8两，赤芍8两，陈皮8两，五灵脂6两，三七2两，木香2两，大黄14两，玄胡（醋炙）8两，续断8两，乌药8两，三棱（醋炙）8两，莪术（醋炙）8两，红花8两，川芎8两，郁金8两，枳实（炒）8两，丹皮8两，青皮（炒）8两，防风8两，威灵仙6两，归尾14两。
出处：《中药制剂手册》。
主治：由跌打外伤引起之筋骨扭伤，瘀血积聚，红肿疼痛，闪腰岔气等症。
加减：None
功效：活血止痛，舒筋活络。
用法用量：每服15丸，黄酒或温开水送下，1日2次。小儿酌减。
制备方法：取香附至木香等8味为细末；取大黄至归尾等15味，用煮提法提取2次，浓稠汁约50两左右。取香附等细末，用大黄等膏汁（可酌加冷开水），按泛丸法制成丸（每10丸干重5分），晒干或低温干燥，挂衣，玻璃瓶装，密封。
临床应用：None
各家论述：None
用药禁忌：孕妇及外伤出血过多者忌服。
附注：None
