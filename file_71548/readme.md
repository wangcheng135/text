序号：71564
名称：揭障丹
组成：黄荆子1斤（晒干，去壳，净温水洗3-4次，又用童便浸3日夜，早、晚换童便，浸完又用温水洗3-4次，炒，研细听用，号揭障磨翳丹头）。
出处：《眼科秘诀》卷一。
主治：内外障眼。
加减：如内外留障重者，加雌雄石（活磁石）末3钱（银锅内煅红，醋淬7次）；如两目红如血，此三焦余热所攻，号曰珠玲，加山栀仁、玄参、麦冬各3钱；如两珠蛮大，突起如怒像者，号曰鼓睛，有风热，加防风、白蒺藜、车前子各2钱；如含浆眼，上下眼包合，不能自开，用手分开，泪倾如米汁之状，此风热太甚，攻于肝肺二经，加龙胆草3钱、防风、羌活各2钱，桑白皮2钱，白芍、柴胡各1钱；如烂弦红皮者，加桑白皮3钱，草决明、防风各1钱5分；如眼内红丝多者，加山栀仁（炒）3钱；如红气上侵黑珠，加桑白皮（蜜制）3钱；如眼中泪多者，加柴胡、升麻各3钱；如血灌瞳人，加石膏（煅）3钱，炒黑栀仁2钱，炒大黄2钱，归尾3钱；如瞳人侧身，加柴胡、升麻各5钱；如瞳人端正，我把手招他，他不把手招，谓之水火未济之像，左右轮中气不贯，加蜣螂（瓦上略焙，为末）2钱；如上眼皮盖下眼皮，作睡人之状，乃脾之倦也，加白术（陈壁土炒）3钱；如双目黑睛红侵，白珠不红，号曰血热侵肝，加当归尾、白芍、茜草各3钱，炒栀仁2钱；如白珠血红，黑珠不红，号曰余热伤肺，加百合、宣连、炒栀仁各2钱；如眼内翳突起，乃水盛火衰，号曰乌睛，乃肝不纳水之故，加木贼，花椒各3钱，柴胡、白芍各2钱；如系蟹眼虾眼者，老膜突起，加千里光3钱，磁石1钱5分；如目黑白不分，混浊污秽，触冲瞳人者，加黄柏、知母各2钱；如眼上下四角作痒，加白蒺藜（去刺）3钱；如视人长大，一人似二人者，号曰轮不分白，加青葙子3钱；如两目并太阳穴作胀，加蔓荆子3钱；如青光者，一双好眼，视物不见，号曰青光瞎子，此三轮有厚病也，加赤茯苓、玄参、黄芩各1钱5分；如鸡宿即目不明，双目黑暗，乃肝不纳余血，血倒转攻心之症，加上好藏浪大黄3钱，黄柏、知母各2钱；如眼内如针刺；谓之血热，加好大黄2钱，栀仁（炒）2钱；如一双目细小者，号曰夹视，加白茯苓、白术、枸杞子各1钱。
功效：None
用法用量：每服2-3钱，食后煎淡竹叶汤送下，1日2次。
制备方法：每用丹头1两，加当归（酒洗）、川芎、生地各2钱半（各为末），白芍（酒洗，为末）1钱半，谷精草、羌活、白芷、升麻、柴胡、草决明、木贼草（各为末）各1钱，龙胆草1钱半，荆芥、薄荷（各为末）各1钱半，诸末和研令匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
