序号：63869
名称：清暑痢疾丸
组成：姜连1两，归身1两5钱，白芍2两半（炒），黄芩2两半（炒），槟榔2两半，枳壳2两半，半夏2两，地榆2两（炒），焦楂5两，川朴2两（炒），木香1两，熟军2两，二丑2两（炒），扁豆5两（炒），滑石2两，青皮2两，干姜3钱，生草2两。
出处：《揣摩有得集》。
主治：一切暑痢，不论红白，或肚痛泻泄，或食积、水积、茶积，受热头痛，小便黄而短少，口焦而不欲饮，内有积滞湿热。
加减：None
功效：None
用法用量：大人每服3钱，小儿1-2钱，皆用红白糖冲开水送下。
制备方法：上为细末，以荷叶煎水成丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
