序号：32437
名称：忍冬汤
组成：金银花大3-5钱、中2-3钱、小1钱，赤芍大1钱、中8分、小5分，土贝母大3钱、中2钱、小8分，牛旁大2钱、中1钱、小8分，连翘（去实）大3钱、中2钱、小1钱，术通大2钱、中1钱、小8分，荆穗大1钱、中8分、小5分，红花大8分、中5分、小3分，枯黄芩大3钱、中2钱、小1钱，羌活大2钱、中1钱、小5分，甘草大5分、中3分、小2分。
出处：《痘疹正宗》卷下。
主治：一切痘后余毒之。
加减：毒留下部，如牛膝；痘疔，加归尾、青皮、地丁；痈，加生地、皂角刺、地丁；牙疳，加天花粉大3钱、中2钱、小1钱。
