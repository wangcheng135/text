序号：66457
名称：小儿七珍丹
组成：胆南星20两，天麻30两，半夏曲30两，滑石60两，寒食曲60两，全蝎30两，巴豆霜7两5钱（含油量不得超过百分之十）。
出处：《北京市中药成方选集》。
主治：急热惊风，痰涎壅盛，感冒风寒，呕吐泄泻。
加减：None
功效：清热败毒，镇惊安神。
用法用量：4-5岁服20丸，10岁服40丸，白开水化服。
制备方法：先将胆南星、天麻等6味研为细末，过罗；取巴豆霜研细，陆续兑入上细末，和匀，用冷开水泛为小丸，朱砂为衣，每16两干丸药用朱砂粉8钱，纸袋包装，每袋100粒。
临床应用：None
各家论述：None
用药禁忌：痘疹及久泻脾虚忌服。
附注：None
