序号：26810
名称：延龄固本丹
组成：天门冬（水泡，去心）2两，麦门冬（水泡，去心）2两，生地黄（酒洗）2两，熟地黄（酒蒸）2两，山药2两，牛膝（去芦，酒洗）2两，杜仲（去皮，姜酒炒）2两，巴戟（酒浸，去心）2两，五味子2两，枸杞子2两，山茱萸（酒蒸，去核）2两，白茯苓（去皮）2两，人参2两，木香2两，柏子仁2两，老川椒1两，石菖蒲1两，远志（甘草水泡，去心），泽泻1两，肉苁蓉（酒洗）4两，覆盆子1两半，车前子1两半，菟丝子（酒炒烂，捣成饼，焙干）1两半，地骨皮1两半。
出处：《回春》卷四。
主治：五劳七伤，诸虚百损，颜色衰朽，形体羸瘦；中年阳事不举，精神短少；未至五旬，须发先白；并左瘫右痪，步履艰辛，脚膝疼痛，小肠疝气；妇人久无子息，下元虚冷。
加减：妇人加当归（酒洗），赤石脂（煨）各1两。
功效：延龄固本，壮阳事，驻颜色，乌须发，强健身体。
用法用量：每服80丸，空心温酒送下。服至半月，阳事雄壮；至1月颜如童子，目视10里，小便清滑；服至3月，白发返黑；久服神气不衰，身轻体健。
制备方法：上为细末，好酒打稀面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：延龄固本丸（《仙拈集》卷三）。
