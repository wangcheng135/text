序号：64122
名称：猪肝丸
组成：猪肝1具（去膜，切，以米醋2斗煮令极烂），柴胡（去苗）2两，泽泻2两，槟榔（锉）2两，附子（炮裂，去皮脐）2两，熟干地黄（焙）2两，当归（炙，锉）2两，蜀椒（去目及闭口者，炒出汗）1两，桃仁（去皮尖双仁，炒令黄，研）1两，蒺藜子（炒去角）1两，牛膝（酒浸，切，焙）1两，木香1两，秦艽（去苗土）1两，桂（去粗皮）1两，芜荑仁（炒）1两，干姜（炮）1两，黄连（去须，炒）1两。
出处：《圣济总录》卷八十六。
主治：肾虚劳气，腰胯疼痛，脚膝无力，耳中虚鸣，夜多小便，饮食减少：女人血劳，面色萎黄，心腹刺痛，经脉不利。
加减：None
功效：None
用法用量：每服40丸，空心温酒送下。
制备方法：上17味，除肝外，捣罗为末；取肝火砂盆内研烂，同药末入臼内捣，滴余醋并熟蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
