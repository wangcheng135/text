序号：26403
名称：异香膏子
组成：黄腊1两半，斑蝥20枚（去头翅足），巴豆肉49粒，草乌1两，南星1两，皂角1两，麻油1斤（熬净蜡），猪脂半斤，白胶香半斤（上以麻油先煎斑蝥、巴豆草乌、南星、皂用五味焦黑，滤去滓，次入黄蜡、白胶香、猪脂煎成膏，瓷盒盛贮，旋和后项12味药末，重汤炖烊丹），吴茱萸3两，蛇床子3两，槟榔3两，雄黄3两，川椒半两，藜芦3两，白芷3两，黄柏3两，剪草3两，苦参3两，白矾（枯）3两，闾茹1两。
出处：《医方类聚》卷一六九引《新效方》。
主治：顽癣疥癞，一切痒疮。
加减：None
功效：None
用法用量：和前膏子敷．乘痒搔动疮靥，以指蘸药擦之，不要药多，只要擦得入肉。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：休敷面目、男女前后阴及奶上，休洗热汤。
附注：None
