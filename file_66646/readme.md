序号：66662
名称：鹿胎膏
组成：梅花鹿胎1具，祁艾3两，香附2两，川芎1两，当归1两半，白芍1两，炮姜炭5钱，红花3钱，熟地4两，吴萸5钱，桂5钱，楠5钱，黄芩5钱，川牛膝5钱，元胡5钱，杜仲1两，川断1两，丹皮5钱，丹参5钱。
出处：《全国中药成药处方集》（抚顺方）。
主治：男女一切虚劳，气血虚弱，营养不足，腰腿疼痛，精神疲倦，经血不调，子宫虚寒，经血参差，腹痛脐冷，白带稠凝，血枯经闭。
加减：None
功效：调经温寒，养血益气。
用法用量：每服1钱，开水或元酒化服。
制备方法：上药水煎数滚，滤滓，再用水熬数滚，一连4-5次，澄清，再熬成膏，兑元酒数壶，共炼成膏为度。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
