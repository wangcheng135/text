序号：34323
名称：万补丸
组成：人参1两半，当归1两半（切，焙），草豆蔻1两半（炮，去皮），嫩茄茸1两半（酥炙），乳香1两半，白术1两，阳起石1两（火煅，细研），肉桂1两（去皮），缩砂仁1两，赤石脂1两，钟乳粉1两，肉豆蔻1两（面裹煨熟），沉香1两，白姜1两（炮），荜茇1两（牛乳半盏，用火煎干，焙），茴香1两（炒），丁香1两，厚朴1两（去皮，姜制），白茯苓1两，地榆半两，大麦芽半两（炒），神曲（炒）半两，大附子7钱（炮，去皮脐），肉苁蓉2两（净洗，用酒浸1宿，切，焙），罂粟壳（和米者）20枚（炙）。
出处：《奇效良方》卷十三。
主治：脾胃久虚，大肠积冷，下痢白脓，或肠滑不固，久服诸药不效。
加减：None
功效：None
用法用量：每服30丸，食前用米饮送下。频并者加至50-70丸。
制备方法：上为细末，研匀，用木瓜15个，去瓤蒸烂，同药末捣和得所为丸，如梧桐子大，晒干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
