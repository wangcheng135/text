序号：33722
名称：灵砂丹
组成：消石（与砒1处细研，入瓷罐子内，用石灰盖口，炭火烧半日，取出，去火毒），信州砒霜半两，腻粉半两，粉霜（研）半两，黄丹（研）1两半，枯矾（研）1两半，朱砂（研，飞）1两，乳香（研）1两，桂府1两，滑石1两。
出处：《局方》卷六。
主治：脏腑怯弱，内有积滞，脐腹撮痛，下痢脓血，日夜无度，里急后重，肠鸣腹胀，米谷不化，少气困倦，不思饮食，或发寒热，渐至羸瘦。
加减：None
功效：None
用法用量：每服5丸，温粟米饮送下。未愈加丸数再服。小儿可服1-2丸。
制备方法：上为末，用蒸饼2两4钱和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
