序号：59750
名称：调经丸
组成：香附（炙）200两，阿胶（炒）100两，熟地100两，川芎50两，白术（炒）75两，橘皮50两，茯苓50两，当归75两，白芍75两，麦冬50两，法半夏50两，川断50两，丹皮50两，玄胡索（醋炙）25两，没药（炙）26两，益母草100两，吴茱萸（炙）25两，小茴香（炒）25两，甘草15两，黄芩50两，艾叶（炒炭）50两。
出处：《北京市中药成方选集》。
主治：经血不调，行经腹痛，气滞血凝。
加减：None
功效：理气调经，止痛散瘀。
用法用量：每服1丸，温开水送下，日2次。
制备方法：上为细末，炼蜜为丸，重3钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
