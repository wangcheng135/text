序号：74044
名称：暖宫万灵丸
组成：川芎3两，当归3两，芍药3两，熟地黄3两，生地黄3两，白茯苓2两，牡丹皮2两，肉桂2两，玄胡2两，黄耆2两，泽兰2两，卷柏2两，牛膝（酒浸）2两，香附子（炒）2两，白术2两，甘草2两，没药（另研）2两，吴茱萸（炒）2两，木香1两，薯蓣1两，山茱萸1两，桂心1两，石斛1两半（去根），钟乳粉3分，藁本1两，五味子1两。
出处：《普济方》卷三二二引《德生堂方》。
主治：冲任虚损，下元久冷，脐腹(疒丂)痛，月水不调，或前或后，或多或少，过期不来，或来时崩下，或月内再行，淋沥不止，带下五色，经脉时至，肢体倦怠，饮食不进，渐至羸瘦。及子宫久寒，不成孕。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
