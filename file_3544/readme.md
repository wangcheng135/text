序号：3544
名称：少阳丹
组成：苍术（乃天之精也。米泔浸半日，先刮去皮，晒干，捣罗细末）1斤，地骨皮（乃地之精也。温水洗，捶打遍，去心，晒干，捣罗细末）1斤，桑椹（乃人之精也。用黑熟者）20斤（入瓷盆内揉烂，绢袋内榨汁，去滓）。
出处：《扶寿精方》。
主治：None
加减：None
功效：服至1年，发白返黑；3年面有童颜，寿年无算。健脾去湿，息火消痰，久服身轻。
用法用量：每服10丸，无灰酒送下，1日3次。
制备方法：将前二末投在汁内，调匀，放大瓷盆内晒干，从朝至暮49日，采日之精；夜有月明时，高置净露台上，采月之华，亦须49度。复捣为末，炼蜜为丸，如小赤豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
