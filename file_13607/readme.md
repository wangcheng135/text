名称：龙胆丸
组成：龙胆3两，柴胡（去苗）2两，秦艽（去苗土）2两，萎蕤2两，升麻2两，百部2两，枳实（麸炒，去瓤）2两，玄参2两，旋复花2两，马兜铃2两，百合2两，紫菀2两，射干2两，木香2两，防风（去叉）2两，黄芩（去黑心）2两，栀子仁2两，大黄（锉，炒）2两，栝楼实2两，沙参2两，贝母（去心）2两，消石（研）2两，凝水石茵2两，陈蒿2两，石膏（碎）2两，知母（切，焙）2两，青蒿2两，款冬花2两，麦门冬（去心，焙）2两。
