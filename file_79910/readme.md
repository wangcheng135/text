序号：79926
名称：磁石丸
组成：磁石2两（烧，醋淬7遍，细研，水飞过），柏子仁1两，黄耆1两（锉），防风3两（去芦头），干姜半两（炮裂，锉），白茯苓1两，远志2分（去心），桂心3分，附子1两（炮裂，去皮脐），地骨皮半两，巴戟1两，牛膝1两（去苗），熟干地黄1两，覆盆子2两，鹿茸2两（去毛，涂酥，炙微黄），肉苁蓉1两（酒浸1宿，刮去皱皮，炙令干）。
出处：《圣惠》卷三十三。
主治：眼见黑花，冲风泪出，远视不明。内外障翳，不问远近。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，入磁石研令匀，炼蜜和为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌动风炙煿物。
附注：None
