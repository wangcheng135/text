序号：69185
名称：黄帝护命千金丸
组成：野葛7寸（炙），斑蝥20枚（去足翅，熬），雄黄1两（研），雌黄1两，鬼臼1两，瓜丁1两，丹砂1两（研），礜石1两（泥裹，烧半日），沙参1两，莽草1两（炙），椒（去目，汗）1两，地胆15枚（去足翅，熬）。
出处：《外台》卷十三引《古今录验》。
主治：羸瘦历年，胸满结疹，饮食变吐，宿食不下，中风鬼疰疾瘦。
加减：None
功效：None
用法用量：每服5丸，1日2次。卒中恶气绝不知人，服如小豆大2丸，老小半之；牛马所触践痈肿，若虫毒所啮，取1丸著掌中，唾和涂疮中毒上，立愈；正月旦以酒率家中大小各1丸，1岁不病；若伤寒身热，服1丸；若欲视病，服1丸，与病者共卧不恐。
制备方法：上药治下筛，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌生血物。
附注：None
