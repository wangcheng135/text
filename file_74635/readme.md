序号：74651
名称：解钳丸
组成：木鳖子2两（净，去油，用木通2两，切碎，同炒焦黑色，去木通），茴香2两（用斑蝥21枚，去头翅足，同炒香熟，去斑蝥），黑丑2两（用萝卜子2两，略研碎，同炒香熟，去萝卜子），半夏2两（用薛荔2两，向阳者，锉碎，同炒黄色，去薛荔），补骨脂2两（用猪芩2两，锉碎，同炒，去猪苓）。
出处：《赤水玄珠》卷十五。
主治：小肠气及膀胱气，横痃竖痃，木肾偏坠，下部诸疝气痛。
加减：None
功效：None
用法用量：每服30丸，空心炒姜盐灯心酒送下，午后、临睡各1服。
制备方法：上为末，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
