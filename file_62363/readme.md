序号：62379
名称：旋覆花汤
组成：旋覆花1两，赤茯苓（去黑皮）1两，犀角屑1两，紫苏茎叶（锉）1两，桂（去粗皮）半两，陈橘皮（汤浸，去白，焙）1两，前胡（去芦头）2两，白前1两。
出处：《圣济总录》卷八十二。
主治：脚气攻心，烦闷至甚者。
加减：如小便涩者，加桑根白皮2两；胸膈气满者，加半夏2两，以小便利，腹中气和，脚肿消为度。皮肤犹如隔帛者，宜服犀角麻黄汤。
功效：None
用法用量：每服5钱匕，水1盏半，入生姜半分（拍碎），大枣2枚（擘破），香豉半合，同煎至8分，去滓温服；如人行20里再服，即气下。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
