序号：58382
名称：真碓儿眼药
组成：炉甘石2两，朴消1两半（2味竭内固济，烧1日，水飞末，出火毒），黄丹1两，密陀僧1两（水飞），朱砂半两，铜绿半两，青盐3钱，玄精石3钱，硇砂3钱，硼砂3钱，白丁香（雄者）3钱，轻粉2钱。
出处：《永乐大典》卷一一四一三引《经验普济加减方》。
主治：远年日近，翳膜瘀肉，昏晕隐涩不明。
加减：None
功效：磨消翳膜。
用法用量：每夜点2-3箸，觉眼内及涩是验，便睡至早晨。觉眼微效，频点；点有涩难开莫怪，是效，点10日验。
制备方法：上为细末；用黄连1两碎，水1碗，熬煮至1盏，滴水不散，去滓，下白沙蜜1斤，用慢火熬，入前药拌匀，文武火熬至稀稠得所。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
