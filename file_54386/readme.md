序号：54401
名称：柴胡复生汤
组成：藁本3分半，川芎3分半，白芍药4分，蔓荆子3分半，羌活3分半，独活3分半，白芷3分半，柴胡6分，炙草4分，薄荷4分，桔梗4分，五味子20粒，苍术5分，茯苓5分，黄芩5分。
出处：《原机启微》卷下。
主治：目红赤羞明，泪多眵少，脑顶沉重，睛珠痛应太阳，眼睫无力，常欲垂闭，不敢久视，久视则酸疼，翳陷下，所陷者或圆或方，或长或短，如镂如锥如凿。
加减：None
功效：None
用法用量：作1服。水2盏，煎至1盏，去滓，食后热服。
制备方法：None
临床应用：None
各家论述：以藁本、蔓荆子为君，升发阳气也；川芎、白芍药、羌活、独活、白芷、柴胡为臣，和血补血疗风，行厥阴经也；甘草、五味子为佐，为协诸药，敛脏气也；薄荷、桔梗、苍术、茯苓、黄芩为使，为清利除热去湿，分上下，实脾胃二土，疗目中赤肿也。此病起自七情五贼劳役饥饱，故使生意下陷，不能上升，今主以群队开发，辅以和血补血，导入本经，助以相协收敛，用以清利除热实脾胃，如此为治，理可推也。
用药禁忌：None
附注：柴胡复明汤（《医统》卷六一）。
