序号：47325
名称：祛风大丸
组成：芎1两，赤芍药1两，防风（去芦）1两，白僵蚕（直者，炒去丝）1两，天麻1两，麻黄（去节）1两，朱砂（研，水飞）1两，石膏1两，龙齿（煅，别研）半两，白花蛇（好酒浸，取肉）半两，甘草半两，川大黄2钱，蝎梢（炙）3钱，麝香3钱（别研）。
出处：《魏氏家藏方》卷一。
主治：一切风痰，手足麻痹，语言謇涩，痰涎壅盛，头目眩晕，耳鸣忪悸，举动艰难，口眼斜，半身不遂，牙关紧急，不省人事等。
加减：None
功效：None
用法用量：每服1丸，用生姜自然汁化开，却用温汤浸，食后服，1日2次。
制备方法：上为细末，炼蜜为丸，每两作5丸。
临床应用：None
各家论述：None
用药禁忌：忌食酒、面、鸡、鱼、一切海鲜。
附注：None
