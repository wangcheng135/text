序号：25911
名称：收带六合丸
组成：白术2两（米泔浸，焙），苍术2两（米泔浸，焙），白茯苓2两（去皮），陈皮2两（盐水洗，去白），当归2两（酒洗），白芍（酒炒）2两，熟地黄1两半（酒洗），半夏（姜制）1两半，椿根白皮1两2钱（洗，炒），牡丹皮1两2钱，黄柏（酒炒）1两2钱，防风9钱，甘草（炙）1两，升麻8钱（一方加香附、枳壳）。
出处：《回春》卷六。
主治：赤白带下，肚腹疼痛。
加减：None
功效：和脾胃，燥中宫之湿，提下陷之气，化痰清火。
用法用量：每服100丸，空心米汤下；盐汤亦可。
制备方法：上为末，酒糊丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：益气固肠丸（原书同卷）、收带丸（《简明医彀》卷七）。
