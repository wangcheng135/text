序号：70530
名称：紫金丹
组成：胆矾3两，黄蜡1两，青州枣50个。
出处：《本事》卷三。
主治：男子、妇人患食劳、气劳，遍身黄肿，欲变成水肿；及久患痃癖，小肠膀胱，面目悉黄。
加减：None
功效：None
用法用量：每服20-30丸，茶、酒任下，如久患肠风痔漏，陈米饮送下。
制备方法：上于瓷盒内用头醋5升，先下矾、枣，慢火熬半日，取出枣去皮核，次下蜡一处，更煮半日如膏，入好腊茶末2两同和为丸，如梧桐子大。
临床应用：None
各家论述：《本事方释义》：胆矾气味咸酸微凉，入足太阳、阳明；黄蜡气味甘平微温，入手足太阴、阳明；腊茶气味苦寒直降，欲其速下也。炼药以醋者，约之也；送药以米饮者，扶中也。此治脱力劳伤，饥饱不调，宿有痃癖，周身发黄，欲变成水蛊，非渗湿之药不能引药入于病所，故效验独捷耳。酒客发黄便血尤宜服此药。
用药禁忌：None
附注：胆矾丸（《卫生宝鉴》卷十四）。
