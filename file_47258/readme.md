序号：47273
名称：砂淋丸
组成：黄色生鸡内金1两，生黄耆8钱，知母8钱，生杭芍6钱，硼砂6钱，朴消5钱，消石5钱。
出处：《袁中参西》上册。
主治：砂淋，亦名石淋。
加减：None
功效：None
用法用量：每服3钱，食前开水送服，1日2次。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：鸡内金为鸡之脾胃，原能消化砂石；硼砂可为金银铜焊药，其性原能柔五金，治骨鲠，故亦善消硬物；朴消《本经》谓能化七十二种石，消石《本经》不载，而《别录》载之，亦谓其能化七十二种石，想此二味，性味相近，古原不分，即包括于朴消条中，至陶隐居始别之，而化石之能则同。然诸药皆取消破之品，恐于元气有伤，故加黄耆以补气分，气分壮旺，益能运化药力。犹恐黄耆性热，与淋证不宜，故又加知母、芍药以解热滋阴，而芍药之性，又善引诸药之力以至膀胱也。
用药禁忌：None
附注：None
