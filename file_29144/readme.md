序号：29152
名称：阿胶散
组成：阿胶（炒）、桑寄生、吴白术、人参、白茯苓各等分。
出处：《活人书》卷十九。
主治：妊娠劳役过度，喜怒不常，服饮失时，冒触风冷，遂至胎动不安，腰腹俱痛，胞漏下血，疲极眩晕。
加减：None
功效：安胎。
用法用量：每服2钱匕，煎糯米饮调下，1日2次。
制备方法：上为细末。
临床应用：None
各家论述：《医略六书》：妊娠伤寒，邪热已解，里气虚馁，散胎孕因之不安而潮热焉。人参扶元补气以承载其胎，阿胶补阴益血以滋养其胎；白术健脾生血以安胎，寄生补肾强腰以护胎；茯苓清治节以安子室也。为散米饮下，使气阴内充，则冲任融和，而胎得所养，胎无不安，何潮热胎动之不已哉。
用药禁忌：None
附注：阿胶糯米白术汤（《鸡峰》卷十六）、阿胶汤（《妇人良方》卷十四）、安胎阿胶散（《卫生宝鉴》卷十八）。
