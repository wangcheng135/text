序号：27380
名称：补益柏子仁丸
组成：柏子仁1两，防风半两（去芦头），续断1两，桂心3分，白茯苓1两，羚羊角屑3分，牡丹半两，人参半两（去芦头），当归半两（锉，微炒），黄耆3分（锉），白术半两，枳壳半两（麸炒微黄，去瓤），赤芍药半两，木香半两，附子1两（炮裂，去皮脐），细辛3分，羌活3分，芎3分，牛膝1两（去苗），熟干地黄1两。
出处：《圣惠》卷七十。
主治：妇人风虚劳损，下焦伤冷，膈上风痰，头目旋眩，或时吐逆，心胸烦躁，不思饮食。
加减：None
功效：None
用法用量：每服30丸，空心及晚食前以温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
