序号：72609
名称：温经益元汤
组成：熟地黄、人参、白术、黄耆、芍药、甘草、当归、生地、茯苓、陈皮、肉桂、大附子。
出处：《伤寒六书》卷四。
主治：伤寒汗后大虚，头眩振振欲擗地，肉瞤筋惕；及因发汗太过，卫虚忘阳，汗出不止；或下后痢不止，身疼痛者。
加减：如饱闷，加枳壳，去地黄；瘦人去芍药；有热，去附子、肉桂；利不止，加炒白术、升麻，陈壁土，去当归，地黄；呕者，加姜汁制半夏；渴者，加天花粉；汗后恶风寒，属表虚，去附子、肉桂、生地，加桂枝、胶饴。
功效：None
用法用量：上以水2钟，加生姜3片，大枣1枚，糯米1撮，捶法煎之温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
