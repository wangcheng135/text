序号：63350
名称：清咽散
组成：甘草、桔梗、荆芥、防风、牛蒡、枳壳、薄荷、前胡。
出处：《喉科家训》卷二。
主治：一切咽喉肿痛，或红或白，形寒恶热，头疼身痛，汗少不得宣达，风痰壅塞，汤饮难咽。
加减：郁热痰多，加川贝、蒌仁；食滞不快，加神曲、谷芽；呃逆，加橘络、竹茹；便泻，加葛根、荷叶；血热，加丹皮、栀子；热甚，加黄芩、黄连；火毒，加银花、连翘；便秘，加青宁、明粉；溺赤，加赤苓、木通；胸下痞闷，加川朴；咳嗽，加杏仁、杷叶；秽浊、加佩兰、去甘草；痉厥动风，加羚羊、钩藤。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
