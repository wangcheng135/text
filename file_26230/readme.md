序号：26238
名称：百嚼丸
组成：槐角（炒）半斤，槐花（炒）半斤，桔梗（炒）半斤，薄荷叶（去土）半斤，蝉蜕（净洗）半斤，荆芥穗4两，甘草4两（炙），枳壳（麸炒，去瓤）4两，白僵蚕（炒去丝嘴）4两，川芎2两，羌活（去头芦）2两，防风（去头芦）2两，香白芷2两，白茯苓（去皮）2两，天麻1两（去苗），细辛（去叶土）半两，藁本（去土）半两，白附子（炮）半两，细松烟墨（烧红，醋淬）半两。
出处：《杨氏家藏方》卷二。
主治：风壅涎实，头目昏晕，眼多紧涩，肌肉瞤动，手足烦热，浑身疼痛，腰重脚弱，大便多秘，夜间少睡。
加减：None
功效：None
用法用量：每服1丸，食后、临睡细嚼，茶清送下。
制备方法：上为细末，炼蜜为丸，每1两作10丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
