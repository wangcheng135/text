序号：74045
名称：暖宫丸
组成：当归（洗，焙）2两，续断1两，藁本（去土）1两，吴茱萸（汤洗7遍，焙干）1两，五味子1两，人参（去芦头）1两，白茯苓（去皮）1两，白术1两，绵黄耆（蜜炙）1两，川芎1两，香白芷1两，缩砂仁1两，干姜（炮）1两，萆薢（酒浸1宿）1两，石斛3两（去根），牡蛎（煅通红，研碎）2两，香附子（炒）2两，熟干地黄（洗，焙）2两，山药2两，菟丝子（好酒煮软，焙7分干，砂盆内研碎，焙干）2两，羌活（去芦头）2两，白龙骨（别研）2两，茴香1两半（炒），山茱萸（去核）半两，延胡索半两，川椒半两（炒出汗）。
出处：《杨氏家藏方》卷十五。
主治：冲任脉弱，经候不调，因成带下；妊娠不牢，久无子息，日渐羸瘦，手足烦热，欲变骨蒸。
加减：None
功效：大益气血。
用法用量：每服50丸，空心食前，温酒或醋汤送下。
制备方法：上为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
