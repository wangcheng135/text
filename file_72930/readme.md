序号：72946
名称：湿热壅遏汤
组成：甘草梢15g，生地15g，桔梗6g，柴胡6g，连翘9g，赤芍9g，桃仁9g，当归12g，红花3g，土茯苓30g。
出处：《古今名方》引《老中医经验选编》。
主治：狐惑病（口、眼、生殖器综合征）。眼红赤疼痛，口腔舌侧溃疡，生殖器或阴部亦有溃烂。
加减：若眼赤不退，加木贼草、刺蒺藜各9g，车前子6g；目赤已退，适逢经至，去土茯苓，连翘，加川芎、益母草；阴部外用牛黄青黛散（人工牛黄、冰片各1.5g，青黛6g，橄榄核30g，煅存性，煅西月石9g，共研细末）外擦溃疡处。
功效：清热解毒、活血利湿。
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
