序号：68375
名称：黄耆膏
组成：黄耆1分，白芷1分，白及1分，白薇1分，当归1分，芍药1分，防风1分（去叉），甘草1分，细辛1分（去苗叶），嫩桑枝1分，垂柳枝2两，乳香（研）1分，铅丹6两，清麻油1斤。
出处：《圣济总录》卷一三五。
主治：灸疮。
加减：None
功效：止痛，抽火毒，吮脓。
用法用量：用薄纸上涂贴疮，日2次；或脓水多，易药时，用葱汤软帛浸洗，贴膏。以愈为度。
制备方法：上除乳香、油、铅丹外，细锉，以油浸1宿，次日煎候白芷黄黑色，绵滤去滓，下铅丹，以柳蓖搅候变黑色，滴水中为珠子，即入乳香末，足搅令匀，以瓷盒盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：黄耆散（《普济方》卷二七七）。
