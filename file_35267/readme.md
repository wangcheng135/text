序号：35277
名称：乳香散
组成：肉桂3两，干姜3两，牛膝4两，羌活4两，白芷2两，川芎4两，细辛4两，姜黄4两，骨碎补6两，当归6两，芍药4两，草乌4两，川乌4两，苍术6两，桔梗10两，赤小豆1升，乳香半斤（别研），没药5两（别研），何首乌14两，木鳖（去壳，麸炒）6两（一方去木鳖，加海桐皮）。
出处：《理伤续断方》。
主治：跌扑伤损，皮肉破绽，筋骨寸断，败血壅滞，结肿烂坏，疼痛至甚；或劳役所损，背肩四肢疼痛，损后中风，手足痿痹，不能举动，筋骨乖纵，挛缩不舒。
加减：None
功效：续筋接骨，结血止疼，生力。
用法用量：每服2钱，温汤调下，不拘时候。
制备方法：上焙，为本，续入乳、没末和汤。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
