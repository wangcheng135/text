序号：3390
名称：太一膏
组成：赤芍药1两，大黄1两，香白芷1两，官桂1两，玄参1两，当归1两，生干地黄1两。
出处：《百一》卷二十。
主治：一切恶疮。已成脓未成脓者。并治蛇、虎、蝎、犬、汤火、刀斧所伤。
加减：None
功效：None
用法用量：发背，先以温水洗疮，拭干，用帛子摊膏药贴之，温水送下1丸；久远瘰疬，摊贴，温水送下1丸；诸瘘疮，盐汤洗，贴，酒送下1丸；打扑伤损，摊贴，橘皮汤送下1丸；腰膝疼痛，盐汤送下1丸；妇人血气，木通、甘草场送下1丸；赤白带下，酒送下1丸；唾血，桑白皮汤送下1丸；风赤眼，摊贴，栀子汤送下1丸；咳嗽，咽喉肿，绵裹1丸含化；一切风劳病，柴胡汤送下1丸；一切疮疖并肿痛，及诸般疥痨，别炼入油少许，打膏令匀，涂之。其它诸疾亦度情而用。
制备方法：上锉。先煎清油2斤令香，候沫尽，即入药煎至黑色，取出不用，将油滤过，然后入黄丹1斤，用青柳枝不住手搅，滴于水中成珠不粘手为度，倾入瓷器中，以砖盖口，掘窖子埋树荫下，以土覆3日出火毒，丸如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：神仙太一膏(《局方》卷八吴直阁增诸家名方)、太乙膏(《证治要诀类方》卷四)、神效太乙膏(《保婴撮要》卷十六)、太乙灵应膏(《外科经验方》)、太乙清凉膏(《饲鹤亭集方》)、太乙膏丸(《杂病源流犀烛》卷三)。
