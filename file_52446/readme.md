序号：52461
名称：香汗丸
组成：牛黄（研细）1分，麝香（研细）1分，朱砂（研细）1分，龙脑（研细）1分，干蝎（微炒）1分，鹿角胶（捣碎，炒令黄燥）1分，白僵蚕（微炒）1分，白花蛇肉（涂酥炙微黄）1两，白附子（炮裂）1两，白蒺藜1两，天麻1两，赤茯苓1两，白芷1两，羌活1两，独活1两，蔓荆子1两，麻黄（去根节）1两，汉防己1两，木香1两，乌蛇肉（涂酥，炙微黄）1两，槟榔1两，藁本1两，防风（去芦头）1两，芎藭1两，当归1两。
出处：《圣惠》卷二十五。
主治：一切风。
加减：None
功效：None
用法用量：每服3丸，以温酒嚼下。服经1月，汗出皆香，经100日，洗面水自香。凡有风气，随汗而出。如是中风者，薄荷酒研下5丸。
制备方法：上为末，入研了药，更研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
