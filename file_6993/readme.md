序号：6993
名称：四时药茶
组成：川羌活1两5钱，法半夏3两，北杏仁2两（去皮尖，炒），漂茅术2两，紫川朴2两，尖川贝2两，软秦艽2两，明玉竹3两，陈建曲3两，正川芎2两，广陈皮1两5钱，藿香叶1两，煨天麻1两5钱，芽桔梗2两，苏扁豆3两，香白芷1两5钱，陈枳壳1两6钱，苏薄荷1两，北防风2两，结云苓3两，薏苡仁3两，白归身3两，京赤芍2两，飞滑石3两。
出处：《寿世新编》。
主治：风寒外感，发热恶寒，头目胀疼，腰脚痠痛，伤风咳嗽，鼻涕流清；以及食积痰滞，呕吐泄泻，饮食无味，似疟非疟，汗出不彻，一切四时不正之气。
加减：None
功效：祛风逐湿，清热散寒，宽胸导滞，和气化痰，汗不伤元，攻不克正。
用法用量：临用时再取大撮，开水泡服。汗若出透，不可再进，病自轻愈。
制备方法：外加淡姜片2两，大红枣50个，同煎，除姜、枣，须选道地咀片，照戥依制，将大铜锅煮取浓汁，铁锅亦可用，仍须擦净油；再将红茶叶5-6斤或7-8斤袭入炒匀，取起另烘干，庶免伤火，候冷，瓷罐收贮，封紧勿走药性，如走药性或兼受霉，恐不应验。
临床应用：None
各家论述：None
用药禁忌：孕妇虚入亦可用之，但病久化热，唇焦舌赤，汗大出，口大渴者，则不可服。
附注：None
