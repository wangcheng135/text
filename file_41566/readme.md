序号：41581
名称：金粟丸
组成：叶子雌黄一两。
出处：《证类本草》卷四引《胜金方》。
主治：久嗽，暴嗽，劳嗽。
加减：None
功效：None
用法用量：每服3-5丸，甘草水服，服后睡良久妙。
制备方法：上为细末，用纸筋泥固济小盒子1个，令干，勿令泥厚，将泥入盒子内，水调赤石脂封盒子口，更以泥封之，候干，坐盒子于地上，上面以未入窑瓦坯子弹子大壅盒子，令作一尖子，上用炭10斤，簇定顶上，着火一熨斗笼起，令火从上渐炽，候火消3分去1，看瓦坯通赤，则去火候冷，开盒子取药，当如镜面光明红色，入乳钵内细研，汤浸蒸饼心为丸，如粟米大。
临床应用：None
各家论述：None
用药禁忌：None
附注：雌黄方（《普济方》卷一五七），雌黄丸（《准绳·类方》卷二）。
