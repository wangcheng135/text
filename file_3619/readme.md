序号：3619
名称：巴戟丸
组成：巴戟天（去心）1两半，肉苁蓉（酒浸，去皴皮，切，焙）2两，牛膝（去苗，同苁蓉酒浸）1两，山芋1两，杜仲（去粗皮，炙，锉）1两半，续断1两，蛇床子1两，菟丝子（酒浸，焙，别捣）1两1分，白茯苓（去黑皮）1两，山茱萸1两1分，五味子1两1分，远志（去心）1两。
出处：《圣济总录》卷九十二。
主治：虚劳，肾气衰弱，小便白浊，阴囊湿痒；羸瘦多忘，面无颜色；男子阳道衰弱。
加减：如精涩，更加柏子仁3分，如精虚，加五味子1两半，阳弱加续断1两半。
功效：服药50日后，筋骨健壮，百日后面如童颜，久服令人精满充溢；令人多子。
用法用量：每服30丸，空心温酒送下，日晚再服。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：《御药院方》有益智仁一两。
