序号：62391
名称：旋覆花汤
组成：旋覆花半两，半夏（汤洗去滑，炒）3两，陈橘皮（汤浸，去白，焙）3分，杏仁（去皮尖双仁，炒）30枚。
出处：《圣济总录》卷八十三。
主治：脚气。呕逆不下食，行坐不安。
加减：若腹中胀满，食不消者，加槟榔3枚（锉）；大便难坚者，加大黄1两；不能食者，加白术1两半；胸中寒热闷者，加羚羊角、犀角屑、青木香各半两；心下坚者，加鳖甲1两（醋炙，去裙襕），防葵、芍药各半两。
功效：None
用法用量：每服3钱匕，水1盏。入生姜1枣大（拍破），同煎至6分，去滓温服，空心、日午、近晚各1次。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
