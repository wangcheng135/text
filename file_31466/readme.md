序号：31475
名称：杜仲丸
组成：杜仲（去粗皮，炙，锉）3分，防风（去叉）3分，附子（炮裂，去皮脐）3分，石菖蒲3分，桔梗（炒）3分，秦艽（去苗土）3分，细辛（去苗叶）3分，厚朴（去粗皮，生姜汁炙）3分，桂（去粗皮）3分，半夏（汤洗2-7遍，焙）3分，熟干地黄（焙）半两，沙参半两，蜀椒（去目并闭口者，炒出汗）半两，干姜（炮）半两。
出处：《圣济总录》卷一五七。
主治：子宫久冷，妊娠数堕胎。
加减：None
功效：None
用法用量：每服15丸，渐加至20丸，空心温酒送下。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
