序号：44142
名称：复春丸
组成：萆薢4两，破故纸（炒）2两，杜仲（去丝）2两，胡芦巴（炒）2两，木通2两，骨碎补（去毛）1两，虎骨（酥炙）1两，乳香（研）1两，槟榔1两，没药1两，木香1两，甜瓜子（炒）2两，牛膝（去芦，酒浸，焙干）2两，巴戟（去心）2两，胡桃仁100个（去皮，另研极细末），黑附子1两（炮）。
出处：《瑞竹堂方》卷二。
主治：腰脚风湿劳损，手足麻痹，筋骨疼痛，不能屈伸。
加减：None
功效：活血驻颜，补虚。
用法用量：每服40-50丸，食前温酒送下。
制备方法：上为细末，再与胡桃仁同研极细，酒糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
