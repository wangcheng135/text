序号：5346
名称：宁志膏
组成：人参（去芦）1两，酸枣仁（微炒，去皮，研）1两，辰砂（水飞）半两，乳香1分（以乳钵坐水盆中研）。
出处：《本事》卷二。
主治：惊恐失志，心气虚耗，健忘，失眠，癫狂，赤白浊，睡卧不宁，梦涉危险，一切心疾;心气虚耗，赤白浊甚;癫狂失心不寐。
加减：None
功效：宁神定志，安眠止痛。
用法用量：每服1丸，薄荷汤化下。
制备方法：上为细末，炼蜜为丸，如弹子大。
临床应用：失心：予族弟妇缘兵火失心，制此方与之，服20粒愈。
各家论述：1.《寿世保元》：此方朱砂能镇心安神；酸可使收引，故枣仁能敛神归心；香可使利窍，故乳香能(害炎)痰达心志；许学士加人参，亦谓人参能宁心耳。_x000D_
2.《本事方释义》：人参气味甘温，入脾胃；枣仁气味苦平，入心；辰砂气味苦温，入心；乳香气味辛微温，入手足少阴。以薄荷汤送药，乃手太阴之引经药也；甘温护持中土，佐以苦味入心，辛香开窍，使以轻扬为引，表里皆得安妥矣。
用药禁忌：None
附注：本方方名，据剂型，当作“宁志丸”。
