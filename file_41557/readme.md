序号：41572
名称：金黄散
组成：大黄（锉，炒）半两，郁金（锉，炒）半两，鲮鲤甲（炙）半两，谷精草半两，龙骨半两，山栀子仁半两，木鳖子（去壳）半两，独角仙（皂荚株上黑虫）半两，乌贼鱼骨（去甲）半两，黄柏（去粗皮）半两，甘草（锉）半两，铅丹半两，白蔹半两，不灰木半两，麒麟竭（研）半两，黄芩半两，腻粉1分，藜芦（去苗）1分。
出处：《圣济总录》卷一三二。
主治：积年恶疮，及透掌漏疮，外臁疮。
加减：None
功效：None
用法用量：每看疮大小掺之，有脓水即用温盐浆水洗净敷之，透掌漏疮，以津调纳于疮内，不过3-5上。
制备方法：上为散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
