名称：仙方膏
组成：白芷2两，紫荆皮2两，独活2两，石菖蒲2两，赤芍2两，高良姜5钱，蜈蚣5钱，刺猬皮5钱，蛇蜕草5钱，麻仁5钱，鳖甲5钱，白僵蚕5钱，甘草5钱，海风藤5钱，连翘5钱，天花粉5钱，白及5钱，牛蒡子5钱，大黄5钱，川黄连5钱，白蔹5钱，当归5钱，千金子5钱，血余5钱，金银花5钱，黄柏5钱，穿山甲5钱，防己5钱，猪牙皂5钱，柴胡5钱，川贝母5钱，桃仁5钱，白附子5钱，巴豆5钱，明天麻5钱，苦参5钱，荆芥穗5钱，红花5钱，黄耆5钱，桔梗5钱，黄芩5钱，牛膝5钱，防风5钱，全蝎5钱，麻黄5钱，草乌5钱，肉桂5钱，乌药5钱，羌活5钱，半夏5钱，大戟5钱，苏木5钱，桃枝1寸长，槐枝1寸长，桑枝1寸长，柳枝1寸长，各24段。
出处：《疡医大全》卷七引《经验方》。
主治：痈疽发背，一切外症，并贴五劳七伤，筋骨疼痛，跌打损伤，妇人症瘕、带下。
加减：None
功效：None
用法用量：None
制备方法：用大磨真麻油13斤，将上药入油内泡7日，入铜锅内熬至药枯滤去滓，复将油仍入锅内，熬至滴水成珠，再撤净药脚，下丹。每油1斤，下飞过黄丹8两为则，药已成功，入有铕(钅敝)缸内，以槐棍搅冷，再入后末：血竭4钱，乳香（去油），没药（去油）各3钱3分，藿香4钱5分，研末搅匀，又入后药：珍珠、冰片各1钱，沉香（不见火）4钱7分，当门子2钱1分，木香（不见火），松香各5钱4分，檀香（不见火）6钱，雄黄5钱5分，搅匀，入潮脑3钱收功。
