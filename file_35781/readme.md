序号：35791
名称：参归补益汤
组成：人参（无者、或以山药3钱炒黄代之），蜜耆2-3钱，当归身2钱，白芍（酒炒）1钱半，大川芎8分，肉桂7分，山楂肉6分，熟地2-3钱。
出处：《会约》卷二十。
主治：痘疹气血两虚，脓不充满，白陷、灰陷等证。
加减：如气虚痒塌不起，加穿山甲（土炒）6分，或加白芷；如血热红紫不起，加紫草1钱半；如胃气虚寒多呕者，加干姜（炒用）1钱，或加丁香4分；如元气大虚，寒战咬牙，去白芍，加附子1钱半，干姜（炒用）1钱；如泄泻，加白术1钱半，肉豆蔻1钱。
功效：None
用法用量：水煎服。或加糯米、人乳（泄者不用），好酒合服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
