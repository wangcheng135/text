序号：37361
名称：宝珍膏
组成：生地3钱，苍术3钱，枳壳3钱，五加皮3钱，莪术3钱，桃仁3钱，山柰3钱，当归3钱，川乌（生）3钱，橘皮3钱，乌药3钱，三棱3钱，大黄3钱，首乌3钱，草乌（生）3钱，柴胡3钱，防风3钱，刘寄奴3钱，猪牙皂3钱，川芎3钱，官桂3钱，羌活3钱，威灵仙3钱，赤芍3钱，南星（生）3钱，香附（生）3钱，荆芥3钱，白芷3钱，海风藤3钱，藁本3钱，续断3钱，良姜3钱，独活3钱，麻黄3钱，甘松3钱，连翘3钱。
出处：《北京市中药成方选集》。
主治：风寒麻木，筋骨疼痛，腰痠腿软，跌打损伤。
加减：None
功效：追风散寒，舒筋活血。
用法用量：用时微火化开，贴患处。
制备方法：上药酌予碎断，用麻油64两炸枯，去滓过滤，炼至滴水成珠，加入黄丹30两，搅匀成膏，取出浸入冷水中，去火毒后，加热溶化。另兑以下细料：肉桂1钱、麝香1钱、附子2钱、木香2钱、冰片3钱、茴香3钱、乳香3钱、没药3钱、细辛3钱。共研为细粉，过罗，搅匀摊贴。大张代光重1两2钱，中张重1两，小张6钱。
临床应用：None
各家论述：None
用药禁忌：孕妇忌贴。
附注：None
