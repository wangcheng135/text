序号：25559
名称：朱砂丹
组成：朱砂1两，麝香2两，山豆根1钱（生，杵），巴豆3分，雄黄1分，黄丹1分，斑螫1分（1半生，1半熟，以糯米炒），苦药子1分（炒1半），续随子1分，没药1分，蜈蚣2个（炒1个）。
出处：《鸡峰》卷二十五。
主治：None
加减：None
功效：解蛊毒。
用法用量：每服1丸，用无粉茶送下。凡中毒者，便下药，少时觉心头拽断皮条声，即是毒下，或从口中出，或从下部出，嫩即成血，老即成蜣螂、鳖子带命之物，因凝血裹出，即知当初下者毒物；其药泻出，便以洗过香汤再度仍以朱砂收之。每1丸可治男子5人，妇女3人，服药后转泻不止，白粥止之。
制备方法：上以糯糊为丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：忌一切秽恶。
附注：内中毒即服之，外中毒则醋磨涂之。如才觉中毒，先吃黑豆，觉其味香甜者，是中毒也。
