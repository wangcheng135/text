序号：50153
名称：茯神煮散
组成：茯神36铢，麦门冬36铢，通草30铢，升麻30铢，紫菀18铢，桂心18铢，知母1两，赤石脂42铢，大枣20个，淡竹茹（鸡子大）1枚。
出处：《千金》卷十三。
主治：心实热，口干烦渴，眼卧不安。
加减：None
功效：None
用法用量：以帛裹方寸匕，井花水2升半，煮取9合，时动裹子，为1服，每日2次。
制备方法：上为粗散。
临床应用：None
各家论述：《千金方衍义》：以肺燥不能胜热，故用麦门冬、桂心蒸发津气于上，又以升麻、通草上升下泄，辅佐清热导火之力。
用药禁忌：忌生葱、酢物。
附注：茯神汤（《圣济总录》卷四十三）。本方方名，《千金方衍义》引作“茯神散”。
