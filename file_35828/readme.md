序号：35838
名称：参术启脾丸
组成：人参（去芦）2两，白术（麸皮炒）4两，黄耆（蜜炙）2两，白茯苓（去皮）2两，山药（微炒）2两，甘草（炙）5钱，陈皮（去半白）1两，黄连（炒）8钱，法制半夏粉1两，砂仁1两，神曲（炒）5钱，白芍药（炒）1两5钱，山楂肉1两，藿香（水洗）3钱，麦门冬（炒，取末）5钱。
出处：《医便》卷五引钱雷方。
主治：None
加减：None
功效：补脾胃，益元气，壮精神，化痰涎，灵饮食。久服肥健延年。
用法用量：每服2钱，白汤下，空腹食远皆可服。
制备方法：上各为细末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
