序号：38128
名称：松节酒
组成：松节30斤（细锉，水4石，煮取1石），猪椒叶30斤（锉，煮如松节法）上澄清，合渍干曲5斤，候发，以糯米4石5斗酿之，依家酿法肆酘，勿令伤冷热，第一酘时下后诸药：柏子仁5两，天雄5两，萆薢5两，芎5两，防风10两，人参4两，独活15两，秦艽6两，茵芋4两，磁石12两（末）。
出处：《千金》卷八。
主治：历节风，四肢疼痛犹如解落。
加减：None
功效：None
用法用量：上(口父)咀，纳饭中炊之，如常酘法，酘足讫，封头28日，押取清。适性服之，勿至醉吐。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：松质坚劲，岁寒不凋，故能治风逐湿、坚骨强筋。松节和猪椒叶酿酒，后下柏子仁等除风湿药，所谓虽有汤药，不及松膏、松节酒也。
用药禁忌：None
附注：None
