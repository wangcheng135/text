序号：57545
名称：润肺丸
组成：百部2两，黄连1两，明矾1两，桑皮1两，使君子3两，鹤虱2两，楝根2两，甘草5钱。
出处：《医略六书》卷二十二。
主治：湿热生虫蚀肺，咳嗽不止，烦心恶热，脉数者。
加减：None
功效：杀虫，化湿热。
用法用量：每服3钱，滚水送下。
制备方法：上为末，白及膏糊丸。
临床应用：None
各家论述：湿热不消，变化生虫而侵蚀肺叶，故咳嗽不止，烦心恶热焉。黄连清热燥湿，百部温肺杀虫，明矾却湿以杜生虫之本，鹤虱杀虫以绝化虫之源，桑皮清肺气以肃金，甘草缓中州以和胃，使君子健脾气化积，楝根皮泻湿热杀虫，白及膏润补肺之残缺也。糊丸，滚汤下使湿热顿消则虫积自化，而肺损复完，安有咳嗽成劳之患乎，此杀虫化湿热之剂，为虫蚀肺叶咳嗽之专方。
用药禁忌：None
附注：None
