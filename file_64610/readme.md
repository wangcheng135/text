序号：64626
名称：甜菜膏
组成：甜菜3两，生地黄2两，猪脂2两，大戟（炒）1两，当归（切，焙）半两，续断半两，白芷半两，莽草半两，芎半两，防风（去叉）半两，甘草（炙）3分，芍药3分，蜀椒（去目并合口者，炒出汗）1分，细辛（去苗叶）1分，大黄（锉，炒）1分，杜仲（去粗皮，酥炙）1分，黄耆（炙，锉）1分，黄芩（去黑心）1分。
出处：《圣济总录》卷一八三。
主治：乳石发痈疽疮。
加减：None
功效：止痛生肌。
用法用量：涂敷疮上，每日3-5次。
制备方法：上除猪脂外，锉碎，先熬脂令沸，下诸锉药，煎候白芷赤色，绞去滓，瓷合盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
