序号：60136
名称：透空丸
组成：香附子1两，藁本1两，藿香叶1两，地龙（去土）1两，川芎1两，白僵蚕（炒）1两，干姜（炮）1两，甘草（炙）1两，干蝎1两，天麻（去苗）1两，天南星（生姜制）1两，白芷7分，神曲（碎炒）2两半，茴香（炒）2两半，麦糵（净，炒）2两半，胡椒1两，川乌头（炮制）1两2分。
出处：《御药院方》卷一。
主治：男子、妇人一切诸风顽麻疼痛，上攻头目，下注腰脚，手背颤动。
加减：None
功效：None
用法用量：每服1丸，细嚼，食前茶、酒任下。
制备方法：上为细末，每药末3两，白面6两，水为丸，如小弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
