序号：22432
名称：阴阳至圣丹
组成：金银花1斤，生地8两，当归3两，川芎2两，牛膝1两，丹皮1两，麦冬3两，生甘草1两，荆芥1两，防风5钱，黄耆3两，茜草根5钱，玄参5两，用麻油5斤煎数沸，将药滓滤出，再熬至滴水成珠，入下药：广木香1两，黄丹2斤（炒飞过，去砂），没药1两，乳香1两，血竭1两，象皮（为末）5钱，麝香1钱。
出处：《石室秘录》卷四。
主治：膏梁之客，失志之人，心肾不交，阴阳俱耗，又加忧愁抑郁，拂怒呼号，其气不散，结成阴症痈疽。
加减：None
功效：None
用法用量：发背疮必须用1两，其余疮口，量大小用之。
制备方法：上各为细末，入油中，少煎好，藏瓷罐内。
临床应用：None
各家论述：None
用药禁忌：None
附注：阴阳至圣膏（《洞天奥旨》卷十五）、阴阳起死膏（《外科十三方考》）。
