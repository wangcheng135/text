序号：53577
名称：圆明内障升麻汤
组成：干姜1钱，五味子2钱，白茯苓3钱，防风5钱，白芍药6钱，柴胡7钱，人参1两，炙甘草1两，当归身（酒洗）1两，白术1两，升麻1两，葛根1两，黄耆1两5钱，羌活1两5钱。
出处：《兰室秘藏》卷上。
主治：脾胃元气衰弱，心火与三焦俱盛，饮食不节，形体劳役，心不得休息，而致内障。内障初起，视觉微昏，空中有黑花，神水变淡绿色，次则视物成二，神水变淡白色；久则不睹，神水变纯白色。
加减：None
功效：None
用法用量：上(口父)咀。每服5-7钱，水3大盏，煎至2大盏，入黄芩、黄连2钱，同煎数沸，去滓，煎至1盏，食远热服。
制备方法：None
临床应用：None
各家论述：《准绳·类方》：因肝木不平，内挟心火，故以柴胡平肝、人参开心、黄连泻心火为君；酒制当归荣百脉，五味敛百脉之沸，心包络主血，白芍药顺血脉，散恶血，为臣；白茯苓泻膀胱之湿，羌活清利小肠之邪，甘草补三焦，防风开胆之降，为佐：阴阳皆总于脾胃，黄耆补脾胃，白术健脾胃，升麻、葛根行脾胃之经；黄芩退壮火，干生姜入壮火为导为使。
用药禁忌：None
附注：冲和养胃汤（原书同卷）、内障升麻汤（《杏苑》卷六）。
