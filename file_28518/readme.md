序号：28526
名称：芫花丸
组成：芫花半两（醋拌，炒令干），甘遂半两（煨微黄），甜葶苈1两（隔纸炒令紫色），川大黄1两（锉碎，微炒），枳壳1两（麸炒微黄，去瓤），大戟半两（锉碎，微炒），郁李仁1两（酒浸，去皮尖，微炒），海藻1两（洗去咸味），桂心1两，杏仁1两（汤浸，去皮尖双仁，锉，研如膏），巴豆30枚（去皮心，研，纸裹压去油，细研）。
出处：《圣惠》卷五十一。
主治：痰冷癖饮，腹中结聚成块；亦疗大腹水肿。
加减：None
功效：None
用法用量：每服3丸，空心以粥饮送下。
制备方法：上为末，入巴豆、杏仁同研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
