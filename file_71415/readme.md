序号：71431
名称：解毒丸
组成：山豆根3两，板蓝根2两，甘草4两（炙），山茨菇3两，土马鬃2两，续随子仁2两，黄药子2两，大黄1两，紫河车3两，木通2两，盆消2两，五味子2两，藿香2两，寒水石2两，雄黄2两，贯众2两，白僵蚕2两，干葛1两，茜草根1两，薄荷2两，绿豆粉3两，百药煎2两，朱砂1两，麝香半两。
出处：《普济方》卷二五一。
主治：诸恶物，蛊毒，砒毒，菌毒，河豚鱼毒。
加减：None
功效：解世间一切不测等毒。
用法用量：每服半丸，用生姜蜜水化下。
制备方法：上为细末，蒸饼为丸，如弹子大，用螺青3两和匀，一半为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：《奇效良方》无藿香，有蔻仁二两；《摄生众妙方》无寒水石、五味子，有石膏、五倍子。
