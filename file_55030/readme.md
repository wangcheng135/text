序号：55045
名称：桂枝去芍药汤
组成：桂枝3两（去皮），甘草2两（炙），生姜3两（切），大枣12枚（擘）。
出处：《伤寒论》。
主治：太阳病，下之后，脉促胸满者。
加减：None
功效：解肌祛风，去阴通阳。
用法用量：以水7升，煮取3升，去滓，温服1升。
制备方法：None
临床应用：外感咳嗽《临证指南医案》：某，44岁，寒热咳嗽，当以辛温治之，桂枝汤去芍加杏仁。
各家论述：①《尚论篇》：用桂枝之辛甘，以亟散太阳之邪；其去芍药之意，酸收二字不足尽之，以误下故不敢用，恐其复领阳邪下入腹中也。②《伤寒贯珠集》：邪气仍在阳分，故桂、甘、姜、枣甘辛温药，从阳引而去之；去芍药者，恐酸寒气味，足以留胸中之邪，且夺桂枝之性也。
用药禁忌：None
附注：None
