序号：47804
名称：神仙秘宝丹
组成：白花蛇头1枚（酒浸3日，焙干），乌蛇头1枚（酒浸3日，焙干），赤足蜈蚣2条（酒浸3日，炙），附子1枚（重6钱者，炮，去皮脐），白花蛇项后肉（离项7寸后取2两，酒浸3日，去皮骨，焙干）7钱，朱砂6钱（别研，纳2钱入药，4钱为衣），白僵蚕半两（直者，炒，去丝嘴），雄雀1枚（去毛肛肠，入硇砂1钱，用泥固济，晒干，用文武火煅，青烟出为度，别末），全蝎（去毒，炒）半两，天麻（去苗）半两，天南星（炮）半两，人参（去芦头）半两，沉香半两，五灵脂8钱（炒，别末），川芎1钱，脑子（别末）1钱，乳香（别研）1钱，没药（别研）1钱，牛黄（别研）1钱，血竭1钱，麝香（别研）1钱。
出处：《杨氏家藏方》卷一。
主治：一切中风，左瘫右缓，手足軃曳，牙关紧急，口眼斜，语言謇涩，昏塞如醉，或痛连骨髓，或痹袭皮肤，瘙痒顽痹，血脉不行，及小儿心肺中风，涎潮抽溺，妇人产后中风。
加减：None
功效：None
用法用量：每服1丸，空心温酒磨下；小儿急慢惊风者，以1丸分作4服，薄荷汤磨下。
制备方法：上为细末，入脑子等末，拌研极匀，用好无灰酒和丸，每1两作15丸，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：方中硇砂，《普济方》作“硼砂”。
