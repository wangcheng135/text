序号：41698
名称：金蝉脱壳方
组成：当归3两，川芎3两，防风3两，滑石3两，天麻3两，芍药1两5钱，桔梗1两5钱，僵蚕2两，大黄2两，人参2两，独活2两，山栀2两，黄连2两，白术2两，蝉蜕2两，黄芩2两，石膏2两，苦参4两，连翘2两，黄柏2两，细辛1两，荆芥3两5钱，羌活2两，全蝎2两，芒消1两，沉香1两，枫子肉4斤。
出处：《秘传大麻风方》。
主治：麻风。
加减：血盛，加泽兰叶、革巴萆、蚵蚾草各2两；麻木，加淫羊藿、天麻各2两：血少，加生地、熟地，血竭各2两：鼻塞不通，不闻香气，乃血气不行，不能贯通，以致滞塞，加片脑、朱砂、硼砂、牙消、玄明粉各等分为末。吹鼻内，1日3次即通。
功效：None
用法用量：每服100丸，1日3次，茶汤送下。
制备方法：先将枫子肉为膏，余药为末，用黄米饭打糊和膏，打千捶为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
