序号：78343
名称：酸枣仁煎
组成：酸枣仁（生用）3两，败龟（醋炙）1两，海桐皮（锉）2两，仙灵脾（去粗茎）1两，赤石脂1两，萆薢（炒）1两，羌活（去芦头）2两，虎骨（涂酥炙）1两半，蒺藜子（炒去角）1两，石斛（去根）1两，牛膝（去苗，酒浸，切，焙）1两，巴戟天（去心）1两，附子（炮裂，去皮脐）1两，木香1两，杜仲（去粗皮，炙，锉）1两，熟干地黄（焙干）1两，白蜜（次入）4两，牛酥（次入）1两半，桑枝1握（长1寸，锉）。
出处：《圣济总录》卷七。
主治：柔风，身体疼痛，行履不得；中风筋骨拘急。
加减：None
功效：None
用法用量：每服半匙许，温酒调下，空心、日午、夜卧服。
制备方法：上19味，将16味捣罗为末，后用清酒1斗，先煎桑枝令黄色，滤去桑枝，却下药末，更煎取沸，次下白蜜、牛酥，煎如稀膏，用瓷盒盛。
临床应用：None
各家论述：None
用药禁忌：None
附注：原书卷八有“赤箭”，无“赤石脂”。
