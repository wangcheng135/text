序号：60965
名称：大黄耆酒
组成：黄耆2两，桂心2两，巴戟天2两，石斛2两，泽泻2两，茯苓2两，柏子仁2两，干姜2两，蜀椒2两，防风2两，独活2两，人参2两，天雄1两，芍药1两，附子1两，乌头1两，茵芋1两，半夏1两，细辛1两，白术1两，黄芩1两，栝楼根1两，山茱萸1两。
出处：《千金》卷十五。
主治：肉极虚寒，为脾风，阴动伤寒，体重怠堕，四肢不欲举，关节疼痛，不嗜饮食。
加减：None
功效：None
用法用量：上(口父)咀，绢袋贮，以清酒3斗渍之，秋、冬7日，春、夏3日。初服3合，渐渐加，微痹为度，每日2次。
