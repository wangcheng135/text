序号：64840
名称：续命汤加紫苏陈皮方
组成：竹沥1升2合，生姜汁5合，生地汁1升，龙齿（末）4两，防风4两，麻黄4两，防己2两，附子（炮）2两，石膏2两，桂枝2两，陈皮（去白）半两，紫苏半两。
出处：《医方考》卷五。
主治：痫疾。发则仆地，闷乱无知，嚼舌吐沫，背反张，目上视，手足搐搦，或作六畜声者。
加减：None
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：是方也，有麻黄、桂枝、防风。紫苏，则可以泄在经之邪；有竹沥、姜汁、陈皮，则可以行痰涎之滞；有生地汁、石膏则可以清心肺之壅；有龙齿则可以安魂；有防己可以通塞；若夫沉痼之痰，非附子不足行其滞，而其大热之性，又足以益火之原而消阴翳，譬之太阳中天，幽谷之翳障无不消灭，此古人用附子之意也。
用药禁忌：None
