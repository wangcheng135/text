序号：46956
名称：独活酒
组成：独活（去芦头）1斤，金牙4两，细辛（去苗叶）4两，地肤子4两，莽草4两，熟干地黄（切，焙）4两，蒴藋根4两，防风（去叉）4两，附子（炮裂，去皮脐）4两，续断4两，蜀椒（去目并闭口，炒出汗）4两。
出处：《圣济总录》卷九。
主治：肉苛。肌肉不仁，遍身(疒帬)重。风不仁，不能行步。
加减：None
功效：None
用法用量：温服2合，每日3次，渐增之。其金牙为末，别以练囊盛，纳大囊中。
制备方法：上除金牙外，并细锉，盛以绢囊，清酒5升渍之，密泥器口，夏3宿，冬5宿，酒成。
临床应用：None
各家论述：None
用药禁忌：None
附注：金牙酒（原书卷十一）。
