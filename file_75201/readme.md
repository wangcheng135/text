序号：75217
名称：腽肭脐丸
组成：腽肭脐1两（酒刷，炙微黄），荜澄茄1两，附子1两（炮裂，去皮脐），泽泻3分，芎3分，沉香1两，石龙芮3分，肉豆蔻3分（去壳），牛膝1两（去苗），蛇床子3分，薯蓣1两，覆盆子1两，巴戟3分，槟榔1两，桂心1两，木香1两，麝香1两（细研），白术3分，远志3分（去心），石斛1两（去根，锉），补骨脂1两（微炒），山茱萸3分，肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），母丁香半两。
出处：《圣惠》卷九十八。
主治：None
加减：None
功效：补益丹田，固济水脏，安神益智，明目驻颜，壮腰膝，充肌肤，补虚冷，安脏腑。
用法用量：每服30丸，渐加至40丸，空心以温酒送下。
制备方法：上为末，入麝香研令匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
