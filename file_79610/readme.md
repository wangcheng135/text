序号：79626
名称：鳖甲三棱丸
组成：鳖甲（九肋，重4两以上者，水浸洗，去脊骨裙襕，醋浸1宿，炙，为末）3两，京三棱（水浸2宿，锉，醋浸1宿，焙干，为末）3两，干漆（炒烟出）3两，木香1分，干姜（炮）1分，补骨脂（炒）1分，槟榔（锉为末）1分，没药（研）1分，硇砂（研）1分，墨（研）1分。
出处：《圣济总录》卷七十三。
主治：男子、妇人、小儿虚中癖气，脏腑不调，食饮不消，久致瘦弱者；又治虚气膨胀，心胸闷滞；并妇人产后血积蓐劳，瘦瘁甚者。
加减：None
功效：None
用法用量：每服20丸，生姜、盐汤送下；妇人血病，醋汤送下。
制备方法：上为末，醋煮面糊为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
