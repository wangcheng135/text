序号：33104
名称：万应灵丹
组成：水银5钱，青盐5钱，皂矾1两，生铅2钱5分（与水银同研碎），生矾1两5钱，火消1两2钱5分，白砒1钱5分，硼砂1钱5分，明雄黄1钱5分。
出处：《疡医大全》卷七引《济生》。
主治：一切痈疽发背诸毒。有脓怕开刀者。疔疮对口，发背痈疽初起。
加减：None
功效：拔毒。
用法用量：以针挑破浮皮，用丹1厘，醋调点患处，即溃头出脓；或发背痈疽大毒，每用1厘，针挑破，醋调点患处，1日上3次。药性内攻，深可寸余，毒气有门而泄，则毒易消。如根盘大者，用丹5厘，川贝母末1钱，浓茶卤调敷周围，必起黄泡，自有黄水流出，其毒自消。
制备方法：上为极细末，入小瓦罐内，炖炭火上熔化，俟药枯结住罐底，用瓦盆1个，将有药罐倒置盆内正中，罐口以盐泥封固。另用一大盆盛水，将药罐安置水内，罐口四围以砖围罐半截，下衬冷灰、然后砖上及罐底俱架炭火，先从顶上着火，从上而下，先文后武，3炷香为度；冷定开看，盆内丹药刮下，研细，瓷瓶密贮。
临床应用：None
各家论述：None
用药禁忌：None
附注：万应灵膏（《何氏济生论》卷八）。
