序号：56836
名称：消乳岩丸
组成：夏枯草4两，蒲公英4两，金银花2两，漏芦2两，山茨菇1两5钱，雄鼠粪1两5钱，川贝母（去心）1两5钱，连翘1两5钱，金橘叶1两5钱，白芷1两5钱，甘菊花1两5钱，没药（去油）1两5钱，瓜蒌仁1两5钱，乳香（去油）1两5钱，茜草根1两5钱，甘草1两5钱，广陈皮1两5钱，紫花地丁1两5钱（1方去瓜蒌仁，加天花粉、桔梗、广胶，用夏枯草熬膏为丸。）。
出处：《疡医大全》卷二十。
主治：乳岩。
加减：None
功效：None
用法用量：每服2-3钱，早、晚食后送下。
制备方法：上为细末，炼蜜为丸。
临床应用：None
各家论述：None
用药禁忌：戒气恼。
附注：None
