序号：42992
名称：保惊丸
组成：明天麻1两，白附子5钱，防风8钱，蝉蜕8钱，荆芥1两，胆星1两5钱，半夏（法制者）8钱，白术1两，茯苓（去皮）1两，川芎8钱，粉草4钱，僵蚕（炒）5钱，土朱5钱，羌活8钱，当归5钱，全蝎（洗，炙）3钱，牙皂（去皮筋）4钱，人参4钱，山药1两，雄黄3钱，麝香2钱。
出处：《万氏家抄方》卷五。
主治：小儿诸惊。
加减：None
功效：None
用法用量：惊风，薄荷汤送下；冷吐泻，干姜豆蔻汤送下；热吐泻，木瓜汤送下；焦躁不宁，桃、橘叶各7片，炒葱姜煎汤送下；温疟，用1丸遍身擦过，灯心薄荷汤服1丸；夜啼，灯心柿蒂汤送下；受惊粪青，薄荷汤送下。
制备方法：上为末，米糊为丸，如龙眼肉大，朱砂为衣。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
