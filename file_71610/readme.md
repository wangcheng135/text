序号：71626
名称：善应白膏
组成：光粉1斤（别研），商陆粉2两（生），续断2两，当归（洗焙）1两，赤芍药1两，白芍药1两，柳枝2两，香白栀半两，川芎半两。
出处：《杨氏家藏方》卷十二。
主治：痈疽发背，一切肿毒恶疮，骨节疼痛，筋脉拘挛，及诸打扑伤损。
加减：None
功效：None
用法用量：若一切疮肿伤折，并于所患处贴之。
制备方法：上锉，如麻豆大，用清麻油1斤，以铁铫或瓷器内入上药，以文武火煎药黑色为度，然后去药滓，留清油再上火煎，次入光粉，以柳枝子搅匀，与油相和得所，滴入水内试之，以不散为度，倾入新水内澄凝，然后取出，以绵子试干，再入钵内，以文武火再煎，熔入蜡半两，乳香末3钱，再以柳枝搅匀，倾入新水内，方取出试干，入瓷器收之。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
