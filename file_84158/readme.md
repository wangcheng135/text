序号：84176
名称：五石汤
组成：白石英2两，钟乳2两，赤石脂2两，石膏2两，紫石英3两，牡蛎2两，人参2两，黄芩2两，白术2两，甘草2两，栝楼根2两，芎2两，桂心2两，防己2两，当归2两，干姜2两，独活3两，葛根4两（1方有滑石2两、寒水石2两，枣20枚）。
出处：《千金》卷三。
主治：产后卒中风，口噤，倒闷吐沫，瘛疭，眩冒不知人，及湿痹缓弱，身体痉，妊娠百病。
加减：None
功效：None
用法用量：以水1斗4升，煮取3升半，分5次服，日3夜2。
制备方法：上药以五石为末，(口父)咀诸药。
临床应用：None
各家论述：1.《千金方衍义》：产后非极虚寒，何致卒然口噤？五石汤治证最剧，急需钟乳、白术相反之性，激发二英、脂、膏、姜、桂、人参奋力祛风，则防己、独活方克有济。_x000D_
2.《中风斠诠》：方以五石为君，明是潜阳镇逆之意，而黄芩、蒌根、人参、甘草，又皆清热养阴之品，则所谓治产后中风，口噤倒闷等证者，岂非血去阴伤，肝阳暴动，内热生风之病，是与古方之豆淋酒、独活紫汤等法治外感风邪而痉厥、瘛疭者不同，惟桂心、干姜终不脱惯用温药之套法。善学古人者，必不可不知所变化也。
用药禁忌：None
