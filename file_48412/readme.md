序号：48427
名称：神效种子丸
组成：大熟地2两4钱，肉苁蓉2两4钱，萆薢4钱，灯草5尺，木香2两4钱，山萸肉2两4钱，荜澄茄2两，大茴香2两，马蔺实（阴干，研）8分，干漆2两，巴戟肉2两，蛇床子1两4钱，龙骨2两，全当归1两，牡蛎粉2两，母丁香2两，桑螵蛸2两2钱，全蝎（去尾）5钱，茯苓1两半，蜘蛛14个，威灵仙2两，菟丝子2两，沉香2钱，车前子2两，木通2两4钱，远志肉2两。
出处：《外科传薪集》。
主治：None
加减：None
功效：种子。
用法用量：每服1-2钱，清晨开水送下。
制备方法：上为细末，炼蜜为丸，如绿豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：每于经期转时，服汤药一剂。煎方如下：桂枝三分，白芍二钱，甘草二钱，姜皮三钱，再加玉盆二钱，枣二枚。
