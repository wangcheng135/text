序号：27591
名称：补阴汤
组成：熟地3两，元参8两，生地4两，麦冬3两，白芍5两，丹皮3两，沙参3两，地骨皮5两，天门冬3两，陈皮5钱。
出处：《石室秘录》卷三。
主治：瘦人火有余，水不足者。
加减：None
功效：None
用法用量：每日5钱，白滚水送下。
制备方法：上为末，炼蜜为丸；或加桑叶6两为末，同捣为丸。
临床应用：None
各家论述：元参去浮游之火，而又能调停五脏之阳，各品之药，阴多于阳，则阴气胜于阳气，自然阴胜阳消，又何必石膏、知母之纷纷哉：虽石膏、知母原是去火神剂，不可偏废，然而用之于火腾热极之初，可以救阴水之熬干，不可用之于火微热退之后，减阳光之转运，此瘦人之治法。
用药禁忌：None
附注：本方方名，据剂型当作“补阴丸”。
