序号：11962
名称：仙授外伤见血主方
组成：归尾2钱，川芎2钱，地黄2钱，白芍2钱，益母草2钱，藁本2钱，乳香（炙）2钱5分，没药2钱5分，川续断3钱，苏木1钱5分，白芷1钱，甘草5分，生姜3片。
出处：《梅氏验方新编、伤科》。
主治：外伤见血者。
加减：头顶伤，加升麻1钱，肉桂2钱；头骨沉陷，加白芷3钱；脑门肿痛，加茯苓、白术各2钱；脑髓出，加香附2钱，白附子、苍耳子、牡蛎各1钱；面青懒食，加柴胡、茯苓各1钱5分，陈皮8分，升麻、半夏、黄耆各1钱；破处生蛀，加细辛、青黛、蝉蜕各1钱，蛀即化为黄水滚出；脑侧近耳边寒热作痛，加丹皮1钱，石枣、泽兰各2钱；目伤出血不止，用人乳饭内蒸过涂之；黑睛脱出，用手掌趁热按进，将绢紧紧包住，3日不开，外用生地捣烂贴退其血，内服方加木贼草、石决明、菊花各1钱；目眶伤损，胬肉脱出，用杏仁去皮尖，嚼细吐于掌上，趁热以绵裹著头，按胬肉上4-5次，送按目内，再用鲜地肤子汁点之自愈，如无鲜者，即浓煎熬膏亦可，后以清水调生半夏末搽6-7日，眉毛即生；目伤睛肿，水调生半夏末涂之；耳伤，加磁石1钱；鼻伤，加辛荑2钱，鳖甲3钱；颊伤，加独活、细辛各2钱；唇伤，加升麻、秦艽各2钱，牛膝3钱；舌伤，加石膏2钱，升麻3钱，用黄芩片贴舌上含之，以断其血；齿伤，加独活、细辛、谷精草；血流不止，用灯心紧咬立止；左肩伤，加青皮2钱；有肩伤，加升麻1钱；喉项伤，加羌活、独活、谷精草各1钱；手伤，加桂枝，禹余粮各1钱、姜汁5匙；胸伤，加川贝3钱，柴胡1钱，枳壳2钱；乳伤，加川贝、百合各2钱，漏芦1钱；胸腹伤，强言乱语，加辰砂，茯神各1钱，远志1钱5分，金银箔10张，盆子2钱为引；吐黄水，加木香、木瓜、扁豆、大茴各1钱，大黄2钱、砂仁14粒；左胁伤，加北芥子1钱，柴胡1钱5分；右胁伤，加北芥子1钱，升麻2钱；腹伤，加大腹皮2钱；腹破肠出，加黄耆、鹿茸各2钱，其肠将手轻轻按入，不可犯指甲，其伤口用柿饼（众人嚼碎）填塞7日全愈，若不便以手按者，用磁石末、滑石粉各2钱，米饮调服，其肠自入，如不入，将病人卧席上，4角用人拿定举摇，其肠自入，或以小麦煎浓汤，待冷，不令病人知，含噀其背，渐渐自入，不令多人相见，并止旁人说话；小腹伤，加小茴1钱，槐花2钱；背伤，加香附、木香各1钱，羌活钱半；腰伤，加木鳖子1个，杜仲、牛蒡子、破故纸、小茴、白芷各1钱、大茴8分、巴戟2钱；臀伤，加白蜡、自然铜各2钱；寒热发搐咬牙，唇口牵动，加天麻，升麻各1钱，柴胡8分；囊肿痛不愈，饮食少思，加人参、白术、柴胡各1钱；两足腿伤，加牛膝2钱，木瓜、苡仁、五加皮、槟榔、石斛、苏梗各1钱；伤口作痒，加干葛1钱，防风、荆芥，连翘壳各1钱5分，赤芍2钱；血出多瘦弱，加人参、麦冬各1钱，烦躁不止，加柴胡5分，丹皮1钱；面黑喘急，加人参5分，苏梗1钱；脓出口噤流涎，加人参3钱，柴胡、升麻各1钱；脓出不干，加滑石、苍术各1钱，白术1钱5分；手足微搐，眉目微动，加钩藤，柴胡各1钱；手撒目闭，汗出如雨，加人参1两，附子5钱；眼开能言，气不相接，加人参、黄耆、白术各1钱。
功效：None
用法用量：水煎服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
