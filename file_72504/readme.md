序号：72520
名称：温补羌活丸
组成：羌活半两，川芎半两，人参半两，白附子半两，赤茯苓半两，天麻1两，僵蚕（炒）1分，干葛（炒）1分，白花蛇（酒浸，焙）1分，川附子（炮）3钱，防风3钱，麻黄3钱，肉豆蔻2钱，母丁香2钱，藿香叶2钱，沉香2钱，木香2钱，轻粉1钱半，珍珠末1钱半，牛黄1钱半，龙脑半字，麝香1字，雄黄2分，辰砂2分（以上7味各另研）。
出处：《幼科类萃》卷四。
主治：小儿吐泻失治，脾肺俱虚，致被肝木所乘，变为慢惊。
加减：None
功效：None
用法用量：每服1-2丸，食前薄荷汤或麦门冬汤送下。
制备方法：上为末，熟蜜剂旋丸，如大豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
