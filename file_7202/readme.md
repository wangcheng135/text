序号：7202
名称：四斤丸
组成：宣州木瓜1斤（去瓤），牛膝1斤（去芦，锉），天麻1斤（去芦，细锉），苁蓉1斤（洗净，切，各焙干，秤），上药用无灰酒5升浸，春5日，秋5日，夏3日，冬10日足，取出焙干。再入：附子2两（炮，去皮脐），虎骨（涂酥炙）2两。
出处：《局方》卷一（绍兴续添方）。
主治：肾经不足，下攻腰脚，腿膝肿痒，不能屈伸，脚弱少力，不能踏地，脚心隐痛，行步喘乏，筋脉拘挛，腰膝不利，及风寒湿痹，脚气缓弱。
加减：None
功效：补虚除湿，大壮筋骨。
用法用量：每服30-50丸，空心煎木瓜酒送下；或盐汤吞下亦得。
制备方法：上为细末，用浸前药酒打面糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：虎骨四斤丸（《准绳·类方》卷四）、虎骨木瓜丸（《重订通俗伤寒论》）。
