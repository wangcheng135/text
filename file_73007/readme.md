序号：73023
名称：椒肾丸
组成：汉椒2两（去目及闭口者，微炒出汗），白术1两半，肉桂1两半（去皱皮），白龙骨2两，白矾灰1两半，桑螵蛸1两半（微炒），鹿茸2两（去毛，涂酥，炙微黄），鸡头实2两（生者），补骨脂1两半（微炒），干姜1两半（炮裂，锉）。
出处：《圣惠》卷二十九。
主治：虚劳内伤，肾气衰冷，小便余沥，精气滑泄。
加减：None
功效：None
用法用量：每服30丸，食前以暖酒送下。
制备方法：上为末，入盐花2两，研令匀，用獖猪肾10只，切去脂膜，研令烂，以酒2升渐入，熬成膏，次入药末，和捣为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
