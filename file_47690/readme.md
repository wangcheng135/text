序号：47705
名称：神仙巨胜丸
组成：巨胜（酒浸1宿，9蒸9晒）1两，牛膝（酒浸彻，焙）1两，巴戟天（去心）1两，天门冬（去心，焙）1两，山芋1两，熟干地黄（焙）1两，柳桂（去粗皮）1两，酸枣仁1两，覆盆子1两，菟丝子（酒浸，别捣，焙干）1两，远志（去心）1两，菊花1两，人参1两，白茯苓（去黑皮）1两。
出处：《圣济总录》卷一九九。
主治：None
加减：None
功效：轻身壮阳，却老还童，去三尸，下九虫，除万病。
用法用量：每服20丸，空心温酒送下。服至1月身轻体健，万病不侵。
制备方法：上拣择净，为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
