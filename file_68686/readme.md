序号：68702
名称：小红升
组成：真水银2两，净明矾2两，提净火消2两。
出处：《外科方外奇方》卷一。
主治：一切疮疡疔肿疖各毒初起出脓时。
加减：None
功效：呼脓拔毒。
用法用量：疮疡疔肿疖初起出脓时，用此掺疮口，外用膏药盖之。
制备方法：上为末，安铁耳锅内，盖以高深宫碗，居中平稳，用煨石膏研细封口，放于风炉上，以先义后武之火，炼3柱香为度，过夜待冷，以刀刮去封口石膏，将碗揭起，用小刀刮下升丹，或绿或黄或红，各自贮开，瓷瓶盛之，听用，颜色虽殊，功效则一，陈一年者，出尽火气，愈陈愈佳。
临床应用：None
各家论述：None
用药禁忌：男子肾囊、女子乳头及眼珠上下两角或生疮毒，切勿用此丹，恐受水银之气，受患莫测，慎之。
附注：如脓腐去净者，另用生肌长肉粉霜。
