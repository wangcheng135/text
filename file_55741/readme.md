序号：55756
名称：桃枝当归膏
组成：当归身（去细梢，洗去土，干）1两，杏仁（汤浸，去皮尖）100个，肥嫩柳枝3两半（切寸许，水洗，干），肥嫩桃枝1两半（切寸许，水洗，干），黄丹（水飞）6两，脂麻油1斤。
出处：《东垣试效方》卷三。
主治：一切热疮。
加减：None
功效：None
用法用量：None
制备方法：上件先令油热，下桃枝、柳枝，熬至半焦，以绵裹当归、杏仁熬至桃枝、柳枝黑焦为度，去药滓滤油净，抹出铫子中滓秽合净，再上火令沸，旋旋入黄丹，熬成滴水中不散为度，或只摊纸上不透为度。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方方名，《医学纲目》引作“寒疮热膏药”。
