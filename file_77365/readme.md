序号：77381
名称：遇仙膏
组成：豨莶草4两，海风藤4两，大半夏4两，蓖麻子4两，麻黄4两，川乌4两，草乌4两，南星4两，羌活4两，桂枝4两，独活1两，细辛1两，玄参1两，当归1两，荆芥1两，金银花1两。
出处：《遵生八笺》卷十八。
主治：风湿骨节疼痛，或痰核肿痛，皮肤麻木，瘙痒，一切风疾。
加减：如牙疼，去轻粉。
功效：None
用法用量：厚纸缎绢摊贴。肉痒出冷汗方去之。
制备方法：以上用真香油7斤，葱汁、生姜汁各2碗半，浸煎药1宿，用铜锅文武火熬煎，药色不易黑，必待滴油变黑，去滓；每药油1斤，下飞过好丹9两，候成膏，再加白水煮过松香1斤，黄蜡1斤，化搅匀，气温方入：没药、乳香、木香、轻粉、胡椒各4两，白芥子1斤，五味研为细末，入膏内，每膏1斤，入蟾酥5钱。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
