序号：30801
名称：灵龙丹
组成：五灵脂（去沙）7两（别研），草乌头（生，去皮尖）半两，木鳖子（新者，去壳）2两（别研），白胶香（别研）半两，地龙（去土）半两，乳香半两（别研），麝香1钱（别研）。
出处：《魏氏家藏方》卷一。
主治：一切风疾，卒中潮搐，口噤不语，舌强脚弱，鹤膝瘫痪，半身不遂，偏风口眼斜。
加减：None
功效：None
用法用量：每有患者，以1片分作3服，用酒磨下；卒中急风，以白矾1小块研末，用童便同酒磨下；或口噤灌少许入鼻中，待口开1时灌尽；小儿惊风，分作6服，薄荷汤入酒化下；手足疼痛，薄荷酒下，或姜汁磨涂患处；如治牙痛，以少许塞之。
制备方法：上为细末，入诸别研药拌和，以辰年辰月辰日辰时取辰方上野水搜做小阿胶片，风干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
