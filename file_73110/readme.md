序号：73126
名称：山芋丸
组成：山芋1两，石斛（去根）1两，牛膝（去苗酒浸，切，焙）1两，鹿茸（去毛，酥炙）1两，白茯苓（去黑皮）1两，五味子1两，续断1两，巴戟天（去心）1两，山茱萸1两，人参1两，桂（去粗皮）1两，熟干地黄（焙）1两，泽泻1两，杜仲（去粗皮，炙）1两，蛇床子（炒）1两，远志（去粗皮，炙）1两，菟丝子（酒浸1宿，别捣末）1两，天雄（炮裂，去皮脐）1两，覆盆子（去梗萼）1两，肉苁蓉（酒浸，切，焙）1两。
出处：《圣济总录》卷一八七。
主治：虚损。
加减：None
功效：补脏腑，利腰脚，壮元气，充骨髓。
用法用量：每服30丸，空腹、晚食前温酒送下。
制备方法：上为末，炼蜜和捣300-500杵为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
