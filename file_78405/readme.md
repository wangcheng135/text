序号：78421
名称：麝香猬皮丸
组成：鸡冠花8两（微炒存性），牛角腮（烧存性）半两，贯众（去毛净）半两，槐花（微炒）半两，油发灰半两，香白芷半两，当归半两，枳壳（去瓤）半两，玄参半两，黄连半两，黄耆半两，防风（去芦）半两，鳖甲1钱（炙黄色），麝香半钱（别研），猬皮（烧存性）1个，诃子（去核，净微炒）半两，猪左悬蹄5个（烧存性）。
出处：《普济方》卷二九七。
主治：肠风痔漏，疼痛不止，大便下血，时举发。
加减：None
功效：None
用法用量：每服30-40丸，空心饮下。
制备方法：上入麝香和末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：忌动风发病之物。
附注：None
