序号：43357
名称：养津益胃汤
组成：玉竹30g，石斛30g，白芍12g，生地黄12g，麦冬12g，胡麻10g，甘草6g，钩藤10g，石决明30g，何首乌10g。
出处：《千家妙方》引刘渡舟方。
主治：风阳走于肢体，消灼血液，肌肉萎缩而游走作痛。
加减：None
功效：滋养胃液，以生营血，平息肝风。
用法用量：水煎服，每日1剂。
制备方法：None
临床应用：杨某某，女，28岁。1971年开始，两手掌鱼际肌肉渐见萎缩，并伴麻木感，四肢及后背时现游走性疼痛，屡治无效。饮食日减，更厌荤腥，口咽发干而不多饮，月经提前，量少，心烦不安，两颊绯红，舌红苔薄黄，脉大而软。证由胃液不足，血虚不能养肝以熄风，而使风阳走于肢体，消灼血液所致。治宜滋养胃液以生营血，投以养津益胃汤，共服30余剂而愈。
各家论述：None
用药禁忌：None
附注：None
