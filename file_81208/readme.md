序号：81224
名称：麝香天麻丸
组成：紫背干浮萍草（去土）4两，麻黄（去根节）2两，防风（去芦叉）1两，天麻（去芦，郓州者佳）1两（上为细末），没药（别研极细）2两，朱砂（研飞）2两，安息香（别研细）1两，乳香（研）1两，麝香（研）1两，血竭（别研极细）3两，槐胶（别研细）1两半。
出处：《局方》卷一。
主治：风痹。手足不随，或少力颤掉，血脉凝涩，肌肉顽痹，遍身疼痛，转侧不利，筋脉拘挛，不得屈伸。
加减：None
功效：None
用法用量：空心服1丸，以温酒或荆芥汤化下，患处微汗为效，如不欲化服，即丸如梧桐子大，每服30丸，依前汤使下。
制备方法：上药，除研药外，将碾出药同研拌匀，炼滤白沙蜜与安息香同熬过，搜成剂，入臼捣杵熟为丸如弹子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
