序号：37293
名称：定痛消毒饮
组成：蒲公英1钱2分，紫花地丁1钱2分，当归（乳房用身，乳顶用尾）1钱，白芍（醋炒）1钱，赤芍1钱，花粉1钱，浙贝母（去心，研）1钱，皂角刺7分或5分，柴胡梢8分或1钱（乳顶肿结用之，若乳房易白芷），牡丹皮8分，广皮8分，明乳香5分，没药5分，生草3分。
出处：《胎产心法》卷下。
主治：乳顶劳或乳房吹乳成痈，并乳结之证，发热恶寒，冷汗自出，势欲破而疼痛难忍。
加减：None
功效：None
用法用量：水3钟，加红枣2个（去核），灯心50寸，煎8分，临服加无灰酒小半酒杯入药，滚数滚服之，不时用槐艾水洗。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：None
