序号：7797
名称：圣石丹
组成：礞石4两（炭火烧1伏时），不灰石2两（炭火烧1伏时），母丁香1两，木香1两，人参（去芦）1两，白茯苓1两，半夏半两（汤洗7次，焙干），真阳起石半两（先同上为末），阿魏（汤化，去砂石，以面和成饼，焙干）1钱，巴豆14个，杏仁14个（并连皮，灯上烧作炭，略存性）。
出处：《卫生总微》卷十引《经验方》。
主治：小儿胃虚伤冷吐逆。
加减：None
功效：None
用法用量：每服1粒，水1大盏，入姜皂子大。擘破同煎，至5分放温，时时服之。煎药须用银石器。
制备方法：上为细末。汤泡蒸饼为丸，如荔枝子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：本方原名圣石散，与剂型不符，据《普济方》改。
