序号：40125
名称：苓甘五味加姜辛半夏杏仁汤
组成：茯苓4两，甘草3两，五味半升，干姜3两，细辛3两，半夏半升，杏仁半升（去皮尖）。
出处：《金匮》卷中。
主治：支饮，水去呕止，其人形肿。
加减：None
功效：None
用法用量：以水1斗，煮取3升，去滓。温服半升，每日3次。
制备方法：None
临床应用：痰饮《经方实验录》：叶瑞初君，咳延四月，时吐浊沫，脉右三部弦，当降其冲气。茯苓三钱，生甘草一钱，五味子一钱，干姜一钱半，细辛一钱，制半夏四钱，杏仁四钱。两进苓甘五味姜辛半夏杏仁汤，咳已略平，惟涎沫尚多，咳时不易出，原方加桔梗，服后竟告霍然。
各家论述：None
用药禁忌：None
附注：苓甘味姜辛夏仁汤（《普济方》卷一四○）、茯苓甘草五味姜辛汤（《医门法_x000D_
律》）、茯苓甘草五味辛夏仁汤（《方剂辞典》）。
