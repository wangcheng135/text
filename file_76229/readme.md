序号：76245
名称：腊鸦散
组成：腊月鸦1只（去爪咀），腊月野狐肝1具（并腊月鸦同入瓷瓶中，以盐泥固济候干。以大火煅令通赤，去火取出，细研为散），天麻3分，天南星半两（炮裂），白附子半两（炮裂），桑螵蛸半两（微炒），藿香半两，干蝎半两（微炒），蚱蝉1分（微炒），乌蛇肉3分（酒浸，炙微黄），白僵蚕半两（微炒），天竹黄半两（细研），阿胶半两（捣碎，炒令黄燥），麝香1分（细研），牛黄1分（细研），龙脑1分（细研），腻粉1分。
出处：《圣惠》卷二十二。
主治：急风。手足挛急，口噤项强，不知人事。
加减：None
功效：None
用法用量：每服2钱，以温酒调下，不拘时候，频服，以效为度。
制备方法：上为细散，与前2味相和，更令研匀。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
