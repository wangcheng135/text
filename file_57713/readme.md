序号：57728
名称：烧丹丸
组成：玄精石（烧赤）1钱，轻粉1钱，粉霜半钱，硼砂半钱。
出处：《幼科类萃》卷十九。
主治：小儿食癖、乳癖，每日午后发寒热，咳嗽，胁下结硬。胎惊发痫。
加减：None
功效：None
用法用量：1岁5丸，2岁10丸，夜卧以温浆水送下，至天明取下恶物是效，如不下，渐加丸数，如奶癖解未消尽，隔2-3日又1服，癖消尽为度。
制备方法：上先将硼砂研细，入3味研匀，更入寒食面1钱研匀，滴水和成饼，再用面煨了，慢火内煨黄，取出去面，将药饼再研为细末，滴水和丸，如黄米大。
临床应用：痫《东医宝鉴·内景篇》：一少女患痫，遇阴雨及惊则作声似羊鸣，口吐涎沫，知其胎受惊也，其病深痼难治，先予烧丹丸，继以四物汤入黄连，随时令加减，且令淡味以助药功，半年而愈。
