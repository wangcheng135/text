序号：37560
名称：抵圣大道丸
组成：硫黄1两，水银（与硫黄结为砂子）1两，自然铜1两，雄黄1两，朱砂1两，铅霜1两，定粉1两，黄丹1两，消石1两，上为细末，入瓷瓶内，以纸筋盐泥固济，盖上钻1窍子，4边泥令合，候极干，用炭火3斤，渐渐逼出阴气令尽，以泥塞窍子令密，用炭火10斤以来，烧令通赤，待稍冷，便以湿砂土3-2斗，掩盖瓶子，勿令透气，1复时取出，向好纸上，于湿地摊，以盆盖1宿，出火毒，研令极细。天麻2两，白僵蚕2两（微炒），羌活2两，桂心1两，犀角屑1两，阿胶1两（捣碎方炒令黄燥），芎2两，白附子2两（炮裂），天南星2两（炮裂），附子2两（炮裂，去皮脐），干蝎1两（微炒），麻黄1两（去根节），川乌头1两（去皮脐，锉碎，以酒拌炒令黄），白龙脑半两，麝香半两，牛黄半两。
出处：《圣惠》卷二十五。
主治：一切风。
加减：None
功效：None
用法用量：每服1丸，以温豆淋酒研下。
制备方法：上为末，入前药末及龙脑等，都研令匀细，炼蜜为丸，如鸡头子大。
临床应用：None
各家论述：None
用药禁忌：忌猪、鸡、毒、鱼，粘滑动风物。
附注：本方方名，《普济方》引作“抵圣大通丹”，见该条。
