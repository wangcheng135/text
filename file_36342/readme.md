序号：36352
名称：和中解托汤
组成：柴胡1钱，干葛1钱，山楂1钱，泽泻1钱，陈皮8分，甘草3分，生姜、大枣。
出处：《不居集》上集卷十。
主治：外感之症，手足厥冷，恶寒淅沥，肢节痠疼，有似阳微者；口渴欲饮，舌上微苔，有似阴弱者。
加减：如头痛者，加川芎8分；呕恶者，加半夏5分；兼寒滞不散者，加桂枝、防风；胸腹有微滞者，加厚朴8分。
功效：None
用法用量：None
制备方法：None
临床应用：None
各家论述：此外邪不解，里郁内热之方也。若体虚之人，过于清凉，邪愈不解。只用柴胡提清，葛根托里，此二味者，一则味甘性寒，一则气清味辛，清辛而不肃杀，甘寒而不壅遏，能使表气浃洽；陈皮辛以利气，山楂酸以导滞，泽泻渗以分消，此三味者辛而不烈，渗而不燥，导而不峻，虚弱者宜之；更有甘草以调表里之和，姜、枣平营卫之逆也。
用药禁忌：None
附注：方中生姜、大枣用量原缺。
