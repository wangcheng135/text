序号：48553
名称：神效墨附丸
组成：香附子1斤（去毛，作4份，1份好酒浸，1份米泔浸，1份童便浸，1份醋浸，各浸1日夜），艾绵4两（用醋2大碗，同香附1处煮干，石臼内杵以烂为度，捻作钱样厚大饼，以新瓦炭火焙干，捣为末），白茯苓（去皮，净）1两，当归（去芦，净，酒浸1宿）1两，人参（去芦）1两，川芎（大实者，去土，净）1两，熟地（用淮生地酒浸，9蒸9晒）1两，上等微墨（火煅，醋淬）1两，木香5钱。
出处：《万氏家抄方》卷五。
主治：妇人久无子，经事不调，及数堕胎者。
加减：None
功效：None
用法用量：每服50丸，空心好酒送下。
制备方法：上为末，醋糊为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：墨附丸（《医学入门》卷八）。
