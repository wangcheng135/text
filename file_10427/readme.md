序号：10427
名称：牛黄丸
组成：牛黄5分，珍珠5分，麝香5分，朱砂，龙齿（各另研）2钱，犀角2钱，琥珀2钱，天门冬（去心）4钱，人参4钱，茯苓4钱，水银5分，麦门冬（去心）4钱，防风半两，黄芩半两，知母半两，龙胆草半两，石菖蒲半两，白芍药半两，全蝎半两，甘草半两，蜂房3钱，金箔70片，银箔70片。
出处：《医统》卷四十九。
主治：癫狂、风痫、心风，邪气惊心，神不守舍，时发无常，仆地吐涎，不自知觉。
加减：None
功效：None
用法用量：每服15丸，食后临夜新竹叶汤送下。
制备方法：上除另研药外，余为细末，共和匀，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
