序号：38962
名称：泻胃热汤
组成：栀子仁2两，射干2两，升麻2两，茯苓2两，芍药4两，白术5两，生地黄汁1升，赤蜜1升。
出处：《千金》卷十六。
主治：胃实热，苦头痛，汗不出如温疟，唇口干，善哕，乳痈，缺盆腋下肿痛。
加减：None
功效：None
用法用量：上(口父)咀。以水7升，煮取1升半，去滓，下地黄汁煮两沸，次下蜜，煮取2升，分3服。
制备方法：None
临床应用：None
各家论述：《千金方衍义》：泻热，反用白术，可见泻胃之热，而非泻胃之实也；方用升麻专走阳明，升发胃热，地黄滋阴退阳，赤蜜和胃解毒，射干散腹中结气，栀子仁滋胃中热气，芍药治邪气腹痛，茯苓宁五脏正气，而佐白术除热消食之功，祛邪养正，全赖苓、术之力耳。
用药禁忌：None
附注：None
