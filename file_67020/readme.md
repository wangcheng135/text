序号：67036
名称：麻黄人参芍药汤
组成：人参3分，麦门冬3分，桂枝5分，当归身5分，麻黄1钱，炙甘草1钱，白芍药1钱，黄耆1钱，五味子2个。
出处：《脾胃论》卷下。
主治：久患吐血，愈后复发；吐血外感寒邪，内虚蕴热。
加减：None
功效：None
用法用量：上（口父）咀，都作1服。以水3盏，煮麻黄1味，令沸，去沫，至2盏，入余药同煎至1盏，去滓，临卧热服。
制备方法：None
临床应用：吐血：一贫士，病脾胃虚，与补药，愈后继居旷室，卧热坑而吐血数次。予谓此人久虚弱，脐附有形，而有大热在内，上气不足，阳气外虚，当补表之阳气，泻里之虚热。冬居旷室，衣服復单薄，是重虚其阳，表有大寒变遏里热，火邪不得舒伸，故血出于口。因思仲景太阳伤寒，当以麻黄汤发汗，而不与之，遂成衄血，却与之立愈，与此甚同，因与麻黄人参芍药汤。
各家论述：《医方集解》：此足太阳手太阴药也。桂枝补表虚，麻黄去外寒，黄耆实表益卫，甘草补脾，白芍安太阳，人参益元气而实表，麦冬保肺气，五味子安肺气，当归和血养血。盖取仲景麻黄汤与补剂各半服之。但凡虚人当服仲景方者，当以此为剂也。
用药禁忌：None
附注：麻黄桂枝汤（《兰室秘藏》卷中）。
