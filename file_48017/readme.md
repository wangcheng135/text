序号：48032
名称：神曲饼子
组成：天南星3钱，乌蛇3钱，天麻半两，麻黄（去节）半两，全蝎1分半，白附子3钱半，白僵蚕4钱，大附子1枚（炮裂，去皮脐）。
出处：《幼幼新书》卷十引《孔氏家传》。
主治：吐泻过后，精神困顿多睡，不吃乳食，四肢逆冷，欲变惊，或已变痫，哭声如鸦，面色青黄，手足瘛疭，咽中不利，或变痫滑利。
加减：None
功效：None
用法用量：治小儿吐泻过后，精神困顿多睡，不吃乳食，四肢逆冷，欲变惊，以神曲4两，龙、麝少许，每服量多少，以温水调下；若已变痫，哭声如鸦，面色青黄，手足瘛疭，咽中不利，加朱砂、龙、麝并曲服之；变痫滑利，即以蜜丸曲末鸡头大，温水化下。
制备方法：上为末，水1升，浸3日，布滤去滓，寒食面1斗和匀，踏作片子，用楮叶罨7日，取出，用纸袋吊起，14日可用。丙日作曲，丁日治药。
临床应用：None
各家论述：None
用药禁忌：None
附注：丙丁膏（《幼幼新书》卷十引《孔氏家传》）、丙丁散（《卫生总微》卷五）。
