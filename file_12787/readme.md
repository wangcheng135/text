序号：12793
名称：石斛丸
组成：石斛1两（去根），牛膝1两（去苗），狗脊3分（去毛），杜仲1两（去皱皮，炙微黄，锉），肉苁蓉1两（酒浸1宿，刮去皱皮，炙干），鹿茸半两（去毛，涂酥炙微黄），附子1两（炮裂，去皮脐），桂心1两，萆薢3分（锉），羌活3分，木香1两，牡丹皮1两，人参3分（去芦头），黄耆1两（锉），山茱萸3分，防风半两（去芦头），芎藭半两，槟榔1两半，熟干地黄1两。
出处：《圣惠》卷六十七。
主治：None
加减：None
功效：打扑损伤后，止疼痛，补虚损。
用法用量：每服30丸，以温酒送下，日3次。
制备方法：上为末，炼蜜为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
