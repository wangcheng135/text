序号：78925
名称：酸枣仁散
组成：酸枣仁3分（微炒），犀角屑半两，黄耆3分（锉），赤芍药3分，枳壳半两（敖炒微黄，去瓤），防风半两（去芦头），细辛半两，茯神1两，当归3分（锉，微炒），龙齿3分，桑根白皮1两，独活半两，子芩3分，麦门冬3分（去心），石膏2两，人参1两（去芦头），羚羊角屑3分，甘草半两（炙微赤，锉）。
出处：《圣惠》卷六十九。
主治：妇人血风，心神惊悸，头痛，眠卧不安，四肢烦疼，不思饮食。
加减：None
功效：None
用法用量：每服4钱，以水1中盏，入生姜半分，大枣2枚，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
