序号：47000
名称：独活散
组成：独活半两，附子3分（炮裂，去皮脐），防风半两（去芦头），麻黄3分（去根节），桂心半两，芎半两，薏苡仁1两，赤茯苓3分，牛膝3分（去苗），人参半两（去芦头），白术半两，茵芋半两，海桐皮半两（锉），枳壳半两（麸炒微黄，去瓤），甘草半两（炙微赤，锉）。
出处：《圣惠》卷二十一。
主治：风，身体疼痛，腰背拘急。妇人血风攻注，脚膝虚肿，或上焦不利。
加减：None
功效：None
用法用量：每服3钱，以水1中盏，加生姜半分，煎至6分，去滓温服，不拘时候。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：忌生冷、油腻、毒鱼、滑物。
附注：独活汤（《圣济总录》卷一五○）。
