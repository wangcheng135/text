序号：43793
名称：厚朴丸
组成：厚朴（去粗皮，生姜制）1斤，生姜半斤（净洗，切片），大枣100枚，附子（7钱者）4枚（炮，去皮脐，每枚切作4片），人参（去芦头）2两，诃子（煨，去核）2两，白术2两，白茯苓（去皮）2两，肉豆蔻（面裹煨香）2两，木香2两。
出处：《杨氏家藏方》卷六。
主治：脾虚受湿，胜则濡泻，故肠鸣夜起，四肢浮肿，多困少力。
加减：None
功效：None
用法用量：每服50丸，食前米饮送下。
制备方法：上将前4味于银、石器中，以水1斗，慢火熬干，取大枣去皮核，四物一处捣如泥；将人参等6味为细末，和匀，入少面糊同为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
