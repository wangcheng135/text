序号：69710
名称：琥珀蜡矾丸
组成：白矾1两2钱，黄蜡1两，雄黄1钱2分，琥珀1钱（另研极细），朱砂1钱2分，蜂蜜2钱（临入）。
出处：《外科正宗》卷一。
主治：痈疽、发背、粉瘤、瘰疬、痰核、痔漏、杨梅结毒。痈疽、发背已成未成脓之际，恐毒气不能外出，必致内攻者。遍身疮如蛇头，鼻痣。斑痧痘疹。
加减：None
功效：护膜护心，散血解毒。祛毒化脓，生肌补漏。
用法用量：每服20-30丸，食后白汤送下，病甚者，早晚日进2次。
制备方法：上4味先研极细，另将蜜蜡铜杓内熔化，离火片时，候蜡四边稍凝时，方入上药搅匀，共成一块，以一人将药火上微烘，众手急丸，如小寒豆大，用朱砂为衣，瓷罐收贮。
临床应用：None
各家论述：《成方便读》：方中黄蜡、白矾，皆固涩之品，为护膜之主药，故以为君；雄黄、朱砂、琥珀，不特镇心神、安魂魄，且皆有解毒之功；白蜜甘平而润；护膜解毒，两擅其长。
用药禁忌：None
附注：蜡矾丸（《全国中药成药处方集》吉林方）。
