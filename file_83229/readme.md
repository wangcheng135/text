序号：83247
名称：乌鸡丸
组成：黄连（去须）2两，附子（炮裂，去皮脐）2两，柴胡（去苗）1两，白附子（生）1两，当归（切，焙）1两，秦艽（去苗土）1两，槐胶1两，甘草（炙，锉）1两，苁蓉（薄切，酒浸，焙）1两，续断1两，远志（去心）1两，巴戟天1两，陈橘皮（去白，焙）半两，乳香（研）半两，雄黄（研）半两，丹砂（研）半两，丁香1分，干姜（炮）1分。
出处：《圣济总录》卷九十三。
主治：传尸劳，尸注骨蒸，眼目昏涩，面色青黑，咽喉噎塞，痰涕咳嗽，或痃癖攻刺疼痛，胸膈满闷，时或恚怒，或多感伤，情思不乐，梦寐虚惊，精藏滑泄，失血，憎寒，潮热盗汗，肩背拘急，腰膂肢节烦疼，四肢少力，不喜饮食，小便黄赤，大便不调，乍进乍退。
加减：None
功效：None
用法用量：每服20丸，以暖酒吞下，空心、日午、临卧各1服。
制备方法：上为极细末，先养乌雌鸡1只，以硫黄3两为末，分作30份，每日拌饭喂1月，日用大麻子1升喂尽，宰，治去毛及嘴、爪并肚肠，留心、肝；将前药末入在鸡腹内，以麻线缠定，再用无灰酒1斗，银石锅内用文武火煮，令鸡熟，取出，除去粗骨，将鸡并腹内药同细研，更将鸡骨焙干，捣罗为末，同研药，搜和入臼中，捣千百下为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
