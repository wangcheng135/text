序号：28538
名称：苏荷汤
组成：紫苏1两，南薄荷1两，青蒿1两，条参8钱，连翘8钱，槐花7钱，玄参7钱，柴胡6钱，川芎2钱，生黄耆5钱。
出处：《验方新编》卷十五。
主治：中疳蛊、中蛇蛊、中肿蛊、中癫蛊。
加减：蛇蛊，加白芷1两、三七2钱；大便秘结者，加茨菇菜1两更妙。
功效：None
用法用量：水煎服。服药之后，病渐减者，即是对症，久服自愈。
制备方法：None
临床应用：None
各家论述：方中条参清肺火，消肿胀；连翘泻心火，败毒；紫苏入肺、心、脾，杀蛇发表；薄荷入肝杀蛊；槐花清肝火解毒；生黄耆败毒发表，玄参清肾火解毒，川芎发表，青蒿杀蛊解毒。凡中蛊治方大要，不外杀蛇解毒，发毒散毒之品。
用药禁忌：戒鸡、鸭、鱼、虾、蚌、蛤。
附注：None
