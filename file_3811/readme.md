序号：3811
名称：引水散
组成：石燕子1双（醋淬），海马1两，海蛤1两，滑石1两，琥珀1两，赤茯苓1两，川木通1两，通草1两，越桃（炒，系山栀子仁）1两，泽泻1两，猪苓（去黑皮）1两，车前子（微炒）1两，茴香（微妙）1两，瞿麦穗1两，萹蓄1两，苦葶苈（纸衬炒）1两，忘忧根1两，木香1两，白丁香1两，鬼棘针1两。
出处：《御药院方》卷八。
主治：小水秘涩不快或不通，及肿满、脚气、一切湿证。
加减：None
功效：None
用法用量：每服5钱，水1盏半，灯心30茎同煎，取清汁8分，纳麝香1字，拌匀放温，食前服。
制备方法：上为粗散。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
