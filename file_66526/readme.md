序号：66542
名称：丁香匀气丸
组成：丁香1分，木香1分，沉香（锉）1分，肉豆蔻（去壳）1分，桂（去粗皮）1分，京三棱（煨，先捣取末）1分，当归（洗，切，焙）1分，陈橘皮（汤浸，去白，焙）1分，槟榔（锉）1分，荜澄茄1分，附子（炮裂，去皮脐）1分，安息香（酒化，去滓）1分，乳香（绢包，汤内摆过，候干，研）1分，硇砂（飞）1分，丹砂（研）1分，巴豆21粒（去皮，热灰内炮令紫色，研）。
出处：《圣济总录》卷六十二。
主治：膈气痰结，呕逆减食；及积聚留结，心腹胀满。
加减：None
功效：None
用法用量：每服5-7丸，温生姜汤送下。
制备方法：上为末，与安息香等一处搅和研匀，酒煮面糊和，再捣三二百下，丸如麻子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：丁香丸（《圣济总录》卷七十一）。
