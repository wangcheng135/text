序号：47764
名称：神仙枕
组成：川椒1两，桔梗1两，荆实子1两，柏子仁1两，姜黄1两，吴茱萸1两，白术1两，薄荷1两，肉桂1两，川芎1两，益智仁1两，枳实1两，全当归1两，川乌1两，千年健1两，五加皮1两，蒺藜1两，羌活1两，防风1两，辛夷1两，白芷1两，附子1两，白芍药1两，藁本1两，苁蓉1两，北细辛1两，猪牙皂荚1两，芜夷1两，甘草1两，荆芥1两，菊花1两，杜仲1两，乌药1两，半夏1两。
出处：《中国医学大辞典》引丁其泰方。
主治：None
加减：None
功效：种子，除百病。
用法用量：100日后诸病消除，精神倍长，如夫妇皆以此作枕，更见奇效。
制备方法：上为细末，绢袋盛之，另用槐木薄板做枕1个，高3寸3分，宽4寸5分，长1尺2寸，如天盖地，一面上钻孔128个，如梧桐子大，将前药装入枕中，其药每间3-5个月一换。
