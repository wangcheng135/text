序号：68814
名称：黄连枳壳汤
组成：川黄连1钱，枳壳8分，当归8分，白芍药1钱，茯苓7分，泽泻7分，青皮7分，槟榔7分，木香5分（磨汁入），甘草4分。
出处：《医统》卷三十六引《质疑》。
主治：痢疾初作，多由湿热，但下之后，即服此汤1-2剂，并无再作。
加减：湿热积滞，初作炽迫者，宜下之，加大黄、朴消各2钱；血痢，加黄芩、地榆、川芎、桃仁各6分；白痢，加吴茱萸（炮）5分；腹痛者，倍芍药，加玄胡索、泽兰叶；赤白兼下者，加桃仁、滑石、归尾、陈皮各5分；赤痢久弱，下后未愈，去芩、连，加归尾、芍药、川芎、熟地黄、白术、阿胶珠各1钱；湿甚，小水少，加木通、泽泻、山栀、茯苓各5分；下后二便流利，惟后重不去，此气陷于下也，升麻、川芎提之；痢久气血两虚者，八物汤养之；痢久滑泄，二便流利，腹中清，加粟壳、诃子、阿胶之类涩之。
功效：None
用法用量：上（口父）咀，作1服，水2钟，加生姜3片，煎1钟，食前温服。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：None
附注：《证治宝鉴》有厚朴，无木香。
