序号：40020
名称：肺灵药片
组成：黄芩1两，郁金4钱，丹皮4钱，菊花炭4钱，甘草4钱，贝母4钱，生石膏4钱，白芍8钱（以上8味共研为细粉，过罗），麦冬4钱，解石斛4钱，知母4钱，竹茹4钱，饴糖4钱，生地8钱，蒌仁8钱（以上7味共熬膏）。
出处：《北京市中药成方选集》。
主治：肺气虚热，咳嗽痰盛，喘促咽干，津液不足。
加减：None
功效：润肺养阴，止嗽化痰。
用法用量：每服4片，1日2次，温开水送下。
制备方法：用膏拌前药，阴干，研为细粉，以水成滓，如黍米大，再砸成饼，约重1分（每5两2钱兑羚羊粉2钱；每16两细粉，兑胶粉1两）。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
