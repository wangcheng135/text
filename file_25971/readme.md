序号：25979
名称：百伤饮
组成：干葛3两，净香附2两，升麻（净洗），青皮（去白）1两，陈皮（去白）1两，谷芽（净洗，焙干）1两，麦芽（净洗，焙干）1两，桔梗（锉，炒）1两，紫苏（和根）1两，缩砂仁1两，甘草1两，神曲（炒）1两，赤芍药1两，麻黄（不去根节）7钱半，枳壳（切片，麸炒微焦）7钱半。
出处：《活幼心书》卷下。
主治：感冒风寒邪气，不拘冷热二证。
加减：有积，加水酒曲；热多，添灯心、竹叶。
功效：None
用法用量：每服2钱，水1盏，煎7分，不拘时候温服；或加生姜、葱白同煎。
制备方法：None
临床应用：None
各家论述：None
用药禁忌：慢惊、慢脾不用。
附注：None
