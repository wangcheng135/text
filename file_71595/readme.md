序号：71611
名称：喜树碱小蜜丸
组成：喜树果实干品5斤，喜树果实粉（80目）250g，法半夏粉（80目）250g，蜂蜜适量。
出处：《中药制剂汇编》。
主治：胃癌等。
加减：None
功效：抗癌肿。
用法用量：每服2钱，1日3次。
制备方法：喜树果实粉碎成绒状，加0.1％氢氧化钠水溶液15000毫升，煎煮2小时，pH＝6～7，过滤，滤渣加0.1％氢氧化钠水溶液10000毫升，煎煮1.5小时，过滤，两次滤液合并浓缩，浓缩至300毫升，加喜树果细粉吸收清液，80℃以下烘干，干后研粉过80目筛，再加半夏粉，混合均匀，加炼蜜制成小丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
