序号：41280
名称：金花如圣散
组成：苍术6两（米泔水浸1宿，去皮），川乌4两（火煨），草乌4两（生用），川芎3两5钱，细辛2两5钱（净），防风2两5钱（去芦，生用），白芷2两5钱，白术2两5钱，蝎梢5钱，雄黄5钱（净，研）。
出处：《摄生众妙方》卷一。
主治：金刃伤，破伤风牙关紧闭，伤寒，蝎螫伤，蛇虫伤，妇人产后淋血不止，多年恶疮，脑风，诸般疮，火烧伤，咳嗽，疽疖，鱼眼红丝疔疮，偏正头风，疥疮，瘰疬，杖疮有血者，舌根疮，外臁疮，无名肿毒，诸风倒地，不省人事，揭鞍风。
加减：None
功效：None
用法用量：每服2钱或1钱半，好酒送下，汗出为度；金刃伤，用小便洗，贴上药；割伤，破伤风牙关紧，好酒调服1钱，着被盖蒙头约重车行5里地，汗出即愈，如无汗出，涎水亦好，若涎水汗俱无，再服1钱；伤寒，用药1钱，热酒服，汗出立效；蝎螫，用唾津调贴；蛇虫伤，用飞矾和药贴患处，酒服1钱，妇人产后淋血不止，红花煎酒服1钱；多年恶疮，去痂，水洗净，贴净药；脑风，口噙药吹入鼻或用生姜调贴两太阳穴；诸般疮，或茶、或薄荷汤调服1钱出汗；大烧伤，用凉水调贴；咳嗽，用白桑皮汤服1钱，同酒送下；多年疮，用小便洗贴；恶疮疽疖，鱼眼、红丝、疔疮，调药用鹅翅扫上患处；偏正头风，用口噙药水吹入鼻内；遍身疥疮，瘰疬，杖疮有血者，将水贴药，继服2钱，温酒下，不拘时候；舌根疮，外臁疮及一切无名肿毒，新汲凉水调敷患处；诸风倒地，不省人事，好酒调服1钱，被盖出汗立效；骡马等畜揭鞍风，诸药不效，抽身抱膝者，药5钱，酒半碗灌下，将毡盖出汗立效。
制备方法：上为细末。
临床应用：None
各家论述：None
用药禁忌：忌热物半日；得汗时切忌不可见风。
附注：None
