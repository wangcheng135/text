序号：43457
名称：养营益卫补心清肺育脾和肝滋肾膏子
组成：人参3两（切，并隔纸焙），熟地8两（切块，焙），枣仁3两（炒熟），当归身2两（酒拌晒干，炒），鸡腿白术4两（人乳拌，晒干，炒），白芍2两（蜜酒拌，晒干，炒），白茯神2两4钱（焙），远志肉（去心，甘草汁煮透，晒干，焙）1两5钱，雄牛膝2两（酒拌晒干，炒），麦门冬（去心，拌老米炒燥，去米用）2两，五味子1两2钱（打扁，蜜酒拌蒸，炒），肉桂（临磨刮尽粗皮）8钱。上为细末，入后膏子：熟地6两（切块），酸枣仁3两（炒熟，捣碎），当归身2两（酒拌，晒干，炒），鸡腿白术4两（人乳拌，晒干，炒黄），白芍1两5钱（蜜酒拌，晒干，炒），白茯神2两4钱，远志肉（去心，甘草煮透，晒干）1两5钱，怀牛膝2两（酒拌，晒干），五味子1两（捣碎），麦门冬（去心，用老米同拌，炒黄）2两，肉桂（临煎去尽粗皮）8钱。先用建莲子（去心农）2斤，入清水煎取头汁、二汁，去莲子，入上药煎取头汁、二汁，滤去滓，慢火熬成浓膏子。
出处：《冯氏锦囊·杂症》卷十一。
主治：None
加减：None
功效：养营益卫，补心清肺，育脾和肝滋肾。
用法用量：每晚服4钱，用桂园汤送下。
制备方法：将膏子入前药细末和膏子为丸。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
