序号：44799
名称：急救痧气丸
组成：真茅山苍术（米泔水浸3日，炒，研末）1两2钱，真干蟾酥1两2钱，腰面雄黄（另研细）1两2钱，丁香（另研细）1两2钱，枯骨广木香（微烘，勿炒，另研细）1两2钱，飞滑石（另研细）1两2钱，辰砂（水飞过，另研细）1两2钱，麝香3分（要取真原麝黄香佳）。
出处：《春脚集》卷三。
主治：None
加减：None
功效：None
用法用量：凡修合完备，于太阳旺时晒热，盛大碗内，上盖瓦盘，乘热摇颠，丸色发亮，用瓶盛贮备用，勿令泄气。每服3丸，阴阳水送下。痧重者加2丸。
制备方法：将诸药末和匀，再碾千余下，以烧酒浸烊蟾酥捣为丸，如芥菜子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
