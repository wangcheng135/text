序号：70424
名称：紫参丸
组成：紫参1两，秦艽1两，乱发灰1两，紫菀1两，厚朴（炙）1两，藁本2两，雷丸半升，白芷1两，(庶虫)虫半两（熬），贯众3两（去毛），猪后悬蹄甲14枚（炙），虻虫半两（去翅足，熬），石南半两（炙）。
出处：《外台》卷二十六引《小品方》。
主治：五痔，大便肛边清血出；脱肛。
加减：有人热，可除羊髓，以赤蜜代。
功效：None
用法用量：每服15丸，未食酒服，日再，亦可饮下，剧者夜1服。4日肛边痒止，8日脓血尽，鼠乳悉愈，满60日终身不复发，久服益善。
制备方法：上为末，以羊脊骨中髓合猪脂各半升煎，为丸如梧桐子大。
临床应用：None
各家论述：《千金方衍义》：草根木实之可以辟湿热者，痔家恒取用之。如秦艽、白芷、藁本、紫参皆风热药；紫菀、贯众、厚朴、雷丸散结滞药；乱发、猪蹄、虻、(庶虫)，破积血药；独石南一味为肾家风药，肾主二便，特取以祛下焦风毒也。
用药禁忌：忌鱼、猪肉。
附注：None
