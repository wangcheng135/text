序号：52407
名称：香归饮
组成：木香（不见火）1两，人参1两，牡丹皮1两，白芍药1两，干姜（炮）1两，官桂1两，熟地黄1两，丁香（不见火）1两，香附子（炒去毛）1两，藿香1两，厚朴（姜制）1两，茯苓（去皮）1两，缩砂仁1两，莪术（炮）1两，白芷1两，当归1两，沉香1两，青皮（去瓤）1两，白擅1两。
出处：《普济方》卷三二二。
主治：妇人营卫不调，气血不顺，或气盛血弱，经脉不匀，或前或后，或多或少，临行腹痛，淋沥不绝，身体无力，四肢倦怠，筋骨颈疼，面色萎黄，不思饮食，腹肚膨胀。
加减：None
功效：调顺荣卫，通利三焦，温暖五脏，调和饮食，滋血顺气，疏风益损。
用法用量：每服4钱，水1大盏半，加生姜3片，煎至1盏，食前温服，滓再煎服。
制备方法：上为粗末。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
