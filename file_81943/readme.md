序号：81959
名称：镇心丸
组成：紫石英（别研）半两，丹砂（别研）半两，茯神（去木）半两，银屑（别研）半两，雄黄（别研）半两，菖蒲半两，人参半两，桔梗（锉，炒）半两，干姜（炮）半两，远志（去心）半两，甘草（炙半两，锉）半两，当归（切半两，焙）半两，桂（去粗皮）半两，防风（去叉）1两，细辛（去苗叶）1两，铁精（研）1两，防己1两。
出处：《圣济总录》卷十四。
主治：心气虚弱，风热所乘，惊悸不宁，胸中逆气，魇梦参错，谬忘恍惚。
加减：None
功效：None
用法用量：每服15丸，渐加至20丸，米饮送下。
制备方法：上为末，入别研药和匀，炼蜜为丸，如小豆大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
