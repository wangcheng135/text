序号：35164
名称：乳香丸
组成：乳香半两（锉，研入），木香半两，肉豆蔻半两（去壳），当归半两（锉，微炒），青橘皮半两（汤浸，去白瓤，焙），京三棱半两（煨，锉），干漆半两（捣碎，炒令烟出），紫菀1两（去苗土），干姜1两（炮裂，锉），附子1两（炮裂，去皮脐），鳖甲1两半（涂醋，炙令黄，去裙襕），朱砂1分（细研），巴豆1两（去皮心，研，纸裹压去油）。
出处：《圣惠》卷四十九。
主治：宿食不化，心膈气滞，中焦不和，及癥癖积聚，或多呕逆。
加减：None
功效：None
用法用量：每服3-5丸，以温水送下。若有久积聚，常于临卧服5丸愈。
制备方法：上药除乳香、朱砂、巴豆外，并为末，每2匙药末用细荞面一匙相和，更研令匀，滴水为丸，如绿豆大，候干，以浆水煎令沸，下药丸子，煮1-2沸，候药丸子浮上，乃滤出，于竹筛子内晒干。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
