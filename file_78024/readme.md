序号：78040
名称：赛宝丹
组成：炉甘石1两（用火精石制），蕤仁1两（制），琥珀（用新布包，捶碎，研细末）5钱，小珍珠（光明者，用豆腐煮过，温水洗3-4次，布包捶碎，研细末）5钱，玛瑙（重煅）5钱，珊瑚（稍重煅）5钱，车渠（轻煅）5钱，石蟹（又轻煅，热水研）5钱，雌石（入醋内煅7次）5钱，雄石（入醋内煅7次）5钱，金箔200张，银箔200张（共制过，细研水飞，以不刺牙为度，方合作一处，擂十万下，将药包紧，勿令泄气），荆芥5钱，真薄荷5钱，草决明5钱，防风5钱，羌活5钱，白菊花5钱，木贼5钱，千里光5钱，蕤仁（去油）5钱。
出处：《眼科秘诀》卷一。
主治：翳膜红丝遮蔽瞳仁。
加减：None
功效：None
用法用量：点眼中。
制备方法：荆芥以下9味锉细，加水3大碗，浸2-3日，用水煎十余滚，倾出，去滓澄清，熬膏。须留1钟，加乳汁1钟，同熬至1钟，调前项药，捶之，令膏尽捶于药中，名为万捶膏。又捶成大条为赛宝丹。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
