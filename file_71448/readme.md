序号：71464
名称：紫粉灵宝丹
组成：黑铅4两，水银2两（不别修制，每2两水银即入硫黄半两，结成砂子，细研如粉）。
出处：《圣惠》卷九十五。
主治：筋骨风气。
加减：None
功效：添精益髓，神气清爽，好颜色红悦，久服轻健，补暖水脏。
用法用量：每日1丸，空心以津送下。
制备方法：上取火消石于铛中心作一堆子，尖尖装之，堆四面流下些些子，令盖铛底，即取砂子末，细细掺于堆子上，勿令四面散，讫，更研入少许硫黄末盖之，又以碗子盖铛口，四面以湿纸固济缝子，上又以泥如法固济，候干，铛下渐渐以火3-5两，候看得所，加至1斤以来，可烧半日久，又加3两，如此叠叠加至4斤，火断之当铛上下通赤，即渐去火，待冷，轻手揭取，药成一团，以甘草2两，水5升，煎至2升，去滓，煮药泣干，出火毒，干了细研，水飞过，以煮枣肉为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
