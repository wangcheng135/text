序号：77002
名称：黑神丸
组成：漆6两（半生，半用重汤煮一半日令香），神曲4两，茴香4两，木香半两，椒红半两，丁香半两，槟榔（除椒外，5物皆半生半炒）4个。
出处：《苏沈良方》卷四。
主治：肾气、膀胱痃癖，七疝下坠，五膈血崩，产后诸血，漏下赤白，死胎，难产，血瘕。经候前先腹痛不可忍。
加减：None
功效：None
用法用量：凡肾气、膀胱痃癖，七疝下坠，五膈血崩，产后诸血，漏下赤白，并丸分4服；死胎1丸，皆无灰酒下；难产，炒葵子49枚，捣碎酒煎下1丸。诸疾不过3服，元气10服，膈气癥癖5服，血瘕3丸。
制备方法：上丸如弹子大，取茴香末12两，铺盖阴地阴干，候外干，并茴香收器中，极干乃去茴香。
临床应用：血瘕：余族子妇病，腹中有大块如杯，每发痛不可忍，时子妇已贵，京下善医者悉，常服其药莫愈，陈应之曰：此血瘕也，投黑神丸三丸，杯气尽消，终身不复作。
各家论述：None
用药禁忌：None
