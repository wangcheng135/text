序号：4407
名称：木香丸
组成：木香2两，白术1两，槟榔2两，高良姜半两（锉），益智仁半两（去皮），红豆蔻半两（去皮），草豆蔻半两（去皮），神曲半两（微炒），吴茱萸半两（汤浸7遍，焙干，微炒），青橘皮半两（汤浸，去白瓤，焙），蓬莪术1两，枳壳半两（麸炒微黄，去瓤）。
出处：《圣惠》卷九十八。
主治：一切冷气，脏腑久积，脐腹多疼，宿食不化，颜色萎弱。
加减：None
功效：None
用法用量：每服30丸，以生姜、橘皮汤或温酒送下，不拘时候。
制备方法：上为末，以酽醋5升，煎药末一半成膏，入余上药末为丸，如梧桐子大。
临床应用：None
各家论述：None
用药禁忌：None
附注：None
